שְׁאֵרִית עַמְּךָ

חתימה: שלם יוסף


שְׁאֵרִית עַמְּךָ יָהּ / מְקַוָּה חַסְדֶּךָ

שְׁעֵה אֶל שַׁוְעָתוֹ / בְּקָרְאוֹ אֵלֶיךָ

בַּשְּׂרוֹ בִּמְחִילָה / עֲשֵׂה לְמַעַן שְׁמֶךָ

כִּי הִשְׁלִיךְ יָהְבוֹ / בְּיוֹם זֶה עָלֶיךָ

יֹאמַר זְכוֹר לִי אֵל / לְאַהְבָתֶךָ

תְּמוֹךְ אֲשׁוּרַי / בְּמַעְגְּלוֹתֶיךָ:


לְבָבוֹ הֵכִין / מִכָּל טֶנֶף

הֱיֵה עֶזְרָתוֹ / בְּשָׁעַת קֶצֶף

וְאַל יִמְשׁוֹל בּוֹ / אַף וָזַעַף

וְהִצִּיל נַפְשׁוֹ / מִכָּל טֶרֶף

כִּי הִיא לְנֶגְדָּךְ / מִשְׁתַּפֵּכָה:


מְחֵה נָא כָעָב / חֶטְאָהּ וּפִשְׁעָהּ

וְאַל תְּדִינָהּ / כִּזְדוֹן רִשְׁעָהּ

כִּי הוּא אַתָּה / יָהּ צוּר יִשְׁעָהּ

הַרְחֵק מִמָּהּ / לַעֲווֹן בִּצְעָהּ

אוּלַי תִּמְצָא / חֵן בְּעֵינֶיךָ:


יְלִידֵי אַב הֲמוֹן / בָּךְ יִבְטָחוּ

וּבְזִכְרֶךָ / הֵם יִשְׂמָחוּ

רְאֵה נָא עָנְיָם / כִּי אָז שָׁחוּ

וּבְגָלוּתָם / נִשְׁתַּלָּחוּ

כִּי חָטְאוּ עַל / מִצְווֹתֶיךָ:


וִדּוּיֵיהֶם / תְּקַבֵּל בְּרָצוֹן

כִּי הֵם בְּדַלּוּת / נְפוּצִים כַּצֹּאן

בְּיַד כָּל לוֹעֵג / וּמַרְבֶּה לָצוֹן

יָשַׁב בִּפְנִים / וְהֵם בַּחִיצוֹן

עֲשֵׂה עִמָּהֶם / לְמַעַן שְׁמֶךָ:


סְלַח לַעֲווֹנָם / וּלְפִשְׁעֵיהֶם

וּבְרַחֲמֶיךָ / תַּקְבִּילֵהֶם

וְתַצִּיל אוֹתָם / מִשּׁוֹסֵיהֶם

הַמַּכְבִּידִים אֶת / עֻלֵּיהֶם

וְהֵם עַמֶּךָ / וְנַחֲלָתֶךָ:


פְּדוּתָךְ תָּחִישׁ / לְיוֹנָה עֲלוּבָה

כִּי מִקֶּדֶם / הִיא לְךָ אֲהוּבָה

וּבְמַעֲשֶׂיהָ / טוֹבָה חֲשׁוּבָה

וְהַיּוֹם בָּאָה / בְּתוֹם וּבִתְשׁוּבָה

לְבַקֵּשׁ מִמְּךָ / סְלִיחָתֶךָ:
