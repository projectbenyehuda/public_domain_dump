חלום חלם ידיד נפשי בשחר

חתימה: חבור שבזי


חֲלוֹם חָלַם יְדִיד נַפְשִׁי בְּשַׁחַר / וּפִתְרוֹנוֹ יְהֶא טוֹב לָךְ וְנָעִים

בְּתוֹרָה שִׂכְלְךָ נִדְבָּק וּבָחַר / וְנַפְשָׁךְ חָדְלָה מִכָּל פְּשָׁעִים

וְנִדְרָךְ שַׁלְּמָה אַל נָא תְאַחֵר / וְתַצִּיל גּוּפְךָ מִכָּל פְּגָעִים

רְצוֹן אֵל חַי אֲשֶׁר לִוָּךְ וְסָחַר / עֲלֵי אֹהָלְךָ מִצֵּד רְשָׁעִים

שְׁבָה בִּמְקוֹם חֲכָמִים לָךְ יְבוֹחַר / וְאָז יִהְיוּ לְךָ אַחִים וְרֵעִים

בְּלֵצִים תִּמְאֲסָה אָחִי וְתִתְחַר / אֲשֶׁר הֵמָה בְמַעֲשֵׂיהֶם גְּרוּעִים

זְכוֹר יוֹם דִּין אֲשֶׁר לִבָּם יְחַרְחַר / וְעוֹרוֹתָם יְזֹרֶה בָּם נְגָעִים

יְחִי לִבָּךְ בְּחֵן תוֹרָה וְתִבְחַר / וְתִקָּרֵא נְטַע שַׁעֲשׁוּעִים

וְשָׁלוֹם עַל עֲדַת רֵעִים וְאַחִים / כְּמוֹ נָהָר מְרַוֶּה לַזְּרָעִים:
