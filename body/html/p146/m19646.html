אִם נִנְעֲלוּ דַּלְתֵי נְדִיבִים

חתימה: אלשבזי (55: סינן אלף ואללאם ואלמים [סאלם])


אִם נִנְעֲלוּ דַּלְתֵי נְדִיבִים/ דַּלְתֵי מְרוֹם לֹא נִנְעֲלוּ

אֵל חַי מְרוֹמָם עַל כְּרוּבִים / כֻּלָּם בְּרוּחוֹ יַעֲלוּ

כִּי הֵם אֲלֵי כִסְאוֹ קְרוֹבִים / יוֹדוּ שְׁמוֹ וִיהַלְּלוּ

חַיּוֹת שְׁהֶם רָצוֹא וְשָׁבִים/ מִיּוֹם בְּרִיאָה נִכְלְלוּ

גַּלְגַּל וְאוֹפָן רוֹעֲשִׁים

מוֹדִים שְׁמוֹ וּמְקַדְּשִׁים

מִזִּיו כְּבוֹדוֹ לוֹבְשִׁים

וּבְשֵׁשׁ כְּנָפַיִם סְבִיבִים / עָפִים בְּעֵת יִתְגַלְגְּלוּ

יַעְנוּ בְּקוֹל שִׁירִים עֲרֵבִים / יַחַד בְּאוֹתוֹת נִדְגְּלוּ:

לַאכְּרַאמַהוּ אַלכֻּל אַוְגַׄד / סֻבחַאנַהוּ אַללַּהּ אֲחַד

רַבּ אַלגַׄלַאלִי אַלמֻמַגַּׄד / אִחְסַאנַהוּ מִן גַיְר חַד

אַגְׄסַאם וַאַרְוַאחֻן תֻסַאגַׄד / מַן לַם יֻכַּשׁאפֹהּ קַד גְׄחַד

וַאכְׄתַץ מִן סַבעַת אֲקַאלִים/ אַלקֻדְס אלַאוְסַט אַפְצַׄלוּ

סַכַּן בְהוּ אַבַאיַנַא

וַאעְטַאהּ גַׄמִיע אַסְבַאטַנַא

וַאלתַּוְרִיַה אַעְטַא לַנַא

וַאגִׄב נֻוַחִדֹהּ בְתַעְצִׄים / פַצְׄלֹה עֲלַיְנַא שַׁאמִלוּ

נַאדַא אלנַּבִי מוּסַי בְתַכְּלִים/ קַאל: כֻּן לִקַוְמִי מֻרְסַלוּ:


שִׁמְרִי יְחִידָתִי לְדָתוֹ / זַכִּי לְגּוּפִי יִשְׂמְחָה

קִשְׁרִי בְּלִבִּי אַהֲבָתוֹ / בִּשְׁמוֹ לְבַדּוֹ יִבְטְחָה

מֵחֵן גְּבֶרֶת רַעֲיָתוֹ / אוֹר לַשְּׁרִידִים יִזְרְחָה

וִימַלְּאָה בּוֹרוֹת נְגוּבִים / גִּשְׁמֵי בְּרָכָה יִזְּלוּ

וִיפַרְנְסָה עָנִי וְרָשׁ

גּוֹלֶה וְנִרְדָּף נֶחֱלָשׁ

צוּרִי לְבֶן אִישׁ תָּם דְּרָשׁ

כִּי סָבְבוּ עָלָיו כְּלָבִים / מִכָּל עֲבָרִים נִקְהֲלוּ

פֶּרֶא וְאַרְיֵה עִם זְאֵבִים / לִבּוֹ וְנַפְשׁוֹ הִבְהֲלוּ:


בַין אלְפֻנוּן תַשׁתַאק נַפְסִי/ חַיְתׄ אלְעֲרַאיִס וַאלשְּׁרַאב

וִאלַּא דַׄכַרתֻ בַיְת קֻדְסִי / וַאסְוַאר צִיּוֹן אלְכַׄרַאב

אַכְּסַף צִׄיַא עַקלִי וְחִסִּי / וַאלרּוּח יַלחַץ̇ אִכְּתְרַאב

קֻם יַא מֻחִבּ הַאת אלְמַשַׁאמִים/ יֻחְיִי לְרוּחִי אלגַאפִלוּ

זַיְן אלְכַּנַאיִס וַאלְמַעַאלִים / קֻדְסִי וִסִעְדִי אלְמֻקְבִלוּ

פַאלְקַצְד אַעְבֻד כַׄאלִקִי

פִי קַלְב מוּמִן עַאשִׁקִי

יוּפִי בְּוַעְדֻן צַאדִקִי

קַד כַׄצְּנַא פִי עִלְם וִתַעְלִים / לַא יַבְק אלִאנְסַאן גַׄאהִלוּ

מַן נַאלַהוּ צִׄיקֻן וְתַסְקִים / גַׄאלַס לִשַׁכְׄצֻן כַּאמִלוּ:


זֶה הוּא מְקוֹם שָׁלוֹם וְאַחְוָה / חָתָן וְכַלָּה יִשְׂמְחוּ

רִנָּה וְקוֹל שָׂשׂוֹן וְחֶדְוָה/ לִבְנֵי שְׁבָטִים זָרְחוּ

מִבַּת יְפֵהפִיָּה וְנָאוָה / לִי כוֹס יְשׁוּעוֹת שָׁלְחוּ

וָאֶשְׂמְחָה בֵין הַמְּסֻבִּים / מַה טּוֹב וְנָעִים נִקְהֲלוּ

צִירֵי תְעוּדָה יָשְׁרוּ

עַל סוֹד וּמִשְׂרָה חָבְרוּ

לָאֵל, וְלִי מַה יָּקְרוּ

רֵעִים אֲהוּבִים הֵם וְטוֹבִים / סוֹדוֹת נְכוֹחִים מִלְּלוּ

תּוֹרָה נְבִיאִים עִם כְּתוּבִים / מִזֶּה אֱלֵי זֶה יִשְׁאֲלוּ:


יַא נַפְס תוּבִי פִי אלְחֲיַאתִי / מִן לַמְס זַגְיַאת אלְגִׄסוּם

מִן קַבְל מַא יַאְתִי אלְמַמַאתִי / יַהְבֻט כַּמַא רִיח אלסִמוּם

מַן קַד גַלַט פִי אלסּוּ יַאְתִי / וַאללַּהּ גַפַארֻן רַחוּם

לַא בֻדּ מַא לִלדַּיְן תַסְלִים / עִנְד אלְפַתַא אַלפַאצִׄלוּ

יַא מַן אלַאנְפֻס כַׄלְקַהוּ

מִן פַיְץ̇ גַׄוְהַר נוּרַהוּ

וַכֻּלַּהַא פִי כַּפַהוּ

כֻּלּ אלְוֻגׄוּד חַכַּם בִתַחְכִּים / וַהוּ לִכֻּל שַׁי פַאעִלוּ

סִינֻן אַלִף וַאללַּאם וַאלְמִים / מִן כַּרְם גׄוּדַךּ סַאאִלוּ:


אֶחְתּוֹם לְשִׁירִי זֶה בְּהַלֵּל / וּשְׁבָח לְרָם עַל מַהֲלָל

הַשֺּׁם בְּפִי אֹמֶר לְמַלֵּל / לוֹ כָל יְצוּר מוֹדִים כְּלָל

נַפְשִׁי עֲדִי רִקְמָה וּמִכְלוֹל / גּוּפִי לְנַעַלְךָ שְׁלָל

בֵּין הַשְּׂרִידִים הַשְּׁלֵמִים / חִזְקוּ פְעָמַי וַעֲלוּ

בֵּין הַוְּרָדִים נִשְׁכְּנָה

לִי הַגְּבֶרֶת זִמְּנָה

טֶרֶף לְבֵיתָהּ נָתְנָה

פַּתְבַּג וְיֵין מִדַּם עֲנָבִים / יֵינוֹ מְשֻׁמָּר צָלְלוּ

וּבְנֵי מְסִבָּה הָאֲהוּבִים / חֶלְקָם בְּעֵדֶן יִנְחֲלוּ:
