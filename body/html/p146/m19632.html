אַהְבַת הֲדַסָּה

חתימה: אלשבזי משתא


אַהְבַת הֲדַסָּה עַל לְבָבִי נִקְשְׁרָה / וַאְנִי בְּתוֹךְ גּוֹלָה פְּעָמַי צוֹלְלִים:

לוֹ יֵשׁ רְשׁוּת לִי אֶעֱלֶה אֶתְחַבְּרָה / תּוֹךְ שַׁעֲרֵי צִיּוֹן אֲשֶׁר הֵם נֶהְלְלִים:

שַׁחְרִית וְעַרְבִית בַּת נְדִיבִים אֶזְכְּרָה / לִבִּי וְרַעְיוֹנַי בַּחֵשֶׁק נִבְהֲלִים:

בִּנְעִים זְמִירוֹת מִנְּדוֹד אֶתְעוֹרְרָה / וַאְנִי וְרַעְיָתִי בְּרִנָּה צוֹהֲלִים:

בִּינוּ עֲדַת קֹדֶשׁ בְּשִׁירָה חֻבְּרָה / חָתָן וְהַכַּלָּה בְּחֻפָּה נִכְלְלִים:

זֶה יוֹם שְׂמָחוֹת לָאֲיֻמָּה יָקְרָה / כִּי הִיא וְדוֹדָהּ חֵן וְחֶסֶד גּוֹמְלִים:

יַחְדָּו עֲלֵי שֻׁלְחָן וְכוֹס דּוֹדִי קְרָא / זַמֵּן שְׂרִידֵינוּ וְכָל הַנִּסְגְּלִים:

מִכּוֹס יְשׁוּעוֹת אֶשְׂמְחָה וַאְזַמְּרָה / אוֹצִיא לְכָל סוֹדִי וְאָשִׁיב שׁוֹאֲלִים:

שֵׁם טוֹב לְמַשְׂכִּילִים בְּדַעַת יָשְׁרָה / כִּי הֵם עֲלֵי יִצְרָם בְּוַדַּאי מוֹשְׁלִים:

תַּאְוַת לְבָבָם לַעֲשׂוֹת טוֹב גָּבְרָה / יַעְלוּ לְגַן עֵדֶן וְחַיִּים נוֹחֲלִים:

אַהְבַת יְחִידָתִי לְטוּב אֵל נָהֲרָה / בָּרוּךְ שְׁהוּא נוֹתֵן שְׂכַר כָּל פּוֹעֲלִים:

שָׁלוֹם כְּנָהָר לַעֲדָתִי יִנְהֲרָה / זָקֵן וְגַם בָּחוּר וְכָל הָעוֹלְלִים:
