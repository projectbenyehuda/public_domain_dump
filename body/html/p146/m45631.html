אֵיכָה חֲבַצֶּלֶת

חתימה: אלשבזי


אֵיכָה חֲבַצֶּלֶת שָׁרוֹן / תִּתְעַלְּמִי עַל הֲמוֹנֵינוּ

אַחַר הֱיוֹתָם לְבִצָּרוֹן / מָרוּ וְהִסְטוּ לְבָבֵנוּ

וַיִּרְדְּפֵם אַף וְגַם חָרוֹן / וַיְעוֹרְרוּ עוֹל עֲווֹנֵינוּ

חֶבְרַת סְגֻלָּה תְּנִי דּוֹרוֹן / אוּלַי תְּרַצִּי בַּעֲדֵנוּ

לְמַעַן תְּסַפְּרוּ לְדוֹר אַחֲרוֹן / כִּי זֶה אֱלֹהִים אֱלֹהֵינוּ:


לָמָּה נִשְׁכַּב לְמַעְצֵבָה / כִּי יֵשׁ תִּקְוָה לְעָם שָׁבִים

אֵחַר סְמָאֵל וְהֵן לֹא בָא / וּבַעְלֵי עֲצָתוֹ מְעֻרְבָּבִים

הֻשְׁחָר וְאָז הֶחְזִיק טוֹבָה / נַעְשָׂה סְנִיגוֹר לְבַת רַבִּים

קוּמוּ יְדִידִים וְנָשׁוּבָה / אֶל אֵל אֲשֶׁר רַחֲמָיו רַבִּים

נַזְכִּיר זְכוּת שׁוֹכְנֵי חֶבְרוֹן / בְּשֵׁם יְוֹיָ אֱלֹהֵינוּ

לְמַעַן תְּסַפְּרוּ לְדוֹר אַחֲרוֹן / כִּי זֶה אֱלֹהִים אֱלֹהֵינוּ:


שִׁפְעַת רְבִיבִים תְּצַמֵּחַ / תַּשְׁקִיט חֲרוֹן זַלְעֲפֵי חֵמָה

לִבְנֵי תְשׁוּבָה וְהַצְלֵחַ / הַסּוֹבְלִים עוֹל בְּתוֹךְ תֵּימָא

עָם הַמְּשַׁעְבֵּד וּבוֹטֵחַ / בְּאֵל חַי וְהוּא לוֹ יְקוֹמֵמָה

רַחוּם וְחַנּוּן וְסוֹלֵחַ / חֶבְרַת עֲנִיָּיו יְרַחֵמָה

בְּסֵפֶר מְחִילָה וְזִכָּרוֹן / הִרְחִיק מִמֶּנּוּ אֶת פְּשָׁעֵינוּ

לְמַעַן תְּסַפְּרוּ לְדוֹר אַחֲרוֹן / כִּי זֶה אֱלֹהִים אֱלֹהֵינוּ:


בַּעְלֵי תְשׁוּבָה הֲלוֹא בִינוּ / כִּי זֶה עֵת רַחֲמִים וְרָצוֹן

לִבְּכֶם מִתְּמוֹל הָכִינוּ / יִטַּהֲרָה לֵב וְגוּף חִיצוֹן

וּבְבֵית מִקְדָּשׁ מְעַט לִינוּ / אוּלַי תְּעוֹרְרוּ לְעָלוֹת צֹאן

לַהֲמוֹן עֲדַתְכֶם הַזְמִינוּ / וְלִבָּם יַעֲזוֹב חֵטְא וְלָצוֹן

אָז יֵדְעוּ כִי יֵשׁ יִתְרוֹן / וּמִי צוּר זוּלָתִי אֱלֹהֵינוּ

לְמַעַן תְּסַפְּרוּ לְדוֹר אַחֲרוֹן / כִּי זֶה אֱלֹהִים אֱלֹהֵינוּ:


יוֹם זֶה זְמוּנִים בְּנֵי חֶלֶד / לַעֲרוֹךְ הַדִּין וְהַחֶשְׁבּוֹן

וְזֶה יַעֲלֶה כַף וְזֶה יֵרַד / הֵם כַּעֲבָדִים לְמוּל רִבּוֹן

אִישׁ תָּם יִשְׂמַח וְרָע יֶחֱרַד / מֵאֵין וַתְּרָה וְעֵרָבוֹן

רָשָׁע מְנָאֵץ שְׁאוֹל יוּרַד / וְאִישׁ בֵּינוֹנִי יִתְלַבֵּן

וְצַדִּיק שְׁהוּא נֶאֱמָן יָרוֹן / לַחֲזוֹת בְּנוֹעַם אֱלֹהֵינוּ

לְמַעַן תְּסַפְּרוּ לְדוֹר אַחֲרוֹן / כִּי זֶה אֱלֹהִים אֱלֹהֵינוּ:


יְשָׁרִים אֲשֶׁר חוֹמְדִים יוֹשֶׁר / זוֹכִים וְעוֹשִׂים רְצוֹן קוֹנָם

חֶפְצָם אֱמוּנָה וְטוֹב כּוֹשֶׁר / תַּמִּים בְּלִבָּם וּבִלְשׁוֹנָם

רַק נוֹאֲלִים רוֹדְפִים עוֹשֶׁר / תּוֹעִים בְּהֶבְלָם וּבִזְדוֹנָם

רָגַע מְעַט קָט וְהֵן קָשַׁר / שָׂטָן מְקַטְרֵג עֲלֵי אוֹנָם

יִכְלוּ יְמֵיהֶם שְׁנוֹת חֶסְרוֹן / יַצְמִיתֵם יְוֹיָ אֱלֹהֵינוּ

לְמַעַן תְּסַפְּרוּ לְדוֹר אַחֲרוֹן / כִּי זֶה אֱלֹהִים אֱלֹהֵינוּ:
