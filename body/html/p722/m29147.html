שְׂמַח חָתָן עֲלוֹז בִּימֵי נְעוּרִים

[לחתן]


שְׂמַח חָתָן עֲלוֹז בִּימֵי נְעוּרִים / וְיִיטַב לִבְּךָ בִּימֵי בְּחוּרִים:

וְתָגֵל נַפְשְׁךָ בָּאֵל וְתָשִׂישׂ / אֲשֶׁר הִלְבִּישְׁךָ בִּגְדֵי פְאֵרִים:

עֲשֵׂה לָךְ כֹּל אֲשֶׁר יֵשׁ עִם לְבָבָךְ / וְשַׂמַּח עִמְּךָ לֵב הַחֲבֵרִים:

כְּבָר נִרְצוּ פְעָלֶיךָ לְאֵל חַי / וְהֵנִיחַ לְךָ מִכָּל עֲבָרִים:

וְהִרְחִיב לָךְ וְאִמֵּץ אֶת לְבָבָךְ / וְשָׂם לִבּוֹת מְשַׂנְאֶיךָ שְׁבוּרִים:

וְלָכֵן לֵךְ אֱכוֹל וּשְׁתֵה בְּשִׂמְחָה / וְאַל תִּירָא חֲמַת קָצִים וְצָרִים:

רְוֵה דּוֹדִים וְהִתְעַלֵּס בְּדַדֵּי / צְבִיָּה נִמְשְׁלוּ כִּשְׁנֵי עֳפָרִים:

בְּפִיהָ צוּף וְחָלָב עַל לְשׁוֹנָהּ / וְכַחַמָּה לְחָיֶיהָ מְאִירִים:

וְיִפְעָתָהּ כְּמוֹ סַהַר בְּתֻמּוֹ / וְקוֹמָתָהּ כְּמוֹ קוֹמַת תְּמָרִים:

וְאֵיךְ תִּשְׁאַל לְאוֹר שֶׁמֶשׁ וְסַהַר / וּמֵאוֹרָהּ יְאִירוּן כָּל חֲדָרִים:

רְכַב עַל בָּמֳתֵי עֶרֶשׂ אֲהָבִים / וְהִתְעַנֵּג בְּמַרְבַדִּים יְקָרִים:

עֲצַת שָׁלוֹם תְּהִי בֵּינוֹת שְׁנֵיכֶם / וְתִשְׁגֶה בָּהּ וְלֹא תִשְׁגֶּה בְּזָרִים:

וְהָאֵל יִתְּנָה בָרוּךְ מְקוֹרָךְ / וְיַטֶּה לָךְ שְׁלוֹמוֹת כַּנְּהָרִים:

וְיַפְרֶךָ וְיַרְבֶּךָ וְתִרְאֶה / בְּנֵי בָּנִים לְבָנֶיךָ יְשָׁרִים:
