קְחוּ נָא מוּסָרִים

קְחוּ נָא מוּסָרִים / סְגֻלּוֹת הָעִבְרִים / וְהַקְשִׁיבוּ שִׁירִים / וְשִׁפְטוּ מֵישָׁרִים

בְּשִׂיחֵי הַהַגְרִי / לְבִֵין אֶחָיו יַפְרִיא / וּבַמָּרוֹם יַמְרִיא / הֲמוֹן הַקַּדָּרִים

אֲשֶׁר שָׂח הַנִּכְאֶה / וְהוּא כָעוֹף יִדְאֶה / בְּמַעֲשֵׂהוּ יַרְאֶה / לְנֶגֶד הַמּוֹרִים

וְהִצִּיל הוֹן מֵהֶם / וְעָמַד לִפְנֵיהֶם / וְעָרֵךְ הוּא לָהֶם / תְּאֵנִים עַזְרוּרִים

וְהָיוּ לִתְרוּפוֹת / לְרַגְלֵי נִגָּפוֹת / לְמֵרָחוֹק צוֹפוֹת / לְתַנִּים וּשְׂעִירִים

וְרַגְלֵיהֶם שׁוֹנוֹת / וְנִסְתָּרוֹת עוֹנוֹת / לְנִשּׂוּאֵי בָנוֹת / לְנִבְזִים וִיקָרִים

בְּנִיבוֹ יִתְפָּאֵר / בְּחוּצוֹתָיו נִאֵר / וּמַחְשִׁיךְ לֹא יָאֵר / לְנִין הָאַשּׁוּרִים

בְּרִיבוֹ הִתְגַּלַּע / וְכִמְעַט הוּא נִבְלַע / וְרֹאשׁוֹ עַל סֶלַע / כְּחֶרֶס נִשְׁבָּרִים

וְאִם יָדַע נִכְסַף / לְתַפּוּחִים אָסַף / בְּמַשְׂכִּיּוֹת כֶּסֶף / בְּזָהָב נֶחְבָּרִים

וְעַל כַּמָּה פָנִים / חֲכָמִים וּנְבוֹנִים / בְּתִפְאַרְתָּם שׁוֹנִים / וְהֵם הָאַבִּירִים

לְשׁוֹנוֹת מַגִּידוֹת / פְּשָׁטִים עִם סוֹדוֹת / מְשָׁלִים עִם חִידוֹת / כְּמַיִם נִגָּרִים

לְמוּלָם תַּלְפִּיּוֹת / מְתוּקוֹת לַפֵּיוֹת / לְרָחוֹק צוֹפִיּוֹת / בְּיַמִּים וּנְהָרִים

וּמֵהֶם אַגָּדוֹת / לְיוֹדְעָם נִכְבָּדוֹת / פְּשׁוּטוֹת וּמְעִידוֹת / וּבָהֶם נִסְתָּרִים

וְדִבְרֵי קַבָּלָה / וְהֵם מַעְלָה מָּעְלָה / וְעוֹלִים בִּמְסִלָּה / לְנַפְשׁוֹת הַיְצוּרִים

וּבָהֶם הֵיכָלוֹת / בְּחָכְמָה נִבְדָּלוֹת / לְמַלֵּא מִשְׁאָלוֹת / לְבָבוֹת הַבָּרִים

לְטוֹב בִּבְלִי קִצְבָּה / וְהוּא עוֹלָם הַבָּא / בְּנַחַת וּבְשׁוּבָה / לְמַעְמַד הַשָּׂרִים

וְזֶהוּ הַתַּכְלִית / בְּרֹאשׁ גֻּלּוֹת עִלִּית / וְאֶפֶס הַתַּכְלִית / חֲסִידִים נִשְׂכָּרִים.
