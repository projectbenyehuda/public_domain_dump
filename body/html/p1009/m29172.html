לִקְרַאת פְּנֵי שַׁבָּת אָשִׂישׂ וְאָגִילָה

לִקְרַאת פְּנֵי שַׁבָּת / אָשִׂישׂ וְאָגִילָה

אֶתְעַלְּסָה כִּמְשׂוֹשׂ / חָתָן עֲלֵי כַּלָּה


זִיו הוֹד מְאוֹר פָּנַי / שֵׁשֶׁת יְמֵי הַחוֹל

יוּעָם וְרַעְיוֹנַי / נַפְשִׁי בְּטוֹב תִּבְחֹל

חֶבֶל הֲמוֹן מוֹנַי / מִתְקוֹמְמִים לִזְחֹל

אָשִׁיב לְמוּל עֵינַי / דָּת הָאֱמֶת לִנְחֹל

אֶתְבּוֹנְנָה סִבַּת / פִּלְאֵי אֲדוֹן מַעְלָה

הַסּוֹד בְּלֵב אֶכְמֹס / לַשֵּׁם וְלִתְהִלָּה


כָּלוּ יְגוֹנוֹתַי / יוֹם בּוֹא מְנוּחָתִי

עֹצֶם פְּלִיאוֹתַי / מִשְׁנֶה אֲרוּחָתִי

אֶפְתַּח צְרוֹרוֹתַי / לִכְבוֹד סְעוֹדָתִי

אֵיטִיב זְמִירוֹתַי / שִׁיר עַל נְגִינָתִי

כּוֹסוֹת כְּעֵין לַבַּת / יִדְמוּ וְלִסְגֻלָּה

שֻׁלְחָן יֳפִי לִפְרֹשׂ / אָלוּן בְּתוֹךְ כִּילָה


רֶכֶב אֱלֹהִים חַי / רִבְבוֹת וְלַאֲלָפִים

שׁוֹמְעִים לְקוֹל שִׂיחַי / הֵם עַד כְּרוּב עָפִים

תּוֹךְ מִקְהֲלוֹת אַחַי / אֶהְגֶּה וְהֵם צוֹפִים

סוֹדוֹת לְבֶן יוֹחַאי / בַּחֵךְ דְּבַשׁ צוּפִים

גַּם אֶמְצְאָה קִרְבַת / שַׁדַּי בְּכָל מִלָּה

אֶשְׁתַּעְשְׁעָה לִנְסֹס / בָּהֶם וְאוֹחִילָה.


יוֹשֵׁב עֲלֵי עָב קַל / שִׂכְלוֹ מְאֹד בַּפַּשׁ

סוֹד מַחֲצִית שֶׁקֶל / חִבַּר לְכָל נֶפֶשׁ

חֶצְיָהּ וְגַם סִקַּל / מִטִּיט וּמֵרֶפֶשׁ

דַּרְכָּהּ וְלֹא עִקַּל / עַד תִּמְצְאָה נֹפֶשׁ

תִּדְחֶה לְכָל דִּבַּת / עַמִּים וְתַשְׁפִּילָה

רוּחָם וְגַם תִּרְמֹס / נֵצַח וְעַד סֶלָה


הִנֵּה שְׁתֵּי תֵּיבוֹת / אַקְדִּים לְוַיְכֻלּוּ

אֵל חַי רְצֵה נִדְבוֹת / מִפִּי וְלֹא יִכְלוּ

עוֹד מִמְּצוֹא אֶשְׁבֹּת / בַּלֵּב וְאִם יַעְלוּ

וּזְכֹר זְכוּת אָבוֹת / קֶדֶם יְמֵי חִילּוּ

גָּלוּת לְאוּם רַבַּת / יֵשׁ קֵץ לְכָל תִּכְלָה

מִיּוֹם לְיוֹם תַּעֲמֹס / רַחֵם וְלֹא תִכְלָא.
