משל נח: נַעַר וְאִישׁ

דּוֹר תַּהְפּוּכוֹת הֵמָּה / נוֹתְנֵי דְמֵי שָׁלוֹם בַּמִּלְחָמָה


נַעַר לְפִי תֻמּוֹ הוֹלֵךְ בַיַּעַר / וְרוּחַ קָדִים סֹעָה מִסָּעַר / וְיָדָיו הָיוּ בְחֵיקוֹ נְחִתִּים / כִּי מִמְּזָרִים קָרָה וְהַקֶּרַח כְּפִתִּים / וְשֶׁלֶג וּבָרָד, וְחֹם הָאֵשׁ יַחְמֹד / לִפְנֵי קָרָתוֹ מִי יַעֲמֹד / וַיִּרְאֵהוּ אִישׁ הַיַּעַר שֹׁכְנִי סְנֶה / וַיַחֲמֹל עָלָיו וַיֹּאמַר: הִנֵּה / דֵּי בָעֵר וּמְדוּרָה גְדוֹלָה / וַיְסִיתֵהוּ עַד בֹּאוֹ הָאֹהֶלָה / וַיְדַבֵּר עַל לִבּוֹ כַּאֲשֶׁר הוּא קְרוֹבוֹ וְגֹאֲלוֹ / וַיְכַסֵּהוּ בַּבְּגָדִים וְלֹא יֵחַם לוֹ / וּבְטֶרֶם בֹּאוֹ שָׁם בְּרוּחַ פִּיו וּבְאַפָּיו / נָפַח עַל צִפָּרְנָיו לְגָרֵשׁ קָרַת כַּפָּיו / וְהָאִישׁ מִנָּה אֶת מַאֲכָלָיו / בַּאֲשֶׁר עֵינוֹ חָסָה עָלָיו / וּמִמַּשְׁקָיו הֵבִיא לְאַמֵּץ אֶת רוּחָם / וַיִּקַּח מֵהֶם וַיִּחַם / וַיִּתֵּן לַנַּעַר לִשְׁתּוֹת עִמּוֹ / וַיָּשֶׁת אֶל פִּיו וְלֹא שָׁתָה מֵרֹב חֻמּוֹ / וַיִּפַּח בְּרוּחוֹ לְהָקֵר יֵין כּוֹסוֹ / וַיֹּאמֶר לוֹ הָאִישׁ בְּכַעֲסוֹ / מַהֵר נָא וָצֵא מֵאִתִּי / אִישׁ כָּמוֹךָ בְּקִרְבָתִי לֹא חָפַצְתִּי / הֲלֹא לְהַעֲבִיר קָרַת יָדְךָ נָפַחְתָּ בַבֹּקֶר / וְלָעֶרֶב עַל חֹם יֵינְךָ נָפַחְתָּ לְהָקֵר / חָלִילָה לִי לְפֶה תַהְפֻּכוֹת כָּמוֹךָ לְקָרֵב לְרֵעַ וְעָמִית / אָכֵן מֵחֶבְרַת יִשְׁרֵי לֵב אוֹתְךָ אַצְמִית.


וְהַמָּשָׁל בְּאַנְשֵׁי חֹנֶף אֲשֶׁר לֹא כֵנִים / בְּלֵב וָלֵב וּבְהַרְבֵּה פָנִים / נוֹתְנֵי דְמֵי שָׁלוֹם בַּמִּלְחָמָה / כִּי דוֹר תַּהְפּוּכוֹת הֵמָּה / וּמִכָּל אֶחָד תֵּצֵא חֲנֻפָּה / וְחִשַּׁק רָאשֵׁיהֶם וְצִפָּה / בְּחֹנֶף וּבְתַרְמִית לְדַבֵּר אֶל שְׁכֵנוֹ / עָמָל וָאָוֶן תַּחַת לְשׁוֹנוֹ / וְאַחֲרֵי דַבְּרוֹ הַוַּת נַפְשׁוֹ וְחֵפֶץ לְבָבוֹ / יִלְעַג וְיַהֲפֹךְ אֲמָרָיו מִדֵּי דַבְּרוֹ בוֹ / בַּאֲשֶׁר לִבּוֹ לֹא נָכוֹן עִמּוֹ / עַל כֵּן לֹא עָמַד בּוֹ טַעְמוֹ / כִּי אֵין בְּפִיהוּ נְכוֹנָה / וְאִם גָּבַר בָּאָרֶץ לֹא לֶאֱמוּנָה / וּמַכּוֹת חַדְרֵי בֶטֶן דִּבְרֵי בֶלַע / וּכְפַטִּישׁ יְפֹצֵץ סָלַע / וְלֹא יַחְפֹּץ אִישׁ בְּקִרְבָתוֹ / כִּי אִם בְּצַלְמוֹ וְכִדְמוּתוֹ.
