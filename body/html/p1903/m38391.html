משל צה: פֶּסֶל וְאִישׁ

מַעֲנֶה רַךְ יָשִׁיב חֵמָה / וְלֹא תָבֹא לְמַשְּׂטֵמָה

פֶּסֶל פְּסָלוֹ יֹצְרוֹ וְכִלָּהוּ / חָרָשׁ עֲשָׂאָהוּ וְלֹא אֱלֹהִים הוּא / עָשִׁיר אֶחָד דָּרֹשׁ דָּרַשׁ / לִקְנוֹת אֹתוֹ מִן הֶחָרָשׁ / כִּי בָחַר לַעֲשֹׂתוֹ אֱלֹהוֹ / יָפְיוֹ קוֹמָתוֹ וְגָבְהוֹ / לִמְאֹד בְּעֵינָיו יָשַׁר / חָקוּק וּמָשׁוּחַ בַּשָּׁשַׁר / וְאִישׁ אַחֵר בִּשְׁכוּנָתוֹ / הָאֹמֵר לִקְנוֹת אֹתוֹ / לַעֲשֹׂתוֹ לַמֵּת מַצֵּבָה / עַל אֶבֶן מַשְׂכִּית טוֹבָה / וִיכַסֶּה בּוֹ קֶבֶר קְרוֹבוֹ / אֲשֶׁר הוּא כְנַפְשׁוֹ אֲהֵבוֹ / וְהַפֶּסֶל לַאֲשֶׁר פְּסָלוֹ / בָּכָה וַיִּתְחַנֶּן לוֹ / לָתֵת לוֹ גוּפוֹ בִשְׁאֵלָתוֹ / וּכְבוֹדוֹ בְּבַקָּשָׁתוֹ / וַיֹּאמַר: הִנְנִי שֶׁלְּךָ וְאַתָּה בִּי עֹצֵר / אֲנִי חֹמֶר וְאַתָּה יֹצֵר / וּבְיָדְךָ כָּל רוּחִי וְהוֹדִי / וּקְלוֹנִי לְהָמִיר בִּכְבוֹדִי / יָדֶיךָ עִצְּבוּנִי וַיַּעֲשׂוּנִי / בִּמְחוּגָה וְשֶׂרֶד תְּתָאֲרֵנִי / מֵעֵץ לֹא יִרְקַב נִבְחַרְתִּי / מֵאֲשֶׁר יָקַרְתִּי בְּעֵינֶיךָ נִכְבַּדְתִּי / וּמְלֶאכֶת נְקָבַי וְתֻפַּי / פָּעַלְתָּ וּזְרוֹעִי וְכַפַּי / מֵרֵאשִׁית דָּבָר רְאֵה מַה סּוֹף / וּלְמַעֲשֶׂה יָדֶיךָ תִכְסוֹף / וְאַל תִּתְּנֵנִי בְּנֶפֶשׁ צָרָי / אֲשֶׁר אָמְרוּ לִהְיוֹת בְּעֹכְרָי / לִקְנוֹתִי מִמְּךָ לְכַסּוֹת מֵת / מִמֶּנִּי, נְבֵלָה סְרוּחָה וְנִרְדֶּמֶת / וּלְחַלֵּל יְפִי יִפְעָתִי / מִי יִתֵּן תָּבֹא שֶׁאֱלָתִי / וְתִמְכְּרֵנִי לַאֲשֶׁר אֹתִי יִירָא / וַאֲשֶׁר בִּשְׁמִי יִקְרָא / כִּי בִּכְבוֹדִי לְךָ כָּבוֹד / וְהָאִישׁ אֲשֶׁר אֹתִי יַעֲבוֹד / יְכַבֶּדְךָ כִּי לִי יִתְפַּלֵּל / וְאֶת כְּבוֹדִי לֹא יְחַלֵּל / וְכֻלָּם בָּאִיִּים יְחַוּוּ / כִּי לְמַעֲשֵׂה יָדֶיךָ יִשְׁתַּחֲווּ / אִם לֹא תַחְפֹּץ פָּעָלְךָ וִיקָרוֹ / מָה הוֹעִיל פֶּסֶל כִּי פְסָלוֹ יֹצְרוֹ / וַיִּשְׁמַע יֹצְרוֹ אֶת תְּפִלָּתוֹ / וּלְעֹבְדוֹ וְלַמְכַבְּדוֹ מָכַר אֹתוֹ.

וְהַמָּשָׁל לְהַחֲנִיף כָּל אֲשֶׁר תִּגְבַּר יָדוֹ עָלֶיךָ / לְהַמְתִּיק לוֹ סוֹד לְהַמְשִׁיךְ לִבּוֹ אֵלֶיךָ / וּלְפַתּוֹת לְבַב עִקֵּשׁ וְלִמְצֹא שִׁלּוּמִים / לְמַעַן יִיטַב לְךָ וְהַאֲרַכְתָּ יָמִים / וְאָמַר אֶחָד מִן הַחֲכָמִים / אֲשֶׁר מַחֲזֵה שַׁדַּי יֶחֱזֶה / מֻתָּר לְהַחֲנִיף הָרְשָׁעִים בָּעוֹלָם הַזֶּה / וְחָכָם אַחֵר אֶת אָזְנִי גֹּלֶה / תַּעֲלָא בְּעִדָּנֵהּ סְגִיד לֵהּ.
