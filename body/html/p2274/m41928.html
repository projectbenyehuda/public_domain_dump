עוֹשֶׂה שָׁלוֹם

סימן: עבדאללה.


עוֹשֶׂה שָׁלוֹם / מִשְׂגָּב שְׁמוֹ

אֶל קֵץ עֵילוֹם / יוֹחִיל עַמּוֹ

יָשִׁיב הֲלוֹם / אֶל מְקוֹמוֹ

מֵיטַב כַּרְמוֹ / עֲשִׂירִיָּה.

אֲיֻמִַָּתִי / יִפְדֶה אֵל יָהּ

וְנִשְׁמָתִי / תְּהַלֵּל יָהּ

לְאֵל עֶלְיוֹן / עֲלִילִיָּה

עַל עַיִן עַל עַיִן.


בְּצוֹק עַמִּים / שׁוֹכֵן אִתָּם

לָךְ נוֹאֲמִים / אֶת רִנָּתָם

לְרַחֲמִים / תִּתֵּן אֹתָם

וְשַׁוְעָתָם / חִישׁ שְׁמַע יָהּ.

אֲיֻמִַָּתִי יִפְדֶה


דָּבְקָה נַפְשָׁם / אַחֲרֶיךָ

תֵּן בְּרֹאשָׁם / מְשִׁיחֶךָ

לְהוֹרִישָׁם / אֶת אַרְצֶךָ

וּבְנֵה בֵיתְךָ / שְׁלִישִׁיָּה.

אֲיֻמִַָּתִי יִפְדֶה


אֵל, שׁוּר רַבִּים / לוֹחֲמִים לִי

כֻּלָּם אוֹיְבִים / וְשׂוֹנְאִים לִי

וְנִצָּבִים / לְהָרַע לִי

וּמַחְסֶה לִי / וְעֻזִּי יָהּ.

אֲיֻמִַָּתִי יִפְדֶה


לַשָּׁוְא חוֹזִים / יִתְאַמְּרוּ

שׁוֹכְבִים הוֹזִים / עֵת יָגוּרוּ

עָלַי עַזִּים / פֶּה פָעֲרוּ

וַיֹּאמְרוּ / לֹא יִרְאֶה יָהּ.

אֲיֻמִַָּתִי יִפְדֶה


לָכֵן הָשֵׁב / גְּמוּל לָהֶם

מֵאֵין יוֹשֵׁב / בְּעָרֵיהֶם

שְׁמַע הַקְשֵׁב / לְדִבְרֵיהֶם

עֲשֵׂה לָהֶם / פְּלִילִיָּה

אֲיֻמִַָּתִי יִפְדֶה


הָאֵר פָּנִים / לְעַם נִבְדָּל

חַדֵּשׁ שָׁנִים / יָשִׂישׂ, יִגְדַּל

בְּמַעְדַּנִּים / וְלֹא יִדַּל

עֹז וּמִגְדָּל / מַעֲשֵׂי יָהּ.

אֲיֻמִַָּתִי יִפְדֶה
