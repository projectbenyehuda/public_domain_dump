אֲנִי אֲנִי בְּתוֹךְ אֲנִי

אהבה


אֲנִי אֲנִי / בְּתוֹךְ אֲנִי

וּמַחֲנִי / כְּמוֹ אֳנִי

בְּיַם עֳנִי; / וְהִנֲּנִי,

בְּחַנְּנִי / לְמַעֲנִי, / כְּאוֹבוֹת;


בְּחַבְּלִי / בְּאָהֳלִי,

בְּמַעֲלִי / בְּפוֹעֲלִי

מְחוֹלֲלִי / וְגוֹאֲלִי,

בְּסִכְלִי / בְּעַקְּלִי / נְתִיבוֹת.


רְעוּץ אֲרִי / בְּלִי צֳרִי,

דְּחוּי חֳרִי / בְּשׁוֹרֲרִי

מְצוֹרֲרִי – / וְזֶה פְּרִי

דְּבַר מְרִי, / בְּבַקְּרִי / מְשׁוּבוֹת.


מְכוֹן צְבִי / בְּתַעֲבִי –

וְאוֹהֲבִי / דְּמוּת צְבִי,

וְסוֹחֲבִי / וְיֵרְדְּ בִּי

בְּבֵית שְׁבִי / בְּאַף – לְבִי / קְרָבוֹת.


חֲסִין, לְכָה / וְאֶשְׁלְךָ

וְחֶבְלְךָ / פְּדֵה לְךָ

בְּגָדְלְךָ; / וְעֻלְּךָ

וְסִבְלְךָ / וְאָפְלְךָ – / לְאוֹיְבוֹת!


חֲדַד מְכוֹר / וְחֵת עֲכוֹר

וְלוֹ הֲכוֹר / וּפִיו סְכוֹר,

בְּבוּז נְכוֹר / עֲדֵי שְׁכוֹר

וְאַךְ זְכוֹר / לְבֶן בְּכוֹר / אֲהָבוֹת;


וְשׁוּב שְׁנֵה / וְחִישׁ פְּנֵה

כְּיוֹם סְנֶה, / וְעַם קְנֵה

וְעִיר בְּנֵה / וְתַחֲנֶה,

וְתַעֲנֶה / לְנַעֲנֵה / לְבָבוֹת:


׳אֲקָרְבֵם, / אֲשַׂגְּבֵם

וְאֶקְּבֵם, / אֲשׁוֹבְבֵם,

אֲיַשְּׁבֵם, / אֲסוֹבְבֵם,

אֲחוֹבְבֵם / וְאֹהֲבֵם / נְדָבוֹת.׳
