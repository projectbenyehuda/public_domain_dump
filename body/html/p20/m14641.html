אֶת עָצְבִּי אַגִּיד

אֶת עָצְבִּי / אַגִּיד – וְיֶחֱרַד / אוֹיְבִי וְיֵבְךְּ לִי.

אֵיךְ לִבִּי / יִשְׁכֹּן – וְנָע בְּיַד / צָרִים, וּבוֹעֲלִי

בָּעַל בִּי? / אֵין לִי צֳרִי – לְבַד / תִּקְוָה לְגוֹאֲלִי.

לוּלֵי שִׁמְךָ, חִזְקִי וְחִשְׁקִי, / סַפְתִּי בְּבֵיִת שְׁבִי!


בְּחֶבְלִי / הִשְׁכִּין מְחַבְּלַי / אֵלִי בְּזַעֲמוֹ,

בֻּלַּע לִי / מִבּוּז מְקַלֲלַי – לוּ כִבְּדוּ שְׁמוֹ.

בְּכִסְלִי / רֶשֶׁף, וְשׁוֹאֲלַי / יִרְאוּ אֲשֶׁר כְּמוֹ

חוֹלַת אַהְבָה אָנִי – לְרָחְקִי / מֵאֶרֶץ הַצְּבִי.


רַב שִׂיחִי / מִבּוֹא בְמֶחֱקָר / כִּי רַע מְאֹד וּמַר.

מַה כֹּחִי עַתָּה – וְנֶעֱקָר / סוֹד הַיְּסוֹד כְּבָר?

אֵיךְ אֶחִי – / כִּי אֶחֱזֶה יְקָר / אֵל מִכְּרוּב לְהַר

עוֹלֶה? אַחַר שָׁכַב בְּחֵיקִי / בָּרַח כְּמוֹ צְבִי.


'מָה תֶּהֱמִי / עָלַי בְּלִי דֳמִי? / דֹּמִּי – וְהִנְנִי!

יוֹם קוּמִי / פִּתְאֹם תְּנֻחֲמִי. / אָכֵן לְמַעֲנִי

מִקָּמִי / אֶקֹּם; וְאֵיךְ שְׁמִי / יַכְעִיס – וְאֵל אֲנִי

וְאֵלֶּה בָנַי, בְּנֵי חֲשׁוּקִי, / אַבְרָהָם אוֹהֲבִי.'
