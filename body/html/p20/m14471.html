אֵיכָה לְבָבִי יַעֲמֹד רֶגַע

אֵיכָה לְבָבִי יַעֲמֹד / רֶגַע וְאֵיךְ לֹא יֶחֱרָד —

וּבְבֵית אֱלֹהַי יַעֲבֹד / טָמֵא, וְעָרֵל בִּי יְרָד?


בָּזוּ לְצַלְעִי, אָמְרוּ / כִּי אֵין לְמַכָּתִי צֳרִי,

לִטְרֹף עֲדָרַי נָעֲרוּ / כִּכְפִיר וְשָׁאֲגוּ כָאֲרִי,

מָרוּ בְךָ וִימָרֲרוּ / חַיַּי בְּעוֹדָם בֵּית מְרִי —

לָעַם אֲשֶׁר עָמְדוּ בְסוֹד / דָּת נִתְּנָה יוֹם מַעֲמַד

סִינַי, אֲשֶׁר הָיְתָה יְסוֹד / עוֹלָם בְּיוֹם מַה נֶּחֱמָד!


רָצוּ וְרָדְפוּ לַעֲקֹר / אוֹתִי וְגֶדֶר פָּרְצוּ,

וּכְבוֹד אֱלֹהַי לַחֲקֹר / דִּמּוּ — וְלִשְׁמוֹ נִאֲצוּ,

חֹק חָלְפוּ, עָזְבוּ מְקוֹר / חַיִּים וְהַמֵּת יֶחֱצוּ.

לִבִּי לְזָר לֹא יַחֲמֹד, / יִבְטַח בְּאֵל שַׁדַּי לְבָד,

נִשְׁבַּע — וְלֹא יָמִיר כְּבוֹד / אֵל חַי בְּמֵת נָתוּן בְּבָד!


מֶה עָרְבוּ צָרוֹת עֲלֵי / שֵׁם גּוֹאֲלִי, מַה נִּכְבְּדוּ!

מַה לִּי, וְאִם צָר יַעֲלֶה / כָּל יוֹם — וְעַמִּי יֵרְדוּ.

מֻבְטָח אֲנִי כִי נוֹחֲלֵי / שֶׁקֶר כְּרֶגַע יֹאבְדוּ.

מַה תֶּהֱמֶה, לּבִּי, לְדוֹד / סָר מִמְּךָ? כִּי לֹא לְעַד

יִזְנַח, וְאִם הִרְחִיק נְדוֹד — / חַכֵּה לְיוֹם קוּמוֹ לְעַד!
