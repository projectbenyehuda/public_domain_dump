אֲנוּשָׁה אַל תִּירְאִי

תוכחה


אֲנוּשָׁה, אַל תִּירְאִי / [……]בונֵךְ

מְשׁוּבָתֵךְ [ה…<אי>] / […]ה צְפוּנֵךְ

וְלוֹ[….] קִרְאִי / וְשִׁפְכִי תַחֲנוּנֵךְ

שִׁמְעִי בַת וּרְאִי / וְהַטִּי אָזְנֵךְ.


בְּתַאְוָתֵךְ וְהֶבְלֵךְ / עִם גָּזְלֵךְ וְעָשְׁקֵךְ

תש[…] מְנַת חֶבְלֵךְ; / וּמַאְוָיֵךְ וְחִשְׁקֵךְ

וְכָל עֲמָלֵךְ וְסִכְלֵךְ – / בְּכָל מַאְכָלֵךְ וּמַשְׁקֵךְ;

קוֹל תּוֹעֵךְ וְנִכְלֵךְ – / בָּם מִשְׁתֶּה לְיֵינֵךְ.

רְאִי, סֹעֲרָה עֲנִיָּה, / בְּרֹב עָנְיֵךְ [שְׁ]לוּלָה

מי […]ניה וּב[.]בך בְּלוּלָה

מוש[…<ניה>] / בְּיַם חֶטְאֵךְ צְלוּלָה

בְּבוֹאֵךְ [….<ניה>] / לֹא יוֹעִיל הוֹנֵךְ.


מְכוּרָה בְאַשְׁמָתֵךְ / סְלִיחָה תֶּהֱמִי –

וְאַתְּ בִּתְנוּמָתֵךְ / וּבְעַצְלֵךְ תִּדְמִי.

בְּשׁוּבֵךְ לְאַדְמָתֵךְ / אֲזַי כַּיָּם תִּנְהֲמִי,

[…]תודי [..<תֵךְ>] / בְּתַעְתּוּעֵי זְמַנֵּךְ.
