[בְּלִי תִקְוָה וְלֹא חֶדְוָה]

בְּלִי תִקְוָה / וְלֹא חֶדְוָה / בָּרְחוּ כַצֵּל יְמוֹתָי

וְנוֹסָפוּ וְנֶאֱסָפו / כְּחו­ל הַיָּם / אַשְמוֹתָי

עֲדֵי תַרְתִּי / וּבָחַרְתִּי / מָוֶת מֵעַצְמוֹתָי

וְשָאַלְתִּי / שְׁאוֹל מַחְסֶה / וְעָפָר אֹהֶל וּמִכְסֶה

וְשָׂנֵאתִי / אֶת הַחַיִּים / כִּי רַע עָלַי הַמָּעֲשֶׂה:

קה' ב, יז


מְתֵי לְבָבוֹת / וּמַחְשָבוֹת / וְכָל נֶפֶש מוּדָעַת

לְמַאֲמָרִי / וּמוּסָרִי / נְטוּ אֹזֶן שוֹמָעַת!

וְעַל פְּשֲעִים / לִרְגָעִים / תְּהִי עַיִן דּוֹמָעַת

וּמוֹעַצָה / בְּיָד אַמְּצָה / קְחוּ מִפִּי הַדָּעַת

והַתַּאֲוָה / תְּנוּ דָוָה / וְהָרוּחַ נִכְנָעַת

הֲכִי תֵבֵל / כְּעָש תִּבֶל / וְיִתְרוֹנָהּ מִגְרָעַת

וְתַחְרִיב / טֶרֶם תוֹשִׁיב / וְתַעֲקֹר אַחֲרֵי טָעַת

וְתִשְׁתַּפַּךְ / וְתִתְהַפַּךְ / כְּמוֹ חוֹתַם טַבָּעַת

וְאִם הוֹנוׁ / בֶאֱנוֹשׁ תְּנַסֶּה / לְמָחָר אוֹתוֹ תְּשַסֶּה!

וְשָׂנֵאתִי / אֶת הַחַיִּים / כִּי רַע עָלַי הַמָּעֲשֶׂה:


שְׁמָהּ תֵּבֵל / כִּי הֶבֶל / הֵמָּה כָל שְׁבִילֶיהָ

וְהִיא כְגִנָּה / רַעֲנָנָּה / וְסַם מָוֶת בְּעָלֶיהָ –

שִׁית זוֹנָה / וְאֵין אֱמוּנָה / בְּפִיהָ לִבְעָלֶיהָ

תַּפְתְּ אוֹתָם / לְהַחְיוֹתָם / וְרַבִּים הֵם חֲלָלֶיהָ

נַאֲפוּפִים / וּכְשָפִים / הֲמוֹנֵי מַעֲלָלֶיהָ

וְאִם תָּחְלִיק / אוֹ תַמְתִּיק / לְךָ אִמְרֵי הֲבָלֶיהָ –

אַל תַּאֲמֵן / כֵּי אֹמֶן / בְּפִיהָ וְשָׁוְא פְּעָלֶיהָ!

וְאִם תַּרְנִין / חֲמַת תַּנִין / תִּמְסֹךְ בְּיֵין מִלֶּיהָ

זְנַח יָפְיָהּ / והוֹד עֶדְיָהּ / בָּם תַּרְהִיב כְּסִילֶיהָ

וְאַל תִּרְדֶה / צוּף דַּדֵּי / בְּתוּלֶיהָ וּתְעַשֶּׁה

וְשׂנֵאתִי / אֶת הַחַיִּים / כִּי רַע עָלַי הַמַּעֲשֶׂה:


הִוָּעֲדוּ / וְהִתְעַתְּדוּ / לְעִי מִדְבָּר מֵעִיר מְתִים

אֲבָל יְשֵׁנִים / בְּאַשְמַנִּים / אַךְ הֵם פְּגָרִים מֵתִים

אֲשֶר חָיוּ / וְהֵם הָיוּ / הָאוֹכְלִים וְהַשּׁוֹתִים

מֵחֲמֻדּוֹת / וּבֵית מִדּוֹת / וּמֵרָהִיטֵי בְּרֹתִים

וּמֵעֲלִיּוֹת / לְגֵאָיוֹת / אוֹ לֲאַחַת הַפְּחָתִים

יָשִׂיחוּ וְיוֹכִיחוּ / בְּלִי פֶה / עֲרוּמִים וּפוֹתִים

דְּעוּ הוֹלְכִים / כִּי מְשׁוּכִים / אְתֶּם מִבְּלִי עֲבוֹתִים

לִשְׁאוֹלהָ / כְּצֹאן לְמִכְלָה / וּכְבָקָר לָרְפָתִים.

וְלוּ יִהְיֶה / אֱנוֹשׁ אַרְיֵה / קוֹלָם לְבָבוֹ ימְסֶה

וְשׂנֵאתִי / אֶת הַחַיִּים / כִּי רַע עָלַי הַמַּעֲשֶׂה:


נ­ותֵן חֵן / וְהַבּוֹחֵן / לֵב עַבְדּוֹ וְכִלְיוֹתָיו

וְהַנּוֹצֵר / כִּי יֵצֶר / אֱנוֹשׁ הֶבֶל וּמַחְשְבוֹתָיו

אַף הָסֵר / וְאַל תְּיַסֵּר / אוֹתוֹ בְאֵש חַטּׁאוֹתָיו.

וְקַח מַלְכִּי / נַפְשׁוֹ כַי / לֹא טוֹב הוּא מֵאֲבוֹתָיו

הֲכִי חֲטָאָיו / וְתַחֲלֻאָיו / יִרְבּוּ בְאֹרֶךְ שְנוֹתָיו

וְהוּא יִתְאָב / אֱלֵי מַשְׁאַב / מֵי מַעְיְנֵי יְשׁוּעוֹתָיו

וְגַם יִכְלֶה / אֱלֵי הֵיכְלֵי / עֹנֶג וּמַחֲנוֹת מְשָׁרְתָיו!

וְאוּלַי שָׁם / אֲשֶׁר אָשַׁם / יִמְחוּ מֵי סְלִיחוֹתָיו

וְאָז יֵרוֹם / וְיִתְנַשֵּׂא / וְאוֹרְךָ עָלָיו תְכַסֶּה

וְשׂנֵאתִי / אֶת הַחַיִּים / כִּי רַע עָלַי הַמַּעֲשֶׂה!
