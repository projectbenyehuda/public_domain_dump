אֶל מִי אֶשָּׂא

אֶל מִי אֶשָּׂא דִבְרֵי רִיבוֹת / וּבְאֹזֶן מִי אָשִׂים אַחְוָה,

וּמְיֻדָּעַי קָשְׁרוּ עָלַי / וַיּפֵרוּ אֶת־הָאַחְוָה,

הִשְׁפִּילוּ נֵס אֵמֻן חִנָּם / וַיָּרִימוּ חֹטֶר גַּאְוָה.

לֹא אֵדַע אִם חָמְדוּ הֵפֶךְ / אוֹ אִם פִּשְׁעִי דַרְכָּם עִוָּה,

הִקְצוּ גֶבֶר גָּבַר שֹׁדוֹ / אַךְ נַפְשׁוֹ כָּל הַיּוֹם דָּוָה.

הֶעְרִימוּ סוֹד יַחַד עָלָיו / גַּם לֵב לֵוִי עִמָּם נִלְוָה.

אֵיכָה יִשְׁגֶּה צֶדֶק מַהְגֶּה / שׁוֹמֵר תּוֹרָה וִירֵא מִצְוָה,

מַשְׂכִּיל, זִכְרוֹ נֹפֶת פִּיּוֹת / וּלְכָל עַיִן אִמְרוֹ תַאְוָה,

רֻכְסֵי מוּסָר הֵשִׁיב בִּקְעָה / וּפְנֵי אַרְצוֹ דֵּעוֹ שִׁוָּה,

בַּהְדַר שֵׂכֶל הִכְתִּיר דּוֹרוֹ / וַעְלֵי פָנָיו הוֹדוֹ שִׁוָּה.

יוֹדוּ יוֹדְעֵי בִינָה כִּי לֹא / אֶחָד מֵהֶם אֵלָיו שָׁוָה,

כִּי כָל נִמְצָא יֵשׁ לוֹ מוֹצָא / וּלְיָם שִׂכְלוֹ אֵין לוֹ תַאְוָה,

אַךְ מַעְבָּדָיו וּנְעִים דּוֹדָיו / רוֹשׁ יָגוֹן עִם צוּפֵי חֶדְוָה,

כַּאְשֶׁר נוּדוֹ וּכְאֵב בִּגְדּוֹ / פֶּצַע הוֹסִיף עָלָיו מִכְוָה.

מַצְפּוּן לִבִּי הֵן בִּכְתָבִי / וּלְשׁוֹן שִׁירִי סוֹדוֹ חִוָּה.

אִם עֵץ אַהַב הִזְקִין שָׁרְשׁוֹ / וּבְאֵשׁ בֶּגֶד גִּזְעוֹ נִכְוָה –

מֵרֵיחַ מֵי דוֹדַי יִפְרַח, / כִּי עַב חִשְׁקִי אוֹתוֹ הִרְוָה,

אוּלַי יָשׁוּב לַעְשׂוֹת מִגְדֵי / חֶסֶד, כִּי יֵשׁ לָעֵץ תִּקְוָה.

אִם יֵיטִיב לִי אוֹ יָרֵעַ / עֵדוּת דּוֹדִי כָּל־עֵת נָאוָה,

יָגוּר בַּאְשֶׁר יִבְחַר, בֵּינוֹת / צַלְעוֹתַי לוֹ מוֹשְׁבוֹת שַׁלְוָה,

אָכֵן נַפְשִׁי אֵלָיו תִּתְאָו / מִכְתַּב יָדוֹ רַק מִתְאַוָּה.
