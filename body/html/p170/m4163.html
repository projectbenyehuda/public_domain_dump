[מִפִּלְאִי מְרוֹמוֹת]

מִפִּלְאִי מְרוֹמוֹת / נִטָּיוּ בְחָכְמָה

וּמִמּוֹפְתֵי אֲדָמוֹת / נִתְלוּ עַל בְּלִימָה

יָדַעְתִּי כִּי כֹל תּוּכָל / בְּאוֹן וְעָצְמָה

וְלֹא יִבָּצֵר / מִמְּךָ מְזִמָּה:


שְׁחָקִים אֲשֶר נִתְלוּ / כְּאֹהֶל נִמְתָּחוּ

לָעַד לֹא יִבְלוּ / וְגַם לֹא יִזָּחוּ

וְהָאֲרָצוֹת נִתְלוּ / וּלְמַטָּה שָׁחוּ –

וְלוּ יְאַו אֵל יִכְלוּ / כֶּעָשָׁן נִמְלָחוּ

וְעִמָּם הָאֲדָמָה / תִּשָּׁאֶה שְׁמָמָה:


הְכִינוֹתָ פְּגָרִים / וְנָפַחְתָּ נְשָׁמוֹת

אֵלֶּה לָעֲפָרִים / וְאֵלֶּה לַמְּרוֹמוֹת

וְכֵן חֵלֶק יְצוּרִים / נְקָמוֹת וּנְעִימוֹת

וּנְפָשִׁים טְהוֹרִים / יִרְאוּ בְעֵין מְזִימּוֹת

אוֹתוֹת תַּעֲלֻמָה / הֲכִינוֹתָ בְּחָכְמָה:


יְגִיעַ כַּפֶיךָ / אֳלוֹהַּ אַל תְּחַבֵּל

וּבַעֲלוֹת קִצְפֶךָ / מְלַאכְתְּךָ אַל תְּנַבֵּל

הֲיֶחֱרֶה אַפֶּךָ / עַל סָס כְּצִיץ נוֹבֵל

אוֹ תִקַּח נְקָמָה / מֵעָפָר וְרִמָּה?!
