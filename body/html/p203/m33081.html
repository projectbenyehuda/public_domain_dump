משולש [אהל צדק ומקדש קדש]

סימן: א"ב יצחק בן יהודה גיאת [א]ליסאני

ח


         אֹהֶל צֶדֶק / וּמִקְּדַּשׁ קֹדֶשׁ

         בֵּית תּוּשִׁיּוֹת / וְאֶדֶר הַיְקָר

         גָּר בְּמִשְׁכְּנֵי / סוֹד רוּחַ הַקֹּדֶשׁ

         דֵּעָה לִנְחֹל / וְחֹסֶן וִיקָר

5        הִנְחִיל בְּרָכוֹת / אַדְנֵי הַקֹּדֶשׁ

         וְאוֹרָה וְשִׂמְחָה / וְשָׂשׂוֹן וִיקָר

          זָחַל וְיָצָא / מֵאֵת פְּנֵי הַקֹּדֶשׁ

          חָרַד מִשּׂוֹטֵם / וּמִמְּקוֹמוֹ נֶעְקָר

          טִבְעֵי פְּגִיעָה / בִּמְקוֹם הַקֹּדֶשׁ

10        יִסַּד וְעֵינוֹ / רָאֲתָה כָל יְקָר

          כְּבוֹד יְקַר הֲלִיכוֹת / אֱלֵי מַלְכִּי בַקֹּדֶשׁ

          לְמִי יַחְפֹּץ הַמֶּלֶךְ / לַעֲשׂוֹת יְקָר

          מַחֲזֶה נוֹרָאוֹת / נֶאֱדָר בַּקֹּדֶשׁ

          נִגְלָה וּדְבַר־יְיָ / הָיָה יָקָר

15       סֻלָּם חָזָה וְסוֹד / שַׂרְפֵי קֹדֶשׁ

          עוֹלִים וְיוֹרְדִים לִרְאוֹת / דְּמוּת אָדָם בִּיקָר

          פִִלְאֵי רָזֵי / סִינַי בַּקֹּדֶשׁ

          צָפָה וּלְיִרְאֵי אֵל / מַה נַּעֲשָׂה יְקָר

          קָדֹש נִגְלֵיתָ / בְּהַדְרַת קֹדֶשׁ

20        רְפוּאַת עֶצֶב שְׁבוּר לֵב / בְּעֵינֶיךָ יָקָר

          שִׁמְךָ בַּעַל לְכָל שָׂרֵי קֹדֶשׁ

          תִּקְרָא וּלְבַעֲלֵיהֶן / יִתְּנוּ יְקָר

          יְצִיאֵי מֵעֶיךָ / זֶרַע הַקֹּדֶשׁ

          צְבָאֵימוֹ כִּצְבָא רוּם / מִסְפַּר לֹא יֶחְקָר

25        חֵילֶיךָ נְדָבוֹת / בְּהַדְרֵי קֹדֶשׁ

          קֹדֶשׁ מֵחָכְמָה / וּמִכָּבוֹד יָקָר

          בַּר לֵבָב / שֻׁלַּח עֶזְרוֹ מִקֹּדֶשׁ

          נֶעֱזַר וּדְבַר / חַסְדְּךָ מַה יָּקָר

          יִעֵד הָעֲשִׂירִי / יִהְיֶה קֹדֶשׁ

30        הוֹן אֶמְצָא לְעַשֵּׂר / וְכָל מַעְשַׂר בָּקָר

          דַּרְכּוֹ הִצְלַחְתָּ / וְהָיָה קֹּדֶשׁ

          הִגְדַּלְתּוֹ וְחִנַּנְתּוֹ / כָּל הוֹן יָקָר

          גִּנַּנְתּוֹ בְּשׁוּבוֹ / אֶל עִיר הַקֹּדֶשׁ

          יְחִידָתוֹ מֵאֲרַמִּי / נִשְׂגְּבָה וַתִּיקָר

35       אַךְ עֵת דִּמָּה / לְהָרַע בַּקֹּדֶשׁ

          תְּמַכְתּוֹ וּמִבֵּיתוֹ / רַגְלוֹ הוֹקַר

          זָר לֶאֱכוֹל / אֶת פִּרְיוֹ קֹדֶשׁ

          יָזַם לִהְיוֹת תָּפֵל / וְהוּא עִקָּר

          סָבַל כִּי זָר / לֹא יֹאכַל קֹדֶשׁ

40        אַךְ שִׁבְטוֹ יִמְחַץ / פַּאֲתֵי מוֹאָב וְקַרְקַר

         נִהַגְתּוֹ בְּשָׁלוֹם / אֶל בֵּיתוֹ קֹדֶשׁ

          בִּטְחוֹנוֹ לֹא נִכְלַם / וְשִׁבְרוֹ לֹא שִׁקַּר

          יָהּ כֵּן תָּחוֹס / עַל עַם הַקֹּדֶשׁ

          לְחַלּוֹת פָּנֶיךָ / מִהַר וּבִקַּר

45        מֵחֵטְא תְּטַהֲרֵם / וְהָיוּ קֹדֶשׁ

          כַּיּוֹם וּבְעֵינֶיךָ / דָּמָם יֵיקַר

          תַּעֲנֵם שָמַעְתִּי / בְיוֹם מִקְרָא קֹדֶשׁ

          תְּפִלַּת כֹּל / וְלֹא יִהְיֶה בְךָ עָקָר
