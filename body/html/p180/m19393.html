[קְחָה הַשִּׁיר וּמַצְפֻּנָיו]

ולה רחמה אללّה [ולו ירחמהו האל]

קְחָה הַשִּׁיר וּמַצְפֻּנָיו / וְדִקְדּוּקָיו וְעִנְיָנָיו

עֲלֵה וַעְמֹד בְּאֶשְׁנַבּוֹ / וְהִתְיַצֵּב בְּחַלּוֹנָיו

וְרַחֵף עַל־בְּנֵי מוּסָר / כְּמוֹ־נֶשֶׁר עֲלֵי בָנָיו

מְאַס כָּל־שִׁיר וְשִׁפְלוּתוֹ / וְהִתְרַהֵב בְּעֶלְיוֹנָיו

בְּחַר הַשִּׁיר וְהַדָּבָר / אֲשֶׁר דָּבֻר עֲלֵי אָפְנָיו

אֲדוֹן נַפְשִׁי יְהִי לִבּוֹ / וְקַשֻּׁבוֹת יְהוּ אָזְנָיו

לְהָבִין אֶת־מְלִיצָתִי / וְגַם־שִׁירִי וְתַחְנוּנָיו

אֲשֶׁר יִדְמֶה אֱלֵי־מֶלֶךְ / וְהוּא דֶגֶל עֲלֵי פָנָיו

וְהוּא יִהְיֶה מְעִיל בּוּצוֹ / וְהוּא זָהָב לְפַעְמֹנָיו

וְאָז כָּל־שָׁר לְךָ יַבִּיט / וְאָז יִשָּׂא לְךָ עֵינָיו

וְיֹאמְרוּ הֵן כְּתָמָר זֶה / וּמִי יֹאחֵז בְּסַנְסִנָּיו.
