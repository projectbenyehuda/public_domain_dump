בְּפִי חַרְבִּי

בְּפִי חַרְבִּי וּבִלְשׁוֹנִי ־ חֲנִיתִי, / וּמָגִנִּי וְצִנָּתִי ־ שְׂפָתִי

וְשִׁירִי עַל לְבַב שׁוֹמְעָיו כְּפַטִּישׁ / יְפוֹצֵץ צוּר וְאָרִיק בַּחֲמָתִי

אֲנִי כִדְבַשׁ וְחָלָב לִמְאַהְבַי / וְרֹאשׁ פֶּתֶן לְכָל מֵמִיר עֲצָתִי

לְבָבִי עֵר וְגַם עֵינַי פְּתוּחוֹת / לְכָל סָתוּם, וְלֹא עָרְבָה שְׁנָתִי

וְהִנֵּה הַתְּבוּנָה אֵם לְנַפְשִׁי / וְהַחָכְמָה וְהַדַּעַת אֲחוֹתִי

יְקָרָה מִפְּנִינִים הִיא לְעֵינַי / וְהָאָרֶץ לְפָנַי כַּאֲמָתִי

לְמַעַן הִיא כְזוֹנָה, שֶׁבְּיוֹם זֶה / בְּבֵית רֵעִי וְיוֹם אַחֵר בְּבֵיתִי

וְעַל כֵּן לֹא דְרַשְׁתִּיהָ, וְשַׂמְתִּי / שְׁמָר חֻקִּי דְרִישָׁתִי, וְדָתִי

וְחֶלְקִי מֵהֲמוֹנֶיהָ ־ לְבָבִי, / וְנַפְשִׁי ־ מִשְּׁלָלֶיהָ מְנָתִי.
