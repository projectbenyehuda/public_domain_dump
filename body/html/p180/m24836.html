שׁוּבִי בְּתוּלָה

שׁוּבִי בְּתוּלָה / לִנְוֵה זְבוּלָה / בְּרִנָּה וְצָהֳלָה / אֲקַבֵּץ פְּזוּרֵךְ

לְבַשֵּׂר נֶחָמָה / לַשּׁוֹמֵמָה / כִּי בְאַחְלָמָה / אֶבְנֶה דְבִירֵךְ

מֵחָבוֹר וַחֲלַח / אֲקַבֵּץ מְשֻׁלָּח / מִקֵּן וּמְפֻלָּח / לְעוֹבְרֵי דָרֶךְ

הֵן בָּא מְשִׁיחֵךְ / לְטִירַת מְנוּחֵךְ / יַעַן בִּטְחֵךְ / בְּאֵל חַי צוּרֵךְ

קוֹל יֶשַׁע אֲבַשֵּׂר / לְגוֹלִים עֶשֶׂר / וְשׂוֹטֵן אֲיַסֵּר / וּבוֹצֵעַ בֵּרֵךְ

טְמוּנִים אֲגַלֶּה / לְעַם לִי יְחַלֶּה / מְשִׁיחָם יַעֲלֶה / פּוּרָה דוֹרֵךְ

נֹאַם דְּבָרִי / אֲקוֹמֵם לְעִירִי / קוּמִי אוֹרִי / כִּי בָא אוֹרֵךְ.
