התלאה מנשא

ומיצירתו השקולה בישוב דעתה וקינתו המצערת שבה הוא מקונן על אביו.

ואמר על אביו את הקינה הזאת והיא נקראת מסע המות.


הֲתִלְאֶה מִנְּשֹׂא אוֹ מִי עֲזָרָךְ / וְאוּלַי הַדְּמִי הַיּוֹם גְּבָרָךְ

וְעוֹד לֹא עָבְרָה כָזֹאת לְבָבִי / וְאַל תָּגוּר סְפוֹת, כִּי בָא מְגוֹרָךְ

אֲחָזָתָךְ רְעָדָה מִכְּאֵבִי / כְּאִלּוּ מַסְמְרוֹת נִטְעוּ בְקִירָךְ

וּמֵאַנְחָתָךְ קָדְרוּ שְׁחָקִים / וְדַע כִּי לַיְלָה יִבְכּוּ לְצִירָךְ

וְאוֹמֵר: הַכְּאֵב הֵמַס לְבָבוֹ, / עֲנִיתִיו: לִבְּךָ הֵמַס וְהֵרַךְ

לְאַט, כִּי מִשְּׂאֵתִי הַיְּגוֹנִים / עֲזָבוּנִי קְשֵׁה לֵב ־ וַאֲנִי רָךְ

וְהַחָכְמָה תְשִׂיחֵנִי בְאֶבְלִי: / עֲזַבְתַּנִי וּפִתַּחְתָּ חֲגוֹרָךְ

אֱמֹר: לָמָּה תְהַלֵּךְ אַט, כְּאִלּוּ / עֲלֵי גַבֵּי בְנֵי עַיִשׁ מְדוֹרָךְ?

וְלָמָּה תַעֲבֹר עָלַי וְתִפְנֶה / בְּפָנֶיךָ ־ וְאֵין שָׁלוֹם בְּעִירָךְ

וְלֹא שַׂמְתִּי לְךָ בַבִּין מְנַחֵם / וְהָעוֹזֵר תְּבוּנָה לֹא יְבֹרָךְ?

וְחַיֵּיכֶם, לְבוּשִׁי לֹא קְרַעְתִּיו, / אֲבָל מִנִּי כְאֵב נִקְרַע וְנִדְרָךְ

עֲמֹד, מָוֶת, וְאַל תִּקְצֹר אֲנָשִׁים ־ / בְּאָבִי הַגְּבִיר נִשְׁלַם קְצִירָךְ

הַשּׁוֹכֵן בֵּין מְעֵי בֶטֶן אֲדָמָה, / לְךָ מֵעַי, לְךָ יֶהְמוּ בְזָכְרָךְ

תְּמַהּ מֵעֹז פְּנֵי תֵבֵל, אֲשֶׁר לֹא / לְבוּשׁ תִּקְרַע כְּמוֹ נִקְרַע סְגוֹרָךְ

עֲלֵי תֵבֵל תְּהִי אֶלֶף קְלָלָה / וְעַל זֶה הַזְּמַן שֶׁהוּא אֲסָרָךְ

וְחָלִיתָ ־ וְהָיָה חָלְיְךָ בִי, / וְכָבֵד הֶחֳלִי עָלַי וְאָרָךְ

כְּאִלּו הֶחֳלִי זוֹלֵל וְסוֹבֵא / אֲשֶׁר אָכַל עֲדֵי כָלָה שְׁאֵרָךְ

וּמֵאֵן הַזְּמַן לָתֵת לְתֵבֵל / כְּפִי אָכְלָהּ לְבַד עַצְמָךְ וְעוֹרָךְ

מְצָאָךְ הֶחֳלִי רֵעַ טְהַר לֵב / וְעַל כֵּן לַאֲחִי נַפְשׁוֹ קְשָׁרָךְ

וּבֶגֶד אַחֲרֵי מִזֶּה אֲזָרָךְ / וּפִתַּח בָּחֳרִי זֵרָךְ וְזֵרָךְ

הֲמִזֶּה הַזְּמַן אֶקַּח נְקָמָה / וְאִם מֵהֶחֳלִי אוֹכֵל בְּשָׂרָךְ?

אֱמֶת, כָּל מִשְׁבְּרֵי תֵבֵל מְצָאתִים, / וְעָלַי כָּבְדוּ, קַלִּים לְשִׁבְרָךְ

קְבָרָךְ הַזְּמַן בִלְבַב יְגוֹנִים / וְהַיָּגוֹן בְּתוֹךְ לִבִּי קְבָרָךְ

וְשֶׁמֶשׁ אָסְפָה נָגְהָהּ בְּעֵינַי / כְּאִלּוּ נֶאֶסְפָה שֶׁמֶשׁ בְּקִבְרָךְ

וְסַבּוּךָ בְּאַנְחוֹתַי עֲנָנִים / וְרֵיחַ מֹר עֲלֵי פִיחַ עֲפָרָךְ

וְכִמְעַט מִבְּכִי עֵינַי וּבִכְיַת / שְׁחָקִים קִבְרְךָ יִמַּס וְיֵרָךְ

וְלוּ אֶבְכֶּה מְעַט מִדֵּי יְגוֹנִי / אֲזַי מֵרֹב בְּכִי נַחַל עֲבָרָךְ

וְלוּ מֵימֵי פְרָת כִּדְמֵי דְמָעַי ־ / לְעוֹלָם לֹא עֲבָרוֹ אִישׁ בְּמִדְרָךְ

וְאִלּוּ קָרְבוּ מֵאֵשׁ לְבָבִי / אֲזַי בָּהּ נִשְׂרְפוּ מֵישַׁךְ וְשַׁדְרָךְ

וְלוּ חָכַם לְךָ, אָבִי, לְבָבִי ־ / אֲזַי נִגְזַר, כְּמוֹ מָוֶת גְּזָרָךְ

הֲכָזֹאת תִּהְיֶה נִסְגָּר בְּתַחְתִּית / וְאֶשָּׁאֵר אֲנִי לִשְׁכֹּן חֲצֵרָךְ?

וְנֵר יִכֶב וְיֵעָדֵר מְאוֹרוֹ ־ / וְכָבִיתָ וְלֹא נֶעְדַּר מְאוֹרָךְ

אֲהָהּ עַל נָפְלְךָ, כֶּתֶר זְמַנִּי / וְאוֹי כִּי מֵעֲלֵי רֹאשִׁי הֱסִירָךְ

אֲהָהּ גַּם הָהּ וְאוֹי לִי וַאֲבוֹי לִי / אֲשֶׁר לֹא אֵדְעָה מִמִּי מְכָרָךְ

וְלוּ שָׁווּ בְךָ חֶפְצֵי אֲדָמָה ־ / אֲזַי נָקֵל כְּמוֹ יָקָר מְחִירָךְ

וְלוּלֵי כִי לְבָבִי לֹא יְנֻחָם ־ / בְּנֵי עָשׁ נִחֲמוּנִי בַעֲבוּרָךְ

בְּמוֹתָךְ אָבְלוּ אַנְשֵׁי אֲדָמָה / וְאוּלָם שָׂמְחוּ אַנְשֵׁי מְגוּרָךְ

וְאִם שׁוֹכְנֵי שְׁאוֹל יוֹדְעִים ־ עֲלֵיהֶם / לְהִתְדַּפֵּק עֲלֵי פֶתַח דְּבִירָךְ.

שְׁקֹט, אָבִי, וְאַל תִּשֹּׁם לְמַעֲנִי ־ / בְּיוֹם מָחָר נְהִי רֵעִים בְּקִבְרָךְ

וְהַחַיִּים בְּיָדֵנוּ שְׁאֵלָה / נְשִׁיבֶהָ, אֱמֶת, לַאְשֶׁר יְצָרָךְ.

אֲשֶׁר הֵמַר לְבָבִי בַיְּגוֹנִים / וּבַיֹּשֶׁר וּבַתֹּם הֶאֱמִירָךְ

יְשַׁוֶּה חַסְדְּךָ עִם רֹב חֲסָדָיו / וְצִדְקָתָךְ וְתֻמָּתָךְ וְיָשְׁרָךְ

וְיַעַל נַפְשְׁךָ, הִיא הַיְקָרָה / וְעַד נַחַל עֲדָנָיו יַעֲבִירָךְ

וּבִצְרוֹרוֹ, אֱמֶת, תִּהְיֶה צְרוּרָה / וְיִהְיֶה שֵׁם אֱלֹהֵינוּ מְבֹרָךְ!
