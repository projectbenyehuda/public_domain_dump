[שיר־שבח]

וּגְבִיר יְמִינוֹ כַעֲנָנִים עֵת / גִּשְׁמֵי בְדֹלָחִים מְפַזֶּרֶת

גֹּזֵר עֲלֵי יָמִים וְאֵין מֵנִיא / וּבְנֵי זְמַן הֵבִיא בְמָסֹרֶת

גֹּדֵר בְּכַף חַסְדּוֹ וְאֵין פֹּרֵץ / גָּדֵר אֲשֶׁר בֹּנָה וְגֹדָרֶת

אָסַף נְפֹצֵי דָת וְלִבּוֹ לָהּ / אָרוֹן וְצַלְעֹתָיו כְּכַפֹּרֶת

יֹעֵץ חֲכַם רוּחַ כְּאִלּוּ הִיא / מִבֵּין כְּרוּבֵי אֵל מְדַבֶּרֶת

הוֹנוֹ וְקִנְיָנוֹ רְצוֹן קֹנוֹ / מֹאֵס בְּגִנְזֵי דַר וְסֹחָרֶת

אַבִּיר עֲנָוָה מַחְגְּרוֹתָיו [שָׂם] / וַיַּעֲטֶה צֶדֶק לְאַדָּרֶת

תִּשְׂמַח אֲדָמָה בוֹ וּמֵרָב־גִּיל / חַסְדּוֹ בְּלִי לָשׁוֹן מְשׁוֹרָרֶת

וּנְשִׂיא נְשִׂיאִים לַצְּמִיתֻת כָּל / מִשְׂרָה מְכוּרָה לוֹ בְּמִמְכָּרֶת

עֹרֵךְ לְכָל עֹבֵר אֲרֻחֹתָיו / אֶל יַד שְׁעָרִים עַל מְרוֹם קָרֶת

הֵקִים אֱמֶת עַל דָּת וּמִשְׁקֹלֶת / אַחַר הֱיוֹת סָרָה וְנֶעְדָּרֶת

רָמָה גְדֻלָּתוֹ עֲלֵי גֹדֶל / כָּל שָׂר כְּרוּם קוֹמָה עֲלֵי זָרֶת

חַסְדּוֹ כְסַהַר עַל פְּנֵי יָמִים / בָּהִיר וְכָל חֶסֶד כְּבֶהָרֶת

שֶׁמֶשׁ צְדָקָה לַעֲדַת רֵעָיו / מַרְפֵּא וְלִמְשַׂנְאָיו מְבַעָרֶת

אָנִי יְדִידוּתִי לְךָ חֹנָה / מִנִּי מְקוֹם כָּבֵד וְיֹתָרֶת

שֹׁתָה רְבִיבֵי חַסְדְּךָ עֹשָׂה / פֶרִי וְלִרְגָעִים מְבַכָּרֶת

אִם מַעֲלָלָךְ רָם עֲלֵי שִׁירִי / אַהְבָה תְהִי עָלָיו מְכַפָּרֶת

תִּשָּׂא בְרָכָה מֵאֱלֹהִים עוֹד / יוֹנָה עֲלֵי עָנָף מְזַמָּרֶת.
