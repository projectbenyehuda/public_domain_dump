שְׁלַח מִנְזָר

שְׁלַח מִנְזָר / לְעַם נִגְזָר / אֲשֶׁר הָפְזַר / הֵן וָהֵן

פִּינְחָס בֶּן / אֶלְעָזָר / בֶּן אַהֲרֹן / הַכֹּהֵן


לְנַחֵם צֹאן / וְקֵץ קִיצוֹן / וְיוֹם רָצוֹן / קְרָא לָהֶן

מְשֹׁךְ בְּחֵמָה / בְּנֵי אָמָה / וְיוֹם נְקָמָה / עֲשֵׂה בָהֶן

הֲפֹךְ כִּסְדוֹם / אֶרֶץ אֱדוֹם / אֲשֶׁר אֲנִי הֲדוֹם / לְרַגְלֵיהֶן

חִישׁ יְרֻשָּׁה / לִגְרוּשָׁה / וּלְרֹאשָׁה תְּהִי עֲלֵיהֶן

זֶה חֹדֶשׁ / נְוֵה חַדֵּשׁ / וְכָל קָדֵשׁ / נְתֹץ נְוֵיהֶן

קְרָא יְשׁוּעוֹת / לַיְגֵעוֹת / וְצֹאן רוֹעוֹת / עַל יְדֵיהֶן.
