אַתָּה אָפַדְתָּ

אַתָּה אָפַדְתָּ שַׁפְרִיר עַל קַו מְמַדֶּיךָ / אַתָּה אֱלֹהִים לְבַדֶּךָ

אַתָּה בִסַּסְתָּ צָפוֹן וְיָמִין עֻזְּךָ יְשַׁנֵּנוּ / אַתָּה בְרָאתָם תָּבוֹר וְחֶרְמוֹן בְּשִׁמְךָ יְרַנֵּנוּ

אַתָּה גִלַּפְתָּ עָצְמִי וַתְּכוֹנֵן גָּלְמִי / אַתָּה גוֹחִי מִבָּטֶן מַבְטִיחִי עַל שְׁדֵי אִמִּי

אַתָּה דָפַקְתָּ צִנּוֹר עַל מְדוּרַת לַהַב / אַתָּה דִכִּאתָ כֶחָלָל רָהַב

אַתָּה הִסְגַּרְתָּ סוּפָה בְחַדְרֵי כֶלֶא / אַתָּה הָאֵל עוֹשֵׂה פֶלֶא

אַתָּה וְעַד עוֹלָם לֹא יִתַּמּוּ שְׁנוֹתֶיךָ / אַתָּה וּמִי יַעֲמֹד לְפָנֶיךָ

אַתָּה זֵרִיתָ לָצוֹן בָּאוֹצָר הָנִּכְמָס / אַתָּה זָנַחְתָּ וַתִּמְאָס

אַתָּה חַקּוֹתָ יוֹם זֶה לְטַהֵר נֶאֱלַחַת / אַתָּה חָשַׁקְתָּ נַפְשִׁי מִשַּׁחַת

אַתָּה טִהַרְתָּ חֶלְאָה לְנַקּוֹת יְרֵאֶיךָ / אַתָּה טוֹב וְסַלָּח וְרַב חֶסֶד לְכָל קוֹרְאֶיךָ

אַתָּה יָשַׁבְתָּ עַל כֵּס לִרְאוֹת תַּעֲלוּמִי / אַתָּה יָדַעְתָּ שִׁבְתִּי וְקוּמִי

אַתָּה כָּרַתָּ חַבְלֵי עָווֹן נִקְשָׁרִים / אַתָּה כּוֹנַנְתָּ מֵישָׁרִים

אַתָּה לִמַּדְתַּנִי לָשׁוּב בְּדָת הָנְחַלְתִּי / אַתָּה לִדְבָרְךָ יִחַלְתִּי

אַתָּה מִלֵּאתָ חֶפְצִי וַתַּשְׁלִים חֻקִּי / אַתָּה מַחְסִי אַתָּה חֶלְקִי

אַתָּה נָחִיתָ כַצֹּאן גֶּזַע אֱמוּנֶיךָ / אַתָּה נוֹרָא אַתָּה וּמִי יַעֲמֹד לְפָנֶיךָ

אַתָּה סָלִיתָ קָמַי בְּקוּמְךָ לְעָזְרֵנִי / אַתָּה סֵתֶר לִי מִצַּר תִּצְּרֵנִי

אַתָּה עָרִיתָ יְסוֹד כָּל קוֹרְצֵי קָרֶץ / אַתָּה עָשִׂיתָ אֶת הַשָּׁמַיִם וְאֶת הָאָרֶץ

אַתָּה פִלַּגְתָּ צוּלוֹת וַתַּשְׁבִּיחַ דָּכְיָם / אַתָּה פוֹרַרְתָּ בְעָזְּךָ יָם

אַתָּה צָרִים שִׁקַּעְתָּ וַתַּעֲבִיר עֲבָדֶיךָ / אַתָּה צִוִּיתָ פִקּוּדֶיךָ

אַתָּה קִבַּצְתָּ בָנִים לְמִקְדָּשׁ אֲרִיאֵל / אַתָּה קָדוֹשׁ יוֹשֵׁב תְּהִלּוֹת יִשְׂרָאֵל

אַתָּה רִצַּפְתָּ כֵרָה לְנִינֵי אֵיתָן / אַתָּה רִצַּצְתָּ רָאשֵׁי לִוְיָתָן

אַתָּה שַׁלִּיט וְאֵין מִי יַעֲרֹךְ לְנֶגְדֶּךָ / אַתָּה שִׁמְךָ יְיָ לְבַדֶּךָ

אַתָּה תָשׁוּב תְּקוֹמֵם חָרְבוֹת אַפִּרְיוֹן / אַתָּה תָקוּם תְּרַחֵם צִיּוֹן.
