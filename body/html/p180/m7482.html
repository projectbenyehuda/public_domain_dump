[רפא צירי נשיא עמך והבט]

וקאל איצֹא

È — — — / È — — — / È — — / / È — — — /È — — — / È — —

רְפָא צִירֵי נְשִׂיא עַמְּךָ וְהַבֵּט / לְאִישׁ יָחִיד לְכָל-צָרְכּוֹ לְבַדּוֹ

וְאֵין עֹלֶה וְאֵין גֹּדֵר בְּפֶרֶץ / וְאֵין עֹזֵר וְאֵין עֹמֵד לְנֶגְדּוֹ

זְכָר-נָא בָחֳלָיָיו טוּב פְּעָלָיו / וְאַל-תִּנְשֶׁה אֱמוּנָתוֹ וְחַסְדּוֹ

בְּךָ יִדְרשׁ וְלֹא יַחְפֹּץ בְּרֹפְאִים / חֲנַן אֹתוֹ כְּמוֹ אָדוֹן לְעַבְדּוֹ

וְתֵן חִנּוֹ בְעֵינֶיךָ עֲדֵי-עַד / וְנִינָיו יֹֽצְאֵי מֵעָיו וְנֶכְדּוֹ

יְהִי שִׂכִּים בְּעֵין אֹיְבוֹ בְּשׁוּרוֹ / וְלִצְנִינִים וְחֵץ שָׁנוּן בְּצִדּוֹ

וְיַעְנוּ כָּל-בְּרָכוֹת הַכְּתוּבוֹת / בְּתוֹרָתְךָ בְּיַד-חַגַּי וְעִדּוֹ

וְתִשְׁפָּת-לוֹ מְאֹד שָׁלוֹם לְעֵילוֹם / סְפוֹת חֵילוֹ וְכָל-מִינֵי מְגָדוֹ.
