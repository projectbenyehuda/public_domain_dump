שְׁאֵרִית קְהִלָּה

שְׁאֵרִית קְהִלָּה / סַלְסְלִי תְהִלָּה / יַשְְּׁרִי מְסִלָּה / סַקְּלִי אֹרַח

לְנוֹרָא וְעֶלְיוֹן / אֲשֶׁר שָׁב לְצִיּוֹן / וְיָחִישׁ פִּדְיוֹן / מִפְּאַת מִזְרָח

מַה לָּךְ מְבַכָּה / מְעֻנָּה וּמֻכָּה / הַיֶּשַׁע מְחַכָּה / תָּצִיץ וְתִפְרָח

הַנֶּאֱנָקָה / מִנְעִי צְעָקָה / כִּי יוֹם צְדָקָה / תִּטְלָל וְתִפְרָח.
