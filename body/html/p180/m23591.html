בְּאֵל תָּרוּם

אחר. ואמר עוד. יש לעשות


בְּאֵל תָּרוּם יְמִינֶךָ / וְעָלָיו בִּטְחוֹנֶךָ

וְהָעֹשֶׁר וְהַכָּבוֹד / פְּקִידִים עַל אֲרוֹנֶךָ

וְתָמִיד לִבְּךָ נָדִיב / וְחֶסֶד עַל לְשׁוֹנֶךָ

וְגָדַל חַסְדְּךָ עָלַי / כְּגֹדֶל רַעֲיוֹנֶיךָ

וְאֶחְשֹׁב כִּי חֲשַׁבְתַּנִי / כְּמוֹ יֶלֶד זְקוּנֶיךָ

וְהָיִיתִי כְּמוֹ חוֹתָם / עֲלֵי אֶצְבַּע יְמִינֶךָ

וְסַרְתִּי מֵעֲלֵי יָדָךְ / וְזֹאת מֵאֵין רְצוֹנֶךָ

הֲבָגַדְתָּ, יְדִיד נַפְשִׁי, / הֲבָגַדְתָּ בְּבָנֶיךָ

הֲשָׁכַחְתָּ אֱמוּנָתִי / בְּיוֹם עָמְדִי לְפָנֶיךָ

הֲלֹא אַתָּה נְטַעְתַּנִי / וְצָמַחְתִּי בְגַנֶּךָ

וְלָמָּה לֹא דְרַשְׁתַּנִי / וְהִנֶּנִּי לְעֵינֶיךָ

הֲלֹא אַתָּה נְטַעְתַּנִי / וְצָמַחְתִּי בְגַנֶּךָ

וְהָיִיתִי כְּעֵץ שָׁתוּל / עֲלֵי נַחַל עֲדָנֶיךָ

עֲדֵי כֻלָּם קְרָאוּנִי / חֲבַצֶּלֶת שְׁרוֹנֶיךָ

וְעַתָּה שׁוּב יְדִידִי לִי / וְהַעְדֵּנִי עֲדָנֶיךָ

אֲנִי אֶכְתֹּב שְׁבָחֶיךָ / אֲנִי אֶלְקֹט פְּנִינֶיךָ

וְאִם אַתָּה כְּמוֹ מֶלֶךְ ־ / אֲנִי אַחַד סְגָנֶיךָ

וְאִם אַתָּה כְּמוֹ אָדוֹן / אֲנִי עֶבֶד לְפָנֶיךָ

וְאִם אַתָּה כְּמוֹ שַׁחַק ־ / פְּרֹשׂ עָלַי עֲנָנֶיךָ.
