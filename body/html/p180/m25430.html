שֵׁשׁ מֵאוֹת

שֵׁשׁ מֵאוֹת / נִקְרָאוֹת / בְּדָת הָאֵל יְסוֹדֵיהֶם

לְבַת מִי זֹאת / נִרְמָזוֹת / עַל כַּנְפֵי רְדִידֵיהֶם

מְחֻקָּקוֹת / מְזֻקָּקוֹת / כְּמוֹ צִיצִית פְּקוּדֵיהֶם

הִזְהִיר אֵל / לְיִשְׂרָאֵל / חֹק שָׁמוּר בִּידֵיהֶם

וְעָשׂוּ לָ-/הֶם צִיצִית / עַל כַּנְפֵי בִגְדֵיהֶם

שְׁמֹר לָבָן / כְּמוֹ דָרְבָן / מְקוֹם תְּכֵלֶת הַנִּצְבָּע

לָךְ נִפְתָּר / סוֹד נִסְתָּר / חוּט צִיצִית הֱיוֹת מְרֻבָּע

מִשְׁפָּטָיו / הֱיוֹת חוּטָיו / כְּפוּלִים בְּתוֹךְ שְׁלֹשׁ אֶצְבַּע

הַמְשֻׁלְשָׁל / יְהִי נִפְשָׁל / כְּדֵי אֶצְבָּעוֹת אַרְבַּע

חָק סְדִינָא / כְּרַב קְטִינָא / בְּדַת צִיצִית לֹא נִקְבַּע

זְקֹף עֵרֶךְ / לְחוּט כֶּרֶךְ / עֲלוֹת מִמִּנְיָנֵיהֶם.


שְׁקֹד עָנָף / עֲלֵי כָנָף / וּמִקֶּרֶן יְהִי נִבְדָּל

לֹא יַדְחִיק / אֲבָל יַרְחִיק / כִּמְלֹא קֶשֶׁר אֲגֻדָּל

מְנָת טַלֵּית / נִכְפֶּלֶת / כְּרַב שִׁמְעוֹן פְּטֹר וַחֲדָל

הֵן כְּפָלָהּ / וְתָפַר לָהּ / מִצִּיצִית לֹא תֶחְדָּל

חוּט בֶּגֶד / וְעוֹר נֶגֶד / כְּמִין בֶּגֶד לְחֹק תִּגְדָּל

זְנַח וּנְעֹר / חֲקוּקֵי עוֹר / וְאִם בֶּגֶד בְּכַנְפֵיהֶם

שְׁלֹשִׁים יוֹם / כְּדַת אָיֹם / כְּסוּת שְׁאוּלָה מְעֻכֶּבֶת

לְעֵת הָלַךְ / כָּאן וְאֵילַךְ / הֲלֹא צִיצִית חַיֶּבֶת

מְכוּרַת שׁוּק / לְעַם חָשׁוּק / כְּחֻקָּהּ הִיא נֶחְשֶׁבֶת

הֲמוֹן תַּגָּר / וְגוֹי מִתְגָּר / כְּשֵׁרָה הִיא מְחֻטֶּבֶת

חֹק נִפְרָץ / וְחוּט נִקְרָץ / פְּסוּלָה הִיא וְנֶעֱזֶבֶת

זְכֹר וְהָתֵם / וּרְאִיתֶם / פְּרַט לַיְלָה מְעַכֶּבֶת

קֵץ נִקְבַּע / עַל אַרְבַּע / וְלֹא שָׁלֹשׁ בְּסוֹדֵיהֶם.
