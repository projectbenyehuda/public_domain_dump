יי שמע נא את קולנו (סליחה)

יְיָ שְׁמַע נָא אֶת קוֹלֵנוּ / חוּסָה וְגַם רַחֵם עָלֵינוּ!


אָקוּם חֲצוֹת לַיִל לְהִתְחַנֵּן, / אֶקְרָא לְאֵל עֶלְיוֹן אֲשֶׁר כּוֹנֵן

אֶרֶץ וְשָׁמַיִם, וְאֶתְבּוֹנֵן / פִּלְאֵי גְבוּרוֹתָיו וַאֲשַׁנֵּן

מִקְצָת תְּהִלּוֹתָיו לְהִתְרוֹנֵן; / אָנָּא, אֲדוֹן עוֹלָם וְכֹל חוֹנֵן,

שִׁמְעָה וְהַאְזִינָה לְקוֹלֵנוּ, / הַסְכֵּת לְרַחְשֵׁנוּ וּמִלֵּינוּ!


בִּינָה הֲגִיגִי, אֵל, וְשַׁוְעָתִי / בּוֹרְאִי וְהָחִישׁ אֶת יְשׁוּעָתִי,

בִּיטָה בְצָרָתִי וְרָעָתִי, / הָסֵר אֲנִינָתִי וְדִמְעָתִי,

צוּרִי, דְּלֵה נָא אֶת טְבִיעָתִי, / מַלֵּא וְהָבֵא אֶת תְּבִיעָתִי,

הָשֵׁב שְׂשׂוֹנֵנוּ וְגִילֵנוּ, / שַׁכַּח יְגוֹנֵנוּ וְאֶבְלֵנוּ!


רַבִּים וְנִפְלָאִים מַעְשֶׂיךָ – / מִי זֶה אֲשֶׁר יַעַשׂ כְּנִסֶּיךָ?

יָחִיד בְּכָל קַצְוֵי אֲפָסֶיךָ, / מִשְׁעָן וְגַם מִבְטָח לְחוֹסֶיךָ,

קַבֵּל תְּפִלַּת עַם עֲמוּסֶיךָ, / הָסֵר חֲרוֹנֶךָ וְכַעְסֶךָ,

הַבֵּט וְשׁוּרָה אֶת עֲמָלֵנוּ, / רַחוּם, וְאַל תַּבֵּט בְּפָעְלֵנוּ!


הַקְשֵׁב לְקוֹל שִׁירִי וְרִנּוּנִי, / אֵלִי, וְאַל תִּבְזֶה עֱנוּת עָנִי,

מַלְכִּי וְצוּר עֻזִּי וּמָגִנִּי, / נוֹרָא לְבַדֶּךָ בְּלִי שֵׁנִי,

קוֹלִי בְּעֵת אֶשָּׂא בְּחִנּוּנִי / יֶעְרַב כְּמִנְחָתִי וְקָרְבָּנִי,

כַּפֵּר חֲטָאֵינוּ וּמַעְלֵנוּ, / רַפֵּא כְּאֵבֵנוּ וּמַחְלֵנוּ!


מֶלֶךְ מְבַלֶּה כֹּל וְלֹא בוֹלֶה, / כְּבוֹדוֹ וְהִלּוּלוֹ מִי יְמַלֵּא,

תַּקִּיף עֲלֵי הַכֹּל וּמִתְעַלֶּה, / עוֹשֶׂה גְדוֹלוֹת הוּא וְכָל פֶּלֶא,

כָּל רָם וְכָל שָׁפָל לוֹ יְחַלֶּה, / הַבֵּט בְּעַנְיוּת דַּל וְגַם נִקְלֶה,

וּרְאֵה בְעָנְיֵנוּ וְשִׁפְלֵנוּ / מִמַּעֲמַקֵּי יָם לְהַדְלֵינוּ!
