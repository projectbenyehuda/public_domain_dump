[אֶל אֵל אֲשַׁחֵר וַאֲפַלְּלָה]

רשות לנשמת


אֶל אֵל אֲשַׁחֵר וַאֲפַלְּלָה, / עוֹד בִּי נְשָׁמָה וַאֲהַלְּלָה.

בֹּקֶר בְּקוּמִי אֶדְרְשָׁה שְׁמוֹ, / לֵילִי וְיוֹמִי אֶזְכֹּר נָאֳמוֹ,

דִּמְעִי וְדָמִי יִגְּרוּ כְּמוֹ / נוֹזְלֵי נְחָלִים, וַאֲסַלְסְלָה.

רֹאשׁ מַאֲמָרִי שִׁיר אֲקַדְּמָה, / שִׁמְךָ אֲפָאֵר וַאֲרוֹמְמָה.

פְּרִי שְׂפָתַי לָךְ אֲשַׁלְּמָה, / הוֹדְךָ אֲ[פָ]אֵר וַאֲגַדְּלָה.

הִנֵּה שְׂפָתַי מִיָּקְרְךָ / לֹא אֶכְלָא עוֹד, וַאֲבָרְכָה,

גֹּדֶל כְּבוֹדְךָ וַהֲדָרְךָ / אַגִּיד לְכָל יָבוֹא וַאֲמַלְּלָה.

מַרְפֵּא וּמָזוֹר תֵּן לְמַחֲלִי, / שָׁלְחָה – וְתִסְלַח נָא לְמַעֲלִי,

קָרְבָה לְנַפְשִׁי חִישׁ לְגָאֳלָהּ / כִּי לָךְ אֲקַוֶּה וְאֲיַחֲלָה.
