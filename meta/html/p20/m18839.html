אֵל בְּרֹב חַסְדּוֹ וְטוּבוֹ

(אהבה (לחנוכה)

אֵל בְּרֹב חַסְדּוֹ וְטוּבוֹ / יַחֲמֹל עַל עַם קְרוֹבוֹ,

צַעֲדִי יָכִין וְדַרְכִּי / עוֹד, לְמַעַן אֶשְׂמְחָה בוֹ.


בַּעֲמֹד יָוָן לְכַלּוֹת / נַחֲלַת שַׁדַּי וְחֶבְלוֹ –

כֹּהֲנַי חָשׁוּ לְחַלּוֹת / אֶת פְּנֵי אֵל, גַּם דְּרָשׁ לוֹ,

קִדְּמוּ יַחַד תְּפִלּוֹת / נִשְׁמְעוּ בִמְרוֹם זְבוּלוֹ.

יוֹם עֲוִיל לִבּוֹ נְדָבוֹ / עַל יְשֻׁרוּן לַאֲדִיבוֹ –

פַּעֲמִי אִמֵּץ וּבִרְכִּי / אָז, וְצַר נֶעְכָּר כְּאֵבוֹ.


רָגְשׁוּ גוֹיִם לְהַשְׁמִיד / שׁוֹמְרֵי חֹדֶשׁ וְשַׁבָּת,

גָּזְרוּ עַל עֵץ לְהַעְמִיד / מוּל וּמָל וּשְׂרֹף בְּלַבַּת

אֵשׁ, וְחָק חָכָם וְתַלְמִיד / לַעֲזֹב כָּל בֵּן וְכָל בָּת.

אָמְרוּ: עַם זֶה עֲצָבוֹ / צוּר, וְעַל חֶטְאוֹ עֲזָבוֹ!

חָמְסוּ בֵיתִי וְסֻכִּי / לַהֲרֹס וּלְהַחֲרִיבוֹ.


הָאֱלֹהִים חָס וְחָמַל / עַל מְעוֹן קָדְשׁוֹ וְעַמּוֹ,

אוֹיְבִי הֶחְפִּיר וְקָמַל / בַּחֲרוֹן אַפּוֹ וְזַעְמוֹ,

כִּי לְטוֹב זָכַר וְגָמַל / בֵּן, אֲשֶׁר הֶחְזִיק בְּתֻמּוֹ;

קָם לְמִשְׁפָּטוֹ וְרִיבוֹ / בַּהֲדֹף שׂוֹטְנוֹ וְאוֹיְבוֹ,

בֶּאֱמֹר עַמִּי: בְּמַלְכִּי / אֶחֱזַק לָצֵאת וְלָבוֹא.


מִפְּדוּת לֹא קָצְרָה יַד / אֵל, וּמַדּוּעַ יְאַחַר

קֵץ גְּאֻלַּת עַם מְפֹרָד, / נָד לְמֵרָחוֹק וְסָחָר?

הֵן בְּשִׁבְיוֹ לוֹ יְקַו, עַד / נוֹדְדָיו יִדְרֹשׁ וְשִׁחָר.

שָׂם בְּעַבְדּוּתוֹ לְבָבוֹ / אֶל כְּתָב נָבִיא כְּתָבוֹ,

בֶּאֱמֶת בּוֹ יַחֲזִיק, כִּי / אַהֲבַת נַפְשׁוֹ אֲהֵבוֹ.
