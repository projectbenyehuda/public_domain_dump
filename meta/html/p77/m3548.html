נכה רוח נדוד

נכה רוח נדוד / וידאל בנבנשת

כתובת: אחר כן בניתי בית בדמות שיריו ובמתכנתם


נְכֵה רוּחַ נְדוֹד פִּלַּח סְגוֹרוֹ / וְהֵרִיק בָּחֳרִי אַחְרָיו סְגוֹרוֹ

סְגוֹרוֹ שָׁת פֵּרוּד קִרְבַת יְדִידָיו / וְלֹא יִכְהֶה וְלֹא יִגְהֶה מְזוֹרוֹ

מְזוֹרוֹ שָׂם בְּיַד חֶבְרַת יְדִידִים / וְעַל גַּחְלֵי נְדוֹד יָסַד מְגוּרוֹ

מְגוֹרוֹ בָא וְלוּלֵי יַם זְמִירָךְ / בְּאֵשׁ נוּדָךְ שְׁמוֹ אָבַד וְזִכְרוֹ

וְזִכְרוֹ הֶחֱיָה רוּח יְדִידוֹת / שְׂעִפַּי יֵלְכוּ חֹשֶׁךְ לְאוֹרוֹ
