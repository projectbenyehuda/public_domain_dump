תְּבֹרַךְ יָהּ

תְּבֹרַךְ יָהּ אֲדוֹן עוֹלָם / בְּפִי כָל הַיְּצוּר כֻּלָּם

וְאִם אַתָּה מְאֹד נַעְלָה / עֲלֵי שִׁירָם וּמַהְלָלָם

בְּנוּסִי מִפְּנֵי צָרִים / צְרָרוּנִי בְּרֹב נִכְלָם

קְרָאתִיךָ לְגַלּוֹת סוֹד / אֲשֶׁר לֹא מִמְּךָ נֶעְלָם

וְגִלִּיתָ עֲמֻקּוֹת לִי / וְלֹא שָׁב עַבְדְּךָ נִכְלָם

וְרָאִיתִי בְשִׁירָתְךָ / כְּאִישׁ רָאָה וְהֵן סֻלָּם

לְזֹאת אוֹדֶה פְלָאֶיךָ / וְלֹא אוּכַל לְהַשְׂכִּילָם

וְלֹא אֶשְׁכַּח דְּבָרְךָ לִי / בְּעֵת לִבִּי חֲלוֹם חָלָם

וְהָרָשִׁים יְהוּ שָׁרִים / וְאָז תָּרֹן לְשׁוֹן נֶאְלָם

שְׁמַע שִׂיחִי פְּדֵה רוּחִי / וְאֵיךְ תֶּחְרַשׁ וְתִתְעַלָּם

וְהִנֵּה בִקְּשׁוּ נַפְשִׁי / לְקַחְתָּהּ טָמְנוּ חַבְלָם

לְעַג לָהֶם וְרִיב בָּהֶם / וְתֶן לָהֶם כְּמוֹ פָעְלָם

עֲשֵׂה פֶלֶא לְבֶן כֶּלֶא / וְהַרְאֵנִי דְבִיר אוּלָם

וְאָבוֹא בֵית תְּפִלָּתְךָ / אֲשֶׁר עָלָיו אֲנִי נֶהְלָם

וְאַקְדִּישְׁךָ וְאַעְרִיצְךָ / בְּסוֹד עַמִּי וּבִקְהָלָם

וְשָׁם אִכַּף וְאֶפְרֹשׂ כַּף / וְאֶעְבָדְךָ בְּלֵב נִשְׁלָם.
