[אֵיךְ לֹא אֶדְאַג וְאֶשְׁתּוֹמֵם]

וקאל איצ͘א


אֵיךְ לֹא אֶדְאַג וְאֶשְׁתּוֹמֵם / וּבְצַלְעִי אֵשׁ לְהִתְחַמֵּם

תִּשְׁאַל אֵי לִבְּךָ יִרְעֶה / וּלְבָבִי עִמְּךָ זֹמֵם

קוּמָה וּרְאֵה מְיֻדָּעִי / בָּרָק יִשְׂחַק וְאֶשֹּׁמֵם

יָרֵחַ כַּחֲצִי סָמֶ“ךְ / עָלָיו עָנָן כְּצוּרַת מֵ”ם

בָּרָק נָפַל בְּלֶב־לַיִל / כִּנְפֹל שִׂמְחָה בְּלֵב שָׁמֵם

כַּמָּה אָעוּף בְּעֵת יָעוּף / לִרְאוֹת פָּנָיו וְיֵרוֹמֵם

הִזְכִּירַנִי פְּנֵי רֵעִי / נֶהְפַּךְ עָלַי לְמִתְקוֹמֵם

דִּמָּה לִשְׂחוֹת בְּגַלֵּי־יָם / יֶהְמוּ גַלָּיו וְהוּא רוֹמֵם

וּבְקֶרֶב הַקְּרָב יָבֹא / גִּבּוֹר חַיִל וְיֵהָמֵם

יִמְשׁוֹךְ עָלַי זְנַב קִנְאָה / עַד־לֹא יָכֹל וְיִתַּמֵּם

עִקְּבֵי הֶרֶס תְּהִלָּתְךָ / גַּם פִּנַּת זִכְרְךָ קוֹמֵם

לָךְ כָּל־הַיּוֹם אֲנִי נִכְסָף / כִּדְוֵה לֵבָב וְכִמְשׁוֹמֵם

עוֹרֵר מוּסָר וְאַל־תִּגְרַע / חֻקֵּי אַהְבָה וְתַשְׁלִימֵם

וּזְכֹר חֶסֶד וְעֵינֶיךָ / עַל־שָׂם עַיִן בְּךָ שִׂימֵם

שִׂים לִוְיַת־חֵן עֲלֵי רֹאשְׁךָ / וּבְתֵבֵל נִסְּךָ רוֹמֵם.
