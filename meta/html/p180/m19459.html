[וְאָלִין וַאֲנִי נִבְהָל כְּאִלוּ]

ולה רחמה אללה [ולו ירחמהו האל]


וְאָלִין וַאֲנִי נִבְהָל כְּאִלוּ / שְׁנַת עֵינַי עֲלֵי עֵינַי אֲסוּרָה

וְהַשַׁחַק כְּאֹהֶל בַּחֲבָלָיו / אֲשֶׁר הַלַּֽיְלָה בָהֶם קְשׁוּרָה

וּמַרְאֵה הַלְּבָנָה בַעֲיָשָׁהּ / כְּכַלָּה נַעֲרוֹתֶיהָ מְאִירָה

כְּאַיֶּלֶת סְבָבוּהָ צְבָאיִם / אֲשֶׁר הַלַּֽיְלָה לָהֶם כְּפִירָה

וְהַכִּימָה בְּכוֹכָבִים אֲגוּדָה / כְּנֵרוֹת הֹעֲלוּ עַל הַמְּנוֹרָה

וְרֹאשׁ הַלַּֽיְלָה כָּפַף בְּשֵׂיבוֹ / כְּבֶן־שִׁשִּׁים וְעֶשְׂרִים וַעֲשָׂרָה

וְהִ[נֵּה] כֽוֹכְבֵי שַׁחַק לְנֶגְדִּי/ כְּרָצִים שֹׁאֲלִים קַחַת בְּשֹרָה

וְעָלוּ כַּעֲלוֹת שָׂרֵי תְעוּדָה / עֲלוֹת שֶׁמֶשׁ בְּאֶרֶץ לֹא סְדוּרָה

מְלִיצַי הַנְּבוֹנִים הַגְּאוֹנִים / וְחַכְמֵי דָת יְשֻׁרוּן הַטְּהֹרָה

שְׁנֵי…הַ[מְּא]וֹרִים /..ים שְׁנֵי צִדֵּי מְנוֹרָה

וְזֶה סַפִּי[ר וְזֶה] נֹפֶךְ וְיַהְלֹם / וְזֶה תַּרְשִׁישׁ וְזֶה אֶבֶן יְקָרָה

בְּנֵי הַחֵן וּבַעְלֵי הָאֱמוּנָה / אֲחֵי כָבוֹד בְּנֵי מַשְׂאַת וְיִתְרָה

בְּדָת אֵל סָחֲרוּ תָמִיד אֲשֶׁר לָהּ / מְאֹד יִתְרוֹן וְעָזְבוּ כָל סְחֹרָה

וְהִנֶּנּוּ מְחַכִּים בֹּא אֲלֵיהֶם / שְׁאֵלוֹת מִנְּהַרְדֵּעָא וְסוּרָא

יְשִׁיבוֹת……/ במלים……סְגוּרָה

וְהֵם שָׂמוּ בְּיָדֵינוּ חֲרָבוֹת / שְׁלוּפוֹת לַעֲמֹד־לָנוּ בְּצָרָה.
