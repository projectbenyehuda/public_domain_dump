[אִם אַחֲרִית]

ולה איצ ͘א מת ͘ל ד ͘לך [ולו עוד דוגמתו; מכוון לכותרת של עז]

— — È — / — — È — / — — È — // — — È — / — — È — / — — —


אִם אַחֲרִית שִׂמְחַת אֲדָמָה אֵבֶל / גַּם סוֹף מְנוּחָתָהּ לְטִיט וָחֵבֶל

וִימֵי אֱנוֹשׁ כַּצֵּל אֲשֶׁר לֹא-יַעֲמֹד / פִּתְאֹם יְהִי נִשְׁבָּר כְּשֵׁבֶר נֵבֶל

עַל-מַה-נְּבַקֵּשׁ בִּלְעֲדֵי שַׁדַּי הֲלֹא / כָּל-מַעֲשֵׂינוֹ בִּלְעֲדֵי זֹאת הֶבֶל.
