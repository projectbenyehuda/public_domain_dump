שָׁמֵם הַר צִיּוֹן

שָׁמֵם הַר צִיּוֹן / בַּעֲווֹנוֹת אֲבוֹתֵינוּ

לֹא מִקְדָּשׁ וְלֹא כֹהֵן / לְכַפֵּר עַל עֲדָתֵנוּ

הֵבֵאנוּ דְבָרִים / תַּחַת קָרְבְּנוֹתַינוּ / וּנְשַׁלְּמָה פָרִים שְׂפָתֵינוּ.


חָרֵב מְקוֹם שִׁבְתִּי / וְשָׁמֵם בֵּית תִּפְאַרְתִּי

אָפֵס עֲנַן קְטָרְתִּי / וּמִזְבֵּחַ אֵין אִתִּי

לֹא נִשְׁאַר בִּלְתִּי / תְּפִלָּתִי בְגִשְׁתִּי

שָׂא מִרְיִי וּבָשְׁתִּי / כְּמוֹ בְיַד אִישׁ עִתִּי / תָּשׁוּב תְּרַחֲמֵנוּ תִּכְבֹּשׁ עֲווֹנוֹתֵינוּ.


עַם נִרְפָּשׁ וְנִדְלָח / רְצֵה אֵל טוֹב וְסַלָּח

וְשַׁוְעָתָם תִּצְלַח / כִּקְטֹרֶת מְמֻלָּח

וְרָב כֹּפֶר תִּשְׁלַח / לְכָל עָווֹן נֶאֱלָח

וְחַטָּאתָם תִּסְלַח / כְּעַל שָׂעִיר מְפֻלָּח / בְּעֵת רָצוֹן עֲנֵנוּ וְאַל תִּבְזֶה עֱנוּתֵנוּ.
