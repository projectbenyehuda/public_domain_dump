שֵׁם אֱלֹהֵי צְבָאוֹת

שֵׁם אֱלֹהֵי צְבָאוֹת אֲיַחֵד / לֵאלָהוּתוֹ לֹא אֲכַחֵד

מַקְהֲלוֹתָיו אֶקְרָא מְפַחֵד / הָבוּ לַיְיָ הֲמוֹנוֹ

בָּרְכוּ יְיָ כָּל צְבָאָיו / מְשָׁרְתָיו עוֹשֵׂי רְצוֹנוֹ


שֵׁם הָאֵל יוֹצֵר הַכֹּל / אַמִּיץ אֲשֶׁר הַכֹּל יָכוֹל

תּוֹלֶה עוֹלָם כְּמוֹ אֶשְׁכֹּל / קָשׁוּר בִּזְרוֹעַ יְמִינוֹ


לִשְׁמוֹ גַּלְגַּלִּים יְאִיצוּן / וְכוֹכָבִים לוֹ יַעֲרִיצוּן

וּמַלְאָכִים אַחֲרָיו יְרוּצוּן / כְּמוֹ צָבָא אַחֲרֵי קְצִינוֹ


מֶרְכָּבוֹ צָפוּן וְסָתוּם / וּבְשֵׁשׁ טַבָּעוֹת חָתוּם

וְעַל אַרְבַּע חַיּוֹת רָתוּם / קְצוּבוֹת תַּחַת כֵּס מְעוֹנוֹ


הָדוּר בְּמֶרְכָּבָה אֲחוּזָה / תַּחַת כִּסְאוֹ שָׁת מְזוּזָה

נֶפֶשׁ יְשָׁרִים שָׁם גְנוּזָה / לְהִתְעַדֵּן מִזִּיו שְׁכִינוֹ


הִצִּיב שְׁנֵי שָׂרִים פְּנִימִים / עוֹמְדִים עַל שִׁבְעָה רְקוּמִים

וְצִיץ שִׁבְעִים שֵׁמוֹת חֲתוּמִים / עַל רֹאשׁ סַנְדַּלְפוֹן נְתָנוֹ


קַרְנֵי כֵס נִשְׂגָּב וְתַקִּיף / בְּסוֹד אַרְבַּע מַחֲנוֹת הִקִּיף

יְרֵאִים כְּבוֹדוֹ מֵהַשְׁקִיף / כִּי מִכָּל עַיִן צְפָנוֹ


טִכֵּס מִיכָאֵל בְּיָמִין / וְגַבְרִיאֵל בִּשְׂמֹאל הִזְמִין

וְנוּרִיאֵל לְפָנָיו הֶאֱמִין / וּרְפָאֵל אַחֲרָיו הֱכִינוֹ


נִצָּב כָּל שָׂרָף וְשִׁנְאָן / כְּמוֹ בָזָק לֶכְתָּן וּבוֹאָן

יְרוֹ[צֵ]צוּן אָנֶה וָאָן / וְלֹא יֵדְעוּ אַיֵּה מְכוֹנוֹ


חוֹנִים בְּסוֹד שׁוֹכֵן חֶבְיוֹן / הַשְׁמִיעוּ לְנוֹרָא וְעֶלְיוֹן

קְדֻשַּׁת עַם עָנִי וְאֶבְיוֹן / אוּלַי יִמְצָא חֵן בְּעֵינוֹ


זְכוּת אֵיתָנִים הַשְּׁלֹשָׁה / יִזְכֹּר בְּעֵת שִׁלּוּשׁ קְדֻשָּׁה

יְרַחֵף עַל צִפּוֹר גְּרוּשָׁה / כְּמוֹ נֶשֶׁר יָעִיר קִנּוֹ.
