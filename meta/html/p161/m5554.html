יה למתי צפנת חזיון תנחומיך

יָהּ, לְמָתַי צָפַנְתָּ / חֶזְיוֹן תַּנְחוּמֶיךָ!

עַד מָתַי מַאַנְתָּ / לְהַשְׁקִיט שְׁאוֹן קָמֶיךָ?

עַד מָתַי הִשְכַּנְתָּ / עָלַי עֲנַן זַעְמֶךָ?

עַד מָתַי עָשַׁנְתָּ / בִתְפִלַּת עַמֶּךָ?

הִתְיָאֲשׁוּ מַחְשְׁבוֹתַי / אֶת‑פָּנֶיךָ לִרְאוֹת,

כִּי אֶחֱרוּ עִתּוֹתַי ‑ / וְכָל‑הָעִתּוֹת בָּאוֹת.

קוֹל מַר צָלֲלוּ שְׂפָתַי / לְךָ, אֱלֹהֵי הַצְבָאוֹת:

אֲדֹנָי, עַד מָתַי / קֵץ הַפְּלָאוֹת

כָּמוּס בַּאֲסָמֶיךָ / וְחָתוּם בְחֹתָמֶךָ?

וּבֵן שַׂמְתָּ מוֹשָׁבוֹ / מֵעַל כּוֹכְבֵי נְגֹהִים –

אֵיךְ נָפַל כּוֹכָבוֹ / וּמְאוֹרוֹתָיו כֵּהִים,

וּמִתּוֹךְ נְוֵה אֶשְׁנַבּוֹ / נִכְסַף לְבֵיתְךָ וְיָהִים!

מָתַי, צוּרִי, אָבוֹא / וְאֵרָאֶה פְּנֵי אֱלֹהִים

לִשְׁקֹד בְּאוּלַמֶּיךָ / וְלַחֲזוֹת בְּמַנְעַמֶּיךָ?

דַּלּוּתִי אֵיךְ אֶכְסֶה / אַחֲרֵי אֲשֶׁר נִגְלֵיתִי?

מַלְכוּת אֲגַג תִּנַּשֵׂא ‑ / וְתִשָׁפֵל מַלְכוּתִי,

וְשֵׂעִיר חַיִל עוֹשֶׂה / וְיִתְגַבֵּר בֶּן‑אֲמָתִי.

מָתַי, צוּרִי, אֶעֱשֶׂה / גַּם אָנֹכִי לְבֵיתִי –

בֵּית מִדְרַךְ פְּעָמֶיךָ / וְאָחוּז בִּמְרוֹמֶיךָ?

הֵן שׁוֹמְרִי לֹא יָנוּם / וַאֲנִי לֹא אֶנְשֶׁנּוּ,

אַךְ בְּחֶטְאִי הֶעָצוּם / כְּדֹב שַׁכּוּל אֶפְגְּשֶׁנּוּ.

מָתַי רֹאשִׁי יָרוּם / וְעֵת רָצוֹן אֶדְרְשֶׁנּוּ?

מָתַי, צוּרִי, אָקוּם / אוֹסִיף עוֹד אֲבַקְשֶׁנּוּ? –

בְּקַנֹּאתְךָ לְשִׁמְךָ / וּלְעִירְךָ וּלְעַמֶּךָ!
