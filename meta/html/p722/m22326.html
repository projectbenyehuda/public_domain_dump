הִתְבַּשֵּׂרוּ בְנֵי עֶלְיוֹן

חמישה עשר באב


הִתְבַּשֵּׂרוּ בְנֵי עֶלְיוֹן / כִּי נִחַם יוי צִיּוֹן:


אַזְכִּיר חֲסָדֶיךָ מַלְכִּי / לְנֶגֶד כָּל נִבְרָאֶיךָ

וּדְבַר גְּבוּרוֹת יְחַו חִכִּי / אֲשֶׁר פְּעַלְתָּם לִירֵאֶיךָ

בַּחֲמִשָּׁה עָשָׂר בְּאָב, כִּי / נִמְלְצוּ לְחִכִּי פְלָאֶיךָ

וְאֶכְתְּבֵמוֹ עַל גִּלָּיוֹן / וְאָשִׁיר בָּם עֲלֵי הִגָּיוֹן:


בְּיוֹם זֶה נֶעְצָר מִקֶּדֶם / נֶגֶף מִמֵּתֵי מִדְבָּר

וּבִנְיָמִין צָעִיר רוֹדֵם / לְשִׁבְטֵי יָהּ בּוֹ הִתְחַבָּר

וְרָצָה בָהֶם צוּר פּוֹדָם / אַחֲרֵי בָמוֹ הִתְעַבָּר

כִּי טִהֵרוּ לֵב וְרַעְיוֹן / וְרָחֲצוּ כַפָּם בְּנִקְיוֹן:


רוֹן פָּצְחוּ שְׁבָטִים בְּיוֹם זֶה / וְיֶשַׁע וְעֹז הִתְאַזָּרוּ

כִּי בוֹ לָבוֹא זֶה בָזֶה / וּלְהִתְחַבֵּר הֻתָּרוּ

וַהֲרוּגִים בְּיַד צָר נִבְזֶה / בְּבֵית תֵּר הַיּוֹם נִקְבָּרוּ

כִּי לֹא יַעֲשֶׂה כִלָּיוֹן / צוּר, וְיָחוּס עַל דַּל וְאֶבְיוֹן:


הַיּוֹם בִּטֵּל בֶּן אֵלָה / שׁוֹמְרֵי אֹרַח וְתַהֲלוּכָה

שָׂם יָרָבְעָם עַל מְסִלָּה / לְהַשְׁמִיד חַג מֵעִיר בְּרוּכָה

וּבוֹ פָסְקוּ שָׂרֵי סְגֻלָּה / לִכְרוֹת עֵצִים לְמַעְרָכָה

כִּי מָצְאוּ בְשֶׁמֶשׁ רִפְיוֹן / וְחֻמָּהּ נִסְתָּר בְּתוֹךְ חֶבְיוֹן:


מָתַי יְחֻדָּשׁ הוֹד בּוֹלֶה / לְבֵן בַּגּוֹלָה זָנוּחַ

וּמָתַי יְנוּחָם לֵב חוֹלֶה / וּמֵעוֹל אוֹיְבַי אָנוּחַ

מְהֵרָה מְשִׁיחִי יִגָּלֶה / וְנָחָה עָלָיו הָרוּחַ

וְיִקְרָא לַאֲסִירַי פִּדְיוֹן / קוּמוּ וְנַעֲלֶה צִיּוֹן:
