משל יג: עֹרֵב וְשֻׁעָל

גָּדוֹל כֹּחַ הַגַּאֲוָה / עַד כִּי תִדְחֶה מִדַּת הַתַּאֲוָה


עֹרֵב עָלָה עַל הַתְּאֵנָה / וּבְפִיהוּ נָשָׂא גְבִינָה / וְשֻׁעָל עַמַד תַּחַת הָעֵץ / חֹשֵׁב בְּתַחְבּוּלוֹתָיו וְיוֹעֵץ / בַּגְּבִינָה אֲשֶׁר רָצָה / וְאֵיךְ יוֹרִידֶנָּה אָרְצָה / וַיִּקְרָא אֶל הָעֹרֵב / הֶהָדוּר הַיָּפֶה וְהֶעָרֵב / הַטּוֹב וְהַנָּעִים וְהַנֶּחְמָד / אַשְׁרֵי כָּל אֵלֶיךָ נִצְמָד / אַךְ אִם הָיוּ כָּל יָפִים אֶצְלֶךָ / לֹא דָמוּ בְיָפְיָם אֵלֶיךָ / וְאִם נִסִּיתָ שִׁירִים לְשׁוֹרֵר / עַל כָּל הָעוֹפוֹת תִּשְׂתָּרֵר / תִהְיֶה כָּלִיל בַּיֹּפִי / כִּי לֹא נִמְצָא בְּךָ כָּל דֹּפִי / רְאֵה אִם קוֹל לְךָ לְפִי יְפִי קוֹמָה וּפְאֵר נוֹצָה / לֹא נִמְצָא בְךָ כָּל שִׁמְצָה / וַיֹּאמֶר הָעֹרֵב בְּלִבּוֹ: אַשְׁמִיעַ לוֹ קוֹלִי / וְיוֹסִיף מַהֲלָל עַל מַהֲלָלִי / וַיִּפְתַּח פִּיו וַיָּרֶם קוֹלוֹ / וְהַגְּבִינָה חִישׁ נָפְלָה לְרַגְלוֹ / וַתִּפֹּל אֵצֶל רֹאשׁ הַשֻּׁעָל / וַיֹּאמֶר: מִמֶּגֶד שָׁמַים מֵעָל / בָּא לִי זֶה מִן קוֹל מֵרִים / לֹא אֶשְׁמַע עוֹד בְּקוֹל שָׁרִים / וַיֵּלֶךְ לוֹ אֶל אַרְצוֹ / כַּאֲשֶׁר מָצָא מִן הָעֹרֵב חֶפְצוֹ.


וָאֶשָּׂא מְשָׁלִי וָאֹמַר:


פִּתִּיתַנִי דּוֹד וָאֶפָּת / הֱיוֹת פֶּתִי כִּי כֵן תּוּכַל,

אֲבָל גָּדוֹל מֵהַסְּכָלִים / הָעוֹשֶׂה אֶת עַצְמוֹ סָכָל.
