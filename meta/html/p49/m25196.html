[שמע השׂר]

U //– – – U // – – – U – – –// – – –

שְׁמַע, הַשַּׂר, / אֲשֶׁר כָּל שַׂר / בְּךָ נִקְרָא / וְנִתְכַּנָּה:

בְּבוֹא סִפְרָךְ / אֲדַמֶּה כִי / בְּיָגוֹן בָּא / וּבִמְגִנָּה,

וְכִי אֵין בּוֹ / לְבַד תַּגְמוּל / לְחֵטְא אַדִּיר / וְלִתְלֻנָּה

וְחוֹרֵשׁ תּוֹ־ / אֲנָה יִקְצֹר / מְהֵרָה כַ־ / אֲשֶׁר אִנָּה.

לְקַחְתִּיהוּ – / וְלִבִּי בִי / כְּמוֹ גַנָּב / בְּתוֹךְ פִּנָּה,

יְבַקֵּש צַד / לְהִסָּתֵר – / וְאֵין מִסְתּוֹר / וְאֵין צִנָּה!

וְנִחַשְׁתִּי / בְּחוֹתָמוֹ / אֲשֶׁר הָיָה / כְּשׁוֹשַׁנָּה,

וְאָמַרְתִּי: / אֲדַמֵּהוּ / אֱלֵי שׁוֹשָׁן / עֲלֵי כַנָּה,

וְאִם יוּמַר / דְּבַר שׁוֹשָׁן – / יְהִי שָׂשׂוֹן / וְיוֹם רִנָּה.

קְרָאתִיהוּ – / וְלֹא נֶחְלַף / דְּבַר נַחַשׁ, / וְלֹא שֻׁנָּה!

מְצָאתִיהוּ / כְּשׁוֹשַׁנָּה / אֲשֶׁר צָצָה / בְּתוֹךְ גִּנָּה,

וְעָצַם בּוֹ / שְׂשׂוֹן לִבִּי / כְּיוֹם חֻפָּה / וְכַחְתֻנָּה.
