אהבה [יושב במעמק מצולה]

יוֹשֵׁב בְּמַעֲמַק / מְצוּלָה, וּמָעֳמָד

אַיִן, וְחָמַק / דּוֹדוֹ וּמַחְמַד

עַיִן נָמַק, / יוֹם יוֹם מַלְמָד

יְשַׂדֵּד לְתֶלֶם / יוֹנַת אֵלֶם,

וּפַעַם הוֹלֵם / בְּגֹדֶל וְגַאֲוָה.


"צֶדֶק, יְדִידִי, / וְאַהֲבַת כְּלוּלִים

אֶזְכֹּר בְּחַסְדִּי, / לֶאֱחֹז מְחַבְּלִים

כְּרָמִים, וְיָדִי / תְרוֹמֵם שְׁפָלִים.

אֶפְרֹשׂ שְׁלוֹמִי / אֶל בַּת עַמִּי –

אַךְ אִישׁ חֶרְמִי / לַמַּעֲצֵבָה".


"חֶלְקִי וְצוּרִי / עֲדֵי מָה יְלוֹנֵן

אוֹיְבִי וְצָרִי / בְּבֵיתְךָ? חוֹנֵן

אֵין לִי, וְאוֹרִי / הֶחְשִׁיךְ מְעוֹנֵן,

וְעַל בַּת נְגִידִים / יָצְאוּ גְדוּדִים

וְעֶבֶד עֲבָדִים / בְּחֶרֶב לְהוּבָה".


"קוּמִי וְשִׁכְנִי / אֶל בֵּית מְנוּחֵךְ!

אֶשְׁכַּח יְמִינִי / אִם אֶשְׁכָּחֵךְ,

וְהִנֵּה לְשׁוֹנִי / תִּדְבַּק אֶל חֵךְ –

אִם לֹא אֶזְכְּרֵךְ / וְאֶחֱבֹשׁ לְשִׁבְרֵךְ

וְנֵזֶר פְּאֵרֵךְ / תַּחְבְּשִׁי, עֲלוּבָה!"
