יְפֵה נוֹף

1

יְפֵה נוֹף2 מְשׂוֹשׂ תֵּבֵל / קִרְיָה לְמֶלֶךְ רָב3

לָךְ נִכְסְפָה נַפְשִׁי / מִפַּאֲתֵי מַעְרָב


הֲמוֹן רַחֲמַי נִכְמָר4 / כִּי אֶזְכְּרָה קֶדֶם

כְּבוֹדֵךְ אֲשֶׁר גָּלָה /וְנָוֵךְ אֲשֶׁר חָרָב


וּמִי יִתְּנֵנִי עַל־ / כַּנְפֵי נְשָׁרִים עַד

אֲרַוֶּה בְדִמְעָתִי / עֲפָרֵךְ וְיִתְעָרָב5


דְּרַשְׁתִּיךְ וְאִם מַלְכֵּךְ / אֵין בָּךְ6 וְאִם בִּמְקוֹם

צֳרִי גִלְעֲדֵךְ נָחָשׁ / שָׂרָף וְגַם עַקְרָב


הֲלֹא אֶת־אֲבָנַיִךְ / אֲחוֹנֵן וְאֶשָּׁקֵם

וְטַעַם רְגָבַיִךְ / לְפִי מִדְּבַשׁ יֶעְרָב

  1. בראשי הבתים השם יהודה: י‘פה, ה’מון וכ"ו.  ↩

  2. יפה־נוף, נוף יפה.  ↩

  3. קריה למלך רב, עיר המלך הגדול והוא ה'.  ↩

  4. המון רחמי נכמר. נתעורר בי המון רחמים.  ↩

  5. ויתערב, דמעי בעפרך.  ↩

  6. ואם מלכך אין בך, אף־על־פי שמלכך אינו בך בכל זאת דרשתיך, השתוקקתי אליך.  ↩
