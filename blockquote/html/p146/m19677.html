שְׁמוּעַת דּוֹדִי זֶה טְהוֹר הָעֵינַיִם

חתימה: שלם שבזי (29: ילוד יוסף)

שְׁמוּעַת דּוֹדִי זֶה / טְהוֹר הָעֵינַיִם

תְּעוֹרֵר קוֹל תּוֹדָה / יְבַשֵּׂר בִּנְתַּיִם

וּמֵי חֶסֶד יַזֶּה / לְחַזֵּק בִּרְכַּיִם

וְיַצִּיל עַם קָדְשׁוֹ / בְּשִׂמְחָה וּבְשִׁירִים:


לְצִיּוֹן יַשְׁפִּיעַ / חֲסָדָיו הָרַבִּים

בְּשִׂמְחָה תַרְגִּיעַ / לְבָנִים שׁוֹבָבִים

בְּרָצוֹן יוֹפִיעַ / יְרַחֵם הַשָּׁבִים

עֲבוּר כֹּהֵן מַקְרִיב / וּבַעְלֵי מִשְׁמָרוֹת:


מְחַכָּה לִקְרָאתִי / לְדוֹדָהּ תּוֹאֶבֶת

תְּעוֹרֵר אַהְבָתִי / לְפָנָיו נִצֶּבֶת

עֲלֵי חֻקֵּי דָתִי / לְפָנָיו סוֹבֶבֶת

לְרַצּוֹת אֶת דּוֹדָהּ / בְּקִיּוּם הַמִּצְוֹת:


שְׁבוּיָה נִכְסֶפֶת / לְהַקְבִּיל אֶת דּוֹדָהּ

בְּכָל יוֹם שׁוֹאֶפֶת / לְתִפְאֶרֶת סוֹדָהּ

וְטוּבוֹ תוֹכֶפֶת / וּפוֹשֶׁטֶת יָדָהּ

בְּעֵינָיו תִּמְצָא חֵן / בְּקִשּׁוּר הַמִּצְוֹת:


בְּחֵי נַפְשֵׁךְ תַּמָּה / תְּרַצִּי עַל גּוֹלִים

פְּזוּרִים תּוֹךְ תֵּימָא / בְּאַהְבָתֵךְ חוֹלִים

מְצַפִּים הַקְדָּמָה / בְּיִשּוּב הָעוֹלִים

לְאֶרֶץ הַקֹּדֶשׁ / שֶׁהִיא נַחְלַת אָבוֹת:


זְמָן יֶשַׁע יָחִישׁ / אֱלֹהֵי עוֹלָמִים

וְאוֹיְבֵנוּ יוֹרִישׁ / וְיֶשְׁבֹּר הַקָּמִים

וְהַר שֵׂעִיר יַעְנִישׁ / וְיַצִּיל אִישׁ תָּמִים

יְרַחֵם עַל בַּנַיו / בְּרָצוֹן וִישׁוּעוֹת:


יְחִידָתִי תִּשְׂמַח / בְּבִיאַת מָשִׁיחַ

וְתִתְעַטֵּר בִּרְמַח / בְּגַנָּהּ תָּנוּחַ

כְּהַצְמִיחַ צֶמַח / לְדוֹדָהּ תַּפְצִיחַ

תְּדַבֵּר שִׁיר חָדָשׁ / תְּזַמֵּר בִּרְנָנוֹת:


יְלוּד יוֹסֵף הִשְׁלִים / לְשִׁיר זֶה שֶׁחִבַּר

וְסוֹד רַעְיָה הֶעְלִים / וּבִרְמִיזָה דִּבַּר

אֱלֹהַי אַל תַּכְלִים / לְעֶבֶד לָךְ סִבַּר

וְשָׁלוֹם רָב לַכֹּל / וְטוֹבוֹת נִכְפָּלוֹת:
