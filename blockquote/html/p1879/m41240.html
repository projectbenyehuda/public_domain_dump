צְאוּ הַדֵּרוּ

צְאוּ הַדֵּרוּ / גַּם זַמֵּרוּ / לָאֵל דָּר מַעְלָה

בָּרוּךְ אֵל נוֹרָא / אֲשֶׁר בָּרָא / חָתָן וְכַלָּה

מוֹשִׁיב יְחִידִים / רֵעִים דּוֹדִים / בְּהוֹד וּגְדֻלָּה.


אֶשְׁאַל בִּתְחִנָּה / בְּקוֹל רִנָּה / מִדָּר זְבוּלִי

יִתֵּן בְּמַתָּן / אֶל הֶחָתָן / גְּבִיר בִּקְהָלִי

חַיֵּי וּמְזוֹנֵי / וּבְנֵי בְנֵי / לְשֵׁם וּתְהִלָּה.


לָקַח יוֹנָתוֹ / הִיא זוּגָתוֹ / כַּלָּה כְנוּסָהּ

יָפָה בְתֵבֵל / וּבְתוֹךְ בָּבֶל / הִנּוֹ יִחוּסָהּ

זָהָב רְפִידָהּ / וַחֲסוּדָה כַּלָּה כְלוּלָה.


חֵן בֶּן זְקוּנִים / לְשׁוֹשְׁבִינִים / אֵל יִתֵּן לָהֶם

בְּנִשּׂוּאֵיהֶם / אֲבוֹתֵיהֶם / יִרְאוּ עֵינֵיהֶם

אֶת כָּל אוֹיְבֵיהֶם / וְשׂוֹנְאֵיהֶם / יַעֲשׂוּ כָלָה.


חַיִּים וְשָׁלוֹם / תֵּן אֵל עֵילוֹם / לְשַׂר יִשְׂרָאֵל

עֹז וּמֶמְשָׁלָה / בְּלִי תִכְלָה / עַד בִּיאַת גּוֹאֵל

הוּא נְשִׂיאֵנוּ / פְּאֵרֵנוּ / רֹאשׁ כָּל הַגּוֹלָה.


זֶרַע צַדִּיקִים / אֵל חַי יָקִים / לָכֶם בְּרִיתוֹ

יוֹסֵף עֲלֵיכֶם / אֶלֶף כָּכֶם / עַם סְגֻלָּתוֹ

יִשְׁלַח אֵלִיָּה / דָּר עֲלִיָּה / יִקְבֹּץ קְהִלָּה.
