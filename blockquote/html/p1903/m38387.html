משל צא: זְאֵב וְשֻׁעָל

טוֹב בֶּעָשִׁיר לְהִתְגָּרוֹת / מִבַּאֲשֶׁר יָדָיו קְצָרוֹת

הַזְּאֵב אָמַר לַשֻּׁעָל / שְׁמַע וַאֲחַוְךָ אֶת אֲשֶׁר תִּפְעָל / וּבֹא אִתִּי וְלֹא יֵדַע הַמֶּלֶךְ / אַשְׂכִּילְךָ וְאוֹרְךָ בְּדֶרֶךְ זוּ תֵלֵךְ / כִּי אָרְכוּ הַיָּמִים אֲשֶׁר עִם הָאֲרִי חֲנוֹתֶךָ / וַיְעַנְּךָ וַיַּרְעִיבֶךָ לְמַעַן נַסּוֹתֶךָ / הִנֵּה רָאִיתִי לְךָ מֵעֵבֶר לַגָּדֵר אֶפְרֹחִים / מֵעַל אִמָּם תּוֹעִים וְנִדָּחִים / וּפִתְאֹם תָּבֹא עֲלֵיהֶם וְהִכִּיתָם / לֹא תִדְרֹשׁ שְׁלֹמָם וְטֹבָתָם / וּפְרֹץ בַּגָּדֵר מִדָּה לְרֹאשֶׁךָ / מַהֵר וְתִמְצָא מַעֲדַנִּים לְנַפְשֶׁךָ / וּבְאֹרֶךְ הַחוֹר וּבְרָחְבּוֹ / שֻׁעָלִים הִלְּכוּ בוֹ / וַיַּעַן הַשֻּׁעָל: אִם עָשִׁיר אֲדֹן הַבַּיִת וְהֶחָצֵר / מִכָּל אֲשֶׁר אָמַרְתָּ לִי לֹא יִבָּצֵר / וְאִם הוּא עָנִי וָדָל / מִלִּשְׁמֹעַ עֲצָתְךָ לִבִּי יֶחְדָּל / פֶּן יִרְדֹּף אַחֲרַי וְהִשִּׂיגַנִי / וּבְמַר נַפְשׁוֹ לַמָּוֶת יַכֵּנִי.

וְהַמָּשָׁל תָּבִין אֲשֶׁר אֲנִי לְךָ מוֹרֶה / וְאַל יְהִי לְבָבְךָ סוֹרֵר וּמוֹרֶה / כִּי אִם תַּשְׂכִּיל מִמִּשְׁלֵי מוּסָר / מֵעֲדַת נְבוֹנִים אַל תְּהִי מוּסָר / וְטוֹב לְךָ בְּהוֹן הֶעָשִׁיר לְהִתְגָּרוֹת / מֵאֲשֶׁר תִּתְגַּר בְּמִי שֶׁיָּדָיו קְצָרוֹת / בַּאֲשֶׁר יַד הֶעָשִׁיר תִּסְרַח וְעֹדֶפֶת / וְנֶפֶשׁ שְׂבֵעָה תָּבוּס נֹפֶת / וְאִם בַּבֹּקֶר מֵאֲשֶׁר נֶאֱבַד לָהֶם יֵעָצְבוּ / בְּבֹא עֵת עֶרֶב לֹא יִדְאָבוּ / בַּאֲשֶׁר יְמִינָם עֹז נֶאְדֶּרֶת / בִּפְלֵיטָתָם הַנִּשְׁאֶרֶת / אַךְ הָאֶבְיוֹנִים מָרֵי נֶפֶשׁ הֵמָּה / לָתֵת דְּמֵי שָׁלוֹם בַּמִּלְחָמָה / כִּי אֵין לָהֶם פָּלִיט וְשָׂרִיד / וְכָל גֹּזֵל לֹא יַפְחִיד וְיַחֲרִיד / מֵרֹב רָגְזָם וְעָצְבָּם / לֵב הָאֲרִי יִצְמַח בָּם / מִבְּלִי תֵת לָהֶם נֹפֶשׁ / וְקָבַע אֶת קֹבְעֵיהֶם נָפֶשׁ.
