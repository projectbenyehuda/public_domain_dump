משל ס: פָּר וָשׁוֹר

מְתוּקָה שְׁנַת הָעֹבֵד / מִהְיוֹת זְמַנּוֹ אֹבֵד


פָּר רָאָה שׁוֹר בַּמַּחֲרֵשָׁה / וְיַד הָאִכָּר עָלָיו הָלֹךְ וְקָשָׁה / בַּאֲשֶׁר נָתַן עָלָיו מוֹסֵרוֹת וּמֹטוֹת / וּמִן הָעֹל לֹא יוּכַל צַוָּארוֹ לִנְטוֹת / וַיֵּט מִן הַדֶּרֶךְ וַיֹּאמֶר לוֹ הַפָּר / בְּרִיתְךָ אֶת שָׁלוֹם כֻּפָּר / מֵרֹב עֲבוֹדָה וּמֵרֹב טֹרַח / וַאֲנִי כְפֶרֶא רָץ עֲלֵי אֹרַח / וּבַיַּעַר אָשִׂים דֶּרֶךְ וּבִכְרָמִים מִשְׁעוֹל / וְלֹא עֻבַּד בִּי וְלֹא עָלָה עָלַי עֹל / וְאֶתְהַלֵּךְ בְּכֶרֶם זַיִת וּבִסְבָכֶיהָ / כְּבִכְרָה קַלָּה מְשָׂרֶכֶת דְּרָכֶיהָ / וַיַּעַן לְמוּלוֹ הַשּׁוֹר / בְּשִׂפְתֵי דַעַת וּבְמִישׁוֹר / כָּל עֵת אֲשֶׁר הָעֲבוֹדָה עָלַי תִּכְבַּד / וְכִי אֲנִי אֶל הַשָּׂדֶה נֶעְבָּד / וְאֶמְשֹׁךְ בְּעֹל בְּמַאֲמַצֵּי כֹחִי / אֶשְׁכֹּן בֶּטַח וְנַפְשִׁי תֶחִי / וְאַתָּה בְּחַיֶּיךָ אַל תַּאֲמֵן / כִּי שִׁלְּחוּךָ חָפְשִׁי לִהְיוֹת דָּשֵׁן וְשָׁמֵן / וְלֹא בָא צַוָּארְךָ בַּעֲבוֹדוֹת רַבּוֹת / בֶּחָרִישׁ וּבַקָּצִיר תִּשְׁבֹּת / בְּפֶתַע פִּתְאֹם יִקָּחֲךָ שַׂר הַטַּבָּחִים / עַל כֵּן סְבִיבוֹתֶיךָ פַּחִים / וַיְהִי מִקֵּץ חֹדֶשׁ יָמִים / אַחֲרֵי הַפָּר בָּאוּ רַבִּים עַמִּים / וּבַחֲבָלִים עַל צַוָּארוֹ מְשָׁכוּהוּ / וַיְהִי נִדְחָף לָלֶכֶת וַיַּבְהִילוּהוּ / וְהַשּׁוֹר חֹרֵשׁ בְּרִנָּה / וְהַפָּר מֻכֶּה וּמְעֻנֶּה מֵאֵין חֲנִינָה / וַיִּקְרָא לוֹ הַשּׁוֹר: הֲלֹא טוֹב לְךָ לַעֲבֹד אֶת הָאֲדָמָה / מִלֶּכֶת לַחֲצַר מָוֶת כִּי לְמַפַּלְתְךָ אֵין תְּקוּמָה.

וְהַמָּשָׁל בְּאִישׁ אֲשֶׁר לְמַלֵּא סִפְקוֹ הוּא בְּעֹל כָּבֵד / מְתוּקָה שְׁנַת הָעֹבֵד / וּבַקָּהָל תִּגָּלֶה צִדְקָתוֹ / כִּ יֵשׁ תִּקְוָה לְאַחֲרִיתוֹ / וְלֹא כֵן מִדַּת הֶעָצֵל / כִּי חָזוּת בְּפִיו לֵאמֹר, יִנָּצֵל / בְּלֹא טֹרַח וּבְלֹא עֲבוֹדָה / וְלֹא יֵדַע אֶת עִתּוֹ כַּדָּגִים שֶׁנֶּאֱחָזִים בִּמְצוֹדָה.
