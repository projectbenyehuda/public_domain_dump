משל קיז: שֻׁעָל וּזְאֵב

כְּבָר אָמַר שְׁלֹמֹה בְּחָכְמוֹתָיו / צַדִּיק מִצָּרָה נֶחֱלָץ וַיָּבֹא רָשָׁע תַּחְתָּיו

שֻׁעָל לְטַיֵּל עַל הַמַּעְיָן הָלַךְ / וַיַּרְא צֵל הַלְּבָנָה אֲשֶׁר אֶל הַמַּיִם מֻשְׁלָךְ / וְחָשַׁב בְּלִבּוֹ שֶׁהָיְתָה גְבִינָה / וּמְאֹד חָמַד לֶאֱכֹל מִמֶּנָּה / וְהָיוּ שָׁם מִשְׁקֹלוֹת תְּלוּיִן לְפָנָיו / וַיַּבֵּט אוֹתָן בִּשְׁתֵּי עֵינָיו / וַיּוֹשֶׁט יָדוֹ וַיֹּאחֶז בַּחֲבָלִים / לַעֲשׂוֹת מִן הַגְּבִינָה חֲבָלִים / וַיִּכָּנֵס בְּתוֹךְ אַחַת הַמֹּאזְנַיִם / וַיֵּרֶד בְּתוֹכָהּ אֶל הַמַּיִם / וַיִּצְעַק צְעָקָה גְדוֹלָה וּמָרָה / וַיַּמְתֵּן אִם יָבֹא שָׁמָּה שׁוּם אָדָם לְעֶזְרָה / וְהִנֵּה הַזְּאֵב אֶל הַמַּעְיָן בָּא / וַיַּרְא הַשֻּׁעָל אֲשֶׁר שָׁם נֶחְבָּא / וַיֹּאמֶר הַזְּאֵב: מָה אַתָּה עֹשֶׂה שָׁם / הַגֶּד נָא לִי מַה בַּמַּיִם נִרְשָׁם / וַיַּעַן הַשֻּׁעָל וַיֹּאמַר: גְּבִינָה הַרְבֵּה אָכַלְתִּי וְשָׂבַעְתִּי / וּמֵהֶם יוֹתֵר לֹא יָכֹלְתִּי / הִנֵּה גַּם לְךָ גָּנַזְתִּי / וְעַתָּה בֹא הֵנָּה בַעֲצָתִי / וְתֹאכַל אוֹתָהּ וְנִהְיֶה חֲבֵרִים יָחַד / טוֹבִים הַשְּׁנַיִם מִן הָאֶחָד / וַיִּשְׁמַע הַזְּאֵב בַּעֲצָתוֹ / וַיֹּאמֶר בְּלִבּוֹ אֵלֵךְ נָא אִתּוֹ / וְיָדוֹ שָׁלַח בְּכַף מֹאזְנַיִם אֲשֶׁר כְּנֶגְדּוֹ / וַיִּקַּח אֹתוֹ בְּיָדוֹ / וַיְמַהֵר וַיֵּשֶׁב בְּתוֹכוֹ בִנְחִיצָה / וַיֵּרְד אֶל הַמַּעְיָן בִּמְרוּצָה / וְהַשֻּׁעָל עָלָה מִכָּבְדוֹ הִכְרִיעוֹ / וְהַזְּאֵב פָּתַח פִּיו אֶל הַצֵּל כְּבָלְעוֹ / וַיַּרְא כִּי לֹא יוּכַל / וַיֹּאמֶר הַשֻּׁעָל / חֵי רֹאשִׁי מִמֶּנָּה לֹא תֹאכַל / וַיַּחֲזֵק בְּיָדוֹ וַיִּמְשֹׁךְ בַּחֶבֶל / וַיַּרְא כִּי הַכֹּל הֶבֶל / וַיִּשָּׂא קוֹלוֹ בִּבְכִי אֶל הַשֻּׁעָל / וַיֹּאמַר: לָמָּה מָעַלְתָּ בִי מָעַל / וַיַּעַן לוֹ הַשֻּׁעָל: נָא הָשַׁע / כִּי לִפְנֵי צַדִּיק יֵצֵא רָשָׁע / וְעַל זֶה אָמַר שְׁלֹמֹה בְחָכְמוֹתָיו / צַדִּיק מִצָּרָה נֶחֱלָץ וַיָּבֹא רָשָׁע תַּחְתָּיו / וַיְהִי הַיּוֹם וַיֵּלֶךְ הָאִישׁ אֶל מַעְיָנוֹ / וְאִשְׁתּוֹ וּבִתּוֹ וּבְנוֹ / וְהַנַּעֲרָה בָאָה אַחֲרֵיהֶם בְּנַחַת / וְכַדָּהּ עַל שִׁכְמָהּ מַנַּחַת / וַיֵּלְכוּ כֻלָּם בְּזֶה עִנְיָן / עַד כִּי עָמְדוּ אֶל הַמַּעְיָן / וַיִּמְשֹׁךְ הָאִישׁ הַמֹּאזְנַיִם וַיִּרְאֵהוּ / אֶת הַזְּאֵב, וַיֹּאמַר: כִּי מִן הַמַּיִם מְשִׁיתִהוּ / וַיִּרְאוּ כֻלָּם אֶת הַזְּאֵב / וּלְהַכּוֹתוֹ הָאִישׁ תָּאֵב / וַיָּסֹבּוּ כֻלָּם אֹתוֹ / וַיַּתְחִילוּ לְהַכּוֹתוֹ / בְּמַקְלוֹתָם עַד כִּי נִלְאוּ / הַזְּאֵב רָץ וְהֵם אַחֲרָיו בָּאוּ / וַיִּרְדְּפוּ כֻלָּם אַחֲרָיו / אֶל אֶרֶץ מְגוּרָיו / וַיָּשׁוּבוּ אֶל הַמַּעְיָן כֻּלָּם / לִשְׁאֹב מַיִם לְמַאֲכָלָם / וַתְּמַהֵר הַנַּעֲרָה וַתְּמַלֵּא כַדָּהּ / וַתִּקַּח אֹתוֹ בְיָדָהּ / וַתִּשָּׂא כַדָּהּ הָעַלְמָה / וַתָּשֶׁב אֹתוֹ עַל שִׁכְמָהּ / וַיֵּלְכוּ וַיָּבֹאוּ אֶל בֵּיתָם / וַיִּשְׂמְחוּ שָׁם עִם אֲבוֹתָם.
