משל מז: חֲמוֹר בְּעוֹר אֲרִי

אִם רוּחַ גַּאֲוָה נָשְׁבָה בוֹ / רוּחַ חָכְמָה אֵין בְּקִרְבּוֹ

חֲמוֹר מִבֵּית אֲדֹנָיו הָלַךְ לִרְעוֹת / עַל הֶהָרִים וְעַל הַגְּבָעוֹת / וַיִּמְצָא שָׁם עוֹר אֲרִי מָפְשָׁט / וַיֹּאבֶה לְלָבְשׁוֹ וְיָדָיו פָּשַׁט / לְהַשְׁלִיכוֹ לְפָנָיו וּלְאַחֲרָיו / עַל יָדָיו וְעַל חֶלְקַת צַוָּארָיו / וַיָּרָם לְבָבוֹ וּבְגַאֲוָה יֵלֵךְ הָלֹךְ / מִתְנַשֵּׂא לֵאמֹר: אֲנִי אֶמְלֹךְ / וַיֵּלֶךְ לְהַפְחִיד הַחַיּוֹת / עַל הֶהָרִים וְעַל הַגֵּאָיוֹת / וְכַאֲרִי יִטְרֹף לָהֶם אֹרֵב וְצֹדֶה / הָאָדָם וְהַבְּהֵמָה אֲשֶׁר יִמְצָא בַשָּׂדֶה / מִפַּחְדּוֹ הָרֹעִים צֹאנָם עָזְבוּ לִשְׁמוֹר / כִּי חֲשָׁבוּהוּ לְאַרְיֵה וְהִנֵּה הוּא חֲמוֹר / וְעֹבְדֵי הָאֲדָמָה מִפָּנָיו נָסְעוּ / גַּם הַצֹּאן וְהַבָּקָר אַל יִרְעוּ / וְאִישׁ לְאָחִיו צֹעֵק: שֹׁמֵעַ לִי הִנְּךָ / שְׁלַח הָעֵז אֶת מִקְנְךָ / כִּי אֶת אֲשֶׁר יִמָּצֵא בַשָּׂדֶה לֹא יִחְיֶה / וַיְהִי הַחֲמוֹר לָהֶם לְאַרְיֵה / וּמֵרֹב חֶדְוַת לִבּוֹ אֲשֶׁר חֲשָׁבוּהוּ לַאֲרִי / עִם אֲדֹנָיו הָלַךְ בְּקֶרִי / וַיֹּאמֶר כִּי מַשָּׂא לֹא יַשְׁלִיךְ עוֹד עָלָיו / וַחֲמוֹר אֵבוּס בְּעָלָיו / לֹא יָדַע, וְלֹא יָשׁוּב אֶל בֵּיתוֹ / אָכֵן יִתֵּן פַּחְדָּתוֹ / עַל כָּל הוֹלֵךְ אַרְבַּע / כִּי כֵן בִּגְאוֹנוֹ קָפַץ וְנִשְׁבַּע / וְהַקּוֹל נִשְׁמַע: עָלָה אַרְיֵה מִסֻּבְּכוֹ / שִׂימוּ אִישׁ חַרְבּוֹ עַל יְרֵכוֹ / בִּרְמָחִים וּבַחֲרָמִים לְסָבְבוֹ / אוּלַי נוּכַל נַכֶּה בוֹ / נְצוּדֵהוּ בַיַּעַר אוֹ בַשָּׁרוֹן / לַמֶּלֶךְ נִשְׁלְחֵהוּ דּוֹרוֹן / וַאֲדֹנָיו אוֹמֵר אֶל שְׁכֵנָיו: אָבַדְתִּי חֲמוֹרִי / וְאֵיכָכָה אֲבַקְשֶׁנּוּ פֶּן יִטְרְפֵנִי הָאֲרִי / וַיְהִי הַיּוֹם וַיֵּלֶךְ אֲדֹנָיו / וַיָּלֶט בְּאַדַּרְתּוֹ פָּנָיו / וַיָּבֹא בְּאַחַד הֶהָרִים / וַיַּרְא חֲמוֹרוֹ כַּאֲרִי בְּמִסְתָּרִים / לְהַפְחִיד כָּל רֹאֵי תָאֳרוֹ / וַיִּתְבּוֹנֵן כִּי הוּא חֲמוֹרוֹ / בְּקוֹמָתוֹ וּבִזְנָבוֹ וּבְאָזְנָיו / וְכִרְאוֹתוֹ אֲדֹנָיו הִשְׁפִּיל עֵינָיו / וַיֹּאמֶר לוֹ: יַעַן גָּבַהּ לִבְּךָ / יְהִי אָרְבְּךָ בְּקִרְבְּךָ / כַּאֲשֶׁר גָּבַרְתָּ לִשְׁפֹּךְ דַּם נְקִיִּים / וַתִּתֵּן חִתִּיתְךָ בְּאֶרֶץ הַחַיִּים / הָלֹךְ הָלַכְתָּ נְטוּי גָּרוֹן / חָלַמְתָּ וְתָבִין הַפִּתְרוֹן / זְדוֹן לִבְּךָ הִשִּׂיאֲךָ / לְהִתְנַכֵּר מֵאֲדֹנְךָ וּמִמַּשָּׂאֶךָ / תִּכְבַּד הָעֲבוֹדָה וְעֻלָּךְ / עַל עָפָר תְּהִי מֻשְׁלָךְ / וְלֹא תִרְדֹּף הַצֹּאן כַּאֲשֶׁר הוּא שִׁבְיְךָ / וְעַתָּה הוֹרֵד אֶת עֶדְיְךָ / וְהִצַּגְתִּיךָ כְּיוֹם הִוָּלְדְךָ / וְיַהֲפֹךְ הוֹדְךָ לְאֵידְךָ / וְתֵדַע מִי אַתָּה וּמֵאַיִן בָּאתָ / וְאִם אָבִיךָ מֶלֶךְ עַל הַחַיּוֹת רָאִיתָ / וַיִּתְנַצֵּל הַחֲמוֹר מֵעֶדְיוֹ / וַיִּקַּח מֵאֲדֹנָיו כִּפְלַיִם בְּמֶרְיוֹ.

וְהַמָּשָׁל בְּבֶן־נָבָל וּבֶן־בְּלִי־שֵׁם / אֲשֶׁר יַעֲבֹר עָלָיו רוּחַ וְאָשֵׁם / וְרוּחַ גַּאֲוָה נָשְׁבָה בוֹ / וְכָל רוּחַ חָכְמָה אֵין בְּקִרְבּוֹ / וְעַם אֲשֶׁר לֹא יְדָעוּהוּ יַעַבְדוּהוּ / וּבְלֶכְתּוֹ וּבְשִׁבְתּוֹ יְכַבְּדוּהוּ / כִּי לֹא נִכָּר בְּלֶכְתּוֹ בַּחוּצוֹת / כִּי אִם בִּלְבוּשׁ וּמַחֲלָצוֹת / וְיִתְגָּאֶה עֲלֵיהֶם וְיֹאמַר: לַעֲבָדִים מְצָאתִים / וְיִתֵּן צַמַּרְתּוֹ אֶל בֵּין עֲבוֹתִים / וְיֵלֶךְ נְטוּי גָּרוֹן וְלֹא יִזְכֹּר מִי אֲבוֹתָיו / וַיִּיף בְּגָבְהוֹ וְאֹרֶךְ דָּלִיּוֹתָיו / וְאַחֲרֵי אֲשֶׁר מוּמוֹ וּמוּם אֲבוֹתָיו יַכִּירוּ / כְּבוֹדוֹ בְּקָלוֹן יָמִירוּ.

וָאֶשָּׂא מְשָׁלִי וָאֹמַר:

חֲמוֹר עוֹר הָאֲרִי מָצָא, לְבָשׁוֹ / וְהִפְחִיד דֹּב וְנָמֵר עֵת פְּגָשׁוֹ

וְכֵן בֶּן־בּוּז בְּלָבְשׁוֹ בּוּץ וְגָאָה / וְלֹא יָדַע נְשֹׂא רִַגְלָיו וְרֹאשׁוֹ.
