[יִפָּלֵא בְעֵינֵי כָל־אִישׁ] (פִּזְמוֹן)

פִּזְמוֹן


יִפָּלֵא בְעֵינֵי כָל־אִישׁ / מַדּוּעַ אֵחַר יִשְׁעִי

וְיִתְמְהוּ בִי כָל־גּוֹיִם / זֵרוּ אָרְחִי וְרִבְעִי

הוֹשִׁיעָה אֵל וַעֲנֵנִי / וְאַל־תֵּפֶן לְרִשְׁעִי


מַלְכִּי אֵלֶיךָ אֶתְפַּלָּל / הַקְשִׁיבָה לְקוֹל שַׁוְעִי.


וּרְאֵה צַר רֹאשִׁי בְכַשִּׁיל / וְכֵילַפּוֹת יַהֲלֹם

סוּרוּ טָמֵא קָרְאוּ לָמוֹ / סוּרָה אַל־תִּקְרַב הֲלֹם

פְּנֵה וְהוֹלִיכֵנִי אֶל ־/ מְחוֹז חֶפְצִי לְשָׁלוֹם


וְאֶל מִשְׁכְּנוֹת מִבְטַחִים / גְּדִיּוֹתֶיךָ רְעִי.


אֵלֶיךָ אֶקְרָא בַּעֲטֹף / לִבִּי כִּי תַעֲנֵנִי

וּבְיוֹם צַר לִי לְךָ אָבֹא / כִּי בְצִלְּךָ תַּסְתִּירֵנִי

כִּי אַתָּה גוֹחִי מִבֶּטֶן / וְאַתָּה אֵל עוֹשֵׂנִי


אַתָּה יָדַעְתָּ שִׁבְתִּי / וְקוּמִי בַּנְתָּ לְרֵעִי.
