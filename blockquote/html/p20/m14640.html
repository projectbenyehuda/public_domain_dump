אַשְׁפִּיל לְךָ לִבִּי וְעֵינִי

אַשְׁפִּיל לְךָ לִבִּי וְעֵינִי / חָפֵץ בְּלֵב שָׁפֵל וְשָׁח.

עֵינְךָ פְּקַח וּרְאֵה בְעֵינִי, / כִּי רַב עֳנִי עַם נֶאֱנָח.


בֵּן בַּשְּׁבִי יָגַע וְנִלְאָה / יִצְעַק וְיִשְׂבַּע מַמְרוֹר,

לֹא יַעֲצֹר רוּחַ נְכֵאָה – / כִּי אֵין לְנוּגִים מַעֲצוֹר –

לֵאמֹר: תְּבַכֵּר בֶּן שְׂנוּאָה / עַל בֶּן אֲהוּבָה הַבְּכוֹר!

תַּחְלִיף דְּבַר פִּיךָ, אֲדוֹנִי / לִי וַאֲנִי בוֹעֵר כְּאָח –

כִּי אֶחֱזֶה מִקְדַּשׁ מְעוֹנִי / מוּרָשׁ בְּיַד עֶבֶד וְאָח!


רָחְקוּ אֲבוֹתַי מֵרְצוֹנְךָ, / חָטְאוּ – וְחֶטְאָם נָשְׂאוּ,

וָאַחֲזִיק בִּכְנַף גְּאוֹנְךָ – / וּנְאוֹת יְקָרִים נִשְּׁאוּ!

אָבִי, הֲיוּמְתוּ בַחֲרוֹנְךָ / בָּנִים אֲבוֹתָם חָטְאוּ?

אֶדְבַּק בְּךָ לוּ תִקְטְלֵנִי, / מֵעִמְּךָ לֹא אֶאֱרָח,

אִם אֶשְׁכְּחָה אֶחָד בְּשֵׁנִי / עַמִּי בְעַם נָבָל שְׁכָח!


הִנְנִי בְךָ אֶבְטַח וְאֶחְסֶה, / כִּי בִלְתְּךָ אֵין לִי בְמִי.

אִם לֹא לְךָ, אוֹמֵר וְעוֹשֶׂה, / אֶזְעַק הֲמוֹן זַעֲקִי – לְמִי?

הָאֶצְעֲקָה לָעֵץ – וְיַעְשֶׂה? / אֶבֶן – הֲתִשְׁמַע נָאֳמִי?

אַתָּה לְבַד אָדוֹן, קְנֵנִי! / שִׂימָה בְפִי מֶתֶג וְחָח!

וּרְצַע – וּמַרְצֵעַ בְּאָזְנִי / אֶשָּׂא מְקוֹם נֶזֶם וְחָח.


מוֹשֵׁךְ בְּשִׁבְיוֹ עֹל נְדוּדִים, / צוּרִי, מְשֹׁךְ אֵלָיו צֳרִי.

בִּיבוֹש קְצִירוֹ – טַל חֲסָדִים / הוֹרֵד וְהַעְלֵה לוֹ פְרִי.

חַדֵּשׁ יְמֵי קֶדֶם וְדוֹדִים / קָרֵב – וְהַרְחֵק בֵּית מְרִי.

כִּי מִי אֱלוֹהַּ יִשְׁמְרֵנִי? / לִי שַׁעֲרֵי יֶשַׁע פְּתָח!

אֶקְרָא – וְאַתָּה תַּעֲנֵנִי, / בַּאְמִתְּךָ אוֹרְךָ שְׁלָח!
