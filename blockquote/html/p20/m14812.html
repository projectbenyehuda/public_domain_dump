אֵיכָה אֶשָׂא לְאֵל פָּנַי

אֵיכָה אֶשָּׂא לְאֵל פָּנַי, / וַאֲנִי מֵחֲסָדִים עָרוֹם?

בַּמֶּה אֲקַדֵּם אֲדֹנָי, / אִכַּף לֵאלֹהֵי מָרוֹם?


בְּקָרְאִי שְׁמָעֵנִי, מַלְכִּי, / וּרְצֵה שַׁוְעִי כְנִסְכִּי,

וְעָזְרֵנִי עַל אוֹיְבַי, כִּי / רַבִּים לֹחֲמִים לִי, מָרוֹם.


רְאֵה וְהַבֵּט, צוּרִי קְדוֹשִׁי, / וְקַנֵּא לְחֻרְבַּן מִקְדָּשִׁי,

וְלִפְדּוֹת נְפוּצַי מִמּוֹקְשִׁי / שְׁלַח יָדֶיךָ מִמָּרוֹם.


הַשְׁקִיפָה וְחֹן בְּחֶמְלָתְךָ, / כִּי אֵין לָנוּ זוּלָתָךְ,

חַנּוּן וְצַדִּיק; צִדְקָתְךָ, / אֱלֹהִים, עַד מָרוֹם.


מָעוֹז לַדַּל, הַעֲלֵנוּ / מָרוֹם, כִּי לְךָ הוֹחַלְנוּ,

עַד יֵעָרֶה עָלֵינוּ / רוּחַ מִמָּרוֹם.
