[עוֹבֵר שְׁאַל עָלַי וְתֵדַע כִּי כְבָר]

עוֹבֵר שְׁאַל עָלַי וְתֵדַע כִּי כְבָר / דַּרְתִּי בְאַרְמוֹנִים אֲשֶׁר פִּסָּגְתִּי

דַּרְתִּי בְאַרְמוֹנִים כְּמֶלֶךְ בַּגְּדוּד / זֶה הֶחֱיֵיתִי אָז וְזֶה הָרָגְתִּי

עַפְתִּי כְנֶשֶׁר עַל־מְרוֹמֵי הַזְּמָן / אֶתְמוֹל וְאֶבְרַת אוֹיְבַי מָלָגְתִּי

גִּבּוֹר וְאַמִּיץ הֶחֱלַשְׁתִּי בֶּעֱזוּז / כֹּחִי וּבִרְכֵּי כוֹשְׁלִים הִבְלָגְתִּי

אֵיזֶה גְדֻלָּה בַזְּמָן לֹא מָצְאָה / יָדִי וּמַעְדַנִּים וְלֹא הִשָּׂגְתִּי

לֹא עָזְבָה נַפְשִׁי אֲשֶׁר לֹא נִסְּתָה / שַׂשְׂתִּי בְכָל־שָׂשׂוֹן וְהִתְעַנָּגְתִּי

שַׂשְׂתִּי בְכָל־שָׂשׂוֹן בְּחֶבְרַת אוֹהֲבִים / לִקְרַאת צְבָאוֹת כַּצְּבִי דָלָגְתִּי

הִנְנִי נְבֵלָה סוֹרְחָה הִנְנִי בְחוּר / פֶּתֶן וְלִנְחָשִׁים גְּבוּל הִסָּגְתִּי

לוּ יֵשׁ בְּעֵינַי אוֹר וְגוּפִי אֶחֱזֶה / אוֹתִי סְחִי שַׂמְתִּי וּבִי לָעָגְתִּי

בָּאתִי לְכָאן רֵיקַן תְּעוּדָה בֹּר וְתֹם / עַל־זֶה לְבַד עַל־זֶה לְבַד דָּאָגְתִּי

אֵיזֶה חֲלוּקָתִי דְרַבָּנָן וְלִי / בֶגֶד בְּגָדִים אַלֲלַי אָרָגְתִּי

אוֹי לִי וְאֵיךְ לֹא אֵשְׁבָה בַּמַּחֲשָׁךְ / חשֶׁךְ וְלֹא אוֹר לִבְּכִי נִסָּגְתִּי

כָּל־כַּךְ זְדוֹנוֹת יֵשׁ עֲלֵי גַבִּי אֲשֶׁר / אֶחְשֹׁב לְהֶבֶל אֶת־אֲשֶׁר שָׁגָגְתִּי

בָּאתִי בְלִי מַעְצוֹר כְּבַעַר וֶאֱוִיל / לִסְבַךְ תְּשׁוּקוֹתַי וְנִשְׂתָּרָגְתִּי

נַסְתִּי כְאַיִל מִשְּׁמֹר מִצְוָה וְחֹק / עַל־זֹאת כְּאַיִל בֶּאֱיָל עָרָגְתִּי

אוֹי לִי אֲבוֹי כִּי אַחֲרֵי הַמָּוְתָה / אֵין מַאֲזִין אִם כַּאֲרִי שָׁאָגְתִּי

מִי־יִתְּנֵנִי מִי וְאֶתְמוֹל יַחֲזֹר / מִי־יִתְּנֵנִי מִי וְאָחוֹר סָגְתִּי

אֵפֶר כְּלֶחֶם אֹכְלָה עוֹד כָּל־יְמֵי/ חַיַּי וּמַשְׁקִי בַּבְּכִי מָזָגְתִּי

לוּ יִפְתְּחוּ לִי הַשְּׁחָקִים שַׁעֲרֵי / דִמְעָה בְּבִכְיִי יַהֲלֹם מוֹגָגְתִּי

אֶנְקוֹת לְבָבִי יִבְקְעוּ שַׁחַק וְרוֹם / הַיּוֹם וּפִתְחֵי הַזְּבוּל פִּלָּגְתִּי

אֶבְלֹם עֲדִי יִצְרִי וּפִיהוּ אֶחֱסֹם / גַּם־עַל־פְּנֵי יָם תַּאֲוָה חֹק חָגְתִּי

הוֹי עוֹבְרִים אַחַר אֲשֶׁר לֹא אוּכְלָה / לִבְכּוֹת בְּכוּ עָלַי וְלוּ זָלָגְתִּי

שׁוּבוּ בְּטֶרֶם בֹּא לְגוּשׁ פֶּן־תַּחְרְגוּ / מִמִּסְגְּרוֹתָיו כַּאֲשֶׁר חָרָגְתִּי

פֶּן תֻּצְּגוּ לִפְנֵי אֱלֹהִים מֵעֲדִי / צֶדֶק עֲרוּמִים כַּאֲשֶׁר הֻצָּגְתִּי.
