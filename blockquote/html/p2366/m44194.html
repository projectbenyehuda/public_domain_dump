הוֹי לֹא

ועוד קונן עליו


הוֹי לֹא יוֹם בְּשׂוֹרָה לָךְ, נַפְשִׁי, / יוֹם יַם תְּלָאָה חָבוּשׁ לְרֹאשִׁי

יוֹם זֶה קִינוֹת לְמִצְרַיִם רַחֲשִׁי / יוֹם זֶה כְבֵדֵךְ לִגְזָרִים בַּתְּרִי.


נַפְּצִי נְבָלַיִךְ, קִרְעִי מִנִּים, / הָמִירִי יִפְעַת שִׁירַיִךְ בְּקִינִים

הֲגִי כַיּוֹנָה, יַלְּלִי כַתַּנִּים / עִמָּם בְּקוֹל יְלָלָה תִתְחָרִי.


עַל מוֹת יְדִידִי הַשַּׂר, בֶּן אֲצִילִים, / גְּדוֹל הָעֵצָה, גַּם רַב פְּעָלִים

כִּיאוֹר מִצְרַיִם הָיָה לְדַלִּים / שְׁמוֹ אָדוֹן נִסִּים בֵּיךְּ מוֹצִירִי.


שְׁמֵי מִצְרַיִם יוֹם זֶה שַׁחֲרוּ / אֶבְיוֹנֶיהָ בִּבְכִי מַר נַחֲרוּ

בְּבֵית מִדְרָשׁוֹ מִסְפֵּד קִשְׁרוּ / יְשִׁיבָתוֹ, עָלָיו שַׂק חִגְרִי.


הוֹי, תּוּגָה פָּרְצָה תּוֹךְ בִּתְרֵי לִבִּי / תְּגָרֵם עַצְמוֹתַי כְּמוֹ לָבִיא

שְׁבִיב אִשָּׁהּ יֵקַד יְקֹד תּוֹךְ חֻבִּי / יַעַל עֲשָׁנָהּ קֶטֶב מְרִירִי.


בּוּז לְתֵבֵל וְכָל יְקָרֶיהָ / כִּי תֶאְטַר פֶּה עַל בְּחִירֶיהָ

בּוֹגֵדָה הִיא, תַשְׁלִיךְ נְזִירֶיהָ / תַּקְשֶׁה לִבָּהּ כְּאוֹיֵב אַכְזָרִי.


אֱלֹקִים תֵּן חַבְלוֹ בַנְּעִימִים / בִּצְרוֹר הַחַיִּים יִשְׁכֹּן מְרוֹמִים

וּבָנָיו כַּחוֹל יַרְבּוּ יָמִים / בְּאֹשֶׁר וְעֹשֶׁר יִשְׂאוּ פֶרִי.
