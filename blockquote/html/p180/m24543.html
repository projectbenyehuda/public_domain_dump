שׁוֹכְנֵי רוּם

שׁוֹכְנֵי רוּם וְתַחַת / צִבְאוֹתָם וְחֵילָם

לְךָ מָעֹז יִתְּנוּ עֹז / יְמֵי עוֹלָם כֻּלָּם

מְמַלְּלִים וּמְהַלְלִים / בִּלְבָבָם וְקוֹלָם

יְהִי שֵׁם יְיָ מְבֹרָךְ / מֵעַתָּה וְעַד עוֹלָם.


בַּשַּׁחַק אֵלִים / חֲצוּבֵי אֵשׁ וָמָיִם

וּבָאָרֶץ נְדִיבִים / הוֹגִים דַּת יוֹמַיִם

מְמַלְּלִים וּמְהַלְלִים / יִחוּדְךָ פַעֲמַיִם

מִקְצֵה הַשָּׁמַיִם / וְעַד קְצֵה הַשָּׁמַיִם.


בַּשַּׁחַק שִׁמְךָ מְקֻדָּשׁ / בְּסוֹד כְּרוּבִים

וּבָאָרֶץ עֹז לְךָ / יִתְּנוּ עַם נְדִיבִים

מְמַלְּלִים וּמְהַלְלִים / לְמוּלְךָ נִצָּבִים

מֵעֲלוֹת הַשַּׁחַר / וְעַד צֵאת הַכּוֹכָבִים.


בַּשַּׁחַק שִׁמְךָ / יוֹדוּ תַרְשִׁישִׁים

וּבָאָרֶץ בְּחִירִים / לְשִׁמְךָ מַקְדִּישִׁים

מְמַלְּלִים וּמְהַלְלִים / וְנֶגְדְּךָ נִגָּשִׁים

מִנַּעַר וְעַד זָקֵן / וְעַד טַף וְנָשִׁים.
