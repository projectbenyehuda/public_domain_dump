הֲלֹא אֶצְדַּק

הֲלֹא אֶצְדַּק בְּאָמְרִי כִּי אֲמָרָיו / כְּגִשְׁמֵי עָב אֲשֶׁר עַל גֵּז הֱרִיקָם

סְפוֹת חֶסֶד עֲלֵי חֶסֶד בְּשָׁפְטוֹ / כְּמִשְׁפַּט שׁוֹמְרֵי חֶסֶד וְחֻקָּם

וּמַה יַּמְרִיץ מְחוֹקֵק בֶּאֱנוֹשׁ? לוּ / יְהִי צַוָּאר אֱנוֹשׁ - הָיָה עֲנָקָם

וּמַה יֹּאמַר לְרֵעֵהוּ אֱנוֹשׁ? לוּ / קְרָאוֹ מֵעֲפַר אֶרֶץ אֲזַי קָם

וְאוּלָם כָּלְאוּ רַגְלַי נְגָעִים, / בְּגֵוִי יַעֲשׂוּ כָלָה וְנָקָם

אֲשֶׁר אִם אֹמְרָה יִכְלוּן - יְנוּבוּן / וְיַעַל מִבְּשָׂרִי אֵד וְהִשְׁקָם

כְּאִלּוּ הֶעֱלָם גַּבִּי בְּחָנֵס / לְמוּל פִּיחִים אֲשֶׁר מֹשֶׁה זְרָקָם

חֲמָתָם כַּחֲמַת עַכְשׁוּב וְתַנִּין / וְאֶחְשֹׁב כִּי אֲבִיר נָקָם נְשָׁקָם

וְאַל תִּתְמַהּ לְכָל אֵלֶּה! תְּמַהּ כִּי / בְּאֵין עֵצִים וְאֵשׁ יֹאפוּ בְּצֵקָם

וְעוֹרֵיהֶם כְּמוֹ עוֹרֵי יְרִיעוֹת / אֲרָגָם בֶּן אֲחִיסָמָךְ וְרָקַָם

אֲשֶׁר יִתְרוֹצְצוּ בָנִים בְּקִרְבָּם / עֲדֵי הַלַּיְלָה דָּמִי הֱנִיקָם

וְלוּ רָאָם רְאֵם - יָשׁוּב וְחָשַׁב / גְּאוֹן יַרְדֵּן, לְפִיהוּ לֹא יְרִיקָם

וְיִדְמוּ, בַּהֲמוֹת לִבָּם, לְהָרוֹת / אֲשֶׁר הַצַּר וְהַצִּיר הֶחֱזִיקָם

כְּשׁוֹשַׁנִּים נְשָׂאוּם לוֹקְטֵיהֶם / עֲדֵי כִּי כָרְתוּ שׁוֹקָם בְּשׁוּקָם

וְיַזְכִּירוּן לְבָבִי נַחֲלֵי נֹף / אֲשֶׁר הָיוּ לְדָם - בִּרְאוֹת אֲפִיקָם

וְהָעֵת יִפְעֲרוּ פִיהֶם - יְשִׂימוּן / לְבוּשִׁי חוּר וְאַרְגָּמָן בְּרֻקָּם

כְּמוֹ דֶרֶךְ אֱוִיל דַּרְכָּם, וְאִלּוּ / אֱוִיל שָׂמָם בְּמַכְתֵּשׁ - לֹא שְׁחָקָם

וְהָעֵת יִדְרְכֵם אִישׁ - יִתְּנוּ קוֹל / כְּשֵׁמַע קוֹל שְׂפָתַיִם בְּנָשְׁקָם

כְּאִלּוּ נַעֲשׂוּ בִידֵי בְצַלְאֵל / וּבִשְׁתֵּי כִכְּרֵי כֶסֶף יְצָקָם

וְהָאֶחָד קְרָאתִיו בֶּן נְתַנְיָה / וְהַשֵּׁנִי גְּדַלְיָה בֶן אֲחִיקָם

וְהָיִינוּ תְמוֹל רָצִים כְּסוּסִים / יְכַסֶּה אֶת פְּאַת שַׁחַק אֲבָקָם -

וְהַיּוֹם כָּשְׁלוּ בִרְכַּי וְנֶעְלַם / מְקוֹמָם מִמְּךָ, אָחִי, וּפִיקָם

וְשָׁלַחְתִּי לְךָ שִׂיחִי בְיַד צִיר / לְבַל יָשׁוּב לְךָ נָעוּר וְרֵיקָם

עֲדֵי אָשִׁית חֲסָדֶיךָ לְנֶגְדִּי / וְשִׁלַּמְתִּי אֲזַי חֻקָּם לְחֵיקָם.
