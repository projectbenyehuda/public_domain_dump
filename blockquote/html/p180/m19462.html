[אֲנִי הָאִישׁ אֲשֶׁר שִׁנֵּס אֱזֹרוֹ]

וכאן סאהרא פי בעץ͘ ליאלי אלסבות עלי אלנט͘ר וכאן מקתריא בצ͘ו אלקמר אלי אן תגיّם אלג͘ו ותכדّר אלאפק פאחתג͘ב ענד ורוד אלמטר ואמתנע ען נט͘רה פקאל פי וצף תלך אלחאלה קצידה מסתחסנה [ והיה ער בליל שבת אחד והיה מסתכל. והיה אור הלבנה מורה את דרכו עד שהעיבו העננים על האויר והתקדר האופק. ברדת הגשם נעלמה (הלבנה) ולא יכול לראותה. אז תיאר אותו המצב ואמר שירה שזכתה להוקרה]


אֲנִי הָאִישׁ אֲשֶׁר שִׁנֵּס אֱזֹרוֹ / וְלֹא יֶרֶף עֲדֵי יָקִים אֱסָרוֹ

אֲשֶׁר נִבְהַל לְבָבוֹ מִלְּבָבוֹ / וְנַפְשׁוֹ מָאֲסָה לִשְׁכֹּן בְּשָׂרוֹ

וּבָחַר בַּתְּבוּנָה מִנְּעוּרָיו / וְאִם כּוּר הַזְּמַן שֶׁבַע בְּחָרוֹ

וְיַהְרֹס כָּל אֲשֶׁר יִבְנֶה וְיִתּשׁ / אֲשֶׁר יִטַּע וְיִפְרֹץ אֶת גְּדֵרוֹ

וְגָשׁ לוּלֵי אֲשֶׁר תֵּקַד תְּלָאָה / וְיֶלֶד מִבְּנוֹת יָמִים סְגָרוֹ

לְקַצְוֵי מַעֲלוֹת חָכְמָה וּמוּסָר / וּמוֹסַד אֽוֹצְרוֹת שֶׂכֶל חֲקָרוֹ

וְדַע כִּי לֹא יְגַלֶּה עַד יְכַלֶּה / צְפוּנֵי תַעֲלוּמוֹת אִישׁ שְׁאֵרוֹ

וְקָנִיתִי תְמוֹל שֶׁמֶץ תְּבוּנָה / וְהִשְׁכִּים הַזְּמַן לִדְרשׁ מְחִירוֹ

וְעוֹדִי אֶרְכְּבָה לִדְרשׁ תְּבוּנָה / וְאִם לֹא יַחֲבשׁ הַיּוֹם חֲמוֹרוֹ

וְלֹא יֶחְלַשׁ לְבָבִי מִזְמַנִּי / אֲבָל יָקִים אֱסָרוֹ בַּל הֲפֵרוֹ

וְיָגֹרְתִּי מְיֻדָּעַי אֲשֶׁר בָּא / וְלֹא יָבֹא לְאִישׁ בִּלְתִּי מְגוּרוֹ

בְּעֵת לוּנִי וְהַשַּׁחַק נְקִי כָף / וְהַסַּהַר טְהָר־לֵבָב וּבָרוֹ

נְהָגַנִי עֲלֵי אָרְחֵי תְבוּנוֹת / וְהוֹרַנִי בְּאוֹר נָהֹג וְהֹרוֹ

וְחָמַלְתִּי בְּפַחְדִּי מִתְּלָאוֹת / עֲלֵי אוֹרוֹ כְּאָב עַל בֵּן בְּכֹרוֹ

וְרוּחַ שִׁלְּחָה בוֹ מִפְרְשֵׂי עָב / וּפָרַשׂ עַל פְּנֵי סַהַר אֲפֵרוֹ

כְּאִלּוּ אִוְּתָה זֶרֶם גְּשָׁמִים / וְתִשָּׁעֵן לְעָב עַד כִּי תְקֵרוֹ

וְשַׁחַק הֶעֱטָה קַדְרוּת וְסַהַר / כְּאִלּוּ מֵת וְהֶעָנָן קְבָרוֹ

וּבָכוּ אַחֲרָיו עֲבֵי שְׁחָקִים / בְּכוֹת כִּבְכוֹת אֱדֹם עַל בֶּן בְּעוֹרוֹ

וְלָבַשׁ לַֽיְלָה שִׁרְיוֹן אֲפֵלָה / וְרַעַם בַּחֲנִית בָּרָק דְּקָרוֹ

וְהַבָּרָק אֲזַי שַׁחַק מְעוֹפֵף / כְּאִלּוּ הוּא מְשַׂחֵק עַל דְּבָרוֹ

אֲשֶׁר פָּרַשׂ כְּנָפָיו כַּעֲטַלֵּף / וְעָפוּ עֹֽרְבֵי חשֶׁךְ בְּשׁוּרוֹ

וְסָגַר מַחְשְׁבֹתַי אֵל וְחֵפֶץ / לְבָבִי מִשְּׁנֵי פָנָיו אֲסָרוֹ

וְנֶאְסַר בַּעֲבֹת חשֶׁךְ לְבָבִי / וְהִתְעוֹרֵר כְּגִבּוֹר מִמְּצוּרוֹ

וְעֵת אוֹחִיל יְדִידַי וַאֲקַוֶּה / לְאוֹר סַהַר אֱשׁוּן חשֶׁךְ הֱמִירוֹ

כְּאִלּוּ קִֽנְאוּ עָבִים לְנַפְשִׁי / וְעַל כֵּן מָֽנְעוּ מִנִּי מְאוֹרוֹ

וְאַשְׁקִיף עֵת יְגַל פָּנָיו וְאָגִיל / כְּגִיל עֶבֶד אֲשֶׁר אָדוֹן זְכָרוֹ

בְּהִלָּחֵם אֱנוֹשׁ יֻכַּת חֲנִיתוֹ / וְעֵת יָרוּץ אֲזַי יִמְעַד אֲשׁוּרוֹ

וְכֵן אִישׁ יַדְבְּקוּ אֹתוֹ תְלָאוֹת / וְלוּ יָשִׂים בְּבֵית נֹגַהּ דְּבִירוֹ.
