אֶל מוּל הַשָּׂר אֶדֶר הַיְקָר

אֶל מוּל הַשָּׂר אֶדֶר הַיְקָר / שָׁלוֹם שָׁלוֹם עַד אֵין חֵקֶר

הָרָב זָקֵף בֵּי רָב טָרֵס / נַפְשׁוֹ לִפְנֵי הָאֵל תִּיקַר

חֶבְרָתוֹ הִיא בִי נִדְבֶּקֶת / חֶבְרַת בִּלְתּוֹ כַּעְנַן בֹּקֶר

חֶבְרַת בִּלְתּוֹ תִּמְצָא בַזּוֹל / אַךְ חֶבְרָתוֹ הִיא בַיֹּקֶר

חֶבְרָתוֹ הִיא אֶצְלִי בֶאֱמֶת / חֶבְרַת בִּלְתּוֹ שָׁוְא עִם שֶׁקֶר

חֶבְרָתוֹ הִיא אֶצְלִי נִצְחִית / חֶבְרַת בִּלְתּוֹ כֻּלָּהּ כִיקַר

עַל שֻׁלְחָנוֹ תִּשְׂבַּע תָּמִיד / זוּלַת בֵּיתוֹ רַגְלָךְ הוֹקַר

סוֹד הָעִבּוּר הִשִּׂיג אוֹתוֹ / עַל מַתְכֻּנְתוֹ לַכֹּל נֶחְקַר

סִבַּב גַּלְגַּל עִם חוּג אֶרֶץ / לִבּוֹ עָמַד עַל הָעִיקָר

שִׂיחַ בִּלְתּוֹ עַל לִבִּי חָם / אָכֵן שִׂיחוֹ כַמַּיִם קַר

שִׂיחוֹ אֵלַי מָתוֹק כִּדְבַשׁ / שִׂיחַ בִּלְתּוֹ לִבִּי דָקַר

אִם בָּא לִרְאוֹת מַכְאוֹבִי סָר / אַךְ זוּלָתוֹ לַשָּׁוְא בִּקַּר

הָאֵל קָרֵב בּוֹאוֹ אֵלַי / נִרְוֶה דוֹדִים עַד הַבֹּקֶר.
