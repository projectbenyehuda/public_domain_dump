הוֹי כָּל־מְתֵי שֵׂכֶל

קינה על מות יצחק בן ברון


הוֹי כָּל־מְתֵי שֵׂכֶל וְיוֹדְעֵי סֵפֶר / הוֹי לַעֲנוֹת עִמִּי לְאֶשְׁכֹּל כֹּפֶר,

וּבְכוֹת כְּעַל יָחִיד וְהָמֵר עַל בְּכוֹר / וּסְפֹד בְּצוֹם וּבְכִי וְשַׂק וָאֵפֶר,

כִּי חֹק תְּעוּדָה הַזְּמָן חָלַף בְּיוֹם / לֻקַּח וְהַדִּבֵּר בְּרִיתוֹ הֵפֵר,

וּשְׁמֵי חֲסָדִים נִמְלְחוּ בִלְתּוֹ וְנָ/גֹלּוּ תְהִלּוֹת הָאֱנוֹשׁ כַּסֵּפֶר,

יִצְחָק אֲשֶׁר הָיָה לְאוֹת עַל מִפְלְאוֹת / חָכְמָה וְהַנּוֹתֵן לְאִמְרֵי שֶׁפֶר,

אֵלָיו דְּבַר שַׁחַק לְבַד גֻּנַּב וּפִיו / רוֹגֵל צְפוּנוֹתָיו וְסוֹדָיו חוֹפֵר,

נוֹקְשׁוּ בְנֵי יָקְשָׁן בְּאִמְרֵיהֶם כְּמוֹ / עָפוּ בְנֵי עֵיפָה וְיַלְדֵי עֵפֶר,

הָעֵט וְהַדִּבֵּר לְמָאוֹס הֵם, הֲכִי / תָּעוּ מְתֵי צַחוֹת וְיוֹדְעֵי סֵפֶר.

כָּל עֵץ בְּגַן אֵל לֹא עֲמָמֻהוּ, וְאֵיךְ / תִּשְׁאַל בְּעַד עֵץ הַהֲדַס אוֹ נֹפֶר?

אֵיךְ חוּג אֲדָמָה יַהֲלֹךְ אוֹ לֹא יְהִי / שַׁחַק עֲשִׂירִי מִנְּתִיבוֹ צוֹפֵר?

בּוֹשָׁה לְבָנָה מֵעֱזוּז שֹׁדוֹ וְכֵן / קָדְרוּ פְנֵי חַמָּה וְחָפְרוּ חֵפֶר,

תַּחְתָּיו תְּאַו כִּימָה הֱיוֹת נַפְשָׁה וְשׂוּם / עַיִשׁ וּבָנֶיהָ לְנַפְשׁוֹ כֹפֶר.

בֶּטֶן שְׁאוֹל צַר מִנְּשׂוֹא שֶׁמֶשׁ וְזָר / כָּל הָאֱנוֹשׁ קֶבֶר לְסַהַר חוֹפֵר.

הַקֵּץ בְּעֵת יָקִיץ יְהִי רֶגֶל אֱנוֹשׁ / קַל כַּצְּבִי אֵלָיו וְרָץ כָּעֹפֶר.

לֻקַּח כְּתִשְׁבִּי בַּעֲווֹן הַדּוֹר, הֲכִי / לֹא מֵת בְּחַטָּאתוֹ כְּמוֹת בֶּן חֵפֶר,

נִגְנַז כְּמוֹ נִגְנַז אֲרוֹן־קֹדֶשׁ, אֲשֶׁר / צֻפָּה בְּזָהָב טוֹב וְלֹא בַכֹּפֶר.

נִִסְתַּר בְּצֵל שַׁדַּי וּבִכְתָב עִם בְּנֵי / עֶלְיוֹן שְׁמוֹ הוּחַק בְּשֵׁבֶט סוֹפֵר,

לִלְבֹּשׁ בְּכֵס אֵל מַחְלְצוֹת כָּבוֹד, / לֹא מַחְלְצוֹת אָרְגוּ יְדֵי אִישׁ תּוֹפֵר.

כָּל שֹׁד אֲשֶׁר נוֹלָד – וְיִכְלֶה, בִּלְעֲדֵי / זֶה, כִּי בְּמֵי דִמְעַת מְבִינִים יִפֶר.

מָה אַאֲרִיךְ בִּסְפוֹר וְסַפֵּר אֶת יְקָר / גָּדְלוֹ, וְלֹא יִמַּד וְלֹא יִסָּפֵר.
