מָרוֹם וְקָדוֹשׁ שְׁעֵה

לקרובה א'


מָרוֹם וְקָדוֹשׁ שְׁעֵה שַׁוְעַת שְׁבוּיִם / מִמְּעוֹנִים

הִתְעַשֵּׁת לָהֶם וְכָתְבֵם לְחַיִּים / עִם אֱמוּנִים

וּקְרָא שְׁנַת רָצוֹן לְרִיב הָעֲנִיִּים / וְהָאֶבְיוֹנִים: / ישע' סא,כ;מא,יז

התעשת


שְׁלוֹמִים נִפְלָאוּ / מִמְּעוֹנִים

אֶל עַם נִצְבָּאוּ / עִם אֱמוּנִים

בִּבְכִי יָבֹאוּ / וּבְתַחֲנוּנִים: / ירמ' לא,ח

התעשת


הַעֲלֵם עִם בְּנֵיהֶם / מִמְּעוֹנִים

וּסְלַח חֲטָאֵיהֶם / עִם אֱמוּנִים

אֲשֶׁר כָּשְׁלוּ בָהֶם / אָבוֹת וּבָנִים: /שם ו,א

התעשת


קְשֹׁב נָא בָנֶיךָ / מִמְּעוֹנִים

וְהַשְׁקֵם לְפָנֶיךָ / עִם אֱמוּנִים

בְּאֵר מַיִם חַיִּים / עִם גַּנִּים: / שה"ש ד,טו

התעשת


טַל סְלִיחָה הָרֵק / מִמְּעוֹנִים

שַׁעֲרֵי חֵן הָפֵק / עִם אֱמוּנִים

וְיָבֹא גוֹי צַדִּיק / שֹׁמֵר אֱמֻנִים: / ישע' כו,ב

התעשת


נְבָרִים הֵבִינוּ / מִמְּעוֹנִים

כַּשֶּׁלֶג יַלְבִּינוּ / עִם אֱמוּנִים

אִם יִהְיוּ חֲטָאֵיהֶם / כּשָּׁנִים: / שם א, יד

התעשת

שְׁלַח קוֹל חֲסָדֶיךָ / מִמְּעוֹנִים

לְהוֹשִׁיב עֲבָדֶיךָ / עִם אֱמוּנִים

מֵעַבְדֵי אֲדֹנָי / הַקְּטַנִּים: /ישע' לו,ט

התעשת
