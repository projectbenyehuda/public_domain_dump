[לַמִּשְׁפָּט אִמְרָתוֹ יִשְׁלַח]

לקרובה א'


לַמִּשְׁפָּט אִמְרָתוֹ / יִשְׁלַח שָׁלֹחַ

אָנֹכִי מָלֵאתִי כֹחַ: [מי' ג,ה]

נֶאְזָר בִּגְבוּרָה / מֵאֵין שְׁתִי וְנוֹחַ

נֹתֵן לַיָּעֵף כֹּחַ: [ישע' מ,כט]

יְדֵי גַאֲוָתוֹ / יְנַצְּחוּ נָצֹחַ

יַחְפְּרוּ בָעֵמֶק וְיָשִׂישׂ בְּכֹחַ: [איו' לט,כא]

מַלְכֵי עוֹלָם לְפָנָיו / נִשְׁטְחוּ שָׁטֹחַ

מֵרֹב אוֹנִים וְאַמִּיץ כֹּחַ: [ישע' מ,כו]

שֵׁבֶט גְּאוֹנוֹ / הִבְרִיחַ בָּרֹחַ

שַׁדַּי לֹא מְצָאנֻהוּ שַׂגִּיא כֹחַ: [איו' לז,כג]

הֲקָצְרָה יָדוֹ מִפְּדוּת / וְדֶלֶת לִפְתֹּחַ

הֲלֹא יִשְׁפָּט-בָּם כִּי אֵין בָּנוּ כֹּחַ: [דהי"ב' כ,יב]

בָּרן יַחַד כּוֹכְבֵי חֲיָלָיו / בַּעֲרֹב וּבִזְרֹחַ

בָּרְכוּ אֲדֹנָי מַלְאָכָיו גִּבֹּרֵי כֹחַ: [תה' קג,כ]

רוּם תִּפְאַרְתּוֹ / יִפְצְחוּ פָצֹחַ

רַב לְאֵין כֹּחַ: [דהי"ב' יד,י]

יִדְגּוּ דּוֹרְשָיו / וְיַשִּׂיגוּ מָנוֹחַ

יָנוּחוּ יְגִיעֵי כֹחַ: [איו' ג,יז]

עָנִי וְנִדְכֶּה / יִדְכֶּה יָשֹׁחַ

עָזַרְתָּ לְלֹא כֹחַ: [שם כו,ב]

קוֹרֵא לַאֲסִירָיו / פְּקַח-קוֹחַ

קוֹל אֲדֹנָי בַּכֹּחַ: [תה' כט,ד]

בְּרָא לִי אֱלֹהִים / לֵב חָדָשׁ וְרוּחַ
חַנּוּן הַמַּרְבֶּה לִסְלוֹחַ: [ישע' נה,ז]
