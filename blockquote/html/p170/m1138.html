הריח מר מאחז האפסים

[הֲרֵיחַ מֹר מְאַחֵז הָאֲפָסִים] / ר' משה אבן-עזרא


הֲרֵיחַ מֹר מְאַחֵז הָאֲפָסִים / וְאִם־רוּחַ תְּנֹפֵף הַהֲדַסִּים

וְעָנָן אוֹ עֲתַר קָנֶה וְקִדָּה / וּבָרָק אוֹ בְרַק כֹּסוֹת עֲסִיסִים

וְהֶעָבִים יְרִיקוּן הַבְּשָׂמִים / וּמֵרָאשֵׁי הֲדַסִּים הָֽרְסִיסִים

וְהָרִים יִפְצְחוּ רִנָּה בְּלִי־פֶה / וְאִם תֹּרִים עֲלֵי בַדִּים וְסִיסִים

וְאֶרֶץ מִשְׁבְּצוֹת זָהָב לְבוּשָׁהּ / וּמֶשִׁי כֻתֳּנוֹתֶיהָ וּפַסִּים

וְיָשְׁרוּ כָל־נְתִיבֶיהָ לְצֹעֵד / וְשָׁבוּ כַבְּקָעוֹת הָֽרְכָסִים

וְהַבָּתִּים יְרַנֵּנוּ וּמִקִּיר / צְרוֹר יַעַן וּמֵעֵצִים כְּפִיסִים

וּפָנִים זֹעֲפִים לָבְשׁוּ עֲלִיצוּת / וְעַלִּיזִים מְתֵי יָגוֹן וְשָׂשִׂים

וְשִׂפְתֵי עִלְּגִים צַחוֹת יְנוּבוּן / לְקוֹמֵם בִּנְיְנֵי גִיל הַהֲרוּסִים

הֲכִי נִרְאוּ פְלָאִים הַצְּפוּנִים / אֲשֶׁר הָיוּ בְּלֵב יָמִים כְּמֻסִים

וְאֹהֶל הַֽיְקָר חֻבַּר וְלֻלְאוֹת / תְּהִלּוֹת הֽוּבְאוּ בָהֶם קְרָסִים

וְיָמִים קֵֽרְבוּ עֶצֶם לְעַצְמוֹ / וְדֻבָּקוּ בְּשָׂרִים הַפְּרוּשִׂים

שְׂמָחוֹת נַעֲלוּ עַל־הַרֲרֵי מֹר / וְעַל־גִּבְעוֹת לְבוֹנָה הֵם כְּנִסִּים

וְעַל־כַּנְפֵי שְׁחָרִים קוֹל מְבַשֵּׂר / בְּתֵבֵל לֹא עֲלֵי כָרִים וְסוּסִים

שְׁלֹמֹה אֵֽרְשׂוּ דֹדָיו כְּבוּדָּה / וְאִם־סַהַר וְחַמָּה הָאֲרוּשִׂים

אֲשֶׁר אָשְׁרוּ נְתִיב־שֵׂכֶל פְּעָמָיו / וְרַגְלָיו דֹּֽרְכִים מוּסָר וּבוֹסִים

וּמִנֹּעַר שְׂעִפָּיו עַל־שְׁחָקִים / וְרַעְיוֹנָיו עֲלֵי־עַיִשׁ עֲמֻסִים

וְגָבַהּ מִבְּנֵי גִילוֹ כְּגֹבַהּ / בְּנֵי אִישׁ מִבְּהֵמוֹת אוֹ רְמָשִׂים

וְיָקְרוּ מַעֲלָלָיו מֵעֲלֵיהֶם / כְּמוֹ יָקְרוּ פְנִינִים מֵחֲרָשִׂים

יְרוֹצֵצוּ אֲבַק רַגְלָיו לְהַשִּׂיג / וְאֵיךְ צִפּוֹר יְהִי רֹדֵף פְּרָסִים

עֱנוּ אֵיכָה כְסִיל נִפְגַּשׁ בְּכִימָה / עֲלֵי־אֶרֶץ בְּלִי דֹרֵשׁ וּמֵשִׂים

וְאִם אֵינָם הֲלֹא כַנְפוֹת הֲדָרָם / עֲלֵי אַפֵּי תְהִלּוֹתָם פְּרֻשִׂים

וְאֵיךְ גָּזְלוּ מְאוֹרֵיהֶם הֲנִרְאוּ / אֲנָשִׁים בֹּזֲזִים אוֹרִים וְשֹׁסִים

יְדִידָיו מַהֲרוּ אֶל־יֵין יְדִידֹת / שְׁתוֹת הַיּוֹם בְּמִזְרַק הַמְּשׂוֹשִׂים

וְעוֹד הִתְעַתְּדוּ לִכְפֹּל שְׂמָחוֹת / וּפִתְחוּ הַהֲנָחוֹת מַאֲבוּסִים

כְּמַיִם קָֽפְאוּ הֵם הַגְּבִיעִים / וְגֶחָלִים בְּתוֹכֵמוֹ נְמַסִּים

שְׁתוּ־נָא כַּאֲשֶׁר יִסַּד בְּבֵיתוֹ / שְׁתוֹת כַּדָּת לְבַל־תִּהְיוּ אֲנוּסִים

וְגִיל בָּחוּר בְּאַיֶּלֶת אֲהָבִים / וְהַרְנִינוּ שְׁנֵיכֶם נֶעֱלָסִים

וְהִתְעַדֵּן בְּקוֹמַת־חֵן כְּתָמָר / אֲבָל תָּנַע כְּבַדֵּי הַהֲדַסִּים

וְאַל־תִּפְחַד לְקוֹל חֶלְיָה בְּצַוָּאר / לְעֵת נֶשֶׁף וְרִגְשַׁת הַשְּׁבִיסִים

וְאַל־תִּרְהֶה לְעֵינַיִם כְּיוֹנִים / אֲשֶׁר שָׁכְרוּ אֲבָל מִיֵּין חֲמָסִים

וְיַעְמֹד לִבְּךָ בַּחְבוֹק זְרֹעוֹת / בְּאֶצְעָדוֹת וְתִפְאֶרֶת עֲכָסִים

וְאַל־תִּבְרַח לְצִפְעֹנֵי קְוֻצּוֹת / עֲלֵי פָנִים בְּמֵי־בֹשֶׁת כְּבוּסִים

הֲכִי לִקְרָֽאתְךָ יֵצְאוּ לְשָׁלוֹם / וְאִם־סֹתְרִים הֲדַר פָּנִים וְכֹסִים

וְרִמּוֹנִים בְּגַן שׁוֹשָׁן וְאָכֵן / בְּרָאשֵׁי מַשְׂמְרוֹת בֹּשֶׂם תְּפוּשִׂים

וְיָדֶיךָ בְּעָבְרָם עַל־פְּנֵיהֶם / לְאִטָּם מֹעֲכִים אֹתָם וְעֹשִׂים

וְדַע כִּי־הַזְּמָן עֶבֶד רְצוֹנְךָ / וּבָנָיו לַעֲשׂוֹת חֶפְצָךְ כְּנוּסִים

לְהַקְרִיב מִשְׁאֲלוֹתֶיךָ יְחִישׁוּן / וְאֶת־כָּל־תַּאֲלַת לִבְּךָ מְנִיסִים

וְהֵא שִׁיר דּוֹד בְּךָ יִשְׂמַח לְבָבוֹ / וּמוֹרָשָׁיו לְךָ פֹּרְחִים וְטָשִׂים

מְעִיל כָּבוֹד לְבַל־יִבְלֶה לְעוֹלָם / עֲדֵי יִהְיוּ יְסוֹדוֹתָיו נְתוּסִים

הֲכִי מֵימֵי יְדִידוּתוֹ כְּשֶׁלֶג / מְאֹד זַכּוּ בְּרֶגֶל לֹא רְפוּשִׂים

וְאִם־יִסָּֽתְרוּ מִמְּךָ בְּלִבִּי / בְּנֵרוֹת דֵּעֲךָ יִהְיוּ חֲפוּשִׂים

לְמַעַן לַמְּבִינִים הַפְּנִינִים / וְלָרֵקִים בְּרִיאֵי הַכְּבָשִׂים

וְתַאְוַת הַגְּבָרִים לַאֲמָרִים / וְהַפֹּתִים לְבַרְבֻּרִים אֲבוּסִים

יְלִיד טוֹבִים חֲיֵה עִם בַּת־נְדִיבִים / בְּצֵל שַׁדַּי שְׁלֵוִים בּוֹ וְחֹסִים.
