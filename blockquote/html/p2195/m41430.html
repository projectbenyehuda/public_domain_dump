אֲנִי לָךְ

לכבוד אהרן שלום ב״ר יעקב גבאי

סימן: אהרן שלום בכר יעקב יצו, נסים.


אֲנִי לָךְ דּוֹד אֲזַמֵּר בַּתְּהִלָּה / לְשַׂר גָּדוֹל וְטַפְסָר בַּתְּהִלָּה

הֲלֹא הִפְלִיא חֲסָדָיו בִּכְתָבוֹ / וְהוּא הִקְדִּים לְשָׁלוֹם בַּתְּחִלָּה

רְאוּ הִדּוּר כְּתָבוֹ וַאֲמִירָיו / בְּאַהְבָה גָבְהוּ לִשְׁמֵי זְבוּלָה

נְדִיב לֵבָב וְהוּא נִסְתָּר וְאָכֵן / כְּתָבוֹ אַהֲבָה מוֹדָע בְּנִגְלָה

שְׂפָתָיו יִנְטְפוּ נֹפֶת וְיוֹתֵר / מְתוּקִים הֵם וְזֶה דֶרֶךְ שְׁאֵלָה

לְעוֹלָם יִהְיֶה דָּשֵׁן וְשָׁמֵן / וְדַרְכּוֹ תִהְיֶה דֶרֶךְ סְלוּלָה

וְיַחְזֹר אֶל נְוֵה קָדְשׁוֹ בְכָבוֹד / בְּעֹשֶׁר רַב וְגַם טוֹבָה כְפוּלָה

מְיֻדָּעָיו הֲלֹא הֵם יִשְׂמְחוּ בוֹ / וְיָשִׂישׂוּ בְרֹב שִׂמְחָה וְגִילָה

בְּבוֹאוֹ חִישׁ הֲלֹא יִזְכֶּה לְבָנִים / כְּזַיִת עַל פְּנֵי מַיִם שְׁתוּלָה

רְצֵה שִׁירִי וְקַבֵּל מַאֲמָרִי / יְהִי רָצוּי כְּמוֹ מִנְחָה בְלוּלָה

עֲשֵׂה עִמִּי לְמַעַן אַהֲבָתְךָ / קְדוּמָה הִיא וְהִיא עוֹלָה לְמַעְלָה

בְּכָל יוֹם חוֹב עֲלֵי רֵעַ וְאָהוּב / יְהִי זוֹכֵר לְדוֹדוֹ בַתְּפִלָּה

צְעָדָיו יִשְׁמְרָה הָאֵל בְּחַסְדּוֹ / וְיִרְאֶה אֶת בְּנֵי בֵיתוֹ בְצַהְלָה

דְּעֵה אָהוּב בְשִׂמְחָתְךָ לְבָבִי / מְאֹד יִשְׂמַח בְּרֹב שִׂמְחָה גְדוֹלָה

גְּלִילַת אֶצְבְּעוֹת יָדְךָ תְּרַפֵּא / נְפָשׁוֹת הַמְּעֻנּוֹת עַד נְקַלָּה

וְשִׁירָתִי לְדוֹדִי הִיא בְחִנָּם / וְאֵינֶנִּי מְצַפֶּה אֶל גְּמוּלָה

וְהוּא הִקְדִּים וְהִגְבִּיר בַּחֲסָדָיו / גְּמוּל הַטּוֹב נְשַׁלֵּם לוֹ בְמִלָּה

נְשַׁבֵּחַ לְשׁוֹכֵן בָּעֲרָבוֹת / לְאֵל עֶלְיוֹן וְהוּא נוֹרָא עֲלִילָה

שְׂבַע רָצוֹן יְהִי דוֹדִי וּמָלֵא / לְבִרְכַּת אֵל וְלֹא יֵדַע גְּבוּלָה

יְמַלֵּא מִשְׁאֲלוֹת לִבּוֹ לְטוֹבָה / וְיַצְלִיחַ וְיִמְצָא חֵן וְחֶמְלָה

מְאֹד יִגְבַּר וְיִתְנַשֵּׂא וְיָרוּם / עֲלֵי אוֹיְבָיו יְהִי עוֹלֶה לְמַעְלָה.
