אֲהַלֶּלְךָ בְּטוּב

סדר אלפא ביתא

סימן: א״ב נסים.


אֲהַלֶּלְךָ בְּטוּב מַעְנֶה וּמִלָּה / חֲשֹׁב אוֹתָהּ כְּמוֹ מִנְחָה בְלוּלָה

בְּזִמְרָתִי אֲדוֹן תִּבְחַר וְתִכְסַף / וְאַף אִם לַשְּׁגִיאוֹת הִיא עֲלוּלָה

גְּבוּרוֹת אֵל רְצוֹנִי אֶזְכְּרָה בָהּ / וְאָז תִּהְיֶה בְתוֹךְ לִבִּי שְׁתוּלָה

דְּרָכֶיךָ ה׳ לַמְּדֵנִי / וְאֵלוּ הֵם דְּבָרַי בַּתְּחִלָּה

הַלְלוּיָה הַלְלוּ כוֹכְבֵי אוֹר / וּמַזָּלוֹת שְׁנֵים עָשָׂר בְּצַהְלָה

וְשִׁבְעָה כוֹכְבֵי לֶכֶת וְעִמָּם / שְׁנֵי הַקְּטֳבִים עִם הָעֲגָלָה

זְרִיזִים הֵם בְּשִׁמּוּשָׁם לְאַהְבַת / רְצוֹן בּוֹרְאָם וְהֵם עוֹשִׂים בְּגִילָה

חֲסָדֶיךָ אֲדוֹן עוֹלָם מְלֵאָה / בְּחַסְדֶּךָ שְׁלַח לִי חִישׁ מְחִילָה

טְהוֹר לֵבָב יְהִי נִצָּב לְמוּלָךְ / וְאוּלַי יִהְיֶה שׁוֹאֵל שְׁאֵלָה

יְשַׁו אֵל חַי לְבַדּוֹ הוּא לְנֶגְדּוֹ / וּמִימִינוֹ בְּלִי יִמּוֹט לְסֶלָה

כְּבוֹדָךְ הוּא לְעוֹלָם אַתְּ יְסַדְתּוֹ / בְּלִי שִׁפְעָךְ אֲזַי תִּהְיֶה חֲלוּלָה

לְדֹר וָדֹר אֱמוּנָתְךָ יְסוֹדָהּ / וְכוֹנַנְתָּהּ וְתַעְמֹד בַתְּחִלָּהּ

מְאֹד גָּבְרוּ חֲסָדֶיךָ וְגָדְלוּ / פְּלָאֶיךָ עֲדֵי לִשְׁמֵי זְבוּלָה

נְצֹר עֵדָה טְהוֹרָה הִיא חֲמוּדָה / כִּימֵי קֶדֶם נְצָרְתָּהּ בַּתְּחִלָּהּ

סְמֹךְ אוֹתָהּ שְׁמָךְ סוֹמֵךְ וְעוֹזֵר / עֲזֹר אוֹתָהּ וְחִישׁ קַבֵּץ קְהָלָהּ

עֲשֵׂה עִמָּה לְטוֹבָה אוֹת וְאַל נָא / תְּשַׁלֵּם לָהּ בְרֹאשָׁהּ אֶת גְּמוּלָהּ

פְּדֶהָ מִמְּצוּקוֹת צַר וְחַלֵּץ / נְשָׁמוֹת עַמְּךָ מִן הַמְּצוּלָה

צְדָקוֹת תִּזְכְּרָה רַחֵם וְהוֹשַׁע / לְעַמֶּךָ שְׁלַח טוֹבָה כְפוּלָה

קְדוֹשִׁי אַתְּ מְאֹד נִשְׂגָּב מְרוֹמָם / עֲלֵי כָל הַבְּרָכוֹת גַּם תְּהִלָּה

רְבָבוֹת מַלְאֲכֵי רוּם אִם, יְהוֹדוּן / וְהַחַיּוֹת וְאוֹפַן בַּהֲמֻלָּה

שְׂרָפִים עוֹמְדִים בְּשֵׁשׁ כְּנָפוֹת / וּמַקְדִּישִׁים לְשֵׁם נוֹרָא עֲלִילָה

תְּמִימֵי עָם בְּנִיב שַׁוְעָם וּמִלָּם / לְעֻמָּתָם תְּהִלָּתָם כְּלוּלָה

נַהֲלֵנִי בְעֻזֶּךָ וְצִדְקָךְ / שְׁלַח לְעַבְדְּךָ מַהֵר תְּעָלָה

סְעָדֵנִי וְתַנְחֵנִי בְדַרְכָּךְ / תְּהִי דַרְכִּי לְפָנַי אָז סְלוּלָה

יְהִי חַסְדָּךְ לְעֻמָּתִי וְאִתִּי / תְּקַבֵּל אֶת תְּפִלָּתִי כְעוֹלָה

מְשֹׁךְ חַסְדָּךְ עֲלֵי עַבְדָּךְ אֲקַו לָךְ / אֲהַלֶּלְךָ בְּטוּב מַעְנֶה וּמִלָּה.
