[היום ילך וירחיק]

ולה גואב אלי בעץ אכואנה ען קציד כאטבה בה

ולו תשובה לאחד מרעיו על שיר תהלה שכתב אליו

U – – – U – – – U //– – U – – – U – – – U – –

הֲיוֹם יֵלֵךְ וְיַרְחִיק נֶאֱמָנָי // וְיוֹם יָבוֹא וְיַקְרִיב אֶת זְקוּנָי

וְלֹא אָשִׁית לְבָבִי אֶל אֲשֶׁר סָר, // וְלֹא אֶרְאֶה, אֲשֶׁר יָבוֹא, בְּעֵינָי?

וְהַזִּקְנָה שְׁאֵר יוֹם הַפְּקֻדָה, // וְהַמָּוֶת אֲחִי פֵרוּד, נְבוֹנָי,

וְאֶשְׂמַח בַּזְּמַן כִּי יַאֲרִיךְ לִי – // וְהוּא כִלָּה אֲנָשִׁים מִלְּפָנָי

וְלִי יִקְרֶה כְּמִקְרֵיהֶם וְאֵרֵד // אֱלֵי שַׁחַת כְּמוֹ רֶדֶת זְקֵנָי.

יְגוֹנַי עַל תְּמוּתָה הָעֲתִידָה, // וְכוֹסָהּ אֵשְׁתְּ עֲלֵי כָרְחֵי יְגוֹנָי,

וְאֶשָּׂאֶה כְּלֵיל מָוֶת בְּלַיִל // יְהִי מִמָּחֳרָתוֹ נֹד שְׁכֵנָי,

וְיִדְוֶה עַל פְּרִידָתָם לְבָבִי // כְּמוֹ יִדְוֶה בְּהִפָּקֵד עֲווֹנָי

וְלֹא לִמְעַט וְגַם לֹא עַל נְקַלָּה // נְסִיעָתָם תְּעוֹרֵר אֶת שְׁאוֹנָי:

חֲשָׁבוּנִי כְּאִישׁוֹנָם בְּעֵינָם // וְכִימִינָם, וְשַׂמְתִּים לִי אֲדָנָי

וְהִרְבּוּ, יוֹם אֲשֶׁר נָסְעוּ, יְגוֹנִים, // כְּמוֹ הִרְבּוּ, בְּיוֹם חָנוּ, נְגִינָי,

וְהֵם הָיוּ בְּלֵב צָרַי רְמָחַי // וְהֵם הָיוּ בְּצִדֵּיהֶם צְנִינָי

וְאֵיךְ יַעַל עֲלֵי לִבִּי נְדוֹדָם // וְאָגִילָה וְיִשָּׁמְעוּ רְנָנָי

וְאֵיךְ אֶזְכֹּר פְּרִידַת מוֹר פְּלוֹנִי // וְלֹא אֶשְׁכַּח בְּיוֹם מוֹעֵד שְׂשׂוֹנָי

אֲשֶׁר נָסַע – וְנָסְעוּ כָל בְּשָׂמַי // בְּמַסָּעוֹ, וְרֵאשִׁית כָּל שְׁמָנָי,

וְסָרָה מֵעֲלֵי רֹאשִׁי עֲטֶרֶת // וְסָרוּ מֵעֲלֵי גִלְדִי עֲדָנָי,

יְדִיד נַפְשִׁי וְהַפָּדוּי בְּנַפְשִׁי // וְאַף כִּי יַהֲלוֹמַי אוֹ פְּנִינָי

וְהָעוֹרֵךְ תְּעוּדַת אֵל בְּישֶׁר, // כְּאִלּוּ לִמְּדוֹ משֶׁה בְּסִינָי,

אֲשֶׁר יָבֹא כְּתָבוֹ יוֹם מְבוֹאוֹ // וְיִתְרָאֶה שְׂשׂוֹן לִבִּי בְּפָנָי,

וְאֶתְהַלֵּךְ בְּשִׂמְחַת לֵב כְּשׁוֹלֵל // וְיִתְבּוֹנֵן שְׂשׂוֹנִי עַל סְדִינָי

יְהַלֶּלְךָ זְמַנִּי עַל לְשׁוֹנִי, // כְּאִלּוּ מִמְּךָ יָמַי וְשָׁנָי

וְיַגִּידוּ שְׁבָחֶיךָ עֲצָמַי, // כְּאִלּוּ אַתְּ אֲשֶׁר תָּכִין מְזוֹנָי

וְנֹעַם סִפְרְךָ חִדָּה לְבָבִי, // כְּחֶדְוָתוֹ בְּנִטְעֵי נַעֲמָנָי

וְשָׁלַחְתִּי לְךָ בֶּן עֵט בְּנַחַץ // וְאֵדַע כִּי יְקוּצוּן בּוֹ מְבִינָי,

לְמַעַן לֹא הֲכִינוֹתִיו בְּיָמִים // וְגַם לֹא יֻגְּעוּ בוֹ רַעֲיוֹנָי.

אֲנִי תוֹמֵךְ בְּכוֹסִי בָךְ, לְמַעַן // מְנַת כּוֹסִי בְּךָ מֵאֵת אֲדֹנָי,

וְאֶתְמֹךְ אֶת אֲהָבֶיךָ בְּחַיַּי, // וְאַנְחִילֵם בְּיוֹם מוֹתִי לְבָנָי!
