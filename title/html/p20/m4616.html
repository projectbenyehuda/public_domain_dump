[אל חי בקרבי]

מוחרך


אֵל חַי בְּקִרְבִּי, / בִּשְׁמוֹ יְרַנֵּן לִבִּי.

בַּנְתִּי בְסוֹדִי, / שַׁרְתִּי: אֲנִי הוּא עֵדִי,

יָגֵל כְּבוֹדִי / כִּי פוֹעֲלִי עִמָּדִי,

לָמָּה בְּעוֹדִי / אֶתְעַצְּבָה בִּנְדוֹדִי?

אֶזְכֹּר בְּחֻבִּי / טוּבוֹ, וְסָר כָּל עָצְבִּי.

רוּחַ לְרוּחִי / זִכְרוֹ וְהֶעְצִים כֹּחִי,

הוּא בֵּית מְנוּחִי, / הוּא מִשְׁכְּנוֹת מִבְטַחִי,

אֶדְבַּק בְּגוּחִי, / הוּא בְהָקִיצִי שִׂיחִי,

שַׁחְרִי וְעַרְבִּי / יִתְדַּפְּקוּ מוֹפְתָיו בִּי.

הוֹדוּ לְמוּלִי / שַׁתִּי וְאִם הוּא פֶלִיא,

מִכָּל עֲמָלִי / נַחְתִּי וְהוּא גוֹרָלִי,

מַה יַעֲשֶׂה לִי / נוֹצָר – וְהִנֵּה אֵלִי

קַשְׁתִּי וְחַרְבִּי, / עֻזִּי וְצוּר מִשְׂגַּבִּי.

מִלָּיו בְּדַרְכִּי / כִּי נַעֲלֶה עַל עֶרְכִּי,

וּשְׁמוֹ בְּחָשְׁכִּי / נֵרִי וְצִלּוֹ סֻכִּי,

אֶשַּׁח לְמַלְכִּי – / אֶתְנַשְּׂאָה בּוֹ, עַד כִּי

גַּלְגַּל מְסִבִּי / דַּעְתּוֹ תְּשַׁוֶּה רִכְבִּי.
