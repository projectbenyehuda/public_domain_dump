[אחרה עת מועד]

מאורה


אֵחֲרָה עֵת מוֹעֵד- / מוֹעֲדִים-וָחֵצִי,

בַּשְּׁבִי אֵין סוֹעֵד / לֵב לְגַלּוֹת קִצִּי

לִי, וּמָתְנִי מוֹעֵד / שָׁם וְאָנוּשׁ חִצִּי.

אֵין יְשׁוּעָתָה לִי / בָּאֱלֹהִים דִּמָּה

צַר בְּאֹרֶךְ שִׂבְרִי – / וַאֲדֹנָי שָׁמָּה!


בֶּאֱשׁוּן לֵיל שַׁחְתִּי, / נָד בְּאַרְבַּע רוּחוֹת,

מִצְּבִי נִסַּחְתִּי – / אֵיךְ אֲדַבֵּר צָחוֹת?

אַךְ בְּמַר לֵב שַׂחְתִּי / לַעֲדִינוֹת בּוֹטְחוֹת:

נָא שְׁמַעַן קוֹלִי, / יַעֲבֹר עָלַי מָה,

עוֹד יְרַפֵּא שִׁבְרִי / אֵל יְמִינוֹ רָמָה!


רֹאשׁ מְשַׂנְּאַי הֵנַע, / שֵׁן לְאֵידִי חָרַק,

יוֹם כְּבוֹדִי נִכְנַע / שָׂש וְעָלַי שָׁרַק.

וַאֲנִי עַד הֵן נָע, / נָד כְּיַיִן הוּרַק

מִכְּלִי אֶל כֶּלִי; / כֵּן בְּשׁוֹר הָאָמָה

קָט בְּמָרוֹם תַּמְרִיא – / עַל גְּבִירְתָהּ קָמָה.


מֵאֲנָה הִנָּחֵם / בַּת בַּחַיִּים קָצָה,

מִפְּנֵי צַר לוֹחֵם / אַחֲרֶיךָ רָצָה.

בַּחֲזוֹנָךְ נַחֵם / עַם בְּגָלוּת יָצָא –

עֵת פְּלָאִים תַּפְלִיא / בִי, וְתִהְיֶה חוֹמָה

לִי, וְתָאִיר נֵרִי / אָז כְּאוֹר הַחַמָּה.
