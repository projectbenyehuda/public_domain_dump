[יעלה יפת קול]

אהבה


         ’יַעְלָה יְפַת קוֹל, / מַה יָּפוּ דְבָרַיִךְ!

         עָלִית עֲלֵי כֹל / בְּחָכְמַת סְפָרַיִךְ!'

אֲנִי שְׁחוֹרָה, / הַשֶּׁמֶשׁ שְׁזָפַתְנִי,

כִּי חֹק וְתוֹרָה / עָבַרְתִּי, וְרָאַתְנִי

בַּת עוּץ וְאָמְרָה / מִלִּים בָּם הֲרָגַתְנִי:

         עָבַר קְצִירֵךְ, / יִתַּמּוּ בְצִירַיִךְ!

         נָפַל אֲמִירֵךְ, / יִמַּלּוּ קְצִירַיִךְ!

'בְּתוֹךְ מַשּׂוּאוֹת / כִּי נִפְרַדְתְּ מֵאִתִּי –

עִמָּךְ פְּלָאוֹת / אַעַשׂ לָךְ בְּבוֹא עִתִּי.

דֶּרֶךְ נְבוּאוֹת / צוֹפַיִךְ לְכִי, בִּתִּי,

         עַל מִשְׁעֲנוֹתָם / תַּעַמְדִי בִשְׁעָרַיִךְ,

         וּבְעִקְבוֹתָם / תִּרְעִי אֶת עֲדָרַיִךְ.'

רָפוּ זְרוֹעַי, / פִּי הֵילִיל – וְאֵין מַרְפֵּא.

רַבּוּ נְגָעַי, / כִּמְעַט קָט אֲנִי נִסְפֶּה.

יִרְאֶה צַלְעִי / צָר, יִלְעַג וְיַרְחִיב פֶּה:

         יָבוֹא אֲדוֹנֵךְ, / אָז יָבִיא לְצִירַיִךְ

         צֹרִי, וְיִקְנֵךְ / וִישַׁלַּח אֲסִירַיִךְ!

מָתַי יְסוּרוּן / מִבֵּיתָךְ בְּנֵי אֹמֶר?

צָרֵי יְשׁוּרוּן – / תַּשְׁקֵמוֹ בְכוֹס חָמֶר!

כָּבוֹד יְעִירוּן / וְשִׁירַיִךְ תְּמוּר תּוֹמֶר –

         ’שׁוּבִי, וְשַׁבְתִּי / אֲנִי לִבְנוֹת גְּדֵרַיִךְ,

         כִּי זָכַרְתִּי / לָךְ חֶסֶד נְעוּרַיִךְ!'
