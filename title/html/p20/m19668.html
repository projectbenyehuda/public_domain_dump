בְּרוּם גַּלְגַּל סוֹבֵב

רשות לקדיש


בְּרוּם גַּלְגַּל / סוֹבֵב עֲלֵי מַעְגָּל –

שְׁמָךְ יִגָּל, / נוֹרָא מְאֹד, נִדְגָּל!


רְאִיתִיךָ, / יוֹצְרִי, בְּעֵין לִבִּי,

קְרָאתִיךָ / רָחוֹק – וּבִמְסִבִּי

מְצָאתִיךָ, / הִנָּךְ בְּתוֹךְ קִרְבִּי,

לְךָ מִכְלַל / יֹפִי וְצִיץ מַהְלָל

בְּכָל מַעְלָל / יָשִׁישׁ וְגַם עוֹלָל.


כְּבוֹד עוֹלָם, / גָּדוֹל וּמִתְנַשֵּׂא,

וְאַתְּ נֶעְלָם / מִכֹּל וְכָל נוֹשֵׂא –

וְאֵיךְ אוּלָם / וּדְבִיר לְךָ מִכְסֶה?

לְךָ מִמְשָׁל / עַל כָּל גְּבִיר מָשָׁל,

וְלֹא נִמְשַׁל / הוֹדָךְ בְּכָל מָשָׁל!


יְהִי חֵקֶר / אֶל כָּל יְקָר נִמְצָא,

וְאֵין חוֹקֵר / כֵּס עֻזְּךָ יִמְצָא,

וְכָל בֹּקֶר / מִשְׁפָּט לְאוֹר יָצָא,

וְלֹא יֶחְדַּל / לָעַד וְלֹא יִדָּל,

אֲבָל יִגְדַּל / שִׁמְךָ וְיִתְגַּדָּל!
