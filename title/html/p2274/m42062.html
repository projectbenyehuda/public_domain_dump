עִזּוּז וְגִבּוֹר

סימן: עבדיה. לחן: אליך קוני עיני.


עִזּוּז וְגִבּוֹר / יִשְׁבֹּר / יַד צָר בּוֹ גָבְרָה / חַי

וּלְכָל הַנּוֹפְלִים / עוֹלִים / מִבּוֹר לְאוֹרָה / חַי

וְצַר מִתְגָּאֶה / יִרְאֶה / יַד אֵל לֹא קָצְרָה / חַי

רַב לְהוֹשִׁיעַ

יֶשַׁע יַבִּיעַ

אֹמֶר יַשְׁמִיעַ / יָרִיעַ / רָנִּי עֲקָרָה.


בְּחַסְדּוֹ יִפְנֶה / אוֹדֶה / כִּי אֵין בִּלְעָדוֹ / חַי

הוּא מֶלֶךְ עוֹזֵר / אִזֵּר / יִשְׂרָאֵל עַבְדּוֹ / חַי

כָּל מַלְכֵי אִיִּים / גּוֹיִים / כְּאַיִן נֶגְדּוֹ / חַי

לְרָמִים מַשְׁפִּיל

לָאָרֶץ יַפִּיל

שִׁמְשָׁם מַאֲפִיל / וּמַכְפִּיל / סוּפָה וּסְעָרָה.


דִּבְרֵי צַדִּיקִים / יָקִים / נַפְשִׁי שׁוֹאֶלֶת / חַי

וּבְלִי מַחֲלָה / סֶלָה / אַמְשִׁיךְ תּוֹחֶלֶת / חַי

כִּי יֵשׁ לִי מַצִּיל / מִכְּסִיל / יִפְרֹשׂ אִוֶּלֶת / חַי

שָׂמַנִי מִרְמָס

הָיִיתִי לָמַס

וְעַל לֹא חָמָס / לִבִּי נָמָס / אַפּוֹ בִי יָרָה.


יֵצֶר לִי אוֹיְבִי / וּבִי / שׁוֹאֲפִי חֵרֵף / חַי

הַכְרֵת אֶת זַרְעוֹ / גִּזְעוֹ / כִּי הִקְשָׁה עֹרֶף / חַי

כַּצֵּל כִּנְטוֹתוֹ / מוֹתוֹ / טוֹרֵף טֶרֶף / חַי

תִּרְמֹס כַּכְּפִיר

עֲצָתוֹ תָפִיר

יַבְאִישׁ וְיַחְפִּיר / הַצָּפִיר / עוֹשֶׂה בְעֶבְרָה.


הַבֵּט בָּךְ נִסְמָךְ / עַמָּךְ / מֵאַלְפֵי זָהָב / חַי

מַהֵר שְׁלַח לוֹ / שִׁילוֹ / אֲשֶׁר יֶאֱהַב / חַי

וְיָעִיר כֹּחוֹ / רוּחוֹ / וּמָחַץ רַהַב / חַי

יִשְׁפֹּט בְּצֶדֶק

טוֹבָם כְּחֵדֶק

מַחְזִיק כָּל בֶּדֶק / וּמְהַדֵּק / מַרְבֶה הַמִּשְׂרָה.
