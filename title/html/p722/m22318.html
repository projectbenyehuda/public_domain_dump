בְּחִירֵי אֵל רְאוּ

לשבת


בְּחִירֵי אֵל רְאוּ כִּי יְיֹיָ / נָתַן לָכֶם הַשַּׁבָּת

עַל כֵּן שִׁקְדוּ נֶאֱמָנַי / לַעֲשׂוֹת אֶת יוֹם הַשַּׁבָּת:


שַׁבַּת אֵל לִמְאֹד יָקְרָה / וְהִיא הָאוֹת עַל גְּדֻלָּתוֹ

מִימֵי בְרֵאשִׁית הִיא סְדוּרָה / עֲדֵי הוֹרִישָׁהּ לְנַחֲלָתוֹ

וְלִיצִיאַת נוֹף בָּהּ זְכִירָה / וּבָהּ שַׁבָּת מִפְּעֻלָּתוֹ

וּבַעֲבוּרָהּ צִוָּה בְסִינַי / זָכוֹר אֶת יוֹם הַשַּׁבָּת:


נַחֲלָתִי עָלַי שָׁפְרָה / בְּתֵת אֵל שַׁבָּת לִי לְמָנָה

כִּי סְעוֹדָתִי בָּהּ הֲדוּרָה / וְכִפַּת עַרְשִׂי רַעֲנָנָה

וּצְנִיפִי הוֹד וְתִפְאָרָה / וּכְסוּתִי נְקִיָּה לְבָנָה

זֹאת נַחֲלַת עַבְדֵי יְיֹיָ / וְצִדְקַת כָּל שׁוֹמֵר שַׁבָּת:


אִם רַגְלִי לֹא תִשְׁקוֹט בְּכָל / שֵׁשֶׁת יָמִים מִנּוֹעַ

בְּשַׁבָּת לְהִנָּפֵשׁ יָכוֹל / אוּכַל, וְאֶמְצָא מַרְגּוֹעַ

בָּעֶרֶב בָּשָׂר לֶאֱכוֹל / וְלֶחֶם בַּבֹּקֶר לִשְׂבּוֹעַ

וְלֹא אֶחְסַר כֹּל כִי יְיֹיָ / בֵּרַךְ אֶת יוֹם הַשַּׁבָּת:


הַשּׁוֹמְרִים אֶת שַׁבְּתוֹתַי / וּמְעַנְּגִים אֶת יוֹם קָדְשִׁי

אַרְבִּיצֵם עַל בָּמוֹתַי / וְאַזְרִיחַ לָהֶם שִׁמְשִׁי

וַאֲשִׁיבֵם אֶל מִשְׁכְּנוֹתַי / וְיַעְנוּ לְוִיַּי בְּמִקְדָּשִׁי

בְּקוֹל עָרֵב עַל דּוּכָנָי / מִזְמוֹר שִׁיר לְיוֹם הַשַּׁבָּת:
