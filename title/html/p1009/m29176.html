מוּל בֵּית דְּבִירִי אֶצְעַק

מוּל בֵּית דְּבִירִי / אֶצְעַק לְשׁוּב מַלְכִּי וְצוּרִי


אַדִּיר וְנָאוֹר / הוּא נַעֲלֶה עַל כָּל גְּבוֹהִים

אָמַר יְהִי אוֹר / וַיֵּצְאוּ אוֹרוֹת נְגוֹהִים

קָדוֹשׁ וְטָהוֹר / יוֹסִיף וְיִפְדֶּה עַָם כְּמֵהִים

עוֹבְדִים לְהַגְרִי / תָּמִיד וְאֵין חוֹבֵשׁ לְשִׁבְרִי


בִּרְאוֹת עֲמָלִי / גָּבַר וְלֹא צָלְחוּ שְׁנוֹתַי

יָגוֹן בְּחֶבְלִי / עָלָה לְפִי רֹב מַעֲווֹתַי

סוֹמֵךְ לְנָפְלִי / רָחַק מְמַלֵּא מִשְׁאֲלוֹתַי

קֶטֶב מְרִירִי / גַּם לַעֲנָה וָרֹאשׁ בְּחֶדְרִי


נִחַר גְּרוֹנוֹ / בִּנְךָ וְאֵין מַקְשִׁיב לְשִׂיחוֹ

כָּל צַר לִימִינוֹ / יִלְעַג עֲדֵי הִבְאִישׁ לְרֵיחוֹ

קוֹרֵץ בְּעֵינוֹ / הִדְאִיב מְאֹד נַפְשׁוֹ וְרוּחוֹ

כִּלּוּ שְׁאֵרִי / אוֹיְבַי וּמִי יִרְפָּא מְזוֹרִי


רֵאשִׁית נְגָעָי / נִמְכַּר לְעֶבֶד עֲבָדִים

רַבּוּ פְצָעָי / חִנָּם וְיָדַי הֵם כְּבֵדִים

עַל רֹב פְּשָׁעָי / תָּמִיד בְּאָזְנַי קוֹל פְּחָדִים

סָמַר בְּשָׂרִי / מִדְרַךְ לְכָל רֶגֶל פְּאֵרִי


מַה לָּךְ עֲצוּרָה / בִּתִּי וְאַל תִּהְיִי זְנוּחָה

מַלְאַךְ בְּשׂוֹרָה / אֶשְׁלַח לְהוֹצִיאֵךְ בְּשִׂמְחָה

פֶּתַע וְאֶקְרָא / רַעְיַת צְבִי שִׁמְשֵׁךְ זְרוּחָה

עֵת בּוֹא דְבָרִי / הִתְנַעֲרִי קוּמִי וְאוֹרִי.
