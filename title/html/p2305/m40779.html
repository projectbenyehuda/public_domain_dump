[בְּשִׂפְתֵי רְנָנוֹת וְשִׁירֵי נְגִינוֹת]

Biniamin Coen Vitale

]נולד באָלֵסַנְדְּרִיָה בשנת 1651 ומת ברֶג֗וֹ בשנת 1739]


͝ – – ͝ – –

בְּשִׂפְתֵי רְנָנוֹת / וְשִׁירֵי נְגִינוֹת / נְהַלֵּל בְּקוֹל עֹז / לְשׁוֹכֵן שְׁחָקִים

נְעִימוֹת יְמִינוֹ / לְעוֹשֵׂי רְצוֹנוֹ / בְּרָכָה וְשָׁלוֹם / בְּכָל־עֵת מְרִיקִים

יְשָׁרִים בְּעוֹדָם / בְּמִשְׁכָּן כְּבוֹדָם / בְּלֵיל זֶה שְׂמֵחִים / בְּמָחוֹל מְשַׂחְקִים

מְאִירִים בְּנֵרוֹת / לְכָל־הַחֲצֵרוֹת / בְּתוֹךְ עִיר קְדוֹשָׁה / בְּמַרְאֵה בְרָקִים

יְעִידוּן יְגִידוּן / בְּפִיהֶם יְהוֹדוּן / לְאֵל רָם וְנִשָּׂא / אֲשֶׁר בּוֹ דְבֵקִים

נְדָבוֹת גְּשָׁמָיו / וְיִזְלוּ בְשָׂמָיו / וְלִשְׁאֹב בְּשָׂשׂוֹן / לְמֵימוֹת מְתוּקִים

כְּמוֹ־כֵן עֲבָדָיו / וְזֶרַע יְדִידָיו / בְּאֵימָה בְיִרְאָה / הֲלֹא מַחֲזִיקִים

הֲלִיכוֹת אֲבוֹתָם / דְּבָרִים בְּעִתָּם / בְּבֵית־אֶל בְּלֵילוֹת / וּמִמַּעֲמַקִּים

נְכוֹנִים לְסַפֵּר / וְנִכְתַּב בְּסֵפֶר / כְּאַיָּל בְּשַׁחַר / וְעַל־הָאֲפִיקִים

יְהִי שֵׁם יְיָ / וּבִקְהַל אֱמוּנַי / מְבֹרָךְ לְעוֹלָם / בְּכָל־הַפְּרָקִים

צְבָאוֹת יְחוֹנֵן / וְעַמּוֹ יְגוֹנֵן / וְיָחִישׁ מְבַשֵּׂר / לְקָרֵב רְחוֹקִים

וְיָבִיא לְצִיּוֹן / גְּאֻלָּה וּפִדְיוֹן / וְיִשְׁלַח מְשִׁיחוֹ / לְהַצִּיל עֲשׁוּקִים

וְיִבְנֶה דְבִירוֹ / וּמוֹשַׁב יְקָרוֹ / בְּשָׁלֵם לְסֻכּוֹ / מְהֵרָה לְהָקִים

תְּהִי־נָא רְוָחָה / וְאוֹרָה וְשִׂמְחָה / לְהַלֵּל בְּקוֹל עֹז / לְשׁוֹכֵן שְׁחָקִים.
