משל מד: נֵץ וְיוֹנִים

נוֹתֵן כָּבוֹד לְאִישׁ חֶרְמוֹ / קוֹנֶה אָדוֹן לְעַצְמוֹ

הַנֵּץ בֵּין הַיּוֹנִים שֹׁכֵן וְחוֹפֵף / וְאִם יִתְבּוֹדֵד הָאֶחָד עִמּוֹ יִתְעוֹפֵף / וּכְפֶתַע יֶהְדֳפֶנּוּ וְיִטְרְפֶנּוּ / הֲתָעִיף עֵינֶיךָ בוֹ וְאֵינֶנּוּ / וַיַּבְחִינוּ הַיּוֹנִים בֵּינֵיהֶם / כִּי מִתְאַנֶּה הוּא לָהֶם / וְיָעֲצוּ לִדְרֹשׁ אֶת שְׁלוֹמוֹ / וַאֲשֶׁר יִכְרְתוּ בְּרִית עִמּוֹ / לְהַמְלִיכוֹ עֲלֵיהֶם מֶלֶךְ / אִם בְּתֹם וְיֹשֶׁר עִמָּהֶם יֵלֵךְ / וַיֹּאמְרוּ לוֹ: קָצִין תִּהְיֶה לָנוּ / בֹּא אַתָּה וּמָלְכָה עָלֵינוּ / וְתֶחֱזַקְנָה יָדֵינוּ הָרָפוֹת / בַּאֲשֶׁר יִירְאוּ מִפֶָּנֶיךָ הָעוֹפוֹת / וַיִּתְנוּ לוֹ הוֹד הַמַּלְכוּת וְהַמִּשְׂרָה / וְאָרַח עִמָּהֶם לְחֶבְרָה / וּבְכָל יוֹם בְּחֵמָה שְׁפוּכָה / הָיְתָה עֲלֵיהֶם הַמְּלוּכָה / כִּי בָז לָהֶם וְלַחֲבוּרָתָם / וַיִּקְרַח וַיִּמְרֹט נֹצָתָם / וַיֹּאמֶר לָהֶם: אֵיךְ תַּעַמְדוּ בִמְקוֹם בּוֹאִי / אִם אֲדוֹנִים אָנִי אַיֵּה מוֹרָאִי / וַיִּטְרֹף בָּהֶם וַיֹּאכְלֵם בְּכָל קָצֶה / וְאֵין נֹדֵד כָּנָף וּמְצַפְצֵף וּפֶה פֹּצֶה / וְהַנִּשְׁאָרִים הֶרָה נָסוּ / בְּחַיֵּיהֶם קָצוּ וּמָאָסוּ / וְנָפְלוּ בְּמִשְׁפְּחוֹתָם מְהוּמוֹת / יוֹנֵי הַגֵּאָיוֹת כֻּלָּם הֹמוֹת / וַיֹּאמְרוּ: מַה זֹּאת עָשִׂינוּ / לָשׂוּם מֶלֶךְ עַז פָּנִים עָלֵינוּ / כִּי מַה שֶּׁעָשָׂה מִקֶּדֶם בַּסֵּתֶר / עֹשֶׂה בַגָּלוּי וּמִי יוֹתֵר / וְטוֹב לִחְיוֹתֵנוּ בִלְתּוֹ / כַאֲשֶׁר הִמְתַּקְנוּ סוֹד אִתּוֹ.

וְהַמָּשָׁל לְנֹתֵן כָּבוֹד לְאִישׁ חֶרְמוֹ / הוּא קֹנֶה אָדוֹן לְעַצְמוֹ / טוֹב אֲשֶׁר עֵינָיו מִמֶּנּוּ יִגְרָע / מִדַּבֵּר עִמּוֹ מִטּוֹב וְעַד רָע / כִּי זֶה צֹפֵן זְדוֹן לִבּוֹ / וְתָמִיד אָרְבּוֹ בְּקִרְבּוֹ / וְהָאִישׁ אֲשֶׁר עוֹבְדוֹ וּמְכַבְּדוֹ / זֶה יָגֵל וְיִשְׂמַח לְאֵידוֹ / וְאָמַר הֶחָכָם: כַּבֵּד אֶת אֲשֶׁר אָהַבְתָּ וְהוּא יֶאֱהָבֶךָ / וְהִשָּׁמֵר מִמִּי אֲשֶׁר יִשְׂנָא לְבָבֶךָ.
