משל קיב: אִישׁ סֶלַע וְעַכְבָּר

אֲשֶׁר נְכֹחוֹת לֹא יַשִּׂיגוּ / קוּרֵי עַכָּבִישׁ יֶאֱרֹגוּ

אִישׁ הָיָה מִתְפָּאֵר בְּעָרְמוֹתָיו / וּבִמְזִמּוֹתָיו וְתַחְבֻּלוֹתָיו / וְכִי בְלַהֲטוֹ יִהְיֶה מְעֻבָּר / סֶלַע גָּדוֹל אֲשֶׁר בַּמִּדְבָּר / וְיוֹלִיד וְיַפְלִיט נֶפֶשׁ חַיָּה / עַד חֹדֶשׁ בְּלֹא שִׂפְתֵי רְמִיָּה / וְאַנְשֵׁי הַמָּקוֹם אַחֲרָיו תָּעוּ / אֶל מְעֹנְנִים וְאֶל קֹסְמִים יִשְׁמָעוּ / וְהוּא עַכְבָּר גָּדוֹל שָׁמַר / וְלַמּוֹעֵד אֲשֶׁר אָמַר / שָׂם הָעַכְבָּר בְּנִקְרַת הַצּוּר / וַיְהִי שָׁם עַד בֹּא הָעָם עָצוּר / וְאֶת נְקִיקֵי הַסֶּלַע סָתָם / וַיֵּלֶךְ וַיַּזְעֵק אוֹתָם / לֵאמֹר: מַהֲרוּ בֹּאוּ אַחֲרַי / וּמִלֵּאתִי אֶת דְּבָרַי / גַּם אֲשֶׁר לֹא הֶאֱמִינוּ עִם מַאֲמִינִים יַחַד / נָסְעוּ לָלֶכֶת כְּאִישׁ אֶחָד / וַיֹּאמְרוּ בְשִׂפְתֵי עֲלִיזוֹת / מִי שָׁמַע כָּאֵלֶּה וּמִי רָאָה כָּזֹאת / הֲיוּחַל אֶרֶץ וְגַם יוֹלִיד הָר / כִּי זֶה לְאַמֵּת דְּבָרוֹ נִמְהָר / וְהָאִישׁ הַמִּתְהַלֵּל בְּמַעְבָּדוֹ / מַטֶּה אֶחָד לָקַח בְּיָדוֹ / אֲשֶׁר עַל הַסֶּלַע הֵרִים / וַיֹּאמַר: שִׁמְעוּ נָא הַמּוֹרִים / הֲמִן הַסֶּלַע הַזֶּה אוֹצִיא / נֶפֶשׁ חַיָּה וְלָכֶם אַמְצִיא / וַיַּךְ הַסֶּלַע בְּמַטֵּהוּ / לְהַפִּיל מִן הַמָּקוֹם הַהוּא / אֶת כָּל אֲשֶׁר שָׂם לִפְנֵי הַחוֹר / וַיֵּצֵא אַךְ עַכְבָּר שָׁחוֹר / וַיֹּאמַר: עַכְבָּר יֻלַּד הִנֵּה יָצָא / לֹא נָפַל מִכָּל דְּבָרַי אָרְצָה.

וְהַמָּשָׁל בְּשֹׁמְעֵי כָזָב וְהֹלְכִים בָּאֹפֶל / וְיַאֲמִינוּ דִּבְרֵי טָחֵי תָפֵל / וּפֹעֲלֵי מִטִּיט הַיָּוֵן / הָרֹה עָמָל וְיָלֹד אָוֶן / בָּטֹחַ עַל שָׁוְא וְעַל תֹּהוּ / כָּל הַמְחֻבָּר לוֹ יִהְיֶה כָמוֹהוּ / וְהַזּוֹנִים אַחֲרֵיהֶם מִדֶּרֶךְ הַשֵּׂכֶל וְהַנָּעִים / הֶבֶל הֵמָּה מַעֲשֵׂה תַעְתּוּעִים / בַּאֲשֶׁר נְכֹחוֹת לֹא יַשִּׂיגוּ / וְקוּרֵי עַכָּבִישׁ יֶאֱרֹגוּ / וְלֹא לְבֶגֶד יִהְיוּ קוּרֵיהֶם / וְלֹא יִתְכַּסּוּ בְמַעֲשֵׂיהֶם / וְחֹשְׁבִים לְהַטְעוֹת עַמִּים בְּתַחְבֻּלוֹתָם / בַּחֲלוֹמוֹתָם בְּשִׁקְרֵיהֶם וּבְפַחֲזוּתָם / וְכַמָּה סָכְלוּ / אֲשֶׁר בְּפַח סִכְלוּתָם כָּשְׁלוּ / כִּי הוֹעִיל לֹא יוֹעִילוּ / וְהַצֵּל לֹא יַצִּילוּ.
