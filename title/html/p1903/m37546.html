משל ד: תַּרְנְגֹל וְאֶבֶן יְקָרָה

כְּסִיל אֶת עֵינָיו מֵחָכְמָה יִגְרַע / מוֹאֵס בַּטּוֹב וּבוֹחֵר בְּרָע

תַּרְנְגוֹל עֲלֵי אַשְׁפָּה עָלָה / וְהַדֹּמֶן בְּרַגְלָיו פִּזֵּר וְגִלָּה / לִמְצֹא תוֹלָעִים לְמַאֲכָלוֹ / מִשָּׁם חָפַר אֹכֶל בְּרַגְלוֹ / וַיִּמְצָא יָשְׁפֵה אֶבֶן טוֹבָה / וּלְנַקֵּר עוֹד בָּאַשְׁפָּה לֹא אָבָה / וַיֹּאמַר: חָשַׁבְתִּי לִמְצֹא יִתְרוֹן טוֹב בַּעֲמָלִי / תּוֹלָע אוֹ זְבוּב אֲשֶׁר יִיטַב לִי / וְעַתָּה יָשְׁפֵה הִנֵּה רְאִיתִיךָ / מַה יִּתְרוֹן לִי כַאֲשֶׁר מְצָאתִיךָ / אָמְנָם אַתְּ נֶחְמֶדֶת לְמַרְאֶה / וְאִם יִמְצָאֵךְ עָשִׁיר גֵּאֶה / תִּהְיִי בְיָדוֹ נִקְפֶּצֶת / וּבְזָהָב טוֹב מְשֻׁבֶּצֶת / וְתָאִירִי עַל אוֹרֵךְ שִׁבְעָתַיִם / אַךְ עַתָּה בֵּין רַגְלַיִם תִּהְיִי רְמוּסָה וּדְרוּסָה / כִּי נַפְשִׁי עָלַיִךְ לֹא חָסָה / בַּאֲשֶׁר לֹא מָצָאתִי בָּךְ חֶפְצִי / טוֹב לִי תוֹלָע אוֹ חֵצִי / לְהָשִׁיב נַפְשִׁי הָרְעֵבָה / מִמְּצִיאַת אֶבֶן טוֹבָה.

וְהַמָּשָׁל הַרְבֵּה אֶחֱזֶה / בְּאִישׁ אֲשֶׁר מְכֻבָּד יִבְזֶה / וְעֵינָיו מִצַּדִּיק יִגְרָע / מוֹאֵס בַּטּוֹב וּבוֹחֵר בָּרָע / כָּכָה מִשְׁפָּטוֹ וְחֻקּוֹ / לֹא יְקָרְבֶנּוּ רַק בְּשָׁעַת דָּחֳקוֹ / וְאָמַר שְׁלֹמֹה בִּמְשָׁלוֹ / בָּז לְדָבָר יֵחָבֶל לוֹ.

וָאֶשָּׂא עָלָיו בְּקוֹלִי / לְקַלֲלֶנּוּ בִמְשָׁלִי:


וְלֶב אִישׁ יַהֲפֹךְ מֵרְצוֹת / חֲכַם לֵבָב בְּמוֹעֵצוֹת

כְּתַרְנְגֹל יַעֲזֹב יָשְׁפֵה / בְּתוֹךְ אַשְׁפָּה בְּטִיט חוּצוֹת

בְּלִבּוֹ יִפְרַח נֶגַע / וְיָשׁוּב אַחֲרֵי הִקְצוֹת

וְהַנִּכְבָּד יְהִי נִקְלֶה / וּמָלֵא בוּז וְנֶאָצוֹת

וּבוּז לַבִּין לְהִתְמַשֵּׁל / וְהַצּוֹאִים לְמַחְלָצוֹת

וְאַבְנֵי גִיר מְנֻפָּצוֹת / בְּתוֹךְ זָהָב מְשֻׁבָּצוֹת.
