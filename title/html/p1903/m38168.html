משל סה: אַרְיֵה וַחֲמוֹר

אִישׁ אֲשֶׁר מַכִּיר מְקוֹמוֹ / תִּגָּלֶה עֶרְוָתוֹ וְלִבּוֹ בַּל עִמּוֹ


אַרְיֵה קָרָא לַחֲמוֹר לְהַרְאוֹתוֹ / אֶת יְקָר תִּפְאֶרֶת גְּדֻלָּתוֹ / וַיִּתְגָּאֶה הַחֲמוֹר בְּלִבּוֹ לֵאמֹר: מִי כָמוֹנִי / לְמִי יַחְפֹּץ הַמֶּלֶךְ לַעֲשׂוֹת יְקָר יוֹתֵר מִמֶּנִּי / בַּאֲשֶׁר אֲנִי לַחֲבוּרָתוֹ הוֹלֵךְ / וְעַתָּה יִקְרְאוּ לִי מִשְׁנֶה לַמֶּלֶךְ / כִּי שָׂמָנִי אֶל מִשְׁמַעְתּוֹ / אֹתִי בָחַר אֵין זָר אִתּוֹ / וַיֹּאמֶר לָאַרְיֵה: הִנֵּה עַל הַבְּהֵמוֹת אֶתֵּן חִתִּיתִי / מִקּוֹלִי יֵחָפְזוּן וְיָנוּסוּ מִגַּעֲרָתִי / וַיַּעַן הָאֲרִי: אַתָּה תַשְׂגִּיא פָעֳלֶךָ / לְהַשְׁמִיעַ בַּמָּרוֹם קוֹלֶךָ / וְנֵצֵא לִשְׁלֹל שָׁלָל רָב / וּפָנֶיךָ הֹלְכִים בַּקְְּרָב / אַתָּה תִשְׁאַג וַאֲנִי אֶטְרֹף / תֹּאחֵז טֶרֶף וַאֲנִי אֶעֱרֹף / וַיֵּצְאוּ הַשָּׂדֶה וַיִּמְצְאוּ צֹאן וּבָקָר זְכָרִים / וַיָּשִׂימוּ מְגַמַּת פְּנֵיהֶם לִטְרֹף וּלְהַחֲרִים / וַיֹּאמֶר הָאֲרִי: הֲמוֹן קוֹלְךָ הַשְׁמִיעֵם / וּבְקוֹל כָּמוֹנִי תַּרְעֵם / וְהִנֵּה בְּקוֹלוֹ הַחֲמוֹר נֹעֵר / וְהַקּוֹל בְּמַחֲנֵיהֶם הֹלֵךְ וְסֹעֵר / וַתְּפוּצֶינָה בְקוֹלוֹ הָרְחֵלִים / מִשֵּׂתוֹ יָגוּרוּ אֵילִים / וַיְהִי לְחֶרְדַּת הַמַּחֲנֶה אֲשֶׁר הֹלֵךְ עַל אַרְבַּע / וּבְהַפְנוֹתָם לָנוּס וַיִּטְרֹף בָּם הָאֲרִי וַיֹּאכַל וַיִּשְׂבַּע / וַיָּנוּסוּ אָנֶה וְאָנָה לְקוֹל הֲמֻלּוֹת גְּדוֹלוֹת / וַיֹּאמֶר הָאֲרִי: אַל תַּשְׁמִיעַ עוֹד קוֹלְךָ וְרַב מִהְיֹת קוֹלוֹת / וַיְהִי כַאֲשֶׁר לִטְרֹף כִּלָּה / וְהַחֲמוֹר וְהָאַרְיֵה עֹמְדִים אֵצֶל הַנְּבֵלָה / וַיֹּאמֶר הַחֲמוֹר: הֲלֹא שָׁמַעְתָּ הוֹד נַחֲרִי / וְרָאִיתָ כִּי רַעְמָה לָבוּשׁ צַוָּארִי / בְּחָכְמָתְךָ אֹתִי תִבְחַר / וְהָיָה כָזֶה יוֹם מָחָר / וְאַפְחִיד הַבְּהֵמוֹת וְאוֹרִיד לָאָרֶץ נִצְחָם / וְלַמֶּלֶךְ אֵין שֹׁוֶה לְהַנִּיחָם / לֹא מֵהֶם וְלֹא מֶהֱמֵהֶם / וְאֵלֶּה הַמִּשְׁפָּטִים אֲשֶׁר

תָּשִׂים לִפְנֵיהֶם / מִגַּעֲרָתִי יְנוּסוּן וְיֵחַתּוּ מִמֶּנּוּ / נַגְבִּיר שְׂפָתֵנוּ אִתָּנוּ מִי אָדוֹן לָנוּ / שְׁמָעֵנִי אֲדֹנִי הַמֶּלֶךְ / הֲלוֹא דְבָרַי יֵיטִיבוּ עִם הַיָּשָׁר הֹלֵךְ / וַיַּעַן לוֹ הָאֲרִי: הַמַּכִּיר תּוֹלַדְתְּךָ מִמַּעַל וּמִתָּחַת / מִקּוֹלְךָ לֹא יַעֲנֶה וּמֵהֲמוֹנְךָ לֹא יֵחָת / בָּזוּי וְשָׁפָל וּקְשֵׁה עֹרֶף / הֲתָצוּד לַלָּבִיא טֶרֶף / עוֹד יֹאמְרוּ בְאָזְנַי בְּנֵי שִׁכֻּלַי לֵאמֹר / אֵיךְ טָרַפְתָּ בַּצֹּאן וְלֹא שִׁבַּרְתָּ אֶת הַחֲמוֹר.


וְהַמָּשָׁל בְּאִישׁ נִכְבָּד אֲשֶׁר יְקָרֵב בָּזוּי וְנִקְלֶה / וְרוּחַ גְּאוֹנוֹ תִּצְמַח וְתִגָּלֶה / לְהִתְגָּאוֹת עַל חֶבְרָתוֹ / וְלֹא יַכִּיר אֶת שִׁפְלוּתוֹ / וּזְדוֹן לִבּוֹ בְהִתְגַּלּוֹת / לָשׁוֹן מְדַבֶּרֶת גְּדוֹלוֹת / בְּחָכְמָתִי כִּי נְבוּנוֹתִי / כָּזֹאת וְכָזֹאת עָשִׂיתִי / וְאָמַר הֶחָכָם בְּחָכְמָתוֹ / שְׁתִיקַת הַכְּסִיל הִיא תְשׁוּבָתוֹ.
