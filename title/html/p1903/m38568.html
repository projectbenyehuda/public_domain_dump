משל קט: סַרְטָן צְפַרְדְּעִים וְעָגוּר

מֹשֵׁל מַקְשִׁיב עַל דְּבַר שֶׁקֶר / הֵיטֵב שֹׁאֵל וְדֹרֵשׁ וְחֹקֵר

סַרְטָן גָּדֵל בַּמַּיִם / וּמִיָּדוֹ לוֹ קַרְנַיִם / כֻּלּוֹ הָפַךְ שָׁחוֹר / וַיֵּלֶךְ לֹא לְפָנִים כִּי אִם לְאָחוֹר / וְהַצְּפַרְדֵּעַ עוֹמֶדֶת בְּצִדּוֹ / וּמְקַרְקֶרֶת בְּאָזְנוֹ לְפַחֲדוֹ / וַיֹּאמַר: מָה הַקּוֹל הַזֶּה בְאָזְנַי / הוֹי עַל כָּל שְׁכֵנָי / מִקּוֹל הָמוֹן שׁוֹדֵד / שָׁמַעְתִּי וָאֶתְנוֹדֵד / לָלֶכֶת אֶל הַדָּג לְהִמָּלֵט / פֶּן יִהְיֶה עָלֵינוּ שֹׁלֵט / וַיֹּאמֶר לוֹ: אַחַר עֲצַת הֶעָגוּר / שְׁאֵרִית חֵמוֹת תַּחְגּוֹר / פִּיו צַר וָחַד / וְאֹכֵל הַצְּפַרְדְּעִים יַחַד / צַוָּארוֹ אָרֹךְ עַד מְאֹד / לְפָנָיו מִי יוּכַל לַעֲמֹד / וַיָּבֹא אֶל הֶעָגוּר וַיֹּאמַר / מִפַּחְדָּתִי שַׂעֲרִי תִסְמַר / מִקּוֹל שַׁאֲגַת הַצְּפַרְדֵּעַ / וַאֲנִי עֵד הַיּוֹדֵעַ / בִּי אֲדֹנִי שְׁמָעֵנִי / חָנֵּנִי כִּי אֻמְלָל אָנִי / וּמַלְּטֵנִי מִן הַתַּרְעֵלוֹת / וְרַב מִהְיוֹת קוֹלוֹת / הַצְּפַרְדְּעִים הַמְקַרְקְרִים וּפֹעֲרִים / בְּפִיהֶם וּבְאָזְנַי סֹעֲרִים / וַיַּעַן הֶעָגוּר / הֵן אֲנִי גֹר אָגוּר / בְּסֵתֶר קָנֶה וּבִצָּה / לְהַכְרִית מִשָּׁם כָּל שֶׁרֶץ וְשִׁמְצָה / וּמֵאַיִן בָּאתָ עֲרִירִי / הַגִּידָה לִי מַה שְּׂכָרִי / וְתֵרְדְּ עַל יְדֵי שֹׁחַד / פֶּן אֲבַלְּעֶךָ בְּפַעַם אֶחָד / וַיֹּאמֶר הַסַּרְטָן / אֲנִי קָשֶׁה אֲנִי קָטָן / אֲנִי מִן הָאֵבָרִים מְחֻסָּר / מִן הַחֵלֶב וּמִן הַבָּשָׂר / מַה תִּמְצָא בִי מֶגֶד / בְּיוֹם עֲמָדְךָ מִנֶּגֶד / בִּין תָּבִין אֶת אֲשֶׁר לְפָנֶיךָ / וְשִׂים אֶל הַצְּפַרְדְּעִים פָּנֶיךָ / וְשַׂמְתָּ שַׂכִּין בְּלֹעֶךָ / וְאָכַלְתָּ מִשָּׁם כְּנַפְשְׁךָ שָׂבְעֶךָ / וְהֶעָגוּר הֵשִׁיב בְּלָשׁוֹן רַכָּה / אֵין חָכָם וְנָבוֹן כָּמוֹכָה / אַתָּה תִּהְיֶה עַל בֵּיתִי / כָּל יְמֵי חַיַּי עַד יוֹם מוֹתִי / וַיִּכְרְתוּ שְׁנֵיהֶם בְּרִית / עַל הַצְּפַרְדְּעִים לְהַכְרִית / אוֹתָם מִן הָאֲגַמִּים / וּמִן הַנְּהָרוֹת וּמִן הַיַּמִּים / וְהַסַּרְטָן וְהֶעָגוּר נָחוּ בְלִי פָחַד / כִּי לֹא נִשְׁאַר בָּהֶם אֶחָד / כְּשֶׁבֶת יַחַד אַחִים וְרֵעִים / וּמֵאֹיְבֵיהֶם הַצְּפַרְדְּעִים / אֲשֶׁר הָיוּ מִקֶּדֶם מְקַרְקְרִים / אֵין אֹמֶר וְאֵין דְּבָרִים.


וְהַמָּשָׁל בְּאִישׁ מָלֵא חָכְמָה / בְּלִ תֹךְ אָוֶן וּמִרְמָה / אִם יַעֲלִילוּ עָלָיו עֲלִילוֹת / יִבָּטֵל עִם הוֹלְכֵי אֳרָחוֹת עֲקַלְקַלּוֹת / וְכַאֲשֶׁר מֹשֵׁל מַקְשִׁיב עַל דְּבַר שֶׁקֶר / הֵטֵיב שֹׁאֵל וְדֹרֵשׁ וְחֹקֵר / לְבַטֵּל פֻּרְעָנוּת בַּחוּץ / יִהְיֶה זָרִיז וְנָחוּץ / פֶּן יַגְדִּילוּ עָלָיו עָקֵב / מִן הַגֹּרֶן וּמִן הַיָּקֶב / וּמֵאַנְשֵׁי בֵיתוֹ / וּמִמִּשְׁפַּחְתּוֹ וּמִמּוֹלַדְתּוֹ / וְטוֹב יַעֲשֶׂה בְּמַעֲשָׂיו וּבִשְׂפָתָיו / כַּאֲשֶׁר עָשָׂה עִם אֲבוֹתָיו / וְשִׁלֵּם לְרֵעֵהוּ מַשְׂכֻּרְתּוֹ / בַּיּוֹם הַהוּא וּלְמָחֳרָתוֹ / כַּאֲשֶׁר גָּמַל הֶעָגוּר לַסַּרְטָן / וְשֵׁם אָחִיו יָקְטָן.
