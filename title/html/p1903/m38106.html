משל מה: שָׁלָךְ וְעוֹפוֹת

אִישׁ מָלֵא חָמָס / בְּאֶרֶץ מְגוּרָיו הוּא לְמִרְמָס

הַשָּׁלָךְ בֵּין הָעוֹפוֹת / הָיָה לָבַז וְלַחֲרָפוֹת / כִּי הָיָה לוֹ חֶרְפַּת אָחוֹר / מַשְׁלִיךְ צֹאָתוֹ מִן הַחוֹר / רֶגַע אֶחָד אֶלֶף פְּעָמִים / מַבְאִישׁ רֵיחוֹ בְּכָל פְּעָמִים / וְכַאֲשֶׁר הָלַךְ עִמָּהֶם / וַיְטַמֵּא אֶת מַחֲנֵיהֶם / וַיְמָרֲרוּהוּ וַיִּשְׂטְמוּהוּ / וּמִתֵּבֵל יְנִידוּהוּ / לֵאמֹר לוֹ: מִחוּץ לַמַּחֲנֶה יְהִי חֲנוֹתֶךָ / וְשַׁבְתָּ וְכִסִּיתָ אֶת צֵאָתֶךָ / בַּאֲשֶׁר לְךָ מַכַּת טְחוֹרִים / וְחֹסֶמֶת הִיא אֶת הָעֹבְרִים / וּמִשְׁפַּט מָוֶת לְךָ נֶחֱזֶה / וַיַּעֲמֹד עַל כַּנּוֹ נִבְזֶה / וּבִהְיוֹתוֹ לְלַעַג וּשְׂחֹק / וַיַּרְא אֶת הַמָּקוֹם מֵרָחֹק / עַל רֹאשׁ גִּבְעָה נִשָּׂאָה / שָׁם יָשַׁב בְּנֶפֶשׁ נְכֵאָה / גַּם שָׁם לֹא מָצָא מָנוֹחַ / כִּי בַעֲלֵי כָנָף שָׁלְחוּ לוֹ שָׁלוֹחַ / וַיֹּאמַר: כָּל עוֹף בְּאֶרֶץ מְגוּרַי אִיַּם / אָעוּפָה וְאֶשְׁכֹּנָה בְאַחֲרִית יָם / וְאַרְחִיק נְדֹד בְּאֶרֶץ מֶרְחַקִּים / וְלֹא אֶשְׁמַע עוֹד קוֹל לֹחֲצִים וְדֹחֲקִים / וְאִכָּבְדָה שָׁם תַּחַת הֱיוֹתִי נִבְזֶה וְנִקְלֶה / וְאֵין אֶחָד מִתְחַזֵּק עִמִּי עַל אֵלֶּה / וַיָּעָף וְאֶל הַיָּם הֹלֵךְ וְקָרֵב / וַיִּפְגַּע בּוֹ הָעֹרֵב לְעֵת עֶרֶב / וַיֹּאמֶר לוֹ: אָנָה תֵלֵךְ הַשָּׁלָךְ / הֲלֹא תְבַקֵּשׁ מָנוֹחַ אֲשֶׁר יִיטַב לָךְ / וַיַּעַן: מָנוֹחַ לִי אֶחֱזֶה / מֵעֵבֶר לַיָּם הַזֶּה / כִּי רֵעֶיךָ גֵּרְשׁוּנִי מִזְּבוּלִי / וְטָמֵא טָמֵא קָרְאוּ לִי / וַיֹּאמֶר הָעֹרֵב: הֲתִשָׂא עִמְּךָ אֲחֹרֶךָ / הַמַּרְחִיקְךָ מֵאֶרֶץ מְגוּרֶיךָ / וַיַּעַן הַשָּׁלַךְ: אֶת אֲחֹרַי / לֹא אוּכַל לְהָנִיחַ אַחֲרַי / כִּי נָשָׂאתִי חֶרְפַּת נְעוּרַָי / וַיֹּאמֶר הָעֹרֵב: הִנֵּה הוּא לְחֶרְפָּה וּלְלַעַג לְךָ / בְּכָל מָקוֹם אֲשֶׁר תִּדְרֹךְ כַּף רַגְלְךָ / וְכַאֲשֶׁר בִּגְלָלוֹ נֶהְפְּכוּ לְךָ הֵנָּה אֹהֲבִים לְאֹיְבִים / גַּם שָׁם יַרְחִיקוּךָ כָּל עוֹבְרִים וְשָׁבִים / וְלֹא יוֹעִיל לְךָ רֹב נוֹצָה וּגְדָל אֵבֶר / כִּי בְאֶפֶס יָד תִּשָּׁבֵר.

וְהַמָּשָׁל בְּאִישׁ מָלֵא חָמָס / וּבְאֶרֶץ מְגוּרָיו יְהִי לְמִרְמָס / בַּאֲשֶׁר דָּרַךְ לְשׁוֹנוֹ קַשְׁתּוֹ שֶׁקֶר / הוֹלֵךְ רִָכִיל מְגַלֶּה סוֹד וְחוֹקֵר / מַה זֶּה עַל חֲבֵרוֹ יְדַבֵּר / וַהֲמוֹן שְׁקָרִים עִם הַדָּבָר יְחַבֵּר / וּבְעֵת אֲשֶׁר יִרְאֶה כִּי רֵעָיו עָרָיו / לָמָּה יַרְחִיק אֶת אֶרֶץ מְגוּרָיו / אֲשֶׁר לֹא עָשָׂה טוֹב בְּעַמָּיו / וַיִּשְׁלַח יָד בִּשְׁלוֹמָיו / כִּי שָׁם יִשְׂנָאוּהוּ כְּמִשְׁפַּט הָרִאשׁוֹן / מָוֶת וְחַיִּים בְּיַד הַלָּשׁוֹן / וַחֲכָמֵינוּ אָמְרוּ עַל הַמְסַפֵּר לְשׁוֹן הָרָע / זֹאת תִּהְיֶה תּוֹרַת הַמְּצֹרָע / וְאָמַר הַיִּשְׁמְעַאלִי: בְּטֶרֶם תּוֹצִיא דְּבָרְךָ חֲשֹׁב מַה סּוֹפוֹ / כִּי כַאֲשֶׁר יִסְבֹּל אִישׁ מִכְוַת אֵשׁ לְתַקָּנַת גּוּפוֹ / רָאוּי לוֹ לִטְרֹחַ בְּתַקָּנַת רִפְאוּת נַפְשׁוֹ / מִמִּקְרֵה פְתַיּוּת וְאַכְזְרִיּוּת וּלְהָסִיר מִלְּשׁוֹנוֹ מוֹקְשׁוֹ.
