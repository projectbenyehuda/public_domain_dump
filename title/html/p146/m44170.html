מֵהִרְהוּר לִבִּי

חתימה: ממשתא


מֵהִרְהוּר לִבִּי וְעֵינָי / לִבִּי רוֹתֵת מֵעָנְשׁוֹ

חָלִיתִי מֵרוֹב יְגוֹנָי / וְהִבְעִיר בִּי לִבִּי אִשּׁוֹ

בִּלְבַּל דַּעְתִּי וְרַעְיוֹנָי / וְחָזַר מִצְטַעֵר בְּחָלְשׁוֹ

בָּרְכִי נַפְשִׁי אֶת יְוֹיָ / וְכָל קְרָבַי אֶת שֵׁם קָדְשׁוֹ:


מֵרוֹב יִסּוּרִין שַׁחְתִּי / וְרִפָּא אֱלֹהַי מַכְאוֹבִי

וּמֵחֵטְא וְעָווֹן חָדַלְתִּי / וְהִרְהוּר עֵינַי וּלְבָבִי

לְבַקֵּשׁ רַחֲמִים עָמַדְתִּי / אוּלַי יְמֻשֵּׁנִי אָבִי

אָמַרְתִּי וְכָל אֲשֶׁר לָאִישׁ / יִתֵּן בְּעַד נַפְשׁוֹ:


שִׁוִּיתִי יְוֹיָ לְנֶגְדִּי / יוֹדֵעַ כָּל תַּעֲלוּמוֹת

וּכְאַיִן נֶגְדּוֹ חֶלְדִּי / וּמַעֲשַׂי בִּכְתָב רְשׁוּמוֹת

וְיִחוּד קוֹנִי מַחְמַדִּי / הַנִּקְדָּשׁ בְּשִׁבְעִים שֵׁמוֹת

וְהַחַי יָשִׂים אֶל לִבּוֹ / וְנָתַן פִּדְיוֹן נַפְשׁוֹ:


תַּאֲוַת נַפְשִׁי רְצוֹן קוֹנִי / יִגְעַר בְּאוֹיְבַי וְיִפְדֵּנִי

וּשְׂמֹאל מְנָאֵץ מְסִיתֵנִי / גּוֹרֵם חֶטְאִי וְרוֹב עֲווֹנִי

צוּרִי בְּחַסְדּוֹ יְקִימֵנִי / יִרְצֶה בְּיוֹם דִּין יְזַכֵּנִי

וּבְיוֹם תְּחָיָה יְחַיֵּינִי / חַסְדּוֹ כָּעֵץ יִפְרְחָה שָׁרְשׁוֹ:


אַחְרִית וְתִקְוָה לְעָם גּוֹלֶה / עוֹשִׂים תְּשׁוּבָה וּמִתְעַנִּים

יִשְׁתּוּ וְיִרְווּ בְּלִי דּוֹלֶה / נַפְשָׁם בְּתוֹךְ מַעְיַן גַּנִּים

זִכְרָם בְּיוֹם זֶה לְטוֹב עוֹלֶה / שָׁבִים בְּטַהְרָה כְּשׁוֹשַׁנִּים

מִזִּיו שְׁכִינָה יְקָר נֶהְנִים / מִדֵּי חֹדֶשׁ בְּחָדְשׁוֹ:
