מְגוּרֵי אוֹהֲבַי

אל שלמה


מְגוּרֵי אוֹהֲבַי נוֹתְרוּ שְׁאִיּוֹת / וְשָׁבוּ אַרְמְנוֹתֵיהֶם כְּצִיּוֹת,

וּמִשְׁלָח נוֹעֲדוּ לִבְנוֹת עֳפָרִים / וּמִרְמָס נִקְרְאוּ לִבְנֵי צְבִיּוֹת,

וְתַחְתָּם רָבְצוּ הַיּוֹם נְמֵרִים / וְתוֹכָם נַעֲרוּ גוֹרֵי אֲרָיוֹת,

וְגַנִּים קִנְּנוּ שָׁם סִיס וְעָגוּר / לְקוֹנֵן נִקְבְּצוּ דַיּוֹת וְאַיּוֹת.

אֲשׁוֹטֵט אַךְ עֲלֵי קִירוֹת נְטוּיוֹת / וְאֶסֹּב הַגְּדֵרוֹת הַדְּחוּיוֹת,

וְעַפְרוֹתָם אֲחוֹנֵן אַט וְאֶרְצֶה / אֲבָנִים מֵעֲרֵמוֹתָם לְחַיּוֹת,

וְאַזִּיל מִדְּמֵי עֵינַי נְחָלִים / אֲשֶׁר לֹא יַעֲבֹר חוֹבֵל בְּצִיּוֹת.

אֲדַבֵּר בָּם וְאֵין מַקְשִׁיב וּמֵשִׁיב / לְבַד תַּנּוֹת יְתַנּוּ תַאֲנִיּוֹת.

זְמָן יָרָה בְיָדוֹ הַחֲזָקָה / שְׁכוּנֵיהֶם וּבִִזְרוֹעוֹת נְטוּיוֹת,

וְעָלוּ כָל מְשׂוֹשֵׂינוּ בְרַגְלָם / וּבִידֵיהֶם נְפָשֵׁינוּ שְׁבוּיוֹת,

וְאֵיכָה נִחְיֶה בִלְתָּם וְהָיוּ / כְּרוּחוֹת וַאֲנַחְנוּ הַגְּוִיּוֹת.

הֲמֵרֵעַי אֲנִי בוֹכֶה וְאִם עַל / נְדֹד אַחִים וּמִפְקַד הָאֲחָיוֹת,

אֲשֶׁר יָזוּב בְּאֵשׁ נוּדָם כְּבֵדִי / וְהַחֵלֶב אֲשֶׁר עַל הַכְּלָיוֹת,

וְאִם אָנוּד עֲלֵי פֵרוּד שְׁלֹמֹה / אֲשֶׁר דּוֹדָיו כְּנֹפֶת צוּף לְפֵיוֹת,

אֲשֶׁר הָיָה מְנָתִי מִגְּבָרִים / הֱיוֹת כָּל הַחֲסָדִים לוֹ מְנָיוֹת.

חֲמַדְתִּיהָ בְּמַעְלָלָיו אֲשֶׁר הֵם / לְכִתְרֵי בַעֲלֵי חֶמְדָּה שְׂכִיּוֹת,

וְכִי נִתְחַבְּרוּ רֵעָיו בְּחָכְמוֹת / מְחֻבָּרוֹת בְּטוּחוֹתָיו לְחַיּוֹת.

בְּבֹר לִבּוֹ לְבַד קָנָה נְפָשׁוֹת / הֲכִי מַעְלוֹת תְּעוּדָה לוֹ קְנוּיוֹת,

וְנָחָם בֶּאֱמוּנָתוֹ עֲדֵי כִי / בְּלִי שֵׁבֶט לְחֶפְצוֹ הֵם נְחוּיוֹת.

אֲהָבָיו הֵם כְּנֵס נִרְאוּ וּמֵהֶם / מְאֹד רַבּוּ אֲשֶׁר לֹא הֵם רְאוּיוֹת.

חֲמָסִי עַל זְמָן הִבְהִיל לְבָבִי / וְיָדָיו תּוֹקְפוֹת עָלַי וְהוֹיוֹת,

בְּנוֹתָיו פָּרְשׂוּ לִי פַח נְדֹדִים – / וְאֵיךְ אַרְיֵה יְהִי טֶרֶף לְשֵׂיוֹת?

וּמִיּוֹם נֶעְצְרוּ עָבֵי כְתָבָיו / בְּלִבִּי נִשְּׂקוּ רִשְׁפֵי כְּוִיּוֹת,

וְלִרְאוֹתָם הֲכִי עֵינַי כְּעֵינֵי / עֲרוּגוֹת אֶל יְדֵי עָנָן תְּלוּיוֹת.
