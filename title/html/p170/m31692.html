לְאַט בַּלֵּב אֲשֶׁר רָגַז

אל אבו עמר [יוסף] אבן קמניאל, במות עליו אחיו שמואל


לְאַט בַּלֵּב אֲשֶׁר רָגַז וּפָחַד / לְמִקְרֵי הַזְּמָן הָרַע וְיַלְדּוֹ,

לְאַט עָלָיו, הֲכִי חָלוּ מְצוּקָיו / וְיָסְפוּ עַל מְגוּרוֹתָיו וּפַחְדּוֹ,

וְאֵיךְ לֹא מָשׁ בְּשָׁמְעוֹ כִּי שְׁמוּאֵל / בְּכוֹר מָוֶת מְשָׁכָהוּ לְאָבְדוֹ,

וְשָׂם אֵלָיו מְגַמָּתוֹ לְהַכְרִית / דְּמִי יָמָיו וְהִשְׁחִית מַחֲמַדּוֹ.

אֲמִיר בֹּשֶׂם בְּגַן מוּסָר וְהַשְׂכֵּל / כְּפָרָיו נָתְנוּ רֵיחַ וְנִרְדּוֹ,

קְטָפוּהוּ וְעוֹדֶנּוּ בְאִבּוֹ / וְלֹא בָשְׁלוּ תְּנוּבוֹתָיו וּמִגְדּוֹ.

שְׁאֵרוֹ הֶחֳלִי בִּלָּה וְעַצְמוֹ / וְאָכַל הַכְּאֵב בַּדָּיו וְגִלְדּוֹ,

בְּטֶרֶם יַעֲלֶה אֶל כֵּס אֲבוֹתָיו / וְיִירַשׁ אֶת־גְּדֻלָּתָם בְּחֶלְדּוֹ.

וְעַד לֹא אָסְרוּ עַיִשׁ לְרִכְבּוֹ / בְּיוֹם מֶרְכַּב אֱנוֹשׁ סוּסוֹ וּפִרְדּוֹ,

אֲלֵיהֶם קָרְאוּ אוֹתוֹ מְאוֹרִים / וְאִם יַעְקֹב לְהַשְׁכִּינוֹ לְצִדּוֹ.

בְּכוּ עָלָיו וְסִפְדוּהוּ כְמִסְפֵּד / אֲשֶׁר נָפַל בְּתוֹךְ בִּקְעַת מְגִדּוֹ,

אֲשֶׁר טֶרֶם הֱיוֹתוֹ מַעֲלוֹתָיו / יְדוּעוֹת מִלְּשׁוֹן חָזוּת יְעִידוֹ.

וְנוּדוּ לוֹ וְלֹא כִמְעֹט שְׁנוֹתָיו / אֲבָל כִּרְבוֹת תְּהִלּוֹתָיו וְחַסְדּוֹ.

וְיוֹסֵף דַּבְּרוּ נָא עַל לְבָבוֹ / הֲכִי נִשְׁאַר בְּלִי אָחִיו לְבַדּוֹ.

סְגוֹר לִבּוֹ זְמָן קָרַע, וְיָרָה / אֱשׁוּן עֵינוֹ בְּחֵץ בִּגְדּוֹ וְשֻׁדּוֹ,

וְדִמָּה כִּי יְהִי מַשְׁפִּיל גְּאוֹנוֹ, / וְאֵיךְ אָדוֹן יְהִי נִכְנָע לְעַבְדּוֹ?

עֲנוּ בוֹ לַעֲרֹךְ שִׂכְלוֹ לְפָנָיו / וּמוּסָרוֹ הֱיוֹת תָּמִיד לְנֶגְדּוֹ,

כְּמוֹ רָדָה דְבַשׁ חָכְמָה וְיָנַק / חֲלֵב הַשְׂכֵּל בְּפִי מַדָּע וְדַדּוֹ.

וּבִין כִּי הַזְּמָן כָּל חַי יְנַהֵל / לְפִי שַׁחַת כְּמוֹ אִכָּר לְצִמְדּוֹ.

שְׂפַת אָסוֹן יְהִי מאפל יקוננו(?) / וְרֻקּוֹ………. וְכַדּוֹ.

לְזֹאת יֵאוֹת לְהִתְאַמֵּץ בְּדֵעוֹ / וְלֹא יַחֵל לְבוּשׁ הוֹדוֹ וּמַדּוֹ.

וְהוּא הַהוֹד אֲשֶׁר…… / ………………עֲבָדוֹ.
