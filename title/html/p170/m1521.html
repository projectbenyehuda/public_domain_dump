[אוֹרִי יִבָּקַע]

סימן: משה


אוֹרִי יִבָּקַע / יוֹם אֱהִי לְמִגְדֹּל

צוּרִי, וְיִתָּקַע / בְּשׁוֹפָר גָּדוֹל: [ישע' כז, יא1]

מִגְדְּלֵי עֲוֹנַי / נִבְצְרוּ לָרֹב

וּבְמֵימֵי עֵינַי / אַחֲרִיבֵם חָרֹב

וְשֶׁמֶשׁ שְׂשׂוֹנַי / אֲשֶׁר עָרְבָה עָרֹב

תִּזְרַח כִּי קָרוֹב / יוֹם הַגָּדוֹל: [צפנ' א, יד]

שִׁבְטֵי יִשְׂרָאֵל / יְקַבֵּץ כְּאַחַד

וְעַל הַר אֲרִיאֵל / יַעַמְדוּ יַחַד

בְּבֹא הַגּוֹאֵל / שְׁלֵוִים מִפַּחַד

יַעֲמֹד מִיכָאֵל / הַשָּׂר הַגָּדוֹל: [דב' יב, א2]

הָשֵׁב צְבִיָּה / אֶל בֵּית עֹפֶר

וְגֶפֶן פּוֹרְיָּה / לְאֶשְׁכֹּל הַכֹּפֶר

וְתוֹצִיא שְׁבִיָּה / מֵעֵיפָה וְעֵפֶר

בִּזְרֹעַ נְטוּיָה / וּבְמֹרָא גָּדֹל: [דב' כו, ח]

  1. צריך להיות ישעיהו כז יג: “וְהָיָה בַּיּוֹם הַהוּא יִתָּקַע בְּשׁוֹפָר גָּדוֹל” (הערת פב"י).  ↩

  2. צריך להיות דניאל יב א: “וּבָעֵת הַהִיא יַעֲמֹד מִיכָאֵל הַשַּׂר הַגָּדוֹל” (הערת פב"י).  ↩
