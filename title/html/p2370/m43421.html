צוּרִי חָנֵּנִי

סימן: משה ישועה חזק. לחן: יא מחיל יאבני סרת גומני.


צוּרִי חָנֵּנִי / כִּי בָךְ בִּטְחוֹנִי / מַהֵר הַעְלֵנִי / לְעִיר שְׂשׂוֹנִי

מִשְׁתּוֹקֵק לִרְאוֹת / אֶת קֵץ פְּלָאוֹת / עוֹשֵׂה נִפְלָאוֹת / הָפֵק רְצוֹנִי

שְׁנָתִי תִדַּד / וְאֵשְׁבָה בָדָד / עַל כִּי עֲדֵי עַד / אֵחַר פִּדְיוֹנִי

הַגִּידָה לִי, דּוֹד, / עֲדֵי אָן אֶדֹּד / צִפִּיתִי וְעוֹד / לָךְ תִּכְלֶה עֵינִי

יָקַד לְבָבִי / בִּרְאוֹתִי אוֹיְבִי / רוֹדִי וְעוֹלְבִי / חִיל אֲחָזָנִי

שִׂבַּרְתִּי חַסְדָּךְ / תַּטֶּה עַל עַבְדָּךְ / כִּי הוּא דַל וָדָךְ / נִבְזֶה מֵעֹנִי

וְלָמָּה, דּוֹדִי / תְּחַל כְּבוֹדִי / מָגֵן בַּעֲדִי / שׁוּב חַלְּצֵנִי

עֲלֵי עֲשׂוֹרִי / וְגַם כִּנּוֹרִי / אֲהַלֵּל צוּרִי / רִנָּה בִלְשׁוֹנִי

הֲלֹא בָךְ חִשְׁקִי / שְׁמַע נַאֲקִי / מִידֵי כָל עוֹשְׁקִי / פְּדֵה הֲמוֹנִי

חֲמֹל עָלַי כִּי / אַתָּה הוּא מַלְכִּי / הַגִּיהַּ חָשְׁכִּי / שְׁעֵה הֶגְיוֹנִי

זָכַרְתִּי יְמֵי / נִרְוֵיתִי מִמֵּי / יִשְׁעָךְ וְעַל מֶה / דּוֹד, זְנַחְתָּנִי

קַוֹּה קִוִּיתִי לְדוֹד חֶמְדָּתִי / יִרְצֶה שִׁירָתִי / חֵלֶף קָרְבָּנִי.
