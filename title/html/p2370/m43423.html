אוֹדֶה לָאֵל

שיר זה חיברו על הנס שעשה לנו הקב״ה בליל ראשון של חנוכה של שנת התר״ן שבאו גנבים לשלול הבית ועקרו מזוזות השער וגררו את המנעולים וברוב רחמי האל נתעוררה אמי מב״ת והצילנו מידם. ובתוך שיר יסופר אופן המעשה.

סימן: א״ב. משה ישועה. לחן: אדון עולם מתהפך לכמה נגינות.


אוֹדֶה לָאֵל / פּוֹדֶה גוֹאֵל / שִׁיר לְמוּאֵל / עַל נֵס נוֹרָא

בְּבוֹא נַפְשִׁי / אֶלֶף שִׁשִּׁי / שְׁנַת שִׁמְשִׁ״י / לַיְּצִירָה

גַּם זוֹ קָרָה / בְּלֵיל קָרָה / לֵיל נֵס נִקְרָא / לֹא אֶשְׁמֹרָה

דַּע אֲהוּבִי / כִּי בְשָׁכְבִי / עַל מִשְׁכָּבִי / בְּצֵל קוֹרָה

הַיְּשׁוּעָה / לֵיל אַרְבָּעָה / בְּחֵי״ת שָׁעָה / רֹאשׁ מִשְׁמָרָה

וּבָא לְאַט / גַּנָּב בַּלָּאט / וּלְבֵיתִי נָט / לְהַעְדִירָה

זְדוֹן לִבּוֹ / וּכְלֵי חַרְבּוֹ / שָׂם עַל לִבּוֹ / גַּם מְגֵרָה

חָשַׁב עָלַי / לִשְׁלֹל כֵּלַי / עִם כָּל חֵילַי / בְּאַף עֶבְרָה

טִירַת הַדַּל / וּמְזוּזוֹת דַּל / הוּא נִשְׁתַּדַּל / וּפִזְרָה

יְדֵי גָמוּל / בִּכְלִי פָּעוּל / נֹכַח מַנְעוּל / שָׁבַר תִּקְרָה

כֹּחוֹ נִגְמַר / וְהִתְמַרְמַר / עָקַר מִסְמַר / וֶהֱסִירָהּ

לָבַשׁ אַכְזָר / עֹז הִתְאַזָּר / וְשָׁב חָזַר / לְשַׁבְּרָהּ

מְעַט מִזְעָר / הִשְׁאִיר בַּעַר / מִן הַשַּׁעַר / לַעֲקִירָה

נָט אֵל חַסְדּוֹ / עַל בֵּית עַבְדּוֹ / סָךְ בַּעֲדוֹ / וְהֱעִירָהּ

סָמְכָה יַד לִי / וְאָמְרָה לִי / שְׁמַע קוֹלִי / פִּיךָ סָגְרָה

עָנְתָה בֵיתִי / עוֹד לִקְרָאתִי / קוֹל לְדַלְתִי / צוֹעֵק מָרָה

פִּתְאֹם בָּאתִי / מְבוֹא בֵּיתִי / הֲרִימוֹתִי / קוֹל וָאֶקְרָא

צַר הַבִּיטָה / לִבְּךָ נָטָה / אֶל הַמִּטָּה / וּכְלֵי פַחְרָא

קָרָא גַּנָּב / סְלָעִים הַב / כִּי נִתְעַכַּב / וְעֵת עָבְרָה

רָאָה רָשָׁע / כִּי לֹא נוֹשָׁע / בָּרַח נִפְשָׁע / קַל מְהֵרָה

שִׁיר הוֹדָאוֹת / לָאֵל יָאוֹת / עַל נִפְלָאוֹת / אֲזַמְּרָה

תָּמִיד שִׁירָה / לוֹ אָעִירָה / בְּעֵת צָרָה / יִשְׁלַח עֶזְרָה

מֵאֵל שׁוֹאֵל / הוּא יְשַׁכְלֵל / חוֹמוֹת וָחֵיל / עִיר הַיְקָרָה.
