[יְשַׁלֵּם הַסְּתָיו נִדְרוֹ וְיָקִים]

וקאל איצ͘א


יְשַׁלֵּם הַסְּתָיו נִדְרוֹ וְיָקִים / דְּבָרוֹ לַחֲבַצֶּלֶת עֲמָקִים

וְיוֹם קַיִץ לְיוֹם חֹרֶף מְצַפֶּה / יְבוֹאוּהוּ לְבַשֵּׂר הַבְּרָקִים

אֲשֶׁר תֵּלֵד בְּלִי עֶצֶב אֲדָמָה / וְהֶעָבִים יְלָדֶיהָ מְנִיקִים

רְאֵה כִּי שָׂחֲקָה תֵבֵל וְרַכּוּ / לְחָיֶיהָ אֲשֶׁר הָיוּ חֲזָקִים

וְכָסּוּ הָעֲרֻגוֹת כָּֽתְנוֹת חוּר / וְשָׁנִי שִׁבְּצוּם עֲבֵי שְׁחָקִים

וּמַרְאֵיהֶן כְּמַרְאֵה הַכְּתָמִים / וְגַבֵּיהֶן כְּגַבֵּי הַיְֽרָקִים

וְקָם הַתּוֹר וְהֶעָגוּר וְהַסִּיס / לְהִתְפָּאֵר עַל יוֹנֵי אֲפִיקִים

וְעֵת כִּי יִתְּנוּ רֹאשָׁם לְקַפֵּץ / תְּדַמֶּה פַעֲמֵיהֶם בַּאֲזִקִּים

בְּהִגָּלוֹת הֲדַר שֶׁמֶשׁ עֲלֵיהֶם / יְכַסּוּ צַוְּרֹנֵיהֶם עֲנָקִים

אֱמֹר אַתָּה הֲתוּכַל אוֹר לְכַסּוֹת / וְאִם תּוּכַל לְהַשְׁפִּיל הָאֲרָקִים

הֲלֹא דַי כִּי תְשַׁוֶּה לִבְּךָ לִי / כְּמוֹ סֶלַע עֲדֵי שַׂמְתּוֹ נְקִיקִים

וְאֵיכָה תֹאֲנָה עָלַי תְּבַקֵּשׁ / וְדִבְרֵי תֹאֲנָה דַּקִּים וְרֵקִים

וְאֵיךְ תִּזְרֹק רְמָחֶיךָ לְנֶגְדִּי / וּמָגִנַּי כְּבָר עָרְצוּ אֲפִיקִים

וְסוּסֵינוּ יְרוּצוּן בַּסְּלָעִים / יְכַסּוּ הַשְּׁחָקִים בַּאֲבָקִים

וְאֵיךְ תִּבְזֶה מְעַט נִיבִי וְתַקְרִיב / עֲלֵי מִזְבַּח אֱלֹהֶיךָ נְתוּקִים

וְהַשָּׁנִים יְקוּמוּן מֵחֳדָשִׁים / וְהַיָּמִים יְקוּמוּן מֵחֲלָקִים

וְאִם אַתָּה בְּרֹאשׁ אַנְשֵׁי לְבָבוֹת / דְּלֵה מִלֵּב אֱנוֹשׁ מַיִם עֲמֻקִּים.
