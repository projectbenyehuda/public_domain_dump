צִיצַת חֲבַצֶּלֶת

צִיצַת חֲבַצֶּלֶת זְמִירוֹת תַּעַן / צִפְצְפָה מֵאֶמֶשׁ צִדְקְךָ וְלֹא דַמָּה

וְתִשְׁקֹל אֲמָרֶיהָ בְמֹאזְנֵי כְנַעַן / וְתֶהְמֶה בְשִׂיחָהּ כַּהֲמוֹת יָמָּה

מְחַץ מַצְמִיתֶיהָ כִּי טָעֲנוּ טַעַן / מֵאֲשֶׁר יֵאֲשׁוּהָ נַפְשָׁהּ נָדַמָּה

הִנֵּה נִגְזַרְתְּ יַעַן וּבְיַעַן / הוּמַר קִצֵּךְ וְתוֹחַלְתֵּךְ תַּמָּה

עִצְּבוּ לְבָבִי וְהִרְווּנִי לַעַן / עַד כִּי קִנְאָה בְלִבִּי הוּחַמָּה

שְׂמֵחוֹת לְאֵידִי קוֹלִי שְׁמַעַן / שִׂבְרִי אִם אָרַךְ יְיָ שָׁמָּה

וְעַד אָן אֶקְרָא וְלֹא אֶשְׁמְעָה מַעַן / וְלָמָּה תַּסְתִּיר פָּנֶיךָ וְלָמָּה

רְאֵה תְּבוּסַת גֶּפֶן הִסַּעְתָּ מִצֹּעַן / רַב זֶה כַּמֶּה שָׁנִים הָשַּׁמָּה

שֶׁתְּעוֹרֵר אַהֲבַת אֹהֶל בַּל יִצְעַן / שְׁקֹד צְפִירַת צֶדֶק בִּשְׁאוֹל הוּעַמָּה

לְבַשׁ מַעֲטֵה חֶסֶד לְעַם בְּךָ נִשְׁעָן / לָשֵׂאת עָווֹן וְחַטָּאָה הוּזַמָּה

מָרוֹם עֲשֵׂה לְמַעַנְךָ וּלְמַעַן / הֲגִיג יוֹנַת אֵלֶם בְּהַר מוֹר הוּרָמָה

הָעֵת קָרָא לְאָבִיו וַיַּעַן / טָבְחֵנִי נָא וְנַפְשׁוֹ לֹא נָשַׁמָּה

חֹן זַרְעוֹ בְקָרְאוֹ וּכְמַעֲשֵׂה אֶרֶץ כְּנַעַן / אֲשֶׁר אֲנִי מֵבִיא אֶתְכֶם שָׁמָּה.


שֶׁתֶּחְפָּץ לְקַבֵּץ מִקַּצְווֹת הָאִיִּים

בְּדוּדִים רְדוּדִים בְּזוּרִים יְרוּיִים

יְאֻסְּפוּ הֲמוֹנֵימוֹ וַהֲמוֹן דָּרֵי הָאִיִּים

לְהִתְהַלֵּךְ לִפְנֵי אֱלֹהִים בְּאוֹר הַחַיִּים.
