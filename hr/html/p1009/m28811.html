אֲשֶׁר נָטָה וְהִרְקִיעַ שְׁחָקִים

אֲשֶׁר נָטָה וְהִרְקִיעַ שְׁחָקִים / כְּסַפִּירִים רְאִי מוּצָק חֲזָקִים

וְגַלְגַּלִּים יְסוֹבֵבוּ בְּלֶכְתָּם / וְגֶשֶׁם עַל פְּנֵי תֵבֵל מְרִיקִים

וְאֶרֶץ עַל בְּלִימָה הִיא תְּלוּיָה / וְהֶעֱמִיד כָּל אֲפָסֶיהָ וְהֵקִים

וּבָחַר לוֹ בְּיִשְׂרָאֵל סְגֻלָּה / לְמַעַן כִּי בְּרִית כָּרַת לְהָקִים

אֲשֶׁר עָשָׂה בְּמִצְרַיִם פְּלָאוֹת / וְיִסַּד עֹז בְּפִי עוֹלְלִים וְיוֹנְקִים

וְהִנְחִיל דָּת בְּתוּשִׁיָּה בְּסִינַי / לְיִשְׂרָאֵל בְּקוֹלוֹת עִם בְּרָקִים

תְּמִימָה הִיא מְשִׁיבָה לַנְּפָשׁוֹת / בְּרֹב עֵדוֹת וּמִשְׁפָּטִים וְחֻקִּים

וְחָנַן בָּהּ יְחִידִים הַשְּׂרִידִים / פְּשָׁטֶיהָ וְסוֹדוֹת הָעֲמֻקִּים

גְּאוֹן הַדּוֹר סְעַדְיָה בֶּן יְהוֹסֵף / שְׁמוּעָתוֹ מְעִידָה בַּשְּׁוָקִּים

גְּבוּרָתוֹ בְּאַרְבָּעָה וְעֶשְׂרִים / וְדִקְדוּקָם בְּלִבּוֹ הֵם חֲקוּקִים

שְׁמוֹ נוֹדָע בְּטִיב מַשָּׂא וּמַתָּן / וּמֵבִין הוּא לְאַרְבָּעָה פְרָקִים

וְלוֹ מִשְׂרָה וּמוּסָר וַעֲנָוָה / וְתוֹכָחוֹת בְּפִיו כַּמָּה מְתוּקִים

וְעַל זֹאת מַלְאֲכֵי חֵן חוֹפְפוּהוּ / וְעָרְבוּ לוֹ לְשָׁמְרוֹ מִנְּזָקִים

לְדַעְתּוֹ מָאֲסָה נַפְשׁוֹ בְּרֶשַׁע / וּמִדּוֹת הָאֱמֶת אֶצְלוֹ חֲשׁוּקִים

וּמִדִּבְשׁוֹ יְקַבֵּצוּ מְשִׁיקִים / וּמִשִּׂכְלוֹ מְזָוֵינוּ מְפִיקִים

וְכָל הַשּׁוֹאֲבִים מֵימֵי מְקוֹרוֹ / חֲסִידִים הֵם וּמִכָּל חֵטְא מְנֻקִּים

וּבוֹ לָבַשׁ זְּמָן עֶדְיוֹ וְהֵאִיר / פְּנֵי תֵבֵל כְּשׁוֹשַׁנַּת עֲמָקִים

וְכַמָּה סָבְבוּ עַל חוּג אֳרָחָיו / וְנִלְאוּ מִלְּהַשִּׂיגוֹ עֲנָקִים

וְנִלְאֵיתִי לְהַשִּיג קֵץ שְׁבָחוֹ / וְכֹחוֹתַי וְרַעְיוֹנַי רְחוֹקִים

וְאִם לָקְחוּ מְיֻדָּעַָיו לְחֵלֶק / בְּאַהְבָתוֹ וְלִי תִשְׁעָה חֲלָקִים

וְנַפְשִׁי תַעֲרֹג לִדְבַשׁ לְשׁוֹנוֹ / כְּאַיָּל תַּעֲרֹג עַל הָאֲפִיקִים

כְּאִישׁוֹן יִנְצְרֵהוּ הָאֱלֹהִים / אֲשֶׁר יָרָה לְאֶבֶן הָאֲרָקִים

וְאוֹדֶה אֵל אֲשֶׁר הֵאִיר לְעַמּוֹ / בְּמוֹרֶה טוֹב לְהַצִּיל הָעֲשׁוּקִים

וְאֶתְחַנֵּן לְמוּל בֵּיתוֹ בְּכָל לֵב / בְּנֶפֶשׁ קוֹרְאָה מִמַּעֲמַקִּים

אֲקַוֶּה לַחֲזוֹת מִשְׁכַּן דְּבִירוֹ / אֲשֶׁר נָטָה וְהִרְקִיעַ שְׁחָקִים.
