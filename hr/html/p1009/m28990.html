לְאַדָּר אֶעֱנֶה בִשְׁבָח וְשִׁירִים

לְאַדָּר אֶעֱנֶה בִשְׁבָח וְשִׁירִים / לְמֵקִים דָּל וּמַשְׁפִּיל אַף וּמֵרִים

אֲשֶׁר עָמַד לְעַם קֹדֶשׁ בְּעֵת צָר / בְּקוּם הָמָן וּבָנָיו הָאֲרוּרִים

וְתָלוּהוּ בְעֵץ הֵכִין לְנַפְשׁוֹ / וְרֹב בָּנָיו אֲשֶׁר שָׁבוּ עֲרִירִים

וְהִשְׁבִּית מִשְׁכְּנוֹת זֶרַע עֲמָלֵק / וְהִחְלִיפָם בְּצַדִּיקִים יְקָרִים

וְצִוַּנִי לְהַזְכִּירָם וְאֶשְׂמַח / וְאָגִילָה לְעֵת כָּזֹאת בְּפוּרִים

וְאֶשְׁתַּעְשַׁע בְּפִלְאֵי הַפְּלָאוֹת / אֲשֶׁר נֶהְפַּךְ יְמֵי אֹפֶל לְאוֹרִים

שְׁתוּ דוֹדִים בְּיֵין מַלְכוּת וְשִׁכְרוּ / בְּנֵי יַעֲקֹב זְקֵנִים עִם נְעָרִים

עֲדֵי כִי נַחֲלִיף אָרוּר בְּבָרוּךְ / וּמִכָּל מַעֲנֶה נִהְיֶה פְטוּרִים

וּמִשְׁלוֹחַ יְפֵה מָנוֹת לְרֵעִים / וּמַתָּנוֹת לְאֶבְיוֹנִים וְגֵרִים

בְּשַׂר אֵלִים אֲשֶׁר נִכְסָה בְחֶלְבּוֹ / וְדָג גָּדוֹל יְהִי רוֹבֵץ יְאוֹרִים

וְסוֹף נוֹדֶה לְמַשְׂבִּיע רְעֵבִים / בְּגַאְוָתוֹ וְהִרְעִיב הַכְּפִירִים.
