שְׁאָלוּנִי מְתֵי שֵׂכֶל

שְׁאָלוּנִי מְתֵי שֵׂכֶל וּמוֹרִים / סְגֻלַּת אֵל מְעֻלִּים הַיְּקָרִים

בְּעַד הַקֵּץ אֲשֶׁר נֶעְלָם וְהֶאְרִיךְ/ וְסוֹדוֹתָיו וְקוֹרוֹתָיו סְתוּרִים

הֲשִׁיבוֹתִים שְׁנַת יוֹבֵ״ל לְסִימָן / קְבוּעָה הִיא לְמִחְיַת הַפְּגָרִים

וְלַמַּפְרֵעַ תְּהֶא מוֹנֶה לְגוֹאֵל / שְׁנוֹת מִדְבָּר יְדוּעִים הֵם סְפוּרִים

וְאִם תִּרְצֶה שְׁאַל נָא אֶת בְּנוֹת חֵת / וְתַעְזֹב הָאֲלָפִים בַּיְּעָרִים

וְצַדִּיק בֶּאֱמוּנָתוֹ יְהִי חַי / לְקֵץ יָמִין וְלִתְשׁוּעַת אֲסִירִים

וְיִסּוּרִין יְהִי סוֹבֵל בְּאַהְבָה / וְקַח מוּסָר לְנָכְרִים גַּם וְגֵרִים.

וְהָאֶחָד עֲפַר אֶרֶץ מְלַאכְתּוֹ / בְּכָל יָמָיו חֲדָשִׁים לַבְּקָרִים

וְהָאֶחָד מְשַׂדֵּד בָּעֲמָקִים / וְזוֹרֵעַ וְחוֹרֵשׁ בַּבְּקָרִים

בְּרֵאשִׁיתָם כְּשׁוּעָלִים חֲשׁוּבִים / וְגָבָהוּ עֲדֵי שָׁבוּ כְפִירִים

וְאַף כִּי יַעֲשֶׂה הָאֵל לְצֹאנוֹ / גְּבוּרָתוֹ תְּאַסֵּף הַפְּזוּרִים.
