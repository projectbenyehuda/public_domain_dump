[אהי כפר צבי]

∪ — — — ∪ — — — ∪ — — // ∪ — — — ∪ — — — ∪ — —

אֲהִי כֹפֶר צְבִי הֵפֵר בְּרִיתִי // וְאַהְבָתוֹ בְּתוֹךְ לִבִּי שְׁמוּרָה1

אֲשֶׁר אָמַר לְסַהַר בַּעֲלוֹתוֹ: // הֲתִרְאֶה אֶת מְאוֹר פָּנַי וְתֵרָא2?

וּמַרְאֵה הַלְּבָנָה בָאֲפֵלָה // כְּבָרֶקֶת בְּכַף עַלְמָה שְׁחֹרָה3.


  1. אהיה כפרת האהוב שהפר את ברית האהבה ובגד בי, ובכל זאת אהבתו שמורה בתוך לבי.  ↩

  2. ותרא־ ותראה, שכן פניו יפים מפני הסהר. השוה לעיל קס 2: כסתה לבנה את מאור כוכבים דודי עלה כסה מאור הלבנה.  ↩

  3. אף על פי שמראה הלבנה באפלת הלילה יפה מאד, כברקת בכף עלמה שחורה.  ↩
