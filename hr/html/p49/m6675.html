לְךָ עשֶׁר

וכאן אלחכים אבו מדין באיתא ענדה פאנבהה פאעלמה במא ראי ולם ישך פי תלף אבן אבי מוסי פקתל אלי איאם יסירה פקאל ענד מא [בש]ר בהלאכה וד ͘ לך שנת שמונה מאות וסמאהא בשורה

והיה הרופא אבו מדין לן אצלו והעירו (הנגיד) והודיעו מה שראה (בחלומו) ולא פקפק במותו של אבן אבי מוסי, ואכן נהרג אחרי ימים מועטים, ושר (הנגיד) כאשר נתבשר במותו — וזה היה בשנת שמונה מאות — וקראה:

בְּשׂוֹרָה


È — — — È — — — È — — — È — — —

לְךָ עשֶׁר / וּמַמְלָכָה / וְהָעֹז מִ- / לְּפָנֶיךָ,

וְאַתְּ מוֹשֵׁל / בְּכָל מוֹשֵׁל / וְנַפְשׁוֹתָם / בְּיָדֶיךָ!

אֲנִי אֶרְאֶה / וְאַגִּיד לַ- / אֲנָשִׁים מַלְ- / אֲכוֹתֶיךָ,

וְהָעִוֵּר – / אֲשֶׁר לֹא יֶ- / חֱזֶה הַדְרַת / כְּבוֹדֶךָ,

5 וְהָאִלֵּם – / אֲשֶׁר יִדֹּם / וְלֹא יַגִּיד / פְּעָלֶיךָ.

לְךָ הַהוֹד / וְהַכָּבוֹד / בְּאַדְמָתְךָ / וְשָׁמֶיךָ,

וְכַמָּה הֶ- / עֱלִיתָ אִישׁ – / וְהָיָה מִ- / מְּשַׂנְאֶיךָ,

וְכִבַּדְתּוֹ – / וְהוּא בִקֵּשׁ / לְהָרַע אֶל / יְדִידֶיךָ,

וְכִמְנַשֶּׁה / בְּךָ כִחֵשׁ / וְאָמַר: לֹא / רְאִיתִיךָ!

10 מְבַשֵּׂר טוֹב, / הֲבִינֵנִי / וְהָרֵם קוֹל, / בְּחַסְדֶּךָ,

וְצַפְצֵף לִי / כְּסוּס עָגוּר, / וְאַל תַּמְעִיט / אֲמָרֶיךָ:

הֲצֶדֶק הוּא / אֲשֶׁר אֶשְׁמַע, / וְאֵיךְ הָיָה, / וְאֵיכָכָה?

הֲבִינֵנִי: / הֲגִוְיַת בֶּן- / אֲבִי-מוּסַי / מְחֻתָּכָה?

וְנִבְלָתוֹ – / הֲנִסְחָבָה / עֲלֵי חוּצוֹת / וְהֻשְׁלָכָה?

15 הֱשִׁיבַנִי: / חֲדֵה וּשְׂמַח / וְיֵעָלֵץ / לְבָבֶךָ,

וְקוּם וּרְקַע / עֲלֵי רֶגֶל / וְשׁוּב וּמְחָא / בְּכַפֶּךָ,

וְהֵרָפֵד / בְּתַפּוּחִים / וְהִסָּמֵךְ / בְּיֵינֶךָ,

וְגַדֵּל שֵׁם / אֱלֹהֶיךָ / וּמַלֵּא מִ- / שְּׂחוֹק פִּיךָ,

וְקוּם וּקְרָא / בְּשִׁיר שִׁירִים / וְאַל תִּקְרַב / אֱלֵי אֵיכָה.

20 וְתֶן לִי בַ- / בְּשׂוֹרָה אֶת / חֲלִיצָתָךְ / וּמַדֶּיךָ,

לְמַעַן בֶּן- / אֲבִי-מוּסַי / כְּבָר נָפַל / בְּמַהְפֵּכָה,

כְּמוֹ נִכְסַף / לְבֶן-עַבַּאס / אֲשֶׁר בִּקֵּשׁ / לְהָרְגֶךָ,

וְהָלַךְ לַ- / חֲזוֹת פָּנָיו / בְּצַלְמָוֶת / וּבַחְשֵׁכָה

וְהָעִיר, מִ- / מְּשׂוֹשׂ קָמָיו, / מְשֻׂמַּחַת / כְּמוֹתֶךָ,

25 וּמִשּׁוֹאַת / בְּנֵי בֵיתוֹ / מְבֹהָלָה / וְנָבוֹכָה.

הֲשִׁיבוֹתִיו: / דְּבַשׁ מָתוֹק / וְחָלָב עַל / לְשׁוֹנֶךָ,

וְתִתְבַּשֵּׂר / מְהֵרָה כִּי / כְּמוֹ זֹאת בָּא / לְצָרֶיךָ!

קְחָה מַדַּי / וּמַלְבּוּשַׁי / וְשִׁית נִזְרִי / בְּרֹאשֶׁךָ,

וְלֵך אֶל כָּל / מְיֻדָּעַי / וְהַשְׂמִיחֵם / בְּמִלֶּיךָ,

30 וְיֵדְעוּ כִי / יְחִידָתוֹ / לְגֵיהִנֹּם / מְמֻשָּׁכָה,

וְכִי מֶלֶךְ / הֲרָגָהוּ / וְרָאָה בוֹ / אֲשֶׁר חָכָה.

בְּנוֹ-עַבַּאס, / שְׂמַח כִּי בָא / לְשַׁחֶרְךָ / חֲבֵרֶךָ!

שְׁבוּ יַחַד, / וְהַכֵּהוּ, / בְּצַלְמָוֶת, / וְיַכֶּכָה,

וְאִם תַּחְפֹּץ — / קְשֹׁר חֶבֶל / עֲלֵי רֹאשׁוֹ / וְרֹאשֶׁךָ,

35 וְיָנוּד עַל / אֲשֶׁר הָיָה – / וְתָשִׁיב לוֹ / בְּקִינֶיךָ,

וְיִהְיֶה לָּךְ / כְּמוֹ פֶסֶל – / וְתִהְיֶה לּוֹ / כְּמוֹ מִיכָה,

וְחִרְשׁוּ עַל / בְּנֵי שַׁחַת, / שְׁנֵיכֶם, רַע / וְתַהְפּוּכָה,

וְיִרְקַב בֶּן- / אֲבִי-מוּסַי / כְּמוֹתֶךָ / בְּקִבְרֶךָ,

וְיִשָּׁכַח, / כְּמוֹ נִשְׁכַּח, / שְׁמָךְ הָרַע / וְזִכְרֶךָ.

40 כְּבָר בָּאתֶם / אֱלֵי צַוָּאר / בְּפַח יָקוּשׁ / וּבִשְׂבָכָה,

וְנָחָה הַ- / מְּלוּכָה מִן / מְשׁוֹאַתְכֶם, / וְשָׁכָכָה,

וְאֵין בָּכֶם / אֲשֶׁר יִצְלַח / לְמוֹעֵצוֹת / וְלִמְלוּכָה.

וְאַתְּ, חוֹפֵר, / אֲשֶׁר תַּחְפֹּר / קְבוּרָתוֹ / בְּיָדֶיךָ —

כְּרֵה לוֹ כַּ- / אֲשֶׁר כָּרָה, / בְּדִינָךְ וֶ- / אֱלֹהֶיךָ!

45 וְהַעְמֵק עַד / תְּהוֹם לוֹ – וַ- / אֲנִי אֶתֵּן / שְׁכָרֶךָ,

וְהַךְ בָּאֵת / לְכָל עַיִן / אֲשֶׁר עָלָיו / תְּהִי בוֹכָה.

לְזֹאת אֶלְבַּשׁ / וְאֶתְעַטֵּף / וְאֶתְבַּשֵּׂם / וְאָסוּכָה,

וְאֶתְנַהֵג / בְּטוּב לֵבָב, / וְאָשׁוּבָה / וְאֵלֵכָה,

וְאֶדַּמֶּה / אֱלֵי יוֹנִים / עֲלֵי מִלֵּאת / וּבִבְרֵכָה,

50 וְאֶתְאַמֵּץ / כְּשִׁמְשׁוֹן עַל / יְתַד אֶרֶג / וּמַסֵּכָה,

וְאָגִילָה / בְּמוֹת צָרַי / בְּבֵיתָם אוֹ / בְּמַעְרָכָה.

וְאוֹמֵר: מִי / אֲשֶׁר עָשָׂה / לְךָ אֶת זֹאת / כְּנַפְשֶׁךָ?

עֲנִיתִיהוּ: / הֲלֹא תֵדַע? / אֱלֹהִים יַ- / עֲשֶׂה כָכָה!

שְׁתֹק וּכְתֹב / לְךָ טוּרַי / לְזִכָּרוֹן / בְּסִפְרֶךָ,

וְתִקְרָאֵם, / קְרִיאָתָךְ / בְּיוֹם שַׁבָּת / בְּדָתֶךָ,

וְהָיוּ עַל / לְבָבֶךָ, / וְהוֹדַעְתָּם / לְבָנֶיךָ.
