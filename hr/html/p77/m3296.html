[נְכֵה לֵב וְדָוֶה]

כתובת: עוד לו


נְכֵה לֵב וְדָוֶה שְׂבַע בּוּז וְרָוֶה / נְדוּדִים וְנַפְשׁוֹ שְׁתוּלָה בְנָוֶה

וְנוֹאָשׁ בְּחֵימָה לְמִדְבַּר שְׁמָמָה / וְלִבּוֹ סְבִיב מַ-חֲנֶיךָ יְשַׁוֶּה

וְיָרָד וְעָלָה וְהָלַךְ בְּגוֹלָה / וְלַיְלָה לְלַיְלָה עֱזוּזָךְ יְקַוֶּה

יְקַוֶּה מְאוֹרָךְ וְעָבֵי נְדוּדָךְ / לְבֵינוֹ וּבֵינָךְ זְמַנָּךְ יְהַוֶּה

לְזֹאת מֵת בְּקִרְבּוֹ וְנֶעְצָב לְעָבָר / וְיָרֵא לְעָתִיד וְנִבְהָל לְהֹוֶה

לְמַעַן כְּבוֹדָךְ שְׁלַח טַל זְמִירָךְ / וְצִמְאוֹן קְרָבָיו בְּטַלָּךְ תְּרַוֶּה

וְיִשְׂבַּע דְּמוּתָךְ בְּלַהְטֵי נְאוּמָךְ / וּמֵעִיר מְגוּרָךְ יְקָרָךְ יְתַוֶּה

כְּמַשִׂיג מְאוֹר נֵר בְּנֹגַהּ בְּרָקָיו / וּמַבִּיט תְּמוּנָה מְכֻסָּה בְמַסְוֶה
