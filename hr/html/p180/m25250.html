שמך יגדל

שִׁמְךָ יִגְדַּל / וְלֹא יֶחְדַּל / גָּדְלְךָ מִפִּי אֲיֻמָּה

לְהַעֲצִימְךָ / בַּהֲדוֹמְךָ / יְרוּצוּן דָּרֵי אֲדָמָה

מֶלֶךְ עָל / אֲשֶׁר פָּעַל / הַכֹּל בְּדַעַת וְחָכְמָה

הֲמוֹנֶיךָ / אֱמוּנֶיךָ / עוֹנִים וְאוֹמְרִים בְּאֵימָה

יִתְגַּדַּל / וְיִתְקַדַּשׁ / שְׁמֵה רַבָּא בְּעָלְמָא


לְךָ אֶקְרָא / בְּרֹב מוֹרָא / וְאַגִּיד גָּדְלְךָ נָעֲלָה

וְגַם חִכִּי / יְצַפְצֵף כִּי / לְךָ יְיַָ הַגְּדֻלָּה

כְּבוֹדֶךָ / וְהוֹדֶךָ / אֲחַוֶּה יוֹם וָלַיְלָה

וּמַקְהֶלְךָ / בְּהֵיכָלְךָ / מְתַנִּים צִדְקְךָ הֵמָּה


מְקֻדָּשִׁים / וְתַרְשִׁישִׁים / יְבָרְכוּךָ בְּמַקְהֵלִים

יְבַקֵּשׁוּ / וְיִדְרֹשׁוּ / אַיֵּה כְבוֹד אֵל אֵלִים

וְיִפְחָדוּ / וְיִרְעָדוּ / בְּעֵת אִי כְבוֹד אֵל שׁוֹאֲלִים

כִּי רָמוּ / וְגַם עָצְמוּ / מַעֲשֶׂיךָ הַגְּדוֹלִים

וְיָעִידוּ / וְיַגִּידוּ / צִדְקָתְךָ הָעֲצוּמָה.
