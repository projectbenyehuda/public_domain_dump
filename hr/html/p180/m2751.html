שוכני בתי חומר

שֹׁכְנֵי בָתֵּי­‑חֹמֶר / ר' שלמה אבן גבירול


שֹׁכְנֵי בָתֵּי­‑חֹמֶר / לָמָּה תִשְׂאוּ עָיִן?

וּמוֹתַר הָאָדָם / מִן‑הַבְּהֵמָה אָיִן!


לָנוּ יֵשׁ לָדַעַת / כִּי אֲנַחְנוּ תוֹלָעַת:

לְגַב חֹמֶר גַּבֵּינוּ ‑ / וְאֵיךְ יִגְבַּהּ לִבֵּנוּ?


מַה יִתְרוֹן לַגֶּבֶר? / וְאַחֲרִיתוֹ לַקֶּבֶר,

וְזֶה יִהְיֶה לוֹ חֵלֶף, / לוּ חַי שָׁנִים אָלֶף.


הֲלֹא אִם יֵלֵךְ מֶרִי, / יְבֻלַּע בַּחֲמַת‑קֶרִי,

וְיִשָּׂרֵף בַּלַּהַב ‑ / וְלֹא יוֹעִיל הַזָּהָב.


הָאָדָם הַנִּכְאֶה, / פְּקַח עֵינֶיךָ וּרְאֵה

מֵאַיִן בּוֹאֶךָ / וְאָנָה מוֹצָאֶךָ.


קִצְּךָ, עָנִי וְאֶבְיוֹן, / נִמְשַׁל כְּמוֹ קִיקָיוֹן

שֶבִּן‑ליְלָה הָיָה / וְעַד בֹּקֶר לֹא חָיָה.


טוֹב שֶלֹּא נוֹצַרְתָּ / וְעָמָל לֹא קָצַרְתָּ,

וְאֵיכָה גְּדוֹלוֹת תְּבַקֵּשׁ ‑ / וְאַתָּה בְּתוֹךְ הַמּוֹקֵשׁ!


נוֹפֵל אַתְּ מֵרֶחֶם / וְתִפְשַׁע עַל פַּת לֶחֶם,

וְאִם תִּתְמַהְמַהּ רֶגַע ‑ / אֲזַי תִּמְצָא כָּל‑נֶגַע.


בְּשָׂרְךָ כְּאוֹר זָרוּחַ / בְּעוֹד יֶשׁ‑בּוֹ הָרוּחַ,

וְעֵת תֵּצֵא הַנֶּפֶשׁ ‑ / נִשְׁאַר טִיט וָרֶפֶשׁ.


רְאֵה כִּי אֵין בְּיָדֶךָ / מְאוּמָה מִכְּבוֹדֶךָ,

לְזָרִים חֵילְךָ יוּקָם, / וְאַתָּה תֵּלֵךְ רֵיקָם.


בְּכָל זֹאת לֹא חָשַׁבְתָּ, / לַתַּאֲוָה הִקְשָׁבְתָּ,

וְאִם לֹא טוֹב רֵאשִׁיתָהּ ‑ / מַה תַּעֲשֶׂה בְּאַחֲרִיתָהּ?


יַעֲזֹב רָשָׁע דַּרְכּוֹ / וְיָשֹׁב לִפְנֵי מַלְכּוֹ,

אוּלַי צוּר יֵעָתֵר / וּמֵחֲרוֹנוֹ יִסָּתֵר.


יְהִירִים, הִתְקוֹשֶׁשׁוּ, / וְזִכְרוּ וְהִתְאֹשֶׁשׁוּ,

שְׂאוּ לֵב וְכַפָּיִם / אֶל אֵל בַּשָּׁמָיִם!


הָהּ עַל‑חַטֹּאתֵינוּ / וְאוֹי עַל נַפְשׁוֹתֵינוּ,

כִּי רוּחַ רָעִינוּ / וְגַם כַּצֹּאן תָּעִינוּ!


וּמַה‑נְבַקֵשׁ וּמַה‑נִּדְרֹשׂ / וַעֲוֹנוֹת עָבְרוּ רֹאשׁ,

מְאֹד רַבּוּ עֲוֹנֵינוּ ‑ / וְאֵיךְ נִשָּׂא פָנֵינוּ?


דְלֵה עַמְּךָ מִשַּׁחַת, / מוֹשֵׁל רוֹם וָתָחַת,

וְאִם הִקְשֵׁינוּ עֹרֶף / יָדֶיךָ אַל­‑תֶּרֶף.


הָפֵק חֶמְלָתֶךָ / לְעַם דּוֹפְקֵי דַלְתֶּךָ,

כִּי אַתָּה אֲדֹנֵינוּ, / וְעָלֶיךָ עֵינֵינוּ!
