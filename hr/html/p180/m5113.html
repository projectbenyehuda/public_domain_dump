[אשר ראה]

ולה פי אלאעתראף פי אלרבוביה [ולו על ההודאה במרות (הקב"ה)]

È — — — / È — — — / È — — / / È — — — /È — — — / È — —

אֲשֶׁר רָאָה לְכָל צָפוּן וְחָזָה / וְלֹא שִׁקַּץ עֱנוּת עָנִי וּבָזָה,

לְהָבִין דַעְתְּךָ הַרְחֵב לְבָבִי / וְשִׂימָה נָא לְעַבְדָּךְ בָּהּ אֲחֻזָּה –

וְאָז אֶשְׂבַּע תְּמוּנָתָךְ בְּהָקִיץ / וְתַחַת כִּסְאֲךָ נַפְשִׁי גְנוּזָה!
