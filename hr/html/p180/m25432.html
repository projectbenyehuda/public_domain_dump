שְׁתֵּי פְעָמִים

שְׁתֵּי פְעָמִים / מְקֻיָּמִים / בְּכָל יוֹם עַל הֲמוֹנֵנוּ

לְיַחֵד צוּר / בְּלֵב עָצוּר / וּבְהֶגְיוֹן לְשׁוֹנֵנוּ

מִמְּקוֹמוֹ / מִמְּרוֹמוֹ / יִשְׁלַח אוֹר לְעֵינֵינוּ

הוּא לְבַדּוֹ / יְצַו חַסְדּוֹ / וְיַאֲזִין קוֹל רִנּוּנֵנוּ

בְּקָרְאִי שְׁמַע / יִשְׂרָאֵל / יְיָ אֱלֹהֵינוּ


שְׁאָר יַעֲקֹב / לְךָ אֶקֹּב / יְסוֹד דָּת אֵל אֲשֶׁר נָתָן

לְכָה וּשְׁמַע / תּוֹרַת שְׁמַע / אֲשֶׁר נָחֲלוּ בְנֵי אֵיתָן

מִצְוָתָהּ / לִקְרוֹתָהּ / לְנוֹחֲלֵי גַן הַבִּיתָן

הֲגוֹת עַרְבִית / אַל תַּשְׁבִּית / כְּתַנָּאִים בְּמִשְׁנָתָן

בְּהִכָּנֵס / כֹּהֲנֵי נֵס / לֶאֱכֹל בִּתְרוּמָתָן

רְשׁוּת נוֹעָד / לְקָרְאָהּ עַד / עֲלוֹת שַׁחַר כְּמוֹ חָתָן

יְסוֹד אַרְבַּע / לָהּ נִקְבָּע / וְלָעַד לֹא לְהַכְרִיתָן

הֵן עֲרוּכוֹת / שְׁתֵּי בְרָכוֹת / לְפָנֶיהָ כְּהִלְכָתָן

וְחֹק בְּרִיתָהּ / לְאַחְרִיתָהּ / שְׁתֵּי בְרָכוֹת לְהוֹרוֹתָן

דָּת וָחֹק / מֵרָחוֹק / הֲגוֹת בָּהּ עַל מְלוֹנֵנוּ


שְׁתֵּי בְרָכוֹת / נֶעֱרָכוֹת / בְּרֹאשׁ שַׁחֲרִית לְהִתְהַלֵּל

לְאַחְרִיתָהּ / כִּבְרִיתָהּ / סְמֹךְ אַחַת לְהִתְפַּלֵּל

מְסוּכָתָהּ / וְהִלְכָתָהּ / עִם הָנֵץ מְאוֹר הֵילֵל

הֲלֹא נִסְעַד / וְגַם יָעַד / שְׁלֹשׁ שָׁעוֹת לְהִתְחוֹלֵל

בְּעֵת הָלַךְ / כָּאן וְאֵילַךְ / לֹא הִפְסִיד ולֹא חִלֵּל

רוֹחֲשִׁים כֹּה / אִישׁ כְּדַרְכּוֹ / וַהֲלָכָה כְּבֵית הִלֵּל

יָד יִפְסֹק / אֲשֶׁר יַעֲסֹק / בְּרֹאשׁ פְּרָקִים לְהִתְמוֹלֵל

הָיָה רָץ / לֹא יֶחֱרָץ / אֲבָל יַעֲמֹד וִיפַלֵּל

וְכֵן יִפְעַל / קְרֹא עַד עַל / לְבָבֶךָ לְהִסְתּוֹלֵל

דִּין נִשְׁמַע / קְרִיאַת שְׁמַע / בְּכָל לָשׁוֹן לְהִתְמַלֵּל

הֲלֹא נוֹדְעָה / בְאַרְבָּעָה / אֲשֶׁר פֵּרְשׁוּ זְקֵנֵינוּ


שֵׁם אֶחָד / יְרָא וּפְחַד / וְהִשָּׁמֵר בְּאוֹתוֹתָיו

לְבַד הַחֵת / אַל תַּשְׁחֵת / וְהַאֲרֵךְ דַּל כְּחֻקּוֹתָיו

מַאֲרִיכוֹ / לְהַמְלִיכוֹ / עַל אַרְבַּע רוּחוֹתָיו

הַמְפַחֵד / אַל יְכַחֵד / בָּאֶמְצַע שְׁאֵלוֹתָיו

בְּרִית יָקִים / בֵּין פְּרָקִים / לְהָשִׁיב לְכָל שְׁלוֹמוֹתָיו

רֶוַח יָקִים / בֵּין דְּבֵקִים / וִידַקְדֵּק בְּאוֹתוֹתָיו

יְרָא וַעֲשֵׂה / כְּרַב יוֹסֵי / וְהַאֲזַנְתָּ לְמִצְווֹתָיו

הַקּוֹרֵא / אַל יוֹרֶה / בְּאֶצְבְּעוֹתָיו וְשִׂפְתוֹתָיו

וְאִם טָעָה / בְאַרְבָּעָה / הֲלֹא יַחֲזֹר לְרֵאשִׁיתָיו

דַּת סָפֵק / לְהִתְאַפֵּק / וְיַחֲזֹר לְרֹאשׁ קְרִיאוֹתָיו

הָלַךְ אָן / וּמְצָאָן / שְׁהֵן קוֹרְאִין לְעֻמָּתָיו

יְהִי עָסוּק / בַּפָּסוּק / הָרִאשׁוֹן כְּדִינֵנוּ.
