[הֲפָרַשׂ לַיְלָה עַל־יוֹם כְּנָפוֹ]

ולה פי אכ֗תיאר אלחלאואת [ולו מן המובחר שבעניינים ערבים]


הֲפָרַשׂ לַֽיְלָה עַל־יוֹם כְּנָפוֹ / וְעֵת דִּמָּה לְכַסֹּתוֹ חֲשָׂפוֹ

וְרָדַף אַחֲרֵי שֶׁמֶשׁ לְחָיָיו / וְלוּ צֶדֶק רְדָפוֹ לֹא רְדָפוֹ

כְּיָרֵחַ סְבָבוֹ זֵר יְרַקְרַק / וְהֶעָנָן אֲפָפוֹ עֵת אֲסָפוֹ

כְּמוֹ אָמִיר הֲדַס נַפְשׁוֹ יְשׁוֹבֵב / וְהָרוּחַ בְּלִי־רוּחַ יְנִיפוֹ

וּבָחַר בַּהֲדַס הָאֵל לְצַוֹּת / וְלָקַחַת בְּיוֹם רִאשׁוֹן עֲנָפוֹ

וְנָשָׂאתִי קְצֵה עַיִן לְהַבִּיט / וְצֹפַי אָֽמְרוּ־לִי מַה־לְּךָ פֹה

יְדִידַי דַּבְּרוּ אֵלָיו חֲלָקוֹת / וְיָשִׁיב אֶת־לְבָבִי אוֹ שְׂעִפּוֹ

והוּא פִזַּר דְּמָעַי עֵת אֲסַפְתִּים / וְעֵת נִפְזַר לְבָבִי הוּא אֲסָפוֹ

וְלוּ הֻשְׁלַךְ בְּשֵׂאתוֹ הַתְּלָאוֹת / בְּקֶרֶב אוּר נְבוּכַד לֹא שְׂרָפוֹ

אֱמֹר לַיּוֹם וְיִרֶף כִּי אֲנִי בוֹ / בְּמִצְרַף הָעֳנִי שֶׁבַע צְרָפוֹ

קְרָאוּנִי טְרָף־טֶרֶף עֲנִיתִים / אֲבָל כֶּרֶם כְּסִיל בֹּסֶר קְטָפוֹ

וְלוּ טוֹב לָאֲרִי חָצִיר וְיָשָׁר / בְּעֵינֵי הָאֲרִי עֵשֶׂב טְרָפוֹ

וְנַפְשׁוֹ לָֽמְדָה בַּקֵּשׁ גְּדֹלוֹת / עֲדֵי נָקֵל בְּעֵינָיו סוּף וְיָפוֹ

וְחָלִילָה לְנַפְשִׁי מִֽהְיוֹת אִישׁ / יְרַדֵּף הַזְּמַן אִוָּה הֲדָפוֹ

אֲבָל אֶהְיֶה לְעוֹלָם רַב וְנָדִיב / יְדַכֵּא הַזְּמָן תַּחַת כְּנָפוֹ

וְאִם־לֹא יִזְּלוּ יוֹרֶה עֲנָנָיו / וְאִם־לֹא יַעֲרֹף לֶקֶשׁ עֲרִיפוֹ

וְאִם־תַּנִיד אֲדָמָה אַחֲרַי רֹאשׁ / וְאִם־בִּי מָאֲסָה מִהְיוֹת צְנִיפוֹ

וְרָעַת הָאֲדָמָה לִי כְּטוּבָהּ / וְגַם רֹאשׁ הַזְּמַן עִמִּי כְּצוּפוֹ.
