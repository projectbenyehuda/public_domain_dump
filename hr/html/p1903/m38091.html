משל ל: שׁוֹר, אֲרִי וְתַיִשׁ

אֲשֶׁר לוֹ אֹרֵב / יִפְחַד בֹּקֶר וָעֶרֶב


שׁוֹר רָאָה אֲרִי וַיִּבְרַח / כִּי אַחֲרָיו יִשְׁאַג וְיִנְהֹם וְיִצְרַח / וַיִּתְחַבֵּא בְּאַחַת הַפְּחָתִים / תַּחַת צַמֶּרֶת הָעֲבוֹתִים / וְהִנֵּה שָׁם תַּיִשׁ נִסְתָּר / מִפַּחַד לִבּוֹ יֶחֱרַד וְיִתָּר / וַיֹּאמֶר לוֹ הַתַּיִשׁ: לָמָּה תִּפְחַד / הֲלֹא אֲנִי וְאַתָּה הָיִינוּ בְּעֵדֶר אֶחָד / וַיַּעַן הַשּׁוֹר: כַָּל הַבְּהֵמוֹת אֲשֶׁר אֶרְאֶה עוֹד חַיִּים / הָיוּ בְעֵינַי כִּשְׁחָלִים וָאֲרָיִים / וְאִם מְצָאתִיךָ לְבַדְּךָ לֹא פָחַדְתִּי / אַךְ עַתָּה מִפְּנֵי הָאֲרִי נִצְמַתִּי וְנִבְעַתִּי.


וְהַמָּשָׁל בְּאִישׁ אֲשֶׁר לוֹ אֹרֵב / תָּמִיד יִפְחַד בֹּקֶר וָעֶרֶב / בְּלֶכְתּוֹ בְּשִׁבְתּוֹ בְּקוּמוֹ וּבְשָׁכְבוֹ / כָּל אִישׁ וָאִישׁ יַחְשֹׁב כְּאֹיְבוֹ / עַתָּה יֹאמַר אֶהְיֶה נִרְדָּף / וְרָדַף אֹתוֹ קוֹל עָלֶה נִדָּף.
