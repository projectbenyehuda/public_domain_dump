משל נא: אַרְבָּעָה שְׁוָרִים וַאֲרִי

חֲבוּרָה בְלִי פַחַד / שֶׁבֶת אַחִים גַּם יַחַד


שְׁוָרִים אַרְבָּעָה לְחֶבְרָה נוֹעָדוּ / יִתְלַכְּדוּ וְלֹא יִתְפָּרְדוּ / וּבְרִית כָּרְתוּ בֵינֵיהֶם / וְרוּחַ אֶחָד לָהֶם / וְיָמִים רַבִּים בְּחֶבְרָתָם הֶאֱרִיכוּ / בְּעָמְדָם יַעֲמֹדוּ וּבְלֶכְתָּם יֵלֵכוּ / וְהָאֲרִי סוֹבֵב לְאַרְבַּעְתָּם / וַיַּרְא כִּי לֹא הִגִּיעָה עִתָּם / בַּאֲשֶׁר יַעַמְדוּ יַחַד בְּמַעֲרֶכֶת / וְאַרְבָּעָה הֵמָּה מֵיטִיבֵי לֶכֶת / וַיָּשֶׁת הָאֲרִי עֵצוֹת בְּנַפְשׁוֹ / מִדֵּי חֹדֶשׁ בְּחָדְשׁוֹ / לַעֲשׂוֹת בָּהֶם כָּלָה / אִם נִתְפָּרְדָה הַחֲבִילָה / וַיְהִי הַיּוֹם וַיַּעֲמֹד הָאֲרִי מֵרָחֹק / וַיִּפְעַר פִּיו לִבְלִי חֹק / בַּאֲשֶׁר הוּא רָעֵב וְעָיֵף / וּמִתּוֹחֶלֶת מְמֻשָּׁכָה סָר וְזָעֵף / וַיִּקְרָא אֶל הַשּׁוֹר / אֲשֶׁר עָמְדוּ רַגְלָיו בַּמִּישׁוֹר / וַיְסִיתֵהוּ בְּחֵלֶק שְׂפָתוֹ / עַד כִּי הִפְרִידוֹ מֵחֲבוּרָתוֹ / וּבְהִתְפָּרְדוֹ מֵחֲבֵרָיו לְרָעָה הִבְדִּילוֹ / כִּי בְאַפּוֹ טְרָפוֹ וַאֲכָלוֹ / וְהַשְּׁלֹשָׁה הַנּוֹתָרִים בַּמַּחֲנֶה / מִקּוֹלָם לֹא יֵחָת וּמֵהֲמוֹנָם לֹא יַעֲנֶה.

וְהַמָּשָׁל הִנֵּה מַה טּוֹב וּמַה נָּעִים / בְּהִתְאַסֵּף בְּמֶתֶק סוֹד רֵעִים / וַחֲבוּרָתָם בְּלִי פַחַד / שֶׁבֶת אַחִים גַּם יָחַד / וְהַמְפַתִּים אֲשֶׁר בְּפִיהֶם כְּמֵי אַכְזָב / יָעֲצוּ לְהַדִּיחַ יִרְצוּ כָזָב / לְהַפְרִידָם מֵחֲבוּרָתָם / עַד אֲשֶׁר זָרוּ מִתַּאֲוָתָם / וְאֶת אֲשֶׁר מֵחֶבְרָתָם נוֹדֵד / הַבּוֹגֵד בּוֹגֵד וְהַשּׁוֹדֵד שׁוֹדֵד / וְהַחֲכָמִים לֹא יָשִׂימוּ עַל לֵב אִם הֶעָמִית לָהֶם יַלְשִׁינוּ / אָזְנַיִם לָהֶם וְלֹא יַאֲזִינוּ / בַּאֲשֶׁר רוּחָם טְהוֹרָה וּנְקִיָּה / וְיִשְׂנְאוּ לְשׁוֹן שֶׁקֶר עֹשֵׂי רְמִיָּה / וַחֲבוּרַת שְׁנֵי חֲכָמִים לֹא תֵרָתֵק / וְהַחוּט הַמְּשֻׁלָּשׁ לֹא הִמְהֵרָה יִנָּתֵק.
