[יוֹמָם וָלַיִל לְאֵל עֶלְיוֹן] (פִּזְמוֹן)

Ganso Josef

[נולד בפֵּרוּגָ’ה במחצית הראשונה של המאה הי"ז, חי בעיר בְּרוּסָה בארץ תוגרמה ונפטר בירושלים]


פִּזְמוֹן

יוֹמָם וָלַיִל לְאֵל עֶלְיוֹן / אָקוּם לְהוֹדוֹת

אָבוֹאָה אֶל־מִשְׁכְּנוֹתָיו / לִי מַה־יְּדִידוֹת

אֲגַדֵּל וַאֲהַלֵּל שְׁמוֹ / בְּשִׁיר יְדִידוֹת


מִזִּמְרַת שִׁיר הֶגְיוֹנַי / לוֹ אֲשַׁלֵּם תּוֹדוֹת.


וַאֲנִי עָנִי וְכוֹאֵב / יַעֲלֹז לִבִּי

וּבְשִׁירִי אֲהוֹדֶנּוּ / כָּל־עוֹד רוּחִי בִי

אֲרַנֵּן לוֹ לַבֹּקֶר / כִּי יִהְיֶה מִשְׂגַּבִּי


וּבְיוֹם צַר לִי יִהְיֶה / מָעוֹז לְבֵית מְצוּדוֹת.


סְבִיבוֹתָיו אֲסוֹבֲבָה / וּמְעוֹן כְּבוֹדוֹ

בְּמוֹ־פִי אֵלָיו אֶתְחַנֵּן / אֶשְׁאֲלָה חַסְדּוֹ

כִּי חִישׁ עָרֵי יְהוּדָה / יִבְנֶה לְזֶרַע עַבְדּוֹ


אוֹהֲבֵי שְׁמוֹ, וְיִשְׁכֹּן / עָרִים נִכְחָדוֹת.


פְּנֵי אֱלֹהַי אֲבַקֵּשׁ / אֵלָיו אוֹחִילָה

אֲחַלֶּה פָנָיו כָּל־הַיּוֹם / עַל־הַגְּאֻלָּה

כִּי שָׁתִיתִי קֻבַּעַת / כּוֹס יֵין הַתַּרְעֵלָה


אֶשְׁאֲלָה מֶנּוּ כִּי יִשְׁלַח / לִי אִישׁ חֲמוּדוֹת.
