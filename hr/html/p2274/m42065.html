עֲטֶרֶת פָּז

סימן: עבדאללה. מנגינה: תורכי חוסיני


עֲטֶרֶת פָּז תָּשִׁית / לְרֹאשׁ קָצִינִי, דּוֹדִי

גּוֹאֲלִי מֵרֵאשִׁית / עוֹלָם תַּנְחֵנִי, דּוֹדִי

אֶת נַפְשִׁי מִצַּר לַחָפְשִׁית / מַהֵר פְּדֵנִי.

חַי, אָז בְרִנָּה, בּוֹא יָבוֹא נָא

אֲהַלְּלָה, אֲהַלְּלָה

חַי, בְּשִׁיר רְנָנִי

בְּשִׁיר רְנָנִי.


בְּתוֹךְ נַפְשִׁי צִירִים / אָנָּא רְפָא אֵל, דוֹדִי

כִּרְפוּאַת אַב הוֹרִים / עַל יַד רְפָאֵל, דוֹדִי

וּבְקוֹל נָעִים, שִׁיר הַשִּׁירִים / תָּרֹן לְשׁוֹנִי.

חַי אָז


דּוֹרֵשׁ טוֹב צִיר נֶאֱמָן / הוּא אֵלִיָּהוּ, דּוֹדִי

מְבַשֵּׂר טוֹב לִזְמָן / נָדָב אָבִי הוּא, דּוֹדִי

יְגַלֶּה עֵת קֵץ הַנִּטְמָן / תִּרְאֶינָה עֵינִי.

חַי אָז


אֵל חַי יָבֹא שִׁילֹה, / שֵׁבֶט מִיהוּדָה, דּוֹדִי

יִשְׁמַע תּוֹךְ אָהֳלוֹ / מִזְמוֹר לְתוֹדָה, דּוֹדִי

יוֹבִילוּ מְלָכִים שַׁי לוֹ / עַבְדֵי אֲדוֹנִי.

חַי אָז


לְהַשְׁקוֹתָם פּוֹרִיָּה / יַיִן וְחָלָב, דּוֹדִי

לְהַרְבּוֹתָם רִבְיָה / צַחוּ מֵחָלָב, דּוֹדִי

וּמִרְעֶה טּוֹב עִיר צְבִיָּה / תִּרְעֶה אֶת צֹאנִי.

חַי אָז
