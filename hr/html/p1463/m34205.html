אֶפְתַּח פֶּה בְרֶנֶן

אֶפְתַּח פֶּה בְרֶנֶן / אֲהַלֵּל לְשָׂם פֶּה

בְּשִׂפְתֵי רְנָנוֹת / גֹּדֶל אֶתֵּן לַמֶּלֶךְ.


גָּדְלוֹ אֲשַׁנֵּן / בְּקָהָל וְעֵדָה

דְּבָרָיו אֱמוּנָה / וְהוּא לְעוֹלָם מֶלֶךְ.


הֲלֹא הוּא נוֹתֵן / מַעֲנֵה לָשׁוֹן

וּלְכָל אֲשֶׁר יַחְפֹּץ / יַטֶּה לֵב, מֶלֶךְ.


זִכְרוֹ לְדוֹר דּוֹר / שְׁנוֹתָיו לְאֹרֶךְ יָמִים

חַי הוּא לְעוֹלָם / וְאֵין כְּנֶגְדּוֹ מֶלֶךְ.


טַעַם אַמְתִּיק לוֹ / בְּחֵיךְ וְגָרוֹן

יֶהְגֶּה לְשׁוֹנִי צֶדֶק / תְּהִלּוֹת מֶלֶךְ.


כָּל עַצְמוֹתַי / וְחַדְרֵי קְרָבַי

לְשֵׁם קָדוֹשׁ אֲהַלֵּל / כִּי הוּא לְעוֹלָם מֶלֶךְ.


מַעֲנֶה אֶשְׁאֲלָה מֶנּוּ / וְלֹא מֵאֵל אַחֵר

נַפְשִׁי יְשׁוֹבֵב / בְּנַחַת לְגַדֵּל מֶלֶךְ.


שְׂפָתַי תַּבַּעְנָה / חֵן וְתַחֲנוּנִים

עֲנוֹת סִיחַ בְּפִי / בְּסוֹד נֹעַם מֶלֶךְ.


פָּעֳלוֹ אֲשַׁנֵּן / בְּפֶה וּבְלָשׁוֹן

צִדְקוֹ אַגִּיד / כִּי הוּא לְבַדּוֹ מֶלֶךְ.


קְרָבַי וְכִלְיוֹתַי / זֶמֶר יַבִּיעוּ

רֶנֶן יְתַנּוּ / כִּי אֵין זוּלָתוֹ מֶלֶךְ.


שִׁיר חָדָשׁ אָשִׁיר / לְיוֹצֵר בְּרֵאשִׁית

תְּהִלָּה אַמְתִּיק לוֹ / כִּי מִמֶּנּוּ מַעֲנֵה לָשׁוֹן.
