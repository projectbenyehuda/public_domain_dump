עַל מָה שַׂר

שיר ששלח אלי יצחק בו כ׳בזא תרנ״ג חשון מעי״ת גאבש ותשובתו בפנים.


עַל מָה שַׂר / בִּלְתִּי אוֹכֵל בָּשָׂר / גַּם מֵעוֹף הַשָּׁמַיִם

הֵן הַסַּכִּין בְּדוּקָה / כַּדָּת וְכַחֻקָּה / בְּבָשָׂר וְצִפָּרְנַיִם

וְעַל הַטַּרְפִיּוֹת / עֵינֵי הַשּׁוֹחֵט צוֹפִיּוֹת / בְּפֶתַח עֵינָיִם

וְהַנְּפוּחָה אֲסוּרָה / נְתוּנָה אוֹ מְכוּרָה / לְכוּשַׁן רִשְׁעָתַיִם

וְהַשּׁוֹחֵט בָּקִי / צַדִּיק וְגַם נָקִי / מְזֻקָּק שִׁבְעָתַיִם

וְעַל הַמְּלִיחָה / עֵין הָאִשָּׁה פְתוּחָה / בַּחֲרִיצוּת יָדָיִם

אִם כֵּן אֵיפֹא / לָמָּה יוֹשֵׁב פֹּה / בְּנִקְיוֹן שִׁנָּיִם

וּמַדּוּעַ / נַפְשׁוֹ תָנוּעַ / אֶל דְּגֵי הַמָּיִם

חִישׁ תַּעֲנֵנִי / וּמַהֵר תְּשִׁיבֵנִי / בְּשִׁיר וּמְצִלְתַּיִם

תְּשׁוּבָה נִצַּחַת / אֶת כֹּל וְנוֹכָחַת / בְּמֶתֶק שְׂפָתָיִם

וְטוּבָךְ אֲחַוֶּה / וְאֶקֹּד וְאֶשְׁתַּחֲוֶה / מוּלְךָ עַל אַפָּיִם

וּבְכָל הַזָּוִיּוֹת / עַד שֶׁיִּתְפָּקְקוּ כָל הַחֻלְיוֹת / אֶכְרַע עַל בִּרְכָּיִם

וְהַחַיִּים וְהַשָּׁלוֹם / לָךְ עֲטֶרֶת יַהֲלֹם / אִישׁ יְרוּשָׁלַיִם

סֻלָּם מֻצָּב אַרְצָה / בְּשִׁירוֹ מֵי הַיָּם חָצָה / וַיַּעֲבֹר בָּרַגְלָיִם

כֹּה דִבְרֵי יִצְחָק / חַי בְּאֶרֶץ מֶרְחָק / עֲרֵל שְפָתָיִם

אֲבִי לֶחֶם יְכֻנֶּה / מְקַוֶּה רַחֲמֵי קוֹנֶה / בְּרַחֲמָיו אֶרֶץ וְשָׁמַיִם.
