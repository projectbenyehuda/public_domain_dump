הֱנִיאַנִי זְמָן סוֹפוֹ

תנחומים לאחיהו למות אחיו


הֱנִיאַנִי זְמָן סוֹפוֹ כְרֹאשׁוֹ / וְהֶלְאַנִי אֲכֹל צוּפוֹ כְּרוֹשׁוֹ,

וְנִסִּיתִי יְמֵי הַבְלֵי פְעָלָיו / וְכָל־צִדְקוֹ מְצָאתִיהוּ כְּכַחְשׁוֹ,

וְכָל יוֹם יַהֲלֹךְ אֶבְכֶּה תְּמוֹלוֹ / וְלַיִל יַעֲבֹר אָנוּד לְאַמְשׁוֹ,

וְאִם חָדָשׁ יְהַרְאוּנִי יְלָדָיו / לְיָרְאֵנִי – כְּבָר הָיָה חֲדָשׁוֹ,

וּבוֹ לֹא אַאֲמִין, כִּי שָׁוְא יְדַבֵּר, / וְרוֹשׁ פֶּתֶן יְהִי צוֹפֵן בְּדִבְשׁוֹ,

וּמֵאָז יַחֲלִיף נוּחוֹ בְנוּעוֹ / וְאֶת־עָשְׁרוֹ יְהִי מֵמִיר בְּרִישׁוֹ,

וְיָכִין פַּעֲמֵי סֶכֶל, וְרִשְׁתּוֹ / לְיַד מַשְׂכִּיל יְהִי פֹרֵשׂ וּמוֹקְשׁוֹ,

וְיַשְׁוֶה אֶת־כְּפִירָיו לַעֲיָרָיו / וְיַעְרֹךְ אֶלְגְּבִישׁוֹ אֶל גְּבִישׁוֹ,

וְיָרֵא מִפְּנֵי צוּרוֹ וְחָרֵד / לְמִדְבָּרוֹ – לְפִי חַרְבּוֹ חֲלָשׁוֹ.

וּפֶתִי יַחֲשֹׁב רִגְעוֹ כְיוֹמוֹ / לְהִתְעַשֵּׁר, וְאֶת־יוֹמוֹ כְּחָדְשׁוֹ;

וְלֹא בָן כִּי אֲשֶׁר יִקְנֶה לְאֶחָד / הֲכִי הוּא מִן יְדֵי אַחֵר יְרָשׁוֹ,

וְכִי כַסְפּוֹ יְהַכְלִימוֹ בְחַיָּיו / בְּעֵין צוּרוֹ בְיוֹם מָוֶת יְבִישׁוֹ.

וּמַה־בֶּצַע בְּהוֹן יֶהְבַּל – וְתַעַל / לְאַפּוֹ צַחֲנַת דָּמוֹ וּבָאְשׁוֹ,

וְתֵבֵל בֵּין מְבַקְשֶׁיהָ כְּפֶגֶר / יְרַד כָּל אִישׁ יְנַתֵּחַ לְנַפְשׁוֹ,

לְזֶה יָרֵךְ וְזֶה כָתֵף, וְכַמָּה / אֱנוֹשׁ פִּדְרוּ יְהִי חֶלְקוֹ וּפִרְשׁוֹ,

וְהַמָּוֶת מְרַדֵּף אַחֲרֵיהֶם / וְנֶפֶשׁ כָּל־אֱנוֹשׁ יִקַּח כְּנוֹשׁוֹ,

כְּמוֹ מַגָּל אֲשֶׁר שֻׁלַּח בְּשָׂדֶה / וְאֶת־זַרְעוֹ יְהִי קוֹצֵר כְּלִקְשׁוֹ.

לְךָ, הַתָּם, אֲשֶׁר יִרְאֶה בְאִישׁוֹן / תְּבוּנָתוֹ וְעֵין שִׂכְלוֹ לְטָשׁוֹ,

אֲשֶׁר חָלָה בְמַחֲלָתִי, וְלָזֹאת / אֲנִי מוֹצֵא יְקוֹד לִבּוֹ וְאִשּׁוֹ,

אֲשֶׁר דִּמְעוֹ כְנִטְ[פֵי עָב וְרוּחוֹ] / תְּלַהֵט גַּחֲלֵי תֹפֶת וְנַפְשׁוֹ.

וְאֵיךְ יִחְיֶה אֲשֶׁר לַהַב בְּקִרְבּוֹ / וְיֵרָאוּ אֲפִיקֵי יָם בְּרֹאשׁוֹ.

וְיִקָּרַע סְגוֹר לִבּוֹ, לְמַעַן / קְשׁוּרָה הָיְתָה נַפְשׁוֹ בְנַפְשׁוֹ,

וּמַה־לִּבְכּוֹת לְאִישׁ עָטָה עֲנָוָה / וְהַצֶּדֶק מְקוֹם מַדּוֹ לְבוּשׁוֹ,

וּפָשַׁט מַחְלְצוֹת גֵּאָה וְחָמַד / פְּאֵר חֶסֶד וּמִנֹּעַר חֲבָשׁוֹ,

וְצִיר הָיָה לְדֶלֶת הַתְּעוּדָה / וּבְרִיחַ לְמִשְׁכַּן תֹּם וְקַרְשׁוֹ,

וּמָוֶת רָץ כְּבוֹד יַעְרוֹ לְשׁוֹטֵט / וְכַרְמֶל וַעֲדֵי טָרַף לְיָשׁוֹ,

וְהָרוּחַ בְּעֵת תֶּחְזַק יְהַשְׁלִים / עֲצֵי רֹתֶם וְתַפִּיל אֶת־בְּרוֹשׁוֹ,

וְהָאֹפֶל יְכַס שֶׁמֶשׁ וְסַהַר / וְלֹא יַחְשִׁיךְ כְּסִיל שַׁחַק וְעָשׂוֹ.

אֲחִיהוּ, חֲיֵה לָעַד וְאַל נָא / תְּהִי יָרֵא וְרַךְ הַלֵּב אֲנוּשׁוֹ.

הֲכִי כָל הַיְּקוּם נוֹצַר לְחֶרֶץ / וְכָל הוֹיֶה עֲפַר אֶרֶץ לְבוּשׁוֹ,

וְשֹׁרֶשׁ הָאֱנוֹשׁ עָפָר וְאוֹתוֹ / בְּעַל־כֹּרַח יְהִי מוּשָׁב לְשָׁרְשׁוֹ,

וְגַלְגַּל עַל מְרוּצָתוֹ וְכָכָה / יְרֵחוֹ עַל תְּקוּפָתוֹ וְשִׁמְשׁוֹ,

לְיַשֵּׁב הָאֲדָמָה מַהֲלָכוֹ / בְּבוֹא יַלְדּוֹ וְהַסִּיעַ יְשִׁישׁוֹ,

וְאִם חָלַף סְפִיחַ הָאֱמוּנָה / הֲלֹא מִיָּדְךָ נֹאכַל סְחִישׁוֹ.

לְזֹאת הַשְׁקֵט וְהִנָּחֵם, וְהַתָּם – / אֲשֶׁר יִצְרוֹ בְּיַד שֵׂכֶל כְּבָשׁוֹ,

וְהוּא נָסַע לְבֵית מוֹעֵד לְכָל־חַי / וּמַרְגֹּעַ אֲשֶׁר מֵאֵל דְּרָשׁוֹ,

וְשָׁם יִקְצֹר אֲשֶׁר זָרַע, וְאַשְׁרֵי / כְּמוֹתוֹ אִישׁ אֲשֶׁר יִקְצֹר חֲרִישׁוֹ,

וְיִתְלוֹנֵן בְּצֵל שַׁדַּי וְטוּבוֹ / וְנַפְשׁוֹ תֶחֱסֶה בִּכְנַף קְדוֹשׁוֹ.
