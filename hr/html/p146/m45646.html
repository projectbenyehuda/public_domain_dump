אָרְחוֹת יוֹשֶׁר

חתימה: אלשבזי


אָרְחוֹת יוֹשֶׁר תְּיַשֵּׁר / בָּנִים הַשּׁוֹבָבִים

לֵילֵךְ בְּדֶרֶךְ מִישׁוֹר / בֵּין יְשָׁרִים נֶאֱהָבִים

כָּל מַעֲשֵׂיהֶם בְּהֶכְשֵׁר / לְאֵל צוּרָם נֶחְטָבִים

דִּגְלֵי שְׁבָטָיו קוֹשֵׁר / עַל שַׁעַר בַּת רַבִּים

וְשָׁם סֶרַח בַּת אָשֵׁר / תְּבַשֵּׂר לִבְנֵי נְדִיבִים

תּוֹסִיפוּ שָׁנִים רַבּוֹת / וּמוֹעֲדִים טוֹבִים:


לֵב נִדְכָּאִים יָשׁוּב / וְיִחְיֶה וְיַאֲרִיךְ יָמִים

וְשִׂיחוֹ יִהְיֶה קָשׁוּב / לְאֵל חַי נָקִי וְתָמִים

בְּסֵפֶר חַיִּים כָּתוּב / בְּאוֹתִיּוֹת נִרְשָׁמִים

וְשָׂבֵעַ מִכָּל טוּב / בֵּין עַם קוֹדֶשׁ חֲכָמִים

וְיָקִיצוּ נִרְדָּמִים / בְּחֵן צוּרָם נִצָּבִים:


שׁוּבִי הַשּׁוּלַמִּית / אֶל הֵיכַל הַמֶּלֶךְ

בְּגָלוּתֵךְ הֵיךְ נִדְמֵית / כְּוֶרֶד בְּאַשְׁפָּה הֻשְׁלַךְ

וְרַבּוּ בַּעֲלֵי תַּרְמִית / וְצָר מַחְזִיק בְּפֶלֶךְ

וְאַהְבָתֵךְ עוֹלָמִית / וְאָנָה דּוֹדֵךְ הָלַךְ

אוֹתִי נָהַג וַיּוֹלֵךְ / רְדָפוּנִי הָרוֹבִים:


בְּעֶזְרָתוֹ אָקוּמָה / כִּי בִשְׁמוֹ בִטְחוֹנִי

אָקִיצָה מִתְּנוּמָה / וְיֵיחוּד שְׁמוֹ בִלְשׁוֹנִי

וְאֶתְפַּלֵּל וְאָצוּמָה / בְּיוֹם זֶה כִּפּוּר עֲווֹנִי

נַפְשִׁי בְכִסְאוֹ עֲלוּמָה / וְכָל שַׁחַר יַנְחֵנִי

בְּחַסְדּוֹ יַעֲמִידֵנִי / בְּאוֹר שֶׁמֶשׁ וְכוֹכָבִים:


זְעוּם הַלֵּב אַל תֵּרָדַם / קוּם קְרָא אֶל אֱלֹהֶיךָ

אֲשֶׁר חַסְדּוֹ לְךָ קָדַם / וְשָׂם שִׂכְלָךְ וְנַפְשֶׁךָ

בְּטִפַּת לוֹבֶן וְאוֹדֶם / הִעְמִיד בִּנְיָנֶךָ

וְרוּחוֹ תַּנְהִיגֶךָ / בְּתוֹךְ חוּצוֹת וּרְחוֹבִים:


יְדִידִים הִקָּבְצוּ / וְשׁוּבוּ מִדַּרְכֵיכֶם

וְלַעֲשׂוֹת הַטּוֹב חִפְצוּ / לְמַעַן יִרְבּוּ יְמֵיכֶם

מֵעָפָר הִתְנַפְּצוּ / וּפְקַדְתֶּם אֶת נְוֵיכֶם

בְּנֵי יַעֲקֹב אַשְׁרֵיכֶם / בְּמִצְווֹת וְחֻקִּים עֳרֵבִים:
