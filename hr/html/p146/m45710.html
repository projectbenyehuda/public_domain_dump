אֵל הַרוֹפֵא

חתימה: אלשבזי


אֵל הָרוֹפֵא לְכָל נִכְאָב / עֶזְרָתִי בְךָ צוּר עוֹשִׂי

בְּחַסְדְּךָ חֲמוֹל עַל דָּל נֶעְצָב / כִּי בַּדְּמָע אֶמְסֶה עַרְשִׂי

וְאַף גַּם זֹאת שֶׁאֲנִי נִתְעָב / בְּעָרֵי חִתִּי וִיבוּסִי

וַיְהִי יְוֹיָ לִי לְמִשְׂגָּב / וֵאלֹהַי לְצוּר מַחְסִי:


לוּלֵי חַסְדְּךָ לִי קָדַם / אֵיכָה תִשְׁקְטָה חַיָּתִי

גּוֹפִי בְחֶטְאוֹ יֵרָדֵם / וּתְכַסֵּהוּ כְּלִמָּתִי

טוֹב כִּי אָשׁוּבָה וְיִדַּם / שָׂטָן מְבַקֵּשׁ רָעָתִי

וּבַגּוֹיִם לֹא יִתְחַשָּׁב / עַם דָּתוֹתַי וְטַכְסִיסִי

וַיְהִי יְוֹיָ לִי לְמִשְׂגָּב / וֵאלֹהַי לְצוּר מַחְסִי:


שְׁעֵה שַׁוְעָתִי כְחַסְדְּךָ אֵל / אַל תִּזְכְּרָה יוֹם שָׂטִיתִי

וָאֶעֱזוֹב הֵיכָל וְהַרְאֵל / בְּעָשְׁרִי וְהוֹנִי תָעִיתִי

וַיִּמְשֹׁל עָלַי סְמָאֵל / כִּי מָרֹה מָרִיתִי

וְטוֹב לִי מִכֶּסֶף וְזָהָב / תּוֹרָה וְקַסְתִּי וְקוּלְמוֹסִי

וַיְהִי יְוֹיָ לִי לְמִשְׂגָּב / וֵאלֹהַי לְצוּר מַחְסִי:


בְּשִׁמְךָ עַתָּה בָּטַחְתִּי / אַתָּה מָגִנִּי בְּצָרָה

וְאֶת כָּל הֲבָלַי זָנַחְתִּי / וָאֶשְׁכְּנָה בְאֶרֶץ זָרָה

עוֹל בְּנֵי נֵכָר סָבַלְתִּי / עָלַי נוֹגְשִׂים וּמִשְׁמָרָה

בַּשֵּׂר נָא דֶרֶךְ כּוֹכָב / מִיַּעֲקֹב הוּא מְנָת כּוֹסִי

וַיְהִי יְוֹיָ לִי לְמִשְׂגָּב / וֵאלֹהַי לְצוּר מַחְסִי:


זְכוּת אָבוֹת תַּגִּין עָלַי / יָסוּר כָּל מִכְשׁוֹל וְעָווֹן

וּרְצֵה תְשׁוּבָתִי, אוּלַי / יֻצַּל עַבְדְּךָ מִדָּווֹן

בְּשׁוּבִי לֹא יִגַּשׁ אֵלַי / דּוּמָה וְשַׂר טִיט הַיָוֵן

וּבוֹטֵחַ בַּיוֹיָ יְשֻׂגָּב / אַתָּה עָזִּי וּמָנוּסִי

וַיְהִי יְוֹיָ לִי לְמִשְׂגָּב / וֵאלֹהַי לְצוּר מַחְסִי:


יָחִיד בְּךָ אֲנִי בוֹטֵחַ / אַתָּה מָעוֹז לַדַּלִּים

אַתָּה מוֹחֵל וְסוֹלֵחַ / רָם עַל שְׂרָפִים וְחַשְׁמַלִּים

שְׁעֵה תְפִלָּתִי כְּשַׁי רֵיחַ / וְאָזְנְךָ מִנִּי אַל תַּעְלִים

בַּעֲבוּר שׁוֹמְרֵי עֵקֶב רָב / סְלַח נָא לְעָווֹן נִכְמָסִי

וַיְהִי יְוֹיָ לִי לְמִשְׂגָּב / וֵאלֹהַי לְצוּר מַחְסִי:
