אֱוִיל מַבְאִישׁ

חתימה: אלשבזי


אֱוִיל מַבְאִישׁ / הֱיֵה נָא אִישׁ / נַחְפְּשָׂה אֶת דְּרָכֵינוּ

וְנִתְכּוֹנֵן / לְהִתְבּוֹנֵן / לַעֲשׂוֹת רְצוֹן אֱלֹהֵינוּ

בְּזֶה הַיּוֹם / יְהֶא פִדְיוֹם / לְרַפֹּאת מַחֲלָתֵנוּ

וְהַשָּׁבִים / נִצָּבִים / יְהֶא שָׁלוֹם בְּעִירֵנוּ

דְּעוּ וּרְאוּ / וְהִמָּצְאוּ / בְּנֵי יַעְקֹב שְׁבָטֵינוּ

וְנִתְוַדֶּה / לְצוּר פּוֹדֶה / וְיִמְחֶה אֶת פְּשָׁעֵינוּ

יָשׁוּב יְרַחֲמֵנוּ / יִכְבּוֹשׁ עֲווֹנוֹתֵינוּ:


לְבַב נִדְכֶָּה / אֲשֶׁר הֻכָּה / בְּיִסּוּרִין וְנִתְמָרַק

וְיִטָּהֵר / וְיִזָּהֵר / בְּמֵי חֶסֶד אֲשֶׁר זוֹרַק

וְיִתְיַסֵּד / בְּלִי הֶפְסֵד / וְעוֹל יִצְרוֹ יִתְפָּרַק

בְּשׁוּב נַפְשׁוֹ / אֱלֵי שָׁרְשׁוֹ / וְאָז יִזְכֶּה בְּטוּב חֵלֶק

וְזֶה מִשְׁפַּט עֲדָתֵנוּ / בְּיוֹם צוּר יְקִימֵנוּ:


שְׁבוּ בָּנִים / מְזֻמָּנִים / דְּעוּ יוֹם דִּין וְהַחֶשְׁבּוֹן

וְהַמּוֹדֶה / הוּא יִפְדֶּה / אֶת נַפְשׁוֹ בְּעֵרָבוֹן

בְּיוֹם יַעְזוֹב / שְׂמֹאל כּוֹזֵב / וְיִרְצֶה בּוֹ הָרִבּוֹן

שְׁהוּא חוֹפֵשׂ דְּרָכֵינוּ / וְגוֹלֶה מִצְפְּנוֹתֵינוּ:


בְּיוֹם עָשׂוֹר / תִּתְבַּשֵּׂר / אֲיֻמָּתִי בִּמְחִילָה

וְתָכִינָה / הַבִּינָה / לְדַלְתֵי שַׁעֲרֵי גִּילָה

וְאֵם כָּל חַי / לְכָל אַחַי / תְּהוֹסִיף חֵן וְתַשְׂכִּילָה

וְזִיו תַּנְהִיר / בְּאוֹר בָּהִיר / לְשִׁבְטֵי יָהּ וְתַגְבִּילָה

וְתַגִּיהַּ לְחָשְׁכֵּנוּ / וְתוֹסִיף אֶת הֲדָרֵנוּ:


זְרִיז שֵׂכֶל / אֲשֶׁר יוּכַל / לְטַהֵר מַחְשְׁבוֹת לִבּוֹ

בְּצַד יָמִין / יְהֶא מַזְמִין / שְׂמֹאל יִמְאַס וְיִגְעַר בּוֹ

וְיִמָּצֵא / כָּל חֶפְצוֹ / בְּעֵת יַקְבִּיל פְּנֵי רִבּוֹ

וְיִתְיַשֵּׁר / בְּיוֹם הֶכְשֵׁר / לְמוּל מֶלֶךְ בִּמְסִבּוֹ

וְשָׁם מֹשֶׁה נְבִיאֵנוּ / וְגַם שִׁבְעִים זְקֵנֵינוּ:


יְשׁוּרוּן רוֹן / וְלָךְ סִבְרוֹן / לְהִתְוַדּוֹת בִּתְשׁוּבָה

בְּצֵל שַׁדַּי / אֱזוֹן עָדַי / וְחֵפֶץ נַפְשְׁךָ יוּבָא

וְהִתְפַּלֵּל / בְּיוֹם וָלֵיל / וְתִתְרַפֵּא מִמְּשׁוּבָה

וְחֵן תִּרְאֶה / וְלֹא תִגְאֶה / וְאָז נַפְשְׁךָ תְּהֶא אֲהוּבָה

וְחֶסֶד אֵל יְלַוֵּנוּ / וְלַחַיִּים יִכְתְּבֵנוּ:
