שׁוּר יָהּ מִמְּעוֹן קָדְשֶׁךָ

חתימה: שבזי


שׁוּר יָהּ מִמְּעוֹן קָדְשֶׁךָ / שֵׂיוֹת אוֹבְדוֹת תְּקַוֶּךָ

הָחִישׁ יְשׁוּעָה לְבָנֶיךָ / וּבְנֵה לְהֵיכַל קָדְשֶׁךָ

אָז נִמְצְאָה חֵן בְּעֵינֶיךָ / בְּיוֹם יִכְלוּ כָּל עוֹזְבֶיךָ

קוּם יְהֹוָה וְיָפוּצוּ אוֹיְבֶיךָ:


לִקְרָאת אֱלֹהֶיךָ הִכּוֹן / זֶרַע יְשׁוּרוּן בְּעוֹד לַיְלָה

רַב לָךְ בְּגָלוּת זְמָן תִּשְׁכּוֹן / זֶה עֵת תְּשׁוּבָה וְיוֹם חֶדְלָה

מִלַּחְמְךָ פַּרְנְסָה מִסְכֵּן / לְעֵירוֹם מְשׁוֹטֵט תְּנָה שִׂמְלָה

יִהְיֶה יְדִידִי לְךָ חֶמְלָה / וְיָגֵן יְוֹיָ עָלֶיךָ:


מַהֵר וְשׁוּבָה בְּרֹאשׁ תִּשְׁרִי / יָמִים עֲשָׂרָה מְזֻמָּנִים

כִּי זֶה הֲדָרִי וְכָל יִשְׁרִי / נִתָּן לְעָם זַךְ וְנֶאֱמָנִים

כִּי בוֹ יַעֲלֶה לְטוֹב זִכְרִי / חֶבְרַת שְׁבָטַי מְכֻוָּנִים

וּבְיוֹם עֲשִׂירִי מִתְעַנִּים / צוּרִי סְלַח נָא לְעַמֶּךָ:


שַׁחַר אֲחַלֶּה פְּנֵי אֵל חַי / יַאְזִין לְקוֹלִי וְשַׁוְעָתִי

הִתְעוֹרְרוּ מַהֲרוּ אֶחָי / שִׁבְטֵי יַעֲקֹב סְגֻלָּתִי

נַפְרִיד מְפַתַּי וּמַדִּיחָי / הַתּוֹאֲבִים לַחֲמוֹד דָּתִי

צוּרִי רְצֵה נָא עֲזוֹר אוֹתִי / קַבֵּל רִנַּת עַמֶּךָ:


בִּזְכוּת חֲסִידִים וְצַדִּיקִים / וּבִזְכוּת יוֹנְקֵי שָׁדַיִם

בְּרִית אָבוֹת צוּרִי יָקִים / לְעַם דָּל בֵּין הַמִּשְׁפְּתַיִם

וְיִרְאֶה דִּמְעַת הָעֲשׁוּקִים / הַדּוֹלְפָה עַל לְחָיַיִם

יְרַחֵם מְחוֹלַת מַחֲנָיִים / וִינַחֲמָה אֵם בְּמַר בּוֹכָה:


זָכְרָה בְרִיתְךָ בְּיוֹם דִּבֶּר / מֹשֶׁה לְהַצִּיל אֲבוֹתֵינוּ

עָלָה וְנִהֲגָם וְיָם עָבַר / נִתְחַבְּרוּ בוֹ שְׁבָטֵינוּ

פַּרְעֹה וְחֵילוֹ כְּלָל נִשְׁבַּר / וְיִכְָּלְמוּ כָּל מְשַׂנְאֵינוּ

אֵל חַי כְּקֶדֶם תְּשִׁיבֵנוּ / כִּי קְרוֹבָה יְשׁוּעָתֶךָ:


יַעֲלֶה שַׁוְעִי עֲדֵי מָרוֹם / יַאֲזִין אֱלֹהַי וְצוּר מַחְסִי

יִסְלַח עֲווֹנִי שְׁהוּא גּוֹרֵם / עָלַי לְמִרְיִי וְרוֹב כַּעְסִי

מִיָּם וְצָפוֹן וּמִדָּרוֹם / וּמִזְרָח בְּחֵן יַעֲלֶה נִסִּי

יְהֹוָה עֻזִּי וּמְנוּסִי / קַבֵּץ פְּזוּרַי בַּחֲסְדֶךָ:
