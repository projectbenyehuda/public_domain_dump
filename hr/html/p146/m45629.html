אוֹדֶה לָאֵל וְאָבוֹא

חתימה: אלשבזי


אוֹדֶה לָאֵל וְאָבוֹא / בִּתְפִלָּתִי וְשַׁוְעִי

כִּי לִבִּי בְּמַכְאֹבוֹ / גָּרַם חוֹבִי וְנִגְעִי

עֲדֵי נִגְעַלְתִּי בְחוֹבוֹ / וַיִּתְפְּשֵׂנִי פִשְׁעִי

חֶמְלַת יְוֹיָ וְטוּבוֹ / יִמְחֶה חֲטָאַי וְרִשְׁעִי

אֱלֹהֵי צוּרִי אֶחֱסֶה בּוֹ / מָגִנִּי וְקֶרֶן יִשְׁעִי:


לָמָּה צְעָדַי קָצְרוּ / וָאֶתְבּוֹדֵד בְּגָלוּת

וְאוֹיְבַי עָלַי חָבְרוּ / מֵרוֹב עוֹנִי וְדַלּוּת

כִּי הָעֲווֹנוֹת גָּבְרוּ / וְסָבְבוּ עָלַי בְּקַלּוּת

וַאֲחָזַתְנִי עַצְלוּת / וְעָזַבְתִּי מוֹשִׁיעִי

אֱלֹהֵי צוּרִי אֶחֱסֶה בּוֹ / מָגִנִּי וְקֶרֶן יִשְׁעִי:


שׁוּבִי יוֹם זֶה יְחִידָה / וְאַל יַשִּׂיאֵךְ סוּרֵךְ

וְחוּלִי כַּיּוֹלֵדָה / וְהִכּוֹנִי תוֹךְ חַדְרֵךְ

וְהָכִינִי לָךְ צֵידָה / אוּלַי זְמַנֵּךְ יִקְרֵךְ

רַב לָךְ מִתְבּוֹדֵדָה / בְּעֵמֶק גּוֹלֶם יִצְרֵךְ

וְאִם תָּשׁוּבִי אַשְׁרֵךְ / חָמַד שִׂכְלִי בְּמַסָּעִי

אֱלֹהֵי צוּרִי אֶחֱסֶה בּוֹ / מָגִנִּי וְקֶרֶן יִשְׁעִי:


בַּעֲווֹן חַטֹּאת נְעוּרַי / אֱלֹהַי אַל תְּבִישֵׁנִי

רַבִּים רוֹדְפַי וְצָרַי / וַאֲנִי בְךָ בִטְחוֹנִי

הֲלָמוּנִי יִסּוּרַי / בְּרוֹב דַּלּוּת וּבְעוֹנִי

פְּדֵנִי מִתַּמְרוּרַי / וְאַחֲרֶיךָ מָשְׁכֵנִי

אַתָּה עוֹז מִבְצָרִי / מְצוּדָתִי וְגַם סַלְעִי

אֱלֹהֵי צוּרִי אֶחֱסֶה בּוֹ / מָגִנִּי וְקֶרֶן יִשְׁעִי:


זָנַחְתִּי כָּל טוֹבָה / וְשִׁמְךָ לֹא זָנַחְתִּי

צוֹפֶה דָבָר שֶׁנִּבָּא / כִּי שֶׁמֶץ שָׁמַעְתִּי

יְשׁוּעָה לָבוֹא קְרוֹבָה / לְהָשִׁיב אֲשֶׁר עָזַבְתִּי

וְאִם הִגְדַּלְתִּי חוֹבָה / מֵחוֹבִי נִחַמְתִּי

אֶת יְוֹיָ זָכַרְתִּי / וְאֵלָיו אֶשָּׂא רֵעִי

אֱלֹהֵי צוּרִי אֶחֱסֶה בּוֹ / מָגִנִּי וְקֶרֶן יִשְׁעִי:


יְזַלֵּף עָלַי חַסְדָּךְ / כִּי אֵל רַחוּם אַתָּה

וְיָסוּר מִנִּי פַחְדָּךְ / וְאָשׁוּבָה מֵעַתָּה

וְאַתָּה מָעוֹז לַדָּךְ / וּמַחֲסֶה לְעָם גָּאַלְתָּ

וְכָל תַּאֲוָתִי נֶגְדָּךְ / וְסוֹדִי יָהּ חָקַרְתָּ

קַצְווֹת עוֹלָם שָׁקַלְתָּ / בְּכַפּוֹת קָו אֶמְצָעִי

אֱלֹהֵי צוּרִי אֶחֱסֶה בּוֹ / מָגִנִּי וְקֶרֶן יִשְׁעִי:
