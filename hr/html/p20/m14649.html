יַעְלֹז יְשֻׁרוּן

יַעְלֹז יְשֻׁרוּן בְּהִתְחוֹלֲלוֹ / לָאֵל, כְּשָׂכִיר יְקַו פָּעֳלוֹ.


אָמוּן מְצַעֵק בְּעֹל סֻבֳּלוֹ / כִּי גוֹעֲלוֹ כֹּה וָכֹה טִלְטְלוֹ,

יָכִין לְבָבוֹ בְּהִתְפַּלֲלוֹ / אֶל אֵל וְיִזְכֹּר אֲשֶׁר יִחֲלוֹ –

מִכּוּר בְּצֹעַן הֲלֹא נִטְּלוֹ, / לִמְחוֹז חֲפָצָיו בְּעֹז נִהֲלוֹ.


בֶּן הוֹר מְיַסֵּר עֲלֵי מַעֲלוֹ – / יוֹדֶה חֲטָאָיו בְּהִתְנַפְּלוֹ

לִפְנֵי אֲדֹנָי אֲשֶׁר חוֹלֲלוֹ, / וַיְלַמְּדוֹ דָת וְיֵשׁ נִחֲלוֹ –

כִּי הוּא לְבַדּוֹ אָהֵבוֹ וְלוֹ / חֻקּוֹ נְתָנוֹ וְהוּא קִבְּלוֹ.


רָם בִּי מְשַׂנְאִי בְּהִתְהַלֲלוֹ, / יָבוּז לְעַמִּי בְּהִסְתּוֹלֲלוֹ.

יִדֹּם כְּהֵלֶךְ בְּיַד שׁוֹלֲלוֹ, / אַרְיֵה טְרָפוֹ וְדֹב שִׁכְּלוֹ.

הִשְׁחִית זְבֻלּוֹ בְּאַפּוֹ וְלֹא / רִחֵם עֲלֵי כֵּס יְקָר, נִבְּלוֹ.


מַה יַּחֲשֹׁב צַר בְּהִשְׁתּוֹלֲלוֹ? / יִכְלֶה וְיֹאבַד כְּמוֹ גֶלְלוֹ,

בַּעְוֹן אֲשֶׁר חִלֵּל פְּאֵר כִּלֲלוֹ / קָדוֹשׁ; אֲדֹנָי, אֲשֶׁר חִלֲלוֹ –

יָשִׁיב מְהֵרָה שְׁבוּת אָהֳלוֹ, / יַקְרִיב לְמוֹעֲדוֹ וּבָא גוֹאֲלוֹ.
