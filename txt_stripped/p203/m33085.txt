


	
	תוכחה [מה נפש נוכחת]
	



סימן: א"ב ויצחק בבית האחרון

יב


         מה נפש נוכחת / תשיב ביום תוכחת / והנושה בא לקחת

         איכה ערף מקשה / ומהתרפס מחשה / איש אשר לו נושה

         בני ימים לרגעיהם / לנושיהם מקריהם / נושים בהם

         גם ממעל ותחת / תוכם משתלחת / צרעת פורחת

5        מה נפש נוכחת…





         דבקו בציץ התלמו / ובתוהו ימי הבלמו / ישיא מות עלימו

         הוד בעלים ורכושו / יקרא וחיל כחשו / נשיתי אתכם נשוא

         ונע משובה ונחת / פתע לשאל יחת / ויפל אל הפחת

         מה נפש נוכחת…





         זחלי נפשי למריך / בקרא ליך ולגויך / באי ושלמי את נשיך

         חקקי לבך בעודו / נגלו וכתבם בסודו / כל בעל משה ידו

         טוב ידיך ומנחת / שוא צררו באמתחת / הנה נשחת

         מה נפש נוכחת…





         יד איש תכחישהו / ויחיש לענות פיהו / אשר ישה ברעהו



15          כן חובתו רשומה / ועלי מצחו חתומה / משאת מאומה



         לעולמה זונחת / ובזה לא מונחת / נמצאת מכאן ומכאן קרחת

         מה נפש נוכחת…





         מי יתן בעוד טובי / בטרם לארץ שובי / לא נשיתי ולא נשו בי

         נואלתי כאיש בוזה / ודבקתי באוחזי / את המשא הזה



20       ספות על חם קדחת / ובשאת או ספחת / כי יהיה בקרחת



         מה נפש נוכחת…







         ענן סכלות עיניך / כסה ורעיוניך / כי ישה לך אלה מעוניך

         פנות אל חטא בקרותו / בלי הבין אחריתו / נשא בו אלה להאלותו

         אל צרור חטאת צורחת / להתחבר לנשחת / אחת אל אחת

25       מה נפש נוכחת…





         קרבו למשפט לריבו / ירשיעו פיו וניבו / האיש אשר אתה נושה בו

         רועה בהתיצב ליד / מעגל והיה ציד / משא כל יד

         שפטוך כעיר הנדחת / בלהבת קודחת / ואש מתלקחת

         מה נפש נוכחת…





         תום תם לאין תום תמשה / לנושה בו כי תשה / לא תהיה לו כנושה



30       יד צדק הן נטויה / קדם לך היא גלויה / צדקתך בארץ נשיה



         אמתך פדה משחת / כי לך מתוכחת / אנכי בורחת

         ובכן מה רבו מעשיך יי.
