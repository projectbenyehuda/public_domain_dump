


	
	תוכחה
	



סימן: אלף־בית

מז






   מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי





   איכה בשר ודם / יעמד פני כל קדם / מוכיח אדם

   בשומם נגד פניהם / צפוני תעלומיהם / ויוכח עליהם

   גמולם יקים נוכח / ויראה זכור ונשכח / אלהי אבותימו ויוכח

   מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי



135    דבקת עפר שחה / נפש לדין לקחה / תהיה ביום תוכחה



   הלא בחזותה פתוחות / מגלות מדברי צחות / אין בפיה תוכחות

   ובעת מרום צוחה / תקרא אנשי שוחה / לכו נא ונוכחה





   מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי

   זמן בא לראות ישע / לאיש אמת ועל פשע / מוכיח לרשע



140    חלתי כי שמעתי / גושו לריב אתי / שמעו נא תוכחתי



   טמן חטא מצפוניך / אז אגלה ועל פניך / אוכיחך ואערכה לעיניך

   מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי

   יד בדמים מלאתם / ומעצתי נטיתם / ותוכחתי לא אביתם

   כמה אטמו תנוכם / ואזנם מבוא בתוכם / הוכח לחכם



145    לכו נערך דיננו / למוכיחים בינינו / ויוכיחו בין שנינו



   מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי

   מי זה פיהו ירים / לפני דן כל יצורים / והוכח בדברים

   נענה איכה יכון / לעומת כל מכון / הוכח בדבר לא יסכן

   סכלי פעלים השחוך / נפשי ובשאול הנחוך / ומשובותיך תוכיחוך



150    מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי



   עמדי וצפי ציגתו / ועת למשפט שבתו / אל תקוץ בתוכחתו

   פניו חלי בשיח / כי כל נע יניח / אשר יאהב יי יוכיח

   צור הנודע בשמו / איך אנוש גבר מומו / ישר נוכח עמו

   מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי



155    קורא בחיל ודאבון / בדין מיסר כסיל ונבון / בתוכחות על עון



   ראה שיחי ונחני / בצדקה והניחני / ואל באפך תוכיחני

   תראה בעמדי נגדך / כי בחנותך וחסדך / הוכחת לעבדך

   מאד חלתי וזחלתי / לתוכחת לקחתי / מה אשיב על תוכחתי
