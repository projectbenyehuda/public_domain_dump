


	
	אספר גדולות
	








אספר גדולות / עושה גדולות / שח ויפעל / מעשיו לעולם



אמץ דוק לכס / ותבל להדום / אור גלה ליום / ואפל החביא לנשף.





ברר רקיע / לעמוס פלגי מים / ויתרם הניס לנבך / ולהדשיא שיח ועשב



ביהק עולם / במושלי יום ולילה / היות לפקד ימים / ולרגעי עתים.





גאון בריחי תהום / / ודגת קשקשת / רמש ובני רשף / השריץ במי שועל



גם באדמת חלד / עש יצורים לרוב / והעלה ללב עשות / אדם לרדות בכל.





דמהו בצלם / העטהו כבוד / והניחו בחפת יקר / והורהו צו לבוחן



דבק לו עזר / מנו ולא זרה / הסיתה זוחל / ורשת פרסה לעפר.





[הפסחו פעמיו / וטרפו עפר / היא] לעצב והוא ליגע / והכל לרקב



[הרגלו ליצר / ושבו לתשוקה] / וצאצאיהם / עובד ורועה.





[והובילו שי / לבוחן לבבות / וש]ע ברך וקץ ברב / ושיחת אח ברצח



וזעק דמו לאל / ונענש הורג / ונתודה פשע / ונתמגן באות.





זרע דור שלישי / לצלם הוחל קרוא בשם / ונוקם לצריו / החל קרוא לים



זדו אומרי לאל סור / עד נמחו במים / ומצפוני בסך / קמו משפחות חלד.





חשבו בוגדים / עלות לעב ונפצו / ואב מעיר בצדק / בסודם לא בא



חבר פושעים / לחי עולמים / ונסהו זה בעשר / עשות בן שי ולא אט.





טלה השלם לחרב / וליקוד אש / יה יראהו אפר / עת תצר לנו



טובו משכנות תם / ובם נסתופף / מצוא בכל־עת / צורו למשגב.





 יקר שמו למעלה / וכיחש לו שר / ושבר לפניו / חרב ורשפי קשת



יופי מטות עוז / צצו מחלציו / בם למלך ולחוזה / לשרת ולכהן.





כבירי שבט / יעלסו בגורל / איש כוח לבית אב / סגן כחוק מלואים



כיהון ילמדוהו / תחכמוני שבת / להכין ביום זה / עתים לכל־פועל.





לדם לרקח לנרות / להגיש בראש ורגל / הג סדר זבחים / ומתודה בערב צום



לעת ערב ירעב / פן יישן וייקר / ונשבע לשרי מטהו / פן יהתל.





מכים באצבע / משוררים לעוררו / עד נחלק לילה / ואספו ברורים



מצביעים פיס / מי ירים דשן / שני לערוך / שלישי לקטרת.





נמו: ברק נוגה, / ופשט וטבל / ועטה שמונה / וקדש עשות כבש



נטל דם וזרק / והקטיר והיטיב / והעלה ראש ונתח / מנחה חבתים ונסך.





סר לבית פרוה / וקדש וטבל / ועטה בדים / ועוד נתקדש



שש כגבור לארח / בגשתו אצל פרו / וסמך שתי ידיו / עליו ומתודה.





וכך היה אומר: אנא השם חטאתי עויתי
פשעתי לפניך אני וביתי, אנא השם כפר נא סלח
נא על החטאים על העוונות על הפשעים שחטאתי
שעויתי שפשעתי לפניך אני וביתי ככתוב
בתורת משה עבדך כי ביום הזה יכפר עליכם לטהר
אתכם מכל חטאתיכם לפני השם. והכהנים והעם
העומדים בעזרה בזמן שהם שומעין את־השם היוצא
מפי כהן גדול בקדשה ובטהרה היו כורעים
ומשתחוים ונופלין על פניהם ואומרים ברוך שם
כבוד מלכותו לעולם ועד. אף הוא היה מכון לגמור
את־השם אחר המברכים ואומר להם: תטהרו.





עד שער היסוד בא / ושם שעירי עם / והגריל עליהם / לשם ולגזרה



עתדם בשני / זה לטבח וזה לדחי /



[ונגש ש]ניה אצל פרו / להתודות בעד עצמו



ומטהו.





וסומך שתי ידיו עליו וכך היה אומר
אנא השם חטאתי עויתי ופשעתי לפניך אני וביתי,
אנא בשם כפר־נא לחטאים ולעוונות ועל הפשעים
שחטאתי ושעויתי ושפשעתי לפניך אני וביתי
ככתוב בתורת משה עבדך כי ביום הזה יכפר
עליכם לטהר אתכם מכל חטאתיכם לפני יי. והכהנים
והעם, העומדים בעזרה כשהיו שומעים את־השם
הנכבד והנורא מפורש יוצא מפי כהן גדול
בקדשה ובטהרה היו כורעים ומשתחוים ונופלין
על־פניהם ואומרים ברוך שם כבוד מלכותו לעולם
ועד. ואף הוא היה מכון לגמור את השם כנגד
המברכים ואומר להם: תטהרו. ואתה בטובך הגדול
תעורר רחמיך וסלח לאיש חסידך.





פנה לטבחו / ודם ימרס פרח / והוא במחתת אודם / חת גחלי אש



פ[ס]ע וחפן / ממגס לתוך כף / ונטרה בשמאל / וקם בין בדים.





צבר כלי קטרת / ונתעשן בית / ושוע בהיכל / בקצר בית זבול



צלח קחת דם / ושב למקומו / והצליף אחת למעלה / ושבע למטה במספר.





וכך היה מונה אחת אחת ואחת אחת ושתים וגמריה





קדם וזבח שעיר עם / ועשה לו כדם פר / וכן משניהם יזה / בהיכל על הכפרת



קרב למזבח פז / וחטאו בדם תערבת / וארבע סביב / ושבע על טהרו.





וכך היה מונה

ואחר כך בא לו אצל שעיר המשתלח



וסמך שתי ידיו עליו ומת׳ עוון עמך ונחלתך יע׳
ובהתודותו מתודה וכך הוא אומר: אנה השם חטאו נ׳.





רגיל משלחו לצוק / ובא להקטיר חלב / חניכיו ישרפו בחוץ / עור ובשר ופרש



רץ ושינן כסדר / וקדש וטבל / ועטה שמנה / ועוד נתקדש.





שי יעש עולות / ופשט וטבל / ועטה בדים / ועוד נתקדש



שובב כלי קטרת / וקדש וטבל / ועטה שמונה / ועוד נתקדש.





תמיד לניחח / הגיש לעת ערב / ותמר מרקחת / בין נתחים לנסך



תכלית כל־פועל / הטיב והדליק נרות / וקדש עשירית / ועטה כלי הון.





תרנן עדה / לקראת ציר אמת / כי חפץ יוצרם / בידו הצליח



תמימים ישרים / לנוהו ילווהו / יחד ישמחו בטוב / בצאתו בשלום בלא פגע.
