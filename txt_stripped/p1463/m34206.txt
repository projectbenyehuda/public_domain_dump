


	
	אתן תהלה
	








אתן תהלה / לאל המהלל / אספרה ביראה / מעט ממעשיו:

אלוה מעולם / ועד לא עולם / לפניו ואחריו / לא נוצר אל.





בלבדו הוא / ואין אתו זר / אומר ועושה / ואיש לא נעדר

באחד ישפט / ואין בו עולה, / אמת אמרתו / ועל שפתיו חסד.





גבור בכח / נושא עוון / גדול העצה / ועובר על פשע

גולה עמוקות / מני חשך, / יושב בסתר / ורואה כל סתר.





דעת ותבונה / יוצאות מפיו / ועיניו משוטטות / ועין לא תשורנו

דברו המשל / וממשלתו סלה / מלא כל־הארץ כבודו / ורום לא יכילנו.





הדור בלבושו / הוד תפארתו / הדר עזו / כשמו גבורתו

האל תמים / צור עולמים; / הכל יעבורון / והוא לבד יעמד.





ועד לא רקע / תבל על תהו / ומקולות מים / אדר ישמיעו

ומרום שבתו / והוא רם על כל / ומביט אל עני / ונכה רוח.





זרתו תכנה / שמים בתבונה / מאורות וכוכבים / מעשה אצבעותיו

זמם במחשבת / ותקם ותעש / יועץ בלבו / ותעמד עצתו.





חסיד במעשיו / ורחום למנוסיו / צדיק בדרכיו / וקרוב לקוראיו

חפץ להצדיק / יצורי כפיו, / תמיד ישקד / בטובת בחיריו.





טהור עינים / רוצה בתשובה, / מרבה לזכות / להשיב אפו

טובו בכל־דור / מדבר בצדקה / פאר ראשו / כובע ישועה.





ימינו פתוחה / לכל שבי פשע, / אומר בכל־עת: / שובו אלי ואשובה

יצר מחמר / צלם דמותו / רחמיו על כל־מעשיו / יודע יצרנו.





כלו כבוד / משגיח בדכא / מאזין כל־לחש / רנת לשון

כל־צבא השמים / משתחוים לו / שמש וירח / יכרעו לפניו.





לא ינטה קו / כנגד יצוריו / בשבתו למשפט / בריב לא יעמיק

למענו פעל / מפעלות עולם / קורא דורות / מי יערוך אליו.





ממעמקי לב / וסתרי כליות / מבין מחשבות כל־בני אדם

מאון ומעמל / יסתיר פניו / מראות ברע / יעלים עיניו.





נוגה סביביו / ושת סתרו חשך / כי לא יראהו / האדם וחי

נורא עלילה! / השב אנוש עד דכא / עד כי ישוב / ויאריך ימים.





שגיא כח / לובש צדקה / מי אל כמהו / ארך אפים

שונא ומתעב / גזל בעולה / אוהב טהר לב / שמוע מזבח טוב.





עוצר בגערתו / ומחריב ימים / צורר כשמלה / כל־מי בראשית

עונה בחסדו / מקרני ראמים / מוחה עוונות / כעב וכענן.





פעל ועשה הכל / בדבור פיו / כי כל־דרכיו / צדק ומשפט

פיו יאמר: / זאת עשו וחיו / כי לא אחפץ / במות הרשע.





צופה רחוקות / נשא ונורא / מרומם כסאו / ושפל יראה.

צועה ברוב כחו / ודואה בכנפי רוח, / בוחן לבות / ואין לפניו שכחה.





קורא בקול רם: / אני אל שדי, / שובו עדי / ואשובה אליכם

קרבו עדיו / בכח התשובה, / ואזניו קשובות / לקול זעקת דל.





רחום וחנון, / שפטינו ברחמים / צדקנו במשפט / כי אנחנו עפר ואפר

רם בעליונים / מלך בשחקים / שומע ממרומים / זעקת אביונים.





שמים כסאו / והארץ הדום / צדקה שריונו / ומאור שמלתו

שמו נורא מאד / חנון ורחום / בכל דור ודור / היא צדקתו.





תמים דרכו / ואין לפניו עול / בכל־בקר יתן / לאור משפטו

תוכן לבות / רב העליליה / מי ימלל / גבורות יי.
