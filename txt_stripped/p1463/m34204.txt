


	
	אכרעה ואברכה
	








אכרעה ואברכה / אען לשון רכה / אקומה ל…. / אתיצב כשפל ודכה.

בפיפיות מישרים / ב…… / בשפתות מישרים / בחן עשרה הסדרים.

ג….. אפלל / גאות לא אמלל / גרוני קול יהולל / ג… ועולל.

דעת יקר / דבר גדולות יקר / דוק…. / [אז?] בדעת חקר.

הציב רקיע / הכפיף והרקיע / הבהיק והבקיע / הוא יסד והשקיע

ואברך במקהלות / ואביע תהלות / ולתוכן עלילות / אספר גדולות.

זכר נפלאות / זבד להנאות / חשות קץ הפלאות / חי זכרה לי לטובה אות.

טוב אילות / טע פעמי כאילות / טעמיני בשבעה קולות / אמץ גדולות.

יקר בצרחי / יחי סלה רוחי / יגל במוסר רוחי / יערב עליו שיחי.

כהביט אל / כהתודותי ישע אל / כשפל אאשר לאל / ארומם לאל.

לבי אפנה / לרחמני יפנה / לערך שוע במענה / טרם אקרא ויענה

מלולי הורות / מפי במהמורות / מבשר טוב בתוך חבורות / אזכיר גבורות.

נעם אקרב / נגינותי יערב / נוראות מלך רב / אבשר בקהל רב.

שיר אסלסלה / שמחה וגילה / סוד רב עלילה / אזכר סלה.

עוד אלבלב / ערוך בפה [ובלב / ע]… נותן לב / לאדם מערכי לב.

פתחתי ביניתה / פנינות גוף הכנ[תה / פ…] קרבי בחנתה / אתה כוננתה.

צרפתה לבי / צור בחנתה קרבי / צפנתה לך אבי / רוח יי דבר בי.

קולות יחונינו / קוה אל יכוננו / קרבת אלהים בתוכינו / אדיר אדונינו.

רצה קולי / ראותך פלולי / רם יי חילי / אדברה וירוח לי.

שעה בדברי מישרים / שלום שפתי פרים / שופכים חדשים לבקרים / אדיר באדירים. תכנת כל / תומך ומושל בכל / תחבולותיך לרחם כל / תוכן כל.

תוכן וגם יכול / תכנתה במדה כל / תושיע עולמים לשכל / אז באין כל.

יחד תהלות אדבר / ערוך….. בר / [ק]ול לא ארחיש מלדבר / בעמדי … בר.
