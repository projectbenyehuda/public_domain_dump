


	
	אנוסה לעזרה
	








אנוסה לעזרה / אמצא נגדי / אל קרוב לי / בעת קראי בקול

אשר בעדת אל / בקרבי נצב / ופה במקדש מעט / אצפצף לו בקול.





בקרני דרשיני / שה פזורה אני / נגזזתי ונאלמתי / בלי להרים קול

באמור גוזזי: / נדחה היא / שומרה וצלה / לא ישאג קול.





גלתי שיח בחקיו / וחכי ערב / הטה אזן ושח: / השמיעיני קול.

גז וברח מני / כעפר על הרי בתר / בבקשו דת ואות / במשכנותי ואין קול.





דלג מבתר לבתר / השיבהו אלי / אולי ישא פניך / עקב שמעך בקול

דרוש טובה למו / וראה שה מוריה / אלם פיהו יהי צדק / ללא שמעה בקול.





הסר מחלק / ידים שעירות / הוגה בתחנונים / כי לך הקול

הושע כי לא / תשכח עדות / מפי זרעו / לא יסוף קול.





וחוזי ומליצי / הם בני אמי / ואז ניחרו בי / למען אשמע קול

ועל משמרתם / יעמדו ויזעקו / וגל להם סוד / והוא יענם בקול.





זה חמק מני / אסובב ואבקשנו / בכל-מקום הוא / אנה אשא קול

זכר דודי למטה / אדיר במרום / מלא כל-הארץ כבודו / למעלה קוראים קול.





חילי מים / אשר בם שבילו / דברו: תמונה לא ראינו / זולתי קול

חפשתי ישימון / היש, ואמר: אין, / קדם תתו עוז / בעתתני בקול.





טהור דלג / הרים ועבר / וממעו הר מור / אז נתן קול

טמאתי ידידות / שכנו ועלה / ליום כסא יבוא / באזני קול.





יקרתי בעיניו / ונלוה לי בשבי / עמו אנכי / אז הבטיחני בקול

ירד בשנער / ושם כס בעילם / והשמיע כארי / וכנחש קול.





כלה מני דוב / כהתרפא ממלאכת / חק במכתב / ויעבר קול

כבש לי / ארבעה ראשי נמר / וגם אני בהודיות סלה / אשמיע לו בקול.





לחית קנה / אז מכר ארץ / ומי לי בשמים / אז הרימה קול

לאלהי ישעי / משני ברזל שועתי / ומרגל עב טיט / הצרחתי קול.





מדת קצי / לא הודיעני / מתי בארצי / תור ישמיע קול

מיודעי ספר / סתם קצי / לבל דעת צופי / עת ישאו קול.





נא הבט וראה / עניי ומרודי / אין לי מכיר / למי אשא קול

נצח אקוה / כי לא יפול דבר / ממקשיבי / דממה וקול.





שוש ישיש / לבי בקרבי / בשמעי דודי דופק / על פתחי קול

סלה ישימני / כחותם על לב / כאז תחת התפוח / עוררני בקול.





עליתני אל / על כל-בנות / כי בעבורי / בחורב תתה קול

על כל-אלהים / מאד נעליתה / נצח תתעלה / בתרועת קול.





פעה ממדבר / צפור ממצרים / ויונה השמיעה / מאשור קול

פקוד צפור בית / דרוש יונת אלם / תקע למו בשופר / ושרוק למו בקול.





צור חקים, מני / לבל יעופו כנשר / ובל יכנפו / משמיעי קול

צרופה אלמד / ועיני למורי / כאז ציר מדבר / ואל משיבו בקול.





קרב קץ / בא עת משפט / קם מליץ יושר / להתחנן בקול

קדש חדש / והוכן מועד / אתקע בשופר / ויענה לי בקול.





רגש מקבר / צוחה מסלע / בתת יבישי עצם / מעפר קול

ראו נס בהרים / וקול שופר בארץ / להשמיע רנן / מדמומי קול.





שנג לב הותל / עוד בל יטנו / בלי להכביד אזן / משמוע בקול

שובב לי כקדם / דת מורשה / אשר בה עלפוני / בלפידים וקול.





תבונה הפיק / איש נבון דבר / וחק נעם זמירות / בנעימת קול

תהלה יתנו / אז לכל השמיעו / לאל מושל בכל / ימתיקו בקול.





תחת בני ציון / בני יון שחו / הברקת חצים / ותהמם בקול

תרעם לבוזזי / תתקע בשופר / בסערות תימן / אז ילך קול.
