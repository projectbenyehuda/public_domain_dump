


	
	אז בדעת חקר
	








אז בדעת חקר / בבינה יעץ / בערמה זמם / בשכל התבונן

אפס אין בלתו / וכי במי נועץ / ואין עמו זר / להפיל דברו.





באחת בימין / טפח מעונו / ונחשב כנשא / אך הוא נושא

בלבדו הוא / אדון רבותים / מלך על צבאיו / ועמו המשל.





גם יסד ארץ הדום / והציב גבולות / והוא ממלא ארץ / ואתו מקום

גריה שוכניה / הלוא כחגבים / רוזניה נדיביה / יחשבו כאין.





דובר והאיר / וחילק בקו / אור לעבודה / ואפל למנוחה

דברם ינוסו / זה מפני זה / ובם יתברך / מגיה ומחשיך.





הביט יוצר / והנה קרים / מכסים מראות / את כבוד מעשהו

החצם ברקיע / והם תלויים באמר / ומהם יגשים / ופלג בל יחסר.





ובם תעשה / רוח סערה דברו / במים תכן כצמר / כאפר וכפתים



ויתרם הניס / לנבכי מצולה / דברו ברוח למו / עד יקראם.





זבד חלד בדשא / ועשב מזריע זרע / למדם להועיל / …יע כל פועל

זכר עשה בם / לנפלאותיו נצח / קציר ביום זרע / ובציר ביום נטע.





חשב להעלים אור / ועמו ישכון. / ואיך ימששו באפל / יצורים אשר יצר?

חק לפיד ליום / ונרות לאישון / נתיבם יכירו / גבול בל ישיגו.





טסי סנפיר / ודאי כנף / במאמר פיו / פרצו לרוב

טוב התוה אכל / לעם אשר יבחר / והכל כירק עשב / לכל אשר ימאס.





יתר ביושב / וגם בנשיה / המון רומסי רגל / וזוחלי עפר

ידע בם / מקרין ומפריס / לארוחת מלך / ופתבג צבאיו.





ככלותו / עד לאין מחסור / כמלאו ספק / ואין מאומה נעדר

כראותו / כי טבח טבוח / ושלחן ערוך / ואין מי ידשן.





למד אורח / ענוה ונועץ / בנשמות יצורים / ויושבי נטעים

לרקם גלם / בצלם מוראו / לגדלו, לחסרו / מעט מאלהים.





מזריח מעריב / מרדים ומקיץ / מטריף מרעיב / מרומם ומשפיל

מביט לאחור / וצופה לפנים / מבין מה בשחק / ושר חקרי ארץ.





נסך והקפה / ויסד שאר / סכך בגיד ועצם / ויקרם עור

נפח נפש / ויצוק רוח / הצית נשמה / ויבער נר.





סכל מלאו / דיעה אסמו / ודת שש מצוות / לפניו ערך

שח: אל תמיתך / תאות עץ דעת / מואס בחיי עת / עת חשוק בחיי עד.





עליו שקד / לבל יהי ערירי / אעש עיור אלמד דרך / גבר בעלמה

עדנו שנה / שכרו תנומה / ומעצם צלעו / הכין לאידו.





פח שם פתן / לרגלי פותה / והיא משכה לפתי / יחד למדחיפות

פתגם אלהיו / לא אבה לשמור / עשה רצון יצר / ולא רצון יוצר.





צאנן עמדו / חשופי נבלות / והומר כבודם / למעטה נובילות

צפע הושת צע.. / למכה בלשון / כמכת פעמיו / סלף מאכלו.





קדר ולא לן / במסוכת עדן / כי ביום חתונה / הודיע אולת

קושי משבר / החזק ללידה / עצב ומות לכל יצור / וציה לחרול.





רחש יצרו / לרבע תשוקת אשת / והבכירה משדד / ומשנהו רועה

רצוי תשורה / ליוצרם הובילו / בכור ממותר / וצעיר מפטר.





שדי קץ ברב / ונרצה בצעיר / וקנאת אכזר שיחתה / אח ברצח

שומע קול דם / ענשו… / ושוע ומגנו / …….





תאב עוד לרבע / והעמיד לו זרע / מעמיד יחס / תולדות נרצח

תחלה לפושעים / הוקמו בני שת / וקראו בשם אלוה / ללא אלהים ואבדו.





תבל העצימו / עשירי שלוה / דברו לאל: סור / מה תטיב לנו

תעתעו בבוז / כלפיד מזהיר / …טו כזרים / ולא כבעלי דעת.





שופט התיר / צרור עבי מים / ובקע מי תהום / עליהם ונפצו

שפע אדמה / בצדיק ונוהו / ובמעט מספר / נצורי תיבה.





רצה אלהיו בניחח / ונשבע שלא [להכ]ות כל חי / וקשת שם בעב / מעבור מי נוח

רצו לקנן במעלה / תושבי מטה / ואל שוכן מעלה / הפיצם מלמטה.





קרב ויאת / ריע מרחוק / רפא ארוחת / משפחות האדמה

קדם והבין / מי אדון לארץ / ומי מושל בשחק / ומנהיג צבאם.





צעדי צולעים / כונן במישור / לבל דרוש למתים / בעד החיים

צורף באש / ונרדף במלחמתו / האומר לו: לך לך, / לעד סגבו.





פרי החניט לו מפריח יבשים / בחנו ונס בו לשי / ולא אחרו

פדאו בנסבך בקרן / קחת לאבות צדק / ולבנים ישע / מנוגחות קרן.





עוקב בבטן / אז קרוא בשם / כי בו גדלו / גדודים לגיאה

עוז דת שנן / במושב אהלים / ואל חק לניניו / מורשה עוז.





סר חנן לרום / וצר הכחש ממנו / כבטיחות אומן / נצב עליו

שובח קדומים / ליופי פירותיו / והוא נשא פרי / כברכות הוריו.





נמנים במספר / כמזלות רקיע / ואין למו מספר / כאין לגדודיו

נגלה סוד לאם / בבן השלישי / בזה ילוה אישי / דברה צנועה.





מרצה אב לבן / ברצוי פועל ידים / ושם קטורה באף / וכליל במזבח

מכרם ידידי / אשר בקרן בן שמן / צץ מפריח בגפן / שלשת סריגים.





לנגיד מהם / נגלה בסנה / לצוען לשעל / לצוה לח…

לנבואת אשת / מחולת מחנים / רצה באר / ורותה צמאים.





כ……. / ברית לקדושיו / היות לו משרתים / ולצ….

כ…….. / חלקם לנצח / ומיתר שלחן מלכם / לע……





ישבו על פתח / אהלו שבעה / ברחיצות ידים / וקידוש ושמן

יומנו ל…קם / טרם הוקם אהל / וכן לדורותם / טרם יום עשור.





טהורי שבט / יפילו גורל / מי יעלה בראש / היות גביר אחים

טירת גירות / מעונו בקדש / להסתופף בם / חוק מלואים.





חולפיו יכינו / כהן משנה / פן……ם / ותופר עבודה

חניכיו יזו עליו / מי טהר / כי תמור דמים / נתנו לדורות.





זה יחלוקו / חצי יתר הניחח / ולא חלק כחלק / ממכר אבות

זבחי שבעה / וחפצם יעשה / [בד]ם ורקח ונרות / וראשון בראש ורגל.





ו…….. / וזקנים יסובבוהו / וישננו יחד / בחקי עשור

ו……. / התיכון….. / זבחי היום / לפניו יובילו.





הן ע…. אות לה / …..מרו / וישם ב…… / …….

ה…… ….ו / ערב טרפו ימע[יטו] / ….שר לא יישן / פן יקרה קרי.





דביקיו יחברוהו / ע…… / ….. רקח / בשם ישביעוהו

דת אל תש[נה] / …. אל תקטיר / כי אם לפני / אדון נסתרות.





גועש בבכי / כי נחשב לפתי / והם פן [ירשי]עו / חף מעוון

גש לדרוש שכל / כחכם ונבון / [ולפני] גדולים / אל יגביה לב.





בהיות לבו / צפון מש[כל] / בהגה כתבי קדש / בפה יעסיקוהו

בער אם יהיה / ולא למד לקח / בשיח מלכי קדם / לבו יפיגו.





אם תגבר עליו שנה / ובקש להתנמנם / בחיך נעם צרדה / משוררים יעוררוהו

את קול שאון / מעיר ישמע / ותפרד שנתו / עד יחצה לילה.





אצילי שבט / זאת היתה למו / קדם, בכשרון כוח / לקחו מצוות

אבירים שרו / הדיפת שבר / והכינו פיס / מי ירים דשן.





במ……. / ……השחר / ועל דשון מזבח פנימי / ונירות ב…….

ב….. / להפיס….. / לנחול בברך / יי חילו.





גבר הפקיד / … שלח ציר / והוא יתן אמר / אם ברק נוגה

גדלה יעשו / לגדול מאחיו / ויפרסו מסך / בינו לבין עם.





דלג וטבל בחוץ / ועטה שמונה / ומהר וקדש / ידיו ורגליו

דבק בתמיד / קרצו כצווי / ואחר ימרס / אך הוא יקבל.





התיז דם / [והי]טיב נרות / והעלה נתחים / חבתים ונסך

הביאוהו לפרוה / לבית טבילת קדש / ויפרסו מסך / בינו לבין עם.





וקדש עוד / ידיו ורגליו / פשט וטבל / ועטה כלי בד וקדש

ונגש אצל פר / אשר מהונו / העומד בין / האולם ולמזבח.





זיו פניו יפו העם / פני פרו גש / וסמך שתי ידיו / עליו בכבד

זדונות יגלה / שגגות יודה / ובעדו ובעד / נויהו יכפר.





חש לשער איתון / שהוא צפון למזבח / וסגן וראש בית אב / מפו ומפו

חלשי פז / שם ימצא בכלי / מוכנים צד / שעירי עם.





טרף והעלם / בימין ובשמאל / זה חקוק לשם / וזה לגזרה

טוב לב ומשוש / ינתן לדור / בעלות בימין / גורל שעיר עם.





ישמיע לו סגן: / הגבה ימינך, / בעלותו / ביד הימנית

יען ראש בית אב: / הגבה שמאלך, / בעלותו / ביד השמאלית.





כהנתנם יאמר / לשם חטאת / ויתום בשני / לזבח ולהרי צוק

כדברו בא לו / אצל פרו שנית / להתודות / בעד עצמו ומטהו.





לשחטו צוה / ודם הזחיל לכלי / ואחר ימוגגנו / ברובד ההיכל

למזבח עלה / לצד פני מערב / וחתה בוערות / מלא מחתת אדם.





מטה ברבד / עזרה נתנה / ויציאו לו / כף ומגס קטרת

מגדישת / דקת הדק / מלא חפניו / ונתן לתוך כף.





נטלה בשמאל / ומחתה בימין / העמיס כבד / ימין על שמאל

נכנס בהיכל / והכניס בפרכות / ובא מצד ומש / בקיר צפון.





שם פ[ניו לדרום] / ובא עד חצי בית / בעוד ארון שם / בבדים פגש

סלק פרכת / לאחריו ברוב כוח / נחלץ ונצב / מול…





על אבן שתיה / הכין מחתה / ושם סמים בכלי / ותמר עשן

עקביו הצדיד / ויצא לאחור / ושוע בהיכל בקצר / פן יבעת עם.





פרח הממריס / הגיש דם פר / ושב ונתיצב / נוכח הכפרת

פעם למעלה במצחה / ושבע למטה בפניה / במספר הזיות / באצבע יצליף.





צעד ונתנו בכאן / ושחט שעיר עם / ובא ועשה לו / כמשפט דם פר

מעד ונתנו בכן / והמירו בדם פר / וחוץ נתיצב / מול הפרכת.





קצב מעשה פנימה / עשה בחוץ / וכן בדם שעיר / ובללם בעקב

קרב למזבח / זהב וחטאו / ורבע קרניו / ושבע על טהרו.





רגיל ועתי / משכו לשלחו / עמוס עוונות עם / ופשעיהם

רצון חלבי פר / ושעיר והקטיר / וישרפו אחרים / לחומם במפקד.





שלח שעיר / לראש המדבר / שנן סדר יום / במכתב, על פה

שמונה ברך / וקדש ופשט / וטבל ועטה / שמונה וקדש.





תכן מוספי יום / הכליל וקדש / וטבל ועטה / כלי בד וקדש

תאר לפנים / והוציא כלי קטרת / וקדש ופשט / והכמין כלי בד.





תכלית חמש / טבילות השלים / וטבל ועטה / שמונה וקדש

תמיד ערב עשה / כחוק ראשון / והקטיר פרס קטרת / בין נתחים לנסך.





תמו העבודות / והבעיר נרות / וקדש עשירית / ופשט כלי פז

תלבושת הונו / צעיריו החישו / לבשם ולווהו / יקרים לנוהו.





תן לנו מופת / באזניו יאמרו / במה נדע / כי יכפר עוון?

תקות חוט / שני הלבינה, / ציר הנאמן / לשולחיו יען.
