


	
	אהללה אלה
	








אהללה אלהי / אשירה עזו / אספרה כבודו / אאפדנו מלוכה

אשגב לפועל / אשר שח ופעל / אאנויהו כי לו / יאתה מלוכה.





בעזו נצח אשנן / כי צבאו אני / ולו נאה שיח / גדל המלוכה

בקהל אבשר / ברב עם אהלל / למי שאת ויתר עז / ולמי המלוכה.





גשו גוים / ובואו ממלכות / ראו מה נהדר / במזח המלוכה

גדלוהו אתי / ונרוממנהו יחד / ואל תתגאו / בנזר המלוכה.





דרכים בעת / נעשתה מצולה / התבוננו יחד / למי נזרקה מלוכה

דרך סוס בים / כנגד שש מאות רכב / ומה-יועיל גבר / עוז במלוכה.





האזינו רוזנים / אז וירגזו / הביטו חתת / ומאסו מלוכה

הגידו כוחו / לאמים ודברו / לזה יכתב / שם המלוכה.





ונלחם ראשית / גוים ואבד / כי נשבע חי / בכסא מלוכה

וילעג בכל-דור / כי לא למד / מי נלחם בים / ועטה מלוכה.





זד על אדוניו / עבד ישב נגב / בזאת תרגז ארץ / בשאת עבד מלוכה

זרע ברוכים / החרימו ארורים / כי נתנו קול / לאדיר במלוכה.





חשבון ובשן / עוררו מלחמת / בלי לתת דרך / לצבאות מלוכה

חילם נשמד / וארצם חלקה / ומעל זרועם / נפלה מלוכה.





טפשו בני כנען / כי נכרים הם / באדמת בני שם / זרע המלוכה

טבחם בן-נון / עד יפנה ארץ / לפני ארון הברית / אדון המלוכה.





יושבי חרושת / אז הקשה לחץ / עזרוהו בלי בצע / אפסי מלוכה

יה הלחים בם / צבא בלי בצע / כן יאבדו שאר / ולאל המלוכה.





כארז בלבנון / אשור גדל / וחירף: אוריד / כביר מלוכה

כליל אש הממם / בליל שמורים / ואז ידעו הכל / כי לאל המלוכה.





לשחת כרע בל / כחשבו עלות לשחק / וסר מנו לבב אנוש / ורד ממלוכה

לכנו הושב / וכח אל הכיר / למרים ומשפיל / השלים מלוכה.





מגרה צאן לטבח / ונתכנו עלילות / בלבוש צעיר / רודם המלוכה

מכורי בלא הון / פדויי בלא כסף / סולו למטה כמים / לב המלוכה.





נמכרו יונים / לבני יונים / וריחקום מעל / גבול מלוכה

ניארו ברית ודת / והמרו עם באל / ומגרום בלא כח / מכהני מלוכה.





שעיר החניף / להוריהו בצידו / ויירש בקול בכי / חרב ומלוכה

סגב חלק / היות גביר לאחים / ועוד תיסוב / לישורון מלוכה.





עשה לך בציון / שם נוראות / כאז תצליחנה / בכסא מלוכה

עורר והקץ / משוש כל-הארץ / וכונן כסאך / בקרית מלוכה.





פני מאור לבנה / וחמה תחפיר / ויבושו עובדימו / בשאיתך מלוכה

פאר עיר יופי / לברה כחמה / וגלה לנגדינו / כבוד המלוכה.





צבאות גאולי צען / שוררו בליל חג / והוא לילה נשמר / לסחף מלוכה

צעדו במי שעל / צפו ברוח שכל / אנה ינטעו / ויקבלו מלוכה.





קמטו שערי זבול / בית עולמים / כי מבינימו / שבתה מלוכה

קדוש יבוא / בם לעולמים / ואז ישאו ראש / בחדשך מלוכה.





רבצה עדינה / שקטה מאלמון / כי ארך לה / קץ ומלוכה

ריבו מושיעים / שאו אדר מאדום / ושיתו על אדון / הוד המלוכה.





שוא שנא אל / והוא על לשונינו / בקש אמת ואין / וריחקה מלוכה

שדי הסר / און מצבאיך / ויריעו לך / תרועת מלוכה.





תחגור גאות / תתאזר עוז / לבל ישתרר / זר במלוכה

תכון תבל / כי ינער רשע / ושם צדק לרגליו / ויצנף מלוכה.





תקום גוים / תוכיח לאמים / תשבר מטה רשע / מושל במלוכה

תחליף אלילים / תשגב לבדך / תקרא נצח / יחיד במלוכה.
