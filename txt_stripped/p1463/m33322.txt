


	
	אפחד במעשי
	








אפחד במעשי / אדאג בכל-עת / אירא ביום דין / בבואי לזכרון.

אדרוש לחנון / אחלה לרחום / אחנן לחק לי / יום לזכרון.





בבואי למשפט / במי אשען? / ומי יחפש-לי / צדק לזכרון?

באבות בטחתי / ופעלם אכלתי, / הם היו לי / קדם לזכרון.





גברה זרועי / כשח: מחיני נא, / לבל ימח מני / שם וזכרון.

גבר אם יעמוד / לפניו היועיל / בעת יבקש מני / זכות לזכרון.





דצתי בלובש אפוד / וחושן המשפט / אשר בם הוחק / שמי לזכרון.

דלג במחתה / עד יעצר נגף / בגשת זר מכהן / יקוד לזכרון.





הביטה אל / בעמדי לפניך / אין בקרבי אנוש / תוי לזכרון.

היש מי יפלל / וישיב חימה / ויחק לדורות / שמו לזכרון.





ומי שונא בצע / ידבר: ענו בי, / ויען ויאמר: עד / אדון לזכרון.

ובטלה חלב / יכפר בעדינו / וכנגד שני עפרים / יעל לזכרון.





זעף וירא, ויאמר: / תהי ידך בי, / אשר כלב אלהיו / חשוב לזכרון.

זעק והשיב / חרב אל נדנה / והושת לו כשמש / כס לזכרון.





חכיתי בעת עצר / לפותח וממטיר / כמשיב רוח לילד / אבוד מזכרון

חי לראות ברית / בדברו: קנאתי, / כי עזבו עם / ברית לזכרון.





טרם היה לי / מכפר פנים / מנחה הולכת / כשוחד לזכרון.

טרף נרד וסמים / למיטב חדריו / דם וחלב לניחח / ולחם לזכרון.





יצגתי על / גחלי ריקה / כי לא אלמן / שתי לזכרון.

יה אבטח בך / ולא בנדיבים / כי הם בקבר / ולנצח שמך לזכרון.





כל אלה סמכוני / ורחמיך בקשתי / לולי הם תמתי / ואיני לזכרון.

כי הם בזרוע / עדיך לא באו / רוממתיך בפיהם / שמו לזכרון.





ליום זה נכמס / סכם חשבונות / תחלה לימים / וראש לזכרון.

להקרא בו / כתב עט ושמיר / גלוי ובאר / וידוע לזכרון.





מות וחיים / שלום ומלחמת / צחצחות ושובע / באו לזכרון.

מעללי גבר / ומספר צעדיו / נשכחו מאנוש / ולאל לזכרון.





נסתרה דרכי, / מי יוכל שיח, / לשוא נכתב לי / חטא לזכרון.

נגד פני גבר / מעשיו יוכיחו / ויענה כחשו / עד לזכרון.





שיחו מזימות אל / יחד כל בני איש / עוברי תחת שבט / כצאן לזכרון.

סוגר דלת / בעד תמימים בזעם / עד בוא קצם / צאת לזכרון.





עשה פלא לחיים / לבל יהו כמתים / היש אמתיך / בקבר לזכרון.

עוררתה אז בפועל / עוררנו בלא פועל / הלא לנפלאותיך / תעש לזכרון.





פנה אלהים / ביושבי גנים / מקשיב לנדברי / בדת לזכרון.

פעלם לפניך / ושכרם אתך / אכלי לחם העצבים / בספר זכרון.





צצו שועלים / מחבלים כרמים / להכרית מגפן / שורש וזכרון.

צררום בפרך / נאקו ונושעו / בכושר הררי קדם / הוחק לזכרון.





קדם בנתה לדורות / ואין בהם חפץ / חלפתם ואבדתם / מהיות לזכרון.

קחתה דור מאלף / אמריך הנחלתם / למענם בכל דור / חקתה לזכרון.





רם חשק מאד / בכלת נעורים / הרוגיה וענייה / באו לזכרון.

רצה אחריך / בגיא ציה וצלמות / אהבת כלולותיה / תעש לזכרון.





שיחתו עם אל / שאת שמצה בקמיהם / שם פסל ולא אל / שתו לזכרון.

שכלו כמעט רגע / לולי קם בפרץ / מעורר שבועות / ברית זכרון.





תיעבתה מאז / עדת כל אמים / חשקתנו מהם / עדות לזכרון.



תמור כסף נמאס / דור נשכח קחתה / קנינו שנית / כי שכחנו מזכרון.





תר איש תם / במי זרעו יכנה / שכל ידיו / לאות לזכרון.

תרפק משול אפרים / בשעשוע ילד / הלא בכורי הוא / חקתה לזכרון.
