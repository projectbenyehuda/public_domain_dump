


	
	אזכיר גבורות אלוה
	








אזכיר גבורות / אלוה נאדר /
יחיד ואין עוד / אפס ואין שני,

אחריו אין בחלד / לפניו אין בשחק / אין בלתו קדם / ואין זולתו בעקב.





אדון לחשוב / אלהים לעשות / נמלך ואין נעדר / שח ואין מאחר

אומר ועושה / יועץ ומקים / אמיץ לשאת / וגבור לסבול.





אשר לו רננות / מפי יצוריו / ממעלה ומטה / ישא תהלה

אל אחד בארץ / וקדוש בשמים / ממים רבים / אדיר במרום.





אדר מתהומות / שבח ממארות / אמר מימים / וזמר מלילות

אש תודיע שמו / עצי יער ירננו / בהמה תלמד / עזוז נוראותיו.





אמון שחקו / דת שעשועיו / היא הגיונו / עד עמוד סגלה

אז קדם לאלף דור / עלתה במחשבת / וממנה תכונת / כל מלאכות התבנית.





במרום הכין / כסא הדרו / פרשז עננו / ודוק נטה לאהל

בל יצען / ובל יסע יתדותיו / עד בוא קצו / ויתחדש באומר.





בירר על־מים / עמודי חלד / ושנס מתניה / בתוהו ובסערה

בל תמוט / ובל ימעדו מכוניה / עד תבלה כבגד / ותומר כמאז.





בוהו ואפלה / כסו פני חלד / ותבהק אור / מאור פני מלך

ביהק נוגה ליום / לצאת אדם לפועל / ושת חשך לאישון / לרמס חיתו יער.





ברחף רוח / בין שחקים למים / ושת תקרה בתוך / שאת חצי מדודים

בם יודע / פלאו לפעלו / בטוב פרי גשם / פלג מלא מים.





בהבדילו / בין מים למים / הסיק עלוקה / תפתה לבלי חק

באש לו נפח / ולא נכבה נצח / ושטף מים רבים / לא יגיענו.





גרש למקוה / מים אדירים / ושם להם חול / דלתים ובריח

גדר בעדם / בל יכסו ארץ / עד נאמן בית / יעשם גזרים.





גילה פני אדמה / לעבודת אנוש / והדשיאה שיח / ועשב מזריע זרע

גם נטע בעדן / גן רעננה / לחפת כבדת / אדיר כל חפץ.





גמר להעלים / אור שבעתים / וערך נרות / למשל ביום ובלילה

גבול לא ישיגו / וללכת לא יאחרו / עד משרת אהל / ידמימם בעמק.





גאו ממים / בריחי תהומות / אפיקי מגן / שנוני חצים

גנז מהם / למשתה נצחים / והכין בם כלא / לציר הבורח.



גביהי עוף / צצו מרקק מים / לאוכלי שלחן מלך / וחיל צבאותיו

גזר לתועבת / עוף לא טהור / עד בוא תשבי / ועורבים כלכלוהו.





דגו מחלד / מקרנים לזבח / חיות מאכל / עד בהמה ורמש

דשן בהמות / ביבול הרי אלף / כי ביום זבחו / יגש חרבו.





דץ יוצר / ושמח במעשיו / בהביטו / כי טוב פעלו

דשא מנחות / ואכלי מגדים / ושלחן ערוך / ואין מי ידשן.





דבר בלבו / מי יסור הנה / לטבוח הטבוח / ויין המסוך

דברי אם יעש / יהיה כאלהים / ואם ימיר אמר / לחמרו אשיבנו.





דרש והבין / וידע מתי שוא / כי לא יבין / ללין ביקר

דמהו בצלם / חקקו בתבנית / להיות בדוק וחלד / פחד מוראו.





דגול בתוך עדן / יפה יצועו / בפז ואבן יקרה / חק מסכתו

דעת קשטו / ורוח חננו / להשכיל מקדם / קץ אותיות.





המשילו / כצדיק ומושל / לכלכל שמות / לאל ומפעליו

הזהירו / על עץ הדעת / פן ינקשו / במאכל תאוה.





הרדימו / בנעם שינה / והרים ממנו עצם / וכונן עלמה

העדה ככלה / עדי קשורים / והדביקם יחד / היות שאר אחד.





השיא פותה / זוחל עפר / להניא אמר / ולשוב לעפר

המיתה איש / בתאות העץ / והפילה רבים / והרגה עצומים.





הכה נחש בטרפו / ובפעמיו לנצח / והוא ישלח ציר / לפורצי גדר

הערמו וכסו / בעלה נובלת / תמור לבוש ענן / חתלת ערפל.





הריון עצב / נתן לתולדות / והוא לעמל ועצבון / לבאי עולם.

הומר תארם / והוסר פחדם / להרדות מכל / ולא לרדות בכל.





הורמו מעדן / וגורשו מגן / והוכן נקיק סלע / בית מלון למו.

הפתה יציר פתי / בשיח פתיון פתן / וגיא בם קללה / כאם על בנים.





ויוצר החיש / תאות יצורו / והרגיל עזרו / לרבע תשוקתו

ויחמה והזריעה / וחבלה וילדה / ובכרה משדד / תלמי אדמה.





והוסיפה שנית / והכריכה ושלחה / עויל תומך שבט / רועה עדרים

ובתשורת שי / קדמו פני מלך / זה בפרי ארץ / וזה בפטרי צאן.





והביט קדוש / ללב ולא לעינים / ושע ערך / ניחחי צעיר

ושגב אח / קודר למרום / ברצותו שי / ערך ניחוחו.





והקדיר פני רב / וגם חרה לו / כי מאס בשי / יתר מאכלו

ויפעל זה / כמשפט הורהו, / כי לא נצר / מצות מצוהו.





וטרף אף / ושמר עברה / וחבל מארץ / אח ומשפחותיו

והצריח זעקה / דם השפוך / ונענש בנע ונד / ראש לשופכי דם.





ונתודה על פשע / ונתן לו אות / פן ישפך / דמו באדם

והחל תשובה / לבאי עולם / כי כל מודה ועוזב / בתשובה ירוחם.





זה חבש עצבו / יציר קדמוני / כי החל שתות כוס / מסוך להודות

זכר לתת לו / זרע לזרע / להחליף לו / פרי האמלל.





זרעו השריש / שרש פורה רוש / ושגשג נטעו / עשות פרי באושים

זממו והחלו / דור עקלקלות / להמיר בשם אליל / שם אל עולם.





זעף מים עצורים / עליהם קרא / פרצו גבולם / ויהפכו ארץ

זכר תקות אנוש / האביד מן הארץ / ושטף יסודי / ספיחי עפר ארץ.





זלזלים אשר / בעצי יער / עד דור עשירי / לא עשו פרי

זך ותמים / בקרבם נמצא / וריח מפעליו / כרקח בדורותיו.





זדו ומרדו / בשבעת ימי נוגה / ומרוב כל טוב / אמרו לאל סור

זועמו במי מעלה / ודכאו במי מטה / ורצהו צדיק בשי / מעבור מי נוח.





חדשו פני חלד / ושבו לקדמתן / ממתי מספר / יוצאי ממסגר

חשבו ונועצו / אז דור חמישי / לרום ולשום / קן במרום.





חי לא בצר / מהם מזמה / מליץ ללצים / חפצם השלים

חלק שפת לשון / בעלי מחלקת / וכתתו זה בזה / ונלחמו למשחית.





חלפו כעבור / סופה ואינמו / והוקם צדיק / יסוד עולם

חשב להבין / סוד מפעלות / המתנהגות / בשוטר ומושל.





חזותו מרוצת / שואף ו[זורח / שש כגבו]ר בצאתו / וחלש בבואו

חלוני שחק / אשר בקדמה וימה / אשר בם תנתר / לבנה יום יום.





חצי ברקים / ודוהר כוכבים / רצים ושבים / ואיש לא נעדר

חכם עוג לב / ושכל מאליו / ושח: אדון לאלה / אחריו ארוצה.





טרם הקדים אל / בירר לו דרך / לך־לך ממות / לדרך החיים

טהור במעשיו / דבר גבורות: / נא אלהי המשפט / שפט־נא ברחמים.





טרף רוזנים / בלא מגן וחרב / שגבו מאש / עזרו במלחמת

טפש דורו / וצור לא הכירו / והודיעם באשל / שם אל עולם.





טבע חותם ברית / שם בשארו / בה תפן פוערת / ואז תקפץ פיה

טרפי צמח זקונים / הנצו ממנו / בהפתח מקור / לערוגת הבשם.





טורח בוחן מנסיהו / עמס על־שכם / וגם משל ביצר / תת שיו לטבח

טנא בכורים / למנחה הוביל / אב לא חמל / ובן לא אחר.





טובח בעד שה / ב[ע]ת [תפש] חרב / מלאכי ש[לום] / מר צעקו בכי

טוב ורחום שח: / אל תשחת נער / נרצה פעלכם / כזובח ונזבח.





טלה הוחן / והושת כפרו איל / ויוחק שרוף / להצדיק עמוסיו.





יסד איש תם / בפתחי תעודה / טרם תאמר / שנן אמריה

יה מעולם / בו נתפאר / ובשמו לעולם / יושב תהלותיו.





יודעו נצב / עליו במלון / ושח: אני שומרך / צל על ימינך

יורדים ועולים / קדושים למענהו / להכיר תארו / חקוק במרומים.





יוצר חסרו / מעט מאלהים / ליחם במקלות / כחמר ביד היוצר

יראת חזון / ארמי בעתתו / אל תגע בבן / עמוס מני בטן.





ידוד אש לוהט / עמו בהאבקו / לו כי לא יכול / נתחנן בבכי

ישע הפליא לו / מחגור חרב / שוטם ושומר / איבת עולם.





ירש תאות / ברכות הוריו / שנים עשר שבט / כשוטרי רקיע

יוחד שלישי / לראות פני מלך / לשורר לשרת / לבוא [חד]ריו.





כעץ שתול / על פלגי מים / [הפ]ריח מטהו / שלשת בדי קדש:

כפופי סבל / [לא]הל וקרשיו, / תוקעי יתד / פורשי יריעות,





כתף נטויה / [כ]עגלה מלמדת / לעול עבודת קדש / אדון כל הארץ.

כגפן אדרת / יפת פרי וענף / הצמיח עמרם / משרש לוי,





כי שלחה שלשת / שריגי חמד: / מכהן ורועה / ואשה נביאה.

כגשת עת דורים / פרחו הוקם / לשבר מוסרות צען / ולפרץ גדר שועל.





כסה בענן / ונתקדש שבוע / ונצב בתוך / ביום תת אמר

כח אדירים / לפניו הכנע / וממוראו / לא עמד איש.





כלכל צאן קדש / שאר בישימון / מלחם שחקים / עד בואם לארץ.

כרתה באה לעם / עלמה מתופפת / גועה ונאספה / ולא היה מים.





לוויי ענני הוד / ידידים נחננו / על ידי מכהן / בשלום ובמישור

לו ולזרעו / ברית מלח נחקקה / לבל תשבית ברית / מלח הניחח.





לקנאת חמת גבר / גמול גמל אל / וחדש חקו / לדורות עולם

לזר לא יאתה / נחלת כבודם / ולא יירש אנוש / מתנת חלקם.





לצלמות הורידו / עדת אנשי ש[ם / והושיבו] עזיהו / בבית החפשית

לא יקנאו בחבל / [אחזת] אחים / כי אלהים נקרא / מנת גורלם.





לחם הא[שים] / להם הוכן / בהתקרבם לאל / בעת רחוק ע[גל]

למדם מחוקק / סדרי עבודה / כי בשבתם פת[ח] / עליהם הפקד





לרחץ ולסוך / לקדש יד ור[גל] / ללבש בדים / ולחגר במזח

למלא ידם / ימ[ים] שבעה. / וישימה לחוק / לדורו עולם,





משפחות ברורים / גורל יטילו / לגדל סגן / לעלותו בעשור

מערש יצועו / שארו יבדילו / פן יטמא שבוע / בשגגות דוה.





מקום עצרת סגנים / בית מושבו / הלא שם ילין / כל ימי שנה

מעלה מנחה / ונותן סמים / ועורך נרות / וראשון בראש ורגל.





ממ[כר] שבתות / וחלק משמרות / חלקו בראש / כנגד כלם

מחכימי פתי / יעמתו לו / לשננו לחנכו / בחקי עשו[ר].





מביאים לו / [בערב י]ום סליחה / פרו ואילו / וזבחי קהלו

מה נאה / עבודתך [יאמרו], / חזק ואמץ / כי כל אלה פעלך.





מאכל ימעיטו / מנו [לעת] ערב / פן ירדם וי[קר] / מקרה לילה

מחנכיו יחברוהו / לזקני שבטו / ובבית [רוקחי רקח / ב]שם ישביעוהו.





נעגם בבכי / כי הרשע כפת[י] / והמה יבכו / פן צדיק ירשיעו

נעימות מדרש / ומקרא ישוחחו / אף בכתבי קדש / הם יעסיקוהו.





נפשו ישובבו / בשיח מלכי קדם / אם בער הוא / ולא למד לקח

נטה לנום / שנתו יפרידו / בנעם שיר צרדה / בפה ולא בכינור.





נותנים שאון / עיר הומיה / להשביעו / נדודים עדי נשף

נוצרי חקים / ובוזזי מצוות / להרים דשן / חצות לילה יקדימו.





נהדפו רצים / במעלות מזבח / ושקדו בפיס / להשבית מדנים

נוערו וטבלו / וענדו עטרה / והפיסו תחלה / על־תרומת דשן.





נצבעו שנית / על־מעשה כבש / ועל־דישון מזבח / פנימי ונרות.

נתחדשו / שלישי לסמים. / רביעי לתערבת / [לערך הנתחים].





שר פקיד נגיד / מלא[כת הקדש] / ישלח לקדים / צירי אמונה

שח למו: הבי[טו] / אם עלה השחר / אם הפיץ אור / קדים עלי ארץ.





ס[פרו] לו / כי ברק נוגה, / ורץ למקוה טהר / ל[בית] העזרה

סכו מחיצ[ת] שש / בינו לבין עם / [לנהוג] בו בכבוד / בלי חז[ות מ]ערהו.





שמלת ע[ורו] / ימהר ויפשט / וי[ר]ד ויטבל / ויעל ויס[תפג]

סגן המלבוש / יעטיהו בדים / ויוסיף לג[דלו] / בעדי בגדי פז.





שש במעטה הוד / ולב לא הגביה / לבשם לכבוד אל / ולא למענהו

סדר מלחמות / בהם ישאל / עיניו למורהו / כתלמיד לרב.





סוד יגלה לו / במשפט האורים / אם עלה אם חדל / אם לחרב אם לישע

סולו לאלהים / בני גוי גדול / קרוב בכל־עת / מבשר ישועות.





עלץ במלאך / בשיבוץ מנכסי בד / כפרש מוכן / ציר אמת לשלוחיו

ערות זמה / בהם יכסה / כי כן [מ]צותם / לכסות ערוה.





עלמות קומה / ימלא [בכותנ]ות / כפולה משבצת / עד פסי יד

עוון בית יעקב / [יכ]פר כזאת / מוכרי צדיק / על כתונת פסים.





עטרת [ר]אשו / בהוד המלוכה / צנוף צפירת שש / לכבוד [ול]תפארת

עסק פריעת ראש / יסיר מקדשות / ויצדיק קדושים / בנזר טוטפות.





עוד ידבק בלחי / כאזור מתנים / תועי בצלמי / חגורי מתנים

עוז אבנט / [כ]מזח יתאזר / ארוג בד, ולא כלאים / ככל־ימי שנה.





עטוי מעיל תכלת / כזהר הרקיע / ממלא בית יד / גלילי זרועותיו

עגול שפת פיו / ראשו כחתרא / מסב קלע / בלי להרקיע.





על שפת שוליו / רמוני צבעים / ופעמוני פז / סביב בתוך

עת ישיקו / קול זה בזה / יכפר בזאת / מכה רעהו בסתר.





פעלת חושב / אריגת האפוד / משזר שש ושני / ותכלת וארגמן

פתיל הזהב / נטוה בכל־חוט / ונתלבש בו / מלב ועד טבור.





פנות כתפיו / כזהר המאורות / מבהיקות אור / מאבני השהם

פתחם בשמיר / ז[כרון] שבטים / המתחלקים / בשמות ואותיות.





פ[רש] ממנו / חשב האפוד / ארוג במעשהו / ויענ[דו] לאחריו

פורק עוון צורי / ראש דברות קדש / אפד[ת] מסכת / אלהי זהב.





פני לב האפד / קרוע זרת ע[ל] זרת / מקום ירכס חשן / בטבעות זהב

פקד ט[ורי] אבן / כמספר דגלים / שמם חקוק בם / וימלא[ו] חשן.





פז מסבות / במלאותם / ומראה דמות[ם] / כמראה עינים /

פרץ דבקים / אשר בין אפוד לחש[ן] / יכסה בשרשרות / עבותות הזהב.





פשע הופכי משפט / ימחה בחשן המשפט / ויצדיקנו במש[פט] / אלהי
המשפט

פעמוני פז מקף / בין אפוד לחשן / חבר לנזר הקדש / על־שתי כתפיו.





ציץ פז מקף / צצי פנינים / דבק וח[בר] בחוט / לנזר הקדשה

צניף הוד שם במצחו / ועליה הציץ / ושטת שיער / בינו למצנפת.





צור עולמים / האל הנערץ / חקוק על־ציץ / במכתב חותם

צואת טמאת / עוון דם ונסך / ירצה הציץ / כניחוח טהור.





ציץ נובל / תולעת ולא איש / נחשב לשרת / לצור עולמים

צופה פני מלך / ובא בחדר קדש / לבוש בדים / ולא בכתר אופיר.





צו עונש מיתה / לבאי עזרה / לקדש ידים ורגלים / בבואם לשרת

ציר לכן בטרם / גשתו לשרת / ימהר ויקדש / ידיו ורגליו.





צעיריו יגישו לו / את־תמיד השחר / קרצו וקבל / דמו וזרק

צעד להקטיר קטרת / ולהטיב נרות / להקריב ראש ונתח / חבתים ונסך.





קדם בצאתו / לבית פרוה בקדש / וסכו מחיצת שש / בינו לבין עם

קדש ופשט / וטבל וספג / ועטה כלי בד / ושב ונתקדש.





קרב אצל / פר חטאתו / והוא עומד / בין אולם ולמזבח

קם תת ידו / על־ראשו בסבל / להתודות בעד / עצמו ונויהו.





סומך שתי ידיו עליו ומתודה, וכך היה או׳ אנא השם חטאתי עויתי פשעתי לפניך אני וביתי אנא השם כפר־נא על החטאים על־העוונות ועל הפשעים שחטאתי שעויתי שפשעתי לפניך אני וביתי. ככת׳ בתורת משה עבדך לאמר כי ביום הזה יכפר עליכם [לטהר אתכם] מכל חטאתיכם לפני השם. והכהנים והעם, העומדים בעזרה
והמשמ[שים] בהיכל, בזמן ששומעים את־השם המפורש המפואר שיוצא מפי כהן גדול בקדשה ובטהרה היו כורעים ומשתחוים ונופלים על־פניהם ואו׳ ברוך שם כבוד מלכותו לעולם ועד. אף הוא מתכון לגמור את השם כנגד המברכים ואומר להם: תטהרו.





קדמה לשער איתון / לצפון המזבח / ילווהו בכבוד / כהני משנה.





קימת שעירים / הלא שם היתה / ובקלפי הנחו / גורלות הזהב.

קובצים על יד / וטורף ומעלה / זה לחי עולם / וזה למיתת צוק.





קחתו בימין / פור חקוק לשם / סגן ישמיע לו / הגבה ימינך,

קשי מעלי דור / בשמאל אם יעל / ראש בית אב ישיח / הגבה שמאלך.





קפץ ונתנם / על שני השעירים / והצריח בקול מר / זה לשם יהוה חטאת

קשר שני / בראש שעיר המשתלח / פניו למדבר / וזה לבית השחיטה.





וכך היה אומר אנא השם יהוה חטאתי עויתי פשעתי לפניך אני וביתי ובני אהרן עם קדושך אנא בשם יהוה כפר־נא לחטאים ולעוונות ולפשעים שחטאתי שעויתי שפשעתי לפניך אני וביתי ובני אהרן עם קדושיך ככתוב בתורת משה עבדך כי ביום הזה יכפר עליכם לטהר אתכם מכל חטאתיכם לפני יהוה. והכהנים והעם העומדים בעזרה בשעת שהיו שומעים את־השם המפורש יוצא מפי כהן גדול בקדשה ובטהרה היו כורעים ומשתחוים ונופלים על פניהם ואומרים ברוך שם כבוד מלכותו לעולם ועד. ואף הוא היה מתכון לגמור את־השם אחר המברכים ואומר להם: תטהרו.





רצחו כדת / וקבל דם בכלי / ואחר ימרסנו / במעלות האולם

רמס כבש / ועלה לראש / ובידו מחתת / פז האדום.





רשפים מלאה / מפני מערב / ירד ושמה / ברבד המרוצפת

רעיו יוציאו לו / את המחתה / וכלי בית צבירת / דקת הדק.





רוקח מלא חפניו / נתן לתוך כף / נטלה בשמאל / ומחתה בימין

רכב כבד / ימין על שמאל / והשמאיל בהיכל / בעקב הפרכת.





רגל בקרבם / מנגד לצפון / ובהגיעו שב / עד חצי הבית

ראשי הבדים / כמראה שדים / לוחצים בפרכת / ואין דרך לעבור.





רב כח יודע / וידחה בפרוכת / ויבא בלא פסע / בין שני הבדים

ראש עפרות תבל / אבן השתיה / מקום מוכן שם / שם מחתת פז.





שפך בה קטרת / ותמרות עשן / והצדיד פניו / ויצא כדרכו

שוע בעד עם / בקצר בהיכל / אשר לא ישור / רב על צעיר.





שובב לו דם / הממרס בו / ובא ונתיצב / במקום עמדו

שלח אצבע מימנת / וטבל והצליף / אחת למעלה / ושבע למטה במספר.





וכך היה מונה אחת, אחת ואחת אחת ושתים אחת ושלוש אחת וארבע אחת וחמש אחת ושש אחת ושבע, יצא והניחו על כן זהב שהיה בהיכל.





שב ושמו בכן / וזבח שעיר עם / וקבל דמו / ובא והצליף כדרכו

שוהו בכנו / ונטל דם פר / וכן יזה באהל / מול ארון חוצה.





וכך היה מונה אחת, אחת ואחת אחת ושתים אחת ושלוש אחת וארבע אחת וחמש אחת ושש אחת ושבע, יצא והניחו על כן השני שהיה בהיכל.





תודה נתנו שב וימירנו / בדם השעיר / ויפעל בו / כמשפט דם הפר

שלמו כדת / סדרי הזיות פנימה / בללם ויצא / לחטא מזבח הקטרת.





שיתחיל מקרן / צפונה מזרחה / ובמקום יגמור / שם יזה שבע

שופך שאר בחוץ / ביסוד פני פתח / כי בם יכפר / על עזרת הקדש.





ואחר כך בא לו אצל שעיר המשתלח וסומך שתי ידיו עליו ונתודה בעדינו סומך שתי ידיו עליו ומתודה בעדינו וכך היה או׳ אנא השם חטאו עוו פשעו לפניך עמך בית ישראל אנא השם כפר־נא סלח נא לחטאים לעוונות ולפשעים שחטאו שעוו לפניך עמך בית ישראל ככת׳ בתורת משה עבדך כי ביום הזה יכפר עליכם לטהר אתכם מכל חטאתיכם לפני השם תטהרו. והכהנים והעם, העומדים בעזרה והמשמשים בהיכל, בזמן ששומעים את השם המפורש המפואר שיוצא מפי כהן גדול בקדשה ובטהרה היו כורעים ומשתחוים ונופלים על פניהם ואו׳ ברוך שם כבוד מלכותו לעולם ועד אף הוא מתכון לגמור את השם כנגד המברכים ואומר להם: תטהרו.





תכן איש / מועד לשלחו / ועלה והקטיר / חלבי פר ושעיר

תקפם במוטות / חוץ למחנה עיר / ושרפום כליל / עור ובשר ופרש.





תאר שעיר / לראש המדבר / משכוהו לשנן / בחקי עשור

תחלה יקדש / ידיו ורגליו / ויפשוט כלי בד / ויעט בגדי עשרו.





תם הגיונו / ופשט וטבל / ועטה שמונה / ושב ונתקדש

תשורת כבשים / ואלים הכליל / וקדש ידיו / ורגליו ופשט.





תרגל פעמיו / לבית הטבילה / וטבל וספג / ועטה כלי בד

תלמודו יעש / כדברו, וקדש / ויוציא מפנימה / כף ומחתת פז.





תכן עבודת / כלי בד השלים / וקדש ופשט / והניחם נצח

תכף טבילה / לבש כלי פז / ושב וקדש / ידיו ורגליו.





תמיד הערבים / יעשה כסדר / והקטיר קטרת / בין נתחים לנסך

תכלית מעשים / נרות הבעיר / פירש כפים / וקדש ופשט.





תלבשת שארו / חניכיו יריצו / לבשם בכבוד / ויצא במשוש

תארו הקרן / כצאת השמש בגבורתו / משלח לשולחיו / צדקה ומרפא.





תקות קהל / לצאת איש מהיר / מבשר ישועות / ומודיע סלוחות

תהלה נתן / לשם אל פני עם / בחמש טבילות / ועשר קדשות.





תחלה השמיעם: / לכו שתו בלב טוב / אל נושא עוון / ועובר על פשע

תן לנו מופת / באזניו יאמרו, / במה נדע / כי כפר עווננו.





תתו ציר שעיר / והוא יבשר סליחה, / והלבין כשלג / אדם השני

תודה נתנו / כי טוב השמיעם / על ציר אמונים / המשלח למדבר.





תמימים ישרים / לנוהו ילווהו / ושמחה יעשו / בצאתו בלא פגע.

אשרי העם / שככה לו / אשרי העם / שיי אלהיו.
