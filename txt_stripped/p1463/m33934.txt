


	
	אין לנו כהן גדול
	








אין לנו כהן גדול / לכפר בעדנו / ואיך יתכפר לנו / על שגגותינו



אין לנו אורים ותמים / לשאול בהם / ואיך יאור לנו / ואוויינו מאפל.





בטלה עבודה / מבית עבודה / ואיך נעבוד לזך / וזר מעביד בנו



בטלו ממנו / אשים ואשם / ואיך נעש אשים / ואש מזבח כבה.





גיל גורלות / חדל ממנו / ואיך נעל בגילה / ואנו בגולה



גודרי פרצות / נפרצו באשמינו / ואיך נגדור פרץ / ואין עומד בפרץ.





דמעה ומלאה / דללו ממנו / ואיך נתרום דמע / ודמעה בעינינו



דם זריקת זבח / פס מבית זבח / ואיך נזרוק דם / ונשפך דמינו.





הקטרת ריח סמים / הוסר ממנו / ואיך נקטיר ריח / והבאש ריחינו



הטבת נרות / הדעכו מאוויינו / ואיך נטיב נרות / וכבה נרנו.





וניחח חדל / ממנוחתינו / ואיך נביא מנחה / ואין בית מנוחה



ובכורים / חדלו לבכר / ואיך נעל בבכור / ואין בכורה.





זבחי כליל / פסו ממכלל יופי / ואיך נעש זבח / וחרב בית זבח



זוללה / זלזלוה זרים / ואיך נבוא לנחלה / ונחלתינו לזרים.





חלבים / פסו מהקטירם / ואיך נקטיר חלב / ואין חלב חטה



חדלו ממנו / שעירי חטאת / ואיך נעש חטאת / וגברו חטאינו.





טבילת טהרה / אין בפרוה / ואיך נטבול בטהר / ופסקו טבילות



טורי אבן / פסו ממלואת אבן / ואיך נסיר לב אבן / וידו בנו אבן.





יער הלבנון / הושת למרמס / ואיך תלבב לדוד / כלת הלבנון



יפינו הושם / ואורינו אפל / ואיך תיפה לדגול / יפת עינים.





כפר פר / פס מאפריון / איך נקריב פר / ואין מי יכפר



כפרותינו / פסו מאוויינו / ואיך נגיש שי / ותמס פגרינו.





לבונה לא עשינו / בגבעת הלבנון / ואיך ילבן חטא / יפה כלבנה



לוג שמן פס / מקרן בן שמן / ואיך נצוח שמן / על בלולה בשמן.









מר דרור אפס / ממקטרת מור / ואיך בהר המור / ילין צרור המור



מחתה הוחתה / מלחתות גחלים / ואיך נעש מחתה / ואנו במחתה.





נרד וכרכום / נדד ריחו / ואיך נען: נרדי / נתן ריחו



נסכי שכר / פסו מהסך / ואיך ננסך / כנסכתו מראש.





סולת סלפה / מסדר חלות / ואיך ניפה סולת / ונלכדנו מסוללות



סמים הוסר / ממלוא חפנים / ואיך נעש סמים / והומסנו במסים.





עולה עתקה / מיער עולות / ואיך נעש עולות / והכבד עלינו



עצי ערך / חדלו ממזבח עץ / ואיך נטהר בעץ / וכשלנו בעץ.





פר ושעיר / פסו מלהשרף / ואיך נשרוף פר / ומקדש נשרף



פרכת פורכה / וכפורת כורתה / ואיך נפציח פלל / ואנו גם בפרך.





ציון ציה / ירושלים לעיים / ואיך נציב לנו / נציב ציונים



ציץ נזר נצל / ממצח מרצה / ואיך נרצה בציץ / וחסדנו כציץ.





קול מדבר / חדל מאהל / ואיך ננטה אהל / ושדד אהלינו



קרבן חדל / מטלה קרבן / ואיך נקריב קרבן / ושלם בחרבן.





רבבה הרחקה / רבוכה הקרה / ואיך נעש מרבכת / ואנו מרבכים



רחבה הצרה / ונסבה הסגרה / ואיך נרחיב פה / ואויבינו פלילים.





שלמים האנחו / שלמים שבתו / ואיך נשלם שי / ומשלום זנחנו



שבעה תמימים / תמו מארצנו / ואיך נהלל שבע / ויסרנו בשבע.





תפלה פסקה / מבית תפלה / ואיך נען תפלה / וסך בעד תפלה.



תמידים כלו / מיער תמידים / ואיך נעש תמידים / ובטל התמיד.
