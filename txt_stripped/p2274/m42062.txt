


	
	עזוז וגבור
	



סימן: עבדיה. לחן: אליך קוני עיני.






עזוז וגבור / ישבר / יד צר בו גברה / חי



ולכל הנופלים / עולים / מבור לאורה / חי



וצר מתגאה / יראה / יד אל לא קצרה / חי



רב להושיע

ישע יביע

אמר ישמיע / יריע / רני עקרה.





בחסדו יפנה / אודה / כי אין בלעדו / חי



הוא מלך עוזר / אזר / ישראל עבדו / חי



כל מלכי איים / גויים / כאין נגדו / חי



לרמים משפיל

לארץ יפיל

שמשם מאפיל / ומכפיל / סופה וסערה.





דברי צדיקים / יקים / נפשי שואלת / חי



ובלי מחלה / סלה / אמשיך תוחלת / חי



כי יש לי מציל / מכסיל / יפרש אולת / חי



שמני מרמס

הייתי למס

ועל לא חמס / לבי נמס / אפו בי ירה.





יצר לי אויבי / ובי / שואפי חרף / חי



הכרת את זרעו / גזעו / כי הקשה ערף / חי



כצל כנטותו / מותו / טורף טרף / חי



תרמס ככפיר

עצתו תפיר

יבאיש ויחפיר / הצפיר / עושה בעברה.





הבט בך נסמך / עמך / מאלפי זהב / חי



מהר שלח לו / שילו / אשר יאהב / חי



ויעיר כחו / רוחו / ומחץ רהב / חי



ישפט בצדק

טובם כחדק

מחזיק כל בדק / ומהדק / מרבה המשרה.
