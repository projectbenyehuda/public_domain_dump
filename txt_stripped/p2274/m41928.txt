


	
	עושה שלום
	



סימן: עבדאללה.






עושה שלום / משגב שמו

אל קץ עילום / יוחיל עמו

ישיב הלום / אל מקומו

מיטב כרמו / עשיריה.

אימתי / יפדה אל יה

ונשמתי / תהלל יה

לאל עליון / עליליה

על עין על עין.





בצוק עמים / שוכן אתם

לך נואמים / את רנתם

לרחמים / תתן אתם

ושועתם / חיש שמע יה.

אימתי יפדה





דבקה נפשם / אחריך

תן בראשם / משיחך

להורישם / את ארצך

ובנה ביתך / שלישיה.

אימתי יפדה





אל, שור רבים / לוחמים לי

כלם אויבים / ושונאים לי

ונצבים / להרע לי

ומחסה לי / ועזי יה.

אימתי יפדה





לשוא חוזים / יתאמרו

שוכבים הוזים / עת יגורו

עלי עזים / פה פערו

ויאמרו / לא יראה יה.

אימתי יפדה





לכן השב / גמול להם

מאין יושב / בעריהם

שמע הקשב / לדבריהם

עשה להם / פליליה

אימתי יפדה





האר פנים / לעם נבדל

חדש שנים / ישיש, יגדל

במעדנים / ולא ידל

עז ומגדל / מעשי יה.

אימתי יפדה
