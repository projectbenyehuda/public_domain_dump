


	
	עדות יי
	



למתן תורה. סימן: עבד יה.






עדות יי / קהל אמוני / נחלו אותה



ומשה עלה / עד למעלה / ובידו מלא / ספיר גזרתה



ואל אמונה / הוא כוננה / ימינו קנתה



חי, פריה שמר / יין חמר / ריחו לא נמר / עין לא ראתה.





בשמאלה עשר / כבוד ואשר / ימים בימינה



כי מוצאה ימצא / רוח עצה / יפוצו חוצה / יפיק תבונה



לשונו תצמיד / חכמה תמיד / נפשו אותה.

חי





דרשו השרים / לחם אבירים / שכיות חמדה



גנוזה במדור / לאלף דור / לאדם תדור / זאת התעודה



מה אדיר שמך / תנה הודך / לנו יאתה.

חי





יה נתן הודו / למשה עבדו / פתח בתשובה



מה לכם אישים / אתם דורשים / מצות אנשים / אשר ילכו בה



לקח מתנות / איש אמונות / ידו עשתה.

חי





הצל איש חכמה / מיד איש מרמה / וארור נוכל



חייו ימרר / פריו ארר / ובו יתגורר / יגיעו אוכל



שלח אלופו / ומרודפו / קומה עזרתה.

חי
