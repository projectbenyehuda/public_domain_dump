


	
	עלי עשור
	



לחתן. סימן: עבדאללה ח׳זק






עלי עשור / שיר תהלה / בשרי כמה



אהללה / דר עלילה / תוך סגלה / עדה תמימה



שם יה חתן / וכלה / יסד בחכמה

אל רב עצמה / ארוממה

בין ישרים / שמו בעזרי.





בחור מעם / נפשו תנעם / טוב מצא אשה



בת נעימה / ואימה / ואור חמה / שמה לבושה



ומחלצות / ממשבצות / זהב לבושה

וחן אישה / כתר ראשה

רב מאורים / גבור כארי.





דודי צלח / בכל משלח / ידך מטוב יה



ברב שמחה / והצלחה / לך שלחה / ברכת אליה



בהוניך / ואשתך / גפן פוריה

כעת חיה / תתן פריה

ציץ כתמרים / עשות כל פרי.





אל נאמן / ואב רחמן / לפניו אברך



חתן תמים / ארך ימים / לעולמים / ברך יברך



ובתורתו / ובצדקתו / ענוים ידרך

אויבים דורך / ישים מרך

בלב צרים / אל עז נאדרי.





לבנות עליון / האפריון / מקדש ההראל



במוצאיו / ובמובאיו / וזיו מראיו / מראות יחזקאל



ממרומו / בשוב עמו / ישמח ישראל

שאול שאל / ישלח גואל

בנים ירים / ישעי ואורי.





חזק חמל / חסד גמל / אתה תשלם



לאיש נעצר / עבד נמסר / ביד איש צר / כרחל אלם



שור לוחמי / ואמר בן מי / זה הוא העלם

מלך שלם / אל תתעלם

תיטיב שירים / משור פר ומריא.
