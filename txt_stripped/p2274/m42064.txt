


	
	פרוח תפרח
	



סימן: עבדאללה בר כצ׳יר ח׳זק






פרוח תפרח / מטה לבית לוי

ביאת נין זרח / לאל כל יום קוי.



עיר קרית חנה / בא יבא ברנה / וכבראשונה / שלות עולם שלוי



בסתר עליון / ישיב לבת ציון / נגה אור חביון / בנוה שלום נוי



דוד נצרתני / וכבן אמצתני / למה שמתני / אט למכים גוי



אל למושעות / הקרב לנו שעות / אתה אל דעות / הנותן לשכוי



לה קום גאלה / והפר האלה / ושור מלולה / משפחת הישוי



בר לבב אני / עבדך רמני / הוא הצפוני / שכם בן החוי



כציר נאמן / מרפא בבא זמן / איש רב חסד מן / מזור לבב דוי



חזק נרוה עד / הבקר נאכל עד / ותכן לעד / הדרי וזיוי.
