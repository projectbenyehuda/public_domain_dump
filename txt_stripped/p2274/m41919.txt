


	
	עלץ לבי
	



סימן: עבדיה.






עלץ לבי / בך משגבי / ושבחי

בקול זמרה / אזמרה / עם אחי

עם אחי

חי אלהים חי.





בכל לבי / דרשתיך / מורישם

מקום משכן / כבודך / בראשם

בראשם

שם יי שם.





דרכיך / הודיעני / ולאט

תנהל, צור / את עם עני / ולי הט

ולי הט

הט אזניך הט.





יפה נוף עיר / ציון תבנה / ותגדר

סכת דוד / תקים לשני / דור ודור

דור ודור

דור ישרים דור.





הטה שלום / לעבדך / כמי גל

ואביטה / נפלאותיך / עיני גל

עיני גל

גל כבודי גל.
