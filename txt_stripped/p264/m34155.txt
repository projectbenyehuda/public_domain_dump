


	
	לשד עירי
	







קינה בצורת שיר אזור. חתימת השם: יצחק ברבי ששת חזק.






לשד עירי / גדול כים שברי, / ולחמי ראש ותלאה.





       יחתני / צור חלקי ולבבי,

       יקימני / למטרה של אויבי,

ותחתני / ארגז, ובבשרי / אין מתם, ונגע נראה.





       צעדי צד / מלכת ברחובותי,

       ושם בסד / פעמי מרכבותי,

והן יסד / בציון אל עירי / פנת יקרת מסלאה.





       חמודותי / למרגיזי אל לנחלה,

       ודמעותי / למכתי כי נחלה,

ובחומותי / ובביתי, כבוד יערי, / באו רבים ונכאה.





       קרא אבל / ומספד – אל יודעי נהי,

       וציר וחבל – / אל כל עובר לו יהי

כלי נבל / לקול בוכים וכנורי, / לנחלתי כי נלאה.





       בריחיה / בעדי ארצי לעולם,

       פתחיה / נסגרו ואנו באבלם,

צמחיה / עבשו, ועיר הדרי / נגאלה והוי מוראה!





       ראות נקם / אקו בשעיר ובעמון,

       להתיקם / ליום הרגה, שכול ואלמון,

גמול לחיקם, / זרוע אלי עורי, / בממלכה החטאה!





       בהגיגי / תבער אש בקרבי.

       למבליגי / שד ושבר בכל סביבי,

אזכרה נגי/נתי ערבי ושחרי – / ואשתומם על המראה.





       יה, הקצור / קצרה ידך מפדות?

       וצר יעצר / בעמך בכבדות,



ואמרה צר / ״האח״, בנפל נזרי, / ״החרבה אמלאה!״





       שבי גבור / היקח או ימלט?

       ואם ישבר / מוטות צר, בי שולט?

בבית הבור / ירדה בי בחרי / ובכבד משאה!





       שבי נאנחה, / בת עמי, ושק חגרי,

       וגם קרחה / הרחיבי כמו נשרי,

ולשמחה / מה זה עושה, אמרי, / התקרב ותבואה?





       תהי מסעד / ועזר אל גולתי

       לבל תמעד, / ושובבה אל נחלתי,

ומגלעד / הביאה נא הצרי / לשארית הנמצאה.





       חזק נמהר / לבב נכאה להר ציון,

       ברוש תדהר / בנות היכל ואפריון,

בראש כל הר / יהיה נכון הרי, / ועל כל גבעה נשאה!




מקור: שירמן – השירה העברית בספרד ובפרובאנס
