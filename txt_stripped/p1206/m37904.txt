


	
	איש הביא אורח
	




האשה והאורח





איש הביא אורח לזבח הימים / ואשתו הראתה לו פנים נזעמים / בסאון רעש ושמלה מגוללה בדמים.



ויהי כבאי הביתה ונצבתי אל עבר הבית אל הפנה / ותשא אשת אדני עיניה אלי ותאמר בקול שאונה: ׳בן מי זה העלם אשר הבאת הנה?׳ / ויען אישה: ׳איש עובר ארח / ממשפחת אסיר אביאסף בני קרח / מתמנת סרח / על יום טוב הבאתיו הנה׳ / ותשא קולה ותאמר: ׳הלא שבע אלה עיני משוטטות אלי לתת אכלם בכל עת ועונה / אני ושני ילדים וילדה הקטנה / ועבד ואמה ואת עלית על כלנה / כי מאכלך בריאה, וחלקך בהמה שמנה / ויאמר לה הזקן: תן חלק לשבעה וגם לשמנה.



ותקם בחמתה ותלך מהרה / לכבות יקוד אש הבערה מן הכירה / היא מוצקת / ועיניה מורקות / ושניה חורקות / ויחזק בה אישה ויאמר: ׳הלא יום טוב היום ולמה תהיי לשמה ולקללה / ותרדי בדם שאולה / בעבור המכשלה׳ / ותאמר: ׳הנה כלכל לא אוכל / אך יליד בית המה יאכלו בלחמי וכל זר לא יאכל׳.



ויאמר לה אישה: ׳את טובת חן ובעלת שכל / ידעתי כי כל תוכל / יונתי רעיתי יפה את כלך / זיגא זבינא דמטי עד ארעא לך / אך יאכל הנער אתנו בערב, ואור לו וילך׳.



ויהי עד כה ועד כה הקולות יחדלון
ותנחם / וערוך השלחן וצפה הצפית / וישב לאכל לחם / חשך משחור תארם כאלו פעל בפחם / וישב הזקן במושב הקיר על מכונו / וישב לפניו הבכור כבכורתו והקטן כפי קטנו / ונצבה השגל לימינו / ומשרה המוציא ופתת פתים ויחלק להקרואים מקצה לקצה / ותשא קולה בריב ומצה / ותאמר: ׳למה תהללני לעיני עמי בפתותי לחם?

לכלם נתת לחם גדול כביצה / ולי לא הגיעני כפול / כי פליג בהמוציא / בחרפת גדופה ושמצה / כלה ונחרצה / בקול תרועות / בפרוע פרעות / ולקחתי הנותרת מן החלה / ופרסתי לי פרוסה / ואין עיניהם רואות / כי נתתי מגרעות / ואכלתי
והיא נקטה / עד אשר שקטה.



ויהי עד כה והביאו את המרק ויגמיאו ואני ישבתי / המה חשבו ודמו כי לקחתי בשנה ובשתי ממנה / אחרי כן ידה חלקה לי בכף אחת קטנה / קטן הוא מי יכילנה / זאת לא זאת אף הוא נשבר ונפלג / ולא יכלתי לשאב מלא פלג / זולתי כאשר יעלה המזלג / וכאשר תם והורק גמיאת המרק /
והביאה האמה / הבשר והירקות צח נעמה / אשפר ואשישה / ותאכל ותתן גם לאישה / חלק אחד מששה / ותאמר אלי: ׳גם לך אתן!׳ ותהי בעיניה כמבשר / ותצבט לי שני עצמים ועליהם כשני חצאי זיתים בשר / ותבט אל העוגה אשר פרסתי ממנה פרוסה והשתכחת חסרה / והרימה קול שאונה ותאמר: ׳מי אשר פתי אכל שניו תקהינה / ויגרס בחצץ שנו חיל בלע ויקיאנה׳.



ולא יכול הזקן להתאפק ויהס קולה / והיא נשאה קולה למעלה / ותהי המצה גדולה / הוא הסיר את המצנפת והצפת אשר על ראשה / ותכסה בושה / והיא תפשתו בידו / ותכהו את שנו / ועפרה בעפר את עינו / וכפישה לו בזקנו / וכאשר ראיתי דמיתי כי נעמי תאחז לו בזקנו לנשק לו / אך מקול מר הנשמע הבנתי כי נעמה בת גילו / עמדה למולו / ודלדלה בשערו / עד כי זב דמו מבשרו /
והיא היתה בעכרו / ושלמה אדר / בגדי הדר / קרעה בתוך החדר / ועדין המה מנגחים כשורים / וכיקר כרים / וכשני עפרים / ויעפרו בעפר / ויקומו כל הקרואים בוש וחפר / והרי שלחן והרי סכין והרי מזון, ואין אשר משם אכל חפר / ואושיט ידי אל הבריה המלאה והרחבה / וגמלתי חסד עם נפשי העלובה / ואכלתי ושבעתי עד שנתקררה דעתי / והלכתי אל המקום אשר תקעתי.
