


	
	אהללך בטוב
	



סדר אלפא ביתא

סימן: א״ב נסים.






אהללך בטוב מענה ומלה / חשב אותה כמו מנחה בלולה

בזמרתי אדון תבחר ותכסף / ואף אם לשגיאות היא עלולה

גבורות אל רצוני אזכרה בה / ואז תהיה בתוך לבי שתולה

דרכיך ה׳ למדני / ואלו הם דברי בתחלה

הללויה הללו כוכבי אור / ומזלות שנים עשר בצהלה

ושבעה כוכבי לכת ועמם / שני הקטבים עם העגלה

זריזים הם בשמושם לאהבת / רצון בוראם והם עושים בגילה

חסדיך אדון עולם מלאה / בחסדך שלח לי חיש מחילה

טהור לבב יהי נצב למולך / ואולי יהיה שואל שאלה

ישו אל חי לבדו הוא לנגדו / ומימינו בלי ימוט לסלה

כבודך הוא לעולם את יסדתו / בלי שפעך אזי תהיה חלולה

לדר ודר אמונתך יסודה / וכוננתה ותעמד בתחלה

מאד גברו חסדיך וגדלו / פלאיך עדי לשמי זבולה

נצר עדה טהורה היא חמודה / כימי קדם נצרתה בתחלה

סמך אותה שמך סומך ועוזר / עזר אותה וחיש קבץ קהלה

עשה עמה לטובה אות ואל נא / תשלם לה בראשה את גמולה

פדה ממצוקות צר וחלץ / נשמות עמך מן המצולה

צדקות תזכרה רחם והושע / לעמך שלח טובה כפולה

קדושי את מאד נשגב מרומם / עלי כל הברכות גם תהלה

רבבות מלאכי רום אם, יהודון / והחיות ואופן בהמלה

שרפים עומדים בשש כנפות / ומקדישים לשם נורא עלילה

תמימי עם בניב שועם ומלם / לעמתם תהלתם כלולה

נהלני בעזך וצדקך / שלח לעבדך מהר תעלה

סעדני ותנחני בדרכך / תהי דרכי לפני אז סלולה

יהי חסדך לעמתי ואתי / תקבל את תפלתי כעולה

משך חסדך עלי עבדך אקו לך / אהללך בטוב מענה ומלה.
