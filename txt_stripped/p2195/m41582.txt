


	
	נורא אתה
	



סימן: נסים. מנגינה: חוסיני






נורא אתה / קבץ עתה / יונתך תמה / תמימה



השב אותה / אל נותה / גרש האמה / בחמה,





סבו ציון / משכן עליון / והקיפוה / ספרוה



חומות רמות / גביש רמות / כל שעריה / עריה,





יה צור חבלי / וגורלי / בנה היכלי / וזבולי



ושכן עליון / באפריון / אל שמחת גילי / גואלי,





מהר אשכן / מקדש מכון / ידי כוננה / רב חנה



זכו עתה / יהיה עתה / חיש אחישנה / ברנה,



נאמן.
