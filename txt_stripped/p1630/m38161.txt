


	
	וירא הזמן כי נסתחפה : מקאמה
	








חתנת זקונים







– – – וירא הזמן כי נסתחפה שדי / ויבא לנוד לי ולסעדי / ולדבר על לבי ולנחמי / כי עורי חרה על עצמי / וירא כי גדל הכאב מאד / ויאמר: ׳זכור אזכרך עוד / את בתי אהבה המרצה / אותך עזבה ותצא החוצה / חיתך למות אל תתן / בשתים בי תתחתן / בת אחת יש לי שמה שיבה / חיתך תבלה בה / ומארתך ודמעך לא תאחר / ונהפכת לאיש אחר / קלקלתך עתה אתקן / לא תקרא עוד פרא כי אם זקן / וכבד אכבדך עוד כמלך בגדוד / תשב ראש לסעוד / בראש עם קדש תמני / ספר תקרא בנהני / אליך אל עצה יבאו /
יראוך נערים ויחבאו / מצות ׳והדרת׳ בך תקים / ותהיה יראוי ומאים׳.







ואמר: ׳אנא אדני אם תשאיבני מים בששון / היטבת חסדך האחרון מן הראשון / אבל דאגתי וכל מגורי / אולי לא תאבה האשה ללכת אחרי׳ / וישא ידיו השמים וישבע בחי העולמים / כי לא תעזב אותי כל הימים / ויקח הזמן שיבה בתו ויתנה לי לאשה / תחת זית יצא חוח ותחת שערה
אשה / כי מהשם היתה סבה / ויאמר לעשות לה כתבה / ברביעי בשבת / משושי שבת / ונרי כבת / בחמשה ימים לירח בול / שנתים אחר המבול / מאתים ועשר לבריאת עולם / אשתקל מלולי ואשב
נאלם / הגיעו ימים אין בהם חפץ נשיתי טובה / בא זקן ואמר למרת שיבה / הוי לי לאנתו ואנא אפלח ואוקיר / נגשש כהלכות גוברין / דמרירן נפשין יהו כגדין מררין / דמוקרין וזנין בקושטא / בקטטה חרטה ועלטה / ודא נדוניא דהנעלה ליה / הפסח והעור והחולה / השעול והקיא והבושה / דהנעלת ליה מבי נשא / וצבי ר׳ זקן ואוסיף לה מנה אחת אפים / פיק ברכים / כהיון שנים / כאב עינים ורפיון ידים / ושחין על הברכים ועל השוקים / סך הכל קבל עליו החתן / עם זוזי קצף מאתן / מאתים זקוקים למשקל עול ברזל
מנקים / מכל גדל ורחם משכיל ושדים צומקים / וצבאת מרת שיבה והות ליה לאנתו / בכוס אחד ישתו / כדת משה וישראל / ולא יפרידם קרוב או גואל / עד עמוד השר מיכאל.







וזאת הברכה אשר ברך הזמן זקן ושיבה / ויאמר: ׳היו לאלפי רבבה / מרבים להוציא מטבע / רק כיח פיע וניע / ורוחות פורחות לעת השינה / תחתיים ועליים תעשנה / ותמד מימיכם מצויות / גלות עליות וגלות תחתיות / ובצנורות
מקלחין במרוצה / יפוצו מעינותיהם חוצה / פיכם תתנו לחטיא את בשרכם / בלכתך בדרך ובשכבכם / ושרט תתנו בבשרכם / ומן החזה השיר יהיה לכם / אסטומכה מלאה ומעים רועמות / מעט שינה ומעט תנומות / תזרעו גרב ושחינים תצמיחו / תנובון בבנים ובשיבה תפריחו.







ויהי לעת ערב / בבא השמש וליל קרב /
וירח לא נתנה אורה / ויביאה הזמן אלי החדרה / ואקרב אל הנכאה / ויהי בבקר והנה היא לאה / ועיני לאה רכות / אפלות כהות חשוכות / שניה עקומות וחרוקות / צנומות דקות / מחוש השנוי שפתותיה / והלבינו שערותיה / כפלח הרמה רקתה / מבעד לצמתה / וצלעותיה ומפרקתה /
אחת גבוהה מחברתה / קמטים קמטים בפניה / וגדלו ריסי עיניה / חצי נשרו / ועד היסוד ערו / פיה בנוי לתלפיות / ברוך משנה הבריות / כמותה לא ראיתי כעורה / על עצמה צפד עורה / ויאכל חצי
בשרה / ועל רע פניה הדלים / כבר אמרו
המושלים:







מאד נכאה אשר יראה בעוד יום

פני שיבה ותבער בו חמתו

אזי יקום חצות לילה לדלג

חלון ביתה וישבר מפרקתו.







ויהי כאשר קרבתי אל המשנה / נתעברה מביאה ראשונה / ותהר לי באותה לילה /
את חולשה ואת מחלה / שתי בנות אומרות הב הב / הזלות מכיס זהב / ויהי בבקר ותפעם רוחי / ואמר: אבד נצחי / כי נראה בעיני הנגע / ועת ופגע / קראני וראש לחלי / כי באה שיבה בגורלי / מה חפצי
עוד בביתי / ואנה אוליך את חרפתי / העיט צבוע היתה לי נחלתי / ולא עצרתי עוד כח / כי תחת החטה עלה חוח / ואצעק אל הזמן ואמר: ׳אהה אדני / הן בצדק עבדתי עמך ולמה רמיתני / ותגנב את לבבי / ותסף מכאוב על מכאובי / כי מאז ערכי נגרע / והחלפתי טוב ברע / איך תשוה שיבה עם קמטיה / לחפצי בה ויפי להטיה / ואין עדני בשר לשיבה / כמתק עדנת חפצי בה / ועתה אדני הושיעה /
ומלטני מן הרעה׳ / ויען הזמן: ׳ההיטב חרה לך / הס אל תשמיע עוד את קולך / בתי שיבה אל תגנה / אל תבוז לקמטים שבפניה / ואל תבוש שיבה לעבד / כי נגד זקני כבוד / ואיך תהיה כפוי טובה / הן תפארת זקנים שיבה / ועתה רב לך אל תוסף דבר / עם בתי שיבה תתחבר / תדבק בך והיתה עמך / לא תוכל שלחה כל ימך׳. – – –
