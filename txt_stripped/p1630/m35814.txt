


	
	אמרו כי באחת האיים
	











מתוך ספר אחיטוב וצלמון


אמרו כי באחת האיים הרחוקים / אשר
מתחת למצוקים / מעבר לארץ החוילה / היתה אי אחת
גדולה, / ומימינה ומשמאלה / חומת אש סביב לה: /
סובב אותה נחל / בוער יום ולילה כגחל. / ודריה –
בעלי שני יסודים: / אש ואויר מצמדים / וממים
ועפר נבדלים, / על כן בתנועתם קלים. / ושם מלכה
אשה חשובה, / כבודה בת מלך ובת מלכת שבא, /
והיו עבדיה סגנים ושרים / והיא שופטה את עמה
בצדק ובמישרים. / והיה לה עשר וכבוד ונסיכות /
ומכרעת עצמה לכף זכות, / דוברת שלום לכל זרעה
ועמה / ופיה פתחה בחכמה, / מנהגת את עמה
הנהגה טובה / ושופטת ברוח נדיבה. / ועמה כלם
צדיקים, / אבל בלי נימוסים וחקים, / כי מקצתם
מעגלותם נעו / ואת יוצרם לא ידעו, / ומקצתם עובדים
לשר של מים / ומקצתם לצבא השמים / ומקצתם
עובדים לחמה – / ולא היתה ביניהם הסכמה. / והמעט
שבהם אשר בעין השכל רואים / את האלהים היו
יראים. / גם היתה המלכה אשה משכלת / משכמת
ומערבת ומתפללת, / בכל יום פעמים / נשאה ידה
לרום שמים, / אך רעיוניה היו תמהים / במה תעבד
את האלהים.



ויהי כי ארכו הימים והמלכה שוכבת
על מטתה / ומעיניה נדדה שנתה / ועל משכבה
שנתה לא ערבה / מרב נביכות המחשבה, / כי שנוי
הדעות הציקה – / ותעל בלבה עצה עמקה / להמציא
לעמה תקנות / לבלתי יהיו דעות שונות. / ויהי
בבקר ותשלח ותקרא לשרים ולסגנים / מרבים
המשרה ולחכמים והנבונים / יועצי נדיבות /
ולמבינים אנשי לבבות. / ויבואו וישתחוו
לפניה באימה, / ותדבר אליהם בחכמה.



ותאמר ״אתם שרי וסגני / עבדי
ונאמני, / הנה ארצנו בכל טוב מברכת / ועל כל
אקלימים נערכת, / ויוצרנו ברך את גבולנו – / מה
טוב ומה נעים גורלנו! / כי יי השפיע עלינו
טובתו / וצוה לנו את ברכתו. / ועלינו לשבח
ולתן כבוד – / ואנחנו לא נדע מה נעבד, / ואני
רואה בינינו דברי עוונות, / אשר בינינו דעות שונות.
/ לדבר הזה עצה עוצו / ומעינותיכם חוצה
יפוצו!״



ויקם אחד מן האנשים / נכבד ושר
חמשים, / ויאמר ״אדונתי המלכה תחיי, / שלוה
ורעננה תהיי! / הנה דבר עמק עולה במחשבך, /
הטיבות כי היה זאת עם לבבך: / כי האל בשמים
כבודו / אין אלוה מבלעדו / ועל כל בריה לעבד
בשמו, / להודות לו ולרוממו. / אבל קבלה היא
בידינו מישישים / שאין עבודה בלי נימוסים, /
והמשתחוה לזולתו / משחית נפשו ונחלתו. /
ואני שמעתי כי בקצוי ארץ הרחוקים / יש להם
נימוסים וחקים. / ועתה אם טוב בעיני אדונתי / קצר
עצתי ומען מלתי, / נמתיק סוד בינינו / ונשלחה
אנשים לפנינו / ועברו ימים ונהרות / לתור לנו את
התורות / ויחפרו לנו את כל הארצות / מקצה ארץ
ועד קצות / ויודיעו לנו בנימוסי העמים חדשות /
ללמד וללמד לשמר ולעשות, / ונהיה כלנו לאל
עובדים / שפה אחת ודברים אחדים״. / וייטב הדבר
בעיני המלכה ובעיני השרים / ויסכימו לבקש
אנשים ישרים, / בעלי מדות ונבונים ותמימים /
נכבדי ארץ וחכמים / משיגי האמת בתבונתם, /
ויבקשו ואלה שמותם: / הראשון צלמון האחוחי, /
והשני עקר הקרחי, / והשלישי היה אב בחכמה – /
אחיטוב מארץ תוגרמה. / ותשלח אחריהם המלכה /
ברוח נדיבה ובלשון רכה / ותשביעם המלכה
באל אשר בראם / בשמים ובארץ ובכל צבאם.









ותאמר אליהם ״עברו את הנהר / עלו זה
בנגב ועליתם את ההר, / ושוטטו בכל מחוזות
ולשונות / ובכל האקלימים והמדינות, / וחקרו אחרי
כל האמונות, / אם הם אחת אם הם שונות, / והשיבו לנו
דבר אמת בלתי נסתם / מנימוסי העמים ועבודתם״.
/ ויענו פה אחד ״אדונתנו הגברת, / נעשה כאשר
את דוברת״. / ויקחו האנשים רשות מן המלכה
וילכו הלם / ותאמר אליהם ״לכו לשלום!״





שלוחי הנעימים! / יתנכם לרחמים

שוכן במרומים / אשר לו תועפות

וחפציכם ישלים / וחסדו לא יעלים,

תקלו כאילים / וכצפרים עפות.

תסובבו הגבולים / וימים ונחלים

וחפשו כל אקלים / מדינות ונפות,

ועל כל המעשים / תחקרו מישישים

ודרשו נימוסים / ודתות היפות.

פנו לכם וסעו / למען תשמעו,

בשובה תושעו, / ידיכם לא רפות.





ויקחו האנשים צידה לדרך, / כי לא
ידעו לה קצבה וערך. / ויעברו את הנחל / אשר
היה בוער כגחל, / וילכו דרך ארץ החוילה /
ויבואו עד כלח היא העיר הגדולה. / וידברו
האנשים יום אחד / ״למה נלך שלשתנו יחד? / כי
אין אנו ומדינתנו נכרים / ומשפיקים אנו בין
ילדי נכרים. / יש לנו לחוש מעמי הארצות, / בעלי
מרמות ומועצות, / פן יהיו עלינו מערימים ומעלילים
/ ויתנו אותנו כמרגלים / ויהיו עלינו מתועדים /
לקחת אותנו לעבדים. / ועתה נקח עצה טובה / ונפרד
איש מעל רעהו באהבה / ולא ירגיש בנו איש כי
יכרה הבור / ונחלקה לנו את הארצות לעבר. / וזמן
תשובתנו יחד נקבע – / עד קץ שבע״. / ויסכימו
האנשים לדבר ההוא / ויפרדו איש מעל רעהו. /
ויעבר הראשון / בארץ הנגב מול נחל פישון, /
והשני ארח לו לקראת / הנהר הגדול נהר פרת, /
והשלישי לצד אסתן שת ערכו / לנהר חדקל
דרכו.



ויעברו האנשים נהרות וימים /
וישוטטו ארצות ואקלימים / ויעברו מדינות ונפות,
/ נוראות ויפות, / ובכל מדינה ומדינה עיר ועיר /
חוקרים מן רב וצעיר / על אלהיהם ועל דתם / על
נימוסיהם ועל אמונתם. / וימצאו כי האמונות /
לשלשה חלקים משתנות, / וידעו כי היהודים
למספרם / שומרים תורת משה בן־עמרם, / אך הם המעט
מכל העמים / ושפלים מכל האמים. / והאמה
הישמעאלית בכללה / מאמינים כי מחמט רשות
השני לאלוה / ואלוה שמו בעולם ממנה / וירכב
אותו במרכבת המשנה. / ומלכות רומי הם הנוצרים /
אמונת ישו הם נוטרים / ומאמינים גדולם וקטנם /
שסבל מיתה כדי לפדותם מהגיהנם.

וצלמון עבר בכל ארץ הצידון / ויבוא עד
הר שיאון / ובכל ארץ שבילייא עד השיחור / והירדן
יסב לאחור, / ויעבר עד בית חרון / ויבוא עד חברון.
/ וירא שם המערה הגדולה / הנקראת מערת
המכפלה / אשר שמה היו האבות קבורים, / בעלי
נטיעות המה היוצרים, / ועל פני כלה בנין נאה /
ובית גדול למראה, / ונותנים שם לדלים צדקות /
ושם היו תמיד עששיות דולקות, / יום ולילה לא
יכבו / ורבו כמו רבו, / ומקטרים שם לבונה וסמים
/ וכל ראשי בשמים. / וירא צלמון בנימוסי העדה /
ותכנס בלבו העבודה / ויאמר: אך זה היום
שקיויתי / יגעתי וגם מצאתי! / זהו עקר האמונה
והמעשה / ואחרי אמונה אחרת לא אשאל ולא אנסה,
כי ודאי נימוסיהם ודתיהם / הן אשר יעשה אותם
האדם וחי בהם. / ויפשט צלמון את בגדיו /
וילבש מעל למדיו אדרת שער עד רגליו, / והיה
אזורו מוכיח עליו, / ובעטיפת ישמעאלים על שפם
יעטה, / עוד מנימוסי ישמעאלים לא יטה, / ושבע
פעמים ביום נופל על ברך, / כורע למחמט
ומברך.



ועקר גם הוא נוסע דרך ערביא / ויבוא
עד ים אנטוכיא / וישם פניו אל ארץ ארמניא /
ויבוא עד קושטנטינא, והיא היתה עיר גדולה מאמונת
נוצרים / אשר סוחריה שרים. / ואחר שם ימים שלשה /
לעשות חקירה ודרישה / על אמונתם ועל דתם. /
ויראוהו בבית נכותם / עץ אשר ישו בו נתלה /
ואומרים שהוא טוב לכל מחלה / וכי ירפא אשר חץ
מחצו / כי יתן בכוס עצו. / והראוהו כתר קוצים
שעל ראשו / יום הערה למות נפשו, / ומסמרות אשר
נתקעו בין רגליו / כאשר הנוצרים מאמינים עליו. /
ועקר טפש כחלב לבו / ותבוא האמונה כמים
בקרבו / ובעצמותיו כמו שמן – / כי פתי לכל אשר
ישמע יאמן! / ויספרו לו איך היהודים עטרוהו
ויקצרו ימיו, / ויבקש עקר לבכות כי נכמרו
רחמיו. / ויבקש עקר להודיעהו ולהורות / את החקים
ואת התורות, / ויאמרו ״אם תבקש לדעת עקר
האמונה, / לך אל רומי המדינה: / תלמידי ישו שמה
נתגדלו / ותורת ישו שמה העלו״.



והאיש עקר לא נתן לנפשו דומה / עד
הגיעו למדינת רומה, / כי היה מאמין לכל הדברים /
ומתפתה לכל הספורים, / ולא היה מעמיק במחשבה
/ להבחין בין דרך רחוקה לקרובה, / להבחין בין
האמת לבין השקר. / ויהי כי הגיע שמה עקר /
ויבוא עקר בתוך הבאים / וירא שם בנינים
מפלאים / בנויים לשם ולתהלה. / ויבוא על הבמה
הגדולה / אשר נקראת על שם קדש פיטרי, / תלמידו
של ישו הנוצרי. / ויראוהו שם מטפחת / בבית
גנזיהם מנחת, / קבלה בידם אב לבנים / שהנוצרי
קנח בה פנים. / וילך עקר מכומריא לכומריא /
מבית כריא לבית גליא, / ויהי הולך ומפשפש /
אחרי אמונת אותו ישו מחפש – / עד כי היתה בלבו
שמורה / ונעשה כמר לעבודה זרה. / ויגלח ויחלף
שמלותיו / ויתן עליהם אותותיו / שתי וערב על
לבושו, / ויקרח קרחה בראשו, / וילבש לבושי
הכמרים / וידבק בדת הנוצרים.



ואחיטוב גם הוא עבר מעברות / ויסובב
ימים ונהרות, / לא הניח גוי וממלכה / אשר רגלו
לא דרכה. / וילך במדינות כמין גאם / לדעת
באמונות פתגם: / כי אמונת מחמט בלבו לא עלה /
היות בשר ודם בזו המעלה – / ויחשבה לחוכה
ולטלולא, / ולאמונת ישו לא פנה, / כי אמר: בלתי
יסוד ופנה / היות אלוה נולד מאשה / ויצא ממקום
טנפת ובאשה, / והיות אחר כך אלוה מת – / קרוב
לשקר ורחוק מן האמת!



וילך ונבוכים רעיוניו / וישת אל
המדבר פניו, / וילך מהלך גדול וחשוב / עד
שבא לכלל ישוב. ולא עבר דרך שדות וכרמים /
מהלך עשרה ימים, / עד הגיעו לעיר גדולה ברמש, /
וילן שם כי בא השמש. / בלילה ההוא חלם אחיטוב
/ והנה הוא עומד בגן רטב, / ובגן היה אילן סמוך, /
שפל מאד ונמוך, / אבל היו רבים פארותיו / והיו
מרבים פרותיו. / ויקרב אל האילן לטעם טעמן /
וימצא בהם טעם מן. / ויהי כראותו כי אורו עיניו /
וילקט מהם מלוא חפניו / וימלא מהן תרמילו /
ויתאב לשבת בצלו. / וייקץ אחיטוב והנה חלום /
ויאמר: חלמא טבא חזאי, יהי שלום!



וישכם בבקר ורעיוניו נבוכים / ולבו
לצאת אל הדרכים, / ויעבר לפני בית גדולה /
וישמע שם המלה. / ויאמר בלבו: נבהלו רעיוני /
מה הקול הזה באזני? / עתה אסורה נא ואראה. /
ויעמד פתח הבית מחריש ומשתאה, / וירא שם
עדת אנשים / כלם מעריצים ומקדישים / ומתפללים
באימה וביראה, / וישתומם על המראה.&gt;
וירא עוד כי אחר התפלה / עמד זקן על
גב מעלה / ובהלכות היחוד להם דורש / ופרק אחד
שונה ופורש. / ויהי ככלותו לדרש לקהלו / וילך
כל אחד אל אהלו, / והזקן לחזר למעונו פנה / עבר
בשוק אצל כל פנה. / והנה לקראתו עדת דבורים /
מקטני בני הנוצרים / ויסקלו בו שועלים קטנים /
וירגמוהו באבנים / ויקראו ״כלב כלב!״ / אל
הזקן עלוב ולא עולב. / וירא אחיטוב ויבטלם /
ויגער בהם ויקללם. / ויהי כי הקריב הזקן אל
ביתו – / והנה איש בליעל לקראתו / ויתמרמר
לנגדו / ויתפשהו בבגדו / ויחרפהו בחץ
לשונו, / ויגזל ממנו מטרונו / ויך אותו בחרון
קצפו./ ויסר ממנו כפו / ויאמר ״הבה לי דינר
כסף / או ראות מטרונך אל תוסף!״ / ויפתח הזקן
כיסו והדינר יצבט / וישב לו האיש את
העבוט.



וישב הזקן לביתו / ויהי מעט בבית
שבתו – / ויקח השוטר עבוטו / ותמיד שכמו לסבל
יטו, / כי נתחיב לשר הבירה מס, / ולבו לב ארי לא
המס ימס. / אך אמר לעלוקה ״הב הב״, / ויפדה את
עבוטו בשלשה שקלי זהב / ושכמו לסבל נטה /
ובלבו ובשפתיו לא חטא, / רק אמר ״יהי רצון שיהא
גלותנו / כפרה לנפשותינו!״ / וישב הזקן אל
המסבה / והנה אחיטוב בא / ויתיצב שם עמו /
וזה דבר נאומו:





אתה אנוש ישיש אשר אלי מכים

נותן לחי והון לכל אשר יתבע.

לי נסתרה דרכך – מה נשתנית

אשר מכל איש בוז וגם חרפה תשבע?

אחשב היותך בא מתולדות כהן

מדין אשר רועים גרשו בנות שבע.





ויאמר לו ״יהודי אתה?״ / ויאמר הזקן
״כן דברת״. / ויאמר ״עוד מצפון לבך ממני אל
תכמין / הודיעני במי אתה מאמין״. / ויאמר לו ״אני
מאמין באלהי אברהם יצחק ויעקב״.
