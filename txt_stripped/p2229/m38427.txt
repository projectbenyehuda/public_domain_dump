


	
	[אקרא צור עליון יושב סתר חביון (תחנה)]
	



[מרומא, מת אחרי שנת 1280]






תחנה על־צרת הזמן





אקרא צור עליון / יושב סתר חביון

אל גמר עלי / תגמולוהי עלי.





צור עליון אקרא / עונה בצרה

בך היא תוחלתי / נאזר בגבורה

מלוחמים רבים / תהיה לי עזרה





בקמים עלי / סביב שתו עלי.





מה-רבו צרי / רודי שודדי

רבים קמו עלי / כלם היו עלי.





צרות כמה מסים / איך אוכל שאת

מדי שנה בשנה / מתן אשא משאת

לשחק עושים / בשאט נפש ישאו את





רדידי מעלי / הפסים אשר עלי.





דמו לעבדי האליל / כהני הבמות

התעיבו עלילות / תוך און ומרמות

כמריבי כהן / מעשרות ותרומות





מגדילים עלי / מחשבותם עלי.





קרבו רודפי זמה / הם מנהיגיהם

קרבם קבר הוות / חץ שחוט פיהם

כוננו חצם לירות / מן-מחבואיהם





יתלחשו עלי / כל-עצתם עלי.





יצאו מיני מינים / שועלים חובלים

מהגים לובשי שקים / ויחפים חובלים

דורשים מקלות לרדות / נעם את-חובלים





פצו פיהם עלי / יסלו עלי.





הן מהם פרח / שונא בלא רחמים

קץ מנד נושך / בשר ועצמים

אח לצרה ילד / ולאף וזעמים





הגיוניו עלי / לא יחמל עלי.





הצילני נא / אבי אביר יעקב

מיד אחי אכזר / עקוב יעקב

להשמיד שאר ישוב / שאר יעקב





רחקו מעלי / עמדו עלי.





ריבה אל ריבי / שפטה משפטי

מצר איש חרמי / בעל משפטי

נתן לבו ועיניו / מעמדי למוטטי





אויבי עלי / הגדיל עלי.





ואני תמיד עמך / תאחז בימיני

אל תניחני / לכופרים ומלשני

מחריבים ומהרסים / יצאו ממני





נאספו עלי / שחקו עלי.





פצה-פה ומצפצף / אחרי חוקר

על-שרשי רגלי / מתחקה ומבקר

יפח חמס / ומקים עדי שקר





ההפך עלי / לבו עלי.





אחת היא / על-כן חשבתי אמרתי

אמנה אנכי / חטאתי הרשעתי

כל-זאת באתני / כי בברית שקרתי





במוט רגלי עלי / תכבד עלי.





חי צורי וברוך / האל הנאמן

זכרה חסד וברית / אל-עם לא אלמן

קרבה אל-נפשי גאלה / אב הרחמן





בקרב עלי / נוסדו עלי.
