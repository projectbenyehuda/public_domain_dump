


	
	[שכולה גלמודה סערה עניה]
	



Samuel da Castiglione

[חי במנטובה באמצע המאה הט"ז]





פזמון




שכולה גלמודה / סערה עניה

מדי העבודה / לך היא הומיה

ותמלא כדה / מדמעת לחיה

ונאמר כי פדה / והנה בשביה

עד־מתי העדה / תהיה בזויה





ואתה אבינו / ובניך אנו / אם תעזבנו / מי יקחנו



למה יי תשכחנו / יי חננו לך קוינו.





מיטב שדך / השמיד הברד

וצאן מרעיתך / מפזר ומפרד

כל־נדיב עמך / מכבוד הורד

הנהיה כזעמך / מן־יום בה"רד

עד־מתי מחמתך / לבנו יחרד





ואתה רוענו / וצאנך אנו / אם תטשנו / מי יושיענו



למה יי תאנף בנו / יי חננו לך קוינו





ואנחנו מה־ / נשיב לשואל

כי תולעי אדמה / עם פרס וישמעאל

יוצאים ביד רמה / לעיני כל־ישראל

וארצנו שממה / עם גולת אריאל

עד מתי לכלמה / תחרש ותשקט אל.





ואתה מלכנו / ועמך אנו / אם תשכחנו / מי יזכרנו



למה יי זנחתנו / יי חננו לך קוינו.





למשל ולשנינה / נזכרים ונעשים

בכל־שנה ושנה / אצים הנוגשים

ירושלם כאלמנה / בעלוה חומסים

קרית דוד חנה / שסוה שוססים

עד־מתי ועד אנא / לא תושיע חוסים.





ואתה נוטרנו / וכרמך אנו / אם תרחיקנו / מי יקרבנו



למה יי תעזבנו / יי חננו לך קוינו.





הרופא לכל־בשר / הוא יפליא לעשות

יוציאנו ממאסר / במופתים ובמסות

מכל־נגיד וטפסר / ביי טוב לחסות

טובנו לא יחסר / האמת אין לנסות

עד־מתי נבשר / מפנים חדשות





ואתה אלהינו / וסגלתך אנו / אם תגאלנו / מי ילחם בנו



למה יי תריבנו / יי חננו לך קוינו.
