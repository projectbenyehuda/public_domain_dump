


	
	מחברת "נאום אשר בן יהודה"
	



מחברת “נאום אשר בן יהודה” / שלמה אבן צקבל




הכון, אנוש, לקראת אלהיך, ואש־ / רי איש מצאו בר נקי כפים!



היום ליום מחר חסדים קדמה – / ולמחרת תקח שכר יומים.



תנעם בעולם זה ועולם בא, ומי / או מי אשר יתחתנה בשתים!


  


נאם אשר בן יהודה: ויהי היום ואשב בשער בקרב אחי וחברי, וכל איש ואיש משיב בדבריו / ומגיד משפטי אמריו. / ואמר: את חטאתי אני מזכיר היום! / הייתי בימי נעורים / כאחד העפרים / דולג על ההרים / ושוכן ביערים, / שמה חדשי ושבתי / ואין איש אתי, / כתנת פסים כתנתי / ויעלת-חן חבלי ומלכדתי. / וחלפו עלי זמנים / ועברו ימים ושנים / עד אשר בלתה מעלי כסותי / ונכספתי לביתי. / ואבוא העיר לילה, וכל אנשי גאלתי / וכל אשר בביתי / טרם ישכבו, באתי אליהם / ודפקתי הדלת עליהם. / המה מתיאשים ממני / ואני אקרא אליהם “הנני!” / ויפג לבבם בקרבם / ולא האמינו, / עד אשר האזינו / והכירו קולי. / ואז פתחו לי / וינשקו על פני / ועל עיני / ויאמרו לי “נחיה אחרי ראותנו פניך / ועמדנו לפניך!” / ורחצו ידי ורגלי / והסירו הבגדים הצואים מעלי / והלבישו אותי חליפות יפות / באהבה רצופות / וישבתי עמהם שלשה ימים ושלשה לילות / לאכל ולשתות.



וביום הרביעי יצאנו הלומי יין / והכל לנגדנו כאין, / מתנהלים על מי מנוחות / ושואפים רוחות צחצחות. / ועל התעלות – / יעלות / יפפיות / ויונים הומיות, / אני מתבונן אליהן / ומאזין לקוליהן, / והנה כוכב משגיח מן החלונות / ומציץ מן הפנות, / קורץ אלי בעינו / ומולל בימינו. / התאפקתי לעיניהם / והתהלכתי ביניהם, / עד אשר עברו את המעבר / ויצאו אל יד המדבר. / נמלטתי מהם כצבי מיד / ושבתי אל המקום מיד. / אני חולף ועובר / והנה קול מדבר /“קרב עד הנה ואדברה אליך / ואשפך את רוחי עליך”. / קרבתי / ולעמת הקול נצבתי – / ויפל עלי תפוח / בשמן מר משוח, / על צדו האחד כתוב לאמר:


  


אשר ירוץ לצודד העפרים / במדבר ים והרי הבתרים –



חדל, כי הם כלואים בחצרים / ונמצאים בחדרי החדרים!


  


ובצד השני כתוב לאמר:


  


אשר שכר ביין חמר ותירוש / וינוע כעץ תמר וכברוש –



לך ישכר וינוע לבבי / והאהבה תשדד־בי ותחרש!


  


וכאשר ראיתים / והבינותים, / גבהה נפשי / ונגהה בעיני שמשי, / ואמרתי: "על כן נקרא שמי אשר, כי אשרוני בנות / ויאהבוני עדינות. / מי זאת אשר ראתה צביי / וחשקה ביפיי? / חי נפשי, בארך נדודים אכעיסנה / ובאהבתי אניחנה / ואעזבנה בחילה / ואדעה מה יהיה לה!” / ונטיתי מעליה / ולא פניתי אליה. / וכאשר התהלכתי / ומן המסלה ההיא נמשכתי, / הוצאתי את התפוח / אשר היה אלי שלוח, /שבתי לקרותו/וביפיו לחזותו,/ דבקו בלבי הבתים / ופתות אותו פתים. / נחמתי / על אשר ללכת קדמתי / ואמרתי: מה זאת עשיתי / אשר לא עניתי? / הרעותי / אשר לא השיבותי! / ואשב היום והלילה, מים לא שתיתי / ולחם לא אכלתי / ותנומה לא ידעתי. / ולעלות השחר שחרתי / ולא התאחרתי / אל מקום הקול / אשר מאתמול, / וקולי נשאתי / ועד העליה קראתי:


  


שלום לבת רבו מצוקיה, / שלום לעלמה מחשוקיה! שלום ליעלה מבנות ארמון – / האהבה דתה וחקיה!



אם אהבה לעדות יפי שירי – / חוטי חרוזים אעניקה,



או צֽמאה אל מי ידידותי – / הנה חלב אהבה אניקה,



או נכספה לראות הדר פני – / אבוא ואשתעשע בחיקה.


  


כליתי שירותי / ותמו דברותי, / ועמדתי משתאה / עד אשר אדעה ואראה – / והנה אין קול ואין עונה / ואין קשב ואין פונה! / וכאשר גדל כאבי / ורב מעצבי, / שבתי במחשבי / וגערתי בלבבי: / למה תתראה / לכל אשר אתה רואה, / ולמה תתגאה / ותגע עדיך ותלאה? / למה לא עמדת / כאשר חמדת? / ואיך רדת / ועתה נעבדת? / התקושש / והתאושש / או התבאש / והתיאש! / ולבי קט לגערתי / והשיב את אמרתי:





נפשי – אמתך – נבהלה / ובפח מארבים נֽפלה.



מה תגערה בי – ואני / נלכד בחשק יעלה?



מה יש בידי לעשות? / עיני לנפשי עוללה!


  


עד אשר רחמתיהו / ודברתי אליו ונחמתיהו / והוא הולך ושומם / וימאן להתנחם. והתהלכתי כל היום ההוא על היאורים ועל האגמים / ועיני נוזלות זרמים, / והגיתי כל הלילה ההוא קינים / והגה והי, אח הייתי לתנים / ורע ליענים. / וכאשר זרחה החמה / והאירו פני האדמה, / שבתי אל המסלה / אשר עמדתי שם בתחלה, / שחתי / ונאנחתי / ועניתי ברן / וקראתי בגרון:


 


שלומותי שאו נא אל עדינה,/לעלמה נשקפה כדמות לבנה, ואמרו נא באזנה, כי קרואה –/אשר קראה תמול – שמע וענה, והאהבה אשר חרשה בלבה – / בלבו נמצאה היום צפונה,



אבל הוד תארה הניא לבבו / ביום נגלות, ולא מצא תבונה!


  


ועל משמרתי עמדתי / וכל היום שקדתי, / צפיתי ברק יעבר־בי / לראות מה ידבר בי. / וכאשר ראיתי היום פונה / ולא מצאתי מענה, / צעקתי / ולא התאפקתי / “מה לי איחל, ותוחלתי / הלא נכזבה! כי עת אקוה לאור – תוסף אפלתי. / נפשי בפח צביה כצבי אשר נשבה, / דרכה קשתה באיתן ולא שבה / לבד כי לקרב עת אסיפתי / חצי נדודי / תשלח תוך כבדי, / ואור תבעיר בקרבי ולבת אש בלבתי, / גזלה שנתי / ותשבע לבל תתן פוגת לעיני ולא מרפא למחלתי”. / והתלוננתי באחד הרחובות / אשר היו אלי קרובות, / סר וזעף / יגע ויעף. / וכאשר כשל כחי / וקצרה רוחי, / הוצאתי את התפוח בכפי / והגשתיהו לאפי, / הריחותי בשמיו / וחייתי במנעמיו. / וכאשר נגלו מפרשי ליל ונטיו נחלי יומם, נקריתי / אל המקום אשר נקראתי / ועמדתי / במקום אשר נלכדתי / ונשאתי קולי בבכי / בדמעות עלי לחי:


  


בחי האהבה, יעלת אהבים, / בנדבת יֽולדך, בת הנדיבים,



בשושני לחייך, במלי / שפתיך אשר המה ערבים,



בתפוח בראו אל להרגי, / בטורים הם באצבע אל כתובים – הלא תטי לדודך רחמים, פן / ימיתוהו בפרודך אהבים,



ויחרקו לחליו שן משנאים / וילעיגו עלי מותו מריבים?


  


ובקצה מעני / נבהלו רעיוני / ונפלתי על פני / ולא מצאתי חזון מאדני. / ונאלמתי / ונרדמתי, / ולא קמתי, / עד אשר הקימוני / והרימוני / ידי נשים רחמניות / מנשי העליות / ופקחו עיני / ואמרו באזני / “בן אדם, עמד על רגליך / ונדברה אליך!” / ותבוא בי רוח ותעמידני על רגלי ואמרתי / “תדברנה גבירותי, כי התאפקתי / ובדבריכן התחזקתי”. / ותאמרנה “שפחותיך אנחנו / ועדיך שלחנו: / שלום לך שלום מאת הצביה, / היושבת בעליה! / שמעה בכיתך / וראתה דמעתך, / והנה יצאה אליך בסודה / ושלחה כתב ידה”. / ואראה והנה בידיהן אגרת / מהדרת / קטנה ככף איש / יקרה משהם ותרשיש / ועליה שמני מר, / כתוב לאמר:


  


דופק עלי דלת ולא / מאז בפניו ננעלה –



רק הוא אשר תעה וח־ / לף לו באפס תוחלה.



אם אפתחה – איך אשכחה? / או אנעלה – איך אֽוכלה?


  


וכאשר קראתים / והבינותים, / נכר אלי מהם תוכחת / ואש בקרבי מתלקחת / ונהפך עלי לבי ורחמי / יחד נכמרו נחומי. / שלחתי ידי / והחזקתי בבגדי / ואמרתי לקרע אותו / ונבצרה ממני שפתו, / כי לא היה בי כח. ותאמרנה אלי “מה זאת ועל מה זאת? / למה חרדת אלינו את כל החרדה הזאת? / שוב אל נפשך / ועזב את לבושך! / התעורר / והתגבר / והתאפק / והתחזק, / ושמר אשורינו / ולך אחרינו, / כי למען הנחותך באנו”. / האמנתי אליהן / והלכתי אחריהן, / ולא זכרתי דברי קהלת (הולך אחריה פתאם כשור אל טבח יבא), / עד אשר עברתי הדלת / ובאתי באש ובמים. אז רפו ידי / וצדו צעדי / ונבהלתי / וחתרתי / להשיב אל היבשה – ולא יכלתי. / ותאחזני זלעפות / והנה קול קורא "חזקו ידים רפות. / וכאשר שמעתי, חיתה נפשי / וכבתה אשי / ועניתי “זכר־דבר לעבדך על אשר יחלתני, / זאת נחמתי בעניי כי אמרתך חיתני”. / והאמהות ההן בהבלי פתיות משכוני / ועל במותי יעמידוני / והוליכוני בחצריהן / והביאוני בחדריהן / ואל העליה העלוני / ואל פח יוקשים הפלוני. / אני הולך בצד העליה, / טרם אבוא אליה / ואחסה בצליה, / שמעתי אחת משננת / ובכנורה מנגנת / “יבא דודי לגנו / ויאכל פרי מגדנו” / וענתה אחרת / ועל נבלה משוררת / "מרבדים רבדתי ערשי חטבות אטון מצרים. וכאשר באתי אליהן / ונגליתי עליהן, / ותחרדנה אלי / ותשתחוינה לפני / ותאמרנה “בוא, ברוך יי!” / וישבתי על מפרשי חור תכלת וארגמן, / מעשה ידי אמן. / הנה משחקות / והארץ לפני מנשקות / וכל אחת ואחת מבשרת / “עתה תבוא הגברת”, / עד אשר ראיתי פני כלן זועפות / כפני כבשן שרופות. / שחתי אליהן “מה לכן / ולמה נשתנו פניכן? / היש בלבבכן מטמון? / ותאמרנה “הנה אדון הארמון!” / ואשא את עיני / וארא בא לפני / איש כלו מלא זין, / משתכר ולא מיין. / מצנפת בד על ראשו / ובאש קנאה בערה נפשו, / פניו לוהטים, / כאחד מבני פלשתים / מתעטף ברדידו, / וחרבו שלופה בידו. / קמתי והשתחויתי / וקראתי חטאתי וישר העויתי” / ואחר כן קדמתיהו “שלום, שלום!” / ויאמר “חדל לך ולשלום / ואמר: מי הביאך הלם? / ככה יגלו סתרים / יבעו צפוני חדרים? / אמר, מי זה העניק אותך גאוה / וכסה את עיניך בתאוה?” / נפעמתי / ודוממתי / ולא מצאתי כח לדבר / ומלים לחבר. / ויוסף ויאמר “הבגדת – / ועתה חרדת? / וכחשת – / ועתה התבוששת? / והתקוממת – / ועתה נאלמת? / אמר, על מי בטחת / בעת התלקחת? / ובמי נעזרת / ולא יגרת? / ומי הסיתך בהבלים / והפלך בחבלים? / ברית נדיבים הפרת / וסתר מלכים קרעת – / חי נפשי, כי בן־מות אתה, / והיום הזה בא עתך / ואראה במפלתך”. / והחזיק בי ויעירני / ויתפש חצני / ויאמר “פשט בגדיך / ותאכל מפרי ידיך”. / ויהי לבי כדונג / ונפשי עלי תתמוגג, / חבלי מות אפפוני / ומצרי שאול מצאוני. / וכאשר ראה כי יראתי וזחלתי, / פן תדבקני הרעה ומתי, / התגוללו בדמעה עיניו / ויסר את המסוה מעל פניו / ויקרא באזני “בן־אדם, פקח עיניך / וראה את המראה אשר לפניך!” / התבוננתי – והנה אור בהיר / ולחי כזהר הרקיע יזהיר. / התעוררה לשוני / ואמרתי “מה זה אדוני?” / ויאמר "בי אדוני, אני מיעלות החדרים / ואילות החצרים, / לא שכנתי ביערים / ולא ידעתי הרי בתרים. / אני מבנות ארמון, / בחיק היקר אמון.



אני עפר בעפעפיו יצודד / אנוש חיל ויפל בחבליו – וחנתי אשר אחן ואשמר, / לבלתי אמנעה טוב מבעליו".



וכאשר הבהילני מאורה / כסתהו בשערה / ואמרתי “ברוכה את ליי, בתי, היטבת חסדך האחרון / מן הראשון! / האת אשר מן העליה עניתני / ואחרי כן עניתני / ותחת התפוח עוררתני / ואחרי כן הרגתני?” / ותענני “לא, אדוני, היא גברת הגבירות / ומחפרת המאורות, / היא נעם השפתים / ותאות העינים. / אמשיל – וחלילה להמשילה! - לאור חמה בשמים, / עדי תהיה לשבעתים! / אני שפחה משפחותיה / ואחת מאמהותיה, / שלחה אותי לשעשעך / ולדבר על לבך ונפשך׃ / כי אמרה אקדמה פניו במנחה ההולכת לפני / ואחרי כן אראה פניו – אולי ישא פני. / לרקמות תובל למלך – / בתולות אחריה רעותיה מובאות לך”. / וטרם כלתה דבריה, שמעתי קול המון / וצלצל פעמון / והריחתי קנה וקנמון / וארא והנה שפעת עדינות / מבחורי בנות / ובתוכן



עלמה משכמה ומעלה גבוהה על כלן / ונס מפניה צלן. / כלילי מלכות כליליה / ופעמונים בשוליה, / בצעיף גאוה מתעטפת / וכסעיף הדס מתנופפת, / בלאט מתנהלת / ועל רעותיה מתגדלת, / והנה יענו אותה בקולות, / בתפים ובמחולות / “אמרי נא המרך חמשך / יקרך / או שכרת מתירושך”. / ולא ראו עפעפי עיניה, / כי כסתה פניה. וכאשר הושיבוה על מכוניה / יצאו מלפניה, / ונשארתי אני לבדי / והעלמה יושבת לצדי. / ואמרתי בלבי: החלק לשונך מחמאותיך / תקרב ועשית כחכמתך! / וסבתי פני אליה / וקראתי עליה / “לבבתני, אחותי כלה, לבבתני באחת מעיניך, / באחד ענק מצורוניך. / גלי, אחותי, מאורך / ופרשי עלי שערך, / הסירי המסוה מפניך / והראיני את מראיך!” / וכאשר הסירתי הצניף / וגליתי הצעיף, / ראיתי זקן ארוך / והמות עליו ערוך, / ופיו פתוח / כסיר נפוח / והוא יאמר "איך ראית הלחיים / ואדם השפתים! / אז נבהלתי / ומלא קומתי נפלתי / וחפו פני / וברחו ששוני. / וישא קולו בצחוק, / עד אשר נשמע למרחוק, / והחזיק בי / וידבר על לבי / ויאמר “התיצב / ואל תתעצב! / אני רעך העדלמי – / קום שתה עמי. / הלא ידעת כי נחש ינחש איש אשר כמוני?” / ואפתח את עיני – / והנה הוא חברי / ובשר מבשרי. / ואמר לו “הלא ברחל עבדתי עמך ולמה רמיתני / ועכר עכרתני?” / ויפצר בי לנחמני / ובפת לחם סעדני, / ויבא אלי מטעמים / ושהיתי עמו ימים, / ואחרי כן השיאני לבתו / והשליטני בכל אשר אתו.





שעו, דודי, יפי שיחי וטובו / ושמרו־נא, לבל תנֽקשו בו:



הלא הם לעגי עוגבים ומלים / אשר בדא ידידכם מלבבו!
