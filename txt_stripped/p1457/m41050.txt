ובכן יי קנני ראשית דרכו





אראלי ערפל / ושוכני שפל / עד אין לא היו

אפל מחשך / ואור וחשך / טרם יהיו

אלפים שנים / קדם ראשונים / שפתותיו לי חוו



אני ואת נשב / בקרה צל מושב / אשר לנו שוו



אדקדק אמריך / אבאר דבריך / כי מאד נאוו

אור פניו / ודבר אפניו / רקחי עסיסי רוו



אות עם אות כתבתי / ובפרגוד שלו ישבתי / ופקק דברי אצבעותיו טוו



מגלה / ברוכה גלולה / ידיו לי אוו



אמץ השמיעני / בדורות אשר שעשעני / במה ברואים יהיו



ארקה אנטיע / דוק ארקיע / ושוכניהם עבורך יהיו

קד והודה / מחיני נא / אם לא תכפר

קראתיו למעונה / ושמע וענה / ואני אספר

קולו בשנים עשר מיל / על שנים עשר מיל / תלמוד מספר

קנאה לא אחז / קלסתורי חז / תפלים תופר

קהל בת רבים / הגיונך יוצא כשלהבים / בך להתכפר

עבורם ברא כל

ר[…]ה תמימה / שלמה ומשלמה / מען בשמחה

ראש הנביאים יבוא / בלבו אבוא / ואקבלנו בשמחה

ראשו אנשק / גופו אחבק / רוחו לרוחי נחה

רום ירום בי / ישאני אוהבי / בכתבה פתוחה

רעיו ישעשעוני / בבתיהם יטעוני / כמזוזה המתוחה

ריביו אריב / קמיו אחריב / ובלשוני אני שיחה



רווי ירונו / חסדי יננווינו / כגן רקוחה



רחבו וארכו יקום / בהיכל הרקום / זאת המנוחה

רעי נאמני / לך הניני הנני / אוצר ששון ושמחה

עבורם ברא כל

ישוב העולם וכלכול כל

במעמד תמימה השיב קדוש





שערים פתחו / עליות רתחו / לשמחת כלה

שמים מור נטפו / הרים עסיס הטיפו / בקטרת כלולה



שערים גן עדן / כל בשמים עדן / והעולם נתמלא



שמיו הטה / ולבש ועטה / ואפרסמון מלא

שחו הרים / מעוכים ודקורים / ועשן עלה

שוררו אלים / מלאכים קלים / רנה וצהלה



שקר החתן / יום המתן להתחתן / בחפה משכללה



שליח גש / לערפל נגש / כבני מעלה

שבה נות בית / לבני הבית / תלמוד להועילה

שנן שמחה / לאוהבי שמחה / שמחת חתן וכלה





ושבעה קדם כל

עבורם ברא כל

ישוב העולם וכלכול כל

במעמד תמימה השיב קדוש





תהומים שאגו / נשי ומים דאגו / בהגבלת סיני

תמים ירד / והכל חרד / וגבה סיני

תבל וכל אשר בה געשה / ונתפלצה ונתפלשה / בשמעה אנכי ייי

תקפה תורה בדברה / והטלילה אורה / לגפן סנסני

תבור וחרמון / ענתה להם אמון / הפיקו עניני



תמה חורב […] / וסיני נגמל / ושופך גע באזני



תם והימן עלה / בקרני הוד הפלא / ברדתו תוך מאמיני

תורה בידו הורד / עלה וירד / ללמד המוני





אור הוקדר / לכל חי החרד / עולם הפרד



זה כי זה הורד / רך הפרד / עלה שמים וירד.
