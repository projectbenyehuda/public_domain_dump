


	
	אופן
	







ט1


וחיות הקדש אשר בשמים / ברואות הם עד לא ארץ

גיויותם אש מתלקחת בשמים / דמות מכונות לכנפות הארץ

הדר מעל ימינו טפחה שמים / ומתחת ידו יסדה ארץ

זיו מושב יקרו בשמים / חונה היה מתחלה בארץ

טס וידא היות בשמים / יצורים עת יצר בארץ

כבא יום זה שמחו שמים / לתת תורה לגוי אחד בארץ

מיד הכריז בצבאות שמים / נם להודיע גבורות בארץ

שיח השיבוהו רום על שמים / עזך איך תוכל להכיל הארץ

פרשז ענן מרומי שמים / צר והכין על פני כל הארץ

קול הדבר השמיע משמים / רעדו וחלו יסודי הארץ

שדי הנוטה כדוק שמים / תנה לעמו ארכה מארץ

העידם בשמים האזינו השמים / והסהידם בארץ ותשמע הארץ

כי אני הוא בשמים / ומביט בארץ

ברוך בשמים / ומבורך בארץ




בקול רעש גדול משמיע לעומת שרפים לעומתם…





ט2


תרשישי שחק חצובים מאש / שאונם מוציא זיעות כמים

ראית פניהם כלפידי אש / קול דבריהם כקול מים

צפות אין בם בכלו אש / פן ישרפו כעצי מים

עבור כי יש אש שורפת אש / סבובה ערפל וענן מים

נאור בעת ירד לסיני באש / מקף היה שבע ענני מים



לבושי גאות ועוז […א]ש / כתב באצבעותיו משולת מים



יריעה לבנה חתך מן האש / טפים שחורות המשיך כמים

חרת ונתן מימינו דת אש / זאת התורה למשוי ממים



וזיק יצא כברקי אש / הולך ו[…]יל על נחלת מים



דבר כשמעו נלהטו באש / גשם נדבות הונף עליהם כמים

באהב קבלו דברו בית אש / אף אם יבואו באש ובמים

לא שכחו שמו בלהב אש / בכבשן אש ובבור מים

כי עזה כמות אהבה רשפיה רשפי אש / ולא יוכלו לכבותה מים

ומיחדין בנשף ושחר בכל יום פעמים




ט3


אחד יחדנו לשמו בסיני / ברוב עוז נהלנו לציון

גם יתר מחמדת סיני / דרש ואהב את הר ציון

הלא לשעה היה קדוש סיני / וקדשת עדי עד הקדשה ציון



זמ[…] והיה מתהלך באהל סיני / חנותי לביתי מצבה בציון



טכוסים במחן נהגנו בסיני / ישבנו ככלה בחפתה בציון

כנטשנו דת צוינו בסיני / למעננו נטש ועזב ציון

מורדים אשר באו ונלחמו בסיני / נמו ערו ערו עד היסוד בציון

סלה עד תפתח תקוה מסיני / עלות מושיעים משם לציון

פרשות סתם שנאמרו בסיני / צור תפרש לנו בציון

קול פה אל פה השמעתנו בסיני / ראות עין בעין נחזה בציון

שבעה מיני אש היו בסיני / תקיף כן יהיה חומת אש לציון

וכהשמעתנו בסיני / כן תשמיענו בציון



להיות לכם לאלהים
