


	
	ח אלה החוקים
	







סלוק לעצרת לאלעזר






אלה החקים / אשר הם נתנו משחקים / בקולות וברקים

דברים עתיקים / מדבש מתוקים / בשבעים לשון נחקקים

על לוחות חקוקים / חיים מחשקים / מלאים ולא ריקים

לוית חן וענקים / מזן אל זן מפיקים / חאט ואשמה מנקים

חמאה וחלב מיניקים / זיו פנים מבהיקים / ישרים וטובים וצדיקים

קרובים ולא רחוקים / חלב לפני יונקים / דבש לחיך דבקים

נשמעים מחך ממתקים / עץ חיים היא למחזיקים





וכדברו בפרהסיה מראש סיני / בלשון מצרית אנכי יי

הרים נזלו מפני יי / כרמל ותבור וחרמון נעשו גניי / כי נתגאו לפני יי

ועלה ונתעלה הר סיני / ומרחוק סקרוהו המוני / ובאצבע נמו זה סיני

ובשמחה וירד יי על הר סיני / לגלות לענו צפונות לפניי

ששה בכתב בהגא עניניי / וששה בהלכות שנוניי

זה ספר תולדות אדם וכל איתניי / ויציאת מצרים גאלת בניי

וקרבנות תורת כהניי / וחמש פקודי מניניי

ומשנה תורת יי / וספר עוצי ירא יי

להודיע סכם לויתן ובהמות סעודת מזמני

ערך זרעים ללמד נבוניי / וסדר מועד לבונן תחכמוני

וספוק נשים איך לטהר במחני / ומשפט נזיקים להודיע דיני

ואזהרת קדשים לשמיע נבוניי / ותורת טהרות ועסקי עניני

ואז האמינו אמוני / כי הלכה למשה מסיני

ונהגו לשנות גדולי וקטני / ומשה קבל תורה מסיני

ואז קרן מראיתו / כמלאכי שרת דמותו / כגבורי כח גבורתו

כארז גבוה קומתו / בעלותו וירידתו

וכל חכמים למדו מחכמתו / וכל הנביאים מנבואתו

כי על שלשה נתן יחידתו / על עמו ועדתו

ועל משפט צדקתו / ועל חקי תורתו

לכן מסרה ליהושוע במותו





כי אלפים היתה במצפונים / מתעלסת בכנף רננים

משחקת בחביון עוז מעונים / בסתר שדי ובצל רעננים

וכנתחתנה נזלו גבננים / ועל פיה היו נוסעים וחונים

כבדה בת מלך יקרה מפנינים / כבתולה מצנעת לפנים

ומפניה נקרע ירדן ועברו בנים / ועושה קרב משרת במעונים

אף הוא במותו מסרה לזקנים





כי כאגן הסהר / מזג בל יחסר / נדמה בה כל נגיד ושר

לדעת חכמה ומוסר / ואם השכים והעריב כוח יווסר

אמריה מרפה לכל בשר / והתרה לכל שבועת אסר

וברכה לבד לא תחסר / וכל חולי ומכאוב מנו יוסר

אף זקנים מסרוה לשופטים ארבעה עשר





ועד לא נמתחו יריעות ברואים / ובטרם נתכוננו עמודי נבראים

חקיה היו צפונים ונחבאים / אלפים שנה צנועים ונפלאים

גם ריקיה כפלח הרמון מצוות מלאים / שלמים ובכל נאים

לעבוד עבודתה ביראה באים / ושופטים מסרוה לנביאים




ובעבורה בכבוד נגלה / יי בורה לקדושים וגלה


ביקר ובגדלה / ברנה ובשמחה ובגילה

ונם לזקני אום נדגלה / והייתם לי סגלה

אף הנביאים מסרוה לאנשי כנסת הגדולה





ולמעלה גהר כאריה מסבכו / ויקרא כה וכה

ארבעים וארבעים בענן סככו / ויפקידהו לצאנו היות נסיכו

ומרשפי חצובי אש חסכו / אש ויין וחלב ודבש המסיכו

ויט על חמד סכו / ומאנשי הכנסת נתנה לאיש סוכו





ממדבר נתנה מתנה / למשולי כבכורי תאנה

קרואי עדת מי מנה / חלק ונחלה גורל ומנה

שרה מראש אמנה / עדות נאמנה / למו באהב נתחתנה

ובעברה עניו נתענה / ופץ בתחנון סלח נא / ואם אין מחני נא

איש סוכו כפנה / מסרה לאיש ירושלים וחסיד בכהנה





וכל הנראות האלה בשבילי / סביבות סיני כהגבילי

בקצף בלי לחבלי / לרצותי ולשעותי ולקבלי

בשלום ובשמחה להובילי / על כנפי נשרים לסבלי

והיא חשקה נפשי משחת בלי / בעת נמסרה לבן פרחיה מתיי הארבלי





ומראש היתה עוז מבטח / לב בעלה בה בטח

הנה אל ישועתי אבטח / ובעבורה וישכן ישראל בטח

כנמסרה לבן טביי ולשמעון בן שטח





כי ככלה מסתרת בעוז חביון / יושבת בסתר עליון



צפונה מראיון / נתנה לעניו בצויון / בשורש ובחפץ וברציון



שפל מכל נתן עליון / ונתעלפו גבננים בקהיון

ונתנו לזועה ולבזיון / כנחקקה דת בגליון

להמסר לשמעיה ואבטליון





יעלוזו רעים ואיחים / כנשתלחו בפרדס שלחים

כעץ מבכר לירחים / ריח טוב מפיחים / בלקח טוב סוחחים

ששים ושמחים / כשושן מפריחים

וכמו קבלוה מראש שני אחים / כן נמסרה בסוף לשני אחים





וקרב יום שמחתה להנתן / לכלת לבנון להתחתן

רעשו דוק וגא והים קול נותן / ונאספו עם אל קוסם בלי להתמתן

ונמו מה זה בעולם להנתן / והשיבם יי עוז לעמו יתן





והשיבוהו חושבי בלבם ובפיהם שלום / כל זאת עבור נתיבותיה שלום

מאת אדון השלום / ועל בלבו להושיב עמו בנוה שלום

במתג ורסן עדינו למה יבלום / יי יברך את עמו בשלום





ורבותים אלפי שנאני קדש / ירדו היום לסיני בקדש

לשמחת תורה הכתובה בלשון קדש / במאמר מלכי בקדש

ושנים עשר מיל נתאחרו אנשי קדש





וכדברו אנכי העיפו נפש / והטיף מטל קווצותיו וחזרה בם נפש

ודברו לרועה צאן קדש / דבר אתה עמנו דברות קדש

ואל ידבר עמנו נאדר בקדש / ונתנו זמר ושלשו קדש



לנאדר בקדש / מלכי בקדש / נערץ בקדש / כסוד אילי קדש
