


	
	קינה על־פטירת משה זכות: על־מה בני תבל
	




Josef Fiammetta


]נולד באנקונה ומת שם בשנת 1721]






– – ͝ – – – / – – ͝ – – – /– – ͝ – – – / – – – ͝ – –



מות



על־מה בני תבל / בוכים ברב־אבל / כבנהרות בבל / נשמע קול יללה.

הקול למרחוק / נשמע, ולבלי־חק / פי נמלאה בשחק / בשמע אמלה.

לא יחטיא דעתי / לחשב תבונתי / מני ומאתי / נעֽשתה נבלה.

חרבי משוש לבי / שבה צבי שבי / חכם ואיש נביא /
שתה כוס רעלה.





חיים



מה־לך פאר אכזר / אם אל לך גזר / נעזר כעבד זר / איך עצמך תפנק

לשמר דבר שדי / כל־חֽיתו שדי / יעוף כזיז שדי
/ וכאריה יזנק.

אך רע לרע ישלח / לא זה וזה יצלח / כצי אשר מלח
/ בו עולל ויונק.

קום בוש לבש חרפה / מוכן למדחפה / לסקל ולשרפה / ולהרג וחנק.







מות



מה־לך להרחיב פה / נספה ולא תצפה / כי אין לך מרפא / מציר מחלתי.

אקצר אשר תזרע / תרבה אני אגרע / דת יועצים
אפרע / תתקים עצתי.

אין דל ואין עשיר / עת בא עדי ישיר / אף מעשיו
יכשיר / ילכד ברשתי.

דם כהדם רגלי / מדרס למעגלי / מהר ובוא אלי
/ תשתה כוס חמתי.







היום שבה בדד / כי אהלך שדד / ראש דת כעוף נדד / צדד תוך מצודי.

משה זכותו רב / למד כרב לקרב / בי רב ולא ערב
/ אליו חץ כבדי.

נלחם בעז כחו / שח שך נתיב ארחו / סוף נאסף רוחו
/ ומסרו בידי.

הוא עף מקום עפעף / רץ רץ ולא ייעף / על־זאת לבד נועף / כי נבהל בפחדי.





חיים (בינו לבין עצמו)



אוי אוי לשבר זה / איך על־אבי חוזה / מזה
ובן־מזה / שלט בן־זדונים

גדול ורם ישפיל / הר לעלות יעפיל / נזיר כזיר
יפיל / יבנה הלבנים

בר בר ישלח בר / ישבר לבב גבר / שוסה גרנות בר
/ לטע קמשונים

איך ראש גדוד שדוד / נפל בלי פדות / נשבה ארון
עדות / משער זקנים.





(אל פני המות)



אויבי ואיש ריבי / לביא עלי לבי / שש
דץ במכאובי / על־חובי משושך.

איך ריב למול אדון / תדון בלב זדון / תשלח
בפקדון / האל יד מבישך.

איך תחבש לכבש / עיר צור, ולא תבש / גסות כסות
תלבש / ולבושך לבושך.

לא לא כגורלך / מלאך כרוב הלך / דם מזדון מלך /
כי הוא יחרישך.







מות



על־מה דברים עוד / הבל וריק לסעד / לעג
ובוז תסעד / עצמך תפתה.

כי כל־בשר החי / יפל בתוך שוחי / הולך לעדן חי /
לשאולה ותפתה.

אין עיר ואין מקלט / אין נס ואין נמלט / נגזר וכן
נחלט / על־ערום ופותה.

יפל במכמרתי / בן־צוף ואפרתי / את־כוס רעלתי /
כל ירוה וישתה.







חיים



דם דם שתק שוטה / חוטא בעת בוטא /
עוטה ומתחטא / על־פשע וזדון.

אם מוֽסדי תבל / מטו כציץ נובל / יבכה ויתאבל /
עבד שר ואדון.

אם בארזים על / להב מלא שעל / מה־יעשה יפעל / אזוב
קיר כנדון.

דרי זבול ידכו / כי לא בדין יזכו / אם כן במר
יבכו / מות ואבדון.





מות



אודה עלי פשעי / כי פֽשתה נגעי /
על־שר ואיש שועי / אור עמים וצלם.

מעלה ומטה הם / בוכים למיניהם / עלי בחבליהם /
משמיעים בקולם.

אוי לי לאן אברח / אם מערב מזרח / אזרח בתוך זרח / או אעלם בעילם.

כפוף בלב מסעיר / אלך לאח צעיר / עמו נהי אעיר /
נאלם בוש ונכלם.





מות חיים



קראו בקול ענות / תנות בנות בינות /
קינות כמו תנות / מספד מר כתנים

שברו בני־תבל / נבל כמו־נבל / חבל ורב־אבל / לא
עוגב ומנים.

שולל מעיל מעלה / על־מועלי מעלה / אדר יקר גלה /
ולבוש הסדינים.

היום בחטאתכם / לקח וסר מכם / מעדה ומלבישכם
/ שני עם־עדנים.





נר מערב כבה / ראש דת ושר צבא / שב
מצבא צבא / גם לא יעבד עוד.

כי לעבד עליון / הלך לרום חביון / שמה באפריון /
לא ימוש וימעד.

אך לי כאב חלי / סר שר וראש חילי / מלכי מנת חבלי
/ לי משען ומסעוד.

שקע ובא שמשי / שאהבה נפשי / הומר במר דבשי
/רוש אכל ואסעד.





רב טוב לישראל / משיב לכל־שואל / בקי
ברזי אל / ובסתרי בראשית.

ידע בכל־מפלא / נגלה והוא פלא / עולה ומתעלה /
על־דעת אנושית.

ודבר אמת מורה / תבן וקש זורה / מורה לכל־מורה /
אל־דפן חמישית.

תועה בגללו / נמכר לעבד לו / הוציא ברב חילו /
לחרות וחפשית.





מליץ ומן־אלף / יחיד ואין חלף / מן־תיו
ועד־אלף / בנה נחלת־הון.

לא יש דיו דיו / לכתב פאר עדיו / גבה צבי עדיו
/ שמהו לגאון.

השיב בדרושיו / לב עם לצור עושיו / דתן בדת
עכשיו / בן־פלת והוא און.

השפיל זדון קמים / הלוחמים תמים / הצליל בלב
ימים / כל־סואן בשאון.





רועה אשר צועה / איך צאן בלי מרעה /
תטש ולא תשעה / תועה על־גבעות.

שה שור בנקרת־צור / עצור באין עצור / נצור ואין לה
צור / גם פועל ישועות.

תשמע ולא תחוש / תפנה ולא תחוש / תראה ולא תחוס
/ אל־בכי ושועות.

רב על־פרידתך / גם על־פרי דתך / שותה פרידתך /
בשליש דמעות.







מי זה ינחם עם / נפעם ביום זעם / לבו
כמו־פעם / הולם מהלמות.

בוכה לדור יתום / בלתי מתם נתם / חזון חתם וסתם /
חכמות נעלמות.

כי נֽפלו ראשים / עירין וקדישין / שרים ושלישים /
כל־דורש רשומות.

נשאר אנוש חלין / מכביר לבד מלין / ביקר ולא
ילין / נמשל כבהמות.





אך זאת ינחם כל / אש זה לכל־אשכל / חכם
ורב יכל / עם־תמים וארים.

ציר נאמן נפטר / דוד בכליל עטר / כי כן חתום
בשטר / על־כל הֽיצורים.

הכל זמן יכלה / עשיר ורש נקלה / שוקט וגם גולה /
יושב על־שמרים.

אכן לאיש ישר / יחסר לבד בשר / רוחו אזי יכשר /
לצאת לדרורים.







לראות אשר פעל / מעלה ורום יעל / שער
בלי נגעל / ודרכים גבהים.

יראה ויתפלא / טוב אוֽצרות מלא / צפון באור נגלה /
ודברים תמהים.

שמה לקבל לו / באים בני גילו / אותו להיכלו /
ילווהו כמהים.

כי שם בהוד נגלו / יראו ויבהלו / עת עוד לרום
יעלו / להם האלהים.





שמה ברום חביון / יחזו לאל עליון /
יראו בלי דמיון / אור זיו השכינה.

אז יכרעו נגדו / גם ישאלו חסדו / אל־עם וצאן ידו
/ חמלה וחנינה.

יחמל לנשארים / יכרית פני זרים / יקבץ מפזרים /
מכל־עיר ופנה.

“די די” ליום רעה / יאמר “ואל שועה” / הזורעים
דמעה / יקצרו ברנה.




(1697)
