


	
	אמרו כי איש היה
	







אדם וביתו






– – – אמרו כי איש היה בארץ הבשן / ושמו חמדן בן דישן / ושם אשתו מי באל בת מי זהב / מארץ החוילה אשר שם הזהב / ופלגשו ושמה ראומה / בת אהליבמה / והיה האיש תם וישר ירא ה׳ וחפץ במצוותיו / היורדים על פי מדותיו / רק בדבר החפצים והקנינים / זהב ורב פנינים / דבק נפשו לאהבה / יום ולילה לא ישבות מן המדהבה / ויולדו לו שלשים בנים ושלשים בנות / ויגדל האיש מאד ויעשר מכל האדם / ויפרץ מכל בני קדם / ויעש לו שדה ושדות ותענוגות בני אדם / והלכו בניו ועשו להם משתה איש יומו וקראו לשלשים אחיותיהם / לאכל ולשתות עמהם / ויהי כי הקיפו ימי המשתה / וישלח חמדן ויקדשם והשכים בבקר / והעלה עולות מספר כלם מן הצאן ומן הבקר / ועם כל זה היה כילי אובר בממונו / כי יתן בכיס עינו / מלחמו לא נתן לדל / ומעני ואביון מתנת ידו חדל / גם היה חומד להעכיר שארו / ולהונות את חבירו / הן יעשק נהרות ולא יחפז / ואם יתן לו איש כל הון ביתו בוז יבוז / כי אמר מה אעשה לבני ולבנותי / אשר רבבה כצמח השדה נתנם האל בביתי ובחומותי / לדבר הזה יסלח לי אלהי קדם מעונה / כי יקח מנחתי מידי אשר הבאתי לו בתחנה / ואקדם פניו בתשורה וארוחה / כי אמר אכפרה פניו במנחה.



ויהי היום וחמדן בבית בנו הבכור / והוא שותה שכור / ואש יצאה מלפני ה׳ ותקחהו
בציצית ראשו / ותקצר נפשו / ולא קמה בו עוד רוחו / והנה הצרעת זרחה במצחו / ויפל ארצה מלא קומתו / ויקחו עבדיו אותו / וישאוהו אל ביתו / וישכיבו אותו בחדר מטתו / ויגדל עליו הכאב מאד עד כי חשך תארו / וצפד עורו ויאכל חצי בשרו.





וישלח ויקרא את כל אוהביו מסביב עד אלון צעננים / ואת הרופאים אשר עשתרות קרנים / ויחל פניהם אולי יוכלו לכלכל מחלהו / ועל ערש דוי יסעדוהו / ויחלצוהו מחבלו / ושב ורפא לו / ויפנו האנשים להביט אל מכת מכהו / המה ראו כן
תמהו / עד כי נוסדו יחד / ויענו כלם פה אחד: ׳עצתך לא תקום ולא תהיה / ועתה צו לביתך כי מת אתה ולא תחיה׳ / וישמעו אשתו ורב בניו כי אפס עצור ועוזר / וישאו את קולם ויבכו בבכי יעזר / ויענו ויאמרו: ׳ישים נפשנו תחת נפשך, צורי ומשגבי / מי יתן מותנו תחתך אבי אבי / אנה במותך נלך ואנה נפנה / על מי נטשת מעט הצאן ההנה / ואיפה נוליך את חרפתנו / נשכבה בבשתנו ונלינה בחרפתנו׳ / ויזעקו מרה בצירם וחילם / ותבקע הארץ לקולם.



ויהי עד כה ועד כה ויקר מקרהו / ויבא הלך אל מחנהו / איש מארץ התימן / צדיק
ונאמן / אב הרחמן / ושמו איתן האזרחי / בן בלדד השוחי / ויועד לראותו / ולשחר אודותו / וישמח חמדן לקראתו / ויאמר אליו: ׳מאין תבוא אזרחי / השלום אתה אחי / ומה מעשיך ופעליך / למי אתה
ואנה תלך?׳ / ויאמר: ׳אני רופא אמן / וציר נאמן / לרפאות צרעת נעמן / וכל מכת השחין לח ויבש / אהיה חובש / ורק אני לבדי נותרתי מיתר הרופאים אשר בגלעד / הסמוכים לעד / ובלא בצע הצרעת ממצורע אאסף / חנם אין כסף / אחבש את חילם / וזה שמי לעולם.







ויהי כשמע חמדן את דבריו / ויחזק
בכנף מעילו / בכה והתחנן לו / ויאמר: ׳אל נא תעבר מעל עבדך / כי למה אמות נגדך / יבואו נא דבריך להושיעני / יחיני מיומים, ביום השלישי יקימני / ואם את ה׳ אלהיך האמרת / כן תעשה כאשר דברת׳ / ואשתו ובני ביתו יצאו נצבים עליו / וכל אוהביו מסביב שתו עליו / ויפלו לרגליו / לחלות את פניו / ויאמרו: ׳אנא, עבד צור
ישראל וגואלו / רפא נא לו׳ / ויוסף חמדן שאת משלו / ויתן בבכי קולו: / ׳לא על נפשי אשא קינה אף לא על עלומינו / ישבנו גם בכינו / אך על זה דוה לבנו / ועל אלה חשכו עינינו / על מרבית ביתי / אשר טפחתי ורביתי / מני רחם נשואים /
איפוא יהיו רועים / ואשתי הענגה ורכה / אשר לא נסתה להציג על הארץ כף רגלה / מי יקום לה / איכה תשב בדודה / שכולה וגלמודה / לכן כאפעה אשאף ואבלה / וארד אבל שאולה׳.





ויען איתן האזרחי: ׳אלהי קדם מעונה / יהיה לי למשען ומשענה / אחת דברתי ולא אשנה / ואתה התיצב לתשועת ה׳ ואותותיו חזה / למחר יהיה האות הזה / ואין לי אתך לא כסף וזהב / ולא אפנה אל שום רהב / אך אשאל ממך שאלה קטנה / אותה אבקש כי תתנה / ובה עד מהרה תמצא ארוכה
לחולך / אשר עולל לך׳ / ויען חמדן: ׳הנה דבר גדול דבר אלי אעשה / וכל אשר תאמר אתן / אף כי דבר קטן׳.





ויען איתן: ׳זה הדבר אשר תעשה ויקחו אליך / ממתי אהליך / אשתך ובני ביתך /
המרננים ישועתך / מן הדם אשר באצבע קטנה שבידם / והחביב חביב קודם / ואת הדם תמציא כמר מדלי / ונתן אותו בתוך מים חיים אל כלי / ורחץ בשרך במים שבע פעמים ואחר תטהר / כברוש ותדהר / אך לא בחרב ולא בחנית / ולא בשום דבר ממאיר את הדם תמצה / כי לא ירצה / ולא יצליח למלאכת הרפואות / ולא יחשב למושעות / כי אם צלי אש וחשש להבה / ושלחו איש את אצבעו בה / עד כי יעשה הבשר צלקת אדמדם / וגידיו יעלעו דם / זאת חקרנוהו שמענוהו / ממים רבים משיתיהו / וממצולות חכמי הטבע דליתיהו / ואם את הדבר הזה תעשה מחלתך אכלכלנה / אנכי אערבנה / מידי תבקשנה / ואתה דע לך׳.





ויהי ככלותו לדבר וימהר חמדן ויקרא את נשיו / ואת בניו / ויבאו כלם וישתחוו אפים לפניו / ויספר להם את כל הדברים האלה ואיך מצא רפאות תעלה / ויען ויאמר: ׳אתם ראיתם את אשר עשיתי לכם מעודי / ונתתי עליכם מהודי / ופרשתי עליכם ענן כבודי / ואשא אתכם על כנפי נשרים / ואנוה לחייכם בתורים / ואלביש אתכם שני עדנים / ומלאתי את ידיכם מטמונים / ועתה בצר לי אנא שחדו בעדי
כבודכם / מסת נדבת ידיכם / פדעוני מרדת שחת / ומפני שמי נחת׳ / ויענו כלם בדמע: ׳את כל אשר דבר אדני נעשה ונשמע / הננו נכונים להיות קרוצים / נתחים ערוכים על האש והעצים / כליל נהיה אזכרה לרוח אפינו / עולה אשה ריח ניחוח לאדננו׳ / ויאמר איתן אל אשתו: ׳גשי הנה / ושלחי אצבע קטנה׳ / ותתן את ידה, ויקח במלקחים רצפה / וידבק אל כפה / ויהי טרם נשקה את בשרה / לאכל את עורה / עלה עשן באפה / כי חרה לה /
ותשא את קולה / כדברה רתת נשאה ותאשם בבעלה / ותאמר: ׳הלמענו אסבול צערי / להכאיב את בשרי / וכי יראה שחת מה אחוש / האם כח אבנים כחי ובשרי כחוש / העוד איש אין בארץ / לבוא עלי כדרך כל הארץ / וכי תמו נכרתו רעים אהובים / בחורי עגבים / ותלבש צוארה רעמה / ותפן ותלך בחמה / ויקרא את בנו הגדול הנצור לו כאישון / ויעש לו כמשפט הראשון / והיה כי יחם לו ויקפץ ידו אליו / ויעבר על פניו / וישא את רגליו במרוצה / וינס ויצא החוצה / והנשארים נסו לקולו, ויבא חדר בחדר להמלט / במלבן ובמלט / וישאר אך חמדן לבדו / ועיניו ראו כידו / ויתהפך בציריו וחבליו / ואימת מות נפלה
עליו.







ויען איתן האזרחי: ׳ההיטב חרה לך אחי?׳ / ויאמר: ׳היטב חרה לי עד מות ואיככה אוכל ואראה ילדי / ומעשי ידי / בקרבם יעריצו ארבם / הפך לשנא אותי לבם / רוחי זרה לאשתי / וחנותי לבני בטני / ויאמר אחד מן הנערים הנצבים עליו:
׳שמעתי אומרים העוד לנו / חלק ונחלה בבית אבינו / לכבדו ולברך בשמו / הלא צבא לאנוש על הארץ וכשכיר ירצה יומו / הן מלאו ימיו / וכלו רחמיו / גם אהבתו / גם שנאתו / כי לא יקח הכל במותו / הוא ילך למיתה וינוח על משכבו בשלום / ואנחנו נשאר הלום / כלנו חיים / עד לא שמים׳ / וישמע חמדן וימת לבו בקרבו / והוא היה לאבן וכל רוח אין בו. – – –
