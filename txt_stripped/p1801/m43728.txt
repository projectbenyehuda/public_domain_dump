


	
	ישיחוני סעפי
	




ישיחוני סעפי מאמרים / ואט יתלחשו לי בדברים

וניבם צח ומלה אין בפיהם / בעיניהם לבד קורצים ומורים



יראים פן אנשים ידעון סוד / ו״האח!״ יאמרו זרים וערים



וזה ניבם בכל עונה וכל עת / וכל רגע, ערבים או שחרים:

״הככה יעשה רע לרעו? / היש ככה במשפט החברים?

התהיה אהבה עזה עזובה / נטושה על פני מדבר חררים?

ופוריה, אמת, היתה לדודה / ומאז שלחה עד ים קצירים

מגדיה כציציה ופרחה / תבשל בעמקים או בהרים״.

וכזו דברו אחד אלי חד / במלים יקרו מפז ודרים

ואמרתי: אני אצפן תנואות / ולעד יהיו בלב שמורים

עדי כי נצבאו עלי אהבים / ורובים בחצציהם ומורים

צדיה פלחו בם את כבדי / ומדם לב לבד רוים ומורים

לכה נשפטה יחד, ונשים / עלינו יודעי משפט ומורים

מביני דין ודת חשק ומשפט / אהבים מימי אבות והורים

ונקריב עצמותנו לנגדם / לבלתי נהיה סוררים ומורים

אבל זעתי לבל נהיה למלה / בפי היקום, זקנים עם נערים

ונערתי לזאת חצני, וכפי / ליראת בוז, אמת, שמתים נעורים

לזאת אדם, ופי אבלם במתגים / ושפתותי אשימם נאסרים

ואבחר משפטי פיך לנפשי / הכי לך משפטי צדק ישרים

ושלחתי אני זאת המגלה / לך תעוף כעל כנפי נשרים

תמלא תוכחות פיה, ואם היא / מנגנת בקול זמרה ושירים

ואם רכות לך תענה, ראה כל / דבריה חזקים משמירים

תבונות מעניה מר אהלות / ועץ בשם נרדים עם כפרים

ומוסכת בריב, יין אהבה לך / בטורי שיר ובתי הזמירים

כסירים בעצי בשם צמחים / וצוף ודבש ביערת הדבורים

ומגדת אשר היה לפנים / בצחות שיר וכמאורות מאירים

וזאת היא התלונה הצפונה / תצעק מר ותשא קול ותרים:

המלכתב אלי ידך כבדים / ועביך כהיום נעצרים?

ומאז ערפו גשם נדבות / עלי ארצם, רביבים או שעירים

ומטרות חן בלי רעם וברק / ומימיהם מאד זכים וברים

לתוכחת מגלה קום אזר־נא / חלציך, כגבר שים חגורים

קרא את ואני כן אענך / ואשאל וענה מלים ברורים

שאל־נא בעדה ארחות דדנים / ואדם, אהלי מואב והגרים

וישמעאל, ונא חל הערבים / וקדר השכונים ביערים

בני יפת וגם כל משפחות שם / ותולדות חם, צחורים או שחורים,

שמע אלי וקבל מועצותי / הכי לך, לא לזרים, הם מסורים

וחן חן תעשה עמי וחסד / שלח קל חיש לאלה הספרים

קנה לי בחמשה או בששה / זהובים אם תרבה במחירים

תכפר המשובה בתשובה / וייטב לי מקום אילים ופרים

השיבני ואשובה ושובה / ונשיב אהבה לימי נעורים

ורע יענה רעו ברנה / ישיחוני סעפי מאמרים

ואמרו: יש הנחות באנחות / נשיאי רוזנים ובני נזירים

והן לו הדאגות איש ישיחם / אזי לא יהיו לו עוד מצרים

לפצעי אהבה העלה ארוכה / ומרפא צו ומזור למזורים

עשה חפצי ואחדל ממריבה / ויחדל ריב ומצות עם מרורים

פשעי הנקלים לא נקלים / ונכבדים שגגותם חמורים

ומאז כן ועתה מחשבותי / בצדק ואמת ובלי שקרים

ובקצה היריעה שם חקקתים / בעט ברזל, ולך שמתים סדורים.
