


	
	[זמן הכה בחץ שנון לבבי] (תלנה על־הזמן)
	




Leone Ebreo או Jehuda Abravanel


[נולד סמוך לשנת 1460 בליסבון, הלך בשנת 1492 לאיטליה
וחי בנפולי ג’נובה וברלטה בפלך אפוליה ושוב בנפולי.
מת אחרי 1521]





תלנה על־הזמן


ں – – – ں – – – ں – –



זמן הכה בחץ שנון לבבי / וכליותי יפלח בקרבי

והכני ומכתו אנושה / ודכני ושם נצח כאבי

פצעני ובלה את־בשרי / ואכל הכאב דמי וחלבי

וגרם כל־עצמי בחמתו / והתנשא ויקם־בי כלביא

ולא די כי צנפני צנפה / ושמני מגרש נד באבי

וכשכור הניעני בתבל / והוא על־פאתי ארץ מסבי

והן זה־לי כעשרים משנותי / אשר לא שקטו סוסי ורכבי

ומדד שעלי מים ועפרות / ארצות כל והם כלו אביבי

והבריח ידיד מני והגלה / בני־גילי והרחיק עם קרובי

ולא אראה ולא אדע מידע / ואת־אמי ואת־אחי ואבי

ופזר מחמדי זה לצפון / וזה קדמה וזה אל־מערבי

לבל עוד יוכלו השקט זממי / ולא אמצא מנוחה גם בחשבי

ובהיות מעדות פני למזרח / פרידתם תשופני עקבי

ורגלי תט ולבי נד ונודד / ולא אבחן בלכתי או בשובי

הלא כלה זמן לבי ושסע / ונתחהו כדבי או זאבי

והכוהו אנחות עם בהלות / בשד שודד ובזת הון ובשבי

בעיניו זה מעט מזער ובקש / לכלני לכבות את־שביבי

והולדתי שני בנים זכרים / בני־חמד בהוד ויקר ובצבי

והקטן קראתיהו שמואל / וגבהו זמן נוגשי ואורבי

והכהו לחמש משנותיו / וגדל בעדו רגזי ועצבי

והגדול אשר יצחק קראתיו / אברבנאל כמו צור מחצבי

כשם גדול בישראל כהורו / כבן־ישי והוא נר מערבי

בהולדו ראיתיו טוב לשכן / בלבו בין וטוב הוריו וטובי

ובן־שנה אהה מני הדפו / והדיחו זמן שונאי ועוקבי

בעת גרשו בני גלות ספרד / יצו מלך להציב מארבי

לבל אצא ואעבר בין גזרים / ולקחת בני יונק חלבי

להכניסו בדתו בעבורו / וגלה אזני איש טוב אהובי

שלחתיו עם מניקתו באישון / חצות ליל כאלו הוא גנובי

לפורטוגל ושם מלך מגאל / אשר היה לפנים מחריבי

ושם גדל כבוד אבי ועשרו / בחי אביו והוא מלכי חשובי

וזה קם זד בליעל ואכזר / ובוצע ואיש חמדן וכלבי

ועת קשרו סגניו בו ואחיו / הלא אבי בקושריו שם בכזבי

ובקש בהרג אחיו להרגו / והציל נפשו רוכב כרובי

והוא נמלט לקשטיליה אשר שם / מקום הורי ושמה בית חציבי

ואז שלל רכושי עד־בלי־די / ובז טובי וכספי עם זהבי

ובראותו בני כי בא לארצו / ובית אבי באיטליה בחשבי

עצרהו לבל ישוב וצוה / לבל יהיה לנדחי משיבי

במותו אחריו מלך כסיל קם / והוא אדוק בדתו איש נבובי

ואנס כל־עדת יעקב והעביר / עלי־דת כל־בני עמי נדיבי

ורבים הרגו עצמם לבלתי / עבר תורת אלהים מרחיבי

ולקח מחמד נפשי והומר / שמו הטוב כפי צורי נקובי

כבר הוא בן־שתים עשרה עדן לא / ראיתיו בעון בצעי וחובי

ואהימה ועל־ראשי חמסי / ועל־נפשי תלנתי וריבי

הניסותיו אלי־פח מפחדים / ומרשף שלחתיו תוך להבי

אקוהו ותוחלתי משוכה / ולמה תאחר עפר אהובי

ולמה זה תדכא לב ילדך / ותהיה חץ בכליותי כמביא

ותחשיך בעריפי אור ותשים / זרחתו למול עיני כערבי

וסהר יחשך תמיד בפני / וכוכבי ישו נסתם בעבי

ולא יאיר שביב חרסי מעוני / ולא יזרח באשנבי ועבי

ולא תפרח חבצלת שרוני / ולא ירד עלי עשבי רביבי

ותגזל את־שנת עיני בזכרך / ולא אדע בעת קומי ושכבי

ולא אטעם במאכלי ונפת / לחכי מר וסף־רעל ערבי

בעני אלחמה עגת רצפים / ובבכי אכלה לחמי חרבי

בדמעה אמזגה מימי ואשתה / ולא יבוא במו פי דם ענבי

והייתי שתוי מים ונזיר / עדי כי חשבוני בן־רכבי

ועת כי אחלם שובך ואראה / תמונתך בעיני מחשבי



חבלים נפלו־לי בנעימים / ופני לבשו מראה צהבי



וישנתי וערבה לי שנתי / הקיצותי שבע רצון וצבי

ושתיתי ומימי לי ערבים / ונמתק לי בחברתך רגבי

הכי עת אזכרה נודך כמו־אש / לבבי חם ויכני שרבי

והייתי כאיש נדהם ונרדם / שפל קומה קצר ארכי ורחבי

בזכרך ששוני עם יגוני / צרי אתה ואת צרי ואויבי

וצורתך עלי לבי חרותה / ומרחקך בתוך לבי חטובי

ויותר מהיות לי מעליזי / דמותך נדודך מריבי

פרידתך תסכל את־עצתי / גרושך סך ועוה את־נתיבי

ובעבורך גאון נפשי מאד שח / ושפל בעבורך רהבי

עדי שקמים גבהים מברושי / עדי רם על־ארזי עץ אזובי

עדי גבה עלי נצי עטלף / ועל־כנפי נשרי עף זבובי

וכנערי יצורי נחלשים / וגבר על־אריותי כשבי

ואמאס שיר ושברתי נבלי / וכנורי תליתים על־ערבי

והפכתי לקול בוכים זמירי / ועוגבי במכאובי כאובי

עגורי נהפך קפוד ותורי / כתנים שב ויונתי כעורבי

ותעבתי בהיכלי המלכים / ותאבתי שכן מדבר ערבי

בני פרץ גרושך יפרצני / ישופני ושם פחים סביבי

ושם מרך בתוך לבי ורגז / במורשי ובעצמי רקבי

ואשמע אמך כל־יום מבכה / בקראה לך "ידיד לבי רטבי

ומי זה מעלי חיקי גנבך / ומי שמך פרי בטני עזובי"

וכי לא אעצר כח לסבלה / ולא לטמן דאגתי בחבי

עזבתיה והלכתי לשרת / פני מלכי נתנו אל מטיבי

וכן נדתי ולא נחתי ונעתי / ואשכן בין אדום תוך עם להבי

ולא אמצא מזורי אל־מזורי / ומי יישיר זמן מתעי ועולבי

ומאסתי ימי עמל ולילות / ומהם מותה אבחר ואבי

וחיים כבדו עלי וימים / כחול ימים עלי שכמי וגבי

ומה בצע בחיי היגונים / והגיע אלי קצי קצובי

למר־נפש הלא חיים תמותה / ויש־לי רב ומעוטי כרבי

ומה אוחיל לרב ימים ושנים / ראות טובה ודב שכול באורבי

ילידי יום בני אשפה וקשתם / ברום שחק זמן מורי ורובי

והושמתי למטרה אליהם / והמה סובבים על־חוג קטבי

ואשובה לדבר אל־יחידי / לבלתי יהיה עוד מאדיבי

בכורי שים לבבך דע הכי בן / חכמים את מחכמים כנביא

והחכמה ירשה לך ואל־נא / תאבד עוד ימי ילדות חביבי

ראה עתה בני חמוד ללמוד / קרא מקרא והבן מכתבי

שנות משנה למד תלמוד במדות / שלש עשרה ופרוש עם מתיבי

ואיך אוכל להתאפק בעברו / וזה חליי וחרחורי וחרבי

וזה יקרע בתער השכירה / סגור לבי ולא תער גלבי

וזה יביא בלבתי אמולה / ברק חרבו בתוך בטני וקרבי

למי זה אמסרה רבי תבונות / למי אשקה עסיס גתי ויקבי

ומי יטעם ויאכל אחרי עת / פרידתי פרי דתי וכתבי

ומי יבין ספוני בין בספרי / תעודות מעשה אבי נציבי

ומי ישאב בצמא מי מקורו / ומי ישתה בציוני וחרבי

ומי ילקט לאט פרחי נטעי / ועץ־דעת יהי אורי וחוטבי

ומי יעצר וישלים מחברותי / ומי יארג שתי בגדי וערבי

ומי יעדה עדי דתי במותי / ומי ירכב עלי פרדי וצבי

לבד אתה ידיד נפשי ויורשי / אני חיב לצורי את ערבי

לך צמא לך כמה בשרי / בך אשקה צמאי אף רעבי

לך יאתה תבונתי הדורה / והוד דעי וחכמה נוססה בי

קצתה לי ירשת־אב מאלפי / אבי חכמות והוא מרי ורבי

ומקצתה קנויה מיגיעי / כבשתיה במו קשתי וחרבי

ועמקו מחשבותי בה וחכמי / אדום היו בעיני כחגבי

והלכתי בבתי מדרשיהם / ואין מי יערך אתי קרבי

אנצח אנוש ירום כנגדי / ואכניע וארשיע מריבי

ומי זה יערב לבו להגיד / יסוד ראשית וסוד מרכב ורוכבי

ולי נפש יתרה נהדרה / עלי נפשות בני דורי עלובי

ולי צורה בעז צורה בצורה / עצורה ואסורה תוך כלובי

וכלתה תעלה על־מדרגתי / וחשקתי תקומם על־שלבי

ידידי מה־לך בין עם טמא־לב / כתפוח בתוך יערי חרובי

ונפשך הטהורה בין עממים / כשושנה בבין חוחי ועשבי

נהג ולך ובא עדי ידידי / ברח ודמה אלי־עפר ולצבי

והלך־נא לבית־אב צור ילדך / ישגבך אלהי משגבי

והוא אלי יישר את־נתיבך / וממצר יביאך אל־רחובי

ויתן ברכות הורי לראשך / ועוד לך ברכות אבי ושבי

ויאיר מחשך רוחי וישיב / ברב חסדו למישור את־עקבי

אני אפקד לאל רועי זרעי / ואשליכה אלי אבי יהבי

ויראני צבי חשקי לפניו / אני קורא חמודי מקשיבי

ואז אשיר ליוצרי שיר ידידת / אזמר־לו בעודי שיר עגבי

ואגישה לפניו מנחתי / ואערכה תשורתי להביא

ובה נפשי קשורה אל־קדושי / ובה ראשי ושם עיני ולבי

וייטב לו כאילי מהללי / ומשור פר זמירתי וניבי

ויראני יקר ציון ויפיה / בהוד מלכות ושם ממשח כרובי

ויחדו יהיו תמים בראשה / שני אורים בנו דוד ותשבי

ולא אויב יחרפה לדורים / ולא יהל לעולם שם ערבי.


(נפולי 1503)
