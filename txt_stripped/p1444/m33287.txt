


	
	יחידה עקבך כרוך
	



תוכחה. חתימת השם: יצחק הלוי בר זרחיה ירונדי זכרו לברכה אמן.

כל מחרוזות התוכחה מסתיימות בפסוק מקראי.






יחידה עקבך כרוך / בקץ וזמן לא ארוך,

ופעלך למשפט ערוך – / יהי מכירך ברוך!





יצורי איש נרדמים, / סכלים ולא מחכמים,

לבשתם וכלמתם / יעורו משנתם.





צמאים להשיג תאוה / ולעלות במתי גאוה,

כאלו הון יצילם / וממות יגאלם!





חתת מות לא יראה / אנוש, בתאוה מתגרה,

רודף אחרי שלומים / נאהבים ונעימים.





קט בהיותו רענן, / בביתו שלו ושאנן,

יצו אל פגע זעמו: / שאהו אל אמו!





הולך על משענתו / בטח לעשות מלאכתו,

ומות שוטטה בעדו / ותתפשהו בבגדו:





״לכה נא אקחך, / הרפה מרוחך,

דבר המלך נחוץ – / למה תעמד בחוץ?





ומה זה עוד תתאחר? / היש בצע כי תשחר?״

הישא כפר מאומה? / ושחד בחיק חמה?





יקרא לאנשי אמוניו, / יצוה על קניניו,

ימהרו לחצב קבר / טרם כלה לדבר.





בפתע נפשו תפרח, / בדרך עולם תארח,

ותעבר מעליו / ולא יספה שוב אליו!





רעיו נקבצו לספדו: / הוי אדון והוי הדה!

רק הלילה בביתו / לא תלין נבלתו.





זועפים פניהם ולבם, / יעטף חמס בקרבם,

ליום מחר יעלזו – / בזו כסף בזו!





רצים אתו למדבר, / בארץ איש לא עבר,

ונושאו על כתפים / יעשה לו כנפים.





חבל צופנו ימד, / אץ להעמיק, לא יעמד,

והשליכו בבור חפר, / וכסהו בעפר.





ירד הגוף שאולה, / ורוחו תשוב למעלה,

יחד שבים ליסודם – / וזאת תורת האדם!





הנה אחרי כל זאת / באות חדשות רגזות,

גדולים חקרי פלא / ונכבדים מאלה –





יום האלהים ידרשם / ולמשפט יגישם,

וישמיע קול חרדה / ״באו ימי הפקדה!





רב לכם ולתלאות! / המעט מכם הלאות,



לחטאים האלה?״ / ויאמרו ״לא נעלה!״





ורוח מאל יענה: / עוד יש שבר משנה,

חזקו מאד פשעיכם / ואשימם בראשיכם!





נצב הגוף באימה, / דברי פיהו מרמה



״מי זה יוסיף סרה? / ומי כמוני יקרא:





דיי לשאת חמסים / ולשמע קול נוגשים –

בחרי עפר אהלי / ושנתי ערבה לי!





יש מעשה ודעת / ברוחי המרשעת,

פקדו וענו עזות / את הארורה הזאת!״





זמות רוח נתקו / בשמעה אמרים עתקו,



״מה תריבני, חורפי – / גם אני לא אחשך־פי:





כרשעו, אל, שפטהו, / בריבו אל תנקהו,

בדם פשעו מתגולל – / הוצא את המקלל!





רעות בכל עת חרש, / ותאות נעורים ירש –

ואיך הנצל יאבה? / כזב חשד ירבה!״





והגוף ישיב גדולות, / ערות רוח לגלות

״אל תחמלו עליה, / טמאתה בשוליה,





לוח כתב נפרטת / ׳הנפש החוטאת׳ –

ואיכה אל אדוניה / העזה פניה?





בפשעה פה נקראתי, / אשר הסתה אותי,

לכת בחלק דברים / אחרי הבחורים.





ראו: מיום נסעה / התמצא בי רעה?

באשר אשכב שמה / לא עשיתי מאומה!״





כשמן רכו דבריו, / אכן סביביו צריו:

״אני אשיב כהתמך, / הנצבת עמכה!





הן בשובך לעפרים / ולמחלות הקברים

באתי אני אחריך – / לו יהי כדברך״.





״אך אל שאול ירדת / ומחיים נכחדת –

ואשובה אל שרשי / ותנצל נפשי!״





מבין כל נסתרות / ויודע כל ספורות,

בגדלך, שוכן שחק, / מדבר שקר תרחק!





נורא בצדק יענם, / במשפט האמת ידינם,

יחברם כפעם בפעם, / וישלחם ביד פשעם.





וכאשר חטאו יחד / בעודם לבשר אחד,

כן יצו עשותם כלה / את אשו הגדולה!





מי זה ירא וחרד, / מחטאיו יפרד,

לבלתי ינהגו התולים / איש על הגחלים!





אם עיני לא שמתי / עלי ועל נשמתי

או יום או יומים – / מי לי בשמים?
