


	
	[למנצח על־מגנות לבני פאדובה]
	




Isac Vita Cantarini


[נולד בפדובה בשנת 1644 ומת שם בשנת 1723]






למנצח על־מגנות לבני פאדובה / שיר
מזמור להזכיר



לך יי חסד / אשר הרחבת לנו בצר



שפתי כזב קשרו שקרים / לאסר אסר עלי־נפש



עלילות קמים והגיונם / נצבו לבלענו חיים



הבודאים לנו לדאבון / והאבדים חשבו לאבד



בימי מצור למצור היינו / והקרב אלינו נקרב



נפשנו תמיד תדם תזכר / ואלול היה אללי



בעשירי בו רעש אחזנו / סמר שערת בשרנו



פשטו גדודים ברגש רב / וסביבות המחנה נגשו



שעריו רעשו מדחי / וידו־אבן במו



דלתי ארזים התפצצו / ובריחי ברזל גדעו



פערו פיהם לבלי־חק / ויבאו כבירים שוטפים



אזי המים שטפונו / לולי יי שהיה לנו



וישלח נושקי רומי־קשת / ויגרשום בקול המלה



ויחר אף יי בעמו / ויסת עוד שטן למשחית



וירגנו ברחבות קריה / וילינו כל־עם הארץ



וידיחו עשקי עשק / זונה עניה בשית חמס



צעקה החוצה ותבך / ואחריה כל־רשעי לב



צעקה רצוא ושוב / ותאמר כי שרפנו בנה



כל־פנים קבצו פארור / והשרים נבהלו נחפזו



חורי העיר עמדו בפרץ / ויאמרו לרגם אתם



אלפי סוררים נוסדו יחד / לנתוש ולנתוץ ולהאביד
שנית



בכשיל וכילפות וכלי קרב / ואש זלעפות בידם



וישובו עוד להרס / וכל־לב נמס במעיו



בנשף בערב יום / שערי צלמות נפתחו



אך שוא תשועת אדם / ותוחלת אונים אבדה



ויהמו חללים בדרך / לקולם נבקעה הארץ



ויקראו חרשים באישון / לשמר את־בתי יהודה



השמרים שאגים לטרף / שדדי לילה לחלק שלל



שדי האיר לילה כיום / ויהפך הקללה לברכה



פחד ופחד היה לנו / המות והפצע הרחיק



ויוציא עמו מאסון / מקינה את־בחיריו



ויתן אותנו לרחמים / לפני שרי המלוכה



ונשא לבבנו אל־כפים / ונשיר נוראת ימינו



את־אלה נזכרה לאלף דור / ונזמרה שמו עליון



רחום ישמע ממכון שבתו / תפלת עני בשביה



בשנות מספר יאתיו / יושיע ממועקה וצוקה



ישבר מסרות עולה / לשלח רצוצים חפשים



יושב בשמים ישחק / ולשוננו תמלא רנה



תחת מגנות־לב מנגינות תבאנה / ישחק לפחד ולא
יחת



לשפתי־שוא יעקב יצחק / ופחד לא יהיה עוד.




(1648)
