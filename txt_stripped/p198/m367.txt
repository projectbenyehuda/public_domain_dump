


	
	מנחת יהודה שונא הנשים
	



[303]: [חזון תחכמוני הזקן]


אמר תחכמני המפלא:/"אני טרם אכלה / לדבר אל לבי – ורעיוני רוהים/ ואראה מראות אלהים:/ והנה השמש והירח קדרו אורם,/ חשך משחור תארם, והכוכבים ממסלותם / שק הושת כסותם, / והגלגלים התגלגלו / והגבעות התקלקלו והארצות התחלחלו, / ואדם ובהמה חרדו / וכל עוף השמים נדדו. / ואמר בלבי; עזב אלהים את האדמה / והארץ תשאה שממה, / כל האדם כוזב / והארץ תעזב, / ואקום כמתאונן / ואעמד להתבונן / ההיתה זאת לכלה וחרץ – / והנה אופן אחד בארץ, / קול קורא / ומי כמוהו מורה / "בן־אדם! עמד על רגליך / ואדברה אליך. / שמעתי את תלנותיך, / ראיתי את דמעותיך, / הסר כעס מלבך / ובערת היגון מקרבך, / ואל יבהילך זמן המרבה עולה / וכל פה דובר נבלה: / כי כל האנשים נבקה עצתם / טחו מראות עיניהם מהשכיל לבותם. / ואם גדל פשעם וחובם – / נשיהם הטו לבבם! / כי מי יתיר עבותות / אשר הנשים עושות? / דרך נכוחה יגרשו / ואת כל הישרה יעקשו, / חברתן ואהבתן כרגע, / על פיהן יהיה כל ריב וכל נגע. / כמה אנשים / כגבה ברושים / היו לחרפה ולבושה / על אודות האשה! / בי נשבעתי: לא הנחש הערים על חוה / ולא עליה צוה – / היא שמה אישה נודד ממקומו / ונחש עפר לחמו, / ואם בניה פשעם ידכאם – / היא היתה האם! / והלא על התרפים / יעקב ובניו נרדפים / ועל עורות גדיי העזים / אתם לחוצים ונבזים. / והלא על דינה / היתה לאביה ולאחיה שטנה, ולולי הנזמים – / לא נעשה העגל לקסם קסמים. / ועל מי נפתו ילדי אמוני / ותהי המגפה בעדת אדני? / ומאשר גברו על צלפחד דברי עונות / לא היו לו בנים כי אם בנות. / ואברם על ישר פעלו / בת לא היתה לו, / ובזאת תולה ארץ כאשכול / ברך את אברהם בכל. / והוסיף לאיוב משנה על כל הטובות – / מלבד הנקבות, / וצוה למשיח על עם קדושים: / לא ירבה לו נשים. / ועל מי נגרע שבט בנימין / צפון וימין? / ועל מי כשל כח / שמשון בן־מנוח / וירדפוהו עד השברים / ויהי טוחן בבית האסורים? / ובעת עוה עמי נתיבו – / נשים משלו בו!



ויען המלאך הדובר בי ויאמר:



ראה שכל פרעה והתבונן דעו



בעת הפליא עצות לענות עם בנות



אשר ענה אותם והגדיל תוגתם



והמית הבנים – וחיה הבנות.



ובעבור זאת העמדתיך לגלות אליך רז כל נביא וחוזה, / כי אותך ראיתי צדיק לפני בדור הזה, / ונגלה נגליתי למענך / ולמען תספר באזני בנך.







[304]: [זרח פותח בתעמולה נגד הנישואין]


ויכל תחכמני לצוות את בנו ואת כל אנשי שלומיו / ויגוע ויאסף אל עמיו. / ויספדו לו כל יודעיו לפנים / באבל גדול ובקינים / וכבדה עליו האנקה / והקיפה הזעקה. / ויקבר אותו בנו זרח / בגבול נחלתו בתמנת סרח / אשר מצפון להר געש, / שנתים אחר הרעש. / בן מאה שנה היה תחכמני בהאסף רוחו, / לא כהתה עינו ולא נס לחו, / מלבד עשרים שנה היה נשוי בימי עלומיו – / על כן לא באו במספר ימיו! / כי בקחתו אשה היה צעיר / אשר לא ידע ללכת אל העיר, / וכאשר גדל וראה את ערותה / נתן לה את כל אשר לו ויגרש אותה / ולא יספה שוב אליו עוד / ולא ידע בפש מאד.



ויכתב זרח את כל דברי אביו ואת כל אשר אמר / ויתן אותם במשמר, / מן היום ההוא והלאה כל המזכיר לו אשה – עליו ילעג וישחק / וגער בו ונס ממרחק, / ותאחזהו רעדה וחלחלה / כי יבעל בחור בתולה. / ויבחר לו שלשה רעים / נדיבים ושועים, / כי רדפו אחרי השכל מימי נעורים / לקחת מוסר השכל, צדק ומשפט ומישרים./ ואלה שמותם / למשפחותם לבית אבותם: / נתנאל בן־מהללאל, אבינעם בן- אבינדב, אחיטוב בן־אבישוע, וילוה אליהם / ויתחבר עמהם. / ויאמרו לו “הנה המקום צר ממנו / והנשים יושבות בקרבנו, / ואנחנו פה יראים מתרמיתן, / פן נלכד בשחיתותן. / ועתה נבחרה לנו מקום פן נאשם – / ארץ אשר לא עבר בה איש ולא ישב אדם שם”. / ויאמר אליהם “חפצי כחפצכם / ורצוני כרצונכם, / פנו וסעו לכם!”



משם הלכו הלוך ונסוע / למצא לנפשם מרגוע, / ועברו מעברות / וחלפו נהרות / ודרכו מסלות, / ארחות עקלקלות, / עד אשר באו אל נחל הבשור, / מקום אשר אין שם מחסור: / על שפתו כל מיני מגדים, כפרים עם נרדים, / וכל עצי פרי ועצי עדן עליו שתולים / מימינים ומשמאלים, / וכל אלה עבות ופארות בבדים תשלחנה / ויחצו הנה והנה. / וכל עפר וצבי יתלונן / תחת כל עץ רענן, / וצפור כל כנף יסתופף בענפיהם / ועל עליהם יפרשו כנפיהם, / פעם יהגו ופעם עלגים, / המצפצפים והמהגים./ ויראו המקום כי טוב וכי שם כל הנעימות / וימצא חן לפניהם מכל המקומות./ וישאו משלם ויאמרו:

ראו, מה נעמו סיסים בצצים / יצפצפו וכנפיהם פרושים,



וגלות יורדות על החצצים / והרוח תרחף בין הדסים,



וכל עץ וכל איש שש בצלו, / ופארותיו בפרותיו עמוסים!

ויאמרו “עד כה דרכה רגלנו, / כי באה נחלתנו אלינו, / פה תחנותנו / כי היא מקום מנוחתנו”. / וישבו בין ההדסים / שמחים נעלסים / פעם יורו חקות, פעם יורו משפטים, / פעם לאכל ולשתות נוטים, / פעם משחקים, פעם מלעיגים, / כי כן משפט ימי התענוגים. / חדש בשנה יעלה זרח בכל שכניו / להוכיח שלומי אמוניו, / ובכל מקום אשר יפנה / כה יענה: / “גורו לכם מפני חרב נטושה / ואל תגשו אל אשה! / בקשו רוחה / ומצאו הנחה, / הרחיקו בעלי נזמים – / למען תאריכו ימים, / להבאיש ריחם – / תפארת בחורים כחם: / כי לא יעטו כלמה ובושה, / אם נשמרו הנערים אך מאשה. / רע הנשוי לעד לא יבלה / מנפש ועד בשר יכלה. / כל הנושא – / בחשך שמו יכסה, / כאבו יגדל וכבודו ידל, / השומע ישמע והחדל יחדל. / כל בעל־אשה גרש יגרש / פן יורש, / והמשכיל יבין ויסר, / ודבר שפתים אך למחסור”.






[305]: [כינוס נשים]


כה יעשה זרח, וכה רעיוניו פתאים מחכימים / וכה משפטו כל הימים. / המשכילים שמרו תורתו לא עזבוה כל ימיהם / ורבים מעמי הארץ מתיהדים כי נפל פחד זרח עליהם, / וכל מדינה ומדינה מקום אשר דבר זרח ודתו מגיע – / לענה ורוש לכל הנשים משביע, / ותועדנה כלן קטנות וגדולות, / אלמנות ובתולות, / רוגזות וחולות, / לקחת עצה להתעולל עלילות, / וצירים אחזון וחלחלה / ונפשם מאד נבהלה. / ויתגודדו כמשפטן, / כי הוביש מבטן, / וגדלה צעקתן ואנקתן / ועצמה אנחתן / ורבה מהומתן / ותעל שועתן / ותאמרנה “מה לעשות באיש אשר שד ושבר הנחילנו / אשר כלנו ואשר דמה לנו? / הנה נודעה חרפתנו ומעללינו – / ואין איש בארץ לבוא עלינו, / כי מי הוא אשר ערב לבו לגעת אלינו / ויתן אותנו לחרפות? / תתעלפנה הבתולות היפות, / כל בת אבד פרי מטעה, / בתולה – ואיש לא ידעה. / הנה זרח שמנו למורא ופחד / והחזיקו שבע נשים באיש אחד! / דענה וראינה כי רעתנו רבה: / כל בתולה סגורה ומסגרת אין יוצא ואין בא. / למה נשכבה בבשתנו / ותכסנו כלמתנו? / קמנה והתיעצנה – / והנשים תשכבנה, / ונעמד על זה השבר: / נקבה תסובב גבר! / ואל יהי בעינכם הדבר הזה נקל ונבזה – / משפט מות לאיש הזה, / סקול יסקל ואל יגאל, / כי הוציא שם רע על בתולת ישראל. / ואיה בעלי מרמה / אשר בידיהם זמה? / ואיה תרמית הנשים ונכליהן / ואיה מועצותיהן ותחבולותיהן? / ואיה ערמתן / ואיה חכמתן? / ואיה מרמתן / ואיה תרמיתן? ומדוע אבדה חכמתנו / ותושיה נדחה ממנו? / על איש אחד נרנו יכבה / וחוטא אחד יאבד טובה הרבה / הבה נתחכמה לו ומגוי נכחידנו / ונקחה נקמתנו ממנו, / וחכמת זרח נאבד מהרה – / ואיש ואביו ילכו אל הנערה!”



ותקם מביניהם אשה חכמה / בעלת אוב ומרמה, / ברב כשפיה ונאפופיה / נהרות לא ישטפו רשפיה, / ושמה כזבי בת־ירשה, / כי אחזה התרמית למורשה, / ולה איש זקן ושמו שקר בן־חפר / ורעתו לא יכלכל ספר. / ותאמר להן “שמענה, נשים שאננות, / ואל תגברנה לשונות! / כי שחין אסונים / לא יחינו דבלת תאנים, / ונביא עצות מרחוק / ותחבולות לבלי חק. / כשלו גבורים / ונמסו ההרים – / בשתים קרני שכלו אגדע / ועצתו אבלע, / לבל יראה בעיניו / וישמע באזניו. / צאינה וראינה בכל המקומות / ובקשו בכל העלמות / נערה תמימה כלילת יפי / אשר אין בה דפי, / יפה לכל מוצאיה / נושאת חן בעיני כל רואיה / ברה כחמה יפה כתרצה / כשמש מציצה, / מקבלת כל רואיה בנפש חפצה, / בעלת מוסר ועצה, / מתבוננת משל ומליצה, / נגידים תדבר / ושירים תחבר, / אמריה כנפת מתוקים / כראי מוצק חזקים, / מצפון תעיר צפוניו / וממרחק תביא עניניו / ובכור השכל תצרפנו – / גחלים בערו ממנו, / יודעת נגן על כנור ועל נבל, / מעוררת השמחה והאבל. / בנערה הזאת ימסו אסוריו, / כי יצר לב האדם רע מנעוריו. / ואם לא ישוב כאשר יכון ויאות, / למות תוצאות, / הזהב יפתנו / ולבאר שחת יורידנו, / הוא ינתק מוסרותיו / והוא יפר מחשבותיו. / הוא מוריד המלכים מכסאותם / ויוציאם מחצריהם ומטירותם, / הוא מחכים הפתאים / והוא מכסה על החטאים, / והוא משפיל בני ענקים / ומסלף דברי צדיקים, / והוא מישר המעקשים / והוא משען ליוקשים / ועינים לעורים ואזנים לחרשים, / המשמח אלהים ואנשים, / והוא לכל אחיו / ציר נאמן לשולחיו”.



ותשא משלה ותאמר:

שחתי “למי כבוד, למי יקר?” / ואשמעה אומרים "למי זהב –



בו הכסיל יעז ויתרומם, / בו כל אנוש ישיג אשר יאהב“.

ועתה אחיותי ורעיותי, / אלה מועצותי ואלה מזמותי: / נשים העלמה לנגדו והזהב למולו / אולי יפתה ונוכלה לו”. / ויאמרו אליה “עלינו לתת לך את כל אשר תחפצי, / אולי תוכלי אולי תערצי”.






[306]: [נשואי זרח שונא נשים. – אימות ליל החתונה]


ותיקד אש האהבה בקרבו / ושביב החשק בוער בו, / ויקרא לשקר ולכזבי המרשעת – / והוא בבלי דעת! – / ויאמר להם “הרבו מאד מהר ומתן עלי – / ואעשה כאשר תאמרו אלי! / והשמיעו זאת בכל איי אלישה / ותנו לי את הנערה הזאת לאשה”. / ויהי כשמע שקר את דבריו, / נעמו אזניו וצהלו פניו / ויאורו עיניו / וכל תוגותיו נשכחו, / ויקד ארצה וישתחו. ותמהר כזבי לבשר את העלמות הבתולות / בגבעת הערלות, / ותאמר להן “הידעתן זרח המוכיח / אשר לבבו עלינו הקשיח – / הסגירו יי ביד אשה / והנה הוא לחרפה ולבושה! / נתן אלהים בידינו / את זרח אויבנו, / פרשתי עליו רשתי / ונלכד במצודתי, / נפתה לבו על אילה שלוחה / והיום ירד כצאן לטבחה. / לעת ערב אחליפנה באחרת – / ותחתיה תעמד הבהרת!” / אז תשמחנה הנערות – / השפחות והגבירות / על דבר הבשורות: / זאת מהללת ומשחקת, / וזאת עוברה חומקת, / וזאת לכזבי חובקת, / וזאת ידיה מנשקת, / וזאת לרגלה סוגדת, / וזאת מעלז רוקדת / – וזאת ערשה רובדת. / ויתנו לה עשרה זהב וחגורה / הבכירה והצעירה, / על כן קרא למקום ההוא נחל הבשור / על הבשורה שנתבשר באותו מישור.



ויאסף האיש הזקן את כל אנשי המקום לרבוא ורבותים / וסגרו דלתים, / ויבואו האנשים / על הנשים, / ויעש זרח משתה גדול לרחוק ולקרוב / ויזבח שור ומריא וצאן לרב. / ויעש שם שמחה גדולה – / כל שבעת הימים שתיה ואכילה / בתפים ובנבלים / ובמנענעים ובצלצלים. / ואחרי כן קם האיש הזקן מביניהם / ויקח ספר הברית ויקרא באזניהם:

נסח הכתבה



ברביעי בשבת / היתה לאיש הזה סבת / ומשוש לבו נשבת / לגלות ערותו / ולהראות חרפתו. / בארבעת אלפים ותשע מאות ושבעים ושבעה / הבדילו יי לרעה. / בשלשה עשר לחדש אדר / לא תאר לו ולא הדר, / איך זרח החתן / אשר כבודו לאחר נתן / אמר לה למרת־רוח רצפה בת־איה מבנות פותיאל / מטמאות בני ישראל: / ימים רבים תהיה־לי לעבד את עבודת בני ישראל / וכל מחסורך עלי אין להוחיל עד בוש, / לחם לאכל ובגד ללבש, / וכלכלתי אותך כחק כל איש ומפעלו / ואנכי אעלך גם עלה. / ויהיבנא ליכי / מהר בתוליכי / מאה ערלות / כמהר הבתולות, / וייטב הדבר בעיני הקדשה / ותהי לו לאשה. / וצבי הוא ואוסיף לה מדילה: נקיון שנים / ופיק ברכים / וחלחלה בכל מתנים. / ועוד רצה ונתן לה במתנה / את ארבעת אלה למנה: / תחת המעילים – / נגעים גדולים, / ותחת הטבעות – / צרות רבות ורעות, / ותחת פתיגיל – מחגרת שק / ותחת בשם – מק. / ודא נדוניא דהנעלת לה הכלה: / חרפה ושמה ושערורה וכלה. / נמצא סך תוספת כתבה ומתנה ונדוניא – / כויה תחת כויה / ופצע וחבורה ומכה טריה!



וכך קבל זרח על עצמו / שלא יהיה אלהיו עמו, / בטרם ימלא שנה / יהיה למשל ולשנינה. / ויקנו מיד זרח קנין שלם, / כי יהיה כחרש לא ישמע וכאלם, / ויעידו עדים נאמנים ויכתבו בספר: / בן־טבאל עיפה ועפר, / ונעל בן־עבד ועבדון, / ובלהן וזעון ואסרחדון.



וביום הרביעי נקהלו אל עמק הברכה, / כי שם ברכו את זרח – על כן קרא שם המקום עמק הברכה. / ויהי כל העם נדון / “הוי אדון / והוי הודו! / עתה ידל כבודו, / באה פקדתו / ונבקה עצתו. / איה כל נפלאותיו / ואיה כל תוכחותיו / ואיה מועצותיו / ואיה מחשבותיו? / הזה האיש מרעיד האנשים / ומרעיש הנשים? / היום כלו זממיו / ונכרת מעמיו!”



ותקח כזבי את הנערה / ותתן תמורתה אשה מצרה / כעורב שחרה, / שפתיה כנאד על נאד נפוחים / אשר כל רואיה נאנחים. / ושער בשרה כקוצים כסוחים, / כסו פניה חרלים, / מברחת יונקים ועוללים, / היתה בעיני כל בזויה, / ושמה רצפה בת־איה. / ותקח כזבי את השחרה / ותביאה אל זרח החדרה, / ויפנו וילכו שקר וכזבי בשמחה, / כי כרו לפני זרח שוחה, / ויתנכלו על הכסף אשר הכינו / וילכו ויטמינו.



וישאר זרח עם רצפה בת־איה / עירם ועריה, / ויהי עד כה ועד כה / ויזרח לו השמש והוא צולע על ירכו, / והנה רצפה – מקום אילה שלוחה, / ותחת מעשה מקשה – קרחה! / ויקרע את לבושו / וישם עפר על ראשו / ויקרא “קשר שקר / רב־שקר, / על זאת אקונן ואילילה /ומספד מר אגדילה. / הכסף אזל ונשאר הדפי / אין זהב ואין יפי! / כי היה הקשר אמיץ ועלי גבר – / פתי יאמין לכל דבר. / עד כה רבו ששוני / ועד הנה עזרני אדני. / אזי קרתי / ואת האשה הזאת לקחתי, / ולא אענה עזות – / חפרתי את הבאר הזאת / ולקחתי לנפשי מהרסיה ומחריביה / ונתתי נפשי בכף אויביה. / אוי לי, מי יועילני / ומי יצילני? / ולחצי מות לא יועיל שרין / כי בא החלום ברב ענין”. / וישא משלו ויאמר:

למי אשא תלנותי / ואגיד רב יגונותי?



חמסי על זמן הרבה / מצוקותי ואנחותי



ופרש פח במעגלי / ושלח אש בעצמותי,



והימים הדפוני / והפרו מזמותי.



לזאת אהים ואשתומם / ואגז את קוצותי,



ולבנות יענה אקרא / להתקבץ סביבותי



ותשנה נהי עלי / ואבכה על בחורותי.



אהה היום מרי שיחי / וזה מספר חלומותי!

ותאמר לו אשתו “חגר מתניך / ואל תתן שנה לעיניך / ואל תמרה דבר פי וקולי / ולך קח לי: / כלי כסף וכלי זהב ושמלות / הנטיפות והשרות והרעלות / ומשכן ודירה / וכסא ומנורה / ושלחן וכפות / ועלי וריפות / ומכבר וכישור / ומחצלת וכיור / וטנא ופלך ופרור ובקבוק וקלחת / ומכתש ומטאטאת ומטפחת / וסיר ותנור וכירים / וחבית למים / ומזרקות ויעים / וכלים וגביעים / ובתי הנפש והמעטפות / ורדידים וצניפות / ומחלצות וסדינים / ונזמי האף וגליונים / וחריטים ושהרונים / ולחשים וקשורים / ורקמה ובוץ ופארים,/ הטבעות ושביסים / וצעדות ועכסים – / ומלבד אלה: בגדים חמודים / לשבתות חגים ומועדים, / וכהנה וכהנה / ידיך תביאינה / ואם מנשא כל אלה תלאה / והיית משגע ממראה עיניך אשר תראה – / וזה־לך האות: השנה הזאת תהיה לך לעצבון / והשנית לדאבון, / לעבד תהיה אחרי היותך שר / ודמן על ראשך אל יחסר, / כי לא יאות לך להיות אחד מעבדי / לצקת מים על ידי”!



ויהי כשמע זרח את דברי האלה / ויצעק צעקה גדולה / ויאמר “העבד ישראל? / אם אין לו גואל? / עתה קמו השפחות / על הסגנים והפחות, / ואזנים רצועות / ימשלו על המגבעות”. / וישא משלו ויאמר:

שחקים וארצות עם גבעות, / בכו על זאת והורידו דמעות!



הבן־חורין וילד שעשועים / יהי עבד למי אזניו רצועות?

ויוסף עוד שאת משלו ויאמר:



שתים מררו חיי ואותם / לדור ודור אספר ואחוה:



יליד חכם אשר עבד באשה / ובן משכיל ליד סכל יקוה.

ותאמר לו אשתו “אל תשא משליך / ואל תגביה קולך! / אין לי חפץ בשכל ובמוסר, / כי אם בלחם ובבשר. / משליך ושיריך נגדי כאין, / בשיר לא ישתו יין. / אמרת אך דבר שפתים / לבוא על רחם רחמתים? / קום קח מחרשתך ואתך, / שא נא תליך וקשתך, / והקף את העיר מערב עד בקר / גנוב ורצוח והשבע לשקר! / ובמרבית ביתך תטריף את לחם חקם / ולא יראו פני ריקם, / כי האשה – עליך מחסורה / לעבדה ולשמרה. / ואם לא תביא את כל אשר אחמד – / בחוץ תעמד! / וכל זה נקל עד בוא עת הלדה – / אז תכבד העבודה: / אחי וקרובי לנגד עיניך / סביב לשלחנך, / ותעשה להם בכל יום מטעמים – / שה תמים, / ולמילדת לא יחסר / יין לחם ובשר, / ולבניך תקרא אשה מינקת – / הם מגישים אליה והיא מוצקת – / ואמהות ושפחות תקנה להם / לאכל את צואתם ולשתות את מימי רגליהם, / וידי לא תגע בהם!”



ויפל זרח מלוא קומתו ארצה / ונפשו בחיים קצה, / ויבך על בחוריו / ועל ימי נעוריו, / כי ירדו למצער / וכי הלבין שער ראשו – והוא נער. / וישא משלו ויאמר:

ראו בי פלאות ונודו לילד / אשר שב כאלו שנותיו שמונים,



והלבין שערו וקדר מאורו, / וזמן שחרים – ומצא לבנים!






[307]: [אבלו של זרח]


וישמעו שלשת רעיו / הקרובים אליו / את כל הרעה הבאה עליו, / ויצאו איש ממקומו / לבוא לנוד לו ולנחמו. / וישאו עיניהם מרחוק ולא הכירוהו / כי משחת מאיש מראהו. / ויען נתנאל בן־מהללאל ויאמר “הצבי, איך נלכדת / ומכסאך איך הורדת / ועם יורדי בור איך נמשלת? / כנפל לפני בני עולה נפלת / ודבריך הנעימים שחת – / מה הבטחון הזה אשר בטחת! / הנך מת על האשה אשר לקחת, / חטאותיך לרעתך הקריבוך, / על כן עלמות אהבוך”. / וישא משלו ויאמר:



אוי לך, זרח, ואוי לך! / דעתך איה ושכלך?



צמחו שערי פארך – / עזך חלף והלך!



שד ושבר עם יגונים / נפלו היום בחבלך,



נבהלו בך רעיונים / בי- ומה עוד לעשות לך?

ויען אבינעם בן־אבינדב ויאמר “כעצם מעלתך ותפארתך / היתה מפלתך. / נפלה פנת יקרתך / וערבה כל שמחתך, / מהרה נתקו זממיך / ונגדעת בחצי ימיך. / זהר נעוריך עטה שחרות / ושמי עזך לבשה קדרות. / דבר שקר וכזבי הקשבת / ודברי אמונים עזבת / ומגדל עז בנה אביך ידך הרסו, / עתה הסכלת עשו. / האיש כמוך נוקש ונכשל / כי היית היום למשל? / עליך אשא במר רוחי: / צר לי עליך, אחי!” / וישא משלו ויאמר:



שמש נעורים אשר חשכו מאוריה



תצרח במרה למכאובה וציריה,



תצעק: אהה, נפלה קומת ברושיה



עד ים רחוקים תשלח את קציריה.



יודעי נהי, מהרו ספדו לאיש אסרה



כזבי להרגו וקמו כל אסריה.



זרח בכו נא והתאספו לתנות וכל



נפש חכמה ממררת אמריה,



מה לי בחיים ומה לי עוד בטוב הזמן?



תבל – חלום שוא וזה חזיון שקריה!

ויען אחיטוב בן־אבישוע ויאמר “אל תבכו למת ואל תנודו לו, / בכו בכה לאשר ברשת הנשוי נלכדה רגלו. / יונה יצא ממעי הדגה – ואין לאסיר אשה תקוה ולא פוגה! / הרכבת אנוש על ראשך / ולקחת מוקש לנפשך. / זדון לבך השיאך / ובפח יקוש הביאך, / איה מועצותיך / ואיה חכמתך ותבונותיך? / תמול היית למושל בארץ – / והיום לכלה וחרץ, / תמול היית כמלך בגדוד – / והיום בזוז ושדוד, / תמול היית נגיד ומצוה לאמים – / והיום חרפת עמים, / תמול היית גבור כאריה – / והיום לא תקום ולא תהיה, / תמול היית כגבה ברושים – / והיום נבזה וחדל אישים, / תמול היה דברך נחוץ – / והיום לא שם לך על פני חוץ. / באמת נקרא שמך זרח, כי זרחה במצחך הצרעת / והקשבת לכזבי המרשעת! / המירות דברי תחכמני בדברי שקר / לכן תהיה כענן בקר”.



וישא משלו ויאמר:



כל איש אשר ימרה עצת אביו / ימות ביד אשה כמות נבל. לו יהיה בימי בנו עמרם – / בו הקללה לו בהר עיבל!

וזאת ליהודה ויאמר:



גם אני לא אחשך־פי / בעוד בקרבי שעפי, / לזאת אכה כפי / ואבכה ימי חרפי / ועליך אשא במר קולי, / היטב חרה לי. / מאד קשה עלי תוגתך / ועצמה לי תלאתך, / זרית ארחך ורבעך – / ובחשך רפדת יצועך, / אילה שלוחה חשקת – / ועם רצפה בת־איה דבקת! / ולא שמעת לקול מוריך – / כזה וכזה יקרך, / עזבת דרך המישור – / ככה יעשה לשור, / כל הממרה דבר אביו וקולו – / כמשפט הזה יעשה־לו! / שקר וכזבי לחרפה שמוך – / כמוני כמוך, / הנשוי שמך לחרדה / ויגל את מסך יהודה". / וישא משלו ויאמר:

אהה זרח, אבוי לך – ואבוי לי, / לכה נבכה וננוד לאחדים לזמן הרע אשר רדף כאויב / ושמנו ביד נשים עבדים.
