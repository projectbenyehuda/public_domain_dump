


	
	וילך האופן : מקאמה
	







בגנות האשה והנשואין






וילך האופן כאשר כלה נאומו / ותחכמני שב למקומו / ויקרא לזרח בנו / וישימהו על יד ימינו / ויאמר לו: הנה נא זקנתי /
לא ידעתי יום מותי / ועתה, בני, שמע בקולי וקבל עצתי, והחזק במוסר אל תרף ידך / שמע לאביך זה ילדך / אם תהיה צואתי על לוח לבך חרושה: אל תקח לך אשה / שלום בפיה, הוות בקרבה / לא ינקה כל הנגע בה / שש ומשי תלבישנה /
ומעדנים תשביענה / ואיש אחר ישכבנה / בור הנשוי / למות הוא עשוי / והמשכיל – ממנה נפשו ישמר / ונפל שמה שור או חמור / כל הנושא אשה / מכתו אנושה / אם יתנו לו זהב מלוא ביתו / לשמה ולחרפה תהיה אחריתו / כי לרעתו ולאידו / צרור הכסף לקח בידו / בני, אם חכם לבך, כספן וזהבן ממך רחוק יהיה / ושונא
מתנות יחיה / לכל מקרה רע יושת כפר / וכפר מקרה הנשוי לא ימד ולא יספר.





וישא משלו ויאמר:

אם בצפון אשה תשכן / שים פניך נגד דרום
אם תשמע בין עדה שמעה / מן העדה ההיא תרום

אם תשלח יד אל מלבושך / הנצל ממנה ערום.







בני, אני היום ילדתיך / ובמעגלי ישר הדרכתיך / ושכל טוב הודעתיך / אמצתיך
עזרתיך / אף תמכתיך / פן תלכד בפח יקושה / ואיבה אשית בינך ובין האשה / אל תחמד יפיה ואל ישיאוך תענוגיה / כי רבים חללים הפילה ועצומים כל הרוגיה. / בני, אל תתן לאחרים הודך / פן יהיה לך למוקש וקיקלון על כבודך / אם תהיה נשא וגאה / שערי מות תראה / חיים לבאר שחת תורידך / ואין לאל ידך / והיא תשאר אלמנה בעבור תהיה לי לעדה / כי לא אלמן ישראל ויהודה. – – –





בני, יגיע כפיך וכל מחמדיך / יהיו לך לבדך / ולא תהיה רועה זונות / ואל יהיו לך בנים ובנות / כי את אשר מאסת לשית עם כלבי צאנך / יתחתנו בך מאין רצונך / ויבזו זרים יגיעך / וכל קניניך / אשר למצוק אותך יביאו / מחלציך יצאו.







בני, השמר לך פן יהיה עם לבבך דבר בליעל / ואחרי מותי בתוכחתי תגעל / כי תאמר: ׳הלא בישישים / חכמה ודעת קדושים / וישאו להם נשים׳ / אל תעשה כמעשיהם לשים נשף חשקך לחרדה / אם זונה ישראל אל יאשם יהודה / ואל תמית
בנער נפשך / ובחצי ימיך אל תחשיך שמשך / הרף מן הנשוי ורעתו / כי לא תוכל שאתו / כי יקח איש אשה / מכל מחמד נפשו יבשה / ימיו מועטים ורעים / ודבריו אינם נשמעים / פניו לא יחשך מכלמות ורוק / כל עובר עליו ישם וישרק / יצרו צעדי אונו / ויבש מקורו ויחרב מעינו / מחלתו תגדל / ומהיות אביון לא יחדל / כל ימיו מכאבים ושנתו נהיתה עליו / המק בשרו והוא עמד על רגליו / ומעדני ימי נעוריו יכלו ויבלו / והימים הראשונים יפלו / והאיש הפנוי / למגדל עז הוא בנוי / הוד והדר לבושו / ונזר אלהיו על ראשו / מריח מים יפריח / וכל אשר יעשה יצליח / כל ימיו שליו שלאנן / ומתערה כאזרח רענן / קל מהרה יבוא כל אשר יקוה / ואיננו חסר לנפשו מכל אשר יתאוה / יצו ה׳ אתו את הברכה / ועוד לו אך המלוכה.





וישא משלו ויאמר:

פדות לאסירים יש / ושחר כתם ליל

ומרפא למכאבים / ויש קץ לכל תכלה

ורון אחרי יגון / ותנחום לכל אבל

לצרה ולמצוקה / רוחה והצלה

אבל לאסיר אשה / בכו נא למפקדו

הכי קץ פדות נפשו / נסתם ולא נגלה.





בני, אל תתן לנשים חילך / ואל תשכן עולה באהליך / פגוש דב שכול וחברת זאבי יערים / ולא עם אשה בחדרי חדרים / חכמי קדם הפטירו בשפה: / ׳מן החדר תבא סופה׳ / בין שמיר ושית / ולא עם אשה בירכתי הבית / בין תנור וכירים / ולא בין שני שדים / בין קוצים כסוחים
וחרולים / ולא בין קוצות תלתלים / בין קמשנים / ולא בין שהרונים / סוף לראש חבוש / וערום מכל לבוש / ולא כסות וצדה / עם בוגדה / קול נהי וקול יללה /
ולא קול חתן וקול כלה / אבל יענים / ולא מצהלות חפות חתנים / אל בית משתיהם לבך אל יפתה / טוב ללכת אל בית אבל, מלכת אל בית משתה / משוש חתן
וחפתו / לבשתו ולחרפתו / ולא יראה בטובתו / מלוש בצק עד חמצתו / אויבי איש נשי ביתו.





וישא משלו ויאמר:

יעצבו הנשאים נשים ומ/טוב הזמן ישכחו שכוח

כמה אנשים גברו, עת נשאו / נשים ראיתים הולכים שחוח

לו הכלבים נשאו יסכרו / פיהם עדי לא יוכלו לנבח.





בני, אל יפתוך בעלי נשים בפיהם / כי תעי לבב הם / סר מעליהם הודם וצלם / צאנה ואלפים כלם / דרכיהם ומעלליהם לא תלמד / המה יאבדו ואתה תעמד / הנשוי כמה מרורות השביע / ונערים עד שאול הגיע / ובחורי ישראל הכריע / ישועו ואין מושיע / מן העשירים הניס רבבה / ויכם וירדפם עד חובה / נער הייתי / וגם זקנתי / ולא ראיתי נשוי מצליח דרכו / וה׳ הגיה חשכו.





וישא משלו ויאמר:

אם יתנו בך בעלי נשים / עין פתח יד ואמר חמש

גור מפני אשה ולא תחמד / יפיה אשר ימס כחם שמש

אם תעלה על לבך היום / השח גאון תפארתך אמש

לו דמתה קומתך תמר / חיש תהלך כפוף כמו חרמש.





בני, הט אזנך ושמע דברי חכמים / למען ייטב לך והארכת ימים / כל המרבה עם האשה שיחה / סופו לגיהנם פתוחה / בעודך רך תקרבך / וככלות כחך תעזבך / כל נתח טוב תאכל מן המטעמים / ותגיש לפניך העצמים / תרמית הנשים ונכליהן / מי יגיע אליהן? / כמה גדולים השפילו / וכמה יגונות הנחילו / כמה גבורים כשלו בהן ואין מקימם / וכשל גם יהודה עמם.





וישא משלו ויאמר:

כל היקום הכו בעורון / ונבון דברים תם וסף דעו

כי יחזו גבר בים נשוי / טבע ויבא אחריו רעו.





בני, אל תחמד יפיה בלבבך / ובערת הרע מקרבך / כי הנשים אינן יפות / כי אם
לרקחות ולטבחות ולאופות / ברוך השם שלא שם חלקנו כהן / ומעשינו כמעשיהן / כמוהן יהיו בועליהן / וכל הבוטח בהן / באש ישרפו אותו ואתהן / ואני מודה לאשר עשה את הים ואת היבשה / שלא עשני אשה / טוב לפני אלהים ימלט מקרבה / וחוטא ילכד בה / אלה תולדות הנשוי וקורותיו / ולא זכרתי לך חצי תלאותיו. – – –
