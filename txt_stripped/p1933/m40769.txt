


	
	חגיגת הקרנול אצל הנוצרים (מתוך הספר “פלפול זמן זמנים וזמניהם”)
	



Jona Coen Rappa

[חי בקזלי מונפרטו לערך 1680]






ובימים ההם ובעת ההיא / לא נשמע קול וי
אוי הוה ונהי / ונס יגון והי / ואין דורש / ואין
מבקש / כי אם־רבוי ושנוי יינות ומאכלים / ואין
קול ואין ענה כי אם מנים ועוגב תפים וכנורות
ונבלים / וקל קרנא משרוקיתא קיתרס וחללים /
ושבכא ופסנתרין וכל זני זמרא נבובין וחלולים /
ושרים כחללים / עוגות מצות וחמץ והאסקריטין
והסופגנין / והחלות והאצטגנין / זבות וזבים חלב
ודבש ותבלין / ורקיקין ורבוכה אשר על־מחבת
בשמן תעשינה / בכל־סטר ואתר מצינא / ויין
מרקח ומעטר / בכל־מקום מגש מקטר / וכל־צפור
כל־כנף / אשר הורם ואשר הונף / ושאר כל־העופות
וברברים אבוסים / המה נקראו המה נבחרו והמה
הנגשים / ושור פר / בריתו לא תופר / ועגל / נושא
הדגל / וכמעט כלתה רגל / ואיל / עושה חיל / ושה /
אשר כל בו חוסה / ובכל־עיר ועיר / קול קורא משעיר /
וגדי / גד גדי / וטלה / מקצר ועולה / והארנבת /
מיד ליד נגנבת / ושפן / בא אחריתן וסופן / ובשר
ושמן החזיר / הוא הסובב והוא המחזיר / ואין מי
שממנו יזיר / ומאכלי גבינה וחלב אשר לא ימנו
ומרב לא יספרו / תשבחות ישמיעו וזמירות יאמרו
/ וכותח הבבלי וזיתוס המצרי / אינו הולך ערירי /
ויטבלו בם בשר שור ועגל ומריא / וכל־מיני לפתן
ומטעמים מרכבים / ומאכלים חשובים / ומעלים
וערבים / כלם ברורים כלם אהובים.

סוף סוף כל־הדגים / אחור נהוגים ונוגים /
וירק / בפניו יורק / ויצא יצא כברק / ומיני קטנית
/ אין תכנית ותבנית / וכל־הרואה אותן בחלום יושב
עליו בתענית / ובשר הרזה / נמאסת ונמבזה /
וכל־התבשילין / צלוים שפודים שלוקין וכבושין
מכל־מין ומין בטלין ומבטלין / לא שרירין ולא
קימין / ובמספר ובמפקד יתר העדונין / אשר על־זה
אנחנו דנין / נער וזקן טף ונשים בשנותם את־טעמם
את־טבעם יתחפשו ושנותיהם יסתירו ויעלימו /
ובלכתם בשוקים וברחובות הכלם לא יכלמו ולא
יאדימו / וסתר פנים ישימו.

הן־אלה קצות דרכיהם / ותענוגיהם / וטוב
מאכליהם ומשתיהם / וערבות משקיהם / יחרה אף יי
בהם / זה הכלל בימים ההם / ובזמן הזה יבקש
החכם ואיננו / המשכיל ולא תמצאנו / הנבון
ובעל דעת מקצוי ארץ ומים עד־ים / ולא נודע מקומם
אים / אם בעל המוסר / צלו סר / ואם המליצים
והחרוצים / כלם עברו חלוצים / וחלפו למו דחופים
ורצים / ומשכו ידיהם את־לצצים / בעלי ענוה /
שטופים במיץ הענבה / כאלו הקבע הדבר חובה /
החסידים / חסרים חסודים וחשודים / הפקחים / נהפכים
והופכים / ובכוסות ובצלוחיות נופחים / הסופרים /
כופרים ונחפרים / השופטים / נשפטים ומפתים /
ואחד אחד נשמטים / אנשי אמת והישר / אכלו בסר /
ולא ידעו כי יבואם חסר / הנגידים / נאגדים /
ובגדו בגד בוגדים / השוטרים נפטרים / ונסתרים
בסתרים / המושלים / למשלים / היועצים / כקוצים
ועצים / צחי הלשון / ללעג ולקלס וללצון / והשבע
לעשיר איננו מניח לו לישון / המשוררים / סוררים /
והמה צוררים וצרורים / הטבעיים והתכוניים /
בדאים ופנויים / והפילוסופים והאשפים
והמכשפים / נכספים ונהדפים / ולא נאספים /
החרטמים / אטמים וסתמים / והקוסמים / נחשבים
כקיסמים / ההוברים / כשבורים וקבורים / ואין אמר
ואין דברים / ובעלי מבטא / בבלי הבטה / באה להם
החבטה / וכלם מלאי חרטה.



נמצא הדקדוק / בבקבוק / ההגיון /
בשגיון / חכמת השיר / בדוד וסיר / הפעלה /
בבתולה ובעולה / הצרוף / לגדוף וחרוף / חכמת
המחקר / אחרי הצאן והבקר / התחבולה / נתבהלה
/ ההנדסה / בדיסא / והיא נתושה ודרוסה / ההלצה /
לכלה ונחרצה / החשבון / לעזבון / ולזבון /
התשברת / נראתה והיא שחרחרת / מלאכת היד /
מגיד / הרפואה / כאיסטיס ופואה / חכמת התכונה /
מגנה / כי אין בפיה נכונה / החיטים / בועטים /
הספרים / מספרים ספורים שפירים / הכובסים /
מיצרם נכבשים / והרצענים / נשענים ונזונים /
ואינם מתענים / הספן / שמח בכל־אופן / החמר /
אזל ומזמר / הגמל / הרה לו עמל / הרועה / מטיל
ותועה / וכל־האומר שאין לו חלק בהתולים הללו
אינו אלא טועה / הנגר / דומה לתגר / ואינו רץ
לבאר המים כהגר / הבנאי / בונה דרך גנאי /
וכמעט שאין לו פנאי / ובלן / ראש לכלן / כשהוא
רואה הצוחקות ושומע לקולן / אומר: “בריך רחמנא
דיהבך לן” / חרש / יחרש / וחושב חושב מחשבות /
אשר לא טובות / ורוקם עוקם מלאכתו / וללצים הוא
יליץ בקימתו ושבתו / וגם המלך מוחל על־כבודו /
ואינו חושש לאימתו והדרו והודו / והמלכה
מלכה עלה ישפר / ויפה־תאר ומראה בעיניה תחפר
/ הדוכוס / על־יקרו לא יחוס / והשר / מפארו יחסר /
והפחות פוחתים והולכים אחר נוגנים / והשלטונים
והאחשדרפנים והסגנים / נכשלים בבני אדם
שאינם מהגנים / ושרי הצבא / בטנם צבה / ושרי
אלפים ומאות וחמשים ועשרות / מניחים
את־חילותיהם ואת־השמירות / משמרתן ומשמרת
העירות / והולכין אחר הבתולות היפות והנשים
היקרות / האפיפיור / שותה יין במזרקי־כסף
ובכיור / והוא עומד ומטיל על־שפת היאור /
החשמנים / אותן דרבו יתירא היה נותן עליהם
סימנים / באשמנים / שמנים ודשנים ורעננים /
ההגמון / אשר כאגמון / ראשו יכניע ויכף / גם הוא
במעלליו דומה לקוף.

והכהנים והעם העומדים בעבודה זרה /
נהנין ונזונין מזיו השכונה והחבורה / וחטאו
ועוו ופשעו ועברו העברה / אין יום שאין סעודתו
מרבה מחברתה / ולא איברי סיהרא לשינתא / אלא
לזנותא והפקרותא.

שבן־לילה היה ובן־לילה אבדה שמחתם /
וסרה מעליהם אולתם / ויקומו איש איש ממסבתם /
ורקודם וצחיקותם / בשמעם את־רנתם / ואת־קול
תרועתם / בבית און אלהיהם / ויפרדו איש מעל
אחיהם / רעיהם וחבריהם / וירגנו באהליהם.

ויהי ממחרת זה נגוע וזה מכה / זה
נשבר וזה נדכה / זה כואב וזה דואג / זה מתאנח
וזה שואג / זה חושש בגרונו / וזה בשרירי בטנו /
זה ממלוי אצטמכתו צועק כאלו נשתטה / וזה
מזרעותיו ומרגלותיו דואב כאלו נפלה עליהם מוטה
/ וחולה ונופל על־המטה / זה חש בראשו / וזה אבד
תאותו ואפשו / ולא ידע כי הוא בנפשו / כלם
אחזי חרב וקרח באו בית שמץ אלהיהם / והמה כרעו
על־ברכיהם / ויזרקו עפר על־ראשיהם.
