


	
	סדר תעניות
	




תענית נוצרים





– – – סדר תעניות כיצד? בכל מדינה ומדינה / ובכל עיר ועיר, וצד ופנה / מקום אשר דבר האפיפיור ודת אותו האיש מגיע / אין מי ישקיט וירגיע / ויצו עליו במפגיע / כל הבשר אסיר / צלי שלוק ומבשל וכבוש בדוד ובסיר / וחלב וחמאה וגבינה וביצים / יצאו מלפניו בצים / והירק וכל מיני קטנית והדגים / אשר עד הנה הם אחור נסוגים / כלם קרואים ראש ומבחר לתענוגים והנשואין והמשתאות והמחולות והשירים / ומושבות הנגונים / והשמחות ושאר דברים / המושכים לבו של אדם והרם ירים / הן הנה נאסרים.



ואלה הבגדים אשר יעשו מכסף וזהב ופארי המגבעות / אצעדה וצמיד ועגיל ונזמים וטבעות / לא התרו לצנועים ולצנועות / ובכל יום ויום הזקן שבהם אומר לפניהם דברי כבושין / כדי שישובו מהעברות שבידיהן וכדי שיהיו בושין / לפעמים הוא מוכיחן על הגזל ועל העריות / ופעמים על שפיכות דמים ושנאת הבריות / פעמים על הפריצות והגאות והגאוה / ופעמים הוא מזהירן על הבשת פנים והצניעות והענוה / ועל הנזירות ופרישות התאוה / לפעמים על הרחקת כל מדה מגנה
/ ופעמים על תפלותיהם ותחנותיהם שהם בחסר הכונה / לפעמים על ההזמה / ופעמים על ההרעמה / והמזמה ומרמה / לפעמים על ההרהורים הרעים ומחשבות הרעות / ופעמים על לשון הרע ושבועות /
פעמים הוא דורש להם תועלת הסגופים והבקשות / ושכר המקים בצדקה את הנפשות / וכל אשר צוה אלהים לעשות.



ויהי כדברו אליהם יום יום כדי שיהא כל אחד ואחד ממעשיו הרעים נבדל ונחדל / השומע ישמע / והחדל יחדל.



הא לחמא עניא די אכלין על מצות ומרורים ולפיד יום / זה ששה וארבעים יום / בתבואה חמשת המינין / וקטניות ושאר זרעונין / וירק השדה והקשואין והאבטיחין / וכל פירות האילן בין יבשין ובין לחין / וכן דגים בין טפלין ובין מלוחין.



כל דכפין אוכלים ושותים ביום לצבות בטן ולפי מלאות תאותן / לחם ויין ולפתן /
ובערב מלמדין את האדם שלא יהיה רעבתן / ככה יעשה אויב כל הימים עד שמצטמקין עורותיהן / ויבשין גופותיהן / ונשתנו מראיהן / ותששו כחותיהן / ולא יכלו עמוד על רגליהם / זה מראו כמלא שחין ספחת ושאת / וזה דומה כמי שכפאו שד / זה בטנו הולך וסוער וקרבו כתנור בוער / וממכאובו וצערו כחמור נוער / זה כרסו נפוח ומלא יגונים / וצוח וי וי על הזרעונים / והרופאים פוסקים ודנים / כי רוח הוא באופנים / ויהי העם כמתאננים / רע על דברי הצומות האלה אשר בעינינו נחשבו כאין וכאפס / והמה אמרו בכל לב ובכל נפש / הגיעו הימים אשר נאמר אין לנו בהם חפץ / ומעיקרא ראיתי בני אוי אויה / המתענים והנם מועטים / ואותן אחת לאחת נשמטים / הזקנים מחמת חולשתם / ונערים מחמת תשות כח תכונתם / האמנין מחמת מלאכתם ועבודתם / מלכי ארץ וכל לאמים והשרים וכל אדם מחמת אסטניסותם / החילים מחמת שמירתם ונמו
שנתם / והחכמים ובתי דינין מחמת יגיעתם / ללמד בתורתם.



זה הכלל לא נמצאו מתענין כי אם העניים מחמת עניותם / ועובדי האדמה בשעה שהם יושבין בביתם / והכלים מחמת כליותם / ובכל זאת אין קצבה לדמעתם / ואין קץ לבכיותם / ואין שיעור לזעקתם / ואין מספר לקולות מררותם / וכל מי שאינו
לא חגר ולא סומא ולא חולה, וכיוצא בהם / כדי שינתן לו רשות לאכל בשר ושאר הדברים הנאסרו להם / עושה עצמו כאחד מהם / אז יכנע לבבם הערל וכל אדם שוקל את מעשיו בכף מאזנים / בהגיע שבועה הקדשה היא בעינים / וכל כך למה? הם אומרים לכפרת הנפשות / וזה החלם עשות. –––
