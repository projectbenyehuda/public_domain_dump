


	
	צרי חליי
	



סימן: צאלח. מנגינה: ראשט.






צרי חליי / סלח מריי / וזדוני

נתחרטתי / כי מרדתי / באדוני

בי חזרתי / לך שבתי / מרצוני, צור גאוני.





אנא הושע / שב מפשע / מרודיו

שמח אבלו / נגד סבלו / מרודיו

אם הוא בער / חשב צער / נדודיו, נד מדודיו.





לך מתגאה / כמה נאה / לסלח

לאיש הלך / לך הולך / שחוח

דך מעודו / גם כבדו / פלוח, מר מלוח.





חשב מעני / כקרבני / וזבחי

לצאת חפשי / הנפש נפשי / ורוחי

רחשי הקשב / וחיש השב / מדוחי, צור מבטחי.
