


	
	צאו הדרו
	




צאו הדרו / גם זמרו / לאל דר מעלה

ברוך אל נורא / אשר ברא / חתן וכלה

מושיב יחידים / רעים דודים / בהוד וגדלה.





אשאל בתחנה / בקול רנה / מדר זבולי

יתן במתן / אל החתן / גביר בקהלי

חיי ומזוני / ובני בני / לשם ותהלה.





לקח יונתו / היא זוגתו / כלה כנוסה

יפה בתבל / ובתוך בבל / הנו יחוסה

זהב רפידה / וחסודה כלה כלולה.





חן בן זקונים / לשושבינים / אל יתן להם

בנשואיהם / אבותיהם / יראו עיניהם

את כל אויביהם / ושונאיהם / יעשו כלה.





חיים ושלום / תן אל עילום / לשר ישראל

עז וממשלה / בלי תכלה / עד ביאת גואל

הוא נשיאנו / פארנו / ראש כל הגולה.





זרע צדיקים / אל חי יקים / לכם בריתו

יוסף עליכם / אלף ככם / עם סגלתו

ישלח אליה / דר עליה / יקבץ קהלה.
