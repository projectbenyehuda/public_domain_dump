


	
	צוה האל
	




צוה האל / לישראל / לשמר בריתו ודתו

ובחותמו / הוא יה שמו / חתם עדתו. ובחותמו





אות זה אל רם / שם בבשרם / יגלה אהבתו

בראותו ילדיו / מעשה ידיו / עם סגלתו. בראותו





לאל אקד / אשפיל קדקד / לאל נעלה במעלה

אשר נתן / אל במתן / לראש הגולה. במעלה





חסד וחן / ולב בוחן / וגם גדלה כפולה

לעד יכון / בטח ישכן / בתוך טירתו. כפולה





חי גואלי / חן נמולי / בן שר קהלי וחילי

כתם הטוב / יהי רטב / לעד מושלי. וחילי





אל חי ישוב / לעם חשוב / בתוך זבולי בגבולי

ישכן עליון / באפריון / בתוך נותו. בגבולי
