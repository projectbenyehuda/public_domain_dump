


	
	אשורר שיר
	



עוד לו שלח אותו להנשיא שי׳יצחק דוד ישועה גבאי

סימן: אני צאלח






אשורר שיר יפה / לכל בשר מרפא / בעזרת אל צופה / ומביט מסתרים



נאום גבר גובר / בטורי שיר חבר / בצדק הוא דובר / ולא גזמת שרים



יבשם לך שירי / פארי גם נזרי / כצפרן וצרי / ביום הכפורים



צבי עפר הטוב / רדף אויבך וחטב / תהי תמיד רטב / לילדי העברים



אדמה המוסר / לקב אחד עשר / עשרת לך השר / ואחד ליצורים



לכה כוכב צדק / והחזק כל צדק / לאיש דובר צדק / ומגיד מישרים



חנה תמיד עמו / בשכבו ובקומו / למתק את טעמו / ואת נסו תרים



אנשים ירחיקו / כליהם יריקו / בעבד יספיקו / בילדי הנכרים



שפתיך שבו / אנשים נקבו / לעבדך יתאבו / ולא יצאו חורים



לבבי שבית / בשכל שרית / ואותי קנית / לעבד לדורים



לפאר ולהלל / בכל יום ובכל ליל / ובשמך אתפלל / ערבים ושחרים



ואעתיר אל קונך / להרבות את הונך / והוא יתן חנך / בעיני השרים.
