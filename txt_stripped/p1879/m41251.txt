


	
	צאן מרעיתי
	



סימן: צאלח. מנגינה: חוסיני.






צאן מרעיתי / אשיב אתי / אלי ביתי / ולגני



אבנה בית אל / וההראל / והשב אל / מכוני



נסכי וקרבני / ואשכן גם אני / בין בדי ארוני

הרי הרי הרי הרי / ענפכם ופריכם / תשאו לישראל בני.





אבי ראה / אני נכאה / מכל באי / מעוני



אכלו אכול / פרי אשכל / ונשכח כל / ששוני



שור כי לחצוני / וגם מחצוני / מידם פדני

בזכות הורי הורי הורי / במהרה היה עזרה / לבן מלך נשאר עני.







למה תזנח / אותי נאנח / כי אויבי נח / במכוני



עד אן גבור / עצמי תשבר / ובתחתית בור / שתני



מאז אהבתני / מבור מלטתני / עתה נטשתני

ביד ערי ערי ערי / חן עלי ועל עולי / בישועתך הראני.





חלץ אותי / מגלותי / בתפלתי / ענני



אצלי חנה / ביתי בנה / ותן מענה / בלשוני



שוב אל תוך משכני / ושם תניחני / ואתה תרעני

ובנה ערי ערי ערי / דוד, אל איום, חיש לי פדיום / בצדקתך נחני.
