


	
	[ארץ ארץ ארץ מה־תהמי עלי] (תוכחה)
	



Josef Carmi

[חי במודנה במחצית הראשונה של המאה הי"ז]





תוכחה (קטע)


ארץ ארץ ארץ מה־תהמי עלי / ותאחזי
בשולי / להסתירני



בסתר אהלך / ולשכב עוד אצלך / אם תרופפנה
עמודיך /



ותמוטנה יסודותיך / הן בך יד ההעדר הויה /
וכוסך ראש



רויה / לא ינקה כל בך נוגע / אם־חכם אם־משתגע /
אדמך



לאשת זנונים / יקרה היא מפנינים /הצמידים על־ידיה



וצרור המר בין שדיה / קוצותיה בזהב תעצר /
ושש



וארגמן תקשר / והנה תחתיה תעמד הבהרת / נגע
צרעת ממארת.



ארץ ככה רמיתיני / והכרע הכרעתיני / כי אך
לאויר העולם



הובאתי / מאתך נקראתי / ותעברי לפני / כערך
שני /



בכלים מכלים שונים / ורוח החיה באופנים / הן
למדתיני



מנעורי / לבלתי שמע לקול מורי / עד מפריך
ישביעוני /



ובנאות דשא ירעוני /וכי נפתולי אלהים נפתלתי
/ כי



בבית המדרש שתלתי / תתראי לי בערמה /
בבגדי־חפש



ורקמה / אז אתאו לאפוד בד ומעיל / ועד איננו לפי
צמיד



פתיל / אחרי־כן באסתירא בלגינא קיש קיש קריא /
מעה



ודינר ופונדיון / סלע דרכון ודרכמון / תשובי
אלי שלישיה /



ובלעדם נאלמתי דומיה / וכי שרשי פתוח אלי־מים
/ יתחיל



להציץ מיומים / אותו תשחיתי ותכריתי בהמצאותיך
/ כי



תקחיני בחלקלקותיך / לשוט בארץ ולהתהלך
בה / לארכה



ולרחבה / על־ההרים הרמים / ולרכב אניות בלב
ימים /



במדבר ובערבה / מדמשק עד־חובה / ודבר יי /
כספר



החתום לעיני / כי תאמרי "מה־בצע בדמך / כי
ימלאו



ימיך / ושכבת עם־אבותיך / וכל־יקר לא ראתה עינך
/ הלא



טוב לך / עודך בחור ורך / לאחז בזה וגם מזה /כבא
מועד



צאתך / תכבד ותשב בביתך / ותקח מפריו ותאכל"
/ וכי



באלה כף רגלי / לא נסו ומעגלי / הן תרהיבני
בנפשי עז /



לירד אל־גנת אגוז / רחם ורחמתים / ערב ובקר
וצהרים /



ושמה ארוה דדים / אתעלס באהבים / או ללכת
אל־משתה



היין / עד מותרי מן־הבהמה אין / או לחגר חרב על־



חלצים / קסדא ושריון ומגפים / או ללבש כתנת
פסים /



בגאה וגאון וחמסים / או בכנור שיר עגבים החביר /
ואש



זרה בלבבות הבעיר / ונוסף על־אלה כמה וכמה /
ארץ



רבת המהומה /אשר בם גדוד תגדדי / ונפש יקרה
תצודי.



נפשי נפשי אם הארורה הזאת / לאלפים ולרבבות /
חללים



הפילה / הן לא נתבוננה ונשכילה / הנאמרה שלום /
והנה



חלום / הלב נקשה / והנושה ישה / הניגע להעשיר /
והנה



מות בשיר / הנעלה אבר / ומחר בקבר / כי גם־בנו
הפילה



הגורל / ושתה גם־אתה והערל / הן אמרה בלבבה /
ותאסר



את־רכבה עם אף וחמה / לערך מלחמה / כי רשע רע



כבא יום נפלו / ארץ מתקוממה לו.
