


	
	ויהי היום יום
	



מתוך מלחמות העט והמספרים [המשורר מגלה שהעט עלול להפיג ממנו את השעמום.]









א.


ויהי היום יום צאת שלג וקרה, / יום
הקרח הנורא / על פני השדה נטה שפרירו / ויורד
רירו, / נס מפניו כל חם וכל שרב / ונפל ממנו רב –
/ כאמתים על פני כל הארץ, / ואין פורץ בעזו
פרץ. / הגה ברוחו הקשה / והפך ים ליבשה, /
ויחבל רחים, / ולא יכלו העם לשתות מים, / כי כל
מעין ונחל נקפא, / הגלדה טפה. / והשמש לא עצר
עמו כח / ויתגנב בהחבא לברח, / ישת חשך
סתרו / ומפרשי עב ילבש על בשרו, / יסכוהו
צאלים צללו / ויכסוהו בבגדים ולא יחם לו – / כי
הקרח הולך וחזק מאד, / לפני קרתו מי יעמד?



וכל העם אל הכלים נחבא – / אין יוצא
ואין בא, / ומעמים אין איש אתי / וגם מהסרים
למשמעתי, / אין רע ואין חבר / לספר עמי ולדבר,
/ כי הקר סגר עליהם / וכלם קופאים על שמריהם. /
ואביט – ואין איש, ואשתומם – ואין מפגיע, /
אמרתי: תסעדני ימיני וזרועי לי תושיע, / לבלתי
זבוב פרידת דודים ורע / שמן ששוני יבאיש, /
במקום שאין אנשים תשתדל להיות איש, / ומי
שיש לו עט ודיו – / מכל החברים דיו! / על בהן
ידו הימנית – / להב חנית, / לשון שנית, / אצבע
יתרה, / לו משפט הבכורה, / לו עזר כנגדו / ומטה
האלהים בידו. / הוא אחד השיחים / שפתים ישק
משיב דברים נכוחים, / ושמו למה יגרע / מעץ הדעת
טוב ורע? / כל אשר בו נוגע / טוב ורע כאלקים יודע.
/ ובמהלל הרימותי קולי / ואשא משלי:





סופר, בימינך להב אש / או להב חרב וחניתות?

עץ הדעת טוב ורע או / מטה תעשה בו את האותות?




ב.

נסיון הכתיבה מסתיים בכשלון






אז במסבי ישבתי / וקסתי במתני
שמתי / לכתב כחקי וכמשפטי, / ידי שלחתי
לעטי / לטבל אותו בדיו – / ויאמר אלי ״ואיו?״ / כי
ברוח הקרח נערמו מימיו, / קפאו בלב קסת תהומיו. /
ודפקתי שנית לטבלו בחזקה – / ושניו נתעוו
ויסעו מדפקה / ויכרע על ברכו – / והוא צולע על
ירכו! / ואני קצפתי על קסתי, / נטשתי את מגלתי
/ וכמעט בחרות אפי קרעתיה / ואת האח לפני
מבערת השלכתיה. / והעט ירד אל ירכתי קסתו /
וישכב על מטתו.

ובחמתי ואפי / לבלי חק פי / עליו
פערתי, / ומרב שיחי וכעסי דברתי / ״משלם רעה
תחת טובה, צא מן התבה! / למה נחבאת לברח, / מה
עזרת ללא כח? / הזה שכרי על כי כבדתיך / ומטה
האלהים קראתיך / ושבע ביום הללתיך / ונתתיך
עליון וראשון / על כל פה ולשון? / ואיה הגרם
ששברת / ואיפה הצור שבקעת? – / ולשון רכה
תשבר גרם / ואשר נטה ידו וירם / להכות
במטהו פעמים – / בקע צור ויזובו מים! / ואתה
שבת מפני נערת שרויה, / ומה תעשה אל זרוע
נטויה / ובצמר גפן נשכה שנך / ונס לחך, /
התרפית ביום צרה סר כחכה! / ותלנותי עלי נפשי
/ וחמסי עלי ראשי, / כי בחרתיך מכל אהובים ורעים
/ להיות לי יום יום שעשועים, / ומשברי הבישותני /
עתה ריקם שלחתני. / וככה יעשה למניח ברוש
וחוסה תחת הנעצוץ, / ואשען על משענת קנה
הרצוץ!





בחרתי עט הללתיהו / וזבדתיו זבד טוב זבוד,

חשבתי כי על כן בשכר / זה מאהבה אותי יעבד –

אך לשקר שמרתי, כי / טרחי עמו אבד אבוד.

כצרור אבן במרגמה / כן הנותן לכסיל כבוד!״





והעט קם בחמתו / מירכתי קסתו /
ויאמר ״שמעני, הודיעני, / על מה תריבני / עד
אשיחה במר רוחי: / אם כח אבנים כחי? / אם בשרי
נחוש? / ומי יאכל ומי יחוש? / קנה רצוץ רפה ידים – /
המן הסלע הזה נוציא לך מים? / אם אפצר עוד להיות
לי למוקש – / ומקרא מלא הוא ״צנים פחים בדרך
עקש״. / האם כף רגל עגל חשבתני / ביען לשתי
פרסות שסעתני / לשדד פני עמקים / כראי מוצק
חזקים? / או קראתני מטה האלהים / וכי דרכי מדרכי
כל לשון גבהים – / להכביד עלי משא והוה על הוה,
/ מקל וחמר ומגזרה שוה? / מקל וחמר – לשון גרם
לרצץ, / ומגזרה שוה – של מטה סלע לפוצץ. / וזה
לאות כי אתה מבקש עלי תואנה / או בפי הפסוק אין
לך הבנה, / וכל שער עמי יודע, / כי הנביא שהצור
בוקע – / לא בחיל המטה ולא בכחו, / כי אם ברוחו. /
וכן הלשון לא תשבר הגרם בעצמה, / כי אם
בחלקת נאומה, / שנאמר ״מענה רך משיב חמה״. / אף
כי אוי לי / אשר לא נסתה כף רגלי / הצג על הארץ
ודרך / מהתענג ומרך – / ותוחיל: כמטר לקחי
יערף / בהכות באבן או באגרוף! / ועוד כי חלשתי
אתה סבות – / ומדוע אלי ריבות? / כי עד דק
פסלתני / וכעצי עולה בקעתני / וכשסע הגדי
שסעתני. / זכר נא כי כחמר עשיתני / לצרך
הכתיבה / עד כי תשש כחי כנקבה. / ואף גם זאת:
בכחשי ורפיון מחי, / אתה ידעת כי בכל כחי /
עבדתיך בתם לבבי ונקיון כפי, / בזעת אפי, /
כעבד נרצע וחבוש / ערם הלכתי מבלי לבוש. /
רגלי מיחף לא מנעתי / ובמצולת הדיו טבעתי, /
כי שלחתני כל היום לבדק החר / לשתות מי
שיחור, / ואבואה במים / מי ברכים. / ובכל שוטי
ולכתי / הגידה לי מה משכרתי? / ומאשר חקותי
בספר / מיד מי לקחתי כפר? / וכתבתי לך שלישים
/ לבקרים חדשים, / מכל אשר לבך עבר / לא נפל
דבר. / אם ימלאו עטיני גשם דיו – על המגלה
אריקנו, / חיל בלעתי ואקיאנו, / ולא דבק בידי
מאומה – / ערם יצאתי מבטן קסתי וערם אשוב שמה /
ואם עתה עצרני הקרח מלדת, / הקשיא במקומה
עומדת. / ועתה אם לכתב נכספת, / למה חרפת /
ואותי גדפת? / הלא טוב לחלות פני אש ושביבו /
למוגג הדיו ולהדאיבו / ולהסיר לב האבן מקרבו. /
או כתר נא זעיר וחם השמש ונמס / ואשיתהו לרגלי
מרמס / והקרח מפזר ומפרד / ויעה ברד, / ומן
המים הנגרים במורד / עינות ותהומות יוצאות
בבקעה, / ושאבתם מים בששון ממעיני הישועה
/ להשקות המגלה ולכתבה / לארכה ולרחבה, /
ולעשות לך אות באות תבה בתבה.





היש קשור בלי חבל / ואם אסיר בלי כבל?

וכל רכב בלי מלוה / ורחים היש חובל?

ורק האש וחם שמש / יקו לקרא שנת יובל

ואז לעבר נהרות – שוק / יגלה, יחשף שבל.





זאת לא זאת: / אם לא תאמין לקול האות, /
הנה קטנך עבה ממתני / ומעיך חזקים משני /
וצמר הדיו רך לפי דבריך – / למען תצדק בדברך
/ טבל אתה באצבעותיך / ותראה מה יהיו
חלומותיך!״





הדבר מפיו יצא / ומהרתי במרוצה /
לטבל אצבע בדיו / ונקרע עור ובשר בקשיו /
וצפרן נעתק ממקומו / ונמצה דמו. / ובבשתי
רציתי לכסות חליי / והעט לועג לי / ונמלא שחוק
פיהו / ויאמר ״זה היום שקויתיהו!״ / ויקרא אלי
בלעג נאומו / ״אל תדין את חברך עד שתגיע
למקומו!״ / ואני בשתי ונכלמתי / ומשתומם
עמדתי / מעזות הקרח ומעשה תקפו, / כי גיד
ברזל ערפו / ומצחו נחושה / ומחמס העט תכסני
בושה. / ואמר: חטאתי הפעם / ודברי העט דברים של
טעם / ותודתי כמנחה ירח / ואהה, הקרח! / מי יאמין
לשמועתנו / שתמשש כל כלי ביתנו / ועל קן
הדיו תשטח חרמיך? / עד כאן, כפור, / יגיעו
רשמיך / לשום הנערת מצור חזקה / שלמה
ממשרקה.





אומר אמש ״מחר אכתב״ / כן כי אין לדבריו פדיום –

וכפור שמים ישחק לו, / ילעג קרח נורא אים:

אל תתהלל ביום מחר / כי לא תדע מה ילד יום!




ג

המספריים מציעים את עזרתם






והבאתי ידי לחיקי ונתעלפתי /
ובחלון ביתי בעד אשנבי נשקפתי / לראות דבר
מי יקום – מן הקרח או מן השמש / ואם יכהו חם
היום אל החמש. / ועבר חצי היום ומרביתו / והקרח
עומד בחזקתו / עודנו מחזיק בתמתו, / ואור
השמש – יקרות וקפאון, / מוצאי מים לצמאון, /
ועצר כל לח ונקשר כל מיץ / ויהי הקשר אמיץ. /
ואני יושב נכלם / עצוב ונאלם / על היותי לבדי /
בלי עזר חבר לנגדי / ועל עצר ידי וכלי, / ואשמע
את מדבר אלי / כתמה על מצבי / ״הברכה אחת היא
לך, אבי? / למה ידיך אתה יועט? / המבלי העט /
רקמת הכתב אין לרקם? / והרבה פתחים למקום! / ואם
יאורי הדיו חרבו ודללו / וקנה וסוף קמלו – / הלזה
חדלו פרזות בישראל חדלו? / ואם נצבו כמו נד
נוזלים ונקרשו / קר יעצר במים וייבשו – / הלא
תכרת עד תמוז עצים יבשים / ועשית את הקרשים. /
ואם דבק לשון קנה לחכו ודיו אין – הלזה תשכח
תורה / ואין נבואה שורה? / איה אפוא חרמיך, / איפה
טוב טעמך? / אם נשבר העץ ורצץ הקנה – / לבנים
נפלו וגזית נבנה, / ואם צנה גם קר מימי הדיו גוזל
/ ואין העט נוזל – / קח לך עט ברזל / יעז לך מעשרה
עטים / ועשה שטים / ותבות כפתוחי חותם מפתחות, /
מכתב אלהים חרות על הלוחות / עומד לנס עמים, /
והיו תואמים / בידך רצוא ושוב כמראה הבזק, /
איש את רעהו יעזרו ולאחיו יאמר ״חזק״. / רוכבים
צמודים / והיו לאחדים / בידך שנים – / הם הם
המספרים. / ברזל בברזל יחד, / טובים השנים מן
האחד / תחת העצים ברזל ימלא משאלך / ותגזר
אמר ויקם לך!״ / ויהי כאשר כלה לדבר אלי /
ויעל מעלי / אני ברב שמחתי / דלגתי בירכתי
ביתי / ואמרתי: זה היום יום בשורה / כי יש לעט
תמורה. / והמספרים השיבוני / הם הרהיבוני / ״יש
ויש, תנה ידך, / נס נא את עבדיך / וכתב וגם תוכל
כל / ולא נשוב מפני כל, / ולא יבוא לעזרתנו /
בלתי אם גויתנו״.




ד

סיום המשפט וההכרעה






[ויאמר העט]: ״ואם לא תאמינו כן כי לא
תאמנו, / נחלה פני אדונינו / לבל ישית ידו על
שנינו / ובמקומנו יניחנו, / ונבקש בשוק / אדם
עשוק. / אז ברצונו אל הבית נביאהו / ודבר מכל זה
לא נודיעהו, / וגם כל אחד ממנו ישבע / אשר לא
יורה באצבע / ולא יקרץ בעינים / ולא ימלל
ברגלים, / ונבלבל כל כלי האהלה, / עליונים
למטה ותחתונים למעלה, / ואנחנו בכללם, / ונאמר
לאיש הנעלם: / משש כלי הבית כלם, / תן להם
כפעלם! / ואשר את פיהו ימרה – / או סקול יסקל או
ירה יירה, / ונראה אם יכתב עמי או עמכם /
ויבחנו דבריכם״. / ויאמרו המספרים ״כן
דברת / ואמת אמרת, / בוא ועשה כן!״ / ויצא העט
אל רחוב העיר וימצא בו איש מסכן, / חכם בכל חכמה
/ ויודע דעת ומזמה. / ויגש אליו ויאמר ״בי
אדוני, / לכה אתי אל מקום פלוני, / ומי ימלל
גבורות חסדך / אשר תעשה את עבדך / כי תשבית
היום מדנים עצומים / ותחשך רציחה ושפיכות דמים״.
/ ויען האיש ויאמר ״צנים פחים אדרך / בלי נעל
ובלי שרוך, / אחליק פטיש ופעם אהלם / משום
דרכי שלום״.

ויביאהו העט אל אהלו / ויאמר לו /
״אדוני, כטוב בעיניך / הנה הבית לפניך, /
לארכו ולרחבו, / שים עיניך בו / ובכל הכלים /
הקטנים עם הגדולים, / ושים כל כלי על כנו / את
כל עורב למינו״. / ויבוא האיש מחזיק בפלכו /
ויפן כה וכה / וירא צפחת בשמן זית / ומנורה
בירכתי הבית / ויעש פתילותיה / והעלה את
נרותיה. / וימצא מחט ובגדיו היו קרועים / ויתפר
בו את כל הקרעים. / ועוד נתן את עיניו כמרגלים /
וירא בתוך הכלים עט ירקרק כברק מזהיר, / והנה
הוא היה סופר מהיר, / וימהר להוציאו מביניהם /
ויכתב בספר לעיניהם. / ויהי ממשש עד צהרים /
וימצא המספרים, / ויעש בהם צפרני הידים /
ויגלח שערו ועשה שפמו. / והעט קם ממקומו /
ולא יכל להתאפק, / כי לבו דופק, / ויקרא בקול רם
ויאמר ״די!״ / ואל הזקנים אמר ״אתם עדי!״ / ויחזק
באיש וישקהו ויאמר ״טרחת בעדנו כמה וכמה
/ תהי משכרתך שלמה!״ / והתחיל בשמחה מפזז
ומכרכר / מספח חמתו ואף שכר, / יזיל כטל
אמרים / ויכתב בשעה ההיא מאה ספרים, / כלפי
המספרים דברים יטיח / יריע אף יצריח / ועליהם
יתגבר. / ולא יכלו לענות אותו דבר, / כי נבהלו
מפניו ונאלמו / בושו ונכלמו, / כי העט שת להם
מורא, / אין קול ענות גבורה / ואין קול ענות חלושה /
והיתה הפצירה פים למחרישה.



ויעש העט משתה גדול לכל אוהביו /
ביום הציל יי אותו מכף כל אויביו. / ויקומו כל
עדת הקנה / ויעבירו קול במחנה / ״מטעם המלך
וגדוליו: / כל אדם יפקד כליו, / כל כלי כאשר
פעל יי למענהו / ולא ישלח ידו במלאכת רעהו, /
פעל פעל מתחלת בריתו / לא יחליפנו ולא ימיר
אותו, / לא יהיה כלי גבר על אשה / ולא ילבש גבר
שמלת אשה. / העט לכתב והמספרים לגלח, / ואש
על המזבח, / חרמש לקצר ומתג לבלם – / וכל העם על
מקומו יבוא בשלום״. / והקול נשמע בכל פנה /
ותעבר הרנה / וכל כלי לעברו פנה / וסדר חקו
ומנהגו לא שנה / ותשקט הארץ ארבעים שנה.




מקור: שירמן – השירה העברית בספרד ובפרובאנס; הוצאת מוסד ביאליק ודביר תשי״ז תשכ״א
