


	
	פליאה דעת העט
	




תהלת העט





פליאה דעת העט ממני, זר מעשהו / את מי נועץ ויבינהו / מחשבות לבי בלי הגידי אותם / יראה יעשה כתבניתם / בלי אזנים יקשיב בכל רזה / ובלתי עינים כל צפון חוזה / מזה בן מזה יזה / נחל נובע מקור חכמה / לתת לפתאים ערמה / יערף כמטר לקחו / ולפני אדניו ישפך שיחו / לרוץ לפניו צו לצו / יאזר כגבר חלציו / אשאלנו ויודיעני / ביום קראתי ויענני.







עט בלי לב יבין לחשי / ובלי אזן ישמע רחשי

יאזר מתניו לרוץ ארח / בשליחותי כרצון נפשי

כל אות יפגש ישליך אחרי / גבו ישלח אותו חפשי

דרך ככר הגליון / רץ ויעבר את הכושי.







לכן העט בחיר דודים ורעים / בן יקיר ילד שעשועים / מדי דברי בו זכר יזכירני
הנשכחות / ויקבץ הנדחות / מארבע רוחות / ויתנם במשמר לבלתי ישתכחו / בדור אחר פן ימחו / כי למן העת יכתבון מלין בספר ויחקו / כרגבים ידבקו / לא נדים משם ולא נעים / כמסמרות נטועים / שם ישמר עט מוצא שפתו / לא יחליפנו ולא ימיר אותו / לעולם דברו נצב במשמרת / לאות ולמזכרת / יכין צעדו בהצנע לכת / יש רגל מברכת / כל המקום אשר תדרך כף רגלו / בנעימים נפל חבלו / גליון מצרע כשלג בעברו בו מצרעתו יאספהו / מנחל עדניו ישקהו / ישלח מרגלים אל השטים לחרש לאמר וקו אל קו אל המגלה רוקע / והוא מדבר על לבו, רק שפתיו נעות וקולו לא ישמע / ויחשיבוהו לשכרה / בשובו לשאב פעם ועשרה / והוא באהבה מכלכל את השורה / אוכל ואין לשבעה שותה ואין לשכרה / רק להשקות הספר ולהלעיטו / כבולעו כך פולטו / ומאומה לא ישא בעמלו / ישלם ה׳ פעלו / ועוד מה ארחיב פה בתהלותיו / ואאריך לשון בחמודותיו / ודוד מלכנו ברוב רצונו / להפליג במהלל לשונו / עט סופר מהיר יקראנו / לעשות לו יקר יותר ממנו / כי כן דרכם של בני אדם / ומנהג בשר ודם / כל רוצה להגדיל כל בריה / והחפץ להלל תבנית כל חמדת שכיה / לא יערכנה / כי אם לרעותה הטובה ממנה.
