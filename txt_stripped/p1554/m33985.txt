


	
	והשמועה באה
	



מתוך: צלצל כנפים או אוהב נשים. אבן־שבתי יורד ארצה מגן עדן.










והשמועה באה עד יהודה בן־שבתי בין
צבאות נגהים / בעדן גן אלהים – / הוא יהודה הידוע
בין האנשים / שונא הנשים. / ויתאפק האיש
בשמעו מלי / ויעף אלי, / ויחרד ושערת ראשו
תסמר, / ויטש סכתו בגן על ידי שומר / ויפקד
שר על מנעוליו ובריחיו, / וירד יהודה מאת אחיו. /
ויתיצב על אם הדרך לדבר אלי – / ואני
התחזקתי כיד אלהי הטובה עלי. / ויתכסה באפר
על עיניו / וקסת הסופר במתניו, / שוקיו אני
תרשיש, גויתו כתרשיש, / והוא חגור חרב כמראה
חרב בני עמון / וקול דבריו כקול המון. / פתח
בבדיחותא / לשנא דצחותא, / מתוקות מדבש
מדברותיו, / נפת תטפנה שפתותיו. / ואתבונן
אליו ואכירהו / כי מבני הכסילים הוא, / ואמר
״השמיעני, אדוני, ספון סודך, כי שומע
עבדך״.

ויאמר יהודה בן־שבתי ״במה סברת
להחריד יחידתי? / למה הרגזתני להעלות אותי? /
אשר חפצתי עזבת / אשר שנאתי אהבת – / ואולם
עתה הריב נטש ובקש שלום, / כי הביאותני עד הלם!״
/ ואען ואמר ״בי אדוני, אם כיום הפרתי כבודך – /
אל יחר אפך בעבדך, / כי לא דמיתי להשיב אל
אדוני עזות, / בתם לבבי ובנקיון כפי עשיתי זאת!
/ כי הנשים בשכלם – / יסוד עולם, / והמון היעלות
היפיפיות – / מגדל עז בנוי לתלפיות. / ותאמר,
אדוני, למולידים בצלמם ׳לא תרבו אשם!׳ / כמעט
שכב אחד העם את אשתו והבאת עלינו אשם. / ותאמר
לשומר התולדות ׳נפשך טבחת! / הנך מת על האשה
אשר לקחת׳. / חלילה לאדוני אם לשד האילות תשחק
– / מדבר שקר תרחק! / חלילה לך מכתב להמון
היעלות גט – / והיה בך חטא! / ואולם עד כה רעיוני
יועצים: / נחנו נעבר חלוצים / לפני שני בני
הפרתמים / הנאהבים והנעימים, / נושאי צנה, /
יודעי בינה, / אנשים אחים, / טובים ונכוחים, / תפארת
הסגלות, / עטרת התהלות, / אשר נפשם בקהל
המעלה והמדע תחד – / תורה וגדלה במקום אחד. /
התהלה נמשכת עמם / ותהי המשרה על שכמם, /
כל אחד דעה יורה / השרידים אשר יי קורא, / אשר
במזור חמדתן ירפאו כל נגע ופשע, / אנשי אמת
שונאי בצע, / ראשי הפנות, / שרי המדינות, / כל
אחד יודע צפונות / נגיד ומצוה ומחזה שדי יחזה, /
היושבים בארלדי בגלילות פרווינצא במקצוע
הראשון מקצה מזה. / לפניהם ירן רחשי / בסודם אל
תבוא נפשי./ עצתם תנחנו כרצון שוכני סנה
וכוכבי נגהים – / כאשר ישאל איש בדבר אלהים, /
אשר לבבי לחזות בנעמם נחפז – / שני טורי שהם
הממלאים בפז.



הטור הראשון – / מאיר בן־שלומון, /
העומד בפרץ / המאיר לארץ, / מר ואהלים, / יושב
אהלים, / עוטה אפוד בד, / והאיש נכבד / רענן כגן
רטב / הולך וגדל וטוב. / נשאל כיום את פיו
להישירנו / והגיד לנו את דרכינו, / בעצתו
הטובה ידע צפון הנולד וחבו – / כל אשר ידבר בוא
יבוא.

הטור השני – יהודה אחיו בן־שלמה אשר
מן החמודות אליו יתלקטו סביב מחנהו / והאיש גדול
ומעשהו. / איש תאר ויי עמו, ולו בחמודות – / עשר
ידות, / מסלת לבבו לידידים רחבה / תוכו רצוף אהבה,
/ בחצרות לבבו וטירתו ינחם, / לא ישקר ולא
ינחם. / וכמוהו מנעול סתר תשוקתי והודי / קמתי
אני לפתח לדודי, / זכרו אדון לרעיוני שמתיו /
ונגיד לי נתתיו. / הוא אשר מצאתיו יערי ודבשי – /
את שאהבה נפשי, / איש אמת אשר החמס זעם, / כי אין
כמוהו בכל העם. / אשר לבותינו לא ימצאו
ברעיוניהם / ישפטו הם, / כי המה המחזיקים כל
בדק / ושפטו את העם משפט צדק״.

ויאמר יהודה בן־שבתי ״טוב דברת!
השופטים יאירו רעיונינו הבאים / והבאת אתה את
הדברים אל האלהים״. / ונבוא יחדו אל ביתם /
בחצריהם ובטירותם / ונמצאם בעלית המקרה פת
התבונות סועדים, / יושבים איש על כסאו מלבשים
בגדים. / וירוצו לעמתנו / וישמחו לקראתנו, /
ואמר אל יהודה בן־שבתי ״דבר אתה לפני כל המוני,
/ דרש נא כיום את דבר יי!״

וידבר יהודה בן־שבתי לפניהם ויאמר
״אמרתי: לפני אל ידרכו אנשים / דרך נשים! /
ואמר לכל נושא אשה בנעם משלי: / הפרד נא מעלי!
/ ולכל עוזבה ולכל המוני: / בוא, ברוך יי! / כי
הנה נמבזות ונקלות, ואם טובות
מראיהן – / נשים דעתן קלה עליהן. / והאיש הזה
טוביה החליש חסני / השיב אחור ימיני – / יגיד לכם
כיום מחברתו אשר הציב להאדיבי / וספר כתב
איש ריבי״. / ויענו שני האצילים, / שני המאורות
הגדולים, / ויאמרו ״שמע נא, יהודה המליץ הגדול!
אל תמעל באילות מעל, / אל תתן את האשה לפני
בת בליעל – / והנה בימי נח ירדו בני האלהים
להעמידם בסודם, / אף כי שוכני בתי חמר אשר בעפר
יסודם, / כי היא מכבוד האלהים נאצלת / ומיי אשה
משכלת, / הוד והדר נסוכה, / חלק משמן חכה. /
וליל נבצר מן האיש יעלה – נבצר ממנו מגדל עפל,
/ הלילה ההוא יקחהו אפל! / על כן חדל לך וחתם
דבריך, / אל תתן את פיך לחטיא את בשרך! /
וטוביה הפליא לעשות בנעמו / והדין עמו. / ואולם
לכבודך יחלק המשפט לחצי, ולבעבור ספרך חיה
יחיה – / גם לו גם לך לא יהיה! / ושמעו הדין אשר
אמת יסובבהו, / כי המשפט לאלהים הוא: / יביא כל
איש עפרה אל ביתו / לעמד לפניו לשרתו / והעמיד
ממנה סגנים וסגנות / וילדה לו בנים או בנות, / כי
לא כל רע יגורהו / אם בעל אשה הוא. / כל ימי חיותה
יתננה אצעדה וצמיד / באהבתה ישגה תמיד, /
וביום מותה יספד חמדתה – / וישמר מקחת
תמורתה / ומתת מלכותה לרעותה. / ואם המות
העבירה / ויסירה מגבירה, / ישמר את ילדיה חסדו
ואמתו / כאשר היתה באמנה אתו, / כי כל בריתה
יפר במעלו, / אם אחרת יקח לו. / ולא יתכן
שישכח ראשונה ודדיה / ויצאו ילדיה / וינשא
בן השנית ורגל הראשון יתן למוט – / האחד בא לגור
וישפט שפוט! / וכה דברה לנו בדעת וחכמה /
תורת יי תמימה – / אשר בנחלתו בנו הגדול יזכר
/ ואת בן השנואה יכיר כי הוא הבכור: / כי איככה
בעין לבבו ירצה לטל מזה / ולתת לזה, / אף כי
הגדול עומד בפרץ / לו שם כשם הגדולים אשר
בארץ! / והקטן לא נסה בתהלות רוחו להעיר / אשר
לא ידע ללכת אל עיר. / על כן אמרנו: תעמד עפרה
בגבולו / אחת, לא ישנה לו / ולבד עינו בנו הגדול
בנחלתו יפקח / גם את הכל יקח״. / ויען להם
יהודה בן־שבתי כאיש נבר / ויאמר ״טוב הדבר!״ /
וישב לקראתו ויפרש כנפים / על פני רקיע
השמים / ומלאכי מחנים אותו נושאים / וטוביה
ומאיר ויהודה רואים / ויאמרו ״נהלל הבורא שוכן
ערץ / כי ראינו אלהים עולים מן הארץ״.







מקור: שירמן – השירה העברית בספרד ובפרובאנס; הוצאת מוסד ביאליק ודביר תשי״ז תשכ״א
