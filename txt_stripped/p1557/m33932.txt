


	
	מקרי הזמן רדפוני
	




מתוך ספר עזר הדת. ראשית הוויכוח בין התורני והפילוסוף.





מקרי הזמן רדפוני והדפוני / וכמטרה
לחצים תולדות הימים שמוני. / והוציאוני ממקומי
והרחיקוני / מאנשי עמי / ונוה הדומי, / ותשאני
רוח על כנפים / ותשליכני עיר ירושלים. / ובעודי
מהלך בדרכיה / ומסובב את מעגליה / לבקש ספקי
וארחתי / או נדיב מסיר אנחתי, / ובבת עיני תנהר /
כעוף המשוטט סביבות הנהר – / נשאתי את עיני וארא
והנה להקת אנשים יושבים / וברחוב העיר נצבים, /
כעגלה המסובבת סביב הירח / או כקלפת הפרי
הפורח, / ובתוכם נשמע קול הברה / קול המלה
נעכרה, / קול ענות גבורה.



ואקרב להתיצב עליהם / לראות מה
מעשיהם, / והנה בתוכם שני אנשים נצים, /
מתקוטטים ומתלחצים, / שבים ורצים, / כל אחד לנצח
לאחר אצים. / האחד זקן ונשוא פנים / בעל שיבה ובא
בשנים, / עד אשר סרו ממנו שני המעדנים. /
מראהו נהדר ומתקן, / בעל כרס וזקן, / גבהה מאד
קומתו, / מעטף בטליתו, / בכבד ראש כפוף /
כאגמון יכף. / והשני נער נאה / יפה תאר ויפה
מראה, / כל רואה עליו ישתאה, / עיניו כיונים /
לחייו שושנים / ושפתותיו ששונים.





וישא הזקן את קולו ויאמר ״שמעו
אלי, עמים, / והטו אזנכם, לאמים! / אשמיעכם מה
שנסיתי בימי חרפי / האזינו, השמים, ואדברה,
ותשמע הארץ אמרי פי. / שמעו אלי, כל אנשי
הקהלה, / כי לא תדברנה שפתי עולה / הן כל אלה
שמעה אזני ותבן לה. / האל אשר מבין האמות
בחרנו / וברחמיו גאלנו / מחשך הוציאנו / וברב
חסדיו על צרינו המשילנו – / תורה תמימה נתן לנו /
וחקים ישרים העניקנו / להדריכנו להישירנו /
בכל מעשה ידינו, / וגם להיותה סבה / לקנות בה
חיי עולם הבא. / וברצותו לזכותנו / הרבה
בחקותינו ובמצוותינו / להיטיב אותנו באחריתנו. /
ועתה הנה עם יצא ממצרים / ערלי שפתים, /
מבקשים לפרש רשתם עלינו / לבטל את אמונתנו /
ולהפר עצתנו / לבשתנו ולחרפתנו, / להבאיש את
ריחנו / בין מבקשי רעתנו, / להאריך את גלותנו /
ולהכביד את עלנו. / בחקותינו הישרים שוגים /
ובחכמינו מלעיגים, / לשונם נכרית / מואבית והגרית,
/ דעותיהם משנים / וספריהם חיצונים, / מחברים
ולקוטים מהידיעה הזרה החיצונית / היא החכמה
היונית / אשר האפיקורסין חברוה / והמינין
ילמדוה / והכופרים ידעוה, / והיו לנו לשטנה /
לעקר ולסתר יסודות האמונה. / כל היום עלינו צרים
וצוררים / מבישים ומחפירים, / גוברים עלינו
ומשתררים / דולקים אחרינו ורודפים – / הם
הנקראים פילוסופים. / וכל איש חסיד אליהם לא
יתרועע, / חכמים המה להרע. / והנה הנער הזה עצת
זקנים פרע, / כי בא אחריהם / והולך בעקבותיהם /
ומתעסק לקרא בספריהם / וממלא את דבריהם /
ושואב בכלי מימיהם / ונוהג במנהגי שגעוניהם, /
יסית וידיח / יריע אף יצריח / יבאיש יסריח, /
גלי שאון דתנו ישביח / וכל אשר יעשה לא
יצליח. / ועתה לכו ונהרגהו / ובאבנים נרגמהו /
או באש נשרפהו. / וישא משלו ויאמר:





ראו נער אשר דרכו להשחית / וכפר חק אשר שם אל והברית

וידיח מתי מוסר ויסית / להאמין שוא בבן מואב והגרית.

לזאת התרחקו מנו, אמונים, / והזכו והתרחצו בברית

ורגום תרגמוהו באבנים / לבל ירא וישאר לו שארית

ושרוף תשרפו תוך אש וגפרית / וישכח שמו ויהי להכרית.
