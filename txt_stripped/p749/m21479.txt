


	
	פורים
	




ובארבעה עשר לחדש אדר / בחורי ישראל לכבוד
ולהדר / יתפארו ויתהללו / כי ישתגעו וכי
יתהוללו. / זה זכר עשה לנפלאות ופרסום לנסים /
כי יעטה כמעיל תפארת העכסים / וזה לכבוד מרדכי
ואסתר / סחיף דקולא ארישה וכופה עליו פסכתר. /
זה ילבש שמלת אשה ולגרגרותיו ענקים, / וזה
יתחקה כאחד הריקים, / תף ומחול שמחה ושלישים, /
אלו עם אלו אנשים עם נשים. / ולעת ערב סעודת
פורים / איל וצבי ויחמור וברבורים, / משלוח מנות
איש לרעהו משבעת המינים – / אבל מתנות לאביונים
מתמעטות בכל השנים!



בלילה ההוא יתנו רוזנים לאין / הבאים לחקור
ממסך השותים במזרקי יין. / אז ירננו אוהבי
אשישי ענבים / יזהירו כזהר הרקיע וככוכבים, /
ונכרת זכר בית הרכבים. / בה בליליא יחפש בית
היין בנרות / ונשתו מים מן הבתים ומן החצרות,
/ גזרה היא מלפני חמת אלהי ספרוים: / בלילה
ההוא אל תאמרו: “מים,מים!” / השותה היין חי – אור
כשלמה עוטה, / ואשר אחר המים נוטה – / כמדקרות
חרב בוטה. / ראוי לברח מהם כמי שנשכו כלב
שוטה, / הקרב הקרב אליהם דמו בראשו, / הנוגע
בם הרי זה מתחיב בנפשו. / לכן התקינו הקדמונים
/ ויעצו חסידים הראשונים / שמי שהניח לו אביו
מעות / ורוצה להרויח בם יותר מכל עסק וקרקעות – /
וקנה מהם יין חשוב ומפלא / באדר יעשה והצליח
דבריא מזלה. / כל אותו הלילה מספרים כבוד יין
ושכר / ודורשים מהם דרוש וקבל שכר.



בעדנא ההיא קרבו פקדות כל חבית וקנקן / מלא
וריקן – / והריק לחרפות מנוה יגרש / והמלא יסיעו
וגם דמו הנה נדרש. / ישתו וימצו / אין אונס
ומעכב אבל לכשירצו, / אדמים שרקים ולבנים, /
חדשים גם ישנים./ זה ישתה עד די אפוהי ישתנא
/ וזה עד דנפיק קיוהא מאונה, / זה ישתה בגביע
גדול יין חי ומלא / עד שתחסר הצפחת והכד
יכלה, / וזה למדת בטנו עד דקפי טחלה. / כל פתי
וכל חסר מח / שורה בני מעיו ביין, עד אם יהיה
ברזל – ירך, ואם אבן – הוא נמוח. / ואליהם ילוו
בשמים וזנים משבחים / ומגדלות מרקחים.



כללו של דבר: / הגבור – גבור לשתות יין
ודאלים גבר, / המשתכר – משתכר עד שלא יכיר
בין נבל לנבר / ובין עתיד לעבר, / עד שלא ידע בין
המן למהומן / ובין “ברוך מרדכי” ל“ארור המן”, /
ואשר ישכר בתוך האהלו / עד שלא ידע בין ימינו
לשמאלו – / מה טוב חלקו ומה נעים גורלו! / חק
לישראל הוא להשתכר מאדם כשנים ומלבן כצמר
צחר / ליל ארבעה עשר באדר עד שיעלה עמוד
השחר, / ולהפיק רצון נורא ואים, / ינתן גם מחר
ליהודים לעשות כדת היום! / כי כדאי הוא החדש /
להוסיף מחל על הקדש /



ואף כי טוב ביום ההוא לזנק כאריה מן הבשן /
לכבוד היהודים אשר בשושן. /



אשריך ישראל, מי כמוך ומי לך דומה / לקים מצות
“מחיב אנש לבסומי”! /
