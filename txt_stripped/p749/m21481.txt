


	
	הרופא
	




ויש שיתפאר בחכמת הרפואות, / יספר על עצמו
לעשות נפלאות / וכי עמו סודות בקירות לבו חרותים
/ להחזיר נשמות לפגרים מתים. / יאמר ״יש לי ספר
רפואות הגנוז / וספרי אבוקרט וגלינוס / וספרי
שאר החכמים וסודותיהם / הבאים אחריהם, / הכל
בחדרי הכינותי / והכל קראתי והבינותי. / כמה
מלאכות נערכות עשיתי בזמן / ימה וקדמה צפון
ותימן! / רפאתי אנשים משחין חזקיהו וצרעת
נעמן״ – ואותו המזיף מכל אלה יתום ואלמן. /
והמון הסכלים בראותם טבעת בידו ועליו בגד
ארגמן / יחשבוהו רופא ממחה ונאמן, / יאמרו
שארבע עשרה שנה נטמן / בבית עבר הגלעדי ראש
רופאי גלעד, / מפרי שפתיו לבב אנוש יסעד / הרבה
למד מרבותיו וחכמה גדולה קנה, / מפליא לעשות אחר
היאוש במה שאין בו סכנה – / סכנת עלוקה – הב הב
/ דינרי זהב – / תנה את המנה!



שמעו, עמים כלם, כמה יתנהג זה באמונה
במלאכתו! / כשיקרב לחולה ביום מפלתו / יאמר
שהוא מסכן ממרסא גדולה במח תחת גלגלתו /
״בינותי יפה ענינו מתוך דפקי הידים / ועביט של
מימי רגלים, / אותותינו אותות ידועים לא מאתמול
ומאמש – / ברור לי הדבר כשמש! / ועתה הבו
שכרי חמשים כסף, / ואם לא – ואדעה מה חדל אני, /
מה אני בשכר אתם גם אתם, כי לא בחנם מלמדני
הורני. / עשו אתם ראשונה פרעון חשוב / והנשאר –
אחר אשר לבריאותו ישוב. / בזאת אאות לכם להתעסק
בו, / אשיבנו רענן באבו / והסירותי מחלה
מקרבו.״ / כללו של דבר: אין לו בשת / לבקש שכר
גדול על המשקה ועל התחבשת, / בחלקלקות
יבטיח לעשות מרקחת / להתיך המרסא ולהסיר
הקדחת – / והעניים גואלי החולה אשר ברפואתו
ירצו / כסף לא יחשבו וזהב לא יחפצו, / ככל
היוצא מפיו יעשו, / מה שגמר אמר – / ימדו בעמר.



והרופא אחר שמעות משך והכל בכיסו שם / יקרא
אל הבשם / לאמר ״כתב זאת זכרון בספר: / קח לך
סמים – נטף ושחלת הבאים מארץ חפר, / ושחקת הדק
כעפר ואפר, / תערב עמהם קמוש וחוח / ועפר
תחוח, / שרף ערלה ושרש רתמים, / עלי זית ודם
צפרת כרמים, / קלפת צנונות / ומיץ חלב מעשר
אתונות, / פרח שושן וזרע מרעים ואבק לשון הרע – /
לא תוסיף על אלה ולא תגרע! – / תרדין לרפואה זרע
וירק / ותשים הכל במזרק. / תחבר אליהם שבוש
הסברא / ומעט קול הברה, / ותעביר הכל בכברה, /
בד בבד יהיה – / בטחוני בקוני בזאת ההרכבה
מחלי זה חיה יחיה. / עשה ממנו רטיה / והניחה
לו כנגד הלב מקום שהנשמה תלויה – / מיד יסור
כאבו ועין בעין תחזה, / למחר יהיה האות הזה! /
והמרקחת שעשיתי לצירו – מלבי יסדתיו, / אני
היום ילדתיו. / זה תארו: קח אזוב יון ואזוב כחלי /
שיש בו סגלה לכל חלי, / שרוך המנעל / ושן של
שועל / יתדות האהל וסכתא בגדא – / מכל אחד
אגדה. / מיץ אבן שיש, / צפרני ליש, / ראש שבלת /
וחלב תרנגלת, / עצי הכלוב, / דדלוז ודדלוב /
ומסמר הצלוב, / אבוב רעיא / זרקא תביר וגעיא, /
כמו שכתוב בספר שגנז חזקיה – / מכל אחד
אוקיא. / קלפי רמונים / ועקצי תאנים, / אבן תקומה
/ ודמדומי חמה, / ברק חנית / וסלע שעל גבי
הצנית, / קדה וכמון / ומי רמון – / מכל אחד
דרכמון. / קלוף חור די על חטרא, / צל דפנות ענן
בקר זקא דבתר מטרא – / מכל אחד חצי לטרא. / הדק
היטב הסמנים הקשים מפה ומפה / וערב בהם
כראוי דבש צרעה שהוסר קצפו, / עשה ממנו
מרקחת מפלא כמו שתוכל / והבאת לחולה ואכל. /
ואיש אשר ירקח כמוהו לאכלה / לא ידע שכול ולא
מליף תכלא. / ואולם המשקה / אשר בו מכל רע
ינקה – / גרוגרות וצמוקים / מחרצניהם מנקים /
עסיס רמוני בדן וקלפותיו / ועלי כרוב ורוביא כדי
שירבו זכיותיו / חלב המנעול ודמו / ודגים
קטנים מהים שעשה שלמה, / לגלונות הבאות מארץ
תימא, / יותרת הכבד ומורני בהמה. / מכל אלה חלק
בחלק, שוה בשוה, / ותבשל הכל עד שיהיה מעט
עבה. / ישתה אותו בערב מזוג במי אפסים, מי שחו –
/ מריחם יפריח שדה, ישגה אחו, / ויתן אחריו שנה
לעינו / וישוב לפנות בקר לאיתנו״.



והיה ככלות הרופא דבריו / ותחבולותיו ושקריו, /
ישוב לביתו שמח, ישתה ויאכל – / והאנשים קרובי
החולה ימהרו אל בית הרוכל / להביא הרפואות / וכל
אותן הנסחאות. / ולתועלת אשר יקוו / יעשו כל מה
שנצטוו. / בלב יחשבו, ביד יקחו – / לא
יעברו ממצוותיו ולא ישכחו. / כל אחד ואחד
בזריזות יעבדנו – / ועם כל זה בחצי הלילה
המות יתפשנו, / רוח עברה בו ואיננו! / והרופא
על זה לא יקפיד אחר שנפרע, / לא יבכה לו וקרוע
לא יקרע, / לא יעטה על שפם וראשו לא יפרע / לא
ינהג אבלו לא שבעה ולא שלשים, / אף על פי שלקח
מממונו קדם כל היורשים!



כן דרך רופאי זמננו: עיניהם בממון ולא יחושו
למה שיהיה, / אם ימות גבר היחיה, / לא יחושו אם
בעתם ימותו כמה עניים / ואם תהיה נפשם צרורה
בצרור נקוב או בצרור החיים. / ורעה מזאת – יצעק
חמס זה המזיף על הקרובים והידידים / לאמר ״אתם
המתם את האיש בדברים המתנגדים! / אותו בידכם
שויתי – / ואתם מריתם פי בכל אשר צויתי. /
כלכם פשעתם בו בשומכם מצותי לאין / ותשקו
את החולה יין״. / סוף דבר – כלם טופלי שקר, רופאי
אליל, יי לא ייראו, / ידיהם דמים מלאו. / על כן
אמרתי: רבותינו הקדושים לא אמרו בחנם / ״טוב
שברופאים לגיהנם!״
