


	
	[צבי נעים נתנו]
	




U – – – U – – – U //– – U – – – U – – – U – –



צבי נעים, נתנו אל ברכה // בארצו, העלני משבכה

ורוני צדקה מלשונך, // כצפיחית ביין הטוב מסוכה.

היש יתרון לך כי תך לבבות // באור פנים ושער כחשכה,

ובת־עין שחרחרת כליל // עלי לחי אדמדמת, כבוכה?

ואיך תעש מלאכתך בקרבים // ובלבבות – ולא תדע מלאכה!

ותגבר על גבירים, לא בנשק // ובשלחים ולא במערכה,

ותרפא האנושים מבלי שית // צרי, או שום עלי מכה ארוכה!

הבינני, היש תכלה לנודך, // ועד מתי? ואיככה ואיכה

תהי עומד בתוך רעים, ותורם // בחצך וקשתך הדרוכה?

ואיך תבחר לצדיקים תמותה – // וחייהם ומיתתם בידכה!

ותעלץ ברעתם כאויב – // ולמה יעשה את זאת כמותכה?

ידידי, תשמעו שירי, ונפשי, // כמו תדעו, ליראת אל סמוכה,

ולו פשר כפשר שיר שלמה // ב’דודי צח' ו’עין כברכה'.

שמע, אדון, משרת את עבדיו, // כמו שמש לשרתנו ערוכה,

אשר שמע שמו ירח לרחוק // כמו על אש, ושמן טוב בסיכה,

תצוני להשלים את אנשים // מצאתימו כמו עוגה הפוכה,

מכרוני ביום רעה כיוסף // בלי עוון, ואולם בעבורכה,

ודמו להניאני ולדוך // יחידתי, כנקלה, במדוכה

ולי לבה, כמו תדע, יקרה, // ונפש לא לבזיון נסוכה!

אמר להם: הלנצח מריבה, // ואם איבה לעולמים משוכה,

ואם שטנה – התלד עד זקונים, // ובת שלום – המלדת חשוכה?

אני אעש כמצותך, למען // דבריך בנפשי כהלכה

ועל ידי אהי נושק בעברם, // ואולם אל תהי להם נשיכה

ואם באו בצל ביתי – אכבד // יצועם, ואכסם בשמיכה,

ומקדם אשרת הגבירים // בתוך ביתי ואין נפשי נמוכה,

ואחשב את עבודתי לנגדם, // בהשקותם והברותם, מלוכה.
