


	
	[ידידיך בך ירבון]
	



ולה גואב אלי נסים זקל

ולו תשובה אל רב נסים זכר קדוש לברכה






U – – – U – – – U //– – U – – – U – – – U – –



ידידיך בך ירבון ששונים, // וצריך יניקון ראש פתנים,

ותורתך שמחה בך, ושמך // במרחקים כמו ריח שמנים.

לך הטוב, לך ישר, לך תם, // לך המה במתנה נתונים,

ומיום צאת שמך הטוב בבבל // קראוך בשם ראש המבינים.

לך משרה, לך כתר תהלה // ותפארת, לעולמים נכונים,

ולך נתנו בני עירך שררה, // כמו נתנו להורך, איש אמונים,

זקן יראה, חבר תורה ומוסר, // ישיש תלמוד, שארית הנבונים,

והנכר בעושי הגדילים, // והנכבד בעוטי הסדינים.

ובראותם באפריקי כבודו // חשבוהו כאחד הגאונים,

וחלוהו, כמו חלו אנשים, // בדור הלך, פני שבעים זקנים!

ויש משרה בלי אדן – והיתה // שררתו יסודה באדנים,

ובנית לך בנין יקר על // יסודותיו, ולא בנין אבנים.

ויש בונה לנפשו בית גדלה // בלא חמר אדמה או לבנים,

כבנינך, אשר בתהום יסודו // ותקרתו עלי ראשי עננים.

אהבתיך בטרם אחזה את // כתביך, וחי שוכן מעונים!

והוסף אז לך אהב בלבי // עשרה מאשר היה לפנים.

תהלתך גברת לתהלות // ומכתבה לכל מכתב אדונים,

מאזנת במשקל ישמעאלים // ומוכנת בחכמת היונים,

ובה מלים כאלו הם כלילים, // וענינים כאלו הם פנינים,

ואיך אשיב גמול שירך, בשיחים // כמו שני, עלי שיחים לבנים?

ומי יצא להלחם, במקל // בידו, עם מתי חצים שנונים?

קחה, חלף שבחיך, לבבי // במתן עם כסלי הצפונים,

ושירה, כמנורה הטהורה, // מהדרת, מקום טורים, בקנים,

ואם אינה כמו שירך – בתוכה // דברים, הם לצריך צנינים.

לקח אותה ותעיד אצלך כי // אהבי לך לעולם נאמנים,

ודבר בה, ותחשב כאלו // אדבר עמך פנים בפנים.
