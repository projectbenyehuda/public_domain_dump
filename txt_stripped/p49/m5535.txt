


	
	אהה שבתי
	



וזאר קברה פי לאתאני מן מואראתה פי לחדה וקאל

בקר את קברו ביום השני אחרי הטמנתו בקברו ואמר


אהה, שבתי / בצר רוחי, /  אלהים יח- / נך, אחי!



קברתיך  /  ביום אמש / וגם היום /  מרי שיחי.



לך שלום!  / הלא תשמע / בקראי לך / בכל כחי?



השיבני:   / התכיר מ- / ענה קיני /  בהצריחי?



ואיך לנת  / בקברתך  /  עלי עפר  /  בבית דחי,



הרפתה מ- /  מך עצם  /  ומכתש מ-  / מקום לחי,



עזבתיך  /  בכור אבי, / לפקדון  /  ביד גוחי,



ואבטח, כי  / אלי שלום / תהי הולך, / במבטיחי.
