


	
	[אחי מוסר]
	



ולמה וקף אבי אעזה אללה עלי אלאביאת אלמתקדמה אלדכר כתב אליה

וכאשר קרא אבי, יוקירהו האל, את הבתים הנ"ל כתב אליו




אחי מוסר  / ובר תורה! / לך שמחה, / לך אורה,

לך ישר, / למען כי /  נתיבתך  /  מישרה,

ומכל מום /  יחידתך /  כמו ספיר /  מטהרה,

ואת נבחר /  וטוב מכל /  מלאכת אל  / אשר ברא!

הבינני / עלי מה זה / ומה זה רו– / חך סרה,         [5]

ואיך ירך /   לבבך / ותתרפה  /  ביום צרה,

ואיכה מ–  / פני רעת  /  זמן קצת, /  ואיך תירא

בשומו חץ  /  עלי יתר  /  והציבך /  למטרה,

כאילו את /  הלינות  /  בני עמך  / בתבערה,

ואת קמת /   וריבות /  עלי משה  / במי מרה,     [10]

ויפית / לישראל /  להמיר אל /  בבן פרה,

וצוית / בני אהרן /  להמרותו  / באש זרה,

ומשלת, / כמו יבין, /  בחזק יד /  ובגבורה,

ומאסת / בברק בן /  אבינעם  /  ובדבורה,

וספחת / נביאי דן /  בככרות  /  ובאגורה,        [15]

ומאנת / לאדם ראש / נביא שמרון / בחבורה,

וישרת / שנא דודים / בעין סרה / וסוררה,

ופזרת / לבן עפרת  / לבנון מר /  אשר ארה,

והכיתו, / להרגו את / בני אדם  /  בחץ ירה,

ושמת על  /  צבי, המית / צבי, כפר  / וכפרה,         [20]

ולא חשת, /  בהקבץ /  ידידים, ל– / ענות שירה,

ולא חלת, /   ביום פרוד /  מידעים,  /  כמבכירה,

ולא תרת  /  למרעים  /  מקום מלון / ובית דירה!

דעה כי ה– /  זמן אשת  / נעורים לך / מעברה,

ולא תדע  /  אשר תלד /  ביום מחר/  לך הרה!         [25]

בטח באל /  ובידידך /  כמו נפשך, / והמירה,

‘הנחתי / כעב עברה /  ואנחתי /  משמרה’

ב’אנחתי  /  כטל הולך /  ועל ביתי /  פאר קרה'.

אני אשכי– /  חך רעת /  זמן, ענה  / בך סרה,

ואם ‘כים /  שטפתיך’  / באהבה לא / מסתרה –     [30]

כים אשא /  לך טובה,  / תהי לדור /  מספרה,

ותפיח, / בהזכר, /   פעלתי, / כאזכרה,

ולא ‘תמות /  בצמאון  /  בתוך צרה / ובצרה’,

ואם יכרה /  זמנך בור  / לך – יפל /  בבור כרה,

ואם ידרך /  נתיב ביתך – / בראשו אד– / רכה פורה,     [35]

ואעשהו / לך כדת  /  וכהגן / וכשורה,

ויתלבן / לך, אחר /  אשר היה /  כפי כירה,

ויישן מ–  /  להרע לך /  כעיף או /  כשכורה,

ותעש לו / אשר תחפץ / ותשחק בו / כצפורה

ואם תמהו  / בני אדם  / בהמתיקי  / לך מרה –     [40]

חכמים י–  / ענו: מה זאת / ומה נפלא  / ומה נורא

עלי דוד / להתעורר  / ולקחת  /  ביד עירא!
