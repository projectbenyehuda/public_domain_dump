


	
	[אהה שומרים]
	








∪ — — — ∪ — — — ∪ — — — ∪ — — —



אהה, שומרים // מצאוני1 // והכוני, // פצעוני!



אהה, שרים אהבוני2 — // ביד צרים3 //
עזבוני,



טפחוני // ורבוני4 —
// ולא שעו // לעלבוני5,



והשקוני // וכסוני6 — // וסוף דבר7 //
מאסוני8!









שומרים מצאוני והכוני פצעוני– בשוטטי
בלילה לבקש את אהובי. על פי שיר השירים ה ז: מצאוני השומרים הסובבים
בעיר הכוני פצעוני.  ↩



אהבוני – אשר אהבוני.  ↩



צרים – השומרים שהכוני.  ↩



טפחוני ורבוני – לפנים. על פי איכה ב כב:
טפחתי ורביתי.  ↩



ולא שעו לעלבוני – עתה.  ↩



והשקוני וכסוני – לפנים. על פי שופטים ד יט:
ותשקהו ותכסהו; יחזקאל טז י: ואכסך משי.  ↩



לסוף דבר –ולבסוף.  ↩



מאסוני – על פי ירמיה ד ל: מאסו בך
עוגבים.  ↩
