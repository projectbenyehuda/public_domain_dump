


	
	[באל אחד אשר]
	



ולו אל אוהב.






U – – – U – – – U //– – U – – – U – – – U – –



באל אחד אשר יחיד יצרך, // ומלבי ומורשיו גזרך,

והיוצר אשר הטיב לנפשך, // ועל דרכך, ביום פחד, נצרך:

אני מצר לפרודך, ואולם // הבינני: הפרודי יצרך?

ואיך אוכל סבל פרוד ידידי // ובן־דודי ואחי – ואני רך!

חמסי על זמן בוגד אשר יום // נתנך לי, ויום שני הסירך,

וליל לא מנעך מלהלך // עלי דרך, וגשם לא עצרך,

עזבתני בלי קרוב – ואתה // נדודך משארך אל שארך,

ועשקת, בלכתך, את חברי – // הטוב לך כי תהי עושק חברך?

לכה, אחי, ואל יהיה בעזרך, // ומשרים, כמו אשר, תברך,

ויסך לך באברתו אלהי // תשועתי, ויוסף על הדרך!

ועת תגע לעיר מבצר – בחסדך, // קראה את שלומי על מברך,

וספר את חשיקתי להביט // אלי פניו – ואל יתן שכרך!

וכה תאמר: הנשית ידידות // אשר לעד בדודים לא ימירך?

וקשרת עלי עמית ורע // כקשורים בקיר לבו קשרך

אשר ישלח לך שלום כחול ים // בכל שנה – ואין שלום בעירך!

ויום תשמע בקול שרים ושרות, // וליל צנור וקול כנור יעירך,

ושיר תורים עלי מים, ויונים, // והגיון סיס בפה רחב וארך,

בתוך גנים ובינות מעינים // ונצנים – תנה הלל לצורך

ואל תשכח, בחסדך, רעך, כי // בזכר טוב בכל רגע זכרך.

ואם תפר ברית אחים כרותה // ולא תשתה עלי זכרו בפורך,

ולי תבוז ביום פסחך, ותשים // נקדה על שמי בחג בשירך

ועין תעלים, כזאת ינקר // אלהים עין ימינך מבשרך!

ואם יעש – פתת פתך והכן // קערתך וקום ורכב חמורך!
