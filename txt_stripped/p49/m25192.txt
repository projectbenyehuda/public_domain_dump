


	
	[היש גשם נדבות]
	




ולו אל קצת אנשי החכמה בארצות המזרח



U – – – U – – – U //– – U – – – U – – – U – –



היש גשם נדבות עוד בעבים – // והארץ צמאה מנדיבים,

ושכבת טל התעל עוד – ואנשי // תעלה שכבו תחת רגבים!

ואם ישקו נשיאי עב רגבים – // ועלו על נשיאי עם עשבים!

כפירים, ישחקו אמש לקול סוס, // עליהם ישחקו היום כלבים,

ואישים, הזמן מחה שמותם – // ומעשיהם עלי מצחו כתובים,

ודן ערב ובקר על שמותם // להצען כמו אהל ערבים,

והמה, כל אשר נותר לליל // ויום־זכר לזכר טוב נקובים.

ומה יתרון ביין אהב ולכת // למשתה אחרי לכת אהובים?

ומה ימית כאב נידת ידידים, // במותם שלחו בלב כאבים?

שכח את זאת ותן פוגת לנפשך // ואל תשכח שארית החשובים,

ואל תשכח אשר שכחו בימיו // בני תורה ותלמוד העצבים,

וכבה מעלי לבך, בזכר // שמו הטוב ומפעליו, שביבים.

אני אומר, ואין כזב בחכי, // – ואיכה יענה שוע כזבים? –

פעליך כמו כסף צרופים // ושיחיך נקיים כזהבים

ויש יין בתורת כל חכמים – // ותורתך לתורתם ענבים,

ואת שרשה, ומפיך יניקון // שריגיה וישיקו יקבים,

כאלו למדך נביא, נתנה // בתוך ארון ותחת לכרובים,

ולמדת נביאים מזכריה // ומלאכי, ומחגי – כתובים,

והורך רחבא תלמוד, ורבא // הביאך בחדריו הרחבים.

אמר לי: איך בפיך רוש ונפת, // ומי נהר בחכך עם להבים,

לסוד רם, נמנעו תחת לשונך, // וארבעתם בפה לא נערבים?

שנים – תאכיל מהם אנשים // אשר הם אל תעודתך רעבים,

ובשנים תענה כל קשי לב // אשר הם עם נבוני לב מריבים.

ואיך תטש קרוביך למען // חמדוך וזכים לך קרובים.

ואיך תשב בעבר ים ותצוד // בעבר זה בלי רשת לבבים?

ואיך תבוא בכל מלה קרובה // ומענה רך ותשכן בקרבים?

ידידי! אם בני אדם בבבל // חבריך, ורעי מערבים –

אני אמשך במו חכת דברים // מתוקים ממך אלי אהבים,

ואם את נעלה על במתי עב – // בעטי ממך אזיל רביבים,

3 ואעל אחריך אל עננים, // ואשיגך בסלם מכתבים!
