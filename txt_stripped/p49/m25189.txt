


	
	[עשה אחי]
	



ולה אלי שכץ כלפה אלאשראף עלי מקאבלה אלמקרא אלדי ככתב בכט ידה

ולו לאדם אשר הטיל עליו את ההשגחה על הקבלת (הגהת) המקרא שכתב בכתב ידו






U // – – – U // – – – U // – – – U
– – –



עשה, אחי, / בחייך, / שאלה ש– / שאלתיך,



ואל תהיה / כקץ ארך / ותוחלת / ממשכה,



והסופר / אשר תכין / יהי מבין / כמותך,



וצוהו / ותהיה כל / עבודתו / עלי פיך,



וישמר / בכל טעם, / ובמנידה / ובסמיכה,



ובפתוחים / ובקמוצים, / ובמלאה / ובחשוכה,



ותהיה את / לעזר לו – / יהי צורך / בעזרך! –



אני אומר, / ואדירי / אנשים יא – / מרון ככה:



בישר כל / פעליך / וצדק מל – / אכותיך



וטוב טעם / בך נמצא, / ועזרה ל – / חבריך!
