


	
	[היום ילך וירחיק]
	



ולה גואב אלי בעץ אכואנה ען קציד כאטבה בה

ולו תשובה לאחד מרעיו על שיר תהלה שכתב אליו


U – – – U – – – U //– – U – – – U – – – U – –



היום ילך וירחיק נאמני // ויום יבוא ויקריב את זקוני

ולא אשית לבבי אל אשר סר, // ולא אראה, אשר יבוא, בעיני?

והזקנה שאר יום הפקדה, // והמות אחי פרוד, נבוני,

ואשמח בזמן כי יאריך לי – // והוא כלה אנשים מלפני

ולי יקרה כמקריהם וארד // אלי שחת כמו רדת זקני.

יגוני על תמותה העתידה, // וכוסה אשת עלי כרחי יגוני,

ואשאה כליל מות בליל // יהי ממחרתו נד שכני,

וידוה על פרידתם לבבי // כמו ידוה בהפקד עווני

ולא למעט וגם לא על נקלה // נסיעתם תעורר את שאוני:

חשבוני כאישונם בעינם // וכימינם, ושמתים לי אדני

והרבו, יום אשר נסעו, יגונים, // כמו הרבו, ביום חנו, נגיני,

והם היו בלב צרי רמחי // והם היו בצדיהם צניני

ואיך יעל עלי לבי נדודם // ואגילה וישמעו רנני

ואיך אזכר פרידת מור פלוני // ולא אשכח ביום מועד ששוני

אשר נסע – ונסעו כל בשמי // במסעו, וראשית כל שמני,

וסרה מעלי ראשי עטרת // וסרו מעלי גלדי עדני,

ידיד נפשי והפדוי בנפשי // ואף כי יהלומי או פניני

והעורך תעודת אל בישר, // כאלו למדו משה בסיני,

אשר יבא כתבו יום מבואו // ויתראה ששון לבי בפני,

ואתהלך בשמחת לב כשולל // ויתבונן ששוני על סדיני

יהללך זמני על לשוני, // כאלו ממך ימי ושני

ויגידו שבחיך עצמי, // כאלו את אשר תכין מזוני

ונעם ספרך חדה לבבי, // כחדותו בנטעי נעמני

ושלחתי לך בן עט בנחץ // ואדע כי יקוצון בו מביני,

למען לא הכינותיו בימים // וגם לא יגעו בו רעיוני.

אני תומך בכוסי בך, למען // מנת כוסי בך מאת אדני,

ואתמך את אהביך בחיי, // ואנחילם ביום מותי לבני!
