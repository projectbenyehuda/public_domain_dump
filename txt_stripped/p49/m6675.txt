


	
	לך עשר
	



וכאן אלחכים אבו מדין באיתא ענדה פאנבהה פאעלמה במא ראי ולם ישך פי תלף אבן אבי מוסי פקתל אלי איאם יסירה פקאל ענד מא [בש]ר בהלאכה וד ͘ לך שנת שמונה מאות וסמאהא בשורה

והיה הרופא אבו מדין לן אצלו והעירו (הנגיד) והודיעו מה שראה (בחלומו) ולא פקפק במותו של אבן אבי מוסי, ואכן נהרג אחרי ימים מועטים, ושר (הנגיד) כאשר נתבשר במותו — וזה היה בשנת שמונה מאות — וקראה:

בשורה

  
  È     — — —  È     — — —  È      — — —   È  — — —


לך עשר  /  וממלכה   / והעז מ-   / לפניך,



ואת מושל /  בכל מושל /   ונפשותם   / בידיך!



אני אראה  /  ואגיד ל-  /   אנשים מל- /  אכותיך,



והעור –  /  אשר לא י-  / חזה הדרת  / כבודך,



5  והאלם –  /  אשר ידם  /  ולא יגיד  /  פעליך.



לך ההוד  /  והכבוד  /  באדמתך /   ושמיך,



וכמה ה-  /  עלית איש – /  והיה מ-  /  משנאיך,



וכבדתו –  / והוא בקש  /  להרע אל  / ידידיך,



וכמנשה  /  בך כחש  /   ואמר: לא  / ראיתיך!



10 מבשר טוב,  / הבינני  /   והרם קול, /  בחסדך,



וצפצף לי /  כסוס עגור,  /  ואל תמעיט / אמריך:



הצדק הוא /  אשר אשמע,  / ואיך היה, /   ואיככה?



הבינני:  /  הגוית בן-  /  אבי-מוסי  /  מחתכה?



ונבלתו –  /  הנסחבה  /  עלי חוצות  / והשלכה?



15 השיבני:  /  חדה ושמח  / ויעלץ  /  לבבך,



וקום ורקע  / עלי רגל   /  ושוב ומחא / בכפך,



והרפד  /  בתפוחים /   והסמך   /  ביינך,



וגדל שם  /  אלהיך  /   ומלא מ- /  שחוק פיך,



וקום וקרא  / בשיר שירים /  ואל תקרב /  אלי איכה.



20 ותן לי ב-  /  בשורה את  / חליצתך  /  ומדיך,



למען בן-  / אבי-מוסי  /  כבר נפל /  במהפכה,



כמו נכסף  / לבן-עבאס  /  אשר בקש /  להרגך,



והלך ל-   /  חזות פניו  /  בצלמות  /  ובחשכה



והעיר, מ-  / משוש קמיו,  / משמחת  /  כמותך,



25 ומשואת  /  בני ביתו  /  מבהלה  /  ונבוכה.



השיבותיו:  / דבש מתוק /  וחלב על  /  לשונך,



ותתבשר  /  מהרה כי   /  כמו זאת בא / לצריך!



קחה מדי  / ומלבושי  /  ושית נזרי /  בראשך,



ולך אל כל  /  מידעי   /  והשמיחם  /  במליך,



30 וידעו כי  /  יחידתו /  לגיהנם  /  ממשכה,



וכי מלך  /  הרגהו  /   וראה בו  / אשר חכה.



בנו-עבאס,  / שמח כי בא /  לשחרך  /  חברך!



שבו יחד,  /  והכהו,  /   בצלמות, /  ויככה,



ואם תחפץ — / קשר חבל   / עלי ראשו  / וראשך,



35 וינוד על  /  אשר היה – /  ותשיב לו  / בקיניך,



ויהיה לך /  כמו פסל – /   ותהיה לו /   כמו מיכה,



וחרשו על  / בני שחת,  /  שניכם, רע /  ותהפוכה,



וירקב בן-  /  אבי-מוסי  /  כמותך  /   בקברך,



וישכח,  /  כמו נשכח, /  שמך הרע  / וזכרך.



40 כבר באתם /  אלי צואר / בפח יקוש /  ובשבכה,



ונחה ה- /   מלוכה מן /  משואתכם,  / ושככה,



ואין בכם /  אשר יצלח  / למועצות /  ולמלוכה.



ואת, חופר, /  אשר תחפר /  קבורתו  /  בידיך —



כרה לו כ- /  אשר כרה, / בדינך ו- /   אלהיך!



45 והעמק עד /  תהום לו – ו- / אני אתן  /  שכרך,



והך באת  /  לכל עין /   אשר עליו  /  תהי בוכה.



לזאת אלבש / ואתעטף  /  ואתבשם /  ואסוכה,



ואתנהג /   בטוב לבב,  /  ואשובה  / ואלכה,



ואדמה  /   אלי יונים /  עלי מלאת /  ובברכה,



50 ואתאמץ  /  כשמשון על /  יתד ארג  /  ומסכה,



ואגילה  /  במות צרי  /  בביתם או  / במערכה.



ואומר: מי  /  אשר עשה  /  לך את זאת / כנפשך?



עניתיהו:  /  הלא תדע? /  אלהים י-  / עשה ככה!



שתק וכתב /  לך טורי /   לזכרון  /  בספרך,



ותקראם,  /  קריאתך  /  ביום שבת /  בדתך,



והיו על  /  לבבך,   /   והודעתם /  לבניך.
