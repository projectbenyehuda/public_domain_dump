


	
	[יחידתי לבבך יפת]
	



ולה ענד אנתקאלה פי אלצבא מן קרטבה ופיהא דכר אלאנתראב

ולו (שיר) בצאתו בנעוריו מקרטבה ובו הזכרת ההליכה לנכר






U – – – U – – – U //– – U – – – U – – – U – –



יחידתי, לבבך יפת חידה // תהי בדבר נדד אחים יחידה,

באותיות כמשכיות כתובה // ובנקדות מחשקות נקודה,

והלמד כקהלת תעודה // שבחיו למדי קהל ועדה

ובאי והגי למועדים את // חסדיו כאשר תבוא חסידה

ואל תנשי פרידתך ידידך // ותשובי ותבכי על פרידה,

למען אחרי נשכח זכרתיו // ושכחתך אהובך עין מרידה.

ושואל על מקום אש הנדודים, // השיבותיו: בלבי היא יקודה,

בהשיאה רתיחת דם לבבי // אלי ראשי – עלי עיני ירדה.

ואיך יוכל אנוש ריבי להכחיש // אשר אמר – ודמעתי מעידה,

כעדות עש לבת־עיני, אשר היא // לבניה עדי בקר יעודה!

בנודי עששו עיני ועבשו // עצמי כאשר עבשה פרודה,

ומדדתי פני ארץ ברגלי, // כאלו הם שני חבלי מדידה,

והיו לי דמי עיני בביתי // ללחם, ועלי דרכי לצידה,

ולולי כי כתב שלח ידידי // בדבר טוב לנפשי החרדה

ובשלומו – אזי באה לנפשי, // בלא עתה, תמותה העתידה!

דברים נמצאו כצל לעבד, // וכגואל לאנשי עיר לכודה,

וכמטר עלי ארץ צמאה, // וכשלום עלי נפש פחודה.

קראתימו וקראתי בקול רם: // דרור נקרא לנפשי השדודה,

ושלם קץ יגוני ועווני, // ובא שלום, וסרה הקפדה!

ואם בכתב פקדני ידידי – // יחידתי בנושעים פקודה,

ובא נוח לידי העמלה, // ומשלוח לרגלי העקודה!

אני נתפש ביד זרים, כאלו // אני שני לאיש צלה ועדה,

ואסור תוך מצודת נוד – חברי, // התתרפו להתיר המצודה?

ומי יוכל להציל את ידידו // ויתעלם – כזובח לשדה!

אהובי, אם לעקר נעבדתי – // שארי את, פדני מעבודה!

ואם נגזל אני מבין בני אב, // כבנימין – היה אתה יהודה!

ואם אובד אני – השב לנדחי, // ולך על אל שכר השב אבדה!

ואם אין – לאל בי סוד, ועתה // תבואני בנוראות פקידה!
