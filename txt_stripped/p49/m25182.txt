


	
	[היום מחר תמישון]
	



ולה אלי פעצל מן אכואנה במצר

ולו אל נכבד מרעיו במצרים






U – – – U – – – U //– – U – – – U – – – U – –



היום מחר תמישון אהליכם? // תנו נא את לבבי בין כליכם,

וצוו עב, אחותכם בנדיבות, // בהקדרה, בלכת יעליכם,

להתאפק, ולא תבכה לנודם, // ולא תריק עליהם ועליכם,

אבל עת נוחכם תרבה בכיתה, // כידכם, או כמו פי שואליכם.

אני אדאג לנוד עפר, ימאן // שמע לי, והוא ישמע אליכם,

ויתהלך לרחוק אחריכם, // וילין בעריסי צאליכם,

ויתקנא לבבי, בעבור מן // עפר רגלו מלאים שעליכם,

וכי ילך בדרך זו הלכתם // ויטה מן שבילי גואליכם.

עלי מה תחפזו לנוס? תנו לי // במתן נעלו או נעליכם

וארוץ בו אלי נף, עיר מגורי // תחוללכם וקרית סבליכם,

ואלך עם רבי סהלאן אצילה // אלי ציון, תהלת חולליכם,

ואשיחה לשכני הר לבנון // וירדן ואמנה נחליכם:

תנו כבוד ליורש מהללי // אבותיכם ומוריש הולליכם,

כפיר דעה ומועצה, ושחל // תבונה, אחרי מות שחליכם,

צרי המחליא מרפי ידיכם // וצריכם, ומרפא לחליכם,

ומקל לח בידכם הוא – ואיכה // דאגתם מן יבשת מקליכם?

ולמה תשברו מים – ולבו // רויתכם, ובלשונו דליכם!

ואיככה חלי כתם תבקשו // להתיפות – והוא מבחר חליכם!

ואיך יוכל מכבה אור פחרים // והוא אשכם, לכבות גחליכם?

ואבטח כי עלי ידו יבשל // קציר צרים – קחו את מגליכם!

שמעוני, עדת זכים, קרוביו, // אני נכסף, בחיי, לו ולכם,

למען כי כמו עץ קור ידיכם // ויראה על פניכם מי עליכם,

וסות צדק אשר מאב נחלתם // לבשוהו בניכם נוחליכם,

ושבר ימצאון באים אליכם, // ולא רוח מלאות שבליכם,

ונלאו החסידים הנבונים // עשה חסד כמעשי נואליכם,

ויד רשע במכתשכם דכיתם, // וראש עולה שברתם בעליכם,

ועל זדון נעלתם בבריחים // ובנחשת וברזל מנעליכם,

ולכם מפעלי ישר ידועים // במרחקים ובינות מקהליכם,

כזקניכם כנעריכם, כהורות // בבתיכם צעירי עולליכם,

ומשכרת ביד האל שלמה // ומוכנה לשלם פעליכם!
