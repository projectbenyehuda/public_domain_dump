


	
	[באשמרת ערב לבבי]
	



ולה אלי ראש כלה ר שמואל אלבגדאדי גואב ען שער כאטבה בה מן דאניה

ולו (שיר) לראש הכלה רב שמואל הבגדדי, והוא תשובה על שיר שבו פנה אליו מן העיר דניה

U – – – – – //

U – – – – –
באשמרת ערב / לבבי בי ירב / ותוכחות הרב / במוסר הכתירם



ואמר לי: קומה, והנח תרדמה, / והבט שמימה, / וברך צור קרם!



ראה את צבאותם,/ וספר פלאותם / ודע יצר אותם / וידע מספרם;



ושמש מתהלך / בתוכם כמלך, / וכלם על פלך / אלהיך זרם,



כשושני סוגה / נפוצות בערוגה / ירוצון במחוגה / עלי קו כבירם;



והבט רמיהם / בהלם בשמיהם / זכר חבר להם / ומאיר כמאורם



אשר ישב שער / בתוך ארץ שנער ובגדולם גער / ומאס שנערם



זכר חסיד ותם / תבונות לא חתם, / אבל פזר אותם / ורפא נשברם



שמואל בן יוסף / שמע דברי כוסף / שמואל בן יוסף / וענדה ספירם:



חכמים נועדים / בכל יום מגידים / כבוד ארץ כשדים / בשיח ספורם;



ולא לאדירים / מקומות מפאירים / וגם לא נהדרים / בארצם ומגורם,



אבל ארצם תרום / בכל נבון מרום / בצפון ובדרום / שמם נשא ורם.



ויסורו משם – / ואדמתם תאשם, / ותראה מדרשם / כיער מדברם;



ועל כן מי יבזה / לאלוף בן מזה, / אשר הוא תוך עם זה / קצינם וגבירם



ומחסיה בתה / לנודו, ובדיתא, / ואספמיא שותה / ורוה מבורם.



ולו עתה יהיה / בארץ־אביי / אזי במקום האיי / תעודת אל הורם,



ופשטה תורתו / ותרשישי דתו / בפיות אמתו / ונהג את דורם,



ושמוהו על ראש / בני דורו לדרש / וקראו אותו ראש / ונזרם ונזירם.



בשירך שמחתי / ובדבריו ששתי / וכדת אשרתי / תבונת חבורם.



בטוריו נשאתי, / וחכמות מלאתי, / וראשי נשאתי / להתיר נקשרם.



אמריו ספירים, / ואותותיו שרים, / ומלואת טורים / בחשן – סדורם,



וענין כפרחים / ומעשה פתוחים / מכסים כירחים / בלילי עבורם.



לך חלק עליון / בעט ובגליון, / וגולת עם אביון / בצלך מסתורם.



וסוד אמתיק עמך / ללמד את עמך / כמתנשא כמך / כרבם כצעירם,



להשכיל ולהורות / עצלים בתורות / ולזרע אורות / ולניר את נירם.



בך ובמכרי / אנגח צרי, / ולך יתרון שירי / ועלי מחסורם!
