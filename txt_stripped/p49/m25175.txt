


	
	[דבריך בתוך]
	







פראנעה ען דלך ובעת אליה מא הצר

והשיב לו (הנגיד) על זה ושלח לו מה שנזדמן (בכיסו)




U – – – U – – – U //– – U – – – U – – – U – –



דבריך בתוך לבות נטועים // ומדיך בתחבולות שרועים,

ושיריך שקולים וחזקים // ומשבר וענין רע מנועים,

ואת מושל עלי מלים טהורים // כספירים, וענינים יגעים,

ואת מולך עלי לבות חכמים // מחכמים, ומדעת שבעים,

ועל פיך, לדודיך, נשיקות, // ובימינך, לצריך, פצעים,

וגדופים לך – נגעו אלי לב // זמן בוגד כמו חצים תקועים,

בגמלו רע לכל מטיב כמותך, // וגמלו טוב לרעיו המרעים,

ורעתו ארכה עד שמטות // ועד יובלות, וטובתו – רגעים,

ושמת ‘חטא ועוון’ שם זמנך, // ושם ימים הסבות ‘פשעים’

ויש רבים אשר זכו וצחו – // ומטובה, כמותך, גרועים,

זמן הרע ולא אחר לך, כי // זמן יפרע בנשאים פרעים.

ואם חטאך יבקשך – חמסך // עלי יצר ורעיונים נגועים!

ועוד היום ויפתחון ספרים // ויבדלו חסידים מן רשעים

ועד לא בא הירע הזמן לך // ויש לי הון ויד נדיב ושועים!

אני זר מקהת, אם לא אשו לך // תנוכי הזמן הרע רצועים!

ונכרי ממררי, אם יהי לי // רכוש – וזמן יהי שליט ברעים!

לקח נא משרוקיו לך סמדר // ועוד תאכל ענבים מנטיעים,

ועוד תשתה עסיס רמון, ותשכב // במטת שן ומקלעת פקעים,

ועוד תכה, כמכה הכך, את // זמנך, או תנגע נגעים!

ושחת: 'איך תהי נראה כאישים // אשר המה בבור יגון שקועים?

ומה תירא ומה ירך לבבך – // והימים עבדים לך ידועים!' –

התבין כי ימי מבין־לאחרית, // ואם טבו, מעט המה ורעים!

ואין טוב אם אהי שוכב, כפותים // בלי לבב, בבית מדות רבועים!

אני שותק – ושמע מעשי עד // למרחוק כשופרות תקועים;

ואתחפש – ולי דבר ומפעל // אשר יהיו בדור אחרון שמועים;

ואתהלך בבקעה עם בני איש – // ויש־לי לב עלי ראשי סלעים!

וענותי כמו תדע, ואולם // לאט אקרע בלב צרי קרעים.

ואת רעי, ואהבתך זרועה // בתוך לבי כמו אורים זרועים,

ושבתי עמך אתאו ואבחר // עלי השקט ומשכב על יצועים,

ושבת עם חכם לבב ונבון // לנפשי, מחיותה, טוב ונעים!
