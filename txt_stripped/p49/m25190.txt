


	
	[אמת לאל אשר]
	




U – – – U – – – U //– – U – – – U – – – U – –



אמת, לאל אשר אין לו תחילה // ואין לו אחרית, תאות תהילה,

אשר תגיד יצירתו כבודו // בלי אמר, ותתן לו גדלה,

לראשון, כל אשר ייצר יכלה, // וכל יצר בראשית – שב וכלה,

ונשאר אחרי כל מפעלותיו // ונבראיו, אשר חדש ובלה,

אשר אין לו מקום שבת בעולם – // ועולם מכבוד הודו ממלא,

ועבים שת לסתר לו, ואם לא // נשורהו – גבורתו מגלה.

בחי מבין צפוני הלבבות // ועל כל לב וסודיו שת חתלה,

אהבתיך, אהוב נפשי, כנפשי, // ושמתיה לך לעד סגלה,

לבבך שר ואתה על, ולכן // בישראל לך משרה מעלה,

ואת נקלה בעיניך – וגדול // לאל, שת לך גדלה לא נקלה,

ולב מבין בך, ושתי כליות // למועצות, כמו זיתים וגלה,

ואל ביתך, אשר בו התבונה, // לכל דורש תבונה יש מסילה,

ולך פתרון בתלמודך, יישר // עקבותיו וישוה כל מחלה,

וחכמה, תענה כי כמחוגה // שמי שחק והארץ עגלה,

ויש לך מעלה על כל נקי כף // ופעם בחסידי הפעלה,

בידים קשורות בתפילין // ורגלים מהירות לתפלה.

קחה אחי דברים, נבחרו מן // בדלחים, חרוזים על מגלה,

ויש מלה בצד מלה שנואה // ואחרת אהובה היא למולה,

וכל מלי אשר אבחר אהובים // במחברת, כמו חתן וכלה.
