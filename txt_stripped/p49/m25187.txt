


	
	[נשמה מאשר תתאו]
	



ולה פי אלצבא ענד אנתקאלה מן קרטבה

ולו (שיר) בנעוריו בצאתו מקרטבה






U – – – U – – – U //– – U – – – U – – – U – –



נשמה מאשר תתאו גרועה // ונפש מאשר תשאל מנועה,

וגוף – שמן ושבע ודשן, // ונפשו הכבודה לא שבעה,

ואיש ענו מהלך על אדמה – // ומחשבתו עלי שחק שרועה,

ומה יועיל לאיש־בשר בשרו // וטובתו – והנפש ברעה!

ויש רעים מרעים, לא מטיבים, // גדולי גוף – וזמתם גרועה,

ידמו כי להוסיף הון אני סר // ונעתק מחניה אל נסיעה,



ואם־ראשי בסופת יום פרועה, // ובת־עיני בפוך ליל קרועה –



ולא ידעו מרעי סוד לבבי, // אבל לא דברו רעי בדעה,

ונפשם מבלי הכר והבין, // כנפש הבהמה השסועה!

היתאפק אשר נפשו לבנה, // וכלבנה להתנשא יגעה,

וישקט עד אשר יאזר במתניו // כנפיה, כאיש אוזר יריעה,

ועד יפעל וישמעו פעליו, // ויוסיף עוד כים על השמועה?

בחי האל וחי עבדי אלהים! // – וכמוני יהי שומר שבועה –

ברגלי אעלה סלע, וארד // אלי שחת, במעמקים תקועה,

ואתפר את שפת מדבר במדבר, // ואקרע ים בכל שוחה קלועה,

ואשוט עד אשר ארום ואעל // אלי שיאה, תהי לעד ידועה,

ותמצא לצרי בי חרדה, // ותמצא לדודי בי תשועה,

וארצע את בני חורים באזן, // ולי אזן למודעי רצועה,

ולי נפש לרעים מחזקת, // ולי נפש למשטינים מניעה,

ולך תוכה ערוגה, מידידות // מלאה, על נהר אהב נטועה,

ידידות השמורה מנעורים, // כחותמת בטבעת קבועה,

מפתחת כפתוחי ירקרק // עלי שמשה, בסף היכל קרועה!

יהי לך כאהבתך, ונפשך // אשר אהב תהי מצר פדועה,

וישלח אל־למושעות לך, עד // בלי שמש וירח, ישועה!
