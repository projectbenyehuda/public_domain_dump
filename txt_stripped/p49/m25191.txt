


	
	[לך שיחים כמו]
	



ולה [אלי ראש ישיבה כביר] פי אלמשרק

ולו אל ראש ישיבה גדול במזרח.


U – – – U – – – U //– – U – – – U – – – U – –



לך שיחים כמו נפת מתוקים, // לך חידות ידועות בשוקים,

ותחבולות בתורתך כאורים, // ומועצות כמו מים עמקים,

ומחשבת תגלה הצפונות, // שבחה על ענקך לענקים,

ואת כהן, ואולם לא תרפא, // לבד מלב נסוגי לב, בהקים,

והכרת, בלב תוכו כברו // ובמזמות קרוביהן רחוקים,

למחלקות כאיש שעיר שעירות, // בפתרונים כישראל חלקים,

ונשבעת להכות העשוקים // – וכמוך יהי נשבע ומקים –

ברוח פה ושפה, עד ישיבון // ביד מה יש בידם לעשוקים,

ועד תכרית גאון רעים, ותרכיב // לבב נפל עלי גופת ענקים.

בביתך ירביון יושבים, ואולם // בתלמודם ירוצון כאפיקים,

וחכמי לב מהללים בתלמוד // סדרים, לא בתלמוד הפרקים,

וכל כותב בלב תורה – ומוחה // התולים שעלי לבב חקוקים,

וראשית ריב, כליל גשם, חשכה – // ופסקי דין יאירון כברקים,

ומשפטים נטועים בתהומות – // וצמרתם גבוהה משחקים,

ותוכחת שאת נפש מרפא, // ומכסל מנתקת נתקים.

קחה מאת מידעך דברים, // במעבה רום, בככר עש יצוקים,

ידי איש טוב, ושם טוב מחזיקים, // ושמן טוב עלי מדות מריקים.

ואדם יחלש ובנו ונכדו // ובן נכדו – והם כראי חזקים!
