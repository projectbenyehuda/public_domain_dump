


	
	רכב. [אשר נטה]
	








אשר נטה שחקים / עלי ארצו כקבה

ושת שכנו בחשמל / ובינות אש ולבה

ואימה לו חתלה / ואין יוצא ואין בא —

סלח פשע והרחק / מליצי חטא ודבה,





אל נערץ בסוד קדושים רבה!





אלהי, עב לבושך / וענן וערפל,

והנגה יגורך / ותדע מה באפל,

ואת בורא מתוקים / וכל מרור ותפל,

וכל שרש וענף / וכל עקר וטפל,

ולך שחו גבהים / וכל נשא ושפל,

ואם אמר ואגדיל — / לך יתר וכפל!

ואיך תפקד חטאים / עלי תולע ונפל?

זרה חטאו כמו מץ, / כעשן מארבה!





אל נערץ בסוד קדושים רבה!





אשר כל העפרים / וטיט חוצות בחמר,

ומד שחק בזרת / והמים בעמר,

ושת גלד שחקים / בכוכביו כנמר

והוא מוליד וממית / ואין ממרה וממר,

וכל יפל ברשתו / כתוא מכמר וזמר,

וכל נברא כמו צאן / והוא נוצר ושומר,

ולנו שוא וכזב / והוא אומר וגומר,

ונגה לו יסובב / ורוח, כל תסמר,

ואור ירק ושחר, / ואור לבן כצמר,

ואש תבער בתוך אש, / ואש כל אש מכבה,





אל נערץ בסוד קדושים רבה!





לך שעו שרפים / ומלאכים צפופים

ומפחדך יבואון / בתוך חורים וכפים

וכוכבים מאורם / בחשך מחליפים

ואת ממליך מלכים / ונותן חן ליפים

ומזעמך אבדים, / כרגע נאספים

ואת תעטיף קשורים, / ואת תקשר עטופים,

ואת גומל חסדים / למאות ואלפים —

גמל טוב על סגלה, / ואם מחטא עקבה,







אל נערץ בסוד קדושים רבה!





לך רפו שחקים / אשר כראי חזקים

ולך שחו נפילים / ושפל רום ענקים

וכל יאמר ויחליף — / ואת גוזר ומקים,

ואת מביט לארץ / ותרעד מן מצוקים,

ויגלו יסודות / ויראו אפיקים,

וממך קול רעמים / ומשלוח ברקים

ושמת חק ומשפט / בלבנו חקוקים

והסכלנו עשו בם / כמו לצים וריקים.

נתק פשע ועוון / לגופנו דבקים,

ואם רב העוינו — / סליחתך מרבה,





אל נערץ בסוד קדושים רבה!





אשר ישליך קרחיו / כלחם רך ופתות

וקורע חזיזיו / כמו בגד שחיתות —

שלח כבוד לחרד / ולפיד בוז לעשתות

והלבן חטא מאדם / כשני או כמו תות

ואל תמכר עדתך / לזרים לצמיתות

והעל נא ארוכה / לעם נדכה וכתות

ולשכולה גרושה / בלי ספר כריתות —

תארשה באמונה / ותכתב לה כתבה,





אל נערץ בסוד קדושים רבה!
