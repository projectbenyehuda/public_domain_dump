


	
	ובעלי הכספין
	



מסכת עשירים


––– ובעלי הכספין / בכל מערבין ומשתתפין / והחניות פתוחים / לערבי פסחים / ומנתם נכונה / מראש השנה / כלם אחוזי חרב / ישבו בסכה למו ארב / עורכי מערכות / רב ברכות / והנה יעלו בשררה / משרתי עבודה זרה / ובקדש
מעלין / הקרוב לחלין / וכסף נקי יחלף / כל היד המרבה לבדק / וההוגה בהם אומרים כי לא יחסר / לחם ובשר / אשר בניהם כנטיעים / כערוגות זרעים /
וקוראים את עצמם צדיקים / העוסקים בנזיקים / ואם אין בידם כספים / מה יחלקו השתפים / הלא בכסף וברכושים / ובעדי עדיים ומלבושים / תתפארנה הנשים / ובפרוטה הנתנת / האשה נקנית / וכל
משכיל יאהב / פרק הזהב / וראש כל המשנאות / אלו מציאות / ומצא שק או מעות / עקר הדעות / ואין עת עתה להורות / בקדשים וטהרות / ואם יעיזו מבטא על השחיטה / הלא הורו הגמרא / הכל שוחטין ושחיטתן כשרה / ושמענו מפי המורים / שהשחיטה כשרה בזרים / כנשים וכעבדים / אף כי אנחנו שרים ונכבדים / ואם ידינו את הדם לא שפכו / פן יתלכלכו / בגדינו החמודים / הצניפות
והרדידים / ואם לכל דבר נוסיף באור / אלו דברים שאין להם שעור / והכסף את הכל יענה / הן יהרס ולא יבנה / ובנינו אין להרס ולמגר / וסגר ואין פותח ופתח ואין סגר / ועקר כל היחסים / עשר ונכסים / ומה בצע בשרש לא יעשה פרי / והוא בין
אחים יפריא / ואם החכמים גדלו ועל עשרה יוחסין עלו / הלא הלכה כדברי האמרים / הכסף מטהר את הממזרים / ומעלין ליוחסין על פי עד אחד / כל שכן בהתחבר יחד / שניהם בהצד השוה שבהם.





והמטים עקלקלותם / ונלוזים במעגלותם / זה דרכם בארחותם / על־עשרם אשר
גדל / ומהלכם עקב בצד גודל / זוקפים קומתם / ומתהפכים במסלותם / כאשר רוח רעה מבעתם / פעם על ימינם / פעם על שמאלם / מתנענעים במעגלם / רגליהם רגלים ישרות / ושתי כתפת
חברת / להם / בכל דרכיהם / הרגל מהלכת ראשונית / והכתף השנית / ועיניהם רמות / על רקמות / כל דמות
מלבושיהם / אשר עליהם / מנעל דחוק וכתנת דקה / גמד ארכה / ואפוד ומעיל ובגדים אחרים / מפז יקרים / רחבים מלמטה על שוליהם / וקצרים מלמעלה על חזיהם / ורמוני תכלת וארגמן סביב ראשיהם / ופתוחים מצדיהם / לארבעת רבעיהם / ואזוריהם משבחים / על רקעי פחים / והם מחשקים כסף / וחגורים על האצילים / כי מתניהם אחזו חלחולים / ונשתברו רב צלעותיהם / ונלאו מהכיל אזוריהם / אכן אל אחוריהם / תלוים עד שתותיהם / וכיס תלו לצדו / איש איש כאשר תשיג ידו / וכסף וזהב ימלאו / ונשמע קולו בבאו / גם אצבעותיו /
נושאות טבעותיו / מוסבת משבצת זהב / במלאתם / פתוחי חותם.





ובראשם כבע ששרה / דקה וקצרה / ועליה מאחיזים / מטוה העזים / על־חצי הראש שומה / אשר שם מלון הכנימה / ואין חשקם / רק לכסות נתקם / אך שערותם גדולים / קוצותיו תלתלים / וסרח
העדף / בציצת ראשם נוטף / על הכתף / ואם בהק פרח בצורונו / שער ראשו יכסנו / קצוצי פאה ובלורית / והם בעלי ברית / לשעירים אשר הם / זנים אחריהם / ואולי יעופו / בשערותם אשר עדפו / ילכו ולא
ייעפו / וכל אחד מהם מחזיק באולתו / אין איש נחם על רעתו / ואם אשאלנו על דרך המוסר: / ׳דבר לי אליך השר׳ / אקרא אליו ולא יענני / אשר בשערה ישופני / ושבועת אלה אם יבטיא / אל השערה ולא
יחטא / כי פרעו לשמצה / לאמר אם אבחר להתלוצצה / או אם עולה בשפתי תמצא / יפל משערת ראשי ארצה / ולולי ראיתים לעין / השתים במזרקי יין / ומטמאים ראש נזרם / במעשיהם ובמוסרם / אמרתי
עליהם / כי נזירים הם / ואשרי ארץ צדקו שריה / זכו נזיריה / ואולי נשברו המספרים והתער / או האומנים אחזו שער / על־כן יער שערם גדל וסוער / וחיתו אין די בער / וכי ירים כניו / לאחז בסבך קרניו / והנה השורים מלינים עליהם / כי
לקחו קרנותיהם / לעשות להם מסרקים / קשים וחזקים / כי בסבך שערותם תמיד משתברים / כשבר נבל יוצרים / ומדוע נחתכה האלה / אשר בה אבשלום נתלה / אולי תוריד למצער / בעלי השער. – – –
