


	
	מתוך המחברת "משא גיא החזיון"
	



[מרומא, חי באמצע המאה הי"ג]






מתוך המחברת “משא גיא חזיון” (על־יהירות העשירים)





עקר כל-היחסים / עשר ונכסים /
ומה-בצע בשרש לא יעשה פרי / והוא בין אחים יפריא / ואם החכמים גדלו / ועל-עשרה יוחסין עלו / הלא הלכה כדברי האומרים הכסף מטהר את-הממזרים / ומעלין ליוחסין על-פי עד אחד / כל-שכן בהתחבר יחד / שניהם / בהצד השוה
שבהם.



ומודעת זאת בכל-הארץ / כי אלה תולדות הזמן: פרץ הוליד את-רם וחסרון הוליד את-עובד / ויצו אותו הכבד ושב בביתך / או הכן בחוץ מלאכתך / ועם-שונים אל-תתערב / ואליהם אל-תקרב / ופרץ לכד את-המלוכה על-כל-המלאכה / לעשות אותה והותר / ואשר נתן כתר מלכות בראשו / כי רב הונו ורכושו / ומשפחת החסרוני ישב לו ממול / משפחת העמול / וישגב אביון מעוני / לחסרון משפחת החסרוני / לעמול משפחת העמולי / לשאול משפחת השאולי / ויהי מושבם בשפלה / משפחת הריב לה / ארץ כבול / ונסב להם הגבול / מנגב / ארץ הנגב / וממשפחת פרץ / עלה הפרץ / לפניהם / ויתקעו את-אהליהם / מול הגלגל / אשר במעגל / ויבנו להם בתים / ויטעו כרמים
וזיתים / ויקחו להם נשים מכל אשר בחרו / בכספם אשר אצרו / ויסעו מגבורות / התאוה ויחנו בחצרות / ותאר להם הגבול / בין מגדל ובין ים יפו / מפו ומפו / ותשבתם הרמתה / בדרך אפרתה / וישימו כצאן / משפחות לעג ולצון / הלעג השאננים / הבוז לגאיונים / על-כן קראו למקום הזה / גיא החזיון עד-יום הזה.



והמטים עקלקלותם / ונלוזים במעגלותם / זה דרכם בארחותם / על-עשרם אשר גדל / ומהלכם עקב בצד גודל / זוקפים קומתם / ומתהפכים במסלותם / כאשר רוח רעה מבעתם / פעם על-ימינם / פעם על-שמאלם / מתנענעים במעגלם / ורגליהם רגלים ישרות / ושתי כתפת חברת / להם / בכל-דרכיהם / הרגל מהלכת ראשונית / והכתף השנית / ועיניהם רמות / על-רקמות / כל-דמות מלבושיהם / אשר עליהם / מנעל דחוק וכתנת דקה / גמד ארכה / ואפוד ומעיל ובגדים אחרים / מפז יקרים / רחבים מלמטה על-שוליהם / וקצרים מלמעלה על-חזיהם / ורמוני תכלת וארגמן סביב ראשיהם / ופתוחים מצדיהם / לארבעת רבעיהם / ואזוריהם משבחים / על-רקעי פחים / והם מחשקים כסף / וויהם כסף / וחגורים על-האצילים / כי מתניהם אחזו חלחולים / ונשתברו רב צלעותיהם / ונלאו מהכיל אזוריהם / אכן אל-אחוריהם / תלוים עד-שתותיהם / וכיס תלו לצדו / איש איש כאשר תשיג ידו / וכסף וזהב ימלאו / ונשמע קולו בבאו / גם אצבעותיו / נושאות טבעותיו / מוסבת משבצת זהב / במלאתם / פתוחי חותם.



ובראשם כבע ששרה / דקה וקצרה / ועליה מאחיזים / מטוה העזים / על-חצי הראש שומה / אשר שם מלון הכנימה / ואין חשקם / רק לכסות נתקם / אך שערותם גדולים / קוצותיו תלתלים / וסרח העדף / בציצית ראשם נוטף / על- הכתף / ואם
בהק פרח בצורונו / שער ראשו יכסנו / קצוצי פאה ובלורית / והם בעלי ברית / לשעירים אשר הם / זנים אחריהם / ואולי יעופו / בשערותם אשר עדפו / ילכו ולא ייעפו / וכל-אחד מהם מחזיק באולתו / אין איש נחם על-רעתו / ואם אשאלנו על-דרך המוסר / “דבר לי אליך השר” / אקרא אליו ולא יענני / אשר בשערה ישופני / ושבועת אלה אם יבטיא / אל-השערה ולא יחטא / כי פרעו לשמצה / לאמר אם אבחר להתלוצצה / או אם עולה בשפתי תמצא / יפל משערת ראשי ארצה / ולולי ראיתים
לעין / השתים במזרקי יין / ומטמאים ראש נזרם / במעשיהם ובמוסרם / אמרתי עליהם / כי נזירים הם / ואשרי ארץ צדקו שריה / זכו נזיריה / ואולי נשברו המספרים והתער / או האומנים אחזו שער / על-כן יער שערם גדל וסוער / וחיתו אין
די בער / וכי ירים כניו / לאחז בסבך קרניו / והנה השורים מלינים עליהם / כי לקחו קרנותיהם / לעשות להם מסרקים קשים וחזקים / כי בסבך שערותם תמיד משתברים / כשבר נבל יוצרים / ומדוע נחתכה האלה / אשר בה אבשלום נתלה / אולי תוריד למצער / בעלי השער. ואם הכל הוא בטבע הזמן וגלגליו / אשר יושת עליו / או הימים ימי אולת / ושניהם מלאים פסלת / מה-לי עוד לעצבים / או לדאגות ומכאובים / לבקש חשבנות רבים / בתהפכות הזמן והימים הבוגדים / אלכה-לי אל- הידידים ואצא אחריהם / ואעשה כמעשיהם / כי טוב לי היות
זנב לאריות / ולא ראש לשועלים קטנים / מחבלים כרמים.
