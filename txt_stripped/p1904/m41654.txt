


	
	[בת ציון עד אנה נודדת כמלונה (תחנה)]
	



תחנה




בת ציון עד אנה / נודדת כמלונה

יוצאת אנה ואנה / משוטטת בכל-פנה

ידיה פרושות למעונה / נושאה עיניה לקונה



אתה תקום תרחם / ציון כי עת לחננה.







אהובה מכל-עלמות / אחזה רעדה ואימות

בזכרה עתות קדומות / דרכה על-כל-במות

גברה בעז ותעצומות / לעלות מעלות רמות



ועתה רבת שכנה / היציע התחתנה.





דודה אלוף נעוריה / אז בימי נעוריה

העיר ודפק שעריה / ליעדה לו לשחררה

ומלכות עגלה ושריה / הכניע בעבורה



ועתה אויבה ענה / ויקרא שמה שטנה.





זה דרכה במישרים / ביד רמה לעין זרים

חלפה ועברה גזרים / בשמחת לבב ובשירים

טעמה לחם אבירים / ולמדה חקים ישרים



ועתה שבעה בשנה / שרש פרה ראש ולענה.







ירשה ארץ חמדה / נחלת צבי מיחדה

כל-הארץ חרדה / והתגעשה מפחדה

לכל-בנות עלתה בהודה / כמו בת-מלך כבודה



ועתה שכנו במחנה / תנים ובנות יענה.







מרב שקט שלותה / וממשמן מרעיתה

נאצה להפר באולתה / ברית אלהי ישועתה

סררה דבקה בחטאתה / לא זכרה אחריתה



ועתה כשלה בעונה / וירד הדרה והמונה.





עשקוה בני עולה / מיום צאתה בגולה

פוערים פיהם למולה / שובי שובי סגלה

צבי תפארתך בלה / ומה-לך עוד להוחילה



ועתה בת-עוץ עדינה / האלהים אשר נתנה.



קולה נשאה בנדבה / מה-תעוררו את-האהבה

ראו זאת היתה סבה / מפני מוציאי דבה

שטרי קודם וכתבה / בחותם כל-חזה נבא



ועתה יזכר ברית ישנה / אמת לא-ישוב ממנה.





תמה תמותה בהתחזקה / ועל-דודה מתרפקה

באלוף נעוריה לדבקה / ודלתי דדיו לדפקה

נדבת רצונו הפיקה / לבלתי היות עוד לפוקה



ועתה תשוב אל-מלונה / ולא יכרת מעונה.





יאמן דבר כל-נביא / וצופה אשר חזו-בי

מלכך יבוא להביא / נדחך מבית השבי

נוה-צדק ארץ צבי / לבדד לבטח תשבי



ועתה שובי ובאי נא / וארשתיך לי באמונה.





חורפי אשיב במלי / המגדילים עלי

זאת הפעם במכללי / ילוה אישי אלי

קרבה באה שנת גאולי / ויום נקמת היכלי



ועתה אחרי זקנה / היתה-לי עדנה.
