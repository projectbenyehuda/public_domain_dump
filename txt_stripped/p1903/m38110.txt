


	
	משל מט: חזירת יער, צבי וחיות
	




נפשו מוראה ונגאלה / לא ירפא על נקלה

חזירת יער בטיט חוצות מתבוססת /
ובאשפה אשר תרפש בו מתעלסת / והנה ביער
קול משתה ושמחה / ומיני מטעמים וארוחה / כי צבי
נשא קרובתו / ויתאספו אל חפתו / מכל הולך על
ארבע / ויאכל כל אחד וישבע / לחם וקמח בשר
ודגים / ובשמחתם הם אוכלים וחגגים / לבלות עמהם
בנעימים / למגדול ועד קטן משתה שבעת ימים /
והחזירה שמעה שאון עליזות / ולא שתה לבה גם
לזאת / ויבט צבי אחד בחזירה / הנגאלת ובטיט
פערה / פיה אשר אשפות חתר / ותאכל ותשבע
ותותר / ויאמר הצבי לה / באי נא בית הכלה /
אשר קול המון שלו בה / ולא תחסר כל בה / וכל אשר
שאלו עיניך שמה תקחם / לא במסכנת תאכל בה
לחם / שם כל עוף למינהו / וצבי עדיו לגאון שמהו /
ורוח שמחה בלבבו עצם וגבר / ובעל כנפים יגיד
דבר / לא נעדר אכל מכל מינים / וכלים מכלים שונים
/ והצביה מה נעמה אשר לצבי מתיפה / כי על כל
כבוד חפה / ותען החזירה: אם היה שם אכל מכל מין /
האכל אכלתם סבין שרוין בחמין / או מטיט הנרמס
בחוצות / כי כמהם לא נמצא מעדן בכל הארצות / ואם
אלה השנים שם נעדרו / כל השאר בעיני לא
ישרו / כי מיום הולדי כהם לא טעמתי / לחם
חמדות לא אכלתי.

והמשל באשר נכריה עבודתו /
ומכתמי חטאיו לא זרה תאותו / עד כי נפשו מוראה
ונגאלה / וירפא פשעיו על נקלה / כי סעפיו בו
ששו / לעשות מעשים אשר לא יעשו / ודרך עיניו
לבו זנה / גם כי יזקין לא יסור ממנה / ובעיני
שכניו נתעב ונאלח / כי מים חיים ברגליו ידלח /
והיה כי יאמרו לו רעים / הנה מה טוב ומה נעים /
להדבק בדרך הישרים והנקיים / להיות חלקם
בחיים / ומה רב טוב הצפון להם / התחזק ועשה
כמהם / אז יאמר: אשר נסיתי אהבתי / כי שמתי כזב
מחסי ובשקר נסתרתי / ואשלים חפצי ואמלא
תאוה / למען ספות הרוה / וטוב לי מראה עיני /
ולהתעדן רעיוני / מלצפות בעולם אחר להיות מלך
/ שלום יהיה לי כי בשררות לבי אלך.
