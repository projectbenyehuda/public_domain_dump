


	
	משל כט: ערב ושאר עופות
	




מתפאר במה שאין בו / רוח השטות נוססה בקרבו



ערב חשב כי רע יצרו / למה חשך
משחור תארו / כי לו נוצות מזפפות / ולכל עוף
נוצות יפות / אדם לבן צהב ירק / על כן כל עוף עליו
ישרק / באשר הוא מכלם שחור / על כן נאנח וישב
אחור / מכליותיו דלה עצה / לקחת מכל עוף נוצה /
אחת מעוף אך לא יותר / ללכת במקום סתר / להשליך
נוצתו שמה / ולשום אחרת במקומה / אשר יהיה
מכל צבעים / להיות יפה מכל רעים / כאשר זמם פעל
וכלה / אל אם הדרך על המסלה / בחדות לבו מהרה
ירוץ / ואברותיו בירקרק חרוץ / שם יתגאה בכל
מיני צבעיו / ויתאספו עליו רעיו / גם ערב רב עלה
אתם / וישימו סביבו שבתם / וכלם עליו תמהו /
מי הוא זה ואיזה הוא / אין כמוהו בכל הערב / אך
קומתו קומת ערב / וכאשר בנו את תבניתו / הכיר כל
אחד נוצתו / זה אומר: זאת נוצתי / וזה אומר: עכרת
אותי / זה את נוצתו מרט / וזה נותן בבשרו שרט /
ונשאר ממשך ומורט / כי לנגדו דרכו ירט.



והמשל באיש מתקשט במה שאין בו /
ואם חכמה שמע יאמר כי בדה מלבו / ובעת ישאלו
לו עצה ידם כאבן / ויש אשר יתן במקום בר תבן /
ומחכמתו ילך שחוח / תחת חטה יצא חוח / וטוב
להיות אויל מחריש / מלהיות בדעתו חכם משריש /
ואם פיו יפתח יגלה שטותו / וישחקו הכל על עצתו
/ והרבה כבו נר הודם / ויסלו ארחות אידם.
