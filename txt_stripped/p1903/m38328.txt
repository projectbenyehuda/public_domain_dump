


	
	משל עט: קוף ושעל
	




הוי האמר לרע / קצר המצע מהשתרע



קוף ראה שעל על נתיבו / ארכה ורחבה
זנבו / עד עפר יגיענה / וברגליו ירמסנה / קוף
הלך עד תכונתו / הפיל לפניו תחנתו / לאמר: הנני
קוף אהבך / תנה לי מעט קט מזנבך / בלי זנב מבטן
יצאתי / וחשופי שת ערותי / לכסות בה די לך
והותר / והיתרון יהי לך לבתר / או לבני גדול יהיה
/ ויתפלל בעדך ותחיה / בחרי אף ענה השעל /
כדי קצף וחרי ומעל / אשר מזנבי שאלת / אשר לא
עמלת בו ולא גדלת / אם יהיה לי כפלי כפלים /
לא אתן לך לכסות בו את השולים / גם בנך ממנה
לא יגדיל הודו / והגעתיהו אל הארץ ונגלה
יסודו.



והמשל באבירי לב הרחוקים מצדקה /
עת ישמעו מדלים צעקה / ואין אחד אמר הנה / שם
יצעקו ולא יענה / ועת יענו יחרה אפם / כי יקר
בעיניהם כספם / כיסם רחב לבם קצר / מלחונן
דלים בצר / ויראו הצדיק נעזב / ויעה ברד מחסה
כזב / והוי האמר לרע / קצר המצע מהשתרע.
