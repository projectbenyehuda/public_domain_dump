


	
	נאם ברכיה ב״ר נטרונאי הנקדן / אשר משלים אלה הצמיחן והולידן:
	




איככה אוכל וראיתי באבדן / אם לא
אכתבם בספר לזכרון / בעמלי מה־יתרון / ואם
אחשב כסיל או יהיר / לשוני עט סופר מהיר / מי יתן
ויכתבון מלי / וסבת עט מכתב משלי / על־אופן
גלגל העולם / אשר מעיני שכלי נעלם / המתגלגל
באי הים / אלה המית ואלה חים / ויפשט במרחבי
ארץ / לפרץ פרץ על־פני־פרץ / ובגלגל עגלתו המם /
הישרים עד תמם / ואופן מרכבותיו / מתהפך
בתחבולותיו / על הטובים ויכניעם / ועל הרעים
ויושיעם / וירא כי האמת עמדה במישור / וכי ידיה
שלחה בכישור / ויך אותה אל החמש / וינף על
רגליה חרמש / ויקצצם וירעצם / הבשר עם העצם /
ויסירה מגבירה / כאשר לזנונים הרה / ותהי לדפי
/ ותלך שפי / ותחתיה את השקר אהב / ויושט לו
את שרביט הזהב / ויקראהו לפניו / וימצא חן
וחסד בעיניו / ויעמידהו על
רגלים / וייפהו ביפי נעלים / ויקרב כל קרוביו
ואהביו / וירחק בוזיו ואיביו / עד כי עלו במעלות
/ פועלי עולות / וישימו עלילות / להכרית עוללות /
ונטישות וזלזלים / מאביונים ודלים / ואשר לא
יעשו כמוהם / כלי קרב בפיהם / חרבות שפתותיהם /
חנית וחצים שניהם / לשומם מרמס כטיט חוצות / על
כן מעטו גודרי פרצות / כי הרשעים נושעים / ולא חן
היודעים / והצדיקים נאנקים / והמרים נמתקים /
ובני בליעל עלו / והגבהים ישפלו / והתפלה
תפלה / והתהלה תהלה / והעולה עולה / והמזמה
זמה / ושרשם פתוח אלי מרמה / יוצאים ביד רמה /
וקראו בשמותם עלי אדמות / אשר נמשלו כבהמות /
ואנשי צדק / כמשכת חדק / וטחי תפל / יושבים
בעפל / ורום אנשים שפל / בחשך וערפל / כפופי
ראש כאגמון / וייראו מקול המון / אישים אשר מאסו
אבותם / הישבים בחצריהם ובטירותם / ענוקי גאוה
ורהב / כוספי כסף, אהבי זהב / היראים לשלח יד
בהונם / אך לשקר בלשונם / ולא הביאו צוארם /
בעבודת צורם / מגאוה נענקו / והענק לא העניקו /
אך שמנו עשתו / כל אשר יזמו עשתו / בבצע
מעשקות / להחניף בחלקות / ויעשו ויצליחו /
כימים אשר נחו / והלך צדק ודבר מישרים / צלמות
ולא סדרים / ומקול מחצצים / המתלוצצים / סעפי
בי נצים ואצים / לנטות עטי למשל / על השקר אשר
משל / לשבת על מרומי קרת / ותהי האמת נעדרת /
וסר מרע משתולל / ורשע על תאותו הלל / ומלאו
פני תבל / ריקים עושי תבל / ומכבדי אל יקלו /
ורדפיהם יקלו / הה כי נלכדה במצודה / האמת אשר
עמדה / והנבלים עלו למעלה / והנדיבים ירדו
שאולה / ושב כל ראש לזנב / וישתרר כל בליעל
וגנב / וקצרו דרכי אמת ונחבאו / ויאריכו
הבדים ויראו / והעניים בעיני העשירים כדבים
/ ואהבי עשיר רבים / והאויל משריש / והחכם
ישבע ריש / וירבו ויעצמו / עינים קרצו ועצמו /
ונשיקות אחים / אשר בלחיהם חחים / וקוי למתק סוד
רעים / כי ימצאום רעים / ומלשני בסתר / יאמר לו:
אכל והותר / בפיו מהלל / ובקרבו מקלל /
משתחוה וכרע / מחשב אך להרע / והאופן אשר כל
זה יצמיד / עבר עלינו תמיד / ויך בזעפו ויגה /
כל חך באמת יהגה / ומתהלל במתת שקר / לכבודו
אין חקר / נשיאים ורוח ואין גשם / בני נבל גם
בני בלי שם / ושפתים דלקים ולב־רע / יאריך ימים
יראה זרע / ובמעיל בוץ מכרבל / וזר על צוארו
סרבל / בלי כל שלום, בלי בגד / ושלו כל בגדי
בגד / בפה חנף ישחית רעהו / ויחבקהו וינשקהו /
חנף בפיו ואון בלבו / ובקרבו ישים ארבו / דבר
בליעל יצוק בו / וכל אח עקוב יעקוב / ובראותי זה
הזמן אקוב.





ואשא משלי ואמר:



נהה עטי עלי אופן עגלה / ולגלגל אשר נגלל ונגלה

ומתהפך עדי כשלה ונפלה / שפת האמת ושקר קם למעלה

והיתה כל תפלת זך לתפלה / וחף ראש חף והתהלה לתהלה

והקם על בליעל ועולה / ימין עולה והתם תם ונקלה

באי הים עדת שכל שכולה / ועטר בוז ומשפחה שפלה

ואזן רב עשיריה ערלה / לכל שאל ולנתן מהולה

וחפשו בוגדי בגד עלילה / שלל בגד ישר־דרך ושמלה

עדת חנף שמחה היא וצהלה / ותוך מחנה ישרים קול יללה

ואשרו פה אשר דבר נבלה / והדבר נכחות לקללה

ורע כטוב ובלה כבתולה / כמתוק מר ואורה כאפלה

וברכיה זמן קלל ואלה / וסבותיו קטנה וגדולה

וטוב לי פת חרבה ועמלה / וזולתם ולא עמם בנחלה.





כחשבי על זה יבהלוני רעיוני / ותדד
שנתי מעיני / לשום בסתר דברתי / מלים אשר
מלאתי / ואמר לאשר יבא מכתבי לחזות / אכל את
המגלה הזאת / כי מהאכל יצא מאכל / ושכל יצא
משכל / ואל יאמר אשר משלי יזכור / חוח עלה ביד
שכור / ומשל בפי כסילים / כרב חלומות והבלים /
כי מרשעים יצא רשע / ומן המושיעים ישע / אם איש
עלי ימרה חכו / חכמת ערום הבין דרכו.





והנני ממשל בחיות ועופות / לחזק
ידים רפות / וברמש ובשרץ / הנם למשל ליושבי
הארץ / ואחל באריה המלך על כלם / מקטנם ועד
גדולם / כי כבודו בקלון המירו / כן לעשיר
שנורש חבריו לו יתנכרו.
