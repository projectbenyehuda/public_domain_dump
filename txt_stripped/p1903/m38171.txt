


	
	משל סח: ארי ואיש ובור ונחש
	






הזמן קצר והמלאכה מרבה / ובעל הבית דחק לקבה



אריה מסבכו עלה / אל אם הדרך
במסלה / מצפה את טרפו וארב / מן הבקר ועד הערב
/ אולי יגרהו בחרמו / והנה איש לפי תמו / וירא
הארי עליו שואג / ובלב רגז מפחד ודואג / כי רץ
אחריו בזעם אפו / כאשר יהגה הארי על טרפו /
ובמצאו מות מר / אל בארת בארת חמר / שם ינוס אל
בור אגלים / והבור ריק אין בו מים / ובקיר הבור
בתוך מוצא / יתד מזה ומזה יוצא / וישמח מאד על
היתדות / אשר שם רגליו היו עמדות / כי חשב למצא
מפלט ומקלט / מן הארי אשר עליו שלט / וכי נמלט
לעד מרצח / ובל יסע יתדתיו לנצח / אך בירכתי
בור רואה במארב / נחש שרף ועקרב / מצפה לבלעו
כשאול חיים / ויהי לו פחדים שנים / בתוך צהרים
כליל צלו / כי לא מועף לאשר מוצק לו / נס מן הארי
ופגעו / נחש אשר פער פיו לבלעו / אם מדרך הארי
תועה / תהרגהו לשון אפעה / מצא נחש אחרי אריה /
ויאמר: אוי מי יחיה / אם אעלה יכני הארי / אם ארד
פה נחש קברי / ומאין יבא עזרי / בין ארי ונחש לי
מחנים / גם כי לי נקיון שנים / מחסר לחם וכל
מאכל / ולמצא מים לא אוכל / אך זאת היא נחמתי / כי
יתד נאמן מצאתי / ומטות עז לסמך על רגלי / עד אשר
יסור הארי מעלי / ואתיצב עליהם כגבור / ולא ארד
אל ירכתי בור / ובדברו כזה על לבו / עכבר יוצא
מעל גבו / ויבט אליו ויבן / כי אין שער שחור
בו רק לבן / ומצד השני כנגד החור / יוצא עכבר
כלו שחור / וביתדות שלחו שניהם / אשר עמדו
רגליו עליהם / ומעט מעט בהם אכלים / מדי יום ביום
מצבו מכלים / וכראות האיש כי התרועעו / ידע כי
היתדות ינועו / שלשה המה קרותיו / ויתאונן על
סבותיו / אוי מארי אוי מנחש / אשר בפניו רגלי
תחש / אוי לי משני העכברים / אשר מקום מצבי
נקרים / אוי מצמא אוי מרעב / וכי אהיה מהר ללעב /
ותשח נפשי בבור ובעני / כפשע בין המות וביני
/ ולא אשוב לראות רעי / בחשך רפדתי יצועי /
ואת בור חשך אשר אעמד בגבולך / אם ארעב לא
אמר לך / המן הסלע אוציא לי / לחם ומן מאכלי /
וכדברו פעמים ככה / ותלנתו לא שככה / כאשר
הוא כבתוך כף קלע / וינקהו דבש מסלע / הצור
אשר הכל יכל / ויאמר: פתח פיך ואכל / ולא יחסר
כל ימיך / דבש מצאת אכל דיך / ויתעדן בטוב
דשנו / דבש וחלב תחת לשונו / ויפשט חרדות
אשר לבש / ויאמר: מה מתוק מדבש / שכח מה
למעלה מה למטה / וכי הוא ישב בעלטה / ומה
לפנים ומה לאחור / עכבר לבן ושחור / אשר
ממצבו יהדפהו / ומתבל ינדהו / לא זכר ארי
ממעל לו / ולא נחש מתחת רגלו / בדבש אשר יאכל
יתענג / כמו בחטי מנית ופנג / מצפיחת בדבש
לעט / וישמן ישרון ויבעט / ובהיותו שמן ודשן
העכברים אשר שלחו שן / בכל יום במקום מצבו /
העבירו ממנו טובו / פתע פתאם נשברו /
וממקבת בור נגזרו / ונתקו מקיר בור היתדות /
במראה ולא בחידות / ונגדעה כל יתד ונפלה /
והאיש אשר עליה עלה / בא בירכתי בור בפה נחש /
אשר לא ישמע לקול לחש / לא יראה עוד ביתו ובני
עמו / ולא יכירנו עוד מקומו.



והמשל מוסר טוב לנפש / ללבש בגדי
חפש / הארי אשר הלך וקרב / הוא המות אשר לאיש
כארב / לא יהפך אליו ערף / ולא ישכב עד יאכל טרף
/ והבור הוא עולם אשר נצב בקרבו / עד בור ינוס אל יתמכו בו / כל גר בו בעיניו
תושב / והוא כמו יתד נחשב / להאריך ימים לראות
זרע / יצר לב האדם רע / הנחש אשר בתחתיתו / פה
שאול הפתוחה לעמתו / האומרת תמיד: הב הב / עתה
תמאס אשר תאהב / העכבר אשר לבן כלו / הוא יום
המאיר למולו / והשחור הוא הלילה / שניהם חברו
לעשות כלה / באדם עד תמו / כי הלך האדם לבית
עולמו / הלכים סביביו חוזרים וסובבים / כי ממנו הם
ערבים / לכלות מקום מצבו להדפו / והוא לא יבין אל
סופו / והדבש אשר כל זה ישכיחו / הנאת העולם
העולה על רוחו / ומשעשעי הבליו / אשר פרשו פחים
לרגליו / צמח גילו אידו עבש / וטעמו כצפיחת
בדבש / ויתאו אל מטעמותיו / וכרגע שרשי
שמחותיו / יבשו וצמרת קצירו ישבר / עם אבותיו
שכני קבר / ואמר החכם אשר חבר תבונות כללאות
בקרס / הנאת העולם כדבש מסוך בארס / ואני אתן
לצורי שבח ותהלה / על המחיה ועל הכלכלה /
ואשר לא יעשה כמוני, לו יום מהומה ומבוכה /
וכלכלה והאבן בתוכה / באשר מלאו חלב עטיניהם
/ וירגנו באהליהם / ולא שמו במדעם / להלל צורם
ורעם.



ואשא משלי ואמר:



ארי מות שאול נחש / אשר לא ישמע לחש

ולבן עכבר היום / ושחור לילה, רחש

לבבי כי ערבים הם / לכלות חי אשר נחש

ימי חייו ועולם בור / ופריו רק פרי כחש

יתד בור נשבר ומת / ולא יכסה בעור תחש.
