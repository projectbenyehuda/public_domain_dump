


	
	משל כו: חמור, דבורים וצרעה
	




אל תהי בז לכל דבר / כי לכל עת וזמן יגבר



חמור רעה באחד ההרים / והנה סביביו עדת
דבורים / והצרעה מרחפת לפניו / עד אשר נחבאת
באחת מאזניו / וירעש ויגעש ובקולו נוער / הנה
והנה הולך וסער / וימאס בחייו ויקץ / כי היה
נשוך ונעקץ / ויצר לו ולמאד חלה / לקול ההמולה
הגדולה / ויאמר: היום רבו עיני / ומה הקול אשר
באזני / אין זה כי אם רוח רעה / ואם את בגבולי
הצרעה / מה לך לטמן בחבי / צאי נא והלחם בי /
ונתראה פנים אל פנים / היש לך כח ואונים / לצאת
ולאסף עדת דבורים / להלחם בחבורת החמורים /
ותען הצרעה / מי יתן והיה ככה / מערכה מול
מערכה / ונלחמה יחד בבקעה / צרעה לחמור וחמור
לצרעה / והנצוח יבדל מן העדה / ויען החמור: יום
המועד נגידה / אשר נצא למלחמה / ויגד המועד
שמה / וילך החמור להמלך / בארי אשר על כלם
מולך / ויגד לו את כל אשר קרהו / מן הצרעה אשר
באזן הציקתהו / והמועד אשר לו יעדה / ואיך לבו
חונה בחרדה / פן תכנס שנית באזן / ויאמר לו
הארי: האזן / איעצך ושמע בקולי / ולא יקרך פגע
וחלי / סתם וסגר ברצועות כל נקביך / אז לא יוכלו
לך איביך / וכראותם זה מפניך יבהלו / ומשלת
בם ובך לא ימשלו / ושית אל המלחמה פניך / ואתה
תאזר מתניך / ויעשו כן כל החמורים / ולמועד
באו מחצצים וסדרים / והצרעים נלחמים אתם /
ובראותם כי נסתם / כל חור וחור / פנים ואחור /
תחת הבטן נתחבאו / שם גבורתם הראו / להמציא
לחמורים כפלים צרות / והחמורים מתחככים וחופפים
על העצים ועל הצרורות / ומתנכלים עליהם להמיתם /
עד אשר נתקו מוסרותם / ויראו החורים והנקיקים /
והצרעים לכנס בם מתחזקים / ונשכים ועוקצים
ולוחצים / מפה אל פה נפרצים / ויהי ארבם בקרבם
/ ויכנע בעמל לבם / ויאמרו להם החמורים / לא
נהיה עוד ממרים וסוררים / הננו לכם לעבדים / רק
תהיו מעלינו נפרדים.

והמשל במתגאה בגבהו וברב
דליותיו / והקטן ימשל בו בסבות תחבלותיו /
על כן אל תחטיא בשרך בלשונך / לאמר לקטן: קטני
עבה ממתניך / כי הרבה נוקשו ונשברו ברב
גאוה / אין מלחמה טובה רק מלחמת התאוה.
