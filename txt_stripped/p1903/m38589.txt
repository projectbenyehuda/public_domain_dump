


	
	משל קיט: הקנאי והחמדן
	







החמדן והמקנא יחדו ילבשו בשת / ואחריהם נזק על כל אחד ישית



שמעו נא דברי קרשפיא הנקדן /
אשר שפט ודן / לב קנאי וחמדן / לחרב הרג ואבדן /
ולמשחית איבת עולם / ולהשיב דך ונכלם / אשר
אפו חרה / על המשפט ועל הגבורה / ועל דרך
הגזרה / לעות משקל ומשורה / כאשר לאדם הקרה /
ולאנשי זדון ועברה / אחד חמדן ואחד מקנא / איש
אחיו ישנא / ומדברים על יוצרם / הידע רע יצרם /
הרבה מתלננים / וברעה מתאננים / חמדן כה יאמר /
ראה כי רע ומר / כל מעשה האלהים / השפיל את
הגבהים / מדוע אני רש ועני / ויעשיר איש פלוני
/ איבי שכני / הישב לימיני / וחנה על דגלו / למה
ירים את רגלו / על סוס ומרכבה / ונפשי תאבה /
וחומדת לו / כי מחמדים כלו / ולבי אליו יפנה /
להיות במקומו חנה / ויען המקנא / בשנאה אשר
שנא / לא יפנה השם לך / ולא ישמע לקולך /
להיותך קצין עם / אמותה הפעם / אם תעשר / ואם
תכשר / להיות כמוני / לבוש שני / אמר לאלהים אל
תרשיעני / כי מי יחוש חוץ ממני / ואוי מי יחיה /
לא הואלתי ולא יהיה / לכל בני אדם / דיך מחית
בשר ודם / ואתהלך לבדי / כל ימי חלדי /
בעשרי ובכבודי / וכי כביר מצאה ידי / ובעת אשר
אני רש ועני / ידל שכני יותר ממני / וכל עברי
על פני / יהיו כחגבים בעיני / וכלם ילכו חשכים /
ויהי המה הלכים / הלך ודבר / וחרה אף המדבר /
הוא יעשה השם / הוא ידע ואשם / וימצאם מלאך
השם / במדבר לשם / ויהי כאשר ראם / המלאך
קראם / ויאמר: מה ראיתם / ומה בקשתם / ועברי
דרך מה שאלתם / את יוצרכם הידעתם / ויאמרו:
ידענו / ולא נגיד ממנו / כי זר מעשהו / ואין מי
יכילהו / באשר נתן לשכני / כל מחמד עיני /
וממני כל החסיר / וחסדו מאתי הסיר / עד אשר
נפשי מענה / מרב קנאה ותאנה / אשר אלי לא תאנה
/ וזה היה מענה / החמדן באולתו שנה / וסמל
הקנאה המקנה / לא יהיה ולא יבא / כל אשר חמדה
בו / כי אם יריח בצידי / עשיר כנגדי / טוב מחיי
מותי / ורקב עצמות בקנאתי / ואם תשמע לתפלה /
ואם תעלה למעלה / לאלהים אשים תפלה / ותחת
תהלה תהלה / ויריבו שניהם / והם מריקים שקיהם /
ויאמר המלאך: החרישו / למפלט לכם החישו / הלא
תבושו מדבר אל יוצרכם / למה תריבון אלי כלכם /
הנה שלחתי לפניכם / לתת היום לכם / לכל אחד
שאלתו / לעשות בקשתו / נגד אמריכם די / יתן
לכם גמולכם שדי / ולא תשאו עלי נהי / כי הוא
אמר ויהי / יבא לכם פתאום / ויאמץ מלאום ולאום /
ונתתי זאת לכם / יהי לאחד משניכם / כל אשר יפצה
פיהו / ברגע אחד יקראהו / ופי שנים בדברו /
יהיה בשל חברו / מפני שהוא לאחרית / וזאת לכם
הברית / אשר לא תפירו / עברו עברו / ולכו לדרככם
/ כי מהם יבא אליכם / כל אשר תבחרו / וישמעו לו
ויאמרו / עבדיך הננו / ואתה תהיה אדנינו / יהי
חסדך עלינו / כי נחמתנו / ויהי כדברם דבריהם
/ והמלאך הלך מעמהם / כי הרוח נשאתהו / ועין
לא ראתהו / ועקבותיו לא נודעו / אז הבינו וידעו /
כי מלאך השם הוא / ותורת אמת בפיהו / ויען
החמדן אשר חמד כפלים / וטמאתו בשולים /
ויאמר: שאל נא / ויען לו המקנא / איך אשאל דבר
/ ואתה עלי תגבר / להכפיל המשרה / לקחת חלק
בכורה / ואתה האחרון / ויהיה לך היתרון / ועלי
החסרון / ואני באתי ראשון / ואיך אחלה לטוב להיטיב
גורלך / ולהרחיב גבולך / הזה משפט פלילי /
ויאמר: המקנא אתה לי / העמק שאלה / או הגבה
למעלה / עד אשר לא בו תחסר / ולא תהיה מעיר מוסר /
ולמה תקנא קנאה / ונפשך מכל טוב מלאה /
תקותך באה ושמת בטנא / ויען לו המקנא / אם
דברת נכונה / שאל נא ראשונה / מטעם דבריך /
ואני אבא אחריך / ובכפלים לא תחמד / ואנכי לא
אעמד / ולא אוכל לשאת ראשי / ותבחר מחנק נפשי /
אם תוסיף עלי מנה / ואם תהיה באחרונה / ויקצף
החמדן בלבו / וחמתו בערה בו / ויפן אליו בחמה
/ ויכהו ביד רמה / באבן ובאגרף / ובמקל יד
ערפו יערף / ויאמר: על פניך תשאל / ואני אחריך
אגאל / ואם לא, אשחיתך / ומהר אמיתך / ויהי
כראותו / כי אבדה תקותו / ועודנו מחזיק בטמאתו
/ ולהקדים תפלתו / עוד לא יקו / ויעש לו משפח
לקו / וצעקה למשקלת / בקע לגלגלת / החל ראשו
לבצע / הכה ופצע / כאשר גבר חילו / וכאשר צר
לו / חל את פניו / ויתחנן לפניו / ויאמר: אדני /
הרף ממני / ואשאל ראשונה / וכפלת באחרונה /
לשים שלום בינינו / וירף ממנו / אך לא
יחפשנו / ובכנפו יאחזנו / ופן יבקש תאנה
לברח אנה ואנה / ויען המקנא ויאמר: אנא / השם
תגמל לעבדך / הפוך מחסדיך / לשלם לו / את
גמולו / עור לי עין אחת מהשנים / ולאיבי פי
שנים / והפל אחת מהידים / ולשנאי זה תן
כפלים / ויהי כדברו ככה / והנה אימה חשכה /
נפלה עליהם / ותך את שניהם / בעורון / ויהי
לאחרון / מכבד רעהו היתרון / להיות לו הכפלים
מחברו / ולא נפל מכל דברו / ויפן לרעהו פניו /
והנה חשכו שתי עיניו / ובכנפי בגדיו / נשארו
שתי ידיו / ויסר כחו מעליו / והנה קטע משתי
רגליו / וישארו שניהם שמה / לבשת ולכלמה /
ויסר מהם את חמדתם / וגם את שנאתם / כי החמד נח
מחמדתו / בחר מחיתו / ובהותו לא יעוד / ולא
יחמד עוד / הון בית גבר / כי אם בית הקבר / כי עוד
לא יכל / לשים בפיו לאכל / כי אם ללחך במרק /
כלחך השור את ירק / כי בחסרון העינים / יהי לו
נקיון שנים / לא יראה טוב טעמו / ולא ימיש
ממקומו / מאכלו ומוצאו יחד / יהיו למקום אחד / כי
נחשב כבהמה / טוב לו מות כמהומה / וגם המקנא /
את אחיו שנא / לא יקנא עוד באחרים / כי יוצא
בראשי אברים / אשר אבד בעזותו / ואשר הפסיד
בקנאתו / ויכת עד החרמה / ועל זה משלה חכמה /
ודברה על המקנא / בשפתיו ינכר שונא / ובקרבו
ישית מרמה / וישפך עליו חמה / אך המשפט
כמדה / ואז היה איש החמדה / החמדה סוף עשרת
הדברים / והקנאה סוף שיר השירים / שהיא קשה
כשאול / ואין לה גואל לגאול / ואין בה טוב מאומה /
כי אם בקנאת החכמה / לקנא במעשה היראים / ולא
במעשה החטאים / וזולת אולת טובה אינה / החמדה
יוצאה ממנה / כי הקנאה ראשונה / ואחריתה מרה
כלענה / ומרה תהיה החמדה שבאה / כי אם המעשה
מביאה / והכל דוה אחת / והצרעת פרחת / רבה
והלכת / עד אשר מולכת / ולב האדם מושכת / אל מבית
לפרכת / כי בסוף תשליכנו / על פני השדה
תגעילנו / בקנאו ממון אחרים / יכרו עליו רעים /
וזה משפט הקנאי / הלא משנאי / אשר השם
ישנאהו / הנקדן קרשפיהו / מבקש מאת יוצרהו
/ אשר מקנאה ישמרהו / ומחמדה יצילהו / וזה
הכתב מעשהו / של קרשפיהו / השם יסירהו /
מחמדה ומקנאה / משאול ומשנאה / וכל דבר רע /
אליו לא יארע / ומחמד הון אחרים / בבני הנעורים /
וגם בימי הזקנים / מלקנא בשכנים / בממונם
ובקנינם / רק בחכמתם ובענינם / ובמעשיהם
הטובים / אשר לעד נצבים / לשמר לעשות כהם /
כמוהם יהיו עשיהם / וזאת הקנאה / מכל טוב מלאה /
וכן דברו הנביאים / אל יקנא לבך בחטאים / כי
אם ביראת אדני / ובדברי הר סיני / ובקנאת הנקמה
/ נתן שלום במרומה / ובשה תמימה / בברית עולם
חתומה / הצור אשר שמו לעילום / הוא יברך את עמו
ישראל בשלום.



אמן



ושלמו משלים, תהלה למשל, / לברכיהו משליו ממשל

תבונות מאסף כזהב וכסף / פתיות ישסף ואהבם ינשל

משלו ענוים מעודד וגאים / מנודד וראשי גאונים יחשל

אמרות טהורות ילקט כאורות / ונותנם בסירות נכונים לבשל

כראשי בשמים חשבום חכמים / ואם הם רתמים לכל לץ וכשל.





סימו

המשלים כלם

שבח

למלך על כל העולם.
