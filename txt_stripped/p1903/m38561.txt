


	
	משל קב: גמל ויעל־סלע
	




הבלתי מסתפק בדי ספקו / יאבד גם לחם חקו



גמל ראה יעלי סלע אחר המדבר / יחלמו
בניהם ירבו בבר / על ידיהם ירבצו אילים וקרנים
בראשם / וכל חית השדה ישחקו שם / והגמל בא
המחנים / ויבא עד האיל בעל הקרנים / ויתפאר
בצלמו ובדמותו / בגבורתו ובגבה קומתו / וכי כל
הלכי על ארבע אשר לפניו / הנם כחגבים בעיניו /
ויען האיל: אמנם נחמדת ונעמת / ואם היו לך
קרנים כמוני מאד עצמת / וישם הגמל את הדברים
אל לבו / וילך אל בעליו המשל בו / ויאמר: אם
הגדלה והגבורה נפלה בגורלי / כל זה איננו שוה
לי / בראותי חיות וקרנים בראשיהם / אשר קטני
עבה ממתניהם / וינגחו ימה צפונה ונגבה / ועל
גבה קומתי הוציאו דבה / ורסן מפני שלחו / לאמר:
כי אתי ינגחו / ועתה אדני שמע בקולי / אזנים
כרת לי / חסדך האחרון מן הראשון השגיבה /
וקרנים במצחי לי הבה / ויפצר בו עד בש / ויאמר
לו אדניו: הגם אתי תכבש / ומדוע לשאול הקשת /
טוב אשר במשאלך החשית / ויתמרמר אליו אדניו /
ויכרת עד החצי אזניו / ולא השיב ידו ולא חמל / על
כן אזנים קטנות לגמל.



והמשל באשר לא די לו בספקו /
וישאל טרף יותר מחקו / ואמר לשכניו כי מאשר
בידו לא יחיה / שרשו כמק יהיה / ופרחו כאבק
יעלה / וכרגע הונו יכלה / כי חרב לזמן מצמדת
בתערה / להשמיד ולכבות גדלתו אשר נשארה /
ואני מצאתי בספר חכמת הזכרון / התאוה שתפת
העורון / וקהלת כתב במליצת משלי חכמתו / דבש
מצאת אכל דיך פן תשבענו והקאתו.



ואשא משלי ואמר:



גמל עת שאל קרנים / ויכרתו מנו אזנים

כן יקרה אל חמד תמיד / אם לו הון רב ומחנים.
