


	
	משל יב: אריה, עז, רחל ופרה
	




מתחבר עם מי שממנו גדל / לסוף כבודו ידל



אריה הלך לטרף טרף / לנפיל ירך
ולכרת ערף / לצוד ציד להביא / ועם חברתו בני לביא
/ עז ורחל ופרה / ובבאם אל היערה / אמרו: כל אשר
יפגענו נפיל חלל / הלא ימצאו יחלקו שלל /
ויראו צביה וילכדוה בשחיתותם / ויאמרו לארי
להראותם / מה ירצה ממנה למלא ספקו / ומה יקח
כל אחד לחלקו / והארי חקרי לבו גדולים / וימינו
להלמות עמלים / ויאמר להם: שמעו נא עצה / לא
תוציאו מן הבשר חוצה / הצביה נפלה בגורלי / אני
מצאתיה, כלה שלי / חלק ראשון, כי אני מלך /
והשני, בהיותי הולך / אחריה במרוצה והשגתיה
/ חלק שלישי, לי חלקתיה / כי אני גבור מכל
היקום / חלק הרביעי, מי יקום / להפרידו מן השלשה
/ מי יענני על זאת קשה / לקחת אותה שלא ברצוני /
לא אכזר כי יעורני / ומי הוא לפני יתיצב / להפר
דין מפי נחצב / ויהי כשמעם מעניו / רגזו וחלו
מפניו / ולכל השואל חלק ישחק / וגער בו ונס
ממרחק / ואחר אשר נסו לקולו / כל הציד נשאר
לו.

והמשל לאדם המתחבר / עם חזק ממנו
אם ידבר / משתפות שביניהם / העשיר אז כארי
נוהם / וישיב לו דבר בזעמו / אין לו חלק ונחלה
עמו / ויאבד הונו ואהבתו / ביום ההוא או למחרתו
/ והטוב תגלה צדקתו / כצאת השמש
בגבורתו.



ואשא משלי ואמר:



ואם דרש דרש / לחברת עשיר רש / בלא גורל נגרש / כמו רעי הארי



בעת שאלו חלקם / בגורל אל חיקם / השיבם חיש ריקם / בזעף אך וחרי


ותבן במקום בר / לחלק איש חבר / / לאיש עליו יגבר / ותחת מר מרי


מבקש שר אתו / לעמת מחברתו / בתוך סכלות שבתו / וגם ילך קרי.
