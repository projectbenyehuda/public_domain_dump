


	
	משל מד: נץ ויונים
	




נותן כבוד לאיש חרמו / קונה אדון לעצמו

הנץ בין היונים שכן וחופף / ואם
יתבודד האחד עמו יתעופף / וכפתע יהדפנו
ויטרפנו / התעיף עיניך בו ואיננו / ויבחינו
היונים ביניהם / כי מתאנה הוא להם / ויעצו לדרש
את שלומו / ואשר יכרתו ברית עמו / להמליכו
עליהם מלך / אם בתם וישר עמהם ילך / ויאמרו לו:
קצין תהיה לנו / בא אתה ומלכה עלינו / ותחזקנה
ידינו הרפות / באשר ייראו מפניך העופות /
ויתנו לו הוד המלכות והמשרה / וארח עמהם
לחברה / ובכל יום בחמה שפוכה / היתה עליהם
המלוכה / כי בז להם ולחבורתם / ויקרח וימרט
נצתם / ויאמר להם: איך תעמדו במקום בואי / אם
אדונים אני איה מוראי / ויטרף בהם ויאכלם בכל
קצה / ואין נדד כנף ומצפצף ופה פצה / והנשארים
הרה נסו / בחייהם קצו ומאסו / ונפלו במשפחותם
מהומות / יוני הגאיות כלם המות / ויאמרו: מה זאת
עשינו / לשום מלך עז פנים עלינו / כי מה שעשה
מקדם בסתר / עשה בגלוי ומי יותר / וטוב לחיותנו
בלתו / כאשר המתקנו סוד אתו.

והמשל לנתן כבוד לאיש חרמו / הוא קנה
אדון לעצמו / טוב אשר עיניו ממנו יגרע / מדבר
עמו מטוב ועד רע / כי זה צפן זדון לבו / ותמיד
ארבו בקרבו / והאיש אשר עובדו ומכבדו / זה
יגל וישמח לאידו / ואמר החכם: כבד את אשר
אהבת והוא יאהבך / והשמר ממי אשר ישנא
לבבך.
