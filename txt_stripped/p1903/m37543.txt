


	
	משל א: אריה וחיות בהמות
	




אוהבי עשיר רבים בעת קרן הודו / ויתהפכו כי ימוך ומטה ידו



אריה זקן היה חלה / וכסליו מלאו נקלה /
ותקצר נפשו בעמלו / ולא נודע מה היה לו / ואם
ימות או יחיה / ולראות מפלת האריה / באו כל בהמה
וחיה / מכל קצוי ארץ נשיה / יש מאהבה לבקר
אותו / ויש לראות תלאתו / ויש לקבל משלו הולך /
ויש לדעת מי אחריו מולך / ויחלה עד אשר לא נודע
באמת / אם עודנו חי או מת / ויבוא השור וינגחו /
אולי תם לריק כחו / והעגל רמסו ברגלים / והשעל
נשכו בשנים / בבדל אזנים / ורחל בזנבה
ממששת שפמו / ותאמר: מתי ימות ואבד שמו /
והתרנגול מנקר עיניו / ויגרס בחצץ שניו /
וישב רוחו אליו ויתבונן / כי שמחו בו איביו
ויקונן / הה ליום אשר תעבוני מתי סודי / וכי הפך
כח ידי והודי לאידי / ואשר היו לי לעבדים / עלי
משלים ורדים / ונהפכו לי לאיבים / אשר מקדם היו
לי לאהבים.

והמשל באדם מלא עשר וכבוד / וכל אחד
משכניו אותו יעבוד / ואם יגיע יום אידו / כי ימוך
ומטה ידו / מנגד נגעו יעמדו / ומעליו יתפרדו /
וצדקתו ממנו יסירו / וימאסו את אשר בחרו.

ואני ברכיה בראותי זה המרי / אשר
הלכו עם נדיב בקרי /



ואשא משלי ואמר על זה
בשירי:



הה כי אריים לפני עגל / רבצו וראשים נשקו רגל

רגלי אמונים רמסו בטיט / ובני שקרים נשאו דגל

וירדו רדים והעוילים / עלים ואיש און מאד יגל

ואמת לשפחה נכבשה היום / נצבה כבר לימין כמו שגל

אין רגלים לאמת כי ה/שקר לקחן בשפת רגל.
