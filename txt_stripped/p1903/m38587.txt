


	
	משל קיז: שעל וזאב
	






כבר אמר שלמה בחכמותיו / צדיק מצרה נחלץ ויבא רשע תחתיו



שעל לטיל על המעין הלך / וירא צל
הלבנה אשר אל המים משלך / וחשב בלבו שהיתה
גבינה / ומאד חמד לאכל ממנה / והיו שם משקלות
תלוין לפניו / ויבט אותן בשתי עיניו / ויושט
ידו ויאחז בחבלים / לעשות מן הגבינה חבלים /
ויכנס בתוך אחת המאזנים / וירד בתוכה אל
המים / ויצעק צעקה גדולה ומרה / וימתן אם יבא
שמה שום אדם לעזרה / והנה הזאב אל המעין בא /
וירא השעל אשר שם נחבא / ויאמר הזאב: מה אתה
עשה שם / הגד נא לי מה במים נרשם / ויען השעל
ויאמר: גבינה הרבה אכלתי ושבעתי / ומהם יותר
לא יכלתי / הנה גם לך גנזתי / ועתה בא הנה
בעצתי / ותאכל אותה ונהיה חברים יחד / טובים
השנים מן האחד / וישמע הזאב בעצתו / ויאמר
בלבו אלך נא אתו / וידו שלח בכף מאזנים אשר
כנגדו / ויקח אתו בידו / וימהר וישב בתוכו
בנחיצה / וירד אל המעין במרוצה / והשעל עלה
מכבדו הכריעו / והזאב פתח פיו אל הצל כבלעו /
וירא כי לא יוכל / ויאמר השעל / חי ראשי ממנה
לא תאכל / ויחזק בידו וימשך בחבל / וירא כי
הכל הבל / וישא קולו בבכי אל השעל / ויאמר:
למה מעלת בי מעל / ויען לו השעל: נא השע / כי
לפני צדיק יצא רשע / ועל זה אמר שלמה בחכמותיו /
צדיק מצרה נחלץ ויבא רשע תחתיו / ויהי היום
וילך האיש אל מעינו / ואשתו ובתו ובנו /
והנערה באה אחריהם בנחת / וכדה על שכמה
מנחת / וילכו כלם בזה ענין / עד כי עמדו אל
המעין / וימשך האיש המאזנים ויראהו / את
הזאב, ויאמר: כי מן המים משיתהו / ויראו כלם
את הזאב / ולהכותו האיש תאב / ויסבו כלם אתו /
ויתחילו להכותו / במקלותם עד כי נלאו / הזאב
רץ והם אחריו באו / וירדפו כלם אחריו / אל ארץ
מגוריו / וישובו אל המעין כלם / לשאב מים
למאכלם / ותמהר הנערה ותמלא כדה / ותקח
אתו בידה / ותשא כדה העלמה / ותשב אתו על
שכמה / וילכו ויבאו אל ביתם / וישמחו שם עם
אבותם.
