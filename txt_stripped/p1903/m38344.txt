


	
	משל פו: תרנגולת וגברתה
	




המסיתך בחלק שפתיו / אל תתאו למטעמותיו



תרנגולת אכלה משטטת / ובין רגליה
מלקטת / שבלים מפרדת ומנתקת / ואת רגליה
מפשקת / לגלות העפר בצפרן / ומץ יסער מגרן /
כמה פעמים לריק יגעה / כי לא מצאה די שבעה /
ותבט אליה גברתה / ותרא כי לבה נפתה /
להחיות נפשה בעמלה / נכמרו רחמיה ותאמר לה /
הנה אותך רביתי וטפחתי / על כן עליך חמלתי /
באשר דגנך מפזר ומפרד / אשר השאיר הברד /
ותקצר נפשך לבקש אתהם / עבשו פרדות תחת
מגרפתיהם / על כן את רזה ותרדי פלאים / כמלקט
שבלים בעמק רפאים / ותרבצי בעמק עכור / הלא טוב
לך לתך חטים וכור / וישבת בפתח תקוה /
מהיותך כל היום דוה / הנה אתן לפניך כל דגני
/ מדשתי ובן-גרני / שוה מאה כסף עבר לסחר / אל
תלכי ללקט בשדה אחר / הנה גלית העפר וכסית /
איפה לקטת היום ואנה עשית / יגעת ולא מצאת
אשר עמלת בו / לפני לחמך אנחתך תבא / וכדבר
הגברת זאת באזניה / ותקל גברתה בעיניה /
ותאמר לה: אם תתני לי בכל חדש חטים / מלא גרנך
כתושים וחבוטים / יחשב לי לחם עצלות / טוב לי
לסבל הדלות / ועוללתי בעפר קרני / מאשר אכזר
מן הגרן ירדפני / כי אשר שם על אחר תקותו /
יגל יבול ביתו / וטוב לי לצאת חפשי / אבקשה את
שאהבה נפשי / לאכל את אשר אמצא / טוב לי ללכת
חוצה / ולתעות כשה אבד / מתוקה שנת העבד /
והשבלת אשר מכל איש תנקה / על שרשי רגלי
תתחקה / שלך שלך, שלי שלי / כי זה חלקי
מכל עמלי / ואדדה בעמלי כל שנותי / כי לא טוב
אנכי מאבותי.

והמשל שמע והבן / האמר לך: קח בר
במקום תבן / אל תבז את אשר ידך מצאה / כי לא תדע
את התלאה / אשר תמצא בשמעך בקולו / בז לדבר
יחבל לו / חשב דברי המפתים לאין / אל תלחם את
לחם רע עין / והמסיתך בחלק שפתיו / אל תתאו
למטעמותיו / כי בטרם תשבע ישבעך / ומחר
מחבורתו יגיעך / איש חמס יפתה את רעהו / ובשבט
פיו יכהו / וגם אם כרת עמו הברית / יזרה על נוהו
גפרית / דרך לץ לאהב לצון / ושחר טוב יבקש
רצון.
