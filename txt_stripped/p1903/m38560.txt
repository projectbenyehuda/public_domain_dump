


	
	משל קא: סחר ולסטים ופרש
	




הגזברים והשפטים נצו גם נעו / חדשים מקרוב באו

סחר ארח עם אוילים לחברה / ויהי
בצהרים בבואם היערה / ויראו עליו בגדי חמדות
/ פחזים אנשי בגדות / כריקים בקקים בקקוהו /
והכוהו שללוהו פצעוהו / וישאר הלום ומדכה /
נגוע פצוע ומכה / ומיתר קשתם נאסר וחבוש /
ערום יצא מבלי לבוש / וללכת להתרפא מן המכים
רצה / ולא התחזק ונפל ארצה / ויצק אל חיק הארץ
דמו / ואין אחד מתחזק עמו / והזבובים נאספו על
צלעיו וזרועיו / ויחנו סביב על כל פצעיו / ללק
וללחך דמי הגזרים / כאשר תעשינה הדבורים /
ויאכלו וישבעו ויותירו / ומרב שבעם עליו נחו
וישארו / והנה דרך שם פרש רכב / ויפן וירא
את האיש שכב / וכי ידיו מאחרי גוו אסורות / וכי
יובל ליום עברות / באשר ערמה כל גויתו /
והזבוב למינו שם תחנותו / וישכב על עפר הארץ
כרסו / ויחמל עליו וירד מעל סוסו / וינתקהו
ממוסרותיו / ומענפי היער ופארתיו / וישב
הדבורים ויניסם יחד / ולא נשאר בהם עד אחד /
ותשאר גוית הפצוע מפרצת / ויקלל את הפרש
קללה נמרצת / וכאוב מארץ קולו מחרף ומגדף /
ויאמר לו: למה את הזבובים תרדף / יעכרך עכרי,
למה עכרתני / מה עשיתי לך ומה הלאיתני / אך
מרשעים יצא רשע / אנוש חצי בלי פשע / כי הקיפו
עלי ארבים ושדוני / הלמוני הכוני פצעוני / אוי
ואבוי הגרימוני שוללי / נשאו את רדידי מעלי /
והנה קמת אחריהם להכאיב לבבי / יספת מכאוב על
מכאובי / ויען הפרש: אך מצרותיך פדיתיך / מה
עשיתי לך ומה הלאיתיך / אשר ככסיל תוציא עלי
דבה / כי אערוץ המון רבה / אשר גויתך מכלים
וכורתים / ותקרב לשחת נפשך וחיתך לממתים /
אשר היו על בשרך הערמה / והדחתיו אל ארץ ציה
ושממה / ואסיר מעל שכמך סבלם / לא תוסיף לראתם
עוד עד עולם / ולא תוסיף עוד לדאבה / למה שלמת
רעה תחת טובה / ותחת ברכתי קללה / ותחת תהלתי
תהלה / ויקש דבר הסחר על הפרש / ויען לו: נקם
נקמות אל יחרש / הקלתני ועכרתני בזרועך
הנטויה / ובידך אשר הויה / לרדף הזבובים מעמי
/ אשר כבר שבעו מדמי / וגויתי כלבוש כסו /
ואחרי אשר תניסם ונסו / באו נכרים אשר דמי
ישאבון / וערקי לא ישכבון / וטוב לי לסבל זבוב
ודבורה / אשר אכלו לשבעה ושתו לשכרה / מלסבל
אשר עדן לא שבעו / על כן דברי לעו.



והמשל בגזברים ומלצרים / אשר העם
מחליפים וממירים / אחרי אשר הריקו הכסף מכיסנו /
האנשים האלה שלמים הם אתנו / כי השבענו
מעדנים כרסם / ומלאנו לרויה כוסם וכיסם /
ומכספנו קצפם נסתם / ויעמידו אחרים תחתם /
אשר לגזל לא נלאו / חדשים מקרוב באו / על כן
כאבן שאבת ישאבו / כפירים רשו ורעבו / אשר
אחר הבצע יטו / ראשיה בשחד ישפטו / ואל
מכיריהם יתנכרו / ומחיה לאיש לא ישאירו / ואין
איש תחת ידם חפשי / ויצו גם את השני גם את
השלישי / לכן אמרי האזינו ועצתי שאו / שפט
ישן מפני חדש אל תוציאו / הזמן אשר על איש גבר
/ פעמים שלוש עם גבר / אשר יפעל בחבריו / על כן
איש בימי מגוריו / יאהב השפט אשר כבר הכיר אתו
/ כי החדש יקח ממך עמדתו / ולעזרו בממשלתו
היה מעיר / פן יהיה מוסר מעיר.
