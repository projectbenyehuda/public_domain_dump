


	
	משל צג: ידע ציד איש שדה
	




מאמיני דברי תפל / לבם באפל

ידע ציד איש שדה / את החיות ארב וצדה
/ והם הכירוהו בלכתו / כאשר נשא אתו תליו
וקשתו / רוחו בלילה לצוד יער / בו תרמש כל
חיתו יער / וטרם באו המה נסעו / ועקבותיהם לא
נודעו / עד כי ראוהו פנים אל פנים / אילים צבים
ושפנים / ואקו ודישן ותאו וזמר / וכל שערות
בשרם תסמר / מפני חמת המציק אשר הכירו / כל
אלה לברח חברו / ונמר תחת אחד השיחים / ראה כי
כלם ברחים / וכי הובא בלבם מרך / ויתיצב על אם
הדרך / לדעת מה זה ועל מה זה / אם בחלום או
במחזה / נסו כלם יחד / והם אמרו לו פה אחד / הנה
האדם ראינו / במדבר ארב לנו / עמו כלי מלחמתו
/ ותשב באיתן קשתו / ולירות תמיד מתוח /
אשפתו כקבר פתוח / שם החצים המה גדודיו /
ויפזו זרעי ידיו / ויען הנמר: אל תמושו מן
המחנה / עמדו עמדו ואין מפנה / וכהפנותם שכמם
ללכת / נמר לא מש מן המערכת / ויאמר: לא את האדם
יראתי / כי אם יקרה לקראתי / אטרף אתו רגע ואצמית
/ אחת דתו להמית / שבו עמי אל תיראו חמה
ואכזריות / ואולך אתכם קוממיות / והאדם אשר שם
ארב / לירות אליו הלך וקרב / דרך קשתו חציו ירה
/ לשלח לו למטרה / ויצא כברק חצו / ובצלע
השנית מחצו / ובלב הנמר דאגה ורגזה / ויתחזק
לנוס בחפזה / וישג את החיות אשר אין מנוח לכף
רגליהם / ובטרם יקרב אליהם / קראו לו: היתכן אשר
ינוס המלך / וכמוג לב בחפזה ובמנוסה ילך /
והארב אם לא טרפת ולא רצחת / מה הבטחון הזה
אשר בטחת / ויען להם כאשר נכנע בעמלו /
ויהי כאוב מארץ קולו / הנה נבהלתי ברעיוני / כי
האדם לא ראו עיני / אך זבובי מות נשכוני במצותו /
הם הם הנקראים בני אשפתו / והה ליום אשר
בטחתי בגבורתי / אוי לי על שברי נחלה מכתי /
כי לא יתעלה כמוני גבור ברב כח ואונים /
כאשרר אחד מעבדיו הקטנים / שלח עד תכונתי
והשברתי / ותדבקני הרעה ומתי.

והמשל בראותך נבהלים וברחים /
הנאנקים והנאנחים / במרירות ובשברון מתנים /
ונמס כל לב ורפו כל ידים / מפני חרב לטושה /
מקול ענות חלושה / במקום רע אל תעמד / כאשר
חייך תחמד / וגבורתך אל תסיתך ותחבלותיך /
פן יתנגפו רגליך במזמותיך.
