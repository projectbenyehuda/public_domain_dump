


	
	משל י: עכבר־בית ועכבר־יער
	




מבקש גדולות ישאר מכאן ומכאן קרח / ויהיה מר צורח



עכבר־הבית נסע ממקום מולדתו / לראות
בעיר אחרת משפחתו / וילך כל היום עד כי פנה
לערב / וירא והנה יער אצלו קרוב / ובו עכברי
יער משעשעים בעדן נפשם / ויפגע במקום וילן
שם / עם עכבר בר אשר שמח בו / וזה לזה הגיד כל
לבו / וירא את מאכליו ואת משקיו / וידו חלקתה
להם בקו / העשבים וכל הנלוים עמם / ושרש
רתמים לחמם / ויאמר אל עכבר היערי / רגלך תאחז
באשורי / ומחר אניעך עמי / ואשקך משקויי
ותאכל לחמי / וטוב לך יום בחצרי מאלף הנה ביער /
במצור ובמצוק וברוח סעה מסער / ובחסר כל מאכל /
ובעת הסתו איך תוכל / למצא מחיה והעשבים יבשו
/ וישביהם קצרי יד חתו ובשו / ואתה מן החרכים
מציץ / יבש חציר נבל ציץ / בא אתי ושא ראשך /
ואתן מעדנים לנפשך / ויסיתהו לבא מן היער
העירה / מקום חטה ושעורה / וזרע במגורה / ויהי
כבואם אל בית לחם / ויאמר עכבר הבית: עליך ארחם
/ הנה כל ביתי לפניך / ואכל כטוב בעיניך /
וכאשר המה אוכלים וחוגגים / קמח ולחם בשר ודגים
/ ויבא שם האדם כדרכו / לקחת לחם לערך ערכו /
וינס עכבר הבית ויתחבא בסכו / ועכבר היער
נחפז אחריו / ולא פחז כזאת מנעוריו / ובעת משם
האדם נפרד / עכבר הבית מנקיקיו ירד / ויצאו
אלה ואלה ממגרשם / מן החורים אשר התחבאו שם /
ועכבר הבית בדגן ובקמח שלח שן / ואכל ושבע
ודשן / והיערי מרחוק עמד / לראות מה יעשה לאשר
אתו נצמד / כי נבעת מן האדם אשר ראה / סמר בשרו
ונפשו נכאה / על הדרך אשר בא בו / וינחם
במחשבו / וכחשבו בלבו ככה / עכבר הבית נס
מן המערכה / ויתחבא בבית סתריו / ויאמר לו
היערי, אשריו / אשר מזה בשלום יפרד / ולא
יהיה עוד ירא וחרד / כי מעת אשר עזבתי מלוני /
עוון עקבי יסבני / ראה ביתך ומקומך / כי לא אוסיף
לגור עמך / והפחד בריתנו הפר /
בא בצור והטמן בעפר / מפני האדם פן יגרך
בחרמו / והארב מגיח ממקומו / כל חייך כעס וזעם
/ ולחם סתרים ינעם / ולמה הוצאת על מקום היער
דבה / טוב פת חרבה ושלוה בה / מבית מלא זבחי
ריב / ולכל אדם אשר יקריב / תנוס בחפזה ותבהל /
להתחבא מאהל אל אהל / ובמקומי אמצא שיר ורון /
הדר הכרמל והשרון.



והמשל למבקש גדולות ונפלאות /
יניח הטוב והנאות / ולא יסתפק בחלקו / ולחם
חקו / יבקש לעלות במעלות נוספות / ולרב ילך
לחרפות / בא זה ואבד את זה / וגם את השני יבזה /
אז יאמר בפה ובלשון / אלכה ואשובה אל דרכי
הראשון / כי טוב לי אז מעתה / ולפעמים ירד מטה
מטה / ויהיה מר צורח / וישאר מכאן ומכאן
קרח.



ואשא משלי ואמר:



טוב פת חרבה לי ושלוה בה / מברבור אבוס ובמריבה

טוב לי באהבה כף מלא קמח / מעשרים כפות ובם איבה

טוב לי מעט ירק בבטחון / מצבי מקום ריבה ומדהבה

שלו בביתך שב ואל תתור / בית זר בזעף אף ולהבה.
