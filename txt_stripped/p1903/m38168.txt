


	
	משל סה: אריה וחמור
	






איש אשר מכיר מקומו / תגלה ערותו ולבו בל עמו



אריה קרא לחמור להראותו / את יקר
תפארת גדלתו / ויתגאה החמור בלבו לאמר: מי
כמוני / למי יחפץ המלך לעשות יקר יותר ממני /
באשר אני לחבורתו הולך / ועתה יקראו לי משנה
למלך / כי שמני אל משמעתו / אתי בחר אין זר
אתו / ויאמר לאריה: הנה על הבהמות אתן חתיתי /
מקולי יחפזון וינוסו מגערתי / ויען הארי: אתה
תשגיא פעלך / להשמיע במרום קולך / ונצא לשלל
שלל רב / ופניך הלכים בקרב / אתה תשאג ואני
אטרף / תאחז טרף ואני אערף / ויצאו השדה
וימצאו צאן ובקר זכרים / וישימו מגמת פניהם
לטרף ולהחרים / ויאמר הארי: המון קולך השמיעם /
ובקול כמוני תרעם / והנה בקולו החמור נער /
והקול במחניהם הלך וסער / ותפוצינה בקולו
הרחלים / משתו יגורו אילים / ויהי לחרדת המחנה
אשר הלך על ארבע / ובהפנותם לנוס ויטרף בם
הארי ויאכל וישבע / וינוסו אנה ואנה לקול
המלות גדולות / ויאמר הארי: אל תשמיע עוד קולך
ורב מהית קולות / ויהי כאשר לטרף כלה / והחמור
והאריה עמדים אצל הנבלה / ויאמר החמור: הלא
שמעת הוד נחרי / וראית כי רעמה לבוש צוארי /
בחכמתך אתי תבחר / והיה כזה יום מחר / ואפחיד
הבהמות ואוריד לארץ נצחם / ולמלך אין שוה
להניחם / לא מהם ולא מהמהם / ואלה המשפטים
אשר

תשים לפניהם / מגערתי ינוסון ויחתו
ממנו / נגביר שפתנו אתנו מי אדון לנו / שמעני
אדני המלך / הלוא דברי ייטיבו עם הישר הלך /
ויען לו הארי: המכיר תולדתך ממעל ומתחת /
מקולך לא יענה ומהמונך לא יחת / בזוי ושפל
וקשה ערף / התצוד ללביא טרף / עוד יאמרו באזני
בני שכלי לאמר / איך טרפת בצאן ולא שברת את
החמור.



והמשל באיש נכבד אשר יקרב בזוי
ונקלה / ורוח גאונו תצמח ותגלה / להתגאות על
חברתו / ולא יכיר את שפלותו / וזדון לבו
בהתגלות / לשון מדברת גדולות / בחכמתי כי
נבונותי / כזאת וכזאת עשיתי / ואמר החכם בחכמתו
/ שתיקת הכסיל היא תשובתו.
