


	
	משל ע: ליש ובהמות
	






טוב במתק שפתים לפתות / למען הסיר גחלי איבה מחתות



ליש גבור בבהמה / וסביבות שניו
אימה / הביא בצאן אלמון ושכל / ולא ישוב מפני כל /
צפון וימין ילחם / אכזרי הוא ולא ירחם / וגם כי
נסו מפניו / הבהילם בחרדת שאוניו / ובעלי חצים
ישטמוהו / ויורו לו וימררוהו / פעם בבטנו פעם
בגבו / ידו בכל ויד כל בו / על כל הולכי ארבע
לא חמל / רק על הצבי והגמל / ויקללוהו כל
הבהמות / קללות נמרצות במהומות / כאשר מפניו
חבאו / ובנקרת הצור באו / וצבי וגמל חבריו / לא
רחקו מארץ מגוריו / וכל אחד מטיב את לבו / וכשמעו
מקלל כהה בו / והנרדפים מקללים ובוכים / כלם
מקללים והמה מברכים / ויקראו לבהמות ולחיות /
רועי שדות וארץ ציות / לבעבור כי תקללו אתו /
יעיר עליכם כל חמתו / ואם תהפכו הקללה לברכה /
הנה מעלה עליכם ארוכה / עשו כמונו ויבחן / אשר
תמצאו בעיניו חן / ולא עוד אתכם יהלום ויתן לכם
את בריתו שלום / וראו ועשו כמונו / כי לא שלח יד
בשלומינו / וישמעו אל העצה / וסמך ידו
ונרצה.



והמשל להסיר גחלי איבה מחתות /
לשנאו במתק סוד לפתות / כי מאד יסכל אשר ישאף
/ להעיר אף כנגד אף / מחשב לרצח רצוח / נצח ברע
נצוח / ואם שנאך תבאיש ותקלל / כבודך תמיר
ותחלל / ראה דרכי הצור תמימים כל מעבדיו / כי
לא תמו ולא כלו חסדיו / קלל נחש עפר לחמו / בכל
אשר ילך עפר עמו / אם יעלה או ירד / ממזונותיו לא
חרד / קלל אשה בנעוריה / והכל רצים אחריה / קלל
אדמה אשר כוננה / והכל נזונים ממנה / קלל עבד
כי העז פניו / אכל ושתה כמו אדוניו / שמע מוסר
תכונן ותבנה / תלך בטח וגם תחנה / ואמר החכם:
אם יעלה לשמים שיאך / ושנאך כפוף לך ממחשב
רע יחטיאך.



ואשא משלי ואמר:

מה תזעם בעל / לשון ולא זעם

הצור, ומפיך / כל־היצור נפעם

אתון בלא חרב / לא השכיל בלעם

להרג בפיו העת / הלך לקלל עם.
