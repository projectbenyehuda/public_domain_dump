


	
	משל כב: נחש ועשיר
	






משלם רעה תחת טובה / מביא על נפשו חובה



נחש היה בא ושב לעשיר בגבולו /
וארוחתו ארוחת תמיד נתנה לו / מאת העשיר עת בואו
/ וחלב לשתות לצמאו / והעשיר יום אחד נפעם /
ושפתיו מלאו זעם / והתקצף וקלל עמו / וביתו
ואנשי שלומו / והנחש בא כחקו / למצא בבית
ספקו / ורגלו בפני העשיר תחש / ויך בזעפו את
הנחש / כי לא נסתר מחמתו / וישב הנחש אל
ביתו / נגוע ומכה ומענה / לחלקו שברון משנה /
ולא יסף עוד להראה / בגבול העשיר ומבואו / ויהי
מקץ ימים באים / וירד העשיר פלאים / נסע עשרו
בא לו רישו / ומת מקנהו ורכושו / ודאגתו הלך
וקשה / תחת שערה באשה / ויסלו ארחות אידו /
ותקצר מהושיע ידו / ויאמר: בפשעי זה כי הכיתי
/ אתמול הנחש בביתי / ופצעתיו ברב חמה / על לא
חמס עשה ומרמה / אלך לשחר פניו באהלו / אולי
יפתה ואוכלה לו / בדבר פתוי וחלק לשון / לשוב
כמשפט הראשון / ויבא עדיו ויקונן / ויבך
לפניו ויתחנן / להעביר מעליו שנאתו / ולא יעיר
כל חמתו / וכה אמר האיש בדמע / נחש, לעצתי תשמע
/ יען התרגזך אלי / שוב וכל מחסורך עלי / ידעת
כי אהבתיך / ואשר בביתי קרבתיך / וחסדי עליך
גבר / לא חסרת כל דבר / ואם בקצפי הכיתיך /
הנה ברצוני רחמתיך / שא נא לפשע מן הפצע /
ואהיה לך עבד נרצע / ויען הנחש למולו / לא אשיב
לאיש כפעלו / אם חזק עלי נגעי / כי איש הרגתי
לפצעי / לא יתמהו עלי השומעים / כי כל בני ביתך
יודעים / אשר לא בעתתך אימתי / ומה תוסיף לפתותי
/ לא אשוב עוד לביתך / ולא אתאו למטעמותיך /
באשר ראשי מחצת / וגדר שלום פרצת / להניף
עלי חרבך / ואם תשחק היום מחר אעצבך / ולא
תשמח אם מותי תחזה / כי הבאת עלי את המות הזה /
ולא את עוונך אפקד / ולא על רעתך אשקד / אך בכל
עת שאני זוכר וראה / פצעי גדול ולבי נכאה / ואש
קדחה באפי / סור פן אמותתך בזעפי / כי עצמה לי
המחלה / ותרבה שברי על נקלה / ואם לא, אפקד
עוונך בחמה / לבך יהגה אימה / מבני הגדול אפע /
כי משרש נחש יצא צפע / ופריו שרף מעפף / אשר
לנקמני עליך חפף / ומגערתו פתע תתמה
ותתרופף.



והמשל באיש יזיד על עמיתו / לעשות לו
רעה בשנאתו / וזה לא איב לו ולא מבקש רעתו /
ישב לבטח אתו / ולא הרע לו ולמשפחתו / ומפני
הרעה אשר לו הרע / זרע שנאה בלבו זרע / אשר
יקצר כראות מקומו / כי יראה כי יבא יומו / שתים
כאחת זה גמולו / ואם זה לא ישלם לו / אחד משאר
בשרו / היודע את יצרו / לנקם נקם לא יתיאש /
בזעף אף ולהב אש / על כן עשה רע לא יתמה / אם
ימלא עליו חמה / המקבל רעות מידו / ידיו
תקרבנה אידו / ובכל דרכיו יהי לו לצורר / ישמח
בפידו ויתעורר / ואיש מראות עיניו טח / אם באשר
הרע לו יבטח / ואמר החכם: מי שאינו מושל בחמתו /
איך ימשל באיש זולתו / ולבי אחרי האומר נטה /
זרע שנאה יקצר חרטה.

ואשא משלי ואמר / לשמים אור לחשך ומתוק למר:



כתב ברכיה בחרט / זרע שנאה יקצר חרט

ובנפשו הוא נתן שרט / ויצר אתו בחרט

פצע רעו או מרט / חרב מלחמה לו מרט

אם דרך האיב ירט / ילקט מן איבתו פרט.
