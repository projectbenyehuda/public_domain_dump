


	
	משל נב: לביא וזאב, שעל, שור ועגל
	




יש בלא משפט נספה / ולשון חכמים מרפא



לביא המלך בגבורתו / זאב ושעל קרא
לחברתו / ויאמר להם: שמעוני עמי / הלביא אשר
עמי / זה חדש ימים גור לי ילדה / נלכה יחד להביא
צדה / וילכו עמו שניהם / ויעבר מלכם לפניהם /
ומחברתו לבבם מגיל יסעד / שלשה המה מיטיבי צעד
/ ובהיותם בשדה כטרפים השקיפו / ושור ופר
ועגל ביום ההוא יערפו / ויאמר הארי לשעל /
הקמת על / הלבך בך הערל / ולא תדע לחלק שלל
בלי גורל / ויען השעל ויאמר: צעיר אני לימים /
והזאב נחשב בין חכמים מחכמים / ורב שנים יודיעו
חכמה / ותהיה משכרתו שלמה / ויאמר הארי לזאב
/ החכם למה כואב / לברר החלקים / כי כל היוצא
מפיו יקים / ויען הזאב: השור לאדני הלביא /
והפר אל הלביא רעיתו להביא / והעגל ביני ובין
השעל / ויאמר הארי: במרד ובמעל / וברשע
ובזדון ובמרי / לא הזכרת חלק לבני בכורי / על
כן דמך בראשך / ואאדיב את נפשך / ויט ידו
עליו ויכהו / ומעור ראשו ורגליו יפשיטהו / עד
אשר קדקדו ועורו בדם מגלל / ויאמר אל השעל: על
פיך יחלק שלל / ויען: אחלקה בצדק ובמישור /
לאדני המלך השור / והפר ללביא ולגור אשר
בו תגל / דנתי כי לחלקו העגל / ויאמר הארי: אין
נבון וחכם כמוך בכל ממשלתי / מן היום והלאה
אשימך אל משמעתי / כי בעיני מאד נחמדת /
לחלק שלל ממי למדת / ויען השעל: כי ראיתי
המצנפת / אשר על ראש הזאב בדם מעלפת / כאשר
נוקש באמרי פיהו / ולב הותל הטהו / וישת מכוס
חמת אדני / על כן לא נתתי לחטא לשוני.

והמשל אל ימהר איש לדבר / לפני אדם
ממנו יגבר / אך יכין מעצר לרוחו / ותחבלות
יקנה ואל יסלף ארחו / אם יש בלא משפט נספה /
ולשון חכמים מרפא / כי במזמות יחשב / ודבר פיו
יאחר / חכם יוסר על ידי אחר / ואמר החכם בספר
המוסרים / לץ תכה ופתי יערים / והישמעאלי אמר
בחכמתו / חלק אדם מאזנו לנפשו וחלקו מלשונו
לזולתו.
