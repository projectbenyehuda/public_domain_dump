


	
	משל עז: צפעון ואיש
	




תרחיק איש אשר יקרבך / עבור כספך וזהבך



צפעון איש אחד ראהו / ובחלק שפתיו
פתהו / לבא להתהלך בגבולו / ברב לקחו התחנן
לו / עד בקולו שמע הצפעון / להתהלך סביב המעון
/ וירא האיש כי בטח בו / שמח ויגל בלבו /
ויכבדהו מאד במעונו / יצרנהו כאישון עינו /
ויאכילהו האיש לרצון / חמאת בקר וחלב צאן / גם
הצפעון אל האיש תשוקותיו / עד כי אמר לו להראותו
אוצרותיו / ימים רבים כן נצמדו / מאהבתם לא נפרדו
/ צפעון לנסות תר בזממו / אם לב האיש נכון עמו /
קרא לאיש ואמר: אהבי / שמעני איש כלבבי / אגלה
אזנך כל דרכי / ואתה אנוש כערכי / הנני שב
לארץ מולדתי / לבני עמי ולמשפחתי / ואדני,
מזמותי / את היקר מאוצרותי / להביא אותו עדיך /
אך בו אל תשלח ידיך / ויען לו: אשמע קולך /
תגזר אמר ויקם לך / הצפעון יצא החוצה / ויבא אל
האיש ביצה / ויאמר לו: זאת הביצה לבנה וזכה / לא
ידע אנוש ערכה / ולא הבין בכל אלה / בכתם טהור
לא תסלה / ופן נפשי ממך תקע / שמר אתה פן
תבקע / כי בהבקעה אבד נצחי / בידך אפקיד
רוחי / תכונתי בה נקשרה / אם היא ערוכה ושמרה /
הצפעון נפרד מאתו / והאיש שמח על לכתו / ומהר
יתחמץ לבבו / און יחשב על משכבו / ויאמר
לאשתו / אשר היתה יועצתו / הביצה אשר הפקדה
אתנו / נבקיענה אלינו / ושמה נמצא אוצר / ועוד
ידנו לא תקצר / כי הצפעון יהיה חלל / ונמלא
בתינו שלל / כאשר חשבו כן הרעו / ביצי צפעוני
בקעו / ויצאו מן הביצה / כמים המגרים ארצה /
ויבשו ממועצותיהם / לא יתכסו במעשיהם /
הצפעון מדרכו שב / למחוז חפצו אל עיר מושב /
וירא הביצה נבקעה / ויאמר לאיש: לבך תעה / על
אהבתך אמר: אללי / כי קרבתי אתך אלי / וכי
ישבתי בקרבך / עד נמצא עולתה בך / ואתה
קרבתי לא חפצת / אך גדר שלום פרצת / ויאמר
האיש, כי ברצונו לא נשברה / ולא את מקורה הערה /
ולא לשבר אליה נגע / כי אם במקרה ופגע / כה וכה
השיב לקראתו / וכפר בעדו ובעד ביתו / והצפעון
אמר: הלכת עמי בקרי / על כן אעלים ממך אוצרי /
אלכה ואשובה אל קני / על מאורת צפעוני / ואתה
תשאר בוש ונכלם / לא תוסיף לראותי עד עולם.







והמשל להרחיק איש אשר יקרבו /
בעבור כספו וזהבו / כי בעת יפקיד בידו תכונתו
/ זה יערים לו בערמתו / להוציא ממנו נפשו /
בעבור כספו ורכושו.



ואשא משלי ואמר:





אחלל רע אשר רע חשבתיו / ביום טובה ועת רעה בחנתיו

ליום שמחה והרוחה פגשתיו / ליום זעוה ועת רעה אבדתיו.
