


	
	משל עה: זאב ורעה
	




כאשר תעשה לאיש טובה / אל תראה לו חובה



זאב היה נרדף במסלה / בערבה בהר
ובשפלה / ומרב פחדו בדרך תעה / ויבא עד
הרעה / ויאמר:קול שאון אחרי צרח / ואני מפניו
ברח / מקול פרש ורמה קשת / כי ללכדני פרשו
רשת / ויאמינו וישמאילו / באו בעבים ובכפים
עלו / לבעבור כי זאב ערבות ישדדם / אמרו: לכו
ונכחידם / ולא ישאר מהם פרסה / מי יתן ועינך
עלי חסה / ובנפש צרי אל תתנני / יסערו להפיצני
/ ומן היער אתי הדיחו / תחת חרול יספחו / ואתה
מעודד לנדד / היה סתר לי מפני שדד / ותהי נפשי
בעיניך יקרה / אל תגדל פיך ביום צרה /
ונסתרתי עד שב הרדפים / אשר באות נפשם אתי
שאפים / וכביאתי אתך כעת ארבה עצבם /
ושן־בהמות אשלח בם / ויאמר הרעה: לך והסתר /
בנקרת הרי בתר / ואני אסכל עצת הארב / ואמלטך
מאבחת חרב / וינס הזאב להחבא בנקרת הצור /
ידכה ישח ויהי עתור / והרדפים אשר אחריו תעו /
עד אצל הרעה הגיעו / לאמר לו: הראית הזאב אשר על
ערינו ישקד / וטרף זרוע אף קדקד / ויען להם: ראו
את נתיבו / זה הדרך תלכו בו / כתרו אתו
הרדיפוהו / מנוחה הדריכוהו / מהרו אחריו להשיגו
/ ארור מנע חרבו מלהרגו / כל אחד לרוץ אחריו
תעה / ותהי מרוצתם רעה / וכאשר המה נלאו /
חדשים מקרוב באו / וכל אחד חרבו יריק / ויגעו
עמים בדי ריק / כי הרעה בפיו ובידו יסיעם /
להראותם הדרך אשר יתעם / אכן בזאב יתן עיניו /
ואת פניו אל מול פניו / וכאשר הרדפים שבו / מזרם
הרים ירטבו / הרעה עד הזאב הלך / ויאמר אליו:קום
לך לך / הקימותי את דבריך / לא נתתי את פי לחטיא
את בשרך / וכל אנחתי השבתי / לחטא חכי לא
נתתי / מלטתיך מרדת שחת / חזק ואמץ אל תערץ
ואל תחת / ויען הזאב: הגדלת לי חסדיך / מדבר
פיך ומנטית ידיך / באשר למדת הרדפים מסלות
/ ללכת ארחות עקלקלות / אך על עיניך תלנתי / כי
תמיד הביטו אתי / ולא נחו ולא שקטו / עיניך לנכח
הביטו / והמה היו בעוכרי / לתת אתי ביד צרי / מהם
היית גדר פרץ / כי מנגד תראה את הארץ / אשר
שמה נסתרתי / כמעט אל מבטך נוקשתי.



והמשל פן יהיו עיניך רכילים / אם
יהיו דבריך פלילים / לדבר לרעך על איש ממעלליו
/ השמר אל תפן אליו / למראהו ולקומתו / לצלמו
ולדמותו / כי עיניך יגידו לו / דבריך בלבב
שכלו / ואמרת החכם שמעתי באזני / ברית כרתי
לעיני / מהביט על איש בדברי בו / כי ממראה עיני
יבין בלבו / אם אאריך לשוני מקלונו או מחסדו /
עת עפעפי יישירו לנגדו / אם יבין הקלון יהפך לו
לחרפה / ואם החסד יחשב לחנפה / ואמר שלמה
בחכמתו הנשגבת / קרץ עין יתן עצבת.



ואשא משלי ואמר:



חכם הוא אך כאשר יעץ / על איש שם את עינו יפנה

הן המדבר מחריץ חלב / לא יביט אל כל הגבינה.
