


	
	משל קיח: בחור וריקים ואשה ושפט
	







אשר שבט יבקש לגו אחיו בשטמה / הוא נקלה בשבטו ויעטה כלמה



ויבא הלך לאיש עשיר / ידע לנגן בכל
כלי שיר / העשיר היה יפה תאר / כאשר במחוגה
מתאר / וייף בגדלו בארך דליותיו / ובחור מעם
הרימוהו אבותיו / וישאל לאיש הלך / מאין באת
ולאן אתה הלך / ויאמר: מארץ רחוקה באתי / ושרים
רכבים על סוסים ראיתי / וייטב בעיני ארץ וטוב
סחרה מסחר כסף / לא יחסר בה כסף / ויאמר לו: מה
העיר ומה שמה / מיום היותך שמה / ויאמר לו:
מעבר לארם נהרים / אלכסנדריה של מצרים /
ויאמר: כך שמעתי אמרים / אספה לי חברים /
ואלכה לי שם בסחורה / בבגד עזים ובאבן יקרה /
ובבגדי צבע תולעת שני / אלכה מצרים בים
ובאני / ותכלת וארגמן ובגדים וכלי יקר / עד
משקל לא נחקר / והכל הכין לא חסר דבר / והנה
הסתו עבר / וימלא את האניה / עד אשר הותר ודיה
/ וישם בה צנה ומגן ורמחים / ויסע עם המלחים
/ ותשאם רוח קדים / ובאו לארץ ששים וחדים /
ויבאו עד לפני אלכסנדריה / ותהם הקריה /
לראות צי אדיר / כי לא נמצא להאדיר / לחוף ימים
ולחוף אניות / כל כח נגדה דיות / והבחור מן
הספינה יצא / ויקח בתוכה פרד אשר מצא /
וירכב עליו וילך להתאכסן / בעדי במתג וברסן
/ ויצא הקול ברחובות ובשוקים / ויתלקטו
אנשים ריקים / ויקשרו קשר / לראות הספינה
והעשר / יען התחברות אליו / להתגולל ולהתנפל
עליו / ולבקש תאנה ומזמה / כי הם אנשי מרמה /
והבחור הלך אנה ואנה / לראות את העיר ואת המדינה
/ ויקר איש בא מבוא / ויאמר לו: אלי תבוא / הצור
לפני אותך הקרה / כי שנינו בעלי סחורה / ואקנה
כל סחורתך אם תאבה / כי יש אני תרשיש עם אניות
אבה / ויען הבחור: אם יש לך רוחה / אמכר לך הכל
בשמחה / ואם רצונך במקנה / מה תתן מכל אשר
בספינה / ויאמר: כאשר תאמר וכאשר תבחר /
אמלא אתה מכל מסחר / ויאמר: כדברך לפניך הנה
/ בא אל הספינה משך וקנה / ויקן הסחורה מאתו
/ ויפרדו איש לביתו / ויהי בבקר למחרתו /
וילך לראות את ספינתו / ויראהו איש לפניו /
וחגרתו אשר במתניו / והסכין אשר בצדו / חגור
עלי מדו / ויאמר לו: סורה אלי סורה / כי הסכין
אשר בחגרה / גנבת ממני / ואת כל ממוני /
ויושם בערכות לועד / לפני השפט ליום מועד /
והבחור היה עניו / וירא ויפלו פניו / ויהי ביום
השלישי בהיות הבקר / קם הבחור להיות חקר / עזר
כנגדו / וקרהו איש אחד לבדו / מבעלי מומין /
נקור לו עין ימין / ויקחהו ויחזק בו / ותפעם
רוחו בקרבו / ויאמר: למה לקחתני / יען כי
גזלת עיני / ואשים נפשי נגד נפשך / כי זה עיני
אשר בראשך / ולשפט מצא ערב / להיות נכון ליום
קרב / ולא יהדר פני דלים / ונתן בפלילים / אך
השפט דיניו ישרים / ולא ישא פני עשירים /
והאיש הלך לדרכו / והבחור הלך הלך ובכה /
כאשר הלך לבקש חנינה / ותמצאהו זקנה /
ותאמר: מדוע פניך זעפים / אשר היו יפים / ויספר
לה כל הקרות / ויתחנן לה להורות / מה יעשה /
ותאמר: לא אשאל ולא אנסה / אך אגלה לך סודם /
כל תפארתם והודם / על פי זקן חכם מאד / והם
יחצו לו כל המאד / אשר ירויחו / כי אליו ישיחו /
יום אחד לפני מועדם / ויספרו לו כל ועדם / וכל מה
שהם מדברים / והוא ספר את הדברים / והם יראים
ממנו / פן יגיד ומה יענו / לאותם שכנגדם /
ויחצו לו כל מאדם / והזקן דר בעליה / ואני
בבית תחתיה / ובעד רחמי הצור / אשימך בנקרת
הצור / ואחביאך מפניהם / ותשמע כל דבריהם /
ויהי ביום בואם להועץ / ותצפנו בפשתי העץ /
ויגלו לזקן דברי סתר / בבתיהם והוא חתר / וחתך
אותם לרעה / והבחור שמע את השמועה / והאזין אל
ניבו / ויטב לבו / ויאמרו: אדנינו נבון וחכם /
והוא פתי ולא חכם / וילכו להם לביתם / ולא ידעו
כי המליץ בינתם / ויהי היום ויבאו להתיצב /
ויאספו חרש וחצב / וכל בעלי מלאכה / לתת היום
ברכה / לבחור להצילו / להשיב אתו גמולו / ויאמר
השפט: עמדו עמדו / אשר להם דבר ויגידו /
והסחר הראשון / בגאותו חרץ לשון / הרבה עלי
ואתנה / ואמלא את הספינה / מכל מה שתאמר /
ויען הבחור ויאמר / לא חפצתי ממך מאומה /
מזהב וכסף ובגדי רקמה / ולא אפרסמון נפת ונדף /
אחר פרעש אני רודף / שאלתי ובקשתי / מהם מלא
ספינתי / ואיש על אחיו מביט וחזה / לאמר: הנמצא
איש נבון כזה / והשני קם וענה / אדני השפט שמע
נא / בביתי הייתי שלו ושקט / ממה שהייתי מאסף
ולקט / עד אשר אספתי הון רב / ובא אל ביתי וארב /
וגנב ולקח הכל אליו / גם הסכין אשר עליו / וכסף
וזהב ובגדי צבעונים / ואעידה עליו עדים נאמנים /
אשר הסכין הכירו / פסקו דיניו וגמרו / וישלם
לי מביתו / ואם אין לו ונמכר בגנבתו / ויקם
לפני השפט ויען / אדני, באתי מארץ כנען /
ומצרפת ומספרד / לזאת לבי לא חרד / והדרך
לנגדי לא ירט / עד באי אל ארץ אררט / ומשם
הלכתי לאספמיה / ועתה באתי לאלכסנדריה /
לגלות הסכין ולהראות / הנמצא איש אשר יאות /
להורות בו ולהכירו / יתברך הצור וזכרו / אשר
הצליח נתיבתי / יגעתי ומצאתי / הן בסכין זה
נהרג אבי / ויגדל כאבי / וגם נמצא בבטנו /
אדני השפט הנו / בידך רק את נפשו שמור / עד
ארכב על החמור / ואביא עדים בדבר / כי לא יהיה
נבר / והשלישי קם בחרון אף / להרג הבחור שאף /
וידבר עליו גבוהות / בעיני כל היו תמוהות / אדני
זה האיש / בא לטרף אתי כליש / והגדיל עלי
בגבורה / ועין ימיני נקרה / ושם עיני בראשו /
בכשפיו ולחשו / ועתה אם לא יודה / בפרך בו
ארדה / ואלחם בו ואנצחו / ואקלע אבן במצחו /
אך גמר דיני / וישלם לי עיני / וצער ורפוי ושבת
ובשת ונזק / כי בא אלי בחזק / ויאמר אל השפט:
בי אדני / ידעת מארץ רחוקה אני / ורחק מאב ואח
ובנים / שמע נא אם דברי כנים / ואם משפט ארצי
ישר / ובעיניך יכשר / נקר נא את עין שמאלו / וגם
ימיני לפי דבריו שהיה שלו / תשים במאזנים / את
שתי העינים / אם שוקלות בשוה / מדברו נאוה /
ואם תכבד זו על זו / כדין ראוי לא יחזו / לעשות
בו משפט / לעין כל בעמק יהושפט / ועמי לא ילחם
/ אך מודה ועזב ירחם / וינצחו הרמאים / וישארו
חלכאים / ויתנו לו כל הונם / ויתש כחם
ואונם.



משלי זה הלא נמשל ונדמה / לבן אדם מלא און ומרמה

אשר בקש עשות און ועול / לאיש תמים, פעל תוך ומזמה

ובא עתו לשלם לו גמולו / ונתמלאה סאתו גם שלמה

בתוך גומץ אשר חפר וכרה / הלא יפל ברגז ומהומה

ויש גבר אשר שבט יבקש / לגו אחיו בזדון ושטימה

ובא העת והוא נקלה בשבטו / אשר בקש ויעטה בו כלמה

ויש פרץ גדרות לשכניו / עשות בהם חרפות גם נשמה

ובא יומו והנחש נשכו / בתוך פרצו ועשה בו נקמה

תנו לב למשלי זה בני איש / ואל תרבו עליכם ריק ואשמה

כמו עשו שלשה בכתובים / ונהפך הוא ונשארו בלי מה.





לכו משלי לאיש תבונות / חזות אמרי היות נכונות

וגם ישרות כדי לאסף / בתוך צרופות ואף בחונות

ואם אמת הוא אשר בעיניו / מצאתם חן בלי תלנות

אזי אחלה פני יברך / להרביצכם בתוך מעונות

וגם כמו כן יחו ויגיד / היש בתוככם מאום זדונות

ויצפין את כתב ידיכם / לבל ראות בם זכר עוונות.
