


	
	משל פג: גנב ומכשפה
	




לא תשמע לרשעים / מחזקי ידי מרעים



גנב ינהק בין השיחים / ויקטף מהם
מלוחים / בערב כבא השמש יעמוד / בחסר ובכפן
גלמוד / מקטב ישוד צהרים / עיף ויגע ורפה ידים
/ ולשתות נפשו נכספה / דרך שם באה מכשפה /
ותשמע כי יצעק למים / ותמלא החמת ממטר
השמים / והוא נכנע ומתודה פשעיו / מקלל
בזעמו את רגעיו / כאשר כלה בבשת ימיו / וחטאו
פרש רשת לפעמיו / אוי לי אמי כי ילדתני / כי כל
היקום מקללוני / בפשעי וחטאתי / השיגוני
עוונותי / והמכשפה מן החמת תשקהו / ועל אשר
נחם על הרעה תריבהו / ותמהר לגשת אצלו / העזה
פניה ותאמר לו / הנה מצאתיך מתבודד / ושמעתיך
מתנודד / על אשר רבו פשעיך / ותקלל ימיך
ורגעיך / על כן נגשתי אליך / פן תמנע מרע
רגליך / אך סעפיך אותך יאיצו / רגליך לרע ירוצו
/ ותרצח ותגנב ותנאף / ואל כל מלאכת רמיה תשאף
/ ואז תצליח דרכיך / ואברכה מברכיך / ואל תירא
מרדת שחת / חזק ואמץ אל תערץ ואל תחת / כי גם
בעת אשר תמצא במחתרת / ידי תהי אותך עזרת /
וזרועי לך משלה / למלטך ממכשלה / בלהט וקסם
ולחש / על מרמה רגלך תחש / וממלאכתך ראשונה
אל תרחק / לשד ולכפן תשחק / ואני אלך אל
חבורתך / ושמעת בקול אמתך / לא תוסיף ולא תגרע
/ נפש רשע אותה רע / לעשת מלאכתו ראשונה / חזר
לטמאתו ישנה / ועברתו שמרה נצח לפתח פה
ברצח / ולהיטיב נפשו לא זכר / גנב וטבח ומכר /
יבחר לו לא יושיע / כי היא היתה יועצתו להרשיע /
ותלמדהו כשפים וקסמים / עד אשר מאתו לא
נעלמים / קלקל בחצים שאל בתרפים / ויאמר לבלה
נאפים / ויהי היום ויצא השדה / ואת הרכלים ארב
וצדה / וישמעו אליו גוים בשחתם נתפש / טמנו
חפש מחפש / זה מכיר עליו בגדו / וזה אמר כי הרג
דודו / ויעצמו עיניו ויכוהו במקלות / ויוליכוהו
אל העץ לתלות / ויקרא למכשפה: מהרי נא למלטני
/ פן הרעה תדבקני / כי הנה איבי בי שלט /
ואנכי לא אוכל להמלט / ופרשת עלי כנפיך /
עמדי נא בחבריך וברב כשפיך / ותען לו: עד הנה
עזרתיך / ובקסמי תמכתיך / וענקתיך גאוה /
למען ספות הרוה / כה פעלתי עמך פעמים רבות /
ובראותי כי אין למותך חרצבות / למה זה הבל איגע /
כי תהיה לשמה כרגע / ולזאת עצתי סכלה וימיני
קצרה / ובא עליך רעה אשר לא תדע שחרה.

והמשל לבלתי שמע אל הרשעים /
מחזיקי ידי מרעים / ולמעננים ולקסמים /
ולמכשפים ולחלמים / לא תבא בעדם הרעה / וגם
נבנו עשי רשעה / חח הזמן באפיהם / ומתגו
בשפתותיהם / אנשי דמים ומרמה לא יחצו ימיהם /
כי באחת יכשלו / וצדיקים יקומו אם שבע
יפלו.
