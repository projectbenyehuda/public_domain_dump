


	
	משל סט: זאב ושעל ויונה
	






חנפים ואנשי תעה / חברתם תמיד לרעה



זאב ושעל ויונה / באו בברית ובאמונה
/ לטרף טרף למאכלם / ולחלק בשוה את שללם /
ויבאו במו ארב / כי ינטו צללי ערב / וילכו יחד
ולא מצאו / רק עטלף אשר הוציאו / מחורי עפר וכפים
/ וישובו רעבים ויעפים / ויאמר השעל: איך יחלק
שללנו / הישחט ומצא לנו / הלכנו בערבה בהר
ובשפלה / ולא מצאה היונה מנוח לכף רגלה / על
כן יאות לתת לה חלקה / ולא תהיה בינינו מעשקה /
ולא נשים עליה עלילות / לכו ונפילה גורלות / מי
יגיד ראשון זקנתו / ואשר יהיה זקן מעמיתו / שמע
נשמע בקולו / והעטלף ינתן לו / והשנים אשר
לא כמוהו בימים באים / והיו מן השלל חטאים /
ויענו השנים: עצתך טובה ונכונה / ויפילו
גורלות ויפל הגורל על יונה / ותאמר היונה: אם
עמכם יגעתי ולא מצאתי מנוח / אני היונה אשר
יצאתי מתבת נח / ובין עצומים אחלק שלל ואשסף /
כנפי יונה נחפה בכסף / ויאמר השעל: זקנתי
ואין מגיד מספר שנותי / אך כך מקבלני מאבותי /
אשר היו מימים קדמנים / סמן לדעת מספר השנים /
אשר עברו עלי מיום הולדי / כל שער לבן אשר בי
עדי / לכל שער שנה, ואשר האמת יבקר / למספר
שני אין חקר / ויען הזאב ויאמר בחרי / שעל
ויונה, שמעו נא דברי / זקן ממני אין אתי / ואני
זקנתי ושבתי / אך סמן שנותי לא באדם ולא
בלבון / אספרם מחול ירבון / ואני יגעתי לדעת
ומצאתי / כאשר משתי שנים יצאתי / באתי אל
השלישית / אך לא אדע אחרית וראשית / ומי יורה אתי
ויאלף / לעזב לקטן ממני העטלף / ויאכל אותו
לעיניהם / ומאומה לא השאיר להם.



והמשל בחנפים ואנשי תעה / זדים
ועושי רשעה / המקרבים עמים בפיהם / ומרחקים
מכליותיהם / כי יהיה לאיש דבר אצלם / ישיבו
במענה רך כרך ונכלם / וארחות ישר מעיניהם
יליזו / אלמנות שללם ויתומים יבזו / ובפה יאמר:
חפצי לחפצו נקשר / לעשות הטוב והישר /
ובהגיע דבר משפט לאחד מהם ושפטום / ויאמר
השפט: שלם, אז יהי אטום / מלשמע דבר אשר לא
יחפץ / יחרק שן ופה יקפץ / ואז יעיר כל חמתו /
כי יבטח כגבור בגבורתו / להעיר גזלות וחבלות /
ללכת ארחות עקלקלות / כאשר עשה הזאב ליונה
ושעל / אשר המתיק עמהם סוד ובאחרית מעל / לקחת
שלא כרצונם חלקם / תחת רשעים ספקם / אך
שבעתים אל חיקם זרעי האמת יקצרו / אל מול פני
המנורה יאירו / כי חובה לחיב וזכות לתם / בפני
נכתב ובפני נחתם.
