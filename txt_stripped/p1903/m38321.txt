


	
	משל עב: איל ורחלים עשרה
	




אין אמונה בפי המקנא / כי השקר לו למקנה



לאיל היו רחלים עשרה / וחברתו להם
שפרה / ויאהב את השתים אשר אין במחנה כמוהן /
ושכולה אין בהן / אך אם יקרב אל אחת מהנה / יביא
בלב השנית קנאה ומגנה / ותרא האחת אשר אהב
יותר ממנה את רעותה / ותקנא רחל באחתה / ובחץ
שחוט לשונה / תדבר עליה שטנה / ואת קשתה
שקר דרכה / כי חנית הקנאה בלבה מעוכה / לאמר:
רעיתך חדשים לבקרים / תחת אישה תקח את זרים /
מבלעדי אישה נתן בה האיל שכבתו / להסיתה
ממך ולשומה אל משמעתו / ומסתופפת בצלו
ברוח נדיבה / בערב היא באה ובבקר היא שבה / ואיך
תחמל עליה / טמאתה בשוליה / משפטה להיות
ערופה / באות נפשה שאפה / רוח, להפרד מעליך
בזנותה / בתוכה טמאתה / תתם חלאתה / ותדבר
כאלה דברי תך ומרמה / ועוד נוסף עליהם דברים
רבים כהמה / ויען האיל: לדברך לא האמנתי / כי
עתק ובוז תחת לשונך ידעתי / אשר בדית מלבך
דברים לא טובים / ותדברי עליה כזבים / בשנאתך
אותה ובקנאה / אשר דרכי ישר מניאה / ואם היתה
נפשך בנפשה קשורה / באהבת עולם ערוכה בכל
ושמרה / אז לדבריך אשית לבי / אך כל עוד נפשי
בי / ארחיק מוציא דבה / אם משנאה וקנאה תהיה
נסבה / ולא אתן משכרתו מעוג / בחנפי לעגי
מעוג.



והמשל אם על איש כרשעו / יגיד עליו
רעו / תשכיל אם הדברים ברורים / עיניך תחזינה
מישרים / כי אם רעו נהפך לערו / תבין כי בזדון
דברו / ואם הוא כנפשו אהבו / ומימיו לא עצבו /
חשב כי אמת הדבר / ולב השני מן הראשון תם וגבר
/ חצי רכיל שנונים עם גחלי רתמים / דברי נרגן
כמתלהמים / ההפכים אהבה לחלוף / ונרגן מפריד
אלוף.



ואשא משלי ואמר:



רחק נרגן בהפרידו לחשקים / וחץ שטנה ישלח כברקים

ולבו מר וחכו ממתקים / אשר חנף שפתיו הם מריקים

מאס זדים בפיהם לך מנשקים / ותוכם נחלי איבה עמקים

ביום גיל לך קרובים גם דבקים / ויום איד תמצאם חיש רחוקים.
