


	
	משל יז: צרעה ונמלה
	




הסר מסוכת העצלות / ולא יהיו עיניך רואות וכלות



צרעה באה אל נמלה / לשאל ממנה
מאכלה / כי חסר גדול יש לה / ממאכל נגד החרף /
והנמלה הפכה ערף / ובזה מאמרה ולעגה / ותאמר:
אחריתך תוגה / וראי נא ממי נעמת / כי בימי
הקציר נרדמת / ולחרף לא הכנת מאכל / כמה
יהיה הלב סכל / הנותן לך אשר אצר / דעי כי ידי
לך תקצר / כי מה עשית עת הקציר / נראה דשא גלה
חציר / וכל היקום ללקט נטו / איש לפי אכלו לקטו /
ולהכין לחרף כל נחוץ / אמר עצל: ארי בחוץ / כי אשר
ירצה חטים לאצר / זרועו שבלים יקצר / כמוני אשר
נשאתים על שכמי / והכנתי בקיץ לחמי / ותען
הצרעה: כל עת הזמיר / בעת ימלאו העגלה עמיר /
למדתי לשיר שיר נעים / והרביתי עלי רעים /
והברחתי מלבי יגוני / כי ישרו השירים בעיני
/ והם אמרו כי קולי ערב / ועתה אכלני חרב / וקר
וחרף וקרח / היו עלי לטרח / ותאמר הנמלה: הגרן
והיקב / לא יושיעוך עתה עקב / למדת בשירים
קולך / ולא חשבת על מאכלך / ואם את מקננת
בארזים / ואם צוארך בחרוזים / תלכי נא אל בית
עשיר / היטיבי נגן הרבי שיר / למלא נפשך כי
תרעב / הן מאמרך אלי נתעב / לכי מעלי והפרדי /
כי תעבוך כל מתי סודי / באשר אני שפלת קומה /
ובי אין כח ועצמה / מלאתי כל טוב אהלי / אך אם
חכמתי חכמתי לי / ואת לא תתפרנסי בשירים /
בבא הקרה ממזרים / כי קרעים תלביש נומה /
ועצלה תפיל תרדמה.



והמשל למתעצל להמציא טרף ביתו /
הישמעו אחרים צעקתו / ישן בקציר ונרדם / לאיש
ההוא יחשב דם.



ואשא משלי ואמר:



בני, צלח רכב עבר / רחוב ועיר בחם, צבר

לעת סתיו כסות וסות / ושור ושה ואך בבר

ועין ימין אויל נקר / ופיו בלם ושן שבר.
