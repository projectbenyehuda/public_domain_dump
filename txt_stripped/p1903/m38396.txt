


	
	משל ק: שד ואניה
	




משלחי מדנים / ישחיתו עצמה ואונים

השד עם אמו ארח על ארץ רבה / לשוט
בארץ ולהתהלך בה / עד אשר על שפת הים באו /
וישאו עיניהם ויראו / אניה מלאה סוס וגבר /
והאניה חשבה להשבר / כי התגעשו וירעשו
שאון ים וגליהם / ויטילו את הכלים אשר באניה
אל הים להקל מעליהם / ומפחד הים כל בשרם סמר /
וקרא זה אל זה ואמר / להשליך בים מחמד עיניו
ועדיו / ולהתודות מפשעיו ומריו / אז מחמדיהם
מאמתחותיהם יריקו / וייראו המלחים ויזעקו /
וכל איש מכלי חפצו חצנו נער / כי הים הלך וסער
/ ויאמר השד לאמו: אם כהיום היית עמהם / יאמרו
כי מידך היתה זאת להם / כן וכן פעמים רבות /
הוציאו עליך דבות / כבא להם יום מבוכה ומהומה /
על לא חמס עשית ומרמה / והאם תשיב אמריה למולו /
העזה פניה ותאמר לו / אם אין אני שם, הרבה מצאתי /
מאשר גדלתי טפחתי ורביתי / אשר שלחתים שם
רצוני יאבון / מידי היתה זאת להם למעצבה ישכבון
/ ומאד ממעשי בארץ ייראו / מהרסים ומחריבים
ממני יצאו.

והמשל במשלחי מדנים / רכילים
ומלשינים / מזעם לשונם זרע יזרעו בחכמים / להרע
לבם ופיהם לשלומים / ויחזו משאות שוא ומדוחים
שקרים / וכל אחד מהם על רעהו יערים / ברעתם
ישמחו מלך ובכחשיהם שרים / וכאשר יצליחו
במלשינותם / אז יתהללו בקרב עדתם / ויאמרו כי על
ידי שליח / ברכילותו הצליח / והחכם אמר בחכמתו
/ שלוחו של אדם כמותו.
