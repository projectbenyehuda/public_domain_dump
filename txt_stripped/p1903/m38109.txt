


	
	משל מח: שעל ועגור
	




לישר מסלת פעלו / יתחבר עם בן גילו

השעל אמר לעגור: למה כל־עת אתה סגור
/ בא וסעד לבך עמי / וראה ביתי ובני עמי /
ויסיתהו לבא עמו / לשמח בו ולאכל בלחמו / ואל
הבשר השעל רץ / ובפתע בהמון תרנגל פרץ /
ויך תרנגלת אשר מול השמש סגרה עפעפיה /
תתיפח תפרש כנפיה / והמאכל בעת הכינו /
חשב בתחבלות הנו / ויאמר בלבו: הנני סכל / אם
מאשר הכינותי לא אכל / והמאכל מתוק לחך / ואני
לא אהיה רק לחך / והעגור בפעם אחד / יבלע כל
המאכל יחד / ויועץ בערמתו / אשר היתה קרובתו
/ והמאכל דק דק טחן / ויפרשהו על השלחן /
ויקרא העגור לאכל / ולקחת בפיו ולא יכל / כי
לאכל דק מפזר ומפרד לא נסה / ועין השעל עליו לא
חסה / וישאיר את השלחן ריק / כלחך השור את ירק
/ והעגור שב יגע ויעף / לביתו רעב וסר וזעף / ולא
דבר עם השעל / לא במרד ולא במעל / אך בכליותיו
ישתונן / ולנקם נקמתו כונן / ובהיותו
בתחבלות יועץ / ותורהו מזמתו עץ / מלא נקבים
בכל עבריו / וימלא טרף חוריו / והמאכל אשר הכין
בנקב יתקענו / עמק עמק מי ימצאנו / לו יהיה אין
זר אתו / ויקב חור בדלתו / דרך אשר יבא
בגבולו / שם מנה את מאכלו / ויקרא אל השעל
ויגיעהו / לבא עמו ויראהו / המאכל ויאמר לו:
הרי שלך לפניך / אכל כטוב בעיניך / והשעל לא
יוכל שם ללחך בלשונו / לחפשי ישלחנו תחת
שנו / לא בלשון ולא ברגל / ועל זה ישמח ויגל /
העגור אשר להוציא יוכל / ובתאותו לעיני השעל
אכל / והשיב לשעל כגמולו / כאשר עשה כן יעשה
לו.

והמשל במבקש תחבלות / וסבות
ועלילות / להונות את רעהו / וזה תם וישר הוא ויושב
לבטח אתו / בפעלתו ובמתכנתו / זה ירקח כמהו
/ בדרך אשר הורהו / אשתוללו אבירי לב / וטפש
לבם כחלב / חשב למצא טוב ועשה רע / איך יקצר
חטים קוצים זרע / חרש על רעהו אידו / כלי מפצו
בידו.
