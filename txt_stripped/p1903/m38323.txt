


	
	משל עד: איל וכלבים
	






המתגאה ברב חילו / יפל ואין עזר לו



האיל על אפיקי מים כבאו / רצה לשבר
את צמאו / ויתן את עיניו במים / וירא בצלו כי
הוא בעל קרנים / ויאמר: עשרתי מצאתי און לי / מי
יערך אלי להתהלל כמהללי / קרני לא כקרני
השור / אך נטועות במצחי במישור / וסדורות בגרם
המעלות / ובחן ויפי שתיהן כלולות / ומי הוא אשר
לפני יתיצב / משפט מות לו חרוץ ונחצב / לאשר לי
עליו תלנה / בהן אנגח ימה וצפונה / וכאשר
רגליו במים נדים / הנה כלבים הלכים צמדים /
ומנהיגיהם מסיתים בו / והאיל לנוס בחפזה אמר
בלבו / וינס ולא פנה אל אם דרך ואל המשעולים /
אך אל עבי היער ואל סבכי החרלים / ותכשילהו
עצתו / וינקש בגאותו / כי בשיתו אל הסבכים
עיניו / והנה נאחז בסבך בקרניו / ושם הכלבים
הדביקוהו / והצידים נתחוהו / וינקש בקרניו
אשר היו לו לאון / לפני שבר גאון / וכל אבריו כהם
לנתוח נמסרו / ואם הם על כלם בעיניו ישרו.



והמשל תהי עינך ראה ואזנך שמעת /
ואל תירא לגשת אל עץ הדעת / פן תהי מהיר
ברעיוניך / לקרב לכל אשר ישר בעיניך / כי את
אשר תקרב ותבחר ברעיונים / לפעמים הוא כחמת
תנינים וראשי פתנים / ואם מאהבה וגאוה בהם
תנקש / יהי לך לפח ולמוקש / ואשר תחשב
כמדרך נעל / עז ותפארת בלבך יעל.
