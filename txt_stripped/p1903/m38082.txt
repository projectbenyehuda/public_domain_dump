


	
	משל כא: זאב ועזים
	




מרבים דברי פה לבם מקשים / ובתוכם ישימו ארבים מוקשים



עז מגדלת עזיה בעדר / וסביבה בית
חמה וגדר / ועת אשר ישכים האדם לחרש / עז אחר
כל ירוק ידרש / ולתור בהרים מרעה / ותאמר לתיש:
דעה / כי רחוק מזה מרעיתי / לא אשוב עד מלאתי /
בטני וקרבי הרזים / להספיק חלב לעזים / סגר
דלתיך בעדיך וחבי / לא תפרד עד עת שובי /
ולפתח אל תשלח ידיך / עד אשר תראני לפניך /
אני אעביר כל טובי על פניך / ויען התיש לאמר /
נלכה יחד נרעה נשמר / וילכו יחד גם שניהם /
ולעז קטנה אחריהם / אמרו לסגר הדלת / והיא
אחריהם נעלת / וסגר הבית מבוא / והזאב אורב
במחבא / אשר להשחית כונן / ובכליותיו ישתונן /
לבא בעדר ולערף / ולצוד ציד ולטרף / וידע כי אי
שם העז והתיש / ולא אדניהם האיש / ויתנכר
ובפיו לועז / כאשר היתה אם העז / ויאמר: פתחי לי
כי רב חלבי / תוך שדי אשר לך אביא / נלאיתי
כלכל ולא אוכל / להביא בהם רב מאכל / ותען העז:
הקול קול אמי / אך יראתי כי לא עמי / עמד אחר
כתלנו / במדבר ארב לנו / לא אפתח עד הערב /
מפני הזאב הארב / פן יתחלף לא־עמי באמי /
וזאב חרמי באב שלומי / ואל יאמרו מתי סודי /
הנבואה החלה בעדי / וגר זאב עם כבש ונמר עם
גדי / וירא הזאב כי בדברים לא יכל לה / ויאמר:
אם לא תפתחי אורידך ביגון שאולה / ולבעבור כי
לבך מלא חמס / אפרץ גדריך ותהיי למרמס / ואכין
בדלת בלהטי מפתח / ותאמר העז: אל יתהלל חגר
כמפתח / לא בפתוי ולא ברגז ולא בתאנה / אצלי
לא תכנס הנה.



והמשל באנשים / אשר לבם ודבריהם
קשים / עם מרי בנים כחשים / לובשי אדרת שער /
למען כחש בשער / בתוך מרמה שבתו / איש איש
ממלאכתו / ועת צרה יתנכרו / בלב ולב ידברו /
כי אשר יראה אהבה בענינו / הלא שקר בימינו /
ואם לא יהי הגר ביניהם כמוהם / לגנב למגנבים
לבביהם / ולהערים המערים בתרמית לבו / ולהכין
לו ארבו בקרבו / ולחשב ערמות כנגד ערמות /
ומזמות כנגד מזמות / לא יוכל ביניהם להתפרנס /
כי אם במעשה נס / באשר הרעים נחו / והטובים
שחו / והשקרים עמדו / והאמונות מעדו.



ואשא משלי ואמר:



בית ברכיה מאיש מרמה / יתפרדו לא יתלכדו

כאשר שבעה כוכבים אל / עת עוף קורא לא נצמדו.
