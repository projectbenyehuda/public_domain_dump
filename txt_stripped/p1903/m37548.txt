


	
	משל ו: שעל ודגים
	




רבו יועצים לפי דרכם / להעלות זולתם בחכם



שעל הלך על שפת הנהר / וירא כי דג
אחד מהר / לנוס ודג אחר רדפו / עד כי השיגו
ובאיבה הדפו / וינצו שניהם / ואין מציל ביניהם /
זה אל זה ירוץ בחמה / ולא דבר אתם מאומה / אכן
נגש אל המים / אולי ישלח בם שנים / או ילכדם
ברשת ערמתו / אך המים חוצצים בפני תאותו /
ויפן משם אל מקום אחר / ולדבר אליהם לא יאחר /
ובראותו שם את הדגים נצים / והגדולים את
הקטנים לוחצים / ואדיריהם נשכו צעיריהם /
ותכבד המלחמה ביניהם / ויקרא להם: הכל אחד
מכם סכל / כי השקט לא יוכל / הזאת הברית אשר
ביניכם / הקהל חקה אחת לכם / כי כל אחד ישחית רעו
/ הגדול מחברו בולעו / ובשצף קצף אותו ישסף /
אם את כל דגי הים יאסף / לאמר לי: עלינו מלכה / לא
אצא מן המערכה / כאשר כל היום תפחדו ותרהו /
ונלחמו איש באחיו ורעהו / ודרכם לנגדם ירט /
ממשך וממרט / ועל זה שנו תעו / דרך שלום לא
ידעו / שמעו נא את טעם מלי / הטו אזנכם ולכו אלי
/ צאו משם ובואו הלום / בשמחה תצאו ובשלום /
ואם תשמעו לעצתי / וברכתם גם אתי / צאו מן הים
ליבשה / ויחדו את הארץ נירשה / ואז שלום לכם
ירב / ולא ישא גוי אל גוי חרב / כי אין בנו פורץ
פרץ / נחה שקטה כל הארץ / הן אתנו משוש פראים /
להחיות לב נדכאים / וכל שכני ארץ בחדותם /
ירננו על משכבותם / יומם ולילה לא ישבתו / לא
ירעו ולא ישחיתו / ויען האחד: אם מלך תמלך
עלינו / האם תשפת שלום לנו / הן בהיותנו במי
מנוחות / וסביבנו המשפחות / בשלום שודד
יבואנו / יעלה הכורת עלינו / וממצודות בני אדם
אנחנו נמוגים / ודיגונו רבים דיגים / וצדונו
צידים רבים / ונהמה כלנו כדבים / ואם ידעת
מקום חנותנו / גם אתה חלית כמונו / אם אתה
בערמתך תאכל שד / כהתמך שודד תושד / כי פתע
יקומו נשכיך / ולא תצליח את דרכיך / ואיך אתה
רשת מרמה עלינו שטח / ובארץ שלום אתה בטח /
הלא לעוף השמים ולדגי הים ולחית השדה /
לכלם יש ארב וצדה / כי גם בני אדם נצים מקנאה
אשר ביניהם / אבל גבה מעל גבה שמר וגבהים
עליהם.



והמשל להזהר מן היועצים לפי דרכם /
ולהשמר ממתק חכם / כי בפשע הארץ רבו חנפי
לב / ופני הדור פני כלב / ולהזהר כל איש
ממוקשו / החכם עיניו בראשו.



ואשא משלי:



פני הדור פני כלב / ודבריהם בלב ולב

כמו חמאה דבר פיהם / והמחשב כמו חלב

ומצו את שדי גילי / כמו יונק בעת חולב

מרגלים הם, אבל אין בם / יהושע ולא כלב.
