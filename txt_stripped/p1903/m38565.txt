


	
	אריה ובנו
	




טוב להזהר מהרשעים על כל פנים / כי מעשי אבות יעשו בנים



אריה מלך בחיות / שכן במעונות אריות
/ לעשות משפט עמו / בנפשו יביא לחמו / כאשר
עשו אבותיו / ומחנק ללבאתיו / ובחליו אשר ימות
בו / והנה נפשו בחלה בו / ויקרא לבנו בכורו /
אשר היה בעירו / המשל בכל אשר לו / ויזעק
הכפיר לקולו / ויקרא אליו: בני / ויאמר לו: הנני
/ ויקד וישתחוה למולו / ויצוהו ויאמר לו /
בני, שמר מצוה והיה / משל בשיך והיה / גבור
ובן חיל / ראה כבש ואיל / לכף רגלך מדרך /
בעודך נער ורך / וכשיגדל כחך / לא תקצר ידך
ורוחך / וכרעת ורבצת כלביא / לצוד ציד להביא /
תקל אליהם ברגל / אל מעשה העגל / ואל עיר בן
אתונות / לא תעזב חנות / כי באף תקפץ / כאשר
תחפץ / ומסבך תעל כאריה / וכל אשר ימרה את
פיך לא יחיה / אז תהיה מלך / על כל ארבע הלך /
תשמח ותגל למשלח רגל השור והחמור / וכל
בהמה לשמור / דרך מבא השמש מוצאה / תמיד
להיותה לאכלה / ולא לסחורה רק לאחזת נחלה /
וכאשר תריד ורבצת לא יבצר ממך מזמה / מכל
חיה ובהמה / מן הטהורה / לך אסורה / הארנבת
והשפן / תשחק לשד ולכפן / וגם מן החזיר / אל
תעשה נזיר / כי בשרו שמן / וענית ואמרת: אמן /
וממך לא יחסר / ובכל אות נפשך תאכל בשר /
ומכל הבהמה / תוציא ביד רמה / ובכל חיה / ידך
הויה / ותשחית את הודם / אך השמר מבני אדם / כי
רבות תחבלותיו / ילכדך בשחיתותיו / ואם עצמת
ממנו / במרמה לא תצילנו / כי ימלא ידיו בקשת
/ או מכבר מעשה רשת / לשים רגלך בסד / פנת
יקרת מוסד / פח ופחד ופחת / תרד לבאר שחת / אתה
לא תוכל לו / כי אין על עפר משלו / ומי לו דמה / כי
כל הבהמה / כלם לאדם עבדים / גם הסוסים
והפרדים / חמד אותם לו לשבתו / ישים למרכבתו /
ויום יום יעמס על החמורים / גם יטבח טבח והכן
ויחרש בבקרים / חזק בגזת הצאן / שור וארי וצאן
/ נפשם עליהם תאבל / ושה לטבח יובל / על כן אותך
צויתי / לשמע לעצתי / ולשמר מצותי / פן תשחית
אחר מותי / אם תפר דברי אמת / והנה אנכי מת /
כדרך כל הארץ / ופרצת עליך פרץ / אל תערץ ואל
תחת / אך בלי רדת שחת / תהי נפשך שמר / השקט
והשמר / מחפר גומץ / וחזק ואמץ / ואחרי כן צוה
המלך את בנו / להושיבו על כנו / לעיני כל שריו /
למען יהיו בעזריו / ואחרי אשר צוהו / נפל עליו
וישקהו / ויהי אחרי דברו יצאה רוחו / וכהתה
עינו ונס לחו / ולבו עליו נשפך / ולמשחית הודו
נהפך / ויאסף אל עמיו / והבן נכמרו עליו רחמיו
/ ויצעק: אבי אבי / רכב איל וצבי / והוד המלוכה /
ראש המערכה / כי תסגר בארץ ציה / אויה לנו
אויה / וישא קולו ויבך עליו וישק לו / ולקולו
חרדו משפחתו / וכל עזריו אתו / ויבאו כלם
לספדו / הוי אדון והוי הודו / אם ישנה עז פניו / מה
יעשו יתומים בניו / אחריו כעורים נגשש /
ומדוע בשש רכבו לבוא / וינס במחבא / למה
מעלינו סר צלנו / לו יוסף ישטמנו / והשב ישיב
לנו / את כל הרעה אשר גמלנו / אוי נא לנו כי
חטאנו / גוענו אבדנו כלנו אבדנו / אם נפל
מלכנו / אשר יתהלל באמים / ויספדו לו העמים
/ ובנו לא קבל תנחומים / ויעש לאביו אבל שבעת
ימים / ויהי בעבר האבל / מכשל כח הסבל / כי עוד
לא יכירוהו / וישאו המת ויקברוהו / בסלע
ביתו הרמה / ויקראו שמה במה / כי היא למצבה
תהיה / לראות את מפלת האריה / ויהי אחרי קברם /
כבודם ויקרם / וישב העם וילך / וישיבו בן
המלך / וינחמו אתו בביתו / ויתקעו כף לאהבה
ולעבדה אתו / וכשב המלך לביתו / ישב על כסא
מלכותו / כדת וכהלכה / ותצלח בידו המלוכה /
ויפן אנה ואנה בהמוניו / וכל החיות לא יעמדו
לפניו / ירופפו ויתמהו מגערתו / וכטוב לב המלך
בשמחתו / ויגבה מאד רוחו / ויפתח את ידיו /
ויעש משתה לכל עבדיו / והשתיה כדת אין אנס /
וישאו על ההרים נס / להשביע כל חי רצון / חמאת
בקר וחלב צאן / ויכסו פנים בחלבם / עד כי היה
לבם / אל מלכם נפתה / ויהי בימי המשתה /
ויקרא המלך למשרתיו / ויפקד שרי חילותיו /
לשמח הבחורים / לבטח בגבורים / לשים שרי
צבאות / שרי אלפים ושרי מאות / כלם נקבצו יחד /
ויהיו לקשר אחד / דבים ונמרים / אריות וחזירים /
סוסים ופילים / הרכש והגמלים / ויאחזו שעלים /
לבא יחד במקהלים / שנהבים וקופים / לאלפי אלפים
/ עד אין מקום לעמד / ויהי המחנה כבד מאד /
והמלך לראות חכה / ויעש שם סכה / ויקרא
למשפחתו / משרתיו ובני ביתו / ויאמר להם: ראו
/ איזה מהם לפני תראו / בחרו לי הגבורים / להיות
בצבא שרים / להשחית עיר ופלך / ויעשו כדבר
המלך / ויתנו הנמרים משנים / והדבים סגנים /
ופחות הזאבים / להלחם בכלבים / והצבאים היו
רצים / והשעלים יועצים / כי ערם יצא מבטן אמה /
ויקומו ביד רמה / וישמחו וישירו / וישמיעו
קול ויאמרו / יחי המלך אדנינו / וירא העם
וירנו / ויפלו על פניהם / והמלך שמח עליהם /
ולאהלו איש הלך / מבית נכת המלך / כי בחדש
האביב / פתח אוצרות אביו / הרבה מעדנות / ויתן
מתנות / לכל באי הלום / ויברכם וילכו לשלום /
וילך העם לדרכו / והמלך נשאר בסכו / עם
פרחי משפחתו / מחזיקי ממלכתו / ויבא אל
אוצרותיו / ויוצא כל אוצרות אבותיו / וירא כי
המעיט מטמונות / וישח במעונות / ויאמר: אלביש
גבורה / ואעבר המדברה / ואעיר כל חמתי / על אשר
יבא לקראתי / אשיב נקם לצרי / ואמלא טרף חרי /
ואם אדם יקרני / הלא אבי צוני / להשמר מפניו /
כי אורו עיניו / ראה מרחק / כי התחתון ידחק /
והעליון יכסיף / כה יעשה לי וכה יוסיף / כי אעביר
מצותו / הלא כאיש גבורתו / ואם גבורתו פחז
כמים בחונה / ותקרא אם לבינה / אך דבר שפתים
למותר / ואני חכמתי יותר / טרם יערים אכנו /
בדבר ואורשנו / ועצמותיו אשבר / טרם כלה
לדבר / דבריו אל לבו / ויחגר את חרבו / ויצא
מן היער / ופיו פער / וימצא בצאתו / פרא למד
מדבר וישאג לקראתו / והפרא לעמתו / אליו
מתאנה / מהמונו לא יענה / ויפלא בעיני המלך /
ויאמר: מי ההלך / אשר לא חזר לקולי / והוא יושב
ממולי / הזה האדם אשר אמר אבי / עתה אעשה אשר
בלבבי / ויבא אליו ויכהו / והפרא לא הכירהו /
ועד לא השיב ידו / שפך לארץ כבדו / ועד לא מות
יאכילנו / צעק: הושיע אדנינו / המלך האריה /
איפה אתה ואיה / טובך וחסדך / להציל עבדך /
מיד מכהו / כי איש חיל הוא / ויהי בשמעו / וידע
כי נגעו / ויאמר לו מי אתה / אשר בי מרדת / הלא
אני ארי המלך / אשר עם הנשר הלך / וכשמעו
מליו / ויפל לפני רגליו / וישתחו לפניו / כי
הוא אדניו / ויאמר: אני הפרא עבדך ואתה הרף,
חטאתי / כי לא ידעתי / המלך והמוניו / והמלך
לאט את פניו / אשר נגאלו בדם / ויאמר: הראית בן
אדם / מה מעשהו ומשפטו / כי הוביש מבטו /
ויאמר: לא ידעתי / ואת קולו לא שמעתי / לא
עברתי בארצו / ולא באתי מלון רבצו / ולא ידעתי
אנה / כי לא ראיתיו עד הנה / אם הוא נכבד או נבזה /
ועתה לך מזה / ובאת אל ארץ המישור / ומצאת חמור
ושור / המה הכירהו / כי מעולם עבדוהו / תמו
דברי באזניך / בין תבין את אשר לפניך / וידבר
עד כה / וילך הארי לדרכו / וימצא ערוד ופיל /
גבה קומתם השפיל / ויצא משם לפרץ פרץ /
להתהלך בארץ / ויהי בצאתו מן היערה / וירא
הסוס אשר לו גבורה / יפה קומה / לבוש רעמה / חלק
משמן תורק / מרוצץ כברק / ומרגליו עד גבו / כל
מום לא היה בו / ובראות האריה המראה / בערה בו
קנאה / והסוס לבש אימה / ביום חמה / לאמר: מי האיש
הלזה / וקרא זה אל זה / מר צעקים ברב חילם /
ותבקע הארץ בקולם / ובהגיש האריה אליו / ויקם
הסוס על רגליו / לפל עליו להשחיתו / והאריה
בקלותו / קפץ ונמלט / ובו לא שלט / ויהי מאחריו /
להיותו בעכריו / והוא מאד נזהר / והסוס מהר / ברב
גאונים / לאחור ולא לפנים / וקנאה כמעיל יעט /
וישמן ויבעט / מרגליו אחרים / ויכה האריה
בחלצים / ותרד המכה על זרועו / וירא הארי כי
נגעו / ויאחז בזנבו / וחמתו בערה בו / וירץ אליו
בחרונו / ויט זרוע ימינו / וישלם לשנאיו אל
פניו / אימה סביבות שניו / וישברהו כקנה רצוץ /
אז הלמו עקבי־סוס / וטרם שב אפו / יהגה האריה על
טרפו / וטרף זרוע אף קדקד / ויפל הסוס ויקד /
על אפיו בכריעה / וצעק: הושיעה / הנה עבדך אני /
אשר עצמת ממני / ועתה רוחך לא יבאיש / ויאמר:
האתה האיש / אשר רב חילו / ויאמר: לא / כי אם הסוס
אנכי / אשר לא נמצא כערכי / ואתה אשר חזקת /
מי אתה / ויאמר: אני הליש / ואיה אפוא האיש / אם
נפשך קצה / מדוע לא הכיתו ארצה / ויאמר: בי
אדני / אני מרכבת אדני / לעשות חפצו / ולכל עם
ארצו / ויאמר המלך: אפוא הוא / ויאמר: הנו על
שדהו / אסף מעשהו / ואם למלחמה יצאת / לך מזה
ומצאת / הנו אחרי האכר / אדם בל ילין ביקר /
ויבא אל המישור / וירא האדם אשר יקשור /
בעבודתו לגמור / וירא החמור / יגוע בבאו / רבץ
תחת משאו / וגם הצאן יחד / נפל עליהם פחד /
ויאמרו: אדנינו איה / ויהי כשמע האריה /
ויגש עד הלום / ויקרא להם לשלום / וישאל להם
מה מעשיהם / ומי אדניהם / ויאמרו: עבדים אנחנו /
לאדם אשר הקננו / ופת־בגו כל אחד ממנו נשא /
ולא נדע מה נעשה / ונפשנו קצה בלחם זולתו / טוב
לנו עבודתו / מהיות עבדים לך / ויאמר: אנה הלך /
הגוי הגדול הזה / ויאמרו: האיש נסע מזה / ביער
לחטב עץ / כי בעצתו יועץ / משפחתו רכבו עמו /
עודם מדברים עמו / והנה השעל לקראתו / וישמח
המלך לראותו / ויאמר: זה מבשר / ואנכי לא
אפשר / ובהגיע השעל אליו / ויקד לרגליו /
וישאל: השלום אדני / ויען: שלום בני / ויאמר:
המלך, יצאתי קן־צפר ואפרחיו ארבתי / לבעלי כנף
/ לא אשאיר ענף / ויען המלך ויאמר: והאדם לא
נשמר / שם איננו / או חזקת ממנו / ויאמר:
ארבתי / ולא בחזקה לקחתי / כי מי אני ומי עמי /
ויאמר המלך: שב עמי / עד אמצאנו / אם לא בחמתי
אכנו / פעם אחת ולא אשנה / והמקום מקום מקנה /
תרעה במרעה שמן / ויאמר השעל: אמן / כדברך
אדני המלך / אני אחריך אלך / אך השמר ממוקש /
האיש אשר אתה מבקש / בסבכי יער תמצא / כי
עתה מביתו יצא / ואתה להלחם תקרב / לשלל שלל
רב / וילך האריה ביד רמה / והשעל נשאר
שמה.



והמשל להזהר מן הרשעים על כל פנים
/ כי מעשה אבות יעשו בנים / כאריה אשר הרג בקר
ושחט צאן / הן בטבע הן ברצון.
