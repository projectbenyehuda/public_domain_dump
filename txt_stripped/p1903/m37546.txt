


	
	משל ד: תרנגל ואבן יקרה
	




כסיל את עיניו מחכמה יגרע / מואס בטוב ובוחר ברע



תרנגול עלי אשפה עלה / והדמן
ברגליו פזר וגלה / למצא תולעים למאכלו / משם
חפר אכל ברגלו / וימצא ישפה אבן טובה / ולנקר
עוד באשפה לא אבה / ויאמר: חשבתי למצא יתרון
טוב בעמלי / תולע או זבוב אשר ייטב לי / ועתה
ישפה הנה ראיתיך / מה יתרון לי כאשר מצאתיך /
אמנם את נחמדת למראה / ואם ימצאך עשיר גאה /
תהיי בידו נקפצת / ובזהב טוב משבצת / ותאירי
על אורך שבעתים / אך עתה בין רגלים תהיי רמוסה
ודרוסה / כי נפשי עליך לא חסה / באשר לא מצאתי
בך חפצי / טוב לי תולע או חצי / להשיב נפשי הרעבה
/ ממציאת אבן טובה.



והמשל הרבה אחזה / באיש אשר מכבד
יבזה / ועיניו מצדיק יגרע / מואס בטוב ובוחר ברע
/ ככה משפטו וחקו / לא יקרבנו רק בשעת דחקו
/ ואמר שלמה במשלו / בז לדבר יחבל לו.



ואשא עליו בקולי / לקללנו במשלי:



ולב איש יהפך מרצות / חכם לבב במועצות

כתרנגל יעזב ישפה / בתוך אשפה בטיט חוצות

בלבו יפרח נגע / וישוב אחרי הקצות

והנכבד יהי נקלה / ומלא בוז ונאצות

ובוז לבין להתמשל / והצואים למחלצות

ואבני גיר מנפצות / בתוך זהב משבצות.
