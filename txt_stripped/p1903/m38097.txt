


	
	משל לו: זאב ובהמות
	




אשר רגליו ללכת בגזל נסה / עינו על הון זולתו לא חסה



זאב אשר היה משנה למלך ושר / מגמת
פניו שם לשחת כל בשר / גזל וחמס טרף ועקר / כל
הנמצא ידקר / והחיה והעוף והבהמה / אשר שלח
במשפחותם מהומה / לצעק עליו הלכו לארי / ויאמר:
אך רע ומרי / אם כדבריכם שניו גלה / הכצעקתה
הבאה אלי עשה כלה / אשפטנו עד אשר מלהשחית
יפנה ערף / ומשניו אשליך טרף / ויגד לזאב:
בא אלי ביום מחר / הקשיבה ועשה אל תאחר / ויבא
אליו וידבר עמו הארי עזות / ויאמר לו: למה
עשית כזאת / אל תוסף עוד בבהמות לערף / ולא
בחיות טרף לטרף / ואשר לך למאכל המה / כל נבלה
וטרפה מן העוף ומן הבהמה / אשר תמצא מת בשדה /
ולחי לא תהיה ארב וצדה / ואם לשמר דברי לא תוכל /
השבעה לי פן תאכל / בשר עד שנתים ימים / לכפר
פשעיך כתובים וחתומים / וזה משפטך אשר לך
אחזה / וישבע לו על הדבר הזה / לבלתי אכל
בשר שתי שנים / מיום יטרף כל אשר בחיות חנים
/ ויפרד הזאב משם וילך לדרכו / והארי נשאר
כמלך בסכו / ולא אכל הזאב מכל הלך על ארבע /
משמרו את השבועה אשר נשבע / אם לא מצא בשדה
או במסלה / טרפה משלכת או נבלה / ויהי היום
וירעב ויפן אנה ואנה / וירא והנה רחל שמנה /
נחמדה למראה וטובה למאכל / ויאמר: לשמר מצוה מי
יוכל / ויריבון בו רעיוניו / אם ישית אליה פניו /
ויאמר בלבו: אם תנצחני תאותי / עוד להכות כל
חי כאשר עשיתי / מן היום ההוא יהיה לי שתי שנים
/ לבלתי אכל בשר מיום אל יום נמנים / זה הדבר
אשר למלך נשבעתי / לקים השבועה בלבי
תרתי / מנין ימות השנה שלש מאות ששים וחמשה
/ פתיחת עיני יהיה יום והעצימה לילה חדשה /
ויפתח עיניו אחר אשר עצם אותן יחד / ויהי ערב
ויהי בקר יום אחד / וכן עשה לשתי שנים במספר /
וסר עוונו וחטאתו יכפר / ועינו פונות לעז אשר
בחרו / והביטו אליו אשר דקרו / ויאמר: הן
קדמתי כפרתי לפני הטרף / ויאחז את העז בערף /
וינתחה ואכלה / כאשר מנהגו בתחלה / ועוד
ידו נטויה על החיות / כימי עולם וכשנים
קדמניות.



והמשל באיש אשר לגנב ולגזל תמיד
נסה / ועינו על הון אחרים לא חסה / שלל ובוזז הונם
ועמלם / ואם ישבע במעמד כלם / יערים בלבו על
שבועתו / ויחשב להיות נקי מאלתו.
