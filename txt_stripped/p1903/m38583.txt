


	
	משל קיג: איש וזאב
	







אשר עינו ולבו על בצעו / פיו יגיד רשעו



איש לזאב אותיות מאלף / ויאמר לו:
אמר אל״ף / זאב אחריו אל״ף ענה / עוד אמר: בית אמר נא
/ הזאב מוצא שפתיו שמר / בי״ת וגמ״ל כמוהו אמר /
ויאמר האיש: שמע נא באזניך / את אשר אערך
לפניך / ותכיר האותיות לחבר / למצא חפצך ודבר
/ ועת תחבר אותם יחד / והיינו לעם אחד / אל״ף בי״ת
כאשר אעשה / זאב ענה: הנה השה.

והמשל באשר עינו ולבו על בצעו /
פיו יגיד רשעו / ושפתיו יענו בו / להתגלות זדון
לבו / ותצא מבטנו הרשעה / עת תמצא בפיו רעה /
ויש מחשבתו נכרת מתוך מעשיו / מאס ביעקב
ובחר בעשו / וצדיק צדקו יקדם / וסלף בגדים
ישדם / ומשלי שלמה לא יגרע / הלוא יתעו חרשי
רע.
