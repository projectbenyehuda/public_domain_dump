


	
	משל לד: צפרדע ושורים
	




המתפאר בפעלתו / יזהר פן תגלה עליו ערותו



צפרדע ראתה שורים חרשים / אצל קנה
וגמא אהלה נגשים / ותמהר ללכת לקראתם /
ותשאל שמם ושם אבותם / כי היא להכירם רוצה /
ותאמר: לי החכמה והעצה / למצא לכל פצע ומחלה /
מזור ורפאות תעלה / במראות השתן ובדפק הזרע /
אכיר אם לטוב אם לרע / חולים לרב הוצאתי לרויה /
כאשר פתרתי כן היה / ושור זקן הביט אל אותה /
המתהללת ברב חכמתה / ויאמר לה: אם דברך אמת
/ למה מראך דומה למת / מכל רמש תארך משחת /
כאשר מחר תרדי שחת / וחכמה נכרת במעלליה /
החכמה תחיה בעליה / אין בך אדם רק כל ירק /
מפניך לא אחשך רק / וחפרה הצפרדע ובושה /
ותשם בין שתי ירכותיה ראשה / ותלך להחבא
במרוצה / בסתר קנה ובצה.

והמשל במתפאר בחכמתו ובפעלתו /
יזהר פן יוכיחו עליו חרפתו / כי טוב להחריש עד
יגידו אחרים הודו / מלהמיר בקלון את כבודו / ויש
אשר אחר יהללנו / ולפני גדולים ינחנו / ואמר
החכם: יהללך בבאך ובצאתך / זר ולא פיך נכרי
ואל שפתיך.
