


	
	משל פ: אשה ובעלה ופרש ומשנה
	




מעט מהחיים יעלו על לבביהם / הנאהבים והנעימים בחייהם



אשה ספקת כפיה / יזלו מים עפעפיה
/ ובמר נפשה בדד ישבה / מתפפת על לבבה / על
בעלה אשר מת ונקבר / והנה אחר המדבר / פרש
היה תלוי על עץ / ומשנה למלך ויועץ / מעיר אל עיר
מתהלך / ויזעק מטעם המלך / כי כל אשר יפיל
התלוי / גופו יהיה באש קלוי / וירא פרש אחר
אחיו הנתלה / כאשר הנה מן המסלה / זה הדבר מאד
הכלימו / אחיו הוא מאביו ומאמו / ולא יפלס נתיב
לאפו / וישם בלילה את נפשו בכפו / וינתק את
מוסרותיו / ויקברהו עם אבותיו / ויירא מאד
לנפשו / פן יסירו מעליו ראשו / כי מצות המלך לא
שמר / מפחדו כל בשרו סמר / ויברח להחבא בית
האשה / המבכה על בעלה במר נפשה / וידבר על
לבה במאמרים נעימים / דברים טובים דברים נחמים
/ לאמר: לי תהי למנה / וארשתיך לי באמונה / שכחי
יגונך ומבעלך תנחמי / ובאי שכבי עמי / והיא
גם היא אמרה / מזלך לפניך הקרה / האשה אשר
עתה נפלה לגורלך / שלום תענך ופתחה לך /
ולדעת כל מעשיו הציקתהו / ויגד לה את כל אשר
קרהו / כי שחה נפשו עד עפר / באשר בלילה ההוא
מצות המלך הפר / על כן ימות בלא עתו / ולא ישוב
עוד אל משפחתו / ותאמר לו האשה: אל תחת / בעלי
הנקבר נתלה תחת / התלוי על העץ ההוא / בטרם
יכיר איש את רעהו / ומי ידע אשר נעשה במסתרים /
אין אמר ואין דברים / עתה תשמע לעצתי / כי על כן
באת בצל קרתי / ותוליכהו חיש אל הקבורה / אהבה
מקלקלת השורה / והקבור מקבורתו הסירו /
ויעשו כאשר דברו / ותלך ותתע במדבר /
לתלות את בעלה הנקבר.



והמשל מעט מן החיים יעלו על לבביהם /
הנאהבים והנעימים בחייהם / ויכתתו כל אהבתם
טחון / לכל אשר ימות אין בטחון / מאשר חבקוהו
וינשקוהו / כמת מלב ישכחוהו / כי לא יתנו לב
להזכיר את צדקתו / ולא לשמר את מצותו / האמר:
זכר נא את חסדי / אל תשקר לי ולניני ולנכדי / מת
האיש ויכסוהו בעפר / והחי את מצותו הפר.
