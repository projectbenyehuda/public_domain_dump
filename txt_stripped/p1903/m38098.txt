


	
	משל לז: נשר ועופות וארי וחיות ועטלף
	




מדבר בדברי חלקות / אל החזק יתחבר במעשקות



הנשר מלך על כל בעלי כנף / הן בארץ
הן בענף / והארי על כל בהמה וחיה / מכל קצוי
ארץ נשיה / ויערכו הנשר והעופות מלחמה / עם
הארי ועם כל חיה ובהמה / ויאמר העטלף בלבו /
כי דבר בליעל יצוק בו / אמנם כעוף יש לי כנפים
/ וכחיה וכבהמה יש לי רגלים / ולא אדע איזה חיל
זה על זה יגבר או מי ימות או נשבה או נשבר /
אראה ואהיה צפה / החזק הוא הרפה / ועם אשר תגבר
ידו ושלומו / והיתה ידי עמו / וירם מן החיל
ויעמד למולו / לדעה מה יעשה לו / וילך ויעמד
על עץ גבה למרחוק / ויאמר: עת מלחמה לעופות ולא
עת שחוק / אל הנצחים לעזר אשא עיני / אשקטה
ואביטה במכוני / וירא כי חזקו הבהמות / ובהמון
העופות נפלו מהומות / וירד מן העץ וילך על ארבע
/ ולעזר במאמצי כחו נשבע / אל כל חיה ובהמה /
ויאמר: חזקו ועשו מלחמה / אל ימלטו בעלי כנף /
כל אחד עליהם יתאנף / וכאשר רפה היום לערב
ותחזק המלחמה / ויתגברו העופות ובהמון הארי
נפלה מהומה / ויאמר העטלף בלבו: אל עמי אשובה
/ ולא אוסיף עוד עם המון הארי לדאבה / ואעופה
ואשכונה / ורגלי תחת כנפי אטמינה / ויאסף
רגליו בין הכתפים / עשה יעשה לו כנפים / ויעף
חיש בין העופות / ויאמר: חזקו ידים רפות / לעשות
פצע וחבורה ומכה טריה / בהולכי על ארבע,
בהמה וחיה / ויכירו העופות את העטלף / ויאמרו:
מדוע ככה תתחלף / ללכת עמנו בקרי / בקר
ראינוך עם המון הארי / הלך על ארבע כמוהם / העת
אשר גברה ידיהם / ושלחת יד בשלומנו / ועוף
היית כמונו / וכאשר גברנו פרשת עלינו כנפיך
/ דם שנאת ודם ירדפך / מדוע בעורך התלת /
מרמה ואון פעלת / ויהי ארבך בקרבך / עד נמצא
עולתה בך / העת שבתך על עקב בשתך / זה יהיה
משכרתך / כ תחשב ממין שרץ / נע ונד תהיה
בארץ / כאשר התנכרת מרעיך / כה יהיה זרעך /
שחר ומרוט וקרח / עור ובינינו כארח / לא תלך עוד
נטוי גרון / אור שמש יהיה לך לעורון / קרח ושרץ
תקרא / מכל פגע תהיה ירא / ויקשרו עליו קשר /
ויביאוהו אל הנשר / ויאמר: הזה הוא אשר דמה
לבהמה וחיה / לבעבור כי ברוחו רמיה / יהי נא
מרוט וקרח / ומאור השמש בורח / ולערב יעוף כמו
נבהל / גם דור עשירי לא יבא לו בקהל / ולא יוכל
לעמד על רגליו / אם זרחה השמש עליו / וימרטוהו
ויקרחוהו / ועיניו עורו ויבהילוהו / ממגוריהם
אשר בם גרו / וכל עוף השמים ממנו יתנכרו /
ולא ידמה עוד לעוף ולבהמה / ושרץ יקראו לו על
האדמה.



והמשל באשר יראה אהבו או קרובו /
נפל ויוכל לו איבו / אל זה ידבר בחלקלקות / ועם
החזק יתאמץ בבצע מעשקות / ואם השפל יתחזק
ויתגבר / בלב ולב ישוב אליו להתחבר / באשר לא
ידע בשת וכלמה / ומן ההתחברות אליו יעשה מרמה
/ ויאמר לו: הנה אמצתיך עזרתיך / ובימין צדקי
תמכתיך / כאשר יד איבך עליך גברה / לא
התרפיתי ביום צרה / וכמקרה העטלף ימצא באחרית
/ לבלתי היות לו שם ושארית / ומישר על המחקה /
כל עשה רע לא ינקה.
