


	
	משל כה: כבשים, איל וארי
	




אל תהיה ללעג ולצון / בראותך כי אמריך לא יעלו לרצון



כבשים בעדר נשארו / והרועים הלכו
באשר בחרו / ולא סגרו הדלת בעדם / ורחוקים המה
מאדם / ויצאו מן העדר כלם / לתור בשדה מאכלם /
הנה והנה בהפזרם / ורעו כבשים כדברם / ועל
העשב לאכל סוד המתיקו / המה יצאו מן העדר לא
הרחיקו / והנה ארי נסע אחר המדבר / ולרוץ עליהם
יתגבר / וכאשר התבוננו לו זה אל זה קרא / אריה
שאג מי לא יירא / ויראו כי ארב הוא להם / ומנוס
אבד מנהם / ויאמרו אל האיל ההלך בראשם / לך נא
אל הארי אל הדרך אשר בא שם / ובחלק לשון
תפתנו / אולי יפתה וילך מעלינו / ויסע האיל
מהמוניו / וישת אל המדבר פניו / ובטרם יקרב
אליו / ויסיתנו בחלק מליו / ויאמר לו: אדני
המלך ברוך אתה בבואך / ויום טוב תמצא בעיני
כל רואיך / ויען לו הארי: יום טוב אשר אהיה מוצא /
ממך ומחבריך לי יצא / מה לך אלי לשלומים / שחת
דבריך הנעימים / המלך כמוני מדבר פיך יבשר /
אין שמחה אלא בבשר / אשר אכל מעליך / ולא מן
דבר פיך אהללך.



והמשל באשר יראה עת צרה / וכיסו
ותכונתו בעיניו לרב יקרה / מבלי תת לנפשו כפר /
ויחשב להמלט באמרי שפר / ולשון רך וחלקלקות
/ וכל דבריו קדחי אש מאזרי זיקות / ולא ימלט
בעת זעם / ויש אשר לו יהי נעם / באמרו: קח את
החיל הזה / וכספו וזהבו בוזה / כסף כסף רבה /
אל יוציאו עליו דבה / כי בעת צרה ישא ראשו / כל
אשר לאיש יתן בעד נפשו.
