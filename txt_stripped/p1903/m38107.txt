


	
	משל מו: חתול ועכבר
	




יש הצלה ומנוס / לשבועה במקום אנוס

חתול הלך לתור מאכלו / לעת בקר וירא
למולו / עכבר בכלי שכר מלא / ותקראנה אתו
כאלה / כי לצאת משם לא יכל / והחתול רעב לאכל /
שמח ויצא לקראתו / שמה תגלה איבתו / באיבת
עולם יאשם אשם / שנא הוא לו מתמל שלשם / על שפת
הכלי נחפז ברגליו / וארב לו וקם עליו / וקולו על
העכבר הרים / וישם לו עלילות דברים / לאמר:
צדקתי ורשעך אותך שם הדפו / הוי משכימי בבקר
שכר ירדפו / מהר אל תעמד וצא / וחפצי בך אהיה
מוצא / אבלך נהפך אלי לחדוה / באשר בשרך
בשר תאוה / והעכבר השיב מענה / אני נגש אני
נענה / אני מרוט וקרח / בלי טעם ובלי ריח / כי
נפלתי ולא קמתי / ונבעתי ונצמתי / ומשמן
בשרי רזה / מיום נפלי בבור הזה / ומרב רגזי
ועצבי / הכרת פני ענתה בי / וכבר חדש ימים / לא
בא בפי לחם ומטעמים / ואם גיל עלי בך צמח /
בנפל איבך אל תשמח / ואל תגמל עלי עתה רעה /
הצילני ואל אטבעה / ויהבי אשליך עליך / למועד
אשוב אליך / ואהיה בריא ויצמח שערי / ובשובי
תשמח בבשרי / אך תוציא מצרה נפשי / כי צפו
מים על ראשי / אל החתול שלח ידו / כי ישוב אליו
במועדו / ועד חדש אתו יחזה / וישבע על הדבר
הזה / אם יוציאנו ממצודה / לשמר את בריתו
לעמדה / והחתול אשר היה בעכריו / נהפך לו
להיות בעזריו / ואת איבתו שם לא זכר / והוציא את
העכבר מן השכר / ואחר אשר מן החתול נפרד / הפך
ערף ובו מרד / ולבו מכל טוב סועד / ויוחר מן
המועד / והחתול הגיד לו לאמר / ואתה את בריתי
תשמר / לשוב כאשר נשבעת לי / שמן ודשן
בגבולי / ואל תאחר פעמיך / כי הוספתי על ימיך /
ואם בתוך מרמה שבתך / תן נא בנך תחתך /
העכבר הודה בשבועה / אך אמר כי אנוס והרעה /
נגדו כאשר נשבע היה / ולצאת משם לרויה / קפץ
ונשבע אל יד נכר / והשבועה על ידי שכר / שכר
ואנס היו שם שניהם / וחי בהם ולא שימות בהם /
ובעלילה כזאת נמלט / אשר החתול בו לא שלט.

והמשל בהיות דבר בין שני אנשים /
שניהם יהיו חקרים ודרשים / פן יהפך הדבר על
ידי עלילה וסבה / להוציא שניהם מדבה / כי לא
יתקים באנס ושכרות / הדבר, אך יחשב למעשה
נערות / ואמר החכם לבנו / לב חכם לימינו / אם חכמת
תתגדל ותתנשא / אל תהי חכם בדברים אם לא
במעשה / כי דבר פי איש אל מעשיו לא דמה / אולת
היא לו וכלמה.
