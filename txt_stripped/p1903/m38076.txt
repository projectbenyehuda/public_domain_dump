


	
	משל טו: אריה, עכבר ורשת
	




טוב להיות טוב ומטיב לכל / כי יש עת וזמן לכל חפץ להיות יכל



אריה נרדם במקום מדבר / וידרך עליו
עכבר / עד כי הקיץ משנתו / ויפן לראות מי אותו /
אשר מלך גדול כמוהו / ברגלים ירמסהו /
ויתבונן כי בצדו / העכבר עמד לבדו / הוא היה
אין זר אתו / ויאמר: הגיעה עתו / ויקרא לו: עכבר
נבזה / משפט מות לך אחזה / כי הקיצותני ובזיתני
/ ולמה זה עכרתני / זדון לבך השיאך להיות עלי
צעד / ואנכי היודע ועד / ולא תוכל להתעלם / עתה
גמולך אשלם / ויען לו העכבר / בלב מר מאד
ונשבר / הלא אנכי נבר ובר / אחלה נא בכל אוני /
יכנסו דברי באזני אדני / ואל יחר אפך בעבדך
/ ושכך חמתך בחסדך / כי אתה יכל לעשות מה
שלא נסית / ולא תוכל להשיב מה שעשית / ואם
לבך חם / מודה ועזב ירחם / ידעתי כי הסכלתי /
ושגיתי אבל לא זדתי / כי מי שלח יד באדני ונקה
/ ולא היה לו לפוקה / חלילה למלך גדול וגבור /
לשלח יד בעכבר נבזה לשבור / ובאמרו ככה
נכמרו עליו רחמיו / ויכרת לו את ברית שלומיו /
ובשפתי רצון אליו קרא / השמר והשקט אל תירא /
ויהי בחצי הלילה / באישון חשך ואפלה / עת
שאגת הכפירים / הארי שאג לטרף באחד היערים /
ויפל ברשת מזורה / ויזעק זעקה גדולה ומרה /
וכאשר התחזק לקום הרשת החזיקתהו / תעה לבבו
פלצות בעתתהו / ותדד שנת העכבר לקולו /
וילך עדיו וימצאהו בסבך מראשו ועד רגלו /
ויאמר לארי: מי אתה כי הקיצותני / ויען לו הארי:
הנני / ויאמר העכבר: העם אשר מאסת בו / צא נא
עתה והלחם בקרבו / אך בא העת והגיע הזמן /
אשר לשלם גמולך אהיה נאמן / כחסד אשר עשית
עמדי / להצילך מפח יקוש יש לאל ידי / כי אקבץ
עמי חברים / לכרת מן הרשת מיתרים / וננתקה את
מוסרותיך / אז תצליח את ארחותיך / ויאמר לו הארי:
אדני מהרה / טובה חכמה מגבורה / וצדיק אתה
ממני / אם מרשת זו תוציאני / והעכבר את חבריו
חבר / ויעש לו כאשר דבר / ויכרתו בשניהם
הרשתות / ויהיו כרגע כרותות / ויאמר לו הארי:
עתה גמולך עלי / כי העמדת במרחב רגלי / ויפן
העכבר שכמו ללכת לדרכו / והארי שב שלם אל
סכו.



והמשל באביונים ועניים / אשר
מפשע וממרד נקיים / נגד עשירים משכנותם / אך
אם חטאו בשגגתם / בבלי דעת, והעשירים / על
שגגתם מכפרים / בבא לעשירים יום אידם / העניים
במאמצי ידם / בהם ידבקו לעזור / כאשר ידבק
האזור / כי יש אהב דבק מאח / ויש איש אם אחיו תוך
אח / לא יציל נפשו משרפה / ואמרת החכם צרופה /
באמרו: אדם גדול כענק / לפעמים בזבוב קטן נחנק
/ ושאלו לחכם, אשר חפץ ביקרו / ומי אהב יותר,
אחיו או חברו / ויען להם: שמעו נא דברי / אינני
אהב אחי עד יהיה חברי.



ואשא משלי ואמר:



מטיב לך טובו גמל / אם ישגה ברע חמל

כי יש בעת צר יעזור / מחר ואם בזוי תמל

לבך בך השפל ומל / לתם וראש עקש קמל.
