


	
	משל עח: קוף ומלך
	




ארח עם שפת שקר לחבורה / נפשו בעיניו לא יקרה



קוף מלך גדול גדלו / ימים רבים עמד
אצלו / עד אשר הכיר כל מעבדיו / מאכל שלחנו
ומושב עבדיו / ואשר מלך עליהם וגבר / לא נעלם
ממנו כל דבר / לצאת משם היה נחוץ / ולמשל על
אחיו בחוץ / ויהי היום ויכן את מברחיו / ויצא
השדה אל אחיו / במדברות על ההרים / בסעיפי סלע
ביערים / ויאמר לפני קופי יער / אי לך ארץ
שמלכך נער / על כן ירדו טובים מטה / כל ראש
מקרח וכל כתף מרוטה / ואין אחד מכם גדר פרץ / מי
יתנני שפט בארץ / להשיב לצדיק כצדקתו / ולא
תונו איש את עמיתו / ואעשה לכל כאשר זמם / אני
לכד חכמים בערמם / יאות לי על רעי להיות משלם
ורדם / בן חכמים אני בן מלכי קדם / הקופים דבריו
האזינו / ויאמרו: קצין תהיה לנו / ויבחר מכל
הקופים / הגבורים היפים / להיות נצבים עליו /
ויש להכין מאכליו / והנבזים ורפי מתנים / נתנם
חטבי עצים ושאבי מים / והזקנים הקים שפטים /
ויהי בימי קציר חטים / שני אנשים במדבר תעו /
ויבאו עד המקום אשר תקעו / הקופים שם את
אהליהם / ויראו כי בקרב מחניהם / מלך וכל אחד
הלך / לאמר: יחי אדנינו המלך / וקוף קטן עמד
לפניו / וימצא חן וחסד בעיניו / על כן חשבוהו
האנשים לבנו / נצבה שגל לימינו / אצל החרלים על
שיחים / אכלים ושתים ושמחים / זה שר האפים וזה שר
המשקים / וזה שמר כלים ריקים / זה נקלה וזה נכבד
/ כחק ומשפט בין אנשים / אשר רשים עבדים ראשים
/ והקוף באנשים התבונן / ובכליותיו ישתונן /
ויהי כאשר ראם / בשפתי רצון לו קראם / לגדול
אמר: אני משביעך / להגיד לי מה במדעך / אם הוד
המלכות עלי / אם גבורים נשאי כלי / ואם בני
למאד יפה / החזק הוא הרפה / ואם יפת מראה אשתי /
אשר נצמדת אתי / ואם למאד נעמו משרתי / ושמרי
ביתי ואוצרותי / האיש הנאמן אשר השביע / לפי
ראות עיניו יביע / ויאמר: מכל אשר ראיתי / לא
אשקר באמונתי / כאב כבן כאמה כבתה /
כעבד כאדניו כשפחה כגברתה / קופים אתם
וקופים אבותיכם / ברוך אשר מאס בכם / קופית
אשתך וקוף בנך / נצבה שגל לימינך / תארה
משחת והועם / נבזים ושפלים לכל העם / אתם פריצי
חיות ועקשים / ומעדת כל חי נגרשים / תחת
כבודכם חרפה / ותחת חגורה נקפה / קוץ מכאיב
וסלון ממאיר / אחריכם נתיב לא יאיר / אך תבאישו
כל מלון / ירשתכם בוז וקלון / קוף המלך הדבר
שמר / לאיש השני שאל ואמר / מי יתן ואשכנה
במגרשיך / אשרי עבדיך ואשרי אנשיך / השמעים
את חכמתך / ורואים יפי חבורתך / אשר למאד גברה
ידם / יפיפית מבני אדם / ואם כל יפים אצלך
נקהלו / אור פניך לא יפילו / ואשתך מאד נעמה /
כלילת יפי וחכמה / לב כל מלך בראותה יגאה /
יפת תאר ויפת מראה / בנך לא נחקר הודו / מכף
רגלו ועד קדקדו / אין בו מום וכל דפי / לא דמה לו
כל חי ביפי / קופים קראו אל המשקר / ויכבדוהו
עד אין חקר / ולנאמן בגדיו קרעו / ואחרי כן
הכוהו ופצעו / והמלך צוה להשיבו אל מסלותיו /
להודיע לבני אדם גבורותיו.



והמשל באיש אשר אין רמיה
בעשתנותיו / ועולה לא נמצא בשפתיו / וארח עם
שפת שקר לחבורה / נפשו בעיניו לא יקרה / כי זה
להחניף האמת יסלף / וזה לא ימיר ולא יחלף / ואם
בעבתות האמת יהיו נצמדים / אחריתם להיות מכבדים
/ והה ליום אשר איש בפיו נפשו יטנף / לקבץ און
לעשות חנף / ולבעבור כי חנפה רבו אהביה / הארץ
חנפה תחת ישביה.
