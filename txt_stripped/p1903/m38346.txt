


	
	משל פח: פרס וסיר
	




טוב טעם ודעת בכליותיך / תבאמו ותטעמו בהר נחלתך





הפרס יבש גרונו מצמאה / ולשונו
בצמא נשתה ונלאה / לבקש נהרים יארים רחבי
ידים / וילך שלשת ימים במדבר ולא מצא מים /
וביום הרביעי מצא סיר ומים בו / ולא ידע אם חציו
או רבו והסיר עמק וארך / והפרס לשתות ערוך /
ולא יכל עד המים להשפיל ראשו / ויכנע בעמלו
וכפלים שקקה נפשו / וירד להדפו ממצבו
ולהפך את הכלי ואת המים לשפך / וילך סביביו
בחתף / אולי יוכל לו בצד או בכתף / ולהפילו בפיו
וברגליו / וירא כי לא יכל אליו / וילקט אבנים
וקטנות היו רבם / מסיע אבנים יעצב בם /
ולהשליכם בסיר יורוהו מזמותיו / וירומו המים
וילכו על כל גדותיו / וישבר צמאו ויהיו לו
לתרופה / מים קרים על נפש עיפה.



והמשל לקנות חכמות ומזמות /
ותחבלות התבונה וערמות / כי יש רפה מתנים
ומזמה בלבו קשורה / טובה חכמה מגבורה / על כן
מדע ושכל תשריש במזמותיך / תבאמו ותטעמו
בהר נחלתך / ומזרעוני בין תזרעהו / וממעין
השכל תשקהו / ואל תסור אחרי אולת / פן תהי
נפשך בם מוראה ונגאלת / ואמר החכם: המתעסק
בחכמה לא יכלם / אם אין לו צואר עב וגוף שלם /
ואשר שת אל דברי לבו / מצודות סלעים משגבו /
יראו חכמים משלי ועוד יחכמו / ושמעו אמרי כי
נעמו.
