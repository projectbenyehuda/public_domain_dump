


	
	משל קה: חזיר וארי ושעל
	




המכביד לבו לפי שעה / יהיה לו תמיד רעה



חזיר יער הלך ושב כדרכו / בחצר ארי
המלך ובתוך סכו / והארי אמר לו: מה לך אצלי /
אל תוסף לבא עוד בגבולי / ולא תטמא משכני אשר
חמדתי / ויען לו החזיר: אדני שמעתי / ויהי
בבקר והנה החזיר מתהלך / סביבות סכת ארי
המלך / ויזעף אליו ויכרת אחד מאזניו / ויקרא
אל השעל ויאמר לו בפניו / אם החזיר לא יוסר מעליו
/ לא תחמל ולא תכסה עליו / כי סכתי אשר מחסה
מזרם ומסער / יכרסמנה חזיר מיער / גם כי
נכריה עבדתו / ועתה אל תנקהו והמיתו / והחזיר
הכביד לבו ויקש ערפו / והארי בחרי אפו / כרת
מראשו אזנים / וינקר את העינים / ועודנו מכעיסו
בגללי צאתו / ויטמא סביבות סכתו / ויאמר הארי
אל השעל: סב פגע בו / ואת ראשו ואת כרעיו
וקרבו / וכל נתח טוב כתף וירך / שים לקראתי על
הדרך / והשעל לנתחיו נתחו / וירא כי לבו
שמן ולקחו / לדעה מה יעשה לו / ולא יכל להתאפק
ואכלו / ויערך ערך אבריו / והארי בא לראות
בתריו / וירא אשר מהנה לא נעדרה / כל עצמותיו
אחת מהנה לא נשברה / אך מצאם כסדר / את הראש
ואת הפדר / ובהגיעו לקרבו / לא מצא את לבו /
ויאמר בחרי אל השעל / איך באדניך תמעל מעל /
איה לב החזיר אשר מחצת / ואיך להסתירו ממני
חפצת / ועתה אשיב לך כגמולך / כאשר עשית כן
יעשה לך / והשעל בחכמתו ענה / אדני לא עלי
התלנה / אך אל אמרי הטה אזנך / הן זאת לא צדקת
אענך / תבין במריו ותשכיל בסכלו / כי לב
בקרבו לא היה לו / ובערכי אבריו לעמתי / יגעתי
ולא מצאתי / על כן בדבר המלך לא יוסר / וגם
בדרך כשהסכל הלך לבו חסר / ובחסר לבבו
לקחת אזניו ועיניו / כאשר על מצותך העז פניו /
ואדני המלך חכם יתהלך במישרים / ואל ישים על
עבדו עלילות דברים / ויען הארי: שמעתי בקולך
/ אם חכמת חכמת לך / וצדיק אתה ממני / על כן
כל ימי תשב לימיני / אין חכם כמוך ושמך אכנה
נעמי / אתה תהיה על ביתי ועל פיך ישק כל
עמי.



והמשל אם יצרך תוקפך לעשות שלא
כהגן / חפש בשרשי מזמותיך להנצל כאיש מגן /
ואל תקנא באנשים אשר לבם ואזניהם הערלו /
בצאתם ממצרים לא מלו / כי איש אשר לבו ערל
ואזנו ערלה / רעיוניו הרעים ציצין המעכבין את
המילה / אך ראה דרך אשר תצא מחטאתך לרויה /
ואז נפשך תכונן וחיה / כי חכמה תחיה הבעלים
/ ומרפא יניח חטאים גדולים / ואני מצאתי בספר
הנסיכים / המלכים שפטי ארץ והחכמים שפטי
המלכים.
