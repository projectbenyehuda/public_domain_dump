


	
	משל צד: שעל וחתול
	




אשר אין בו רק מהלל שפתיו / יעטה בשתו וילבש כלמותיו

השעל קרא לחתול / תחת עץ על מים שתול
/ לאמר: שמעני, מעתה / נכרתה ברית אני ואתה / כי
רגלי קלות כאילות / וכנגדם נסית לעלות /
במקום אשר לא יעלה שעל / הנה ראיתי ממעל / בשר
כבש לח ושמן / ואם דברי לא יאמן / בא ואראך
המקום ההוא / ותחת זה העץ נאכלהו / והחתול השיב
מענה / על פיך אסע ואחנה / ויכרתו ברית ביניהם /
וילכו יחדו שניהם / ויאמר לו השעל: פקח עיניך
/ וראה המקום אשר לפניך / ואם תוכל הבשר להביא
/ עלית למרום שבית שבי / ואני אוחיל לך הלום / עד
שובך אלי בשלום / והחתול במעלות עלה / אל הבשר
אשר נתלה / באשר לאכל אתו חבב / עלה לכבש ופנה
לסובב / והבשר בין שניו נשאו / שעל המצפה עת
באו / שמח ויצא לקראתו / באשר שמר את מצותו /
ובאזני החתול יועץ / נלכה נא תחת העץ / ובגורל
יחלק שמה / בלא ערמה ובלא מרמה / ובלכתם יחד
במסלות / שעל חשב בתחבלות / ולגזל הבשר ארב
/ ואל החתול הלך וקרב / והחתול לעלות על עץ מהר /
בטרם השעל נזהר / כאשר רעב וחש לאכל / שעל
לעלות שם לא יכל / ויקרא לו: חתול בוגד / התאכל
ואעמד מנגד / ולא תשמר את בריתך / ואת מוצא
שפתיך / להשיב את נפשי הרעבה / תן לי ולא אוסיף
לדאבה / ויען לו: באות נפשי אכל בשר / ובטן
רשעים תחסר / אני אכל ואתה תרעב / והמלעיב
יהיה לנלעב / לי הבשר, בריאה ושמנה / ושני
רשעים תקהינה / ויאמר השעל: הזה מאתך גמולי /
ואם ככה את עשה לי / תחת און עמל תקצר / הנני
עליך במצור / לשלם לך מדה במדה / ומה תעשה
ליום פקדה / כי משפטך חרוץ ונחצב / וצרתי
עליך מצב / לא אמוש מזה עד רדתך / איפה תמצא
דרך לכתך / תחת הבשר אקח נפשך / ואסיר
מעליך ראשך / חתול ענה: זאת לא תהיה / לא אמות כי
אחיה / באשר מן ההר ירדים / פרשים רכבי צמדים /
לפניהם כלבים יתגאו / דרך עליך יבאו / ברח
ותמצא לך נפש / כי למה יכוך נפש / שים לדרך
פעמיך / טרם בידם תנתן / צפה דרך חזק מתן /
שעל ענה: לא יגרתי / ולא מן הכלבים נבעתי / כי
מי יוכל אלי להרע / לבי מערמות לא יגרע / ממאה
תחבלות וסבות / וכהנה עמי רבות / אשר מהם
אמלט נפשי / לצאת מידם חפשי / אך לא אלך עד
סכותי / אשר תתפש במצודתי / ובין שני תהיה
נשבר / לא תרדף עוד אחרי עכבר / ויהי כדבר לו
ככה / כלבים באו במערכה / והפרשים הסיתום בו
/ זה נשך אתו על גבו / זה בזנבו וזה באזניו / זה
בצוארו נתן שניו / ויש אשר אתו נשך / בגיד
פחדיו ובאשך / וכאשר קראתהו התלאה / חתול
קרא: אחד ממאה / תחבלות עשה נא והמלט / אז
תהיה בגופך שלט / ומה יועילו ערמותיך / אם לא
יושיעוך בעת רעתך.

והמשל כי זה מנהג המתגאים / תרבות
אנשים חטאים / האמרים: ידנו רמה / מהחל חרמש
בקמה / מאד מתהללים בערמתם / ומתפארים
באמנותם / אני שקל אני ספר /אני חרש אני תפר /
אני צרף אני סחר / אני חכם ומי כמוני אחר / אשר לי
ידמה וישוה / אני רפא לכל מדוה / ולא יציל נפשו
מצרתו / ולא יכלכל בני ביתו / באשר לא נמצא
חכמה בו / וכל רוח אין בקרבו / ויעטה בשתו
וילבש כלמותיו / איש אשר אין בו רק מהלל
שפתיו.
