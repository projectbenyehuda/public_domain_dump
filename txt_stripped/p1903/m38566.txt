


	
	משל קז: פיל ואיש שדה
	




אם היה מצער ראשיתו / מאד ישגה אחריתו



הפיל מפלא בחיות / בדמות בקומה
בראיות / כי הוא מלא עצמות / וסביבות שניו אימות /
צוארו ארך מאתים / אמות ויוצא בינתים / מבין
השוקים ומן החזה / לדעת מה זה ועל מה זה / ללקט
מאכלו בפיו בפעם אחת / עגלה מלאה טעונה שחת / עיר
ויושביה רוכבו / פיו פער לעלות על גבו / כאשר
יוכל שאת משא / פסיעותיו פסיעה גסה / וידע ציד
איש שדה / מן החיות ארב וצדה / והנה פיל יוצא
לקראתו / בדמותו במראהו בקומתו / והאיש אליו
הלך וקרב / על ירכו חגורה חרב / ויהי כאשר הקריב
הפיל אליו / חרק שניו עליו / ויפער פיו לבלי חק /
וישליכהו למרחק / והאיש עיף ויגע / הלך שוק על
שוק נוגע / להסית הכלבים בו לנשכו / לא יכפר
לדם כי אם בדם שפכו / ויבן אליו האיש ההוא /
אולי יוכל ברשתו להפילהו / ויפרש הרשת
והמיתרים / מכמרות ופחים ביערים / וירדפהו עד
באו שמה / וישברם וינתקם עד החרמה / וילטש
חרבו / אולי יוכל להלחם בו / וילך ויפגשהו
ויבקש המיתו / כי תאנה הוא מבקש לעמתו /
וכאשר הרים ידו עליו וגבר / חרבו ככלי יצר נפץ
ונשבר / וירא כי לא יוכל לו / ויזעק כל העם
לקולו / ויאמר להם: שלי הוא זה הפיל / חמדתיו
ופני בו אפיל / כי לא יכלתי בערמתי / לצודו
במצודתי וברשתי / גם לא בחרבי ובקשתי / ואני
כאשר שכלתי שכלתי / רדפו אחריו אולי
תשיגוהו / כי תאחזנו רעדה ופלצות תבעתהו /
והאספסוף אשר בקרבו / התאוו תאוה לשלט בו /
ויפלוהו מהר בשחיתותם / וילכדוהו בחפזם
ובחמתם / ויביאוהו אל אדניו איש שדה / ויאמר
בלבו אדניו, כי בפרך ירדה / ויכנע בעמל לבו
/ ויעל אדניו על גבו / ויאמר לו: מדוע כה עשית
ללכת עמי בקרי / בחרון אפך ומרי / ואתי
השלכת אחרי גוך / אכן צנים ופחים סביבך /
וירא כי אין זר אתו / וילט פניו באדרתו /
וירכב עליו עד בא השמש / מהחל בקמה חרמש /
ויראו אתו הקוצרים / ויתחבאו ממנו ביערים /
ישרגו גידי פחדיהם / כי יזמו השד עלה בשדיהם /
ויאמרו איש אל רעהו / השד על
סוסו ראינוהו / ויצעקו ויבכו כל הלילה / קול
המלה גדולה / והנה אימה חשכה עליהם נפלה /
ותחזק ביניהם היללה / עד באם אל עיר מושב /
ויאמרו: הוציאו כל גר ותושב / להבריח השד
מעלינו / הלנו אתה אם לצרינו / ותהם עליהם כל
העיר / ויאמרו: נגד שד ושעיר / לא נלחם מלחמה /
פן נהיה לבוז ולכלמה / ויעזבו את אהליהם / ואת
מקניהם ואת חמוריהם / וינוסו על נפשם / אנשים
ונשים משם / והאיש עם הפיל בא שמה / ואהליהם
כאשר המה / מצא ובם כל הון יקר / זהב וכסף צאן
ובקר / וישלח ויבא את אהביו / ויקרא אליו כל
קרוביו / ויוצא מן העיר כל כלי חמדה / בתבה
במגדל ובשדה / ועל מיטב צאן והבקר והמשנים /
שמחה האם על בנים / והעיר תשאה שממה / מאין אדם
ובהמה / מצאן ובקר לרב / כמלך הבא עליה לחרב /
וילך ויבא לעיר אחרת / ויעש לה כמו לראשונה
הנשארת / וכן לשלישית ורביעית / חמישית ששית
שביעית / ושמינית והתשיעית / ועוד בה עשיריה /
ושבה והיתה לבער בה כל שביה / וישב בשלום
עשיר לביתו / ויודיע להם מה משכרתו / אשר
יבריח מהם השד בשמות / אשר בארץ שם שמות /
וידרו לתת לו חצי השבי / להחזיר להם חציו
ולהביא / ויעש כן האיש ההוא / וימחלו לו את אשר
כבר עשהו.



והמשל באיש מסכן וחכם / לרדף אחרי
טרפו ממוכן / הוא בכל יום עיף ויגע / ולא מצא
לנפשו מרגוע / ומאומה לא ישא בעמלו / כמקרה
האיש לבדו עם פילו / וכאשר הגיע דבר המלך
אליו / להתגבר מזלו עליו / להעלותו מעלה מעלה /
כבעל הפיל לגדלה / ירחיק מידו גזלה וגנבה /
בפעלתו אשר עבד בה / כאשר פעל האיש עם
השבי / להחזיר חציו נתגבר כלביא / ואם היה
מצער ראשיתו / מאד ישגה אחריתו / ולחם מטעמים
יסעד / וצדקתו עומדת לעד.
