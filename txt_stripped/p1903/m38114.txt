


	
	משל נג: שמש ורוח ואיש
	




החכם יעשה בחכמתו / יותר מהגבור בגבורתו



השמש והרוח הגישו עצמותם / ויאמר
השמש לרוח: נקיתי ואיתם / וברב יפיי שמוני
ביום לממשלת / ואם תתגאה לנגדי תחשב לך
אולת / באשר לי הגבורה והחן והזהר / וכעצם
השמים לטהר / ויען הרוח: אם כפלים לך
כמעלליך / רוח המושל תעלה עליך / כי בלילה לא
תוסיף להראה / שמש ידע מבואו / וממשלתי ביום
ובלילה / על כן יאות לי הממשלה / ומעשי בארץ
נודעים / מפרק הרים ומשבר סלעים / ויאמר השמש:
ננסה את גבורתנו / הנה האיש היוצא לקראתנו /
בשמלתו ובאדרתו / צא נא עתה והלחם אתו /
ובתחבלות תעשה לך מלחמה / עד תפשיט אדר ממול שלמה / ואם יעזב בגדו
אצלך / הוד המלוכה אתן עליך / ותשגיא פעלך
ותפשיטנו / שמלה לך קצין תהיה לנו / ויאמר
בלבו הרוח: אפתנו בערמתי / או אפשיטנו
בגבורתי / ויצא הרוח מאת השמש לעיניו / ורוח
חזק מרחפת לפניו / והאיש הולך דרך ישרה / ובבא
ממזרים קרה / יגלם אדרתו ושמלתו ויתעטף בהם
/ ורוח לא יבא ביניהם / והיה כאשר התחזק הרוח בכל מאמצי כחו / והאיש ללאות
אדרתו יקשר סביב מצחו / ובימינו יחזיק כנפיו
ויצנף סביב ראשו / ולא גלה הרוח פני לבושו /
והיה כי נראה כי נלאה / הרוח אשר גאה גאה /
ולגזל האדרת לא נשגבה / ותהי תוחלתו נכזבה /
ותאמר השמש: עתה תראה את אשר אעשה לו /
ותקרע סות מאפל ואת צללו / ויפרץ בצללים פרץ /
השמש יצא על הארץ / ותתצב בכחה לעמתו /
כצאת השמש בגבורתו / להשקיף על האיש ממרום /
בהשקט ארץ מדרום / ועין השמש בנחת שזפתו /
ומכבד בגדיו בעתתו / ומאדרתו תפשיטהו /
ערום ינוס ביום ההוא / ולא יוכל עוד לשאת כליו / אם
זרחה השמש עליו / ותפשיטהו בהשקט כאשר עליו
הגיעה / ולא כן הרוח אשר בגבורה ומסער
סעה.

והמשל כי החכם יעשה בחכמתו / אשר
לא יוכל הגבור בגבורתו / כי דברי חכמים כחצים
ביד לוחמים / וכלבות אשים בלהב / תחתיו שחחו
עזרי רהב / כאשר בקירות בין המה מפתחים /
להמשיך לבות ואם הם כרשת ופחים / באשר הם
נשמעים בנחת / אך מקול כסיל גאה איש לא
יחת.
