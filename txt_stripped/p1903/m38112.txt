


	
	משל נא: ארבעה שורים וארי
	




חבורה בלי פחד / שבת אחים גם יחד



שורים ארבעה לחברה נועדו / יתלכדו
ולא יתפרדו / וברית כרתו ביניהם / ורוח אחד להם /
וימים רבים בחברתם האריכו / בעמדם יעמדו
ובלכתם ילכו / והארי סובב לארבעתם / וירא כי
לא הגיעה עתם / באשר יעמדו יחד במערכת /
וארבעה המה מיטיבי לכת / וישת הארי עצות
בנפשו / מדי חדש בחדשו / לעשות בהם כלה / אם
נתפרדה החבילה / ויהי היום ויעמד הארי מרחק /
ויפער פיו לבלי חק / באשר הוא רעב ועיף /
ומתוחלת ממשכה סר וזעף / ויקרא אל השור / אשר
עמדו רגליו במישור / ויסיתהו בחלק שפתו / עד
כי הפרידו מחבורתו / ובהתפרדו מחבריו לרעה
הבדילו / כי באפו טרפו ואכלו / והשלשה
הנותרים במחנה / מקולם לא יחת ומהמונם לא
יענה.

והמשל הנה מה טוב ומה נעים /
בהתאסף במתק סוד רעים / וחבורתם בלי פחד / שבת
אחים גם יחד / והמפתים אשר בפיהם כמי אכזב /
יעצו להדיח ירצו כזב / להפרידם מחבורתם / עד
אשר זרו מתאותם / ואת אשר מחברתם נודד / הבוגד
בוגד והשודד שודד / והחכמים לא ישימו על לב אם
העמית להם ילשינו / אזנים להם ולא יאזינו /
באשר רוחם טהורה ונקיה / וישנאו לשון שקר
עשי רמיה / וחבורת שני חכמים לא תרתק / והחוט
המשלש לא המהרה ינתק.
