


	
	משל פט: פרש נורא ואים
	




אל תתהלל בגאותך / כי לא תדע אחריתך



פרש נורא ואים היה / ונודע שמו בכל
קריה / והמון עם יחתו לקולו / ורזנים משחק לו /
מרחק יריח מלחמה וייראו מגשת אליו / בחגורתו
אשר במתניו ובנעלו אשר ברגליו / ותהי לו
לישועה / רעם שרים ותרועה / כי בשמעו את קול צעדה
/ בראשי הבכאים ידו הדה / לשפך דם רהבים
ביבשת / לא יבריחנו בן קשת / ומצחת נחשת על
מצחו / ומשך אבירים בכחו / וישחק לשד ולכפן /
קול שוט וקול רעש אופן / לנתק ולשבר שריונים
ימינו נאדרת / כאשר ינתק פתיל הנערת / וידיו
בעדיו יריבו / פרסות סוסיו כצר נחשבו / ויצא
שמו עד למרחק / רדף ומשיג היה לו לחק / על כראות
לחמיו כי ברושיו הרעלו / באו בעבים ובכפים עלו
/ כלם כאחד מפניו נסים / אלה ברכב ואלה בסוסים
/ ותהי על שכמו מן הפרשים המשרה / ויתנו עליו
הוד הגבורה / ויפגע בסחר כבד בכסף ובזהב /
ויעש לו מטעמים כאשר אהב / גם בלסטים מזין
שלח ידו / ויהפך לארץ הודו / ובעוד כבודו חדש
עמו / הזמן שלח ידו בשלומו / ויפרש פח לפעמיו
/ גבר לא יצלח בימיו / ובכל עת אשר בקרב בא / או
נשבר או נשבה / ואם ירוץ יכשל בהליכתו / אין
כהה לשברו נחלה מכתו / באשר לו חלחלה
במתנים / ולב נמס ופיק ברכים / וקרא לו שם חדש
חלש ולא גבור / קנה רצוץ לא ישבור / ויגרהו
בחרמו כל שפל / וממעלת גבורתו נפל / ויהי
כנפלו ואין מקים / ושוקיו רתקו בזקים / כי עלה
מפיץ על פניו / ויעצם את עיניו / וידיו מקשת
אסרו / והביטו אליו את אשר דקרו / גם הסחר
והגזלן נצמדו יחד / וממכון שבתם חזר ונעור כל
אחד / ויהי תמיד הלך וברח / ויזעקו אחריו: עלה
קרח / ולו כי נתן כפר נפשו / יסירו מעליו ראשו /
ומן היום ההוא הלך אחרנית / מפני להב חרב וברק
חנית / ותחת אשר על אחיו עד כה עצם וגבר / לא יצא
בצבא ולא יעבר עליו לכל דבר / ועבדיו מקדם
יכבד ויעבוד / וישבע קלון מכבוד / וישבע
לעיני כל משפחתו / לשרף את כל כלי מלחמתו /
ולא יצא עוד למלחמה / ויגדל המדורה להשליך
כליו שמה / כי אמר: אשרף ואשמידה / סוס דהר
ומרכבה מרקדה / שריון וכובע עמם ימח / מגן אם
יראה ורמח / חרבו תקע בבטן סוסו אשר אהב / ויבא
גם הנצב אחר הלהב / ואחרי כן החרב שבר /
ויעש לכליו כאשר דבר / אם מהם כל המחנות
החריד / לא השאיר מהם פליט ושריד / ולמחרתו זכר
את שופרו / אשר בו תקע תוך כרמלו ויערו /
וכאשר נשאו באש להשליכו / הלך ילך ובכה /
ותאמר לו אשתו / מדוע זה לשרף זכרתו / ומכלי
מלחמה איננו / ולמה זה באש תשרפנו / לבך
מלהשחיתו יתיאש / הלא זה אוד מצל מאש / הנני
במעשיך נכלמת / אל נא תהי כמת / ויהי כדבר אליו
צלעתו לא אחת ולא שתים / ויך את הצלע במטהו
פעמים / ויאמר לה בחמה: הנך בת המרי / כדבר
אחת הנבלות תדברי / הנה גם הוא היה בעכרי /
להעיר עלי שעשועי נעורי / כי כאשר קול שפר
שמעתי / כל אנחתי השבתי / בדי שפר אמרתי:
האח / על כן אשליכנו תוך האח / כי קול תרועה
בשמעי / על פיו אכין מסעי / והתעררתי על איבי
להלחם / ותבא חרבי ותקחם / ומן המלחמה בשובי
/ לצוד ציד להביא / תקעתי בו ויחרצו כלבי
לשונות / כי הוא מאסף לכל המחנות.



והמשל להזכיר כי זה משפט חרוץ
כזרק כמושיט / ואחז הרגל כמפשיט / כי גם
השפר אשר לא מכלי מלחמת הפרש / לשרפו עם
כלי מלחמתו דרש דרש / אם העלם יעלים הראה
באשנב / דין המעלים כדין גנב.
