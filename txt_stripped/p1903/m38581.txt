


	
	משל קיא: שממית ומלך ועבד
	




אשר העשירים עליהם לא חמלו עלו למעלה / והמה ירדו מטה שאולה



שממית היא בהיכלי מלך / ידה תמכה
פלך / לטוות ולעשות מטוה / אל עמק שוה / ובכל
קרני הכתלים / מן הבתים ומן האהלים / ויבן הבית
מאבני אלגביש / בית המלך תוכו עשתה בית
עכביש / עד שהמלך במסבו / וחמת המלך בערה
בו / והעבד נקרב כדרכו / לכבד הבית ברחבו
וארכו / ויבט את העכביש בקרנות / ויעל ארבע
פנות / מזרחית צפונית מערבית דרומית / ויאמר אל
השממית / לא נוכל לשבת יחדו / את המלך עם עבדיו
/ וטאטאתיך במטאטא השמד / ותלך היא אל היין
ואל המים ואל התמד / להטיל ארסה במשקה /
והאיש נקה לא ינקה / לצבות בטנו ולנפל ירכו /
והעבד ילך הלך ובכה / להגיד לאדניו הדברים /
ויהיו המים המאררים למרים / ויצו המלך לשפך
לחוץ המשקים / ברחובות ובשוקים / ויאמר העבד:
מר ממות אני מוצא / ויבקש הדבר וימצא / כאשר
באו מן היערים / מן השדה העזים והחזירים /
ישתו יין נסכיהם / וצבתו בטניהם ונפלו
ירכותיהם / מן המשקים אשר שפכו / שתו ולעו
וכפשתה דעכו / והאנשים הרגישו בדבר / את
אשר נעשה כבר / כי בזבוב קטן נחנק / אדם גדול
כענק / לשתות כוס התרעלה / בלי רפואה ארך
ותעלה / וכאשר ראה המלך הנס בלבו עלץ / כי
מצרה גדולה נחלץ / ממיתה משנה לחיים / והיו
חייו מנגד תלויים / ויעש משתה לכל אהביו /
ולאנשי ביתו ולקרוביו / ויאמר, כי טוב לי עתה /
להעביר השממית מרעתה / ויבא אליה האהלה /
ויצו אל עבדיו להגיד לה / לבוא בדבר המלך /
ויאמר: שמתי שלום בגבולך / כי בך מרה מריתי /
מרבדים ערשי רבדתי / לולא יגרתי חרבתיך
ושממותיך / הרחיבי מקום אהליך ויריעות
משכנותיך / ובאי ליני באשר אלין / מה יפו
פעמיך בנעלים / והנה בית זבול לך בניתי / תחת
אשר לך הרעותי / ולא תוסיפי לדאבה עוד עולמית /
ולא תעלה בלבי עוד עליך רעה ותרמית / כי
הביאתני עד הלום / ושמתי פקדתך שלום / ואם את
עשה לי ככה / וחמתך מני שככה / ותען השממית
קשות / עת לדבר ועת לחשות / גרשוני מהסתפח
בנחלתך / הנני משליח בך ובבתיך / ערב כבד
מאד / ומפני לא תוכל לעמד / ולא תוכל להרפא / כי
ממות תהיה נספה / ותלך ותצבר כל השממיות
/ מן הבתים ומן הגאיות / ותצברנה כלנה יחדו /
לבא אל המלך ואל עבדיו / ותגשנה לאבדם
ולהמם / האנשים עד תמם / ותעשינה בהם ארבים
ונגשים / בנפל תרדמה על אנשים / ויפלו
וימותו לרב מהם / ולא נשאר עד אחד בהם.



והמשל באביונים ודלים / והעשירים
עליהם לא חמלים / ואחרים הסיתום להרע להם / כאשר
היו מקדם איביהם / והעניים יבינו לאחריתם /
ויעשו חרצבות למותם / ולנקם נקמתם ישכילו /
לשלם להם מעלם אשר מעלו / והיו לסכים בעיניהם
/ ולצנינים בצדיהם / עד אשר באו
וכרעו ונפלו לרגליהם / לעבר על פשעם ולמחל
חטאתם / ויכנעו וישובו על עקב בשתם / וירדו
מטה מטה שאולה / והעניים עלו מעלה מעלה / ואמר
שלמה בחכמתו / יש מתרושש והון רב מסיר אזנו
משמע דל בזעקתו.
