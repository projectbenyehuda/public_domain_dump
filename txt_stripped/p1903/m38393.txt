


	
	משל צז: שחל ועזים ושעל
	




תמחל למי שהרע לך / ותן למי שמנע ממך בשאלך

שחל שלח בעזים חרון אפו / ויהי
כאשר טרף החביא טרפו / והשעל הבין נתיבות ביתו
/ ויתגנב ויבא בלט בערמתו / ומבשריו אכל
ומגרמיו גרם / וישבע מהם ולא נתן מכרם / כאשר
בשדה מצא מציאה / לנתחיה לנתחיה הוציאה /
והשחל ראה כי השעל / לא שמר אמונים וימעל מעל
/ ויקראהו בדברים רכים ולא בחרי / זאת הפעם עצם
מעצמי ובשר מבשרי / אשר טרפתי בעמלי וביגיע
כפי / אך כליותי יעצוני והשיבוני שעפי / לאמר:
לך עזב צידי ואכל מצידך / ואם כה תוסיף שנית ידך
/ אשנה לך ׳לא תגנב׳ ו׳לא תחמד׳ / ושלשת תרד
מאד / כאשר תהיה במריך עמד / וטאטאתיך
במטאטא השמד / ולשבר עצמיך ולנתק גדיך לא
אתיאש / כאשר ינתק פתיל הנערת בהריחו אש /
היאכל שעל ושחל ימחץ / לא הדריכוהו בני שחץ /
ואם יפל בידך אשר הקיף מצודו / לא יחרך רמיה
צידו / ואיך עלה על רוחך לחפש מצפוני / ולמה
ככה רמיתני / את מוסרי השכילה / ולא תשוב עוד
לכסלה / ויען השעל במתק פתוי חכו / ההסכן
הסכנתי לעשות לך כה / שמעתי מוסרך ומתק
ניבך / הן קלותי מה אשיבך / אך במעשי בושתי
ונכלמתי / לא אוסיף עוד און כאשר פעלתי.

והמשל להצילך מפח יוקשים / הלא
כתבתי לך שלישים / חידות משלים ומליצות / להיות
קשה לכעס ונח לרצות / אך תשים לך בלב התבונה
שבילים / כי מרפא יניח חטאים גדולים / ואם תראה
רעך הלך דרך פריצים / משך ידו את לוצצים / ולא
ישבת להוסיף על חטאתו / מלוש בצק עד חמצתו /
ודבר בליעל יצוק בו / באמרי נעם קרע סגר לבו /
ושנה ושלש במוסרך להשיבו / כי ידעתי את
מכאובו / ואמר לו: שנה באולתו ועשה שערוריה /
אחת דתו תאניה ואניה / למי אוי למי אבוי למי
מדינים / לאשר באולתם כמוך שנים / ואם עד
השנים בא ועבר / פעמים שלוש עם גבר / אז אין
תקוה לאחריתו / כי גם השור בנגיחתו / מועד
בשלש נגיחות / חרות על הלחות / ואם לא ישיב אל
לבו לא דעת ולא תבונה / ידך תהיה בו בראשונה /
ואמר החכם: מחל למי שהרע לך / ותן למי שמנע
ממך בשאלך.
