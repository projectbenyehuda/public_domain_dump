


	
	משל סב: זאבים, עזים וכלב
	






נתן חרבו מידו / אל שונאו נתן יום אידו



הזאבים אשר כל אחד בלבו גאה ורם / נלחמו עם העזים והכלב שמרם / ותגבר יד
הכלב והעזים / וינצחו השפלים את
העזים / וידם בזאבים חתרת / והיה החסן לנערת /
ויאנחו הזאבים וינאקו / והעזים עליהם שרקו
ויחרקו / אז אמרו לעזים בעקבה ובמרמה / לא נצא
עוד עליכם למלחמה / אך בזאת יאות לנו / לשמע
ולהיות כמונו / לבלתי העיר עליכם עוד קרבות / אם
תתנו לנו תערבות / ועת תפרדו ועת תאבו להעז /
וגר זאב עם כבש או עז / והמנכם בארץ ישלם /
וברית שלום בינינו ובינכם עד עולם / ויאותו
העזים ויאמרו / מי משלנו אשר אתו לכם תבחרו /
להיות ערבון אל ידכם / כי נכספנו לחבורתכם /
ויענו הזאבים בתרמית לב / לא נשאל מכם רק
הכלב / ויקראו העזים את הכלב ויתנוהו לערבון
/ ויהי הדבר להם למפח נפש ולדאבון / כי הכלב
להלחם אתם יצמיד / הכל הלך אחר המעמיד /
באמרו: עמדו עמדו ותצמחנה ישועות / אספה עלימו
רעות / התיצבו להלחם כדמותי / ובל יאמר שכן
חליתי / ובכל עת אמץ את רוחם / ועתה אבד תוחלתם
ונצחם / כי הזאבים ערכו אתם מלחמה / ויכום
ויכתום עד החרמה.



והמשל אל יגרע איש ממה שיש בידו /
להוסיף אל שונאו וחפץ אידו / כי זה ישריש יגון
לבו / לתת חרב בידו להרגו בו / ועשה אלה מכה
בסנורים / זה ישפיל וזה ירים / והנתן בסכלותו
נשבה או נשבר / והמקבל עליו ירוץ
ויתגבר.
