


	
	משל סז: שני קופים וארי
	






אנשי תככים מלאים קנאה / זה על זה יעירו שנאה



שני קופים התהלכו לפני הארי המלך /
ולבקש מתת ממנו כל אחד הלך / ויבן הארי
כאשר זה ידבר וזה יענה / כי האחד חמדן והשני
מקנא / ויאמר להם: אם לשאל משלי עיניכם עלי /
אתנה לכם כאשר תאמרו אלי / אך בזאת יאות לי / אם
שמע תשמעו בקולי / אחד מכם יגלה לאזני
בקשתו / ואחרי דברו לא אשנה לתת את שאלתו /
והשני אשר יחריש בפה ובלשון / עד תם בקשת
הראשון / הנה הוא לי תאות עינים / לתת לו פי שנים
/ אשר אתן לחברו / כי אחפץ בכבודו וביקרו /
והחמדן אמר בלב תך ומרמה / החרש אחריש ותהי
לי חכמה / ואשמע מה ישאל חברי ומה ישמיע
בקולו / ויהי לי פי שנים ככל אשר ימצא לו /
ויהי כי החריש והקשיב קשב / ויאמר בלבו
המקנא: חברי המחריש חכם לא יחשב / גם לי לבב
כמהו אשר מתחבלות לא יגרע / וגם הוא חכם ויבא
רע / ויקרא המקנא אל המלך לאמר / מוצא שפתיך
תשמר / ועשית כאשר דברת באזני / שאל אני
ממך לנקר אחת מעיני / הנה גליתי באזנך את
תאותי / מי יתן תבא שאלתי / ויאמר הארי: לעשות
בקשתך אשמע בקולך / כי לא אעזבך עד אשר אם
עשיתי את אשר דברתי לך / ולחמדן אמר: הנני
לקח ממך שתי עיניך / בין תבין את אשר לפניך /
וישב להם כגמול תחבלותם / ויתן להם את
שאלתם.



והמשל באנשי תככים מלאים קנאה /
אשר זה על זה יעירו שנאה / זה יאמר בלבו: איך
שכני הורם והונף / אכנו בלשון ולא אשאיר לו
שרש וענף / כי סרבים וסלונים הוא אתי / ובטרם
יפל מאשר זממתי / אתן שדות כרמים ובתים / תמת
נפשי עם פלשתים / ולעשות רע לשכנו יתנכר /
ואל צרור נקוב משתכר / ויחפשו בלבם מהתלות /
וסבות וערמות ותחבלות / גם אם יאמר להם המלך:
המתיקו סוד יחד / למה אשכל גם שניכם יום אחד /
אזנים להם ולא ישמעו / כי מדרך השכל תעו / עד
אשר זה להון שכנו שבר ואבד / ירהבו הנער
בזקן והנקלה בנכבד / ואמר החכם: דיך מן
המקנא בראותך / אשר יאבל לשמחתך
ולטובתך.
