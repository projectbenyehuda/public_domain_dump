


	
	משל קיב: איש סלע ועכבר
	







אשר נכחות לא ישיגו / קורי עכביש יארגו



איש היה מתפאר בערמותיו / ובמזמותיו
ותחבלותיו / וכי בלהטו יהיה מעבר / סלע גדול
אשר במדבר / ויוליד ויפליט נפש חיה / עד חדש
בלא שפתי רמיה / ואנשי המקום אחריו תעו / אל
מעננים ואל קסמים ישמעו / והוא עכבר גדול שמר /
ולמועד אשר אמר / שם העכבר בנקרת הצור / ויהי
שם עד בא העם עצור / ואת נקיקי הסלע סתם / וילך
ויזעק אותם / לאמר: מהרו באו אחרי / ומלאתי את
דברי / גם אשר לא האמינו עם מאמינים יחד / נסעו
ללכת כאיש אחד / ויאמרו בשפתי עליזות / מי שמע
כאלה ומי ראה כזאת / היוחל ארץ וגם יוליד הר / כי
זה לאמת דברו נמהר / והאיש המתהלל במעבדו /
מטה אחד לקח בידו / אשר על הסלע הרים / ויאמר:
שמעו נא המורים / המן הסלע הזה אוציא / נפש חיה
ולכם אמציא / ויך הסלע במטהו / להפיל מן
המקום ההוא / את כל אשר שם לפני החור / ויצא אך
עכבר שחור / ויאמר: עכבר ילד הנה יצא / לא נפל
מכל דברי ארצה.



והמשל בשמעי כזב והלכים באפל /
ויאמינו דברי טחי תפל / ופעלי מטיט היון / הרה
עמל וילד און / בטח על שוא ועל תהו / כל המחבר
לו יהיה כמוהו / והזונים אחריהם מדרך השכל
והנעים / הבל המה מעשה תעתועים / באשר נכחות
לא ישיגו / וקורי עכביש יארגו / ולא לבגד יהיו
קוריהם / ולא יתכסו במעשיהם / וחשבים להטעות
עמים בתחבלותם / בחלומותם בשקריהם ובפחזותם
/ וכמה סכלו / אשר בפח סכלותם כשלו / כי הועיל
לא יועילו / והצל לא יצילו.
