


	
	משל כד: צפרדעים, אילן ונחש
	




השוכן לבטח בעולמו / לא יקנה אדון לעצמו



צפרדעים נועצו ביניהם / להמליך מלך
עליהם / ויריבו על דבר המלוכה / זה אומר ככה וזה
אומר ככה / עד אשר התאחדה עצתם / להיות מלך
בינותם / עץ שתול על פלגי מים / אשר גבהו עד
לשמים / ויאמרו אל העץ: מלכה עלינו / שמלה לך
קצין תהיה לנו / ויחריש ולא ענה אותם דבר / וקול
ההמון מהם גבר / ויאמרו פה אחד כלם / יחי אדננו
המלך לעולם / וילכו למחוז חפצם / וגיל לבם מאד
עצם / מיום אשר המלך המלך, הוחל / לראות באבי
הנחל / זאת באה אל המלך וזאת שבה / אך אין שנאה
ולא אהבה / בעת ההיא אשר קראו מלכם / למועד שמו
עדיו דרכם / לשבת תחת צלו / יכרעו אף ישתחוו
לו / ויאמרו: אדננו המלך, באנו אליך / אל מי
דמית בגדלך / נפשנו אל אמרתך נכספה / אל טוב
עבדיך ממעון קדשך השקיפה / כי תנחם ותורם
דרך הטובה / על כל פשעים תכסה אהבה / ויקראו
אליו בקול גדול ואין קול ואין עונה / ותאמרנה כלם
הנה / אם לא תגיד לנו את שמך / וסרו הצפרדעים
ממך / ויהי כראותם כי לא הגיד סודו / וימירו
בקלון את כבודו / ויועצו להפרד מעליו /
ויאמרו: לא טוב עציו ולא טוב עליו / כי אם יהיה
בארץ רעה / לא תמוש היתד התקועה / ממנו לא בכי
ולא שחק / על כן עברו תורות חלפו חק / והוא נגוע
אלהים מכה בסנורים / ונאספו שם כל העדרים /
לקראו אלון בכות / כי לא לו נאה הוד מלכות / באשר
לבו כחלמיש / ממקומו לא ימיש / נתאמת המשל
שההמון בוטה / כל־ארך שוטה / ועמנו מיום המלך
זה עלינו היו אמה שפלה / מראיהם רע כאשר
בתחלה / וירמסו אותו ברגליהם / וימאסוהו
ממלך עליהם / ויהיו עליו מקיאות / וישימוהו
למחראות / ותאמר האחת: הנה ראיתי לנו ראש / נעזב
נעצוץ נבחר ברוש / ותאמרנה: לא נמליך עלינו קנה
רצוץ / לא הברוש ולא הנעצוץ / לא נביט אל גבה
קומתו ואל מראהו / אין נשמה באפו כי במה נחשב
הוא / נמליך מלך נורא ואיום / חי חי כמנו היום /
אשר ילחם מלחמותינו / פה אל פה ידבר לנו /
וימליכו עליהם את הנחש / אשר לא ישמע לקול לחש
/ וארח עמהם לחברה / וחברתו עליהם ישרה /
צפרדע ונחש הלכו שירות / שתי כתפות חברות /
ורוח לא תבא ביניהם / עד יום אחד אשר עמהם / על
פני המים צף / והיה כי ירעב והתקצף / ויאמר
להם: תהיו מידי נכאים / הלדרש אתי אתם באים /
ויך בהם מכה רבה / ויוציא עליהם דבה / ויוסף
עוד לשנא אותם / ויאכל בשרם ועצמותם / ויכלה
כל אשר ביארים / ויצבר אותם חמרים חמרים /
ויאכל עשר ידות / מן הבתים ומן השדות / וטוב להם
להיות בלי שטר ומושל / מלהרבות במחניהם
כשל.



והמשל בבני אדם אשר יש לאל ידם /
לשבת שלוים בביתם / ויועצו בתחבלותם /
ובארבם אשר בקרבם / ודבר רע יצמח בם / להיות
משלומם נוסעים / כאשר תעשינה הצפרדעים /
ואמר החכם: נדמה המלך לאש / אשר לבך ממנו
מתיאש / תצטרך לו בסופך / ואם תקרב לו
ישרפך.
