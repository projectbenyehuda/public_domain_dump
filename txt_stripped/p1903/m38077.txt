


	
	משל טז: יונה ופשתן
	




אשר לעצה לא ישמע / באחרית עינו נגרה ולו תדמע



יונה ראתה זרע פשתן / אשר ביד
הזרע נתן / ויפיצנו על האדמה / ותמלא היונה
חמה / ותאמר אל כל העופות / חזקו נא ידים רפות /
אחד מכם לא יגרע / נצא לחוץ נאכל זרע / אשר זרע
אדם עקש / פן יהיה לנו למוקש / כי עבדי פשתים
שריקות / יעשו רשתות דקות / לצוד כל בעלי כנף /
הן בארץ הן בענף / ולבבי נד בראותי הזרע
פעמים / כאשר ינוד הקנה במים / ויאמרו לה על
עצתה / אין לב ליונה פותה / או מחשבה נפלא ממנו
/ עמק עמק מי ימצאנו / ויוציאו עליה דבה / כי
ממרום היתה נסבה / ובראותה כי בזו דבריה /
ותספר לקרוביה ועוזריה / אשר יעשה להם האדם /
וכי קרוב יום אידם / ויאמרו רק זועה / הבין שמועה /
ויאספו כל המשפחה / למצא להם פדות ורוחה /
וכל אחד בלבו מקונן / מפני חמת המציק אשר כונן
/ ויפתו הזרע בלשונם / בשיחם ובטוב מענם /
ויאמר להם: בחורי ביתי תשכבו / בטוב בעיניכם
שבו / אם אשמאילה או אימינה / אל תלכו הנה והנה
/ והיונים במשכנותם בטחו / כימים אשר נחו /
והאיש מן הזרע אשר זרע / עשה מצודים וכל מוקש רע
/ ויפל ברשתו כל עוף למינהו / אך היונים ביום
ההוא / אשר שמעו לעצה / לא היו לקמיהם לשמצה /
ויהיו על ארבותיהם / לא בא קול נוגש להם / ואשר
ליונה בזו ולעגו / יותר ממנה שגו / ויקוננו
בעברי פי פחת / ולא מצאו לכף רגליהם נחת / ולא
נמלטו רק היונים בארבות / אשר נשארו באלה
סבות.

והמשל לאשר עצה לא ישמע / באחריתו
עינו תדמע / עת יראה כי אזלת ידו / ואידו יכבה נר
הודו / ויאמר: לבי ועיני היו בעכרי / כי לא
שמעתי לקול מורי / והחכמים אמרו אשר לב השכל
בתוכם / שמע לעצה חכם.



ואשא משלי ואמר:



ערלות אזני סכל בחקם / ולבו מן תבונות ריק וריקם

ולא נזהר לקול מזהיר, ולכן / יהי נקרא אשר נפל ולא קם

וערום יראה רעה ונסתר / בעת רבים זמן פתע הדקם

והה ליום אשר מאן שמע / ליוחנן גדליה בן אחיקם.
