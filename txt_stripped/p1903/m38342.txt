


	
	משל פד: סוס וסחר ואיש
	




אשר באדם יחסה / לא ישאל ולא ינסה



סוס נאה היה לסחר / ושכנו את פניו שחר
/ לאמר: מכר לי סוסך / את ערכו קח ושים בכיסך /
והנה הסחר ענהו / הסוס בינך וביני מה הוא /
להזכיר דמיו אל תוסף / הנה הוא לך בלא כסף / וזה
ענה: לא כן יהיה / שנא מתנות יחיה / אך את דמיו
גלה אזני / ואת הכסף קח ממני / ויען לו הסחר /
לתתו לך אהיה בחר / אמנם זה הסוס לא מצאתיו /
בעשרים זהובים קניתיו / והוא שוה כפל / כפלים
מארבעים זהובים לא נפל / ואם בחנם לא תחפץ אתו /
על פי מעריך תהיה מקנתו / ונשכימה לכרמים
שנינו / וראשון היוצא לקראתנו / כאשר יעריך
בין טוב ובין רע / לא תוסיף עליו ולא תגרע / ויקומו
בבקר ובמסלותם / איש יוצא לקראתם / מעין ימינו
היה נשחת / ואין לו רק עין אחת / ויאמרו לו: אל
תתן לחטא חכך / הנה סוס זה נמכר כערכך / דין
ערכו אתה פלילו / כסף ערכך עליו וקם לו /
ויאמר: חלילה לי לשקר באמונתי / ערך הסוס
בלבי תרתי וראיתי / כי הוא שוה זהובים חמש / סוס
כמהו ראיתי אמש / בחמש זהובים נמכר / עד הרעה
והאכר / ויזעף הסחר וילך משם סוסו / ושכנו
הזיל זהב מכיסו / חמש זהובים ולא אבה / בעל הסוס,
וזה באהבה / מביא הזהב באהלו / אולי יפתה ויוכל
לו / ויאמר: תן לי הסוס, וזה ערכו / וזה ימרה עליו
חכו / הקונה צועק לשפטים / וחכמי לב אליו נטים /
צוו להשיב הסוס אליו / אם כפר יושת עליו / הסחר
נתיעץ במזמה / בדבר מרמה ותך וערמה / ויספר
אליהם כה וכה / וכי האיש אשר העריכו / אם היו לו
שתי עינים / יעריכנו פי שנים / אך בנקרת עין
ימינו / עת נתן בסוס עינו / בחצי ראיה ראהו / על
כן בחצי ערך יעריכהו / אז אמרו השפטים: יהי לך
אשר לך / כי השכל והבינה נפלה בגורלך / כאשר
צדקת ממנו / כי בעינו אחת לא יעריכנו.



והמשל כן יקרה לאיש
אשר באדם יחסה / מה תשאל ומה תנסה / כי
בהשענך עליו תשבר / בלב ולב לך דבר / כאשר
הסחר אמר לשכנו / סוסי בלא כסף לך הנו / ופי
שנים על דמיו אבה למכרו / ואיש כזה לא חפץ
ביקרו / ועוד תבין משל ומליצה / אל תבוא במשפט
בלא עצה / התיעץ במזמותיך / טרם תדברנה
שפתותיך / כאשר הסחר בהשפטו / הוציא לאור
משפטו.



ואשא משלי ואמר:



אם חסד תוציא מן כלי / מים מצור הוצא תוכל

כי כל מסמר מברזל רע / חשב לכפף אתו סכל.
