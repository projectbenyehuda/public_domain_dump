


	
	משל צט: שעל ועגלה ודגים
	




מלא תך ומרמה / ירבה משטמה

שעל לדרכו הלך ובא / לשוט בארץ
לארכה ולרחבה / ויפן פניו כה וכה / והנה
עגלה באה לדרכו / מלאה דגים מן הים / ולתאות
רוחו בעים / תחבלות חשב במעש / נפשו אותה
ויעש / כמת בדרך עשה עצמו / עודנו בעצם תמו
/ ויפרש כפיו למעלה / נחשב כמת במסלה /
בערבה בכשל ברך / נפל על אם הדרך / וישא
העגל את עיניו / וירא השעל לפניו / כרע שכב שם
ורבץ / כי אחזהו השבץ / ויקח אתו וישאנו /
לא קם ולא זע ממנו / וישא אתו על האמתחת /
ויכסהו העגל כי יחת / פן יכירו אתו רדפיו /
ויסתר אתו תחת כנפיו / פן יראה אתו השאלו /
ויכסהו בבגדים ויחם לו / עם הדגים אשר בעגלה
/ מהם עשה השעל כלה / וימלא כרסו מהם / לא חסה
נפשו עליהם / מהם חלק אכל ויבתר / בלבו אמר:
אכל והותר / לא אשקיט עד בלעי / גם את המלא
תסיעי / ויאכל מן הדגים וישבע / גם בפיו לקח
עד שבע / ויאכל וישבע ויותר / גם את האמתחת
חתר / ויצנח מן העגלה וינס / ובמצודה אשר מצא
נכנס / ויתחבא והטרף בין שניו / ויעבר העגל
על פניו / ולא ראה כי השעל נמלט / וילך השעל
בלט / ויעמד במשעול הכרמים / כי ירא מן הקמים
/ ועקבותיו לא נעו / ומצפניו לא נודעו / וכאשר
היה נסתר שמה / בפחד ויראה ואימה / ויפגע
במקום הזאב / עיף ויגע ודאב / כי לטרף עיפה
נפשו / וישא הזאב את ראשו / ויבא לקראתו
ויברכהו / והשעל כמשפט ענהו / אך חשב
מחשבות להרחיקו מעל גבולו / כי ירא אתו פן יבא
ויקח אשר לו / וישאל לו הזאב: השלום לו / ומה
בפיו נשא למולו / ויען בקול רם השעל / בעקבה
במרד ובמעל / שלום לי ושלום בואך / ששתי מאד
על אמרתך / כי הנה יצאתי המימה / אשר נמצא
שמה / וקפאו המים מקרח / הלכתי כגבור לרוץ
ארח / ובקרח חור עשיתי / ובחור זנבי כשעה חדה
שמתי / וסביב הזנב באו דגים / וכתרוני כסיר
דוגים / ואחזתים כמו במצודה / ומהם עשיתי צידה /
ומשם אכלתי ושבעתי / תלך ותעשה כאשר
עשיתי / כי ידעתי כי לריק יגעת / אאמן כי לא
מצאת / ולהשיב נפשך העיפה / תאכל אחד ויהיה
לתרופה / ויתאו הזאב למטעמיו / ויפרש עליו
שלומיו / ויברכהו הזאב וילך / ויאמר לו: יהי
לך אשר לך / ולא אאמינך להכניס זנבי בחורים / פן
יבאו בי המים המאררים ומרים / וירץ הזאב עד
התעלה / וכלה נקפית מקרח לילה / ויבט החור
אשר נקרו / הרעים אשר שמרו / צאנם כחם היום
להשקותם / ויאמר בלבו: נקיתי ואיתם / חטאתי כי
חשדתי בשעל / ולא שלח ידו במעל / אמנם בפיו
נכונה / ויהי ידיו אמונה / כי אני ראה התעלה / גם
החור אשר למעלה / וישם בחור זנבו / וישב שם על
גבו / להמתין עד בא עתו / ויאספו הדגים
במלכדתו

והשמש הלך ועבר / והקרח עלה וגבר /
והזנב הקפיא בתעלה / והזאב מלא גילה / כי חשב
במחשבת לבו / לאחז דגים בזנבו / ויאמר: מאד
כבדה זנבי / ידעתי כי איתן מושבי / טוב זה מסחר
כסף / אם כל דגי התעלה יאסף / עתה אהיה זריז
ונשכר / אך ירא אני מבני נכר / פן יראוני
והרגוני / וריקם ישיבוני / ויתחזק הזאב לקום
ממקומו / ולא יכל, כי הקרח נפל לפעמו / ויצר לו
מאד מזנבו אשר מצא צרה / ויזעק זעקה גדולה
ומרה / כי לא יכל לקום על רגליו / כי מעדו קרסליו
/ וישמעו הרעים ויתמהו לקולו / ויבאו לראות מה
היה לו / ויתירו הכלבים להסיתם בו / כל אחד הסית
את כלבו / ויפגעו בו וינשכוהו / ומכל צד
כתרוהו / בעל כרחו נשאר שמה / ויכוהו
ויכתוהו עד החרמה / ויתחזק בחמת רוחו / אבל
תם לריק כחו / כי נלחץ זנבו בקרח / והשמש עליו
זרח / והרעים להכות הרבו / וימררהו ורבו /
וישטמהו בעלי חצים / ויזרקו לו כל כלי
מפצים / כי שנאתם מקדם / ויאמרו: לכו ונכחידם /
ולא נשמר דרך נתיבו / והרעים פשעו בו / ויהמו
כלם עליו / ולא יראו מגשת אליו / ויתמרמר בחמת
כחו / כי כהתה עינו ונס לחו / עד כי נשבר הקרח /
ולחבורתם לא ארח / ויברח עד היער / ברוח סעה
מסער / ובכל זאת לא מצא נפש / כי הכלבים עזי
נפש / תמיד אתו הדביקו / ובקול רם עליו צעקו /
רדפו מהר כי תשיגו / וינבחו הכלבים ויצעקו /
וימלט מן הרדפים / כי מן היער נרפים / מלרדף
אחריו שבו / ללכת ביער לא אבו / ויתחבא הזאב
תחת הסנה / מכה פצוע ומענה / ויתאונן הוא
לבדו / ולא מצא עזר כנגדו / ויאמר: אכן סר מר
המות / כי השעל דרכי עות / וזמם להמית אתי /
להאמין אתו נסכלתי / נראה כי בי התעולל / לבקש
בתעלה שלל.

והמשל באיש מלא תך ומרמה / יחשב על
רעהו משטמה / ובקרבו ישים ארבו / רשת טמן על
נתיבו / ושמח כי הגדיל אונו / אוי לרשע אוי
לשכנו / כי בכל עת מבקש תאנה / ואין בפיו נכונה
/ אזניו הכבד ועיניו השע / מרשעים יצא רשע / על
שכניו מדבר סרה / שנאה מקלקלת השורה.

ואשא משלי ואמר:

אני המדבר נקי תך וחף בר / יהי נא בסבר נשיקות בתך בר

ואון מחבר כמו מץ בתך בר / שפת חן יגבר פני זך וגבר.
