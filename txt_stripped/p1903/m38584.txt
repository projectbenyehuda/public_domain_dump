


	
	משל קיד: תנשמת ורכלים
	







מלא דפי ושמצה והוללות / נחשב לבני אדם כרם זלות



תנשמת נברת בשניה / אחרי השרשים
והעשבים אשר תחתיה / ותשלך את העפר למעלה
המנה / ללכת אנה ואנה / למען רבות כהנה וכהנה
/ ותמלא השדה תלי תלים / ויבאו שמה הרכלים
/ לבקש שרשים לרפואה / ויחפרו אחריהם עד אשר
יצאה / ממקומה לקראתם התנשמת / ויאמרו: על פי
שנים עדים יומת המת / וירדפו אחריה עד התל /
ותמלט בתוכו ובהם תהתל / ותקרא: מדוע
דלקתם אחרי / כי ידעתי ביתי סתרי / להציל
ממות נפשי ולחיותי ברעב / אתהלך בארצות
החיים ואל אהיה ללעב / ולא תוסיפו לראותי עוד עד
עולם / והשרשים אשר אתם מבקשים עקרתי כלם /
כי בלבי יום נקם / ואתם תשובו ריקם / בשים
וחפורים ונכלמים / בין הרפאים ובין העמים / ורפאי
אליל כלכם / אתם במעלליכם ובנכליכם / סורו
מפני מהרה / ואל תופע עליכם נהרה / ועליכם כל
אדם ישרק / ומפניכם לא אחשך רק / שפלים ונבזים
מכל אדם / אשר המיר בקלון כבודם / ויפנו משם
האנשים / אל מקום אחר אחר השרשים / ותשלח
אחריהם בתה / למען עשותה מזמתה / לעקר
השרשים מן הארץ / ותאמר: ישכנו נחש גדר פרץ
/ אשר ירדו האנשים לחיינו / לקחת טרף ומזון
לנפשם ממנו / ותלך בתה אחר עצתה / ותעש
ככל אשר עליה אמה צותה / ותבא עד מקום תחנותם
/ ולא שבה עד כלותם / השרשים אשר בארץ כלם /
ותשימם כלה עד עולם / ולא נמצא שרש אחד /
וילכו משם האנשים ביחד / והתנשמת אחריהם
רצתה / להגיד להם כי זאת עשתה / ותאמר: למה
יגעתם לתהו / מי הוא זה ואיזהו / אשר מלאו לבו
לעשות כן / לא נמצא חכם בכם / אני עורת ואתם
פקחים / ממני תקחו בשמים ומרקחים / לי העצה
והחכמה / להכיר כל שרשי אדמה / איזה מהם יפים
לאספלניות / מכל מיני גוניות / לבן אדם ירוק
שחור / ואתם תלכו אחור / יומם ולילה אתם נלאים /
ושרש אין אתם מוצאים / והאנשים לא ידעו מאומה /
תחת הארץ איזהו מקומה / כי לא לגבורים המלחמה
/ ולא לקלים המרוץ לבא שמה / ויבאו האנשים
האלה במחנה / אין קול קשב ואין ענה.



והמשל באיש המתפאר ברב חכמתו /
ביפי בנוי ובקומתו / לאמר בארץ מי כמוהו / מי הוא
זה ואיזה הוא / ולבו ריק מתבונה / מכל אנשי אמנה
/ כי אם מלא דפי ושמצה והוללות וסכלות / ונחשב
לבני אדם כרם זלות / ומחכמתו ילך שחוח / תחת
חטה יצא חוח / ובכל עצב יש מותר ומישור / ודבר
שפתים אך למחסור / ואמר קהלת במשלי הנבואה /
תחת גערה במבין מהכות כסיל מאה.
