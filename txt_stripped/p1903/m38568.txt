


	
	משל קט: סרטן צפרדעים ועגור
	




משל מקשיב על דבר שקר / היטב שאל ודרש וחקר



סרטן גדל במים / ומידו לו קרנים /
כלו הפך שחור / וילך לא לפנים כי אם לאחור /
והצפרדע עומדת בצדו / ומקרקרת באזנו לפחדו
/ ויאמר: מה הקול הזה באזני / הוי על כל שכני /
מקול המון שודד / שמעתי ואתנודד / ללכת אל הדג
להמלט / פן יהיה עלינו שלט / ויאמר לו: אחר עצת
העגור / שארית חמות תחגור / פיו צר וחד / ואכל
הצפרדעים יחד / צוארו ארך עד מאד / לפניו מי
יוכל לעמד / ויבא אל העגור ויאמר / מפחדתי
שערי תסמר / מקול שאגת הצפרדע / ואני עד
היודע / בי אדני שמעני / חנני כי אמלל אני /
ומלטני מן התרעלות / ורב מהיות קולות /
הצפרדעים המקרקרים ופערים / בפיהם ובאזני
סערים / ויען העגור / הן אני גר אגור / בסתר קנה
ובצה / להכרית משם כל שרץ ושמצה / ומאין באת
ערירי / הגידה לי מה שכרי / ותרד על ידי שחד /
פן אבלעך בפעם אחד / ויאמר הסרטן / אני קשה
אני קטן / אני מן האברים מחסר / מן החלב ומן
הבשר / מה תמצא בי מגד / ביום עמדך מנגד / בין
תבין את אשר לפניך / ושים אל הצפרדעים פניך /
ושמת שכין בלעך / ואכלת משם כנפשך
שבעך / והעגור השיב בלשון רכה / אין חכם ונבון
כמוכה / אתה תהיה על ביתי / כל ימי חיי עד יום
מותי / ויכרתו שניהם ברית / על הצפרדעים
להכרית / אותם מן האגמים / ומן הנהרות ומן הימים
/ והסרטן והעגור נחו בלי פחד / כי לא נשאר בהם
אחד / כשבת יחד אחים ורעים / ומאיביהם
הצפרדעים / אשר היו מקדם מקרקרים / אין אמר
ואין דברים.



והמשל באיש מלא חכמה / בל תך און
ומרמה / אם יעלילו עליו עלילות / יבטל עם הולכי
ארחות עקלקלות / וכאשר משל מקשיב על דבר שקר /
הטיב שאל ודרש וחקר / לבטל פרענות בחוץ / יהיה
זריז ונחוץ / פן יגדילו עליו עקב / מן הגרן ומן
היקב / ומאנשי ביתו / וממשפחתו וממולדתו /
וטוב יעשה במעשיו ובשפתיו / כאשר עשה עם
אבותיו / ושלם לרעהו משכרתו / ביום ההוא
ולמחרתו / כאשר גמל העגור לסרטן / ושם אחיו
יקטן.
