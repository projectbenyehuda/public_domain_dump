


	
	משל יא: שעל ונשר
	




החומס לרב גדלו ועשרו / סוף יאבד גם את יקרו



שעל הלך לראות באחד הגנים / ועמו
צאצאיו שעלים קטנים / ויטש הנשר על האחד
וישאנו / וירא השעל כי האחד איננו / ולא
ידע אם גנב או נחבא / או מת או נשבר או נשבה /
וישא עיניו וירא כי הנשר נושאו באברתו /
וירם קולו ויבך ויצעק לקראתו / לאט לי לבני
והשיבהו / והארכת ימים אם תנקהו / למה
תכריתנו ממשפחתו / אשר לא עמלת בו ולא
גדלתו / שמע השכל ודעה / למה תתגרה ברעה /
ויען הנשר: מי אתה אשר אחרי דלקת / ומה לך
כי נזעקת / ולא אבה שמע בקולו / והשעל הלך
בדרך למולו / עד באו אל עץ יער גבה קומה /
וימלא השעל חמה / כי הנשר השליכו אל קנו
לגוזליו / וירא השעל כי לא יוכל לעלות אליו /
ויקרא לו: לאדיב את נפשך / אשרף קומת ארזיך
ומבחר ברושיך / וילך השעל וישם אש בכדיו /
צמר ופשתים יחדיו / עלי אילן ועצים יבשים /
ונערת וקוצים קשים / ולמדורה עצים הרבה / והאש
תוקד בו לא תכבה / ושלח את בערו וביער בצמר
ובפשתים / ויאמר: תמות נפשי עם פלשתים /
וישרף בני עם הנשר ובניו / בטרם יהיה למאכל
בין שניו / יהיה עמהם כלה / הנה לאש נתן
לאכלה / ובראות הנשר האש הגדולה / ויזעק זעקה
גדולה ומרה / ויקרא לשעל: קח את בנך אשר נפשו
לך יקרה / וכבה את האש אשר הצתה בחרונך /
למה נמות לעיניך.

והמשל למתהלך באשמיו / לשלח יד
בשלומיו / חמד וחמס גזל ולקח / יבאיש יביע
שמן רוקח / לבעבור כי הוא חזק ועשיר ועליון /
שלל ובזז עני ואביון / גם כי תחנונים ידבר לו
רש בלשונו / כמו פתן חרש יאטם אזנו / לסוף נבעו
מצפניו / ויקטן גם הוא בעיניו.

ואשא משלי ואמר:



גבר יקרה גבר / שפל ורשע לבר

עת יתקפו זה והוא / ירד ויחליק דבר

פיו: נא אדני לחטא / אל תפקד כי כבר

נחמתי, ואהי / לך אב ואת לי לבר.
