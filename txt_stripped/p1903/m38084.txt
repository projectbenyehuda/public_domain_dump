


	
	משל כג: אריה ושעל
	






נסתר בעת רעה / משמועה לא יירא



אריה חושב בתחבלות / באישון לילה
ואפלות / ודבר בליעל יצוק בו / ויאמר הארי
בלבו / כל מלך יושב בביתו חפשי / ולמה זה אני
עמל ולריק מחסר נפשי / בנע ונד אחר טרפי / ולא
יבא דבר תוך פי / כי אם במקרה ופגע / למה זה הבל
איגע / לשבר גופי במרוצתי / ומנוחה לא מצאתי /
ומי בכל עבדי אשר עלי חמל / טוב מלא כף נחת ממלא
חפנים עמל / ויגד לכל בערמתו / כי מאד עצמה
מחלתו / ויזעק בהמה וחיה / בחלקות ובשפת
רמיה / מהרו ובאו לראותי / לא ידעתי יום מותי /
כי אין דן דיני למזור / רכוש ושלל לכם אבזור /
וכל מחסורכם עלי / ואתנה כאשר תאמרו אלי /
ואשיב לאיש ואיש כצדקתו / לרב תרבו נחלתו / אם
לא עכשו אימתי / הנני שוכב עם אבתי / ויסעו ללכת
כלם / מקטנם ועד גדולם / ויהי בעוד כברת ארץ
לבא / ויגד מטעם המלך וניבו / הבא לראותו יבא
אחד אחד / ואל ילכו שנים יחד / לראות חולה כמוהו /
פן בקולם יבהילוהו / וילך האחד לבדו /
וימצאהו שכב על צדו / הארי בסכו כמת / ויאמר
לו: אדני באמת / לראות שלומך מהרתי אשורי /
ויען הארי: מה לך ולשלום, סב אל אחרי / הלאתני
כמה פעמים / הלא תכלם שבעת ימים / בשרך אכל
ועצמיך אשבר / אל ינוס הקל ואל ימלט הגבור /
ויעש ממנו כדברו / וישסעהו ויאכל בשרו /
עוד זה אוכל וזה בא ויאמר / אדני המלך מחליך
בשרי סמר / ויפתהו בתוך סכו בחלק לשון /
ויעש לו כמשפט הראשון / גם השלישי גם כל
ההלכים אחרי העדרים / לא זז משם עד שנעשה
אברים אברים / וכן כלם עד תמם / לא נשאר רק
השעל אשר בלבו זמם / לדעת ולשמע מה המלך
בלבו דרש / מה יעשה לו כי לא פרש / ויפג לבו
כי לא האמין אשר יאמרו בעמים / לכלם יתן המלך
שדות וכרמים / ואם אמת אתם, למה העיד בעבדיו /
לבלתי לכת שנים יחד עדיו / לא אלך פן אפל בפח
מרמה ורשת / וילך וישב מנגד הרחק כמטחוי
קשת / ויתחבא בדד במסתרים / לשמע שריקות
עדרים / ויאמר: בראותי כי הלך העכשוב / רוח הלך
ולא ישוב / והנה כלם הלכים בלי פחד / אך לא
ישוב אף אחד / כי לא ידעתי מה הוא / והמנהג
כמנהג יהוא / והיתה להם המסלה / כחלקלקות
באפלה / כי מתאנה המלך להם / ולא ישובו
לאהליהם / ובא יום איד לאשר קוו לראות אידו / ולא
יספדו לו: הוי אדון והוי הודו / ועל כי לב השעל
מפחד יתר / ערום ראה רעה ונסתר / ויהי כאשר
כלה הארי לטרף כל הבא / ויתבונן כי השעל
בערמתו נחבא / ויקרא: למה זה ועל מה זה אחרו
פעמיך / הלא שלחת יד בשלומיך / ויען לו: דרך
ההלכים ראיתי / ועקבות השבים לא מצאתי.

והמשל על כן יאמרו המשלים / צוק
העתים בסבות הגלגלים / ובבוא דבר פלא כל איש
ידרש שלומו / שבו איש תחתיו אל יצא איש ממקומו
/ כי לא טובה השמועה / יוסף את דבתם רעה / ויש
נסתר ביום עברה / משמועה רעה לא יירא / דיה צרה
בשעתה בבואה אליך / ויש שבר באהלך.
