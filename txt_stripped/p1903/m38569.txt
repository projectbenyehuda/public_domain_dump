


	
	משל קי: נמלה ועכבר
	




כמו אש על עץ רענן / לפיד בוז לעשתות שאנן



הנמלה עם לא עז / הוציאה על העכבר לעז
/ כי אגרה בקיץ לחמה / ותבא בקצה הערמה /
בתחלת קציר שערים / ללקט בין העמרים / ותחמל
על נפשה בעמלה / להכין בקציר מאכלה / ותרא
שמה עכברים רבים / חטים ושערים מרבים / וימדו
שש שערים להם / להביא אל חוריהם / עד אשר הותירו
משבעם / והנמלה מלאה זעם / ותאמר להם דברי
ריבות / ותקרא למותם חרצבות / לעשות לי כה למה
הסכנתם / מפני אפי וחמתי לא יגרתם / כי אני
הנמלה / אני קצין לי שמלה / אני משל אני שטר /
אני נקם אני נטר / ואין בי ועלי מלך / מכל שרץ על
גחון הלך / למה הרגזתם אתי / לקחת מגרני
ומדשתי / ויען העכבר / לי הגרן לי הבר / למה
הוצאת עלי דבה / ותהי לי לחרבה / הלא בזויה את
לנגדי / ואיך להלחם בי למד תלמדי / שבי וקומי
מעפר התנערי / וגורי באשר תגורי / ולא תדרכי עז
נפשי / אכלתי יערי עם דבשי / והמלאכה היתה
דים / מי יעבר לנו את הים / והנמלה תשיב אמריה /
מים ועד ים קציריה / לשלח צעיריה באדיריהם /
בעכברים למשפחותיהם / ובעמלם אשר עמלו /
בימי שלג מים יגזלו / ותשב אל חברותיה / לספר
להן כל קורותיה / ותאמר: יגעתי ומצאתי / ואכלתי
וטעמתי ושבעתי / מאכל ולחם מטעמים / על הרי
בשמים / והעכברים הרעים / והשרצים הרשעים /
באו עדי נפשי / ומהם לא אשא ראשי / וילקטו
בשבלים ובמאכלים / ויחבלו כרמינו כקטנים
שעלים / צאינה אתי לקראתם / בקרב ידעתי תחנותם
/ וישמעו אליה הנמלים /ויבאו אתן החתולים /
להיות עמן בעזרתן / לארב דמם במחבואתן /
ויבאו עד מקומן / ויחנו שם עד תמן / ויכנסו
בחוריהן הנמלים / ויכו פעמיהם בנעלים / ויעלו
על גב העכברים / כאשר תעשינה הדבורים / עקצו
בבשרם ובנפשם / עד אשר יצאו ממגרשם /
ויציאו החטים והשערים / ויצברו אותם חמרים
חמרים / ומן העכברים מלאו החתולים בטנם /
ויעשו בשנאיהם כרצונם / והנמלים עם לא עז
ונבזה / השמידום וישבו תחתם עד היום הזה.



והמשל באיש עני ונכאה / אצל איש
עשיר גאה / ותחנונים ידבר לו רש / כי באהבתו
דרש דרש / והעשיר יענה עזות / מי שמע כזאת /
לבקש עליו עוללות / להמציא לו עלילות / לגרשהו
ממחיתו / ותעל עליו צחנתו / וכמו אש על עץ רענן /
לפיד בוז לעשתות שאנן / נכון למועד רגליו / כן
העשיר במכשליו / ורדף אתו כל עלה נדף /
והאלהים יבקש את נרדף / ואשא משלי / ואמר
במעני / אלהים שפט צדק מוריש את העשיר / והעני
מעשיר.
