


	
	משל יט: שני צבאים
	




אויל מחריש חכם יחשב / ובמושב נבונים יתישב



שני צבאים עמדו על שפאים / מתלחשים
בלט דבריהם / ואין שמע ביניהם / ואזן זה לפי
חברו / להגיד לו את דברו / ויעבר איש דרך עליהם
/ וילך לשאל להם / למה דברו בלט עצתם / כי אין
המליץ בינותם / גם אם יצעקו בכל כחם / אין שמע
דברי שיחם / כי מאדם הם רחוקים / ויענו: הננו
נדבקים / להמתיק סודנו יחד / ומגלה סוד אין גם
אחד / והעצה אשר לנו / לבעבור כי נרפים אנו.

והמשל לאוילים מחרישים / ויש אשר
יחד מתלחשים / אז יאמרו בעמים / הננו חכמים
מחכמים / ובעת יגלה עצתם / אז יודע כי נבערה
חכמתם / ואמר במשליו קהלת / נחלו פתאים אולת /
ואמר החכם: טוב לי לסבל שוטה בכל עניניו /
מלסבל שוט חכם בעיניו.

ואשא משלי ואמר:

חכם בני ובין דעה / אמת קנה ותם רעה

חשב אויל כצל אליל / ומן עצת כסיל שעה

ואם יצו הרס בנה / ואל נבון דבר שעה.
