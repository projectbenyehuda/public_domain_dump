


	
	משל צא: זאב ושעל
	




טוב בעשיר להתגרות / מבאשר ידיו קצרות

הזאב אמר לשעל / שמע ואחוך את אשר
תפעל / ובא אתי ולא ידע המלך / אשכילך ואורך
בדרך זו תלך / כי ארכו הימים אשר עם הארי
חנותך / ויענך וירעיבך למען נסותך / הנה
ראיתי לך מעבר לגדר אפרחים / מעל אמם תועים
ונדחים / ופתאם תבא עליהם והכיתם / לא תדרש
שלמם וטבתם / ופרץ בגדר מדה לראשך / מהר
ותמצא מעדנים לנפשך / ובארך החור וברחבו /
שעלים הלכו בו / ויען השעל: אם עשיר אדן הבית
והחצר / מכל אשר אמרת לי לא יבצר / ואם הוא עני
ודל / מלשמע עצתך לבי יחדל / פן ירדף אחרי
והשיגני / ובמר נפשו למות יכני.

והמשל תבין אשר אני לך מורה / ואל
יהי לבבך סורר ומורה / כי אם תשכיל ממשלי מוסר
/ מעדת נבונים אל תהי מוסר / וטוב לך בהון העשיר
להתגרות / מאשר תתגר במי שידיו קצרות /
באשר יד העשיר תסרח ועדפת / ונפש שבעה תבוס
נפת / ואם בבקר מאשר נאבד להם יעצבו / בבא עת
ערב לא ידאבו / באשר ימינם עז נאדרת / בפליטתם
הנשארת / אך האביונים מרי נפש המה / לתת דמי
שלום במלחמה / כי אין להם פליט ושריד / וכל גזל
לא יפחיד ויחריד / מרב רגזם ועצבם / לב הארי יצמח
בם / מבלי תת להם נפש / וקבע את קבעיהם נפש.
