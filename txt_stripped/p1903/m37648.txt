


	
	עכבר אמר בלבו : מקאמה
	




גבה מעל גבה





עכבר אמר בלבו: זכר בלא אשה מה טיבו / וכל החי מכל בשר ראיתי / ואשה בכל אלה לא מצאתי / והנה נפשו נכספה / לדרש ולתור אשה יפה / וטובת חן ולבנה אדמדמת / ואשר לא מאכל טועמת / ובעת ללכת יפנה שכמו / ולא יצאה אשתו עמו / ולא מצא ברעיוניו אשה בלא דפי / רק השמש אשר לה יפי / ויאמר: ׳אם כל־ישבי הארץ חשכו בעריפיה / שמש צדקה ומרפא בכנפיה׳ / וכאשר זרחה השמש לפניו / ותשא חן וחסד בעיניו / ויאמר לה: ׳אהבת עולם אהבתיך / על־כן חסד משכתיך / ומהר אמהרך לי לאשה / ולי משפט הגאלה והירשה׳ / ותען השמש במרמה: / ׳הזאת חשבת לחכמה / לקחת אור אשר חשך אמש / וזרח השמש ובא השמש / התעיף עיניך בו ואיננו / תמיד הענן יכסנו / והנני לענן שפחה נכבשת / כי ברצונו אני מסות חשך מלבשת / אך קח לך אשה ממשפחת אמך וממולדתה׳ / ועמד ואמר: ׳לא חפצתי לקחתה׳ / ותאמר: ׳אז אל הענן תשא עיניך / ידעתי כי לא תשיב פניך׳ / העכבר בכליותיו ישתונן / ללכת בחפזה אל הענן / ויאמר לה: ׳היום יגעתי ומצאתי / רעיתי יפתי תמתי / אשר בעצת השמש אקחך / ותהיי לי ולא אזנחך׳ / ותען הענן ותאמר: ׳גבה מעל־גבה שמר / ביד הרוח נתנני / באשר תחפץ תטני / צפון דרום מזרח מערב / בזרע נטויה ובכח רב / ואם אשה
כמני תרץ / והיית נע ונד בארץ / עזב שפחה וקח גברת / כי אני ביד הרוח נמסרת / בא אל הרוח ועמה הסכן / תפתה וגם תוכל, צא ועשה כן׳ / וילך אל הרוח העכבר / ימצאה בארץ מדבר / ויאמר אל הרוח: ׳אל תכלאי / אתי מלבנון כלה תבאי / כי מכל־הנשים אשר לי אחזה / אתך ראיתי לפני בדור הזה / ואשה אחרת לא אקח עליך / ותהי לי ואני אליך׳ / ותען לו: ׳העת באת לקחת אתי / לא ידעת את שפלותי / כי אין בי כח ועצמה / להפיל בגערתי חומה / אם מאבן או מעפר / ריתך
מאתי תופר / באשר חומה ממני גברה / ואם היא בעיניך ישרה / ותוכל אתה לפתות / תהי לך משגב לעתות׳ / וילך אל החומה ויאמר: ׳שמעי / בעצת השמש והענן והרוח דעי / כי באתי לפקדך לשלומים / ארשתיך לי בחסד וברחמים׳ / ותען החומה בחמה: ׳להוכיח עלי חרפתי וכלמה / שלחוך עדי להבאישני / כי באת אלי להזכיר עווני / באשר המה עולים ויורדים / ואבני ועפרי לא נדים / &gt; ובי אין כח ולא גבורה / וכל עכבר ורמש מקורי הערה / וגלו יסודי לשום בי מסלות / אני חומה ושדי כמגדלות / וינפצוני בפיהם וברגליהם / העכברים למשפחותיהם / וישכנו בי אם על בנים /
יותר משתי מאות קנים / ולא יכלתי נגדם לעמד / האשה כמוני תחמד׳ / ובראות העכבר כי נכזבה תוחלתו / ויקח לו אשה ממשפחתו / אשר נולדה בשכונתו בצדו / וימצא לו עזר כנגדו.



והמשל ברדף אחר כבוד להשיגנו / ושררה וכבוד בורחים ממנו / והמתגדל יותר ממה שבמתכנתו / סופו לשוב על עקב בשתו / כי ישוב כל אחד על כנו / כבוד למכבד ונקלה למינו / ואמר החכם הישמעאלי: צבע שיבה יהיה נקלה / עת
גדול השער אותו יגלה / ואמר שלמה: עולה בלא רשות יקח בשנה / כי טוב אמר לך עלה הנה.



ואשא משלי ואמר: בראש כפוף ולב נשבר / ועין שפל רגלים יצעדו ראה אש יעלה ירד / ומים נגרו סוף יעמדו.
