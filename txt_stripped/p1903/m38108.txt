


	
	משל מז: חמור בעור ארי
	




אם רוח גאוה נשבה בו / רוח חכמה אין בקרבו

חמור מבית אדניו הלך לרעות / על ההרים
ועל הגבעות / וימצא שם עור ארי מפשט / ויאבה
ללבשו וידיו פשט / להשליכו לפניו ולאחריו / על
ידיו ועל חלקת צואריו / וירם לבבו ובגאוה ילך
הלך / מתנשא לאמר: אני אמלך / וילך להפחיד
החיות / על ההרים ועל הגאיות / וכארי יטרף להם
ארב וצדה / האדם והבהמה אשר ימצא בשדה /
מפחדו הרעים צאנם עזבו לשמור / כי חשבוהו
לאריה והנה הוא חמור / ועבדי האדמה מפניו נסעו /
גם הצאן והבקר אל ירעו / ואיש לאחיו צעק: שמע
לי הנך / שלח העז את מקנך / כי את אשר ימצא
בשדה לא יחיה / ויהי החמור להם לאריה / ומרב
חדות לבו אשר חשבוהו לארי / עם אדניו הלך בקרי
/ ויאמר כי משא לא ישליך עוד עליו / וחמור אבוס
בעליו / לא ידע, ולא ישוב אל ביתו / אכן יתן
פחדתו / על כל הולך ארבע / כי כן בגאונו קפץ
ונשבע / והקול נשמע: עלה אריה מסבכו / שימו
איש חרבו על ירכו / ברמחים ובחרמים לסבבו / אולי
נוכל נכה בו / נצודהו ביער או בשרון / למלך
נשלחהו דורון / ואדניו אומר אל שכניו: אבדתי
חמורי / ואיככה אבקשנו פן יטרפני הארי / ויהי
היום וילך אדניו / וילט באדרתו פניו / ויבא
באחד ההרים / וירא חמורו כארי במסתרים /
להפחיד כל ראי תארו / ויתבונן כי הוא חמורו /
בקומתו ובזנבו ובאזניו / וכראותו אדניו השפיל
עיניו / ויאמר לו: יען גבה לבך / יהי ארבך
בקרבך / כאשר גברת לשפך דם נקיים /
ותתן חתיתך בארץ החיים / הלך הלכת נטוי
גרון / חלמת ותבין הפתרון / זדון לבך השיאך /
להתנכר מאדנך וממשאך / תכבד העבודה ועלך
/ על עפר תהי משלך / ולא תרדף הצאן כאשר הוא
שביך / ועתה הורד את עדיך / והצגתיך כיום
הולדך / ויהפך הודך לאידך / ותדע מי אתה
ומאין באת / ואם אביך מלך על החיות ראית /
ויתנצל החמור מעדיו / ויקח מאדניו כפלים
במריו.

והמשל בבן־נבל ובן־בלי־שם / אשר
יעבר עליו רוח ואשם / ורוח גאוה נשבה בו / וכל
רוח חכמה אין בקרבו / ועם אשר לא ידעוהו
יעבדוהו / ובלכתו ובשבתו יכבדוהו / כי לא
נכר בלכתו בחוצות / כי אם בלבוש ומחלצות /
ויתגאה עליהם ויאמר: לעבדים מצאתים / ויתן
צמרתו אל בין עבותים / וילך נטוי גרון ולא יזכר
מי אבותיו / וייף בגבהו וארך דליותיו / ואחרי
אשר מומו ומום אבותיו יכירו / כבודו בקלון
ימירו.

ואשא משלי ואמר:

חמור עור הארי מצא, לבשו / והפחיד דב ונמר עת פגשו

וכן בן־בוז בלבשו בוץ וגאה / ולא ידע נשא רגליו וראשו.
