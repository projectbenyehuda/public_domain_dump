


	
	משל פה: ארי וזאב ודב ושעל
	




חרש על רעהו רעה / קללת חנם עליו באה



הארי חלה חליו אשר ימות בו / ויבאו
הזאב והדב לדבר על לבו / וכל חיתו ארץ למינו /
רק שעל אשר מסתתר הנו / לדעת סוף דבר לבו
נפעם / ומה יהיה באחרית הזעם / ואם הוא מאת
המלך יפקד / ולשמע מה ידברו במסתרים שקד /
וישמע באמר הזאב אל הארי / אדני, שמע נא את
דברי / שלח נא השעל להביא צרי מגלעד / ובשמים
וזנים אשר לבך מהם יסעד / כי לו אזן מלין תבחן
/ וידו המרקחת תטחן / והוא ידע מכלנו השבילים
/ ובגלעד מכיר את הרכלים / חכם ונבון אין כמהו
/ויאמר המלך: קראהו / והזאב עשה בעקבה /
להוציא על השעל דבה / להטריחו ולהעציבו /
ולהרחיקו מעל נתיבו / ושם לו עלילות כי לא בא אל
המלך לבקר / ויגדל עליו שפת שקר / והשעל
שמע קולו / פשק שפתיו מחתה לו / באמר למלך:
השעל איננו / כמו חי כמו חרון ישערנו /
וכשמעו דברי הזאב / מלבו ערמתו שאב / לתת
בלב הזאב עצבת / ועל שפתיו כאש צרבת / לעשות
לו כאשר זמם / בכליותיו ישתונן וישב דומם / עד
אשר ראה פני המלך זעפים / הרצים יצאו דחופים /
אנה ואנה השעל לבקש / ויירא השעל פן ינקש /
ויבא בלאט שוק אל שוק כף אל כף נגע / כאשר היה
עיף ויגע / ויאמר: יחי אדני המלך לעולם / אשא
ממנו נסתר ונעלם / טרח דרכי והתלאה / לבקש
לך ארוכה ורפואה / מיום אשר שכב אדני ויתחל /
חדל נחי ונעי הוחל / עד אשר באתי בגלעד ושבתי /
לא נחתי ולא שקטתי / זה ארבעים יום שבעתי
נדדים / וכל לחם צידי יבש היה נקדים / עברתי על
הר וגבעה / ועל רכסים ובקעה / בחשך ואפלה לא
ידעתי איי / קפדתי כארג חיי / אך הרפאים נתנו
לי ואכל / מכל אבקת רוכל / וכלם פה אחד אמרו / כי
לחליך רפואה בחרו / להפשיט עור זאב חי מעל בשרו
/ ויקח חם את עורו / להיות על ראש המלך למצנפת /
והיה לאות ולמופת / ויבאו אהביך / ומדמו יזו
סביביך / ולא תאמר אבד נצחי / וימרחו על השחין
ויחי / ולא אחר המלך לעשות כדברו / ויקחו את
הזאב ויפשיטו את עורו / אך נמלט בעור שניו /
והיה חרפה לשכניו / כי ערם אתו השאירו /
ועצמותיו כמוקד נחרו / השעל תפשו במצודתו /
ויסע כעץ תקותו / ובלעגו יתיצב למלו /
ויחרק שן ויאמר לו / הלא אתה אשר המלך חפץ
ביקרו / היהפך כושי עורו / אך באמרי פיך
נוקשת / וברשע אשר עלי חרשת / ורשעתך עליך
תתך / כי ישבתי לבטח אתך.



והמשל כי שלום לנפש לא דרש / אשר
על רעהו רע חרש / לדבר עליו בזדון / פוטר מים
ראשית מדון / ואחת דתו להמית / מלשני בסתר רעהו
אתו אצמית.
