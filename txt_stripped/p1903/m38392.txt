


	
	משל צו: שפנים ושפן
	




טוב שתהיה לרגלי אצילים / מהיותך מלך לאוילים

השפנים עם לא עז / עמקי שפה ועם נועז
/ וישימו בסלע ביתם / ויבזו ארי המלך בינותם
/ ויאמרו: נבחרה ממשפחותינו אדנים / סלעים
מחסה לשפנים / ואם נשים מלך עלינו / האריה מה
יעשה לנו / הירדף אחרינו בחורים ובמערות /
בסלעים ובצרחים ובברות / וישלחו מלאכים בכל
גבולם / ויצעקו ליום מועד כלם /
ויבחרו להם שפן גבה קומה / זקן וגבור ומלא
חכמה / ויאמרו לו: בחרנוך למלך חבש ועצר /
לעשות כל היוצא מפיך לא יבצר / כי זה הארי אשר
תחת מקולו / לא ידענו מה היה לו / ובטרם יבהילך
ויהיה ערך / ופרקת עלו מעל צוארך / ויען להם:
חלילה לי להיות מלככם / ולא אמשל אני בכם / כי
לא הוד מלכות וכבוד ומשרה / למלך על אמה שפלה
ומשפחה צעירה / ומאד יסכל המבקש להשתרר
עליהם / נשמטו בידי סלע שפטיהם / ולמה
הרגזתוני לעלות / ללכת ארחות עקלקלות / כי
המלכות אשר תתנו לי והממשלה / מן הממלכות
תהיה שפלה / טוב לי להיות אחד מעבדי הארי
הקטנים / ממלך על עם בלא עצמה וכח ואונים /
ואכפי עליכם לא יכבד / ויהיה כבודנו בד בבד /
וישסע בדבריו אותם / ויסכל את עצתם.

והמשל כי טוב היותך לרגלי אצילי עם /
מהיותך בין אנשים ריקים מלכם ורעם / באשר בין
הגדולים שם לך תקנה / ובחבורתם תכונן ותבנה /
ובין הנבזים אשר תשתרר עליהם / גם אתה כאחד
מהם / וכבר אמרו חכמים גדולים / הוי זנב לאריות
ואל תהי ראש לשעלים / וחכם קרא לבנו ויאמר לו:
בני / בכורי אתה וראשית אוני / דע כי הצור יעץ
להגביה שפלים / ולהשפיל אשר בגאונם עולים /
והשררה בורחת מכל מבקש גדלה / על כן תשפיל
עצמך ותנשא למעלה.

ואשא משלי ואמר:

לבך שררה לרדף טרח / אל תתמה אם נרדף ברח

דוד, נפשך תשפיל ותנשא שם / כי לענוים חן ועז פרח

לא ימצא כבוד גבה לב אך דע / כי על נכה רוח יהי זרח

תבחר גדלה תברח ממך אז / תמשילך לשחין אשר פרח

תדמה היות שעל והנך אריה / תחשב היות אריה ואת קרח

אם תאהב להיות בישר ותם / תהיה בעולמך כמו ארח

וכפי אשר תשח ותכנע יום / ליום הדר כבוד לך סרח

גאה וגאון תרחיק באשר רע / טעמו ועל דרכו באוש הריח

יאות להתגאות עלי כל משלים / למשל ועל שמש ועל ירח.
