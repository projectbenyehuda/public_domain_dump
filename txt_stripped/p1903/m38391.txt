


	
	משל צה: פסל ואיש
	




מענה רך ישיב חמה / ולא תבא למשטמה

פסל פסלו יצרו וכלהו / חרש עשאהו
ולא אלהים הוא / עשיר אחד דרש דרש / לקנות אתו מן
החרש / כי בחר לעשתו אלהו / יפיו קומתו וגבהו /
למאד בעיניו ישר / חקוק ומשוח בששר / ואיש
אחר בשכונתו / האמר לקנות אתו / לעשתו למת
מצבה / על אבן משכית טובה / ויכסה בו קבר קרובו /
אשר הוא כנפשו אהבו / והפסל לאשר פסלו / בכה
ויתחנן לו / לתת לו גופו בשאלתו / וכבודו
בבקשתו / ויאמר: הנני שלך ואתה בי עצר / אני
חמר ואתה יצר / ובידך כל רוחי והודי / וקלוני
להמיר בכבודי / ידיך עצבוני ויעשוני / במחוגה
ושרד תתארני / מעץ לא ירקב נבחרתי / מאשר
יקרתי בעיניך נכבדתי / ומלאכת נקבי ותפי /
פעלת וזרועי וכפי / מראשית דבר ראה מה סוף /
ולמעשה ידיך תכסוף / ואל תתנני בנפש צרי /
אשר אמרו להיות בעכרי / לקנותי ממך לכסות מת /
ממני, נבלה סרוחה ונרדמת / ולחלל יפי יפעתי /
מי יתן תבא שאלתי / ותמכרני לאשר אתי יירא /
ואשר בשמי יקרא / כי בכבודי לך כבוד / והאיש
אשר אתי יעבוד / יכבדך כי לי יתפלל / ואת
כבודי לא יחלל / וכלם באיים יחוו / כי למעשה
ידיך ישתחוו / אם לא תחפץ פעלך ויקרו / מה
הועיל פסל כי פסלו יצרו / וישמע יצרו את
תפלתו / ולעבדו ולמכבדו מכר אתו.

והמשל להחניף כל אשר תגבר ידו
עליך / להמתיק לו סוד להמשיך לבו אליך / ולפתות
לבב עקש ולמצא שלומים / למען ייטב לך והארכת
ימים / ואמר אחד מן החכמים / אשר מחזה שדי יחזה /
מתר להחניף הרשעים בעולם הזה / וחכם אחר את
אזני גלה / תעלא בעדנה סגיד לה.
