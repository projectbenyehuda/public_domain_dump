


	
	משל מה: שלך ועופות
	




איש מלא חמס / בארץ מגוריו הוא למרמס

השלך בין העופות / היה לבז ולחרפות /
כי היה לו חרפת אחור / משליך צאתו מן החור / רגע
אחד אלף פעמים / מבאיש ריחו בכל פעמים / וכאשר
הלך עמהם / ויטמא את מחניהם / וימררוהו
וישטמוהו / ומתבל ינידוהו / לאמר לו: מחוץ
למחנה יהי חנותך / ושבת וכסית את צאתך /
באשר לך מכת טחורים / וחסמת היא את העברים /
ומשפט מות לך נחזה / ויעמד על כנו נבזה /
ובהיותו ללעג ושחק / וירא את המקום מרחק / על
ראש גבעה נשאה / שם ישב בנפש נכאה / גם שם לא
מצא מנוח / כי בעלי כנף שלחו לו שלוח / ויאמר:
כל עוף בארץ מגורי אים / אעופה ואשכנה באחרית
ים / וארחיק נדד בארץ מרחקים / ולא אשמע עוד קול
לחצים ודחקים / ואכבדה שם תחת היותי נבזה
ונקלה / ואין אחד מתחזק עמי על אלה / ויעף ואל
הים הלך וקרב / ויפגע בו הערב לעת ערב / ויאמר
לו: אנה תלך השלך / הלא תבקש מנוח אשר ייטב
לך / ויען: מנוח לי אחזה / מעבר לים הזה / כי
רעיך גרשוני מזבולי / וטמא טמא קראו לי / ויאמר
הערב: התשא עמך אחרך / המרחיקך מארץ מגוריך /
ויען השלך: את אחרי / לא אוכל להניח אחרי / כי
נשאתי חרפת נעורי / ויאמר הערב: הנה הוא לחרפה
וללעג לך / בכל מקום אשר תדרך כף רגלך /
וכאשר בגללו נהפכו לך הנה אהבים לאיבים / גם
שם ירחיקוך כל עוברים ושבים / ולא יועיל לך רב
נוצה וגדל אבר / כי באפס יד תשבר.

והמשל באיש מלא חמס / ובארץ מגוריו
יהי למרמס / באשר דרך לשונו קשתו שקר / הולך
רכיל מגלה סוד וחוקר / מה זה על חברו ידבר /
והמון שקרים עם הדבר יחבר / ובעת אשר יראה כי
רעיו עריו / למה ירחיק את ארץ מגוריו / אשר לא עשה
טוב בעמיו / וישלח יד בשלומיו / כי שם
ישנאוהו כמשפט הראשון / מות וחיים ביד
הלשון / וחכמינו אמרו על המספר לשון הרע / זאת
תהיה תורת המצרע / ואמר הישמעאלי: בטרם
תוציא דברך חשב מה סופו / כי כאשר יסבל איש
מכות אש לתקנת גופו / ראוי לו לטרח בתקנת
רפאות נפשו / ממקרה פתיות ואכזריות ולהסיר
מלשונו מוקשו.
