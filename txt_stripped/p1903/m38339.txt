


	
	משל פא: חלה ובתו ורפא
	




חשבים להכשיל בתחבלותם / יפלו במועצותם



רפא הקז דם מחלה / ויאמר לו את
הדברים האלה / הדם יהי בידך למשמרת / עד יום
שלישי באשמרת / והיה ביום השלישי / אראה אם
תצא חפשי מן החלי אשר משלך / ואיך יקל מעליך /
בדם אכיר כל מכשלה / כל נגע וכל מקור מחלה / כי
כל סבות הנגעים / הדמים מודיעים / ירק שחור לבן
אדמדם / כי נפש כל בשר הדם / החלה צוה לבתו /
לשאת הדם מאתו / ולשמרו כבבת עינה / פן
ישפך אנה ואנה / כי כל אשר הכלי הפך / באדם
דמו ישפך / הדם לקחה ותשאהו / תחת תבתה
שמתהו / חתול הלך לפי תמו / ושפך את דמו / אז
רפו ידיה כהתה רוחה / ותאמר: אוי אבד נצחה / איך
אמר לאבי איננו / וכי יפקד מה אשיבנו / אשר
שפכו יטרפנו האריה / אם בהמה אם איש לא יחיה /
סעפיה לחצוה / כליותיה יעצוה / להריק בכלי
מדמה / כמדה אשר היה שמה / זממה מקיז
ותביאהו / בחדריה ותצוהו / להקיז מדמה
בכלי / ולדעת מקור החלי / הרפא בא בזמנו /
והחלה הגיד לו ענינו / וזה שאל אתו לאמר / הדם
אשר צוה לשמר / והחלה השיב כי לא נגרש / וגם
דמו הנה נדרש / בתו את הדם הביאה / וכאשר
הרפא ראה / אדם הדם ומראהו / דם יחשב לאיש ההוא
/ חרה אפו ויתעבר / ויאמר לו: הנך מעבר / לא
יהיה כזאת ולא נהיה / בגוף זכר ולד ושליה / לא
אתמה אם אתה חלה / עוונותיך הטו אלה / לאיש
כמוך רשעך / ושמת שכין בלעך / הרפא שב אל
נתיבו / ויתעצב אל לבבו / החלה כי לבו תעה /
בתו קרא והשביעה / פעמים רבות לדעת / אם דמו
בלא מגרעת / הביאה כאשר קבלתו / או אם על הארץ
שפכתו / ודם אחר שמה במקומו / על כן לא עמד בו
טעמו / הנערה ענתה: לא שפכתי / הדם ושפכו לא
ידעתי / אך מצאתי הכלי נהפך / וכל הדם מתוכו
נשפך / ופן יחרה אפך עמי / תחת דמך שמתי
דמי / ויאמר: חשבתיך חומה ואת דלת / מסבין
קויתי סלת / הנה חשבתיך מסגרת / ואת מאיש
מעברת / ותזרעי מאנשים זרע / קויתי טוב ויבא
רע / הרבה השבעתיך בחדרים / לבלתי לכת אחרי
הבחורים / אך בחבלי ערמתך נוקשת / הוד והדר לא
לבשת.



והמשל בחשבים להכשיל אחרים
בתחבלותם / פתאם יהיו מכותם ובמארב אשר הם
יארבו / יתפשו במזמות זו חשבו / ותגלה רעתם
ומעלם / ברשת זו טמנו נלכדה רגלם / חשב ללכד
בחבלי שוא עמיתו / שוא תהיה תמורתו / כאשר
קרה לנערה במעלה / כי אביה חשבה לבתולה /
וידע כי איש מקורה הערה / ויבז הנפש אשר
היתה בעיניו יקרה.
