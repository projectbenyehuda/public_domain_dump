


	
	משל מא: טוס, חטים ועגור
	




גאות אדם תשפילנו / ולא יוכל לדון עם שתקיף ממנו



טוס הנקרא ברבור / מצא מן החטים
צבור / ויבא העגור גם הוא / ובדגן שלח פיהו /
ויאמר הברבור אל העגור / למה ממני לא תגור / ומי
אתה אשר לי תתחבר / מה לתבן את הבר / ומי
כיפיי בכל העופות הוה / ואל מי תדמיוני ואשוה /
ומי ימצא בי כל שמצה / ככתנת פסים לי נצה /
והשרים והשרות / זנבי בראשם לעטרות / ממנו
יעשו כובעים / וראה אתה אם לי רעים / כי מי יוכל
לספר הודי / אשרי עוף לו אגיד סודי / בראשי כתם
פז לעטרה / לי הכבוד והמשרה / ליפיי כל עוף לא
דמה / לי הנצה לי הרקמה / ולזנבי לא נמצא ערכה /
אמתים וחצי ארכה / ואתה דומה לגנב / אשר
כרתו ממנו זנב / למה לא תירא ולא תחת / לבא עמי
במחנה אחת / סור נא ונהיה לשני מחנות / חסרון לא
יוכל להמנות / ויען העגור: אם סעפי בי החשו /
בדיך מתים יחרישו / מה תתהלל בזנבך ונצותיך
/ יהללך זר ולא פיך, נכרי ואל שפתיך / ואם אתה
בעיניך יפה / לפי גדלך אתה שפל ורפה / שפל בקול
שפל בקומה / לא תצא ביד רמה / מגערת כל עוף
תתרופף / ולא תוכל כמוהם לעופף / על כן תדור
בחרבה / והנה קמה אלמתי וגם נצבה / צואר ארך
ונטוי גרון / בין כוכבים אשמיע רון / שוקי ישרות,
עלי נאמר / זאת קומתך דמתה לתמר / וזנבי אשר
קצרה / לא ברפיוני אך בגבורה / לתת על כל שנאי
אימה / אאזר מתני למלחמה / ותפארתך אך לך
לדפי / שקר החן והבל היפי / כשמע הברבור
קשות / וידם כי עת לחשות.

והמשל באיש המתהלל ביפיו / אם
לבו ריק מחכמה זה דפיו / כי גאות אדם תשפילנו
/ לא יוכל לדין עם שתקיף ממנו / ואמרת החכם
הישמעאלי מה נעמה / לא הנחיל האב את בנו דבר
טוב מחכמה / ומתפאר ביפיו ועשרו מכה בסנור /
כי מה תועיל פתיחת העין אם הלב עור.
