


	
	משל ט: כלבה הרה וכלבה
	




המשליט זולתו עליו / ירכיבהו עד יהיה בעליו



כלבה אחת בית ללון חומדת / כי קרבו
ימיה ללדת / והסתו הולך וקרב / ותבא אל רעותה
בערב / אשר בית וחדר בגבולה / ותבך ותתחנן
לה / לשמע את צעקתה / ולאסף אותה הביתה / עד
אשר תפלט מן התלאה / ותריק בטן המלאה /
ותחמל עליה ותאמר: הנה ביתי לפניך / עשי כטוב
בעיניך / וצאצאי מעיך בקטנותם / שבי עד גמלך
אותם / והכלבה שם נחה ושקטה / עד אשר ילדה והמליטה / בדמותה כצלמה
כלבים / ובעיני גברת הבית לא טובים / כי לשמים
עלתה צעקתם / ולא זרו מתאותם / וירבו ויעצמו
/ בשר אכלו ועצם עצמו / שבעו וירם לבם /
ובקרבם ישימו ארבם / ולא ישאירו בבית מחיה /
ותאמר הגברת אל הנכריה: רעיה / צאי מביתי
ומעלי סורי / וגורי באשר תגורי / ואם אני קרבתך
אוהבת / לא נוכל יחדו לשבת / כי אין לי עם ילדיך
נפש / והכלבים עזי נפש / ותמיד מכעיסים אותי /
ולזעקתם נגזלה שנתי / אויה לי כי גרו משך עמי
/ שתי מימי אכלי לחמי / ותקצר נפשי בעמלם / ואין
לי מנוחה בעולם / ותען לה בשפת חלקות הנכריה /
גברתי הכבדה והנקיה / תנתן לי נפשי
בשאלתי / ועמי בבקשתי / אם קרבתי תשנאי
אצלך / שלח תשלח את האם ואת הבנים תקח לך /
כי אמרו לי צאצאי והמוני / שובי אל גברתך
והתעני / עוד לא יחרץ כלב באזניך לשון /
ותיטיבי לי חסדך האחרון מן הראשון / והניחי לי
עד יעבר החרף והקר / ובעת הזמיר בית אחר נחקר /
ואל תורידי שיבתי בדם שאולה / פן יאכלנו חרב
ביום וקרח בלילה / אם לעת כזאת בני מבלי אהל
ישארו / וכלבת הבית עליה רחמיה נכמרו /
ותאמר: שלי שלך וביתי ביתך / כשאלתך
ובקשתך / ובבא עת הזמיר / את לבם שמו שמיר /
הנכריה ובניה / והגברת אומרת באזניה / הימיני
או השמילי / צאי את ובניך מאהלי / לא תקום
פעמים צרה / למה את מתנכרה / לשלם רעה תחת
טובה / והנכריה בזה את ניבה / ותועץ בלבה
ושכלה / למען הרחיקה מעל גבולה / ותאמר לה:
הבית שלי / צאי את ממגרשי וגבולי / והפרדי
נא והנלוים עליך מעמי / מהרו צאו מתוך עמי /
ויבהילו להוציא אותה / ותלך נכאבת מביתה /
ובני הנכריה יירשום / ויבאו הרעים
ויגרשום.



והמשל באיש נעים וערב / אשר בעל
תהפוכות יקרב / בקרב ביתו עשה רמיה / ומערים
להונותו בצדיה / ובעת הלך ושוב / דברי מרמות
יחשב / ולהתעות אדניו יאנח / כאשר משלום נפשו
תזנח / ועיניו יתנו מימיו / והאדון עליו יכמרו
רחמיו / למלאות כל תאותו / ולהיות שורר בביתו /
והאחד אשר בא לגור יערים / לי הבית ולי החדרים /
ואתה בביתי אורח / וזה בקולו מר צורח / ואם יום
שמח בחבורתו / שנים יאבל למחרתו / ואמר החכם
בחכמתו / יש צדיק אבד בצדקתו.



ואשא משלי ואמר:



בתי אמונים ותלם / שממו ואין איש בכלם

דבר נכחה וישר / הזמן הדמם וכלם

חנף ושקר יחיה / תם מת ונסתר ונעלם

ירדו שאולה ישרים / עלו נלוזים בסלם

מתוק כמר, אור כחשך / מים סרוחים כטלם

נקלה כנכבד לכבד / שמשם להאיר בצלם

אם איש עמיתו יקרב / לו רע עבור טוב ישלם

ואשר יגרה בחנף / מי הקשה לו וישלם

משיב בטוב רע יעור / מראות מאורות בהלם.
