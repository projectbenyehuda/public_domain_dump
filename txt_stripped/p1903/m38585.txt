


	
	משל קטו: זקן ובניו והגמון ואנשיו
	







המכבד אביו ואפלו בשתיה / לא יהיה שכרו רמיה



איש אחד בארץ היה / זקן ועשיר מכל
בריה / אשר בכל תכונתו / ובניו היו בביתו /
ויקרא להם לצוות / שמעו בנים מצוות / אביכם כי
זקנתי / ולא ידעתי יום מותי / האזינו נא אלי
רוזנים / ושמעו אם דברי כנים / הקשיבו לי כל
בני / ממון יש לי במטמני / וקחו נא הכל לכם /
וחלקו אתו ביניכם / ולא תעזבו אתי מאומה / רק
היין תנו לי מימים ימימה / והזקן צויו גמר /
ויעשו לו בניו כאשר אמר / וייין בכל יום לו
מצאו / הרבה שנים עד כי נלאו / ויקחו עצה ביניהם
/ וישכרו את אביהם / מיין לבן ואדמדם / והוא
לפניהם נרדם / ויקראו אל השכנים / ויאמרו:
באו הנה רדפי שלמונים / וראו מה הקרה לנו / כי
בעוונותינו מת אבינו / וילכו ויבכו אתו /
ויעשו לו כהלכתו / וילבישוהו תכריכים בניו /
וישאו אתו במערה אשר לפניו / ויהי ביום ההוא
ויבא שם ההגמון / ועמו היה רב וקול המון /
וישבו במערה כתות כתות / ויתחילו לאכל
ולשתות / ויקץ האיש מיינו / ויקם לילך למעונו
/ ויראו אתו מרחק כלם / ויניחו את מאכלם /
ויאמר להם ההגמון: באמת / הנה האיש הלזה מת /
ויקומו האנשים וייראו מאד / וילכו ויעזבו את
כל המאד / ובניו באו במערה לראותו / וימצאו
עשר גדול אתו / וישמחו שם שמחה גדולה /
וילכו לביתם עם סגלה.

והמשל לאיש הממלא ממעדנים בטנו והמעים / ולשתה נתן יין.
