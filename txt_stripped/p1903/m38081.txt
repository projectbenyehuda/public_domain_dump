


	
	משל כ: נשר, חמט וערב
	




החכמים בערמתם וסודם / יתחכמו על בני אדם



חמט הלך לפי תמו / ובכל אשר ילך
ביתו עמו / כי הוא נשאו על גבו / והנה יצא ראשו
ורבו / קרניו זקופות בתוצאותיו / ממנו היו
קרנותיו / והנה הנשר מרחף לפניו / ויתבונן לו
וישפיל קרניו / ולבו מחרדתו יתר / ערום ראה
רעה ונסתר / ויתחבא מהר באהלו / בעלית
המקרה אשר לו / וירא הנשר כי נסתר ונחבא /
ולא נראה דרך הקבה / ויבא עד האהל ויהפכהו
למעלה / ויבן כי שם נחבא אשר ראה בתחלה / וכי
שמה תחנתו / ישאהו על אברתו / ויאמר כי לא
ימלט מקרבו / עד עשותו ועד הקימו מזמות לבו /
וכאשר נשאו בין השמים ובין הארץ / וינסה
לפרץ בו פרץ / ולשברו בפיו וברגליו / וירא כי
לא יכל אליו / והנה נצב לקראתו הערב / וילך הלך
וקרב / ויקרא לנשר ויאמר: בגד בך השכל /
הנך ראה בעיניך ומשם לא תאכל / ומה תעיר על זה
חמה / לא לגבורים המלחמה / כי מי פתח דלתי
פניו / גאוה אפיקי מגניו / יצוק כמו אבן גבו /
העשו יגש חרבו / ולא לך חכמת השבירה / מעוף
השמים נסתרה / רק ממני עתה באתי / בחכמתי כי
נבונותי / והשלל מאד נחמד / אשר אליך נצמד / ואם
תתן בו לי חלק וחבל / תשברנו כשבר נבל /
ותראה הנסתר בערמתו / אז תבין נתיבות ביתו /
ויען הנשר: אחרי אשר ראשו מחצת / כן משפטך
אתה חרצת / ולהסיר מכסהו הורני / חלק כחלק לך
כמוני / ויאמר הערב: לך על שן סלע ומצודה /
ותנפצהו משם בירידה / ותקלענו ארצה כבקלע /
ותנפצהו אל הסלע / והיה כאבק דק מכסהו /
והבשר שם תאכלהו / והנשר שמע לעצה /
וישליכהו מן הסלע ארצה / וטרם מידו השלך /
ערב בתחתית הסלע הלך / וימצא החמט מפזר
ומפרד / ויאכלו ולא פלט ולא שרד / ולנשר צר מאד
אשר שמע בסודו / כי יכין לערב צידו / ושללו
בערמתו ירף / וישלך משניו טרף.

והמשל מאד יערב / טובה חכמה מכלי
קרב / והערומים בחכמתם ובסודם / ישחיתו אשר
בנו בני אדם / אף כי יבטח אדם במבצריו אשר עשה
ובמגדליו / אם שונאיו יערימו סוד עליו / ויהי
בלבו מסלול חכמה מישר / פן יתחכמו לו כחכמת
הערב לנשר.



ואשא משלי ואמר:



בהון יריב אנוש רעו והודפו / ועבר זר ואספו בכנפו

כמו עצם יריבון בו שנים / כלבים, בא שלישי וחטפו.
