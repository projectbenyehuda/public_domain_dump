


	
	משל יד: חמור, כלב ואדם
	




הקנאה תביא להתאוות / דבר בלתי יאות



כלב קטן היה / בית הפקיד מן הקריה /
פתו יאכל חיקו ישכב / עמו ילך העת ירכב /
ובבית חמור ארב / כי האדון הכלב יקרב / וכי
ישחק בו לרגעים / כאשר היו שני רעים / ויאמר
החמור בלב / לא נופל אנכי מכלב / אלך גם אני אל
אדני / אולי ככלב יקרבני / ואחבקהו ואנשקהו /
ובלשוני אלחכהו / ואשים עיני אצל עיניו / ואת
פני למול פניו / ואתחבר אתו / לעמת מחברתו /
וכל מלבושיו אגאל ואדרך / מחוט ועד השרוך /
ולמצא חן עליו אזעק / כאשר הכלב חרץ וצעק / בזה
הדבר אבחן / אם אני ככלב אמצא חן / וכאשר בא
אדניו הביתה / החמור השלים לבו עלתה / וירץ מהר
אצל אדניו / וילחכהו סביב אזניו / ויחבקהו
בשתי רגליו / וירם קולו עליו / וראשו אל ראשו
נצמד / כאשר מן הכלב למד / האדון ירא וחרד / כי
החמור בו עולה ויורד / ומפה אל פה הקיפו / עד כי
מכסאו הדפו / ובצוארו חבקו כרביד / ומשאו
עליו הכביד / ויאמר האדון: מי אתי אלי / הוציאו את
החמור מעלי / כי גדלה לו החרדה / ותקהל כל העדה
/ זה ברמח וזה במקל / וזה באבנים יסקל / זה
בשבט מכה החמור / זה אומר הך אך את נפשו שמר /
וירדפו החמור עד אבוסו / הלם כל גוו
וכרסו.

והמשל לאיש הולך שובב / רחב עינים
וגבה לבב / וישתרר בגאותו / כי לא יתכן לאיש
כמותו / וברצותו להיות מושל / פתע פתאם יבא
כשל / ויהדפנו ממצבו / ואין איש מאזין אל ניבו
/ ומשל שלמה לא נגרע / כי לא תהיה אחרית
לרע.



ואשא משלי ואמר:



לפני שבר איש דרכיו מעקש / לפני כבוד שפלות לו למוקש

נא זכר מי מצא מלוכה ונבא / כאשר שב מן האתונות לבקש.
