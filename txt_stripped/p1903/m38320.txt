


	
	משל עא: זרזיר ובת־מלך
	




איש מלא חכמה ודעת / לו עין ראה ואזן שומעת



זרזיר בת־מלך גדלתו / ולדבר צחות
למדתו / ותביאהו בחדריה / בבית אביה בנעוריה
/ ויחכם מכל העופות / בדברי צחות ובמליצות יפות
/ ויתבונן בשמעו חדשות / עת לדבר ועת לחשות /
אך לא יכל לצאת ולבא / ויתעצב אל לבבו / כי תבת
קני ברזל עשתה לו / ובין הקנים נשמע קולו / יום
אחד פרש מתהלך / בחצר המלכה והמלך / ולעבר
הים ישים פניו / ויצו על ביתו ועל בניו / ועל
ארצו את המלכה / וחלק משמן חכה / לאמר: שמע
אשמע בקולך / עיני אשית על אשר לך / זרזיר הבין
בפרש / כי לעבר את הים דרש / ובלכתו דרך
אצלו / בשמו קרא ויאמר לו / הנך את הים עבר /
נכבדות בך אהי דבר / אם תראה עוף כדמותי /
וכצלמי, אז תזכר אתי / ובשמי תפקדהו לשלומים /
ושאל לו בחסד וברחמים / לצאת ממסגר כלאי עצה /
באשר נפשי בו קצה / כי סביבותי גדר וחומה / גם
פה לא עשיתי מאומה / כי ממשפחתי גנבתי /
ובין אנשים סגרתי / עשה עמי חסד ואמת / עצור
באסור חשוב כמת / הנני עצור לא אוכל / לצאת מזה
ומאכל / לא יערב לי, עד לחפשי / אצא, אז אשא ראשי
/ על כן אני משביעך / לזכרני במדעך / ובשובך
הגד לי אמרותיו / ומה יהיו תשובותיו / הפרש
משם נפרד / וילך ואל הים ירד / ועוף יעופף
לעמתו / כצלם עוף המדבר אתו / ובטרם יקרב אליו
/ וירם את קולו עליו / לאמר: לשלום פקד אתך / עוף
כצלמך כדמותך / אשר הוא בין אנשים גר /
ובמסגר כלאם סגר / השביעני ממך לשאול / עצה
איך נפשו לגאול / להוציא ממסגר אסיר / וישע
לנפשו לא תחסיר / ואם יוכל עוד ביער להראה /
ושנה את בגדי כלאו / ויהי ככלותו לדבר / העוף
את כנפיו חבר / ויפל על הפרש בעלפה / כאשר
היה ממות נספה / והפרש אתו לקחו / ולהשיב אליו
רוחו / שם את עיניו למול עיניו / ויז מים חיים על
פניו / ואחרי כן שם אתו בחיקו / אולי יחם לו,
וזרקו / בראותו כי אין תקוה / על זה היתה נפשו
דוה / כי לא ידע מה היה לו / ויפרד מהר מאצלו /
ללכת אל מחוז חפצו וישב / עד משפחתו בעיר
מושב / ובשובו למלך שמחהו / ויחבקהו
וינשקהו / ויהי הולך עד כה ועד כה / הזרזיר
הבין כי מדרכו / שב הפרש וישאלו / אם זכר את
ימי אבלו / ואם מצא עוף כצלמו / מה פעל ומה דבר
עמו / ויספר לו את כל הקרות / וכי היה יום עברות /
לעוף כשמע אמרותיו / ואיך נתקו מזמותיו / כי
מבלי ארב וצדה / נפל מת על פני השדה / וכי
להשיב רוחו במים הטבילו / ויכסהו בבגדים ולא
יחם לו ובטרם תעלה צחנתו ובאשו / השליכו בגעל
נפשו / וישם הזרזיר על לבו הדברים / וכבא בת
המלך בחדרים / ולא שמעה העוף מצפצף וחמתה
בערה בה בקצף / ותפתח את התבה ותראהו /
ותשלח את אמתה ותקחהו / והוא כמת משכב
ותתאנף / ותרא כי לא היה נדד כנף / כאשר
אחזתו השבץ / במקום אשר כרע רבץ / ותזרק מים
על ראשו / אולי תוכל להשיב נפשו / ותקחהו
ותשת אתו בחצנה / ולא קם ולא זע, והיא בדומה
שכנה / ותשליכהו מידיה ארצה / ומדמעת עיניה
פניה רחצה / ותבך עליו ותקונן לו / ותפרד
מבכה מאצלו / כשמע העוף כי הלכה / ועוד לו אך
המלוכה / ויפרש כנפיו ויעף / ולנוס באברותיו
לא מועף.



והמשל באיש מלא חכמה ודעת / לו עין
ראה ואזן שמעת / לשמע מה בחוץ ידברו /
ועפעפיו נגדו יישירו / ובטרם יבינו זממו בחוץ /
להחל וכלה יהיה נחוץ / במשא ובמתן במקח
ובממכר / וכזרזיר יהיה זריז ונשכר / אשר
בחכמתו נמלך וילך הלך / מבית הסורים יצא
למלך.



ואשא משלי ואמר:



אנוש שכל לפי שכלו יהלל / והסכל לפי סכלו יחלל

ויש חכם במראה לא בחידות / ויש פתי בסות סכלו מגלל

ואולתו לעמת מחברתו / ולא תסור ואם כל יום יקלל

ונבון ברמיזות בן דרכיו / ויכירו תבונתו במעלל

כמו זרזיר אשר נפל לחלל / ונפשו היתה אליו לשלל.
