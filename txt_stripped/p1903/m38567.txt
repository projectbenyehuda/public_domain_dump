


	
	משל קח: חלדה ועוף ושעל וארנבת
	




איש מלא רע ותהפוכות / משלח מדנים בין אחים ומכות

חלדה דרה בעקרי בתים / ומן החרכים
הציצו חטים / והעוף אכל אותם קמעא / ותחשב את
נפשו לקבעה / ותאמר: מדוע נפשי קצרה / ולעוף
הרבה חטה ושעורה / וימי משתה ושמחה / כל ימיו
בלי פרץ וצוחה / לשחק אמרתי מהולל / והיתה לי
נפשי ובשרי לשלל / באות נפשי אכל בשר /
וברעב מזון לא אחסר / ותלך אליו בלט / ומפח
יקוש הצפור נמלט / ויעף ממנו דרך קצרה / פן
יהיה לו לחץ מטרה / רחק כמטחוי קשת / והחלדה
קראה לו בארשת / ותאמר: למה ברחת ממני / ואם
אשה תחפץ כמוני / ופרשת כנפך על אמתך / כי
אהבתי מעון ביתך / ועליך השלכתי יהב / ואלי
קראתיך באהב / ותהיה בכל עבר רדה / לכה דודי נצא
השדה / ואני דרס ואכל / ואתה טרף ומשכל /
ונחלק בגורל השלל יחד / טובים השנים מן האחד /
והעוף השיב לה דבר / כי הנה הסתו עבר / והגיע
עת הזמיר / לא אחליף ולא אמיר / אשתי הטובה ברעה
/ לא אוסיף עליה לא אגרעה / עיני וזוגתי בחייה /
אפלו בצבי או בצביה / ולא תמצאי בי חבר חבר /
ולא תוסיפי עוד אלי לדבר / ותען החלדה / אני
שכולה וגלמודה / וממני לא תפחד / כי בביתי
הבאתי לך שחד / ולא ידעת את שפלותי / כי ידעתי
אמרתי אחרי מותי / אשר פי זרות עמקה שוחה /
הטתו ברב לקחה / לבא אליה עוד ולגשת / ותהי לה
לכלמה ולבשת / ויהי כאשר הם מדברים / והנה מן
סבכי היערים / באו שעל וארנבת / ויכו את
החלדה עד מות / והעיט נמלט מכל פגע רע / כי לא
איב לו ולא הרע / והחלדה אשר חשבה נפשו
לקחתה / הושב גמולה בראשה / ונעשה לה כאשר
זממה / עניה סערה לא נחמה.

והמשל באיש מלא רע ותהפכות /
משלח מדנים בין אחים ומכות / להפיל ידידיו
ורעיו / בפחים ומכמרות מדהרת אביריו / ונמצאת
לעשות לו כאשר זמם / הוא לאנשים עד תמם.
