


	
	משל ב: עכבר עם צפרדע ונשר
	




בור כרה ויחפרהו / וישלם לו כפרי מעללהו



עכבר ישב על מפתן בית הרחים /
וימלא קמח שפתים ומלקוחים / והנה השמש זרחה
עליו / ובכפות ידיו ורגליו / ישתעשע וממשש
בשפמו / כמשפט אביו ובני עמו / ובהיותו שם
בלי פחד ואימה / צפרדע עוברת לפי תמה / ותעמד
לחקר כל ענינו / ואם אדון בביתו הנו / ויען לה
העכבר / לי הבית ולי הבר / ולי הזרע במגורה /
לאכלה ולא לסחורה / והנה ביתי סתרי / ואמלא טרף
חורי / ובנות הרחים והשכב / סביבותם אני שוכב /
ויספיקו לי רב מכלת / שניהם מלאים סלת / כה
משפטי כל הימים / וכשמעי המון עמים / ואתחבא
בנקרותי / ומשם תוצאותי / בראותי כי אין גבר /
ואלך לשבר שבר / ותרא הצפרדע לבן על כל
אבריו / ועל ידיו ועל חלקת צואריו / ותאמר לו:
מדוע לבן ללבושך / האם שיבה זרקה על ראשך /
ויאמר: עם לבן גרתי / ומסלת חטים טעמתי /
והקמח עלי ירד / מפזר ומפרד / כי לא נסגרו
בפני דלתות / הקמחים והסלתות / ותאמר לו
הצפרדע במרמה / כל חייך פחד ואימה / מה
תתהלל בסלת ומגד / וחייך תלואים לך מנגד
מכל הבא אל האהל / תגע עדיך ותבהל / ומבני
איש שם מקהלות / ורב מהיות קולות / וכל בך עצר
וחבש / ומאכלך קשה ויבש / ובגדיך תמיד
מעפרים / אך בגדי נקיים וננערים / ובמשכנותי
אין ארב / ומאכלי דשן וערב / סביבותיך פחים /
ומשכנותי מבטחים / שמע אמרי ומלי / וכל מחסרך
עלי / בא אתי במרוצה / בסתר קנה ובצה /
ותדיחנו בחלק שפתיה / ותלך לפניו משכת
ירכותיה / והעכבר לא נזהר / עד בואם אל שפת
הנהר / ויאמר לצפרדע: ואנה / ביתך אשר את
בו חנה / ותען לו: מעבר לנהר הזה / ואתה מכל
אשר באהלי תחזה / הטוב והישר בעיניך / ושים
אל הנהר פניך / לעבר מי מנוחות / כאשר יפרש
השחה לשחות / ועין העכבר על נפשו חסה / ויואל
ללכת כי לא נסה / ויאמר לה: משכנותיך אשר
אמרת אים / מי יעבר לנו אל עבר הים / לעבר בים
לא נסיתי / טוב לי לשבת בביתי / ולהיות שם כל ימי
חלדי / כי ירט הדרך לנגדי / ותאמר לו: אדני
שמעני / כי תעבר במים אתך אני / ונהרות לא
ישטפוך / ותרחיק שונאיך אשר רדפוך / את החוט
הזה ברגלי וברגליך תקשר / ויעמדו רגליך
במישור / ואלך עמך לנחותך הדרך / בצמיד פתיל
אשר תקשר לי בירך / ובמעשה ובשמועה / לא
תאנה אליך רעה / ואתה בין אחים תפריא / עתה
תראה היקרך דברי / ויאמר: אם כדברך ברית לך
אכרת / גלי שוק עברי נהרות / הטתו ברב לקחה /
ורוח חבורה עליהם נחה / וילכו צמדים הצפרדע
והעכבר / בצמיד פתיל אשר ברגליהם חבר /
ובהגיעם לעמקי תהום ומצולה / ותחשב הצפרדע
להטביעו ולהורידו שאולה / ובהתחזק העכבר על
פני המים / דרך הנשר בשמים / וירא אותם זה
בזה מגלל / ויעט אל השלל / אכזרי הוא ולא ירחם /
יפרש כנפיו יקחם / ובפתיל חזק אשר יחדו נצמדו
/ יתלכדו לא יתפרדו / ויהי הנשר חש לאכול /
וירא כי אל העכבר לא יכול / כי צפד עור חזק על
גבו / ושער שחור צמח בו / וישת אל הצפרדע
פניו / נחמדת למראה וטובה למאכל בעיניו /
ויבלענה בפעם אחת ולא שנה / ויפן העכבר הנה
אנה ואנה / וירא כי מוסרותיו נתקו / לנוס מהר
רגליו התחזקו / וימלט רק הוא לבדו / והצפרדע
אשר נקשרה בצדו / נפלה בפח אשר יקשה / והושב
גמולה בראשה.



והמשל אראה באיש / אשר יחפיר
ויבאיש / לחרש על רעהו רעה / לעשות חנף לדבר
תועה / וכל מזמותיו לענשו / גמולו ישוב בראשו /
כאשר זמם לעשות לו / וימלט נקי שלו באהלו /
ואמר שלמה בחכמתו / מרדף רעה למותו.



ואשא משלי ואמר:



פתיל מרמה אשר חבר / צפרדע והעכבר

הלא נתק ושב שלם / אשר מלא אסמיו בר

ולא נאכל ולא נקבר / ולא נשבה ולא נשבר

ומתה המתיקתהו / ולא שבה מקום מדבר

וכן יהיה גמול מסית / בפה חלק יהי נדבר

בלבו מרמות ותך / ולא יועיל לסוד עבר

וחרש רע לרעו מו/קשו ימצא ויתעבר

תמימים ישכנו ארץ / ודרשי טוב ונושקי בר.
