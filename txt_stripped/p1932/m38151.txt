


	
	הייתי וזמן הנעורים
	








בני מקרא במצרים





– – – הייתי וזמן הנעורים מכלי החברה



אל כלי הנדוד לא הורק / וימי העלמות לשמי הטוב שמן תורק / ועתות העמל רחוקים ממני בל תקרב / כרחוק מזרח ממערב / ושד יין רקחי השמחות מוצה ויונק / ועסיס רמוני המנוחה להרי הבשמים סוחר ומזנק / וכל תענוגי העולם מנפשי הדשנה / לא ילכו אנה ואנה / ואני על ברכי הילדות והשחרות משתעשע ואמון / ושוכב בחיק שלות הזמן במדינת נא אמון / המדינה המלאה ביקר החן כרמון
/ ושמע יפיה נשמע בקצוי העולם כפעמון / העיר המלאה המאשרה בכל חן בבנין הטוב מזמן קדמון / וקרן כל המדינות לפניה כפוף כאגמון / בעלת המגדל והארמון / כגבה הר בשן וחרמון / והדירות הרחבות / וקורותם ארזים בזהב ובששר
משוחות / ובכל מיני אבנים מפתחות / ואבני בנינה כאלו בזהר האור טוחות / ויסודותיה החמודות כאלו הם על אדני שש מיסדות / ומכל חצרותיהם לא נעדר / בהט ושש ודר / והגנות והפרדסים / כמים
לים מכסים / ובדרכיה הגשם יום יום יגבר / זרם מים עבר / ובעת תבקשנו / ולא תמצאנו / ויעבר והנה איננו / כי תעלות יסודותיה בהדלות בעיר / ויוציאוהו ויניחוהו מחוץ לעיר / וכל צמחיה מצאו
חן בעיני / כאהלים נטע ה׳ / עד היה שמה בפי שוכניה ויושביה / יקרא חפצי בה / וסוחרי כל העולם בה בחרו וחשקו כי שפע ימים יינקו / וכמה אניות סוחר בלילה ויומה / ממרחק תביא לחמה / ובכל עת יבואוה ספינות מארם ופלשתים / וצים מיד כתים / כי המלך הגדול אלכסנדר
בחזקתו ירה אבני יסודותיה / ובעצמתו הציב דלתיה / חומתה גבהה ובצורה / לעתות בצרה / ולה החן והתפארה / על צור המעטירה / כי לה ובעדה אמר האל בחזיונו: / ׳התיטבי מנוא׳ / וכמה לשמרה בנו פסגה וצריח / חומה גבוהה דלתים
ובריח / ובכל לילה להיותה לשוכניה שומרת / סגרת ומסגרת / וזה השיר עשיתיו לעד ולאות על גדלה / מספר אחד אחוז מחמש המאות ממהללה.







לנוא אמון פאר הוד ועטרה / עלי כל עיר יפפיה יקרה

ולה כל המדינות כשפחות / בטוב מושב והיא להן גבירה

ומי ראה בכל עולם כמותה / בחומתה אשר בה היא בצורה

בבנינה יקר לקחה עלי כל / מדינות חן וצור המעטירה.







ובעודי בה בין חברי ורעי / ודודי ומידעי / בגילי ושעשועי / תמיד לרגעי / לקחה
אזני שמץ דבר / מאיש היה לי נחבר / כי במצרים העיר הגדולה / גוי טהור ועם סגלה / דגל יקר מעלתו רם ונעלה / על כל קהלות הגולה / חכם ונבון בכל דברי נבואה וחזיון / ויודע דעת עליון / ונפשו בלבד חומדה ומתאוה / לשמר התורה והמצוה / ושמו נודע בין העמים ונקרא: / עם בני מקרא / המאמין בה׳ כי הוא אחד ואין עוד מלבדו / ובמשה עבדו / ובשאר הנביאים הנבחרים / הנזכרים בארבעה ועשרים / ודבר ה׳ אשר הוריד בתורתו / לא יחליפנו ולא ימיר אותו / היא התורה
אשר על יד משה נתונה / ואין זולתה לא תלמוד ולא משנה / ובדעתו נכונה / כי כל מצוה זולת מצוות התורה שבכתב החמודה / מצות אנשים מלמדה.





וכשמעי זה הדבר ממקומי קמתי / ולבי לנסיעה שמתי / ולשור אשר שמעתי / חשתי ולא התמהמהתי / כי אמרתי לנפשי אולי בלכתי אחזה / את המראה הגדול הזה / והעם אשר בחר בכשרון ותמימות / אלכה ואראנו בטרם אמות / ונסעתי ועזבתי את ארצי / ורכבתי על צי / להגיע לחפצי / במשוש ועלצי / ואמצאה עיר על
נהר גן עדן מושבה / וכל חן ויפי ויקר נמצא בה / ויהי כאשר שמעו עליה אזני / בנטעיה ומגדיה כגן ה׳ / ואנשיה ככוכבי השמים / והעיר רחבת ידים / ובתיה ערבים / גבוהים ורחבים / ובה שירי
עגבים / לבב שומעיהם משיבים / וגנות טובים / נפשות משיבים / על הים קצובים / ובורות חצובים / ובשורי גדלה ורוב ההמון אשר בה מתחבר / נפעמתי ולא אדבר / ודברתי ללבי הנפעם: / זאת העיר רבתי עם / על זאת יאות תאמרנה בעדה כל הלשונות / רבתי בגוים שרתי במדינות / ואפתחה פי בטוב דבור וזמר / ואשא משלי ואמר:







למצרים עלי כל עיר גדלה

בהוד מלכה אשר בה רם ונעלה

וחילה כעפר ארץ במספר

ורוב עמה ככוכבים במעלה

ואם היא אז לקדר בית מלוכה

לפרעה בה יקר היתה תחלה

בכל עת בה כבוד מלכות יחדש

אשר נשא ורם על כל תהלה.







ואחר השתכלתי לעצם העיר / רוחי
ולבי האל העיר / לחקר את הדבר אשר בעירי שמעתי / ולעמד על סוד הענין אשר בעבורו מנוי נסעתי / ואשאל בעד גבול העם הנזכר / אשר ריח שמניו שלח לי הזמן לאשכר / ויורוני אנשים עליו / ויגיעוני אליו / ואמצא קהל בפניו שבע
שמחות / יגון מלבב כל איש דוחות / ובתים פתוחים לגר ולתושב / והוא לבדד ישכן ובגוים לא יתחשב / ואישון עין דרישתו לכתוב בפוך האמת נקרע / תם וישר וסר מרע / ובמלחמת התורה אשר לב
ונפש משיבה / אחד ירדף אלף ושנים יניסו רבבה / ומרב תקף מלוליו / ייראו מגשת אליו / ובקרב מלאכתה העצומה / כלם אחוזי חרב מלמדי מלחמה / ומפחדם לתעות בדרך העיון יתהלכו / איש חרבו
על ירכו / רוממות אל בגרונם / ולהשיב דבר לשונאיהם / בדברי תורה שננו לשונם.





ובראותי זה העם הנכבד ידעתי / כי לעץ חיים נגעתי / ובלבי אני אמרתי / כי לא לריק יגעתי / ועתה ידעתי / כי כל אשר שמעתי / על העם הזה בארצי / והנה לא הגד לי החצי / ואקד ואשתחוה על פני / ואברך את ה׳ / אשר מתוך נוי ומעוני / בחסדו ובטובו הנחני / למלאות חפצי ורצוני / בעם אשר אין לו בזמן שני / וישאלוני העם ויאמרו לי: ׳ה׳ אלהיך ירצך / מה מלאכתך ומאין תבא ומה ארצך / ואני נכלמתי מהם כי טעם הגרות בפי מר. ואשא משלי ואמר:







בנוא אמון ילדוני ושרשי / בארץ מערב מעיר ספרד

והשיר לי מלאכת מנעורי / ולבי לו מאד יהמה ויחרד

שקלתיו מבלי חסרון ויתרון / ומלאתיו מחבר גם מפרד

ורצה לעבד אותי לבדי / וגם פשע בזולתי ומרד

ולא שקט עדי עלה בסלם / חמודותיו ודבר בי וירד.







וכשמע העם ערך שירי אשר ערכתי / ידעו ארצי ומלאכתי / וגדלה בעינם מעלתי / והסכימו על אהבתי. – – –
