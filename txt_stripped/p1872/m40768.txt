


	
	וכוח רכב ובענה
	




Immanuel Frances


]נולד בליוורנו בשנת 1618, חי איזה זמן בפֿירנצי ובאלג֗זיר, מת בליוורנו אחרי שנת 1710]






רכב: סורה אלי בענה אחי.



בענה: הרף ממני כי היום מרי שחי / כי לא נתנני הזמן השב רוחי.



רכב: איך נבהלו רעיוניך / ולמה נפלו פניך / ותחרק שניך/ ומה־ירמזון עיניך.



בענה: הנני נשבע בחיי ראשי / כי כמעט אבחר מחנק נפשי.



רכב: גרסה נפשי לתאבה / לדעת הסבה / הגידה־לי המחלה / אולי אמצא לה תעלה.



בענה: תשאות חן חן לך רכב רעי / אלופי
ומידעי / אמנם דע כי אנושה מכתי / ואין מרפא
למחלתי / כי איככה אוכל / ולבי אש קנאה לא תאכל /
בראותי ישב בסתר היכל / כל־טפש וכל־סכל / אשר
בשם רבנים יקראו / ומפניהם זקנים עמדו ונערים
נחבאו / ואני הדל והמך / בהמות הבית הייתי
עמך / לא יביטו ולא יראו בי / ולא הגעתי להקרא
רבי / על־זה אשחה בכל־לילה מטתי / ואמסה ערשי
בדמעתי / ואקרא למות להפיק מאויי / כי טוב מותי
מחיי.



רכב: צחוק עשית לי באמת ואמונה / ועתה
שמע בקולי אאלפך בינה / ולמרבה לתקופת השנה
/ תקרא הרב כמהר"ר בענה.



בענה: רכב מרים ראשי וכבודי / אם החסד
הזה תעשה עמדי / אהיה לך מאח דבק / ובעפר
רגליך אתאבק.



רכב: הסכת־נא אלי / ועשה אזנך
כאפרכסת למלי / שים הסרבל על־הכתפים /
שיהיה ארך עד הרגלים / ובתי עינים קשורים
לאזנים / וכובע גדול הכנפים / ואבנט חגור
על־המתנים / ובתי ידים / ובשבתך בבית הכנסת
בשפלות עינים / תניע השפתים / ובברכת “בונה
ירושלים” / תשא עיניך לשמים / ותוריד דמעות
כמים / ובברכת הודאה תקד עד הקרסלים /
וב“עושה שלום” תשתחוה שבעתים / ואם תוכל גדל
זקנך שיגיע לברכים / כי מי יראה איש זקנו
מגיע לטבור / גם כי יהיה רק ובור / ולא יאמר
שהוא ראוי להיות רב ממנה על־הצבור.



בענה: מה־נמלצו אמרותיך לחכי / ואולם
רכב מלכי / הישר לפני דרכי / כי אתה תגיה
חשכי.



רכב: גם בדרך אחרת תשא על־חבריך דגל / והוא בדרש נכון למועדי רגל.
בענה: סכלותי אודיעך ואולתי לא כסיתי / כי מעולם בדרש לא נסיתי.



רכב: יש דרש טוב לעמו / ומי יודע אם הוא
קמח שטחנו בעצמו / ולא תשמע לקול מלחשים /
שיאמרו כי מה הם דורשים / בפשטים נאים ופרושים
/ מלבם הם מחדשים / כי גברו ולא לאמונה / ובפיהם
אין נכונה / כי רבו כמו רבו ספרי הדרשות / ישנות
גם חדשות / והדרשנים יעשו מהם / דרושים
לכל־חפציהם / גם יש דרשן בזמננו / שראינו
בעינינו / ושמענו באזנינו / אשר היה בעיני העם
לאות / ויספרו מדרשותיו נפלאות / ולא שמעתי
מפיו כי אם גמטריאות למאות / ונוטריקון וראשי
תיבות / לאלפים ולרבבות / ועל־כמה מאמרים ופסוקים
/ שקשים להלמם וכראי מוצק חזקים / יאמר פרושים
דחוקים / ומהאמת רחוקים / כל־העם מקצה ועד־קצה /
כל־דבור אשר מפיו יוצא / בצמא שותה ומוצה /
וקרא זה אל־זה ואמר / הפרוש הזה כמין חמר /
וכל־הדרוש בעיני כנתינתו מסיני.



בענה: בא אל־הפרטים כה יושיעך היוצר /
ודבר־מה לך לא יבצר / ובמקום שאמרו להאריך
אינך רשאי לקצר / ודבר ממני לא תכסה / ואדעה
מה אעשה / בעלותי על־הדוכן או בשבתי
על־הכסא.



רכב: תתעטף בציצת בעלותך על־הדוכן /
או בשבתך על־הכסא לדורשים מוכן / וקדם תתחיל
הדרשה תירק / והחטם מן הזהמא יורק / וקנח פיך
כאלו שתית מרק / וזקנך באצבעותיך תסרק. /
אחר זה דרש דרושך בלי בושה / בעזות פנים
ובמצח נחושה / ובסוף דבריך שים פניך נגד העם /
והגד להם חטאתם ופשעם / באמר “איכה זהב המוסר
יועם / בטובו לאנשים יטעם ולמוכיחים ינעם / רבו
חטאת הדור מחטאת ירבעם / ואולם אי לאו דמסתפינא
שיקראוני אכזר מבלעם / אזעם מה־שהוא לא זעם /
אראה הדור והוא פרוץ / וכל־אחד ואחד לרע ירוץ /
זריז וחרוץ / לאסף פז וחרוץ / ולכנס החכמה
מתרשלים ומתעצלים / כאלו הם נכשלים ונחשלים
/ ואין מכלים / וגם על־פרנסי הקהל אשפך שיח /
וכלפי שרי העם דברים אטיח / ולו יש־בינינו
מוכיח / יריע אף־יצריח / עד־מתי עמי תועה / כצאן
אשר אין־להם רעה. / קדשו צום קראו עצרה / ושובו
לאל במהרה / פן־ישלח בכם את־המארה / עברה
וזעם וצרה / ואל־נא תקשו ערפכם / והסירו את־אלהי
הנכר אשר בתככם / הוא יצרכם הרע הצורר אתכם /
ודרשו יי בכל־לבבכם / אולי יחננכם וישלח גואל
ויקבץ נדחכם / כאשר דבר לכם”. / ותאמר הכל
ברמה / ותספק על־הקרשים בחמה / ותהי לך לחכמה
/ ואם אינך יודע עד־מה / וכל־זה תעשה בנקלה /
ותהי לך לתהלה / וכל־העם מזקן ועד־נער / יאמרו
“אלו לא היה אדם גדול, לא היה מוכיח בשער.”

בענה: אין לי עסק בנסתרות אפילו
כגרגיר חרדל / ואראה כל־דרשן מדבר בם לא
יחדל / ואם לא אעשה כן אני הדל / הקטון שבהם
ממני יגדל.



רכב: דע כי רבים מהם בהמה המה להם /
וגם אתה עשה כמעשיהם / ודבר בם כאשר בלבבך
/ ואמר כי נהירין לך שבילי הסודות כשבילי עיר
מושבך / ואם לא אפו מי יכזיבך / וקרא בגרון
“שמעו עמים כלם / כי יש־לי סוד נסתר ונעלם /
אשר שמעתי במדינת עילם / מפי המקבל הגדול ר'
חירה מעיר עדלם / שחבר ספר “חמור עולה בסלם” /
מי־הקשה אליו וישלם / ומפני שהוא מכבשונו
של־עולם / הלא הוא כמס עמדי / ויהיה לי לבדי / עד
אמצא איזה יחידי/ ראוי לבא בסודי”.

בענה: ואם יבוא לפני דבר אסור והתר.

רכב: אם תוכל מהם הסתר / ואם לא תוכל
להתעלם החרש והתאפק / ואמר כי אתה מספק /
ודעתך בדבר נבוכה / והמחמיר תבוא עליו
ברכה.



בענה: ובלכתי אל־הישיבה / איך אתנהג
בה / כי לא אדע הויות אביי ורבא / ואני אנה
אני־בא.



רכב: יש למקום הרבה דרכים / ואני
אתן־לך בהם מהלכים / אמר כי שלחו לך שאלות
מתם / ואתה עסוק בתשובותם / או אמר כי מרב
עסקיך לא נתת לעיניך שנה / ושמעתא בעיא
צלותא כיומא דאסתנא.



בענה: בך יברך ישראל / כי היית לי לאב וגואל.



רכב: גם זאת עשה אם בישיבה תשב / דעת
בעלי תריסין הקשב / ודבר־מה קדם אל־תשב /
וסכלותך לענוה תחשב / וזאת הדרך אשר תבר
לעולם / נטה אחר סברת הגדול שבכלם / ולא תשוב
דך נכלם / ואמר “יחדלון אחים ורעים / כי דברי
חכמים בנחת נשמעים / ובאמת כי דברי הרב פלוני
בזאת המימרא / סברא נכונה וישרה / וממנה אין
להמיש”.

בענה: ואם ישימו החולקים פניהם
כחלמיש / ויאמרו לי באף ובזעם / “קולך כקול
הרעם / אמר־לנו טעם / ולא תלך עמנו כפעם
בפעם”.



רכב: ענה להם כבר אמרתי די והותר / ולא
באתי כאן לקנטר / כי כל־ימי בין חכמים עמדתי /
ובדלתות מדרשיהם שקדתי / ולעמקה של־הלכה
ירדתי / ולא עמדתי על־הנצוח / בדברי הבל
ורעיון רוח / וזכורני כי מנעורי שלש פעמים
הודיתי לדברי חברי / וגם המה פעמים רבות הודו
לדברי / ולמען כבודי לא יתבזה / לא אוסיף עוד
דבר בדבר הזה / ולכן בענה לך ואכל לחמך
בשמחה / ונסו יגון ואנחה / ומבטח אני כי למועד
הזה כעת חיה / תקרא רב בסיעתא דשמיא.



בענה: מי־יתן והיה / ואולם בהתהלכי
עצתך תנחני / והקיצותי והיא תשיחני / ואמצא חן
בעיניך כי נחמתני / ועד־הלם הביאתני / ועתה
שמעה שירתי / והקשיבה רנתי:







– – ͝ – –

רוחי גבוהה / תמיד צמאה / אל־הסמיכה /

סמוך אשישות

ויהי לבבי / חלל בקרבי / אף־מחשבותי /

כל־עת אנושות

לאמר ואיכה / אסמך סמיכה / בקרב הלכה /

ידי חלושות

אמר לשכלי / מי יעמד לי / לפרות בתורת /

האל ולפשות

דוחק ומצר / לבי וקצר / היום והנו /

בין השמשות

גם כי לשחרות / אשוב ובחרות / אוי לי ודעות /

שכלי קלושות

אמר בעצמי / מי יעמד מי / יום החקירות /

יום הדרישות

יום בי ידבר / לשמי יחבר / כנוי חמור בר /

אוכל חששות

נודע לנפשי / ריק שבראשי / גם זה ימלא /

לבי חששות

כמה הכנות / צריך ובינות / כמה פרישות /

כמה קדשות

ללבש תחלה / מעטה תהלה / לתת ברגלי /

יצרי נחושות

כי כל־שנתי / תדד שנתי / לשכב בספרי /

אל־המראשות

שמתי למכרח / אותי ומצרך / לידע לפלפל /

תרץ והקשות

על־זה בצערי / אפרע שערי / ופאות זקני /

מחיל תלושות

אבכה ולחיי / נשטף בבכיי / היו לכעסי /

עיני עששות

הה לב מעקל / מאמין בנקל / הפך זממך /

ידי ממישות

יושם לרבן / ראוי לדרבן / אישים ראוים /

אל־מחרשות

אספו בחפנם / רוח וגרנם / תבן ואין כל־ /

בר במדושות

צאנה זמירות / פי בחצוצרות / צאנה מהירות /

לרוץ וחשות

צאנה בדיצה / שפתי מליצה / משיר ורנה /

ושחוק גדושות

מי יאמין כי / יספיק לצרכי / ארבע הגדות /

ושתי דרשות

פלפול גמרא / שקול סברא / נחשב להבל /

ולמעקשות

חכמות עמקות / מפה מחוקות / בחוץ נטושות /

בטיט רטושות

נפשי ערמה / מבין וחכמה / יש יש כמותך /

הדר לבושות

יש יש מנויות / בכלל ראויות / לפאר וממך /

רעות וטפשות

נפש מקבל / לא עוד תחבל / לא יחטאו /

על־הנפשות

כלם בריאים / גסים ונאים / מראות פניהם /

אינם כבושות

כל־מחשבות רב / הנם להון רב / רגליו ברשת /

חמדה יקושות

נדמו לבלעם / כל־הון בבלעם / ושמם כמשה /

למשך ולמשות

כל־בור וסכל / ישר ויוכל/ לא עת לשוני /

לשתק ולחשות

הנני הגם כי / שפל בערכי / אדם ולא קוף /

למאס ולנשות

מתי אברר / להיות כמהר"ר / הוד רב כשעור /

שלש עדשות

אמרות תחנה / נא תֽהינה / היום בלבות /

מורים חרושות

עיני גאונים / אזני נבונים / היו עדנה /

עורות וחרשות

אמנם באהיה / אבטח יחיה / עצמות כבודי /

אף כי יבשות

אוחיל לחסדו / כי הוא לבדו / פועל גבורות /

עושה חדשות.







רכב: יששום מדבר וציה למדברך /
ותגל ערבה לערבות זמירך / ועתה לך לשלום



לאהלך.



בענה: יאר ה' פניו אליך / ויהי שלום בחילך.
