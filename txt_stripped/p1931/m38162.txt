


	
	ויקם החושק השלישי : מקאמה
	







החושק שהתחפש לנזיר. תוכן המעשה: מעשה בבת מלך יפיפיה, שאביה כולא אותה בארמון כדי למנוע את חושקיה להגיע אליה. חמישה בני מלכים מנסים בתחבולות שונות להיכנס לארמון. הראשון הופיע בדמות מנגן, השני כרוכל בשמים, אך שניהם נכשלו. השלישי התחפש כנזיר, ולהלן הסיפור עליו:






ויקם החושק השלישי וילבש בגדיו הנחמדים / ועל גרונו רבידים / וילך תחלה כבן מלך רוכב על סוס לבן והוא עומד בין ההדסים / עם כתנת הפסים / ותפארת העכסים / ויתיצב מרחוק לראות את יפי הבנין והעיר וחומותיה / בנויה לתלפיה / באבני שישה / כוחלא ומרמרא / וירא
והנה גמדים במגדלותיה / ויסתכל באנשי
המגדל והנה הם מעט מזער לא כביר / ויחשב החושק היותו על כל אדון וגביר / על ההיכל והדביר / וישב לבית זבולו / ולמשכן אהלו / ויערים תחבולות ומועצות / עד כלה ונחרצות / להיות באחרית הזעם / נותן טעם בר נותן טעם / ויאמר:
׳עתה אפשיט בגדי החשק ואלבש בגדי נזירים ארוכים וישנים / וכלים מכלים שונים / ואכריז ספרי נזירות / ובזה אצא לחרות / כי אהיה מוחזק בכשרות / וישא משלו:







לבי ועיני קהל נאמני / אספו עדתם וגמרו ונמנו

סותם להפשיט והדר ותכשיט / ולבוש הדרם בנכרי ישנו

לעבר בערמה בספרי נזירים / ובשוק צביה בקול עז יתנו

אולי בחכמה בדעת מזמה / יעלת תהלה יפתו ויונו

ישבו במרה ויחנו במתקה / ולפי כבודם חדרים ימנו.







ויתלבש החושק בגדי נזיר ארוכים / שמורים וערוכים / ויתחכם לדעת ידיעת ההפכים / ויגדל שערו ושער הזקן / ויקח ספרו כדי שיסתכלו מה שיש בקנקן / ולא ימצאוהו מכל וכל ריקן / וישם הצניף הטהור על ראשו / ויתן מטפחת סביב גרונו להעלים כחשו / ויתחיל ללכת ונעריו עמו / והוא נושא ספר הנזירות על שכמו /
בשנותו את טעמו / וילך תחלה בעיר וגבולותיה / ויסב כל חומותיה / פעם ופעמים / וילך כל היום בשוה קריתים / כדרך יום כה וכדרך יום כה וכאמתים / להודיע לכל רואיו חכמת נזירותו / ורב
פרישותו / ולא פריצותו / עד אשר הוחזק לנזיר / בהיותו מראה טלפיו כחזיר / וכל זה עשה כל הימים אולי יוכל אבדת חבריו להחזיר.





ויצא שמו בכל גבולות העיר / כגדול כצעיר / וישמע שמעו בבית הגבירה / אשר עם לבונת הנזירות קטורה / ויגידו לה את כבודו ועזוזו ונפלאותיו / ושם ספרו המעורר לב ישנים בדברי גבורותיו / ותצו לקראו לראות ספרי חמדתו / ואת יקר תפארת גדלתו / ויבא לביתה ובדרך האריך שולי בגדיו / ויסחב על הארץ מדיו / ויסרק במסרק זקנו להאריך השער / וילך משער לשער / ומצער לסער / ויהי בהגיעו עד הסלם ובהיכל נטפל / מאריכות בגדיו בין רגליו כרע נפל / ויתחיל לעלות במדרגות הסלם / ויצא משם לקול דברים בלי נשמע קולם / כי אם לנזיר המזיף במעשה תעתועיו / והבלי שעשועיו / והקול שמע באזנו / בעלותו במדרגות גאונו / וישמע קול רעם גדול ממקום המשתמר / וכזה אומר:







מי זה אשר עלה ברום דרגי / ידו בעץ סלם בעז תמכה

לבוא בגן עדני ללקט אור / מרור ועולשין וגם תמכה

כי בא דמות שועל בסות לביא / שחק לבבו בו והוא בכה

קרב להתעלף בשושני / ויעבר עמק וגיא בכא

ובכן להביט זיו יפפיה / נפשו בשמן גיל ושיר סוכה

מה לו לטמא משכנות היכל / יעלה ברוח חן הלא סוכה

לכן אשבר מדרגותי כי / לא יעלה פסל ומסכה

אלבש מעיל קנאה להפיל איש / כסוי בבגדי תם ומסכה

בל יעלה אל בית חצר מלכות / רצפת הדר כל גוי וממלכה

ככבי נבוכה לו נבוכים כי / לא יעזרו גלגל ומערכה.







וישמע הנזיר את הדבר הרע הזה ויתאבל / וישתומם כשעה אחת ויעמד מבלבל /
כי נסתפק אם השפחות הפחידוהו לבלתי יעלה / או אם הקול היה ממזל טלה / ויעמד בסלם תלוי ועומד / ולא עולה ולא יורד / וכמעט חום שמשא אריש דיקלי / ויהי כאשר ראה שמשו שוקע היה צוח או
גנוחי או ילולי / עד אשר באת שפחת הגבירה ותאמר לו: ׳עלה נא הנזיר הגדול / ואל תחדל חדול / בוא מבחר הפרושים / וזרע הקדושים׳ / וכשמע את קול השפחה ידע כי הקול אשר שמע בתחלה / לא היה מבית גברת התהלה / ועם כל זה נתבהל כאיש אשר ישן / וארכבתיה דא לדא נקשן / ויעל בסלם ויתנועע / ובגדו הארוך נאחז במסמר המדרגה ויקרע / ויתבהל מאד מהסימן הרע הזה / ומשמן בשרו ירזה / ויכנס בנס דגלו וספרו בפתח היציע / והוא לפני כל רואיו שק ואפר יציע / ויעלה עוד עד ההיכל העליון / וטרם הכנסו רחץ פניו בנקיון / וירא שם מראות אלהים בכח הדמיון / ויבא עד חדר הצביה / והוא בנוי לתלפיה / וישתחו ויקד ולנגדה כרע / ולבבו כלו חנף ומרע / ותרא הצביה את מראהו ואת גבוה קומתו / והצניף אשר בראשו והדור תבניתו / ותאמר: ׳ודאי זה הוא אשר המשרה על שכמו / והביא ספר הנזירות עמו׳ / ותמהר להקימו מהארץ והרצפה / ותושיבהו בכסא האולם והכפה
/ ותאמר לו: ׳האתה האדם הגדול בענקים / אשר שמעתי שמעך מפי עוללים ויונקים?׳ / וימהר המזיף להשיב פתגם / בנותן טעם לפגם / ויאמר לה: ׳אני עבד הנזירים ואתאבק בעפר רגליהם / ואשתה בצמא את דבריהם / ואם גבירתי על ראשם
שמתני / הן מדרך לכף רגליהם אני׳. – – –
