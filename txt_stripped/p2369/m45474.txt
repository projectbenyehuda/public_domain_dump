


	
	יחזקאל ב״ר סלימאן ב״ר עבדאללה חנין
	



קרובו (דודו, אחי אביו?) של עבדאללה חנין הנ“ל, היגר להודו ויסד דפוס ככלכתה, The Jewish Press, ונדפסו בו
בשנים תרל”א-תרנ“ג (1893–1871) עשרה ספרים ושבועון בשם “מבשר”, שהוא עצמו
ערך והוציא לאור. בשבועון זה פירסם שירים משלו. בשנת תר”מ הדפיס שיר
בשבועון “פרח”, שיצא בכלכתה אחרי שה“מבשר” חדל לצאת. שיר אחד על צרות הגלות
וערגה לציון, והיתר שירי אישים, בני משפחות שיצאו מבגדאד והתיישבו בהודו
והיתה להם השפעה על הקהילה.





סימן: יחזקאל ח׳זק. לחן: אליך קוני.






ידידי חלקי / חשקי / למה שמתני

למכים גוי / כוי / ועזבתני

ביד צר מוזר / אכזר / וישטמני

צא לעזרתי, עז ישועתי, נא לקדמותי, אותי, השב על כני.





חשקך לי טמון / אמון / מאז עזבתיך

על יד ימיני / אני / כחותם שמתיך

שובי רעיתי / אתי / חסד רחמתיך

אבנה מקדשי, ואשלח לך שי, מגזע ישי, ישי, קומי ורני.





זכר את עניי / דיי / מי ראש, לענה

עד אן בגלות / דלות / אהמה כיונה

שעה לשועי / ישעי / עת לחננה

צורי משגבי, שמח לבבי, וגואל תביא, תשבי, יבשרני.





קומי נא עורי / שירי / וזמרה פצחי

נפשי נכספה / יפה / אשלח משיחי

יפדך משביה / רעיה / שישי ושמחי

בגדי שש רקמה, תלבשי רמה, ברה כחמה, תמה, זיוך הראיני.





אליך צורי / עזרי / תמיד איחל

ירומם דגלי / אלי / ישועות וחל

שוכן רום חביון / ציון / אותי הנחל

מקום מקדשי, אותה נפשי, אקריב לך אשי, אישי, תודת קרבני.





חזקי ואמצי / עלצי / חזק גואלך

את אויבך יגמר / יזמר / סחב והשלך

ופצחי גרון / ברן / צהלי קולך

זמרה הללי, ציון תעלי, משכן זבולי, גילי, בשיר רנני.
