


	
	להכנסת שבת.
	




שבת, שבת, / מאין באת?

שלום לך, בת, / מה־טוב מראך!

מראך נאוה, / את גן רוה:

כל־הדוה / יביט וחי.

וחי גם־אתה תהיה מעתה;

מדוע חתה? / עורה אפוא.



אפוא קום גילה; / רק־זאת אואילה:



הנה אובילה / מרגוע לך.

לך אשמע־נא, / מכלל עדנה;

ממך אמנה / אדרוש דבר.

דבר אדרוש / קדם וראש:

מה ראשיתך? / מתי? ואן?

ואן נסכתי, / ואן ברכתי,

ואן מלכתי, / הקשב דוד צח.

דוד צח עשני, / וימליכני;

ויקראני / ענג לכל.

לכל? ואיך / יחל הודך?

אך מי דודך? / אך מי ומי?

ומי יהיה / אתך יחיה?

איה, איה / חלמיש צור?





צור רם, רב־דע, / בי ישתעשע;



ויהי לי רע, / ואני אליו.



אליו נקניתי, / אליו נשאתי;



אולם נשביתי / מגבר טוב.





טוב הדבר: / משה גבר;



האיש גבר / זכה וייטב.

וייטב לי, / אף לקהלי;

ומשוש גילי / היה היה.





היה איש יושר, / הדריך בכשר;



מאס העשר, / תורה קבל.



קבל רב־זבד / אז בן־יוכבד,



ויכנה עבד / אל־אל טהור.



טהור היום, / לנו פדיום;



גם סבטיום / עדות נתן.



נתן אלינו / יום שבתנו;



פצחו, רננו / על המתן.
