


	
	ליום שמחת תורה.
	








בשגיון / תגל ציון,

לאל עליון / תשא זמרה.

ועם־חמדה / יתן תודה,

וביהודה / תופע אורה.

כדור ראשון / ישבע ששון,

וכל־לשון / תשיר שירה.

ושיר מזמור / נאמר אמור

ביום נגמור / את־התורה.





וגיל חתן / יהיה איתן,

ויתעדן / בקרוא מקרא.

הלא יירש / גיל לא פרש

אשר ארש / יפה ברה.

וברכתו / תשכן אתו,

ומנתו / מלין יקרא.

תנו שכר / ששון ויקר

לאיש יקר / השלים תורה.





ואתה ראש, / מגיד מראש,



אשר תפרש / עולם נברא,

לך ולך / ששון ושמחה,

בהחלך / דברי נורא.

ואל יחיד / צריו יכחיד,

ובן־דוד / ירבה משרה.



וישמע־אל / אל־ישראל,



ובא גאל / בגלל תורה.
