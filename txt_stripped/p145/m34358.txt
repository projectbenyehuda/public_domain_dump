


	
	ליום הפורים
	








שאו שירים / ביום פורים,



וברבורים / לאיש אביון.

שאו רנה / בכל־פנה,

שתו, שכרו, / במות המן.



שאו זמרה / לאל נורא;



קחו כנור, / קחו עוגב.

במקהלות / תנו קולות,

תנו תודה / לאל נשגב;

אשר פדה / לעם־חמדה,

והוא מעז, / והוא משגב,

צאו שרים, / שאו שירים,

כמו דוד / כמו הימן.





הטוב מרפה, / ולחסם־פה?



ומה־נסתיר / גבורתו?

הלא נגיל, / ברן וגיל;

הלא רבה / אמונתו.

ולחפשה / כבר נשא

לישראל / באברתו.

וגם נלחם / בזרע חם;

שלו הביא, / והמטיר מן.







בכל היקום / הלא נקום

וצדקותיו / נתנה־נא.

ורוב חסדו, / ועז ידו,

הלא נגיד / בכל פנה;

הלא נרן / לזכרון

במועד זה / בכל־שנה.

ומה אחסום / לפי מחסום?

בטח רוחי / בצור רחמן.





והוא יסלח / לאיש נאלח,



וכל־חטאת / כבר ילבן;

והוא יורה / שפת ירא,

והוא יחן / פתיות בן.

הלא אבטח / (ואף אם־טח

מאד עיני / ולא יבן)

באל מישיר / לדבר שיר,

ושם שפה / לכל־נאמן.





ומשכנו / ואפדנו

ברום עולם / שמי שמיו;

ושם ישקיף / גבר תקיף,

וחטאתו / וכל־מומיו.

והוא ינעל / אנוש יפעל

אשר לא־טוב / בתוך עמיו;

ויפיל צר / באין מעצר

במכמרתו אשר טמן.





תשואות חן/ לך בחן



עצת לבב / הלא נשא.

במו גרון / הלא נרן

לברא ים / ויבשה.

לך נודה, / לך פודה,

מאבד צר / אשר הקשה;

ולא נשעה / לעם תעה,



ואל־פסלם, / ואל־צלמן.





אלהים רב, / ובו מי־רב?



ורב כחו / ואזרעו.

והוא ירדוף, / והוא יהדוף

לזד יהיר, / כמו פרעה.

אבל יאהב / לעם נכאב,

ויתהלך / להרגיעו.

הלא כל־אל / פעלות אל,

ולא מעשה / ידי אמן.





אחשורוש / לשר וראש

עמלקי / כבר גדל.

והוא שובב / ברום לבב

מאד קצף / בעם נבדל.

ורוב כספו / בחם אפו

אזי נדב / להשמיד דל.

ובוכיה / קהלת־יה;

והומיה. / כשוד שלמן.





וכל־העם / כאיש נפעם,



וכאבל / חפוי קדקד.

וקול קינה / בכל פנה;

ועת לבכות, / ולא לרקוד.

ועיר מלכות / מלון בכות,

וכל־אדם / בצום יקד.

ויום וליל / בכי וילל,

וישראל / כאיש אלמן.





ויתעטף / ושב וטף,

והחתן, / והכלה –

מאד יזחל, / ויתחלחל,

בלב נמס / ובהלה.

והמלכה, / כאיש נדכא,

במר תבכה / בהיכלה.

וקם שאון, / ותמהון,

והשמה / מקום משמן.





אלהים חי, / חזו אחי,



אשר עשה / לישראל.

גבורותיו / ואותותיו,

אשר הפלא / באון ואל.

ימין תקפו / ראו אפוא;

זכור עמי / והבהל.

זכור צדקו / בעם־חלקו

בכל־שעה / ועת וזמן.





ומה־פעל / אלהי על



ואיך מלט / ידיד נפשו;

ואיך הבדיל, / ואיך הגדיל,

ואיך רומם / בחיר קדשו;

ואיך החל / בבן־רחל,

בהעיר שר / עלי ערשו;

ואיך כבד / לאיש אבד,

ואיך נהפך / לאיש נסמן.





אנוש צורר, / ומשתרר,



ובר־לבב / בלב ישטון,

להכריתו / ולתלותו.

לדבר בא / לאיש שלטון;

והסלם / כבר נשלם,

ועץ נשגב / כהר קטון;

והתלוי / לאל גלוי,

ולא הוכן, / ולא זמן.







כצאת חמה, / מלא חמה



ממהר בא / לבית השר;

ותוך קרבו, / ברום לבו,

אני(אמר) / מאד אשר;

ואנכי / למו מלכי

מאד אהוב, / מאד נקשר;



ומי־טמא / אשר־ידמה



לשר־שרים, / לאיש חשמן?





אויל נמהר, / ולא נזהר,



בראש נשא / ועין גבהות,

הלא כי־כן / כאיש מסכן

בפיו נוקש / ברוב גאות.

אזי שפל / אזי נפל;

ובא שודו / ואין רפאות.

כצלמונע / אזי נכנע;

כמו פרעה / כמו נעמן.





אזי צוה / (ראו הוה!)



להלביש איש / אשר שנא;

ברכבו / להרכיבו,

בסוס נכבד / כשר משנה;



ועל־כרחו / בכל־רוחו



קרא אברך / לאיש נענה.

אנוש נבל / אזי נאבל,

ושת לבו / אלי סימן.





אבל אבלו / אמת כלו,



ולא אחר / אשר־יגור.

הלא לאל / ואון ואל,

הלא אליו / חנית וסגור.

והוא אוה, / והוא צוה;

ומי יפתח / והוא יסגור?

ומי יחמס, / ומי ירמס

ירא אל חי / בלב נאמן?





בעת־משתה, / והוא שתה



עלי שלחן / אחשורוש;

ומתפנק / בכוס חנק,

ומתענג / במשקה־רוש;

ולא חכמה / ורק אשמה,

ורק זמה / לבד ידרוש;

ובזה יה / ותושיה;

כדרך איש / אשר שמן.





ולא ידע / אשר נגדע,



ולא הבין / אשר־הכה.

אזי נגרז / ונגלה רז;

ואין מחסה, / ואין סכה.

והנקם / אזי הוקם,

ולא ריקם / דבר מלכה.

ובא קצו / עלי עצו,

אשר־הכין / תמול וימן.





ורש ודל / אזי גדל,



כמו המן / ועוד יותר;



וכל־חילו / וכל־גדלו,



הלא כלו / לאב אסתר.

והוא פליט, / והוא שליט,

והוא מושל, / והוא שטר.

באין מצה / אזי יצא

בתכריך בוץ / וארגמן.





לאל אלים, / במקהלים,



וכנבלים / הלא נפצח.

ונתכונן, / ונתחנן,

אשר יבא / ברוח צח;

וירא רש / אשר־גרש,



וגם־נגזל, / וגם־נרצח.



ואל־מזרח / תני יצרח,



ואל־צפון, / ואל־תימן.



אסוף אחי / ונדחי,



וישראל / קבוץ על־יד.

תנה רשיון / לעם־ציון,

ויבן עיר, / ויציב יד.

וישבר־מוט, / ולא ימוט,

ולא יירא / פני ציד.

ומארבה / מאד ירבה,

ולא ימד, / ולא ימן.





ועד־עילום / יהי שלום,



ורק שירים / וקול זמרה.

וגר נמר / באין שמר

בבית השור / והפרה.

והצפיר / מאד יחפיר,

ורב יעבר / לעם נברא.

ומלך שר / בכל־בשר,

וממשלתו / בכל־מכמן.





יובל שי / לבן־ישי,



אשר ימלך / בעיר ציון.

ולא ימעד, / ולא ירעד;

לעולמים / יהי עליון.

שאו שירים / ביום פורים,

וברבורים / לאיש אביון.

שאו רנה / בכל־פנה,

שתו,שכרו, / במות המן.
