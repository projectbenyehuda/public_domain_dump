


	
	[חרון אפי כאש בוער ובבשרי כמו קוצים]
	




͝ – –

חרון אפי / כאש בוער / ובבשרי / כמו קוצים

עלי מקרה / אשר קרה / בלכתנו / משבצים

ביום שבת / לשוט בחוץ / למקום שם /

בני סכלות / משנים את־ / לבושיהם / ומתפרצים

וכל מבאיש / לאיש, ויש־ / טמונו ב־ / עלי ביצים

הקימונו / שחוק להם / כמטרה / אלי חצים

וקול אומר /“ירה” ויור / לאישים ע־ / ברים נצים

ובחרנו / חנות חנות / והנוגשים / אזי אצים

וגם שמה / רדפונו / ולא הונח / לנלחצים

עדי נפש / בכף שמנו / ויצאנו / מאד רצים

וכל־אחד / כנשר קל / מדלג הר / וחריצים

בכן באנו / ונמלטנו / ידי שוסים / ופריצים

אהה אל חי / תנה אותם / עלי־אש או / עלי עצים

והדבר / אשר ירו / יהו מהם / מקצצים

ואנו נא / נברך יה / ומעתה / ועד־קצים

ולא נוסיף / ביום קדש / להתערב / לבין לצים

ורק תורה / וזיו אורה / ובמורים / נהי רוצים

אשר יורו / כללי־דת / בגן עדנה / מלא צצים.
