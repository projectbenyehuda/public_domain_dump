


	
	[משורר יערים שריקות עדרים]
	




͝– – ͝– –

משורר יערים / שריקות עדרים /
בהקשיב להדפיס / מצוה וכופה



מליצה ושירה / סתומה וזרה / צריכה לרש"י / וקמחי
וצופה



בידך להכתיר / ואוסר ומתיר / עטרה לחכם /
כשהם וישפה



לשונך חרוזי / לשון אשכנזי / לעיל או לרע לא /
תבקר ותחפה



והנע תניחו / והנח בכחו / תנידו, ודגש / וחזק
ורפה



ואיה נעימות / באיכות וכמות / לקורא בשירך /
ברישה לסיפה



בכל־טור ומלה / המלה ומחלה / ונופל וחלש /
וצריך לרופא



ומה־זה תפרסם / כנביא וקוסם / נבואות נגינות /
לעין טוב ויפה



אהה זו למדה / וחטאת יהודה / כנפח כתופר /
כמשקה כאופה



והמית ישיבה / לך היא תשובה / ומשיר שתק־נא /
ושים יד למו־פה.
