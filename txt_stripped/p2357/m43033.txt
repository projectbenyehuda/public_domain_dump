


	
	קנה לך
	




קנה לך מוסר / ומדות נעימות / ובלתיהן תחסר / לעופות ובהמות

נמלה אין לה שר / ומגזל תוסר / וחטה אוספת / בתוך האדמות

ותרנגול שורר / בחצות יתעורר / ואתה מתגורר / ברב התרדמות

ועוד נדיבות לו / בעת מאכלו / קרב רעיו אצלו / ואם הם בהמות

וגמל צנוע / ואם הוא גרוע / ואתה פרוע / וחושב מזמות

ובטל איך תשב / לעכביש הקשב / אשר תמיד חושב / ואורג בנעימות

ותשוה לדבורים / ביחד נחברים / ובונים חדרים / בשוה בכמות

ושור האילת / ללא לה חומלת / ולכל גומלת / חסדים במימות

ראה העורבים / אשר אינם טובים / והמה אוהבים / אהובות עצומות

ותרנגול הבר / באנסו שגבר / שבועתו עבר / ונחנק על מות

ותורים ויונים / אשר אצלך חונים / ואינם מזנים / כמעשה בהמות

ועופות דרכיהם / שתות את מימיהם / ונושאים עיניהם / למעלה למרומות

ואיך לא למדת / והטוב השלכת / ואכן הוספת / בעול ומרמות.
