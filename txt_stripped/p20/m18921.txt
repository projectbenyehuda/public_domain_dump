


	
	אבינו היום בלי אב דמינו
	



זולת (לשבת איכה)




אבינו, / היום בלי אב דמינו,



בזינו / דברך וכפלים לקינו,



ראינו / יום נשרפה עיר ששם עלינו,



היינו / חרשים ואלמים ולא נפתח פינו,



מצינו / קבעת כוס שתינו.



עליך הרגנו ואליך קוינו,



זכר חסדך ואל תזכר את אשר העוינו,



ראה, כי ארבע גליות גלינו.



גלות מצרים ראשונה, / שנית גלות שמרונה,



שלישית גלות ציונה, / רביעית גלות אחרונה;



בצען ביאור מתו בני, / בחבור נפזרו עדת אביוני,



בבבל נשבו כלי בית אדני, / באדום נפזרו מתי
איתני –



יחד כלם יצאוני, / צרות רבות מצאוני.



שתים היו קראותי, / שתים היום רעותי:



אחת – השד והשבר, / שנית – הרעב והדבר;



המוני / ממרור שבעו,



זקני / בעיר גועו,



כהני / יחד כשכור תעו,



נביאי / זעקו ולא נשמעו.



הנה בגלות מצרים / פרעה הכה אחורים,



החשיך פני ככירים, / העביד בפרך אפרים;



הנהבתגלת פלאסר / לקח בנותי העשר,



נתנם לרעב ולחסר / ועודם צועקים מאסר;



הנה בגלות בת בבל / מעשי נבוכדנאצר חבל,



כלה שארית נחלה חבל, / ענה רגלי בכבל;



הנה בגלות החל הזה / משל בי אדום הלזה,



יבזה אותי – והוא נבזה; / כלם יחד, זה כמו זה,



לחצוני, / מחצוני, / הליצוני, / הריצוני,



ועד קצה הארץ הפיצוני;



נאלמתי, / נהלמתי, / נעלמתי, נכלמתי,



אבי, מצוקי ממך לא נעלם –



כי עזרת אבותינו אתה הוא מעולם.
