


	
	אמלא פי תוכחות(תוכחה)
	






אמלא פי תוכחות / בשירות נכוחות וצחות.



גשו יצורי למשמעתי, / שמעו נא תוכחתי!



ראשי, בא והכנע / ומגאוה המנע,



ותלך פני אל לדרש / אבל וחפוי ראש.



מצחי, חזקך הרף / מחטא! אמנם, לחרף



חטאים היה; עת תלחם – / חזק את מצחם.



עפעפי, הישירו / נגד אל ותאירו,



והסירו באף ובחימה / עפעפי תנומה.



עין, פעל צורך / הבט, ועל יצרך



געל – ובתוך גן רטב / תשוב עיני לראות טוב.



לחיי, תורי זהב / עזבו וגאון רהב,



ותהיו נתונות לשאטים, / לחיי למרטים.



שפתי, לאל שבחו / ובצדקה תשיחו;



לחן בכל עתותי / צור על דל שפתי.



לשוני, מדבר שקר / רחקי ערב ובקר,



היי בשיר ובתפלות / לשון מדברת גדלות.



שני, נא כל טמא / אל תאכלו וכל נדמה,



אולי בבוא יום יי / אתמלטה בבר שני.



חכי, מתק שיחות / תן ואשיר בתשבחות;



צורי, מעסיסיך – / לחכי אמרתך.



אזני, חיש האזיני / למוסר אל ותביני,



כי הוא למד דעה / אזן שמעה.



גרוני, קרא והתפלל / והכנע לפני אל,



ואל נא כנשי יתרון / נטויות גרון.



ערפי, קשיך הרפה / לצורך, פן תספה;



הפנה אל ילדי זנונים / ערף ולא פנים



צוארי, על עולתך / שבר ומוט חטאתך,



פן אמר בהוכחי: / צוארי הכשיל כחי.



כתפי, משא עוני / פרקו, ומשא יי



תשאו יחד, קשורות / שתי כתפת חברת



זרועי, קבצו פזורים – / היה זרעם לבקרים! –



תנו אותה למחיה / בזרוע נטויה.



כפי, שאו ידיכם / לאל בעד חטאתכם,



ופרושות על עוני / כפי אל יי.



בטני, לא עוד מרמה / תבין, ושמר מזמה!



ותהיה ביום דיני – / ומה בר בטני.



מעי, אל אל אוחילה / עליכם, ואשא תפלה:



תנה בינה בתוך צלעי / ותורתך בתוך מעי!



כליותי, במו אמצא – / תיעצוני בטוב עצה,



לשאת יראת מלכי / כליתי בחקי.



חלצי, אזרו גבורה, / לבל אהיה ביום צרה



כגבר, ממעמסיו / ידיו על חלציו.



ירכי, טהרו גויה / ואל תשגו בחיק נכריה –



היא מכשול ופיק ברך / לנפיל ירך.



ברכי ושוקי, כרעו / לאל – ותושעו



ממכשול ונחשתים / ופיק ברכים.



רגלי, לבית אלהיכם / הכינו פעמיכם,



לרוץ תהיו קלות, / רגלי, כאילות.



לבי, יצורים כלם / תחתך; ואם במעלם



שוא – בעדם תהי נגמל, / לבי, על כל העמל.



הדריכם באמונה / לבית אל ברוח נכונה,



לך פנה רם על כל ראש – / ועל לבי לדרוש.



יצורי, יסרתיכם – / תנו נא כבוד לצורכם!



תדבקו בו, בעודכם / חיים כלכם.



אשרי אם תוכחו / מרע, יצורי, ותקחו



בלב שרים מוסרים / ותוכחתי לבקרים.



ובואו כלכם יחד / לפני רחום בפחד,



לשפך לפניו שיחה – / לכו נא ונוכחה!
