


	
	[אני תמה יפת צמה]
	



אהבה

  


אני תמה / יפת צמה – עדן שמה / ביד שמה, / והחמה



שזפתני, / שאפתני, / שרפתני; / זכר, דודי, בריתך!



נפלאתה לי אהבתך.




בבור שביי, / בכור עניי / ברב מריי, / עדן בכיי / עלי לחיי,



ואת עזרי, / בך שברי, / חבש שברי / וכל שבר עדתך!




ראה לחצי, / רפא מחצי, / גדר פרצי / ומים צי / לאור תוציא,



תקימני, / תרימני, / תשימני / לראש כדבר שפתיך.




היה סלעי, / ביום צלעי / קרא ישעי / וסר נגעי, / ומכניעי



באף תהרס / כגוי פתרוס, / אזי תפרש / כנפך על אמתך.




מנת חלקי / וצור חזקי / וצוף מתקי / וכל חשקי, / בך דבקי,



חצוצרותי, / נגינותי / ושירותי / אנגן עוד בביתך.
