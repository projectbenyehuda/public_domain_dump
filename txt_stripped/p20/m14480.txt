


	
	אל פעליו גברו
	




אל פעליו גברו, / על יצורים משלך —

הן צבא רום נפארו / נתנו כתר לך!

אם שרפים קצרו, / אל, לספר גדלך,

סודך / וכבודך —

מי יהלל / מי ימלל

איש, אשר בצעיר יצורך / הוא, ואיך יקרב לשירך?



בעלי בין בערו / באמתת סודך —

ואשר בך נעזרו / ידעו יחודך,

גם בך יתגברו; / לחזות בכבודך

יחפזו; / עת יחזו

זיו יקרך / גם מאורך —

יעלו יום יום בהרך, / שם יאירון בהדרך.



רם, גדלה גדלה / לך לבדך יאתה!

אם להמציא מעלה — / מיקרך היתה

לו, ונפש נבהלה — / בך וממך חיתה!

יֽהיו / בך יֽחיו

הנפשות / הקדושות;

ינעמו מהוד פארך, / ילכו כלם לאורך.



מה גדלת איש ומה / מול גדלה קדמה?

מה גבורה נרדמה / מול גבורה עצמה?

אם נשמה נעמה / מתבונות חכמה —

אין לאל / ידה באל,

רק תשורר / שיר תעורר:

ברכו, עם אל, בניב רך / את אדני המברך!
