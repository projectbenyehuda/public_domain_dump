


	
	אתה אל ראי
	




אתה אל ראי, / את מעריצי, את מוראי!



בחנתי לבי – / אל חסדך אדם אתחולל,

שמך משגבי / יום סוללות ישפך מסתולל.

הואל ופנה בי, / הוצא כפי יקר מזולל

ומשנאי / קום לערץ! / גדרו פרץ, / יבוא רבץ / תחת משאי!



רחש מרחיקי / למצא כבוד אל בחקר דבר,

קרוב מצדיקי / למצא עונו, כי בך נדבר.

מהודך חזקי, / מקול בכח בהדר

קורא לצבאי: / שכל בתם / תורה חתם, / אותי בתם / דרשו בהמצאי!



הרבה מזבחות / לחטא אויל יערך: יתיבב

פעם, תוכחות / יענה – גדולים חקרי לבב!

אחר למנחות / נגש לעצו הולך שובב.

נחי מלטאי / הוא חבלך / ישאלך / הכין לך / בחסד כסאי.



מושיע חוסים, / צופים לבשר שואפים משח:

'משרי מסים / קראתי לבני ואקח

מקרב שוסים; / אבחר בו, מטהו יפרח.

עוד למקראי / יושב ששון / ישעי, שכן / אור על מכון / הר מקראי!'
