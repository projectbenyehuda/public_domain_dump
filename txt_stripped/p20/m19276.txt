


	
	אני אני בתוך אני
	



אהבה






אני אני / בתוך אני



ומחני / כמו אני



בים עני; / והנני,



בחנני / למעני, / כאובות;



בחבלי / באהלי,



במעלי / בפועלי



מחוללי / וגואלי,



בסכלי / בעקלי / נתיבות.



רעוץ ארי / בלי צרי,



דחוי חרי / בשוררי



מצוררי – / וזה פרי



דבר מרי, / בבקרי / משובות.



מכון צבי / בתעבי –



ואוהבי / דמות צבי,



וסוחבי / וירד בי



בבית שבי / באף – לבי / קרבות.



חסין, לכה / ואשלך



וחבלך / פדה לך



בגדלך; / ועלך



וסבלך / ואפלך – / לאויבות!



חדד מכור / וחת עכור



ולו הכור / ופיו סכור,



בבוז נכור / עדי שכור



ואך זכור / לבן בכור / אהבות;



ושוב שנה / וחיש פנה



כיום סנה, / ועם קנה



ועיר בנה / ותחנה,



ותענה / לנענה / לבבות:



׳אקרבם, / אשגבם



ואקבם, / אשובבם,



אישבם, / אסובבם,



אחובבם / ואהבם / נדבות.׳
