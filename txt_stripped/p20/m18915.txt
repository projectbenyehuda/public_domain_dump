


	
	אחי שמעו נא לי
	



קינה לט׳ באב




אחי שמעו נא לי, / אזכיר בהרים קולי / צאתי
ממצרים;



ואש תוקד בכסלי / בזכרי, ביום רב אפלי, / צאתי
מירושלים.



בחר שוכן בקרבי / משה לשסע לביא, / בצאתי
ממצרים;



ונבא בזמן חובי / ירמיהו הנביא, / בצאתי
מירושלים.



גמולי טוב גמלני, / ביד משה גאלני, / בצאתי
ממצרים;



ובזמן אשר געלני / ביד ירמיה הגלני, / בצאתי
מירושלים.



דרשני ברב אותות / ביד בן לו אותות, / בצאתי
ממצרים;



ורמני בחניתות / כהן מענתות, / בצאתי מירושלים.



היה לי למושעות / בעשר מכות רעות, / בצאתי
ממצרים;



ונסי שכינה נעות / בעשר מסעות, / בצאתי
מירושלים.



ויצא בשיר ונבל / עם מעני סבל, / בצאתי ממצרים;



ולעבדות ביום אבל / הגלני לבבל, / בצאתי
מירושלים.



משה זכר ואות / הראני ונפלאות, / בצאתי ממצרים;



וירמיהו חקק להראות / לי מכאוב ותלאות, / בצאתי
מירושלים.



משה, טרם גאולה, /שח: / ׳כחצת הלילה׳, / בצאתי
ממצרים;



וירמיהו על גולה: / ׳בכו תבכה בלילה׳, / בצאתי
מירושלים.



משה, בשירה ערוכה, / שורר: ׳מי כמכה׳, / בצאתי
ממצרים;



וירמיה מקונן ׳איכה׳, / ובאה קשת דרוכה / בצאתי
מירושלים.



משה שם בארון / לוחות אדירירון, / בצאתי ממצרים;



וירמיהו, ביום חרון, / גנז את הארון, / בצאתי
מירושלים.



משה קרא לבני: / ׳עם נושע ביי׳, / בצאתי ממצרים;



וירמיה שח להמוני: / ׳שלח מעל פני׳, / בצאתי
מירושלים.



משה קבל שכינה / בעוד ארבעים שנה, / בצאתי
ממצרים;



וירמיה קינות ענה / בעוד ארבעים שנה, / בצאתי
מירושלים.
