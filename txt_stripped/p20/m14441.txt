


	
	אליה לשבויה בוא יבוא
	




אליה / לשבויה / בוא יבוא בבשורה

לפני בוא / יום יי / הגדול והנורא.



בתחנונים / כאביונים / נקדם את פניך,

זכר הברית / לשארית / צאצאי אמוניך

ואיך אחיהם / ועבדיהם / יעבידום לפניך —



והם חבלך, / עבדים לך, / ככתוב בתורה!



ראה עתה — / הלא אתה / אל אחד ולא תשנה!

אני בזוז / כשה גזוז / ואקראך ולא תענה.

שלח מרפא / ללב נכפה / ותביא מגיד משנה

כדברך, / לבכורך / הלא משפט הבכורה!



'מה תוחיל, / עם הוחיל — / ומועד מועדים ישלם!

ואשיבך / ואושיבך / בהיכל חן והוד אולם,

ושם גואל / לישראל / יהיה לך לאור עולם

ולילה כ-/ יום יאיר, / כחשיכה כאורה.'
