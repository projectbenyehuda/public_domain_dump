


	
	אמונת אלהי אודיעה
	



מי כמוך




אמונת אלהי אודיעה / וחידות קדם אביעה,



גבורת בראשית אשמיעה, / הקבצו ושמעו בני
יעקב.



בטרם היות אורה / ושמים וארץ ברא –



אמון אצלו תורה / ויעמידה ליעקב.



רבה בארצו דשאיו / להחיות נבראיו,



עליהם יברכו יראיו / והקדישו את קדוש יעקב.



הכין במרומים צבאם, / להאיר לאדם בראם,



והם יעבדום בחטאם – / לא כאלה חלק יעקב.



מיני חיות הולידם, / בים וביבשה יסדם,



ויאמר: נעשה אדם, / להקים את שבטי יעקב.



בששה היו צבאות / מעלה ומטה נבראות,



שבת ליי, לאות / בינו ובין יעקב.



נתנה לאדם מצלעו / עזר בגן שעשועו –



הוא המוכן לזרעו, / הנקראים בשם יעקב.



ערום השיא השא / האשה בדבר קשה,



ככתוב בתורת משה, / מורשה קהלת יעקב.



זעומים מעדן גרשו, / ואלה הדברים פרשו,



מכות עליהם התחדשו – / כה אמר יי אל בית יעקב.



רועה – מעשהו זר, / זרעו אכר גזר,



ויארב לו כאכזר, / כי כל אח עקוב יעקב.



העוה היקום ישרו / ומבול נמחה יצרו,



לולי תמים בדורו – / מי יקום יעקב?



מגדל בנו מורים / ומשם היו נפזרים,



אחר ונולדו הורים, / אברהם יצחק ויעקב.



נעקד נטע שתילים – / הגדול איש התלים,



והקטן יושב אהלים; / מה טבו אהליך, יעקב!



מכר הרב בכורתו / לצעיר לקח ברכתו,



ויאמר, כהשמיעו צעקתו: / ׳הכי קרא שמו יעקב!׳



דברי סבות התעתדו, / עליהם בנינים יסדו,



ושנים עשר ילדו – / אלה תלדות יעקב.



נמכר ילד שעשעים / וימלך כאחד השועים,



וירדו מצרימה שבעים / יצאי ירך יעקב.



תמימה שועה בעניה / ובכל מצרים הכו לחיה



ויאמרו: ׳לא יראה יה / ולא יבין אלהי יעקב.



תקף צרה וצוקה / ובכל מתנים מועקה,



וברית אבות צעקה: / ׳מי נתן למשסה יעקב?׳



טוב מארצו ברח, / בזקנתו כשמש זרח,



ממטה לוי פרח – / דרך כוכב מיעקב.



יהירים במכות נצמתו / וכמעט תמו נכרתו,



ראשית אונים הומתו – / ותחי רוח יעקב.



לילה יצאו חפשים / ביד רמה, חמושים,



ומרכבות עם פרשים / רדפו אחרי בני יעקב.
הדורים בים בבואם – / והנה האויב ראם;

׳אריק חרבי, נאם, / ואהרגה את יעקב!׳



ואל ביבשה העבירם – / וסוררים בים נערם,



אז ישיר משה בקול רם: / ׳גאל יי עבדו יעקב!׳
