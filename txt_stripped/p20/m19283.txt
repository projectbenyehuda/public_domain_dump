


	
	אצולים מהודך
	



קדושה




אצולים מהודך, / ברואים מכבודך,



העומדים בסודך / לעשות דברך חשים.



בכחך מתאזרים / וראשם בשמך מעטרים



ורגלם בזכרך מישרים / ובגדי תהלתך לובשים.



רנתם כפי שפתם / ושפתם דברותם



ודברותם נפשותם / ונפשותם סוד הנפשים.



הנה הם תמימיך / ורוחות שמיך



ועל פני מרומיך / הנם נטושים.



מראיהם נוראים / ומעשיהם פלאים,



מאתך יוצאים – / אליך נגשים.



בכסא כבודך מתלכדים / ופעליהם לאחדים –



ובלב נפרדים / והיו לארבעה ראשים.



רוממות אל בפי גבריאל / והשר ענאל



ורפאל, ומיכאל / ראש השלישים.



מהלל בפיהם / כפי מעלותיהם,



ורקיע עליהם / הוא קדש הקדשים.



אמונת שמשך / מהיכל קדשך,



להעמיד באשך / זרע אנשים.



ירוץ הירח / כגבור צורח,



ואורו זורח, / יבכר לחדשים.



רמת מעבדיך – / ומי יועידך?



ומעשה ידיך / ישנים וחדשים!



ויודו שמים / פלאך, יי,



אף אמונתך / בקהל קדשים.



אלהיהם מעריצים, / במסלותם רצים



כגבור עריצים / לעד ולא נכשלים,



בשם הנכבד יפגשו / ומזמר לא יחשו,



יום יום שיר יחדשו / והם אינם בלים.



רגליהם לא יגפון / וברצונם לא ייעפון



ומעפעפי שחר יעופון / ולמקומם לא גולים.



הנם לצורות התכבדו, / נועדו והתעתדו,



זו עם זו יתאחדו / ואלה מאלה נבדלים.



ממקום קדוש באים, / מהם שפלים וגאים,



וכלם נפלאים, / קטנים וגדולים.



במתכנתם עומדים / ועל היכלו שוקדים



ומפעלותיו לומדים / ואין בהם סכלים.



נצח ישכנו שם / וממאורם לבושם,



וכתוב בראשם / קדש הלולים.



עינים מלאים / וברוח נשאים,



ומברקים יוצאים / נחלים וגחלים.



זקופים, נצבים / על סוסי להבים –



והמון הכוכבים / נושאי כלים.



ראשיתם אחריתם / ואחריתם ראשיתם



ונוסעים בתבניתם / דגלים לקראת דגלים.



הכל יעידון / וכבוד אל יגידון



ושמו לבדו יודו – / וזולתו הבל הבלים.



כי מי בשחק / יערך ליי,



ידמה ליי / בבני אלים?



אליו פונים / והם הם הפנים,



עליו חונים – / ואין יוצא ואין בא.



בשמות נקבו / וכעבדים נצבו –



וכאדונים נחשבו / על ראש המצבה.



רוחות נאמנים / במעלות שונים –



ושבעה איתנים / נתנו להם לרכבה.



המה הגבורים / בחיים צרורים,



טהורים, ברורים, / בלי שמץ ודבה.



מה נכבד כל איש, בשבתו / על כסא ממלכתו



לעשות מלאכתו, / והמלאכה מרבה.
