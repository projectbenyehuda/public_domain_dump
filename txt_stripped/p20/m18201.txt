


	
	אנא שא נא לפשע כנה (תחנון לצום גדליהו)
	




אנא שא נא / לפשע כנה,



מענה, נתונה / למשל ולשנינה.



פנה נא לכוננה, / היה לה משענה



מראשית שנה / ועד אחרית שנה.



בשמך, פודה, / קרא עם עולם,



ולך מודה / כנוקש וישלם.



הנו בכף צודה / מכה ונכלם,



כרחל בידי / גוזזיה נאלם.



בשבט רודה / מרדף ונהלם,



גולה, מנדה / ונגרש מאולם –



ויוצא השדה / שנה שנה.



רודפו יכלנו / ועונו הרצה,



כל שן יאכלנו / מנפש עד נוצה;



כל לשון יקללנו / והיה לשמצה,



כל גא ישפילנו – / ופיו לא פצה.



באש יפילנו – / ומהאש יצא,



והאש תאכלנו / שנה שנה.



הקם מפלתו, / יושב במרומים,



והרם שפלותו / ופקדהו ברחמים,



והעבר חטאתו / ואל ימות באימים.



זמן רצון חקקתו / בעשרת ימים –



לכפר קבעתו, / למחות אשמים



ועון איש וביתו / דבר שנה בשנה.



מעוז ראשו – / לפקדו הזכר,



והסר כחשו / ועוכרו יעכר;



ופדה נפשו / מיד בני נכר,



ופאר מקדשו / ומעינו הסכר,



אשר רקת ראשו / כחוט, ולא יסכר,



וכל עץ לחורשו / לחדשיו יבכר –



חדש בחדשו / לחדשי השנה.
