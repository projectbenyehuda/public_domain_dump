


	
	אין אלוה מבלעדיך
	



מי כמוך






אין אלוה מבלעדיך / ומעשה ידיך עדיך,



ולאין חקר חסדיך / וצדקתך כגלי הים.



בטרם עגלה וחלדה, / בחכמה נועצת לבדה –



ארכה מארץ מדה / ורחבה מני ים.



רבות עשית ועצומות, / נלאו לחקרם הנשמות –



אלו מלאו חכמות, / כמים יכסו על ים.



השמים ימינך נטתה / והארץ ידך עשתה,



ותהו ובהו היתה / כדמה בתוך הים.



מאורך אור נברא / ובדברך יום נקרא,



ראשו ממזרח ירא / ואת עקבו מים.



בינות המים, להודיע / פלאך, הבדלת ותרגיע –



חצים ממעל לרקיע / וחצים אל הים.



רמת ודברך גאה / בגבורה מכל נפלאה,



יבשת הארץ הראה / ויסך בדלתים ים.



ביבשה צמחו דשאים / לתת מחית נבראים,



להשקותם יעלה נשיאים / הקורא למי ים.



ישתחוו לו עבדים / שמש וירח וגדודים,



בגלגל עולה מקדים / ונסב לפאת ים.



מחשבות אל מה אמצו / וחקותיו לכל חרצו,



וממים בחמישי שרצו / צפור שמים ודגי הים.



אדמות ששו בהולידם / בהמות, בעפר יסודם;



ויאמר: ׳נעשה אדם, / וירד מים עד ים.׳



יצר מצלעיו עלמה / המאכילה עץ בערמה,



השמה הנושבת נשמה, / המחרבת ים.



רמם נחש, ונגעו / בעץ – וקול שמעו,



דק וחוג התרועעו / ממזרח ומים:



׳עתה, הנחבאים, אמרו: / הלא כל מאתי נוצרו –



אראם, ואם יסתרו / מנגד עיני בקרקע הים.׳



זדונם הודו באחרונה: / חננו, חננו, שוכן מעונה,



והעבר חטאת ראשונה / ותשליך במצלות ים!׳



רומם – לארץ חללו / ובגן השכין חילו



מקדם, ויאמר לו: ׳נא הבט דרך ים!׳



הורג אח בשדה / בקש במה יפדה,



ויאמר: עוני אודה, / כי גדול כים.׳



ברבות האדם נסכלו, / דרכם השחיתו ונואלו,



השקט לא יוכלו – / והרשעים כים.



מבול שטפם וגועו, כי מימי זבול נגעו,



ומעינות תהום נבקעו – / ויהי סער גדול בים.



דורכי דרך תמימה / נמלטו בספינה בחכמה,



ולשנה יבשה האדמה / ונשתו מים מהים.



ילדו בנים כחפצם, / ובבנות מגדל לרבצם –



משם אלהים הפיצם / ונעו מים עד ים.



נגה אב, ובשכלו / יצא על פי מחוללו



מארצו, ויט אהלה / בית אל מים.



תפשוהו לאור זרים / ונמלט, והיו אומרים:



׳על כן בארים / כבדו יי באיי הים!׳



רעשו שמי מעונו / בשלח יד בבנו,



ומצוקי ארץ למענו / קולם כים.



וישמע: ׳הרף ידך / מאת בנך יחידך!



עקב זאת – ארבה הודך, / ושמתי את זרעך כחול
הים!׳



מרום, רב עליליה, / שלח נזיר למחיה,



בשבעים ירדו, והיה / מספר בני ישראל כחול
הים.



העבידם תנין במרדו – / ולפדותם שלח עבדו



נטה שמים לבדו / ודורך על במתי ים.



חנסים לשמע מאנו / ובעשר מכות נבחנו,



וידידים נסעו ויחנו / בין מגדל ובין הים.



בן נשא עיניו – / והנה סביבותיו עונניו,



ביבשה העבירו עניו / ומטהו על הים.



רדף אחריהם פרעה, / ובסור אופני ישעו –



חשף אל זרועו / והרג את התנין אשר בים.



וישראל נפלאים ראה, / וישר שירה נאה:



׳אשירה ליי, כי גאה גאה, / סוס ורכבו רמה בים!׳
