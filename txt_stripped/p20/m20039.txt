


	
	ארך זמני כמה וכמה
	



אהבה




ארך זמני / כמה וכמה, // סר הוד ופנה / שמש ובא.



נחר גרוני, / יבש בצמא // לדבר אמונה / שנכתבה.





בשבי איחל / עד בא דברי, // ימים ושנים / אין לי
דמי,



ירא וזוחל / שבתי בצירי, // לקח מעני / כל נעמי.



איך צר, ינהל / עמו דבירי, // יאמר לבנים: /׳מי צור
ומי?



חוזכם פלוני / פתה בערמה // אתכם – ועלה; / שוא
כי צבא



אל קם להניא / לב אח ואמה, // אך זאת דעו נא / כי
נכזבה.׳



׳רכסי אנשים / באו לרגלי,׳ // אמרה כבודה / תוך
בית שבי,



׳בני גרושים / מבית זבולי, // ויקר וחמדה, / ארץ
צבי,



תופשים קדשים, / ויאמרו לי: // בא יום פקדה, /
שפלי שבי!



אנא, אדני, / רחם אימה // מהיות שנינה; / לא
שגבה,



בכאב ועני / נשאה כלמה, // בדד בחונה / היא
יושבה.׳



׳הרבות דברים / עזבי, גבירה, // עלה באזני / קול
צעקך,



שם בספרים / חזות שמורה: // שופטך יי /
ומחוקקך,



אשא וארים / ידי מהרה // לשפך חרוני / על דוחקך



וברב גאוני / לעשות נקמה // בבני עדינה /
אתיצבה,



לשמי תתני / שירה נעימה // כי עת חתנה / הן
קרבה.׳



׳מר ואהלים, / קנה וקדה // עם קנמן ב־/שם תרקחי,



נסו צללים / כיום והפדה // אותך גביר בו / עוד
תשמחי,



והמון נבלים / שמוך לנדה // מלכי בחרבו / ישים
סחי.



יום צר חרוני / אהבה קדומה // אזכר ליונה, /
ואשובבה



הור בת המוני, / גם אקראה: מה // יפית ומה נ־/עמת
אהבה.׳
