


	
	לספר את כבודך
	




לספר את כבודך / וגדל נוראותיך

ולהגיד הודך / דפקתי דלתיך

ונפלתי בידך / אקוה חמלתך –

ואני ברב חסדך / אבוא ביתך.



אמונתך מיטיבי / הלבבות יעידו

והקיצו שוכבי / יצועיהם, והנידו

שנתם, ומשיבי / אמרים העמידו

בעד חובי- / הם אשר בהם זדו,

ומדינך הרבה / נבהלו ופחדו –

וייראו ישבי / קצות מאותתיך.



בחסדך, אל נעלם, / אמוניך בטחו

והשכימו כלם / ודברי שיר לקחו,

ואת עזם וחילם / מידך לידך ברחו

ובטוב מהללם / כבוד שמך שבחו.

האזינה קולם / ואם בריתך זנחו

ונודה לך לעולם / נספר תהלתך.



רחם עת קומך / לחקר את פעלינו,

והנה מזעמך / אימתך עלינו

נפלה, ומעמך / סליחת מעללינו

ניחל, ובהדומך / נשא את קולנו:

נא הטה שלומך / כנהר אלינו –

ואנחנו עמך / וצאן מרעיתך.



מרום למיחדיך / פקח עין לראות,

באים עדיך / ונושאים קול תשאות

למולך ידידיך / לקרב קץ פלאות;

ושוב נא לחוכיך / אשר מצאום תלאות –

למען עבדיך / שבטי נחלתך.
