


	
	אמרי הקשב
	



סליחה




אמרי הקשב / ואל נא תשב / פני בחסדך,



לשמך אוחיל / ולבי יחיל / מרב פחדיך.



יה, סמכני / גם תמכני / בעמדי לפניך –



ואל תשכחני!



עזרתי היית, אל תטשני / ואל תעזבני.



בנשאי עיני / נפלו פני / מרב זדונותי;



ברגז רחם / תזכור, ונחם / לי מיגונותי,



מהם דלני / והצילני / וכבש עונותי –



ואל תדיחני!



רחפו עצמי / מרב אשמי / ולמאד נכלמתי,



ומעי המו / לעת בי קמו, / דומם נאלמתי,



ואין לי פעל / לכפר מעל, / ומראות נעלמתי



יום תוכיחני.



הנה דמעי / לגדל פשעי / עיני שוטפת,



ונפש נמוכה / ובשמך תמוכה / בשיחה עוטפת;



סלח שגיונה / בחין הגיונה / למולך נוטפת –



עצתך תנחני.



מחילה אשבר / ולכן לדבר / הנה נא הואלתי:



תמך לי פעם / אך הפעם / אותה שאלתי,



כפר פשעי / ושא את רשעי / ואשר נואלתי –



ואל תזנחני!
