


	
	איך אני לבדי נד
	



מאורה






איך אני לבדי / נד – וכל זאב / ירדפני?



מה לצור לבבי / סר – וכל כאב / יאחזני?



בעון אבותי /נהפך אבי-/עד לאורב,



ואקו למתי / מועדי צבי / חן יקרב.



שממו בנותי, / החלם שבי, / עד לחרב



נהפך לשדי, / עד כלות חלב / שד המוני –



אך לבב יריבי / יאמץ כלב / תחכמוני.



רב לפי לישים / עד טרוף, והן / עד לכלה.



בקשו אנשים / כל צרי – ואין / לי תעלה,



כי עדת קדושים / נמשלו עדן / כחללה.



[…] לעבדי / שב ויחצב / עז גאוני,



יאמין בנביא / קם ויעצב / בו לקוני.



מי ינחמני, / אוהבי, וצר / לי וכחי



תם וישטמני / דוד ומעצר / אין לרוחי?



אחזה בעיני: / נתנה חצר / אל לאחי.



יעלז לנגדי / צר בהחרב / בית מעוני.



אור שלח והביא / לי – והערב / אור למוני!
