


	
	ובכן אשפך לפני אל
	



רהיט




ובכן אשפך לפני אל לבבי / בזכרי דברי אליהו
הנביא.



אל, לשועי היה קרוב / ולטוב עבדך ערב / בפנות
היום לערב.



אז נסע עם אל – / וישלח מלאכים אל / כל גבול
ישראל.



נקבצו נביאי תהו / לחקר אלוה מי הוא; / ויאמר
אליהו:



׳יחדו אל, נאלחים, / ועל סעיפים מתנצחים / עד מתי
אתם פסחים?׳



אז, בשמעם דבר נעם, / נלאו להשיב טעם – / ולא ענו
העם.

׳בינו סוד מחשבי, / כי לאלהי אבי / אני נותרתי
נביא.׳



רבו, באפס ישרים, / נביאי אלהים אחרים; /
ויתנו לו שנים פרים:



׳האחד יהי לכם / ואש לא תשימון כדתכם / וקראתם
בשם אלהיכם.



עשו ראשונה יחד / וקראו בשם הנכחד – / בחרו לכם
הפר האחד.



זאת תעשו – וכן אפעל!׳ / ופר עשו מתי מעל – /
ויקראו בשם הבעל:



׳ראה, מקרא גרוננו / נחר, באש חננו! / הבעל,
עננו!׳



אז הנביא בשכלו / התל במלולו: / ׳כי שיח וכי
שיג לו.׳



בקול רם כלם צורחים, / מתגודדים נאנחים / בחרבות
וברמחים.



ראה כי אין שומעם / קראם משכיל טעם – / ויגשו כל
העם.



מזבח הרוס לפנים / אז רפא במנינים / שתים עשרה
אבנים.



אמונת שדי נעלה / בישר פעלו גלה – / ויעש
תעלה,



יערך ונתח ושח: ׳פנו, / וארבעה כדים תנו!׳ /
ויאמר: ׳שנו! וישנו,



רצו ושלשו כאלה / ומים – החל וכלה, / וגם את
התעלה מלא.
