


	
	איך היה כמו ציה
	




איך היה / כמו ציה / בית קדשי ואולמי,

וסר הודי / וכבודי / וגם עזי ותעצומי?

ברב פשעי / וגם רשעי / וזדוני ואשמי

אסרתי / ונמסרתי / בכף קמי וגם ל[…]

אכלוני / וכלוני / והגלוני ממקומי

ובאפיקים / בכל ארקים / נזרו אהלי לאמי –

אל תאיצו לנחמני על שבר בת עמי.



ביום צרה / [ויום] עברה / וגם אילילה,

ואת אב[לי] / כמו אבל / על עיר המהללה,

[…] שמם / וגם דמם / מקום שירה ות[הלה],

וסר נרי / ומאורי / ופס זבח וגם עולה,

וכל שמחה / אזי ברחה / והשמו שיר גילה,

ויד צרי / בכל ערי / ואני בא[ת…..] לה –

ומשנאי / ומקניאי / נתנו [על על] שכמי.



[רם], באפו / ובזעפו, / אש העביר בחומותי,

[וי.ש.] / וגם השח / אל ארץ דמעותי.

וצר טרף / וגם חרף / והרבה כלמותי,

ויחלל / מקום הלל / וגם הרבה שוממותי,

ויגדע / מקום מדע / וכל אנשי מזמותי,

ודכאני / והלאני / והגדיל את נקמותי –

והן כלה / וגם בלה / אלהי בשרי ועצמי.



המון ש[ר]י / ואדירי / באף ספו וגם כלו,

גם חכמי / ותמימי / במו חרב שכלו,

ובן תרדיון – / ברב קשיון / ענוייו הגדילו.

ועת נתפש / ביד נמאס / וצרים בו משלו –

הוא ואשתו / וגם בתו / ביד אויב נפלו

ברב צרות, / וגזרות / עליהם אז הטילו;

ושלשתם, / בעת צאתם, / קבלו וגם סבלו

דין יוצרם / ואבירם, / והודות לאל מללו –

לזאת אקונן / ואתאונן / בעת שבתי ועת קומי.



מרב דויי / וגם שביי, / אלהי, שור והביטה,

וכל אויבי / ומריבי – / אותם באף המעיטה,

ואליהם / כפעליהם / השב וגם תשפוטה,

וכוס חמה / ביום נקמה / עוד להם הלעיטה,

לענותם / להכותם / ורום למא[…] תנקיטה.

וכל אונן / אזי ירנן / בעת אותי תשקיטה,

ואז נשמח / בעת […] / תוגרמי ואדומי.
