


	
	אל עליון שעה הגיון (פזמון)
	






אל עליון, שעה הגיון / עם באים עדיך!



אזן רחשם / שמע לחשם / מכסא כבודך!



לך היום / מצוא פדיום / נאספים ידידיך –



למשפטיך עמדו היום כי הכל עבדיך.



ברית האב, / אשר יאב / עשות תמיד רצוניך,



ובן נפקד / אשר נעקד / במזבח לפניך –



זכר נא ברית / ואל תשבית / אמונת כל בחוניך!



זכר לאברהם, ליצחק ולישראל עבדיך.



רשעתי / ופשעתי / ואכלתי זכות אבותי –



ואין מעשה / היות מחסה / ביום תפקד עונותי.



מחק פשעי, / מחה רשעי / ואל תזכר חטאתי.



זכר, אדני, רחמיך אל עם סגלתך.



האזינה / והבינה / עם ייחל ישועתה,



בלב זוחל / לך מיחל / לפתח בני תמותה.



אזר חיל / כאיש חיל / וקומה נא לעזרתה,



אל, לנו ופדנו נא למען בריתך.



מבחיני / ומאזיני, / האזינה אמרתי,



מאד תערב / וגם תקרב / לפניך רנתי.



צור עזי / ומעזי, / לך היום תפלתי



לך אדני עת רצון אלהים ברב חסדך.
