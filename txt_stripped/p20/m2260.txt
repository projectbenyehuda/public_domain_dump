


	
	על הים
	



[א]


אפרש כף / אף אכף / אל אלי,



הוא חילי / מגדלי / בצר לי,



אקד לו / על גדלו / שהפליא.



בדבר תך / מטה תוך / ים רגלי



ואמר / “רזי לי, / רזי לי!”



וישמע / מהיכלו / קולי.



בין גלים עברתי / אמרתי: נגזרתי –



אמר צור עולמים / “אשביח שאון ימים!”


  


בים בוגד / מנגד / השלכתי



את נפשי, / ולמוקשי / דרשתי,



ובשורי / יום נכרי / נחמתי,



וים בזמן / זה נאמן / חשבתי.



וביגוני / על פני / נרדמתי



ובספינה / כמדינה / עברתי,



עזבתי / את ביתי / נטשתי.



מה לך ים תבהיל מך? / גר אנכי עמך!



למה אפך חרה, / מה זה רוחך סרה?


  


רוחי בי / בלבבי / הומיה,



וים אמר / "אל תאמר: / מה היה?



הלדרך בי / ובקרבי / תושיה?



וליועץ / לוח עץ / אין מחיה



וכל מלח / הן נמלח / בשאיה,



ורב חובל / רב חבל / רב בכיה,



והאניה – / תאניה / ואניה.



אל נא תלון עלי, / כי אתה בא אלי,



ואני איך אנוח – / ותבוא בי רוח!"


  


"החרש, ים, / כי קים / בטחוני,



ובסער / הן יגער / צור קוני.



ואין שבתי / או לכתי / ברצוני



ובכתבו / בך לבוא / צוני



ובראשית / קץ השית / אל אוני,



גם קצים / נחרצים / מי יניא?



בהכינו / שמים / שם אני!



סודו איך אביננו / ואסתר ממנו?



בידו ים וחרבה – / ואני אנה אני בא?"


  


משיב לי / ים "בבלי / לב שחת



ובאונך / וזדונך / הלכת –



ואת עצור / ואף נצור / נותרת!



זכר דגים / תענוגים / אכלת –



בלי שכל / תאכל / גם אתה!



איש חמדה, / קח מדה / מדדת



הלא ידע– / ת אם לא / שמעת:



עין אל על מפעלו / גמול ידו ישיב לו,



בראשך דמך יהיה – / מת אתה ולא תחיה!"


  


אל גאה, / תראה / גאותך,



פדה וראה / הנכאה / בחמתך



הנשא / על כסא / חמלתך!



הופיעה / והושיעה / בן אמתך,



השואל / לצאת אל / אדמתך



אעבדך / גם אודך / בעדתך,



לדור ודור / אודיע / אמונתך,



אגיד זאת בכל מהלך / ואשלם תודות לך



אשירה ברנני / ישועתה ליי!


  

[ב]


ממים רבים תמשני, / רוחך טובה, תניחני!


  


איך לב נע ונד ינפש?



יירא מרוח ורפש,



כי באו מים עד נפש.



קומה, אלי, הושיעני / ומצולה אל תבלעני!


  


בידך עתותי – ואני מה



אחשב בים או באדמה,



ועמך הייתי כבהמה?



בחלדי בעצתך תנחני / ואחר כבוד תקחני.


  


רפו ידי מאימיך



יום סער עלי ימך,



ואחזק ידי בשמך:



אם אתה לא תעמידני, / חטאי אל בור יורידני.


  


מדי היותי זאת אזכרה



בבוא רוח צפון בסערה



ועבר עלי בים צרה



יום קראתי ותענני / לאמר “הנני הנני!”
