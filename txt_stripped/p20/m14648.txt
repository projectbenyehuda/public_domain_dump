


	
	יהמה לבבי על נדוד
	




יהמה לבבי / על נדוד כרובי / וגואלי.

דל מאד בשרי, / שח כאוב דברי / למחלי.



ארכו שנותי / בבית שבי,

קצרו ימותי / בעיר צבי,

נתנו נאותי / לאויבי.

אויבי סביבי, / בערו שביבי / וגחלי,

מי בכי נהרי / לי ולי יאורי / ונחלי.



בוערים לבולם / יהללו,

יבערו בפסלם / ויכסלו.

צור, גער בחילם / ויכשלו!

שואגים כלביא / לי ונחרו בי / בהללי

את שמך, וצרי / קם במאסרי / לקללי.



רננת רשעים / כעב מחה

ועדת מרעים / כקש דחה.



נודדים ונעים / לאט נחה.

אעלז בשובי / עוד לחיק אהובי / באהלי,

אשכחה מרורי / שם בשיר מררי / וקול כלי.



משכנות תהלה / כאז בנה,

מחנות סגלה / קנות שנה,

משפטי גאלה / לך – קנה!

יוצרי ואבי, / ציר שלח ונביא / לנהלי;

שוב שכן בחדרי, / קום היה בעזרי / לגאלי!
