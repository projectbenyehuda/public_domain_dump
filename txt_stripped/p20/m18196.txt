


	
	חרדו רעיוני (אופן לראש השנה)
	




חרדו רעיוני, / כי אלהי משפט / שוכן עד וקדוש, /
מדבר בצדקה.



באהבו עם המוני, / שם לו חק ומשפט – / ודת יום זה
קדוש / תהיה לו צדקה,



ויגבה יי / צבאות במשפט / והאל הקדוש /
נקדש בצדקה.



אבירי בריאות / ארץ, החישו / לרצות בוראכם /
וחסדו יאמצכם,



ובנפשות צמאות / לו שיר הגישו, / הוא ירוה צמאכם
/ ודשנו יחליצכם –



את יי צבאות – / אתו תקדישו, / והוא מוראכם / והוא
מערצכם.



במבטא שפתים / לך אערך רנני / ועניני תהלתך
/ עלי לבי חרושים;



גדול במחנים / שמך, מעון מעוני: / ברום –
בהמלתך, / למטה – בנפשים,



ויודו שמים / פלאך, יי, / אף אמונתך / בקהל
קדשים.



ראך מבשרי / לב, שמתו לצופה, / שעפי – הושיבם /
במקום הקדש.



מאד אשמח, / בזכרי / יום תת חק יפה / לבנים, בבוא
מחובבם / מרבבות קדש –

רכב אלהים ר-/ בתים אלפי / שנאן, יי בם / סיני
בקדש.



הכל מגידים / גבורתו וגדלו / בעולמים שנים /
יהללוהו יחד:



משתחוים ומודים / מתים חוסים בצלו, / וצבא
השמים / ישרתוהו בפחד –



שרפים עמדים / ממעל לו, / שש כנפים / שש כנפים
לאחד.



מבקשים איה זה / מחנך, קדוש, / בני אלהים, כי
אות / כבודך בם והודו;



ראהו במחזה / איש אלהים קדוש / על כסא פלאות /
וסביביו גדודו –



וקרא זה אל זה / ואמר: קדוש, קדוש, קדוש / יי
צבאות, / מלא כל הארץ כבודו!
