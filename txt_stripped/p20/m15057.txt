


	
	אסיר תאוה שמע מצוה
	




אסיר תאוה, / שמע מצוה / לחפשי תשלחך:



בדת אל הגה / ואל תשגה / והיא נפלאות תשיחך.



ואם נפשך / תבקשך – / שמעני, אוכיחך

עלי חכמות / בני אדמות, / אשר בקשו להדיחך:

איך תדרש / סוד הראש – / ותלאה לחפש רוחך?

ואיך תחזה / למה זה? / ונקל מזאת כחך!

ועל אמונות / התבונות / מי זה אשר יבטיחך?

בצל אל חסה / וחקיו עשה, / כי הוא יישר ארחך.

הוא נצחך, / הוא נכחך, / הוא בקרבך, הוא סביבך –

ואהבת / את אדוני / אלהיך בכל לבבך!



בכל מראה / אשר תראה / עד, כי אין זולתו.

וכל שמע / אשר תשמע / ישמיע תהלתו.

וכל ריח / תריח / יודיע עלילתו.

וכל טעם / אשר תטעם / יבאר סוד גדלתו.

וידיך – / עדיך, / להמיש בם פעלתו.

וכל מדע / אשר תדע – / בך נכון במלתו.

וכל שכל / אשר תשכל – / מאתו תחלתו.

הוא בראך, / הוא נשאך / באהבתו וחמלתו,

יעודדך, / ילמדך, / באחריתך להיטיבך!



ראש וסוף / אם תכסף / לדעת משכרתך –

בראשיתך / ואחריתך / אתה כפי מחשבתך;

כטובתך / כרעתך / נתונה לך כיכלתך,

ויוצרך / יחקרך / וידע קומך וצאתך.

ואיך תשטה, / ואן תטה? / ומדעתו – דעתך!

עזב הבל / מתי תבל / ושמר את משמרתך!

ילוד אדמה, / חשב: למה / תיגע לזרים בלתך?

פקח עינך / והט אזנך, / ראה: שאלתך אתך!

והבורא / בקול קורא: / ‘אם תשוב – ואשיבך!’



מה ישאל / לב נואל – / וטובת אל עודפת,

ומה תירא – / והעזרה / עליך חופפת.

גן עדנו / הוא רצונו / ובלתו מקרא תפת,

כי יחידת-/ מואס-בדת / באש הנדוד נשרפת.

שמר חקים / צדיקים / המתוקים מנפת,

תתרומם, / אם תשימם / על ראשך מצנפת.

הם הודך – / ועל ידך / תנמו לאות ולמופת,

בם הבן – / והיו בין / עיניך לטוטפת –

אשר תפן / אליהם, פן / יסורו מלבבך.
