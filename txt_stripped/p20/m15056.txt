


	
	אלי האזינה לקול תחנוני
	




אלי, האזינה / לקול תחנוני,

את הגיגי בינה, / שוכן מעוני.

עם בא בתחנה / שמעה, אדני,

צועקים: רחמיך / רבים, אדני!



בין בני מוניך / שופכים לחשם

היום בניך, / ונושאים נפשם;

עשה למענך / ותמחל ענשם,

צור, ויאמרו: תמיד / יגדל אדני!



רחם מי-מנה / ושכך זעמך,

מלט והושע נא / שארית עמך.

לך תצעק יונה / ותקרא בשמך

כמאז, אשר נטעה / ימינך, אדני.



היום, אדון עולם, / תסלח עונם

תשמע את קולם / והגיון לשונם,

צור, כי סר צלם / ואוי‎ב ענם –

הלעולמים / יזנח אדני?



מעלי ימחל / בחסדך, אלי

ופדה בן רחל / ועשה להפליא.

שובבו, כי יחל / לך נס גואלי –

חי, עורה! למה / תישן, אדני?



ממך יראתי / ושחר קמתי

אל בית תפלתי / ולב שפכתי.

העבר חרפתי / אשר יגרתי!

לך אמרה נפשי: / חלקי אדני.
