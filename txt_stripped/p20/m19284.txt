


	
	אבי רכב ישראל
	



אהבה




אבי, רכב ישראל, / שר אל מלאך, רד עם אל –



נפל כרע / מרע / מרע / מסתולל,



כסאו מגר, / יגר / חוגר / מתהלל.



בן יקיר מחיק נגרש, / גם בן האמה ירש,



ישכיל עבדי / נגדי, / בדי / בל שרש,



משחית הודי / רודי; / דודי, / בל תחרש!



רב למשנאי יתנשא, / ילעג – ולשור ולמשא



גנת ביתן / איתן / נתן / למשסה;



רוח יבצר, / יעצר, / יצר / במשא.



השב למזרח לצבי / בת נדיב משבי,



וצבי מעם לועז / העז, / צור, עז-/רתי בי,



מבית כלא / עולה / אעלה / אל אבי.





׳מושיעך הנה / זה בא, / בתי, ירתם מרכבה,



עדן נחלי / יחלי, / בחלי / פז צבא,



יקח ריחו / רקחו / לב חו-/לת אהבה.׳
