


	
	אל חי יקרא
	






אל חי יקרא / המקרה / במים עליות



ובישעו פודה / עם יורדי / הים באניות.



בים שם דרך / אל צרך / עוברי ארחות ימים,



באו בספינות / כמדינות / בינות גלים רמים,



על מים עמדו / כי למדו / מתבת איש תמים,



תבות במכונות / משתנות / תחתיות ושניות –



ואני מצרה לי / אקרא אלי / מבור תחתיות.



רוח אל תנחם / ותניחם / צדקותיו יגידו,



בשלשה נסים / אות עושים, / יעלו גם יורידו,



ומלחיהם / בכליהם / המים ינידו.



ברוך מחכימם / ומשימם / לשכם אחד, להיות



רבי החובלים / בה מושלים / כנפשים בגויות.



הן באני כפר / הוא כופר / החונים על ידו,



ואני קטן לו / על דגלו – / ההוא בעל סודו,



וכלי ברזל בו, / וסביבו / ילך עד יעמידו;



ואנשים קשים / ושלשים / עומדים בשלישיות,



גם חצי רבים / בם רבים / עם חרב פיפיות.



מי יספר מעשיו / או נסיו? / הוא רב עליליות,



הוא מפליא סבות / במסבות, / הוא מגדיל תושיות,



אין חכמה נגדו; / הוא לבדו / ידע האותיות.



אל נחל עדנו / צופיות / כל נפשות החיות



ובידו גלות / עליות / גם גלות תחתיות.
