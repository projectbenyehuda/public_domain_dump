


	
	[חיה עצומה מכל בריאה]
	



אופן

  


חיה עצומה / מכל בריאה, // מבן אדמה / דעתה פליאה.


  


    אנה מעונה / משם נשמה?



     באיש נתנה, / כמה חכמה!



      אל חי צפנה / עד נעלמה.



איכה ועל מה / היתה כלואה // בגו, ובמה / שמע וראה?


  


     במה יקרא / שם רם וגאה? –



     ודמות וצורה, / משמע ומראה



     אין לו, ונורא / זכרו לנראה!



שבתי כדמה / מזו קריאה // ואשמעה מה / אמרה נבואה.


 


     רצו שחקים / לילה ויומם,



     מצר חזקים – / מי זה הרימם?



     כל המצוקים / ארץ מקומם,



כלה בחכמה / קנין מלאה // אך על בלימה / תוכם תלואה.


 


     מראות המלות / אין קץ לאורם,



     חוזרים חלילות / במעון יקרם,



     נושאים תהלות / תמיד לצורם;



שירה נעימה / על פה נשואה, / עונים באימה, / אומרים ביראה
