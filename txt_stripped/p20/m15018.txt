


	
	אל אלהי הרוחות
	




אל אלהי הרוחות / שת חכמה בטוחות, / הרחמים והסליחות.

אמנם הסכלתי עשה / במרד לבי בעושו, / גדול עוני מנשוא!

נעורי כימי שכור / ושיבה ראשי תעכר – / וראיתיה לזכר.

ישובון עוד רעיוני / בתם להעמיד בניני, / לפני שחת יי.

אשרי מואס הבליו / ומישר מעגליו, / בחמלת יי עליו.

בחק שם לי השיאני, / יום בו הוכיחני – / בדרך נחני.

ראש בואי ומהלכי / איך לא אבין בתוכי, / כי עירם אנכי?

היך לבי הכבדתי / ותאוה, בה נלכדתי, / בכל כחי עבדתי?

מרגוע אמצא, כי / אשא קולי בבכי – / כי ירא אנכי.

בנתי ואדעה, / כי מחר אגועה – / ואיך אעשה הרעה?

חלד כצל עבר – / ויש תקוה ללב נבר, / כי נכון הדבר:

בשוב הגו אל שרשו, / שב רוחו לקדושו / ונפשו קשורה בנפשו.

לאל חיי נתיבותי, / עד אם תמו שנותי / ושכבתי עם אבתי.

אני בצל שדי אחסה, / לא אשאל ולא אנסה / לדעה מה יעשה!

יחידתי חפשה / מכל חמדה חדשה / ומעבדה קשה;

ויום גוף מנשמה יחסר – / לתולעת יהי נמסר / ואכלו את הבשר.

אשרי דורש תבונה, / בפיהו דבר נכונה / ויהי ידיו אמונה.

בוערים כסילות יפרשון – / ומחברים יכסון / המעשה אשר יעשון.

מה אנוש ומחשבו? / כי כל סתר לבבו / עד האלהים יבא.

מי רוח אלהים בו / ירים תרומה בחבו, / אשר ידבנו לבו;

תרומתו נשמרת / לנפשו הנשמרת / לכבוד ולתפארת,

בהגיעה לגבהים / עם כוכבי נגוהים, / כתבים באצבע אלהים.

היה לכל עול מלאכתו – / מי יענה, בעת בשתו, / איש אשר נמצא אתו?

לא יוסיף ברשעתו / לכבות נר נשמתו / עד יום העלתו!

יעזב דרך נטה / ויתבונן כי שטה – / והתודה אשר חטא.

ורע להקשיב לא יאבה, / את תאותו מרבה, / תוקד בו, לא תכבה –

פעלו לשרץ נדמה / ולבו מלא מרמה, / כל אשר בתוכו יטמא!

כל יגיעיו שוא ותהו / וחליו אין כמהו – / נגע צרעת הוא!

ייגע איש בכל מאדו / והכל יבקש בעדו, / אשר לא תשיג ידו;

ואת אשר עליו ישפר, / שחת, קברו חפר / וכסהו בעפר.

ואיך ביצרו יתגולל / ובכבודו יתהולל? / קדש יי חלל!

משחיתים דרכיהם – / ימאסם אלהיהם, / כי משחתם בהם;

ומשכיל סר בחכמתו, / כי הוא בחייו ובמותו / ביד הקנה אתו.

אשר בוטחים בקונם / להסיר מחשבות אונם – / והתודו את עונם!

ובוגדים בגדו בגד – / והוא יושב מנגד, / וכסו אותו בבגד;

ונפשם הנשאה / חמדת תבל מלאה, / ונסתרה והיא נטמאה.

אשיב, בטרם מותי, / לטהר את טמאתי – / ואל אראה ברעתי!

מה לאיש נפשו רזה, / וסוד האמת לא חזה, / כי דבר יי בזה?

ספו מתים בהבליהם, / ונוסרו משכיליהם – / ולא ימותו גם שניהם.

איך תמעד אשורנו – / כי המות יקרנו, / אנחנו ובעירנו?

איש יעות מסלולו / ויצרו מסית שכלו – / בדרך לשטן לו.

שובבים, מה לכם / רדוף אחרי עיניכם? / כי צררים הם לכם!

תכין נפשם פעלה / להנקות ממעלה – / ויי יסלח לה.

בינו, יהירים, בסודכם, / כי לכבוד אל בראכם – / והשתחויתם לכם!

להושיבכם מקום תהלות / סבב בעדכם מעלות – / ולא אביתם לעלת.

בטרם יכלו שני / אודה ואעזב זדוני, / פן יחרה אף יי.

יודע ראשיתו / יבין, כי באחריתו / יי הוא נחלתו.

לנצח הוא במעלה – / ובאדמה השפלה / אין לו חלק ונחלה.

יעירהו מחשבו / סוד מאין יבוא, / לבלתי רום לבבו.

פתי, ייסר אותו / אל – ירוץ לרצותו, / פן יסיף להכתו.

ימהר, עד יאבד, / ויכבד בכל מעבד / את השם הנכבד;

מה לרוחו ולעצבה, / בסוד חלאים בקרבה / אשר חלה יי בה?

עוני טמון בחבי – / ואיך יעלה על לבי, / כי אין אלהי בקרבי?

סוד חיים ושבילו / וסוד מות הראה לו / הצור תמים פעלו.

נהלל יה כלנו / על חסד גמלנו – / תורה צוה לנו.
