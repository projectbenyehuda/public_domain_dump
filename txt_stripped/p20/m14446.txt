


	
	אמרתך אלי הקם לשבויה
	




אמרתך, אלי, / הקם לשבויה / כלבבי:



‘הנני שולח / לכם אליה / הנביא!’



בת ציון איכה / בינות אכזרים / נטשה?

או איך רעיתך / מחדר הורים / גרשה?

ונוה קדשך / איך לבני זרים / מורשה?

ירשו את אהלי / מקדם היה / מושבי —

ואני בורח / ארץ נכריה / בין אויבי.



ראשי הרימה, / כי לך מאהבה / משתאה,

וראה בן אמה / על בן האהובה / מתגאה.

ימה וקדמה / נשבר או נשבה, / אין רואה.

הכבידו עלי / בוזי תושיה, / ירדו בי —

ואני צורח / נפשי בוכיה / אל אבי.



'מרום שמעתי / קולך, בת עמי, / ואענה:

ישעי קרבתי / לפדותך לשמי, / ואקנה

את בית תפארתי / הרס איש חרמי / אין בונה.

למקום היכלי / מקצוי נשיה / תשובי,

ובלב שמח / הר המוריה / תקרבי!'
