


	
	צורי בשורך אוחיל
	



מאורה




צורי, בשורך / אוחיל עצור / על צר וצור,



אשבע באורך; / די לי שכור / כוס ממרור!



אשב מיחל / עת יעלה /אור ישעך,



מרעיד וזוחל / על אהלי / מטעך,



נאלם כרחל – /ואחלה / מנסעך.



עד מה חזונך / לחתם וצור? / נעלם חקר!



יפיק רצונך / עצור בבור, / יקוט בדור.



בינות צלעי / יקד יקוד / אל גואלי,



על רב פשעי / הרחיק נדוד / מאהלי.



אחי ורעי, / שאלו לדוד / על מחלי,



אם יש ארוכה / לחבש מזור, / וגהות מזור־



תקוה משוכה / אל קץ ותור / לקרא דרור.



רום לב מעני / השפל ושח / קולם כאוב.



רבי עוני / מלב זנח, / אל תחשב.



אורך לעיני / עתה שלח, / מתוק וטוב.



כיום לבנך / חסד גמור, / הורם זכור,



עליו ימינך / שים לעזר, / זה הבכר!



הקשב בזעקי / את עזרך, / וצלח רכב,



צורי וחשקי, / פן צרך / נרי יכב,



וכאז בחיקי / תוך הדרך, / דוד, שוב שכב.



על עם רחקך / הרק, סגור! / קנאה חגור



אל עם דבקך, / יגור בבור / בין דב וגור.



מאור צפנתו / אל יודעי / שמך – חלק



אל בן בחנתו / בין פורעי / מוסר וחק,



שוב וחננתו, / קולו שעה, / עלו פרק.



אגיל בצאתך / לקב, ואר / צרי, וצור



כפך לביתך / אוסיף עבור / לאור באור.
