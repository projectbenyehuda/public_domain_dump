


	
	אשחר בשירה
	



סליחה (?)




אשחר בשירה / לנורא עלילה



בפחד ומורא / ונפש שפלה



בעד שה פזורה / סורה וגולה;



שוכן מעוני, / כקרבן רצני!



בקראי ענני, / אלהי צדקי.



במה אקדם / פני מושיעי?



להלבין אדם / חטאי ופשעי



אמאס רדם / בליל על יצועי;



ברוח נמוכה / דמעי שפוכה



וחון, ארחמך / יי חזקי.



ביד בן רובה / המוני מענה



וזולל וסובא / עלי ממנה



ואין משובה / לנפשי קונה



ונין בא נבה / דוחק ומונה –



ידו הדה / על בת יהודה



ושם לחרדה / נשף חשקי.



׳יונת אלם / בהר נודדת



לציד ואלם / ובי מורדת,



ו[..] בצלם / כשה אובדת.



פניהם אכלם, / וירדו ברד[ת]



יע[רו] אל בור, / וידם אשבר,



ועמי – אשבר / עליהם חקי.׳



תורה תמימה, / חלי פני אל



בעד עם נדמה / כאביון שואל,



ברעב וצמא / ידם אין לאל



ומיד דומה / אין לו גואל.



יען, עושי, / תרים נסי –



אתה כוסי / ומנת חלקי.



דין יתומים / וגם אלמנות



ינחם עגומים / באו[ן] ויגונות



בצדקת חכמים / חוקרי תבונות;



כשה ושלמים / יקבל תחנות



מבן, בצוקה / יגדיל צעקה



בפיו, יען ק־/רב מצדיקי.



יה, ריב היום / ריבי נפשי



והמציא פדיום לי מנ[ושי]



ואותי היום / הוציא חפשי;



ומדת קוני / השביעני



והטריפני / לחם חקי.
