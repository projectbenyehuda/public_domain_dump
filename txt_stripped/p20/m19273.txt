


	
	מי כמוך אפס יש מבלעדיך
	



מי כמוך






מי כמוך? / אפס יש מבלעדיך; / אודך במעשי ידיך –



ארוממך, אלוהי המלך.



מי כמוך? / במתחך אהל מישרים, / הואלת לפתח
ספרים –



ויהיו נקראים לפני המלך.



מי כמוך? / רחמיך ושפטיך נצבו / וברואיך כצאן
נחשבו,



עוברים לפני המלך.



מי כמוך? היודע צפון חבם, / אנה יסתרו, ולבם –



הוא מספר למלך?



מי כמוך? / מקשיב לקול משוע, / עני אני וגוע,



מפיל אני תחנתי לפני המלך.



אחד לבדו נעלה / כסא הכבוד הנפלא,



אבני קדש ממלא – / אפריון עשה לו המלך.



בנה מוסדי עולם / ובפתוח קדש נעלם,



באותיות שמו כללם / ונחתם בטבעת המלך.



גזר אמר להיות / בגלות עליות ותחתיות



אור מתרוצץ, לחיות / באור פני המלך.



דבר שלח והרקיע, ומקוה / ימים כונן ויצוה,



ומוצק העמיד בשוה – / הוא עמק המלך.



הדשיאה בכחו אז חרבה / ומלאו פני תבל תנובה –



ועוד אורח לא בא / אל שלחן המלך.



ויתרוצצו סביב עליונים, / שבעה פנימים חיצונים –



המה השרים הראשונים / המשרתים את פני המלך.



זמירות רצים לקוניהם, / בלי יסבו פניהם,



והשמש הולך לפניהם – / השליש אשר למלך.



חקות הליכתה נודעה: / בבקר יצאה ונסעה,



ולערב עת נגעה / ותבא ותשתחו למלך.



טובות אוצרותיו בידו, / יעל אור ויורידו,



ויתנהו לירח פקידו – / והוא יתן מעדני מלך.



ימלא ויחסר לזמן, / ופעם יראה – ונטמן,



בשחק לעד נאמן, / אמר: אני – מעשי למלך.



כמלאת חיות שמים / שרצו חיות במים,



לעשות לתושיה כפלים, / כי כן יסד המלך.



להוציא מארץ בהמות / קרא, וירא בחכמות,



כי הנברא באדמות / רב כיד המלך.



מדד רוחות עולמו / ויברא אדם בצלמו,



על כל היקום הרימו, / ויאמרו: יחי המלך.



נוצר הכל בעתו / ויעמידהו על מתכנתו,



ויכל כל מלאכתו / ביום השביעי, כטוב לב המלך.



סוד באדם נצר, / בגללו בכל עצר,



אכן הושיבו בחצר / גנת ביתן המלך.



עצה הגדיל והפליאה / ומצלעו רעיה הוציאה,



עדיו ככלה הביאה – / כבודה בת מלך.



׳פרה ורבה בחדלך! / ראה, העשבים לאכלך



והחיות לעבדים לך – / ואתה הוה להם למלך!׳



צום למנע ידיהם / מעץ הדעת – והם



האמינו בנחש, ושניהם / שמעו בצות המלך.



קל מהרה יראו / מקול מתהלך – ונחבאו,



ומגן עדן יצאו / מבהלים ודחופים בדבר המלך.



רעות חשב הפכפך / וידו כאכזר הפך,



ודם אחיו שפך – / והוא צעק אל המלך.



׳שרש פורה שרשת / ונחל נובע יבשת;



הרצחת וגם ירשת? / וכל דבר לא יכחד מן
המלך!׳



תמים דעות הורה / אותו בדרך ישרה



הודה חטאו, וקרא / כי קרוב המלך.



ארץ טהורה החניפו / ונביאים להם הטיפו,



אז למרות הוסיפו / בהשמע דבר המלך.



בראותו הדור לא שב / משרירות לבו חשב –



יי למבול ישב, / וישב יי מלך.



גאל נח במועצה, / כי חן בעיניו מצא,



ומתבה לימים יצא / כשך חמת המלך.



דבר בעתים באות: / ׳לא אשחית עוד הבריאות!׳



וקשת בענן לאות / ביום ההוא נתן המלך.



התחברו עליו, בהתיעצם / לבנות מגדל כחפצם,



לארבע רוחות להפיצם / הדבר יצא מפי המלך.



ואב הטיב במעלליו, / הכבוד חופף עליו,



ומלאך נראה אליו: / ׳הנה חפץ בך המלך!׳



זנח אלילי הורו / ויגדף מלך דורו –



נשמע בגוים דברו / וידברו הכשדים למלך:



׳חקותיך לא שמר / ולעבד אלהיו אמר,



בכבוד שמו יתימר – / וזה אשר הרים יד במלך.׳



׳טעה לבך ויתגבר – / למי באש תשבר?



מדוע אתה עובר / את מצות המלך?׳



ידעתי, ועיני צופיות / הליכות רב עליליות,



ומחר נפשות וגויות / יבאו למשפט אל המלך.



כשמו כן תהלתו, / משלם לכל כפעלתו,



ונפש תעבד זולתו – / לא תבוא עוד אל המלך.



לאור כשדים הורידוהו, / וכאשר נצל – הודוהו:



׳אלהיך אמת, נעבדהו!׳ / וגם ככה אמר המלך:



׳מעבודה נכריה צאו, / אך האלהים תיראו,



ובצאת נפשכם – תבואו / בית המלכות אשר למלך!׳



נסהו בבנו יחידו / ובהר המוריה עקדו,



ולמאכלת שלח ידו, / כי כן צוה לו המלך.



סערו בשמים תורות, / וסביבות הכס מדות



הדין והרחמים עומדות, / ותדברנה לפני המלך.



עליונים היו לאגדה / בעמד מליץ חמדה,



וטלה באיל נפדה / על יד איש משרי המלך.



׳פדה בני בשועם, / בהר הנבחר בפגעם,



והיום הזה לזרעם / יזכר נא המלך!׳



צמחו תאומים ותמים / בשנים עשר שלמים



לבש מלבוש מרומים, / אשר לבש בו המלך.



קדוש הפליא מעשים – / ובעל כתנת הפסים



נמכר לעבד, ותחפנסים אתו משחו למלך.



רעב קרא בהגיע / קץ, לאב המון הביע,



ובן לאביו הודיע / כי הכינו יי למלך.



שבה רוחו בהתבשרו / ומצרימה ירדו בעבורו –



וימת יוסף ודורו / ויקם מלך.



׳תראו יועצים ותחשו, / טרם ארצנו ירשו –



ועתה יקשרו ויבקשו / לשלח יד במלך.



אשר יולידו – נשמידם, / אולי מגוי נכחידם,



והחיים – בפרך נעבידם!׳ / ויחזק דבר המלך.



בנים צעקו מעבודה – / ורגשת מרום נועדה,



וברית הורים עמדה / ותדבר לפני המלך.



׳רועה, שלחתיך ממרומי / להוציא ממצרים עמי,



התחזק באותות שמי / ובאת אל המלך!׳



מתניו שנס עניו / והודיע לאחיו עניניו,



וינאמו לתנין וקציניו: / ׳כה אמר המלך:



בני בארצך בעבדות – / שלחתי אליהם פדות!׳



ויען בן המרדות: / ׳מי הוא זה מלך?׳



׳רבה תפלצת בקרבך / והשיאך זדון לבך,



וגם אין תבונה בך – / מי אתה קראת אל המלך?



מי כמהו מורה / ויש מאין בורא,



ודורות מראש קורא? / יי צבאות הוא מלך!׳



אמרו למשה מכים: / ׳בעינינו שמתם לשכים;



ולמי אנו מחכים – / ושמע אין לך מאת המלך!׳



ישוב אליו חוזה: / ׳דבריך תנין בוזה,



למה יקלל הכלב המת הזה / את אדני המלך?׳



׳ראה תראה בסוף, / מי אפזר ואאסף;



ימין מחוללת אחשף – / ביום ההוא יאבד לב המלך!׳



׳עריץ, שלח בני, / פן תספה בחרוני;



מאסת את דבר יי – / וימאסך ממלך!׳



זו מטות רשעים / במטה אהרן נבלעים,



יכם בדם, וצפרדעים / תבאינה בהיכל מלך.



׳ראה, אשלח המוניך – / והסר מעלי חרונך!׳



ויאמר: ׳אצא מפניך / ומחר אעשה כדבר המלך.׳



הקשה לבו חוטא – / ורוח יי תבטא:



׳הך העפר במטה / ובבקר אמר למלך.׳



קמטם בערוב ודבר / ושחין אבעבעות ושבר,



ארבה מביש משבר / להנקם באיבי המלך.



טבעו בחשך זידונים – / ועוד העזו פנים



עד מות ראשית אונים; / בלילה ההוא נדדה שנת
המלך.



נקהלו ויחד במר צרחו / ׳צאנכם ובקרכם קחו –



וקומו צאו וזבחו / כטוב בעיניכם, בשם המלך.׳



חמושים עלו ממצריהם – / והם מקברים בכוריהם,



ומרכבות ושלישים אחריהם / יצאו דחופים בדבר
המלך.



זרוע רמה התעוררה / ובים עבדים נערה,



וביבשה עברה העברה / לעביר את בית המלך.



קול נשאו כלם: / ׳אשירה ליי במהללם,



׳יי ימלך לעלם / והיה יי למלך!׳
