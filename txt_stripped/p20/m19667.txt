


	
	ישע בשרו אם גולה
	




פיוט לחנוכה





ישע בשרו אם גולה / כי בא יום גאולים,



ויהי לה לשם ותהלה / אל שוכן זבולים.



אהבת אב המונים העיר / היום לעדתו –



וישפך עלי איש שעיר / כאש את חמתו



ויאביד שרידיו מעיר / וישים נותו;



וימציא צרי ותעלה / לעניים אמלים,



החוכים פדות וגאלה / ולישע מיחלים.



באו להגות רב טובה / עשה מלפנים



ותנו לו תהלה ערבה / וחמודי רננים,



כי השמיד בחמה רבה / מלכות היונים –



יום באו להשים תהלה / בילדים אצילים



ולתתן מעי מפלה / מקדש עם גבולים.



רם לב צר בבואו לפנים / ויגדל גאונו,



ולכלות בני איתנים / השיאו זדונו,



אולם שב בבשת פנים / ויכלה המונו,



מתנם אחזה חלחלה – / כל עובדי אלילים –



ולדך נתנה הצלה / וגבורה לכושלים.



הכהנים בבואם ראו / פך אחת חתומה,



שמן בה להדליק מצאו / יום אחד, והמה



הדליקו שמונה מלאו / בברכה עצומה.



אז הודו לשוכן מעלה, / כי מעשיו גדולים,



כי רפאות למחלה העלה / וצרי לעמלים.



משכילים שמונת ימים / האלה לדורים



קבעו, להיותם שומים / יום שמחה ושירים,



גם נרות יהוא מוקמים / על פתחי שערים:



זכרון לנס הנפלא; / לכן על מחולים



שירו, ועשו יום צהלה / ותנו מהללים.



׳בת ציון, בכל לב עלזי / ושתי כוס ישועות!



הנה לך אגלה רזי / ואקים שבועות.



נשבעתי עלי יד חוזי / לאסף מתנועות



העם, שלקחתיו נחלה / לי, ובכל שבילים



אקרא: סקלו המסלה / חיש לעבר גאולים!׳
