


	
	[את תפארתי טוב לך תתי]
	



רשות לקדיש




את תפארתי / טוב לך תתי; / קומי, בתי, / ולכי אתי / והתגדלתי / והתקדשתי.

בוגד יען / דורש לי אין; / רם – ואען: / בדרכך סואן, / להלך מאן / אדוני לתתי.

ראש מתאנה / דת אח ישנה; / מתי מענה / אמן יענה: / אברח מפני / שרי גברתי?

מתי תעיר / לפדות צעיר? / ושית צר מבעיר / אובד מעיר, / ושעיר כשעיר / ביד

איש עתי!
