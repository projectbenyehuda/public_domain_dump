


	
	מן האדה האיסלנדית
	



מן האדה האיסלנדית / משוררים אלמונים /תרגום שאול טשרניחובסקי



הקדמה

האדה (אמא דסבתא) אינה אלא אוסף שירים של משוררים אלמונים מן המאה התשיעית ועד המאה השתים-עשרה למניין הנוצרים והיא נכללת בספרות האיסלנדית הואיל והשירים נקבצו באיסלנדיה. שפתם של כתבי-היד – איסלנדית – אמנם דומה עד היום ברובה לשפה הנורבגית הקדומה, אולם רוב השירים חוברו בנורבגיה.

איש איסלנדי אחד, עלום-שם, כינס את השירים האלה בתחילת המאה ה-13 לאוסף אחד; סבורים כי הוא-הוא שצירף גם את המבואות שבפרוזה, את דברי הנעילה ואת המאמרים המוסגרים, שאנו מוצאים ברוב השירים. כתב-היד העתיק ביותר הוא ה- Codex regius מסוף המאה ה-13 והוא שמור בספרייה אשר בקופנהגן.

מתוך שירי האדה אנו לומדים לדעת את המיתולוגיה הגרמנית העתיקה ואת אגדות הגיבורים שלה; אין צורך להרבות מלים עד כמה גדול ערכה של זו.

השירים כתובים במשקל קבוע שאת חוקיו גילה בחקירותיו החכם המפורסם בתחום זה – אדוארד סיברס, והרי סיכום דבריו בקווים כלליים:

בדרך כלל יש להבחין בין שלושה משקלים. בשיר הסיפורי נהגו להשתמש בשני משקלים, ה- Fornydislag וה- Malahattr. השיר בנוי בתים-בתים, בני ארבעה חרוזים (שורות); מספר השורות הוא לפעמים יותר גדול; החרוז השירי משתמש באליטרציה (דמיון ראשי התיבות), זאת אומרת הישנותה של הברה אחת בעלת נגינה בתחומו של חרוז אחד או זוג חרוזים.

ארבע השורות – ארוכות, והחתך ( Cäsura ) מפריד אותן לשתי שורות קצרות בעלות שתי הברות בנות-נגינה – “הגבהות”.

מספר ההברות קבוע: במשקל הראשון – ארבע, בשני – חמש. מעין דוגמה ישמש לנו החרוז (בגרמנית): Met trinkt Mimir / am Morgen taeglich שבו ההגבהה ניכרת יפה באות M הגדולה.

בעברית: מי -דבש יגמא / מי מיר לבקרים.



נבואת הרואה


האזן האזינו / דורות הקדש,



בניו של הימדל / גדולים וקטנים;



ולפטר כי רצה, / אני אספרה



תולדות ימי-קדם, / זכרונות ראשונים.


  


שמורים בזכרוני – / אבות-אבות ענקים,



שהם ילדוני / לפני עדנים;



תשעה עולמות ידעתי / תשעה דורי עץ-העולם



המעמיק שרשיו / בבטן האדמה.


  


אז בימות-עולם / ואימיר עודו חי,



ואין חצץ ואין ים / גם לא משבר קר;



לא היתה ארץ / ולא רקיע,



אך תהום פוערת פה, / ואין זכר לדשא.


  


והנה בצבצו ועלו / בני בור מן הבר



והם יצרו / את מידגרד היפה,



מדרום האיר / השמש את הארץ,



ויעל בקרקע / עשב מוריק.


  


ושמש דרום / נלוה אל ירח,



נגע בימינהו / בקצה השמים;



לא ידע השמש / איפה זבול לו,



לא ידע ירח / מה כחו ואונו.



לא ידעו הכוכבים / מקומם איו.


  


אז הלכו לעצרה / האלים כלמו,



מושלי הקדש, / ושם נועצו:



קראו שם ללילה, / לירח ולכסא,



בקר וערב, / צהרים וערבית,



כל המועדים / לספירת השנים.


  


לאידפלד באו / האזים נקבצו



לעשות מזבחות / ולבנות היכלות;



יסדו כורים / לחשל הזהב,



נגדו צבתות / וכל-כלי עבודה.


  


בחצר השתעשעו, / עליזים במשחק-הקרש,



כי לא חסרו / זהב פרוים –



עד שבאו שלש / הנערות האדירות,



בנות הענקים / מטורסנהים.


 


[אז הלכו לעצרה / האלים כלמו,



מושלי הקדש, / ושם נועצו:



מי ומי יצור / המון הננסים



מדמיו של ברימיר / מאיבריו של בלאין.


 


ויהי מוטסוגניר / האדיר ביניהם,



בכל הננסים, / ודורין השני.



ויעשו הננסים / כמה צורות בני-אדם



באדמה / כמצות דורין.


 


ניאי ונידי, / נורדרי וסודרי,



אוסטרי ווסטרי / אלטיוף, דולין,



נר ונאין, / ניפינג, דאין,



ביפור, בפור, / בומבור, נורי,



אן ואונר, / איי, מיודביטניר.





ויג וגנדלף / וינדולף, טורין,



טרור וטראין / טק, ליט וויט,



ניר ורגין / נירד ורדסביד –



נכונה מניתי / את הננסים.


  


פילי וקילי, / פונדין, נלי,



הפטיפילי, / הנר, סויאור,



פרר, הורנבורי, / פרג ולוני,



אאורונג, יארי, / איקינשלדי.


  


עת לכנות בשיר / את-האדם בשמו,



בני עם הננסים / אבי אבות לופר;



גזעם עלה / מארץ הטרשים



דרך בקעות-טיט / אל שדמת-החול.


  


ואלה שמותם: דראופניר / ודולגטרסיר



הר, האוגספורי, / הלונג, גלואין,



דורי, אורי / דוף, אנדורי,



שירפיר, וירפיר, / שפיד, איי,


  


אלף ואינגוי, / איקינשלדי,



פיאלר ופרוסטי / גנר ופיד:



עד בלתי ירח / ידעו ידע



שושלת ארכה זאת / של אבות-לופר.]


  


ושלשה אזים / נאדרים ויפים



בני הזרע ההוא / עד שפת-הים באו;



בשדה הנרחב / מצאו את אסק ואת אמבלה



תשושי-כח / וגורלם רופף.


  


לא היה בהם / לא רוח לא נפש,



לא חם לא תנועה, / לא צבעים פורחים;



רוח נתן אודין, / הוניר – הנפש,



לודור – החם / וצבעים מאירים.


 


אני יודעת מילה / איגדרזיל יכנוה,



מהל לבן / מרטיב את עץ-האדירים;



משם בא הטל / המרטיב עמקים;



מוריק הוא תמיד / על מבוע האורד


 


על יד גזע העץ / עומד אולם,



שלש בנות-חכמה / גרות בתוכו:



[האחת שמה אורד, / השניה ורדנדי –



הן חרטו ברעפי-עץ – / והשלישית היא שולד;]



הן פה חרצו / גורל החיים



לבני-אדם, / גורל כל-גבר.


  


ראשונה למלחמות-עולמות / אני ידעתי



עת דקרו / את גולויג ברמחים



וישרפו באש / באולמו של הוי,



שלש פעמים שרפו / את-הילודה שלש



[כמה וכמה פעמים / והיא חיה עודה].


  


היד קראו לה / כשנכנסה ביתה



אותה קוסמת / לב-חכמה לה וחזון;



קסמים בידיה / מטריפים הדעת,



נשים קלות-דעת / חשקו בה תמיד.





הלכו לעצרה / האלים כלהם,



מושלי הקדש, / נוסדו נועצו:



הישאו מס / האזים גם הם



או כל האלים / יהנו מן הקרבנות.


  


הטיל אודין / העץ בחיל –



גם זה ארע / בקרב הראשון –



אז נשברה הסוללה / של בירנית האזים



הונים אנשי-המלחמה / בוססו השדה.


  


הלכו לעצרה / האלים כלהם,



מושלי הקדש / נוסדו נועצו:



מי הוא אשר מלא / כל האויר רעל



ויתן לזרע הענקים / כלתו של אוד?


  


אך טור הוא הסכים / באמץ לבו –



לעתים רחוקות / ישב כי ישמע זאת –



אז נתרופפו האלות / הדברים והשבועות,



חוזה-המבטחים / אשר כרתו קדם.


 


אני יודעת מחבואה / של קרן הימדל



תחת אילן-הקדש / השותה אויר-שמים;



רואה אני הזרמים / הרטבים נופלים עליו



מערבונו של אודין, / – התבינו שאר-דברי?


 


בדד ישבתי / כשבא אלוף האזים



הבא-בימים / והביט בעיני:



"מה תאוה לדעת? / מה תבחנני?



אודין, אני יודעת, / איפה הצפנת עינך!"


  


ידעתי כי צפונה / עינו של אודין



במבועו של מימיר / המהלל בכל;



מי-דבש יגמא / מימיר לבקרים



מערבונו של אודין, / – התבינו שאר דברי?


  


אבי-צבאות חנני / ענק וטבעות,



כי לבי לב-שכל / ובי רוח-חוזה.



לי הרבה חכמה / אוכל להרחיק שור,



עיני חלפה-עברה / כל העולמות.


  


סביב סביב ראיתי / הולקירות בבואן,



מוכנות לרכוב / במערכות הגבורים;



שולד נשקה מגן, / שוגול אחריה,



גוד, הילד, גונדול / וגירשוגול.



הנה גליתי לכם / את נערות הרין,



נכונות ועומדות / לרדת ארצה.


  


ראה ראיתי / נגזרה גזרה



על בנו-אודין / על בלדר האציל;



רם על קרקעו / עלה מתמר



הדבקון הזקוף / שצמח יפה.


  


אותו הענף / הדק למראה



היה לחץ-צער: / ירה ירה הוד;



אך תכף הולד / אחיו של בלדר,



בן-לילה נלחם / בנו של אודין.


  


לא סרק ראשו, / לא נטל ידיו עד



שכב על גל-יקוד / שונא-בלדר;



אך פריג בכתה / בפנסליר



אסון ולהלה. / – התבינו שאר דברי?


  


את לוקי הרע, / את דמות המארה,



ראיתי עקוד / בחרש שביר,



שם יושבת סיגין / שקועה בצער



על מכאובי אישה / – התבינו שאר דברי?


  


ממזרח שופך מימיו / הסלידר המקציף



עם חרבות ופגיונות / דרך עמקי-תרעלה.

בשדמות נידה / התנוסס בצפון



אולם בנוי פז / לצאצאי סינדרי;



על הר אוקולניר / עמד עוד אחד –



ברימיר, בית-השכר / לענקים.


  


ראיתי אולם / רחוק משמש



בנסטרנד, ושעריו / קרועים צפונה;



בעד חור-עשנו פרץ / גשם של רעל,



כי כתלי האולם / חתולים בנחשים.


  


שם ראיתי גברים / נשבעים לשקר ומכי-נפש



בחצותם / זרמי-פרא



[וגם אלה / שפתו אשת-איש];



שם ינק נידהוג / מגויות מתים,



הזאב טרף בני-אדם, / – התבינו שאר דברי?


  


מקדם ישבה הזקנה / ביער-הברזל,



צאצאי פנריר / פה ילדה;



עוד יבוא יום / ואחד מאלה



יחנק השמש / בדמות הזאב.


  


מאכלו בשר / גברים נפלו,



בדמים יגאל / מושב האלים;



יקדר אור-שמש, / הקיצים יבאו,



ימיהם עכורים, / – התבינו שאר דברי?


  


שם על הגבעה / ישב שומר-הענקית,



אגטר העלז, / פורט על נבל;



למראשותיו קרא / ביער-הצפרים



שכוי אדם יפה, / המכנה פיאלר.


  


ולאלים קרא / גולינקמבי,



מעורר-הגבורים / בהיכל אל-האלים;



ועוד אחר קורא / במעמקי הארץ,



בחלל-ההל / ונוצותיו אמצות.


 


גרם צועק בקול / לפני גניפהליר:



קורע כבליו, / בא-רץ הזאב!



לי הרבה חכמה, / אוכל להרחיק שור



גורל הזעם / הקרב לאלים.


  


מדנים בין אחים / ומפילים זה את זה,



קשרי דמים מנתקים / בני-האחיות;



רע ומר בעולם, / רבה התועבה –



ימות-גרזן, ימות חרב, / מגנים מתבקעים,



ימות-סער, ימות-זאב, / עד תבל שוקעת,



לא יחמול, לא יחוס / אדם על משנהו.





בני-מים נחפזים, / קולה הצלול



של קרן-גיאלר / מכריז על הסוף;



הימדל תוקע בקול, / השופר באויר,



אודין מדבר / עם ראשו של מים.


  


איגדרזיל מרעיד / הרמה במילות,



רועש העץ העתיק, / לענק קרא דרור;



אחז פחד / כל-יושבי שאול-תחת,



עד שעלה בשבילו / רע-דמיו של סורט.


  


מה נשמע בין האזים? / מה להם לאלבים?



ריזנהים כלה הומה, / האזים נועצים;



הננסים נאנחים / על שערי האבנים



שלמושלי הר-היער, / – התבינו שאר דברי?


  


גרם צועק בקול / לפני גניפהליר:



קורע את-הכבלים / בא-רץ הזאב!



הרבה חכמה לי, / אוכל להרחיק שור



גורל הזעם / הקרב לאלים.


  


מקדם בא הרים / מגנו בזרועו,



בגלים מתגולל / נחש-העולם



בחמת ענקים; / צרודות נואק העיט השב,



ברטשו פגרים; / ינתק נגלפר.


  


מצפון שטה / אניה בים,



בה אנשי-הל / ולוקי בה נוהג;



אחרי הזאב באים / קהל פראי-אדם,



אתם בקהלם / אחיו של ביליפט.


  


מדרום בא סורט / עם מכחיד-הזמורות,



שמש אלי-קרב / מזהירה מן החרב,



קורסים הרי-האבן, / כושלות הענקיות,



הל בולע בני-אדם, / נבקע הרקיע.


  


צער חדש / בא על הלין,



בצאת ולפטר / לעמוד בפני הזאב,



ובפני סורט / חונק-בלי הלבן;



נפול תפול אז / שמחתה של פריג.


  


אז יבוא וידר / בנו של ולפטר,



הגבור העצום / להלחם בזאב:



יתקע הלהב / בלב בן-הענק,



בעד גרונו, / וינקום את אביו.


  


גם בנה של הלודין / הנהדר יבוא אז;



חובק-הארץ / יפער מפהק



את לעו הרחב / עד כפת השמים –



אך בנו של אודין / יוצא לקראת פריץ-החיות.


 


יכנע לחמתו / גונן-העולם;



הכל אנוסים / לעזוב מולדתם;



בנה של פיורגין / נרתע תשע פסיעות



לפני הנחש, / לא יחת מפני רשע.


  


החמה משחירה, / הארץ שוקעת בים,



נופלים משמים / הכוכבים הצחים;



נתז הקיטור / וחונן-החיים,



הלהב החם / לוקק הרקיע.


  


גרם נובח בקול / לפני גניפהליר,



קורע הכבלים / בא-רץ הזאב!



הרבה חכמה עמדי, / אוכל להרחיק שור



הגורל המר / הקרב לאלים.


  


רואה אני הארץ / בעלותה שוב



מתוך המים, / עוטה ירק רענן;



על פני אשדות-קצף / מרחף הנשר,



שולה דגים / על יד נד הסלע.


 


על שדה-אידפלד / נקבצים האזים,



מדברים בענק / חובק-הארץ,



נזכרים בעלילות / המארעות הגדולים



ובכתב-הרונות / העתיק לאל עליון.


  


שם שוב יגלו / בתוך העשב



לוחות-הזהב / הנפלאים



אשר היו / לאזים לפנים.


––––––– / –––––––




על שדה לא זרוע / יעלו שבלים,



וכלה כל-הרע, / כי בלדר יופיע.



בבית-המנצחים של הרופט, / מעון אלי-הקרב,



יבואו הוד ובלדר, / – התבינו שאר דברי?


 


אז יבור לו הוניר / זמורת-הנבואה


––––––– / –––––––


ובני אחי-טוגי / ישבו על כסאם



בבית-הרוחות הרחב, / – התבינו שאר דברי?


  


אני רואה אולם / על רמות הגמלה



מצפה זהב, / יפה מחמה:



שם יגורו / גדודי גבורים



יהנו מאשר / אשר לא יסוף.


  


ממרום יבוא / השליט הנהדר



הכל-יכול / ליום-הדין הגדול


––––––– / –––––––

––––––– / –––––––




ולמטה יעוף / הדרקון הרע,



העכן הנוצץ / מסלע-הליל:



באברותיו ישא / נידהוג פגרים –



טס על פני שדות / אך שקע ישקע.


  

חלומותיו של בלדר


חשו האזים / כלם אל הטינג,



גם האזות / כלן באו לעצרה:



האלים האדירים / רצו לדעת



למה חלומות רעים / מענים את בלדר.


  


ויקם אודין / היוצר העתיק,



שם את האכף / על גבו של סליפניר;



רכב ירד / למעמקי הניפלהל,



ויצא לקראתו / מן המחלה כלב.


  


היה חזהו / מגאל בדמים,



הלך וצעק / על אבי-הקסם;



המשיך דרכו אודין, / ותרעד הארץ –



עד שהגיע / לבית-הל הרם.


  


נטה אודין / עד שערי מזרח,



ידע כי ימצא שם / גבעת הקוסמת;



שר לערמומית / את קסם הפגר,



עד שקמה / בעל כרחה האשת;



משפתי המתה / מלים נבעו:


  

הרואה


מי הוא הגבר, / אני לא ידעתי,



המכריחני / לצאת בדרכים קשות?



כסני שלג, / הכני גשם,



טפטף עלי הטל, / זה כבר אני מתי.


  

אודין


וגטם זה שמי / אני בנו-ולטם;



ספרי דברי שאול-תחת / (ואני – מה למעל):



בשביל מי הספסלים / עם טבעות מבריקות,



הרצפות הנוצצות / המצפות בפז?


  

הרואה


מועדות לבלדר / פה מי-דבש הוזדו,



מגן מכסה / המשקה המזהרר;



אבדה תקותם / של בני-האזים.



על אפי דברתי, / רצוני ואשתוק.


  

אודין


אל דמי לך, הרואה, / הוסיפי ואמרי לי,



לשואלך עני / עד ידע הכל:



מי ומי ישפוך / דמיו של בלדר,



ומי יקרב קץ / בנו של אודין?


 

הרואה


הוד ידו תביא / החטר הנהדר,



הוא-הוא שישפך / דמיו של בלדר



ויקרב קצו / של בן-אודין;



על אפי דברתי, / רצוני ואשתוק.


  

אודין


אל דמי לך, הרואה, / הוסיפי ואמרי לי,



לשואלך עני / עד ידע הכל:



מי ומי ינקום / דבר התועבה,



יביא על גל-היקוד / את הורגו של בלדר?


  

הרואה


רינד תלד את ולי / באולם המערבי,



בן-לילה ילחום / בנו של אודין;



לא יסרוק ראשו, / ידיו לא יטול



עד ישכב על גל היקוד / שונא בלדר.



על אפי דברתי / רצוני ואשתוק.


  

אודין


אל דמי לך, הרואה, / הוסיפי ואמרי לי,



לשואלך עני / עד ידע הכל:



בנותיו של מי הן / הנמסות בדמעות,



זורקות השמימה / ציציות הצוארים?


  

הרואה


אבל אינך וגטם / כאשר דמיתי קדם,



כי אתה אודין / היוצר העתיק.


  

אודין


אינך חכמת-לב / גם לא נביאה את,



בחיקך כי נשאת / את-שלשת הטורסים.


 

הרואה


שמח בתהלתך / ושובה הביתה!



לא יבקרני / כל-גבר אחר,



עד יפרק לוקי / הכבלים מעליו,



ובאו המהרסים / למגר את האלים!


  

שירת טרים


בער אף וינגטור, / כשנתעורר



ולא מצא את מיולניר, / הפטיש האדיר;



נענע את זקנו, / נפנף שערו,



משש בידיו / בן -האדמה.





היתה מלתו / הראשונה אשר אמר:



"שמע נא, לוקי, / אשר אודיע –



איש לא ישער זאת / לא במלא -ארץ,



לא בשמים ממעל – / נגזל מן האז הפטיש!"


  


הלכו אל חצרה / הנהדר של פריה.



היתה מלתו / הראשונה אשר אמר:



"התלוי לי, פריה / שמלת -נוצותיך



למען אשיבה / פטישי שגזלו?"


  


"בחפץ -לב / לך אשאילנה,



לו גם היתה מכתם / או מכסף מאיר!"



אז טס לוקי, / והשמלה רועשת,



זנח אחוריו / את חצרות האזים,



תכף הגיע / למכורת הענקים.


 


ישב על הגבעה טרים / אלוף הטורסים,



קלע מוסרות -כלבים / מכתם אדם,



הישיר החליק / רעמות -הסוסים.


טרים


מה שלום האזים? / מה שלום האלבים?



מה תבוא לבדך / אל ריזנהים?


 

לוקי


רע לאזים / רע לאלבים;



האתה טמנת / פטיש הלורידי?


 

טרים


אני טמנתי / פטיש הלורידי



שמונה מילין עמק / בחיק האדמה.



איש לא יקבלנו / עד אם הביאו



הנה את פריה / לאשה לי.


 


אז טס לוקי, / השמלה רעשה,



מכורת הענקים / זנח אחריו,



ונחפז עף / אל חצרות האזים.



ועוד בחצר / פגשו טור,



היתה מלתו / הראשונה אשר סח:


 


"ההצלחת בדרך / וכדאית טרחתך?



הגד בחוץ פה / ספורך הארך!



היושב מרבה לשכח / את אשר לו להגיד,



והשוכב / משקר ברצון!"


  

לוקי


"הצלחתי בדרך, / כדאית הטרחה:



טרים אלוף הטורסים / בידיו המקבת;



איש לא יקבלנה / עד אם הביאו



שמה את-פריה / להיות אשה לו."


  


הלכו לחצרה / הנהדר של פריה.



היתה מלתו / הראשונה אשר סח:



"קשטי עצמך פריה, / בצעיף כלולותיך,



עלינו השנים / ללכת לארץ ענקים!"





בחמת קצף / נשמה-נשפה פריה.



בירת האזים / רעד רעדה,



נשבר-נשמט / ענק-הבריזינגים:



"המשגעת לגברים – / כך היו לי קוראים,



לו נסעתי / אתך לארץ ענקים."


  


האזים כלם / נחפזו אל הטינג,



גם האזות באו למועצה כלן;



שם נועצו / האלים המהללים,



איך יחזירו / פטיש-הלורידי.


  


ענה הימדל / הלבן באזים,



יודע הבאות / כאחד הונים:



"נקשט את-טור / בצעיף של כלה,



ישא את ענק / הבריזינגים הרחב.


 


תנו לו הטבעת / עם המפתחות המקשקשים,



תרחשנה שמלות-נשים / על ברכו סביב,



קשטו את-חזו / באבנים רחבות,



עטרו לראשו / עדי-אמנים."


  


ענה-אמר טור / האדיר באזים:



"אשה יכנוני / האלים המושלים,



אם אתקשט / בצעיף של כלה."


 


ענה לוקי / בנה של לופי:



"החרישה, טור, / חדל מלהג!



במהרה ימלכו / הטורסים באזגרד,



אם אינך מחזיר / פטישך שלך."


  


קשטו את טור / בצעיף של כלה,



בענק הרחב / של הבריזינגים.


  


נתנו לו הטבעת / עם המפתחות המקשקשים,



רחשו שמלות-נשים / על ברכו סביב,



קשטו חזהו / באבנים רחבות



ועטרו לראשו / עדי-אמנים.


 


אז אמר לוקי, / בנה של לופי:



"אתחפשה כאמה / ואלך אתך,



עלינו השנים / ללכת לארץ-ענקים."


  


דפקו הביתה / התישים המקרינים,



אסרו ליצול / הרצים הקלים.



ההרים נבקעו, / בערה הארץ,



טור בן-אודין / הלך לארץ-טורסים.


  


אז אמר טרים, / אלוף הטורסים:



"קומו הענקים, / ערכו הספסלים;



הנה מביאים לי / את פריה לאשה,



את-בתו של ניורד / מנואטון.


  


פרות עם קרני פז / הולכות החצרה,



פרים שחורים משחור / לשמחת לב טורס;



לי הרבה סגלות / וקשוטי-חמד,



אך פריה האחת / חסרה לי.


  


ירד הערב / בא בעתו,



הביאו לטורסים / משקה השכר,



אכל טור שור-פר / ושמונה אלתיות



וכל מיני-תבלין / שנועדו לנשים,



שתה בעלה של סיף / עוד שלש חביות מי-דבש.


  


אז אמר טרים / אלוף הטורסים:



"איפה מצאת מעולם / כלות זוללות כזאת?



מימי לא ראיתי כלות / בולעות בלע שכזה,



או נערה מרבה / לשתות מי-דבש."


  


ישבה ליד פריה / האמה השנונה,



ידעה מה להשיב / לדברי הענק:



"מאום לא אכלה פריה / זה שמונה לילות,



כי כמהה מאד / לארץ-ענקים."


  


הרים צעיפה / כי רצה לנשקה,



נבעת ונרתע / לאחורי האולם:



"מדוע מבריקות כה / עיניה של פריה?



אמר אמרתי: אש / מתלקחת בהן."


 


ישבה ליד פריה / האמה השנונה,



ידעה מה שתשיב / לדברי הענק:



זה שמונה לילות / לא סגרה עיניה,



כי כמהה מאד / לארץ-ענקים."


  


נכנסה אחות טורס / הבאה-בימים,



העזה לבקש / מתנת הכלות:



"חלצי מזרועך / צמידיך האדמים,



אם יש נפשך / לקחת לב-הישישה,



לב-הישישה / וכל-חסדה."


  


אז אמר טרים / אלוף הטורסים:



"הביאו המקבת / לקדש הכלה,



שימו את מיולניר / בחיק הנערה



וקדשו בריתנו / בשמה של ור!"


  


צחק בחבו / לב הלורידי,



האיש הקשה / למראה המקבת.



הרג ראשון את-טרים, / את-אלוף הטורסים,



ויפוצץ את-זרע / הענקים כלם.


  


הכה גם את אחות / הטורס הישישה



אשר בקשה / מתנת הכלות;



ספגה מכות / תחת שילינגים,



ומהלמות-פטיש / תחת צמידים.



ככה החזיר לו / בן-אודין המקבת.


  

שירת הימיר


פעם ישבו יחדיו / האלים המנצחים



על מי-דבש וציד, / אך הסעודה דלה;



הם נענעו זמורות, / התבוננו בדם:



מצאו את-אגיר / והנה עמו שפע.


 


ישב שוכני-הסלע / עלז כנער



אשר לו עינים / סמויות עדין;



הביט בעיניו / בן-איג בעז-לב:



“ערך-נא לאזים / כרה בשפע!”





הטריח הגבור / את חובב-המדנים,



שחשב רעה / על האזים;



בקש מבעלה של סיף / להביא הקלחת:



“אז אבשל ברצון / לכלכם השכר.”


  


לא עלה בידי / האלים המהללים,



חכמי-העצות, / להמציא הקלחת,



עד שיעץ טובות / טיר תם-הלבב



להלורידי / ויאמר בסוד:


 


"באליוגר / מזרחה שמש



חי הימיר החכם / על קצה השמים,



יש לו לאבי / האביר קלחת,



כלי רב ועצום, / מיל אחד עמקו."


 


“הנוכל לקחת / את מרתיח-המים?”



“נוכל ידידי, / אם נבוא בערמה.”


  


נחפזו נסעו / אותו יום כלו



מן האזגרד / עד מעון אגיל;



הוא נטר התישים / עטורי הקרנים.



נכנסו שניהם / באולם הימיר.


  


מצא טיר את אם-אביו / אשר שנאה מאד



היו לה ראשיה / מאה פעם תשע;



אך האחרת נוצצה / בזהב, והביאה,



לבנת-הגבות / לבנה את השכר.


  


"קרובם של הענקים, / עזי-הלבב,



את שניכם אושיבה/ תחת הדודים,



כי, לצערי, צרה / עין דודי באורחים



לעתים קרובות, / וסר וזעף הוא מאד."


 


בשעה מאחרת / שב הענק מן הציד,



הימיר המכער / שב נכנס הביתה.



נכנס באולם, / קשקשו לשונות-קרח



על פני הבא, / כי קפא היער.


 


אשריך, הימיר! / ישמח מאד לבך!



היום באולם פה / נכנס בננו,



אשר לו חכינו, / מדרכים רחוקות.



נוסף-נלוה אליו / אויבו של הרודר,



גומל החסדים לאדם, / ויאור זה שמו.





"ראה איך הם יושבים / על גמלון האולם,



מאחורי הקורה / שם יסתתרו."



פקעה הקורה, / כי הביט הענק בה.



נשבר לשבבים / העמוד החזק.


  


שמונה דודים / נדרדרו מן הקרש,



אחד נגוד קשה / נשאר שלם.



קרבו האורחים. / הענק השב



הביט בעז / בעיני שונאו.


 


טובות לא נבא לו / לבו כי זה בא,



המאביל / נשות הטורסים.



להוציא מן הרפת / שלשה פרים,



לשלוק בשרם / צוה הענק.


  


כלם התקצרו / כמלא ראש אחד,



העבירו בשרם / בלהבה.



בעלה של סיף / אכל הוא לבדו



שנים פרים / עד שהרגיע.


  


לא קטנה בעיני / איש-סודו של הרונגניר



לבן-השער / סעודתו של טור:



"אנו השלשה / אנוסים לאכל



בערב הבא / משלל הציד."


 


היה ויאור נכון / לחתור הימה



אלו נתן לו הענק / עז-הלב השרגוג:



"לך אל העדר, / מכניע הטורסים,



אם לבך מלאך / לקחת שרגוג."





"סבורני, בנקל / תקח ידך



פתיון-חמד / מבשר השור."



מהר ובא / טור אל היער,



שם פגע בו / שור-פר שחור.


  


מכניע הענקים / קטם מן הפר



את משגבו / עטור הקרנים.


——-  ——-




"רב-האניה / רואה אני שקידתך



נוראה אף / ממנוחתך בסעודה."


——-  ——-




צוה בעל-התישים / לבן-הקופים



להוסיף ולהסיע / את סוס-הגלילים;



אך הטורס העצל / ענה לו ואמר:



אין לי כל חשק / להוסיף ולשוט.


  


שלה הימיר העז / שנים תנינים



בחכה מן הים / תוך זיזה אחת;



אבל בנו-אודין, / ויאור הערום,



כונן תרן / מתח החבל.


  


גונן-האדם, / מכה-הנחשים,



קבע בחכה / ראשו של השור.



בתאבון חטף / שונא-האלים,



חובק-הארצות / את נתח-החמד.


  


ביד אמיצה / שלה טור עז-הלב



התולעת הארסית / אל שפת הסירה;



ממעל הלם, / הכה במקבת



את הר-השער / של אחי הזאב.


 


המו המפלצות, / נתנו קול ההרים;



רעד רעדה / ארץ-קדומים.


——-  ——-


ושמט הימה / הדג המוזר


  

——-  ——-


הוסיפו וחתרו, / וילן הענק,



ישב על הרצועה, / שתק-החריש.



ויט הסירה. / אל היבשה.


  


"עליך לחלק / אתי שאר העבודה,



הדק בחוף / את תיש-המים,



או שא התנינים / אל מעוננו



דרך הנקיק / בצל היער."


 


אז תפס הלורידי / את קנה החרטום,



הרים הסירה / עם מי-הקרקע,



נשא לבדו / עד חצר הטורסים



משוטים, גיגית-משאב / וחזירי-הדכי.


  


אולם הטורס / העקש רצה



להוסיף התחרות / עם טור בכח,



אם-גם היטיב / חתור במשוטים;



אדיר יכנה / אך מפוצץ-הכוס.





תפס הלורידי / בכפיו הגביע,



נפץ בזכוכית / את עמוד האבנים,



בשבתו בקע / העמוד לשנים;



אפס שלמה שבה / הכוס לידי הימיר.


  


בחכמה יעצה / רעיתו הנחמדה



של הענק / כי ידעה:



"זרק בגלגלת / הענק העצל,



מצחו כי קשה / מכל זכוכית."


 


התרומם למחצה / אדון-התישים,



קם התאזר / בכל כח האזים,



שלם נשאר / גרדם-הקובע,



אך כוס-היין / היפה נשברה.


  


"הנה אבדה לי / סגלת-חמד,



כי הספל נפץ / שמט מחיקי;



לא אוסיף כקדם / לאמר בשמחה:



הנה תססת / משקה שכרי!"


  


" הנה התנאי / האחרון בתנאים:



הוצא את דוד / השכר החוצה!"



פעמים תפס / טיר החבית,



אך זו לא זזה, / נשארה על עמדה.


 


תפס אבי-מודי, / אחז בשפה



וירד לאולם / במעלותיו,



הרים הקלחת / בעלה של סיף על ראשו,



צלצלו אזניה / על עקביו.


  


לא הרבו לנסוע / ובנו של אודין



צפה התבונן / עוד פעם לאחור.



ראה המון עם / מרבי-ראשים



בא עם הימיר / ממערות ההרים.


  


אז הרים מערפו / את-הדוד הגדול,



את מיולניר, / צמא הדמים,



ויך את תניני / השממה, את-כלם.


 


לא הרבו לנסוע, / ויצנח אחד



מתישיו של הלורידי / ארצה, מת למחצה:



התעקמה רגלו הדקה / של סוס-היצול



בעוונו של לוקי / רע הלבב.


  


הלא שמעתם: / מי ומי היטיב



לספר דברים / על סוד-האלים?



מה קבל טור / כתמורה



מידי הטורס – / שני בניו קנס נתן…


  


אל טינג האלים / בא טור האדיר,



בידיו הקלחת / שהיתה להימיר;



עתה יכולים האלים / לסבא כהלכה



באולם אגיר / עד ימות הסתו.


  

מלחמת הדברים של לוקי

  

אגיר (הוא גם גימיר) עשה משתה בירה לאלים אחרי שקיבל לידיו את הדוד הגדול (כמסופר ב“שירת הימיר” – שיר IV ) אל המשתה באו אודין ופריג אשתו. טור לא בא כי נסע אז מזרחה שמש, אבל אשתו סיף באה אל המשתה. היו שם בראגי ואשתו אידון; היה גם טיר גידם ידו האחת, שכירסם הזאב פנדיר, בהיותו כפות. בא ניורד עם אשתו סקאדי, פריר ופריה, בא גם וידר, בנו של אודין, כן בא לוקי והמשרתים של פריר – ביגויר ובילא, ועוד כמה מן האזים והאלבים. אגיר היו לו שני משרתים, פימאפנג ואלדיר. במקום האש האיר לאלים הזהב; השכר עצמו הגיש את עצמו. האורחים שיבחו מאוד את המשרתים הזריזים של אגיר. לוקי לא יכול לשאת זאת והרג את פימאפנג. אז הרימו האלים את מגיניהם ופרצו בקריאות זעם כנגד לוקי וגירשוהו מהבית היערה. אחר-כך ישבו שוב למשתה. לוקי חזר ומצא בחוץ את אלדיר, פתח לוקי ואמר לו:

לוקי


אמר זאת אלדיר, / עד שתפסע



פסיעה אחת קדימה:



מה אומר קהל אלי-הקרב / על כד השכר



בהיכל הטירה פנימה?


  

אלדיר


קהל אלי-הקרב / מהלל כלי-מלחמתם



ורוח גבורתם;



אין בפי האזים והאלבים / הנמצאים פה



מלה בשבחך.


  

לוקי


רצוני להכנס / בהיכל אגיר



לראות קהל השותים;



אביא כעס / וסלידה לאזים



ואטיל מרה בתמד-זהבם.


 

אלדיר


בהכנסך / בהיכל אגיר



לראות קהל השותים,



לנבל בבוז / את האלים הנהדרים,



נגב ינגבוהו בך.


  

לוקי


דע לך אלדיר / כי אם נלחם



בדברי שנאה שנינו;



ידי בתשובתי / תהי על העליונה,



אם תקים שאון רב מדי.




בתוך כך נכנס לוקי בהיכל, וכשראו המסובים מי הנכנס, נשתתקו כולם.

לוקי


צמא נכנס לופט / בהיכל הזה,



מדרך רחוקה בא



לבקש מן האזים, / אולי יתן לו אחד



ממשקה התמד הרך.


  


איך נאלמתם, / האלים הגאים?



אינכם פוצים פה?



תנו מקום / ומושב לי במשתה,



או תצוו ואלך מפה.




בראגי


מקום ומושב / לך במשתה



לעולם לא יתנו האזים;



כי יודעים האזים / שלא כל יצור



יאה לו לבוא בחג.


  

לוקי


הלא תזכור אודין, / כי בימי קדם



ערבנו שנינו דמינו?



נשבעת לא לטעום / לעולם מן השכר,



אם לא יציעו לשנינו גם יחד.


 

אודין


קומה וידר, / יהנה אבי-הזאב



ממושב של אורח;



ולא ינאצנו / לוקי בחרפות



בהיכל-אגיר פנימה.


  

וידר קם ומזג ללוקי, אבל קודם שזה שתה פנה אל האזים ואמר:

  


שלום לאזים! / שלום לאזות!



ולכל האלים הקדושים מאד!



מלבד לאחד / בין האזים אך הוא:



בראגי, היושב שם על הספה.




בראגי


סוס וטבעת / מאוצרותי אני נותן



וקנס בראגי מציע – חרב,



לבל תרעים / מתוך צרות-עין;



סור מחרון אף האלים!




לוקי


לעולם לא תעשר / בסוסים וטבעות



מזה ומזה תוחלתך נכזבה;



אין באזים ובאלבים, / בכלם פה יחד,



נמנע מקרב כמוך,



מפחד מפני יריה!


  

בראגי


אלו הייתי / בחוץ ולא בפנים



היכלו של אגיר,



מהרה נשאתי / ראשך בכפי,



שלומים בעד הכזב.


  

לוקי


עז רוחך בכסא / ובעלילה מפגר



אתה בראגי, הדר הספסל!



צא לקרב-בינים / אם חרה אפך:



אמיץ-הלב אינו מרבה מחשבות.


  

אידון


התעשת, בראגי, / ראה בקשתיך



כי בו בחר אודין לבן-כנפשו,



על כן אל תשעה / אל לוקי בחרפות



בהיכלו של אגיר.


 

לוקי


שתקי, אידון, / שאין בנשים כמותך



לרדוף אחרי גברים!



בזרועותיך הצחות / התרפקת עליו,



אשר דקר חזה אחיך.




אידון


אינני דוברת ללוקי / בדברי חרפות



בהיכל אגיר פנימה;



אני פיסתי את בראגי, / הלום השכר,



ולא יעיר כעס מלחמה.


  

גפיון


איכה תרהיבו פה פנימה / האזים השנים,



להלחם בדברי-איבה?



יצא שם ללוקי – / פה דובר נבלה,



שונא הוא יצורי העולם כלם.


  

לוקי


שתקי את גפיון, / לעולם לא אשכח



מי פתך לתאוה;



העלם הלבן / הוא נתן לך עדי,



ותשלבי סביבו ירכיך.


  

אודין


מטרף אתה, לוקי / וחסר-לב



אם תעיר קצפה של גפיון,



כי היא יודעת / יפה כמותי



את גורל העולם.


  

לוקי


שתק, אתה אודין, / המחלק בלא צדק



אשר קרבות בין לוחמים!



לעתים קרובות / נתת נצחון לרע בקרב,



לאשר לא היית צריך לתת.


  

אודין


אם נתתי מימי / לרע נצחון בקרב,



לאשר לא צריך הייתי לתת:



בפנים האדמה / עשית שמונה סתוים,



חלבת פרות בדמות נערה;



גם בנים יולדו לך שם,



היה טבעך טבע נשים.


  

לוקי


אומרים עליך: / בסמסי כשפת,



מעשי קוסמים בידיך,



מבית אל בית / הלכת כמכשפה,



היה טבעך טבע נשים.


  

פריג


מוטב שלא תספרו / באזני הגברים



את אשר עוללתם,



שני האזים, / בימים מקדם,



אשר היה – לא יזכר!


  

לוקי


שתקי את, פריג, / אשתו של פיורגין



את, זונה, היית תמיד משגעת לגברים:



אשתו של וידריר / נטתה את חסדה



לוילי ולוה – לשנים.


  

פריג


אלו היה לי / בהיכל אגיר פנימה



בן שרוחו כרוחו של בלדר,



לא מצאת מוצא / מפני בני האזים



עד שטעמת טעם דו-קרב.


 

לוקי


רצונך, פריג, / שאוסיף ואספר



על האסון אני המטתי?



הלא זאת אשמתי, / שאינך רואה לעולם



את-בלדר עושה דרכו היכלה.


  

פריה


משגע אתה, לוקי, / שאתה מספר עלילות,



שעשוכם לשם ולשנינה,



יודעת פריג / את-כל הבאות



אם כי אינה מגדת בעצמה.


  

לוקי


שתקי את, פריה, / אני ידעתיך היטב,



אינך נקיה מעון:



כל האזים והאלבים / אשר פה בבית,



את כלם בחפץ-לב האשרת.


 

פריה


לשונך כוזבת, / אבל ימים יבואו, אשורם,



והיא תזמר לך אסון.



יזעמוך האזים, / יזעמוך האזות,



ביגון עוד תשוב ביתה.


  

לוקי


שתקי את, פריה, מכשפה רעה



מכתמת בזדון מאד:



אותך מצאוך האלים / במטתו של אחיך



ושם נפחת נפיחה.


  

ניורד


קטן הנזק / אם אשה יפת-תאר



מוצאת לה ידיד או עוגב;



אך תמוה הדבר / שאז-נקבה,



שילד ילדים, אלינו בא.


  

לוקי


שתוק אתה, ניורד, / כמה פעמים שלחוך



בן-תערובות אל האלים,



סיר-שתן היית, ובנותיו של הימיר



הטילו מים בפיך.


  

ניורד


תמורה קבלתי / ואם הרחיקו לשלחני



אל האלים בן-תערובות;



נער קניתי, / שאין לו שונאים,



ראשון לאזים יכנוהו.




לוקי


אל תגביה גבה / עד מאד, ניורד!



את אשר שמעתי לא אעלים:



מחיק אחותך / נולד לך בן



אשר יצלח כמקוה.


  

טיר


פרי הוא הראשון / בכל הגבורים,



אשר בהיכל האזים;



אינו מקניט לא / אשת-איש אף לא ילדה,



נותן לכפותים דרור.


  

לוקי


שתק, אתה טיר, / מעולם לא הצלחת



להיות מתוך בין גבר לגבר:



את זרוע ימינך / קרע לך פנריר,



ברצון אני זוכר את זאת.


  

טיר


לי אבדה היד / לך – הזאב הנהדר,



צערי וצערך – מחסור.



גם לא טוב לפנריר, / המחכה בשלשלאות,



עד שיגיע ליל-האלים.


  

לוקי


שתק, אתה טיר, / הלא קרה לאשתך



שילדה ילד לי;



לא שלמו לך / בעד החרפה גם פרוטה,



או שוה אמה, עלוב שכמותך!


  

פריר


פנריר שוכב / מול שפך הנהר



עד שיכלו האלים;



גם אתה תבוא במהרה, / אם לא תשתוק,



בכבלים, הוי, נוגד-הרעות.


 

לוקי


בזהב קנית / את בתו של גימיר,



נתת מתנה חרבך,



אך ברכב בני-מוספל / דרך מירקויד,



אתה, המסכן, חסרת כל כלי-זין.


  

ביגויר


אלו אציל אני / כמו אינגונאר-פריר,



ומושב נהדר לי,



הייתי מועך עד-דק / את עורב-הרעה,



ורועץ את כל גרמיו.


  

לוקי


מי הוא הקטנטן / המכשכש בזנבו,



רב לחטף ולבלע?



המתמיד ושוהה / באזניו של פריר,



כשאינו רוטן על רחים.


  

ביגויר


ביגויר זה שמי / בפי האזים והאדם,



זה שבחי שאני זריז,



על זאת גאותי, / כי בניו של הרופט



מסבים פה כלם למשתה.


  

לוקי


שתק, אתה ביגויר, / מעולם לא המצאת



לגברים טרפם במדה!



נחבאת-שכבת / בקש על הרצפה,



כשירדו לוחמים לקרב.


  

הימדל


לוקי, המשקה / לטפש עשך,



מדוע אינך שומע לעצה הטובה?



מדת הפרזה / זה דרכה בגבר,



מתירה ללשון הרצועה.


 

לוקי


שתק, הימדל, / מאז מעולם נגזרה



עליך הגזרה הרעה;



כל הימים עליך / לעמד בקשי ערף



על משמר השמים.


  

סקאדי


שמח אתה, לוקי, / אפס לא תוסיף ותרבה



לכשכש בזנבך,



כי האזים / ירתקוך אל סלע



במעיו של בן קר ככפור.


  

לוקי


דעי לך, אם אל סלע / ירתקוני האזים



במעיו של בן קר ככפור,



אני ראשון ואני אחרון / הייתי בקרב,



בו נפל מת טיאצי.


 

סקאדי


דע לך, אם היית / אתה ראשון ואתה אחרון



בקרב בו מת טיאצי,



מבתי ומחרשותי / תבואך כל-הימים



העצה הקרה.


  

לוקי


נעמו דבריך יותר / בדברך עם בן-לופי



כשהזמנתני אל יצועך;



אם כל חטאינו / יצאו במספר,



אזכרנו גם את זה.


  

לקחה סיף מן התמד בגביע של גביש ותגש לו ותאמר:


שלום לך לוקי / קח הגביע



מלא התמד המשמר,



ואשאר אני לבדי / מכל בני האזים



חפה מכל פשע.


 

קיבל התמד ושתה ממנו.


אך עליך חסתי / כי בישנית וקפדנית היית תמיד



מתרחקת מגברים;



רק אחד ידעתי, / שהפיר ברית נשואיו



גם עם אשתו של טור:



מעשה לוקי הרע הוא.


  

בילא


רעש ירעשו הרים, / סבורתני, המרעים הוא



ההולך מביתו הנה,



הוא ישתיק מהרה / את הנבל המתעב,



הבז לאזים ולבני-הארץ.




לוקי


שתקי, את בילא, / את אשתו של בגויר,



ומכסה כלך עולה;



מעולם לא באה בין האזים / מפלצת רעה ממך,



אמה מטנפת בזבל!


  

נכנס טור ואמר:


שתק, האמלל, / פן יסגר את פיך



מיולניר, הפטיש החזק,



אתיז מצוארך / את סלע-הכתפים



וכביתי אור חייך.


  

לוקי


הנה נכנס / בן האדמה;



בעזות פנים תנשום ותנשוף, טור.



יבוא יום ולא תעז / להתגושש עם הזאב,



שיבלע את אבי-הקרבות.


  

טור


שתק, האמלל, / פן יסגר את פיך



מיולניר, הפטיש החזק,



אזרקך מעלה / מזרחה שמש,



מקום לא תראך נפש.


  

לוקי


אל תספר לעולם / לגברים, כי היית



מימיך במזרח השמש;



באצבע קטנה של כסיה / שם ישב הגבור,



לא האמין בעצמו, כי הוא טור.


  

טור


שתק, האמלל, / פן יסגר את פיך



מיולניר, הפטיש החזק,



ידי תמיתך / בכלי בו הומת הרונגניר



ותרעץ את עצמותיך כלן.




לוקי


אני זאת תקותי / כי עוד ארבה שנים,



ואף אם תרים הפטיש בגערה;



אמרת: קשות רצועות / ילקוטו של סקרימיר



ולא מצאת שבר,



אבדת בזלעפות רעב.


  

טור


שתק, האמלל, / פן יסגר את פיך



מיולניר, הפטיש החזק,



שאול תחתית ישלחך / בכלי בו הומת הרונגניר



עמק לעולם-המתים.


  

לוקי


כאות נפשי שרתי / לפני האזים



ולפני בני-האזים;



רק מפניך / בלבד ארתע אחור,



אני יודע שאתה מיטיב ללחם.


  


הזדת, אגיר / שכר השנה,



אולם שוב לא תכרה כרה;



כל טובך, / אשר פה אתך



יאכל יקוד האש,



אש תצלה את גבך!


  

אחרי הדברים האלה נחבא לוקי ולבש צורת אלתית. באשדות פראנאנגר צדוהו האזים ואסרו אותו במעיו של בנו ולי, אבל את נרפי, בנו השני, הפכו לזאב. סקאדי נטלה נחש ארסי וקבעה אותו על פניו של לוקי, והיה ארסו של זה מטפטף עליו; אבל סיגין אשתו של לוקי ישבה על ידו וספל בידה לקבל את הארס, וכשנמלא הספל היתה מוציאה אותו החוצה. בינתיים היה הארס פוגע בלוקי וזה היה מושך בכבליו וכל הארץ היתה מזדעדעת - מכאן רעש האדמה הפוקד אותנו.



שיר על אלויס

אלויס


עתה אצלי תכס / הכלה ספסלי,



תכף תלך אתי ביתה;



טלטולי השוא / ארכו די ויותר,



איש לא יפריע מנוחתי בבית.


  

טור


מי אתה, הבחור? / מה חורים כה פניך?



השכבת הלילה עם מתים?



איש מן הענקים / לא יגיע עדיך,



לא לך יאה הכלה.




אלויס


אלויס - זה שמי, / מעוני עמקי אדמה,



שם תחת אבנים ביתי;



אדיר חפצי / למצא את נהג-המרכבה,



אל יפר איש מוצא פיו.


  

טור


אני אפירנו, / כי לי האב



יאה למהרנה;



לא הייתי בבית / עת יעדו אותה לך,



אין בלתי אל לתתה.


 

אלויס


מי הוא הענק / התובע לו רשות



לתת העלמה המבריקה?



נוד שכמותך – / נכרי לאדם;



מי נשאך בחיקו, הוי גלם?


  

טור


וינגטור זה שמי / ממרחקים אני בא,



אני - בנו של סידגרני,



ברצותי, לא תקבל / לעולם הנערה,



לעולם לא אסכים לברית.


  

אלויס


תכף אקבל / הסכמתך שלך,



נחג בשמחות החתנה;



אמלל הייתי / בלי העלמה,



אשר לבנה מחפיר השלג.


  

טור


לא אמנע ממך, / עלם חכם,



ידה של הנאה,



אם תודיעני / את אשר ארצה לדעת



מכל העולמות.


  

אלויס


[ינסני וינגטור / אם ברצונו לנסותני,



אם יש חכמה בננס;



אני שוטטתי-עברתי / בתשעת העולמות כלם,



ועיני פקוחות לכל דבר]




טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש:



מה שם האדמה / רחבת-הידים,



לה יכנה בכל העולמות?


  

אלויס


“אדמה” בפי אדם, / “הארץ” בפי האזים,



הונים קוראים לה “הדרך”;



“הירקה-לעולם” – ענקים, / האלבים – “הככר הפורחת”



האלים העליונים – “השדה הלח”.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש:



מה שם השמים, / (שרואים בם הגבורים)



להם יקרא בכל העולמות?


  

אלויס


“שמים” בפי אדם, / “כפת-מרום” בפי האלים,



בפי הונים – “אורגי הרוח”,



“עולם-עליון” בפי הענקים, / בפי האלבים – “הגג היפה”,



בפי הננסים – “אולם חונן-צוף”.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש:



מה שם הירח / שרואים בני-האדם,



מה שמו בכל העולמות?


  

אלויס


“לבנה” בפי אדם, / “האש הרכה” בפי האלים,



בפי הל “האופן המתגלגל”.



בפי הענקים – “הרץ”, / “אור” בפי הננסים,



בפי האלבים – “מונה-הזמן”.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר-דבר לכל היש:



מה שמות השמש / שרואים בני-אדם,



מה שמו בכל העולמות?


  

אלויס


בפי האדם – “שמש”, / בפי אלים – “חמה”,



בפי אצבעוני דולין – כעס,



“מאור-נצח” בפי הענקים, / בפי האלבים “הגלגל היפה”



“אור צח” בפי עם האזים.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש:



מה שמות העננה / הרת החזיזים,



מה שמה בכל העולמות?




אלויס


“עננה” בפי אדם, / בפי האלים “תקות-סגריר”



בפי הונים “שעשוע-הרוח”,



“מביא-מים” בפי הענקים, / “און חזיזים” בפי האלבים,



בפי הל “קובע הלוט”.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש:



מה שמות הרוח / המרחיק לחדר,



מה שמו בכל העולמות?


 

אלויס


בפי אדם – “רוח”, / “המשלהב” בפי האלים,



“הצוהל” בפי המושלים-בכל,



“הנוהם” בפי הענקים, / “הרועש” בפי האלבים,



“סער” – במשכנותיה של הל.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש: –



מה שמות האויר / הלא-נע ולא-זע,



מה שמו בכל העולמות?


  

אלויס


בפי אדם: “שקט-האויר”, / בפי האלים – “גלינה”,



“מנח-הרוח” בפי הונים,



בפי הטורסים – “מועקה”, / “נפש היום” – בפי האלבים,



“מפלט-הרום” בפי הננסים.


 

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש: –



מה שמותיו של ים / בו עוברים האדם,



מה שמו בכל העולמות?


  

אלויס


בפי בני-אדם – “ים”, / בפי אלים – “אגם”,



בפי שבט הונים – “משבר”,



“בית-הצלופות” – בפי הענקים, / בפי האלבים – “חמר-שתיה”,



ננסים יכנוהו “עמק”.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש: –



מה שמות האש / הלוהטת לאדם,



מה שמה בכל העולמות?


  

אלויס


בפי אדם “אש”, / “להבה” בפי בני-האזים,



בפי הונים “יקוד רותח”,



בפי הענקים – “האוכלת”, / בפי הננסים – “השורפת”



בביתה של הל – “הזריזה”.


  

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש: –



מה שמות היער / הצומח לבני-אדם,



מה שמו בכל העולמות?


 

אלויס


“יער” בפי אדם, / “רעמת-השדה” בפי האלים,



“אצת-הלדה” בעולם המות,



“דלק” בפי הענקים, / בפי האלבים – “רב-הענפים”,



“סבך” – בעולם הונים.




טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש: -



מה שמות הלילה / בתו של נור,



מה שמו בכל-העולמות?


 

אלויס


“לילה” בפי אדם, / “ערפל” בפי האלים



“המעטה” בפי השולטים בעולם,



“חסר-אור” בפי הענקים, / “ענג-שנה” בפי האלבים,



“יוצר-החלומות” בפי עם-הננסים.


 

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש: -



מה שמות המזרע / שזורעים בני-אדם,



מה שמו בכל העולמות?


  

אלויס


“דגן” – בפי האדם, / “נבג” בפי האלים,



“לבלוב-צמיחה” בפי שבט-הונים,



“טרף” בפי הענקים, “משקה” בפי האלבים,



בביתה של הל – “הסוף הרופף”.


 

טור


ענני, אלויס, / כי דומני, לא נעלם



ממך פשר דבר לכל היש: –



מה שמות המשקה / ששותים בני-אדם,



מה שמו בכל העולמות?


 

אלויס


“בירה” בפי אדם, / “סודן” בפי האזים,



“לחות-רקח” בפי הונים,



“המשקה הזך” בפי הטורסים, / בעולם-המתים “תמד”



ובפי בני-סוטוינג – “עסיס”.


 

טור


מעולם לא מצאתי כל כך / חכמת-זקנים



בלב האחד;



אבל עלה בידי / לכלותך בערמה.



אור-יום ממית את הננס,



הנה החמה מאירה בהיכל!


  

משלי הר


הגיעה השעה לדבר / מכסא הנואמים!



על המעין הקר של אורד



מחריש השקפתי, / משתאה התבוננתי



והאזנתי לאמרי הר;



הוא דבר על הרונות, / בלי שתאבד לו עצה;



זאת שמעתי בביתו של הר:


 


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך לתועלת, כי תקבלנה:



אך בלילה תקום, / כי תתחקה על אויב



או תתור לך מקום בחוץ.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



לעולם אל תשכב / בחיק מכשפה,



לא תלפתך שוקה.


 


"כי תקח את לבך, / ותשכח יום מועד,



תאחר מועצת-עם;



תרחק מאדם, / לא ימתק לך מאכל,



בדאגה תבקש שנה.


 


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



לעולם אל תדיח / להתנות אהבים



אשת-חיקו של אחר.


 


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



אם עליך לעבור / בחרשה או על צור,



תרבה צידה לדרך.


 


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



ירא בכל עת / לגלות לאיש רע



פרענות באה לך;



מספקני מאד, / אם יגמלך איש רע



תחת יצר לב טוב.


 


"ראיתי אדם / וראשו לקה



מלהג רע של אשה;



קפח חיתו / פה דובר נבלה,



לשון מוציאה לעז.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



אם מצאת ידיד, / אשר תאמין בו,



לעתים קרובות תפקדנו,



כי עשבים רעים עולים / וישגשג רב עשב



בדרך בה איש לא עובר.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



קנה איש חרוץ / לחברה נאמנה



ולמד כל-ימי חייך,



אם ידע אמרי-ישע עוזרים.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



אם ידיד נאמן לך / אל תהיה הראשון



לנתק עבותות לב ונפש;



כי הצער יאכלך / אם לא תוכל



לפתח לבך לאחר.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



לא תבוא לעולם / בדברים עם עלוב,



חסר לב ושכל.


  


כי מספקני מאד, / אם יגמלך איש רע



טובה תחת טובה;



אך לא כן אדם טוב, / הוא יצליח תמיד



לחבבך בשבחים.



"ידיד-נפש ידידך, / אם תגלה לו בכל-עת



את מחשבותיך כלן.



הרעה ברעות / לא להיות לב-שלם –



האומר אך נעימות לא יסכון לידיד.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



לעולם לא תדבר / שלשה דברים בריב



עם אדם פחות טוב,



לעתים קרובות / נשתה גבורת טוב



במכותיו של איש רע.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



עושה-נעלים / וחורש עץ-החנית



תהיה אך לך לבדך;



אם הנעל לא רעה, / לא עלה יפה העץ,



על ראשך תחלנה קללות.


 


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



אם נגרם לך הפסד, / הודה בהפסד,



אל תענה לשלום את-אויב.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



לעולם לא יהיה טוב / בעיניך הרע,



אך שמח אל הטוב בכל-לבך.


 


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



בתשואות-קרבות / אל תביט אל מרום,



כי הפחד מטפיש / את בני-האדם,



הלכודים בכשפי-משנאיהם.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



רצונך לשדל הנאוה / להתעלס במסתרים



ולהשיג נעימות ממנה,



תבטיח לה דבר יפה / ותעמוד בדבורך:



כל-האדם שש על דבר טוב.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



הוה זהיר מאד / אבל לא מוג-לב,



ביחוד בשתיה / ובאשתו של אחר,



ושלישית אכנה / ערמת-גנבים.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



לא תנהג לעולם / לעג וקלס



בנכרים ועוברי-דרך.


  


"לעתים רחוקות / יבוא יושב-בית



עד חקר טיבו של נכרי:



אין אדם טוב, / שאין בו כל-פגם,



גם רע איננו שלא יצלח-לכל.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



אל תלעג לשר / אם זרקה בו שיבה,



לעתים תכופות טובים דברי שב;



רבות פעמים מפיק / נכחות גלד-קמטים,



עומד תלוי בין העורות,



ובינות שלחים ירפרף ברוח,



על יד גלדה והמסס יתנדנד.


  


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



אל תגרש משער / נכרי בחרפות



שים לבך אל נצרך.


  


"יהי בריחך חזק, / יסוב בלי הרף,



יתן כניסה לכל-אדם;



שית עליו טבעת, / לבל יביא עליך



כל מיני פרענות.


 


"איעצך, לודפפניר, / לעצתי תשמע!



תצמח לך טובה, כי תשמע לה,



תהיה לך תועלת כי תקבלנה:



שכרון שכרך / גרשנו באון-האדמה,



[כי שדה יונק לחות / ואש לוקחת מדוה,



האלון רפאות לשלשול, / מלילה – לכשפים,



קרני-שפון לשבר, / ירח – לשגעון,



צריף לגרב, / הרונות – להרעלה]



השדה מושך רטיבות.


  


וכך נאמרו / משלי-הר באולם,



תועלת לבני-אדם, / ללא-תועלת לבני-הענקים.



אשריהו אומרם! / אשרי יודעי-אותם!



ישתמש בם השומע אותם!



שלום על כל השומעים!


  

נבואתו של גריפיר

היה אדם אחד ושמו גריפיר, והיה בנו של גילימי ואחיה של היורדיס. הוא היה מלך במדינה אחת ועלה על כל האדם בחוכמתו וגם ידע את הבאות. פעם אחת רכב סיגורד, הוא לבדו, ובא עד היכלו של גריפיר. היה קל מאד להכיר את סיגורד. בפתח ההיכל פגש אדם אחד והחליט לדבר אתו. שמו של אדם זה היה גיטיר.

  

שאל אותו סיגורד:


מי היושב פה / באולמי הבירה?



מה שם קוראים הגבורים / למושל העם?


  

גיטיר


גריפיר זה שמו / של האלוף הטוב,



המושל ביבשה / ובעם-הארץ.


  

סיגורד


הבביתו הוא / המלך החכם?



הנכון הגבור / לדבר אתי?



נכרי זקוק / לעצתו בידידות,



בחפץ-לב הייתי / נפגש עם גריפיר.


  

גיטיר


המלך העליז / ישאל את גיטיר



מה שם הגבור / הרוצה שישמעוהו.


 

סיגורד


שמי סיגורד / יורשו של סיגמונד,



והיורדיס היא / אם נוטר-הצבא.


  


וילך גיטיר / להגיד לגריפיר:



"פה בחוץ / עומד איש נכרי;



נאה ויפה / מאד הגבור,



הדורש לדבר / אך עם הנסיך."





יצא מן ההיכל / שורר-העם,



הסביר פניו / לאורח הזר:



"שלום בואך, סיגורד, / מדוע לא באת קדם?



אתה, גיטיר, קח / ממנו את גרני."


  


התחילו לדבר / על כמה וכמה דברים,



בהפגש הגבורים / חכמי-העצה.


  

סיגורד


הגד לי, אחי-אמי, / אם אך ידעת,



איך יעברו / ימי חיי סיגורד?


  

גריפיר


תהיה הנהדר / ונשוא-פנים



מכל הנסיכים / תחת השמים,



נדיב בזהב, / אך עצל לנוס,



תענוג לעינים, / חכם בדבורים.


 

סיגורד


מלך נעלה, / הוסף וגלה



לאיש סיגורד / את אשר אתה צופה:



איזה אשר / ראשון יפגשני



כשאטוש היום / חצרך, שליט?


  

גריפיר


קדם כל תקום / אביך וסבך,



תקום בדמים / בעקב התוגה.



בקרב תמגר / את בני-הונדינג



עזי-הלב / ותנצח.


 

סיגורד


מלך חכם, / הגד לקרובך



אמת לאמתה / בשיחה גלוית-לב:



היש עלילות זהר / בגורלו של סיגורד



המתרוממות / עד הרקיע?


  

גריפיר


אתה לבדך תכה / התולעת הנוצצה



השוכבת תאבה / על שדות גניטה;



תכה נפש / שני האחים



רגין ופפניר. / אמת דברתי.


  

סיגורד


יש די זהב / אם כך נלחמתי,



ואצא בשלום / כדבריך;



מה אתה רואה עוד / בחיי סיגורד?


  

גריפיר


אתה תמצא / מערת פפניר



ותשא האוצר / המבריק שם;



תעמיס הזהב / על גבו של גרני



ותסע אל גיוקי / גבור איתן.


  

סיגורד


הוסף והגד / בדברי חכמה



למלך, / אלוף עז-לב:



הייתי אורחו / של גיוקי בצאתי –



ואיך עתידים / לעבור חיי?




גריפיר


על סלע ישנה / בתו של הנסיך,



מאירה בשריון, / כי מת הלגי,



בחרב חדה, / במכה-פפניר,



תקום ותחתוך / השריון הנוצץ.


 

סיגורד


השריון נשבר, / הכלה תדובב,



האשה הנעימה, / שתקיץ משנתה;



מה תגיד לסיגורד / היפה-בטבעות



ויהיה למושל / לאשרו בעתיד?


  

גריפיר


אותך הגבור / תלמד הרונות,



למו נכספים / כל האדם;



ועוד שפות / עמים זרים,



ודעת רפאות – / אשריך, המושל!


  

סיגורד


זה עלה בידי, / קבלתי התורה,



אני נחלץ / הלאה ללכת;



העמק לצפות / הוסף והגד;



מה אתה רואה עוד / בחיי-סיגורד?


 

גריפיר


תבא, הגבור, / למעונו של הימיר,



אורח עליז / תהיה לאלוף;



יותר לא אדע / לספר, סיגורד.



ואל תוסיף לשאל / את אחי-אמך.




סיגורד


תוגה תוקפתני / בעקב דבריך,



כי אתה, האלוף, / מעמיק ראות:



הרבה עמל תראה / בגורלו של סיגורד



ואינך מעז / לאמר לי, המלך.


  

גריפיר


היו נעוריך / מאירים בזהר,



גלויים לעינים / ועיני בהירה;



אבל לא בצדק / קראוני רב-עצה,



גדול כנבואה: / הקץ לחכמתי.


  

סיגורד


איש לא ידעתי / במלא כל-הארץ,



המיטיב לראות / מגריפיר עתידות;



אל תעלים דבר / ממני, ואם רעות;



ואם גם אאשם / בעלילות רעות.


 

גריפיר


לא נכתמו / חייך בעוונות,



האמן תאמינה / בזאת, סיגורד;



תהלתך עומדת / עד בלי-ירח,



ולעולם לא יסוף / שמך, הגבור.


 

סיגורד


לא ישרה זאת בעיני; / אבל על סיגורד



לפרד מן המלך, / כי כן נגזרה.



הנח לעתידות, / יבוא מה שמכרח,



אך תראה לי דרכי, / דודי החכם!


 

גריפיר


עתה עלי להגיד / הכל לסיגורד,



האלוף המהלל / מאלצני לאמר,



הלא ידעת, / כי אמת דברתי:



אני יודע / היום בו תמות.




סיגורד


אין ברצוני להכעיס / את-המלך האדיר;



אלא לשמע / עצה טובה מפי גריפיר.



רציתי לדעת – / ואם גם אין זה שמח –



מה נכון וקים / בגורל סיגורד.




גריפיר


עלמה נאוה / חיה בבית הימיר,



בסוד גבורים / ברינהילד יכנוה.



בתו של בודלי; / הנסיך הנעלה



הימיר אומן / את העקשנית.


  

סיגורד


מה לי ולה / לבת המלך



יפת-התאר / הגדלה עם הימיר?



דבר אל נכון / הגד לי, המלך,



כי אתה צופה / כל-הגורלות.


  

גריפיר


תאבד ממך / שמחת החיים.



בשל הבת היפה / שהימיר אומן;



תדד שנתך, / קרב לא תערך,



תרחק מאדם, / תראה אך הנערה.


  

סיגורד


האין מזור / לתוגת אהבים?



הגד לי, גריפיר, / מה אתה צופה!



האקנה הנערה / באוצרות-כרה,



את בתו היפה / של השר הנעלה?


 

גריפיר


כלכם תשבעו / שבועות זה לזה,



אך לא תיטיבו / לעמד בנדריכם.



לילה אחד תשהה / בנחלת גיוקי



ותשכח היפה / שגדל הימיר.


  

סיגורד


איך קרתה זאת? / אותה לי תגיד.



כלום הפכפך / לבו של הויקינג?



האפר דברי / אמרתי לאשת



אשר אהבתי / בכל לבבי?


  

גריפיר


תהיה קרבן / תרמית אחרים.



מזמות גרימהילד / אתה תשלם;



את-בתה שלה / צהבת-השער



תתן לך / בערמה רעה.




סיגורד


אני אתחתן / בשבט גונר



וגודרון היא לי / תהיה לאשה;



הלא אשרתי / באשה טובה,



לולא זכרון / עול קדומים.


  

גריפיר


גרימהילד תקח / את לבך כליל,



היא תשכלך / לבקש יד ברינהילד



בשביל גונר, / מלך הגותים;



לאם המלך / תדור לנסע.


  

סיגורד


רעות תבאנה; / אני צופה יפה,



בינת סיגורד / תאבד מנו כליל,



ברצותו להשיא / לגבר זר



הנערה היפה, / שככה אהב.


 

גריפיר


שבועות תשבעו / כלכם זה לזה



הוגני וגונר, / ואתה, הגבור, שלישי;



עליכם בדרך / להחליף תארכם



אתה וגונר. / גריפיר לא ישקר.


  

סיגורד


איך זה בזה תלוי? / מדוע שנינו



מחליפים בדרך / מראה ותנועות?



ודאי רמיה / גוררת רמיה,



מחבלת מאד – / הגד לי, גריפיר!


  

גריפיר


מראה גונר / ותנועותיו תקח,



אבל בינתך / וקולך ישארו לך;



אז תארש / הנערה הנאוה



שגדל הימיר – / איש לא יוכל למנוע זאת!




סיגורד


רע הוא בעיני, / כי אדם רע



הלא יכנוני / בסוד הגבורים;



על אפי ארמה / בתו של השליט,



שהיא בעיני / העדינה בנשים.


  

גריפיר


אלוף-צבאות / תעלה יצוע



הנערה, כאלו / היא אמך-שלך;



תהלתך עומדת / עד בלי ירח,



לעולם לא יסוף / שמך, הגבור!


  

סיגורד


התהיה לגונר / אשה טובה,



לו למכבד מכל – / ענני, גריפיר! –



אף כי תישן / שלשה לילות



עמי כלת-החרב? / מעולם לא היתה כזאת!


  

גריפיר


שניכם בבת-אחת / תחגו חתנתכם,



גונר וסיגורד. / באולמי גיוקי;



שוב תחליפו / בבית את ברכם;



אולם תוככם / נשאר כשהיה.


 

סיגורד


הלטוב לנו, / לשני הגבורים,



יהיה אשר הרהבנו? / הגד לי, גריפיר!



האם להבא / יצמח מזה



טוב לגונר / ולסיגורד גם הוא?


 

גריפיר


תזכור השבועה, / הלא תחרישה,



ותראה חיים טובים / עם גודרון;



אולם ברינהילד תאמר / כי לא טוב התחתנה,



תהיה שואפת / לנקום את הכחש.


  

סיגורד


התפיס תשובה / את לב האשה



אשר רמינו / בערמת-רמיה?



לה נשבעתי / ולא קימתי,



ואשר מעט / ידעה האשה.


  

גריפיר


אמר תאמר / לגונר אשתו,



כי לא שמרת / לו הבטחותיך,



כי האמין השר / הנעלה באמונה שלמה,



בנו של גיוקי, / לאיש לסיגורד.


  

סיגורד


איך יפל הדבר? / אל תעלם מני דבר!



האמנם עשיתי / את-התועבה?



או לעז תוציא / עלי המלכה



וגם על עצמה? / הגד לי, גריפיר!


  

גריפיר


מעלבון גאותה / מתוך כאב וכעס



תרע לך / העדינה;



לא הכלמת / את היפה,



ואם גם ביודעים / רמיתם את אשת המלך.


  

סיגורד


הישמע גונר החכם / גם גוטהורם והוגני



לקול דבריה / התובעים נקם?



היאדימו החניתות / בניו של גיוקי



בלב גיסם? / אל תעלים, גריפיר!


 

גריפיר


אבל-זעם / מלא לב-גודרון



כשימיתוך / האחים הבוגדים;



לא יצמח שוב / משוש בחיי



חכמת-הנשים – / גרימהילד זאת עוללה.


  

סיגורד


אשרי הנפרדים! / איש לא יעמוד בפני הגזרה.



יפה מלא גריפיר / את משאלותי;



ברצון היית / מגיד אך שמחות



מימים יבאו / אלו יכול פיך.




גריפיר


נסיך-צבאות גבור, / זכור והנחם,



כי אך מזל מרהיב / נפל בגורלך:



לא יתהלך לעולם / על עפר באור שמש



גבור עדין / ממך, סיגורד.
