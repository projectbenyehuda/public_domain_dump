


	
	ה' שבור זרוע רשע
	



ה' שבור זרוע רשע / רבי שמעון בן צמח דוראן (הרשב"ץ )




ה', שבור זרוע רשע / מבטן פשע / ירחקו בניו מישע



חילים מגבר / ושלום כי אדבר / עובר ומתעבר



‘מה לך ולשלום?’





מכים עבדים / ונוגשים ורודים / בוזזים ושודדים



ופסו אמוננו / ומרב עווננו / כלינו שנינו  ( 5 )



קוה לשלום.





עתק מפיו מוצא / הבל פיהו יפצה / עושה אשר ירצה



אויב כמוץ ידפם / יהדפם / ירדפם



יעבר שלום.





והוא מתעקש / להיות לי למוקש / להדיחני מבקש ( 10 )



אומר לי: לך עבוד אל זר / ויקם לך אומר תגזר / ובו תהיה נעזר



וישם לך שלום.



נחש השיאני / אך עתה הלאני / ובכלאו הביאני



להיות למחיתה / אומר לי אם עתה / תמיר כבודך אתה



תבוא אל אבותיך בשלום.  ( 15 )





בונה במה / טלואה ורמה / ובקול דממה



משכים ומעריב /מבית מלא זבחי ריב / לחם כזבים מקריב



ויקרא לו ‘ה’ שלום'.





רבו כמו רבו / במי מריבה רבו / בריבם עמדי נצבו



ואני אמרתי בדמי ימי / אלך שאול באשמי / כי מפני חטאתי בעצמי
( 20 )



אין שלום.





צור לבבי וחלקי, / ארחמך, ה' חזקי! / שם לי לחרדה נשף חשקי



אויב שניו חורק / ואין מציל פורק / וזרע אמת נטעתיו שורק



זרע השלום.





מבשר גאלה / לשם ולתהלה / לעדה סגלה ( 25 )



אשלח בבוא עתו / באיתן קשתו / בריתי היתה אתו



החיים והשלום.





חיים וארך ימים / נתתי לו לעולמים / וכי באמת ובתמים



קנא לאלהיו בחילו / יכפר על עמו וממות גאלו / הנני נותן לו



את בריתי שלום. ( 30 )
