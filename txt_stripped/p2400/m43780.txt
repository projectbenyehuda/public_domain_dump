


	
	אסדר שבחא
	



סימן: א-י. לחן: שוכנת בשדה.






אסדר שבחא / לאלה כל רוחא / די עבד רוחא / לבנין קדישין

בידא תקיפא / וחרבא שליפא / דן עמא חציפא / וכל זיני חרשין

גלי לן נסוהי / עבד לן אתוהי / אחזי לן תמהוהי / דאנון תקיפין

דכרין יהודאי / ית מופתין סגיאי / מחא ית מצראי / בעשר מכתשין

הרס ית קרויהון / אבד ית גיתיהון / שצא ית טפלהון / וגברין ונשין

ואפיק ית עמיה / אצשך לון רחמי / עבד בדיל שמיה / נהיר לון חמישין

זכות אבהתנא / הוא אגין עלנא / דחס רחמנא / על רוחין ונפשין

חמן בני עמא / תקפא דרב עלמא / כד בזע ית ימא / בזעין נפישין

טבע ית מצראי / במין סגיאי / נחתו לעמקאי / במנין ולבושין

ישפר האי שבחא / קדם אלה רוחא / וישלח משיחא / לבנין קדישין.
