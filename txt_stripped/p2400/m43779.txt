


	
	שמחו נא
	



שיר לכבוד ברכת הלבנה הרומזת על גאולתנו שבב״א (שתבוא במהרה בימינו אמן)

סימן: חותם חיים חי






שמחו נא שמחו נא / בברכת הלבנה

שמחו נא שמחו נא / בשמחת הלבנה



חדוש הירח / בו לבי שמח / וצר ננגח / באור השכינה

ופועל האמת / פעלתו אמת / ותהיה נשלמת / בשלמות בנינה

תתחדש גברת / בעלוי עטרת / עד כתר תפארת / בתפארת בינה

מצמיח ישועה / התר השבועה / בא והודעה / בעיר העליונה

חדוש הנשמות / יהיה בעולמות / יאירו השמות / מעבר ופנה

ימשך אור עליון / הנעלם מרעיון / ממקור העליון / עד מדה תחתונה

ישראל הוא יחל / בנין בית וחיל / יעקב ורחל / באו נא ברנה

מלכי לי תעזר / ולמחלי שים מזור / וחיש נא תחזר / עטרה לישנה

חלו נא פני אל / ישלח נא הגואל / יקבץ ישראל / עם זו אשר קנה

יה שמע תחנה / ושלח נא חנינה / אנא הושיעה נא / אנא הצליחה נא.
