


	
	אל חי
	



לכבוד משיח בן דוד. סימן: אהבה נצר לנו





ב






אל חי מעלה / לך אוחילה / ממך אשאלה / תן-לי נחלה

בבן ישי.

השמד הזרים / אויבים וצרים / נכרים הצוררים / אשר הם אומרים

מי בן ישי.

בנה אריאל / למענך אל / ושלח נא הגואל / מלך ישראל

בן לישי.

השב עטרה / על ראש גבירה / והזרח נא אורה / חטר תפארה

גזע ישי.

נא הושיעה נא / והצליחה נא / כי עת לחננה / יבוא ברנה

שרש ישי.
צמח הצמחת / כסאו כוננת / ואותו אהבת / כי בוחר אתה

בבן ישי.

רגלי המבשר / שלום ואשר / הם קלו מנשר / אותי יבשר

בא בן ישי.

לו יבקשו / אותו ידרשו / אויביו יבושו / אליו יגישו

מלכים שי.

נכון הוא כסאו / אל חי נשאו / כל אנשי צבאו / דרשו יום בואו

מאל עושי.

וחיש נקריבה / בשמחה רבה / וברב אהבה / שלמי נדבה

לחם אשי.
