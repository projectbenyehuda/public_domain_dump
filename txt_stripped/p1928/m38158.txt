


	
	ויהי לימים אחר אשר
	








ויהי לימים / אחר אשר מצאונו אימים /
ואלך לתור מנוחה / והשקט ובטחה / ואמצא שם אניה באה תרשיש / ויאספו אליה כל זקן וישיש / והרוח אשר שם היה מרעיש / ואשכור אותה / היא וכל אשר בה / במאה אלף פרחים / ואשאל על אודות רב הספינה, ויגידו לי כי הוא חכם גדול, חושב מחשבות עד אין חקר, ויודע להוליך הספינות / בין הימים והרוחות / הקשות והחזקות / ויודע לכון השעות / הטובות והרעות / ואבקש לראות פניו / שהוא טוב ועניו / ואמצא איש אחד בין חכמיו ונבוניו
/ טוב ויפה / קורא בתורה על פה / מראהו כמראה האלהים / כקדוש כמלאך אלהים / ואומר ליושבים לפניו: ׳בחייכם! מי הוא זה / אשר הוא כנביא וחוזה?׳ / ויאמרו: ׳לא ידעת? כי הוא עמיתך / והוא בן בריתך / ודמותו כמותך / אלא יושב בין חכמים / נבונים וידועים / ולא שזפתו שמש / ולא שרב ורפש / ושפתותיו דובבות בארש׳ / ואדע נאמנה / אחרי אשר הגידו לי באמונה / כי הוא איתן האזרחי / אלופי ומידעי / ואפול לפניו בודוי ותחנה / ואומר לו: ׳חטאתי לך נכונה / שלא שמעתי לעצתך
הנכונה / ותברח ממני בכל פנה / ואני אנה אלך מרוחך אנה / ועתה, בבקשה ממך, שתלך עמי ביחד / ולא אוסיף עוד אמעד / מיום שהלכת גופי רעד רעד / ויאמר לי: ׳הן!׳ / ואשמח בלבי ואומר:
׳יהי כן׳. / ונלך שנינו יחד / ויספר את אשר קרהו, ואשר ראה פלאי פלאות / הווות ועתידות / יש מהם במקראות / ויש במראה הנבואות / ויפתח שפתיו היפים / וידבר במשליו הנעימים / ויאמר: ׳אודיעך אחי, חזמק התימני / כי יש מעשה רעיוני / ממשל הקדמוני / ראיתי אותו בעיני / ואומר אליו: ׳דבר, כי שומע עבדך!׳



ויאמר: אחי ורעי שמעו / הטו אזנכם ואל תפרעו / ובראשכם איש הנחמני / חזמק
התימני / כי כשהלכתי מאתו / וחשבתי להפר בריתו / הלכתי אל המדבר / ואסע מלא דבר / ולא היה עמי אפלו פת קיבר / והייתי אוכל מלוחים / ועשבים ופרחים / ואגיע לשפת הים רעב וצמא וערום / ואראה שם זקן ועילום / ואומר לו: ׳עליך
השלום!׳ / ויאמר לי: ׳מי הביאך עד הלום?׳ / ואומר: ׳אלהי עילום!׳ / ויאמר לי: ׳בני, לא ידעת שמקום זה יש בו לסטים / ועל פני המים שטים / גוזלים וחומסים?׳ / ואומר לו: ׳אני עכשו רוצה ללון / לנוח
מצערי ולזון / ואתן כל אשר לי, ואתן שנה לעיני / ואטעם פת בשני / אולי אשכח יגוני׳ / ויאמר: ׳ברוך הבא! / באהבה וחבה׳ / ויוליכני לביתו / ויראני את כל בית נכותו / וחדרי משכיתו / ואמצא מגלה כתובה משלים וחידות / טובות וחמודות /
ונאכל ונשתה ונברך לאל ההודאות / ונשב כל הלילה, ונקרא שם זה המעשה על אודות השועל והזאב:



ויהי לימים / בימים קדומים / היו המלך והחיות מסבים / ויאמר הזאב למלך: ׳אדוננו המלך! / עבדך רעב והלך / ויש לנו שלשה ימים לא טעמנו מאומה / מכל פרי האדמה / נלך ונשוט על פני האדמה / אולי נמצא לנו תקומה / וצריכים אנו אחד חכם, לתת לנו עצה וחכמה׳ / ויאמר המלך: ׳מי בעיניכם נבון?׳ ויאמר הזאב: ׳הנה השועל אתנו / תן לו רשות וילך אתנו / כי עצתו כעצת אחיתופל קמה / וידו על הכל רמה / והוא פטור מכל מס / ואין עליו שום חמס׳ / וכונת הזאב שיביאנו לידי הרס / ואז לבבו ימס / ויביאוהו לפני המלך / והוא מתעטף בשלמה של כלך / וחוט החרוזים על צוארו / ועניו אין כמותו / שפתותיו דובבות / בתחנות ותפלות / וכיון שהגיע אצל המלך, ויכרע וישתחוה ויאמר: ׳שלום עליך אדוני! למה שלחת לעבדך?׳ ויאמר אליו: ׳תדע לך שאתה חכם / ויודע בעצות יותר מכלם / והזאב ידע בחכמתך / כי לא תחטיא בעצתך / ולבו שלם עמך / אם דבריו אמת ונכונים / ואין בלבו מינות וטינים / תלך עמו ותורהו איזה טרף׳ / ויאמר השועל: ׳תן לנו כתב, לילך אל הישוב / ונראה להם כתב כתוב׳ / ויתן להם כתב מפאר / כל אשרה ימרהו יואר.





וילכו שניהם / והכתב בידיהם / ויקרבו אל הישוב / ויאמר השועל אל הזאב / הרעב והתאב: ׳תדע נאמנה / כי היום ערב יום טוב ליהודים הנאמנים / תקח הכתב בידך / שאם יבואו מולך / תראם ולא יוכלו לך׳ / וכיון שהגיעו למדינה, והיא סגורה מכל פתחים / ולא מצאו כי אם חור בין הפתחים / ויכנסו דרך שם / על לא חטא ועל לא אשם / וימצאו הרבה נתחים / ברחובות ובשוקים / ויאכל השועל מעט להחיות / כי ידע סוף הדבר להיות / ומחמת שלא מלא בטנו / יצא מן החור,
לא בא אדם לשטנו / והזאב השוטה שמע לעצתו הנבערה / ותאותו עליו גברה / בראותו הנבלות מרבות / בשוקים וברחובות / פתח בטנו לבלי חוק /
ולא יכל לילך למרחוק / ויכבד עליו גופו / אוי לו ואוי לגופו / כי כשזרח השמש / הרגישו בו היהודים ולא יוכל למש / כי גופו גדל / והחור קטן לא יגדל / ויבואו היהודים / ובמקלות חובטים / ויצעק צעקה גדולה ומרה: ׳הלא ראיתם כתב המלך הנקרא׳ / ויצעק אל השועל ויאמר: ׳זאת עצתך הנבערה / הכנסתני לזאת הצרה!׳ / אז פתח השועל ואמר: ׳לא ידעת מאמר החכם וחידותיו? / צדיק מצרה נחלץ ויבא רשע תחתיו׳. – – –
