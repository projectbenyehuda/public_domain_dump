


	
	להבין משל ומליצה
	








להבין משל ומליצה דברי חכמים וחידותם
/ לדעת מוסר נביאי השיר ותוכחתם / חפש המבקש
משורר אשר לו בשיר עשר ידות / מנצח בנגינות
ומשכיל שיר ידידות / ולומד מנעוריו לדבר
בנכבדי ארץ נכבדות. / וימצא משורר אשר
המליצה היקרה בפיו ונכבד ניבו / וחזון השיר
נפרץ בלבבו, / איש דברים ורב מהללים / אזן וחקר
תקן משלים, / נהרי נחלי דבש וחמאה בזמרותיו, /
נפת תטפנה שפתותיו / וצוף יזוב משירותיו, /
ימלא הפה שחוק ורנה באמרו אמריו / וצור יצוק
פלגי שמן בדברו דבריו. / בשוררו שיריו ועל
פיו יעלם / אזידלג כאיל פסח
ותרן לשון אלם, / יכפר כל עוון וכל חטאת ואם
מנשא גדולה וכבדה / באמרו מזמור לתודה. / בחן
שפתיו יצוד הנשמות / וימשך לו בנעם מליו שרי
אדמות, / יתנו לו מפרי אמריו ובשערים יהללוהו,
/ ריח שמניו טובים, שמן תורק שמו – על כן עלמות
אהבוהו. / בשירותיו ינשא למעלה נדיבי עם
ושועיהם / וישפיל למטה בני בליעל והעושים
כמעשיהם.





משורר מעשיו תמה לרואים, / למען יעשה בשיר פלאים

בהגביהו עדי מרום שפלים / והשפילו עדי ארץ לגאים.





אמר לו המבקש ״אדם המשורר, / אשר
בחכמת שיריו ישנים יעורר, / כתמי התוגות
במליצותיו ילבן ויברר / וכל אמרותיו אמרות טהורות
/ והוא נעים זמירות! / צופה הייתי בשירותיך אשר
מפניני הצחות חצובות, / והסתכלתי בזמירותיך –
והם אור זרוח ובקר לא עבות, / ודברותיך מצוף דבש
ערבות / ושירותיך ראויות להלל בהן לרוכב ערבות /
וכל דבריך במשפט תכלכל / להבין אמרי בינה
ולדעת מוסר השכל. / ואחרי שחננך יי דעת / חובה
עליך להתעסק בחכמות האמתיות וסתריהן לדעת, /
כי הן צורת האדם וצלמו / ובשכל נבדל האדם על
כל נבראי מטה ויי יהיה עמו. / ואינו מהנכון
לבלות הימים בהבלי המשוררים / אשר פיהם דבר
שוא ולשונם תהגה שקרים / ולטוב רע ולרע טוב
אומרים / הבונים על יסוד השוא שיריהם / ותהי
האמת נעדרת בדבריהם, / הנושכים בלשונם וחקקי
און חוקקים / ובכזבי הדמיון מחזיקים / ויתלקטו
להם אנשים ריקים. / וצדקו באמרם המושלים / כי
החכמה כפרי והשיר כעלים. / ואמר החכם: המצליח
מאד מבני אדם מי שמתעסק מנעוריו בחכמת
המופת לא בהבלי המשוררים. ואמרתי אני:





כלה ימותיך בחדש סוד / חכמה ודעת – לא בדברי שיר,

ועמד להתחכם, ולדעת / דעת קדושים דרכך תישיר.





ראיתי והנה הפכפך דרך המשורר וזר
/ נכרי לחכמי לב ולאנשי אמת מוזר, / כי לכילי
יאמר שוע ונדיב אל נבלים / דבר בליעל יצוק בו
ועמו התולים, / זה דרכו כסל לו וחמה / ושוא
ודברי כזב בשירותיו ומזמה, / ועל יסוד השקר
בונה בית אפל / וטח קירותיו תפל / ויצפה אותם סיגים
מחוץ ומבית / נוטע כרם והנה עלה כלו קמשונים
כסו פניו חרלים ועלה שמיר ושית, / והמבקש
מדבריו והנדרש / להיות בהם הכזב מפרש. /
אומרים אמור שיר עגבים, / ופונים אל רהבים ומלאו
בתיהם









שקר, ושטי כזבים. / חמר הבתים גרוע
ושפל – / מאסו הבונים ביסוד האמת והנם טחים אותו
תפל, / והוא מחמר ההקש החמישי הגרוע והאפל – /
אין נגה לו כי אם חשך ענן וערפל. / ובעבור כי
חכמת השיר מהאמת רחוקה / ורוח שקר בפי נביאיה
ומהאמת נעורה וריקה – / לא ישמשו בעליה אלא
בשמות המשאלים והמעתקים / אשר הם מהורות על
האמת רחוקים, / ולא ישמשו בשמות המסכמים /
אשר ישמרם פי חכמים.

וכשמע המשורר מליו גער בו גערה /
וחם לבו בקרבו ובהגיגו אש בערה / ויאמר ״צדק
שהקדימני במשלו / כי מי שלא ישכיל הדבר הוא
שונא לו, / ולולי שאני נושא פני המוסר – בהבל פי
שרפתיך / ולחרפות לדראון עולם שמתיך, / כי
חרפת מערכות המשוררים / אשר כל דבריהם על
אפניהם דבורים, / וזה דרכם להזהיר האדם לקרות
מוסר וחכמה / ולבקש סגלת הנפש החכמה וכל
מעשה רמה / ולשבח מעשה הטובים / ולהלל
ולאשר מעללי הנדיבים / ולנבל ולגנות בני
בליעל / האומרים לאל ״סור ממנו!״ וישעו לבמות
בעל, / ולפרסם החנפים אשר ביוצרם מעלו מעל /
ומעשי הרעים להודיע ולהלשין / וכמו שאמרו
חכמים זכרונם לברכה: האי מאן דסני שמענה שרי
לבזויה בגימ״ל ושי״ן.





אמר איך תערך איש אל משורר? / והוא הלב ישמח עת ישורר

וימתיק עם חבריו סוד ורעיו / וחיי המקנאים בם ימרר

ולרדף הוד מידעיו יעורון / ולדרש כל יקר אותם יעורר.

וישים מחשכי הדברים / באור, ודבר שפתים יברר,

וראש תנין ילידי יום ישבר / בעז שיריו, וים תוגות יפורר,

ויניח במליו אוהבי תם / והישר, ושונאיהם יצורר,

יברך מעשה נדיב ופעלו / כמו כל מעשה כילי יארר,

יחרף הזמן, יען יעקש / ישר לבב ואת דרכו יסורר





קום, המבקש, ושמע ועדי האזינה, / ולץ,
שמע גערה, ופתי, חכמה הבינה! / ראה כי בשיר
אומרים תהלות לאל / אשר שוררו אנשים מלפנים
משה ובני ישראל, / ושירת ״האזינו״ שהיא כוללת
כל עקרי התורה / כלה על דרך השיר אמורה. /
ומהמשוררים היו אנשים צדיקים וישרים / ברוח
הקדש מדברים, / וספר תהלים ברוח הקדש כתוב
וכלו מזמורים / וחכמינו קראו קדש הקדשים ל״שיר
השירים״, / יען בו נאמר דבקות הנפש במקורה /
ואהבתה וחשקה אל יוצרה. / ודרך השיר להעיר
האהבה ולהמתיק סוד האוהבים / ולפרסם השנאה
ואיבת האויבים / והחכמים הקדמונים אמרו כי הפסד
היצורים והויתם – / השנאה והאהבה סבתם: / כי
בשנאה יתפרדו וזה מעל זה ירחקו / ובאהבה
יקרבו זה לזה ואיש ברעהו ידבקו, / ומטבע
המשורר לרדף אחרי הנאות בדבורו ובפעלו /
ושפת יתר נאוה לו. / והשיר הטוב – ראיה כי טבע
זך טבע אומרו, / ושהוא מגדולי הנפש ואשר
המלכים חפצים ביקרו. / וישא משלו ויאמר:





בשיר בעת יפל לנפשך חבל – / רבים יהודוך ושרי תבל.

תתן אליהם את רעות רוח והם / לך יתנו כספם – וגם זה הבל!









וזכרו כי באמה מן האמות / מלך רב
ואוזר חיל ותעצומות, / העמיד במשפט ארץ / ויפרץ
באויביו פרץ, / והרשעים לא שכנו ארצו ויתנם
לכלה וחרץ. / ויהי כשמש הודו, / יראוהו אנשים
ולא מחל על כבודו, / ובתקפו ובגבורתו / נשאת
למעלה מלכותו, / בן־חרין וחכם בכל חכמה /
ובתחבולות עושה לו המלחמה, / גבור חיל נודעו
בעמים פעליו – / ואני הוא האומר עליו:





לא עצרו כח לנגדו לוחמיו, / אך נחבאו כצאן [למול] טובח.

לא יחרד ברבות חיליהם – ואיך / ברבות תישים יחרד זובח?





והוליד בן ואין בידו מאומה ממדות
אביו המעלות / ונולד רש מקניני מעלותיו
הגדולות. / ויתבונן אביו בדרכיו / והביט און בו
ולא ישרו בעיניו מהלכיו / כי אינם כדת המלכים
וכהלכה / והוא נער ורך ולא יצלח למלוכה. /
והמלך נעצב אל עמו, / בראותו כי אין ממלא
מקומו: / כי בהיות המלך מכלכל במשפט דבריו /
והוא חכם ומבין לאשוריו – / ישמח כל עם הארץ
ותשקט כל עיר, / ואם הוא סכל – יאנח עם וקול מהומות
בארצו ובלהות בעיר. / ובחלום הלב ידוו כל
האברים ותהי אנושה מכתם – / וכן העם לפי שכל
מלכם תהיה הצלחתם. / ואמר שלמה ״מלך במשפט
יעמיד ארץ״, / ואמר החכם: כשירצה השם להיטיב
לאמה ישים השכל במלכיה / והמלכות בחכמיה. /
ואמרתי אני:





ברצות האל דרכי העם / ישים השכל במלכיו,

כי טוב עם ועם יהיה על / פי שכל שריו ונסיכיו.





וישלח המלך לחכמים יודעי העתים
ללמדו חקי המלכות ולהדריכו בהם / ולהורות אליו
מוסר המלכים והנהגותיהם, / ויסרו אותו – ולא היה
שומע אליהם / והיה הולך בשרירות לבו וחבור
עצבים. / ויתאבל המלך על בנו ימים רבים / עד
שנואש ממצא מרפא למחלת נפשו / וישלחהו
מעליו וגרשו.

ולקץ ימים בא משורר גדול במלכותו /
מקצוי ארץ אשר אין כמותו, / ויגידו לו דאגת
המלך על סכלות בנו ורע הנהגתו. / ויבוא לפני
המלך ויקד לפניו וישתחוה / ויאמר ״יחי
אדוני המלך, ובוראו יחישה מפלט לו וכל אשר
יאוה!״ / ויאמר המלך לו ״מה מלאכתך ומאין
תבוא ומאיזה עם אתה?״ / ויאמר ״ערבי אנכי ומלאכת
השיר מלאכתי מאז ועד עתה, /



ואני בא מהאי הנקרא אי השיר, / יען כל שוכן בו
יפתח שפתיו בזמירות וישיר. / ואני הגד הגד לי
דאגתך ויגונך על אודות בנך, / ואם כשר הדבר
בעיניך – / ישב הנער אתי ימים או עשור / ואני
אדריכהו במוסר המלכים ואיסרהו יסר. / רוח
השיר אתן בקרבו – / ואז תבוא חכמה בלבו /
ונחה עליו רוח דעת וחכמה / ורוח עצה וגבורה
למלחמה״. / ויאמר המלך ״קחנו ועיניך שית עליו /
ואולי בלמדך אותו יתנכר הנער במעלליו /
ויהיו ישרים וזכים פעליו, / ותהי משכרתך
שלמה מעמי / ואתה תהיה על ביתי ועל פיך ישק
כל עמי״.

ויחל המשורר לפעמו במחנה המשוררים
/ וללמדו ספר הצחות ולשון השירים, / ויהי
בדרך וימצא להקת נביאי השיר / ונהפך לאיש
אחר וישיר. / אז אמרו הרואים / ״מה זה היה
לבן־המלך? הגם הוא בנביאים?״ / ויהי כנוח הרוח
עליו / דבר נכבדות וישא משליו:





לי לב יאוה לעלות אל ראש / עש או היות נקבר בתחתית הר,

נפשו להשיג מעלה עליו / תקל, ולרדף היקר מהר.

כי מי בעפרה חשקה נפשו / יקר בעיניו לו יהי מהר?





וראה אדם מיסר בשוטים / לאחד מהשוטים, / וישא משלו ויאמר:





בזד יהיר בלי תחת גערה – / ואך כי לילה ויום תשיחו,

ואל תכה בשבט גו חסר לב / לישר מעוה לבו ורוחו,

אבל הכה ופצוע לראשו, / למען כי מקום השד במחו!





אחרי כן גרה המשורר מדון בינו ובין
אחד מרעיו / ויהי ריב בינו ובין אחד ממידעיו. /
ויאמר בן־המלך להכותו בחרב שיריו / וירה אליו
חצי אמריו, / ואני ענינם העתקתי / ובזה הספר
אותם חקקתי:





אשאל לאדם בחלום אחזה / אותו ״הזה הריק והנבזה



מצאצאיך?״ וענני / ״חוה אגרש, אם אמת הוא זה!״





וילך בן־המלך הלוך וגדול ויי עמו /
ונשמע שמו בגויים ויגדל שמו, / וישלח המלך
בעדו / ויבוא אליו עם המשורר אשר למדו. /
וכראות המלך פניו אמר ״התנכר הנער הזה
במעלליו / והכרת פניו ענתה בו כי צלחה רוח
אלהים עליו״. / ויגד לו המשורר כל אשר קרה לו /
ויתן לו ככל חפצו ומשאלו.
