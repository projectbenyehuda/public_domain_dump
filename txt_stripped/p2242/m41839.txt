


	
	אליך שועתי
	



סימן: א״ב. בראש הצלעיות האחרונות בכל טור: משה חוצין.
מנגינה: באבא טאהר. לחן: יאמר נא ישראל.


אליך שועתי / בשמך בטחתי / גדלך ספרתי / מהלל יי



דלתך שקדתי / הקשיבה רנתי / וגם לתפלתי / שמעה יי



זכר נא בריתי / חושה לעזרתי / טרם לך קראתי / הושיעה יי



ישעך צפיתי / כונן בית תפארתי / להקריב מנחתי / חדשה ליי



משגב בצרתי / נקם את נקמתי / סלעי ומצודתי / ומנוסי יי



ענה עתירתי / פנה לתחנתי / צוה ישועתי / צור ישעי יי



קרב קץ פדותי / רחם על נשמתי / שמך שמעתי / יראתי יי



תאיר אפלתי / תחיש גאלתי / תשיב נחלתי / נחני יי.
