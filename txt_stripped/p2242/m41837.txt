


	
	אעורר בשירה
	



להרב הכולל מוהר״ר משה חוצן בן לאותו צדיק מר זקנינו הרב הגדול ראש
גולה שבבבל מוהר״ר צדקה חוצן ז״ל.

סימן: אני משה חוצין.


אעורר בשירה / ברתת ומורא / לאל רם ונורא / ולו עז ומשרה.



נעים שיר יהלל / לצור רב מחולל / לפניו אמלל / בשפה ברורה



יברך לעמו / ברב עז שלומו / ויבנה הדומו / באבן יקרה



מעודד ענוים / סחופים דוויים / אשר הם מקוים / לטובה בשורה



שמי רום רכבו / וישר נתיבו / מכלכל בטובו / לכל נצר נברא



הלא כל צבאיו / יעידון פלאיו / אשר חי ברואיו / בידו מסורה



חסדיו אצפה / מהלל בכל פה / לצירי ירפא / יהי לי לעזרה



וישלח מנחם / לעמו ירחם / ואותם ינחם / בבית הבחירה



צמאי ירוה / ישועות יצוה / והר טוב ונוה / יכונן מהרה



יקבץ נפוצות / ונשמע בחוצות / צבי כל ארצות / נעים שיר וזמרה



נשמה בעוד בי / בקומי ושכבי / אהלל בניבי / לאל כי יצרה



ואשא לכפי / ואשיר במו פי / לאל שם באפי / נשמה טהורה.
