


	
	מלכי ארימה
	



סימן: משה חוצין ח׳זק.


מלכי ארימה / קולי בנעימה / אשיר באימה / לך שוכן רומה



שדי השקיפה / בנעים יפה / היתה לחרפה / לבני האמה



השב עטרה / לבני גבירה / חשכם האירה / כאור החמה



חסין וקדוש / דמיהם דרש / ושימם לראש / עלי כל אמה



וזכר אליהם / זכות אביהם / ובכל אויביהם / עשה נקמה



צור תמים דעות / צוה ישועות / כי שבעה רעות / יונתך תמה



יושב על כסא / לעם בך חוסה / פשעם תכסה / עוון ואשמה



נותך שכלל / צור רב מחולל / ולך תהלל / כל הנשמה



חזק בריחי / צור גואלי חי / וקבץ נדחי / קדמה וימה.
