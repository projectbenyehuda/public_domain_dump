


	
	מלט אותי
	



סימן: משה חוצין ח׳זק.


מלט אותי / מגלותי

ובנה ביתי / ותפארתי

ושוב אתי / כקדמותי

יום היותי / על אדמתי.





שוכן עליון / אין לך דמיון

חזק רפיון / ידי אביון

ובנה ציון / ואפריון

וחיש פדיון / קהלתי.





הקם דגלי / וסמך נפלי

ושמע קולי / ומהללי

כי את אלי / וצור ישעי

ומנוס לי / יום צרתי.





חושה אבי / לעם שבי

אמר שבי / תוך עיר צבי

ושעה ניבי / מקום חלבי

בהקריבי / תוך נותי.





וזה כמה / אני צמא

אשאב ממי / ישעך, אפעה

ולך אהמה / ולא אדמה

קרב ימי / גאלתי.





צור מחולל / ואור וליל

לך אמלל / שיר והלל

ויום וליל / לך תחולל

ותתפלל / יחידתי.





ישעי אורי / האר נרי

ושלח צרי / לכל צירי

ושעה צורי / אלי שירי

מקום צורי / תוך קטרתי.





נא ציץ גזעי / ושא פשעי

ושעה שועי / להרגיעי

כי את רועי / ומושיעי

וגם סלעי / ומצודתי.





חזק לולי / חסדך עלי

דל ונקלה / היה כלה

לך אחלה / חסדך הפלה

וחיש גלה / קץ פדותי.
