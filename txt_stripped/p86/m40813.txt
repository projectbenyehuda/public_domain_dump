


	
	מחברת הנדיבים
	








נאום יהודה בן חריזי הספרדי: בעוד



שדי עמדי / ובימי חרפי וכבודי / על נהרי שלוה



פרחתי / ובגלגל השכל זרחתי / וענני גילה רוו



תלמי / והפיחו מתוך גנזי אסמי בשמי / ונגה אור



לשמי / עד בגד בי הזמן והתמכר להרע לי /



ויעזוב בגדו אצלי / ושלח ידו בשלומיו /



וידבר לסלע עיני ונתן מימיו / וגדר דרכי ועוה



נתיבותי / ושם אל ים הדמעות תוצאותי / ויסר את



אופן מרכבותי / וינהגני בכבדות / וקשר ביני



ובין הפרידה ידידות / וכאשר הכה במטה היגון



סלעי צלעי / וזורמו דמעי לרגעי ואען ואומר

אלו כפי אידי דמעי יזלו / לא דרכה רגל אנוש יבשת

באמת דעו כי לא למי נח לבד / כי אם לדמעי נראתה הקשת.





וכאשר נסעו אהובי מבין זרועי /



ויחנו עצבי בין צלעי / על כנפי פירוד רפדתי



יצועי / ונחפזתי למסעי / אשקה בדמעתי ארץ



ציה / כאלו דמעתי ים ואני אניה / ואען



ואומר

כל הדמעות יזלו גם יחדלו / עתים לבד דמעות אסירי אהב

עת מכבדם איד אנחות יעלה / יהיה למים בעלות הלהב.





וימלך עלי לבי לקשור עפעפי



בעפעפי שחרים / והנני בכנפי נשרים / לדרוך



ציות / ולרכוב אניות / עד תגיע הנפש לרצונה /



ותשוב לתכלית גאונה / ובצאתי מארצי / וממכון



רבצי / הוכיחתני לשון שכלי על הרחיקי נדוד / מכל



רע ודוד / ואען ואומר

אל נא עצל לבקש לך / מחיה ושים קנך מלון אורחים

כי לא תהי מוצא מנוחה במ/נוחה ובעליות מרוחים

הבט לשון קדש ושור כי לא / יתקבצו תוכו שני נחים.





ושמתי מגמתי אל אמון מנא / ושכרתי



לי מבנות הים כושית ולא מבנות חוה / שחורה



ונאוה / כברק ממהרת / ועל כנפי רוח דוהרת



ונוהרת / נוסעת בלי רגלים / מעופפת בלי כנפים /



עולה ויורדת / הרה ויולדת / ובמשבר איננה עומדת



/ תמיד יתרוצצו הבנים בקרבה / ויש נשמה לעם



עליה ורוח להולכים בה / בהתנוסס ביד הרוח



דגליה / תתעטף במעיליה / וחמתה בשוליה /



ואם רפתה רוחה מעליה / תחשוף על פניה שוליה /



ותסר צעיפה מעליה / ויהי כי עלינו בגבה /



ונהי כסוד סתום בלבה / ונסע מחורב וארץ ציה /



אל האניה / ותוציאנו לרויה / ונשינו נשיה /



כאלו היבשת / לבשה מפנינו בשת / ונעלמה



למראה עינים / ואין כי אם מים ושמים דומים לימים



שנים / ואין מבדיל בין מים למים / ונבוא בצי



אדיר ויסגר יי בעדו / והספינה צפונה בלב הים



וכבדו / והרוח כרכיל יגלה סודו / והמשוטים



חלוצים נחוצים / ודומים לשני חצים / כברקים רצים /



והארן והתרן / כמוץ יסוער מגרן / והקלעים /



נקרעים / וידי המלחים / כבלואי מלחים / והגלים /



יתגלגלו כגלגלים / עד קרבנו לנוה מנוחנו /



ושובבה רוח השלום רוחנו / ויראו אפיקי ים לרב



החובל / ויגלו מוסדות תבל / והמגדל אשר לחוף



הים בצפיתינו צפינו / ובין מגדול ובין הים



חנינו / ויצאנו מעבודת הים הקשה / למנוחת



היבשה / ושוררנו לאל שירה חדשה / על גאלתינו



/ ועל פדות נפשינו / ובאנו לעיר נא אמון ללטש



עינינו בציצי צמחיה / וגרש ירחיה / וכפתוריה



ופרחיה / והעיר ככלה כלילת יופי / בלי דופי /



ושם קהלה נכבדת אך מצאתי בהם איש מחפיר



ומבאיש היה לי משענת קנה רצוץ / חשבתיו הדס



מצאתיהו נעצוץ / והוא מכל חכמה נעור ורק / כגפן



בוקק / והחכמה מאין תבוא לבני עוננה / או מי נתן



לשכוי בינה / תדע עינו בהונו / לתת שלום



לשכנו.



לשון כוסו דבק / לחכו בצמא / ושפתי שלחנו / עטופים ברעב



להסך את רגליו / יפחד פן יצמא / וכן יירא לבדוק / נקביו פן ירעב





ורוע צורתו / תגלה לך נבלותו / וכל



מום אשר יעלימנו / ממראה פניו תשרנו / מבית



ומחוץ תצפנו

ויש אישים מתוקי ניב ושפה / ובלשונם יפתוני וניבם

יגלו לי פני רצון ואהבה / וידעתי הכי ארבם בקרבם

ואיך יוכלו צפון רוח לבבם / והכרת פניהם ענתה בם.





ואם אאריך בו אמרתי / אחלל כבוד



מליצתי / ואשים לאל מלתי / וכאשר רתמתי מנא



אמון מרכבת הפרידה / לקחתי טוב זכרה לצידה /



ובאתי באניות שיחור / נוסעות על יד היאור / בין



גלות הגלים כעב תעופינה / ותבאנה אל קרבנה /



ולהם מן המשוטים כנפים ככנפי החסידה ולא



שקטנו לדאות / בכנפי האניות / עד קרצה לנו



מצרים בעיניה / וגלתה לנו צעיף פניה / ויצאו



לקראתנו בשמיה / במרקחות סמיה / וכאשר קרבנו



לנוה הדומה / שאלנו בלשון התשוקה לשלומה /



אף היא תשיב אמריה / בלשון רוחות שחריה /



הנושבות מעבריה / ויהי בבואנו העירה תמהנו



בנועם בניניה / וחסן ארמוניה / וערוגות גניה /



וריח שמניה / ונעבור אל עבר היאור / לנוה הזוהר



והאור / אשר שם היה מעמד משה איש האלהים /



והשתחוינו שם לפני האלהים / ונחנו שם ימים



שלשה וביום השלישי שמנו אל העיר מגמת פנינו



/ וקרענו בפוך יפיה עינינו / והנה כלה זיו



וזוהר / כעצם השמים לטוהר / כאלו העולם חשך



והיא מנורת המאור / כלה מקשה אחת זהב טהור /



והיא נחלקת לשתי מדינות / עדינות / הראשונה



מצרים הנסוכה / והשנית עיר המלוכה / ובשתיהן



מבני הגולה קהלות / לאל בלבם מסילות / ובתוכם



הכתם הטוב / הנכתר בשם טוב / רבינו מנחם ירום



הודו אשר במדותיו הלשונות תקצרנה / וכל



תודותי [ואם] תארכנה / במהללו תמעטנה /



ותהלותי ואם מים עד ים תלכנה /



רק ביאור תשארנה / כי אותו ראיתי נדיב בדורו /



ואין לאל ידו / ויתחנן כשואל / לכל שואל / וכל



אשר יתנהו / מדם הלב יקחהו / ומחלב כליות



יתיכהו / ומבין שניו יוציאהו / וישמח לבו



בתתו / כשמחת השואל בקחתו / ונדבתו לא



לשכר תהלה / ולא לבקש גדלה / רק לנפש יקרה



אשר בו / ורוח קדשה אשר נתן אלהים בלבו /



ובאמת ראיתי פניו כראות פני אלהים.

יום לבשו שמי ענן איד גם / מבול דמעי בא להשחיתו

היה מאור פניו כמו קשת / בענן זמן נראה להחיותו.





ובעוד אני תמיה לטובותיו אשר קדמו



אותי / והנה כפיר אריות שואג לקראתי / מקולו



יחתו כפירים / ומשאגתו ימס לב הגבורים / הוא



גור החכמות / שחל המזמות / רבינו אברהם בן



אדונינו נזר גליות כל ישראל רבי משה ז״ל גם הוא



גבר עלי חסדו / ויאצל עלי מכבודו / ובחדר אהביו



הביאני / ולוית חן לראשו שמני / ומה יוכל לשון



העט לחבר / וחיליו לגבר / לפני אדון המדות



הנכבדות / אשר לו על החכמים עשר ידות / ואם



תשאל היקר האתה אדון / יאמר לא כי עבד אברהם



אנכי / ולא קנני וסגלותיו / אבל ירשתי



מאבותיו.





אם היקר אדון לכל אדון / הוא באמת עבד לאברהם

מיום כבודו נעלם מני / אלך משוטט בעדו נדהם

כמהו סעפי לו בעברי א/רץ נוד ואתי יעבור כמהם.





אחר הדברים האלה יצאנו ממצרים /



לראות בטוב ירושלים / ונפן ונעבור דרך



המדברה / בארץ ציה וצרה / ובחרבוני קיץ עצמי



חרה / וקראתי שם המקום ההוא תבערה / כי שם בני



ימים מקריבים אש זרה / ותהי נפשי ברשפיהם



שרופה / ותנים ובנות יענה יבכו את השרפה /



ונלך במדבר הגדול והנורא ההוא מקום דאבון



נפש וכליון עינים / נחש שרף ועקרב וצמאון אשר



אין מים / אבל שמנוהו בזעת אפינו / ודמעת



עפעפינו מקום נהרים / יאורים רחבי ידים / ויהי



בבואנו עד עזה / אחרי יגיעה קשה ועזה / ערכנו



לאל תהלה בתוף ומחול / על אשר עברנו מעבר החול /



ויצאנו מן הקדש אל החול / ולא חדלנו לרוץ כנחל



שוטף / וכברק חוטף / יסתירנו כנף האפלה /



ויורנו הנתיבות אישון לילה / עד נפתחו לנו



שערי שמים / היא עיר יי ירושלים / ונקוד



ונשתחוה לאל עילום / אשר פקדנו בפקדת שלום /



ונחנו שם חדש ימים / ובימים ההם חבלים נפלו לי



בנעימים / ימים כאלו מפנינים נחצבו ומעצי



חיים נקצבו / ומככבי שחק גנבו / והיינו



מתהלכים בכל יום לבכות על ציון המתאבלת על



בניה / עד כרו יבלי הדמעות שיחות בפניה /



ולקונן על הריסות ארמוניה / ושארית בניניה /



ולסובב על ציוני הצדיקים אשר סביבותיה /



ולספוד על האם ועל בנותיה / ובקצת הימים



תמשכנו יד התשוקה בעבותים / לעלות אל הר



הזיתים / לערוך תפלות לעושה נפלאות /



להשתחוות למלך יי צבאות / והיינו בעין דומעת



/ ונפש נכנעת / משקיפים על מקום העזרה / אשר



נחלף לעבודה זרה / והיינו מסתירים פנינו מהביט



לנוה הכבוד והגדלה / אשר היה שם אהלו בתחלה



/ אשר מקדם שכנה בו שכינה / ועתה שכנה עליו



עננה / ולא נשאר מן הבנינים ההם / כי אם אותות



יכאיבו לב זוכריהם / וידאיבו בבכי עין רואיהם /



ובמלאת הימים האלה שמנו מגמת פנינו מדברה



דמשק / וקראנו שם הזמן עשק / כי התעשקנו



עמו / ובבואנו עדיה ראינו עיר יפהפיה / בנויה



לתלפיה / דרכיה דרכי נועם / וכלה יפה ושרת



טעם / דשנה כגפן פוריה / בלחי שושניה / ועיני



מעיניה / שוחקת ובוכיה / וכל נאותיה ערוגות



הדסים / וכל יעריה גנות ופרדסים / כמים לים



מכסים / על דרכיה נחלים נטיו / ובלחי הרריה אש



ושלג ישליו / ומצוף שפתיה צמאים ירויו / ובכל



המסלות / תעלות / ועל הדרכים / למים מהלכים /



ועל הנתיבות / מראות השואבות / וריחה כריח



לבנון / וסביב לה תבור וחרמון / ובקיץ ובחרף



תשלג בצלמון / ובתוך העיר מבני הגולה / קהלה



גדולה / ובראשם יחיד הדור ופלאו / ממזרח שמש



עד מבואו / נגיד הזמן ונשיאו / ידיו ים נדבות /



ופניו בקר לא עבות / השר הגדול הרופא ר׳ משה ירום



הודו.

שואל בעד משה פאר שרי זמן / צורי סלעים דשאו מנוזליו

מושל אשר משל בראש שרים אבל / נכנע לכל דורשיו עדי הם מושליו

נפשו לאל מכר ועיניו אל ידי / שואל כעין עבד אלי יד גואליו



מה נואלו דורשיו אשר חשבו הכי / הוא יגמלם [חסד] אבל הם גומליו



יתן ויתנדב כנדבת ענני / שחק ויתחנן תחנת שואליו.





הוא לבדו מבני עירו כבדני / וזבד טוב



זבדני / אין בם אנוש יעטה סות המהללים כמדו /



יראו קרוב לנדבה ונגש משה לבדו / וכמעט אבדה



נפשי בעניה / לולי מצאתי בם איש שכל בפעלותיו



/ ונחמד במדותיו / האח הנאמן / צנצנת המן / נטע



נעמן / ואינו מבני המדינה / רק מבנות עדינה / הוא



עזרא עלה מבבל / להאיר פני תבל / ומצאתיו בים



יגוני רב החובל / והשכיחני במתק חברתו החמודה /



לענת הפרידה / והגדיל כבודי והודי / לאיים



הרחוקים אשר לא שמעו את שמעי ולא ראו את כבודי /



צור ברחמיו ישלם גמולו / וירצה פעלו.





כוכב משחקים ירד / החריד רום והרעיש

מצבאות מרומים שרד / לשכון בין בני איש

לבו אש בעצם ברד / וגויה כתרשיש

בו מדות ובם עם אל רד / מדות עיר וקדיש

מסוה חן ותום על עיניו / וכנף הוד ינופף

בשתים יכסה פניו / בשתים יעופף.





עזרא בזמן הוא אחד / אך נחשב קהלה

כי התחברו בו [יחד] / שכל עם תהלה

ובסוד אל כבודו תחד / עת יערוך תפלה

ובמעלות יקר התאחד / ויעלה תחלה

מגדל עוז יחוה גדלו / עת עליו יחופף

יריע ומהוד קולו / הגלגל ירופף.





ללחום מלחמות התורה / פיו חרב לטושה

ותהיה בעט תפארה / על לבו חרושה

ובשפה ברורה יקרא / ונעימה קדושה

קול ענות אבל בגבורה / אך לא קול חלושה

כל הומה ביגון נאנח / או על לב יתופף

בו ישמח וזעמו יזנח / כל נוגן ותופף.





עת יעטוף לאל כשואל / אמרי פיו לרצון

ונקיה תפלתו אל / אל ממום ולצון

היא עולה כעולה לאל / טוב מקרבנות צאן

לבו זך בלי מתגאל / כפנימי כחיצון

על לב דוד בחילה יפעה / טל אורות ינופף

ובלב אויביו הוא אפעה / עם שרף מעופף.





ועתה קצרה נפשינו ובחיים זנחנו /



כי באשר נחנו לא נחנו / ונוסעים אנחנו / לנוה



החמודות / ומקום המדות הנכבדות / היא ארם צובה



אשר שם גלגל הגדלה / ונקדת העגלה / הוא השר



הגדול מופת הזמנים / וראיה לאחרונים על



הראשונים / מנהיג העדר / וראש הסדר / יוסף השליט



על הארץ / המשביר לכל עם הארץ.



שאו שלום לגביר / חמודות בין אוסף / תהלתו מפז / יקרה וכסף



זמן לספור מקצת / חמודותיו כוסף / ובקש לו מליץ / היות סודו חושף.





ואת יהודה שלח לפניו אל יוסף שמש



תהלות עלה ממערב / ויחן במזרח ויהי שם לגוי



גדול ועצום ורב / בנה אוצרות החמודות / וימלא



אותם בר התעודות / ויצבר יוסף בר / ובא לערוך



שלחן במדבר / לדעת לעות את יעף דבר / הרוה



צמאים / ורפא נכאים / והשיב לב אבות על בנים /



ומחה דמעה מעל כל פנים.

בך הוביש אלהים ים דמעות / ובך אמר לצולתם חרבי

בלבך בית זבול בנה לשבתו / לשון קדש ויהל שם ערבי

והיית גביר ורב במערב / ובמזרח משחך אל לנביא

ואמנם כי ארם צובה כמקדש / ואתה בו כמו נר מערבי.





בך חדש הזמן אחרי בלותו נעורים /



ורחץ במים טהורים / ופשט את בגדיו ולבש בגדים



אחרים / וביום טהרתו לקח לו שני תורים / תור



התורה ותור המשרה אשר בך נחברים / ובשניהם



נאוו לחייו בתורים / צוארו בחרוזי השירים.

השר בתוך פיהו לשון זהב וא/דרת תהלתו מאד נאדרת

בזז חמודות בין וגנב לב מתי / שכל ולא נמצא בתוך מחתרת

ובעת פלילים חפשו תוך אהלו / מצאו לשון זהב והאדרת.





אשרי האיש אשר ילדהו / וצרור המור



אשר צנפהו / גבר על בני דורו / וכבן כהורו /



וכשרש אמירו / ואין למי אמשיל אחד מהם ואשונו



/ כי ממדות בני הדור יוסף איננו / ושמעון



איננו / הראיתם אשר בחר בו יי / זה סיני / אבן



יקרה לא הבין אנוש ערכה / אלהים הבין



דרכה.

היה בארץ מערב אודם בתוך / הבוץ ואין איש סוד יקרו חוסף

עד כי ידי היקר לדרשו מהרו / וימשכו ויעלו את יוסף.





הוא כולל המדות החמודות / ובסודות



התעודות / לו עשר ידות / ובסוד הרפואות / יגלה



נפלאות / ובחכמת המדות / בנה בית המדות /



ובשאלות המנין / משיב כענין / ובנתיבות



הגלגל / יפלס כל מעגל / ובתעודות ההגיון /



יסכל כל רעיון / ורזי התורה / יוציאם לאורה / וסוד



הדקדוק / על לבו חקוק / ויפתח בגמרא / כל דלת



סגורה / ובסוד המשנה / מפיו דעת ותבונה /



וחכמתו כבדה בת מלך / ובבואה לפני המלך /



תשואות חן חן לה / ויבהל את תמרוקיה ואת מנותיה



לתת לה / וארוחתו ארוחת תמיד / ומנורת שכלו



באהל מועד העמיד / להעלות נר תמיד.

ואחריו החזיקו בני אלהים וקדושי



עליון / תומכי הדגל הנעלה / והם בני המעלה /



אשר אל גויות המשרה הם נשמות / ולככבי



התהלה מרומות / ובראשם הרב הנכבד המעלה רבי



משה / אשר בו ינגן המנגן ולא יחשה.





יד כל רודפי המעלה / אזלה / מהשיג ומשה עלה





מעבדיו / עמוסי טובות

הם עדיו / ליחס אבות

גם ידיו / כים לנדבות

כל הבא ופניו חלה / העלה / מי זהב ודלה דלה.









קרבתו / לדודיו רצון

אהבתו / למר לב ששון

נדבתו / כליל היגון

כמנורה וזהב כלה / גלה / על ראשה לשם ותהלה.





רוב דעו / כרוב ענותו

כצנעו / ישו גאותו

הכנעו / כפי מעלתו

אם נפשו לעפר שפלה / גדלה / היא יורדה והיא העולה.





זך מעלל / נתיב טוב סקל

עת מכלל / הדרו נשקל

כל מהלל / לערכו יקל

ולנפש ברום לה מעלה / נקלה / מה נעשה יקר וגדלה.





מושך עול / אמונה וברית

תום יסבול / ועול שוא יכרית

הוא אל כל / תחלה אחרית

אך סוד מהללו נפלא / נעלה / כי לו ראש ואין לו תכלה.





האדיר אל / במשה משרה

דניאל / כפיר התורה

יוסף אל / חשכים אורה

לו אור זהרם באפלה / נגלה / חשך סר ושחר עלה.





תמת בעון אללה תעאלא



ברוך נותן ליעף כח ולאין אונים עצמה ירבה

משקל

שמעו זאת ותבינו / ותנו לאל שירה

שירו לו בתשבחות / ובכנור וקול זמרה

תישר איש לאלהיו / יזמרו לאל הנורא.

תם.
