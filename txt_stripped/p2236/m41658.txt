


	
	[אני גבר בסבות הזמנים] (שיר חתימה לספר מעלות המדות)
	



שיר חתימה לספר מעלות המדות (קטע)

̮̮ _ _ _ ̮ _ _ _ ̮̮ _ _


אני גבר בסבות הזמנים / שבע רגז שבע בוז עם-קלונים

ודלקו אחרי צרות מצרות / ושד על-שד ועל-שבר חרונים

וצללתי בעמקי הדאגות / ועלי עֽברו גלי יגונים

וטרם אמצאה מרפא לצירי / בצדם צֽמחו האחרונים

להותי אמת שחקו צעירים / למו ימים ושועלים קטנים

מאסתימו ותעבתי אבותם / לשיתם שוֽמרי עזים וצאנים

ותבחר מחנק נפשי ומות / ראותי בוז ולעג שאננים

בשמריהם מאד שוקטים ויושבים / בנויהם דשנים רעננים

אשר אמרו לנפשי אין תשועה / ולא חלק ביה רוכב עננים

נגינתם אני שבתם וקומם / וכנורות ועוגבים ומנים

והייתי למשל וֽשנינה / לשתוקים אסופים וֽנתינים

וכל-רואי הכי שממו ותמהו / לקורותי ויודעי מלפנים

ויאמרו לי חלציך תשנס / כאיש גבור ברב-כח ואונים

וחץ בינֽתך תורה בלב ה-/זמן בגד בך בגד כמונים

חנית חכמֽתך הכה בקרבו / וחרבות שכלך הרק וצנים

וזמן שכלך תמיד לנגדו / ותתלונן בסתר דר מעונים

ותנוח ותשקט מזמנך / ותנפש לך מדאבונים

ושמעתי למצותם למען / חשבתימו לרעים נאמנים

ולקחתי לנפשי תוֽכחותם / והיו לי יקרים מפנינים

ואמרתי בלבי אֽלכה נא / בעקבי בעלי שכל נבונים

ואוכיח לנפשי בתחלה / ובי יתיסרו שרים ורוזנים

ואחרי-כן איסר להמון עם / ואוכיח פתאים עם-מבינים

ומוסרי לנפשם יערבון / כטל יום חם ועת שלג וצנים

ואף אמנה חמסי על-זמן רע / ובו אריב ואשא רב תלנים

דלקני בילדותי ורדף / עדי בחרות ונגע עד-זקנים

ושלח-אש בעצמתי ותיקד / שאול תחתית והרגיזה אדנים

והניר לחכה גם דלדלה הר / וסכסכה לעצים ואבנים

הכי בני יצאני ואינם / לנכרים נחשבו-לי גם לשוטנים

עזבוני ונותרתי לבדי / וככלי ריק חשבוני כעוינים

ולא-אבו שמוע לי והלכו / במועצותם לאחור לא לפנים

לזאת אספד אקונן ואיליל / ואבל אעשה כבנות יענים

לקול בכיי אעורר כל-מקוננות / וכל-יודעי נהי עם-כל-מקוננים

ואתביש להרים ראש למען / כלמה כסתה פני כשנים

ועם-כל-זה העירני לבבי / ישיבוני סעפי הנכונים

ויאמרו לי חמל על-הפתאים / להסיר מפניהם כֽשלונים

וחברתי יסוד ספר יסדתיו / באגדות אמתיים וכנים

במקראות ומשלי החכמים / וחידותם גלוים עם-צפונים

חצובים ממקור בינה והשכל / במו-דע מיסדים נכונים

כללתיהו בכל-חכמה ומוסר / להוכיח לבנים ההגונים

להגן בעדם מן-הזמן מר / למתוק שם ומתוק ראש פתנים

לבלתי ימצאום נגע ומחלה / בחייהם ואף-לא קול שאונים

ואחריתם בעדן גן אלהים / בצד אבות יהו שוכנים וחונים.
