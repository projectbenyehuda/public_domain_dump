


	
	[זעומי אל ומנואל חברם]
	




ں – – – ں – – ­­– ں – –



זעומי אל ומנואל חברם / ועוזריהם ואברהם גבירם

וכל־בוז עם אשר לו יד ברשעם / והקרב לחרחר ריב עבורם

אשר תעו והתעו את־עדתם / והחשיכו בעב רשעם מאורם

ונגעה עד נשיא הדור לשונם / וכמעט זרקו בו כוס מרורם

כאחד הם נשותיהם וטפם / וקנינם וכל־הונם יחרם

ויתלקח חרון אף האלהים / בעצמיהם ובבשרם ועורם

ברום מבצר מעונם יעלה צר / וצורר יחנה במרום הררם

עצי פרים בקרדמות ישחת / במזמרות גאון רמי אמירם

וכרמי נעמנים יטעו אך / לגוי אכזר וזר יהיה בצירם

ושדות יזרעו אך בתבואות / לגרנו יאסף נכרי עמירם

גדוד אויב רכוש חילם ישדד / וכל־מקנם וצאנם אף־בקרם

וכלחך שור ירק שדה ילחך / יבול עצם וכל־דשא חצירם

שבי חרב ושד ירב בארצם / וסף־רעל שפוכה על־מגורם

וכל־אלות יגורהם אשר צור / ביד צירו לישראל מסרם

מנדחים יבקשו בפתחים / בחסר כל כסותם עם שארם

רעבים מבלי לחם לשבעם / צמאים מבלי מים בבורם

יחפים מבלי מנעל לרגלם / וסות בלוי להתכסות בקרם

בשבתם יצמחו תוגות בלבם / ויכשל בלכתם חוץ אשורם

אשר הם ימאסו יבחר אלהים / וימאס בחרי אפו בחירם

יאמץ עז משנאיהם וכדמות / גדי עזים ישסע גו אבירם

ורשעם ועון בצעם ילהטם / ודלקה בם לשון אש מנחירם

ויומם יחשך שמשם ולילה / מאור אש יאפיל נגהו בנרם

דצ“ך עד”ש באח"ב ידבקו בם / לכלות מגדולם עד־צעירם

ויחוגו וינועו נבכים / כשכורים אשר יין עברם

וכל־חלי ומכה בם תקנן / ואין צרי למכאובם וצירם

עדי תדבק לשון חלקם לחכם / ותמקנה בנות עינם בחורם

ונכד אל־יהי להם ולא נין / הכי ישבית אלהים את־מקורם

וחטאם לצמיתות ימכרם אל־ / מצוקות יום ואין נותן שכרם

ואיכה יתנו כפרם וממכר / עדי־עד חטא חדל שעור מכרם

ובעבור הם בנכליהם אחיהם / זמנם כן בלי חמלה מכרם

זמן בם יעשה יום יום שפטים / וירבה באשר יפנו מגורם

וחציו השנונים בם יפלח / וישברו ואין חובש מזורם

ונפלו לעפר מות חללים / ואכלו עוף וחיה את־בשרם

ופגריהם בקברם יבאשו עד / אשר ילאה שאת באשם עפרם

כחכם בם וכסכל יסופון / ויכרתו כדלם כעשירם

אשר חפשי יהי בהם ידקר / ולטבח זמן יוציא אסירם

ומימיהם ולחמם רוש בלולים / וסם מעץ בקלחתם וסירם

היבכה איש וכף יכה במותם / והמה גֽֽרמו בכים לדורם

ומי לא יֽהיה כיום בעוכרם / והם עכרו אשר הונף והורם

ואם הם מאסו חכמה היחדל / חכם־לב מערות נגע שארם

זבול יגיל ויעלזו ערבות / להעדרם וישאו קול זמירם

ושבעה כוֽכבי לכת כאחד / בחבה יחגרו חמדת אזורם

ומזרות ברן יחד זמירות / יריעון מול גאון נצחם וצורם

וצור ישמח להמסרם ואבדם / הכי נחם אשר ככה יצרם

ותפצחנה ברנה פה אדמות / וההרים יכסם סות הדרם

לחלפם בעלי כנף ישישון / ועל־ראשם ינוצץ הוד אוירם

בהמות עם המון חיות בשדות / בנפשותם לנצל זיו פארם

ותוך סיר נחלי גפרית יענו / ולא יחסר מדורתם ואורם

כקרח סף יהי סופם להכרית / ואחריתם כדתן ואבירם.


(?1519)
