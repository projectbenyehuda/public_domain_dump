


	
	ויאמר השכמתי בבוקר: מקאמה
	








משפט היין







– – – ויאמר: השכמתי בבקר ואצא לדרכי / לעשות חפצי וצרכי / ובעודי מתהלך ברחוב המדינה / אנה ואנה / מצאתי צבאות בני אדם כצבאים רצים / ועם רב מכל פנה נקבצים / ושני אנשים עברים נצים / האחד זקן יאמיר באמרתו המזמות / והשני עלם יגלה במליצתו העלומות / זה אומר: ׳תן לי פקדוני / פן אקרע לבך בקלשוני׳ / וזה אומר ׳תשב גזלתי / פן אכלה בך בני אשפתי׳ / וכאשר גלו האיבה / וגדלה המריבה / ארו להם העם: ׳עד מתי אתם נרפים / ופוסחים על שתי הסעפים? / עשו ביניכם פשרה / או לכו עד השופט אשר בשערה׳ / ויהינו ללכת אל השופט / ולב כל אחד מהם בחייו נוקט / ואומר בלבי: ׳אלכה ואחזה / המראה הגדול הזה׳ / וכאשר עד השופט הגענו / ובמכון כסאו פגענו / נגש הזקן בערמה ויתמרמר / ויען ויאמר: ׳בי האדון / אשר בין עצומים יפריד וידון / יש לי עם זה הבחור פקדון / ברוך הוא וברוך טעמו / והשמחה על שכמו / וכבר נשבעתי בזהר צלמו / וזכר בשמו / לא אזכרנו ולא אדבר עוד בשמו / אבל אספר לאדני קצת מהלליו / אולי תכירנו ביפי פעליו׳.





דע כי היתה לי עלמה תמה / אדמה אימה
/ ברה כחמה / רקה – צוף למלקחים ועינה – רצפה במלקחים / בת יושבת בפתח עינים / לא תתעלם מרואה / ואם היא צפונה בכרמה בכלאה / ולא תסתיר מצפוניה / ולא תעלים ריח שמניה והכלמה
לא כסתה פניה / דלה ודקה / ליגון תחליף רקה / לחיה כאדם השושן ועינה כדם אדם / ובה שמחת אלהים ואדם / תפיל רפאים / ותשים הפלטים / בחרבה פלטים / ותלבב לבאים / ושלחנה ערוך ליוצאים ולבאים / דלה ירקרקת / כאלו היא חושקת / ומעיליה רצופים אקדח וברקת / תרשיע צדיקים והיא מצדקת / מרה ונמתקת / כי רודיה תמיד נושקת / ועליהם מאורה חולקת / הם מגישים אליה והיא מוצקת / בלתה ולא נמר ריחה / ורבו
ימיה ולא כהתה עינה / ולא נס לחה / בקרבה שמחות תבל צפונות / רבתי בגוים שרתי במדינות.





עלמה בזיוה פאתי הארץ

תאיר, עדי תחפיר מאורי ערץ

אחר צבא יגון חנית תשלף, ויד

תחשף, עדי תחרץ חיליו חרץ.

תזקין – ועדנה יום ליום תוסיף, ואם

תבלה – תחדש הוד ותפרץ פרץ

רקה צרי גופות נגופות הוא וציר

נאמן לרפא ציר ולחלים קרץ

אם תעלימנה – ברקחה תהלך

רכיל ואף תקרץ בעינה קרץ.





ואפקידה אצל זה הנער / הסכל והבער /
ויבוא אליה בלאט / ויקרב לה מעט / וירח ריח בשמה / ותפשה ושכב עמה / וצעקה הנערה ואין מושיע לה / ויבא בזדונו עליה / ויוציא דם בתוליה / ותהר ותלד בן נחמד ונאה / יפה תאר ויפה מראה / אדמוני עם יפה עינים / ישיב החשכה צהרים / וסביבו יפיץ זהר / כעצם השמים לטהר / בו גדל משושו וחדל כעסו / ותהיה השמחה מנתו וכוסו / וישם אותו על זרועו / ויתן בו שעשועו / ויכבדהו כאביו / וישם פיו אל פיו / ויקרב עוד אליה ותולד בן / מעטף ברדידי חן / זקן בא בימים / לא ישנה… / וישמח בו כפלים / וישב אור גילו לשבעתים / ויוסף עוד לדעת את הבתולה / כי לא ישקט האיש כי אם כלה / ותדבק נפשו באהבתה… קדשה / ותלד לו בנים ששה / ותשאר נבזית אחר היותה נכבדת / ונמאסת עקב היותה
נחמדת / ותעמד מלדת / וכראותו כי נשארה כנבלי חרש / אחר היות אורה כאור חרס / מהר ויקחנה / וידבר על לבה ויוכיחנה / לאמר: ׳לא טוב עזבך אביך וצאתך מגבולו / כי הוא אדוניך והשתחוי לו / כי אני לנסע מזה המקום צריך / לכי שבי, אלמנה, בית אביך׳ / ותבוא אלי לעת ערב / מתעלפת כחללי חרב / ואמר לה: ׳המחלי תם כחך / או מן החם נס לחך / או נפלה שבא ותקחך / כי מדוע שנו פניך לחושקם / ויראו פני ריקם?׳ / ותאמר: ׳האיש אשר אצלו הפקדתני / הוא אכלני
וכלני / ולא עזב בי אונים / כי ילדתי לו ששה בנים׳ / וכשמעי מקרה העלמה המפארת / ירדה דמעתי כים כנרת / ויהי בלבי כאש בוערת / ועתה הושיעה לבן אמתך ועשית כחכמתך.





ויחרש השופט שעה תמה על חידתו / כי לא הבין הסוד הצפון במליצתו / אבל לפני העם התחזק / וירחץ פניו ויתאפק / יאמר: ׳אם דבריך נכונים / וטענותיך נאמנים / יתן כמהר הבתולות השוכב אותה ויענה / והאשה וילדיה / תהיה לאדניה׳.





ויגש הנער / ויאמר: / ׳בי אדני / שמעני / ואחר תצדקני / או תרשיעני / אדני
יודע, כי מלך התאוה יפתה בני איש בשקריו / ויצר לב האדם רע מנעוריו / וכאשר הפקיד העלמה אצלי מצאה חן בעיני / ושגו בה רעיוני / ואשאל אותה על מספר ימיה / לדעת היש לאביה ממשל עליה / ותאמר: ׳מימי המבול נולדתי / ובחיק נח נאמנתי / ולבנות לוט בהתיעצם התחברתי / ואני זקנתי ושבתי׳ / וכשמעה ידעתי כי היא משכלת / ועל נפשה מושלת / ואמר לה: ׳אם טוב בעיניך, הבת הקדושה / אשר לרואה מבישה / שתהיי אצלי אמונה / ארשתיך לי באמונה׳ / ותאמר ׳בצלך אחסה / וכל אשר בלבבך עשה׳ / ולא לקחתיה אך כמשפט הנשים / על יד חפה וקדושים / ותלד ששה בנים /
החליפו לי כח ואונים / ויצא הראשון אדמוני / ואחריו יצא אחיו אשר על ידו השני / הראשון / האיר לעיני כל אישון / וצוה לפקח לו האישון / ולערוך שיר עורר הלשון / והשני / דלק אחר צבא יגוני /
והשלישי / שלח סוד לבבי חפשי / והרביעי השכירני ברקו עד חשבתיו שביעי / והחמישי / האיר נגה שמשי / והששי / הפיל לארץ ראשי / וכראותה כי פתתני והשיאתני / עזבתני באהבתה חולה / ותרץ הנערה ותגד לבית אביה כדברים האלה / ועתה, אדני, איך אעזב מאור עיני / ואתן לו משוש רעיוני / את אשתי ואת בני? / והנה עשק עשק (?) זה (?) ומעל מעל / על האשה אשר לקח והיא בעולת בעל.





אמר המגיד: וכראות השופט כי טענותם
חזקות / וחידותם עמקות / וכרחוק מזרח ממערב ממנו רחוקות / נבהל רעיונו / וידבק לחכו לשונו / ותעל חמת קצפו באפו / ויטרף נפשו באפו / ויושט כפו אל צרור כספו / ויוציא שבעים כסף / ויאמר ׳כרתו באלה המריבה והקצף!׳ / ויחטפם הזקן במרמה / ולנער לא נתן מאומה / ויאמר: ׳חצים חלקי מן המתנה / וחלקך –
מהר העלמה הנאמנה / מהר לקחת אשתך אשר לקחת / וראה חיים עם אשה אשר אהבת׳.





וכשמע הנער / גדל עליו הצער / ורעיוניו נשערו שער / ויהמו מעיו / וירדו
פלגי דמעיו / ויגש אל השופט בעין נוטפת / ודמעה שוטפת / ואנחה שורפת / וערמה עודפת / אש בקרבו – ומראה כי לבו חמרמר / ויען ויאמר:





בי אדוני / הנני ואגלה לך סודי / כי נהפך עלי למשחית הודי / דע כי העלמה אשר לקחתי לי לאשה / אשישה השמחות ללב מחישה / והילדים הנחמדים / גביעים משקדים / ועתה מחירה יקח לו למנה / ואחלק עמו במתנה / ויעמד שמם מחכה / ודמעו מעיניו מפכה / עד המו רחמי השופט עליו / ויקרא אליו / ויוציא מתחת בגדו שלשים כסף ויתנם אליו בלאט / ויאמר: ׳קח אלה, ואל תשובו לעמד לפני במשפט!׳





וכאשר פרשו לו מצודותם / ונלכד בשחיתותם / והאכילוהו לחם קצף / וענשו אותו מאה כסף / יצאו מאתו בששון שלם / ואצא אחריהם מתעלם / וכאשר הרחיקו, הגדילו שמחה ושחוק / עד נשמע קולם מרחוק / ואקרב אל הזקן ואמר: ׳השבעתיך, באשר שם מתק שפתיך כיער /
ולשונך לטוש כתער / וחמת צוף ורוש פתן תער / ומום אוהביך תסתר ומום אויביך תער / ולב מקנאיך לחרבו תער / הבנך זה הנער / אשר הוא בגילו כתפוח בעצי היער?׳ / ויאמר: ׳בני ותאות עיני /
אני יחיד במליצות ואין שני / והוא יגזר עליהם ואין מניא / טרם נקרא לשיר יענה / ועל פינו יסע צבאו ויחנה / ובמלחמותינו ניבנו – חרבנו / וקלשוננו / לשוננו / וקשתותינו – שפתותינו / וחניתותינו – אמרותינו / ולנו הגבורה במערך המבטא / ושם נקלע אל השערה ולא נחטא / ולא ישיגו הברקים אבק מרכבותינו / ולא ידעו מקום תחנותינו / כאלו סוסינו חצובים מלהבה / אחד ירדף אלף ושנים יניסו רבבה׳ / וישא משלו



ויאמר:





נריק דבש השיר בפי אוהב, וצור

חזק נשבר בו וכל הר נחריב

נריק עלי צרים חניתות, ובצור

חרב לשוננו לשונם נחריב.







ויקראו עלי שלומם / ויסבו ללכת למקומם / ואעזבם וזכרם בעדי ערב / וילכו לאהליהם שמחים וטובי לב. – – –
