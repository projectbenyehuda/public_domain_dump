


	
	אגרת עמנואל לרבי הלל
	



זה הכתב ששלח ר׳ עמנואל בר׳ שלמה בר יקותיאל ז״ל לר׳ הלל






זמן הרע תכונת איש גרע / וסבותיו ועתיו לו למוקש

והחכמות מאד רמות, ואתה / תבקש לך גדולות? – אל תבקש!





אל תבהל על פיך ולבך אל ימהר / בלם
פיך מלדבר ולבך מלהרהר / כי יש אלהים
שופטים / על כן יהיו דבריך מעטים / והחכמות רמות
/ והבינות נעלמות / והלבבות נאלמות / וההשגות
דוממות / כגחלים עוממות / ויסעו מיטבתה ויחנו
בישימות / והשכל היה לחרבה / כסומא בארבה /
אשר לא ידע בין ימינו לשמאלו ובהמה רבה / והיום
קצר והמלאכה מרבה / והנפש עוונה בה / היא
שובבה / בערב היא באה ובבקר היא שבה / ותבואתה
מרבה / לאל נכר אשר בקרבה / לעשות כונים
להעציבה / והזמן בוגד והמקרה מוקש / יפל בהם
נפתל ועקש / ואתה תבקש לך גדולות? – אל
תבקש! /

השמר לך פן תוקש / ועתה, מה יקחך
לבך ומה ירמזון עיניך / כי תטריד רעיוניך /
ותשא מרום עיניך / ואל קדוש ישראל / ראש גולת
אריאל / איש הבינים / נחמד הוא להשכיל ותאוה הוא
לעינים / שקל בפלס שירים ומליצות במאזנים /
פלוני נר המערבי אשר בילידי הרפה / אשר עלהו
לתרופה / ומפריו יאכלו כל בעלי שפה / ויחיו
המתים במגפה ואתה חק המוסר פרעת / וקלון
מכבוד שבעת / הגדת והודעת והשמעת / מה
לך כי נזעקת / כל קבל די דנה ידעת / ופערת
פיך לבלי חק / והשמעת קולך למרחוק / כקול
הסירים תחת הסיר / תקילא במאזניא ואשתכח הסיר
/ ותלך דרכים לא סלולים / ברב חלומות והבלים /
והוצאת מפיך מלין / ותלעג ואין מכלים / ודברי
תורה המצילים / הכית לשבעה נחלים / והדרכת
בנעלים / דרך לעבר מגאלים / תרמסנה רגל, רגלי
עני, פעמי דלים / ועתה, אל תפל עליך אימתה ופחד
/ הזכירני, נשפטה יחד / יחדו למשפט נקרבה /
הנה קמה אלמתי וגם נצבה / אם תוכל – השיבני,
ערכה לפני התיצבה / הדרכת בים החכמה סוסיך /
ושפעת התבונה תכסך / או תבטח אל מעשיך / לצאת
ביד רמה / ולבקע ים בחכמה / במטה החמה ובחרב
שנונה / לא באמת ולא באמונה / הגד אם ידעת בינה /
וכל איש אשר מחזה שדי יחזה / משמן בשרו ירזה
/ בשמעו הדבר הזה / ואתה על לבך וכל עניניו /
אשר הוא חכם בעיניו / איך לא הרגיש בעוונו /
כאשר פנה לימינו / ועשה ישנו אינו / וישת מן
היין וישכר / חלל את קדש ה׳ אשר אהב ובעל בת
אל נכר / ויצא בחמה גדולה להשמיד ולהחרים רבים /
ויפלו ארצה מן הצבא ומן הכוכבים / וצלעות
הבית והעבים / ועמודי התוך אשר הבית נכון
עליהם / הסיר ממקומם ושם שיחות תחתיהם / ותלה
יסודי החכמה / ומחקרי תעלומה / על בלימה / לא
דקדק ולא חפש / וחצב מעמודים גדולים אויר
שאינו נתפש / ונטש אמונת העבריים והספיק
בילדי נכרים / אהלי אדום וישמעאל, מואב והגרים /
ודבר מדבר זרים / ואשר תחת מגרפותיהם עבשו /
ומעטה גאוה לבשו / מן גו יגרשו / לא זרו ולא
חבשו / גם נשף בהם וייבשו / היא האגרת
הראשונה / השונה בלשונה / אשר דבריה רשו
ורעבו / עמל כתבו / ודובר דברים תמים יתעבו /
וילכו אחר הכזב אשר אהבו / ואני, בראותי
הקדמתה / שמחתי על יפי נעימותה / ויאמר לבי
בקרבי: ואל בשורה טובה תבוא; וכאשר ירדתי אל
עמק כונתה / והכרתי קלקול אמנתה / עלתה
באשה וצחנתה / והנה ריקה יוצאת / ולה אין
בשורה מוצאת / ועוונה צרור על שכמה / נטשה על
אדמתה, אין מקימה / וגם אנשי המקום אמרו: לא
היתה בזה קדושה / אוי לה לאותה בושה! / ויקרא
שמה: כזבי קטנת אמנה / אשר מכזיב המדינה /
והאגרת השניה / תאניה ואניה / ודמעתה על
לחיה / וכל חכמי לב בצל שדי יתלוננו עליה /
ילונו ויתאוננו / כי אשר לא ספר להם ראו ואשר
לא שמעו התבוננו וכו׳.
