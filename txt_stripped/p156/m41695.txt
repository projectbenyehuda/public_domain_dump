


	
	[הה לזמני כתב שטנה]
	




הה לזמני כתב שטנה / עלי לא־השאיר לי עדנה

בימי חרפי השיר על־פי / רתם רכבו על־פי חנה

כל־השירות מבור שירו־ / תי תקחנה ותמלאנה

מרמס רגלי הם תרעינה / מרפש רגלי הם תשתינה

לו המליכו מלך שירי / חשק שירותי תמלכנה

הייתי שעשועי לבות / וצביות חן בי תגלנה

בי תשכחנה אידן אותי / על־ראש שמחתן תעלינה

ובחצריהם ובטירתם / הייתי קרוא ראשונה

דדי ערבו לצביות חן / ויבאו אל־קרבנה

לא־שגבה עפרה ממני / לא־תבצר עיר ציונה

אם־תסתר במסתרים / האנכי לא־אראנה

לו תחבא אל־תוך הים / משם ידי תקחנה

אם־תגביה כנשר או / בין כוכבים תשים קנה

או כעת במרום תמריא / משם ידי תורידנה

אשפילנה עד־הארץ / עד־העפר אגיענה

אנכי בדרך וזמ־־ / ני נחני אל־תוך גנה

לראות הדודאים לרעות / בגנים ללקט שושנה

שם ראיתי עפרה ברה / ומקטרת מר ולבונה

יום ציר חונן הצורות / אותה מכל־עשק פנה

לא־הוליד על־תבל כדמו־ / תה וכצלמה עוד לא־שנה

יום הלדת הנכבדת / כל־ההרים פצחו רנה

ראשה עליה ככרמל / ושפתיה מר תטפנה

ומאורי אור בשמים / קודרים מול זהר יפעתה

שררה אגן סהר לא־יח־ / סר וערמת חטים בטנה

ולחייה כל־האורים / בם תזרחנה בם תערבנה

ובנרדם לב נרדם יעור / ובניר־דם כאור תארנה

עיניה להט החרב / תתהפכנה אל־כל־פנה

עשתנותי ומזמותי / גם־עיני נכחן תבטנה

משתאות מחשות לדעת / אם־תגרשנה או תקראנה

לא־הבחנתי אך האמנתי / כי הן אומרות “לכנה שבנה”

חמותי ראיתי אור כל־ / יותי כאש כן תבערנה

לא־ידעתי אם אש חשקה / הוא את־נפשי ילהיבנה

או ממרום שלח בי אש / על־עצמותי וירדנה

אמרתי "מה־יפית עפרה / יפעת הודך מי זה מנה

אם־עיניך לי תמחצנה / ידיך אותי תרפינה

חוסי עלי ותני־נא לי / מדודאי חשקך יונה"

כשמע עפרה שש לבה / גם־עיניה בי תצפינה

אמרה "בשתים תתחתן / בי היום נזר הבינה

אחלק גופי לשני מנות / ובחר־לך אחת מהנה

האזור הוא יגבל אותם / בין עליונה לתחתונה

העמק שאלה או הגבה / כי מנתך לך אתנה"

עת שמעתי לא־ידעתי / אם־אשמילה אם־אימנה

עיני יזקו מטרות עז / ותבאנה ותדלנה

אז אמרתי "נגזרתי כי / נפשי כיום לקחה בשנה

לא־ידעתי בלתי היום / נפשי עד־כה לא־הבינה

כי עפרה יד חשקה תקפץ / ובאיש חיקה תרע עינה

ונקלתי עוד מזאת כי / כאלה אותי תקראנה

אך לכבודך עפרה אקח / לי המנה העליונה"

כשמע זאת עפרת החן / אז כליותיה תעלזנה

ותשבע "חי החשק / חי חשקך לבי ענה

יען היתה זאת עם־לבך / כי לנדיבות לבך פנה

חי־אני האזור תחת / כפות רגלי שם אכינה

ומשלת בכל ושתי / הממלכות לך תהיינה"

אז הלכתי עם־עפרת חן / עד־באתי אל אפריונה

אכלתי יערה אריתי / מורה גם־שתיתי יינה

וזמני עתה הלאני / וצביות חן לי תבזינה

לא־תחשקנה בי ובקולי / לא־תשמענה לא־תאבינה

אם אמרתי לחשק עפרה / חושק אחר ישכבנה

לא־תחפצנה מידי ז־ / בח עולתי לא־תרצינה

אצעק חמס לצביות כי / כגמול ידי לא־תגמלנה

לא־נשיתי לא־נשו בי / כי רעה עלי תחרשנה

אתמה על־כי תקצנה בי / כי טמא וכסיל תשכרנה

לא נחלו על־שברי כיום / אלו מתי לא־תבכינה

לו נמכרתי בקשיטה אל / עקר אותי לא־תפדינה

לו אהיה פטר לחמורם / לא תפדוני לי תערפנה

כי הלבין שערי וזקני / מזהב אל־כסף שנה

כי־יצא קצף השיבה / גם־החל נגף הזקנה.
