


	
	הירשה (השער הי"ד מספר המחברות)
	




עמנואל בן־שלמה הצפרוני המכנה עמנואל הרומי



Manoello Giudeo או Immanuele Romano


[נולד ברומא לערך 1272–1268, חי ברומא, פרוג’ה, פֿרמו, פֿברינו, קמרינו, אנקונה וורונה, מת אחרי שנת 1328]





הירשה (השער הי"ד מספר המחברות)






אמר עמנואל המחבר: בהיותי בבית
השר שמן ודשן / שוכב על־מטות שן / ובמעדניו
אשלח השן / הייתי בחוצות השמחות מגמא / וסובב
כל־מעיני הששונים סבוב הצמא / ויהי היום ואני
את־רגלי חליתי / ועמדתי ימים שבעה ופני השר
לא־ראיתי / ויין ושכר לא־שתיתי / וכאשר
התחזקתי ויצאתי מפתח ביתי / ואשא עיני ואראה
והנה אם השר באה לקראתי / וארוץ לקראתה ואמר
“השלום לך גברתי / השלום לשר השלום לילדים
כי באת הלום” ותאמר “שלום / אמנם השר שמועה שמע
וחרד חרדה / וצירים אחזוהו כצירי יולדה / סוער
כהמות הים לגליו / ושנתו נהיתה עליו / ועתה
הדוד הנאמן / החומל עליו והרחמן / למה תעמד
ברחוק ותעלים לעתות בצרה / והיגונים / חונים /
על־השר איש על־דגלו באתת / באה אליו החדרה
ונהלהו על־מי מנוחת / ויהיה־לו בראותו פניך שבע
שמחות / ושא אליו חידותיך המשמחות / יכבו אש
היגונים / ויאירו נרות הששונים”.

ואען ואמר “הנני גברתי” / ואקום קל
חיש נשען על־משענתי / ואלך אל־השר חדר בחדר
ומצאתיו הומה וסוער / ובכל מחמדי הזמן גוער /
ולבו באש היגונים בוער / ועמו חברת מידעיו /
וגואליו ורעיו / ובראותם אותי מרחוק / ובידי
המשענת החלו לשחק / ויאמרו “מי הביאך הלום
ואתה צריך למשענת” / ואען ואמר “אהבת השר
הנאמנת” / ויאמרו “הידעת חידה משמחת / תהיה
אל־השר בשמע אותה נחת / ולא ימות לשחת” / ואען
ואמר “ידעתי ולא אוכל להגידה / ועתה חוסו
על־נפשי האובדה / להכין אתה ולסעדה / הטרם
תדעו כי אבדה / הלא תראו עצמי אשר התדלדלו /
והעגבות התקלקלו / ואתם את־החידה תשאלו /
למה־זה הבל תהבלו / הבו שכרי ואם־לא חדלו” /
וישחק השר ויאמר “ומה־יהיה השכר” / ואען ואמר
“אטף לך ליין ולשכר” / ויצו השר הנערב /
ויבא יין מלכות רב / ואמר “ברוך יי צורי המלמד
ידי לקרב” / ואקום ואשתה יין לשבעה / כי לא
שתיתי ממנו ימים שבעה / ויהי כאשר שבעתי
מן־היין / היו יגוני השר בעיניו אין / ויאמר
“עד־מתי חידתך בתוך חכך תמנענה / חודה חידתך
ונשמענה” / ואען ואמר "האזינו מידעי ואדברה /
וזה־חזיתי ואספרה:

איש היה בארץ רומא משה שמו / רצוי
לרוב אחיו ודרש טוב לעמו / ושם אשתו רושולה /
אשר ירד עליה אבל שאלה / ושם בנו דניאל אויל
שפתים / חמר גרם רבץ בין המשפתים / לא מצא
בביתו נחת / רק פח ופחד ופחת / ברד ואש מתלקחת
/ כי היו מדי יום יום נצים / הנה האש והעצים /
יקומו זה על־זה כהר־פרצים / ויצעק האיש אל־יי
מפני לחצים / כי בנו משך ידו את־לצצים / ואשתו
תורה בלבו מן־הכלמה חצים / והנגשים אצים /
ויאמר האיש בלבו: משבת עוד אתם נבעתי /
פן־תדבקני הרעה ומתי / אין־לי טוב רק להניח
אלה האמללים / ולגלות בראש גולים / אולי יקרה
יי לקראתי / ואמצא מקום תהיה־שם מנוחתי. / ויהי
היום ויקח האיש הונו ורכושו / ויקם וילך
אל־נפשו / ויקר מקרהו בארץ מארצות יון אשר שם
אנשי אמונה / היו עם־השכל באמנה / בשבטי
ישראל מודעת נאמנה / ושם העיר שבמה / וירא
האיש מנחה כי טוב ואת־הארץ כי נעמה / ויטע שם
אהלי אפדנו / וישכח שם צרת אשתו ובנו /
ויאמר “זאת הארץ אהבתיה / אויתיה חמדתיה /
זאת־מנוחתי עדי־עד פה אשב כי אויתיה” /
ויאהבוהו אנשי המקום על־ישרו / ועל־נעם פעליו
וכבוד עשרו / כי כליל הוא בהדרו / ויאמרו
“בא־יבא ברנה נזר הודנו / המשביע בטוב עדינו
/ המחדש כבודנו / זה ינחמנו ממעשנו ומעצבון
ידינו”.

ויואל האיש לשבת אתם / ולבו עלז
בחברתם / ויהי שם כעשרים שנה גדול ליהודים
ונאהב / כבד מאד במקנה בכסף ובזהב / כי
הצליחו האל עושיו / והרויחו בכל־מעשיו / ויהי
היום חלה את־חליו אשר מת בו / ובראותו כי גדל
צירו ומכאובו / קרא אליו אנשי מקומו / אשר ידע כי
הם דורשי שלומו / ויאמר “שמעוני אנשי שלומי /
הנני נאסף אל־עמי / ואין לי יורש וגואל / רק בן
אחד ברומא ושמו דניאל / אחלה לפניכם האצילים
להיות הוני בידכם למשמרת / והודיעו לבני מיתתי
באגרת / ובבאו הנה בעדות מבארת / תנו לו
את־נחלתי הנשארת / ואשר יבוא בלא עדות ומזכרת
/ יחשב כגנב נמצא במחתרת” / ויענו האנשים
ויאמרו “אדננו נזר היקום / כאשר צוית את־עבדיך
כן יקום” / ויהי ככלתו לדבר עם־אנשי שלומיו /
ויאסף רגליו אל־המטה ויגוע וימת ויאסף
אל־עמיו / ויבואו כל־אנשי מקומו לנוד לו ולספדו
/ “הוי אדון והוי הדה” / ויקברוהו במעלה קבריהם
על־צדקתו / וכבוד עשו־לו במותו.

ויזכרו האנשים את־מצות האיש
וישלחו לרומא אגרת / אדות מיתת האיש מספרת /
ואת־כבוד עשרו אשר הניח ליורשיו / ואשר יבא
בעדות אמת כל־דורשיו / ובכן לא ייעפו כל־מבקשיו.
/ ויהי כאשר הגיעה השמועה בעיר / ונגלה הדבר
למגדול ועד־צעיר / נמצא אז ברומא איש אחד מזרע
היהודים / ושמו אפרים רעה רוח ורדף קדים / זולל
וסבא / כל־היום כזב ושד ירבה / פריץ חיות / רע
לשמים ורע לבריות / עקוב הלב ואיש מהתלות /
ולו לשון מדברת גדלות / היה איש בעל נכל וערמה
/ ולבו חורש און ומרמה / גבר במרמות ותך על־כל
אשר היו לפניו / והיה דברו דבר דבר על־אפניו /
ובשמעו השמועה חשב מחשבת רעה / לעשות חנף
ולדבר תועה / וישת עצות בנפשו / לקחת ממון
המת ולירשו / ומן־היום ההוא והלאה רוח אחרת
עמו / ולא־עשה רגליו ולא־עשה שפמו / ולבש
שחורים כאלו מתאבל על־מת / להראות העמים כי הוא
בנו באמת.

ויקח זהבו וכספו / וישם את־נפשו
בכפו / ויעבר מדברות וציות / והוא מתהלך בין
החיות / וירכב אניות / עד־אשר הגיע אל־מחוז
חפצו / ואל־מקום המת ואל־ארצו / ויהי בהגיעו
אל־המלון קרא אל־הנער לימים צעיר / ויאמר "קרא לי
עשרה אנשים מטובי העיר / אשר ידעת כי־הם זקני
העם ושפטיו / אשר יורו חקי אל ואת־משפטיו /
ואמר אליהם: הגיע פה דניאל הנכאב / אשר דמי
כבדו בבכי ישאב / יחפץ לעשות חק הבן על־מיתת
האב / ואחלי לפני משרת מעלתכם / ויקר תפארת
גדלתכם / ללכת עמו עד־קבר אביו קדושו / רכבו
ופרשו / כי שם ישפך עליו את־מרירות נפשו. /
וילך הנער ויקבץ כעשרים איש מזקני העיר
ונכבדיהם / כלם אנשים ראשי אלפי ישראל הם /
וילכו עם הנער אל־המלון אשר שם יחנה / והנה
שם קול המלה כקול מחנה / וימצאוהו בדמי דמעיו
מתגולל / וקולו אילות יחולל / יאמר “אבי אבי רב
מחולל / המוציא יקר מזולל / איך נפלת משמים
הילל” / ובראותו הזקנים קם בענוה ובמוסר / כאלו
בחבלי העצב והדאגה נאסר / ויאמר “אחלי לפניכם
מורי התורה וההלכה / המצליחים למלוכה / עשו אתי
ברכה / ובאו אתי אל־עמק הבכא / והראוני מקום
מנוחת צדיק אבי / ואעשה שם כאשר עם־לבבי” /
ויאמרו אליו האנשים “הננו נעשה כל־חפצך / ויי
אלהיך ירצך” / וילכו עמו ויראוהו מקום הקבורה
/ ובראותו הקבר צעק מרה / ויאמר “אבי אבי רכב
ישראל ופרשיו / קומת ארזיו מבחר ברושיו / רד
עם־אל ועם־קדושיו / מי האמין כי תמות בארץ נכריה
/ ותעזב אשת חיקך תאניה ואניה / בכו תבכה
בלילה ודמעתה על לחיה / אבי אבי כתר ראשי /
הלא קראת לשפך עליך את־מרירות נפשי / מי יעץ זאת
עליך אבי אבי / למה היתה מכתי אנושה ונצח כאבי
/ חנני חנני אתם רעי כי יד־אלוה נגעה בי” /
ויבך האיש בכי נמרץ / עד אשר גבול הבכי פרץ /
כי דבר הממון נתן בפיהו חזון נפרץ / וכאשר
ימנעוהו מלבכות אז יחרץ / ולהכות על־פניו
ועל־חיקו לא ירחם / ושערותיו ימרט ובידו יקחם /
ויקומו כל־האנשים אשר שם לנחמו וימאן להתנחם
/ ויאמר כי נרד אני ואמי רושולה / אל אבי אבל
שאלה. / ויהי כראות האנשים קול בכיתו כאלו
האל זעם / ובקול כמהו ירעם / וכי מלבושיו בידיו
יקרעם / ויחזיקו האנשים בשתי ידיו / לבלתי
יקרע עוד בגדיו / ויזעק האיש ויאמר “הנני נשבע
בנפש אבי / כי אם תמנעוני מקרע בגדי אקרע
לבבי” / ויבך האיש עד־אשר גרונו נחר / וישבע
כי למות על־אביו יבחר / ומעשות כזה יום יום לא
יתאחר / והיה כזה יום מחר / ויפל על־הקבר
מלא־קומתו / וילחך עפרותיו ואדמתו / ויהיו
האנשים בראותם צרת נפשו תמהים / ויאמרו “אין זה
כי אם־איש אלהים” / ויהי ככלתו לבכות בכל־אות
נפשו / ולמרט שער ראשו / ויאמר “אחלה לפניכם
הזקנים יודעי צרת לבבי / לקרא צדוק הדין על־קבר
אבי / אולי יחשך כאבי” / ויהיו האנשים לקרותו
מהירים / והם לא ידעו כי ערם יערים / ועוד יעשה
על־מיתת האיש הזה פורים / ויהי כקרא הקורא
“שומרי גנזי גן עדן” והזכיר שם המת / התאנח
אנחה גדולה להראות הפך האמת.

וישב אל־ביתו אבל וחפוי ראש / בפניו
תחנה הדאגה ובלבו און יחרש / בחשבו כי תחת
הנעצוץ יעלה ברוש / והזקנים וכל־העם עד־ביתו
לווהו / ומלבכות עוד יזהירוהו ויצווהו /
וישאר האיש בביתו מתעסק באבלו / וגדודי
השמחות יחנו בלבו איש איש על־דגלו / וכל־העם
שב איש לאהלו / ויאמר בלבו האיש / רשע מחפיר
ומבאיש / יזכר עון אבתיו אל־יי וחטאת אמו
אל־תמח / “עת לפזר היום ומחר אשמח / הנני עשה
חדשה עתה תצמח” / וישלח ויזבח צאן ובקר /
לעשות לעניי הארץ אורה ושמחה וששון ויקר /
ויחלק לכל־העם למאיש ועד־אשה / אשפר ואשישה
ויאמר כי זה יעשה לכפרת נפש אביו הקדושה /
וישלח למחלקי הצדקה אל־האמללים / כסף שלשים
שקלים / לחלק אותם לאביונים ודלים / ומשלוח
מנות לאלמנות וליתומים / ויעש לאביו אבל שבעת
ימים.

וישת האיש עצות בנפשו / להציב ציון
על־קבר האיש אשר חשב לירשו / לבנות כמו־רמים
מקדשו / בחשבו כי בזה יגיע אל־מבקשו / ויאמר
בלבו אם הוני היום חולף ואוזל / ועמלי ויגיע כפי
אני גוזל / לשלח אותו לעזאזל / תחת הכסף אביא
זהב ותחת הנחשת אביא כסף ותחת האבנים ברזל. /
וישלח ויקרא לחרשים ולבונים / ויתן את־הכסף
בידיהם כאבנים / ויאמר אליהם “אני עם־לבבי /
לבנות ציון על־קבר אבי / לכבוד ולתפארת / ולכתב
מקצה בחברת / בעט ברזל ועפרת / שם הצדיק אבי
ושם אביו ואמו / ואחיו ושער מקומו” / ויאמרו
האנשים “חייך, לא־נשאל ולא־ננסה / ככל אשר
תצוה את־עבדיך כן נעשה” / ויבנו על־הקבר
במצות האיש ציון נפלא / בכתם טהור לא יסלא /
ויקימו עליו מצבה גדולה / לשם ולתהלה / ויכתב
שם האיש בידו / זה המכתב להעלים סודו / “פה
שוכב נזר החסידים / ותפארת הנגידים / משה
בן־פלוני מעיר רומא ובנה זה הציון לשמו
ולכבודו / דניאל הקודר והנכאב בנו יחידו /
אשר אהבו בכל־לבבו ובכל־נפשו ובכל־מאדו /
וימאן להתנחם בעודו / על־אשר לא־היה על־פניו
בהפרדו / לנוד לו ולספדו / הוי אב והוי אדון והוי
הדה”.

ויעמד שם האיש חדש ימים / כאיש
מתאבל על־מת נעצב ומשמים / ולא שאל על־ירשת
אביו לעולם / פן יתגלה סודו וישב דך ונכלם /
ומדי יום יום יעלה בית־התפלה להעלים סודו /
וילבש צדקה כמדו / ותפלין בראשו וציצת
בבגדו / עד־אשר בטחו בו הלבבות / ונפלא בעיני
המחשבות. / ויהי כראות אנשי הקהלה / כי־זה
האיש חרד על־מת חרדה גדולה / ולא סר מלבכות
ומהתאבל / ומלקרא “על־נהרות בבל” / ולא־שאל
ממון המת כאלו לא־הלך שם לקבל / וכי יפזר
כל־רכושו / לתתו לכפרת נפשו / וכי הקים על־קברו
מצבת אבן / לא־חשבו כי לבו יקבץ־און / ויאמרו
איש אל־אחיו “מדוע אנחנו מחשים / והנה האיש
הלזה זרע קדושים / אין צרך לאיש כזה עדות וכתיבה
/ עד הגל הזה ועדה המצבה / ברוך יי אשר
לא־השבית לאביו גואל / ויקרא שמו בישראל /
ועתה נתן לו את־ירשתו / אחרי אשר ידענו
את־קדשתו” / והם לא־ידעו כי נלכדו ברשתו.

וישלחו האנשים ויקראו־לו / והוא
עודנו מחזיק באבלו / ויאמרו לו “אדוננו אשכל
הכפר וצרור המר / אביך צוה לפני מותו לאמר /
הכסף אשר נתתי בידכם לשמר / תנו אותו לדניאל
בני וירש אותו / ובתנאי שיביא ראיה או עדות
מארץ מולדתו / ובעדות זקני קהלו יהיה נעזר /
למען אשר לא־יקרב איש זר / ואנחנו ראינו בך
נוראות / ונגד עינינו עשית על־מיתת אביך נפלאות /
ובא המופת והאות / כי פעליך הנכבדים / ודבריך
הנגידים / ואנחותיך ודמעותיך כמאה עדים /
ובציון אשר על־קברו בנית/ ואת־אשר עליו
התענית / כל אשר לאביך קנית / מלבד אשר
גברת על־אנשי הדור בטוב שיחיך / וכוכבי הענוה
זרחו במצחך / ועתה קומה השר למנוחיך / הנה
כפתוריך ופרחיך / עלה רש / והאנחות מלבך גרש
תגרש / כי אין זולתך גואל ויורש” / ויקד האיש
וישתחו / ועיניו יבכיון ויצוריו ישמחו /
ורעיוניו כעלה יפרחו / ועשתנותיו בגני החדוה
ותרעינה באחו / ויאמר אל־האנשים “יענה יי
שלומכם / ועל־טוב יזכר שמכם / מה־יקר חסדכם /
אשר עשיתם את־עבדכם / ואני מה / ישלם יי
פעלכם ותהי משכרתכם שלמה / ועתה האצילים
והשועים / שלחו מכם אנשים לשבטיכם ידועים /
וישוטטו בחוצות העיר / וישאלו גדול וצעיר / אם
אבי נתחיב ממון / לאיש מכל־ההמון / במכר או
במקח / יבא ויקח / ומי האיש אשר שרת אותו
במחלתו / או התעסק בקבורתו / הנה שכרו אתו
ופעלתו / ומי האיש אשר נתחיבתי לו דבר יקחנו
ממני / בטרם אלך ואינני” / ויענוהו האנשים
“תברך מהאל זרע קדושים / כי כבד כבדתנו / ולא
עשקתנו ולא רצותנו / ודבר־עצב ממך ומאביך
לא־סבלנו / את־הכל התקבלנו / ברוך יהיה אשר
הביאך הלום / אחינו לך לשלום”.

ויקח האיש מידם את־ההון / ויאמר
בלבו אך עשרתי מצאתי און / שמע אלהים קול בכיי
/ הכי קרא שמי אפרים כי־הפרני אלהים בארץ עניי
/ והפך מספדי למחול / כאשר הביאני נחלת אחרים
לנחל / ברוך המבדיל בין קדש לחול. / ויהי כאשר
הגיע הכסף לידו / ירא פן־יבוא שם איש ויחשף
סודו / ויאמר “אין־לי טוב כי אם לרכב על־צי / ולשוב
אל־מחוז חפצי / ואלכה אל־מקומי ולארצי / פן־יבא
בנו האמתי / או ישלח הנה איש עתי / ויוציאו
את־בלעי מפי / ומבין שני טרפי / ואשם על־עקב
בשתי / אם יקחו ממני את־אשר לכדתי ברשתי /
הכסף והזהב אשר לקחתי בחרבי ובקשתי” /
וימהר וירתם פרדי הפרידה / ויקח הנסיעה לו
לצידה / וישב אל־ביתו קל חיש כנשר חש לאכל /
בשמחה ובטוב לבב מרב כל.

ויהי מקץ שנתים ימים / וישם דניאל
בן־המת על־לבו לרכב ימים / ולהוציא את־הכסף
ואת־הזהב אשר הניח אביו מבין קרני ראמים /
ולא־ידע כי אפרים / לקח ברכתו והושיבו בין
תנור וכירים. / ויקח דניאל מזקני קהל רומא
עדות וראיה / כי הוא בן־פלוני אשר מת בארץ
נכריה / ועל־החתום היו זקני קהל רומא ושועיה /
נקובי ראשית החכמה וידועיה / וילך למסעיו ויבא
אל־חוף ארם צובה / וימצא אניה ויתן שכרה וירד
בה.

וילך אחת הנה ואחת הנה עד־אשר
הגיע אל עיר־קברות אבותיו / ובהגיעו שם לא־שאל
על־אביו ועל־אדותיו / ואם נקבר בקברי ישראל או
באשפה / ואם עשו לו כבוד במותו או חרפה / לא־בכה
ולא־התאבל למטוב ועד־רע / לא־מרט את־ראשו ובגדיו
לא־קרע / רק שאל ביד מי הניח אביו הונו / ומקנה
קנינו / כי הוא כחו וראשית אונו / לו משפט
הירשה / אוי לה לאותה כלמה אוי לה לאותה
בושה / ויאמרו “מה־לך הפתי אשר על־ריב לא־לך
תתעבר / ועתה מה־לתבן את־הבר / כי תבוא בנכל
ובערמה / כאיש בעל זדון ומרמה / לשאל ירשת
אנשים צדיקים / כהגלות נגלות אחד הרקים /
ולא־זכרת “ארור אשר לא־יקים” / ועתה חדל־לך
הסכל והנבזה / שוכב והוזה / ואל־תוסף עוד דבר
אלינו בדבר הזה” / כשמע האיש תשובת האנשים
עמד מרעיד / ויוצא אגרת מחיקו ויאמר “הנה זאת
עלי תעיד / כי אנכי דניאל בן־פלוני המת /
ואינני בא אליכם במרמה כי אם באמת” / ויקראו
האנשים את־האגרת / והנה היא על־אדות האיש הלזה
מדברת / ועל־החתום זקני קהל רומא ורבניה /
אציליה זקניה וקציניה.

ויתמהו האנשים איש אל־רעהו /
ויקהלו שם קהלה גדולה כל־העם מקצהו / ויהי
כל־העם נדון / לאמר “האיש אשר לקח הירשה היה
איש מדון / ובא עלינו בדרך מרמה וזדון” / וקצתם
חשבו רעה על־זה האחרון / ויאמרו “מה־מאד גדול זה
העורון / אשר לא התנהגנו בכבדות / בתתנו
הממון ההוא בלא ראיה ועדות / לאיש אשר לא־ידענו
אותו וחין ערכו” / זה אמר בכה וזה אמר בכה /
ויאמר האיש אל־האנשים / “אראה אתכם מתלחשים /
האם נתתם ירשתי לאחר” / ויאמרו “נתנו עבר
לסחר” / ויחרד האיש חרדה גדולה / ויצעק צעקה
כקול המלה / ויאמר “מי הוא אשר הרבה מבוכתי /
ובא עד־הלום ולקח את־ברכתי” / ויאמרו “בא הנה
איש־חיל רב־פעלים לבו בקרבו חלל / בוכה
ומתאבל ובדמי דמעיו מתגולל / ויעט אל־השלל /
שאל על־קבר אביך ופלגי־מים ירדו עיניו / ויקרע
שמלתיו וישם שק במתניו / וימרט שערות ראשו /
ויחלק כמה זהובים לכפרת נפשו / ומדי שבת
בשבתו לטובה אותו יזכר / ספד עליו כמספד
על־היחיד והמר עליו כהמר על־הבכור / בנה ציון
על־קברו מהונו לא־נעשה כן לפנים / ופזר ונתן
לאביונים / ועמד נעצב ונכאב / ומדם כבדו דמעיו
ישאב / ועשה מה־שלא עשה לעולם בן על־אב / לא
שאל לעולם ירשה / רק היה על־מיתת אביך כאבו נצח
ומכתו אנושה / והיו לעולם ציצת בבגדו ותפלין
בראשו / וקנה הון אביך ורכושו / בשערות אשר
תלש מראשו / ובזהובים אשר חלק לכפרת נפשו /
ואת־כל־מקנה קניניו / בדמעות אשר הוריד מעיניו /
ואם אתה בנו כאשר עדים העדת / וכפי החידה אשר
חדת / איה החרדה אשר עליו חרדת / ודמעות אשר
מעיניך הורדת / והמספד אשר עליו ספדת / איה
בגדיך אשר קרעת / וראשך אשר פרעת /
וממרורים אשר עליו שבעת. / ואף אם נאמנו דברי
אגרותיך / מדוע אחרו פעמי מרכבותיך / בא אחיך
במרמה ויקח ברכתך”.

ויאמר האיש “מה־גדלה זאת המבוכה / אין
למכתי תעלה וארוכה / הלא אצלתם לי ברכה” /
ויאמרו “אצלנו לך הברכה אשר בהר עיבל /
ובשרך יכאב ונפשך תאבל / כי על־מיתת אביך לא
תתעצב / ולבקש ירשתו על־מצור תתיצב / ועתה
אמר־נא החלד והעכבר והצב / מי יאמין כי תהיה
בנו / ומי הוא הבן אשר על־אביו לא תחוס עינו / ואף
אם אתה בן־אשתו שמע חרפתך / זנתה אמך
הובישה הורתך / ולכן על־מיתתו לא נכמרו נחומיך /
רק תבקש הונו לבשתך ולבשת ערות אמך / ועתה
אל־תשמע קולך עמנו ותמצא נפש / פן־יפגעו בך
האנשים מרי נפש” / ויאמר האיש “חמסי עליכם אנשי
הקהלה הקדושה / מדוע מהרתם לתת לאיש נכרי
הירשה” / ויענו העם אותו קשה / ויאמרו “ראה
ראינו צרת נפשו / והתפלין אשר בראשו /
ואת־אשר נפל על־קבר אביך והתעסק באבלו /
עד־אשר חשבנו כי לא יחיה אחרי נפלו / לא כאשר
עשית אתה אויל משריש / בן מחפיר ומביש / כי הוא
מלקרא “אבי אבי” לא יחריש / יקרא “אוי ואבוי”
וישאג כאריה / ונתנו לו הכל בטרם תבוא ולא
אמר איה / ונברכהו גם־ברוך יהיה” / ויאמר האיש
“ארור יהיה מן־העולם ועד־העולם / האיש אשר התאבל
על־אבי והקשה אלי וישלם / ולקח את־ברכתי
ואל־ביתי ישיבני דך ונכלם” / ויפרד מהם והוא
בוש ונאלם / והיתה לו חרפת עולם.

וימהר וירכב אניות / וישב אל־ביתו
בידים ריקניות / ותשאלהו אמו על־דבר הירשה
/ ויאמר “הנני שלוח אליך קשה / אין קול ענות
גבורה רק קול ענות חלושה / כי הלך שם איש בליעל
/ ומעל בנו מעל / אל־ידרשהו אלוה ממעל / בכה
וספד על־אבי / ולקח כספי וזהבי / ותלש שערות
ראשו / וחלק ממון לכפרת נפשו / וירש כל־רכושו
/ הקים על־קברו מצבה / ועשה כונים להעצבה / ובנה
אצלו ציון / ולדעת מי היה זה הנבל צריך עיון /
הלכתי שם לקבל הון / והכתי בשגעון ובעורון
ובתמהון / קבלתי חרפה מכל־החרפות מקבצת /
וקללוני קללה נמרצת / שמו לי עלילות / ויבקשו
את־נפשי להקלות / וליחמנה במקלות / ועליך
דברו כאחת הנבלות / ואני זאת נחמתי אחרי אשר
הדביקתני הרעה / כי לא הורדתי אל־אבי דמעה / ולא
קרעתי ולא ספדתי / ועליו חרדה לא חרדתי /
נשאתי אגרת ועדים העדתי / ופרשתי רשתי ולא
לכדתי / וכאשר אבדתי אבדתי” / כשמע אמו
חרפתו וחרפתה / ותרא כי נוחלה אבדה תקותה /
ותסר מעליה בגדי אלמנותה / ותאמר כי מלבכות
עליו גם־היא תחרש / אחרי אשר מהונו לא תירש /
ותהי להם לנחמה / ויחדו הלכו בכלמה".

כשמע השר קול החידה דופק / לא־יכל
להתאפק / ואז נמלא פיו שחק / והשמחה פערה פיה
לבלי־חק / והקול נשמע עד־למרחוק / כאלו נתן
תהום קולו / ואל־הכבוד הרעים מזבולו / וישמעו
אנשי העיר את־קול השר ברעה / וישמעו מצרים
וישמע בית פרעה / ויהי האנשים על־שחק השר
תמהים / כי למבראשונה היו עליו פוחדים ורוהים /
ויאמר השר “צחק עשה לי אלהים / כל־השמע
יצחק־לי / כי נשני אלהים את־כל־עמלי” / ויהי
כאשר־תם כל־העם לשחק לרצונו / ויאמר השר
“הידעת מה עשה אפרים הבוגד מהונו”.

ואען ואמר "זה פתרנו:

אפרים היה עגה בלי הפוכה / לא השאיר
אחריו ברכה / כי היה ברשע נודע / לכל־איש רע
ובליעל גואל ומודע / אכלו זרים כחו והוא לא ידע /
ואחרי־כן התחבר עם־אנשי לצון / ובוגדי בגד יעלו
על־מזבחו לרצון / הרג בקר ושחט צאן / ישתה יין
לשבעה / ויקרב אל־כל־בהמה לרבעה / עד־אשר חלף
יגיעו ועמלו / וישכן באהלו מבלי־לו / ונשאר
במקלו ובתרמילו / כי הגזלן זה־משפטו / ודבר
העושק כה מבטו / כבולעו כן פולטו".

ויאמר השר “אף אם יש לשחק על־זה
המאורע / הוא ענין רע / לבא איש חנף ומרע / חלקו
מחמאת פיו וקרב־לבו / יקרב כתנור לבו בארבו
/ ויקח את־אשר לא־עמל בו / ואחשב כי זאת הרעה לא
היתה הראשונה אשר אפרים עשה / רק נגעו בעור
רבים פשה”. / ואען ואמר "חייך אפרים היה אבן
מעמסה / ופח היה לרבים ורשת פרושה / נשרטו בו
עומסיו / ונוקשו ונלכדו במעשיו / לא תוכל הארץ
להכיל את־חמסיו / ואתו חברת פריצים / עצרת בוגדים
ומנאצים / ישבו במושב לצים / מלבד התועבות
והשקוצים / וברית עם־השכרות יכרתו / יום ולילה
לא ישבתו / ומן־הפלא איך לא היה בזמן כליון
חרוץ / לרב היות הדור פרוץ / כי רבו העבדים
מאדניהם מתפרצים / והמורים בקשתות הרכילות
ובעלי חצים / אשר בידיהם זמה / וימינם מלאה נכל
וערמה / וחפרה הבינה ובושה החכמה / כי מלך
הנכל בעברת זדון / ונטה ידו בכידון / והחזיק
מלכות בחלקלקות / והיו דדי המרמה מיניקות /
ושבו דדי התבונה להיות צומקות / ופני השכל
מוריקות / ואין מעמד רק ללשונות המחליקות /
ולקדחי אש ולמאזרי זיקות / ותבלענה הפרות
הרעות והדקות / דבר מישרים והלך צדקות / ומאס
בבצע מעשקות / ויהיו לראש אנשי אין / ואחות
תובל־קין / השתים במזרקי יין / ובראותי כי
בכנפיהם נמצאו דם נפשות אביונים נקיים /
ואנשי־צדק נתונים מאכל לעם לציים / בחרתי מות
מחיים / ולו שתקו האוילים והחסרים / ושמרו
מחסום לפיהם השכורים / או יערו כדם אל־השקת /
אז נשתו מי־ים המחלקת.

ואשא משלי ואמר:



הה כי מדד / מרמה תרוה / תבל ושדי / בינה צומקות



ופני המון / פלח רמון / ופני הש־ / רים מוריקות



איכה היקום / יעמד יום קום / כתות על־ה־ / צדק חולקות



גזלו מי־ / רבעם עגלי / זהב אף מ־ / עמרי חקות



ולשונם ח־ / רב וידיהם / מלאו דמים / כמזרקות



הה על־ראש איש / כעש יחרש / הפורה ראש / איש מעשקות



אין מעמד רק / לנרגנים / או ללשונות / המחליקות



ולאיש עוין / הבוחר יין / מלכות בכלי / זהב השקות



נפשי בראות / כל־המצאות / רעיוניה / מות חושקות



לולי יראת / אל על־פני / היו ידי / אותי חונקות



כי מה חפץ / עוד בחיים / ויאורי ה־ / בין שוקקות



שפל רום ה־ / בין ופתיות / החזיק מלכות / בחלקלקות



מדוע ה־ / פרות טובות / תבלענה או־ / תם הדקות



מה־לשכל / ולצאן אובדות / עלות וגמ־ / לים מיניקות



איך לא־ישתק / איש סכל ומ־ / חרחר־ריב ומ־ / אזר זיקות



אלו שתק / איש לא ידע / אז בטלו כל־ / המחלוקות.
