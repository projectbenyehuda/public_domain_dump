


	
	יחידה החטאה
	



תוכחה






יחידה החטאה / ביום דין הנה באה

להתודות על מעלה – / ואדני יסלח לה!





צידה תם ועתרת / קחי לך, מצטערת,

וחסד הכיני עוד, / מרב הדרך מאד.





חלי נא פני יוצרך / ושברי את גאון יצרך,

בכי עליו והתעני, / קומי נא והשתני!





קומי, מהרי ונהרי / אל אדני, והטהרי,

כי עד כה מפשעי / לא רחצת למשעי.





במי דמעה התרחצי / וגם לקונך התרצי,

וחיל תהי סובלת / כאשה מתאבלת.





נחלית בשחין חליך – / הכה אותך מעלך,

דבלת תפלה מרחי / על השחין ויחי!





יגעת ברב הבל / בדרשך תענוגות תבל,

כי תתיף בלבושה / ותיטב את ראשה.





הקריבי מקום מנחה – / לבבך, ושפכי שיחה,

והאל כתפלתך / יתן את שלתך.





ואם דמעך תזילי – / בזאת נפשך תצילי,

בכנף עצתי תחסי, / דעי וראי מה תעשי!





דרשתי רעיוני / להוכיחם בהגיוני:

העתירו אל מלככם, / נכח אדני דרככם!





הרבו בביתו שירה, / ונביא לפניו תשורה

מטוב הגיונינו, / לברך את אדונינו.





בואו שעריו בתודה, / תודה וזמרה חמודה,

ותשמחו בו בפעלכם, / וישמח גם הוא בכם!





נגידים לפניו אמלל, / ואם אמנע מהלל –

למה על אדמתי / החיה אדני אותי?





נעלה על כל רמים / ואין שני ביקומים

כמותו אין בנשיה / וכמוהו לא היה.





תכן איש על אדמה / ונפח באפיו נשמה –

ושת בכח הנפשים / ארבעה חרשים.





נבראתם, בני אדם, / בנפלאות אשר קדם,

מאפס נקראו / ומאין יבואו.





אזנים ואפים / שת בכם, וידים,

ועין לראות פעלו, / וחך אכל יטעם לו.





לבב בעל חכמה, / וגופה חן מרקמה –

וצר משחית אחריו, / יושב לו בחדריו!





השיא לבו בחטאו, / כי לחטא משיאו,

ולא הבין כבער, / כי עודנו נער.





איך נסכל אחי רמה, / איש נוצר מאדמה!

יצרו בתבל עובד – / ואחריתו עדי אובד.





יאחר על היין / ויאהב כל יפת עין,

ויחשק, נעור וריק, / אשה בנחל שורק!





נפתה! איך לא תזכר / עוון, אותך יעכר?

ויום דין כל מפעלך / למועד שמור לך!





איך תשמח, הגבר? / מחר אתה בקבר –

ותהיה לשאול סוחר, / ונהפכת לאיש אחר!





הלא תראה מות גברים / קברום אנשים אחרים,

הלא תביט בעין שכלך: / המה עושים גם לך?





וזה עודנו לא שב, / עד כמתים הוא נחשב,

ויהיו אותו נושאים / בקצה עמק רפאים.





זרקה בו שיבתו, / וזעו שומרי ביתו,

ויקרב אל הפחתים, / וחיתו לממיתים.





והנה בא עד סופו, / וישחת את גופו,

ואת רוחו יגביהו, / כי מלאך יי צבאות הוא.





מאתו נפרדות / ארבע היסודות,

וממנו נתכים / חמשת המלכים.





לא יבין ולא ירגיש / וממקומו לא ימיש,

ומשפתיו הברורים / אין אמר ואין דברים.





טרם ידיחוהו / במים ידיחוהו,

וגופתו האמולה / היא לוטה בשמלה.





כעת מחר בנסעו, / יעזב לאחר יגיעו

ויניח כל מאדו – / ומאומה אין בידו!





העת שב להבל / ינדוהו מתבל,

ויודעיו, יום שברון, / נושאים את הארון.





לבלעו פיהו פצה / אבדון, והאיש יצא

מרעה אל רעה, / ונסב הקרקעה.





אחרי אותו קברו / עליו עפר צברו,

ואיש איש ירים קולות: / גלו אבנים גדולות!





רעים אשר חרדו / מאתו נפרדו,

והוא ישאר נגנז / אל הסלע הלז.





יצא יום הקפדה / מחרדה לחרדה,

ועבים בשית ימיר / והוא יושב בשמיר.





הגויה היקרה / בתבת חמר סגורה,

גל עפר שוכבת – / והיה היא רוכבת.





וייקצו המזעזעים / המה התולעים,

אז יעלו באישוניו / וינקרו את עיניו.





לו יש אדם דובר / בעת שכבו בקבר –

יענה בלבב נמס: / פגעתי את סס!





יאכל עש ולא יחמל / בשרי, והנה אתמול

קם בגופתי באשי, / ועתה ירום ראשי.





למשלח עש ולמרמס, / עורי רגע וימס.

כל עורקי יבשו / ועצמי עששו.





עד שנים עשר חדשים / נשכח מלב אנשים,

הוא רקב ויבל, / ושם האיש נבל.





אתמול בבכי זעקו / רעיו – ועתה ישחקו,

ויתמו ימי בכיתו – / והנה משתה בביתו!





ככה יקרה לנבזה – / כל ילוד אשה, וכזה

ימותון כל בני נשיה, / ואין קצה לגויה!





ורע על כל אלה – / יום לדין האל עולה,

ומלאכי משפט יחזה / מזה ומזה.





ישבע מפרי ידיו, / פרי מגדיו על חסדיו,

ובו לעד יקום כחשו, / וישוב עמלו בראשו.





לזאת אוכיח לבי / כל יום רוחי בקרבי,

כי ידעתי: לא / יחיה אחרי נפלו.





עתה בינו, זממי, / ועזבו ערות עלומי,

שמעו והטו אזניכם, / וישמע אלהים אליכם!





ולכן שמע, איש לבב, / וכל איש בדרכו שובב,

לשרת פני אל יחד, / ולעבדו שכם אחד:





אם נא תדרשו לשכנו, / תשבעו מזיו עדנו,

כחיות משרתות כלם / העומדות על תלם.





אליכם אקרא, אישים, / יחדו כראשים כרשים,

לעבד פני אל קדם, / כי זה כל האדם!
