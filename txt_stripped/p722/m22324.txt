


	
	דודים שוררו אהבה עוררו
	



לשר שלמה אבי משה לרגל חתונת בנו אברהם




דודים שוררו / אהבה עוררו



ובשירי נגידים / לאל זמרו:





שירו שיר ידידות / לצבי נחמד



כי הוא יום חרדות / ללב מעמד



ובכל החמודות / לו שם טוב ויד



מעשיו שפרו / טוביו יקרו



ועניים מרודים / בשמו נעזרו:





שמעו במדינות / הולך הוא ורב



ולכל הלשונות / זכרו יערב



כי מעין אמונות / לו לא יחרב



מימיו גברו / עד לא חסרו



ושדי החסדים / לשקות חברו:





הוא השר שלמה / נודע בזמן



כי האל הקימו / נטע נעמן



איתן, שם מקומו / צלו נאמן



שונאיו נעדרו / בושו חפרו



מירוא קול פחדים / פתאום נשברו:





יגדיל אל כבודו / ובטוב יחנן



את משה חמודו / ויהי רענן



גם זכאי בהודם / ישב שאנן



מעשיו יכשרו / ידיו יצרו



צדק עם ידידים / בו יתפארו:





גיל היום בשמחת / בנך אב המון



וברנן וגילת / צלצל פעמון



ואנחה ומחלת / לב וגוף טמון



כי טוב ספרו / בך, עם נדברו



ושלשה שרידים / בך התחברו:





המה השלשה / כשלשת קני



גלה הקדושה / מול אורם חנה



בם שירה חדשה / כל פה יענה



הדר יחגרו / עז יתאזרו



וכלל היהודים / בם יתאשרו:





אל עושה פלאות / הסר זעמך



ופדה מתלאות / נטעי כרמך



ובחן חן תשועות / תקרא עמך



לעלות מהרו / אל תתאחרו



ולאסוף נדודים / דרך ישרו:
