


	
	יונות אהבים מתעופפים כעבים
	



שבח לנדיב






יונות אהבים / מתעופפים כעבים:



רועות בגנים / בין זהרי שושנים

הוגות רננים / יקרו מאד כפנינים



דוב[ב יש]נים / שירם, בעת כי עונים



ובעת ערבים / אורן כמו כוכבים:



לבם יסובב / על משכנות הרועים

פיהם ינובב / לשאול לכל נודעים

איה משובב / רוחות ביגון תועים

כל הנדיבים / לדבר עצתו שבים:



על מי נתיבות / תרתם יפות עינים

נתנו תשובות / לרמוז בעפעפים

על עז לבבות / יוסף נקי כפים

לו הקרבים / המו ולא נושבים:



גן שעשועים / נמצא ללב כל דורש

יפה ונעים / משרה וכבוד יורש

תמים בדעים / ובעד תבונות חורש

מרפא כאבים / שמחה ללב נעצבים:



אהל יקרו / על קו אמונה העמיד

טל מאמרו / כל נפרדי לב הצמיד

ובחץ דברו / נרגן בזעם השמיד

כי עת שביבים / מליו, ועת כרביבים:



לבות נבוכים / רועה בתם כלבבו

קלו נסיכים / לדרוך עפר מרכבו

הבהיל מלכים / חזק דבר מחשבו

כי הכרובים / לו סוד אנוש מקריבים:



האל יצוה / שלום בתוך אהליו

חסדו ישוה / תמיד מחופף עליו

עד כי יחוה / דור דור נעים מהלליו

ועלי כתבים / לעד יהו נכתבים:
