


	
	אשר חפץ קנות חכמה
	



[הלכות שחיטה]






אשר חפץ קנות חכמה ובינה / גשה אשכילך דת האמונה:

גשה אשמיעך דברי נגידים / ואדריכך עלי דרך נכונה:

פנה אלי ואל תקוץ במלי / ועין לבך להשכילם פקח נא:

הלוך אחר עבודת אל ודתו / שמור מסור לך אנה ואנה:

ויראתו בפניך לעדה / ואהבתו בלבך שים צפונה:

ירא אותו לבל תט מאמריו / ואהבהו למען תעשנה:

ואל תט אחר אות לבבך / לבל תעמוד עלי ימין לשטנה:

ואל תט אחרי אכל ומשתה / ואל תט אחרי דברי רגינה:

ולא תברה בבשר כי לתאוה / להקים נפשך רק לא לעדנה:

והזהר בתורת השחיטה / כמשפטה בעדות נאמנה:

ונשחט כהלכה תאכלהו / אשר בו נקבצו אבות שמנה:

ולא תאכל בהמה המצפה / ליד מות היותה לו למקנה:

והיא היא הנבלה בחיותה / משמרת ימי מות ומונה:

והזהר במנין הנבלות / אשר מנו לך אנשי תבונה:

והם שוחט בכובע או במשעיר / ובמקום אשר תרבץ מכנה:

והשוחט בהקנה במקום / אשר יצא לחוץ מאין רצונה:

ושלא נבדקה אחר שחיטה / והשוחט חצי ושט וקנה:

והשוחט בפגום או מחבר / והשוחט בבתי השכינה:

והשוחט ביום שבת וכפור / והשוחט לחמה או לבנה:

והשוחט לשם קדש אשר בא / בנדר מבלי חטא או תאונה:

וכן אם שחטה סכין בנפלה / ואפלו שחיטתה כתקנה:

והשוהה והדורס ומגרים / ועקור עם שחיטה הטמונה:

וישראל אשר ישחט לבדו / בלי שני ולא למד ושנה:

והחרש והשוטה וקטן / והשכור שאין דעתו נכונה:

ומינים או משמדים וכן כל / אשר סר מעבודת אל ופנה:

וגם הן אספרה לך הטרפות / ואערכם כשושנים בגנה:



ואלו הן: דרוסה או נקובה / בהתרבץ ובלב או בקנה:



ועור מוח והמרה וכרס / קני כבד וקבה הנתונה:

ובית כוסות ומעים והמסס / ומעים בהפכם ממכונה:

וחסרון המנקבים ויתרם / וראה הנקובה או משנה:

והראה והמוח בהמוס / והסמפון בהמוק תחזה נא:

וכן סמפון בהנקב, וכן כיס / אשר מרה שחורה בו שכנה:

והנקב קני ראה למטה / וכן ראה חלקה כגבינה:

והנמצאת נפוחה או אטומה / וכן אם צמקה מרב מגנה:

וכן אם חסרה מקצת, ויבש / קצת גופה ונפרך כלבנה:

וכן מים סרוחים או עכורים / ואם ריחם בלי נמר ושנה:

וכן ליחה סרוחה אם תהי בה / וכן אם נסמכה אונה לאונה:

והאונות בחסרון או ביתרון / וחלופם אשר לא כתבונה:

והרגל בחסרה או ביתרה / וצאת ירך אשר בכף אמונה:

וכן אם נטלה צמה וכבד / והלחי וכליה הקטנה:

ומימיה סרוחים או עכורים / וליחה בה ומכה נאמנה:

וחוט שדרה בהפסק ומוחו / בהתמסמס, ופשט עור בצנה:

וסימנין בהדלדל, וושט / בההפך אדמדמת לבנה:

ודיני הנפולה או קרועה / ברב בשר אשר חופה לבטנה:

וכן אם נשברו רב הצלעות / ונעקרו לחוץ ותגבהינה:

וכן אם נעקרה חליה, וצלע / בחליתה אשר היא בה ספונה:

וגלגלת בהחבס ברבה / וכן אם חסרה סלע הגונה:

הרי שבעים טרפות הם מנויים / שמור אותם וסורה מתלונה:

ואל תקל בדיני אל, ושמרם / כבת עין, וקולן האזינה:

למען אם למשמעתו תהי סר / יהי חסדו לך מגן וצנה:

תנה נפשך ליוצרך למנה / והוא יתן לך חסדו למנה:



והלך בענוה על שבילו / ומאמר פיו ומצוותיו שמר־נא:



והעלה לבך מים חטאתו / ועורר נפשך מבוא בשינה:

דבק לך באשר אהב לדבקה / וסורה מאשר יזנח וישנא:

הכי אם תעשה כזאת, תפלט / יחידתך ותמציאה חנינה:

ובשת ילבשו כל שונאיך / וישמע עלי פיך רננה:

ותזכה לחזות ישע, ותהיה / יחידתך במעדנים דשנה:



בעת־בוא קץ גאולה העתידה / לעורר אהבת אבות ישנה:



ויחן בת שחרחרת עניה / וישבור עז גברת העדינה:

ושמשה יעריב לצהרים / וישכין על מאור יומה עננה:

וכפתה ואגמונה יקוסס / ולא תהיה לעולם רעננה:

בבוא קצה ואין עוזר לנפשה / וישבר בפתע ראש גאונה:

וראש שעיר בתער השכירה / יגלח גם יכס פניו בבשנה:

למען גבהה קומת ארזיו / בהמשך שררותו עדנה:

אלהים ריב לאויבנו והרק / למולם חץ ופי חרב שנונה:

ושובב יעקב אל משכנותיו / ויתעלס באהבה נעמנה:

והזריח מאור שמשו ותמשוך / רצונך אל דביר קדשך ונבנה:

למען כי יספר חסדך אל / לדור ודור ומשנה לשנה:
