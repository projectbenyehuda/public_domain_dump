


	
	ואסע ואלכה : מקאמה
	








הכילי והנדיב





ואסע ואלכה מן הכפרים / לראות בארץ ויפי הערים / ויהי היום ואבא בקהלה מפארה / ובתוכה בטלנים יותר מעשרה / וראיתי כל העיר שוממה / ועל פני החוצות דממה / אין קול ואין קשב / אין גר ואין תושב / רק נער אחד נושא יין ולחם עשרה חמרים / ואשאלהו ויאמר: ׳היום ערב יום
כפורים׳.







וארוצה הנה והנה לבקש מלון / ואשא עיני וראיתי איש נשקף בעד החלון / ואמר: ׳הושיעה אדני / הודיעני בחסדך בית מלוני / כי מאד חליתי ואפס כחי / הלא עברי אנכי / ומחר יום צום וזעקה /
ואיך אצום ונפשי ריקה?׳.







והנה האיש עומד וכוס יין על כפו / ותמרות עשן עולות מאפו / ויהי כאשר שאלתי בשלומו / ויסב את פניו וישאג בשאונו: ׳גש הלאה הרחק מביתי / פן אפגע בך מרב כעסי / הלא מנעורי מתעב אנכי כל גר ואורח / אשר על העדה לטורח / ומה לנו להמון האורחים / אשר עלינו
כיונים בארבה פורחים / נרפים המה ושונאי מלאכה / ויבקשו מעדנים ומזון ארוחה / ויקומו לשחק בנו / וללחך כלחך השור את כל אשר לנו / ואקח היום את פתי וייני / ואתנה לזה הנכרי העני? / והתורה צותה לתת הנבלה לגר אשר בשעריך / ולא לאורח אשר יבא עליך / וכאשר אמר איוב: דלתי לאורח אפתח / שחין וצרעת בו פרח / כי רוב העוברים ושבים / מצרעים הם וזבים׳ / ואני אמרתי
פירוש אמת על זה הפסוק / דבר נחמד ומתוק / אך אין כאן מקומו להאריך / כי כבר שלחני ערוך.







ויכל לדבר ויסגר הדלת בעדו / וארוץ אחריו בתוך ביתו / ותרב כעסו / ויאמר: ׳עז פנים מאנת הכלם / מי הביאך הלום?׳ / ואמר: ׳לא באתי על זבחיך / ולא אוכיחך על דבריך / לך אכול בשמחה לחמך / ומכור נבלתך לגר אשר בשעריך /
אך תנה לי פת לחם ומלח / ומחר יום הדין לעשיר והלך / וגם אני קרוא למלך׳ / ויאמר: ׳הבט נא הבית אשר ממולי / שם שוכן חסר דעת ואוילי / הוא האיש המאסף לכל האורחים / ויין נסיכו דורכים / ולא ראיתי איש מפזר כמותו / ולא יניח מאומה אחרי מותו / ועוכר שארו ובני ביתו / וגם הוא גר בתוכנו / ולא יתחבר את בני עדתנו / ובקהלו לא תחד כבודנו / ועתה שטה ועבר מעל פני / ואכין לבי לתפלה ותחנוני / אשר אתפלל מחר לפני אדני׳ / ואמרה: ׳ישמע ה׳ את קול תחנוניך / ויחזק ויאמץ לבבך / לקפץ את ידך מעניי עמך / וכל זר לא יקרב באהלך / ויהיו עניים בני ביתך׳.







ואבא אל בית פלוני / ויצא לקראתי נער
אדמוני / ויאמר: ׳מהר סורה אדני / ועלה העליה בחדר המשתה / וגם אתה לשבעה תאכל ותשתה׳ / ואמר: ׳מה זה מהרת בני / ואדניך לא ידע את שמי / כי מארץ מרחק אני׳ / ויען: ׳הלא בית אדני פתוח /
לכל גר ואורח ושם ינוח׳ / ובשתי ונכלמתי
להרים פעמי / כי לא אכלתי לחם עני כל ימי / אך חצי הכפן דולקים בלבבי / וכנושה אשר בא לקחת את יהבי / כן יריבון ריבי / ואסיר את מסוה הבושה /
ולחדר האכל אחישה / והנה עדת אנשים / שם אוכלים חמושים / ולא מתבוששים.







ואשבה לי בקצה השלחן / ופני חפו והנערים ירוצו ויעופו / ויציגו לפני כל נתח
טובה / נחמד למראה ובשר תאוה / ואת הכל אכלתי בחפזון / כי קפץ עלי חמת הרזון / ונאכל די והותר / עד כי תם כל הבשר / אחרי כן הביאו כדים וכוסות
/ מלאים יין אשר שם בנו האותות / כי היין בוער ויפיח להבים / הניס כל הדאגות והעבים / ונמלא פינו שחוק אני ורעי הקרבים / ונשא כוס היין ונריע / וכל איש גבורתו יודיע / ושחוק והתלים יביע.







ויקם האדון מעל כסאו ויאמר: ׳איכה כיום הזה שמחות נחבירה / ומחר יום הקדוש והנורא / ומי יעמד אם לא נפשו מטהרה? / אבל אשמים ראיתי בבני אדם / אשר חמס ושד בידם / ויתעוללו ויהוללו כל השנה / אין איש אל יוצרו פנה / ובהגיע הסליחה והתשובה / אז יחשבו תפלתם לחובה / וינחמו על רעתם / ויתודו את חטאתם / וכל היום יתקדשו בצום ותפלה / וינדרו נדר לשם ותהלה / אך כמעט פנה היום וישובו לביתם / וישובו לחטאתם / ולא נודע כי בא התשובה בקרבו / והיצר ישים עליו שנית את ארבו / וככלב שב אל קיאו וצחנתו / כן הרשע אל רשעתו׳.







ואמר: ׳מה צדקו דברי אדני / כזאת וכזאת זה היום קרני / איש אחד אשר יתקדש ליום מחר / מאן תת פת לחם לנכר / ואיך יאמין כי ישמע ה׳ תפלתו / ועוד לא רחץ מצואתו / ואיך יחשב לטהר בטבילה אחת את בשרו / והנגע פשה בעורו / ואם זה בטמאת הגוף מרפש / אף כי בטמאת הנפש?. – –
