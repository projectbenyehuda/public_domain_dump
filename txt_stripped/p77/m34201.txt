


	
	דורש דבר צחות
	




מתוך הספר עפר ודינה




א






דורש דבר צחות ואמרי שפר,

שור מעשה דינה ותולדות עפר!

דת שעשועים שם לילדי הזמן

לקרא בהבלי שוא ומשלי אפר.

שם תוכחות מוסר, ושמה ימצאו

חזון נבוני לב ויודעי ספר.





איש היה בארץ נוד / ויהי לו עשר וכבוד /
ועבדה רבה מאד / ושדות וכרמים שפחות ועבדים, /
צאן ובקר וגמלים וסוסים ופרדים, / אין קצה לקניניו
וחילו / ואין חקר לכבוד עשרו ורב גדלו. / ושם
האיש עפר / ממשפחת בוז מארץ חפר, / מפני שיבה
חשך משחור תארו / ומרב ימים בלה בשרו ועורו,
/ והאיש קשה ורב מעללים / אחרי שרירות לבו הולך
ואין מכלים / וכל חפציו ישלים. / ״אכל ושתה!״ יאמר
לך – / ואחרי לא יועיל הלך. / משמע מוסר אוטם
אזניו / ואין פחד יי לנגד עיניו, / בפעל ידיו חומס
נפשו / ומורא לא יעלה על ראשו. / ושם אשתו
מהיטבאל, / בת איש חיל רב פעלים מקבצאל, / אשה
חכמת לב ונאמנה / מכבדת את יי מהונה, / ותורת
חסד על לשונה, / מאשרת במפעליה / יפה
במעלליה / ותהי נושאת חן בעיני כל רואיה. / על
כן יאמרו במהללה / ״אשת חיל עטרת בעלה!״ / בטוב
לקחה מעת לעת תוכיחנו / ובחלקת אמריה לרגעים
תבחננו, / אף לילות על ערש יצועיה תיסרנו /
ומנחל עדניה תשקנו – / והוא כמו פתן חרש
בקולה איננו שומע / שונא תוכחת ומוסר פורע /
בכל עת אוהב הרע. / ויהי כדברה אליו יום יום ולא
שמע אליה / ולא הטה אזניו לנעם מליה – / מרב
כעסה ושיחה / אתכרית רוחה / ותגרע שיחה /
ולשונה מדבק מלקוחה / ולא תוסף תת כחה, /
ותמת האשה מעצבון מעשה אישה, / באה שמשה
גם פרחה נפשה. / וישאר האיש בירכתי ביתו /
וערפל חתלתו.




ב

עפר מחזר אחרי דינה






ויהי מקץ ימים ויעברו ימי בכיתו /
וינחם האיש אחרי אשתו / ויצא אל רחוב העיר
נשען על משענתו. / וירא נערה מלבנת הספיר
יוצרה הכינה / ושמה דינה, / יפת תאר וברה
כחמה, / בעלת חן ונעימה, / בחרבות לשונה כל
הנמצא ידקר / ובחצי אישונה עיני האנשים תנקר. / וישא משלו ויאמר:



עפרה אשר חן כמעיל לובשת

אחשק – ונפשי יראה מגשת.

הן אהבתה עוררה ויהי סגור

לבי כמטרה ועינה – קשת.

חכה לצודני כחכה העלה

ואני כבן צפור וניבה – רשת.





ויוסף עפר להביט נכח פני הנערה /
והנה כעצם השמים תארה / וכאספקלריא
המאירה / זכה וברה. / ותיקר בעיניו מכל כלי
חמדה / וכל אוצרות זהבו כאין נגדה / ולא
יערכנה אדם ופטדה. / ותדבק נפשו בדינה /
ויאמר: האלהים לידי אנה / לעת זקנתי מגן וצנה, /
מפני הצנה, / בימי הסתו תהיה לנפשי משענת /
ותהי לי סוכנת, / היא האשה אשר הוכיח יי /
להתעלס בה כחפצי וכמאויי / ולמלא כרסה
מעדני.

ויחמד עפר את יפיה בלבבו / ובקרבו
ישים ארבו, / באהבתו אותה על לוח לבו יכתבנה,
/ משחית נפשו הוא יעשנה. / ולכן אמר
המשורר:





אל נא במתק מענה תבטחו,

גם לא ליפי מחזה תשמחו –

פן יקרכם כעוללים עת יחזו

אור גחלי אש כי ביד יקחו.





וילך עפר וידבר באזני אביה /
ויאמר לו ״את דינה / ראיתי ואהבת עולם אהבתיה, /
תנה אותה לי לאשה כי אויתיה! / היא ישרה בעיני
מכל בנות עירי / לעשות ענף ולשאת פרי, / בה
שמתי מבטחי וכסלי – / אין כמוה, תננה לי. / על
כפים אשאנה / ועל ביתי אצונה / ואבנה גם
אנכי ממנה. / כי מה חפצי בביתי ברב עשרי, /
ואנכי הולך ערירי?״ / וישא משלו ויאמר:





נפש נכאה תעמד נבהלת

גם נוהמה, כי אבדה תוחלת.

איך תעלז אם יום וליל תעמל

רוחה, ולא תדע למי עומלת?





ועוד אמר עפר ״והיה אם שמוע תשמע
בקולי – / בידך אפקיד אוצרותי וחילי, / ואם תטה
אזנך לשועתי / לתת את שאלתי – / אתה תהיה על
ביתי, / ואם יכנסו דברי באזנך / ולעשות את
בקשתי ייטב בעיניך – / הנה כל אשר לי בידך, /
ואנכי אכלכל את שיבתך / ונתתי לך ערך בגדים
ומחיתך, / וחיית אתה וביתך״. וישא משלו
ויאמר:





הקשב עצת זקן יועץ בלחשו

ושמע לקול ישיש גם האזן רחשו:

את בתך דינה יפה תנה אליו,

כי כל אשר לאיש יתן בכד נפשו.





ואבי הנערה איש דל ונענה / מכה
אלהים ומענה. / ויען ״כל אשר דבר אדוני אעשה /
ובצל כנפיך אחסה / לאכל לשבעה ולמכסה. / אולם
את פי הנערה אשאל, / וידעתי: לא תשים מלתי
לאל. / בחלק שפתים אפתנה / ולכל אשר אחפץ
אטנה. / ובעוד אני מדבר על לבה, / כיד אלהי
הטובה – / קשט עצמך והתיפה / וגם את הזקן
תספה, / ולך הסר מעליך שמלת אלמנותך ובגדי
אבלך / ולבש בגדיך החמודות החדשות / לכסות את
ערות העצמות היבשות״. / וישא משלו
ויאמר:





מעטה תהלה או צניף תפארת

כסו גויה תהלך קודרת –

כן כסתה רקה אדמדמת עלי

טוחות בקרבם תעמד בהרת.





ויוסף עוד שאת משלו ויאמר:





מכאוב לבבך בטוב גהה תכסנו

ורקב עצמות בחן מראה תחפנו.

תדמה לצורף אשר נשחת בידו כלי

כסף – וזהב לכסות מום יצפנו.





וילך האיש שמח וטוב לב לביתו /
ויספר לדינה בתו את כל המוצאות אותו / ויאמר
אליה ״בתי, בתי, מצאתי לך מנוח, / לא בחיל ולא
בכח: / כי הקרה יי לפנינו / את עפר שכננו /
להחיות את נפשנו / ולמאור עינינו. / אמש אמר אלי
במענה רך ובלשון בקשה / לתתך לו לאשה – /
ואנחנו ידענו את כבוד עשרו וממשלתו / באשר
יי אתו, / כי מלאה כל הארץ תהלותיו, / ואין קצה
לאוצרותיו. / ועתה, בתי, מלאי משאלותיו / בנפש
חפצה, / כי מיי הדבר יצא. / וכן אמרו קדמונינו
ז״ל / ״הכל תלוי במזל״. / ואם יי חפץ לעשותו – /
מי יעץ להפר עצתו / או מי יוכל לתקן את אשר
עותו?״ / וישא משלו ויאמר:





יוצר אנוש כל מעשיו שוה / על חק ומי יפר אשר צוה?

מי זה יעות את אשר תקן / או מי יתקן את אשר עוה?





ותען דינה ותאמר / ״מר לי מר! / העל יפי
הזקן וצבי תפארתו / לבך תשית לדעתו? / ואם על
חמדת פניו והדרתו / פקחת עיניך לעמת
מחברתו? / ובמה יודע כי עת דדים אתו – /
בקרחתו או בגבחתו? / אין בו רק שבתו / לאכל
ולשתות וישבת מכל מלאכתו! / ואיך תאמר שאל
יעקב בעזרו – / ויאכל חצי בשרו! / ישב בדד
ממקומו לא ימיש, / חי נזון בלתי מרגיש, / ולא
נודע אם בהמה אם איש. / ומה בצע בכספו וזהבו /
ועל חמדת נשים לא יבוא? / ימיו נזעכו ורוחו חבלה
– / ומה יתבונן על בתולה? / ואיך אשא בימי עלומים
/ כביר מאביך ימים! / האמנע מחפץ בחורי חמד
למשכב דדים בימי בחורותי, / ושכבתי עם אבותי? /
אבי, אבי! ראה גם ראה לחלשת האיש ותולדתו, /
מתהלך בחוץ על משענתו, / כי תם כחו וחילו /
ומקלו יגיד לו!״ / ותשא משלה ותאמר:





נחם לנפשה מאנה לקחת

דינה בקרחת וגם גבחת:

כי ראתה שיבה בדגל אהבה

על קיר לבבה הוא כמו מספחת

לא יעלז רוחה בחמדת אוצרות

כסף ולא מצאה לנפשה נחת.

יערב משוש גילה בבן גילה ותוך

חברת צבי עפר זמיר פוצחת.

מאין מחיר דוד נאהב נחשב זהב

אופיר, ופניו – מגדלות מרקחת.

אכן מלא ימים בפנים זועפים

עם די זהבו נחשב אמתחת!





ויען אביה ויאמר לה ״בתי,בתי, /
חלקת חמדתי, / ידעתי את האיש ואת שיחו, / עוד עמד
בו טעמו וריחו, / לא כהתה עינו ולא נס לחה. / ואם
שיבה זרקה בו – / עודנו באבו, / אחרי בלותו היתה
לו עדנה בצלם דמות תבניתך, / ולא יהיה זקן
בביתך! / הן עוד היום בתוך עלמות יתיצב, / ובא
ולן ושכב.

עוד ידו נטויה / לקחת חנית מסע ושריה,
/ בגבורה נאזר / ובכבוד יתימר ובאשה שמר, /
אשת חן יתמך ובחיקו ישאנה, / גם כי יזקין לא
יסור ממנה!״ / וישא משלו ויאמר:





אהבת בני נער ללא תועלת,

כי על לבבם נקשרה אולת:

מעת לעת נתהפכה רוחם ולא

יפנו ליעלת חן ואל אילת.

אל הזקנים אמרה נפשי ״שבו

בזה!״ וחברתם תהי שואלת,

כי אהבתם תעמד לעד ובם

מתפארה נפשי ומתהללת.





״בתי,בתי! את אשה חכמת לב
ומשכלת, / אל תהי דלת משכלות לפניך ננעלת, /
ולא תשימי הקדמה חלקית ככוללת: / כי לא כל
הבחורים אבירי הטבע חזקי ההרכבה / ולא כל זקני
ארץ חלושי מזג מפני שיבה: / יש בחור מעם וטעם
זקנים יקח, / גם שב, גם ישיש כגבור לרוץ ארח״. /
וישא משלו:





לא מפני שיבה בני איש נחתו,

גם לא לילדות עז וחיל אזרו:

כי אחזה בנים – וחשך משחור

הוד תארם, ופני זקנים נהדרו!





״הלא תדעי לך: אבינו הראשון אחרי מות
שרה, / לקח אשה ושמה קטורה / ויולד ממנה
ששה בנים, / והוא בן יותר ממאה שנים. / שמעי, בת,
וראי כי רות המואביה / אשת חיל ורבת העליליה /
כמונו היום ירדה לשערים / ולא נטתה ללכת אחרי
הבחורים, / ולטוב מדותיה וכשרון פעליה / מלכים
יצאו מחלציה / ויהללו בשערים מעשיה״. / וישא
משלו:





כל בת כבודה לפי שכלה מיקרת

וכפעלה כן שמה יעמד למזכרת,

כי אם ליפיה ונעם תארה יקרה

כיקר תמונה בקיר היכל מצירת.





״בתי, בתי! זה לי
עשרים שנה בביתי גדלתיך, / על כפים חקותיך /
ועל לוח לבי שמתיך / ואהבת עולם אהבתיך. /
יגעתי באנחתי למלאת משאלותיך – / הנטיפות
והמעטפות וריח שלמותיך / מר ואהלות קציעות כל
בגדותיך – / ואני בעניי הכינותי בתי הנפש
והלחשים / ותמרוקי הנשים, / ותרבי ותגדלי
ותבואי בעדי עדיים / ואלבישך משי וצבע
רקמתים. / אך עתה הלאוני ימי השיבה / ותשש
כחי כנקבה, / מעני ומרב עבודה, / אסלדה בחילה, /
ובביתי אין לחם ואין שמלה. / ועתה, בתי, שמעי
בקולי, למען ייטב לי / ויהי שלום בחילך / וחיתה
נפשי בגללך: / כי זה האיש אלהים שלחו לפנינו
למחיה, / ברוך שאמר והיה״. וישא משלו
ויאמר:





הצור עלי כל היצור משגיח

לא יעזב מעשיו ולא יזניח,

לספר תהלותיו וכח מעשיו

כלו תפלותי וכלה שיח.

הפליא חסדיו לי ועל רב רחמיו

עמל וריש מאהלי השכיח,

תחת היותי איש שבע רגז – אהי

היום שבע רצון ואיש מצליח!





ודינה נשקפה בעד החלון / ותרא את
עפר יכף ראשו כאגמון, / שפל רוח ומרים קלון. /
ותאמר ״אמותה הפעם אחרי ראותי, / ואל אראה
ברעתי! / התבטח כי רב כחו / איש אין מעצר לרוחו /
ויורד עז מבטחו? / התאמין כי ישוב לימי עלומיו – /
ואין שלום בעצמיו / ונפל בעצומיו? / כלו כאדרת
שער מלאה נוצה, / על עפעפיו צלמות ואפל פניו
כסה. / כן יהיה תמיד – הענן יכסנו, / חבור עצבים
ומי ישאנו? / חכלילי עינים, / רפה ידים, / נשבר
ונדכה ממתנים / ועד ירכים, / כחם היום שוכב
בין שפתים, / מבקר לערב לא יקום פעמים. / מרבה
בגדים ומקר מזגו גויתו תהלך קודרת – / ואת
האח לפניו מבערת, / כל רואהו ידד ויפליא מאד / אם
יום או יומים יעמד. / ואיך הבל תנחמני / ולעפר
מות תשפתני! / ואם אבינו הראשון בגדולות
ובנפלאות נשא, / לאו בכל שעתא ושעתא מתרחש
נסא. / וגם רות המואביה היתה אשה אלמנה /
וכפתה לא רעננה: / והיא גם היא בימי בחורותיה
שוש תשיש באלוף נעוריה / וככלה תעדה כליה.
/ ואני עניה אנה אוליך את חרפתי / ואנה מפניך
אברח להנצל מרעתי? / ברכני גם אני, אבי, / באיש
עושה מזמות כאשר עם לבבי: / כי אם ענה תענה
אותי נגד אחי / לחלל צבי עדיי, נטע שעשועי – / טוב
מותי מחיי!״ / ותשא משלה ותאמר:





דינה לנפשה מחנק בחרה

תצעק במר שיחה וכה אמרה:

איה מתי בין חושבים כי האמת

לא נעלמה מהם ולא נסתרה?

איה מדמים כי פעלת תולדות

כל היצורים לא לשוא חברה?

לו יחזו עפרה לעפר נתנה –

לא כחשו בה, כי לריק נוצרה!




ג

עפר נושא את דינה לאשה






ויהי ממחרת ויאסף עפר את כל זקני
עמו / ואת כל בני שער מקומו, / לשמח בשמחת
חפתו / וביום חתנתו. / וילכו לבית דינה בתפים
ובמחולות / וכל העם רואים את הקולות. / וכשמע
דינה את קול העם בכנורות ובמצלתים / וימס
לבבה ויהי למים, / והיא יושבת בין ההדסים
בקשורי עדיה / ודמעתה על לחיה. / ויגש עפר
וישם את הטבעת על יד ימינה / ויאמר ״שמעי נא
זאת, עדינה, / בטבעת זאת תהא לי למקנה״.

ויקם אחד מהעומדים אצלו / וישא את
קולו / ויאמר: בששי בשבת / קול חתנים ממשתה
שבת, / בתשעה ימים לחדש אב / הבתולות תתעלפנה
ובשרן עליהן יכאב. / איך עפר דנן החתן / חלל
יקרן / וחבל הדרן / ושד פניו יחיתן / אמר לה
לדינה אשר בתוך עלמות מהללת: / הוי לי לאנתו
והיית כאשה זה ימים מתאבלת / וכאלה עלה נובלת.
/ ואנא אוקיר ואזון ואפלח – / נצר נתעב ונאלח /
ורוחו כעשן נמלח, / ומעל לותיכי כארח כל ארעא
– / אל תאמן בשוא נתעה. / ודא נדוניא דהנעלת
לה כלתא: / קל קרנא משרוקיתא, / ותעבר המנחה
על פניו – / קרן חזות בין עיניו. / ועוד הביאה לו
בנדוניא: שמה ושאיה / וההעדר הקודם להויה, /
ולהסיר השחין מעל גבו – / חרש להתגרד בו. /
ותחת הסדינים – / פחים וצנים, / ותחת הגליונים – /
רבע הקב חריונים. / וצבי החתן והוסיף לה מדילה /
פגר מובס כרקב יבלה / וכסליו מלאו נקלה, / ולב
נשבר ונדכה, / נגוע ומכה / בשברון מתנים /
וכל מדוי מצרים / ושחין פורח על הברכים / ועל
השוקים. / סך הכל בין כתבה נדוניא ותוספת / את
הקדחת ואת השחפת / את הספחת ואת הבהרת / את
המהומה ואת המגערת / וצרעת ממארת. / וקנינא
מנה במנא דכשר למקניא / כי במהרה ישוב אל
הארץ כשהיה, / וכן קבל על עצמו בדרך פשרה /
דלהוי בטיל וחשיב כעפרא. / ואלו פינו מלא שירה
כים, / אין די להעלות על ספר כבד מכאוביו ועז
דכים – / והכל שריר וקים. / ויכתבו בספר
הדדרמון / בן טברמון / מרבת בני עמון, /
ויעידו עליו עדים: פגעיאל בן עכרן / ועוץ וארן /
ואשבן ויתרן וחמדן וכרן. / וישורר:





ראו חתן שבע רגז וחרפה / וחרפתו בחפתו מחפה,

ויעלתו כלילת חן בצדו – / כברקת אשר נמצא באשפה!





וישורר עוד:





חתן נכה רוח, ופניו קבצו / פארור, ובין עיניו תמשש חשך,

נגדו יפת תאר כאור שמש, ובם / יבדיל אנוש בין אור ובין החשך.









ויעש עפר משתה גדול / לאחיו וקרוביו
ולכל יודעיו מאתמול / ויאמר לאשר על ביתו
בשמחה ובששון / ״הרוג בקר ושחוט צאן!״ / ויתן
לכל הנמצאים מתנות / ומשלוח מנות, / לכלם נתן
אשישה ואשפר / וחדל לספר כי אין מספר. /
וילכו לביתם אחרי אכלם ואחרי שותם / ויברכו את
עפר ואת דינה אחותם, / איש כברכתו ברך
אותם.

ויהי בלילה ויקח עפר את דינה אשתו
/ ויביאה החדרה אצלו לעשות מלאכתו / ויחבקה
וינשקה ולפיסה בדברים הרבה / וידבר על
לבה. / ויאמר אליה ״אחותי רעיתי, משושי וגילי! /
שיתי לבך לחילי, / יש לי רב להפיק משאלך – / משי
אלבישך ותחש אנעלך, / ובאשר תלכי אלך. /
ואנכי פניתי הבית / מחוץ ומבית / ולשוש עליך
כחפצי / מרבדים רבדתי ערשי״. וישא
משלו:





ערש יקר תוך אהלי רבדתי,

עליו רפידת אהבה רפדתי.

גם מר דרור נפתי במשכבי, ובו

לבות אסירי תאוה עוררתי,

שכבי, צבית חן, ואתעלס בהוד

מראך וחן יפיך אשר חמדתי!





וישכב עפר ויאסף אל המטה רגליו /
ותרדמה גדולה נופלת עליו, / כל הלילה שכב עיף
ויגע / ואין מקיץ ואין יודע. / ודינה בפאת המטה
שכבה למעצבה / ולא ידע בשכבה. / ותאמר דינה
בלבה: / דנני אלהים משפט מות באיש הזה, / כל
הלילה אוהב לנום שוכב הוזה, / נמאס בעיני ונבזה.
/ ותשא משלה:

דינה בחיק שיבה שכבה למעצבה

ובמעטה זקן עוטיה ונכלמת,

תשכב בחיק איש רע זקן וגם מאוס

רוחו ביד שנה נתונה – ונפעמת,

דמתה בבית עפר אל שוכבי עפר –

אל נא, רפא נא לה, אל נא תהי כמת!




ד

קצו העגום של עפר






וישמע עפר ויתעצב אל לבו / ולא יכל
לענות מרגזו ומעצבו. / ויאמר: עתה ירננו בקהל
רב בשתי וכלמתי / ועל פני כל חוצות יגידו
חרפתי / ויגילו מפני רעתי. / ואל מי אשא
תלנתי? / נוקשתי בפחי מזמתי, / נלכדתי
ברשת תאותי, / בקברות התאוה פרשתי פח
למעגלי / ורשת טמנתי לרגלי / בקברי אשר
כריתי לי. / הצרי אין בגלעד למחץ מכתי? / אם רופא
אין שם להעלות ארוכה למחלתי? / ואם ברב חילי
ועשרי / לא ימצא מזור למזורי / וצרי לצירי? / אוי
לי על שברי! / זאת היתה לי, / כי שמתי זהב כסלי /
ברב אולתי וסכלי / והוא עוון פלילי. / וישא
משלו ויאמר:





שמו ילידי הוללות כסלם – זהב

חילם, ומבטחם – ברצי כסף.

הם בטחו על ריק, ולהם נאמר

״לא תעשון אתי אלהי כסף!״





ויאמר חרבונה אחד מעבדיו / היושבים
בצדיו / ״יצוה אדוני לבקש ולחקר / מאין ימצא
הדג שנקרא סקנקור / כי יש לו כח יפה ומסגל /
לעורר תאות המשגל, / בחמו יעיר חם הטבע
הנכבה / ולאין אונים עצמה ירבה. / האיש האוכל
מעט מטבורו / יצא חתן מחדרו. / ושעור הלקיחה
לרב / כשני גרעיני חרוב, / וביין חי ישתה / וכל
הישר בעיניו יעשה: כי בזה די למלאת תאות
האשה עד פעמים ששה. / אולם ישמרו מטבורו
ולמעלה / כי הוא חזק מאד ואולי יביא לידי
מחלה״.

ויאמר עפר לאחד מהנערים ״לבקש הדג
הזה קום צא״ / ויבקש הדבר וימצא. / וישמח עפר
שמחה גדולה / (ובמקום גילה שם תהא רעדה וחילה!) /
ולהראות את תקפו, / לקח לו כזית ממול ערפו, /
להיותו יותר תקיף וחזק, / כי אמר: הזקן אשר
מזגו קר בזה לא יזק. / ויאמר: בלילה הזה תהיה
נפשי עדינה / ואוציא לאור משפטה ודינה / בתתי
לדינה / חלק לשבעה וגם לשמנה. / והוא לא ידע כי
בזה הדג כח ארסיי / יוריש לאוכליו אש פרסיי. /
ויאכל ממנו וימלא כל גופו נגעים גדולים /
ויאחזוהו מכאובים וחבלים, / כל הלילה ההוא צועק
ומהים / ויהי לחרדת אלהים, / ותחת מעשה מקשה /
היה כאבו נצח ומכתו אנושה. / וימת האיש בצירים
כצירי יולדה / וילך בלא חמדה.

וישמע שר העיר אודות עפר ועניניו /
ויתן עיניו לקחת את הונו וקניניו / ויאמר לאנשי
ביתו ״את האיש הזה המתם / וכוס רעל אותו השקיתם, /
הרעותם אשר עשיתם! / ועתה, חי יי, כי בני מות
אתם, / כי את אדוניכם לא שמרתם, / ואתם האסרו
בבית משמרכם / ויבחנו דבריכם. / ועל כל אשר לו
– יש לי זכות, / כי כן חק המלכות, / ומשור עד חמור
ועד שה / לא אחמל ולא אכסה״. / ויתן את כל אוהביו
וקרוביו בבית האסורים / וישם עליהם עלילות דברים
/ ויענם בכבלי ברזל ומיני יסורים. / על כן אמר
החכם לבנו: / אוי לרשע ואוי לשכנו! / וכל חיל עפר
היה לבז ולשמה / ולא נותר לנבל מכל אשר לו
מאומה, / ברע מעלליו עליו ועל יורשיו אחריו רעה
הדיח – / ולנו ולכל ישראל שלום הניח. / על כן
אמר המשורר:





לבות מתי כשר במותם נחו / והמון לאמים בעדם נאנחו,

אכן במות עול לבבו יאבל / עליו – והעם אחריו ישמחו!
