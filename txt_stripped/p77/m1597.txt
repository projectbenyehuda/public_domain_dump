


	
	לחולי הזמן
	



כתובת: לאחד מן החברים התחיל להתעסק במלאכת הרפואה


לחולי הזמן זה יום בשורה / להצמיח ארוכתם מהרה



ויום רצון לחבש נשברי לב / ומרגע לכל רוח שבורה



ומרגע לכל נחלה כאלו / ארוכה על פני היקום מזורה



וכל נפש נכאה אל מזורה / וצירה מצאה את מזורה



הכי קם בזמן אברם לרופא / ברב בינה הכינה גם חקרה



והיתה התבונה לו למקנה / מכורה לו ולא ימשל למכרה



ברחב לב יגלה העמקות / ותופע על פני שכלו נהרה



ויודע יסוד כל סוד וכל רז / וכל סתום בטבע היצירה



ינופף יד בטבעי היצורים / וכל מרכב במו חמר וצורה



ויכיר במזגים איש כערכו / ביד מופתיו ובראיה ברורה



ואיך יתמזגו ארבע הפכים / להתאחד במתכנת ישרה



בהתלכד יסוד אויר ועפר / וגם מים ואש ארחו לחברה



והם היסוד לכל הוה ונפסד / כפי החוש ומשפט הסברה



ובגוף יולדו ארבע כנגדם / וכל אחת ידמה לחברה



והם הדם ומיני הלבנה / והאדמה ומיני השחורה



והכבד מקורם גם יסודם / ובאו שם כמימי השעורה



ולשמור הבריאות יעריכם / חכם לבד במערכת שמורה



והליחה אשר תמשל ותגבר / בעוז סמיו יסירה מגבירה



ויפעל בם בכמות או באיכות / ובסגלה מאד נפלאת וזרה



והתחכם בארבע מעלותם / ומזכרת שמם על פיו שגורה



ושת לבוא במשקל המתקן / והמוביל ומשקל התמורה



ויעביר כל חלי עצום כהעביר / אנוש שערו בתער השכירה



כאלו גנזו ספרי רפואות / בתוך רוחו והוציאו לאורה



וכחו רב בכח כל פעולה / ואם הלב והמוח מקורה



בכח רעיון ציור ודמיון / וזכרון וכח השמירה



וחושי הראות טעם ומשוש / והרח ושמע קול הברה



ויודע בנתוח עצמים / ובעצבים ומיתרי הקשירה



ובדפק ימישנו אצבעותיו / להתבונן בסוגיו העשרה



ועליו נפתחו שערי תבונה / ואין חכמה לפניו נעצרה



וירגיש מעגלות קבוץ כפשוט / ולא עמד זמן רב בחקירה



וכל צפון יבונן על נקלה / ולא ישב כסקראט במערה



ובכלי הזכוכית יחזה כל / חלי נסתר וכל מרה סגורה



וכל מחלה אשר על הכליות / וכל סתום במעבר המררה



ויניע ביד שכלו קרבים / כהניע אנוש בר בכברה



וימצא בו אנוש אנוש תעלה / לכל עלה גדולה או צעירה



וכל אשה מצרה ואנושה / ברוחו מצאה רפאות לשרה



ויודע ביולדת והרה / אשר לא ידעה פועה ושפרה



לכל מדוה צרי גלעד ישוה / ובו מקוה למלאות מחסורה



ואם תשכב למעצבה ישיבה / כאזרח רענן ביבוש קצירה



ואלו אב המון גויים ידעו / אמת לא היתה שרי עקרה



ושם אברם ללא עזר ימירון / באברהם ולא שרי בשרה



ובסגלות לבד רועה עקרות / כעשת שן מגרר במגרה



הכי נגלית לפניו כל מזמה / וכל חכמה כקלה כחמורה



וכרמי בין זמן נטע ברוחו / ואין בלתו אשר יכרת זמורה



ומצא חן בעיני אל ואדם / וקנה הלבבות באמירה



ונקשר כל אנוש בעבות אהביו / כשלהבת בגחלת קשורה



והוא משען לכל כושל ונחשל / ולנכשל זרוע עם גבורה



והוא מרפא לכל נספה ונדכה / והוא לכושלים חומה בצורה



ומצאו בו מידעיו תשועה / אהה ואיך רוחי לא נעזרה



ומפיו צוף דבש ירדו ידידיו / אהה לבי ואיך ירדה מרורה



ולבות יודעיו שבעו שמחות / ואיך רוחי שבע רגז ועברה



ינהל אוהביו על מי מנוחות / ונפשי תהלך גולה וסורה



אהה נפשי היש רפאות לשברה / ורופא כל נכה רוח עכרה



היש חתול לחבש עצבונה / והוא יריק סגור לקראת סגורה



היש מזור למחץ חץ נדודיו / אשר לא יחבלו שריון ותחרה



המטיר בי נדודו אש וגפרית / כעל אנשי סדומה ועמורה



ושת לבי כלבת אש וגלמי / כקש יבש בתוך רוח סערה



ואין זולתך עוזר וסומך / ואל מי בלתך אנוס לעזרה



תמוך רוחי בסמים מחזיקים / ומר עובר לאפי שים קטורה



ותחבשת כפרים עם נרדים / במי ורד לכבות המדורה



ומחלת הנדוד אכלה קרבי / בשושני נדודך העבירה



ותפוח זמירך תסמכני / הכי יד רופאי גלעד קצרה



וכל דורש לזולתך ארוכה / כעוזב אל ודורש לאשרה



ויום קמת לרופא בין חכמים / לחולי הזמן הוא יום בשורה


  


סימן



בכל לב נחלה תעש פלאים / ביד שכלך בלא לחש ונחש



התמכה יד בנו עמרם ימינך / והמטה אשר נהפך לנחש
