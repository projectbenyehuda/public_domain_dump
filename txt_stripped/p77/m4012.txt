


	
	בני אדם הבל
	



על העניין


בני אדם הבל / בתוך מאסר תבל / אסורים בכבל / ונפשי יודעת



תכלה גופיהם / ותשחית מעשיהם / ומחמד עיניהם / לעינם פוצעת



באף שן חורקת / ונפשם בוקקת / והיא כמשחקת / ובם משתעשעת



אנוש רם תכניע / ובתהום תשקיע / ולבות תבקיע / ונפשות קובעת



לזאת איש אל ישמח / בנפשו אם יצלח / אשר היום יקח / למחר גורעת



ואל יבטח לבו / בעודו במסבו / אשר עודנו בו / לבבו שוסעת



ואל יבטח גבר / עלי כסף צובר / ומחר בקבר / גאונו תולעת



ואם עת הגישה / לפניו את דבשה / לסוף אוכל ראשה / וישתה קבעת



ועת יחשף שת / וצועק ברשת / תשועה חרשת / וצרה שומעת
