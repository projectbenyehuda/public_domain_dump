


	
	הכיתי בית מאהבי – לחיך צבית חן
	




הכיתי בית מאהבי / בהתולי החשק ובדברי הבאי / בהפוך ה' את עשב השדה לגאון ולתפארת / בהקבץ נערות להוציא ממסגר אהבה מסתרת / ובבתי הנפש והלחשים / מצודדות נפשים / ובחלק שפתים / מושכות כסיל ונעקש דרכים / ויאסרוהו בנחשתים / ובזהר הלחיים / ולבן השנים / ושחרות העפעפים / האדם יראה לעינים / ערב ובקר וצהרים / וארבות השמים / ובמראות הנשים / הצובאות לרעות בגנים / ראו עיני נערה עיניה יונים / נושאת חן בעיני כל רואיה / באהבתה החזקתי ולא ארפה / היא ישרה בעיני מכל בנות עירי / לאחוז בסכלות ולחטיא את בשרי / ותשב באיתן / גנת הביתן / בלבוש שני וצבע רקמתים / כסות עינים / ובגד כלאים / ותעד נזמה וחליתה / לבאים לחקור ממסך ידידותה / והיה במחשך מעשיהם / אור נגה עליהם / מה גדלו מעשיה / מה עצמו ראשי תהלותיה / יפה כלבנה ברה כזהר הרקיע / מבטא שפתיה לבבות יבקיע / ובים אהבתה לב שומעיה ישקיע / ובחורי ישראל תכריע / גויתה כתרשיש ספיר גזרתו / מבינות לגלגל צבי תפארתו / יוצא מחפתו / כצאת השמש בגבורתו / ומי העומד בהראותו / התשלח ברקים מבין עיניה / והיו למאורות לעיני מאהביה / התוציא מזרות מבין שדיה ומחיקה / ומבעד לצמתה שמש צדקה / התלביש צוארה / צפירת תפארה / ותופע עליו נהרה / התקשר מעדנות כימה בקשורי עדיה / ודמדומי חמה על לחיה / מי יעצר כח אהבת כלולותיה / ומי יתיצב לפניה / ולילה כיום תאיר אל עבר פניה / אין קצה לתכונות נעם תכונותיה / אין חקר אל תכלית חמדת מדותיה / על כן כאשר ראיתיה / אהבת עולם אהבתיה / ובזמרת זמירותי לילה ויום הללתיה ואען ואמר:

לחיך צבית חן כיום תאיר ובת / עינך ואישונך כאישון לילה

לילה ויום אראה בפניך אבל / גבר מאור היום להאיר לילה
