


	
	אנא חיש
	



זה השיר הוא על סדר א״ב. ופסוק אחד אומרת כנסת ישראל להקב״ה, והקב״ה משיב לה בפסוק שני.

סימן: א״ב. משה ישועה יצ״ו. לחן: יאמר נא ישראל.






אנא חיש ישענו / אבינו מלכנו / ושעה הגיוננו / דוד אשכל הכפר



בחרתי בך כלה / יסוד עיר תהלה / איסד בחמלה / מגדלה אספר



גדול, לישועתך / הודך וזמרתך / תיחל עדתך / את פשעם תכפר



דעי כי אל תמים / לנצח לך ישים / גברת עולמים / ודרכך ישפר



השב שביתנו / תפארת ראשינו / כי אתה אבינו / אל, כוכבים סופר



ואפדך במהרה / בעז עם גבורה / ולזרע קטורה / אשים עפר אפר



זכר לי אדירי / את חסד נעורי / ותשלח את צירי / קל כמו העפר



חמדתיך, רעיתי / אנחך בבינתי / ואשפך חמתי / על נין עיפה עפר



טוב וישר שימה / עלי רוח חכמה / וראשי הרימה / על צר גמץ חופר



יחידה ותמה / בכתר אחלמה / אשימך רשומה / גלוף בעט סופר



כל בשם טוב וראש / וכתר שים בראש / בן ישי הוא בראש / בשיריו נספר



לעדה שפלה / חכמה עם נחלה / אתן ואשלח לה / פדיון וגם כפר



מרום, חוס עלינו / בבינה תנחנו / ושעה הגיוננו / שיר באמרי שפר



נאוו לחייך / בחסד הוריך / אקבץ קהליך / מעיר ומכפר



סלעי ומצודתי / אל גבור, תקותי / בהגיון תורתי / כדם זבחי כפר



עורי מתנומות / תפארת עלמות / אקבצך מאמות / אביאך הר שפר



פאר בית המקדש / לנצח יתחדש / אשירה שיר חדש / ואכתב בספר



צביה אהובה / את הודך אשיבה / לציון בחבה / עם כהן המכפר



קומה, דודי, ושוב / יסד יסוד חשוב / לעיר קדשך נשוב / בשבחך נספר



רנני בשירה / מלכות גם עטרה / אשיב לך גבירה / עם נרד ומר כפר



שלח לי הגואל / צור עונה לשואל / עם זרע ישראל / בריתך אל תפר



תמתי ידידתי / כלה היחידה / את אויבך אשמידה / ולך יהיו כפר



מלכי שור הגיוני / יה שעה ופדני / עליון הקדמוני / ידמה צבי ועפר.
