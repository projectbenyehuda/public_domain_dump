


	
	צורי חנני
	



סימן: משה ישועה חזק. לחן: יא מחיל יאבני סרת גומני.






צורי חנני / כי בך בטחוני / מהר העלני / לעיר ששוני



משתוקק לראות / את קץ פלאות / עושה נפלאות / הפק רצוני



שנתי תדד / ואשבה בדד / על כי עדי עד / אחר פדיוני



הגידה לי, דוד, / עדי אן אדד / צפיתי ועוד / לך תכלה עיני



יקד לבבי / בראותי אויבי / רודי ועולבי / חיל אחזני



שברתי חסדך / תטה על עבדך / כי הוא דל ודך / נבזה מעני



ולמה, דודי / תחל כבודי / מגן בעדי / שוב חלצני



עלי עשורי / וגם כנורי / אהלל צורי / רנה בלשוני



הלא בך חשקי / שמע נאקי / מידי כל עושקי / פדה המוני



חמל עלי כי / אתה הוא מלכי / הגיה חשכי / שעה הגיוני



זכרתי ימי / נרויתי ממי / ישעך ועל מה / דוד, זנחתני



קוה קויתי לדוד חמדתי / ירצה שירתי / חלף קרבני.
