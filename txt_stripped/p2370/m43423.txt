


	
	אודה לאל
	



שיר זה חיברו על הנס שעשה לנו הקב״ה בליל ראשון של חנוכה
של שנת התר״ן שבאו גנבים לשלול הבית ועקרו מזוזות השער וגררו את המנעולים
וברוב רחמי האל נתעוררה אמי מב״ת והצילנו מידם. ובתוך שיר יסופר אופן
המעשה.

סימן: א״ב. משה ישועה. לחן: אדון עולם מתהפך לכמה נגינות.






אודה לאל / פודה גואל / שיר למואל / על נס נורא



בבוא נפשי / אלף ששי / שנת שמש״י / ליצירה



גם זו קרה / בליל קרה / ליל נס נקרא / לא אשמרה



דע אהובי / כי בשכבי / על משכבי / בצל קורה



הישועה / ליל ארבעה / בחי״ת שעה / ראש משמרה



ובא לאט / גנב בלאט / ולביתי נט / להעדירה



זדון לבו / וכלי חרבו / שם על לבו / גם מגרה



חשב עלי / לשלל כלי / עם כל חילי / באף עברה



טירת הדל / ומזוזות דל / הוא נשתדל / ופזרה



ידי גמול / בכלי פעול / נכח מנעול / שבר תקרה



כחו נגמר / והתמרמר / עקר מסמר / והסירה



לבש אכזר / עז התאזר / ושב חזר / לשברה



מעט מזער / השאיר בער / מן השער / לעקירה



נט אל חסדו / על בית עבדו / סך בעדו / והעירה



סמכה יד לי / ואמרה לי / שמע קולי / פיך סגרה



ענתה ביתי / עוד לקראתי / קול לדלתי / צועק מרה



פתאם באתי / מבוא ביתי / הרימותי / קול ואקרא



צר הביטה / לבך נטה / אל המטה / וכלי פחרא



קרא גנב / סלעים הב / כי נתעכב / ועת עברה



ראה רשע / כי לא נושע / ברח נפשע / קל מהרה



שיר הודאות / לאל יאות / על נפלאות / אזמרה



תמיד שירה / לו אעירה / בעת צרה / ישלח עזרה



מאל שואל / הוא ישכלל / חומות וחיל / עיר היקרה.
