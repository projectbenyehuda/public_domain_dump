


	
	אמרו כי היה באשדוד
	








מתוך השער השני של משל הקדמוני – החסיד ואשתו הנואפת





אמרו כי היה באשדוד איש תמים



במדותיו / וצדיק היה בדורותיו, / איש ישמעאל /



נאמן עם אל / ועם קדושים / זקן בא באנשים. / ותהי



לו אשה מיחסת ותמה, / יפה כלבנה ברה כחמה, /



ומרב צדקתו היה נחפר ונכלם / ולא נסתכל בהוד



יפיה לעולם / ובחזקת אשת חיל מחזיק אותה, / והוא



לא ראה את ערותה / ולא הכיר בפעליה, / כי לא



הביט אליה. / ותחשק האשה בחור אחד מבני העיר /



אדמוני יפה תאר רך בשנים וצעיר, / ותבער



בלבה אש תשוקתו – / נפלאתה אהבתו / מים רבים



לא יכלו לכבות אהבתו.

ויהי בימי קציר חטים / ורעיוני האשה



אל תאותה מטים / ומדרך הצניעות נדים ומתמוטטים



/ כמעשה שטים. / ויגע ערב חגם בבטחה – / החליל



והשמחה – / ויהיו שמחים ועליזים, / ויפקד



האיש את אשתו בגדי העזים. / ויהי כעלות המנחה



/ לעת מצא נפש והנחה / וילך האיש בית התפלה /



במעטה עז ותהלה / בגילה וצהלה.

ויאמר לאשתו ״הכיני השלחן



והמנורה / והנני הולך להתפלל בכנסת



ההדורה״.

ותעמד אשתו להבין בבנים חסר לב



ובער / נכח פתח השער – / והנה הבחור ברנה /



עובר בשוק אצל פנה. / ויצעד ויבוא לקראתה /



ויקרב אל פתח ביתה, / ותסגר האשה בעדו /



ותתפשהו בבגדו. / ותרא את הוד יפיו וגדלו, /



העזה פניה ותאמר לו / ״זבחי שלמים עלי מנעורי /



היום שלמתי נדרי. / והייתי עצובה על נדרי /



ולבי המות בוחר / למיקם עלה משום ׳בל תאחר׳. /



וכבר הזהירוני חכמי הדורות / להבין ולהורות /



ואמרו: אחד הנודר ואחד המעריך / ומלקים



מאמרו האריך / עובר ב׳בל תאחר׳ אם אינו משלים /



כיון שעברו עליו שלש רגלים. / ואמרו: חמר



בנדרים משבועות / וכל אלו הלכות ידועות /



שהנדרים על גב שבועות עולים / ולבטל המצוה



חלים. / ואמרו: בעוון נדרים אשתו של אדם מתה /



כי היא מצוה צרופה ועל לוח לב חרותה. / ועתה אחרי



אשר תוחלתי ממך לא אבדה, / בוא אתי הביתה



ונסעדה, / ותשבע מהודי ומזיו פני / ואשלם את



נדרי אשר נדרתי ליי״. / ויאמר הבחור ״היה לך



לדון אותי מקל וחמר, כי אני תאב מאד לחבורתך /



ופי פערתי למלקוש ידידותך – / אבל לא ראיתי שעת



הכשר / באמת ובישר, / ואני מפחד מבעל הבית /



אשר קנאתו שמיר ושית״. / ותאמר האשה ״הבינותי



הוד מחשבותיך / ונלאיתי נשא על אהבתך / ולבי



בענק חשקך נפשי העניק / יותר ממה שהעגל רוצה



לינק – פרה רוצה להניק״.





ותוסף עוד לפתותו בדבריה / ותחליק



עליו אמריה / ותאמר ״דע כי בעלי – השם ישמרהו /



ויוצרו מבטן יעזרהו! – / הוא בעל תשובה / ונפש



נכבדת וחשובה / ונקי בכל עניניו, / וענתה בו



הכרת פניו / כי לא נסתכל בהוד יפיי / להשלים



את נשיי. / והוא הגדול מבני מתיבתא / מקדימי



ומחשיכי לבי כנישתא / ושעה אחריהם מתאחר /



ובבקר לפניהם שוחר / מתפלל ומתחנן יותר



מחבריו / מברך לפניו ולאחריו / בקשות רבות



ותחנונים / ומרבה רננים. / עתה בוא אתי אל הבירה



/ סורה, אדוני, אלי אל תירא״. / ויסר אליה אל החדר /



בעלית מגדל עדר / ויאכלו וישתו וישמחו /



ויעלו העולה ויצנחו. / ותאמר ״שמח בילדותך



ואל תתעצב / השכם בבקר והתיצב, / ועתה בוא



אלי, זולת יפיי אל תחמד / מהרה חושה אל תעמד, /



כי האיש ישכים לתפלתו להשלים תחנוניו



כמשפטו ודתו / לעשות מעשהו – זר מעשהו



נכריה עבודתו! / ועתה, חשוקי, במאמרי בסתכל



וחשב / לך ושוב!״

ויסב הבחור אל בית אביו כחקו, /



ותשאר האשה לכודה בחשקו, / ותמהר ותעש



מטעמים / קנמן־בשם וסמים / ותכבד ביתה



במיני מרבדים מלאי עינים / חטובות אטון מצרים, /



ותערך נרותיה כפי חפצה ורצונה, / טבחה טבחה



מסכה יינה / אף ערכה שלחנה.

ויהי ככלות האיש להתפלל כדתו /



ויפן ויבוא בית מנוחתו, / וימצא רפידתו ערוכה



ושמורה, / מטה שלחן כסא מנורה. / ויאמר לאשתו



״ברוכה את בפי כל עם ולשון! / היטבת חסדך



האחרון מן הראשון״. / וימהר וישב במסבו /



ויאכל וישת וייטב לבו. / ויהי כאשר כלה



לאכל פתו, / ותאמר לו הענגה אשתו / ״הואיל



ושבענו מטוב הארץ ותפארתה, / לימא לן מר מלי



דאגדתא!״ / ויחל האיש לדבר / ובריחי אמונתו



לשבר / ויאמר ״שמעי, בתי, וראי והטי אזנך /



למאמר יוצרך וקונך, / ואודיעך מוסרים מפרשים /



אשר בארו זקנים וישישים / בעניני הנשים. /



אמרו: צריכה האשה להיות אשת חיל במדותיה, /



צנועה ושלמה במעלותיה, / ובעבודת בעלה מהירה



וזריזה / ובאור פניו מאירה ובשמחתו עליזה, /



ולהתבונן תמיד בדברו ואמרתו / כדי להפיק



רצונו בעבודתו / ושלא יצטרך לקראתה פעמים /



ותיראנו כמורא שמים / ותשמר עת כעסו ושמחתו



/ עת אהבתו ושנאתו / עת שבתו וקומו / עת קיצתו



ונומו, / ואל תשנא אשר יאהב למנה / ואל תאהב את



אשר ישנא, / ואל תחשף סודותיו / ואל תבוז



מעלותיו / תולדותיו ומשפחותיו. / ואמרו: צריכה



לעזב חקה ומנהגה, / תכשיטי עדיה ותענוגה, /



ואל תקשה לשאל ממנו / ומתנו לאל אל תשימנו,



/ וכאשר יתן לה אל תבעט / בין רב למעט, / ותאהב



מאכלו ומזונו / כפי חפצו ורצונו. / ובקימתה



ושבתה / תשמר עתו ולא עתה, / קשבת ולא



דברנית / מישבת ולא יצאנית, / שעל שכל ודעה /



חצבה עמודיה שבעה, / צופיה הליכות ביתה ומעונה



/ ותורת חסד על לשונה. / ואמרו: צריכה לקשט



עצמה דרך צניעות / כדי שלא ימצא בה דפי



וטעות, / ואל תהיה עליו הומיה וסוררת / וביחסה



משתררת, / ואל תראה לו פנים זועפים / ורוח קנאה



שואפים, / ויגדל בעיניה הודו וגדלו / בקרב איש



להשתחוות לו. / וכל מום וכל איבה מלבה תשמיד /



ותתפלל בעדו תמיד, / וכבבת עינה תנצרהו / כל



היום תברכהו. / והאשה אשר אלה ארחיה – / עצמו



מספר שבחיה, / ועליה אמרו זמרה, / רחוק מפנינים



מכרה, / ועל שלמות גדלה ומהללה / בטח בה לב



בעלה. / ועתה אכונן לבי ורעיוני, / הפעם אודה



את יי: / כי רב המדות אשר דברתי / והמהלל



אשר זכרתי – / לך הם למקנה / מימי נעורים



ועדנה״.

וכשמע האשה את דבריו / ונעם אמריו /



ותקם ותחזק בבגדו / ותשתחוה לנגדו / ותאמר



״זהו מאמר חכמי ונבוני / זה השלחן אשר לפני יי,



/ והנה אם צדקת כי במהללך תזכירני – / אמצא



חן בעיניך, אדוני, להכירני!״ / ויכל האיש ענינו /



לדבר על שלחנו, / ויקומו שניהם בחפזון / אחר



המאכל והמזון / ויסר אל העליה וישכבו שם, /



ויתענגו ככל תאות נפשם. / ויהי באשמרת



הבקר ויעור האיש משנתו – / ויפזו זרועי



תנומתו. / ויאמר ״אשתי, קומי, אחותי, / הכיני



שמלתי וכסותי, / כי כבר נתכנו רעיוני: / עת



לעשות ליי! / ועתה לכבד ביתך ולערך מאכלך



עורי / בחטי מנית ופנג ודבש ושמן וצרי, / ואחר



תשימי לבך למסלה / לעלות אל בית התפלה / ולתת



הודאה על טוב חלקנו / ואור נגהנו, / כי קדוש היום



לאדונינו״.

ותאמר האשה בנפש נכאה / ״אל תשת



בי עוון ושגיאה / כי לא אצא מביתי, / פן אשחית



נחלתי. / גאל לך אתה את גאלתי / והתפלל



בעדנו / להגדיל מהללנו והודנו, / כי בנות העיר



הזאת / מאד הן קשות ועזות: / כי כל אחת מהן



לתאותה שוחרת / ועל הוד בעלה אמריה ממררת /



הומיה וסוררת, / מתעסקת בדברי כלמה / פתיות



ובל ידעה מה. / וכל אחת בדברי חברתה החזיקה /



נכריה – אמריה החליקה. / ואין ביניהן נבונה / כי



אם פותה ודברי עגבים מונה, / שית זונה. / ולא



תתחברנה בבתי התפלות / לשמע תחנונים



ותהלות, / כי אם לפני הבחורים להתנאות / לעלות



ולראות / והם לראותן יארבו / ולמלאת פגימתן



יחשבו / וירמזו בעינים ובגבותם / יצפנו



לנפשותם. / וכאשר שמעתי סודיהם / ואתבונן



עדיהם, / אמרתי: חובה להתרחק מכל לב הזיד ושגג –



/ טוב לשבת על פנת גג! / ונשבעתי, אדוני,



בחיי ראשך / חי יי וחי נפשך, / אם אצא מפתח



ביתי / וענתה בי צדקתי! / כל שכן שאין האשה



נחשבת / ובמצות היום אינה חיבת / וכן פרשו



בעלי המסורות: / כל מצות עשה שהזמן גרמה –



נשים פטורות״. / ויאמר האיש ״ברוך יי אשר נתן



כזאת בלבך / והביא רוח חן ותחנונים בקרבך /



ותשימי לאל דברי כסילות ונבזות: / לזאת יקרא



אשה כי מאיש לקחה זאת! / ועתה שבי בית



תפארתך והודך – / ואני אתפלל בעדך״. / וילך



האיש לדרכו / להתפלל לפני מלכו.

ויהי כאשר יצא מבית מנוחו – / והבחור



השכים אל פתחו / וירמז אל האשה במטהו, /



ותפתח ותראהו / ותאמר לו בקול נחוץ / ״בוא,



ברוך יי, למה תעמד בחוץ? / כי אין האיש בביתו



ובזבולו / חלף והלך לו״. / ויבוא ויתהלל



בעלומיו / ויגרשו מימיו. / ויאכלו וישתו כחק



החושקים הנחמים / ויצנחו לארץ שבע פעמים /



וישבעו מעדנים לפי תאותם / הרעו מאבותם. /



ויהי כאשר כלו לאכל את השבר – / אשת מדינים



ובית חבר! – / ויפן הבחור אל דרך גבולו / וילך



לו. / ותמהר האשה להכין מאכלה ומזונה /



ותתעטף ותשב על מכונה.

ויבוא אישה מבית מדרשו / הדור



בלבושו, / ותקם לפניו בשפלות עינים / ותקד בת



שבע אפים / ותסירהו אל מקומו והדרו / ואל מושב



יקרו. / ויבואו אוהביו וכל בני משפחתו /



ויפקדו אותו ואת אשתו / ויאמרו ״השלום לכם מאל



נורא ועילום?״ / ויאמרו ״שלום!״ / ויהי אחרי כן



וישב האיש אל מזונו / והאשה ככתם אופיר לימינו,



/ וישמחו בחגם ובמועדם / כי זה כל האדם – / חק



וזמן נתן להם שלא ישנו את תפקידם.





וירבו הימים / בחטאים ואשמים / ולא



הבין האיש עניני אשתו ומעבדיה, / כי מצודים



וחרמים – לבה, אסורים – ידיה. / ויהי לתקופת



השנה / בהתעורר לב כל זונה / וכל ממרים / ולמלא



תאותו יערים – / ויתאו הבחור לצחק / וילך מאביו



ויתרחק / ויצא בית השחוק / כמשפט כל זונה



וכחק. / ותפל עליו גדר הדחויה – / ויצחק בקביא



/ ויחסר הונו וכסותו גרע / ואבד העשר ההוא



בענין רע. / ויתחבא כל היום ההוא לברח /



וישכח בית אביו שכוח / וישכר לו בהמה לרכב /



ולן ושכב.

ויהי בלילה בהגיע עת תפלתם ועונה



/ ראש האשמרת התיכונה / ויעור האיש משנתו /



וילך להתודות על עוון אשמתו. / וישכם הבחור



ללכת לדרכו / ולנוס מעירו ומקהלת פלכו, /



ויקרא אל האשה אל הפתח / לפתח פה ברצח /



ויאמר לה ״הנני הולך מביתי ואולמי / אל מלון



אורחים ואעזבה את עמי״. / ותאמר לו האשה ״אל



תמנעני, חשוקי, מעצמך – / הלוך אלך עמך!״ /



ותקח מבית בעלה כל אשר תאהב: / כסף וזהב, /



בגדים ורדידים, / נזמים וצמידים. / ויבוא הבחור אל



היכלה / ויקח את רכוש בעלה, / עדיים ושביסים, /



תפארת העכסים / כלילם ואצעדתם ועגילם / עשרה



זהב משקלם. / וישכרו להם בהמה אחרת / חזקה



ונבחרת, / ויקח כל אחד משאו ומבחרו / ויעמס



איש על חמורו, / ולא הניח לבד הכבד / וכל דבר



האבד / אשר לא היה עומס / וכל המלאכה נמבזה ונמס.



/ ויקומו וילכו בנשף / מזי רעב ולחומי רשף, /



וישימו מגמת פניהם אל אדמת מצרים לחגא / דרך



אשדות הפסגה, / ויסבו את כל ערי האמורי



ומגרשם / ויבואו עד חרמה וישבו שם.





ויהי בבקר עת שמחת הלבבות / בקר לא



עבות – / ויכל האיש להתודות על מעלו – / דם ליי



והתחולל לו – / וירץ אל ביתו במרוצה / ופתח פתוח



מצא. / וירא – והנה ביתו שדוד, אין בו מחיה /



ועירם ועריה. / ויכנס אל גנזיו – והנם נסוגים /



כמצולה שאין בה דגים. / ויאמר: אין זה כי אם



אשתי התמימה / השאילה מיטב חפצינו לכלה יתומה



/ כדי לגמל חסדים / בעת דודים / (ולא ידע החסיד



הענו / כי קרן עור פניו!) / ועתה איחל עד עת בואה



/ ואדעה מוצאה. / וייחל האיש למצא פדיום / עד נכון



היום / וירא והנה אחרה לבוא / ויתעצב אל לבו. /



ויאמר: אלך להבין מי הלאני / ואדעה מה חדל אני. /



ויקם וידרש בעדה בכל מושב / נכבד לכל



ונחשב – / ולא מצא לשאלתו מענה / ואין קול ואין



עונה. / ויבוא עד לפני שער העיר / שואל לכל ישיש



וצעיר, / והנה איש מדרך הקדים / בא והקדים /



וירא את האיש ויכירהו – / ויקר מקרהו.

ויאמר ״הלא אתה האיש בעל התשובה /



הולך בדרך חשובה, / ואשתך היתה בעיני בחזקת



הצנועות – / והנה היא מן הזונות הגרועות! / כי



היום פגשתיה עם פלוני הבחור / רצים באין אחור,



/ וחמוריהם נושאים משא כפלים / הולכים להוריד



מצרים. / ואין איש יכול להשיג אותם / לתקף מהלכם



ומרוצתם״. / ויהי כשמע החסיד עניניו, / כסתה



כלמה פניו / ויתן את קולו בבכי ויתנהג כשולל



/ ונאק נאקת חלל. / ויצעק ויאמר ״הן בעוון צדקי



נטמאתי, / אוי לי כי נדמיתי! / וסבת בשתי



וכלמתי / ויגון לבי וחרפתי / היתה לי דמעתי, /



ועיון תפלתי ותשובתי – / היתה מחלתי. / כי אם



התנהגתי בגאוה ובזדון / והייתי בביתי איש מדון



– / הייתי מכיר את אשתי ואודותיה / מלקחיה



ומחתותיה / וזדוני בכל יום ידענה / וגאותי בכל



עת תדרשנה / ותשחת עגבתה ממנה. / אבל



פתתני המשובה ואתפתה / ואמצא כוס חמת אשתי



ואשתה, / ויכלו בהבל ימי / ויבוא רקב בעצמי. /



ועתה, שבים, סורו מדרך תשובה / ובחרו לכם דרך



עזות ומשובה, / פן יקרה אתכם כאשר קראני /



וישבר הדרתכם היגון אשר שברני / ותהיו למשל



עמים וקינה / לחרפה למשל ולשנינה״.

וישב האיש לביתו / כאשר כלה לספר



אנחתו / וישאר לחרפה ולבזיון / לדאבון נפש



וכליון. / ויתן תחת שמחתו – אנחה / ומנוד ראש –



תחת הנחה / ותחת מעשה מקשה – קרחה / ותחת כבוד –



מום ודפי, / כי – תחת יפי.
