


	
	אמרו כי היה איש אחד בארץ תוגרמה
	








מתוך השער השלישי של משל הקדמוני





אמרו כי היה איש אחד בארץ תוגרמה /



היה עובד אדמה, / ויהי שוכן בכפרים / ובתי



החצרים, / והיה מתפתה ומתהולל בעמלו / אדם אשר



לא ידע בין ימינו לשמאלו, / ובינו ובין עיר



הסוחרים הרמים / מהלך שלשת ימים. / ושנה אחת



גדל בביתו זרעים / לעבודת המשי ידועים / וירם



תולעים, / ותהי לחפצו העבודה נמשכה / ותשלם



כל המלאכה.

ויקם יום אחד ללכת בחפזה / אל העיר



העליזה / למכר המשי הגדל בהיכלו / להרבות



חילו. / וימהר להפיק רצונו, / ויחבש את אתונו, /



וישים את המשי באמתחת / בשובה ונחת, / ויתן



אבנים בצד האחר / ולמשקל המשי אותם בוחר, /



ויקח צידה למזונו / ויעמס בשוה על אתונו. /



ויצו לביתו וילך / דרך המלך. / ויהי הוא הולך



לבדו – / והנה איש אחד קרב לנגדו, / והוא עני



מסכן, / בנפשו מסתכן. / ויירא מאד העובד / כי



ראה אותו מבלי לבוש אובד / ויאמר אל האיש ״השלום?



/ גש נא הלם!״ / ויאמר לו האיש ״אל תירא מפני / כי



שלום לך ואין דבר, חי יי!״





ויאמר העובד ״לאיזה מקום מגמתך? /



אולי אוכל ללכת בחברתך״. / ויאמר ״אל הסחורה



מגמתי / ולפי משקל המשי תחנותי״. / וישמח



העובד על הבשורה / וארח עמו לחברה. / ויאמר לו



״הנני בחברתך בוטח / ובאהבתך שמח. / ועתה



הודיעני מלאכתך ועניניך, / אם נא מצאתי חן



בעיניך״. / ויאמר האיש ״מנעורי אני שוכן רמים /



עיר גדולה של חכמים / ושם הכינותי מנוחתי



ושבתי, / ואני סופר מהיר במלאכתי, / ויש לי



בתוגרמה אח וקרוב / ורעים לרב: / והנני שוחר



לקראתם / לדרש שלומם וטובתם״. / ויאמר ״הודיעני



דבר במלאכתך הרמה / ויעבר עלי מה…

ויאמר העובד ״ראה דבריך טובים ונכוחים



במלולך / ושומע אין לך! / כי העמל הפליאני



הפלא ופלא / ולא נסיתי ללכת באלה, / ואיני מכיר



בדבורך לבד חתוך שפתים / פעם ושתים״. / ויהי



הם הולכים הלוך ודבר / וכל אחד בחברו מתחבר – /



ותצנח האתון ותכרע / ויחלש כחה ויגרע, / כי



לא יכלה לסבל המשא. / ועין העובד עליה חסה /



ויצעק ויהמה לקולה / וישת לבו לחילה.





ויאמר הסופר ״למה תשומם, / חרד



והומם? / קרב אל האתון והסר המשא אשר עליה – /



ותנוח מעט מזער מחבליה״. / ויגש העובד להסיר



המשא מעל אתונו / והסופר עזרו כפי כחו ואונו. /



וכראות באמתחתו אבן מעמסה / ויאמר לו ״מה



המשא?״ / ויאמר ״אמתחתי האחד מלא מהמשי הקל



– / והאבנים לנגדו למדה ולמשקל״. / ויאמר



הסופר ״למה מחשבותיך חפצות / באבני גיר



מנפצות?״ / ויאמר העובד ״ואיך ינשא המשי לבדו



/ ואין משקל לנגדו?״ / ויאמר ״בענין המשא



אלמדך דברים / שמע בקולי איעצך מישרים״. /



ויצו להרים האבנים מן האמתחת / ולהציל נפש



בהמתו מני שחת / ולחלק המשי באמתחותיו /



משתי רוחותיו. / ויעש העובד כפי מאמרו ורצונו /



ויקל המשא מעל אתונו.

ויקומו וילכו שניהם חברים / והאתון



לפניהם מעבירים. / והסופר השלים חידתו הנפלאה /



להעביר מהם יגון ותלאה, / והעובד תמה מאד



מחדודו / וגדל תבונתו והודו / ויאמר ״אחרי אשר



הוד תבונת לשונך גדל / מדוע אתה ככה דל?״ /



ויאמר הסופר ״והלא חכמים עשו תעניות וצדקות /



על הסופרים ועל מלמדי תינוקות / שלא יתעשרו



במלאכתם, / וכן נכתב ונחתם! / ועליהם אמרו: נאה



עניות לישראל וצערא / כורדא סומקא לסוסיא



חורא״. / ויאמר העובד ״הואיל וזאת היא מחלתך – /



העבודה! לא אקבל עצתך / ולא אתחבר עוד לחברתך.



/ ועתה הפרד נא מעלי / אל תוסף עוד דבר אלי /



ולך לשלום דרך מגמתך ערוכה – / ואני אתנהלה



לאטי לרגל המלאכה״. / וילך האיש לבדו ארח



נתיבו ויתפרדו איש מעל אוהבו.

וישב העובד ויקח את האבנים אשר



הניח / ונגד עצת הסופר דברים הטיח / ללקט אותם



במספר / מערמת העפר / ויתן אותם באמתחתו /



אשר נמצא אתו / ובמשקל המשי אותם סלה / כאשר



היה בתחלה. / ויקם וילך לבדו מעט מעט / לרגל



האתון לאט, / ויעש בדרך שלשה ימים / וברק



ממרורתו יהלך עליו אימים. / ויגיע אל העיר יגע,



/ עני ועמל שבע. / וילך בלילה ההוא במלונו /



ויאכל לאתונו.

ויהי בבקר ויבוא הסוחר, ויקנה



המשי מידו / וישקל הכסף בעדו / ויאמר ״יש



עוד באמתחתך דבר למכרו? / ואני אתן שכרו /



וארבה כסף שברו״. / ויאמר ״אין בו לבד דבר נקל –



/ אבני משקל / הבאתי לישר המשא, / אבן מעמסה״.



/ ויגש הסוחר לראותם / והנה הם שונות במראיתם!



/ ויבן כי הם טובות / לחדד ולמרט החרבות. /



והמלך צוה על כל בני העיר / יחד ישיש וצעיר /



לתקן כל כלי מלחמה ולחדדם / להראות הדרם



והודם / ולתקן כל נפש אשר לבוש שריון עוטה /



חרב חדודה וגם מרטה, / וכל האנשים אשר לא יראוה



בזמן קבוע / חרפתו וענשו ידוע, / כי כל חשך



לצפוניו טמון / ובתיהם יתשמון. / ובראות האיש



האבנים הספורות / אשר היו מזמן רב נעדרות /



ובעת הצרך המה נסתרות, / אבנים גדולות אבנים



יקרות ־ ־ / דבר אל העובד דברים טובים / אל לב כל



משכיל קרובים / ויאמר ״נקבה אלי באלה האבנים



מחירך / ואני אתן את שכרך״. / ויהי בעיני העובד



לשחוק / ויביא עצות מרחוק / ויאמר ״אני אמכרם



בעבור נעלים / ובמחיר המשי כפלים״. / ויאמר



הסוחר ״אשלם לך כל אשר תשאל / ואל תשים



מלתך לאל״. / ויקח האבנים ויוציאם לחוץ, / כי



היה דבר המלך נחוץ, / ויתן לעובד כסותו ומנעלו



/ וכסף הרבה לו. / וישמח העובד בסכלו ויתהלל



/ כי הוליך יועצים שולל / וסופרים יהולל / ויאמר



״זה תגמול כל עמל וכל רועץ / אשר לא שמע לקול



יועץ״. / וישב לבו שמח / ובהוללותו משתבח /



ויהי לו מקנה צאן ומקנה בקר / שמחה וששון



ויקר.
