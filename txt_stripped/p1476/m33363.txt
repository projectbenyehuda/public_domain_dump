


	
	אמרו הקדמונים
	








מתוך השער הראשון של הספר משל הקדמוני.





אמרו הקדמונים / ופותרי הדמיונים /



כי היה בארץ צען איש משכיל בעניניו, / איש חסיד



איש ענו. / והות שעתא דחיקא לה / ובניו בני דלא



מעלי, / לא היה משיג די מחסורו / ברב חכמתו



וישרו, / והיה חוקר כל תעלומה / ברעב ובצמא, /



וכל היום צועק ומהים / נגוע ומכה אלהים, / ובניו



חוזרים על הפתחים, / נאנקים ונאנחים. / והיו



שכניו שמחים לאידו / ומתענגים בפידו, / כי אמרו:



מה לו ולחכמה / ולחקר תעלומה? / וכבר קדם הנביא



בנבואתו / ״אל יתהלל חכם בחכמתו!״



ויהי כאשר גדל כאבו וצערו / והקיץ



חלל נזרו / ויתחזק בשנה ההיא חם גדול והיבש



רד, / כי חדל המטר והברד – / ויארע לו חלי היבש



במחו, / סר כחו: / והוא חלי המחשב והזכרון /



ואין לו על הבהמה מעלה ויתרון. / ויחל היגע /



הולך ומשתגע, / ותסתר מלבו החכמה והתשוקה



/ ונפשו שוקקה, / ויחזק בבגדיו הצנועים /



ויקרעם לשנים עשר קרעים. / ויתלקטו אליו



אנשים ריקים ופוחזים, / שודדי ערב ובוזזים, /



ויסובבוהו הנערים בריבו / בשמעם צעקו וניבו, /



ותעלולים משלו בו. / ויבוא עד לפני שער המלך /



מצחק ותומך בפלך – / והמלך יושב בעלית



המקרה אשר לו / והשגל יושבת אצלו. / ויתאו



לשמע רבוי אמריו אשר להג, / כי בשגעון ינהג, /



ויצו להביאו ברגילות / ולאחז בסכלות, /



וישתעשע המלך במהלליו / והשגל במעלליו. /



ויתן לו המלך כתנתו / ולחם אמר לו להברותו, /



ויפקידהו עם שריו הנכבדים / וילבישהו בגדים,



/ ויהי לפניהם כל הזמנים / שותה ממתקים ואוכל



מעדנים. / וכמו כן צוה לתת / לאשתו ולבניו



משאת. / ויהי יושב במושב השרים / עם בני המלך



והנערים, / כי היה משעשעם בדברי התול ושחוק /



כמשפט כל כסיל וכחק. / וכל יושבי העיר, / יחד



ישיש וצעיר, / ישימון עין עליו לשמרהו, / כי



מצות המלך הוא. / ויאמרו כל יודעיו כאשר ראו



מעלתו / בחצר המלך עם כל ביתו / ״לטובתו /



נשברה רגל פרתו!״





ויהי מקץ שנתים ימים – ויעתר אלהים



לאדמה / תחת אשר היתה שממה / והשגיח על כל



יושביה / ממכון ערבות, / וינף גשמי נדבות /



לטובה ולשבע ולברכה / שמורה בכל וערוכה. /



ויחלו הנצנים להתנאות / לעלות ולראות / ולהיות



כל עץ פורח, / הגפנים והסמדר נתנו ריח. / ויקם



היבש ויברח / וישיש כגבור לרוץ ארח, / ויבוא



חדש האביב בשמחה / ונסו יגון ואנחה. / ויכנס



השמש במזל טלה / וקראו אחריו ״מלא!״ / בהיותו



מעלה דשאים / בזרועו יקבץ טלאים. / והיום



והלילה באהבה נחבקו, / איש באחיהו ידבקו /



ונשתוו במעלתם וגבהם / במראיהם וגבה להם. /



וירבה הזך והנקי מן הדם / וישב עין כל בחור



מאדם, / ונראה כל לבן אדמדם, / והמרה השחרה



לא תחשב / ונקתה לארץ תשב. / והטבע החכם מסיר



הדאגות / ומשוה הזוגות / והעמלים מתחזקים



לשמח ועמלם לשכח. / ויחסר בעת ההיא הכסל



ויגרע / ואסף המצרע – / וישב הכסיל לדעתו



ולענינו / בפנות בקר לאיתנו.

ויהי לעת ערב / בהתעצב כל לב וקרב. /



וידרש המלך בעד סכלו / וירא כי לא יכול לו. /



וישאלוהו השרים והשרות / שיחה עמקה פי זרות,



/ ויתהוללו עמו – ולא ענה / ואל שטי כזב לא פנה./



ויהי כאשר האליצוהו / ובדברי פחזותם האיצוהו, /



וירם קולו ויקרא בשועו / ״אך אוילים שרי צען,



חכמי יועצי פרעה!״ / ויתנכר אליהם במלולו /



ובאר שיחו הרים קולו / ויפתח וישא משלו:





          תריבוני עלי חמדת אמרי – / תבקשו חטא וסכלות מדברי:

          הלא סכל ואיש הולל – בצרי, / בני הבל, אשר שונאי וערי!

          למדוני מליצה מנעורי, / והשכל והבין – מבחורי.





ויתעצבו האנשים על מאמרו / כי חלל



דברו / ויאמרו ״לא בא זה אלא לרמותנו / לראות



ערותנו, / וכל סכלותו היתה בערמה / על פיו היתה



שומה, / כאשר הערים המשורר נאומו / וישנה את



טעמו / ויורד רירו על זקנו / ויתהולל ברצונו. /



והנה הוא חיב מלקות ארבעים / כמשפט המטעים״. /



ויאמרו אל השוטר להלקותו / וכמספר הזה



להכותו.





ויצו המלך להסיר מעליו מחלצות



הטובות / וילבישהו בלויי הסחבות / וכנצר נתעב



מביתו השלך / ויגרשהו וילך. / וילך האיש אל



ביתו סר וזעף / יגע ועיף, / ויגד לאשתו ולבניו



מחלתו ושביו / בחלותו ויחי מחליו / ואת כל אשר



אמרו השרים להכותו / למען ענותו. / ויאמר ״עתה



ידעתי / וסברתי סברתי / כי העמיק החכם



במשפטו אשר שפט: / עקר מחכמה מכבוד סכלות



מעט!״
