


	
	היה איש אחד בארץ התימני
	








מתוך השער השני [השודד והנזיר]





היה איש אחד בארץ התימני / ממשפחת



הדני / ושמו חבר האחוחי / בן-בלדד השוחי, /



ולא למד תורה בקטנותו / ואל חק ומשפט לא הטה



אמונתו, / שלח ידיו בשלומיו חלל בריתו / והיה



מזלזל אביו ואת בני עמו, / וכסיל אדם בוזה אמו, /



ויתחבר אל בני בליעל וימעל מעל. / וכאשר



ראה אביו ארחותיו / מהלכיו ונתיבותיו, / הדיחו



מביתו לרע מעשיו / ונדהו והורידו מנכסיו. /



ויצא האיש מבית האיש הנאמן / אל מדבר צן נגבה



מקצה תימן, / ויהי לכל עובר ושב משא וטרח / נחש



עלי דרך שפיפון עלי ארח.

ויהי היום והנה נזיר אחד עובר / ליראי



אלהים רע וחבר, / ויתפש בו האיש לבהלו / ולגזל



הונו וחילו – / ולא מצא בידו דבר, / כאשר חשב



וסבר. / ויהי כראותו כי האיש התמים / אין לו דמים,



/ החל להלאותו ולהכאיבו / ולענותו ולהרעיבו. /



וכראות הנזיר צרתו וחילו, / ויבך ויתחנן לו /



ויאמר ״למה תעשה כן לעבדך / ולמה אמות נגדך? /



לך עמי, אל תחדל חדול – / ואני אעשרך עשר גדול



/ ואתן לך כסף וזהב / וכל אשר תאהב״. / ויאמר לו



״אם כן / שאתה עשיר ולא מסכן, / הודיעני נא את



דרכיך / ובמה נעשרת ומה מלאכתך / ומאין



תבוא ובאיזה מקום ישבת / ומה ארצך ואי מזה עם



אתה״.

ויאמר ״אני הייתי מתעסק בסחורה /



בצור המעטירה, / ואהי רוכל בכל מיני תמורה /



בראש כל בשם ובאבן יקרה. / ותגדל מאד יפעתי



ותבונת מחשבותי / ותעש כסף וזהב באוצרותי. /



ויבעט לבי ביוצר אור ובורא שחר / מרב כל הון



ביין חלבון וצמר צחר, / ויתגבר יצרי על נפשי



ההדורה / ויסירה מגבירה, / וילך לו לבקש גאוה



/ אל קברות התאוה. / ויהי יום אחד בפקדי עווני /



הוא יום יודע ליי – / והנה כאב גדול – עד למות



שאלה נפשי / ויקחני בציצית ראשי. / ואשלח אל



הרופאים לדרש / לרפאות כאב הראש, / וארבה להם



מהר ומתן / בגדי משי ורקמה ושריקות פשתן. /



וישתדלו לעשות לי רפואתם, / איש איש כפי



עבודתם, / ויצוו עלי ״ממאמרינו אל תגרע /



ונשמרת מכל דבר רע – / והוא כל מאכל קשה וכבד /



אשר לא יעבד – / בקנה וקדה להשלים מעלתו /



וקנמן-בשם מחציתו״ / להמלט מדבר ואסון / למשח



ראשי בשמן ששון. / ואעשה כפי מצותם ולא



אחרתי, / ובכל אלה נשמרתי ונמשחתי – / ולא



שלותי ולא שקטתי ולא נחתי. / ואשלח בעד רופא



אחד מגדוליהם / שר צלם מעליהם, / והנה הוא בקי



בכל ענין / גדול בחכמה ובמנין. / ואמרה אליו ״על



כל אשר הבינו והשכילו – / כי לא יועילו ולא



יצילו. / ומיום עשיתי מאמרם – כמעט לא אבדתי



אבוד / כי הכאב גדול מאד״. / ויחר אף הרופא



ויריבני, / אפו טרף וישטמני, / ויאמר ״כל



המשתדל ברפואת פתי וסכל / השקט לא יוכל, / כי



לא יבין תבנית המרפא בענינו – / לא ידע שלו



בבטנו. / והנני אודיעך שרש הרפואה ועקריה / אף



היא תשיב אמריה…

ויהי בשמעי דברי הרופא וחכמתו /



והתבונני בהנהגתו ותבונתו, / ואדר על פשעי



ועווני / נדר נזיר להזיר ליי, / ואחפש מעשי /



והקדשתי כל נכסי. / ויעש לי הרופא רפואות /



למופת ולאות / וישלח אלהים מרגיע / מלאך מליץ



ומושיע, / ויתחזק הטבע בלבי / וילך המקרה



מקרבי / וינס הכאב והנזק מעל נפשי, / ואסיר



עטרת ראשי. / ואקום ואבנה בית מקדש / לאלהים מעט



מקדש, / ואכין אותו מקדם לעיר / לקרא לכל עובר



ושב ולהעיר. / ואחבר אלי חברים ותלמידים / רבים



ונכבדים, / ואמאס סלונים וסרבים / וארביץ תורה



ברבים. / ואתן לכל איש כפרי מעלליו / בבית



אשר נקרא שמי עליו, / וארבה בתוכו עשר וכבוד /



וכסף וזהב הרבה מאד, / והכל מוכן לעניים /



ולפדיון שבויים.

ויהי היום ואני יושב על מכוני /



שמעתי ותרגז בטני – / והנה איש מבני עמנו



נחבא / ובין הגויים נשבה. / ואלך היום לפדותו /



ולהשיב את שבותו / והנה שבתי לדרכי / ללכת



לביתי ואל קהלת פלכי. / ועתה מהר לך עמי / אל



ביתי ואולמי / וארבה לך מהר ומתן להשכיח



רישך / ככל אשר תשאל נפשך״. / ויאמר האיש



״איך אלך? אולי יתחברו תלמידיך / ואתה ועבדיך /



ויודיעו למלך דבר איש כמוני / ויאחזוני ימי



עני. / ועתה השבעה לי בצור מעזי וקוני / אם



תשקר לי ולניני!״ / ויאמר הנזיר ״אנכי אשבע, /



שפה לפי סביב לא יקרע״.





ויקומו משם האנשים / וילכו הלוך



והמה מחשים / ויגיעו אל בית הועד / באימה ורעד



/ וכל התלמידים יושבים במסבם / יסבו על דרכם



ונתיבם. / וישתחוו לו וישמחו לקראתו /



וישבו לפניו הבכור כבכורתו. / וישאל הנזיר



בשלומם ובשלום תורתם / ושלום תלמודם ומשנתם.



/ ויאמרו ״שלום – תהלה לשוכן מעוני! – / ברוך



אתה ליי. / אמנם נסתפק עלינו הלכה ונשארה



לפרש / ארבעה חלוקי כפרה שהיה רבי ישמעאל



דורש״. / אמר להם ״שלשה הם / ותשובה על כל אחד
מהם״…

וישמחו התלמידים במדרשו / ושום



שכל והבן בפרושו / ויאמרו לו ״שלום לך ושלום



לדבריך הנחמדים / אשר שמת שלום בין



התלמידים!״





ויהי בהתבונן האיש בשאלת התלמידים



/ ויהי כראות את הנזם ואת הצמידים / ובשמעו את



דברי הנזיר ופרושו – / וישאל באלה נפשו /



ויאמר בלבו: אהה כי נדמיתי באשמתי, / אוי לי על



שברי, נחלה מכתי! / אללי לי כי בא קלון / ויבוא



זדון! / אוי לי, אמ, כי ילדתני איש ריב ואיש מדון –



/ ולא שמעתי לקול הורי / ולא הטיתי אזן למלמדי



ומורי! / ומה אעשה כי יקום אל וכי יפקד / והעת



אשר על הרעה ישקד? / ומה אשיבנו עת ימהר על



האדמה להבקיענה / לאחז בכנפי הארץ וינערו



רשעים ממנה? / למה היה כאבי נצח ומכתי אנושה /



מקצר רוח ומעבודה קשה?

ויהי הוא טרם כלה להרהר מחשבותיו /



בגדל אשמיו / ורע חטאותיו – / והנה קול דממה באה



לבשרו / ותסמר שערת בשרו / ותאמר ״שובו



בנים שובבים / הרחוקים והקרובים, / כי לא אחפץ



במות הרשע נחפר ונכלם, / כי חסיד אני, נאם יי,



לא אטר לעולם!״ / ויתן האיש את קולו בצעקה /



בשמעו קול דממה דקה. / וישמע הנזיר את קולו /



וירא והנה הוא מסלד בחילו. / ויאמר ״מה לך כי



נזעקת, / הרימת קולך וצעקת? / אולי יראת



מתלמידי נאמני / העומדים שם לשרת את פני יי? /



חזק ואמץ, אל תירא, / כי אתך אני לצור מעז



ולעזרה! / והנני נותן לך כעת מחר / כסף וזהב ככל



אשר תבחר, / ואשלחך בשירים ובשמחה / בהשקט



ובבטחה, / ולא אגלה סודך הנעלם / אשר נשבענו



שנינו בשם יי אל עולם״.

ויאמר האיש ״על זה לא נהייתי ונחליתי,



/ לא צעקתי כי יראתי – / ויראתי קשה ועצומה /



כמעט שכנה נפשי דומה: / כי אין בי כח להגיד



עוונותי, / עצמו מספר תלאותי. / כי בעוון



חוללתי / ובדמי חטאתי התגוללתי, / אשמתי



בגדתי גזלתי / והבל נחלתי, / ועל כן חלתי



והתחלחלתי / ובעפר קרני עוללתי: / כי אשמי



הדאיבו כחי / וחטאותי הרפו מחי / אשר חמתם שותה



רוחי, / ועווני מהר להכלימני, / ותושיה נדחה



ממני. / ועתה, אם יש לך על זה עצה / יום עברה



ונאצה / ואם ידעת ראיה ומחזה – / האחיה מחלי



זה?״





ויאמר הנזיר ״אם תשוב אל יי, תזכה



ותחיה / למעלה בזמן תהיה / ובצל ארבעה דברים



הללו תתלונן ותחסה: / צעקה, צדקה, שנוי השם,



שנוי מעשה, / ואף שנוי מקום תועלת גדולה / כלי



גולה״. / ויאמר האיש ״הנה עוני מנשא גדל / ולבי



מהבין מאס וחדל / ואיך יתכפר בשנויים, / דברים



קלים מצויים, / צדקה וצעקה ושיחה / או בזבח



ובמנחה?״ / ויאמר הנזיר ״אם בזבח ומנחה אין לו



כפרה, / מתכפר בדברי תורה״. / ויאמר האיש



״הנני משביעך בשם האל הנעלם / שכוחו מלא



עולם / שתיעצני עצה נכונה / בדעת ובתבונה: / איך



אוכל להרים ראשי / ולהכניע גאותי וכחשי – /



ותגדל חסדך אשר עשית עמדי להחיות את נפשי. /



בשבועה אקבל דבריך ועצתך / ואבוא בברית



אתך״.

ויאמר הנזיר ״אם כן – הדבק בתשובה /



ולך בדרך חשובה / ומאד מאד הוי שפל רוח /



ותשקט ותנוח, / ורדף אחרי הצדקה החמודה /



ושמרת את החקה הזאת במועדה. / ואם תראה רעב



וצמא / חרד והומה – / תמהר להאכילו ולהשקותו, /



וכי תראה ערם וכסיתו. / ואין בזו המצוה שעור



ידוע / על כל אדם קבוע / אמנם הכל כפי הונו



וכבודו, / איש כמתנת ידו. / ואפלו אין לאל ביד



ממונך / לעשות כפי חפצך ורצונך – / היה מפיס



לדל ודובר על לבבו / ונותן הודאה לשעבר וצועק



לעתיד לבוא: / זו היא צעקה וצדקה גדולה / על כל



עולה״. / ויאמר האיש ״הריני מקבל כל אשר תאמר, /



הושיעני, אדוני, כי גמר״. / ויאמר הנזיר ״מה שמך



האזרחי?״ / ויאמר ״חבר האחוחי״. / ויאמר ״חבר שמך



יקרא / ויהיה על תשובתך מורה, / ושנא חברת



הזדים והגאים / יתמו חטאים, / יהיו בלבבך כל



עושי רשעה למכאוב / וחובר חבר ושואל אוב, / ושנה



את שמך כאשר אדוני דובר, / אהוב ורע וחבר. /



ושבת וראית בטוב שדי והודו / בין עובד לאלהים



אשר לא עבדו. / והעד עליך לצור שוכן מרומים / ועל



כל פעליו הנעימים, / ואמר לפניו בכל הדברים



אשר ימצאוך: / חבר אני לכל אשר יראוך! / זהו



שנוי השם ושנוי המעשה / ותחת כנפיו תחסה. /



ושנוי מקום כללים ופרטים – / מעין מבית יי יצא



והשקה את נחל השוטים: / כי דרך רב בני אדם /



שוכני בתי חמר אשר בעפר יסודם / להיות בדעת



נמשכים / וכמנהג מדינות מושבם הולכים. / על כן



צריך אדם להתחבר לחכמים / ולשבת במושב



התמימים, / ויתרחק מן הרשעים / ומחבורת מרעים. /



ואם היה במדינה שיושביה ממרים / ושריה סוררים /



ואינם מתנהגים כשורה / על פי התורה – / ישוב אל


&lt; נפשו ויעירה / ומצנים ופחים יסירה / באמרו:


ראי, נפשי, דרכי ההמון / בתהו ילל ישימון / כמה



מעשיהם מכערים / זקנים בחורים ונערים / איך



במנהגם כל תם וישר יזיקו / ויד תלמיד חכם לא



יחזיקו / וכסילים שמו לשריך ולשופטיך / וזדים



והוללים חזקו לשפט שפטיך / אלונים מבשן עשו



משוטיך! / ועתה, נפשי, לכי ונלכה באור אלהינו



/ להרבות מהללנו ונגהנו / אל מקום אנשים צדיקים



/ ובתורה מחזיקים, / ונתחבר להם בכל מיני חבור /



לדעת ולבור – / ואם אינו יכול ללכת / מפני מחלה



ואלה נתכת / או מפני אמה אדירה או ממלכת / להט



החרב המתהפכת – / יתנהג יחידי ויתבונן



במפעליו / ישב בדד וידם כי נטל עליו. / ואם



לבדו לא יניחוהו – יצא למדברות ולציות / להרים



ולגבעות / לאפיקים ולגיאיות, / ויתנהג בענוה / על



פי המצוה / ויסתכל במעשה הנשא והגבה /



ומבשרו יחזה אלוה. / אלה הם ראשי מדות השב



וכללם – / ותלמוד תורה כנגד כלם!״





ויאמר האיש ״אחרי אשר מצוף דבריך



השבעתני / ומעלות מדות טובות הודעתני, / הנני



מקים אמרותיך הצרופות / בסבר פנים יפות / וכל



ימי חיי אעבר לפניך / כה יעשה לי אלהים וכה



יוסיף, כי המות יפריד ביני ובינך!״

ויקחהו ויביאהו אל בית רפידתו



הנאהב / אשר שם הזהב / ויתן לו ככר כסף



במתנה / וחמש חליפות שמלות לו למנה / ומזהב



שבא לטרא אחת מזקקת / טור אדם פטדה וברקת. /



ויאמר לו ״לך והעבר קול בכל מדינות הנושבות /



בשוקים וברחובות / ושלם לכל עשוק ממון ואמרי



שפר / את מי עשקת את מי רצות ומיד מי לקחת



כפר, / ואל ירע לבבך בתתך לו ממונו וכליו /



וחמישיתו הוסף עליו״.

ויעבר האיש ויעש ככל אשר צוה, /



כי מדרשו למושב לו אוה, / וידבר על כל לב עשוק



וגזול עד אשר נתרצה, להשיב את הגזלה אשר גזל



או את העשק אשר עשק או את הפקדון אשר הפקד



אתו או את האבדה אשר מצא, / ויתן לעניי



המדינות צדקה / וירב על חטאו תפלה וצעקה /



וישנה מעשיו לטובה / כמנהג כל בעל תשובה /



ויעד על פעליו הטובים שמו / וישנה את טעמו /



וילבש שחורים ויתכס / וגבה לב שם למרמס, /



וישב בבית הנזיר באהבה / גרסה נפשו לתאבה. /



וכראות הנזיר שנוי פעליו ומעשהו / וירח את



ריח בגדיו ויברכהו / ויאמר ״תחיה ותזכה בתורה



כפי חפצך, / יי אלהיך ירצך!״ / ויהי ממחרת



וישב הנזיר ללמדו תורה / וילמד כל אשר הורה: /



תורה נביאים כתובים ביחד, / שלשת הרועים בירח



אחד. / וילמד פרושים ודקדוקים, / חקים ומשפטים



צדיקים, / מקרא משנה תלמוד ואגדתא, / ספרי ספרא



ותוספתא. / ויגדל האיש במעלת העשר והכבוד /



וילך הלוך וגדל עד כי גדל מאד. / וישימהו



הנזיר על תלמידיו נגיד ומצוה / מחמיר ומקל



ומשוה, / כי כל דבריו על פי התורה שקלן וחלקן /



חזקה על חבר שאינו מוציא מתחת ידו דבר שאנו



מתקן.

וישמע בלדד אביו ענינו / וייטב



הדבר בעיניו על אודות בנו / וישלח אליו ספרים



וארחה / מנחה היא אל הנזיר שלוחה / כסף וזהב



ובגדים כאשר יאות וקדה חמש מאות. / ויתן לו



הנזיר בתו לאשה – / תהלה תפארת קדשה – / כי



אמר: לעולם ימכר אדם כל אשר לו / להרבות מהללו,



/ וישיא בתו לתלמיד חכם / ויבין ויחכם, כי בת



החכם הנשואה לתלמיד / עולה תמיד: / כי הוא דבר



מתקבל ונאה / יעלה ויבוא יגיע יראה, / משל



לענבי הגפן בגפן, / ואל און נפשו אל תפן. / כי



התלמיד איש נחמד / ודרך תשובה למד / וישמש את



גדול החכמה / ויהי להם לחומה. / והמשיא בתו לעם



הארץ כאלו כופתה בפני הארי / כעת במרום



תמריא. / וכבר אמרו, / נמנו וגמרו: / אין בור ירא



חטא ולא עם הארץ חסיד. / ואמרו: אם תלמיד חכם נחש



הוא – חגרהו על מתניך, / ואם עם הארץ חסיד – אל תדור



בשכונתו והרחק ממנו מעונך. / ועל זה צריך



להשיא בת החמודה / למשכיל על דבר נוצר תעודה /



״אשר יכה קרית ספר ולכדה״. / ויעש החבר חפה



רעננה ושלמה / עד מלאות לו שנה תמימה, / וכל



התלמידים גמלו אותו חסד לרב אהבתו / ויאכלו עמו



לחם בביתו.





ויהי אחרי הדברים האלה וישמעו בני



המדינה / מהלל הכנסת והעדינה / והמדרש הקבוע



והמסבה / אשר הנביא הזקן יושב בקרבה – /



ויתקבצו כלם בנפש נכונה ורוח נדיבה / אל הנהר



הבא אל האהבה. / וישלחו מלאכים אל הנזיר



והחבר, / מור עובר, /ויאמרו ״ראה ראינו כי היה יי



עמכם, / ונאמר: תהי נא אלה בינותינו וביניכם. /



בואו ורשו הארץ הערוכה / ושבו בתוכה, / וכל איש



אשר פיכם ימרה – / ירה יירה, / ותהיו בינינו



למופת ולאות / שרי אלפים ושרי מאות״.

ויאמר הנזיר ״הנה נא זקנתי, / יום



מותי לא ידעתי, / ואיני יכול להלאות עצמי / ולנוד



ממקומי: / אמנם חתני החבר הנאמן / הנה מוכן



ומזמן, / יכול ללכת בהנחה / לתור לכם מנוחה״. /



ויאמר החבר ״איתן מושבך, / חי יי וחי נפשך אם



אעזבך!״ / ויאמרו לו האנשים ״אל נא תמנע מהלך



אתנו, / כי אלהים הדמנו, / ולא נשאר לנו חכם



מורה / ולא תלמיד קורא, / ואתה תהיה לנו למליץ



ולסבר – / על כן ידעת חנותנו במדבר״. / ויאמר



הנזיר ״הנה נא העיר הזאת קרובה, / ועליך המצוה



והחובה / להדריכם בדרך טוב ונעלה / וארח ישרים



סלולה, / וכל זמן תוכל לעלות אל בית מלוני, / כי



תבוא לראות פני״.

ויקם החבר וילך בחבורתם / במחוקק



במשענותם. / ויכינו לו בית ועבדים משבחים /



ועליות מרוחים. / ויהי לכל בני העיר מגן וצנה, /



נחה שקטה הארץ פצחו רנה, / וינהגם על פי



התורה והמוסר / ויהי להם לשר, / ויורם את משפט



התשובה / כאשר היא על ספר הישר כתובה. /



ולפעמים היה מקביל פני חברו בנחת וזמרה / חדש



ושבת קרא מקרא. / ויהיו לו בנים ותלמידים /



עוסקים בתורה ובגמילות חסדים. / ויהי רוכב פעם



בשבוע, / בזמן קבוע, / לישר את בני הסחורה /



במדה במשקל ובמשורה / ושב אל מדרשו שוקט



ושמח / ורב שלום עד בלי ירח.




מקור: שירמן – השירה העברית בספרד ובפרובאנס
