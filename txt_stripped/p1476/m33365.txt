


	
	איש היה בכפר חנן
	








מתוך השער השלישי של משל הקדמוני – הכילי שבא על ענשו





איש היה בכפר חנן / בצל הממון יתלונן,



/ ויהי לו בית חמדה / ומקנה בקר ועבדה. / ויהי



כילי בממונו ומאדו, / משחית תפארתו והודו, /



ודעתו באמונתו נבדלת / בהרבות דברים לאין



תועלת, / וממון עני ואביון יפזר / ויתום ואין עוזר



/ ברב דבריו היה עצל וכבד / כילי אובד.

ויהי היום ויבוא אל ביתו הלך / כתאר



בני המלך / איש אחד נכבד מבני עמו, / ושני



נעריו עמו. / ויאמר לו ״האמצא מלון הלילה הזה



בקרבך?״ ויאמר ״הואל נא ולין וייטב לבך!״



והאורח לא הבין סוד תחבולותיו / שיורד על פי



מדותיו. / ויתן לתחנותיו מטה מצעת ויפה, /



ויעש לו משתה ומצות אפה. / ויוסף לכבדו



ולהדרו / ויתן מספוא לחמורו, / ויתן מים לרחץ



רגליו / ורגלי האנשים הנלוים אליו, / ויאכלו



כלם בביתו ואולמו / וישתו וישכרו



עמו.

ויהי אחרי הסעודה / ויפתח האורח



במשלי תעודה / ויאמר אל האיש ״שמע נא דברי, /



ישר לבי אמרי: / הנה אתה בהדרך כבדתנו /



ומטוב נעמך האכלתנו / ומיין רקח נדיבותך



השבעתנו / וממיטב פעלתך שמחתנו. / על כן



ראוי להודיעך דברים ערבים, / גומל חסדים טובים, /



למען תבין במהלל לבי ודרכו / למי עוללת כה״. /



ויאמר ״דבר כפי צדקך וחסדך, / כי שומע



עבדך״.

ויאמר האורח ״אמרו הקדמונים / אשר



במחנה החכמה חונים / טוב המלון ושלמותו / קונה



בן־חרין בנדיבותו, / ופרי הנדיבות והרוח / הוא



המהלל והשבח. / ואמרו: נדיבות הנכבדים – /



גמילות חסדים, / ומפעלות הנאהבות – / גשם נדבות,



/ כי הזורע נדבה / יקצר הוד ואהבה. / ואמרו:



הנדיב יבקש החן והישר / והכילי – הממון



והעשר. / והמתנדב בנפשו יהלל / וקובץ על יד –



חלקו יקלל. / ואמרו: הנדיבות יחזק הבדק /



ומשם נולד הצדק. / הצדק יצמיח השבח והכשר /



וממנו יולד הישר. / הישר יוליד הנצח והעזרה



/ וממנו תצא החבורה. / החבורה – גפן חזקה /



תצמיח צדקה. / הצדקה תבוא הלם / ותוסיף אהבה



ושלום. / השלום – סוד התבונה / ועקר האמונה. /



האמונה מוכנת להבין ולהורות / והיא ראש התורות. /



התורות – משפטים וחקים / מצוף ומנפת מתוקים, /



ובהם ישוב הארצות / קרבת אלהים חפצות. / ראה כמה



הנדיבות נכבדת / אשר בראש המעלות עומדת /



וכמה גדולה מעלת השמחים / בהכנסת אורחים!״





ויאמר האיש אל האורח / אשר בנגינותיו



ינצח / ״היש לך מחר להשכים, / להתיצב בבקר



לבך מסכים? / עת הוא להכין מטות ולהציע – / אחר



תחוה דעתך ותביע״. / ויאמר לו ״עוד יש לי לתקן



באמתחתי מגרע / אשר היום נקרע״. / ויקח



אמתחתו לתפרו, / ושם נקרב האיש בעל הבית



לעזרו, / וירא בתוכו צנצנת / לתכלית היפי



מתקנת. / ויאמר אל האיש ״מה זה בידך / למעלת



כבודך והודך?״ / ויאמר ״אני רוכל בארץ החוילה, /



נורא עלילה. / בשנה הזאת חסר בעירי צרי גלעד /



למרפא ולמסעד, / ואני הלכתי אל גלעד בעבורו /



וארבה זהב במחירו, / וזהו הצרי הגדול אשר אין



חקר לתפארתו / לכל אשר נמצא אתו״. / ויאמר לו



״בחסדך הודיעני בתועלותיו דבר נרשם / זעיר שם



זעיר שם״. / ויאמר האורח ״ומי יוכל למלל גבורותיו



/ ושבח סגלת מעלותיו? / אבל הרופאים נמנו וגמרו



במהלליו אחד ותשעים / לכל משכיל נודעים / ואם



תרצה לדע סימן במהללו / ״צא״ תאמר לו / כי בכל



מעלה ומעלה / יועיל לכל כאב ולכל מחלה / והשותה



סם המות – בטובו יצילהו / ובשתותו ממנו מיד



יועילהו״. / ויאמר לו ״תן לי מעט מזער לזכר



מעלותיו / למען אספרה כל נפלאותיו״. / ויאמר



״שמעתי לקולך / אני אמלא את משאלך״. / ויתן



לו ממנו בכלי קטן / צלוחית של פליטון. / ויאמר



לו ״הרי לך מהר ומתן / בצלוחית קטנה נתן / ועתה



שים עיניך לשמרהו / כי לא דבר ריק הוא!״

ויהי ככלות האורח לתקן אמתחתו /



ובא ולן במשכב תפארתו – / ויעל בלב בעל



הבית / שמיר ושית. / ויאמר לו ״אדוני, הואיל ואין



רצונך להתאחר / ובבקר אתה תשחר – / צוה לפרע



ההוצאה / אשר לרגלך באה״. / ויאמר ״מה הוא הוצאתי /



אשר בסתר עשיתי?״ / ויאמר ״לא חסרתי דבר



ממשאליך / בל קרב אליך, / וכל דבר במחיר גדול



נרצה / ובספר זה נמצא – / כל שכן בחפזון / בר



לחם ומזון! / ועתה יש לך לתת ככל אשר הוצאת פה /



עם התבן ועם המספוא – / כסף עשרה שקלים /



בשקל העיר נשקלים״. / וישמע האורח הדברים



ויתעצב / ויקם על פתח המשכן ויתיצב /



ויאמר בלבו: ומה יועיל כעסי וחמתי / ולמה



אחלל עמו צדק אמרתי? / ויאמר לו ״קח את כל אשר



תאמר, / כ יין עסיסך לשותיו ימר״. / וישקל לו



הכסף כאשר היה בוחר / עובר לסוחר.

ויהי באשמרת הבקר וישכם האורח



לעמס חמורו / ועמו נעריו עושים דברו. / ויקרא אל



האיש פתח השער – / וינע לבבו כנוע עצי היער /



ויקם בכבד ובעצלתים / כשכור אסור בנחשתים



/ ויאמר ״תן לי שכר המלון והחניה / ושכר מטה



יפה־פיה!״ / ויאמר האורח ״כמה המחיר אשר תדרש



/ ומפני אדם לא תבוש?״ / ויאמר ״ביד האיש היכלת –



/ בקע לגלגלת!״ / וימהר ויתן לו המחיר אשר



שאל, / כי הוא מטמא ומגאל. / ויפתח, וילך



האורח לנתיבו / ושני נעריו סביבו. / ויאמר להם



״ברכו, בני, בכח לאל / יי ממקור ישראל, / אשר



הצילנו מיד האיש הנבל / אשר הרשיע לנו וחבל. /



ואם הייתי יודע מעלו ורשעו / והבינותי נכלו



ובצעו – / לא הייתי לן במלונו / ולא קרבתי אל



שלחנו. / והנני מתנחם לרב פשעו ומעלו / על



הצרי אשר נתתי לו״. / ויאמרו לו הנערים ״אל יחר



בעיני יחידתך / כתר לו זעיר ותראה נקמתך. / אולי



הצרי יהיה סבה לחלותו / חרצבות למותו״. / וישא



האיש את עיניו אל רום מעוני / ויאמר ״אמן, כן יעשה



יי!״ / וילך לדרכו נפלא ורעיוניו תמהים – /



ויפגעו בו מלאכי אלהים.

והכילי קם ממשכבו / ויאכל וישת



ויטב לבו / ויתהלל וישתבח / על כל אשר



עשה את האורח / ועל דבר הצרי אשר שאל לשכרו /



ולא הרבה במחירו, / ויאמר בלבו: יקרבו ימי



הקיץ ואלך לי אל המדינה / מקום הדעת והבינה /



ואמכר את הצרי לאחד הסוחרים / אשר בינה שוחרים



/ ואקנה בדמיו מלבושים / ויין ענושים.

ויהי מימים והנה איש עובר / והצמא



בקרבו גובר / ויסר אל ביתו לשתות / ויואל



הכילי אותו לפתות / ויאמר לו ״מה מלאכתך ומאין



תבוא?״ / לדעת עסקו ומחשבו. / ויאמר לו ״אנכי איש



ציד, / בחרבי וקשתי צבי מיד. / והלכתי לקנות



למלאכתי סם / מאת איש אחד בשם, / לתת ממנו



בראש החצים / להיות נוגשים ואצים / להמית כל



בהמה וחיה / בכל כלי צדיה. / והסם הזה הוא חזק



/ וכל הנוגע בו יזק / ולא יועיל לו מרפא ומסעד, /



זולתי הצרי הגדול הבא מגלעד״. / ויאמר הכילי



״תן לי מעט ממנו ואשים מארב / אל נחש שרף



ועקרב / אשר הם לוחצים אותי / ואת בני תנובתי״. /



ויתן לו מעט מזער / וחצנו נער. / ויפן הציד



וישב לביתו / והחיות יגורו משתו.

והכילי חשב מחשבות און בתאותו, /



כי נתמלאה סאתו, / ויאמר: עתה אבחן הצרי



ותפארתו / ואדע מחירו ותועלתו. / אם סגלת הסם



והצרי כאשר למדתי יפה – / אני אמחץ ואני



ארפא. / ויקח מן הסם ויתן בלשונו / וילכד



בעוונו / ויגוע לבו כרגע – / המקרה והפגע! – /



ויתהפך בצירו ויתחלחל / ופחד ורהה וזחל. /



וירץ אל גנזיו לקחת את הצרי / להסיר אף וחרי – /



ויתבלבלו ידיו במפתח, / אין סוגר ואין פותח.



/ וישאג וינהם במכאובו / לרע עצתו ומחשבו. /



ותרא אשתו כי נלכד במעלליו / וציריו נהפכו



עליו, / ותפתח ותקח הצרי השמור כבת עין /



ותבא לו יין – / ואין שותה ואין לוקח, / אין מבריח



ואין בורח! / ויגוע וימת באשמיו / ונפל



בעצומיו, / והמרפא לא הועיל / צריו עבר – ולא



יגעיל.
