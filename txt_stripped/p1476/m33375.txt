


	
	היה בציון בחור אחד
	




מתוך השער הרביעי של משל הקדמוני – מעשה כשפים





היה בציון בחור אחד נחמד / אשר דרך



המליצה למד, / ויהי לו בכל חכמה מבוא / כי לכל



תבונה הכין לבבו. / ויבקש לדעת בענין הכשפים



/ ויחפש ולא מצא את התרפים: / כי נאסר הכשוף



לאמה חמודה / ליושב ירושלים ולאיש יהודה. /



וכראות הבחור כי התשוקה ללבו נוגעת / ובקשתו



בארץ הצבי נמנעת, לקח מהונו דבר נרשם / וירד



מצרימה ויגר שם.

ויהי בתחלת בואו אל העיר / ומחשבתו



העיר – / והנה זקן אחד יושב / חרש וחושב. / וישא



עיניו וירא את האיש האורח / אור זורח, / כי כלו



מחמדים / לבוש בדים. / וישאלהו הזקן לדעת



מחשבו / ״אנה תלך ומאין תבוא?״ / ויאמר לו ״עברי



אנכי נוצר תעודה / מבית לחם יהודה״. / ויאמר הזקן



״סורה, אדוני, אלי – / רק כל מחסורך עלי״. / ויסר



אליו ויביאהו אל ביתו וחצרו / ויתן מאכל



לחמורו, / וירחצו רגליהם ויאכלו וישתו /



ופיהם בדבור שתו. / ויאמר הזקן ״בהתחיב כל



אדם באהבת החכמים / ולבחר מושב זקנים ותמים – /



כמו כן נתחיב להתנהג עמו בשכל טוב ומוסר / וכל



טוב ממנו לא יחסר, / ולקרב לביתו כל אורח /



ולהיות בכבודו שמח, / ולדבר דברים על לבו /



להניחו מרגזו ומעצבו, / ולהדריכו בדרך ישרה /



כפי הדעת והסברה: / ולא יתכן זה זולתי בדעתו



רעיוניו / ולאיזה ענין מגמת פניו – / ועל פי



דרכו ייעצהו ויועילהו / ועל מי מנוחות ינהלהו. /



ועתה הואיל ובבואך כבדתני / ולחבר ולרע לך



שמתני / וארחת הלילה במעוני / סביב לשלחני –



/ השמיעני מצפון לבך ומהלכיך, / הודיעני נא את



דרכיך ואדעך, / למען אמצא חן בעיניך. / איעצך



כפי שכלי אענדך עטרות לי״.





ויאמר הבחור ״באמת צדק החכם במוסרו



/ וגמר אמר במאמרו, / כי כל אדם צריך להועץ /



מפי זקן ויועץ – / ואף אם יהיה משכיל ובעל מליצה,



/ אין לו יתרון בלי עצה: / כל שכן איש אורח / נודד



ממקומו ובורח / ודעתו בנדוד דחויה / בארץ



נכריה. / ועתה אודיעך מחשבות לבי, / אם דרך



עצב בי: / אני הייתי שוכן בציון / כפלגי מים



בציון, / ונפשי היתה לכל מליצה מודעת / ולבי ראה



חכמה ודעת. / ואבקש לדעת בענין הכשפים דבר /



ובמה כחם גבר – / ולא מצאתי בעירי מענה / ואין קול



ואין עונה. / ואחשבה לדעת זאת / באחת מערי



הפרזות, / ועל זה מארץ הצבי יצאתי / ועתה באתי /



לבקש מחסורי די / לכל חרטם ואשף וכשדי. /



ועתה אם ישך עושה חסד ואמת עמי / אחרי אשר כל



זאת נדדתי מעמי – / הודיעני הימצא איש בקי בזאת



החכמה, / ויעבר עלי מה, / ומהר ומתן ירבה עלי /



ואתנה כאשר תאמרו אלי / ויאצל עלי מהודו



וילמדני / ואותיות החמה יגידני!״

ויאמר הזקן ״אחרי אשר עד רחבי ארץ



התבוננת / והלילה בסתר אהלי התלוננת /



בבואך לבקש חכמה / מעיני יושבי ארצך נעלמה – /



על כן יהבך עלי השלך / ואני אלמדך להועיל,



אשכילך בדרך זו תלך, / ואודיעך עקר הכשפים /



ואון ותרפים: / כי החכמה הזאת לי למלאכה / ומאבות



אבותי לי נמשכה״. / וישמע הבחור עניניו / ויהי



כמצחק בעיניו, / ויחשב בלבו מחשבות / אשר



תבונת הזקן מתעבות. / ויאמר: אין זה כי אם הטעהו



יין מסבו / ולהורות נתן בלבו, / ויחמד הוני וחילי



/ ויחפץ לרמות דעתי ושכלי. / ויאמר לו



״מהאלהים תמצא משכרת / ויערב שיחך כשיר



וקטרת, / כי דברת על נפש אנושה ומרה, / רוח



נשברה. / אבל איני חפץ להכביד עלך / ולהטריח



עליך, / כי החכמה הזאת גדולה / כנחש תתן



קולה, / ורצוני לבקש בחור בגילי / ויגידה



ויערכה לי: / והוא ישאלך על הספק / להגדיל רצון



ולהפק – / ואתה תעזרהו ותועילהו בחסדך / ולי



אני עבדך״. / וידע הזקן כי לא האמין הבחור



לחכמתו / ולא חשש לתבונתו, / ונדר בלבו נדר: /



אם אבוא באהל או בחדר / ואם אעלה על ערש יצועי, /



עד אראהו חכמתי ודעי / ואודיעו את גבורתי / שכלי



ותבונתי. / ויכן תחבולות ויחנן קולו / ויסב



פניו ויאמר לו / ״לין הלילה והיה בבקר אראך איש



אשר לתבונתו אין חקר: / הוא יורך וילמדך – / וגם



אני אודך״.

וייטב הבחור שרעפיו / וישתחו



לאפיו / ויאמר ״ברוך השם אשר מהללו לא יצמת /



אשר הנחני בדרך אמת!״ / וישבו שעה אחת



מתחברים – / עודם מדברים, / וישאל הזקן לשתות



/ לרמות הבחור ולפתות. / ויקח הכוס בימינו /



וישת מן היין לרוות צמאונו. / ויאמר אל הבחור



״שתה – וישתכחו תלאותיך, / ושכבת – וערבה



שנתך״. / וישתה הבחור, וישתנה מחשבו /



ויקם וילך אל בית משכבו, / ויהי הולך – /



והזקן לנגדו פחים ישלך. / וירא והנה באר



בשדה / והנה שם מקום אשר עמלו צודה, / ויחפץ



לרגלו ויחשל / ויפל בבור סיד ויכשל. / והבור



ריק אין בו מים / וישא לבבו אל כפים / ויצעק



ויהמה בכחשו / כאשר ראה צרת נפשו. / וילך כל



הלילה במחשכים / וגוו נתן למכים, / ויהי בבקר



וירא בירכתי הבור דלת, / ונפשו עליו מיללת. /



ויפתח ויצא לחוץ / נדהם על כאבו ולחוץ. /



וימצא גנה מלאה עצים / ושיחים וצצים / כתמר



ותפוח גדולים וגבהים / כעדן גן אלהים, / ונהר



גדול אותה סובב, / ואיש נבוב ילבב / להביט מהללה



ויפיה / לאכל מפריה. / וישכח בראותו מעדנים



/ חדשים גם ישנים, / וישכח רישו וחילו / ולא



חשש לעמלו. / וילך הלוך עד קצה הגנה / ויעמד



בראש פנה, / וישא עיניו וירא – והנה גשר /



בנוי על קו ישר, / ובו שני מגדלים חקוקים בששר



/ עשויים באמת וישר. / ויעמד הבחור וישתאה /



וישתומם על המראה.





ויאמר בלבו: ארוץ ואעבר את הגשר /



כאשר ידאה הנשר, / אולי אמצא בלכתי ישוב /



נחמד לשכן וחשוב. / וילך במרוצה וידהר /



ולעבר הגשר מהר / וימצא עיר גדולה / ומיפי



כלולה: / ובה שוקים ורחובות / מלאות וטובות, / ובהם



כל בעלי מלאכה, / מערכה מול מערכה. / ויגיע אל



שוק אחד גדול ונחמד / והיה כל הבא אליו ועמד: /



כי שם נמכר כל חפץ נחמד ויפה – / תרשיש שהם



וישפה, / ושם הסופרים יושבים איש על דגלו / כל



אחד שוכן עם בני גילו. / וישמח הבחור על הפלאות



/ וישמח לראות.

ויען סופר אחד נצב על משמרתו /



בראותו הוד הבחור ותפארתו / ויאמר ״למי אתה



ואנה תלך, / אשר חץ ידידות בלב רואיך תשלך?״ /



ויאמר ״אנכי איש ציון שוכן ירושלים / ירא יי



אלהים בשמים, / ובאתי למצרים לבקש חכמת



הכשוף – / את הכוס ואת הינשוף״. / ויצחק הסופר



מדבריו / ויקרא אל הסופרים חבריו / ויאמר



״הידעתם על עיר – ירושלים שמה, / ואורח יבקש



בארץ מצרים חכמה?״ / ויאמרו ״לא שמענו בלתי



היום / כדברהזה נורא ואים״. / ויאמר ״הנה זה



אורח מארץ רחבת ידים / אשר אור לו בציון ותנור



לו בירושלים״. / ויתמהו הסופרים על דברו /



ויותר על תפארתו והדרו / ויאמרו לו ״הידעת



בענין הכתיבה / חקיקה או חציבה?״ / ויאמר ״ידעתי



בדבר הזה נפלא / בכל טוב נמלא״. / ויכתב להם



שורות הפלא ופלא, / ויקרא באזניהם שני הבתים



האלה:





         חוסו עלי נודד, / בקש להתבודד –

         ובשעלו מודד / לעג ומי מרה!

         נפלא מאד נצב, / משכן נדוד חצב

         ויעמד נעצב, / כי צחקה צרה.





ויתבוננו הסופרים במליצת סברתו /



ויתמהו מגערתו, / ויקח אותו הממנה / אשר



משאר הסופרים לטוב משתנה / ויביאהו אל ביתו /



ויראהו את בית נכותו. / ויעש משתה שמנים /



מיין רקח מעסיס רמונים, / ויתנו לו החברים מנות /



ויביאו לו מגדנות, / וישבו כלם סביבו /



וישתעשעו בניבו. / ויעמד על עמדו האורח /



בנגינותיו מנצח / ויכתב להם עוד שתי שורות /



לפקח עינים עורות:





         מיחל, עת שתות יין משמר / ולאכל מסעודת אל וקוויו!

         ראו מאכל בבית נדיב מתקן / והיין משמר בענביו.





ויתמהו האנשים מכתיבת ידו / ומליצת



שירו והודו / ויאמרו ״הכזה ראוי לבית המלכות /



אשר בו יתרון וזכות!״ / ויגידו בית המלך פנימה



/ לאמר ״פעמיך למסעות הרימה, / ותראה סופר מהיר /



כזהר הרקיע יזהיר, / האיש אשר בא היום אלינו – /



הנה זה עומד אחר כתלנו״. / וישלח המלך



בעבורו, / ויכן לנגדו פעמי אשורו. / ויבוא



וישתחו האיש הנאהב / ויושט לו המלך את



שרביט הזהב. / וישבו הסופרים להודות מהללו /



מימינו ומשמאלו / וידברו למלך ארמית / באזני



כל רע ועמית / ״הנה זה האיש אשר בו רוח יתירה /



ומלתא די מלכה שאל יקירה / לשמשא קדמוהי גבר



די חכמתא ושכלתנו בלבה / די רוח אלהין



קדישין בה״. / ויהי בהתבונן המלך מעניהם /



וישא משאות מאת פניו אליהם / ותרב משאת האיש



ומתנותיו החמודות / ממשאות כלם עשר ידות. /



ויבקש ממנו המלך בקשה, / חזות קשה – /



למנותו על הסופרים בבית מלכותו / ולעמד על



משמרתו / ולהיות ראש לכל אנשי שלומיו, / סגנו



על כל חכמיו.





ויאמר האיש ״לא, אדוני, / ולגור בארץ



אין רצוני, / כי לא יצאתי מארצי לבקש כבוד /



בחצר המלך לעבד, / אמנם יצאתי לבקש חכמה /



ולהרבות עצמה. / וכבר אמרו בעלי הבין והמדע: /



אל תתראה לרשות ואל תתודע! / ואמרו: המלך



והאש מתדמים / בפעליהם העצומים / תצטרכם – אם



תתרחק מהם, / ישרפוך – אם תתקרב אליהם״. / אמר



המלך ״והם כתבו דברים / להגיד מישרים / ואמרו:



בטח בבורא ובדתו / ותשען במלך ובעבודתו /



ותתחבר אל החכמים / בטוב ובנעימים. / ואמרו:



המלך והאמונה אחים / בשמן משחת קדש משוחים, /



ולא יעמד העולם זולתי בשניהם – / אחד באחד יגשו



ורוח לא יבוא ביניהם. / ואמרו: אל תשכן בעיר אשר



אין בה מלך יכול / ומהיר צדק שולט על כל, / ורופא



חכם אחד ונרצה / ונהר עובר ושוק נמצא. / ויצו



המלך להחזיק באמונתו / ולקים חקיו על פי דתו /



ולשמר גבולי הדינין כלם, / כי הם להעלותו סלם,



/ ואל יניח פלפולים ודקדוקים / אשר לא יקים. /



ויגדיל מעלתו בכל זה / ויכבד דבר כל נביא



וחוזה, / ויתרצה לבוראו במעבדיו / ויהיה נחמד



ואהוב לעבדיו. / ותמיד בהכרת פניו / יהיה חסיד



ועניו / טוב הבחינה, / רחב המחשב וזך הבינה, /



עוזר נכשלים ומרחם – / כאשר אבלים ינחם. /



וכאשר יקצף – אל ימשך איבתו / ולא יעיר כל חמתו.



/ ואם יניעהו הרוח המתאוה / ישיבהו בשכלו



הנאוה / אל עמק שוה / הוא עמק המלך המושל



בנפשו ובתאותו / כאשר ימשל בעמו ובצאן



מרעיתו. / ואמרו עוד: יתחיב המלך לדבר אל



האביונים / ויהיו במושב הדרו חונים / ויגדיל



מעלת החכמים והודם / הכהנים המשוחים אשר מלא



ידם.

ואני במדות האלה נכבדתי, /



הגדלתי והוספתי – / והן הנה תכסיסי מלכותי, /



יקר תפארת גדלתי, / ועל כן אויתי לשמע אל



דבריך / ולהקשיב מאמריך. / ואתה ידעת / ושמע



שמעת, / כי המלך אשר באלה המדות חמדתו /



ויקח מהן עמידתו – / ראוי הוא לעבדו / ולשמח



בהודו, / וכל איש שכל ואיש מדע – / במראה אליו



אתודע. / ועתה שב עמדי ואל תירא מעמלך /



וידעת כי שלום אהלך! / ואתן לך עשר וכבוד /



וכסף וזהב ארבה לך מאד מאד, / כי לא בא בארצי



משכיל אשר לא הגדלתי / וחכם אשר לא כבדתי /



ועל דברי מוסר שאלתי / ואני אושיבך במושב אחי



ורעי, / אלופי ומידעי״.

וירא האיש מדבר המלך ואמרתו / כי



בחר בחברתו / ויאמר ״אעבדך שנה אחת ברוח



נכונה, / מראשית ועד אחרית השנה״. / ויאמר לו



״שבה עמדי / להרבות הדרי והודי / ועל נכון



תבונתך העמידני – / אנהגך אביאך אל בית אמי



תלמדני״. / ויעמד עמו בנפש חפצה ושלמה / עד



מלאת לו שנה תמימה.

וישמח המלך בדברי תבונתו / ויבוא



עד תכונתו / ויתענג עמו במשלים ומליצות /



קרבת הידידות חפצות: / פעם בשיר, פעם במליצה, /



פעם בסוד פעם בעצה, / פעם תבונות פעם תעודה, /



פעם חידה פעם הגדה. / ותתם השנה ההיא בדברים



ערבים / לשרת את השם קרבים. / ויאמר האיש



״שלחני אל מקום חפצי, / ואלכה אל מקומי וארצי״. /



ויצו המלך לתת לו הון יקר ונערב / מזהב ומפז רב



/ ויאמר ״אל נא תעזב אותי, / כי על כן ידעת חנותי,



/ ושבה עמדי שנה אחרת / לכבוד ולתפארת״. /



ויאמר ״גם לדבר הזה נשאתי פניך / למען אמצא חן



בעיניך. / ואל יוסף המלך לעכב חסדו / לבלתי



שלח עבדו!״ / ויעבד עמו שנה אחרת / בנפש יקרה



ונהדרת, / ויתעסקו בדברי התורות / להבין



ולהורות: / פעם דרשות פעם פשטות, / פעם כללים



פעם פרטות, / פעם פרוש פעם סברא, / פעם תלמוד



פעם גמרא.

ותתם השנה השנית בחכמה עצומה



אשר הלב כובשת – / בשלש עשרה מדות התורה



נדרשת. / ויאמר האיש ״אל תאחר אותי, / כי אתה



ידעת את עבודתי!״ / וישלח המלך ויתנו לו עשר



ונכסים, / אלה ברכב ואלה בסוסים, / ויאמר לו ״איך



תלך ותרחק / אל ארץ מרחק? / והנה נפשי קשורה /



בנפשך ההדורה. / ועתה לכה ונכרתה ברית / להיות



שארית, / ואתן לך את בתי לאשה / בתוספת



קדושה, / ותמלך אתה במקומי / בעבור הזכיר את



שמי: / כי אין לי בן זכר / וכאבי נעכר, / ושכלך



יעלה ארוכה לחלי / יחזק במעזי יעשה שלום לי. /



כי לא חפצתי באחד המלכים / ובבני הנסיכים, /



אמנם חפצתי בישרך ובתבונתך / חכמתך



ואמונתך. / כי האמונה – ראש המלכות, / והחכמה –



שרש נסיכות, / והתבונה – עקר הדעת / אשר היא לכל



חכמה מודעת, / והישר – יסוד כל בדק, / בו מלכים



ימלכו ורוזנים יחוקקו צדק״. / ואמר החכם בזה משל



עליון / וזה צורת דמיון:





         העולם – פרדס יתנהג בטוב הדור,

         הדור – שלטון ישגבהו החק,

         החק – הנהגה ישמרהו החכם,

         החכם – מלך ינשאהו השכל,

         השכל – רועה יכבדוהו אנשי הצבא,

         הצבא – עוזרים יסעדם הממון,

         הממון – עשר יקבצהו ההמון,

         ההמון – עבדים יסמכם הישר,

         הישר – מאזני צדק / מחזיק הבדק,

         והוא – הצלחת תבל ומהלליה,

         המאיר לארץ ולדרים עליה.





ואתה עתה, ברוך יי, / מחמד לבי



ורעיוני, / אחרי אשר האמונה – גזעך / והחכמה –



שרש מדעך / והתבונה – מקום מרעך / והישר –



אורך וישעך, / לך יאות כל מעלה נסוכה / ונאה לך



אך המלוכה, / ולהיות שר ונגיד על כל המוני / בך



בחר יי, / ובשתים תתחתן בי היום / למצא



פדיום: / בבתי העדינה ובמלכותי – / אתה תהיה על



ביתי!״

ויאמר האיש ״ומי אנכי ומי חיי / חפצי



ומאויי, / כי אהיה חתן למלך – / ואני נקלה



הלך?״ / ויאמר המלך ״איני חפץ במהר ומתן / כי



אם איש משכיל ודעתן. / ועתה מהר למלא שאלתי /



ולעשות את בקשתי, / כי היום קצר והמלאכה



ארכה / ובעבותות החפץ משוכה״. / ויאמר האיש ״גם



את הדבר אשר דברת אעשה / ובצל מאמריך



אתלונן ואחסה, / ואטש בעבורך מחשבות לבי



ורעיוני / כי מצאת חן בעיני״. / ויאסף המלך



פרשיו וגדודיו / ויעש משתה לכל שריו ועבדיו /



וידבר באזניהם דברים נמרצים, / אהבת האיש



חפצים, / מקדישים ומעריצים. / וייטבו בעיניהם



מליצותיו / ודעת שפתיו / ויקבלו אותו לאדון



ולמלך / כל עשיר וחלך / ויתן לו המלך את בתו



לאשה / והוציא את אבן הראשה. / וישמח כל עם



הארץ והעיר שקטה, / רפא שבריה כי מטה, /



ויקראו לפניו ״אברך״ / כאשר כלו לברך.





ויעברו ימי המשתה והשמחה /



בהשקט ובבטחה, / ויבנה לו האיש בית מדה / את



כל שכיות החמדה, / ויכרה בור אחד תוך חצרו /



להשלים הודו והדרו, / ויבן לאשתו בית מדה, /



משוש ומחמד עינים, ומגדל וראשו בשמים. /



וינשה ביפיה את בית אביו וישכח – / ושמח את



אשתו אשר לקח. / ותהר האשה ותלד בן ותאמר



״ילד בן לנעמי /להיות נגיד על עמי״. / ויקרא שמו



עובד להראות מעלתו – / והיה בכזיב בלדתה אותו! /



ויגדל הילד ויגמל / ואביו חסד לכל בני



המלכות גמל, / ויעש משתה והנחה למדינות /



ומשלוח מנות.

ויהי היום ויעמד האיש בפתח אהלו /



בחצר הכבוד שלו / והיה משתעשע בספר אחד



נמרץ, / והנער רץ / וישחה אל הבור וישפל /



ידכה ישח ונפל. / וירגז האיש לקול הילד / כי



נסע מאתו אל מתי חלד / ויאמר ״מי יתן מותי אני



תחתיך / בבוא עתך!״ / וימהר וירד אחריו אל הבור



וזרוע עזו נשבר, ויבקש ולא מצא הנער, / וינע



לבבו כנוע עצי היער. / ויפן כה וכה / הלוך ילך



ובכה, / וירא את המכשף אשפיזו / גאותו ועזו



/ ויאמר לו ״למה תבכה / ודמע תפכה?״ / ויאמר ״כה



קראני היום רעדה ופחד / להשמיד אותי ואת בני



יחד״. / ויאמר ״ומאין לך בן? / אל תהיה כסוס



וכפרד אין הבן, / כי כל מעשיך הבל / צירים וחבל!״ /



ויאמר האיש ״הלא אנכי גודר פרץ / נורא למלכי ארץ:



/ כי בת מלך גדול נשאתי / לי כפר מצאתי, / ויהי לי



ממנה בן גאה / יפה תאר ויפה מראה / ויגמל



הילד ויהי רוכב וחונה / על משענת הקנה, / ויפל



אל הטיט ואל היון נחבר / אל הבור הזה אשר



במדבר, / ועד לא מטה לארעית גבא / מת או נשבר



או נשבה. / ועל זה בכיתי לקשה יומי / אריד בשיחי



ונאמי: / הנה אנכי בשברי תועה / כיולדה אפעה /



כעלה נדף אבלה, / כי ארד אל בני אבל שאולה״. /



ויאמר לו המצרי ״אל תעצב! / קום והתיצב / ושוב



אל לבך ותנוח – / שגם זה הבל ורעיון רוח: / וכל



הענין אשר ספרת / והימים אשר עברת / וכל



המקרה הגדול והפגע – / לא היה אלא כרגע. / והנה



בידי הכוס אשר שתית / והתרעלה אשר מצית, /



ובעבור תבין חכמתי ותגידנה / למען הראותך



הובאת הנה!״





וירגז האיש מאמרתו וניבו / ישב



יתפלא בו / ויאמר לו ״נעניתי לך, כי אין כמוך



איש מכשף / אשר רוח חובר נושף / ואין דומה לך



בכל תעלומות נרמזות / אחרי הודיע יי אותך כל



זאת. / ולך משפט הבכורה / ובידך כח וגבורה /



להראות ענינים / כנף רננים – / במושב חפצך



ישכנו ויחיו, / ושתו ולעו – והיו כלא היו. /



ועתה אעבדך כפי כחי / ולכל בהם חיי רוחי /



ותחיני ותחלימני, / ועתה אל תכלימני / העמידני



על אמונת עתך / אתהלך באמתך״. / וישב עמו



ימים אחדים / ללמדו ספר ולשון כשדים.
