


	
	נורא עלילה
	




נורא עלילה / המצא לנו מחילה / בשעת נעילה

אליך צועקים / ישישים ויונקים / מבקר ועד לילה / בשעת נעילה

נגדך נורא ואיום / עמדו כל היום / מתחננים בתפלה / בשעת נעילה

ישעך סוברים / עם חתימת ספרים / שואגים במלה / בשעת נעילה

יחתמו לחיים / והוציאם נקיים מכל חטא ועולה / בשעת נעילה

ורצה ענוים / והפק מאוים / לנצבים בחילה / בשעת נעילה

שגבם מפשע / ונקם מרשע / כי הם נחלה / בשעת נעילה

פנה אליהם ברחמים / צור שוכן מרומים / לעדי עד סלה / בשעת נעילה

ברך שאריתם / וקבץ גלותם / ברנה וצהלה / בשעת נעילה

רומם את ראשם / ושובבם לבית מקדשם / בשמחה וגילה / בשעת נעילה

יחיד שוכן שמי ערץ / בורא קצות הארץ / אשר על כל נעלה / בשעת נעילה

צפה ענים ודלדולם / ושבר מוטות עלם / ושימם לתהלה / בשעת נעילה

חן בחסדך עם זה / ותפלתם אל תבזה / אל נאדר בגדולה / בשעת נעילה

קבל שועתם / עת חתימתם / צור שוכן מעלה / בשעת נעילה

חזק ידם ואמצם / ומכל צרה חלצם / כי הם לך סגלה / בשעת נעילה

זמן קרב נא לפדיונם / ושמחם מיגונם / וחיש יום הגאלה בשעת נעילה.


מקור: הארץ עמ׳ 8 ערב יום כיפור 15.9.2021.
