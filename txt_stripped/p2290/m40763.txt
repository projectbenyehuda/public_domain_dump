


	
	[אלהי צמאה נפשי ורוחי]
	




[חי במאה הי"ז במנטובה, מודנה ובוסטו]





͝– – – ͝– – – ͝– –

אלהי צמאה נפשי ורוחי / לך גוחי
ומבטחי וכחי



בעת אראה ואשתאה בפלאי / המון צבאי יצוריך
לנכחי



גדול דעים מאד נעים אשר עם / היותך ראש לכל־ראש
רם ונצחי



דרשתני חקרתני למיום / יצרתני בכל־רבעי
וארחי



הלא תחקר בתוך מקור ותתיך / כמו חלב יסוד גופי
ולחי



ותקפיא כגבינה עור ובשר / לבוש יכשר אלי
נפשי לשלחי



זבול מבצר ומצע צר כאוצר / מעי אמי הכינות
לנוחי



חשכי תאיר ותגיה ותזהיר / בהלך נר עלי ראשי
ומוחי



טהורה היא נשמה התמימה / שביב אורך אלי גופי
לפקחי



יצרתה אל ושוכן־עד למסעד / וחיי־עד לגוף נגוף
סרוחי



כבוד הודה ועז ידה היותה / מנהלת ומושלת
בנצחי



לגופי זה, ואחזה מבשרי / לך צורי ברב חכמה
לשבחי



מלא עולם כליל הדר ונשלם / יצרתני דמות אילן
וצמחי



נטיעתי מקום טהרה בראשי / והוא שרשי ותו חיים
במצחי



סעיפי הם זרועותי וידי / ירכותי ורגלי ציץ
ופרחי



עצי־עדן ליפיי לא ידמו / בעת קרבי לך אלי
וגוחי



פרי צדיק בעץ חיים פעלים / עשויים באמת ישר
ורוחי



צבי עדיי עזוז שמת לכל־יו־ / שבי תבל כרב חובל
ומנחי



קבועים בי עליות גם שניות / ותחתיות כמו
היכל נכוחי



רמזים אל־שלשה המדורות / יצירות במאורות הם
לזרחי



שערים בי חמשה עם חמש שו־ / ערים השומרים
תמיד בפתחי



תהלתך ידבר פי יברך / בשרי שם קדשתך
ויחי



ידידות משכנותיך יי / צבאות, אכספה תמיד
בטוחי



ולך אקד כפוף קדקד ואשקד / לספר נוראותיך
בשיחי



סמך אותי אילותי וקומה / בעזרתי בקומי או
בשטחי



פנה אלי לקול מלי ואחלי / וממצר לך אנוס
בברחי



במו עיני ומעיני יי / בתחנוני לך אפנה
בבטחי



ראה קוני ורבוני באזני / אני אקשיב לתורתך
ולקחי



וכי אטעם בצוף נעם וגשוש / בחוש משוש ואפי
בהריחי



כאחד יהיו אצים ורצים / נחוצים אל־רצונך בל
לזנחי



משוש חיי תרננה שפתי / לזמר־לך בחם לבי
ורתחי



אדני נגדך כל־תאותי / וחשקי בך לדבקה אש
קדוחי



ולך ארוץ מאד חרוץ בכל־לב / ועם נפשי ונשמתי
ורוחי



ראה עניי יסוד נויי ועדיי / ועל־מריי שמע בכיי
בצוחי



בגדל חסדך עלי תכפר / לשגגותי ואשמותי
בסרחי



ירה יצרי והוא צרי וקטב / מרירי רודפי תמיד
לטבחי



נטה צדקך ובו תקח בצלך / יחידתי וגם גופי
שחוחי



ומה רב טובך צפון וספון / יהי חלקי מנת כוסי
בטרחי



ויוסף עמך לראות פלאות / ונוראות ובן ישי
משיחי



ואל ברוך תברך נחלתך / לבקר עוד בהיכלך
בזבחי



אזי מאור בנוך יהי אור / וקבץ שם פזורי עם
ונדחי



מבשר טוב לבת־ציון יבשר / ויאמר צהלי רנה
ופצחי



למלכותך ולשכינת כבודך / אלהי צמאה נפשי
ורוחי.



(1659)
