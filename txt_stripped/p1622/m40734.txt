


	
	[מלך עולם / אשר נעלם] / רשות לקדיש
	



[מאי מיורקה, חי בפלרמו ונהרג שם
בן כ"ד שנה סמוך לשנת 1430]





רשות לקדיש


מלך עולם / אשר נעלם / שיר ותהלה / לך
יאתה



באימה ובמורא / אספרה / מעשיך / אשר פעלת



בראת / ויצרת / בששת ימים / כל כלית



השפלים / וגלגלים / ושכלים / בם כוננת



טובך יתמידם / ויעמידם / ואת־כל־אלה / ידך
עשתה



בך קימים / צור עולמים / אתה הוא / ברוך אתה



יתהלל שמה / דקודשא בריך הוא / לעילא ולעילא /
מן־כל־ברכתא.





שפע טובו / יעבר־בו / על־שכלים /
ראשיתו



ומשכלים / על־גלגלים / תעבר את / השפעתו



והגלגלים / בעולם שפלים / כי בידם אל / נתן
ממשלתו



והם עגלים / וכגלדי בצלים / זה בתוך זה / ורוח
אין אתו



וכלם סובבים / ובהם כוכבים / ומהללים / כל־יום
אותו



וזה מספרם / לפי סדורם / תשעה המה / מאור
עשית.







הגלגל העליון / ברום חביון / הוא
ערבות / בחר אותו



שנים עשר / נשיאים קשר / יעקב שם / חקוקה צורתו



שבתי וצדק / יחזק כל־בדק / מאדים מול /
מלחמתו



לשמש שם / אהל שם / ואין נסתר / מחמתו



וגם נגה / אשר יגה / וכוכב חמה / בכתיבתו



והלבנה / היא תחתונה / תוסיף תגרע /
בטובתה.





ראש ושלטון / הם בתחתון / כי־כן רצה /
ברוך הוא



חמר הראשון / התחלה וראשון / אל־גלם כל / מרכב
הוא



ומתפעל / אינו פועל / כמעשה לבנת / ספיר
מעשהו



כל־צורה / מנו נעדרה / והארץ / היתה תהו



פושט צורה / ולובש צורה / כי ההעדר / דבק לו הוא



קבל צורות / ספורות / הם היסודות / אתה יצרת.





מהיסודות / קלות וכבדות / נפעל בחמרם / פועלים בצורתם



זך עב חבר / אש ואויר / מים ועפר / הם ארבעתם



טבע האש / חם ויבש / עליון לכלם / ומקיף אותם



האויר הלח / הוא חם ולח / שני אלה הם /
עולים לקלותם



ומים נבעם / קר לח טבעם / ועל־ארץ / מושבותם



היבשה / קרה ויבשה / ועל־בלימה / אותה
תלית.





והמרכבות / המתהוות / מערובם /
ארבעה הם



הדומם / עד תמם / והצמחים / למינהם



ומרגישים / בעלי חושים / אשר בלתי / מדברים
הם



מרגיש מדבר / אשר סובר / בצלם אלהים / עשה אתהם



הוא האדם / אשר בידם / נתן הכל / וירדו בהם



ובחרת אל / בישראל / ואת־תורתך / להם
נתת.





סתר ומגן / בעדם הגן / ופדה אותם /
מכל־צרה



למען שמך / תכונן עירך / ותקבץ שה / הפזורה



וגם תביא / הנביא / אליהו / בבשרה



וישיב לבב / הולך שובב / לפני בוא יום / אל הנורא



והגואל / שלח הואל / בימינו / במהרה



תפדנו / ותקבצנו / ותביאנו / אל־עיר
בחרת.
