


	
	[יה מלכי מנעה מבכי] (רשות לקדיש)
	



[חי ברומא בסוף המאה הי“ג ובראש המאה הי”ד]






רשות לקדיש של סכות





יה מלכי / מנעה מבכי / עניה סערה / לא נחמה



ודבר על־ / לבה וגעל / איש שעיר / ובן־האמה



אשר חרפו / וגם גדפו / עקבות משיח / שוכן רומה



ברב אונם / ובגאונם / יאמרו / ידנו רמה





יתגדל / ויתקדש / שמה רבא / בעלמא.





והותר בני / תמותה ובנה / את־הר ציון / אשר שמם



כי על־זה / בשרי רזה / ולבי דוה / ודומם



ואז אודך / ולכבודך / את־בית־אל / נרומם



השב שבותי / ומחלתי / רפא־נא לי / ואותי תקומם





אחלי אבי / אל־ימשלו־בי / זדים ואנשי / דמים ומרמה.





אשר אתה / מעט קצפת / והמה לרעה / עזרו



טמנו־לי / רשת לרגלי / חנם שחת / לרגלי חפרו



הבט שבתם / וקימתם / כי את־רוח / קדשך מרו



זכר איום / להם את־יום / האומרים / ערו ערו





כמעשה ידיהם / תן להם / על־הגלותם / גלות שלמה.





בחרתני / בחנתני / בכור עני / ביד זרים



כהודעת / והשמעת / לאבינו / בין־הבתרים



ובחזיוניו / נגזר לבניו / במרדם בא / שעריו נכרים



ולהגלותם / מאדמתם / טרם ירדו / שאול ככופרים





כמראה / אשר הראה / חשכה גדלה / נופלת ואימה.





חזק עמך / ויהי נעמך / עלי וחסדך / יקדמני



חיש קצם / ראה לחצם / פנה אלי / וחנני



זרעם יכון / ויהיה נכון / לפניך / ותציבני



זכר דבר / כמדבר / על אשר / יחלתני



קרב מהר / עדתך טהר / מכל־טמאה / כהבטחתני





קומה והעיר / את־שם העיר / מיום היותה / יי שמה.





אמן יי / יאמן לעיני / דברי נביאי / וכל־חוזינו



מלכים יראו / ויבואו / להשתחות / למלכנו



נפלאות / נוראות / תחזינה / עינינו





בשובך לציון / ברחמים ובחזיון / כי ברוכי / יי המה.





סלה תשכן / בתוכנו ותכון / בית קדשנו / ותפארתנו



לעד תקומם / וגם תרומם / קרן לדוד / משיחנו



העם אשר / בחרת תאשר / לעין הכל / כבשרתנו





אלי אלי / יי חילי / ושמו למשגב / לי ולחומה.
