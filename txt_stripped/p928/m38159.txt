


	
	ובעת ההיא העיר ה'
	








הישוב והמדבר





– – – ובעת ההיא העיר ה׳ את רוח חכם אחד אמתי / ושמו אתי הגתי / ולו רעה נאמן איש צדיקי / ושמו חושי הערכי / וישבו יחדו בבית לחם יהודה / על התורה והעבודה / ומדי שנה בשנה הלכו בגלגול לסבב את בית אל / לדעת מה יעשה
ישראל / ויראו כי רבו רעות האדם באדמה / אין תורה ואין חכמה / אין יראה ואין אהבה כי אם תחבולה ומרמה / ויאמר אתי הגתי: ׳שמע נא חושי רעי ועמיתי / השמת לבך על בני דורנו / כמה הרעו והשחיתו מאבותינו / כל חכם לב יבא לספד לצרה ולבכות על המצוקה / וכל חונף, ולבבות גונב, כאזרח בישראל ישב בסכה / עשויה לצל ולענג / וכל חכם לב נמס כדונג / כי כל איש מרמה ודמים / בעיני הדור חכמים / וכל אנשי צחוק ולצון / לפניהם לרצון / פנימי וחיצון / ויאמרו ׳רדה!׳ / לכל ממשל
ובעל מדה / ולכל בעל מוסר יאמרו סר / וכה אומרים השרים וההמון: / ׳אין חכמה ואין תבונה ודעת זולת אל הממון׳ / לכן אתה חושי, רעי ועמיתי / ונאמן ביתי, ננוסה נא מפני המצרים / אל אחד ההרים / אם במדברים או ביערים / במקום שאין בני אדם דרים / ומפני צרה גוברת אנו בורחים / ובמדבר מי יתן לנו מלון אורחים.





ויען חושי הערכי: / ׳אתה ידעת את דרכי / מנעורי עד עתה / אין לי אלא אתה / ובזמן אין לי ממך תמורה / ולא יפרידני ממך שום מקרה / זולת הקבורה / ועוד כל ימי חיי / אתה חלקי ומאויי / אם במדבר תשים קנך / שם אעמד לעבד
ולשרת לפניך׳.





ויקומו האנשים בבקר טרם יכיר איש את רעהו / ואל המדבר שמו את פניהם לדעהו / ויסעו מעמק הרפאים / ויבאו במדבר פראים / ומשם נסעו למדבר זיוף, אשר בתבערה / ויחנו במרה / ומשם נסעו דרך מדבר שור / קדמת אשור / וימצאו שם אילנות / נטועות על מי מעינות / ולפנים ישבו שם עם בינות / וכשרבה האפיקורסת והמינות / נחרבו המדינות העדינות / ונשארו האילנות / אל האריות למעונות / ושם קננו תנים ובנות היענות.





ויאמר אתי הגתי: / ׳פה בחרתי / להסתופף עם האריות והנמרים / ולא עם בני עמי האכזרים / כי חיות היערים / אינם בעלי בחירים / לעשות לנו רעה כבני האדם ארורים / ונקל לה׳ לשנות הטבע הברורה / מלשדד את הבחירה / ורק אותה לבדה נתן אלהים לבני האדם לענות / ולא כן עשה אל התנים ובנות היענות / ואם נעשה את רצון אלהינו / האריות והגורים יגורו לפנינו / ואם ה׳ עמנו / כל חיתו יער ישלימו אתנו׳.





ויכריתו משם אילנות / ויעשו להם מעונות / ושם ישבו ועסקו בספרי תבונות / ויאמרו אך טב / לתורה שבכתב / ויפה / לתורה שבעל פה / ולפעמים הלכו לטיל בפרדס / לידע את אמתית הדת / כפי החכמה המסורה / מבעלי כח וגבורה / אנשי רוח הקדש אשר קבלו בשפה ברורה / את הקבלה כפי התורה / ובשעה שאין לא יום ולא לילה / שלא לילך לריק ולבהלה / למדו מליצות ומוסרים יפים /
מחסידי הפילוסופים / כדי שלא יהיו סכלים כקופים בהשמים והעולם / ובנפש וצורה וגולם / ובדעות ומדות / קנו עשר ידות / במראה ולא בחידות /
והתבוננו במעשי ה׳ הנורא / בכל מה שבעולמו ברא / ועל הכל שם חושי את עיונו / ואתי הגתי שם עליהם את הגיונו / ונתנו שבח ליוצר בראשית / אשר ברא את כל בימים ששת. – – –
