


	
	דוד בחלום נטה
	



ליצחק ומשה בני עזרא






דוד בחלום נטה / ללון בחיק עפרה –

מה נעמה מטה / לו החלום פתרה.





נפשי פדות שנה / ארכו חבליה

ויקרבו יונה / רחקו גבוליה,

מופת ואות נתנה / מקצת פעליה –

מלחיה שחטה / לי כוס משכרה,

דר מענה לקטה, / מנהו ענק קשרה.





מה לזמן יפריד / ביני ובין חפצי?

אם אעלה יוריד – / עד יחטיא חצי.

מזרח לים יחריד / הן מערב ארצי.

אם הזמן יחטא / עלי ואין עזרה –

אברח ואמלטה / לשני בני עזרא.





שם הגביר יצחק / צוף אטעמה בשמו,

מלאו קצות שחק / מתמרות בשמו.

ספר יקר הוחק / על פיו וחותמו



בן־עט ובן־מבטא / ההוד והמשרה,



חרב בפיו לוטה / עד לה בפי תחרא.





משה, הינשה / מפי ורעיוני?

או איך אהי מחשה / על נד מאור עיני?

בושש מאד משה; / נא שאלו סיני:

שם אהלו נטה? או מסנה נקרא

יום סות נדוד עטה / מה מר ומה נורא?





עיני בנות מערב / אל פאתי מזרח,

לראות גביר ורב / וכרוב תמול ברח,

קוראות בשיר יערב / אל עוברי ארח:



״חביבי קד אכ׳טא; / מן חבס אלבדרא



עני? לקד אבטא / ואשגל אלסרא״.



(״אהובי החטיא (דרכו); / מי הסתיר הירח



ממני? אכן התמהמה / והעיק אף הכאיב מסע־הלילה (המתארך של האהוב אלי)״.
