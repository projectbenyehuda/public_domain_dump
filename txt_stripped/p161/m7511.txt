


	
	יודעי מכאובי אל תתמהמהו מהביא
	



[יודעי מכאובי] / ר' יהודה הלוי


יודעי מכאובי, / אל תתמהמהו / מהביא



צריי מגלעד ‑ / הוא אליהו / הנביא!

העתירו בעדי / אל אבי – אולי / יעתר



ויגלה סודי / וחזון משכילי / יפתר.



עד כמה, דודי, / מלפנות אלי / תסתר?



וצבי מאס בי ‑ / מתי אעלהו / בית אבי,



הדוד הנועד / להיות מחנהו / בלבבי?

ויבושש רכבו, / תמו מועדיו / ואין,



מאין יבא / עזר לידידיו, / מאין?



אצמא אל ניבו, / כי טובים דדיו / מיין.



מלכי וכרובי, / כספי אעדהו / וזהבי,



ובבית הועד / ייני אשקהו / וחלבי.

דודי, זנחת / אותי שוממה / תוך מוקשי



אכן שלחת / פרא בן‑אמה / לחפשי,



ואדום שמחת, / כי ידו רמה / על קדשי.



שמחת אויבי / ותרכיבהו / מרכבי,



תן מתניו המעד / עד תשכיבהו / משכבי!

הלב לא ירך, / אם הקץ ארך / וירבה,



אך בך יתברך / שארית עדרך / הנשבה.



גר מאז נערך / וכוכב דרך / לא יכבה.



אם עבר טובי / ועיני כהו ‑ / כוכבי



לא יכבה לעד, / כי הנר ההוא / מערבי!
