


	
	אשחר עדתי
	



קינה על החורבן (לראש חודש אב)






אשחר עדתי – / ולמאד עניה, / ואפקד נותי – / ורבה שאיה,



גדרות דחויות / וקירות נטויות / ועיפה ואפל / ועיף וציה.



וקפוז תקנן / ועורב יקונן / ויענות ותנות / מתנות אניה,



לטאה וחיה / ואיה ודיה / ומספד ואבל / וקול תאניה.


שאלתים: ״ואיה / עפרים, צביי / חמודות, ויוני / אפיקי עליה?


ואיה עליות / באקדח בנויות / וגבהי ארזים / ואבן שתיה?



ואיה דבירים / יסודי יקרים / וחצרות עפרים / והיכלי צביה?



ואיה עלמות / לבושות רקמות, / כבודות פנימות / בחדרי שכיה?



ואיה עדנים / ואיה שמנים / וייני רננים / ודת השתיה?



ואיה עבודה? / ואיה תעודה? / ואיה כהנה? / ואיה לויה?



ואיה מלוכה / במשפט נסוכה / בשפה ברורה / באפס רמיה?



וענו בניהם: / ״שאול משכניהם, / בחורי עפרים / ועמקי נשיה.



ועלו בגבם / דשאים ועשבם / למרמס יעלים / ומרעה גדיה.



והורום רגבים / בחצי רקבים, / משוכים שכובים / בתוך מאפליה.



ובקר וערב – / לקרח וחרב, / למכות זרמים / וזרמי כויה.



והלכו במו ים – / ואין קץ לשבים / לענות נפשות / וכלות גויה.



וים הלכו בו – / אבדון נתיבו / ומות לחובל / וקבר לציה.



חרדה ושמה / ופחד ואימה, / ואין קול ועונה / בכל האניה.



ועדה הדורה / צמיתות מכורה, / נתושה, נטושה / לחרב נטויה.



וזרעם נפוצים, / לחוצים, מחוצים, / בתולדות נגעים / ומכה טריה.



ובחל אביהם / באמם ובהם / ומשלו עליהם / בנות נכריה –



אמורית וחתית / והגרית וכתית / והמדינית / והמואביה״.



אלהים ברחמיו / ירחם יתומיו / ויקבץ זרוייו / לארץ נקיה.



אשר רם ונעלה / והוריד – והעלה, / ומחץ – ורפא / והמית – והחיה.



ישובב הדרו / ויבנה דבירו / וישכן חצרו / ויאסף דחויה.



מחדש חדשים / יקבץ קדושים, / אנשים ונשים / לעיר הבנויה.



וזה החדש / לטובה יחדש / ורצון יצו אל / רב העליליה.
