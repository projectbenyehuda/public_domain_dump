


	
	יבשו אלי
	



אופן. חתימה: יהודה






יבשו אלי / וספו אלי / ואיד בא אלי / ארחי גדר.



חדלו רעים / ואפסו רועים / ושחתו רעים / גפן סמדר.



ולסוד מפלא / אין איש גולה / והדר גולה / וחסד נעדר.



אויבי רבים / ושוטני אורבים / לי כעקרבים / בין קוץ ודרדר.



אויה לי כי / גרתי משך, / שכנתי עם / אהלי קדר!





ההפך עלי / רע פעלי, / נבלו עלי / וצל הודי סר,



וביד דומה / אשכן דומה / בתוך על, דומה / כצפור במוסר,



ועלי משא / הכביד משא / כדבר משא / בעלי מוסר.



ועפר ועיפה / בעפר ועיפה / שכנו יפה / ועדיה הוסר,



חלקו בחבל / אחזת חבל / אחוזת חבל / נתונה במאסר –



אמר הצר / להחשיך, והצר / ועשה יום צר / כאשר נדר.





ובני ערב / חגורי חרב / הפכו לערב / יומי וחשך,



זרע עדה / כשחל עדה / וכליו עדה / וקשת משך.



ולך אנוס, / אלי, ואנוס; הראה לאנוס / זרוע קדשך,



וגזוז כשה / בחסדך כסה / והיה מחסה / תמיד לדורשך.



הקשב ניב דל / רק לך נבדל, / בכל יום ידל / ממרט ממשך –



במצוק ומצור / ומחסור ומעצור / בקדר וחצור / שמשו קדר.





דלי ישורון / טובך ישורון, / לשמך ישירון / על פי נבלים.



חרב תער / מתוך תער / ותעביר תער / עח גו נבלים –



ותעל שמורים / מעל שמרים, / תסיר שמירים, / תנפץ נבלים.



כליהם הרק / לדורשי הרק / וחרב הרק / לעובדי פסילים,



שרשם רצץ / ופרים קצץ / ופטיש מפוצץ / שלח באלילים –



וידעו רחוקים / כי צבא שחקים / בפיך יצוקים / איש לא נעדר.





הדר ערבות – / שירות ערבות / חצובי להבות / לו יערכו,



גופם אשים / לא כגוף אישים / וכלם קדושים / עז יתמכו.



קדוש נאומם / וקדוש מקומם / וקדוש הדומם / אשר ידרכו,



אומרים ״קדוש / קדוש קדוש״ / עד כלות שלש, / לא יחשכו –



שוחחים הודו / כל יציר חלדו / צור אשר מדו / שת הוד והדר.
