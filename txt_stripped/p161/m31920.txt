


	
	יוני גאיות
	








יוני גאיות / הומות בפחדן,

גלה מחמדן / ונקלה נכבדן.

יתרן וחמדן – / ליתרן וחמדן,

להרג ואבדן / הצבו לבדן.

זה אלף שנים / צנינים בצדן,

כי לדינן אין דן / ויספרו כל חלדן

עדנין ועדן / ופלג עדן.





החשכו נשפיהם / ודעכו רשפיהם

ורפו כנפיהם – / וקלו רודפיהם,

ונערו כפיהם / מכבוד כספיהם

ובאשר אפאיהם / תנים שואפיהם.

ערב לנשיאיהם, / אדום לאלופיהם,

ומשך ופלשת / נתנו על פיהם –

ונבדלו הם לבדן / ואין עוזר כנגדן.





ונביות וכושים / רדו בחמושים

והניחום רטושים, / בכל פנה נטושים.

גדוליהם דרושים / ובתיהם בלושים

ונשים ואנשים / בכל רגע ענושים.

ומרב הכחשים / דלקום נחשים

והניחום חבושים / ובושה מלבשים –

נתונה על מדן / כמו עורן ועדן.





דברי הימים / נפתרו לחכמים –

וקצים נעלמים / סתומים וחתומים,

ואין נביאים וחוזים / ואין אורים ותמים,

וצדק עולמים / נחלף בקסמים.

ובנות מלכים / שפחות לעמים,

פרקו הנזמים / וזרקו הבשמים –

ועבדן יתעדן / בקנמונן ונרדן.





העבר טבעתם / מעל יד געתם

ומעל ראש קדר – / את פאר מגבעתם,

ואוהבי מפלתם – / השקם קבעתם.

וגלה קץ נחתם / והאר אור נעתם

ודגלך הרם על / דגלי משמעתם,

ומחנה כהניך / תנה בתוך ארבעתם –

ויציעו מרבדן / מבאר צבע ועד דן.
