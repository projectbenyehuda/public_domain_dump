


	
	יונה בפח מצרים
	



אהבה. חתימה: יהודה






יונה בפח מצרים / בין אויבים וצרים, / קנה עזבה –

למה באף נטשתה / אחר אשר משכתה / בעבות אהבה?





הומים לנוד ידידי / מעי וכואבים,

איך יעטו רדידי / צרים ואויבים?

איך נשאו רבידי / בוגדים ואורבים?

איך רעיה ערומה / מסות יקר ערמה / דומה שכבה,

אחר אשר שמאלו / לי מחסה, ודגלו / עלי אהבה?





וזכר ברית נעורים, / דודי ואוהבי!

איך תעזב לזרים / נחלתך שבי?

או אם עלי בתרים / תברח כמו צבי –

אלי נטה, אדוני! / אך אם ברב עווני / עירי חרבה –

סורה שכן בקרבי! / הן אהלך בלבי / רצוף אהבה.





דורשי פדות וצלם / נגלה ונחשף,

הוד יעטו, וחילם / צור יחבש, ואף

יכבש, ולא לעולם / יקצף ויאנף.

שריד יצו להשאיר / מהשבי, ויאיר / שמש ערבה,

עזרי, חמל עלי עיר / קדשי, ושוב והעיר / את האהבה!





העוד אקו לחלות / הר הצבי, ועוד

אשיר ימי כלולות / שיר אהבה לדוד?

מה תפרצו לגלות / תכלית ימי נדוד?

אשביעכם בעפר / הרחיק נדד והפר / את האחוה,

אם הבכי תעירו, / רעי, ואם תעוררו / את האהבה.
