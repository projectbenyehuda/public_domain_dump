


	
	אותותי במה אדע
	



לג / שמות ג א – ומשה היה





א


אותותי במה אדע / בחור בעת אץ לדע

גלתה אדון כל מדע / דרכיך אותו לידע.





הבמלאכתו היה מהיר / ובמעלליו זך וזהיר

זמנתו צאן להנהיג / חילי צאן אדם היות מנהיג.





טרם נס יראה / ידעתו כי נסיך יראה

כלולי נמתה זה ירעה / למען כי באמונה היה רועה.





יקום דברך / יעמוד דבורך / למגנינו כדברך.




ב


מדדה היה איש האלהים / נוהג ובא בנסי אלהים

שאתו עיניו בהר אלהים / עליו נחה רוח אלהים.





פלידת אש אוכלה / צפה וסנה לא אכלה

קימת לפידים שר ונמלא / רך יונקתיו לא נאמלה.





שבח פעלך בקרוב תראה / שדי מרחוק תראה

תקפך עד לא יראה / תחלה מלאך פניו נראה.





חסד ורחמים / תט ממרומים / להחיות בטל רדומים.




ג


יה חסוף ידך / ואסוף צאן ידך / במטה עזך / ובשבט מעזך



נדמינו לצאן תועה מפורד ופועה / באין לה מרעה וגם לא רועה.

ירעו כדברם……..




ד


בכל צרותינו לך צרה / והבטחתנו עמו אנכי בצרה

כי אתה אלהינו ואנחנו עמך / כי אתה רוענו ואנחנו צאנך

במצרים בעת הצר לנו שונא / נגלית במקום צר מתוך הסנה

פרוכים בלבנים לנו עת ראית / תחת רגליך לבנה הראית



ראו ראיתה / וידוע ידעתה / פקוד פקדתה / וקרוא קראתה



למשה ממרעה / מתוך מראה / ועת אשר ראה / אשר לא הראה



כבודך הנראה / באימה ויראה / האמירך קדוש.




ה


אחר המדבר ציר נהג צאן / אחר המדבר הנהיג עמו כצאן

בלא רגלים רץ והריץ מרעיתו / ללכת למקום חזיון מראיתו

גדולי דשא היו נבראים לפניו / ואחר כך היו נבלעים מלפניו

דרך גדולה ליום אחד הלך / כי אוהב מישרים ישר לו הלך

הר האלהים עת כי הגע / מטרחו הונח ומיגיעו הרגע



ותחלה נראה לו כדמות מלאך / ל…….. דמותו כדמות מלאך



זה לפי דרכו חנכו לראות / להיות בקי בכל מרא מראות

חזק לבו כשר לבת אש / בעבור ללבבו בכל מיני אש

טהור בתוך טמאה יקרו הופיע / גבוה על סנה כבודו הודיע

יען כי צרת עמו היא צרתו / וישועתם היא ישועתו.




ו


אשר נתכנה יקותיאל / כי בנים הקוה לאל

בסבלות אחיו לראות כהואל / נודע כי הוא הגואל.





גדול מרבית בת פרעה / ומנו ובו קם ופרעה

דגול אשר………… / דגלו אהבה…….





ה……………….. / ……..צאן אל אמונה……



ו……………… / ……………..





ז………שר במפעליו / וטוב מעשרו כונן עליו



חשוב הרהור גלה לו ונגלה אליו.





טעון את משאוי אחים / אשר היו כשושן בין החוחים

יגע לידע עד כמה הם נאנחים / וידעתו כי הם ננוחים.





כולל הקץ והוא קרוב / ופרט חשבון והוא רחוק

לבו השכיל כי הוא קרוב / כקרב ויאת מרחוק.





מנפש בהמתו הודיע / ומרת נפשו לבו יודע

נודע לו מתי שוא יודע / קץ ידוע תדע להודיע.





סמוך ובטוח בשבועה / אשר הבטחה לאב בשמועה

עודנו משתאה ברן ישועה / צלחה וצמחה בה בשעה.





פקוד מילדותו איש גדול / להפקיד על גוי גדול

צעד לראות המראה הגדול / מיכאל השר הגדול.





קראוי ומכנה שמות עשרה / בחכמה אשר תעוז מעשרה

ריצוי זה שם אשר הקרא / חבב ובו פעמים נקרא.





שפל ועניו ונבחר בו / ובהשפלה ובענוה תחלה דבר בו

תקיף אשר מלא כל הארץ כבודו / ועל הסנה לחץ כבודו.





ובכן וירא מלאך יי אליו (בלבת אש)





אש אשר היא אוכלה אש אש בוערת ביבשים ובלחים

אש גוחלת בשלג וקיטור אש דומה לארי רובץ

אש המראה כמו מראות אש ודי כי לא כבה

אש זורחת ומשוטטת אש חוששת ומתלקחת

אש טסה ברוח סערה אש יוקדת באפס עצים

אש כי כל יום מתחדשת אש לא נפחה מאש

אש מעלה לולבין אש נצוציה הם הברקים



[אש ס………… אש ע…………….]



צאש פלדות פל……. אש צ………..



אש קודחת עד שאול תחתית אש ר…………… קרה



אש שחורה כעורב אש תלתלים כצבעי קשת.




ח


[…….?] מתחלפת ברב דמיונות / על הסנה



בלהבים / ועל סיני בלפידים למעלה שביבים תלהט /



ולמטה רשפים תלחך / ומנה גבילת השרפים / לכן



בה לא נשרפים / ומזעה אשר הם מזיעים / נהר



דינור מושך אור / וגדותיו גושי שלג / האש לא



תרתיח שלג / ומשלג לא יצין אש / כי יוצר האש



ובורא השלג / מתויך שלום בין אש ושלג / שופט



באש ודן בשלג.

ומי יסתר מחמתו / ומי יעמד בקרתו /



כי אם ישראל עמו / אשר באו במים ובאש / והם



משולים במים ובאש / לכן לא ישטפו במים ולא



יכוו באש / כי כבלולי מים ואש / יפתחו פיהם



בקדשה / להעריץ בקדש.




ט


בתום וצדק ואומן יקדישוך צבאות קדש תמיד יענו ויאמרו:



קדוש קדוש קדוש / יי צבאות מלא כל הארץ כבודו.



קדוש מהולכי תמים / קדוש מפועלי צדק / קדוש מדוברי אמת

קדוש למשה הודיע ישר והלוך תמים

קדוש חכמו ענות העקב ופעלת צדק

קדוש למדו שליחות הקושט ודברת אמת.





ממקומו בתום מפרכים צפה ודלג על קץ והוא במקומו

ממקומו ראה צדקת עמרם וקפץ חשבון והוא במקומו

ממקומו ידע אמונת משה ונגלה וצמח והוא במקומו.
