


	
	אב לבו כלבך
	



בראשית טו א. אחר הדברים באלה: במחזה









א


אב לבו כלבך מצאתה / בטהר לבו כלבו לו המצאתה

גאות חרבו בכל צר המצאתה / דברי חסידותו מציתה.





הממלחמה אחר הובא / ודיי הרהורים בלבו חבא

זדים אשר קטל בדן ובחובה / חרד פן יחשב לו חובה.





טעם השיבותו בעתירה / יופי רוב שכר [ל]ך אותירה



כי דמם תרתי לך להתירה / לך אנכי מגן אל תירא.





אדם בדמות צרתה / ובצלם צײנתה / מגן עם זו יצרתה.




ב


מה יתרון לי אב נתחנן / נפשי עגומה ולבי מתאונן

שמחה בלבי מה ירנן / עץ סרק אנכי ולא רענן.





פעלי מה בכל יקרי / ציץ עקרי עקרי במקורי

קנין ברכתי יירש ארורי / ריק אם אפנה וערירי.





שם צדק יש לי / שתל צדק אין לי

תתי לך ונתתה לי / מה אתן לך ומה תתן לי.





אזניך קשובות / לכל מחשבות / מטליל תחי [בר]וחות נשבות.




ג


יה מאמרות עשרה ח[תכ]תה / ובלשונות עשרה נבואה קראתה



[נבו]אה חזון צווי דבר אמירה / הטפה [מש]ל מליצה משא חדה



יקר לאיתן א[ז יי]דעתה / כי בשלשת היקרים שבהם לו נודעתה



ידוע דבר ואמר וחזון / [לדבר] ולאמר לשמע ולאזון.




ד


לאב עד לא חז מחזה / והבין בכל מעשה

נראיתה במחזה / ונהיתה לו מחסה

ואשר הרהר בלב / וחשב בכליות

ידעתו חוקר לב / ובוחן כליות

מאשר נתירא / נמתה לו אל תירא

ובדבר אשר נהם / בו בדבר נחם

ובאמר כי נתחשב / בו באמר נתישב



אנכי מגינך / מי ימגינך / [אנ]כי מחזיקך / מי יזיקך



אנכי משכירך / מי יחסירך / הבטחתו קדוש.




ה


יי אלהים הננך ……נך / אשר ברחמים ובדין חננתו בכל מתנך



בכן פץ אם אין לי / ברחמים תן לי / ונם אם יש לי / בדין ינתן לי



געה מה תתן לי / ויש לי ואין לי / מוליד יש לי / וילד אין לי



די שכר טוב בכל נתתה לי / ושכר פרי הבטן לא נתתה לי



הלא מה יתרון לי / בכל עמלי / [אם] בן אחי או בן ביתי / [יורש] אותי



ונמתה לו אל תירא לא מזה ולא מזה / לא יירשך לא זה ולא זה



זחל וערג / על אשר הרג / ….ה…. לדרוש / פן דמם תדרוש



חרד פן יהיה צדיק ביניהם / או נר עתיד לעמד מביניהם

טעם השיבותו אל תהי צדיק הרבה / רשעים הכחדתה שכרך הרבה

יראך כחכם ירא וסר מרע / ובכל רע אשר ירא טוב מפיך בשר.




ו


אלף המגן לאב תליתה / ועל מגני ארץ עליתה

במגן אשר אותו בירכתה / במגן אברהם נתברכתה.





ג……. שמש ומגן לו הייתה / ומגן [יש]עך לו נתתה



דגול בעת אליו נראיתה / מגנינו א…….איתה.





הראיתה לו שכרו……. [חו] / כי [לא ה]לין שכר שכיר אתו



ובשרתו ל……… מסגורתו / ולשלם לו משכרתו.





[ז]……ך לפניו תלך / בצדק אשר לפניו יהלך



ח[ז צד]קו לפניו מהלך / בצדק אשר לפניו יהלך.





טרוח עלי ירשתו / פן לזר [ת]היה מ[ור]שתו



ידעתו מראשיתו / כי באחריתו / בנו יירש אותו.





כבודך עליו גליתה / וכו[בד א]זנו גליתה



לענות אשר חג פיו המגדתה / [בטו]בות אשר בפיך לו הגדתה.





מראש צורים צור חצבתו / ומראש אמנה מאמין הצבתו



נוה מרומים חוצה הוצאתו / ו[צב]א כוכבים הצפיתו.





סחתה [לו כ]ה יהיה זרעך / שלושים כערכך מגזעך



עדי [עד] נרך [ל]א ידעך / אשר היאר[תה במ]דעך.





פלאות הפלאתה למימינך / וח[שבתה צד]קה למאמינך



צבי לצדיק…… בנינך / אשר הקנית קנינך.





קראתה למ[ק]ריא ת……. לך / וקרבתה ………ם לך



רחמים משלך אתן לך / וצדק בשמך אקרא לך.





שמועה טובה לאב [שמעתה] / ומשמועה רעה אותו שמרתה



תשורת שלישים מנו שעתה / ומש[ליש]ים אותו [הוש]עתה.




ז 1


אל תירא / אנכי תוחלתך / אל תשתע / אנכי שומרך.



אל תרעד / אנכי ר……ך / אל תקנא / אנכי קונך.



אל תצר / אנכי צורך / אל תפחד / אנכי פודך.



אל תערץ / אנכי עמך / אל תסמר / אנכי [סומכ]ך.



אל תנ[וד] / אנכי נוחך / אל תמוג / אנכי מגינך.



אל תלאה / אנכי לוויך / אל תכעס / אנכי כוסך.



אל תחת / אנכי ישעך / אל תטרף / אנכי טורך.



אל תחלה / אנכי חילך / [אל ת]זחל / אנכי זרועך.



אל תהולל / אנכי הלולך / אל תדאג / אנכי דגלך.



אל תגר / אנכי גואלך / [אל תבוהל / אנכי בוראך.



אל ת[אפ]ל / אנכי אורך.




ז 2


[שכר]ך באש אשר באתה / הרבה מאד בני[רים] אשר דבקתה



שכרך בהלי… אשר ועדתה / [הרבה מאד] בזכות אשר חשבתה



שכרך ב[ט….] אשר ידעתה / הרבה מאד בזכות אשר לבבתה



[שכרך] במלכים אשר נצחתה / הרבה מאד בשיח אשר עניתה



שכרך בפושעים אשר צדקתה / [הרבה מאד] בקושט אשר רביתה

שכרך בשבועה אשר שמעתה / הרבה מאד בתניים אשר תבעתה




ח


[שכרך] בדין / הרבה מאד ברחמים /
כי אלהים דין / ויי רחמים / בו קדשתה שמך
[בדין] / ובו נתקדשתה ברחמים / כקדושת
עליונים / ככ וקרא




ֿ
