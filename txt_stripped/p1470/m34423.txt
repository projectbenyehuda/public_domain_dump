


	
	אות לטובה
	







א






אות לטובה חשבתה / באחר למבול ישבתה

גם ביתה יחידים הושבתה / דגלים בתיבה ישבתה.





התרתה אסורים / והוצאתה א[סירי]ם



זכרתה סגורים / חשבתה סגו..ם.





טפם בתיבה עבור הצניעו / ישיבה עלי גיא נציעו

כי מעט היו והרבה הצבאו / לעת למשפחותם יצאו.





ישמיד צריו / ישמר צוריו / ימג[ר] צורריו / ימגן נצוריו.




ב






מחלציהם אדמה השריצה / נטיעתם בארץ הפריצה

שריגים גפנם הניצה / עולתה כעלתה נצה.





[פלטו ב]טוב מעט לצדיק / צמחי איש צדיק



[ק]… יסוד עולם הוא הצדיק / רכיו עמו הצדיק.





[ש]… וכחלה הפרשתם / שפע טוב עולם הורשתם.





פליטים מסגרירי שטף / בהסכר מעינות ובהס[תם]…



צפונים עד יעבר זעם קצף / ויצאו נצולים משצף.





[ק]….. שמותם כלולה להם / ומורא מוציאם עליהם



רחק או…… בחם / ונארר ונשתנה מהם.





שנים מאחד הוטבו / כי מעש[יהם] היטיבו



תבל וחרבותיה הושיבו … ובברכה זה לזה השוו.




ז 1






נה….ו / מאלה אפסי ארץ נאחזו / מאלה בתי בריות נבנו



מאלה גבולות גיא [נג…..] / מאלה [ד….] דורות נדרכו



מאלה המוני המונים נה….ו / נה….ו / ותורי ועדים [נו…]



מאלה ז…… זאת נזרעו / מאלה חדושי חלד נחלפו



מאלה טכס טירות ניט….. / מאלה יקרות יל….ם נראו



מאלה כנת כל נכללו / מאלה להקות למו…. [נ]לקחו



מאלה מחנות מושבות נמלאו / מאלה נוות נחלות ננחלו



מאלה סרידי ש… [נס…] / מאלה עובדי עדות [נע]שו


מאלה פנות פעל נפרדו / מאלה צדדי ציה נצ….


מאלה קיני קהלים נקראו / מאלה רבבות רבים נרבבו



מאלה שרשי ש…. [נש…] / מאלה תודות תבל נת….ו.




ז 2


בתקוה / [אשר למו ת….] / [בש….… / אשר למו ש…]



[ברצון / אשר] למו רציתה / בקושט / אשר למו [ק….]



[בצ…. / אשר למו צ…../ [בפ….] / אשר למו פעלתה



בעידות / אשר למו ערכתה / בסימן / אשר למו סדרתה



בנעם / אשר למו נצרתה / במחיה / אשר למו מניתה



בלווי / אשר למו לבבתה / בכעס / אשר למו כבשתה



בישע / אשר למו ימנתה / בטוב / אשר למו טיעמתה



בחיל / אשר למו חזקתה / בזכות / אשר למו זכרתה



בותיקות / אשר למו ויעדתה / בהשכל / אשר למו הוריתה



בדבר / אשר למו דנתה / בגזרה / אשר למו גזרתה



בברית / אשר למו ביררתה / באמירה / אשר למו אמנתה.




ח


כי תרבה להשיב אפך / ותשגה להחזיר



זעפך / וכלה לא תעש / כי למענך תעש / ורחמיך



על כל מעשיך / וחמלתך על כל ביריותיך / ודור



לדור ישבח / ודור לדור יתנה / וזה מזה ישמעו /



וזה לזה ילמדו / כי לריק לא נבראו / ובעבור זאת



נבראו / לברך שמך / להלל זכרך /ולהודות לך /



ולהאמין בך / ולהקדישך ולהאדירך / כאדירי



מעלה באדרת קדשה נאדר בקדש אדיר



וקדוש.




ט


יוצאי ממסגר אשר הוצאתה לרוחה יצואיהם יאמרו:



קדוש קדוש קדוש / יי צבאות מלא כל הארץ כבודו



קדוש מבעלי שם ומעשה / קדוש מבעלי לשון מבחר / קדוש מבעלי שכר טוב



קדוש…..



קדוש משם העמיד עוטה ציצית בשם ומעשה.

קדוש מיפת העמיד ממליצי דת בלשון מבחר.

קדוש מהם העמיד משכילים דבקים בברוכים.



קדוש……





ממקומו בחר בשם כי דרש אש דת.

ממקומו בחר ביפת הוגי נפת צופים.

ממקומו קרב מהם גרים וחושבי שמו.





ממקומו הוציא מתיבה ילדי…

[ממקומו] בנה עולמו מגזע תמים

ממקומו הוריש ל….. עולם לחלקים.
