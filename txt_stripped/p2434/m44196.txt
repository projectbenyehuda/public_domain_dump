


	
	הלנצח רעיה
	




הלנצח רעיה / צביה נקיה / לשפחה וגויה / חמדת מאויה

זרויה דחויה / בארץ מאפליה / סוערה עניה / הומיה כאניה

צועקה בוכיה / דמעתה על לחיה / איך צר ביד הויה / ובזרוע נטויה

הכה אותה בצדיה / ובחיל מכה טריה / עשה בה פליליה / עלי דת ופליליה

הציגה גלויה / בערום ועריה / שת גפנה לשאיה / שמה שערוריה

ועדנה גן רויה / למדבר ארץ ציה / מקום תנין וחיה / מעון דיה ואיה

אל רב עליליה / דר בשמי עליה / איך עדה המנויה / במפקד עשיריה

סגלה שלישיה / שומרה דת שלישיה / היתה כעוטיה / על עדרי נכריה

התמיד יפהפיה / כאמה בזויה / שים עניה תחת מריה / יוצר עולמים ביה

בשרה באליה / השיבה לנוה צביה / השבע בטוב עדיה / השב עדיה וחליה

חיש תהיה בנויה / בזיוה ובנויה / וכהנה ולויה / אל הר המוריה

בשירים תהלל יה / בתוך רחובות קריה / אנו ליה, וליה / עינינו צופיה.
