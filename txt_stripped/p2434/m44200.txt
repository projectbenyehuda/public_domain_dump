


	
	אל מחוללי
	




אל מחוללי / נא מחל לי / למחולי / הפך אבלי

שמע קולי / נפץ קולי / נטה קו לי / יסד חלי

רפא חילי / הסר חילי / אלי חו לי / קץ גאולי

צורי קום לי / השמד קומלי / נוטר קום לי / ומשפילי

חזק חלי / ברך חילי / קבץ חילי / אל גבולי

סחה שבצי / אלי שובצי / השב שבצי / הרם דגלי

נצר נצרי / צורי נוצרי / מידי נוצרי / וחוללי

זרה סירי / זרה צירי / זרה צירי / וגואלי

זרה חורי / הגבר חורי / קרא חורי / התר כבלי

סתר קירי / בנה קירי / אל חי קירי / ועז משלי

ארה אורי / ארה אורי / קומי אורי / אמר נא לי

שלח נפלי / הקם נפלי / יסד נוף לי / עיר היכלי

העלה ציצי / הפרח ציצי / גלה ציצי / גם מעילי

חדש רוחי / השב רוחי / כונן רוחי / ומפעלי.
