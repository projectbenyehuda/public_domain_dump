


	
	חזק בדק
	




חזק בדק / הדק חדק / כרת וקצר

אל תבטחה / כי תמרחה / נגעי מצור

גרש פשע / לחם רשע / אל תאצר

קום מהרה / והעירה / ואל תעצר

לדרש פדיום / מאל איום / כי הוא ישר

דע והבן / לא כיבין / מלך חצור

אל תתהלל / כאיש הולל / ולא תפצר

באל קונך / הוא יחנך / ולא יצר

ערכה שירה / לו זמרה / עלי עשור

כל משאלך / ומעגלך / אליו תמסר

הוא יטיב לך / כל מפעלך / בלי מחסור

אנה תברח / אן תארח / בעיר מצור

בכל דירה / שם האורה / עמו ישר

הבט לזמן / אין נאמן / ולא ינצר

אהבתך, / לשחתך / ירדה יעצר

על כן רדף / טוב, ותהדף / רע, ותבצר

אולי תזכה / ולא תכה / כמלך צור

כי אל צורך / הוא מעז לך / כמשגב צור.
