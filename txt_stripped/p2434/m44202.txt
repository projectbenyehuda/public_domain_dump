


	
	התעוררי נפשי
	



שיר לעורר הנשמה לעבודה / טרם בא יום הפקודה




התעוררי נפשי התעוררי / ערכי שירה לאל וזמרי

הטיבי מעשיך התאזרי / חזקי באל מחסך התגברי

שורי מאין באת להעירי / ולאן את הולכת תסתתרי

במקום עפר רמה שם תגורי / חשך וצלמות תסגרי

מה תעני ליום דין, מה תאמרי / לפני נורא איום מה תפטרי

עת יפקדך צורך אן תחתרי / להעלות ארכה לך וצרי

אויה לך ואבוי מה תפצרי / אז תבושי מאד ותחפירי

הלא יענו לך איך תקשרי / עלי צורך ובו תשקרי

תמיד טובותיו לך לא תחסרי / איך תשכחים ממך, תתנכרי

אזי תזעקי מר עת תצרי / איכה חשך משחור תארי

התמרמרי נפשי התמרמרי / על עצם חטאתך השתערי

עד מתי יחידה תשתכרי / את יינך מעליך הסירי

פקדי נא את בת מי תתאורי / עם גוש סחי מאוס איך תתחרי

איך יעלת החן תמסרי / ביד איש בליעל איש מצרי

ברה כחמה איך תתקדרי / יפה כלבנה תשתחרי

אנה תפני לעזר מצרי / אלה בני שעיר יושבי החורי

שימי זאת בלבך ותזכרי / תמיד עדי שובך תוך מצורי

לזאת שובי אליו ומהרי / חיש קומי מעפר, התנערי

כבסי את בגדיך וטהרי / את סותך מכל כתם צהרי

התלבשי בגדי עז וחגרי / במקום מחשכים קומי אורי

קומי רני בליל ראש אשמורי / לאל חיש התגברי כארי

נדדי שנתך והאירי / לילה וחשך שימי למאורי

שקדי על דלתותיו אל תעצרי / מזוזות פתחיו תמיד שמרי

לעשות רצון צורך בקרי / תמיד אליו רוצי ודהרי

את רע מעלליך פזרי / התרחצי התזכי הברי

הקימי פרצותיך וגדרי / פסגי ארמנותיך הדרי

שועי לאל צורך ודברי / כי הוא מעז לך משגב מבצרי

משלי על אויביך השתרי / אלה משפחות בני מררי

הבדלי מחברתם וסורי / אל תלכי דרך אתם, הנזרי

נתצי נתיבותיהם עדרי / את כסאותם לארץ מגרי

כל ראש בשמים לקטי ארי / מנחה לפניו ערכי סדרי

טרם פרי מעשך תקצרי / אזי מות מהחיים תבחרי

אולי תזכי ליום עת תנהרי / אליו ובגן עדנו תתגוררי

אז יענך צורך התחברי / לי, את מראש אמנה תשורי.
