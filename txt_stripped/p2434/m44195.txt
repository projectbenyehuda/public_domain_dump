


	
	עיר כלילת
	



סימן: ע׳יר ציון נעמת האשיגה עוזך. לחן: ידידי השכחת.






עיר כלילת יפי / משוש כל הארץ / לך אותה נפשי / מפאתי מזרח

צעוק אצעק מר ו/אהמה כיונה / בראות כי זיו הודך / והדרך ארח

יחרד לבבי / יתר ממקומו / על כי מלכך אין בך / ממך כצבי ברח

ואיך אחריש את/אפק בראותי ד/רכיך אבלות / שבת עובר ארח

נטושה רטושה / ביד זרים כבושה / את כל עדיך הציל / וראש נזרך קרח

נחלת אל עליון / וגיא החזיון / איך נהפכת לזרים / לזאת במר אצרח

עיני נגרה, לא / תדמה בראותי כי / מטה עזך נשבר / ומטה זר יפרח

מי יתן והיה / אחונן עפרך / ואת אבניך / כמר דרור ארח

תאבתי לגוע / בעפרך לרגע / כי גוי שם יציץ / ציץ ויוצא פרח

המון בחיריך / הם שוכני עפרך / כל אשר גר אתך / וכל האזרח

אשרי יבחר יקרב / ישכן חצריך / כי שוכן בך נשוי / עוונו אל ימרח

שם ימצא מנוח / ושם ירגיע / כמו בשמים ראש / ואהלות ירח

יחיד רם ונשא / בך בחר ונסה / להקריב יחידו / אברהם בן תרח

גם מכון לזבולו / בחר לנחלה לו / וכבודו שם נגלה / ואורו שם זרח

התנחמי ציון / יושבת בציון / כי מלכך יבא לך / וחסדו לך יסרח

עוד יאצר בך ששון / ישראל בך יחסון / לא ימר עוד ריחו / לא יבאש ויסרח

ולשאטים אותך / יטח את ימינו / תבלעמו ארץ / כמו עדת קרח

זריקת דם זבחך / וריח ניחוחך / יעלו לרצון / בל יהיו לטרח

כל מדבר וציה / ישית גן רויה / ותגל ערבה / ופרוח תפרח

עוד יבנך ונבנית / קריה נאמנת / אור שמש לא יאיר / לך ולא אור ירח

לא יבוא עוד שמשך / כי אל צור קדושך / יאמר קומי אורי / וכבודו לך יזרח.
