


	
	איך ימצא
	



שיר המדבר על פרידת הנשמה / לאדם אשר להבל דמה.

סימן: אהה אבוי אהה לי. לחן: אשמחה כי אפתחה.






איך ימצא חן / בעין בוחן / לב וטוחות

איש כל חושיו / במשאות שוא / ומדוחות

וסוף שברו / בתוך קברו / אין מנוחות

רדוף רמה / עדי חרמה / לאכל לחמו.





הון רב יכסף / יקר יאסף / ביד רמה

און יגביר / ובלב יחביר / תוך ומרמה

יטמן רשת / ידרך קשת / יחפר גמה

לכל איש דל / ולא יחדל / לשפך דמו.





הוא לא ידע / קרנו נגדע / באף חמה

וזיו הודו / גם כבודו / שם לכלמה

ולא ימצא / לו כל עצה / גם מזמה

עת ירתק / אז ינתק / כל זממו.





איך הוא הולל / והשתולל / בלי חכמה

אשר חלל / והתגולל / במי אשמה

ואחריתו / לשחתו / ביד דומה

ברב קצף / עת יאסף / אלי עמו.





בעת יגדל / חליו ידל / על משכבו

עד אין רופא / ואין מרפא / למכאובו

הלא יזכר / זמן יעכר / את מצבו

כאיש שונא / לו מתאנה / באף זעמו.





ויתגבר / בלי אבר / הוא לפרח

אולי יצר / בעיר מצור / שם לברח

אך לא יפליט / כי אין שליט / הוא ברוח

לכלא אותו / משחתו / כי בא יומו.





יפתח עיניו / נגד פניו / איש משחיתו

וביד הדה / חרב חדה / לעמתו

לבו יתר / ואין נסתר / מחמתו

עודו נגדו / להשמידו / עדי תמו.





איך לא ישוב / חמת עכשוב / מרעתו

טרם בא יום / אין לו פדיום / למחלתו

יום בא ימש / מאור שמש / לחשכתו

וירח / אין זורח / עת הדמימו.





הבל דמה / איש אדמה / רב אנחות

מלא עולות / תוך מגלות / מפתחות

ויזכר / לפני זוכר / כל נשכחות

ויהי נוכח / עת יתוכח / ישר עמו.





הה מה יען / על מי נשען / איש גוש עפר

כי בז לדבר / אל, ועבר / וברית הפר

הלא ידאג / ומר ישאג / בקול שופר

איך הוא נגרש / ולא דרש / טוב לעצמו.





לזאת יעיר / נפשו, יאיר / את נתיבו

יסור מרע / וגם יקרע / סגור לבו

ישאל מאל / כדל שואל / להשיבו

יעטה בשת / לב אל ישת / עת הכלימו.





יהי עופף / להסתופף / בית אל איום

לערך תמיד / שיר גם יצמיד / לילה עד יום

בקול תורה / אז מהרה / ימצא פדיום

טוב ונעים / יכתב עם / חושבי שמו.
