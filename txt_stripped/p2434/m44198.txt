


	
	שברתי לתשואות
	



סימן: ששון ישראל.






שברתי לתשואות / המוני נפלאות / אלהים צבאות / גואלי, מעוז בצרה.

שוב ישוב להראות / עוד עז ונוראות / יעשה לטובה אות / להפליא, בעז וגבורה.

וכימי צאת מבור / חגר עז כגבור / קרע הים לעבר / קהלי, לאום נבחרה.

נשאו עין לגמר / הלל שיר ומזמור / ויאמרו לאמר / זה אלי, לו אזמרה.

ירה בחרונו / תוך הים ושאונו / פרעה והמונו / כי צר לי, באף ועברה.

שבר גאון זדים / הפושעים ומורדים / היו עולים ויורדים / כעלי, וכמץ בסערה.

רבי מי זדונים / עברו בחרונים / ירדו כאבנים / עד לבלי, בנפש מרה.

אזי שתו מי ראש / בהמון ים לגרש / כל לבב וכל ראש / לחלי, פצע וחבורה.

לזאת זעקו מר / זה אל זה ואמר / הן כל גוים כמר / מדלי, וכרוח עברה.

כל שרפי מעונה / הם בקשו ראשונה / לשאת שיר ורנה / לצור לי, מגן וסתרה.

אכן נאם האל / צור פודה וגואל / אז ישיר ישראל / עם חבלי, לי את השירה.

אולי צורי יעתר / לי ולא יסתר / ינתק ויתר / כבלי, ויזרח אורה.

יקבץ שה פזורה / יל ידי איש משרה / אל נות תפארה / גורלי, עיר הבצורה.
