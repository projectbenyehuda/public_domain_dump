


	
	[ידידותך בתוך לבי]
	



פראנעה אבי אעזה אללה בהדה אלאביאת

והשיב לו אבי, יוקירהו האל, בבתים האלה






U – – – U – – – U //– – U – – – U – – – U – –



ידידותך בתוך לבי יצוקה // ואין ביני ובינך דין חלקה,

ואהבה לא כאהבתך, כתבה // אנוש על קיר לבבי – חיש מחוקה

ונפשי מאשר תרחיק רחוקה, // ונפשי באשר תדבק דבקה.

ויש אוהב אשר כפו ספוקה // בצרתך, וצר – שנו חרוקה,

ויש מראה לך טובה ביום טוב – // ויום צוקה יהי חבר לצוקה,

אשר אם תשאלו רקיק לאכלה // אזי יתן בפניך רקיקה;

ותאמר לו להנחותך בעמק – // וישליכך בתוך שוחה עמקה;

ואם תדליק אבוקה לך – יזורר // עלי אורת אבוקתך אבקה.

ואתה תאחז לב טוב ברעך // ועל שונאך ידך חזקה,

ואם תשך – נשיכתך נשיכה, // ואם תשק – נשיקתך נשיקה.

ושמעתי דברך באנשים // אשר אין לך ולי בהם חשיקה,

נחשים הם בעיניך ועינם – // והם אצלי כתולעת ירקה;

חמדוני ויעיקו, להודי // והוני, כעגלה המעיקה,

ומה־יוכל עשה תיש לליש // ומה כח בכח ואנקה?

ישלח אל עלי אישים זכרתם // אשר שלח עלי נסי עזקה,

ויכלימם בתוך ארץ, וירדו // מכסים בכלימה אל עלוקה!

אני אעיד בצרתך לצירי // וכי תחיל בעת אנאק נאקה,

ואצדיק, על אשר בא, אל אמונה, // כמוסרך, בעתותי זעקה,

והרעה אשר הביא ישו על // מגלת פעלי הטוב חקוקה!

יגוני על אשר מתו, ומרה, // לבד מותם, בפי כדבש מתוקה,

ואשמח כי אשר לקח – בכבוד // נתתיהו, ולא היה לפוקה;

חשבתיו על שני גבי כמשא – // וכל טעון מנוחתו פריקה.

וסרה אחרי זאת כל תביעה // והיתה בה למשטיני שתיקה,

וכל בית יפנה משאור, לא // תהי אחר בדיקתו בדיקה;

וכל גויה, בהתגלח, שעירה // תהי, אחר אשר היתה חלקה,

ויש פרי בבן כרם, ויוסף // פרי, אחר זמירה ועזקה,

ואור פני אזי ירבה, כהרבות // מאור בית, בהקצץ, אבוקה.

כבר הפיג דברך את יגוני, // ואל יחשב לך את זאת צדקה,

וייטיב באדמה אחריתך // ולא ישמיעך קול צעקה.





קרא, אחי, תשובה לך שלוחה // בחפזון, כמו עלה דפוקה,

ומי יכין כשירתך, אשר מר // ושמן טוב עלי ראשים מריקה?

וכל דל מזבח פר וכבש // יהי נעתר בתורי המליקה!
