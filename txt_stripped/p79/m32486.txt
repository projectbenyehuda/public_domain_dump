


	
	ארך גלותי
	



כתובת בראש השיר: ואמר בשבח החכם אבי יצחק אברהם בן עטא (= בן נתן) יאריך ה' ימיו.


ארך גלותי / בעניי ודלותי,



והנני, אילותי, / מגלה־לך ריב[י].



כחי עזבני / וכמת חשבני

וחטאי סבבני / ודויי ומכאובי.



[ו]תמיד בשועתי / לך צור ישועתי,



ואשחה בדמעתי / לערסי ומשכבי.



[ו]מרב עוונותי / ועצם זדונותי



מרבים יגונותי / בנפשי ובלבבי.

ובשתי בתוכחתי / וכמת נשכחתי

בעזבי למשפחתי / ואחי ובית אבי.



[ו]לולי אשר נועד / לעמו בעת־צעד



וסיני מאד רעד / בעמדי במצבי,



[ו]הקים למושיעים / בכל־דור לנושעים



נדיבים וגם שועים / מפיגי למעצבי.



[ו]מהם נשיא דורו / נסיכו וכבירו



מאירו ומנהירו / ופודו כמשבי,

בחירו בנו נתן / נדיב לב כמו איתן

מהדר כמו חתן / בחדרי ומושבי.



[ו]פניו פני חמה / ולבו לבב חכמה



וגבור במלחמה / כאריה וגם לביא.

מישר מסלתו / לנגד קהלתו,

ומנהיג בחמלתו / קטני וגם שבי.

והוא שר לכל־שרי, / מחזק ידי מורי,

וחרבו עלי צרי / וחומה להסביבי,

והוא אור שני עיני, / והוא אב לכל־בני

ומשען לנשעני / ותקרי ומעזבי.

בנותי בהוד עלה / בקשוט עדי כלה,

וכיסי הלא מלא / בכספי ובזהבי,

וצוה להלבישי / תכלת וגם משי

וצניף עלי ראשי / וסוסים למרכבי.

שמו טוב כאזרחי / וריחו כמרקחי

מהלך במזרחי / צפוני ומערבי.

לבנים וגם־בנות / יתומים ואלמנות

ימינו במתנות / כמטר להרביבי.

והוא אור להלוכי / וסהר במחשכי

וגפן לסכוכי / ופריו להתניבי.

לדת אל בפה גמר / וחקיו מאד שמר,

מעלה כמו תמר / ועניו כאזובי.

ומשתיק למדמימי / ומפעים למפעימי

ונלחם בכל־קמי / לבדו ובקרבי.

אדוני אבא אסחק / קחה שיר בך הוחק

קרובים וממרחק / ירוצון……..בי

הדרתיו במו טורים / בנפך וספירים

ומשתיק לכל……/ …………[בי]



[ו]ירום נשיא הדור / יחי עוד לדור [ו]דור



יהיה עתיד לגדור / …לשגוב …[בי].
