


	
	לשנים דשנים
	



ברכה לידיד






לשנים דשנים / וזמנים לא נמנים / וזקונים נכונים / ברננים תגיעו



בבנים נבונים / המונים אמונים / רוזנים וסגנים / פנינים יביעו



וכמה וכמה / בחכמה ומזמה / כחמה וכימה / בנעימה תופיעו



ובעצמה בקומה / שלמה ותמימה / כחומה רוממה / מאימה תרגיעו



כנהרים נגרים / ספירים ומהירים / נהדרים נבחרים / חדרים תציעו



בדברים יקרים / אוצרים שמורים / מבצרים ושערים / נגזרים תושיעו



בגבורה יתרה / בתורה וספרא / וגמרא ברורה / אורה תשפיעו



נא אל נא שמע נא / תחנה למנה / נתונה נתונה / ורננה יריעו.
