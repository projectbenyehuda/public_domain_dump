


	
	[בשפתי רננות ושירי נגינות]
	




Biniamin Coen Vitale


]נולד באלסנדריה בשנת 1651 ומת ברג֗ו בשנת 1739]






͝ – – ͝ – –



בשפתי רננות / ושירי נגינות / נהלל בקול עז / לשוכן שחקים

נעימות ימינו / לעושי רצונו / ברכה ושלום / בכל־עת מריקים

ישרים בעודם / במשכן כבודם / בליל זה שמחים / במחול משחקים

מאירים בנרות / לכל־החצרות / בתוך עיר קדושה / במראה ברקים

יעידון יגידון / בפיהם יהודון / לאל רם ונשא / אשר בו דבקים

נדבות גשמיו / ויזלו בשמיו / ולשאב בששון / למימות מתוקים

כמו־כן עבדיו / וזרע ידידיו / באימה ביראה / הלא מחזיקים

הליכות אבותם / דברים בעתם / בבית־אל בלילות / וממעמקים

נכונים לספר / ונכתב בספר / כאיל בשחר / ועל־האפיקים

יהי שם יי / ובקהל אמוני / מברך לעולם / בכל־הפרקים

צבאות יחונן / ועמו יגונן / ויחיש מבשר / לקרב רחוקים

ויביא לציון / גאלה ופדיון / וישלח משיחו / להציל עשוקים

ויבנה דבירו / ומושב יקרו / בשלם לסכו / מהרה להקים

תהי־נא רוחה / ואורה ושמחה / להלל בקול עז / לשוכן שחקים.
