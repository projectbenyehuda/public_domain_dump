


	
	אמונה יצרה יסוד היסודות
	




אמונה יצרה / יסוד היסודות / תרנן לפניו / בשירי ידידות

בטרם שחקים / וארץ מתוחים / ועד לא ארקים / ותבל יסודות

גנזה לפניו / אלפים שנים / ודורות עשרים / וששה סדורות

דבש מלשונה / ליודעי פשטים / וכבוד וחיים / למבין בסודות

הדורים תישר / במיטב אמרים / ותשיב לנפש / במשל וחידות

ותחכים לפתי / וסכל ונמהר / ואלם ירנן / וחגר בקידות

זמנים משנים / והיא לא תשנה / לנצח נצחים / במצוות תמידות

חקוקה בחקים / ומצוות ישרים / ועדות בכל עת / בחכמה מעידות

טהורה וברה / מאירה לעין כל / מסלות נתיבות / בכל לב אפודות

ימין עז ועזוז / נתנה למשה / בסיני מסרה / ליד איש חמודות

כנוח נבואה / לבלתו ברוח / אלהים צבאות / בלבשם חרדות

לבד בן לעמרם / מדבר בעמדו / בכחו ושיחו / כמו אש פלדות

מנוחה ושלום / וחיים ארוכים / לרב לנביאים / צדקות רבודות

נתנה בלב טוב / וצוה לשמרה / הלא בעבורה / נפשות פקודות

שמאלה יעשר / לריקים וכבוד / וארך לימים / בימין אגודות

ערבה בכל פה / ומרפא חליים / לדבר בסודה / ידי עט כבדות

פתח לי אלהים / לבבות סתומות / ולא יהיו עוד / בהבל שדודות

צדקה עשה נא / למען שמך / והתר לרגלי / בתוך פה לכודות

קדשה ומלכות / תשובב לציון / ונזכה לשרת / בארבע עבודות

ראה זכרך יה / אמת תוך לבבי / ואם רב עווני / ורבו בגידות

שלח ממרומים / וקבל לשירי / כחטאת כאשם / כעולות ותודות

תקבל לעבדך / ואכרע ואודה / ותמיד אזמר / לך טוב להודות.
