


	
	אל אל אקרא מראש צורים
	






אל אל אקרא / מראש צורים / מקים לדל / משפיל ומרים



בינותי עם / שכלי כי את / מאיר תבל / גם לדרים



גל מעלי / חרפה ובוז / אהגה תמיד / עוד בספרים



דברי הרב / יוסף אלבו / נרשם בעל / העיקרים



הוא לי חמדה / גם לסגלה / תוך ארבעת / המאמרים



וישת לי / חכמה רמה / היא כמים / האדירים



זכני אל / לידע סודה / הפשט מני / לב נמהרים



חטאתי אל / אל תזכר עוד / הן בזקונים / הן בנעורים



טרם תבוא / על עבדך / במשפטים / הנעכרים



יען אוכל / לקנות עולם / אוסיף לקח / טוב בסגורים



כונתי / לרדף צדק / אדרך מבחר / המישרים



לכן יצרי / הוא שעכב / הטביל עיסתי / בשאורים



מה אוחיל עוד / מן היצר / תבכה נפשי / במסתרים



נין האמה / על כל אלה / הוא השקני / כוס עיקרים



סלה מני / כל אבירי / הושיבני / במאסרים



עמד נגדי / הדאיבני / ואני מכה / בסנורים



פרץ בנין / גדר אחוה / הפך לבו / לאכזרים



צמתו בבור / חיי שרים / ופניהם עוד / לא נהדרים



קנא קנא / ונקם נקם / קדש שמך / מול הצרים



רחם עמך / וגאל אותם / מיד אויבים / מן ההגרים



שמעה ופנה / אל הערער / וזכר אבות / הם היקרים



תכלית מלי / עם רב שיחי / אשא עיני / אל ההרים.
