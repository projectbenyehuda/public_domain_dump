


	
	בני עליון בני חכמה בני איש
	








בני עליון בני חכמה בני איש / סגלת כל מתי משרה וחברה

דעו האל בראנו לעבדו / ומצותו תהי בלב שמורה

ומשפטיו ותורותיו וחקיו / להתהלך בדרך הישרה

בעוד לנו אמת חיי בשרים / ולב מרפא והנפש ברורה

והם לנו כמגדל עז בשם אל / והם לנו אמת חומה בצורה

בטרם נרדה פתאם שאולה / ונשכן בית אבנים לקבורה

לכל אדם הלא יבוא חשבון / ועיר סיחון לקלה וחמורה

ואם יעלה בחשבונו אבנים / כאבני פז וכל אבן יקרה

יבילוהו לגן עדן מהרה / לחפתו ועל ראשו עטרה

וצדקו יהלך תמיד לפניו / כבוד האל יחופף לו לעזרה

ואם ימצא בחשבונו אשמים / וחטאות וכל מדה חסרה

יבילוהו לפח מוקש כמו אש / תאכלהו באף זעם ועברה

כמו כסף בסיג לצרף לזקק / ויבחן כזהב במדורה

ובדין יעמד אחלי ואולי / במליץ טוב להליץ לו לכפרה

ואם לא ימצאה ישכן בדומה / והתגלח בתער השכירה

והוא ימות בתבערה ומסה / ודראון לכל חמר וצורה

בני אל חי וכל מאזין לקולי / אמת דרשו תהי לכם לסתרה

ימותינו גבולם עד לשבעים / ופלא יעלו לשני גבורה

ואם כן הוא נסקל המסלה / נתיב ישר והיא דרך קצרה

וההגיון בעתותיו לשמרה / ביום וליל להבין כל סברא

והיראה תהא אוצר לכלם / מעלה היא לכל חסיד מסורה

ועקב לענוה היא נתונה / נעימות הן בכל מדה גמורה

ומה יקר אנוש הולך לתמו / בהוד יצא ומאפל לאורה

ואם יעשו פרי נטעי דברי / שבח אשיר ואתבשר בשורה

בגיל אשמח ואודה ואהלל / הכי נמצא במהללי תשורה

לאל עליון אלהי הצבאות / שהוא גדול והוא גבור ונורא

והוא חומל יכפר מעללי / אקוה לחזות פניו בטהרה.
