


	
	קחו נא מוסרים
	








קחו נא מוסרים / סגלות העברים / והקשיבו שירים / ושפטו מישרים

בשיחי ההגרי / לבין אחיו יפריא / ובמרום ימריא / המון הקדרים

אשר שח הנכאה / והוא כעוף ידאה / במעשהו יראה / לנגד המורים

והציל הון מהם / ועמד לפניהם / וערך הוא להם / תאנים עזרורים

והיו לתרופות / לרגלי נגפות / למרחוק צופות / לתנים ושעירים

ורגליהם שונות / ונסתרות עונות / לנשואי בנות / לנבזים ויקרים

בניבו יתפאר / בחוצותיו נאר / ומחשיך לא יאר / לנין האשורים

בריבו התגלע / וכמעט הוא נבלע / וראשו על סלע / כחרס נשברים

ואם ידע נכסף / לתפוחים אסף / במשכיות כסף / בזהב נחברים

ועל כמה פנים / חכמים ונבונים / בתפארתם שונים / והם האבירים

לשונות מגידות / פשטים עם סודות / משלים עם חידות / כמים נגרים

למולם תלפיות / מתוקות לפיות / לרחוק צופיות / בימים ונהרים

ומהם אגדות / ליודעם נכבדות / פשוטות ומעידות / ובהם נסתרים

ודברי קבלה / והם מעלה מעלה / ועולים במסלה / לנפשות היצורים

ובהם היכלות / בחכמה נבדלות / למלא משאלות / לבבות הברים

לטוב בבלי קצבה / והוא עולם הבא / בנחת ובשובה / למעמד השרים

וזהו התכלית / בראש גלות עלית / ואפס התכלית / חסידים נשכרים.
