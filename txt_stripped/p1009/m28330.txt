


	
	אל מסתתר למאד נעלם
	




אקרוסטיקון: א״ב אני זכריה בן שעדיה



אל מסתתר / למאד נעלם / יודוך אל / עמים כלם

בראת ראש / שכל נבדל / מן העשר / ישכיל שכלם

גלגל סובב / לנקדתו / בו כוכבים / מאיר הלם

דרכו סובב / על היסודות / תחבולותיו / יצא פעלם

הן יתהוו / המתהוים / צורות יוצאים / מהיכלם

ותבחר לך / מכל אלה / אדם מבין / משכיל משלם

זהב מופז / מן הבריות / מבדל מהם / בריא אולם

חטאו בגן / הוא עם אשתו / עץ הדעת / סבת אכלם

טבעו ביון / ים סכלותם / עליון צופה / כל מעללם

יחיד ענה / צא מן הגן / רב לך שבת / עוד בגבולם

כביר כח / יסד ארץ / על מתכנתה / עד בוא מעלם

לנשכחים / מני רגל / הנבטחים / על רב חילם

מחה אותם / מן הארץ / הם נשקעו / תוך מבולם

נח צדיק / תוך תיבתו / חי במצולות / נשאו קולם

סרחו דורות / הפלגה / עת נועצו / על מגדלם

עין תלעג / מרום להם / גמר אמר / על בלבולם

פזר אותם / אל כל פאות / ללשונותם / כי סר צלם

צמח איתן / האזרחי / האיר תבל / על ראש תלם

קדוש נעקד / על מזבח / צעקו חוצה / הן אראלם

רועה הצאן / עשרים שנה / בין אהלים / חזה סלם

שבטי השם / עת נטעו / גם שרשו / האל עלם

תמיד היו / עובדי אל חי / עד החלומות / יוסף גלם

אחרי נמכר / אל מצרים / אל יד פרעה / צופה השלם

נתן מלך / על כל ארצו / בשנות שובע / עת החלם

זכר החלום / גם פתרונו / שלם להם / לא כגמולם

כלו ימיו / גם דורותיו / וישימו / חמר סבלם

ראה אל חי / זרע יעקב / ירדו עד כי / עפר משלם

יחיד בחר / משה עבדו / לגאל עמו / מיד נכלם

הביא עליו / עשר מכות / עד שב פרעה / בוש גם נאלם

בקע ימים / וישירו / הן לקטנם / הן לגדולם

נתן תורה / בשני לוחות / תרי״ג מצוות / דת הנחילם

שמעו כלם / על הר סיני / גם שמים / נתנו טלם

ענו כלם / נעש נשמע / מן התועים / אל הבדילם

דרך בים / הוא הנותן / במדברות / מן האכילם

יסד העז / מפי עוללים / מלא רחמים / בורא עולם

השם נפשי / מן החיים / אל מסתתר / למאד נעלם.
