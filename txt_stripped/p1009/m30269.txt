


	
	תמה אחי עלי ידעי משלים
	




תמה אחי עלי ידעי משלים / מתוקים מדבש ונפת צופים

ומעשי הם אני אומר למלך / בדעותיו אמת טובים ויפים

בצאתו עם צבאיו וחיליו / וכל שריו ופחותיו צרופים

הכי מצא אנוש זקן וקשיש / משחר לעבד ארץ ערפים

ואז שאל בערמתו ואיכה / כמתעצל תהי מוציא אלפים

הלא תקדים ותשכים לעבודה / וכוכבים לפניך רצופים

ואז ענה וחי ראשך אדוני / כבר קמתי וקדמתי נשפים

אבל לקחו אחרים את נשפי / והוספתי עליהם עוד כספים

והמלך בשמעו נם לישיש / דעה הנה מנתך עם חנפים

והלך הוא לדרכו מחנים / ואז חנה כאיש עושה כשפים

בעת שאל סגניו הם ושריו / ידעתם שאלותי התכופים

לעובד האדמה שב וישיש / והוא רזה ומשמניו עיפים

ולא יכלו ענות אותו כאלו / בשכליהם ודעותם טרופים

והוא גזר הלא אתם נגידים / מנתי אוכלים מאות אלפים

ומדוע כבשתם את פניכם / ואם לא תדעו אתם הדופים

כשמעם חגרו עזם בשלחם / אלי ישיש והרצים דחופים

בידם מתנות כסף וזהב / ומגדנות ואפוד עם תרפים

בהגיעם אלי ישיש בביתו / והשתחוו וכרעו כפופים

למען יפתרה מלין למלך / עצרם מלפניו ועטופים

והזקן בשורו מתנותם / ופניו צהבו כפני שרפים

ואז השיב שאלני אדוני / ימי חרפי הלא בני צפופים

ואנה הם לכונן את מלאכתי / כמו עשו לשרשים ענפים

עניתיו כן אני האב לבנות / נשאום מבצרי העז אפופים

והמנות אשר גרם בשכלו / למלאת מחסורי הן אסופים

ולערב בשר אילים אבוסים / ולבקר והיא עוגת רצפים

ואז הפכו פניהם השלוחים / להגיד לאדוניהם שטופים

בהגידם בשיח דברותיו / ונחשבו כאלו הם אשפים

ולא ידעו היות מלכם כמערים / ורעיוניו למרחוק כצופים

להציל מהמון הונם לזקן / אסרו הזמן כלוא בכפים.
