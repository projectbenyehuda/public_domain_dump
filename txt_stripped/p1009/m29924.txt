


	
	ראו ספר חקרתיהו בחכמה
	








ראו ספר חקרתיהו בחכמה / ארגתיהו כמשי המזמה

ידי רעיון משכוהו בהשכל / ידי מחשב רקמתיהו ברקמה

ועין הלב ומורשיו יצקום / וסרעפיו בניב ישר חתומה

באזן שמעה כל מחקרי בין / ועין ראתה היכל פנימה

בסימני בני אדם בנויים / מיסדים ברב אונים ועצמה

באיש צדיק במדותיו וחסיד / ואיש ענו ואיש ערום בערמה

והנדיב והשוע וכילי / ואיש רשע שפל ברך וקומה

קצר רוח וארך אף ורגזן / ואיש נואף במעשהו כלמה

והעצל והרכיל והלץ / והנמהר ופתי כבהמה

והנאמן ובעל סוד ומהיר / חכם לבב בדעתו התמימה

וקרח והרצחן וגנב / ועין רע ונוקם הנקמה

ומתרושש ומתעשר ואין כל / והחכם בעיניו אין מאומה



[בגלגלתו באזניו גם ומצחו / ובזקנו ועינו הרשומה]



ואיש חטם והחרום וגבן / והחגר ברגלו העקומה

ושרוע והקלוט בגופו / ואיש קצר וארוך באדמה

והיועץ ואיש נבוב בלבו / והגזלן ואיש עקש במרמה

ועוד נוסף על אלה ברבים / בתוך ספרי הרימותי תרומה

לאפלאטון וגאלינוס ובוקראט / ומתפלסף בחכמתו שלמה

אספתימו ברעיוני להשכיל / למען יהיו בעדי לחומה

לקץ ימי נשאני זמני / לארץ כוש והדו הזעומה

לעיר פתרוס וכסלוחים ואין איש / מאסף לי לנפשי העגומה

והנה איש גויתו מלאה / בכל מומין וקומתו צנומה

ואז שלח לידו להביאה / לפס ידי הקימני תקומה

משכני לבית טוב ורחב / לפת לחמי ועשב לבהמה

ובן בקר זבחו לאריחות / ויין רקח להשיב הנשמה

וכל דבר בכיסו שם מצאתיו / ואז שבתי כרחל נאלמה

וחשבתי לחכמתי לסכלות / יגיעתי לריק תוריש זממה

וגמרתי בלבי על ספרי / להשליכם מצולות ים בחימה

אנוש כזה בסימניו מגנה / ואיך תהיה פעלתו תמימה

ויחשב אויל מחריש כחכם / ואולתו בשער נעלמה

לקץ ימים עשרה אז קראתיו / ואמרתי: תברך ממרומה

תברך מאנשים הקדומים / למען כי פעלתך נעימה

ועמדתי להעמיס על חמורי / כמתעסק והן נפשי כהומה

ואז אמר: ואנה תהלך חיש / וכל מחסורך כתוב בשומה

ופת לחמך ויינך עם בהמתך / כתובים הם ולא נפקד מאומה

עמד שלם פרע עתה לחובך / ולא תטבע בתוך ים התנומה

כשמעי אז משושי רב ונפשי / השיבותיו אני אוכל תרומה

אשלם לך ואוסיף החמישית / ואהבתי לך נפלאת עצומה

למען אין יכלת בידי לגמלך / עלי אחת ואיך אוכל וכמה

ועתה כי הקימות דברי / אני עבדך ונפשי לך לאמה

ושלמתי נקדים עם ברדים / ולא ידע ומחשבתי סתומה

לזד יהיר ולא רשע ונבזה / ואיש מדון ובעל חטא ואשמה

והלכתי לתנות לחכמים / ראו ספר חקרתיהו לחכמה.
