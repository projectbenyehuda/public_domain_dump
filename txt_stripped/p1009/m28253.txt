


	
	אל אל אדיר
	








אל אל אדיר במרומי ערץ / אל אלהים יי דבר ויקרא ארץ

אזכירה עבודת דומה ומשא / אליך יי נפשי אשא

בימי בני קדר גברו וגאו / בחורים טחון נשאו

גדעו קומתי בחמר ובלבנים / גוי עז פנים

דחפוני באמת בצעם / דתיהם שונות מכל עם

הדפוני בבית הכלא / הלמתים תעשה פלא

וגם שמו רגלי בסד / ולך יי חסד

זממו על בניך להכחידם / זה ספר תולדות אדם

חשבו להעבידם במרץ / חסד יי מלאה הארץ

טפלו שקר והתעיבו עלילה / טבעתי ביון מצולה

יחד קוו לאבדני / יהי נא חסדך לנחמני

כל ימי ירא וזוחל / כלו עיני מיחל

לעד העבידוני בפרך / לא אליכם כל עוברי דרך

מעלי המשילני לאבק דק / מה נדבר ומה נצטדק

נפלתי ביד מעני / נפלה נא ביד יי

סבבוני עניי וצירי / סמר מפחדך בשרי

עברתי על מצותך קוני / עון עקבי יסבני

פשעתי והנה נפלו פני / פלגי מים ירדו עיני

צרפני עוני בכור עוני / צר ומצוק מצאוני

קוממיות שברתי וארגיעה / קדמתי בנשף ואשועה

רחום הצילני ואמלטה / ראיתי בוגדים ואתקוטטה

שוקד ומוציא עולמו ליש מאין / שמרני כאישון בת עין

תוכן נשמות בגלגלי מעלה / תבענה שפתי לך תהלה

אדון נקראת צור עולמים / אל מלך יושב על כסא רחמים

באתי לחלות פניך על רב עוני / בצר לי אקרא יי

נכפפתי ואתה זוקף כפופים / נוצר חסד לאלפים

רחם יחידתי בצאתה ובבואיה / רוקע הארץ וצאצאיה

בקש עבדך צועק מצרותיו / בונה בשמים מעלותיו

נין האמה כלה עם מתי השקר / נפשי ליי משומרים לבקר

חשבתי דרכי בלבבי ורחשי / חלקי יי אמרה נפשי

לובש צדקות שיחי אל תשי / למה יי תזנח נפשי

קומי ושבתי שמתיך לנגד עיני / קראתי שמך יי.
