


	
	אלהי פתח פי
	






אלהי פתח פי / וגם האיר נשפי / והשיב יום חרפי / נגינתי ערבה

אקדם בתהלות / לעושה לגדולות / ועלת כל עלות / וסבת כל סבה

בשורי נפלאות / עתידות כי באות / לעולם היו אות / נבואה נכתבה

במשא האמה / היותה לשממה / באשמה נכתמה / וטירתה חרבה

גאונה הוא נשבר / בשבר על שבר / והובל לקבר / שאול לו הרחיבה

גמולה שלם לה / כרע מפעלה / ונפל מזלה / וגם נרה כבה

דברה הנחוץ / כבר נרמס בחוץ / ואישה השחוץ / ביד דומה נחבא

דעו כתות יונים / בלעג שאננים / ובוז לגאיונים / בארץ לו צבא

המון קול הרועץ / כבר יבש כעץ / ושבת הנוגש / ושבתה מדהבה

הלא פתחו נסגר / ובשקים נחגר / ועדר בן הגר / ביד מות נשבה

ושבתו האיבות / וגם דברי ריבות / ונפרעו חובות / ואין עוד דין הבה

ומצחו הרחב / כבר נס בטחב / ועל פניו נסחב / בתוך אש להבה

זמן נתן עדיו / ושלם את נשיו / והתבונן איו / בתקוה נכזבה

זכר דברי נבל / ולשאולה הובל / וקללת הר עיבל / בגופו נצבה

חדי יונה תמה / בשבר האמה / וקומי עד כמה / ברוחך נעצבה

חבי כמעט רגע / לזרע המשגע / ולו עת ופגע / ומכתו רבה

טמא לב עם שפה / הלא שבע חרפה / מגלתו צפה / ותוכה תועבה

טפולה בשקר / לרב עד אין חקר / והיא כענן בקר / לתהו נגנבה

ילודי ישמעאל / בספרם מתגעל / ודלתם תנעל / ושמשם ערבה

ידיהם הם רפו / ובפניהם חפו / בשרשם נקטפו / כמוציאי דבה

כפירם הנזכר / הלא הוא שב נעכר / והיה כזכר / ונחשב כנקבה

כבודו בזיון / ותחת אפריון / למחלון ולכליון / ונפשו שובבה

לקוחים ההגרים / להשמיד ולהחרים / כמהפכת זרים / לממשלתם קצבה

לדכא בנגעים / ומכות ופצעים / המון זדים רעים / לנער עם שיבה

מנוחתם כמה / בתתק״ף נשלמה / שנות מלכי אמה / כרגע נחשבה

מנוחת החגר / בתפת על מסגר / וכסאו מתמגר / ורגלו נעקבה

נשיאים קול שאון / לכל חי דראון / כלהב צמאון / ונפשו דאבה

נשיא ראש כל קדר / בגיהנם קדר / בפניו אין הדר / ועינו הרקיבה

סגלה הוא לכם / וכבו אור אשכם / ומלך הוא לכם / בנו לכם מצבה

סביבתו פחת / באש מתלקחת / ונפשו בשחת / בכל יום היא שבה

עדי אש על גופו / ודומה יניפו / בקלע ודחפו / ונבהל: מי זה בא

עדת חנף גלמוד / כמפעלם למוד / בליעל למוד / בשנאה גם איבה

פני אל חי תראו / בני יעקב וגאו / והביטו וראו / תחלת הטובה

פנו אל אל צורכם / ומיכאל שרכם / והנה ישרכם / בנפש חצבה

צדקותיו רבות / ומדותיו טובות / ובזכות האבות / גאלתכם קרבה

צדקותי קלו / מזמותי יעלו / ועיני לך דלו / ועשקה לי ערבה

קחה נא מנחתי / ועל רב אשמתי / קבועה חטאתי / שעירה או כשבה

קדושים אחרי מות / תפלות נעלמות / לתוכן עולמות / ורוכב במסבה

רצה אל חי עמך / ושכלל אולמך / ועם עני ומך / בציון היטיבה

רצועי ספך / וסובלי קצפך / למען צופך / בשמך נתנבא

שעה את שועתם / מחה את דמעתם / ותקרב רנתם / ותרצה בתשובה

שנתי בה אורה / בהשל״ג ליצירה / תחלה לבשורה / לאמה נאהבה

תקבל מלולי / ותשמע פלולי / ותאזין הלולי / ברצון ובחבה

תמהר במשיחי / ויערב לך שיחי / כריח ניחוחי / כעולה ונדבה

חקר ארך מענית / לחותם התכנית / ואל תוסיף שנית / תדינו תוך לבה

זכר נא כי כרע / וחטאתו פרע / בחסדך תקרע / שטרו כי נגבה

קדורנית רזה / במשמניו אחזה / בעודי חי בזה / ולעולם הבא

ואם לך עז חביון / כבודך תוך ציון / ואתה הוא עליון / עלי כל מרכבה.
