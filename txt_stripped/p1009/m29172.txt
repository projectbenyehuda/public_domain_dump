


	
	לקראת פני שבת אשיש ואגילה
	








לקראת פני שבת / אשיש ואגילה

אתעלסה כמשוש / חתן עלי כלה





זיו הוד מאור פני / ששת ימי החול

יועם ורעיוני / נפשי בטוב תבחל

חבל המון מוני / מתקוממים לזחל

אשיב למול עיני / דת האמת לנחל

אתבוננה סבת / פלאי אדון מעלה

הסוד בלב אכמס / לשם ולתהלה



כלו יגונותי / יום בוא מנוחתי

עצם פליאותי / משנה ארוחתי

אפתח צרורותי / לכבוד סעודתי

איטיב זמירותי / שיר על נגינתי

כוסות כעין לבת / ידמו ולסגלה

שלחן יפי לפרש / אלון בתוך כילה



רכב אלהים חי / רבבות ולאלפים

שומעים לקול שיחי / הם עד כרוב עפים

תוך מקהלות אחי / אהגה והם צופים

סודות לבן יוחאי / בחך דבש צופים

גם אמצאה קרבת / שדי בכל מלה

אשתעשעה לנסס / בהם ואוחילה.





יושב עלי עב קל / שכלו מאד בפש

סוד מחצית שקל / חבר לכל נפש

חציה וגם סקל / מטיט ומרפש

דרכה ולא עקל / עד תמצאה נפש

תדחה לכל דבת / עמים ותשפילה

רוחם וגם תרמס / נצח ועד סלה



הנה שתי תיבות / אקדים לויכלו

אל חי רצה נדבות / מפי ולא יכלו

עוד ממצוא אשבת / בלב ואם יעלו

וזכר זכות אבות / קדם ימי חילו

גלות לאום רבת / יש קץ לכל תכלה

מיום ליום תעמס / רחם ולא תכלא.
