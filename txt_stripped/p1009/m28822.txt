


	
	בני משרה קחו מני רננים
	








בני משרה קחו מני רננים / ותוכחות לנחים שאננים

לעורר המזמות ההגונות / משלים עם מליצות נאמנים

שנו בהם ושלשו לחקרם / ואל תפנו לדברי ידעונים

באיש פתי חסר לבב לתמו / ולו רעיה כמותו אם לבנים

ילדו אב בילדותו כסיל הוא / והתערב לקץ ימיו לשונים

במראה החלום עמד לנגדו / כאיש חרבו שלופה ציר אמונים

אזי נבעת ושאל לו למי את / והוא ענה לאדון האדונים

שלחני לנפשך לקחתה / והוכן לך פצעים נאמנים

והוא צעק במר אל נא תמהר / אדוני להמיתני חרונים

עדי אסדיר לפניך במלין / ואולי יהמה לבך לפנים

ראה אבי ראה רחם בנותי / ומי זה ימצאה להם חתנים

והבנים בלי אונים ועצמה / לפת לחם גדולים עם קטנים

ועד עתה אני צעיר לימים / ערב עבדך לימים אחרונים

עדי אשלים ימי מדת צבאי / ובני אחרי יהיו דשנים

לפתוייו ולחליקות אמריו / אזי השיב לחרב אל נדנים

בטרם יהלך מלאך אלהים / תקפהו שמע מלי נכונים

בחייך בבואך לעשות את / שליחותך שלח אלי לפנים

למען אהיה מכין לנפשי / לכפרת פשעים ועוונים

והוא קבל בערמתו יהי כן / ואם עלו בפניו קמשונים

והאיש בהקיצו שח לאשתו / והננה באבל כיענים

ולא ידעו למתי הפרידה / והוא נעלם בעיני ההמונים

והאריך הזמן שולי מעילו / ונשכחו סאונים ואסונים

ולא פנה לנגד יום תמותה / עדי הרחיב ליתר על שמנים

ומלא משאלות לבו בתבל / ואז ראה בני בנים אמונים

לקץ ימים ובעל החלום בא / לפתרון החלומות הישנים

ועמד מעלי ראשו להשבית / מעון רבצו וישכן בית אבנים

והוא ענה ולמה תהדפני / כאיש נדהם בטומאת רעיונים

ואיה פה שלוחיך בטרם / תבואני ואשתה רוש פתנים

והמלאך בקצפו וחרונו / במענהו: ולא כן בן זדונים

שלוחי הם לשבעה נחלקים / דבוקים בך לך שוים ונמנים

תחלתם לרואות בארבות / והן כהו אשר היו כיונים

ואזניך לקול שופר כאין / ומקודם בלחש מאזינים

ושניך כבר נשרו בפיך / לימים שכבר שחקו אבנים

שערותך שחורות כאפלה / כמו עורב והם עתה לבנים

וקומתך זקופה היא כאגמון / ונכפפה לארך הזמנים

ורגליך שתים מלפנים / והן עתה לשלש הם נתונים

והופרה אביונה בגופך / ועדיך אמת בחוץ ובפנים

והן אלה קצת עדי לנגדך / וכהנה ידועים למבינים

ומה אוסיף עלי אלה שלוחים / לגלות אזנך קוראים ועונים

ואז שלף לחרבו מנדנה / דקרהו ודמעו כעננים

והלך לאבדון סר וזעף / וימיו הם כצל או כעשנים

דעו שיחי ושירי ההגונים / דעו שירי נערים עם זקנים

למען תדעו שדון לפניו / ותעירו נפשות הישנים.
