


	
	בואי שנת אהבה וצאי שנת איבה
	



תפלה לראש השנה






בואי שנת אהבה / וצאי שנת איבה, /למיחלים קצבה, / יגיל שמיה רבא.

בואי שנת בנים / וצאי שנת אונים, / לקראתך עונים / מהר והקריבה.

בואי שנת גואל / וצאי שנת גועל, / עשו וישמעאל / יכלו כנר כבה.

בואי שנת דופק / וצאי שנת חורק / שניו ומתחזק, / ישכב למעצבה.

בואי שנת השם / וצאי שנת אשם / לדין והקדישם, / נעלז ברוב טובה.

בואי שנת ועד / וצאי שנת רעד / אויב שבור המעד, / ישבת כמדהבה.

בואי שנת זהר / וצאי שנת סהר, / ארצה לך יגהר / הצר בתוך לבה.

בואי שנת חרות / וצאי שנת גרות, / צר יהיה כרות, / נדון ברב חובה.

בואי שנת טהרה / וצאי שנת מרה, / אשיר בקול זמרה / שירה לך ערבה.

בואי שנת ישע / וצאי שנת פשע, / נשמע ונושע / בציון והיטיבה.

בואי שנת כבוד / וצאי שנת אבוד / הצר למס יעבד / לו כאשר גבה.

בואי שנת לקוח / וצאי שנת פקוח, / אקטיר לך רקוח / כפר לכל חובה.

בואי שנת מלכות / וצאי שנת מכות, / זכרה ברית וזכות / מי על שמך נבא.

בואי שנת נפש / וצאי שנת רפש, / תיקר לך נפש / עובד במחשבה.

בואי שנת סבור / וצאי שנת שבור, / אבין בסוד עבור / כל סוד להרחיבה.

בואי שנת עצמה / וצאי שנת אימה / טירות בני אמה / מהר והחריבה.

בואי שנת פדיון / וצאי שנת רפיון / על הררי ציון / תשכן והיטיבה.

בואי שנת צדק / וצאי שנת חדק, / צור חזקה בדק / בנך אשר נשבה.

בואי שנת קימה / וצאי שנת נומה, / על זר שלח אימה, / שמעה והקשיבה.

בואי שנת רצון / וצאי שנת רזון, / אל תטשה הצאן, / עובדים לאין צבא.

בואי שנת שילה / וצאי שנת שח לו, / ינון לעת משלו / יעלה למרכבה.

בואי שנת תקוה / וצאי שנת עוה, לעם אשר אוה, / שמך בעז נחבא.
