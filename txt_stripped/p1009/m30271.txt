


	
	תני נפש יקרה בתחלה
	




תני נפש יקרה בתחלה / תני כבוד לאל נורא עלילה

והשתחוי למול ביתו וכרעי / בתחנונים באישון האפלה

וזכרי מעשים בימי נעורך / בעברך על חמורה גם נקלה

זדון לבך הלא נכתב בספר / ולא הכיל ועוד כתבו מגלה

ימותיך הלא חלפו ועברו / בים סכלות טבועה במצולה

ותחת היכלי ענג ופנג / בראש צורים אמת סורה וגולה

ומלבושך למשי עם רקמות / וארגמן ובתחש נעולה

ואיך שפל עדי ארץ יקרך / וסר נזרך כשמשון עם דלילה

ותמיד אשמרה מחסום למו פי / ודומיה אלי נפשי תהלה

אבל נפשי צרי אמצא לצירך / ואשלחך ועברי המסלה

והצפון למול עינך בלכתך / עלי ההר ומישור השפלה

ומנוח לעולם אל יהי לך / עדי תראי לצקלג עם קעילה

בעיר ציון יהא שמה חנותך / ועיר שלם והעיר האבלה

ויום בואך במלוי הלבנה / ושעתך תהא שעה מעלה

שקי בלאט לאדמת העזרות / ושפכי לב והרבי התפלה

ומשם תעלה לנו רפואה / וגם יתר שאת עם עז וחמלה

ולשבים שעריה פתוחים / ולחנף בפניו היא נעולה

אלהים שם שמו נקרא ושדי / גבורה לו ונצח עם גדלה

ומתנתו לכל נוצר ונברא / ולגזל אמת שונא בעולה

והגידי שלומי לו בראשית / וזכרי את שבחו המעלה

וספרי בדלות האסורים / וסר שכלם ואין דעת צלולה

ואין לי בזמן הרע מתיקה / אשר לא תהיה בראש בלולה

ומעי רתחו המו בקרבי / וכל ימי שנת עיני גזולה

ועיני עששה מרב בכיה / ומחלתי לבת יענה משולה

ולו אהיה אני עם עם נמלים / לרב דלות נשאתני נמלה

אני אדן והצרות קרשים / ואין רגע בלי צרה כפולה

אמת כל משברי תבל מצאתים / ורעות רבות תוכפות למעלה

וריש עם מחסור גברו שניהם / ומה כוחי לידם הגדולה

וערום מבלי לבוש לקרה / לכסות את בשרי המגלה

ופת לחמי במי דמעי מסוכה / והם חלקי וכוסי לאכילה

ויסר הזמן צריו בטפח / ולי יסר באמה האצילה

נגינתי חשבתיה לקינה / וקול שירי אכנה קול יללה

ואם לא אצאה ממאסרי / אזי ארד ביגון אל שאולה

ולא אראה לבני עם בנותי / ואהבתם בתוך לבי שתולה

ונשי השתים העזובות / ולא אזון לאמי השכולה

ומי יאיר ביום צרה לעיני / ומי יגלה לאזני הערלה

הלא נפשי אשר נבראת יקרה /יקרה היא ומכסא אצולה

ויום יפקד אלהים את עבדיו / ודרכי היא לעמתו סלולה

אדון עולם ומתיר האסורים / גאלני אסלד לך בחילה

ויאות אל יציר חמר מעילה / ויאות לך אלהים המחילה

לך אכרע שחרים וערבים / ואפרש כף במנחה עם נעילה

ואתחזק כפי כחי לעבדך / אחונן דל ואיטיב הפעלה

ועונה צועקים נקרא שמך / מרומם דל וסומך הנפילה

יחונן צור בחמלה על עבדיו / יקרב קץ ויחיש הגאלה

ויום יקיץ ישני האדמה / תני נפש יקרה בתחלה.
