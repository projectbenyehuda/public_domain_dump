


	
	ידידי אם תבחר
	








ידידי אם תבחר / להבין סוד זהר

עמד נא בשחר / לשער האורה





פתחיו הם רבים / עשרה נכתבים

רכוביהם עבים / במלכות הנורא





שמו אל״ף דל״ת / ונו״ן יו״ד מועלת



והדין פועלת / בעוברי התורה





ופתח השני / יסודו במעוני

ואל חי יעניני / בעת צדיק יקרא





ונצח עם ההוד / ירכים בסוד

ומשפיעים ליסוד / ומהם לעטרה





בשמות נפלאות / אלהים וצבאו

נביאים ונביאות / מקומם הכירה





ומחשבתך רעי / לקו האמצעי

אחוה לך דעי / ולו התפארה





ויו״ד ה״א ראשיתו / ווא״ו ה״א אחריתו



והלב מודעתו / בנפש הברה





וממך לא נעלם / זרועות העולם

ועצם מפעלם / גדלה וגבורה





ומימין אלי / אלהים לשמאלי

והסוד הוא פלי / לבעל המקרא





ועד פה משמרתך / וטוב כונתך

ותיטיב אחריתך / בשכל ובצורה





ושלש אחרונות / והמה ראשונות

בעמק ענינות / למרבה המשרה





והן כתר חכמה / ובינה נעלמה

והנה נשלמה / עשרה בספירה





שמותיהן אהיה / ויה הוה יהיה

ובם חיה תחיה / נשמה נזהרה





ובבינה סודות / אלהים בנקדות

בארבע מועדות / ובהם נכתרה





ואין סוף לתהלות / לעלת העלות

ממלא משאלות / לכל נפש נברא.
