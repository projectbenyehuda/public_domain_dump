


	
	אזר גבר חלציך
	








אזר גבר חלציך מהרה / וקום עין בדברי הגמרא

היה חכם במלחמתו למען / תהי בקי ותהיה לך סברא

ודע איני ומתקיף והתיבון / לקושיות הם לתנא ואמורא

מקום תימה פשיטא דע והבן / לאביי ורבא הבכורה

ואף על פי ואף על גב תתקן / בעיונך לך דרך ישרה

כדי ניסבא בפרושה וענין / תנא נמי להך דרך עברה

מקום תיק״ו ואליבא דרבי / תרין אנון ואנון חד בחזרא

ובמשנה בחסורי מחסר / תהי משכיל שהיא דרך קצרה

והמדות מקום הקל וחמר / ובנין אב ואם תשוה גזרה

ואדרבה מתקנת להפך / בנוטריקון לקלה וחמורה

וטיב משא ומתן בהויות / לרבינא ורב אשי מסורה

ומהם למדו דרך גאונים / בפסקיהם לכל דעה חסרה

לבלתי ייגעו למצא דרכים / בתלמוד בבלי פים הפצירה

והמסלול אשר תמצא בפיהם / וכל תיקו דאסורא לחומרא

וטיב גטין וקדושין מתקן / בחכמתם ודין עדי מסירה

ובכתבות ומאון וחליצה / ודין פסח ומצה העשירה

ואין בינה בבנים עוד ויש אם / למסרת וגם יש אם למקרא

ועוד דיני נגעים ואהלות / טבול יומו וטומאה החמורה

ודין השור והמבעה והבור / וארבעתם כדין מבעיר בערה

והמשפט בבית ועליה / ואם קל הוא כדין תנור וכירה

והשוכר והשומר למיניו / והמקבל ודין חוכר חכירה

והמיתות בארבעתן לבאור / וכריתות ודין עובר עברה

וחלוקים מדיקים ודקים / בערובין לשבת החמורה

ודין ממרה ודין סורר ומורה / ודין עבד ודין אמה מכורה

ודין לקט ודין שכחה ופאה / ומעשרות ודין בית הבחירה

ועוד נוסף עלי אלה ברבים / ומה אעש ואין לי עוד זכירה

ולא אוכל שאת מהלל לנפשי / בידעי בם בתכלית השמירה

אבל חשקי ידיעתם ולבי / לפרודם כלב אשה מצרה

ולא עלי גמירת המלאכה / ולא אמר לנפשי את פטורה

ואשרי איש אשר זכה לנפשו / בקץ ימיו ומשנתו סדורה.
