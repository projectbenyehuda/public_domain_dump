


	
	חכמת היהודי: מקאמה
	




– – – ויאמר לי: דע אדוני



בתחלת עניני / כי אני הייתי בארץ אשור / אשר מראש אמנה תשור / ויהיו לי חברים שנים / נוצרי וישמעאלי בעלי עינים / ויאמרו לי: ׳היהודים חלשים כלם / מקטנם ועד גדולם / ואינם יודעים לעשות דבר גבורה / ולא דבר גניבות גמורה / אנחנו יש לנו שם / אם יערבו לנו כמה דברים בשוק נירשם / כרגע מבעליהן נגרשם׳ / ואמר אליהם: ׳נסו / אבל הטוענים מקודם עשו / ואני יהיה מעשי אחריכם / לבעבור תפרישו ביני וביניכם / על מנת שכל אחד ממנו זוכה בחלקו / כפי ענינו ופרקו׳ / ויאמרו לי: ׳החייתנו / לא עשקתנו ולא רצותנו׳ / ויפילו גורלות ביניהם / ושבועת השם בין שניהם / ויפל הגורל בתחלה על הנוצרי / ואחריו ההגרי.






ונלך לשוק כלנו / על פי הדברים / אשר בינינו / ונראה בשוק שתי חמות / מלאות
מחמאת הבהמות ויאמר הישמעאלי אל הנוצרי: ׳אם אתה רוצה לרפא שברי / עליך לגנב את החמת הגדולה / מלפני בעליה בתחבולה׳ / ויאמר הנוצרי: ׳עלי מידו להפקיענה / ואתה תגביהנה׳ / ויאמר הישמעאלי: ׳אני אעשה / לא תשאל ולא
תנסה׳ / וילך הנוצרי ובידו צבת / והוא צועק מכאב שנו לא יוכל לשבת / ויאמר אל מוכר החמאה: ׳בחייך הסר ממני חלאה / כי אני שמעתי עליך / בנעם מליך / תרפא בני אדם / על ידי הקזת דם / ותלחש על המכה ותתרפא / ובידך סגלת תרשיש וישפה / עשה לי כחסדך / וקח הצבת בידך / והסר את שני / כי לא טעמתי שנה בעיני׳ / ויאמן בעל החמאה / כי על ידו תהיה רפואה / בידו את הצבת / ויגביה את הנוצרי לשבת / ויתקבצו למולו מן השוק אנשים / נערים וישישים /
והישמעאלי בא לקחת את החמת הקטנה / והוא נבהל ושכלו במגנה / ובעל החמאה ממשש / בשני הנוצרי מגשש / ויאמר הנוצרי ברשעו / כאלו הוא מראה נגעו: / ׳קח הגדולה שבהם / כי שני אמרתי אפאיהם / ויבן הישמעאלי כי הוא רומז אליו / ברע מעלליו / ויקח את החמת הגדולה / וילך לדרכו הסלולה / והנוצרי ידע כי הישמעאלי ברח / ויאמר לבעל החמאה: ׳הכאב ממני פרח / ברוך יהיה אשר אליך הנחני / היום ראיתי בך מה שלא ראתה עיני / ומעת אשר מששת בשני / סר נגעי מלפני׳ / ויקח את הצבת וילך לדרכו / ומוכר החמאה צועק כל צרכו / כי לא ידע מי גנבו / ויבער לבו בקרבו / זאת עצת הנוצרי הבליעל / הנבזה והנגעל.






ויהי למחר ויאמר הישמעאלי: ׳זה היום למולי מצבו / נגילה ונשמחה בו׳ / ונלך בשוק הצורפים / אשר בעלי הכסף עודפים / ונראה שלחני אחד זקן ולפניו תיבה / מלאה מכל אבן יקרה וחשובה / ויאמר הישמעאלי אל הנוצרי: ׳עלי לקחת
התיבה / ותחבולתך על לוח לבי כתובה׳ / ויאמר הנוצרי: ׳ואיך תוכל להעלימנה / ושמא תבוא לידי סכנה? / ויאמר ישמעאלי: ׳עתה תראה / הנסה דבר אליך תלאה / וילך אצל השלחני / האיש מלשני / ויאמר: ׳שלום עליך, בעל האמונה / כל ימי עדנה וזקנה / ויענהו השלחני בכפילת השלום / ולא ידע כי הוא כחולם חלום / ויאמר לו הישמעאלי: ׳אני תמה עליך / איך תאמין על זאת התיבה להניחה בין רגליך / ואתה צריך להצניעה עד אשר לכסף תבקשנה / ומן המטמונים תחפשנה / כי יש בזאת העיר לסטים מזינים / גונבים מקרני ראמים ועד ביצי כנים / ואתה הזהר מהם / פן יראו ממונך בעיניהם / כי אפשר להם לעשות כך / ולהניחך מך מערכך׳ / ויוציא התיבה אל פתח החנות מבחוץ / ויהיו דבריו אל השלחני נחוץ / ויאמר לו: כך עושים / וממון בני אדם תחת כנפיהם מכסים / ויסגר הדלת בעדו / כי קרוב יום אידו / ואסר את העניבה / והשלחני יושב על הארכבה / מקוה לפתח / והישמעאלי הלך בשוק שחוח / ויאמר אל הנוצרי: כמעט נשמתי לא נותרה בי / כי הנחתי השלחני על קרן הצבי / ויצעק השלחני מתוך החנות / מתחיל בשבח ומסים בגנות / ויתמהו בעלי השוק עליו / איך הלסטים בחצי היום גברו עליו / ויאמרו איש אל אחיו: ׳ושמא השלחני נשתטה / ומדרך הפקחים נטה / כי בעיניו נגנב ממונו כאלו הוא בעלטה׳ / ויעזבו אותו צועק / וכל עובר עליו שוחק / זאת תחבולת הישמעאלי הממזר / אשר על בעל דתו מתאכזר.






ויהי ביום השלישי ויבואו שניהם אלי / להראות להם מיטב מפעלי / ומה תהיה גבורתי / ועל איזה אפן תחבולתי / ואמר להם: ׳איני כמותכם עושה / כי כתוב בתורתנו אם נבלת בהתנשא / ואתם לא תמצאו / על ראשי העצים אתכם ישאו / או אם יכר דבר מן הנגנב אצלכם / ראו כי רעה נגד פניכם / ואתם בואו ונלך בשוקי העיר / ובחרו לי אחד מן הסוחרים בממון מרחיב ומשעיר / והראוני חצרו / ואיפה כבודו מנח ויקרו / ולכו לדרככם / ואני אראה אתכם / איך אשבית מטהרו / והוא רואה בעיניו סכלות דברו / ונלך במבואות המדינה / אשר היא לכל המדינות עדינה / ויראוני בית גדול ודירה נאה / לסוחר אחד וסחורתו פליאה / ואני חפשתי על שמו ושם אביו ושם זקנו / בכנויו כתקונו / ואלך בשוק ואקנה מפתחות / על כל מין מפתחות / וקניתי ניר לתקן עניני / וקסת הסופר במתני.






ויהי בערב / יצאתי אני וחברי כמו חיה
למו ארב / ונגיע בחצות הלילה / אל פתח בית הסוחר באפלה / ואפתח באחד המפתחות את הדלת / אז ידעתי כי לי תקוה ותוחלת / ונכנס שלשתנו / ואם ידע בנו הסוחר נשכבה בבשתנו / ונדליק את נרנו / בצנעה להעלים חרפתנו / ונפתח את האוצרות / למען נדע איזה בהם גנוזים אבנים יקרות / ונגיע עד אוצר הסחורה / מצינו בו ארגז בתוכו כל אבן יקרה / עם זהב שחוט / נטוה כחוט / עם כלי חמדה ואבני פז / קבועים בזהב מופז / עם אדם פטדה וברקת / לבנה אדמה וירקרקת / עם מעט מבגדי שש ומשי וארגמן / בירכתי הארגז נטמן / ואוציא את הניר והקסת / ואכתב כל אשר בארגז אבן יקרה מתנוססת / עם הזהב כל כלי בצורתו / ואיך תכונת מלאכתו / לא הנחתי בארגז דבר אשר לא כתבתי / ובתוך הארגז הטמנתי / השטר אשר על שמי יסדתי / וכתבתי אחר כמותו / ושמתיו בידי באיכותו וכמותו / הכל על שמי נכתב ונחתם / נתקבל ביד הסוחר אשר עוונו נכתם / ונצא מן האוצר והארגז סגור / וכל אשר בתוכו אגור / ויתמהו חברי על אשר לא לקחתי מאומה / וישבעו חרפה ובוז וכלמה / ואמר להם: ׳אם לא תצאו מזה כאשר באתם / דעו כי אתם חטאתם / ובנפשכם סכנתם׳ / וישמעו לעצתי / אבל לא ידעו סוף דעתי / ונצא מן הבית ולא לקחנו מאומה / למען נוכל בתחבולה וערמה.






ויהי בבקר ואלך אצל הסוחר / ואמר לו:
׳יש לי משפט עמך אצל הקצין לא תאחר׳ / ואני לבוש לבוש הסוחרים / הבאים מארץ ההגרים / ויקם הסוחר וילך עמי / אצל הקצין והוא היה איש חרמי / ונעמד לפניו שנינו / אני והסוחר המשביע בטוב עדינו / ואמר אל הקצין: ׳ידע אדוננו / בעל משפטנו / כי אני נתתי לאיש הזה פקדון / והוא על זה לפניך נדון / כי לא רצה להשיב בערמתו / הפקדון אשר הפקד אתו׳.






ויאמר הקצין אל הישמעאלי: ׳מה תאמר
בטענת זה האיש אשר עומד למולי?׳ / ויאמר הישמעאלי: ׳לא פקדון אתי, לא נגלה ולא נעלם / ולא היו דברים מעולם׳ / ואמר אל הקצין: ׳אדוני / אם אתן ראיה על פקדוני / מה יהיה תכלית עניני?׳ / ויאמר הקצין: ׳זה גלוי וידוע בכל קצוי נשיה / כל המוציא מחברו עליו הראיה׳ / ויוסף הישמעאלי עלי לטעון / כי אמר: ׳לי הדברי ורברבני בעון / אני נאמן בכל הישמעאלים / ואתה תלעג ואין מכלים׳ / ויאמר לו הקצין: ׳אל תגביה קולך / שמע
הטוען למולך / והשיבהו כפי טענותיו / אל תט ימין ושמאל מדברותיו / ומקלקול הדין עיני אנשי המשפט יכהו / כי המשפט לאלהים הוא׳ / אז נתחזק לבי / וסר ממני עצבי / ואמר אל הקצין: ׳דע, אדוני, כי השם הקימך על הישמעאלים / לשפט ביניהם ולעזר האמללים / ואני באתי מימים קדמונים אל זאת המדינה / זה לי שתים עשרה שנה / ונזדמן מכאן דרך על ידי מעט עסק / למדינת דמשק / והיה בידי ארגז מלא חמודות / הביאותיו ממקום המצודות / קבצתי בו כל ממוני / ולהוליכו באותה הדרך חסה עיני / לבל אוליכנו / ולפני הלסטים המזינים אשליכנו / אז אמרתי לעצמי כי לא אמצא נאמן / כמו זה האיש בסוחרי הזמן / ואתן בידו הארגז למשמרת / ובתוכו אגרת / כל אשר בו מזכרת / וכמותה הנחתי בידי / תהיה עדי ושהדי׳ / ויאמר הסוחר: ׳אם אמת דבריך אשר דברת / טול ממקום שהנחת׳ / ואמר אל הקצין: ׳ישלח אדוננו עמנו אחד / לבל מן הסוחר אפחד / ונלך ונביא הארגז שנינו / ממקום שהנחנו׳.






וישלח הקצין עמנו את נערו / למען הקים את דברו / ונלך אל בית הסוחר / עם הנער השוחר / ונכנס דרך האוצר / אשר בתוכו הארגז במקום צר / ונקח אותו ונשאנו לפני השופט הישמעאלי / וכאשר פתח אותו עמד נס דגלי / כי מצא בו אגרת כתובה / כל המחמדים בתוכה חשובה
ואני נתתי גם כן האגרת השניה / והנה היא לתלפיות בנויה / לא פחתה ולא הותירה / מכל אשר באגרת הראשונה בתוכה זכורה / ויתמה הסוחר ויתמרמר ויתאדם / ויאמר: ׳הושיעה אדוני כי גמר חסיד כי פסו אמונים מבני אדם / חלילה
לסוחרים ואמונתם / כי ככה שקר תרמיתם / וילך הסוחר מלפנינו בבושה וכלמה / כי היתה מכתו עצומה / והארגז מלפני הקצין לקחתי / בחרבי ובקשתי / ואלך אצל חברי ויתמהו ממעשי הגדולים / ויאמרו לי: ׳באמת, כי אתה איש חיל רב פעלים / לך הודינו / ולך ראוי לכרע על ברכנו / ותהיה מעתה אדוננו׳.
