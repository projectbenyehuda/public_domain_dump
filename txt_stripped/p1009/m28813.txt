


	
	אשר שת מלכותו
	








אשר שת מלכותו / עלי כל ממלכת / לפניו בורכת / ובו מתברכת



יסודה הוא אל חי / בנצח עם ההוד / ולו התפארת / בקו מתהלכת



זרועות העולם / גדלה וגבורה / לנגד כל סורר / כקשת דורכת



פעלתם תמיד / בבינה ובחכמה / אצילותם תמצא / לכתר נמשכת



ואין סוף עלתם / וספיר גזרתם / דרש וחקר היטב / בספר מערכת



ותמצא שם בנין / מיסד תלפיות / למעלה ולמטה / בעץ מסתבכת



והוא עץ החיים / לכל מבין סודו / בצלו חמדתי / נתיבות ללכת



ולו שם עמודים / יהודה גור אריה / ומנחתו סלת / ביינה מוסכת



שתו דודים שכרו / וכל רעים אכלו / ראו כי דבש תחת / לשוני נמשכת



והרימו מכשול / וכל דעה זרה / אשר אור השמש / ללילה הופכת



והוא עמוק עמוק / ומי ימצאנו / ועל זאת שאגתי / כמים נתכת



וסוף המחשבה / לאל אשא עיני / אשר לו כל לשון / תהלה עורכת.
