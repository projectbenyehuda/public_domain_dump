


	
	אשר דר במרומו
	








אשר דר במרומו / ושכלל עולמו / והעמיד סלמו / לרחב ולארך



השיבו על כנו / ונטע אפדנו / וטפח בימינו / שחקים יתברך



וקבע מזרות / ושם להם צורות / והיו למאורות / ואין לו בם צרך



ויסד החכמה / גלויה נעלמה / טהורה ותמימה / לסודה אין ערך



והנחיל תושיות / בנויות תלפיות / מתוקים לפיות / לכורעים על ברך



ולי שם מהלכים / לחיים נמשכים / קצרים וארוכים / בצדה לדרך



ועל פי דלותי / וארך גלותי / ומיעוט משנתי / הניקני שד רך



דבש יוצא מנהו / למשכיל יניקהו / ונגד בוזהו / אמת קשתו דרך



יחידים בסגלות / עלו נא במעלות / ולקטו עוללות / בניב עובד פרך



והרכיבו מענה / לבית אל חי פונה / במרכבת משנה / וקראו לו אברך



אשר שח הנכאה / וכמעט קט יראה / ולאל הנראה / תהלותיו ערך



וכל ימיו קורא / שמע נא הבורא / ותמיד הוא ירא / ובלבבו מרך



ושח עד אין מספר / עדי פניו חפר / ונפשו כעפר / לכל רגל מדרך



ונעלה ארצנו / בעת כי תרצנו / בבית תפארתנו / ונשיר שם שירך.
