


	
	פתח פי בנהיה
	








פתח פי בנהיה / וקינה ובכיה / להולכים בשביה / במלכות ההגרים

ואבחר לי קינות / בענינים שונות / ואעזב לנגינות / ומה לי עוד שירים

ואגידה פלאים / לדורות הבאים / ומוסר לפתאים / ומזמות לנערים

יגוני והוא נוטה / שנתו כש״טה / ועל שפם אעטה / כאבל לבכורים

ימותי לעבודה / ולילותי נידה / שנתי נדדה / וחיי נארים

דמעותי אשאב / וגופי הוא נכאב / בעת כי נולד אב / לכל הנזכרים

לכל צרה הוא ראש / ומקדשות לחרש / והשקני מי רוש / ולענה ומרורים

וצר קוה להשמיד / לזקן עם תלמיד / ובו עיני תמיד / בדמעה נגרים

לבבי ארגיעה / בקינות אביעה / לזאת לא נשמעה / בכל המחזורים

עלי עם וקהלה / שבויים בגולה / בהסכמת מעלה / ושבעים השרים

מדכים ומענים / והוגים כיונים / במספד כתנים / בלבות נשברים

ונושאים לסבל / מענים בכבל / וגודל האבל / בחרבן הערים

ולחם לא נמצא / בבית ובחוצה / וקראי לא נרצה / דמעי נגרים

קהלות נערכים / בכל רגע בוכים / בלבות נשפכים / ושותים מי מרים

פזורים במערב / לקרח ולשרב / וכל נגע יקרב / בראשי מבצרים

וכמה יקר סף / לפת לחם נכסף / בגאיות נאסף / ולא בא לקברים

ומהם נודעו / וכעורים נעו / במבצר גועו / לעיני הצרים

ושוכני משכיות / כיוני גאיות / לקדר ונביות / בחנם נמכרים

ומצח עופפת / לבקש טוטפת / הלא היא נהפכת / לכל המשברים

ועין דולקת / לאשפה חובקת / בכל יום צועקת / תרועה ושברים

ולשון לו ישפר / בחכמה אין מספר / ילחך העפר / ערבים ושחרים

ופנים מבהיקים / לעדות ולחקים / מכסים בשקים / באשמה משחרים

ואזנים נגעם / בעת אויב יזעם / ואיך זהב יועם / ופניו נכרים

וידים עורכות / להפך במלאכות / לריחים מושכות / ובהם נקשרים

ורגלים תלך / למצות המלך / לעשיר ולחלך / בכבל נעצרים

ונפש נוספת / יקרה נכספת / ועתה נחרפת / באחד הצורים

לבבות נקרעו / ומתנים זעו / נגידים הטבעו / ביד האכזרים

ואיך אמצא נפש / אנחם הנפש / ומטיט ורפש / למים אדירים

והיגון כתות / ונהרסו שתות / בלילי שבתות / מכבד באורים

נטועה הכנה / ששונה נשתנה / בראשית השנה / ויום הכפורים

אנושות המכות / לדעות המחכות / ויום חג הסכות / בעצם יסורים

ומתני הוא מועד / וגופי הוא רועד / לביאת המועד / ויום הבכורים

והגשם חלף / וגבר הסלף / והציץ לא שלף / בשמחת הפורים

ואין חומל עלי / לחזק נחשלי / ונשארו רגלי / סבוכים בסירים

ואין מקשיב קשב / ואין אומר השב / כקרבן לא נחשב / כאישים נבערים

סעפי אל תנשו / ורעיוני חושו / פרודות עבשו / וחתו גבורים

אטהר הנפש / ואכרע בחמש / למזריח שמש / ומעריב בדברים

ואניע שפה / בקומה נכפפה / לרוח נשטפה / וחיים נעכרים

לעצמותי אמגר / במלכות החגר / לאזרח עם הגר / חבלים עם צירים

לשברי הוא אוה / נתיבותי עוה / ודמעי הוא רוה / גבעות עם הרים

לעדה נזנחה / אסורה השמחה / ותמיד האנחה / חדשים לבקרים

וגבר בן אמה / בקצף ובחמה / עלי יונה תמה / כחיות ונמרים

ויש מהם נדכה / בכל רגע יבכה / ויש בשחין מכה / וגם בסנורים

ולא נשא פנים / ולא חן לזקנים / ואבות על בנים / בכבל נאסרים

ומי יזון לטף / בבוא יום השטף / ואב מהם נחטף / ושבעו תמרורים

ומי יאיר אורה / ללומדי התורה / ינהג כשורה / לבנים נמהרים

ונשים שאננות / עזובות אלמנות / בקינותיהן שונות / מתנות לגברים

אחזק הבדק / בתוך עם כחדק / והוא שופט צדק / במבחר מישרים

ואיך לא יראה יה / לבניו בשביה / ועד כמה הויה / נעילה לשערים

ועוד שער אחר / לדמעה לא תאחר / אני תמיד שוחר / ומי לי לסגורים

אלהי הקשיבה / בכי עין זבה / ותהיה נכתבה / לחיים בספרים

ודתך לא נמר / ואחוה לא נפר / ואם דינך גומר / אנחנו נמסרים

ועת בנים תמכר / ברית אבות תזכר / ואל נא אל תעכר / למעמד היקרים

ואם חטאי גבר / ולבי לא נשבר / ראה נא לנבר / והצל נותרים

היה מעוז ומחסה / לעם נאלם כשה / בצלך יתכסה / ותעזב לעדרים

אלהים חננו / ובוז רב שבענו / גאל נא אותנו / כמיד המצרים

אמר נא אל שדי / לצרותינו די / וענה מעבידי / מלכים עם שרים

ולבם תקשיח / ודעתם תשיח / ויבוא משיח / וקרננו תרים

בבוא בן־עמיאל / לנחם ישראל / וזרע ישמעאל / להכרית ולהחרים

פנה אל חטאתם / וערבב שמחתם / הפך את טירתם / כמהפכת זרים

אבשר גליות / ברדתם תחתיות / ונעלה לעליות / זקנים ונערים

ותרבה הדעת / כשדה נזרעת / וחכמה נובעת / כימים ונהרים

והצל נא גוחי / לרגלי מדחי / ונפשי כי תחי / תהלל בשערים

וגלותי תשיב / ורנתי תקשיב / הלא אתה משיב / נשמות לפגרים

ובמלאת הימים / לגן עדן תמים / ואודה בעמים / בפנים מזהירים.
