


	
	לבי עלי צלעי
	








לבי עלי צלעי / יוחיל למנוח

רבה יגיעתי / למאד בלי כח.



זוחל בבית כלא / לא אמצאה נפש

באמת אני חולה / בגוף ובנפש

נופל ואין דולה / מטיט ומרפש

אל נא עשה כלה / וכתב שטר חפש

ומחה כעב דמעי / ורצה כניחוח

ואדום ובן אמה / לרטש ולזנח.



כלה מאור פני / חשך עדי הקדיר

תמו לריק שני / פשעי לרב אסדיר

גבר המון מוני / ניני ערב העדיר

אשא לך עיני / נורא וגם אדיר

לך ענני דמעי / אשלח כשילוח

דופק שעריך / מהר ולפתח.



רעוץ בתוך מוקש / כמעט כעוף ידאה

ידמה למץ או קש / שפל אנוש נכאה

נשפט כדור עקש / הוא מסבל ילאה

את עבדך בקש / רואה בלי נראה

תמרח עלי נגעי / ישבת כמי נח

הצר והאויב / ידכה וישוח.



ימי בכור עני / תמו בראש צורים

כמעט ואינני / עת אעבד זרים

אגוע עלי קני / נמכר לאכזרים

מהר וחנני / משפיל וגם מרים

כמה יהא בצעי / נכתב עלי לוח

צוה אלהים חי / למחל ולסלח.



היום מרי שיחי / אערך לנגדך

בכיי במר רוחי / שימה בנאדך

אצעק בכל כחי / תענה בחסדך

נפשי בעוד תחי / אפקיד בידך

ששים עלי צלעי / תזרם לכל רוח

שבוי קרא לדרור / אסור פקח קוח.
