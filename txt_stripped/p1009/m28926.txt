


	
	ברכי נפשי את יי
	








ברכי נפשי את יי

ברכי אימה מכל נוצר ונברא / את שם אדון אשר לו הגדלה והגבורה

אחד ומסוד קדשתו נפחד ונירא / אין קדוש כיי





ברכי בחונה מכל נפש חיה / את שם ברוך שאמר לעולם והיה

בחרך והנחילך דת ותושיה / ברוכה את ליי





ברכי גומלת חסד לגוף להחיותו / את שם גואלך למען תורתו ומצותו

גבוה ומשגיח לכל בעבור בריתו / גדולים מעשי יי





ברכי דבקה בגוף בלא רצון / את שם דוברך והזהירך מלצון

דגלך בשכל פנימי וחיצון / דעת צדקות יי





ברכי השוכנת בחדרי חדרים / את שם המכין מקומך לעתים ספורים

הטהרי ועמדי במקום טהורים / הנגשים אל יי





ברכי ועודה להשיג המעלות הרמות / את שם ועד המחכמך בחכמות נעלצות

והחושים קראו בשמותם עלי אדמות / ועליך יזרח יי





ברכי זכה מטנף הגוף ומעללו / את שם זוכר חסד נעוריך בהיכלו

זמרי לשמו בכל רגע במהללו / זה השער ליי





ברכי חגורה בעז לעבודת יוצרך / את שם חוננך דעת מלמדך ומורך

חפץ לזכותך אם תקבלי מוסרך / חן וכבוד יתן יי





ברכי טהורה כגלגלי מעלה / את שם טהור בחר בעמו סגלה

טוב ומטיב ולו דומיה תהלה / טעמו וראו כי טוב יי





ברכי יושבת בגוף עד יום פקדך / את שם יושב תהלותיך ויודע סודך

יראו החטאים כבודך והודך / יזכרו וישובו אל יי





ברכי כלולה מן המעלות כלם / את שם כביר כח מכפר לחטאים מעלם

כונן כסאו בחסד ומשפט וצוה לעולם / כי באלה חפצתי נאם יי





ברכי לובשת כבוד מהדר אל והודו / את שם לובש גאוה ולא נודע סודו

לקחך מתחת כסא כבודו / לעמד לשרת בשם יי





ברכי מיחלת לעלות למחצבה הראשון / את שם מורך מצוה בלא פה ולא לשון

מחכה למלכה השומרה כאישון / מדי עלותה בבית יי





ברכי נשקפת בכל עת על הגוף לעוררו / את שם נצב בשמים עד עת בוא דברו

נועדים מלאכיו לפארו בהלו נרו / נושאי כלי יי





ברכי סופרת לעצמה ימים ושנים / את שם סולח עוונות אדמים כשנים

שמחה להצילה ממים הזידונים / סומך צדיקים יי





ברכי עלובה באודות הגוף וקורותיו / את שם עליון על כל גבוה וגבוהים משרתיו

עונה שועת צועק מצרותיו / עזרי מעם יי





ברכי פותחת בכל עדה במקהלות / את שם פודה לעמו ממלא כל שאלות

פונה אל פלולך בקומך לחלות / פני האדון יי





ברכי צורה מצירת בצלם אלהים / את שם צור עולם המופיע מרום גבוהים

צדק במאמרו וכל יודעיו תמהים / צדיק הוא יי





ברכי קרובה אל מעלות הסלם / את שם קרוב לכל קוראיו נגלה ונעלם

קנא ומי הקשה אליו וישלם / קדוש ומקדש יי





ברכי רמה ממחצב היקר חצובה / את שם רוכב שמים לא ישבתו ממסבה

רחש והקדים לחטאים תשובה / רם על כל גויים יי





ברכי שוקלת מעשיה בכל שנה / את שם שומע צעקת יתום ואלמנה

שוקל בדעתו ודעתו לא תשתנה / שם יתנו צדקות יי





ברכי תמה והשכל לכבודה משתאה / את שם תולה עולם בעמוד נעלם ממראה

תקן גוף ונשמה סובלת לו ולא תלאה / תחלת דבר יי





זוקף כפופי ראש יחיד הקימנו / יהי שמך לעולם בכל נתיבותינו

סוד עשר דברות יה נסה עלינו / אור פניך יי

ברכי נפשי את יי
