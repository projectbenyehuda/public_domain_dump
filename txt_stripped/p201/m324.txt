


	
	מוחרך
	




יפתה עלמת / חן ומעין / נעלמה:



בעלי חכמת / לב, דעו “אין, / איך ולמה”.


  


ירדה אל בית / טיט בהתאסף / גו וצורה



ותהי נחבאת / שם באין כסף, / רק עצורה.



לא ביד נשבית / גם בלי כסף / היא מכורה



לעבד אדמת / גוף, ולשים לו / הוד ואימה



חק, ובמזמת / לב להבדילו / מבהמה.


  


סוד פליאתך / מה מאד יגבר, / צור, להבן!



עז בריאתך / באנוש – חבר / פז לאבן



ראש פעלתך / היא – ואת הבר / מה לתבן?



חברה עצמת / ידך גלמי / אל נשמה,



כוננה רקמת / מחזה צלמי / מאדמה.


  


שאלה חפש / מאשר עבד / בה יחידה



יום להנפש / בו, ואל תכבד / העבודה,



כי יסוד נפש / חי בלי יאבד / יום פרידה,



באשר קדמת / גוף יהיה נמצא / היא קדומה,



אך לפי תמת / מעשים נרצה / או זעומה.


  


יה, תהלתך – / צוף, ומלתך – / מן ונפת,



עז פעלתך / על גדלתך / אות ומופת.



חן קהלתך / בית תפלתך / נאספת!



תעטף נשמת / כל בני חלד / לך כאמה,



קול בלי נצמת / “הללו יוצר / כל נשמה!”
