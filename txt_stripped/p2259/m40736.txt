


	
	[יודו לשמך עליון מורה] (רשות לנשמת)
	




Agnelo d’Ascoli


[חי בשנת 1436 בעיר קמרינו]





רשות לנשמת של יום הכפורים




יודו לשמך עליון מורה / דרך לשוב לך
כל־חי



עושה כל יוצר ובורא / נפש רוח נשמת כל־חי.





עם זו בחר אדון הכל / לסגלה
מכל־עמים



ועד־לא ירה פנת כל / הגדיל יום זה מכל־ימים
ובכן מאז תכן הכל / אריכות אפו ברחמים





כי יצר לב־איש ונסכו / ידע מאז יוצר
כל־חי



לכן ייטיב כל־איש דרכו / ובכל־לב ישוב
לאל־חי.





קהלות עמך בתפלתם / לפניך הם
נעצרים



בחמש ענויי תאותם / ודוי פשע מסדירים



אנא סלח על־חטאתם / בזכרם מדות היקרים





כהבטחתם על־יד חוזה / משה עניו מבחר
כל־חי



לכפר עליהם ביום הזה / ולקרא להם בני אל־חי.

בריח מסגר גלות שבים / התר־נא למען
שמך



ותשקיע במצולות ים / פשע ועון חטאת עמך



הפק להם כל־מאוים / והצמת עם זונה ממך





ושבות עמך השב מהר / ואז יודה לשמך
כל־חי



ובאור פניך תנהר / לעם חוסה בשמך אל־חי.







ושכינתך בתוך ציון / מכון שבתך
תשיבנה



ולהאירה מרום חביון / על־נחלתך תכוננה



ולתתה על־כל־גוי עליון / כהבטחת אל אמונה







וכן תקים כל־נאמיך / מהרה לעיני
כל־חי



וכל־לשון יקרא בשמך / ויענו יחד אל אל־חי.





חסדך אשר הוא מעולם / הגדל על־עם בך
חתומים



מהר לגלות קץ הנעלם / כי לא תזנח לעולמים



למקטנם ועד־גדולם / ידעוך רם על־כל־רמים







וכל־עם יודה כי אין גואל / זולתך אדון
כל־חי



פודה ומושיע ישראל / אשר חבל חלקו אל־חי.





יהיו לרצון תהלות עמך / ולא תבזה
את־ענותם



ושלוחיהם מבקשים ממך / רחמים עליהם בתחנתם



רצונם ישיגו מעמך / ופנה־נא אל־קול שועתם





בטחונם בך בזכות הורים / פותח יד
ומשביע לכל־חי



רצה בם כעולות פרים / רוממים לך נשמת כל־חי.
