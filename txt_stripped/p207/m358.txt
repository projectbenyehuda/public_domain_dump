


	
	[השערוריה באורווה]
	




ויאמר אלי “עד מתי תעז פניך / לחרפני ולהכני בשוט לשונך?/ אכן אסבל חרפתך / למען שכלך ותבונתך”. / ויכף ראשו ויאמר בבשת פנים / “כזה וכזה ישא האנוש מאוהביו הנאמנים. / ועם כל זה אתמה על חכמתך – / ויותר על אכילתך: / אשר מאומה לא השארת / ושלחני מכל טוב חסרת, / ודבר שלמה אל לבך לא שת / “ושמת שכין בלעך אם בעל נפש אתה”. / הכזבת דברי החכם אשר אמר, כי חכמה ודעה / לא ישכנו בבטן מלאה ונפש שבעה”. / ואמר אליו “עד אן תמלל אלה / ואי זה מאכל בטני מלא? / והלא כל אשר אכלתי הלילה ברעבוני / לא מלא חדר אחד מחדרי בטני”.



ויאמר “המקום יצילני מכל צרה וצוקה, / לא אמשיל בטנך כי אם לעלוקה! / ואתמה על שלמה אשר אמר: שלוש הנה לא תשבענה ארבע לא אמרו הון. איך לא אמר: חמשה – ותהיה בטנך החמישית אשר לא תשבע לעולם, / והיא שקולה כנגד כלם”.



ובעוד היה מדבר עמי בלשון צחות / אמרתי “ברוך זוכר כל הנשכחות! / אדוני בעל הבית, ידעת אם אכל הלילה חמורי מספוא / או נפל עליו הקיר ומלק את ראשו ממול ערפו / אשר לא אשמע נהקתו / ולא אאזין נאקתו?”



ויאמר “אין זה כי אם מרב שביעתו”. / ואמר בלבי: מי יתן ויצדקו מליו / וישבע החמור אחרי אשר לא נשבע בעליו / ויקרא לעבד ויאמר אליו / “נתת הלילה לחמורו את בלילו / מדוע לא ישמע קולו?” / ויאמר העבד "אדוני, שמתי לפניו תבן ומספוא מלוא האבוס / ואכל ושבע עד אשר אותם נפשו תבוס ". / ויאמר “ומי יתן והיתה כקבת החמור קבת אדוניו / ואולי היינו אוכלים מן הנשאר לפניו”. / וחשבתי בלבי כי העבד הממר, אמת היה אומר. / ואמר “אדוני, בעבור אהבתך ונדיבתך  חשבתי כל מאכל כמן / והנה שלחנך לפניך סימן, / ועל כן לא הותרתי ממנו עד בקר”. / ויזעק ויאמר “אין כל דבריך בלתי כזב ושקר! / עד אן תהותת עלי / ותבוז לשכל מלי? / דם והאלם, / כי לא תבוש ולא תכלם. / טאטאת שלחני במטאטא השמד – / ועדן כל אשר בביתי לאכל אתה חומד!” / ואמר "אדוני, כפך מעלי הרחק / ואל תשת לבך לדברי לעג ומשחק ". / ויאמר “ואיך אסיר את אפי? / ואכלת את כל יגיע כפי / וכמעט אותי במכתש לחייך תשחק – / ותאמר כי תלעג וכי תשחק!” / ואמר אליו "אדוני, מיום היותי / כן היתה אכילתי, / ועל כל הנשאר לבי יכאב, / כי מנעורי גדלני כאב / ומבטן אמי אנחנה / לרוות בטני ולהשביענה “. / ויצחק צחוק הדומה לבכי ויענני עזות / “ארור כל בן אשר ילמד מאביו המלאכה הנבזה הזאת!” / ויאמר “ומי היה אביך?” אמרתי לו “אבי ז”ל היה איש תם ונבר, / רופא חכם ומשכיל בכל דבר”. / ויאמר “מדבריך אתבונן כי אמת הדבר ונכון, / כי היה חכם בכל דבר ונבון, / ועדות חכמתו הגדולה – / אשר למדך  חכמת האכילה!”



ובעוד היינו מדברים נדעך הנר אשר היה לפנינו / ויעצם האפל את עינינו. / ויקרא אל עבדו ויאמר לו “הדלק מהרה את הנר!” / ואין קול ואין עונה, כי הרגו אחי ענר. / ואמר “מה קרה לעבד הזה?” / ויאמר “אולי הוא שוכב הוזה / ויושיבני במחשכים / עד אשר אור הבקר מציץ מן החרכים”.



ואמר אליו “אדוני, הבקר אור / הקץ את העבד וילוש וישם בבצק מעט שאור”. / ויאמר אלי “הבשר עודנו בין שניך  – / וכל זה מהרה תחפץ למלא בטנך? / הלא שמעת דבר אבוקרט באמרו, כי הבאת מאכל על מאכל / רב בני אדם המית ושכל”. / ואמר “כן דברת / וצדקת בכל אשר אמרת, / אבל כל אשר אכלתי נעכל במעי ונספה / כאכל קש לשון אש וחשש להבה ירפה”. / ויאמר “המקום יכה בשלשול קבתך  / עד אשר לא תוכל לעצר את צואתך!” / ואמר “ידעתי כי רעה עלי חרשת / אכן התפללת ולא פרשת: / אם אחזני שלשול, אולי יהיה לי מרפא וארוכה, / ויהפך  לי השם את הקללה לברכה, / כי אמר גאלינוס : כאשר יקרה לאדם שלשול מן הלחות אשר הטבע צריך להוציאם מן הגוף ולפנותם / יחזקו בו כל האברים ויהיה סבת בריאותם”. / ויאמר "אם טעיתי בתפלתי, / המקום יודע כונתי ". / אמרתי “ידעתי כונת לבך ומחשבו / אכן קללת חנם לא תבוא”.



ואלך לראות את חמורי ואבוא אל האבוס ואמצא אותו מבלי תבן ומספוא / ולא יכול להגות על טרפו / וכמעט מצאהו מחנק, / כי העבד הבליעל חסם את פיו לבל ינהק. / ואגש אליו להסיר את המחסום, וכאשר הסירותיו נשכני / וכמעט בין רגליו הדריכני. / ואקח את המקל ואך אותו – / ועד לשמים עלתה שועתו . / ויפתח יי את פי החמור ויאמר בעברה וזעם / “למה תכני ואני ממשפחת אתון בלעם?” / ואתמה מדברו וחרדתי מחרונו  / ואמרתי: אולי השטן מדבר מגרונו. / ואמר אליו “ואיך לא אכך – ואתה נשכתני / וכמעט בין רגליך רמסתני?” / ויאמר החמור “הנה ישבת עם הנבל להתענג ולאכל – / ואני ברעב ובצמא ובחסר כל / וכל הלילה אשמרה לפי מחסום”. / אמרתי “הבלעם אנכי אשר ידע לקסם? / מאין אדע כי צמאת ורעבת / והעבד הרע אמר אלי, כי מאשר אכלת ושבעת / באבוס רבצת וכרעת”. / ואזעק אל העבד ואמר אליו “למה המת את חמורי ברעב ובצמא / והרעבת אותו באכזריות חמה?” / ויען אותי וישבע, כי שם לפניו תבן ומספוא ואכל ושבע – / אכן קבת החמור כקבת אדוניו / אשר לא השאיר מאומה לפניו. / אמרתי “ואם נתת לו מספוא לשבעתו , / מדוע חסמת אותו / לבל אשמע נהקתו?” / אמר "בעבור כי הוא חמור שעסקיו רעים / וכל הלילה היה צועק ומרעים, / ועוד כי פחדתי פן ישכני בשניו ".



אמרתי “באמת כעבד כאדוניו! / ואמר הערב, כי העבד מחמר אדוניו קרץ”. / וישמעני אדוניו ולשונו עלי חרץ, / ויאמר עינן “למה תכביר עלי במלים / ותבזני זה שלש פעמים?” / ואמר אליו “על זה הוצאתני מעירי / להמית ברעב אותי וחמורי?” ויאמר “ארור היום אשר הוצאתיך בו, / במספר ירחים אל יבוא! / כי לא הבאתיך אל ביתי כי אם לבשתי וכלמתי / ולהוכיח עלי פשעי ואשמתי. / אבל חי נפשי ונפש זקני אשמדאי, / אם לא תאמר לסכלותך די, / ולקחתיך באצבעי הקטנה / והראיתיך כנה / וקרית חנה!” / ואביט אליו – והנה פני להבים פניו / וכידודי אש עיניו / ומנחיריו יצא עשן / כקיטור הכבשן. / ואמר בלשון רכה / ונפשי עלי מבכה / “אדוני, אשאלך ואל יחר בעיניך: / איזה אשמדאי היה זקנך?” / ויאמר “אשמדאי השר הגדול אשר באברתו / השיא המלך שלמה ממלכותו / והרחיקהו מאדמתו”. / ואמר “אוי לנפשי כי חשבתיך  מן הידידים – / ואתה מבני השדים / אשר רוחות בני אדם מצודדים! / זה חסדך את רעך? / למה לא הגדת לי זרעך / ומדוע ככה רמיתני / ומארץ מולדתי הוצאתני?” / ויאמר אלי “ואיה אפוא שכלך ותבונתך / ומדעך וחכמתך? / והלא אמרתי לך כי שמי עינן הנטש בן ארנן הדש! הפך שמי ושם אבי ותמצא: עינן השטן בן ארנן השד – בן חצר-מות / בן עזמות / בן צלמות / בן חרדה / בן רעדה / בן אימה / בן חמה / בן משכלה / בן חלחלה / בן דבר / בן שבר / בן יגרע / בן שם רע / בן עלון / בן יון / בן לצון / בן אסון / בן מגפה / בן אנפה / בן רם / בן שרם / בן אשמאי / בן ארמאי / בן סרחדון / בן אבדון / בן קלקלה / בן קללה / בן אבדאי / בן אשמדאי / מלך השדים / אשר כלם היו לו לעבדים”.



וכשמעי את כל אלה השמות / אחזוני שאון ושמות / ונפלו עלי פחד ואימות / ושכח לבי כל גיל ועליצות / ואחז בשרי פלצות.



ואמר “אוי לי על שברי / כי פה יהיה קברי! / אמת אמרו חכמים: רגלוהי דבר אנש  אינון ערבין בה / לאתר דמתבעי תמן מובלין יתה ”.



ויאמר אלי “אל תירא ואל תחת / וכל דבריך יהיו בנחת: / כי חי כל הנשמות אשר זכרתי – / לא אעזבך עד אשר עשיתי את אשר דברתי! / קום התהלך בארץ לארכה ולרחבה / וראה יפיה וטובה, / אפס כי עז העם היושב בה”.
