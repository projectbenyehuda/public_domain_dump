


	
	[האשה שלא עמדה במבחן]
	




אמרו כי מלך אחד ממלכי הערב היה חכם ונבון ובעל משפט, והיו חכמיו ונבוניו / יושבים תמיד לפניו. / ויום אחד התחילו לספר בשבח הנשים וגדלתן / ורב חכמתן וסכלתנותן. / ויאמר להם המלך "קצרו דבריכם / ועצרו מליכם, / כי מעולם לא נראתה ולא נשמעה / אשה טובה וכשרה, בעלת שכל ודעה, / וכל אהבתן לפי הנאתן וצרך שעתן / ואינן מושלות בתאותן / וחוטאות בנפשותן ". / ויאמרו לו חכמיו “אל יאמר המלך כן, כי יש נשים חכמות ונבונות, / כשרות ונאמנות, / אוהבות ומכבדות בעליהן, / מכסות ערות בתיהן / ומפרנסות בניהן ובנותיהן, / וכל מום ודבה אין בהן”.



ויאמר המלך “אני אתן לכם אות על דברי / ומופת על מאמרי: / הנה העיר הזאת גדולה מאד, שוטטו וראו אם תמצאו בה אשה אחת בכל הטובות אשר אמרתם / והמדות אשר זכרתם”. / ויבקשו וימצאו בה אשה אחת בעלת שם, צנועה וכשרה וחכמה / יפה כלבנה ברה כחמה, / ובעלה איש עשיר גדול וסוחר / מבקש רצון וטוב שוחר. / וישיבו אל המלך דבר ויאמרו לו “הנה מצאנו אשה חכמה / ומשכלת בלי מום ומרמה / ובלי מרד ומעל, / והיא בעולת בעל”. / וישלח המלך לקרא לאישה, ויבוא לפניו ויכבדהו / וירחיב לו מקום ויושיבהו. / ויאמר לו “דבר סתר לי אליך”. / ויאמר לו “דבר אלי עבדך, / כי ישמע לקול מליך ”. / ויאמר לו המלך “יש לי בת טובה וחמודה / והיא לי רק יחידה, / ולא אחפץ לתתה למלך או לשר, / כי חשקה בלבי נאסר, / רק אבקש איש טוב ונאמן / וזך מכל חלאת הזמן / יאהבנה ויכבדנה / ועל שכמו ישאנה / ועטרות יענדנה / ובסבר פנים יקבלנה / ובחסד וברחמים יכלכלנה. / ואני שמעתי עליך ועל רב מהלליך  / וטוב מדותיך  ומעלליך, / וארצה לתתה לך לאשה – / אבל לא אוכל לתתה לבעל אשה. / הרג את אשתך הלילה ומחר אתננה לך לאשה!” / ויאמר לו הסוחר “אדוני המלך, מי אני ומה חיי, כי אהיה חתן למלך – ואינני ראוי לרעות צאנך?” ויאמר לו “אני אבחר בך, כי אתה מאויי וחפצי, / ובלעדיך לא ירים איש את ידו ואת רגלו בכל ארצי”. / ויאמר לו “אדוני המלך, איך אוכל להרג את אשתי? / והיא חמש עשרה שנה אתי, / מפתי תאכל ותשתה מכוסי, / והיא שמחת לבי ומשושי, / תאהבני ותכבדני / ותשמרני ותעבדני, / וכבודי בכל יום בעיניה יוסיף ולא יגרע, / ונשמרתי מכל דבר רע”. / ויאמר לו המלך “שמע בקולי והרג אותה,/ כי כבוד גדול יהיה לך במותה: / כי על כל שרי מלכותי ארים ראשך / ומשלת בכל אשר תאוה נפשך”. / ויאמר האיש “אנסה ואחזה / אם אוכל לעשות את הדבר הרע הזה”.



ויצא מעם המלך סר וזעף / ודם כבדו מנחירו רועף. / ויבוא אל ביתו / ובראותו את אשתו / גדלה אנחתו / ונכפלה דאגתו. / ותאמר לו אשתו “אדוני הטוב והנעים, / מדוע פניך רעים?” / ויתאפק ויאמר אליה “צורי ישמרך  ממות וקטב, / אין לי דבר ואין בלבי רק טוב”. / ויהי בלילה, והיא ישנה על מטתה , / קם בתמהון לב להמיתה. / ויקח החרב בימינו והנר בשמאלו ויסר את הכסות מעליה – / ויראנה ישנה ושני בניה שוכבים בין שדיה, ויחמל עליה / ויאמר “אוי לי, איך אהרגנה / וחרפתי אנה אוליכנה? / ומי יגדל את בני / אשר הם כאישון עיני? / אין זה כי אם רב פשעי ועוני!” / וישב החרב אל נדנה , / ונפשו מתמוגגת ביגונה, / ויאמר בלבו – ועיניו בוכות: / הנה אשתי טובה מכל הממלכות. / ארורים יהיו כל המלכים / כי הם אחר תאותם הולכים / ולבות בני אדם בהבליהם מושכים / וביין שמחתם מי אנחה נוסכים! / ויעל על המטה וישכב עמה וינשקה, / וישם שמאלו תחת לראשה ויחבקה.



ויהי בבקר ויקם וילך לחנותו / לעשות מלאכתו, / והמלך מיחל אותו. / ויתמהמה, וישלח המלך למענהו / לבוא לפניו ולמהרהו  / ולדעת מקרהו. / וכראותו שליח המלך גדל כאבו / ופחד ורהה לבבו. / ויבוא לפני המלך, ויאמר לו “מה עשית – הרגת אותה?” / ויאמר “לא אדוני, כי גדלה עלי אהבתה וחמלתה”. / ויספר לו את כל אשר קרהו  בקחתו  את החרב להכותה. / ויגער בו המלך ויאמר לו " צא מנגד עיני / אל תוסף ראות פני, / כי אינך מן האנשים / ולבך כלב הנשים“. / ויצא מלפניו, ותרב שמחתו / לזעף המלך וגערתו. ויהי ממחרת לעת ערב שלח המלך אחד מעבדיו להביא את אשת האיש בסתר, / וילך ויביאה אליו. וכראות המלך אותה / תמה על הוד תמונתה / והדר תארה ודמותה, / ויאמר אליה: הנה שמעתי על חכמתך / ועל יפי יפעתך, / ומיום שמעתי שמעך קדחה בלבי אש אהבתך / ונמשכה נפשי בחבלי תאותך , / ועתה חפצתי לקחתך לי לאשה / והמלכות תהיה לך למורשה. / אמנם לא אוכל קחת אשת איש, פן יהיה לבושה. / לכי הלילה והרגי את אישך – / ואחרי-כן אקחך ואעשה כל אשר תאוה נפשך”. / ותען האשה בטוב-לב ובשמחה / “אעשה מצות המלך ואהיה לו לשפחה”. / ויאמר לה המלך “אבל תהיי אשתי וכל נשי תהיינה אמהותיך  / ופילגשי – שפחותיך”. / ויתן לה המלך חרב בדיל לטושה, / כי ידע חסרון דעת האשה, / ויאמר לה “הנה חרבי חרב חדה מאד - / הכי אותו פעם אחת ולא תשני לו / ולא יוכל עוד קום לו”.



ותקח החרב ותלך לביתה / והיא מתרפקת בשמחתה, / וטבחה טבחה, אף מסכה יינה / ותבא מטעמים ותערך שלחנה. / ויבוא האיש אל ביתו / לאכל ולשתות עם בניו ועם אשתו, / ותשקהו יין בלילה ההוא ותשכרהו / ותשאהו  אל המטה ותשכיבהו. / ותקם בעוד לילה, והוא בשכרותו / ישן על מטתו , / ותקח החרב ותך על ראשו / ותחשב כי ברזל באה נפשו – / ותכפל חרב הבדיל ויעור משנתו / וינער מעט משכרותו. / ויצעק ויאמר לאשתו “מי משנתי העירני / ועל ראשי מי הכני?” / וכראות האשה כי הקיץ ולא הזיקתהו  חרבה, / חרדה חרדה גדולה ויצא לבה / ומרב פחד ואימה / כמעט לא נותרה בה נשמה. / ותאמר לו “אהובי, ידיד נפשי, שכב ושוב לנומך – / אולי מכה אותך ראית בחלומך”. / וישב אל שנתו ולא יכל לפקח עינו, / כי עוד לא הקיץ מיינו. / ותרב דאגת האשה עד אין חקר / ותתחלחל עד אור הבקר. / ותשכם לעשות מעשיה ולתקן את ביתה / לכסות ערותה מהמעשה הרע אשר עשתה / ולעשות לבעלה מטעמים / כי כן משפטה כל הימים.



ויהי המלך מיחל עת בואה, / ולא באה עד אשר צוה להביאה. / ויהי כבואה ויאמר לה המלך “עשית את הדבר אשר היה ביני וביניך  / או חסו עליו עיניך?” / ותאמר “אדוני המלך, אני עשיתי מצותך  – / אכן אתה שמת לאל מגמתך / והפרת עצתך הטובה ותסכלה, / כי חרבך הנפתי עליו – ותחללה.” / ותספר לו את אשר עשתה / ואת מעותה  אשר עותה. / וישלח המלך לקרא לאישה ויאמר לו “ספר לחכמים היושבים לפני את כל אשר קרך  עם אשתך, אל תכחד מהם דבר!” ויספר להם את כל המוצאות אותו עם אשתו ואחרי כן צוה המלך את האשה להגיד להם את כל אשר קרה לה עם בעלה ותעש כן. ויען המלך לחכמיו "הלא זה דברי אשר דברתי אליכם / באמרי לכם "עצרו מליכם ". / ויענו “כן” והם מחכמתו תמהים / “מה נאמר – ואדוני חכם כמלאך האלהים!”
