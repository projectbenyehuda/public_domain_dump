


	
	וכוח האדם
	







הוכחת השלושה עשר






וכוח האדם עם אדמתו / ויזכר לה ימי קדמתו

ויצעק עליה בהגיון אמרתו / ומרב אנחתו בצע אמרתו

מוסר לאדם הנרדם אשר להבל דמה / יערך את תוכחותיו בנחת עם האדמה

ויוציא את כל מצפוניו בקול דממה / כי מחר ישכב בעפרה כאבן דומה

לכן קדם אמרי הוכוח / ברב אונים ואמיץ כח

מנגד גאותו הנעדרת יברוח / ולא ידבר בגסות הרוח

יכניע גאונו וידכה ישח / ושאון אמרותיו בענוה ישוח

יבקש לנפשו היקרה מנוח / ויכיר מקומו אנה ינוח

אם לבבו כסיר נפוח / יהלך לאט עד שיפוח.





בן אדם איך תהלך ביד רמה / ולא תכלם מלפני שוכן רומה

מחר את אישון עיניך תאכל הרמה / ויצרך הרע ירדף אחריך עד חרמה

ואחריו תבוא אליך הארץ הנשמה / ובעפרה תדאיב מגופך את הנשמה

תשפיל מעליך את גבה הקומה / ולא תשאיר לך שארית ותקומה

היא ושתי כלותיה עמה / ותשחיר פניך לנגד עמה

אכן הסר מראשך את מצנפת האדמה / ואל תתוכח בעזות פניך עם האדמה

אשר ממנה לקחת / ולבגדיך בבשמיה הרקחת

את כפיך עליה שטחת / וקורות טירותיך בעפרה הטחת

ברחובותיה ימין ושמאל פרצת / ואת בנין גדריה פרצת

מחמר עפרה קרצת / ועיניך עליה קרצת

תחת צל אילנותיה רבצת / ולחצרותיך במי נהרותיה רבצת

את יערותיה בקרדמיך קצצת / ואת אבניה במקבותיך רצצת

על במותיה נעליך חלצת / ואת אבני יסודותיה חלצת

בפעמי רגליך עליה אצת / ואת כל פירותיה נאצת

פיך לרע לנגדה פצת / וכל צמר גפנה נפצת

על גבעותיה כצבי קפצת / ולשכן בתוך ארמנותיה חפצת

וכראותך חסרנך ככלב חרצת / כי משפטיך אתה חרצת

בצל חומותיה הסתופפת / ולתבואתה נכסף נכספתה

לאכל מפריה ולשבע מטובה / ולהתעלס בתוך עליותיה המחטבה

מצמר גפנה תעשה לגופך תלבשת / ומעשביה תקח לחלי מכותך תחבשת

כל ימי חייך על במותיה תכבש / והיא תוציא לך לחם לאכל ובגד ללבש

תתעורר משנתך בתחלת האשמורה / לחרש שדותיה ולעבדה ולשמרה

לאכל מדגן תירושה ועבורה / ולמען ייטב לך בעבורה

תרוץ כצבי בקרב חללה / ותחיה נפשך היקרה בגללה

בפרותיה הנעימים תצהלך / ובחומותיה מהאויבים תצילך

כל ימיך תדרך עליה במנעליך / והיא בחייך ובמותך תשמר עליך

לילה ויומם תרשיע בתוכה בכל מפניך / ואיך באת להוכיח אותה בעזות פניך

מגנותיה תאכל שומיך ולופניך / וכשפחה תשים לחמה לפניך.





ויהי כשמע האדם את דברי הצירה הזאת / עמד בעיניו קרן חזות / לבבו נמלא חמה ורגזות / וישב לו בערי הפרזות / ויקם בחמתו ויכתב את המגילה / ואת כל מצפוניו אליה גלה / וישם שק במתניו וילך בגולה / ויצעק בחרונו ויען ויאמר לה:





עניה סוערה למה את מתנכרה / וקולך מר כקול יולדה במבכירה

לבבך צר כלב אשה מצרה / ונפשי היקרה מחרונך תחיל בצירה

איך נהפכת עלי כארץ גזרה / ואת איברי תשימי בחרבותיך גזורה

נלאיתי מקחת ליך תשורה / ואת תשקיני מים המרים במשורה

בסעיפך תתכסי ותתעלפי / וכיולדה בחבליך תתעלפי

טירתך נשגבה עומדת לתלפיות / ואבניך על ראשי חופיות

לשונך חד כחרב פיפיות / ועפרך יבלה כל הפיות

שאונך יעלה כשאון הימים / ועברתך צפונה לאחרית הימים

קולך יהמה כקול ההמון / ולנצח אין בך אמון

מעפרות גבעותיך יצא פז החרוץ / וראשי הבריות תכרתי במורג חרוץ

כל הנבראים בתוכך קבורים / ובחרונך תכלי הנבונים והבורים

אהליהם על במותיך תקועים / ונבלתם בתוך עפרך שקועים

בחמתך תכלי כל שר ומלך / ולא תחוסי על עשיר והלך

בעפרך יכהו העינים המאירות / ולא יוכלו לראות המאורות

מעבודתך הקשה כל נברא יתחולל / ובתוך חרבותיך ישמע נאקת חלל

הנפשות היקרות בקרבך לחוצות / ולא יוכלו לצאת לחוצות

חרבות חומותיך דם ירעפו / וכל היצורים מתחלאיך יזעפו

כל הרוחות אליך יובלו / ועל נפשותם העדינות יתאבלו

דלתותיך עליהם ינעלו / ובחלאת עפרך יגעלו

פנותיך על פניהם יקצפו / ובחדרי משכנותיך כנודות יצופו

מחמס עברתך היקרים יסופו / וכל המלכים בחדריך יאספו

תסבי עליהם בציריך / ואין מרפא לציריך

תעמיקי להם את התעלה / ולמכותיך לא ימצא תעלה

תכסי את פניהם בשוליך / ועצמותיהם תבשלי בקדרות תבשיליך

בחמתך תכלי הרכה והענגה / וכל בתולה מקצפך נוגה

הקצינים ביתדותיך תלואים / ולנצח בקרבך כלואים

חצרות משכנותיך חשך ואפלה / וכל קומה בתוכך טפולה

גבעותיך קשים כעשת שן / וכל נברא בעפרך ישן

כל הפגרים אליך נקראות / ולא יוכלו לעלות ולראות

נפשי מסורה בידך כאמה נמכרת / ואת איברי תאכלי כתאנה מבכרת

יסודותיך בנוים עלי כגבעה רמה / ותשורתך תקריבי לפני תולעה ורמה

בקולות מחולותיך עלי תצחקי / ואת עצמותי כעפרך תשחקי

שמאלך תחת לראשי / ולזרים תחלקי תירושי

בעפרות חוצותיך תאבקיני / ובימינך לנצח תחבקיני

על עפעפי עיני תדרכי / חצי קשתותיך בבטני תדרכי

חמתך בקרבי תבער כמחבת מרחשת / ותולעתך על תאר פני רוחשת

מתניך בחמה עלי תחגירי / ואת רגלי בסד תסגירי

את בורותיך לקומתי תחפירי / וברב חבריך אותי תחפירי

על זיו תבניתי תהכירי / ומקנאתך לצורתי לא תכירי

את סוללותיך עלי תצבירי / ואת זרועותי כחמרים תצבירי

כמצנפת על ראשי תצפירי / ובתלאותיך בכל יום תצפירי

בקצפך לעצתי הנכונה תפירי / ובזעם אפך לבגדי תתפירי

שפתותיך ברע עלי תפטירי / ומעבודתך אותי לא תפטירי

במכותיך את בשרי תנקיפי / ואת רגלי באבניך תנקיפי

מאשנבי חרכיך עלי תשקיפי / ובלבניך על ראשי תסקיפי

בחרונך על קומתי תקיפי / ועצמותי כברזל הכבד תקיפי

במניפות סערותיך לרוחי תניפי / ומקרב ביתי לגופי תניפי

את צורתי כקוף תחליפי / ואת רקתי כעפרך תחליפי.





ארץ שמעי תוכחתי ולא תמירי כבודי / מאנחתי שפכתי עליך את מרורת כבדי

ואל תקוצי מדרך על במותיך בכבדי / כי מרב כעסי אוכיחך מתוך כובידי

למתי תתעוררי עלי כגבור מתרונן מיין / ותלאותיך יבואו עלי ולא אדע מנין

את דמי תשקלי עין בעין / ותקחי ממני מחמד כל עין

תהמי עלי כהמות ים ונחל / ומחרונך נפשי היקרה ירעה בבחל

לקראתי תשאגי כשאגת אריה ושחל / וחמתך תבער בקרבי כלהבי אש בגחל

לקומתי תפתחי צוארוניך / ופעמי תסגירי בארוניך

מבשרי תמלאי גרוניך / ועלי תשפכי חרוניך

תרבצי בפוך אבניך / ובחרונך תפרי אביוניך

תמלאי כרסי מתמרות אביוניך / ולא תחמלי על אביוניך

מנגד עיני תכבי ניריך / ואת כבדי תשרפי בנוריך

על קומתי תחרשי ניריך / ועל צוארי תכבידי ניריך

כלי מלחמותיך בדמי תמריקי / ואת גופי ביסורין תמריקי

חרונך עלי תריקי / וחרבך אחרי תריקי

תשקיני דמי בסף / ומאימתך אשב בסף

מחרון אפך אוסף / ועברתך עלי תוסף

תשימיני חרפה לעיני האמות / ותשברי ממני את האמות

לילה ויומם תפילי עלי אימות / ותרדפי אחרי עד אחרי מות

שובי השולמית עמי שובי / ובלבבך און אל תחשבי

נעם אמרותיך אלי תשיבי / שמעי אמרי ולתוכחתי הקשיבי

למתי תעמדי בעזות מרדך / ותשרפי את גופי במרדך

זכרי את עניך ומרודך / והביאי לביתך עניים מרודך

הסירי פעמי מעמק מורדך / ואל תשימיני מאכל לירודך

הוציאי בשרי מתוך ורידך / ושפכי על פני מי ורדך.





ארץ השקיני מים ממקורך / ואערך לפניך את חמדת יקרך

מגנזיך יצא דר וסחרת / ומהרריך יחצבו הבדיל והעפרת

מפאת מצפוניך זהב יאתה / ומאורותיך כל נברא יאתה

על במותיך יתקעו האוהלות / ומשושניך יציץ מור ואהלות

מדגן תירושך ימלאו האסמים / ומגנותיך יקחו עצי בשמים

מאבי נחליך יצא לוז וערמון / ומשלחיך יפרח פלח הרמון

על נהרותיך יצמחו החסים / ובצלי גבעותיך החיות חוסים

מפרותיך ישבעו עשירים ורזים / ומאילנותיך יצמחו כל ארזים

מעשבותיך החולים יעשו לציריהם תחבשת / וכל שונאיך יעטו כלמה ובשת

ליערך כל עובר ושב יהלם / וממחצבך יצא נפך ספיר ויהלם

ממנהגך הטוב לא תמושי / ומתולעתך יצא השש והמשי

לריח שמניך כל הרוחות סרות / ומנעם תבואתך יתעדנו השרים והשרות

מתוך מעינותיך יביעו הבארות והברכות / ועל פרותיך יחולו כל הברכות

הכלל העולה גדלתך בכתם טהור לא תסלה / וכל יציר נברא תחת עפרות אבניך יסלה

אכן אערכה לפניך משפט ופי תוכחות אמלא / ואת נקמתי אקח ממך ברב טענות ומלי.





ארץ ארץ למתי התהלכי במרץ / ובעפרך תשחיתי כל עומד בפרץ

כל הנפשות תכלים בכליון וחרץ / ותולעתך בשניה תאכל כל חרץ

מיום היותך את לא מטהרה / ובטנך מלאה חמס כאשה הרה

תלאותיך יבאו עלי כנשר מהרה / ומעלבוני לא תופע עליך נהרה

למתי אקראך יעלה יפיפיה / ואת חמתך עלי חופיה

לשונך חד כחרב פיפיה / ועיניך לנגדי לרע צופיה

קולך יבא באזני כקול המריה / ותלאותיך כבדים עלי כהר המוריה

אנכי נטעתיך שורק כגפן פוריה / ואיך נהפכת כשפחה נחרפת נכריה

כרסי תמלאי מתורי יוניך / ותשכרי אותי בשמרי ייניך

עצמותי תשרפי כאזוב יוניך / ותמכריני לעבד לבני יוניך

כגנב ארדף אחרי פניניך / ומפחדי אארב בעבר פניך

לרע תשימי לקראתי פוניך / ואיך אוכל לברח מפניך

תרדפי אחרי בכל עוניך / ותחרישי לגופי עמקי מעניך

תרהיבי נפשי בעפעפי עיניך / ותרבי עלי את מעניך

הכלמי מלפני האדון קוניך / ואל תלחציני בתוך קניך

הסירי מעל ראשי קניך / ואל תמסריני ביד קוניך.





לכן שמעי התוכחות אשר ערכתי / ואת נפשי בדמים יקרים ערכתי

בנתיבותיך המקלקלים בגאוני הלכתי / ועתה ביצרי הטוב נמלכתי

את נפשי מחטא חשכתי / ואת יינך לא מסכתי

בחצרות אלדי ישבתי / ויראתו בלבי חשבתי

את כפי מהגזל השבתי / ובאזני לדברי התורה הקשבתי

עם הרשעים לא התערבתי / וללחם חמס לא ערכתי

לבתי החוטאים לא קרבתי / ואת חלבי כקרבן הקרבתי

את לבבי לתשובה הסבתי / כי מתלאותיך זקנתי ושבתי

אכן בלבי תרתי / ודרך אמונה בחרתי

כאריה ממקומי נערתי / וחצני מהגזל נערתי

את המצוות עלי החמרתי / ואת עיני בדמעות החמרתי

על יצרי הרע קבלתי / ותוכחות המוסר עלי קבלתי

לכל אביון לא חבלתי / ובגדי הערמים לא חבלתי

בהגיון שפתי ליוצרי הללתי / ועל אנשי עולה התהוללתי

ראה פניך לא פללתי / ולפני חי העולמים התפללתי

ועתה אל תגלי לקראתי נבלותך / כי קרוב הוא יום בלותך

הנה למעלה ספרתי את גדלתך / הנה באפס תקוה יפלו מגדלותך

התרחקי מנגד עיני פן אשבר את קלתך / כי מפני תאות פרותיך באה עלי קללתך

שאי עיניך וראי דלותך / והסירי מראשי עצי דליותך

אל תעלי עלי במעלותך / כי מחר תפלי ממעלתך

אויה ליך ארץ אוכלת יושביה / וכלביא טורפת את הבריות בניביה

שוד ושבר יתהלך על סביביה / ומחמסה בגדו בה כל אוהביה

אנה מפניך אברח / ומעבודתך הקשה אטרח

ברב תלאותיך את נפשי תזיקי / ויד עני ואביון לא תחזיקי

קול הרוגיך צועקים בחמה / ופגריהם משלכים חוץ לחומה

כל באיך לא ישובון / ואת עפרך בפיהם ישאבון

על ספי חרבותיך ישכבון / ומפני זעם עברתך ידאבון

חורי חומותיך מלאים דביונים / ובכנפיך נמצאו דמי האביונים

בחרונך תשחיתי המסלאים והיקרים / ובתוך חוצותיך כלם מדקרים

התינוקות יבואו אליך בחצינים / ולא תשאירי הפחות והקצינים

כל חכמי לב והיועצים / תחת במותך כיתדות נעוצים

המלכים לעבודת שדותיך נעבדים / ואת עפרך ינשקו כעבדים

הפגרים רוחשים בקרבך כנדלים / ולא תשאירי הקטנים והגדולים

מתלאותיך יצעקו האנשים והנשים / ומקללתך הקדומה המה נענשים

תאר חמדת החתנים והכלות / תחת כפות מנעליך כלות

במקום מבחור ברושם / עפר ישימו בראשם

במקום הדרת יקרם / יבש כחרס מקורם

במקום תפארת כבודם / רמתך תפלח כבדם

במקום מרקחת ריחם / חמתך שותה רוחם

במקום מי הורדים / הנחשים בגופם רודים

במקום נעילת העכסים / ישתו ארס העכסים

במקום שמן משחתם / יפלו בבור שחתם

במקום הפארים והקשורים / בצירי דלתותיך קשורים

במקום הנטיפות והרעלות / ישתו כוס התרעלות

חמדת השביסים והשהרונים / כלואים בתוך הסהרונים

נזמי האף והטבעות / בתוך עפרך טבועות

בתי הנפש והלחשים / הנחשים עליהם מתלחשים

בגדי המטפחות והמחלצות / תחת אבניך מחלצות

לובשי החריטים והמעטפות / בתכריך בוז מעטפות

אנשי התענוג והמנוחה / תכלים ביגון ואנחה

רעותיך אחריהם תדבק / ולשונם לחכם ידבק

כאנק חללים יאנקו / וכוס שמריך יינקו

עפרך לבשרם יטחן / ותולעתך תאכל בטוחן

את קשרי הבריות תשברי ביתדות / ותקחי מהם נקמה עשר ידות

כל הפגרים בתוכך נקברות / ונבלתם תתני לרמה לברות

בעלי הגבורה בקרבך סרוחות / ואליך יבאו כל הרוחות

הנפשות ירבצו עליך כעדרים / ומחדרי טירותם תשימים נעדרים

כל הגמלים וכל החמרים / תצברי פגריהם חמרים חמרים

כל הנבראים אשר המה במשכנותיך דרים / תאסרי אותם בקרבך לעד ולדורי דורים.





אי לך ארץ שמלכך נער / ועל כל הבריות חמתך תער

קולך יחריד כקול האריה השואג ביער / ועפרך לא ישאיר כל נבון ובער

מיום היותך תהו ובהו / בכל חוצות יאמרו הו הו

ידיך בראשי היקרים הויה / ולא תשאירי נפש וגויה

המות בראשך תחבשי / ובגדי נקם תלבשי

פרודותיך בדם תעבשי / והבריות לעבדים תכבשי

סם המות דבש נחילתך / וקוץ ודרדר חמס נחלתך

על המלכים תקונני במחולתך / ותקברי אותם בעפרות מחלתך

רוש פתנים ראשית חלתך / ואין מרפא לצירי מחלתך

בושי והכלמי הארץ הנשמה / כי מה בצע תרדפי אחרי הנשמה

הנה ירעוך קדקד תישי אשימא / ומעלבוני יגדל עליך פשע ואשמה

כלך מדברותיך נגעים וחלחלות / ולא תתופפי לקראתי במחולות

הביטי וראי חסדי נורא עלילות / הלא ברחמיו ישאיר לנו עוללות

שמעי קולי ארץ רחבת ידים / ולא תבואי עלי בעדי עדיים

כי מתגרתך רפו כל ידים / אנשים ונשים עוללים ויונקי שדים

תלאותיך יבואו עלי כגדודים / ומחרון אפך בגדי גדודים

אבניך יפלו על ראשי כגדודים / ורמש אדמתך ישים איברי גדודים

בעבור קצפך ארחיק נדודים / ועפעפי לא ישבעו נדודים

מאימתך כל עצמותי מתנודדים / ולילה ויומם בבכיתם מתנודדים

לביותיך יתהלכו לפני כעדר הקצובה / ופעמי בתוך עפרך כעצים קצובה

להריך מי יוכל לירד לקצבה / וקורותיך אין להם תכלית וקצבה

צירי דלתותיך סגורים עלי מבא / ומפחדך לא אוכל לצאת ולבוא

מי הוא זה אשר מלאו לבו / שישרף את גרני תבואתך בגחלת לבו

את יושבת כמלך במסבו / משרתיו ועבדיו הולכים סביבו

ואני כעני שוכב בקרב המחבוא / וביתי סגור עלי לארכו ולרחבו.





ארץ באי בעפר והטמני בצור / עד מתי עריך לקראתי תצור

חרבך הקשה יאכל הנשאר והנצור / וכל יציר נברא במגלך קצור

נתיבות דרכיך העקלקלות סבוכים כמשוכת סיר / ובטנך מלאה אף וחמה ונפוחה כסיר

אלקי ברב רחמיו את גדלתך יחסיר / ויבלע המות לנצח וחרפת עמו יסיר

ואני כל ימי צבאי אסלדה בציר / לא אנוח ולא אשקט בחריש ובקציר

אשא עיני לפני קוני ובתחנוני אפציר / ישלח רפואה לעמו ויצילם מנזק ומציר

וצאן מרעיתו יקבץ על ידי ציר / ויקרב הגאלה ואריה כבקר יאכל חציר.





ועתה הקשיבי לתוכחתי ולא תגורי / והתהלכי למנוחתך וגורי באשר תגורי

מלאי גרוני מדגן תירושך האגורי / והאירי לפני בשמן זיתך האגורי

השקיני מים מתוך קללך / ולא תקללי אותי בקללך

רוצי לפני כצביה במרוצת קולך / והשמיעיני נא את נעימות קולך

אם מצאתי חן בעיניך / אל תרבי עלי מעניך

הרחיבי לי משכנות מעוניך / והשיביני דבר בלשון מעניך.
