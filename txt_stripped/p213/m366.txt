


	
	אופן
	




אראלים / וחשמלים / לאלפים / ורבואות

נקבצים / ומעריצים / ברב אימות / ונוראות,

להקדישם / יחד שם / אל עושה / נפלאות –

אומרים “קדוש קדוש קדוש יי צבאות!”

ושרפים / אשר עפים, / בעלי שש / כנפים,

ממעל / אל מעל / חוג ארץ / ושמים –

דבריהם / וכנפיהם / כקול המון / כקול מים,

כניב נמרץ / לאל נערץ / בנבואות / נבאות.

בכס נקבע / בראש ארבע / דמות ספיר / הטהורה.

ניב רחשם – / ועל ראשם / כעין קרח / הנורא –

דמות בזק / בקול חזק / מברכים בש- / פה ברורה,

ירוממו / וינעימו / קדשה כ- / אשר תאות.
