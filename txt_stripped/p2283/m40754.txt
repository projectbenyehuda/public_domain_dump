


	
	[תהלה ידבר־פי לשואל צרור כספי]
	



Samuel Archevolti

[נולד בצ’זנה (Cesena) בפלך
רומניה בשנת 1515, חי בבולוניה ובונציה ומשנת 1568 ואילך
בפדובה. נפטר בשנת 1611]





שיר על־חנף המשתבח בחנפותו






ں – – ں – – ­­–



תהלה ידבר־פי / לשואל צרור כספי / להסיר דבר מכשול / ומסגר לאמתחתי



ואקהיל קהלת עם / לדבר בלב נפעם / ואטמין בתוך קרבי / שעפי ומחשבתי



במו־פי יהי שלום / לדלים והוא חלום / ואוי לו למי יצפה / לשבטי ומשענתי



ואפרש מצודתי / לעמי באמרתי / ואסתיר צפון לבב / ולי יד באיש עתי



להרבות תלנתו / לנגדי בספרתו / וחלקו יהי עמי / בייני וגם פתי



וסלעי יהי סלעי / ומה־לי למודעי / ידידי ומכרי / ולבני ובן־בתי



וארבה להראות אות / ומופת ונפלאות / לקים דבר כזב / בארצי ומולדתי



וצדיק בדורותיו / אסלף באמרותיו / ופני פני אסתר / וארשע כמו ושתי



במודים אכרע אקד / בברכי אשו קדקד / ואביט סביבותי / לרואי ומי אתי



אחוה לכל־רעי / קרובי מידעי / כבוד כל־קרואי שם / בביתי ומשפחתי



דבש מן וכל־משמן / בחכי ועד נאמן / עלי כל־מזמותי / יהי לי דבר אשתי



ולי כל־כלי מרמה / ובכלל בכל־ערמה / וכל־מין חלקות פה / אני הוא ואין בלתי.
