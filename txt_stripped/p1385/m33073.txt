


	
	המחברת העשירית: גור הזאב והנער
	




נאם למואל בן-איתיאל: ויהי נראו
הנצנים / והנצו הרמונים / וכל עץ השדה פורח /
והגפנים סמדר נתנו ריח / והפיחו ההדסים /
ופשטו הציצים על פני הפרדסים / כמים לים
מכסים / והגיע עת הולכי נתיבות ועוברי דרכים /
ועת צאת המלכים / ובימים ההם הייתי גר במדינה
ושמה עדעדה / יצא ממנה ללכת אל ארץ המועדה /
כי ראיתי גדל בתוכה החטא והרשע / והחמס קם
למטה-רשע / ונדיביה / בין רגלי נבליה / היו
למשלח ולמרמס / ויכבשום והיו למס / ותמלא
הארץ חמס. / והרשע לצדיק זומם / ואתמול עמי
לאויב יקומם / ואמר לנפשי: כי ראיתי כי הצדיק
והתם חלף והלך / בתי הלא אבקש לך / מנוח אשר
ייטב לך / ואלך מעיר לעיר מתנודד / וממדבר אל
מדבר שומם ובודד / עיף ויגע ומתאנה / הלך ונסע
ממחנה אל מחנה / ומתהלך עד קצוי-אדמה / בתוך
אנשים חולפי חוק ותורות חומסים / ומהם אצים ומהם
נוגשים / ומהם שודדים ומהם שוסים / איש איש
ממלאכתו אשר המה עושים / ולא מצאתי כי אם
משתגע מתהולל / וסער מתחולל / והייתי כנוסע
ממי מריבה / וחונה בתבערה ובמסה / ויהי ריב
ומדון אשא / ואהיה עלי למשא.



ואשא משלי ואמר:

הה כי תלאות מצאו דים / כמעט חמודי קט כעש ימסה

הה יד זמני נגעה בי הה / אוצר כלי חמדה זמן ישסה

בגדי יקר והוד פשטני / ויזל זמן מסך והמכסה

מכה בלי סרה הניאתני / נגע זמן בוגד בעור יפשה

הה עיפה נפשי ללבי צר / כי מנעוריו על זמן נושא

דם האנוש לאל והתחולל / ובטח עלי חסדו והוא יעשה

אל תחטא אל תתנה תפלה / לו כי בצרות הוא לך מחסה.



ואמר: אנוסה לי אל אחת ערי המקלט /
אולי אחישה לי מפלט / ואנצל מטיט היון / מרגשת
פועלי און. / ואלך צמה אל קדש בגליל / כהולך
בחליל / והנה עדת זאבים / לקראתי ולקראת חבר
נצבים / ונריע עליהם ונרדפם עד מעונתם / ומצאנו
גורותם / ביום הלדת אותם. / ויקחו איש גורו /
ויבא אל עירו / ואקח גם אני גורי /
ואביאהו אל קדש בגליל ארץ מגורי. / וכבואי אל
שער העיר / פגשתי איש צעיר / תועה מאביו ומאמו
/ ואשאלהו לשמו / ולא ידע להשיבני על דבר
משאלי / ואת שמו לא הגיד לי / ואקח אותו
ואשאהו בחיק / כאשר ישא האומן את היונק / כי
אמרתי אגדלהו / ודברי חכמה אשמיעהו /
ואלמדהו דעת ודרך תבונות אודיעהו / ואם הכין
לבו לצרף ללבן / אני אהיה לו לאב והוא יהיה לי
לבן. / ויגדל הגור / ויהי בביתי
סגור / והאכלתיו מאכל מגורות אשר משדי אמם
חלוצים ונלוזים / ודי חלב עזים / ויגדל גם הנער
בשכל ובערמה / בדעת ומזמה / ואמר לו כאשר
ראיתיו כאחד החכמים / לכן שמר לראשי אשימך כל
הימים / וכאשר ראיתיו מתהלך בכבוד ובענוה /
חשבתיו צדיק וירא מצוה. / וילך לבי כאשר
ראיתי הודו ותפארתו / לעמת מחברתו / ואוליכהו
אתי / ואשימהו על משמעתי / ואראהו כל בית
נכותי / ולא היה דבר אשר לא הראיתיו באוצרותי. /
ואשים הנער על אשר לי ועל בית האוצר / ואשים
הגור שומר הבית מאויב ומצר. / ויגדל הנער
ויגמל / בלבו חכמה ואשמה / וערמה ומרמה /
ומזמה בזמה / יראה ענוה / ותוכו עלוה / יראה
יראה / ותוכו חטאה / יראה אהבה / ותוכו איבה /
יראה צדק / ותוכו חדק / יראה אמונים / והלאה תאונים
/ יראה אמונה / כי אין בפיהו נכונה / ידבר רכות /
ותחתיהם מכות / ידבר נכונות / ועליו ימצאו
תלנות / ובעת ההיא משתחוה וכורע / ומחשב
להרע / ויהי שותה כמים דמי / גם איש שלומי /
אשר בטחתי בו אוכל לחמי / ואפנק מנער עבדי /
ויהי לצנינים בצדי.



ואשא משלי ואמר:

לקחתיו להיותו לי למנון / כמו גבר אשר עבד מפנק

ושרשיו עלי מימי מנוחות / נטעתימו עדי עלה כיונק

עדי כי אמרו רואיו והגדיל / בעדן או המבשן יונק

אהבתיהו כנפשי וחשבתיו / כאח, לן בין שדי אמי ויונק

וזה היה לצדים, לשיכים / כמו אריה ללבאותיו מחנק.



וילכד במכמרתו אנשי התם והדעה /
כדגים הנאחזים במצודה רעה / וחמס סב בכל
מהלכיו / פח יקוש על כל דרכיו.



ואשא משלי ואמר:

אמרו היש מקשים ומכשולים / כאנוש בלב ולב יהי דובר

ילכד במלכדתו מתי צדק / כלכד ברשת בעלי אבר

כעוף אשר יפול בפח יקוש / כן האנוש יפל בפח גבר

עת תהיה שוקט ובוטח / אתו אזי יחצב לך קבר.



ואמר: כאשר חקרתיו ועקשותיו
ידעתי / לריק יגעתי. / ואקום לבקש על יחסו /
ואמצאהו כפלים על חמסו / כי הם נבלים / וסגלת
סכלים / לא נודע בית אבותם, כי הם בבגדיהם
ושמלתם.



ואשא משלי ואמר:

בני נבל ואם תראם / יפי מראה יפי קומה

ועל ראשם צניפי הוד / ומלבושם לבוש רקמה

הדרם אל יבהלך / ואל תפת בהוד שלמה

הלא קרבם מלא הוות / ומצפוניו כלוב מרמה

כתפוח יפה מראה / ותוכו תמצאה רימה

כאיש נראה נשוי-עוון / והוא מלא עוון אשמה.



ואשאלם על זאת / ויענו עזות /
ויתרפקו מליהם / וירגנו באהליהם / ויתגעשו
וייראו / ויבקשו כתבים המתיחסים ולא נמצאו /
על כן אכנם עד אפגשם / בני נבל גם בני בלי שם.
/ ויהי היום ואצא ממעוני / השדה אל צאני / ואלך
עם חברי / ואוליך הגור אחרי / ובהגיענו אל צאני
ואל צאן חברי ורכושם / והנה הטלאים משחקים
במעדנות נפשם / ושעירים ירקדו שם / וכראות
הגור השעירים והטלאים / המדבר עם הפראים /
ואקום אחריו בחפזה / בדאגה וברגזה / וארדפהו
בלב מורא / כאשר ירדף הקרא / ואקרא אחריו
ואריעה עליו תרועה / לאמר: הישלם תחת טובה רעה /
וישב ראשו ולעשות הרע מתנכל / ויענני לאמר:לא
אוכל לעשות כי אם כפי תולדותי / כי לא טוב אנכי
מאבותי.



ואשא משלי ואמר:

מעודך תרבית נותי את ומי / הגיד לך היום כי את בן זאב

כן איש מגדל בן-בליעל אשר / תוסיף להטיב לו ויוסיף לך כאב.



ואמר לגור ולנער אשר מעולם היו
בביתי וסביבי: / בנים גדלתי ורוממתי והם
פשעו בי. / ומהיום ההוא ידעתי כי הענף ילך
אחרי שרשו ונטעו / והבן אחר עמו וזרעו.



ואשא משלי ואמר:

ואיש מצא שתיל עץ מר / ומצל רענן חרשו

וראהו יפה ענף / ורם ותעג בו נפשו

והרכיבו בתפוח / והשקהו בצוף דבשו

ואולי מתקה טעמו / וימוש מפרי כחשו

והנה העלה שרשו / מרירותיו אלי ראשו

והיה (כ)לענה פריו / ומי רוש מר כמו ראשו
ומריח בתפוח / באפו יעלה באשו

ואך כי אכלו יומת / בעוד יומם ובא שמשו

אזי בנת אשר הכל / יהלך אחרי שרשו

המזולל תשו יקר / והחלל התקדשו

לזאת ישם בליעל / ויפרד עדי עבשו

וידעך אור בני נבל / ולא יגה שביב אשו.



ואתחלחל מאד כי הלכתי עמהם בחשך
/ ואמר: אויה-לי כי גרתי משך / ותקצר נפשי
בעמלם / ועל תתי אהלי בתוך אהלם / ויחר עליהם
אפי / בתתם ספם אל ספי / ואלון על הסכלים
הנבלים / אל הזקנים הנבונים והמשכילים /
ויהיו עלי הלם / ויועצו בתבונתם ושכלם /
למען הרחיקם מעל גבולם. / ויתקבצו כל החכמים
אשר סבל על יסבולו / להוציא את הכינם ולא יכולו /
ויאמרו לי: קצנו בחיינו מפני נבלים זו שדונו /
אהה שדוד נשדונו / ועתה הרחק מהם והנצל מפח
יוקשיך / והמלט על נפשך / וירם האחד קולו /
ויוכיחני במשלו.



וישא משלו ויאמר:

פגש דב וארי וזאב / הלא טוב מפגש נבל

פגש אפעה וצפעוני / הלא יצר לך שם בל

ואם תפגש בליעל / כרגע רוחך חבל

ומתחבר לבן-נבל / פשעיו על שכם סבל

ויום תשמח בחברתו / ושנים נפשך תאבל

שמר פן תהלך אחר / הבל נבל ואל תהבל

אהלל בן-בליעל / כבר קולו כהר עיבל.



וישא [השני] משלו ויאמר:
ואם תראה עדי זהב עלי איש / שאל לשמו ודע זרעו ועמו

ואם הוא בן-נבלים תחשבהו / כעמוני ואל תדרש שלומו

הלא נבל וכל עדיו כמו קש / אשר נדף ולא נודע מקומו

חלום נבל לנבל הנדיבים / ישלם לו כפתרון חלומו

וכל חפצו להדיח לבב כל / גביר אולי יגורהו בחרמו

ואולי ילמדו רע מפעליו / וידביק בם קצת רשעו ומומו

בכו אם תפגשם מאה פעמים / שמו תאר ושם אביו ואמו.



וישא [השלישי] משלו ויאמר:
נבל יהי מומת בנבלתו / אור יחשך עליו ואל יופע

כי מעשי נבל, נבלים הם / הוא כאין פעלו אפע

מזולליך איכה יקר תשאל / [ו]הלא נחשים ילדו צפע.



וישא [הרביעי] משלו ויאמר:
ברח מבן בליעל / לבל ידבק בך חמש

לחשך יהדפך מאור / ביומם מאפל תמש

ואם סוד תפקדה אותו / יגל סוד נאמר אמש

ותבטח את בחברתו / ויקטפך קטף חרמש

ואת מומו תכס אך הוא / יגל מומך לעין שמש

וישם בך קצת מומיו / ובהם יכך חמש

קצות דרכי נבלים אל / ומי יגיד לך חמש!



ויאמר: ערי המקלט
ברחתי / כאלו הרגתי ורצחתי / ולא שלותי
ולא נחתי / ואשוטט בארץ / לבקש איש גדר-גדר
יעמד בפרץ / ולא מצאתי כי אם איש בשני לבבות /
להעיר מריבות / ולחשב מחשבות / ובלב ולב מדבר
אחד אל אחד / וארבעה פנים לאחד / ואבוא אל גוי
נמהר ומר / בארות בארות חמר / ולא נמצא בהם נדיב
/ כי אם נפש מדאיב.



ואשא משלי ואמר:

ספו נדיבי עם ולא נשאר / כי אם משקר באמת כחד

נפקד משחר-טוב ולא נמצא / אהב טהר-לב כי כבר נכחד

אבדה אמונה מבני תבל / הלא כבר או נאלח יחד

כי כהניה במחיר יורו / גם שופטיה לוקחי שחד

אם כרתו אתך ברית תופר / אם שלמו אתך אזי תפחד

אוכלי שאר עני ופושטי עור / כי מזאב ערב לשונם חד

על כן בסוד עם יי ובקהלם / נפשי אל תבא ואל תחד

תרתי בתבל איש אמונים בר / לב לא מצאתיהו ולא אחד.



ויחר לו, כי השתרר כל בליעל ועשק וגנב / ועל כי שב הראש לזנב.



ואשא משלי ואמר:

למקדם זנבות עברו ראש / עדי שב ראש עדי הזנבות

וכמה היתה תבל שלוה / עדי קם דור אשר שניו חרבות

ואנחם נפשי על רב קורותיה / ועל תלאותיה ותנואותיה.



ואשא משלי ואמר:

ובהגיע יגונות עד קצותם / ונמסו לבבות בתלאות

ואם נשבר לבבך בתנואות / בתמם קו שבריך רפואות

ואם תשקה ימי זרע יגונות / הלא תקצר שמחות כתבואות

ודע כי קץ לכל תוגה וצוקה / בכן חכה בכל יום קץ פלאות.



ואכשר גם לבבי, כי תמו ימי הצרות / ועברו ימי עברות.



ואשא משלי ואמר:

יגונות מצאו דים / ולב נמס ברב צרה

בסוף כל ציר יהא מרפא / בתום ליל תהי אורה.



וארא כי התעטף עלי רוחי / וכי כשל
כחי / על כי גלו התם והתבונה / ואבדה האמונה /
ולא נשאר כי אם שודדים שדדו / ובוגדים בגדו /
ואריד בשיחי ואהים / כי אין אמת ואין חסד ואין
דעת אלהים.



ואשא משלי ואמר:

לא אתמהה על רוצחים רצחו / רק אתמהה על בגדים בגדו

נבחר מנצח וכל אנשי מרי / מבוגדים מהאמת שדדו

איה מתי אמן ואיה בעלי / התם ואנה הלכו נדדו

כמה אבקשם יום ליום בבני זמן / אך לא מצאתים כי כבר אבדו

הנה באזנינו דבר אמן ותם / נשמע ולא נודע מקום נועדו

אמרו הרוח ים עליהם נשבה / באו מצלות ים ושם עמדו

אם בסערה העלו, בידי אשר / קנא לשם אל נתנו הפקדו

אמרו התדעו אי נתיבות דרכו / לא בלבב ים באני צעדו

אם תאמרו כי נהגם צר בשבי / איך כנבלים בשבי נעבדו

או נחבאו איה חניכיהם ואי / לומדים אשר משפט ודין למדו

או אם זמן הכם ביום הלחמם / איה שרידיהם אשר שרדו

אי סות אמונתם ואי מלבוש הדר / תמם ואפוד בד אשר אפדו

או אי צניפי מהללם צנפו / אי רביד אשרם אשר רבדו

איה עדי הודם ותפארתם ואי / כל מחמדיהם אשר חמדו

איה חלי חמדם ונזר נעמם / איה רפידת תם אשר רפדו

אי שהרון החן אשר על לויו/תיהם ואי קשר יקר ענדו

איה כלי מראה ואי כר נשקו / בם רפדו שרים והתפרדו

אי קשתות תמם אשר מגעת / יתרם גדודים רגזו חרדו

איה מושבות צדקם ואיך התעתדו / גלים על אשדות אמת יסדו

דרשו עלי ספר אמונה איש ואיש / לא נעדרו מהם ולא פקדו

לולי יראת אל אמרה לא נבראו / או כי לאחר נבראו טרדו

כי אך בני תבל בלי תם נשארו / עד איש באיש מרו וגם מרדו

לא אמצאה כי אם-ברית מפר ועת / אשאל היעמדו אזי ימעדו

אין בם לבד כבד עלי-כבד עדי / מחול ועפרת בדיל יכבדו

עת אחשב כי מעשות רע נחמו / אז על דבר פשע ושוא שקדו

לחם כזבים כל ימותם לחמו / לבם בפת עזות ואון יסעדו

יתמכרו יתנכרו גם יהכרו / לאל ולא בושו ולא יפחדו

כי לעשות דברי נבלה נקבצו / ולהעביר אמן ותם נצמדו

חמס ושוד אקרא לסלף בגדים / כי יהרו קש וחשש ילדו

איכה נבלים רום בכל-יום יעלו / איכה נדיבים עד שאול ירדו

הה כי קריות האמונה שממו / הה מעברות תם נתפשו נלכדו

התפצצו הרי חסדים שחחו / גם שמרי חמות אמת נשמדו

אם אמרה לא אזכרם הן בעדם / לבי ומורשיו כאש יקדו

יה אם לפנים נפרדו אמת [ו]תם / יתקבצו היום ויתאחדו
שא נס אמונתם ודגל זהרם / גם חזקה תרנם ויתעודדו.
