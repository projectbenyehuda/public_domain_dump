


	
	המעשה בבחור ספיר, בשפיר אהובו וברשע הזקן האשמאי
	








נאם למואל בן־איתיאל: היה איש בארץ
לבאים / מצודד בתוכם צבאים / ושם האיש ספיר
מגבעת יסוד תרשיש / ושם אמו תרשיש / ויהי
כל־הימים צפרים צודה / איש ידע ציד איש שדה. /
ויבחר לו הצבאים והצבאות / מבין פריצי החיות /
כה יעשה ספיר וכן משפטו, ויהי לו קשת וכידון
וכל־כלי־נשק / והיה גדול מכל־בני־חשק / וילך
הוא וכל־גדודי חשקו ופשט על הצבי / וישב
ממנו שבי / ויבא צבאים וצבאות צרורים / ממעונות
אריות מהררי נמרים.





וישא משלו ויאמר:

אני צודה ולי תאות צביה / אשר אציל צבי מפני לביא

ואי ציד אשר יצוד כצידי / במרבץ האריות הצביה

אבל עיני צבאים יעזרוני / להכות בם ואין קץ לגויה

אני אפיל ואמית אחרי עין / צבי כי היא כמו קשת רמיה

ואם אחת עין צבי ואז / ישיבהו ארי הוסף שניה.





ויעברו הצבאים והצבאות לפניו /
והנה אחד מהם נשא חן בעיניו / ושמו שפיר ושם
אחותו צביה. / ויאהבהו מאד ויהי סביבו / כי
אהבת נפשו אהבו / כי לא היה כתפארתו /
בצבאים אשר גדלו אתו / וישימהו על־משמעתו /
וישכיבהו בחיקו / ויאכילהו מפתו / ויקח
הצבי בחרמו ויאספהו במכמרתו / ויהי חשקו
הלך וגדל, והיה אם יצא מאתו אנה ואנה יצאה
נפשו / כי קשורה בנפשו / ובלחיו זורח שמשו /
ולילו שער ראשו / והיה כי הציקתהו אהבתו / ותחל
לענותו / ויאמר ״איך חשקי עתה הלאני /
הלא־משחק אני״. / ויענוהו יתדות החשק אשר
בלבו נתקעו / ומסמרות האהב אשר בנפשו נטעו /
ויאמרו ״הלא בתחלה התרפית / עתה צר כחך
והתענית / אם שחקת פעם באהבה / ותשחק בך
גם־היא רבבה״.





וישא משלו ויאמר:

נקלה אהבה מראש / ותוסיף עד אשר תכבד

ואם תכבד בלב חושק / תענהו עדי יאבד

ויש דבר בראש נבזה / והוא באחרית נכבד

הלא לוה אסיר מלוה / וחושק לצבי נעבד.





ואם בא לשעשעו בזהר פניו / נחתו
בלבו עיניו / ויבהלוהו רעיוניו / ויתקבצו
להלחם בו גדודי חשקו וחיליו / וייראו מגשת
אליו / וירא כי לא יועילו כלי נשקו / ויקרא לכל
כלי חשקו / ויועצו אנשי המליצה / ולא מצאו
עצה / ויהי משגיח מחלונות מציץ מהחרכים / והנה
בפניו מימי החן מפכים.



וישא משלו ויאמר:

בלחי צבי מי־חן ומי / יפי בפניו נגרו

ישא מאורי רום וכל / אורי שחקים יחפרו

עיניו אריות ימחצו / כל־יום חילים יגברו

על־לחיו שמש וכל / שושן בלחיו יארו.





ויהי עד־כה ויעמד [לפניו איש]
ממשך וממרט / גר בירכתי אררט / ברביעי
בשבת / ושלשה עשר לחדש שבט / ויהי כאשר
הצבי נבט / מט והתמוטט / קט והתקוטט שם והתשוטט
/ ולא נשקט / ויהלך אט / ושם האיש ברשע /
בן־מישע / וישתחו ועל ברכיו כרע / ולבו נפרע
/ ונפשו קצרה מהשתרע / ויבש חכו ורקו לא בלע
/ ומרט ראשו / וקרע לבושו / ויבהל הנער / ויאמר
לו ״ולמה זה הסער״ / ויהי בשמעו מליו / ותשב
רוחו אליו / ויאמר לו ״בן היה לי / ויברח מאתי
וילך / כתאר בן־מלך / על כן בכיתי לנודו
מזבולו״ / כחש לו / אחר־כן שאל ברשע בשלומו /
ויברכהו ויפתהו / אולי אחריו ימשכהו / וישא
הנער את עינו / וירא את זקנו / ויתמה בהדרה /
כי הפך ללבן שערה. / לבנה כשחר / גזת־צמר צחר
/ ארכה רחבה עמקה / בוקה ומבוקה ומבלקה / נכונה
לעלוקה / מכבסת בנתר וברית / מדורתה אש וגפרית
/ בראותו דבר הזקן / חשב בו הכל מתקן / ויאמר:
אין זה כי אם שופט הראוי לדינות / וכרם וזקן חצי
הרבנות / ויחשב כי הוא דין ומהיר צדק / והוא
כמשכת חדק / ויאמר: כבד אכבדנו / ומשירי
אהודנו.





וישא משלו ויאמר:

זקן זקנו זקנה בתם / מימי תבונה האריכוה

שלחו קציריה כבוד חכמה / על־נהרות צדק משכוה

אם־תחשב יום לעשות און / רחבה וארכה הם חשכוה.





ויאמר לו הנער:״הנה נפשי דבקה בך
ותרצך / הגידה נא לי אי־מזה עם אתה איה ארצך״? /
ויענהו לאמר: ״גר הייתי בארץ אררט / ותשובתי
לסדם למשפט / ומעיר ברשע בן־מישע / השופט סדם
עד־לשע / ושם בשליש / דרך לא עבר בה איש״ /
ויאמר לו:״אם אתה מסדם / מי הביאך הלם״? / וישאג
שאגה / ברגזה ובדאגה / ויאמר: ״הלא נתיבותי
דרכתי / וארחות עקלקלות הלכתי / ובאתי יערות /
ועברתי נהרות / וירדתי בקעות / ושפכתי רב
דמעות / וירדתי ימים על ימים ובאתי ציות / שמות
ושאיות שמה נתקבצו דיות / ולא מצאתי מנוחה /
בארץ ערבה ושוחה / והלכתי במדבר שממה / ברעב
ובצמא / ואין כל ביערים / כי אם הר הפרים / ובאתי
עיים ואיים / ואיי כתיים / וכמעט נגזרתי מארץ
החיים / מעם אל עם / נחרף ונזעם / ובאתי אל ארץ
פלשתים ותרשיש וכתים / אשר שם ארם נחתים /
חוילה ודקלה / ותרשיש ורבלה / ואילון ויתלה /
מדי ויון / ודישון ועלון / ועכבור וזעון /
וצבעון ודישן / וחמדן ואשבן וענה ועקן / ושם
רביתי זה הזקן / הדורם ויקטן / ושובל ולוטן /
והוא אחי הקטן / ודודנים וצידונים / וכפתורים
ומצרים / והערקי והסיני וארודי וערבים /
ואלמודד וארפכשד אבימלך וכתיים / ובאתי עד
גרר ועד חול תגרמה / ופלשתים סדומה ועמרה עד
גבול אדמה / ולצבאים דרכתי כל־אלה הדרכים /
והגויים והמלכים / והלכתי חשכים / ורעיוני
נבוכים / עד אשר זקנתי ושבתי / ומדרכי לא
שבתי / ושאלתי מעיר אל עיר / גדול וצעיר / וכה
אמרתי בלכתי: הראיתם צבי־תאר יפה־עין / ויאמרו:
אין! / ועתה אני הולך במר נפשי ובחם לבבי /
ואביט והנך סביבי״ / ויען צבי ויאמר ״למה
תדבר עוד דבריך / השמיעני מטוב שיריך.״





וישא משלו ויאמר:

בחי אהב ראיתם דוד / בלבות אוהבים יארח

מאור חמה בפניו שם / ושם יערב ושם יזרח

וחשקו שם בלב כל איש / ביומו צץ וגם פרח

ורבו אהביו מחול / וחשקי יערב יברח

והוא נקל עלי לבי / ונפשי היתה טרח



בכל נתיב אבקש דוד / בכל־דרך בכל־ארח



נתיבות לא עברם איש / לחרב יום ולקרח

אסובב ים ולדרום / ולצפון ולמזרח



אני חושק עברו יין / ידידות ״דוד״ במר אצרח



ואז פנה צבי החן / ואז הלך ואז אזרח.





ויאמר לו: ״הטיבות / מלים נכונות



השיבות״ / ויהי אחרי כן וייקץ ספיר וירדם /



ויקרא לעיר אין שם איש וקול אדם / ויחרד חרדה /



ויקם בסערה וברעדה / ויחפש ולא מצא / ויפל



מלא־קומתו ארצה / ויקם לגרד העפר / וישב על
האפר / ויתגלל ויתעטף / ויהם לבו



ויתעלף.





וישא משלו ויאמר:



חמי חמי / אשי אשי / חילי חילי / ומאור שמשי



דודי דודי / הודי הודי / בשמי צופי / יערי דבשי



הוא שעשועי / מכל רעי / דברו ייני / פיו תירושי



אך מאוכל / יצא מאכל / כי לקח / מחדרי נפשי.





ויהי לארץ דובק / והאבנים חובק / וקורא
וצועק / כעגל מרבק / ולבו בוקק / נעור ורק /
ושניו חורק / ועיניו בורק / ושפתיו פושק / והוא
יורק / ובלשונו לוקק / ויאמרו: שרי העפרים / אשר
בנפשו קשורים / וידברו על לבו / ויקימוהו מן
העפר אשר שם מסבו / ויהי כאשר הציקוהו
בדבריהם / ויפתוהו בנכליהם / וינס מפניהם /
וירחק מעליהם / ויברח אל ההרים ואל הגאיות /
וילך אל החיות / ותצא אמו ואחיותיו / וצביה
אחות שעיר אל משפחותיו / וכל הצבאים אשר גדלו
אתו וכל־מתי מחשבותיו / ויבקשוהו בכל ההרים /
ובפתחים ובין הגדרים / ובגיא הצבעים / ובין
נקיקי הסלעים / וימצאוהו על־שפת נחלי צבאי
הידידות / והוא בוכה על הפרידות. / וישא עיניו
ויראם / ועיניו כעיני ראם / ולא נשאר בו כי אם
יפה עינים / וזהר הלחיים / ויהי כראותם ברח
לראש הגבעה / מתנקש בדמעה / ויקראו בלט
בטוב מענה / ואין קול ואין ענה. / ויחלו פניו /
ויסב מהם עיניו / ויען אחד מהנערים /
וישביעהו בחי מבחר העפרים / וישמע וינאק /
ולא התאפק.





וישא משלו ויאמר:

אץ כצבי כב להבי / כי אש בלבי נשקה

לא מי נחלים ישטפו / אשי לבד דוד יחשקה

כי מי ירפא מחלה / רפאות מזורה רחקה

רגבי נחלים שוטפים / תחת חבוק דוד אחבקה.





ויפן אנה ואנה / וידברו אליו ולא ענה
/ וירא עדת צבאים / ולהקת פראים / וירץ אליהם /
וילך עמהם, / וישובו גם אלה לאהליהם / ויהיו
כל יום מבקשים לשחרו ולבקשו / אולי יחיו את
נפשו / והיה אם ראם / ינוס מפניהם כי שנאם. /
ותהי לו אחות קטנה / יפה כלבנה / אחות צביה
ושמה אקדח / וביפיה אש בלבבות תקדח / היא
תקרא בשמו / ותשם בסלי בסלע לחמו / ויקרב
הוא והצבאים בלט / ויאכלו ויאכל גם הוא מעט /
והיה בקרב איש יראהו כי ברירו ישחק / וגער בו
ונס ממרחק / ושאג ודאג וזעק כגור אריות נער /
וכעגל מרבק געה וגער / ופיהו לרוח פער / וירד
אל הנחל / ויקרא כקול שחל. / ויהי היום ותצא
אחותו צביה ותקרא בחנוניה / ובטוב שיריה
ורנניה / ויפן אליה / ולא השיב למליה /
ותשביעהו בחי הצבי שפיר, ויפל על האדמה /
ויחשבו כי לא נותרה בו נשמה / ויקרבו אליו
בוכים וסופקים כפים / ויזו על פניו המים /
ויהי כמת עד עלות היום עליו / ותשב רוחו אליו /
וינס מפניהם ויתחלחל / ויבא אל תוך
הנחל.





וישא משלו ויאמר:

הה על־נדוד דוד הה עלי / נפשו לנודו זועפה



כל־יום תצפה אל־מקום / דודה ורוח שאפה



לא ידעה כי חשקה עד־נוד ידיד התנגפה

הנה בתוך נהר והיא / בצבא צבי התעטפה.





ויבוא איש אחד מקרוביו / ויאמר ״אני
ארפא אהביו״ / וימהרו ויבהילוהו / ולשפת הנחל
הובילוהו / וימצאוהו כותב ברירו / וזוכר הצבי
חברו / ויראם ויקם לנוס ויבקש ויקרא לו האיש
״לך [ואראך] את האיש אשר אתה מבקש״ / ויהי
כשמעו מליו / ויחי ויקם על רגליו / וילך הוא
והאיש המרגל / והנה הצבי וברשע על עין רגל /
ויהי כראותו הצבי / ויטרפהו כלביא / ויחבקהו
ויפל על צואריו וישקהו ויבכו / ולחיי ברשע
הכו / ויענוהו וישימוהו במשמר / על כי נפשו
המר / ויען הצבי שפיר מתאונן / מתנפל ומתחנן
/ ויאמר: ״הנני נער קטן / אל יחשב לי אדני עוון /
כי ראיתי זקן ושיבה / ומחליק אמריו כנקבה /
ויקחני בחלקת אמריו / ובתחנון דבריו / הלא
ידעתי כי זקנו חרפות / ויציבני למדחפות / ויתן
לי מזמרת ארצו וכבדני / ואחריו משכני / ואבא
היום אל העין / ואדם אין / עד כי עלינו נחת /
למושיע לי שלחת״ / ויאמר: ״לא אזכיר עוד את
עוונך / אחרי ראותי את פניך.״ / וישמח לבו
בדודו / ויגל כבודו / וירא את העין / והיא רחבת
ידים / ומימיה מתוקים / ישקו בקעות ועמקים / ועל
שפתיה עצי פרי־טעם נטועים / ישקוה לרגעים /
ושלחו ועשו ענפים / וכל עופות בם מצפצפים /
והעין נקיה, כפני צביה / חצציה לבנים / אדמים
גדולים וקטנים / אדמו עצם מפנינים.





וישא משלו ויאמר:

מים עלי חצץ כתרשיש נגרו / ערבו לשותיהם ולמאד מתקו

ידמו למי החן אשר בפני צבי / תראו עלי זהר לחייו ישחקו

עין סביבה עצי פרי אשר / שרשם במים גברו גם עתקו

ידמו אלי קומות צבאות משדי / אמות עדינות ועלמות ינקו

קול תור ומיני עוף ועיט צפצפו / על ראש יניקות זה לזה בם נשקו.

ידמו אלי שרים ושרות נגנו / על ראש עפרים חיק צבאות חבקו.





ויבא ברשע המענה / מדברי ספיר עם
שפיר ויקנא / וימשך לפניו ואחריו / וימסו
אסוריו / ויקם על ספיר הצודה / וינצו שניהם
בשדה / ויאמר ברשע: ״אל נא תהי בינינו מריבה /
הנה־נא העיר הזאת קרבה / ושם שופט אמת / יעיד על
החי כי הרגו המת / הוא יוכיח בינינו / ישת ידו על
שנינו / ושמו אריוך / אשר דם נקי ישפך״ /
ויבואו עד העיר ושמה פתאים /
אשר אין שם מתים / ויבואו עד השופט מטה משפט
גר־יתום / ויספרו לפניו דבריהם וקורותם / וכל
המוצאות אותם / ויחר אפו / ויספק כפו / ויגער
אל ברשע לאמר: ״סור מלפני נקלה ונבזה / משפט
מות לאיש הזה״ / ויקרא בקול גדול: ״הלא על זאת
האדמה תאשם / אשם הוא אשם אשם / על־כן הנער לא
ילך עמכם / ואתם תלכו לדרככם״. / כשמע
ברשע באמרו: ״שמטו הנער שמט״ / ויאמר: ״האחד
בא לגור וישפט שפט / הנה הנער בני ואני
ילדתיו והוא בני יחידי / ואתה תצילנו מידי״ /
וישמע ספיר כי הוא בנער מסתולל / ותצא רוחו
וישתולל / ויהולל ויתמולל / ובלי מדע ימלל /
וכאשה מצרה מגדף ומקלל / ויפל ארצה
ויתהפך / וכמים נשפך / ויקרא לו הצבי /
״אוהבי אוהבי״ / ואין קול ואין – ענה / ואין מביט ואין
פונה. / ויקח הצבי שפיר מלא כף מי אהב / ויז על
פניו וימעט הלהב / ועיניו פקח / וכנף הצבי
ספיר לקח / ויצעק מרב חמו / ויבך הצבי
עמו.





וישא משלו ויאמר:

בני־חשק בני מחלה / והם בלבד בני שבר

בלבות בעלי כנף / בכל יום יעלו אבר

אשר לבי כעוף ידאה / ויתעופף לכל עבר

חלי־אהב חלי כבד /וזולתו כצל עובר.





עד אשר ידע הסער / ומה־יהיה לנער /
ויאמר הצבי שפיר: / ״הראית הזקן ברשע / כי
[זקנו] זקן רשע / וארכה ורחבה / הלא עמל ואון
ברהבה / ומרורת פתנים בקרבה / זה אדני הקנני
/ ובעל הזקן גנבני / והוא רמני״ / ותחמל עין
השופט על־ספיר ועל־אוהבו / ויצו לאשר סביבו /
ויכו לברשע וימחצוהו / ובין הדלתות לחצוהו /
וימת ויסחבוהו / ובאשפתות השליכוהו.





וישא משלו ויאמר:

ומת ברשע בנו מישע / בחטא אדמה וחטא לשע

ברשעתו אשר רשע / ובפשעיו אשר פשע.





וספיר עודנו שוכב בלב־ים אהב /
ובמדורת רהב / ואינו מקשיב ושומע / ולא מקיץ
ויודע / ויקרבו אליו השופט והעפר / ויקראוהו
עד קם מעל האפר.





וישא משלו ויאמר:

וישנתי תבואני דמות דוד / וראיתי תמונתו בנומי

הקיצותי יריבני למען / דמותו אחזה־לי בחלומי



עניתיהו ואם־אחטא אקרב / שעיר חטאת לפניו על־אשמי



הכינותי והקרבתי לחטאת / לפניך סגור לבי ודמי.





ויאמר לו שפיר: / אדני התבשר / וכעס
מלבך הסר״ / ויגד דברי השופט וחמלתו /
ויפל מלא־קומתו / ויאמר לו שפיר: ״שמעתי עליך
כי משפט גר תטה / והדל על־יד רשע תרטה / ולא
עשית עמי כרשעך / ותסיג ברעך / ואותי החיית
/ מטרף בני עלית״.
