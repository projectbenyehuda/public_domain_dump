


	
	ויכוח על תכונותיהן המסתיים בנצחונו של ״איש השיר״
	








נאם למואל בן־איתיאל. באתי אל בית
המדרש / אשר שם כל רז ודבר תבונה פרש / והיה
פתוח לכל דבר חכמה אוגר / האזרח והגר / והחרש
והמסגר / והיו מדברים במבחר אמרים / ובחמדת
דברים / וילקט פנינים / מדברי חכמים ונבונים. /
ואם מעטו ועורם מעליהם / ויקח כל אחד מהם / כל
טוב מלה / והפשיט את העולה / ויאמרו: אי־זה
מהדברים יכשר / שידבר בו איש ואשר. / ויאמר
האחד: מעט מדבר / וענין רב בו מחבר. / ויאמר
האחד: מדבר איש עניו / דבר דבור על אפניו / יש
לשון מלה ערבה מדברת / ותחתיה תעמד הבהרת. /
יש מלה נאמרת ביום פקידה / טובה מכל כלי חמדה
/ וכל המדבר בתאוות לבו / דבר בליעל יצוק בו.
/ מדבר הכסיל יבאיש יביע / והמשכיל יזהירו
כזהר הרקיע / והאויל כי יפתה ודבר דבר / לא
לזרות ולא להבר / והמשכיל לא ידבר דבר, כי אם
בשכלו יפארהו / ובמחוגה יתארהו / מקור שכלות
ועלו / בעלית המקרה אשר לו. / לשון חכם מכל רע
נצורה / ערוכה בכל ושמורה. / בשלשה דברים אמרת
חכם אמורה: / במדה במשקל ובמשורה. / כל
דברי חכמים שקולים כאחד / במאזנים ישאו יחד. /
וידברו בדברי השיר / וביודע ספר ודבריו
במשקל משיר / ויאמרו: חכמת חכם בלי משקל
ומדה ערוכה / היא עוגה בלי הפוכה. / ושם נקרא
[איש] דברים / וירב בשר בשירים / ויאמר: מי
השיר ומי ענינו / הלא שקר בימינו / הנה מיטב
דברי שירה / דבר עשק וסרה / הלא בתי השיר
לאכזב / שוא ודבר כזב. / מגנבי נפשות אנשים
ולבותם / בשקריהם ובפחזותם. / ויחר אף איש
השיר בדברים, ויאמר: האתם תריבון בשירים /
השכונים בבתי החצרים / ואתם נפוצים על ההרים
/ ואין בכם כי אם גולה ותועה / כצאן אשר אין להם
רעה. / וכלכם נפרדים איש מרעהו / איש לבצעו
מקצהו / ואת הנחלות לא חזקתם /
ולנשברת לא חבשתם / ואני לכל שמעם שירותי
ערבות / כעדר הקצובות / והמה שלמות ותמימות /
שכלם מתאימות / ואת הנחלות אחלק ואחלצה /
אוספה הצולעה והנדחה אקבצה / ומלותי אבקר
ואדרש / ולנשבר אחבש.





וישא משלו ויאמר:

ואסיר ממני כל דבר שמיר ושית

לחזק את בדק הבית

וכל דברי במאזני צדק ואיפה שלמה

ואתכם איפת רזון זעומה.





ויען איש דברים ויאמר: שקר אתה דבר
ועל תהו אתה בוטח. הקץ לדברי רוח ומה במשקל
במאזני שיריכם / ואולם אתם טופלי־שקר רפאי אליל
כלכם. / הלא כל דבר שיר חסר תבונות / וחסרון לא
יוכל להמנות / ואני אם אדבר דברי חכמה אכלהו /
מקצה גבול ענין עד קצהו / ומלתי תגיד לך כל
לבה / לא תחסר כל בה / והיא לכל בנחת נשמעת /
ולבב נמהרים יבין לדעת / ירוץ כל פתי קורא במלות
נכחות / ולשון עלגים תמהר לדבר צחות. / ועניני
שיריך רחוקים מלבות אישים / אשא ארחותיהם
עקשים./ והנה אם רחקו בתי המלכים והבליהם /
מי־בעל דברים יגש אליהם?









ויען השיר: במשקל הודו / וקנה
המדה בידו / ויאמר: הלא תדע רע שבריך / למה
תדבר עוד דבריך? / הלא ידעת כי כל הדברים
יגעים / ובעלי השיר נבונים וידועים. / שירתם ערבה
שירת חפש / תאוה נהיה תערב לנפש / כחלי כתם
כל מדבריה / וככלה תעדה כליה. בשיר יגילו
כל יצורים / בחורים גם בתולות / זקנים עם נערים /
כי הוא מחבש אנושים / ומחזק חלושים / המשמח
אלהים ואנשים / ומנער ועד זקן טף ונשים. / השיר
יכתבהו בספריו / כי יתהללו על חבריו / וביום
המשתה הביאני המל[ך] חדריו. / ואחזק לב
המלך ולב חילו אשר סביבו / למלחמה לצאת ולבוא. /
ודברך לא טעם ולא ריח / לא מכאיב ולא משמח /
ואין בו רפואה / לרוח נכאה / ומליך לא לריב ולא
להגעיל. / הבל ואין בם מועיל / לא תנחם אבל לא
תאמץ לבבו / ובית המשתה לא תבוא. / ולמה נפקדו
נפרדו דבריך / ומדוע נפוצו אמריך / הרוח עליהם
יערה / אשר זורה ברחת ובמזרה / לא שקולים ולא
מדודים / כי כלם יחפים וקרועי בגדים / ועת
שומעיהם יפילו תרדמה / וקרעים תלביש נומה. /
כל הוגה בם יהגה אך־נכאים / כמלקט שבלים
בעמק רפאים.



ויען איש דברים ויאמר: רב לך והרף
משירותיך / שמעתי את כל נאצות שירותיך. /
לעגת לי והתלוצצת / וראש דברי מחצת / ואם
באדוניך הרימות יד והנפת / את מי חרפת / ואת מי
גדפת? / ותהלל לך בבתים ובחצרים / בשמחה
ובשירים. / הלא מדברי בנית בתים וחומות וצריחים
/ ומגדלות ומרקחים / ועליות מרוחים. / והשירים
הם צוררי / מגנבי דברי / ומגנבת אמרי / בנית
בית ותחזק בדק / הוי בונה ביתו בלא צדק. /
ומטובו מלותי גבולות תציבו / הוי מגיעי בית
בבית ושדה בשדה יקריבו. / ותחבר לך שירים
כזבים, עיניך בם תעלים / בידיך מתים יחרישו
ותלעג ואין מכלים. / ותעש מדברי אשר הם
מחמדים / בתים לבדים / ותקרב מלת שקר לרעותה
/ אשה אל אחותה. / ואיך תתפאר עלי לאמר: מלים
אמדד בשעלי רוח מלא יבוא לי.



ויחר אף [איש] השיר, ויקם ויקח
דברים מטוב ומרע משמחה ומאבל / וימדד בחבל /
ויאמר: הנה קמתי ואלקט מלים מכל הר וגבעה /
מכל עמק ובקעה / ומכל גאיות וכל פרץ /
וקבצתים מירכתי ארץ. / ולקחתי כל דבר מלה /
כאשר הגה מן המסלה / ועשיתי מהם רנה / היתה
לראש פנה. / ואחפש דברי אהבה ושנאה ובחנתים
כבחן הזהב. / ואקח דברים פזורים אשר היו לי
למשלח רמש ולמרמס־שרץ / ועשיתי אותם גוי אחד
בארץ / ולקטתי מלות מפי כל איש אשר ידבר בם
לכל צרכיו ומעניו / והמתפרצים מפני אדוניו /
ועשיתי מהם שלישים ונסיכים / וסגלות מלכים. /
ואהלל בם סגנים ונגידים / ויהיו על רבידים ועל
ידיהם צמידים / ונזמים על אזנים / וריח מור לאפים
ואתה למה מליך תשבר, הקצור קצרה לשונך /
ופיך כבד והדבר אין בך / ואם יועצך אבד, כי
תעיק לשונך בכל מלה / כאשר תעיק העגלה / למה
בששו אמרי פיך / ונוקשו בכל עת שפתיך / מדוע
אחרו פעמי מרכבותיך? ואות כל לשון חוטב / כנהר
שלום וכנחל שוטף. / ואני כל לשון השר כמוס אגור
/ כתר וסיס ועגור / יתפרקו מלי על לשונם וקלותם
לא יספו / ירוצו ולא יכשלו, ילכו ולא
ייעפו.



ויען איש דברים ויאמר: אמנם השירים
יתהללו ובכזבים יתהללו / וילכו אחרי ההבל
ויהבלו. / ויאמרו: אנחנו משמחים מלכים / ובנו
ישישו נגידים ונסיכים / ואם תדמו, כי דברים דבר
תמים עבר / לא כן הדבר / כי בחרו להם חלקות מלות
/ ויחזו להם מהתלות / ויפתום בפיהם / ויתעום
כזביהם / וישמחו בדברי שקריהם / ברעתם
ישמחו מלך ושרים בכח שיריהם / והנה במבחר
השיר / אשר תשיר / ובמיטב מחברת אשר תחבר /
שקר אתה דבר. / ויקח השיר דבר המלוכה /
ויעשהו פסל מסכה / ומעשק מלי בנה ביתו
ויכינהו / בית גזל ויבנהו / ויבן הבית
ויכלהו. / עשק וגנבה בשיריכם / גזלת העני
בבתיכם / כי אין עיניך ולבך כי אם על מאמרי / עד
אם דברתי דברי וממני שיריך / ונגינות אוהביך
/ מלאכת תפיך ונקביך / תחבר מלותי ותנגנם
בנבלים / ובמנענעים ובצלצלים / ועתה מה לך
ולמחוז חפצי / כי באת אלי להלחם בארצי?



ויקומו שרי החכמים והנבונים /
ונגידי השכל והסגנים / ויגערו במאמרים
והוכח בדברים. ויאמרו: כל המלות / נבזות
ונקלות / והדבר אם נעלו עניניו ואם גבהו / הלא
דבר הוא / ואם יקרו נפרדות / ולמאד יהיו חמודות /
עבשו פרדות / וידמו בהיותם נפוצות / כאבני
גיר מנפצות / לא נכרו בחוצות / ותהיינה למדרך
כף רגלי כסילים / תרמסנה רגל רגלי עמי פעמי
דלים / ומהם הקדש מהחל לא נבדל / לא נכר שוע
לפני דל. / וידמו מקומותם טרם יהיו ברורים
ובמשקל השיר צרופים / חרי עפר וכפים / ולא כן
דבר השיר, כי בתיו מקפים באבני אקדח / לבל
ידח מנהו נדח / ומליו נקרבו אחת אל אחת
ויתאחדו / ויתלכדו ולא יתפרדו / והדברים אם
לא ייעדם השיר לא נצמדו / הילכו שנים יחדו
בלתי אם נועדו.



עד אשר קם השיר ויאמר: כי ממלות



יקרות ונכבדות / אבנה לי בית מדות / ואביאם בכור



מחשבה וחניתי לביתי מצבה. ואביאם במבחר



מאזנים / עד אשר יצא הדבר מהשפתים / מזקק



שבעתים. / ואשיבם על מערכות שירתו / הבכר



כבכרתו / ויקח כל מלה אשר היתה לקלס / ושקל



בפלס.



ויאמרו הדברים לשופטים: למה תשאו
פנים? על שפת שקר תקשיבו / הלא דברי ייטיבו. /
אמרו למהללי השירה ודברי לעו / על־מה אדניה
הטבעו / הלא מאמרותי בחרו נבחר כל אמר / וימדו
בעמר / הלא ידעתם כי בדברים דברו נבונים /
לימים קדמונים / והמה טובים משיר אצל החכמים / כי
זקנים לימים. / ונביאי השיר תמול נבאו / חדשים
מקרוב באו / וקדם לא מצא שירים בשפתותיכם /
אשר לא שערום אבתיכם / והיום אם על השיר נפתה
לבבכם / ראו כי רעה נגד פניכם. / ותבחרו לכם
מאזנים ומשקל / על כן לא יצא משפט מעקל.



ויקש דבר השירים / מדבר איש דברים
/ ויצאו מפני השיר זועפים בחמה / כי על פי
השוטרים היתה שומה / כי יעדו בם השוטרים
ולעבדים לשיר כבשום. / ויבואו הרעים ויגרשום.
/ ויאותו להמליך על כל מלה ודברה / את דברי
השירה. / ויהיו הדברים במשפט עשוקים / כי
הסירו השופטים משפט הצדיקים. / וישמח השיר
על כי גבר שירו וכחשו / ועל צדקו נפשו. / ויגל
כבודו על הגזלה ועל הנאצה / ועל העשק ועל
המרוצה.





וישא משלו ויאמר:



התעוררי נפשי וסוד / מעין [דבר] חכמה גלי



נפשי כאובד קם ומך / מטוב נבונים העלי

שיר איש לשון לבו כפי / שכלו יכשר בכלי

איך שיר נבון־לחש ארי / אזן שר סכך חלי

לא שיר נבונים שיר כסיל / אם תכתשנו בעלי

כי לב נבונים בית יקר / ולבב כסיל בית האלי

השיר פעמים יעלה / ומבור ומשחת בלי

השיר כמו רביד עלי / צואר ועל גרון חלי

בינת נבונים מבלי / שיר נחשבה מר מדלי.
