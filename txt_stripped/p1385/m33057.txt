


	
	המחברת התשיעית – וכימה סהר אהבת
	




נאם למואל בן־איתיאל: ויהי כאשר
הסתיו עבר ותמו ימי חלדו / הגשם חלף והלך לו /
וכי הפך ערף / הקר והחרף / כי הקיץ שפח קדקדם
והפיל אשיותם / ונהפך יסודם / ויסבר על הזקן
בגד / הכר על הגלד / ויחם בשר הילד / והעצים
נתנו פרים / והאניות דוברות בים. / ויהי איש
מבני הנדיבים ואיש ממשפחת מלכים ושמו צלמון
/ מגת רמון / ולו בן אחד ושמו סהר / ולחיו מלבש אור וזהר. / ויברח מאתו / כי
עצבו וגדפו וילך אל ים יפו / וימצא על מבואות
הים אנשים / בורחים מפני רב נשים / וישכר
האניה על כל סוחר וים עובר / והאניה חשבה
להשבר / כי התגעש ורעש הים / ונשאו נהרות
דכים / ויצעקו למשביח שאון ימים להשביח
שאון גליהם. / ויטילו את הכלים אשר באניה אל
הים להקל מעליהם / והעם צועקים ומר צורחים /
ונאנקים ונאנחים. / וייראו המלחים וישפילו
הקריות / אשר באניות / ורב החבל ביריעות /
מפני רוח הנס / בל יחזקו תרנם בל פרשו נס /
וכל איש מהם ידר לתת לעניים כספו / ואולי יעמד
הים מזעפו / וגם הנער בהר ומר צורח / כי מפני
אביו הוא בורח / והאניה רפתה מרוח גדולה וחזק
מפרק הרים / ושברה כשבר נבל יוצרים / ויחזק
סהר בלוח / וישאהו רוח / ותביאהו עד ים הערבה
/ והנה על שפת הים עיר ושמה צובה. / ויהי בצאתו
אל היבשה, / וישא משלו בשירה חדשה:

ומי ראה גבורות אל / ונוראותיו כיורדי־ים

יחילון משאון גליו / ויודו לאשר לו ים

ויחזון נפלאות שדי / ויהמו כהמות הים

ומי יעמד בעת ירעש / ויכיל משברי הים

ומי נסתר ביום אף אל / ולו נחבא בקרקע ים

ולו יסק לרום־שחק / ויציע תהומות ים

ולו ארץ ושמים / ויבשה וים עד־ים.

והוא גוער בזעף ים / ומשביח שאון הים

ונהר יחריב יבש / ואזלו מי תהום מים

והוא בונה עליות רום / והוא קורא למי הים

והוא צוה להרעיש ים / נהרות נשאו דכים

ושבר האניה, כי / כקש נדף ישו עבים

ואנשי האני בוכים / ודמעתם עלי לחים

כמספד על בכור ספדו / ונהו אז נהי כים

ומהם ידרו נדר / ויודו על עוון מרים

לבל־יקשו לבבם עוד / ויעזבו הלך קרים

ובל־ילכו בקרי עוד / ובל ישגו בחטא קשים

ומהם ידרו הענק / ענייהם לפי ענים

עדי רגע בכחו ים / אזי התנצלו עדים

והטילו אלי ים כל / כלי כספם כלי צבים

ובגדי מחמדיהם / לבושי שש לבוש משים



והפילו אזי חובלים / קריותם, כל־כלים

והשחו את יריעותם / וכל ערבם וכל שתים

ולא הועיל כלי נשקם / וחץ קשתם וכל־כלים

גבורה נשתה מהם / ורופאים שכחו צרים

עדי כי מעדה רגלם / ולא עמדו עדי דחים

כאבן ירדו טבעו / ולו אדיר יהי עים.

כאין כל־הדרם שב / ושוא הודם וכל־יפים

וחיות ים קטנות עם / גדולות אכלו דים

וקבריהם מעי דגים / לשמה נהגו שבים

בלב־ים יצאה רוחם / ושמה שלמו נשים

וכלם אבדו רגע / ורואים יאמרו אים

ארומם אל אשר קדם / בבטן כוננם חים

אשר היום גאלני / בחסדו משכן עים

מברך הוא לעולמי־עד / ועד־נצח שמו קים.



ויהי הוא מדבר את השירה / ותשקף
אליו נערה מהבירה / ותרא יפיו ותתמה / ותרץ
ותגד לאמה / ותבא האם ותראהו ותגד
לחברתה / ותרץ גם היא ותגד לגבירתה / ותבאנה
כל נערות הבירה לראותו נועדו / אשה רעותה לא
פקדו / ותתיצבנה לחזות הדר יפיו ולשור / בנות
צעדה עלי שור / ויראוהו והנה פניו פני אריה /
לפניו לא קם כמהו ואחריו לא יהיה / ושני אריות
שניו / וכי קרן אור פניו / ושפתיו בהיות מדבר /
נוטפות מור עובר. / ויהי ככלות הנער שירתו /
להודות על־תשועתו / וישא עיניו לראות הבירה /
וישמע בקרבה קול המון שליו בא / ויאמר: ״מה קול
ההמון בתוך הארמון״ / וירא והנה דלתות הבירה
צללו / וכראותם יפי סהר נבהלו / ולבורא הדרתו
הללו / וכשמעו קול המלה / ויאמר: ״אין זה כי אם
בית התפלה״. / וירא והנה פנים נוצצים / ועינים
מתלוצצים / ושם שני בני כושים חלוצים / וימררהו
ורבו וישטמהו בעלי חצים / ויטמנו לו רשת /
וימצאוהו המורים בקשת / ויאמר להם בראותו כי
בשחתם נתפש: ״תמנו חפש מחפש״ / כי אמר: ״איך
מים נחלצתי / ועתה בבית נמחצתי / ואיך לבי
ונפשי יבריחם בן־קשת / והיו לדם ביבשת /
ואיך יריבון עינים / באיש יגע ורפה ידים.״



וישא משלו ויאמר:

הה כעמלק נחשבו עינים / ללחם ביגע ידים

הנה עמלק מלפנים נקבצו / וילחמו את יוצאי מצרים

כן בעלי חצים לחצוני ביום / הנצלי מעמקי המים

אנא עלי ימין נטו או על שמאל / אל תשפכו דם בר נקי כפים

חץ קשתותיכם גדוד ימית וכל / גבור ולו יהיה כאיש בינים

לב רואכם אמנם בחכה תעלו / רגע והמזלג שלש שנים

אלו יהי הלב כלב אריה ימס / צירים אחזוהו ופיק ברכים.







שמעו שירו ותבהלנה / ולא ידעו על מה
תבטנה / אם לשירו / אם להדרו / אם לטוב מעניו / אם
לזהר פניו / והנה תפוח עבר וחלף / במור ובנרדים
מעלף / וישלך אליו בעד החומה / מאת צביה ושמה
כימה / ברה כחמה / ודבר אהבה בו מחקק /
שבעתים מזקק / כי שמעה שירתו / ונעם
זמירתו.



ותשא משלה / אף־היא אמריה תשיב לה:
שלום לך שלום הרוג עינים / מכואבה משתחוה אפים

חולת אהביך הרוגת חשקך / לקחה כפי כל־חשקך כפלים

מי יתנני יום וליל עם צבי / בשחוק חבק אעניקך שדים.



וישא עיניו ויראה / ויבהל מהדר
מראה / ואל יפי פניה / ותשתחו לו ותשק לו
על־ידה / כמשפט כל עפרה אל־ידידה / ותשקהו
והוא נשקה מרחוק / כמשפט
ידידים הנפרדים כחק / והנה סריס המלך גער בה
ובשאר הנערות / ותשארנה דלתות הבירה לפניו
סגורות / ונשארו בצלעותיו להבות וכידודים /
בוערים כמראה הלפידים / ויהי כפנותה לדרכה /
ותתחלחל המלכה / ולבו בקיר לבה נחבא /
וימת לבה בקרבה. / וילך אל דרך המדינה /
בלא דעת ותבונה / ויאמר לנער אחד צעיר: / ״הראני
נא את מבוא העיר״! / ויבא וישב לו ברחוב העיר בוכה
/ ודבר מהנערה חוכה. / ויהי לעת ערב / כקדח לב
החושק והקרב / ויגדל כאבו / ויאנח לבבו /
וישא את קולו בבכי ותגדל צעקתו / והנה אשה
לקראתו / ותגער בו ותאמר ״הצירים אחזוך / אם
בכימה חשקת / מה לך כי נזעקת״? / ויהי
כשמעו את שם כימה חשוקתו / ותצא נשמתו / ויך
לב הנער אותו / ויאמר: ״מי את גברתי ומי כימה?״ /
ותענהו לאמר: ״הלא כימה / שם אהובתך הנעימה /
בעלת התפוח / אשר בשמן המור משוח״ / וכאשר
הגידה לו את דבר הנערה אשר חשק וחשקתהו /
לבבו פלצות בעתתהו / ותאמר לו: ״כי חשקת
כימה / מי יעלה לך השמימה / הלא יבא בלבך מרך /
כי ירחק ממך הדרך / הלא היא כימה אשר ראית
בבירה / היא בת מלך והיא הגבירה / הלא כל אוהב
כימה / בחצים ובקשת יבא שמה״. / ויהי כשמעו
מליה ויפל כמת ארצה / כי יצאה רוחו ונפשו
קצה. / ותדבר האשה על לבו /
אולי תניחהו מרגזו ומעצבו / ויאנק ויאנח /
ויאמר: ״נפשי משלום תזנח״.



וישא משלו ויאמר:

ראיתיה וראתני / חשקתיה חשקתני

חקקתיה עלי לבי / ועל לבה חקקתני

ועל־יד מלאכי דדים / נשקתיה נשקתני

צרפתני בכור־אהב / ועל־לבה יצקתני

ובגדודה הביאתני / ושמה החזיקתני

ואיך עתה גזלתני / וקדם העניקתני

תמול תגער בעושק דל / ושם היום עשקתני

הניאתני והלאתני / מחצתני מחקתני

ויום מתי. דעו־נא כי / פרידתה המיתתני.





והאשה עשתה בעקבה לראות אהבתו /
ותבא לנסותו / כי אמרה כימה אשלחה ואנסהו /
מה־יהיה משפט הנער ומעשהו / כאשר ראתה
האשה חשקו בה / ותתן לו כתבה / ויפרשהו
והנה ריחי בשמיה הפיחו / ומלי אהבה יריעו אף
יצריחו.



ותשא משלה בכתבה / ותלן על סער אהובה:

די צבי די – הלעשק / יעלת־חן? – לאט לאט

איך תענה לבב צב/יה ונמס מעט מעט

הה המועיל בדכאו? / רפאו או שחט שחט

לב ישוע במר נדוד / אזנך הט והט והט

ויהיה רץ ורפא / כי כבר מת וקט וקט

או התשמח בשאטך / אל אסיר דוד שאט שאט

בעבר דוד חשקו עלי / לבך שט ומט ונט.



ויהי כקרא העפר / את דברי הספר /
וכראותו כי מצא חן בעיניה / וכי הפיק רצוניה /
ויעמד לבו מזעפו / ותאמר לו האשה: ״העבר
דאגה מלבך / והסר כעס מקרבך״ / ויאמר לה:
״החיתיני כי חסד גמולך / וחיתה נפשי בגללך /
והנה שנאתי את החיים ובחיי אקוצה / משכני
אחריך נרוצה״. / ותוליכהו אל הבירה / אשר שם
כימה הגבירה / ותושיבהו תוך החצר / ונשאר
כאיש אין כח עוצר / וכעמיר אחרי הקוצר / ולבו
בוער בחם אהבתו / בתוך הארמון / כדוד נפוח
ואגמון.



וישא משלו ויאמר:

לב כל אנוש דם ובשר / לב חושקים מרשפים

לבות יעופון כנשר / כי כן רשפים כעפים

כל יד תמשש לבבם / תצא כעוגת רצפים.



ויהי כמיחל ויצרו יכבש / ויחיל עד
בוש / והנה באו נערות שתים / כאור השמש בחצי
השמים / וילך עם הנערות / אל מבית לשדרות /
וישכיבוהו במטתו / כל הלילה עד למחרתו / ויהי
כזרח השמש / והנה נערות שתים יפות מנערות
אמש / והריחו בגדיהן כבואן אל הארמון / מר אהלים
וקנמון / ותלעגנה לו להוסיף על חילו / ותאמרנה
לו: / ״איך הנה הובאת / הבכסלתך באת / ואם באהבה
בקשת הלא ידעת / כי הבא אל חדר הנערות נכרת
ונצמת / והבא אל השדרות יומת״. / ויען סהר לאמר:
״הנה הסכלתי אם בחשקי אם בשכלי / והנה אם רע
בעיניכן אשובה לי״.



וישא משלו ויאמר:

מנעוני מתוקי דוד / הניאוני מרורי נוד

ויאמרו כי חשק דוד חטא / והחוטא אשר יבגד

ודוד בוגד בדוד כבגד / בצור מי שלכל יסגד.



ותשתחוינה לו ותפלנה לפני רגליו /
כשמען יפי מליו / ותאמר האחת: ״נראתה / כי כמעט
רוחו נכרתה / וכמעט נפשו נשמדה / באה אתי
הביתה וסעדה״ / ותאמר האחת: ״בא תבא עמי /
אל
החצר הפנימי / כי הנערה הנצבת
עמך / בזה לך לעגה לך״. / והנה ארבע נערות
כצבאות / מהחדר יוצאות / ובצאתן נמלא הבית
מזהרן / ויאמר בראותו רב מאורן / וכי זרח
על־כל־פנה ועל כל פרץ: / ״השמש יצא על הארץ״. /
ויקם וישתחו אליהן / וישתחו על מדרך
רגליהן / ויאמר: ״מי מכם גברתי כימה / בעלת
השירה הנעימה / החקוקה בתפוח המשלך בעד
החומה״? / ותאמרנה לו:



״שמע צבי הנאה / שא נא עיניך וראה״: /
וישא עיניו אל החדר / והנו נסתר בפרכת יקר
ואדר / וכתוב בה ״קרא את השירה / בטרם תבא
החדרה״ / וכתבה בה שיר אהבה / ותגד לו את כל
לבה:



ידידי אחרי רעיה / אני על לחיה מסוה

ואור תלוה לכל מאור / ושמש זהרה לוה

לכה איעצך דודי / בשכל בא והשתחוה

והכנע אלי יפיה / ומבחר שיר תהחוה

רצונה אז ידיד תפיק / אזי תכון ותלוה

תשיחך תניחך / ודדי רעיה תרוה

ואם תענה במליך / ״אבל אין לידיד שוה״

תגרש אז ותמאס / ונשאר לבך דוה

ויוקד בך יקוד־אהב / עדי תחם ותדוה

ואם תיטב דבריך / ברעיה יש לך מקוה.



ויבא והנה נערות מושכות הדרתן / ולא
נודע התרשיש גויתן / ואם ספיר גזרתן / והנה
נערה אחרת באה מגנת הביתן / ויאמר להן: ״אי
הנערה אשר תורה בחצים / אשר בי חציה ילכו
שבים ורצים״. / ותען הנערה אשר מגנת הביתן
באה: / ״הנה החצים ממך והלאה.״ / ויענו לו לבו
ונפשו מחצים מחוצים: / ״רץ מצא נא את החצים״. /
ויגער בם לאמר: ״איך תשלחוני אל הורגי /
ותשימוני ביד מוגי!״ / ויאמר כאשר ראה כי
יבקשו המיתו: / ״אויבי איש אנשי ביתו״.



וישא משלו ויאמר:

נפשי ולבי יקומון / כיום ממיתם לאורב

מותת בחץ עין צביה / מה־טוב ומתוק וערב

תורה בחצים שנונים / באים עמק לב וקרב

המה לשלום ואך היא / תלחם בכל יום ותרב

כי אהבתה בכל־עת / תגדל ותפרה ותרב

תכון בעצה להרגם / קשת וכידון וחרב

תאות לבבות סביבות / בבות שחורות כעורב

לרחיק ולנוד רצונה / ורצון שניהם לקרב

תשחק והמה יחילון / בקר וצהר וערב

על־לב יצורים יהי טל / על־חושקים חם וחרב.







ויאמר אליהן: ״אן כימה אחותי / שאלתי
ובקשתי״ / ותאמרנה לו: ״עד מתי תהלך נפשך
למוקש / גדלות אל תבקש / ואיך אתה בוטח על־תהו
ועל בלימה / התקשר מעדנות כימה״? / ויען
ויאמר: ״עד מתי תהתלו בי / להכאיב את לבי / עד
אשר ישבר ויהרס הרס / ולא ימצא במכתתו
חרש״.



וישא משלו ויאמר:

לבב החושקים נשבר / מכתת ככתות חרס

וישבר כמו נבל / בנוד דוד נהרס הרס

ויזרה עדי כי כל / יצוריו צל בתוך ערס

ולמצא חן בעין עפרה / יפלל צור בכף פורס

ולא ימצא לבד מריב / לחי מכה ושן יגרס

יריבוהו עלי חשקו / ברעיה תעשק חרס

ורב חושק ברב גרב / יהי מכה ובחרס

ויהיו שואלים לחם / עלי דלת ואין פורס

ויכנעו מריבי דוד / ומקלה אוהבים קורס.



ותאמרנה לו: אל יחר בעיניך / ואל־תעל
חמתך וחרונך / הלא אנחנו באות לנסותך / היש
[את] לבבך ישר כאשר לבבה עם לבבך״? /
ויודיעוהו כי בתי הארמון והבירות / חצרות קטרות /
והראוהו מבא הבירה / ולפניו פרכת כתובה בה זאת
השירה:



טרם ידידי בואך לחדר / שא עינך וקרא כתב פרכת

בא בענוה אל דביריה אזי / יתן בבית עפרה לך מהלכת

ובחר קצר אמר ערב ענין ואז / נפשך לבב רעיה תהי מושכת

תשמח בעפרה אז ותנעם נפשך / ובכל־אשר תתאב תהי מולכת

תשב במעדנים ומריח במר / זכרה בפה נושק ויד מועכת.



ויבא והנה שמנה נערות / באורן
השמש מחפירות / וכזהר הרקיע מזהירות. / ויאמר:
״השבעתי אתכם המאורות אשר לקראתי יצאתן / אם
את שאהבה נפשי ראיתן״. / ותענינה לאמר: / ״הן
משאלך נפלא ממנו / עמק עמק מי ימצאנו / איך
כימה תבקש בארץ הלא תרגז ולא תחרד / מי עלה
שמים וירד״.



וישא משלו ויאמר:

כימת אדמה תור בעינה חץ / כי לא לכימת רום ירות חצים

כימת אדמה היא תענה דוד / תגש בעינים מאד אצים

כימת אדמה תראה לחיה / כשני אנשים לוחצים נצים

כימת אדמה תך בעינים / לשפך דמי נקי ותם רוצים

כימת אדמה אש נדוד תדליק / בלב ותוקד בו בלי עצים

אין קודחי אש הנדוד זולת / כימה הלא תדעך כאש קוצים

כימת אדמה חומסה חושקים / כי לא כסיל כימה ועש חומצים.



ויצעק ויאמר: ״רחמוני כל מריבי /
ונחמוני כל אוהבי / וחמלו על לב נגזל אשר עשוק /
והצילו גזול מיד עשוק״. / ותאמרנה לו: ״התבשר
וגיל ושמח / כי ישועתך עתה תצמח / הלא כימה
הגבירה הולכת / אל־מבית לפרכת״. / וימהר לבא
ותאמרנה לו: שא נא עיניך וקרא זאת השירה / ואחר
תבוא אל הגבירה״.



ידיד בא תחזה עפרה / ואך אל סוד היה נאמן

שפתיה כחוט שני / רק פיה כצוף ומן

בשרה עב ושוקיה / חלאים מעשה אמן

ולו שת איש ימינו בין / עבות בטנה יהי נטמן

זרעך לחנקה צו / שפתיך לנשקה מן

וכל איש יחשק לו בה / כמו איש יעבד חמן

ומי יוכל ספר הודה / היספר ואם ימן

אשר תציל מאור שמש / ומאר זהרו אלמן

ולה עין לבב חושקים / כשודד לה כשד שלמן

הרוג חצה בלי יחי / ולו רופאו יהי הימן

ולה לב עז כצור חזק / ומי יתן יהי רחמן

וכן כל־דוד וכל־חושק / כבד־אזן ולב השמן

ואם תועידך מועד / ימותיך תכל בזמן

ודע כי זאת גבולת־חן / ושורת־הוד בשיר נסמן.



ויבא והנה עדת נערות כצבאות / מאת
הגבירה יוצאות / ולקראת סהר באות / מחפירות
השמש בזהר פניהם ומבישות החמה ונוצצים על
הלחיים / פרח ונרות ומלקחים / וכך היו קוראות
ומדברות / ובתחנוני אמרים אמרות: / ״לך אהוב כימה
/ ועמך חשוק חמה.״ / ויען ויאמר: ״אם ישכן איה
כימה גברתי, הלא ראיתן כי רוחי בחשקה חבל /
וכשל כח הסבל״. / כראותן כי רוחו קצרה קצר /
ותבא נפשו במצור / ותאמרנה לו ״קרב אל הפרכת /
אשר אחריה כימה הולכת / וקרא המכתב הזה״. /
ויקרא והנה כתוב במגלת ספר: / ״קרא נא אשכל
הכפר״. / וירא בכתבה / כי הגידה לו את כל
לבה:



אני סתר עלי רעיה / כהסתיר עב מאור חמה

בבואך העלני דוד / ותמצא אחרי כימה

ולא כימה אחות עיש / אשר עם כוכבי רומה

אבל כימה יפת־עין / יפת מראה יפת קומה

בכימת האדמה לא / ידידי תלכו רומה.



ויבא אל הארמון אשר שם כימה הגבירה
/ ואל סהר היתה אורה / והנה לקראתו נערות קהלות /
לחול במחולות / ותערך כל הקהלה / ותהללנה
אותו בשירי תהלה / ובידיהן כלי שיר מכל־מינים /
וכלים מכלים שונים / ובהגיע כל הקהלה אליו /
ותשתחו לרגליו / וליפיו לבם יסגד / וילך
כמלך בגדוד / וכה אמר בלכתו / בזכרו כימה
גברתו: / ״מי יתן ואראה רעיתי כימה / ויעבר עלי
מה / הלא בעדה נתכה כמים שאגתי. / מי יתן תבוא
שאלתי / כי נשארתי באין לב ואין דעת״. / וכימה
שומעת / ותענהו במענה כל־כאב יחבש / מה־מתוק
מדבש / ודברים ערוכים בטוב טעם / צוף דבש אמרי
נעם: / ״הלא לבי נד כפי נודך כפלים / כאשר ינוד
הקנה במים / ויצא לבי מאתי / כי באש נודך
שרף / ואמר:״אך טרף טרף / וילך אל המדבר כערוד
/ וישב בארץ נוד / ואני חשבתי כי לבי אחרי
יעמד / והנה הסכלתי ואשגה הרבה מאד״.



ותשא משלה ותאמר:

וחשבתי להתנודד בנוד דוד / והוחלתי והופר מחשבי

ובטחתי בעז לבי ותקפי / והוא נסחף ולא עמד לבבי

ואמרתי כבר צדתיו ברשתי / והוא עודה והוא טורף כלביא

וגרשתיו ונגרש גם לבבי / והוליכו כאיש הולך בשבי

בהתנודד צבי חשך מאורי / ומזרחי אזי שב מערבי

ונחמתי ונחומי כאין / לאחר נוד אבל נכפל כאבי

והדליק אש נדודו תוך כבדי / והגדיל המדורה על שביבי

אחזתיך ידיד לא ארפך עוד / ועוד לא אשמעה בך קול מריבי

קרב דודי עדי מתי קרבות / וכבה אש נדודים בערה בי

נדודים תחליף היום בדדים / ומי קרבתך תן על שרבי

ארה מורי ידידי עם בשמי / אכל דבשי ויערי עם חלבי.



ותקרב אליו / ותשתחו על שוליו /
ותנשקהו על־ידה כמו משפט ידידים נאמנים / ולא
נשקתהו בפיו מפני הנרגנים / רק עיניה הם
מרקדים ומשחקים / כאילו הם מחבקים / וירא סהר
כי לא נשקתהו ויבז בעיניו / ויחר לו מאד
ויפלו פניו.



ותשא משלה ותאמר:

נשקתיו על ידי מלאך ידידים / וענני: ״הלא פי ציר אמונים

הכמוני ביד מלאך לנשק / הזאת תורת ידידים העדינים״.



ותאמר לו: ״אל ירע בעיניך / למה חרה
לך ולמה נפלו פניך / הלנשק ולחבק הוא חשקנו /
לא יעשה כן במקומנו / אבל חשקנו וחשק הנדיבות
/ לצרף וללבן הלבבות / לא כמשפט בני התערבות
/ כי ישבו יחד לא לנשק לא לחבק / אבל לב [זה]
בלב זה דובק / וכן סבת אהבת בני היקרים / לקחת
מוסר השכל, צדק ומשפט ומשרים / ויהי ידם אמונה
/ ואם מספרים דברי חשקם ואהבתם / תגבר צדקתם
/ ואם הגבורות יספרו אודות חשקם והתלאות
והקורות. / אלה החקים והתורות / וזה כל מתוקי
חשקם / צופם ודבשם / והיו לברות למו וישיבו את
נפשם.



ותשא משלה ותאמר:

אהבתיך אבל לא על דבר מום / וכמוני ידידותם אהובה

וחשקנו נקי־כף בר פעלים / ורוח אהבתנו נדיבה

וספריהם מלאים תם וישר / ויראה וענוה בם כתובה

וכל־חשק בני נבל נבלה / וסוף כל אהבתם מעצבה

ותתמגג כרגע אהבתם / ותתהפך ידידותם לאיבה

ואם תשמע דבר מום על נדיבים / חשב טוב ואמר גם זו לטובה.



ויאמר סהר בשמעו דבר לחשה / צדקה
נפשה / ויאמר: ״טוב הדבר אשר דברת / ברוך
טעמך וברוכה את״ / ויבואו אל הארמון ודלתותיו
ברושים / ממלאים בתרשישים / מעשה חרשים /
וקרקעו וקירותיו וגגו זכוכית כלם / והמים להם
חמה מימינם ומשמאלם / ותבא אליו בתחלה /
[ויפלא] ממנו המעשה הנפלא / ולשחות פשט
הבגדים והנעלים / כי יחשבהו פלגים יבלי מים /
וכי ירא המים הנגרים תחת הקרקע / חשב כי
יטבע במים וישקע / וידלח וירקע ברגליו / עד
אשר מעדו קרסליו. / ויהי כראות סהר הבית
וצורתו / והמים אשר מתחת הזכוכית ותכונתו /
ויבאו בלבו מורא וחתית, / בראותו גלות עלית
גלות תחתית / ויאמר: ״הלא הרוח אל זה הים שנית
תדפך / אם לא ים שנאת וים ירדפך.״ / ותודיעהו
כימה דבר הבית והזכוכית וצורותיו / וגלות המים
אשר תחתיו / וישבו בו בשמחה והנחה / ויכינו
את המנחה / ויאכלו וישתו / ולבותם מתאותם
חתו. / וילינו שניהם מדברים מכל־טוב מלה / ולא
קרב זה אל זה כל הלילה / ונפש איש לאחיו כלתה
ונכספה / נשאותיהם עמוסות משא לעיפה / ואיש
מרעהו אין לו מכל חפצו ורעיוניו / כי ראות עיניו. /
ויהיו כן ברורים מכל־מום ותהלה / ויתחרשו
כל־הלילה / בנגינות שיר אהב ותהלה / עד כי בערה
בם אש תשוקותיהם נחר / עד עלות השחר. / ותגדל
אהבתם עד אין חקר / ותתבונן אליו בבקר,



ותשא משלה ותאמר:

הסהר יחנה מרום / השמש תהלך ארצה

אשר זה… עמד / וזאת לבא בלי אצה

מבקש יום וליל יחד / יגל תאר צבי ימצא

בעיניו יהרג איש תם / והא דמי כבר נמצה.



ותוסף פנות אל הדרתו / ואל יפיו ואל
יפעתו / וזהר לא יתאר כברק לחייו ואל עיניה

ותוסף משלה ותאמר:

מאור שמש ואור סהר / יקנאו בצבי סהר

שניהם יעשקו אורו / ויוסיף לחיו זהר

יריבוהו ואך כל־יום / לחייו יספו סהר

וכרעו לו אזי אורים / לזיוו תארם צהר

וגם הודו עדי נגהם / עלי אור לחיו יגהר

אשר סהר דמות עפר / ואי אדם ואי טהר

ידמה דוד אלי שמש / כסיל בער בלב נמהר

וזכרו מר־דרור ושמו / כשמן טוב וטוב יצהר

ונדבות לב אמהות לו / וימהרם הצבי מהר

ולו שואל יאו הונו / לתתו לו אזי מהר

ולו שאל אנוש נפש / נתנה אך היה נזהר

לזאת אם נוד ינידנו / ואתשוטט בראש כל־הר

אצפצף לי עלי נודו / כסוס עגור, כסוס אדהר

ואהגה אהמה כל־יום / כיונים על־ברוש תדהר

אצו עבים נשוא שלום / לדוד חן רב כמי נהר.



וישמע סהר ויאנח ויאמר: ״אהה כי לבי יקד יקד / ויצמא עד מאד / ראי
רעיתי כי אש נדודך בחיקו יחתה / הטי־נא כדך
ואשתה / טרם מחיים נגזרתי / פן־תדבקני הרעה
ומתי״. / ותען כימה ותאמר: ״אל־נא זה הדרך תט /
ויהיה בך חטא״.



ותשא משלה ותאמר:

עוון דוד רב מאד מנשוא / ואין כפר אלי חטאו

כבדי סהרו אתן / ולבי יהיה כלאו

ולי לבב כבד־אזן / בלי ישמע לקול קוראו.



ויהי סהר עם־לבו במלחמה / באהבתו את־כימה.



וישא משלו ויאמר:

חשק ומלחמה שניהם הורגים / הוי על בני חיל והוי על חושקים

אל ברמחים וחרבות נלחמו / גם אל בחצים לוחצי לב דוחקים

רמח הכי נשבר וחרב תחטיא / אך בעלי חצים לעולם דולקים

ובאף וחמה נלחמו [המה] אבל / עיני צבי יורו והם כמשחקים.

בלבה אהבתו / על כי זה לבזה נתן נפשה הצבי /
לבבה נהג בשבי.



ותשא משלה ותאמר:

למי אלון עלי חשקי / ועיני היתה סבה

לבבי בשבי הלך / ביד שר אהבה נשבה

והוא לא שב עדן אלי / והוא שר הידידת בא

ואם אמר להפדות / הלא דוד במחיר הרבה

ובקשתי וענני / בתוך לב הידיד נחבא

הבמראות נעים שיר דוד / היש נביא בשיר נבא.



ובכל־זאת כימה לא תראנו / וכמסתר פנים ממנו.

רעיה איך תסתרי / לחיך נא גלי גלי

או חבי נא לבעבור / זהרי רום תנצלי

לב ידידך תנתחי / תזבחי גם־תבשלי

בי רצונך עשי עשי / בשלי או צלי צלי

אהבת אוהבים הלא / מול אהבך כמר דלי.



ויהי כן ימים רבים בין מריבה ותשובה /
ואיבה ואהבה / ובין דברי אהבות / ודברי
מריבות.



וישא משלו ויאמר:

רעיה יום ליום תלנות וריבות / יעצר הזמן ונחנו מריבים

האנחנו לבד מענים כאלה / אוהבים או הזאת תעודת אהובים

יענני ידיד בלי ריב כשוכן / עם־שכנו ואי מתוקי אהבים

הה ואם אין באהבה ריב ומדון / אין נעים שיר או נעימת אהבים.



וידבר כזאת וכזאת אולי לרצונו ישיבה
/ ולא ענתה ולא שתה על־לבה / ויהי כראותו
משובתה / וכי היא מחזקת בתמתה / ויבא אל
הארמון ויקרב אל הדלת / ותצא מכבדו גחלת /
ותצא כימה נבהלת ויחפה / לראות את השרפה. /
ותאמר לנערותיה: / ״מה האש אשר בדלת נשקה?״ /
ויריבון אתו בחזקה / ויען לאמר: ״הלא שרפת גחלת
כבדי / מה־תריבון עמדי״! / ויתחנן אליה /
ויאמר: ״אחלי לפני הנערות העומדות אתי /
שתאמרנה באזני כימה גברתי / ואולי תתרצה
נפשי אשר בנפשה נקשרה / וכבו גחלת אשר
נשארה״.



וישא משלו ויאמר:

ויהי בנוע מני צביה / יום יום ותיקד אש בקרבי

בתי לדלתי ביתה כשואל / כעוף יעופון רשפי שביבי

וישרפו את דלתה ולא על / חפצי וזאת לא עלתה בלבי

חרה לרעיה עד־מאד ותשכח / כי אש נדודה אז בערה בי /
שלם אשלם דלתה בכפל / גם היא תכבה אשי ולהבי.



ותרא כימה כי בה התאנף ידידה / ויפלח חץ חמתו כבדה.



ותשא משלה ותאמר:

לבבי זע וציר אחז / ואם בו הידיד אנף

אשר יחרד והתעופף / היש לב עף כעוף כנף

נטע חשקו ביום פרח / בנטעו העלה ענף

עמד לבי הלא לב דוד / בך הורם בך הונף

וגם לבו יצועך / אשר [בו] מר אהלים נף

ולב רעיה אשר ידבק / בזולת אהבה טנף

ואם ידבק בזולתו / ושב אליו הלא יחנף

מעיל חשקו יעטני / צניף דדיו הלא צנף.



ויספר אליה סהר תורת הידידות / וכל
תלאות הפרידות / ויאמר לה: ״לכי שמעי האוהבים
העמלים / ותחלואי החושקים האמללים / וצרות לבותם
הנהלמים / אשר יקרא אותם באחרית הימים״.





וישא משלו ויאמר:

חושק עשוק נפש רצוץ / לבב יהלך צו לצו

יבטח בקרבת רעיה / אך תמשכנו קו לקו

כי אין בפי רעיה אמת / לא תעמיד רגלה בשוא

איך באהבים יאמין / חושק דבר כזב ושוא

כי אם יעדתהו, עדי / שיבת כנף עורב יקו

ליליו אלי עורבים וכל / ימיו בשיבתם יאו

יאמר ישובון עורבים / אולי צמאו אז ירו

אין בין ואין דעה עדי / בין טוב ובין הרע ישו

יקרב מדורות אש ימי / קיץ וחרף גם סתו

נבהל בלי לבב ואם / יחשב לתקן אז יעו

יבער ויכסל עד בריר / פיו מדברי אהב יתו

אם יאריך [ימים] כרו / קברו ואל ביתו יצו

תכיר בני חושק כאיש / התוה עלי מצחו כתו

אלה קצות דרכי חשוק / קורות אהבים מי יחו.



עוד הם מדברים דברי חשקם /
ומתחלחלים בעת צרתם ומצוקם / והנה נערה באה /
נבהלה ונלאה / ותאמר לכימה: ״הכיני ביתך ונותך /
כי הנה אביך בא לראותך / והחביאי סהר ודבר
חשקך מאביך סתרי וכסי / ועתה דעי מה תעשי״. /
ותשמע כימה ותקם [ותאמר]: ״אסתיר את סהר
ידידי ואחביא / כי כלה היא מעם אבי״.



ותשא משלה ותאמר:

ידידי אן אשו דוד אן / אביאו מפני רודפי

אביאו תוך סגור לבהן / רשפים שם והם שורפי

ואן אחביא צבי לפני / מקנאי אויבי צופי

וכבד רתחה כסיד / בגחלי וברצפי

ואין כי אם להסתירו / להחביאו בסרעפי

ולו [בא] באשון עיני / אכסנו בעפעפי

מקום בכי בגיל אשחק / בשמחה אספקה כפי.



ויהי המלך לבוא וייראו והנה
תחביאנו במחבוא / ויך המלך סהר מכה / ויאמר:
״הגם לכבש את המלכה״.



וישא משלו ויאמר:

ומה־יועיל בעז אריה / והן עפר יצודנו

ומה־יועיל בעז מלך / והן עבדו יגודנו

ואם הניא אניאנו / ומתנכל אנידנו

ואם צדק אקרבנו / ומשירי אהודנו.





וישב המלך אל כימה ויגער בה /
ויחרפה ויעציבה / ויאמר המלך: ״קחו לי את סהר /
והוליכוהו אל בית הסהר״! / וישב פניו אל כימה /
והנה כסתה פניה כלמה.



וישא משלו ויאמר:

והננו שני יונים בבית / במעדנים וענג הידידות

מצאונו תלאות הזמנים / ונפרדנו והה על הפרידות.



ותאמר כימה: ״ידעתי כי אהבתך לי כיום תשבך / הן קלתי ומה אשיבך״.



ותשא משלה ותאמר:

והיינו שני ענפי הדסים / בגן רוה דשנים רעננים

מחבקים מחשקים דבקים / ונפרדנו בקורות הזמנים.



ויהי כשמוע המלך ותחס עינו עליהם /
בשמעו טוב מליהם / ויאמר: ״השיבוהו ונדרשה בעד
חשקם / ונחקר בעד רשעם וצדקם״. / ותען כימה
ותאמר: ״אדני המלך כי זה הנער לא אמושהו ולא
המישני / כי צדק לבשתי וילבשני / ומאין
חשקי מבגדי יראתך פשטני / סהר מעיל צדקה
יעטני / ויציענו מצוה / ומלבושנו ענוה / לבוש
יראה לבושנו / ורדידנו דעת קדושינו״.



ותשא משלה ותאמר:

ויעטנו מעיל־חן גם ענוה / ולבשנו לבוש־אהב ויראה

וכרתנו ברית־חשק בלי מום / אשר לא יעלנה ריב וקנאה

אנחנו נושאי על תם וחשק / ולב נושא שתי אלה הנראה

ורוח אהבה תעיר להרע / אבל רוח חכמה היא מניאה

ורוח אהבתי נכבשה בי / וחכמה לבשה גאון וגאה

רוי דדים חשבונו מקנאים / והא יבש גרוננו בצמאה.



ויאמר לה המלך: ״הגידי נא לי מה
החשק הזה ומה־סבתו / ומי החושק ומה
אהבתו״?



ותשא משלה ותאמר:

בני אהב / יקוד להב / ואם הב הב / בקרבם
בלי יכבה / אבל ירבה / כדי־ארבה / שביבם
וישפילו / ויחילו / יחוללו / לחרבם
דוי נפש / בלי נפש / ואין חפש / לעצבם
הכי דודים / ככידונים / ולפידים / בלבם
ולב יקלו / ולא יצלו / ואכלו אין / מכבם
הדיחוהו / שחטוהו / זבחוהו / בחרבם
יכתוהו / יחתוהו / יפתוהו / בעקבם
ועיניהם / מעניהם / ומוניהם / בעגבם





ודע וחרש / חקר ודרש / הלא הם ראש / מסבם
ושם חשים / בני כושים / ידות אשים / בגבם
ודוד ילאו / ולב מצאו / ולא חטאו / בריבם
ולא דמו / ולא שלמו / אבל הלמו / עקבם
בני־חשק / בני־עשק / כלי־נשק / סביבם
לבב דוד דל / מאד ידל / עדי יגדל / כאבם
ועינים / כעין מים / כנחלים / שאבם
וצרותם / וקורותם / קצותם, מי / כתבם
ודדיהם / מנדיהם / יקודיהם / ולהבם
ויד יהפך / ודוד ימשך / ומי ידרך / נתיבם
ויגידו / ויעידו / ויפחידו / בניבם
דבר־שקר / בפה יקר / לאין חקר / כזבם
וזה ימחץ / וזה ילחץ / עדי יחץ / לבבם
ולולי דוד / אשר ידוד / גביר כל־הוד / נציבם
אשר פרח / כאור זרח / בהוד אזרח / נדיבם
אזי מתי / וגועתי / ונצמתי / לכזבם
כליל יפי / בלי דפי / דבש צופי / ערבם
ויד שוה / לדל תקוה / עדי רוה / שרבם
למושיע / ליגע / למשביע / רעבם
נדיבי עם / וכל־שועם / בלי טעם / עזבם.



ויאמר המלך לסהר: ״ואם בכימה חשקת
/ ראה חיים עם האשה אשר אהבת״. / ויאמר המלך
לכל שריו ועבדיו / ולכל ראשי חיליו וגדודיו /
ולכל העם הנמצאים סביבו: / ״הנמצא כזה איש אשר
רוח אלהים בו״? / ויעש להם משתה גדול, וישב סהר
עם כימה / במשתה ובשמחה שנה תמימה. / וינגף
המלך ויקם סהר מיין משתיו / הוא והשרים וכל
משרתיו / ויקברוהו וימליכוהו תחתיו / וישב
סהר המלך עם כימה הגבירה / מושל בצובה
הבירה.

ויהי אחרי כן ויקם סהר בערמה / וירב
בכימה / ויהי ריב מקדש / ודברי חשקם מחדש / כי
ריב אהבים הידידת לוטש / וחושק בלי ריב חשקו
נוטש. / ויפקד עליה עוון מריה / ונדודה ממנו
וקשיה / באהבתם אשר נאהבו / טרם ישכבו /
ויאמר לה: ״הייתי יושב בארמונך / ותפיחי לי ריח
קנמונך / ותכבשי כאותי ורצוני / ולא עשית
כטוב בעיני״. / ותען כימה ותאמר / ״הכזה יעשו
באהבים / דודים הנדיבים / הלא זאת חרפה ובושה /
כי תפקד עלי עוון האשה. / אל תזכר לי עוונות
ראשונים / הלא דברת מימים קדמונים / מי יתן
ואראה כימה / ויעבר עלי מה / ואם לבבך מעלי נטה /
או לבי מעליך שטה / ואם באדמת חשקנו היתה
שמטה / יצא חוח תחת חטה / ואם תרצה אהבתנו
בריב ללטש / לפני התלאה התגלע הריב נטש / הלא
הארמון ארמונך / והקנמון קנמונך / ומדוע
תנידם ממני / ובנדוד תחלימני / שוב והשיבם עד
אריחם באפי / כאשר הייתי בימי חרפי״.





ותשא משלה ותאמר:

ריח בשמים נדדה מני / גם נחרו בי אשכלות כפר

מטרות חרון דוד גברו אין לי / כי לעשות תיבת עצי גפר

אשתוללה אשום ואשתאה / אביט מקום דוד… שפר

אתחבאה עד יעבר זעם / עד מחרונו יהיה צופר

אם לב ידיד תתה עלי דוד זר / או החליף תחת פאר אפר

חטא וגנב לב ידיד על־כן / לא ימצאה על מעלו כפר

לבב קרעתו בנדוד קום־נא / אולי קרעיו תהיה תופר

איך חושקים יהפכו עושקים / דוד שעשועים איך ברית יפר

ואשאלה על־זאת וענוני / כן משפט חשוק וחק עפר

אם אין באהבה ריב ואי מתקי / תוכחות ידיד או אי נעים ספר.



ויאמר סהר: ״אל תשיתי לבך ודבר
הקנאה הלא משחק אני״. / וינחם סהר על אשר דבר
קשות ועזות / ויאמר: ״מה חרדת את כל החרדה
הזאת״. / ויהיו כן מריבים סהר וכימה / מימים ימימה
/ וכזאת היו מוליכים אהבתם ומחדשים / וחרבות
חשקם מלטשים / וחמה ביניהם לא עלתה / ונפש איש
לאחיו נכספה וגם כלתה.
