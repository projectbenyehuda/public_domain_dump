


	
	המחברת השביעית: ישפה ושתי אהובותיו: עפרה וימימה
	




נאם למואל בן־איתיאל: ויהי כאשר ירדו
המלכים מכסאותם / והוסרו שרי צבאות מצבאותם, /
ושם נקרא איש נדיב / והזמן נפשו האדיב, / ושמו
נמואל בן־ברכאל נדיב / מנדיבי ישר, ולו בן אחד
ושמו ישפה / וכמוהו לא היה איש־יפה. / ושם עירם
חצר סוסה / ובני הנדיבים היו שם לשמצה. / וירא
כי הזמן בנדיבים מועל מעל / והקים עליהם בני
בליעל, / ויתעצב לזאת עצבון, / כי היה בן חכם
ונבון. / וישא משלו ורננו / וילן על
זמנו.





וישא משלו ויאמר:





זמני איך נדיבי־עם / תשו כבני בליעל?

ואיך תשפיל בני־שכל / ודור סכל תקימו על?

היתכן חצר סוסה / היות שפחת חצר שועל!





ויאמר לאביו ולאמו / ״לזאת יחרד לבי
ויתר ממקומו / על נבלים כי הכבירו / ורשעים
את־צדיקים הכתירו. / ועתה אלכה ממדינה למדינה
מעיר אל עיר / ואתהלכה לפני גדול וצעיר, /
ואתהלך ברחובות ובשוקים / ואשב בסוד משחקים
/ כאחד הריקים. / ואם אראה יין ותירוש, / אבחר
דרכם ואשב ראש / או אם אלין בערב בכי וארנן
לבקר, / או עשיתי שפטי שקר / אז אולי יחוס עלי
הזמן / ויעשה לי בית נאמן: / כי קצרה נפשי
בארץ הזאת / ורבו בלבי דאגות ורגזות״.





וישא משלו ויאמר:





ואם ארץ לך תצר / ואם בה נפשך תקצר,

ברח מנה לאחרת, / ברח לך ממקום מצר!

ואל תאמר ״ואיך אלך? / בלכתי אמצאה אוצר״.

בטח על רחמי יוצרך, / היד בוראך תקצר?





ותעש לו אמו צידה ומטעמים / ומיני
בשמים / וזנים ומרקח / וינשק לאביו ולאמו ואת
ערבתם לקח. / ויתנו לו מזהב אלפים / ומהכסף
מאתים / ויהי בלכתו / וימצא שם אנשים ריקים
ופוחזים וילכו אתו, / והאנשים נבזים ושפלים /
וכמדותם קנקנים בלים, / נבלים סכלים, / סובאים
וזוללים, / ישתו יינם בנבלים, / רצונם לאכל
ולשתות / ולחבק אשפתות. / ויתחבר ישפה
אליהם / וילך עמהם / ויהי כאחד מהם / בהתלוצצם
יתלוצץ / ובמרוצתם יתרוצץ / ובשבתם ישב /
וכאחד מהם יחשב, / ישתה וישתכר, / מספח
חמתך ואף שכר. / וילך אתם מהר אל הר, / מנהר
לנהר, / מגבעה / לגבעה / מתחבר לבני פרחח, /
והיה כעוף נודד קן משלח. / והיה אם בא לעיר
גדולים, / יהלך עם נבלים / ויתחבר עם כסילים, /
ממשל עמהם משלים, / ישכבו בין שפתים / בין
תנור וכירים. / ולכתו כלכתם / ושבתו כשבתם /
ונחשב כנבל – והוא איש תם, / ומלונו כמלונם, / אך
כי רצונו רחוק מרצונם; / רק יראה קלונם / ויפקד
עוונם.





וישא משלו ויאמר:





זמן רע, אם יסרתני / בגוי נבל ובכלבים

והולדת בני עולה – / הלא הם לך למכאובים,

ואם רבו בני ישר – / הלא תפארתך טובים!

משכתני עלי כל הר / ומעינות ומשאבים,

אהלך עם בני סכלות / להכעיסך בשוא שבים.

יסרתני בשוטים – אך / איסרך בעקרבים!





וילך אל ארץ מרתים, / אל שערים
ועדיתים, / אל ארם נהרים, / בארץ תלאובות ומבועי
מים, / עד בואו אל ארץ מצרים / מקום יאורים רחבי
ידים, / ועמו האלפים והמאתים, / וישב בין
החומותים. / וירא חומותיה כי הלבינו מהגיר /
ואבניה אדמו עצם מפנינים וגזרתם מספיר, /
כתרשישים קצותיה – אבן שחרה אדמה ברדה / ובה
כל שכיות החמדה. / ותדמה כי האבן שתים, / על
אבן אחת שבעה עינים, / ויפארו הבנין בחצצים, /
עד כי נראו משבצים.





וישא משלו ויאמר:

ומצרים מהללה / בחורה מאחותיה,

בצורה מעבריה / וגבהו מגדלותיה

בחומת מעשה תשבץ / ויפו משבצותיה,

אבניה כעין ספיר / כתרשישים קצותיה.

וכאם למדינות היא / והערים בנותיה,

והיא בינם כמו שגל / סביבה נערותיה.

עדי נזמה וחליתה / יאור על גרגרותיה,

ולא תדאג בבוא חם, כי / מטרה נהרותיה.

ומה יפו נאותיה / וטבו משכנותיה!





ויהי יושב בין החומותים / והנה נערים
שנים, ויאמרו לו ״למה ישבת בין המשפתים? /
קומה ונתהלך, / והנה הטוב אשר ייטיב יי לנו –
והיטבנו לך״. / ויוליכוהו בשוקים וברחובות / עד
שוק נערות ערבות, / אשר עיניהן לבבו / ועפעפיהן
לבבות יגנבו, / ויבט אל זהר פניהן / ואל יפי
עיניהן, / ובתוכן נערה קטנה – / ותתן בלבו רעדה
ומגנה. / ויהי עד כה ועד כה ותעבר הרנה, /
והנערה כצביה / ותטרף לבו כלביאה, / ותנהג
נפשו כשבויה: / היא הולכת / ונפשו אחריה
נמשכת.





וישא משלו ויאמר:

ועפרה לא ככל עפרה – / לבבות תמשכה משך,

ואור מלחיה נבקע – / לאורה ילכו חשך;

וחלק לה בלב כל איש / ותשיך ביציר נשך.





ויאמר לאיש המלצר / ״מהר והוציאני מן
המצר!״ / וירא המלצר מעשהו ומתכנתו, / והוא
נשען על משענתו / קרוע כתנתו, / ויבזהו כי
ראה מדיו קרועים, / וחכמת המסכן בזויה ודבריו
אינם נשמעים. / ויאמר ״האתה תקנה הנערה והדרה
– / ורחוק מפנינים יקרה?״ / וירץ ויקנה בגדי
חמודות / ולאצבעותיו טבעות / ופארים לראשי
מגבעות / ופרדה נאה / וסוס בחמשים ומאה / ונער
כושי / וילבישהו בגדי חפשי. / ויקן את כל
אלה אשר על הנערות / וישתחו לו הוא וכל העפרות
/ ותאמרנה ״יחי אדוננו השר, / ואולי אחת ממנו
בך תבשר!״ / ויקח גברת הצבאות / וישקל האלף
ושבע המאות / וירכיבה על הפרדה / וילך הוא
ולפניו הכבודה / ושמה יפיפיה, / ותהי לו
לרעיה.

וירא אחד מהנערים אשר הביאוה כי כן
/ ויאמר ״אני אהיה סוכן / ובביתי יהיה שוכן״, /
ושמו משוש / מארץ פתרוס. / וירץ משוש לקראתו /
ויביאהו אל ביתו, / ויקד לפני ישפה וישתחו,
ויבוא ישפה וירא כל הבית וכל שעריו, / והוא
מתהלך ויפיפיה אחריו. / ומבואות הבית שנים עשר
/ וכל דבר יפי לא חסר, / ובחצר נטועים כל עצי
נעימים / גבהים ורמים, / ובתוכה ברכות מים / וכל
מיני עוף ובעלי כנפים. / וסביבות העינים והברכה /
שבכים מעשה שבכה, / פארות ארכות ויפות / וצפרים
עפות / הוגות ומצפצפות. / ובנין הבית נאה / בית
גדול למראה / והחצר מאחזה / בארבעה בתים מול
מחזה אל מחזה, / בנינם חזק נשגב / ועל פניהם
עמודים ועב. / ויהי בבואו ויראהו.





וישא משלו ויאמר:

ארמון רחב יד נעלה בנין / צורות אריות בו מפתחים,

גם בניני בתיו מחקקים / נאים עליותיו מרוחים.

מפי אריות נגרו מימיו / על רב חצצים כבדולחים,

עצים שתולים על אגם מים / עד ים קציריהם משלחים,

קיץ וחרף רעננים הם, / ובקר וחם ישארו לחים.

ויצפצפו תורים עלי פארות / ויצוחו יונים בבין שיחים,

יתילדו בין ציץ ושושנים / ויעשו ביצים ואפרוחים.





ויבא לו משוש כל כלי מחמדים / וערש
מרבדים / ויתן בבית הפקדה / אוצר כל כלי
חמדה. / ובכל זאת ילך ישפה אל הנערה שחוח, / כי
מאהבתה לא עצר כח, / ויבוא חשקה בלבו
ובכבדו / והמזלג שלש השנים בידו, / והם
נושקי רומי קשת ויורוהו בחצים ברורים / ויחל
מאד מהמורים, / ותרא יפיפיה כי מחצוהו / ובמקום
צר לחצוהו.





ותשא משלה ותאמר:

ידידי, והלא עיני / ללבות פרשה רשת!

תעורר אהבת דודים / ותכליתה מחדשת.

וזאת תדליק בלבות אש / וזאת עצים מקוששת

וזאת תגנב לבב חושק / וזאת קרבו ממששת.

ויום יום קלקלה חצים / ומלחמה מקדשת

ותורה חץ ולא תחטיא / גדודים בו מכבשת.

בני, הרחק והזהר / ושב כמטחוי קשת!









ויען ישפה ויאמר ״ולמה את אותי
מגרשת / באמרך: הרחק כמטחוי קשת? / הנה
בעיניך לבבתני, / ועתה ממך הרחקתני – / אכן
הכרע הכרעתני!״ / ותשא עיניה ותרא יפי יפעתו
/ ותחל מאהבתו, / ותבוא הנערה במשעול האהב /
ותיקד בלבה אש ולהב, / וקדם זאת לא ידעה מה
אהבה מה משעול / ולא עלה עליה על. / ותשב
מאהבתו נפעמה והדוכה / כגרן עת הדריכה / או דכו
במדוכה. / ויבן ישפה, כי מאהבתו חלתה /
ואהבתו בלבה ובכבדה פשתה / וכי נכאה לבבה /
והחשק מעבריה סבבה.





וישא משלו ויאמר:

אהה! לב רעיה נכאה / במשעול אהבה נלחץ,

ולב יבוא נתיב אהב – / לכל רוחות הלא יחץ.

ולו עני עברו או / ידכא או בני שחץ!

הלא אהב יבואנו / לכל יבוא כחץ ימחץ,

והא דמי מהצבי / בדמעותי אני ארחץ.





ותיקד אש בלבות השנים / ויגעו
לכבותה בדמעות עינים – / ואלה חוטבי עצים ואלה
שואבי מים, / וזה לבו מתיאש / וזה יגע בדי אש. /
ותענהו יפיפיה בדברים ערבים / ומעני אהבים, /
לחך מתוקים / ללב דבקים, / ותאמר ״ידבר אדוני
בכל לבבו: / מה מלאכתך ומאין תבוא?״ / ויהי
כאשר הציקתהו, / ויגד לה את כל התלאה אשר
מצאתהו. / ויאמר ישפה למשוש הסוכן / לטבח טבח
והכן. / ויביאו המטעמים, ויאכל וישביענה /
אחת הנה ואחת הנה. / וישתו יין ידידותם /
ויאכלו מפרי אהבתם, / ויהיו השנים / בהחזיק
אזנים / ולקחת לחיים / ונשוק שפתים / וקרוץ
בעינים / ולהלחם בעפעפים / ושלוח ידים /
ומעוך שדים, / וישכבו את משכב הצהרים.&gt;
ובשוק הנערות נקראת נערה יפה /
בספירים מעלפה, / תמשך היפי בשוליה / וכל
הצבאות לא דמו ביפין אליה. / ותרא כי עזבה
ישפה ולא לקחה / ותעל חמתה ותצא רוחה, /
ושמה ימימה / ברה כחמה. / ותקרא למלצר
בחרוניה ושאוניה / ותתן לו אחד מצורוניה /
ותאמר ״קנה לי סוס נאה / נחמד למראה / ובגדי רקמה
/ וכלי מלחמה! / ואם מחירו אלפים, /אתנה לך
ואשתחוה אפים!״ / וירץ המלצר ויקן את כל אשר
אמרה / אחת מהנה לא נעדרה. / ותתן לו אלפים
זהב מסדיניה / אשר לקחה מבית אדוניה.&gt;
ותלבש כל לבושה / ותשם כתר מלכות
בראשה, / ותרכב סוס גבה קומה / ותקח כלי
מלחמה, / ותשאל למקום ישפה ולביתו / אשר היה עם
יפיפיה רעיתו. / ותארב עד הערב / ותרכב סוסה
ותקח בידה חרב, / ותפתח הדלת בידיה
כמגנבת / ותשב במבוא הבית מארבת. / ותקשר
לשון הסוס בשער / ותבוא בלט אל הנער – / והנה
הוא ורעיתו ישנים שניהם / ורוח לא יבוא ביניהם,
יעפים מן השחוק / כדרך כל השוחק וחק, /
ותמשכהו מחיק רעיתו, / ולא הקיץ משנתו. /
ותאסר ידיו אל אחוריו / ותוציאהו מחדריו /
ותקחהו באוניה / ותרכיבהו על הסוס בפניה, /
ותגער בסוס / ותאלצהו לנוס. / ויהי אחרי צאתם
מן המדינה כהרף עין, / וייקץ כצאת ממנו היין /
ויצעק ויאמר ״הנה הקיצותי / ונהייתי נחליתי /
ונעצבתי, / כי גנב גנבתי. / ואם גבור חיל הוא
אשר אסרני / ומחיק רעיתי העבירני – / ינתק
מוסרותי אשר קשרני, / ואם יוכל להלחם אתי –
והכניעני״.









וישא משלו ויאמר:

הה לי, רעיתי, הה לי, כי / מחיקך גנב גנבתי!

לו מוסרותי נתקתי, אז / לא אשקט כי אם נקמתי.





ותגער בו הנערה ימימה אשר גנבתו /
ובעין קנאה שזפתו / ותאמר לו ״אתה הולך
בשביה / ולא ידעת אם למות אם למחיה – / ועודך
תזכר ערש רעיה!״ / ותשליכהו מעל הסוס, וילך
לרגלו כאשר הולך בשבית, / ותכהו בחנית. / אזי
העז פניו / וחרק שניו / ולטש עיניו / ויאמר ״מי
אתה אשר איש כמוני תדכא? / האשר שבית
בחרבך אתה מכה.״ /





וישא משלו ויאמר:

הגבור את וגבורים / בחרבך בשבי תנחם?

אסורי נתקה כיום / וצא עמי להלחם!

ולו את עם גדודיך / תבואון לי – אני אמחם,

ולו יהיו כחול – אדרך / גויתם ואדלחם,

ואשחקם כמו עפר / וארקעם ואסחם.

גבורה משדי אמי / ינקתיה ומרחם,

ולי לב ישאלה חשקה / כעולל ישאלה לחם,

ולבי לא ככל חושק / אשר יחשק וינחם.





ויהי למחר / כעלות השחר / וזרח
השמש / ונכר היום מאמש – / והנה אחד פרש /
מתגעש ורועש וכים נגרש, / והוא כאריה מגוריו
שכל שכול / לא ישוב מפני כל, / וסוסו יגמא ארץ
ויאכלנה אכול, / וירא וידאה / ויעלה ויגאה, /
ירוץ כנחל שוטף / ובכלי מלחמה מתעטף, / ובחניתו
להבת / כאש צרבת, / יחרק שניו ולהבות אש מהם
ישוטטו, / כידודי אש יתמלטו. / וירד מראש ההר
וירץ כברק / וסוסו מתחתו כחץ הורק / על ישפה
וימימה / וינח עליהם כאשר יפל הטל על האדמה, /
והנה היא יפיפיה אהובתו / ולא הכירה מרב חרדתו
– / והיא הכירתו / וממוסרו נתקתהו.&gt;
ותקם ימימה אליה בחניתה, ותשב
אליה יפיפיה / ותרץ לקראתה כצביה. / ותהי
בינותם מלחמה גדולה: / זאת יורדת וזאת עולה, / זאת
צורחת וזאת צווחת, / זאת רודפת וזאת בורחת, / זאת
בכליה מתהפכת / וזאת חנית על חברתה מושכת, /
זאת מימנית וזאת משמאלית, / זאת מחרפת וזאת מגדפת.
/ ובכל זאת חשבם ישפה אנשים – / והנה נשים! /
ויעל בינותם האבק / ובפניהם העפר דבק / עד מרב
עפרם / חשך משחור תארם / ועד אשר לא נראו /
ולמצא זה לזה נלאו. / ותצאנה שתיהן מהמערכה, /
זאת וזאת חניתה לארץ משכה.





ותשא משלה ימימה / ותאמר ליפיפיה בחמה:

הפרש מתהום יעל / המרום יחנה – מפח

להציל ישפה כיום / כהנצל צבי מפח?

ואין זאת כי יקוד לבך / ברוח אהבה נפח

ודכא לבך חשק / ואהבה קדקדו שפח.

ידידות גדלה לבך, / בחיק האהבה טפח,

תפרש כפך מחם / יקוד לבך ותתיפח.





ותשמע יפיפיה ותאנח / ותצא רוחה
וכמעט מעל הסוס תפל ותצנח, / ואחרי כן שבה
אליה רוחה.





ותשא משלה במר רוחה:

הפרש נעשה גנב – / ואם גנב יהי פרש?

ואם אמש גנבתו לך / ומחיק רעיה גרש

ותקנת גנבתך / גנבת מעשה חרש

וחרפתו וחבלתו – / והא דמו לך נדרש!

הגדפת הגנבת / הרצחת וגם תירש?





ויתמה ישפה כי ראה זה אומר וזה
מכחש / ויאמר: אין זה כי אם נחש ינחש! / ויחר אף
ימימה על יפיפיה / ותך אותה בחנית בין הדבקים
ובין השריה / ותפל מעל הסוס על האדמה / וכמעט
לא נותרה בה נשמה, / והוסר מעל ראשה צניפה /
ונגלה צעיפה / ויראו דלות ראשה על זהר הפנים /
נחשים צפעונים. / ויכירה ישפה / ולא נודע החזק
הוא הרפה. / וימהר וירכב סוסה / ויקח כליה
ודגליה ונסה / ויכה את ימימה באחרי החנית אל
החמש / תחת המכה אשר הכתהו אמש, / ומעל הסוס
הפילה ונראה שערה וראשה – והנה היא נערה יפה
/ ובאור השמש מתעטפה. / ותגל בעד הפנים
כשחרים / ויתפרשו עליהם נחשים צפעונים
כעורב שחרים, / ושדיה עומדות כתפוחים /
ויציקוהו בנפלו עליה ויחשבם רמחים. / וירא כי
היא אשה / ותכסהו בושה / ויחר לו כי אמר: עתה
יאמרו היא רמתהו, / ופן יאמרו לו: אשה
גנבתהו.





וישא משלו ויאמר:

לנשים יאתה תרמית, / אחזוה למורשה

ותדמה עיר אחזתן / והיא בלבד כלי אשה.

ונכל נכלה עלי / ופני כסתה בושה –

ולולי כי הדרה רב / ובו לב כל אנוש כבשה

ומצילה מאור חמה, / כמות אהבה קשה!





ויקם ישפה מעליה / ותקם גם היא
בחמה, תמשיך מאורה בין שוליה / ככלה תעדה
כליה. / ותבוא יפיפיה לאדוניה / ותשתחו ותפל
על פניה / ותורד כנהרות מים מעיניה / והיא אומרת
״ואיך גנב מבין זרועי / ידיד שעשועי!״ /
ויגישה אליו ישפה וישם לבו מסבה / וידבר
על לבה.





ותשא משלה ותאמר:

מה נעמה קרבת ידיד / אחר נדוד מה מתקה,

מה ישמחה חושק בבוא / רעיה לאחר רחקה!

מאז גלות מני גביר, / אש הנדוד בי נשקה

וברב דמעות אשטפה / דמעה לדמעה דפקה

מאז נדודו, והלא / רוחי בקרבי נבקה.

מחם לבבי אזעקה / על חומסי שן אחרקה,

עד כי אמתך רדפה / אחר אהובה דלקה.

מהר להר אתהלכה, / ״דוד, דוד!״ במרה אזעקה

שבה שבות דודה וזאת / כל מוסרותיו נתקה.

אמה ימימה כתתה / היום בחרבו בתקה,

אמה יפיפיה לזאת / בכתה והיום צחקה.





וישפה מדבר ומשחק עם יפיפיה – /
ומנגד ימימה צופיה / בנפש מרה הומיה ובוכיה /
היא נפעמה נכלמה / וכרחל לפני גוזזיה נאלמה, /
ויערבו הדמעות אשר על לחיה נגרו / במי החן
אשר בפניה ואלה מאלה לא נכרו, / ותקם אליה
יפיפיה ותאמר אליה ״קומי ולהשתחוות לפני
אדונינו החישי / ואל תכלמי כי לא תבושי! / ולמה
תבכי ובאדוניך תתבאשי? / ועתה, אחותי,
החרישי / ותרבי לפניו תחנוניך / ותשתחוי לו,
כי הוא אדוניך. / ואם תכלמי לדבר דברים
נפרדים, / קרבי אותם אחד אל אחד והיו לאחדים. /
אולי תוכלי להישיר – / עורי עורי דברי שיר!״ /
ותקם ימימה, / כדמות לבנה בהיותה תמימה.





ותשא משלה ותאמר:

הט, הגביר, אזן לקול / אמה לך מתחננה!

משפט אמת שפטה, ואל / עלי תבקש תואנה.

נפשי כיונה עלתה / אבר ותעוף תשכנה,

עד מצאה בית וקן / בלבבך שם קננה.

ואמרה ״שובי״ ולא / שבה עדן, כי מאנה.

הן אהבתך היא אשר / רשת לנפשי טמנה!

אוחיל ואשא על נדוד, / אצפה בבוא עת חננה!

תחן עלי אמה אשר / נפשה בידך נתנה

ובבור שבי האהבה / כמה פעמים נבחנה.

תגזל ונפש תעשק – / ובאחרית התאוננה.





ויהי טרם כלתה לדבר השירה
והנעימה / ונפש ישפה נקשרה בנפש ימימה /
ויאמר לה ״ראי, שיריך בשפתי ארוגים, / מה יפית
ומה נעמת אהבה בתענוגים!״ / ותשבנה שתיהן
לפניו, וישתעשע במי החן היורד ים על הפנים, /
ופניהם כפלח רמונים / אדמו עצם מפנינים /
ומשלג לבנים. / ונתקבצו שם שלשה מינים: / טל
חרמון / ופלח הרמון / ושלג הלבנון. / עליהם
קוצות תלתלים / ועפעפים כגדילים / ואישוני
עינים משחקים מחללים כבחלילים, / וגבות כקשתות
נטויות / להכעיס חושקים עשויות, / ידמו פניהם /
בפרש עליהם / שערות שחרות כעורב / כמראה הבקר
והערב. / ויהיו שלשתם בנשק וחבק, / וחשק זה
בלב יוחק, / ויאמר ישפה לרעיתו ימימה ״והן אם
אהבתני וחשקתני, / איך גנבתני / ובשבי
נהגתני?״ / ותענהו לאמר כי ״זכר יום קחתך אחותי
יפיפיה / ותבחרה לך מכל צביה / ותהי לך
לרעיה: / שם הבישות כל קוויך / ואותי השלכת
אחרי גוך. / הלא אנכי לכל עין מחמד / ולכל רואי
אין לפני מעמד, / ומראי נאה וקולי יערב, / כי היה
הדרי רב״.





ותשא משלה ותאמר:

הלא כל בעלי חשק / ביפיי יתמהו כי רב!

בפני יחנה שמש / ובם זרח ובם ערב,

ומראי – שעשועי דוד, / וקולי מה מאד יערב!

ואשיר שיר – ולב חושק / באש האהבה נצרב.

ולו נחל בקרבו בא – / בחם לבו אזי חרב,

ואם לבו מקור מים – / כרגע יהיה שרב.

ועיני ילחמו אל דוד / ותמיד הולכים בקרה

ודוד יאמר להלחם / אליהם – החרב נחרב,

וחוג פני – דמות סהר, / ותלתלי – דמות עקרב.

גשה נא, אוהבי, ולכה / לדדי רעיה וקרב,

והא לבי בתוקעי כ/פך, כי חשקך ערב!

ואל תזכר אשר עותה / אמתך לך ביום מארב,

ואל תשמע לקול מריב / אשר על אהבתך רב,

אשר ישך כנחש לב / ידידים, עד אשר זרב,

והרחק כל מריבי דוד / ואל תשב בתוך סרב,

וכל נרגן יהי מכה / בשדפון ובגרב!





וישחק ישפה מהדר שיריה ויאמר
״והנה יפיך הללת – / ובי התעללת!״ / וישפה
היה לבן־שנים / עם יפה עינים / והדור בכל מעניו /
וכמוהו לא היה לפניו.





ותשא משלה ותאמר:

בעת ישחק יגל שניו / מצמדים ואין נפרד

כחלב, או כעין קרח / כשלג ככפור ברד.

המחמר כזה נברא, / המרום זהרו ירד?

לנגה זהרו – נגה / מאור חמה כבר מורד!

הבא ממעדן עדן / והוא משם לבד שרד?

ומשם איך יבואנו? / הלדבר מעלו טרד?

עפרים שחחו תחתיו – / והוא עם זהרי רום רד,

הדר לחים לפניו צד / והיום על פניהם רד,

וכל כותב שמו ״עפר״ / ימהר חיש והתגרד:

הלא כל מחנה יפי / לפניו קם ולו חרד.





ויהי הם משחקים / כמשפט החושקים – /
והנה פרש אחר ומגנו מאדם / וחניתו אדמה מדם, /
מפרק הרים בכחו / ומפיל ארזים ברוחו / ומחריב
ימים בגערתו / ומבקע יערים בזעקתו, / ויצוח
ויצרח / וכל שומעיו לפניו יברח. / ויהי בשמעם
קול ההמלה / ויאמרו ״מי יעלה לנו בתחלה?״ /
וירכב ישפה על סוס ימימה / וילך אליו באף וחמה
/ ותעלה בינותם חרבותם / ועד למרחוק השמיעו
מכותם, / ויקרא הפרש אל ישפה לאמר / ״הנה יצאתי
אל ישפה לבקשו / ולהציל ולהחיות נפשו. / המעט
כי אותו מביתי גנבת – / ולהלחם עמי עתה
נצבת!״ / וישמע ישפה כי משוש הוא ויכירהו /
וירץ אליו ויחבקהו / וישקו איש את רעהו. /
וישאל משוש הסוכן לשלומו ולשלום יפיפיה
אהובתו / ויגד לו ישפה חידתו. / וישמח שמחה
גדולה / כי היה לישפה רוח והצלה, וילבשו איש
מדיו / וילכו שניהם יחדו / ויבואו עד הנערות – /
והנה נבהלות ונסערות, / כי אמרו ״אולי הפרש
אסרו / וטבחו או מכרו״. / וישב משוש עמהם
ויגידו לו קורותם / וכל המוצאות אותם, / וישובו
לעיר אל ביתם / ולא ידע איש בבואם ובצאתם. /
ויהיו ארבעתם אוכלים ושותים וישב ישפה עם
הנערות – והנה מלאכים רוכבים צמדים / ולפניהם
משא צמד פרדים / ויאמרו לישפה ״אביך שלחנו /
וברוך אשר בדרך הנחנו! / והנה לך מנחה / מאת
אביך שלוחה, / ואמנם אתה איש לא תחמד / בכסף
ובזהב הרבה מאד, / ושא שלום מאת רעך שר
הצבאות / וגם הוא שלח אליך מנאות, / והא לך כתבו /
ובו תראה כל לבבו, / והנה בו שיר בשורה / כי
לנגידים היתה אורה״. / ויקרא כל דברי חמדת /
פתשגן כתב הדת:





שמח, דודי, והתבשר – / זמני שב למתכנתו

ושב נחל למקום גר / וכל מחצב למחצבתו,

ואיש איש על עבודתו / ומשכנו ומצבתו,

ונשיא שב למשאו / וכל מושל לממשלתו,

ושב הלך למהלכו / וכל מלך לממלכתו,

ודיג שב אלי חרמו / וכל ציד למכמרתו,

ורגלי יהלך רגל / והרוכב למרכבתו.

בליעל בלי יעל / ושח שפל עדי רדתו

ושב עול בראש עול / וצד נלכד במלכדתו,

ובוש סוגד לאיש בוגד / וישתחו עדי גשתו

ויאמר על הרג איש תם / ״מברך את אשר משתו!״

כבר נמס אנוש חומס / וגיל ושמח למפלתו.

הריקם הזמן נקב / בתוך מצחם במקבתו,

אבל ישפה בחיר דודים – / בריתי היתה אתו.





וישמחו שמחה גדולה ותרועה הריעו /
על אבד בני בליעל אשר הרעו. / ויאמר ישפה ״הדור
הדוד אשר בשרנו זאת הבשורה, / לו משפט
הבכורה! / והנה בשירותיו ימלא שחוק / פינו, טוב
קרוב מרחוק״.





וישא משלו ויאמר:

ברית דוד נאמן עמכן / אמונה מברית אחים

ואף כי אח ודוד יחד / כמו הסיר במרקחים,

במדבריו צרי צירים, / יגונות כואבים דוחים.





ויאמר משוש והנערות / ״הנה אמרותיך
מה נברות! / אם מפרדות – נלקט מפנינים / ואם
מחברות – ענק נקשר מצורונים״.





וישא משלו ויאמר:

מה נמלצו מליו לחך! / לשם לקט, אם נפרדו,

שימם רביד זהב עלי / צואר בעת יתאחדו,

כתבם במר בכתב כעין / לחיו, ודודים ילמדו

מה מתקו, מה נעמו, / יפו כפז, מה נחמדו!





ויאמר ישפה למשוש ״ראה כי הזמן
בעת רצון אלינו פנה / ואויבינו וגאונם בפניהם
ענה / ובגדי כלאנו שנה / ולחפשי שלחנו תחת
אשר ענה, / ועתה אמצא חן בעיניך ועשה כל חפצי /
ואלכה אל מקומי ואל ארצי״. / ויחרד משוש חרדה /
וירעד רעדה / והוא עולה ויורד / ויאמר ״איך מעלי
תפרד! / תמול כרתנו ברית אהבה – / והיום בפרוד
נפשי דאבה״.





וישא משלו ויאמר:

ברית אהב מאד תנעם / אבל פרוד יפרנה,

נות דוד נעמה, לולי / אשר פרוד יגורנה,

ומי ראה ידידות – אין / פרידה מי ישורנה.

פרידה אהבה תמית – / וקרבת דוד תחינה,

פרידה תחשיך נפשי – / וקרבת דוד תאירנה.

תשמח אהבה נפשי – / ופרוד דוד יצרנה,

וליל תשכן דאבה בי – / תמונתו תסירנה.

ונפשי אהבה תבחר / בפז רב לא תמירנה,

וככסף תבקשנה, / כמטמונים תתורנה,

ונפש אין ידיד תומת / ורגל כל תזורנה.





וינחם עין ישפה על משוש הנער /
ורעיוניו סערו סער / וישבע לאמר ״בחי אהב ובחי
נפשך, דודי, / אם אפרד מעליך בעודי! / הנה נא לי
אחות קטנה / יפה כלבנה / ברה כחמה / אם טוב
בעיניך – תהי לך לאמה. / ושמה צפור / על כן עמה
תשב ותצפר״. / ויקד הנער וישתחו / ויריעו
כלם וישמחו. / ויקם משוש ממקומו / להודיע
לאביו ולאמו, / ויראו כי אחות ישפה אהב /
ויתנו לו כסף וזהב / ויביאו לו מנחה מכל אפסים /
ויעשו לו מרכבה וסוסים. / ויאמר לנערות ולשאר
אנשים / ״עברו לפנינו חמושים / ואנחנו נחלץ
חושים״. / ותעלה ותצא מרכבה ממצרים, וילכו כל
הדרך בשלוה / ובשלום בשחוק ובחדוה / עד בואם
חצר סוסה / ועין הזמן על נדיבה חסה. / ויצאו
לקראתו אביו ואמו / וכל אנשי מקומו / וישאלו
שלומו. / ויראו יפי משוש והדרתו / וירוצו
לקראתו / וינשקוהו ויחבקוהו / ועל לבם
הדביקוהו, / ויבואו לבית שאנן / כזית רענן, /
ויעש ישפה משתה גדול לרחוק ולקרוב / ויזבח
שור ומריא וצאן לרב. / ויתן אחותו למשוש
ויאהבה / ושנה תמימה לא מש מסביבה. / וישב
ישפה עם הנערות / כשני המאורות, / פעם מנגנות
ופעם מספרות / משלים ושירות / וחידות ותורות /
ומדות וקורות / ארכות וקצרות. / שתיהן חוברות /
אבל לא כצרות / נכונות ישרות / טהורות וברות / ואהב
מעירות / ואש דוד מאירות, / בלא ריב וקנאה / ואיבה
ושנאה, / מאין חטאה / ולא כל שגיאה, / אבל הן
כאחיות / לשמש שניות / ושוקטות שלוות. / ומתן
מניות / ומשלוח בגדים / פארים חמודים / ומיני
נרדים – / כחק הידידים, / ומכתב דברים / בחמדת
אמרים / בהלל ושירים – / כמשפט חברים. /
ותלמדנה לצפור אשת משוש דברי אהבה / ויכינו



בה רוח נדיבה / במשלי צבאות / שמנות בריאות /
ונשים מקנאות / ומנחת קנאות / אהובות שנואות /
ודברי תנואות / הולכות ובאות / ואמרים ערבים /
מתוקים וטובים / בלבות נדיבים / ודברי אהבים. /
ותרא צפור כי שכלן / למלכות הוכן.





ותשא משלה ותאמר:

הבינותן אמת סכלות / עדי למדה מעט מזער,

ולב לא בן ברב חידות / ומשל, הוא כסיל בער,

ועת חידות יחו נבון – / בלא חק פיו אויל יפער.

שפכתן רוחכן עלי / ורוחי אהבה תער:

לבבי רך צעיר ימים / משוח חן ובן נער

וקדם אוהבים שנא / וחצנו מחשק נער

ופתח אהבה לא בא / ונלאה ממצא שער

ולא נסה חבק ונשק – / ועתה יסערה סער!

והן היום ברב חשקו / למות נפשו יער

ובא כיום נתיב חשק / ובו אש אהבה תבער,

עדי שלח בעיר חשקו / ובשדה חשקו בער

ומצפון אהבה חפש / ומצות עם שאור בער,

וחשק כה תכונתו / והיה כן כמו שער.

ומי חושק אשר ערך / אלי חשקי ולא נבער?

ולא נמצא לבב חושק / כלבי עד גבול שנער!

ולי קרבת ידיד לי טוב – / תחי נפשי הלא מצער.

ידיד – אורו כאור חמה / כאזרח רענן יתער,

ולב נשאר בלי חשק – / כקרח בלי שער,

כאיש גלח בלי זקן / ועליו העביר תער,

כאיש נולד בלא פנים / בלי אבר בלי מער,

כאיש ערם בלי מלבוש / ויראה כל יצור מער,

כמשתגע בריר פיהו / וזה חרף וזה יגער,

כפתי אל פתיות יט / אשר נטה לשבת ער.





ותענינה לה ימימה ויפיפיה ותאמרנה / ״רבות בנות עשו חיל – ואת עלית על כלנה!״
