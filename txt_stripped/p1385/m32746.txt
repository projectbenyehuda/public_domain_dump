


	
	תחרות בין משוררים. משוררנו זוכה בתחרות זאת.
	








נאם למואל בן־איתיאל: היינו במושב
חכמים / ובסוד ערומים / והיינו מקלקלים פני
התבונה / מחקרים עמק התבונה / ויען אחד מהם איש
משורר / איש יסודות השיר מצרף ומברר / ושמו
יחיאל בן ירחמיאל / ויאמר: הראיתם שיר ימימה
אהובת ישפה / ומה נעים ומה יפה / על כן על כל
אנשי השיר היא משתררת / כי חנית שירה על חמישה
דמיונות בפעם אחת מעוררת. / ויען האחד אם תעורר
חניתה. / על חמשה דמיונות בפעם אחת / אכן ממנה
דבר לא נכחד. / ואענם, אני נשען על שכלי / ספרו
נא לי / ויאמרו זה נאמר / השיר אם אחד אם שנים /
ויאמרו לי בהלל נא נקראם שנים / והא לך השיר /
ואם תוכל דמיונם להשיר / וחקור מליו ודמיוניהן
ואם תוכל [לתת] רוכבים עליהם.





וישאו משלם ויאמרו:

כחלב או כעין קרח / כשלג ככפור ברד.





ויתמהו כל בעלי השיר על טוב שיחיו /
ויקנאו בו אחיו / כי אמרו הנה אנחנו ממשלים
בחמשה שקלים / והיא שקלתם במשקל אחד ומדה
אחת נכונה / שקצב אחד לכלהנה / וזה מדבר
בקנאה וזה מהלל ביראה / וזה מחריש בשנאה /
וזה יבש גרונו בצמאה / כי תמהו כלם על טוב
השיר ומעניו / כי נבהלו מפניו. / ואען ואמר:
למה נבהלתם / ועל השיר תמהתם / והנה אם השיר
נחמד מזהב ומפז / עבדכם ילך והכה את המשורר
הלז. / ויאמרו לי כמתלהלים / ולמה אנחנו
מתמיהים / אתה תהיה לנו לאלהים, ויסגדו לי כסגוד
למלך / ויאמרו לי אי חנית המלך / והאחד יהתל
בי / והשני ישרתני וילך סביבי / והאחד אומר
במה תכה אנשי השירות / ובעלי אמרות טהורות /
התניעמו בחילך / התורידמו במקלך / התבט אחת או
שתים / התשכיבנו בין שפתים. / ואענם בטוב
מעני / כי לא בחרב ובחנית יושיע יי / ויענוני
לאמר הלא זה עשה כמוה ולמה תתראה / וראה ועשה
כתבניתו אשר אתה מראה / ושמר הענין והדמיון /
ואל תגרע מן הנקיון:





ואשא משלי ואמר:

כצמר או כמו שחר / כדר ציץ או כמו ורד.





ויהי כשמעם שירי החשו / ובאמרם
כחשו / והתבוששו והתקוששו / וכמתגנבים
התלחשו / ויכלמו ולא נשמע קולם / ויפלו
חללים כלם. / ואמר אליהם / והלא אמרתם לי /
בשמעכם משלי / לא תוכל לו / ואמרתם בראותכם
הענין והדמיון / עד מתי תוכל נקיון / הנה
יכלתיו והכיתיו / ותחנה מלה נגד מלה ודמיון
נגד דמיון / כפלגי מים בציון:





משורר אך בכף שירי / ביד מלי אאבד שר

ביום נבחן נגיד שירים / הלא אז מעני יכשר

ואלו שר כשירי שר / אזי במשוררים אשר

ואיש יאמר אשר נמצא / כטוב שירי אזי קשר

משורר ישמעה שירי / כאון מצא באיש עשר

ערב מלה יפה ענין / ושר משקל בלב נקשר

למען יחרד שומעו / הלא זה שיר אשר יושר.





ויודו כלם ויאמרו משלם:

לך השיר ולך יאות / והנה זה לך האות

אשר התחברו צבאות / נגידי שיר לך הלאות

נפצתם בהדר שירך / והפלתם למשאות

ומטובי רנניך / כהיום נשאו משאות

עדי אמרו הבמראות / רנניך הנבאות.





בראותי כי בפיהם מהללים / ובקרבם
מקללים / ומקדם היו לאוהבים / ובשמעם שירי
נהפכו לי לאויבים / ואמר הנני משורר בדמיון
ששה / ותרא אויבתי תכסה בושה.





ואשא משלי ואמר:

לששה אוהבי דמה / ובהם כל יצור כבש

כשמש ככרוב כצבי / כאדם ככפור כדבש.





ויאמרו כלם לא שמענו משורר ישיר
ששה דמיונים במשקלי משקליו / אף כי זה
המשקל [קל] ברגליו / והאחד אומר מקדם שמעתו
/ והאחד אומר גנוב הוא אתו / למה תתהלל בשיר
אשר לא חברתו / אשר לא עמלת בו ולא גדלתו. /
ויען יחיאל המשורר ויאמר: ישים שיר אחר לצדו /
ויעשה לו עזר כנגדו / ובזה יודע כי הוא שירו /
והשיר השני יקים את חברו.





ואשא משלי ואמר:

כעש כעב כמור כארי / כהימן כצרי יחבש.





ויענו כל העם יחדו ויאמרו: לך
השיר ואותו חברת / וטוב הדבר אשר דברת /
אבל באר תבאר השיר האחרון / בחכמה ודעת
וכשרון.





ואשא משלי ואמר:

כעש נעלה כעב נדיב / כריח מור כחיל אריה

כמו הימן בחכמתו / במראהו שחין יחיה.





ויהי ככלותי מהשיר ואקשיב והנם
תהלתי יחוו / ויקדו וישתחוו / ואמר אליהם /
הנני מוסיף עליהם / משל אחד והיו שבעה / ויריעו
עלי תרועה / ויאמרו לך החכמה / ואתך הערמה /
הבה נתחכמה.





ואשא משלי ואמר:





בשבע אוהבי נעלה / וכמעט בם יהי נביא

כעש כעב כמור כארי / כהימן כצרי כצבי.





הלא איש אתה מי כמוך רב פעלים / הנה
בששה דמיונים על כל יצור תשתרע / ובשבע לא
יגע בך רע / אמנם כל לב יקח מאמרותיך / ישא
מדברותיך. / ויהי כאשר הכינו לבם / אחר אשר
קרבו לבם בארבם / ויען יחיאל המשורר ויאמר:
לך יאות על בעלי השיר להשתרר / והוא מכיר בסוד
השיר ובדמיונים / ויחיאל היה איש גדול לבני
השיר ונשוא פנים. / ואמר אהללנו בששה
דמיונים.





ואשא משלי ואמר:

קל כצבי עז כארי מתוק כצוף / ריחו כמור ידיו כעב מליו צרי

מנוזלי ידיו סלעים דשאו / כפו כעב תמטיר ותטריח ברי

כפיו כשרש וענפיו אצבעו/תיו על אגם מים ולא ימיש פרי

דמה בענותו כגבר אין איל / וביום קרב עם לוחמיו נמצא ארי

כל איש אשר ממרה אמרי בו הלא / הוא איש גאיון איש מדנים איש מרי.
