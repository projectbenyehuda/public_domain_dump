


	
	מעשה בעכבור הזקן
	












נאם למואל בן־איתיאל: הייתי מנעורי
מתהלך בחוצות וסוחר / מגוי אל גוי ומממלכה אל
עם אחר / פעם אלין במדבר וביערים / פעם בבתי
החצרים / כמשפט הולכי על דרך והולכי נתיבות /
פעם בחוץ פעם ברחובות / ולילה אהי ישן / על
מטות שן / ושוכבים על מטותם / וסרוחים על ערשותם
/ ולילה סוף לראשם חבוש / וערום ילינו מבלי לבוש,
/ ויום יהלך יעף ונכה רגלים / אין לחם ואין מים, /
כזאת וכזאת יקרה לעוברי דרכים / פעם עדינים, פעם
נבוכים / פעם כנבלים ופעם כמלכים.

ואשא משלי ואמר:

דבר אדם בלי יעמד / בעינו, יד ליד ישנא

ועת יקום ועת ישב / ועת יסע ועת יחנה

ועת יגדל ועת ידל / ועת יעשיר ועת יענה

ועת ישחק ועת יזעק / ועת יהרס ועת יבנה

ועת יגיל ועת יחיל / ועת יצדק ועת יונה

ושחר יעבד צורו – / וערב לא לאל יפנה,

ויש נשען עלי הונו – / והוא על בעלו יזנה,

ויש נשען בהוד תבל – / ומשענתו רצוץ קנה,

ויש ייגע להעשיר, אך / ללבו מחלה יקנה.

דבר איש כל ימי חייו / היספר הימנה?

ומשכרתו בעת ועת / עשרה יחליף מנה,

תבקשו אחרי כל זאת / והן אין קול ואין עונה!





וראיתי נפלאות / גדלות ונוראות /
ופגעתי אנשים ורשים / נבלים ונדיבים. / [ובאתי
לעיר] יושביה עשירים / וסוחריה שרים / ועל זהב לא
יחמלו / וכסף בקנה ישקלו / ותישר המדינה
בעיני / ויברחו וילכו כל יגוני / ואמר כאשר
ראיתיה: / פה אשב כי אויתיה.





ואשא משלי ואמר:

בחר־נא לך שכון ארץ / שכוניה נדיבי עם

עשירי הון שמחים בו, / מתי שכל וטוב טעם

בטוב לב תפגשם תמיד, / באור פנים בלי זעם

ואם בא איד לאיש מהם – / יבואוהו בלב נפעם

בעם זה תשכנה בטח / ובינם נפשך תנעם.





ואשכים בבקר לראות אנשיה /
ויושביה ורחובותיה ומגרשיה / והנה כלם אנשי
מרמה / וילידי אשמה. / ואמר בראותי רשעם: / אלכה
לי אל הגדולים ואדברה עמם / כי המה יורו חוקות
ותורות / אך המה שברו על נתקו מוסרות. / ואראה
והנה שופטיהם שופטי שקר / זאבי ערב לא גרמו
לבקר / ויתלקטו [אליהם] כל טחי תפל / וכל אשר
שקר טופל / ואיש באחיהו בגד ופשע / וצדיק מט
לפני רשע / וכעגלה דשה יפושון / ולמוכיח
בשער יקושון / ולא נמצא כי אם רשע וחוטא – /
והעיר מלאה מטה.





ואשא משלי ואמר:

כמה אהלך נתיבות, / אסב ואדרך דרכים

קצוי אדמה אמודד, / אבוא קצות מהלכים

ארחיק נדוד ביערים, / אקיף שבילים ארכים!

פעם בנפש חרדה / וברעיונים נבוכים

פעם חברי נבלים, / פעם חברי נסיכים

פעם מלוני באשפה, / פעם בבתים ערוכים

פעם אמשש בלי נר, / אתעה ואלך חשכים

לדרש ולתור מקומות / אישים בשכל סמוכים.

אנשי אמת אהבי טוב / משוא ומחטא חשוכים

הן אחרי משאלותי / הם מאויי משוכים

ואפגשה עם חשבתים / זכים ובתום תמוכים –

הן תאותך לבבי – / היא עיר סגלת מלכים

הן אם זמן ינשכני, / פה יחיו הנשוכים.

ואחקרה – והנה / פה מחשבות הפוכים

כי היא מלאה מתי שוא / מזד ומאיש תככים

ותעלה קמשונים / כלה כסירים סבוכים

עובדי מלכת מרומים / ויסכו לה נסכים

כל מאכלם פתנים, / יינם חמסים מסוכים

עשק וגזל בידם / נמצא ודמים שפוכים

חלקם בארץ יקלל, / יהיו כרגע הדוכים

יאבדו מהרה ויהיו / נרות מאורם דעוכים

רמחי זרועות נטויות / יהיו בלבם מעוכים!

ומתי אמונה וצדק / יהיו לצורם ברוכים



וירננו אז בטוב לב / ה׳ צועקים מר ובוכים.



אלך אבקש שכן טוב / תוך עם ומנערים חנוכים

עם ישפטו דל ויתום, / כרש כעשיר כמכים

לא יהדרו דל בריב, לא / ישאו פני דל ודכים

צופים רצון בוראם ול/חסדו מקוים וחוכים

יושבים בסתר שלומים / בכנף חנינה סכוכים

לא ייראו יום חרון אף, / כי אף וחרון שכוכים.





ואמר: כי ראיתי מטי משפט לוקחי שוחד
/ עתה אספה יום אחד / כי דרכם משוכת חדק
וחתחתים. / אין לי טוב כי אם המלט אל ארץ
פלשתים.

אני טרם אכלה לדבר אל לבי / ואשמע
קול שאון וקול מלחמה / ואמר: מדוע קול הקריה
הומה? / והנה איש זקן [יוצא] אל רחוב העיר אל
המעגלה / ויצעקו מכל חוגר חגורה ומעלה. / הוא
מתנהל לאט לרגלו / ונשען על מקלו / ורוח לא יבא
בין פשע לפשע / כאיש חף תם בלי פשע. / ואמר:
עתה תצמח צדקה ויכלה הרשע / תפתח הארץ ויפרה
ישע.

ושמו עכבור / מהר תבור, / ולו זקן
וענפים יוצאים ממנה / ויחצו הנה והנה / כפרש
מברחיה ואגפיה / ואברותיה וכנפיה / שם ירעה
עגל ושם ירבץ וכלה סעיפיה / ויתמהו במראיה /
ויבהלון כל רואיה / ותבא כל העדה לחזותה
ותקהל / ותבא אליה ותלא ותגע עדיה ותבהל. /
ובכבדותה ילך שחוח / ולא עצר כח / וילך הלך
בין צבאו / רובץ תחת משאו / כי העם אחריו כחול
אשר לא ימד / עד בואו אל הרחוב אשר העם שם נצמד /
וכל הבא אליו ועמד.&gt;
ואביט והנה מציב לו יד, / כי הכינו
ברחוב כסא / אשר יעלה שם ביום המעשה. / וילך
ויעמד כמלאך / ועל משרתו היה נסמך./ ואמר:
עתה ייטיב [עמכם] / אשר מלאך שלח לכם / והיה
מכין על כתפו לבושו / וישרק ויניד בראשו /
ואבהל ואשתאה / ואשתומם על המראה.





ואשא משלי ואמר:

אם זקנו זקן אמת / משנת חטא יעוררם

אם פשעים כחול, הלא / זה בצדקו יכפרם

סררו או יעותו / זה כבנים ייסרם

נפרדו או יפזרו / זה כרועה יפקדם

העוו או יעותו / בסליחה יבשרם,

בחטאים נגאלו / אל בלב־ים ינערם

כי בסכו יחבאם / גם בסתרו יסתרם.





וכל העם סביביו מהם מדברים ומהם
מחרישים / מהם מצפצפים ומהם מתלחשים. / ויקרא
לנער משרתו ופקידו / אשר עכבור נשען על ידו. /
ויאמר בגערה: ״אם לצפצף או להתלחש היתה
קבצתכם / פנו וסעו לכם!״ / ויהי כראותם כי
התקצף מאד / עמדו ולא ענו עוד. / ויפתח שפתיו
עכבור במלות נכוחות / ודברי תוכחות / וישא קולו
/ וירם ימינו ושמאלו. / ויאמר [בלשון] רכה /
ובעין מבכה: / ״אשרי האיש אשר לבו זכה / ודאג
מחטאתו ומעוונותיו מדכא / וישב בסתר עליון
ולרחמיו חכה / וישם לו שם סכה!





וישא משלו ויאמר:

לבי פני צור יחלה, / תמיד ישוע למלכו

יערב שמו על לשון איש / זכרו דבש צוף לחכו.

אשרק לחרב וזרם, / כי יצפנני בסכו,

קוו לישעו וחכו – / אשרי אנוש המחכו!





אבאר אמרי בתחלה / בהלל לו נאוה
תהלה / ושמים וכוכבים ברא במלה / והארץ [על]
בלימה תלה / והאדם עליה ברא להיות לו סגלה. / על
כן שמעו נא תהלה! / הטי נא אזנך, [עדתי],
ושמעי והוסרי והכנעי / ודרכי בית תפלה /
ושיתי לבך אל המסלה. / והטהרי והתבררי /
והשמרי והזהרי, / פן ישיאוך בעלי זקן חרפות
ומרדים / שדופות קדים / ואבליגה. ואל יבהלוך
אנשי כחש וכחד / עולות בקנה אחד / המפתים
בדברי ספר ומלה / וירפו את שבר בת עמי על
נקלה / אשר המה נאנקים ונאנחים / ויחזו לו
משאות שוא ומדוחים. / הלא ידעתם אשר נאמר על
זקן חרפות / אשר מקדים שדופות: / ושפטתיך
משפטי נואפות!





וישא משלו ויאמר:

זקן ושיבה אל יבהלוך / אל תאמן בם עד אשר תבחן,

כי רב פעמים תחשב למצוא / חטה – ותמצא חוח ולא יוחן.

כן תחשב איש מבני חסד – / הנו פני עני ודל יטחן,

יתאב אכל פת דל, ככלב שב / אל קיא ומקיא יערך שלחן.

אל תאמרו יצלח וימלט, / כי לא יכפר לו ולא יוחן!





על כן תשפילי גאונך / והרחיקי
זדונך / ואל תתני על בצעך עוונך / ואל תבטחי
בהונך / כי הונך מונך / ובצעך בוצעך / הלא ביום
פקדת עוונך / הרבה ירבה עצבונך. / קחי מצוה /
והרחיקי גאוה / ואל תלכי אחרי התאוה / פן
תורישכם צרבת המכוה / כבסו מרעה לבכם / שמעו
ותחי נפשכם / עד מתי יצעק עשוקכם / וילון
גזולכם / אם על יתום תפילו ותכרו על ריעכם? /
שמעי נא העדה הקבוצה, / לכי איעצך נא עצה /
וכפרי עוונך בהמונך / ופרי קלונך בהונך, / כי
אם העניק תעניקי לעני / בצדקה תכונני / ואם
פרש תפרש לרעב לחמך / יברך [יי] את לחמך ואת
מימיך. / ובדל ובאביון אל תגער / ואל תדכא עני
בשער / ואל ישב דך נכלם / מבשרך אל תתעלם, /
פתח תפתח לעניך אשר בקרבך / והעניק תעניק
לו מגרנך ומיקבך, / ואל נא בראש דלים תשאף /
מתן בסתר יכפה אף.





וישא משלו ויאמר:

ואם תרצה מצא מאה / לכל בצע – שחד עם אל

לעני העניק תעניק / פתח ידך לדל שואל

ויאמר אל: ברכה עד / בלי די קח לך הואל

ומחטא יפדך, ישא / פשעיך ותגאל.





ואני משביעכם בסתר עליון / שתרחמו
עני ואביון / ואל תבזו סתר אל / עת ישביעכם בו
שואל – / משל העני ששאל על דלת / לפתי מכלת. /
ויתן לו בעל הבית פרוטה / וישביעהו בסתר
עליון לאמר ״הוסף לי מעט חטה / כי יש לי נערים
קטנים / ברעב ובצמא מענים.״ / אזי בעל הבית
בחצרו פשע שקר / וכפר בעקר / ויאמר: ״ואיך
תשביעני בסתר עליון? / הלא תראה כי הנני עני
ואביון?״ / ויהי טרם כלה לדבר / ויאחזוהו צירים
ושבר, / ויכהו בוראו בעורון ובשגעון
ובכליון / ויעבר קול: כה יעשה לבוזי סתר עליון! /
ואתם אחי הוסרו / אולי ביום אף וחרון בסתר
עליון תסתרו.





וישא משלו ויאמר:

אשרי אנוש יעתר ויעתר! / צל בוראו עליו כמו כתר



יבטח ביום חרון ולא יירא, / כי צל ה׳ לו יהי סתר.



מדורכי קשת יהי נצל / יום כוננו חצם עלי יתר

נצל וימלט בלב ימים / אך מבלי חובל בלי חותר.

איש בן לסתר אל ובוזהו – / ימס בחטאתו מסוס נתר

הנה כיתרון אור עלי חשך / נסתר עלי נחש יהי יתר.

אחלם עלי שבתי בסתר אל / אך צור ישועתי יהי פותר

עם רחמיו אכרת ברית עגל / אכרת בתוך אעבר בתר

ואהללה ואזמרה, אמר: / אשרי אנוש יעתר ויעתר!





וידבר כל דבר תמים ונבר / ולא ענו
אותו דבר. / ויאמר להם: ״אחי, למה תבקשו בצע
ולמה תכבדו לבכם? / הלא שמעתם משל שני
אנשים שהיו באחד מגרש / אחד עשיר ואחד רש. /
ויהי העשיר סוחר / בכל יום היה משחר, / ויענה
הרש נפשו בצום / שלשה ימים לילה ויום / וכמעט
לא נותרה בו נשמה / ברעב ובצמא.
ויהי ברביעי כי צר לו / ויקח בידו
מקלו / ויצא ברב דאבות / נשען על חסדי רוכב
ערבות / עד אשר [בא] לאיש הסוחר / ויאמר לו: ״סעד
נא לבב עבדך ואל תאחר!״ / ויגער בו ויחשבהו
לאין / כי היה מתרונן מיין. / ויאמר: ״ואיך תבוא
אלי בשחר?״ / ויענהו: ״כי היה לבבי סחרחר״. /
ויהי לרגליו סוגד / וילך וישב לו מנגד. /
ויגער בו שנית ויאמר לו: ״הלא אתה מבקש
קלונך, / ידעתי את זדונך! / מה תעמד ומה תחשב /
ועל מה אתה חושב?״ / ויענהו האורח: ״העל המונך
אני בוטח? / הלא אני עומד על הופך לבבות / ומפר
מחשבות / הוא יהפך לבך״. / כשמוע הסוחר /
ויהפך לו אלהים לב אחר. / ולו היה אוהב ושמו
פקידו / וכל אשר יש לו נתן בידו / ויכתב לו לתת
ארבעה שקלים לרש / ותתע ידו ולבו נגרש /
ויחשב לכתב לו: תן־לו ארבעה שקלים ואל תוסף /
ויכתב: תן־לו ארבע מאות שקל כסף. / ויקח הרש
הספר מידו / ויבא אל פקידו / וישקל לו הכסף
וישם בסדיניו / ויבא אדוניו. / ויבא הסוחר הכסף
והכתב דרש / ויאמר לעני: ״הלא את [אשר]
יורישך אלהיך אותו תירש / והוא התעה את ידי / לתת
לך את הוני והודי / והוא המעשיר והמוריש /
והנוטע והמשריש / עוצר אמונים / ומסיר שפה
לנאמנים. / ולו כל נשא ושפל וכל גאלו שוגג
ומשגה.״





ואדם יחשב נפשו / כלב חכם – והוא משגה,
ויש חכם אשר ידל / ויש סכל אשר ישגה,
ויש צדיק אשר יכשל / ויש מעשיר והוא מוגה,
ויש שואל שאר וכסות / עלי כל הר ועל כל גיא
ואחר ישבעה טוב אל / ברב שלום לרב ידגה
ויש ירון בראשיתו – / בסופו לענה יהגה
אלהים הוא אשר ישפט / ינשא שח ומשפיל גא,
אשר לו כל אנוש חוכה / ולו יוחיל ויעגה.





ויהי ככלותו לחוות המשל / ויקם איש
מזקני המעגל / ויאמר: ״הלא ידעתם זה האיש בן
שמונים / זקן ונשוא פנים / הכלמו מארך זקנו
ומרוב קציריה / אשר בכל פוארותיה רבצו גוריה /
ועתה גמלו חסדים אצלו / ותנו לו איש נזם שללו.״
/ ויהי בעמדו בין המשפתים / וידבר כזה לא אחת
ולא שתים / וימס לבב העם ויהי למים. / ויקרא
האיש שני נערי עכבור / וילכו אל העם כסף לצבר
/ ויהפך לבב העם ורך ונחת / ויתנו לו איש
קשיטה אחת / ואיש בגדו ורדידו / כאשר תשיג
ידו. / וישאו שני נעריו כספם וזהבם /
וישתחו עכבור ויברכם ויאמר אליהם: ״אמרתי
לכם כי לא שאלתי מאתכם / כי אם את נדבת
לבבכם.״





וישא משלו ויאמר:

בורא לבבות הוא אשר / זכים לחפצו יהפכם

הכביד והקשה חזקם, / גם אל רצונו ימשכם,

יקשם יכינם, כאשר / יקרא למי ים ישפכם –

כן נפשכם עמי וכן / הפקד בידי רוחכם.





כן יהי רעוה שיטה לרצונו לבבכם /
ויכון לחפצו תאותכם / ולכן יחכה לחננכם /
וירום לרחמכם / ויצרור בצרור החיים נפשכם.״ /
וישתחו להם עכבור / וילך ויעבר, / והעם
התפזרו מעליו / וילך איש לאהליו.



ואמר בלבי: [אלך] אחר הזקן הזה /
ואדעה אם נביא הוא או חוזה / אם יקר או נבזה. /
וילך עכבור עם שני נעריו / הוא הולך ואני אחריו
/ ויבא אל ביתו ונשל נעלו / והשליך מקלו / וכליו
שם באהלו. / ולביתו עשרה מבואות / כבתי שרי
צבאות. / הייתי מספר צורותיו / וחצרותיו וטירותיו /
לולא כי קצרה לשוני / לספר ראות עיני. / ומי יוכל
לספר המים והברכה / והעוף והשבכה / ומיני
העצים / והשושנים והצצים / והכפות והמחתות
/ וכלי היין לשתות / ומיני בגדים לתלבושת /
וכלי כסף וזהב ונחשת? / ואמר בלבי: המשאלתך
מנבלים או מנדיבים / עשה עשר ונכסים רבים. / הלא
יכלם לשאל מיקר ונבזה / ועשה לו כל החיל הזה /
ואדע כי הוא מבני הנבלים / והאיש קשה ורע
מעללים.



ויש עושה נכסים מחמסים / והתעשר בדמעת העשוקים

ויש אישים בהוד שיבה וזקן – / וטוב מהם בקיר אנשי מחקים

ויש זקן הדורות שערותיו – / וטוב מהם במו זפת דבוקים

ויש זקן ארוכה – ואין ארוכה / בחטאתה, שכניה נמקים.

במחיהם הלא ישקו זקנם / וראשיהם במו מוחם בקוקים

ויש אישים בעיניהם חכמים / ומחכמה הלא המה רחוקים

ויש אישים כחול ימים כבדים / ותחתיהם מתי שכל מעיקים

ויש אישים בצד חכם לשיכים / וכצנינים בעיניו הם מציקים

ויש זקן ובעלה איש תככים / ונכסי דל לחכו ממתקים

ומחיתו גזלת דל, ויעשיר / ויבלע כל שאור מצות רקיקים.

ולא זכרו למקשיב צעקת דל / ושומע לשועת נאנקים

אשר היו ממלאים שק בגזל / למחר אמתחותיהם מריקים.





ויהי עד כה ועד כה – ותצאנה ארבע
נערות / כירח הולכות יקרות / ותשקנה בידיו /
ותפשיטנה בגדיו. / וישבו על השלחן וישימו



לפניו כל מאכליו / ופרי מעדניו / וכפות וגביעים /
מקלעות פקעים / ויין ישן / וציצים ושושן / ומים
מהלכים / וסגלת מלכים / וישב עכבור על מסבו



ויהי כטוב לבו / ויאמר לנערותיו: ״התעוררנה /
ושיר דברנה!״ / ובידיהן כנורות ונבלים /
ומנענעים וצלצלים.





ותען האחת ותאמר:

יין כעין ברק כעין להב / ידליק לבב שותיו בכידודו, /

ימר, ואכן מדבש ימתק, / ריחו יחיה כי כמר הודו.

אם ימצאה יגון בלב כל איש, / ישלח גדוד משוש להשמידו,

ישית לבב כילי כלב נדיב / כי בו קפוץ יד יפתחה ידו.





ותען השנית ותאמר:

יין אשר יאיר בעד כוסות / דמו אלי מים אשר קפאו

לא נסתרה עינו בעד הכוס / ושביב בעד סתר הלא תראו.



ואמרה עת נפלאו מהם / סוד כוס ואור [יינה] ובם נלאו:



כאשר יאירון כוכבי חמה / ובעד שחקים הם יחבאו.





ותען השלישית ותאמר:

ועפרה כסתה לחיה בשני / בידה כוס כאילו דם חללים

ומאדם שתיהן אדמו כל / פני רואים ומלבושים וכלים

עדי ליל מרב מאור יכסה / מאורי רום פני אורם מחלים

ושית עפרה וכוסה באשון [ליל] / אזי יאור ונסו הצללים.





ותען הרביעית ותאמר:

ובת כרם בעת תורק בדמעות / צבית חן בהתנודד ידידה



דאגות תהרג רגע, כאלו / להשמיד כל [כאב] חרב בידה.



ולב נמס כלב אריה תשיתו, / ותעיר אהבת דודים בהודה.

ואם תאמר: ואיך אורה ינוצץ / בעד הכוס ונגלה כל יסודה?

הלא תראה יפת גן יעלה תוך / רדידה – כן חשב הכוס רדידה.

בכן קומה, הלא זה עת שתיה / והתענג, ידיד, מזיו כבודה!





ויאמר עכבור: ״והנה בפיכן הטיבותן /

ובידיכן מלאתן / ובטוב שיריכן / הרביתן חלליכן. /

וכאשר העירותן חשקי / אענה אף אני חלקי.״





וישא משלו ויאמר:

כוסות בדלח מבלי יין / צחות לבנות הן מזקקות

יתמלאון דמעת כרמים, אז / אראם אדמדמות ירקרקות,

וירקדו תוכן בני רשף, / יתמלאו כמאזרי זיקות

התבוששו או נכלמו או איך / לא עין מלאות היא כעין רקות?

כן חק צבאות עת פגש דודים / יתאדמו פנים וגם רקות:

יתאדמו מדם לבב חשקים, / כי לב ידידים גזלות עשקות.

קומי, צבית חן, והשקיני / כסות גדולות אחרי רקות!

מי־יתנה כוסי שפת רעיה / כי מתקה מכל כלי השקות!

כי־כן תעודת חשקים לשתות / בשפת צבית חן וכן חקות.





ותאמרנה לו הנערות: ״הטיבות ושיריך
משירים נחמדים / רבים ונכבדים!״ / ויקם אליהן /
וישחק עמהן / ויהי כמתנפל מחיק אל חיק /
ויקומו לשחק / ובשמחת לבו היה מרקד / כי היה
כאש יוקד / עד כי יגע ושכב וסגר עפעפיו / וכסה
אפיו / וירדם בתרדמות. ותבואנה הנערות בחצר
הפנימית – ובכל זאת אינו רואה / מחריש ויהי בלכתן
ויקרא חרישית / ותצא מהחדר אשה כלה כושית, /
ולה שפה / כאוד מוצל משריפה / ועיניה כרשפים /
ואפיה שקופים. / ואמר ״לכל ניאופים!״ / וארגז
לרוע מעלליהם / ואכם ואפשיט לבושם מעליהם /
ואעזבם ערמים ושוללים / כי קנאתי בהוללים. /
ויאמר עכבור: ״הרף ידיך! / למה תעשה כה לעבדיך?
/ כי אין לב לבוערים / בתאנים השוערים״.





וישא משלו ויאמר:

יום יחלו מתני אזי / סוד על תאנים אערים

אך שעריהן לי צרי – / למדו רפואות, בערים!





ואגער על סכלו ועל מעלו / ויוסף שאת משלו:

מה תמאסו כושית ועינכם לא / תכון בלי הדרת שחרחרת

אבחר שחרחרת ואל ברה / כי אחשבנה לי לבהרת



כתמו בשוק צלמה וחתולה / אל יד שערים או לפי קרת.





ואענהו על חסרון דעתו: / ״ענה כסיל כאולתו!״

מה לשחרחרת אשר תבאש / ולנערה במור מקטרת?

ברה כשושנה ביד חושק / מחמד לכל עין ותפארת

כי כל שחורה היא בלי לבב / פותה והומיה וסוררת

אמר לכל כושית דבר סתר: / שובי, לכי במקום אשר נבראת!





ואמר לו: ״מה לך, עכבור, / לשתות מי
שיחור?״ / ויענני לאמר: / ״הלא כל אוהבי נערות ברות
/ רוכבי אתונות צחורות!״ / ואצעק צעקה מרה / ותבאנה
הנערות כי הרימותי קולי ואקרא. / ואגיד להן דבר
עכבור וגדפוהו / ובחרפות גדולות חרפוהו /
ותלעגנה על שערות זקנו אשר ארכו והלבינו /
ולא ידעו ולא הבינו, / והברה כנגדו כשחורה /
והשחורה כאורה, / ותאותו מהשחורות מי ישיבנה
/ כי אפלות הנה.





ותאמר האחת: ״מרוח אשך / חתר בחשך!
/ והנה ראיתי זקנו כי נתנה בלב רואיה מורא וחתית
/ ותגע עד שאול תחתית / וענפיה רעננים / מחסה
לשפנים, / בין פארה ופארה מחבוא / שועלים הלכו
בו / ופרעושים שם נרדמים, / כי ארכו להם שם
הימים.״





ותשא משלה ותאמר:

זקן עכבור – זקן רועי אוילים / סעיפיה לכל עבר ופנים,

וארכה עד אשר נגעה שאולה / ברב אונים עדי הלאות תאונים.

בתוכה ילכו חיות קטנות / ושאלתי: הקופים הספונים

וראיתי כעוף תוכה יעופון / ופרעושים ירשוה לקנים

השיבוני: זקן עכבור כיער / ושם ציים ושועלים קטנים

ושמה קננה קפוז תמלט / ושמה מחסה אף לשפנים,

ושמה ישכנו בטח והשקט / ואיש תחת גפנים שאננים,

ואיש תחת תאנתו בצל של / זקן עכבור דשנים רעננים.





ותאמר השנית: ״הנה ראיתי עכבור כי
הלבין משיבו / ושער שחר אין בו / ואחשבנו חף
מפשעיו / ובחשך רפד יצועיו / וזנה את בנות חם /
ופעל בפחם. / על כן קראתי על זקנו הנבלה / הוי
מוראה ונגאלה! / ואמנם קדם חשבתיה זקן דעת
וחכמה / והנה זקן בשת וחמה.





ותשא משלה ותאמר:

זקן עכבור חשבתיה / זקן איש מחלה ירפא,

זקן צדק ישר / זקן כבוד זקן חפה,

והנה היא זקן בשת / זקן קלון זקן חרפה

ונאה על שחר פניו / כדמן על פני אשפה.

תגלח אך בלי תער / במו חרש תהי נספה,

ושטן יעמד נגדו / בעיניו השחור יפה,

עדי חשך לאורה שם / ויד יפי וחן רפה

ובחר מק מקום בשם / ותחת החגור נקפה.





ותאמר השלישית: ״הנה ראיתי זקן
עכבור הדורה ויפה / פוריה וענפה, / וחשבתי
שערותיה כשרות ונברות / ותעש בדים ותשלח
פאורות / ותארכנה עד קצה שוליו / ותגע לרגליו. /
והגיע שער לשער ענף / נוגעות כנף אל כנף. /
ואמר: אחשב – ואיך אחשב? – שערות זקנו שערות
דעת וכשרון – והן במורד בית חורון!״





ותשא משלה ותאמר:

ומי גבר יפה אישים כעכבור / כאילו על זקנו הוד מלוכה?

ובו נפתה לבבי כי ראיתיו / ארך זקן – והן אין ארוכה.

חקרתיה והנה היא שתולה / באשפתות עלי דמן מעוכה,

ושרשה על לחי, אולם קצותה / עדי מורד לבית חורון משוכה.

ענפיה לכל עבר פזורים, / וכל פארה לכל פאה נבוכה.

ובה צמר למרדעת פרדים / ומסכה, לגדרת צאן שבכה

שמיכה תהיה לך על אזנך / וצאתך תכסה בשמיכה.





ותאמר הרביעית: ״מי יתן וירחץ אשר בה
חלאתה / והסיר את מראתו בנוצתה! / הלא כי
שערותיה חטאים ורעים – / הבל המה מעשה תעתועים.
/ ואם הזקינה להבל והלבינה כברד, אך אל שאול
תורד.״





ותשא משלה ותאמר:

וראיתי זקן עכבור / ולבי זע וגם חרד

חשבתיהו לאיש תמים / והנה הוא בצור מרד

ואם הזקין זקנו עש, / כשלג שב וכברד

ואם השקה במי חטאת, / הלא אך אל שאול תורד.





ותבואנה ארבעתן והכוהו ומרטוהו /
ומשפט נואפות שפטוהו, / זאת תמשך זקנו וזאת
תמרטנו / וילחצוהו ולא הרפו ממנו / וימת מן
המכים אשר הכוהו / כי המת המיתוהו. / ויאסרוהו
בעבותים / וישליכוהו באחד הפחתים. / ותאמרנה:
״אל ישב איש נאלח עמנו / כי חרפה היא לנו.״



ויהי מימים / ותצאנה אל הכרמים /
לרעות בגנים / וללקט שושנים / מתהלכות
בפרדסים / מתנהלות בין ההדסים / עליהן כתנות
פסים / מספרות משלים וחידות / ושירות חמודות / על
הסמדר והגפנים / ועל כל מיני הנצנים /
ותדברנה על העצים / ועל השושנים והציצים. /
ויהי לעת הצהרים / ותשבנה על בריכות המים /
יושבות תחת הצללים / מנגנות בכינורות
ובנבלים.



והנה ארבעה נערים מקשיבים לקול
רנתן / מאזינים מחולן ונגינתן / משתעשעים לקול
הנגינות / משגיחים מן החלונות. / ותדענה הדבר
ותאמרנה: / ״מי הוא אשר יקשיב לקולינו / ועומד אחר
כתלינו? / הלנו אתה אם לצרינו?״ / ויהי לעת ערב /
ויקומו מן המארב וילכו הלך וקרב / וכאשר
אליהם נלוו / ויקדו וישתחוו / והושיבום איש על
כנו ואיש על מחנהו / ויתמהו איש אל רעהו / על
יפין ומראיהן / ועל זהר פניהן / ויאמרו
כמתלחשים: / ״מי יתן ותהיו לנו לנשים!״



ותשמענה הדבר ותאמרנה: / ״הבינונו
אמריכם / ויבחנו דבריכם! / ובמה יודע כי אמת
דברתם ונדעה כי לא מרגלים אתם? / ואיך [באתם]
לגלות ערותנו ולא נכלמתם? ולמה [לא] אמרתם:
אם עד כה בעיניהן נטמינו / עתה נדבר ואחר [כך]
אליהם נגלינו / ואם כה יאמרו: עלו אלינו! –
ועלינו.״





ותשא אחת מהן משלה ותאמר:

החמסו דודים אשר / דלתות עפרות דפקו?

הה נבערו או יכסלו / או נואלו – או יחשקו!

כי אין לדודים בא עלי / עפרה לבד אם עשקו,

לא לעשוקים לעשק / כי אם בפיהם יחלקו.

אם חושקים הם, יתנו / מופת ואות – יום יצדקו.

אם הם חביבי אהבה / איך אין ידיד יתאפקו

כי חושקים תמיד דוי / לב הם להבים נשקו.

מתאוננים מתהוללים, / כי מכאב לב יצעקו,

אם חושקים אתם – ואי / דמעות לחיכם תצקו?

אי תחנוני אוהבים / אי מאמרים מתקו?

אי חושקי עולם אשר / מעל עפרות חזקו,

הנובבים חשקם עלי / לוח לבבם חקקו?

כלו והעלותם בכל / אם גברו גם עתקו.

דודים, בזאת תבחנו / כי לא בריב תתעסקו,

קומו וחושו מהרו / מוסרות נדודים נתקו,

התנחלו איש רעיה / אכן בגורל תחלקו.

כי יעזב איש אב ואם / וביעלת חן ידבקו,

כי אז ימהרו וינחשו, / כל איש באשתו יחזקו,

ויהי נעורים חדשו, / על אוהבים התרפקו!
