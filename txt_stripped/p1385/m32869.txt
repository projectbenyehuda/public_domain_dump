


	
	ויכוח העט והחרב. החרב נוחלת מפלה ושומעת דברי מוסר ופילוסופיה מפי העט.
	








נאם למואל בן־איתיאל. / ויהי דור
הסכלות גבר ורבה / והשיר אבד ונחבא / ומת או
נשבר או נשבה. / והרבה תבל להוליד כל בן
מנבל / בעבור טמאה לחבל. / ואין יתרון לבעל
הלשון / כמשפט הראשון. / והסיר המן מלבושיו /
החמודים והיקרים / ופשט את בגדיו / ולבש בגדים
אחרים.





ואשא משלי ואמר:

פתיות פשתה בחוץ / כנגע פשתה עקבה

וזנתה את בנות סכלות / ועגבה על חסר לבה

וסכל זע וקנא בי / להכיל הונות הרבה

ודקר בחנית בינה / קבתה בחדר קבה

אזי עלתה חמת פתי / וצבאות בערים רבה

ורדף אחרי שכל / והרדיפו עדי נחבא

ולא נודע המת אמנם / נשבר ואם נשבה

ויאמר אל נביא שכל / חדל לך מה לו נבא

עדי מבין ומשכל / אדמה נשארה חרבה.





והשכלים כל בגד יפרמו / וכל מעין
מים יסתמו. / ויהי כן השכל מפניה נחת / מעין
נרפש ומקור משחת.





ואשא משלי ואמר:

רבו גדלו בערים בעם / ויגברו בם סבאי יין

ויסתמו כל מעיני בינה / מלאם ולא נודע מקום עין

עד כי פתיות ברחוב תקרא / נותר חכם־לב אמרו אין

דמי תבונה כי ישועתך / תשמח נאם גבר שתם עין.





ויהי בשנת שלש ותשעים / בהתם
פשעים / ורוח לבשה את הזמן ואת ילדיו / וקדש
הוא ובגדיו / וימלך מלך ושמו מליץ־החוזה. / ויהי
בעיני הנבלים נבזה. / ובני בליעל אמרו: מה
יושיענו זה! ותצלח עליו רוח צח שפאים / ויקם
ויאבד כל זרע הפתאים / ויהי לחכמה למושיע /
ובכל אשר יפנה ירשיע / וישב הנסחף סוחף /
והנדחף דוחף / והעם הנס אל המדבר נהפך
אל־הרודף.





ואשא משלי ואמר:

פתיות האביד מליץ / לפניו כל כסיל נבעת

ולבות בערים כל יום / ימין מליץ משסעת

עדי תמו מתי סכלות / ונדמו מבלי דעת.





ויבן לתבונות / ערי מסכנות / ויחדש
היכלי החכמה ואת מגרשיה / בני האמונה הקים
ואנשיה / ויבן את סדרי התבונה ואת מדרשיה /
ויצאו מנחבאי החכמה ואת נגרשם / מן החורים
אשר נתחבאו שם. / וישב השכל על כסאו ועל
כנו. / ויטע אהלי אפדנו / ויהי כאשר הכינו
השכל והחכמה לבותם / ויפתחו לצמאים מעינתם
/ ויצאו מים חיים ממקורם / ותשת העדה ובעירם
/ כי הפתאים / גם הם ישקו לצבאים / ישברו
פראים צמאם.





ואשא משלי ואמר:

מליצים פצחו רינה לעינת / תבונה נפתחו לשתות צמאים

עדי כי בוערים שתו ולעו / והיו תאוות לבם מביאים

ועל כן היתה לנו למשל / הגם חברי פתאים בנביאים.





ותשמע נפשי לשכל תבונתו / ותצא
לנסותו / לראות החכמה וכבוד עשרה / לעבדה
ולשמרה. / והנה אשה מראש אמנה תשור / על העין
בדרך שור / ותרץ נפשי לקראתה, כי למימי
שכלה תכסף ותתאב / ותרד העינה ותשאב. /
ויתעורר גם לבבי וילך בחוצות התבונה
והרחובות, / לחזק בדק הרסותיה החרבות. / לעת ערב
לעת צאת השאבת / כחק כל לב חכם אשר ימצא כל
דבר נעלם ונפלא / בנשף בערב יום באישון לילה
ואפלה / הלא זה הלבבות, ילטש וסיגיהם ירטשו. /
ויעופפו מחשבותם כמו חרב / וחדו מזאבי



ערב.





וישא משלו ויאמר:

לעת ערב לעת צאת שואבי מי / תבונה ממקור המחשבות

בהתאחד לבבות במליצה / וחדו מזאבי הערבות

בהגלות פליאות המזמה / אליהם אך בעין לב לא בבבות

ולבות בערים הוזים ושוכבים / בשוט לבות בני התערבות

ישוטון לחזות נעם קדושים / ולדרש לאשר דורש לבבות

ואם תראה סכלים ידרשו אל / תדמה ידרשו כי אם באובות.





ועד הימים ההם חכמה לא נבעלה / ועם
איש לא נתפשה / בתולה אשר לא ארשה ולבבי
ישתאה ויתמה / והנה חכמה יוצאת וכדה על
שכמה / וירץ אחריה כאשר המחשב מצאה /
ולחדרי שכלו הביאה / ויחזק בה בצאתה ממקומה
/ ותפשה ושכב עמה.





ואשא משלי ואמר:

לבי שכב את חכמה כי / מצא אותה בשדה מחשב

צעקה ואין מושיע לה מ/נהו וגם אין אמר השב

אמרו זנה אמר אמה את/כנה לי גם כספה מושב

אז ענתה את עמי אזרח / אך זולתך שכיר תושב

חכם יבעל בעלת חן כי / על כסא חן לבו ישב

וכסיל יבעל בת־אל נכר / כי לב סכל במדמנה שב.





ותלך החכמה לבית מדרשה / וילך
עמה אישה / ובלכת לבבי עם החכמה רעיתו / והוא
כחתן יצא מחפתו / מתהלך בין גנותיה ופרדסיה
/ ושושניה והדסיה / אורה ומורה, ומריח טובי
ריחיה / ומשתעשע ברימוניה ובתפוחיה /
וכפתוריה ופרחיה. / וישא עיניו בין ההדסים
והציצים / והנה שני אנשים עברים נצים. / והאחד
גבור חיל. פני אריה פניו. / חגור חרב מצמדת
על־מתניו / יריע אף יצריח / וכל רואיו ממנו יבריח
/ ושאונו כשאון מימי תהום / ישאג ככפירים
וינהם.





ואשא משלי ואמר:



היש גבור ובחור יערך לי / ומי הוא [זה] אשר יעמד לפני



הלא יבהלון שני מלכים / וחיליהם בצאתם מנדני.





וביד האחד קנה / מקולו לא יחת ומהמונו לא יענה.





ואשא משלי ואמר:

אויל שית יד לפה גם החרש כי / אמתתך בפי עטי וקני

ומקדם הלא הגד לך כי / בחרב לא יהושיע.





ויגש בעל החרב אל בעל העטים /
וידבר אתו משפטים / ויאמר: מה העט אשר יתפאר
בין הקנים / והנה עלה כלו כמשונים / והלא נולד
ונמצא בסתר קנה ובצה. ויבאש ריחו ומחו / וינס
לחה / ביום הלקחו / וישוב ירקרק ונקל / והוא לא
פנים קלקל / נפוחות גופות כנאדים / שדופות קדים /
ולולי החרבות / לא היו לקנים לבבות / כי יבואו
מקצתן / וברא אותם בחרבותם / ויפתחו פיהם לשוח
/ ותנח עליהם הרוח / לכתב כהיום עוונות
ושגיונות / על ספר הזכרונות ומכתבי עמל כתבו /
ועל ספרך כלם יכתבו / ואיך אל החרב יאמר סכל /
והוא אל ההרים אכל / ואני בראתיו אחר היות נקלה
ונבזה / לקום עלי לארב כהיום הזה.





וישא משלו ויאמר:

הלחרב כסיל תאמר / ומי פתי כעט סכל

הלא העט והקנה / לחרב נתנו מאכל

הקש היבש ומוץ נדף / לחרב מרטה יאכל

ואיך בעט יכלת כי / להרים בדיו יוכל.





ויען העט ויאמר: בדעת ומליצה / מה זה
מחשיך עצה / ומי יקלל ויעז פניו / דור חרבות
שניו? ואין דעה ואין חכמה / פתיות בל ידעה מה! /
והם אם יקראו העטים, קנים אחר אשר יבאו / על שם
אחיהם קראו / ובסכלותך חשבת נבוב כמשובב /
ואיש נבוב ילבב. / ותקם על הכלים / כאשר הם
נמולים / הלא נחמד ספרו וכתיבתו / בהמולו בשר
ערלתו. / ותחרף שרשי אשר על יארים יצמיח /
מריח מים יפריח / ואם הגדתי לך מולדתך. / אפס
כי לא תהיה תפארתך. / זכר מולדתך ואל תשכח:
/ ברזל מעפר יקח.





ואשא משלי ואמר:

רדי ושבי אלי עפר ואל ת/לכה רומה ואיך חרב תרומם

והיא עפר אבל נתך עפרה / כחם השמש על עפר תחמם.





ויהי כשמע החרב ותקם בחרונה /
ותצא מנדנה ותאמר: / אוי לעטים למעני בהם. /
בעופפי חרב על פניהם. / הכבד ושב בביתך / פן
בחרב אצא לקראתך / ואל תתהלל בהשכל ודעה. /
ולמה תתגרה ברעה / ואם אפתח פה אחריש בדיך /
וחין ערכך ולשונך אדביק אל חכך / היותה אלי
דמית היות אהיה כמוך הנה החרב אדרת מלכים /
ומשרת נסיכים. / ועזרת נבוכים / ונפש מחזיקה
תחיה / ועזר מצריו תהיה / וכל איש אשר
יחגירני לצדו. / אעשה לו עזר כנגדו / ואויבי
מחזיקים, לו ישמעו, / ואותי יעזרו / ואם לא ישמעו
/ בשלח יעברו.





ותשא משלה ותאמר:

אני אצלח לכל דברי גבורות / ומי יעמד פני פני וחיה

אני ישע ביד ראשים ורשים / ושלחתי לנחשלים למחיה

אני יומם לצד אישים וליל / לראשיהם אני לוטה ולויה

ואחות יקראו לי הסגנים / והשרים לקחוני לרעיה

ולי משרה ולי תכון מלוכה / ובי גדר הריסותיה בנויה

ולולי אאבד חיל מזידים / ואיש מכה באיבה בצדיה

אזי מלאו פני תבל מתי שוא / ושפכי דם ואין קץ לגויה.

ומה לעט מאד נקלה ונבזה / אשר עלות לו העליה

ויתפאר ויתהלל בחכמה / והוא מסכן וחכמתו בזויה.





ויען העט ויאמר: די לך כי תעלי, כי
תרמי / האספי אל תערך הרגעי ודמי / ומה
תתהללי בהרג וברצח / האויב תמו חרבות לנצח /
וכי תשימון ארץ נושבת חרבה? / עמדתם על חרבכם
ועשיתם תועבה. / ואין בם מועיל, כי להרבות חללים
/ למען למוג לב ולהרבות מכשולים / ומפיה דם
נקיים הזילה / כי רבים חללים הפילה / ונקי
וצדיק לא החיית / מכשלת גוייך היית / ובעת אשר
פיהו על אויביו יפתח / כקש נחשבו לו תותח /
וכמוץ נדף יהדפנו בניבו / יתן בעפר חרבו /
וקשתו ישבר / וחילים יגבר / וישמידו /
גדודים ושליטם / בסעיף סלע עיטם ואפתחה במשל
פי / אדרכם באפי / עד אשר יכחשו לו כי
הכנעתים / וכתתו חרבותם לאתים ולפי ייחלו
להורות / אלה החקים והתורות / והשואל ישאל
בעטו / ולא ירשיענו בהשפטו / ופי יודיעך מה
שהיה לפנים / בימים הקדמונים / ודברי הימים.
לשרינו, למלכינו / לכהנינו אשר היו לפנינו / עד
תחשב כי עם כל אחד תתחבר / ופיהו אל פיך ידבר
/ ויודיעך את צדקתו ואמונתו / ורשעו וחטאתו /
ומפי תקח ולקח וערמה / ויגד־לך תעלומות
וחכמה.





וישא משלו ויאמר:

אני גבר ולי תיאות גבורה / ברוק פי אויבי יתן שממה

ואם העט קצר קומה חנית פיו / הלא תגע עדי קצוי אדמה

אני חכמה ושכנתי בערמה / ופי יגיד לך סוד תעלומה

ואגידך אשר היה לפנים / אבל מה שיהי לא אדעה מה.





וכאשר שמעה החרב מעניו, / ותקד
ותשתחו לפניו. / ותאמר: תנה לי קצת חכמתך
ופרשת כנפיך על אמתך / וכאשר התחננה לו
ואלצתהו. / ויגד לה כי הציקתהו. / ויען העט
ויאמר: דעי כי לא החרב גבורה במלחמה / ואין
לבעל העט תבונה וחכמה וכאין כל תבונתו אשר
בלבו תר / ובינת נבוניו תתר / וכאפס חכמי
ונבוני / אין חכמה ועצה נגד יי.





וישא משלו ויאמר:

פתח ספרי ותמצא בו תבונות / והוא יודיעך דעת מזמה

עדי תבין אשר אין בך יכלת / להוסיף או לגרע במאומה

ואז תדע ותשכיל כי כמקרה / בני אדם כמקרה הבהמה.

ושער מותה הכל יבאון / ולא ידע אנוש מה בפנימה

ולא יעמד לבד נוטה שחקים / עלי תהו וארץ על בלימה.





ותתמה החרב ותאמר: ומי הוא זה האדון
הנורא אשר ימשל בכל נברא? ויאמר: הוא יי אלהי
האלהים / ואדוני האדנים ראשון לראשונים / ואחרון
לאחרנים / אחד לכל האחדים / אשר הכל לו מודים /
אשר תבונה נעלמה המציא / ותעלומיה אור הוציא /
ותקח מתבונה הפרט / ויצר אותו בחרט.





וישא משלו ויאמר:

שמונה הם אבות כל האחדים / והאחד מיחד באמונה

אשא אין לו דבר מרבית והוא אל / אמת כי הוא למקדם מעונה

ולא נמצא במקום כי במחשב / חכם לב גם במצרף התבונה

והוא אל חי וקים אין דמות לו / ואין ערך וצורת כל תמונה

והוא המציא תבונה נעלמה / וכל נמצא לאל שדי למקנה

והוא ידע לכל נברא ערכיו, / ומאותו נתונה כל תבונה

והוא ראש כל וגם הוא אחרית כל / והוא סבה לכל סבה הגונה.





ובלתו אין מאחר באחדים. וזולתו הלא
שלש מאות לך אחת לכל הבריאה. ברוח גם בגשם
היא נתונה / וכל גשם בצורה גם תבונה / ורוח לך
אות באמונה / וכל אשר נקרא אחד בלתו ואינו צומת
/ כי שקר הוא מת / ויי אלהים אמת.





וישא משלו ויאמר:

שדי מקרא באמת אחד / לא נחלק לספר ולא יחד

כי כל אשר נקרא שמו אחד / בלתי יחלק אחרי נצמד

אחד אשר לא נחלק יעמד / לעד ואחד נחלק נשמד.





ציד לא צוד / ולא יחמד / כי היקום בלבד
יחכים ברואיו לעבד / אותו מלמדם אכן בלי סוד /
האלהים נעלם מבלי כי מי בסוד יי נעלם עמד /
והאחד אשר אבירי אדמה יסלה, / כקטן כגדול
יכלה / כי הנה קדושי עליון לא נאמנו בכבודם / אף
שכני בתי חמר אשר בעפר יסודם / וכל יושבי
תחתית הכל ימעד / וצדקתו עומדת לעד.





וישא משלו ויאמר:

ארץ ושמים ושכניהם / הכל אלי אל נערץ נעבד

תחתים ועליונים בעין יוצרם / שקר כמו נקלה כמו נכבד

אין מעמד להם אבל יעמד / האל אמת כי לא אמת תאבד

כי לא אמת תכון בנברא רק / האל יי הוא אמת בלבד.





אבל מלאכיו העליונים / וקדושיו
הנאמנים / וכל אשר אורו מאור כבודו יסודו קמו
עמדו. אך במאמר מלך הכבוד. / כי הוא אמר ויהי,
הוא צוה ויעמד / על כן מאור יקרו נגזר העמיד אותו
לפני יי לשרתו.





וישא משלו ויאמר:

ולא יכלו בפעל כי בכח / והכח הלא קרוב לפעל

ומוציא תעלומה אור הלא הוא / מסבותם הלא עשה ופעל.





כי כל נברא מוכן לתכלית / והבורא יכול
להשחית / כי האמנם לו חפץ פתאום / יפקד יי על
צבא מרום / וימסו במיסי ונגולו כספר שמים / ועל
כן ברא שמים מוסבים / למען יעמדו ימים רבים /
וכל אשר נתן בין הארץ ובין השמים / יכלה וישקע
/ וכל אשר בארץ יגוע.





ויחפשו לבות בני האדם / ושטטו
בחוצות המדע כפי תולדות יסודם / ויחקרו
וילמדו / את אשר יעמדו / והשתקשקו בחוצות
התבונה / לראות כל התמונה / ופחדו ורהו / כי לא
ידעו מה הוא / וכאשר התאחדה עצתם / והתעוררה
משנתם חכמתם / ונצרף המחשב / ולב השכל רץ
ושב / ויעל כל רעיון / לדעת דעת עליון / עד
שנמצא בכל נתח / וילאו למצא הפתח / כי לא
השיגה יד התבונה / לראות תכונה / והנה מלאכיו
לא ראו כי אם כבודו / ולא ידעו סודו / כי אם הודו /
אשר יסוד טהר ונעלה יסודם / אף כי לבות בני אדם /
אבל ראינוהו כעין הלבבות / ובמצרף מחשבות /
ומצאנוהו בקרבות / כי הוא אחד לכל אחדים אשר עשה
נפש חיה וצומחת / ונפש חכמה אשר באדם פורחת /
ויצמח ויהי לגפן פורחת / ויתן עצמות וגידים
שלשה גביעים משקדים / והוא האחד אשר הודו שמים
כסה / והוא עשה כל אשר נעשה.
ותחרד החרב ותלבש רגזות / כשמעה
החכמה הזאת. / ותאמר לעט: היטבת מאד להחכימני /
צדיק אתה ממני / ומה לחרבות אל העטים / החכמה
תעז לחכם מעשרה שליטים / כי מה ימתק שיחך
ומה יערב / טובה חכמה מכלי קרב.
