


	
	סיפור אליגורי על הנפש והגוף
	








נאם למואל בן-איתיאל / רב פעלים
מקבצאל / אשר ספר משלים וחבר / לדעת לעות את
יעף דבר. ויהי בעת צאת מלכי הידידות / והתעורר
שרי השחרות והילדות / ויצאתי אל רחוב מרכבותם
/ ולחוצות מרוצתם / והנה מרכבות החשק באים /
ירוצון כצבאים / חילי חילים / דגלי דגלים /
יתהוללון וישתקשקון / יתדפקון ולא יתאפקון
/ והנה אחריהם רכבי האהבה / מרכבה אחרי מרכבה
/ ועליהם ידיד נפשי שר הצבא / דגול מרבבה. /
בראותם אותי / וירוצון לקראתי / ויחבקוני
וינשקוני / ובכל זאת נפשי ישנה. ותחלם והנה
השמש מחבק חיק הלבנה / ותאמר: אין זאת כי אם
ידידי יבואני / ולא ישקט עד אשר יראני.





ותשא משלה ותאמר:

וישנתי ולבי ער / והנה קול צבי דופק

צביה, די להרדם, / לכי על דוד להתרפק

וכשמוע לבבי זע / ולא יכול להתאפק.





ויתגעש וירעש לבי בחדרו /
בשמעו מספר החלום ואת שברו / וימהר
ויחביאני בחדרי מורשיו / לבל יראוני מרכבות
הצבי ופרשיו. / ויקם דודי וכל רכבי אוהביו / וכל
גדוד החשק אשר סביבותיו / ויתנו כסאותם סביבות
לבי / ותשמע נפשי ותאמר: אשקף בעד אשנבי /
ואולי אראה הצבי מאור שמש[י] / ותחי נפשי,
ותדום ותלבש בגדי חפשי / ותפתח גנזי
ותיטב נפשי בבשמיה / ומר ואהלות וקנמונות
וקציעות כל בגדותיה / ותלך היא ונערותיה /
ושולי עדניה / וגדילי תענוגיה ארוכים. ותרץ
יעלת חן / ותעל אל העליה / ותשקף אנה ואנה /
סביבותינה / הצבי אהובה, ויבא לעיניה / מאור
לחייו פתאום / ברק ויהום.





ותשא משלה ותאמר:

הברק זה ירוצון או / פני דוד זהרם זרח

הנטע בי ידיד חשקו / ביום נסע ביום פרח

ירוצון בי ידידותיו / כרוץ גבור עלי ארח

וכל דוד ישכנה לבי / כגר תושב – וזה אזרח

וזולתו כאורח / כהלך יהלך אורח

וזה ילך וזה ישוב / וזה יבא וזה יברח

וזה יכבד וזה יקל / וזה משא וזה טרח

ורפאות פי צבי אל ציר / ומליו על שחין ימרח.





ויהי אחרי כן ותשלח ידה / ותחזק&gt;
בידידה / ויהי הוא בא כגוער / וחצנו ינער / ויאמר&gt;
אליה: סורי ממני וגורי / באשר תגורי / הלא את&gt;
ידידתי / למה לא יצאת לקראתי / הלא ידעת / אם לא&gt;
שמעת / כי באתי לחזותך / להשתעשע בהדרתך /
ובתפארת יפי יפעתך. / ורכבתי על עבות קלות /
והלכתי ארחות עקלקלות / ותחבאי ולא רצית
להתגלות / ותצר לי להתחלות / ובבואי לראות
פניך / ותסבי מנגדי עיניך / ועתה אשובה לי
אני ומרכבות חשקי / כי לא השלמתי חקי /
ותחרפי מרכבות ידידותי / וגדודי החשק אשר
סביבותי / אך תני לי נשיק[ה] אחת לצידה / בעבור
תהיה לי לעדה. / וישב אל עינו ויאמר: עיני, עיני,
קחי גם [את] מן הידידה צידתך / להחיות את נפשך
ולכלכל את שיבתך / והרבי פנות אל פניה / כי
אין לך מן העפרה כי אם ראות עיניה.&gt;





ותשא משלה ותאמר:

קחי עיני ראות עפרה לצידה / וזה חלקך לבד מן הידידה

וגם את, פי, יעד לבך לנשקה / תהללך בכל יום היעודה

וזאת תורת ידידי התבונה / וביניהם הלא זאת התעודה

ומה מרה פרידת יעלת חן / ומה קשה ספוד על הפרידה.





ותקם נפשי ותשתחו למולו / ותבך
ותתחנן לו / ותאמר: הלא ידעת כי לבי לי
ובדברי עם אדם יתאנה / ועז פניו ישנה /
ובראותו אותי לקחני / ולחדריו הביאני / ובמורשיו
החביאני / ואמנם אין עתה רואני. / וישב הצבי
לדרכו סר וזעף / ורפה ידים ויעף. / ויהי
בהפנותו שכמו ללכת / ודמעת עפרה על לחייו נתכת
/ ותשב אל לבה ותאמר: מה לי ולבכות / וירבו
ביניהם דברי ריבת.





ותשא משלה ותאמר:

האהבים ספרו לי / איך דוד יחי אין ידידו

לבי ביד הצבי שם / גם הצבי הוא פקידו

אדם בלי לב היחיה / וצבי נשאו בידו

נוע ינועו משחק / ילעג וישחק לאידו

אם עין צבי תחזהו / חצה יפלח כבדו

ילון בבואו לחשקו / אם נד ישחק לנדו

נבהל לבואו וימס / כעש לנודו חמודו

על כן ימותיו דאגות / אם בא ואם שב בעודו.





ויהי אחר הדברים האלה ותתחלחל
נפשי מחשקה / ותרב אל לבה בחזקה / וכאשר
הגד לי, כי גדל חילה / נבהלתי ונחפזתי לראות
מה-לה. / באתי עדיה והנה סרה וזועפת / ואל דודה
נכספת / ובמרירות פרוד מתעטפת / ורוח ממול ארצו
שואפת. / שאלתי לשלומה ולא השיבה / ולא
הקשיבה / ואקד ואשתחוה / וכל זה איננו שוה. /
ותאמרנה לי נערותיה / דע לך, כי היא חולת אהבים /
ותדבר בך דברים לא טובים. / וכשמעי, כי דברה
בי / וימת לבי בקרבי / ויחר אפי ואמר: הלא זה
קלוני / לטמון בחבי עווני / וימלך לבי עלי /&gt;
ויחרד חרדה בשמעו מלי / ויאמר: מהרה הכנע /&gt;
מהלך אלי[ה] אל נא תמנע / על כן לא תביט אליך /
ולא תראך. / על כי מנעת מדודה / שוב אל
גברתך והתעני תחת ידה. / ויהי טרם כלתה
מעצתו / וייקץ יעקב משנתו / ואמר: אט אני נגע
בבת עיני / הלא שקר בימיני.





ואשא משלי ואמר:

דוד היקצף על רעיתו ונשבע / חי ידידות, לולי לבבי הרגתיך

כזבה לה נפשי ואיני כחושק / הן תמול בעלת אמונים ידעתיך



זאת תעודת דודים: צביה תחרף – / יענה דודה: הן בנפשי פדיתיך.





ואקום בחפזה, ובדאגה וברעדה
הכנעתי / ולא גבהתי / וחשתי ולא התמהמהתי /
ואבא אליה בלט ואקרא: נפשי, נפשי, יונתי תמתי
/ ואמר לה ולנערותיה: מדוע קצפתן על עבדכן /
ותדברנה דברים [אשר] לא כן / הכזה יתכן /
אמנם כל היום יחטא איש לאיש ובן מחפיר ומביש /
ויהי כמחריש. / ותען נפשי ותדבר אתי קשות /
ותאמר: הלאנשים תדמה הנפשות? / ואם יחטא אדם
לאדם / הלא בעפר יסודם / ואיך יערך בן מלך / לבן
מלך? / הלא גזרתי גזרת ספיר / ומחצבי יקיר /
וחזויה דחיל נרויה שפיר.





ותשא משלה ותאמר:

גזעי ברום נתן בתוך אור היקר / שם הוא למקדם ליום המעשה

איך יוצרי שלח קצת גזעי לגוף / אדם להסיר מאנוש רע מעשה

כי איך ידמה גוף לגוש עפר? למי / גזעו כזה ובצל אדוניו יחסה.





ויהי כשמע לבבי וחרה אפי / ועלה
קצפי / וחזק זעפי / ויספק אל כפי. / ויען
בסערה ויאמר: איך יגבהו שכני בתי-חמר? ויאמר
לה: גם אני אודך / כי רב הודך / וגדול כבודך /
ויסוד טהור יסודך / והן אם יסודך נעלה / ומחצבך
ממני נפלא / וגזרתך ברה / ואת צפירת תפארה /
הלא ידעת / כי כל הודך ותפארתך / גנוב הוא
אתך. / הלא אם גזרתך מגזרת טוב נגזרת / והוא
עטר לך עטרת תפארת / עליך יאור אורו / ויגה
נגהו / כי גבה מעל גבה שמר / ואם יסודך ויסוד
יסודך גבהו שניהם / וגבהים עליהם / אף כי כל ימי
מגורך אני הדרך / ובמצבי את נצבת / ובקרבי
את יושבת / את נשאת אותי – ואני משאך / ואת
לי לאלהים – ואני נביאך / אגיד זיוך וישרך /
ונגהך ואורך / והודך והדרך / כבוד עשרך. /
ואולם לא חנותך בגוף אדמה / ואז תהי נעלמה /
ועתה הנני בית מנוחך / ואת לי לאלהים ואני
משיחך. / ולמה בזיתני / ומדוע הקלתני? / אז
גערתי בלבי ואחרפנו / בשמעי אותו מדבר
גדולות ונפלאות ממנו / ואמר: מה לך לדבר
במעלות נעלות / אל תבקש לך גדולות / כן
הנפשות דוברות / הלא הנה בגזרת בוראן חברות /
אך הרחק הרחיקו מבוראן נויהן / והגיא ביניהן /
ואתה מה לך לדבר / ומלים לחבר / הלא אתה גוף
נגוף ופגר מובס / למשלח ולמרמס / ואיך יתחרה
בנפש הטיט והרפש.





ואשא משלי ואמר:

יסוד אדם כמו אפס / ומחצבו הלא רק רע

ומארבע יסודות הוא / וכל מוסיף הלא יגרע.





ויהי כשמע נפשי / באמרי ללבי
נקלה ונבזה / אז רפתה רוחי מעלי בדברי את כל זה
/ ותענני נפשי: אל ירע בעיניך זאת / כי עניתי לך
עזות / על כן דברתי דברי תבונות / ודברי חכמים
כדרבנות / אבל דברתי מקצר רוחי / ומרב כעסי
ושיחי / אף אמנם חמסי עליך / על כן אריב אליך / על
כי כליתני בבתי כלאים / ולא הראיתני דודי בין
הצבאים. / ואני שמעתי עליו, כי ביעלת חן זולתי
דובק / וחיק אילת אהבים אחרת חובק / ואני שמעתי
עליה להשמעות אזנים / כי תאוה היא לעינים /
מי-יתן ויגיד לי שמה ודמותה / ומשפטה
וצורתה. / ויענני לי לאמר: הלא החכמה רעיתו /
ועליה תשוקתו / ועתה מפיל אני תחנתי על
פניך / ותיקר-נא נפשי בעיניך / ונלכה נא יחד
ונצא / ונשאל לנתיבה, אנה תמצא / ושם נרוה
דודי החכמה / ונתעלסה באהבי ערמה / ודדי
שכלה נעשה / ושער דוד נתכסה / ונצלח ונעשה
/ ונרעה בגנים / ונלקט שושנים / ומשך חכמה
מפנינים / ונרד לגן שעשועים / אולי יקרה
לקראתנו הדוד הנעים. / ואצעק צעקה גדלה ומרה /
ואמר: מאין לי גבורה / כי איך דעי וראי כי את
מתאנה / ואיך תאמרי נרד לגן או נעלה / ואנכי
איש-רש ונקלה.





ואשא משלי ואמר:

המסכל יבקש בין / וערמה לפתאים מה

המרעב שאול לחם / ומים מאנוש יצמא

מבקש בין בלב סכל / הלא ישא עוון אשמה

ומה בינה לפתי כי / עלי כסל עשות פימה

הבשחק יבקש גוש / ואם לשאל שאול ריקה

ומה לגוף ולשכל / לנפש יאתה חכמה

ולו גן יעשה עצלות / ופריו נום ותרדמה.





ותען נפשי ותאמר: גם אודך הודך /
דבר, כי חפצתי צדקך / ותקם בצוחה / גם היא
נאנחה / ותאמר: ולמה אני מצעקת והמיה / ועל מה
אני בוכיה? / הלא על היותך עצל בימי המעדנות /
ולא עשית לך פרדסים וגנות / הלא ידעת, כי
קרעים תלביש נומה / ועצלה תפיל תרדמה / כי
זכר באמרי לך קנה חכמה / ושכן ערמה / ותענני
דברי תנואות / מלאות תשואות. / ותאמר: הנה החכמה,
מכל יציר נפלאה / ואיך אבקשנה והיא לא נמצאה
/באזנינו שמענו שמעה / ואיש לא ידעה / והשכל
הלא רם ישכן והוא מרומים ישכן / עד כי נשארת
מן החכמה ערום מבלי בגד / ותלך ותשב לה מנגד
/ ואמנם כי שקר דברת / ומלי כזבים חברת /
ובהבלי סכלות התאננת / ובצל פתיות התלוננת /
ותבחר לך דרך משובות / אמר עצל: שחל בדרך
ארי בין הרחבות / כי רודף שכל ישיגהו ולא יראה /
אפס קצהו תראה / והן אם נתן השכל במרומים /
בתחבולות יורידוהו משם חכמים / ואף כי הנפש
אתו תשב ולא תהיה / ובצלו תחיה / ותחת כנפיו
תחסה / להודיע לאדם מה יעשה / ואם תשיב לה
רוחך / היא תשיחך.

ויהי כשמעי דבריה / ותקצר רוחי, כי
נהפכו עליה ציריה / ונחרפתי ונאלמתי / ובשתי
וגם נכלמתי / וידי למו פי שמתי / ויפלו פני
וגם חמרמרו / עד כי עלי רחמיה נכמרו.

ותאמר: לא עתה יבוש יעקב ולא עתה
פניו יחורו. אם אתה היית עצל ותרדם / אל יפל
עליו לב אדם / הנה ראיתי גן חכמה נטוע / ובזרעוני
בינה זרוע / ומעין ומרוהו ישקהו ומבוע / קום
ונשית לבנו למסלות הליכותיו / וירנו מדרכיו /
ונדרש ונחקר לנתיבותיו / ונלכה בארחותיו.
בשמעי כל זאת אמרתי: ומי הוא זה אשר
נגע אלהים בלבו / ותענני: איש אשר רוח בו; הלא
ראית מרכבות האהבה / אשר עליה שר הצבא / דגול
מרבבה / הוא הנוטע וזורע / החכמה קראתהו רע /
ואני כל היום משתומם / על כי ישטמני / ועל
כבודו כי גלה ממני / ועתה למה נתראה / לכו
ונלכה עד הראה / הלא [ב]אמריו כאבים יחבש /
ומתוקים מדבש / ושמו לא אכנה מפני אויב מקנא /
ואמנם אלון על נודו אשר הכאיבני / אך עתה
הלאני.

ותשא משלה ותאמר:

נע ידיד מני ועיני / ירדה מים לנועו



בם ישקה גן אהבים / גן [כ]נפשי דוד נטעו



בו הלא לבו יהלך, / בו הלא רפד יצועו

יזכרה ימים קדומים / יזכרה דוד שעשועו

אפרעה לבי ואקרע / הן נדוד דודי קרעו.





ואמר: אהה נפשי, מה יועילך / בבא
חבלים לך / ומה נחנת / עד מתי להנחם מאנת /
ותען ותאמר: הנה ללעג תשימני / הלא ידעתי כי
הבל תנחמני. / ואתחנן אליה ואמר: הנה נא
נואלתי / ולדבר לפניך הואלתי / הסירי ממך
כעס וזעם / ואדברה אך הפעם / הגידה לי יסודך /
ומאין חכמתך והודך. ותאמר: אני לך יסוד
הנפשות ומחצבתן / ונצורות לא ידעתן.





ותשא משלה ותאמר:

אלהי נעלה על כל / והוא נפלא לכל נפלא

והוא ראשית לכל ראשית / באין ראשית ולא תכלה

וברא כל בחכמתו / והוא נושאם ברב חמלה

ושכל ראש פעלותיו / ונפש אחריו עולה

שלשת הנפשות אך / [ו]חכמה נתנה מעלה

ותחתיה מקור חיים / ונפש צמחה אצלה

ומגזרת שלשתם ש/שלש ארץ להגדילה

להחכימה ולהחיותה / להצמיחה….. לה



[ו]שמים וכוכבים / בראם סובבים סלה



בהסב השחקים נו/לדו ארבע אבות תחלה



[ו]ארבעתם יסוד כל חי / וכל אבן וכל אלה



ולא כל חי אבל כל מת / הלא זה אשר הוא יכלה

אדמה שת בתוך שחק / וארץ על בלי תלה

וכל אלה בראם אל / במאמר עד אשר כלה

ודע כי נאחזו כלם / וזה מזה הכי נתלה

ומאמר יהיה צעיר / ואור מאור עדי מעלה

ויגבה אור עלי מאור / ומי יגע לאור מעלה

ולא יכלו ולא יבלו / ולא יראו ליום נקלה

אבל יבלו בני ארבע / יסודות, הם בני מחלה



ומחצב זה בהפך זה / [ו]מה זעם [ו]מה כלה



ומפעלם לזאת יבלה / ומעשיהם כלי גולה

והמשל בסיר ארבע / לבל תהיה מבשלה?





נוצרו אלה העולמים כלם / והוכנו על
מכונו כל עולם ועולם / ומערכה לקראת מערכה /
ותשלם כל המלאכה.





משל שכל כבר נשלם / בענין אהבה נעשה



חשב טובה, [ו]לדבר מום / ידיד אל נפשך תשא.
