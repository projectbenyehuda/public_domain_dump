


	
	הקדמת המחבר - סיפורי אהבה
	








משלי יעקב בן־אלעזר / מלאותם השכל מוסר



שירים ספירים גזרתם / שעשועים הם אל־כל־בשר



דמו חידות אך הם בינות / אל־איש משכיל לראותם סר

אך בין יבין את ענינם / כי טוב טעם בם לא החסר.





נודה את־יי בוראנו ומשכילנו /
מלפנו מבהמות ארץ ומעוף השמים יחכמנו / אשר
נתן לאדם לשון לדבר כל־התבונות והמשלים אשר
בן / ופם ממלל רברבן / ויחץ את הלשונות על
הגוים כפי רצונו / ואל־עם ועם כלשונו / ויבחר
בלשון הקדש לשון העברים / מכל־לשונות הגוים /
והמלעיג ללשוננו הלא הוא הנלעג והנבזה / כי
בלעגי שפה ובלשון אחרת ידבר אל־העם הזה.



נאם יעקב בן־אלעזר: סבת ספר משלי /
והתחברות מלאות מלי / כי חכמי הישמעאלים /
היו לשון הקדש מבהילים / ויתגברו ועלי
יתאמרו: / ויענו בגאונם ויאמרו: לשפתינו יאות
כל המשלים להחביר / אשר אמרו ללשוננו
נגביר.





וישאו משלם ויאמרו:





היש לשון להלל או לגדף / ולעיר אהבה כלשון ערבים

ומלחמות וקורות הזמנים / ואי מלים כמלינו ערבים.





ויהתלו ביהודים ברהבם / ויהיו
מלעיגים בם / ויפתו גם־עמי בפיהם / ויתעום
בכזביהם / עד כי היו אומרים: / ״כן בני ישמעאלים
דוברים / כי בשפתם כל־לשונות ידביקון / לשונם
יחליקון / ואין מלים ערבים / כלשון ערבים״. /
ואקנא על־כי חרפוני בחרפתם / ועל אמרם נבלה
שפתם / ויאמרו לי חברי באשר [אתך] שפתנו
אסף חרפתנו / ואשא משלי ואמר:





תלינון לי ותאמרו לי / לשון קדש הלא נחשל

והיא תרנין לשון אלם / ובה ירוץ ולא יכשל

להלל או לגדף בה / וחוד חידה משל משל.





אף כי לשוננו פלאים ירד / ולא נשאר
כי אם פלט ושרד / אשר השאיר הברד / לכו חבולי
עדת ישמעאלים / ואחבירה עליכם במלים / ולמה
תוציאו על־לשוננו דבה / ודעו וראו כי רעתכם רבה
/ ומה־לך כי נזעקת עמי / ומה־אתה חסר עמי /
דבר ככל־אות נפשך במלי / דע כל־מחסורך עלי
/ ורוץ בם ולא תכשל / וחוד חידה משל משל / וקרא
ספר משלי המחבר / ותדע ותשכל מן־מצא דבר /
ושמי במשלי לא כניתי / והסבותיו ללמואל
בן־איתיאל ואותו שניתי / כי כן חקות ישמעאלים /
להסב שמותם במשלים / אך עתה אגידנו כי אני
יעקב בן־אלעזר / למען אשר לא יקרב איש זר /
והגונב ממנו יהא נכרת ונצמת / והזר הקרב יומת
/ זה משפט שבועת משלי וזאת החקה / כי כל־הגנב
ממנה כמוה נקה / אך לא לגנב כי לכסילים ולאיש
רע מכלל נבזה / כי מי יחוש חוץ ממני לדעת ולבור
את־כל־זה / וראשון משלי שכל קראתיהו / בקשתיו
ולא מצאתיהו.





ואשא משלי ואמר:





משל שכל עלי אהב / עשיתיו בעבור יערב

קראהו והתענג / עלי טוב נעמו כי רב!
