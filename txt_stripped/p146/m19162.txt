


	
	אימה בת מלכים
	



חתימה: אלשבזי משתא זיו




אימה בת מלכים התימה / עדי כמה / בגלותך
עלובה



לדוד בקשי / והפריחי לשרשי / בהר קדשי /
העידיני אהובה



שאון גלים / משכני והעלים / בני אלים / ונשמתי
עזובה



בחיק רעיה / יחידתי עניה / ובוכיה / ביד פרא
עצובה



זמן בחרות / משכני בישרות / בדת חרות / בתורת
אל כתובה



יקרה מפנינים / למבינים / והם חונים / בתחלואי
משובה.



מתי שקר / עונם לא יבקר / עדת עקר / וחכמתם
גנובה



שפלים / כבהמות הם משולים / מבהלים / בעץ דעת
תנובה



תכונתם / מכערת בדתם / ואהבתם / בסוד איבה
חשובה



ואין חכמה / ולא דעת ועצמה / אבל ערמה / בלבותם
נצובה



זמונים לעשות / ראשי גיסות / ומחמיסות / בלי מתן
קצובה



ידיד חוסה / עלי עדה אנוסה / ותנוסה / לתוך צערי
תשובה



והגן על בני איש תם / בלכתם / תהא דעתם / בחסד
אל רהובה.
