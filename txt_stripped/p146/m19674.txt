


	
	דוד בחלום ינחני למכוני
	



החתימה: שלם י[וסף].

הכתובת: ישנתי רעב ננערתי בשיר זה.




דוד בחלום ינחני / למכוני / סביב לשומרון



זרח ושש רעיוני / ימשני / זיוו כסהרון



מכוס עסיס רמוני / ישקני / ואשוררה ברון



דוד חסדך



לי תמשכה



ובצלך



אסתוככה



רבון ומיטטרוני / אין שני / מלך ותוסברון



לידע בסוד הגיוני / זכני / עם גונזי ארון:



שרד לנום אסיאני / ואלהאני / שוקי בד̇א אלזמאן



אסבח אללה עאני / לא ת̇אני / עאלי בכל שאן



ד̇ו אלג̇וד ואלאחסאני / רחמאני / מא קאל כן פכאן



רבי רצ̇אך



אלקא חצ̇אך



קלבי עצאך



אכ̇שו קצ̇אך



אג̇על ליא אלגפראני / ואגית̇ני / מן ג̇ור אלאמתחאן



ואצד הוא אלשיטאני / ד̇י ג̇אני / פי צ̇ד ואקתראן:



לבי בהבלו בשש / מתגושש / לולי בריתך



עלי עלילות בולש / ומנקש / על בן אמתך



ואני עצתו כובש / ומבקש / לעשות רצונך



שיר אפצחה



ואשבחה



אם אשמחה



לא אזנחה



נצחי וגם פטרוני / עושני / לא שר ולא סרון

היך אחליף מעיני / מי הוני / בגרש ושנתרון:



מהג̇ת פואדי אלמרגוב / צח מצחוב / פי אלחסן ואלג̇מיל



ופי ג̇בינה מכתוב / סר מחגוב / ואלסעד לה דליל



היים קבילת יעקוב / פי מסבוב / באלנור ואלצ̇היל



אדנא אלג̇באל



פי כל חאל



נאדא וקאל



רב אלג̇לאל



אללה אחד וחדאני / דיאני / לא צ̇ד ולא קראן



מן מצר הו נג̇אני / וארואני / מן צכ̇רת אלצואן:



ים סוף וקנה קמל / יום המל / פחדו ופצמן



פגי בני חם נמל / עם צמל / עבר וקרסמן



[…]
