


	
	מעמד א: אלהים מעשיך
	



החתימה: אבן יוסף






אלהים מעשיך / בצדק נוראו

כתוב את חסידיך / לחיים יבואו

ולשלחנך / ברון יקראו

שומע תפלה עדיך / כל בשר יבואו:





בעדן תנחילם / וימצאו מנוחה

ותמחה מעללם / ופשעיהם סלחה

פני רון תקבילם / ושלום ורוחה

ומנוח וחיים / ותקוה ימצאו:





נחם האבלים / ורפא מכאובם

ובקר החולים / ודבר על לבם

ומדת דין העלים / ועוד לא תמשל בם

שעה שועת עם דל / בכל לב יקראו:





יגדל נא חסדך / על עדת ישראל

ותסיר את קצפך / ונפשותם תגאול

מקוים הם ישעך / עונה כל שואל

כי מזעמך / רעדה מלאו:





ואל מי עוד נסמך / ומי יושיענו

אם לא ממרומך / תרחם עלינו

ותשקיף ממקומך / לנחם אבלינו

את מעוז לעמך / חסדך ימצאו:





סלח פשע ועוון / ואל תזכור חובה

והצל מדוון / נפש שובבה

ומטיט היון / ביום המעצבה

ותחישה לאמר / לאסורים צאו:





פדה נפש תקוה / ישעך לראות

שבחיך תחוה / ותראה נוראות

וחסדך לה ילוה / אלהי הצבאות

ומלאכי השלום / למולה יקראו:
