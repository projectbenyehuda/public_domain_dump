


	
	אם דרכי
	



חתימה: אלשבזי






אם דרכי / במהלכי / ביום שובי חקרתי

לא אוכל / בעין שכל / להעלותן עלי דעתי

כי אצו / לחטא רצו / פעמי ביום מרדתי

ולב הומה / כמו סומא / בעצתו הלכתי

ועינים / כמו מים / במראיהן נמשכתי



אליך יוי / נפשי אשא / אלהי בך בטחתי:





לב טמא / באשמו / בימי נעורי מרד

חטא חמד / ולא למד / וביום הדין חרד

חבל על דל / היך יחדל / ויום אידו יתפרד

ובכלמה / ורוב אשמה / ישוב לעפרו ירד



והנפש / ביום נופש / תדאג אם נזכרתי:





שמע קולי / צור אלי / רצה שיחי וקול ניבי

כי שחתי / בעת לכתי / וחלשתי ברוב חובי

ואיך יצדק / עפר דק / וצד שמאל משל בי

כפוף קומה / ברוב אימה / ואיך אקביל פני רבי



ולבי סר / ונתיסר / ובהבליו נוקשתי:





בכי עיני / לבן אוני / אשר חטא במעלליו

והתבונן / והתאונן / עלי רוע מפעליו

חבצלת / כאילת / בלכתך בקשי עליו

ביום יחזור / להעזר / לרצות דוד במהלליו



וטל אורות / באשמורות / תני עלי ביום לכתי:





זרח שמש / לדל טמש / במי עכור ונצטנן

והוא צופה / לדוד יפה / אשר דעתו בטוב חונן

ובן פרא / לראש פורה / מאד גואה ומתרונן

והרבה מס / ודל חמס / ופי חרבו לחטא שונן



ומתיהר / שהוא טהור / ובחמסו שפלתי:





יקר חפצי / בבן פרצי / ביום יבוא להיכלו

ועת יבוא / למושבו / ושב עמי לגבולו

באהבת עם / מאד ינעם / ונקבץ בגללו

ואש תבעיר / לאיש שעיר / ובן אמה וכל חילו



ובת ציון / באפריון / תעירני ושמחתי:
