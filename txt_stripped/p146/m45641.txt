


	
	אלהי ישראל
	



חתימה: אבו שמעון אלשבזי






אלהי ישראל / יצוה רומיאל / לסגור פי מחבלים

בין לבנה / לוילון חנה / ובו דין ורחמים נכללים



ומחשמל / שע״ח רקיעים תגמל / חסד ודין פועלים



ומנהגם מלכם / עת רוצם בדרכם / ומשפעו מקבלים

ומראה הצורות / על כס יה קשורות / באופן וסגלגלים

כל ברואי מעלה ומטה / ליוי מהללים:





בחסדו קשר / רחמים ודין יושר / וקו המכריע

וירד סלון / עדי הוילון / מזה וזה ישפיע

ומהן תחתונים / ביחד נהנים / יורדים בהצנע

ונפשות חיות / קשורות בגויות / בעולם זה חונים

ונפשות בני אדם / בכסא יסודם / האור תזריע

ונפשות שאר חיים / מן הגלגלים:





ואדם הראשון / נער מפישון / הצמיח שבעים לשון

וסוד תורה גלה / לישראל סגלה / כעץ לח מתדשן

ושבעים האמות / אשר נקבו בשמות / לשרי החלשון

וסגלת המוני / קודש ליוי / ינצרנו כאישון

מיחדים שם כבודו / וטובו וחסדו מיחלים:





שבטי יה עורו / ואל נא תסורו / [……שר]

הכונו דרככם / לעשות רצון מלככם / אשר דגלכם קשר

סורו מהוות / ועסקו במצוות / יהא לכם כושר

ואשרי נשוי פשע / בשובו יושע / ומה יועיל עושר



מעשר ממונו / והשב מעוונו / תשובתו מקבלים:





מה תהמי נפשי / מתמול התקדשי / בין בעלי תשובה

בעשירי בתשרי / תשובתך בחרי / הבת האהובה

יום משה הרגל / ונמחל עוון עגל / לעדתי החשובה

והקבע יום מחילה / בקול שמחה וגילה / לרפאת כל משובה

ומדתו בינה / אשר בה מתנהלים:





עקרת בנים דלי / בן בגלות צוללי / ושגבי לאביון

והעירי ישנים / בחן אם הבנים / כי דללו בשביון

בזכות איתנים / אבות ראשונים / ושרידי הגיון

במחילה בשרי / וקומי התעוררי / בספר גליון

עלי זדון ושגיון / מתודים ומתפללים:





ובשורה טובה / לבעלי התשובה / כי נמחל עוונם

וזו תקנה / לעדת מי מנה / ישמחו מיגונם

ונפשותם תרון / בספר זכרון / יכתבון המונם

וצורי עז מגדל /המשגב לדל / ירצני בגינם

ביום מאזני צדק / זכות ועוון שוקלים:





נוגה תפארת / במלכות מזהרת / להאיר שבטינו

ואברהם ויצחק / ויעקב לא ירחק / לרצות עלינו

ומשה ואהרן / ויוי ראשון ואחרון / ודוד מלכנו

יעירון נעלבים / ביום שוב שובבים / וימחה פשעינו

ברוך אל עליון / הסומך נופלים:
