


	
	ארחות יושר
	



חתימה: אלשבזי






ארחות יושר תישר / בנים השובבים

לילך בדרך מישור / בין ישרים נאהבים

כל מעשיהם בהכשר / לאל צורם נחטבים

דגלי שבטיו קושר / על שער בת רבים

ושם סרח בת אשר / תבשר לבני נדיבים

תוסיפו שנים רבות / ומועדים טובים:





לב נדכאים ישוב / ויחיה ויאריך ימים

ושיחו יהיה קשוב / לאל חי נקי ותמים

בספר חיים כתוב / באותיות נרשמים

ושבע מכל טוב / בין עם קודש חכמים

ויקיצו נרדמים / בחן צורם נצבים:





שובי השולמית / אל היכל המלך

בגלותך היך נדמית / כורד באשפה השלך

ורבו בעלי תרמית / וצר מחזיק בפלך

ואהבתך עולמית / ואנה דודך הלך

אותי נהג ויולך / רדפוני הרובים:





בעזרתו אקומה / כי בשמו בטחוני

אקיצה מתנומה / וייחוד שמו בלשוני

ואתפלל ואצומה / ביום זה כפור עווני

נפשי בכסאו עלומה / וכל שחר ינחני

בחסדו יעמידני / באור שמש וכוכבים:





זעום הלב אל תרדם / קום קרא אל אלהיך

אשר חסדו לך קדם / ושם שכלך ונפשך

בטפת לובן ואודם / העמיד בנינך

ורוחו תנהיגך / בתוך חוצות ורחובים:





ידידים הקבצו / ושובו מדרכיכם

ולעשות הטוב חפצו / למען ירבו ימיכם

מעפר התנפצו / ופקדתם את נויכם

בני יעקב אשריכם / במצוות וחקים ערבים:
