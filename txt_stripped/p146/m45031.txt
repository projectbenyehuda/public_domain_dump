


	
	ביום שבת אהלל אל
	



חתימה: בן יוסף חזק






ביום שבת אהלל אל / ואעיר נוגנים

ואשיר שיר בנועם קול / למול מתבוננים:





נמגד רבנא אלעאלי / אלאה כל אלודוד



נסבחהו באגלאלי / קיאמן ת׳ם קעוד



נוחדהו באכלאמי / ונעבד פי סגוד



ומא אחד יטיק יכמל / בתסביח אלעצ׳ים:



[נהלל את אלוהינו הנעלה / אלוהי כל האהובים



נשבחהו בהדר / עומדים ואחרמ יושבים



נייחדו בדיבור / ונעבדהו בהשתחוויה



ואין אדם יכול להיות שלם / בתשבוחות לאדיר:]





ידידים זמרו הודו / לאל עליון בשיר

במעשיכם תיחדו / לרעיה עם גביר

ובשבחים תסלדו / לבורא כל יציר

אשר הוא מאציל הכל / מקור כל נאצלים:





וג׳ב תעזיז יא אכואן / לדא אליום אלפצ׳יל



אמר פיה רבנא אלרחמאן / בקולה אלגליל



עלא מוסי ולד עמראן / נבי זאכי פצ׳יל



אדי פי חלמהו מכמל / רסולן תם כלים:



[חובה לרומם, הו, אחים / את היום הנכבד הזה



בו ציווה אלוהינו הרחמן / בקולו הנעלה



על משהבן עמרם / נביא טהור ונכבד



אשר במתינותו מושלם / שליח אף דובר:]





שרידים המידעים / בני תורה וסוד

דעו ותיחדו רעים / עטרת עם יסוד

וגם כי בה רמז נעים / כללות עם כבוד

ובה שבת אדון הכל / וכלה אשתלים:





פראיץ׳ אלעקול אדרס / באסרארן גמיק



ואגכר ת׳ם אזיד אחרס / פי עקלן ות׳יק



ואצפי אלקלב מן אלאנשאש / וכלהמן וצ׳יק



לאן אליום זכי צ׳אהל / וכל כרבה עדים:



[מצוות השכל אלמד / בסודות עמוקים



ואזכור אף אוסיף ואשמור / בשכל איתן



ואטהר את הלב מן הזיופים / וכל דאגה ומצוקה



כי היום טהור בוהק / וכל יגון נשלל:]





חיאתן לאלאד׳י חאפט׳ / להאד׳א יומנא



זכין מכרמא פי חט׳ / וירזק פי אלת׳נא



קבולן ת׳ם כ׳יר פאיץ׳ / יכ׳ץ אחבארנא



סלאמן וארדן כאלסיל / מן אלרב אל כרים:



[חיים לאשר ישמור / את יומנו זה



טהור מכובד בחלק טוב / ויתפרנס בכפליים



רצון אף טוב שופע / יוענק לחכמינו



שלום בא כזרם שוטף / מן האל הנכבד:]
