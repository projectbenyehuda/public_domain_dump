


	
	אפתח בתחנוני
	



תיקון חצות

חתימה: אני שלם






אפתח בתחנוני / אשבח קוני

יודע במצפוני / וכל חביוני

יאיר מאד עיני / ויושיעני

יסלח לשגיוני / וגם לזדוני

קומה יוי / הושיעני:





נפשי תקוך / אלהי עולם

תסלח לעבדך / לגלוי ונעלם

נפשי תבקשך / ולבי נכלם

כי קץ בחקיך / וסר לא נשלם

לכן כחטאי / אל תדינני.





ישע והצלה / לעבדך תעמיד

נפשי מאד חולה / בחטאה תמיד

שכלי ונשמתי / ברחמיך תצמיד

אל אשבה בדד / בחטאי גלמיד

חון נא ברחמיך / והצילני.





שעה אל שועת / נפש יחידה

ואם שוגגת / היא או מזידה

כי פוחדת / בחטאה נדודה

ועין ענת / בכעסה מעידה

ולבי תחלה / החטיאני.





למען חסדך / זכור עבדיך

וזכור ידידך / אב שבטיך

ביום בוא פחדך / וגם דיניך

ותרים ידך / עלי יצוריך

תוציא לאור / אלהים דיני.





מחול נא חובי / וחטאי תכפר

ורפא כאבי / עצת רע הפר

ועוון חבי / יפרח כצפור

כי משל בי / ורב מלספר

ישמח לבי בך / ורעיוני:
