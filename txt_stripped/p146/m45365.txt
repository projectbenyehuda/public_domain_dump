


	
	אימתי בהר המור
	



חתימה: אלשבזי משתא






אימתי בהר המור / ואני בבור תחתיות

ברם ידידי שומר / ועיני לו צופיות

ואנעימה בשיר מזמור / בקו משקל אותיות

ורגלי כסאו קומר / בארבע הזויות

ועל דמות הכסא / דמות ארבע חיות:





לב ישרים יתחדש / כנשר בהמצאו

אל נוכח המקדש / יוצא מבית כלאו

וחיל סמאל ידש / יטעה ביין סבאו

וקציר גבולו נגדש / ולא רוה מצמאו

העוון יוציאו / לפי חרב פיפיות:





שאלי נפשי תשועה / מאלהי עולמים

ושמעי שופר תרועה / בעשרת הימים

וצדקה לך זרועה / לטהר טנוף גשמים

ותהיי בגנך נטועה / בלי פחד וזעמים

הזכי מאשמים / והרהור לב וגויות:





בראש השנה עמדי / יחידתי בשחר

ודלתות צורך שקדי / כי בך צורי בחר

ובזבד טוב זבדי / גופי ואל יתאחר

אתבונן יום חלדי / בבוא הילל בן שחר

חמוד עיני אל תתחר / עשה עמי זכיות:





זכור בוראך בן איש / בימי בחורותיך

וסור מחומר מבאיש / ושובה אל אלהיך

ושכל ונפש תקדיש / לאל חי ברצונך

ולך צדקה ילביש / ותזכה ביום שפטך

וירצה יה עליך / בגן הביתן לחיות:





יום ליום יביע אומר / להזכיר עושה רצונו

להעלותו מחומר / הטבוע בעוונו

וחטאו בספר שמר / לגלות ביום דין קלונו

ובקצירת העומר / ואין מענה בלשונו

יצרו צעדי אונו / ועינים בוכיות:





מחשרת העבים / ירוה נפשותינו

גומל חסדים טובים / בתוך יושר ינחנו

כי הוא מקבל שבים / ורוצה תפלתנו

בזכות אבות נדיבים / וראשי אבות שבטינו

וחן אהבת המוננו / בתוך בתי כנסיות:





שובי נפשי ותזכי / לחזות בנועם צורך

וישעך היום חכי / והסירי רוע סורך

כי בא אלהי מלכי / להזכיר חסד נעורך

מבכי תשובי, כי / לא ייבש מקורך

רחל למה תבכי / אורי כי בא אורך

ובלילי כפורך / כל נפשות הומיות:





תהמה כל הנפשות / מפחד יום הדין

בעלות חטא נענשות / יחד יהו מועדין

ובעלות חן נלבשות / באור בהיר כסדין

ביחד מגשות / ומלך על כסא דין

וכל שומרי דת ודין / מבחרות ונקיות:





אהבת כלילת יופי / תרצה על כל עדתי

ואם הלכתי בשופי / ועזבתי נחלתי

בקרוב יבוא צופי / וירפא מחלתי

מנתי וצרור כספי / צורי חמוד אהבתי

בראשיתי ובסופי / עיני לו צופיות:
