


	
	אל הנורא
	



חתימה: אני שלם שבזי משתא






אל הנורא וקדוש שמו / עושה ארץ ושמים

והוא אל חי מקום עולמו / שוקל הכל במאזנים

ובכוח שם תעצומו / יסד ארץ עלי מים

ומאין יש ברא יקומו / זכר ונקבה שנים

ואדם תכלית חותמו / ולבו כמו קו בנתים

וזרע יעקב סוד טעמו / איש תם ונקי כפים

בחר בזרעו והקימו / והוציאו ממצרים

כי נחם יוי עמו / גאל ירושלים:





נאץ רשע והרשיע / והצר להם במלכותו

תקע אהלו והציע / בערב להראות גדלתו

הכריז להרע והודיע / מגמת לבבו ותחרותו

להחליש שרידי להכניע / והחריב אהלי באחריתו

וחשב שאין לי מושיע / מפתני לחמוד דתו

זלזל יקרי והגריע / וקרב עת קץ נפילתו

אוהב לכסף ומטמיע / גואה ומקשה גזרתו

מגיד מדן ומשמיע / און מהר אפרים:





ישרי לב הנאמנים / שיתו לבכם לחילה

העניים והאביונים / כלכם יושבי הגולה

תעלו לציון ברננים / ויאיר צורי האפלה

ויאבדו גוים חמנים / ישתו כוס התרעלה

בבוא מטטרון שר הפנים / וידלה נפש צוללה

וכלים מכלים שונים / תלבש העיר האבלה

יאזין צורי תחנונים / וירצה תפלתי כעולה

לא הפנו אבות אל בנים / מרפיון ידים:





שוב ישורון מחטאיך / כי זה עתך עת דודים

כי אל שדי בעזרך / לעלות הר ציון גדודים

לך עמי בוא בחדריך / עד יעבר-זעם זדים

והיה שדי בצריך / בשוחה יהיו נלכדים

בטח ביוי רופאך / אוהב צדקה וחסדים

יתן לך כלבבך / בעבור אבות נכבדים

וידעת את נוך / יעלו דגלינו ידידים

וזרח בחשך אורך / ואפלתך כצהרים:





לבשו משנאינו פחדם / מיום באה בשורתי

ועמד פרא והזידם / להצר לי ביום גלותי

וגזר לפשוט את ידם / להחריב את בית תפלתי

יחד יבוא יום אידם / ותתעורר עת ישועתי

ויקיצו כל הנרדם / לעלות אל בית מנוחתי

ויראו ציון בעמדם / אשר שם הוא חשק מנתי

וישיגו את כל סודם / וישתוממו בני אמתי

והנה אין האדם / ואין עוף השמים:





מתוך ערים הפרוזות / הקבצו הנפזרים

בשנת היובל הזאת / נעלה לערי מבצרים

יזכו שבטינו לחזות / חמדת יוי מרום הרים

תגיל נפשות הנמבזות / בין בעלי ריב הצרים

וענני כבוד מתפרשזות / על בני ציון היקרים

וערי אדום ובצרה בזוזות / ומגדלי פרא מבערים

וצנורי אש בם נותזות / תכלה זקנים ונערים

מאת יוי היתה זאת / שמחו את ירושלים:





שוב עמי לגדור פרץ / לחלות פני צורי ואבי

ולהנצל מקרץ / תרמוס על שחל ולביא

ולא גור אריה ולא שחץ / יהמה בארצי ומושבי

חזה דת אל בעיון נמרץ / כי דרך מאור כוכבי

ועמד משיחי בן פרץ / ובא אליה הנביא

ועיני פרא אשר קרץ / תחשך בשוב נר מערבי

שמעי והאזיני ארץ / ואמרי לשולמית שובי

יתיצבו מלכי ארץ / בקול צעקה מחרונים:





בני אמה בא עת זמנם / יהמו יחדו בהיכליהם

כי יצרו צעדי אונם / רוח שערה תרדפיהם

שרים יפלו מכנם / ואש תלהט סביבותיהם

נפל שרם ואופנם / פחד ואימה תפול עליהם

ועמי הקוראים לקונם / יוי צבאות יעזריהם

יזכו ויצא לאור דינם / וחן תורה יאיר פניהם

ואויבי יתגלה קלונם / וינגפו תוך אהליהם

למי פצעים חנם / למי חכלילות עינים:





זמרי יחידה באשמורת / פצחי ברנה וקול תודה

בשרי אימה שחרחורת / כי בא פרץ בן יהודה

דרך גאולים מישרת / רוץ נא ומהר ציר תעודה

עורה מגבול צמרת / ופרוש לבן אמה מצודה

בטל גזרה אשר חרת / רגלו בפח מוקש לכודה

תשוב למלכי העטרת / מפז ומזהב חמודה

על הר עלי לך מבשרת / ציון אשר בחן אפודה

אבן יקרה לתפארת / והזהב זהב פרוים:





יבוא נא משיח צדקי / ויקבץ את נדוחינו

הוא כל תאותי וחשקי / יהיה שלום בעירנו

אחסה בצל יוצרי חלקי / יעלה יוי בראשינו

על ירכך בת אדום ספקי / כי באה ישועתנו

כי את באש יעקב תלקי / ובאש יהוסף ידידנו

רעיה לרגלי צבי נשקי / יחיש ויקשור דגלינו

אשמח ויפתחה שקי / יאמר: עלו עם משיחנו

ואשבור עליו חקי / ואשים בריח ודלתים:





מתוגת פרא נמבזה / בתימן יום בא נחבאתי

נמתי: האחיה מחלי זה / כי מהמונו יראתי

ונער שכלי הלזה / וקרר דעתי אני וביתי

ואמר לי: קום עלה מזה / כי בארץ ערב נטמאתי

פתאום יבוא דבר חוזה / ירפא לבי בחלותי

ועתה רוץ מה לך בזה / ולמי ישועה צמאתי

חזיר ופרא יחד תרזה / ואני בדינם החשיתי

ולנשיא מזה ומזה / שודד בצהרים:





שומרי דת אל אשריהם / שרידים אשר יוי קורא

רוח אלהים תניחיהם / הוא היוצר והבורא

חסד אל הוא מסובביהם / וצדיק מגין עלי דורו

צד השמאל לא ינידיהם / כי הוא כתנין בתוך יאורו

כי לא ינום שומריהם / משקיט לפתן בגיא חורו

ובני ישורון יחיהם / יאיר פניהם בזיו אורו

עורה ידידי אמור להם: / והעמדת להם כל מתנים:





תפארת בנים אבותם / ביחוד צורי יתהללו

כי הם עומדים בזכותם / ובזכות […]
