


	
	שבח אפליא
	



חתימה: שלם שץ






שבח אפליא / לצור גואלי / הנותן לי חיים

יוצר כל / ברוב השכל / בלי יגיעת ידים

גוף קבע / מארבע / אש ורוח ומים

והעפר / במספר / הרקימו בנתים

ומזגים / משורגים / ירקי ואדמותים

ושחריי / ולבניי / ארבעה בשויים

כל היצור / הוא עצור / בארבע ארבעתים



הן גוים / כמר מדלי / וכשחק מאזנים:





לשקל זה / כשקל זה / לא יוסיף ולא יגרע

ועוד נדמה / כחומה / מרכב מטוב ורע

והמוח / משוח / שהוא מלך מקום זרע

ולב לו שר / למוסר / מלכותו במאורע

והכבד / שר עובד / מחלק מזון בפרע

ואל הלב / משתלב / רמח ושסה הוא יכרע



והנשמה / נעלמה / ונפש ורוח חיים:





מכון כוחות / נוכחות / מעשיי ועיוני

ושכלים / נכללים / קרואים הם להמוני

והזוכר / והשומר / כוחות לחיוני

ושכל טוב / כעץ רטוב / הפועל רוחני

שהוא גוף דק / איש צדק / במעלתו עליוני

והדמיון / והאביון / הוא כח גופני



ובהמית / הנדמית / בבהמה ובחיים:





שמות כלם / לא נעלם / משרידי חכמינו

ויסודם / הוא נגדם / והמה שהורונו

בכונה / נאמנה / הגיהו חשכנו

ועת חשבון / הרבון / נפש תוכיחנו

והיא תעל / ממעל / ובגן תושיבנו

מקום נועם / בלי זעם / ומי חיים תשקנו



ונהרים / מנהירים / ארבעה נהרים:





צדיקים / נחשקים / שמורים שם במנוח

ועטרותם / בראשותם / מקום טובה וגם רוח

מעופפים / כשרפים / ברוב אונים וגם כח

ושם אבות / כמרכבות / וכל צדיק ומשוח

שכר כלם / בסוד נעלם / והוא כתוב עלי לוח

ושם רעים / וגם שבעים / ושם אהרון וזנוח



וחסידים / נחמדים / הישרים והנקיים:
