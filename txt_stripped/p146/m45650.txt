


	
	במנחת יום כפור
	



חתימה: בן יוסף שבזי






במנחת יום כפור / הסמוכה לנעילה

רוחי ונשמתי / בעת רצון תקבילה

הדרת כבוד מלכי / אל נורא עלילה

מקבל השבים / ברחמים ומחילה

השמיעו בציון / שמחה וגילה:





נתכפר יום זה / עוון חברת קדושים

במנחת זכרון / סר מים ענושים

השמיעו מחילה / בראש מעלות שלושים

הבינו זקנים / בחורים וישישים

וכל הנגשים / אשר לאל סגלה:





יחי לבבכם לעד / כל אנשי אמונה

כי תזכו כלכם / לטובה הצפונה

יחד בצל שדי / יהא לנפשכם מלונה

ולשרידי הדת / ידידים שאלו נא

תרצה עלינו / זכות שבת ומילה:





וזכות אבותינו / ותורה התמימה

היא תעמוד לנו / ודרך תשימה

לתשובתנו / ברצון תקדימה

תרפא לב נדכא / וכוחו תעצימה

ודגלי תרימה / ואויבים תשפילה:





סלח במנחת כפור, / צורי, לעמך

וכתבנו לחיים / ושלום בספרך

ורפא הלבבות / בחסד ימינך

ויכבשו רחמיך / ביום זה כעסך

רחם שובבים / דופקי דלתיך

המוני שבטיך / כקדם תסגילה:





פודה ומרחם / עניים ויתומים

יקבץ עם קדושיו / מבין העמים

כל זרע יעקב / שנקראו שלומים

ברצון ינהגנו / בחסד וברחמים

טהורים מכל חטא / ועוון ואשמים

ויחיש משיחנו / ימהר להצילה:





שולמית יום זה / שבטים השיבי

והמציאי כופר / ומנחה הקריבי

לכל עני ואביון / במזון הטיבי

וביום בוא גאולים / אהלך הרחיבי

ולשרידי חכמה / גברת השגיבי

ורוח אלהים בי / לשכלי תאצילה:





בואי עת רצון / בתפלת מנחה

יום כפור עווני / טוב מכל שמחה

כי נדחף סמאל / משמאלי נדחה

וסר מני עוון / אשר היה טרחה

דלתי התשובה / ידיד נפשי פתחה

ובשני גורלות / אהרן יגרילה:





זה היום מכפר / לשבים עצמו

ביום הזה כל בא / יקרא בשמו

ויצוה חסדו / בורא עולמו

לעם נקרא בכורו / אשר יצר בצלמו

הם חברת ידידיו / פאר נטע כרמו

ונקראים עמו / אשר לשם ולתהלה:





ישורון התישר / סרו כל עוונות

במנחת יום כפור / הביע רננות

בך בחר יוי / משבעים לשונות

ולכם עוררו זכות / בארבע המחנות

ובצל אל שדי / נפשותיכם מתלוננות

ובבקר יחד / שיר ושבח נותנות

ובגופות שוכנות / וחוזרות ביום חלילה:
