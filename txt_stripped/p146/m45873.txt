


	
	אקרא לאל ממעמקים
	



חתימה: אני שלם






אקרא לאל / ממעמקים

לילדי איתן / בריתו יקים

ברון יצילם / מיד עושקים

בזכות אבותם / הנחשקים

אשר שמרו דת / מצוות וחקים

ואם אויבינו / נראים חזקים

עיני יוי / אל צדיקים:





נועם ואחוה / אמונים דרשו

ושלום ותקוה / מאל בקשו

וליום ישועה / חכו והחשו

ואם אויבינו / בחילם רגשו

כמעט רגע / והם ינתשו

במעשם הרע / הם יענשו

כי הם רודפים / אחרי ריקים:





ישע אלהים / לעמו יאתה

ולבם בטובת / דברים יפתה

ועל שונאיהם / אש דת יחתה

וערוך בדינם / תמהר תפתה

עושה משפט / אל לעשוקים:





שפעו יוריד / לעם נאמנים

ויזכור להם / ברית ראשונים

שלשה אבות / האיתנים

זכותם תגין / כאם על בנים

מתגרת צר / וכל זדונים

ומיד מושלים / בעולם דוחקים:





לב עם נדכה / אזי יתנחם

בזכרון צורם / אשר הבטיחם

כי הוא עליהם / כמו אב ירחם

ויקשיב לקולם / בערכם שיחם

וידלה טבעם / ממעמקים:





מלכות משיחו / בקרוב יחדש

ועמו מיד / בוחלים יקדש

וקרני מעניו / יגדע ויגדש

ויקצור לזרעם / כקציר בחרמש

ויפדה לעמו / אשר בו דבקים:
