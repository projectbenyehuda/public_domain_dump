


	
	שזפתני יפת תאר בהיכל אהבה
	



החתימה: שלם יוסף




שזפתני יפת תאר / בהיכל אהבה



ראיתיה עלות שחר / בדעת רחבה



והיא תביע בקול אמר / בקומה נצבה:



ידיד נפשי, בשיר מהר / ואלי תכתבה:



לקחני הדר יפיה / ברב חכמה וסוד



הדר מלך יקר עדיה / כמו נצח והוד



אבל חלת בבית שביה / בגלות הנדוד



ומראיה כאור סהר / לבבי לבבה:



מחשקת באחלמה / וזהב פעמון



יפת עין ולה קומה / מלבשת בחן



למולה מלאכי רומה / יצחצחו ברון



בקול הלל וקול זמר / לבבי להבה:



ירוצון מלאכי מעלה / למולה נכספים



והיא תנעים בקול צהלה / והם שם שואפים



מקשטת כמו כלה / כלילת היפים



הדרה במרום נהר / עלי כל שר צבא:



ודודה ויעדה מרום / בהשפע ורון



וסודה עד מאד עצום / בגובה אדרון



ותבין מעין חתום / ותשיג סהרון



ומדות תקשרם קשר / באיתן תשלבה:



סגלת כל בני חכמה / בשלמות נרקמה



ותשוט על פני ימה / בהודה תשלמה



והיא עלמה מאד תמה / בטובה נפלגה



וטובה רב ולא יחסר / בעת כי שגבה:



פתור דודי לשירי זה / ושים על לבך



והתבונן כמו חוזה / לדרך רחבה



והשתונן ליום מסע / והכן דעך



והכינה לבוש טוהר / למען תרהבה:
