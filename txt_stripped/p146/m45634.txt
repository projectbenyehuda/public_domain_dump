


	
	אימתי לעזרתי
	



חתימה: אלשבזי






אימתי / לעזרתי / חלי עלי פני אל

היי לדל / עיר מגדל / ומאויביו יגאל

והעירי / בני עירי / ונין צמח איתיאל

מה טובו אוהליך יעקב / משכנותיך ישראל:





לב טהור / יזהר / מעוון ומאשמה

על צד שמאל / לא יחמול / כי בו עול וערמה

וצד ימין / לטוב יזמין / לרוות לבב נצמא

ויוסיף דעת וחכמה / להרגיעו ישראל:





שחי נפשי / בסוד רחשי / ושובי אל אלהיך

והכוני / לרעיוני / בנועם ארחותיך

בזה היום / תני פדיום / להצליח מעשיך

ובצלי כנפיך / נעים זמירות ישראל:





בת גלים / אל תעלים / סוד לבי ומחשבו

כי חובי / ענה בי / ונתפשתי במכאובו

שמאל חטא / לרע נטה / ושם בחוץ מושבו

ובספר מכתבו / כל בחור בישראל:





זכות תורה / תעוררה / חסד אל כל היום

לעם גלה / אשר חלה /ומצוותיה מקים

ולדלים / אל תכלים / צור הנורא ואיום

בשפע טובו מחיים / אשריך ישראל:





יצו צורי / בעבורי / מטטרון שר הפנים

ילוני / לארמוני / מפחד לגיונים

בבוא רוחי / למנוחי / להקביל שוכן מעונים

ומלאכי מעלה המונים / בבוא כל ישראל:
