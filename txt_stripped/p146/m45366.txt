


	
	אקום בחצות לילה
	



חתימה: אלשבזי משתא שמו






אקום בחצות לילה / אשיר שיר לאדוני

המרומם על תהלה / והוא אחד ואין שני

צור שומע תפלה / וניב שפתי ולשוני

יחיש לקץ הגאלה / לקבץ שבטי המוני

בזכות עוקד ונעקד / אלהי הושיעני:





לבנו יחידו יצחק / עקד לרצון קונו

להעלותו נתחזק / ובדק את סכינו

והנה איל נדחק / נאחז בסבך בקרנו

קראהו יה ממרחק / להקריבו תחת בנו

וכרת לו בהר המור / ברית עולם לזכרני:





שניהם עמדו בנס / ביום שהלכו יחדיו

ידיד יוי משתנס / וחרבו תחת למדיו

ויצחק מבלי אונס / עמו להחזיק ידיו

ועצי עולה תופס / ושומר דת תפקידיו

ושני נעריו עמו / הנהיגם אב המוני:





ביום השלישי ללכתו / לארץ המוריה

נתכון במחשבתו / בחושק לביא ולביה

למרחוק שם דעתו / ויוי שם היה



נם לנעריו: שבו פה / ויצחק בני ילוני:





זכו שניהם לעלות / במעלה העליונה

ויברכם נורא תהלות / ובאו עד חברונה

והעמיד מהם קהלות / שבטים בני ציונה

זרע יעקב איש תם / סגלת אל חי קוני:





יזכור לנו צורי / זכות חן שלושה אבות

ביום דין יאיר נרי / ויסיר כל מכאובות

ויקנא בעבורי / וישמח לבבות

ויוציא לאור דיני / וימחה את כל עווני:





מבור גלות ידלה / לעם קדשו שטבע

בזכות אבות יתלה / אשר הם בקרית ארבע

ועת רצון יגלה / ויוסיף חן בת שבע

ויקבל השבים / וימינו תסעדני:





שאלי בת הידידה / ביום זה מאלהיך

סליחה, ותני תודה / ורצי על בניך

תגלנה בנות יהודה / בקבוץ מפזריך

ואשמע כי בא מלכי / יפה עין אדמוני:





תכין לך הדרך / בן ישורון להתכונן

לעלות מבית פרך / קום לך ואל תתאונן

פשוט מלבך מורך / ושוב מחטא והתבונן

היום בספר חיים / אל שדי יכתבני:





אהבת כלילת יופי / יום זה בלבי עלה

ואני גולה בשופי / ולבי באהבה חלה

ואזכיר נבואת צופי / בישע ובהצלה

ונפשי הנבהלה / בהקיצי תשיחני:





מתימן שמעתי / דבר חזיון תנחומים

וכארבה נערתי / להזכיר זכות איש תמים

ובאלהי בטחתי / יציב גבולות עמים

למספר בני ישראל / בחותם אל נחתמים

מגדל עוז שם יוי / מנת חלקי ובטחוני:





שלשה אבות בקשו / יום זה רחמים עלי

למלא רחמים דרשו / לקשור באותות דגלי

ומשנאי ינתשו / ואשוב אל אהלי

ביום הדין יענשו / ותלהטם אש גאוני:





מקול שופר יהמו / סנטורי משטיני

כלם כאבן ידמו / ולא יזכרו עווני

ועם קדש לו חכמו / להשכיל זאת במשכני

לא באתי עד הנה / ולא יד צר תנידני:





ורצון צורי ישובב / לבנים הנאהבים

חסדו וטובו יסובב / שבטים השובבים

עמו אשר הוא חובב / ולעבדו נצבים

ומצדיקי הרבים / בזכותם ירצני:
