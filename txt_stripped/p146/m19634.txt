


	
	אגיל ואשמח... נפלו / סעדיה
	



חתימה: אני סעדיה






אגיל ואשמח / ולאל אשבח / ואזבח זבח / ביום
הפסח:



נפלו אויבי / והכו מריבי / ונמחלו חובי / ביום
הפסח:



יום זה קדוש / למועדים ראש / ברוך מקדש / יום
הפסח:



סביב היכלו / ענו מהללו / והקריבו לו / קרבן פסח:



על שם גאלה / ודיצה וצהלה / וחסד וחמלה / ביום
הפסח:



דן מצרים / צור שמים / בתוך המים / ביום הפסח:



ירד פרעה / ונכרת זרעו / ואבד ישעו / ביום
הפסח:



הללו אמונים / אדון האדונים / ופצחו רננים / ביום
הפסח:
