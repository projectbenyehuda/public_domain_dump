


	
	אימה בקשי
	



חתימה: אלשבזי






אימה בקשי על דל / המתאונן בחלדו

בעת רצון יופיע / ויומם יצוה חסדו

והכוני יחידתי / לצורך ועמדי נגדו

מי בכם ירא יוי / שומע בקול עבדו:





לקח מוסר בן דעת / קום קרא אל אלהיך

ולך אוזן שומעת / להשכילך בדתך

בלב טהור בלי בעת / נוטה לימינך

וחן תורה תושיה / בהקיצך תשיחך

אשרי איש נשוא עוון / שמאל מואס ומפרידו:





שלומי לב יבינו / לעשות טוב וגם יושר

ואל שדי בעזריהם / חוט שלחסד קשר

ובתפלה וסוד יחוד / נפשם ושכלם תישר

כשושנים יציצו / ודגליהם באות קשר

ורוח אלהים תניחם / וגוף ישוב אל יסודו:





בינו זאת קהל קדש / בטוב מלין ורמזים

ושמעו סוד והשכילו / דבר תורה וסוד חוזים

רצון אל חי בחן צדיק / ולו טובות נגנזים

יקרים הם ונחמדים / מזהב ומפזים

בעץ חיים נאחזים / ומה נעים פרי הודו:





זמן גרם לבן מלך / ומבית היכלו גלה

וממים זדונים / במשתה אויביו חלה

ולא מצא באר שבע / ביום בא אוהבי דלה

ומעניו בקול המה / והוא נודד בבהלה

רצה צורי בהצלה / ובימינך סמוך ידו:





יצו החן חמוד נפשי / לבן גלה ונתפזר

ויקשיב קול תפלתו / ומקדשו יהא נעזר

ואל ישפל בבית שביו / ולא יפול ביד אכזר

וילבישו עטרת חן / ודת פרא יהא נגזר

וכל ארצו באש תוזר / ואש תוקד בתוך מדו:
