


	
	בגלות גברת
	



חתימה: בן יוסף






בגלות גברת תקשיח / על דל מפורצי פרץ

ועמי בעול זמן מטריח / וירא מבעלי קרץ

ולבו כמו אש מקדיח / להבין סוד בעיון נמרץ

כי אלהי לי הבטיח / לעלות במלכות בן פרץ

ממכון שבתו השגיח / על כל יושבי הארץ:





נחשבתי עם יורדי בור / נבזה, ויוי חלקי

בארץ אויבי מתחבר / ביחוד אל שדי חשקי

ופרא עובר מתעבר / ומבקש תמיד לעשקי

גאה בחילו מתגבר / ומכחיש דת דיני צדקי

צורי רצה עבד שובר / הטריפני לחם חקי

אשר עשיתי בסתר / רקמתי בתחתיות ארץ:





יום אויבי עלי עמד / ותחבולות דברים חתר

פרא ואיש שעיר צמד / להשמיד פליטי בסתר

ולבי וגם שכלי חמד / לעלות אל הרי בתר

כי בא ידידי זכות למד / יום חץ כוננו על יתר

ואני יושב בסתר / עליון למלכי ארץ:





ואראה רשע עריץ / גאה וגבה עינים

עלי תחבולות מפריץ / ואני בין המשפתים

ופיהו בנבלה מחמיץ / על בן מחולת מחנים

בעיניו להרע לי מקריץ / בזכרי את ירושלים

מתי צור קוני יציץ / ויתן שכרי כפלים

ישמחו השמים / ותגל הארץ:





סגלתו בחן יאמירה / להם בארצו יניח

ישמח לבי ולא יירא / בשיר ותהלה אפציח

חסדי יוי אזכירה / יבשרני בבוא משיח

ובעוד שחר אעירה / היום באורו יפיח

לאל חי אשירה / ואור זרוע יזריח

צמח ישועות יצמיח / וצדיקים יירשו ארץ:





פלאי אלהים מי יגיד / רוקע ארץ על מים

וכלם בשערה העמיד / ושם קו ירוק בנתים

ומשגיח עליהם תמיד / וגלגל סובב כרחים

וכוכבים קול שיר תגיד / בלי לשון ושפתים

האזינו השמים / ותשמע הארץ:
