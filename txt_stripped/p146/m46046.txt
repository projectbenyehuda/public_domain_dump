


	
	עוררי בת
	



חתימה: שלם בן שבזי משתא






עוררי בת נדיב / עם פזורי שביון

שיר והלל תיטיב / משבח לאל עליון

כי לבבי מקשיב / הוא ושכל רעיון

אחפצה סוד הגיון / בחצר הר ציון:





שובבי עם גולים / רעיה כי שאו

לך באהבה חולים / אז כציץ יזרחו

חזקי נכשלים / זה בזה יתאחו

ותמכי כל רפיון / בחצר הר ציון:





לי תהא תקנה / תוך סגלת עמי

רחמן עד אנה / צר יהא משטיני

ממך תכנה / קום דרוש נא דמי

אז יהא לי רתיון / בחצר הר ציון:





ממקור הצדק / על פזורים שורה

חזקה כל בדק / דוד ונשמח פורה

איך אהובי חמק / ואני צמרורה

רחמה דל אביון / בחצר הר ציון:





כך בטוחים יחד / עם אשר חקקת

כי אחזם פחד / יום אשר הרחקת

בן אמתי הכחד / כי מרומים אתה

כוננת אפריון / בחצר הר ציון:





נחמה דל צופה / חסדך צור הכל

כי שאון צר חופה / חוץ לאוהל בת כל

קום צלח דוד יפה / ועצתו סכל

יאחזנו שגיון / בחצר הר ציון:





שם אלהי עזרי / מכלל הקמים

זיו סגלת ישרי / חי עדי עולמים

חיל משנאי יפריא / אז יהו נדהמים

ישכרו בחמת יין / בחצר הר ציון:





באלהי עזרי / מכלל זדונים

בת נדיבים אמרי / לו בשיר ורננים

יעלה טוב זכרי / בין שרידים חונים

סודרי דת הגיון / בחצר הר ציון:





זרזו שכליכם / כל בני איש תמים

אל מחוז חפציכם / רם עלי נעצמים

הוא כעת ינחיכם / עם גבולות עמים

נחלקו בחזיון / בחצר הר ציון:





יום ליום יביע / מאלהי אומר

אז לבן יושיע / בעבור דת שמר

נפזרים ישמיע / שיר והלל זמר

נמצאה סוד חביון / בחצר הר ציון:





מהרה בגאלה / ממעון קדשך

רחמה לסגלה / דורשה חסדך

תענה בתהלה / נכספה אליך

תחפשה […] / בחצר הר ציון:





שחרי נשמתי / בתפלה עמדי

עוררי אהבתי / סוד בתורה למדי

כי בזה חמדתי / מתחלת עמדי

אמצאה גליון / בחצר הר ציון:





תוך ארון הקדש / אור מהדר נגלל

תשמעה קול רעש / ממפיצי מהלל

אך אהובי החש / דע פרטים וכלל

מלומדי הגיון / בחצר הר ציון:





אחתמה שיר נועם / משפתי נפצה

בו תנועת טעם / סוד לצורה נרצה

יה פדה לב הועם / עמך חן ימצא

תן לעניי נשיון / בחצר הר ציון:
