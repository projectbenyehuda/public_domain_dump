


	
	ברכי אמונה ביד
	



ד. ראש חודש

חתימה: א״ב, שלם אלשבזי משתא






ברכי אמונה ביד אלהיך שומר אמונים

את שם אל יחיד ומיוחד בפי עם קודש הגונים

אוהבים להגות בתורתו ולשלחנו זמונים

אשרי איש ירא את יוי

ברכי נפשי את יוי:





ברכי בעלת חמשה שמות קבועים

את שם בורא מציאות הנסתרים והידועים

בחר בזרע יעקב גנת כרם נטועים

בו בחר יוי

ברכי נפשי את יוי:





ברכי גברת המתקשרת בארבע יסודות

את שם גוזר ומצוה בחקים ועדות

גדול העצה ולשמו טוב להודות

גדול וקדוש יוי

ברכי נפשי את יוי:





ברכי דוברת צדק לעשות מישרים

את שם דין אלמנות עליון על גלגלים ושרים

דבר למשה לצוותנו בחקים ישרים

דבר שלח יוי

ברכי נפשי את יוי:





ברכי הדורת קומה בת מלך חשוקה

את שם המושיעך מהבלי זמן ומצוקה

העירי שכל אנוש משוחה עמוקה

הללו את שם יוי

ברכי נפשי את יוי:





ברכי ועודה לערוך מעשיך יום דין וחשבון

את שם ותיק דין על כל יצוריו ורבון

ואם תעוררי יצר טוב ותתני ערבון

והקרבתם אשה ליוי

ברכי נפשי את יוי:





ברכי זבודה בזבד טוב מיקר עליון

את שם זוכר הברית לעם עני ואביון

זכי פעלות הגוף ושכל הרעיון

זכרון לפני יוי

ברכי נפשי את יוי:





ברכי חכמה המסככת על בנין הגוף

את שם חי העולמים ממציא בלי שתוף

חוקר כליות ולב וסוד מאמרו צרוף

חונה מלאך יוי

ברכי נפשי את יוי:





ברכי טובת מעשה המחזקת במצוות

את שם טהור עינים ודרך רשעים יעות

טובו וחסדו לעולמי עד ובריותיו לטובו מקוות

טוב לקוייו יוי

ברכי נפשי את יוי:





ברכי יקרה מטנופי גוף ורוב הבליו

את שם יוצר הכל ונותן לאיש כמעלליו

יודעים סוד ומשרה וחוכים ונוהרים אליו

ישראל בטח ביוי

ברכי נפשי את יוי:





ברכי כמוסה ביסוד צדיק שמורה

את שם כביר כוח הקושר גולם וצורה

כתבי על לוח לבך תורת אלהיך תשורה

כי ישר דעת יוי

ברכי נפשי את יוי:





ברכי לובשת הוד שפוע מהר הכרמל

את שם לוקחך עליון מגלגלים וחשמל

לקראת אלהיך תשובי וחסדו אליך יגמל

לראות בטוב יוי

ברכי נפשי את יוי:





ברכי משוכה בחוט שלחסד קשורה

את שם מחייך במזגי הגולם עצורה

מצאו לכם סופר בתשובה גמורה

מנחה היא ליוי

ברכי נפשי את יוי:





ברכי נקיה לשלחן אלהיך קרואה

את שם נאמן בבריתו וכסאו למעלה נשואה

נעלם מכל עין וטובתו לכל מצואה

נפשנו חכתה ליוי

ברכי נפשי את יוי:





ברכי סועדת הגוף מזיו כבוד השכינה

את שם סובל העולם כמו הים לספינה

שאי שלום ובשרי בעלי דעת נכונה

שמחו צדיקים וגילו ביוי

ברכי נפשי את יוי:





ברכי עוסקת במצוות לשמור לעשות

את שם עושה רצון יראיו ובו טוב לחסות

ענני כבודו מתפרשזות ומתרכסות

עושה כלם יוי

ברכי נפשי את יוי:





ברכי פוצחת שיר להלל לקונך

את שם פועלך ומזיו שכינתו מזונך

פתי גולם וישוב מחטא ואמרת ללשונך

פציתי פי אל יוי

ברכי נפשי את יוי:





ברכי צופה נועם אלהיך לראות

את שם צור העולמים אלהי הצבאות

צפי לישועתו וקחי תורתו אות

צורי וגואלי יוי

ברכי נפשי את יוי:





ברכי קרואה להיכל מלך פנימה

את שם קדוש ומקדש בשיר ובנעימה

קבע שני המאורות וכוכבים וכסיל וכימה

קוה קויתי יוי

ברכי נפשי את יוי:





ברכי רצויה לעשות רצון אלהיך

את שם רם ונשא ומשגיח על כל פעליך

רבת בנים בשרי כי יחיש גואליך

רוח אפנו משיח יוי

ברכי נפשי את יוי:





ברכי שמורה מכל חטא וסורי הבלים

את שם שדי מרומם על כל מהללים

שמו מיוחד בפי זקנים ובחורים ועוללים

שמע ישראל יוי

ברכי נפשי את יוי:





ברכי תלויה בחוט שלחסד משוכה

את שם תמים אשר את בשמו ברוכה

תני חן והעיצי בן גברת במלוכה

תמים תהיה עם יוי

ברכי נפשי את יוי:





שועתי לטובך ממציא הכל ומחים

אלהי לעבדך שלח בפדיום

זכות ישרים מפציחי שבח תפארתך איום

הושיענו יוי

ברכי נפשי את יוי:
