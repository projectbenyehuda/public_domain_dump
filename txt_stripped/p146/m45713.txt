


	
	אל נורא
	



חתימה: אלשבזי






אל נורא / יעוררה / חן על קהלה מחכה

כי חשתי / ושמעתי / את קול רחל מבכה

קול ברמה / נעלמה / העיר את המלכה

תאמר רצה / צורי וצא / עזור שפל רוח דכא



אלי אתה / ואודך / אלהי ארוממך:





להציל רש / אשר גורש / יחיש אלהי ישועות

נשבה ביום / חרון איום / ומבית אבותיו שם עת

לקרוא דרור / ולעורר / נפשות הנצנעות



וצור מקים / צדיקים / עד מה אשור תשבך:





שבח תנעים / בת נעים / להודות רם וגם נשא

וגן ביתן / לבן איתן / הדר היכל מבוססה

והוד רוחות / כמטפחות / על נפשות באור כסה



יה שכן / לדל מסכן / תחי נפשי ותהללך:





בשמאלי / גער אלי / ואל יטעה יחידתי

וקרר אש / והתלבש / בחסדך צור ישועתי

ושים חשקי / בך חלקי / ורחם על סגלתי



בעת תחים / ומן הים / תמשוך לויתן בחכה:





זמן לבוש / כסות ללבוש / כי אתה מבצרי

וסוד טובי / וכל יהבי / עליך ומחסורי

לך נפשי / וגוף חלשי / ואתה ישעי ואורי



ענה עתרת / שחרחרת / כסמים ולבונה זכה:





יקר ישת / ויתעשת / האלהים להצילה

לאביונים / בקול עונים / לדרוש חן ומחילה

וירצה בם / בעת שובם / בעת רצון יקבילה



ויבשר / למתיסר / עטרת תפארת תמגנך:
