


	
	מסאן פי רצׄא רבי עלי שעבה תגׄלא
	



החתימה: משתא אלשבזי (84 – 85: סין אלף לאם ומים; ולד יוסף)




מסאן פי רצׄא רבי / עלי שעבה תגלא



אנאר שרקי וגרבי / באחסאנן ופצׄלא



בגׄאהה אלתטף בי / ואועדני בקולא



ונורה חאט קרבי / עלי מא כאן קבלא



שריף אלדין ארסל לי



דׄכי אלנפס ואלעקלי



וינפי פאעל אלגׄהלי



חמוד נפשי וגם לבי / לסודותי מגלה



בכל שחרי וערבי / שאלותי ימלא:



שזפתיו ואני יושב / בתימן נבהלי



ולבי מעשיו חושב / ודורש פועלי



וסוד תורה בחן קושב / ושכלי צוהלי



ושמחתי וסר עצבי / בכׄירן שאמלא



בלג שוקי אלי אלקדסי



ורית אלמלך פי חסי



תהיא ואזהי אלכרסי



שמח דודי ורוץ כצבי / ועודד דל מחלה



שמאלך ידיד חרבי / תרימני ותדלה:



תמכני ידיד נפשי / בפורים לעלות



ולוני להר קדשי / ברומי מעלות



ושם חלקי בקו ששי / ורפא מחלות



ודשנתי בטוב חלבי / ונפל צר כעלה



ורעיה שמחה עם דוד



ושחו ממזרי אשדוד



ושב צפור שהוא בודד



וסרעפי בתוך קרבי / שכר ללה תעלה



ותוחיד אסמהו חסבי / אכׄירן תׄם קבלא:



אהובי בך אני מודה / וארוץ עמך



ואעזב הון וגם שדה / ואבוא אצלך



ועל פשעי אני מודה / רצה על עבדך



תעוצׄני תזיל גלבי / ובעד אלצׄיק אסלא



ואנצׄר היאת אלקרבאן



ויסלי כׄאטרי אלתעבאן



באהל אלעלם ואלרבאן



לנא יפתון מא יגבי / אלאחבאר אלאגׄלא



ומאהם מנבע אלעדׄבי / שביה אלנחל ואלחלא:



אהובי החמוד נתן / עלי הדל תבקש



ועל ביתי והמפתן / שלח יד על מעקש



ונכנס בה יהא נרתן / ותהיה לו למוקש



ועליו יגזרה תשבי / יהא שוטה וחולה



ולי יה יתנה כפל



וירים כל בני שפל



ואורה תנצחה אפל



וזאל אלכׄוף ואלרעבי / ומלך אללה תולא



וחיא כל מתנבי / וטאב מן צאם וצלא:



לבוש סיע וגם לחם / למשתוקק וגם דל



אלהי עבדך רחם / וקבל נחדל



ואת כל לוחמי תלחם / ואל נא יגדל



ימין האל הלא תצביא / לשגב דל מחלה



הדר ציון חזת עיני



ואליה וינוני



ואז אשיר לצור קוני



בשיר חדש נאם ניבי / ינבה לאלגפולא



יגׄלי אלהם ואלכרבי / לקלב אלמסתדׄלא:



שבא שבתי צבי וצלח / עלי מלכותך



נבזבז לידיד תשלח / לבבי עמך



ואש על אויבי תדלח / וירום הודך



גאוניהם יהו שבי / יצאלון אלבלא



ואדעו אללה יעוצׄני



מעא אלאלאסבאט ירחמני



ידׄל מן כאן יהמלני



ואנסי שר חטאם שגבי / אנאל פיץׄ אלכמאלא



יהא שבתי ומצבי / בצל נורא תהלה:



בהר זיתים גבול בקע / ידידי יום עלות



ובן אמה אויל פקע / ירכיו נופלות



גאון שרו תהום שקע / ורגליו צוללות



שלומות רעיה כתבי / וכׄצי לאלמהלא



גׄמיע סלסלת יעקוב



תקי ללה מחבוב



ופי אלתוראת מנסוב



ובשדה אהבה רגבי / לכלאים אפלא



ועלה משאול קטבי / בתפארת מסלא:



זריזים סנדרי לידע / הלכות רוחות



ויורון כל בני דרדע / בסוד המפתחות



וקרן כוזבים נגדע / ועיני זורחות



ואשלוך על ידיד יהבי / אשר אויבי מכלה



והוא חשקי וכל חפצי



ועוז חיי בתוך ארצי



אשר הזריח לבן פרצי



גשי נפשי והתרבי / קני חכמה לנחלה



והאריכי במרחבי / לראשי עב יטלא:



יקול אלעבד אנא אלחלאם / בפכרי פי אלליאלי



לעארף סבעת אלאקלאם / אריד מטלב סואלי



ואסמי סין אלף לאם / ומים פי אלנצׄם סאלי



ולד יוסף פקד אגׄבי / למטלבהו בסהלא



אקו צורי ואלי



בחסדו ימחלה מעלי



ועליו מחמל חדלי



יטיב עישי ושרבי / בסמח אלמסתצׄלא



מחול צורי ואבי / לעבדן צאם וצלא:
