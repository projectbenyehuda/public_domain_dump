


	
	אילת אהבים רצי
	



החתימה: אלשבזי






אילת אהבים / רצי על המוני

כי לקחום שובים / חבורת משטיני

ויכתבו כתבים / להעיץ על צפוני

עוררי שובבים / נערי וזקני

נחפשה דרכינו / ונשובה עד יוי:





למה נתעלם / בבור גלות הומים

ואיך נסבול עלם / מרודים ואשמים

נעלה לעיר שלם / מגבולות עמים

ונשוב ונשלם / פר ואיל תמים

לחי העולמים / ה[…] יוי

נחפשה דרכינו / ונשובה עד יוי:





שבות עמך תשיב / אלהי ישענו

ברחמים תקשיב / לקול תפלתנו

בעבור בת נדיב / עטרת ראשנו

לעיר ציון תיטיב / שם מחוז חפצנו

תן מאויינו / הה[…] חמה יוי

נחפשה דרכינו / ונשובה עד יוי:





בחסדך ורחמיך / תעזרנו מהרה

תכבוש כעסך / ותסיר קטיגורה

תנהג בניך / בתוך קרן אורה

בעבור כבודיך / ובגין אבות ותורה

עמך תעוררה / הבוטחים ביוי

נחפשה דרכינו / ונשובה עד יוי:





זכור יום גאלתנו / ביד רועה נאמן

עבורו פלטנו / בעת רצון נזמן

בכבוד פרנסנו / בזכות אוכלי מן

תקבץ פזורנו / אשר נפרד בתימן

וזה יהיה סימן / טוב לחסות ביוי

נחפשה דרכינו / ונשובה עד יוי:





ישועה לבני תם / יום זה והרוח

תתבשר עדתם / ברצון והצלח

ירצה תשובתם / מוחל וסולח

ותעלה תפלתם / כמו קרבנות ריח

מעשה רקח / קודש ליוי

נחפשה דרכינו / ונשובה עד יוי:
