


	
	מהרהור לבי
	



חתימה: ממשתא






מהרהור לבי ועיני / לבי רותת מענשו

חליתי מרוב יגוני / והבעיר בי לבי אשו

בלבל דעתי ורעיוני / וחזר מצטער בחלשו

ברכי נפשי את יוי / וכל קרבי את שם קדשו:





מרוב יסורין שחתי / ורפא אלהי מכאובי

ומחטא ועוון חדלתי / והרהור עיני ולבבי

לבקש רחמים עמדתי / אולי ימשני אבי

אמרתי וכל אשר לאיש / יתן בעד נפשו:





שויתי יוי לנגדי / יודע כל תעלומות

וכאין נגדו חלדי / ומעשי בכתב רשומות

ויחוד קוני מחמדי / הנקדש בשבעים שמות

והחי ישים אל לבו / ונתן פדיון נפשו:





תאות נפשי רצון קוני / יגער באויבי ויפדני

ושמאל מנאץ מסיתני / גורם חטאי ורוב עווני

צורי בחסדו יקימני / ירצה ביום דין יזכני

וביום תחיה יחייני / חסדו כעץ יפרחה שרשו:





אחרית ותקוה לעם גולה / עושים תשובה ומתענים

ישתו וירוו בלי דולה / נפשם בתוך מעין גנים

זכרם ביום זה לטוב עולה / שבים בטהרה כשושנים

מזיו שכינה יקר נהנים / מדי חדש בחדשו:
