


	
	כפור אשר
	



חתימה: שלם יוסף






כפור אשר עבר / יום המחילה

אחריו תבשרו / בשמחה וגילה:





שלום להדרתכם / זרע ישרים

הודו לאל נורא / משפיל ומרים

ירבה בשמחתכם / זרע חברים

כי כלכם זכים / מכל מעילה:





לכם להתבונן / לשאול צרכים

החן והחסד / יחד משוכים

מה טוב להדרתכם / כבני מלכים

ההוד יסובבכם / כל יום חלילה:





מכל חטאיכם / יחד טהרתם

ממעיני צדק / שלום משכתם

לאור באור חיים / דודי אהבתם

שלום כמו נהר / סובב חוילה:





ישע שלח לנו / צור ממרומך

ובנה לעיר ציון / היכל הדומך

ולכל בני תורה / תשפיע שלומך

כי הם לך צופים / נורא עלילה:





וראה עני יונה / גולה עלובה

עורר ישועתך / תשוב אהובה

כמה בגלותה / תהיה עזובה

מעול משנאיה / תצעק בחילה:





ששון וחן ויקר / לנו תמהר

מבין בני עולה / עמך תטהר

תשפיע לאור הודך / בהיר ונוהר

ולאויבי עמך / תהיה נפילה:





פתאום שלח צמח / לנו יקבץ

ישיב שבותנו / יחד תאמץ

תדין לכל עול / מאוס וחומץ

יכבה כנר נמשך / מן הפתילה:





אחתום בשלום רב / על הקהלה

ואשבחה לאל / בדבר תהלה

ושמי בראש בתים / ואבי ממלא

תמיד אני צופה / יום הגאלה:
