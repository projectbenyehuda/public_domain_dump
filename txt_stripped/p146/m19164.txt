


	
	יום אזכרה ציון ויהמה לבי
	



חתימה: שבזי




יום אזכרה ציון / ויהמה לבי



שכל חמוד רעיון / שואן בתוך קרבי



ונאום דבר חזיון / יעיר צפון חבי



אקרא לאל עליון / צורי ראה עלבי,



שועי אהב / חלשי רהב



יהבי יהב / המר זהב / חטאי בעשת שן.



לב צר בקע / רומי שקע



שורה נקע / עלי רקע / שלום כמו פישון.



מלכי רצה / דל חלצו



קום אמצו / אל ילחצו / דישן ולא דישון.



ציון עדי תקפך / צפי מבוא צופך.



שוכן שמי מעלה / נחם לעם דלים



להם תנה חמלה / צורי ואל תכלים



לדל אשר גלה / הט חן ואל תעלים



חתן עלי כלה / מילת לבוש יגלים,



שורה שעה / קול יודעי



דת שומעי / גין שופעי / דרשה וקבלה.



לבני ברית / שים אחרית



טובה ורת/יון אחרית / טורף אשר שלה



מימינך / חוט תמשכה



לידידך / ותסוככה / על עם אשר חלה



אויב אשר שפך / רוגזו לפיו תהפך.&gt;


בעבור גביר נפקד / לעשות רצוניך



עוקד והנעקד / ואבי שבטיך



תמחל לחטא נשקד / לבני בריתך



ואדום באש יוקד / מכחיש אמתיך,



שלחה פדות / השב שבות



חיש קוממות / גזע לרות / ינון משיחנו.



לו תמשחה / אז יצלחה



בך יבטחה / יום נשמחה / תוך מאוינו,



מצרפת / בשר יפת



עין. שפת / דלות צפת / צופים פדותינו.



ספיר וטוב נפך / היך סג ונשתפך.



זכרה זכות שבעים / וזכות ילוד עמרם



אהרן שהוא נעים / כהן לך אל רם



וזכור לארבעים / יום לבשו כתרם



חון ושלחה רועים / אל חי מנת סברם



שפעת רביב / תרפא עציב



מעין נגיב / תנהיר רביב / ציון והאולם.



לקראת גדוד / ישוב חמוד



קץ הנדוד / אויביו שדוד / הכה – וסר צלם,



מזיוך / צור סככה



יונתך / רעיתך / צופה לכל נעלם.



נפשי חלי הרפך / חזקי בסרעפך.



ירצה צבי החן / ישיב לבת גלים



אז ימצאו פתחון / מלאך וגלגלים



ויעוררו בטחון / שפע בים גלים,



אחרון שהוא בוחן / יקר וזלזלים.



מימיו קבוץ / יה לחלוץ



בנך אמוץ / ילבש לבוץ / יעלה לרום ארמון.



שמעה נהי / רחל שהיא



בוכה והיא / תאמר יהי / רצון להזמן.



תשמח ברב / שיר יערב



צר יחרב / חפץ קרב / אשוב לאיש קדמון



שיחה לדוד תשפך / מצפך ובמנצפך.



שירי חתמתיו זה / בשבח לצור קוני



נורא ולא יבזה / דלות ענות עני



עלי בחן יזה / וינחמה אוני



שמן אדום ירזה / ואני יחיני,



קדוש דרוש / גולה ובוש



בגדו טמוש / ולחן לבוש מימינך



נצרה לדל / הנחדל



מחטא בדל / ואמור גדל / מלכי והתיסד,



מקרנך תשפך / עליו ולא מפך.
