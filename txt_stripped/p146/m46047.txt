


	
	שארית עמך
	



חתימה: שלם יוסף






שארית עמך יה / מקוה חסדך

שעה אל שועתו / בקראו אליך

בשרו במחילה / עשה למען שמך

כי השליך יהבו / ביום זה עליך

יאמר זכור לי אל / לאהבתך

תמוך אשורי / במעגלותיך:





לבבו הכין / מכל טנף

היה עזרתו / בשעת קצף

ואל ימשול בו / אף וזעף

והציל נפשו / מכל טרף

כי היא לנגדך / משתפכה:





מחה נא כעב / חטאה ופשעה

ואל תדינה / כזדון רשעה

כי הוא אתה / יה צור ישעה

הרחק ממה / לעוון בצעה

אולי תמצא / חן בעיניך:





ילידי אב המון / בך יבטחו

ובזכרך / הם ישמחו

ראה נא ענים / כי אז שחו

ובגלותם / נשתלחו

כי חטאו על / מצוותיך:





ודוייהם / תקבל ברצון

כי הם בדלות / נפוצים כצאן

ביד כל לועג / ומרבה לצון

ישב בפנים / והם בחיצון

עשה עמהם / למען שמך:





סלח לעוונם / ולפשעיהם

וברחמיך / תקבילהם

ותציל אותם / משוסיהם

המכבידים את / עליהם

והם עמך / ונחלתך:





פדותך תחיש / ליונה עלובה

כי מקדם / היא לך אהובה

ובמעשיה / טובה חשובה

והיום באה / בתום ובתשובה

לבקש ממך / סליחתך:
