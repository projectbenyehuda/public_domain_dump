


	
	אל מרומם
	



ו. מילה

החתימה: א״ב שלם בירב יהוסף […] בן משתא






אל מרומם על תהלה / אבינו מלכנו

צוה לאברהם תחלה / ראשית אבותינו

למול בשר הערלה / לטהר את נפשנו

להסיר זהמא מגלה / לישר ארחותינו:





בזכר זו מצוה / כי היא כנגד זכור

להסיר זהמא נדוה / מצד עמק עכור

זרק נחש בחוה / והצרכה אל עין סוכר

כי בחשקו נתאוה / לעורר חטא עלינו:





גבולות עמים השיג / חוץ מגבול צמרת

וחיי אדם הפסיג / אבל נפשו נשמרת

צרופה ככסף מסיג / לגן עדן חוזרת

ואיש שדה המלעיג / לא יבוא בעירנו:





דבר בזה אל עליון / לאברהם חק דתו

הופיע מציון / והלבישו נבואתו

ואמר לו בסוד רעיון / למול בשר ערלתו

ובניו עשיר ואביון / מחזיקים בדתנו:





הלא היא אות ברית קדש / בכסא מעלה רשומה

וכל העובר נענש / בכרת נפשו עגומה

ואם בשמיני חלש / יבריא בערלה אטומה

ואחר ימים תומש / כדת בני עמנו:





ושלוש עשרה בריתות / על המילה נאמרו

בשלוש עשרה מדות / שלרחמים נתבררו

לזרע יעקב למודות / שבטי יה נבחרו

האומרים טוב להודות / ליוי אלהינו:





תורת משה / זרע קדושים כלכם

אשר מאדונים קשה / השלח והוציא דגלכם

ולעלות לסיני הרשה / ותורה ודת הנחילכם

וסלק שמאל הנושה / ביום בא בהמונינו:





חסד לאברהם חן / ושבטי יה הקדושים

ופחד יצחק נצחן / יקרא אליכם אישים

ויעקב איש תם נבחן / הוא ראש השלישים

וצדיקים טוב ריחן / כי בו ישמח לבנו:





טללי רצון וברכה / שורה בעליונים

ולבת כל בהמשכה / יורדים פנים בפנים

ומלאך וגלגל ככה / מקבלים והתחתונים

וקהל יעקב זכה / בשפע טוב יחינו:





יחינו מיומים / ויקימנו בשלישי

ובפתח עינים / עתה ירום ראשי

זכות מילה בנתים / תזכה שכלי ונפשי

ובזכות לוחות שנים / נמצא מחוז חפצנו:





כך דת מילה נוהג / ביום השמיני תמיד

ודוחה הוא את השבת / מלכות ויסוד תצמיד

וסוף שעה אחרונה / אם נולד שבות העמיד

ומול אותו ביום ראשון / כפי דת רבותינו:





לכך הנולד בערב / שבת בין השמשות

יום העשירי תערב / מילתו ואל תשת

ספק במנין שמונה / ימים המפורשות

ואם ראשון ערב יום טוב / באחת עשרה ימול ידידנו:





מלבד ראש השנה / אם יום ראשון ערבו

תהא מילת זה הגונה / בשנים עשר לבוא

בברית קדש אמונה / לאיש אשר ישית לבו

וחפש בהלכות מילה / ושם תמצא דברנו:





נורא ואיום שותף / לתן לנו שכר

וזרע יעקב נסף / קדשה על בן נכר

בענן שכינה יקף / בהמול לו כל זכר

ונרגן וחילו נדחף / ויוי בראשנו:





סגלת כל העמים / מה נאוו מעשיכם

בזכות שבת ומילה / רע לא יגע אליכם

נקיים מכל אשמים / בני אל חי אשריכם

בסוד מצות איש תמים / יקום בעזרתנו:





על זקנינו יחמול / ובחורים וטף ונשים

וחסד לנו יגמול / נורא בצבאות קדושים

יברך זה הנמול / ועם זה הנגשים

מזמנים מאתמול / לעשות רצון יוצרנו:





פדות ורפואה ישלח / אלהי קדם מעונה

והנמול זה יצלח / בסוד דעת עליונה

ובחן זכות ברית מלח / נעלה אל ציונה

ואליהו ישלח / בגאלה יבשרנו:





צופה עתידות ירחם / על כל בני הגולה

את בת ציון ינחם / ירפא נפש חולה

וישמיד אויב לוחם / וירצה שועי כעולה

פזורי עמו ינחם / ויחיש להושיענו:





קורא כל הדורות / מראש סדרי בראשית

בעשרה מאמרות / כל המצואים השית

ונפשות הטהורות / מדברת ואנושית

בכסא כבודו מאירות / גנת אגוז חלקנו:





רועד לבי וכוסף / להחזיק בעוז המגדל

ישפיע מאין סוף / צורי על עבד דל

שלם בירב יהוסף / המחבר הנחדל

בן משתא המאסף / טעמי תורתנו:





שלום לכם בני ברית / כמו נהר שופע

בלשון קודש ועברית / ומכל חבר ורע

תהיו לשם ולשארית / ברכה בסוד הצנע

מראשית ועד אחרית / אלהים יחננו ויברכנו:





תומך כל דל ואביון / יקבץ פזורי עמו

מי יתן מציון / תשועתי למען שמו

ונראה גי חזיון / לחזות את מנעמו

וינון על אפריון / ידידי ברוך טעמו

וסנדרי בעלי הגיון / להאיר עין שכלנו:
