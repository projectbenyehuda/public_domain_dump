


	
	שובו שובבים
	



חתימה: שלם בירב יוסף משתא






שובו שובבים / וחכו לצור ישעכם

לשמח נעצבים / ולהחיות רוחכם

בנחמת נדיבים / אשר נבאו בגינכם

ויכתבו כתבים / להשמיע אזנכם

אתם נצבים / היום כלכם:





ליום הדין הכנה / לבת גלים שבה

קהל עדת מי מנה / במרום נצבה

מראש השנה / ועד ערב ערבה

ובת מלכים חונה / והנה דודי בא

ושם רבוא רבבה / וראיתם ושש לבכם:





מחילה וסליחה / חבצלת תצפה

לצורה תשיחה / ולו ישבח כל פה

עדת אל תרויחה / בחצר דוד יפה

וקודם בוא צופה / ריבו באמכם:





בהלה אחזה / בני שעיר ואמה

ומדת דין נתזה / למגדלם דמה

חיליהם בזזה / עדת אל ביד רמה

ובשובה עלזה / לדודה השלמה

ועליה שמה / טרחכם ומשאכם:





ישע מתבשרים / בני יעקב כלם

ולבותם משברים / לצור קונם גואלם

וחן רעיה מעוררים / בדברי מהללם

משבחים ומזמרים / ומצליחים עמלם

ואומרים בהתפללם / שיתו לבכם:





רחמים וישועה / לעם גולים בענים

גנה הנטועה / בנהרות דכים

בצדקה זרועה / כבוד האל מחים

ומעתיקי שמועה / יקר שכינה עדים

ועונים בניב צבים / יהי כן יוי עמכם:





בעת רצון פצה / פי שבח לצור קוני

וחן ושכל מצא / בנועם הגיוני

ונפשי נועצה / תשובה תלמדני

בהבלי שמאל קצה / ותעורר המוני

וכלה ונחרצה / השלח בשוטמני

יה ישמיעני / והתהלכתי בתוככם:





יכתבון ידידים / בספר זכרון

לחיים ועודים / ישובו לבצרון

כלל בני היהודים / סגלת היתרון

שבח והלל תקדים / חבצלת שרון

עבור משה ואהרן / חיים כלכם:





ותשמע אזני / שמץ מנביאים

כי אשוב לכני / ביום קבוץ טלאים

לארץ מעדני / מארץ רפאים

וטל אורות גני / יחיה נדכאים

צאו אסירי כלאים / ותחי נפשכם:





סבלה גויתי / יסורי הזמן

כי ברוב אשמתי / חש לבי ונטמן

וגרם לרעיתי / לעזוב דוד נאמן

ושלט בן אמתי / על המוני בתימן

פזורים תנו סימן / וישבתם לבטח בארצכם:





פדות לבני גברת / בימינו יחיש

ויזכור ברית כרת / רם ונשא, להלביש

עטרת תפארת / לינון, ולהכתיש

לעשו אשר חרת / משטמה והבאיש

ופרא ישבע ריש / לכו באור אשכם:





מארבע פאות / יקבץ לעדתו

אלהי הצבאות / משלשים קדשתו

ולרוחות נכאות / מחיה באהבתו

וכלן נקראות / לעת בוא ישועתו

ונם לסגלתו / לכו אל ארצכם:





שפריר מעזו / ילביש גברת עוז

ולו נפלאות רזו / בחותמו גונז

ועל סגלת מחזי / נביאים הוד יפרשז

ושמץ ברמזו / קול דממה אוחז

אם יהיה נביאכם:





תחתור לי חתירה / אלהי לקבלני

כי לבבי יירא / בהפקד עווני

ברחמים האירה / נתיבי למשכני

ועלי העירה / זכות אהרן אדוני

ואמור לסמכני / אנכי אנכי הוא מנחמכם:





אורחותי יכונן / צור, מחסה לדלים

ואשוב ואתבונן / במעשי הנקלים

כשכור מתרונן / פשעיו מתגלים

אלולי יה חונן / כים סובל גלים

הבו ליוי בני אלים / להשמיע במרום קולכם:
