


	
	אל עליון
	



חתימה: אלשבזי






אל עליון / דלה אביון / אשר טבע בבית שביה

ביום חטא / ויד מטה / ועזב דוד וגם רעיה

רצונך חיש / לבן מכחיש / שהוא ערום וגם עריה



אל איום / שלח פדיום / לבת גלים צופיה:





לך תקרא / תעזרה / ותקומה בעזרתה

שמע קולה / והרפא לה / והחישה ישועתה

כבוש עוון / ומר דוון / ורפא את משובתה

תעורר חן / בני בטחון / לבת נוגה והומיה



אל איום / שלח פדיום / לבת גלים צופיה:





שבות עמך / אשר נמך / תשובב אל נוה ארצו

ויתאפק / דל דופק / דלתיך להמצא

ואל ייעף / ביום זעף / בשעבודו ובית לחצו

הצילו / מבעלי / לשון הרע ורמיה



אל איום / שלח פדיום / לבת גלים צופיה:





בטוחה בך / יונתך / צור מעוז לבוטח

רצונך הב / לעם נאהב / תהא מוחל וסולח

שפע הודך / לבן עבדך / כאור שמש וירח

והעבירה / והסירה / תאניה ואניה



אל איום / שלח פדיום / לבת גלים צופיה:





זכות עורר / ואף קודר / ואל תקצוף עלי הדל

אשר גלה / בחלחלה / ועזב הון ועיר מגדל

והשיבו / למושבו / ביום קראו ולא יחדל

בגין אברם / ובן עמרם / ועקוד ותם ותושיה



אל איום / שלח פדיום / לבת גלים צופיה:





יבשר שב / וטוב יושב / ויסורו עוונותיו

בלב מודה / יתודה / ויטהר מחשבותיו

ולא יחמוד / ולא ילמוד / עצת נחש ומרמותיו

ולא ירכיל / ולא ישכיל / הדר נפש וגויה



אל איום / שלח פדיום / לבת גלים צופיה:
