


	
	דל דופק
	



חתימה: רבי שלם אלשבזי






דל דופק שואל / שמעה אזננו

מגולת אריאל / סגלת המונינו

מקוה חסד אל / צור מגננו

שמי ישראל / יוי אלהינו:





בחצר המלך / בעוד ליל דופק

ממרעים חלך / וחדל מתאפק

יה לאויבי תשלך / זממו אל תפק

ובכפיו יספוק / כי בא אלהינו:





יאתה חן לדלים / לקבץ נדוחיהם

ישועת יה מיחלים / כי הוא גואליהם

אז ישובון גולים / פדויים אל נויהם

בארבעת דגלים / גודל לאלהינו:





שמעי חבצלת / נחמה מדוד

ורוצי כאילת / גבול עזה ואשדוד

חסד מקבלת / כי לך יה יעודד

עם דל מתבודד / זמרה אלהינו:





לוית חן עטרת / כקדם תתבוסס

ותשובה גברת / בטוב ולא תתנוסס

ותקום באשמורת / חצות ליל רוסס

וחיל שעיר גוסס / ומי צור זולתי אלהינו:





מאוצרות חיים / יזלף על חסידיו

ויפרנס עניים / עדת יעקב ידידיו

ועל יושבי איים / הם רבים חסדיו

וכל מעשה ידיו / כי קדוש יוי אלהינו:





אליו הם מקוים / כל המוני יצורים

ושבחו מחוים / כי בטובו עזורים

נפסדים והוים / וברוח קשורים

ומראש אתה הוא / יוי אלהינו:





לחברת בני יעקב / מנהל ברחמים

ומאוצר טוב / מוריד גשמים

ודל לא יעזב / והוא אבי יתומים

ואחרית ימים / יוי אלהינו:





שחקים שפרה / מקום הצדיקים

נפשות הקרה / ברוחו הקים

וגלגלי הדרה / זה בזה דבקים

באור מוצקים / בעיר אלהינו:





בחושק תוכפים / להדרת חשקם

ותמיד שואפים / כי זה חקם

וכמעט רודפים / בזיו יסוד דבקם

והמציקם / שם אלהינו:





זכות יה ילמד / לעני ונקלה

בפתחו עומד / נרדף וגולה

רצונו חומד / והוא רש ונקלה

באהבה חולה / רפאנו יוי אלהינו:

יחזק לנזעם / רם על תהלות

כפעם בפעם / עורך תפלות

ולבו הועם / ושם בו מסלות

ללמוד טעם / שרידי סגלות

ויהי נועם / יוי אלהינו:
