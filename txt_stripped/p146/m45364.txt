


	
	אימתי בהגיוני תעירני
	



חתימה: אלשבזי






אימתי בהגיוני / תעירני בסוד יוקר

וחן לבי ורעיוני / מהרהרים בדברי שקר

כי רבו משטיני / בערב ובני עקר

ובהזכירי יגוני / לאחור גופי נזקר

נפשי לה׳ משמרים לבקר:





וסורי מחטא ואשמה / ועלי אל ארמוני

אז תזרחי כחמה / באור חיים קדמוני

למה יסיתך למה / נחש הצפעוני

שתי מצלול המימה / ואל יטעך הצפוני

ורצי עלי קוני / בכל ערב ובקר:





שורי אורח חיים, וכתבי נפשות עניים

אשר הם לאל נקיים / בצדקם ירוום ציים



[….] / […..]



מבאר שבע ואיים / ישובו לבצרון

ויהיה להם יתרון / באשמרת הבקר:





בחן אהבת חבצלת / תתנהג יחידתי

ותרוצה כאילת / לבית היכל מנוחתי

אבל רגלי צוללת / בגלותי באשמתי

ובת שעיר צוהלת / משחקת עלי דתי

סלח צורי לחטאתי / להגיד חסדך בבקר:





זה היום קוינו / מעשרת ימי תשובה

מקדם אוינו / לרפאת נפש עלובה

ביוי חסינו / להסיר לחץ משובה

בני עליון השגיבו / בני רחל האהובה

ואם ישענו יבוא / וחכינו לאור בקר:





יום בשורה לעם קדש / מראשית השנה

וקול שופר בחדש / יעירני בהזמנה

לשובב את הנפש / בתשובה נבחנה

ותנצל מעונש / והרהור אנה ואנה

עורי עדת מי מנה / יעיר בבקר בבקר:
