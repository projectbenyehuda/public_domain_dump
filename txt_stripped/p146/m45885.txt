


	
	לך עמי בא בחדריך
	



רצון הלוי – שירת ישראל בתימן

חתימה: לאלשבזי






לך עמי בא בחדריך / וחפש מתמול דרכיך

ירא את יוי בלבבך / אולי ירחם עליך

שובה וישוב אליך / וירפא את משובתך

למען תצדק בדברך / תזכה בשפטך:





אהוב צורך ועשה רצונו / והרחק מלבך זדונו

וסורה מהרהור עוונו / הגורם עליו יגונו

טוב עבד אוהב אדונו / ותורת חסד על לשונו

בלכתך תשמור צעדך / והקיצות היא תשיחך:





ליל תעמוד באשמורת / ברוב תחנונים ועתרת

ונפשך לטוב אל נוהרת / מצות וחוקים שומרת

ואורח חיים בוחרת / ומכל פגעים נזהרת

ויפוצו אויביך / וינוסו משנאיך:





שובה ותכון בלב תמים / מכל עוון ואשמים

ולא תירא פחד אמים / והרגשת אויבים וקמים

סור מרע ומאיש דמים / וקח נא מוסרי חכמים /

והיה שדי בצריך / יזכור כל מנחותיך:





בעזרת אל חי מלכנו / ישמח ויגל לבנו

עזרנו ומגננו / יאר פניו אתנו

בספר חיים יכתבנו / ומכל צרה ימלטנו

ויאזין ברצון שוענו / ימחה את כל פשעינו

אלהים ברב חסדך / ענני באמת ישעך:





זכות למדו עלי אבות / לפני צור שוכן ערבות

ישמח נפשות נעצבות / אשר ביום הדין נצבות

בספר חיים נכתבות / יעלו גלגל ומרכבות

נטה לבי לימינך / להיטיבך באחריתך:





יוצר כל בחכמה ודעת / ירחם איומה נכנעת

כי בשם קדשו נושעת / בגלותה מתיגעת

בעמק עכור טובעת / ומעיר אל עיר נוסעת ל

קול נחומים שומעת / בחן תורה משתעשעת

צורי רצה על עמך / יכבשו רחמיך את כעסך:
