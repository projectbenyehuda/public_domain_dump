


	
	אימה בהר המור \[...\] וצמח ארי גבור
	



חתימה: אלשב[ז]י




אימה בהר המור / עלי לך ובשרי



ונשיר בשיר מזמור / בנגון ליוצרי



וצמח ארי גבור / מהדר בשערי



וזמיר וקול התור / בארצי מעוררי



שריה חגור חרבך



והשחת גדוד אויבך



וחזקה ואל תנבך



להוגני אדום תגמור / ותשקיף בשחרי



כרמיו באף תזמור / ותשחית לעץ פרי:



לקית אלפתא לאצׄרגׄאם / בבסטה לחין ינוב



עליה תנצׄם אלאנגאם / ותתצאפף אלחרוב



והו יקתל אלדרגׄאם / ותהדם לה אלדרוב



ונאר אכלה תדׄער / גׄיושה ותעתרי



ותמחי בני אלעמלאק



באלאסכׄאט ואלאחראק



וינפדׄהם אלכׄלאק



ובית מלכהם ידבר / ופיה נחס גׄאירי



ויצהר מלך טאהר / אמיר אלעשאירי:



שכינה בצל כנפי / כרובים תסוככי



לינון וחיל תקפו / וצורך תברכי



וישפוך חרון אפו / בפרא הפכפכי



ועובר ומתעבר / ודתו מכוערי



מהתל עלי דתי



ובוזה לתורתי



והסיר עטרתי



וימיו כצל עובר / ואידו ממהרי



ועמי ביד שומר / משבח מפוארי:



בגׄודׄה עצׄים דאים / ועאלי במלכהו



והו יחרס אלנאים / בתוחיד אסמהו



ואגׄעל סראט קאים / למן כׄץ דׄכרהו



ונור אלפלך דאיר / בשאנה וסאירי



להו אנואר סטאעה



ותכׄדם להו טאעה



ודור לם יקף סאעה



ומן גׄוהרה נאשר / עלי אלבחר ואלברי



בלילן קמר סאמר / ושמס אול אלפגׄרי:



יחידה בשיר הודי / לאל חי ורנני



ועמדי ויחדי / ולבך תכוני



וחשקי וכל סודי / במלכי וכונני



ואליו אני נוהר / בערבי ושחרי



מחיה לנשמתי



ועזי וזמרתי



ואליו תשוקתי



וחסדו אני זוכר / וטובו מעוררי



ולבי כאש בוער / באהבה מזוררי:



תשוקת יא ודי / בנצׄמי ולחנתי



בדׄכר אלודוד סידי / אבלג בניתי



ופי גנת אלכׄלדי / יסכן לחיותי



כׄטא עבדהו יגפר / ויסמח לאוזארי



יסלם עלי נפסי



ביום תנצׄר אלכרסי



ופי צׄלהו תמסי



ובאלגׄוף תעוד תסכן / להו אלחמד ואלשכרי



וגׄסמי בהא יבשר / מעא אול אלסחרי:
