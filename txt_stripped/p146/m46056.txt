


	
	שטוף עברה
	



חתימה: שלם יוסף






שטוף עברה / למוד משרה / בתוך קהל מתענים

אהוב חכמה / ואם תצמא / שתה ממעין גנים

קרא מקרא / ודע סברה / קטוף ורד ושושנים

ובת גלים / לך תשלים / מקום יושר ומעדנים

ושיר תערוך / ותברך / לאל רם על אופנים

ברוך יוי אלהים / אלהי ישראל:





לעד כמה / בני מרמה / להמיר דת יסיתוני

ואיך אטמא / באיש ערמה / ויום הדין יקימוני

לאל נאמן / אני נזמן / ומלאכיו ילווני

ותורתו / וסוד דתו / דבר חכמה יסיחוני

מעט מזער / ואתנוער / ורעיוני יליצוני

בני אשמה / לכלמה / לרעתי תפתוני



וצור קוני / קראני / בני בכורי ישראל:





מנוחתי / בשמחתי / ביום שבת בלי ערך

והיך אסור / לדת אסור / וצורי עזרי בכל צורך

שכרי טוב / כעץ רטוב / וימי יוסיפון אורך

ספוק צרכי / ביד מלכי / שמו יגדל ויתברך

יחנני / וינחני / ואזכה לחזות חדרך



וכל פעם / בטוב טעם / יברך את ישראל:





יחיינו / ויראנו / ביום ישוב להיכלו

ואז נשמח / בבוא צמח / ונתברך בגללו

ובית מקדש / יתחדש / ונשמע שיר מהללו

וקרבנות / מוכנות / ונראה מגדלות שילה



בכל מועד / נתועד / אז ישיר ישראל:





ומה נעים / בבוא רועים / ונסיכים בתוכנו

וסנהדרין / ופלהדרין / סגלת מחמדינו

והמחכה / לטוב יזכה / ביום יבוא משיחנו

ואל יתמה / בהתמהמה / ילוד דוד אדוננו



נחכה לו / ונקבילו / כל זרע ישראל:





שריד אהבה / ברוב חבה / למוד תורה עם דרשה

עלה הגן / עזוב נרגן / אשר עלו מאד קשה

שמע הוגדס / בתוך פרדס / ינגן שיר בהדגשה

בתוך דוכן / ואור מוכן / נפשו שם מקדשה

והיכלות / סגלגלות / וחוט חרב מלטשה



ולשכות / מסככות / שם נפשות ישראל:





פדעני / והמשני / אלהי מאולתי

השב נפשי / אל שרשי / אצליח ביום לכתי

ושים חלקי / מקום חשקי / כי בשמך בטחתי

והובאתי / בחיק דתי / להבלי גוף מאסתי



מבראשית / חן תשית / לעיני כל ישראל:
