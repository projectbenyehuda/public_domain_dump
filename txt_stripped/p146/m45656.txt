


	
	מנחת כפור
	



חתימה: משתא אלשבזי






מנחת כפור תן מנוחה / צורי, לאילה שלוחה

מחטא ומחרב פתוחה / רוחי ונשמתי בטוחה

בחסד אל שדי משוחה / כי מכסאו היא לקוחה

להאיר גוף באפו נפוחה / הגלד מלחה סרוחה

חטא ויפול בשוחה / ושבה נפש נאנחה

יוי שמעה יוי סלחה:





שובי יחידתי שובי / מהרהור לבבי

אחרי עיני מרד בי / גרם חטאתי וחובי

כי לולי אלהי אבי / אזי אבדתי במכאובי

ובעוד נשמתי בי / איחד אלהי אבי

ימין אל חסד אל הרבי / ולי בספר חיים כתבי

עוררי זכות משה רבי / ים השים כמרקחה:





תגיל נפשי אם באתי / בכפור ושבת מנוחתי

עולת תמיד תפלתי / ושיחתי היא מנחתי

יוי נגדך כל תאותי / שים רוב שלום לעדתי

ובצל כנפי רעיתי / תלין נפשי בהרוחה:





אהבת נפשי ורעיוני / ללמוד תורת הגיוני

מיום נתנה מסיני / הוסיף אור שכלי ועיני

עלה משה ראש גאוני / והגביל עמי וזקני



[….ני] / ונערו צוה בהצלחה:





אל חי בחרו וזכהו / ואמר למשה: צוהו

קום חזקהו ואמצהו / ומהודך הלבישהו

והתבונן כי אני אני הוא / בראש קהל עמי אעלהו

אחלצהו ואכבדהו / בצפון ודרום ים ומזרחה:





לקח יהושע משחו / האציל עליו מרוחו

וכל עם קדושיו הוכיחו / כי הוא זך ויפה כוחו

לא כהתה עינו בנוחו / ובמותו לא נס לחו

הוא נאמן אל ושלוחו / במראה הסנה הבטיחו



[…חו] / לדבר בנבואה נכוחה:





שמע יה עם בא להתודות / אסירים תחישה לפדות

בעבור שלש עשרה מדות / ומנעימים שיר לך להודות

ומצוות סגלתנו ועדות / לידע בכל משל וסודות

נשירה שיר על ידידות / בשמחה וצהלה והצלחה:





במנחת כפור הבינו / זרע ישורון והאזינו

שיתו לבכם והאמינו / ולאלהים חי הרנינו

זה היום אשר קוינו / ולסליחה התאוינו

בשקט ובשאנן לינו / וחטאת עמי נשכחה:





זכרון טוב לבני נדיבים / היום אתם נצבים

בשבתות וימים טובים / מקרא קדש נאהבים

שובו בנים שובבים / בספר חיים נכתבים

בקרב עמים רבים / אחד מהם למשפחה:





יקומה בעזרתנו / אלהי קדם וירצנו

מהרה נעלה לארצנו / בחסד ורחמים ינהלנו

יאר פניו אלינו / במעגלי צדק ינחנו

מה נעים גורלנו / שמש צדקה זרחה:
