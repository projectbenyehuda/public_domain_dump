


	
	אם לבי
	



חתימה: אלשבזי






אם לבי / בערבי / ממים זדונים

ואצמא / למימה / ואין בי כח אונים

ועין שכלי / יום חדלי / תזכירני יגונים



רחמנו / צורנו / כרחם אב על בנים:





להקביל דוד / אתנודד / ואין בפעמי כוח

כי פשעי / שם צלעי / כשה מוכן לטבוח

בצלמות / גוף עות / עדי ידכה ישוח



וכמעט אמ/צא רוח / ומשל גוי עז פנים:





שכינת אל / רצי שואל / וזכות למדי עליו

ויתבונן / להתכונן / באות ארבעת דגליו

בניב רחשו / וסוד נפשו / יוי נצב עליו



ובטובת / מהלליו / ידבר שיר ורננים:





באהבת בת / נצבת / למעלה ממרכבה

רצה שועי / צור ישעי / לעמך כפר הבה

בזכות חן / בני בטחון / דרוש סגלה נאהבה



רפא נפש / נעלבה / בין אויבים חמנים:





זה אלי / שמע קולי / לעמך המציא כפר

רפא חולה / שהוא גולה / ברית אבות אל תופר

ושים שלום / לדל חולם / כמי גיחון ומי פרפר



לעזרתי / צלח עופר / ורחם עם נאמנים:





יצו חסדו / לידידו / רם ונשא עלי הכל

ויופיע / להשפיע / לבני דעת והשכל

וטוב מזון / לדל נזון / מחסדו ומתכלכל



והוא נשען / עלי בת כל / אשר היא אם הבנים:
