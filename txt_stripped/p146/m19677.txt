


	
	שמועת דודי זה טהור העינים
	



חתימה: שלם שבזי (29: ילוד יוסף)


שמועת דודי זה / טהור העינים



תעורר קול תודה / יבשר בנתים



ומי חסד יזה / לחזק ברכים



ויציל עם קדשו / בשמחה ובשירים:



לציון ישפיע / חסדיו הרבים



בשמחה תרגיע / לבנים שובבים



ברצון יופיע / ירחם השבים



עבור כהן מקריב / ובעלי משמרות:



מחכה לקראתי / לדודה תואבת



תעורר אהבתי / לפניו נצבת



עלי חקי דתי / לפניו סובבת



לרצות את דודה / בקיום המצות:



שבויה נכספת / להקביל את דודה



בכל יום שואפת / לתפארת סודה



וטובו תוכפת / ופושטת ידה



בעיניו תמצא חן / בקשור המצות:



בחי נפשך תמה / תרצי על גולים



פזורים תוך תימא / באהבתך חולים



מצפים הקדמה / בישוב העולים



לארץ הקדש / שהיא נחלת אבות:



זמן ישע יחיש / אלהי עולמים



ואויבנו יוריש / וישבר הקמים



והר שעיר יעניש / ויציל איש תמים



ירחם על בניו / ברצון וישועות:



יחידתי תשמח / בביאת משיח



ותתעטר ברמח / בגנה תנוח



כהצמיח צמח / לדודה תפציח



תדבר שיר חדש / תזמר ברננות:



ילוד יוסף השלים / לשיר זה שחבר



וסוד רעיה העלים / וברמיזה דבר



אלהי אל תכלים / לעבד לך סבר



ושלום רב לכל / וטובות נכפלות:
