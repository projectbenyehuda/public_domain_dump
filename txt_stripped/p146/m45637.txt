


	
	אויל מבאיש
	



חתימה: אלשבזי






אויל מבאיש / היה נא איש / נחפשה את דרכינו

ונתכונן / להתבונן / לעשות רצון אלהינו

בזה היום / יהא פדיום / לרפאת מחלתנו

והשבים / נצבים / יהא שלום בעירנו

דעו וראו / והמצאו / בני יעקב שבטינו

ונתודה / לצור פודה / וימחה את פשעינו

ישוב ירחמנו / יכבוש עוונותינו:





לבב נדכה / אשר הכה / ביסורין ונתמרק

ויטהר / ויזהר / במי חסד אשר זורק

ויתיסד / בלי הפסד / ועול יצרו יתפרק

בשוב נפשו / אלי שרשו / ואז יזכה בטוב חלק

וזה משפט עדתנו / ביום צור יקימנו:





שבו בנים / מזמנים / דעו יום דין והחשבון

והמודה / הוא יפדה / את נפשו בערבון

ביום יעזוב / שמאל כוזב / וירצה בו הרבון

שהוא חופש דרכינו / וגולה מצפנותינו:





ביום עשור / תתבשר / אימתי במחילה

ותכינה / הבינה / לדלתי שערי גילה

ואם כל חי / לכל אחי / תהוסיף חן ותשכילה

וזיו תנהיר / באור בהיר / לשבטי יה ותגבילה

ותגיה לחשכנו / ותוסיף את הדרנו:





זריז שכל / אשר יוכל / לטהר מחשבות לבו

בצד ימין / יהא מזמין / שמאל ימאס ויגער בו

וימצא / כל חפצו / בעת יקביל פני רבו

ויתישר / ביום הכשר / למול מלך במסבו

ושם משה נביאנו / וגם שבעים זקנינו:





ישורון רון / ולך סברון / להתודות בתשובה

בצל שדי / אזון עדי / וחפץ נפשך יובא

והתפלל / ביום וליל / ותתרפא ממשובה

וחן תראה / ולא תגאה / ואז נפשך תהא אהובה

וחסד אל ילונו / ולחיים יכתבנו:
