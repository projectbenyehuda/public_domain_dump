


	
	ברכו לאל חי נורא עלילה
	



החתימה: אלשבזי שץ

הכתובת: לענין סעודה






ברכו לאל חי / נורא עלילה

והודו לשמו / עלי כל אכילה.





אלהים בחסדך / רצה פרנסנו

ומהוד כבודך / בחן הגבלנו

ורחם ידידך / ואל תטשנו

להבין בסודך / נפשי תחלה:





לבבות ורוחות / כל היצורים

באל חי בוטחות / ובשמו שמורים

בשובע שמחות / תשובב פזורים

ותצליח ארחות / עדתך בגילה:





שכינת יקרך / בציון תסובב

וילדי בחירך / לארצך תשובב

אשר להדרך / מאוה וחובב

שפע זיו מאורך / והאר אפלה:





בשלחן מהדר / תשרה ברכה

בלחם מסדר / לנפש צריכה

פדה רם ונאדר / ברעיה נבוכה

ופרא וקדר / תהא לו נפילה:





זמן פרוסה / אל חי לעבדך

בתימן מנסה / לבבי בפחדך

ונפשי כמוסה / בטוחה בחסדך

ובעפר תבוסה / מדכה טפלה:





ישועה תחיש / לזרע ישרים

ואויבים תעניש / משנאים וזרים

ונצחם תחליש / מלכים ושרים

ועשו תרעיש / וגם תכשילה:





שלמא וקשטא / דמלי מדוריה

לבד תב בחרטא / יסגי נהוריה

בסתוא וקיטא / סביל לניריה

ונפשיה נקוטא / כנפש קטילא:





[שלום ואמת / הממלאים את מעונו

לבד מן השב בחרטה / ירבה זהרו

בחורף ובקיץ / נושא עולו

ונפשו מאוסה / כנפש הרוגה]





צדקתא הב לן / מרן במגן

בטב אשגח עלן / ואעדי לנרגן

ואצלח דגלן / באמצע דדרגן

ואסגי חילן / דנהיר לעילא:





[צדקה תן לנו, / אדוננו, חינם

בטובה השגח עלינו / וסלק את הנרגן

והצלח את דגלנו / באמצע הארמון

והרבה את חילנו / המאיר מעלה:]





לעמך שלמא / רבא תפרשז

ועבדך כקדמא / יחדי ויעלוז

ופתיר לחלמא / אבונא תגנוז

בגנת מרומא / דאיהי כלילא:





[לעמך שלום / רב תפרוש

ועבדך כקדם / ישמח ויעלוז

ואת פותר החלום / אבינו תגנוז

בגן המרום / שהוא מושלם:]





שלמא יקיף / מאלהנא

בפחדא ישקיף / על סנאנא

וקועהון ינקיף / מקמנא

כנשתן יתקיף / דעליה סבילא:





[שלום מקיף / מאלוהינו

באימה ישקיף / על שונאינו

וגרוניהם ישבר / של הקמים עלינו

אומתנו יחזק / אשר את עולו היא נושאת:]
