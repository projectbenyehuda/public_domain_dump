


	
	בלילי כפור למדו
	



חתימה: אלשבזי






בלילי כפור למדו / בעלי נפש נמוכה

בפתחי שערים שקדו / בנפש טהורה חוכה

והכונו והתודו / וחוט שלחסד משוכה

ועל קו צדק עמדו / וצאו מן ההפכה

ומזבד טוב זבדו / ונחמו לאם הבוכה

כי שם צוה יוי את הברכה:





את הברכה צוה / אל חי לעשות רצונו

יהא לו אחרית ותקוה / וימחול את כל עוונו

ופרא מבאיש בגאוה / יכלה יה את עוונו

וכמו אשה דוה / יצרו צעדי אונו

ביום דין יחרד המונו / ולא יצלח למלוכה:





למלוכה לא יצלח / ביום בוא ישועתנו

יהיה נתעב ונאלח / ותעמוד מלכותנו

ולא יחזיק בשלח / ולא ימשל עלינו

ואז תעמוד ברית מלח / לכהני בבית קדשנו

ולא יבוא בתוכנו / ממזר וכרות שפכה:





שפכה בוז על עשו /צורי, ואש תוקד בו

תכלה כל אוכלוסיו / ולא יועיל לו חרבו

בעמקיו ורכסיו / אש יעקב תחריבו

ועת יפול מנכסיו / ימות לבו בקרבו

ויכה בימי עצבו / דל ולא ישא ברכה:





ברכה ליעקב ובניו / ובכורה נתנת

ומלאך הולך לפניו / ושכינה מגנת

בחוריו וזקניו / וטף ונשים אומנת

ולענייו ואביוניו / הון ועושר נותנת

ויום שבת חוננת / שאין בו התר מלאכה:





זכרון טוב יהא לנו / בהיכלי שערי בינה

ישמע צעקתנו / אל נורא ויאזינה

וימחל חטאתנו / כל קהל עדת מי מנה

ובחסדו יפרנסנו / ונרויח בכל שנה

וכל בעלי האמנה / ישמחון יחד ככה:





יחד ככה יפרנס / לעניים ועשירים

גואלנו מאנס / ומרשעים אכזרים

והעבירנו בנס / בתוך מים אדירים

ופזורינו יכנס / וקרן משיחו ירים

ויוסיף מימי נהרים / מעליוני ברכה:
