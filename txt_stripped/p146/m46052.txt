


	
	שוכן שמים
	



חתימה: שלם שץ






שוכן שמים / שור ממרומיך

וקבל נא ברצון / תפלת קוראיך

והסר כעסך / ויגולו רחמיך

למען יחלצון / ביום זה ידידיך

ותהיה יה למשען / ומחסה לדורשיך

הושיעה את עמך / וברך את נחלתך:





לקראם האזינה / לבש חן וצדקה

והשפיע שכינה / לנפש שוקקה

וכפר המצא נא / והסר למצוקה

ומרוע תכונה / לעמך העתיקה

ודבק את לבבם / יה ביראתך:





מהר נא לגאול / נפש נעשקת

מתגרת סמאל / בנגעה צועקת

ולצד מיכאל / תהא משתוקקת

ומאש גבריאל / אל תהא נדלקת

ומעזא ועזאל / תהא מתרחקת

ותתנהג עמה / במדת רחמיך:





שלח צרי ותעלה / לעם הנבחנים

ובשרם במחילה / מחטא ועוונים

ובינה המשילה / שהיא אם הבנים

בהשפע תגדילה / ותגביר תחתונים

ותפארת בחמלה / ונצח והוד נותנים

ליסוד משפיעים / אלי מלכותך:





צדק ושבתאי / ומאדים במרומא

וחמה בשחק / ונוגה התמה

וכוכב באופק / ולבנה שלמא

ומזלות תדחק / ותשיר בנעימה

תספר שבחיך / וכח מעשיך:
