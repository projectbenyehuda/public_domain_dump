


	
	שבח אל חי אשר ברא
	



חתימה: שלם בן יוסף




שבח אל חי / אשר ברא לכל חי / ומלכותו עדי נצח
נצחים:



לשוני דברה / בשבח וזמרה / והוא נורא עלי כל
השבחים:



מנת נפשי / ושכלי האנושי / וכליותי עלי חסדו
בטוחים:



בסוד לבי / ושרעפי בקרבי / אהודנו ורעיוני
שמחים:



נהוג עבדך / אדון עולם בחסדך / ושמח לבבות
נאנחים:



ישועה תשלחה / לבני מנוחה / ותשכילם בכל סודות
נכוחים:



והחישה / לבת גלים דרושה / פדה לישה נדודה
בפתחים:



סלח פשעה / והחישה לישעה / והדריכה להלך
ברוחים:



פדה וזכור / זכות מלך מכרכר / למול ארון וגם זבח
זבחים:



שלומך צח / יחינו לנצח / כצלי עב מדבקים
ונחים:
