


	
	אקו לאל
	



חתימה: אבו יהודה שבזי






אקו לאל / הגואל / רם ונשא בהיכלו

ירחם רש / אשר נגרש / מביתו ואוהלו

ויפרנס / בדברי נס / לדל ואביון אשר אין לו

יאמץ עם / אשר נזעם / ויאירה לעין שכלו

ויצליח / וירויח / בעולמו ביום חדלו

הוא עזי / ומעזי / דודי לי ואני לו



הללוהו / בגבורותיו / הללוהו כרוב גדלו:





בתוך תימן / אני נטמן / ובן אמה הצר לי

ולי חמס / והרבה מס / ודכני ביום חשלי

קרב חפץ / ומתנפץ / יריבני בבית אבלי

מבקש ריב / ולהחריב / כנסיות להרע לי



ואם שקר / הלוא בוקר / וידע יוי את אשר לו:





ודוי פי / בליל נשפי / ירצה יה ויענני

ויופיע / להושיע / בעת רצון להמוני

ויציל בת / נעלבת / בתוך צלה ינחני

וירצה על / אסיר נגעל / ומשפל ירימני

יצו ינון / לדל אונן / יפה עין ואדמוני



אל נורא / דרור יקרא / העם בחר לנחלה לו:





ידיד הכון / בעת תשכון / והתחנן בתפלה

עלות שחר / אל תאחר / עדי הקץ יהא נגלה

וחיש מהר / והטהר / ולך אל יתנה חמלה

מאס קדר / אשר נהדר / ולך לך מבני עולה

ושוב וחדל / וחון על דל / ותמצא חן ורוב צהלה



ואם צר לך / נגה שכלך / דום ליוי והתחולל לו:





היו נכונים / מזמנים / והשכילו להתבונן

באות נפלא / מלמעלה / יבשרה בבוא ינון

וישכלל / לבית מכלול / מקום קדש אשר כונן

ואז יבוא / למושבו / לאביונים יהא חונן

ושיר חדש / בתוך מקדש / לויים בו תתרונן



לאל הודו / והתמידו / שירו לו זמרו לו:

ואם חברו / ביום גברו / משנאים בהמוניהם

כמעט קט / והן נשקט / הדר גבהם ושריהם

ואש תבער / חזיר יער / וההגרים וסגניהם

ואש יצחק / ממרחק / תכלה ארמנותיהם

הפוך חרבם / לתוך לבם / אדון הכל וכליהם

והציל דל / אשר נבדל / ויעבר את אשר לו:





דרור צפור / חשי לספור / מתי מספר שבטינו

וכופר תני / והתבונני / וחלי פני אלהינו

מצוא כופר / צבי עופר / והגן על ידידינו

ותן מרפא / לגולים פה / עשה בזכות אבותינו

פדה מסכן / מסתכן / יהא שלום בעירנו



אדון עולם / הנעלם / ותועפות הרים לו:





הדר קמים / מזוהמים / יהא הולך באש עברה

יזורבו / ויחרבו / ילוד הגר ושעירה

לפי חרב / תתקרב / בני מישא וגם בצרה

ותנצל / כלילת צל / חצר קדשי מקום זמרה

עלי נפשי / להר קדשי / ולך צורי דרור יקרא



ואיש נאהב / בחן נרהב / ושב ורפא לו:





שאון גלי / זמן מלא / לבבינו דבר שקר

ואויבינו / סביבינו / באפם זוללו יוקר

והקיפו / והוסיפו / דבריהם בלי חקר

ומלך זר / נתאכזר / לקדשנו באף הפקר

וזה נחלש / וזה נבלש / וזה הכה וזה נזקר



והנמלט / לעיר מקלט / לא ידענו מה היה לו:





בך ארוץ / גדוד חלוץ / צור עזי וזמרתי

פדה נרדף / כקש נדף / נפוץ חפצי רעתי

ושים תקוה / לבן אוה / להבין סוד בתורתי

והך מכחיש / ברע מרחיש / ומתלעג עלי דתי

וזלזלים / באף תכלים / ורחם על אימתי

שלח צמח / בך ישמח / תאות לבו נתתה לו:





זן ומזין / רצה והאזין / עניי עם והשכינם

הגן יתומים / שהן הומים / ומלא יה לחסרונם

ובן הגר / באף סגר / עלי כספם וממונם

ובא רבץ / עדי קבץ / לתירושם וכל הונם

ומתנאה / ומתגאה / ברעי יעבוד חנם



ובפרך / ישעבד רך / ופעלו לא יתן לו:





יבוא יום / הפדיום / ואקביל את פני דודי

ואמשול על / חזיר מעל / ואתנער ביום חלדי

והפרא / יתערה / ויתבהל מנגדי

רצה אל חי / דבר שיחי / בידך חזקה ידי



שפוט אויבי / מכאיבי / אמר נא: קום, עמוד עבדי!



בעיר ציון / ואפריון / שבה וצלח ברוב חסדי!



כמו נמת / ואמרת: / ובריתי נאמנה לו:
