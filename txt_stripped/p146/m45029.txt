


	
	אקדם בשיר
	



חתימה: אני שלם






אקדם בשיר רחשי / תוך עם סגלתו

ביום שבת קדשי / אסדיר תהלתו

ברכי נא נפשי / בראשית בריאתו

וביום הששי / השלים כל מלאכתו

לקדש יום שביעי / ולזכור ברית עולם

ברוך שאמר והיה העולם:





נקדש בצבאות / כי הוא רב כוח

ודורות הבאות / המה גבהי רוח

עד בוא איש צדיק / אשר נקרא נח

ונכנס לתבה / ומצא מנוח

עד עבור המבול / לטהר העולם:





יצא הוא ובניו / ועופות ובהמות

עם מיני נחשים / ודבים ואריות

וברכם האלהים / לפרות גם לרבות

עד בא ידיד / לסוף עשרה דורות

הוא כענק וכרביד / נסה לסוף העשרה

באומן לאל יחיד / אשר עמד בכלם:





שמח ודץ ושקד / בבן נולד לשרה

במוריה נעקד / וזה סוף העשרה

והוא לאל נחמד / ומדתו גבורה

והוליד זוג תאומים / תם ובעל עברה

עשו איש דמים / אשר מכר בכורה

ליעקב אוהלים / וחזיונות בסלם:





לנסותו ביוסף / והורד למצרים

ילד שם גבירים / מנשה ואפרים

ויעקב עוד ירד / ונרפא בעינים



[…..] / […..]



כי ראה את יוסף / צדיק יסוד עולם.





משכם בעבותות / אל משפיל ומרים

לקים נגזרות / אשר בין הבתרים

להשלים העשרות / ויהיו נטהרים

כתמים וסיגות / עד ליל שמרים

ויצאו בכושרות / ומשה הוא גואלם:





הביאם להר סיני / וקבלו התורה

בלוחות אחרונים / דברות עשרה



ותרי״ג נצפנים / בתוכם נקשרה



ובם האיר עינם / והגיה אפלם:
