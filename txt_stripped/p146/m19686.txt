


	
	לבי מצפה רעיה ודוד צח ויפה
	



החתימה: לאלשבזי




לבי מצפה / רעיה ודוד צח ויפה / תשפיע לחמה
וסהרי



חון דוד והרפה / על דל בגלות מעלפה / אתה כבודי
ויוקרי



אויב מחפה / עלי דברים ונכפה / מחריץ כנאנא עלי
ארי



עצמו ישפה / נרו תכבה ויטפה / ואני תהוסיף
בזהרי



גולה צפוני



החיש גאוני



מלך המוני



מהר עניני / שמך מנתי ושומרי



נביא וצופה / חליי כאבי ירפא / אנעים לקולי
בזמרי:



אצפי פואדי / תוחיד רב אלעבאדי / מא דמת פי דאר חיותי



גאית מראדי / אסראר עלמי רשאדי / אללה יבלג לניתי



סר אעתקאדי / באלראי ואלאעתמאדי / סבחאן כאפל
מעישתי



לאלכׄלק האדי / דׄו אלכון ושי דׄו פסאדי / נאטק
ואבכם וצאמתי



פצׄל באסמה



אסבאט עלמה



קדהם בקסמה



מן תחת חכמה / קומן מקדס וטאהרי



אסראר תכׄפא / אלקן למוסי בלטפא / פי כתב לוחין
גׄוהרי:



לבש יקר הוד / מאש סנה חש לועד / שבטי המוני
באהבה



רעיה וגם דוד / נתחברו שם לסעד / ארבע דגלי
לסובבה



ומשה בכבוד / מנעים בשירה וצועד / עמו מטטרון
לשר צבא



נצב כמו נאד / הים כירא ופוחד / אש יבשתו
והחרבה



נעשה דרכים



טורים כרוכים



לבני מלכים



כנאד סבוכים / לפני שבטי בשחרי



עמד וקפה / טבע לפרעה ונספה / חילו בתוכו
מנוערי:



שאפון אלאעדא / פרעון ותלך אלגׄנודא / אלמוג אוצל
נפדׄ בהם



נאלון כידא / מן שור עמלאק, בזהדא / שדו ריאנאת
כׄילהם



מן גיר גׄהדא / מא נאלהם ראי רשדא / קד אחבט אללה
חאלהם



בלעם אבדא / אסחארהו תׄם תעדא / וקתל במדין
בינהם



ואליאונוסי



ואליומברוסי



אבנא אלמגׄוסי



נאלו אלנחוסי / הם מדׄהב אלזור סאמרי



באלסיף אנפא / ליואן דׄאך אלשריפא / פי יום צׄרב
אלבואתרי:



בזכות אבותי / יתכוננו ארחותי / נעלה לציון
ונשמחה



ובמשכנותי / אראה סגלת נאותי / יבוא משיחי
ויזרחה



ראש מחנותי / וגאון הדר מקהלותי / יה יעזרהו
ויצלחה



בין רבבותי / אראה לנס נפלאותי / אחסה בצורי
ואבטחה



ישוב צבי חן



עלי, ויחון



אראה בפתחון



מים דגיחון / אשקה לצאני בנהרי



לבי אצפה / ואש כבדי כאופה / תחליש שמאל
המאחרי:



זין אליהי / מן כאן זכי עקל באהי / לה עז ונאמוס
דאימא



ואהל אלמלאהי / אפכארהם לא תנאהי / מן כאן מכׄטא
ואתׄמא



דׄאת אכתדאהי / ואלעקל שארד וסאהי / לא יסתמע קול
עאלמא



יבקא מצׄאהי / לאלגׄהל בין אלספאהי / ולא יכׄאף אמר
חאכמא



סאלב לעקלה



תאבע לגׄהלה



ראבי בוהלה



מא בין אהלה / תשתם בהו אלעשאירי



לא עאד יכׄפא / פעלה רדין כתׄיפא / גׄהאר אעיב
מנאכרי:



יכׄתץ לחני / מנטק שפאתי ולסאני / אללה אחד גל
אסמהו



הו דׄי כׄלקני / ואפני אלמהמאת עני / ואגׄב עלי
דׄכר שכרהו



יעלם בצׄני / ואלא דעותה אגׄבני / ולא יכיׄב
לעבדהו



רוחי תמני / תזכו תנאל מא ועדני / ואזור אלי בית
קדסהו



אגנם זמאני



פי קלב האני



חיתׄ אלמעאני



ואנאל אמאני / אסמע לזין אלבשאירי



כאסי אלמצפא / מצהר לאסראר תכׄפא / אפתי למן כאן
חאירי:
