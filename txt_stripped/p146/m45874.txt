


	
	ארוך עין בכוס יין א
	



חתימה: אלשבזי





נוסח א


ארוך עין / בכוס יין / אל ישגה לבך

ולא תלמוד / ידיד לחמוד / את אשת רעך

וסור מרע / וטוב תזרע / צדקה לענייך

ולא תכלים / לדלים / ולא תקפוץ ידיך

ולא תגאה / ותתגאה / ולא תבטח בעשרך

בלבך גמור / דת לשמור / ואז תצדק בדברך

זכור קונך / ביום הונך / ואמור ביום עמדך



טוב אתה / ומטיב / למדני חקיך:





ללב נמהר / יהא מזהר / לעשות את רצון קונו

ויתיסד / להבין סוד / תורת אל בהגיונו

ודת מוסר / לא יחסר / ולא ימות בעוונו

אבל ישוב / יהא חשוב / ואז יחזר למכונו



שמע מלין / ואל תלין / לבבי בחטאיך:





שחי רוחי / במר שיחי / ליום שובך תני צידה

והכוני / ותשכני / בתוך קהל וגם עדה

תני כפר/ לגוף הפר / ברית אחים וגם זדה

והביני / ביום דיני / ועלי שלמי תודה

בבוא צורף / כדוב טורף / ביום הדין ויעמידה



מנת כוסי / וצור מחסי / מחל נא לעבדיך:





בגלותי / ודלותי / אני הומה במר בכיי

ולבי סר / ונתיסר / אלהי רפאה חליי

וחון גוף דל / אשר חדל / וחזק יוצרי רפיי

רצה עלי / במשאלי / ראה שפלי ורב עניי



בך בטחי / סבול טרחי / ביום דפקי דלתיך:





זמן אתה / בהשחתה / והכביד על ורב דלות

ובן אמה / מאד המה / ומתגאה כרום זלות

ועיניו צר / בעם נעצר / שהוא צולל בבור גלות

ורם לבו / בתוך קרבו / בעמך ינהגה שפלות



אלהי חון / לדל בחון / בחשק אהבתך:





ישורון לך / והמלך / בשכלך לעשות טובה

והתישר / ברב הכשר / בלי עוון ולא חובה

גבול תשים / בטוב מעשים / ואת דעתך תישבה

הדר זקן / בחן תקן / ותקום מפני שיבה

עשה לך רב / ביום תקרב / ולא תשכב למעצבה



ביה השלך / יהבך / והוא יהיה מכלכלך:
