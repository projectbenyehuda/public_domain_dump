


	
	יום אזכרה מלכי כסאו בהר קדשי
	



החתימה: אלשבזי (בראשי החרוזות); שלם סאלם שלם (בראשי טורי האזור הראשונים
בחרוזות הראשונה, השנייה והשישית)




יום אזכרה מלכי / כסאו בהר קדשי



ימתק בשיר חכי / מצוף וטוב דבשי



רעיה לדוד חכי / ובאויבי בטשי



רחצי והזכי / חן אהבה דרשי



שורי פני



שבטי בני



איש תם, בנו



עם שוכני / תימן וגם אוזל



להם רצי



כפר מצי



דל אמצי



גם קבצי / המון אשר נגזל



משחי גביר



כהן אביר



יהיה סביר



יבא דביר / אז אויבי יוזל



פרא אשר ארב / יפול לפי חרב:



אהבת יפת קומה / חשקה באות נקמר



יפיה אשר נדמה / הדר כמו תמר



מרב יפי עלמה / שמש בזיו נגמר



מלחיה חכמה / לנוגנים נשמר



סוד תארה



מיוצרה



וביושרה



שם זהרה / ביום ובלילה



אש דת רפא



דל יכספה



מור משפ־



תיו יטפה / מאהבת כלה



לובש כליל



מאור עליל



ובזוג גליל



ישמע חליל / נוגן בקול צהלה



מנעמו



טוב טעמו



קול פעמו



שיר נאמו / ירפא ללב נחלה



בקר וגם ערב / יאתה ויתקרב:



לביא והלביא / מתחברים היום



ואני במושבי / צופה פדות, איום



צורי שלח תשבי / ובריתך קים



רפא לנעצבי / מהר שלח פדיום



אלי תנה



אתכוננה



איך אשכנה



עם בן ענה / תחיש רצונך



לנו קבוץ



בנו חפוץ



איבים נפוץ



העיר לקץ / ושלח משיחך



מים דלה



רש העלה



אל היכלו



יה קבלו / בזכות שבטיך



פרא אשר זורב / יצמת וישתרב:



שובי יחידתי / התאזרי חיל



זכי לעם דתי / לעמוד בעוד ליל



אעיר באהבתי / חכמת אביגיל



אתא בעונתי / תור בת אביחיל



צורי צלח



חון נאלח



ינון שלח



איבי דלח / רפא לנכאבים



לידידך



צו חסדך



ברצונך



ובישעך / תשיב לשובבים



יתרפאו



עם נדכאו



חן ימצאו



כי צמאו / יחדו למימיך



ספסוף וערב רב / זיו תארם יצרב:



בחרי לדל רעיה /חלי לדוד יפה



כמה בתוך שביה / חסדך אני צופה



איכה צבי צביה / פרא בעל כופה



עירם וגם עריה / קלון עלי כל פה



יום גלתה



אז נשחתה



כי טעתה



לא עלתה / לחזות מקום שלוה



שדי קרא



בת נבחרה



חטא כפרה



חוש ישרה / דרך ושים תקוה



רחם עני



עם נצפני



משוכני



מתאונני / שכח לכל חדוה



שר שיר ולא ערב / כי אהלו חרב:



זה עת מבוא דודים / יבא אהוב לבי



ואני לשיר אקדים / יום אשמעה תשבי



שבטי ברון מודים /רוח אלהי בי



חוזים ומגידים / בת רעיה הרבי



שמרי ברית



עד אחרית



בנים זרית



איכן שרית / להם תבקשי



לבשי לבוש



זהב חבוש



קבצי נתוש



מפוט וכוש / לצוללים המשי



מלכי גשה



דת חדשה



בת בוששה



קום קדשה / נעלה להר קדשי



ומאהבי קרב / כשתי וכערב:



יא טאלב אלחכמה / מן גיר אסתאדי



אלצבר לך חשמה / לא תנעכס עאדי



קף קיד אלנעמה / חתא יג̇יך באדי



יפתיך ען כלמה / מא שית תתקאדי



באב אלעלום



תקוא אלג̇סום



תובה תרום



קבל אלעזום / ואחד̇ר מן אלבטרה



תב תסתריח



אנך גׄריח



ג̇סמך טריח



ואלנפס ריח / האוי מעא אלסכרה



רד אלכלאם



כ̇וף אלמלאם



קבל אלצ̇לאם



כ̇ץ·באלסלאם / מן כאן פי אלחצׄרה



כ̇ץ שרק ואלמגרב / ואלקדס וחד יתרב:







את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.  הכל זמין תמיד בכתובת הבאה:https://benyehuda.org/read/19675
