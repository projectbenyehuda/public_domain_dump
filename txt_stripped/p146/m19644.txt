


	
	אימה לבשי עזך בת נדיב הצנועה
	



החתימה: אלשבזי משתא




אימה לבשי עזך / בת נדיב הצנועה



ועלי אל הר המור / ושמעי שופר תרועה



ואמרי לדודך יזכור / זכות אבות ושבועה



ושנים עשר שבטים / וכל מעתיקי שמועה



לעם עני יושיע / בעת רצון הידועה



עזי וזמרת יה / ויהי לי לישועה:



לבשי עזך ציון / בחצות ליל עורי



ועוררי בעלי חזיון / ואלי באהבה סורי



ובשכל הרעיון / אשכיל ביופי צורי



אשיר ואשמח על יין / ואנשה עול תמרורי



יהא לעדתי רתיון / יקנא יה בעבורי



ויזכה את נפשי / לחזות גנה נטועה:



שאלו חברת קדושים / את שלום ירושלים



מראש השלישים / אשר בין אש ומים



וראו מעלות שלושים / ומסכתות כפלים



אז נחלץ חושים / ושכינה בנתים



אליכם אקרא אישים / בעלי שש כנפים



אשר בפתח עינים / לכם לשכה ידועה:



בואו ידידי היום / נחלה פני צורנו



הגוזר ומקים / וגולה מצפוננו



אולי נמצאה פדיום / להחיות נפשותינו



כי ברצונו יחיים / ויכפר בעדנו



אל חי נורא ואיום / יאזין תפלתנו



וישליך את אויבינו / אל ארץ לא זרועה:



זריזי הלב עמדו / בפתח ים כנרת



ולאלהיכם עבדו / כמו שומרי משמרת



ודלתי תשובה שקדו / בנפש מתישרת



ותורת משה למדו / ולכם חן מזכרת



ואת נויכם פקדו / בדת מלכות ותפארת



ויום כפור מטהרת / חליי נפש נגועה:



יום עון עגל הוסר / משמרת קבעתי



סודו למשה נמסר / ובזכותו טהרתי



ולו במחילה בשר / ואמר לו ‘סלחתי’



ומצות שית מאה וחד סר / שתים זו שמעתי



ומעבר ים גינוסר / ודרומה ירשתי



וביום זה חמדתי / מן הבינה שפועה:



מחשקת דוד יפה / לא טעמתי שינה



וענן יקרו חופה / למעלה מציונה



ושמאל מבאיש רפה / ונפשי בתקוה חונה



ולבי רצון אל צופה / אל דעת עליונה



ובתורה שבעל פה / מנתי בראשונה



ואם במלאכה פונה / לה משמרת קבועה:



שרשי וצמח גזעי / בחסדו האל יגביר



ייי אורי וישעי / ופניו לעבדו יסביר



הוא מצודתי וסלעי / חרון אף מני יעביר



וידלה מים טבעי / ורוחי בשכלי יחביר



ואזכה ויירש זרעי / חלקו בציון ודביר



וגאון אויבים ישביר / ויחיש לעמו תשועה:



תכתבון עם ייי / יום זה בספר מחילה



עבור משה וזקני / נחזה יום ולילה



ונתברך בפי כהני / שחרית ומוסף ונעילה



ונגיד תהלות ייי / בעלז חן וצהלה



ובזכות חזה סלם / יקבץ עמו ויושיעה:



אלהינו ואלהי אבותינו / זכור חסדך לבניך



שמרנו בבואנו ובצאתנו / ורחם על שארית צאנך



וזוננו ופרנסנו / למען זה שחלה פניך



וארץ קדשך העלנו / לבנות משכן מעונך



ותפרח כחבצלת ארצנו / ומהררי אל בשיר תודה
נביעה:
