


	
	אמונים יום
	



חתימה: אלשבזי






אמונים יום / מאויים / נקבצים להתועד

ביום כפור / סוד ספור / ודוי דברים מבעוד

במחילה / יקבילה / אל חי לנכלם רועד

ויזלף / בטל דולף / בעבי עננים ובאד



על ברקים / ושחקים / גדול יוי ומהלל מאד:





לו תכרע / על טוב ורע / כל ברך ותשתחוה

ראש הכל / ומכלכל / לכל מצוי ההוה

לזה המציא / וזה הוציא / וזה כעור וזה נאוה



וזה דומם / וזה מרומם / ואחריתו תשגה מאד:





שוא גלים / לו מהללים / ונהרות ימחאו כף

ותנינים / בשיר עונים / ותהום בקול שיר נכסף

וסגלגל / וגם גלגל / ואופן וקו המקף



והוא אל רם / עלי שירם / אור בהיר נורא מאד:





בני יעקב / עוון עקב / יכפר לכם יום זה

והוד סחר / עלות שחר / לראש צדיק הלזה

ובת גלים / גביר יעלים / במי טוהר לה יזה



ומשפעת / לבן נבעת / כי יפה היא מאד:





זבול נפתח / וחוט נמתח / מחסד אלהינו

בסוד רחמים / לבן תמים / הדר סגני שבטינו

ונמשכת / בשלכת / חבצלת לעזרנו



וחוזרת / ואומרת / יוי אלהי גדלת מאד:





יזכנו / צורנו / להתהלך במישרים

יפרנס דך / בכל צורך / וגוזליו מזקרים

להתפרנס / בדרך נס / בחסד אל נעזרים



והוא פודם / ומעמידם / ויברכם וירבו מאד:
