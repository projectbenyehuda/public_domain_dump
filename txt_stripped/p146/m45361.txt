


	
	אהב לבי
	



חתימה: א״ב אלשבזי






אהב לבי לבוא / ביושר ארחות נגוהים

ואתבונן בשובו / לעורר כוחות שוהים

ואיך חוטא יחבא / ויש שומר וגבוהים

אימתי אבוא / ואראה פני אלהים:





בדין הכל עומדים / וכל מעשיהם נשקל

וקצוות ארץ רועדים / ואין שם משפט מעקל

והרשעים המורדים / יהיה דתם מקלקל

ופעמי מתנודדים / צמאה נפשי לאלהים:





גל עיני יה ואחזה / בתורתך נפלאות

ועל בן אמתך תזה / מי טוהר יהיה אות

כי בהבלו נתבזה / בתכונות נטמאות

גם את זה לעמת זה / עשה האלהים:





דרכי בחן יתישרו / בעזרת שוכן ערבות

ואם צעדי נתאחרו / בין בני עולה וריבות

וימי חיי נקשרו / בחסד אלהי סביבות

ובין שרידים שנבחרו / חנני אלהים:





השלך יהבך בן איש / על הזן לכל בריה

כי הוא מעשיר ומוריש / ורוחו לכל מחיה

וגופות ערמים מלביש / ועונה עשוקים בשביה

ולדעת אני מחריש / ישעי על אלהים:





ועד טוב וחן ימציא / לעדת קודש צופים

ומבית כלאים יוציא / עניים הנרדפים

למועד מועדים וחצי / יחיה רוחות עיפים

מעפר התנפצי / מחנה אלהים:





זרזיף ארץ תהמה / לקבל שפע טובה

ושני יצרים תגמה / לרפא נפש עצבה

ושכל כמו אור דומה / ואל השלשה לא בא

להיות בחטאם הומה / כי ירא אלהים:





חשרת מימי חסד / מכבה שביב להבות

מקרר לאבני מוסד / זבול אמצעי סביבות

אבן שתיה יסד / בציון בזכות אבות

וכל הוה וכל נפסד / מפעלות אלהים:





טעמו וראו בבינה / כי טוב יוי ונאמן

לשלם לנפש נכונה / שכר לעתיד מזמן

יום תקביל אור שכינה / ושחקים שוחקים מן

ואיש צדיק ילינה / ויראה פני אלהים:





יזכו נפשות נקיות / לחזות גנת ביתן

ראש גלגלים וחיות / יטיבו באחריתן

כי שם סוד ההויות / חקוקים במחצתן

בעלי הזכיות / יראי אלהים:





כובש עוונות סלח נא / לעוון עניי עמך

כי מחסדך נהנה / הכל עשיר והמך

ועולם בחסד נבנה / ביד ימינך נסמך

יוי עשה למען שמך / כחסדך אלהים:





לתשועת אל חכו / זרע ישורון וגילו

אבל רחצו הזכו / וכוחותיכם החילו

במתי תקוה דרכו / ועצת שמאל הזילו

במקהלות / ברכו אלהים:





מה טוב תשובת מודה / על עוונו וזדונו

נפשו משאול פודה / ומסיר הרהור יגונו

ועדי טהור יעדה / לעשות את רצון קונו

ומתחבולות רודה / טוב לישראל אלהים:





נפש צדיק תעלה / באהבה למעין גנים

וגוף מיגון תדלה / וכל מחשבותיו תשלים

זכה יה דל נקלה / אתה מעוז לדלים

רפא ללבב חולה / להתהלך לפני אלהים:





סגלת נדיבים הומים / עניים לבקש פרוסה

זרע קודש שלמים / למי שאמר ועשה

וביחודו מרוממים / ונפשם בשמו חוסה

הודעת בעמים / עזך אלהים:





עורו חברת קדושים / לבוא בית אלהינו

ונערים וישישים / משפט לבקר דינו

ואז נחלץ חושים / ונקרא לעזרתנו

רם על גלגל ואישים / יוי אלהים:





פתחי רחמים פתוחים / לקבל תשובה גמורה

ומשקלי צדק נכוחים / לשקול זכות ועברה

ורשע עליו שלוחים / והוא נדון בחמורה

ומלאכי דין מוכיחים / נוכח פני אלהים:





צדקה עשה עמנו / נורא על כל מלכים

היה עזרתה לנו / להצליח כל דרכים

כי כל פעלותינו / לפניך נערכים

בזכות תורתנו / יתעשת האלהים:





קוראי דברי תחנות / יענה נורא תהלות

ונפשות מתענות / יטהר מכל מעילות

בחסד האל אפונות / נפשות בני תם סגולות

וביד צורם אמונות / לפני האלהים:





רצי עלי יחידה / אל חי מכל אשמים

ובשעת הפרידה / נקי זהמת גלמים

הביעי שיר וחידה / להודות צור עולמים

כי בעלותך עתידה / למזבח האלהים:





שפע נדבות יוריד / רחום וחנון ברצון

על בן אשר הוא ידיד / ואנוס ביד בני לצון

ואז ישכב ואין מחריד / ואויב נדחף חיצון

ופי בתהלות יגיד / הרנינו לאלהים:





תשועת יוי תבוא / לקבץ עניי עמו

וישוב בן למושבו / ויפרוש סכת שלומו

וישפיע עליהם בטובו / ויקנא למען שמו

והחי ישיב אל לבו / מפעלות אלהים:





אמרות יוי טהורות / בסוד חקים ומשפטים

עין ישראל מאירות / מחשך ועלטים

מיסדים לדורות / בכללים ופרטים

ונפשות צדיקים צרורות / בישע אלהים:





להיכל קודש זמונים / זרע יעקב בשובם

רוח ונפש אמונים / כי שם נאותם וטובם

ואערוך תחנונים / לזכות נפשי סביבם

כי יוי בחר בם / אל בית אלהים:





שומרי מצוות זוכים / אל הקודש פנימה

רצון קוניהם חוכים / לעדן פאת קדימה

וגופותן הנמוכים / להם חן ישימה

כי בחיותם סמוכים / רכב אלהים:





במעגלי צדק ינחם / לשוב לנחלת אבותם

כי שם מחוז מבטחם / יבינו לאחריתם

רוח אלהים תקחם / לזכור להם בריתם

כי בסוד הקור והחום / נחם האלהים:





זוכות נפשות יקרות / תהיה להן רוחה

בשש כנפים מאירות / לעלות בעת הצלחה

אבג יתץ כנפי צורות / בהן נפש תפרחה

על מי חסד נוהרות / עד הר האלהים:





יזכה צורי לנפשי / בין עניי קהל עדתו

וימחול חטאי בדרשי / לראות הדר שכינתו

יחזק רפיי וחלשי / בזכות חן תורתו

ויאיר מקו ששי / בראשית ברא אלהים:
