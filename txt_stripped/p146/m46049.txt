


	
	שובי צפירת
	



חתימה: שלם משתא






שובי צפירת חן / לחדריך

הנה מבשר טוב / בבוא משיחיך

ונשמע שוועך / אל אלהיך

כי שב אפו מ/קצוף עליך

אל תיראי ציון / אל ירפו ידיך:





לבשי לתפארתך / וקומי רני

רב לך ביד צר / תשבי תתעני

מקום בתוך חדרך / לשפעך פני

מלך ומשיח / לעמך מני

אז תצרפי היום / לכל סיגיך:





מקדש וקדשיו / לבואך צופים

כלים ברוב חשקך / וגם נכספים

כן יגזרון עירין / בזה ושרפים

כל שונאי ציון / יהו נדחפים

אז תשמחי ציון / במהלליך:





מנחלי שלום / ופתח תקוה

תתעדני תמיד / בטוב ובשלוה

שובע שמחות שם / והוד וחדוה

מור ואהלות בו / בזלפו רוה

כי זה מקום טובך / ותענוגיך:





שקט וגם שאנן / לילדי איש תם

תוך היכלי טובם / מקום תפארתם

צדק וחיים / ילכו לקראתם

ציון מאד תשמח / ביום מלכותם

הפתי כלילת חן / מקום בניך:





תמו עווניך / כלילת החן

בא עת ישועתך / ובא הבטחון

שפעך מאד גבר / כמו מי גיחון

הנה אמת נצרף / כבוד סיגיך:





אמרי לבת שעיר / שהיא אויבת

תחזיר שלליך / ובנים שבת

כי תעמוד בדין / תהא נצבת

חולה במעשיה / וגם כואבת

עת יעבור ינון / עלי מרכבת

תשוב תלחכה / עפר רגליך:
