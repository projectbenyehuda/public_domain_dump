


	
	חדל לך
	



חתימה: אלשבזי






חדל לך דל / אסיר נבדל / ואל תפול בתרדמה

והתחנן / והתלונן / במגדל עוז ועיר חומה

והנעימה / והשלימה / לב ונפש שוממה

בפתחון פה / לדוד יפה / דרוש אולי יקוממה

חבצלת / אשר גלת / ברוב חסדו ירחמה

ימין יוי עושה חיל ימין יוי רוממה:





אימתי / לעזרתי / הכוני דרכיך

והציצי / והליצי / ללמד זכות לבניך

למול דודך / צבי חמדך / הגי קול תחנוניך

ואספי עם / אשר הועם / באש אהבת רשפיך



ועת דודים / בחן יקדים / לנדכאים ינחמה:





לבן נכאב / ירחם אב / הרחמן להצילו

ישמח לב / מתעלב / מתגרת מבהילו

ביום גזר / ונתאכזר / ודלק בהמון חילו

אדון הכל / ברע סכל / עצת רשע והכשילו

עדי ימק / בתוך עמק / יהושפט ביום נפלו



ואכלוסיו / וכל סוסיו / ירדפיהם אף וחמה:





שועת דל / נתדלדל / יאזין צור לכל נוצר

אשר שקל / ברגע קל / בפלס הררי מבצר

ויצילה / לצוללה / קוראה מן המצר

ויציבה / וישגיבה / ויושיע לרש נעצר



ויופיע / להרגיע / לבת נדיב דוממה:





בינו זאת / ולהזות / לבבכם אוהבי שיתו

ובתשובה / אנוש יובא / שאור וחמץ תשביתו

ולגויה / תהא מחיה / שמאל מבאיש הצמיתו

וידחף / ברע יחף / וכל חילו ירתיתו

וילדי תם / באהבתם / בבית אל יביתו



ויתודה / אויל מודה / בלב נשבר כפוף קומה:





זכות תורה / תבשרה / שרידי עם במקהלות

בסליחה / תצליחה / עדת קדש בחן לעלות

להר המור / בשיר מזמור / שיר חדש ותהלות

ואז יחכים / צעיר משכים / וגם מעריב בתפלות

וימצא / כל חפצו / בנסתרות והנגלות



וחיל יבוז / ולא יבוז / עניי עם במשטמה:





ישועת בן / בסוד חשבון / שני יובל ודניאל

אבל סתום / בדת חתום / ולא נגלה לכל שואל

וכוכב רך / כבר דרך / כמו מקל בחסד אל

ובאתתקל / עלי עב קל / שנים כוכבי גואל

בן יוסף / וינון סוף / שני מלכי ישראל



ובתשובה / דודי בא / ויבנה עיר וגם חומה:
