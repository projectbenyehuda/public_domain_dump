


	
	אודה לאל ואבוא
	



חתימה: אלשבזי






אודה לאל ואבוא / בתפלתי ושועי

כי לבי במכאבו / גרם חובי ונגעי

עדי נגעלתי בחובו / ויתפשני פשעי

חמלת יוי וטובו / ימחה חטאי ורשעי

אלהי צורי אחסה בו / מגני וקרן ישעי:





למה צעדי קצרו / ואתבודד בגלות

ואויבי עלי חברו / מרוב עוני ודלות

כי העוונות גברו / וסבבו עלי בקלות

ואחזתני עצלות / ועזבתי מושיעי

אלהי צורי אחסה בו / מגני וקרן ישעי:





שובי יום זה יחידה / ואל ישיאך סורך

וחולי כיולדה / והכוני תוך חדרך

והכיני לך צידה / אולי זמנך יקרך

רב לך מתבודדה / בעמק גולם יצרך

ואם תשובי אשרך / חמד שכלי במסעי

אלהי צורי אחסה בו / מגני וקרן ישעי:





בעוון חטאת נעורי / אלהי אל תבישני

רבים רודפי וצרי / ואני בך בטחוני

הלמוני יסורי / ברוב דלות ובעוני

פדני מתמרורי / ואחריך משכני

אתה עוז מבצרי / מצודתי וגם סלעי

אלהי צורי אחסה בו / מגני וקרן ישעי:





זנחתי כל טובה / ושמך לא זנחתי

צופה דבר שנבא / כי שמץ שמעתי

ישועה לבוא קרובה / להשיב אשר עזבתי

ואם הגדלתי חובה / מחובי נחמתי

את יוי זכרתי / ואליו אשא רעי

אלהי צורי אחסה בו / מגני וקרן ישעי:





יזלף עלי חסדך / כי אל רחום אתה

ויסור מני פחדך / ואשובה מעתה

ואתה מעוז לדך / ומחסה לעם גאלת

וכל תאותי נגדך / וסודי יה חקרת

קצוות עולם שקלת / בכפות קו אמצעי

אלהי צורי אחסה בו / מגני וקרן ישעי:
