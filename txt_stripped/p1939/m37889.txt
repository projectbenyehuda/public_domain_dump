


	
	פני משתנים
	







זולת. שיר תלונה על שעבוד הגלות. סימן: אפרים ברבי יצחק.





פני משתנים / יגוני לא נכחד / הן ביתי שוטנים / שלום מפחד

האם כפר נותנים / לך האל האחד / וכי תשא פנים / וכי תקח שחד.





רשע מתגאה / על עם העמוסים / בהודי מתגאה / ושם עלי מסים

וכל איש משתאה / לרבוי הנוגשים / האינך רואה / מה המה עושים.





ישלחו ריקם / עניים מכיהם / יאכלו את חקם / כדתי חקיהם

שבו את חלקם / בכח עושקיהם / מציקים להדקם / ומריקים שקיהם.





מה רבו רודפים / רבים עלי קמו / ומאימת חורפים / שפתי נחסמו

בראש דלים שואפים / ואותותם שמו / עובדי התרפים / קסם יקסמו.





ברבות מפלצתם / משטרם מיתר / עלי כל עצתם / וכרו לי מחתר

למסע קיצתם / הן לבי יתר / הנה כל עליצותם / לאכל עני במסתר.





ישבתי בגולה / ובארך עבדות / נשיתי גאלה / ויאשתי מן פדות

הנה בן עולה / נעות המרדות / מושל בבן סגלה / וינהגהו בכבדות.





ישועה תאבתי / וממני רחקה / חסן חדותי / עברה ונמחקה

בתוחלתי נכזבתי / ובלשוני חכה / לעגה אויבתי / ולמה זה צחקה.





צמק את שפתה / וקרב חבליה / לארץ עיפתה / תדחה לרגליה

גלה חרפתה / וטמאתה בשוליה / חרבך הנפת / עליה ותחללה.





חומס את נפשי / כמים פחז / מלק את ראשי / בחצו הנשחז

בחרבו החלישי / ובערפי אחז / מלכי וקדושי / צא נא לקראת אחז.





קומה לי לישועתה לכה / וקרב את קצי / הגאלה היא לך / למועדים וחצי

תשיב המלוכה / ומבור זה תוציא / שלחני ואלכה / אל מקומי ולארצי.
