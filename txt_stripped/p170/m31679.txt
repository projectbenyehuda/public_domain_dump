


	
	אליכם אמונים
	



אל יוסף






אליכם אמונים / בחיק האמונים / מתי רעיונים / כברק לטושים,



אדוני קהלות / ואדני תהלות / מפיצים תהלות / לכל אור מבישים,



ונרות באפל / זמנים, ועפל / ומסעד לנופל / ומשען יקושים,



וצהלת עצבים / ורפאות כאבים / ואוצר רעבים / ועשר לרשים,



ידי תם לחזק / ועול להדק / וחפים לצדק / חנפים להאשים,



פזורים להקהיל, / עגומים להצהיל, / חרדים להאהיל, / חררים להנשים,



עתידים נכוחות / להורות וצחות, / וישר ארחות / לשטים ועקשים,



צנועי לבבות / כפירי קרבות / מתוקי אמרות / למרי נפשים,



ושרים, אבל הם / כשרי אלהים / קדושים, ואכן / כקדש קדשים,



בקרבם נשמות / והם על מרומות / במשכן נעימות / וחיות קדושים,



צמאים כמהים / לטוב האלהים / ומעלות גבוהים / לאחז מחישים,



וגופות בתבל / ראות שד ותבל / ימיהם בהבל / ומגו גרושים,



תפתה בגילה / ותשיא בחילה, / אסורים בכבלה / ואם הם נטושים,



ותוליד פתאים / בני איש קרואים, / בעין כל פראים, / בעינם אנשים,



והם המרעים / ונראים כרעים, / והם הצבועים / בצבע נחשים.



להרחיב גרונם / והאריך לשונם / ולנשך בשנם / ואין לם לחשים,



אנושים לשבר / וזדים לגבר / ואיה מדבר / עלי לב אנושים?



ואזנם פתוחה / ועינם פקוחה / ולפני נכוחה / כעורים וחרשים,



להרע יהירים / שבעים שכורים / בלחם שקרים / ומיין ענושים,



חנפים ורקים / בפנים חזקים / כנחוש יצוקים / ונפשות קדשים,



עלילות לבקש / סלולות לעקש / וחנם לנקש / לחיל כנושים,



וארגים בגדים / בחמס אפודים / ועגים מרודים / להאכיל ולשים,



זמורי פתנים / וסורי גפנים / יקו איש ענבים / ויעשו באשים.



צאו מנויהם / וזעף פניהם / וטוב משניהם / מעונות לישים.



וסורו, תמימים, / לחזיון חכמים / ולא אל קסמים / ולקראת נחשים,



לגבעת לבונה / ומר האמונה / וגליון תבונה / לזוכרים ונושים,



כשמן חלקים / וכראי חזקים / לגרון ענקים / וצפת לראשים,



כנטעי ערבים / להורות ערבים / בצחות ערבים / נבוני לחשים,



וחכמת יונים / כנטפי עננים / ונחלי עדנים / כים לא יבשים,



עגילי מליצות / נזמים לעצות / צבי מחלצות / ועדיי לחשים,



יסודות וידות / לכסאות תעודות / משלים וחידות / עליהם קרשים,



חלי צואריהם / ותורי לחיהם / חכמים עליהם / מידים חלשים.



נגידי אמרים / להחכים ולערים / מתוקים ומרים / ורכים וקשים,



לגלות כסויים / וכסות גלויים / לחדש בלויים / ובלות חדשים,



לישב מעונים / לאישים מבינים / והחריב לשונים / נויהם ולשים.



ואלה ילדים / לשכלי מעידים / ואם הם עבדים / בצור פי כבושים,



בני רעיוני / ונוזלי ענני / לפי צור לשוני / ועטי חלושים,



לדעה וחכמה / כדלת וחומה / וסתרי מזמה / לחפצם דרושים,



בפיהם פלאים / ודברי נביאים, / תרופת חלאים / ועז נאנשים.



לאיש מעלומים / בבאר עלמים / רגעיו כימים / וימיו חדשים,



מפאר במעניו / אבות שיר ובניו / ונעים לפניו / הררים ומשים,



ונוסעים וחונים / בארצות מבינים / והם שאננים / בביתם כנשים,



ולמד לרעים / תבונות ודעים / ואיך הם רצועים / ויצאו כחפשים.



כשלחן ערכתיו / כלחם פרשתיו / ומופת נתתיו / לחכמי חרשים,



שלחתיו כמשאות / לשרי בריאות / כשיות בריאות / ויין האשישים,



ולפני שורים / – אבל הם עירים – / כלחם שעורים / וכנזיד עדשים,



כקר מי גבאים / לחם הצמאים / ובלבב מקנאים / כחמם כאשים,



מדמים פתאים / לנכר לבאים / ולערך דשאים / לגבה ברושים.



היהיו חנפים / בתבל כחפים / ושרי אלפים / כשרי חמשים?



ונחל עדנים / כמו רוש פתנים / ועורות גדיים / כעורות תחשים?



ומשלי חכמים / כהבלי חלומים / ואם אלגבישים / כדר או גבישים?



ורעי מזמה / כרועי בהמה / ואם עש וכימה / כעפר וגושים?



ויפרו צמחים / בגבי צחיחים / ומים שלוחים / היהיו נתושים?



ואם איש יאסף / חסדים כיוסף / אשר פז וכסף / לרגליו נדושים,



לגזע חכמים / ונצר עצומים / וצעיר לימים / ומחכים ישישים,



אשר מפעלותיו / כפי עז אבותיו / וארך עבתיו / כחזק שרשים,



ושרף כגחל / ושטף כנחל / ושאג כשחל / בעדרי תישים,



קחה סוד וחבר / לערב ועבר / ומזון ושבר / לפי כל מבקשים.



עלה בם והצלח, / הכי הם כמלח / לאמר, ושלח / לרדף חמושים,



ורום על גבוהים / וכוכבי נגוהים, / שרה עם אלהים / שרה עם אנשים.
