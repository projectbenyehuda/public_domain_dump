


	
	הריחותי הדסי השלומים
	



[הריחותי הדסי השלומים] / ר' משה אבן-עזרא




הריחותי הדסי השלומים / במכתב או ערוגת הבשמים



ואריתי ביד רעיון כפרים / וראיתי בעין־לב יהלמים



ברמים מעשה ספיר והמה / בנפך או ברק העט רקומים



נאם הערים בחכמתו פתאים / וניב פתה בערמתו ערומים



ושיר נפל צבא־שחק לחנו / וצופי מעניו כושל מקימים



קראתיהו ומי עיני יזובון / והמים אמת כדם אדמים



ומחיתי יפי טוריו בדמעי / ושחתי דבריו הנעימים



כתב דוד נתקו חבלי אהביו / עדי חבלי יצריו בחלמים



מצודים אפרשה להם בערשי / ואשטח על־ידי נומי חרמים



ונפשי מבלי שנה אישן / ולבי ער ואם־עיני עצומים



ומה־אוחיל ומסעיהם רחוקים / ומוראים שבליהם ואמים



ילל ליל ילהלה ואך ימ־ / צאו עיני בנות־שחר שתמים



יגון פרוד יקיצני וצירו / יציקני רעות כוכבי מרומים



והשחק כמו־אהל תכלת / בפניו פזרו אבני לשמים



חרוזים הם בצוארו וכימה / ועיש על־שתי אזניו נזמים



וליל יעטה שקו וברק / יקצץ בעדו חוטי כתמים



אדמהו לכושי רץ בפצע / ושמלתו מגוללה בדמים



רעיתימו עדי היו מעילי / אפלתו ביד־שחר פרמים



פתילי זהרו נראו בנשפו / כשיבה זֽרקה בשער עלמים



למי אבכה ואתנודד ועל־מי / דמעי אתנה נגדי אגמים



הלאחים אמר או לקרבים / ואכן בענן מרחק עלמים



ולבנים אשר עטו יגונים / והיו מפאר הורים ערומים



ואם על־אהבים שלום בפיהם / זמן אתי בני ימים שלמים



אשר אשאף לרוח אהבתם / ואם לא־חוללה מאז גשמים



בצור חרבו דרכי שך זמני / ונתן כל־פרציהם סתמים



ואין איש בעדם נדרש לבד אוב / ולא נשאל לבד קסם קסמים



לך שכן מעון שכל וחפש / בנר דעו חדריו הפנימים



והאשר בדרך האמונה / והמשים ליד־חסד פעמים



לך אצעק חמס ימים אשר ג־ / ברו מני ובי קמו כקמים



והמצוני ביד גרות לבדי / ואך נפשי והיגון תאומים



וזקן שֽקדו עלי חיליו / ועדנה הגלתה הגלת שלומים



בשרי הנדד בלה ושבר / כנבל יוֽצרים כתות עצמים



ולא התוה לנודי קץ ומועד / וחזיוני ימי חברה חתומים



קוצות הזמן שבו ושבו / זקני הֽיגונים לעלומים



לחומי מאכל חלף ללחמי / ודם דמעי מקום דמעת כרמים



והמתוק במשקי ראש פתנים / והדשן לפי שרש רתמים



יבהלני בגרותי שמעות / ומכאבים גדלים ועצמים



הכי קרא זמן לשבר בחורי / בני־חסד ומחמדי תמימים



וכי חתם בעד כוכבי תהלות / והשכין מגדלי־חסד תהמים



אהה לי אוֽמרה על־מפקדיהם / אהה לי ואבוי אלף פעמים



אבל מות חלי ישוה יקומים / ומקרה הכסיל יקרה חכמים



לבנים מאנוש יפיל כגזית / וערמנים יהי שבר כשקמים



והם רצו לפנינו ורכבי / הליכינו להשיגם רתומים



ואמנם משפטי הצור ישרים / ולו הדין ונחנו האשמים



כתבי זה ומעגלי בארח / ומדברות מגמתי וימים



לנחת על־פני־ארץ גדודים / ורות מדמי עיני תלמים



עדי אגוע בארץ ימעטו בה / עלי קברי מאד בכים והמים



ופקדוני ביד האל עזבתיו / אשר נקרא שמו אב הֽיתומים



ואבטח באשר ישפיל גבהים / וידיו יֽרדי דומה מרימים



ואולי לי הנחה באנחה / וכמה תחלאים מחלימים



והימים והלילות כהרות / ומי־בן תוֽלדות לילות וימים



וטל ישקה ארצות בם לבבי / כמי עיני אבל נחל קדומים



ועל־השֽכנים שם לעדי־עד / כחול הים וכמעתיו שלומים.
