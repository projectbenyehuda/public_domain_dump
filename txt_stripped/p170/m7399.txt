


	
	נטפי דמעות מערף נגרעו
	



נטפי דמעות מערף נגרעו / ר' משה אבן-עזרא




נטפי דמעות מערף נגרעו / או גחלי־אש משרף שקעו



כי־אמצאה אלה נבכים בין שני / צדי ואלה בין לחיי תעו



אריב יקד אשי ואוכיח המון / דמעי הכי בו משלט נמנעו



ויענו גבר מאד השד עדי / חקם אשר־לו נבראו פרעו



לא־מֽצאו עזים ידיהם לאלות / לא־אֽמצו לנוד ודומם פעו



איך עֽמדו לבות ואיך לא־שֽלפו / ויֽצאו מגו לשד שמעו



איכה עלי צורי שלחים יפלו / לבות נפליהם ולא יבצעו



הוה לחילה הנפשים שֽכנו / רעה לחולה הקרבים נעו



יתלחשו אישים בהגידה הכי / רהו להרים קול וגם־ירעו



מי האמין כי מגדלי עז נֽפלו / או כי עצי־עדן באף גדעו



או בא מאור־ארץ אשר בין כֽוכבי / אור עקבותיו מאנוש נודעו



אחי אבל אח למאורים אך ביום / יתנוססו יוסיף ויגרעו



צעד עלי־ארץ וילדי מחשבות / שכלו עדי כנפי כרוב נגעו



סודיו עלי כני אמונה נֽוסדו / מליו עלי אדני אמת הטבעו



מאז זממיו נתכנו לפיק רצון / צורו וכל היום בחקיו שעו



אצלו אצילי מאמר שחו כמו / נגדו נגידי מענה כרעו



שמו לפיהם כף מתי צחות ביום / אמר ודבריהם לפניו לעו



הרבו גבהה לענות יום אֽלמו / אך נאלמו עמו ואליו שעו



סרו למשמעתו מתי־ספר ולו / אזני מביני הספר רצעו



אתמה עלי עטיו אשר לא־פֽרחו / מנחלי ידיו אשר נבעו



נדבו עדי נגדם ידיהם אֽספו / עבים ומהגשים עננים שעו



טוריו למר על־הכפר דמו ואם / נפך עלי־שהם ביד קלעו



הוריו לפי־חסד ידיהם זֽרעו / וירו במי צדקו אשר זרעו



ההר אשר חמד אלהים איך בני / איש נֽשאו גבהו ובו פשעו



נגהם צבא־שחק במותו אֽספו / כי־דל מאור שמש ופניו רעו



אלו לכוכבים קרבים נהמו / כים ועינים אזי דמעו



הפליא זמן למרד ובניו נֽשאו / יד באדניהם ובו פשעו



לקחו אבי־לקח וסוד המאמר / שבו ומופת הֽיקר קבעו



לקחו אדני מעלי ראשי וראש / הדור באף פרעו ולא פרעו



אבי ומחכים רעיוני יום אשר / ישתערו לבין וישתעשעו



לא־מֽצאו נוח עלי תבל עדי / בינות צלעי אהלם תקעו



ידו אבנים בי בבור גרות ולא / חטאו סגור לבי בעת קלעו



אסרו להשמיד את־בני אבי אבל / לפריד בני־עיש הלא נשבעו



דמי לאט שתו ולא־רוו כמו / אכלו שאר גופי ולא שבעו



אתי בפצעי הנדד ענו עדי / יזמו להמית אחרי פצעו



אחיו אשר מיום כבודו פֽקדו / לא נעזרו משד ולא נושעו



חלו לחול אידו וערתם עלי / עצמם כרגע נמאסו רגעו



לא־חֽגרו שק כי־לבושם משחור / הלב ובדמי הבכי צבעו



לא־אֽספו אליו להשית יד עלי / עיניו ולא־פרעו ולא־קרעו



אמרו הנפשו בחלה בהם ואם / הם מאסו אתו ובו נקעו



או אם־זמן בגד הפיצמו בלי / חמלה ובמכוני נדד גועו



חלף ולא־חלפו סעפתיו אשר / ידיו ימי השחרות נטעו



הוא אם־ערירי נאסף לא־נכחדו / ילדי תהלותיו ולא נכנעו



יבלה עפר־ארץ והם יתחדשו / יצרו עריפיה וישתרעו



איך אתנה שנה לעפעפי ולא / יתחברו לנום ולא יתרעו



או איך אחי אם־לא אקוה לעלות / עוד שחרי פניו ויבקעו



מיקוד צלעי אהמה תמיד ואם / רגלי בצולת הבכי טבעו



לא־חֽמדה נפשו שכן־לו עם־מתי / רע אשר רוח התלים רעו



דור נאלח עיניו מאד רמו אבל / אתם להביט האמת השעו



רפו ידיהם לעשות חסד ותם / אכן ברב־כח להרע צעו



קול העביר מות בכל־נוצר שכן / כפים בכף־רגל תמול רקעו



מקרה כסיל יקרה חכמים נֽודעו / ספו וחכמתם והתבלעו



איה מלכים יסדו חפצם שאל / אם־נמשכו כמעט עדי בצעו



ילדו לבהלה וחיל עֽתקו / עזבו ולריק כל־אשר יגעו



כסף כחול צברו ופח היה ליד / מעגל תבוסתם והתגלעו



שמו אדמה מאבסיהם והם / סביב בלי־לב כשורים געו



אכלו ושבעו והשמינו עדי / קיו לפי־רגע אשר בלעו



מות אשר השוה נברים צֽדקו / במעי אדמה עם־אשר רשעו



לו יֽהיה הדר פני שרים אזי / ידיו לאחי משאול פדעו



תמהו בני־יום לחיותי אחרי / המֽצאות אתי והתרעעו



לא־יֽדעו כי־לי מזמות יפגעו / בהם בטרם בי באף יפגעו



מתי אריות פחדו מקול אנוש / או מעבר רוח הררים זעו



שלום לאור מערב אשר ערב כפי / שאת לבבות בעדו שסעו



יבלי סליחה יענו קברו לבל / יבלה ותחתיו רחמים יצעו



יעמד לגורלו ביום ימֽשכו / בו שֽופרות ישע ויתקעו



לבכות עלי אחי פעמי אחרו / כמעט ואסע אל־אשר נסעו.
