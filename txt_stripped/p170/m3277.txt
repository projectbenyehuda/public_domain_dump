


	
	עיני עננים בבכי נמאסו
	




עיני עננים בבכי נמאסו / עד עקבי גן־ההדס נחמסו



בכו בחזק נֽוזלים וישחקו / שפתי ערגותיו אשר כעסו



ענן אשר הגשים ושחק רעמו / פניו ואפי האדמה ששו



הרעים ואכן בעבור הרעים פני / שחר ועפעפיו לקולו נסו



כנפי ברקיו אחזו כנפות פני / חלד וצבאי לֽילה פרשו



רצים ושבים כנשרים נחפזו / העת עלי־אכל בארץ טשו



לולי בני־עב יֽרדו לא־נעלו / צצי אמירי־גן והתנוססו



היו אפילות עד מטר־גשם וחם / שמש להראות זהרם נגשו



בשמי ערגת זֽרחו אכן ידי / צפון צפוני מרקחיהם שסו



ריח אבק רוכל והרוח כמו / רכיל בסודיהם אשר כמסו



הראו יפי־פנים אשר באפר סתיו / חפו ימי הקר והתחפשו



הודו בלי לשון לעבים יֽנקו / אתם ועיניהם עליהם חסו



כרים מעיל תולע וחור עטו עלי־ / בגד תכלת בעדם פרשו



עיני בעיניהם פלאים יחזו / כי מעשה תשבץ פניהם עשו



בם גרגרי שהם והם לחים הכי / אתם רסיסי לֽילה רססו



מה־יֽקרו לו החרוצים לֽקטו / אתם וכפי רֽכלים כנסו



נצהל במשכיתם וקולם יתנו / עוף בין עפאיהם ויתעלסו



נשיש בקול הסיס ותר אם־יענה / כנור ועוגב עֽגבים ימאסו



חי סֽבאים סבאו בבת־כרם אשר / אתה נדיבים לעלץ ארשו



תמשול בראש שרים ומן־אז פעמי / דלים גאונה דֽרכו עססו



זקנה ואור לא־ראתה מיום אשר / אתה באוצרתם ידיהם עשו



שפה זמן עצמה וכמעט נעלמה / לולי גביעים זהרה עמסו



אש בערה אך הֽיגונים בערה / מעם ברשת הנדד נתפשו



עדני בני עדנה ואון נפשות מתי / אונים לחול זקן מאד גרסו



בי מזרקי ספיר כמים קֽפאו / בם גחלי האש אבל נמסו



איך גֽמלו חסד עלי כל־כאבים / ואני לבדי מאנוש חמסו



אשתה ואשתאה ואמהל את־עסיס / כוסי במי עיני אשר נמאסו



איכה לפי־רגע גביעים יֽוכלו / לרפוא אשר ליל ויום הרסו



ילדו בנות ימים עלי ברכי ולא / גמלו וילדיהם בצלי חסו



עד על־שמי יקֽראו קרת ולי / יתילדו תוגות ויתיחסו



עדי לבד עדו לנצל את־צבי / עדיי וחיתי באף רמסו



באתי באחרית הזמן עת גֽמרו / כל־החסידים ואמונים פסו



ובני בלי־שם גֽבהו כאשר מתי / השם פלאים כֽרעו קרסו



ספו כפירי דת ותחתם נבראו / חרפת בני־אדם אבל רמשו



שלחן כזבים עֽרכו בו מאכל / קשר ולחם לחמים פרסו



זכו בעיניהם והם יטמאו / בדמי כסילותם ויתבוססו



רכסי נתיבים בארג שיר דֽרכו / לא־סקלו להם ולא־פלסו



קוריו כעכביש ולזאת ילעגו / לו שֽמעי שמעו ויתקלסו



לו חֽכמו בצור קלונם טֽמנו / וסחי חרוזיהם בעפר כסו



אבכה פליטי תם אליהם נֽגעו / נגעי בני־הפך ובהם פשו



ויגאלו בהם ונפשם טנפה / חלאת פעליהם ולא־כבסו



עד־נדלחו מי מעיני־צדק אשר / ישתו ונחלי האמת נרפשו



ושדי אהבים מעכו שונים ביד / בגד ודדי הֽידידות עשו



קמתי להשיב את־גמוליהם אבל / אתי מליצי אהבה אנסו



קח מגדלי בצר להבצר הכי / שורי בנות־שיר בערים נתסו



או טבעות כסף ידי אמן ליד / שמש בחוטי מאמר רכסו



שרים לדבר אם־יעושון יקראו / להם בני־עיש פתאים הסו.
