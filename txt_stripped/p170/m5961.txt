


	
	[מציון אל עליון]
	



סליחה

סימן: משה




מציון    /    אל עליון  /  אשר לו נתכנו עלילות –

צור קונה  /    הבונה  /     בשמים מעלות

תנה ישע   /    לשבי פשע   /   וזנח חטא ומעילות,

ולמהללים   /   במקהלים  /   נושאים כפיהם תהלות

עבדי אדני  /    העמדים  /   בבית אדני בלילות:      [תה' קלד, ב]

מה-ידידות  /    נכבדות    /   לדבר בבית מעוניך! –

ולנטש מחוזים  /  ולאחז  /     בשולי פעמוניך

ולנער זמן  /    לא נאמן   /   ולשמר מאד רצונך

ועת תהלה  /    ועת תפלה  /   לבקע ענניך

וכל בכיל   /    כנף ליל   /    ואחז כנף ארונך

ואשמרות   /   עת שמורות  /   עיניו יזיל לפניך

ולך יכסף  /    כי תאסף  /    תהלתך בחפניך

ושם בלבות   /  מסלות   /   אל השחקים לעלות:

שחרתיך    /   ושרתיך  /    בעיני לבי ושכלי

ולהחזיק    /   במסבך   /   נטשתי נוה אהלי

והון חרוץ,  /    וארוץ   /    אחר עקבות אלי

ולא נודעו,   /   אבל פגעו    /  רעיוני לצור גואלי

וכי אעטף   /   יום תטף  /    מלתי במהללי

אשתאה   /    ולא אראה –  /  והוא ישב ממלי!

ואמנם על   /   ידי פעל   /    גבורותיו אשר הפליא

אני רואה   /    ולא במסוה /    אבל בחידות נראה לי

והוא רוכב  /   על עגלות  /  ויושב על התהלות:      [תה' כב, ד]

היש יתרון   /   לבן מרון  /    ואם ירבה תחנוניו?

ואותו   /     בהותו   /    וחטאו מלאו עטיניו

ואין זוכר  /    לב עוכר  /    ליום יקרוהו אסוניו

ליום רחוק  /    ירחק חק  /    אשר בו יעלו עשניו

וכי יסחר   /   ליום מחר  /    תשופנו לשון פתניו

ואיך יגאה  /    ולא יראה  /    זמן ישכן בית ענניו,

והוא יצבר  /    והוא יעבר /    ואסף רוח בחפניו

ואיך ירכב   /   ואיך ישכב /    תחת נחלי עדניו

ויום יבא  /    במחלות  /   ביד יחתום על מגלות?     [איו' לז, ז]
