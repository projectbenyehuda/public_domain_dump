


	
	לזמן בגד
	



אל יצחק בן משכראן


לזמן בגד / אותי רמה / לא אשם כי / הרבה אשמה,

ולרב מעלו / לא אמר לו: / הרף מאף / ועזב חמה.

כי מי יעתיר / קשר העתיר / ומאד רבה / המשטמה.

לא אמן בו / במשך אפו, / לרע יעשה / את־הזמה,

ולטובותיו / במזמותיו / כל דתותיו / הבל המה.

סותי מעל / יעטה כמעיל / בגדי בגד / לו כשלמה.

לכאיב רוח / יתן כח, / לחלש אונים / ירבה עצמה.

יכין צעד / אך חיש ימעד / ירחיב טפח / יציק אמה.

בבני שכל / הוא מתנכל / ויכעס על / ילדי ערמה.

בהם אידו / יהפך ידו / כאשר חרמש / יקצר קמה.

יחמד שובב / ערל לבב / ושפל שכל / וגבה קומה.

כל איש ישכל / כי הוא סכל / ועלי כסל / יעש פימה.

תבנית גלם / לו יאלם / ודמות צלם / לפני במה.

עיר צעה / רוח רעה / אך צוארו / ילביש רעמה.

אזנו תשנא / משלי בינה / גם עת ישאל / לא מחכמה.

לזמן יכסף / כארי לטרף / נפשי, רעי, / שאלו: על־מה.

כאנוש פתי / השיא אתי / כי החליק לי / שפתי מרמה.

בימי חברה / חצי עברה / הבר וסגור / לבי רמה,

וזמן שבתו / דרך קשתו / עד מחשבתו / עלי קמה.

כמה החריד / טרם יפריד / כי על פיהו / היתה שומה,

ובהתחבר / מירא שבר / לבי כל יום / יהגה אימה.

השקה את־פי / מזרק צופי / ועלי כפי / כוס החמה.

כדמות נחש / אין לו לחש / בו החמה / לו הרקמה.

יואר ערב / עת הערב / קרא: שבר / נדמה נדמה!

הרחיק בינות / חלדי בינות / יום החצות / גוף ונשמה.

מהם ארח / אל יד מזרח / מהם הפאה / לפאת ימה,

ובעת רחק / דודי יצחק / נפשי בלתו / שכנה דומה.

רדף שכל / אבקת רוכל / אהב תמה / שנא תרמה.

מליו עתקו / מצוף מתקו / רכו אמרי / פיו מחמה.

כל איש זחל / היה לו חל / וביום חמה / עליו חמה.

מיום נדד / גרתי בדד / רעה עיש / שמר כימה.

עיני תטף / דמעה תשטף / כטל יערף / על האדמה.

הוקד קרבי / בשביב עצבי / מעב עיני / לחיי גשמה.

איך לב דוה / מנהו רוה / אכן יצעק / צחה צמא.

וכחותם צר / תבל תצר / בי, כי אחריו / נותרה שמה.

אם גיל יקרב / או כוס תערב / ולעפעפי / אתן נומה

אהיה מוסר / מכת מוסר / אלך קדר / מאין חמה.

רע תמים / הם הימים / כל אולת / הם וכלמה,

גם היא תבל / תעשה תבל / בעזוז מצח / וביד רמה.

אוחיל עד בוש / אולי תבוש / או עוד אכבש / אותה אמה,

אולי תשאף / לשוב מאף / ולמפלתי / תשית קימה,

וקרב תצית / אתו תשבית / כי עת שלום / עת מלחמה.

תרים אדם / מהאדמה / ושבו תשיב / ממרגמה.

אט בת־עיני / על פקדוני / היא הנפש / השוממה,

אט לה, אולי / תשוב אלי / או תוחיל עד / באי שמה.
