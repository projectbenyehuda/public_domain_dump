


	
	רוח ששוני וחן מנעמיו
	




רוח ששוני   /   וחן מנעמיו



הפיחי גני    /   יזלו בשמיו!





מהר מורי   /  וגבעת הלבונה



צפון, עורי,   /  ובואי תימנה!



אולי עפרי   /   ירד אל גנה



לאכל פריי   /  וללקט שושנה



וכימי עלומיו   /   יחדשו ימיו:


  


שאלו וראו  /   בעלי אמרים



וחפשו וקראו  /  צפוני סתרים



אולי תמצאו   /  לחליי מזורים



ולדודי תשאו   /  שלום אשת נעורים -



ויעיר רחמיו    /  להפיק את שלומיו:


  


הצבי, אלי  /    כמאז שובה



ועדי כלי   /   עלי השיבה



ונרי גילי    /   לפני היטיבה



וקומם אהלי  /   ואת יריבי ריבה



ובן מטו פעמיו  /   ומחץ מתני קמיו!


  


חזקו דברי   /   בן-אמתו עליו



ויום יום יכרה  /  שחת לרגליו



ומדון יגרה   /  וריב יגיש אליו



וענף ראש פרה  /  ישים מאכליו



ושרש רתמיו   /   יתן מטעמיו!
