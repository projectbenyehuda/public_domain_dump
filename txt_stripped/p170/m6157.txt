


	
	שמר דת חנכה
	



“מאורה לר' משה ן' עזרא”

סימן: משה הקטן


שמר דת חנכה /   להבין ולהורות /  בהעלתך את הנרות


  


מצוה להדליק   /  שמנה שלמים  /  וברך ראשון



שלש פעמים



וזמן למעט   /   בשאר הימים  /  פתילות ושמנים



אסרו חכמים



בשבת    /   לחנכה יתמים  /  ואם כבו



הדלק מקימים



כאשר פרשו   /  מבארי סתומים  / אמרות אדני



אמרות טהרות:




תה' יב, ז


שעת דלקת   /   שקיעת החמה /  עד כלות רגל



משוק באימה



ונר איש וביתו  /   חקה שלמה  /  נר אחד לחברה



מצוה תמימה



אך המהדר   /  לאל בעצמה   /  נר לכל אחד



כדת שומה



בדברי נבונים   /  לומדי אימה  /  המשפטים



והתרות:



         ויק' כו, טז




החזק במאמר  /  בני תושיה   /  היותו על



פתח החניה



בטפח סמוך   /   לדירה ראויה  /  ושעור כמבוי



וסכה עשויה



ועת סכנה    /  בזמן שהיה   /  על שלחנו



מצוה רצויה



ובחלון אם  /  דר בעליה   /   לפרסם את הנס שהיה



ולהודיע כל הקרת:



 ברא' מב, כט




הרצות מעות   /  אסור לאורו  /   לבל יהי מבזה



אנוש חק צורו



להשתמש בו  /   הצור אסרו   /  ועל יד אל יהי



עת מבעירו



שמא יאמרו /   לצרכו בחרו /  ובפנים אל יהי מכין אורו



בהיטיבו את הנרת

 שמ' ל, ז





קבעו היותו  /   בשמאל נתונים  / ואל ידליקו בם



נרות משנים



אך זה מזה   /   חקים הגונים, /   ובשבת אם



דולקים ופונים



מכבים אותם  /  ולהדליק שונים  / וחרש ושוטה וקטן משנים



ונשים באות אותה מאירות:



   ישע' כז, יא

  


טהורים פרשו  /  בדת יומים   /   נר שיש לו



פיות שתים



מוציא חובת   /  אנשים שנים  /  פתילות נתונות



ולו למאתים



בקערה יוצאה  /   בלי עצלתים   /  ובלבד לכפות



כלי בידים



וחצר שיש לה  /  פתחים שנים  /  צריכה שתים לעינים



היותם על פני ארץ מאירות:




ברא' א, טו

  


נאמו בימים   /  אלה פליאה   /  שלא לספד



בם לתלאה



ולגמר הלל   /   בם בקריאה   /  ולזכר חנכה



בהודאה



ובברכת מזון  /   בסוד באה  /   וזהיר במצות



ודת נוראה



יזכה לבנים  /   בעלי הוראה /   חקת עולם



תהיה לדרת:




ויק' ז, לו
