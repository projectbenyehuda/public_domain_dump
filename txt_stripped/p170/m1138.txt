


	
	הריח מר מאחז האפסים
	



[הריח מר מאחז האפסים] / ר' משה אבן-עזרא




הריח מר מאחז האפסים / ואם־רוח תנפף ההדסים



וענן או עתר קנה וקדה / וברק או ברק כסות עסיסים



והעבים יריקון הבשמים / ומראשי הדסים הֽרסיסים



והרים יפצחו רנה בלי־פה / ואם תרים עלי בדים וסיסים



וארץ משבצות זהב לבושה / ומשי כתנותיה ופסים



וישרו כל־נתיביה לצעד / ושבו כבקעות הֽרכסים



והבתים ירננו ומקיר / צרור יען ומעצים כפיסים



ופנים זעפים לבשו עליצות / ועליזים מתי יגון וששים



ושפתי עלגים צחות ינובון / לקומם בניני גיל ההרוסים



הכי נראו פלאים הצפונים / אשר היו בלב ימים כמסים



ואהל הֽיקר חבר וללאות / תהלות הֽובאו בהם קרסים



וימים קֽרבו עצם לעצמו / ודבקו בשרים הפרושים



שמחות נעלו על־הררי מר / ועל־גבעות לבונה הם כנסים



ועל־כנפי שחרים קול מבשר / בתבל לא עלי כרים וסוסים



שלמה אֽרשו דדיו כבודה / ואם־סהר וחמה הארושים



אשר אשרו נתיב־שכל פעמיו / ורגליו דֽרכים מוסר ובוסים



ומנער שעפיו על־שחקים / ורעיוניו עלי־עיש עמסים



וגבה מבני גילו כגבה / בני איש מבהמות או רמשים



ויקרו מעלליו מעליהם / כמו יקרו פנינים מחרשים



ירוצצו אבק רגליו להשיג / ואיך צפור יהי רדף פרסים



ענו איכה כסיל נפגש בכימה / עלי־ארץ בלי דרש ומשים



ואם אינם הלא כנפות הדרם / עלי אפי תהלותם פרשים



ואיך גזלו מאוריהם הנראו / אנשים בזזים אורים ושסים



ידידיו מהרו אל־יין ידידת / שתות היום במזרק המשושים



ועוד התעתדו לכפל שמחות / ופתחו ההנחות מאבוסים



כמים קֽפאו הם הגביעים / וגחלים בתוכמו נמסים



שתו־נא כאשר יסד בביתו / שתות כדת לבל־תהיו אנוסים



וגיל בחור באילת אהבים / והרנינו שניכם נעלסים



והתעדן בקומת־חן כתמר / אבל תנע כבדי ההדסים



ואל־תפחד לקול חליה בצואר / לעת נשף ורגשת השביסים



ואל־תרהה לעינים כיונים / אשר שכרו אבל מיין חמסים



ויעמד לבך בחבוק זרעות / באצעדות ותפארת עכסים



ואל־תברח לצפעני קוצות / עלי פנים במי־בשת כבוסים



הכי לקרֽאתך יצאו לשלום / ואם־סתרים הדר פנים וכסים



ורמונים בגן שושן ואכן / בראשי משמרות בשם תפושים



וידיך בעברם על־פניהם / לאטם מעכים אתם ועשים



ודע כי־הזמן עבד רצונך / ובניו לעשות חפצך כנוסים



להקריב משאלותיך יחישון / ואת־כל־תאלת לבך מניסים



והא שיר דוד בך ישמח לבבו / ומורשיו לך פרחים וטשים



מעיל כבוד לבל־יבלה לעולם / עדי יהיו יסודותיו נתוסים



הכי מימי ידידותו כשלג / מאד זכו ברגל לא רפושים



ואם־יסֽתרו ממך בלבי / בנרות דעך יהיו חפושים



למען למבינים הפנינים / ולרקים בריאי הכבשים



ותאות הגברים לאמרים / והפתים לברברים אבוסים



יליד טובים חיה עם בת־נדיבים / בצל שדי שלוים בו וחסים.
