


	
	מרום וקדוש שעה
	



לקרובה א'




מרום וקדוש שעה שועת שבוים  / ממעונים



התעשת להם וכתבם לחיים  /  עם אמונים



וקרא שנת רצון לריב העניים  /  והאביונים: / ישע' סא,כ;מא,יז



התעשת





שלומים נפלאו  /     ממעונים



אל עם נצבאו  /     עם אמונים



בבכי יבאו  /        ובתחנונים: / ירמ' לא,ח



התעשת





העלם עם בניהם  /    ממעונים



וסלח חטאיהם   /     עם אמונים



אשר כשלו בהם  /      אבות ובנים: /שם ו,א



התעשת





קשב נא בניך  /    ממעונים



והשקם לפניך  /    עם אמונים



באר מים חיים /   עם גנים: / שה"ש ד,טו



התעשת





טל סליחה הרק  /   ממעונים



שערי חן הפק  /    עם אמונים



ויבא גוי צדיק  /  שמר אמנים: / ישע' כו,ב



התעשת





נברים הבינו   /   ממעונים



כשלג ילבינו   /   עם אמונים



אם יהיו חטאיהם  /   כשנים: / שם א, יד



התעשת



שלח קול חסדיך  /  ממעונים



להושיב עבדיך  /   עם אמונים



מעבדי אדני  /     הקטנים: /ישע' לו,ט



התעשת
