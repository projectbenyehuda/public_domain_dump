


	
	היעמד לבבי ונפשי
	



על מות בת הנגיד אברהם






היעמד לבבי / ונפשי מחוצה / ותחזק ימיני / והנה רצוצה,



וארגע בנודי / ויגון מנידי / ותערב שנתי / ותוגה מקיצה,



ואסתיר יגוני / ומופתם בעיני / ואשקיט שאוני / ודמעה מאיצה,



לכבות שביבי / ולהב בקרבי / ומבית ללבי / מלהט וחוצה,



וישגו מצרי / כאחו במים / ויגאו מצוקי / כגמא בבצה,



אקוה הנחה / ותחוש אנחה / ולהיות רוחה / והנה נאצה.



ואשנא חיותי / ואשאף למותי / ומה־לי בחיים / אשר הם לשמצה.



ימותי בתבל / מרורים ואבל / וחפץ להבל / ואוה חרוצה.



התשנא אדמה / מתי בין וערמה / ותקוץ בחכמה / כשמש מציצה.



בשימה שחותי / כפי גאותי / והוד מעלתי / בלי חטא נתוצה.



ויום יום תגבר / המון שד ושבר / וצוקים תחבר / וגילים מפיצה.



וימים כקמים / מציקים וקמים / להגיש עצמים / ומלה חמוצה(?)



ועם בת מלכים / וזרע נסיכים / יגרו מדנים / וריבות ומצה.



להשמיד שארית / בניהם ונצל / עדיהם ושומם / כחומה פרוצה.



ושרת גבירים / וצרת מאורים / לאפל קברים / שלוחה ואצה.



ואשה חכמה / ותמה אימה / וברה כחמה / ויפה כתרצה.



וקומה כתמר / ופנים כסהר / ולחי במי חן / ובשת רחוצה.



ובינות עלמות / כשמש מרומות / בידה מקורות / לנעם ומוצא,



וגפן ענפה / ופריה תרופה / ופארה עלי גב / בנות עש ונצה.



פנימה כבודה / בנפש שפלה / וממור וקדה / לבד היא קרוצה.



מבישה לבנות / ושרה לבנות / לשמר אמונות / והסר מרוצה.



לשד הכבודות / חליצות ומדות / מסירות וקורעות / כתנת חמוצה.



ומפריד אמונים / עלי מי אמונים / וחברת עפרים / כרגע נפוצה.



וחל בן נדיבים / לעפרת אהבים / ונפשו בחיים / מאוסה וקצה,



מצעק בתוגה / לציצת ערוגה / אשר פרחה בין / צלעיו וצצה.



כעפר בציה / עלי נד צביה / אשר בין זרעיו / בפתע חלוצה.



מרי פי ושיחי / אשלח לאחי / ומרד ברוחי / בשירה נחוצה



לאבי ואב כל / בני בין והשכל, / ואם לא יכילון / לפניו מרוצה.



ואיכה אנחם / ארי עז ולחם / בתוגות וישחם / בנפש חפצה.



הלריק מאד בן / תעודת חכמים / וידע משלים / והבין מליצה,



ומבור תבוניו / והוד רעיוניו / בני הזמן ש/אבו בין ועצה.



ונמהר יבונן / ויאמר לאונן / בעד יד נטויה / ועצה יעוצה,



הלא חי לחרץ / ונוצר לקרץ / ותבל לכלה / וגם נחרצה.



שקט, כי בכי לא / ישובב אבדה / ולא יחזיק לב / ונפש רעוצה.



ואיך נד למי כוס / פרידה בידו / זמן שם ושתה / שמריו ומצה,



ויאות היות לו / כנף נץ ויעוף / ראותך כבעל / אברים ונוצה.



אקוה לאלי / לקבץ נדודים / ואברת פרידה / לשומה קצוצה,



ואולי תהי ל/נפילה תקומה / וישים לנומת / ששונים יקיצה.
