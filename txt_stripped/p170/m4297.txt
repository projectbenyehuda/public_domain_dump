


	
	היתאפק אנוש כלו עלומיו
	




היתאפק אנוש כלו עלומיו / וכלו בנדד שניו וימיו



ודמו לענן־בקר משושיו / ואיך יתֽפשו הבלי חלמיו



ומעיו יהמו כים בזכרו / זמן עבר כברק עם־נעימיו



זמן היה ביד חפצו כחמר / ביד יוצר עדי רכבו מרומיו



וחיש נהפך בלי־פשע לאכזר / ופרש לחבל חבלי חרמיו



ואהל יֽדעיו החריב ואחיו / בלי שריד וכל־אנשי שלמיו



ולמד פאתי עולם קראו / ולשקות מדמי עיניו תלמיו



ומבור הנדד יצעק יחידי / ואין מקשיב ומעמקי תהמיו



ולולי להבי־אש בין צלעיו / מגוריו החריבו מי זרמיו



ועוד ידו נטויה על־לבבו / להרגיזו בפחדתו ואימיו



ומכתב מידי עזרא עזרו / הכי מצוף־דבש מתקו טעמיו



וחלקו מחמאת דברתיו / והם שכים בעין חמדיו וקמיו



צעיר שנים ולו טעם זקנים / ועול ימים ונקרא ראש בעמיו



כתב יעיד לישר מעלליו / ומהלליו ותפארת קדומיו



בתמכו נֽטפו־מר אצבעתי / ונשא אל־נחירי ראש בשמיו



כנפך על־פני שהם מצאתיו / וכתכלת עלי חורי גלומיו



כים גדול חשבתיהו וגליו / כנפת־צוף וקול כנור רעמיו



בשמים יגרשו מימיו ואדם / גבלתו ועל־חופו לשמיו



מעיל עז יעטו כל־מאמריו / וגם צנפו צניף הוד כל־נאמיו



ואריתי לאט מגדי אהביו / ורויתי ביין דדי כרמיו



בשיר הושת צמיד יד התבונה / ועל־אף הֽיקר כעדי נזמיו



לכה כוכב אשר יזהיר בארץ / והודו לו שני אורים בשמיו



לכה איעֽצך עצה ומה־טוב / ידיד יועץ בתם־לבב תמימיו



דרש מוסר בעודך לח ותצלח / ותרום על־מתי־שכל ורמיו



ועל־דלתי אמת תשקד ותקד / לכל־משכיל להתבונן עלמיו



וההדר תישר את־הדוריו / והמדע תפתח סתמיו



לימי הֽמזמות רד בכל־עת / וים אחר זנח לשחות במימיו



ואל־תלעג לאיש חובר חבריו / ואל־תבז בעת יקסם קסמיו



הכי יש לנבונים רעיונים / וכל־עם יחשק גנזי ברמיו



והסכל יחרחר ריב פתאים / והשכל יהפריד בין עצומיו



ויגע בילדותו יהי נח / בשחרותו ויחליץ את־עצמיו



ולא ירדה בזקן צוף תבוניו / עדי יאכל בנער את־רתמיו



ואל־תחמד מתי מדות ומדות / ואיך יחמד אנוש תבנית צלמיו



ומה־בצע באיש תאר בלי־בין / ולו אופיר יהי־לו עם־כתמיו



כלחם־בין יהי שבע לרעב / וסות־שכל יכסה מערמיו



ואל ישאך מדבר מהלל / ולו יקריב לך זבחי שלמיו



הכי כל־האנוש יחליק לשונו / ומצפניו שטר שקר חתומיו



ותור אחר דבר אנשי מליצה / וחידתם בכל־דור וחכמיו



והתרע אל־מתי אמן וצדק / ומה־נחמד שכן כל־איש לדומיו



והקטן עצמך לפני עצומים / הכי גאים בעיני אל זעומיו



ואיש משכיל יהי נשה חסדיו / וכל־רגע יהי זכר אשמיו



ושא שלום כמי־ים או מעתיו / וגוש עפר ומספר כל־יקומיו



ועל־האב אבל אב החסדים / כחיל־שחק וכל־נטפי גשמיו



יהי שלו ימי חייו ושלום / ואל ישים אלי־דרך פעמיו.
