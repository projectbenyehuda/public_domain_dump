


	
	בלבב בר
	



פזמון. חתימה: משה


בלבב בר / מאד נשבר / באתי העת בקומי,

יעצני / והנחני / להתודות על אשמי –

והנני / על עווני / מקריב חלבי ודמי

ומתפלל ומתודה חטאתי וחטאת עמי.





מי יבאר / ויתאר / עזוז מלכי וגאוניו?

וכל גבר / בפיו דובר / – ערל שפתים לפניו,

והמשכיל / לא יכיל / נשא גדלו ברעיוניו!

ואם אפר / יאו לספר / מקצת דרכי אדוניו –

ישוב הלום, / כי הוא חלום / והמות פתרוניו.

ועיניו ראו גלמי / ומנהו לא נכחד עצמי.





שוא יבקר / החוקר / סוד מהאנוש נעלם,

איך צופה / ברוח פה / ברא את העולם,

ואיך תבל / בפיו סובל, / ואת כל היקום כלם.

במפלא, אנוש, אל תדרש, / פן תשוב דך נכלם!

ושוב, לבי, / למשגבי / והסכן עמו ושלם,

טרם יבוא יומי / ואשובה אל מקומי.





הסר זעם, / ופעם / הרימה למשואות,

במי אהבה / מחה חובה / וקרע שטרי חטאות,

וחן עבדך / יום נגדך / יהי נשפט עם בריאות –

יום נברים / נכרים / לפני אדני צבאות.

ונפשים / בקדושים / יהיו למופת ולאות

יום קול יעבר בעמי: / ההולכים מי ומי?
