


	
	[פתאם יצרי אהבים עזבתי]
	




פתאם יצרי אהבים עזבתי / תעים באישון לֽילה פגעתי



חבלי שעפי נפרשו להם ופח / שנה עלי ערש דוי הצעתי



איך שוטטו מדבר ואיך התחפשו / באפר כסות ליל ולא־ידעתי



חלו פני ערשי בליל כלו עדי / נשף נדדים מחלי שבעתי



לנו בבין שדי עדי ארו פני / מזרח ופרע מאפל פרעתי



לא־נאחז לי מפניהם בלעדי / ריח צרור המר אשר קבעתי



לו אֽדעה כי־החלום יראה פני / דודי לרגלי החלום כרעתי



אך מעדו רגלי משושי בהדף / אתם ידי שחר והתרועעתי



וינפפו גשמי דמי עיני הכי / את־צור כבדי בבכי בקעתי



לולי יגונים סֽגרו אז בעדי / שחתי בצולת הבכי טבעתי



ממעמקי המצוקים אצעקה / חמס ומבטן נדד שועתי



לא־עֽרבה שנה לראותי למן / היום אשר ממערב נסעתי



קצתי בחיי מסבב תבל וגם / למד ממדיה ולא־בצעתי



רדת תהום ועלות עלי־ראש במתי / עבים עדי אמר לרום הרקעתי



עם־בהמות יער הליכותי ומ־ / ראש הררי־טרף כעוף הופעתי



קצוי אפסים כברקים רוצצו / רגלי וגם מים עדי־ים נעתי



מסע אהלך אחרי מסע ולא / תרתי מנוחה לי ולא־הרגעתי



יומם שזפני מאור שמש וליל / עיני בפוך המאפל קרעתי



ובכל־חליפות הזמן פני בלי / מסוה לנגד קר וחם הוקעתי



עד כי־למשחית נהפך הודי וגם / התנכרו פני ולא־נודעתי



עבר זמן קציר וכלו כל־ימי / קיץ ותם חרף ולא־נושעתי



ספו חדשים חֽלפו שנים ולא / באתי מחוז חפצי ולא־הגעתי



נשכח כמת מלב אני או אם־כלי / אבד ואם־אבן בים שקעתי



כמה אפתה את־לבבי באמר / יחל ואתו באליל הוגעתי



אשא בשני את־בשרי אֽמרה / האח ובפעמי בעז רקעתי



לא־חֽמלה נפשי עלי נפשי ולא / חסתי עלי הודי ולא פרעתי



על־מי אהי מלין ונפשי עוללה / צוקי ובימיני אני פצעתי



כי לא נבנותי במוסר יֽדעי / ועצת בני־אהב מאד פרעתי



כמה יעצני לבל־אבטח עלי / בגד בנאמניו ולא שמעתי



מי־זה ישיחם כי לתהו תם המון / כחי וכל־ימי לריק יגעתי



אנשי שלומי שלמו רעה מקום / טובה ושמחו־לי ביום צלעתי



איך נהפכו־לי אל־זמרי זר בני / שרק וזרע האמת נטעתי



איך העלו קצים שדי־חמד מקום / חטי ידידתי אשר זרעתי



האח עלי אחים אשר בגדו כמו / נחל ופשעו בי ולא־פשעתי



נעבד לימים נֽתנו אתי ואז / להם תנוך אזן זמן רצעתי



חסד גמלתים מבלי תקוה עדי / להם הטיבותי ולי הרעתי



ובעת זמן יצא למלחמתי ועד / ארץ ברע מעשי נגדעתי



רצתי ברגלי גאוה להם ועת / שרתי פניהם זעפים נכנעתי



ולאהלם סרתי נטוי גרון ויום / באי ברגל בֽצקה פשעתי



חלו ליום חולי עליהם כאשר / ממהתלות מדבריהם זעתי



פתו בשקריהם אמונתי ועין / דעי להביט פעלם השעתי



עד־נמכרו בידי עצת־כסל ולא / לבם הנידתי ולא־הנעתי



בגלל חמסי לֽבשו־בשת ובה / נוספה תהלתי ולא־נגרעתי



יונים תעופינה כעב אל־מערב / אתכם בחי האהבה השבעתי



לשאת שלומי אל־ידידים רחקו / אך בין צלעי אהלם תקעתי



ותספרו להם הכי לבי עלי / נודם כשסע הגדי שסעתי



ודרר אשר קן לה בצלעותי ובה / שמח כבוד נפשי והשתעשעתי



לולי חיותה את־חיותי לא אהי / שאל ובמות לבד הפגעתי



הנה לרגלה נלכדה רגלי בפח / גרות כתוא מכמר והתגלעתי



ולמענה באתי בכור עני ואת־ / נפשי בקלע הנדד קלעתי



אחי בכה עמי ואוי לגר ענה / כי עם־אדום גרתי ונתבלעתי



כה יעשה־לי אל ויוסיף מחרון / אפו עלי נגעי זמן נגעתי



אם־נשכחו מפי חסדיך ואם / מבית לבבי אהבה הקצעתי



לבנת לבושי משחור עיני עלי / נודך לארך הנדד צבעתי



כבלי זמן אסרו פעמי מעמד / אף כי־לחפצי מהלך נמנעתי



אוחיל ואולי אל יהי מחה במי / דמעי מצוקי ואשר הרשעתי



יחן ורוח חן יהי מעיר עלי / נפשי ואחיה אחרי גועתי.
