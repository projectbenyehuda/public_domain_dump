


	
	ברשפי הבכי נפשי שרופה
	




ברשפי הבכי נפשי שרופה / ובבכי הֽיקד לחיי שטופה



וצפרני נדד תפשו לבבי / והוא במתלעות זקן טרפה



וליל נדי בלי־שחר ולשנת / פרידתי בעולם אין תקופה



והרחיק הזמן מני קרובי / ובניו אספו עדי אספה



וכמה שֽלפו עלי חרבות / ובימינו אני חרב שלופה



ובי יצנף להתפאר וכדור / צנפני לקצוי־בור צנפה



נתנני כציץ נבל ביער / ובין חוחים כשושנה קטופה



והמצני ביד־אפל וכוכבי / שחקים זֽרחים המה בעיפה



זמם שטים לבד הפיק ותאות / לבבי שם ביד־איבה הדופה



ויום טרפי יהי מכין וטרפם / לעול יאחז איפה ואיפה



לחרפה הֽיתה דעתי בארץ / תבוסתי וחכמתי גדופה



ולנגוע מאד נפשי תמאן / ותיקר מֽהיות נגדו נגופה



ותתגבר למלחמת ילדיו / בחזק יד ואזרוע חשופה



ותבליג על־סבל צוקיו ולשאת / פרידת הגביר יצחק יעפה



לכנת תם עלי יבלי חסדים / ופריה בכל־עת וענפה



פרי חכמה תבכר לחדשים / לפי היקום ועלה תרופה



ומגדל תם עלי רצפת תעודה / באבני עז ותפארת רצופה



בני־יום לא יניעוהו ותבור / ברוח פיו הלא מניף תנופה



בצור עט יערך לחם ולשון / לרדף דֽברי צחות חרופה



תהלותיו בלב חמדיו כחצים / מזמתו בעין קנאה שזופה



וכמה ישחקו אליו בפיהם / ואוד מצל לבבם משרפה



ודמו לחלש ימין חזקה / ביד־כסל מאד דלה ורפה



ואיכה יהדף כבש לאריה / ופרס איך יהי טרף אנפה



והחול איך יהי עלה במאזני / אמת עם־קש אשר יעוף כסופה



זרעיו יעשו מגדי מליצה / ושבלת תנובתם שדופה



היתגרו באיש תתאו לבנה / לשרתהו כמו שפחה חרופה



ושמש תעדה גאון וגבה / היותה לו כרקחת ואפה



ידידי נכספה נפשי לחבר / לשון עבר ופי עפר ועיפה



ראות מלים במצרף לב בחונים / ואמרת הוד בכור־דעת צרופה



והתלונן בצל כנת בשמים / עלי־כל־עץ בגן־האל זקופה



סעפיה בלב־ערץ ולארות / בשמיה עדי־ארץ כפופה



הכי נעתי בעי־כסל ונפשי / ברעב אל־תנובתה עטופה



ותעיתי בתלאבות פתיות / והיא אל־קר רסיסיה עיפה



והא אחי לך בת־רעיונים / מאירה ממאור סהר ויפה



אחותך או אחות שמש ואם־היא / בלי־יד מבנות־עיש חטופה



ולולי הֽודך לבשה ערמה / תהי מהוד ומהחן יחפה



תשיחך בעד אהבה בלבי / מזקקה בלי־פשע וחפה



אהבים נֽוספו אכן נדבה / ואין למו ימי חיי חליפה



ותתחנן להזיל מי כתבו / עלי נפש באור פרוד שרופה



אשר עינה כעין גנה צמאה / מיחלת ליד העב וצפה.
