


	
	תנו ציץ לשירות
	



שיר הקדשה לאברהם אבן מהאג׳ר בראש ספר הענק.


תנו ציץ לשירות / ומלות נכוחות / לאחז בתבל / קצוות ורוחות,



ללמד לכל נמ/הרי לב תבונות, / ועמקי שפתות / לדבר בצחות,



ולשור נגידי / אמרים, במעבה / לבבי יצקתים, / צרפתים בטוחות,



היותם כאזן / לשכל, ועין / לדעה, ואל גוף / תעודות כרוחות.



יפיחון כקנה / לאפי מלכים / ויהיו מלכות / ברקחם רקוחות,



והם שעשועים / לכל שאננים / וצדה לדרך / לעוברי ארחות,



ואזן לחרשים / גלויה לשמעם / ורואות לעורים / לשורם פקוחות



ונפש לנפש / מבינים, וקרב / כסילים – כמו אש / קרב ואנחות,



וחלקו מאד מ/חמאות ורכו / כשמן – והמה / לחומדים פתיחות.



נתתים כמכתב / אלהים לאב כל / המונים וגויים / לנשאם כלוחות,



ושמתים ככתר / עלי ראש חסדיו, / ירימו עלי כל / סגנים ופחות,



ועל גרגרות מ/עלתו – ענק חן, / וטוריו מלאים / באבני שבחות.



גביר יהלך על / אדמה ועל ראש / כסיל נתכנו לו כחפצו מנוחות,



ירוצץ כברק / עשה תם וצדק – / בעת מהלכי כל / גבירים טפחות,



ומשרה בנויה / באבני תהלות – / ובלתה בתפל / בנויות וטוחות.



וטובות בידיו, / ודלתות חסדיו / לכל גר ואזרח / ימותיו פתוחות,



ומציל למטים / להרג בהונו / ומשגיב נפשות / למות לקוחות,



נעימות לשואל / במתן ימינו / ולו גיל בפניו / ושבע שמחות.



קחה לך תשורה / בדלח אמרים – / ומה טוב מנחות / כמו זאת שלוחות,



וכנות אהבים, / יסופון ארזים / בגן אל – והן ר/עננות ולחות.



והא לך גברת / לשירות ילדון / בני שיר – וכלן / לנגדה שפחות.



אמת, את ושמש / תאומים בעולם, / ולכם תהי זאת / לבדה לאחות.
