


	
	אט על אנוש
	



על מות רפיא בן בזאז


כתב נחם שלחו אל אהוביו / אשר יראם בעין לבו סביביו,

בנו הרב וילדי אח ואחות / ותלמידיו ואל יתר קרוביו.



אט על אנוש חל מגורו / פתע וצר בו מגורו,

אין לו רפואת תעלה, / אין דן למזור מזורו,

חליו מאד רע ונאמן, / אכן צרי אין לצירו,

בינות יענים יאוה / לשום ימותיו מדורו.

אסר זמן רע להכאיב / לבו וקם על אסרו,

אותו בהנחיל יגונים / הכיר ושמו בכורו.

אמרו ״אלהים עזבו״ / רעיו ״וצורו מכרו״,

ויבגדו בו, ושפע / ימים בראשו עזרו,

קרבו כתפת, ונחל / מימי דמעיו גברו,

מוצא אשר לא יכזב, / כי מדמי לב מקורו,

[א]ו מאגמים בנוף יום / נהפך לדם כל יארו,

דמע יפוצץ סלעים, / ישחק אבנים באורו.

עמקו מאד רעיוני / בזמן ושכלי חקרו,

מיום מעט מחזות את / הפכו כרגע יעירו,

בוגד בבעל בריתו / שודד להון אם צברו,

אם יעמד איש עלי קו / ישר להמעיד אשרו,

יחביר להפריד ועשר / יתן בריש להמירו,

ירנין בנשף וערב / יחליף במספד זמירו,

כספו כרשת למעגל / יפרש ומכמר סגורו.

יכסל אנוש שם בהונו / כסלו ובטח בעשרו,

לריק יגיעו וכחו / ילוד לעמל והרו.

מי יאמין בזמן או / לבו יפתה בשקרו

יזכר תמותת רפיא / הרב ויודה לצורו,

יודה ויאמר: הכי כל / נוצר לכלה יצרו,

איך הזמן קם ונצל / עדיו וחלל [ל]נזרו!

איכה להר האלהים / יכל ואיכה עקרו!

קם לי עלי איש ברוח / פיו מהיותו גערו.

עולם למותו כגבר / נדהם ויין עברו,

ויעמד חוג שחקים / מרוץ ופחד עצרו,

אספו כסילים למותו / נגהם ושמש מאורו,

מסוה אפלה עלי אור / סהר וקדרות אפרו,

וילבשו שק בנות עש / וכסיל חבלים חגורו,

ולים תהלות מאד זע / הים ושב לאחורו.

נעו גבעות וסיני / חרד עלי מות חברו,

כל החסדים יתנו / כי נבראו בעבורו,

היכל פנימי ייליל / כי סף כבודו ונרו,

נר מערבי, וילכו / שוכנים כקדר לאורו,

שוכן פאת ים וצפון / בשמו ירנן וזכרו.

האמיר אלהים וכן על / כל היקום האמירו,

בחר בדתו ובזאת / מכל בני תם בחרו.

חכם, ולא קם לפניו / חכם בתמו וישרו,

כל חף בתבל בצדקו / יצנף ויכתיר בכתרו.

משלה ענוה בלבו / כמשל פעליו ביצרו.

עצרו חכמים במלין / נגדו והודו לאמרו,

שמו ידיהם לפיהם / יחד עדי בא דברו.

עושו בכו נא עלי אות / עולם ומופת בדורו,

ונהו לשופט בצדק / בין איש לאחיו וגרו,

עוטה אמונה וחסד / כמעיל וצדק אזורו,

אוצר תעודה, ואכלו / חנם לרעב ושברו.

משיב לשואל כחפץ / לבו ודי מחסורו,

משקה צמאים למי בין / מאין דלי מי בארו,

גן התבונות ופריו / שכלו ומדע עבורו,

שמט לכל בא רדות את / דבשו ואכול ליערו,

ובלי מחיר יארה את / בשם תבוניו ומרו.

רועה מנהל לצאנו / על מי מנוחות לדברו

לא החרידו ארי כל / ימיו ואיש לא שערו.

נטר לכרם אלהיו / וכעין ימינו נצרו,

סקל נתיביו וכלה / קוציו ואת־כל־שמירו,

הן אחריו היה לבער / חרץ ופרץ גדרו.

נושאי ארונו בכתף / אט בו, וסובלים לפגרו,

כי את־ארון האלהים / תשאו ולוחות דבירו,

לא שערו כי שרפים / צבאו ללוות הדרו,

נפשו להיכל קדושו / נשאו ואל כס יקרו.

ידו תחופף עלי גוף / טהור וירגע בשרו,

כי אין יכלת בעפר / בלות יצוריו וצורו.

בלשון כתבי ילדיו / אקרא ואל כל שארו:

הנחמו, כי פאר כל / משכיל ודעו בשברו

חיי בני איש בעולם / כחלום ומות כשרו,

ינהג יקומים עלי פי / שחת כרועה לעדרו,

מות כמגל, ונחנו / שדה ובשל קצירו.

אלו זמן רע יבקש / כפר ופדיון גבירו

שתנו לבבות ובבות / תחתיו שמחים בכפרו.

אחלי ידידי, ויתן / לי עוף כנפו ואברו

לעוף לארץ אשר בה / נחצב מכונו וקברו,

קבר ינוצץ לתועה / דרך באפל לאורו,

לקראת משחר תשלח / ריח בשמו וזכרו,

ריח יחיה נשמות, / כי מר־דרור הוא עפרו.

עמדו עלי עמדכם כל / רעיו להשיר צעירו,

יקום למלאת מקומו / מהר ולחבש פארו,

יפרה, למען כשרש / עץ גן אלהים אמירו,

ישאג בדת אל, וכארי / חכמה יהי קול כפירו.

יגאה ועד ים תבונה / לעד ישלח קצירו.

עד יאמרו מושלים, כי / בן־האמונה כהורו,

הוא מאשר רם ונכבד / תאב ונכסף לצורו,

ישכן לפניו בחלק / אהרן ומשה בחירו,

קברו ירוה בכל עת / מלקוש מרומו ויורו,

ילכו לפניו פעליו / לאור ואתו שכרו.
