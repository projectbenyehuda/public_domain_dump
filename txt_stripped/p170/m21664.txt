


	
	בשם אל אשר אמר
	




בשם אל אשר אמר / והעז לאמרותיו,



וצוה בלי מורה / ומורה למצותיו,



ופעל ועשה כל / אשר אותה נפשו,



והחל בלי עזר / וכלה פעלותיו.



אהללה לממשלתו / ואודה למהללו,



ואקד לכס גדלו / לזכרו ולשמותיו;



ומי זה אשר יצרו / מאחז יקר צורו,



ומי זה אשר יעמד / בסודו וסתרותיו;



ואיך מה יקו לספור / וספר גאון החי,



וחסרון הימנה / להות יתרותיו,



לראשון בלי ראשית / ואחרון בלי אחרית,



ואיך לו יהי תכלית / ואפסו תהלותיו;



ואחד, ואך לא הוא / כאחד אשר נתן



יסוד האנוש לחשב / ולכלול ספורותיו;



ואחד, ולא ירבה / וימעט ויחלק



כאחד אשר הוא ראש / אלפיו ורבבותיו;



אבל הוא אשר נודע / בראשית עלילותיו,



בכחו וחכמתו / ועז תעצומותיו;



ורחוק והוא קרוב / וקרוב והוא רחוק,



ונמצא בכל מקום / ושגבו מקומותיו;



ונסתר והוא ראה / ונעלם ויראה



בעיני גבורותיו / ועל יד פליאותיו;



ואיך לא אהי צופה / כבודו בעין לבי,



ודעה אשר שם בי / ונפש מעידותיו;



ונעלה בלי מעון / ונשא בלי מכון,



אבל הוא מקום חלדו / וסובל יצירותיו;



ובבלי כרוב יעוף / ועבים לסתר לו,



יהלך עלי כנפי / ענניו ורוחותיו;



ועשה משרתיו אש / ולהט, ומלאכיו



ורוחות, ואף קרה / במים עליותיו;



ולילה כיום יאיר / לפניו, ואם נתן



ערפל חתלתו / וחשך סביבותיו;



ומביא בלב נוצר / כבודו ואם נבצר



וארץ ושמים / מלאים תהלותיו;



והוא כל ובו הכל / ונמצא בקרב כל,



ואם גבהו מכל / דרכיו ומדותיו;



וכרחוק פאת מזרח / ליד מערב, רחקו



מאד מחשבות איש / מעזוז עשתונותיו;



הכי נמצא בזמן / ומקום, והוא מאין



זמנים ומשכנות / מציאות מזמותיו;



ועלה במחשבה / תחלה בראש חמר



וצורה והחפץ / ואליו תשוקותיו;



והוא ראש לכל חמר / ושרש לכל צורה,



ואין לחמריו קץ / ומספר לצורותיו;



וזמם עשות היש / בלימה ונתכנו



עלילות למחשבתו / וסבות לעצותיו,



ונטה כדק שחק / לספר בלי אמר



כבודו ומלכותו / וחות עלילותיו;



מרצע כמו עבד / להשיג רצון קונו,



ואליו להתרצות / בפחדו ואימותיו;



מתחו כמו אהל / אבל קם ברוח פיו,



והביא בלי יד את / קרסיו בללאתיו;



ועש עם כסיל לפנות / בצפון ובדרום



קצותיו, ועל גבם / הליכיו ועקבותיו.



ושבעה רקיעים / הובאו זה בזה להיות



מגורות לכוכבים / ולשני מאורותיו;



בליל שבתאי גם חול / לצדק וחוג מאדים



ושמש, ואת אפלם / יכסה באורותיו;



ונגה וחוג כוכב / וסהר אשר יזהיר



ברחקו, ואם יקרב / ילבש באפלותיו;



גלילים וכוכביהם / ירוצון, ובשמיני,



חקקים באין מרוץ / גדודי חילותיו;



ועלים במתכנתם / ורצים במשמרתם



עלי קו והעגול / נכונות מסבותיו;



והמה במסמרים / נטועים כנגה,



ירוצון לארך על / שבילי תהלותיו;



והוא בגבורתו / מסובב במזלות,



לכלות חדשותיו / וחדש ישנותיו;



והוא אל גדל כח / אשר הוא לכל סובל



ובו נחשלו זמות / אחי בין וחכמותיו;



והוא אל אשר יסב / גליליו ביום פעם



למערב ולמזרח / ועליו תשובותיו;



ויחד שרפיו הם / ירוצון ויקדישו,



ואם לא יבינון את / מקומו ומושבותיו;



יצרם כספירים / מאירים, ולא זכו



בעיני שכינתו / וזהר מגורותיו;



ואז נתכנו אישים / וצורות בעולמו,



בעצם מרוצתו / וכח תנועותיו.



ושמש במזלות / והמה שני עשר



יסובב ימי שנה / לכונן תקופותיו;



והם קר וחם, וזמן / לקיץ וחרף לא



יסופון וישתנה / זמן בחליפותיו;



ושש יורדים מהם / ושש יעלה בהם,



ימי עד לקומם / חוץ ימותיו ולילותיו;



שלשים בכל מזל / להשלים שלש מאות



וששים, יהלך על / חלקיו ומעלותיו;



וישאף ויזרח אל / מקומו בראש טלה,



ויסב עלי בתיו / כמלך בטירותיו;



ויבא בצל ארץ / ואז יחשך אורו



בצלו, למען אין / חשכים תמורותיו;



ובינם כחתן / מחדרו יהי יוצא



להאיר עלי טבור / וירוץ בארחותיו;



וסהר בכל מזל / ביומו יהי חנה,



להשלים ימי חדש / בחדשו ולשנותיו;



ויתכס ויתגלה / וישח ויתעלה,



ויחסר וימלא / להאיר כחקותיו;



ויגביר בהמלאו / גאון ים ושפעותיו,



וגרש ירחים גם / להבשיל תנובותיו;



וכימה תקשר / מעדנות ותקשיח,



ונגדה יפתח / כסילה ומושכותיו;



כסילים למזלות / נשמות, והם גופות,



להראות זרוע אל / בעולם ואותותיו;



להחיות להמית גם / להשפיל וגם רומם,



לרושש ולעשיר כל / ימותיו מרוצותיו;



ולבנות וגם להרוס / ולעקר ולנטוע,



ומגיד לכל מלך / יקרו, וכסאותיו;



ועז התלי לעזור / ולכשיל דמות תנין



זנבו תבוסתו, / והעז מראשתיו;



ועשה רקיע / מבדיל הוא לבין מים,



ולהיות מקום קרוב / להשקות לאדמותיו;



וראשית לכל נברא /יסודות אשר ברא,



ומה נפלאה חכמה / בארבע יסודותיו;



והם אש והרוח / ומים והעפר,



בהתחברם עולם / אזי בם ותלדותיו;



ויסד בכל אחד / שתים במפעליו



ליובשו ואת חמו / וקרו ולחותיו;



הכי הוא אשר יצמיד / משנאים והוא יפריד



אהובים, ואין בלתו / מחבר פזורותיו;



וכבד ונקל בם / ופעל ונפעל בם,



ועלה ויורד בם / רצון אל ודתותיו.



והם נקראו אמות / הכי ילדו בנות,



אשר חק אלה בם / המון היקום ומתיו



וארץ תלויה על / בלימה באמרתו,



קשורה לבל תזח / בחבלי אמונותיו



ככדור, והרוחות / מפיחות והן ארבע



ונושבות בכל רגע / בעולם לרוחותיו



בצדם, וארבעה / ועשרים אלפים מיל



אמת מהלך עולם / אפסיו ומדותיו;



ונגזר עלי כפות / ספירות והם תשעה,



ובהם שבע עשרה / אלפים מדינותיו.



ובהם חי אשר יצעד / וישחה ויתעופף,



וחפה בבין ארץ / ותוך מהמרותיו;



בזהב וגם כסף / ונחשת ועפרת,



ובדיל ובברזל / ובשאר יצוקותיו;



ועוד התכו בשאול / בארץ המון מיני



פנינים וכל אבן / יצוקה משוכותיו.



וזרע אשר נזרע / וכל עץ אשר נטע



וכל דשני ארץ / ושיחיו ועשבותיו;



ולבן בעיניהם / ושחור בציציהם,



וירוק במראיהם / ואדם בצמחותיו;



ומתוק מר מהם / וחמוץ בעליהם;



ותפל ומלוח / בטעם לאורותיו,



ומהם כמו בשם / ובם עוברים רושם



ומבאיש פני רבוא / חלליו ורבותיו;



וחציר ועשב / לבהמה ולחיה,



ומהם למותת חי / ובהם רפואותיו;



והשוה אדמה ל־ / עבדה וצוה על



ערבה בעולם / ועמקיו והרותיו;



ועפר בשליש כל / ומים ביד מדד,



ורומש בגוש ארץ / ושרש בטירותיו;



ואמר לים עד פה / תהי בו ולא תוסיף,



והתגעשו מימיו / והחוץ גבלותיו;



והעלה נשיאים מ־ /קצות הארצות גם



ברקים למטר שם / ורוח באוצרתיו;



ובקע עינות גם / תהמות להשקות כל,



וחיתו שדי יער / בעריו וחצותיו;



והכל בשם עולם / קראו, אבל גדול



לארכו וגם רחבו / גבהיו ותחתיתיו;



וזאת יצאה מאל / נדבה בלי צרך,



אבל כי להודיע / לאדם גבורותיו;



ולחיות לאות לעד / הכי מעשיו המה,



והוא בלבד קדמון / וכלם חדשותיו;



ויום בראת אדם / בצלמו ובדמותו



במעשיו להמשילו / ומיני בריאותיו,



ועשה כנגדו מ־ / צלעיו לעזר לו,



להשאיר כתבניתו / ודמיון הויותיו,



כחלב להתיכו / בשליא להקפיאו,



ובשר להלבישו / בגידיו ועצמותיו;



ואחור וקדם צר / ותחת ורם יצר,



להימין להשמיל חק / ואלה תכונותיו;



וגופו בעת עצם / וחזק יתרותיו;



ויקרם עליהם עור / ובשר בשלמותיו;



בבטן יצרהו / ודם אם להיניקו,



וטובו ענקו / באפלו וחנותיו;



ואל קץ זמן יצא / לעולם להבטיחו,



לאט על שדי אמו / והחק לאמותיו;



ובאר בהרגשות / חמשה תבונותיו,



אשר הן עדי גופו / וחלית חמדותיו;



יצרו יפה מראה / ותאר והכין את



צלעיו ומפרקתו / וערך שדרותיו;



ועיניו להשגיח / ופיהו להצריח,



ואפיו להריח / בנחרת נטישותיו;



ולשוך אשר הוטל / והועד, וכעיט



יצפצף לבאר / מחקרי לב וחידותיו;



ויליץ בעד סודו / לשואל, ורעיוניו



יבאר ויודיע / סעפי עלומותיו;



וידיו להניע / ורגליו להסיע,



ואזן להגיע / ללבו שמועותיו;



בגופו להמישו / בכפיו לאחזהו,



בשוקיו להציעו / וישר הליכותיו;



ויצר בראשו / מזכרת ומשמרת,



ונתן לשכוי בין / וחכמה בטחותיו;



בקרב אנוש הושם / כמלך עלי כסאו,



וכל האברים ל־ / עבדיו ואמהתיו;



וכיס מר עלי ימין / ובו ראש לבל יצוק,



ובשמאל לחם טחול / באורו ומוקדתיו



לקרר, להרבות את / ימותיו במפעליו



ויאמץ בהתיעץ / במתניו וכליותיו;



וקבה להחיותו / וכבד להפרותו,



וקדקד להורותו / סעפי תבונותיו;



לדם יהפך כבד / אכליו ושל פגרו



ישלח ארחותיו / בכל עת ומשאותיו;



ועל לב, כנף ריאה / יחופף לרוחותיו,



להוציא עשן אפו / והביא לקורותיו;



ורוח עלי גרון / שבילו, לתתו קול



וחתך בחך ובשן / ולשון בשפתותיו;



נתיבות שתים גם / שלשים פלאותיו



בחכמה ביה חקק / למבין עלומותיו;



לספר ולספר / ולספור, ואותיות



שתים ועשרים הם / ועשר ספירותיו;



והשלים מלאכת כל / נקביו וגם תפיו



מסלות למי רגליו / ודרך לחלאותיו;



ומהם נתיבות חק / לכל תוצאות הגוף



ברירו ודמעותיו / ורקו וניעותיו;



מעט מאלהים ח־ / סרהו, הכי הם לעד



בחיים, והוא עפר / וישוב לעפרותיו;



וכן עטרהו הוד / במדע אשר נמשך,



אבל ממקור צורו / ומעין ישועותיו,



מדבר באין לשון / ומביט בלי אישון,



וחושב באין רעיון / וצופה עתידותיו;



מהלך באין רגל / מרומות ותחתיות,



וצופה בלי עין / לעולם וכנפותיו;



ונפש טהורה / נגזרה מכבודהו,



ותחסר מעט מבא / לקצוי תהלותיו;



ותשאף להתימר / במדע אשר קדם,



וכבוד אשר נגזר / לכל אדם בדרתיו;



הכי היא בגוף נבל / אסורה בלי כבל,



וחיה ולא תבל / בשחק וצבאותיו;



וזה הוא בשם עולם / מכנה אבל קנין



ומי הוא אשר בא עד / תכונת יצירותיו;



וכלה אלהים כל / מלאכתו ואז ראה



הכי טוב מאד הכל / ויפה בעתותיו;



ואדם אשר ילד / לעמל, כפי יכלתו



יחוה גאון צורו / ויטיף גדלותיו;



ונאחז בפח תבל / ונשקע ברב הבל,



ואכן פליאותיו / להלל מעידותיו



ואם בין צלעיו צר / לנפשו מאד יצר,



ימינו בטוב תקצר / ותארך ברעותיו;



ויפרש כצפעוני / ויקשר ולא יניא,



וישך ואם ישק / פצעים נשיקותיו;



בתאנה יסיתהו, / ולבכות, יחדהו



בעולם, יפתהו / בחמדת שכיותיו;



וישיא אמונתו / וייגע לפתותו,



בהונו וגם בשלל / צבעו ורקמותיו;



אמת גדלו מעשי / אדני וקניניו,



אשר נעשו כלם / בצדקו וחכמותיו;



ואלה מעט מזער / גאונו ומלכותו,



ואלה קצות דרכו / ופלאי מסלותיו;



אנוש מת אלהי חי / יזמר בעוד נפח



באפיו, ומה אדם / ומה הם זמרותיו;



ויעיר לבב שכלו / להמעיט קצת חבלו,



וימעיט ימי הבלו / כליליו תמונותיו;



ויודו לשם, אמרו / בפיהו להגבירו,



להשכיל לשון קדש / וחזיון נבואותיו;



ומדבר יהודית / המדבר עלי אפניו,



ועוד מאמר מערב / ומעניו וצחותיו,



וצעדו במועצה / בחכמה ובמליצה,



ומוסר לשון יון / משליו וחידותיו;



עדי יבשו מימי / נעוריו, והזקינו



קצירו עצי גילו / ושרשו שלחותיו;



ועבר זמן ילדות / כברק והשחרות,



ונדמו ימי עדנה / להבל חלמתיו;



ומשו לפי שעה / וחשו ימי רעה,



להפרות מצוקותיו / להספות אנחותיו;



והפך זמן הודו / למשחית, ועז פניו



ישגא, וכבה נר / סעפיו ורואותיו;



וילך כפוף קומה / כאגמון, וצדו און



צעדיו, ופעמיו / מדרוך את מסלותיו;



ושחו בנות שירו / וגדל כים שברו



ומתנודדים תחת / זמריו ושירותיו;



וקול יעבר נחוץ / אבל אל חצר מות



מגורי מידעיו / והוריו והורותיו;



ונושה יבקש / הנפשים ופקדונו



יבקש, ומי יוכל / תפוס את חבולותיו;



ומדביר ינדהו / למדבר יריצהו,



וחמדות מעונתיו / בזדון מקיאותיו;



הכי סוף אנוש לשוב / לעפרו ובא מועד



לכל חי, ובשחת / יכונן מלונתיו;



ישובון יסודותן / כמראש לקדמותן,



ורוחו להיכלי אל / וצלו נעימותיו;



ונודע בעולם / הפגרים, אשר זרע



ויאסף בעולם / הנפשים תבואותיו;



ויובא במשפט על / גלויו ונעלמיו,



ושמה ישלם לו / אלהים פעלותיו.



אלהי אחזתני / רעדה בהזכירי



פשעי וחם לבי / ועצמו מררתיו;



הכי מימי נער / לשלם חטאים סר



והזקין עדי לאה / נשא את אלמותיו;



והזיד בחטאתו / ושגה בצדקתו,



והמעיט תשובתו / והרבה משובתיו;



והעז כצור מצחו / והקשה כצור מוחו,



עדי סף זמן כחו / וסר צל בחורתיו;



ועפה שנות חייו / וחפצו ומאוייו,



ורבו כחול הים / חלייו בחטאתיו;



ועשש כעש עצמו / וסר מאנוש טעמו,



ופרץ זמן כרמו / ונשחתו זמירותיו;



ושלח גאון ידו / בגופו ובכבודו,



וקדר מאור פניו / והלבין קוצותיו;



והמר במר רוחו / עדי מתקו נכחו,



לפי טעמים שרש / רתמיו ולענותיו;



ואם דם בפיו יוריד / ומטר נדוד יריד



ויסתם בעב צדקו / ויסיר תפלותיו;



ואיך זד, פני יוצרו / יחלה ויתחנן,



מטנף בדם סכלו / ובדמי זדונתיו.



ואדון היתחקה / עלי צעדי חוטא,



אשר לעשות הסכיל / והרבה תנואותיו;



ויפקד עלי רמה / וידרש לתולעה,



והוא יעשה בשתו / וילבש כלמותיו;



ברחמיו יהי פנה / לעבדו ואל יבזה



ענותו בגרותו / וצוקיו וקורותיו;



ונודו יהי סופר / וישים בנאדו את



דמעיו, ויעבר על / המוני עונותיו;



ואל נא ידינהו / במעלו כמפעליו,



וימחה זדונותיו / במימי סליחותיו;



וחלקו ישו עם / החפצים ביראתו,



ונוצרי תעודותיו / והלכים בתורתיו;



ויזכה ביום יקרא / לעמו פקח-קוח,



ויקבץ נפוצותיו / ויום החיות מתיו.
