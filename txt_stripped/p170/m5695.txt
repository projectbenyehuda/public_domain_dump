


	
	[מארץ שפלנו ברבי עונינו]
	



כמנהג קהלת ארגיל(אלג’יר)




מארץ שפלנו  /  ברבי עונינו

שחנו מהרים  /  אליך פנינו

הרענו ואיך נוחיל /  לראות בעינינו

בכל אפסינו –       את ישועת אלהינו: [ישע' נב, י]

אלהינו, איך  /  עפר ואפר

אדר יקרך   /   יבאר על ספר

אנוש כחציר  /   יציץ ולא יפר

יספר –  /        את מעשה אלהינו:    [ירמ' נא, י]

אלהינו, בך  /   אם פשענו

ביון מצלות  /   מעשינו טבענו –

באנו להתודות /  כי לא נכנענו

ולא שמענו –  /      בקול אדני אלהינו:    [דנ' ט, י]

אלהינו, גוי /    בך יתפאר

גוו במוסר  /   אכזרי מתאר

גויים כחשו  /   בגדלך המפאר

ונשאר –   /       גם הוא לאליהנו:     [זכר' ט, ז]

אלהינו, דין   /  עני ונכלם

דון, והרף   /   כי נגדך נאלם

דברו מארץ  /   שח ונעלם

ויקום לעולם –  /     דבר אלהינו:       [ישע' מ, ח]

אלהינו, הקרב נא / את רנני

הפלא חסדיך  /  לעין כל ועיני

השמיע מיחלים /  יראו ממעוני

כבוד אדני – /       הדר אלהינו:    [ישע' לה, ב]

אלהינו, ובורא  /  כל נשמה

ותיקים אמץ  /  לובשי חיל ואימה

ולאט כי יגרנו  / מפני האף והחמה

ונצומה –   /       ונבקשה מאלהינו:  [ישע' לה, ב]

אלהינו, זה  /   עני עזוב ועצור

זדוניו הציקוהו /  להטמן בצור

זכרך לא שכח  /  כי בלבו נצור

ומי צור –   /      זולתי אלהינו:      [תה' יח, לב]

אלהינו, חנון  /  ורחום ומקשיב

חרפה תאסף  /   וחרבה תושיב

חסדך תשלח /   רוח חן משיב

להשיב – /        חרון אף אלהינו:     [ע.' י, יד]

אלהינו, טוב זכרך / אל חכי

טעמתי מצוף  /  דבשי ונסכי

טעם זמירך   /  נר באשון חשכי

כי –    /        טוב זמרה אלהינו:    [תה' קמז, א]

אלהינו, יסרתי  /  לב קשר

ידי מעגליו  /   בדתך לאשר

יבש כי מאס  /  שום שכל וכשר

וכל אשר –  /      יאמר אלהינו:      [דב’ה, כד]

אלהינו, כואבים / ועניים ועצבים

כהיום בעינינו  /  נחשבנו כחגבים

כלנו אומרים  /   האם לנבזי עצבים

אלהים קרבים  /     כאדני אלהינו:      [שם ד, ז]

אלהינו, לקחנו  /  עמנו תחנות

לבוא עדיך   /  עם ברים להמנות

להודות קמנו  /  וזדוננו לענות

ולהתענות –  /      לפני אלהינו:       [עז' ח, כא]

אלהינו, מדברי /  דת זנחנו

מדרך סרנו   /  ותורתך שכחנו

מודים באנו  /  ומבשת נאנחנו

אנחנו –    /      מעלנו באדני אלהינו:   [שם י, ב]

אלהינו    /   נבוכים מחובם

נסה עליהם  /   אור סליחה סביבם

נעוו משמע  /   ולא אמרו בלבבם

נירא נא   /       את אדני אלהינו:     [ירמ' ה, כד]

אלהינו, שריד חרב / ולבבו יחיל

שק יציע   /   תחת פתיגיל

ספו בהתנצל  /   עדי טבעת ועגיל

גיל –    /       שמחה מבית אלהינו:   [יוא' א, טז]

אלהינו, עוינו  /  ופשענו, אכן

עותנו דרכנו  /   ודברנו לא כן

עיני עבד   /   ליד אדון ומסכן

כן –     /       עינינו אל אדני אלהינו:   [תה' קכג, ב]

אלהינו, פשענו בך / עוינו ומרינו

פעלנו און   /   ועון לא כסינו

פקודי עדותיך  /  זנחנו ושגינו

ולא חלינו –  /      פני אדני אלהינו:     [דנ' ט, יג]

אלהינו, צבא /   חובי ועוני

צוה להפאות  /  מנגד עיני

צופני מצוקות /  יחדשו את רנני

בעיר אדני –  /      צבאות בעיר אלהינו:    [תה' מח,]

אלהינו, קרא דרור / לנו מחובים

קצרי רוח   /  מערב נצבים

קרבן אמריהם  /  אשר הם מקריבים

קרבים –   /      יהו אל אדני אלהינו:   [מל"א ח,]

אלהינו, רחמיך  / מאד יתאוו

רוהים, ישברו  /  אליך ויקוו

רוחשים לכל יבואון / לביתך וילוו

השתחוו –  /       רוממו אדני אלהינו:    [תה' צט,]

אלהינו, שקר נחלו / אך נכאים

שמענו אחרינו  / התולי חטאים

שופט, הלא תחקר / לבות נבראים

אם –     /      שכחנו שם אלהינו:    [שם מד, כא]

אלהינו, תן עז  /  לבן נוקש הודש

תשועת כפרה /   עליו תתחדש

תמימי סגלתך  /  ישוררו במקדש

שיר חדש   /      תהלה לאלהינו:      [שם מ, ד]

אלהינו, חובינו /  כדונג ידאבו

זמירינו לך לרצון / יעלו וייטבו

קדושי עליון  /   יתבשרו כי נקשבו

הבו גדל לאלהינו:              [דב' לנ, ג]
