


	
	מרומם אל בישראל
	



סימן: מש[ה]


מרומם אל /  בישראל! –  /   בצדקה שמו נקדש



ויתעלה  /  בעם אלה,   /  שנת רצון כי תחדש, –



שנת חיים  / שנת פדיום   /   לבנין בית המקדש



ישתבח  /  ויתפאר   /    ויתגדל ויתקדש!


  


מלך באימה / תמים חכמה,   / והכניע כל מולך



ולגבולו  /  בבית זבולו  /   השב עם דל וחלך



וברננים  /  עלי מנים   /   אל בית אדני נלך



בשופרות /  ובחצוצרות   /  נריע לפני המלך



בקול המון /  בתוך ארמון  /   ונכד אדמון ידש:


  


שלח ישע  / לעם נושע  /   בעת תקרא שנת רצון



ואז אשמח /  בבוא צמח   /  לשמח שאר הצאן



ונגאלים /   יהו עולים   /  לבית פנימי והחיצון



ובמועדים /  יהו חדים    /  לעת הקץ והקיצון



והחדש  /  בהתחדש  /    לקרא: מקדש מקדש



ישתבח  /  ויתפאר   /    ויתגדל ויתקדש!
