


	
	קראני גביר הקדיש קראיו
	



קראני גביר הקדיש קראיו / ר' משה אבן-עזרא


קראני גביר הקדיש קראיו / והקרני לחברה עם־ידעיו



לדודים מֽתקו אל־פי כדודיו / ורעים יֽקרו עלי כרעיו



בגן־ביתן אשר החן סביבו / והיפי לארבעת רבעיו



מלבש כֽתנות משי ועטוי / מעיל תולע ושש ושלל צבעיו



ערגותיו בלי רקם וחשב / משבצים ושושנים יצועיו



כאלו הזמן קלע פניהם / בתמריו ומקלעת פקעיו



כזהב מעשה מורד עליהם / מאור שמש והשיחם רקיעיו



ומימיו יגרו ארצה כדמעי / עלי מפקד אחי לבי ורעיו



ותור יתור לזמרתו הדסים / וסיס ישיש עלי גפי נטעיו



ירנן בין עצי שרק ושרק / ויתרונן כשכור מדמעיו



ועופות פֽרצו על־כל־פרציו / ושם דגרו ובקעו בין בקעיו



ביום ערו ישני גיל ועצם / זמן עיניו ונזרו פגעיו



ועתיו מהרו לעות יעפיו / עליצתם ולרגיע רגעיו



והשלחן דמות גלגל ואגן / כמו־סהר וכוכבים גביעיו



קערותיו מלאות מעדנים / ומחתתיו בשמים ראש ויעיו



וכוס שהם אשר עלה ככוכב / ביד משקה ומי־אדם במעיו



כחשק קֽפאו מימי דמעיו / עלי לחיו והאש בין צלעיו



יקצץ מזהב אופיר פתילם / עלי ידיו להאיר את־זרעיו



להסיר מלבב כאב עצביו / ולרגיע בלב רגז רגעיו



לאט יהגה וראש יגה וישגה / באהבתו ויאמנו פצעיו



חמדותיו לאין קצה וקצב / כאברהם במהלליו ודעיו



אשר יזל כטל אמרו וחסד / בידו מחנותיו הם ומסעיו



רחב לבב רחב־צעד להיטיב / ולרע מה־מאד צרו פשעיו



ולכתו עם־אלהים ואנשים / בהצנע וחכמה את־צנועיו



ולאבד במחשך הֽיגונים / כתרן אהלו נטע לתעיו



ובו ינֽפשו מרי נפשים / וינוחו יעפי־און יגעיו



ומי נקרע סגור לבו בחצי / נדד פיהו יהי תפר קרעיו



הכי רכו כשמן דברתיו / ויום אמר יפצץ בם סלעיו



אנשים כֽבשו להם עבדים / ומאין הון בני חורים רצועיו



עדי בשו לנדבתו עננים / והודו לו נדיבי עם ושועיו



ומי זרע ימי חייו חסדים / יהי קוצר לפי־חסד זרעיו



יהי שלו עדי ינחיל כבודו / ומעלתו לילדי שעשועיו.
