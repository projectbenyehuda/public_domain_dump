


	
	[לאנשי לבבות בני מחשבות]
	




לאנשי לבבות / בני מחשבות / בידם נעמות / למחית ברואים

ועיני שעפים / עתידות מצפים / ואסר זמנים / בלי פה מניאים

ודעות כמראות / ראות הפליאות / ועצות כמראות / וחזיון נביאים

ישורון עצמות / וסוד תעלומות / והמה במסגר / פגרים כלאים

ומה טוב נפשים / במצער אנשים / במחנות קדשים / ואל על קראים [5]

כעצם שחקים / וזהר ברקים / עלי עש עמסים / וכימה נשאים

ולולי יצועים / בלבות לדעים / אזי גברו מ-/ גברים לבאים

ויצר ישיחם / ושכל יריחם / עדי נבדלו מ-/ עירי פראים

ויש בעלמים / מסכל חכמים / וצעיר לימים / מבאר פלאים

ובקרב מליצים / ללחם חלוצים / להפיל נפילים / ולחלש רפאים [10]

כגורי נמרים / בילדי שעירים / ושאגת כפירים / בעדרי טלאים

לאחור משיבים / עלמים ושבים / ואתם באתות / אמת מחטיאים

וכמה ישישים / כגבה ברושים / גדלות מבקשים / והם אך נכאים

תמונת גברים / תבונת עירים / ודעות גמולים / וגשמי מריאים

מתי הוד ומדות / צניפים ומדות / ומסיג אבוסים / ועול בריאים [15]

ומפת פתיות / שבעים וסכלות / ומיין כסילות / ימותם סבואים

ומהם אלפים / אבל כאלפים / עלי מאבוסים / בכל טוב מלאים

גבהות ישיחון / ויום ריב יריחון / להשכל יבאון / בתוך מחבאים

ופצחים במדבר / כרוח במדבר / ואך לא לחבר / כרוח שפאים

ומשל לפנים / הכי הון עונים / מסתר ועשר / מכסה חטאים [20]

קחו לי מספר / בפי עט וספר / לצפצף בשפר / כעוף בין ענפים

לכתב אמרים / לשכני עברים / בספיר ספרים / בנפך מלאים

לאחים נפוצים / בלבי קבוצים / צפוים בעיני / זממי ונראים

ובין בעריהם / יבלו ימיהם / בהבל כגרים / וגרי כלאים

היו נא נכנים / לשלחן אמונים / וגשו רעבים / והוי כל צמאים [25]

לרות אמרים / כנוזלי שעירים / בגני כפרים / וירק דשאים

כנפת לפיות / ועל חם כליות / כפג בציות / ומלקוש נשיאים

ולעות יעפים / דברים כצופים / וקר מי עדנים / ולא מי גבאים

ואל ים מזמות / משלים וחכמות / וגביש וראמות / בחופו מצואים

במימי מצולות / פניני תהלות / ומר ואהלום / נבכיו מקיאים [30]

לברים כצופים / ורוש לחנפים / ועשר לחפים / וחסר משנאים

ומכל עבריו / עצי מר לאריו / וטרפי אמיריו / תרופת חלאים

נגידי אמרים / לעברים והגרים / כאלים לעמקי / שפתות ותאים

ולקח לרוגנים / וצחות לשונים / ועצה ואמצה / לשרי צבאים

פרדי לשנות / לאסף תבונות / וחמאת אמרות / לחמדת נשיאים [35]

לחרש באבו / בגן אל מסבו / וכל עץ אשר בו / הדרו מקנאים

מגבר חילים / בפי עט ומלים / ולו עז שחלים / וחן הצבאים

ולשון תשובב / מליצות לנובב / נפשים תשובב / ותחכים פתאים

קרסים למעניו / בני רעיוניו / בללאות אמריו / בלי יד מביאים

באזני כסילים / נתנם עגילים / ועל צורוני / לבנה חלאים

ובידי חסדים / נאמיו צמידים / וחייו בחיי/ בני עש תלואים

והנה ילדים / להקיץ עתידים / ישני עפרות / ועורר רפאים

ונשאים בפיהם / ולא על ידיהם / גדלת אביהם / עדי גוי שבאים

וכוכבי אמרות / להאיר והורות / אנשים במסגר / אפלה יראים.
