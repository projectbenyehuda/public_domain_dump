


	
	[הנפשי בחרו לשכן יגונות]
	




הנפשי בחרו לשכן יגונות / ואתי חמדו יחד מהומות

ושטו על־פני תבל ובינות / צלעי נתכנו להם מקמות

וילדי יום הלום לבי והכשיל / בכשיל רוצצו ובמהלמות

ודמים ידרשו מיד בני־איש / וממני לבד יקחו נקמות

בפני חזקו פנים ובושו / לכל־לא־בוש ולא הסיג כלמות

קרבות יערכו יום יום לנגדי / ותמיד יקראו להם שלומות

בעין קצף ישורון בעלי־תם / ועין חנות בני דמים ומרמות

יכחשו באיש כחש בשרו / דרש חכמה ופתח סתמות

ויתרעו לרץ משד וחמס / עלי־כסל עשה עשת ופימות

גבול גאים עליהם חק להציב / והשיב מוֽשבות ברים נשמות

לאט כי לא ככל־פעם זמן קם / לריבתי והגיש העצמות

אבל דמה להשוות צר ואהב / ונפשותם ביגון תת עגומות

וכל־עת יהפך ידו בגופות / ולא־נגף לבד היום נשמות

ועוד בקש עלי נפשו תנואות / כאלו לו עלי נפשו אשמות

והורד את־פאר עזו אשר־בו / ארצות אחזו על־רום תרומות

ונצל הוד צבי עדיו ונפץ / כנבל יוֽצרים גביש וראמות

והכוכב אשר יפיץ מאורו / ביום וליל פני שכני הדמות

רחב נפש כפי שכלו ודעו / אשר היו רחבים מתהמות

שפל לב עם־בני תבל והיו / לידיו על־בני־עיש תשומות

לצריו נמצאו־בו חיל וחמות / ולקשרי לבבו חל וחומות

ומשמן מאד רכו דבריו / ומתקו מעניו מצוף וחמות

מחדש מחלצות חסד ביום כ־ / תנתיו על־בני תבל פרמות

וכמה מעלות הוד נקבו־בו / אשר לא־נקבו מאז בשמות

ואין נפתל ועקש באמריו / וצחות פיו כמפעליו שלמות

לאם־דרך עמד רעה מבשר / והשמיע אלי קצוי אדמות

והרם קול לכל־שאל אשר מר־ / כבתיו להלך אליו רתומות

עמד כי־נעצרו עבי נדיבות / וימין בה לך נצח נעמות

ולכשל צפר כי־סף גביר פיו / ואמרותיו לכל־כשל מקימות

ולהלך נטה כי־סר נדיב לב / שכון תחת מכון ארמון ישימות

והאמון עלי תולע לחבק / בעי מדבר מקום תולע ורמות

כהיום אהבי קראו מבכות / ותבואנה ושלחו לחכמות

והחישו בנות יענה ויהיו / כהמיתי בלי פוגה מהימות

ומי־יתן כים ראשי ועיני / כנאדות יגרו־מים וחמות

ויזוב דם כבדי עד־אשוה / נהרות מדמי עיני אדמות

לתוגה שחרו פנים אדמות / והלבינו קוצות העלמות

לדוד היה כחומת אש סביבי / ובנפל בו לבד אמצא תקומות

אשר עיניו פקחות על־אהבי / ומחטאי ביום אשגה עצומות

למותו נֽקטה נפשי בחיי / ומות בחרה־לה מעצמות

וחלו כל־מגורי בו ואין־לי / למות בלתו לעד פחד ואימות

למי אפחד ביום פחד ואשמע / בהתיעץ עתידי תעלמות

לרחוק אקראה אחיו לנחם / לבבות על־יגונות העצומות

לשון מכתב יחוכם מה־בלבי / ושפתות אגרות אבל חתמות

מידעי זמם כל־איש תפיקון / ואיכה נגדכם אערך מזמות

וכזכר בעלי דעות כבודם / אליכם לערך אתם ודמות

שכר מבחן אנוש הודות עלי־כל / בפיות מספד ונהות חסומות

ומה־יועיל קצף על־הזמן או / שפך חרון עלי ימים וחמות

ועינכם מֽראות אינה עצומה / ואזניכם שמע לא אטמות

הכי תבל בעינינו כליל / ונחנו בה וימינו חלמות

ילדתנו ובינינו כאיבה / זמן נתן לבין כלה וחמות

ויד־מות תניפנו לשחת / כהניף יד־אנוש חרמש בקמות

ודי כי נחלו־לו כל־יקומים / ובכו עד־אשר בכו מרומות

וחלו־לו בנות עיש וגחו / כאלו נוֽתרו מעש יתומות

ושולל יהלך חסד למותו / והבינות למפקדו ערמות

וזכרו עד־מאד גבה בתבל / והן נכפל ונשא אחרי מות

וחלה בחלי־מות ואכן / פעליו לו בעדן מחלימות

למען יצדקו האֽמרים כי / תמותת בעלי־ישר תנומות

כפי גדלו יכונן מעלתו / אשר־לו עז ולבש תעצמות

ונחומים כפי שאת לבבכם / יצו לכם ומשכרות שלמות.
