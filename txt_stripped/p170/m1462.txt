


	
	הא לך מכתב מאח נכאב
	




הא־לך מכתב / מאח נכאב / חטאו פרש / פח לפעמיו



וזמן הצביא / צרות עליו / עד כי־שלח / יד בשלמיו



מי־ראש בלבד / נתן משקיו / שרש רתם / רק מטעמיו



אחליו אלו / בימי נער / יאבד ויהי / מת בעלומיו



אז ינוח / לו כי אחר / כבוד יהיה / בא אל־עמיו



הן מאויו / לכלות חייו / מות יבחר / לו מעצמיו



כי לא־ישכב / לילה לבו / גם מכאבים / הם כל־ימיו



פצו עליו / פיהם צריו / גם כל־היום / שפתי קמיו



עד כי־אחיו / הקשה לבו / ובלי־פשע / שחת רחמיו



לו דברתי / מאז כזאת / אל־הסלע / נתן מימיו.
