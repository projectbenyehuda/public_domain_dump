


	
	ילדי ימים חשו לצבא
	




ילדי ימים / חשו לצבא / על לב הכחיד / יגון רבו



כמעט מזער / מנהו נשאר / שלף עליו / פרוד חרבו



אך בו ישב / יהפך ידו / ישוט אכן / אליו סבו



קשר עליו / קשר אמיץ / וישם תוך / קרבו ארבו



כל מבטחיו / בדמי שיחיו / ודמי עיניו / רק משגבו



וכתב דודו / תמך ידו / העת פידו / העיר עצבו



מכתב דמה / לפני שחר / פרש עליו / טורי ערבו



טורי שירים / יקרו מפז / אחד מהם / ירדף רבו



משי חבר / מהדבר / הבין שתיו / התם ערבו



הנה כל שיר / יקר כגוף / נגדו אך הוא / רוח קרבו



לולי נתן / הודו עליו / היה ככלי / אין חפץ בו



איך בן נעים / וצעיר שנים / יעמס הרי / בין על גבו



או עול ימים / יהדף רמים / ובעוד נער / הוא ובאבו



הן משעיר / זרח לעיר / ארך עולם / ומכון רחבו



וביד רמה / מראש כימה / נשקף כי לא / מאשנבו



איש כמוני/ באשון שכלו / ראה אתו / ובעין לבו



על יד שמעו / הבין רעו / ויראה את / מטמון חבו



עבר ארח / בעדו ישאל / אז מדוע / בשש רכבו



נא אמרו לו / לרוץ אל גן / אהב ורעות / בשמי עשבו



אל גן נתן / נרדו ריחו / רבעו ישים / בו ומסבו



וארות מגדי / דדים בשלו / ואכל חנם / את טוב חלבו



ובבית מדות / לגור תוכו / יום כל בית / סגר מבוא.
