


	
	אני גבר יגון פרוד דחפו
	



[אני גבר יגון פרוד דחפו] / ר' משה אבן-עזרא


אני גבר יגון פרוד דחפו / טרדו החרי כדור צנפו



והציבו למטרה זמנו / לחץ יגון ומכל־צד אפפו



והחל בו נשב רוח צפוני / ולא עמד ויד־רגז הדפו



ורוח הדרומי קם בקנאה / גדולה מצניף הודו חשפו



ומזרחי לזאת קנא וזעף / ברוח עז מאד קשה שאפו



ומערבי לזאת הגדיל ולמאד / ביד רמה בכור עני צרפו



וסוד מחשב לארבעתם מיחד / וקם סודם כמו קישון גרפו



וילד־יום מאד איום כרגע / כארב קם ובחרב רדפו



ואם־נשבע לענתו זמנו / אזי קים תחלתו וסופו



ואם־נדר נחתו אט הפרו / כמלוח עלי־שיח קטפו



ויום חשב לבבו כי־נשאו / בחמלה הזמן אט על־כתפו



נשאו אז לים הולך וסער / ושמו תוך אני־און אל־כתפו



ורב חבל יגון תבל הקימו / וקם מרעיד כמו יונה ביפו



הטילהו בים זעף וחמה / ודג ילד דאגה הוא חטפו



וצעק ממעי צרות לצורו / ומהים הקיאהו לחופו



ועבר נחלה עליו וכמעט / נהר עברה וגל חמה שטפו



ולא־שקט והנו רב תלאות / ולולי רחמי יוצר טרפו



והנה על־מסלתו מצפה / ארי יגון וכמעט קט תקפו



והוא הולך עקלקלות ארחות / עקלתון כקט עקב ישופו



לאט־נא הזמן על־דל ונספה / לאט התנהלה נגף ורפוא



אנוש חלש עלי־תקיף היוכל / אשר כדור לכל־עבר צנפו



ואם־גרזן עלי חצב היסכן / היות גדול ומשור על־מניפו



אנוש נואש ראות אפי ידידיו / ולהט אש תשוקתם שרפו



וגם־עבר זמן עדנה וקפץ / זמן זקן ובחרחר שדפו



ואור עיניו בהתחבא מהלך / ופתע בא מטר־אפל סחפו



וקול עבר כקול כרוז בחיל / קרא שחרות ואמר מה־לך פה



עדי ברחו ימי־נער ושבו / קוצותיו והתלבן צניפו



יהי רצון וחפץ מאלהים / לשובב נענה יומו דחפו



ויזרח־לו מאור שמש צדקה / ומרפא לו יהי־לו עם־כנפו.
