


	
	[קומי בת ולכי]
	



סימן: משה




קומי בת ולכי  /   שובי לעריך  /   שלישיה

הנה אנכי   /    שלח לכם  /    את אליה: [מלא' ג, כג]

מאהבה חולה  /   יונתי בחגוי סלע /  נדדה

אין מושיע לה  /   צרה כמבכירה  /  יולדה: [ירמ' ד, לא]

      נוהמת בבכי

      תוך מכמר

      חלף זמר

      מבור תחתיה   [אי' ג, נה]

הנה אנכי   /   שלח לכם   /   את אליה:

שש משי רקמה /   תעדי עדי פז  /   לבוש סהרון,

טורי אחלמה  /   אל צורניך    / יהי יתרון

וכאור תעלמה  /  הוד תלבשי אבני / זכרן: [שמ' כה, יב]

      בת נדיב ערכי

      שיר מהלל

      כפרט כלל

      לשלל שלל

      עוד בי משכיה     [יחז' לח, יג]

הנה אנכי    /  שלח לכם   /  את אליה:

הילל בן שחר  /   ינון שמו, הכל   נכספים

כי אותו בחר  /   צור לפני שמש /  ושרפים –

מה דודי אחר  /   את פעמי רכבו /   היפים: [שופ' ה, כח, שיה"ש ז, ב]

      המאיר חשכי

      תקרא דרור

      וצרר צרור

      וברר ברר

      יושבי מאפליה    [ ירמ' ב, לא]

הנה אנכי  /    שלח לכם    /  את אליה:
