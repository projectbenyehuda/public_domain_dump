


	
	אני אמרתי אבוא בתפלה
	




Abraham da modena


[נולד במודנה בשנת 1511, חי במנטובה, ארצו, פֿררה, פֿורלי ובערים אחרות. בשנת 1575 היה עוד בחיים]





תפלה


אני אמרתי אבוא בתפלה / לפניך אל
נורא תהלה / כי אין לי מנוחה / רק בעת שפכי שיחה /
באימה לפני כסא כבודך / ובתחנה אבוא עדיך /
באמרי ענני נא אלי אלי / הביאני אל־שמחת גילי /
לישב בביתך לארך ימים / עם יראי שמך החכמים /
לעסק תמיד בדברי תורה / בפלפול התלמידים
וסברה / לקים מצותיך באמונה / ולתת לעני ואביון
מנה / משמים ממדת רחמיך / ברכני אלהי
בשלומך / והראני בנעם התאוה / לקח תורה בגילה
וחדוה / אותה לבד לבי מבקש / ועל־שערי רחמים
נוקש / לפניך אדון התחנונים / זה כמה וכמה שנים
/ יי הושיעה ביום קראי / ענני מהרה טרם קראי /
לפניך ברוח נשברה / לבקש מחילה וכפרה / על־רב
חטאתי / פשעי זדונותי ועונותי / אל־תזכר עון
ראשונים / וגם לא האחרונים / הראני את־ חסדך
לעולם / ואל־אשוב כדך נכלם / ואת־תפלותי שרבו
/ לפניך כאשה יחשבו / עם ערכי בקול שועות /
ועל־לחיים דמעות / תהיינה כלנה מקבלות /
תובלנה לפנים בגילות / ורבים מאנשים הנאנחים /
ישבו עמי ששים ושמחים / ואמר בלב שמח וטוב /
הודו ליי כי טוב.
