


	
	עלז לב אסיר
	







בכתב יד רשום עליו: להחכם הר׳ שלמה בונפיד, בעבור הסתיו והגשם חלף והעת עת דודים שלחתי זה השיר לידידי יהוסף ואמר.






עלז, לב אסיר תקוה, / אשר לכפור חולה,

עלז, כי כבר שמש / מסובב בראש טלה

והלך למסעיו / למעלות ולרגעיו

ויתגל ברום גלגל / להצהיל לבב גולה,

ונטש פאת דרום / ועלה לראש מרום

ונכח למשוה יום / יחצץ ויגלה.

סתו עם כפור ברח / וכל עץ פרי פרח,

וחרס בחוג יזרח / וסהר באור מלא.

ויחשר בכח שור / גשמים, ואז ישר

לבנון באגלי טל / וכרמל בגיל מלא.

ולבשו כתנת פס / תכלת וחור כרפס

עמקים, וכתמי פז / ירקרק עלי עלה,

ועל צוארי יונים / רבידים ושהרונים

עשויים ביד עב טל / וענן באד יעלה,

והתור והזמיר / ישירון בראש אמיר,

ככנור וקול נעים / בשיר יעשו פלא.

ואחים בבין אחים / ישיחון עלי שיחים

וסודם ליעלות חן / לאט כל ידיד גולה.

בתולות כאילות / בתפים ובמחולות

בתוך גן תרקדנה / בזמר ושיר נפלה

ועם פה וניב חלק / ואישון כמו ברק

תשימון בגן ירק / עלמים בבית כלא.

אזי נחלו חושקים / באש אהבה דולקים

ורעיות יקר דולקים / כמו הזאב טלה –

אבל כוס מלא יין / ביד כל יפת עין

לעורר עלי עין / לבב כל ידיד נחלה.

תנו עז, מתי חשק, / תנו עז לבת גפן

אשר תעשיר פתאום / לבב רש ואיש נקלה

וחזקו באהבתכם / וסבלו עבודתכם –

צבא שם קחו לכם / למופת וסוד מפלא:

עגלות וכדורים / יקרים כספירים

ירוצון להשיגו / וחשקם בלי יכלה,

ורוצו אלי עפר / יהוסף כליל שפר

אשר לו צפון חכמה / וכל סוד וכל נגלה

והלכו לפי ערכו / אלי יעלת חשקו –

עדי עד יהי סכו / בשלם ולא יבלה!

ואמרו אלי דוד צח / ברנה ושיר פתח:

אסיר הנדוד פתח / אשר בעדו חולה.
