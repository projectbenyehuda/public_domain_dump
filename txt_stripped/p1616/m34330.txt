


	
	הולך הלוך ונסוע
	







מתוך קונטרס המליצות והשירים נגד נכבדי סאראגוסה.

עליו רשום בכתב יד: וזאת המליצה בלתי שקולה שלחתיה לשרי סאראגוסה הסוררים.





א


הולך הלוך ונסוע / בהרי הגלבע /
פגעתי בעיר גדולה אשר אבניה יסדו על דמים
ומרמה / והעם אשר ראיתי בתוכה אנשי מדות מגנות
ימותו ולא בחכמה, / אנשי בוגדות ואשתדור עבדין
בגוה וקשר של קימא, / לא ירעו לא ישחיתו כי אם
גנוב ורצוח ונאוף ועשות המזמה. / לא חמור אחד
מהם, כי כל העדה פתיות ובל ידעה מה! / עדה פחותה
מעשרה באיכותה – יש בה לאלפי רבבה / אדם, אשר
לא ידע בין ימינו לשמאלו ובהמה רבה. / עם כבד
עוון המה, שריהם ושופטיהם את יי לא דרשו /
ונודעו כי ערומים הם ולא יתבוששו, / כלים ריקים
חלולים חלולים / כעוללים, / לא ראו אור רק החוצה אל
העין, / בני בלי שם נכאו מאין / וגויות מחטבות
כתבנית איש, ואדם – אין! / בוערים בעם והדיוטות –
/ ראשי המטות, / החיטים – / שופטים, / ורצענים – /
דינים / וראשי סנהדראות, / אין איש בהשכל –
דבריו יכלכל – ואין לדין אלא מה שעיניו רואות.
/ כל בור אינו ירא / חטא – הוא יורה, / וכל תפוש זהב
וכסף וכל רוח אין בקרבו / עושה רוש ובראש הקהל
יבוא. / סוסים מיזנים / משכימי קום בתפלות
ובתחנונים, / ואחר אשר יתפללו – / איש אל אשת
רעהו יצהלו. / קדמו עיניהם אשמורות – / וטמאותם
בשמלותם צרורות: / ביד שמאלם אגדות קשר של
תפילין קשורות / וביד ימינם השופרות. / ואם באחת
יתקעו, / יתהללו ביום מחר ונשבעו / כי ברבות
בנות כשלו ופשעו. / וכל אשה חכמת לב יפת תאר
ויפת מראה תשב בפתח עינים / ועם העוגבים – התנו
אהבים – ומשקרות עינים. / תמרוק ברע על עיניהם
והזונות רחצו / ובילדי נכרים בעלי נעורים מאסו,
/ וכי תחלינה לזנות / תלות עצמן בגוי ודתיהן
שונות. / ורב הצנועות שבהן / יתנו יקר לבעליהן:
/ כי איש עברי לבד יערה פתהן / וכל בן נכר לא יאכל
בו, / והבעל העלוב ישמח ויאמר בלבו: /
ממשפחתנו הוא! וכפר על קרנותיו וחרפה לא נשא
על קרובו. / והבחורים יצאו מן החרים ונפשותם לא
תשבענה, / יחתרו בחשך בתים והנשים
תשגלנה, / הקטנים עם הגדולים גדלו מעשיהם
בדברי רכילות, / עברו משכיות לבב בגאוה ובגדל
לבב כרום זלות. / וזקני העם ושוטריו נשבעו
לשוא על פת לחם ולכחש ילבישו אדרת, / נכרתה
האמונה ותהי האמת נעדרת. / ובראותי שמה
ושערורה / וכי נגע לשמים משפטם ונתמלאה
סאתם וחטאתם גברה, / יגרתי מפני האף והעברה /
וקנאתי לאלהי, נשאתי תפלה בעד השארית
הנשארה / ונשאתי קולי כקול אשה מצרה / ובבית
פרטתי עוונותם פעמים עשרה – / ואין מקיץ כי כלם
ישנים, / ערלה אזנם שוכבים בחיק הזדונים, /
ותוכחתי בדרשותי לא השיבו פושעים על לב /
ויהמו ככלב / אין עושה טוב אין נבון וחכם…




ב

מעל למליצה זו רשום: וטרם אכלה לדבר אכתוב התקנות והחקים לא טובים, אשר עשו ביניהם שטי כזב פונים אל רהבים.






עוד ראיתי תועבות גדולות / בהתאסף
ראשי עם רשי תבונות לתקן תקנות או תקלות / לשים
מכשולות במקהלות / ולגזל בחירתם אשר לא צוה
אותם יושב תהלות. / ועניי הצאן לא ראו / כי
בנפשותם הוא, / להתעולל בהם עלילות / ולעולל אותם
במסלות, / ולא נתנו להם כתף סוררת, / נגזרת
עצרת, / בוגדים קימו וקבלו / וכל עיניהם על עשרת
בני המן תלו, / והם חשבו לכתב על קצת יחידיהם
שטנה, / ויפל הגורל על יונה: / כי אותי אשר לא
עשיתי עמהם רק טובה עד כי חדל לספר צוד צדוני
כצפור. / לכן קדם אפרט ההסכמות, שבעה מהם
בספרי אזכרם / ובמחוגת עטי אתארם / ובציור
רשעתם אצירם, / למען ירוץ הקורא בכתבי, ואנשי
אמת עת דברי הספר הזה יקראו / דמות פניהם עין
בעין יראו. / אל אלהים יצמיתם וימיתם כאחד /
ויפלו שבעתם יחד!

הראשון משה עפיה המתנשא לכל לראש
ראש ותלאה, / כמוהו לא נראה, / משתרר גם השתרר
על הני יהודאי טפשאי, / הולך רכיל ואוכל קרציהון
דיהודאי, / לוקח נפשות בארשת רשת שפתיו ונפל
בעצומיו חלכאים, / רגליו לרע ירוצו כאחד
הצבאים. / עושה עשר / ולא בישר, / פיהו מלא חצים
ומות כמתלהלה היורה זקים, / הוא אמון הרבה
אשמה וברית לא יקים. / וכי יכרת ברית עם רעהו – /
לערב ימולל ויבש / ויהי ידיו אמונה עד בוא
השמש! / להט החרב המתהפכת, / אשר בשלכת /
הרע מתנשא ומתגאה – / אפס קצהו תראה וכלו לא
תראה.

והשני אברהם אביוד האדם האדם / הוא
עשו אבי אדום, / הבוגד בוגד לבו יקבץ־און לו,
וכל רע לבו רחש, / שפיפון עלי ארח וערום כנחש. /
ובשרו נחוש וגיד ברזל ערפו, / יחשכו כוכבי
נשפו, / סער מתחולל – / אילות הקהלות יחולל. /
והן שמים יעיב באפו, / פורק על ונפשו כחש
מלאה וזדונים, / וגם היא הרה לזנונים. / קטן קומה
בדעה, ועת ימלא עברה וזעם – / משכמו ומעלה
גבה מכל העם!

והשלישי / דומה לעורב וכושי / הוא
משה בן־ימין / הן בקדושיו לא יאמין, / זפת בוערה, /
גלם בלי צורה. / הרוכל הנוכל, השרץ השורץ /
כלו ארס, / ובנשך ובמרבית כל עמו אוכל. / עושה
סמים / מגחלי רתמים / ומרמות ותך יתן בראשי
בשמים. / איש שותה כמים עולה / אשר חמתו חמת
תנינים ויין תרעלה, / הפכפך דרכו וזר / וראש
פתנים אכזר, / איש ריב ומדון / ביום הבראו כוננו
כחש וכזב ופרח הזדון. / ערירי ימות וחמסו בראשו
ידרש / וביתו שדה תחרש!

והרביעי וידאל אבולפידה / על זכרונו
פחד קראני ורעדה. / איש צר ואויב, המן הרע, /
הסופר הכופר כלו חנף ומרע. / עיני יורדה מים
ותנוד כנוד הקנה כי אעלה על ספר זכרונו, / כי
ברשע ובפשע עבר על אנשי מינו. / אבי אבות
הטמאה ואונו בשרירי בטנו, / עפר ואפר / לא למד
ספר / ולא נשמר מחטא בלשונו, / קלמוסו כנשא
יערף כמטר שוא / ודבר כזב, / ותשב קשת רמיה
ברשתו וקנו / ויהי כמצחק כי יקיף השטר
ויקדים זמנו, / ומכתבים עמל ילמדו לקח מרע
מעללו וזדוניו, / כי ידבר הוות מולל ברגליו
וקורץ בעיניו, / און יחשב על חברו / וטבחו או
מכרו / וישא בשניו בשרו. / אבן מקיר תזעק על
זדונו, / יגלו שמים עוונו!

והחמישי איש שלומי
אשר בטחתי בו הוא בונאשטרוק די שלינש אשר
בריתי אתו ברית מלח – / הוא ים המלח, / מלח סדומית,
/ מסמא את העינים בתרמית. / הוא הקהיל עלי את כל
העדה, כי כה משפטו להרע לכל רע ועמית / העצמות
היבשות איש כזה לכל דבר אמונה אמן / והיה דשן
ושמן / לחרחר ריב ולכל דברי בלע / ונפש אוהביו
יקלענה בתוך כף הקלע. / כלי ריק מלא מרמות
כרמון, / תולעת ולא איש וקול דבריו כקול המון, /
והוא מתקדש ומטהר ושרץ בידו, / ויתן למופת
כי אביו שרף – והנה צרף ביום אידו. / במאזני
מרמה לבבו, זכות אבותיו ישים לקלבון, / אוהב פשע
ואוהב מצה לתאבון, / וכפיו מלאות חמס וגוזל קהלו
בעת יושב בחשבון.





אלה החטאים בנפשותם / נמנו וגמרו
בסכמותם / לצודד נפשות בתרמיתם, / וזאת תורת
האדם לקימה ולחזקה נתנוה לשני חזנים
זהירים / לקראה בכל חדש בשמחה ובשירים /
לפרסומי נסא דלא תשתכח באגרת פורים, /
ואזכיר תארי שניהם על מתכנת פשעם ורשעם /
לאחד קראתי חובלים והאחר בלעם.
הששי אברהם שועיב נקשה ורעב, רב
החובל, שר הטבחים דאתיליד במאדים, / שופך
דמים מכמה צדדים, / ויצא מחלב עיניו ופניו על
זדוניו עדים. / מצות ״לא תנאף״ / ירדף באף, / אף כי
אמר אלהים ״לא תקח שחד״ – / כלו רודף שלמונים
ואוהב שחד. / חזן הזן כל העולם בזנוניו /
ושעירת עזים לחטאת מצא חן בעיניו.
והשביעי הטמא והערל / אברהם מוריאל,
/ דומים פניו למגדל חזירים ונותן לפגם טעם, / שמן
כעגלון, ממאיר כסלון, וסר טעם / אוכל תרטימר
בשר ולג יין ועברה וזעם, / אחי אשכול פיכל שר
צבאו / וכהתעות שכור בקיאו. / הולם הבריאות
כהלם פעם – / קול ישורר בחלון כקול רעם, / וזה
הסכל יעלה לתבה כפעם בפעם / ויפתח פיו לקרא
התקנות ויקלל את יומו וכפתחו עמדו כל העם /
וההמון העלוב בשמעם את דברי האלה והקול הולך
וחזק ילבשו חרדות / ופניהם איש אל אחיו יאמר ״מה
החקים האלה והעדות הנכבדות?״ / ולא ידעו כי כל
אלה משטח חרמים ומצודות!




ג

מעל למליצה זו רשום: טופס התקנות שהסכימו ואותם החכימו.

לפי סגנונה הרי זו פארודיה על נוסח התקנות המקובל.






זה טפס החקים / המיסדים במחול
משחקים, / והם רמז למומיהם דכלהו איתנהו בהו
איש לא נעדר, / אמנם לשוא צרף צרוף, כי לא ידעו
הכלם ואין תבונה בם ולא הדר.

תחלה העשרה נבררים הסכימו / שכל
מלאכת יי יחרימו / ובערו הקדש מביתם ואש לא
ישימו.

עוד הסכימו תת קדש וצבא מרמס / ולביא
ממזר בקהל ולהתיר את השרץ ואת הרחמה ואת
התחמס, / וכי יקום עד חמס / שיהיה פטור מן המס, /
וכי יכרת ברית איש עם רעהו – וחם השמש ונמס! /
וכל אשה יפת תאר לא תעדה עדי כי אם עגיל
וכומז.

עוד הסכימו שכל הולך רכיל, / אלפים
בת יכיל, / וכל דין שיטה את הדין ושחד על נקי
יקח – / שיתן דנקא לצדקה וישקוהו מיין הרקח.
/ וכל הגונב תפוחי ספר תורה / או הסיר המצנפת
והרים העטרה / יעטרוהו בקוצי המדבר
ובשושנים לצפירת תפארה.

עוד הסכימו שכל מקרין מפריס ישא קרן
חזות בין עיניו, גנב / שיכנס בלילה בעד
האשנב – / ביומו ינתן שכרו ושמו מסוה על
פניו.

עוד הסכימו שקדם שתבוא הכלה לבעל
/ שכל העם מקצה אותה יבעל, / ואם אין לו דמים – /
נא יתן לערבון הטלית והמאכלת ולא ישפך עוד
דמים.

עוד הסכימו שכל מי שימצא איש אחד עם
אשתו במטה / תחת השמלה לוטה – / שלא יהיה
רשות ביד הבעל להתיר אגדות מוטה, / ויען הבחור
ללון בצל קורתו נטה, / ממקומו לא ימיש והוא יעלה
מעלה מעלה והבעל ירד מטה מטה.

עוד הסכימו שכל מי שממעט את הדמות
ולא יעסק בפריה ורביה להרבות ילדי העברים /
שישקוהו מן המים המאררים. / באשתו אמרו, קל
וחמר – באשת חברים! / וכל אשה שתתן עיניה
באחר ותשאל גטה / כופין את הבעל לגרשה
ויאכילוהו שעורים תחת חטה.









עוד הסכימו שלא תכנס אשה למרחץ /
כי אם עם בחור שיורה כחץ, / ושלא תכנס כלה
לחפה / כי אם עם אגמון וכפה / ותחת חגורה –
נקפה.

עוד הסכימו שלא יהיו לכלה
שושבינים / כי אם בני זנונים, /ולצדי החתנים
יקומו אשרים וחמנים, / ושלא יאכלו בסעודת החתן
/ כי אם תרש ובנתן / ואבירם ודתן / בגנת
הביתן.

עוד הסכימו שכל איש טהר־לב / עלוב
ואנו עולב / שישליכו אותו לכלב, / וכל מי שלא
ישבע לשוא / ובמושב לצים לא ישב / שיהיה
נחשב כגר תושב / ובגויים לא יתחשב.
עוד הסכימו שבמקום שש ורקמה ושני
עם עדנים / והמחלצות והגליונים / שתצאנה
הנשים במחצלת של קנים, / ותחת בשם – מק, / ותחת
פתיגיל – מחגרת שק, / והטבעות ונזמי האף / ילכו
לחרי אף.

ועוד הסכימו שכל מי שלא ילך
ברחובות בגלוי הראש / וישא כובע על ראש ובית
ראש / שישתה קבעת לענה וראש / וילך אבל וחפוי
ראש.

עוד הסכימו שכל נערה יפה והגונה
הלובשת תלבשת בשת וענוה כנזם על אפה /
וכנפיה בכסף הצניעות נחפה / ולא שמעה קול נוגש
ולא החל לפעמה רוח החשק ולא בערה בה אש
ותבערת קברות התאוה ורשפה / ותצער כל נפש
מאהביה והיא אשה בבתוליה הגיעה לחפה – / ויען
צערה את הנפש ורבים חללים הפילה ביפיה –
תהיה לבשת וגם לחרפה / ותבאנה תמנע פילגש
ורצפה / ובידן רצפה / והיה למאכלת אש על כל
כבוד חפה. / ואם עוד תקשה ערפה, / תצאן ערפה
וזלפה / ומלקו את ראשה ממול ערפה, / ותקצצן
את כפה / ויגש גיחזי להדפה.
