


	
	ויהי בימים הרבים ההם
	








גזרות קסטיליה

ויהי בימים הרבים ההם / בהיותנו נלחצים מהגוים ומלכיהם / בגלות החל הזה הרע / אשר עטרת ראשנו פרע / כי אדום הכביד מאד בעלנו / חזק ונתחזק בעד עמנו / עד כי היינו נרפים / כל היום נגפים / כי כל שר ומושל בכליון / ינופף
ידו על הר ציון / עד שהיו לשרקה / מרב צרה וצוקה / וארך הגלות אשר בם ארך / והאויב על שכמם דרך / ומציק בכל עת ובכל שעה / עד שקרן עזו נגדעה / וברב המצוקות והיגונות / נשכחו הצרות הראשונות / וקורות הזמן אשר דחקונו / על ההרים דלקונו / כי חשב להשחית אנשים / עצומים, עם קדושים / ובהלכו בדרך הזה רגיל / נאסף שמחה וגיל / איך אמה חכמה וקלסה / נפלה משמים ארצה /
אשר הכאיב אויב את נפשנו / ויסר עטרת ראשנו / עד כי היינו לחרפה ולבזה / כל היום לרגזה / כי לכל הדרך אשר היהודי יפנה לבבו / בחצים ובקשת יבוא / וכל היום הולך נרדף / כקש נדף / ולא מצאו מנוח / וילכו בלא כח / עד כי ברב
הקורות / ויחנו על עי״ן חר״ד / וברב עמלם וענייהם / קצו בחייהם / כי איך יחיה האדם בין גוי עז פנים / אשר חציו שנונים / לאבד ולהשמיד / ליהודים
תמיד / עד שנדדו מקול המון / מנחל ארנו״ן על הר חרמון / ותפארתם נשפלה / ודרכם נבהלה / וכל צבי מדח מעוף צוקה ואפלה מנדח / ואמנם בהיות אבותינו בתוכם / עברו זמנם עמם ועם מלכם / ולא
דחקום ברב התלאות / כי י״י צבאות היה צופה עליהם / ומביט בענייהם / ומצילם מן הצרות / ומן המקרים והסערות / ואם בזמן ההוא אפל כסמו / מארץ כתים נגלה למו / וראה האל אפלם / והיה להם לאור
עולם / או פרחי שמד עליהם נתחדשו / וגם נשף בהם ויבשו / וידם לא קצרה וזכותם לא נאסר / כי י״י סגר פה כל שר / לבל יחדשו גזרות עליהם / הגוים ההם / אמנם בזמן הזה העל גדל / עד כי לספר חדל / מכל צרות משנות / בכל יום עלינו נתונות / הנחמדים מזהב ומפז / לשלל שלל ולבז בז / כי גברו מאד המסים / והנגשים אצים / והמס היה לאלפים ולמאות / וידל ישראל מאד / וכל מושל ידו הדה / ויצעקו בני ישראל מן העבוד״ה / ומלכם בעלם מאד הרבה / וירדפם עד חוב״ה / ועושים שמד בענינים אחרים / אנשים מדני״ם סוחרי״ם / אשר אויב
תפלתם רכל / ובא אל מקדשו להתפלל ולא יוכל / וכל היום מכלות / המסות הגדולות / כי רעת המס רב על כל מדה / הוא ישסה אוצר כל כלי חמדה / והוא
יפשיט כל הוד איש מעליו / המק בשרו והוא עומד על רגליו / ועל כל הלחץ והתלא / נתון במאסר ובכלא / עד כי משחת מאיש מראהו / וחשב עם קונהו /
ובראותי כזה בכל עת ועונה / נשאתי עיני לאל שוכן מעונה / כי קצתי בקורות האל / ותקצר נפשי בעמל ישראל / ואמרתי: ׳אנא, אלהינו, אשר ממצרים הוצאתנו / ומבית עבדים נשאתנו / והבאתנו על כנפי נשרים / לציון וקרית יערים / איך נתתנו לבזה ולשמה / בין מזה ושמה / וגאולה ממנו רחקה /
אף כי קרבה צרה וצוקה / ואיך לא לצרותינו תצפה ולמכאובינו תרפא / ואיך תרצה בכלותנו / ביגון ימינו ושנותינו / ולא תרחם בעמך הפזורים / בכל
צער נמסרים / ורבה עליהם המצוקה / וכל מכשול ופוקה / ואם מאמה יחדלו חדל / ומה תעשה לשמך הגדול / ולמה שמתנו כעבד נבזה / וזכר כי עבדך הגוי הזה / ואל תכרית שמנו מעולם / יודע כל נעלם. – – –
