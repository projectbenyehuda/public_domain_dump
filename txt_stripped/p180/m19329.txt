


	
	[ביום קצר אפי]
	



ורתאה איצא [וקונן עליו עוד]




ביום קצר אפי / במשל אפתח־פי / השחק עפעפי / ואם־לחיי שריון

ומה־יתאב תאוב / לקול שיח כאוב / לבבי ממכאוב / וידי מרפיון

וייגע ויעף / מאד מדבש רעף / ונשאר העפעף / כחומת בת־ציון

וחמה ימלא / לסוד לא־יפלא / ואיך לא־יגלה / והוא על־גליון

גערתיו שוב לבב / ואל־תלך שובב / הלא כמעט סבב / לך הכליון

ואיך תאהב מטה / ומדעת תשטה / ואל־תבל תטה / והיא כחלום חזיון

ואיך חיים תבחר / ותדע כי מחר / אמת ידבק אחר / אלימלך כליון

ותשמח ברננה / בארץ כמלונה / כמו שמחת
יונה / עלי הקיקיון

והאדם הלך / ובחשך הולך / והנפש מלך / והגוף אפריון

והמות יגל / וישלח המגל / ונרץ הגלגל / אלי בור השאיון

ועיני גבור טח / ולשמש ישטח / ועל־קשתו יבטח / ויתעל בסריון

בכל יפנה ירשע / וכל־היום יפשע / כצדיק
עם־רשע / ועשיר עם־אביון

וחיים בהרג / יקפד כארג / ובהם ישתרג / כהשתרג שריון

ומהם לא־ישגה / למלא בם כל־גיא / ושיתו שח לגא /והבוז לגאיון

ויאכל כחסיל / בני־עיש וכסיל /
וכל־בער וכסיל / לשניו נקיון

והראני חלד / בעודני ילד / אשר תבל תלד / בטרם הריון

ולא מעד מעוד / אשרי בה מעוד / כמו־הוד היה עוד / לרעי
נסיון

בעת בא בגבורה / מסעף כל־פארה / ולקח
הצורה / ועזב הדמיון

ועינו זב בכיה / ונפשו דק
עביה / ועלתה־לה אל־יה / לגיא החזיון

ונשארתי חרש / כעץ מאין שרש / ונכרת החרש / אשר היה
חביון

ואנחתי רבה / במוט מוט הקבה / ועל־ארון נשבה / ויקר הפדיון

לבבי הנמזח / ביגון לא יזח
/ עזב שירי מרזח / והמר שגיון

וכף אל־כף הניף / ויד עוד אל־תניף / והורד הצניף / והסר הגליון

קרע ופרם כל־מד / למקדש כי־נשמד / והיה כל־מחמד / לעינינו כליון

וקונן על־מקל / פאר בפז נשקל / והחלד נקל / בעיניו מדפיון

תהי שלום נפשו / בסור אל־מקדשו
/ ויזריח שמשו / כדברי הגיון

תמול היא שכבת / בחיל ובמעצבת / והיום ישבת / בצל סתר עליון.
