


	
	ששון לבי
	








ששון לבי / משגבי / הנשגב שמו לעד

סלח פשעי / וחיש ישעי / הבא ציר הנועד

אליה / התשבי / מתושבי גלעד





לנדכאים / וחלכאים / שבעי בוז וגם לצון

פנה שדי / ותאמר די / לארך קץ הקיצון

שלח רועה / לצאן תועה / ואל תטש מעט הצאן

אשר נגז / וגם נבז / מלפנים ולחיצון

והוא מיחל / כמו רחל / ליום תקרא שנת רצון

וציץ צמח / ואז נשמח / בשמחת בית הועד





מחצוני / לחצוני / פריץ חיות ואריות

הנני גר / וגם נסגר / ביד ארבע מלכיות

נבו קרס / וקם פרס / ואליפז ונביות

זה יתפש / וזה ירפש / וזה ידיח שיות

ובכל זה / אני אחזה / ועיני לך צופיות

וכל דורש / קץ יפרש / ויספר האותיות



אולי תם / אשר נחתם / ו[ב]יתי תכין עדי עד.
