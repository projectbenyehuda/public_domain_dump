


	
	אכה כפי
	








גורו מיום משפט / ובואו ליום המשפט / כי באש יי נשפט





אכה כפי / על רב אכפי

אך לי תקריב / ותגיש לריב / ואותי תביא במשפט





נוחל סרעף / וכלה עפעף

נבהל רעיון / בקראי חזיון / וקרבתי אליכם למשפט





יום בוא נושה / במו דין קשה /

יצורים ירעיש / ואותם יגיש / לעשות בהם משפט





שדי מורה / לשלש עשרה

שקל נא פשע / במאזני ישע / ואל תבוא במשפט





לכן תחמל / וחסד תגמל

למען תצדק / מאבק דק / המתעבים משפט





מה אנוש נמס / יהלך תמס

מיום נוצר / עולה קצר / והשג אחור משפט





הננו נצפה / חסד כלפי

הכי מהרת / ולנו הזהרת / כי יי אוהב משפט





הצור תמים / במדת רחמים

הכל ינהג / צדקה לוהג / כי כל דרכיו משפט





קל וחמר / קרוצי חמר

קדוש למולם / אל תשב גמולם / השופט כל הארץ לא יעשה משפט





טהר לבב / פתי ושובב

טובך תמד / כי כן ללמד / ידרך ענוים במשפט





נקה תנקה / פשע מחקה

נסוגים כחדק / תשפט בצדק / וענייך במשפט





חרפה תסר / צדק תבשר

זדון תכלה / קמים תבלה / תעשה ברודפי משפט.
