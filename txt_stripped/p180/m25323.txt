


	
	שני חיי
	








שני חיי / ומאויי / לריק ספו בחובותי

למי אפנה / ומה אענה / וחטאתי סביבותי

האל חמל / וחסד גמל / ואם רבו משובותי

כי גר א־/ נכי ע־/מך תושב ככל אבותי



זד יצרי / עלי יוצרי / ולא נכנע ולא נכבש

והותו / בתאותו / כראש פתן בקרב דבש

ומה בצעו / עלי פצעו / אשר לא זור ולא חבש

ולא נמלך / ביום ישלך / בבור שאון וגם יחבש

ולא זכר / כבן נכר / ביום עפר וגוש ילבש

ימי כצל / נטוי ו/אני כעשב איבש

בקום שדי / לריבותי / מה יהיו תשובותי



עזב לבב / נתיב שובב / ורוץ כצבי להנצל

פשט הבל / לבש אבל / והנזר והאצל

וקום ליל / כבן חיל / ואל תישן שנת עצל

ולין בודד / והתנודד / ובבכי אזנך צלצל

והתנחם / ואז תרחם / ואולי נפשך תצל

ומה יצדק / אבק דק / ואיך יוכל להנצל

כציץ יצא / וימל / ויברח כצל

ומה יתרון / לחרבותי / ולאלפי ולרבבותי



גדל חסד / אשר יסד / גלמי וגם רקמתי

אדון עולם / אשר נעלם / והוא נצב לעמתי

לך ארנן / ואתחנן / בכל שבתי וקימתי

עין פקח / ושועי קח / ואל תפקד עלומתי

פדות מהר / ולב טהר / והעבר את כלמתי

רעה אותי / עדי מותי / ויום שובי לאדמתי

ותיטיב סוף / ביום תאסף / לך רוחי ונשמתי

צור מבין / נתיבותי / ויודע מחשבותי.
