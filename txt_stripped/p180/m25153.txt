


	
	שובי נפשי
	








שובי נפשי למנוחיכי / כי יי גמל עליכי

הלא מרוח פיו נפחך / ומכף רגל ועד ראש מתחך

ומכח תכונתו במעט הכינך / ומעצם תבונתו במקצת הבינך

והשכילך לתור חכמות / והחכימך לדעת מזמות

והכתירך בדעת ותבונה / והדריכך על דרך נכונה

ואת שטית תחת צורך / ופרקת עלו מעל צוארך

ותתעבי את שכלך / ותמשכי בכסלך יקר הולך

ותבעטי בחכמתך / ותשימי הרע מגמתך

ותפשטי בגדי אהבות / ותלבשי תחתם בלויי סחבות

ותכבי את נגה שמשך / ותשליכי את עטרת ראשך

ותחרשי רעות לבקרים / ותשוכי את דרכך בסירים

ותרעי עוונות עדרים עדרים / ותצברי מרמות חמרים חמרים

ותשחיתי את נעם יפיך ותתחבלי / ותלכי אחרי ההבל ותהבלי

על מה נאצת אלהיך כל זאת / ותשחריהו היום בפנים עזות

על מי בטחת / כי נסרחת

ותרבי חמסך / ותשכחי עושך

ולא אמרת איה אלוה עושי / צופה עלומי ובוחן מעשי

מה אמלה לבתך / מה טריה מכתך

מה אנושה מחלתך / מה צרובה מכותך

אין חובש לשברך / אין צרי לצירך

על מי תנוסי לעזרה / יום אלהיך אליך יקרא

מה תשיבי ורעתך רבה / מה תעני וחטאתך כתובה

בעט ברזל ועופרת / על סגור לבך למשמרת

לכן שמעי בת וראי והטי אזנך / להנצל מפח פקחי נא עינך

הנזרי מפשע טרם יאמץ / ורחקי מרשע טרם יחמץ

לחמי את יצרך בחרבות תורה / ושובי ליוצרך בדעת טהורה

הנחמי נצח על עשות זמה / ובחטאתך שלחי חרמש בקמה

מהרי התנערי מרב עווניכי / ולפני תשובה שימי פניכי

וכבסי מרעה לבך / ורחצי בנקיון קרבך

הטהרי מעוונות עלומיכי / והתכבסי ממרמות זממיכי

פלסי בצדק מעגליכי / וישרי במשפט מסלוליכי

והיטיבי בחסד מעבדיכי / וגאלי משחת חייכי

והמתיקי מנפת זמירות חכך / וקומי רני בלילה למלכך

ושאי אליו כפיך / והשתחוי לו על אפיך

והכוני לקראתו בזמר נעים / ושחרי את פניו בשירך לרגעים

והמליכיהו עליך באמונה / ויחדי את זכרו בדעת נכונה

ורעדי מפחדו / והתחנני נגדו

ורעשי מכחו / והתנפלי נכחו

זעקי אליו / והתיצבי עליו

ויחלי רצוניו / והשתחוי לפניו

ועל משמרתך עמדי והתודי לשמו / והללי לכס מרומו.
