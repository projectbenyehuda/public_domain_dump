


	
	שש מאות
	








שש מאות / נקראות / בדת האל יסודיהם

לבת מי זאת / נרמזות / על כנפי רדידיהם

מחקקות / מזקקות / כמו ציצית פקודיהם

הזהיר אל / לישראל / חק שמור בידיהם

ועשו ל-/הם ציצית / על כנפי בגדיהם

שמר לבן / כמו דרבן / מקום תכלת הנצבע

לך נפתר / סוד נסתר / חוט ציצית היות מרבע

משפטיו / היות חוטיו / כפולים בתוך שלש אצבע

המשלשל / יהי נפשל / כדי אצבעות ארבע

חק סדינא / כרב קטינא / בדת ציצית לא נקבע

זקף ערך / לחוט כרך / עלות ממניניהם.





שקד ענף / עלי כנף / ומקרן יהי נבדל

לא ידחיק / אבל ירחיק / כמלא קשר אגדל

מנת טלית / נכפלת / כרב שמעון פטר וחדל

הן כפלה / ותפר לה / מציצית לא תחדל

חוט בגד / ועור נגד / כמין בגד לחק תגדל

זנח ונער / חקוקי עור / ואם בגד בכנפיהם

שלשים יום / כדת אים / כסות שאולה מעכבת

לעת הלך / כאן ואילך / הלא ציצית חיבת

מכורת שוק / לעם חשוק / כחקה היא נחשבת

המון תגר / וגוי מתגר / כשרה היא מחטבת

חק נפרץ / וחוט נקרץ / פסולה היא ונעזבת

זכר והתם / וראיתם / פרט לילה מעכבת

קץ נקבע / על ארבע / ולא שלש בסודיהם.
