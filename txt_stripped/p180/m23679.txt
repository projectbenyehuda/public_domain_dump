


	
	אפכה דמעי
	








אפכה דמעי / על רב פשעי / חמרמרו מעי / ביום קראי בקול



בחסדך אבטח / כי אתה מבטח / וכפי לך אשטח / בעת ארים קול



גדל פשעי / העבר צור ישעי / וסלח רשעי / בצרחי בקול



דלתך פתח נא / לבן בא לך ברנה / ובשרהו מעונה / חננתיך בזעקך בקול



הקשב תפלה / שוכן מעלה / והסר תהלה / ממבקשיך בקול



וכתב לחיים / עם נבחר מאיים / וידעו גויים / כי חשקת בקול



זכה בדינך / מחלה פניך / וחן על בניך / מקשיבי קול



חוקר עלומים / זכר ברית עולמים / לעם נכלמים / מלהרים קול



טהר נקיה / ביד צר שבויה / והיא הומיה / שואגת לך בקול



יערב שיחה / ויכון שבחה / לחוקר טוחה / בעת תתן קול



כבודך גלה / על עם אלה / והפלא פלא / יום יצרח בקול



לאנה אברח / ועם מי אארח / ולמי אצרח / על כן לך אתן קול



מוחץ ורופא / חבש נא לנספה / מחוץ ונכפה / בלי הרים קול



נהל בחסדך / לעם דל ודך / מלא פחדך / והוא מתחנן בקול



שגב רחומים / והער רחמים / ופדה עגומים / כעגור יצפצפו בקול



עוונם תכבש / וצירם תחבש / ועמך אל יבוש / בעתר בקול



פדה נא עמך / והקם נאמך / והפר זעמך / ממנעימי קול



צדקה תעשה / לעם על נושא / ופשעם תכסה / ובשרם בקול



קבץ פזורה / אים ונורא / וקנא לתורה / נתנה בקול



רפא נא שברה / וחבש מזורה / ויזרח אורה / ורוממה בקול



שעה נא לעם דל / ועוד אל ידל / וכחך יגדל / וחזקהו בקול



תחדש עדיך / לבאים עדיך / אזי אני אודך / בתרועת קול.
