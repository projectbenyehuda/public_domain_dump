


	
	אתה אל קדם
	








אתה אל קדם / הנוצר אחור וקדם

עדתך קנית קדם / אלהים מלכי מקדם





אתה בימינך טפחת שמי ערץ / אף ידך יסדה ארץ

לך שמים אף לך ארץ / בורא קצות הארץ





אתה גבהת מים בעליה / ומהם כנסתה גנזת בתחתיה

ובדברך הכל נהיה / גדול העצה ורב העליליה





אתה דורש גלויים וסתורים / ונקראתה משפיל ומרים

בורא רוח ויוצר הרים / דובר צדק מגיד מישרים





אתה השם לאדם עין / [נעלם מ]כל עין

ומשוטט בכל עין / הנותן רוזנים לאין





אתה ונופח באדם נשמת רוחו / מכין מצעדי ארחו



[מ]שפט עורך לנכחו / ומגיד לאדם מה שחו





אתה זוכר הברית והחסד לאלף דור / וממשלתך בכל דור ודור

כן פצתה לאבי גדור / זה שמי לעולם וזה זכרי לדור דור





אתה חי ומחליף לקוייך כח / [ו]משרתיך גבורי כח

נותן ליעף כח / חכם לבב ואמיץ כח





אתה טהור עינים אלהים חיים / רחמן [ומ]לך חפץ בחיים



סודך גלית לברים ונקיים / טוב חסדך מחיים





אתה יחיד ושמך מיחד חוקר ל[ב] / רואה כליות ולב



הרופא לשבורי לב / יודע תעלומות לב





אתה כסאך לעדי עד קים / [אתה] מושל בגאות הים

ויסך בדלתים ים / כונס כנד מי הים





אתה לעולם שם כבודך ומרומם / ועל כל תתגדל ותתרומם

דור מבול מחית יקומם / לוכד חכמים בערמם





אתה מכסה [ה]שמים בעבים / וממטיר נטפי רביבים



מכלכל רעים וטובים / מונה מספר לכוכבים





אתה נותן למלכים תשועה / ומאזין נאקה ושועה

רוכב שמי שבעה / נוטה שמים כיריעה





אתה שמ[ת בי]ם דרכו / ובנעלים נמת [להדריכו]



[ ]שליכה שמה בתוכו / סופה וסערה דרכו





אתה על שמים ידיך רומה / נותן עז ועצמה

גבור ועושה נקמה / עוטה אור כשלמה





אתה פותח את ידיך / משביע לכל הבריות מאוצרותיך

מאד עמקו מחשבותיך / פלאות עדוותיך





אתה צור ומעמיד עולם / ודבר ממך לא נעלם

יודוך עמים כלם / צדקתך צדק לעולם





אתה קנית קנין קדומה לדרש / נסוכה מעולם מראש

לך יי הממלכה והמתנשא לכל לראש / קורא הדורות מראש





אתה רחום משען ומסעד / וחסדיך עד עולמי עד

המביט לארץ ותרעד / רם ונשא שוכן עד





אתה שמעת לאסורים פקח קוח / ומשרתיך אין שכחה לשכח

צועה ברב כח / שדי לא מצאנוהו שגיא כח





אתה תקיף ידך רמה / יהב דע[ה] וחכמה



[ מה / תולה ארץ על בלימה].
