


	
	שנאני אלים
	








שנאני אלים / חצובי להבה

לז ללז שואלים / איה מקום המרכבה

מעריצים ומהללים / דגול מרבבה





שרפים / אוזרי עז / עטויי הוד כלילו

עומדים / בכם מפרשז / בחתלת זבולו

ממעל / גלגל / ומתחת גלילו

לו / דגל ישאו / ויסבו להללו

שרפים עומדים ממעל לו





שש כנפים / הם / מרעידים בפחד

שש / וארבע פרודות / לארבע חיות יחד

כנפים / זקופות / ופני חד אל חד

לאחד / חוקרים שיר / וזמר בלי כחד

שש כנפים שש כנפים לאחד





בשתים / טסים / סביבות כס מכוניו

יכסה / יקר עזו / מצבאות המוניו

פניו / כמו יבקשו / נסתר בחביוניו

להט יקוד פרגוד / בינימו וביניו

בשתים יכסה פניו





ובשתים / מחנות / מימין מהלליו

יכסה / נגה שמאל / בשתי מחנות זבוליו

רגליו / סביבי זיו / ניצוצי חיליו

ערכם כצפות בזק / רצים ושבים אליו

ובשתים יכסה רגליו





ובשתים / פארות / זיו הולך וטופף

יעופף / ציץ כסא / ועלימו יחופף

קרן עזוז חביון / סביבותם ינופף

רפיונם צפוי / נסוגים להסתופף

ובשתים יעופף





וקרא זה / שנאן / למול שנאן במשמר

אל / תקיף ירוצון / בכל מהלל וזמר

זה / שלש ושתים / וזה אחד במגמר

ואמר / לז שני / ללז חק לא יומר

וקרא זה אל זה ואמר





קדוש / מארבע / חיות נשאות

קדוש / הוא בארבע / מחנות נוראות

קדוש / באופנים / עינים מלאות

יי צבאות / רם / בסוד שרפי פלאות

קדוש קדוש קדוש יי צבאות





מלא / יצוקי חצצים / וחשמלי גדודו

כל / היום ירוצון / זמר להענידו

הארץ / וחילה / התלויים בידו

כבודו / דובבים כן / הגות הכל בידו

מלא כל הארץ כבודו.
