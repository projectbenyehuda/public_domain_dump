


	
	שעה זכרון דתי
	








שעה זכרון דתי כמעשה / למעז היה לי ולמחסה / ועל כל פשעי תכסה



- - - -



אלה שוכן מרומות / אלף ציר אחרי מות / שני בני אהרן

בוא אל יבוא בכל עת / בקדשי פן יבעת / דבר אל אהרן

גמור יגמר בחיל / גשת פר ואיל / בזאת יבוא אהרן



דלג ורחץ מכבד / דרש בגדי הבד / והיו על אהרן



הכין שעירים ואיל / הגיש פרו בחיל / והקריב אהרן

והודה על חטאתו / בעדו ובעד ביתו / וכפר אהרן

זמן שעירים להעמיד / זוג גורלות יצמיד / ונתן אהרן

חטאת לשם שפר / חזר אצל הפר / אשר לאהרן

טפל ודוי מטו / טל פרו ושחטו / ויפן אהרן

יקח מחתה גברת / ימלא חפניו קטרת / ובא אהרן

כמס ושפך נחוץ / כסה ויצא לחוץ / ויקרא אהרן



[ל ל אהרן]



מזה שבע להעיר / מהר לשחט שעיר / ויעש כן אהרן



נגש לטהר אריאל / נצב להזות כאל / וכלה אהרן



סבב להקריב שעיר / סטים עליו יעיר / וסמך אהרן

עוון נסלח ונמלח / עת לעזאזל נשלח / על פי אהרן

פשט בגדים לבשם / פנה והניחם שם / את בגדי אהרן

צהבי זהב העטיר / צעד חלב להקטיר / ויקרב אהרן

קדם להקריב עולה / רצף לעצמו כלולה / זה קרבן אהרן

שלמה עבודת כפור / תחשב כקרבן ספור / על כפי אהרן

שדי כמאז תציץ / להשיב ספירת הציץ / על מצח אהרן

מהרה תמשח לגן רטב / היותו כשמן הטוב / על הזקן זקן אהרן.
