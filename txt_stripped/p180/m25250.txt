


	
	שמך יגדל
	




שמך יגדל / ולא יחדל / גדלך מפי אימה

להעצימך / בהדומך / ירוצון דרי אדמה

מלך על / אשר פעל / הכל בדעת וחכמה

המוניך / אמוניך / עונים ואומרים באימה

יתגדל / ויתקדש / שמה רבא בעלמא





לך אקרא / ברב מורא / ואגיד גדלך נעלה

וגם חכי / יצפצף כי / לך יי הגדלה

כבודך / והודך / אחוה יום ולילה

ומקהלך / בהיכלך / מתנים צדקך המה





מקדשים / ותרשישים / יברכוך במקהלים

יבקשו / וידרשו / איה כבוד אל אלים

ויפחדו / וירעדו / בעת אי כבוד אל שואלים

כי רמו / וגם עצמו / מעשיך הגדולים

ויעידו / ויגידו / צדקתך העצומה.
