


	
	אנושים וענושים
	








ובכן מה אדם ותדעהו בן אנוש ותחשבהו

יי מה אדם





אנושים וענושים בקשים / אמלים ועמלים במרים

אף רואיהם יאמרו אים / אזלו מים מני ים / אך הבל בני אדם





נפלים שפלים נטמו / נודדים הבל יהמו

נצחם וכחם יתמו / נמשל כבהמות נדמו / נבער כל אדם





יום יום זדנו ואשמנו / יחדו בשנו ונכלמנו

יעצנו רע ותך יזמנו / יחפשו עולות תמנו / יצר לב האדם





שוקד גדיש ורגב חובש / שוכן גוש ורמה לובש

שוה לחציר גגות עבש / שקדמת שלף יבש / שלא ימצא האדם





לקוחים לדבר אתם עזות / לתת להם בתפת אחזות

לכודים כצפרים האחוזות / לו חכמו ישכילו זאת / לבות בני אדם





מועד ומוכן לקיטור גחל / מתקן ונסכן לתגרת שחל

מאוס ימאס חייו בבחל / מתקו לו רגבי נחל / מה יתרון לאדם





המונם סג וישר אין בם / הוות בקרבם ועמל בארבם

המבין חבם וחופש קרבם / היוצר יחד לבם / השקיף על בני אדם





בבוא קצו תסערהו / באמת בצעו תנערהו

בחרמו מות יגרהו / בור כרה ויחפרהו / באשר הוא סוף כל אדם





רעוץ ורצוץ חליו ועדיו / ראוי לשלם נשיו בקשיו

רגבים יבלו שרשיו ופריו / רואיו יאמרו איו / רבה רעת אדם





במלאת ספקו עצר יניאנו / בלע חיל ויקיאנו

בדד ישב ולא יראנו / בשלום שודד יבואנו / בקום רשעים יסתר אדם





יקרו נשחת וקרנו נגדע / יחידתו משחת בל יפדע

יום אשר אליו יתודע / ישלם אליו וידע / יודע מחשבות אדם





ידד כחזיון לילה בחמה וסער / יסער כמוץ בסופת יער

יגעש בגעש כישיש כנער / יחד כסיל ובער / יוקשים בני האדם





הנה אם נחפש ונבדק / הושת כלו סג ונחדק

היה יהיה כאבק דק / האנוש מאלוה יצדק / המלמד אדם





וזרים בנוהו יאהילון / ונכרים באוצרותיו יגילון

והוא לקברות יובילון / וכאורח נטה ללון / ויחלש ויגוע אדם





דל לא יחדל מעוון ואשמה / דתו און עצתו תרמה

דרכו עול נתיבו ערמה / דברי פיו און ומרמה / דברת בני אדם





הובר אשמים בצרור נחתמים / הרי פשעים עליו נכתמים

המס ימס בגחלי רתמים / הנה בהיותו תמים הבל כל אדם





גזור מטיט ועל טיט נסמך / גוו פתאם יתר וימך

גוש עפר עני ומך / גר אנכי עמך / גם בני אדם





בתקף עליו כידור מלך / בכן ישוב לארצו כהלך

בשובו שם ערם כחלך / בהבל בא ובחשך ילך / במות אדם





יעזב לאחרים רכושו / יצועו רמה ועש לבושו

יקר פדיון נפשו / ישוב עמלו בראשו / יום ענות אדם





רודף בחייו רוח נשוב / רודה במותו חמת עכשוב

ראוי כאפס ואין חשוב / רוח הולך ולא ישוב / רוח בני האדם





ומרב מצוקיו בחייו ירב / ומותו ערב ויומו עורב

וקצו בקרבו לנפשו אורב / וילך הלוך וקרב / ורחק יי את האדם





לריק ייגע וידל מאדו / לשוא יעמל ויהבל בעודו

לכן לא ישא מאומה בידו / לא ירד אחריו כבודו / לא יכון אדם





חדשיו חצץ שוכן שמי ערץ / זעוך נזעך ונקרץ קרץ

קמוט נקמט לכלה וחרץ / זכרו אבד מני ארץ / זה חלק אדם.
