


	
	עטה הוד
	




עטה הוד ועדה ולבש גאונים / והנחם והרף מיגונים



ודם אם בבכי תלין וקוה / הלא תקום ולבקר רננים



ודע כי הנך יחיד בדורך / ושירך כחרוזי
צורונים



עניתיהו: ואיך ארף ואדם / ואיכה ידמו כים
שאונים



אפפוני הדפוני מצוקים / וחצי הזמן נגדי נכונים



ויורוני בנות ימים כאלו / אני שערה והם קלעי
אבנים



ומכנפי זמנים יאחזוני / ועלי נוסעים תמיד וחונים



ואדע הזמן היטב והוא לא / ידעני אבל כי ביגונים



ולא ידעו ילדיו כי לבבי / מאוד חזק והיה כאבנים



ולא ידאג ולא ישם אליהם / וישחק אם יבואוהו
המונים



אני הבן אשר טרם יילד / לבבו בן כמו בן
השמונים



וגוי יהלך על האדמה / ונפשי תהלך על העננים



ויתר מעלות חכמה תבקש / ומואסת ברב עשר והונים



ויגעתי בחכמה מנעורי / למען אחריתה מעדנים



והיא היתה אחותי מנעורי / ומודעה קראתני בבנים



ונפשי מכלי חמדה יקרה / ותחשב הפנינים
כצנינים



ומראש מאסה נעם אדמה / וחייה בעיניה אסונים



למען כי ששוניה יגונים / ומדוחות, ואוניה
כאונים



ובתי מחמודיה הריסות / ובתי מהלמיה נכונים



תפתה כל חסר לבב ביפיה / כבעלת אוב ואשת
ידעונים



בשניה תשנה המצוקים / וזאת לא זאת תשלח
בזמנים



ותשית רוזני היום כמו כן / מחסרים בדעותם
ושונים



לקחם על זרועותיו - ודכם / ואם היו עלי חיקו אמונים



והתל בחבורתם ושחק / לעמתם - והראמו ששונים



ראה עתה, אחי מוסר ובינה / והבט יום כלותם
כעננים



ויום יחשפו בו מעריהם / ויראה כל אנוש מהם קלונים



חמתי נתכה עלי בשורי / פתאים חשבו כי הם
מבינים



ואשתק מעדתם הנבלה / ואכבד ואשקיט השאונים



והם מתפארים עלי בשירים - / ומי יתן והיו
מאזינים



ולו התפארו עלי מאורי / שחקים - החשכתים
כעננים



פתאים חשבו סכלות להשוות עצי עדן כשורות
הארנים



הכאלה לפני יערכון / צעירי הנמלים הקטנים



אשר לא ידעון בשיר לדבר / ואינם להגות בו
נאמנים



ואלו יעלה שירם במשקל / אזי יהיה כאבק ועשנים



וטרם ילדו כרו קברים / לשיריהם ועמהם טמונים



עדרים מאסו רועה אמונה / ולולי אל טבחתימו כצאנים



ואלו נפלו ידי, עליהם / נשאתים - נפלו הם ביונים



וכלם הטבעו בבץ צמיתות / ולא יכלו שאתם העננים



ואולם מפני שירי ישובון / אשר יכנעו אליו
גאונים



ויתרצו לכפר השגגות / ויתודו להעביר העוונים



ערכוני והם לא ידעו כי / אני שמש אכסה במעונים



אני אחקר צפוני המליצה / ואפתח שערי דעת ובינים



ואקבץ מנפוציה חרוזים / ואלקט מפזוריה פנינים



וגבנני הרריה שרונים / לפני, וארחיה נכונים



אני באתי לחדריהם אשר הם / לכל מבין וכל חכם
צפונים



ואשיר [שיר] ישמח הנפשות / והלבבות יחלץ
מיגונים



וכל לא ימצאו עמי ידיהם / ויודו לעצתי בלשונים



וישובו מקנאי מעלתי / מלאים מחמת אש החרונים



ואת חחי באפיהם אשוה / ואתן על לחייהם רסנים



ואשחקם ואדרך במתיהם / ואשביע פניהם
בקלונים



ויהיו כנערת מפני אש / תלהט בעזוזה החסנים



ופי כל דוברי שקר יסכר / וימותו באין כח ואונים



ולא ירום לבבם לאמר שיר / ואף לא יזכרוהו
בלשונים



והם לא יעלו על לב לעולם / וכמתים בחייהם טמונים



לאידם השחוק עשו בנפשם / בשומם כעצי קש
קנמונים



ואמנם לא לעולם יהיה אור / כמו אפל, / והיום
כאשונים



ושירנו כילודי אמנה / ושירתם כמו ילדי זנונים



וכי נפשם קרוצה מאדמה / קרצוה רגבים מדמנים



ונפשנו פניניה ועל כן / גזרנוהו אנחנו
מפנינים



ובו ארום ואנשא עלי כל / מתי דורי ועל כל
הזמנים



ואתגבר בשירתי ואחבש / פני שונאי לעולם
בטמונים



אשר המה ערירי לב ורעיון / ואינם שומעים כי אם
מאנים



ודת אל חשבו בה מחשבות / ושגו בה והרבו עוד
זדונים



ומתכנת לשון עברית עמקה / וקשה לאחי מוסר
ובינים



ובינה מפליאי התבונות / ואינה כי לחדי רעיונים



כבודה היא לנכבדי תבונה / ואף כי לנמלים
הקטנים



אשר אין די בבינתם להפלות / צעירי הזבובים
משמנים



אהה כי הזמן חסר תבונות / ונמלא מעדת רוגנים
ושונים



ועל כן כל ימי חיי אקונן / ולבי יעשה מספד
כתנים



למען נעדרו אנשי אמונה / וספו כל נשואי שם
ופנים



ותחת ההדס סרפד וחוח / ובמקום הארזים - קמשונים



ואם התמוטטו גבעות תבונה / ואם התפוצצו הרי
אמנים



הלא טוב השאר נשאר וזרח / כמו שמש עלי יום
העננים



וחשב האלהים זאת לטובה / להשאיר בין שאר
עמו אמונים



לשובב יעקב אליו ולאסף / פזורי דת לאל נוטה
מעונים



ולבנות נגדעי הנהלאה / ולסאן שופטי הרע סאונים



לנחם לב שפלים ואמלים / ומכותם לרכך בשמנים



ולרעות צאן מנדח באפסים / והרזים לעדן כשמנים



אמת כי נהלם על מי מנוחות / והובילם לחיי מעדנים



והרום בתבונה עד לשבעה / והשקם כי הרגום
צמאונים



אשר חזק ידי דת אל אמונים / והעמידה עלי ראש
המכונים



אשר השיב פזורי צאן תעודה / והושיב נוחליה
מלפנים



אשר הודע בעטו משפטי אל / ואליו יעלו עמו
נדונים



וכמה תוצאות משפט יגלה / בחכמתו ויתיר
האטונים



והוא בעל לשון דת הקדושה / ופיהו הממלל
כנבונים



והוא פתח חגוריה, וחלץ / חלוציה, וחדש בה
ישנים



ויסד משגוביה בצורים / וחשף מסתריה גאונים



והחיה משחותיה מקורים / והפיץ מנפוציה עינים



והעמיד מפרוציה שערים / וחלק מחלקותיה לפנים



נשוי קדקד, ולו שחק ערכו / גערו גערת אבות לבנים



אשר התם והישר לבושו / ועטה החסדים כסדינים



וירש מעלותיו מאבותיו / ויורישם לאחריתו לבנים



אבי הכל נשאהו עלי כל / והרים מעלותיו כתרנים



והשמיע כבודו בארצות / עדי אפסי שבאים אף
דדנים



והשים את תהלתו בפי כל / מתי מוסר, גדולים עם
קטנים



ואמנם הוא לכל חכמי אדמה / כמו שמש, והמה -
כעננים



וכמוני יאהב כי אשוה / תהלתו בפי כפעמונים



ישירוה מתי מוסר ויהגו / בצדקותיו ימי עולם
ושנים.
