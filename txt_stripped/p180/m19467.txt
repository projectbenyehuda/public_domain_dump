


	
	[אשר תשב שכולה כנהרות]
	



ומן מטאולאת קצאידה ואג͘זאלה קצידה ירת͘י פיהא בעץ͘ אלמפקודין [ומשירותיו
הארוכות והנמלצות ביותר שירה שבה קונן על אחד הנפטרים]




אשר תשב שכולה כנהרות / דמעות על לחייה פזורות

ותאנח ויד תשים ללחיה / והיא מרה כהמר על בכרות

ופניה ידעתים כמאורות / ונהפכו בבכיתה שחרות

למה תבכי כלילת הנערות / ולא מטל ענניך עצורות



עלי פרוד ידידיך כאלו / גויתם לפניך קבורות

ומדי חי אנוש רעיו יקוה / ואין מקוה לשוכב בקברות

בכי בכה עלי חיים ומות / אשר נפל באחד היערות

ואלו מת כמות רבים בביתם / ויקבר בכבוד והדרות

אזי היה ללבי קט הנחה / ונחמים וחבש לשבורות

ולא ימות מטען בחרבות / ובמרוצות אשר כאש בעורות

ולא תעש גויתו כברה / וידקר בכל גופו דקרות

אשר נשלח היות עזרה בצרות / והלך להדף חילי מצורות

טרפוהו הדפוהו לבאים / והאריות וגורי הכפירות

אשר עמדו ורמחיהם מרוקים / וחרבותם שלופות מתערות

חרבות נֽשכות כאש תלהט / תגרר האבנים כמגרות

אדמים שֽפכו דם איש נקי כף / ונפשו הֽרגו מהבחורות

והאח אֽמרו עליו בהרגו / ולבי סר ועופף באוירות

שחטוהו כאחת העגלות / וגם הובל אלי טבח כפרות

והתגלל בדמו בֽמסלה / והרתיח כבשר בקדרות

לדמו רעשה ארץ בשפכו / גבעות רֽגזו עליו והרות

ולו ראה וידע מחשבתם / אזי בא מפניהם במערות

אבל עת־בא פקדת איש ויומו / הימלט ואם ימצא עזרות

ואין שלטון ביום מות לאדם / ולא יעבר אנוש מיום עברות

אהה איך התשה חכמה כחומה / אשר היתה בצורת הבצורות

וחומת־אש סביב כל הגבירות / ומגדל עז עלי כל העירות

והר תבור מחזק לב חשולים / ולצמאים בצחצחות מקורות

וצוף אהב ורוש אויב אבל כי / בכל כחו אמת המתיק מררות

תמים לבב ובר מכל נבר לב / אשר אחז אשורו בֽישרות

למען יֽדעיו חלו לשמעו / ואחזו ציר כצירי הֽמצרות

וקינתם בדמעתם עשתם / בחיתם תמור יין ושירות

ותחת מעטה כבוד והדר / חגורים מלבוש שקים חגרות

חפויי ראש מקום הוד העטרות / ושית אפר עליהם ועפרות

ושבת דם בלב יחם כתנור / מקומם מעליות הֽמקרות

וכל בנות עדינות שאננות / אשר ברך ובתפנוק זכורות

מקוננות הן ואפיהן גדודות / וכל ערב וכל שחר שחרות

ומר צרחו ורוש לען במות ראש / לכל ראשים ושבעו ממררות

וכוס רעל תמורת הֽרעלות / חבש חבל מקום שרים ושרות

והן הולכות וזו אל זו סמוכות / כיולדות בצירים לא עצרות

ובכות על אדניהן אביהן / והומיות זקנות עם נערות

וכלן נֽשאות קולן בחילן / כאלו יֽלדו אתו עקרות

ואיכה לא יבכו המאורות / לאור תבל אשר כבה נהרות

וכמעט קט ותבכינה אבנים / וכמעט קט תפוצינה נהרות

בכו עליו קהלות הֽישרות / להוציא מעיניכם כֽיאורות

ושימו יום כיום גלות בכיתו / ויום אבל יהי לכם לדורות

ומהרו נא וקראו לחכמות / ושלחו עוד והוסיפו אחרות

וספדו לו כמספד הֽמדקר / מחדש בית אלהים בטהרות

וכתבו אל פלילי דת גאונים / עגלות כחצי גרן ושורות

ושלחו שיר אליהם חברו שר / דבריו מבדלחים יקרות

אשר בחן אמרות הצרופות / וגם הלבין וברר הברורות

ונשקלות בפלס מחשבות / ומאספי תעודה הן אגורות

תמימות הן כמשפטן וחקן / כמתכנתן ואיניהן שבורות

ישיחוהן קציני החבורות / בשיחתן כחק דתות ותורות

אשר יטו בחנם הלבבות / וכל נפש ישנה הן מעירות

בכות על איש אשר הסיר משוכה / ביום סורו ופרץ הגדרות

ועל ארז אשר היה מגורות / אשר האריך סעפת לו ופארות

וכל עוף שֽכנו תחתיו וקננו / ושמו בין סעפתיו מדורות

כזית רענן היה ונכרת / ביד איב ושרשיו עקורות

יעקר עֽקריו האל מעקר / חמשת המלכים במערות

נתשוהו נתצוהו סערות / אחזונו לשד שברו שערות

למענו צללו אזנות סגורות / ועליו רעשו בתים וקרות

אספו אל ורוחו הטהורה / ושמו בנפשות הטהרות

ונרו יעלה תמיד כמו אור / עלות שמש עלי כל המארות.
