


	
	מי כמוך שוכן בגבהי
	








מי כמוך שוכן בגבהי מרומים

שומר נפשות יקומים / נוצר האדם





מי כמוך לובש בעזו צדקות

מגלה פלאי עמקות / יודע מחשבות אדם





מי כמוך מושל בגבורתו עולם

מחיה יצורים כלם / ויוצר רוח אדם





מי כמוך הנותן לכל חי נשמה

ומשים אותותיו באדמה / ובישראל ובאדם





אזכיר בפחד ומורא / גדלת אדיר ונורא

אליכם אישים אקרא / וקולי אל בני אדם





ברא רוכב ערבות / שלשים ושתים נתיבות

אמרות צרופות קצובות / להודיע לבני האדם





גזר בכסאו לקשר / ארבע חיות במישור

נשר אריה ושור / ודמות כמראה אדם





דק וצבאיו תכן / גלגל עגל מעשה כן

וצבא במרום שכן / והארץ נתן לבני אדם





האדמה חציר הפריא / ועוף במרום המריא

ועץ עושה פרי / ועשב לעבודת האדם





וכמו השלים ענין / ותמלא הארץ קנין

חתם תכנית בנין / ויברא אלהים את האדם





זרח עליו הודו / והשליט בכל ידו

ונטע גנת יסודו / וישם שם את האדם





חנון צלעו סלה / ובשר תחתנה מלא

והפאיר צורת כלה / ויביאה אל האדם





טהור עליו צוה / הרחק מעץ תאוה

השמר ושמר מצוה / כי זה כל האדם





טוב טעמו הפר / ועלה תאנה תפר

השיבו שדי לעפר / ויגרש את האדם





יהיר נמאס ערכו / ויקם אל אחיו ויכו

וכל בשר השחית דרכו / ויאמר יי אמחה את האדם





כלה דור צוחן / תוכן לבות ובוחן

ונח מצא חן / בעיני אלהים ואדם





לב עליזים גדל / לבצר מעז ומגדל

וירד יי לראות את העיר ואת המגדל / אשר בנו בני האדם





מגדל עזם הפיל / ובלל שפת עם העפיל

והפיץ אותם והשפיל / עיני גבהות אדם





נגה אזרח ונהר / ובא מעבר הנהר

והוא בן שלש נזהר / ויחכם מכל אדם





סמך את ידו / על ראש בנו יחידו

קראו צור ילדו / יפיפית מבני אדם





עבר במקלו נחלים / תם יושב אהלים

שב ובידו דגלים / נגד בני אדם





פרצו לרב בארץ / ובני מף חרצו חרץ

לאכל עניים מארץ / ואביונים מאדם





צעקו מן המצר / לאל העונה בצר

הבה לנו עזרת מצר / ושוא תשועת אדם





קדוש גאון תעדה / והדף רשעים ורדה

אך פדה תפדה / את בכור האדם





ראה צרת בניו / ושלח מלאך פניו

והאיש משה ענו / מאד מכל אדם





שדי חזק זרועו / ואותות רבות הודיעו

ומצרים אמרו לפרעה / למה תשמע את דברי אדם





תנין איפה חכמיך / אשר דרשת מימיך

ואיך יצדקו חרטמיך / וחרשים המה מאדם





אל אלהים לקחני / ובדרך אמת הנחני

לבשר ענוים שלחני / נורא עלילה על בני אדם





נוטה שמים לבדו / ורוקע אפסי ממדו

שאול ואבדון נגדו / אף כי לבות בני אדם





יעמדו נא ויושיעוך / ההוברים קדקד ירעוך

הלא בלהטיהם יתעוך / אשר יעשה אותם האדם





שני רשעים אשברה / ויספתי אותם ליסרה

יי לי לא אירא / מה יעשה לי אדם





לכן שמעה דברי / אל תאמר לי יאורי

שלח בני בכורי / כי ליי עין אדם





מאנו תת פדיום / עד כי שפטם אל אים

שפטים אשר לא נבראו למן היום / אשר ברא אלהים אדם





הפיל צור פחדו / וגאל ישראל עבדו

יודו ליי חסדו / ונפלאותיו לבני אדם.
