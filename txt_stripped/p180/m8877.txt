


	
	[פקד ברק-שחק באש יקדת]
	



וקאל איצ ͘ א

— — È — / — — È — / — — È — // — — È — / — — È — / — — —


פקד ברק-שחק באש יקדת / עין נדדי לֽילה פקדת



פרש כנפיו על-פני-שחק כמו / שדה תפזז בלבוש רקדת



יחג וינוע כחולת אהבה / וכנערה לא-נסתה ללדת



ברק אשר לא-אחזה עד-אחזה / הלֽילה בלֽילה אבדת



חלה פני חשק פני דודים אשר / ישחו בים-חשק כמו רפסדת



יברק בשורו בת בתולה תהלך / בינם והיא כי-תעלה ירדת



תניד בראש כי-נשאה על-שעלי / משקים כשר משקים ביום הלדת



תזרח כמו תנין בטלה ותהי / כאש בעיני על-כפור שקדת



מזרק חמתו בערה בו בעבור / יחפץ להסתירה והיא מרדת



כחש בשרה בכלי ואחשב / בת-הכלי מאין כלי עמדת



תמשך לבב-שמש בחוט אהב עדי / כי-תעמד על-כף אנוש סגדת



רעי קראוה מכשפה הכי / תצוד יגוניהם בלי מלכדת



ובמעלות רוחי יגונים גערה / בהם למען מאנו לרדת



ילדה אדמה רבבות בנים עלי / ברכי ועוד לא-עֽמדה מלדת



שובה מידעי אשר הרחיק נדד / נפשי כמו צפור לך נודדת



לא-לֽמדה לנוד למיום נבראה / אולם לעופף נגדך עמדת



לא-נֽטשה על-רב זמנים אהבת / רעים אשר היא על-אמת נוסדת



תהגה לשוני צדקך תמיד הכי / מאין יגיעה חסדך מגדת



קח זאת אשר רעית ברקים נקראה / כי פאתי מדבר וים עקדת



יחם לבב- סהר בעת כי-יחזה / אורה ושמש תארה חמדת.
