


	
	לבי שמח
	




לבי שמח / ובשיר פצח / דברך



בית גן פתח / סב נא וקח / מרך



ורד קטף / שושן חטף / והדס טרף



זמר בתף / עת יערף / אורך



וביד תפש / מזרק וכוס / ומנת וכוס



קח דוד, ושוש / תשיש בכס / שריך



ששון דרש / יגון גרש / גיל תחרש



וכמו ברוש / תהיה לראש / ממריך



רן תענה / ובמחנה / בין תחנה



חכמה קנה / תאיר פני / חורך



תשיר בשיר / אל הגביר / איש תם בחיר



ושם וניר / לו מאביר / צורך



כי על שמו / הוד טעמו / זיו נעמו



צוף טעמו / ערב כמו / יערך



מה נהדר, / נקי ובר / לו מאמר



ראש ליקר / קצין ושר / דורך



שם גיל לגיל / מחכים כסיל / גם ככסיל



על הגליל / יאיר שביל / שירך



עץ רענן / ציץ נעמן / ציר נאמן



מתוק כמן, / הוא בזמן / שמרך



דובב בניב / יצית סביב / צריו שביב



אך לו רביב / מאיר נתיב / אמרך



הוא היסוד / אל רז וסוד / ענו מאוד



לו לעבד / הן יחמד / סהרך



לו חן צבי / אימת ארי / ובפיו צרי



בו בחרי / יך בחרי / עריך



יעלה בלי / סלם תלי / מליו חלי



הם לך ולי / חובש חלי / שברך



ובמעמד / המחמד / עמד לבד



לו שם ויד / עוטה כמד / אורך



אור שחרו / מתארו / הוד סהרו



מצהרו / מזהרו / זהרך



טוב מעלל / ענו, שפל / רוח, אבל



עלה לגל/גל רום ועל / שיריך



דעה יחו / שוקל בקו / גן בין ירו



פריו תאו, / אם בו תקו / אשריך



עיניו סגור / נאה כתור / כל עת ותור



עופף כתור / מוציא סגור / הריך



לדת כקן / לבו, ואין / בו תך, וחן



מצא כבן / למך בעין / צורך



אדם וצח / הוד דוד ואח / גיל נאנח



לבי שמח / ובשיר פצח / דברך.
