


	
	שוממתי ברב יגוני
	








שוממתי ברב יגוני / ליום יפקד זדוני / מה אמר לאדוני

אמללתי ונאלמתי / בזכרי אשר אשמתי / בשתי וגם נכלמתי

בהבל כלו ימי / מפני בשת עלומי / אין שלום בעצמי

גחלתי קודחת / כי מגלה נפתחת / והנושה בא לקחת

דבקתי במחשכי / ונפשי לא ידעה כי / גר ותושב אנכי

הוי כי יבוא יומי / אזי איקץ מחלומי / ואשובה אל מקומי

ועל חטאי אשר עבר / ועל פשעי אשר גבר / מה אשיב שולחי דבר

זדון לבי השיאני / ועוון אשר הלאני / מבטן קראני

חושב בנפשו נבל / כעץ שתול על יובל / והוא לקברות יובל

טפל שקר בתוכו / ונטה ממהלכו / ויפן כה וכה

ישלך כאבן דומה / ולא ישא לבית מהומה / מכל אשר לו מאומה

כחו לא סמכתהו / עת נפשו נשאתהו / אף כי אש אכלתהו

לכדתני אשמתי / הלא זה דבר נשמתי / עד היותי על אדמתי

משוגתי נשאתי / ובלבבי קראתי / אנכי חטאתי

נכוחה לא חשקת / ובשרירות דבקת / מה לך כי נזעקת

שאי נפשי אשמתך / ראי חטאך לעמתך / גם את שאי כלמתך

עת אדאג מעווני / השיבוני רעיוני / נפלה נא ביד אדני

פנה ממכון שבתך / ופתח לי דלתיך / כי אין בלתך

צור הגן בעדני / ומעוונותי פדני / ארחותיך למדני

קולי שמע כחסדך / ביום אעמד נגדך / אל תט באף עבדך

ראה עניי וענני / אני בידך הנני / ואתה יי חנני

שלח אמתך וחסדך / לעם צועקים נגדך / ולי אני עבדך

תסלח לאשמינו / ואל תפקד עלומינו / כי צל ימינו.
