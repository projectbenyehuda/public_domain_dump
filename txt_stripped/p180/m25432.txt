


	
	שתי פעמים
	








שתי פעמים / מקימים / בכל יום על המוננו

ליחד צור / בלב עצור / ובהגיון לשוננו

ממקומו / ממרומו / ישלח אור לעינינו

הוא לבדו / יצו חסדו / ויאזין קול רנוננו

בקראי שמע / ישראל / יי אלהינו





שאר יעקב / לך אקב / יסוד דת אל אשר נתן

לכה ושמע / תורת שמע / אשר נחלו בני איתן

מצותה / לקרותה / לנוחלי גן הביתן

הגות ערבית / אל תשבית / כתנאים במשנתן

בהכנס / כהני נס / לאכל בתרומתן

רשות נועד / לקראה עד / עלות שחר כמו חתן

יסוד ארבע / לה נקבע / ולעד לא להכריתן

הן ערוכות / שתי ברכות / לפניה כהלכתן

וחק בריתה / לאחריתה / שתי ברכות להורותן

דת וחק / מרחוק / הגות בה על מלוננו





שתי ברכות / נערכות / בראש שחרית להתהלל

לאחריתה / כבריתה / סמך אחת להתפלל

מסוכתה / והלכתה / עם הנץ מאור הילל

הלא נסעד / וגם יעד / שלש שעות להתחולל

בעת הלך / כאן ואילך / לא הפסיד ולא חלל

רוחשים כה / איש כדרכו / והלכה כבית הלל

יד יפסק / אשר יעסק / בראש פרקים להתמולל

היה רץ / לא יחרץ / אבל יעמד ויפלל

וכן יפעל / קרא עד על / לבבך להסתולל

דין נשמע / קריאת שמע / בכל לשון להתמלל

הלא נודעה / בארבעה / אשר פרשו זקנינו





שם אחד / ירא ופחד / והשמר באותותיו

לבד החת / אל תשחת / והארך דל כחקותיו

מאריכו / להמליכו / על ארבע רוחותיו

המפחד / אל יכחד / באמצע שאלותיו

ברית יקים / בין פרקים / להשיב לכל שלומותיו

רוח יקים / בין דבקים / וידקדק באותותיו

ירא ועשה / כרב יוסי / והאזנת למצוותיו

הקורא / אל יורה / באצבעותיו ושפתותיו

ואם טעה / בארבעה / הלא יחזר לראשיתיו

דת ספק / להתאפק / ויחזר לראש קריאותיו

הלך אן / ומצאן / שהן קוראין לעמתיו

יהי עסוק / בפסוק / הראשון כדיננו.
