


	
	שנאני מרומים
	








שנאני מרומים / אדרך יראו

ולהקת הדומים / בשמך יקראו

מאז הבקר / גשים יתראו

יהללו את שם יי / כי הוא צוה ונבראו





שנאני מרומים / דרים ברום תקר

ולהקת הדומים / הוגי שיר הבקר

מאז הבקר / וערב ובקר

גדול יי ומהלל מאד / ולגדלתו אין חקר





שנאני מרומים / זקי הלתך

ולהקת הדומים / חשובים סגלתך

מאז הבקר / טועמים גדלתך

ואני תמיד איחל / והוספתי על כל תהלתך





שנאני מרומים / יד ליד תסלסלך

ולהקת הדומים / כמו כן תצלצלך

מאז הבקר / לעד אגדילך

אספרה שמך לאחי / בתוך קהל אהללך





שנאני מרומים / מנעימים שיר בקול צעדה

ולהקת הדומים / נצבים ברעדה

מאז הבקר / שוחחים בקול חרדה

אהללה שם אלהים בשיר / ואגדלנו בתודה





שנאני מרומים / עונים איה כבודו

ולהקת הדומים / פוצחלים יחודו

מאז הבקר / צועקים לנגדו

יהללו את שם יי / כי נשגב שמו לבדו





שנאני מרומים / קרני זיו תהלה

ולהקת הדומים / רצופים ברב קהלה

מאז הבקר / שמו יהללו במלה

מרומם / על כל ברכה ותהלה





שנאני מרומים / יד ליד תסלסלך

ולהקת הדומים / כמו כן תצלצלך

מאז הבקר / לעד אגדילך

אספרה שמך לאחי / בתוך קהל אהללך





שנאני מרומים / תפארתו מעידים

ולהקת הדומים / שבחיו מגידים

מאז הבקר / סלסולו מסלדים

שירו ליי שיר חדש / תהל7תו בקהל חסידים.
