


	
	בצרכם כל עצל
	








בצרכם כל עצל / והוא להם כצל / ולמקלות פצל /
וגזר לגזרים

ונשא את ראשם / והזריח שמשם / והרוה את נפשם /
כמלקוש ומטרים

בלבו יסתירם / למען יזכרם / ועינו תשמרם /
נשפים ושחרים

לכן ידע כל עם / אמת כי הוא רועם / וכי הוא מושיעם /
בצוקה ומצרים

וזה מפעל נבר / ומבין כל דבר / אשר בתם גבר /
כזכים וברים

בתורת אל בחר / ומאס כל מסחר / בלי אמור: מחר /
אשנן בספרים

ביען הם אמו / ואביו ולאמו / ומתק מטעמו /
הגומר בפשרים

יזכהו האל / ראות בנין אריאל / ו[יש]וב ישראל /
לקדשו נטהרים

ופיד אח [לא יראה] / ושאה לא ישאה / אבל גאה
יגאה / ויעל כנשרים

והימים האל / כשמום ישראל / ישימהו האל / לדורי
הדורים

עד / לבבנו [סועד] / ואת לנו מועד / בשמחה
[ו]בשירים

ואתה פסחנו / ואתה [כח]נו / ואתה רוחנו /
בגויות ופגרים

ואת חדותנו / ואת תאותנו / ואת תקותנו /
בקום האכזרים

ואתה מלכנו / מישר [ד]רכנו / וממתיק חכנו /
בנע[ם] מדברים



אלהים יפרך / בקרב עירך / ואת כל עריך / ישו
כיקר כרים.
