


	
	אלהים אל ראשון
	








אלהים אל ראשון ואחרון / נעלה על כל גבוהים

איחדנו ואקרא בגרון / בראשית ברא אלהים.





אלהים אדני אל עולם / יעידוהו לבות אמוני

אף מעשיו יודו כלם / אכן יש אדני.





אדני אחד מאז נקדש / בלב נמצא ולא נכחד

אמת יורה כי כל יציר חדש / נתנו מרועה אחד.





אחד אדני רם ונשא / עד לא שפרירי מעוני

אכן הוא פעל ועשה / כל אשר חפץ אדני.





אדני אל אפס בלתו / יודע לכל שואל

אף הוא ראשון והכל בעתו / ואל מי תדמיון אל.





אל אור אז קרא במאמר / ויגרש מאפל ואור

אפדת עז לבש ואמר / יהי אור ויהי אור.





אור ארץ ושחק לילה ויומם / יביעו אמר במרץ

אין קדוש כאל מרומם / אשר בידו מחקרי ארץ.





ארץ במדה יסד וקלקל / חוג מתהלך על אגדה

בעשותו לרוח משקל / ומים תכן במדה.





במדה בראתם תוכן שחקיו / ועמדו יחדו עת קראתם

בהטביע אבן בתוך קו / צפון וימין אתה בראתם.





בראתם בלימה להיות בהו / לעת תחליף כלבוש אדמה

בנטות צפון על תהו / תולה ארץ על בלימה.





בלימה בגבורה שת ממדים / בקרב סגרירי סערה

בהקצותו פנות וצדים / נאזר בגבורה.





בגבורה במאזנים הרים / זרומי תחתיות שנים

בפלס שקל הרים / וגבעות במאזנים.





במאזנים בכחו לעשות הגדיל / בצנור הנגדש בגיחו

ברקיע ביניו הבדיל / מכין הרים בכחו.





בכחו בעתו קרה עליות / מפלג שטף לקול תתו

ברוחו יערפו בציות / יורה ומלקוש בעתו.





בעתו גדולות תכן מזריו / כמוס במו שלג מצולות

גפי רשף העמיק לצריו / עושה גדולות.





גדולות גבורות שת באוצר / אבני אלגביש יקרות

גנז קפאון לעת צר / ודבר גבורות.





גבורות גדול מספר עצומות / בהבדיל תחתיים בדול

גם בקראו למקוה תהומות / זה הים גדול.





גדול גליהם יחדו הגיר / וישבר חקו עליהם

גבול גדר בעדם להסגיר / שאון ימים שאון גליהם.





גליהם גאולים שת וזקק / מחוז יבשת ממצולים

גרפם עד יפלס מחוקק / דרך לעבר גאולים.





גאולים גליו אז לעבר / הוליך בנתיבי שביליו

גזרם בשבט אל גבור / רוגע הים ויהמו גליו.





גליו גבורותיו בם יקוו / להראות שנית נוראותיו

גדלתו אז איים יחוו / ורעם גבורותיו.





גבורותיו דשא דמו להפרות / ועשב לעבודת נקשה

דשני ארץ צצו להברות / ותוצא הארץ דשא.





דשא דשן לאלף מנויים / ומאתים ותשעים לדשן

דעץ גם שלשים שנויים / ומעגליך ירעפון דשן.





דשן דרכיו יזם להפאיר / צור מעז ומחסה לחוכיו

דביר עדן להם התאיר / הן אלה קצות דרכיו.





דרכיו דברו השלים וזרח / שמש וסהר לאור באורו

דצים כגבור לרוץ ארח / לשמע בקול דברו.





דברו דבר לרוץ תמיד / עד בן נון חילים יגבר

דהר מרוצתם העמיד / כי פי יי דבר.





דבר דגלו שבעה להצמיד / בשבעה גלגלי זבולו

דגלים בגלגל רום העמיד / איש על דגלו.





דגלו דגים תהום שרץ / ועופות מרקק פלגים

דגר תנינים פרץ / ובצלצל דגים.





דגים האכל התיר לחוסיו / והכין לויתן לתכל

הן עליו יכרו עמוסיו / לעת האכל.





האכל הבא מטוב ימינו / ערך על שלחן והרבה

הן כל זה שדי הכינו / לעשות לאורח הבא.





הבא הביתה עת קראו / שם נפש בגו עמותה

הזמין הלך להביאו / ויבוא הביתה.





הביתה הודו עליו יאור / בהכתירו יפעת כבודו

הפיקו נות בית נאור / כסה שמים הודו.





הודו הזה נתן עליו / לעשות לו עם שאול חוזה

השית מתחת לרגליו / את כל הכבוד הזה.





הזה הגן הפקיד בידו / ונחת שלחנו ממגן

השליטו לאכל מעודדו / מכל עץ הגן.





הגן היא תתן מגד / וזה יתן לכם ויהי

הרחק מן העץ נגד / כי מצות המלך היא.





היא והוא אוו אכל / ושניהם שניהם קהו

והשיאם ערום נוכל / ותאכל היא והוא.





והוא ונסתר חשב דרכו / ממגלה מצפון ומסתר

ובהתהלך קול בסכו / ראה רעה ונסתר.





ונסתר ונלוז גער צורי / לא תוסיפו עוד לעלז

זמר לשם יושב תהלות / עד יעבר זעם.





זעם זה מלאו נסוגים / עד שוב מישרים חוזה

זבדם שנים שריגים / את זה לעמת זה.





זה זכרם שלחו למקנה / להתעדן בפרי סחרם

זרע יבול ורעות צונה / מארץ זכרם.





זכרם זמרה לקחו לקרב / פרי ובכרת צאן תשורה

זמן למוביל שי ערב / תודה וקול זמרה.





זמרה זמם עת רצון / ורב ניבו נבזה נשמם

זר לעשות חומד לצון / לו כאשר זמם.





זמם זועה יד להפך / חורש על אחיו רעה

זעף דם נקי לשפך / והיה רק זועה.





זועה זר מראש רוחש / פשע כסה ונזר

זך ענהו מה תכחש / ועיני ראו ולא זר.





זר חרב שלף נכנע / ואות שם חופש קרב

חלץ בלכתו נד ונע / ובמלחמה מידי חרב.





חרב חיל לא קרב / ויחלפו יום וליל

חבור עצבים מאד רב / עתקו גם גברו חיל.





חיל חנף רב תקפו / ומטה רשע צץ וענף

חבלים חלק אל באפו / כי כלו חנף.





חנף חי נאץ ואמר / לאל סור מני ארחי

חוקר סתר אז גמר / להכות את כל חי.





חי חשך יד מתי און / ומסכה עליהם נסך

חבלם יחד בטיט יון / ואת עבדו חשך.





חשך חק זרם כביר / וקמטו ולא עת מרחוק

חשרת מים עליהם העביר / כי עברו תורות חלפו חק.





חק חסיד לא הכרית / במחותו יקום מפסיד

חיה לארץ שארית / כי הפלה יי חסיד.





חסיד טרף מכל שנים / ובעץ גפר קץ וחרף

טלטל יונה לעינים / והנה עלה זית טרף.





טרף טוב כל ברואיו / והצילו מש[אול קטב]



טרף נתן ליראיו / ודורשי יי לא יחסרו כל טוב.





טוב טובם לא נער / ושם איתן מושבם

טחו תפל יושבי שנער / הן לא בידם טובם.





טובם טיט השלך בשאט / באמרם להכביד עבטיט

טאט מעזם ולא אט / וכמו יוצר ירמס טיט.





טיט טבור זדים רמס / לבל יקום דבר דבור

טלטל אנשי תוך וחמס / יושבי על טבור.





טבור טהור לאור יצא / ומאליו נזהר נזהור

טועים נמצאו סיג ונמצא / זהב טהור.





טהור טוב בחר בחור / בן שלש כגן רטב

טעם לנסוגים אחור / טעמו וראו כי טוב.





טוב יעץ להעיר איתן / ומאש גבור לא נרעץ

יזם כרת גלול מפתן / ונדיב נדיבות יעץ.





יעץ ישרו מנעוריו / לאחז רגל באשורו

יה צרפו מבין סתריו / להגיד לאדם ישרו.





ישרו יבחן לשחט יחיד / עד יקר דמו ויוחן



[יראת ש]די לא הכחיד / יי צדיק יבחן.





יבחן יראה ממכון כסאו / תבוסת עם עני ונכאה

יזכר ברית איתן בקראו / יי יראה.





יראה ישעו גזע אמוני / וימחה בשכר זאת פשעו

ישא ברכה מאת יי / וצדקה מאלהי ישעו.





ישעו יהגה בן נבחר / כארז בלבנון ישגה

יושב אהל דת שחר / ובתורתו יהגה.





יהגה יפרח בית מגוריו / תם נעור וריק ברח

יעטוהו בדי אמיריו / צדיק כתמר יפרח.





יפרח כדת גזע ידידות / עד בני חם הקשו עבודת

כבשו אדוניהם לעבדות / אשר לא כדת.





כדת כלו גן ננעל / הציץ ויעשר חילו

כרם לחבל יזם שועל / והנה עלה כלו.





כלו כלבנון זרח / מתחת סבלות מנון

כשושן בין חוחים פרח / ויך שרשיו כלבנון.





כלבנון כבוד רב העדם / ויאבדו זרים אבוד

כרעו שרי נף נגדם / ונגד זקניו כבוד.





כבוד כעדר שלחם / בישימון חדר בחדר

כסוס במדבר הנחם / וינהגם כעדר.





כעדר כאלה בנסעם / עת מסך פרוש יעלה

כזאת מי שמע על עם / מי ראה כאלה.





כאלה כהנים הם לאלי / קרואי ראשית אונים

כאמור ואתם תהיו לי / ממלכת כהנים.





כהנים לסבל קדש אנה / והזר הקרב יבל

לוי לעמד לפניו מנה / ויט שכמו לסבל.





לסבל לעולם פנת יקרת / אביר יעקב האצילם

לאהרן קדושו ברית כרת / אתה כהן לעולם.





לעולם ליי נבדל / לכהן בהיכל לפני

לו ולזרעו כי יגדל / יכתב ידו ליי.





ליי לעשות חקים נעימים / תהיינה כפיו פרושות

למלא ידו שבעת ימים / כאשר צוה יי לעשות.





לעשות לשולחיו דת יפנה / שאר גדול מאחיו

ללשכת פלהדרין יפנה / ציר נאמן לשולחיו.





לשולחיו לקח טוב יזמן / בהקהיל תחכמוני פקח

להורות חק עשור נאמן / ישמע חכם ויוסף לקח.





לקח לשולחיך תקשיב / ואז יישר כוחך

לכן התאמץ להשיב / אמרים אמת לשולחיך.





לשולחיך מתניך תשנס / וישמע אל ויענך

ממרום יתן לך נס / ואתה תאזר מתניך.





מתניך מפח מועק ימלט / באפס תקות מפח

מעגלותיך יתמך פלט / כי הוא יצילך מפח.





מפח מליץ טוב נאצל / עת יכונן נפשו להחליץ

ממות פתאם נצל / מלאך מליץ.





מליץ מרפא [וארוכה] / ועבודות ארבע ייפה



משמ[ר הלילה והיום] מלא[כ]ה / וציר אמונים מרפא.





מרפא משיחו ישלח תמיד / בהראותו זבחי עם [נכוחו]



[מעבירים] פר ואיל להעמיד / נגד יי ונגד משיחו.





משיחו מזה סר לבית רקח / עם זקני כהנה במחזה

משביעים עליו שם בלקח / אל נא תמוש מזה.





מזה מכבד בקול תשואה / ונעם צרדה בפה בלבד

ממעט אכל פן תקר תנואה / לקדוש יי מכבד.





מכבד נכוחים יהגו לפניו / בספרי קדש או בשיחים

נדודים ישבע בית מלוניו / משיב דברים נכוחים.





נכוחים נפשם חצות ליל / יעירון לעבודת קדושם

נצבעו בפיס לא בחיל / להציל ממות נפשם.





נפשם נכוחה ראשונית / תרם דשן עולה קדוחה

נתכן פנימי ונרות שנית / ידעו עשות נכוחה.





נכוחה נאמנה שלישית / [קטרת סמים באמנה]

נתחים ברביעית לשית / עדות יי נאמנה.





נאמנה נחפזו צנופים / לעשות [ זו]

[נגיד מלאכ]ת ישלח צופים / נבהלו נחפזו.





נחפזו נס לעשות ולחזות / אם שחר קדים [עלה כנס]

נגה כצאתו על פרזות / נתתה ליראיך נס.





נס נאמנים צור יצוה / בהוסדם על ציר [אמונים]



נכון לצאת חוץ למקוה / מימיו נאמנים.





נאמנים סבבוהו בנסעם / ולשער המים קרבוהו

סות פרשו בינו לבין עם / אשר סבבוהו.





סבבוהו שמלות ונחפז / לטבל ולהסתפג ולעלות

סגן יעטהו כלי פז / חליפות שמלות.





שמלות סבה הם לדחות / במכנסי בד שמץ ודבה

סאון חרפת שטים למחות / כי היתה סבה.





סבה סלולה בכתנת בד / לפני נורא עלילה

סוד פסים תסיר ומעבד / דרך לא סלולה.





סלולה סביביו אמרת צורו / באבנט יחגר שר קרביו

סחות צלמי כשדים בחגרו / על כל סביביו.





סביביו סביב מצנפת תעביר / פרע ותצדיק אות חביב

סכלות לשון בקול יגביר / על שולי המעיל סביב.





סביב סבלו אפוד ערב / וחשן שאת לפני מחוללו.

סמל העשוי בחורב / ביום ההוא יסור סבלו.





סבלו עולם בציץ יצמיד / רצות דם ונסך ומעלם

על מצחו יהיה תמיד / כירח יכון עולם.





עולם עדי עד ציר נקדש / עת לעשות תמיד יצעד

עשרה בעשור יקדש / ביי עדי עד.





עד עמו עז כקרץ / התמיד וקבל דמו

עמד להקטיר ולהטיב רץ / ויי עמו.





עמו עטרת רצון בערכו / ראש ונתחים לעתרת

עת יקריב סלת ונסכו / תשית לראשו עטרת.





עטרת עמו אץ ככלות / תמיד השחר מהקימו

עד פרוה ימהר לעלות / עם נדבי עמו.





עמו עליו סך וקדש / וירד וטבל מול אציליו

עטה בד ועת קדוש חדש / הוד והדר תשוה עליו.





עליו עומד אל עולם / צדק עדוותיו ללמד

עת בין מזבח לאולם / הכהן הגדול עומד.





עומד פניו בעת יקרב / לחלות ולגלות טמוניו

פר הקריב ופאת מערב / אל עבר פניו.





פניו פשע בל יכסה / ובסמכו יסמך ישע

פה יענהו צור מחסה / אשרי נשוי פשע.





פשע פה כעת יכפר / סגן וראש בית אב אפפו

פנה ויצג הפר / עד בואו פה.





פה פניהם שמו כהואל / ללכת עמו שניהם

פתח יסוד לצפון אריאל / מגמת פניהם.





פניהם פאר עטה במצאו / שעירים וקלפי פור מבאר

פורי פז טרף בנשאו / כחתן יכהן פאר.





פאר פעלת עת רצונך / ושם על ימינו הטלת

פץ סגן העל ימינך / כל מעשינו פעלת.





פעלת פחד כקמץ / בשמאל וירהו יחד

פי ראש בית אב חזק ואמץ / לא היה פחד.





פחד צדקו ציר יעטה / תמיד כי ישעה בחקו

צור שלום אליו יטה / יי חפץ למען צדקו.





צדקו צבי ופאר נכתם / בתפשו שני שעירים כלביא

צמד גורלות בתתם / לעטרת צבי.





צבי צבאות תעלז בנשאו / לשם חטאת בקול תשואות

צלצלו צדיקים בקראו / בשם יי צבאות.





צבאות ציון ראו אורה / בשמעם שם אלהים עליון

צפצפו יחד בקול זמרה / ורננו במרום ציון.





ציון צדקה לא להכרית / כדת שעיר העזאזל חקה

צמד בו תקות זהורית / עד יצא כנגה צדקה.





צדקה צפנת להראות / מליץ לשועו האזנת

צעד בין אולם לראות / טובך אשר צפנת.





צפנת צדקי בדרשו / לצאת עם שבטו נקי

צעק על הפר ברחשו / ענני אלהי צדקי.





צדקי קדושו אז יענה / ככלות לצקת לחשו

קם לשחט הפר לפני / גואל ישראל קדושו.





קדושו קולו יאזין רב / כקבל דם לרצות פעלו

קרא למרסו ואז יערב / את הוד קולו.





קולו קדשו אז יעלה / במחתת פז לחתות אשו

קרב למלא אש למלא / את דבר קדשו.





קדשו קטרת בשמאל נתונה / מלא חפניו למצא עתרת

קח מחתה בימין ופנה / להקטיר קטרת.





קטרת קדם מזבח פנימה / לפני הפרכת יקדם

קצה צפון יתור בעצמה / לרוכב בשמי שמי קדם.





קדם קרן ישעו ירים / לשאת נס ולחזק תרן

קדם מחתה בין שני עפרים / הנושאים קרן.





קרן קדשו כסתה עננה / בתתו קטרת על אשו

קצר עת צאתו רננה / בהיכל קדשו.





קדשו רבות רץ נחוץ / קחת דם פר ממסבות

רץ להזותו פנים וחוץ / פעמים רבות.





רבות רצה צור ערכו / עת לשחט שעיר יצא

רחש להזות ושב לדרכו / כי כבר רצה.





רצה רצון לכפר משוחי / והצליף על קיצון וחיצון

רצף השעיר החי / שוחר טוב יבקש רצון.





רצון ראש פעלו יחבר / שתי ידיו עליו וידרש

רחש פשעי עם ודבר / יחולו על ראש.





ראש ראש מדבר צוק ישלח / שעיר ביד עתי נפרש נפרוש

רצץ בהשליכו ופלח / מכף רגל ועד ראש.





ראש רבות יעש בהקטיר / חטאת לצור תוכן לבות

רחץ וקדש ועולה העטיר / וכהנה רבות.





רבות רע לא ימצא / וחק טבילה לא יגרע

רגל כף ומחתה ויצא / ואין פגע רע.





רע שמור בפשטו כלי בד / וטבל וקדש כחק אמור

שקד ללבש פז מכבד / כי למועד שמור.





שמור תרצה תמיד כעורך / ועת יקטיר וייטיב תפצה

תר לשאת ידו לברך / ופעל ידיו תרצה.





תרצה שלום לעם מאשר / בצאתו באין פגע לבלם

שמר תם וראה ישר / כי אחרית לאיש שלום.





שלום לעמו ציר יזבד / והם יסובבוהו לרוממו

לביתו ילוו בכבוד / דורש טוב לעמו.





לעמו מופת טוב יבשר / באמרם לציר שעיר שופת

מי יודע כי עוון סר / תנו לכם מופת.





מופת האות עת יעירון / אז יבשר ציר פלאות

הוסר עוונך עם ישורון / וזה לך האות.





האות לבן מאד כברית / וטהורים נצבו כדרבן

לראות פלא חוט זהורית / כלו הפך לבן.





לבן עוון וסר פשע / דברה בת קול בהדומו

חסידים זמרו קול ישע / כי רוצה יי בעמו.
