


	
	ארנן לאל חי
	








ארנן לאל חי / למכין ארחי / אסדר שבחי / לעוטה מלוכה



ביום דין וזכרון / אקרא בגרון / תת הלל ורן / לפני כבוד מלוכה



גלה יקרך / והראה הדרך / לעם לך ערך / סדר מלוכה



דגלך הרם / ואויבך החרם / ולען הברם / למכחישי מלוכה



הראה פלא / לאסיר כלא / ומבור העלה / בן הנחלתו מלוכה



ומיום הגלם / ומיום נפלם / יחכו לגואלם / אדון המלוכה



זרה נא פשעם / וקרב ישעם / ונקה זרעם / וחדש מלוכה



חיש נא ישע / לשבי פשע / ומחטא ורשע / נקה זרע מלוכה



טהר מחסה / לעם בך חוסה / ומיד שוסה / חלץ בני מלוכה



ישעך הראה / אל עם נלאה / מכה ונכאה / מיחל קץ מלוכה



כחך יגדל / עלי רש ודל / ואל יהי נבדל / מישע מלוכה



לך רם ותלול / אעמד בפלול / ואשמיע הלול / ואיחד מלוכה



מקים בריתו / לעם נחלתו / ומכתיר עדתו / בכתר מלוכה



נחם ביום זה / עם דל ונבזה / וישעך נחזה / בחדשך מלוכה



סמך נא לעטוף / ביד צר קטוף / בכל עת חטוף / ביד מרגיזי מלוכה



עונה בצר / ענה נא לנעצר / ונפשו תקצר / מכבד על מלוכה



פודה וקונה / שעה למענה / בחסדך תפנה / למחכה מלוכה



צפה בעניו / והתר שביו / והשב עדיו / לקרית מלוכה



קרא נא דרור לו / והסר סבלו / ושבר עלו / והקם כתר מלוכה



רננו תרצה / וחפשי יצא / אזי לך יפצה / בעז המלוכה



שעה נא רחשו / והעבר כחשו / והרחק מוקשו / ועטרהו מלוכה



תדלה נאסר / אשר כל יחסר / מיום מנהו סר / כתר מלוכה



תזכר ברחמים / לעם נכלמים / פני כל עמים / כונן קרית מלוכה.
