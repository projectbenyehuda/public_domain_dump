


	
	שליט יוצר
	








מי כמוך ואין כמוך מי דומה לך ואין דומה לך





שליט יוצר כל פעל / בתחת ובמעל / מודד מים בשעל / הנוטה כדק שמים



לובש צדקה כשריון / אל מסתתר בחביון / נורא ועליון / בהכינו שמים



מושל בגבורתו עולם / מחיה יצורים כלם / ברא שחקים וכללם / וצבא השמים



השם עבים רכובו / ובמים עזים נתיבו / ענן וערפל סביבו / היושבי בשמים



– – – – – – / – – – – – / – – – – – / – – – – – 



אחריש הגיון / ואשאל רשיון / אלהים עליון / נוטה שמים



ברא במבע / יסודות ארבע / תקע וקבע / גבהי שמים



גזר נאור / ויהי אור / הולך ואור / תחת השמים



דלה זרומים / לגבהי מרומים / והקוה תהומים / מתחת השמים



הציץ דשאים / וכל צאצאים / וזיז ופראים / ועוף השמים



והריץ מאורות / והשריץ יאורות / ורקם צורות / ויכלו השמים



זהורים עדו / וצדו וזדו / נעו ונדו / מתחת השמים



חבקו אשכר / רועה ואכר / וצעיר נזכר / לאל השמים



טחה טחה / ואחיו דחה / זכרו מחה / מתחת השמים



יהירים עמסו / ונתיב חמסו / נחשו וכחשו / באלהי שמים



כמס רבה / והפליא סבה / בתהום רבה / וארבות השמים



לאמים התיר / ועבדו הסתיר / קטר והעתיר / לאלהי השמים



מרעים שלו / וגאו ועלו / ומגדל תלו / וראשו בשמים



מצדם נשחק / נדו למרחק / להם שחק / יושב בשמים



נדיב נודע / מאור נפדע / אזרח [י]דע / חקות שמים



סחף שרים / וסעף זרים / ידו הרים / אל אל בשמים



עמד ושקד / ובנו עקד / זכרו פקד / כימי השמים



פצה עתרת / לנכח גברת / זכר עקרת / בורא השמים



צאצאי תאומים / שערים וימים / זכרו לרחמים / נצב בשמים



קפץ תמים / מאיש דמים / הוליד רחומים / עמודי שמים



רגישת שבטים / ירדה לפוטים / ורבו מעוטים / ככוכבי השמים



שאון תקיף / מצודו הקיף / ראה והשקיף / יי משמים



שבר גאונם / ושקע שאונם / כל[ו] ואינם / עד בלתי שמים



תמה חפושה / בים חדשה / שירה חדשה / לעושה השמים.
