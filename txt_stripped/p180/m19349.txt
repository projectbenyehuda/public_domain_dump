


	
	[עצמותי הה]
	



וקאל איצא הזגיה [ואמר גם כן (שיר) במשקל התנועות]




עצמותי הה / יתמוללו / כי צרותי / יתבוללו

ותבונותי / במזמותי / לולי השר / אשתוללו

ומידעי / משכו לבי / ומנוחתי / עת אזלו

על־כן אבכה / גם־דמעותי / לילה ויום / לא־יחדלו

אם־סגר אל / שערי פלול / שערי דמעה / לא־ננעלו

כי צוקותי / ומגורותי / עלי רבו / ויגדלו

כרבות חסדי / שר על־עמו / שטובתיו / לי נכפלו

יקר ונדיב / לבב יתן / חייו ובט־ / רם ישאלו

ובנות ימים / על־כל־שאל / מתת ידו
/ לא־משלו

שמחת פניו / ושחק פיהו / עם־נדבותיו /
בו הללו

ובני עמו / בטוב ירם / ובדרך לא /
יכשלו

סרו גלים / ממקדשים / ימיו אמרו / כי
לא־גלו

מי בעמים / ירום לבו / אם מפניו / לא
יחילו

הנה השיר / יגיד חסדו / בתיו למאד / בו
יגילו

פצחו רנה / לאדניהם / ובחמלתו / לא ישלו.
