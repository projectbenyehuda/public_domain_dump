


	
	אמרה גולה
	








ובכן יי מעון אתה היית לנו

פזמון יי תשפת שלום לנו





אמרה גולה וסורה / אשר ביד צרים עצורה

בעודי בבור גלות אסורה / שם אדיר יי לנו





בחונה בגלות יושבת / כאניה בלב ים שוכבת

בכל עת תעורר מחשבת / לאמר מי יעלה לנו





גרושה מחל ועפל / סגורה במחשך ואפל

כמעט תאבד משפל / לולי יי צבאות הותיר לנו





דחויה עם כל ממלכת / כיונה קצוצה משלכת

צדו צעדיה מלכת / במדבר ארבו לנו





הפוכה כצפור בתוך רשת / בכל עת לשחט נדרשת

והיא בכל זאת אורשת / אולי יתעשת האלהים לנו





וכמה זמנים נחשבו / ועמך בגלות ישבו

ואויבים מאחריהם לא שבו / כי עתה יאמרו אין מלך לנו





זממו שקר להחביר / וכחשו בדת אל כביר

אשר אמרו ללשוננו נגביר / שפתנו אתנו מי אדון לנו





חסין הופע בהודך / ושלח כמאז ידך

וקנא לשם כבודך / לא לנו יי לא לנו





טאט מלכות נביות / וטרף פריצי חיות

טורפי פזורים כשיות / אשר אמרו נירשה לנו





ירש שעיר הגלנו / והיום כמה אכלנו

האיש אשר כלנו / ואשר דמה לנו





כאין תמנו ואבדנו / מרב אדונים עבדנו

כמעט מהם נכחדנו / לולי יי שהיה לנו





לרגלי עבדים כפפנו / ועל בור ומסגר אספנו

על צוארנו נרדפנו / יגענו לא הונח לנו





מרום איה חסדיך / גמלת מאז ידידיך

אשר שמעו עבדיך / ואבותינו ספרו לנו





נאר גאים ושקץ / וגלה פליאת הקץ

וצעיר רודם הקץ / קומה עזרתה לנו





סגב שפלים משחוח / ושיחם יערב כניחוח

ולקט שושן מבין חוח / ולכה לישועתה לנו





עצובה עזובה תשמח / ודמעת לחיה תמח

וצמח צדיק יצמח / כי ילד ילד לנו





פרק עלי והסר / וזרים תוסיף ליסר

ושלח מהרה מבשר / ויבוא כגשם לנו





צהל יקיר נבזה / ומלך ביפיו נחזה

ויושר השיר הזה / בארץ יהודה עיר עז לנו





קרית עזי תצלח / ואתה תשמע ותסלח

ותשא עוון נאלח / אל יי ויאר לנו





רנת עושי תשובה / תהיה כעולה חשובה

ויודו נשויי משובה / לא כחטאינו עשה לנו





שלום וחדוה עשית / ועוון עמך נשית

פשע וחטאה כסית / כי גם כל מעשינו פעלת לנו





תמיד תהיה נגדך / שועת עמך ועבדיך

הראנו יי חסדך / וישעך תתן לנו.
