


	
	שור ידיד
	








שור ידיד כאבי / וחמס יריבי / ולוחמי

ילעגה לצירי / צר וירד ערי / והולמי



לוהטים לאידי מהללים / את כבוד זקני מחללים / בוזזים יקרי ושוללים



שלחו בקרבי / גחלי שביבי / וטעמי

מר במאסרי / גם דמע נהרי / בלי דמי



מועדי כקדם תחדשם / שוב אלי המונך וקדשם / אויבי באיבה תגרשם



להגי וניבי / האזין וריבי / מרוממי

ריב וגם יקרי / שא ביד הדרי / ונעמי



העלה מיחלי פדותך / לחצר דבירך וביתך / שם לך תרנן עדתך



ישרה נתיבי / יה ומחריבי / וזועמי

אר ואז בהרי / אודך בשירי / ונאמי.
