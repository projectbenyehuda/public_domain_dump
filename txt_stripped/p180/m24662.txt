


	
	שדודים נדודים
	








שדודים נדודים / לציון תקבץ

מכורים בלא הון / תשובב למרבץ

השב עבודת / לובשי תשבץ

תרעת שמעת / משפחות יעבץ

להודות לזמר / לשמך עליון / אתה תקום תרחם ציון.





לגויי ארצות / תחיש שאת נס

ינון תגבר / ומתניו תשנס

עתרי תאסף / ופוצי תכנס

תחדש גאלה / כטנס וחנס / צדיק לובש צדקה כשריון.





משנאי ידבר / עזות בפני

מבקש לאבד / תקות אמוני

יריבני עד אן / תוחיל לעיני

עניתיו כי לא יזנח / לעולם אדני / מעז לדל מעז לאביון.





השב גרושה / לנחלת ירושה

ופקד אנושה / ביד צר ענושה

אכולה בכל פה / נתושה ורטושה

וחדש צדקת / אבות שלשה / וזרים תכניע כחרב בציון.
