


	
	שאול ישאלון
	








שאול ישאלון / לז ללז ימללון / מהללים אהיה / ואומרים איה





בכסא רם ונשא / אדר ארבע חיות

ובארץ כנגדם / בחירי דר עליות

גדלת אל מעידים / ולפניו עומדים

וכן ראה חוזה / דמות קמים להללו / שרפים עומדים ממעל לו.





בכסא רם ונשא / הוד כנפים ארבע

ובארץ כנגדם / ועודי חוט מרבע

זרויים בארבע כנפות / כנגד ארבע זקופות

וכן ראה חוזה / חסן אל מיחד / שש כנפים שש כנפים לאחד.





בכסא רם ונשא / טכס אדם נקדם

ובארץ כנגדם / יעודים צאן אדם

כזכרם עווניהם / יכסו את פניהם

וכן ראה חוזה / לפני דר מעוניו / בשתים יכסה פניו.





בכסא רם ונשא / מכונת אריה יעודה

ובארץ כנגדם / נצני יהודה

סולדים בלב מאשר / וברגל מישר

וכן ראה חוזה / עז דגלי זבוליו / ובשתים יכסה רגליו.





בכסא רם ונשא / פאר דמות שור במישור

ובארץ כנגדם / צאצאי בכור שור

קהלם יאספון / וכיונים יעופון

וכן ראה חוזה / רום כבוד מחופף / ובשתים יעופף.





בכסא רם ונשא / שביב נשר נכתם

ובארץ כנגדם / תמימים כנשר בלכתם

שיר ליוצר ובורא / זה אל זה קורא

וכן ראה חוזה / לעומדים על משמר / וקרא זה אל זה ואמר.





בכסא רם ונשא / מראה גחלי אש

ובארץ כנגדם / ההולכים במו אש

במונים שלשה / ישלשו לך קדשה

וכן ראה חוזה / רחש שרפי פלאות / קדוש קדוש קדוש יי צבאות.





בכסא רם ונשא / ידות רצים ושבים

ובארץ כנגדם / המון רצים בניבים

ושבים לצור ערץ / מלא חוג וארץ

וכן ראה חוזה / דברי המון גדודו / מלא כל הארץ כבודו.
