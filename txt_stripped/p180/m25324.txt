


	
	שעה ניבי
	








שעה ניבי / בהיטיבי / שגיונותי והגיוני

למכאובי / צרי תביא / ותקריב יום פדיוני

להושיבי / בת נדיבי / בארמוני כקדמוני

מחץ רבי / וסרבי / ועקרבי וסלוני

היה סביבי / וריב ריבי / והשב גמול גאיוני

שלח אבי / משוש לבבי / המבשר אביוני

אליה / הנביא / לפני בוא יום יי



לך האל / אני שואל / וגם אפרש לך ימין

קרץ אדבאל / ומגדיאל / וכל מכזיב בך יאמין

ושובב [א]ל / עיר אריאל / בני יהודה ובנימין

וחיש גואל / לישראל / וקרב קץ הימין

ומיכאל / וגבריאל / יעמדו בדוכני





מעד חומסי / וחיל בושסי / כטיט חוצות הד[ק רקע

ולרמוסי / ולרפו]סי / יתד נאמן תתקע

[ואו]ר חמוסי / ונר עמוסי / כשחר יבקע

ומכעיסי / ומו[א]סי / שחוח ילכו לפני

הדר ישי / נשיא אישי / כאור נוצץ באפריון

ונענשי / ונאנשי / בימיו ימצאו פדיון

ונחבשי / ונכבשי / ברנה יעלו ציון

ויובל שי / למקדשי / בהגיון ושגיון

ואל רשי / וחלשי / שלל מוני ושלמוני.
