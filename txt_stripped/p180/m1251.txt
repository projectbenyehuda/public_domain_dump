


	
	אדני מה אדם
	



אדני מה אדם / ר' שלמה אבן גבירול


אדני, מה אדם? / הלא בשר ודם!



ימיו כצל עובר / ולא ידע בנודם,



פתאם יבא אידו / וישכב וירדם.

אדני, מה אדם? / גו נרפש ונרמס,



מלא תוך ומרמה / וגם און וחמס,



כציץ נבל יצא ‑ / וחם השמש ונמס.



אם תפקד עליו / עונו הנכמס, ‑



אפך וגם קצפך / איך ישא ועמס?



לכן חמל וגמל / כי אין לאל ידם.

אדני, מה אדם? / בטיט יתגלל,



בשקר יתהלל / ובשוא יתהלל.



הטהור מטמא / ויקר מזולל?



אם תפקד עליו / עון יצר מעולל



כמו עשב ייבש, / וכחציר ימולל:



לכן חמל וגמל / חסד ביום אידם.

אדני, מה אדם? / יהיר לא נחם,



מי חמס ישתה / ולחם רע ילחם,



וכים הוא נגרש / וכתנור יחם.



אם תפקד עליו / חטא אשר בו יוחם –



יספה כחלש / עם גבור נלחם.



לכן חמל וגמל / וכפר מעבדם!

אדני, מה אדם? / רק נתעב ונאלח,



כל‑היום יבגד / ופניו ברע ישלח.



אם תפקד עליו ‑ / יהי נרפש ונדלח,



וכבגד יבלה / וכעשן נמלח.



לכן חמל וגמל / וכפר‑נא וסלח



לשכני בתי‑חמר / אשר בעפר יסודם!

אדני, מה אדם? / והוא כעץ נקסס,



בבוא אמת בצעו / יהי כמו‑קש מרסס



ויבכה מאשר שש / יום ירקב כמו‑סס,



אם תפקד עליו / עונו הנכסס –



כמו שבלול יהלך / וכדונג נמסס,



לכן חמל וגמל / ואל תדקדק בעדם!

אדני, מה אדם? / לבד נדף כעלה,



במאזנים לעלות / בהבל יסלה,



וככלוב מלא עוף / כן מרמה מלא



אם תפקד עליו / ועל לבך יעלה –



כמו עשן יכלה / וכרקב יבלה.



לכן חמל וגמל / כחסדך, לא כחסדם!
