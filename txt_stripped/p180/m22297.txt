


	
	יגון חשק
	



ולו בתהלה






יגון חשק ואהבת הנעורים / עזבוני בלי לב
ביצורים



ועיני נפתחו מאין שנתם / לדלתי האהבים הסגורים



ושם אל כוכביו עמי, כאלו / אני רועה והמה
העדרים



ונסני כאברם המנסה / באמרו: קום ספר שאין
ספורים



ראות אלה וכאלה, ידידי, / יביאוני ברעה לקברים



צררוני, הציקוני יגונים / ואך שבע צרפוני בכורים



והיום, שת שפתיו צוף לחכי, / הלא נהפך והיה
ממרורים



וחצי הזמן - שירי, ואלו / אהבני - עשיתים לו
כתרים



והם הגות בפי כל החכמים / ובלשון כל מתי מוסר
שמורים



וראשם יהיה שמור לעולם / ויפדה בכל שוטנים
וערים



אשר בחר בשם טוב משמנים / וטוב מכל בדלחים
ודרים



אשר נתן לכתר המלכים / ולעטרת, ונזר מנזרים



מבלה שוטני שרית ישורון / והמתיעצים אינם
אזורים



ולו מרו ולא אבו שמע / למועצותיו - אכלם ככפירים



הלא יחפזו מגערתו / וגם יבהלו זדים ארורים



אדום ובני צפו צופים לדעתו / וקדר שוחחים אליו
וקודרים



יליד משרה ובן יתרה, ואמון / בחיקי הסגנים
הגבירים



נדיב רוח, אשר צוה ענניו / לבל יהיו לעולמים
עצורים



רצוץ מתן ומזהיר כל לתתן / לכל שואל, ואם אינם
זהירים



ודיני אל וחקותיו ודתיו / עלי פיהו ערוכים גם
שמורים



אשר המה כתובים על לבבו / ובין עיניו לטוטפות
קשורים



אשר ניביו יקודים עת יניבם / ואינם אש אבל כאש
בעורים



אשר לבש כמו שרין צדקה / ושם כל משפטי עמו
ישרים



אשר אם נע שפתיו קט לדבר - / לאט, מתוך שפתיו קח
דברים



כפיר יציל טרפת עם ישורון / וידק בחמתו הכפירים



וכל אובד וכל נדח מבקש / וחובש בחסדיו
השבורים



ומסיר מעליהם שן בהמות / וחיות עם חמת זוחלי
עפרים



ומה אגיד בתורו המעלה / ומה אמר - ורעיוני
קצרים



בשכנו מעלות שמש, והאל / הביאו מלפנים בחדרים



ואולם מפלאי השיר יעירון / מזמותי ונוקצים וערים



לקבץ מגדודיו הפזורים / ומהשיר לגלות הסתרים



והשיר גדלני מנעורי / כמו אב ואני לו בן
בכורים



וקרן עור פני שירי, כאלו / עליהם מתהלתך
אפרים



ואליו יחלו כל המשוררים / ופיהם פערו לטל
פעורים



אשר לנו בנפשותם אריות / והשכימו לפניו
כשורים



וממשרה ומכבוד קנויים / וללעג ולחרפה מכורים



אחי מוסר, מעט קט כתרו לי / עדי ינערו מני
נעורים



ותראו שיר יבהל רעיונים / אשר בתיו ממלאים
בדרים



בזהב וחרוזי הבדלח / במדברים וענינים יקרים



ישימוני בעין אנשי זמני / כמו אריה בעיני
השורים.
