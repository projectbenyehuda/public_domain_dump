


	
	לך שדי
	






לך שדי / ואמר די / לגלת רום וחיליה

לך מרום / ים ודרום / וכל ששת גבוליה

הלא תעיד / וגם תגיד / גדלתך במליה

המאיר / לארץ / ולדרים עליה



לך עליון / מקום חביון / הדרך והודך

לך התו / אשר תחתיו / גלגלי יסודך

וכל גלגל / וכל מעגל / יהודך ייחדך

הכי הכל / כמו אשכל / תלויים הם בידך

ומי לא נע / ולא נכנע / בזכרו שם כבודך

ומ יכיל / ומי ישכיל / ומי יעמד בסודך

גדולתך / פעלתך / לבב איש לא יכילה



לך כל יסוד / ותדע סוד / הגלגל העליון

אשר ארח / ממזרח / ויתגלגל כגליון

וסוד ימתיק / בעת יעתיק / גליל שמש בלא רפיון

עשות רמש / וצאת שמש / כמו חתן באפריון

והוא העד / בכל מועד / באין אמר ולא הגיון

וראש דברו / כי יצרו / אל אחד בלי דמיון

עבר ציה / ותחתיה / להאיר מאפליה





לך דין / כל ענין / וכל קנין פלוסיך

לך אדיר / לך יאדיר / המון קצוי אפסיך

מי יעצר / ומי יבצר / פלאיך ונסיך

מי חכם / ומי יחכם / ומי יעשה כמעשיך

ואת נאור / ותעטה אור / ארץ לך וחיליה

לך וילון / והחלון / אשר לפאת צפונך

לך שנאן / ורבו רבבן / והם כאין לפניך

לז אל לז / בקול מעלז / הלא שואלים למעונך

ויעידון / ויגידון / גבורתך וחנך

והנעלה / והמעלה / תרבה מהלליה



לך קושר / חין ישר / סנדלפון לכותרת

לך ענף / ועיט כנף / מהדרת ומעטרת

לך יקר / בלי נחקר / הוד והדר ותפארת

ונוראות / ונפלאות / עלילות למזכרת

שית רום דור / לדור ודור / לגבעה וכל חיליה.
