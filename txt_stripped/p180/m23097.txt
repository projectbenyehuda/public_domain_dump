


	
	אליכם יודעי
	



ולו באותו הענין. ומשיריו השבעה יקונן על אחד הנפטרים בלשון צחה ובדרך הרג׳ז.


אליכם יודעי נהי, / אני אקרא בשיחתי



לכו באו והקבצו / המו עמי בהמיתי



ומהרו נא וגם שלחו / מקוננות ותבאנה



ותשאנה המון נהי / ומספד מר תקוננה



ואל תבכו מידעי / למת נקבר באדמתו



בכו בכה לאיש הלך / אשר לא שב אלי ביתו



שלחהו גביר עמים / היות תעל למכותם



והאדומים אחזוהו / דקרוהו בחרבותם



אשר שלח בתם נפשו / ובלבבו כמו איש תם



והצרים סבבוהו / בשקריהם ופחזותם



משנאים טמנו פח לו / לצודו בו וגם חצב



והעת צרחו עליו / אזי צרח בקול נעצב



ומתחנן לעמתם / (וחושבים רע בלב עקב



אשר לא יחזה שדי / ולא יראה קדוש יעקב):



הרפוני ועזבוני / ואל תראו ברעתי



ומה בצע בדמי יש / ומה תועיל הרגתי



לבבו [סר וגם עופף] / 	וגם נדאב



ואז צר[ח] במר [נ]פשו / כמו יתום ואין לו אב



ואין מקשיב ואין עונה / ואין שועה לשועתו



ביום קרבה נפילתו / ביום [ב]אה פק[דתו]



ואלו [מת] במגפה / ואם אין ־ [ב]מות [משבר]



ולא ימות והוא נדקר / וכל גופו כמו מכבר



יולדתי והורתי / אהה צירים אחזוני



וצד[ו שו]טני נפשי / ובחרב המיתוני



[ו]לא אראך לעולמים / ולא אביט לפניך



בכי תמיד ואל נא ת/תני פוגה לעיניך



אהה צרים רדפוני / ולמות הדפוני



וברמחים נשאוני / ולחיות נתנוני



ולפני מי במכאובי / במר אהמה ומר אצרח



[ואנה זה] א[ני] אפנה / ואין מנוס ואין מברח



בכי, עדה, והילילי / וגזי את הדר נזרך



בראשך ה[עלי] עפר / ב[הכבות] מאור נרך



ואבירך וא[דירך / אשר זדים הרגוהו]



ואת דמו בחרבותם / בדי		שפכוהו



[אדו]ם, אל תשמחי על זאת / ותשישי ליום אידי



די]
