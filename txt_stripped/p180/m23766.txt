


	
	שטר עלי בעדים
	








שטר עלי בעדים וקנין / לשלם מחרת המנין / והיה יי לדין



- - - - - - - - -



אזי בחשבי למרחוק / בכיתי לבלי חק / עת לבכות ועת לשחק

נפלה עלי חרדה / בזכרי יום הקפדה / פחד קראני ורעדה



יען שרירות לבבי / דאגתי במחשבי / כי איך אעלה אל אבי



שד למי רב זדונהו / יום יבקש עוונהו / וחשב עם קונהו



ליום לא ישא פני שר / בהתנגש למוסר / הנפש עם הבשר



משפטיהם ודיניהם / עד האל אדוניהם / יבוא דבר שניהם

הנפש תמהר תשובה / אין עלי חטאת כתובה / צדקה נפשה משובה

הרפני ועזבני / הלריב תציבני / הודיעני על מה תריבני

קדושי לא אהבני / כי במסגר הביאני / במחשכים הושיבני

טנף גוף טורחת / ומנבלה סורחת / אנכי בורחת

נלאיתי מאד כלכל / כי הגוף הנוכל / השיאני ואכל

בכל חטא יתגלע / ואותי בכף קולע / לא השיב ידו מבלע

רעצני הניאני / ולשחת קראני / אך עתה הלאני

בכל תאוה הוא נאסר / ומאכל ומשקה לא סר / ובטן רשעים תחסר

יגוף לטמא עצמו / יען כי הוא כשמו / נבל שמו ונבלה עמו

יצודני פותע / והוא בתתו מלתע / בעיניו כמתעתע

הן חטאו לא יספר / לכן יכלם ויכפר / כדי רשעתו במספר

וממני אל תפרע / ומוסר גו אל תגרע / כי כלו חנף ומרע

דבר גוף להושע / חדל ממני והשע / זך אני בלי פשע

הן כאבן מעמסות / מלא בוז וקלסות / ומה יכלתי עשות

גם אני נשכל שכול / דומה לעץ האשכל / אשר לא יצלח לכל

בלי נפש אחשב / כבית מבלי יושב / ואין עונה ואין קשב

יען אשכים ואאמיש / משלך כמו חלמיש / ממקומו לא ימיש

ראש הפקדה כלה / על נפש ומעללה / שלמו לה כפעלה

ושאי נפשי מריך / ואכלי נא את פריך / ושלמי את נשיך

לכל רע לא צמאתי / אבל אני בך נדמיתי / איך תאמרי לא נטמאתי

מחשבותיך הבינו / וידיך הכינו / ונחנו מה כי תלינו

אמלים אשר רגשו / וצדק לא פגשו / אחד באחד יגשו

לב דורש ומעבדיו / יחביר שניהם עדיו / ובערו שניהם יחדו

קלקול עוונם יוסר / ולא יכבד המוסר / בשגם הוא בשר

יהיר שרירות יחמד / אם כמפעלו תמד / אדני מי יעמד

חשך זעם אימתך / ואל תיסר בחמתך / וינפש בן אמתך

קרב נא להרויחני / ויום לריב תקחני / אל באפך תוכיחני.
