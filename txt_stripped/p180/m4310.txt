


	
	[אזי בשעיף]
	



וממ ّ א חוא פיה כל כואכב אלת͘אבתה מת͘ל אל ّ ד͘י לם יטורהא כ͘יאל ולא ילחקהא ארתג͘אל קצידה או ّ להא [וממה שמכיל את כל כוכבי-השבת (ר"ל: המזהירים) ואשר לכמותה לא יקרב הדמיון ולא ישיגנה כוח האימפרוביזאציה זו שירה שראשיתה]

אזי בשעיף חלום רוח נשאו / ועבר כעבר ברק בפלאו



ולא שר אחריו שמץ לבבי / ולו מדי ראות עמד רפאו



וקץ לשור דמותו משעפיו / ולא שער לבבי כי בזאו



ויסתיר מאחי נער אמריו / לבלתי יֽאמרו חשק מלאו



הכי נלכד בחבלי התלאות / ושבר אחרי שבר קראו



ומשברך תמול הניא זמנו / היתמלא והוא עז להניאו



כאלו אזנך העם אשר צר / להכיל מחשבותיו יום צבאו



ואחז לבך כנפות אדמה / והשח במתי-שחק בשיאו



ובקע שכלך מדבר דברים / ותחת רגלך תדבר לביאו



ונגה ממך אור על-אנשים / כשמש עמך היה מבואו



ולא נזכר לשירך שיר מנחם / ושיר דונש ושיר אבון קריאו



כצח יסתיר בעודו על-אדמה / בני-עיש ויראון בבואו



ואיכה תחריש כיום להפיץ / יאור בו כל-יאור שבר צמאו



ונפש בהמות ימים תדכה / ועטרם כחנית גלית בדכאו



ולשון כחצץ ברק ילהט / ויפיק כאפיק נחל בשואו



ונטשת גדוד השיר מפזר / בלי-ערוך ואתה שר צבאו



לכל נתעה לבבו בכסילות / כהתעות אנוש שכור בקיאו



וכל-שגה ונקל ממשוגה / אשר השיר במצרף לא הביאו



ומקר נוזליו כי-נצבו נד / עדי כי-קפאו ימים לקפאו



זכר משה אשר היה כבד-פה / ועל-כן אהרן היה נביאו



עניתיו איך אשורר שיר והשיר / לעמת כל-כסיל ירז בריאו



ונפשט מבגדיו החמדות / ונותר כאנוש עני בכסאו



ונמאס מכלי אין בו תעלה / וממלך אשר הורד לכסאו



ונקל מכתב מדרש ביד מין / ומפסל אשר נביא מצאו



ולא נשאר אנוש מבעלי כן / ונמלא הזמן מבעלי או



ואינם אהבי שכל ובינה / אבל הם אהבי שמע וראו



כפתי כחכם לבב כזולל / כיקר כארז דלת כתאו



הכשול באחי שפלות וסכלות / ופגוש בהמות יער ופראו



כזיזי יערות ינוק שדי הר / כעתודי רעי לחוך לדשאו



עירים לו אנוש ידקר בשרם / בפטיש גב אחרם לא יראו



וישתרר אשר ישיג מעט בין / כהשתרר נבל-נפש במצאו



והם לו אספו על-כף נמלה / נקלים להיותם די מלאו



הכי מופת באולתם לחסד / אדון עולם אשר להם בראו



ואמר נשאו כי חברו שיר / עניתיו לא כשיר זמר וכתאו



והחגר לבד יבז גבורה / ומלחמת פרז אפס בגאו



והעז בקרב יבא ויגבר / ויעשה מעשה פינחס בקנאו



והחפץ בעודו לעלות עב / בתוך בטן שאול תחתית כלאו



ושיר כל-בחני מלין ימולל / וייבש כחציר שדה וגמאו



וכתב הזמן שירי בפניו / ואם-הוא לא ידעו לא שנאו.
