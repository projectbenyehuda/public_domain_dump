


	
	שני זיתים
	








שני זיתים / נכרתים / בגן נעול יצהירו

לראש קהתי / ואפרתי / שתי עטרות יכתירו

מול מנורה / הטהורה / כמו נרות יזהירו

הן במשאת / כל איש את / רעהו יעזרו

הן במחנה / אל מול פני / המנורה יאירו.





לבן פרת / וציץ יקרת / אשר היה כציץ נובל

ולנמשח / אשר הושח / כמו אניה בלי חובל

זכר מראה / אשר ראה / זכריהו בעיר בבל

כמו זכרתם / והושעתם / אזי על יד זרבבל

זכר זרעם / להושיעם / בעת תקרא שנת יובל

ושים אותם / מקום אבותם / היות מושלים בכל תבל



ומעל יד / איש ציד / טבעתם יסירו.





ממלכה / ממשכה / לעיר ציון תשיבנה

ובת כבודה / ברב עבדה / לבית אמה תביאנה

והגבירה / והצפירה / לראש דודה תקימנה

ומצנפת / מעלפת / לראש אהרן תשימנה

והתמיד / אזי תעמיד / והמנחה תרימנה

והמרקח / אשר לקח / ולא ראיתיו עד הנה

והקטרת / מקטרת / לאהרן יקטירנה

והנרות / במנורות / בני אהרן יבעירו.





הנכאה / והצמאה / אשר תמיד לך תיליל

סמך ידה / בבן דודה / אשר נקרא שמו חכליל

והכניסם / לגן בשם / וחפתם בהוד תכליל

והוליכם / והמליכם / בהר ציון והגליל

וחק ישר / אזי יושר / עלי נבל וגם חליל

זעם עוזבי / שמך ואוהבי / חזון שקר ושוא ואליל

ועם פרוכים / יהו ברוכים / כל רואיהם יכירו.
