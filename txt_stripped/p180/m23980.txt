


	
	אתה אמון
	








       ובכן פזמון יי מעון אתה





אתה אמון קנית / אתה אצלה חנית

אתה ארמון בנית / אתה נורא אתה





אתה תוכן שחקים / אתה תומך חזקים

אתה תולה ארקים / רבות עשית אתה





אתה בלי חק נקצב / אתה באין גבול נחצב

אתה בכל נצב / אם אסק שמים שם אתה





אתה שדי ברום ערץ / אתה שליט בכל הארץ

אתה שוכן באין קרץ / נכון כסאך מאז מעולם אתה





אתה גאות לבשת / אתה גבורות כבשת

אתה גדלה חבשת / כי גדול אתה





אתה רחום רב ישע / אתה רוצה מחות פשע

אתה רחוק מרשע / כי לא אל חפץ רשע אתה





אתה דורש עמקות / אתה דוחה מצוקות

אתה דובר צדקות / זך וישר אתה





אתה קדוש בראתני / אתה קדם נשאתני

אתה קטן קראתני / יצרתיך עבד לי אתה





אתה הדקות רהב / אתה הפכת להב

אתה הגית באהב / קראתי בשמך לי אתה





אתה צוה חסדיך / אתה צהל ידידך

אתה צדק עבדיך / כי תפארת עזמו אתה





אתה והעבר מרינו / אתה והבט ענינו

אתה וגאל שבינו / כי גואל אתה





אתה פלטים תנהר / אתה פשעים תטהר

אתה פדיון תמהר / ואמרת ללא עמי עמי אתה





אתה זכים תושיב / אתה זועקים תקשיב

אתה זרויים תשיב / ולאמר לציון עמי אתה





אתה עשוקים תחלץ / אתה עריץ תאלץ

אתה עוצית תפלץ / אומרים לעץ אבי אתה





אתה חלץ נדשת / אתה חזק נשת

אתה חתת קשת / אומר לרשע צדיק אתה





אתה סעד אם חלושה / אתה סער בן גרושה

אתה סור מירשה / כי בן אשה אחרת אתה





אתה טלאים תקבץ / אתה טרופים תרבץ

אתה טירה תשבץ / אלוף נעורי אתה





אתה נכים תשסף / אתה נפוצים תאסף

אתה נוהג כצאן יוסף / ראש שבטי ישראל אתה





אתה יהירים תנאר / אתה ינון תפאר

אתה ישעו תבאר / הוא יקראני אבי אתה





אתה מלך עולם / אתה מכל נעלם

אתה מושל על כלם / משל בנו גם אתה





אתה כתמים תדלח / אתה כלם תסלח

אתה כפר תשלח / כחסדך זכר לי אתה





אתה לחזק הבדק / אתה לישר כל חדק

אתה לובש הצדק / כי צדיק אתה.
