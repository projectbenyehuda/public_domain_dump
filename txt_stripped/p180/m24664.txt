


	
	שזופה נזופה
	








שזופה נזופה / בשבלת שטופה

מה לך כפופה / בלי קומה זקופה

תשב עטופה / כשושנה קטופה

בכל עת אשר תגור / כלי מות חגור

תצפצף כעגור / בתוך מסגר סגור

היה לה צור צנה / מאוסה תהיה לראש פנה.





לפני משנאי / אני תמיד נכנע

שאנן ורענן / ואני נד ונע

ילעג וישאג / וידו לא ימנע

אקוה לעת מרפא / ואוחיל ואצפה

ואשים יד על פה / בלב נכפה ונספה

כחנה מול פננה.





מה לך דוה לב / במר לבב יעטף

אפיו וכפיו / בדמעתו ישטף

זרע לך צדקה / כי חסד תקטף

הלא עול ימס / וגם יהיה למרמס

וחץ שחוט נכמס / לכל אוהב חמס

לזרע מי מנה.





הלכו שנותי / ביגון ואנחה

אקוה רוחה / והנה קול צוחה

בטרם צרי בא / נפשי נלקחה

דוה לבב נאנח / קוה ואל תנח

לאל כי לא יזנח / ומאויב תהי נח

ותפרח כמו גנה.





חלו וכלו / עיני מיחל

זעפך אשא / ונאלמתי כרחל

אולי מצוקי / יהי לפשעי מוחל

בחסדך אל תדכה / נגוע ומכה

בלב נשבר ונדכה / ליל ויום יחכה

לישעך אל קנא.





קשורה ביד צר / כצפור בכף תינוק

ידחק וישחק / והיא תאנק אנוק

שמעי נא עניה / יושבת צינוק

צפי לי וחכי / מהרה אנכי

אשלח מלאכי / לפנות את דרכי

בערב בכי ולבקר רנה.
