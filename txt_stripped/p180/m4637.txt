


	
	[כנשר את מרחף על-ילדיו]
	



[כנשר את מרחף על-ילדיו] / שלמה אבן גבירול

ולה תצאנע אלי בעץ ͘ תעל ّ ם עלם אלאדב [ולו דברי מחמאות לאיש שהיה בקי בחכמת המוסר]

È — — — / È — — — / È — — / / È — — — /È — — — / È — —

  


כנשר את מרחף על-ילדיו / וכאדון אשר משל עבדיו



ואת בלט עלות השיר מנהג / כמו-מלך ינהג את-גדודיו



והמוסר בך נאסר ולא-סר / בך ילך ולא-יצר צעדיו



ואם-התפֽרדו קדם פזוריו / הלא אתה תקבץ את-נדודיו



ועל-אזן תבונה את כעגיל / וגם על-צורניה רדידיו



ואת נמשל כעץ חיים וקצו / פני-שחק ובארץ יסודיו



ומי יתן ישימני ידידי / כמו זרו וכחוט על-בגדיו



ויבין מתבונתו לבבי / ויראני כאחד מילדיו



אשורר שיר תהלתו לעולם / ויתן עץ תהלתו מגדיו.
