


	
	מי כמוכה
	








מי כמוכה ואין כמוך

מי דומה לך ואין דומה לך





מי כמוכה שוכן עד לא ראשית / ומי דומה לך

שבילו ודרכו בים השית / יי אלהינו יי אחד





מי כמוכה לפניך יערך ויועיד / ומי דומה לך

לשמך הכל יודה ויעיד / יי אלהינו יי אחד





מי כמוכה מרום לא ראתה עין / ומי דומה לך

מוציא כל יש מאין / יי אלהינו יי אחד





מי כמוכה הנוטה כדק שמים / ומי דומה לך

הרוקע הארץ על המים / ומי כעמך ישראל גוי אחד





אז בראשית קנין / אל הפליא ענין

יסד ראש ומנין / ועשה את האחד





בהיות הכל בנוי / נודע כי הוא קנוי

גם לכל דבר מנוי / הלא אב אחד





גלם שפריר נבצר / וארבע חיות עצר

ובהוד עזו יצר / ארבעה פנים לאחד





דפן אבן ארקיו / ונטה עליה קו

ויט גלגלי שחקיו / ויהי המשכן אחד





הליכות אפלה פוצץ / ויהיה אור נוצץ

סבב עלי חוג מרוצץ / ויהי ערב ויהי בקר יום אחד





ויצו נבך להשקיע / ויתו תוכו רקיע

ולשני גלים הבקיע / את הים האחד





זמם במלאת יומים / ונאם יקוו המים

מתחת השמים / אל מקום אחד





חניטי דשאים הפריץ / ומאורי אור הריץ

ומיני חיות השריץ / זאב וטלה ירעו כאחד





טהור יצוריו קדם / ואמר נעשה אדם

להיות מושל בסודם / והקימותי עליהם רועה אחד





יצק עפר וספח / ועשת מרקם טפח

ונפש באפיו נפח / ויהי איש אחד





כבודה השיב לצדו / להיות עזר כנגדו

לא טוב היותו לבדו / טובים השנים מן האחד





לטירה יקרה שבץ / ותרשיש ושהם רבץ

ואותם לתוכה קבץ / והיו לבשר אחד





מרום הזהיר משמעת / ידך בכל עץ נוגעת

אך מעץ הדעת / תהיו לעבר אחד





נפתה הפכפך וזר / ויצא למות ונפזר

עליו האל גזר / שתי אלה רגע ביום אחד





סנסן שנים ספיחים / בוקר ועובד שיחים

קדמו תשורת מחים / ודבר חד את אחד





ערכו שמן ורזה / זה נכבד וזה נבזה

ויקם זה על זה / ויכו האחד את האחד





פלל לצורו נכנע / אל נא חסדך תמנע

והייתי נד ונע / עתה אספה יום אחד





צורי עווני עודף / והנה אותי הודף

אחרי מי אתה רודף / אחרי פרעוש אחד





קבל בנחת ושובה / בשובו להודות משובה

על כן יצוו תשובה / הנביאים פה אחד





רבה רעת מתי געל / והוחל לקרא בשם בעל

כחשו לאל ממעל / ויאמרו גש הלאה ויאמרו האחד





שקף אל טוב וסלח / על דור אשר לא יצלח

כלו סג יחדו נאלח / אין עושה טוב אין גם אחד





תקיף הכחיד סודם / ונהר יוצק יסודם

כי מקרה בני האדם / ומקרה הבהמה ומקרה אחד





אזנו פקח ויגל / טרם יסובב גלגל

ויצו מפלס מעגל / קח לך עץ אחד





אנה לידו עצי גפר / והיה הכפר כפר

סגורים נצלו כעפר / איש אחד איש אחד





נקבצו אנשי בקעה / ונועדו אנשי רשעה

יזמו לפסג גבעה / ושמו להם ראש אחד





יזמו מזמה להעיר / הבה נבנה לנו עיר

ונבצר גדול וצעיר / ויאמר יי הן עם אחד





שואת מגדל חפץ / ושפת עליזים קפץ

ואותם משם נפץ / ולא נותר מהם אחד





לעשרים דור פרח / ציץ הנקרא אזרח

יצא כשמש ויזרח / והוא איש אחד





מסכת אלילים פרץ / ומאש זרים לא ערץ

נטה אחריך רץ / לב אחד ודרך אחד





הצלתו בשבעה דרכים / והוכחת עליו מלכים

חשכתו מהלך חשכים / את פני פחת אחד





העת איתן סקר / כי הוא ערירי עקר

דרש האל וחקר / חלום בלילה אחד





קרא שושן בתוך שית / הנני נשל כזית

והיה מחר בן בית / כהן לבית איש אחד





טעם שא זרועך / וספר כוכבי ישעך

כה יהיה זרעך / ועשיתי אותם לגוי אחד





נדיב מופת דרש / במה אדע כי אירש

כאשר צור ארש / נתתי לך שכם אחד





באר בן סוד לועז / בעגלה ואיל ועז

ותור וגוזל העז / וקרב אותם אחד אל אחד





רזם לו במבע / סוד מלכיות ארבע

ויקם אשר נשבע / לא נפל דבר אחד





בשיבו צהר צהור / ויולד זך וטהור

שמרו כבן כהור / תורה אחת ומשפט אחד





יזם חוקר רזה / להראות לכל צר בוזה

אי זה יכשר הזה או זה / ואם שניהם כאחד





יה בחנו עשיריה / כגפן ענפה פוריה

לך לך אל ארץ המוריה / ואחז לך אחד





הקדים מבחר בדיו / ועמס עצים בבדיו

וילכו שניהם יחדו / לפניך בדרך אחד





ויען בן משתאה / אבי הזבח איה

השיבו אלהים יראה / את הכבש האחד





דמעת אב תפכה / והבן מות יחכה

זה ייליל וזה יבכה / לפאת האחד





הה לשד אבי ופחדו / כי אפל חלל לצדו

ונותר הוא לבדו / ואילו האחד





אבי אל נא תבכה / וקוה ליי וחכה

כי לשכר טוב תזכה / לכל מקרה אחד





במותי נחם אמי / כי אאסף אל עמי

ומחר אתם עמי / הכל הולך אל מקום אחד





נעו וזעו צבא ערץ / בשורם כי בא הקרץ

וכמעט תפל על הארץ / קומת הכרוב האחד





געשו לפני אים / חנון הפק פדיום

ולא ישחט כיום / טלה חלב אחד





בכו בראות התער / נלקח לשחט הנער

עד קראו פתחו השער / שלחו מכם אחד





ידיד בקחתו הזין / נם עזרי מאין

וירא בשאת לך עין / והנה אופן אחד





רחש אסף ידיך / מאת בנך יחידך

כי כבר נודע סודך / וימלט בן אחד





ובעת רדתו מערך / נשא עיניו שפל ברך

וירא בסבך הדרך / והנה איל אחד





להג בתתו פדיום / ויצעק לנורא ואים

לזרעי זכר זה היום / והיה יום אחד





מעין חכמה נסתם / עד כי נולד איש תם

בכסא כבוד נחתם / ויכוננו ברחם אחד





אזן הולך ושב / ולחוט משלש נחשב

באהל תבונה ישב / לחבר את האהל להיות אחד





לבדו לגור נמסר / ובעז ושלום בשר

נכתר בשנים עשר / נשיא אחד נשיא אחד





קרבו ימים ולילות / גם נתכנו העלילות

הפליא שדי פלילות / ויצא האחד





יוסף לפתרוס הורד / ומחיק אביו נפרד

זעק עליו וחרד / נגדע היום שבט אחד





חבוש במסגר אסר / עד עת למלך הושר

שמו לבית טפסר / מלך עליהם ורועה אחד





זלזל כצפור מקן / זרע מאשר מתקן

יחדו נער וזקן / בני איש אחד





קננה בחנס צפור / ורבתה עד אין לספר

ישבה שלוה מחפר / כימי מלך אחד





ויהי כנאסף יוסף / חשף זרועו חושף

ויאמר רב שוסף / ישנו עם אחד





והנו עצום ורב / והיה בהיות הקרב

ונוסף על צר וארב / אל העיר כאיש אחד





יחידים ענו במס / ובטיט ובחמר למרמס

ויזעקו שד וחמס / טף ונשים ביום אחד





אתה צפירת שיחור / והקוץ יסוב לאחור

שלח ידו מן החור / וימלא את יד אחד





מעץ שלישי הנעים / צמח שלשה נטיעים

כמראה שלשה גביעים / משקדים בקנה אחד





צופה בלכתו בדרך / פגע מציץ מחרך

שלחו להוציא מפרך / בית אב אחד





לך נא אל איש חרמי / ודבר שלח עמי

ועשה אותות בשמי / ולקח עץ אחד





בא ויקרא במרץ / שלח עמי מארץ

בדבר אחד ברום ערץ / ויאמר מי אחד





בדרשו מופת להזמין / השליך מטה בימין

ויאמר לא אאמין / על פי עד אחד





כשפים עשו בלט / ומטה אהרן שלט

מטות בדים חלט / והיו לעץ אחד





מרגיש עצמו ישטם / אזנו אטום יאטם

דבר גם זה חרטם / ואם יתקפו האחד





כפש היאור והכו / ולא שת לבו עד כה

הוסיף אמץ בדרכו / הלך בדרך אחד





להקו צפרדעים בפנים / בבתי פחים וצנים

ויך בהם כנים / שני העמר לאחד





הביא עליהם ערוב / ארב לנפשם ארוב

ודבר ממית לרב / וממקנה בני ישראל לא מת אחד





מרחם בשחין וגרד / והמטיר עליהם ברד

וארבה בארצם ירד / לא נשאר ארבה אחד





יעץ חומד לצון / בהחשיך נוה קיצון

יצג בקר וצאן / ואתם תלקטו לאחד אחד





חוקק מענו הפאיר / בשמעו סלון ממאיר

במצרים לא אשאיר / פר בן בקר אחד





להג תנין ונסכל / דע כי חרב תאכל

לך מעלי למה אשכל / גם שניכם יום אחד





ידיד ענה עכשוב / כן לא אוסיף לשוב

אך יש למועד חשוב / עוד נגע אחד





מחר אגף ילדיך / והשתחוו לי עבדיך

צא אתה וכל גדודיך / לעבדו שכם אחד





לכודים נזכרו זכור / ולודים נעכרו עכור

ויהרג בהם כל בכור / לא נשאר אחד





יום גדלה הצוחה / אז היתה הרוחה

ויצאו מאנחה לשמחה / כל הקהל כאחד





המונים כחנו בחירות / חדשו צרות אחרות

חפרו חנסים בארות / ותקהל העדה כאיש אחד





ויענו חפצי אידנו / מה זאת עשתה ידנו

כי שלחנו מעבדנו / את העם הזה כאיש אחד





הוות יעץ וחרש / ולקח גלגל ופרש

לרדף עני ורש / ויצאו כאיש אחד





ידידים סחרו סחור / ויפנו העם אחור

והנה מרכבות שיחור / עולות בקנה אחד





ויהי כראות רכבם / חרדו ויצא לבם

הים בקרבם והצר לגבם / מזה אחד ומזה אחד





מחוקק צעק וצרח / כי הצר הציץ וזרח

השיבו הך ים ורקע / ונטה מטך ותקע





כפף יהודה ונכנס / ואזור חילו שנס

הים ראה גם נס / מפני גערת אחד





פשו אופן וגלגל / לרדף סלולי מעגל

ויכס עליהם הגל / לא נשאר מהם עד אחד





רואה חנסים וגוים / מתים על שפת הים

יאמר איפה אים / כפה ואגמון ביום אחד





יעצו להשליך במים / נשלחו במימי תהומים

שפטם שוכן שמים / מדה אחת וקצב אחד





מצולות ידידים הדריכו / ונעלם על צר השליכו

רוממו לאל והמליכו / יי אחד ושמו אחד.
