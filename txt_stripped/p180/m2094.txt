


	
	[ידידי סב]
	



ולו לאחד הידידים הקרובים ביותר אליו אשר שלח אליו עטים




ידידי סב ודמה / לחכמה תעז מ- / עשרה שליטים



מרומים שבתך, / ודורש בלתך - / כדורש לאטים



תנפף תלפיות / וחלילה להיות / שפתיך שטים



אבל לא יתאפק / לבבך עד יפק / זממי השטים



והצדקת עדי / ביום באו עדי / עשרת העטים



אשר גבה דרכם, / ואם אגיד ערכם - / סעפי
ממעיטים



כזהב עת צקת / כמראה ברקת / כדרים ובהטים



כספיר וכלשם / כחוטי הגשם, / כגבי שרביטים



וכתאשור וברוש / ושרביט אחשורוש, / כחצים
ושבטים



כלבנת קרן שן / כצהר בין דשן, / כסלת החטים



כקרני מזבח / כמו ראש גבח, / כתורים נמרטים



כקנה עם קדה, / כאצבעות שדה, / כצוארי עיטים



כעמודי בירה, / כקנה המנורה, / כבדי השטים



אשר הם למחוקק / בעד כל דב שוקק / רמחים ושלטים



לשונם דק כחוט / וכחץ השחוט / ידבר משפטים



תניפם יד-אמן / והם לפאת תימן / כלב חכם נוטים



לאט ילכו ולאט, / ואם בם יד תשאט / ינוסון
כפלטים



וחרקו שן חרוק, / והאגרת - רק / פתנים מלעיטים



חשב האגרת / כמו ים כנרת, / והמה - כמשוטים



יגלו בה סודות / כצצי השדות - / ואינמו בוטים



כברק בה יברק, / כצפרים ישרק / אשר אכל
לוקטים



ודרך עוף דרכם / אשר ביבש חכם / ירוצון לרהטים



בלי גרון קורא / יביאון כל דברי / ספרד לפוטים



דברם ינכר / ופימו יסכר / בקרב ילקוטים



שאר טלם חרב / וכמרירי שרב / כסותימו עוטים



וברבות הקרה / וקרח הנורא - / לבושם מפשיטים



קשורי גביהם / כאלו מתניהם / חגורים אבנטים



שמך במו אחרש / בשיר בשם ראש / כסותיך לוטים



יישר כל עקב / ויתנך, יעקב, / כיעקב לשבטים!
