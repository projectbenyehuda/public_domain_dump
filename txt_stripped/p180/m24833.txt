


	
	נשמת שבט
	








נשמת שבט רא[ובן / באדם מציר / יעלה / במזל טלה]



גם שבט שמעון / [בפטדה ומזל שור מכיר

זה לעמת ניסן] / וזה לעמת איר





[נשמת להקת לוי / בברקת גמוז] / תמים / במזל ת[אומים



גם לאם יהודה / בנפך ומזל סרטן רמוז]

זה [לעמת סיון / וזה לעמת תמוז





נשמת מטה יששכר] / בספיר כלול / ח[יה / במזל אריה



גם מולדת זבולון / ביהלום ומזל] בתולה / תלול



[זה לעמת אב / וזה לעמת אלול





נשמת המון] דן / בלשם מ[כון] ים / במזל מאזנים



גם המלת נפתלי / בשבו] ומזל עקרב [בארגון]



זה לעמת [תש]רי / וזה לעמת מ[רחשון





נשמת קהלת גד / באחלמה מישבת / נדרשת במזל קשת

גם קבוצת אשר / בתרשיש ומזל גדי מיטבת

זה לעמת כסלו / וזה לעמת טבת





נשמת [ט]כוסי יוסף / בשהם המסדר / מפליא במזל דלי



גם [נאמני בנימין / בישפה ומזל דגים מהדר]



זה לעמת שבט / וזה לעמת אדר.
