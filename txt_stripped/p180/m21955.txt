


	
	תהלת אל
	



למנויו של אחד מתלמידי החכמים




תהלת אל בראש כל התהלות / וצדקותיו לתנות,
הגדולות



אשר נעלה עלי ערץ ונשגב / ושת בינו ובינינו
גבולות



ימינו רוממה במאד פלאיו / אשר גדלו עלי כל
הגדלות



אשר תתמך אדמה על בלימה / ותחריב בעזוזה
המצולות



אשר תעור ותפדה הישנים / בגלותם, ותחיש
הגאלות



לשה גולה, ומבנים שכולה / וגלמודה, שכולה
משכולות



נדודה מהנחה לאנחה / אכולה גם בנותיה אכולות



והיום לה כמו אלף בגלות / וכמעט קט ואלפים
כפולות



בכל פיהם אכלוה יריבים / ועוד ידמו שאריתה
נטולות



בעצם עצמוה העצומים / עלי חילה אשר היה נחלות



והיא מיד ליד נדחה ושחה / בשוחות וחלקלקות
אפלות



בחבלי העני תשב אחוזה / עניה פעמותיה כבולות



מפרק על! פרק על צוארונה / והמוטות תמוטט
בחילות



ומשכרתה לגלותה תשלם / ותשב לה גמול,
משיב גמולות!



ותבנה מהרוסיה בצורות / ותקים מנתוציה תלולות



ותבעל מבנותיה עזובות / אשר היו לצודיהן בעולות



וגלות חיל מדינה העדינה / תחדה במכלולות כלולות



אשר גלו אלי ארץ ספרד / והשיגו, והשיגום קללות



ומשאות נשאו כי את גזרתה / והכילו עדי אין
בם יכולות



רעבים, מבלי מים צמאים, / כאילות חשוכות מאילות



ולולי נחלו נחל אמונה / שארית יעקב עשו פלילות



אזי בם נעקר עקר תבונה / ויבשו האשלים
השתולות



והיו האסורות כפטורות / והיו הכשרות כפסולות



אשר העמיד יסודות משפטי דת / והשים מפלותיה
בטלות



ולנו עם ישורון בחלקות / והשכימו באורי האפלות



כדין דין אמת, הולך נכוחו / והצדק לבקש גם
פעלות



אשר יפסיק לדין ראוי להפסיק / בצדקתו בלי און
ועולות



וממתין בו, ואם זקוק לנחץ, / כאזהרת ישישי
העגלות



ודינו מבלי טעות מתקן / כארח שאבניה סקולות



והקלות השיתן כחמורות / וראה ככבדות הנקלות



ואם ישפטו נצים לפניו / ומתעולל אזרם בעלילות



יזכה זה, ואם את זה יחיב / ולא ישא פני פנים
צהולות



וכל משפט לאור יוציא בצדקו / ומשפטיו וארחותיו
סלולות



ובמקום השבועה שם פשרה / ובה גזר מקום גזרו
באלות



וימציא תעלומי האבדות / ומהעושקים ישיב גזלות



ושם מלוה לאיש עני להשיב, / בעוד שמש, בגדיו
החבולות



והלביש מערמי התבונה / והיו היחפות בו נעולות



ופתח לחכמים האטומות / ומי יפתח דלתות שם
נעולות?



והוא נבחר, בעת אין שם זבחים / לאל, משי וקרבנות
ועולות



והוא הזהיר עלי צדק וצוה / - הירצה בעגלים או
עגלות -



אמת כי בו עוונינו מחויים / וחטאת דורשי חפצו
מחולות



וכל חכמות לחכמתו עזובות / וגם מבעליהן הן
שלולות



ונקראות תשובותיו בבבל / ונפרשות בראשי
הקהלות



בסוד ראשי נהרדעא וסורא / עירות הפלילות
הגדולות



הרימוהן עלי כל התשובות / ובזהב כתבום במגלות



ואומרים: מצאו אנשי ספרד / פליאי התבונות
המעלות



וחזו האמת גבר ברבם / וחזינו אנחנו מהתלות



ולבשנו לעמתם קרעים / והמה לבשו חמדת שמלות



והיתה גאותם כאריות / והיינו לעמתם רחלות



ושריגי תבונה הם קטופים / ואין עולל, וקטפנו
מלילות



ועיני עוריהם נפקחו בם / ואזניהם, ואזנינו צלולות



ברב הרבה ששוניהם וחדם / וחדותם להגלות
תהלות



ורב האיי כלא היה לפניו / וקטן מענות אותו במלות



מזמותיו יפיצון כל מזמות, / ותחבולות לכל אדם -
חבולות



וחכמת שוטרי חכמות גדולות / אשר הורה עלי דרך
גמולות



פני רב למדנו הכעורות



ומלא הנפשות היבשות / עדי היו במי הבין
טבולות



במים לא אנוש ישחה בדכים / ואם תרד לסוף תמצא
מצולות



וחכמתו רחבה מאדמה / ואם יש לה גבול - אין לה
גבולות



יפצל מפליאיה פצלות / כאיש פצל ליחם בם
במקלות



ותלדנה תבונותיו בלי ציר / ולא קשות ולא רוגזות
וחלות



וכל עת הן מחדשות ועלות / ואם הנה מישנות ובלות



אשר הנה לכל צמא ועיף / מזי רעב כצמוקי דבלות



ודעות הגלויות - לחכמים, / והוא ידע לנסתרות
ונגלות



והכל פוצחים רנות לשונות / ובלבבם לרבם רב
מסלות



אשר הזהיר להדליק נר נשיהם / והפרשת תרומיות
וחלות



ומצוותיו וחקותיו ודתיו / מישרות נכוחות
כשפלות



ותוכחותיו לכל ראשים עטרות / וכצפירות עלי קדקד
סבולות



אשר הנה בעינינו בתולות / וכמונו ועמנו שגולות



והן רפאות שרירי נוססיהן / ולמזורות מזורות גם
תעלות



והנה למגששים כמאורות / וכנרות דלוקות
באפלות



ומימי אב אבי מלאך עתרות / ותלמיד רב, אשר הנחיל
נחלות -



אמת לא קם בישראל כמותו / אבל כי איש בגבעת
הערלות



אשר מל הלבבות הערלות / והשכיל הנפשות הסכלות



וקרע כל סגורי לב אוילים / ונפשותם מפלשות
חלולות



וכל סתום וכל נעלם ועמום / מגלות הן לקשיות
עפלות



והוא אחד ואין שני לערכו / ונפשו בם כמלך
בחילות



עלולה היקרה מתבונה / אצולה מאצילי האצולות



מהירה לעשות תחבול ולו ה-/חרוצה נחשבה מהעצלות



לכן יקל ויקטן מכבודו / אגר שבח וחבר מהללות



וקציתי קצת הודו, וחסר / דלי מיאור ונטף מתעלות



ובראותי יקר השיר ויתרו / הכינותיו לנוהו מסלות



והעדיתיו בעדי הבדלח / והענדתיו עטרות
מתהלות -



ואמרותי אמרות הטהורות / במאזני שיר ומשקלות
שקולות



ומצאתי בצורות כעזובות / וחשבתי במחשבות
כלולות



והיו יודעי שפה ברורה / וכל אמיץ לדבר צח במלות



סחרחרים ולא מצאו ידיהם / והנה לעזוז ידי אזולות



ומשלתי לשון עברית, ועזי / כבדיה לעמתי
נקלות



ועשיתי לפוציה מסלות, / כמו כן לערוציה תעלות



ולו עוכר עכרני עכרתיו / כחדקל וכפרת בהמלות



ורגלי בה כרגלי הראמים / ורגליו בה כרגלי
הנמלות



ועננוהו וכסוהו אבקים / ולא יכל לצעדו בנעלות



והיו לי לביאותיו כשיות / וגם נפלו לאותותי
קטולות.



ואל תתמה בשירות היקרות - / תמה מהשמונה
הכפולות.
