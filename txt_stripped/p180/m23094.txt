


	
	אנוש חציר
	








אנוש חציר ייבש / ובמות נכבש / וגוש
רמה ילבש / ויובל לקברים



ולמות נצב / ואליו הוא נחצב / וסופו
הנקצב / לעפרות מדברים



והחי הנגבר / עלי מה יתגבר? / ומחר
למדבר / יהושם מדרים



ולא יועיל הונו / וחילו והמונו / ועוד
כמעט אינו / בבית ומדורים



ולא יועיל קרוב / ולא ממון לרב / והמות
יארב / לחטפו כנמרים



ולא יועיל מטמון / ולא יעזר המון / ולא
יתר אמון / ביום מות עוזרים



וגבור הנאזר / ליום מות נחזר / ולקרוב
יכזר / ויהיו לו זרים



והחי הנמעד / ולקבר מועד / ולא ימצא
מסעד / לנפשו מבורים



וזה הוא המקרה / לכל אדם יקרה / כברחת
נזרה / יהובל מן זורים



ואדם הגבור / עדי מתי יגבר / ובעפר
לקבר / ויוסר כגרים



והודיעם גדלו / כבר אין הועילו / ולא
יוכל גאלו / ביום איד ושברים



ולעפר נשבר / ויום מחר נקבר / ויעבר
המעבר / ויהיה מעברים



וישכח זכרו / בדורו ומדורו / ורעו
ושארו / בלי לשמו זוכרים



ויאמר איו / בעת יקח דיו / ולשכן
את עיו / ברגבי מסתרים



ויוסר האביר / אשר היה כביר / ועל כל
מכביר / בחייו לדברים



ותאבד תוחלת / ותסור ממשלת / לעולם
אולת / ודברי נעתרים



ושהיה כוכב ־ / והיום הנה כב / ועוד
אינו נעכב / ונעתק לעפרים



והעם הרוהים / בפחדו ותמהים / עלי
יומו נוהים / והומים נשברים

וסרו הבינות / ובטלו חשבונות / ואבדו
עשתונות / אשר היה מערים

ובשל הקציר / כגפן הבציר / ונמשלים
חציר / במגל נקצרים



וחילים עם צירים / אחזום ומצרים / ואינם
נעצרים / וגם לא נבצרים



וגבור הבצור / כלו היה נצור /
וכאשכול בצור / ולזה נוצרים



והכל הוא בולה / וימות גם יכלה / ויסור
לכלא / וישב דור דורים



וישתה תרעלות / ויפל בנפילות / ולא
יוכל לעלות / ביום איד ומזורים



וילך לתעלות / רגבים ומחילות / ומזור
ותעלות / לחליו נעדרים



מהרה ישתולל / כגפן יתעולל / כדשא
יתמולל / וישכב בחורים



ואיה העמים / ואי משפחות קדומים /
ואיה האימים / ואיה גבורים?



ולא מצאו פדיום / ביום מות איום /
וממוקשי היום / כליהם נעדרים



ידידי, מה תכאב / ועל מה זה תדאב /
ותבכה על האב / במספד תמרורים



ותשא ביללה / ביומם ולילה / וגם קול
כחולה / ודמעתך תזרים



ועד מתי תחיל / במו צירים וחיל? / ודם
לאל הוחיל / אלהי היצורים



ינחם לבך / וירבה טובך / וירים אביך
/ בחברת הישרים



ויגיה צהרו / וידליק את נרו / וארחו
יזהירו / כנגה המאורים.
