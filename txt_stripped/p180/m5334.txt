


	
	[אפלס מעגלי]
	



ומן כלמאתה אלוסאס וקצאידה אלג͘האר קצידה יד͘ם ّ פיהא קומא ויפתכ͘ר בנפסה [ומדבריו הנאים ומשירותיו המוכרות שירה שבה הוא מגנה אחרים ומשבח את עצמו]

אפלס מעגלי / ואיטיב מעללי / ואלך-תם אולי / לאל יערב שיחי

ואתן תודתו / ואשמר תורתו / ואעמד בבריתו / ואשמר מסחי

ומעי ארתיחה / ולב לא-אקשיחה / בזכרו אשיחה / ותתעטף רוחי

זממי לא-נבצר / וידי לא-תקצר / בקראי ממצר / אדני מבטחי

מגלה תושיות / וחפש משכיות / לבבות וכליות / ומגיד מה-שחי

באף יזכר רחם / לאדם ינחם / אלהי מרחם / ומבטן גחי

והציל שה תעה / בחסדו מרעה / ועיני מדמעה / ורגלי מדחי

והוא יתמך ידי / ויכיר מעבדי / ויכין מצעדי / כמו זרה ארחי

בלבי אש תיקד / לעל פשע נשקד / ובצואר נעקד / כהעקד מזחי

ולי רעיון נדבר / יעירני דבר / לך לבב נשבר / ונפשי הוכחי

ושובי מגאונך / ומחשבת אונך / והטי את-אזנך / ועיניך פקחי

שמר מחטא משכיל / אשר עשה יסכיל / אלי אם-לא אשכיל / ואם-אשכיל פצחי

הלא נפל אשת / מזרה הרשת / ולמה עוד בשת / עלומים תשכחי

ואתה משתולל / וברע מסתולל / אמר איך תתהלל / ותאמר זך לקחי

והסכנת הסכן / לחשב רע מכן / והנה אך על-כן / סביבתי פחי

לך ישן עורה / ומשדי גורה / ומתבל סורה / ואם-תמות תחי

תפרש-לך כפה / ויום תקשה ערפה / ותלטש עפעפה / תהי מאוס וסחי

לזות שפה הסר / ואולי תוסר / ומה-אתה חסר / ודרך אל נכחי

ודרכי אל שמרם / ומצותיו זכרם / ועל-אצבע קשרם / וכתבם על-לוחי

ושפלי כל-היקום / בסודם אל-תקום / והשקט אם-יקום / בך כל-פרחחי

השיבני על-זאת / וענני עזות / אחזוני רגזות / ונחר מפחי

בקרבי חיל כבד / יפלח הכבד / ותקותי תאבד / ולא-אבד נצחי

אנחנו עם-אנשי / זמני הקשה / כמצרים עם-משה / וכשדים עם-תרחי

תמונות נמשלות / ולבות נבדלות / ואינם נקהלות / כשלג עם-אחי

התהיה הברכה / לעולם כברכה / ואם-צלם מיכה / כצלמון האחחי

ואם-יהיה עמרם / כמו משפחת רם / ואם-יהיה אברם / אבירם הקרחי

ואם-יהיה הידד / אשר בו יוד כהדד / בלי יוד או אלדד / כבלדד השוחי

וחלילה להיות / שורים כאריות / ואיזה קל חיות / אשר יצעד פתחי

מידעי זמרי / ורעי בן-עמרי / ושנאי-בן-אורי / ואיתן אזרחי

ולו יש משאלת / ומצאו קהלת / לחצוהו דלת / והכוהו לחי

בריתם לא ימוט / לשום שכל שמוט / ועל-כן בם אקוט / ולא-אשכח שחי

ובראותם אחוש / לקלל היום חוש / ולו בשרי נחוש / וכאבנים כחי

נפשות נדאבות / וגבותם עבות / כגלי מצבות / ושפתי נפתחי

וגבהם כתדהר / ולבם אץ נמהר / כעתודי ההר / וקטפי מלחי

כחסן אלונים / כקומת ערמונים / כסירי סלונים / כקמשי חוחי

ישיבון את-מלי / ויסבו עלי / עדי אשליך כלי / ואשליך תותחי

פניהם נפעמתי / ודומם נאלמתי / ולו ארצה שמתי / באפיהם חחי

ורפיתי קשים / ורפאתי חלים / אבל שמתים כים / וספרי מזרחי

הכי אין דומיה / לנפש המיה / וקצוי תושיה / יגרמו חחי

ואניף יד רמה / בדעת ומזמה / וציץ נזר חכמה / אשוה על-מצחי

ואליה אתאב / ומימיה אשאב / ולבין אקרא אב / ולמוסר אחי

ודרכי בין אצר / ובשכל אעצר / וממנו אקצר / שחיסי וספיחי

והוא גן פרדסי / וייני ועסיסי / וכפרי והדסי / וריח ניחחי.
