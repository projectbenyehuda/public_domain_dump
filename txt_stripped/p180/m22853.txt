


	
	ביום קצר
	



ואמר עוד, נשמתו עדן, בהספידו נפטר שהיה יקר לו.






ביום קצר אפי / במשל אפתח פי: /
השחק עפעפי / ואם לחיי שריון?



ומה יתאב תאוב / לקול שיח כאוב /
לבבי ממכאוב / וידי מרפיון?



וייגע [גם] ייעף / מאד מרב שרעף /
ונשאר העפעף / כחומת בת ציון



וחמה ימלא / לסוד לא יפלא / ואיכה לא
יגלה / והוא על גליון?



גערתיו: שוב לבב / ואל תהיה שובב! /
הלא כמעט סבב / לך הכליון



ואיך תאהב מטה / ומדעת תשטה /
ואל תבל תטה ־ / והיא כחלום חזיון



ואיך חיים תבחר / ותדע כי מחר, /
אמת, ידבק אחר / אלימלך כליון



ותשמח ברננה / בארץ כמלונה /
כמו שמחת יונה / עלי הקיקיון?



והאדם הלך / ובחשך הולך / והנפש
מלך / והגוף אפריון



והמות יגל / וישלח המגל / ונרץ
הגלגל / אלי בור השאיון



ועיני גבור טח / ולשמש ישטח / ועל
קשתו יבטח / ויתעל בסריון



בכל יפנה ירשע / וכל היום יפשע /
בצדיק עם רשע / ועשיר עם אביון



וחיים בהרג / יקפד כאורג / ובהם
ישתרג / כהשתרג שריון



ומהם לא ישגא / למלא בם כל גיא /
ושית ישע לגא / ותת בוז לגאיון



ויאכל כחסיל / בני עיש וכסיל / וגם
בער וכסיל / לשניו נקיון



והראני חלד / בעודני ילד / אשר
תבל תלד / בטרם הריון



ולא מעד מעוד / אשורי בה מעוד / כמו
הור היה עוד / לרעי נסיון



בעת בא בגבורה / מסעף כל פארה /
ולקח הצורה / ועזב הדמיון



ועיני בוכיה / ונפשי רק עריה /
ועלתה פוריה / לגיא החזיון



ונשארתי חרש / כעץ מאין שרש /
ונכרת החרש / אשר היה חביון



ואנחתי רבה / במוט מוט הקבה / ועל
ארון נשבה / ויקר הפדיון



לבבי הנמזח / ביגון לא יזח! / עזב
שירי מרזח / והמר שגיון



וכף על כף הניף / וידך אל תניף /
והורד הצניף / והסר הגליון



קרע ופרם כל מד / למקדש כי נשמד /
והיה כל מחמד / לעינינו כליון



וקונן על מקל / פאר בפז נשקל /
והחלד נקל / בעיניו מדפיון



תהי שלום נפשו / בסוד אל מקדשו /
ויזריח שמשו / כדברי הגיון



תמול היא שוכבת / בחיל ובמעצבת /
והיום יושבת / בצל סתר עליון.
