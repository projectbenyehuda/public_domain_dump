


	
	אחלה פני אל
	








אחלה פני אל / צור ישראל / אקוה פדות אל / ביום זכרון



בזכרי זדוני / יפלו פני / יחדשו יגוני / בבואי לזכרון



גדלך אקוה / מעז ומקוה / הסר מעוה / מספר זכרון



דרש נא עדתך / ברב הדרתך / ברב צדקתך / נקה ביום זכרון



המוני מרומים / בקשו רחמים / על גזע תמימים / ביום צדקה וזכרון



ואבות קדושים / הרבו רחשים / על עם חבושים / שוטחי זכרון



זעקה הרבו / על עם נשבו / ובגלות החבו / ואין זכרון



חנון חסדו / יעטה כמדו / לזכר עבדו / ויהיה זכרון



טעותו יפר / וזדונו יכפר / וימציא לו כפר / ביום זכרון



יגלה צדקו / לנחלתו וחלקו / וימחה דוחקו / מספר הזכרון



כובש עוונות / העבר זדונות / ונפשות אנונות / חבש ביום זכרון



לחסדך אפנה / פודה וקונה / כי לי תענה / ׳סלחתי׳ ביום זכרון



מחסה ועז / בך צור אעז / כי אתה מעז / לדל ביום זכרון



נורא ואים / המצא פדיום / וסלח היום / לעמך ביום זכרון



שגב סגלה / מכל מחלה / ותהיה נחלה / לחק וזכרון



ענה לנחשלת / תערג כאילת / כמו דל על דלת / תעטף ביום זכרון



פתח כבלה / וישר שבילה / ויהי חבלה / חקוק בספר זכרון



צפה לנפזרת / ביד צר נמסרת / ועזר גברת / ותהיה לזכרון



קבל שיחה / והכן ארחה / ויערב לקחה / ביום זכרון



רחם צנועה / נדודה ונועה / תרבה שועה / לחפשה יום זכרון



שובב כמטרם / עם דומה לכרם / ותשליך בתוך זרם / חטאם ביום זכרון



תמציא מנוח / לעם הולכי שחוח / כאשר לנח / ושמתו לזכרון.
