


	
	[אולי דמעות]
	



ולה איצא

אולי דמעות בחרי נדחו / השקו לחיי עד אשר-צמחו



יתמהמהו כי-יֽגעו מרוץ מעט / ובשור אנחה תרדפם יברחו



ליל נגרו עיני כאלו הדמי / בשכר עמידת כֽוכביו לקחו



לו תחזמו תחזמו אחרי / מסך תכלת אחריו יזחו



תקע לבבי עת בכותי כף לבל / יכלו וכמעט קט בגדיו קחו



לולי אשר סגר יגוני בעדי / שחו שכני כי בדמעי שחו



תבכה אחי מוסר ימי רעה ולא / תבכה ימי-נער אשר נלקחו



ימים אדמה כי פניהם חצבו / מדר ומלשם שבו נתחו



מה-קֽדרו ימים עדי לא נכרו / לילות והלילות לפנים צחו



אפגש מידעי באנחותי אשר / לו פֽגשו הרים אזי נסחו



יום רֽצחו דודם במזרק שֽפכו / בו דם אחי ענר ולא-רצחו



אז נֽפחו רוח אלהים בי ואש / לבי בריח מר-דרור נפחו



מה-מֽצאו קירות לבבי באמר / שוא חֽשקו חשק ותפל טחו



כי חֽשקים בראות אנשים ישחקו / ובאהלם יבכו ומר יצרחו



בשתחות שמש לחשך אזכרה / חשך לשמש הלחי ישתחו



ומשחתים כפרם אני כי-יֽרדו / לנגף ובראות דם אזי פסחו



איך נהפכו פנים כתפוחים ואיך / היו אדמים אחרי טפחו



נפש עמיתך יֽצאה עת יֽצאה / נפש בשמיך אשר מלחו



פתח ידידי מֽוסרות חלל אשר / עליו תלאות חרבך פתחו



ימשח בדמו פאתי-ליל עדי / כי-פאתי קדים בדם נמשחו



שלח לך לבו מקום קרבו ושם / ציריו כמו ציריו ולא יצלחו



צירים סרוחים על-כבדו שֽכבים / ועלי יריעות לֽילה יסרחו



ארך עדי קרניו אשר ארכו כמו / קרני ראם כימה ועש נגחו



ליל כֽתבו עטיו עלי-שחק המון / צורות וצורות האדמה מחו



צורות תדמה עין אנוש כי נפתחו / בם שערי-שחק ולא נפתחו



שחק כמו פרדס אדמה כֽוכביו / צצים ואולם בסתיו פרחו



ידמו בגביהם לברקת ופז / על רצפת בהט ודר שטחו



כלם כאחד לאלמים נצבו / כאלמים לחֽרשים סוד שחו



יודו ויגידו לאדם נפלאות / אל ממסלותם אשר ארחו



מעת היותם עד-כלותם ישמרו / חקות אלהיהם ולא יזנחו



כי הפקדו לרעות יצורים עֽמדו / על משמרותיהם ובם צנחו



ובשום לבבי לתבונה אחזה / כי הֽיקום כצאן ותבל אחו



מהם רחב-נפש גדל לבב ובם / חפצו מעט ובחלקהם ישמחו



מתי תמלא הזמן לי משאלות / לבי אשר מלבך נשכחו



איך יֽדעו כחי מצוקיך אשר / ישענו עלי ובי יבטחו



עד-אן אלמדך ולא-תבין ואור / אפיץ לעיניך ולא-נפקחו



ימצא תנואות כי ישיבני אבל / תדם עדי גבעות הדרך שחו



איך הֽיקר תשמט והשיר תעזב / עד כי-שמי תפארתך נמלחו



או ישקטו היום מזמות מזמן / נח אני-נח אמת לא-נחו



אכן אצילי עמך יקלו כבוד / השיר ומימיו חֽשבו מי שחו



וביום קרב כנחשלים נסחפו / וכנוססים דרדר וקוץ ידלחו



אם-אין לך עון לבד שכל שמח / כי ערפהם יקשו ולא-יסלחו



עד-אן פתאים תאהבון ריק ולא / תודו ואנה בערים תברחו



האשמרה ככם אני מחסום לפי / בגלל כלבים אלמים נבחו



כבחן אנוש נודע ולא-נודע דבר / יונה וערב כי בעת שלחו



מה-בצעך איש כי תצוה חוג לבל / יסב וכוכביו לבל-יזחו



לא-אחריש עד-אחרש שירי עלי / לוח לבב תבל לבל-ימחו



שים לבך לזאת והעבים ראה / כי-יסרם רוח ולא יוכחו



וזכר יקר-שמש ותפארתה ושור / כי-זהריה לאנוש ישחו



ביני וביניה אשר בין פֽנחס / לקהת וכשמואל לבן-צוף תחו.
