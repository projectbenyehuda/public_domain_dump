


	
	שעה לחש קוראיך
	








שעה לחש / קוראיך / וענותם אל תבזה

לשועתם / הט אזן / כבימי אבי כל חוזה

ממרום / כהודעתו / במראה ובמחזה

והשמיע / סלח נא ל־/ עוון העם הזה





שקד דלת / ותוחלת / אזי תמצא אנוש רזה

למד היטב / בלא קטב / כפץ צורך ביד חוזה

מאס הבל / עטה אבל / עלי גופך אשר ירזה

הדף פשעך / עזב רשעך / ושים דבקך בדת כי זה

תהי עסקך / וגם חקך / בת מזה ובן מזה

וגן עדן / תהא נדן / לנשמתך ואת תחזה

תהי קשוב / שאר ישוב / שאר יעקב לאלי זה





באותי / ותאותי / אני באתי לאל אדרש

רחש מוסר / להתיסר / ותורתו אזי אפרש

במשפט בר / כמו דבר / כשמן טוב עלי הראש

יהי ריחו / ומשביחו / ברב בשם כמו מראש

ותוך קדשי / יהי נפשי / ושם אקפא ושם אקרש

ואסיר צר / אשר הצר / ויתלה כפיר בברוש

ואלי חיש / בעת תחיש / חדל אישים וגם נבזה





ודע פצעי / ושור בצעי / וגלותי באספמיא

הלם הולמי / ומכלימי / יהא סחוב בסרטיא

וחיש יומו / להתמו / כמו תמו באנטוכיא

זקר ועקר / באבני קר / נשר שבערביא

ואתפאר / בנין ושאר / כהבטיחי יליד בתיה

וכל אויבי / ומכאיבי / יפלו בארץ נשיה

ותוך מקדש / החדש / דמי קרבן אזי אזה




מעון תקר / כשתה קר / וגם החריב יאור מצור


הבלים פץ / ובם חפץ / וקם כרוז בחומת צור

דמי השה / ביום כסא / חקקוהו ובו לפצר

ובא מצד / לכל נד צד / וכל איש גש ליד מעצור.
