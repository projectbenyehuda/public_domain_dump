


	
	[במוצאי שבת]
	



ולה מרת ͘ יה פי אביה [ולו קינה על אביו]

È — — — — — / È — — — — — / È — — — — —


במוצאי שבת / צררוני רבת / ונגרזתי חפזי



ואיכה לא-אכף / ולא-אספק על-כף / ולא-אמר רזי



ואשתה קבעת / ואסיר טבעת / ואשליך כומזי



ואשכח מאויי / ואמאס את-חיי / ואבידה עלזי



להוה נצחת / ואש מתלקחת / בצמרת ארזי



ומתי אתאפק / לשבר מתרפק / והמות גוזי



וכביר ירוע / ולא-נכר שוע / כדברי הבוזי



ארון האל נלקח / ביום הודי לקח / ותפארת עזי



ורוח מעשו / נשאתהו כנשא / יחזקאל בן-בוזי



אבל היה רע / אמונת הושע / וכלב הקנזי



באור חיים יאור / ועליו ישא אור / אלהי מעוזי.
