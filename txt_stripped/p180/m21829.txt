


	
	תחלת החכמה
	








תחלת החכמה / וראשית הערמה / ליד אל הרמה /
ויוצר היצורים



וראשון בתחלה / ואחרית כל תכלה / מרומם בתהלה
/ ומשפיל אף מרים



אשר לא יתוה / במעון גם נוה / ואיכה ישתוה /
בשבת נוצרים?



וכל דעת נוצר / להגביל אל קצר / ולבם כים צר /
ודעות כשקרים



אשר מאז קדם / והכל סר נרדם / ועד לא שם אדם /
ולא יצר הרים



ושם ארבע ידות / נכונות מועדות / ולהנה סודות /
צפונים וסתורים



וצורם אבירם / אנוש מהם קרם / ויחד החבירם – /
ואינם נחברים



ויזם וישלים / עשות הגלגלים / כתבנית אהלים /
כמעשה ספירים



פלאיו לדעת / ואור מארבעת / שחקים לגעת / לעיני
היצורים



והציג כח בם / כפי מתת רבם / לבל ימוט רבם /
לדורי הדורים



ושת בהם תבל / תלויה בחבל / ושת תלה סבל /
במאזני מישרים



ובמרוצת מרץ / ירוצון בחרץ / להעמיד תל ארץ /
יצוקה מעפרים



ורוח עושיהם / תרחף ביניהם / ויאחז כנפיהם /
ויהיו ננערים



ופרש את צלם / עלי כל העולם / וחצץ את חילם /
למחנות וסדרים



והוא הוא התואם / שנים אב ואם / בשמות יקראם /
ואינם נעדרים



וחוזי כוכבים / להודות סרבים / ואינם קשבים /
וממוסר סרים



וכחדו אל שדי / באמרם כי יש די / בארבע מוסדי
/ והם העקרים



להמציא הנעדר / באל חי הנאדר – / והוא קוץ אף
דרדר / לרגלי ההוברים



הלא הנה אותות / לרגלם כרשתות / ואל יהיו שפתות
/ עלי אל מגבירים



היוכל תת צורה / אשר עוד לא נברא / אבל האל קרא /
וגלה נסתרים



וחדש חם יפצר / עלי קרית מבצר / ורוח מאוצר /
וקרה ממזרים



בשום האד נפרד / ומארץ נטרד / וישוב במורד /
עלות הקיטורים



ועוד אות כי נטר / אמת, כי לא אטר / עננים ממטר /
והיו נעצרים

אשר הם נחמה / ואף הם לנקמה / ובהם למהומה / ומוסר
לסוררים



ובם ידין עמים / לקצף ולרחמים / ואכל מטעמים /
ישו למכבירים



והעת ימנעו / רביבים – יכנעו / יצורים, כי פשעו /
והלכו נקדרים



וערבה כל שמחה / ביגון ואנחה / ושיחה וצוחה /
ומספד תמרורים



ומשקה כוס רעל / בקצף ובגעל / ויגון נקשר על /
לבבות נתרים



ואיש כשור גועה / והכל סר תועה / ולא מצאו מרעה
/ לצמד אכרים



ואנשי תושיה / ישיחון בשאיה / וקול כנהי
נהיה / לשבים ונערים



וערגו אילים / ועטפו עוללים / ושבו עולים /
תשובת הברים



ועלתה שועתם / לאל ראה אותם / ושמע אנקתם /
והראה נסתרים



והחסד לבש / והעוון כבש / וחתל אף חבש / מזורים
למזורים



ונשא הזעם / ונפל הרעם / ובא בן אבינעם / במים
כבירים



וקולו עז קולות / יחולל אילות / ולנו הנגלות / ולו
הנסתרים



ומלא העבים / במלקוש ורביבים / והריק לשבים /
בדיי מחסורים



ובלע המות / בארץ צלמות / ולמאד עז מות / בעיני
נקברים



ושמח כל יוגב / וכל אנשי רגב, / וארץ הנגב /
ירוה ביאורים



ושקט לב הומה / ולב רעב, צמא / ובמה יש, במה /
להקריב אשכרים?



ומה לי קרבנות / כנעם הגיונות / ושירי שגיונות /
בפי ראש השרים



שמואל, המורה / פתאים, הפורה / תבונות, היורה /
חניתיו כמורים



אשר בו נכבדות / נכונות מועדות / ולו עשר ידות /
עלי כל המורים



וכשמואל היה / בדת ופליליה / ואנשי תושיה /
למולו נמהרים



ופתח גם השרה / אגדות המקרא / ובדק בתורה /
מלאים וחסרים



והשיב לנכוחות / קמוצות ופתוחות / כמוריד על לוחות /
עשרת הדברים



ואסף נערצות / בדעת מועצות / עלי כל הארצות /
דבריו נזכרים



ומדברו להב / ומכתבו רהב / כעדיי הזהב /
ישורר בשירים



יגלה ענינים / צפונים וטמונים / והיו נבחנים /
ככסף בכורים



והתיר נאסרים / ואסף נפזרים / והוריד נבצרים /
ופתח נסגרים.
