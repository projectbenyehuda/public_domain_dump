


	
	[תהלת אל בראש כל־התהלות]
	



לה פי בעץ תלא… ל. אנדלס והו קד ולי אלחכי… [ולו אחד מתלמידי (החכמים) של
אנדלוס כשהוכתר…]


תהלת אל בראש כל־התהלות / וצדקתיו לתנות הגדלות

אשר נעלה עלי־ערץ ונשגב / ושת בינו ובינינו גבולות

ימינו רוממה במאד פלאיו / אשר גדלו עלי כל־הגדלות

אשר תתמך אדמה על־בלימה / ותחריב בעזוזה המצולות

אשר תעור ותפדה הישנים / בגלותם ותחיש הגאלות

לשה גלה ומבנים שכולה / וגלמודה שכולה משכולות

נדודה מהנחה לאנחה / אבלה גם־בנותיה אבלות

והיום לה כמו־אלף בגלות / וכמעט קט ואלפים כפולות

בכל־כחם אכלוה יריבים / ועודם ידמו שרית נטולות

בעצם עצמוה העצומים / עלי חילה אשר היה נחלות

והיא מיד ליד נדחה ושחה / בשוחות וחלקלקות אפלות

בחבלי העני תשב אחוזה / עניה פעמתיה תמוטט כבולות

מפרק על פרק על צורנה / והמוטות תמוטט באילות

ומשכרתה לגלותה תשלם / ותשב־לה גמול משיב גמלות

ותבנה מהרוסיה בצורות / ותקים מנתוציה נפולות

ותבעל מבנותיה עזובות / אשר היו לצריהן בעולות

וגלות חל מדינה העדינה / תחדה במכלולות כלולות

אשר גלו אלי ארץ ספרד / והשיגו והשיגום קללות

ומשאת נֽשאו כי־את גזרתה / והכילו עדי אין־בם יכלות

רעבים מבלי־מים צמאים / כאילות חשוכות מאילות

ולולי נחלו נחל אמונה / שארית יעקב עשו פלילות

אזי בם נעקר עקר תבונה / ויבשו האשלים השתולות

והיו האסורות כפקורות / והיו הכשרות כפסולות

אשר העמיד יסודות משפטי־דת / והשים מפלותיה נטולות

ולנו עם ישרון בחשכות / והשכימו נאורי האפלות

בדין דין אמת הלך נכחו / והצדק לבקש גם־פעלות

אשר יפסיק לדין ראוי להפסיק / בצדקתו בלי־און ועולות

וממתין בו ואם־זקוק לנחץ / כאזהרת ישישי העגלות

ודינו מבלי טעות מתקן / כארח שאבניו בו סקולות

והקלות השיתן כחמורות / וראה החמורות כנקלות

ואם־ישפטו נצים לפניו / ומתעולל אחדם
בעלילות

יזכה זה ואם־את־זה יחיב / ולא־ישא פני פנים צהולות

וכל־משפט לאור יוציא בצדקו / ומשפטיו וארחתיו פלילות

ובמקום השבועה שם פשרה / ובה גזר מקום גזרו באלות

וימציא תעלמי האבדות / ומהעֽשקים ישיב גזלות

ושם מלוה לאיש עני להשיב / בעוד שמש בגדיו החבלות

והלביש מערמי התבונות / והיו היחפות בו נעלות

ופתח לחכמים האטמות / ומי יפתח דלתות שם נעלות

והוא נבחר בעת אין־שם זבחים / לאל משי וקרבנות ועלות

הלא הזהיר עלי־צדק וצוה / הירצה בעגלים או עגלות

אמת כי בו עונינו מחוים / וחטאת דֽרשי חפצו מחולות

וכל־חכמות לחכמתו עזובות / וגם מבעליהן הן שלולות

ונקראות תשובותיו בבבל / ונפרשות בראשי הקהלות

בסוד ראשי נהרדעא וסורא / עירות הפלילות הגדלות

הרימוהן עלי כל־התשובות / ובזהב כתבום במגלות

ואמרים מצאו אנשי ספרד / פליאי התבונות הנעלות

וחזו האמת נכר ברבם / וחזינו אנחנו מהתלות

ולבשנו לעמתם קרעים / והמה לֽבשו חמדת שמלות

והיתה גאותם כאריות / והיינו לעמתם רחלות

ושריגי תבונה הם קטפום / ואין עלל וקטפנו מלילות

ועיני עֽוריהם נפקחו־בם / וסכליהם ואזנינו צלולות

ברב הרבה ששוניהם וחדם / וחדותם להגלות תהלות

ורב האיי כלא היה לפניו / וקטן מענות אתו במלות

מזמותיו יפיצון כל־מזמות / ותחבלות לכל־אדם חבלות

וחכמות שֽטרי חכמות גדלות / אשר הורה עלי דרך גמולות

פני רב למדנו הכעורות / והשגנו נחוצי הבהלות

ומלא הנפשות היבשות / עדי היו במי־הבין טבולות

במים לא אנוש ישחה בדכים / ואם־תרד לסוף תמצא מצלות

וחכמתו רחבה מאדמה / ואם־יש־לה גבול אין לה גבולות

ופצל מפליאיה פצלות / כאיש פצל ליחם־בם במקלות

ותלדנה תבונותיו בלי־ציר / ולא קשות ולא רגזות וחלות

וכל־עת הן מחדשות ועלות / ואם־הנה מישנות ובלות

אשר הנה לכל־צמא ועיף / מזי רעב כצמוקי דבלות

ודעות הגלויות לחכמים / והוא ידע לנסתרות ונגלות

ובהם יפצחו רנה ידידיו / ותלמידיו לשנותם צהולות

והכל פֽצחים רנות לשנות / ובלבבם לרבם רב מסלות

אשר הזהיר להדליק נר נשיהם / והפרשת תרומיות וחלות

ומצותיו וחקותיו ודתיו / מישרות נכחות כשפלות

ותוכחותיו לכל־ראשים עטרות / וכצפירות עלי קדקד סבולות

אשר הנה בעינינו בתולות / ועמנו וכמנו שגולות

והן רפאות שרירי נססיהן / ולמזרות מזרות גם־תעלות

והנה לֽמגששים כמאורת / וכנרות דלוקות באפלות

ומימי אב אבי מלאך עתרות / ותלמיד רב אשר הנחיל נחלות

אמת לא קם בישראל כמותו / אבל כי־איש בגבעת הערלות

אשר מל הלבבות הערלות / והשכיל הנפשות הסכלות

וקרע כל־סגורי לב אוילים / ונפשותם מפלשות חלולות

וכל־סתום וכל־נעלם ועמום / מגלות הן לתושיות ומפלות

והוא אחד ואין שני לערכו / ונפשו בם כמלך בחילות

עליליה יקרה מתבונה / אצולה מאצילי האצולות

מהירה לעשות תחבול ואלו / חרוצה נחשבה מהעצלות

לכן יקל ויקטן מכבודו / אגר שבח וחבר מהללות

ואם־אגיד קצת הודו וחסר / דלי מיאור ונטף מתעלות

ובראתי יקר השיר ועזו / הכינותי לנוהו מסלות

והעדתיו בעדיי הבדלח / והענדתיו עטרות מתהלות

ואמרותי אמרות הטהרות / במאזני־שיר ומשקלו שקולות

ומצאתי בצרות כעזבות / וחשפתי במחשבתי פלילות

והיו יֽדעי שפה ברורה / וכל־אמיץ לדבר צח במלות

סחרחרים ולא־מצאו ידיהם / והנה לעזוז ידי אזולות

ומשלתי לשון עברית ועזי / כבדיה לעמתי נקלות

ועשיתי לפוציה מסלות / כמו־כן לעצוריה תעלות

ולו עכר עכרני עכרתיו / כחדקל וכפרת בהמלות

ורגלי בה כרגלי הראמים / ורגליו בה כרגלי הנמלות

ועננהו וכסהו אבקם / ולא־יכל לצעדו בנעלות

והיו לי לביאותיו כשיות / וגם־נפלו לאתותי קטולות

ואל־תתמה בשירות היקרות / תמה מהשמנה הכפולות.
