


	
	שחקים רקע
	








שחקים רקע / ארקים תקע / ניצוץ בקע / אלי מלכי בקדש



לפת רקיע / ועץ פרי התקיע / מאורות הבקיע / השתחוו ליי בהדרת קדש



מקוה ויבשת / גם איש ואשת / כלה בששת / וביום השביעי מקרא קדש



הוסת נפרד / ומגן נטרד / יען אשר מרד / ואת אשר חטא מן הקדש



בוגד נזעק / לשום מועק / דם אח צעק / ונשמע קולו בבואו אל הקדש



רעים פשח / רמים השח / איתן נמשח / שמן משחת קדש



ימינו ידה / בקנה וקדה / יחידו נפדה / בצאתו ביובל קדש



הבאר נסתם / עד צאת איש תם / בכס יה נחתם / פתוחי חותם קדש



ומעת הוסר / לגור בשר / בשנים עשר / שומרי משמרת הקדש



דחויי חנס / ברחמיו כנס / הרימו נס / וקראו להם עם הקדש



הגו המוני / בהצליל מוני / מי כמוכה באלים יי / מי כמוכה נאדר בקדש.
