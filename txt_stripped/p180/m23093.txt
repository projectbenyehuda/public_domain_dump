


	
	ואומר אל תחשה
	




ואומר: אל תחשה / לבבך פן יקשה /
עניתיו: איך אחשה / וכליותי חרות



ועמי תוחלת / לבבי אוכלת / כלהב גחלת
/ תקרקר הקירות



קרבי תהמינה / ועיני תבכינה / ואולי
תכבינה / כליות בוערות



אשר אש בם נחבא / ביום הלב נשבה /
ולו תפת נכבה ־ / שטפוהו נהרות



מאד נכחד עצבי / ולא ידע לבי / בערות
מקרבי / דמעות נגרות



וזה מכתב ידי / יהי לכם עדי / ומופת על
סודי / ואותות נכרות



בסודי כיד נכתם / במו ספר נחתם / תם
/ להבין ולהורות



ואולי איש נוסר / אשר בתם נאסר /
ומדרך לא סר / לנתק מוסרות



וארום על אויבי / בנפשי ולבבי / ולו
היה אבי / גערתיו גערות



ושחח על קצפו / וכפף על כפו / ולו
שחה ערפו / רות



עדי מתי אוחיל / והתוחלת ־ חיל / ונפשי
לה תחיל / כחילי מבכירות



הכל היום אפנה / כמו נגש נענה / ולא
אמצא מענה / כפיות נסכרות



והשיב על ישאל / זמני הנואל / ואיככה
אשאל / גדולות ובצורות



זמני הלוחץ / אשר ראשי מוחץ / ירה
אותי כחץ ־ / אנחנו מטרות!



אשר קרני גדע / ואלי יתודע / כאלו
לא ידע / לבבי נסתרות



זמן שקר עמס / ובי קם החמס / והשפיל
אף רמס / נפשות היקרות



והחזיק בתרמית / ובין רעים הצמית /
ופרודו העמית / לעמת הצרות



ואינני גבר / יקוה השבר / אבל אחצב
קבר / לנפשי בצורות



ואלו חכיתי / וסבר אויתי / מעט
שקויתי / להשיג מזרות



ואיך לא אואש – / והנפש כאש / ושים
על עצים אש / יקומון תבערות



אקוה לצדקה –/ ואמצא מועקה / ותבל
כמתוקה – / ומתקה כמרורות

והיא הנוסדת / לעולם עומדת / והיא
כיולדת / והימים ־ הרות



והיא גם רע ממשק / והכל בה חשק /
למען כי תשק / נשיקות נעתרות



והעולם רועץ / והאדם כעץ / והמות יועץ
/ יקצץ הפארות



ועין לא יעלים / אנשים מהשלים / אבל הם
זלזלים / והוא כמזמרות



כצודד מגיח / אליהם ישגיח / וכלם
יגיח / אלי פיו כנהרות



יגלה אזניהם / בעת יקרא להם / ויפקח
עיניהם – / והיו נסגרות



ותבל תנחנו / עדי תניחנו / ביום
תשלחנו / לבתה בתשורות



אנחנו נפזרים / בפיה נקברים / כעצים
נגזרים / מכסים בגדרות



ואל שוכן ערץ / לך יגדר פרץ / ותהיה
בארץ / ברכה לדורות



שלומך כנהר / וכשחק תזהר / כמו
אתה נזהר / בקלות וחמורות



גער לך בדאגה / ותן ללב פוגה / ומה
תועיל תוגה / ואף על העברות?



היה דומם זומם – / לעולם תרומם / ולמה
תשתומם? / תחלץ מצרות



הלא מדד חבל / עלי אנשי תבל / להשחית
ולחבל / ולזרות ולזרות



אנוש כציץ נמל / וכקנה קמל / ומה
יתרון עמל – / ויובל לקברות!



ומות ירענו / הלא ישיגנו / ולו
ימצאנו / בבין המזרות!



ורוחו לא יצל / אשר ימיו כצל / ואיכה
ינצל – / ועליו משמרות!



והמות מסך / עלי כל חי נסך / והיה להם
סך / ועברו העברות



כאלו הוא מסעד / לכל אדם נועד / ומי זה
חי לעד / ועמד לדורות?



ומי עז התאזר / וממות נגזר / ובחיים
נגזר / ונמלט מבורות?



ושרש כל יקוט / ולא יוכל לשקט /
והאדם ילקט / כאורה האורות



וכל ישאל שאול / לבל ירד לשאול – / ומי
יוכל לגאל / נפשות מנקרות



גביר אחים תהיה – / הלעד איש יחיה /
והתבונן איה / מלכים וגבירות?



ואיה אבירים / ואיה כבירים / ואיה
השרים / ואיה השרות



ואיה כל המון / ואוצר כל מטמון / אשר
בנו ארמון / וישבו הטירות?



הלא מותם ארב / לעתם כי קרב – / ועשו
חיל רב / ומלאו אוצרות!



ויקר פדיונם / ונקל כל הונם / והמות
חנם / לקחם ממכרות



והחי כעגור / בתוך מסגר סגור / כאחד
בא לגור / ויצא ממגורות.



אשר אין לו מעצור / לך חסד יצר / אשר
רוחות יבצר / ונודע בגבורות



ויבנה ההרס / ויקים הקרס / ותהיה
כחרס / עלי כל המאורות



ימלא ימיך / ושם נר לשמך / ויעביר
קמיך / בשואות ועברות!
