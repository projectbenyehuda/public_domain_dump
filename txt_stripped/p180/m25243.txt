


	
	שחר קמתי גם
	








שחר קמתי / גם נכלמתי / לשאת לך פני

בך חסיתי / ולא כסיתי / עווני וזדוני

כי ידעתי / גם שמעתי / אל עולם אדני

בנת לרענו / יי מלכנו / הוא יושיענו





לך צופיה / בת הומיה / מבית מגוריה



תשחה ערפה / תפרש כפה / תקח אמריה



עניה תפנה / ביתה תבנה / נפרצו גדריה

על רב פשעינו / יי מלכנו / הוא יושיענו





מלך מקשיב / מתי תשיב / צפור גרשת

נא אל תבזה / ענות עני זה / כמה נטשת

בעבור ידידך / אתה ידך / גויים הורשת

ותטענו / יי מלכנו / הוא יושיענו





היום תגאל / אביון ישאל / גואל חסדיך

אם ענה בו / רע לבבו / הנו בידך

קח שועתו / וגם דמעתו / שימה בנאדך

אלהי ישענו / יי מלכנו / הוא יושיענו





חזק רפיון / מלך עליון / ורחם עמך

זכה אותם / בזכות אבותם / שלשת תמימיך

קרב קצם / וגדר פרצם / למען רחמיך

שעה שועתנו / יי מלכנו / הוא יושיענו.
