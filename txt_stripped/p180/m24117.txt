


	
	שזופי פלטי
	








שזופי פלטי / אפופי עיטי / שוקדי ועוטי / על סף רהיטי



בחורי חניטי / ילדי מעוטי / בעדת קשוטי / כללי ופרטי



תמו מקשטי / וחוגרי אבנטי / תופשי מוטי / ומנצחי ופורטי



שלמו שליטי / ונושאי שלטי / שפכו בהטי / ושדדו חריטי



בלע מלעיטי / רוש ומעליטי / בזר שומטי / וחובטי ולובטי



תטאט מטאטאי / תלהט מלהטי / תפשט מפשטי / תקמט מקמטי



ושועטי ובועטי / ושוחטי ושוחטי / ומורטי ויורטי / וחולטי ושולטי



נהר מקלטי / ומהר מפלטי / נחת לקוטי / מבין שורטי



העז משוטטי / מפולי ופוטי / השב שופטי / ותומכי שרביטי



ומיד מטי / הקל עבטיטי / ומיון טיטי / משך משוטי



אשר מבטי / וישר מעבטי / אמץ קולטי / מזדון שטי



שבר שאטי / שפר לאטי / להלבין חוטי / לסורי וחוטאי



מפצל בעטי / מקלות רהטי / הלץ שבטי / הקוראים את משפטי.
