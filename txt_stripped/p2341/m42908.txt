


	
	סתרי ענה
	



סימן: סלימאן. מנגינה: חוסיני






סתרי ענה / לדל נענה / ותן מענה / בלשוני



סלח רשעי / מחה פשעי / אורי וישעי / וקוני



אתה עשיתני / אל חי יצרתני / מאז בראתני / ואצלת אורי אורי אורי



האר עיני / ואישוני / באור פני/ך נחני.





לך אתפלל / אל מהלל / ענה אמלל / ועני



הבט וראה / לבו נכאה / השקף וראה / בעני



אדני חנני / בקראי ענני / אלהי פדני / מיד צרי צרי צרי



ריבה ריבי / נקם אויבי / מבור שבי / יה דלני.





יה אל רוחי / שמע שיחי / ואסף נדחי / למכוני



צור מבטחי / אתה גחי / בך כחי / ואוני



תשוב תחנני / אל חי הקימני / וסב נחמני / ושלח צירי צירי צירי



קל כצבי / עם איש תשבי / ישמח לבי / תשור עיני.





מלכי שעה / שובה ורעה / בטוב מרעה / את צאני



אנא אל דל / מעפר דל / אשיש, יגדל / עלצוני



אל לעד תנשני / ואל תבישני / יחיד ואין שני / הקם שורי שורי שורי



בנה מעון / ואפריון / אל הר ציון / העלני.





אדון עולם / מלך נעלם / חן דך נכלם / ורצני



יוצר כל חי / שור שבחי / בך אל חי / רעיוני



אדני נגדך / תאות עבדך / צורי בחסדך / רפא צירי צירי צירי



שמע ניבי / מגן אבי / ממכאובי / רפאני.





נר נשמתי / וצורתי / אילותי / מגני



אל נאמן / אב הרחמן / מידך מן / מזוני



אלהי עזרני / לטובה זכרני / ואתה תורני / דרכי מורי מורי מורי



נא אל יחיד / שלח נגיד / לבן דוד / נא הראני.
