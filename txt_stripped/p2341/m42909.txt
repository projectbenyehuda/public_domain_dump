


	
	סמך נפלי
	



סימן: סלימאן.






סמך נפלי / וראה שפלי / והושע נא / חי נאמן אל



חן עבדך / תן לעבדך / מענה לשון / חי נאמן אל





לי אל מפליא / האר אפלי / והופע נא / חי נאמן אל



בך חילי / רפא חליי / יושב עליון / חי נאמן אל





יה אל עורה / ופדה סורה / וכוננה / חי נאמן אל



קבץ עמך / לאולמך / ובנה ציון / חי נאמן אל





מלך נורא / בנה בירה / קרית חנה / חי נאמן אל



עוטה אורה / שים גבירה / תוך אפריון / חי נאמן אל





אל דר רומה / חושה קומה / ושוב חן נא / חי נאמן אל



גלה סודך / הראה הודך / ושלח ינון / חי נאמן אל





נא קוממה / עיר שוממה / וקומם נא / חי נאמן אל



את אימה / ברה תמה / בה אור ישכן / חי נאמן אל.
