


	
	[אדון הכל וחי עולם]
	




ں – – –



אדון הכל /וחי עולם / ועד אין סוף / תחלתו



בכתר אור / והוא נעלם / בחכמתו / ובינתו



רצון חסדו / לאברהם / ומי כאל / גדלתו



הליכותיו / במדותיו / ועל יצחק / גבורתו



מקבל ה־ / אמת יעקב / ולהקימו / בתפארתו



מקום נצח / והוד צדיק / יסוד עולם / ומלכותו



יהי רצון / לאמרי פי / והגיון לב / ויראתו



כמו חרב / בפיפיות / לצר צורר / להכריתו



אמונתי / יחידתי / תני לאל / ואחדותו



למגן לי / כאברהם / רדף חסדו / כמדתו.
