


	
	אמר אויב לקדחות
	



מאמרי הרופאים. שער שלישי: שער תפלות






אמר אויב: לקדחות דבריות / הרופא
ידבר נפלאות לרמות בהם את הבריות, / יעביר קול
בכל הארץ גם באיים / ״מי האיש החפץ חיים? / גשו
נא אלי – ואתנה לכם הנהגה ישרה / והיא קדמי
מאשמדאי מלכא דשדי ערוכה ושמורה. / שמעו אלי,
עמי, / לכו שברו ואכלו בבקר השכם ולחמו
בלחמי, / ערכו מאכל מלכים גדולים ורמים – / את
הבצלים ואת השומים / מערבים בקש וגבבא / לאכל
לשבעה בפת חרבה. / גם בצהרים ובערב תאכלו
בשר שקצים / ורמסים, אפרחים או ביצים, / וכן
תשכבו הנשים בכרס מלאה – / זו היא סגלה נעלמת
ורפואה! / זאת עשו וחיו מדי יום ביום, ויי עמכם /
ולא יתן המשחית לבוא אל בתיכם. / ומי שלא יאמין
ויעשה הפכו – / אולת אדם תסלף דרכו! / ואם
אולי יצא הדבר כרגע / כי פשה הנגע – / נשלחה
אנשים אל הגלעד ונבקש שם מעט צרי ולוט / או יין
ושכר במערת שתי בנות לוט: / כי מגפן סדום גפנם
ומשדמות עמרה, / ורפואה לבוא קרובה – לא ידע אנוש
ערכה ולא ידע מחירה. / ואם באולי גרם מזלו – /



אל תבכו למת ואל תנודו לו!״.
אמר אויב: למחלת הכבד או ללב או לראה
/ יאמר הרופא ״אני בקי ברפואתם ולבי ראה הרבה
חכמות ודעת – / בכל הארץ זאת מודעת״. / ובאמת הוא
כחמור נושא מרדעת!

כי יפתח פיו בדברי תפלות, / במקום
חכמה – הוללות ושכלות, / וכה ידבר ״לרעת הכבד /
בחכמתי אכבד, / ולכאב הלב / לא כמוני בלתי אם
כלב, / ולכל תחלואי הראה / מי יחוש חוץ ממני
לדעת תרופה נפלאה? / ובחנוני נא בזאת מה תחזו בי
/ כי שבע חכמות בלבי״ – / ובאמת כי שבע תועבות
בלבו / וכל רוח אין בקרבו! / האף אמנם כל עוד
אשר יגדיל לדבר – / והחולים באו עד משבר, / כי
מצאוהו נעור וריק מכל חכמה, / ולא יעזב מעשיו, אם
יביאוהו לכלמה.



אמר אויב: לכו, חזו מפעלות הרופא
ותחבולותיו / נושא אלמותיו, / פוער פיו לבלי חק
גוזר באמרותיו, / כי הוא ידע חלי ומקרה וסבותיו.
/ להכין עזר מזור ומרפא / יגדיל שפה וירחיב פה, /
ילך אל הבשם ברומה בבוא עתו, / ירים ידו לכתב
להראות טבעתו, / יסדר בלשון זר כי נכריה
חכמתו / להסיר את מראתו בנוצתו, / ובין מלה ומלה
זורק מפיו את רקו – / ככה משפטו וחקו, / ובדבר
שהוא קל / יחמיר במדה ובמשקל. / אז יאמר אל
הבשם ״קח את המשקה ואל החולה תביאנו, /
בערכך נפשות תעריכנו, / ושאל את מחירו /
בטרם ידעך נרו. / ואם צרה ויגון ימצא, אנחנו בני
חרין – / כל המקלקלין פטורין!״ / ושניהם שמחים
בחלקה / כאשר אנשים אשר עשו צדקה. / ומה להם
אם במותו יתקעו בשופר / מאחר אשר יכסוהו בעפר?
/ ״ומה לנו – / נקיים אנחנו!״.



אמר אויב: שמעו נא כל העמים / דברי
הרופא הסכל והוא ישכן מרומים / אשר יאמר ״בן
מלכי קדם אני, בן חכמים״. / מיום ברא אלהים אדם /
מי הגיע בחצי ידיעתו בטבעי העשבים לבוא בסודם?
/ גם בשמים הנפרדים / לכמה מעלות עולים
ויורדים – / כל אחד בשמו / הוא ידע מקומו /
ובמעשה ההקזות / הלא יאמר ״שכלי פעל כל זאת! / כי
הבינותי להבחין בין שעות ורגעים / הטובים
והרעים״. / והוא לא ידע מולד הירח ומלואו, / כי לא
ידע מבואו. / ובימים של סימן / יתן קולו ״גם אני
ידעתי, ויום שנתלה בו המן / שיש בו סכנה דם
להוציא / אם לא בברכת יין והמוציא״.

אמר אויב: כמה הרשיע הרופא לדבר
פתיות ובל ידע מה / להעלות עצמו ברב חכמה, / אוי
לאותה בושה אוי לאותה כלמה! / גם כי ישאלהו
איש מהו גדר הרפואה, / אז יענה עזות ״הממני דעת
פליאה? / ומה זו שאלה? שנים הם ואתה תחזה: /
גדר מזה וגדר מזה״. / ואם יבקשו ממנו על
הקדחת / ישיב כי היא באפס עצים מתלקחת. /
ובעונות חלוקות / שוקדות ודבקות / יגזר ״המשפט
לאלהים, הוא נעלם – / זה לא ידע אנוש מעולם!״ /
ובהתחלת החלי ובעליתו / והעמדה ובירידתו – /
מי זה בא עד תכונתו?״ / ועל ימי הגבול והקפו / ומה
יהיה בסופו – / כל דבריו דברי תהו / והבל הביא גם
הוא!

אמר אויב: שגגת הרופא עולה זדון / וריב
ומדון, / מן המת אומר שהוא חי ומן החי מת, / אין
בפיהו נכונה ולא דבר אמת, / אשר מלא כפו קוצר /
כעל סלסלות בוצר / הוא משרת אל מלאך המות
ושלוחו, / זה כחו. / יקבץ ספרי רפואות חדשים גם
ישנים / מכמה מינים, / משום מראית עין יחזיקם
בביתו / ולא על קריאתו. / ואם יש בם שגיאות – לא
יבינם, / יעמדו כאשר המה ואין מי שיתקנם. /
ואם ישאלוהו פרק או מאמר נבחר – / ידחה אותם עד
למחר / והיה לעת מצא יאמר אליהם / שישובו
לאהליהם / ״וכי לנסותי אתם באים? / האיש כמוני
יענה לפתאים? / ואני הרופא לכל תחלואים!״
אמר אויב: זה הכלל בשחיטת רופאי



הזמן – / שכר הרבה לטל, / ויקרא רופא-אמן. /
ולמלמדים – להם אזהרתם תחלה: / כל המתעסק
ברפואה ואינו נוטל שכר – חיב סקילה! / למדו קיחה
קיחה משדה עפרון / ובזה יהיה לכם תועלת ויתרון,
/ משכו וקחו לכם גם אתם / ושחטתם בזה
ואכלתם. / היו נכונים לבקר החולים / ולא בדברים
בטלים / להיות עזרם ומגנם – / במחיר רב ולא
בחנם! / וזה יהיה לכם האות: / חזקיה גנז ספר
רפואות / שלא היו נותנים לרופא שכר עמלו / ודוחים
אותו לאמר ״עד כי יבוא שילה / שוב לביתך לשלום /
ולשאל שכרך אל תקרב הלם!״ / לכן הזהרו
בפעלתכם / ובית החולה אל תבואו עד אשר תרימו
תרומתכם, / ובידים ריקניות אל תצאו מן הבית /
במעט כביצה או כזית!
