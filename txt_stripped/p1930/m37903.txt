


	
	שמע נא מלך
	







משל העולם הזה






שמע נא! מלך אחד היה לו פרדס רמונים
וערמונים / והושיב שם פועלים שונים / להרים מכשול ולהסיר קמשונים / והבטיח משכרתם מונים / הישרים על דגליהם חונים / וגדרים בונים / הגפן סמדר / ולהם עז והדר / על משמרתם עומדים / ובעבודתם חומדים / באדניהם לא מרו / ומים קרו ומשם לא סרו / עד כל עץ נשא ענף / ותחתיהם שכנו כל כנף / אליהם נסים / ותחתיהם חסים / שמחים וששים / הדודאים נתנו ריח / וכל עץ הפריח /
סעפותם ארכו / ולפרים ברכו / והידד כדורכים צרחו / פועלים כזאת על אנשים ישרו / וכל בעלי השיר עליהם ישוררו / ואשר שכנו בצל דליות / ישכנו במרומי עליות.



אבל פועלים אשר במלאכתם עצלים / במלאכת חקם מועלים / ובשלום שואלים / כל אחד בכורה בתאנה חטף / ראש ינקותיו קטף / הן פועל כזאת כלמה עטף / או כי הכרם לא עזק / ויפקדה בבזק / ולא שמרה מנזק / תוכו נחר / בטוב לא בחר /
הגפן מתרושש / והוא עצים קושש / ועל כי בהשתדלות המלאכה בושש / ערוגת צמח עבשו / ושרשיהם יבשו / ושם שכן כל הולך על גחון / בין הפועלים קול ענות מהומה / ובידם אין כל מאומה / וקח על מים רבים צפצפה שמו / ופועל כזאת בראשו דמו.
