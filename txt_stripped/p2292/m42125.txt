


	
	הוי קול
	



קינה לר׳ יחזקאל ישועה






הוי קול כחולה / מספד יללה / נשמע בבהלה / זרם הגה ונהי וקינה



על רב פעלים / רחמן לדלים / אב לאמללים / ולכל איש מר נפש ונענה



צבי ישראל / שר יחזקאל / קראו האל / ויטשנו ברב מגנה



במר אצרחה / ראשי אקרחה / שמחה אשכחה / ואבל אערך באנינה



את מהללו / כל לא יכילו / מצות עמלו / עובד מיראה ורב אמונה



כל היודע / מדיו קורע / נכרי כרע / כאבל תנים ובנות יענה



גדל שברנו / צדיק ממנו / נפקד ואינו / עמוד הימני גם ראש הפנה



בצל שכינה / נפש עדינה / גם שאננה / במנוחה נכונה תהיה חונה.
