


	
	שבים ועוברים
	



מצבת אסתר אשת סי׳ שמעון אלקנווי בווהראן איר תרנ״ג






שבים ועוברים / בארח קברים / עמדו בצירים / ושאו הגה ונהי וקינה



קינים וצירים / וקול שברים / ובמקום שירים / אשא הגה ונהי וקינה



כי פה נקברה / גויה יקרה / בנשים שרה / יראת אלהים נפש עדינה



רבת פעלים / רחמנית לדלים / אם לאמללים / ולכל איש מר נפש נענה



קול מר אצרחה / שער ראשי אקרחה / שמחה אשכחה / ואערך אבל באנינה



לחמה ברה / אסתר המאשרה / זכה וישרה / יקרת רוח רבת תבונה



בדמי ימיה / נצמתו עלומיה / כי רבו ציריה / על בתה נקטפה כשושנה



סורה גלמודה / הלכה בלי חמדה / והיא בעודה / רכה ענוגה ורעננה



כל השומע / את מדיו קורע / נכרי אח רע / יחדו ספדו מר ברב מגנה



בצל שכינה / נפשה תהיה חונה / גם שאננה / במנוחה נכונה ועליונה



אל יחונן עפרה / יהי לסתרה / בעד בחירה / היקר אליהו בנה.
