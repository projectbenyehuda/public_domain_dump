


	
	ידיד חכם
	



תשובת שירתי מיצחק בו כבזא.






ידיד חכם לבב / מעיר קדש תוב״ב / הוא מבני חובב / נודעים בשמותם



כליל המדעים / חרוזו מה נעים / כילדי שעשועים / בחיקי אמותם



קראתיו חרוזים / כי מאזניו נלוזים / המה שוכבים הוזים / לא הטו כפותם



אמר יושב כסאות / מי יבין שגיאות / אדון בלי ראות / נלכד בשחיתותם



וגם עטו הדק / הנה הוא לא צדק / ואותו לא בדק / ושם עליו חותם



הנה לשון אינו / לא עמד בעינו / במקרא איננו / בכל מקומותם



ומלות מעתיקים / בעלי צחות מרחיקים / ואף כי להקים / אותם בשיר מכתם



שר פיו נגדי פצה / כי בלתי לחוצה / תושע רצוצה / ואתה עניתם



תנוע עם בלתי / יצאו לעמתי / יצעקו לקראתי / יום שלפו חרבותם



לבבות חמודות / עניתי בכבדות / בצדק נוסדות / במקום תחנותם



הן איננו אחות / לבלתי בלוחות / וזו מזו לוקחות / ברצון משרתם



תנוע רעיתי / בצדק אמרתי / אני לא מריתי / עווני לא נחתם



עדות נאמנה / גפן רעננה / וזית ותאנה / בדבר איש יותם



בין השופטים הובא / דברו לא נחבא / דברתו כתובה / תצהיר בין שורותם



שירתי בדיצות / קרא ובמועצות / והבן במליצות / ובא עד תכליתם



ובעינך חן אמצא / תשיב לי בעצה / בשיר ובמליצה / עם כל נתיבותם



במליצות ערבות / רמות ונשגבות / מהצור חצובות / עמקה חידתם



שמחה וששונים / לידיד איש אמונים / ובניו נהנים / יושבים על כסאותם.
