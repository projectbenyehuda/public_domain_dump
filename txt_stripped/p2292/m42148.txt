


	
	על מה שר
	



שיר ששלח אלי יצחק בו כ׳בזא תרנ״ג חשון מעי״ת גאבש ותשובתו בפנים.






על מה שר / בלתי אוכל בשר / גם מעוף השמים

הן הסכין בדוקה / כדת וכחקה / בבשר וצפרנים

ועל הטרפיות / עיני השוחט צופיות / בפתח עינים

והנפוחה אסורה / נתונה או מכורה / לכושן רשעתים

והשוחט בקי / צדיק וגם נקי / מזקק שבעתים

ועל המליחה / עין האשה פתוחה / בחריצות ידים

אם כן איפא / למה יושב פה / בנקיון שנים

ומדוע / נפשו תנוע / אל דגי המים

חיש תענני / ומהר תשיבני / בשיר ומצלתים

תשובה נצחת / את כל ונוכחת / במתק שפתים

וטובך אחוה / ואקד ואשתחוה / מולך על אפים

ובכל הזויות / עד שיתפקקו כל החליות / אכרע על ברכים

והחיים והשלום / לך עטרת יהלם / איש ירושלים

סלם מצב ארצה / בשירו מי הים חצה / ויעבר ברגלים

כה דברי יצחק / חי בארץ מרחק / ערל שפתים

אבי לחם יכנה / מקוה רחמי קונה / ברחמיו ארץ ושמים.
