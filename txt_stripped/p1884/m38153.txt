


	
	ואשא עיני ואראה
	








סעודת חסידים







– – – ואשא עיני ואראה / והנה שולי הבית מלאים מהמון החסידים / על שלחן וכסא כאילים רוקדים / לחטוף הנותר מקערת קדושיהם / ויאכלוהו בכל פה כזבח פסחיהם / והודו לו כי טוב כי יטעם
לחכיהם / ועד השמימה תעל שועתיהם / ומכל עבר הביאו סירות דוגה ובשר / וכל טוב לא יחסר / כי מאת אלהים יצא הדבר / וחיש עמדו על הכלים / כמו אם שבעת הימים היו להם ערלים / והכו בדוד או בפרור בלי עצלתים / זה בא במזלג וזה ישמש בנטילת ידים / וילקקו כאשר ילקק הכלב במים / עד בלי השאיר שריד ופליט בשולים / וכנחלים נזלו נהרי נחלי דם עינב / וכתולע האדים פני שותיו / כי ביד כל איש כוס מלא מסך, בת הין יכיל הגביע / ויעבר הכוס אל פיהו והריק יסיע / וישנה וישלש היין אל קרבהו / לאמר: זה אלי ואנוהו.







אז בהלך היין אל ראשימו / וערפל חתלתימו וצר ואור חשך במו / עמד הקדוש ממקומו / ודרש קבל עם בסתרי קבלה / וישביעם בשבועה ובאלה / להדבקה בו בכל ימיהם / ולהשיב לו כספיהם / כי אז טוב להם ולבניהם אחריהם / ויפתחו להם שערי עדן ומבואיהם / כי הוא הולך לפניהם / וידבר אליהם שלשת אלפים משלים / דברי רוח והבל הבלים / לציץ נובל ולאבק פורח נמשלים / ויראם את המלאך טטרי-אל / ומימינו מיכאל ומשמאלו גבריאל / והיה חוזה ומתנבא כשמואל / ויצוה לשמע מפיהו תורה / כי ברה היא זכה וטהורה / כשמן כתית המנורה / ולעשותה עטרה לראשיהם / ואור לנתיבתיהם / כי אז בל תגוף באבן
רגליהם / וישבו בטח ואין מחריד במחניהם / ואשר לא ישמעו לקולו אוי להם ואוי לשכניהם / כי בנער ימותו שניהם / ויעשה בתיהם קבריהם.







ככלותו לדרש שרו עוד זמירותיהם /
וקול רעש אופן השמיעו בגרונותיהם / עד אשר תרדמה נופלת עליהם / ועל הארץ הציעו משכביהם / וישכבו כל הלילה אצל קדושיהם / והוא האציל מרוחו עליהם / ובתמהון לבב ראו מלאך גבריאל עומד לפניהם / ויוליך השמימה את נשמותיהם וישקל שם כל חטאותיהם. – – –
