


	
	הזרע והלחיים והקבה
	



Efraim da Modena

]חי באמצע המאה הט"ו]





הזרע והלחיים והקבה


אחי שמעו קולי / רעי האזינו לי /
הקשיבו מלולי / בדעת ברורה



פי אפתח במשל / שוֽמעי לא ישל / ובשל יתבשל
/ כבשר בסירה



רק ולא יאמין / אפיקורוס או מין / יתעבד הדמין /
אחריתו מרה



יוסיף ימים עשר / דעה בינה כשר / כל־מעשיו ישר /
לו תהי המשרה



מאזין אמרי־פי / ולא ימצא דפי / הוד הדר ויפי /
לנפשו הטהורה



בינו את־שירתי / חרשו בעגלתי / השכילו אמרתי /
אשר היא ישרה



נוראות אשמיעכם / נפלאות אראכם / דעה אחוכם /
בדרישה חקירה



יתבוננו שמועות / בשלשת הרעות / ראשיתן זועות /
וסופן מארה



ואשר יפתה / ערוך אליו תפתה / בעציצו ישתה /
כל־שנותיו עברה



אף נאספה שמחה / וגילה ורוחה / ויום המנוחה / הוא
אליו לזרא.





בער יסיתהו / צחוק ויכלהו / נסתחפה
שדהו / עצתו נבערה



מאכלו לא לשבוע / ולא ימצא מרגוע / תמיד ינוד
ינוע / ושוע ההרה



מנדח לאפלה / התקצף בקללה / ופנה למעלה / גם
ידבר סרה



ומעות בכיסו / כחשים לשוסו / ירגז יראה כעסו /
במדון ותגרה



דלות וחסרון כיס / שראצר רב־סריס / נרגל רב־מג
עכיס / אף זעם וצרה



יחישה מפלט לו / יגעה על־בלי־לו / יחנן את־קולו /
כאיש קדוש נורא



נחוץ בית נדיבים / בשער בת־רבים / לאסף זהובים /
וערותו ערה



אך ישוב לאיתנו / וישב על־כנו / בכיס ערו עינו /
לפניו בור כרה



חושה אל־תעמד / הדמים אל־תמד / מהר יתחיל ללמד /
בספרי וספרא



זאת יפסיק רגע / יד אחרת תיגע / וימצא כל־נגע /
למות נפשו הערה



קורא במר נפשו / צועק אוי לו ראשו / ומצחנת
באשו / נמלאה הבירה



חנון לא יחונן / כל־ימיו יתאונן / ובני ביתו כונן /
בשבר וסערה.





ורעה עוד מזאת / אשר ראוי לבזות / נבלה
גם לזות / צרה כמבכירה



אשר אליו ינעם / לאכל רב מטעם / וגם בתוך העם /
הוא מעלה גרה



דבר יישר אליו / אף כי אין בעליו / יצעק בחבליו /
יחיל כמו הרה



יאכל בלא עתו / אם נתמלאה סאתו / לא יוכל שאתו /
לבני ביתו יקרא



כי אחזו שבץ / לרב אכל קבץ / תחת משא רבץ /
ונפשו בוערה



את־הארץ השקה / ונפשו רוקקה / בוקה ומבלקה /
כל־מאכלו ירה



ואל־המרקחות / עיניו הן פקוחות / ושפתיו פתוחות /
פיהו כפי פרה



ואם יאכל בשר / לבדו אל יישר / ודג גדול יכשר /
עמו לחברה



ורב המטעמים / יכלו הדמים / אחזוהו אימים /
מכתו מרורה



ימכר את־לבושו / להחיות נפשו / ישתה ישכח
רישו / לא תחת בו גערה.





ומכת הזנב / אשר אליו יענב / וגנב
יתגנב / יחכם התנכרה



יצא בעוד לילה / באישון ואפלה / מתכסה בשמלה
/ ובידו תשורה



וילך אל־ביתה / חביטא מברכתא / תשיתהו בתה /
מפרקתו שבורה



יקרב אל־אשת איש / כל־משפחתו יבאיש / בן
מחפיר ומבאיש / ויערה מקורה



יקרב אל־הגויה / מכתו טריה / אוי לגויה / אשר
בה הערה



יקרב אל־העריות / ואל־שתי אחיות / לאוחים ונמיות /
יֽהיה לברה.





ואל־כל־המכות / תמצאין ארוכות /
ורפואות ערוכות / לכל־גו נשברה



כי עת אשר יצחק / ולו תהי לחק / עניו אותו ידחק /
ויובל כפרה



כי ישאר ערום / תחת שבתו ברום / ותולעים ירום /
ישתה מי מארה



ימלט אי רוקקת / כגפן בוקקת / נפשו לו עוקקת /
ברעב צרורה



גם את־אשר יצרו / ואש מבערו / ופיו מרה מרה /
לאומרת סורה



אז כי יהיה תפל / נשתו ימי עפל / עומד אבל אפל /
תבוא לו מגורה



אך איש בעל קנה / יפרץ ולא יבנה / בית האופים חונה
/ בו בחר לדירה



עד פיו עפר יקרב / עד מעינו יחרב / מעדן לו יערב /
נפשו היא ארורה



הן אל ישועתי / יחזק לבתי / להגות בתורתי /
לעבדה ולשמרה



אפרים בן־יואב / בשרו לא יכאב / ונפשו לא תדאב /
ביראת אל קשורה



אקרא אל־אל עליון / אשר לו עז חביון / בנחמת ציון
/ אזכה לראות הדרה



יקבץ פזורים / יכנס עדרים / ישראל חברים / יובל
שי למורא



בו אשים מגני / והוא יעזרני / והוא יזכני / ליום
הבשרה.
