אָשִׁיר לְאֵלִי כִּי לַעֲשׂוֹת לִי הִפְלִיא

[חנוכה]

אָשִׁיר לְאֵלִי / כִּי לַעֲשׂוֹת לִי הִפְלִיא:

בִּקְהַל אֱמוּנִים / אַגִּיד פְּלָאִים עָשָׂה

כִּי מִלְּפָנִים / עֵינוֹ עֲלֵיהֶם חָסָה

וּבְרִית לְבָנִים / זָכַר, וְרֹאשָׁם נָשָׂא

וּבְעֵת בְּאָפְלִי / שַׁבְתִּי, יוי אוֹר לִי:


רָמָה יְמִינִי / עַל צָר לְהָרַע חָשַׁב

רֶגַע, לְעֵינִי / נִכְלָם, אֱלֵי אַרְצוֹ שָׁב

יָשַׁב לְדִינִי / צוּרִי, וְשַׁוְעִי קָשַׁב

בִּכְיִי וְאֶבְלִי / הָפַךְ אֱלֵי מָחוֹל לִי:


הִמְצִיא בְרָכָה / הָאֵל בְּפַךְ נִמְצְאָה

נֵר מַעֲרָכָה / שָׂמַח, וְהוֹדוֹ נִרְאָה

וּלְאוּם נְסוּכָה / מֵרֹאשׁ, תְּשׁוּעָה בָאָה

וּכְאָז בְּאָהְלִי / שָׁכַן וְהֵרִים דִּגְלִי:


מַלְאוּ בְּנֵרוֹת / פִּתְחֵי שְׁעָרִים בָּעִיר

לִהְיוֹת לְדוֹרוֹת / זִכְרוֹן פְּלָאִים מֵעִיר

עַד בּוֹא בְשׂוֹרוֹת / לִקְרוֹא לְנִינֵי צָעִיר

יֶתֶר קְהָלִי / שׁוּבוּ אֱלֵי הֵיכָלִי:
