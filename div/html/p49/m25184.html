[עלי כל איש]

עֲלֵי כָל אִישׁ יְעִידוּן מַאֲמָרָיו, //וְחַיָּיו בֵּין שְׂפָתָיו, אוֹ קְבָרָיו,

וְיֵשׁ יִתְרוֹן לְמִתְהַלֵּךְ בְּתֵבֵל, // וְאֵין יִתְרוֹן לְשׁוֹקֵט עַל שְׁמָרָיו

וְיֵשׁ נִרְדָּם עֲדֵי נֶשֶׁף בְּלִי לֵב, // וְאַחֵר נָם וְרַעְיוֹנָיו מְעִירָיו.

הֱקִיצַנִי לְבָבִי מִתְּנוּמָה // בְּתוֹךְ לַיִל, בְּטֶרֶם בּוֹא בְּקָרָיו,

וְהִרְאִני כְּסִיל, מֵאִיר לְעֻמַּת // בְּנֵי עַיִשׁ, כְּמוֹ שַׁלִּיט וְשָׂרָיו,

וְנֹגַהּ בַּחֲדַר תֵּימָן, כְּכַלָּה// אֲשֶׁר מֶלֶךְ הֱבִיאָהּ בַּחֲדָרָיו,

וְיָרֵחַ כְּקֶשֶׁת, שָׁלְחוּ אוֹר // בְּצִדֶּיהָ, וּמִתּוֹכָה אֲפֵרָיו.

בְּנֵי עוֹלָם! תְּנוּ לֵבָב לְסוֹד אֵל// בְּעוֹלָמוֹ, וְחִקְרוּ אֶת סְתָרָיו,

וְאַל תִּהְיוּ כְּחָצִיר צָץ וּפָרַח, // וְלֹא הֵבִין אֱלֵי עֵת בּוֹא קְצִירָיו.

פְּנוּ אֵלַי בְּכָל עָקוּד וְקָשׁוּר// וְרָז אָסוּר – וְאַתִּיר מַאֲסָרָיו,

וְאַל תִּבְזוּ לְאִישׁ אַרְצוֹ פְּאַת יָם // וְתָבוּזוּ לְכָל שׁוֹקֵד שְׁעָרָיו,

וּבִינוּ כִי נְבוֹנֵי לֵב יְשִׂימוּן // לְיָם יֶתֶר עֲלֵי קֶדֶם וְעָרָיו.

הֲיִשְׁוֶה מִי צְבָא מָרוֹם // מְהַלֵּךְ אֱלֵי אַרְצוֹ, לְמִי שָׂמוֹ אֲחוֹרָיו?

וְאִם יִשְׁווּ, לְאַלּוּפִי בְּחָכְמָה// וּבִתְבוּנָה וּבַדַּעַת, חֲבֵרָיו?

אֲשֶׁר שָׁקַד וְהוֹצִיא מִמְּעֵי דָת – // בְּכוֹרֶיהָ, וּמִזּוֹלֵל – יְקָרָיו,

אֲשֶׁר הִשְׁקָה וְרִוָּה הַחֲכָמִים,// וְלֹא לוֹ דַבְּרוֹ הִשְׁקָה וְלֹא רַו.

אֲכַבֶּדְךָ כְּכַבֵּד בֵּן לְהוֹרָיו // וְאִירָאָךְ כְּמוֹרָא אִישׁ לְמוֹרָיו,

וְאֹהָבָךְ כְּאַהְבַת בֶּן זְקוּנִים // יִחיד אָב, אוֹ כְּאַהְבַת שָׂב צְעִירָיו,

וְאֶדְבַּק בָּךְ כְּיוֹנָתָן בְּדָוִד,// וּמשֶׁה עִם מְשָׁרְתוֹ מִבְּחוּרָיו,

וְנִגְלֶה זֶה לְכָל נוֹצָר, כְּמוֹ יָם// אֲשֶׁר נִכָּר לְבָא מִכָּל עֲבָרָיו!
