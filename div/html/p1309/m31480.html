מִי זֶה בָא בְגִיל וּבְשִׂמְחָה

שיר יין

מִי זֶה בָא בְגִיל וּבְשִׂמְחָה / אֶל תּוֹךְ הַלְּבָבוֹת?

כָּל יָגוֹן וְכָל אוֹן דָּחָה, / כִּי גִילוֹת קְרֵבוֹת,

גַּם סֵפֶר מְצוּקִים מָחָה / וַיִּכְתֹּב לְטוֹבוֹת –

בָּא יַיִן אֲשֶׁר מִנְיָנוֹ / שִׁשִּׁים וַעֲשָׂרָה,

יָצָא סוֹד וְהוּא עִנְיָנוֹ / גַּם הוּא בַּסְּפִירָה.


״שָׁלוֹם״ אֹמְרָה ״כּוֹס רִאשׁוֹן״ / אֶל הַלֵּב בְּבוֹאוֹ,

גַּם שֵׁנִי יְהָאִיר אִישׁוֹן / הָעַיִן לְנוֹשְׂאוֹ

וּשְׁלִישִׁי יְצַוֶּה לָשׁוֹן / יֶהְגֶּה שִׁיר לְבוֹרְאוֹ

וּרְבִיעִי יְגָרֵשׁ אוֹנוֹ / זֹאת הִיא הַחֲקִירָה

אַשְׁרֵי מִי יְהִי קִנְיָנוֹ / שִׂמְחַת לֵב וְשִׁירָה!


הָבוּ לִי חֲצִי הַר סִינַי / הוּא שִׂמְחַת לְבָבִי

הוּא יָנִיס מְאֹד לִיגוֹנַי / הוּא יָסִיר כְּאֵבִי,

הוּא יַעַל עֲלֵי רַעְיוֹנַי / בִּרְכָתִי וְטוּבִי.

הַשְׁקוּנִי אֲדַמְדַּם עֵינוֹ / עִם נֶפֶשׁ יְקָרָה,

כִּי כָל מִין יְבַקֵּשׁ מִינוֹ / עֵת שֶׁבֶת בְּחֶבְרָה.


כָּרַת עִם רְנָנִים אָמְנָם / הַיַּיִן בְּרִיתוֹ,

הַמּוֹכֵר מְכָרוֹ חִנָּם – / לֹא יָדַע שְׁתוֹתוֹ

נֶעְגָּמִים יְנַחֵם אוֹנָם / זֹאת חֻקּוֹ וְדָתוֹ

הַכִּילַי יְפַזֵּר הוֹנוֹ / בִּשְׁתוֹתוֹ מְהֵרָה,

גַּם חַלָּשׁ יְהִי נָא רֻנּוֹ / רֹן בַּעַל גְּבוּרָה.
