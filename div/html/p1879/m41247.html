צִוָּה הָאֵל

צִוָּה הָאֵל / לְיִשְׂרָאֵל / לִשְׁמֹר בְּרִיתוֹ וְדָתוֹ

וּבְחוֹתָמוֹ / הוּא יָהּ שְׁמוֹ / חָתַם עֲדָתוֹ. וּבְחוֹתָמוֹ


אוֹת זֶה אֵל רָם / שָׂם בִּבְשָׂרָם / יְגַלֶּה אַהֲבָתוֹ

בִּרְאוֹתוֹ יְלָדָיו / מַעְשֵׂה יָדָיו / עַם סְגֻלָּתוֹ. בִּרְאוֹתוֹ


לָאֵל אֶקֹּד / אַשְׁפִּיל קָדְקֹד / לָאֵל נַעְלֶה בְמַעְלָה

אֲשֶׁר נָתַן / אֵל בְּמַתָּן / לְרֹאשׁ הַגּוֹלָה. בְמַעְלָה


חֶסֶד וָחֵן / וְלֵב בּוֹחֵן / וְגַם גְּדֻלָּה כְּפוּלָה

לָעַד יִכּוֹן / בֶּטַח יִשְׁכֹּן / בְּתוֹךְ טִירָתוֹ. כְּפוּלָה


חַי גּוֹאֲלִי / חֹן נִמּוֹלִי / בֶּן שַׂר קְהָלִי וְחֵילִי

כֶּתֶם הַטּוֹב / יְהִי רָטֹב / לָעַד מוֹשְׁלִי. וְחֵילִי


אֵל חַי יָשׁוּב / לְעַם חָשׁוּב / בְּתוֹךְ זְבוּלִי בִּגְבוּלִי

יִשְׁכֹּן עֶלְיוֹן / בְּאַפִּרְיוֹן / בְּתוֹךְ נְוָתוֹ. בִּגְבוּלִי
