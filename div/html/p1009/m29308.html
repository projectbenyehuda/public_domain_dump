מַלְאֲכֵי מָרוֹם יֶהֱמָיוּ

מַלְאֲכֵי מָרוֹם יֶהֱמָיוּ / עַל עָם רָבוּץ נֶעֱשָׁק

צָעֲקוּ מַר הוֹי אֶרְאֶלָּם / נַאֲקוֹת חָלָל יִנְאָק


אֶת עַם עָנִי תּוֹשִׁיעַ / לִמְנוּחָתָם תִּשְׁקֹד

גֶּפֶן מִנּוֹף תַּסִּיעַ / שֵׁנִית פָּקֹד תִּפְקֹד

וּלְמַעַן אָב מַפְגִּיעַ / תִּזְכֹּר לִבְנֵי עָקֹד

אֵיכָה צָרִים יִשְׁלָיוּ / דּוֹבְרֵי עַל צַדִּיק עָתָק

תּוֹךְ טִירָתָם בָּרִיא אוּלָם / וּבְפִיו רָעָה תִּמְתָק


בּוֹעֵר כָּל לֵב כִִּמְדוּרָה / בִּצְדִיַּת עַם לֹא אַלְמָן

מִי יוֹבִיל שַׁי לַמּוֹרָא / יָשִׁיר שִׁירַת הֵימָן

יַעֲמִיד נֵס הַבִּירַָה / יוֹפִיעַ אוֹר מִתֵּימָן

עוֹד חַשְׁמַנִּים יֶאֱתָיוּ / אָז תָּרִיץ כּוּשׁ מִמֶּרְחָק

יַעַבְדוּהוּ עַמִּים כֻּלָּם / כֹּה בִנְבוּאוֹת סֵפֶר הוּחָק


נִין אֵיתָנִים לִשְׁנֵים עָשָׂר / לִבְנֵי אָמָה הָיוּ מַאֲכָל

וּבְנֵי הָאָח שָׂר עִם טִפְסָר / אֶל הֶהָרִים עֵינָיו אָכָל

זֶה כַּמֶּה שָׁנִים נִמְסָר / מִיַּד נָבָל אֶל יַד סָכָל

אַיֵּה צוּר בּוֹ חָסָיּוּ / אוֹ מִכְתָּבוֹ כִּי לֹא נִמְחָק

יִזְכֹּר לִבְרִית חוֹזֶה סֻלָּם / וּבְרִית הָאָב עָקוּד יִצְחָק


רָם מַה יּוֹעִיל אוֹ יִסְכֹּן / לָךְ אִם נִזְרִי מֻקְרָח

דְּבַר שֶׁקֶר אֵיךְ יִכּוֹן / וּצְבִי עֶדְיוֹ סָרָח

מָתַי מַלְכוּתִי תִּכּוֹן / צַדִּיק כַּתָּמָר יִפְרָח

אִם עַד כַּפַּי לֹא נִקָּיוּ / הֵן כַּף אֶל כַּף אֶסְפָּק

וּנְחֵנִי דֶּרֶךְ עוֹלָם / כִּי לֹא אַחֲרִישׁ עַד אֶתְאַפָּק


מַה לָךְ יוֹנָה הוֹלֶכֶת / עַל מִשְׁמַרְתֵּךְ לַעֲמֹד

שׁוּבִי מִתְבָּרֶכֶת / לִי אֵרַשְׂתִּיךְ לַעֲבֹד

וִישִׂימֵךְ עֹז דּוֹרֶכֶת / מַלְכֵּךְ מֶלֶךְ הַכָּבוֹד

אִם תִּבְעָיוּן בְּעָיוּ / לִמְחַזֵּק אֶת הַבֶּדֶק

וִיכוֹנֵן יַד מַעְגָּלָם / עַל דִּבְרָתִי מַלְכִּי צֶדֶק.
