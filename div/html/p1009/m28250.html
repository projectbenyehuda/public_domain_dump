אִישִׁים אֶקְרָא מֵהָר

אִישִׁים אֶקְרָא מֵהָר אֶל הָר / בּוֹאוּ לִמְדוּ סֵפֶר זֹהַר

בּוּעוּ וּבְעוּ מַצְפּוּנוֹתָיו / כִּי מֵאוֹרוֹ יָאִיר סַהַר

גֹּשׁוּ לִקְטוּ מִסּוֹדוֹתָיו / יוֹדְעָיו הֵמָּה יַעֲלוּ בָהָר

דֶּרֶךְ הָאֱמֶת יִקָּרֵא לוֹ / הָרָגִיל בּוֹ אֶל טוּב יִנְהָר

הִיא הַדֶּרֶךְ הַיְפֵהפִיָּה / כִּי בָהּ יַשְׂכִּיל לֵבָב נִמְהָר

וַיּוֹרֵנִי וַיֹּאמֶר לִי / הֶחָפֵץ בִּי מֵחֵטְא נִזְהָר

זֵכֶר צַדִּיק הוּא לִבְרָכָה / רִבִּי שִׁמְעוֹן כָּאוֹר יִזְהָר

חָשַׂף לַכֹּל כָּל חֶמְדּוֹתָיו / כָּל סוֹד נֶעְלָם שָׁב לָטֹהַר

טַעְמֵי מִצְווֹת הַנֶּעֱלָמוֹת / עוֹזֵב אוֹתָם אַרְצָה יִגְהָר

יוֹדֵעַ בָּם עַל מַתְכּוּנְתָּם / יִרְדֹּף צֶדֶק בַּכֹּל יִטְהָר

כָּל שׁוֹנֶה בּוֹ יִמְצָא חַיִּים / וּלְהוֹסִיף עוֹד דָּרְשׁוֹ מִהַר

לֹא יִרְעַב כָּל הַמְּעַיֵּן בּוֹ / לֹא יִצְמָא עוֹד אֶל מֵי נָהָר

מָשַׁךְ יָדוֹ מִכָּל מָתוֹק / מָאַס דָּגָן תִּירוֹשׁ יִצְהָר

נַפְשׁוֹ תִּשְׂבַּע מִטּוּב חֶלְבּוֹ / וּגְבוּרָתוֹ כַּסּוּס דָּהָר

סָרִים תָּמִיד אֶל מִשְׁמַעְתּוֹ / חַכְמֵי דוֹרוֹת לָהֶם יִצְהָר

עוֹרְכִים כָּבוֹד מִמֶּרְחַקִּים / כָּל עֵץ אֶרֶז וּבְרוֹשׁ תִּדְהָר

פָּעַל הָאֵל סוֹד הַגִּלְגּוּל / אֶל כָּל גֶּבֶר אִמּוֹ תַּהַר

צוֹעֵק חָמָס מִצָּרוֹתָיו / לֹא יֵדַע בּוֹ קֹדֵם יֻהַר

קָדוֹשׁ יוֹשֵׁב בִּשְׁמֵי מָרוֹם / יִפְדֶּה עַבְדּוֹ מִן הַסֹּהַר

רָחַשׁ לִבִּי בַּשִּׁיר תִּלִּים / לִפְנֵי הָאֵל כִּבְנֵי יִצְהָר

שׁוֹקֵד לַעֲשׂוֹת כָּל אִמְרוֹתָיו / יַהֲפֹךְ עָוֹן צֶמֶר צַחַר

תֶּחִי נַפְשִׁי וּתְהַלֵּל עוֹד / תָּמִיד עַד כִּי גָרוֹן יִנְחַר

וּשְׁמִי יוֹשֵׁב שִׁבְעָה עַל כַּר / וַחֲצִי הַשֵּׁם יָהּ כַּשַּׁחַר.
