שִׁמְעוּ שִׁירִי הַקַּדְמוֹנִים

שִׁמְעוּ שִׁירִי הַקַּדְמוֹנִים / כָּרִאשׁוֹנִים הָאַחֲרוֹנִים

כִּי לִי בַשִּׁיר עֶשֶׂר יָדוֹת / בַּתַּחְבּוּלוֹת עֶשֶׂר מוֹנִים

דִּרְשׁוּ הֵיטֵב לִמְדוּ מִשְׁפָּט / קִרְאוּ שִׁירִי כָּל הַחֲתָנִים

כִּי קִוִּיתִי יוֹם לִשְׂמֹחַ / לִשְׁתּוֹת מֵימֵי מַעְיַן גַּנִים

חָלִיל מוּכֶּה עִם קוֹל כִּנּוֹר / נָשִׁים שָׁרוֹת שָׂרִים עוֹנִים

נֶהְפַּךְ שִׁירִי קִינָה בִּרְאוֹת / כַּלָּה יֵשׁ לָהּ כֵּעוּר פָּנִים

גַּם שִׁנֶּיהָ נָשְׁרוּ כֻלָּם / וּקְוֻצּוֹת לָהּ הֵם מַלְבִּינִים

כָּל יָמֶיהָ עַל מִטָּתָהּ / עַל אַפִּרְיוֹן רַבַּת שָׁנִים

וַתֹּאמַר לִי: יֶשׁ לִי זָהָב / יֶשׁ לִי כֶּסֶף עִם קִנְיָנִים

עִם מַחֲמַדִּים לֹא נֶחְקָרוּ / גַּם לִי גַּנּוֹת גַּם בִּנְיָנִים

אֵיזֶה תִּבְחַר תִּשְׁכֹּן מֵהֶם / תֹּאחֵז תָּמָר בַּסַּנְסַנִּים

אַל יַבְהִילָךְ רֹעַ פָּנַי / אַל תִּסְתַּכֵּל בַּקַּנְקַנִּים

כִּי הַיֹּפִי אֵינוֹ חָשׁוּב / כִּי הָאִשָּׁה הִיא לַבָּנִים

וָאֹמַר לָהּ: כָּעֵת אֶסַּע / אֶבְרַח מִמֵּךְ לִמְעוֹן תַּנִּים

אִם לֹא תִהְיֶה אִשְׁתִּי יָפָה / אֶתְעַלֵּס לִי בָּהּ בִּרְנָנִים

כִּי הָאִשָּׁה הַמַּשְׁחֶרֶת / בַּעְלָהּ נֶחְשָׁב לָהּ לִזְנוּנִים

כָּל רוֹאֶיהָ יַעֲבֹר עַל דָּת / יִדְמֶה עוֹבֵד לַכַּוָּנִים

דֶּרֶךְ עוֹבֵר עַל קוֹל שִׁירִי / דֶּרֶךְ עִקֵּשׁ פַּחִים צִנִּים

וּשְׁנוּ שִׁירִי וְגַם שַׁלֵּשׁוּ / שִׁמְעוּ שִׁירִי הַקַּדְמוֹנִים.
