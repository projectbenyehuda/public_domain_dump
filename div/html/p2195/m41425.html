נְחַבֵּר שִׁיר

זה השיר שלחתי להיקר שי׳ מרדכי שי׳ יצחק ש׳ הנז״ל.

סימן: נשים.


נְחַבֵּר שִׁיר לְשַׂר וָרָב בְּמַעְלָה / וְהוּא נִין לַנְּשִׂיאִים בַּקְּהִלָּה

שְׂבַע רָצוֹן וּמָלֵא כָל בְרָכוֹת / יְהִי תָמִיד וְגַם טוֹבָה כְפוּלָה

יְהִי זַרְעוֹ כְגַן רָטוֹב וְשָׁמֵן / וְכִנְטִיעָה עֲלֵי מַיִם שְׁתוּלָה

מֵשִׁיב נֶפֶשׁ בְּדַעְתּוֹ הָרְחָבָה / מְקוֹר חָכְמָה וְהוּא יָדַע שְׁבִילָהּ

וְרַב חֶסֶד וְהוּא מַרְבֶּה לְהֵיטִיב / צְדָקוֹת יַעֲשֶׂה, בּוֹ חֵן וְחֶמְלָה

וּבָא מִמִּזְרְחָה שֶׁמֶשׁ לְהָאִיר / גְּבוּלֵנוּ בְּצֶדֶק גַּם בְּתוּלָה

פְּזֵר גָּדוֹל לְמָמוֹן רַב מְפַזֵּר / זְהַב שְׁבָא וְרֹב כֶּסֶף וְשִׂמְלָה

וְכָל פֶּרֶק וּפֶרֶק וַחֲנֻכָּה / וּבַפּוּרִים בְּעֵת קוֹרְאִים מְגִלָּה

מְאֹד יַרְבֶּה צְדָקוֹת לַעֲנִיִּים / וּמַתָּנוֹת לְחַכְמֵי לֵב בְּצַהְלָה

וְעֵת יַשִּׂיא אֱנוֹשׁ לִבְנוֹ וּבִתּוֹ / מְאֹד יַרְבֶּה לְבָחוּר וְלִבְתוּלָה

וּמִלֵּא אֶת הָאָרֶץ בַּחֲסָדָיו / וְלוּלֵי רֹב חֲסָדָיו הִיא חֲלוּלָה

וְשִׁירָתִי לְדוֹדִי הִיא בְחִנָּם / וְאֵינֶנִּי מְצַפֶּה אֶל גְּמוּלָה

אֲצַפֶּה אֶל אֱלֹהֵי הַצְּבָאוֹת / לְעָבְדָךְ אֵל יְהִי נִצְרָךְ שְׁאֵלָה.
