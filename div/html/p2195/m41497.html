אוֹיְבִי דִמָּה

אוֹיְבִי דִמָּה / לַעְלוֹת כִּימָה / שָׁכַח רִמָּה / וְסוֹפוֹ שְׁאוֹל

לָבוּשׁ רִקְמוֹת / גָּבִישׁ רָאמוֹת / מַעְלוֹת רָמוֹת / נַפְשׁוֹ לִשְׁאוֹל

אִם זֶה נִגְעַל / פְּרֹשׂ שֹׁעַל / דַּלְתֵי רוּם עַל / פָּנָיו נָעוּל

שָׂם עֹל צָרִי / עַל צַוָּארִי / אֶשְׁאַל צוּרִי / עֲדֵה נָא עֹל.
