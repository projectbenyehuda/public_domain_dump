משל לז: נֶשֶׁר וְעוֹפוֹת וַאֲרִי וְחַיּוֹת וַעֲטַלֵּף

מְדַבֵּר בְּדִבְרֵי חֲלָקוֹת / אֶל הֶחָזָק יִתְחַבֵּר בְּמַעֲשָׁקוֹת


הַנֶּשֶׁר מָלַךְ עַל כָּל בַּעֲלֵי כָנָף / הֵן בָּאָרֶץ הֵן בֶּעָנָף / וְהָאֲרִי עַל כָּל בְּהֵמָה וְחַיָּה / מִכָּל קַצְוֵי אֶרֶץ נְשִׁיָּה / וַיַּעַרְכוּ הַנֶּשֶׁר וְהָעוֹפוֹת מִלְחָמָה / עִם הָאֲרִי וְעִם כָּל חַיָּה וּבְהֵמָה / וַיֹּאמֶר הָעֲטַלֵּף בְּלִבּוֹ / כִּי דְבַר בְּלִיַּעַל יָצוּק בּוֹ / אָמְנָם כָּעוֹף יֵשׁ לִי כְנָפַיִם / וְכַחַיָּה וְכַבְּהֵמָה יֵשׁ לִי רַגְלַיִם / וְלֹא אֵדַע אֵיזֶה חַיִל זֶה עַל זֶה יִגְבַּר אוֹ מִי יָמוּת אוֹ נִשְׁבָּה אוֹ נִשְׁבָּר / אֶרְאֶה וְאֶהְיֶה צֹפֶה / הֶחָזָק הוּא הֲרָפֶה / וְעַם אֲשֶׁר תִּגְבַּר יָדוֹ וּשְׁלוֹמוֹ / וְהָיְתָה יָדִי עִמּוֹ / וַיֵּרֹם מִן הַחַיִל וַיַּעֲמֹד לְמוּלוֹ / לְדֵעָה מַה יֵּעָשֶׂה לוֹ / וַיֵּלֶךְ וַיַּעֲמֹד עַל עֵץ גָּבֹהַּ לְמֵרָחוֹק / וַיֹּאמַר: עֵת מִלְחָמָה לָעוֹפוֹת וְלֹא עֵת שְׂחוֹק / אֶל הַנֹּצְחִים לַעֲזֹר אֶשָּׂא עֵינִי / אֶשְׁקְטָה וְאַבִּיטָה בִמְכוֹנִי / וַיַּרְא כִּי חָזְקוּ הַבְּהֵמוֹת / וּבַהֲמוֹן הָעוֹפוֹת נָפְלוּ מְהוּמוֹת / וַיֵּרֶד מִן הָעֵץ וַיֵּלֶךְ עַל אַרְבַּע / וְלַעֲזֹר בְּמַאֲמַצֵּי כֹחוֹ נִשְׁבַּע / אֶל כָּל חַיָּה וּבְהֵמָה / וַיֹּאמַר: חִזְקוּ וַעֲשׂוּ מִלְחָמָה / אַל יִמָּלְטוּ בַּעֲלֵי כָנָף / כָּל אֶחָד עֲלֵיהֶם יִתְאַנָּף / וְכַאֲשֶׁר רָפָה הַיּוֹם לַעֲרֹב וַתֶּחֱזַק הַמִּלְחָמָה / וַיִּתְגַבְּרוּ הָעוֹפוֹת וּבַהֲמוֹן הָאֲרִי נָפְלָה מְהוּמָה / וַיֹּאמֶר הַעֲטַלֵּף בְּלִבּוֹ: אֶל עַמִּי אָשׁוּבָה / וְלֹא אוֹסִיף עוֹד עִם הֲמוֹן הָאֲרִי לְדַאֲבָה / וְאָעוּפָה וָאֶשְׁכּוֹנָה / וְרַגְלַי תַּחַת כְּנָפַי אַטְמִינָה / וַיֶּאֱסֹף רַגְלָיו בֵּין הַכְּתֵפַיִם / עָשֹׂה יַעֲשֶׂה לוֹ כְנָפַיִם / וַיָּעָף חִישׁ בֵּין הָעוֹפוֹת / וַיֹּאמַר: חַזְּקוּ יָדַיִם רָפוֹת / לַעֲשׂוֹת פֶּצַע וְחַבּוּרָה וּמַכָּה טְרִיָּה / בְּהוֹלְכֵי עַל אַרְבַּע, בְּהֵמָה וְחַיָּה / וַיַּכִּירוּ הָעוֹפוֹת אֶת הָעֲטַלֵּף / וַיֹּאמְרוּ: מַדוּעַ כָּכָה תִּתְחַלֵּף / לָלֶכֶת עִמָּנוּ בְקֶרִי / בֹּקֶר רְאִינוּךָ עִם הֲמוֹן הָאֲרִי / הֹלֵךְ עַל אַרְבַּע כְּמוֹהֶם / הָעֵת אֲשֶׁר גָּבְרָה יְדֵיהֶם / וְשָׁלַחְתָּ יָד בִּשְׁלוֹמֵנוּ / וְעוֹף הָיִיתָ כָּמוֹנוּ / וְכַאֲשֶׁר גָּבַרְנוּ פָרַשְׂתָּ עָלֵינוּ כְנָפֶיךָ / דָּם שָׂנֵאתָ וְדָם יִרְדֳּפֶךָ / מַדּוּעַ בְּעוֹרְךָ הֵתַלְתָּ / מִרְמָה וְאָוֶן פָּעַלְתָּ / וַיְהִי אָרְבְּךָ בְקִרְבְּךָ / עַד נִמְצָא עַוְלָתָה בְּךָ / הָעֵת שִׁבְתְּךָ עַל עֵקֶב בָּשְׁתְּךָ / זֶה יִהְיֶה מַשְׂכֻּרְתְּךָ / כִּ תֵחָשֵׁב מִמִּין שֶׁרֶץ / נָע וְנָד תִּהְיֶה בָאָרֶץ / כַּאֲשֶׁר הִתְנַכַּרְתָּ מֵרֵעֶיךָ / כֹּה יִהְיֶה זַרְעֶךָ / שָׁחֹר וּמָרוּט וְקֵרֵחַ / עִוֵּר וּבֵינֵינוּ כְאֹרֵחַ / לֹא תֵלֵךְ עוֹד נְטוּי גָּרוֹן / אוֹר שֶׁמֶשׁ יִהְיֶה לְךָ לְעִוָּרוֹן / קֵרֵחַ וְשֶׁרֶץ תִּקָּרֵא / מִכָּל פֶּגַע תִּהְיֶה יָרֵא / וַיִּקְשְׁרוּ עָלָיו קֶשֶׁר / וַיְבִיאוּהוּ אֶל הַנֶּשֶׁר / וַיֹּאמַר: הֲזֶה הוּא אֲשֶׁר דָּמָה לִבְהֵמָה וְחַיָּה / לְבַעֲבוּר כִּי בְרוּחוֹ רְמִיָּה / יְהִי נָא מָרוּט וְקֵרֵחַ / וּמֵאוֹר הַשֶּׁמֶשׁ בּוֹרֵחַ / וְלָעֶרֶב יָעוּף כְּמוֹ נִבְהָל / גַּם דּוֹר עֲשִׂירִי לֹא יָבֹא לוֹ בַקָּהָל / וְלֹא יוּכַל לַעֲמֹד עַל רַגְלָיו / אִם זָרְחָה הַשֶּׁמֶשׁ עָלָיו / וַיִּמְרְטוּהוּ וַיִּקְרְחוּהוּ / וְעֵינָיו עִוְּרוּ וַיַּבְהִילוּהוּ / מִמְּגוּרֵיהֶם אֲשֶׁר בָּם גָּרוּ / וְכָל עוֹף הַשָּׁמַיִם מִמֶּנּוּ יִתְנַכָּרוּ / וְלֹא יִדְמֶה עוֹד לָעוֹף וְלַבְּהֵמָה / וְשֶׁרֶץ יִקְרְאוּ לוֹ עַל הָאֲדָמָה.


וְהַמָּשָׁל בַּאֲשֶׁר יִרְאֶה אֹהֲבוֹ אוֹ קְרוֹבוֹ / נֹפֵל וְיוּכַל לוֹ אֹיְבוֹ / אֶל זֶה יְדַבֵּר בַחֲלַקְלַקּוֹת / וְעִם הֶחָזָק יִתְאַמֵּץ בְּבֶצַע מַעֲשַׁקּוֹת / וְאִם הַשָּׁפָל יִתְחַזֵּק וְיִתְגַּבֵּר / בְּלֵב וָלֵב יָשׁוּב אֵלָיו לְהִתְחַבֵּר / בַּאֲשֶׁר לֹא יֵדַע בֹּשֶׁת וּכְלִמָּה / וּמִן הַהִתְחַבְּרוּת אֵלָיו יַעֲשֶׂה מִרְמָה / וְיֹאמֶר לוֹ: הִנֵּה אִמַּצְתִּיךָ עֲזַרְתִּיךָ / וּבִימִין צִדְקִי תְּמַכְתִּיךָ / כַּאֲשֶׁר יַד אֹיִבְךָ עָלֶיךָ גָּבְרָה / לֹא הִתְרַפֵּיתִי בְיוֹם צָרָה / וּכְמִקְרֵה הָעֲטַלֵּף יִמְצָא בְּאַחֲרִית / לְבִלְתִּי הֱיוֹת לוֹ שֵׁם וּשְׁאֵרִית / וּמְיֻשָּׁר עַל הַמְּחֻקֶּה / כָּל עֹשֵה רָע לֹא יִנָּקֶה.
