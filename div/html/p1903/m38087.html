משל כו: חֲמוֹר, דְּבוֹרִים וְצִרְעָה

אַל תְּהִי בָז לְכָל דָּבָר / כִּי לַכֹּל עֵת וּזְמָן יִגְבָּר


חֲמוֹר רֹעֶה בְאַחַד הֶהָרִים / וְהִנֵּה סְבִיבָיו עֲדַת דְּבוֹרִים / וְהַצִּרְעָה מְרַחֶפֶת לְפָנָיו / עַד אֲשֶׁר נֶחְבֵּאת בְּאַחַת מֵאָזְנָיו / וַיִּרְעַשׁ וַיִּגְעַשׁ וּבְקוֹלוֹ נוֹעֵר / הֵנָּה וָהֵנָּה הוֹלֵךְ וְסֹעֵר / וַיִּמְאַס בְּחַיָּיו וַיָּקָץ / כִּי הָיָה נָשׁוּךְ וְנֶעֱקָץ / וַיֵּצֶר לוֹ וְלִמְאֹד חָלָה / לְקוֹל הַהֲמוּלָה הַגְּדוֹלָה / וַיֹּאמַר: הַיּוֹם רַבּוּ עֹיְנַי / וּמַה הַקּוֹל אֲשֶׁר בְּאָזְנַי / אֵין זֶה כִּי אִם רוּחַ רָעָה / וְאִם אַתְּ בִּגְבוּלִי הַצִּרְעָה / מַה לָּךְ לִטְמֹן בְּחֻבִּי / צְאִי נָא וְהִלָּחֵם בִּי / וְנִתְרָאֶה פָּנִים אֶל פָּנִים / הֲיֵשׁ לָךְ כֹּחַ וְאוֹנִים / לָצֵאת וְלֶאֱסֹף עֲדַת דְּבוֹרִים / לְהִלָּחֵם בַּחֲבוּרַת הַחֲמוֹרִים / וַתַּעַן הַצִּרְעָה / מִי יִתֵּן וְהָיָה כָכָה / מַעֲרָכָה מוּל מַעֲרָכָה / וְנִלָּחֲמָה יַחַד בַּבִּקְעָה / צִרְעָה לַחֲמוֹר וַחֲמוֹר לְצִרְעָה / וְהַנָּצוּח יִבָּדֵל מִן הָעֵדָה / וַיַּעַן הַחֲמוֹר: יוֹם הַמּוֹעֵד נַגִּידָה / אֲשֶׁר נֵצֵא לַמִּלְחָמָה / וַיֻּגַּד הַמּוֹעֵד שָׁמָּה / וַיֵּלֶךְ הַחֲמוֹר לְהִמָּלֵךְ / בָּאֲרִי אֲשֶׁר עַל כֻּלָּם מוֹלֵךְ / וַיַּגֶּד לוֹ אֶת כָּל אֲשֶׁר קָרָהוּ / מִן הַצִּרְעָה אֲשֶׁר בָּאֹזֶן הֱצִיקַתְהוּ / וְהַמּוֹעֵד אֲשֶׁר לוֹ יָעָדָה / וְאֵיךְ לִבּוֹ חוֹנֶה בַחֲרָדָה / פֶּן תִּכָּנֵס שֵׁנִית בָּאֹזֶן / וַיֹּאמֶר לוֹ הָאֲרִי: הַאֲזֵן / אִיעָצְךָ וּשְׁמַע בְּקוֹלִי / וְלֹא יִקְרְךָ פֶּגַע וָחֹלִי / סְתֹם וּסְגֹר בִּרְצוּעוֹת כָּל נְקָבֶיךָ / אָז לֹא יוּכְלוּ לְךָ אֹיְבֶיךָ / וְכִרְאוֹתָם זֶה מִפָּנֶיךָ יִבָּהֲלוּ / וּמָשַׁלְתָּ בָּם וּבְךָ לֹא יִמְשֹׁלוּ / וְשִׁית אֶל הַמִּלְחָמָה פָנֶיךָ / וְאַתָּה תֶּאֱזֹר מָתְנֶיךָ / וַיַּעֲשׂוּ כֵן כָּל הַחֲמוֹרִים / וְלַמּוֹעֵד בָּאוּ מְחַצְצִים וְסֹדְרִים / וְהַצְּרָעִים נִלְחָמִים אִתָּם / וּבִרְאוֹתָם כִּי נִסְתַּם / כָּל חוֹר וָחוֹר / פָּנִים וְאָחוֹר / תַּחַת הַבֶּטֶן נִתְחַבְּאוּ / שָׁם גְּבוּרָתָם הֶרְאוּ / לְהַמְצִיא לַחֲמוֹרִים כִּפְלַיִם צָרוֹת / וְהַחֲמוֹרִים מִתְחַכְּכִים וְחוֹפְפִים עַל הָעֵצִים וְעַל הַצְּרוֹרוֹת / וּמִתְנַכְּלִים עֲלֵיהֶם לַהֲמִיתָם / עַד אֲשֶׁר נִתְּקוּ מוֹסְרוֹתָם / וַיֵּרָאוּ הַחוֹרִים וְהַנְּקִיקִים / וְהַצְּרָעִים לִכָּנֵס בָּם מִתְחַזְּקִים / וְנֹשְׁכִים וְעוֹקְצִים וְלוֹחֲצִים / מִפֶּה אֶל פֶּה נִפְרָצִים / וַיְהִי אָרְבָּם בְּקִרְבָּם / וַיַּכְנַע בֶּעָמָל לִבָּם / וַיֹּאמְרוּ לָהֶם הַחֲמוֹרִים / לֹא נִהְיֶה עוֹד מַמְרִים וְסוֹרֲרִים / הִנְנוּ לָכֶם לַעֲבָדִים / רַק תִּהְיוּ מֵעָלֵינוּ נִפְרָדִים.

וְהַמָּשָׁל בְּמִתְגָּאֶה בְגָבְהוֹ וּבְרֹב דָּלִיּוֹתָיו / וְהַקָּטָן יִמְשֹׁל בּוֹ בְּסִבּוֹת תַּחְבֻּלוֹתָיו / עַל כֵּן אַל תַּחְטִיא בְשָׂרְךָ בִלְשׁוֹנֶךָ / לֵאמֹר לַקָּטָן: קָטָנִי עָבָה מִמָּתְנֶיךָ / כִּי הַרְבֵּה נוֹקְשׁוּ וְנִשְׁבְּרוּ בְּרֹב גַּאֲוָה / אֵין מִלְחָמָה טוֹבָה רַק מִלְחֶמֶת הַתַּאֲוָה.
