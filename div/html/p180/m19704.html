[תלונה אל איש רם־המעלה]

נְשִׂיא אֻמּוֹ נְגִיד עַמּוֹ נְסִיכוֹ / וְהַגּוֹאֵל אֲשֶׁר בּוֹ אֵל סְמָכוֹ

וְהַנֵּזֶר וְהָעֵזֶר וְהָעֹז / וְהַמָּעֹז לְמִגְדַּל כֹּל נְסָכוֹ

אֲרִי וּצְרִי חֳרִי וּפְרִי תְעָלָה / כְּזָר דַּרְכּוֹ וְכֹה וָכֹה עֲרָכוֹ

קְרָא מִכְתָּב לְמִתְאַוֶּה וְנֶאְנָח / בְּעֵת כָּתְבוֹ בְּדִמְעָתוֹ מְסָכוֹ

בְּדִמְעַת לֶחֱיוֹ וּדְיוֹ מְסָכוֹ / וְלֹא יָכֹל לְשׁוּר הָיָה בְתוֹכוֹ

וְשָׁלוּחַ בְּיַד אָפֵל וְשָׁפֵל / אֲשֶׁר הַבַּז וְהַבּוּז הֶחֱשִׁיכוֹ

וּבַשַּׁעַר כְּבַיַּעַר מְדֻכֶּה / וּמֵחָמָס וְאִם הָלֹךְ וּבָכֹה

וְאֵלַי נָס וּבִי נֶאְסַר [בְּעָבוֹ]ת / וְאֵל עָלַי בְּמַעְלָלַי הֲפָכוֹ

כְּטֹבֵעַ אֲשֶׁר נֶאְחַז בְּנִטְבָּע / וְכָעִוֵּר אֲשֶׁר עִוֵּר מְשָׁכוֹ

אֲדֹנִי הַט יְמִינָךְ לִי וְלִבּוֹ / אֱזֹר קִנְאָה וְהִתְעוֹרֵר לְתָמְכוֹ

תְּהִי שָׁלוֹם אֲדוֹן נַפְשִׁי לְעוֹלָם / נְשִׂיא אֻמּוֹ נְגִיד עַמּוֹ נְסִיכוֹ.
