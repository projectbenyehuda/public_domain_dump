[שמעה ודם רגע ודם]

ולה בית טֹריף [ולו בית רב חן]

— — È — / — — È — // — — È — / — — È —

שִׁמְעָה וְדֹם רֶגַע וְדֹם / אָדֹם אֲשֶׁר בָּא מֵאֱדֹם

לָאֵל אֲהַבְתִּיךָ אֲבָל / לֹא אַהֲבַת אַנְשֵׁי סְדֹם.
