[אֱמֶת כִּי הַפְּרוֹד]

[לידיד שנפרד ממנו]


È — — — / È — — — / È — — / / È — — — /È — — — / È — —


אֱמֶת כִּי הַפְּרוֹד מַחְלֶה לְבָבוֹת / וּמַבְעִיר בַּקְּרָבוֹת אֵשׁ לְהָבוֹת

וְתָמִיד יִבְעֲרוּ מוֹקְדֵי אֲהָבִים / בְּתוֹךְ הַלֵּב כְּאִלּוּ הֵם חֲרָבוֹת

עֲלֵי דוֹדַי יְחַיּוּ הַנְּפָשׁוֹת / וְיָאִירוּ לְכָל-דֹּרֵךְ נְתִיבוֹת

יְדִיד נַפְשִׁי אֲהָבֶיךָ בְלִבִּי / וְאִם-לֹא אֶזְכְּרֵם תָּמִיד בְּנִיבוֹת

וְלֹא אֶשְׁכַּח יְדִידוּתְךָ אֲהוּבִי / וּמִלֶּיךָ בְּתוֹךְ לִבִּי חֲצוּבוֹת

וּפֵרוּדְךָ אֲדֹנִי עַז וְקָשֶׁה / יְסַמֵּר שַׂעֲרַת אַנְשֵׁי קְרָבוֹת

יְשַׁלַּח-לָךְ אֱלֹהַי טַל רְצוֹנוֹ / וְנַפְשָׁךְ יִשְׁמְרָה שֹׁכֵן עֲרָבוֹת.
