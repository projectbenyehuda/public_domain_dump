הברק או ברק

הֲבָרָק אוֹ בְרַק חֶרֶב לְטוּשָׁה / וְרַעַם אוֹ שְׁאוֹן כּוֹבַע נְחוּשָׁה?

הֲבִינֵנִי מְיֻדָּעִי וְאֵדַע / וְהָקִיצָה לְעֶזְרָתִי וְחוּשָׁה

וְאִם אַיִן - סְמֹךְ עַל לְבָבִי / בְתַפּוּחַ, וְרַפֵּד בַּאֲשִׁישָׁה

וְיֵין חֶלְבּוֹן אֲשֶׁר דַּאְבוֹן יְגָרֵשׁ / וְיִתֵּן אוֹן לְכָל נֶפֶשׁ חֲלוּשָׁה

אֲשֶׁר יַזְכִּיר, לְרֹב אֹרֶךְ זְמַנּוֹ, / זְמַן נֹחַ וְתַרְשִׁישׁ וֶאֱלִישָׁה

וְהַשְׁקֵנִי שְׁלֹשָׁה וַעֲשָׂרָה / וְחַד עָשָׂר וְאַרְבָּעִים וְשִׁשָּׁה

וְאוּלַי בּוֹ יְהִי מַרְפֵּא וְשִׁקּוּי / לְלֵב דָּוֶה וְנֶפֶשׁ נֶאֱנָשָׁה

אֲשֶׁר צָמְאָה לְמוֹדָעָהּ וְדוֹדָהּ / וְהִנֶּהָ בְּרֹב תַּאְוָה יְבֵשָׁה

וְנִלְכָּדָה בְחַבְלֵי הָאֲהָבִים / כְּמוֹ צִפּוֹר אֲשֶׁר בַּפַּח יְקוּשָׁה

וְיָשְׁבָה מֵחֲרוּץ גִּילָהּ חֲרוּצָה / וּמִמִּגְרַשׁ מְנוּחָתָהּ גְּרוּשָׁה

וְלִבִּי נֶהֱפַךְ מַיִם וְנִתַּשׁ / בְּדִמְעוֹתַי עֲלֵי עֵינַי נְתִישָׁה

וְכָל לַיְלָה אֲנִי אֶשְׂחֶה בְדִמְעִי / וּמִכַּעַס בְּכִי עֵינִי עֲשֵׁשָׁה

וְכַחוֹלֶה בְּחִיל אֶפְעֶה, [כְּאִ]לּוּ / לְשׁוֹן אֶפְעֶה פְּרָשַׁנִי פְּרִישָׁה

וּבִכְבֵדִי מְדוּרַת אֵשׁ עֲצוּמָה / וְתַאְוָה מִשְּׁאוֹל עַזָּה וְקָשָׁה

וְאֶשְׁתָּאֶה לְהַבִּיט אֶל דְּמוּתָךְ / אֲשֶׁר שֶׁמֶשׁ חֲצִי יוֹמָם מְבִישָׁה

וּבָעוֹלָם שְׁנֵי אוֹרִים גְּדוֹלִים / וְהִנָּם עִם תְּמוּנָתָךְ שְׁלֹשָׁה

וְשִׁבְעַת כּוֹכְבֵי רוֹם בָּךְ שְׁמוֹנָה / וְאַרְבָּעָה שָׁרָשִׁים בָּךְ חֲמִשָּׁה

וְאֵיךְ אַחְרִישׁ לְפֵרוּדָךְ - וְהִנֵּה / תְּמוּנָתָךְ לְמוּל עֵינַי חֲרוּשָׁה?

וְאָמְנָם כִּי תְהִלָּתָךְ יְשָׁנָה / וְאוּלָם לַבְּקָרִים הִיא חֲדָשָׁה

וְאֵיךְ כַּיּוֹם תְּיָרֵשׁ הַיְּדִידוּת / וְהִנֵּה הִיא בְיָדֵנוּ יְרֻשָּׁה

וְלָמָּה זֶה בְּךָ אַחְזִיק - וְתִטֹּשׁ / אֲהָבַי מִבְּלִי פֶשַׁע נְטִישָׁה?

וְהֵן נַפְשִׁי לְאָמָה לָךְ, אֲבָל עַל / מְנָת שֶׁלֹּא תְהִי לָעַד חֲפוּשָׁה

וְחָלִילָה הֱיוֹתָךְ כַּמְּסִיתִים / אֲשֶׁר מַכַּת אֲסוּרֵיהֶם אֲנוּשָׁה

הֲיִהְיֶה רוֹם לְעוֹלָמִים כְּתַחַת / וְאִם אִישׁ יִהְיֶה לָעַד כְּאִשָּׁה?

וְאִם יִכּוֹן בְּאַף אָדָם הֱיוֹת מֹר / כְּמוֹ דֹמֶן אֲשֶׁר יַבְאִישׁ בְּאִישָׁה?

וְאֵיךְ אֶהְיֶה חֲלַל כַּחַשׁ וְכָזָב / וְלֹא אֶדְרֹשׁ דְּמֵי דוֹדָךְ דְּרִישָׁה?

וְלוּ תִרְצֶה אֲזַי מַחְצִי תְרַפֵּא / וְנִשְׁבַּרְתִּי בְּךָ תִהְיֶה חֲבוּשָׁה

וְאָז אֶהְגֶּה בְצִדְקָתָךְ וְאֶרְחַשׁ / תְּהִלָּתָךְ יְמֵי חַיַּי רְחִישָׁה.
