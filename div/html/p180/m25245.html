שְׁחֹרָה וְנָאוָה כָּל

שְׁחֹרָה וְנָאוָה / כָּל הַיּוֹם דָּוָה

לַעַג שַׁאֲנַנִּים / שְׂבֵעָה וְרָוָה

מְרוּדֶיהָ תְבַשֵּׂר / הִנֵּה יוֹם חֶדְוָה

שׁוּבוּ לְבִצָּרוֹן / אֲסִירֵי הַתִּקְוָה


בַּת נָדִיב תִּצְעַק / וְתֶהְגֶּה בְנִיב לָשׁוֹן

מָתַי יָשׁוּב לִי / אִישִׁי הָרִאשׁוֹן

לְךָ כַּפָּהּ תִּפְרֹשׂ / וְגַם תִּשָּׂא אִישׁוֹן

קְרָא הָאֵל דְּרוֹר לָהּ / מִיַּד עוּץ וְדִישׁוֹן

וּשְׁלַח נִין נַחְשׁוֹן / וְהָמֵר בּוּז בְּגַאֲוָה


עַד מָתַי אִכַּף / וְגַם לֹא אָרִים רֹאשׁ

מִתִּגְרַת לוֹחֵץ / עַל גַּבִּי יַחֲרֹשׁ

אֲנִי בְכָל יוֹם אִדַּל / וְהוּא רַעֲנָן כִבְרוֹשׁ

יִשְׁתֶּה מַמְתַקִּים / וְיַשְׁקֵנִי מֵי רוֹשׁ

הוֹפִיעַ עֶלְיוֹן / וְגַם דָּמִי תִדְרֹשׁ

מִיַד עוּץ מְבַקֵּשׁ / לְהָפֵר הָאַחֲוָה.


אַל תֶּחֱרַשׁ וְאַל תִּשְׁקֹט / וְאַל תָּשִׁיב יָדֶךָ

מִקֶּרֶב חֵיקְךָ כַלֵּה / הַשּׁוֹאֲגִים בְּמוֹעֲדֶךָ

וְיִוָּדַע בַּגּוֹיִים / נִקְמַת דַּם עֲבָדֶיךָ

וְהוֹשִׁיעָה יְמִינְךָ / לְמַעַן יֵחָלְצוּן יְדִידֶיךָ

אֲשֶׁר סָבְלוּ עֻלְּךָ / בְּחִבָּה וּבְאַחֲוָה


בְּכוֹר שׁוֹרוֹ הָדָר לוֹ / כְּשׁוֹר יוּבַל לָטֶבַח

וּפֶרֶא גַם אִישׁ שָׂדֶה / יָרִיקוּ עָלַי אֶבַח

וְאֵין בְּיָדִי מִכֹּל / לְבַד תְּחִנָּה וָשֶׁבַח

נִדְבוֹת פִּי רְצֵה נָא יְיָ / כְּמוֹ עוֹלָה וָזֶבַח

וְתִיטַב לְךָ מִשּׁוֹר פָּר / וּמִמִּנְחָה הַמַּרְוָה


יֵאָמֵן דְּבָרְךָ / צַדִּיק יְשׁוּעוֹת פּוֹעֵל

לְסוֹעֲרָה וַעֲנִיָּה / תַּעֲרֹג כְּאַיָּל וּכְיָעֵל

אוֹיָה לִי כִּי גַרְתִּי / בְאָהֳלֵי אֱדוֹם וְיִשְׁמָעאֵל

לֹא יֵאָמֵר לָךְ עוֹד עֲזוּבָה / וְלֹא עוֹד תִּגָּעֵל

כִּי חָפֵץ יְיָ בָּךְ / וְאַרְצֵךְ תִּבָּעֵל

וְיוּשַׁר בְעֵמֶק יִזְרְעֶאל / וּבְאֶרֶץ יְהוּדָה הַנָּוָה.
