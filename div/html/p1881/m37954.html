חשקי בבת מלכים תמה

חֶשְׁקִי בְּבַת מְלָכִים תַּמָּה / אֶעְנֶה בְּמַהְלְלִי

אָשִׁיר בְּשִׁיר וְאִמְרֵי חָכְמָה / עַל תּוֹף וְנֵיבְלִי:


זִיו תָּאֳרָהּ כְּאוֹר הַשֶּׁמֶשׁ / אָדוֹם כְּמוֹ אֲרִי

תּוֹאַר וְאוֹר אֲשֶׁר לֹא יָמֵשׁ / שִׁמְשִׁי וְסַהֲרִי

בָּהּ אֶמְצְאָה לְנַפְשִׁי נוֹפֶשׁ / הַצּוּף וְהַצֳּרִי

מִן הָעֲסִיס בְּפִיהָ שָׂמָה / עָלָה בְגוֹרָלִי

לֹא אֶרְעֲבָה וְגַם לֹא אֶצְמָא / חֶלְקִי וּמַזְלִי.


קוּמָה יְדִיד שְׁתֵה מֵימֶיהָ / נֵטַע וְנַעֲמָן

כִּצְרִי מְעַט וּבֵין שִׁנֶּיהָ / כִּדְבַשׁ וְצוּף וּמָן

כִּדְמוּת אֲפִיק כְּלִי עֵינֶיהָ / הָיְתָה כְּנֶאֱמָן

כַּמָּה אֱהֶא בְּחִשְׁקֵךְ עַלְמָה / נִרְצָע בְּאֵין כְּלִי

אִם תִּלְבְּשִׁי כְּלִיל עִם רִקְמָה / עֶדְיֵךְ תְּנַצְּלִי.


פִּתְחִי לְדוֹד בְּיָפְיוֹ כָלָה / לַחְזוֹת בְּתָאֳרֵךְ

אִם תֵּשְׁבִי בְּכִסֵּא מַעְלָה / לִשְׁכֹּן בְּשַׁעֲרֵךְ

אָז תִּתְּנִי בְּאָזְנֵךְ עִגְלָה / יוֹפִי בְּצַוְּארֵךְ

הַרְוִי לְמִי לְבָבוֹ צָמָא / מָזוֹן תְּכַלְכְּלִי

דּוֹד יְקַו בְּלִבּוֹ שָׁמָּה / שִׁכְנִי בְּאָהֳלִי.


מֵרוֹב פְּרוֹד אֲהוּבִי, יָרַד / דִּמְעִי כְּטַל שְׂנִיר

מִן הַנְּדוֹד בְּשָׂרִי יִיקַד / נִכְסָף אֱלֵי גְּבִיר

אַבְרָם אֲשֶׁר אֲנִי לוֹ עֶבֶד / נִרְצָע בְּאֵין מְחִיר

בּוֹ אֶעֱלוֹז וּבוֹ אָקוּמָה / עַל אוֹיְבִי, וְלִי

יֵקַר וְחֵן וְיָדוֹ רָמָה / עֶזְרִי וּמִגְדְּלִי.


קָמָה בְּיָדְךָ מֶמְשָׁלָה / רוֹעֶה וְנֶאֱמָן

אֵל שָׂמְךָ פְּאֵר לַגּוֹלָה / נֶטַע וְנַעֲמָן

אַבְרָם יְלוּד יְהוֹסֵף עָלָה / אֶל כָּל בְּנֵי זְמָן

אַבְרָם אֱמֶת וְיוֹסֵף הֵמָּה / זִכְרָם בְּמַהְלְלִי

אוֹר תָּאֳרָם כְּאוֹר הַחַמָּה / אֶעֱנֶה לְשׁוֹאֲלִי.
