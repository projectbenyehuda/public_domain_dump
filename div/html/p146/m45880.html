דָּל דּוֹפֵק

חתימה: רבי שלם אלשבזי


דָּל דּוֹפֵק שׁוֹאֵל / שָׁמְעָה אָזְנֵנוּ

מִגּוֹלַת אֲרִיאֵל / סְגֻלַּת הֲמוֹנֵינוּ

מְקַוֶּה חֶסֶד אֵל / צוּר מִָגִנֵּנוּ

שְׁמַי יִשְׂרָאֵל / יְוֹיָ אֱלֹהֵינוּ:


בַּחֲצַר הַמֶּלֶךְ / בְּעוֹד לֵיל דּוֹפֵק

מִמְּרֵעִים חֵלַךְ / וְחָדַל מִתְאַפֵּק

יָהּ לְאוֹיְבִי תִּשְׁלַךְ / זְמָמוֹ אַל תָּפֵק

וּבְכַפָּיו יִסְפּוֹק / כִּי בָא אֱלֹהֵינוּ:


יֶאֱתָה חֵן לְדַלִּים / לְקַבֵּץ נְדוּחֵיהֶם

יְשׁוּעַת יָהּ מְיַחֲלִים / כִּי הוּא גּוֹאֲלֵיהֶם

אָז יְשׁוּבוּן גּוֹלִים / פְּדוּיִים אֶל נְוֵיהֶם

בְּאַרְבַּעַת דְּגָלִים / גּוֹדֶל לֵאלֹהֵינוּ:


שִׁמְעִי חֲבַצֶּלֶת / נֶחָמָה מִדּוֹד

וְרוּצִי כְּאַיֶּלֶת / גְּבוּל עַזָּה וְאַשְׁדּוֹד

חֶסֶד מְקַבֶּלֶת / כִּי לָךְ יָהּ יְעוֹדֵד

עַם דָּל מִתְבּוֹדֵד / זַמְּרָה אֱלֹהֵינוּ:


לִוְיַת חֵן עֲטֶרֶת / כְּקֶדֶם תִּתְבּוֹסֵס

וְתָשׁוּבָה גְּבֶרֶת / בַּטּוֹב וְלֹא תִּתְנוֹסֵס

וְתָקוּם בְּאַשְׁמוֹרֶת / חֲצוֹת לֵיל רוֹסֵס

וְחֵיל שֵׂעִיר גּוֹסֵס / וּמִי צוּר זוּלָתִי אֱלֹהֵינוּ:


מֵאוֹצְרוֹת חַיִּים / יְזַלֵּף עַל חֲסִידָיו

וִיפַרְנֵס עֲנִיִּים / עֲדַת יַעֲקֹב יְדִידָיו

וְעַל יוֹשְׁבֵי אִיִּים / הֵם רַבִּים חֲסָדָיו

וְכָל מַעֲשֵׂה יָדָיו / כִּי קָדוֹשׁ יְוֹיָ אֱלֹהֵינוּ:


אֵלָיו הֵם מְקַוִּים / כָּל הֲמוֹנֵי יְצוּרִים

וּשְׁבָחוֹ מְחַוִּים / כִּי בְּטוּבוֹ עֲזוּרִים

נִפְסָדִים וְהֹוִים / וּבְרוּחַ קְשׁוּרִים

וּמֵרֹאשׁ אַתָּה הוּא / יְוֹיָ אֱלֹהֵינוּ:


לְחֶבְרַת בְּנֵי יַעֲקֹב / מְנַהֵל בְּרַחֲמִים

וּמֵאוֹצָר טוֹב / מוֹרִיד גְּשָׁמִים

וְדָל לֹא יַעֲזֹב / וְהוּא אֲבִי יְתוֹמִים

וְאַחֲרִית יָמִים / יְוֹיָ אֱלֹהֵינוּ:


שְׁחָקִים שִׁפְרָה / מְקוֹם הַצַּדִּיקִים

נְפָשׁוֹת הִקְרָה / בְּרוּחוֹ הֵקִים

וְגַלְגַּלֵּי הֶדְרָה / זֶה בְּזֶה דְּבֵקִים

בְּאוֹר מוּצָקִים / בְּעִיר אֱלֹהֵינוּ:


בְּחוֹשֶׁק תּוֹכְפִים / לְהַדְרַת חָשְׁקָם

וְתָמִיד שׁוֹאֲפִים / כִּי זֶה חֻקָּם

וְכִמְעַט רוֹדְפִים / בְּזִיו יְסוֹד דִּבְקָם

וְהַמֵּצִיקָם / שֵׁם אֱלֹהֵינוּ:


זְכוּת יָהּ יְלַמֵּד / לְעָנִי וְנִקְלֶה

בְּפִתְחוֹ עוֹמֵד / נִרְדָף וְגוֹלֶה

רְצוֹנוֹ חוֹמֵד / וְהוּא רָשׁ וְנִקְלֶה

בְּאַהֲבָה חוֹלֶה / רְפָאֵנוּ יְוֹיָ אֱלֹהֵינוּ:

יְחַזֵּק לְנִזְעָם / רָם עַל תְּהִלּוֹת

כְּפַעַם בְּפַעַם / עוֹרֵךְ תְּפִלּוֹת

וְלִבּוֹ הוּעַם / וְשָׂם בּוֹ מְסִלּוֹת

לִלְמוֹד טַעַם / שְׂרִידֵי סְגֻלּוֹת

וִיהִי נוֹעַם / יְוֹיָ אֱלֹהֵינוּ:
