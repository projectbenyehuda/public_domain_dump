תבל ככלה

לדון וידאל בנבנשת שלחם לר’ שנאור

תֵּבֵל כְּכַלָּה תַעֲדֶה כֵלֶיהָ / אוּלַי לְבָבִי יַעֲמֹד עָלֶיהָ

כִי הַסְּתָו עָבַר חֲלִיפוֹת תַּחֲלִיף / לַבְנוֹת יְעַרֶיהָ וְכַרְמִלֶּיהָ

רִקְמַת צְבָעִים רָקְמָה עַל מַחְלְפוֹת / כָּתְנוֹת שְׁתִילֶיהָ וְזַלְזַלֶּיהָ

לָבְשָׁה שְׁנִי תוֹלָע וְשָׂמָה מִשְׁבְּצוֹת / זָהָב לְרַגְלֶיהָ וְעַל שׁוּלֶיהָ

בֵּינוֹת עֲרוּגוֹת הַהֲדַסִּים נָטְעָה / כַּנַּת מְעוֹנֶיהָ וְאֹּהָלֶיהָ / 5

אֹהֶל בְּשׁוֹשַׁנַּת עֲמָקִים רָצְפָה / תוֹכוֹ וְעָלָיו חוֹפְפוּ עָלֶיהָ

וּבְתִמְּרוֹת בָּשְׂמָהּ וְרֵיחָהּ שׁוֹבְבָה / רוּחַ נְכֵאֶיהָ וְנֶחְשָׁלֶיהָ

אַךְ לַאֲשֶׁר לֹא יָדְעוּ דַת אַהֲבָה / תַּכְאִיב בְּקוֹצֶיהָ וּבַחְרוּלֶיהָ

וּבְיַד חֲבַצֶּלֶת שְׁרוֹנִים מָר דְּרוֹר / תָּפִיץ לְדוֹדֶיהָ וְלִרְגִּילֶיהָ

וּבְיַד יְלִיד שׂוֹרֵק שְׁלוֹמִים פָּרְשָׂה / וּבְבַת זְמוֹרָה תַּחֲלִים חוֹלֶיהָ / 10

הַבַּת אֲשֶׁר נוֹלְדָה בְיַד שֶׁמֶשׁ אֲבָל / תָּאִיר כְּאוֹר יוֹמָם חֲצוֹת לֵילֶיהָ

יָפָה כְמוֹ תִרְצָה וְהִיא מִתְעַלְּסָה / עַל פִּי נְחָלֶיהָ וְיוּבָלֶיהָ

הִתְעַתְּדָה תֵבֵל כְּיַעְלַת חֵן אֲשֶׁר / דּוֹדָהּ יְחַכֶּהָ וְיוֹחִילֶהָ

הִתְיַצְּבָה נֶגְדִּי לְפַתּוֹתִי בְטוּב / טַעְמָהּ בְחוֹתָמָהּ וּבִפְתִּילֶיהָ

וּבְמַעְגְּלוֹת הָאַהֲבָה הִתְקָרְבָה / אֵלַי וְלִבִּי בַל קְרוֹב אֵלֶיהָ / 15

יקר בְּעֵינֵי שׁוֹחֲרֶיהָ וַאֲנִי / אָבוּס גְּאוֹנָהּ וַאֲנִי אַקְלֶהָ

כִּי נוֹדְעוּ לִי מַעְגְּלוֹתֶיהָ וְלִי / נִגְּלוּ דְרָכֶיהָ ומַסְלוּלֶיהָ

תִּדְמֶה כְאִלּוּ עַל חֲלִיפוֹת הָטְבְּעָה / אוֹ עַל תְּמוּרוֹת יִשְׁכְּנוּ רַגְלֶיהָ

וּשְׁנוֹת צְבָאֶיהָ כְצֵל הִתְנוֹדְדוּ / גַּם כַּחֲלוֹם חָלְפוּ יְמֵי גִילֶיהָ

שׁוּר כִּי זְמָן עָבָר כְּבָר עָבַר וּמַה / יִּתְרוֹן בְּטוּב אַמְשָׁהּ וְאֶתְמוֹלֶיהָ / 20

וַעְמוֹד עֲלֵי הֹוֶה וְאֵין לוֹ מַעֲמָד / כִּי הַזְּמָן יָרוּץ כְּגַלְגַּלֶּיהָ

עָתִיד בְּלִי נִמְצָא וְאֵיךְ בּוֹ יִמְצְאוּ / נַחַת הֲלוּמֶיהָ וְנִכְשָׁלֶיהָ

אוֹ אֵיךְ מְנוּחָה בַזְּמָן אֶמְצָא וְהוּא / מִסְפַּר תְּנוּעָתָהּ וְגִלְגּוּלֶיהָ

תִּרְעַשׁ כְּיַם סוּף עַל יְלִידֵי תַחְפְּנֵס / כִּי לֹא כְרֶגַע יִרְגְּעוּ גַלֶּיהָ

עַל כֵּן מְאַסְתִּיהָ וְלֹא אֶחְמֹד גְּאוֹן / יָפְיָהּ וְחֶלְיָתָהּ וּמַכְלוֹלֶיהָ / 25

לִשְׁכּוֹן בְּסֵתֶר הַתְּבוּנָה אִוְּתָה / נַפְשִׁי וְלִשְׁמוֹעַ נְעִים מִלֶּיהָ

הִיא יַעֲלַת חִשְׁקִי וְנִכְסַפְתִּי אֱכוֹל / מִמַּן פְּרִי פִיהָ וּמִיבוּלֶיהָ

וָאֶעֱלֶה לִפְנֵי שנֵיאוֹר הַגְּבִיר / יִקְרַת חֲכָמֶיהָ ומַשְׂכִּילֶיהָ

אֶשְׁתַּחֲוֶה שָׁם לַהֲדֹם רַגְלָיו וְהוּא / יָפִיק רְצוֹן רוּחִי וּמִשְׁאָלֶיהָ

הַשָּׂר אֲשֶׁר אֶמְצָא בְיָדוֹ מַפְתְּחוֹת / כָּל מִסְגְּרוֹת חָכְמָה וּמַנְעוּלֶיהָ / 30

עָצְמוּ תְהִלּוֹתָיו וְיִדְמֶה תוֹךְ אֳנִי / תֵבֵל כְּמַלָּח אוֹ כְרַב חוֹבְלֶיהָ

הוא מִשְּׂרִידֵי מַחֲשַׁכֶּיהָ וְלֹא / יִיגַע בְּדֵי הֶבְלָהּ וּבְהִתּוּלֶיהָ

לֹא יַחֲרִידוּהוּ כְלֵי זַעְמָהּ וְלֹא / יִירָא בְמוֹט חֵילָהּ וּמַקְהֵלֶיהָ

אֵין לוֹ בְתוֹכָהּ נַחֲלָה לוּלֵי מְנַת / חָכְמָה וְעִנְיָנָהּ וּמֻשְׂכָּלֶיהָ

הָאֵר בְּאוֹר שִׂכְלָךְ שְׁנֵיאוֹר עַרְפְלֵי / רוּחִי ומחשכיה וּמַאְפֵלֶיהָ / 35

רַוֵּה גְבִיר צִמְאוֹן יְחִידָתִי בְמֵי / מִטְרוֹת מְזִמָּתָךְ וּבִטְלָלֶיהָ

וּפְרוֹשׂ כְּנַף חַסְדָּךְ לְנֶפֶשׁ שֹׁחֲרָה / הוֹדָךְ בְּתֻפֶּיהָ וּבִנְבָלֶיהָ

וּבְצִלְצְלֵי שָׁמַע לְךָ מִשְׁתּוֹחֲחָה / אוּלַי לְבָבָךְ יַעֲמֹד עָלֶיהָ
