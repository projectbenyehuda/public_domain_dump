אֲנִי עֲטֶרֶת

סימן: א-י.


אֲנִי עֲטֶרֶת תִּפְאֶרֶת / בִּל אַוַּל כִּנְתוּ כַאתּוּנָה

מֵאָז הָיִיתִי גְבֶרֶת / פִי גַ׳מִיע אִל-כַיר מַשְׁחוּנָה

עַתָּה מִבֵּיתִי נֶעְדֶּרֶת / הַלְ סַאע כִלּוֹנִי מִסְגוּנָה

בְּחַגְוֵי סֶלַע מִשְׁכָּנִי. / בִפְשׁוּק אִל-צֵּכֶר מַסְכַּנִי.


בְּזָכְרִי אֶת יְמֵי חָרְפִּי / מִן אַדְכִּר אִיָאם שָׁבַּאבִּי

בִּהְיוֹתִי בְחֵיק אַלּוּפִי / מִן כּוּנְתְּ פִי חִצְ׳ן חַבִּיבִי

לֹא אֵדַע אֵיךְ אָשִׁיב חוֹרְפִי / לַם אַעְרֶף אַעְטִי גַ׳וָאבִּי

עַתָּה, כִּי הוּא יַלְעִיגֵנִי. / קִדַּאם נַאסְ אִל-בַּאגִצִ׳ינִי.


גֵּרְשַׁנִי צַר בִּמְחִתָּה / הִגֹּגֹנִי פִי כִּתְרַת כוֹפָה

שָׁמַנִי כְיוֹנָה פוֹתָה / כִלּוֹנִי אַמְשִׁי צְ׳עִיפָה

אֵיךְ שִׁפְחָה תִירַשׁ גְּבִרְתָּהּ / אַשְׁלוֹן יִסִיר אִל-וַצִיפַא

וְעֶבֶד יִמְלֹךְ בִּמְכוֹנִי. / תִּקְעִד תִּסְכִּן פִי מַכַּאנִי.


דִּבַּרְתִּי אֲנִי עִם לִבִּי / אִפְתַּכַּרְתְּ פִי וַצַט קַלְבִּי

הֲיֵשׁ מַכְאוֹב כְּמַכְאוֹבִי / וּלַם שֻׁפְתוּ כַּעַנַאיְבִּי

כִּי עַתָּה רֵעַי בָּגְדוּ בִי / חִתַּא אַצְהַאבִּי כַאנוּ בִּיִי

הֵנִיעוּ רֹאשׁ עֵת רָאוּנִי. / הַזּוּ רַאסְהוּם מִן שַׁאפוּנִי.


הֲמוֹן מַעְלָה תִוָּעֲדוּ / אִל-מַלַאיְכָּה בִּיִי אתְיַאעְדוּ

הֲלֹא תִרְאוּ וְתָעִידוּ / קוּמוּ שׁוּפוּ וַאשְׁהַדוּ

אֲזַי לְדוֹדִי תַגִּידוּ / הַתַּא לְהַבִּיבִּי תִפִידוּ

כִּי חוֹלַת אַהֲבָה אֲנִי. / לִעִשְׁקוּ וִגְ׳עַאנַא אַנִי.


וְקוֹל כְּחוֹלָה אָרִימָה / אַתְחַנַּן, לִרַבּ אִל-עַאלִי

כְּמוֹ קוֹל יוֹנָה אָהִימָה / לִאַגִל יִנְצִ׳ר אַחֶוָאלִי

אֵין לְעַפְעַפַּי תְּנוּמָה / וּחַתַּא פִי אִל-לִּיָאלִי

לֹא אֶתֵּן שֵׁנָה לְעֵינִי. / לַם אַגְעַל נוֹמַא לְעֵינִי.


זֶה לִי כַיּוֹלֵדָה אֶפְעֶה / יַא רַבִּי אִקְבַּל שִׁפַאעִי

עַל מְעַט הַצֹּאן כִּי תוֹעֶה / עַלַא גְנַאמַךְ אִסְבַּאעִי

מָתַי תָּקִים לָהֶם רוֹעֶה / כֵּיף הַליוֹם לֵיסלְהוֹם רַאעִי

רְאֵה כִי נָפוֹצוּ צֹאנִי. / יִרְעִיהוּם פִי קַלְבְּ רַחְמַאנִי.


חִלִּיתִי אֵלֶיךָ דּוֹדִי / אִנְתַּכַּלְתְּ עַלֵיךְ, יַא רַבִּי

שִׁוִּיתִי שִׁמְךָ לְנֶגְדִּי / צַוַּרְתְּ אִסְמַךְ פִי קַלְבִּי

רְאֵה צַר חִלֵּל כְּבוֹדִי / אִנְתַּא כַאלְקִי וִאַבִּי

מֵעִיר שִׁבְתָּךְ הִדִּיחַנִי. / מְעַיְשְׁנִי וּמְרַבִּינִי.


טֶרֶם אֶקְרָא, דּוֹד, תַּעֲנֶה / בִּפַצְ׳לַךְ אִסְמַע כַּלַאמִי

כִּי לֹא אֵדַע אָנָה אֶפְנֶה / וְחִנִיד׳ אִשְׁפִק עַלַא קוֹמִי

אִם עוֹד צַר אוֹתִי יְעַנֶּה / אַסְתַּרְגִ׳י לַךְּ כֻּל אֵיָאמִי

אֵיכָה תֹאמַר אֲהַבְתָּנִי. / פִי כִּתְרִת רַחְמַתַּךְּ אִרְחַמְנִי.


יָהּ, שְׁלַח נָא לִי אִישׁ תִּשְׁבִּי / אִפְתַכַּר נִחְנַא עַבִּידַךְּ

יְבַשֵּׂר שָׁלוֹם בְּקִרְבִּי / לֵיס לִנַא גִ׳יר אִנְתַא וַחְדַּךְּ

שׁוּבִי, הַשּׁוּלַמִּית, שׁוּבִי / אִבְעַתִ אִל-רַסוּל מִן עִנְדַּךְּ

לְצִיּוֹן הִיא עִיר שְׂשׂוֹנִי. / פִי פַכַּאכִּי יִבִּשְׂרְנִי.
