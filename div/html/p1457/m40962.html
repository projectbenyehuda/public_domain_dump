אופן

ט1

וְחַיּוֹת הַקֹּדֶשׁ אֲשֶׁר בַּשָּׁמַיִם / בְּרוּאוֹת הֵם עַד לֹא אֶרֶץ

גִּיוְיוֹתָם אֵשׁ מִתְלַקַּחַת בַּשָּׁמַיִם / דְּמוּת מְכֻוָּנוֹת לְכַנְפוֹת הָאָרֶץ

הַדָּר מֵעַל יְמִינוֹ טִפְּחָה שָׁמַיִם / וּמִתַּחַת יָדוֹ יָסְדָה אָרֶץ

זִיו מוֹשַׁב יְקָרוֹ בַּשָּׁמַיִם / חוֹנֶה הָיָה מִתְּחִלָּה בָּאָרֶץ

טָס וַיֵּדֵא הֱיוֹת בַּשָּׁמַיִם / יְצוּרִים עֵת יָצַר בָּאָרֶץ

כְּבָא יוֹם זֶה שָׂמְחוּ שָׁמַיִם / לָתֵת תּוֹרָה לְגוֹי אֶחָד בָּאָרֶץ

מִיָּד הִכְרִיז בְּצִבְאוֹת שָׁמַיִם / נָם לְהוֹדִיעַ גְּבוּרוֹת בָּאָרֶץ

שִׂיחַ הֱשִׁיבוּהוּ רוּם עַל שָׁמַיִם / עֻזְּךָ אֵיךְ תּוּכַל לְהָכִיל הָאָרֶץ

פַּרְשֵׁז עָנָן מְרוֹמֵי שָׁמַיִם / צָר וְהֵכִין עַל פְּנֵי כָל הָאָרֶץ

קוֹל הַדִּבֵּר הִשְׁמִיעַ מִשָּׁמַיִם / רָעֲדוּ וְחָלוּ יְסוֹדֵי הָאָרֶץ

שַׁדַּי הַנּוֹטֶה כַדּוֹק שָׁמַיִם / תִּנָּה לְעַמּוֹ אֲרֻכָּה מֵאֶרֶץ

הֱעִידָם בַּשָּׁמַיִם הַאֲזִינוּ הַשָּׁמַיִם / וְהִסְהִידָם בָּאָרֶץ וְתִשְׁמַע הָאָרֶץ

כִּי אֲנִי הוּא בַּשָּׁמַיִם / וּמַבִּיט בָּאָרֶץ

בָּרוּךְ בַּשָּׁמַיִם / וּמְבוֹרָךְ בָּאָרֶץ


בקול רעש גדול משמיע לעומת שרפים לעומתם…


ט2

תַּרְשִׁישֵׁי שַׁחַק חֲצוּבִים מֵאֵשׁ / שְׁאוֹנָם מוֹצִיא זֵיעוֹת כַּמַּיִם

רְאִיַּת פְּנֵיהֶם כְּלַפִּידֵי אֵשׁ / קוֹל דִּבְרֵיהֶם כְּקוֹל מַיִם

צְפוֹת אֵין בָּם בְּכֻלּוֹ אֵשׁ / פֶּן יִשְׂרְפוּ כַּעֲצֵי מַיִם

עֲבוּר כִּי יֵשׁ אֵשׁ שׂוֹרֶפֶת אֵשׁ / סְבוּבָה עֲרָפֶל וַעֲנַן מַיִם

נָאוֹר בְּעֵת יָרַד לְסִינַי בָּאֵשׁ / מֻקָּף הָיָה שֶׁבַע עַנְנֵי מַיִם

לְבוּשֵׁי גֵאוּת וְעוֹז […אֵ]שׁ / כָּתַב בְּאֶצְבְּעוֹתָיו מְשׁוּלַת מַיִם

יְרִיעָה לְבָנָה חָתַךְ מִן הָאֵשׁ / טִפִּים שְׁחוֹרוֹת הִמְשִׁיךְ כַּמַּיִם

חָרַת וְנָתַן מִימִינוֹ דָּת אֵשׁ / זֹאת הַתּוֹרָה לְמָשׁוּי מִמַּיִם

וְזִיק יָצָא כְּבִרְקֵי אֵשׁ / הוֹלֵךְ ו[…]יל עַל נַחֲלַת מַיִם

דִּבֵּר כְּשָׁמְעוֹ נִלְהֲטוּ בָאֵשׁ / גֶּשֶׁם נְדָבוֹת הוּנַף עֲלֵיהֶם כַּמַּיִם

בְּאַהַב קִבְּלוּ דְבָרוֹ בֵּית אֵשׁ / אַף אִם יָבוֹאוּ בָּאֵשׁ וּבַמַּיִם

לֹא שָׁכְחוּ שְׁמוֹ בְּלַהַב אֵשׁ / בְּכִבְשַׁן אֵשׁ וּבְבוֹר מַיִם

כִּי עַזָּה כַמָּוֶת אַהֲבָה רְשָׁפֶיהָ רִשְׁפֵי אֵשׁ / וְלֹא יוּכְלוּ לְכַבּוֹתָהּ מַיִם

וּמְיַחֲדִין בְּנֶשֶׁף וָשַׁחַר בְּכָל יוֹם פַּעֲמַיִם


ט3

אֶחָד יִחַדְנוּ לִשְׁמוֹ בְסִינַי / בְּרוֹב עוֹז נִהֲלָנוּ לְצִיּוֹן

גַּם יֶתֶר מֵחֶמְדַּת סִינַי / דָּרַשׁ וְאָהַב אֶת הַר צִיּוֹן

הֲלֹא לְשָׁעָה הָיָה קִדּוּשׁ סִינַי / וּקְדֻשַּׁת עֲדֵי עַד הָקְדְּשָׁה צִיּוֹן

זמ[…] וְהָיָה מִתְהַלֵּךְ בְּאֹהֶל סִינַי / חֲנוֹתִי לְבֵיתִי מִצָּבָה בְּצִיּוֹן

טְכוּסִים בְּמַחַן נִהֲגָנוּ בְסִינַי / יָשַׁבְנוּ כְּכַלָּה בְּחֻפָּתָהּ בְּצִיּוֹן

כְּנָטַשְׁנוּ דָת צֻוֵּינוּ בְסִינַי / לְמַעֲנֵנוּ נָטַשׁ וְעָזַב צִיּוֹן

מוֹרְדִים אֲשֶׁר בָּאוּ וְנִלְחֲמוּ בְסִינַי / נָמוּ עָרוּ עָרוּ עַד הַיְסוֹד בְּצִיּוֹן

סֶלָה עַד תִּפָּתַח תִּקְוָה מִסִּינַי / עֲלוֹת מוֹשִׁיעִים מִשָּׁם לְצִיּוֹן

פָּרָשׁוֹת סְתָם שֶׁנֶּאֶמְרוּ בְסִינַי / צוּר תְפָרֵשׁ לָנוּ בְּצִיּוֹן

קוֹל פֶּה אֶל פֶּה הִשְׁמַעְתָּנוּ בְסִינַי / רְאוֹת עַיִן בְּעַיִן נֶחֱזֶה בְצִיּוֹן

שִׁבְעָה מִינֵי אֵשׁ הָיוּ בְסִינַי / תַּקִּיף כֵּן יִהְיֶה חוֹמַת אֵשׁ לְצִיּוֹן

וּכְהִשְׁמַעְתָּנוּ בְסִינַי / כֵּן תַּשְׁמִיעֵנוּ בְצִיּוֹן

להיות לכם לאלהים
