מי כמוך

מִי כָמוֹךָ בָּאֵלִים יְיָ מִי כָמוֹךָ נֶאְדָּר בַּקֹּדֶשׁ.

אוֹפַנֵּי מְעוֹנַי / סוֹבְבִים עַל פָּנַי / וְהִשְׁתַּחֲווּ לַייָ / בְּהַר הַקֹּדֶשׁ.

בְּשֵׁם צְבָאָיו קוֹרֵא, / כִּי בִשְׁתֵּים עֶשְׂרֵה / מַעֲלוֹת הֵם שֹׁמְרֵי / מִשְׁמֶרֶת הַקֹּדֶשׁ.

רָצִים בְּמַעַגְלֵיהֶם; / בַּל יִקְרַב אֲלֵיהֶם / נֵכָר, כִּי עֲלֵיהֶם / עֲבֹדַת הַקֹּדֶשׁ.

הֵא גַלְגַּל יַנְחֶה, / פִּתַּח עָלָיו חֵי / עוֹלָם פִּתּוּחֵי / חֹתָם קֹדֶשׁ.

מִי יִתֵּן וְחֶדְלִי / תָּמִיד יַרְאֶה לִי / הֲלִיכוֹת אֵלִי / מַלְכִּי בַּקֹּדֶשׁ.
