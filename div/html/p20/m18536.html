אֵימוֹת עָלַי נָפְלוּ

תוכחה ליום הכיפורים


אֵימוֹת עָלַי נָפְלוּ / וּשְׂעִפַּי בִּי נִבְהֲלוּ,

בְּהַעֲלוֹתִי עַל לְבָבִי / דְּבַר יוֹנָה הַנָּבִיא.


אֵין יֵשׁ כִּי אִם אִתְּךָ, / וְאֵין לִי מָנוֹס בִּלְתְּךָ,

וְעָדֶיךָ כָּל בָּשָׂר / יָבוֹא תַּחַת מוּסָר.


בְּךָ מַצַּב עוֹלָמְךָ, / וְכָל שֵׁם נִמְצָא בְּשִׁמְךָ –

וּמִי יִסָּתֵר מִמְּךָ? עֹז וְתוּשִׁיָּה עִמְּךָ!


רַמְתָּ מֵהַבִּיעַ, / וְאָן אֶסַּק וְאַצִּיעַ –

עֹז כְּבוֹדְךָ בַּמָּרוֹם / וּשְׁאוֹל נֶגְדְּךָ עָרוֹם?


הֵן כָּאוֹרָה כַּחֲשֵׁכָה / לְךָ, וְהַחֹשֶׁךְ – אֵיכָה

אֹמַר כִּי יְשׁוּפֵנִי, / וְלַיְלָה אוֹר בַּעֲדֵנִי?


מַחֲשַׁבְתִּי לוֹ נִתְכְּנָה, / וּבְאַחֲרִית יָם אֶשְׁכְּנָה –

הֲלֹא גַם שָׁם הִנֶּךָּ / וְתֹאחֲזֵנִי יְמִינֶךָ.


עָלֶיךָ אֶשָּׁעֵן, / וְאֵלֶיךָ עֵינִי כְּעֵין

עֶבֶד אֶל יַד אֲדוֹנָיו, / לָשֵׂאת אֵלָיו פָּנָיו.


זְכֹר, אֲנִי – אִם מוֹבָאִי / לְאַיִן, כְּמוֹ מוֹצָאִי

מֶנְהוּ, אֵי חֲסָדֶיךָ? / הֲשָׁוְא מַעֲשֵׂה יָדֶיךָ?


רַבִּים הֵם אֲשָׁמַי – / סְלַח נָא, כִּי מְעַט יָמָי;

כְּחָמְלָךְ עַל פָּעֳלֶךָ / שׁוֹב אָשׁוּב אֵלֶיךָ.


אֶבְטַח עַל רַחֲמֶיךָ / וְסָר מֶנִּי זַעְמֶךָ

וּמִפִּשְׁעִי לֹא אִירָא, / כִּי שֵׁם יְיָ אֶקְרָא.
