בִּידֵי אֵל יְהִי אוֹרֶךָ

אַנְשֵׁי אֵל סְחִי לִבְקָרִים / לִבְקָרִים,

כִּי לֹא נִבְדְּלוּ מִנְזָרִים / מִן זָרִים,

וּטְמֵאִים עֲלֵי נִיד בָּרִים / נִידְבָּרִים.

יוֹם הִבְאִישׁ מְרִי מוֹרֶיךָ / מוֹרֶךָ – / אָז צָעַד עֲלֵי שׁוּריֶךָ / שׁוּרֶךָ.


בּוֹזֶיךָ, אֲבִי- עַד, אָנָּה / עַד אָנָּה?

וּזְרוֹעַ יְקָר אֵל קָנָה / אֶל קָנָה

לֹא שָׁבָה, וְעִיר אַלְמָנָה / אַלְ מָנָה

לָהּ בָּךְ, עַד נְשׁוֹת תּוֹרֶיךָ / תּוֹרֶיךָ; / אֶל מִי תַּעֲזֹב בָּרֶיךָ / בָּרֶיךָ?


'רַבּו מִבְּכִי מֵי עֵינָךְ / מֵעֵינָךְ,

יוֹם אֶל לִבְּךָ, שַׂר אוֹנָךְ / סָר אוֹנָךְ.

עוֹד תִּרְאֶה, בְּבוֹא פִדְיוֹנָךְ, / פִּד יוֹנָךְ–

חֹזֶק יָד כְּמוֹ צָרֶיךָ / צָרֶיךָ, / עוֹד תִּרְאֶה הֲדוֹם צוּרֶךָ / צוּרֶךָ.


מַעַל שָׂם מְתִים כִּכְסִילִים / כִּכְסִילִים

יַעַן קָרְאוּ לִבְעָלִים / לִבְעָלִים,

וַיְהִי כַנְּךָ לִנְבָלִים, / לִנְבָלִים.

עַתָּה יַחֲבֹש צִירֶיךָ / צִירֶיךָ, / יַכְרִית מִנְּוֵה עָרֶיךָ / עָרֶךָ.'
