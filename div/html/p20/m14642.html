אַתְּ שֶׂה פְזוּרָה

אַתְּ שֶׂה פְזוּרָה, אֵיךְ / מָרִית בְּמוֹרָיִךְ?

אָנָה הָלַךְ / רוֹעֶה עֲדָרָיִךְ?


בִּנְאוֹת דֶּשֶׁא / רָבַצְתְּ יוֹם רָעָה

אוֹתָךְ משֶׁה, / עַד בּוֹא יְמֵי רָעָה;

רוֹעֵךְ יֶחֱשֶׁה – / וַתִּהְיִי תוֹעָה,

מַרְאֶה לְכָל רוֹאֵךְ, / יִדְּפוּךְ הֲרָרָיִךְ,

נִזְרֵךְ הֻשְׁלַךְ, / אֵין קֵץ לְצִירָיִךְ.


רָאָה צַלְעֵךְ / הָאֲרִי וְלָךְ אָרָב,

מִי יוֹשִׁיעֵךְ / מִפִּיו בְּעֵת יִקְרָב?

מִי יַשְׂבִּיעֵךְ / מַיִם בְּבוֹא שָׁרָב?

אוֹ אָן יְהִי בוֹאֵךְ? / נִתְּשׁוּ גְּדֵרָיִךְ,

כָּלִית כֻּלָּךְ, / לָעוּ דְבָרָיִךְ.


'מַלְבּוּשׁ קִנְאָה / אֶלְבַּשׁ, אֲנִי רוֹעֵךְ.

עֵינִי רוֹאָה / פִּצְעֵךְ בְּרֹב פִּשְׁעֵךְ,

תִּתֹּם חֶלְאָה / מִמֵּךְּ וְסָר נִגְעֵךְ.

שַׁבְתִּי לְהוֹצִיאֵךְ / מִבֵּין כְּפִירָיִךְ,

זָכַרְתִּי לָךְ / חֶסֶד נְעוּרָיִךְ.'
