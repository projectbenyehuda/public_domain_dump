כל עצמותי

כל עצמותי / ר' אברהם אבן-עזרא


כָּל עַצְמוֹתַי תֹּאמַרְנָה: יְיָ / מִי כָמוֹךָ?


אֶפֶס בִּלְתְּךָ בְעֵת צָרָה, / עַל כֵּן לְבָבִי

בָּטוּחַ בָּךְ: וּמָה אִירָא – / וְאַתְּ סְבִיבִי

מָגֵן בַּעְדִּי בְיוֹם עֶבְרָה / מִצָּר יְרִיבִי?

         אַתְּ מַחֲנוֹתַי, / אִישׁ מִלְחֲמוֹתַי,

         שָׂר רִבְבוֹתַי / מוּל שׂוֹנְאוֹתָי!

           אַתָּה בְחֵיק מוֹנַי / קֶשֶׁת דְּרוּכָה!


בָּךְ לִדְרֹש אֶת רְצוֹנֶךָ – שִׁמְךָ קְרָאתִיו

וּבְהִתְנַפֵּל לְפָנֶיךָ – / לִבִּי מְצָאתִיו,

אֵלֶיךָ אֶל מְעוֹנֶךָ / מַשְׂאֵת נְשָׂאתִיו –

         כִּי מַחְשְׁבוֹתַי / הֵם מַעְשְׂרוֹתַי

         וְדַבְּרוֹתַי / הֵם מִזְבְּחוֹתַי,

              אִשַּׁי וְקָרְבָּנַי – / נַפְשִׁי שְׁפוּכָה.


רוּחִי תַיְשַׁר וְתַנְחֵנִי, / יָהּ, לְעָבְדֶךָ,

תִּפְקַח לִבִּי וְאֶת עֵינִי / לַחְזוֹת כְּבוֹדֶךָ.

מַעֲנֵה לָשׁוֹן תְּחָנֵנִי, / הַכֹּל בְּיָדֶךָ.

         טוּב מִפְעֲלוֹתַי / הֵם קָרְבְּנוֹתַי

         וּמַתְּנוֹתַי / פְּרִי שְׂפָתָי,

               מִפִּי וְרַעְיוֹנַי / קַח נָא בְּרָכָה.


מֶלֶךְ אֶחָד אֲשֶׁר אֵלָיו / כָּלוּ נְפָשִׁים,

כִּי מַהֲלָלָיו – בְּמַעְלָלָיו / רָאוּם דְּרוּשִׁים,

יוֹשֵׁב קֶדֶם – וּמִפְעָלָיו / כָּל עֵת חֲדָשִׁים:

         עִם דּוֹר אֲבוֹתַי / וְעִם שְׁנוֹתַי

         וְתוֹלְדוֹתָי; / כִּי כָל יְמוֹתַי

               בּוֹ יֶחֱזוּ עֵינַי / אוֹת הַמְּלוּכָה.
