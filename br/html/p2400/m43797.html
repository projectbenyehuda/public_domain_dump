חָרַד לִבִּי

לשיר על דרך המוסר בענין התפלה


חָרַד לִבִּי בְּרֹב פַּחַד וּרְעָדָה / אֵיךְ אֲדַבֵּר לִפְנֵי אֵל, צוּר תְּעוּדָה

וּמָה אֲנִי וּמָה עַצְמִי וְעֶרְכִּי / כִּי אֲבַקֵּשׁ מֵאֵת מַלְכִּי כָּל צָרְכִּי

וְאָמְנָם זֹאת עֲנֹתוֹ הִיא תַּרְבֵּנִי / וּבְרֹב חַסְדּוֹ וְטוּבוֹ הוּא יִרְצֵנִי

לָכֵן אָחִי בֹּא נָא וְנִוָּכְחָה / הַסְכֵּת וּשְׁמַע וַאֲדַבֵּר נְכוֹחָה

עֵקֶב אֲשֶׁר הִגְדִּיל נַפְשָׁךְ וְהִרְצָךְ / לְהַלְּלוֹ וְלִשְׁאֹל אֶת כָּל חֶפְצָךְ

אֵיךְ לֹא תֵבוֹשׁ בְּעָמְדָךְ לְהִתְפַּלֵּל / וְלִבְּךָ לֹא עִם פִּיךָ יְמַלֵּל

שְׂפָתֶיךָ הֵמָּה נָעוֹת לְבַדָּם / אֲשֶׁר מֵאָז זֶה דַּרְכָּם וְלִמּוּדָם

עֵת פֶּה יִצְעַק לַאֲדוֹן הַיְּצוּרִים / הַלֵּב יַחְשֹׁב בְּמַחְשָׁבוֹת הַזָּרִים

גַּם יְבַקֵּשׁ לְרָצוֹן הֶגְיוֹן לִבּוֹ / וְלֹא הִכִּיר מַה הוּא שֵּׂחוֹ וְנִיבוֹ

עַד אָן יִהְיֶה לִבָּךְ תּוֹעֶה כְשִׁכּוֹר / וְאֶת דְּבַר יֶתֶר בְּאַיִן תִּמְכֹּר

חַזֵּק עַצְמָךְ וְתַעֲרֹךְ מִלְחָמָה / נֶגֶד יִצְרָךְ וְאַל תַּחְשֹׁב מְזִמָּה

שִׂים לְנֶגְדָּךְ לִפְנֵי מִי עֲמִידָתְךָ / וְאָז תּוֹסִיף יִרְאָה עֲלֵי יִרְאָתְךָ

בָּרֵךְ אֵל חַי כִּי לוֹ נָאוָה תְהִלָּה / אֲשֶׁר הִרְשָׁךְ לַעְרֹךְ שְׁבָח וּתְהִלָּה.
