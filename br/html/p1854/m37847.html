אֵלִי הֱשִׁיבַנִי בַאֲנָחָה

אֵלִי הֱשִׁיבַנִי בַאֲנָחָה וּמְהוּמָה / אֱלֹהִים הֲדִמַּנִי נְתָנַנִי שׁוֹמֵמָה דָּרְכוּ אַחַי בְּגָדְלָה וְעָצְמָה / חַטָּאתִי עַד שְׁמֵי שָׁמַיְמָה וְנוֹסַף עוֹד בְּתוֹךְ חֶטְאִי אַשְׁמָה / וְהִתְלַחֲשׁוּ יַחַד וְגַם חָשְׁבוּ מְזִמָּה נְתָנוּנִי לְחֶרְפָּה וְלִכְלִימָה / לַעַג וְקַלָּסָה בְּתוֹךְ כָּל עַם וְאֻמָּה יְשַׁלְּמוּ רַע כְּמוֹ נוֹקֵם נְקָמָה / מְצוּדָתָם הִקִּיפוּ כְחוֹמָה מִקְדַּשׁ אֵל אֲשֶׁר הוּרַם תְּרוּמָה / נְתָנוּהוּ לְחָרְבָּה וּשְׁמָמָה הֲפָכוּהוּ כְּאַדְמָה וּסְדוֹמָה / מְכָרוּהוּ לְמוֹאָב וֶאֱדוֹמָה לְהַכּוֹתִי יוֹעִילוּ בְמִרְמָה / אוֹיְבַי וַאֲנִי עוֹד יְתוֹמָה וְלוּלֵי הַשּׁוֹכֵן מְרוֹמָה / אֱלֹהַי וַיָי אֲשֶׁר הָיָה שָׁמָּה יוֹשֶׁבֶת כְּבַת מֶלֶךְ פְּנִימָה / לִבְשִׁי עוֹז וְהוֹד תּוֹרָה תְמִימָה בִטְחִי בָאֵל וְאַל תִּרְהִי מְאוּמָה / מִמּוֹאָב וְיִשְׁמָעֵאל וְדוּמָה רַם יַשְׁלִיךְ בִּמְצוּלוֹת תְּהוֹמָה / כָּל פֶּשַׁע וְכָל חַטֹּאות אֲיֻמָּה נָשׁוּבָה בִתְשׁוּבָה שְׁלֵמָה / וְהוּא יִזְכֹּר כִּי בָשָׂר הֵמָּה שֵׂכֶל טוֹב נִמְצָא וּנְעִימָה / וּמַרְגּוֹעַ וּמְנוּחָה עֲצוּמָה יוֹצֵר כֹּל יְקִימֵנוּ בְקִימָה / וִיכַלֶּה צָרֵינוּ בְחֵמָה מִקְדָּשִׁי הַשָּׁמֵם יְרוֹמֲמָה / בִּימִינוֹ וּזְרוֹעוֹ בְרָמָה זֹאת אָשִׁיב אֶל לִבִּי בְחָכְמָה: / חַסְדֵּי יְיָ לֹא כָלוּ לְרַחֲמָה חַסְדֵּי יְיָ כִּי לֹא תָמְנוּ כִּי לֹא כָלוּ רַחֲמָיו.
