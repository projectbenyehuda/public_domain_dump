מעמד ז: הוֹי עַל רַב

החתימה: שלם יושף


הוֹי עַל רַב נָעִים / מְלַמֵּד תּוּשִׁיָּה

מָתְנַי נִרְתָּעִים / וְעֵינִי בוֹכִיָּה:


שְׁמוּעָתוֹ טוֹבָה / בְּכָל עִיר וּמְדִינָה

בְּכָל סוֹד יְשִׁיבָה / וּמִלִּים יַאְזִינָה

רֹאשׁ הוּא לַיְּשִׁיבָה / בְּרוֹב שֵׂכֶל וּבִינָה

כְּאַבִּיר הָרוֹעִים / בְּחָכְמָתוֹ הָיָה:


לְבָבִי יֵאָנַח / בְּכָל עֵת אַזְכִּירוֹ

וְלַשִּׂמְחָה יִזְנַח / וְיִנְשֶׁה אֶת שִׁירוֹ

יְקָר הָעִיר צָנַח / וְנִכְמָס עִם הֲדָרוֹ

וּפַחַד וּזְוָעִים / בְּכָל מָקוֹם וְקִרְיָה:


מִי זֶה יִדְמֶה לוֹ / וְיָבִין חָכְמָתוֹ

וְיַנְעִים אֶת קוֹלוֹ / בַּהֶגְיוֹן כְּמוֹתוֹ

בְּיוֹם שָׁמְעִי אֶבְלוֹ / וְיַלָּלוּת בֵּיתוֹ

אֲחָזוּנִי פְגָעִים / וּמִסְפֵּד וּנְהִיָּה:


יוֹרֶה לַנְּבוֹכִים / בְּסוֹדוֹת וּרְמָזִים

בּוֹ הָיוּ סְמוּכִים / בְּבוֹאוֹ נֶעְלָזִים

וְהַיּוֹם הֵם בּוֹכִים / אַחֲרָיו נֶחְפָּזִים

וְלִבּוֹתָם נְגוּעִים / וְנַפְשָׁם צוֹפִיָּה:


וְכֵיצַד אָבִינָה / בְּתוֹרָה וּבְמִשְׁנָה

וְעוֹד לִבִּי נַעְנָה / עַל רָב הַתְּבוּנָה

וְאֵיךְ אַבִּיעַ רִנָּה / וְהֵן עָלַת קִינָה

וּבְגָדַי קְרוּעִים / כְּנֶאְסָר בַּשִּׁבְיָה:


שְׂשׂוֹן הַמִּדְרָשׁ סָר / וְאֵין בּוֹ כָּל דּוֹרֵשׁ

אַחַר הַטִּפְסָר / אֲשֶׁר הָיָה בָּרֹאשׁ

כְּמוֹ גוֹלֶה וְנֶאְסָר / לְבָבִי מִתְגָרֵשׁ

עַל חֶמְדַּת נְטִיעִים / אֲשֶׁר הוּא כִּדְלָיָה:


פּוֹתֵר הַמִּלּוֹת / וּמֵבִין אֶת סוֹדָם

נְשִׂיא רֹאשׁ הַקְּהִלּוֹת / וְסֵגֶן מַעְמָדָם

וְאֵיךְ נוּכַל לְחַלּוֹת / וְהַמָּוֶת קָדַם

וְאֵין כֹּל נוֹשָׁעִים / מִיּוֹם תַּאֲנִיָּה:
