אֵל נוֹרָא

חתימה: אלשבזי


אֵל נוֹרָא / יְעוֹרֵרָה / חֵן עַל קְהִלָּה מְחַכָּה

כִּי חַשְׁתִּי / וְשָׁמַעְתִּי / אֶת קוֹל רָחֵל מְבַכָּה

קוֹל בְּרָמָה / נֶעֱלָמָה / הֵעִיר אֶת הַמַּלְכָּה

תֹּאמַר רְצֵה / צוּרִי וְצֵא / עֲזוֹר שְׁפַל רוּחַ דַּכָּא

אֵלִי אַתָּה / וְאוֹדֶךָּ / אֱלֹהַי אֲרוֹמְמֶךָּ:


לְהַצִּיל רָשׁ / אֲשֶׁר גּוֹרָשׁ / יָחִישׁ אֱלֹהֵי יְשׁוּעוֹת

נִשְׁבָּה בְּיוֹם / חֲרוֹן אָיוֹם / וּמִבֵּית אֲבוֹתָיו שָׂם עֵת

לִקְרוֹא דְרוֹר / וּלְעוֹרֵר / נְפָשׁוֹת הַנִּצְנָעוֹת

וְצוּר מֵקִים / צַדִּיקִים / עַד מָה אַשּׁוּר תִּשְׁבֶּךָּ:


שְׁבָח תַּנְעִים / בַּת נָעִים / לְהוֹדוֹת רָם וְגַם נִשָּׂא

וְגַן בִּיתָן / לְבֶן אֵיתָן / הֲדַר הֵיכָל מְבוֹסֵסָה

וְהוֹד רוּחוֹת / כְּמִטְפָּחוֹת / עַל נְפָשׁוֹת בְּאוֹר כִּסָּה

יָהּ שַׁכֵּן / לְדָל מִסְכֵּן / תְחִי נַפְשִׁי וּתְהַלְלֶךָּ:


בִּשְׂמֹאלִי / גְּעַר אֵלִי / וְאַל יַטְעֶה יְחִידָתִי

וְקָרֵר אֵשׁ / וְהִתְלַבֵּשׁ / בְּחַסְדְּךָ צוּר יְשׁוּעָתִי

וְשִׂים חֶשְׁקִי / בְךָ חֶלְקִי / וְרַחֵם עַל סְגֻלָּתִי

בְּעֵת תְּחַיָּם / וּמִן הַיָּם / תִּמְשׁוֹךְ לִוְיָתָן בְּחַכָּה:


זְמָן לְבוֹשׁ / כְּסוּת לִלְבּוֹשׁ / כִּי אַתָּה מִבְצָרִי

וְסוֹד טוּבִי / וְכָל יָהְבִי / עָלֶיךָ וּמַחְסוֹרִי

לְךָ נַפְשִׁי / וְגוּף חָלְשִׁי / וְאַתָּה יִשְׁעִי וְאוֹרִי

עֲנֵה עֲתֶרֶת / שְׁחַרְחֹרֶת / כְּסַמִּים וּלְבוֹנָה זַכָּה:


יְקָר יָשֵׁת / וְיִתְעַשֵּׁת / הָאֱלֹהִים לְהַצִּילָה

לְאֶבְיוֹנִים / בְּקוֹל עוֹנִים / לִדְרוֹשׁ חֵן וּמְחִילָה

וְיִרְצֶה בָם / בְּעֵת שׁוּבָם / בְּעֵת רָצוֹן יַקְבִּילָה

וִיבַשֵּׂר / לְמִתְיַסֵּר / עֲטֶרֶת תִפְאֶרֶת תְּמַגְנֶךָּ:
