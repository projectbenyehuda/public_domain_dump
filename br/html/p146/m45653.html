יוֹם זֶה

חתימה: אלשבזי משתא


יוֹם זֶה עֲווֹנֵנוּ יְכֻפַּר / בִּרְצוֹן אֱלֹהִים אֱלֹהֵינוּ

וּבְרִית אַחִים לֹא תוּפָר / בָּא מֵלִיץ טוֹב בַּעֲדֵנוּ

וְהוּשְׁחָר הַמְקַטְרֵג בְּאֶשְׁפָּר / וְהִנֵּה שָׁמְעָה אָזְנֵנוּ

תִּקְעוּ בַחֹדֶשׁ שׁוֹפָר / בַּכֵּסֶא לְיוֹם חַגֵּנוּ:


אַהְבַת רַעְיָה תְּלַמֵּד חֵן / וְחֶסֶד תִּמְשׁוֹךְ לְבָנֶיהָ

וּתְעוֹרְרָה בַּעֲלֵי בִטְחוֹן / תִּפְרוֹשׂ עֲלֵיהֶם כְּנָפֶיהָ

לַעֲשׂוֹת תְּשׁוּבָה יְהֶא פִתְרוֹן / שַׁעֲרֵי רַחֲמִים לְפָנֶיהָ

בִּינוּ זֹאת כָּל הָעַמִּים / הַאֲזִינוּ תּוֹרַת אֱלֹהֵינוּ:


בַּמֶּה יְרַצֶּה אֱוִיל מַבְאִישׁ / יוֹם דִּין וְחֶשְׁבּוֹן בְּבוֹא צוֹרֵף

וּלְזֶה יְזַכֶּה וְזֶה יַעְנִישׁ / זֶה רַךְ וְרוֹתֵת וְזֶה כּוֹפֵף

הוֹי מִשְּׂמֹאל צַעֲדַי תַּרְגִּישׁ / לִבִּי וְדַעְתִּי בְּרָע טוֹרֵף

יְוֹיָ הוֹשִׁיעָה / הַמֶּלֶךְ יַעֲנֵנוּ:


זַכַּאי יְרַחֵם לְדָל חוֹסֶה / בִּשְׁמוֹ וְלֹא יַחְמְדָה זָרִים

סוֹבֵל וְדוֹמֵם בְּיַד שׁוֹסֶה / לֹא יִגְבְּהָה רֹאשׁ וְלֹא יָרִים

לְמַעַן שְׁמָךְ יוֹצְרִי תַּעֲשֶׂה / לִפְדּוֹת בְּנֵי דַּע וּמֵישָׁרִים

וַאֲנַחְנוּ לֹא נֵדַע מַה נַּעֲשֶׂה / כִּי עָלֶיךָ עֵינֵינוּ:


יִרְצֶה אֱלֹהַי תְּפִלַּת עָם / הוֹמִים וְנָעִים בְּגָלוּתָם

צוֹפִים עֲדֵי יַאֲזִין שַׁוְעָם / דַּיָּם בְּיָקְשָׁם וְעַנְיוּתָם

הֵיךְ צָר יְאִיצֵם בְּתַעְתּוּעָם / וּבְתוֹאֲנָה יַחְמְדָה דָּתָם

הוֹשִׁיעֵנוּ אֱלֹהֵי יִשְׁעֵנוּ / קַבְּצֵנוּ וְהַצִּילֵנוּ:


מִשְׁפַּט אֱמֶת יָהּ שְׁפוֹט אוֹיְבַי / כֵּן תִּרְדְּפֵיהֶם בְּסַעֲרֶיךָ

תִּמְחֶה פְשָׁעַי וְכָל חוֹבַי / אָז יִמְצְאוּ חֵן בְּעֵינֶיךָ

כִּי בַהֲבָלַי וּבִכְזָבַי / חָטָא לְבָבִי לְפָנֶיךָ

וַתָּקֶם אֶת דְּבָרֶיךָ / וַתֵּרֶא אֶת עֳנִי אֲבוֹתֵינוּ:


שִׁית אַחֲרִית עַמְּךָ טוֹבָה / וּשְׁלַח נְקָמָה עֲלֵי פֶרֶא

וּבְאֵשׁ הֲמָסִים תְּסוֹבֵבָה / מִגְדַּל שֵׂעִיר צַר וּמִתְחָרֶה

תִּרְדּוֹף שְׁנֵיהֶם עֲדֵי חוֹבָה / תִּבְעַר בְּזָנָב וְרֹאשׁ פּוֹרֶה

יְהִי יְוֹיָ עִמָּנוּ / כַּאֲשֶׁר הָיָה עִם אֲבוֹתֵינוּ:


תִּשְׂמַח וְתָגִיל יְפֵהפִיָּה / חֶבְרַת שְׁבָטִים בְּשִׁיר תָּרוֹן

דּוֹד יוֹסְפָה חֵן עֲלֵי רַעְיָה / יִזְכּוֹר זְכוּת שׁוֹכְנֵי חֶבְרוֹן

יַשְׁחִית מְנָאֵץ כְּלוֹא הָיָה / וּלְבֶן מְלָכִים יְהֶא זִכְרוֹן

כַּאֲפִיקִים בַּנֶּגֶב / שׁוּבָה יְוֹיָ אֶת שְׁבִיתֵנוּ:


אַהְבַת עוֹלָם נִקְשֶׁרֶת / לְיַעֲקֹב וּבָנָיו נִפְלָאָה

גָּלוּת עֲלֵיהֶם מְכַפֶּרֶת / וְחוֹכִים גְּאֻלָּה הַבָּאָה

כְּעָבְרָם בְּתוֹךְ יָם בְּאַשְׁמוֹרֶת / וְאָמְרוּ כִּי גָאֹה גָּאָה

כִּי שֵׁם יְוֹיָ אֶקְרָא / הָבוּ גוֹדֶל לֵאלֹהֵינוּ:
