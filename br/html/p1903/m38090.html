משל כט: עֹרֵב וּשְׁאָר עוֹפוֹת

מִתְפָּאֵר בְּמַה שֶׁאֵין בּוֹ / רוּחַ הַשְּׁטוּת נוֹסְסָה בְקִרְבּוֹ


עֹרֵב חָשַׁב כִּי רַע יִצְרוֹ / לָמָּה חָשַׁךְ מִשְּׁחוֹר תָּאֳרוֹ / כִּי לוֹ נוֹצוֹת מְזֻפָּפוֹת / וּלְכָל עוֹף נוֹצוֹת יָפוֹת / אָדֹם לָבָן צָהֹב יָרֹק / עַל כֵּן כָּל עוֹף עָלָיו יִשְׁרֹק / בַּאֲשֶׁר הוּא מִכֻּלָּם שָׁחוֹר / עַל כֵּן נֶאֱנַח וַיָּשָׁב אָחוֹר / מִכִּלְיוֹתָיו דָּלָה עֵצָה / לָקַחַת מִכָּל עוֹף נוֹצָה / אַחַת מֵעוֹף אַךְ לֹא יוֹתֵר / לָלֶכֶת בִּמְקוֹם סֵתֶר / לְהַשְׁלִיךְ נוֹצָתוֹ שָׁמָּה / וְלָשׂוּם אַחֶרֶת בִּמְקוֹמָהּ / אֲשֶׁר יִהְיֶה מִכָּל צְבָעִים / לִהְיוֹת יָפֶה מִכָּל רֵעִים / כַּאֲשֶׁר זָמַם פָּעַל וְכִלָּה / אֶל אֵם הַדֶּרֶךְ עַל הַמְּסִלָּה / בְּחֶדְוַת לִבּוֹ מְהֵרָה יָרוּץ / וְאֶבְרוֹתָיו בִּירַקְרַק חָרוּץ / שָׁם יִתְגָּאֶה בְּכָל מִינֵי צְבָעָיו / וַיִּתְאַסְּפוּ עָלָיו רֵעָיו / גַּם עֵרֶב רַב עָלָה אִתָּם / וַיָּשִׂימוּ סְבִיבוֹ שִׁבְתָּם / וְכֻלָּם עָלָיו תָּמְהוּ / מִי הוּא זֶה וְאֵיזֶה הוּא / אֵין כָּמוֹהוּ בְּכָל הֶעָרֵב / אַךְ קוֹמָתוֹ קוֹמַת עֹרֵב / וְכַאֲשֶׁר בָּנוּ אֶת תַּבְנִיתוֹ / הִכִּיר כָּל אֶחָד נוֹצָתוֹ / זֶה אוֹמֵר: זֹאת נוֹצָתִי / וְזֶה אוֹמֵר: עָכַרְתָּ אוֹתִי / זֶה אֶת נוֹצָתוֹ מֹרֵט / וְזֶה נוֹתֵן בִּבְשָׂרוֹ שָׂרֶט / וְנִשְׁאַר מְמֻשָּׁךְ וּמוֹרָט / כִּי לְנֶגְדּוֹ דַרְכּוֹ יָרַט.


וְהַמָּשָׁל בְּאִישׁ מִתְקַשֵּׁט בְּמַה שֶּׁאֵין בּוֹ / וְאִם חָכְמָה שָׁמַע יֹאמַר כִּי בָּדָה מִלִּבּוֹ / וּבְעֵת יִשְׁאֲלוּ לוֹ עֵצָה יִדֹּם כָּאֶבֶן / וְיֵשׁ אֲשֶׁר יִתֵּן בִּמְקוֹם בָּר תֶּבֶן / וּמֵחָכְמָתוֹ יֵלֵךְ שְׁחוֹחַ / תַּחַת חִטָּה יֵצֵא חוֹחַ / וְטוֹב לִהְיוֹת אֱוִיל מַחֲרִישׁ / מִלִּהְיוֹת בְּדַעְתּוֹ חָכָם מַשְׁרִישׁ / וְאִם פִּיו יִפְתַּח יְגַלֶּה שְׁטוּתוֹ / וִישַׂחֲקוּ הַכֹּל עַל עֲצָתוֹ / וְהַרְבֵּה כִּבּוּ נֵר הוֹדָם / וַיָּסֹלוּ אָרְחוֹת אֵידָם.
