משל פח: פֶּרֶס וָסִיר

טוּב טַעַם וְדַעַת בְּכִלְיוֹתֶיךָ / תְּבָאֵמוֹ וְתִטָּעֵמוֹ בְּהַר נַחֲלָתֶךָ


הַפֶּרֶס יָבֵשׁ גְּרוֹנוֹ מִצִּמְאָה / וּלְשׁוֹנוֹ בַצָּמָא נָשְׁתָה וְנִלְאָה / לְבַקֵּשׁ נְהָרִים יְאֹרִים רַחֲבֵי יָדַיִם / וַיֵּלֶךְ שְׁלֹשֶׁת יָמִים בַּמִּדְבָּר וְלֹא מָצָא מַיִם / וּבַיּוֹם הָרְבִיעִי מָצָא סִיר וּמַיִם בּוֹ / וְלֹא יָדַע אִם חֶצְיוֹ אוֹ רֻבּוֹ וְהַסִּיר עָמֹק וְאָרֹךְ / וְהַפֶּרֶס לִשְׁתּוֹת עָרוּךְ / וְלֹא יָכֹל עַד הַמַּיִם לְהַשְׁפִּיל רֹאשׁוֹ / וַיִּכָּנַע בַּעֲמָלוֹ וְכִפְלַיִם שֹׁקְקָה נַפְשׁוֹ / וַיֵּרֶד לְהָדְפוֹ מִמַּצָּבוֹ וְלַהֲפֹךְ אֶת הַכְּלִי וְאֶת הַמַּיִם לִשְׁפֹּךְ / וַיֵּלֶךְ סְבִיבָיו בְּחֶתֶף / אוּלַי יוּכַל לוֹ בְּצַד או בְכָתֵף / וּלְהַפִּילוֹ בְּפִיו וּבְרַגְלָיו / וַיַּרְא כִּי לֹא יָכֹל אֵלָיו / וַיְלַקֵּט אֲבָנִים וּקְטַנּוֹת הָיוּ רֻבָּם / מַסִּיעַ אֲבָנִים יֵעָצֵב בָּם / וּלְהַשְׁלִיכָם בַּסִּיר יוֹרוּהוּ מְזִמּוֹתָיו / וַיָּרוּמוּ הַמַּיִם וַיֵּלְכוּ עַל כָּל גְּדוֹתָיו / וַיִּשְׁבֹּר צְמָאוֹ וַיִּהְיוּ לוֹ לִתְרוּפָה / מַיִם קָרִים עַל נֶפֶשׁ עֲיֵפָה.


וְהַמָּשָׁל לִקְנוֹת חָכְמוֹת וּמְזִמּוֹת / וְתַחְבֻּלוֹת הַתְּבוּנָה וְעָרְמוֹת / כִּי יֵשׁ רְפֵה מָתְנַיִם וּמְזִמָּה בְלִבּוֹ קְשׁוּרָה / טוֹבָה חָכְמָה מִגְּבוּרָה / עַל כֵּן מַדָּע וְשֵׂכֶל תַּשְׁרִישׁ בִּמְזִמּוֹתֶיךָ / תְּבָאֵמוֹ וְתִטָּעֵמוֹ בְּהַר נַחֲלָתֶךָ / וּמִזֵּרְעוֹנֵי בִין תִּזְרְעֵהוּ / וּמִמַּעְיַן הַשֵּׂכֶל תַּשְׁקֵהוּ / וְאַל תָּסוּר אַחֲרֵי אִוֶּלֶת / פֶּן תְּהִי נַפְשְׁךָ בָם מוֹרְאָה וְנִגְאֶלֶת / וְאָמַר הֶחָכָם: הַמִּתְעַסֵּק בְּחָכְמָה לֹא יִכָּלֵם / אִם אֵין לוֹ צַוָּאר עָב וְגוּף שָׁלֵם / וַאֲשֶׁר שָׁת אֶל דְּבָרַי לִבּוֹ / מְצוּדוֹת סְלָעִים מִשְׂגַּבּוֹ / יִרְאוּ חֲכָמִים מְשָׁלַי וְעוֹד יֶחְכָּמוּ / וְשָׁמְעוּ אֲמָרַי כִּי נָעֵמוּ.
