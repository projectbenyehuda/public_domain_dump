משל פו: תַּרְנְגוֹלֶת וּגְבִרְתָּהּ

הַמְסִיתְךָ בְּחֵלֶק שְׂפָתָיו / אַל תִּתְאַו לְמַטְעַמּוֹתָיו


תַּרְנְגוֹלֶת אֹכְלָה מְשֹׁטֶטֶת / וּבֵין רַגְלֶיהָ מְלַקֶּטֶת / שִׁבֳּלִים מְפָרֶדֶת וּמְנַתֶּקֶת / וְאֶת רַגְלֶיהָ מְפַשֶּׂקֶת / לְגַלּוֹת הֶעָפָר בְּצִפֹּרֶן / וּמֹץ יְסֹעֵר מִגֹּרֶן / כַּמָּה פְעָמִים לָרִיק יָגְעָה / כִּי לֹא מָצְאָה דֵּי שָׂבְעָהּ / וַתַּבֵּט אֵלֶיהָ גְּבִרְתָּהּ / וַתֵּרֶא כִּי לִבָּהּ נִפְתָּה / לְהַחֲיוֹת נַפְשָׁהּ בַּעֲמָלָהּ / נִכְמְרוּ רַחֲמֶיהָ וַתֹּאמֶר לָהּ / הִנֵּה אוֹתָךְ רִבִּיתִי וְטִפַּחְתִּי / עַל כֵּן עָלַיִךְ חָמַלְתִּי / בַּאֲשֶׁר דְּגָנֵךְ מְפֻזָּר וּמְפֹרָד / אֲשֶׁר הִשְׁאִיר הַבָּרָד / וַתִּקְצַר נַפְשֵׁךְ לְבַקֵּשׁ אֶתְהֶם / עָבְשׁוּ פְרֻדּוֹת תַּחַת מֶגְרְפֹתֵיהֶם / עַל כֵּן אַתְּ רָזָה וַתֵּרְדִי פְלָאִים / כִּמְלַקֵּט שִׁבֳּלִים בְּעֵמֶק רְפָאִים / וַתִּרְבְּצִי בְעֵמֶק עָכוֹר / הֲלֹא טוֹב לָךְ לֶתֶךְ חִטִּים וָכוֹר / וְיָשַׁבְתְּ בְּפֶתַח תִּקְוָה / מִהְיוֹתֵךְ כָּל הַיּוֹם דָּוָה / הִנֵּה אֶתֵּן לְפָנַיִךְ כָּל דְּגָנִי / מְדֻשָּׁתִי וּבֶן-גָּרְנִי / שָׁוֶה מֵאָה כֶסֶף עֹבֵר לַסֹּחֵר / אַל תֵּלְכִי לִלְקֹט בְּשָׂדֶה אַחֵר / הִנֵּה גִלִּית הֶעָפָר וְכִסִּית / אֵיפֹה לִקַּטְתְּ הַיּוֹם וְאָנָה עָשִׂית / יָגַעַתְּ וְלֹא מָצָאת אֲשֶׁר עָמַלְתְּ בּוֹ / לִפְנֵי לַחְמֵךְ אַנְחָתֵךְ תָבֹא / וּכְדַבֵּר הַגְּבֶרֶת זֹאת בְּאָזְנֶיהָ / וַתֵּקַל גְּבִרְתָּהּ בְּעֵינֶיהָ / וַתֹּאמֶר לָהּ: אִם תִּתְּנִי לִי בְּכָל חֹדֶשׁ חִטִּים / מְלֹא גָרְנֵךְ כְּתוּשִׁים וַחֲבוּטִים / יֵחָשֶׁב לִי לֶחֶם עַצְלוּת / טוֹב לִי לִסְבֹּל הַדַּלּוּת / וְעוֹלַלְתִּי בֶעָפָר קַרְנִי / מֵאֲשֶׁר אַכְזָר מִן הַגֹּרֶן יִרְדְּפֵנִי / כִּי אֲשֶׁר שָׂם עַל אַחֵר תִּקְוָתוֹ / יִגֶל יְבוּל בֵּיתוֹ / וְטוֹב לִי לָצֵאת חָפְשִׁי / אֲבַקְשָׁה אֶת שֶׁאַהֲבָה נַפְשִׁי / לֶאֱכֹל אֶת אֲשֶׁר אֶמְצָא / טוֹב לִי לָלֶכֶת חוּצָה / וְלִתְעוֹת כְּשֶׂה אֹבֵד / מְתוּקָה שְׁנַת הָעֹבֵד / וְהַשִּׁבֹּלֶת אֲשֶׁר מִכָּל אִישׁ תִּנָּקֶה / עַל שָׁרְשֵׁי רַגְלַי תִּתְחַקֶּה / שֶׁלְּךָ שֶׁלְּךָ, שֶׁלִּי שֶׁלִּי / כִּי זֶה חֶלְקִי מִכָּל עֲמָלִי / וְאֶדַּדֶּה בַּעֲמָלִי כָּל שְׁנוֹתַי / כִּי לֹא טוֹב אָנֹכִי מֵאֲבוֹתַי.

וְהַמָּשָׁל שְׁמַע וְהָבֵן / הָאֹמֵר לְךָ: קַח בָּר בִּמְקוֹם תֶּבֶן / אַל תִּבֶז אֶת אֲשֶׁר יָדְךָ מָצְאָה / כִּי לֹא תֵדַע אֶת הַתְּלָאָה / אֲשֶׁר תִּמְצָא בְּשָׁמְעֲךָ בְקוֹלוֹ / בָּז לְדָבָר יֵחָבֶל לוֹ / חֲשֹׁב דִּבְרֵי הַמְפַתִּים לְאַיִן / אַל תִּלְחַם אֶת לֶחֶם רַע עָיִן / וְהַמְסִיתְךָ בְּחֵלֶק שְׂפָתָיו / אַל תִּתְאַו לְמַטְעַמּוֹתָיו / כִּי בְּטֶרֶם תִּשְׂבַּע יִשְׂבָּעֲךָ / וּמָחָר מֵחֲבוּרָתוֹ יְגִיעֲךָ / אִישׁ חָמָס יְפַתֶּה אֶת רֵעֵהוּ / וּבְשֵׁבֶט פִּיו יַכֵּהוּ / וְגַם אִם כָּרַת עִמּוֹ הַבְּרִית / יְזֹרֶה עַל נָוֵהוּ גָפְרִית / דֶּרֶךְ לֵץ לֶאֱהֹב לָצוֹן / וְשֹׁחֵר טוֹב יְבַקֵּשׁ רָצוֹן.
