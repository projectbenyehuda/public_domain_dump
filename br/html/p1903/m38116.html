משל נה: דָּג קָטָן בִּמְצוּדָה

אַל תַּשְׁלֵךְ מַה שֶּׁבְּיָדֶךָ / לְהַעֲלוֹת זָרִים בִּמְצוּדֶךָ


דָּג קָטָן נֶאֱחַז בִּמְצוּדָה / וַיֹּאמֶר לַדַּיָּג: קָטֹנְתִּי מִהְיוֹת צֵדָה / כִּי תְשִׂימֵנִי בְסִיר דּוּגָה / הָשַׁע מִמֶּנִּי וְאַבְלִיגָה / וְלִבְּךָ מִמֶּנִּי יִתְיָאֵשׁ / לֹא מְבֻשָּׁל וְלֹא צְלִי אֵשׁ / אֶהְיֶה לְךָ לְמֵשִׁיב נָפֶשׁ / אַךְ הַרְפֵּנִי וְיִנָּפֵשׁ / עַבְדְּךָ שְׁנָתַיִם יָמִים / וְאָז תִּמְצָאֵנִי בְנַחַל קְדוּמִים / גְּדָל־קוֹמָה וְדָשֵׁן שִׁבְעָתַיִם / וּבַשֵׁל מְבֻשָּׁל בַּמָּיִם / אֶהְיֶה בְּבֵיתְךָ כְּיוֹם מוֹעֵד / וּלְבָבְךָ אָז תְּהִי בִי סוֹעֵד / וַיַּעַן הַדַּיָּג: טוֹב לִי דָּג קָטָן אֲשֶׁר עַתָּה בֵין חָפְנַי / מִלִּוְיָתָן גָּדוֹל אֲשֶׁר יִרְדּוּ עַד שָׁנָה שְׁכֵנָי.

וְהַמָּשָׁל נוֹדַע בְּכָל קִרְיָה / וְהַפִּתְרוֹן שָׁגוּר בְּפִי כָל בִּרְיָה / כִּי טוֹב כַף נַחַת בְּיָדֶיךָ / מִמְּלֹא חָפְנַיִם תִּקְוָה בְיַד זוּלָתֶךָ / טוֹב לִי עוֹף סָגוּר בַּחֶדֶר / מִשְּׁנַיִם קוֹפְצִים עַל הַגָּדֵר / גַּם אֶת הַטּוֹב תְּקַבֵּל, וְאִם מְעָט הוּא / שִׂים יָדְךָ עָלָיו תֹּאחֲזֵהוּ וְלֹא תַרְפֵּהוּ / וְאַל תִּבְחַר בְּתוֹחֶלֶת מְמֻשָּׁכָה / לְהַרְבּוֹת עַל הַסָּפֵק טוֹבָה וּבְרָכָה.
