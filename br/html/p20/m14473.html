אֶל אֵל גָּדוֹל וּמְהֻלָּל

אֶל אֵל גָּדוֹל וּמְהֻלָּל / אֶצְעַק מֵעָנְיִי וְצִירִי,

עַד יִבְנֶה בֵית הוֹד וְחֶמְדָּה — / אָז יָשׁוּב מַלְכִּי וְצִירִי.


בָּטַחְתִּי, חַנּוּן, בְּשִׁמְךָ, / כִּי אֵין לִי בִּלְתָּךְ לְמַחְסֶה.

עָנְיִי שׁוּרָה מִמְּרוֹמְךָ, / כִּי כָל יוֹם עַבְדָּךְ מְנֻסֶּה,

וַעְוֹנִי נֶגְדָּךְ בְּשׂוּמְךָ / עֵד יִהְיֶה יוֹם דִּין וּמַעְשֶׂה.

הַבֵּט אֶל דַכָּא וְאֻמְלָל / מַלְכִּי וּמְנוּסִי וְצוּרִי,

אַל תִּגְמֹל מִדָּה בְמִדָּה, / פֶּן אִסָּפֶה עִם שְׁאָרִי.


רַחוּם, אַתְּ הַכֹּל מְחוֹלֵל, / הַבֵּט לִבְרִית נֶאֱמָנָי!

עַד מָתַי יִהְיֶה מְחַלֵּל / הַצַּר אֶת שִׁמְךָ לְעֵינָי?

לֹא אוּכַל נֶגְדּוֹ לְמַלֵּל / פֶּן יָקוּמוּ בִי מְעַנָּי.

נִשְׁאַרְתִּי עָרוֹם וְשׁוֹלָל, / נִפְזַרְתִּי מִבֵּית מְגוּרִי,

נֶחְשַׁבְתִּי לָהֶם לְנִדָּה / אַחַר נִיחוֹחִי וּמוֹרִי.


הַאֻמְנָם יָסִיר אֱלֹהִים / אֶת חֶרְפַּת עַמּוֹ בְחִירוֹ?

יוֹפִיעַ מֵרוּם גְּבוֹהִים / לִגְמֹל טוֹב לִבְנוֹ בְכוֹרוֹ,

וִימַהֵר לִפְדּוֹת כְּמֵהִים / וִישִׁיבֵם אֶל תּוֹךְ הֲדָרוֹ!

אָז יָשִׁישׁ יָשִׁישׁ וְעוֹלָל / עֵת תִּקְרָא שְׁנַת דְּרוֹרִי,

וּלְבֵיתִי קָנֶה וְקִדָּה / יָשִׁיב, עִם רֹקַח דְּרוֹרִי.


מָרוֹם, פָּנֶיךָ מְחַלִּים / כָּל יוֹם סוֹבְלֵי הַתְּלָאוֹת,

עַד מָתַי יִהְיוּ מְיַחְלִים / קֵץ תַּנְחוּמֵי הַנְּבוּאוֹת?

מָתַי יַעַבְרוּ גְאוּלִים / עֵת בּוֹא הַמּוֹפֵת וְהָאוֹת?

תַּקְדִּישׁ שִׁמְךָ הַמְּחֻלָּל / וּתְשׁוֹבֵב אֶל תּוֹךְ דְּבִירִי,

עִם שׁוּב בַּת מֶלֶךְ כְּבוּדָה — / יַהַלְכוּ גוֹיִם לְאוֹרִי.
