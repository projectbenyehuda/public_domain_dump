[שָׁמַיִם מְסַפְּרִים כְּבוֹד אֵל]

שָׁמַיִם מְסַפְּרִים כְּבוֹד אֵל,

אֶרֶץ תַּעֲנֶה: אֵין כָּאֵל!


אוֹתוֹת בְּכָל בְּרוּאִים / פָּגְעוּ פָנַי,

כִּי כֹה וְכֹה פְלָאִים / רָאוּ עֵינַי,

גַּם מַחֲזוֹת נְבִיאִים / שָׁמְעוּ אָזְנַי; / סוֹד שַׂר צְבָא אֲדוֹנָי

מִיכָאֵל שְׁמוֹ, וְשַׂר עֲנָאֵל

וּרְפָאֵל, וְאִישׁ גַּבְרִיאֵל.


בָּחַר בְּאַב הֲמוֹנוֹ, / בָּחַן לִבּוֹ,

וַיֶּחֲזֶה בְעֵינוֹ / הַר חָשַׁק בּוֹ,

בָּא בַחֲלוֹם לְנִינוֹ, / בָּא אַחַר בּוֹא / שֶׁמֶשׁ, וְשָׁב בְּשׁוּבוֹ

שֵׁנִית אַחֲרֵי עָבַר פְּנוּאֵל,

קָרָא אֶת שְׁמוֹ: אֵל בֵּית אֵל.


רָצוֹת עֲלֵי שְׁחָקִים / חַיּוֹת רָצוֹא

וָשׁוֹב, דְּמוּת בְּרָקִים, / לַעֲשׂוֹת חֶפְֱצוֹ.

יָעִיר שְּכַן עֲמָקִים / עַד יַעֲרִיצוֹ, כִּי הֶעֱמִיד בְּאַרְצוֹ

הַכָּבוֹד בְּתוֹךְ מִקְדַּשׁ אֲרִיאֵל,

שָׁם יָשׁוּב בְּבוֹא הַגּוֹאֵל.


מָה רַב גְּדוּד מְרוֹמוֹ! / עֲדֵי גָּדְלוֹ

הֵמָּה, וּמִמְּקוֹמוֹ / יִשְׁתַּחְוֶה לוֹ

עַמּוֹ, וּמִמְּרוֹמוֹ / יִתֵּן קוֹלוֹ / שָׂרָף בְּכֵס זְבוּלוֹ;

קוֹל ‘קָדוֹשׁ’ כְּקוֹל הָמוֹן בְּשֵׁם אֵל,

כֵּן עָשׂוּ בְּנֵי יִשְׂרָאֵל.
