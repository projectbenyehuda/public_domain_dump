[לְאַט עַל מְתִים עָבְדוּ]

לְאַט עַל מְתִים עָבְדוּ / וְעַל פִּתְחֲךָ שָׁקְדוּ,

אֱלֹהַי, פְּתַח אָזְנְךָ / לְצַעְקִי, וְשִׂים עֵינְךָ

בְּעִירְךָ וּבַהְמוֹנְךָ, / וְחוֹנֵן שְׁאָר צֹאנְךָ,

אֲשֶׁר בֵּין אֱדוֹם נִפְרְדוּ / לְחַדְרֵי תְהוֹם יָרְדוּ.

בְּחוּנִים בְּכוּר נִבְחֲנוּ / וְהוֹמִים בְּאוֹן נִצְפְּנוּ

אֲסִירִים, וְקוֹל נָתְנוּ / בְּבֵיתָךְ וְשָׁם יַחֲנוּ –

פְּדֵם, כִּי שְׁמָךְ יִחֲדוּ / וּמַהְלָלְךָ יִסְּדוּ.

רְעֵה עַמְּךָ גַּם רְצֵה, / אֱלֹהִים, וּמִצַּר פְּצֵה,

וְשִׂימָה לְקִצִּי קְצֵה, / חֲגוֹר חַרְבְּךָ, צוּר, וְצֵא!

לְחַם לוֹחֲמִים נוֹעֲדוּ / וְעַל עַמְּךָ נוֹסְדוּ!

מְיַחֲלִים – עֲנֵם, יוֹצְרִי, / וְעַל מָה בְּאַף וָחֳרִי

נְתַתָּם בְּפִי הָאֲרִי? / מְחַצְתָּם, וְחָדַל צֳרִי –

וּבֵית קָדְשְׁךָ לָכְדוּ / מְשַׂנְּאִים, וְלֹא פָחֲדוּ.
