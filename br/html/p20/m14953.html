אֱלֹהַי הוֹדִיעֵנִי אֶת אֹרַח הַחַיִּים

אֱלֹהַי הוֹדִיעֵנִי / אֶת אֹרַח הַחַיִּים;

אַתָּה מַחְסִי, חֶלְקִי / בְּאֶרֶץ הַחַיִּים.


כָּל הַזְּמַנִּים הֶבֶל / בִּלְעֲדֵי עֵת רְצוֹנֶךָ

וְכָל הַמְּקוֹמוֹת תֹּהוּ – / בִּלְתִּי בֵית מְעוֹנֶךָ

וְכָל הַמַּעֲשִׂים שָׁוְא / לְבַד עֲשׂוֹת הַטּוֹב בְּעֵינֶיךָ –

לַעֲשׂוֹת כָּל הַיָּמִים / אֲשֶׁר הֵם חַיִּים.


מָה אֶעֱשֶׂה וְאָן אֶפְנֶה – / וְהִנֵּה חֶטְאִי קוֹרְאִי,

וְאַתָּה גֵרַשְׁתָּנִי / וְיַהַרְגֵנִי מוֹצְאִי?

תְּפִלָּתִי עָשָׁנְתָּ / וְאֵין יִתְרוֹן לִי בְקָרְאִי,

וְאִם לֹא אַתָּה חֶלְקִי – / לָמָּה לִּי חַיִּים?


אִם אַתָּה הֲדַפְתָּנִי – / דָּפַקְתִּי שְׁעָרֶיךָ,

וְאִם אַתְּ הִשְׁלַכְתָּנִי– / בָּחַרְתִּי חֲצֵרֶיךָ,

וְאִם קוֹדֵר הִלָּכְתִּי – / נֵר לְרַגְלִי דְבָרֶךָ,

בְּאוֹרְךָ, יָהּ, נִרְאֶה אוֹר, / כִּי עִמְּךָ מְקוֹר חַיִּים.


אֶחְיֶה אִם תְּחַיֵּנִי, / אֱלֹהַי, בִּרְצוֹנֶךָ

וְאָמוּת אִם תְּמִיתֵנִי / בְּקִצְפְּךָ וַחְרוֹנֶךָ.

גַּם אֶת הַטּוֹב וְהָרָע / נְקַבֵּל מִלְּפָנֶיךָ –

וְנוֹחִיל חֲסָדֶיךָ, כִּי / טוֹב חַסְדְּךָ מֵחַיִּים.


בְּמֵימֵי סְלִיחוֹתֶיךָ / אֶת כָּל אַשְׁמוֹתַי מְחַה,

וּלְפִי רֹב חֲסָדֶיךָ / אֶת כָּל יְגוֹנוֹתַי דְּחֵה,

וְעַל מֵי מְנוּחוֹתֶיךָ / נַהֵל נַפְשִׁי וְהַנְחֵה

לְהִתְהַלֵּךְ לְפָנֶיךָ, / לֵאוֹר בְּאוֹר הַחַיִּים.


רָאִיתִי כָּל עָמָל – / וְהִנֵּה הַכָּל הֶבֶל,

אַךְ עֲבוֹדַת מַלְכוּתְךָ / לְבַד חֶלְקִי מִתֵּבֵל,

וְעֵת אֶמְצָא רְצוֹנֶךָ / אָשִׁיר בְּכִנוֹר וְנֶבֶל

וְאֹמַר: אַשְׁרֵי, וְאַחְלַי / מֹצְאִי מָצָא חַיִּים.
