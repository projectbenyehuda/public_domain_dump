סְגֻלַּת אֵל מְתֵי חָכְמָה וְדֵעָה

סְגֻלַּת אֵל מְתֵי חָכְמָה וְדֵעָה / לְכוּ אֶל אוֹר מְנוֹרָה הַקְּבוּעָה

כְּלוּלָה מֵאֲבָנִים הַיְּקָרוֹת / וְלָהּ נֵרוֹת אֱמֶת שִׁבְעָה וְשִׁבְעָה

וּמִלָּא כָל פְּנֵי תֵבֵל מְאוֹרָהּ / וְהוֹד זִיוָהּ בְּשׁוּלֵי הַיְּרִיעָה

אֲשֶׁר תִּקֵּן בְּנוֹ מַיְמוּן וְחִבַּר / בְּעֵין שִׂכְלוֹ בְּתַכְלִית הַיְדִיעָה

וְהוּא עָבַר בְּצִי אַדִּיר וְהִפְלִיג / בְּיַם תַּלְמוּד וְדָלָה כָל שְׁמוּעָה

וְלוּ אֶהְיֶה נְגִיד כָּל הַנְּגִידִים / אֲהֶא כָּפוּף בְּקִידָה עִם כְּרִיעָה

בְּעֵת אֶשְׁמַע דְּבָרָיו הַנְּעִימִים / וְהֵם מַרְפֵּא לְנַפְשִׁי הַנְּגוּעָה

וְאֶשְׁתַּעְשַׁע בְּעִיּוּנָם וְאָגִיל / וְאֶשְׂמַח בָּם כְּמוֹ לֶאֱכֹל לְשָׂבְעָה

וְעַל אוֹר זֶה הֲלֹא יִרְבֶּה שְׂשׂוֹנִי / בְּבוֹר גָּלוּת וְלֹא אֶבְכֶּה בְּדִמְעָה

וְלִכְבוֹדוֹ אֱהִי קוֹרֵא בְּגָרוֹן / לְכָל מַשְׂכִּיל בְּכָל רֶגַע וְשָׁעָה

שְׁאַל מָתוֹק לְחִכֶּךָ בְּסִפְרוֹ / וְאַל תִּשְׁאַל לְכִדְבוֹרָה וְצִרְעָה

וְאַל תִּדְאַג לְקוֹרוֹת הַזְּמַנִּים / וְאִם תֵּלֵךְ עֲלֵי כָל הָר וְגִבְעָה

תְּהֶא שׁוֹנֶה פְּרָקָיו עַל לְבָבָךְ / וּבָם תִּמְצָא לְנַפְשֶׁךָ תְּשׁוּעָה

חֲדַל אָחִי לְהִתְרַפּוֹת בְּשִׂיחִי / חֲדַל אָחִי חֲדַל מִן הַפְּשִׁיעָה

אֱזָר נָא אֶת חֲלָצֶיךָ כְּגֶבֶר / וְאַל תִּירָא לְקֹצֶר הַפְּסִיעָה

וְהִתְבּוֹנֵן בְּבָתֵי הַתְּבוּנוֹת / וְנִשְׁמָתָךְ תְּחַיֶּה מִגְּוִיעָה

בְּנֵי יַעֲקֹב אֱמֶת דִּרְשׁוּ בְּכָל עֵת / וְתָרִיעוּ עֲלֵי הַצַּר תְּרוּעָה

וְאֵל יִשְׁמַע לְשַׁוְעַתְכֶם מְהֵרָה / וְיִזְכֹּר עֵת לְהָקִים הַשְּׁבוּעָה

וְתִזְכּוּ לַחֲזוֹת מִקְדָּשׁ וְתִרְאוּ / מְבַשֵּׂר טוֹב וּמַשְׁמִיעַ יְשׁוּעָה.
