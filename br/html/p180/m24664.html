שְׁזוּפָה נְזוּפָה

שְׁזוּפָה נְזוּפָה / בְּשִׁבֹּלֶת שְׁטוּפָה

מַה לָּךְ כְּפוּפָה / בְּלִי קוֹמָה זְקוּפָה

תֵּשֵׁב עֲטוּפָה / כְּשׁוֹשַׁנָּה קְטוּפָה

בְּכָל עֵת אֲשֶׁר תָּגוּר / כְּלֵי מָוֶת חָגוּר

תְּצַפְצֵף כֶּעָגוּר / בְּתוֹךְ מַסְגֵּר סָגוּר

הֱיֵה לָהּ צוּר צִנָּה / מְאוּסָה תִהְיֶה לְרֹאשׁ פִּנָּה.


לִפְנֵי מְשַׂנְאִי / אֲנִי תָמִיד נִכְנָע

שַׁאֲנָן וְרַעֲנָן / וַאֲנִי נָד וָנָע

יִלְעַג וְיִשְׁאַג / וְיָדוֹ לֹא יִמְנַע

אֲקַוֶּה לְעֵת מַרְפֵּא / וְאוֹחִיל וַאֲצַפֶּה

וְאָשִׂים יָד עַל פֶּה / בְּלֵב נִכְפֶּה וְנִסְפֶּה

כְּחַנָּה מוּל פְּנִנָּה.


מַה לְּךָ דְּוֵה לֵב / בְּמַר לֵבָב יַעֲטֹף

אַפָּיו וְכַפָּיו / בְּדִמְעָתוֹ יִשְׁטֹף

זְרַע לְךָ צְדָקָה / כִּי חֶסֶד תִּקְטֹף

הֲלֹא עַוָּל יִמַּס / וְגַם יִהְיֶה לְמִרְמָס

וְחֵץ שָׁחוּט נִכְמָס / לְכָל אוֹהֵב חָמָס

לְזֶרַע מִי מָנָה.


הָלְכוּ שְׁנוֹתַי / בְּיָגוֹן וַאֲנָחָה

אֲקַוֶּה רְוָחָה / וְהִנֵּה קוֹל צְוָחָה

בְּטֶרֶם צְרִי בָא / נַפְשִׁי נִלְקָחָה

דְּוֵה לֵבָב נֶאֱנָח / קַוֵּה וְאַל תַּנַּח

לְאֵל כִּי לֹא יִזְנַח / וּמֵאוֹיֵב תְּהִי נָח

וְתִפְרַח כְּמוֹ גַנָּהּ.


חָלוּ וְכָלוּ / עֵינַי מִיַּחֵל

זַעְפְּךָ אֶשָּׂא / וְנֶאֱלַמְתִּי כְּרָחֵל

אוּלַי מְצוֹקִי / יְהִי לְפִשְׁעִי מוֹחֵל

בְּחַסְדְּךָ אַל תְּדַכֶּה / נָגוּעַ וּמֻכֶּה

בְּלֵב נִשְׁבָּר וְנִדְכֶּה / לַיִל וְיוֹם יְחַכֶּה

לְיֶשְׁעֲךָ אֵל קַנָּא.


קְשׁוּרָה בְיַד צַר / כְּצִפּוֹר בְּכַף תִּינוֹק

יִדְחַק וְיִשְׂחַק / וְהִיא תֶּאֱנֹק אָנוֹק

שִׁמְעִי נָא עֲנִיָּה / יוֹשֶׁבֶת צִינוֹק

צַפִּי לִי וְחַכִּי / מְהֵרָה אָנֹכִי

אֶשְׁלַח מַלְאָכִי / לְפַנּוֹת אֶת דַּרְכִּי

בָּעֶרֶב בֶּכִי וְלַבֹּקֶר רִנָּה.
