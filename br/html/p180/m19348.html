[עֵינַי מֵרֹב בְּכִי כָּלוּ]

וממّא רתֹאה בה איצֹא [ומן הקינות שבהן קונן עליו עוד (על אביו]


עֵינַי מֵרֹב בְּכִי כָּלוּ / וּקְרָבַי מִכְּאֵב חָלוּ

כִּמְעַט כָּסּוּ בְאַנְחָתִי / תּוּגוֹתַי בָּם וְהָצְלָלוּ

דַּי כִּי־לִבִּי וְצַלְעֹתַי / כָּלוּ מֵהֶם וְלֹא־כָלוּ

אַחַי נָפְלוּ פְּנֵי נַפְשִׁי / יוֹם שֶׁבּוֹ אֹהֲבַי נָפְלוּ

אֶבְכֶּה אָבִי אֲשֶׁר־מֵת אוֹ / אֶבְכֶּה אַחַי אֲשֶׁר גָּלוּ

מָסָךְ נָפַל וְעַמּוּדָיו / נָפְלוּ לִשְׁאוֹל וְהָרְעָלוּ

אָחַז רַעַד בְּמוֹ מָתְנַי / אָחְזוּ צִירִים וְנִבְהָלוּ

עֵת כִּי־רָאָה אֱנוֹשׁ הוֹרִי / עָמְדוּ עָלָיו וְשָׁאָלוּ

נָהָר יֶחְרַב וְיִבָשׁוּ / מֵימֵי הַיָּם וְאָזָלוּ

אֵיכָה חָרְדוּ מְתֵי לֵבָב / וּלְפִי שִׂכְלָם יְהֻלָּלוּ

רָאוּ כִי מֵאֲנָה נַפְשִׁי / הִנָּחֵם יֹֽדְעַי חָדְלוּ

לִבִּי תֹעֶה וְשַׁעְשׁוּעֵי / תַנְחוּמַי אַחֲרָיו חֻלּוּ

גָּדַל כַּיָּם כְּאֵבִי עַד / רָאוּ עֵינַי וְנָזָלוּ

אוֹי כִּי זֶה הַכְּאֵב נִלְחַם / עִם־דִּמְעוֹתַי וְנֶחְשָׁלוּ

שִׁבְרִי לוּ יָדְעוּ בָנַי / בָּכוּ עָלַי וְאָבָלוּ

לוּ הִשְׂכִּילוּ עֲצֵי־יַעַר / יָבֵשׁ פִּרְיָם וְלֹא־עָלוּ

אַף לוּ חָכְמוּ מְאוֹרֵי אוֹר / קָדְרוּ עָלָיו וְאָפָלוּ

קַח מָוֶת כָּל־לְבָבִי הֵן / כִּי מַה־יּוֹעִיל אֲמָרְךָ לוּ

אַיֵּה שָׂרִים אֲשֶׁר שָׁלְטוּ / עַל־הָאָרֶץ וּמָשָׁלוּ

אַף כֻּלָּם נִפְרְשָׂה לָהֶם / רֶשֶׁת מָוֶת וְנִכְשָׁלוּ

פָּדוּ נַפְשָׁם בְּהוֹנָם לוּ / לָקַח פִּדְיוֹם וְנִגְאָלוּ

עָמְדָה עַל־זֹאת וְהִתְבּוֹנֵן / בִּמְשַׁל שָׂרִים אֲשֶׁר בָּלוּ

כִּי אָדָם כַּחֲצִיר שָׂדֶה / יָבֹא קִצָּם וְיִמָּלוּ.
