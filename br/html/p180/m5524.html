[בְּשָׂרִי דַל]

וקאל איצ ͘ א


È — — — / È — — — / È — — / / È — — — /È — — — / È — —


בְּשָׂרִי דַל מְאֹד מֵרֹב דְּאָגָה / וְאִם-אֵשֵׁב בְּעֵין יָשֵׁן הֲיִזַּח

וְחֹתָמִי אֲשֶׁר הָיָה בְיָדִי / אֲנִי חֹגֵר לְמַחְגֹּרֶת וּמֵזַח.
