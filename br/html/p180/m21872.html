כּוֹכָב אֲשֶׁר דָּרַךְ

כּוֹכָב אֲשֶׁר דָּרַךְ בְּמַשְׁקוֹפֵנוּ / הֶחְשִׁיךְ בְּאוֹרוֹ כּוֹכְבֵי נִשְׁפֵּנוּ

עָבַר כְּבָרָק בַּחֲלוֹם וַיֶּאֱחֹז / הַלֵּב וְלֹא נֶאְחַז לְעַפְעַפֵּינוּ

גָּלָה תְּמוֹל מִפִּיו שְׂחוֹק וָאֹמְרָה: / שָׁלוֹם לְךָ! וַיַּעֲנֶה: ״הַרְפֵּנוּ!

אֵיךְ תַּחֲשֹׁב לִתְפֹּשׂ מְאוֹרִי, שׁוּב וּרְאֵה / אִם נִתְפְּשָׂה שֶׁמֶשׁ בְּמוֹ כַפֵּנוּ?״

אָמִיר הֲדַס רוּחַ הֲדָפוֹ עַד אֲשֶׁר / נָדַד, וְלָמָּה תַחֲשֹׁב הָדְפֵנוּ?

דַּע כִּי אֲנַחְנוּ עַם בְּרוּחָם נִפְרְקוּ / הָרִים וְכָלוּ בַּחֲרוֹת אַפֵּנוּ

טֶרֶם רְאוֹת תֵּבֵל יְדַעְנוּהָ וְגַם / דִּבֵּר בְּלִי לָשׁוֹן, אֱמֶת, טַפֵּנוּ

לוּ הַזְּמַן יָבוֹא, וְתֵבֵל אַחֲרָיו, / עַד הַיְּקוּם - תָּעוּ בְסַרְעַפֵּנוּ

אַכְעִיס בְּנוֹת יָמִים וְנֶגְדָּן אֶצְחֲקָה / כִּי צוֹעֲקוֹת: מִי זֶה אֲשֶׁר יִכְפֵּנוּ

הַאַתְּ הֲדַפְתָּנוּ וְאֵין כֹּחַ בְּךָ / אוֹ יַד יְקוּתִיאֵל עֲלֵי עָרְפֵּנוּ -

הַשַּׂר אֲשֶׁר לוּלֵי שְׁטָפְךָ בַעֲנַן / יָדָיו שְׂרַפְנוּךָ בְאֵשׁ קִצְפֵּנוּ

נִשְׁבַּע וְלֹא יָמִיר לְבִלְתִּי יַרְפְּךָ / וּבְיָד חֲזָקָה מִמְּךָ יַרְפֵּנוּ

מַחְמַד לְעֵינֵינוּ וְכָל עַיִן, אֲבָל / נֹפֶת לְשׁוֹנֵנוּ וּמֹר אַפֵּנוּ

הָנַח, שְׁלַח וּלְקַח, פְּקַח וּשְׂמַח, צְמַח / וַהְפֹךְ, הֲדֹךְ וּדְרֹךְ עֲלֵי אַפֵּנוּ

תֵּבֵל תְּהוֹדֶךָ לְמַעַן נִמְלְאָה / שִׂמְחָה, וְהַיּוֹתֵר צְרוֹר כַּסְפֵּנוּ.
