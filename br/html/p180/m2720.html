[ידידי החליתי] / שלמה אבן גבירול


ולה פי אלתצאבי [ולו מסוג שירי האהבה]

יְדִידַי הָחֳלֵיתִי מֵאֲהָבוֹת / וְעוֹד לֹא שָׁב לְבָבִי מִמְּשׁוּבוֹת

וְעָרְבוּ הַתְּלָאוֹת בָּם לְנַפְשִׁי / הֲרָאִיתָ תְּלָאוֹת הָעֲרֵבוֹת

וְדַקּוֹתִי עֲדֵי לֹא-תֶחֱזוּנִי / בְעֵינֵיכֶם אֲבָל בַּמַּחֲשָׁבוֹת

וְאִם לֹא תִמְצְאוּנִי בַּקְּשׁוּנִי / בְּמָקוֹם תִּשְׁמְעוּ בוֹ קוֹל כְּאֹבוֹת

וְהוּא קוֹל אִישׁ תְּלוּי לֵבָב וְנֶפֶשׁ / אֲשֶׁר נִטְבַּע בְּלֶב-יָם הָאֲהָבוֹת

קְרָאתִיכֶם לְהַמְשֵׁנִי יְדִידַי / וְלֹא אֶשְׁמַע וְלֹא אֶרְאֶה תְּשׁוּבוֹת

אֲנִי רֹכֵב עֲלֵי-יָם אַהֲבַתְכֶם / וְאֶשָּׁעֵן עֲלֵי רֹכֵב עֲרָבוֹת

וּמִי יִתֵּן יְדִידַי תַּעֲלֻמוֹת / לְכַבּוֹת אֵשׁ בְּלִבִּי רַב שְׁבִיבוֹת.
