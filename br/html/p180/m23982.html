שְׁעֵה לַחַשׁ

שְׁעֵה לַחַשׁ מְעֻנֶּה / הַיּוֹם לְךָ נִבְדָּל

לְךָ יִשְׁפֹּךְ שִׂיחוֹ / וְקוֹלוֹ לֹא יֶחְדַּל

מָרוֹם הַט אָזְנְךָ / וְכֹחֶךָ יִגְדַּל

כִּי הָיִיתָ מָעוֹז לַדָּל


שְׁעֵה לַחַשׁ מְדֻכֶּה / וְגַם נַעֲנֶה וּמֻכֶּה

אֲשֶׁר דִּמְעוֹ יְפַכֶּה / וְכַף אֶל כַּף יַכֶּה

בְּצִדְקוֹ לֹא יִזְכֶּה / אֲבָל צִדְקְךָ יְחַכֶּה

לְךָ יַעֲטֹף וְיִבְכֶּה / כְּדַל יַעֲטֹף עַל דָּל


שְׁעֵה לַחַשׁ מְעֻטָּף / בְּשִׁבֹּלֶת נִשְׁטָף

בְּפִי אַרְיֵה נֶחְטָף / וּכְאֶשְׁכֹּל נִקְטָף

יִתֵּן זַעֲקוֹ לְטוֹטָף / וְכִשְׁחֵלֶת נָטָף

יוֹם יַעֲטֹף כַּטַּף / לְךָ מָעֹז וּמִגְדָּל.
