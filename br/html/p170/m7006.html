[מִשְׁעַן עַמּוֹ וּמִבְטָחָם]

סימן: משה קטן

מִשְׁעַן עַמּוֹ וּמִבְטָחָם / הוֹד תִּפְאַרְתָּם וְעֹז נִצְחָם

רַחֵם כִּי אָבַד נִצְחָם / בְּגָלוּת וְתַם לָרִיק כֹּחָם –

עַל שְׂדֵה יֶשַׁע לְהַבְטִיחָם / וּבְמֵי שָׂשוֹן לְהַפְרִיחָם

כִּי יָדִין אֲדֹנָי עַמּוֹ / וְעַל עֲבָדָיו יִתְנָחָם:

דב' לב, לו

תה' קלה, יד

שִׂפְתוֹתַי וְנִיבֵי חִכִּי / אַקְרִיבָה לִפְנֵי מַלְכִּי

וְאָסִירָה עִקְּשׁוּת דַרְכִּי / וַאֲבָרֵר הֵלֶךְ חֶשְׁכִּי

וּדְמָעַי אֲשִׂימֵם נִסְכִּי / וְעַל אֵשׁ צְלָעַי יְהִי עֶרְכִּי

וְאוֹדֶה עֲלֵי פִשְׁעִי כִּי / מוֹדֶה וְעֹזֵב יְרֻחָם:

מש' כח, יג


הֲמוֹן מִי יִשְׁעוֹ לֹא יְכַזֵּב / לְהַשְׁקוֹת צִמְאוֹן יְדִידָיו

וְאֶת עֶדְרוֹ לֹא יְהִי עוֹזֵב / כִּי יִשְׁמֹר נַפְשׁוֹת חֲסִידָיו

וְאֶת חוֹמוֹתָיו יְהִי עוֹזֵב / לְקַבֵּץ אֵלֶיהָ שְׂרִידָיו

כִּי לֹא אִישׁ אֵל יְכַזֵּב / וּבֶן אָדָם וְיִתְנֶחָם:

במד' כג, יט


קָטָן נִקְרָא אֶפְרַיִם / מֵאָז וְיֶלֶד שַׁעֲשׁוּעִים

אֵיךְ בֵּין תַּנּוּר וְכִיַרִים / נִשְׁכַּח בְּתַחֲלוּאִים רָעִים –

קוּם כְּצֵאתְךָ מִמִּצְרַיִם / וְהָקִימָה שִׁבְעָה רוֹעִים

וּמוֹשֵׁל בִּירוּשָׁלַיִם / וּבְאֶרֶץ יְהוּדָה פֶּחָם:

נחמ' ה, יד
