בְּלֵבָב בָּר

פזמון. חתימה: משה

בְּלֵבָב בָּר / מְאֹד נִשְׁבָּר / בָּאתִי הָעֵת בְּקוּמִי,

יְעָצַנִי / וְהִנְחַנִי / לְהִתְוַדּוֹת עַל אֲשָׁמִי –

וְהִנֵּנִי / עַל עֲווֹנִי / מַקְרִיב חֶלְבִּי וְדָמִי

וּמִתְפַּלֵּל וּמִתְוַדֶּה חַטָּאתִי וְחַטַּאת עַמִּי.


מִי יְבָאֵר / וִיתָאֵר / עֱזוּז מַלְכִּי וּגְאוֹנָיו?

וְכָל גֶּבֶר / בְּפִיו דּוֹבֵר / – עֲרַל שְׂפָתַיִם לְפָנָיו,

וְהַמַּשְׂכִּיל / לֹא יָכִיל / נְשֹׂא גָדְלוֹ בְּרַעְיוֹנָיו!

וְאִם אֵפֶר / יְאַו לְסַפֵּר / מִקְצַת דַּרְכֵי אֲדוֹנָיו –

יָשׁוּב הֲלוֹם, / כִּי הוּא חֲלוֹם / וְהַמָּוֶת פִּתְרוֹנָיו.

וְעֵינָיו רָאוּ גָלְמִי / וּמֶנְהוּ לֹא נִכְחַד עָצְמִי.


שָׁוְא יְבַקֵּר / הַחוֹקֵר / סוֹד מֵהָאֱנוֹשׁ נֶעֱלָם,

אֵיךְ צוֹפֶה / בְּרוּחַ פֶּה / בָּרָא אֶת הָעוֹלָם,

וְאֵיךְ תֵּבֵל / בְּפִיו סוֹבֵל, / וְאֶת כָּל הַיְקוּם כֻּלָּם.

בְּמֻפְלָא, אֱנוֹשׁ, אַל תִּדְרֹשׁ, / פֶּן תָּשׁוּב דַּךְ נִכְלָם!

וְשׁוּב, לִבִּי, / לְמִשְׂגַּבִּי / וְהַסְכֵּן עִמּוֹ וּשְׁלָם,

טֶרֶם יָבוֹא יוֹמִי / וְאָשׁוּבָה אֶל מְקוֹמִי.


הָסֵר זַעַם, / וּפַעַם / הָרִימָה לְמַשּׁוּאוֹת,

בְּמֵי אַהֲבָה / מְחֵה חוֹבָה / וּקְרַע שְׁטָרֵי חֲטָאוֹת,

וְחֹן עַבְדְּךָ / יוֹם נֶגְדְּךָ / יְהִי נִשְׁפָּט עִם בְּרִיאוֹת –

יוֹם נְבָרִים / נִכָּרִים / לִפְנֵי אֲדֹנָי צְבָאוֹת.

וּנְפָשִׁים / בִּקְדוֹשִׁים / יִהְיוּ לְמוֹפֵת וּלְאוֹת

יוֹם קוֹל יַעֲבֹר בְּעַמִּי: / הַהוֹלְכִים מִי וָמִי?
