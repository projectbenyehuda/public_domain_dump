ברשפי הבכי נפשי שרופה

בְּרִשְׁפֵּי הַבְּכִי נַפְשִׁי שְׂרוּפָה / וּבִבְכִי הַֽיְקֹד לֶחְיִי שְׁטוּפָה

וְצִפָּרְנֵי נְדֹד תָּפְשׂוּ לְבָבִי / וְהוּא בִמְתַלְּעוֹת זֹקֶן טְרֵפָה

וְלֵיל נֹדִי בְּלִי־שַׁחַר וְלִשְׁנַת / פְּרִידָתִי בְּעוֹלָם אֵין תְּקוּפָה

וְהִרְחִיק הַזְּמָן מִנִּי קְרוֹבַי / וּבָנָיו אֻסְּפוּ עָדַי אֲסֵפָה

וְכַמֶּה שָֽׁלְפוּ עָלַי חֲרָבוֹת / וּבִימִינוֹ אֲנִי חֶרֶב שְׁלוּפָה

וּבִי יִצְנֹף לְהִתְפָּאֵר וְכַדּוּר / צְנָפַנִי לְקַצְוֵי־בוֹר צְנֵפָה

נְתָנַנִי כְצִיץ נֹבֵל בְּיַעַר / וּבֵין חוֹחִים כְּשׁוֹשַׁנָּה קְטוּפָה

וְהִמְצַנִי בְיַד־אֹפֶל וְכוֹכְבֵי / שְׁחָקִים זֹֽרְחִים הֵמָּה בְּעֵיפָה

זְמַם שָׂטִים לְבַד הֵפִיק וְתַאְוַת / לְבָבִי שָׂם בְּיַד־אֵיבָה הֲדוּפָה

וְיוֹם טַרְפִּי יְהִי מֵכִין וְטַרְפָּם / לְעַוֵּל יֶאֱחֹז אֵיפָה וְאֵיפָה

לְחֶרְפָּה הָֽיְתָה דַעְתִּי בְּאֶרֶץ / תְּבוּסָתִי וְחָכְמָתִי גְדוּפָה

וְלִנגּוֹעַ מְאֹד נַפְשִׁי תְמָאֵן / וְתִיקַר מִֽהְיוֹת נֶגְדּוֹ נְגוּפָה

וְתִתְגַּבֵּר לְמִלְחֶמֶת יְלָדָיו / בְּחֹזֶק יָד וְאֶזְרוֹעַ חֲשׂוּפָה

וְתַבְלִיג עַל־סְבֹל צוֹקָיו וְלָשֵׂאת / פְּרִידַת הַגְּבִיר יִצְחָק יְעֵפָה

לְכַנַּת תֹּם עֲלֵי יִבְלֵי חֲסָדִים / וּפֹרִיָּה בְּכָל־עֵת וַעֲנֵפָה

פְּרִי חָכְמָה תְבַכֵּר לָחֳדָשִׁים / לְפִי הַיְקוּם וְעָלֶהָ תְּרוּפָה

וּמִגְדַּל תֹּם עֲלֵי רִצְפַת תְּעוּדָה / בְּאַבְנֵי עֹז וְתִפְאֶרֶת רְצוּפָה

בְּנֵי־יוֹם לֹא יְנִיעוּהוּ וְתָבוֹר / בְּרוּחַ פִּיו הֲלֹא מֵנִיף תְּנוּפָה

בְּצוּר עֵט יַעֲרֹךְ לַחַם וְלָשׁוֹן / לְרַדֵּף דֹּֽבְרֵי צַחוֹת חֲרוּפָה

תְּהִלּוֹתָיו בְּלֵב חֹמְדָיו כְּחִצִּים / מְזִמָּתוֹ בְּעֵין קִנְאָה שְׁזוּפָה

וְכַמָּה יִשְׂחֲקוּ אֵלָיו בְּפִיהֶם / וְאוּד מֻצָּל לְבָבָם מִשְּׂרֵפָה

וְדַמּוּ לַחֲלֹשׁ יָמִין חֲזָקָה / בְּיַד־כֶּסֶל מְאֹד דַּלָּה וְרָפָה

וְאֵיכָה יַהֲדֹף כֶּבֶשׂ לְאַרְיֵה / וּפֶרֶס אֵיךְ יְהִי טֹרֵף אֲנָפָה

וְהַחוֹל אֵיךְ יְהִי עֹלֶה בְמֹאזְנֵי / אֱמֶת עִם־קַשׁ אֲשֶׁר יָעוּף כְּסוּפָה

זְרָעָיו יַעֲשׂוּ מִגְדֵי מְלִיצָה / וְשִׁבֹּלֶת תְּנוּבָתָם שְׁדוּפָה

הֲיִתְגָּרוּ בְּאִישׁ תִּתְאַו לְבָנָה / לְשָׁרְתֵהוּ כְּמוֹ שִׁפְחָה חֲרוּפָה

וְשֶׁמֶשׁ תַּעֲדֶה גָּאוֹן וְגֹבַהּ / הֱיוֹתָהּ לוֹ כְּרַקַּחַת וְאֹפָה

יְדִידִי נִכְסְפָה נַפְשִׁי לְחֶבֶר / לְשׁוֹן עֵבֶר וּפִי עֵפֶר וְעֵיפָה

רְאוֹת מִלִּים בְּמִצְרַף לֵב בְּחוּנִים / וְאִמְרַת הוֹד בְּכוּר־דַּעַת צְרוּפָה

וְהִתְלוֹנֵן בְּצֵל כַּנַּת בְּשָׂמִים / עֲלֵי־כָל־עֵץ בְּגַן־הָאֵל זְקוּפָה

סְעִפֶיהָ בְּלֶב־עֶרֶץ וְלֶאְרוֹת / בְּשָׂמֶיהָ עֲדֵי־אֶרֶץ כְּפוּפָה

הֲכִי נַעְתִּי בְּעִי־כֶסֶל וְנַפְשִׁי / בְּרָעָב אֶל־תְּנוּבָתָהּ עֲטוּפָה

וְתָעִיתִי בְּתַלְאֻבוֹת פְּתַיּוּת / וְהִיא אֶל־קֹר רְסִיסֶיהָ עֲיֵפָה

וְהֵא אָחִי לְךָ בַּת־רַעֲיוֹנִים / מְאִירָה מִמְּאוֹר סַהַר וְיָפָה

אֲחוֹתְךָ אוֹ אֲחוֹת שֶׁמֶשׁ וְאִם־הִיא / בְּלִי־יָד מִבְּנוֹת־עַיִשׁ חֲטוּפָה

וְלוּלֵי הֽוֹדְךָ לָבְשָׁה עֲרֻמָּה / תְהִי מֵהוֹד וּמֵהַחֵן יְחֵפָה

תְּשִׂיחֶךָ בְּעַד אַהְבָה בְּלִבִּי / מְזֻקָּקָה בְלִי־פֶשַׁע וְחָפָה

אֲהָבִים נֽוֹסְפוּ אָכֵן נְדָבָה / וְאֵין לָמוֹ יְמֵי חַיַּי חֲלִיפָה

וְתִתְחַנֵּן לְהַזִּיל מֵי כְתָבוֹ / עֲלֵי נֶפֶשׁ בְּאוּר פֵּרוּד שְׂרוּפָה

אֲשֶׁר עֶינָהּ כְּעֵין גַּנָּה צְמֵאָה / מְיַחֶלֶת לְיַד הָעָב וְצֹפָה.
