הֶרֶף בֹּכֶה חֻקֵּי בָתִּים

הֶרֶף בֹּכֶה / חֻקֵּי בָתִּים / אַחְרֵי יַלְדֵי / אַהַב נִצְדִּים

וּנְטֵה עִמִּי / אֶל גַּן בֹּשֶׂם / לָשֶׁבֶת בֵּין / צִלֵּי בַדִּים

וּפְנֵי אֶרֶץ / לָבְשׁוּ שָׁנִי / אוֹ מִתַּשְׁבֵּץ / לָהּ מַרְבַדִּים

הֵסִירָה אֶת / בִּגְדֵי אַלְמֹן / וַתִּפְשֹׁט כָּל / סוּתֵי עִדִּים

וַתַּעַד הוֹד / כֶּלְיֵי כַלָּה / אוֹ מִמֶּשִׁי / תַּעְטֶה בַדִּים

תִּשְׁלַח לִקְרַאת / כָל אַף עַל פִּי / רוּחֵי נֶשֶׁף / כָּל מַחְמַדִּים

תֵּן כּוֹס מֵעֵין / שֶׁלֶג יַלְבִּין / רַק כַּתּוֹלָע / יֵינוֹ יַאְדִּים

תָּעִיד כִּי אֵשׁ / הוֹא בֵּין בָּרָד / אוֹ לַפִּדִים / תּוֹךְ הַכַּדִּים.
