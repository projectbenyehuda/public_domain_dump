רהיט [מָה אֱמוּנָה נָאֶה]

סימן: אלף־בית

יג

         מָה אֱמוּנָה נָאֶה לִדְרשׁ בְּאֵין הֶפְרֵשׁ / מָה יְיָ דּוֹרֵשׁ

         מַה בֶּצַע בְּחֵילוֹ וּבַצַּר לוֹ יוֹעִילוּ / מַה בַּבַּיִת וְכָל אֲשֶׁר יֵשׁ לוֹ

         מַה גָּבְהוּ דַּרְכֵי מִדּוֹתֶיךָ מִמִּדּוֹת גְּאוֹנַי / מַה גָּדְלוּ מַעֲשֶׂיךָ יְיָ

         מַה דָּבֵק לַלֵּב וְיָפֶה לַיָּד לַעֲשׂוֹת כְּפִי אוֹנַי / מַה דִבֶּר יְיָ

5          מַה הוֹד כָּלֶה וְלֹא יָקוּם חֵילוֹ / מַה הֹוֶה לְאָדָם בְּכָל עֲמָלוֹ

         מַה וַּעַד חֲכָמִים בְּעַד יוֹצְרָם מֶחְקְרֵיהֶם / וְחָכְמַת מַה לָּהֶם

         מַה זְּעֵיר יְרוֹצֵץ לָתוּר יָמָיו וּרְאוֹת בְּמַחֲזֶה / מַה זֶּה וְעַל מַה זֶּה


         מַה חֵלֶף פֶּתַע וְלֹא יָבִין בִּמְגוּרָיו / מַה חֶפְצוֹ בְּבֵיתוֹ אַחֲרָיו

         מַה טּוֹב לַחֲסוֹת בְּסִתְרֶךָ / וְהִסְתּוֹפֵף בִּצְלָלֶיךָ / מַה טּוֹבוּ אֹהָלֶיךָ

10     מַה יָּפוּ קְרוּאֶיךָ / וְנֶחְמָדוּ יְדוּעֶיךָ / מַה יָּקְרוּ רֵעֶיךָ

         מַה כָּבוֹד טוֹבַת עָמָל יָמָיו וְשָׁנָיו / מַה כִּשְׁרוֹן לִבְעָלָיו כִּי אִם רְאוֹת עֵינָיו

         מָה לְּפוֹעֵל רְצוֹן קוֹנוֹ בְּיוֹמוֹ תִּתֵּן שְׂכָרוֹ / מַה לַּעֲשׂוֹת בָּאִישׁ אֲשֶׁר הַמֶּלֶךְ חָפֵץ בִּיקָרוֹ

         מַה מֶּתֶק מַשְׂכִּיל בַּהֲדַר יוֹצְרוֹ מְלֻבָּשׁ / מַה מָּתוֹק מִדְּבַשׁ

         מַה נָּעֲמוּ אוֹרְחוֹתֶיךָ וְעָרְבוּ מַלְאֲכוֹתֶיךָ / מַה נִּמְלְצוּ לְחִכִּי אִמְרָתֶךָ

15       מַה סִּבְרֵי בִּמְשֹׁל בְּשִׂכְלִי תַאֲוָתִי / מַה יִּסְכֹּן לָך מָה אוֹעִיל מֵחַטָּאתִי

         מֶה עָמְקוּ מַחְשְׁבוֹתֶיךָ וּמִי יַשִּׂיג צְפוּנֵיהֶם / מֶה עָצְמוּ רָאשֵׁיהֶם

         מַה פְּאֵר נָבוֹן כִּי אִם לְהִתְבּוֹנֵן בַּיָּמִים הָאֵל / מַה פָּעַל אֵל

         מַה צֶּדֶק בֶּן חֲלוֹף אֵין לוֹ בְּבוֹא חֲלִיפָתוֹ גוֹאֵל / מַה יִּצְדַּק אֱנוֹשׁ עִם אֵל

         מַה קָּצְרָה רוּחַ אֲסִיר תִּקְוָה מָתַי אֵצֵא לַחָפְשִׁי / מַה קִּצִּי כִּי אַאֲרִיךְ נַפְשִׁי

20       מָה רוֹצִים בָּנֶיךָ תִּתֵּן וְיַבִּיעוּ בְפִיהֶם / מָה רָאוּ עַל כָּכָה וּמַה הִגִּיעַ אֲלֵיהֶם

         מַה שֶּׁנֶּאֱמַר מֵאָז תְּגַלֶּה וְיָרוּם כְּבַתְּחִלָּה וְיִנָּשֵׂא / מַה שֶּׁנָּעֲשָׂה הוּא שֶׁיֵּעָשֶׂה

         מַה תַּחְפֹּץ אֲמַלֵּא תִּקְרָא וְאָפִיק מִשְׂאָלָךְ / מַה תֹּאמַר נַפְשֶׁךָ וְאֶעֱשֶׂה לָךְ
