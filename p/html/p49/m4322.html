[אֲחִי מוּסָר]

ולמה וקף אבי אעזה אללה עלי אלאביאת אלמתקדמה אלדכר כתב אליה

וכאשר קרא אבי, יוקירהו האל, את הבתים הנ"ל כתב אליו


אֲחִי מוּסָר / וּבַר תּוֹרָה! / לְךָ שִׂמְחָה, / לְךָ אוֹרָה,

לְךָ ישֶׁר, / לְמַעַן כִּי / נְתִיבָתָךְ / מְיֻשָּׁרָה,

וּמִכָּל מוּם / יְחִידָתָךְ / כְּמוֹ סַפִּיר / מְטֹהָרָה,

וְאַתְּ נִבְחָר / וְטוֹב מִכָּל / מְלֶאכֶת אֵל / אֲשֶׁר בָּרָא!

הֲבִינֵנִי / עֲלֵי מַה זֶּה / וּמָה זֶּה רוּ– / חֲךָ סָרָה, [5]

וְאֵיךְ יֵרַךְ / לְבָבֶךָ / וְתִתְרַפֶּה / בְּיוֹם צָרָה,

וְאֵיכָה מִ– / פְּנֵי רָעַת / זְמַן קַצְתָּ, / וְאֵיךְ תִּירָא

בְּשׂוּמוֹ חֵץ / עֲלֵי יֶתֶר / וְהִצִּיבָךְ / לְמַטָּרָה,

כְּאִילּוּ אַתְּ / הֲלִינוֹתָ / בְּנֵי עַמָּךְ / בְּתַבְעֵרָה,

וְאַתְ קַמְתָּ / וְרִיבוֹתָ / עֲלֵי משֶׁה / בְמֵי מָרָה, [10]

וְיִפִּיתָ / לְיִשְׂרָאֵל / לְהָמִיר אֵל / בְּבֶן פָּרָה,

וְצִוִּיתָ / בְּנֵי אַהְרֹן / לְהַמְרוֹתוֹ / בְּאֵשׁ זָרָה,

וּמָשַׁלְתָּ, / כְּמוֹ יָבִין, / בְּחֹזֶק יָד / וּבִגְבוּרָה,

וּמָאַסְתָּ / בְּבָרָק בֶּן / אֲבִינֹעַם / וּבִדְבוֹרָה,

וְסָפַחְתָּ / נְבִיאֵי דָן / בְּכִכָּרוֹת / וּבַאְגוֹרָה, [15]

וּמֵאַנְתָּ / לְאַדֵּם רֹאשׁ / נְבִיא שֹׁמְרוֹן / בְּחַבּוּרָה,

וְיִשַּׁרְתָּ / שְׁנֹא דוֹדִים / בְּעֵין סָרָה / וְסוֹרֵרָה,

וּפִזַּרְתָּ / לְבֶן עָפְרַת / לְבָנוֹן מֹר / אֲשֶׁר אָרָה,

וְהִכִּיתוֹ, / לְהָרְגוֹ אֶת / בְּנֵי אָדָם / בְּחֵץ יָרָה,

וְשַׂמְתָּ עַל / צְבִי, הֵמִית / צְבִי, כֹּפֶר / וְכַפָּרָה, [20]

וְלֹא חַשְׁתָּ, / בְּהִקָּבֵץ / יְדִידִים, לַ– / עֲנוֹת שִׁירָה,

וְלֹא חַלְתָּ, / בְּיוֹם פֵּרוּד / מְיֻדָּעִים, / כְּמַבְכִּירָה,

וְלֹא תַרְתָּ / לְמֵרֵעִים / מְקוֹם מָלוֹן / וּבֵית דִּירָה!

דְּעֶה כִּי הַ– / זְמַן אֵשֶׁת / נְעוּרִים לָךְ / מְעֻבָּרָה,

וְלֹא תֵּדַע / אֲשֶׁר תֵּלֵד / בְּיוֹם מָחָר/ לְךָ הָרָה! [25]

בְּטַח בָּאֵל / וּבִידִידָךְ / כְּמוֹ נַפְשָׁךְ, / וְהָמִּירָה,

‘הֲנָּחָתִי / כְּעָב עָבְרָה / וְאַנְחָתִי / מְשֻׁמָּרָה’

בְּ’אַנְחָתִי / כְּטַל הוֹלֵךְ / וְעַל בֵּיתִי / פְּאֵר קֵרָה'.

אֲנִי אַשְכִּי– / חֲךָ רָעַת / זְמַן, עָנָה / בְּךָ סָרָה,

וְאִם ‘כַּיָם / שְׁטַפְתִּיךָ’ / בְּאַהְבָה לֹא / מְסֻּתָּרָה – [30]

כְּיָם אֶשָּׂא / לְךָ טוֹבָה, / תְהִי לַדּוֹר / מְסֻפָּרָה,

וְתָפִיחַ, / בְּהִזָכֵר, / פְּעֻלָּתִי, / כְּאַזְכָּרָה,

וְלֹא ‘תָמוּת / בְּצִמָּאוֹן / בְּתוֹךְ צָרָה / וּבַצָּרָה’,

וְאִם יִכְרֶה / זְמַנָּךְ בּוֹר / לְךָ – יִפֹּל / בְּבוֹר כָּרָה,

וְאִם יִדְרֹךְ / נְתִיב בֵּיתָךְ – / בְּרֹאשׁוֹ אֶדְ– / רְכָה פוּרָה, [35]

וְאֶעְשֵׂהוּ / לְךָ כַדָּת / וְכַהֹגֶן / וְכַשּׁוּרָה,

וְיִתְלַבֵּן / לְךָ, אַחַר / אֲשֶׁר הָיָה / כְּפִי כִירָה,

וְיִישַׁן מִ– / לְּהָרַע לָךְ / כְּעָיֵף אוֹ / כְּשִׁכּוֹרָה,

וְתַעַשׂ לוֹ / אֲשֶׁר תַּחְפֹּץ / וְתִשְׂחַק בּוֹ / כְּצִפּוֹרָה

וְאִם תָּמְהוּ / בְּנֵי אָדָם / בְּהַמְתִּיקִי / לְךָ מָרָה – [40]

חֲכָמִים יַ– / עֲנוּ: מָה זֹּאת / וּמָה נִּפְלָא / וּמָה נּוֹרָא

עֲלֵי דָוִד / לְהִתְעוֹרֵר / וְלָקַחַת / בְּיַד עִירָא!
