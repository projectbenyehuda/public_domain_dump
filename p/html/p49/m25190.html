[אמת לאל אשר]

אֱמֶת, לָאֵל אֲשֶׁר אֵין לוֹ תְּחִילָּה // וְאֵין לוֹ אַחֲרִית, תֵּאוֹת תְּהִילָּה,

אֲשֶׁר תַּגִּיד יְצִירָתוֹ כְּבוֹדוֹ // בְּלִי אֹמֶר, וְתִתֵּן לוֹ גְּדֻלָּה,

לְרִאשׁוֹן, כֹּל אֲשֶׁר יִיצֶר יְכַלֶּה, // וְכֹל יָצַר בְּרֵאשִׁית – שָׁב וְכִלָּה,

וְנִשְׁאַר אַחֲרֵי כָל מִפְעֲלוֹתָיו // וְנִבְרָאָיו, אֲשֶׁר חִדֵּשׁ וּבִלָּה,

אֲשֶׁר אֵין לוֹ מְקוֹם שֶׁבֶת בְּעוֹלָם – // וְעוֹלָם מִכְּבוֹד הוֹדוֹ מְמֻלָּא,

וְעָבִים שָׁת לְסֵתֶר לוֹ, וְאִם לֹא // נְשׁוּרֵהוּ – גְּבוּרָתוֹ מְגֻלָּה.

בְּחֵי מֵבִין צְפוּנֵי הַלְּבָבוֹת // וְעַל כָּל לֵב וְסוֹדָיו שָׁת חֲתֻלָּה,

אֲהַבְתִּיךָ, אֲהוּב נַפְשִׁי, כְּנַפְשִׁי, // וְשַׂמְתִּיהָ לְךָ לָעַד סְגֻלָּה,

לְבָבָךְ שַׂר וְאַתָּה עָל, וְלָכֵן // בְּיִשְׂרָאֵל לְךָ מִשְׂרָה מְעֻלָּה,

וְאַתְּ נִקְלֶה בְּעֵינֶיךָ – וְגָדוֹל // לְאֵל, שָׁת לָךְ גְּדֻלָּה לֹא נְקַלָּה,

וְלֵב מֵבִין בְּךָ, וּשְׁתֵּי כְלָיוֹת // לְמוֹעֵצוֹת, כְּמוֹ זֵיתִים וְגֻלָּה,

וְאֶל בֵּיתָךְ, אֲשֶׁר בּוֹ הַתְּבוּנָה, // לְכָל דּוֹרֵשׁ תְּבוּנָה יֵשׁ מְסִילָה,

וְלָךְ פִּתְרוֹן בְּתַלְמוּדָךְ, יְיַשֵּׁר // עֲקֻבּוֹתָיו וְיַשְׁוֶה כָל מְחִלָּה,

וְחָכְמָה, תַּעֲנֶה כִּי כַמְּחוּגָה // שְׁמֵי שַׁחַק וְהָאָרֶץ עֲגֻלָּה,

וְיֵשׁ לָךְ מַעֲלָה עַל כָּל נְקִי כַף // וּפַעַם בַּחֲסִידֵי הַפְּעֻלָּה,

בְּיָדַיִם קְשׁוּרוֹת בַּתְּפִילִין // וְרַגְלַיִם מְהִירוֹת לַתְּפִלָּה.

קְחָה אָחִי דְּבָרִים, נִבְחֲרוּ מִן // בְּדֹלַחִים, חֲרוּזִים עַל מְגִלָּה,

וְיֵשׁ מִלָּה בְּצַד מִלָּה שְׂנוּאָה // וְאַחֶרֶת אֲהוּבָה הִיא לְמוּלָהּ,

וְכָל מִלַּי אֲשֶׁר אֶבְחַר אֲהוּבִים // בְּמַחְבֶּרֶת, כְּמוֹ חָתָן וְכַלָּה.
