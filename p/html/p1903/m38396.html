משל ק: שֵׁד וָאֳנִיָּה

מְשַׁלְּחֵי מְדָנִים / יַשְׁחִיתוּ עָצְמָה וְאוֹנִים

הַשֵּׁד עִם אִמּוֹ אָרַח עַל אֶרֶץ רַבָּה / לָשׁוּט בָּאָרֶץ וּלְהִתְהַלֵּךְ בָּהּ / עַד אֲשֶׁר עַל שְׂפַת הַיָּם בָּאוּ / וַיִּשְׂאוּ עֵינֵיהֶם וַיִּרְאוּ / אֳנִיָּה מְלֵאָה סוּס וְגָבֶר / וְהָאֳנִיָּה חִשְּׁבָה לְהִשָּׁבֵר / כִּי הִתְגֹּעֲשׁוּ וַיִּרְעֲשׁוּ שְׁאוֹן יָם וְגַלֵּיהֶם / וַיָּטִילוּ אֶת הַכֵּלִים אֲשֶׁר בָּאֳנִיָּה אֶל הַיָּם לְהָקֵל מֵעֲלֵיהֶם / וּמִפַּחַד הַיָּם כָּל בְּשָׂרָם סָמַר / וְקָרָא זֶה אֶל זֶה וְאָמַר / לְהַשְׁלִיךְ בַּיָּם מַחְמַד עֵינָיו וְעֶדְיוֹ / וּלְהִתְוַדּוֹת מִפְּשָׁעָיו וּמֶרְיוֹ / אָז מַחְמַדֵּיהֶם מֵאַמְתְּחוֹתֵיהֶם יָרִיקוּ / וַיִּירְאוּ הַמַּלָּחִים וַיִּזְעָקוּ / וְכָל אִישׁ מִכְּלֵי חֶפְצוֹ חָצְנוֹ נֹעֵר / כִּי הַיָּם הֹלֵךְ וְסֹעֵר / וַיֹּאמֶר הַשֵּׁד לְאִמּוֹ: אִם כְּהַיּוֹם הָיִית עִמָּהֶם / יֹאמְרוּ כִּי מִיָּדֵךְ הָיְתָה זֹאת לָהֶם / כֵּן וָכֵן פְּעָמִים רַבּוֹת / הוֹצִיאוּ עָלַיִךְ דִּבּוֹת / כְּבֹא לָהֶם יוֹם מְבוּכָה וּמְהוּמָה / עַל לֹא חָמָס עָשִׂית וּמִרְמָה / וְהָאֵם תָּשִׁיב אֲמָרֶיהָ לְמוּלוֹ / הֵעֵזָה פָנֶיהָ וַתֹּאמַר לוֹ / אִם אֵין אֲנִי שָׁם, הַרְבֵּה מָצָאתִי / מֵאֲשֶׁר גִּדַּלְתִּי טִפַּחְתִּי וְרִבִּיתִי / אֲשֶׁר שְׁלַחְתִּים שָׁם רְצוֹנִי יֹאבוּן / מִיָּדִי הָיְתָה זֹאת לָהֶם לְמַעֲצֵבָה יִשְׁכָּבוּן / וּמְאֹד מִמַּעֲשַׂי בָּאָרֶץ יִירְאוּ / מְהָרְסִים וּמַחֲרִיבִים מִמֶּנִּי יֵצֵאוּ.

וְהַמָּשָׁל בִּמְשַׁלְּחֵי מְדָנִים / רְכִילִים וּמַלְשִׁינִים / מִזַּעַם לְשׁוֹנָם זֶרַע יִזְרְעוּ בֵחֲכָמִים / לְהָרַע לִבָּם וּפִיהֶם לִשְׁלוֹמִים / וְיֶחֱזוּ מַשְׂאוֹת שָׁוְא וּמַדּוּחִים שְׁקָרִים / וְכָל אֶחָד מֵהֶם עַל רֵעֵהוּ יַעֲרִים / בְּרָעָתָם יְשַׂמְּחוּ מֶלֶךְ וּבְכַחֲשֵׁיהֶם שָׂרִים / וְכַאֲשֶׁר יַצְלִיחוּ בְמַלְשִׁינוּתָם / אָז יִתְהַלֲלוּ בְקֶרֶב עֲדָתָם / וְיֹאמְרוּ כִּי עַל יְדֵי שָׁלִיחַ / בִּרְכִילוּתוֹ הִצְלִיחַ / וְהֶחָכָם אָמַר בְּחָכְמָתוֹ / שְׁלוּחוֹ שֶׁל אָדָם כְּמוֹתוֹ.
