משל ה: כֶּלֶב, גְּבִינָה וָמָיִם

הַבִּלְתִּי מִסְתַּפֵּק לֹא יֵאוֹת / וְעֵינָיו לֹא יִשְׂבְּעוּ אַף לִרְאוֹת

כֶּלֶב בַּבַּיִת לָקַח גְּבִינָה / וַיִּשָּׂאֵהוּ בְפִיו אָנֶה וְאָנָה / וַיִּפֶן כֹּה וָכֹה / וַיִּקֶר עַל הַגֶּשֶׁר דַּרְכּוֹ / וַיִּתֵּן בַּמַּיִם אֶת עֵינָיו / וַיַּרְא צֵל הַגְּבִינָה אֲשֶׁר בֵּין שִׁנָּיו / וַיֹּאמֶר בְּלִבּוֹ: אִם יִהְיֶה לִי גְּבִינַת הַמַּיִם וַאֲשֶׁר בְּפִי יַחַד / טוֹבִים הַשְּׁנַיִם מִן הָאֶחָד / וַיָּשֶׁת עֵצוֹת בְּנַפְשׁוֹ / לָכֹף כְּאַגְמוֹן רֹאשׁוֹ / וְלִבְלֹעַ גַּם הֶחָרִיץ אֲשֶׁר בַּמַּיִם / וּבְפָעֳרוֹ פֶה וְהַרְחִיבוֹ הַשִּׁנַּיִם / לִבְלֹעַ אֲשֶׁר לֹא הִגִּיעוֹ / נָפַל מִפִּיו בִּלְעוֹ / אֲשֶׁר הֶחֱזִיק בּוֹ בַתְּחִלָּה / וְקֵרֵחַ מִכָּאן מִכָּאן עָלָה / מִן הַמַּיִם וְלֹא נִשְׁאַר לוֹ / מְאוּמָה מִכָּל עֲמָלוֹ.

וַאֲנִי רָאִיתִי זֹאת הַמִּדָּה / בְּאִישׁ אֲשֶׁר לִבּוֹ מָלֵא חֶמְדָה / בְּכֶסֶף וְזָהָב כָּבֵד / הַכֹּל חוֹמֵד הַכֹּל אוֹבֵד / וּמִשְׁלֵי שְׁלֹמֹה לִי יֶעֱרַב / כִּי יֵשׁ מִתְרוֹשֵׁשׁ וְהוֹן רָב / וְאָמַר הֶחָכָם: תִּסְתַּפֵּק בְּמַה שֶׁיֵשׁ בְּיָדְךָ וּבִרְשׁוּתְךָ / וְאַל תְּקַנֵּא בְּמַה שֶּׁיֵשׁ לְזוּלָתֶךָ.

וָאֶשָּׂא מְשָׁלִי וָאֹמַר:


טֶרֶף לְכָל חַי נָדְרוּ שָׁמַיִם / כֵּפֶל בְּטַרְפָּם חָמְדוּ פַעְמָיִם

חוֹמֵד וְחוֹמֵס טַרְפּוֹ עַל חֶלְקוֹ / יִקְרֶה כְּכֶלֶב בִּגְבִינַת מַיִם

חָשַׁב לְהוֹסִיף סִפְקוֹ וַיִּגְרַע / אֹכֶל לְהָשִׁיב נַפְשׁוֹ יוֹמָיִם.
