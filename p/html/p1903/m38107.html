משל מו: חָתוּל וְעַכְבָּר

יֵשׁ הַצָּלָה וּמָנוֹס / לִשְׁבוּעָה בִּמְקוֹם אָנוֹס

חָתוּל הָלַךְ לָתוּר מַאֲכָלוֹ / לְעֵת בֹּקֶר וַיַּרְא לְמוּלוֹ / עַכְבָּר בִּכְלִי שֵׁכָר מָלֵא / וַתִּקְרֶאנָה אֹתוֹ כָּאֵלֶּה / כִּי לָצֵאת מִשָּׁם לֹא יָכֹל / וְהֶחָתוּל רָעֵב לֶאֱכֹל / שָׂמַח וַיֵּצֵא לִקְרָאתוֹ / שָׁמָּה תִגָּלֶה אֵיבָתוֹ / בְּאֵיבַת עוֹלָם יֶאֱשַׁם אָשֹׁם / שֹׂנֵא הוּא לוֹ מִתְּמֹל שִׁלְשֹׁם / עַל שְׂפַת הַכְּלִי נֶחְפַּז בְּרַגְלָיו / וְאָרַב לוֹ וָקָם עָלָיו / וְקוֹלוֹ עַל הָעַכְבָּר הֵרִים / וַיָּשֶׂם לוֹ עֲלִילוֹת דְּבָרִים / לֵאמֹר: צִדְקָתִי וְרִשְׁעֲךָ אוֹתְךָ שָׁם הָדָפוּ / הוֹי מַשְׁכִּימֵי בַבֹּקֶר שֵׁכָר יִרְדֹּפוּ / מַהֵר אַל תַּעֲמֹד וָצֵא / וְחֶפְצִי בְךָ אֶהְיֶה מוֹצֵא / אֶבְלְךָ נֶהְפַּךְ אֵלַי לְחֶדְוָה / בַּאֲשֶׁר בְּשָׂרְךָ בְּשַׂר תַּאֲוָה / וְהָעַכְבָּר הֵשִׁיב מַעֲנֶה / אֲנִי נִגַּשׂ אֲנִי נַעֲנֶה / אֲנִי מָרוּט וְקֵרֵחַ / בְּלִי טַעַם וּבְלִי רֵיחַ / כִּי נָפַלְתִּי וְלֹא קַמְתִּי / וְנִבְעַתִּי וְנִצְמַתִּי / וּמִשֶּׁמֶן בְּשָׂרִי רָזֶה / מִיּוֹם נָפְלִי בַבּוֹר הַזֶּה / וּמֵרֹב רָגְזִי וְעָצְבִּי / הַכָּרַת פָּנַי עָנְתָה בִי / וּכְבָר חֹדֶשׁ יָמִים / לֹא בָא בְּפִי לֶחֶם וּמַטְעַמִּים / וְאִם גִּיל עָלַי בְּךָ צָמַח / בִּנְפֹל אֹיִבְךָ אַל תִּשְׂמָח / וְאַל תִּגְמֹל עָלַי עַתָּה רָעָה / הַצִּילֵֵנִי וְאַל אֶטְבָּעָה / וִיהָבִי אַשְׁלִיךְ עָלֶיךָ / לַמּוֹעֵד אָשׁוּב אֵלֶיךָ / וְאֶהְיֶה בָרִיא וְיִצְמַח שְׂעָרִי / וּבְשׁוּבִי תִּשְׂמַח בִּבְשָׂרִי / אַךְ תּוֹצִיא מִצָּרָה נַפְשִׁי / כִּי צָפוּ מַיִם עַל רֹאשִׁי / אֶל הֶחָתוּל שָׁלַח יָדוֹ / כִּי יָשׁוּב אֵלָיו בְּמוֹעֲדוֹ / וְעַד חֹדֶשׁ אֹתוֹ יֶחֱזֶה / וַיִּשָּׁבַע עַל הַדָּבָר הַזֶּה / אִם יוֹצִיאֶנּוּ מִמְּצוּדָה / לִשְׁמֹר אֶת בְּרִיתוֹ לְעָמְדָהּ / וְהֶחָתוּל אֲשֶׁר הָיָה בְּעֹכְרָיו / נֶהְפַּךְ לוֹ לִהְיוֹת בְּעֹזְרָיו / וְאֶת אֵיבָתוֹ שָׁם לֹא זָכַר / וְהוֹצִיא אֶת הָעַכְבָּר מִן הַשֵּׁכָר / וְאַחַר אֲשֶׁר מִן הֶחָתוּל נִפְרָד / הָפַךְ עֹרֶף וּבוֹ מָרַד / וְלִבּוֹ מִכָּל טוֹב סוֹעֵד / וַיּוֹחֶר מִן הַמּוֹעֵַד / וְהֶחָתוּל הִגִּיד לוֹ לֵאמֹר / וְאַתָּה אֶת בְּרִיתִי תִשְׁמֹר / לָשׁוּב כַּאֲשֶׁר נֵשְׁבַּעְתָּ לִי / שָׁמֵן וְדָשֵׁן בִּגְבוּלִי / וְאַל תְּאַחֵר פְּעָמֶיךָ / כִּי הוֹסַפְתִּי עַל יָמֶיךָ / וְאִם בְּתוֹךְ מִרְמָה שִׁבְתֶּךָ / תֶּן נָא בִּנְךָ תַּחְתֶּךָ / הָעַכְבָּר הוֹדָה בִשְׁבוּעָה / אַךְ אָמַר כִּי אָנוּס וְהָרָעָה / נֶגְדּוֹ כַּאֲשֶׁר נִשְׁבַּע הָיָה / וְלָצֵאת מִשָּׁם לָרְוָיָה / קָפַץ וְנִשְׁבַּע אֶל יַד נֵכָר / וְהַשְּׁבוּעָה עַל יְדֵי שֵׁכָר / שֵׁכָר וְאֹנֶס הָיוּ שָׁם שְׁנֵיהֶם / וָחַי בָּהֶם וְלֹא שֶׁיָּמוּת בָּהֶם / וּבַעֲלִילָה כָּזֹאת נִמְלַט / אֲשֶׁר הֶחָתוּל בּוֹ לֹא שָׁלָט.

וְהַמָּשָׁל בִּהְיוֹת דָּבָר בֵּין שְׁנֵי אֲנָשִׁים / שְׁנֵיהֶם יִהְיוּ חֹקְרִים וְדֹרְשִׁים / פֶּן יֵהָפֵךְ הַדָּבָר עַל יְדֵי עֲלִילָה וְסִבָּה / לְהוֹצִיא שְׁנֵיהֶם מִדִּבָּה / כִּי לֹא יִתְקַיֵּם בְּאֹנֶס וְשִׁכְרוּת / הַדָּבָר, אַךְ יֵחָשֵׁב לְמַעֲשֵׂה נַעֲרוּת / וְאָמַר הֶחָכָם לִבְנוֹ / לֵב חָכָם לִימִינוֹ / אִם חָכַמְתָּ תִּתְגַּדֵּל וְתִתְנַשֵּׂא / אַל תְּהִי חָכָם בִּדְבָרִים אִם לֹא בְמַעֲשֶׂה / כִּי דְּבַר פִּי אִישׁ אֶל מַעֲשָׂיו לֹא דָמָה / אִוֶּלֶת הִיא לוֹ וּכְלִמָּה.
