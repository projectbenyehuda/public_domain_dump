משל נז: תַּיִשׁ וְלַיִשׁ

הִזָּהֵר מִן הַיּוֹעֵץ לְפִי דַרְכּוֹ / וְאֹיִבְךָ כִּי יַמְתִּיק חִכּוֹ


תַּיִשׁ עַל שֵׁן סֶלַע וּמְצוּדָה / תֹּעֶה בְּלִי רוֹעֶה בְּנִיעָה וְנִידָה / כִּי אֵין שָׁם צֶמַח וְלֹא לַדֶּשֶׁא מוֹצָא / גָּפְרִית וָמֶלַח שְׂרֵפָה כָל אַרְצָהּ / וְהַלַּיִשׁ בְּתַחְתִּית הָהָר לְמוּלוֹ / וַיִּתְבּוֹנֵן לַתַּיִשׁ וַיִּקְרָא לוֹ / מַה לְּךָ כִּי עָלִיתָ מְקוֹם צְמָחִים צְנוּמִים / נִתַּן הַסֶּכֶל בַּמְּרוֹמִים / בַּאֲשֶׁר יָבְשׁוּ צִמְחֵי שְׁרוֹנֶךָ / עֶלְיוֹן שַׂמְתָּ מְעוֹנֶךָ / בְּאֶרֶץ בְּלִי מַיִם צְמֵאָה וַעֲיֵפָה / רְדָה אִתִּי בְכַר נִרְחָב וְהָיָה מַאֲכָלְךָ לִתְרוּפָה / מְקוֹם מִרְבַּץ אֵילִים וְעַתּוּדִים וְכָרִים / מְשׂוֹשׂ פְּרָאִים מִרְעֵה עֲדָרִים / וְיִתְעַנְּגוּ עַל רֹב שָׁלוֹם גּוּרַי וְגוּרֵיהֶם / יַחְדָּו יִרְבְּצוּ יַלְדֵיהֶם / וְאַל תֹּאמַר כִּי בִמְקוֹמְךָ עֵשֶׂב וָזֶרַע / הוֹי הָאֹמְרִים לָרַע טוֹב וְלַטּוֹב רָע / וַיַּעַן הַתַּיִשׁ: יָדַעְתִּי גַּם אֲנִי יָדַעְתִּי / כִּי לֹא מוֹצָא דֶשֶׁא וָזֶרַע אִתִּי / וּבִמְקוֹמְךָ הֲדַר הַשָּׁרוֹן וְהַכַּרְמֶל / לְרַפֵּא כָּל שְׁבוּרֵי לֵב וְאָמֵל / וּבְטֶרֶם בָּא דְבָרְךָ לָרֶדֶת לִרְעוֹת שָׁם בְּלִבִּי תַרְתִּי / אַךְ עַתָּה כַאֲשֶׁר שָׁמַעְתִּי עֲצָתְךָ נִחָמְתִּי / הַאַתָּה הוּא אֲשֶׁר מָחַץ רֹאשׁ עַל אֶרֶץ רַבָּה / טוֹב פַּת חֲרֵבָה וְשַׁלְוָה בָהּ / בַּאֲשֶׁר לְקוֹלְךָ יִרְעֲשׁוּ מִגְרְשׁוֹת / לִשְׁפָּךְ דָּם לְאַבֵּד נְפָשׁוֹת.

וְהַמָּשָׁל תִּלְמַד לְהָבִין וּלְהוֹרוֹת / פֶּן יִהְיוּ מְזִמּוֹת לִבְּךָ מֵהָבִין עֲקָרוֹת / אִם אֹיִבְךָ לְפַתּוֹתְךָ יַמְתִּיק חִכּוֹ / הִזָּהֵר מִן הַיּוֹעֶצְךָ לְפִי דַרְכּוֹ / וּרְאֵה וְהָבֵן בְּמַעֲשֵׂה הַתַּיִשׁ / כִּי לִבּוֹ לֹא נִפְתָּה לַעֲצַת הַלַּיִשׁ / אֲשֶׁר חָשַׁב לְטָרְפוֹ בְעָרְמָתוֹ / וְהֵבִין כִּי מִתְאַנֶּה הוּא לוֹ בַעֲצָתוֹ / וְאָמַר הֶחָכָם: עֲשֵׂה וְהַשְׁלֵם עֲצַת אֹהַבְךָ / וּשְׁמַע וְאַל תַּעֲשֶׂה עֲצַת אֹיִבְךָ / וּדְבַר הָאֹהֵב תְּקָרֵב כְּסַפִּיר וְנֹפֶךְ / וְדִבְרֵי הָאֹיֵב שְׁמַע וַעֲשֵׂה לְהֶפֶךְ / וְאִם שְׁכֵנֶיךָ בְלֹא עֵצָה וְיִדְמוּ לָעֵץ / וְלֹא תִמְצָא בְּמִי הִתְיָעֵץ / סוּר מִן הָעֵצָה הַקְּרוֹבָה אֶל הַתַּאֲוָה / וּבָטַחְתָּ כִּי יֵשׁ תִּקְוָה.
