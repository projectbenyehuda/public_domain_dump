צְרִי חָלְיִי

סימן: צאלח. מנגינה: ראשט.


צְרִי חָלְיִי / סְלַח מֶרְיִי / וּזְדוֹנִי

נִתְחָרַטְתִּי / כִּי מָרַדְתִּי / בַּאְדוֹנִי

בִּי חָזַרְתִּי / לְךָ שַׁבְתִּי / מֵרְצוֹנִי, צוּר גְּאוֹנִי.


אָנָּא הוֹשַׁע / שָׁב מִפֶּשַׁע / מְרוּדָיו

שַׂמַּח אֶבְלוֹ / נֶגֶד סִבְלוֹ / מֵרוֹדָיו

אִם הוּא בַעַר / חֲשֹׁב צַעַר / נְדוּדָיו, נָד מִדּוֹדָיו.


לָךְ מִתְגָּאֶה / כַּמָּה נָאֶה / לִסְלֹחַ

לְאִישׁ הֵלֶךְ / לְךָ הוֹלֵךְ / שְׁחוֹחַ

דָּךְ מֵעוֹדוֹ / גַּם כְּבֵדוֹ / פָּלוּחַ, מַר מָלוּחַ.


חֲשֹׁב מַעְנִי / כְּקָרְבָּנִי / וְזִבְחִי

לָצֵאת חָפְשִׁי / הַנְפֵשׁ נַפְשִׁי / וְרוּחִי

רַחְשִׁי הַקְשֵׁב / וְחִישׁ הָשֵׁב / מַדּוּחִי, צוּר מִבְטָחִי.
