יְיָ שְׁעֵה

יְיָ שְׁעֵה לַחְשִׁי וְרַחְשִׁי / וֶהְיֵה נָא דוֹרְשִׁי

שׁוּבָה יְיָ חַלְּצָה נַפְשִׁי / הוֹשִׁיעֵנִי לְמַעַן חַסְדֶּךָ

שׁוּב מֵחֲרוֹן אַפֶּךָ / וְהִנָּחֵם עַל הָרָעָה לְעַמֶּךָ


לְהוֹדוֹת לְךָ קַמְתִּי / וְרַחֲמֶיךָ שִׂבַּרְתִּי

וּדְלָתֶיךָ דָפַקְתִּי / לְהַגִּיד בַּבֹּקֶר חַסְדֶּךָ


מֶלֶךְ רָם וּמִתְנַשֵּׂא / הֱיֵה לָנוּ מָעוֹז וּמַחֲסֶה

וְאֶל עַם בְּשִׁמְךָ חוֹסֶה / מְשֹׁךְ חַסְדֶּךָ


הַפְלֵה חֲסָדֶיךָ / לְזֶרַע יְדִידֶךָ

הָאִירָה פָנֶיךָ עַל עַבְדֶּךָ / הוֹשִׁיעֵנִי בְחַסְדֶּךָ


חַנּוּן רְפָא מְשׁוּבָה / בְּעָמְדִי לְפָנֶיךָ בִתְשׁוּבָה

זָכְרָה לִּי אֱלֹהַי לְטוֹבָה / וְחוּסָה עָלַי כְּרֹב חַסְדֶּךָ


זְכֹר דְּבַר חוֹזֶיךָ / לְעַם נִקְרָאִים אֲחוּזֶיךָ

וַאֲנִי אָשִׁיר עֻזֶּךָ / וַאֲרַנֵּן לַבֹּקֶר חַסְדֶּךָ


קָרֵב נָא יִשְׁעָם / כִּי מְאֹד בְּגָלוּת נִפְעָם

סְלַח נָא לַעֲווֹן הָעָם / הַזֶּה כְּגֹדֶל חַסְדֶּךָ.
