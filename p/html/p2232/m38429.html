[אֱלֹהִים זְנַחְתָּנוּ פְרַצְתָּנוּ אָנַפְתָּ תְּשׁוֹבֵב לָנוּ (תחנה)]

[מרוֹמָא, מת אחרי שנת 1284]


תְּחִנָּה עַל־צָרַת הַזְּמָן


אֱלֹהִים זְנַחְתָּנוּ פְרַצְתָּנוּ / אָנַפְתָּ תְּשׁוֹבֵב לָנוּ.


שְׁנוֹת יְמִין עֶלְיוֹן חַלּוֹתִי הִיא / וּבְחַטָּאתִי נִרְחְקָה הַיְשׁוּעָה

הֲלֹא הַמַּחֲצֶבֶת רַהַב הִיא / וְלֹא קָצְרָה יָדוֹ וְלֹא נִגְרָעָה

עַל-זֹאת אֶזְעֲקָה הֶגֶה וָהִי / כִּי בַעֲוֹנִי קַרְנִי נִגְדָּעָה


שׁוּבֵינוּ אֱלֹהֵי יִשְׁעֵנוּ / וְהָפֵר כַּעַסְךָ עִמָּנוּ.


לֹא לָנוּ אֵל כִּי אִם-לְשֵׁם קָדְשָׁךְ / קַנֵּא וְלִימִינְךָ הַנֶּאְדָּרִי

הוֹשֵׁעַ אֶת-עַם מֹרָט וּמְמֻשָּׁךְ / רְאֵה כִּי הֵם לְבַז בְּיַד חוֹרִי

בָּאֵשׁ וּבַמַּיִם וּבַמַּחְשָׁךְ / יִסְּרוּנוּ בַּחֲמַת-קֶרִי


וַנְּהִי כַטָּמֵא כֻּלָּנוּ / כִּי סְחִי וּמָאוֹס שַׂמְתָּנוּ.


מָעֻזִּי וּמְנוּסִי בְּיוֹם צָרָה / אַל-תַּעְלֵם אָזְנְךָ לְשַׁוְעָתִי

לְבַד מִמְּךָ אָבִי אוֹחִיל לְעֶזְרָה / לֹא אֶשְׁאֲלָה אֵל זָר לִישׁוּעָתִי

מַה-תַּעֲשֶׂה לְשִׁמְךָ עָלַי נִקְרָא / אִם נֶאֱבַד נִצְחִי וְתוֹחַלְתִּי


מִי-יִתֵּן בְּיָדְךָ מוּתֵנוּ / וּמִדְּרָכֶיךָ לֹא תַתְעֵנוּ.


הַפְלֵה חֲסָדֶיךָ כִּי לֹא-כָלוּ / וְרַחֲמֶיךָ יִכְבְּשׁוּ אֶת-כַּעֲסָךְ

וְנִפְלְאוֹתֶיךָ מִקֶּדֶם גָּדָלוּ / אֶזְכְּרָה כִּי אֶעֱבֹר בַּסָּךְ

פֶּן-יֹאמְרוּ כִּי מִבִּלְתִּי יְכֹלֶת חָדָלוּ / עוֹרֲרָה גְבוּרָתֶךָ וַהֲרֹס מָסָךְ


הַבֶּט-נָא עַמְּךָ כֻלָּנוּ / וּלְכָה לִישֻׁעָתָה לָּנוּ.


רְאֵה רַחוּם דִּמְעַת הָעֲשׁוּקִים / בַּצַּר פְּקָדוּךָ בְּכָל-לִבָּם

בִּבְכִי וּבְתַחֲנוּנִים נֶאֱנָקִים / זוֹבְחִים תּוֹדָה דָּמָם וְחֶלְבָּם

מַלֵּט יוֹנְקִים וְעוֹלֲלִים צוֹעֲקִים / פְּרֹשׂ כְּנָפֶיךָ וֶהְיֵה מִשְׂגַּבָּם


וְיֹאמְרוּ בָּרוּךְ אֲשֶׁר גְּאָלָנוּ / הִגְדִּיל יְיָ לַעֲשׂוֹת עִמָּנוּ.
