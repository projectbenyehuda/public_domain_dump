עֹז לְעַמּוֹ

סימן: עבדאללה. על כותל הדירה המוקדשת ליורשי יעקב צמח.


עֹז לְעַמּוֹ / כָּבוֹד לִשְׁמוֹ / מִמְּרוֹמוֹ / נָתַן לָמוֹ

בְּהַמְצִיא אֵל / שַׂר וְגוֹאֵל / לִיחֶזְקְאֵל / נְבִיאֵמוֹ

דִּירַת גִּנּוֹת / חֻרְבּוֹת פִּנּוֹת / אִוָּה לִבְנוֹת / בֵּית אוּלַמּוֹ

אוֹרוֹ יִצְמַח / צוּרוֹ יִשְׂמַח / יַעְקֹב צֶמַח / גָּדוֹל שְׁמוֹ

לָאֶבְיוֹנִים / יִהְיוּ חוֹנִים / כֻּלָּם עוֹנִים / רַב שְׁלוֹמוֹ

לָשׂוּם שֹׁרֶשׁ / מֶנּוּ יוֹרֵשׁ / כִּי הוּא דוֹרֵשׁ / טוֹב לְעַמּוֹ

הֵן נִגְמְרָה / מֵאֵל עֶזְרָה / בִשְׁנַת הַתְרָ״ה / לְעוֹלָמוֹ.
