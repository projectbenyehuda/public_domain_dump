פָּרוֹחַ תִּפְרַח

סימן: עבדאללה בר כצ׳יר ח׳זק


פָּרוֹחַ תִּפְרַח / מַטֶּה לְבֵית לֵוִי

בִּיאַת נִין זָרָח / לָאֵל כָּל יוֹם קַוִּי.

עִיר קִרְיַת חָנָה / בֹּא יָבֹא בְרִנָּה / וּכְבָרִאשׁוֹנָה / שַׁלְוַת עוֹלָם שַׁלְוִי

בְּסֵתֶר עֶלְיוֹן / יָשִׁיב לְבַת צִיּוֹן / נֹגַהּ אוֹר חֶבְיוֹן / בִּנְוֵה שָׁלוֹם נָוִי

דּוֹד נְצַרְתַּנִי / וּכְבֵן אִמַּצְתַּנִי / לָמָּה שַׂמְתַּנִי / אַט לְמַכִּים גֵּוִי

אֵל לְמוֹשָׁעוֹת / הַקְרֵב לָנוּ שָׁעוֹת / אַתָּה אֵל דֵּעוֹת / הַנּוֹתֵן לַשֶּׂכְוִי

לָהּ קוּם גְּאָלָהּ / וְהָפֵר הָאָלָה / וְשׁוֹר מִלּוּלָהּ / מִשְׁפַּחַת הַיִּשְׁוִי

בַּר לֵבָב אָנִי / עַבְדְּךָ רִמָּנִי / הוּא הַצְּפוֹנִי / שְׁכֶם בֶּן הַחִוִּי

כְּצִיר נֶאֱמָן / מַרְפֵּא בְּבֹא זְמָן / אִישׁ רַב חֶסֶד מַן / מָזוֹר לֵבָב דַּוִּי

חָזָק נִרְוֶה עַד / הַבֹּקֶר נֹאכַל עַד / וְתִכֹּן לָעַד / הֲדָרִי וְזִיוִי.
