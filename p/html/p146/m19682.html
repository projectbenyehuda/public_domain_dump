אֲהוּב לִבִּי לְבַת נָדִיב בְּאַהְבָה יְחַלֶּה

חתימה: אלש(ש)בזי (43:אלמשתאי)


אֲהוּב לִבִּי לְבַת נָדִיב / בְּאַהְבָה יְחֱלֶּה

וְתָקַע אָהֳלוֹ בִּכְזִיב / מְצַפֶּה יַעֲלֶה

וְעָלַי יִדְלְפָה מִרְבִיב / יְשׁוֹבֵב נֶהֱלֶה

וְיֹאמַר אֶל יְפַת עַיִן

תְּנַחֵם לִי בְכוֹס יַיִן

וְאֶנְשֶׁה לַעֲצַת קַיִן / אֲשֶׁר הוּא נֶאֱלֶה:


לַךּ אַלאִחסַאן וַאלפַצׄלַא / אִלַאהִי מִן קְדִים

וַמֻלכַּף צַאדִק אַלעַדלַא / וְיַא נַעם אַלכְּרִים

וּמִן פַרדוּסַךּ אַלאַעלַא / אְנַאר מוּסַי אַלכְּלִים

וְרַחַל זִינַת אַלאֻמַּה

דְוַאעִי אַלעִלם וַאלחִכּמַה

אְקַאמוּ אַלקֻדס וַאלכִדמַה / בִפַיצֻׄן חַאצִלַא:


שְׁלַח יִנּוֹן מְשִׁיחֵנוּ / וְעוֹרֵר חַסְדְּךָ

מְחֵה אֶת כָּל פְּשָׁעֵינוּ / רְצֵה עַל עַמְּךָ

עֲשֵׂה בִזְכוּת אֲבוֹתֵינוּ / וְחֹן רַעְיָתְךָ

וְצַוֵּה יָהּ לְאֶרְאֶלִּים

וְכוֹכִָבִים וְגַלְגַּלִים

יְעִירוּן חֵן לְבַת גַלִּים / לְגַנָּהּ תַּעֲלֶה:


שְׁמוּעַת דּוֹד תְּשַׂמֵּחַ / לְלִבִּי יוֹם נְדוֹד

וּבוֹ חוֹסֶה וּבוֹטֵחַ / וְלאֹ אָמִיר בְּדוֹד

וְהוּא מוֹחֵל וְסוֹלֵחַ / לְנַפְשִׁי מִשְּׁדוֹד

אֲנִי נִקְשַׁר בְּאַהְבָתוֹ

וְאַהְבַת עַם סְגֻלָּתוֹ

וְאֵיכָה אַחֲלִיף דָּתוֹ / וְנַפְשִׁי קִבְּלָה:


בִּפַצׄלַךּ יַא כְׄלִיל דַּענִי / וְסַלִּי כִׄאטִרִי

וְאִגפִר זַלְּתִי עַנִּי / לִאַנִּי חַאִירִי

וְיַום אַלדִּין אִרחַמנִי / וְאַרגַׄע זַאִירִי

וְאַנֻצר גַׄנַּת אַלכֻׄלדִי

בִּיַוְם אַלנַּפס תַתוַדִּי

וְתַגנִי זַהרַת אַלוַרדִי / בִפַרדוּס אַלעֻלַא:


זְהוּר עַדְנַאן קַאטִפְהַא / צְׄמִירִי פִי אַלחַיַאת

וְלִי עַקלֹן יֻצַׄאעִפְהַא / בִסִרּ חִכּמַה וְדָׄאת

אְנִיר נַפסִי וְלַאטִפְהַא / תְטִיב וַקת אַלמַמַאת

תֻטִיע אַלעַקל אַלאִנסַאנִי

וְתַלחַץׄ פַיץׄ אַלאִחסַאנִי

וְאַפצִח נֻטק לִסַּאנִי / בִקַולֻן כַּאמִלַא:


יְקוּל אַלמַשׁתְאִי קַד תַם / קַולִי דִׄי חְצַל

אִלַאהִי סַלִּי אַלמֻגתַם / בִסִרִּי אַלמֻנפְצַל

וְלַא סִרֻּן עְלַיךּ יֻכּתַם / אִלַיךּ גַׄמעַה נְצַל

סְלַאמִי אַכַׄר אַלמַעְנַא

וְחַיַּא גַׄמע סַאדַתנַא

וְלִלַּה סִרּ גַאַיתנַא / מֻדַבֵּר לִאלמְלַא:
