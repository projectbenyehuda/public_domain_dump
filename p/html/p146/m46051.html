שׁוֹכֵן עַד

חתימה: שלם שץ


שׁוֹכֵן עַד וְקָדוֹשׁ / בְּחַשְׁמַלֵּי מְרוֹמִים

יָחִישׁ יְחַיֵּנִי / וְיִמְחוֹל לַאֲשָׁמִים

וְיִהְיֶה לִי לְעֶזְרָה / מֵאוֹיְבִים וְקָמִים

וְנָשׁוּבָה לְקַדְּשׁוֹ / וְנַקְרִיבָה שְׁלָמִים

שִׁבְתִּי בְּבֵית יְוֹיָ / לְאוֹרֶךְ יָמִים:


לְבֵית קָדְשׁוֹ יְכוֹנֵן / הֵיכָל וַעֲזָרוֹת

וְנָשׁוּבָה בְּחֶמְלָה / וְנִשְׁמַע קוֹל בְּשׂוֹרוֹת

וְנִרְאֶה חֵל וְחוֹמָה / וְעֶשֶׂר הַמְּנוֹרוֹת

וְכֹהֵן הַמְקֻדָּשׁ / בְּיוֹם כִּפּוּר יְשָׁרֵת

וּלְוִיִּם בְּדוּכָן / בְּשִׁירוֹתָם נְעִימִים:


מִזְבֵּחַ וְקָרְבָּן / וַעֲלִיַּת רְגָלִים

וּמָשִׁיחַ וְסֶגֶן / וְאַרְבָּעָה דְּגָלִים

וּשְׂעִירִים שְׁנַיִם / בְּגוֹרָל וַחֲבָלִים

וּפָרָה הַעֲשׂוּיָה / לְטַהַר מִגְּעָלִים

וְתוֹרָה הַהֲדוּרָה / בְּכִנּוֹר וַחֲלִילִים

וְשָׁם נַקְרִיב לְרָצוֹן / קָרְבָּנִים תְּמִימִים:


שְׂכָרֵנוּ יְהֶא שָׁם / לְמוּל הָאֵל מְרֻבָּה

כְּמוֹ גַּם אִישׁ חֲמוּדוֹת / בְּדִבּוּרוֹ וְנִבָּא

וְאַשְׁרֵי הַמְחַכֶּה / כִּי יְשׁוּעָה קְרוֹבָה

וְנִרְאֶה חֶפְצֵנוּ / בְּרוֹב שָׂשׂוֹן וְאַהְבָה

וְאָז נִשְׂמַח וְנִרְאֶה / בְּשִׂמְחַת הַשְּׁאוּבָה

וְסַנְהֶדְרֵי גְּדוֹלָה / סְגֻלַּת הַחֲכָמִים:


צְלַח נָא בֶּן גְּבִירָה / בְּרוֹב שִׂמְחָה וְצַהֲלָה

וְשַׁבֵּחַ לְצוּרָךְ / שְׁהוּא אַדִּיר וְנַעֲלָה

וְהָבִין מַעֲשֵׂהוּ / שְׁהוּא נוֹרָא וְנִפְלָא

וְאָז תִּזְכֶּה וְתִרְאֶה / לְיוֹם בּוֹא הַגְּאֻלָּה

תְּבָרֵךְ אֵל וְתוֹדֶה / אֲשֶׁר לוֹ הַגְּדֻלָּה

וְאִם הֵם הַדְּבָרִים / סְתוּמִים וַחֲתוּמִים:
