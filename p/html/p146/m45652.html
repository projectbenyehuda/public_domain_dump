הִכּוֹנוּ בַּעֲלֵי כִּשְׁרוֹן

חתימה: לשבזי


הִכּוֹנוּ בַּעֲלֵי כִשְׁרוֹן / וְחַכּוּ יוֹם גְּאוּלָתֵנוּ

אָז נָשׁוּבָה לְבִצָּרוֹן / וְיִנָּתֵן מַאֲוַיֵּנוּ

לְחַיִּים בְּסֵפֶר זִכָּרוֹן / בְּיוֹם זֶה צוּרִי יִכְתְּבֵנוּ

לְמַעַן תְּסַפְּרוּ לְדוֹר אַחֲרוֹן / כִּי זֶה אֱלֹהִים אֱלֹהֵינוּ:


לַוִּי יַעֲלַת חֵן קְהָלַי / לְהַקְבִּיל חַי הָעוֹלָמִים

קִשְׁרִי בְּאוֹתוֹת לִדְגָלַי / נִקָּבְצָה מִן הָעַמִּים

יָבוֹא מְשִׁיחִי בִגְלָלַי / נָגִיד וּמְצַוֶּה לְאֻמִּים

עוּרוּ וּנְבַקֵּשׁ רַחֲמִים / הָבוּ גֹדֶל לֵאלֹהֵינוּ:


שׁוֹבֵב נְתִיבוֹת בֶּן פֶּרֶץ / נַעֲלֶה הַר צִיּוֹן בְּרִנָּה

שִׁמְעִי וְהַאֲזִינִי אֶרֶץ / וְשִׁיר הַשְׁמִיעִי וְקוֹל רִנָּה

וְכֶלֶב לְשׁוֹנוֹ לֹא יֶחֱרַץ / וְנָחָשׁ נְמִבְזֶה וּמְגֻנָּה

וְאֲיֻמָּתִי כַּשּׁוֹשַׁנָּה / הַאֲזִינוּ תּוֹרַת אֱלֹהֵינוּ:


בּוֹאִי הַכַּלָּה הַכְּלוּלָה / הַקְבִּילִי אֶת פְּנֵי דּוֹדֵךְ

כְּלִילַת יוֹפִי הָאֲצוּלָה / עוּרִי וְהַגִּידִי סוֹדֵךְ

יִשְׂמְחוּן בָּחוּר וּבְתוּלָה / בְּהַשְׁפִּיעֵךְ מִזִּיו כְּבוֹדֵךְ

יִגְדַּל נָא עָלַי חַסְדֵּךְ / בְּעִיר אֱלֹהֵינוּ:


זָרִים זֵדִים יִכָּלְמוּ / וְשָׁבוּ בָּנִים לִגְבוּלָם

יִַעֲשׂוּ תְּשׁוּבָה לוּ חָכְמוּ / יַשְׂכִּילוּ זֹאת בִּכְלָלָם

מְאֹד מַחְשְׁבוֹת צוּרִי עָצְמוּ / בָּחַר בְּעַמּוֹ וְהִצִּילָם

וְאוֹיְבַי כָּאֶבֶן יִדְּמוּ / כִּי אֵין כַּיְוֹיָ אֱלֹהֵינוּ:


יִרְצֶה צוּרִי יְשַׂגְּבֵנִי / יִגְמְלֵנוּ חֲסָדִים טוֹבִים

אָז יֹאבְדוּ כָּל אוֹיְבֵינוּ / שׁוֹלְפֵי חֶרֶב וְהָרוֹבִים

וְיִכָּלְמוּ בְּאַף שׂוֹטְמֵינוּ / הַגְרִים וְעַמּוֹן וּמוֹאָבִים

שׁוּבוּ בָּנִים שׁוֹבָבִים / הַחוֹכִים רַחֲמֵי אֱלֹהֵינוּ:
