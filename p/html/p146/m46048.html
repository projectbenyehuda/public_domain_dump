שְׁבַח אַפְלִיא

חתימה: שלם שץ


שְׁבַח אַפְלִיא / לְצוּר גּוֹאֲלִי / הַנּוֹתֵן לִי חַיִּים

יוֹצֵר כֹּל / בְּרוֹב הַשְׂכֵּל / בְּלִי יְגִיעַת יָדַיִם

גּוּף קָבַע / מֵאַרְבַּע / אֵשׁ וְרוּחַ וּמַיִם

וְהֶעָפָר / בַּמִּסְפָּר / הִרְקִימוֹ בִּנְתַּיִם

וּמְזָגִים / מְשׂוֹרָגִים / יְרֻקִּי וְאַדְמוּתַיִם

וְשָׁחְרָיִי / וְלִבְנִָיִי / אַרְבָּעָה בְּשֵׁוֻיִּים

כָּל הַיְצוּר / הוּא עָצוּר / בְּאַרְבַּע אַרְבַּעְתָּיִם

הֵן גּוֹיִם / כְּמַר מִדְּלִי / וּכְשַׁחַק מֹאזְנַיִם:


לְשֶׁקֶל זֶה / כְּשֶׁקֶל זֶה / לֹא יוֹסִיף וְלֹא יִגְרַע

וְעוֹד נִדְמָה / כְּחוֹמָה / מֻרְכָּב מִטּוֹב וָרָע

וְהַמּוֹחַ / מָשׁוּחַ / שְׁהוּא מֶלֶךְ מְקוֹם זֶרַע

וְלֵב לוֹ שָׂר / לַמּוּסָר / מַלְכוּתוֹ בִּמְאוֹרָע

וְהַכָּבֵד / שָׂר עוֹבֵד / מְחַלֵּק מָזוֹן בְּפֶרַע

וְאֶל הַלֵּב / מִשְׁתַּלֵּב / רְמַח וּשְׁסַה הוּא יִכְרַע

וְהַנְּשָׁמָה / נֶעֱלָמָה / וְנֶפֶשׁ וְרוּחַ חַיִּים:


מְכוֹן כּוֹחוֹת / נוֹכָחוֹת / מַעֲשִׂיִּי וְעִיּוּנִי

וּשְׂכָלִים / נִכְלָלִים / קְרוּאִים הֵם לַהֲמוֹנִי

וְהַזּוֹכֵר / וְהַשּׁוֹמֵר / כּוֹחוֹת לַחִיּוּנִי

וְשֵׂכֶל טוֹב / כְּעֵץ רָטוֹב / הַפּוֹעֵל רַוְחָנִי

שְׁהוּא גּוּף דַּק / אִישׁ צֶדֶק / בְּמַעְלָתוֹ עֶלְיוֹנִי

וְהַדִּמְיוֹן / וְהָאֶבְיוֹן / הוּא כֹּחַ גּוּפָנִי

וּבֶהָמִית / הַנִּדְמִית / בַּבְּהֵמָה וּבַחַיִּים:


שְׁמוֹת כֻּלָּם / לֹא נֶעְלָם / מִשְּׂרִידֵי חֲכָמֵינוּ

וִיסוֹדָם / הוּא נֶגְדָּם / וְהֵמָּה שֶׁהוֹרוּנוּ

בְּכַוָּּנָה / נֶאְמָנָה / הִגִּיהוּ חָשְׁכֵּנוּ

וְעֵת חֶשְׁבּוֹן / הָרִבּוֹן / נֶפֶשׁ תּוֹכִיחֶנּוּ

וְהִיא תַּעַל / מִמַּעַל / וּבַגָּן תּוֹשִׁיבֶנּוּ

מְקוֹם נוֹעַם / בְּלִי זַעַם / וּמֵי חַיִּים תַּשְׁקֵנוּ

וּנְהָרִים / מַנְהִירִים / אַרְבָּעָה נְהָרִים:


צַדִּיקִים / נֶחְשָׁקִים / שְׁמוּרִים שָׁם בְּמָנוֹחַ

וְעַטְרוֹתָם / בְּרֹאשׁוֹתָם / מְקוֹם טוֹבָה וְגַם רֶוַח

מְעוֹפְפִים / כִּשְׂרָפִים / בְּרוֹב אוֹנִים וְגַם כֹּחַ

וְשָׁם אָבוֹת / כְּמֶרְכָּבוֹת / וְכָל צַדִּיק וּמָשׁוּחַ

שְׂכַר כֻּלָּם / בְּסוֹד נֶעְלָם / וְהוּא כָתוּב עֲלֵי לוּחַ

וְשָׁם רֵעִים / וְגַם שִׁבְעִים / וְשָׁם אַהֲרוֹן וְזָנוֹחַ

וַחֲסִידִים / נֶחְמָדִים / הַיְּשָׁרִים וְהַנְּקִיִּים:
