[מָרוֹם אַיֵּה חֲסָדֶיךָ]

סימן: משה קטן


מָרוֹם אַיֵּה חֲסָדֶיךָ / נִשְׁבַּעְתָּ לַחֲסִידֶיךָ

שׁוּבָה אֲדֹנָי עַד מָתָי / וְהִנָּחֵם עַל עֲבָדֶיךָ: [תה' צ, יג]

שׁוּבָה לָעֵדֶר וְצֹאן טֶרֶף / נְתַתּוֹ בְּפִי עוֹבְדֵי תֶרֶף

הַצֵּל פְּלִיטָיו וְאַל תֶּרֶף / יָדֶיךָ מֵעֲבָדֶיךָ: [יהוש' י, ו]

הוֹזִים בְּשִׁבְיָם וְנִרְדָּמִים / עוֹרֵר, סְדוּרוֹת עוֹלָמִים,

חוּס נָא וְנַחֵם שׁוֹמֵמִים / וְדַבֵּר עַל לֵב עֲבָדֶיךָ: [ש"ב, יט, ח]

קוֹמֵם וְהַצֵּב גְּבוּלֶיךּ / וּבְנֵה מְכוֹן בֵּית זְבוּלֶךָ

וְיִשַּע הֲדָרְךָ לְפָעֱלֶיךָ / יֵרָאֵה אֶל עֲבָדֶיךָ: [תה' צ, טו]

טָבוּעַ בְּמֵי חֲרוֹנֶיךָ / דַּל הוּא וְחָרֵד לְדִינֶךָ

יִזְכֶּה וְתִיקַר בְּעֵינֶיךָ / נַפְשִׁי וְנֶפֶשׁ עֲבָדֶיךָ: [מל"ב א,יג]

נְשָׂא חֵטְא מְגֹאָל וְאִם תִּרְחָק / וְקָרֵב בְּקָרְאוֹ וְאַל תִּרְחָק

וְזֵכֶר לְאַבְרָהָם לְיִצְחָק / וּלְיִשְׂרָאֵל עֲבָדֶיךָ: [שמ' לב, יג]
