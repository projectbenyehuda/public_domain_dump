בָּאתִי לְגַנִּי

בָּאתִי לְגַנִּי צְבִיָּה / בָּאתִי לְגַנִּי

לִפְקֹד גֶּפֶן פּוֹרִיָּה / נְטַע יְמִינִי.


עוּרִי עוּרִי לִבְשִׁי עֶדְיֵךְ / מִטּוּב מַעְדַּנִּי

הָסִירִי אֶת שִׂמְלַת שִׁבְיֵךְ / וְלִבְשִׁי שָׁנִי.


זְכוּת נֹפֶת שִׂפְתוֹתַיִךְ / בְּעוֹד תַּעֲנִי

זֶמֶר עַל פְּדוֹתַיִךְ / אֶל רוּם שׁוֹכְנִי.


רֹנִּי וְשִׂמְחִי, שְׁבִיָּה, / עַל כִּי הִנְּנִי

שַׂמְתִּיךְ כְּחוֹתָם, צְבִיָּה, / עַל יַד יְמִינִי.


אֶבְחַר שִׁירֵי רְנָנַיִךְ / מִכָּל הֲמוֹנִי

קָשׁוּר עַל שִׂפְתוֹתַיִךְ / תִּקְוַת חוּט שָׁנִי.


אֵלַיִךְ דֵּעִי אֲחַוֶּה / אַתְּ בַּת אֵיתָנִי

עַד אָנָה בְּתוֹךְ חַגְוֵי / סֶלַע תִּשְׁכְּנִי.


לָךְ צוּרִי אֶזְכֹּר אֶדַּדֵּם / עַל בֵּית מְלוֹנִי

עִמְּךָ כְּיַרְחֵי קֶדֶם / מִי יִתְּנֵנִי.


יָהּ, כַּיּוֹלֵדָה אֶפְעֶה / הָשֵׁב מַעֲנִי

הַגִּידָה לִי אֵיךְ תִּרְעֶה / נָפֹצוּ צֹאנִי.


אֶשְׁרְקָה לָהֶם, אֲקַבֵּץ / אֶל בֵּית מְעוֹנִי

אֲבִיאֵם לִנְוֵה מַרְבֵּץ / מִכְלְאוֹת צֹאנִי.


חָזָק, חִישׁ צְבִי עֹפֶר / מַהֵר וּפְדֵנִי

אָז יִהְיֶה כְּעֵט סוֹפֵר / מַהֵר לְשׁוֹנִי.
