איך תמו מאויי (תוכחה)

אֵיךְ תַּמּוּ מַאֲוַיַּי / וְיוֹם יוֹם יִרְבּוּ חֳלָיַי,

בְּחֶרְפָּה הִכּוּ לְחָיָי – / נָקְטָה נַפְשִׁי בְחַיָּי.

בְּנֵי אֲדָמָה אֵיךְ יְנַכְּרוּ / אוֹתוֹת אֱמֶת, וְלֹא יִזְכְּרוּ

הֱיוֹתָם לְמָחָר מֵתִים, / וְהִנָּם אוֹכְלִים וְשׁוֹתִים?

רָצוּ יְלוּדֵי אִשָּׁה / אַחַר תַּאֲוָה חֲדָשָׁה,

וּבְהֶבֶל יָגִילוּ – / עַל כֵּן לֹא יַשְׂכִּילוּ.

הָלְכוּ גּוֹדְרֵי פָרֶץ / בְּדֶרֶךְ כָּל הָאָרֶץ,

אַחֲרֵי רָאוּ יְמֵי עֳנִי – / וְהָלַכְתִּי גַם אָנִי.

מִי מֶנְהוּ יִסָּתֵר? – / אֲשֶׁר הַגּוּף יִוָּתֵר,

בְּצֵאת רוּח מִקִּרְבּוֹ, / כִּכְלִי אֵין חֵפֶץ בּוֹ?

בְּבֵית סֹהַר הַנּוֹצָר / אֱנוֹשׁ עָצוּר בְּכֹל עָצַר,

סְבָבוּהוּ גַלְגִּלָּיו – / שֻׁלַּח בְּרֶשֶׁת בְּרַגְלָיו.

נִדָּף כְּעָלֶה יָבֵשׁ, / מְבַקֵּשׁ בֶּגֶד לוֹבֵשׁ,

אֲשֶׁר בְּצֵאתוֹ מֵרֶחֶם / נוֹדֵד הוּא לַלֶּחֶם.

עָרוֹם בָּא בְרֵאשִׁיתוֹ, / כֵּן יֵלֵךְ בְּאַחְרִיתוֹ,

אוֹכֵל לֶחֶם עֲצָבִים – / כָּל יָמָיו מַכְאוֹבִים.

זוֹרֵעַ שָׁוְא יִקְצֹר / רִיק, וְאֵיךְ כֹּחַ יַעֲצֹר?

כְּקָנֶה וָסוּף קָמַל, / כְּצִיץ יָצָא וַיִּמָּל.

רוּחַ אִישׁ בִּלְבָבוֹ / יְסוֹד הַגּוּף לְהַצִּיבוֹ,

דּוֹמֶה לְשַׂר הַצָּבָא – / כִּי הוּא יוֹצֵא וָבָא.

הֵיכְלֵי עֹנֶג גְּוִיּוֹת / מְלֵאוֹת כָּל מַשְׂכִּיּוֹת,

בְּבוֹא הַמָּוֶת אֲלֵיהֶם / סָר צִלָּם מֵעֲלֵיהֶם.

חָק נָתַן לְבֶן אֲדָמָה / כַּמָּה יַעֲמֹד שָׁמָּה,

וּקְצָרִים הֵם עֲלוּמָיו / וְכִימֵי שָׂכִיר יָמָיו.

זֶה סוֹף כָּל הָאָדָם / אֲשֶׁר יִשְׁכַּב וְיֵרָדַם

וְנִשְׁחַת גּוּף מַה נָּעִים / וַיָּרֻם תּוֹלָעִים.

קִדֵּשׁ כְּבוֹד-שָׂם-דָּרֶךְ, / בְּאֵל אָמֵן יִתְבָּרֵךְ,

אָז יֹאמַר: אֵלִי זֶה, / מַחֲזֶה שַׁדַּי יֶחֱזֶה.
