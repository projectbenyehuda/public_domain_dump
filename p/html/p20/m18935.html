אַלְלַי אַלְלַי צָרִי

קינה לט׳ באב


אַלְלַי אַלְלַי, / צָרִי / סָמַךְ עָלַי!

אַלְלַי אַלְלַי, / צוּרִי / סָר מֵעָלַי!


אָמַר אוֹיֵב לְהָתֵל בִּי: / ׳עַם יְיָ אֵלֶּה –

הֲלֹא פִזַּרְתִּים בְּחַרְבִּי, / לֹא אֲדֹנִים לָאֵלֶּה!׳

אֵיךְ יַעְמֹד לִבִּי / וְתִקְרֶאנָה אֹתִי כָּאֵלֶּה?

הֲלֹא עַל כִּי אֵין אֱלֹהַי בְּקִרְבִּי / מְצָאוּנִי הָרָעוֹת הָאֵלֶּה.


בָּנֶיךָ, אֵיךְ תְּשִׁיתֵמוֹ / כִּכְלִי אֵין חֵפֶץ בּוֹ?

וּבַיִת בָּנוּ אֲבוֹתֵימוֹ, / אֱלֹהֵי יִשְׂרָאֵל בָּא בוֹ –

הֶחְרִיב צַר בְּיוֹם אֵידֵמוֹ, / שׁוּעָלִים הִלְּכוּ בוֹ,

וְאָמַר: ׳אֵי אֱלֹהֵימוֹ / צוּר חָסָיוּ בוֹ?׳


רָדָה הַצַּר וְהֵנִיף חַרְבּוֹ / בְּהֵיכַל קָדְשׁוֹ –

וְשָׁב הַמֶּלֶךְ לִמְסִבּוֹ / בִּמְעוֹן קָדְשׁוֹ.

וְאֵיךְ נָכְרִי כָּל מוּם בּוֹ / יָקוּם בִּמְקוֹם קָדְשׁוֹ?

אַיֵֵּה הַשָּׂם בְּקִרְבּוֹ / אֶת רוּחַ קָדְשׁוֹ?


מַה יַּעֲשׂוּ זְרוּיִם / בְּאֶרֶץ לֹא לָהֶם?

אוֹ מַה יַּעֲנוּ שְׁבוּיִם / לִפְנֵי שׁוֹבֵיהֶם –

וּלְחָרֵף אֱלֹהִים חַיִּים / שַׁתּוּ בַשָּׁמַיִם פִּיהֶם?

לָמָּה יֹאמְרוּ הַגּוֹיִם: / ׳אַיֵּה נָא אֱלֹהֵיהֶם?׳


הַנּוֹתֵן בַּיָּם דָּרֶךְ וּבְמַיִם עַזִּים נְתִיבָה,

נְקֹם אֶת נִקְמָתֵנוּ וְאֶת יְרִיבֵנוּ רִיבָה,

וְאֶת אַחְרִיתֵנוּ מֵרֵאשִׁיתֵנוּ הֵיטִיבָה!

הֲשִׁיבֵנוּ יְיָ אֵלֶיךָ וְנָשׁוּבָה.
