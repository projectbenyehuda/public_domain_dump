[אָז בַּעֲלוֹת מְקֻטֶּרֶת מֹר]

פדות


אָז בַּעֲלוֹת מְקֻטֶּרֶת מֹר / רֵיחָהּ לְמַעְלָה

הִתְרַפְּקָה בְּמֵיטַב טַעַם / עַל דּוֹד כְּיַעֲלָה.


בַּלָּט צְבִי מְשָׁכָהּ בִּימֵי / נֹעַר וְנָהֲרָה

אֵלָיו כְּרוּץ צְמֵאִים אֶל מֵי / נָהָר, וְעוֹרֲרָהּ.

לִבָּהּ יְסָעֲרוּ בוֹ יַמֵּי / חִשְׁקָהּ וְאָמְרָה

"יוֹאֵל יְדִיד רְצוֹנוֹ לֵאמֹר – / אִמְרוֹ אֲקַבְּלָה,

מִצּוּף לְפִי בְּשַׁלְוָה יִנְעַם / וּצְרִי בְמַחֲלָה".


רֹאשָׁהּ לְכָל עֲדִינֹות הָיְתָה / בִּגְאוֹן מְחוֹבֲבָהּ,

רַכָּה בְצֵל כְּנָפָיו חָיְתָה, / עַד נָד יְדִיד וּבָהּ

בָּעַל, וְאֶל דְּרָכִים צָפְתָה / מָתַי יְשׁוֹבֲבָהּ.

לַיְלָה כְסִיל וְכִימָה תִשְׁמֹר / תַּעֲמדֹ וְשָׁאֲלָה

בַעֲדוֹ, וְאִם יְאַחֵר פַּעַם – / תִּדֹּם וְיִחֲלָה.


"מַה תִּפְחֲדִי וְלִבֵּךְ זָחַל, / רַעְיָה מְאַהֲבָה?

אַקְדִּישׁ שְׁמִי וּמִקְדָּשׁ נִחָל / צִיר אֶשְׁלְחָה, וּבָא

גּוֹאֵל לְגָאֳלֵךְ כִּי אֶנְחַל / אוֹתָךְ, וְאוֹיְבָה

אִם גִּדְּפָה שְׁנֵינוּ יִגְמֹר / הַקֵּץ וּמִלֲלָה:

זֶה אֵל אֱמֶת וְאַשְׁרֵי הָעָם / בָּחַר לְנַחֲלָה!"
