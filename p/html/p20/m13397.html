אַתֶּם שְׁאֵרִית בַּעֲלֵי הַבְּרִית

אַתֶּם שְׁאֵרִית / בַּעֲלֵי הַבְּרִית / גּוֹרָל אֱחָד לַייָ,

נְכוֹנִים הֱיוּ – / אוּלַי תִּחְיוּ? – / וְהָכִינוּ לְבָבְכֶם אֶל יְיָ,

יַחַד לְרוֹמְמוֹ / וּלְהַקְדִּישׁ שְׁמוֹ / מִלְאוּ יֶדְכֶם הַיּוֹם לַייָ.

אֵין זוּלָתוֹ / וְהַכֹּל פְּעֻלָּתוֹ / כִּי מִי בַשַּׁחַק יַעֲרֹךְ לַייָ?

בְּשֵׁמוֹת נִקְרְאוֹ / בְנֵי אֵלִים, בָּאוּ / לְהִתְיַצֵּב עַל יְיָ,

רָצִים אַחֲרָיו / עוֹשֵׂי דְבָרָיו / לִשְׁמֹע בְּקוֹל יְיָ.

הֲמוֹנִים הֲמוֹנִים / לְעָבְדוֹ נְתוּנִים, / כֻּלָּם קְדשִׁים וּבְתוֹכָם יְיָ,

מֵרֹב מוֹרָא / זֶה אֶל זֶה יִקְרָא / לִקְרֹא כֻלָּם בְּשֵׁם יְיָ,

יָשִׁירוּ בְדַרְכֵי יְיָ / כִּי גָדוֹל כְּבוֹד יְיָ.

ככתוב וקרא זה אל זה ואמר
