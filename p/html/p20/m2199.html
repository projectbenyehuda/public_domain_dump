[אני תמה יפת צמה]

אהבה


אֲנִי תַמָּה / יְפַת צַמָּה – עֲדֶן שָׂמָה / בְּיַד שַׁמָּה, / וְהַחַמָּה

שְׁזָפַתְנִי, / שְׁאָפַתְנִי, / שְׂרָפַתְנִי; / זְכֹר, דּוֹדִי, בְּרִיתֶךָ!

נִפְלְאַתָּה לִי אַהֲבָתֶךָ.

בְּבוֹר שִׁבְיִי, / בְּכוּר עָנְיִי / בְּרֹב מִרְיִי, / עֲדֶן בִּכְיִי / עֲלֵי לֶחְיִי,

וְאַתְּ עֶזְרִי, / בְּךָ שִׂבְרִי, / חֲבֹשׁ שִׁבְרִי / וְכָל שֶׁבֶר עֲדָתֶךָ!

רְאֵה לַחְצִי, / רְפָא מַחְצִי, / גְּדֹר פִּרְצִי / וּמִיָּם צִי / לְאֹור תּוֹצִיא,

תְּקִיֵמִני, / תְּרִימֵנִי, / תְּשִׂימֵנִי / לְרֹאשׁ כִּדְבַר שְׂפָתֶיךָ.

הֱיֵה סַלְעִי, / בְּיוֹם צַלְעִי / קְרָא יִשְׁעִי / וְסָר נִגְעִי, / וּמַכְנִיעִי

בְּאַף תַּהְרֹס / כְּגוֹי פַּתְרוֹס, / אֲזַי תִּפְרֹשׂ / כְּנָפְךָ עַל אֲמָתֶךָ.

מְנַת חֶלְקִי / וְצוּר חִזְקִי / וְצוּף מָתְקִי / וְכָל חִשְׁקִי, / בְּךָ דָּבְקִי,

חֲצוֹצְרוֹתַי, / נְגִינוֹתַי / וְשִׁירוֹתַי / אֲנַגֵּן עוֹד בְּבֵיתֶךָ.
