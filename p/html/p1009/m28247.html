אָז בַּחֲלוֹם לְרֹאשׁ הַר הַמּוֹר

אָז בַּחֲלוֹם לְרֹאשׁ הַר הַמּוֹר / תָּשׁוּר וּפִלְּלָה

שִׂיאָהּ וְאֵיךְ זְהָבָהּ יוּעַם / שַׁחָה וְשָׁפְלָה.


בֹּקֶר וְשָׁאֲלָה כָל חַכְמֵי / לֵיבָב וְזָכְרָה

עָנְיָהּ וְסוֹעֲרָה וּלְחוּמֵי / רֶשֶׁף וּפָעֲרָה

לֶחְיָהּ וְנִגְּרוּ בָהּ זִרְמֵי / עֵינָהּ וְגָבְרָה

מִיּוֹם לְיוֹם תְּקַו לְשִׂיחָה תִימוֹר / תִּשְׁכּוֹן בְּאָהֳלָהּ

בֶּטַח בְּעֹז וְלֹא תִפָּעֶם / רוּחָהּ וְתִסְבְּלָה.


נַחֵם לְבַב אֲסוּרָה בָיְתָה / רָצוֹן תְּסוֹבְבָהּ

כִּמְעַט וְקָט וְהֵן נִשְׁחָתָה / הוֹאֵל וְקָרְבָהּ

תִּמְשֹׁל בְּעֹז לְךָ יָאָתָה / עַל כֹּל תְּשַׂגְּבָהּ

תַּשְׁבִּית לְפוֹרְשֵׂי לָהּ מִכְמוֹר / תִּקַּח כְּפָעֳלָהּ

אֵיךְ תִּזְעֲמָה אֲשֶׁר לֹא תִזְעֹם / אָבְלָה וְנָבְלָה.


רִאשׁוֹן לְדִבְּרוֹת אָז הוּחַל / כִּי לֹא אֲכַזְּבָה

עֵת תִּדְרְכִי זְאֵב עִם שַׁחַל / נֵצַח וְרָחֲבָה

בִּירָה וְחֵטְא וְעָוֹן נִמְחַל / לִמְאֹד וְנָסְבָה

בָּהּ תַּעֲנִי בְּשִׁיר עִם מִזְמוֹר / וּשְׁמִי אֲגַדְּלָה

אֶתְקַדְּשָׁה בְאִמְרֵי נֹעַם / שִׂמְחָה וְצַהֲלָה.
