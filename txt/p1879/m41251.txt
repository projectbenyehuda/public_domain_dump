


	
	צֹאן מַרְעִיתִי
	



סימן: צאלח. מנגינה: חוסיני.






צֹאן מַרְעִיתִי / אָשִׁיב אִתִּי / אֱלֵי בֵיתִי / וּלְגַנִּי



אֶבְנֶה בֵית אֵל / וְהַהַרְאֵל / וְהָשֵׁב אֶל / מְכוֹנִי



נִסְכִּי וְקָרְבָּנִי / וְאֶשְׁכֹּן גַּם אֲנִי / בֵּין בַּדֵּי אֲרוֹנִי

הָרַי הָרַי הָרַי הָרַי / עַנְפְּכֶם וּפֶרְיְכֶם / תִּשְׂאוּ לְיִשְׂרָאֵל בְּנִי.





אָבִי רְאֵה / אֲנִי נִכְאֶה / מִכָּל בָּאֵי / מְעוֹנִי



אָכְלוּ אָכוֹל / פְּרִי אֶשְׁכֹּל / וְנִשְׁכַּח כָּל / שְׂשׂוֹנִי



שׁוּר כִּי לְחָצוּנִי / וְגַם מְחָצוּנִי / מִיָּדָם פְּדֵנִי

בִּזְכוּת הוֹרַי הוֹרַי הוֹרַי / בִּמְהֵרָה הֱיֵה עֶזְרָה / לְבֶן מֶלֶךְ נִשְׁאַר עָנִי.







לָמָּה תִזְנַח / אוֹתִי נֶאְנָח / כִּי אוֹיְבִי נָח / בִּמְכוֹנִי



עַד אָן גִּבּוֹר / עַצְמִי תִשְׁבֹּר / וּבְתַחְתִּית בּוֹר / שַׁתַּנִי



מֵאָז אֲהַבְתַּנִי / מִבּוֹר מִלַּטְתַּנִי / עַתָּה נְטַשְׁתַּנִי

בְּיַד עָרַי עָרַי עָרַי / חֹן עָלַי וְעַל עוּלַי / בִּישׁוּעָתְךָ הַרְאֵנִי.





חַלֵּץ אוֹתִי / מִגָּלוּתִי / בִּתְפִלָּתִי / עֲנֵנִי



אֶצְלִי חֲנֵה / בֵּיתִי בְנֵה / וְתֵן מַעְנֶה / בִלְשׁוֹנִי



שׁוּב אֶל תּוֹךְ מִשְׁכָּנִי / וְשָׁם תַּנִּיחֵנִי / וְאַתָּה תִרְעֵנִי

וּבְנֵה עָרַי עָרַי עָרַי / דּוֹד, אֵל אָיוֹם, חִישׁ לִי פִדְיוֹם / בְּצִדְקָתְךָ נְחֵנִי.
