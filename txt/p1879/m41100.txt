


	
	אֲשׁוֹרֵר בְּשִׁירִים
	



שיר שחיבר להזקן הנכבד סלימאן דוד מעתוק

סימן: אני צאלח






אֲשׁוֹרֵר בְּשִׁירִים / שְׁקוּלִים סְפוּרִים / לְשַׂר רֹאשׁ חֲבֵרִים / גְּבִירִים יְקָרִים



נְפָשׁוֹת מְדֻכִּים / אֲנוּשׁוֹת וּמֻכִּים / דְּבָרַָיו מְחַכִּים / וְיִגְהוּ מְזוֹרִים



יְמַלֵּט גְּוִיּוֹת / לְמָוֶת שְׁבוּיוֹת / יְשַׁו כָּאֲרָיוֹת / לְמֵתִים פְּגָרִים



צְמָחִים וּמַכּוֹת / יְבֵשׁוֹת וְרַכּוֹת / בְּגוּפוֹת עֲרוּכוֹת / יְרַפֵּא וְיָרִים



אֲבִי הַמְּלִיצוֹת / וְחָכְמוֹת וְעֵצוֹת / וְיַשְׁבִּית לְמַצּוֹת / וְיַמְתִּיק מְרוֹרִים



לְכִי נָא בְרִנָּה / מְלִיצָה הֲגוּנָה / לְיַד אִישׁ תְּבוּנָה / וְחוֹקֵר דְּבָרִים



חֲלִיפַת תְּשׁוּרָה / וּמִנְחָה הֲדוּרָה / וְנַפְשִׁי מְכוּרָה / לְעֶבֶד לְדוֹרִים.
