


	
	לִבִּי יָחִיל בְּקִרְבִּי
	








לִבִּי יָחִיל בְּקִרְבִּי / מִיגוּרֵי יוֹם קְפָדָה

וְאֵימוֹת נָפְלוּ עָלַי / וְנַפְשִׁי מְאֹד חֲרֵדָה.

יָרֵאתִי פֶּן תִּהְיֶה / רַגְלִי בְּחֶטְאִי לְכוּדָה –

צִירִים אֲחָזוּנִי / כְּצִירֵי יוֹלֵדָה.





חָשַׁךְ מְאוֹר עֵינֵינוּ / בְּאֶרֶץ צָרָה וְצוּקָה

וְאֵיכָה נָרִים מֵצַח / אִם לֹא לָנוּ צְדָקָה?

וְחַטֹּאתֵינוּ הָיוּ / לָנוּ לְמִכְשׁוֹל וּפוּקָה!

לָכֵן יִסַּרְתָּנוּ / וְשַׂמְתָּ בְגָלוּת דְּחוּקָה

בְּמָתְנֵינוּ מוּעָקָה, / הֲבֵאתָנוּ בַמְּצוּדָה.





זְדוֹנֵנוּ אִם נִגְלָה – / שִׂימָה סִתְרְךָ מְחוֹפֵף

וּלְחַם מִלְחֲמוֹת תְּנוּפָה / בֶּאֱדוֹם אֲשֶׁר יָד יְנוֹפֵף:

לָבִיא וְלַיִשׁ מֵהֶם / אֶפְעֶה וְשָׂרָף מְעוֹפֵף,

כְּזֶרֶם מַיִם שׁוֹטֵף / עַד הַנֶּפֶשׁ אוֹפֵף,

וְנוֹגְשִׂים אָצִים וְלוֹחֲשִׁים, / וַתִּכְבַּד הָעֲבוֹדָה.





קָרוֹב לְכָל הַקּוֹרְאִים / וְשָׁבִים בַּצַּר אֵלָיו,

פְּדֵה עַמְּךָ אֲשֶׁר נִשְׁאַר / כְּאִילָן נֶחְמְסוּ עָלָיו,

וְקָמִים מִכָּל עֵבֶר / כְּאַרְבֶּה נִקְבְּצוּ עָלָיו.

כְּרַחֵם אָב עַל בָּנִים – / רַחֵם עַל עוֹלָלָיו,

כְּנֶשֶׁר יָעִיר קִנּוֹ / יְרַחֵף עַל גּוֹזָלָיו,

וְהָשֵׁב שְׁבוּת אֹהָלָיו / וְחֹן שְׁכוּלָה וְגַלְמוּדָה.





וּרְאֵ כִּי בְעָלוּנוּ / זוּלָתֶךָ אֲדוֹנִים

וְשָׁחֲרוּ עַד כִּי קִבְּצוּ / פָארוּר כָּל הַפָּנִים.

נָא הַשְׁפֵּל עַקְרַבִּים / וְסָרָבִים וְסַלּוֹנִים

וְקָרֵב חֶזְיוֹן ״וְשִׁעֲשַׁע / יוֹנֵק עַל חֻר פְּתָנִים

וְעַל מְאוּרַת צִפְעוֹנִים / גָּמוּל יָדוֹ הָדָה״.
