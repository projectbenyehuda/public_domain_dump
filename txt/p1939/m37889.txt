


	
	פני משתנים
	







זולת. שיר תלונה על שעבוד הגלות. סימן: אפרים ברבי יצחק.





פָּנַי מִשְׁתַּנִּים / יְגוֹנִי לֹא נִכְחָד / הֵן בֵּיתִי שׂוֹטְנִים / שָׁלוֹם מִפָּחַד

הַאִם כֹּפֶר נוֹתְנִים / לְךָ הָאֵל הָאֶחָד / וְכִי תִשָּׂא פָנִים / וְכִי תִקַּח שֹׁחַד.





רָשָׁע מִתְגָּאֶה / עַל עַם הָעֲמוּסִים / בְּהוֹדִי מִתְגָּאֶה / וְשָׂם עָלַי מִסִּים

וְכָל אִישׁ מִשְׁתָּאֶה / לְרִבּוּי הַנּוֹגְשִׂים / הַאֵינְךָ רוֹאֶה / מָה הֵמָּה עוֹשִׂים.





יְשַׁלְּחוּ רֵיקָם / עֲנִיִּים מֻכֵּיהֶם / יֹאכְלוּ אֶת חֻקָּם / כְּדָתֵי חֻקֵּיהֶם

שָׁבוּ אֶת חֶלְקָם / בְּכֹחַ עוֹשְׁקֵיהֶם / מְצִיקִים לְהַדִּקָּם / וּמְרִיקִים שַׂקֵּיהֶם.





מָה רַבּוּ רוֹדְפִים / רַבִּים עָלַי קָמוּ / וּמֵאֵימַת חוֹרְפִים / שְׂפָתַי נֶחְסָמוּ

בְּרֹאשׁ דַּלִּים שׁוֹאֲפִים / וְאוֹתוֹתָם שָׂמוּ / עוֹבְדֵי הַתְּרָפִים / קָסֹם יִקְסֹמוּ.





בִּרְבוֹת מִפְלַצְתָּם / מִשְׁטָרָם מְיֻתָּר / עָלַי כָּל עֲצָתָם / וְכָרוּ לִי מֶחְתָּר

לְמַסָּע קִיצָתָם / הֵן לִבִּי יִתַּר / הִנֵּה כָל עֲלִיצוּתָם / לֶאֱכֹל עָנִי בַּמִּסְתָּר.





יָשַׁבְתִּי בַגּוֹלָה / וּבְאֹרֶךְ עַבְדּוּת / נָשִׁיתִי גְאֻלָּה / וְיָאַשְׁתִּי מִן פְּדוּת

הִנֵּה בֶּן עַוְלָה / נַעֲוַת הַמַּרְדּוּת / מוֹשֵׁל בְּבֶן סְגֻלָּה / וַיְנַהֲגֵהוּ בִּכְבֵדוּת.





יְשׁוּעָה תָאַבְתִּי / וּמִמֶּנִּי רָחָקָה / חֹסֶן חֶדְוָתִי / עָבְרָה וְנִמְחָקָה

בְּתוֹחַלְתִּי נִכְזַבְתִּי / וּבִלְשׁוֹנִי חַכָּה / לָעֲגָה אוֹיַבְתִּי / וְלָמָּה זֶה צָחָקָה.





צַמֵּק אֶת שְׂפָתָהּ / וְקָרֵב חֲבָלֶיהָ / לְאֶרֶץ עֵיפָתָה / תִּדְחֶה לְרַגְלֶיהָ

גַּלֵּה חֶרְפָּתָהּ / וְטֻמְאָתָהּ בְּשׁוּלֶיהָ / חַרְבְּךָ הֵנַפְתָּ / עָלֶיהָ וַתְּחַלֲלֶהָ.





חוֹמֵס אֶת נַפְשִׁי / כַּמַּיִם פָּחָז / מָלַק אֶת רֹאשִׁי / בְּחִצּוֹ הַנִּשְׁחָז

בְּחַרְבּוֹ הֶחֱלִישִׁי / וּבְעָרְפִּי אָחָז / מַלְכִּי וּקְדוֹשִׁי / צֵא נָא לִקְרַאת אָחָז.





קוּמָה לִי לִישׁוּעָתָה לְכָה / וְקָרֵב אֶת קִצִּי / הַגְּאֻלָּה הִיא לְךָ / לְמוֹעֲדִים וָחֵצִי

תָּשִׁיב הַמְּלוּכָה / וּמִבּוֹר זֶה תּוֹצִיא / שַׁלְּחֵנִי וְאֵלְכָה / אֶל מְקוֹמִי וּלְאַרְצִי.
