


	
	אֲנִי אָמַרְתִּי אָבוֹא בִתְפִלָּה
	




Abraham da modena


[נולד במוֹדֶנָה בשנת 1511, חי במַנְטוֹבָה, אָרָצוֹ, פֵֿרָרָה, פֿוֹרְלִי ובערים אחרות. בשנת 1575 היה עוד בחיים]





תְּפִלָה


אֲנִי אָמַרְתִּי אָבוֹא בִתְפִלָּה / לְפָנֶיךָ אֵל
נוֹרָא תְהִלָּה / כִּי אֵין לִי מְנוּחָה / רַק בְּעֵת שָׁפְכִי שִׂיחָה /
בְּאֵימָה לִפְנֵי כִסֵּא כְבוֹדֶךָ / וּבִתְחִנָּה אָבוֹא עָדֶיךָ /
בְּאָמְרִי עֲנֵנִי נָא אֵלִי אֵלִי / הֲבִיאֵנִי אֶל־שִׂמְחַת גִּילִי /
לֵישֵׁב בְּבֵיתְךָ לְאֹרֶךְ יָמִים / עִם יְרֵאֵי שִׁמְךָ הַחֲכָמִים /
לַעֲסֹק תָּמִיד בְּדִבְרֵי תוֹרָה / בְּפִלְפּוּל הַתַּלְמִידִים
וּסְבָרָה / לְקַיֵּם מִצְוֹתֶיךָ בֶאֱמוּנָה / וְלָתֵת לְעָנִי וְאֶבְיוֹן
מָנָה / מִשָּׁמַיִם מִמִּדַּת רַחֲמֶיךָ / בָּרְכֵנִי אֱלֹהַי
בִּשְׁלוֹמֶךָ / וְהַרְאֵנִי בְנֹעַם הַתַּאֲוָה / לִקַּח תּוֹרָה בְגִילָה
וְחֶדְוָה / אוֹתָהּ לְבַד לִבִּי מְבַקֵּשׁ / וְעַל־שַׁעֲרֵי רַחֲמִים
נוֹקֵשׁ / לְפָנֶיךָ אֲדוֹן הַתַּחֲנוּנִים / זֶה כַמָּה וְכַמָּה שָׁנִים
/ יְיָ הוֹשִׁיעָה בְּיוֹם קָרְאִי / עֲנֵנִי מְהֵרָה טֶרֶם קָרְאִי /
לְפָנֶיךָ בְּרוּחַ נִשְׁבָּרָה / לְבַקֵּשׁ מְחִילָה וְכַפָּרָה / עַל־רֹב
חַטּאֹתַי / פְשָׁעַי זְדוֹנוֹתַי וַעֲוֹנוֹתַי / אַל־תִּזְכֹּר עֲוֹן
רִאשׁוֹנִים / וְגַם לֹא הָאַחֲרוֹנִים / הַרְאֵנִי אֶת־ חַסְדְּךָ
לְעוֹלָם / וְאַל־אָשׁוּב כְּדַךְ נִכְלָם / וְאֶת־תְּפִלּוֹתַי שֶׁרַבּוּ
/ לְפָנֶיךָ כְּאִשֶּׁה יֵחָשֵׁבוּ / עִם עָרְכִי בְקוֹל שׁוֹעוֹת /
וְעַל־לְחָיַיִם דְּמָעוֹת / תִּהְיֶינָה כֻלָּנָה מְקֻבָּלוֹת /
תּוֹבַלְנָה לִפְנִים בְגִילוֹת / וְרַבִּים מֵאֲנָשִׁים הַנֶּאֱנָחִים /
יֵשְׁבוּ עִמִּי שָׂשִׂים וּשְׂמֵחִים / וְאֹמַר בְּלֵב שָׂמֵחַ וְטוֹב /
הוֹדוּ לַיְיָ כִּי טוֹב.
