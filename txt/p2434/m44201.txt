


	
	חַזֵּק בֶּדֶק
	




חַזֵּק בֶּדֶק / הָדֵק חֶדֶק / כְּרֹת וּקְצֹר

אַל תִּבְטְחָה / כִּי תִמְרְחָה / נִגְעֵי מָצוֹר

גָּרֵשׁ פֶּשַׁע / לֶחֶם רֶשַׁע / אַל תֶּאֱצֹר

קוּם מְהֵרָה / וְהָעִירָה / וְאַל תַּעְצֹר

לִדְרֹשׁ פִּדְיוֹם / מֵאֵל אָיוֹם / כִּי הוּא יָשֹׁר

דַּע וְהָבֵן / לֹא כְיָבִין / מֶלֶךְ חָצוֹר

אַל תִּתְהַלֵּל / כְּאִישׁ הוֹלֵל / וְלֹא תִפְצֹר

בְּאֵל קוֹנָךְ / הוּא יְחָנָּךְ / וְלֹא יִצֹּר

עָרְכָה שִׁירָה / לוֹ זַמְּרָה / עֲלֵי עָשׂוֹר

כָּל מִשְׁאָלָךְ / וּמַעְגָּלָךְ / אֵלָיו תִּמְסֹר

הוּא יֵטִיב לָךְ / כָּל מִפְעָלָךְ / בְּלִי מַחְסוֹר

אָנָה תִבְרַח / אָן תֶּאֱרַח / בְּעִיר מָצוֹר

בְּכָל דִּירָה / שָׁם הָאוֹרָה / עִמּוֹ יָשֹׂר

הַבֵּט לַזְּמַן / אֵין נֶאֱמָן / וְלֹא יִנְצֹר

אַהֲבָתָךְ, / לְשַׁחֲתָךְ / יִרְדֶּה יַעְצֹר

עַל כֵּן רְדֹף / טוֹב, וְתַהְדֹּף / רַע, וְתִבְצֹר

אוּלַי תִּזְכֶּה / וְלֹא תֻכֶּה / כְּמֶלֶךְ צוֹר

כִּי אֵל צוּרָךְ / הוּא מָעֹז לָךְ / כְּמִשְׂגַּב צוֹר.
