


	
	שִׂבַּרְתִּי לִתְשׁוּאוֹת
	



סימן: ששון ישראל.






שִׂבַּרְתִּי לִתְשׁוּאוֹת / הֲמוֹנֵי נִפְלָאוֹת / אֱלֹהִים צְבָאוֹת / גּוֹאֲלִי, מָעוֹז בְּצָרָה.

שׁוּב יָשׁוּב לְהַרְאוֹת / עוֹד עֹז וְנוֹרָאוֹת / יַעְשֶׂה לְטוֹבָה אוֹת / לְהַפְלִיא, בְּעֹז וּגְבוּרָה.

וְכִימֵי צֵאת מִבּוֹר / חָגַר עֹז כְּגִבּוֹר / קָרַע הַיָּם לַעְבֹר / קְהָלִי, לְאוּם נִבְחָרָה.

נָשְׂאוּ עַיִן לִגְמֹר / הַלֵּל שִׁיר וּמִזְמוֹר / וַיֹּאמְרוּ לֵאמֹר / זֶה אֵלִי, לוֹ אֲזַמֵּרָה.

יָרָה בַחֲרוֹנוֹ / תּוֹךְ הַיָּם וּשְׁאוֹנוֹ / פַּרְעֹה וַהֲמוֹנוֹ / כִּי צָר לִי, בְּאַף וְעֶבְרָה.

שָׁבַר גְּאוֹן זֵדִים / הַפּוֹשְׁעִים וּמוֹרְדִים / הָיוּ עוֹלִים וְיוֹרְדִים / כֶּעֱלִי, וּכְמֹץ בִּסְעָרָה.

רֻבֵּי מֵי זְדוֹנִים / עָבְרו בַחֲרוֹנִים / יָרְדוּ כַאֲבָנִים / עַד לִבְלִי, בְּנֶפֶשׁ מָרָה.

אֲזַי שָׁתוּ מֵי רֹאשׁ / בַּהֲמוֹן יָם לִגְרֹשׁ / כָּל לֵבָב וְכָל רֹאשׁ / לָחֳלִי, פֶּצַע וְחַבּוּרָה.

לָזֹאת זָעֲקוּ מַר / זֶה אֶל זֶה וְאָמַר / הֵן כָּל גּוֹיִם כְּמַר / מִדְּלִי, וּכְרוּחַ עָבְרָה.

כָּל שַׂרְפֵי מְעוֹנָה / הֵם בִּקְּשׁוּ רִאשׁוֹנָה / לָשֵׂאת שִׁיר וְרִנָּה / לְצוּר לִי, מָגֵן וְסִתְרָה.

אָכֵן נָאַם הָאֵל / צוּר פּוֹדֶה וְגוֹאֵל / אָז יָשִׁיר יִשְׂרָאֵל / עַם חֶבְלִי, לִי אֶת הַשִּׁירָה.

אוּלַי צוּרִי יֵעָתֵר / לִי וְלֹא יִסָּתֵר / יְנַתֵּק וְיַתֵּר / כַּבְלִי, וְיַזְרַח אוֹרָה.

יִקְבֹּץ שֶׂה פְזוּרָה / יַל יְדֵי אִישׁ מִשְׂרָה / אֶל נְוַת תִּפְאָרָה / גּוֹרָלִי, עִיר הַבְּצוּרָה.
