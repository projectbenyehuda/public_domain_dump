


	
	לְשֹׁד עִירִי
	







קינה בצורת שיר אזור. חתימת השם: יצחק ברבי ששת חזק.






לְשֹׁד עִירִי / גָּדוֹל כַּיָּם שִׁבְרִי, / וְלַחְמִי רֹאשׁ וּתְלָאָה.





       יְחִתֵּנִי / צוּר חֶלְקִי וּלְבָבִי,

       יְקִימֵנִי / לְמַטָּרָה שֶׁל אוֹיְבִי,

וְתַחְתֵּנִי / אֶרְגַּז, וּבִבְשָׂרִי / אֵין מְתֹם, וְנֶגַע נִרְאָה.





       צְעָדַי צָד / מִלֶּכֶת בִּרְחוֹבוֹתַי,

       וְשָׂם בַּסַּד / פַּעֲמֵי מַרְכְּבוֹתַי,

וְהֵן יִסַּד / בְּצִיּוֹן אֶל עִירִי / פִּנַּת יִקְרַת מְסֻלָּאָה.





       חֲמוּדוֹתַי / לְמַרְגִּיזֵי אֵל לְנַחֲלָה,

       וְדִמְעוֹתַי / לְמַכָּתִי כִּי נַחְלָה,

וּבְחוֹמוֹתַי / וּבְבֵיתִי, כְּבוֹד יַעְרִי, / בָּאוּ רַבִּים וְנִכְאָה.





       קְרֹא אֵבֶל / וּמִסְפֵּד – אֶל יוֹדְעֵי נְהִי,

       וְצִיר וְחֵבֶל – / אֶל כָּל עוֹבֵר לוֹ יְהִי

כְּלֵי נֶבֶל / לְקוֹל בּוֹכִים וְכִנּוֹרִי, / לְנַחֲלָתִי כִּי נִלְאָה.





       בְּרִיחֶיהָ / בַעֲדִי אַרְצִי לְעוֹלָם,

       פְּתָחֶיהָ / נִסְגְּרוּ וְאָנוּ בְאֶבְלָם,

צְמָחֶיהָ / עָבָשׁוּ, וְעִיר הֲדָרִי / נִגְאֲלָה וְהוֹי מוֹרָאָה!





       רְאוֹת נָקָם / אֲקַו בְּשֵׂעִיר וּבְעַמּוֹן,

       לְהַתִּיקָם / לְיוֹם הֲרֵגָה, שְׁכוֹל וְאַלְמוֹן,

גְּמוּל לְחֵיקָם, / זְרוֹעַ אֵלִי עוּרִי, / בַּמַּמְלָכָה הַחַטָּאָה!





       בַּהֲגִיגִי / תִבְעַר אֵשׁ בִּקְרָבַי.

       לְמַבְלִיגִי / שֹׁד וְשֶׁבֶר בְּכָל סְבִיבַי,

אֶזְכְּרָה נְגִי/נָתִי עַרְבִּי וְשַׁחְרִי – / וָאֶשְׁתּוֹמֵם עַל הַמַּרְאָה.





       יָהּ, הֲקָצוֹר / קָצְרָה יָדְךָ מִפְּדוּת?

       וְצַר יַעְצֹר / בְּעַמֶּךָ בִּכְבֵדוּת,



וְאָמְרָה צֹּר / ״הֶאָח״, בִּנְפֹל נִזְרִי, / ״הָחֳרָבָה אִמָּלְאָה!״





       שְׁבִי גִבּוֹר / הֲיֻקַּח אוֹ יִמָּלֵט?

       וְאִם יִשְׁבֹּר / מוֹטוֹת צַר, בִּי שׁוֹלֵט?

בְּבֵית הַבּוֹר / יִרְדֶּה בִי בָּחֳרִי / וּבְכֹבֶד מַשָּׂאָה!





       שְׁבִי נֶאְנְחָה, / בַּת עַמִּי, וְשַׂק חִגְרִי,

       וְגַם קָרְחָה / הַרְחִיבִי כְּמוֹ נִשְׁרִי,

וּלְשִׂמְחָה / מַה זֹּה עוֹשָׂה, אִמְרִי, / הֲתִקְרַב וְתָבוֹאָה?





       תְּהִי מִסְעָד / וְעֵזֶר אֶל גּוֹלָתִי

       לְבַל תִּמְעַד, / וְשׁוֹבֲבָה אֶל נַחֲלָתִי,

וּמִגִּלְעָד / הָבִיאָה נָא הַצֳּרִי / לַשְּׁאֵרִית הַנִּמְצָאָה.





       חֲזַק נִמְהַר / לְבַב נִכְאֶה לְהַר צִיּוֹן,

       בְּרוֹשׁ תִּדְהָר / בְּנוֹת הֵיכָל וְאַפִּרְיוֹן,

בְּרֹאשׁ כָּל הַר / יִהְיֶה נָכוֹן הָרִי, / וְעַל כָּל גִּבְעָה נִשָּׂאָה!




מקור: שירמן – השירה העברית בספרד ובפרובאנס
