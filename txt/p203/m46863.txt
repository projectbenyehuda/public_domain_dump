


	
	משולש
	



סימן: תשר"ק יצחק

מו






   תְּמִימֵי אֹרַח / חַיִל אָזְרוּ

   שַׁוְעָם לְהַקְרִיב מְקוֹם קָרְבְּנֵיהֶם

   רְנָנוֹת לָקְחוּ / וְחָשׁוּ וְקִטְּרוּ /

   קְטֹרֶת לִפְנֵי יְיָ לְרָצוֹן לָהֶם

   צַעַד הֵכִינוּ / שֵׁנִית וְיִשְּׁרוּ /

   פַּעַם וּבֵיתְךָ שָׂמוּ פְנֵיהֶם

   עֲרֻומִים מֵאֵין / סוּת אִם נִשְׁאֲרוּ /

   סִתְרְךָ מַחְסֶה יְכַס מַעֲרֻומֵּיהֶם



120    נוֹשְׂאֵי עַיִן / בְּפָנִים חָפְרוּ /



   מְשַׁחֲרֵי חִנּוֹת מִמּוֹצָאֵיהֶם

   לוּלֵי הֶאֱמִינוּ / בִּסְלִיחוֹת גָּבְרוּ /

   כָּלוּ מִפְנֵי פַחַת מַעֲשֵׂיהֶם

   יְקוֹדֵי מוֹקְדוֹת / סְעִיפִּים בָּעֲרוּ /

   טַלֵּי רָצוֹן יְכַבּוּ מוֹקְדֵיהֶם

   חֲסֵרִים כּשֶׁר / פְּעָלִים חָסְרוּ /

   זְכוּת אֲבוֹתֵיהֶם תְּמַלֵּא מַחְסוֹרֵיהֶם









   וַעֲלֵיהֶם הַיּוֹם / תְּשׁוּאוֹת חִבְּרוּ /

   הֵם מֵעוֹלָם סוֹכְכִים בְּכַנְפֵיהֶם



125    דִּבְרֵי תְהִלּוֹת / מִפָּז יָקְרוּ /



   גְּלִילֵי זָהָב מְרִיקִים מֵעֲלֵיהֶם

   בּוֹרְאָם לְרַצּוֹת / בָּהֶם דִּבְּרוּ /

   אוּלַי יִיטְבוּ בְעֵינָיו דִּבְרֵיהֶם

   יַחֲשֹׁב צוּר חָשַׁק / שִׁירוֹת נֶחְקְרוּ /

   וּבָאוּ בְמֹאזְנֵי הַשִּׁיר וְשִׁקְלֵיהֶם

   בְּשִׁקְלֵי אֲשֶׁר / עַל פְּקוּדֵיהֶם עָבְרוּ /

   בְּבוֹאָם לְכַפֵּר עַל נַפְשׁוֹתֵיהֶם

   יִתְדוֹת אָהֳלִי / תָּמִים בִּקְּרוּ /

   נִינָיו לִתְמֹךְ וְהִתָּלוֹת בָּהֶם



130    יִזְכּוּ בְּקָרְאָם / מִשְׁפָּטַי תַּעֲשׂוּ /



   וְאֶת חֻקּוֹתַי תִּשְׁמְרוּ לָלֶכֶת בָּהֶם.
