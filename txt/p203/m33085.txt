


	
	תוכחה [מַה נֶּפֶשׁ נוֹכַחַת]
	



סימן: א"ב ויצחק בבית האחרון

יב


         מַה נֶּפֶשׁ נוֹכַחַת / תָּשִׁיב בְּיוֹם תּוֹכַחַת / וְהנּוֹשֶׁה בָא לָקַחַת

         אֵיכָה עֹרֶף מַקְשֶׁה / וּמֵהִתְרַפֵּס מַחְשֶׁה / אִישׁ אֲשֶׁר לוֹ נוֹשֶׁה

         בְּנֵי יָמִים לְרִגְעֵיהֶם / לְנוֹשֵׁיהֶם מַקְרֵיהֶם / נוֹשִׁים בָּהֶם

         גַּם מִמַּעַל וְתַחַת / תּוֹכָם מִשְׁתַּלַּחַת / צָרַעת פּוֹרַחַת

5        מַה נֶּפֶשׁ נוֹכַחַת…





         דָּבְקוּ בְצִיץ הֶתְלֵמוֹ / וּבְתוֹהוּ יְמֵי הֶבְלֵמוֹ / יַשִּׂיא מָוֶת עֲלֵימוֹ

         הוֹד בְּעָלִים וּרְכוּשׁוֹ / יִקְרָא וְחֵיל כַּחְשׁוֹ / נָשִׁיתִי אֶתְכֶם נָשׂוֹא

         וְנָע מְשׁוּבָה וָנַחַת / פֶּתַע לִשְׁאֹל יֵחַת / וַיִּפֹּל אֶל הַפַּחַת

         מַה נֶּפֶשׁ נוֹכַחַת…





         זַחֲלִי נַפְשִׁי לְמִרְיֵךְ / בִּקְרֹא לִיךְ וּלְגֶוְיִךְ / בֹּאִי וְשַׁלְּמִי אֶת נִשְׁיֵךְ

         חִקְקֵי לִבֵּךְ בְּעוֹדוֹ / נִגְלוּ וּכְתָבָם בְּסוֹדוֹ / כָּל בַּעַל מַשֶּׁה יָדוֹ

         טוּב יָדֶיךָ וּמִנְחַת / שָׁוְא צָרְרוּ בְאַמְתַּחַת / הִנֵּה נִשְׁחַת

         מַה נֶּפֶשׁ נוֹכַחַת…





         יַד אִישׁ תַּכְחִישֵׁהוּ / וְיָחִישׁ לַעְנוֹת פִּיהוּ / אֲשֶׁר יַשֶּׁה בְרֵעֵהוּ



15          כֵּן חוֹבָתוֹ רְשׁוּמָה / וַעֲלֵי מִצְחוֹ חֲתוּמָה / מַשַּׂאת מְאוּמָה



         לְעוֹלָמָהּ זוֹנַחַת / וּבָזֶה לֹא מוּנַחַת / נִמְצֵאת מִכָּאן וּמִכָּאן קָרַחַת

         מַה נֶּפֶשׁ נוֹכַחַת…





         מִי יִתֵּן בְּעוֹד טוּבִי / בְּטֶרֶם לָאָרֶץ שׁוּבִי / לֹא נָשִׁיתִי וְלֹא נָשׁוּ בִי

         נוֹאַלְתִּי כְּאִיש בּוֹזֶה / וְדָבַקְתִּי בְאוֹחֲזֵי / אֶת הַמַּשָּׂא הַזֶּה



20       סְפוֹת עַל חֹם קַדַּחַת / וּבַשְּׂאֵת אוֹ סַפַּחַת / כִּי יִהְיֶה בַקָּרַחַת



         מַה נֶּפֶשׁ נוֹכַחַת…







         עֲנַן סִכְלוּת עֵינֶיךָ / כִּסָּה וְרַעְיוֹנֶיךָ / כִּי יַשֶּׁה לְךָ אֱלֹהַּ מֵעֲוֹנֶיךָ

         פְּנוֹת אֶל חֵטְא בִּקְרוֹתוֹ / בְּלִי הָבִין אַחֲרִיתוֹ / נָשָׂא בוֹ אָלָה לְהֵאָלוֹתוֹ

         אֶל צְרוֹר חַטָּאת צוֹרַחַת / לְהִתְחַבֵּר לַנִּשְׁחָת / אַחַת אֶל אַחַת

25       מַה נֶּפֶשׁ נוֹכַחַת…





         קֵרְבוֹ לַמִּשְׁפָּט לְרִיבוֹ / יַרְשִׁיעוֹ פִיו וְנִיבוֹ / הָאִישׁ אֲשֶׁר אַתָּה נוֹשֶׁה בוֹ

         רוֹעֶה בְּהִתְיַצֵּב לְיַד / מַעְגָּל וְהָיָה צַיָּד / מַשָּׂא כָל יַד

         שְׁפָטוּךָ כְּעִיר הַנִּדַּחַת / בְּלַהֶבֶת קוֹדַחַת / וְאֵשׁ מִתְלַקַּחַת

         מַה נֶּפֶשׁ נוֹכַחַת…





         תּוֹם תָּם לְאֵין תּוֹם תַּמְשֶׁה / לְנוֹשֶׁה בוֹ כִּי תַשֶּׁה / לֹא תִהְיֶה לוֹ כְנוֹשֶׁה



30       יַד צֶדֶק הֵן נְטוּיָה / קֶדֶם לְךָ הִיא גְלוּיָה / צִדְקָתְךָ בְּאֶרֶץ נְשִׁיָּה



         אֲמָתְךָ פְּדֵה מִשַּׁחַת / כִּי לְךָ מִתְוַכַּחַת / אָנֹכִי בוֹרַחַת

         וּבְכֵן מָה רַבּוּ מַעֲשֶׂיךָ יְיָ.
