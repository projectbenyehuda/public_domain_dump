


	
	עֵדוּת יְיָ
	



למתן תורה. סימן: עבד יה.






עֵדוּת יְיָ / קְהַל אֱמוּנַי / נָחֲלוּ אוֹתָהּ



וּמֹשֶׁה עָלָה / עַד לְמַעְלָה / וּבְיָדוֹ מִלֵּא / סַפִּיר גִּזְרָתָהּ



וְאֵל אֱמוּנָה / הוּא כוֹנְנָהּ / יְמִינוֹ קָנְתָה



חַי, פִּרְיָהּ שָׁמַר / יַיִן חָמַר / רֵיחוֹ לֹא נָמַר / עַיִן לֹא רָאֲתָה.





בִּשְׂמֹאלָהּ עֹשֶׁר / כָּבוֹד וָאֹשֶׁר / יָמִים בִּימִינָהּ



כִּי מוֹצְאָהּ יִמְצָא / רוּחַ עֵצָה / יָפוּצוּ חוּצָה / יָפִיק תְּבוּנָה



לְשׁוֹנוֹ תַצְמִיד / חָכְמָה תָמִיד / נַפְשׁוֹ אִוְּתָה.

חַי





דָּרְשׁוּ הַשָּׂרִים / לֶחֶם אַבִּירִים / שְׂכִיּוֹת חֶמְדָּה



גְּנוּזָה בִמְדוֹר / לְאֶלֶף דּוֹר / לְאָדָם תָּדוּר / זֹאת הַתְּעוּדָה



מָה אַדִּיר שִׁמְךָ / תְּנָה הוֹדְךָ / לָנוּ יָאָתָה.

חַי





יָה נָתַן הוֹדוֹ / לְמֹשֶׁה עַבְדּוֹ / פָּתַח בִּתְשׁוּבָה



מַה לָּכֶם אִישִׁים / אַתֶּם דּוֹרְשִׁים / מִצְוַת אֲנָשִׁים / אֲשֶׁר יֵלְכוּ בָהּ



לָקַח מַתָּנוֹת / אִישׁ אֱמוּנוֹת / יָדוֹ עָשָׂתָה.

חַי





הַצֵּל אִישׁ חָכְמָה / מִיַּד אִישׁ מִרְמָה / וְאָרוּר נוֹכֵל



חַיָּיו יְמָרֵר / פִּרְיוֹ אֵרֵר / וּבוֹ יִתְגּוֹרֵר / יְגִיעוֹ אוֹכֵל



שְׁלַח אַלּוּפוֹ / וּמֵרוֹדְפוֹ / קוּמָה עֶזְרָתָה.

חַי
