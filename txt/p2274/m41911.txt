


	
	עֻזִּי אֵלֶיךָ
	



סימן: עבד יהוה.






עֻזִּי, אֵלֶיךָ אֶשְׁמֹרָה

שַׁבָּת קָדְשֶׁךָ מַה נּוֹרָא



כִּי עַל עַמְּךָ / בִּרְכָתְךָ / מִיָּדְךָ / עֲשִׁירָה



זֵכֶר רַב טוּבְךָ אֶזְכֹּרָה

לְהוֹדוֹת שִׁמְךָ אֶבְחָרָה.

עֵינִי, לִבִּי, רוּחִי, חַי פָּעֳלָךְ

עֵינִי, לִבִּי, רוּחִי, חַי אָשִׁיר לָךְ



בִּקְהַל אֱמוּנִים / וְנוֹגְנִים / בִּרְנָנִים / וְזִמְרָה.





בְּיוֹם הַשַּׁבָּת בָּחַרְתָּ

בּוֹ לְיִשְׂרָאֵל הוֹדַעְתָּ



מַתָּנָה טוֹבָה / בִּנְדָבָה / וּבְחִבָּה / נָתַתָּ



תּוֹרָה בּוֹ לָהֶם הִנְחַלְתָּ

וַתּוֹפַע עָלָיו נְהָרָה.

עֵינִי, לִבִּי





דְּבַר הַמֶּלֶךְ וְדָתוֹ

אֶהְגֶּה בָם בְּיוֹם שַׁבַּתּוֹ



אֶשְׁלַח פּאֹרוֹת / טַ״ל אוֹרוֹת / לְהוֹרוֹת / בִּמְלַאכְתּוֹ



וּבִכְנִיסָתוֹ עַד צֵאתוֹ

יִרְאַת ה׳ טְהוֹרָה.

עֵינִי, לִבִּי





יִיטַב לִבִּי עַל שֻׁלְחָנִי

בֹּשֶׂם רֵיחַ טוֹב בִּימִינִי



יֵין הָרֶקַח / בְּלֶקַח / טוֹב, אֶפְקַח / אֶת עֵינִי



תּוֹרַת חֶסֶד עַל לְשׁוֹנִי חַסְדֵי ה׳ אָשִׁירָה.

עֵינִי, לִבִּי





הַמְּעַנֵּג שַׁבָּת רַב טוּבוֹ

נוֹתְנִים לוֹ מִשְׁאֲלוֹת לִבּוֹ



לֹא יִדְעַךְ נֵרוֹ / וְצוּרוֹ / בְּעֶזְרוֹ / וּסְבִיבוֹ



תּוֹרַת אֱלֹהָיו בְּקִרְבּוֹ

עֲרוּכָה בַכֹּל וּשְׁמוּרָה.

עֵינִי, לִבִּי





וְשׁוֹמֵר שַׁבָּת מְחֹל לוֹ

נַחֲלַת עוֹלָם גְּבוּלוֹ



בְּלֹא מְצָרִים / מִזָּרִים / אַכְזָרִים / יַצִּיל לוֹ



וְאִישׁ לְפִי מַהֲלָלוֹ

זוֹכֶה לְרוּחַ יְתֵרָה.

עֵינִי, לִבִּי





הַנְחֵל עֶלְיוֹן עַם יְשֻׁלָּם

הַר צְבִי קֹדֶשׁ וְאוּלָם



וְשָׁם יַחֲנוּ / יִשְׁכְּנוּ / יָרֹנוּ / בְקוֹלָם



חֲטִיבָה אַחַת בָּעוֹלָם

וְשִׁמְךָ גָּדוֹל בִּגְבוּרָה.

עֵינִי, לִבִּי
