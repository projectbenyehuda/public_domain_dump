


	
	סְמֹךְ נִפְלִי
	



סימן: סלימאן.






סְמֹךְ נִפְלִי / וּרְאֵה שִׁפְלִי / וְהוֹשַׁע נָא / חַי נֶאֱמָן אֵל



חֹן עַבְדְּךָ / תֵּן לְעַבְדְּךָ / מַעְנֵה לָשׁוֹן / חַי נֶאֱמָן אֵל





לִי אֵל מַפְלִיא / הָאֵר אָפְלִי / וְהוֹפַע נָא / חַי נֶאֱמָן אֵל



בְּךָ חֵילִי / רְפָא חָלְיִי / יוֹשֵׁב עֶלְיוֹן / חַי נֶאֱמָן אֵל





יָהּ אֵל עוּרָה / וּפְדֵה סוּרָה / וְכוֹנְנָהּ / חַי נֶאֱמָן אֵל



קַבֵּץ עַמָּךְ / לְאוּלַמָּךְ / וּבְנֵה צִיּוֹן / חַי נֶאֱמָן אֵל





מֶלֶךְ נוֹרָא / בְּנֵה בִירָה / קִרְיַת חָנָה / חַי נֶאֱמָן אֵל



עוֹטֶה אוֹרָה / שִׂים גְּבִירָה / תּוֹךְ אַפִּרְיוֹן / חַי נֶאֱמָן אֵל





אֵל דָּר רוּמָה / חוּשָׁה קוּמָה / וְשׁוּב חֹן נָא / חַי נֶאֱמָן אֵל



גַּלֵּה סוֹדָךְ / הַרְאֵה הוֹדָךְ / וּשְׁלַח יִנּוֹן / חַי נֶאֱמָן אֵל





נָא קוֹמְמָה / עִיר שׁוֹמֵמָה / וְקוֹמֵם נָא / חַי נֶאֱמָן אֵל



אֶת אֲיֻמָּה / בָּרָה תַמָּה / בָּהּ אוֹר יִשְׁכֹּן / חַי נֶאֱמָן אֵל.
