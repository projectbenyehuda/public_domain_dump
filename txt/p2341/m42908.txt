


	
	סִתְרִי עֲנֵה
	



סימן: סלימאן. מנגינה: חוסיני






סִתְרִי עֲנֵה / לְדָל נַעְנֶה / וְתֵן מַעְנֶה / בִלְשׁוֹנִי



סְלַח רִשְׁעִי / מְחֵה פִשְׁעִי / אוֹרִי וְיִשְׁעִי / וְקוֹנִי



אַתָּה עֲשִׂיתַנִי / אֵל חַי יְצַרְתַּנִי / מֵאָז בְּרָאתַנִי / וְאָצַלְתָּ אוֹרִי אוֹרִי אוֹרִי



הָאֵר עֵינַי / וְאִישׁוֹנַי / בְּאוֹר פָּנֶי/ךָ נְחֵנִי.





לָךְ אֶתְפַּלָּל / אֵל מְהֻלָּל / עֲנֵה אֻמְלָל / וְעָנִי



הַבֵּט וּרְאֵה / לִבּוֹ נִכְאֶה / הַשְׁקֵף וּרְאֵה / בְעֹנִי



אֲדֹנָי חָנֵּנִי / בְּקָרְאִי עֲנֵנִי / אֱלֹהַי פְּדֵנִי / מִיָּד צָרִי צָרִי צָרִי



רִיבָה רִיבִי / נְקֹם אוֹיְבִי / מִבּוֹר שְׁבִי / יָהּ דְּלֵנִי.





יָהּ אֵל רוּחִי / שְׁמַע שִׂיחִי / וֶאְסֹף נִדְחִי / לִמְכוֹנִי



צוּר מִבְטַחִי / אַתָּה גֹחִי / בְּךָ כֹחִי / וְאוֹנִי



תָּשׁוּב תְּחָנֵּנִי / אֵל חַי הֲקִימֵנִי / וְסֹב נַחֲמֵנִי / וּשְׁלַח צִירִי צִירִי צִירִי



קַל כַּצְּבִי / עִם אִישׁ תִּשְׁבִּי / יִשְׂמַח לִבִּי / תָּשׁוּר עֵינִי.





מַלְכִּי שְׁעֵה / שׁוּבָה וּרְעֵה / בְּטוּב מִרְעֶה / אֶת צֹאנִי



אָנָּא אֵל דַּל / מֵעָפָר דַּל / אָשִׂישׂ, יִגְדַּל / עֶלְצוֹנִי



אַל לָעַד תִּנְשֵׁנִי / וְאַל תְּבִישֵׁנִי / יָחִיד וְאֵין שֵׁנִי / הָקֵם שׁוּרִי שׁוּרִי שׁוּרִי



בְּנֵה מָעוֹן / וְאַפִּרְיוֹן / אֶל הַר צִיּוֹן / הַעֲלֵנִי.





אֲדוֹן עוֹלָם / מֶלֶךְ נֶעְלָם / חֹן דָּךְ נִכְלָם / וּרְצֵנִי



יוֹצֵר כָּל חַי / שׁוּר שְׁבָחַי / בְּךָ אֵל חַי / רַעְיוֹנִי



אֲדֹנָי נֶגְדְּךָ / תַּאֲוַת עַבְדְּךָ / צוּרִי בְחַסְדְּךָ / רְפָא צִירִי צִירִי צִירִי



שְׁמַע נִיבִי / מָגֵן אָבִי / מִמַּכְאוֹבִי / רְפָאֵנִי.





נֵר נִשְׁמָתִי / וְצוּרָתִי / אֱיָלוּתִי / מָגִנִּי



אֵל נֶאֱמָן / אָב הָרַחְמָן / מִיָּדְךָ מַן / מְזוֹנִי



אֱלֹהַי עָזְרֵנִי / לְטוֹבָה זָכְרֵנִי / וְאַתָּה תוֹרֵנִי / דַּרְכֵי מוֹרִי מוֹרִי מוֹרִי



נָא אֵל יָחִיד / שְׁלַח נָגִיד / לְבֶן דָּוִד / נָא הַרְאֵנִי.
