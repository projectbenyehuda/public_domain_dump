


	
	אֶקְרָא לָאֵל מִמַּעֲמַקִּים
	



חתימה: אני שלם






אֶקְרָא לָאֵל / מִמַּעֲמַקִּים

לְיַלְדֵי אֵיתָן / בְּרִיתוֹ יָקִים

בְּרוֹן יַצִּילֵם / מִיַּד עוֹשְׁקִים

בִּזְכוּת אֲבוֹתָם / הַנֶּחֱשָׁקִים

אֲשֶׁר שָׁמְרוּ דָּת / מִצְווֹת וְחֻקִּים

וְאִם אוֹיְבֵינוּ / נִרְאִים חֲזָקִים

עֵינֵי יְוֹיָ / אֶל צַדִּיקִים:





נוֹעַם וְאַחְוָה / אֱמוּנִים דִּרְשׁוּ

וְשָׁלוֹם וְתִקְוָה / מֵאֵל בַּקְּשׁוּ

וּלְיוֹם יְשׁוּעָה / חַכּוּ וְהֶחֱשׁוּ

וְאִם אוֹיְבֵינוּ / בְּחֵילָם רָגְשׁוּ

כִּמְעַט רֶגַע / וְהֵם יִנָּתְשׁוּ

בְּמַעֲשָׂם הָרַע / הֵם יֵעָנְשׁוּ

כִּי הֵם רוֹדְפִים / אַחֲרֵי רֵיקִים:





יֶשַׁע אֱלֹהִים / לְעַמּוֹ יֶאֱתֶה

וְלִבָּם בְּטוֹבַת / דְּבָרִים יְפַתֶּה

וְעַל שׂוֹנְאֵיהֶם / אֵשׁ דָּת יֶחֱתֶה

וְעָרוּךְ בְּדִינָם / תְּמַהֵר תָּפְתֶּה

עוֹשֶׂה מִשְׁפָּט / אֵל לָעֲשׁוּקִים:





שִׁפְעוֹ יוֹרִיד / לְעָם נֶאֱמָנִים

וְיִזְכּוֹר לָהֶם / בְּרִית רִאשׁוֹנִים

שְׁלֹשָׁה אָבוֹת / הָאֵיתָנִים

זְכוּתָם תַּגִּין / כְּאֵם עַל בָּנִים

מִתִּגְרַת צָר / וְכָל זֵדוֹנִים

וּמִיַּד מוֹשְׁלִים / בְּעוֹלָם דּוֹחֲקִים:





לֵב עָם נִדְכָּה / אֲזַי יִתְנַחֵם

בְּזִכְרוֹן צוּרָם / אֲשֶׁר הִבְטִיחֵם

כִּי הוּא עֲלֵיהֶם / כְּמוֹ אָב יְרַחֵם

וְיַקְשִׁיב לְקוֹלָם / בְּעָרְכָם שִׂיחָם

וְיִדְלֶה טָבְעָם / מִמַּעֲמַקִּים:





מַלְכוּת מְשִׁיחוֹ / בְּקָרוֹב יְחַדֵּשׁ

וְעַמּוֹ מִיַּד / בּוֹחֲלִים יְקַדֵּשׁ

וְקַרְנֵי מְעַנָּיו / יִגְדַּע וִיגַדֵּשׁ

וְיִקְצוֹר לְזַרְעָם / כְּקָצִיר בְּחֶרְמֵשׁ

וְיִפְדֶּה לְעַמּוֹ / אֲשֶׁר בּוֹ דְּבֵקִים:
