


	
	אֲיֻמָּתִי בְהַר הַמּוֹר
	



חתימה: אלשבזי משתא






אֲיֻמָּתִי בְהַר הַמּוֹר / וַאְנִי בְּבוֹר תַּחְתִּיּוֹת

בְּרַם יְדִידִי שׁוֹמֵר / וְעֵינַי לוֹ צוֹפִיּוֹת

וְאַנְעִימָה בְשִׁיר מִזְמוֹר / בְּקָו מִשְׁקַל אוֹתִיּוֹת

וְרַגְלֵי כִסְאוֹ קוֹמֵר / בְּאַרְבַּע הַזָּוִיּוֹת

וְעַל דְּמוּת הַכִּסֵּא / דְּמוּת אַרְבַּע חַיּוֹת:





לֵב יְשָׁרִים יִתְחַדָּשׁ / כְּנֶשֶׁר בְּהִמָּצְאוֹ

אֶל נוֹכַח הַמִּקְדָּשׁ / יוֹצֵא מִבֵּית כִּלְאוֹ

וְחֵיל סַמָּאֵל יֻדַּשׁ / יִטְעֶה בְּיֵין סָבְאוֹ

וּקְצִיר גְּבוּלוֹ נִגְדָּשׁ / וְלֹא רָוָה מִצָּמְאוֹ

הֶעָווֹן יוֹצִיאוֹ / לְפִי חֶרֶב פִּיפִיּוֹת:





שַׁאֲלִי נַפְשִׁי תְשׁוּעָה / מֵאֱלֹהֵי עוֹלָמִים

וּשְׁמְעִי שׁוֹפָר תְּרוּעָה / בַּעֲשֶׂרֶת הַיָּמִים

וּצְדָקָה לָךְ זְרוּעָה / לְטַהֵר טִנּוּף גְּשָׁמִים

וְתִהְיִי בְּגַנֵּךְ נְטוּעָה / בְּלִי פַחַד וּזְעָמִים

הִזַּכִּי מֵאֲשָׁמִים / וְהִרְהוּר לֵב וּגְוִיּוֹת:





בְּרֹאשׁ הַשָּׁנָה עִמְדִי / יְחִידָתִי בְשַׁחַר

וְדַלְתוֹת צוּרֵךְ שִׁקְדִי / כִּי בָךְ צוּרִי בָחַר

וּבְזֵבֶד טוֹב זִבְדִי / גּוּפִי וְאַל יִתְאַחַר

אֶתְבּוֹנֵן יוֹם חֶלְדִי / בְּבוֹא הֵילֵל בֶּן שַׁחַר

חֲמוּד עֵינִי אַל תִּתְחַר / עֲשֵׂה עִמִּי זָכִיּוֹת:





זְכוֹר בּוֹרְאָךְ בֶּן אִישׁ / בִּימֵי בְּחוּרוֹתֶיךָ

וְסוּר מֵחוֹמֶר מַבְאִישׁ / וְשׁוּבָה אֶל אֱלֹהֶיךָ

וְשֵׂכֶל וְנֶפֶשׁ תַּקְדִּישׁ / לְאֵל חַי בִּרְצוֹנֶךָ

וּלְךָ צְדָקָה יַלְבִּישׁ / וְתִזְכֶּה בְיוֹם שָׁפְטֶךָ

וְיִרְצֶה יָהּ עָלֶיךָ / בְּגַן הַבִּיתָן לִחְיוֹת:





יוֹם לְיוֹם יַבִּיעַ אוֹמֶר / לְהַזְכִּיר עוֹשֵׂה רְצוֹנוֹ

לְהַעֲלוֹתוֹ מֵחוֹמֶר / הַטָּבוּעַ בַּעֲווֹנוֹ

וְחֶטְאוֹ בְּסֵפֶר שָׁמַר / לְגַלּוֹת בְּיוֹם דִּין קְלוֹנוֹ

וּבִקְצִירַת הָעוֹמֶר / וְאֵין מַעֲנֶה בִלְשׁוֹנוֹ

יֵצְרוּ צַעֲדֵי אוֹנוֹ / וְעֵינַיִם בּוֹכִיּוֹת:





מֵחַשְׁרַת הֶעָבִים / יְרַוֶּה נַפְשׁוֹתֵינוּ

גּוֹמֵל חֲסָדִים טוֹבִים / בְּתוֹךְ יוֹשֶׁר יַנְחֵנוּ

כִּי הוּא מְקַבֵּל שָׁבִים / וְרוֹצֶה תְּפִלָּתֵנוּ

בִּזְכוּת אָבוֹת נְדִיבִים / וְרָאשֵׁי אֲבוֹת שְׁבָטֵינוּ

וְחֵן אַהֲבַת הֲמוֹנֵנוּ / בְּתוֹךְ בָּתֵי כְּנָסִיּוֹת:





שׁוּבִי נַפְשִׁי וְתִזְכִּי / לַחֲזוֹת בְּנוֹעַם צוּרֵךְ

וְיִשְׁעֵךְ הַיּוֹם חַכִּי / וְהָסִירִי רוֹעַ סוּרֵךְ

כִּי בָא אֱלֹהַי מַלְכִּי / לְהַזְכִּיר חֶסֶד נְעוּרֵךְ

מִבֶּכִי תָשׁוּבִי, כִּי / לֹא יִיבַשׁ מְקוֹרֵךְ

רָחֵל לָמָּה תִבְכִּי / אוֹרִי כִּי בָא אוֹרֵךְ

וּבְלֵילֵי כִפּוּרֵךְ / כָּל נְפָשׁוֹת הוֹמִיּוֹת:





תֶּהֱמֶה כָל הַנְּפָשׁוֹת / מִפַּחַד יוֹם הַדִּין

בַּעֲלוֹת חֵטְא נֶעְנָשׁוֹת / יַחַד יְהוּ מוּעָדִין

וּבַעֲלוֹת חֵן נִלְבָּשׁוֹת / בְּאוֹר בָּהִיר כַּסָּדִין

בְּיַחַד מֻגָּשׁוֹת / וּמֶלֶךְ עֵַל כִּסֵּא דִין

וְכָל שׁוֹמְרֵי דָּת וָדִין / מֻבְחָרוֹת וּנְקִיּוֹת:





אַהְבַת כְּלִילַת יוֹפִי / תְּרַצֶּה עַל כָּל עֲדָתִי

וְאִם הָלַכְתִּי בְּשׁוֹפִי / וְעָזַבְתִּי נַחֲלָתִי

בְּקָרוֹב יָבוֹא צוֹפִי / וִירַפֵּא מַחֲלָתִי

מְנָתִי וּצְרוֹר כַּסְפִּי / צוּרִי חֲמוּד אַהֲבָתִי

בְּרֵאשִׁיתִי וּבְסוֹפִי / עֵינִי לוֹ צוֹפִיּוֹת:
