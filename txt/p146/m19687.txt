


	
	מֵאַהֲבַת בַּת נָדִיב בּוֹעֵר לְבָבִי כָּאֵשׁ
	



החתימה: משתא שץ




מֵאַהֲבַת בַּת נָדִיב / בּוֹעֵר לְבָבִי כָּאֵשׁ



לֹא אֶעֱנֶה קוֹל עַל רִיב / וַאֲנִי בְגָלוּת בּוֹשֵׁשׁ



וְאוֹיְבִי עַָלַי מַכְזִיב / וְאוֹמֵר: סוּר וְהִתְיָאֵשׁ



עָלַי כְּמוֹ קוֹץ מַכְאִיב / לְהַפְחִיד הֲמוֹנַי בּוֹלֵשׁ



קוּמָה יְיֹיָ אַל יָעוֹז אֱנוֹשׁ:



שַׁלַּח רְצוּצִים חָפְשִׁים / וְרַחֵם עַל עֲדָתֶךָ



וּמִיַּד אֲדוֹנִים קָשִׁים / תְּקַבֵּץ לִמְבַקְשֶׁיךָ



וּבְעֵת רָצוֹן תַּגְשִׁים / לְרַוּוֹת אַדְמָתֶךָ



וּמִבְּנֵי רֶשַׁע וְעַוְלָה / דַּם עֲבָדֶיךָ תִּדְרוֹשׁ:



תִּרְצֶה תְּקַבֵּל תְּשׁוּבַת / פְּזוּרֵי גוֹלָה מְחַכִּים



וּתְרַחֵמָה עַל בַּת / בְּפִתְחֲךָ תַּעְרִיב וְתַשְׁכִּים



עַל הֲמוֹנַי סוֹבֶבֶת / כִּי הַיְלָדִים רַכִּים



וְאַתָּה מָעוֹז לְדַלִּים / הַמִּתְנַשֵּׂא לְכֹל לְרֹאשׁ:



אִמְרַת יְיֹיָ צְרוּפָה / וְאַשְׁרֵי עוֹשֵׂי רְצוֹנוֹ



לֹא נִמְצָא בָהּ חֲנוּפָה / לְרָשָׁע לְכַסּוֹת עֲוֹנוֹ



יָחוֹן לְנֶפֶשׁ עֲטוּפָה / וְיָאִיר לִבִּי בְּדִינוֹ



עוּרָה כְבוֹדִי עוּרָה / בְּקִרְבְּךָ קָדוֹשׁ:



שֹוֹא גַלֵּי יָם שָׁקְטוּ / מִקּוֹל שַׁדַּי עָמְדוּ



וּבְנֵי עַוְלָה קֻמְּטוּ / מֵאֵשׁ חֲמָתוֹ רָעֲדוּ



וְזֶרַע יַעֲקֹב נִמְלְטוּ / וּפַרְעֹה וְחֵילוֹ נִשְׁמְדוּ



וְכֵן יוֹם בּוֹא יְשׁוּעוֹת / יְיֹיָ מָלָךְ גֵּאוּת לָבַשׁ:



צַהֲלִי קוֹלֵךְ וְרֹנִּי / בַּת צִיּוֹן הַכְּבוּדָּה



כִּי בָא יִשְׁעֵךְ לְקַבֵּץ / קְהִלַּת קֹדֶשׁ וְעוּדָה



אָז יִשְׂמַח הַר צִיּוֹן / תָּגֵלְנָה בְנוֹת יְהוּדָה



לִפְנֵי בּוֹא יוֹם יְיֹיָ / אָכֵן רוּחַ הִיא בֶּאֱנוֹשׁ:
