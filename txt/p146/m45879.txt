


	
	בֵּן תִּפְדֶּה
	



חתימה: בן יוסף






בֵּן תִּפְדֶּה בְּרַחֲמֶיךָ / אֱלֹהִים מִתּוּגוֹתָיו

כִּי נִמְסָר בְּיַד אוֹיֵב / וְהִִרְבָּה מַכְאוֹבוֹתָיו

וְאֵין לוֹ צַד לְהִשָּׁעֵן / בִּלְתִּי בִּזְכוּת אֲבוֹתָיו

וְאַתָּה תַּקְשִׁיב בְּחַסְדָּךְ / וְתִרְפָּא לִמְשׁוּבוֹתָיו

פְּדֵה אֱלֹהִים אֶת יִשְׂרָאֵל / מִכָּל צָרוֹתָיו:





נִשְׁפָּל בְּרוֹב גָּלוּתוֹ / וְאֵין מִי יֶחֱסֶה עָלָיו

בִּלְתִּי בְּצֵל כְּנָפֶיךָ / חוֹסֶה בְּכָל מִפְעָלָיו

כִּי עִמְּךָ מְקוֹר חַיִּים / לְהַשְׁפִּיעַ טוּבְךָ אֵלָיו

וְתִרְצֶה בִּתְפִלָּתוֹ / וְתִשְׁמַע קוֹל מַהֲלָלָיו

וּבְיוֹם דִּין וְתוֹכֵחָה / תְּמַלֵּא כָּל מִשְׁאֲלוֹתָיו:





יִשְׁעָךְ יְקַוֶּה לִדְרוֹשׁ / וְלִתְבּוֹעַ אֶת דִּינוֹ

תֵּפֶן אֶל תְּשׁוּבָתוֹ / וּתְמַלֵּא חֶסְרוֹנוֹ

וּלְחַיִּים תִּכְתְּבֵהוּ / וְיִצְרוֹ אַל יַשְׂטִינוֹ

לְמַעַן שִׁמְךָ יְוֹיָ / וְסָלַחְתָּ לַעֲווֹנוֹ

כִּי אַתָּה הַשּׁוֹפֵט / וְנֶגְדְּךָ כָּל תַּאֲווֹתָיו:





וְאִם עֲווֹנוֹ בוֹ עָנָה / יִסְמְכוּהוּ רַחֲמֶיךָ

כִּי הוֹדֶה בַּחֲטָאָיו / לְבַקֵּשׁ סְלִיחָתֶךָ

יְשַׁעְשְׁעוּהוּ בְּרָצוֹן / אֱלֹהִים תַּנְחוּמֶיךָ

וְאַל יִמְשָׁל בּוֹ פַּחְדְּךָ / וְיִפְגַּע בּוֹ חֲרוֹנְךָ

לְמַעַן חַסְדְּךָ תִּסְלַח / לְפִשְׁעוֹ וּלְחַטֹּאתָיו:





סְמוֹךְ עָם שֶׁבָּךְ נִשְׁעָן / מִכָּל אוֹיְבָיו וְקָמָיו

הַיּוֹם הוּא מִתְוַדֶּה / עַל חֲטָאָיו וַאֲשָׁמָיו

כִּי לֹא יוּכַל לִסְבּוֹל / מִשִּׁלְטוֹן מַשְׂטִימָיו

וְהוּא כְּהֶבֶל נֶחְשָׁב / וְהוֹלֵךְ כַּצֵּל יָמָיו

וְאוֹיְבוֹ עָלָיו גּוֹבֵר / וְתוֹקֵעַ מַחֲנוֹתָיו:





פְּתַח יָדְךָ הָרְחָבָה / וּשְׁמַע צַעֲקַת עַמָּךְ

וְאַל תֵּפֶן לְחַטֹּאתָיו / כִּי מֵעֲשׂוֹת טוֹב הוּא מָךְ

וְלָמָּה יִשְׁכַּב בְּבָשְׁתּוֹ / וּבְרוֹב עֲווֹנוֹ יִדְמָךְ

וְאַתָּה מְקַבֵּל שָׁבִים / לְכָל הַחוֹסִים בִּשְׁמָךְ

תַּקְשִׁיב בְּקָרְאוֹ נֶגְדָּךְ / וְתַרְאֵהוּ בְּמַנְעַמָּךְ

וְתִפְדֶּנּוּ בְּרַחֲמֶיךָ / מִכָּל עֲווֹנוֹתָיו:
