


	
	אֶפְתַּח בְּתַחְנוּנִי
	



תיקון חצות

חתימה: אני שלם






אֶפְתַּח בְּתַחְנוּנִי / אֲשַׁבֵּחַ קוֹנִי

יוֹדֵעַ בְּמַצְפּוּנִי / וְכָל חֶבְיוֹנִי

יָאִיר מְאֹד עֵינִי / וְיוֹשִׁיעֵנִי

יִסְלַח לְשִׁגְיוֹנִי / וְגַם לִזְדוֹנִי

קוּמָה יְוֹיָ / הוֹשִׁיעֵנִי:





נַפְשִׁי תְּקַוֶּךָ / אֱלֹהֵי עוֹלָם

תִּסְלַח לְעַבְדֶּךָ / לְגָלוּי וְנֶעְלָם

נַפְשִׁי תְּבַקֶּשְׁךָ / וְלִבִּי נִכְלָם

כִּי קָץ בְּחֻקֶּיךָ / וְסָר לֹא נִשְׁלָם

לָכֵן כְּחֶטְאִי / אַל תְּדִינֵנִי.





יֶשַׁע וְהַצָּלָה / לְעַבְדָּךְ תַּעְמִיד

נַפְשִׁי מְאֹד חוֹלָה / בְּחֶטְאָהּ תָּמִיד

שִׂכְלִי וְנִשְׁמָתִי / בְּרַחֲמֶיךָ תַּצְמִיד

אַל אֵשְׁבָה בָּדָד / בְּחֶטְאִי גַּלְמִיד

חוּן נָא בְּרַחֲמֶיךָ / וְהַצִּילֵנִי.





שְׁעֵה אֶל שַׁוְעַת / נֶפֶשׁ יְחִידָה

וְאִם שׁוֹגֶגֶת / הִיא אוֹ מְזִידָה

כִּי פּוֹחֶדֶת / בְּחֶטְאָהּ נְדוּדָה

וְעַיִן עָנַת / בְּכַעְסָהּ מְעִידָה

וְלִבִּי תְּחִלָּה / הִחְטִיאַנִי.





לְמַעַן חַסְדָּךְ / זְכוֹר עֲבָדֶיךָ

וּזְכוֹר יְדִידָךְ / אַב שְׁבָטֶיךָ

בְּיוֹם בּוֹא פַּחְדָּךְ / וְגַם דִּינֶיךָ

וְתָרִים יָדָךְ / עֲלֵי יְצוּרֶיךָ

תּוֹצִיא לָאוֹר / אֱלֹהִים דִּינִי.





מְחוֹל נָא חוֹבִי / וְחֶטְאִי תְּכַפֵּר

וְרַפֵּא כְּאֵבִי / עֲצַת רָע הָפֵר

וַעֲווֹן חֻבִּי / יִפְרַח כְּצִפּוֹר

כִּי מָשַׁל בִּי / וְרַב מִלְּסַפֵּר

יִשְׂמַח לִבִּי בָּךְ / וְרַעְיוֹנַי:
