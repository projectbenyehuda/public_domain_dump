


	
	מעמד א: אֱלֹהִים מַעֲשֶׂיךָ
	



החתימה: אבן יוסף






אֱלֹהִים מַעֲשֶׂיךָ / בְּצֶדֶק נוֹרָאוּ

כְּתוֹב אֶת חֲסִידֶיךָ / לְחַיִּים יָבוֹאוּ

וּלְשֻׁלְחָנֶךָ / בְּרוֹן יִקָּרֵאוּ

שׁוֹמֵעַ תְּפִלָּה עָדֶיךָ / כָּל בָּשָׂר יָבוֹאוּ:





בְּעֵדֶן תַּנְחִילֵם / וְיִמְצָאוּ מְנוּחָה

וְתִמְחֶה מַעֲלָלָם / וּפִשְׁעֵיהֶם סְלָחָה

פְּנֵי רוֹן תַּקְבִּילָם / וְשָׁלוֹם וּרְוָחָה

וּמָנוֹחַ וְחַיִּים / וְתִקְוָה יִמְצָאוּ:





נַחֵם הָאֲבֵלִים / וְרַפֵּא מַכְאוֹבָם

וּבַקֵַּר הַחוֹלִים / וְדַבֵּר עַל לִבָּם

וּמִדַּת דִּין הַעְלִים / וְעוֹד לֹא תִמְשָׁל בָּם

שְׁעֵה שַׁוְעַת עַם דָּל / בְּכָל לֵב יִקְרָאוּ:





יִגְדַּל נָא חַסְדְּךָ / עַל עֲדַת יִשְׂרָאֵל

וְתָסִיר אֶת קִצְפְּךָ / וְנַפְשׁוֹתָם תִּגְאוֹל

מְקַוִּים הֵם יִשְׁעֲךָ / עוֹנֶה כָל שׁוֹאֵל

כִּי מִזַּעֲמֶךָ / רְעָדָה מָלֵאוּ:





וְאֶל מִי עוֹד נִסָּמָךְ / וּמִי יוֹשִׁיעֵנוּ

אִם לֹא מִמְּרוֹמָךְ / תְּרַחֵם עָלֵינוּ

וְתַשְׁקִיף מִמְּקוֹמָךְ / לְנַחֵם אֲבֵלֵינוּ

אַתְּ מָעוֹז לְעַמָּךְ / חַסְדָּךְ יִמְצָאוּ:





סְלַח פֶּשַׁע וְעָווֹן / וְאַל תִּזְכּוֹר חוֹבָה

וְהַצֵּל מִדָּווֹן / נֶפֶשׁ שׁוֹבֵבָה

וּמִטִּיט הַיָּוֵן / בְּיוֹם הַמַּעֲצֵבָה

וְתָחִישָׁה לֵאמֹר / לָאֲסוּרִים צֵאוּ:





פְּדֵה נֶפֶשׁ תְקַוֶּה / יִשְׁעֶךָ לִרְאוֹת

שְׁבָחֶיךָ תְחַוֶּה / וְתִרְאֶה נוֹרָאוֹת

וְחַסְדָּךְ לָהּ יְלַוֶּה / אֱלֹהֵי הַצְּבָאוֹת

וּמַלְאֲכֵי הַשָּׁלוֹם / לְמוּלָהּ יִקְרָאוּ:
