


	
	מעמד ד: נַפְשִׁי מְאֹד
	



החתימה: שלם יושף






נַפְשִׁי מְאֹד הוֹמָה / אֲחָזַנִי רַעַשׁ

עַל רָב הַחָכְמָה / אֲשֶׁר לָמַד וְדָרַשׁ

וְהַמָּוֶת בְּעָרְמָה / לְהַכְחִידוֹ בִּקַּשׁ

תּוֹרַת יְוֹיָ תְּמִימָה / מְשִׁיבַת נֶפֶשׁ:





שְׁכֵנָיו נֶאֱנָחוּ / וְסָר מֵהֶם הֲדָרָם

וּמִתּוּגָה שַׁחוּ / עַל סְגֻלַּת דּוֹרָם

אַחֲרָיו נִזְנָחוּ / וַיִּדְעַךְ אוֹרָם

וְנִתְמַעֲטָה חָכְמָה / וְהַהֶגְיוֹן חָלַשׁ:





לְכַמָּה אֶתְעַצָּב / וּמַה-יּוֹעִיל עָצְבִּי

וּמִי זֶה יִתְיַצָּב / כְּמוֹ פְלוֹנִי רִבִּי

בְּעֵת הָיָה נִצָּב / וְזִיוִי לֹא יַחְבִּיא

אֲדַמֶּה אֶת הוֹדוֹ / כְּמוֹ אוֹר הַשֶּׁמֶשׁ:





מָוֶת זֶה נִפְלָא / וְאֵין מֶנּוּ מַצִּיל

הוּא בָא בַּעֲלִילָה / וְלַכּוֹחוֹת יַעְצִיל

נִבְרָא מִתְּחִלָּה / וּמִפַחְדּוֹ הִאְצִיל

בְּעֵת אָדָם חָטָא / וּמִגָּן נִתְגָּרָשׁ:





יְכַלֶה הַבִּרְיוֹת / וְאֵין בּוֹ רַחְמָנוּת

נְפָשׁוֹת הַחַיּוֹת / עֲלֵי יָדוֹ תָמוּת

וְיַפְרִיד הַבִּרְיוֹת / וְיַנִּיחַ גּוּף צָמוּת

וְיָשׁוּב אֶל עֲפָרוֹ / וּבַחֵטְא יֵעָנַשׁ:





וְאֵיךְ יֻזְכַּר הַהוֹן / בְּשָׁעַת הַפֵּרוּד

כִּי זֶה לֹא יָחוֹן / כְּמוֹ אַרְיֵה יָצוּד

וְאֵין מִי יִמְצָא חֵן / בְּעֵת יִפּוֹל שָׁדוּד

וְלֹא יָחוֹן תִּינוֹק / נוּלָּד מֵאֶמֶשׁ:





שׂוֹטֵם כַּלּוּסְטוֹס / לִבְנֵי הָאָדָם

בְּכָל רֶגַע חוֹמֵס / בְּבוֹאוֹ יַפְחִידָם

עֲלֵיהֶם לֹא יָחוֹס / בְּעֵת יִקְרַב אֵידָם

וְלֹא יַבְדִּיל וְיַפְרִישׁ / בֵּין חוֹל לַקֹֹּדֶשׁ:





פַּחְדּוֹ לֹא יָסוּר / מִלֵב הַבְּרוּאִים

כִּי בְיָדוֹ מָסוּר / דִּין הַנִּמְצָאִים

וְיָבִיא לוֹ סִרְסוּר / לַגּוּף בָּחֳלָאִים

עֵת כִּי אַזְכִּירוֹ / אֲחָזַנִי רַעַשׁ:
