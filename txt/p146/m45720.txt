


	
	אֱמוּנִים יוֹם
	



חתימה: אלשבזי






אֱמוּנִים יוֹם / מַאֲוַיִּים / נִקְבָּצִים לְהִתְוַעֵד

בְּיוֹם כִּפּוּר / סוֹד סִפּוּר / וִדּוּי דְּבָרִים מִבְּעוֹד

בִּמְחִילָה / יַקְבִּילָה / אֵל חַי לְנִכְלָם רוֹעֵד

וִיזַלֵּף / בְּטַל דּוֹלֵף / בְּעָבֵי עֲנָנִים וּבְאֵד



עַל בְּרָקִים / וּשְׁחָקִים / גָּדוֹל יְוֹיָ וּמְהֻלָּל מְאֹד:





לוֹ תִכְרַע / עַל טוֹב וְרַע / כָּל בֶּרֶךְ וְתִשְׁתַּחֲוֶה

רֹאשׁ הַכֹּל / וּמְכַלְכֵּל / לְכָל מָצוּי הַהֹוֶה

לְזֶה הִמְצִיא / וְזֶה הוֹצִיא / וְזֶה כָעוּר וְזֶה נָאוֶה



וְזֶה דוֹמֵם / וְזֶה מְרוֹמֵם / וְאַחֲרִיתוֹ תִּשְׂגֶּה מְאֹד:





שׂוֹא גַלִּים / לוֹ מְהַלְּלִים / וּנְהָרוֹת יִמְחֲאוּ כָף

וְתַנִּינִים / בְּשִׁיר עוֹנִים / וּתְהוֹם בְּקוֹל שִׁיר נִכְסָף

וּסְגַלְגַּל / וְגַם גַּלְגַּל / וְאוֹפָן וְקָו הַמֻּקָּף



וְהוּא אֵל רָם / עֲלֵי שִׁירָם / אוֹר בָּהִיר נוֹרָא מְאֹד:





בְּנֵי יַעֲקֹב / עֲווֹן עָקֹב / יְכֻפַּר לָכֶם יוֹם זֶה

וְהוֹד סָחַר / עֲלוֹת שַׁחַר / לְרֹאשׁ צַדִּיק הַלָּזֶה

וּבַת גַּלִּים / גְּבִיר יַעְלִים / בְּמֵי טוֹהַר לָהּ יַזֶּה



וּמַשְׁפַּעַת / לְבֵן נִבְעַת / כִּי יָפָה הִיא מְאֹד:





זְבוּל נִפְתָּח / וְחוּט נִמְתָּח / מֵחֶסֶד אֱלֹהֵינוּ

בְּסוֹד רַחֲמִים / לְבֶן תָּמִים / הֲדַר סִגְנֵי שְׁבָטֵינוּ

וְנִמְשֶׁכֶת / בְּשַׁלֶּכֶת / חֲבַצֶּלֶת לְעָזְרֵנוּ



וְחוֹזֶרֶת / וְאוֹמֶרֶת / יְוֹיָ אֱלֹהַי גָּדַלְתָּ מְאֹד:





יְזַכֵּנוּ / צוּרֵנוּ / לְהִתְהַלֵּךְ בְּמֵישָׁרִים

יְפַרְנֵס דַּךְ / בְּכָל צוֹרֶךְ / וְגוֹזָלָיו מְזֻקָּרִים

לְהִתְפַּרְנֵס / בְּדֶרֶךְ נֵס / בְּחֶסֶד אֵל נֶעְזָרִים



וְהוּא פוֹדָם / וּמַעְמִידָם / וַיְבָרְכֵם וַיִּרְבּוּ מְאֹד:
