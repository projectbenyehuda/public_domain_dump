


	
	בְּמִנְחַת יוֹם כִּפּוּר
	



חתימה: בן יוסף שבזי






בְּמִנְחַת יוֹם כִּפּוּר / הַסְּמוּכָה לַנְּעִילָה

רוּחִי וְנִשְׁמָתִי / בְּעֵת רָצוֹן תַּקְבִּילָה

הַדְרַת כְּבוֹד מַלְכִּי / אֵל נוֹרָא עֲלִילָה

מְקַבֵּל הַשָּׁבִים / בְּרַחֲמִים וּמְחִילָה

הַשְׁמִיעוּ בְּצִיּוֹן / שִׂמְחָה וְגִילָה:





נִתְכַפֵּר יוֹם זֶה / עֲווֹן חֶבְרַת קְדוֹשִׁים

בְּמִנְחַת זִכָּרוֹן / סָר מַיִם עֲנוּשִׁים

הַשְׁמִיעוּ מְחִילָה / בְּרֹאשׁ מַעֲלוֹת שְׁלוֹשִׁים

הָבִינוּ זְקֵנִים / בַּחוּרִים וִישִׁישִׁים

וְכָל הַנִּגָּשִׁים / אֲשֶׁר לָאֵל סְגֻלָּה:





יְחִי לְבַָבְכֶם לָעַד / כָּל אַנְשֵׁי אֱמוּנָה

כִּי תִזְכּוּ כֻּלְּכֶם / לְטוֹבָה הַצְּפוּנָה

יַחַד בְּצֵל שַׁדַּי / יְהֶא לְנַפְשְׁכֶם מְלוּנָה

וְלִשְׂרִידֵי הַדָּת / יְדִידִים שַׁאֲלוּ נָא

תִּרְצֶה עָלֵינוּ / זְכוּת שַׁבָּת וּמִילָה:





וּזְכוּת אֲבוֹתֵינוּ / וְתוֹרָה הַתְּמִימָה

הִיא תַעֲמוֹד לָנוּ / וְדֶרֶךְ תָּשִׂימָה

לִתְשׁוּבָתֵנוּ / בְּרָצוֹן תַּקְדִּימָה

תְּרַפֵּא לֵב נִדְכָּא / וְכוֹחוֹ תַּעֲצִימָה

וְדִגְלִי תָּרִימָה / וְאוֹיְבִים תַּשְׁפִּילָה:





סְלַח בְּמִנְחַת כִּפּוּר, / צוּרִי, לְעַמֶּךָ

וְכָתְבֵנוּ לְחַיִּים / וְשָׁלוֹם בְּסִפְרֶךָ

וְרַפֵּא הַלְבָבוֹת / בְּחֶסֶד יְמִינֶךָ

וְיִכְבְּשׁוּ רַחֲמֶיךָ / בְּיוֹם זֶה כַּעֲסֶךָ

רַחֵם שׁוֹבָבִים / דּוֹפְקֵי דְלָתֶיךָ

הֲמוֹנֵי שְׁבָטֶיךָ / כְּקֶדֶם תַּסְגִּילָה:





פּוֹדֶה וּמְרַחֵם / עֲנִיִּים וִיתוֹמִים

יְקַבֵּץ עַם קְדוֹשָׁיו / מִבֵּין הָעַמִּים

כָּל זֶרַע יַעֲקֹב / שֶׁנִּקְרְאוּ שְׁלוֹמִים

בְּרָצוֹן יְנַהֲגֵנוּ / בְּחֶסֶד וּבְרַחֲמִים

טְהוֹרִים מִכָּל חֵטְא / וְעָווֹן וַאֲשָׁמִים

וְיָחִישׁ מְשִׁיחֵנוּ / יְמַהֵר לְהַצִּילָה:





שׁוּלַמִּית יוֹם זֶה / שְׁבָטִים הָשִׁיבִי

וְהַמְצִיאִי כוֹפֶר / וּמִנְחָה הַקְרִיבִי

לְכָל עָנִי וְאֶבְיוֹן / בְּמָזוֹן הֵטִיבִי

וּבְיוֹם בּוֹא גְּאוּלִים / אָהֳלֵךְ הַרְחִיבִי

וְלִשְׂרִידֵי חָכְמָה / גְּבֶרֶת הַשְׂגִּיבִי

וְרוֹחַ אֱלֹהִים בִּי / לְשִׂכְלִי תַאֲצִילָה:





בּוֹאִי עֵת רָצוֹן / בִּתְפִלַּת מִנְחָה

יוֹם כִּפּוּר עֲווֹנִי / טוֹב מִכָּל שִׂמְחָה

כִּי נִדְחָף סְמָאֵל / מִשְּׂמֹאלִי נִדְחָה

וְסָר מֶנִּי עָווֹן / אֲשֶׁר הָיָה טִרְחָה

דַּלְתֵי הַתְּשׁוּבָה / יְדִיד נַפְשִׁי פְּתָחָה

וּבִשְׁנֵי גּוֹרָלוֹת / אַהֲרֹן יַגְרִילָה:





זֶה הַיּוֹם מְכַפֵּר / לַשָּׁבִים עַצְמוֹ

בַּיּוֹם הַזֶּה כָּל בָּא / יִקָּרֵא בִּשְׁמוֹ

וִיצַוֶּה חַסְדּוֹ / בּוֹרֵא עוֹלָמוֹ

לְעָם נִקְרָא בְּכוֹרוֹ / אֲשֶׁר יָצַר בְּצַלְמוֹ

הֵם חֶבְרַת יְדִידָיו / פְּאֵר נֶטַע כַּרְמוֹ

וְנִקְרָאִים עַמּוֹ / אֲשֶׁר לְשֵׁם וְלִתְהִלָּה:





יְשׁוּרוּן הִתְיַשֵּׁר / סָרוּ כָּל עֲווֹנוֹת

בְּמִנְחַת יוֹם כִּפּוּר / הַבִּיעַ רְנָנוֹת

בְּךָ בָּחַר יְוֹיָ / מִשִּׁבְעִים לְשׁוֹנוֹת

וְלָכֶם עוֹרְרוּ זְכוּת / בְּאַרְבַּע הַמַּחֲנוֹת

וּבְצֵל אֵל שַׁדַּי / נַפְשׁוֹתֵיכֶם מִתְלוֹנְנוֹת

וּבַבֹּקֶר יַחַד / שִׁיר וּשְׁבָח נוֹתְנוֹת

וּבַגּוּפוֹת שׁוֹכְנוֹת / וְחוֹזְרוֹת בְּיוֹם חֲלִילָה:
