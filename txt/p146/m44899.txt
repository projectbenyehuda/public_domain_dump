


	
	מעמד ו: הוֹי עַל רֵעַ
	



החתימה: שלם יושף






הוֹי עַל רֵעַ טוֹב / רֹאשׁ חַכְמֵי הַדּוֹר

אֲשֶׁר הוּא כְּגַן רָטוֹב / עֲלֵי נַחַל גּוֹבֵר:





שׁוֹפֵעַ בְּסוֹדוֹת / כְּמוֹ אַב הַיְּשִׁיבָה

וּמַבִּיעַ חִידוֹת / בְּכָל סוֹד יְשִׁיבָה

וּבְחֻקִּים וְעֵדוֹת / לְכָל לֵב יֵיטִיבָה

יְשַׂמַּח עָצֵב / בְּעֵת יִהְיֶה דוֹבֵר:





לְבָבִי יוֹחִילָה / בְּזָכְרִי שֶׁנִּפְטָר

וְנַפְשִׁי תֵילִילָה / בְּמִי עוֹד תִּתְעַטָּר

וּמָוֶת בַּעֲלִילָה / בָּא עָלָיו וְנִפְטָר

וְאָרַב לוֹ כַדֹּב / בְּעֵמֶק הַמָּצוֹר:





מִי זֶה יְנַחֲמֵנוּ / עַל רֹאשׁ הַתְּעוּדָה

אֲֲשֶׁר הוּא מַשְׁקֵנוּ / מִמַּעְיַן הַנְּגוּדָה

הֲדָרוֹ בֵינֵינוּ / כְּמוֹ רַבִּי יְהוּדָה

לְתוֹרַת אֵל אוֹהֵב / וְהַמִּצְווֹת שׁוֹמֵר:





יָשָׁר בְּמִפְעָלוֹ / וְהוֹסִיף בְּחָכְמָה

בְּכָל יוֹמוֹ וְלֵילוֹ / כְּמוֹ גִּבּוֹר נִדְמָה

מְרַוֶּה מִלּוּלוֹ / לְכָל מִי שֶׁיִּצְמָא

זִכְרוֹ כְּיֵין הַטּוֹב / נָבוֹן לֵב הַטָּהוֹר:





וְאֵיךְ אֶמְצָא דִמְיוֹן / לְחָבֵר הַנָּעִים

לְבָבִי כָּאֶבְיוֹן / עַל נְטַע שַׁעְשׁוּעִים

וְיָדַי מֵרִפְיוֹן / טוֹפְחוֹת בִּזְוָעִים

וּמִי יוּכַל יָשׁוֹב / וּמִסָּבִיב מָגוֹר:





שְׂשׂוֹנִי פָנָה / אַחַר אִישׁ חֲמוּדוֹת

וְגַם לִבִּי עָנָה / נְהִיָּה וּסְפִידוֹת

וְחָכְמָה נִצְפָּנָה / וּרְמָזִים וְסוֹדוֹת

וְלִבִּי הוּא כוֹאֵב / וְאֵינוּ מִתְעוֹרֵר:





פָּנָה הֲדַר הָעִיר / אַחַר שֶׁנֶּאֱסָף

גָּדוֹל וְגַם צָעִיר / כְּאֵב לִבָּם יָסָף

וְאֵיךְ לִבִּי אָעִיר / וְלַחָכְמָה יִכְסָף

וּמִי אֶמְצָא אוֹהֵב / לְתוֹרַת אֵל חוֹקֵר:
