


	
	אימה בת מלכים
	



חתימה: אלשבזי משתא זיו




אֲיֻמָּה בַּת מְלָכִים הַתְּיֻמָּה / עֲדֵי כַמָּה / בְּגָלוּתֵךְ
עֲלוּבָה



לְדוֹד בִּקְשִׁי / וְהַפְרִיחִי לְשָׁרְשִׁי / בְּהַר קָדְשִׁי /
הֲעִידִינִי אֲהוּבָה



שְׁאוֹן גַּלִּים / מְשָׁכַנִי וְהִעְלִים / בְּנֵי אֵלִים / וְנִשְׁמָתִי
עֲזוּבָה



בְּחֵיק רַעְיָה / יְחִידָתִי עֲנִיָּה / וּבוֹכִיָּה / בְּיַד פֶּרֶא
עֲצוּבָה



זְמָן בַּחְרוּת / מְשָׁכַנִי בְּיַשְׁרוּת / בְּדָת חָרוּת / בְּתוֹרַת
אֵל כְּתוּבָה



יְקָרָה מִפְּנִינִים / לַמְּבִינִים / וְהֵם חוֹנִים / בְּתַחְלוּאֵי
מְשׁוּבָה.



מְתֵי שֶׁקֶר / עֲוֹנָם לֹא יְבֻקַּר / עֲדַת עֵקֶר / וְחָכְמָתָם
גְּנוּבָה



שְׁפָלִים / כַּבְּהֵמוֹת הֵם מְשׁוּלִים / מְבֹהָלִים / בְּעֵץ דַּעַת
תְּנוּבָה



תְּכוּנָתָם / מְכֹעֶרֶת בְּדָתָם / וְאַהְבָתָם / בְּסוֹד אֵיבָה
חֲשׁוּבָה



וְאֵין חָכְמָה / וְלֹא דַעַת וְעָצְמָה / אֲבָל עָרְמָה / בְּלִבּוֹתָם
נְצוּבָה



זְמוּנִים לַעֲשׂוֹת / רָאשֵׁי גְּיָסוֹת / וּמַחְמִיסוֹת / בְּלִי מַתָּן
קְצוּבָה



יְדִיד חוּסָה / עֲלֵי עֵדָה אֲנוּסָה / וְתָנוּסָה / לְתוֹךְ צַעְרֵי
תְשׁוּבָה



וְהָגֵן עַל בְּנֵי אִישׁ תָּם / בְּלֶכְתָּם / תְּהֶא דַעְתָּם / בְּחֶסֶד
אֵל רְהוּבָה.
