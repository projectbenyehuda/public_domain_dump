


	
	אֵל חַי
	



חתימה: אלשבזי בן יוסף אלשבזי






אֵל חַי לַדָּךְ אַל תַּכְלֵם / צוּר הַמֵּשִׂיחַ לְאִלֵּם

שִׂכְלִי יְעִירֵנִי בַּחֲלוֹם / וַאֲנִי בְּתֵימָן מִתְעַלֵּם

וְרַעְיָה אָמְרָה לִי: גַּשׁ הֲלוֹם / אֶת אֲשֶׁר תִּדּוֹר שַׁלֵּם

דְּרָכֶיהָ דַּרְכֵי נוֹעַם / וְכָל נְתִיבוֹתֶיהָ שָׁלוֹם:





לֵךְ עַמִּי בּוֹא בַחֲדָרֶיךָ / אֱזָר נָא כְּגֶבֶר חֲלָצֶיךָ

וּפָקַדְתָּ אֶת נָוֶךָ / וְלֹא תֶחֱטָא לֵאלֹהֶיךָ

כִּי הוּא מוֹחֶה פְשָׁעֶיךָ / בְּלֶכְתְּךָ יִשְׁמוֹר צְעָדֶיךָ

יִשָּׂא יְוֹיָ פָּנָיו אֵלֶיךָ / וְיָשֵׂם לְךָ שָׁלוֹם:





שׁוּבִי וַעֲלִי לָךְ מִבּוֹר / יְחִידָתִי קְנִי לָךְ חָבֵר

כִּי צַד הַשְּׂמֹאל מוּם עוֹבֵר / לְעַצְמוֹת בִּנְיָנֵךְ שׁוֹבֵר

וּבְצַד יָמִין דָּל סוֹבֵר / צֶדֶק וּמֵישָׁרִים דּוֹבֵר

פֶּלֶא יוֹעֵץ אֵל גִּבּוֹר / אֲבִיעַד שַׂר שָׁלוֹם:





בֵּין הַמִּשְׁפְּתַיִם עָמַד / שִׂכְלִי, וּמוּסָר לֹא לָמַד

בִּימֵי בַּחְרוּתִי צָמַד / הִרְהוּר מַחְשָׁבָה חָמַד

הוֹמֶה כְּעֵגֶל לֹא לֻמַּד / וְהַגּוּף סוֹפוֹ יִתְגַּלְמַד

וְאֶל אֲבוֹתָיו / יָבוֹא בְּשָׁלוֹם:





זֵכֶר צַדִּיק לִבְרָכָה / חֶלְקוֹ בְּגַן עֵדֶן זָכָה

וְנֶפֶשׁ רְשָׁעִים נָבוֹכָה / וְרוּחָם אֶל שְׁאוֹל נֶהְפָּכָה

וְצִדְקַת יְשָׁרִים נִמְשָׁכָה / עוֹבְרֵי בְּעֵמֶק הַבָּכָא

הָאֶזְרָח יַעֲשֶׂה כָּכָה / יְוֹיָ יְבָרֵךְ אֶת עַמּוֹ בַשָּׁלוֹם:





יָאֲתָה חֵן לִבְנֵי יְשָׁרִים / דּוֹבְרֵי צֶדֶק וּמֵישָׁרִים

מַה נָּאווּ עַל הֶהָרִים / רַגְלֵי מְבַשֵּׂר בְּקוֹל יָרִים

וִיקַבֵּץ כָּל הַפְּזוּרִים / זְקֵנִים עִם טַף וּבַחוּרִים

צְלַח לַעֲמוֹד בֶּן הַיְּקָרִים / וְאֶת אַחֶיךָ תִּפְקוֹד לְשָׁלוֹם:
