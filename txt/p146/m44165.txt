


	
	בַּחֲצוֹת לַיִל
	



תיקון חצות

חתימה: בן יוסף שלם






בַּחֲצוֹת לַיִל / לְבָבִי יִקָּרֵא

יִזְכּוֹר מַעֲשֶׂיךָ / לְמַעַן תִּוָּרֵא

וְנַפְשִׁי בּוֹ תָקוּם / לְמַעַן תַּזְהִירוֹ

בְּיִרְאָה וּתְשׁוּבָה / וְחָכְמָה תַּעְטִירוֹ

וְעַל מֵי הַיֶּשַׁע / לְהֵיטִיב תַּעֲבִירוֹ

כִּי יֵצֶר לֵב הָאָדָם / רַע מִנְּעוּרוֹ:





נַפְשִׁי, תְּפִלָּתֵךְ עִרְכִי / שׁוּבִי מֵחַטַּאת נְעוּרֵךְ

הִתְנַחֲמִי מִפְּשָׁעִים / וּמִגּוּף אַכְזָר אַזְהִירֵךְ

כִּי יָשׁוּב לְאַדְמָתוֹ / וְאַתְּ תָּשׁוּבִי אֶל חַדְרֵךְ

וְאִם בַּהֲבָלָיו תִּמָּשְׁכִי / פֶּגַע וַעֲווֹנִים יִקְרֵךְ

וּמַה תַּעֲשִׂי אַחַר זֹאת / וּמַה תָּשִׁיבִי אֶת צוּרֵךְ

וְאִם תְּבַקְשִׁי לַעֲרוֹךְ אֶת דִּינֵךְ / בְּחֵטְא תְּטַנְּפִי אֶת חַדְרוֹ:





יוֹשֶׁר לִבְשִׁי וַעֲנָוָה / אִם תִּהְיִי לָאֵל תַּמָּה

הַרְחִיקִי תַּאְוַת הַגּוּף / אָז תִּנָּצְלִי מֵאָשְׁמָה

כִּי הוּא בְּהֶבְלוֹ יַשִּׁיאֵךְ / וְיִזְרוֹק עָלַיִךְ זֻהְמָה

מִצַּד נָחָשׁ עֲקַלָּתוֹן / בַּעַל תַּחְבּוּלוֹת וְעָרְמָה

אֲשֶׁר הִטְעָה אָדָם תְּחִלָּה / וְעַל מִצְוַת אֵל הֶעְבִירוֹ:





שִׁמְרִי מִצְווֹת אֵל וְחֻקָּיו / אַל נָא יְחִידָתִי תִּסְטִי

וּרְאִי פְּלִיאוֹת חָכְמָתוֹ / וְאָזְנֵךְ לְתוֹרָתוֹ הַטִּי

כִּי הוּא לִכְאֵבֵךְ מַרְפֵּא / בָּהּ חֵן וְגַם שָׁלוֹם תַּעְטִי

אַל יַשִּׂיאֵךְ לֵב רָעָה / כִּי הוּא הַסִּבָּה לְהַחְטִיא

וְהָעַיִן תַּבְהִילֶנּוּ / תָּתוּר וּבַחֵטְא תְּעִירוֹ.





לָבַשְׁתִּי אֵימָה וְיִרְאָה / מִיּוֹם הַדִּין וְהַחֶשְׁבּוֹן

אֵיכָה עֲווֹנִי הַנֶּאְדָּם / לְאַחַר צָבְעוֹ יִתְלַבֵּן

אוֹ מַה יְשִׁיבָהּ לִבִּי / בְּיוֹם הַקְבָּלַת הָרִבּוֹן

אִם לֹא יָהּ עָלַי יְרַחֵם / וְיַחְמוֹל עָלַי כְּאָב עַל בֵּן

כִּי לִבִּי עָלַי מָרַד / וְלֹא יָכֹלְתִּי הַחְזִירוֹ:





מָה אֶצְטַדֵּק בִּדְבָרַי / אַחַר שֶׁלִּבִּי עָוָה

וְנַפְשִׁי בַּהֲבָלָיו מָשַׁךְ / עֲדֵי הִטְמִיעָהּ כְּדָוָה

וְגָזַל בַּחֲיוּתוֹ וְחָמַס / וְלֹא שִׁלֵּם מַה שֶׁלָּוָה

וְחָמַד וְנִאֵץ לֵאלֹהִים / וְלָרִיק וּלְתוֹהוֹ קִוָּה

אֱלֹהִים בְּרַחֲמָיו יוֹפִיעַ / וּבִסְלִיחָה יָאִיר נֵרוֹ:
