


	
	אֲקַדֵּם בְּשִׁיר
	



חתימה: אני שלם






אֲקַדֵּם בְּשִׁיר רַחְשִׁי / תּוֹךְ עָם סְגֻלָּתוֹ

בְּיוֹם שַׁבָּת קָדְשִׁי / אַסְדִּיר תְּהִלָּתוֹ

בָּרְכִי נָא נַפְשִׁי / בְּרֵאשִׁית בְּרִיאָתוֹ

וּבַיּוֹם הַשִּׁשִּׁי / הִשְׁלִים כָּל מְלַאכְתּוֹ

לְקַדֵּשׁ יוֹם שְׁבִיעִי / וְלִזְכּוֹר בְּרִית עוֹלָם

בָּרוּךְ שֶׁאָמַר וְהָיָה הָעוֹלָם:





נִקְדָּשׁ בִּצְבָאוֹת / כִּי הוּא רַב כּוֹחַ

וְדוֹרוֹת הַבָּאוֹת / הֵמָּה גִּבְהֵי רוּחַ

עַד בּוֹא אִישׁ צַדִּיק / אֲשֶׁר נִקְרָא נֹחַ

וְנִכְנָס לַתֵּבָה / וּמָצָא מָנוֹחַ

עַד עֲבוֹר הַמַּבּוּל / לְטַהֵר הָעוֹלָם:





יָצָא הוּא וּבָנָיו / וְעוֹפוֹת וּבְהֵמוֹת

עִם מִינֵי נְחָשִׁים / וְדֻבִּים וַאֲרָיוֹת

וּבֵרְכָם הָאֲלֹהִים / לִפְרוֹת גַּם לִרְבּוֹת

עַד בֹּא יָדִיד / לְסוֹף עֲשָׂרָה דּוֹרוֹת

הוּא כַּעֲנָק וּכְרָבִיד / נֻסָּה לְסוֹף הָעֲשָׂרָה

בְּאוֹמֶן לְאֵל יָחִיד / אֲשֶׁר עָמַד בְּכֻלָּם:





שָׂמֵחַ וְדָץ וְשָׁקֵד / בְּבֵן נוֹלָד לְשָׂרָה

בְּמוֹרִיָּה נֶעְקָד / וְזֶה סוֹף הָעֲשָׂרָה

וְהוּא לָאֵל נֶחְמָד / וּמִדָּתוֹ גְּבוּרָה

וְהוֹלִיד זוֹג תְּאוֹמִים / תָּם וּבַעַל עֲבֵרָה

עֵשָׂו אִישׁ דָּמִים / אֲשֶׁר מָכַר בְּכוֹרָה

לְיַעֲקֹב אוֹהָלִים / וְחֶזְיוֹנוֹת בְּסֻלָּם:





לְנַסּוֹתוֹ בְּיוֹסֵף / וְהוּרַד לְמִצְרַיִם

יָלַד שָׁם גְּבִירִים / מְנַשֶּׁה וְאֶפְרַיִם

וְיַעֲקֹב עוֹד יָרַד / וְנִרְפָּא בָּעֵינַיִם

כִּי רָאָה אֶת יוֹסֵף / צַדִּיק יְסוֹד עוֹלָם.





מְשָׁכָם בַּעֲבוֹתוֹת / אֵל מַשְׁפִּיל וּמֵרִים

לְקַיֵּם נִגְזָרוֹת / אֲשֶׁר בֵּין הַבְּתָרִים

לְהַשְׁלִים הָעֲשָׂרוֹת / וְיִהְיוּ נִטְהָרִים

כְּתָמִים וּסְיָגוֹת / עַד לֵיל שִׁמֻּרִים

וְיָצְאוּ בְּכוֹשָׁרוֹת / וּמֹשֶׁה הוּא גּוֹאֲלָם:





הֱבִיאָם לְהַר סִינַי / וְקִבְּלוּ הַתּוֹרָה

בְּלוּחוֹת אַחֲרוֹנִים / דִּבְּרוֹת עֲשָׂרָה



וְתַּרְיַ״ג נִצְפָּנִים / בְּתוֹכָם נִקְשָׁרָה



וּבָם הֵאִיר עֵינָם / וְהִגִּיָהּ אָפְלָם:
