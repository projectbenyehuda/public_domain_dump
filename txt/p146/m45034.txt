


	
	נוֹחֲלֵי יוֹם
	



חתימה: שלום






נוֹחֲלֵי יוֹם מְנוּחָה / שִׁירוּ נָא שִׁיר לָאֵל

כִּי אוֹרָה וְשִׂמְחָה / יוֹם זֶה לְיִשְׂרָאֵל:





שִׂמְחַת גִּיל לְבָבִי / הִנֵּה מְאֹד נִגְדְּלָה

עֵת אֵצֵא בְּעַרְבִי / לִקְרַאת פְּנֵי כַלָּה

לְזֹאת אֶהְגֶּה בְּנִיבִי / לָהּ שִׁיר עִם תְּהִלָּה

בְּשׁוּבָה וַהֲנָחָה / בְּהַדְרַת עַם הָאֵל

עַד לְבֵית מְנוּחָה / כְּמַלְכְּתָא לְיִשְׂרָאֵל:





לִפְנֵי הוֹד כְּבֻדָּה / בַּת מֶלֶךְ פְּנִימָה

יָחִישׁ עָם אֲשֶׁר נִפְדָּה / לַעְרוֹךְ שֻׁלְחָן בְּנַהְמָא

וּמִטָּה בִּכְלֵי חֶמְדָּה / וּמְנוֹרָה נְעִימָה

וְהֵן מֵעֵת הַמִּנְחָה / יַעֲשֶׂה וְגַם יוֹאֵל

בְּעַצְמוֹ לֹא שְׁלוּחָה / כִּגְדוֹלֵי יִשְׂרָאֵל:





וְזֹאת נֹאמַר כַּהֲלָכָה / בְּדָת זְכוּת הֲכָנָה

כִּי כֵן יִשָּׂא בְרָכָה / מִפִּי מַלְאַךְ שְׁכִינָה

לְשַׁבָּת אַחֶרֶת כָּכָה / תְּהֶא כֵן בִּרְנָנָה

וּמַלְאַךְ אַחֵר בְּהַכְרָחָה / עוֹנֶה בְכֹחַ וָאֵל

אָמֵן וְהִיא הַבְטָחָה / גְּדוֹלָה לְיִשְׂרָאֵל:





מִצְוַת זָכוֹר וְשָׁמוֹר / כּוֹלֶלֶת הַתּוֹרָה

בְּדִבּוּר אֶחָד לֵאמֹר / מִפִּי הַגְּבוּרָה

וּמִקְדָּשׁ עַל הַר הַמּוֹר / בִּזְכוּתָהּ בִּמְהֵרָה

וּכְאַיָּלָה שְׁלוּחָה / יְצַו הָאֵל וְיוֹאֵל

בְּהַשְׁקֵט וּבִטְחָה / מָשִׁיחַ לְיִשְׂרָאֵל:





עֲדַת קְדוֹשֵׁי עֶלְיוֹן / דְּגוּלֵי סְגֻלָּה

נֶחָמַת בִּנְיַן אַפִּרְיוֹן / יֶחֱזוּ בִּיקָר וּגְדֻלָּה

חַסְדָּךְ שׁוֹכֵן צִיּוֹן / זָכְרָה נָא לְעַם נַחֲלָה

קוּמָה נָּא לְשָׁלְחָה / מְבַשֵּׂר וְגַם גּוֹאֵל

וּבְקוֹל שָׂשׂוֹן וְשִׂמְחָה / אָז יָשִׁיר יִשְׂרָאֵל:
