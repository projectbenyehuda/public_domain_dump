


	
	שׁוֹכֵן שָׁמַיִם
	



חתימה: שלם שץ






שׁוֹכֵן שָׁמַיִם / שׁוּר מִמְּרוֹמֶיךָ

וְקַבֵּל נָא בְּרָצוֹן / תְּפִלַּת קוֹרְאֶיךָ

וְהָסֵר כַּעֲסֶךָ / וְיִגּוֹלוּ רַחֲמֶיךָ

לְמַעַן יֵחָלְצוּן / בְּיוֹם זֶה יְדִידֶיךָ

וְתִהְיֶה יָהּ לְמִשְׁעָן / וּמַחְסֶה לְדוֹרְשֶׁיךָ

הוֹשִׁיעָה אֶת עַמֶּךָ / וּבָרֵךְ אֶת נַחֲלָתֶךָ:





לְקָרְאָם הַאֲזִינָה / לְבַשׁ חֵן וּצְדָקָה

וְהַשְׁפִּיעַ שְׁכִינָה / לְנֶפֶשׁ שׁוֹקֵקָה

וְכֹפֶר הַמְצֵא נָא / וְהָסֵר לִמְצוּקָה

וּמֵרוֹעַ תְּכוּנָה / לְעַמְּךָ הַעְתִּיקָה

וְדַבֵּק אֶת לְבָבָם / יָהּ בְּיִרְאָתֶךָ:





מַהֵר נָא לִגְאוֹל / נֶפֶשׁ נֶעְשֶׁקֶת

מִתִּגְרַת סְמָאֵל / בְּנִגְעָהּ צוֹעֶקֶת

וּלְצַד מִיכָאֵל / תְּהֶא מִשְׁתּוֹקֶקֶת

וּמֵאֵשׁ גַּבְרִיאֵל / אַל תְּהֶא נִדְלֶקֶת

וּמֵעֻזָּא וַעֲזָאֵל / תְּהֶא מִתְרַחֶקֶת

וְתִתְנַהֵג עִמָּהּ / בְּמִדַּת רַחֲמֶיךָ:





שְׁלַח צֳרִי וּתְעָלָה / לְעָם הַנִּבְחָנִים

וּבַשְּׂרֵם בִּמְחִילָה / מֵחֵטְא וַעֲווֹנִים

וּבִינָה הַמְשִׁילָה / שְׁהִיא אֵם הַבָּנִים

בְּהַשְׁפֵּעַ תַּגְדִּילָה / וְתַגְבִּיר תַּחְתּוֹנִים

וְתִפְאֶרֶת בְּחֶמְלָה / וְנֵצַח וְהוֹד נוֹתְנִים

לַיְּסוֹד מַשְׁפִּיעִים / אֱלֵי מַלְכוּתֶךָ:





צֶדֶק וְשִׁבְתַּאי / וּמַאְדִּים בִּמְרוֹמָא

וְחַמָּה בַּשַּׁחַק / וְנוֹגָה הַתַּמָּה

וְכוֹכָב בָּאוֹפֶק / וּלְבָנָה שַׁלְמָא

וּמַזָּלוֹת תִּדְחַק / וְתָשִׁיר בִּנְעִימָה

תְּסַפֵּר שְׁבָחֶיךָ / וְכֹחַ מַעֲשֶׂיךָ:
