


	
	אֲיֻמָּה בִּקְשִׁי
	



חתימה: אלשבזי






אֲיֻמָּה בִּקְשִׁי עַל דָּל / הַמִּתְאוֹנֵן בְּחֶלְדּוֹ

בְּעֵת רָצוֹן יוֹפִיעַ / וְיוֹמָם יְצַוֶּה חַסְדּוֹ

וְהִכּוֹנִי יְחִידָתִי / לְצוּרֵךְ וְעִמְדִי נֶגְדּוֹ

מִי בָכֶם יְרֵא יְוֹיָ / שׁוֹמֵעַ בְּקוֹל עַבְדּוֹ:





לְקַח מוּסָר בֶּן דַּעַת / קוּם קְרָא אֶל אֱלֹהֶיךָ

וְלָךְ אוֹזֶן שׁוֹמַעַת / לְהַשְׂכִּילְךָ בְּדָתֶךָ

בְּלֵב טָהוֹר בְּלִי בַעַת / נוֹטֶה לִימִינֶךָ

וְחֵן תּוֹרָה תּוּשִׁיָּה / בְּהָקִיצָךְ תְּשִׂיחֶךָ

אַשְׁרֵי אִישׁ נְשׂוּא עָווֹן / שְׂמֹאל מוֹאֵס וּמַפְרִידוֹ:





שְׁלוּמֵי לֵב יָבִינוּ / לַעְשׂוֹת טוֹב וְגַם יוֹשֶׁר

וְאֵל שַׁדַּי בְּעֶזְרֵיהֶם / חוּט שֶׁלְּחֶסֶד קָשַׁר

וּבִתְפִלָּה וְסוֹד יֵחוּד / נַפְשָׁם וְשִׂכְלָם תִּישַׁר

כְּשׁוֹשַׁנִּים יָצִיצוּ / וְדִגְלֵיהֶם בְּאוֹת קָשַׁר

וְרוּחַ אֱלֹהִים תְּנִיחֵם / וְגּוּף יָשׁוּב אֶל יְסוֹדוֹ:





בִּינוּ זֹאת קְהַל קֹדֶשׁ / בְּטוּב מִלִּין וּרְמָזִים

וְשִׁמְעוּ סוֹד וְהַשְׂכִּילוּ / דְּבַר תּוֹרָה וְסוֹד חוֹזִים

רְצוֹן אֵל חַי בְּחֵן צַדִּיק / וְלוֹ טוֹבוֹת נִגְנָזִים

יְקָרִים הֵם וְנֶחְמָדִים / מִזָּהָב וּמִפָּזִים

בְּעֵץ חַיִּים נֶאְחָזִים / וּמַה נָּעִים פְּרִי הוֹדוֹ:





זְמָן גָּרַם לְבֶן מֶלֶךְ / וּמִבֵּית הֵיכָלוֹ גָּלָה

וּמִמַּיִם זֵדוֹנִים / בְּמִשְׁתֵּה אוֹיְבָיו חָלָה

וְלֹא מָצָא בְּאֵר שֶׁבַע / בְּיוֹם בָּא אוֹהֲבִי דָּלָה

וּמֵעָנְיוֹ בְּקוֹל הָמָה / וְהוּא נוֹדֵד בְּבֶהָלָה

רְצֵה צוּרִי בְּהַצָּלָה / וּבִימִינְךָ סְמוֹךְ יָדוֹ:





יְצַו הַחֵן חֲמוּד נַפְשִׁי / לְבֶן גָּלָה וְנִתְפַּזָּר

וְיַקְשִׁיב קוֹל תְּפִלָּתוֹ / וּמִקָּדְשׁוֹ יְהֶא נֶעְזָר

וְאַל יִשְׁפַּל בְּבֵית שִׁבְיוֹ / וְלֹא יִפּוֹל בְּיַד אַכְזָר

וְיַלְבִּישׁוֹ עֲטֶרֶת חֵן / וְדָת פֶּרֶא יְהֶא נִגְזָר

וְכָל אַרְצוֹ בְּאֵשׁ תּוּזָר / וְאֵשׁ תּוּקַד בְּתוֹךְ מִדּוֹ:
