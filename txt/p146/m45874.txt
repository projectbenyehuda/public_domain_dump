


	
	אֲרוֹךְ עַיִן בְּכוֹס יַיִן א
	



חתימה: אלשבזי





נוסח א


אֲרוֹךְ עַיִן / בְּכוֹס יַיִן / אַל יִשְׁגֶּה לִבֶּךָ

וְלֹא תִּלְמוֹד / יְדִיד לַחְמוֹד / אֶת אֵשֶׁת רֵעֶךָ

וְסוּר מֵרָע / וְטוֹב תִּזְרָע / צְדָקָה לַעֲנִיֶּיךָ

וְלֹא תַּכְלִים / לַדַלִּים / וְלֹא תִּקְפּוֹץ יָדֶיךָ

וְלֹא תִּגְאֶה / וְתִתְגָּאֶה / וְלֹא תִּבְטַח בְּעָשְׁרֶךָ

בְּלִבָּךְ גְּמוֹר / דָת לִשְׁמוֹר / וְאָז תִּצְדַּק בְּדָבְרֶךָ

זְכוֹר קוֹנָךְ / בְּיוֹם הוֹנָךְ / וֶאְמוֹר בְּיוֹם עָמְדֶךָ



טוֹב אַתָּה / וּמֵטִיב / לַמְּדֵנִי חֻקֶּיךָ:





לְלֵב נִמְהָר / יְהֶא מָזְהָר / לַעֲשׂוֹת אֶת רְצוֹן קוֹנוֹ

וְיִתְיַסֵּד / לְהָבִין סוֹד / תוֹרַת אֵל בְּהֶגְיוֹנוֹ

וְדָת מוּסָר / לֹא יֶחְסָר / וְלֹא יָמוּת בַּעֲווֹנוֹ

אֲבָל יָשׁוּב / יְהֶא חָשׁוּב / וְאָז יַחְזֹר לִמְכוֹנוֹ



שְׁמַע מִלִּין / וְאַל תַּלִּין / לְבָבִי בַּחֲטָאֶיךָ:





שְׁחִי רוּחִי / בְּמַר שִׂיחִי / לְיוֹם שׁוּבֵךְ תְּנִי צֵידָה

וְהִכּוֹנִי / וְתִשְׁכֹּנִי / בְּתוֹךְ קָהָל וְגַם עֵדָה

תְּנִי כֹפֶר/ לְגוּף הֵפַר / בְּרִית אַחִים וְגַם זָדָה

וְהָבִינִי / בְּיוֹם דִּינִי / וְעָלַי שַׁלְּמִי תוֹדָה

בְּבוֹא צוֹרֵף / כְּדוֹב טוֹרֵף / בְּיוֹם הַדִּין וְיַעְמִידָה



מְנַת כּוֹסִי / וְצוּר מַחְסִי / מְחֹל נָא לַעֲבָדֶיךָ:





בְּגָלוּתִי / וְדַלּוּתִי / אֲנִי הוֹמֶה בְּמַר בִּכְיִי

וְלִבִּי סָר / וְנִתְיַסַּר / אֱלֹהַי רַפְּאָה חָלְיִי

וְחוֹן גּוּף דַּל / אֲשֶׁר חָדַל / וְחַזֵּק יוֹצְרִי רִפְיִי

רְצֵה עָלַי / בְּמִשְׁאָלַי / רְאֵה שִׁפְלִי וְרֹב עָנְיִי



בְּךָ בִּטְחִי / סְבוֹל טָרְחִי / בְּיוֹם דָּפְקִי דְלָתֶיךָ:





זְמָן אָתָה / בְּהַשְׁחָתָה / וְהִכְבִּיד עֹל וְרֹב דַּלּוּת

וּבֶן אָמָה / מְאֹד הָמָה / וּמִתְגָּאֶה כְּרוּם זֻלּוּת

וְעֵינָיו צָר / בְּעָם נֶעְצָר / שְׁהוּא צוֹלֵל בְּבוֹר גָּלוּת

וְרָם לִבּוֹ / בְּתוֹךְ קִרְבּוֹ / בְּעַמָּךְ יִנְהֲגָה שִׁפְלוּת



אֱלֹהַי חוּן / לְדַל בָּחוּן / בְּחֵשֶׁק אַהֲבָתֶךָ:





יְשׁוּרוּן לֵךְ / וְהִמָּלֵךְ / בְּשִׂכְלָךְ לַעֲשׂוֹת טוֹבָה

וְהִתְיַשֵּׁר / בְּרֹב הֶכְשֵׁר / בְּלִי עָווֹן וְלֹא חוֹבָה

גְּבוּל תָּשִׂים / בְּטוּב מַעְשִׂים / וְאֶת דַּעְתָּךְ תְּיַשֵּׁבָה

הֲדַר זָקֵן / בְּחֵן תַּקֵּן / וְתָקוּם מִפְּנֵי שֵׂיבָה

עֲשֵׂה לָךְ רָב / בְּיוֹם תִּקְרָב / וְלֹא תִשְׁכַּב לְמַעְצֵבָה



בְּיָהּ הַשְׁלֵךְ / יְהָבֶךָ / וְהוּא יִהְיֶה מְכַלְכֶּלְךָ:
