


	
	בְּנֵי חָכְמָה
	



ז. שחיטה

חתימה: טור 23: שמי שלום […] ילוד יוסף [ספק לשבזי].






בְּנֵי חָכְמָה בְּנֵי שֵׂכֶל וּבִינָה / הֱשִׁיבוּנִי בְּעֶשְׂרִים גַּם שְׁמוֹנָה

בְּעִבּוּרִים וּבֵאוּרִים יְקָרִים / מְבוֹאָרִים לְכָל חוֹדֶשׁ וְשָׁנָה

אֲשֶׁר אוֹרָן לְהָרַמְבַּ״ם מְסָרָן / וְהִבְהִירָם יְלוּד עַמְרָם וְשָׁנָה

וְנָטַע שֵׁשׁ וְגַם חָמֵשׁ וְשִׁשִּׁים / מְשֻׁלָּשִׁים מְפוֹרָשִׁים בְּבִינָה

וְגַם שִׁבְעָה בְּאַרְבָּעָה קְבוּעָה / בְּתוֹךְ תִּשְׁעָה מְאוֹרִים הֵם לְמִשְׁנָה

שְׁלֹשִׁים מַעֲלוֹת יְמוֹת חֳדָשִׁים / וְנִדְרָשִׁים חֲסֵרִים אֵין תְּבוּנָה

וְעִבּוּר יַ״ט וְכֵיצַד כַ״ט בְּחֶרֶט / וּמַה פֶּרֶט לְמוּל כֶּרֶם נְתוּנָה

וְסוֹד תַּשְׁרַ״ק אֱלֵי צֶדֶק בְּפֶרֶק / וְסוֹד חוֹקֵק בְּמַזָּלוֹת צְפוּנָה

וּמַה עֵדִים לְבֵַין שִׁבְתַּי וּמַאְדִּים / וְאֵין חַדִּים וְכֵיצַד הַתְּלֻנּוֹת

וּמַה נִקְדָּשׁ אֱלֵי שַׁבָּת וְחוֹדֶשׁ / וּמִי גֵּרַשׁ מְעֻבֶּרֶת עֲגוּנָה

וּמַה שִּׁמֵּשׁ לְיָרֵחַ וְשֶׁמֶשׁ / וְכַמָּה מַעֲלוֹת עַד הַשְּׁכִינָה

וְכֵיצַד שׁוֹחֲטִין בִּתְרֵין לְחִטִּין / וְנִשְׁמָטִים בְּסוֹד רָחֵל וְחַנָּה

וְהָעִבּוּר בְּמוֹלָדוֹ לְכָל יוֹם / וְהַשָּׁנָה לְכַמָּה תֶּחֱזַרְנָה

מְעֻבֶּרֶת בְּסִימָנוֹת סְפוּרוֹת / בְּטַל אוֹרוֹת זְמַנִּים הֵם דְּעוּ נָא

וְהַמּוֹעֵד לְאֵיזֶה יוֹם יְסוֹעַד / וּמִי הֵעִיד וּמִי הוּא הַמְּמֻנֶּה

וְלִשְׁטָרוֹת פְּשׁוּטוֹת וַחֲטוּפוֹת / ורחב לנטות ולחיטות ישונה

וְהַתַּלְמִיד יְתַקֵּן זֶה וְיִלְמַד / וְגַם תָּמִיד תְּהֶא דַּרְכּוֹ נְכוֹנָה

וְסַכִּינוֹ רְמוּזָה בַּאֲרוֹנוֹ / לְמוּל עֵינוֹ וְכֵיצַד הַתְּלֻנָּה

וְכֵיצַד יֹאמְרוּ שׁוֹחֵט וְנִשְׁחָט / וְהַשּׁוֹחֵט דְּמוּת הַחוּט מְבֻנֶּה

וְזֶה הַסּוֹד עֲלֵי כָל הַשְּׁאָלוֹת / וּמַזָּלוֹת וְעִבּוּרִים לְשָׁנָה

הֱשִׁיבוּנִי מְעַט מָתוֹק וְנָקִי / וְזֶה חֻקִּי לְכָל מֵבִין אֲמָנָה

וְשִׁלְחוּהוּ אֱלֵי כָל הַמְּדִינוֹת / וְיָדוּנוּ לְדִין יָפֶה חֲנִינָה

שְׁמִי שָׁלוֹם עֲפַר רַגְלֵי חֲכָמִים / יְלוּד יוֹסֵף אֱלֹהַי יַעַזְרָה לִי

וְשָׁלוֹם רַב לְמוּל כָּל הַקְּהִלּוֹת / כְּנִטְפֵי יָם אֲשֶׁר עָלָה עֲנָנוֹ:
