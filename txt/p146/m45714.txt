


	
	אֵל עֶלְיוֹן
	



חתימה: אלשבזי






אֵל עֶלְיוֹן / דְּלֵה אֶבְיוֹן / אֲשֶׁר טָבַע בְּבֵית שִׁבְיָה

בְּיוֹם חָטָא / וְיָד מָטָה / וְעָזַב דּוֹד וְגַם רַעְיָה

רְצוֹנָךְ חִישׁ / לְבֵן מַכְחִישׁ / שְׁהוּא עֵרוֹם וְגַם עֶרְיָה



אֵל אָיוֹם / שְׁלַח פִּדְיוֹם / לְבַת גַּלִּים צוֹפִיָּה:





לְךָ תִּקְרָא / תַּעַזְרָהּ / וְתָקוּמָה בְעֶזְרָתָהּ

שְׁמַע קוֹלָהּ / וְהַרְפֵּא לָהּ / וְהָחִישָׁה יְשׁוּעָתָהּ

כְּבוֹשׁ עָווֹן / וּמַר דָּווֹן / וְרַפֵּא אֶת מְשׁוּבָתָהּ

תְּעוֹרֵר חֵן / בְּנֵי בִטְחוֹן / לְבַת נוּגָה וְהוֹמִיָּה



אֵל אָיוֹם / שְׁלַח פִּדְיוֹם / לְבַת גַּלִּים צוֹפִיָּה:





שְׁבוּת עַמָּךְ / אֲשֶׁר נָמָךְ / תְּשׁוֹבֵב אֶל נְוֵה אַרְצוֹ

וְיִתְאַפֵּק / דָּל דּוֹפֵק / דְּלָתֶיךָ לְהִמָּצֵא

וְאַל יִיעַף / בְּיוֹם זַעַף / בְּשִׁעְבּוּדוֹ וּבֵית לַחְצוֹ

הַצִּילוֹ / מִבַּעֲלֵי / לְשׁוֹן הָרַע וּרְמִיָּה



אֵל אָיוֹם / שְׁלַח פִּדְיוֹם / לְבַת גַּלִּים צוֹפִיָּה:





בְּטוּחָה בָךְ / יוֹנָתָךְ / צוּר מָעוֹז לְבוֹטֵחַ

רְצוֹנָךְ הָב / לְעָם נֶאְהָב / תְּהֶא מוֹחֵל וְסוֹלֵחַ

שְׁפַע הוֹדָךְ / לְבֶן עַבְדָּךְ / כְּאוֹר שֶׁמֶשׁ וְיָרֵחַ

וְהַעְבִירָה / וְהָסִירָה / תַּאֲנִיָּה וַאֲנִיָּה



אֵל אָיוֹם / שְׁלַח פִּדְיוֹם / לְבַת גַּלִּים צוֹפִיָּה:





זְכוּת עוֹרֵר / וְאַף קוֹדֵר / וְאַל תִּקְצוֹף עֲלֵי הַדָּל

אֲשֶׁר גָּלָה / בְּחַלְחָלָה / וְעָזַב הוֹן וְעִיר מִגְדָּל

וְהָשִׁיבוֹ / לְמוֹשָׁבוֹ / בְּיוֹם קָרְאוֹ וְלֹא יֶחְדָּל

בְּגִין אַבְרָם / וּבֶן עַמְרָם / וְעָקוּד וְתָם וְתוּשִׁיָּה



אֵל אָיוֹם / שְׁלַח פִּדְיוֹם / לְבַת גַּלִּים צוֹפִיָּה:





יְבַשֵּׂר שָׁב / וְטוֹב יוּשָׁב / וְיָסוּרוּ עֲווֹנוֹתָיו

בְּלֵב מוֹדֶה / יִתְוַדֶּה / וִיטַהֵר מַחְשְׁבוֹתָיו

וְלֹא יַחְמוֹד / וְלֹא יִלְמוֹד / עֲצַת נָחָשׁ וּמִרְמוֹתָיו

וְלֹא יַרְכִיל / וְלֹא יַשְׂכִּיל / הֲדַר נֶפֶשׁ וּגְוִיָּה



אֵל אָיוֹם / שְׁלַח פִּדְיוֹם / לְבַת גַּלִּים צוֹפִיָּה:
