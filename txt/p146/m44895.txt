


	
	מעמד ב: עַל סְגַן
	



החתימה: שלם יוסף






עַל סְגַן רֹאשׁ גְּאוֹנִי / אִישׁוֹנִי בָכָה

יוֹם פָּסַע לְפָנַי / וְהָעִיר נֶהְפָּכָה

יָהּ, פְּדֵה קְהַל הֲמוֹנַי / מִיּוֹם קִצְפְּךָ

אַתָּה יְוֹיָ / לֹא תִכְלָא רַחֲמֶיךָ:





שְׂשׂוֹן הָעִיר עָבַר / בְּעֵת כִּי בָא שָׁמְעוֹ

וְהָאֵבֶל גָּבַר / כִּי נֶעֱִזַב גִּזְעוֹ

וְנִכְנָס לַקֶּבֶר / וּמִי זֶה יוֹשִׁיעוֹ

וְנַפְשׁוֹ תִּתְחַבַּר / בְּעֵדֶן תִּדְרוֹכָה:





לְקָחוֹ הָאוֹרֵב / שְׁהוּא מַלְאָךְ אַכְזָר

וְעוֹדֶנּוּ קָרוֹב / בְּחָכְמָה הִתְאַזָּר

חֲצֵרוֹ נִתְחָרֵב / בְּיוֹם עָלָיו נִגְזָר

זְכוֹר רַחֲמֶיךָ יְוֹיָ / וַחֲסָדֶיךָ:





מֵרֹב הַבִּכְיָה / לְבָבִי נִתְאָנַח

וְנַפְשִׁי נָמוֹכָה / כִּי הֲדָרָהּ צָנַח

וְהָעִיר נָבוֹכָה / וּכְבוֹדָהּ נִזְנַח

יוֹדוּךָ יְוֹיָ / כָּל מַעֲשֶׂיךָ:





יְחִידָתִי סִפְדִי / עַל גְּדוֹל הַחֶבְרָה

וּבְקִינָה עִמְדִי / בְּחוּצוֹת וּבְחֶדְרָה

וּלְצוּרֵךְ הוֹדִי / וְזֶה סוֹף כָּל נִבְרָא

כִּי הַכֹּל הוֹלֵךְ / לְעֵמֶק הַבָּכָא:





וּמַה-יּוֹעִיל עֹשֶׁר / וְהַהוֹן לִבְעָלָיו

רַק עֲשׂוֹת הַיֹּשֶׁר / בְּמֵיטַב מִפְעָלָיו

וְיוֹם בּוֹא הַקֶּשֶׁר / לְהִתְעוֹלֵל עָלָיו

אִם צַדִּיק נִמְצָא / בְּחֵלֶק טוֹב זָכָה:





סוּר מֵרָע בֶּן אִישׁ / וַעֲשֵׂה-טוֹב תָּמִיד

וְרֵיחֲךָ אַל תַּבְאִישׁ / אָשׁוּרֶיךָ תַמְעִיד

לְאִוַּלְתָךְ תַּכְתִּישׁ / וּבַתּוֹרָה תַצְמִיד

כִּי הִיא חַיֶּיךָ / וְאוֹרֶךְ יָמֶיךָ:





פְּדֵה נַפְשָׁךְ הַיּוֹם / תַּחְזוֹר לִיסוֹדָךְ

וְהַמְצִיא לָהּ פִּדְיוֹם / כְּפִי הֶשֵּׂג יָדָךְ

כִּי גָּדוֹל הַיּוֹם / אֲשֶׁר הוּא יוֹם אֵידָךְ

כִּי הָרַע וְהַטּוֹב / נָתוּן לְפָנֶיךָ:
