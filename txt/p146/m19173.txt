


	
	שבח אל חי אשר ברא
	



חתימה: שלם בן יוסף




שְׁבָח אֵל חַי / אֲשֶׁר בָּרָא לְכָל חַי / וּמַלְכוּתוֹ עֲדֵי נֶצַח
נְצָחִים:



לְשׁוֹנִי דִּבְּרָה / בִּשְׁבָח וְזִמְרָה / וְהוּא נוֹרָא עֲלֵי כָּל
הַשְּׁבָחִים:



מְנַת נַפְשִׁי / וְשִׂכְלִי הָאֲנוּשִׁי / וְכִלְיוֹתַי עֲלֵי חַסְדּוֹ
בְּטוּחִים:



בְּסוֹד לִבִּי / וְשַׂרְעַפַּי בְּקִרְבִּי / אֲהוֹדֶנּוּ וְרַעְיוֹנַי
שְׂמֵחִים:



נְהוֹג עַבְדָּךְ / אֲדוֹן עוֹלָם בְּחַסְדָּךְ / וְשַׂמֵּחַ לְבָבוֹת
נֶאֱנָחִים:



יְשׁוּעָה תִשְׁלְחָה / לִבְנֵי מְנוּחָה / וְתַשְׂכִּילֵם בְּכָל סוֹדוֹת
נְכוֹחִים:



וְהָחִישָׁה / לְבַת גַּלִּים דְּרוּשָׁה / פְּדֵה לַיְשָׁה נְדוּדָה
בַּפְּתָחִים:



סְלַח פִּשְׁעָהּ / וְהָחִישָׁה לְיִשְׁעָהּ / וְהַדְרִיכָהּ לְהַלֶּךְ
בָּרְוָחִים:



פְּדֵה וּזְכוֹר / זְכוּת מֶלֶךְ מְכַרְכֵּר / לְמוּל אָרוֹן וְגַם זֶבַח
זְבָחִים:



שְׁלוֹמָךְ צַח / יְחַיֵּנוּ לְנֵצַח / כְּצִלֵּי עָב מְדֻבָּקִים
וְנָחִים:
