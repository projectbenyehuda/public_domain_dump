


	
	בָּרְכוּ לְאֵל חַי נוֹרָא עֲלִילָה
	



החתימה: אלשבזי שץ

הכתובת: לענין סעודה






בָּרְכוּ לְאֵל חַי / נוֹרָא עֲלִילָה

וְהוֹדוּ לִשְׁמוֹ / עֲלֵי כָל אֲכִילָה.





אֱלֹהִים בְּחַסְדָּךְ / רְצֵה פַּרְנְסֵנוּ

וּמֵַהוֹד כְּבוֹדָךְ / בְּחֵן הַגְבְּלֵנוּ

וְרַחֵם יְדִידָךְ / וְאַל תִּטְּשֵׁנוּ

לְהָבִין בְּסוֹדָךְ / נַפְשִׁי תְּחַלֶּה:





לְבָבוֹת וְרוּחוֹת / כָּל הַיְצוּרִים

בְּאֵל חַי בּוֹטְחוֹת / וּבִשְׁמוֹ שְׁמוּרִים

בְּשׂוֹבַע שְׂמָחוֹת / תְּשׁוֹבֵב פְּזוּרִים

וְתַצְלִיחַ אָרְחוֹת / עֲדָתָךְ בְּגִילָה:





שְׁכִינַת יְקָרָךְ / בְּצִיּוֹן תְּסוֹבֵב

וְיַלְדֵי בְחִירָךְ / לְאַרְצָךְ תְּשׁוֹבֵב

אֲשֶׁר לַהֲדָרָךְ / מְאַוֶּה וְחוֹבֵב

שְׁפַע זִיו מְאוֹרָךְ / וְהָאֵר אֲפֵלָה:





בְּשֻׁלְחָן מְהֻדָּר / תִּשְׁרֶה בְרָכָה

בְּלֶחֶם מְסֻדָּר / לְנֶפֶשׁ צְרִיכָה

פְּדֵה רָם וְנֶאְדָּר / בְּרַעְיָה נְבוֹכָה

וּפֶרֶא וְקֵדָר / תְּהֵא לוֹ נְפִילָה:





זַמֵּן פְּרוּסָה / אֵל חַי לְעַבְדָּךְ

בְּתֵימָן מְנֻסָּה / לְבָבִי בְּפַחְדָּךְ

וְנַפְשִׁי כְמוּסָה / בְּטוּחָה בְחַסְדָּךְ

וּבַעֲפַר תְּבוּסָה / מְדֻכָּה טְפֵלָה:





יְשׁוּעָה תָחִישׁ / לְזֶרַע יְשָׁרִים

וְאוֹיְבִים תַּעֲנִישׁ / מְשַׂנְאִים וְזָרִים

וְנִצְחָם תַּחֲלִישׁ / מְלָכִים וְשָׂרִים

וְעֵשָׂו תַּרְעִישׁ / וְגַם תַּכְשִׁילָה:





שְׁלָמָא וְקֻשְׁטָא / דְּמַלֵּי מְדוֹרֵיהּ

לְבַד תָּב בְּחֶרְטָא / יִסְגֵּי נְהוֹרֵיהּ

בְּסִתְוָא וְקֵיטָא / סָבֵיל לְנִירֵיהּ

וְנַפְשֵׁיהּ נְקוּטָא / כְּנֶפֶשׁ קְטִילָא:





[שלום ואמת / הממלאים את מעונו

לבד מן השב בחרטה / ירבה זהרו

בחורף ובקיץ / נושא עולו

ונפשו מאוסה / כנפש הרוגה]





צְדַקְתָּא הַב לַן / מָרַן בְּמַגָּן

בְּטָב אַשְׁגַּח עֲלַן / וְאַעְדֵּי לְנִרְגָּן

וְאַצְלַח דִּגְלַן / בְּאֶמְצַע דְּדִרְגָּן

וְאַסְגֵּי חֵילַן / דִּנְהִיר לְעֵילָא:





[צדקה תן לנו, / אדוננו, חינם

בטובה השגח עלינו / וסלק את הנרגן

והצלח את דגלנו / באמצע הארמון

והרבה את חילנו / המאיר מעלה:]





לְעַמָּךְ שְׁלָמָא / רַבָּא תְּפַרְשֵׁז

וְעַבְדָּךְ כְּקַדְמָא / יִחְדֵּי וְיַעְלוֹז

וּפָתֵיר לְחֶלְמָא / אֲבוּנָא תִּגְנוֹז

בְּגִנַּת מְרוֹמָא / דְּאִיהִי כְּלִילָא:





[לעמך שלום / רב תפרוש

ועבדך כקדם / ישמח ויעלוז

ואת פותר החלום / אבינו תגנוז

בגן המרום / שהוא מושלם:]





שְׁלָמָא יַקִּיף / מֵאֱלָהָנָא

בְּפַחְדָּא יַשְׁקִיף / עַל סָנְאַנָא

וְקוֹעֲהוֹן יַנְקִיף / מִקַָּמַנָא

כְּנִשְׁתַּן יַתְקִיף / דְּעֻלֵּיהּ סְבִילָא:





[שלום מקיף / מאלוהינו

באימה ישקיף / על שונאינו

וגרוניהם ישבר / של הקמים עלינו

אומתנו יחזק / אשר את עולו היא נושאת:]
