


	
	[לַמְנַצֵּחַ עַל־מְגִנּוֹת לִבְנֵי פָּאדוֹבָה]
	




Isac Vita Cantarini


[נולד בפָּדוֹבָה בשנת 1644 ומת שם בשנת 1723]






לַמְנַצֵּחַ עַל־מְגִנּוֹת לִבְנֵי פָּאדוֹבָה / שִׁיר
מִזְמוֹר לְהַזְכִּיר



לְךָ יְיָ חֶסֶד / אֲשֶׁר הִרְחַבְתָּ לָּנוּ בַּצָּר



שִׂפְתֵי כָזָב קָשְׁרוּ שְׁקָרִים / לֶאְסֹר אִסָּר עֲלֵי־נָפֶשׁ



עֲלִילוֹת קָמִים וְהֶגְיוֹנָם / נִצְּבוּ לְבַלְּעֵנוּ חַיִּים



הַבּוֹדְאִים לָנוּ לִדְאָבוֹן / וְהָאֹבְדִים חָשְׁבוּ לְאַבֵּד



בִּימֵי מָצוֹר לְמָצוֹר הָיִינוּ / וְהַקְּרָב אֵלֵינוּ נִקְרָב



נַפְשֵׁנוּ תָּמִיד תִּדֹּם תִּזְכֹּר / וֶאֱלוּל הָיָה אַלְלַי



בָּעֲשִׂירִי בּוֹ רַעַשׁ אֲחָזָנוּ / סִמֵּר שַׂעֲרַת בְּשָׂרֵנוּ



פָּשְׁטוּ גְדוּדִים בְּרֶגֶשׁ רַב / וּסְבִיבוֹת הַמַּחֲנֶה נִגָּשׁוּ



שְׁעָרָיו רָעֲשׁוּ מִדֶּחִי / וַיַּדּוּ־אֶבֶן בָּמוֹ



דַּלְתֵי אֲרָזִים הִתְפֹּצָצוּ / וּבְרִיחֵי בַרְזֶל גִּדֵּעוּ



פָּעֲרוּ פִיהֶם לִבְלִי־חֹק / וַיָּבֹאוּ כַבִּירִים שׁוֹטְפִים



אֲזַי הַמַּיִם שְׁטָפוּנוּ / לוּלֵי יְיָ שֶׁהָיָה לָנוּ



וַיִּשְׁלַח נוֹשְׁקֵי רוֹמֵי־קָשֶׁת / וַיְגָרְשׁוּם בְּקוֹל הֲמֻלָּה



וַיִּחַר אַף יְיָ בְּעַמּוֹ / וַיָּסֶת עוֹד שָׂטָן לְמַשְׁחִית



וַיֵּרָגְנוּ בִרְחֹבוֹת קִרְיָה / וַיַּלִּינוּ כָּל־עַם הָאֶרֶץ



וַיַּדִּיחוּ עשְׁקֵי עשֶׁק / זוֹנָה עֲנִיָּה בְשִׁית חָמָס



צָעֲקָה הַחוּצָה וַתֵּבְךְּ / וְאַחֲרֶיהָ כָּל־רִשְׁעֵי לֵב



צָעֲקָה רָצּוֹא וָשׁוֹב / וַתֹּאמֶר כִּי שָׂרַפְנוּ בְנָהּ



כָּל־פָּנִים קִבְּצוּ פָארוּר / וְהַשָּׂרִים נִבְהֲלוּ נֶחְפָּזוּ



חוֹרֵי הָעִיר עָמְדוּ בַפָּרֶץ / וַיֹּאמְרוּ לִרְגֹּם אֹתָם



אַלְפֵי סוֹרֲרִים נוֹסְדוּ יַחַד / לִנְתוֹשׁ וְלִנְתוֹץ וּלְהַאֲבִיד
שֵׁנִית



בְּכַשִּׁיל וְכֵילַפּוֹת וּכְלֵי קְרָב / וְאֵשׁ זִלְעָפוֹת בְּיָדָם



וַיָּשׁוּבוּ עוֹד לַהֲרֹס / וְכָל־לֵב נָמֵס בְּמֵעָיו



בְּנֶשֶׁף בְּעֶרֶב יוֹם / שַׁעֲרֵי צַלְמָוֶת נִפְתָּחוּ



אַךְ שָׁוְא תְּשׁוּעַת אָדָם / וְתוֹחֶלֶת אוֹנִים אָבָדָה



וַיֶּהֱמוּ חֲלִלִים בַּדָּרֶךְ / לְקוֹלָם נִבְקְעָה הָאָרֶץ



וַיִּקָּרְאוּ חֲרָשִׁים בְּאִישׁוֹן / לִשְׁמֹר אֶת־בָּתֵּי יְהוּדָה



הַשֹּׁמְרִים שֹׁאֲגִים לַטָּרֶף / שֹׁדֲדֵי לַיְלָה לְחַלֵּק שָׁלָל



שַׁדַּי הֵאִיר לַיְלָה כַּיּוֹם / וַיַּהֲפֹךְ הַקְּלָלָה לִבְרָכָה



פַּחַד וָפַחַד הָיָה לָנוּ / הַמָּוֶת וְהַפֶּצַע הִרְחִיק



וַיּוֹצִיא עַמּוֹ מֵאָסוֹן / מִקִּינָה אֶת־בְּחִירָיו



וַיִּתֵּן אוֹתָנוּ לְרַחֲמִים / לִפְנֵי שָׂרַי הַמְּלוּכָה



וַנִּשָּׂא לְבָבֵנוּ אֶל־כַּפָּיִם / וְנָשִׁיר נוֹרְאֹת יְמִינוֹ



אֶת־אֵלֶּה נִזְכְּרָה לְאֶלֶף דּוֹר / וּנְזַמְּרָה שְׁמוֹ עֶלְיוֹן



רַחוּם יִשְׁמַע מִמְּכוֹן שִׁבְתּוֹ / תְּפִלַּת עָנִי בַּשִּׁבְיָה



בִּשְׁנוֹת מִסְפָּר יֶאֱתָיוּ / יוֹשִׁיעַ מִמּוּעָקָה וְצוּקָה



יִשְׁבֹּר מֹסְרוֹת עַוְלָה / לְשַׁלַּח רְצוּצִים חָפְשִׁים



יוֹשֵׁב בַּשָּׁמַיִם יִשְׂחָק / וּלְשׁוֹנֵנוּ תִּמָּלֵא רִנָּה



תַּחַת מְגִנּוֹת־לֵב מַנְגִינוֹת תָּבֹאנָה / יִשְׂחַק לְפַחַד וְלֹא
יֵחָת



לְשִׂפְתֵי־שָׁוְא יַעֲקֹב יִצְחָק / וּפַחַד לֹא יִהְיֶה עוֹד.




(1648)
