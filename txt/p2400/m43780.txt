


	
	אֲסַדֵּר שְׁבָחָא
	



סימן: א-י. לחן: שוכנת בשדה.






אֲסַדֵּר שְׁבָחָא / לֶאֶלָהּ כָּל רוּחָא / דִי עֲבַד רְוָחָא / לְבָנִין קַדִּישִׁין

בְּיָדָא תַקִּיפָא / וְחַרְבָּא שְׁלִיפָא / דָּן עַמָּא חֲצִיפָא / וְכָל זֵינֵי חַרְשִׁין

גַּלִּי לַן נִסּוֹהִי / עֲבַד לַן אָתוֹהִי / אַחְזִי לַן תִּמְהוֹהִי / דְּאִנּוּן תַּקִּיפִין

דָּכְרִין יְהוּדָאֵי / יָת מוֹפְתִין סַגִּיאֵי / מְחָא יָת מִצְרָאֵי / בְּעֶשֶׂר מַכְתָּשִׁין

הֲרַס יָת קִרְוֵיהוֹן / אַבֵּד יָת גֵּיתֵיהוֹן / שְׁצָא יָת טַפְלְהוֹן / וְגֻבְרִין וְנָשִׁין

וְאַפֵּיק יָת עַמֵּיהּ / אַצְשֵׁךְ לוֹן רַחֲמֵי / עֲבַד בְּדִיל שְׁמֵיהּ / נְהֵיר לוֹן חֲמִישִׁין

זְכוּת אַבְהָתַנָא / הוּא אַגֵּין עֲלַנָא / דְחָס רַחֲמָנָא / עַל רוּחִין וְנַפְשִׁין

חֲמָן בְּנֵי עַמָּא / תֻּקְפָּא דְרָב עַלְמָא / כַּד בַּזַּע יָת יַמָּא / בְּזְעִין נְפִישִׁין

טַבַּע יָת מִצְרָאֵי / בְּמַיִן סַגִּיאֵי / נְחַתוּ לְעֻמְקָאֵי / בְּמָנִין וּלְבוּשִׁין

יִשְׁפַּר הַאי שְׁבָחָא / קָדָם אֱלָהּ רוּחָא / וְיִשְׁלַח מְשִׁיחָא / לְבָנִין קַדִּישִׁין.
