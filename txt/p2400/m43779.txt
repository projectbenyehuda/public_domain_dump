


	
	שִׂמְחוּ נָא
	



שיר לכבוד ברכת הלבנה הרומזת על גאולתנו שבב״א (שתבוא במהרה בימינו אמן)

סימן: חותם חיים חי






שִׂמְחוּ נָא שִׂמְחוּ נָא / בְּבִרְכַּת הַלְּבָנָה

שִׂמְחוּ נָא שִׂמְחוּ נָא / בְּשִׂמְחַת הַלְּבָנָה



חִדּוּשׁ הַיָּרֵחַ / בּוֹ לִבִּי שָׂמֵחַ / וְצָר נְנַגֵּחַ / בְּאוֹר הַשְּׁכִינָה

וּפוֹעֵל הָאֱמֶת / פְּעֻלָּתוֹ אֱמֶת / וְתִהְיֶה נִשְׁלֶמֶת / בִשְׁלֵמוּת בִּנְיָנָהּ

תִּתְחַדֵּשׁ גְּבֶרֶת / בְּעִלּוּי עֲטֶרֶת / עַד כֶּתֶר תִּפְאֶרֶת / בְּתִפְאֶרֶת בִּינָה

מַצְמִיחַ יְשׁוּעָה / הַתֵּר הַשְּׁבוּעָה / בֹּא וְהִוָּדְעָה / בָּעִיר הָעֶלְיוֹנָה

חִדּוּשׁ הַנְּשָׁמוֹת / יִהְיֶה בָעוֹלָמוֹת / יָאִירוּ הַשֵּׁמוֹת / מֵעֵבֶר וּפִנָּה

יִמָּשֵׁךְ אוֹר עֶלְיוֹן / הַנֶּעְלָם מֵרַעְיוֹן / מִמָּקוֹר הָעֶלְיוֹן / עַד מִדָּה תַחְתּוֹנָה

יִשְׂרָאֵל הוּא יָחֵל / בִּנְיַן בַּיִת וָחֵיל / יַעֲקֹב וְרָחֵל / בֹּאוּ נָא בְרִנָּה

מַלְכִּי לִי תַעֲזֹר / וּלְמַחְלִי שִׂים מָזוֹר / וְחִישׁ נָא תַחֲזֹר / עֲטָרָה לְיָשְׁנָהּ

חַלּוּ נָא פְנֵי אֵל / יִשְׁלַח נָא הַגּוֹאֵל / יְקַבֵּץ יִשְׂרָאֵל / עַם זוּ אֲשֶׁר קָנָה

יָהּ שְׁמַע תְּחִנָּה / וּשְׁלַח נָא חֲנִינָה / אָנָּא הוֹשִׁיעָה נָּא / אָנָּא הַצְלִיחָה נָּא.
