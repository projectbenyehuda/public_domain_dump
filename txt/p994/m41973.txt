


	
	[חֲרוֹן אַפִּי כְּאֵשׁ בּוֹעֵר וּבִבְשָׂרִי כְּמוֹ קוֹצִים]
	




͝ – –

חֲרוֹן אַפִּי / כְּאֵשׁ בּוֹעֵר / וּבִבְשָׂרִי / כְּמוֹ קוֹצִים

עֲלֵי מִקְרֶה / אֲשֶׁר קָרָה / בְּלֶכְתֵּנוּ / מְשֻׁבָּצִים

בְּיוֹם שַׁבָּת / לְשׁוּט בַּחוּץ / לְמָקוֹם שָׁם /

בְּנֵי סִכְלוּת / מְשַׁנִּים אֶת־ / לְבוּשֵׁיהֶם / וּמִתְפָּרְצִים

וְכָל מַבְאִישׁ / לְאִישׁ, וַיִּשְׂ־ / טְמוּנוּ בַּ־ / עֲלֵי בֵיצִים

הֲקִימוּנוּ / שְׂחוֹק לָהֶם / כְּמַטָּרָה / אֱלֵי חִצִּים

וְקוֹל אוֹמֵר /“יְרֵה” וַיּוֹר / לְאִישִׁים עִ־ / בְרִים נִצִּים

וּבָחַרְנוּ / חֲנוֹת חָנוּת / וְהַנּוֹגְשִׂים / אֲזַי אָצִים

וְגַם שָׁמָה / רְדָפוּנוּ / וְלֹא הוּנַח / לְנִלְחָצִים

עֲדֵי נֶפֶשׁ / בְּכַף שַׂמְנוּ / וְיָצָאנוּ / מְאֹד רָצִים

וְכָל־אֶחָד / כְּנֶשֶׁר קַל / מְדַלֵּג הָר / וְחָרִיצִים

בְּכֵן בָּאנוּ / וְנִמְלַטְנוּ / יְדֵי שׁוֹסִים / וּפָרִיצִים

אֲהָהּ אֵל חַי / תְּנָה אוֹתָם / עֲלֵי־אֵשׁ אוֹ / עֲלֵי עֵצִים

וְהַדָּבָר / אֲשֶׁר יָרוּ / יְהוּ מֵהֶם / מְקֻצָּצִים

וְאָנוּ נָא / נְבָרֵךְ יָהּ / וּמֵעַתָּה / וְעַד־קִצִּים

וְלֹא נוֹסִיף / בְּיוֹם קֹדֶשׁ / לְהִתְעָרֵב / לְבֵין לֵצִים

וְרַק תּוֹרָה / וְזִיו אוֹרָה / וּבַמּוֹרִים / נְהִי רוֹצִים

אֲשֶׁר יוֹרוּ / כְּלָלֵי־דָת / בְּגַן עֶדְנָה / מְלֹא צִצִּים.
