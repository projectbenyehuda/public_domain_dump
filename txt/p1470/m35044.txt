


	
	אֱמֶת עַיִן רוֹאָה
	



כח – בראשית מב א – וירא יעקב









א


אֱמֶת עַיִן רוֹאָה / אֵל לְתָם הֶרְאָה / בְּעֵת חוק רָעָב / נַפְשׁוֹ לֹא תִרְעָב



[ג……] לַאֲגֻדָּה חִבֵּר / פִּלּוּלוֹ מֵהֶם לֹא עִבֵּר / דָּץ כִּי יֶשׁ [שֶׁבֶר] / וְאֵל בְּעֶזְרוֹ לְסֵבֶר



הִפְצִיחַ אַל תִּירָאוּ / כִּי לֹא תִרְעָבוּ / [ו……..או / וְלָמָּה תִּתְרָאוּ





[אֱלֹהָיו…… ] חַיָּיו / מְשַׂגֵּב בָּנָיו / וּמְמַגֵּן הֲמוֹנָיו.




ג


[מִנַּחַל] עֲדָנֶיךָ / תְּרַוֶּה עֲנָוֶיךָ / נֶצַח עַל יְרֵאֶיךָ / הִתְמַדְתָּה עֵינֶיךָ



שַׂבַּע אֲיֻמָּתָךְ / מִדֶּשֶׁן בֵּיתָךְ / עוֹרֵר אַהֲבָתָךְ / וְתֵבוֹשׁ אוֹיַבְתָּךְ



פַּצְתָּה לָנוּ שִׁבְרוּ / וְלִמְכַלְכֵּל סִבְרוּ / צַהַל שֹׂבַע תָּשׁוּרוּ / וּלְמַשְׂבִּיעַ רָצוֹן תָּשִׁירוּ.




ד


אוֹמֵר וְעוֹשֶׂה גּוֹזֵר וּמֵקִים / נוֹרָא…… נֶפֶשׁ צַדִּיק נֵצַח לֹא יַרְעִיב



ובוטח…….. [לֹא] יַחֲסוֹרוּ / וְקָרָא רָעָב לְכָל דָּרֵי אָרֶץ.




ה


קוֹחַ עַיִן פָּקַח לָאָסוּר בְּבֵית סֹהַר / …………… עַד נֶפֶשׁ

רְצָפוֹ אַהֲבָה חָכְמָה כְּפִתְרוֹן… / …….. נַעֲשָׂה מְכַלְכֵּל וּמַשְׁבִּיר

שִׁמַּעְתָּ לניני………. / כִּי לֹא נֶחְלוּ עַל שֵׁבֶר יוֹסֵף



תָּכְפוּ עֲשָׂרָה ירא……. קְדוֹשִׁים / כִּי בַּעֲדַת אֵל נִצָּב קָדוֹשׁ.





וירא יעקב




ו


אָהוּב כִּי הָיָה בְּפַתְרוֹס / בִּנְיַן צוֹעֲנִים הוּא יַהֲרוֹס



[ג]………… לַחְמוֹ יִפְרוֹשׂ / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.





דִּבָּה עַ[ל אֶחָיו] הוֹצִיא / הָמְרַר בְּיַד נוֹגְשִׂים אָצִים



וַיִּשְׂטְמוּהוּ [בַּעֲלֵי חִצִּים] / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.





זוֹכֵר אֶת חֲלוֹמוֹתָיו / חוֹזֶה סְבִיב אֲלֻמּוֹתָיו



[ט]……..מְאוֹרוֹת בְּמַעֲלוֹתָיו / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.





יָקָר דָּבָר בַּלֵּב [שָׁמָר] / [כ]……. לְהִשְׁתַּחֲווֹת בַּלֵּב אָמָר



לִדְבַר שְׂפָתָיו אֵל גָּמָר / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.





מַקְנִיאִים בּוֹ רָדוּ לְמִצְרָיִם / נִבְהֲלוּ מִמֶּנּוּ וְשָׁחֲחוּ אַ[פָּיִם]



סִיחַת פִּיו בָּא לְחַדְרֵי מַעָיִים / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.





עֲלֵי עַיִן שַׁלִּיט בצ……. / פַּעְנֵחַ קָרְאוּ שְׁמוֹ בְּצַהַל



צָפְנַת אַבְרֵךְ קוֹרְ[אִים] ב…… / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.





קֵץ יָדוֹעַ תֵּידַע בָא / רָעָב לְמַעֲנוֹ לָאָרֶץ בָּא



[ש]לח………. אִישׁ וְנִבָּא / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.





תָּמַהּ לְדִבְרֵי הַקּוֹרוֹת / תוכ…… חֲקוּרוֹת



תְּפִלָּה הֶעְתִּיר לְעוֹנֵה עֲקָרוֹת / לְעוֹלָם דְּבָרָךְ נִצָּב קָדוֹשׁ.




ז


אֶחָיו בְּדִבְרֵי גְרוֹנָם / דִּבְרֵיהֶם ה……. [ו…..]



זָכַר חֲלוֹמוֹת טְעָמָיו / יָרֵחַ וְכוֹכָבִים [ל…]



מְאַלְּמִים נִצְּבוּ סְבִיבָיו / עֵת פְּתִיחַת צב….



צ[ק…..] רְאוֹת שַׁלִּיט / תְּהִלַּת תִּפְאֶרֶת תָּאֳרוֹ.




ט


אִישׁ חוֹלֵם חֲלוֹמוֹת וּפוֹתֵר חֲלוֹמוֹת / בְּעֵת שָׁר א….. חֲלוֹמוֹתָיו



גָּשׁ וְנִתְגַּבֵּר וְדִבֵּר קָשׁוֹת / דָּע וְהִתְנַכֵּר…… מְרַגְלִים אַתֶּם



הִשְׁמִיעוּ קוֹל יַחַד כֵּנִים אֲנַחְנוּ / ו……. עִבְרִים בְּנֵי אָב אֶחָד



זֹאת לֹא תַחְשֹׁב בַּלֵּב עַל עֲבָ[דֶיךָ] / חֵשֶׁק נֹעַם פִּינוּ לָאֵל כְּנֹעַם חַיּוֹת.





טְלָאִים אַחִים שְׁנֵים עָשָׂר אֲנַחְנוּ / יָצָא אֶחָד מֵאָב וְאֵינֶנּוּ

כִּי כְּנֶפֶשׁ אָב קָטוֹן עִם אָב / לִמְאֹד נַפְשׁוֹ קְשׁוּרָה בְּנַפְשׁוֹ.
