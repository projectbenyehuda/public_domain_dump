


	
	עַד בּוֹא מַשּׁוּאוֹ
	



לח / שמות ח טז – השכם בבקר והתיצב





ו


[ט…..] עַד בּוֹא מַשּׁוּאוֹ / לְדָבָר אַחֵר הִשִּׂיאוֹ



יַעַן …………… / יֻכְבַּד עָלָיו מַשָּׂאוֹ.





כִּי אִם אֵינָךְ מְשַׁלֵּחַ עַם מֵעַמָּךְ / ………………..עמך

לְמָחָר יְהִי אוֹת בֵּינִי וּבֵינָךְ / בֵּין עַמִּי וּבֵין עַמָּךְ.





מִבּוּקָה אֲשֶׁר אָדִין אֻמִּים / וּמִמַּכָּה אֲשֶׁר אֶשְׁפֹּט לְאֻמִּים



נַמְתָּ לָדִין וְלִשְׁפֹּט עֲנָמִים / לְהַפְלִיא פְדוּת בְּעַם ….מים.





סִימַת פָּנֶיךָ כָּבוֹד אֲמַלֵּא / וּפְנֵי צַר קָלוֹן תְּמַלֵּא

עֶרֶב יָבוֹא וְאַרְצוֹ תִתְמַלֵּא / בֶּעָרוֹב אֲשֶׁר תִּמָּלֵא.





פיצח עז ענותך לא ימלל / כִּי עַזּוּת לִבּוֹ יְמַלֵּל



צַר אֲשֶׁר בְּעֵינָיו הִתְהַלֵּל / בֹּקֶר לְעֵינֶיךָ יִתְחַלֵּל.





קִדְּמוּהוּ בְּחִיל וּבְבֶהָלָה / ב………. מָלֵא חַלְחָלָה



רוֹדֶה ב……. לו נחלה / יֵרָדְה בְּמַכָּה נַחֲלָה.





ש…… א…………../ ………..למושב ברכו



תּוֹעַ אִוַּלְתּוֹ תְּסַלֵף דַּרְכּוֹ / וְעַל תָּמִים יִזְעַף דַּרְכּוֹ.





ובכן והתיצב לפני פרעה




ז


וְהִתְיַצֵּב בְּאֵימַת אֱיָלוּת אֹמֶץ לִפְנֵי אוֹמֵר אֱלוֹהַּ אָנִי

וְהִתְיַצֵּב בְּבַעַת בִּיעוּת בַּהַל לִפְנֵי בוֹזֵה בְנִי בְכוֹרִי

וְהִתְיַצֵּב בִּגְבוּרַת גְּאוֹן גַּסּוּת לִפְנֵי גְבִיר גּוֹי גָּמוּר

וְהִתְיַצֵּב בְּדֹהַר דָּבָר דָּבוּר לִפְנֵי דוֹחֵק דּוֹדֵי דָּגוּל

וְהִתְיַצֵּב בְּהֵלֶךְ הַדְרַת הוֹד לִפְנֵי הוֹנֶה הֶתֶל הֲפַכְפָּךְ

וְהִתְיַצֵּב בְּזִיו זַעְפַּת זַעַף לִפְנֵי זָר זֵרִים זָרַע

וְהִתְיַצֵּב בַּחֲזוּת חֲנִינַת חֶסֶד לִפְנֵי חָנֵף חֵרֵף חָי

וְהִתְיַצֵּב בְּטַעַם טֶכֶס טֹהַר לִפְנֵי טוֹעֵן טַפִּים טִיט

וְהִתְיַצֵּב בְּיִחוּד יֵילִיל יָגוֹן לִפְנֵי יוֹזֵם יעף יוֹקֵשׁ

וְהִתְיַצֵּב בְּכִיווּן כִּשְׁרוֹן כּוֹחַ לִפְנֵי כָמוּס כֹּבֶד כָּעַס

וְהִתְיַצֵּב בְּלִבּוּב לֵב לָבִיא לִפְנֵי לֵץ לוֹחֵם לְקוּחִים



וְהִתְיַצֵּב בְּמֶלֶל מְהִירַת מ….ת לִפְנֵי מַפְרִיךְ מִקְהֲלוֹת מַמְלֶכֶת



וְהִתְיַצֵּב בְּנוֹאַם נֹעַם נֶצַח לִפְנֵי [נ….נ…..נ…..]נים



וְהִתְיַצֵּב בְּסִיחַ סִימָן סוֹד לִפְנֵי סוֹנֵא סְגוּלֵי סֵבֶר

וְהִתְיַצֵּב בְּעֵרוּם עִוּוּת עִנְיָן לִפְנֵי עוֹמֵס עֲמוּסִים עוֹנִי

וְהִתְיַצֵּב בְּפֶצַח פּוֹעֵל פֶּלֶא לִפְנֵי פָץ פָּקַת פְּלִילִיָּה

וְהִתְיַצֵּב בְּצַהֲלַת צִוּוּי צוּר לִפְנֵי צַר צוֹרֵר צוּרִים

וְהִתְיַצֵּב בְּקִדּוּם קִיהוּי קְרָבַיִם לִפְנֵי קוֹצֵף קְנוֹת קְנוּיִים

וְהִתְיַצֵּב בְּרֹגֶז רִיצַת רַעַד לִפְנֵי רַע רִצֵּץ רֵעִים

וְהִתְיַצֵּב בְּשַׁחֲרִית שְׁזִיפַת שֶׁמֶשׁ לִפְנֵי שׁוֹטֶה שׁוֹבֵר שׁוֹקָיו

וְהִתְיַצֵּב בְּתֹאַר תְּהִלַּת תֹּקֶף לִפְנֵי תוֹעִי תּוֹמֵךְ תָּעַב.




ז 2


אֻמָּה אַבֵּד בְּאֵימָתָךְ / בְּהַשְׁכָּמַת בֹּקֶר בַּלָּהוֹת

גַּדַּע גַּאֲוַת גְאוֹנָה / דַּעֵיךְ דְּלִיקַת דּוֹהֵר

הוֹמֵם הֶמְיַת הֲמוֹנָהּ / וְתִקְרָא וְתִקְרַב וְתָקוּם

זְרִיקַת זִיקֵי זַעְפָּךְ / חַלְחֵל חִקְרֵי חֲדָרֶיהָ



טַאֲטֵי טִבּוּר טִירוֹתֶיהָ / יַחֵת יְקָר יֹשֶׁר …..ותיה



כֻּלָּהּ כָּלִיל כַּלֵּה / לַבְּטָה ליהקי לֶהָבוֹת

מַרְדּוּת מַכּוֹת מְצֵירִים / נוֹרָא נַתֵּק נְגִידֶיהָ

סַחֵף סִימָן סָרָה / עֲדִינָה עוֹלְלָה עַלִּיזָה

פְּרַע פְּאֵר פְּקֻדָּתָהּ / צַמֵּת צִבְאוֹת צחיה

קַרְקֵר קְהִלּוֹת קְרִיאֶיהָ / רְעוֹץ רִבְבוֹת רוֹבֶיהָ

שִִׁיתָהּ שְׁאִיַּת שַׁמָּה / תַּפִּיל תֹּקֶף תִפְאַרְתָּהּ.




ח


וְתִשָּׂגֵב לְבַדָּךְ / וּתְיַחֵד שְׁמָךְ / בְּעֹצֶם
כּוֹחָךְ / וּבְאֹמֶץ חֵילָךְ / בְּנִפְלְאוֹת עֻזָּךְ / בְּנוֹרְאוֹת
עִזּוּזָךְ / בְּגֹדֶל יָדָךְ / בִּגְאוֹן יְמִינָךְ / בְּתֹקֶף זְרוֹעָךְ
/ בְּכֹבֶד מַשָּׂאָךְ / בִּנְטִיַּת קַוָּךְ / בִּנְשִׂיאַת נִסָּךְ
כִּי לְשִׁמְעָךְ / וְלִשְׁמָךְ וּלְזִכְרָךְ יַזְכִּירוּ
וְיַשְׁמִיעוּ יַחַד צִבְאוֹתַי רוּם וְתַחַת עֲנִיָּה אַחַת קְדֻשָּׁה



אֶחָת.
