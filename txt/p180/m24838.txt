


	
	שְׁנֵי זֵיתִים
	








שְׁנֵי זֵיתִים / נִכְרָתִים / בְּגַן נָעוּל יַצְהִירוּ

לְרֹאשׁ קְהָתִי / וְאֶפְרָתִי / שְׁתֵּי עֲטָרוֹת יַכְתִּירוּ

מוּל מְנוֹרָה / הַטְּהוֹרָה / כְּמוֹ נֵרוֹת יַזְהִירוּ

הֵן בְּמַשְׂאֵת / כָּל אִישׁ אֶת / רֵעֵהוּ יַעֲזֹרוּ

הֵן בְּמַחֲנֶה / אֶל מוּל פְּנֵי / הַמְּנוֹרָה יָאִירוּ.





לְבֶן פֹּרָת / וְצִיץ יִקְרַת / אֲשֶׁר הָיָה כְּצִיץ נוֹבֵל

וְלַנִּמְשָׁח / אֲשֶׁר הוּשַׁח / כְּמוֹ אֳנִיָּה בְּלִי חוֹבֵל

זְכֹר מַרְאָה / אֲשֶׁר רָאָה / זְכַרְיָהוּ בְּעִיר בָּבֶל

כְּמוֹ זְכַרְתָּם / וְהוֹשַׁעְתָּם / אֲזַי עַל יַד זְרֻבָּבֶל

זְכֹר זַרְעָם / לְהוֹשִׁיעָם / בְּעֵת תִּקְרָא שְׁנַת יוֹבֵל

וְשִׂים אוֹתָם / מְקוֹם אֲבוֹתָם / הֱיוֹת מוֹשְׁלִים בְּכָל תֵּבֵל



וּמֵעַל יַד / אִישׁ צַיָּד / טַבַּעְתָּם יָסִירוּ.





מַמְלָכָה / מְמֻשָּׁכָה / לְעִיר צִיּוֹן תְּשִׁיבֶנָּה

וּבַת כְּבוּדָה / בְּרָב עֲבֻדָּה / לְבֵית אִמָּהּ תְּבִיאֶנָּה

וְהַגְּבִירָה / וְהַצְּפִירָה / לְרֹאשׁ דּוֹדָהּ תְּקִימֶנָּה

וּמִצְנֶפֶת / מְעֻלֶּפֶת / לְרֹאשׁ אַהֲרֹן תְּשִׂימֶנָּה

וְהַתָּמִיד / אֲזַי תַּעֲמִיד / וְהַמִּנְחָה תְּרִימֶנָּה

וְהַמֶּרְקָח / אֲשֶׁר לֻקַּח / וְלֹא רְאִיתִיו עַד הֵנָּה

וְהַקְּטֹרֶת / מְקֻטֶּרֶת / לְאַהֲרֹן יַקְטִירֶנָּה

וְהַנֵּרוֹת / בִּמְנוֹרוֹת / בְּנֵי אַהֲרֹן יַבְעִירוּ.





הַנְּכֵאָה / וְהַצְּמֵאָה / אֲשֶׁר תָּמִיד לְךָ תֵילִיל

סְמֹךְ יָדָהּ / בְּבֶן דּוֹדָהּ / אֲשֶׁר נִקְרָא שְׁמוֹ חַכְלִיל

וְהַכְנִיסֵם / לְגַן בֹּשֶׂם / וְחֻפָּתָם בְּהוֹד תַּכְלִיל

וְהוֹלִיכֵם / וְהַמְלִיכֵם / בְּהַר צִיּוֹן וְהַגָּלִיל

וְחֹק יָשָׁר / אֲזַי יוּשַׁר / עֲלֵי נֵבֶל וְגַם חָלִיל

זְעֹם עוֹזְבֵי / שְׁמָךְ וְאוֹהֲבֵי / חֲזוֹן שֶׁקֶר וְשָׁוְא וֶאֱלִיל

וְעַם פְּרוּכִים / יְהוּ בְרוּכִים / כָּל רוֹאֵיהֶם יַכִּירוּ.
