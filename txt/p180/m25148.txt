


	
	שְׁבָחָה בְחִכִּי
	








שְׁבָחָה בְחִכִּי / וְשִׁירָה כְעֶרְכִּי / אֲתַנֶּה לְמַלְכִּי / בְּפַחַד וְאֵימָה

לְטוּבוֹ אֲקַוֶּה / וְצִדְקוֹ אֲחַוֶּה / וְחַסְדּוֹ יְצַוֶּה / לְגֶזַע אֲיֻמָּה

מְרוֹמָם בְּפִי כֹל / וְקַדְמוֹן וְיָכֹל / וְתוֹלֶה כְאֶשְׁכֹּל / אֲדָמָה בְלִימָה

הֲדָרוֹ אֲרַנֵּן / וְהוֹדוֹ אֲשַׁנֵּן / וְנֶגְדּוֹ אֲכוֹנֵן / שְׂעִפַּי בְחָכְמָה

הֲמוֹנֵי זְבוּלָה / וְנִינֵי סְגֻלָּה / מְתַנִּים לְנַעְלָה / תְהִלָּה עֲצוּמָה

הֲלֹא הוּא לְבַדּוֹ / וְהַכֹּל בְּיָדוֹ / וְזִכְרוֹ וְהוֹדוֹ / בְּפִי כָל נְשָׁמָה.
