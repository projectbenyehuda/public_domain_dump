


	
	שְׁעֵה זִכְרוֹן דָּתִי
	








שְׁעֵה זִכְרוֹן דָּתִי כְמַעֲשֶׂה / לְמָעֹז הֱיֵה לִי וּלְמַחֲסֶה / וְעַל כָּל פְּשָׁעַי תְּכַסֶּה



- - - -



אֱלֹהַּ שׁוֹכֵן מְרוֹמוֹת / אִלֵּף צִיר אַחֲרֵי מוֹת / שְׁנֵי בְנֵי אַהֲרֹן

בּוֹא אַל יָבוֹא בְכָל עֵת / בְּקָדְשִׁי פֶּן יִבָּעֵת / דַּבֵּר אֶל אַהֲרֹן

גָּמוֹר יִגְמֹר בְּחַיִל / גֶּשֶׁת פָּר וָאַיִל / בְּזֹאת יָבוֹא אַהֲרֹן



דִּלֵּג וְרָחַץ מְכֻבָּד / דָּרַשׁ בִּגְדֵּי הַבַּד / וְהָיוּ עַל אַהֲרֹן



הֵכִין שְׂעִירִים וְאַיִל / הִגִּישׁ פָּרוֹ בְחַיִל / וְהִקְרִיב אַהֲרֹן

וְהוֹדָה עַל חַטָּאתוֹ / בַּעֲדוֹ וּבְעַד בֵּיתוֹ / וְכִפֶּר אַהֲרֹן

זִמֵּן שְׂעִירִים לְהַעֲמִיד / זוּג גּוֹרָלוֹת יַצְמִיד / וְנָתַן אַהֲרֹן

חַטָּאת לַשֵּׁם שִׁפַּר / חָזַר אֵצֶל הַפָּר / אֲשֶׁר לְאַהֲרֹן

טָפַל וִדּוּי מַטּוֹ / טָל פָּרוֹ וּשְׁחָטוֹ / וַיִּפֶן אַהֲרֹן

יִקַּח מַחְתָּה גְבֶרֶת / יְמַלֵּא חָפְנָיו קְטֹרֶת / וּבָא אַהֲרֹן

כָּמַס וְשָׁפַךְ נָחוּץ / כִּסָּה וְיָצָא לַחוּץ / וַיִּקְרָא אַהֲרֹן



[ל ל אַהֲרֹן]



מַזֶּה שֶׁבַע לְהָעִיר / מִהֵר לִשְׁחֹט שָׂעִיר / וַיַּעַשׂ כֵּן אַהֲרֹן



נִגַּשׁ לְטַהֵר אֲרִיאֵל / נִצַּב לְהַזּוֹת כָּאֵל / וְכִלָּה אַהֲרֹן



סָבַב לְהַקְרִיב שָׂעִיר / סֵטִים עָלָיו יָעִיר / וְסָמַךְ אַהֲרֹן

עָווֹן נִסְלַח וְנִמְלַח / עֵת לַעֲזָאזֵל נִשְׁלַח / עַל פִּי אַהֲרֹן

פָּשַׁט בְּגָדִים לְבֵשָׁם / פָּנָה וְהִנִּיחָם שָׁם / אֶת בִּגְדֵי אַהֲרֹן

צְהֻבֵּי זָהָב הֶעְטִיר / צָעַד חֵלֶב לְהַקְטִיר / וַיִּקְרַב אַהֲרֹן

קָדַם לְהַקְרִיב עוֹלָה / רָצַף לְעַצְמוֹ כְּלוּלָה / זֶה קָרְבַּן אַהֲרֹן

שָׁלְמָה עֲבוֹדַת כִּפּוּר / תַּחְשֹׁב כְּקָרְבַּן סִפּוּר / עַל כַּפֵּי אַהֲרֹן

שַׁדַּי כְּמֵאָז תָּצִיץ / לְהָשִׁיב סְפִירַת הַצִּיץ / עַל מֵצַח אַהֲרֹן

מְהֵרָה תִּמְשַׁח לְגַן רָטֹב / הֱיוֹתוֹ כַּשֶּׁמֶן הַטּוֹב / עַל הַזָּקָן זְקַן אַהֲרֹן.
