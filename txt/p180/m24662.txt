


	
	שְׁדוּדִים נְדוּדִים
	








שְׁדוּדִים נְדוּדִים / לְצִיּוֹן תְּקַבֵּץ

מְכוּרִים בְּלֹא הוֹן / תְּשׁוֹבֵב לְמַרְבֵּץ

הָשֵׁב עֲבוֹדַת / לוֹבְשֵׁי תַשְׁבֵּץ

תִּרְעָת שִׁמְעָת / מִשְׁפְּחוֹת יַעְבֵּץ

לְהוֹדוֹת לְזַמֵּר / לְשִׁמְךָ עֶלְיוֹן / אַתָּה תָקוּם תְּרַחֵם צִיּוֹן.





לְגוֹיֵי אֲרָצוֹת / תָּחִישׁ שְׂאֵת נֵס

יִנּוֹן תְּגַבֵּר / וּמָתְנָיו תְּשַׁנֵּס

עֲתָרַי תְּאַסֵּף / וּפוּצַי תְּכַנֵּס

תְּחַדֵּשׁ גְּאֻלָּה / כְּטָנֵס וְחָנֵס / צַדִּיק לוֹבֵשׁ צְדָקָה כַּשִּׁרְיוֹן.





מְשַׂנְאִי יְדַבֵּר / עַזּוֹת בְּפָנָי

מְבַקֵּשׁ לְאַבֵּד / תִּקְוַת אֱמוּנָי

יְרִיבֵנִי עַד אָן / תּוֹחִיל לְעֵינָי

עֲנִיתִיו כִּי לֹא יִזְנַח / לְעוֹלָם אֲדֹנָי / מָעֹז לַדָּל מָעֹז לָאֶבְיוֹן.





הָשֵׁב גְּרוּשָׁה / לְנַחֲלַת יְרוּשָׁה

וּפְקֹד אֲנוּשָׁה / בְּיַד צַר עֲנוּשָׁה

אֲכוּלָה בְכָל פֶּה / נְתוּשָׁה וּרְטוּשָׁה

וְחַדֵּשׁ צִדְקַת / אָבוֹת שְׁלֹשָׁה / וְזָרִים תַּכְנִיעַ כְּחֹרֶב בְּצָיוֹן.
