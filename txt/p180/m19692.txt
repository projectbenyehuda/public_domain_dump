


	
	[אֲלֵיכֶם יֹֽדְעֵי נֶהִי]
	



ומן אשעארה אלסבעה ירתי פי בעץ אלמפקודין עלי אלאטלאק ואלתרגיז [ומשבעת
שיריו (אחד שבו) הוא מקונן על אחד הנעדרים בצורה כוללת ובמשקל הרגז]


אֲלֵיכֶם יֹֽדְעֵי נֶהִי / אֲנִי אֶקְרָא בְשִׂיחָתִי



לְכוּ בֹּאוּ וְהִקָּבְצוּ / הֲמוּ עִמִּי בְּהֶמְיָתִי.



וּמַהְרוּ נָא וְגַם שִׁלְחוּ / מְקוֹנֵנוֹת וְתָבֹאנָה



וְתִשֶּׂאנָה הֲמוֹן נֶהִי / וּמִסְפֵּד מַר תְּקוֹנֵנָּה.



וְאַל תִּבְכּוּ מְיֻדָּעַי / לְמֵת נִקְבַּר בְּאַדְמָתוֹ



בְּכוּ בָכֹה לְאִישׁ הָלַךְ / אֲשֶׁר לֹא שָׁב אֱלֵי בֵיתוֹ.



שְׁלָחָהוּ גְּבִיר עַמִּים / הֱיוֹת תַּעַל לְמַכּוֹתָם



וְהָאדֹמִים אֲחָזוּהוּ / דְּקָרוּהוּ בְחַרְבוֹתָם.



אֲשֶׁר שֻׁלַּח בְּתֹם נַפְשׁוֹ / וּבִלְבָבוֹ כְּמוֹ אִישׁ תָּם



וְהַצָּרִים אֲסָרוּהוּ / בְּשִׁקְרֵיהֶם וּפַחְזוּתָם.



מְשַׂנְאִים טָֽמְנוּ פַח לוֹ / לְצוֹדֶד־לוֹ וְגַם חֻצָּב



וְהָעֵת צָֽרְחוּ עָלָיו / אֲזַי צָרַח בְּקוֹל נֶעְצָב.



וּמִתְחַנֵּן לְעֻמָּתָם / וְחֹשְׁבִים רַע בְּלֵב עָקֹב



אֲשֶׁר לֹא יֶחֶזֶה שַׁדַּי / וְלֹא יִרְאֶה קְדוֹשׁ יַעְקֹב.



הַרְפּוּנִי וְעִזְבוּנִי / וְאַל תִּרְאוּ בְרָעָתִי



וּמַה בֶּצַע בְּדָמִי יֵשׁ / וּמַה תּוֹעִיל הֲרֵגָתִי.



לְבָבוֹ…… /…… וְגַם נִדְאָב



וְאָז צָרַ[ח] בְּמַר [נַ]פְשׁוֹ / כְּמוֹ יָתוֹם וְאֵין לוֹ אָב.



וְאֵין מַקְשִׁיב וְאֵין עֹנֶה / וְאֵין שֹׁעֶה לְשַׁוְעָתוֹ



בְּיוֹם קָרְבָה נְפִילָתוֹ / בְּיוֹם [בָּ]אָה פְקֻ[דָּתוֹ].





וְאִלּוּ [מֵת] בְּמַגֵּפָה / וְאִם אֵין..מות..[בָּ]ר



וְלֹא יָמוּת וְהוּא נִדְקָר / וְכָל גּוּפוֹ כְּמוֹ מִכְבָּר.



יֹלַדְתִּי וְהוֹרָתִי / אֲהָהּ צִירִים אֲחָזוּנִי



וְצָד[וּ שְֹׂ]טְנַי נַפְשִׁי / וּבַחֶרֶב הֱמִיתוּנִי.



וְלֹא אֶרְאֵךְ לְעוֹלָמִים / וְלֹא אַבִּיט לְפָנָיךְ



בְּכִי תָמִיד וְאַל נָא תִ־ / תְּנִי פוּגָה לְעֵינָיִךְ.



אֲהָהּ צָרִים רְדָפוּנִי / וְלַמָּוֶת הֲדָפוּנִי



וּבִרְמָחִים נְשָׂאוּנִי / וְלַחַיּוֹת נְתָנוּנִי.



וְלִפְנֵי [מִי] בְּמַכְאוֹבִי / בְּמַר אֶהְמֶה וּמַר אֶצְרָח



..הנה אֲ[נִי] אֶפְנֶה / וְאֵין מָנוֹס וְאֵין מִבְרָח.



בְּכִי עֵדָה וְהֵילִילִי / וְגֹזִּי אֶת הֲדַר נִזְרֵךְ



בְּרֹאשֵׁךְ הַ[עֲ]לִ[י] עָפָר / בְּ….. מְאוֹר נֵרֵךְ.





וְאַבִּירֵךְ וְאַ[דִּירֵךְ] /……… [וּהוּ]



וְאֶת דָּמוֹ בְּחַרְבוֹתָם / בְּדי….. שְׁפָכוּהוּ.





[אֱדוֹ]ם אַל תִּשְׂמְחִי עַל זֹאת / וְתָשִׂישִׂי לְיוֹם אֵידִי…
