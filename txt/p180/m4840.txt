[אשר גנב] / שלמה אבן גבירול

[על פרידת ידידו]

È — — — / È — — — / È — — / / È — — — /È — — — / È — —


אֲשֶׁר גָּנַב לְבַב דּוֹדָיו בְּהוֹדוֹ / וְעַל-כָּל-אֹהֲבָיו הִקִּיף מְצוּדוֹ



הֲטוֹב לֵאוֹר בְּאוֹר פָּנָיו פְּנֵי-כֹל / וְלֹא-זָרַח כְּבוֹדוֹ עַל-יְדִידוֹ



וּבֵית-אַהַב תְּמוֹל הֵאִיר בְּיָפְיוֹ / הֲטוֹב כִּי-יֶחֱשַׁךְ הַיּוֹם לְנֹדוֹ.
