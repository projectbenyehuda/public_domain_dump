


	
	שׁוֹבֵב כְּלִילִי
	




שׁוֹבֵב כְּלִילִי  /   לְרֹאשׁ בֶּן חַכְלִילִי



לְהַחֲלִיף אֱלִילִי  /  עִם רוֹעֶה אֱוִילִי



מֵהַר גְּלִילִי    /  יְרַנֵּן קוֹל חֲלִילִי



הָעֵת תְּחַדֶּשׁ [לִי]  /  יְמֵי שִׂמְחַת גִּילִי


  


וְעַל אֹזֶן רָגִיל   /  תְּשַׁוֶּה הֶעָגִיל



וְלוֹבֵשׁ הַפְּתִיגִיל  /  בְּרֹב שִׂמְחָה וָגִיל   /   יְצַפְצֵף עַל חָלִיל


  


לִשְׁכָּה וְגַם דְּבִיר  /  בְּבֵית קֹדֶשׁ תַּחְבִּיר



וְכֹהֵן אַבִּיר   /   בְּמִזְבֵּחַ תַּגְבִּיר



קְטֹרֶת לְהַעֲבִיר /   לִפְנֵי אֵל כַּבִּיר


  


וְלָשֵׂאת כַּף מוּכָן  /  וְלַעֲלוֹת לַדּוּכָן



אֲשֶר בּוֹ אֵל שָׁכַן  / וְעֵת כַּפָּיו תָּכַן    /   יְרַנֵּן עַל קוֹל צְלִיל


  


מִזְרָק וְגַם סַף  /   בְּבֵית פְּרָחִים יְאֻסַּף



וְשׁוֹמְרֵי הַסַּף  /   יְהִי זְרוֹעָם נֶחְשָׂף



וְנוֹתְנֵי שִׁיר נִכְסָף /  יְשִׁירוּן שִׁיר לְאָסָף


  


וְיַחֲזִיקוּן כְּמָנוֹר  /  כְּלֵי נֵבֶל וְכִנּוֹר



וְיֶהֱמוּ כַצִּנּוֹר  /   בְּשַׁעַר נִיקָנוֹר  /     וְגַם עַל הַר גָּלִיל


  


הֲמוֹנִים הֲמוֹנִים  /  בְּבֵית אֵל יַעֲלוּ שָׁם



וְשִׂמְחַת עוֹלָם   /  תִּהְיֶה עַל רֹאשָׁם



וְעֵינָם תֶּחֱזֶה   /  כָּל תַּאֲוַת נַפְשָׁם


  


יְשׁוּעָה וּבְרָכָה  /  וּמוֹשֵל מַמְלָכָה



וְכֹהֵן מַעֲרָכָה  /   וּנְשִׂיא הַמְּלוּכָה   /   וְנָבִיא וְגַם פָּלִיל
