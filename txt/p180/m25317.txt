


	
	שְׁמַע נִיב עַם
	






שְׁמַע נִיב עַם / אֲשֶׁר הוּעַם / בְּעֹל עַבְדּוּת מְאֹד יְהִימוּן

[לְהַשְׁמִ]יעָם לְהוֹדִיעָם / אֳרָחוֹת     
יְרִימוּן

מְחַל [פִּשְׁעָם / שְׁעֵה שַׁוְעָם]      יְטַעְמוּן

הֱיֵה זְרוֹעָם / אֵל רוֹעָם /     [מוּן

קְרָא זַר]עָם / בְּקוֹל מֻנְעָם / בְּטָחוּן נָאֳמָךְ   [מוּן

יִהְיוּ מֵתֶיךָ נְבֵלָתִי יְקוּמוּן]



לְקוֹל בִּכְיָם / וּבִטּוּיָם     [מִים

רְאֵה עָנְיָם / פְּדֵה] שִׁבְיָם / בְּיַד צָרִים [מְאֹד] עֲגוּמִים

     מִים

צְבִי עֶדְיָם / וְהוֹד [יָפְיָ]ם      [מִים]

וּפִרְיָם / אֵל חַי וְקַיָּם    [מוּן]





מְעוֹן רִבְצָם /      [צָם  דוֹר

מְח]וֹז חֶפְצָם / דְּבַר יוֹעֲצָם /      [דוֹר

וּמַפְרִיצָם / וּמַ]קְרִיצָם / יְהִי שָׁמֵם [לְדוֹר וָדוֹר]

עֲבֹר      [צָם /     צָם      גְדֹר

צָם      /      צָם / וְתִקְוָתָם] בְּךָ יְשִׂימוּן





[הֱ]יֵה מִגְדָּל / לְעַם מֻבְדָּל / מִכָּל לָשׁוֹן וְאֻמָּה

נִבְזֶה וְדַל /           בְּגָלוּת נַפְשׁוֹ עֲגוּמָה

          כְּמוֹ דַל           אליך ש [מָה]

צִירוֹ גָ[דַל] / וְנִתְדַּלְדַּל / בְּבֵין קֵדָר [יְטוּר] וְד[וּ]מָה

שְׁמָךְ [יִגְ]דַּל / וְלֹא יֶחְדַּל / וְיִחוּדָךְ לְעַד יְקִימוּן



חלף אֵבֶל / וְטָשׂ חֵבֶל / תְּנָה לָהֶם פְּאֵר וְצָהֳלָה

[זְרֻבָּבֶל] / בְּכָל תֵּבֶל / [יְ]שׁוֹבֵב חִישׁ לְעִיר תְּהִלָּה

קְ[רָא] יוֹבֵל / מְקוֹם כֶּבֶל / לְהַחְפִּישׁ אֹם לְהַסְגִּילָה

[וְ]נַחְלַת חֶבֶל / [עֲלֵי נֵ]בֶל / יָשִׁירוּ אֲזַי בְּמִלָּה

מְ[חַץ] בָּבֶל / וְדוֹר מְנַבֵּל / נְוֵיהֶם נְוָלִי יִתְּשָׂמוּן.
