


	
	אֲחַלֶּה פְנֵי אֵל
	








אֲחַלֶּה פְנֵי אֵל / צוּר יִשְׂרָאֵל / אֲקַוֶּה פְדוּת אֵל / בְּיוֹם זִכָּרוֹן



בְּזָכְרִי זְדוֹנַי / יִפְּלוּ פָנַי / יְחֻדְּשׁוּ יְגוֹנַי / בְּבוֹאִי לְזִכָּרוֹן



גָּדְלָךְ אֲקַוֶּה / מָעֹז וּמִקְוֶה / הָסֵר מַעֲוֶה / מִִסֵּפֶר זִכָּרוֹן



דְּרָשׁ נָא עֲדָתְךָ / בְּרֹב הַדְרָתְךָ / בְּרֹב צִדְקָתְךָ / נַקֵּה בְּיוֹם זִכָּרוֹן



הֲמוֹנֵי מְרוֹמִים / בַּקְּשׁוּ רַחֲמִים / עַל גֶּזַע תְּמִימִים / בְּיוֹם צְדָקָה וְזִכָּרוֹן



וְאָבוֹת קְדוֹשִׁים / הַרְבּוּ רְחָשִׁים / עַל עֵַם חֲבוּשִׁים / שׁוֹטְחֵי זִכָּרוֹן



זְעָקָה הַרְבּוּ / עַל עַם נִשְׁבּוּ / וּבְגָלוּת הָחְבּוּ / וְאֵין זִכָּרוֹן



חַנּוּן חַסְדּוֹ / יַעְטֶה כְמַדּוֹ / לִזְכֹּר עַבְדּוֹ / וְיִהְיֶה זִכָּרוֹן



טָעוּתוֹ יָפֵר / וּזְדוֹנוֹ יְכַפֵּר / וְיַמְצִיא לוֹ כֹפֶר / בְּיוֹם זִכָּרוֹן



יְגַלֶּה צִדְקוֹ / לְנַחֲלָתוֹ וְחֶלְקוֹ / וְיִמְחֶה דוֹחֲקוֹ / מִסֵּפֶר הַזִּכָּרוֹן



כּוֹבֵשׁ עֲווֹנוֹת / הַעֲבֵר זְדוֹנוֹת / וּנְפָשׁוֹת אֲנוּנוֹת / חֲבֹשׁ בְּיוֹם זִכָּרוֹן



לְחַסְדְּךָ אֶפְנֶה / פּוֹדֶה וְקוֹנֶה / כִּי לִי תַעֲנֶה / ׳סָלַחְתִּי׳ בְּיוֹם זִכָּרוֹן



מַחֲסֶה וָעֹז / בְּךָ צוּר אָעֹז / כִּי אַתָּה מָעֹז / לַדַּל בְּיוֹם זִכָּרוֹן



נוֹרָא וְאָיֹם / הַמְצֵא פִדְיוֹם / וּסְלַח הַיּוֹם / לְעַמְּךָ בְּיוֹם זִכָּרוֹן



שַׂגֵּב סְגֻלָּה / מִכָּל מַחֲלָה / וְתִהְיֶה נַחֲלָה / לְחֹק וְזִכָּרוֹן



עֲנֵה לְנֶחְשֶׁלֶת / תַּעֲרֹג כְּאַיֶּלֶת / כְּמוֹ דַל עַל דֶּלֶת / תַּעֲטֹף בְּיוֹם זִכָּרוֹן



פַּתַּח כַּבְלָהּ / וְיַשֵּׁר שְׁבִילָהּ / וִיהִי חֶבְלָהּ / חָקוּק בְּסֵפֶר זִכָּרוֹן



צְפֵה לְנִפְזֶרֶת / בְּיַד צַר נִמְסֶרֶת / וַעֲזֹר גְּבֶרֶת / וְתִהְיֶה לְזִכָּרוֹן



קַבֵּל שִׂיחָהּ / וְהָכֵן אָרְחָהּ / וְיֶעֱרַב לִקְחָהּ / בְּיוֹם זִכָּרוֹן



רַחֵם צְנוּעָה / נְדוּדָה וְנוּעָה / תַּרְבֶּה שַׁוְעָה / לְחַפְּשָׂהּ יוֹם זִכָּרוֹן



שׁוֹבֵב כְּמִטֶּרֶם / עַם דּוֹמֶה לְכֶרֶם / וְתַשְׁלִיךְ בְּתוֹךְ זֶרֶם / חֶטְאָם בְּיוֹם זִכָּרוֹן



תַּמְצִיא מָנוֹחַ / לְעַם הוֹלְכֵי שְׁחוֹחַ / כַּאֲשֶׁר לְנֹח / וְשַׂמְתּוֹ לְזִכָּרוֹן.
