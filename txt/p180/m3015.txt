


	
	[פקד ברק-שחק באש יקדת]
	



וקאל איצֹא

— — È — / — — È — / — — È — // — — È — / — — È — / — — —


פָּקַד בְּרַק-שַׁחַק בְּאֵשׁ יֹקֶדֶת / עַיִן נְדֻדֵי לַֽיְלָה פֹּקֶדֶת



פָּרַשׂ כְּנָפָיו עַל-פְּנֵי-שַׁחַק כְּמוֹ / שִׁדָּה תְפַזֵּז בַּלְּבוּשׁ רֹקֶדֶת



יָחֹג וְיָנוּעַ כְּחוֹלַת אַהֲבָה / וּכְנַעֲרָה לֹא-נִסְּתָה לָלֶדֶת



בָּרָק אֲשֶׁר לֹא-אֶחֱזֶה עַד-אֶחֱזֶה / הַלַּֽיְלָה בַּלַּֽיְלָה אֹבֶדֶת



חִלָּה פְּנֵי חשֵׁק פְּנֵי דוֹדִים אֲשֶׁר / יִשְׂחוּ בְּיַם-חֵשֶׁק כְּמוֹ רַפְסֹדֶת



יִבְרֹק בְּשׁוּרוֹ בַּת בְּתוּלָה תִהֲלַךְ / בֵּינָם וְהִיא כִי-תַעֲלֶה יֹרֶדֶת



תָּנִיד בְּרֹאשׁ כִּי-נִשְּׂאָה עַל-שַׁעֲלֵי / מַשְׁקִים כְּשַׂר מַשְׁקִים בְּיוֹם הֻלֶּדֶת



תִּזְרַח כְּמוֹ תַנִּין בְּטָלֶה וַתְּהִי / כְּאֵשׁ בְּעֵינַי עַל-כְּפוֹר שֹׁקֶדֶת



מִזְרָק חֲמָתוֹ בָּעֲרָה בוֹ בַּעֲבוּר / יַחְפֹּץ לְהַסְתִּירָהּ וְהִיא מֹרֶדֶת



כָּחַשׁ בְּשָׂרָהּ בַּכְּלִי וָאֶחֱשֹׁב / בַּת-הַכְּלִי מֵאֵין כְּלִי עֹמֶדֶת



תִּמְשֹׁךְ לְבַב-שֶׁמֶשׁ בְּחוּט אַהַב עֲדֵי / כִּי-תַעֲמֹד עַל-כַּף אֱנוֹשׁ סֹגֶדֶת



רֵעַי קְרָאוּהָ מְכַשֵּׁפָה הֲכִי / תָצוּד יְגוֹנֵיהֶם בְּלִי מַלְכֹּדֶת



וּבְמַעֲלוֹת רוּחִי יְגוֹניִם גָּעֲרָה / בָהֶם לְמַעַן מֵאֲנוּ לָרֶדֶת



יָלְדָה אֲדָמָה רִבֲבוֹת בָּנִים עֲלֵי / בִרְכַּי וְעוֹד לֹא-עָֽמְדָה מִלֶּדֶת



שׁוּבָה מְיֻדָּעִי אֲשֶׁר הִרְחִיק נְדֹד / נַפְשִׁי כְּמוֹ צִפּוֹר לְךָ נוֹדֶדֶת



לֹא-לָֽמְדָה לָנוּד לְמִיּוֹם נִבְרְאָה / אוּלָם לְעוֹפֵף נֶגְדְּךָ עֹמֶדֶת



לֹא-נָֽטְשָׁה עַל-רֹב זְמַנִּים אַהֲבַת / רֵעִים אֲשֶׁר הִיא עַל-אֱמֶת נוֹסֶדֶת



תֶּהְגֶּה לְשׁוֹנִי צִדְקְךָ תָּמִיד הֲכִי / מֵאֵין יְגִיעָה חַסְדְּךָ מַגֶּדֶת



קַח זֹאת אֲשֶׁר רַעְיַת בְּרָקִים נִקְרְאָה / כִּי פַאֲתֵי מִדְבָּר וְיָם עֹקֶדֶת



יֵחַם לְבַב- סַהַר בְּעֵת כִּי-יֶחֱזֶה / אוֹרָהּ וְשֶׁמֶשׁ תָּאֳרָהּ חֹמֶדֶת.
