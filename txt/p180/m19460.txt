


	
	[שֶׁמֶשׁ מְרוֹמִים הֲכִי אַתְּ גְּבִירָה]
	



וקאל איצ͘א




שֶׁמֶשׁ מְרוֹמִים הֲכִי אַתְּ גְּבִירָה / וְאִם אַתְּ בְּפִי כָל אֲנָשִׁים צְעִירָה

וְעָלִית וְרָחַק מְקוֹמֵךְ עֲדֵי כִי / חֲשָׁבוּךְ עֲלֵי רֹאשׁ שְׁחָקִים צְפִירָה

וְאַל תִּתְמְהִי כִּי אֲנִי כֵן בְּעֵינָם / כְּיֶלֶד בְּזוּי גֵּו וְנֶפֶשׁ יְקָרָה

וְאִישׁ בֶּאֱמֶת לֹא יְדָעֵךְ יְדַמֶּה / אֲשֶׁר תֵּֽלְכִי אַט וְלֶכְתֵּךְ מְהֵרָה

וְכָזֹאת יְדַמּוּ אֲחוֹתֵךְ בְּרָאשֵׁי / חֳדָשִׁים חֲסֵרָה וְהִיא לֹא חֲסֵרָה

וְכָכָה יְשִׂימוּן יְדִידַי יְדִידֹת / יְדִידָם חֲסֵרָה וּבִמְאֹד יְתֵרָה

אֲשֶׁר הָֽלְכוּ לַחֲזוֹת צִיץ עֲרֻגוֹת / וְשָׁם חִבְּקוּ אַחֲרֵי כֵן עֲקָרָה

הֲקִנֵּאת לְאוֹרֵךְ בְּשׁוּרֵךְ פְּנֵיהֶם / וְהִלְבַּשְׁתְּ אוֹרָם כְּתֹנֶת שְׁחֹרָה

וְאֵיךְ תּֽוּכְלִי אוֹר לְהַסְתִּיר וְצוּרַת / פְּנֵיהֶם פְּנֵי הָאֲדָמָה מְאִירָה

וְרֵיחַ בְּשָׂמִים בְּרֵעַי מְרַגֵּל /לְאַפִּי וְרוּחַ תְּרַחֵף בְּשׂוֹרָה

בְּגַדְתֶּם בְּרֵעַ עֲזַבְתֶּם וְנַפְשׁוֹ / בְּנֶפֶשׁ צְרוֹר אַהֲבַתְכֶם צְרוּרָה

וְהָיָה לְבָבוֹ לְמַיִם וְצָמֵא / יְאַוֶּה הֱיוֹת רֹאשׁ פְּתָנִים תְּמוּרָה

וְלֹא כָהֲתָה עֵין חֲסַר נוּם בְּקִרְבּוֹ / מְנָעָהּ כְּלוֹת הַשְּׁאֵר מִנְּעוּרָה

וְתֻשְׁלַךְ בְּלֶב־יָם גְּוִיַּת אֲחִיכֶם / אֲזַי הָֽיְתָה מִכְּלוֹתָהּ סְתוּרָה

וְלוּ דַק נְמָלִים נְשָׁכוֹ נְשָׂאוֹ / בְּפִיהוּ וְאָבַד בְּפִי הַמְּעָרָה

שְׁנוֹתָיו בְּתֵבֵל כְּשִׁשִּׁית שְׁמֹנֶה / וּמַחְצִית שְׁלשִׁים וְשִׁשִּׁית עֲשָׂרָה

וְכִמְעַט יְהִי אוֹת בְּשַׂעְרַת לְחָיָיו / אֲשֶׁר הִיא שְׁחוֹרָה וְתָשׁוּב צְחֹרָה

חֲמָסִי יְדִידַי עֲלֵי הָאֲדָמָה / אֲשֶׁר כִּֽשְׂכִיר יוֹם לְפֹתִים מְכוּרָה

וְלוּלֵי כְסִילִים אֲשֶׁר נִשְּׂאוּ־בָךְ / נְשָׂאתִיךְ וְשַׂמְתִּיךְ לְכַפַּי נְעוּרָה

אֲדַמֶּה לְקַבֵּר יְגוֹנִי בְנַפְשִׁי / וְהָיָה יְגוֹנִי לְנַפְשִׁי קְבוּרָה

וְתִדַּד שְׁנָתִי לְזֹאת כָּל שְׁנָתִי / כְּאִלּוּ שְׁנָתִי לְעֵינַי אֲסוּרָה

וְהָמוּ קְרָבַי עֲדֵי יָם חֲשַׁבְתִּים / וְהָעֵת קְרָאתִיו לְבָבִי מְדוּרָה

וְאֶבְעַר לְעַם בֹּעֲרִים הֵם בְּנַפְשָׁם / כְּזִמְרַת אֲנָשִׁים וְהֵם כַּזְּמֹרָה

וְשֵׁפֶל וּבוּז לָֽבְשׁוּ אוֹן אֲבָל כִּי / כְתֹנֶת כְּסִילוּת בְּעֹרָם תְּפוּרָה

עֲוֹנָם יְגַלּוּ וְשִׂנְאָה יְכַסּוּ / וְיִרְאוּ בְעֵינָם וְעֵינָם סְגוּרָה

מְעֻוָּת כְּגָמָל וְשָׁפָל כְּנָמָל / וְנָלוֹז כְּשׁוּעָל וְסוֹרֵר כְּפָרָה

אֱמֹר לַפְּתָאיִם שְׁבוּ אַל תְּנוּעוּן / לְהָעִיר חֲמַת אִישׁ כְּנוֹעַ כְּבָרָה

אֲשֶׁר שָׁת תְּבוּנָה לְמַקֵּל בְּיָדוֹ / וּמֵזַח מְזִמּוֹת לְמָתְנָיו חֲגֹרָה

וְאַל תַּאֲמֵן בָּם אֲבָל כִּי בְעֵדוּת / יְעִידוּן לְאִמָּם אֲשֶׁר הִיא עֲקָרָה

וְאֵיךְ יִפְרְצוּ לַעֲלוֹת אֶל מְקוֹמִי / וְיָדָם לְהַשִּׂיג לְגָדֵר קְצָרָה

וְגֻלַּת שְׁחָקִים עֲצוּרָה בְלִבִּי / וְתֵבֵל בְּגֻלַּת שְׁחָקִים עֲצוּרָה

וְאִם הֵם יְקַלּוּ דְבָרִי אֲשֶׁר עַז / וְחֹלָה תְחַלֵּל וְאִשָּׁה מְצֵרָה

וְאִם הֵן כְּשַׁיִת אֲנִי הַמְּדוּרָה / וְאִם הֵם כְּרוּחַ אֲנִי הַסְּעָרָה

וְאִם הֵם כְּלָבִים אֲנִי הַֽמְגָרֶה / וְאִם הֵם כְּשָׁמִיר אֲנִי הַֽמְגֵרָה

וְלוּ יֵֽדְעוּן כָּל יְפִי הַוְּרָדִים / עֲזָבוּם וְלֹא נִשְׂרְפוּ בַבְּעֵרָה

וְאִם נָֽשְׁכָה הַדְּבוֹרָה בְשַׂר אִישׁ / וְלֹא יָֽדְעָה אַחֲרִיתָהּ דְּבוֹרָה.
