


	
	[בְּיוֹם קָצַר אַפִּי]
	



ורתֹאה איצֹא [וקונן עליו עוד]




בְּיוֹם קָצַר אַפִּי / בְּמָשָׁל אֶפְתַּח־פִּי / הֲשַׁחַק עַפְעַפִּי / וְאִם־לֶחְיִי שִׂרְיוֹן

וּמַה־יִּתְאַב תָּאוֹב / לְקוֹל שִׂיחַ כָּאוֹב / לְבָבִי מִמַּכְאוֹב / וְיָדִי מֵרִפְיוֹן

וְיִיגַע וַיָּעַף / מְאֹד מִדְּבַשׁ רָעַף / וְנִשְׁאַר הָעַפְעַף / כְּחוֹמַת בַּת־צִיּוֹן

וְחֵמָה יִמָּלֵא / לְסוֹד לֹא־יִפָּלֵא / וְאֵיךְ לֹא־יִגָּלֶה / וְהוּא עַל־גִּלָּיוֹן

גְּעַרְתִּיו שׁוּב לֵבָב / וְאַל־תֵּלֵךְ שׁוֹבָב / הֲלֹא כִמְעַט סָבַב / לְךָ הַכִּלָּיוֹן

וְאֵיך תֶּאְהַב מֻטֶּה / וּמִדַּעַת תִּשְׂטֶה / וְאֶל־תֵּבֵל תִּטֶּה / וְהִיא כַחְלוֹם חֶזְיוֹן

וְאֵיךְ חַיִּים תִּבְחַר / וְתֵדַע כִּי מָחָר / אֱמֶת יִדְבַּק אַחַר / אֱלִימֶלֶךְ כִּלְיוֹן

וְתִשְׂמַח בִּרְנָנָה / בְּאֶרֶץ כִּמְלוּנָה / כְּמוֹ שִׂמְחַת
יוֹנָה / עֲלֵי הַקִּיקָיוֹן

וְהָאָדָם הֵלֶךְ / וּבַחֹשֶׁךְ הוֹלֵךְ / וְהַנֶּפֶשׁ מֶלֶךְ / וְהַגּוּף אַפִּרְיוֹן

וְהַמָּוֶת יִגָּל / וְיִשְׁלַח הַמַּגָּל / וְנָרֹץ הַגַּלְגַּל / אֱלֵי בוֹר הַשְּׁאָיוֹן

וְעֵינֵי גִבּוֹר טַח / וְלַשֶּׁמֶשׁ יִשְׁטַח / וְעַל־קַשְׁתּוֹ יִבְטַח / וְיִתְעַל בַּסִּרְיוֹן

בְּכֹל יִפְנֶה יִרְשָׁע / וְכָל־הַיּוֹם יִפְשָׁע / כְּצַדִּיק
עִם־רָשָׁע / וְעָשִׁיר עִם־אֶבְיוֹן

וְחַיִּים בַּהֶרֶג / יְקַפֵּד כָּאֹרֵג / וּבָהֶם יִשְׂתָּרֵג / כְּהִשְׂתָּרֵג שִׁרְיוֹן

וּמֵהֶם לֹא־יִשְׂגֶּה / לְמַלֵּא בָם כָּל־גֵּיא / וְשִׁיתוֹ שַׁח לַגֵּא /וְהַבּוּז לִגְאֵיוֹן

וְיֹאכַל כֶּחָסִיל / בְּנֵי־עַיִשׁ וּכְסִיל /
וּכָל־בַּעַר וּכְסִיל / לְשִׁנָּיו נִקָּיוֹן

וְהִרְאַנִי חֶלֶד / בְּעוֹדֶנִּי יֶלֶד / אֲשֶׁר תֵּבֵל תֵּלֵד / בְּטֶרֶם הֵרָיוֹן

וְלֹא מָעַד מָעוֹד / אֲשֻׁרִי בָהּ מֵעוֹד / כְּמוֹ־הוֹד הָיָה עוֹד / לְרֵעִי
נִסָּיוֹן

בְּעֵת בָּא בִגְבוּרָה / מְסָעֵף כָּל־פֻּארָה / וְלָקַח
הַצּוּרָה / וְעָזַב הַדִּמְיוֹן

וְעֵינוֹ זָב בִּכְיָהּ / וְנַפְשׁוֹ דַּק
עָבְיָהּ / וְעָלְתָה־לָּהּ אֶל־יָהּ / לְגֵיא הַחִזָּיוֹן

וְנִשְׁאַרְתִּי חֵרֵשׁ / כְּעֵץ מֵאֵין שֹׁרֶשׁ / וְנִכְרַת הַחֹרֶש / אֲשֶׁר הָיָה
חֶבְיוֹן

וְאַנְחָתִי רַבָּה / בְּמוֹט מוֹט הַקֻּבָּה / וְעַל־אָרוֹן נִשְׁבָּה / וְיֵקַר הַפִּדְיוֹן

לְבָבִי הַנִּמְזָח / בְּיָגוֹן לֹא יִזָּח
/ עֲזֹב שִׁירֵי מִרְזָח / וְהָמֵר שִׁגָּיוֹן

וְכַף אֶל־כַּף הָנִיף / וְיָד עוֹד אַל־תָּנִיף / וְהוֹרֵד הַצָּנִיף / וְהָסֵר הַגִּלְיוֹן

קְרַע וּפְרֹם כָּל־מָד / לְמִקְדָּשׁ כִּי־נִשְׁמָד / וְהָיָה כָל־מַחְמָד / לְעֵינֵינוּ כִּלְיוֹן

וְקוֹנֵן עַל־מַקַּל / פְּאֵר בַּפָּז נִשְׁקַל / וְהַחֶלֶד נָקַל / בְּעֵינָיו מִדִּפְיוֹן

תְּהִי שָׁלוֹם נַפְשׁוֹ / בְּסוּר אֶל־מִקְדָּשׁוֹ
/ וְיַזְרִיחַ שִׁמְשׁוֹ / כְּדִבְרֵי הִגָּיוֹן

תְּמוֹל הִיא שֹׁכֶבֶת / בְּחִיל וּבְמַעְצֶבֶת / וְהַיּוֹם יֹשֶׁבֶת / בְּצֵל סֵתֶר עֶלְיוֹן.
