


	
	[עֲזֹב הֶגְיוֹן בְּמַהְלַל כָּל־סְגָנִים]
	



ולה איצ֗א


עֲזֹב הֶגְיוֹן בְּמַהְלַל כָּל־סְגָנִים / וְאַל־תִּתְחַר בְּנִרְגָּנִים וְשׁוֹנִים

וְהוֹדֵה רַב יְקוּתִיאֵל לְבַדּוֹ / בְּנוֹ יִצְחָק אֲדֹנֵי הָאֲדֹנִים

אֲשֶׁר לָבַשׁ תְּהִלָּה כַּמְעִילִים / וְעָטָה הוֹד כְּמַעְטֵה הַסְּדִינִים

עֲנִיתִיהוּ וְרַעְיוֹנַי נְבֻכִים / וְאֵין־בִּי לַעֲנוֹת כֹּחַ וְאוֹנִים

לְאַט עָלַי וְתֵן פּוּגָה לְלִבִּי / הֲיָם־אָנִי וְאִם כֹּחִי אֲבָנִים

בְּיַעַן כִּי־הֲלָמוּנִי כְאֵבִים / הֲדָפוּנִי אֲפָפוּנִי יְגוֹנִים

וְהִנֵּה הֱחֳלִי בִּלָּה בְשָׂרִי / וְהִכְאִיב לֵב וְהִבְהִיל
רַעֲיוֹנִים

וְאֶתְגֹּלֵל בְּצִירִים עַל־יְצוּעַי / כְּמִתְגֹּלֵל עֲלֵי קוֹצִים וְצִנִּים

וְיָגַעְתִּי בְּאַנְחָתִי וְאֵידִי / וְנִלְאֵיתִי נְשֹׂא תוּגוֹת וְאוֹנִים

וְקָם עָלַי כְּאַכְזָרִי זְמַנִּי / וְיֵרֶד־בִּי לְמַעְמַקֵּי שְׁאוֹנִים

וְכָשַׁלְתִּי וְנָפַלְתִּי בְרִשְׁתּוֹ / וּמִכְמַרְתּוֹ וּפֶחָיו הַטְּמֻנִים

זְמַן בֹּגֵד אֲשֶׁר בֹּגְדִים יְאַהֵב / וְשָׁב אֶל כָּל־בְּלִיַּעַל בְּפָנִים

וְכִפַּת כָּל־נְדִיבֵי עָם יְיַבֵּשׁ / וּבַדֵי הַנְּבָלִים
רַעֲנַנִים

הֲכִי כִּי־לַפְּתָאִים הַנְּבָלִים / תְּהִלַּת חֵן וְהוֹד לֹא
לַנְּבוֹנִים

לְזֹאת אַךְ יֶחֶרַד לִבִּי וְיִתַּר / אֲמָרֵר בַּבְּכִי כִּבְנוֹת יְעֵנִים

וְעָלָיו אֶעֱשֶׂה מִסְפֵּד כְּתַנִּים / וְעָלָיו נֽוֹזְלֵי עֵינַי כְּעֵינִים

וְעַל־בֹּגְדִים אֲשֶׁר בָּגְדוּ כְנַחַל / וְעָפוּ אֶל־אֲרֻבָּתָם כְּיוֹנִים

וְהַמַּעְיָן אֲשֶׁר קֶדֶם אֲדַמֶּה / לְרַוֹּתוֹ כְּמֵימֵי צִמְּאוֹנִים

בְּחַנְתִּיהוּ וְהִנֵּה־הוּא כְאַכְזָב / וּמֵימָיו נֶהְפְּכוּ־לִי רֹאשׁ פְּתָנִים

וְהַכֶּרֶם אֲשֶׁר שֹׂרֵק חֲשַׁבְתִּיו / וְדִמִּיתִי נְטָעָיו נַעֲמָנִים

בְּחָקְרִי אֶת־יְנִיקוֹתָיו מְצָאתִים / זְמֹרַת זָר וְסוּרֵי הַגְּפָנִים

אֲקַוֶּה לַעֲשׂוֹת עֵנָב וְעָשָׂה / בְּאֻשִׁים וַחֲרֻלִּים קִמְּשֹׁנִים

מְכָרַנִי וְהֵפֵר אֶת־בְּרִיתִי / וְאִהַבְתִּיו וְלֹא־שָׁמַר אֱמוּנִים

וְהַנֶּחְשָׁב בְּתוֹךְ אַנְשֵׁי שְׁלוֹמִי / אֲשֶׁר בָּטַח לְבָבִי בוֹ לְפָנִים

וְהָיִיתִי כְמוֹ שִׂכִּים בְּעֵינָיו / וְאֶל־צִדָּיו וּבִלְבָבוֹ צְנִנִים.
