


	
	שְׁנֵי יָמִים
	








שְׁנֵי יָמִים / מְסֻיָּמִים / נוֹעָדוּ לַעֲמוּסִים

זְכִירָתָם / שְׁמִירָתָם / הֻזְהָרוּ בְךָ חוֹסִים

חֹק שַׁבָּת / לְבַל יָשְׁבַּת / וְיוֹם זֵכֶר כָל מַעֲשִׂים

וְהַיָּמִים הָאֵלֶּה / נִזְכָּרִים וְנַעֲשִׂים





לַמִּשְׁפָּט, / הֱיוֹת נִשְׁפָּט / בּוֹ כָל בָּשָׂר, שַׂמְתּוֹ,

צוּר עוֹלָם, / לְכָל נֶעְלָם, / וְצוּר לְהוֹכִיחַ יְסַדְתּוֹ

עֲלֵי כָל אִישׁ / אֲשֶׁר יַבְאִישׁ / פְּרִי מַעֲלָלָיו לְתִתּוֹ

אִישׁ אִישׁ מִמְּלַאכְתּוֹ / אֲשֶׁר הֵמָּה עוֹשִׂים





מֵרֵאשִׁית / בּוֹ הֵשִׁית / לְחוֹטְאִים צְרִי וַאֲרוּכָה

וְדִין נִקְשָׁב / לְאִישׁ לֹא שָׁב / וְלֹא חָפֵץ בִּבְרָכָה

לְהוֹלְכֵי תֹם / יֵשׁ בּוֹ מְתֹם / בְּרִית שְׁמוּרָה וַעֲרוּכָה

לְעוֹשֵׂי הַמְּלָאכָה / כִּי בֶאֱמוּנָה הֵם עוֹשִׂים





הֵן הַיּוֹם / מְאֹד אָיֹם / וְנוֹרָא כְּמוֹרָאֶךָ

בְּעֵת עָבְרָם / בְּמִסְפָּרָם / כִּבְנֵי מֵרוֹן צְבָאֶיךָ

כְּמַעֲשֵׂה אֱנוֹשׁ / אַל תַּעֲנֹשׁ / עַל שְׂכָרָם בְּבוֹאֶךָ

כִּי בְנִיבְךָ / וְרַב טוּבְךָ / נִשְׁעָנִים נְכֵאֶיךָ

צָפַנְתָּ לִירֵאֶיךָ / פָּעַלְתָּ לַחוֹסִים.
