


	
	צוּרִי בְּשׁוּרְךָ אוֹחִיל
	



מאורה




צוּרִי, בְּשׁוּרְךָ / אוֹחִיל עֲצוֹר / עַל צַר וְצוֹר,



אֶשְבַּע בְּאוֹרְךָ; / דַּי לִי שְׁכוֹר / כּוֹס מַמְרוֹר!



אֵשֵׁב מְיַחֵל / עֵת יַעֲלֶה /אוֹר יִשְׁעֲךָ,



מַרְעִיד וְזוֹחֵל / עַל אָהֳלֵי / מַטַּעֲךָ,



נֶאְלָם כְּרָחֵל – /וָאֶחֱלֶה / מִנָּסְעֲךָ.



עַד מָה חֲזוֹנְךָ / לַחְתֹּם וְצוֹר? / נֶעְלָם חֲקֹר!



יָפִיק רְצוֹנְךָ / עָצוּר בְּבוֹר, / יָקוּט בְּדוֹר.



בֵּינוֹת צְלָעַי / יֵקַד יְקוֹד / אֶל גּוֹאֲלִי,



עַל רֹב פְּשָׁעַי / הִרְחִיק נְדוֹד / מֵאָהֳלִי.



אַחַי וְרֵעַי, / שַׁאְלוּ לְדוֹד / עַל מַחֲלִי,



אִם יֵשׁ אֲרוּכָה / לַחְבֹשׁ מְזוֹר, / וּגְהוֹת מְזוֹר־



תִּקְוָה מְשׁוּכָה / אֶל קֵץ וְתוֹר / לִקְרֹא דְרוֹר.



רוּם לֵב מְעַנַּי / הַשְׁפֵּל וְשַׁח / קוֹלָם כְּאוֹב.



רֻבֵּי עֲוֹנַי / מִלֵּב זְנַח, / אַל תַּחֲשֹׁב.



אוֹרְךָ לְעֵינַי / עַתָּה שְׁלַח, / מָתוֹק וְטוֹב.



כַּיּוֹם לְבִנְךָ / חֶסֶד גְּמוֹר, / הוֹרָם זְכוֹר,



עָלָיו יְמִינְךָ / שִׂים לַעֲזֹר, / זֶה הַבְּכֹר!



הַקְשֵׁב בְּזַעְקִי / אֶת עֶזְרְךָ, / וּצְלַח רְכַב,



צוּרִי וְחִשְׁקִי, / פֶּן צָרְךָ / נֵרִי יְכַב,



וּכְאָז בְּחֵיקִי / תּוֹךְ הֲדָרָךְ, / דּוֹד, שׁוּב שְׁכַב.



עַל עַם רְחֵקְךָ / הָרֵק, סְגוֹר! / קִנְאָה חֲגוֹר



אֶל עַם דְבֵקְךָ, / יָגוּר בְּבוֹר / בֵּין דֹּב וְגוֹר.



מָאוֹר צְפַנְתּוֹ / אֶל יוֹדְעֵי / שִׁמְךָ – חֲלֹק



אֶל בֵּן בְּחַנְתּוֹ / בֵּין פּוֹרְעֵי / מוּסַר וְחֹק,



שׁוּב וַחֲנַנְתּוֹ, / קוֹלוֹ שְׁעֵה, / עֻלּוֹ פְּרֹק.



אָגִיל בְּצֵאתְךָ / לָקֹב, וְאֹר / צָרִי, וְצוֹר



כַּפָּךְ לְבֵיתְךָ / אוֹסִיף עֲבוֹר / לֵאוֹר בְּאוֹר.
