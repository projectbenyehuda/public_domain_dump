


	
	אֶל אֵל אֶקְרָא בְּמֶרְחָב
	



(קינה לט׳ באב)




אֶל אֵל אֶקְרָא / בְּמֶרְחָב וְצָרָה



בְּרָצוֹן וְעֶבְרָה: / צַדִּיק הוּא יְיָ!



בְּמַר לֵב אֲשַׁנֵּן / וְעָלַי אֲקוֹנֵן



וְלֹא אֶתְאוֹנֵן, / כִּי יָשָׁר דְּבַר יְיָ.



גָּבַר הַצָּר / נְבוּכַדְנֶצַּר,



כֹּחַ עָצַר / בְּיוֹם אַף יְיָ.



דָּרַךְ קַשְׁתּוֹ / וְהִצִּיב רִשְׁתּוֹ,



וּבְמַלְכֻּדְתּוֹ / נְתָנַנִי יְיָ.



הֵנִיד רֹאשׁוֹ / בְּעוֹנְנוֹ וְנַחְשׁוֹ,



וְאָמַר בְּפָרְשׁוֹ: / מַה דִּבֶּר יְיָ?



וּבֵיתְךָ חַלְלֵי / חֶרֶב מִלֵּא –



הַעַל אֵלֶּה / תִּתְאַפַּק, יְיָ?



זָבַח בָּנַי / עַל חֵיק זְקֵנַי,



וּדְמֵי כֹּהֲנַי / עַל אִשֵּׁי יְיָ.



חַיִל גִּבֵּר / וְשָׂח: ׳מַה תְּשַׂבֵּר?׳ –



זֶה הַמְדַבֵּר / סָרָה עַל יְיָ;



׳טְרוּפִים נִכְאוּ, / קוֹל אֵיךְ תִּשְּׂאוּ?



צַעֲקוּ וְקִרְאוּ – / לֹא יִשְׁמַע יְיָ!



יָדִי רָמָה / לַעֲשׂוֹת נְקָמָה



בְּהֵיכָל פְּנִימָה – / מֶה עָשָׂה יְיָ?׳



כֹּל שָׁמַעְתִּי / טֶרֶם נַעְתִּי –



וְלֹא יָדַעְתִּי / כִּי מַשְׁחִית יְיָ.



לַעַג שׂוֹטְנִי / שָׁמְעָה אָזְנִי,



וּבְלִבִּי אֲנִי / אֶקְרָא יְיָ.



מֵעֲנוֹת אִלֵּם / אָזְנְךָ תַּעֲלֵם,



וַיִּכָּלֵם / כָּל מְבַקֵּשׁ יְיָ.



נְבִיאַי נִלְאוּ / וְלֹא הִתְנַבְּאוּ,



כִּי לֹא מָצְאוּ / חָזוֹן מֵייָ.



סְכָלִים נוֹאֲלוּ / לְחוֹזִים נִמְשְׁלוּ,



שֶׁקֶר נָחֲלוּ – / כֹּה אָמַר יְיָ.



עָלַי שָׂחֲקוּ / וְדָתִי רִחֲקוּ,



וְחֹק שַׁוְא חוֹקְקוּ, / לֹא דִבֶּר יְיָ.



פָּעֲרוּ פִיהֶם / כַּאֲרִי נוֹהֵם,



סְלֹחַ לָהֶם / לֹא יֹאבֶה יְיָ.



צְפִירָה הֻלֲּלָה / עַתָּה חֻלֲּלָה,



יַעַן מָעֲלָה / מַעַל בַּייָ.



קָמוּ הוֹרִים / עִם אֵל מוֹרִים,



עָבְדוּ זָרִים – / וַיִּתְּשֵׁם יְיָ.



רְאוּ נָא, זַרְעָם, / אוֹרְכֶם הוּעַם,



כִּי אַתֶּם עַם / אֲשֶׁר זָעַם יְיָ.



שְׁבוּ הַשְׁפִּילוּ, / לֹא תַּשְׂכִּילוּ –



וְאֵיךְ תּוֹחִילוּ / אֶל יְשׁוּעַת יְיָ?



תַּם קֵץ נֶחְשַׁב / וְעַם אֵל נוֹשַׁב



בְּגָלוּת, וְלֹא שָׁב / חֲרוֹן אַף יְיָ.



אֵיךְ הוֹפַעְתָּ – / וְלֹא הוֹשַׁעְתָּ,



כִּי שָׁמַעְתָּ / חֶרְפָּתָם, יְיָ?



בִּגְדֵי נְקָמָה / לְבַשׁ בְּאַף וְחֵמָה,



עוּרָה, לָמָּה / תִּישַׁן יְיָ?



רוֹדְפֵי חִנָּם – / גָמְלֵם כְּאוֹנָם,



וְתִרְאֶה עֵינָם / אֶת מוּסַר יְיָ.



מַעְיַן פְּדוּתְךָ / רָוֵּה אֲמָתְךָ –



לִישׁוּעָתְךָ / קִוִּיתִי יְיָ!
