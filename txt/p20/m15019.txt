


	
	יְמֵי אָדָם צָבָא
	




יְמֵי אָדָם צָבָא / וְהוּא בָעוֹלָם הֵלֶךְ,

כִּי בַהֶבֶל בָּא / וּבַחשֶׁךְ יֵלֵךְ.

שׁוֹגֶה בְּאַהֲבַת תֵּבֵל / סָכַל, כִּי הוּא הֶבֶל;

וְנִפְתָּה בְּמֶתֶק שִׂיחָהּ, / הִטַּתּוּ בְּרֹב לִקְחָהּ.

לְרוֹשֵׁשׁ אוֹתוֹ – תַּעֲשִׁיר, / לְקוֹנֵן עָלָיו – תָּשִׁיר,

וְצוּף פִּיהָ וְחֶלְבּוֹ – / מְרוֹרַת פְּתָנִים בְּקִרְבּוֹ.

שְׂפָתֶיהָ בְּצוּף לֶקַח / יְפַתּוּ כְּיֵין הָרֶקַח –

וּמִגֶּפֶן סְדוֹם גַּפְנָם, / חֲמַת תַּנִּינִים יֵינָם.

עוֹלָם זְנַח וּשְׁלוֹמָיו / וְאַל תִּתְאָו לְמַטְעַמָּיו –

וְיִהְיֶה בְּעֵינֶיךָ נִבְזֶה / מְעַט הַדְּבַשׁ הַזֶּה.

אֲנִי אִישׁ בָּעוֹלָם צוֹעֶה / וּבְעִי סְעִפָּיו תּוֹעֶה,

נָגוּעַ וּמְנֻסֶּה, / כִּי רַע עָלַי הַמַּעֲשֶׂה.

בְּיֵין נֹעַר שָׁכָרְתִּי / וְאַחֲרֵי עֵינַי תָּרְתִּי,

וְעֵקֶב הִסְכַּלְתִּי עֲשֹה – / גָּדוֹל עֲוֹנִי מִנְּשׂוֹא.

רָמוֹת הָלְכוּ עֵינָי / וּלְזֹאת נָפְלוּ פָנָי,

וְכָל מַכְמוֹרֵי כַחֲשִׁי / עֵינִי עוֹלְלָה לְנַפְשִׁי.

מִתְּנוּמַת יְמֵי עֶדְנָה / הֱקִיצוּנִי יְמֵי זִקְנָה –

וְהִנֵּה כָּל פְּרִי נֹחַ / הֶבֶל וּרְעוּת רוּחַ.

יַלְדוּת אֱנוֹשׁ וְשַׁחֲרוּתוֹ / כַּחֲלוֹם רָאָה אוֹתוֹ,

וַחֲטָאָיו נִצְמָתִים / וְאַחֲרָיו אֶל הַמֵּתִים.

וְיוֹם יִדַּל הַכֹּחַ / אֲזַי יַחֲלֹף הָרוּחַ,

וְהַגּוּף בִּשְׁאוֹל מִרְמָס / יִהְיֶה, כְּפֶגֶר מוּבָס.

מִשְׁכָּנוֹ יְקִיאֶנּוּ / וְכַנּוֹ יַבְהִילֶנּוּ

לָצֵאת מֵחֲדַר בֵּיתוֹ, / מֻשְׁכָּב עַל מִטָּתוֹ.

וּמוֹדָעָיו יְרִיצוּהוּ / וּמִתֵּבֵל יְנִידוּהוּ

וּבַבּוֹר יֻשְׁלַךְ קִרְבּוֹ, / כִּכְלִי אֵין חֵפֶץ בּוֹ.

וְתִיקַר אָז הַנְּשָׁמָה, / אֲשֶׁר מִתַּאֲוָה עֲרֻמָּה,

וְתַעֲלֶה, אִם לֹא נִטְמֵאת / הַנֶּפֶשׁ הַחֹטֵאת.

הֲשִׁיבוּהָ אֶל מְעוֹנָהּ / כַּאֲשֶׁר הָאֵל נְתָנָהּ –

וְאָז אוֹרָהּ יַבְקִיעַ / כְּזֹהַר הָרָקִיעַ.

זְנַח מָשׂוֹשׂ וַהֲנָחָה, / כִּי לֵב כְּסִיל בְּבֵית שִׂמְחָה;

וּבְכֵה וַחֲגֹר חֶבֶל, / לֵב חֲכָמִים בְּבֵית אֵבֶל.

וְהָבָה נִשְׁפֹּךְ שִׂיחָה / לִפְנֵי אֵל וְנִוָּכְחָה;

לְכוּ נָא, זֶרַע אֱמוּנָי, / וְנָשׁוּבָה אֶל יְיָ!
