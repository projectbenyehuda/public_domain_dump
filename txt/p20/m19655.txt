


	
	… בְּרוּאִים, מֵאַיִן נִבְרֵאתֶם
	



מי כמוך






[…………..]



[…………..]



בְּרוּאִים, מֵאַיִן נִבְרֵאתֶם, / כִּי קַדְמוֹנִים לֹא הֱיִי[תֶם],



הִתְבּוֹנְנוּ אֵיךְ יְצָאתֶם, / הָבוּ לַייָ כָּבוֹד!



רֵאשִׁית פְּעֻלַּת הַבּוֹרֵא / הֲגוּ, וע[….]



וְזֶה וְזֶה עָלָיו יוֹרֶה: / כֻּלּוֹ אֹמֵר כָּבוֹד!



הַכֹּל – בְּלִי יֵשׁ הוֹלִידָם, / בְּלִי זְמַן וּמָקוֹם יְסָדָם,



וַיֹּאמֶר: ׳נַעֲשֶׂה אָדָם, / וְהָיָה לְכִסֵּא כָבוֹד.׳



מִצַּלְעוֹתָיו הוֹצִיא שִׁדָּה, / יְפַת תֹּאַר בְּרֻדָּה,



כְּבַת מֶלֶךְ כְּבוּדָּה – / אֵשֶׁת חֵן תִּתְמֹךְ כָּבוֹד.



בַּהֲרֵעוֹ הַנִּבְרָא בִּרְבוֹתוֹ, / הָאֵל בְּמַבּוּל הִשְׁחִיתוֹ,



וְנֹחַ נִמְלַט בְּצִדְקָתוֹ / וְהָיְתָה מְנֻחָתוֹ כָּבוֹד.



נִינוֹ, בִּבְנוֹתוֹ שׁוּרוֹ – / הֱפִיצוֹ מִשָּׁם וּפִזְּרוֹ,



הֲלֹא יָדַע, כִּי בַּעֲבוּרוֹ / עָשָׂה אֶת כָּל הַכָּבֹד.



עָזַב אֶרֶץ מְגוּרָיו / אָב פִּלֵּס אֲשֻׁרָיו,



וּבֵיתוֹ צִוָּה אַחֲרָיו: / ׳תְּנוּ לַייָ אֱלֹהֵיכֶם כָּבוֹד!׳



זָקֵן זָרַח מִתְעָרֶה, / וַיֹּאמֶר לוֹ הַבּוֹרֵא:



׳בְּיִצְחָק לְךָ שֵׁם יִקָּרֵא, / קַרְנוֹ תָּרוּם בְּכָבוֹד.



רִ[בָּה] זַרְעָם כְּחוֹל הַיָּם / וּבְרִיתוֹ לָהֶם קִיָּם,



[…] שים אַחַר תֵּת פִּדְיוֹם, / כֻּלָּם שָׁכְבוּ בְּכָבוֹד.׳



ה[…] יָרְדוּ שִׁבְעִים / וְשָׁם יכב[…]ם זֶרַע מְרֵעִים,



[…יַלְ]דֵי שַׁעֲשׁוּעִים, / יַעְלְזוּ חֲסִידִים בְּכָבוֹד.



ב[…] בְּיָדוֹ הֲמוֹנִי, / רָבוּ בְנֵי הַיִּצְהָר לְתַנִּינִי:



׳[וַיֹּאמֶר] יְיָ: שַׁלַּח עַמִּי, / הוּא מֶלֶךְ הַכָּבוֹד!׳



[רָחַ]ב פִּיו, תַּנִּין, עַל חֲסִידָי: / ׳אֲנִי עֲשִׂיתִנִי וְהַיְאוֹר
מַעֲשֵׂה יָדָי!



לֹא יָדַעְתִי אֶת יְיָ, וּבִלְעָדַי / מִי זֶה מֶלֶךְ הַכָּבוֹד?׳



מֵלִיץ עָנָהוּ: ׳שָׁוְא הִתְהוֹלַלְתָּ, / עוֹד יְבַכּוּךָ חַרְטֻמִּים
הֶאֱצַלְתָּ:



הֵילֵל בֶּן שַׁחַר, אֵיךְ נָפַלְתָּ? / אֶל מִי דָמִיתָ כָּכָה
בְּכָבוֹד!׳



אָמַר הָאוֹיֵב: ׳מִי הוּא, וּמִי / יוּכַל לְהַחֲרִימֵנִי וְעַמִּי? –



וְלִי הָרֶכֶב וְהַפָּרָשִׁים, וְעִמִּי / עֹשֶׁר וּנְכָסִים וְכָבוֹד!׳



׳יֵהָרְגוּ בָנֶיךָ בְּחֵימָה, / וַאֲנִי כִּבַּדְתִּיךָ – וְלֹא מֵאֵימָה,



כִּצְרוֹר אֶבֶן בְּמַרְגֵּמָה, / כֵּן נוֹתֵן לִכְסִיל כָּבוֹד.׳



רָחַשׁ לְעָנָו: ׳אֱסֹף יָדֶךָ! / לֵךְ מֵעָלַי, לָמָּה אֲאַבְּדֶךָ?



אָמַרְתִּי, כַבֵּד אֲכַבֶּדְךָ – / וְהִנֵּה מְנָעֲךָ יְיָ מִכָּבוֹד!׳



חֲצוֹת לַיְלָה נֶהֶרְגוּ בְכוֹרָם – / וְעַם יָצְאוּ מִמַּאֲסָרָם,



אוֹמְרִים לְפַרְעֹה: ׳לָמָּה לֹא שָׁמַעְתָּ לְאֵל רָם? / שָׂבַעְתָּ
קָלוֹן מִכָּבוֹד!׳



זֵדִים רָדְפוּ אַחֲרֵיהֶם / [….] רֹכְבֵיהֶם



וּמַלְאַךְ יְיָ הֶחֱשִׁיךְ לִפְנֵיהֶם – / [וְנֶגֶד] זְקֵנָיו כָּבוֹד.



קָפְאוּ תְהֹמֹת בְּלֶב יָם, / עֲדֵי עָבְרוּ עַם אֵל בְּעֶדְיָם,



וּבִרְ[אוֹ]תָם הַמֵּתִים עַך שְׂפַת הַיָּם – / יָשִׂימוּ לַייָ
כָּבוֹד.
