


	
	מָתַי יְבַשֵּׂר בְּבִיאַת נְגִידִי
	



(הבדלה)


מָתַי יְבַשֵּׂר / בְּבִיאַת נְגִידִי



אֵלִיָּהוּ / הַגִּלְעָדִי?



אֲיַחֵל לְיִשְׁעִי / בּוֹא יוֹם בְּשׂוֹרָה,



וּבְבוֹא רֹעִי / שִׁיר אָעִירָה,



וּבְהַפְגִּיעִי / בוֹ, אָשִׁירָה



נָא לִידִידִי / שִׁירַת דּוֹדִי.



בְּאַרְצוֹת עָרַי / נַפְשִׁי יָגְעָה,



יָרְשׁוּ עָרַי / זֵדֵי רִשְׁעָה,



שָׁמְעוּ צָרַי / אֵין לִי תְשׁוּעָה –



וְאַתָּה, יְיָ, / מָגֵן בַּעֲדִי.



רוֹדִים וְצָרִים / תּוֹסִיף לְיַסֵּר



וִיגוֹן בְּחִירִים / בְּיִשְׁעָךְ הָסֵר,



וְעַל הֶהָרִים / רַגְלֵי מְבַשֵּׂר



הָחִישׁ, לְבַשֵּׂר / כִּי בָא מוֹעֲדִי.



הַשָּׂם רְכוּבוֹ / עָבִים וְחֵילוֹ,



הַבּוֹטְחִים בּוֹ / יָשִׁיב לְחֶבְלוֹ,



וּפִתְאֹם יָבוֹא / אֶל הֵיכָלוֹ –



יִשְׂמַח לִבִּי / וְיָגֵל כְּבוֹדִי.



מְדֻכָּה, מְעֻנָּה / וְלִבָּהּ רוֹעֵד –



הָסֵר יְגוֹנָהּ, / שְׁלַח לָהּ סוֹעֵד,



כִּי עֵת לְחֶנְנָהּ, / כִּי בָא מוֹעֵד –



אָז מַקֵּל עֹז / אֶקַּח בְּיָדִי.
