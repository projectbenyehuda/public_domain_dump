


	
	אֵלִיָּה לִשְׁבוּיָה בּוֹא יָבוֹא
	




אֵלִיָּה / לִשְׁבוּיָה / בּוֹא יָבוֹא בִּבְשׂוֹרָה

לִפְנֵי בּוֹא / יוֹם יְיָ / הַגָּדוֹל וְהַנּוֹרָא.



בְּתַחֲנוּנִים / כְּאֶבְיוֹנִים / נְקַדֵּם אֶת פָּנֶיךָ,

זְכֹר הַבְּרִית / לִשְׁאֵרִית / צֶאֱצָאֵי אֱמוּנֶיךָ

וְאֵיךְ אֲחֵיהֶם / וְעַבְדֵיהֶם / יַעֲבִידוּם לְפָנֶיךָ —



וְהֵם חֶבְלָךְ, / עֲבָדִים לָךְ, / כַּכָּתוּב בַּתּוֹרָה!



רְאֵה עַתָּה — / הֲלֹא אַתָּה / אֵל אֶחָד וְלֹא תִשְׁנֶה!

אֲנִי בָזוּז / כְּשֶׂה גָזוּז / וְאֶקְרָאֲךָ וְלֹא תַעֲנֶה.

שְׁלַח מַרְפֵּא / לְלֵב נִכְפֶּה / וְתָבִיא מַגִּיד מִשְׁנֶה

כִּדְבָרָךְ, / לִבְכוֹרָךְ / הֲלֹא מִשְׁפַּט הַבְּכוֹרָה!



'מַה תּוֹחִיל, / עַם הוֹחִיל — / וּמוֹעֵד מוֹעֲדִים יִשְׁלָם!

וַאְשִׁיבֵךְ / וְאוֹשִׁיבֵךְ / בְּהֵיכַל חֵן וְהוֹד אוּלָם,

וְשָׁם גּוֹאֵל / לְיִשְׂרָאֵל / יִהְיֶה לָךְ לְאוֹר עוֹלָם

וְלַיְלָה כַּ-/ יוֹם יָאִיר, / כַּחֲשֵׁיכָה כָּאוֹרָה.'
