


	
	אנא שא נא לפשע כנה (תחנון לצום גדליהו)
	




אָנָּא שָׂא נָא / לְפֶשַׁע כַּנָּה,



מְעֻנָּה, נְתוּנָה / לְמָשָׁל וְלִשְׁנִינָה.



פְּנֵה נָא לְכוֹנְנָהּ, / הֱיֵה לָהּ מַשְׁעֵנָה



מֵרֵאשִׁית שָׁנָה / וְעַד אַחֲרִית שָׁנָה.



בְּשִׁמְךָ, פּוֹדֶה, / קֹרָא עַם עוֹלָם,



וּלְךָ מוֹדֶה / כְּנוֹקֵשׁ וַיִּשְׁלָם.



הִנּוֹ בְּכַף צוֹדֶה / מֻכֶּה וְנִכְלָם,



כְּרָחֵל בִּידֵי / גוֹזֲזֶיהָ נֶאֱלָם.



בְּשֵׁבֶט רוֹדֶה / מְרֻדָּף וְנֶהֱלָם,



גּוֹלֶה, מְנֻדֶּה / וְנִגְרָשׁ מֵאוּלָם –



וְיוֹצֵא הַשָּׂדֶה / שָׁנָה שָׁנָה.



רוֹדְפוֹ יְכַלֶּנּוּ / וַעֲוֹנוֹ הִרְצָה,



כָּל שֵׁן יֹאכְלֶנּוּ / מִנֶּפֶשׁ עַד נוֹצָה;



כָּל לָשׁוֹן יְקַלֲּלֶנּוּ / וְהָיָה לְשִׁמְצָה,



כָּל גֵּא יַשְׁפִּילֶנּוּ – / וּפִיו לֹא פָצָה.



בָּאֵשׁ יַפִּילֶנּוּ – / וּמֵהָאֵשׁ יָצָא,



וְהָאֵשׁ תֹּאכְלֶנּוּ / שָׁנָה שָׁנָה.



הָקֵם מַפַּלְתּוֹ, / יוֹשֵׁב בַּמְּרוֹמִים,



וְהָרֵם שִׁפְלוּתוֹ / וּפָקְדֵהוּ בְרַחֲמִים,



וְהַעֲבֵר חַטָּאתוֹ / וְאַל יָמוּת בְּאֵימִים.



זְמַן רָצוֹן חֲקַקְתּוֹ / בַּעֲשֶׂרֶת יָמִים –



לְכַפֵּר קְבַעְתּוֹ, / לִמְחוֹת אֲשָׁמִים



וַעֲוֹן אִישׁ וּבֵיתוֹ / דְּבַר שָׁנָה בְּשָׁנָה.



מָעוֹז רֹאשׁוֹ – / לְפָקְדוֹ הִזָּכֵר,



וְהָסֵר כַּחֲשׁוֹ / וְעוֹכְרוֹ יֵעָכֵר;



וּפְדֵה נַפְשׁוֹ / מֵיַּד בְּנֵי נָכֶר,



וּפְאֵר מִקְדָּשׁוֹ / וּמַעְיָנוֹ הַסְכֵּר,



אֲשֶׁר רֻקַּת רֹאשׁוֹ / כְּחוּט, וְלֹא יִסָּכֵר,



וְכָל עֵץ לְחוֹרְשׁוֹ / לָחֳדָשָׁיו יְבַכֵּר –



חֹדֶשׁ בְּחָדְשׁוֹ / לְחָדְשֵׁי הַשָּׁנָה.
