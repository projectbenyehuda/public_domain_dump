


	
	אֶפְרֹשׂ שְׁעָלַי
	




אֶפְרֹשׂ שְׁעָלַי / וְאַזִּיל דְּמָעָי

עַל רֹב מְעָלַי / וְגֹדֶל פְּשָעָי.

יַשֵׁר פְּעָלַי / וְרַפֵּא נְגָעָי!

צוּרִי, חָנֵּנִי! / עַד תְּדִינֵנִי –

בְּקָרְאִי עֲנֵנִי, / אֱלֹהֵי צִדְקִי.



בְּרֶנֶן וְחִילוֹת / אֶשְׁאַל מְחִילוֹת,

וּבִנְעִים תְּהִלּוֹת / בִּקְהַל סְגֻלּוֹת,

בְּשִׁירוֹת מְעֻלּוֹת / וּמֵיטַב פְּעֻלּוֹת;

וַאֲרוֹמִמְךָ / וְאוֹדֶה שִׁמְךָ,

אַף אֶרְחָמְךָ,/ יְיָ חִזְקִי.



רוֹכֵב עֲרָבוֹת / וְחוֹפֵשׂ קְרָבוֹת

וְרוֹצֶה תְשׁוּבוֹת / כְּעוֹלוֹת חֲשׁוּבוֹת;

סוֹמֵךְ לְבָבוֹת, / שׁוֹבֵב רְבָבוֹת

לְקִרְיַת מְשׂוֹשִׂי – / צוּרִי וּמַחְסִי,

חֶבְלִי וְכוֹסִי / וּמְנָת חֶלְקִי.



הֵן לָךְ אֲקַוֶּה, / מְקוֹר עֹז וּמִקְוֶה

וְעֻזְּךָ אֲחַוֶּה / וּלְךָ אֶשְׁתַּחֲוֶה;

וְיִשְׁעָךְ תְּצַוֶּה / וְלִי, יָהּ, יִלָּוֶה

לְהַקִּיפֵנִי / וּלְחוֹפְפֵנִי,

20 לְהַטְרִיפֵנִי / לֶחֶם חֻקִּי.



מָרוֹם, מַלְכִּי, / בְּךָ מִשְׁעָנִי!

שְׁעֵה רֹן עֶרְכִּי / וּבַשֵּׂר עָנִי:

'אַל תִּירָא, כִּי / אִתְּךּ אָנִי.

אִמָּצְתִּיךָ, / עֲזַרְתִּיךָ,

אַף תְּמַכְתִּיךָ / בִּימִין צִדְקִי!'
