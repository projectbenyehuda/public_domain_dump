


	
	אֶפְתַּח שְׂפָתַי בִּנְגִינוֹתַי
	




אֶפְתַּח שְׂפָתַי / בִּנְגִינוֹתַי, / אֵלִי, תּוֹךְ בֵּיתְךָ,

כְּי מִבְּשָׂרִי, / אֱלֹהֵי צוּרִי, / אֶחֱזֶה גְּבוּרָתְךָ.

אַתְּ אֵל נַעֲלֶה, / עוֹשֵׂה פֶלֶא / בְּגָדְלְךָ וְגַאֲוָתְךָ.

כִּי מָה הָאָדָם? / חָטָאתִי – מָה אֶפְעַל לָךְ, נֹצֵר הָאָדָם?



בְּבוֹאִי לְקִבְרִי, / אֶמְצָא אוֹרִי / חשֶׁךְ וַאֲפֵלָה.

לִי הַפֶּשַׁע, / כִּי מֵרֶשַע / לָאֵל חָלִילָה.

כַּפֵּר פִּשְׁעִי / וְהָחֵשׁ יִשְׁעִי, / אֵל נוֹרָא עֲלִילָה

עַל בְּנֵי אָדָם! / חָטָאתִי – מָה אֶפְעַל לָךְ, נֹצֵר הָאָדָם?



רְאֵה אִיש נָבוּב / נִמְשָׁל כִּזְבוּב / רֵיחַ מוֹר יַבְאִישׁ

וְיִשְׁכַּח סוֹפוֹ, / יוֹם תּוֹךְ גּוּפוֹ / הֲמוֹן רִמָּה יַפְרִישׁ.

אַרְבֶּה בֶכִי, / כִּי אָנֹכִי / תּוֹלַעַת וְלֹא אִישׁ,

חֶרְפַּת אָדָם! / חָטָאתִי – מָה אֶפְעַל לָךְ, נֹצֵר הָאָדָם?



הָכְלְמוּ פָנַי / יוֹם בּוֹא עֲוֹנַי / הֲמוֹנִים הֲמוֹנִים,

עֵת יֵאָסְפוּ / פְּשָׁעִים נוֹסְפוּ / הַיּוֹם וּלְפָנִים.

אֵל תְּמִים דֵּעָה, / נָא הוֹשִׁיעָה, / כִּי פַסּוּ אֱמוּנִים

מִבְּנֵי אָדָם. / חָטָאתִי – מָה אֶפְעַל לָךְ, נֹצֵר הָאָדָם?



מִרְיִי הַעֲבֵר, / צֶדֶק דּוֹבֵר / וְהָעוֹלָם סוֹבֵל!

מָרוֹם, תִּרְאֶה / כִּי אִישׁ נִכְאֶה – / יָמָיו כְּצִיץ נוֹבֵל.

אֵיךְ יִתְגָּאֶה / אוֹ יִתְנָאֶה – / וְיֵדַע, כִּי הֶבֶל

הֵם בְּנֵי אָדָם? / חָטָאתִי – מָה אֶפְעַל לָךְ, נֹצֵר הָאָדָם?
