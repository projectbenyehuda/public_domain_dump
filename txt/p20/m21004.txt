


	
	[אֱלֹהֵי הָרוּחוֹת לְכָל בָּשָׂר… רְחָבִים וַאְרֻכִּים]
	



רשות לקדיש






אֱלֹהֵי הָרוּחוֹת / לְכָל בָּשָׂר אַתָּה



וְחָכְמָה בַּטּוּחוֹת / בְּחָכְמָתְךָ שַׁתָּה



וְגַם דַּבֵּר צָחוֹת / בְּיַד לָשׁוֹן תָּתָּה –



אֱמֶת כֹּל חוֹלַלְתָּ / וּבוֹ לֹא עָמַלְתָּ, / רַק בְּפִיךָ מִלָּלְתָּ!



בְּיָדְךָ עִתּוֹתַי / וּמַה זֶּה פִי יַעְנֶה



וְקָצְרוּ שִׂפְתוֹתָי? / וְכָל רֶגַע הִנֵּה



בְקִרְבִּי אוֹתוֹתַי, / וְאָן לִבִּי יִפְנֶה



וְאוֹתוֹ הִגְבַּלְתָּ / בְּדָת שֶׁהִנְחַלְתָּ, / לִי בְרָכָה אָצָלְתָּ!



רְחָבִים וַאְרֻכִּים / חֲסָדֶיךָ עִמִּי,



וְרַעְיוֹנַי רַכִּים / וְלֹא אָבִין גָּלְמִי,



וְעֵת נוּם וּבְהַשְׁכִּים / בְּךָ עָצַם עַצְמִי,



וְעָלַי חָמַלְתָּ / וְטוֹבוֹת גָּמַלְתָּ / לִי גְּמוּל לֹא שָׁאָלְתָּ!



הֲכִי טוֹב עָשִׂיתָ / וְאָנֹכִי אֶרְשָׁע,



וְחוֹבִי כִסִּיתָ / וְאֶל שַׁוְעִי תֵּשַׁע,



וְאַחַר נִסִּיתָ / לְבָבִי בָּךְ נוֹשָׁע,



וְנַפְשִׁי הִצַּלְתָּ / וְחַיַּי גָּאַלְתָּ, / מֵעֲזֹר לֹא חָדַלְתָּ!



מְרוֹמִים שָׁכַנְתָּ / וְשָׁחִים מָלֵאתָ,



וְכָל יֵשׁ חוֹנַנְתָּ / וְלַעְשׂוֹת הִפְלֵאתָ,



וְכִסֵּא כוֹנַנְתָּ / וְעָלָיו נַעְלֵיתָ,



וְהַכֹּל פָּעַלְתָּ / וּבַכֹּל מָשַׁלְתָּ / ווַעֲלֵי כֹל גָּדָלְתָּ!
