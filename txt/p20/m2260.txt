


	
	על הים
	



[א]


אֶפְרֹשׁ כַּף / אַף אִכַּף / אֶל אֵלִי,



הוּא חֵילִי / מִגְדָּלִי / בַּצַּר לִי,



אֶקֹּד לוֹ / עַל גָּדְלוֹ / שֶׁהִפְלִיא.



בִּדְבַר תֹּךְ / מָטָה תּוֹךְ / יָם רַגְלִי



וָאֹמַר / “רָזִי לִי, / רָזִי לִי!”



וַיִּשְׁמַע / מֵהֵיכָלוֹ / קוֹלִי.



בֵּין גַּלִּים עָבַרְתִּי / אָמַרְתִּי: נִגְזַרְתִּי –



אָמַר צוּר עוֹלָמִים / “אַשְׁבִּיחַ שְׁאוֹן יַמִּים!”


  


בְּיָם בּוֹגֵד / מִנֶּגֶד / הִשְׁלַכְתִּי



אֶת נַפְשִׁי, / וּלְמוֹקְשִׁי / דָרַשְׁתִּי,



וּבְשׁוּרִי / יוֹם נָכְרִי / נִחַמְתִּי,



וְיָם בִּזְמָן / זֶה נֶאֱמָן / חָשַׁבְתִּי.



וּבִיגוֹנַי / עַל פָּנַי / נִרְדַּמְתִּי



וּבַסְּפִינָה / כַּמְּדִינָה / עָבַרְתִּי,



עָזַבְתִּי / אֶת בֵּיתִי / נָטַשְׁתִּי.



מַה לְּךָ יָם תַּבְהִיל מָךְ? / גֵּר אָנֹכִי עִמָּךְ!



לָמָּה אַפְּךָ חָרָה, / מַה זֶּה רוּחֲךָ סָרָה?


  


רוּחִי בִי / בִּלְבָבִי / הוֹמִיָּה,



וְיָם אָמַר / "אַל תֹּאמַר: / מֶה הָיָה?



הֲלִדְרֹךְ בִּי / וּבְקִרְבִּי / תּוּשִׁיָּה?



וּלְיוֹעֵץ / לוּחַ עֵץ / אֵין מִחְיָה



וְכָל מַלָּח / הֵן נִמְלַח / בִּשְׁאִיָּה,



וְרַב חוֹבֵל / רַב חֵבֶל / רַב בִּכְיָה,



וְהָאֳנִיָּה – / תַּאֲנִיָּה / וַאֲנִיָּה.



אַל נָא תִלּוֹן עָלַי, / כִּי אַתָּה בָא אֵלַי,



וַאֲנִי אֵיךְ אָנוּחַ – / וַתָּבוֹא בִי רוּחַ!"


  


"הַחֲרֵשׁ, יָם, / כִּי קַיָּם / בִּטְחוֹנִי,



וּבְסַעַר / הֵן יִגְעַר / צוּר קוֹנִי.



וְאֵין שִׁבְתִּי / אוֹ לֶכְתִּי / בִּרְצוֹנִי



וּבִכְתָבוֹ / בְּךָ לָבוֹא / צִוַּנִי



וּבְרֵאשִׁית / קֵץ הֵשִׁית / אֶל אוֹנִי,



גַּם קִצִּים / נֶחֱרָצִים / מִי יָנִיא?



בַּהֲכִינוֹ / שָׁמַיִם / שָׁם אָנִי!



סוֹדוֹ אֵיך אֲבִינֶנּוּ / וְאֶסָּתֵר מִמֶּנּוּ?



בְּיָדוֹ יָם וְחָרָבָה – / וַאֲנִי אָנָה אֲנִי בָא?"


  


מֵשִׁיב לִי / יָם "בִּבְלִי / לֵב שַׂחְתָּ



וּבְאוֹנְךָ / וּזְדוֹנְךָ / הָלַכְתָּ –



וְאַתְּ עָצוּר / וְאַף נָצוּר / נוֹתַרְתָּ!



זְכֹר דָּגִים / תַּעֲנוּגִים / אָכַלְתָּ –



בְּלִי שֵׂכֶל / תֵּאָכֵל / גַּם אַתָּה!



אִישׁ חֶמְדָה, / קַח מִדָּה / מָדַדְתָּ



הֲלֹא יָדַעְ– / תָּ אִם לֹא / שָׁמַעְתָּ:



עֵין אֵל עַל מִפְעָלוֹ / גְּמוּל יָדוֹ יָשִׁיב לוֹ,



בְּרֹאשְׁךָ דָּמְךָ יִהְיֶה – / מֵת אַתָּה וְלֹא תִחְיֶה!"


  


אֵל גֵּאֶה, / תֵּרָאֶה / גַּאֲוָתְךָ,



פְּדֵה וּרְאֵה / הַנִּכְאֶה / בַחֲמָתְךָ



הִנָּשֵׂא / עַל כִּסֵּא / חֶמְלָתְךָ!



הוֹפִיעָה / וְהוֹשִׁיעָה / בֶּן אֲמָתְךָ,



הַשּׁוֹאֵל / לָצֵאת אֶל / אַדְמָתְךָ



אֶעֱבָדְךָ / גַּם אוֹדְךָ / בַּעֲדָתְךָ,



לְדוֹר וָדוֹר / אוֹדִיעַ / אֱמוּנָתְךָ,



אַגִּיד זֹאת בְּכָל מַהֲלָךְ / וַאֲשַׁלֵּם תּוֹדוֹת לָךְ



אָשִׁירָה בִרְנָנַי / יְשׁוּעָתָה לַיָי!


  

[ב]


מִמַּיִם רַבִּים תַּמְשֵׁנִי, / רוּחֲךָ טוֹבָה, תַּנִּיחֵנִי!


  


אֵיךְ לֵב נָע וָנָד יִנָּפֵשׁ?



יִירָא מֵרוּחַ וָרֶפֶשׁ,



כִּי בָאוּ מַיִם עַד נֶפֶשׁ.



קוּמָה, אֵלִי, הוֹשִׁיעֵנִי / וּמְצוּלָה אַל תִּבְלָעֵנִי!


  


בְּיָדְךָ עִתּוֹתַי – וַאֲנִי מָה



אֵחָשֵׁב בְּיָם אוֹ בַאֲדָמָה,



וְעִמְּךָ הָיִיתִי כִבְהֵמָה?



בְּחֶלְדִי בַּעֲצָתְךָ תַנְחֵנִי / וְאַחַר כָּבוֹד תִּקָּחֵנִי.


  


רָפוּ יָדַי מֵאֵימֶיךָ



יוֹם סַעַר עָלַי יַמֶּךָ,



וָאֲחַזֵּק יָדִי בִשְׁמֶךָ:



אִם אַתָּה לֹא תַעֲמִידֵנִי, / חֶטְאִי אֶל בּוֹר יוֹרִידֵנִי.


  


מִדֵּי הֱיוֹתִי זֹאת אֶזְכֹּרָה



בְּבוֹא רוּחַ צָפוֹן בִּסְעָרָה



וְעָבַר עָלַי בַּיָּם צָרָה



יוֹם קָרָאתִי וַתַּעֲנֵנִי / לֵאמֹר “הִנֵּנִי הִנֵּנִי!”
