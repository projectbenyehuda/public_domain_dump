


	
	אֲסִיר תַּאֲוָה שְׁמַע מִצְוָה
	




אֲסִיר תַּאֲוָה, / שְׁמַע מִצְוָה / לַחָפְשִׁי תְּשַׁלֵּחֲךָ:



בְּדָת אֵל הֲגֵה / וְאַל תִּשְׁגֶּה / וְהִיא נִפְלָאוֹת תְּשִׂיחֲךָ.



וְאִם נַפְשְׁךָ / תְּבַקֶּשְׁךָ – / שְׁמָעֵנִי, אוֹכִיחֲךָ

עֲלֵי חָכְמוֹת / בְּנֵי אֲדָמוֹת, / אֲשֶׁר בִּקְשׁוּ לְהַדִּיחֲךָ:

אֵיךְ תִּדְרשׁ / סוֹד הָרֹאשׁ – / וְתִלְאֶה לְחַפֵּשׂ רוּחֲךָ?

וְאֵיך תֶּחֱזֶה / לָמָּה זֶה? / וְנָקֵל מִזֹּאת כֹּחֲךָ!

וְעַל אֱמוּנוֹת / הַתְּבוּנוֹת / מִי זֶה אֲשֶׁר יַבְטִיחֲךָ?

בְּצֵל אֵל חֲסֵה / וְחֻקָּיו עֲשֵׂה, / כִּי הוּא יְיַשֵּׁר אָרְחֲךָ.

הוּא נִצְחֲךָ, / הוּא נִכְחֲךָ, / הוּא בְקִרְבְּךָ, הוּא סְבִיבְךָ –

וְאָהַבְתָּ / אֶת אֲדוֹנָי / אֱלֹהֶיךָ בְּכָל לְבָבְךָ!



בְּכָל מַרְאֶה / אֲשֶׁר תִּרְאֶה / עֵד, כִּי אֵין זוּלָתוֹ.

וְכָל שֵׁמַע / אֲשֶׁר תִּשְׁמַע / יַשְׁמִיעַ תְּהִלָּתוֹ.

וְכָל רֵיחַ / תָּרִיחַ / יוֹדִיעַ עֳלִילָתוֹ.

וְכָל טַעַם / אֲשֶׁר תִּטְעַם / יְבָאֵר סוֹד גְּדֻלָּתוֹ.

וְיָדֶיךָ – / עֵדֶיךָ, / לְהָמִישׁ בָּם פְּעֻלָּתוֹ.

וְכָל מַדָּע / אֲשֶׁר תֵּדַע – / בְּךָ נָכוֹן בְּמִלָּתוֹ.

וְכָל שֶׂכֶל / אֲשֶׁר תִּשְׂכַּל – / מֵאִתּוֹ תְּחִלָּתוֹ.

הוּא בְרָאֲךָ, / הוּא נְשָׂאֲךָ / בְּאַהֲבָתוֹ וְחֶמְלָתוֹ,

יְעוֹדֶדְךָ, / יְלַמֶּדְךָ, / בְּאַחֲרִיתְךָ לְהֵיטִיבְךָ!



רֹאשׁ וָסוֹף / אִם תִּכְסֹף / לָדַעַת מַשְׂכֻּרְתְּךָ –

בְּרֵאשִׁיתְךָ / וְאַחְרִיתְךָ / אַתָּה כְּפִי מַחְשַׁבְתְּךָ;

כְּטוֹבָתְךָ / כְּרָעָתְךָ / נְתוּנָה לָךְ כִּיכָלְתְּךָ,

וְיוֹצֶרְךָ / יַחְקָרְךָ / וְיֵדַע קוּמְךָ וְצֵאתְךָ.

וְאֵיךְ תִּשְׂטֶה, / וְאָן תִּטֶּה? / וּמִדַּעְתּוֹ – דַעְתְּךָ!

עֲזֹב הֶבֶל / מְתֵי תֶבֶל / וּשְׁמֹר אֶת מִשְׁמַרְתְּךָ!

יְלוּד אֲדָמָה, / חֲשֹׁב: לָמָּה / תִּיגַע לְזָרִים בִּלְתְּךָ?

פְּקַח עֵינְךָ / וְהַט אָזְנְךָ, / רְאֵה: שְׁאֵלָתְךָ אִתְּךָ!

וְהַבּוֹרֵא / בְקוֹל קוֹרֵא: / ‘אִם תָּשׁוּב – וַאֲשִׁיבְךָ!’



מַה יִּשְׁאַל / לֵב נוֹאַל – / וְטוֹבַת אֵל עוֹדֶפֶת,

וּמַה תִּירָא – / וְהָעֶזְרָה / עָלֶיךָ חוֹפֶפֶת.

גַּן עֶדְנוֹ / הוּא רְצוֹנוֹ / וּבִלְתּוֹ מְקֹרָא תֹּפֶת,

כִּי יְחִידַת-/ מוֹאֵס-בְּדָת / בְּאֵשׁ הַנְּדוֹד נִשְׂרֶפֶת.

שְׁמֹר חֻקִּים / צַדִּיקִים / הַמְתוּקִים מִנֹּפֶת,

תִּתְרוֹמֵם, / אִם תְּשִׂימֵם / עַל רֹאשְׁךָ מִצְנֶפֶת.

הֵם הוֹדְךָ – / וְעַל יָדְךָ / תְּנֵמוֹ לְאוֹת וּלְמוֹפֵת,

בָּם הָבֵן – / וְהָיוּ בֵין / עֵינֶיךָ לְטוֹטֶפֶת –

אֲשֶׁר תֵּפֶן / אֲלֵיהֶם, פֶּן / יָסוּרוּ מִלְּבָבְךָ.
