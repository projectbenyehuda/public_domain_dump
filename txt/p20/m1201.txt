


	
	אופן
	



אופן / ר' אברהם אבן-עזרא


חַיָּה עֲצוּמָה / מִכָּל בְּרִיאָה, // מִבֶּן אֲדָמָה / דַעְתָּה פְּלִיאָה.

אָנָה מְעוֹנָה / מִשָּׁם נְשָׁמָה?



בָּאִישׁ נְתָנָהּ, / כַּמָּה חֲכָמָה!



אֵל חַי צְפָנָהּ / עַד נֶעֱלָמָה.



אֵיכָה וְעַל מָה / הָיְתָה כְלוּאָה // בַּגֵּו, וּבַמָּה / שָׁמַע וְרָאָה?

בַּמֶּה יְקֹרָא / שֵׁם רָם וְגָאָה? –



וּדְמוּת וְצוּרָה, / מִשְׁמָע וּמַרְאָה



אֵין לוֹ, וְנוֹרָא / זִכְרוֹ לְנִרְאָה!



שַׁבְתִּי כְדֻמָּה / מִזּוֹ קְרִיאָה // וָאֶשְׁמְעָה מָה / אָמְרָה נְבוּאָה.

רָצוּ שְׁחָקִים / לַיְלָה וְיוֹמָם,



מִצֹּר חֲזָקִים – / מִי זֶה הֱרִימָם?



כָּל הַמְּצוּקִים / אֶרֶץ מְקוֹמָם,



כֻּלָּה בְּחָכְמָה / קִנְיָן מְלֵאָה // אַךְ עַל בְּלִימָה / תּוֹכָם תְּלוּאָה.

מַרְאוֹת הֲמֻלּוֹת / אֵין קֵץ לְאוֹרָם,



חוֹזְריִם חֲלִילוֹת / בִּמְעוֹן יְקָרָם,



נוֹשְׂאִים תְּהִלּוֹת / תָּמִיד לְצוּרָם;



שִׁירָה נְעִימָה / עַל פֶּה נְשׂוּאָה, / עוֹנִים בְּאֵימָה, / אוֹמְרִים בְּיִרְאָה
