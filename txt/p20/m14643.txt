


	
	אַתָּה אֵל רֹאִי
	




אַתָּה אֵל רֹאִי, / אַתְּ מַעֲרִיצִי, אַתְּ מוֹרָאִי!



בָּחַנְתִּי לִבִּי – / אֶל חַסְדְּךָ אֶדֹּם אֶתְחוֹלֵל,

שִׁמְךָ מִשְׂגַּבִּי / יוֹם סוֹלְלוֹת יִשְׁפֹּךְ מִסְתּוֹלֵל.

הוֹאֵל וּפְנֵה בִי, / הוֹצֵא כְפִי יָקָר מִזּוֹלֵל

וּמְשַׂנְאִי / קוּם לַעֲרֹץ! / גִּדְרוֹ פְּרֹץ, / יָבוֹא רְבֹץ / תַּחַת מַשָּׂאִי!



רָחַשׁ מַרְחִיקִי / לִמְצֹא כְּבוֹד אֶל בַּחְקֹר דָּבָר,

קָרוֹב מַצְּדִיקִי / לִמְצֹא עֲוֹנוֹ, כִּי בָךְ נִדְבָּר.

מֵהוֹדָךְ חִזְקִי, / מִקּוֹל בַּכֹּחַ בֶּהָדָר

קוֹרֵא לִצְבָאִי: / שֵׂכֶל בְּתֹם / תּוֹרָה חֲתֹם, / אוֹתִי בְתֹם / דִּרְשׁוּ בְּהִמָּצְאִי!



הַרְבֵּה מִזְבְּחוֹת / לַחְטֹא אֱוִיל יַעֲרֹךְ: יִתְיַבָּב

פַּעַם, תּוֹכֵחוֹת / יַעְנֶה – גְּדוֹלִים חִקְרֵי לֵבָב!

אַחַר לִמְנָחוֹת / נִגַּשׂ לְעֵצוֹ הוֹלֵךְ שׁוֹבָב.

נחי מלטאי / הוּא חֶבְלְךָ / יִשְׁאָלְךָ / הֵכִין לְךָ / בַּחֶסֶד כִּסְאִי.



מוֹשִׁיעַ חוֹסִים, / צוֹפִים לְבַשֵּׂר שׁוֹאֲפִים מְשָׁח:

'מִשָּׂרֵי מִסִּים / קָרָאתִי לִבְנִי וָאֶקַּח

מִקֶּרֶב שׁוֹסִים; / אֶבְחַר בּוֹ, מַטֵּהוּ יִפְרָח.

עוֹד לִמְקֹרָאִי / יוּשַׁב שְׂשׂוֹן / יִשְׁעִי, שְׁכֹן / אוֹר עַל מְכוֹן / הַר מִקְרָאִי!'
