


	
	לִבִּי עֲלֵי צַלְעִי
	








לִבִּי עֲלֵי צַלְעִי / יוֹחִיל לְמָנוֹחַ

רַבָּה יְגִיעָתִי / לִמְאֹד בְּלִי כֹחַ.



זוֹחֵל בְּבֵית כֶּלֶא / לֹא אֶמְצְאָה נֹפֶשׁ

בֶּאֱמֶת אֲנִי חוֹלֶה / בַּגּוּף וּבַנֶּפֶשׁ

נוֹפֵל וְאֵין דּוֹלֶה / מִטִּיט וּמֵרֶפֶשׁ

אֵל נָא עֲשֵׂה כָלָה / וּכְתֹב שְׁטָר חֹפֶשׁ

וּמְחֵה כְעָב דִּמְעִי / וּרְצֵה כְנִיחוֹחַ

וֶאֱדוֹם וּבֶן אָמָה / לִרְטֹשׁ וְלִזְנֹחַ.



כָּלָה מְאוֹר פָּנַי / חָשַׁךְ עֲדֵי הִקְדִּיר

תַּמּוּ לְרִיק שָׁנַי / פִּשְׁעִי לְרֹב אַסְדִּיר

גָּבַר הֲמוֹן מוֹנַי / נִינֵי עֲרָב הִעְדִּיר

אֶשָּׂא לְךָ עֵינַי / נוֹרָא וְגַם אַדִּיר

לָךְ עַנְנֵי דִּמְעִי / אֶשְׁלַח כְּשִׁילוֹחַ

דּוֹפֵק שְׁעָרֶיךָ / מַהֵר וְלִפְתֹּחַ.



רָעוּץ בְּתוֹךְ מוֹקֵשׁ / כִּמְעַט כְּעוֹף יִדְאֶה

יִדְמֶה לְמֹץ אוֹ קַשׁ / שֵׁפֶל אֱנוֹשׁ נִכְאֶה

נִשְׁפָּט כְּדוֹר עִקֵּשׁ / הוּא מִסְּבֹל יִלְאֶה

אֶת עַבְדְּךָ בַּקֵּשׁ / רוֹאֶה בְלִי נִרְאֶה

תִּמְרַח עֲלֵי נִגְעִי / יִשְׁבֹּת כְּמֵי נֹחַ

הַצַּר וְהָאוֹיֵב / יִדְכֶּה וְיָשׁוֹחַ.



יָמַי בְּכוּר עֹנִי / תַּמּוּ בְּרֹאשׁ צוּרִים

כִּמְעַט וְאֵינֶנִּי / עֵת אֶעֱבֹד זָרִים

אֶגְוַע עֲלֵי קִנִּי / נִמְכַּר לְאַכְזָרִים

מַהֵר וְחָנֵּנִי / מַשְׁפִּיל וְגַם מֵרִים

כַּמָּה יְהֶא בִּצְעִי / נִכְתָב עֲלֵי לוּחַ

צַוֵּה אֱלֹהִים חַי / לִמְחֹל וְלִסְלֹחַ.



הַיּוֹם מְרִי שִׂיחִי / אֶעֱרֹךְ לְנֶגְדֶּךָ

בִּכְיִי בְּמַר רוּחִי / שִׂימָה בְּנֹאדֶךָ

אֶצְעַק בְּכָל כֹּחִי / תַּעְנֶה בְּחַסְדֶּךָ

נַפְשִׁי בְּעוֹד תֶּחִי / אַפְקִיד בְּיָדֶךָ

שָׂשִׂים עֲלֵי צַלְעִי / תִּזְרֵם לְכָל רוּחַ

שָׁבוּי קְרָא לִדְרוֹר / אָסוּר פְּקַח קוֹחַ.
