


	
	עַנְנֵי דִמְעִי שְׁקוּדִים
	






עַנְנֵי דִמְעִי שְׁקוּדִים / עַל לְחִי מוֹשְׁכִים וְיוֹרְדִים

שִׁבְּרוּ לִבִּי חֲרָפוֹת / כִּי בְאָזְנַי קוֹל פְּחָדִים

אָחֲזָה גּוּפִי רְעָדָה / גַּם וּמָתְנַיִם מְעוּדִים

אֵשׁ לְהָבוֹת תּוֹךְ כְּסָלַי / תּוֹךְ לְבָבִי הַיְּקוּדִים

כִּי אֲנִי כַּיּוֹם מְאֹד רַד / כַּעֲנִיִּים הַמְּרוּדִים

אִם לְרֶגֶל הַמְּלָאכָה / אוֹ לְרֶגֶל הַיְלָדִים

מִקְהֲלוֹת עִירִי אֲסוּרִים / הֵם בְּתוֹךְ הַפַּח לְכוּדִים

אֶשְׁתְּקַד שָׂרִים וְעַתָּה / הֵם עֲבָדִים לַעֲבָדִים

לֵב תְּנוּ בָּרִים וְשִׁמְעוּ / אֶל דְּבַר אֵל הַחֲרֵדִים

הַנְקַלָּה זֹאת בְּעֵינְכֶם / נִהֲיֶה תַּחַת פְּקִידִים

תַּם זְמַנֵּנוּ בְּמִשְׁמָר / יוֹם וְגַם לַיְלָה עֲתִידִים

שַׁאֲלוּ עוֹבְרִים וְשָׁבִים / שַׁאֲלוּ נָעִים וְנָדִים

יֵשׁ רְפוּאָה לַנְּגוּעִים / אוֹ לְהַתִּיר הָעֲקוּדִים

אֵי חֲכָמִים הַשְּׁקוּדִים / אֵי יְחִידִים הַשְּׂרִידִים

אֵי גְבָרִים סוֹבְלֵי עֹל / אֵי גְבָרִים הַחֲסִידִים

שׁוֹקְדֵי סִפְרֵי תְּעוּדוֹת / בַּחֲצִי לַיִל וְעוֹמְדִים

הַמְחַלִּים אֶת פְּנֵי אֵל / בַּעֲבוּר צֹאן הָאֲבוּדִים

הֵן בְּיַד עֵשָׂו בְּזוּזִים / הֵן בְּיַד עֶבֶד שְׁדוּדִים

פָּרְשׂוּ רֶשֶׁת לְרַגְלַי / פָּרְשׂוּ לִי הַמְּצוֹדִים

טָמְנוּ פַחִים לְנֶגְדִּי / תִּקְּנוּ עוֹל לַיְהוּדִים

זָעֲקוּ מִקִּיר אֲבָנִים / אֵיךְ יְהוּ מַשָּׂא יְדִידִים

הַלְּבֵנִים גַּם וְעֵצִים / רוֹכְבִים יַחְדָּו צְמָדִים

כָּל יְמֵי צְבָאִי אֲיַחֵל / לִרְאוֹת בָּנִים חֲמוּדִים

יוֹדְעֵי מַשָּׂא וּמַתָּן / בַּסְּפָרִים הֵם לְמוּדִים

דָּר בְּרוּם עוֹלָם וּמַחְבִּיר / הַפְּרוּדִים לַאֲחָדִים

תַּעֲלִים עַיִן וְכַמָּה / אַךְ שְׁמַע וּרְצֵה לְמוּדִים

בַּחֲטָאִים מִנְּעוּרִים / עַד זְקוּנִים הֵם מְעִידִים

עָצְמוּ צָרוֹת לְסַפֵּר / כִּי יְדֵי הָעֵט כְּבֵדִים

הַלְעוֹלָמִים אֱלֹהִים / לֹא תְכַלֶּה מַעֲבִידִים

כָּל בְּנֵי אָמָה וְנִינָם / מַחֲנֵה צָרִים וְצָדִים

הַמְשַׂנְאִים עָבְרוּ חֹק / חָלְפוּ דִּבְרֵי נְגִידִים

אֵיךְ וּמָתַי אֵשׁ תְּלַהֵט / כָּל בְּנֵי רִשְׁעָה וְזֵדִים

אֵל לְמַעֲנָךְ מַלְּטֵנוּ / מִיְּדֵי קָמִים וְרוֹדִים

גַּם בְּחַסְדָּךְ תַּעְזְרֵנוּ / לָךְ נְהֶא הַיּוֹם עֲבָדִים

סוֹף דְּבָרַי אֶדְרְשָׁה אֵל / אֶקְרְאָה גוֹמֵל חֲסָדִים

יַעֲלֶה שִׂיחַ שְׂפָתַי / אֶל מְקוֹם עוֹלוֹת תְּמִידִים.
