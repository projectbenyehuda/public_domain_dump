


	
	לְאִישׁ נִכְאֶה זְמָן הִגְדִּיל כְּאֵבוֹ
	








לְאִישׁ נִכְאֶה זְמָן הִגְדִּיל כְּאֵבוֹ / לְמִי יַגִּיד מְרִי נִגְעֵי לְבָבוֹ

כְּאִלּוּ הוּא עֲלֵי כַנְפֵי נְשָׁרִים / וְנוֹדֵד מֵחֳלִי לִמְאֹד עֲצָבוֹ

בְּנֵי יָמִים צְנָפוּהוּ כְּכַדּוּר / עֲלֵי אֶרֶץ גְּזֵרָה שָׁת נְתִיבוֹ

וְסָגַר בַּעֲדוֹ דַּלְתֵי שְׁחָקָיו / וְחָשַׁךְ מֵעֲרֹף עָלָיו רְבִיבוֹ

וְנִלְחַם בִּי חֲשָׁבַנִי לְאוֹיֵב / עֲדֵי אָסַר לְפָרָשָׁיו וְרִכְבּוֹ

עֲלֵי אֵלֶּה נְפוּגוֹתִי בְמִשְׁמַר / וְהֵן לֹא אוּכְלָה לָצֵאת וְלָבוֹא

וְאֵין חוֹשֵׂךְ לְיָם גָּלָיו וְדָכְיָם / לְבַתֵּר מוֹשְׁכוֹת שִׁתְיוֹ וְעִרְבוֹ

וְנִלְאֵיתִי לְהַשִּׂיג קֵץ שְׁבִילָיו / בְּאוּלַי אֶמְצְאָה אָרְכּוֹ וְרָחְבּוֹ

הֲקִימוֹתִי בְּכָל גִּבְעָה מְצַפֶּה / לְהַגִּיד מִי אֲשֶׁר הוּא עַם קְרוֹבוֹ

וְכָל פִּנָּה יְשׁוֹטֵט הַנְּתִיבוֹת / בְּמִזְרָחוֹ וְאִם מִמַּעֲרָבוֹ

וְהִנֵּה הוּא כְמוֹ אַיָּל יְדַלֵּג / לְעֻמָּתִי לְהִתְבַּשֵּׂר בְּנִיבוֹ

וְאָמַר לִי: תְּמוּנַת אִישׁ רְאִיתִיו / בְּהִלּוּכוֹ בְּשׂוֹרָה נִמְצְאָה בוֹ

וּמִנְהָגוֹ כְמִנְהַג אִישׁ חֲכַם לֵב / לְיֹפִי שׁוּר מְרוּצָתוֹ וְשׁוּבוֹ

בְּעוֹדֶנּוּ מְדַבֵּר לִי וְזֶה בָא / וּפָתַח אִגְּרוֹת טוּרֵי לְבָבוֹ

וְהִנֵּה הֵם לְאִישׁ מַשְׂכִּיל שְׁלוּחִים / אֲשֶׁר קָם לַזְּמָן אוֹרוֹ וְרִיבוֹ

וְשׁוּרוֹתָיו כְמוֹר עוֹבֵר יְפִיחוּן / בְּקָרְאִי בָם וְלִבִּי נָס שְׁבִיבוֹ

וּמֵהֶם טוּר בְּתִפְאֶרֶת יְדַבֵּר / וְאָחִיו אַחֲרָיו אוֹחֵז עֲקֵבוֹ

כְּאִלּוּ הֵם כְּכוֹכְבֵי רוּם סְדוּרִים / עֲלֵי שַׁחַק וּמַתְאִימִים סְבִיבוֹ

מְעִידִים עַל כְּרוּב מִמְשַׁח וְהוֹדוֹ / אֲבִי יֹשֶׁר וְכָל חָכְמָה בְּקִרְבּוֹ

וְהַגַּלְגַּל יְסוֹבֵב הַבְּרוּאִים / וְהוּא בֶן אִישׁ וְהַגַּלְגַּל בְּחֻבּוֹ

וְלֹא נִמְצָא סְגוֹר כִּי אִם פְּתָחוֹ / וְכָל סָתוּם כְּאַיִן הוּא חֲשָׁבוֹ

וּבָטוּחַ בְּחוּטוֹ הַמְּשֻׁלָּשׁ / בְּמַעְנֶה טוֹב לְכָל שׁוֹאֵל הֱשִׁיבוֹ

פְּנֵי הַדּוֹר וְתִפְאֶרֶת גְּאוֹנוֹ / מְאֹד עָנָו בְּעֵת קוּמוֹ וְשָׁכְבוֹ

וְלִרְגָעַי לְטוֹבָה אֶזְכְּרֶנּוּ / וְאַפְקִידוֹ לְשָׂם עָבִים רְכוּבוֹ

יְחוֹפֵף זִכְרְךָ שַׁדַּי בְּשָׁלוֹם / וְתִזְכֶּה לַחֲזוֹת נָעְמוֹ וְטוּבוֹ.
