


	
	מְרִי שִׂיחִי וְאַנְחָתִי כְּבֵדָה
	








מְרִי שִׂיחִי וְאַנְחָתִי כְּבֵדָה / וְנַפְשִׁי מִבְּלִי סוֹמֵךְ יְחִידָה

וְנִתָּכוּ כְמַיִם שַׁאֲגוֹתַי / וְדִמְעָתִי עֲלֵי עַרְשִׂי רְבוּדָה

וְרַעְיוֹנַי כְּיַמִּים יֶהֱמָיוּן / וְשִׁינָתִי כְּמוֹ לַיְלָה שְׁדוּדָה

בְּחָשְׁבָם כִּי אֱמֶת נוֹסְעִים לְיָטְבַת / וְתָעוּ בַחֲנִיָּתָם חֲרָדָה

וְעוֹד נָסְעוּ לְאֶדְרֶעִי וּמָרָה / וְאֵין כֹחַ לְהָשִׁיב הָאֲבֵדָה

וְהֵצַר הַזְּמָן צָרָיו לְעִתִּים / וְצָרָתִי בְּכָל רֶגַע לְבוּדָה

וּבָחַרְתִּי לְמוֹתִי מֵחֲיוֹתִי / וּמָה אוֹחִיל וְאֵין לִי עוֹד עֲמִידָה

וְאֵין מֵשִׁיב לְנַפְשִׁי מִיְּקוֹדָהּ / בְּטִיב מָשָׁל וּבִמְלִיצָה וְחִידָה

וּבָגַד אָח כְּמוֹ נַחַל אֲפִיקִים / בְּעֵת עָבַר וְעָצְמָה הַבְּגִידָה

וְאֵין לִי בַזְּמָן הָרָע עֲלִיָּה / אֲשֶׁר לֹא תִהְיֶה עִמָּהּ יְרִידָה

זְמַנִּי הֶעֱטַנִי סוּת כְּלִמָּה / וְלֹא אוּכַל רְתֹם רִכְבֵּי פְרִידָה

וּמִי יִתֵּן וְיַרְפֶּה לִי זְמַנִּי / וְאָשִׂים הַנְּסִיעָה לִי לְצֵידָה

בְּיַעַן כִּי אֲנִי עֶבֶד לְעַבְדִּי / וְגַם רַגְלִי בְּתוֹךְ הַפַּח לְכוּדָה

וְהַבָּנִים לְאַכְזָרִים מְסוּרִים / בְּיוֹם צָרָה וְעֶבְרָה עִם קְפָדָה

וְהָאָבוֹת אֲסוּרִים בַּכְּבָלִים / מְפֻזָּרִים בְּכָל עָרֵי מְצוּדָה

וְנַפְשִׁי תֶּהֱמֶה עַל רֹב עֲווֹנִי / וּמָה אַעַשׂ וְהִיא עָלַי מְעִידָה

וְעַד עַתָּה בְּטִיט יָוֵן שְׁקוּעָה / הֲיֹאמְרוּ לָהּ בְּשָׁמַיִם חֲסִידָה

וְאֵין דּוֹרֵשׁ לְבָתֵי הַתְּבוּנוֹת / חֲתוֹם תּוֹרָה וְגַם צוֹר לַתְּעוּדָה

וְהַחֶרֶב אֲשֶׁר הָיְתָה מְרוּטָה / מְקוּלְקֶלֶת וּפָנֶיהָ חֲמוּדָה

הֲלֹא קֵהָה וְגָבְרוּ הַחֲיָלִים / וְרֹעַ הַזְּמָן הֶעֱלָה חֲלוּדָה

אֲבָל דַּעְתִּי בְּשׁוֹשַׁנִּים לְלַקֵּט / וְתוּשִׁיָּה עֲלֵי לִבִּי אֲפוּדָה

וְאֵל בּוֹחֵן כְּלָיוֹת עִם לְבָבוֹת / וְכָל נֶעְלָם בְּנֶפֶשׁ אִישׁ וְסוֹדָהּ

לְמִי הֵכִין בְּאֶלֶף הָר בְּהֵמוֹת / וְלִוְיָתָן מְתֻקָּן לַסְּעוֹדָה

הֲלֹא לַמְיַחֲלִים חַסְדֵי כְּבוֹדוֹ / מְצָאוּהוּ בְּרֹב נִיעָה וְנִידָה

אֱלֹהִים חַי וּמַתִּיר הָאֲסוּרִים / לְךָ אוֹדֶה וְלָךְ אֶכְרַע בְּקִידָה

תְּנַהֲלֵנִי יְמֵי חֶלְדִּי וְחֶדְלִי / וְזָכְרֵנִי בְּיוֹם תָּבוֹא פְּקִידָה

אֲזַמֵּר לָךְ שְׁחָרִים וַעֲרָבִים / וְתִרְצֵנִי וְתִיטַב הָעֲבוֹדָה

וְיוֹם תָּבוֹא בְּמִשְׁפָּט עַל עֲלֻמוֹת / תְּזַכֵּנִי בְּתוֹךְ קָהָל וְעֵדָה

מְשֹׁךְ חַסְדָּךְ לְיוֹדְעֶיךָ בְּחֶמְלָה / בְּעֵת תֶּעֱרַב לְךָ מִנְחַת יְהוּדָה

וְעוֹד לֹא אַאֲרִיךְ לָשׁוֹן בְּזָכְרִי / מְרִי שִׂיחִי וְאַנְחָתִי כְבֵדָה.
