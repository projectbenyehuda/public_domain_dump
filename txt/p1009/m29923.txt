


	
	רְאוּ מֶלֶךְ אֲשֶׁר מִלָּא חֲמָסִים
	








רְאוּ מֶלֶךְ אֲשֶׁר מִלָּא חֲמָסִים / פְּנֵי תֵבֵל וְנֶעְדָּרוּ מְשׂוֹשִׂים

מְלַאכְתּוֹ הָיְתָה קֶשֶׁת לְתֻמּוֹ / וּמַחְנוֹתָיו עֲלֵי כָתֵף עֲמוּסִים

וְשׁוֹפֵךְ דָּם לְאֶבְיוֹנִים נְקִיִּים / וְעָזַב כָּל מְתֵי חַיִל הֲרוּסִים

בְּעֵינָיו הֵם כְּדֹמֶן לָאֲדָמָה / וְנֶחְשָׁבוּ כְּנִבְלֵי הַחֲרָסִים

עֲדֵי שָׁבוּ לְכָל עוֹבֵר וְכָל שָׁב / בְּכָל חֶרְפָּה וּבִכְלִמָּה מְכֻסִּים

וְאֵין יוֹעֵץ לְהָשִׁיב אֶת חֲרוֹנוֹ / וְלֹא יִהְיֶה כְּאוֹצָרוֹת בְּלוּסִים

וּמִשְׁנֵהוּ אֲבִיר רֵעָיו מְקַנֵּא / לְמַעְשֵׂהוּ וְשַׂרְעַפָּיו אֲנוּסִים

אֲזַי אָמַר לְמַלְכּוֹ: הֵן אֲדוֹנִי / אֲנִי בוֹחֵר לְקֶשֶׁת נֶעֱלָסִים

וְתִהְיֶה לִי כְּחוֹמָה בֵּין גְּדוּדִים / וּבָהּ אֶהְיֶה כְּעוֹמֵד בֵּין הֲדַסִּים

בְּמָשְׁכִי בָהּ לְנֶגֶד צוֹרְרֶיךָ / יְהוּ כוֹרְעִים וְלֹא נִהְיֶה לְשׁוֹסִים

וְהַמֶּלֶךְ בְּשָׁמְעוֹ רָץ כְּאַיָּל / וְעָשָׂה כָל יְפִי קַשְׁתּוֹת כְּמוּסִים

לְמִשְׁנֵהוּ בְּתִפְאַרְתּוֹ לְפָנָיו / וְעֹז עִם הוֹד וְתִפְאֶרֶת לְעוֹשִׂים

וְנָתַן לוֹ וְקִבַּל לָהּ בְּכָבוֹד / וְנָשַׁק אֶת יְדֵי מַלְכּוֹ פְּרוּשִׂים

וְהָפַךְ בָּהּ יְמִינוֹ עִם שְׂמֹאלוֹ / עֲדֵי שָׁבַר עֲדִי קַשְׁתּוֹ כְּמֵשִׂים

וְהַמֶּלֶךְ אֲזַי קָם בַּחֲרוֹנוֹ / וְהִתְקַצַּף וְהִתְרַפַּס רְפָסִים

וּבִקַּשׁ לַהֲרֹס כָּל אַרְמְנוֹתָיו / וְכִמְעַט קָט כְּבִקְעָה לָרְכָסִים

וְהַמִּשְׁנֶה בְּנַחַת דִּבְּרוֹתָיו / בְּחָכְמָתוֹ וְנִפְלָאוֹת וְנִסִּים

מְשָׁכָהוּ בְּחַבְלֵי אַהֲבָתוֹ / וְאִם הָיְתָה חֲמָתוֹ כָעֲכָסִים

וְהֵשִׁיב מַחֲנֵה קִצְפּוֹ וְזַעְפּוֹ / וּמִקִּרְבּוֹ וּמִלִּבּוֹ כְּטָסִים

בְּאָמְרוֹ: הוֹי אֲדוֹנִי לֹא תְדַמֶּה / לְפוֹעַל אֵל בְּיָדֶיךָ רְמוּסִים

וּפֹעַל כַּפְּךָ כָּזֶה לְמֵאוֹת / בְּחָכְמָתָם כְּחָכְמָתָךְ מְנֻסִּים

וְקִנֵּאתָ לְשִׁבְרוֹ עַד לְכִמְעַט / תְּאַבֵּד צוֹרְרֶיךָ כַּיְּבוּסִים

וְאַף כִּי צוּר אֲשֶׁר יָצַר יְצוּרִים / לְבַדּוֹ כִּי בְנַפְשׁוֹתָם תְּפוּסִים

הֲלֹא קַנֹּא יְקַנֵּא עַל בְּרוּאָיו / וְעֶבְרָתוֹ כִּקְדֹחַ אֵשׁ הֲמָסִים

וְהַמֶּלֶךְ כְּשָׁמְעוֹ שָׁב לְאָחוֹר / נְתִיבוֹתָיו בְּשׁוּבוֹ הֵם מְכֻסִּים

וְשָׁמַע לַעֲצַת מִשְׁנֶה חֲכַם לֵב / וְשָׁת לִבּוֹ לְמוֹשִׁיעַ לְחוֹסִים

שְׁאָלָהוּ עֲצָתָךְ מִע יְעָצָהּ / הֱשִׁיבוֹ אִישׁ יְהוּדִי מֵעֲמוּסִים

אֲזַי שָׁלַח וְהֶעְלַנִי לְמוּלוֹ / בְּחָכְמוֹתַי אֲשֶׁר לַיָּם מְכַסִּים.
