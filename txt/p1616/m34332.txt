


	
	עֲלֹז לֵב אֲסִיר
	







בכתב יד רשום עליו: להחכם הר׳ שלמה בונפיד, בעבור הסתיו והגשם חלף והעת עת דודים שלחתי זה השיר לידידי יהוסף ואמר.






עֲלֹז, לֵב אֲסִיר תִּקְוָה, / אֲשֶׁר לַכְּפוֹר חוֹלֶה,

עֲלֹז, כִּי כְבָר שֶׁמֶשׁ / מְסוֹבֵב בְּרֹאשׁ טָלֶה

וְהָלַךְ לְמַסָּעָיו / לְמַעְלוֹת וְלִרְגָעָיו

וְיִתְגַּל בְּרוּם גַּלְגַּל / לְהַצְהִיל לְבַב גּוֹלֶה,

וְנָטַשׁ פְּאַת דָּרוֹם / וְעָלָה לְרֹאשׁ מָרוֹם

וְנֹכַח לְמַשְׁוֵה יוֹם / יְחַצֵּץ וְיִגָּלֶה.

סְתָו עִם כְּפוֹר בָּרַח / וְכָל עֵץ פְּרִי פָרַח,

וְחֶרֶס בְּחוּג יִזְרַח / וְסַהַר בְּאוֹר מָלֵא.

וְיַחְשֹׁר בְּכֹחַ שׁוֹר / גְּשָׁמִים, וְאָז יָשֹׁר

לְבָנוֹן בְּאֶגְלֵי טַל / וְכַרְמֶל בְּגִיל מָלֵא.

וְלָבְשׁוּ כְתֹנֶת פַּס / תְּכֵלֶת וְחוּר כַּרְפַּס

עֲמָקִים, וְכִתְמֵי פָז / יְרַקְרַק עֲלֵי עָלֶה,

וְעַל צַוְּארֵי יוֹנִים / רְבִידִים וְשַׂהְרוֹנִים

עֲשׂוּיִים בְּיַד עָב טַל / וְעָנָן בְּאֵד יַעְלֶה,

וְהַתּוֹר וְהַזָּמִיר / יְשִׁירוּן בְּרֹאשׁ אָמִיר,

כְּכִנּוֹר וְקוֹל נָעִים / בְּשִׁיר יַעֲשׂוּ פֶלֶא.

וְאַחִים בְּבֵין אָחִים / יְשִׂיחוּן עֲלֵי שִׂיחִים

וְסוֹדָם לְיַעְלוֹת חֵן / לְאַט כָּל יְדִיד גּוֹלֶה.

בְּתוּלוֹת כְּאַיָּלוֹת / בְּתֻפִּים וּבִמְחוֹלוֹת

בְּתוֹךְ גַּן תְּרַקֵּדְנָה / בְּזֶמֶר וְשִׁיר נִפְלֶה

וְעִם פֶּה וְנִיב חָלָק / וְאִישׁוֹן כְּמוֹ בָרָק

תְּשִׂימוּן בְּגַן יָרָק / עֲלָמִים בְּבֵית כֶּלֶא.

אֲזַי נֶחֱלוּ חוֹשְׁקִים / בְּאֵשׁ אַהֲבָה דוֹלְקִים

וְרַעְיוֹת יְקָר דּוֹלְקִים / כְּמוֹ הַזְּאֵב טָלֶה –

אֲבָל כּוֹס מָלֵא יַיִן / בְּיַד כָּל יְפַת עַיִן

לְעוֹרֵר עֲלֵי עַיִן / לְבַב כָּל יְדִיד נֶחְלֶה.

תְּנוּ עֹז, מְתֵי חֵשֶׁק, / תְּנוּ עֹז לְבַת גֶּפֶן

אֲשֶׁר תַּעֲשִׁיר פִּתְאוֹם / לְבַב רָשׁ וְאִישׁ נִקְלֶה

וְחִזְקוּ בְאַהְבַתְכֶם / וְסִבְלוּ עֲבוֹדַתְכֶם –

צְבָא שֵׁם קְחוּ לָכֶם / לְמוֹפֵת וְסוֹד מֻפְלֶא:

עֲגֻלּוֹת וְכַדּוּרִים / יְקָרִים כְּסַפִּירִים

יְרוּצוּן לְהַשִּׂיגוֹ / וְחִשְׁקָם בְּלִי יִכְלֶה,

וְרוּצוּ אֱלֵי עֹפֶר / יְהוֹסֵף כְּלִיל שֶׁפֶר

אֲשֶׁר לוֹ צְפוּן חָכְמָה / וְכָל סוֹד וְכָל נִגְלֶה

וְהִלְכוּ לְפִי עֶרְכּוֹ / אֱלֵי יַעֲלַת חִשְׁקוֹ –

עֲדֵי עַד יְהִי סֻכּוֹ / בְשָׁלֵם וְלֹא יִבְלֶה!

וְאִמְרוּ אֱלֵי דוֹד צַח / בְּרִנָּה וְשִׁיר פָּתַח:

אֲסִיר הַנְּדוֹד פַּתַּח / אֲשֶׁר בַּעֲדוֹ חוֹלֶה.
