


	
	קְנֵה לְךָ
	




קְנֵה לְךָ מוּסָר / וּמִדּוֹת נְעִימוֹת / וּבִלְתֵּיהֶן תֶּחְסַר / לְעוֹפוֹת וּבְהֵמוֹת

נְמָלָה אֵין לָהּ שַׂר / וּמִגֶּזֶל תּוּסַר / וְחִטָּה אוֹסֶפֶת / בְּתוֹךְ הָאֲדָמוֹת

וְתַרְנְגוֹל שׁוֹרֵר / בַּחֲצוֹת יִתְעוֹרֵר / וְאַתָּה מִתְגּוֹרֵר / בְּרֹב הַתַּרְדֵּמוֹת

וְעוֹד נְדִיבוּת לוֹ / בְּעֵת מַאֲכָלוֹ / קְרֹב רֵעָיו אֶצְלוֹ / וְאִם הֵם בְּהֵמוֹת

וְגָמָל צָנוּעַ / וְאִם הוּא גָּרוּעַ / וְאַתָּה פָרוּעַ / וְחוֹשֵׁב מְזִמּוֹת

וּבָטֵל אֵיךְ תֵּשֵׁב / לְעַכָּבִישׁ הַקְשֵׁב / אֲשֶׁר תָּמִיד חוֹשֵׁב / וְאוֹרֵג בִּנְעִימוֹת

וְתִשְׁוֶה לִדְבוֹרִים / בְּיַחַד נֶחְבָּרִים / וּבוֹנִים חֲדָרִים / בְּשָׁוֶה בְּכַמּוּת

וְשׁוּר הָאַיֶּלֶת / לְלֹא לָהּ חוֹמֶלֶת / וְלַכֹּל גּוֹמֶלֶת / חֲסָדִים בְּמֵימוֹת

רְאֵה הָעוֹרְבִים / אֲשֶׁר אֵינָם טוֹבִים / וְהֵמָּה אוֹהֲבִים / אֲהוּבוֹת עֲצוּמוֹת

וְתַרְנְגוֹל הַבָּר / בְּאָנְסוֹ שֶׁגָּבַר / שְׁבוּעָתוֹ עָבַר / וְנֶחֱנַק עַל מוֹת

וְתוֹרִים וְיוֹנִים / אֲשֶׁר אֶצְלָךְ חוֹנִים / וְאֵינָם מְזַנִּים / כְּמַעְשֵׂה בְהֵמוֹת

וְעוֹפוֹת דַּרְכֵיהֶם / שְׁתוֹת אֶת מֵימֵיהֶם / וְנוֹשְׂאִים עֵינֵיהֶם / לְמַעְלָה לִמְרוֹמוֹת

וְאֵיךְ לֹא לָמַדְתָּ / וְהַטּוֹב הִשְׁלַכְתָּ / וְאָכֵן הוֹסַפְתָּ / בְּעָוֶל וּמִרְמוֹת.
