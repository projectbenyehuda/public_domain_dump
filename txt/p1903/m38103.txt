


	
	משל מב: חָרַשׁ בַּרְזֶל וּסְנֶה
	




גֵּאֲוַת אָדָם תַּשְׁפִּילֶנּוּ / וְלֹא יוּכַל לָדוּן עִם שֶׁתַּקִּיף מִמֶּנּוּ

חָרַשׁ בַּרְזֶל שָׁת אֶת טָרְחוֹ / וַיִּפְעַל בִּזְרוֹעַ
כֹּחוֹ / בַּרְזֶל גַּרְזֶן אֵין כָּמֹהוּ / פָּנִים קִלְקַל כִּי לֹא
קֵהוּ / וּבְכָל שְׁכֵנָיו יִוָּעֵץ / אָנָה יִקַּח לִרְצוֹנוֹ עֵץ /
לַעֲשׂוֹת בֵּית יָד חָזָק שִׁחַר / וְעֵץ לֹא יִרְקַב יִבְחָר /
הַיֹּעֲצִים הֱשִׁיבוּהוּ מַעֲנֶה / קַח לְךָ מִן דָּלִיּוֹת הַסְּנֶה /
קָשֶׁה מִן הָעֵצִים כֻּלָּם / מִן הָעוֹלָם וְעַד הָעוֹלָם / וַיֵּלֶךְ
אֶל הַסְּנֶה בְּמוֹעֲצוֹתָיו / וַיִּכְרֹת מִן דָּלִיּוֹתָיו / חִבֵּר
הַבַּרְזֶל בַּעֲמָלוֹ / עַל הָעֵץ אֲשֶׁר הֵכִין לוֹ / תִּקֵּן הַגַּרְזֶן
וְכִלָּהוּ / וַיֹּאמֶר לַדֶּבֶק טוֹב הוּא / וּלְנַסּוֹת טוּבוֹ וְהוֹדוֹ
/ בַּגַּרְזֶן נִדְחָה יָדוֹ / לִכְרֹת אֶת הַסְּנֶה עַד תֻּמּוֹ / שֹׁרֶשׁ
וְעָנָף כִּי בָא יוֹמוֹ.

וְהַמָּשָׁל בְּאִישׁ אוֹהֵב אֶת רֵעֵהוּ / אֲשֶׁר
טִפְּחָהוּ וְרִבָּהוּ / יַאֲכִילֵהוּ וְיַשְׁקֵהוּ / כַּאֲשֶׁר הָיָה
רָעֵב עֵרוֹם וָדָל / וּבִרְאוֹתוֹ כִּי שָׁמַן וְגָדַל / נֶהְפַּךְ לוֹ
לְבֶן־בְּלִיַּעַל / וּבְמֵטִיב אֵלָיו יִמְעַל / לְשַׁלֵּם לוֹ רָעָה
תַּחַת טוֹבָה / וְשִׂנְאָה תַּחַת אַהֲבָה / וַיְהִי לוֹ לְשָׂטָן בְּרֹב
מַשְׂטֵמָה / מִכָּל הָאָדָם אֲשֶׁר עַל פְּנֵי הָאֲדָמָה / וְהָיָה
אַדִּיר מִמֶּנּוּ וּמָשְׁלוֹ מִקִּרְבּוֹ / הֲיִתְפָּאֵר הַגַּרְזֶן עַל
הַחֹצֵב בּוֹ / וּמַר מִמָּוֶת זֶה הָאִישׁ מוֹצֵא / מְהָרְסָיו
וּמַחֲרִיבָיו מִמֶּנּוּ יֵצֵא.
