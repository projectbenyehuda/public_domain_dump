


	
	משל נב: לָבִיא וּזְאֵב, שֻׁעָל, שׁוֹר וְעֵגֶל
	




יֵשׁ בְּלֹא מִשְׁפָּט נִסְפֶּה / וּלְשׁוֹן חֲכָמִים מַרְפֵּא



לָבִיא הַמֹּלֵךְ בִּגְבוּרָתוֹ / זְאֵב וְשֻׁעָל קָרָא
לְחֶבְרָתוֹ / וַיֹּאמֶר לָהֶם: שְׁמָעוּנִי עַמִּי / הַלְּבִיָּא אֲשֶׁר
עִמִּי / זֶה חֹדֶשׁ יָמִים גּוּר לִי יָלְדָה / נֵלְכָה יַחַד לְהָבִיא
צֵדָה / וַיֵּלְכוּ עִמּוֹ שְׁנֵיהֶם / וַיַעֲבֹר מַלְכָּם לִפְנֵיהֶם /
וּמֵחֶבְרָתוֹ לְבָבָם מִגִּיל יִסְעָד / שְׁלֹשָׁה הֵמָּה מֵיטִיבֵי צָעַד
/ וּבִהְיוֹתָם בַּשָּׂדֶה כַּטֹּרְפִים הִשְׁקִיפוּ / וְשׁוֹר וּפָר
וָעֵגֶל בַּיּוֹם הַהוּא יַעֲרֹפוּ / וַיֹּאמֶר הָאֲרִי לַשֻּׁעָל /
הֻקַּמְתָּ עָל / הֲלִבְּךָ בְךָ הָעֳרַל / וְלֹא תֵדַע לְחַלֵּק שָׁלָל
בְּלִי גוֹרָל / וַיַּעַן הַשֻּׁעָל וַיֹּאמַר: צָעִיר אֲנִי לְיָמִים /
וְהַזְּאֵב נֶחְשָׁב בֵּין חֲכָמִים מְחֻכָּמִים / וְרֹב שָׁנִים יוֹדִיעוּ
חָכְמָה / וְתִהְיֶה מַשְׂכֻּרְתּוֹ שְׁלֵמָה / וַיֹּאמֶר הָאֲרִי לַזְּאֵב
/ הֶחָכָם לָמָּה כוֹאֵב / לְבָרֵר הַחֲלָקִים / כִּי כָל הַיּוֹצֵא
מִפִּיו יָקִים / וַיַּעַן הַזְּאֵב: הַשּׁוֹר לַאֲדֹנִי הַלָּבִיא /
וְהַפָּר אֶל הַלְּבִיָּא רַעְיָתוֹ לְהָבִיא / וְהָעֵגֶל בֵּינִי וּבֵין
הַשֻּׁעָל / וַיֹּאמֶר הָאֲרִי: בְּמֶרֶד וּבְמַעַל / וּבְרֶשַׁע
וּבְזָדוֹן וּבְמֶרִי / לֹא הִזְכַּרְתָּ חֵלֶק לִבְנִי בְכוֹרִי / עַל
כֵּן דָּמְךָ בְּרֹאשֶׁךָ / וְאַאֲדִיב אֶת נַפְשֶׁךָ / וַיֵּט יָדוֹ
עָלָיו וַיַּכֵּהוּ / וּמֵעוֹר רֹאשׁוֹ וְרַגְלָיו יַפְשִׁיטֵהוּ / עַד
אֲשֶׁר קָדְקָדוֹ וְעוֹרוֹ בְּדָם מְגֹלָל / וַיֹּאמֶר אֶל הַשֻּׁעָל: עַל
פִּיךָ יֵחָלֵק שָׁלָל / וַיַּעַן: אֲחַלְּקָה בְצֶדֶק וּבְמִישׁוֹר /
לַאֲדֹנִי הַמֶּלֶךְ הַשּׁוֹר / וְהַפָּר לַלְּבִיָּא וְלַגּוּר אֲשֶׁר
בּוֹ תָגֵל / דַּנְתִּי כִּי לְחֶלְקוֹ הָעֵגֶל / וַיֹּאמֶר הָאֲרִי: אֵין
נָבוֹן וְחָכָם כָּמוֹךָ בְּכָל מֶמְשַׁלְתִּי / מִן הַיּוֹם וָהָלְאָה
אֲשִׂימְךָ אֶל מִשְׁמַעְתִּי / כִּי בְעֵינַי מְאֹד נֶחְמַדְתָּ /
לְחַלֵּק שָׁלָל מִמִּי לָמַדְתָּ / וַיַּעַן הַשֻּׁעָל: כִּי רָאִיתִי
הַמִּצְנֶפֶת / אֲשֶׁר עַל רֹאשׁ הַזְּאֵב בְּדָם מְעֻלֶּפֶת / כַּאֲשֶׁר
נוֹקַשׁ בְּאִמְרֵי פִיהוּ / וְלֵב הוּתַל הִטָּהוּ / וַיֵּשְׁתְּ מִכּוֹס
חֲמַת אֲדֹנִי / עַל כֵּן לֹא נָתַתִּי לַחֲטֹא לְשׁוֹנִי.

וְהַמָּשָׁל אַל יְמַהֵר אִישׁ לְדַבֵּר / לִפְנֵי אָדָם
מִמֶּנּוּ יְגַבֵּר / אַךְ יָכִין מַעֲצָר לְרוּחוֹ / וְתַחְבֻּלּוֹת
יִקְנֶה וְאַל יְסַלֵּף אָרְחוֹ / אִם יֵשׁ בְּלֹא מִשְׁפָּט נִסְפֶּה /
וּלְשׁוֹן חֲכָמִים מַרְפֵּא / כִּי בִמְזִמּוֹת יַחְשֹׁב / וּדְבַר פִּיו
יְאַחֵר / חָכָם יִוָּסֵר עַל יְדֵי אַחֵר / וְאָמַר הֶחָכָם בְּסֵפֶר
הַמּוּסָרִים / לֵץ תַּכֶּה וּפֶתִי יַעֲרִים / וְהַיִּשְׁמְעֵאלִי אָמַר
בְּחָכְמָתוֹ / חֵלֶק אָדָם מֵאָזְנוֹ לְנַפְשׁוֹ וְחֶלְקוֹ מִלְּשׁוֹנוֹ
לְזוּלָתוֹ.
