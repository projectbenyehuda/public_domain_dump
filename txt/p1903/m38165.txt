


	
	משל סב: זְאֵבִים, עִזִּים וְכֶלֶב
	






נֹתֵן חַרְבּוֹ מִיָּדוֹ / אֶל שׂוֹנְאוֹ נֹתֵן יוֹם אֵידוֹ



הַזְּאֵבִים אֲשֶׁר כָּל אֶחָד בְּלִבּוֹ גָאָה וָרָם / נִלְחֲמוּ עִם הָעִזִּים וְהַכֶּלֶב שֹׁמְרָם / וַתִּגְבַּר יַד
הַכֶּלֶב וְהָעִזִּים / וַיְנַצְּחוּ הַשְּׁפָלִים אֶת
הָעַזִּים / וְיָדָם בַּזְּאֵבִים חֹתֶרֶת / וְהָיָה הֶחָסֹן לִנְעֹרֶת /
וַיֵּאָנְחוּ הַזְּאֵבִים וַיִּנְאָקוּ / וְהָעִזִּים עֲלֵיהֶם שָׁרְקוּ
וַיַּחַרְקוּ / אָז אָמְרוּ לָעִזִּים בְּעָקְבָה וּבְמִרְמָה / לֹא נֵצֵא
עוֹד עֲלֵיכֶם לַמִּלְחָמָה / אַךְ בָּזֹאת יֵאוֹת לָנוּ / לִשְׁמֹעַ
וְלִהְיוֹת כָּמוֹנוּ / לְבִלְתִּי הָעִיר עֲלֵיכֶם עוֹד קְרָבוֹת / אִם
תִּתְּנוּ לָנוּ תַעֲרֻבוֹת / וְעֵת תִּפָּרְדוּ וְעֵת תֹּאבוּ לְהָעֵז /
וְגָר זְאֵב עִם כֶּבֶשׂ אוֹ עֵז / וַהֲמָנְכֶם בָּאָרֶץ יְשֻׁלָּם /
וּבְרִית שָׁלוֹם בֵּינֵינוּ וּבֵינְכֶם עַד עוֹלָם / וַיֵּאוֹתוּ
הָעִזִּים וַיֹּאמְרוּ / מִי מִשֶּׁלָּנוּ אֲשֶׁר אֹתוֹ לָכֶם תִּבְחֲרוּ /
לִהְיוֹת עֵרָבוֹן אֶל יֶדְכֶם / כִּי נִכְסַפְנוּ לַחֲבוּרַתְכֶם /
וַיַּעֲנוּ הַזְּאֵבִים בְּתַרְמִית לֵב / לֹא נִשְׁאַל מִכֶּם רַק
הַכֶּלֶב / וַיִּקְרְאוּ הָעִזִּים אֶת הַכֶּלֶב וַיִּתְנוּהוּ לְעֵרָבוֹן
/ וַיְהִי הַדָּבָר לָהֶם לְמַפַּח נֶפֶשׁ וְלִדְאָבוֹן / כִּי הַכֶּלֶב
לְהִלָּחֵם אִתָּם יַצְמִיד / הַכֹּל הֹלֵךְ אַחַר הַמַּעֲמִיד /
בְּאָמְרוֹ: עִמְדוּ עֲמֹדוּ וְתִצְמַחְנָה יְשׁוּעוֹת / אַסְפֶּה עָלֵימוֹ
רָעוֹת / הִתְיַצְּבוּ לְהִלָּחֵם כִּדְמוּתִי / וּבַל יֹאמַר שָׁכֵן
חָלִיתִי / וּבְכָל עֵת אִמֵּץ אֶת רוּחָם / וְעַתָּה אָבַד תּוֹחַלְתָּם
וְנִצְחָם / כִּי הַזְּאֵבִים עָרְכוּ אִתָּם מִלְחָמָה / וַיַּכּוּם
וַיַּכְּתוּם עַד הָחָרְמָה.



וְהַמָּשָׁל אַל יִגְרַע אִישׁ מִמַּה שֶּׁיֵשׁ בְּיָדוֹ /
לְהוֹסִיף אֶל שׂוֹנְאוֹ וְחָפֵץ אֵידוֹ / כִּי זֶה יַשְׁרִישׁ יְגוֹן
לִבּוֹ / לָתֵת חֶרֶב בְּיָדוֹ לְהָרְגוֹ בוֹ / וְעֹשֵׂה אֵלֶּה מֻכֶּה
בְסַנְוֵרִים / זֶה יַשְׁפִּיל וְזֶה יָרִים / וְהַנֹּתֵן בְסִכְלוּתוֹ
נִשְׁבָּה אוֹ נִשְׁבָּר / וְהַמְקַבֵּל עָלָיו יָרוּץ
וְיִתְגַּבָּר.
