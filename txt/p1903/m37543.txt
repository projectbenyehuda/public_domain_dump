


	
	משל א: אַרְיֵה וְחַיּוֹת בְּהֵמוֹת
	




אוֹהֲבֵי עָשִׁיר רַבִּים בְּעֵת קֶרֶן הוֹדוֹ / וְיִתְהַפְּכוּ כִּי יָמוּךְ וּמָטָה יָדוֹ



אַרְיֵה זָקֵן הָיָה חֹלֶה / וּכְסָלָיו מָלְאוּ נִקְלֶה /
וַתִּקְצַר נַפְשׁוֹ בַּעֲמָלוֹ / וְלֹא נוֹדַע מֶה הָיָה לוֹ / וְאִם
יָמוּת אוֹ יִחְיֶה / וְלִרְאוֹת מַפֶּלֶת הָאַרְיֵה / בָּאוּ כָל בְּהֵמָה
וְחַיָּה / מִכָּל קַצְוֵי אֶרֶץ נְשִׁיָּה / יֵשׁ מֵאַהֲבָה לְבַקֵּר
אוֹתוֹ / וְיֵשׁ לִרְאוֹת תְּלָאָתוֹ / וְיֵשׁ לְקַבֵּל מָשְׁלוֹ הוֹלֵךְ /
וְיֵשׁ לָדַעַת מִי אַחֲרָיו מוֹלֵךְ / וַיֶּחֱלֶה עַד אֲשֶׁר לֹא נוֹדַע
בֶּאֱמֶת / אִם עוֹדֶנּוּ חַי אוֹ מֵת / וַיָבוֹא הַשּׁוֹר וַיְנַגְּחוֹ /
אוּלַי תַּם לָרִיק כֹּחוֹ / וְהָעֵגֶל רְמָסוֹ בָרַגְלַיִם / וְהַשֻּׁעָל
נְשָׁכוֹ בַשִּׁנַּיִם / בִּבְדַל אָזְנַיִם / וְרָחֵל בִּזְנָבָהּ
מְמַשֶּׁשֶׁת שְׂפָמוֹ / וַתֹּאמַר: מָתַי יָמוּת וְאָבַד שְׁמוֹ /
וְהַתַּרְנְגוֹל מְנַקֵּר עֵינָיו / וַיַּגְרֵס בֶּחָצָץ שִׁנָּיו /
וַיָּשָׁב רוּחוֹ אֵלָיו וַיִּתְבּוֹנֵן / כִּי שָׂמְחוּ בוֹ אֹיְבָיו
וַיְקוֹנֵן / הָהּ לַיּוֹם אֲשֶׁר תִּעֲבוּנִי מְתֵי סוֹדִי / וְכִי הָפַךְ
כֹּחַ יָדִי וְהוֹדִי לְאֵידִי / וַאֲשֶׁר הָיוּ לִי לַעֲבָדִים / עָלַי
מֹשְׁלִים וְרֹדִים / וְנֶהֶפְכוּ לִי לְאֹיְבִים / אֲשֶׁר מִקֶּדֶם הָיוּ
לִי לְאֹהֲבִים.

וְהַמָּשָׁל בְּאָדָם מָלֵא עֹשֶׁר וְכָבוֹד / וְכָל אֶחָד
מִשְּׁכֵנָיו אוֹתוֹ יַעֲבוֹד / וְאִם יַגִּיעַ יוֹם אֵידוֹ / כִּי יָמוּךְ
וּמָטָה יָדוֹ / מִנֶּגֶד נִגְעוֹ יַעֲמֹדוּ / וּמֵעָלָיו יִתְפָּרָדוּ /
וְצִדְקָתוֹ מִמֶּנּוּ יָסִירוּ / וְיִמְאֲסוּ אֶת אֲשֶׁר בָּחָרוּ.

וַאֲנִי בֶרֶכְיָה בִּרְאוֹתִי זֶה הַמֶּרִי / אֲשֶׁר
הָלְכוּ עִם נָדִיב בְּקֶרִי /



וָאֶשָּׂא מְשָׁלִי וָאֹמַר עַל זֶה
בְּשִׁירִי:



הָהּ כִּי אֲרָיִים לִפְנֵי עֵגֶל / רָבְצוּ וְרָאשִׁים נָשְׁקוּ רָגֶל

רַגְלֵי אֱמוּנִים רָמְסוּ בַטִּיט / וּבְנֵי שְׁקָרִים נָשְׂאוּ דָגֶל

וַיֵּרְדוּ רֹדִים וְהָעְוִילִים / עֹלִים וְאִישׁ אָוֶן מְאֹד יָגֵל

וֶאְמֶת לְשִׁפְחָה נִכְבְּשָׁה הַיּוֹם / נִצְּבָה כְבָר לִימִין כְּמוֹ שָׁגֶל

אֵין רַגְלַיִם לָאֱמֶת כִּי הַ/שֶּׁקֶר לְקָחָן בִּשְׂפַת רֹגֵל.
