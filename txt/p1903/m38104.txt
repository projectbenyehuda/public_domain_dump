


	
	משל מג: זְאֵב, כֶּלֶב וְעֵדֶר
	




לֹא יְשַׁקֵּר אִישׁ בֶּאֱמוּנָתוֹ / וְלֹא יחַלֵּל אֶת בְּרִיתוֹ

זְאֵב מִן הַכֶּלֶב יֵחַת / בְּפִיו חַלַּת לֶחֶם אַחַת /
נָשָׂא וַיָּבֹא אֶל הָעֵדֶר / שֶׁשָּׁם הַצֹּאן תּוֹךְ הַגָּדֵר /
הַכֶּלֶב הֵקִיץ מִשְּׁנָתוֹ / וְהַזְּאֵב לוֹ בְּעָרְמָתוֹ / אָמַר: דַּע
כִּי שֶׁלִּי שֶׁלְּךָ / קַח בִּרְכָתִי אֲשֶׁר הוּבָאת לְךָ / כִּי לֹא
עַל הַלֶּחֶם לְבַדּוֹ / אֶעֱשֶׂה לוֹ עֵזֶר כְּנֶגְדּוֹ / לְךָ כָּל
מַחֲמַדַּי תִּקָּחֵם / אַךְ עַתָּה תֹאכַל לֶחֶם / וַיַּעַן הַכֶּלֶב:
לָרִיק הוֹגַעְתָּ פְעָמֶיךָ / אִם תַּעְצְרֵנִי לֹא אֹכַל בְּלַחְמֶךָ /
בַּאֲשֶׁר הוּא לֶחֶם כְּזָבִים / וּבְעֵינַי דְּבָרֶיךָ לֹא טוֹבִים / לֹא
לְטוֹבָתִי תָּבִיא לִי לֶחֶם מֵאָהֳלֶיךָ / אַךְ לִסְתֹּם פִּי פֶּן
אַשְׁמִיעַ קוֹלִי עָלֶיךָ / כַּאֲשֶׁר תִּטְרֹף בַּצֹּאן / חָשַׁבְתָּ
לִמְצֹא עֵת רָצוֹן / וְחָלִילָה לִי לְכַבּוֹת נֵר הוֹדִי / כִּי אֲדֹנִי
נָטַשׁ הַצֹּאן עַל יָדִי / וְאֵיכָה אֲשַׁקֵּר בֶּאֱמוּנָתִי / אָהַבְתִּי
אֶת אֲדֹנִי וְאֶת בֵּיתִי / אֲשֶׁר מִנְעוּרַי גְּדֵלַנִי כְאָב /
וְאַתָּה בְּפִתּוּי דְּבָרֶיךָ תִכְאָב / וַיֶּחֱרַץ עָלָיו הַכֶּלֶב
לְשׁוֹנוֹ / וְהָרֹעֶה קָם וַיָּרָץ בַּחֲרוֹנוֹ / לִרְדֹּף הַזְּאֵב עִם
הַכֶּלֶב / אֲשֶׁר הָיָה שֹׂנְאוֹ בַלֵּב.

וְהַמָּשָׁל כָּל אִישׁ יִהְיֶה נִשְׁמָר בְּרוּחוֹ / פֶּן
יִבְגֹּד וִיסַלֵּף אָרְחוֹ / וּבְפִיו וּבְלִבּוֹ מִשְׁפָּט אֶחָד / לֹא
בִמְחִיר וְלֹא בְשֹׁחַד / אַל יְשַׁקֵּר בֶּאֱמוּנָתוֹ / וְאַל יוֹנֶה אֶת
עֲמִיתוֹ / לְהַגְדִּיל עָלָיו עָקֵב / לֹא מִן הַגֹּרֶן וְלֹא מִן
הַיָּקֶב / כִּי מִן הַכֶּלֶב יִמְשֹׁל מָשָׁל / כִּי בַשֹּׁחַד לֹא
נִכְשָׁל / וְאָמַר הֶחָכָם: הִדָּבֵק בֶּאֱמוּנָה וּשְׁמֹר מִצְוָה / כִּי
אֵין עֹשֶׁר גָּדוֹל כַּעֲזִיבַת הַתַּאֲוָה.
