


	
	משל מט: חֲזִירַת יַעַר, צְבִי וְחַיּוֹת
	




נַפְשׁוֹ מוֹרְאָה וְנִגְאֲלָה / לֹא יֵרָפֵא עַל נְקַלָּה

חֲזִירַת יַעַר בְּטִיט חוּצוֹת מִתְבּוֹסֶסֶת /
וּבְאַשְׁפָּה אֲשֶׁר תִּרְפֹּשׁ בּוֹ מִתְעַלֶּסֶת / וְהִנֵּה בַיַּעַר
קוֹל מִשְׁתֶּה וְשִׂמְחָה / וּמִינֵי מַטְעַמִּים וַאֲרוּחָה / כִּי צְבִי
נָשָׂא קְרוֹבָתוֹ / וַיִּתְאַסְּפוּ אֶל חֻפָּתוֹ / מִכָּל הוֹלֵךְ עַל
אַרְבַּע / וַיֹּאכַל כָּל אֶחָד וַיִּשְׂבַּע / לֶחֶם וְקֶמַח בָּשָׂר
וְדָגִים / וּבְשִׂמְחָתָם הֵם אוֹכְלִים וְחֹגְגִים / לְבַלּוֹת עִמָּהֶם
בַּנְעִימִים / לְמִגָּדוֹל וְעַד קָטָן מִשְׁתֶּה שִׁבְעַת יָמִים /
וְהַחֲזִירָה שָׁמְעָה שְׁאוֹן עֲלִיזוֹת / וְלֹא שָׁתָה לִבָּהּ גַּם
לָזֹאת / וַיַּבֵּט צְבִי אֶחָד בַּחֲזִירָה / הַנִּגְאֶלֶת וּבַטִּיט
פָּעֲרָה / פִּיהָ אֲשֶׁר אַשְׁפּוֹת חָתַר / וַתֹּאכַל וַתִּשְׂבַּע
וַתּוֹתַר / וַיֹּאמֶר הַצְּבִי לָהּ / בֹּאִי נָא בֵּית הַכַּלָּה /
אֲשֶׁר קוֹל הָמוֹן שָׁלֵו בָהּ / וְלֹא תֶחְסַר כֹּל בָּהּ / וְכָל אֲשֶׁר
שָׁאֲלוּ עֵינַיִךְ שָׁמָּה תִּקָּחֵם / לֹא בְּמִסְכֵּנֻת תֹּאכַל בָּהּ
לָחֶם / שָׁם כָּל עוֹף לְמִינֵהוּ / וּצְבִי עֶדְיוֹ לְגָאוֹן שָׂמָהוּ /
וְרוּחַ שִׂמְחָה בִּלְבָבוֹ עָצַם וְגָבַר / וּבַעַל כְּנָפַיִם יַגִּיד
דָּבָר / לֹא נֶעְדַּר אֹכֶל מִכָּל מִינִים / וְכֵלִים מִכֵּלִים שׁוֹנִים
/ וְהַצְּבִיָּה מַה נָּעֲמָה אֲשֶׁר לַצְּבִי מִתְיַפָּה / כִּי עַל כָּל
כָּבוֹד חֻפָּה / וַתַּעַן הַחֲזִירָה: אִם הָיָה שָׁם אֹכֶל מִכָּל מִין /
הֶאָכֹל אֲכַלְתֶּם סֻבִּין שְׁרוּיִן בְּחַמִּין / אוֹ מִטִּיט הַנִּרְמָס
בַּחוּצוֹת / כִּי כְמֹהֶם לֹא נִמְצָא מַעֲדָן בְּכָל הָאֲרָצוֹת / וְאִם
אֵלֶּה הַשְּׁנַיִם שָׁם נֶעְדְּרוּ / כָּל הַשְּׁאָר בְּעֵינַי לֹא
יָשְׁרוּ / כִּי מִיּוֹם הִוָּלְדִי כָהֶם לֹא טָעַמְתִּי / לֶחֶם
חֲמֻדּוֹת לֹא אָכַלְתִּי.

וְהַמָּשָׁל בַּאֲשֶׁר נָכְרִיָּה עֲבוֹדָתוֹ /
וּמִכִּתְמֵי חֲטָאָיו לֹא זָרָה תַאֲוָתוֹ / עַד כִּי נַפְשׁוֹ מוֹרְאָה
וְנִגְאָלָה / וִירַפֵּא פְּשָׁעָיו עַל נְקַלָּה / כִּי סְעִפָּיו בּוֹ
שָׂשוּ / לַעֲשׁוֹת מַעֲשִׂים אֲשֶׁר לֹא יֵעָשׂוּ / וְדֶרֶךְ עֵינָיו
לִבּוֹ זָנָה / גַּם כִּי יַזְקִין לֹא יָסוּר מִמֶּנָּה / וּבְעֵינֵי
שְׁכֵנָיו נִתְעָב וְנֶאֱלָח / כִּי מַיִם חַיִּים בְּרַגְלָיו יִדְלָח /
וְהָיָה כִּי יֹאמְרוּ לוֹ רֵעִים / הִנֵּה מַה טּוֹב וּמַה נָּעִים /
לְהִדָּבֵק בְּדֶרֶךְ הַיְשָׁרִים וְהַנְּקִיִּים / לִהְיוֹת חֶלְקָם
בַּחַיִּים / וּמָה רַב טוּב הַצָּפוּן לָהֶם / הִתְחַזֵּק וַעֲשֵׂה
כְמֹהֶם / אָז יֹאמַר: אֲשֶׁר נִסִּיתִי אָהַבְתִּי / כִּי שַׂמְתִּי כָזָב
מַחְסִי וּבַשֶּׁקֶר נִסְתַּרְתִּי / וְאַשְׁלִים חֶפְצִי וַאֲמַלֵּא
תַאֲוָה / לְמַעַן סְפוֹת הָרָוָה / וְטוֹב לִי מַרְאֵה עֵינַי /
וּלְהִתְעַדֵּן רַעְיוֹנַי / מִלְּצַפּוֹת בְּעוֹלָם אַחֵר לִהְיוֹת מֶלֶךְ
/ שָׁלוֹם יִהְיֶה לִי כִּי בִּשְׁרִרוּת לִבִּי אֵלֵךְ.
