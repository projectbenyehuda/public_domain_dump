


	
	לו משאלי יעשה לי אלי
	



 






לוּ מִשְׁאָלִי / יַעֲשֶׂה לִּי אֵלִי

וְיַמְשֵׁנִי / מִתְּהוֹם נַחֲלָתִי / אֶל נַחֲלָתִי!





מָתַי צֹרִי / יִשְׁלְחָה לִי צוּרִי

עַל יַד צִירִי / לַחֲבוֹשׁ אֶת צִירִי?

וּפְנֵי צָרִי / אֲשֶׁר סָבִיב צָרִי

יָשִׂים כִּבְלִי / כִּי הִכְבִּיד כַּבְלִי

כִּי שָׂמַנִי / מִבְּלִי תַאֲוָתִי / גַּם תַּאֲוָתִי.





שָׁמֵם מִבְצָר / בִּידֵי יְלִיד נִבְצָר

וּפְנֵי הַצָּר / נֶחְשְׁבוּ לִי כַּצָּר

עַד כִּי קָצַר / יָד וּמַצָּע קָצַר

וַיּוֹר כִּסְלִי / אַךְ עֲלֵי אֵל כִּסְלִי

לַהֲרִימֵנִי / מִבּוֹר דַּלּוּתִי / חִישׁ לְדַלּוֹתִי.





הִנֵּה אֵילַי / סָפוּ כִּי אֵילַי

יָבְשׁוּ. וְאֵילַי / שָׁמְמוּ כִּי אֵלַי

צָפוּ וְאֵילַי / יִקְרְאוּ נָא, אוּלַי

יָשִׁיב חֵילִי / אֶל הֲדוֹמִי וְחֵלִי

וְיַנִּיחֵנִי / לַעֲרוֹךְ זִמְרָתִי / עַל זִמְרָתִי.





בִּידֵי דוּמָה / יָרַדְתִּי דוּמָה

הַשְׁקוּ חֵמָה / לִי תַחַת חֵימָה

וַאֲנִי חוֹמָה / …רעית חוֹמָה

מִבּוֹר חֵילִי / אֶצְעֲקָה אֶל חֵילִי

לְנַהֲלֵנִי / וְלִגְאוֹל אַדְמָתִי.
