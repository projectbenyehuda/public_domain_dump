


	
	[מְנוּמָם בְּעֵת קוּמָם]
	




מְנוּמָם  /      בְּעֵת קוּמָם  /   לְהַמְלִיכָךְ בְּנֵי רְחוּמִים,

שְׁעֵה שַׁוְעָם   /   מְחֵה פִשְׁעָם  /   וְכָתְבֵם לְחַיִים וּלְרַחֲמִים

חֲמֹל, אָבִי,  /    כִּי תָבִיא   /    בְּמִשְׁפָּט עַל כָּל עֲלֻמִים

שְׁלַח פְּדוּתְךָ, /    מַלְכוּתְךָ    /  מַלְכוּת כָּל עֹלָמִים:       [תה' קמה, יג]

בְּיוֹם אֶקְרָא   /   תְּהִי, נוֹרָא,   /  עֵיְנָך עָלַי לְטוֹבָה

וּבַלָּאט נְחֵה   /  דַל, וּמְחֵה   /   עָוֹן וְחַטָּאת כְּתוּבָה

שָׂא פֶשַׁע   /    וְחִישׁ יֶשַׁע   /   בְּהַשְׁקֵט וְנַחַת וְשׁוּבָה

לְךָ אָשִׁיר   /   כִּי תָסִיר  /    מְשׁוּבַת בַּת שׁוֹבֵבָה

וְלֹא אַעֲלִים  /   כִּי חֲבָלִים    /  נָפְלוּ לִי בַּנְעִמִים:        [תה' טז, ו]

זֶה הַיּוֹם    /   אֲשֶׁר אָיֹם  /    יָצַר רֹאשׁ פְּעֻלָּתוֹ

וְצִוָּהוּ    /    וְטִהֲרָהוּ   /    וְלֹא שָׁמַר מִצְוָתוֹ

וּבוֹ נִשְׁפָּט    /  כִּי לְמִשְׁפָּט  /    נִבְרָא מִתְּחִלָּתוֹ

זֶה לְאֵידוֹ   /   וְזֶה לְצִידוֹ  /   וְזֶה לְחַיָּיו וְזֶה לְמוֹתוֹ

גְּדֹר פִּרְצִי   /    וּלְאוֹר הוֹצִיא  /  דִּינִי כְּשֶׁמֶשׁ בְּצֵאתוֹ

בַּשָּׁמַיִם   /    שִׁבְעָתַיִם  /    כְּאוֹר שִׁבְעַת הַיָּמִים:      [ישע' ל, כו]

אֲנִי עַבְדְּךָ   /    בְּרֹב חַסְדְּךָ   /   עֲשֶׂה עִמִּי לְטוֹבָה אוֹת

וְכִימֵי צֵאת   /   בְּאֵין יוֹצֵאת  /   הַרְאֵנִי נִפְלָאוֹת

וְשָׂא הַנֵּס, /     צוּר, וְכַנֵּס  /    הַפְּזוּרִים בְּכָל פֵּאוֹת

וְהַמְּאוֹרִים   /   הַמְּאִירִים   /  נֵס וְאוֹת לָהֶם יֵאוֹת

בְּעֹז מַלְכִּי  /    בִטְחו כִּי  /    מָלַך אֲדֹנָי צְבָאוֹת -

רְצֵה נִשְׁפִּי   /    וְנִדְבוֹת פִּי /     כְּמוֹ עוֹלוֹת וּשְׁלָמִים:      [תה' קיט, קח]
