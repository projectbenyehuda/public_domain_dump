


	
	שְמֹר דַּת חֲנֻכָּה
	



“מאורה לר' משה ן' עזרא”

סימן: משה הקטן


שְׁמֹר דַּת חֱנֻכָּה /   לְהָבִין וּלְהוֹרוֹת /  בְּהַעֲלֹתְךָ אֶת הַנֵּרוֹת


  


מִצְוָה לְהַדְלִיק   /  שְמֹנָה שְׁלֵמִים  /  וּבָרֵךְ רִאשׁוֹן



שָׁלֹשׁ פְּעָמִים



וּזְמַן לְמַעֵט   /   בִּשְׁאָר הַיָּמִים  /  פְּתִילוֹת וּשְׁמָנִים



אָסְרוּ חֲכָמִים



בַּשַּׁבָּת    /   לַחֲנֻכָּה יַתְמִים  /  וְאִם כָּבוּ



הַדֶּלֶק מְקַיְּמִים



כַּאֲשֶׁר פֵּרְשׁוּ   /  מְבָאֲרֵי סְתוּמִים  / אִמְרוֹת אֲדֹנָי



אֲמָרוֹת טְהֹרוֹת:




תה' יב, ז


שְעַת דְּלֵקַת   /   שְׁקִיעַת הַחַמָּה /  עַד כְּלוֹת רֶגֶל



מִשּׁוּק בְּאֵימָה



וְנֵר אִישׁ וּבֵיתוֹ  /   חֻקָּה שְׁלֵמָה  /  נֵר אֶחָד לְחֶבְרָה



מִצְוָה תְּמִימָה



אַךְ הַמְּהַדֵּר   /  לָאֵל בְּעָצְמָה   /  נֵר לְכָל אֶחָד



כַּדָּת שׂוּמָה



בְּדִבְרֵי נְבוֹנִים   /  לוֹמְדֵי אֲיֻמָּה  /  הַמִּשְׁפָּטִים



וְהַתֹּרוֹת:



         ויק' כו, טז




הַחֲזֵק בְּמַאֲמַר  /  בְּנֵי תוּשִׁיָה   /  הְיוֹתוֹ עַל



פֶּתַח הַחֲנִיָּה



בְּטֶפַח סָמוּךְ   /   לְדִירָה רְאוּיָה  /  וְשִעוּר כְּמָבוּי



וְסֻכָּה עֲשׂוּיָה



וְעֵת סַכָּנָה    /  בִּזְמַן שֶׁהָיָה   /  עַל שֻׁלְחָנוֹ



מִצְוָה רְצוּיָה



וּבַחַלּוֹן אִם  /  דָּר בַּעֲלִיָּה   /   לְפַרְסֵם אֶת הַנֵּס שֶׁהָיָה



וּלְהוֹדִיעַ כָּל הַקֹּרֹת:



 ברא' מב, כט




הַרְצוֹת מָעוֹת   /  אָסוּר לְאוֹרוֹ  /   לְבַל יְהִי מְבַזֶּה



אֱנוֹשׁ חֹק צוּרוֹ



לְהִשְׁתַּמֵּשׁ בּוֹ  /   הַצּוּר אֲסָרוֹ   /  וְעַל יָד אַל יְהִי



עֵת מַבְעִירוֹ



שֶׁמָּא יֹאמְרוּ /   לְצָרְכוֹ בְחָרוֹ /  וּבִפְנִים אַל יְהִי מֵכִין אוֹרוֹ



בְּהֵיטִיבוֹ אֶת הַנֵּרֹת

 שמ' ל, ז





קָבְעוּ הֱיוֹתוֹ  /   בִּשְׂמֹאל נְתוּנִים  / וְאַל יַדְלִיקוּ בָם



נֵרוֹת מְשֻׁנִּים



אַךְ זֶה מִזֶּה   /   חֻקִּים הֲגוּנִים, /   וּבְשַבָּת אִם



דּוֹלְקִים וּפוֹנִים



מְכַבִּים אוֹתָם  /  וּלְהַדְלִיק שׁוֹנִים  / וְחֵרֵשׁ וְשׁוֹטֶה וְקָטָן מְשֹֻנִּים



וְנָשִׁים בָּאוֹת אוֹתַהּ מְאִירוֹת:



   ישע' כז, יא

  


טְהוֹרִים פֵּרְשׁוּ  /  בְּדַת יוֹמַיִם   /   נֵר שֶׁיֵּשׁ לוֹ



פִּיּוֹת שְׁתַּיִם



מוֹצִיא חוֹבַת   /  אֲנָשִׁים שְׁנַיִם  /  פְּתִילוֹת נְתוּנוֹת



וְלוּ לְמָאתַיִם



בִּקְעָרָה יוֹצְאָה  /   בְּלִי עֱצַלְתַּיִם   /  וּבִלְבָד לְכַפּוֹת



כְּלִי בְּיָדַיִם



וְחָצֵר שֶׁיֵּשׁ לָהּ  /  פְּתָחִים שְׁנַיִם  /  צְרִיכָה שְׁתַּיִם לָעֵיַנִים



הֱיוֹתָם עַל פְּנֵי אֶרֶץ מְאִירוֹת:




ברא' א, טו

  


נָאֲמוּ בְיָמִים   /  אֵלֶּה פְלִיאָה   /  שֶׁלֹּא לִסְפֹּד



בָּם לִתְלָאָה



וְלִגְמֹר הַלֵּל   /   בָּם בִקְרִיאָה   /  וְלִזְכֹּר חֲנֻכָּה



בַהוֹדָאָה



ובְבִרְכַּת מָזוֹן  /   בְּסוֹד בָּאָה  /   וְזָהִיר בְּמִצְוֹת



וּדָת נוֹרָאָה



יִזְכֶּה לְבָנִים  /   בַּעֲלֵי הוֹרָאָה /   חֻקַּת עוֹלָם



תִּהְיֶה לְדֹרֹת:




ויק' ז, לו
