


	
	זָדוּ מְתֵי הָאַהֲבָה
	



ליצחק בן באן (ברון?) בן שת






זָדוּ מְתֵי הָאַהֲבָה, אוֹ שָׁגוּ, / אוֹ אִם לְמַסַּת כֹּאֲבִים יִלְעָגוּ,

כִּי הֶחֱלִיקוּ מִדְבְּרֵי פִימוֹ עֲדֵי / מֵהַאֲמִין בָּהֶם לְבָבוֹת פָּגוּ.

סִיג הָאֱלִיל כָּמוּס בְּלִבּוֹתָם וְאֶת / כֶּסֶף אֱמוּנִים מִלְּשֹׁנוֹת הָגוּ,

וַיִּמְאֲסוּ זֵכֶר לְעוֹלָם יַעֲמֹד / וּבְאַהֲבַת מַשְׂכִּית אֲדָמָה שָׁגוּ,

לִפְרֹץ בְּחוֹמַת הָאֱמוּנָה פָּרְצוּ / וּבְכָל גְּבוּלוֹת תֹּם עֲדֵי הֻסָּגוּ.

יֵלְכוּ מְזֵי רָעָב לְפַת מוּסָר, וְאֵיךְ / יֹאפוּ וְיֹאכְלוּ אֵת אֲשֶׁר לֹא עָגוּ?

אַבִּיט וְאֵין חֶסֶד, וְאֶשְׁתּוֹמֵם וְאֵין / אֵמוּן, הֲכִי נָעוּ וְכֻלָּם סָגוּ.

צַר לִי רְאוֹת גּוֹרֵי אֲרָיוֹת יַעֲנוּ / רַכּוֹת וְשׁוּעָלִים בְּעֹז יִשְׁאָגוּ.

הִתְנוּ אֲהָבִים לִי בְּלֵב וָלֵב, וּמֵי / רוֹשׁ עִם דְּבַשׁ אֹמֶר בְּפֶה מָזָגוּ.

אוֹאִיל לְהִוָּאֵל לְבִלְתִּי יֵדְעוּ / אִם סָלְדוּ צִדַּי וְאִם הָשְׁלָגוּ.

אוֹי אִם אֱהִי רוֹחֵשׁ וְאוֹי אִם אַחֲרִישׁ, / אוֹחִיל לְאַט מֵעַי – וְהֵם יִנְהָגוּ.

דַּי כִּי יְקָרִי סָכְלוּ וַיְכַחֲשׁוּ / אֹתוֹ וּבָרֹאֶה כְּעִוְרִים שָׁגוּ.

לוּ חָכְמוּ עַל צוּר אֲמִתִּי נִשְׁעֲנוּ / וּבְאַרְמְנוֹת צִדְקִי אֲשֶׁר פֻּסָּגוּ

אֵיכָה נְתִיבִים לַנְּדִיבִים עָזְבוּ / וּבְרֻכְסֵי הַנּוֹכְלִים נֹהָגוּ?

אִם יִיגְעוּ לִמְצֹא שְׁבִילִים – יִגְּעוּ / אֶל פַּעֲמֵי יִצְחָק וְלֹא יֵעָגוּ,

אוֹ יַעֲטוּ לִשְׁלַל חֲסָדִים – מַעֲנָיו / הֵן עָצְרוּ עִמּוֹ וְלָרֹב דָּגוּ,

אָז לָמְדוּ דַרְכֵי קְדוֹשִׁים גָּבְהוּ / מִכָּל בְּנֵי תֵבֵל וְלֹא הֻשָּׂגוּ.

יָרוּץ עֲשֹׂה חֶסֶד בְּיוֹם הִתְמַהְמְהוּ / רוּץ פַּעֲמֵי גַלְגַּל אֲבָל חָרָגוּ,

כַּאְשֶׁר לְיֶלֶד הַזְּמָן יִכְבַּד, וְאִם / לוֹ רָעֲשָׁה אֶרֶץ וְהָרִים מָגוּ.

עַל צַוְּארֵי יָמִים רְבִידִים יַעֲלוּ / מִיַּד עֲלִילוֹתָיו וְיִשְׂתָּרָגוּ.

תָּרֹן לְשׁוֹן אִלֵּם בְּמַהְלָלָיו וְכֵן / פִּסְחִים לְזִכְרוֹ כַּצְּבִי דָלָגוּ,

הֶחְיָה בְּטוּב יָדָיו קְבוּרִים בַּעֲפַר / חֶסֶר וְיַלְדֵי הָעֳנִי הֹרָגוּ,

יִדֹּב וּפָנָיו יִצְהֲלוּ, וַנֶּחֱזֶה / אַפֵּי עֲנָנִים בַּעֲרֹף יִדְאָגוּ.

נִמְהָר יְכוֹנֵן פִּיו וְרוֹגְנִים יִלְמְדוּ / לֶקַח וְצַחוֹת אַחֲרֵי נִלְעָגוּ.

תָּעוּ כְשִׁכּוֹרִים אֲנָשִׁים יָזְמוּ / לַעְלוֹת לְכֵס גָּדְלוֹ, וְגַם חָגָגוּ.

מִלָּיו עֲלֵי שָׁמִיר בְּעֵט שָׁמִיר, וְהֵם / חֹק עַל פְּנֵי מֵי הַשְּׁקָרִים חָגוּ.

שָׁרִים אֲשֶׁר מֵאָז לְשָׂרִים הִלְּלוּ / הֵם חִפְּאוּ כָזָב וְשָׁוְא אָרָגוּ.

גָּאוּ נְגִידֵי פִי בְמַהְלָלוֹ עֲדֵי / עַל רֹאשׁ בְּנוֹת עַיִשׁ לְאוֹת הֻצָּגוּ.

יִמָּאֲסוּ לִבִּי וְכִלְיוֹתַי כְּמוֹ / מַיִם לְנוּד יִצְחָק וְיִתְמוֹגָגוּ,

לֹא שָׁקְטוּ מִיּוֹם פְּקַדְתִּיהוּ וְלֹא / הִתְנַחֲמוּ בִלְתּוֹ וְלֹא הָבְלָגוּ.

לִבּוֹת אֲשֶׁר מֵאָז אֲחָדוֹת נִבְרְאוּ, / אֵיךְ יִחְיוּ אַחַר אֲשֶׁר נִפְלָגוּ?

אֶבְכֶּה וְנַפְשִׁי תַעֲרֹג אֵלָיו, וּמִן / אָז חוֹשְׁקִים עַל מֵי בְכִי עָרָגוּ.

יִתְעַדְּנוּ עֵינַי בְּמִכְתָּבוֹ וְיַ/עַלְצוּ קְרָבַי בִּי וְיִתְעַנָּגוּ.

הִנֵּה עֲצֵי דוֹדַי אֲשֶׁר הָשְׁקוּ בְמֵי / שִׂכְלוֹ וְעַל יִבְלֵי חֲסָדָיו שָׂגוּ,

יָרוּם בְּעֹז עֵטִי וּפִי, יִתְרוֹצְצוּ / לָשִׁיר תְּהִלּוֹתָיו כְּיוֹנִים הָגוּ.
