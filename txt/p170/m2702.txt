


	
	מלאכים ממליכים מעריצים
	




מַלְאָכִים / מַמְלִיכִים, / מַעֲרִיצִים / בְּכָל פֵּאוֹת,



מַקְדִּישִׁים / מַרְעִישִׁים, / מַרְגִּישִׁים / בְּרֹב תְּשׁוּאוֹת,



נוֹרָאִים, / וְאֵין נִרְאִים, / נִפְלָאִים / וְנִפְלָאוֹת –



אוֹמְרִים: קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!


  


שְׁמוֹ לָעַד / בְּרָב-רַעַד / בְּאֵימָה הֵן / יַעֲרִיצוּן,



שְׁאוֹן שְׂרָפִים / בְּרוֹם עָפִים, / כַּבְּרָקִים / יְרוֹצֵצוּן.



שׁוֹאֲלִים זֶה / רְשׁוּת מִזֶּה / לְהַקְדִּישׁ אֵל בְּמוֹרָאוֹת,



אוֹמְרִים: קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!


  


הֲמוֹן עֵדִים / מַרְעִידִים, / בְּרֶגֶשׁ יַ- / עֲרִיצוּ אֵל,



הֲמֻלָּתָם / קְדֻשָּׁתָם / בְּעֵת שִׁלּוּשׁ / קְדֻשַּׁת אֵל,



בְּקוֹל רַעַם / בְּרֹב נֹעַם / בְּקוֹל מַרְאֶה / לְהִתְרָאוֹת



אוֹמְרִים: קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!
