


	
	בַּמָּה אוֹדֶה
	



אל יוסף בן סהל






בַּמָּה אוֹדֶה לִגְבִיר דֵּעוֹת / עִמִּי כָל־עֵת לַעְשׂוֹת יַפְלֵא,

יִבְלוּ יָמִים כְּרָקָב אַךְ / חֶסֶד יִגְמֹל אֵינוֹ בוֹלֶה.

זָרַח בַּדּוֹר לָאִיר פָּנָיו / כַּחְנוֹת שֶׁמֶשׁ מַזָּל טָלֶה.

אִם נִכְלָאו עָבֵי חָכְמָה / גֶּשֶׁם שִׂכְלוֹ לֹא יִכָּלֶה.

מִמַּשְׂאַת שִׁיר נָעִים שָׁלַח / יָבֹא אֵלַי רוּחַ מָלֵא.

הִמְשָׁה מִיָּם רַעְיוֹן אַבְנֵי / מוּסָר, מֵהֶם טוּרִים מִלֵּא.

מִלִּים דָּמוּ לָאֵשׁ, אָכֵן / הָיוּ עַל חֹם לִבִּי אֶגְלֵי

טַל הַחֶסֶד לִכְבּוֹת רִשְׁפֵּי / לִבִּי, לַשִּׂיג תִּקְוָה יִכְלֶה.

הֵאִירוּ עֵין עִוֵּר וּלְאַט / אָזְנֵי חֵרֵשׁ הוֹדָם גּוֹלֶה.

מוֹפֵת נִמְצָא בַּדּוֹר לִדְרֹשׁ / מִפִּיו מִשְׁפָּט כִּי יִפָּלֵא,

וַיֹּאמֶר כִּי עָרֵב לִקְחִי / עִמּוֹ מָתְקוּ לוֹ נוֹזְלֵי

שִׁירִי, וַאְנִי אַגִּיד אֵמוּן / וּלְבָבִי עַל צִדְקוֹ יִגְלֶה,

כִּי יוֹם אֶצְמָא מֵחֹם כֶּסֶל / מִבּוֹר שִׂכְלוֹ מֵי בִין אֶדְלֶה,

וּבְעֵת יֶחְשַׁךְ אוֹר מַחְשָׁבִי / מֵאוֹר דֵּעוֹ נֵרוֹ אַעְלֶה.

יִתְאַפַּק עַל צֹרֶךְ יָמִים / לִבִּי, וּלְנוּד יוֹסֵף יֶאְלֶה,

וָאֶחְשֹׁב אֶת־רֹחַב תֵּבֵל / מֵאֵין פָּנָיו בָּתֵי כֶלֶא.

דּוֹדִי, חָשְׁבוּ יָמִים עָלַי / רָעָה לָשׂוּם אוֹתִי גוֹלֶה,

וּבְפָנַי הֵעֵזּוּ מֶצַח / וּבְכָל רֶגַע בִּי יִתְפַּלֵא,

תָּקַע אוֹתִי יָתֵד, אַךְ כָּל / כֵּלֵי צוּקָיו עָלַי תּוֹלֶה,

כַּמָּה אָרוּץ לַעְשׂוֹת מִצְוַת / פִּיךָ וֶהְיוֹת כִּמְעַט שׁוֹלֶה,

וַיְמָאֵן לֵב רָצוּף צִירִים / גַּם אֵשׁ פֵּרוּד אוֹתוֹ קוֹלֶה,

אֵיכָה אוּכַל עוֹד הִתְכַּבֵּד / וּזְמָן הָרַע אוֹתִי מַקְלֶה?

וּבְתֵת אֹדֶם לִבְדֹלָחַי / יָעִיד בּוֹ כִּי הוּא מִנִּבְלֵי

חֶרֶשׂ, עַד כִּי שָׁפַל דֶּגֶל / הַשִּׁיר וַיְהִי עֻזּוֹ נִקְלֶה,

וּנְגִידֵי שִׁיר הָעֵת אֹמַר / וָאֶשָּׂא אֶל אָזְנָם מִשְׁלֵי

שֵׂכֶל, יֹאמְרוּ כִּי הוּא מִדְבַּר / עִמְקֵי שָׂפָה אוֹ מַהְתַלֵּי

רֵקִים, עַל כֵּן הִנְנִי כַנֵּר / יָאִיר, אָכֵן גּוּפוֹ יִכְלֶה.

אוֹחִיל לָאֵל אוּלַי יָרִים / יַלְדֵי שֵׂכֶל יָרְדוּ פֶלֶא.
