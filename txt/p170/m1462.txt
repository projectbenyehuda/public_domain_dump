


	
	הא לך מכתב מאח נכאב
	




הֵא־לָךְ מִכְתָּב / מֵאָח נִכְאָב / חֶטְאוֹ פָרַשׂ / פַּח לִפְעָמָיו



וּזְמָן הִצְבִּיא / צָרוֹת עָלָיו / עַד כִּי־שָׁלַח / יָד בִּשְׁלֹמָיו



מֵי־רֹאשׁ בִּלְבַד / נָתַן מַשְׁקָיו / שֹׁרֶשׁ רֹתֶם / רַק מַטְעַמָּיו



אַחְלָיו אִלּוּ / בִימֵי נֹעַר / יֹאבַד וִיהִי / מֵת בַּעְלוּמָיו



אָז יָנוּחַ / לוֹ כִּי אַחַר / כָּבוֹד יִהְיֶה / בָא אֶל־עַמָּיו



הֶן מַאְוַיָּו / לִכְלוֹת חַיָּיו / מָוֶת יִבְחַר / לוֹ מֵעְצָמָיו



כִּי לֹא־יִשְׁכַּב / לַיְלָה לִבּוֹ / גַּם מַכְאֹבִים / הֵם כָּל־יָמָיו



פָּצוּ עָלָיו / פִּיהֶם צָרָיו / גַּם כָּל־הַיּוֹם / שִׂפְתֵי קָמָיו



עַד כִּי־אָחִיו / הִקְשָׁה לִבּוֹ / וּבְלִי־פֶשַׁע / שִׁחֵת רַחְמָיו



לוּ דִבַּרְתִּי / מֵאָז כָּזֹאת / אֶל־הַסֶּלַע / נָתַן מֵימָיו.
