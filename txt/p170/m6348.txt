


	
	זְּמָן יִקְרָא עֵדָיו
	



זְּמָן יִקְרָא עֵדָיו / ר' משה אבן-עזרא


זְּמָן יִקְרָא עֵדָיו / רְאוֹת מַעְשֵׂי יָדָיו / אֲשֶׁר הֵמָּה עֹשִׂים



הֲכִי כִסָּה פָנָיו / בְּתַשְׁבֵּץ נִצָּנָיו / וּבִכְתֹנֶת פַּסִּים



וּמֵאֵין לֵב יָגִיל / כְּבַת תַּעְדֶּה עָגִיל / וְתִתְיַף בִּשְׁבִיסִים



וְרוּח גַּן יִשְׁאַף / וְיִשָּׂא אֶל כָּל אַף / כְּפָרִים וַהְדַסִּים



5    וּמַחְמַדֵּי נִרְדָּם / יְעִירוּן כָּל נִרְדָּם / וְלֵב חשְׁקִים שֹׁסִים



וּבֵינוֹת עָלֵיהֶם / יְעוּפוּן סִיסֵיהֶם / וְעַל אֶרֶץ טָשִׂים



וְלַחְשׂף סוֹדֵיהֶם / יְחוּשׁוּן סוֹדֵיהֶם / וְכֹסֵינוּ כֹסִים



לְאִישִׁים צִלְלֵיהֶם / וְרָאשֵׁי עַנְפֵיהֶם / לְתֹרִים וּלְסִיסִים



וּמֵי שֹׁקֶט צַחוּ / הֲכִי לֹא נִדְלָחוּ / וְאֵינָם נִרְפָּשִׂים



10   בְּלִי פֶה מַטִּיפִים / וְהֵם כִּפְצִירָה פִּים / וְשִׁרְיוֹן קַשְׂקֵשִּׂים



וְרִשְׁפֵּי מִזְרָקִים / עֲלֵי כַפֵּי מַשְׁקִים / כְּאֵשׁ מֵאֵין מֵשִׂים



לְהָקֵל מַעְצֵבִים / יְרֻצוּן כֶּעָבִים / וְכִמְרֻצַת סוּסִים



לְזֹאת אֶשְׁאַל רֵעַי / לְהָבִין אֶת רֵעַי / וּבוֹ נִהְיֶה שָׂשִׂים



הֲכִי רִגְעֵי נוֹחַ / יְחוּשׁוּן לִבְרוֹחַ / וְכַצֵּל הֵם נָסִים.
