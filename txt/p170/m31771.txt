


	
	הֶרֶף נְדֹד
	



אל שלמה בן בתאת






הֶרֶף נְדֹד, כִּי אֵין לְאֵל יָדֵנוּ, / חֶרֶף וְנִשְׂבַּע מִפְּנֵי דוֹדֵנוּ,

דַּי כִּי הֲדַפְתָּנוּ בְאֵיבָה מִגְּבוּל / בָּתֵי יְלָדֵינוּ וּמוֹלִידֵינוּ,

דַּי כִּי הֱקִיאוּנוּ מְעוֹנֵיהֶם וְעִם / חַיְתוֹ יְעָרִים נֶחֱלַק שְׂדֵנוּ,

דַּי כִּי הֲסִבֹּתָ אֲחֹרַנִית לְבַב / אַחִים, בְּלֹא הוֹן מָכְרוּ דוֹדֵינוּ.

קָצַף זְמָן כִּמְעַט וְהֵמָּה עָזְרוּ / עַד מִיְלָדָיו מִהֲרוּ עָדֵינוּ.

סָכְלוּ, וְאִלּוּ שָׂכְלוּ הִתְיַמְּרוּ / בָנוּ יְמוֹת עוֹלָם וּבִכְבוֹדֵנוּ.

נֹאמַר: ״[אֲפִיקֵי יָם], לְאַט, הִתְאַפְּקוּ!״ / כִּי גָבְרוּ מֵימָם לְהַכְחִידֵנוּ,

וַיַּעֲ[נוּ אֵ]דֵי קְרָבִים עוֹלְלוּ / כִּי יַעֲלֶה נֵצַח לְהוֹרִידֵנוּ,

לוּלֵי הֲמוֹנֵיהֶם אֲשֶׁר יִתְהַלְכוּ / אָמְרוּ יְקוֹדֵינוּ לְהַשְׁמִידֵנוּ.

אֵיךְ יַעֲמֹד הַלֵּב וּבוֹ יִתְחַבְּרוּ / גֵּאוּת גְּשָׁמֵינוּ וְלַפִּידֵינוּ,

שַׁעְרֵי יְגוֹנִים נִפְתְּחוּ לָנוּ כְּמוֹ / שַׁעְרֵי שְׂשׂוֹנִים נִסְגְּרוּ בַּעְדֵנוּ.

כַּנְפוֹת אֲדָמָה עַד מְאֹד צָרוּ עֲדֵי / בִּכְנַף שְׁלֹמֹה אָחֲזָה יָדֵנוּ,

אִישׁ מַעֲלוֹתָיו יִקְרְאוּ אֶל כּוֹכְבֵי / אוֹר בַּעֲלוֹת: שִׁפְלוּ לְמִצְעָדֵינוּ,

כַּאְשֶׁר לְנִבְלֵי רוּם יְשִׂיחוּן: שַׁאֲבוּ / שֶׁפַע יְגוֹנֵנוּ וּמֵי אֵידֵנוּ,

וּבְעַד בְּשָׂמִים מַעֲלָלָיו יֹאמְרוּ: / אֵלֶּה פְעָלֵינוּ וּמַעְבָּדֵינוּ,

וּלְלוֹחֲמֵי שֹׁרֶשׁ רְתָמִים: מִדְּבַר / פִּינוּ רְדוּ נֹפֶת וּמִשֹּׁדֵנוּ,

וּלְשׁוֹאֲלֵי פַת בִּין וְיֵין הַשְׂכֵּל: אֲכֹל / וּרְווֹת בְּיֵינֵנוּ וּבִמְגָדֵינוּ,

וּלְכָל דְּוֵי לֵבָב: לְכוּ הִתְעַנְּגוּ / בִּצְבִי כְפָרֵינוּ וּבִנְרָדֵינוּ,

וּלְנֹחֲלֵי יַחַשׂ וְחָכְמָה: מַהֲרוּ / לַכְתִּיר בְּיַחְשֵׂנוּ וּבַחְסָדֵינוּ.

נִמְצָא זְרוֹעֵנוּ עֲדֵי עַל בָּמֳתֵי / אַף הַזְּמָן צִוָּה לְהַצְעִידֵנוּ,

וַיִּהְיוּ דוֹדָיו בְּיָקְשֵׁנוּ בְּפַח / גַּנּוֹת אֲדָנֵינוּ וְעַמּוּדֵינוּ,

וּבְיוֹם בְּמַאְסַר הַנְּדֹד שַׁחְנוּ הֲלֹא / הֵם הַמְּקִימֵינוּ וְהַפּוֹדֵינוּ.

נַכְעִיס בְּנֵי יוֹם בּוֹ, וּמִי זֶה יַעֲמֹד / מֵהֶם לְפָנֵינוּ וְיוֹעִידֵנוּ?

מֶה עָרְבוּ לָנוּ יְמֵי הִתְעָרְבוּ / דּוֹדִים וּמָתְקוּ מִדְבְּרֵי סוֹדֵנוּ,

עַד דְָּהֲרוּ סוּסֵי נְדֹד וַיִּקְרְבוּ / רִכְבֵי זְדוֹנֵנוּ לְהַפְרִידֵנוּ.

נִסְעָה וְנֵלֵכָה, וּמִפְרָדוֹ כְאֵשׁ / יֵקַד בְּלִבֵּנוּ וּבִכְבֵדֵנוּ,

נֵדְעָה מְקוֹם פָּנָיו פְּנֵי שַׁחַק וְהֵן / עַישׁ וּבָנֶיהָ נְשַׁו עֵדֵינוּ,

לָעַד נְתַנֶּה הוֹד פְּעָלָיו עַד בְּלִי / שֶׁמֶשׁ בְּלִבֵּנוּ וּבִמְאֹדֵנוּ,

נָשִׂים נְעִים זִכְרוֹ בְּכָל מַסָּע וְכָל / מַחְנֶה בְּאָרְחֵנוּ מְקוֹם עָמְדֵנוּ,

עַד יֶחֱנַן הָאֵל וְאוּלַי יַחֲזֹר / גַּלְגַּל וְעוֹד יִסֹּב לְהַצְמִידֵנוּ.
