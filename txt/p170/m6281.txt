


	
	[עֶבֶד לֵילוֹ מִדַּד וְעַרְבוֹ]
	



סליחה


עֶבֶד לֵילוֹ   /    מִדַּד וְעַרְבוֹ

וּבְבֵית מְחוֹלְלוֹ  /  גִּלָּה רִיבוֹ

יוֹמָם חִילוֹ  /   יִגְדַּל חוֹבוֹ

וּבַלַּיְלָה לֹא  / שָׁכַב לִבּוֹ:        [קה' ב' כג]

מַחְמַד שִׁירִים  /    וּצְבִי מִלּוֹת

לָקְחוּ בְחִירִים  /     בְּאִישׁוֹן אֲפֵלוֹת

וְדָפְקוּ שְׁעָרִים   /    בְּבָתֵּי תְפִלּוֹת

עֵת כָּל בַּיִת /  סֻגַּר מִבּוֹא:        [ישע' כד, י]

שַׁחַר עָדָה   /   בֵּית אֵל גּוֹעֶה

וְכַיּוֹלֵדָה   /   יְשָׁנֵּס וְיִפְצֶה

יִלְבַּשׁ חֲרָדָה  /  כִּנְשׁוּךְ אֶפְעֶה

יִמְצָא מְרוֹרַת / פְּתָנִים בְּקִרְבּוֹ:      [איו' כ, יד]

הוֹמֶה בְחִילוֹ  /   לְיוֹם הַמָּוֶת

כִּי יַבְהִילוֹ   /  לְגֵיא צַלְמָוֶת

וּמֵהֵיכָלוֹ  /    לַחֲצַר מָוֶת

יֵצַא לְמִדְבָּר /  אָדָם אֵין בּוֹ:       [איו' לח, כ]

סְלַח נָא וַעֲבֹר  /   עַל חֵטְא נְעוּרָיו:

וְהַתֵּר לִשְׁבֹּר   /  מוֹטוֹת אֲסוּרָיו

וּמֵחֲלַח וְחָבוֹר   /    תְּשַׁלַּח אֲסִירָיו

וּבָנָיו מִבּוֹר /  מַיִם אֵין בּו:        [זכר' ט, יא]
