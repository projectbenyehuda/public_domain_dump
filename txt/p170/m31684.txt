


	
	לֹא פָצְעוּ יָמִים
	



לאבא יחיי – יהודה – בן אלרב, בקחתו אשה חדשה






לֹא פָצְעוּ יָמִים עֲדֵי חָבָשׁוּ / וַיְקוֹמְמוּ בִנְיָן וְכֵן נָתָשׁוּ,

לֹא יָזְמוּ שַׂק הַיְגוֹנִים לֶאֱרֹג / עַד כִּי תְמוּרָתוֹ פְּאֵר חָבָשׁוּ.

הִתְנַחֲמוּ כִּי יָזְמוּ רָעָה, וְהֵן / יִכָּלְמוּ יוֹם יוֹם וְיִתְבּוֹשָׁשׁוּ.

סָמְכוּ בְתַנְחוּמִים לְבָבוֹת יָצְאוּ / עַל שֹׁד כְּצֵל עֹבֵר וְכִמְעַט מָשׁוּ.

שָׁבוּ לְחַזֵּק מֵיתְרֵי אֹהֶל אֲשֶׁר / חֶבְלֵי עֲלִיצוּתוֹ בְּאַף נָטָשׁוּ.

כּוֹכְבֵי שְׂמָחוֹת זָרְחוּ פֶתַע בְּבֵית / אֵבֶל וְאָפְלֵי הַנְּהִי גֹרָשׁוּ.

נֻדּוּ אֲשֶׁר נָדוּ וְלִנְהוֹת נוֹעָדוּ / כַּאְשֶׁר בְּנוֹת הַשִּׁיר לְשׁוֹרֵר חָשׁוּ.

הֶרְאָה זְמָן עָרִים אֲשֶׁר לֹא שִׂבְּרוּ / וַיֶּחֱזוּ רֵעִים אֲשֶׁר דָּרָשׁוּ,

נָשְׂאוּ בְרֹן קוֹלָם אֲשֶׁר הִתְחָרְשׁוּ / תָּמִיד בְּמִקְרֵיהֶם וְהִתְלֶחָשׁוּ.

פָּשְׁטוּ בְנֵי שֵׂכֶל לְבוּשׁ יָגוֹן כְּמוֹ / תַחְתָּיו מְעִילֵי צָהֳלָה לָבָשׁוּ,

וַיֵּדְעוּ כִי אֵל לְרַגְלָךְ בֵּרֲכָם / כִּי בָחֲנוּ שֶׁבַע וְכִי נִחָשׁוּ.

אֹרוּ פְאוֹת אֹהֶל אֲשֶׁר נִתַּק לְךָ, / כִּי בוֹ בְנוֹת עַיִשׁ לְהָאִיר עָשׁוּ.

הִנֵּה מְסִבָּתָךְ בְּשַׁחַק נוֹעֲדוּ / תּוֹכָהּ מְאוֹרֵי רוֹם וְהִתְקַדָּשׁוּ.

שֶׁמֶשׁ וְסַהַר אַתְּ וְרַעְיָתָךְ, וְאִם / צֶדֶק וְשָׁלוֹם מֵאֱנוֹשׁ נִפְגָּשׁוּ.

ומָה דְלֵה מֵי גִילְךָ מִתּוֹךְ בְּאֵר / יֶשַׁע, אֲשֶׁר דַּלּוּ וְלֹא יָבָשׁוּ,

וֶאְרֵה לְךָ מִגַּן בְּחוּרִים מִגְדְּנוֹת / חֵן, אֶחֱרוּ כִמְעַט וְגַם בּוֹשָׁשׁוּ.

עֶדְנָתְךָ שָׁבָה וְהֵשִׁיבָה יְמֵי / נֹעַר אֲשֶׁר בָּלוּ וְהֵן חֻדָּשׁוּ.

הִתְבַּשְּׂרָה, כִּי הַזְּמָן חֹמֶר בְּיַד / חֶפְצְךָ וּבָנָיו מִמְּךָ רָגָשׁוּ.

אֶל מַעֲלָתָךְ שָׁאֲפוּ רַבִּים וְאָ/כֵן זָרְעוּ רוּחַ וְסוּפָה דָשׁוּ.

הוֹדוּ לְךָ עָבִים בְּנִדְבַת יָדְךָ / אַף כָּל אֲשֶׁר קָנוּ וְגַם יָרָשׁוּ.

יִזָּכְרוּ לָעַד פְּעָלֶיךָ בְּפִי / תֵבֵל וּבָנֶיהָ וְלֹא יִנָּשׁוּ.

כָּלוּ מְשַׂנְאֶיךָ בְּאֵשׁ קִנְאָה בְּמוֹ / עָשָׁן וְאֵינֵימוֹ בְּעֵת בֻּקָּשׁוּ.

קַח לָךְ תְּשׁוּרַת דוֹד, פְּצָעָיו עָצְמוּ / חִנָּם וְלֹא זֹרוּ וְלֹא חֻבָּשׁוּ,

יָמִים הֱנִיעוּהוּ בְלִי פֶשַׁע וְהוּא / בָם לֹא יְהִי נוֹשֶׁה וּבוֹ לֹא נָשׁוּ

אוֹן הַתְּבוּנָה מָצְאוּ יָדָיו עֲדֵי / עָשְׁרוֹ, וְאִם מֵהוֹן אֲדָמָה רָשׁוּ.

אֵין לוֹ לְבַד לָשׁוֹן וְעֵט, יִתְחַבְּרוּ / לִסְפֹּר חֲסָדֶיךָ אֲשֶׁר קָדָשׁוּ.

זֶרַע פְּרֻדוֹת מַהֲלָלָיו אַט בְּךָ, / כִּי יַעֲשׂוּ פֶרִי וְלֹא יֶעְבָשׁוּ.

הֵרוֹם וְעַל כָּל־בָּמֳתֵי צָרִים דְּרֹךְ / עַד יִכְרְעוּ אֶצְלָךְ וְיִתְכֶּחָשׁוּ,

כִּי הֵם יְקוֹד אֵיבָה בְלִבָּם יִצְפְּנוּ / וַיַּעֲנוּ רַכּוֹת וְיִתְרוֹשָׁשׁוּ.

כֵּן יִהְיוּ תָמִיד בְּעוֹד חוּג יַהֲלֹךְ, / נִרְאֶה בְשִׂמְחָתְךָ וְהֵם יֵבוֹשׁוּ.
