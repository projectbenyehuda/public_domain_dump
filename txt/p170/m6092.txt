


	
	[אֶל־הַשְּׁחָקִים נַפְשְׁךָ תַרְקִיעַ]
	




אֶל־הַשְּׁחָקִים נַפְשְׁךָ תַרְקִיעַ / אוֹ מֵהֲרָרִים לַיְקָר תּוֹפִיעַ

אוֹ לֹא־תְרַפֶּה מִזְחֲךָ עַד יִֽהְיוּ / גַּבֵּי בְנוֹת־עַיִשׁ לְךָ יָצִיעַ

תִּיגַע עֲדֵי תֹאמַר לְשֶׁמֶש דֹּם וְגַם / בֵּין יָם לְמִזְרָח תַּחֲשֹׁב בָּקִיעַ

וּבְעַד אֲדָמָה תֹֽאמְרָה כִּי־הִיא כְמוֹ / כִּכָּר וְהַיָּם תַּחֲשֹׁב גָּבִיעַ

דַּי לָךְ הֱיוֹתְךָ אוֹת בְּאַלְפֵי הַזְּמָן / עַד־הֵם כְּמוֹ־אֶרֶץ וְאַתְּ רָקִיעַ

הַמְעַט הֱיוֹת שִׂכְלָךְ לְכֹל יָבֹא כְּמוֹ / מוֹפֵת וְאָב אֹתוֹ לְבֵן יוֹדִיעַ

אֵיךְ עַל־זְמָן תִּלּוֹן וְהוּא יַצְדִּיק מְתֵי־ / שֶׁקֶר וְדֹרְשֵׁי הָאֱמֶת יַרְשִׁיעַ

יָרוּם לְבָבוֹ עַל־יְלִיד שֵׂכֶל וְאֶת־ / גָּבְהוֹ לְפֶתִי יִֽהְיֶה מַכְנִיעַ

שָׁווֹ בְעֵינַי יוֹם יְהִי בוֹ מַאֲכִיל / נֹפֶת וְהָעֵת מַמְּרוֹר יַשְׂבִּיעַ

אוֹ יוֹם כְּאֹהֵב יַעֲנֶה רַכּוֹת וְיוֹם / עָלַי כְּאֹיֵב יִֽהְיֶה מֵרִיעַ

כִּי זֶה כְמוֹ־זֶה כַּחֲלוֹם יֻדַּד וְגַם / כַּקַּשׁ אֲשֶׁר רוּחַ יְהִי מַסִּיעַ

יִדְמוּ יְלָדָיו לוֹ כְּמוֹ נָחָשׁ אֲשֶׁר / צֶפַע יְהִי מוֹלִיד וְלֹא צָפִיעַ

כֻּלָּם זְבוּב מָוֶת אֲשֶׁר יַבְאִישׁ הֲמוֹן / שֶׁמֶן מְרֻקָּח עַד־אֲשֶׁר יַבִּיעַ

מִי יַאֲמִין בָּם יַאֲמִין כִּי הַזְּמָן / עָפְרָה לְאַרְיֵה יִֽהְיֶה מַרְבִּיעַ

אוֹ כִי אֱנוֹשׁ יִסַּק אֱלֵי־שַׁחַק וְכִי / עַיִשׁ וּבָנֶיהָ שְׁאוֹל תַּצִּיעַ

אוֹ כִי מְאוֹר־שֶׁמֶשׁ בְּמִשְׁקָל לוּ יְהִי / בָא אוֹר בְּנוֹ יַעְקֹב יְהִי מַכְרִיעַ

אֶרֶז יְנִיקוֹתָיו אֱלֵי־קַצְוֵי שְׁאוֹל / שִׁלַּח וְאֶת־רֹאשׁוֹ לְעָב הִגִּיעַ

דֹּבֵר נְגִידִים יוֹם לְשׁוֹן כָּל־בַּעֲלֵי / אֹמֶר בְּחַבְלֵי הַדֳּמִי הִשְׁקִיעַ

הַשַּׂר אֲשֶׁר עִוְרִים מְאוֹרוֹ יֶחֱזוּ / נֵצַח וְאָזְנֵי חֵֽרְשִׁים יַשְׁמִיעַ

לוֹ רַעֲיוֹן יָעוּף כְּבֶן־רֶשֶׁף וְעַל / כַּנְפֵי כְרוּבִים יַחֲמֹד לַרְגִּיעַ

מִקְדַּשׁ חֲסָדִים כּוֹנֲנוּ יָדָיו עֲלֵי / אַדְנֵי תְהִלּוֹת עַל־תְּהוֹם הִטְבִּיעַ

וּמְעוֹן צְדָקוֹת הֶעֱמִיד אֵילָיו וּמִ־ / בָּתֵּי מְגוּרוֹ הָאֱלִיל הִקְצִיעַ

יִסַּר זְמָן בּוֹגֵד בְּשׁוֹט לָשׁוֹן וְאֶת־ / בָּנָיו בְּקוֹץ הַמַּאֲמָר הוֹדִיעַ

עַד־שָׁב בְּיַד חֶפְצוֹ כְּמוֹ־חֹמֶר בְּיַד / יוֹצֵר וּבוֹ יֵיטִיב וּבוֹ יָרִיעַ

לוּ טַל יְמִין חַסְדּוֹ בְּסֶלַע יַחֲנֶה / רֶגַע אֲזַי הוֹלִיד וְגַם־הִזְרִיעַ

דִּבֶּר עֲלֵי לִבִּי בְּיוֹם אַפֵּי זְמָן / שֻׁנּוּ וְרֹאשׁוֹ אַחֲרַי הֵנִיעַ

וַיִּפְגְּעוּ־בִי מַלְאֲכֵי דֹדָיו בְּיוֹם / אַבִּיט לְכָל־עֵבֶר וְאֵין מַפְגִּיעַ

בָּרָק אֲשֶׁר עָבַר בְּלַיִל מִפְּאַת / מִזְרָח וְשַׂק הַמַּאֲפֵל הִבְקִיעַ

שָׂא אֶת־שְׁלוֹמוֹתַי לְיֹאשִׁיָּה וּמֵ־ / אֵין עֵט צְפוּנֵי אַחֲוָה הוֹדִיעַ

וּלְאַט שְׁאַל־נָא אֶת־שְׁלוֹם נַפְשׁוֹ וְכֵן / אֹתוֹ בְּחֵי הָאַהֲבָה תַּשְׁבִּיעַ

לָחִישׁ צְרִי מִכְתָּב לְלֵב כֹּאֵב וְאוּ־ / לַי בּוֹ אֲסִיר הַמַּחֲלָה יוֹשִׁיעַ.
