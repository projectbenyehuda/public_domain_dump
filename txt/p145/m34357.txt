


	
	ליום שמחת תורה.
	








בְּשִׁגָּיוֹן / תָּגֵל צִיּוֹן,

לְאֵל עֶלְיוֹן / תִּשָּׂא זִמְרָה.

וְעַם־חֶמְדָּה / יִתֵּן תּוֹדָה,

וּבִיהוּדָה / תּוֹפַע אוֹרָה.

כְּדוֹר רִאשׁוֹן / יִשְׂבַּע שָׂשׂוֹן,

וְכָל־לָשׁוֹן / תָּשִׁיר שִׁירָה.

וְשִׁיר מִזְמוֹר / נֹאמַר אָמוֹר

בְּיוֹם נִגְמוֹר / אֶת־הַתּוֹרָה.





וְגִיל חָתָן / יִהְיֶה אֵיתָן,

וְיִתְעדַּן / בִּקְרוֹא מִקְרָא.

הֲלֹא יִירַשׁ / גִּיל לֹא פֹרַשׁ

אֲשֶׁר אֵרַשׂ / יָפָה בָרָה.

וּבִרְכָתוֹ / תִּשׁכֹּן אִתּוֹ,

וּמָנָתוֹ / מִלִּין יִקְרָא.

תְּנוּ שָׂכָר / שָׂשׂוֹן וִיקָר

לְאִישׁ יָקָר / הִשְׁלִים תּוֹרָה.





וְאַתָּה רֹאשׁ, / מַגִּיד מֵרֹאשׁ,



אֲשֶׁר תִּפְרֹשׁ / עוֹלָם נִבְרָא,

לְךָ וּלְךָ / שָׂשׂוֹן וְשִׂמְחָה,

בְּהַחֶלְךָ / דִבְרֵי נוֹרָא.

וְאֵל יָחִיד / צָרָיו יַכְחִיד,

וּבֶן־דָּוִד / יַרְבֶּה מִשְׂרָה.



וְיִשְׁמַע־אֵל / אֶל־יִשְׂרָאֵל,



וּבָא גֹאֵל / בִּגְלַל תּוֹרָה.
