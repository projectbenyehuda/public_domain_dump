


	
	הוֹי קוֹל
	



קינה לר׳ יחזקאל ישועה






הוֹי קוֹל כְּחוֹלָה / מִסְפֵּד יְלָלָה / נִשְׁמַע בְּבֶהָלָה / זֶרֶם הֶגֶה וּנְהִי וְקִינָה



עַל רַב פְּעָלִים / רַחְמָן לְדַלִּים / אָב לְאֻמְלָלִים / וּלְכָל אִישׁ מַר נֶפֶשׁ וְנַעֲנָה



צְבִי יִשְׂרָאֵל / שַׂר יְחֶזְקְאֵל / קְרָאוֹ הָאֵל / וַיִטְּשֵׁנוּ בְּרֹב מְגִנָּה



בְּמַר אֶצְרְחָה / רֹאשִׁי אֶקְרְחָה / שִׂמְחָה אֶשְׁכְּחָה / וְאֵבֶל אֶעֱרֹךְ בַּאֲנִינָה



אֶת מַהֲלָלוֹ / כֹּל לֹא יְכִילוֹ / מִצְוַת עֲמָלוֹ / עוֹבֵד מִיִּרְאָה וְרַב אֱמוּנָה



כָּל הַיּוֹדֵעַ / מַדָּיו קוֹרֵעַ / נָכְרִי כָרַע / כְּאֵבֶל תַּנִּים וּבְנוֹת יַעֲנָה



גָּדַל שִׁבְרֵנוּ / צַדִּיק מִמֶּנוּ / נִפְקָד וְאֵינוֹ / עַמּוּד הַיְמָנִי גַּם רֹאשׁ הַפִּנָּה



בְּצֵל שְׁכִינָה / נֶפֶשׁ עֲדִינָה / גַּם שַׁאֲנַנָּה / בִּמְנוּחָה נְכוֹנָה תִּהְיֶה חוֹנָה.
