


	
	שָׁבִים וְעוֹבְרִים
	



מצבת אסתר אשת סי׳ שמעון אלקנווי בווהראן איר תרנ״ג






שָׁבִים וְעוֹבְרִים / בְּאֹרַח קְבָרִים / עִמְדוּ בְצִירִים / וּשְׂאוּ הֶגֶה וּנְהִי וְקִינָה



קִינִים וְצִירִים / וְקוֹל שְׁבָרִים / וּבִמְקוֹם שִׁירִים / אֶשָּׂא הֶגֶה וּנְהִי וְקִינָה



כִּי פֹה נִקְבְּרָה / גְוִיָּה יְקָרָה / בַּנָּשִׁים שָׂרָה / יְרֵאַת אֱלֹהִים נֶפֶשׁ עֲדִינָה



רַבַּת פְּעָלִים / רַחְמָנִית לְדַלִּים / אֵם לָאֻמְלָלִים / וּלְכָל אִישׁ מַר נֶפֶשׁ נַעֲנָה



קוֹל מַר אֶצְרְחָה / שְׂעַר רֹאשִׁי אֶקְרְחָה / שִׂמְחָה אֶשְׁכְּחָה / וְאֶעֱרֹךְ אֵבֶל בַּאֲנִינָה



לְחַמָּה בָרָה / אֶסְתֵּר הַמְאֻשָּׁרָה / זַכָּה וִישָׁרָה / יִקְרַת רוּחַ רַבַּת תְּבוּנָה



בִּדְמִי יָמֶיהָ / נִצְמְתוּ עֲלוּמֶיהָ / כִּי רַבּוּ צִירֶיהָ / עַל בִּתָּהּ נִקְטְפָה כְשׁוֹשַׁנָּה



סוּרָה גַלְמוּדָה / הָלְכָה בְלִי חֶמְדָּה / וְהִיא בְעוֹדָהּ / רַכָּה עֲנוּגָה וְרַעֲנַנָּה



כָּל הַשּׁוֹמֵעַ / אֶת מַדָּיו קוֹרֵעַ / נָכְרִי אָח רֵעַ / יַחְדָּו סִפְדוּ מַר בְּרֹב מְגִנָּה



בְּצֵל שְׁכִינָה / נַפְשָׁהּ תִּהְיֶה חוֹנָה / גַּם שַׁאֲנַנָּה / בִמְנוּחָה נְכוֹנָה וְעֶלְיוֹנָה



אֵל יְחוֹנֵן עֲפָרָהּ / יְהִי לְסִתְרָה / בְּעַד בְּחִירָה / הַיָּקָר אֵלִיָּהוּ בְנָהּ.
