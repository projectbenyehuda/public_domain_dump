


	
	[זְעוּמֵי אֵל וּמָנוּאֵל חֲבֵרָם]
	




ں – – – ں – – ­­– ں – –



זְעוּמֵי אֵל וּמָנוּאֵל חֲבֵרָם / וְעוֹזְרֵיהֶם וְאַבְרָהָם גְּבִירָם

וְכָל־בּוּז עָם אֲשֶׁר לוֹ יָד בְּרִשְׁעָם / וְהַקָּרֵב לְחַרְחֵר רִיב עֲבוּרָם

אֲשֶׁר תָּעוּ וְהִתְעוּ אֶת־עֲדָתָם / וְהֶחְשִׁיכוּ בְּעַב רִשְׁעָם מְאוֹרָם

וְנָגְעָה עַד נְשִׂיא הַדּוֹר לְשׁוֹנָם / וְכִמְעַט זָרְקוּ בוֹ כּוֹס מְרוֹרָם

כְּאֶחָד הֵם נְשׁוֹתֵיהֶם וְטַפָּם / וְקִנְיָנָם וְכָל־הוֹנָם יְחֹרָם

וְיִתְלַקַּח חֲרוֹן אַף הָאֱלֹהִים / בְּעַצְמֵיהֶם וּבִבְשָׂרָם וְעוֹרָם

בְּרוּם מִבְצָר מְעוֹנָם יַעֲלֶה צַר / וְצוֹרֵר יַחֲנֶה בִּמְרוֹם הֲרָרָם

עֲצֵי פִרְיָם בְּקַרְדֻּמּוֹת יְשַׁחֵת / בְּמַזְמֵרוֹת גְּאוֹן רָמֵי אֲמִירָם

וְכַרְמֵי נַעֲמָנִים יִטְּעוּ אַךְ / לְגוֹי אַכְזָר וְזָר יִהְיֶה בְצִירָם

וְשָׂדוֹת יִזְרְעוּ אַךְ בַּתְּבוּאוֹת / לְגָרְנוֹ יֶאֱסֹף נָכְרִי עֲמִירָם

גְּדוּד אוֹיֵב רְכוּשׁ חֵילָם יְשֹׁדֵד / וְכָל־מִקְנָם וְצֹאנָם אַף־בְּקָרָם

וְכִלְחֹךְ שׁוֹר יְרַק שָׂדֶה יְלַחֵךְ / יְבוּל עֵצָם וְכָל־דֶּשֶׁא חֲצִירָם

שְׁבִי חֶרֶב וְשֹׁד יִרֶב בְּאַרְצָם / וְסַף־רַעַל שְׁפוּכָה עַל־מְגוּרָם

וְכָל־אָלוֹת יְגוּרֵהֶם אֲשֶׁר צוּר / בְּיַד צִירוֹ לְיִשְׂרָאֵל מְסָרָם

מְנֻדָּחִים יְבַקְשׁוּ בַפְּתָחִים / בְּחֹסֶר כֹּל כְּסוּתָם עִם שְׁאֵרָם

רְעֵבִים מִבְּלִי לֶחֶם לְשָׂבְעָם / צְמֵאִים מִבְּלִי מַיִם בְּבוֹרָם

יְחֵפִים מִבְּלִי מַנְעָל לְרַגְלָם / וְסוּת בָּלוּי לְהִתְכַּסּוֹת בְּקֹרָם

בְּשִׁבְתָּם יִצְמְחוּ תוּגוֹת בְּלִבָּם / וְיִכָּשֵׁל בְּלֶכְתָּם חוּץ אֲשׁוּרָם

אֲשֶׁר הֵם יִמְאֲסוּ יִבְחַר אֱלֹהִים / וְיִמְאַס בָּחֳרִי אַפּוֹ בְּחִירָם

יְאַמֵּץ עֹז מְשַׂנְאֵיהֶם וְכִדְמוּת / גְּדִי עִזִּים יְשַׁסַּע גֵּו אֲבִירָם

וְרִשְׁעָם וַעֲוֹן בִּצְעָם יְלַהְטֵם / וְדָלְקָה בָם לְשׁוֹן אֵשׁ מִנְּחִירָם

וְיוֹמָם יֶחֱשַׁךְ שִׁמְשָׁם וְלַיְלָה / מְאוֹר אֵשׁ יַאֲפִיל נָגְהוֹ בְּנֵרָם

דְּצַ“ךְ עָדָ”שׁ בְּאָחָ"ב יִדְבְּקוּ בָם / לְכַלּוֹת מִגְּדוֹלָם עַד־צְעִירָם

וְיָחוֹגּוּ וְיָנוּעוּ נְבֻכִים / כְּשִׁכּוֹרִים אֲשֶׁר יַיִן עֲבָרָם

וְכָל־חֹלִי וּמַכָּה בָּם תְּקַנֵּן / וְאֵין צֹרִי לְמַכְאוֹבָם וְצִירָם

עֲדֵי תִדְבַּק לְשׁוֹן חֶלְקָם לְחִכָּם / וְתִמַּקְנָה בְּנוֹת עֵינָם בְחוֹרָם

וְנֶכֶד אַל־יְהִי לָהֶם וְלֹא נִין / הֲכִי יַשְׁבִּית אֱלֹהִים אֶת־מְקוֹרָם

וְחֶטְאָם לַצְּמִיתוּת יִמְכְּרֵם אֶל־ / מְצוּקוֹת יוֹם וְאֵין נוֹתֵן שְׂכָרָם

וְאֵיכָה יִתְּנוּ כָפְרָם וּמִמְכָּר / עֲדֵי־עַד חֵטְא חֲדַל שֵׁעוּר מְכָרָם

וּבַעְבוּר הֵם בְּנִכְלֵיהֶם אֲחִיהֶם / זְמַנָּם כֵּן בְּלִי חֶמְלָה מְכָרָם

זְמָן בָּם יַעֲשֶׂה יוֹם יוֹם שְׁפָטִים / וְיַרְבֶּה בַּאֲשֶׁר יִפְנוּ מְגוּרָם

וְחִצָּיו הַשְּׁנוּנִים בָּם יְפַלַּח / וְיִשָּׁבְרוּ וְאֵין חוֹבֵשׁ מְזוֹרָם

וְנָפְלוּ לַעֲפַר מָוֶת חֲלָלִים / וְאָכְלוּ עוֹף וְחַיָּה אֶת־בְּשָׂרָם

וּפִגְרֵיהֶם בְּקִבְרָם יִבְאֲשׁוּ עַד / אֲשֶׁר יִלְאֶה שְׂאֵת בָּאְשָׁם עֲפָרָם

כְּחָכָם בָּם וְכַסָּכָל יְסוּפוּן / וְיִכָּרְתוּ כְדַלָּם כַּעֲשִׁירָם

אֲשֶׁר חָפְשִׁי יְהִי בָהֶם יְדֻקָּר / וְלַטֶּבַח זְמָן יוֹצִיא אֲסִירָם

וּמֵימֵיהֶם וְלַחְמָם רוֹשׁ בְּלוּלִים / וְסָם מֵעֵץ בְקַלַּחְתָּם וְסִירָם

הֲיִבְכֶּה אִישׁ וְכַף יַכֶּה בְמוֹתָם / וְהֵמָּה גָֽֽרְמוּ בִכְיָם לְדוֹרָם

וּמִי לֹא יִִֽהְיֶה כַיּוֹם בְּעוֹכְרָם / וְהֵם עָכְרוּ אֲשֶׁר הוּנַף וְהוּרָם

וְאִם הֵם מָאֲסוּ חָכְמָה הֲיֶחְדַּל / חֲכַם־לֵב מֵעֲרוֹת נֶגַע שְׁאֵרָם

זְבוּל יָגִיל וְיַעַלְזוּ עֲרָבוֹת / לְהֵעָדְרָם וְיִשִׂאוּ קוֹל זְמִירָם

וְשִׁבְעָה כּוֹֽכְבֵי לֶכֶת כְּאֶחָד / בְּחִבָּה יַחְגְּרוּ חֶמְדַּת אֲזוֹרָם

וּמַזָּרוֹת בְּרֹן יַחַד זְמִירוֹת / יְרִיעוּן מוּל גְּאוֹן נִצְחָם וְצוּרָם

וְצוּר יִשְׂמַח לְהִמָּסְרָם וְאָבְדָם / הֲכִי נִחַם אֲשֶׁר כָּכָה יְצָרָם

וְתִפְצַחְנָה בְרִנָּה פֶּה אֲדָמוֹת / וְהֶהָרִים יְכַסֵּם סוּת הֲדָרָם

לְחָלְפָם בַּעֲלֵי כָנָף יְשִׂישׂוּן / וְעַל־רֹאשָׁם יְנוֹצֵץ הוֹד אֲוֵירָם

בְּהֵמוֹת עִם הֲמוֹן חַיּוֹת בְּשָׂדוֹת / בְּנַפְשׁוֹתָם לְנַצֵּל זִיו פְּאֵרָם

וְתוֹךְ סִיר נַחֲלֵי גָפְרִית יְעֻנּוּ / וְלֹא יֶחְסַר מְדוּרָתָם וְאוּרָם

כְּקֹרַח סָף יְהִי סוֹפָם לְהַכְרִית / וְאַחְרִיתָם כְּדָתָן וַאֲבִירָם.


(?1519)
