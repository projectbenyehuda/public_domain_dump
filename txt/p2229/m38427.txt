


	
	[אֶקְרָא צוּר עֶלְיוֹן יוֹשֵׁב סֵתֶר חֶבְיוֹן (תחנה)]
	



[מרוֹמָא, מת אחרי שנת 1280]






תְּחִנָּה עַל־צָרַת הַזְּמָן





אֶקְרָא צוּר עֶלְיוֹן / יוֹשֵׁב סֵתֶר חֶבְיוֹן

אֵל גֹּמֵר עָלָי / תַּגְמוּלוֹהִי עָלָי.





צוּר עֶלְיוֹן אֶקְרָא / עוֹנֶה בַצָּרָה

בְּךָ הִיא תוֹחַלְתִּי / נֶאְזָר בִּגְבוּרָה

מִלּוֹחֲמִים רַבִּים / תִּהְיֶה לִי עֶזְרָה





בַּקָּמִים עָלָי / סָבִיב שָׁתוּ עָלָי.





מָה-רַבּוּ צָרָי / רוֹדַי שׁוֹדֲדָי

רַבִּים קָמוּ עָלָי / כֻּלָּם הָיוּ עָלָי.





צָרוֹת כַּמָּה מִסִּים / אֵיךְ אוּכַל שְׂאֵת

מִדֵּי שָׁנָה בְּשָׁנָה / מַתָּן אֶשָּׂא מַשְׂאֵת

לִשְׂחֹק עוֹשִׂים / בִּשְׁאָט נֶפֶשׁ יִשְׂאוּ אֵת





רְדִידִי מֵעָלָי / הַפַּסִּים אֲשֶׁר עָלָי.





דָּמוּ לְעַבְדֵי הָאֱלִיל / כֹּהֲנֵי הַבָּמוֹת

הִתְעִיבוּ עֲלִילוֹת / תּוֹךְ אָוֶן וּמִרְמוֹת

כִּמְרִיבֵי כֹהֵן / מַעַשְׂרוֹת וּתְרוּמוֹת





מַגְדִּילִים עָלָי / מַחְשְׁבוֹתָם עָלָי.





קָרְבוּ רוֹדְפֵי זִמָּה / הֵם מַנְהִיגֵיהֶם

קִרְבָּם קֶבֶר הַוּוֹת / חֵץ שָׁחוּט פִּיהֶם

כּוֹנֲנוּ חִצָּם לִירוֹת / מִן-מַחֲבוֹאֵיהֶם





יִתְלַחֲשׁוּ עָלָי / כָּל-עֲצָתָם עָלָי.





יָצְאוּ מִינֵי מִינִים / שׁוּעָלִים חוֹבְלִים

מַהְגִּים לוֹבְשֵׁי שַׂקִּים / וִיחֵפִים חוֹבְלִים

דּוֹרְשִׁים מַקְלוֹת לִרְדּוֹת / נֹעַם אֶת-חוֹבְלִים





פָּצוּ פִיהֶם עָלָי / יָסֹלּוּ עָלָי.





הֵן מֵהֶם פָּרַח / שׂוֹנֵא בְלֹא רַחֲמִים

קֹץ מֻנָד נוֹשֵׁךְ / בָּשָׂר וַעֲצָמִים

אָח לְצָרָה יֻלַּד / וּלְאַף וּזְעָמִים





הֶגְיוֹנָיו עָלָי / לֹא יַחְמֹל עָלָי.





הַצִּילֵנִי נָא / אָבִי אֲבִיר יַעֲקֹב

מִיַּד אָחִי אַכְזָר / עָקוֹב יַעְקֹב

לְהַשְׁמִיד שְׁאָר יָשׁוּב / שְׁאָר יַעֲקֹב





רָחֲקוּ מֵעָלָי / עָמְדוּ עָלָי.





רִיבָה אֵל רִיבִי / שָׁפְטָה מִשְׁפָּטִי

מִצַּר אִישׁ חֶרְמִי / בַּעַל מִשְׁפָּטִי

נָתַן לִבּוֹ וְעֵינָיו / מֵעָמְדִי לְמוֹטֲטִי





אוֹיְבִי עָלָי / הִגְדִּיל עָלָי.





וַאֲנִי תָמִיד עִמָּךְ / תֶּאֱחֹז בִּימִינִי

אַל תַּנִּיחֵנִי / לַכּוֹפְרִים וּמְלָשְׁנִי

מַחֲרִיבִים וּמְהָרְסִים / יָצְאוּ מִמֶּנִּי





נֶאֶסְפוּ עָלָי / שָׂחֲקוּ עָלָי.





פֹּצֶה-פֶה וּמְצַפְצֵף / אַחֲרַי חוֹקֵר

עַל-שָׁרְשֵׁי רַגְלַי / מִתְחַקֶּה וּמְבַקֵּר

יָפִחַ חָמָס / וּמֵקִים עֵדֵי שָׁקֶר





הָהְפַּךְ עָלָי / לִבּוֹ עָלָי.





אַחַת הִיא / עַל-כֵּן חָשַׁבְתִּי אָמָרְתִּי

אָמְנָה אָנֹכִי / חָטָאתִי הִרְשָׁעְתִּי

כָּל-זֹאת בָּאַתְנִי / כִּי בַבְּרִית שִׁקָּרְתִּי





בְּמוֹט רַגְלִי עָלָי / תִּכְבַּד עָלָי.





חַי צוּרִי וּבָרוּךְ / הָאֵל הַנֶּאֱמָן

זָכְרָה חֶסֶד וּבְרִית / אֶל-עָם לֹא אַלְמָן

קָרְבָה אֶל-נַפְשִׁי גְּאָלָהּ / אָב הָרַחְמָן





בִּקְרֹב עָלָי / נוֹסְדוּ עָלָי.
