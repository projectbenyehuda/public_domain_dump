


	
	בני אדם הבל
	



על העניין


בְּנֵי אָדָם הֶבֶל / בְּתוֹך מַאְסַר תֵּבֵל / אֲסוּרִים בַּכֶּבֶל / וְנַפְשִׁי יוֹדַעַת



תְּכַלֶּה גוּפֵיהֶם / וְתַשְׁחִית מַעְשֵׂיהֶם / וּמַחְמַד עֵינֵיהֶם / לְעֵינָם פּוֹצַעַת



בְּאַף שֵׁן חוֹרֶקֶת / וְנַפְשָׁם בּוֹקֶקֶת / וְהִיא כִמְּשַׂחֶקֶת / וּבָם מִשְׁתַּעְשַׁעַת



אֱנוֹשׁ רָם תַּכְנִיעַ / וּבִתְהוֹם תַּשְׁקִיעַ / וְלִבּוֹת תַּבְקִיעַ / וּנְפָשׁוֹת קוֹבַעַת



לְזֹאת אִישׁ אַל יִשְׂמַח / בְּנַפְשׁוֹ אִם יִצְלַח / אֲשֶׁר הַיּוֹם יִקַּח / לְמָחָר גּוֹרַעַת



וְאַל יִבְטַח לִבּוֹ / בְעוֹדוֹ בִמְסִבּוֹ / אֲשֶׁר עוֹדֶנּוּ בוֹ / לְבָבוֹ שׁוֹסַעַת



וְאַל יִבְטַח גֶּבֶר / עֲלֵי כֶסֶף צוֹבֵר / וּמָחָר בַּקֶּבֶר / גְּאוֹנוֹ תּוֹלַעַת



וְאִם עֵת הִגִּישָׁה / לְפָנָיו אֶת דִּבְשָׁהּ / לְסוֹף אוֹכֵל רָאשָׁהּ / וְיִשְׁתֶּה קֻבַּעַת



וְעֵת יֵחַשֵּׂף שֵׁת / וְצוֹעֵק בָּרֶשֶׁת / תְּשׁוּעָה חֵרֶשֶׁת / וְצָרָה שׁוֹמַעַת
