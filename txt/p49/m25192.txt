


	
	[היש גשם נדבות]
	




וְלוֹ אֶל קְצָת אַנְשֵׁי הַחָכְמָה בְאַרְצוֹת הַמִּזְרָח



U – – – U – – – U //– – U – – – U – – – U – –



הֲיֵשׁ גֶּשֶׁם נְדָבוֹת עוֹד בְּעָבִים – // וְהָאָרֶץ צְמֵאָה מִנְּדִיבִים,

וְשִׁכְבַת טַל הֲתַעַל עוֹד – וְאַנְשֵׁי // תְעָלָה שָׁכְבוּ תַחַת רְגָבִים!

וְאִם יַשְׁקוּ נְשִׂיאֵי עָב רְגָבִים – // וְעָלוּ עַל נְשִׂיאֵי עַם עֲשָׂבִים!

כְּפִירִים, יִשְׂחֲקוּ אֶמֶשׁ לְקוֹל סוּס, // עֲלֵיהֶם יִשְׂחַקוּ הַיּוֹם כְּלָבִים,

וְאִישִׁים, הַזְּמַן מָחָה שְׁמוֹתָם – // וּמַעְשֵׂיהֶם עֲלֵי מִצְחוֹ כְּתוּבִים,

וְדָן עֶרֶב וּבֹקֶר עַל שְׁמוֹתָם // לְהִצָּעֵן כְּמוֹ אֹהֶל עֲרָבִים,

וְהֵמָּה, כָּל אֲשֶׁר נוֹתַר לְלַיִל // וְיוֹם־זֵכֶר לְזֵכֶר טוֹב נְקוּבִים.

וּמַה יִּתְרוֹן בְּיֵין אַהַב וְלֶכֶת // לְמִשְׁתֶּה אַחֲרֵי לֶכֶת אֲהוּבִים?

וּמַה יָּמִית כְּאֵב נִידַת יְדִידִים, // בְּמוֹתָם שִׁלְּחוּ בַלֵּב כְּאֵבִים?

שְׁכַח אֶת זֹאת וְתֵן פּוּגַת לְנַפְשָׁךְ // וְאַל תִּשְׁכַּח שְׁאֵרִית הַחֲשׁוּבִים,

וְאַל תִּשְׁכַּח אֲשֶׁר שָׁכְחוּ בְּיָמָיו // בְּנֵי תוֹרָה וְתַלְמוּד הָעֲצָבִים,

וְכַבֵּה מֵעֲלֵי לִבָּךְ, בְּזֵכֶר // שְׁמוֹ הַטּוֹב וּמִפְעָלָיו, שְׁבִיבִים.

אֲנִי אוֹמֵר, וְאֵין כָּזָב בְּחִכִּי, // – וְאֵיכָה יַעֲנֶה שׁוֹעַ כְּזָבִים? –

פְּעָלֶיךָ כְּמוֹ כֶסֶף צְרוּפִים // וְשִׂיחֶיךָ נְקִיִּים כַּזְּהָבִים

וְיֵשׁ יַיִן בְּתוֹרַת כָּל חֲכָמִים – // וְתוֹרָתָךְ לְתוֹרָתָם עֲנָבִים,

וְאַתְּ שָׁרְשָׁה, וּמִפִּיךָ יְנִיקוּן // שְׂרִיגֶיהָ וְיָשִׁיקוּ יְקָבִים,

כְאִלּוּ לִמְּדָךְ נָבִיא, נְתָנָהּ // בְּתוֹךְ אָרוֹן וְתַחַת לַכְּרוּבִים,

וְלָמַדְתָּ נְבִיאִים מִזְּכַרְיָה // וּמַלְאָכִי, וּמֵחַגַּי – כְּתוּבִים,

וְהוֹרָךְ רַחֲבָא תַלְמוּד, וְרָבָא // הֱבִיאָךְ בַּחֲדָרָיו הָרְחָבִים.

אֱמֹר לִי: אֵיךְ בְּפִיךָ רוֹש וְנֹפֶת, // וּמֵי נָהָר בְּחִכָּךְ עִם לְהָבִים,

לְסוֹד רָם, נִמְנְעוּ תַּחַת לְשׁוֹנָךְ, // וְאַרְבַּעְתָּם בְּפֶה לֹא נֶעֱרָבִים?

שְׁנַיִם – תַּאֲכִיל מֵהֶם אֲנָשִׁים // אֲשֶׁר הֵם אֶל תְּעוּדָתָךְ רְעֵבִים,

וּבִשְׁנַיִם תְעַנֶּה כָל קְשֵׁי לֵב // אֲשֶׁר הֵם עִם נְבוֹנֵי לֵב מְרִיבִים.

וְאֵיךְ תִּטּשׁ קְרוֹבֶיךָ לְמַעַן // חֲמָדוּךָ וְזַכִּים לָךְ קְרוֹבִים.

וְאֵיךְ תֵּשֵׁב בְּעֵבֶר יָם וְתָצוּד // בְּעֵבֶר זֶה בְּלִי רֶשֶׁת לְבָבִים?

וְאֵיךְ תָּבוֹא בְּכָל מִלָּה קְרוֹבָה // וּמַעֲנֶה רַךְ וְתִשְׁכֹּן בַּקְּרָבִים?

יְדִידִי! אִם בְּנֵי אָדָם בְּבָבֶל // חֲבֵרֶיךָ, וְרֵעַי מַעֲרָבִים –

אֲנִי אֶמְשֹׁךְ בְּמוֹ חַכַּת דְּבָרִים // מְתוּקִים מִמְּךָ אֵלַי אֲהָבִים,

וְאִם אַתְּ נַעֲלֶה עַל בָּמֳתֵי עָב – // בְּעֵטִי מִמְּךָ אַזִּיל רְבִיבִים,

3 וְאַעַל אַחֲרֶיךָ אָל עֲנָנִים, // וְאַשִּׂיגָךְ בְּסֻלָּם מִכְּתָבִים!
