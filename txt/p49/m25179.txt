


	
	[עלה אלי ועזרני]
	



ולה אתר אלאנתקאל מן קרטבה

ולו סמוך לצאתו מקורדובה






U – – – U – – – U //– – U – – – U – – – U – –



עֲלֵה אֵלַי וְעָזְרֵנִי, יְדִידִי, // עֲלֵי יָמִים גְּזָלוּנִי כְּבוֹדִי,

וְהִתְאַחֵד לְהוֹשִׁיעַ מְזֻמָּן – // לְקָשׁוֹת, מִן זְמַן שָׂמוֹ יְחִידִי!

בְּלִבִּי קֵד, וְלֹא שִׁנּוּ לְהָבָיו // יְצוּרַי, כִּי אֲנִי כוֹסֶה יְקוֹדִי,

וְלֹא יֵדְעוּ אֲנָשִׁים מַה בְּלִבִּי, // וְלֹא אָבִיא בְּנֵי אָדָם בְּסוֹדִי,

אֲבָל יֵדַע זְמַן רַע כִּי בְּזִיתִיו, // וְכִי אֶמְאַס תְּנוּאוֹתָיו בְּעוֹדִי,

וְאִם עָלַי בְּמַלְכֻּדְתּוֹ יְסוֹבֵב – // בְּצוּרִי, אֶלְכְּדֶנּוּ תוֹךְ מְצוּדִי,

וְאִם יָכִין בְּנֵי קַשְׁתּוֹ – אֲפַלַּח // בְּחִיצַּי לֵב זְמַן, פִּלַּח כְּבֵדִי,

וְאִם אֵלִי יְשִיבֵנִי לְרִבְצִי – // אֲזַי יֵבוֹשׁ זְמַן הִרְחִיק נְדֹדִי,

וְאִם עֵינוֹ תְּהִי עָלַי לְטוֹבָה – // עֲלֵי עֵקֶב זְמַן אָגוּד גְּדוּדִי,

וְאִם יָדִי בְּיַד נַחוּם אֲחוּזָה – // זְמַנִּי בֶאֱמֶת אָחוּז בְּיָדִי,

בְּיוֹם צָרָה כְּצוּר חָזָק מְצָאתִיו // וְהָיָה לִי כְּמוֹ אִחי בְּנוּדִי,

רְפָדַנִי – וְלֹא בִיקָר וְלֹא הוֹן, // אֲבָל בָּא אֵיד – וְהִתְעַצֵּב לְאֵידִי

אֲנַי עֶדִי וְכוֹתֶרֶת לְרֵעִי // וְאַתָּה לִי לְכוֹתֶרֶת וְעֶדִי,

וְלֹא אֶחְפֹּץ בְּמַרְבַדִּים, וְאוּלָם // יְדִידוֹת הַחֲבֵרִים הֵם רְבִידִי.

אֲשֶׁר זוֹכֵר בְּרִית־קֶשֶׁת כְּרוּתָה // נְדוֹד יַכְרִית וְיִזְכֹּר אֶת מְרוּדִי,

וְהַדָּן דִּין אֱמֶת יָדִין זְמַן רַע // פְּצַעִני וְנָשָׂא אֶת רְדִידִי!
