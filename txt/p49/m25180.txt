


	
	[לשר רבי נסים]
	



וכתב אליה מע צדיק להמא

וכתב אליו באמצעות ידיד לשניהם






– – U – – U – – – U – – –  //
– – U – – U – – – U – – –



לַשַּׂר רְבִי נִסִּים כְּתָב רִיב וְתוֹכַחַת, // מֵאֵת יְדִידוֹ, עִם יְדִידוֹ מְשֻלַּחַת,

הִנֵּה אֲדַבֵּר לוֹ שְׁתַּיִם וְלֹא אוֹסִיף, // אוּלַי יְשִׁיבֵנִי עֲלֵיהֶם בְּמוֹ אַחַת,

אוּלַי יְכַבּוּ מֵי מְרִיבָה אֲשֶׁר אַמְטִיר // לַהַב תְּנוּאָה תּוֹךְ לְבָבוֹ מְלַקַּחַת

הַאַחֲרֵי הֵחֵל וְשָׁלַח אֱלֵי אָחִיו // שִׁירָה שְׁהִיא עַל רֹאשׁ בְּנוֹת שִיר לְמִטְפַּחַת

וַיַּעֲנֵהוּ מַהֲלָלָיו, וְכָל אַחַת // מֵהֶן עֲלֵי רַקַּת זְמַנּוֹ מְפֻתַּחַת –

הִשְׁלִיךְ אֲהָבָיו אַחֲרֵי גַו, וְדָתוֹתָיו // זֵרָה, כְּמוֹ זוֹרֶה בְמִזְרֶה וּבָרַחַת?

הִנֵּה רְבִי יִשַׁי יְדִידוֹ, אֲשֶׁר אֵין לוֹ // דּוֹמֶה בְתוֹרָתוֹ בְּרִיפַת וּמָנַחַת,

חָקְרָה וּבוֹאָה אֶל חֲדָרָיו וְאָז תִּמְצָא // בּוֹר סִיד וּמַבּוּעַ, וְסוֹגֵר וּפוֹתַחַת,

וֶהְיֵה לְלָשׁוֹן לוֹ וּפֶה אֶל קְהִלָּה אַתְּ // פֶּה לָהּ וְעֵין יָמִין וּפַרְצוּף וּפַדַּחַת,

וּשְׁאַל, יְחַוָּךְ אֶת דְּרָכַי וְתִתְבּוֹנֵן // כִּי לִי וּבִי נֶפֶשׁ מְשִׁיבָה וְנוֹכַחַת

לַשֹּׁד וְלַנַּחַת, וְלַשְׂאֵת וְסַפַּחַת, // לָתֵת וְלָקַחַת, לְהַנְתִּיךְ וְלָפַחַת,

אֶחְתֹּם מְרִיבָתִי בְאַחַת וְלֹא אֵחַת // כַּיּוֹם, וְכָמוֹנִי יְדַבֵּר וְלֹא יֵחַת:

אִם כֵּן יְדִידוּת הַחֲבֵרִים מְשֻׁלַּחַת // פַּעַם, וּפַעְמַיִם צְרוּרָה בְאַמְתַּחַת –

הוֹרֵד יְדִידוּתָם, אֱלֹהַי, וְחֶבְרָתָם, // בִּלְתִּי יְדִידוּתוֹ וְחֶבְרוֹ, אֱלֵי שָׁחַת!
