


	
	[לך שיחים כמו]
	



ולה [אלי ראש ישיבה כביר] פי אלמשרק

וְלוֹ אֶל רֹאשׁ יְשִׁיבָה גָדוֹל בַמִּזְרָח.


U – – – U – – – U //– – U – – – U – – – U – –



לְךָ שִׂיחִים כְּמוֹ נֹפֶת מְתוּקִים, // לְךָ חִידוֹת יְדוּעוֹת בַּשְּׁוָקִים,

וְתַחְבּוּלוֹת בְּתוֹרָתָךְ כְּאוּרִים, // וּמוֹעֵצוֹת כְּמוֹ מַיִם עֲמֻקִּים,

וּמַחְשֶׁבֶת תְּגַלֶּה הַצְּפוּנוֹת, // שְׁבָחָהּ עַל עֲנָקָךְ לַעֲנָקִים,

וְאַתְּ כֹּהֵן, וְאוּלָם לֹא תְּרַפֵּא, // לְבַד מִלֵּב נְסוֹגֵי לֵב, בְּהָקִים,

וְהִכְרַתָּ, בְּלֵב תּוֹכוֹ כְּבָרוֹ // וּבִמְזִמוֹת קְרוֹבֵיהֶן רְחוֹקִים,

לְמַחְלֹקוֹת כְּאִישׁ שָׂעִיר שְׂעִירוֹת, // בְּפִתְרוֹנִים כְּיִשְׂרָאֵל חֲלָקִים,

וְנִשְׁבַּעְתָּ לְהַכּוֹת הָעֲשׁוּקִים // – וְכָמוֹךָ יְהִי נִשְׁבָּע וּמֵקִים –

בְּרוּחַ פֶּה וְשָׂפָה, עַד יְשִׁיבוּן // בְּיָד מַה יֵּשׁ בְּיָדָם לַעֲשׁוּקִים,

וְעַד תַּכְרִית גְּאוֹן רָעִים, וְתַרְכִּיב // לְבַב נֵפֶל עֲלֵי גוּפַת עֲנָקִים.

בְּבֵיתָךְ יִרְבְּיוּן יוֹשְׁבִים, וְאוּלָם // בְּתַלְמוּדָם יְרוּצוּן כַּאֲפִיקִים,

וְחַכְמֵי לֵב מְהֻלָּלִים בְּתַלְמוּד // סְדָרִים, לֹא בְּתַלְמוּד הַפְּרָקִים,

וְכָל כּוֹתֵב בְּלֵב תּוֹרָה – וּמוֹחֶה // הֲתוּלִים שֶׁעֲלֵי לֵבָב חֲקוּקִים,

וְרֵאשִׁית רִיב, כְּלֵיל גֶּשֶׁם, חֲשֵׁכָה – // וּפִסְקֵי דִין יְאִירוּן כַּבְּרָקִים,

וּמִשְׁפָּטִים נְטוּעִים בַּתְּהוֹמוֹת – // וְצַמַּרְתָּם גְּבוֹהָה מִשְּחָקִים,

וְתוֹכַחַת שְׂאֵת נֶפֶשׁ מְרַפָּא, // וּמִכֶּסֶל מְנַתֶּקֶת נְתָקִים.

קְחָה מֵאֵת מְיֻדָּעָךְ דְּבָרִים, // בְּמַעְבֵה רוֹם, בְּכִכַּר עָשׁ יְצוּקִים,

יְדֵי אִישׁ טוֹב, וְשֵׁם טוֹב מַחֲזִיקִים, // וְשֶׁמֶן טוֹב עֲלֵי מִדּוֹת מְרִיקִים.

וְאָדָם יֶחֱלַשׁ וּבְנוֹ וְנֶכְדּוֹ // וּבֶן נֶכְדּוֹ – וְהֵם כִּרְאִי חֲזָקִים!
