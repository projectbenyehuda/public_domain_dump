


	
	רכב. [אשר נטה]
	








אֲשֶׁר נָטָה שְׁחָקִים / עֲלֵי אַרְצוֹ כְּקֻבָּה

וְשָׁת שִׁכְנוֹ בְּחַשְׁמַל / וּבֵינוֹת אֵשׁ וְלַבָּה

וְאֵימָה לוֹ חֲתֻלָּה / וְאֵין יוֹצֵא וְאֵין בָּא —

סְלַח פֶּשַׁע וְהַרְחֵק / מְלִיצֵי חֵטְא וְדִבָּה,





אֵל נַעֲרָץ בְּסוֹד קְדוֹשִׁים רַבָּה!





אֱלֹהַי, עָב לְבוּשָׁךְ / וְעָנָן וְעֲרָפֶל,

וְהַנֹּגַהּ יְגוּרָךְ / וְתֵדַע מַה בְּאֹפֶל,

וְאַתְּ בּוֹרֵא מְתוּקִים / וְכָל מָרוֹר וְתָפֵל,

וְכָל שֹׁרֶשׁ וְעָנָף / וְכָל עִקָּר וְטָפֵל,

וְלָךְ שַׁחוּ גְבֹהִים / וְכָל נִשָּׂא וְשָׁפֵל,

וְאִם אֹמַר וְאַגְדִּיל — / לְךָ יֶתֶר וְכֶפֶל!

וְאֵיךְ תִּפְקֹד חֲטָאִים / עֲלֵי תוֹלָע וְנֶפֶל?

זְרֵה חֶטְאוֹ כְּמוֹ מֹץ, / כְּעָשָׁן מֵאֲרֻבָּה!





אֵל נַעֲרָץ בְּסוֹד קְדוֹשִׁים רַבָּה!





אֲשֶׁר כָּל הָעֲפָרִים / וְטִיט חוּצוֹת בְּחֹמֶר,

וּמַד שַׂחַק בְּזֶרֶת / וְהַמַּיִם בְּעֹמֶר,

וְשָׂת גֶּלֶד שְׂחָקִים / בְּכוֹכָבָיו כְּנָמֵר

וְהוּא מוֹלִיד וּמֵמִית / וְאֵין מַמְרֶה וּמֵמֵר,

וְכֹל יִפֹּל בְּרִשְׁתוֹ / כְּתוֹא מִכְמָר וְזֶמֶר,

וְכָל נִבְרָא כְּמוֹ צֹאן / וְהוּא נוֹצֵר וְשׁוֹמֵר,

וְלָנוּ שָוְא וְכָזָב / וְהוּא אוֹמֵר וְגוֹמֵר,

וְנֹגַהּ לוֹ יְסוֹבֵב / וְרוּחַ, כֹּל תְּסַמֵּר,

וְאוֹר יָרֹק וְשָׁחֹר, / וְאוֹר לָבָן כְּצֶמֶר,

וְאֵשׁ תִּבְעַר בְּתוֹךְ אֵשׁ, / וְאשׁ כָּל אֵשׁ מְכַבָּה,





אֵל נַעֲרָץ בְּסוֹד קְדוֹשִׁים רַבָּה!





לְךָ שָׁעוּ שְׂרָפִים / וּמַלְאָכִים צְפוּפִים

וּמִפַּחְדָּךְ יְבוֹאוּן / בְּתוֹךְ חוֹרִים וְכֵפִים

וְכוֹכָבִים מְאוֹרָם / בְּחֹשֶׁךְ מַחֲלִיפִים

וְאַתְּ מַמְלִיךְ מְלָכִים / וְנוֹתֵן חֵן לְיָפִים

וּמִזַּעְמָךְ אֲבֵדִים, / כְּרֶגַע נֶאֱסָפִים

וְאַתְּ תַּעְטִיף קְשׁוּרִים, / וְאַתְּ תִּקְשֹׁר עֲטוּפִים,

וְאַתְּ גּוֹמֵל חֲסָדִים / לְמֵאוֹת וַאֲלָפִים —

גְּמֹל טוֹב עַל סְגֻלָּה, / וְאִם מֵחֵטְא עֲקֻבָּה,







אֵל נַעֲרָץ בְּסוֹד קְדוֹשִׁים רַבָּה!





לְךָ רָפוּ שְׂחָקִים / אֲשֶׁר כִּרְאִי חֲזָקִים

וְלָךְ שַׂחוּ נְפִילִים / וְשָׁפֵל רוּם עֲנָקִים

וְכֹל יֹאמַר וְיַחְלִיף — / וְאַתְּ גּוֹזֵר וּמֵקִים,

וְאַתְּ מַבִּיט לְאֶרֶץ / וְתִרְעַד מִן מְצוּקִים,

וְיִגָּלוּ יְסוֹדוֹת / וְיֵרָאוּ אֲפִיקִים,

וּמִמָּךְ קוֹל רְעָמִים / וְמִשְׁלוֹח בְּרָקִים

וְשַׂמְתָּ חֹק וּמִשְׁפָּט / בְּלִבֵּנוּ חֲקוּקִים

וְהִסְכַּלְנוּ עֲשׂוֹ בָם / כְּמוֹ לֵצִים וְרֵיקִים.

נְתֹק פֶּשַׁע וְעָווֹן / לְגוּפֵנוּ דְבֵקִים,

וְאִם רַב הֶעֱוִינוּ — / סְלִיחָתָךְ מְרֻבָּה,





אֵל נַעֲרָץ בְּסוֹד קְדוֹשִׁים רַבָּה!





אֲשֶׁר יַשְׁלִיךְ קְרָחָיו / כְּלֶחֶם רַךְ וּפָתוּת

וְקוֹרֵעַ חֲזִיזָיו / כְּמוֹ בֶגֶד שְׁחִיתוּת —

שְׁלַח כָּבוֹד לְחָרִד / וְלַפִּיד בּוּז לְעַשְׁתּוּת

וְהַלְבֵּן חֵטְא מְאָדָּם / כְּשָׁנִי אוֹ כְּמוֹ תוּת

וְאַל תִּמְכֹּר עֲדָתָךְ / לְזָרִים לַצְּמִיתוּת

וְהַעַל נָא אֲרוּכָה / לְעַם נִדְכֶּה וְכָתוּת

וְלִשְׁכוּלָה גְרוּשָׁה / בְּלִי סֵפֶר כְּרִיתוּת —

תְּאָרְשָׂה בֶּאֱמוּנָה / וְתִכְתֹּב לָהּ כְּתֻבָּה,





אֵל נַעֲרָץ בְּסוֹד קְדוֹשִׁים רַבָּה!
