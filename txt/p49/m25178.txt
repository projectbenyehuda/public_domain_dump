


	
	[ידידותך בתוך לבי]
	



פראנעה אבי אעזה אללה בהדה אלאביאת

והשיב לו אבי, יוקירהו האל, בבתים האלה






U – – – U – – – U //– – U – – – U – – – U – –



יְדִידוּתָךְ בְּתוֹךְ לִבִּי יְצוּקָה // וְאֵין בֵּינִי וּבֵינָךְ דִּין חֲלֻקָּה,

וְאַהְבָה לֹא כְּאַהְבָתָךְ, כְּתָבָהּ // אֱנוֹשׁ עַל קִיר לְבָבִי – חִישׁ מְחוּקָה

וְנַפְשִׁי מֵאֲשֶׁר תַּרְחִיק רְחוֹקָה, // וְנַפְשִׁי בַּאֲשֶׁר תִּדְבַּק דְּבֵקָה.

וְיֵשׁ אוֹהֵב אֲשֶׁר כַּפּוֹ סְפוּקָה // בְּצָרָתָךְ, וְצַר – שִׁנּוֹ חֲרוּקָה,

וְיֵשׁ מַרְאֶה לְךָ טוֹבָה בְּיוֹם טוֹב – // וְיוֹם צוּקָה יְהִי חָבֵר לְצוּקָה,

אֲשֶׁר אִם תִּשְׁאֲלוֹ רָקִיק לְאָכְלָה // אֲזַי יִתֵּן בְפָנֶיךָ רְקִיקָה;

וְתֹאמַר לוֹ לְהַנְחוֹתָךְ בְּעֵמֶק – // וְיַשְׁלִיכָךְ בְּתוֹךְ שׁוּחָה עֲמֻקה;

וְאִם תַּדְלִיק אֲבוּקָה לָךְ – יְזוֹרֵר // עֲלֵי אוֹרַת אֲבוּקָתָךְ אֲבָקָה.

וְאַתָּה תֶּאֱחֹז לֵב טוֹב בְּרֵעָךְ // וְעַל שׂוֹנַאֲךָ יָדָךְ חֲזָקָה,

וְאִם תִּשֹּׁךְ – נְשִׁיכָתָךְ נְשִׁיכָה, // וְאִם תִּשַּׁק – נְשִׁיָקָתְך נְשִׁיקָה.

וְשָׁמַעְתִּי דְּבָרָךְ בַּאֲנָשִׁים // אֲשֶׁר אֵין לָךְ וְלִי בָהֶם חֲשִׁיקָה,

נְחָשִׁים הֵם בְּעֵינֶיךָ וְעֵינָם – // וְהֵם אֶצְלִי כְּתוֹלַעַת יְרֻקָּה;

חֲמָדוּנִי וְיָעִיקוּ, לְהוֹדִי // וְהוֹנִי, כַּעֲגָלָה הַמְּעִיקָה,

וּמַה־יוּכַל עֲשׂה תַיִשׁ לְלַיִשׁ // וּמַה כֹּחַ בְּכֹחַ וַאֲנָקָה?

יְשַׁלַּח אֵל עֲלֵי אִישִׁים זְכַרְתָּם // אֲשֶׁר שָׁלַח עֲלֵי נָסֵי עֲזֵקָה,

וְיַכְלִימֵם בְּתוֹךְ אֶרֶץ, וְיֵרְדוּ // מְכֻסִים בַּכְּלִימָה אֶל עֲלוּקָה!

אֲנִי אָעִיד בְּצָרָתְךָ לְצִירִי // וְכִי תָחִיל בְּעֵת אֶנְאַק נְאָקָה,

וְאַצְדִּיק, עַל אֲשֶׁר בָּא, אֵל אֱמוּנָה, // כְּמוּסָרָךְ, בְּעִתּוֹתֵי זְעָקָה,

וְהָרָעָה אֲשֶׁר הֵבִיא יְשַׁו עַל // מְגִלַּת פָּעֳלִי הַטּוֹב חֲקוּקָה!

יְגוֹנִי עַל אֲשֶׁר מֵתוּ, וּמָרָה, // לְבַד מוֹתָם, בְּפִי כִדְבַשׁ מְתוּקָה,

וְאֶשְׂמַח כִּי אֲשֶׁר לֻקַח – בְּכָבוֹד // נְתַתִּיהוּ, וְלֹא הָיָה לְפוּקָה;

חֲשַׁבְתִּיו עַל שְׁנֵי גַבַּי כְּמַשָּׂא – // וְכָל טָעוּן מְנוּחָתוֹ פְּרִיקָה.

וְסָרָה אַחֲרֵי זֹאת כָּל תְּבִיעָה // וְהָיְתָה בָּהּ לְמַשְׂטִינַי שְׁתִיקָה,

וְכָל בַּיִת יְפֻנֶּה מִשְּׂאוֹר, לֹא // תְּהִי אַחַר בְּדִיקָתוֹ בְּדִיקָה;

וכָל גִּוְיָה, בְּהִתְגַּלַּח, שְׂעִירָה // תְּהִי, אַחַר אֲשֶׁר הָיְתָה חֲלָקָה,

וְיֵשׁ פֶּרִי בְּבֶן כֶּרֶם, וְיוֹסֵף // פְּרִי, אַחַר זְמִירָה וַעֲזָקָה,

וְאוֹר פָּנַי אֲזַי יִרְבֶּה, כְּהַרְבּוֹת // מְאוֹר בַּיִת, בְּהִקָּצֵץ, אֲבוּקָה.

כְּבָר הֵפִיג דְּבָרָךְ אֶת יְגוֹנִי, // וְאֵל יַחְשֹׁב לְךָ אֶת זֹאת צְדָקָה,

וְיֵיטִיב בָּאֲדָמָה אַחֲרִיתָךְ // וְלֹא יַשְׁמִיעֲךָ קוֹל צְעָקָה.





קְרָא, אָחִי, תְּשׁוּבָה לָךְ שְׁלוּחָה // בְּחִפָּזוֹן, כְּמוֹ עָלָה דְפוּקָה,

וּמִי יָכִין כְּשִׁירָתָךְ, אֲשֶׁר מֹר // וְשֶׁמֶן טוֹב עֲלֵי רָאשִׁים מְרִיקָה?

וְכָל דַּל מִזְּבֹחַ פַּר וְכֶבֶשׂ // יְהִי נֶעְתָּר בְּתוֹרֵי הַמְּלִיקָה!
