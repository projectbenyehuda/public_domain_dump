


	
	[אהה שומרים]
	








∪ — — — ∪ — — — ∪ — — — ∪ — — —



אֲהָה, שׁוֹמְרִים // מְצָאוּנִי1 // וְהִכּוּנִי, // פְּצָעוּנִי!



אֲהָהּ, שָׂרִים אֲהֵבוּנִי2 — // בְּיַד צָרִים3 //
עֲזָבוּנִי,



טְפָחוּנִי // וְרִבּוּנִי4 —
// וְלֹא שָׁעוּ // לְעֶלְבּוֹנִי5,



וְהִשְׁקוּנִי // וְכִסּוּנִי6 — // וְסוֹף דָּבָר7 //
מְאָסוּנִי8!









שומרים מצאוני והכוני פצעוני– בשוטטי
בלילה לבקש את אהובי. על פי שיר השירים ה ז: מצאוני השומרים הסובבים
בעיר הכוני פצעוני.  ↩



אהבוני – אשר אהבוני.  ↩



צרים – השומרים שהכוני.  ↩



טפחוני ורבוני – לפנים. על פי איכה ב כב:
טפחתי ורביתי.  ↩



ולא שעו לעלבוני – עתה.  ↩



והשקוני וכסוני – לפנים. על פי שופטים ד יט:
ותשקהו ותכסהו; יחזקאל טז י: ואכסך משי.  ↩



לסוף דבר –ולבסוף.  ↩



מאסוני – על פי ירמיה ד ל: מאסו בך
עוגבים.  ↩
