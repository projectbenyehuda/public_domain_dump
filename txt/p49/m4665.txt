


	
	זמרה
	




וְלוֹ עוֹד, יְכַבְּדֵהוּ הָאֵל, בְּמִלְחֶמֶת אִשְׁבִּילִיָּה וְשָׂרְפוּ הַר הַזֵּיתִים. וְאַחַר כֵּן עָבַר עַל אָשׂוּנָה וְאֶצְטֶפָה וְעֵמֶק אוּבֵּדָה. וְהָיָה זֶה בְּסִיוָן שְׁנַת תת"ח [4808; מַאי-יוּנִי 1048].


  


חַיִּים בְּלֹא רִישׁ וְתַחְלוּא / טוֹבִים לְנַפְשִׁי, וְנִפְלוּ,



לוּלֵי זְקוּנִים, וְאֵין לִי / הוֹעִיל בְּ’לוּלֵי' וְלֹא ‘לוּ’



נִרְאוּ לְבָנוֹת בְּרֹאשִׁי, / עָצְמוּ כְמוֹ חוֹל, וְגָדְלוּ,



נִרְאוּ לְעֵינַי הֲלִיכַי / מֵהַר לְשַׁחַת, וְנִגְלוּ,



הָיוּ שְׁחֹרוֹת חֲדָשׁוֹת / הוֹלְכוֹת בְּגֵאָה – וּבָלוּ,



כָּכָה יְבַלּוּ יְמוֹתַי / תֵּבֵל וְחוּגָהּ, וְיִכְלוּ.



חַיַּי, אֲשֶׁר תַּם וְגָמַל / בִּסְרָם כְּאֶשְׁכּוֹל, וְנָבְלוּ,



שׁוּבוּ לְאָחוֹר וְהוֹצִי- / אוּ צִיץ נְעוּרִים, וְגִמְלוּ,



גִּזְלוּ יְמֵי עַד אֲשֶׁר מִ- / כֶּם הֶעֱבִירוּ וְגָזְלוּ,



אִם לֹא – דְּעוּ מַה לְּמָחָר / יֵתֶא עֲלֵיכֶם, וְחִדְלוּ.



מַה יַּעֲשֶׂה אִישׁ, יְרַמּוּ / אוֹתוֹ זְמָמָיו,וְיַשְׁלוּ,



כִּי יַעֲמֹד אֶת פְּנֵי אֵל / כָּבֵד בְּרִשְׁעָה וּמָלוּא?



אַשְׁרֵי אֲנָשִׁים יְעָצָם / לִבָּם בְּלַיִל, וְשָׂכְלוּ,



עָשׂוּ פְלִילָה, וְעִשּׂוּ / יִצְרָם בְּכֹח, וְיָכְלוּ.



לוּלֵא אֲשִׁישׁוֹת, לְקָחוּם / נֶגְדִּי עֲלָמוֹת, וּמָלוּ –



לָקְחוּ יְגוֹנִים לְבָבִי / מִבֵּין כְּסָלַי, וְנָשְׁלוּ.



יָפוּ בְאָמְרָם: גְּלוּ מִן / לֵבָב תְּיַגּוּ, וְגָלוּ –



לוּלֵי קְרָב, כִּי שְׁאוֹנָיו / שָׁכְנוּ בְלִבִּי, וְגָבְלוּ,



שָׁלְטוּ תְשׁוּאוֹת קְרָב עַל / זִמְרַת זְמוֹרָה, וּמָשְׁלוּ,



שָׁמְעוּ רְנָנוֹת בְּמִשְׁתֶּה / רִנַּת קְרָבוֹת, וְחָלוּ.



לוּלֵי פְלָאוֹת, בְּצַר לִי / יָרְדוּ כְמַלְקוֹשׁ, וְנָזְלוּ –



הוֹרִישׁ זְמַנִּי כְבוֹדִי / זָרִים וְסוּרִים, וְנָחְלוּ.



בָּזוּ מְשַׂנְאִים בְּרִית בֵּ- / נֵינוּ וּבֵינָם, וּמָעְלוּ.



בָּאנוּ בְאָלָה, אֲבָל עָ- / מַדְנוּ בְאָלָה – וְנָכְלוּ.



עִוּוּ נְתִיבָם, וְעָנָה / פִּיהֶם הֲתוּלִים, וְהָבְלוּ.



הִרְבּוּ לְדַבֵּר גְּבוֹהָה / בִּי עַל לְשׂוֹנָם, וְרָגְלוּ.



קַמְנוּ אֲלֵיהֶם בְּאִישִׁים / יִשׂאוּ כְשֶׁטֶף, וְיַעְלוּ,



כֻּלָּם בְּכֶתֶר וּמִכְלוֹל / נִלְחָם, וְרָקוּם וְטָלוּא,



קָמוּ בְחֶרֶב שְׁלוּפָה / וּבְשֵׁם אֱלֹהִים, וְדָגְלוּ,



לָבְשׁוּ חֲנִיתָם, וּבָאוּ / בִּגְבוּל יְהִירִים, וְקָטְלוּ,



תָּקְעוּ חֲנִיתוֹת בְּלִבּוֹת / רָאשֵׁי סְרָנִים, וְשָׁתְלוּ,



קָצְרוּ בְּבַרזֶל גּוִיּוֹת / בָא עֵת קְצִירָם וּבָשְׁלוּ,



הָרְגוּ אֲנָשִׁים, וְשָׁבוּ / נָשִׁים, וְכִלּוּ וְאָכְלוּ,



דָּלְקוּ בְזֵיתִים בְּגִבְעַת / זֵיתִים, וְגָדְעוּ וְקָמְלוּ,



תִּרְאֵם כְּאִישׁים שְׁחֹרִים / הֻכּוּ בְחֶרֶב וְנָפְלוּ,



הוֹבִישׁ נְסִיכָם וְסָגַר / פִּתְחֵי שְׁעָרָיו כְּכָלוּא,



רָאוּ מְשַׂנְאִים פְּתָאיִם / מַה בָּא עֲלֵיהֶם – וְאָבְלוּ,



הָיוּ בְלִבָּם גּבוֹהִים / וּבְנֵי אֱלֹהִים – וְשָׁפְלוּ,



חָזוּ הֲמוֹנִים הֲמוֹנִים / חָנוּ כְאַרְבֶּה וְחָסְלוּ,



הָיוּ לְבוּזָה בְאַרְצָם / וּבְכָל אֲרָצוֹת, וְנִקְלוּ,



עָבְרוּ וְחָלְפוּ כְעָבִים / הַבְלֵי לְשׁוֹנוֹת, וְאָזְלוּ,



חָרוּ פְעָלִים וְהִמְסוּ / כָּאֵשׁ דְּבָרִים, וּבָטְלוּ,



עָמְדוּ עֲזוּרִים בְּשָׂשׂוֹן / עַל מֵי יְשׁוּעוֹת, וְדָלוּ,



בִּדְמֵי רְשָׁעִים הֲלוּמִים / רָחְצוּ פְעָמִים, וְטָבְלוּ.


  


אַחַי, אֲשֶׁר עַל יְשׁוּעָ- / תִי שָׂשׂ לְבָבָם וְגָלוּ –



עַל טוּב כְּפָלוֹ אֱלֹהִים / אִמְרוּ זְמִירוֹת וְכִפְלוּ.



קוּמוּ קְחוּ תֹּף וְשִׁירוּ / לָאֵל בְּלֵב טוֹב, וְחוּלוּ.



מֵאֵל עֲשׂה לִי כְמוֹ זֹאת / תָּמִיד בְּכָל יוֹם שְׁאָלוּ!
