


	
	אָנָּא אֲדֹנַי הוֹשִׁיעָה
	



[ראש חודש]






אָנָּא אֲדֹנַי הוֹשִׁיעָה / וְהַצְמִיחַ קֶרֶן יְשׁוּעָה

וְהָחִישׁ שְׁנַת הָקֵם שְׁבוּעָה / כִּי לַיוי הַיְשׁוּעָה



בָּךְ אֵל בָּטְחוּ אֲבוֹתֵינוּ / וְלָךְ זָעֲקוּ וְנִמְלָטוּ

וְאֵיךְ עַד כֹּה בְגָלוּתֵנוּ / נַפְשׁוֹתֵינוּ נָקָטוּ

רְאֵה כִי מֵאֹרֶךְ שְׁבִיתֵנוּ / כִּמְעַט אֲשׁוּרֵינוּ מָטוּ

קוּם וְהָרֵם יָד הַיְדוּעָה / וַחֲשׂוֹף זְרוֹעַ הַיְשׁוּעָה

וְהָמִישׁ יָתֵד הַתְּקוּעָה / וּלְבַשׁ כּוֹבַע הַיְשׁוּעָה:



רְאֵה כִי גָבְרָה יַד אָמָה / וְיַד בַּת גְּבֶרֶת שָׁפְלָה

הַפִּיל עַל הֲמוֹנָהּ אֵימָה / וְתֵבוֹשׁ מֵאֲשֶׁר הוֹחִילָה

וְהָחִישׁ לַעֲשׂוֹת בָּהּ נְקָמָה / כִּי עַל יוי הִגְדִּילָה

וְלָהּ הַשְׁמִיעַ קוֹל זְוָעָה / וְלִי הַשְׁמִיעַ קוֹל יְשׁוּעָה

מִפִּי מְבַשֵּׂר בְּקוֹל תְּרוּעָה / טוֹבָה, וּמַשְׁמִיעַ יְשׁוּעָה:



הָאֵל אֲשֶׁר נָתַן לְעַמּוֹ / רָאשֵׁי חֳדָשִׁים לְכַפָּרָה

לִהְיוֹת מַקְרִיבִים לִשְׁמוֹ / בָּם זִבְחֵי רָצוֹן בְּטַהְרָה

יִרְצֵנוּ וְיָסִיר זַעְמוֹ / וְיִהְיֶה עָלֵינוּ לְסִתְרָה

וְיִשְׁמְרֵנוּ מִכָּל רָעָה / וְיִהְיֶה לָנוּ לִישׁוּעָה

וְיִבְנֶה עִירוֹ הַבְּקוּעָה / וְיָשִׁית חוֹמוֹת וְחֵל יְשׁוּעָה:



מָתַי בְּצִיּוֹן מִזְבֵּחַ / תָּכִין, וְשָׁם נַעֲלֶה עָלָיו

עוֹלַת רֹאשׁ חֹדֶשׁ לְרֵיחַ / וְלֵוִי בְשִׁיר יִשָּׂא כֵלָיו

וְאָז נָשִׁיר בְּלֵב שָׂמֵחַ / עַל בִּנְיַן דְּבִיר וְאֹהָלָיו

יוֹם תִּהְיֶה עֵדָה פְּדוּעָה / כְּסוּיָה בְצֵל עֹז וִישׁוּעָה

וּמַלְכוּת עֲרָב תִּהְיֶה קְרוּעָה / וְאֵין לָהּ כְּחֹם הַיּוֹם יְשׁוּעָה:



מֶלֶךְ אֲשֶׁר לוֹ הַגְּדֻלָּה / וּמְרוֹמַם עַל כָּל תְּהִלָּה

קְרָא שְׁנַת רָצוֹן וּגְאֻלָּה / לְאוּם נוֹשֵׂאת נֶגְדָּךְ תְּפִלָּה

וְהָרֵם קַרְנָהּ כְּבַתְּחִלָּה / וְסַקֵּל לַעֲלוֹתָהּ מְסִלָּה

וּקְרָא לָהּ בְּיוֹם בּוֹא שְׁמוּעָה: / שִׂישִׂי, כִּי קָרָאת יְשׁוּעָה

חוֹמוֹתַיִךְ וְאוֹרָה זְרוּעָה, / קוּמִי לִבְשִׁי לְבוּשׁ יְשׁוּעָה:
