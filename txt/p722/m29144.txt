


	
	שׂוֹשׂ אָשִׂישׂ בְּיֶשַׁע הָאֵל
	



[מילה]






שׂוֹשׂ אָשִׂישׂ בְּיֶשַׁע הָאֵל / אֶתֵּן לוֹ תְהִלָּה

כִּי בָחַר בְּבֵית יִשְׂרָאֵל / וּקְרָאָם סְגֻלָּה:





מֵרוֹב אַהֲבָתוֹ אוֹתָם / שָׂם אוֹת עַל בְּשָׂרָם

צֶאֱצָאָם בְּמִילָה חָתַם / עֵדוּת עַל שְׁאֵרָם

מֵרֹאשׁ נִתְּנָה לָאֲבוֹתָם / כִּי צוּרָם בְּחָרָם

נִקְרָא אַב הֲמוֹן בָּהּ תָּמִים / וַיּוֹסֵף גְּדֻלָּה

גַּם מָצָא חֲתַן הַדָּמִים / בָּהּ רִפְאוּת תְּעָלָה:





עַם קֹדֶשׁ בְּזֹאת הִתְאַחָד / הִיא עָלָיו לְעֵדָה

כִּי עוֹבֵד אֱלוֹהַּ אֶחָד / מַחֲזִיק בַּתְּעוּדָה

מוֹאֵס בָּהּ לְעוֹלָם נִכְחָד / תֹּאחֲזֵהוּ רְעָדָה

כִּי הִנֵּה יוי פּוֹקֵד / עַל כָּל מוּל בְּעָרְלָה

וּלְהָרַע נַפְשׁוֹ שׁוֹקֵד / עַד רִדְתּוֹ שְׁאוֹלָה:





כָּל עוֹמֵד וּמַחֲזִיק בִּבְרִית / מִילָה הַחֲשׁוּקָה

עָרֵב לוֹ מְגַלֶּה אַחֲרִית / בַּל יֵרֵד עֲלוּקָה

גַּם יִהְיֶה לְשֵׁם וּשְׁאֵרִית / וּתְהִי לוֹ צְדָקָה

כִִּי בָהּ יֵרְדְּ עַל תַּאֲוָתוֹ / וּתְהִי לוֹ נְקָלָה

וְלָאֵל תִּהְיֶה אַהֲבָתוֹ / בָּהּ אַהֲבָה מְגֻלָּה:





יַעַן כִּי חֲמוּדָךְ מַלְתָּ / בַּיּוֹם הַשְּׁמִינִי

וּבְאַהֲבַת לְבָבָךְ חַשְׁתָּ / לַעֲשׂוֹת אֶת רְצוֹנִי

אָפִיק לָךְ אֲשֶׁר שָׁאַלְתָּ / וּבָךְ יַד יְמִינִי

תַּחֲזִיק, וְאֶהִי לָךְ סוֹמֵךְ / וּבְעֵין חֵן וְחֶמְלָה

אֵפֶן לָךְ וְאֶהְיֶה תוֹמֵךְ / גּוֹרָלָךְ בְצַהְלָה:





שִׂישׂוּ הַחֲרֵדִים אֵלַי / הַבָּאִים בְּתוֹדוֹת

הֵן אָחִישׁ נְבוּאוֹת אוּלַי / חָזָה אִישׁ חֲמוּדוֹת

רַגְלֵיכֶם בְּתוֹךְ הֵיכָלַי / יִהְיוּ מָעֳמָדוֹת

וּבְפִתְאוֹם יְמַהֵר מוֹרֶה / צֶדֶק בִּגְאֻלָּה

עֵת כִּי תִשְׁמְעוּ קוֹל קוֹרֶא: / סֹלוּ הַמְסִלָּה:
