


	
	לִקְרַאת דּוֹד אֲשֶׁר זָרַח
	



[שבח]






לִקְרַאת דּוֹד אֲשֶׁר זָרַח / וּבָא מִירוּשָׁלַיִם

צְאֶינָה כָּל בְּנוֹת עֵדֶן / בִּמְחוֹלוֹת וּמַחֲנַיִם:







אַחַר הֲלִיכָיו רוּצוּ / וְהִסְתּוֹפְפוּ בְּאֶבְרָתוֹ

וּבִמְאוֹר פָּנָיו עִלְצוּ / וְהֵאָחֲזוּ בְחֶבְרָתוֹ

כִּי פְעָלָיו כְּבָר נִרְצוּ / וְאֵל עֶלְיוֹן סִתְרָתוֹ

וְאוֹרוֹ עָלָיו צָלַח / כִּי הוּא נְקִי כַפַּיִם

וּטְהָר לֵב מֵרְאוֹת רָע / וְלוֹ חֵן בְּשִׂפְתוֹתָיִם:





בְּצַעֲדוֹ בְתוֹךְ מִצְרַיִם / תָּמְהוּ עָלָיו רוֹאִים

כִּי רָאוּ עֵינַיִם / מִצִּדְקוֹתָיו פְּלָאִים

וְנוֹא הָבְקְעָה לִשְׁנַיִם / וְהָיוּ מֶנְהוּ יְרֵאִים

כִּי רָאוּ אוֹרוֹ צַח / מִיָּדוֹ קַרְנַיִם

וְחָזוּ הַלְּבָבוֹת אֲשֶׁר / לֹא שָׁמְעוּ אָזְנָיִם:





רַבָּה תִפְאַרְתְּךָ עוֹד / בְּלֶכְתְּךָ לְמִקְדְּשֵׁי אֵל

מְקוֹם הַנְּבוּאָה וְהַהוֹד / אֲשֶׁר הָיוּ לְיִשְׂרָאֵל



וְקִבְרוֹת כָּל נָבִיא וְדוֹד / וּנְוַת צִיּוֹן וְ[הַ]הַרְאֵל



וּמוּסָד מוּסָּד נוֹכַח / שַׁעַר הַשָּׁמַיִם

אֲשֶׁר בּוֹ נִגְלוּ אֱלֹהִים / לְכָל זֶרַע אֶפְרָיִם:





הָרֵם שְׁלֹמֹה יָדְךָ / וְקוּם מַלֵּא רְצוֹנְךָ

כִּי אֵל שׁוֹמֵר צַעְדְּךָ / וְהוּא תוֹמֵךְ יְמִינְךָ

וְדַע כִּי כָל עַם סוֹדֶךָ / שְׂמֵחִים בְּאוֹר פָּנֶיךָ

כִּי אַתְּ כְּשׁוֹשָׁן פָּרַח / בְּתוֹךְ גַּן רְחַב יָדַיִם

וּבְאוֹרְךָ שָׁבוּ כָל / מְעוֹנוֹת כְּמוֹ צָהֳרָיִם:





מַה נָּעִים יוֹם שׁוּבְךָ / כַּאֲשֶׁר מַר יוֹם רָחְקְךָ

הָאֵל יַרְבֶּה טוּבְךָ / וִימַלֵּא אֶת חִשְׁקְךָ

וְיִקְרָא לְךָ מִסְּבִיבְךָ / שָׁלוֹם, וְיַצְמַח צִדְקְךָ

וְיָשִׁיב דּוֹד אֲשֶׁר בָּרַח / אָדוֹם יְפֵה עֵינַיִם

וְאָז נַעֲלֶה לָחוֹג / בְּתוֹכְכֵי יְרוּשָׁלָיִם:
