


	
	יוֹנָה מַה לָּךְ הוֹמִיָּה
	








&amp;&amp;&amp; יוֹנָה מַה לָּךְ הוֹמִיָּה

סימן: ישראל





יוֹנָה מַה לָּךְ הוֹמִיָּה / עַל נִזְרֵךְ לָאָרֶץ הֻשְׁלָךְ

וּלְקִנֵּךְ יִשְׁכֹּן דַּיָּה / וְכוֹס וּקְאַת וְשָׁלָךְ

מִנְעִי עֵינֵךְ מִבִּכְיָה / כִּי אֵל עָל עָלַיִךְ מָלָךְ

פִּצְחִי שִׁיר יְפֵיפִיָּה / כִּי הִנֵּה מַלְכֵּךְ יָבוֹא לָךְ.





שַׂמְּחִי אֶת לֵב נִשְׁבָּר / אֲשֶׁר בַּשִּׁבְיָה נִתְעַלָּף

הַלְבִּישִׁי סוּת זַךְ וּבָר / אֶת בַּר בִּלְבוּשׁ שַׂק נִתְעַלָּף

הִנֵּה הַסְּתָו עָבַר / הָלַךְ וְהַגֶּשֶׁם חָלָף

נָא קוּמִי לָךְ בַּת נָבָר / רַעְיָה יָפָתִי וּלְכִי לָךְ.





רָאִיתִי עֳנִי עַמִּי / וְיָדַעְתִּי אֶת מַכְאוֹבָיו

לֹא יִתְּנוּ לוֹ דֳּמִי / וְיָסֹבּוּ עָלָיו רַבָּיו

חַכִּי לִי לְיוֹם קוּמִי / לְעַד, לָרִיב אֶת יְרִיבָיו

אָשׁוּבָה אֶל מְקוֹמִי / וְשָׁם אֶתֵּן אֶת דּוֹדַי לָךְ.





אֶל קִנֵּךְ עוּפִי יָפָה / וְעַל גּוֹזָלַיִךְ נוּחִי

מַחַץ מַכָּתֵךְ אֶרְפָּא / וּבְשִׁבְיָה עוֹד לֹא תֵאָנְחִי

לֹא תִהְיִי עוֹד לְחֶרְפָּה / וּלְמוֹנַיִךְ מָאוּס וּסְחִי

אַךְ תִּרְדִי בָם וְאַפַּ/יִם אֶרֶץ יִשְׁתַּחֲווּ לָךְ.





&amp;&amp;&amp; יָפָה אַתְּ יַעֲלָה

סימן: ישראל





יָפָה אַתְּ יַעֲלָה / עֵינַיִךְ יוֹנִים

שִׂפְתּוֹתַיִךְ כַּלָּה / כְּמוֹ חוּט שָׁנִים

פִּצְחִי שִׁיר תְּהִלָּה / בְּתֹף וּמִנִּים

נַעַל בַּמְּסִלָּה / לִרְעוֹת בַּגַּנִּים.





שְׂעַר רֹאשֵׁךְ דָּמָה / תִּקְוַת חוּט זָהָב

לֶחְיֵךְ כְּאוֹר חַמָּה / בְּקִרְבִּי לַהַב

קִרְבָתֵךְ אֲיֻמָּה / לִבִּי יֶאֱהַב

דּוֹדַיִךְ, נְעִימָה / מֵרֹב פְּנִינִים.





רִשְׁפֵּי אַהֲבָתֵךְ / כְּלַפִּיד יִבְעָר

וּבְיָם תְּשׁוּקָתֵךְ / לְבָבִי יִסְעָר

הֵן נְוַת דּוֹדִים נְוָתֵךְ / צְאִי מִשִּׁנְעָר

שׁוּבִי לִמְנוּחָתֵךְ / שִׁכְחִי יְגוֹנִים.





אֶל נֹכַח פָּנַיִךְ / טוּבִי אַעֲבִיר

וּבְנֵי שִׁכֻּלָיִךְ / עַל אֱדוֹם אַגְבִּיר

אֶרְעֶה עֲדָרַיִךְ / אֶבְנֶה הַדְּבִיר

אָשִׁיבָה שׁוֹפְטַיִךְ / כְּיָמִים קַדְמוֹנִים.
