


	
	אֲעוֹרֵר בְּשִׁירָה
	



להרב הכולל מוהר״ר משה חוצן בן לאותו צדיק מר זקנינו הרב הגדול ראש
גולה שבבבל מוהר״ר צדקה חוצן ז״ל.

סימן: אני משה חוצין.


אֲעוֹרֵר בְּשִׁירָה / בְּרֶתֶת וּמוֹרָא / לְאֵל רָם וְנוֹרָא / וְלוֹ עֹז וּמִשְׂרָה.



נְעִים שִׁיר יְהַלֵּל / לְצוּר רַב מְחוֹלֵל / לְפָנָיו אֲמַלֵּל / בְּשָׂפָה בְרוּרָה



יְבָרֵךְ לְעַמּוֹ / בְּרֹב עֹז שְׁלוֹמוֹ / וְיִבְנֶה הֲדוֹמוֹ / בְּאֶבֶן יְקָרָה



מְעוֹדֵד עֲנָוִים / סְחוּפִים דְּווּיִים / אֲשֶׁר הֵם מְקַוִּים / לְטוֹבָה בְשׂוֹרָה



שְׁמֵי רוֹם רְכֻבּוֹ / וְיָשָׁר נְתִיבוֹ / מְכַלְכֵּל בְּטוּבוֹ / לְכָל נֻצָּר נִבְרָא



הֲלֹא כָל צְבָאָיו / יְעִידוּן פְּלָאָיו / אֲשֶׁר חֵי בְרוּאָיו / בְּיָדוֹ מְסוּרָה



חֲסָדָיו אֲצַפֶּה / מְהֻלָּל בְּכָל פֶּה / לְצִירִי יְרַפֵּא / יְהִי לִי לְעֶזְרָה



וְיִשְׁלַח מְנַחֵם / לְעַמּוֹ יְרַחֵם / וְאוֹתָם יְנַחֵם / בְּבֵית הַבְּחִירָה



צְמָאִי יְרַוֶּה / יְשׁוּעוֹת יְצַוֶּה / וְהַר טוֹב וְנָוֶה / יְכוֹנֵן מְהֵרָה



יְקַבֵּץ נְפוּצוֹת / וְנִשְׁמַע בְּחוּצוֹת / צְבִי כָל אֲרָצוֹת / נְעִים שִׁיר וְזִמְרָה



נְשָׁמָה בְּעוֹד בִּי / בְּקוּמִי וְשָׁכְבִי / אֲהַלֵּל בְּנִיבִי / לְאֵל כִּי יְצָרָהּ



וְאֶשָּׂא לְכַפִּי / וְאָשִׁיר בְּמוֹ פִי / לְאֵל שָׂם בְּאַפִּי / נְשָׁמָה טְהוֹרָה.
