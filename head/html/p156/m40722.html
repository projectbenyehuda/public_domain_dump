[נולד ברוֹמָא לערך 1272–1268, חי ברוֹמָא, פַּרוּגָ’ה, פֶֿרְמוֹ, פַֿבְּרִיָּנוֹ, קָמַרִינוֹ, אַנְקוֹנָה ווַרוֹנָה, מת אחרי שנת 1328]


הַיְרֻשָּׁה (הַשַּׁעַר הַי"ד מִסֵּפָר הַמַּחְבָּרוֹת)


אָמַר עִמָּנוּאֵל הַמְחַבֵּר: בִּהְיוֹתִי בְּבֵית הַשָּׂר שָׁמַן וְדָשֵׁן / שׁוֹכֵב עַל־מִטּוֹת שֵׁן / וּבְמַעֲדַנָּיו אֲשַׁלֵּחַ הַשֵּׁן / הָיִיתִי בְחוּצוֹת הַשְּׂמָחוֹת מְגַמֵּא / וְסוֹבֵב כָּל־מַעְיְנֵי הַשְּׂשׂוֹנִים סִבּוּב הַצָּמֵא / וַיְהִי הַיּוֹם וַאֲנִי אֶת־רַגְלִי חָלִיתִי / וְעָמַדְתִּי יָמִים שִׁבְעָה וּפְנֵי הַשָּׂר לֹא־רָאִיתִי / וְיַיִן וְשֵׁכָר לֹא־שָׁתִיתִי / וְכַאֲשֶׁר הִתְחַזַּקְתִּי וְיָצָאתִי מִפֶּתַח בֵּיתִי / וָאֶשָּׂא עֵינֵי וָאֶרְאֶה וְהִנֵּה אֵם הַשָּׂר בָּאָה לִקְרָאתִי / וָאָרוּץ לִקְרָאתָהּ וָאֹמַר “הֲשָׁלוֹם לָךְ גְּבִרְתִּי / הֲשָׁלוֹם לַשָּׂר הֲשָׁלוֹם לַיְלָדִים כִּי בָאת הֲלוֹם” וַתֹּאמֶר “שָׁלוֹם / אָמְנָם הַשָּׂר שְׁמוּעָה שָׁמַע וְחָרֵד חֲרָדָה / וְצִירִים אֲחָזוּהוּ כְּצִירֵי יוֹלֵדָה / סוֹעֵר כַּהֲמוֹת הַיָּם לְגַלָּיו / וּשְׁנָתוֹ נִהְיְתָה עָלָיו / וְעַתָּה הַדּוֹד הַנֶּאֱמָן / הַחוֹמֵל עָלָיו וְהָרַחְמָן / לָמָּה תַעֲמֹד בְּרָחוֹק וְתַעֲלִים לְעִתּוֹת בַּצָּרָה / וְהַיְגוֹנִים / חוֹנִים / עַל־הַשָּׂר אִישׁ עַל־דִּגְלוֹ בְאֹתֹת / בֹּאָה אֵלָיו הַחַדְרָה וְנַהֲלֵהוּ עַל־מֵי מְנוּחֹת / וְיִהְיֶה־לוֹ בִּרְאוֹתוֹ פָנֶיךָ שׂבַע שְׂמָחוֹת / וְשָׂא אֵלָיו חִידוֹתֶיךָ הַמְּשַׂמְּחוֹת / יְכַבּוּ אֵשׁ הַיְגוֹנִים / וְיָאִירוּ נֵרוֹת הַשְּׂשׂוֹנִים”.

וָאַעַן וָאֹמַר “הִנְּנִי גְּבִרְתִּי” / וָאָקוּם קַל חִישׁ נִשְׁעַן עַל־מִשְׁעַנְתִּי / וָאֵלֵךְ אֶל־הַשַּׂר חֶדֶר בְּחֶדֶר וּמְצָאתִיו הוֹמֶה וְסוֹעֵר / וּבְכָל מַחֲמַדֵּי הַזְּמָן גּוֹעֵר / וְלִבּוֹ בְּאֵשׁ הַיְגוֹנִים בּוֹעֵר / וְעִמּוֹ חֶבְרַת מְיֻדָּעָיו / וְגוֹאֲלָיו וְרֵעָיו / וּבִרְאוֹתָם אוֹתִי מֵרָחוֹק / וּבְיָדִי הַמִּשְׁעֶנֶת הֵחֵלּוּ לִשְׂחֹק / וַיֹּאמְרוּ “מִי הֱבִיאֲךָ הֲלוֹם וְאַתָּה צָרִיךְ לְמִשְׁעֶנֶת” / וָאַעַן וָאֹמַר “אַהֲבַת הַשַּׂר הַנֶּאֱמָנֶת” / וַיֹּאמְרוּ “הֲיָדַעְתָּ חִידָה מְשַׂמָּחַת / תִּהְיֶה אֶל־הַשָּׂר בִּשְׁמֹעַ אוֹתָהּ נָחַת / וְלֹא יָמוּת לַשָּׁחַת” / וָאַעַן וָאֹמַר “יָדַעְתִּי וְלֹא אוּכַל לְהַגִּידָה / וְעַתָּה חוּסוּ עַל־נַפְשִׁי הָאוֹבֵדָה / לְהָכִין אֹתָהּ וּלְסַעֲדָהּ / הֲטֶרֶם תֵּדְעוּ כִּי אָבָדָה / הֲלֹא תִרְאוּ עֲצָמַי אֲשֶׁר הִתְדַּלְדָּלוּ / וְהָעֲגָבוֹת הִתְקַלְקָלוּ / וְאַתֶּם אֶת־הַחִידָה תִּשְׁאָלוּ / לָמָּה־זֶה הֶבֶל תֶּהְבָּלוּ / הָבוּ שְׂכָרִי וְאִם־לֹא חֲדָלוּ” / וַיִּשְׂחֵק הַשַּׂר וַיֹאמַר “וּמַה־יִהְיֶה הַשָּׂכָר” / וָאַעַן וָאֹמַר “אַטִּף לְךָ לַיַּיִן וְלַשֵּׁכָר” / וַיְצַו הַשַּׂר הַנֶּעֱרָב / וַיָּבֵא יֵין מַלְכוּת רָב / וָאֹמַר “בָּרוּךְ יְיָ צוּרִי הַמְּלַמֵּד יָדַי לַקְרָב” / וָאָקוּם וָאֶשְׁתֶּה יַיִן לְשָׂבְעָה / כִּי לֹא שָׁתִיתִי מִמֶּנּוּ יָמִים שִׁבְעָה / וַיְהִי כַּאֲשֶׁר שָׂבַעְתִּי מִן־הַיָּיִן / הָיוּ יְגוֹנֵי הַשַּׂר בְּעֵינָיו אָיִן / וַיֹּאמַר “עַד־מָתַי חִידָתְךָ בְּתוֹךְ חִכְּךָ תִּמְנָעֶנָּה / חוּדָה חִידָתְךָ וְנִשְׁמָעֶנָּה” / וָאַעַן וָאֹמַר "הַאֲזִינוּ מְיֻדָּעַי וַאֲדַבֵּרָה / וְזֶה־חָזִיתִי וַאֲסַפֵּרָה:

אִישׁ הָיָה בְּאֶרֶץ רוֹמָא מֹשֶׁה שְׁמוֹ / רָצוּי לְרוֹב אֶחָיו וְדֹרֵשׁ טוֹב לְעַמּוֹ / וְשֵׁם אִשְׁתּוֹ רוֹשׂוֹלָה / אֲשֶׁר יָרַד עָלֶיהָ אָבֵל שְׁאֹלָה / וְשֵׁם בְּנוֹ דָּנִיֵּאל אֱוִיל שְׂפָתָיִם / חֲמֹר גָּרֶם רֹבֵץ בֵּין הַמִּשְׁפְּתָיִם / לֹא מָצָא בְּבֵיתוֹ נָחַת / רַק פַּח וָפַחַד וָפָחַת / בָּרָד וְאֵשׁ מִתְלַקָּחַת / כִּי הָיוּ מִדַּי יוֹם יוֹם נִצִּים / הִנֵּה הָאֵשׁ וְהָעֵצִים / יָקוּמוּ זֶה עַל־זֶה כְּהַר־פְּרָצִים / וַיִּצְעַק הָאִישׁ אֶל־יְיָ מִפְּנֵי לֹחֲצִים / כִּי בְנוֹ מָשַׁךְ יָדוֹ אֶת־לֹצֲצִים / וְאִשְׁתּוֹ תּוֹרֶה בְלִבּוֹ מִן־הַכְּלִמָּה חִצִּים / וְהַנֹּגְשִׂים אָצִים / וַיֹּאמֶר הָאִישׁ בְּלִבּוֹ: מִשֶּׁבֶת עוֹד אִתָּם נִבְעַתִּי / פֶּן־תִּדְבָּקַנִי הָרָעָה וָמַתִּי / אֵין־לִי טוֹב רַק לְהַנִּיחַ אֵלֶּה הָאֲמֵלָלִים / וְלִגְלוֹת בְּרֹאשׁ גּוֹלִים / אוּלַי יִקָּרֶה יְיָ לִקְרָאתִי / וְאֶמְצָא מָקוֹם תִּהְיֶה־שָׁם מְנוּחָתִי. / וַיְהִי הַיּוֹם וַיִּקַּח הָאִישׁ הוֹנוֹ וּרְכוּשׁוֹ / וַיָּקֶם וַיֵּלֶךְ אֶל־נַפְשׁוֹ / וַיִּקֶר מִקְרֵהוּ בְּאֶרֶץ מֵאַרְצוֹת יָוָן אֲשֶׁר שָׁם אַנְשֵׁי אֱמוּנָה / הָיוּ עִם־הַשֵּׂכֶל בְּאָמְנָה / בְּשִׁבְטֵי יִשְׂרָאֵל מוּדַעַת נֶאֱמָנָה / וְשֵׁם הָעִיר שִׂבְמָה / וַיַּרְא הָאִישׁ מְנֻחָה כִּי טוֹב וְאֶת־הָאָרֶץ כִּי נָעֵמָה / וַיִּטַּע שָׁם אָהֳלֵי אַפַּדְנוֹ / וַיִּשְׁכַּח שָׁם צָרַת אִשְׁתּוֹ וּבְנוֹ / וַיֹּאמַר “זֹאת הָאָרֶץ אֲהַבְתִּיהָ / אִוִּיתִיהָ חֲמַדְתִּיהָ / זֹאת־מְנוּחָתִי עֲדֵי־עַד פֹּה אֵשֵׁב כִּי אִוִּיתִיהָ” / וַיֶּאֱהָבוּהוּ אַנְשֵׁי הַמָּקוֹם עַל־יָשְׁרוֹ / וְעַל־נֹעַם פְּעָלָיו וּכְבוֹד עָשְׁרוֹ / כִּי כָּלִיל הוּא בַּהֲדָרוֹ / וַיֹּאמְרוּ “בֹּא־יָבֹא בְרִנָּה נֵזֶר הוֹדֵנוּ / הַמַּשְׂבִּיעַ בַּטּוֹב עֶדְיֵנוּ / הַמְחַדֵּשׁ כְּבוֹדֵנוּ / זֶה יְנַחֲמֵנוּ מִמַּעֲשֵׂנוּ וּמֵעִצְּבוֹן יָדֵינוּ”.

וַיּוֹאֵל הָאִישׁ לָשֶׁבֶת אִתָּם / וְלִבּוֹ עָלֵז בְּחֶבְרָתָם / וַיְהִי שָׁם כְּעֶשְׂרִים שָׁנָה גָּדוֹל לַיְּהוּדִים וְנֶאֱהָב / כָּבֵד מְאֹד בַּמִּקְנֶה בַּכֶּסֶף וּבַזָּהָב / כִּי הִצְלִיחוֹ הָאֵל עוֹשָׂיו / וְהִרְוִיחוֹ בְּכָל־מַעֲשָׂיו / וַיְהִי הַיּוֹם חָלָה אֶת־חָלְיוֹ אֲשֶׁר מֵת בּוֹ / וּבִרְאוֹתוֹ כִי גָדֵל צִירוֹ וּמַכְאוֹבוֹ / קְרָא אֵלָיו אַנְשֵׁי מְקוֹמוֹ / אֲשֶׁר יָדַע כִּי הֵם דּוֹרְשֵׁי שְׁלוֹמוֹ / וַיֹּאמַר “שְׁמָעוּנִי אַנְשֵׁי שְׁלוֹמִי / הִנְנִי נֶאֱסָף אֶל־עַמִּי / וְאֵין לִי יוֹרֵשׁ וְגוֹאֵל / רַק בֵּן אֶחָד בְּרוֹמָא וּשְׁמוֹ דָּנִיֵּאל / אֲחַלֶּה לִפְנֵיכֶם הָאֲצִילִים לִהְיוֹת הוֹנִי בְּיֶדְכֶם לְמִשְׁמָרֶת / וְהוֹדִיעוּ לִבְנִי מִיתָתִי בְּאִגָּרֶת / וּבְבֹאוֹ הֵנָּה בְּעֵדוּת מְבֹאָרֶת / תְּנוּ לוֹ אֶת־נַחֲלָתִי הַנִּשְׁאָרֶת / וַאֲשֶׁר יָבוֹא בְּלֹא עֵדוּת וּמַזְכָּרֶת / יֵחָשֵׁב כְּגַנָּב נִמְצָא בַמַּחְתָּרֶת” / וַיַּעֲנוּ הָאֲנָשִׁים וַיֹּאמְרוּ “אֲדֹנֵנוּ נֵזֶר הַיְקוּם / כַּאֲשֶׁר צִוִּיתָ אֶת־עֲבָדֶיךָ כֵּן יָקוּם” / וַיְהִי כְּכַלֹּתוֹ לְדַבֵּר עִם־אַנְשֵׁי שְׁלוֹמָיו / וַיֶאֱסֹף רַגְלָיו אֶל־הַמִּטָּה וַיִּגְוַע וַיָּמָּת וַיֵּאָסֶף אֶל־עַמָּיו / וַיָּבוֹאוּ כָּל־אַנְשֵׁי מְקוֹמוֹ לָנוּד לוֹ וּלְסָפְדוֹ / “הוֹי אָדוֹן וְהוֹי הֹדֹה” / וַיִּקְבְּרוּהוּ בְמַעֲלֵה קִבְרֵיהֶם עַל־צִדְקָתוֹ / וְכָבוֹד עָשׂוּ־לוֹ בְמוֹתוֹ.

וַיִּזְכְּרוּ הָאֲנָשִׁים אֶת־מִצְוַת הָאִישׁ וַיִּשְׁלְחוּ לְרוֹמָא אִגָּרֶת / אֹדוֹת מִיתַת הָאִישׁ מְסַפָּרֶת / וְאֶת־כְּבוֹד עָשְׁרוֹ אֲשֶׁר הִנִּיחַ לְיוֹרְשָׁיו / וַאֲשֶׁר יָבֹא בְעֵדוּת אֱמֶת כָּל־דּוֹרְשָׁיו / וּבְכֵן לֹא יִיעָפוּ כָּל־מְבַקְשָׁיו. / וַיְהִי כַּאֲשֶׁר הִגִּיעָה הַשְּׁמוּעָה בָּעִיר / וּנְגַלֶּה הַדָּבָר לְמִגָּדוֹל וְעַד־צָעִיר / נִמְצָא אָז בְּרוֹמָא אִישׁ אֶחָד מִזֶּרַע הַיְּהוּדִים / וּשְׁמוֹ אֶפְרַיִם רֹעֶה רוּחַ וְרֹדֵף קָדִים / זוֹלֵל וְסֹבֵא / כָּל־הַיּוֹם כָּזָב וָשֹׁד יַרְבֶּה / פְּרִיץ חַיּוֹת / רַע לַשָּׁמַיִם וְרַע לַבְּרִיּוֹת / עֲקוּב הַלֵּב וְאִישׁ מַהֲתַלּוֹת / וְלוֹ לָשׁוֹן מְדַבֶּרֶת גְּדֹלוֹת / הָיָה אִישׁ בַּעַל נֵכֶל וְעָרְמָה / וְלִבּוֹ חוֹרֵשׁ אָוֶן וּמִרְמָה / גָּבַר בְּמִרְמוֹת וָתֹךְ עַל־כָּל אֲשֶׁר הָיוּ לְפָנָיו / וְהָיָה דְבָרוֹ דָבָר דָּבֻר עַל־אָפְנָיו / וּבְשָׁמְעוֹ הַשְּׁמוּעָה חָשַׁב מַחֲשֶׁבֶת רָעָה / לַעֲשׂוֹת חֹנֶף וּלְדַבֵּר תּוֹעָה / וַיָּשֶׁת עֵצוֹת בְּנַפְשׁוֹ / לָקַחַת מָמוֹן הַמֵּת וּלְיַרְשׁוֹ / וּמִן־הַיּוֹם הַהוּא וָהָלְאָה רוּחַ אַחֶרֶת עִמּוֹ / וְלֹא־עָשָׂה רַגְלָיו וְלֹא־עָשָׂה שְׂפָמוֹ / וְלָבַשׁ שְׁחוֹרִים כְּאִלּוּ מִתְאַבֵּל עַל־מֵת / לְהַרְאוֹת הָעַמִּים כִּי הוּא בְנוֹ בֶּאֱמֶת.

וַיִּקַּח זְהָבוֹ וְכַסְפוֹ / וַיָּשֶׂם אֶת־נַפְשׁוֹ בְכַפוֹ / וַיַּעֲבֹר מִדְבָּרוֹת וְצִיּוֹת / וְהוּא מִתְהַלֵּךְ בֵּין הַחַיּוֹת / וַיִּרְכַּב אֳנִיּוֹת / עַד־אֲשֶׁר הִגִּיעַ אֶל־מְחוֹז חֶפְצוֹ / וְאֶל־מְקוֹם הַמֵּת וְאֶל־אַרְצוֹ / וַיְהִי בְּהַגִּיעוֹ אֶל־הַמָּלוֹן קָרָא אֶל־הַנַּעַר לַיָּמִים צָעִיר / וַיֹּאמַר "קְרָא לִי עֲשָׂרָה אֲנָשִׁים מִטּוֹבֵי הָעִיר / אֲשֶׁר יָדַעְתָּ כִּי־הֵם זִקְנֵי הָעָם וְשֹׁפְטָיו / אֲשֶׁר יוֹרוּ חֻקֵּי אֵל וְאֶת־מִשְׁפָּטָיו / וֶאֱמֹר אֲלֵיהֶם: הִגִּיעַ פֹּה דָּנִיֵּאל הַנִּכְאָב / אֲשֶׁר דְּמֵי כְבֵדוֹ בִּבְכִי יִשְׁאָב / יַחְפֹּץ לַעֲשׂוֹת חֹק הַבֵּן עַל־מִיתַת הָאָב / וְאַחֲלֵי לִפְנֵי מִשְׂרֵת מַעֲלַתְכֶם / וִיקָר תִּפְאֶרֶת גְּדֻלַּתְכֶם / לָלֶכֶת עִמּוֹ עַד־קֶבֶר אָבִיו קְדוֹשׁוֹ / רִכְבּוֹ וּפָרָשׁוֹ / כִּי שָׁם יִשְׁפֹּךְ עָלָיו אֲשֶׁר שָׁם יַחֲנֶה / וְהִנֵּה שָׁם קוֹל הֲמֻלָּה כְּקוֹל מַחֲנֶה / וְיִמְצְאוּהוּ בִּדְמֵי דְמָעָיו מִתְגּוֹלֵל / וְקוֹלוֹ אַיָּלוֹת יְחוֹלֵל / יֹאמַר “אָבִי אָבִי רַב מְחוֹלֵל / הַמּוֹצִיא יָקָר מִזּוֹלֵל / אֵיךְ נָפַלְתָּ מִשָּׁמַיִם הֵילֵל” / וּבִרְאוֹתוֹ הַזְּקֵנִים קָם בַּעֲנָוָה וּבְמוּסָר / כְּאִלּוּ בְּחַבְלֵי הָעֶצֶב וְהַדְּאָגָה נֶאֱסָר / וַיֹאמַר “אַחֲלֵי לִפְנֵיכֶם מוֹרֵי הַתּוֹרָה וְהַהֲלָכָה / הַמַּצְלִיחִים לִמְלוּכָה / עֲשׂוּ אִתִי בְרָכָה / וּבֹאוּ אִתִּי אֶל־עֵמֶק הַבָּכָא / וְהַרְאוּנִי מְקוֹם מְנוּחַת צַדִּיק אָבִי / וְאֶעֱשֶׂה שָׁם כַּאֲשֶׁר עִם־לְבָבִי” / וַיֹּאמְרוּ אֵלָיו הָאֲנָשִׁים “הִנְנוּ נַעֲשֶׂה כָּל־חֶפְצֶךָ / וַיְיָ אֱלהֶיךָ יִרְצֶךָ” / וַיֵּלְכוּ עִמּוֹ וַיַּרְאוּהוּ מְקוֹם הַקְּבוּרָה / וּבִרְאוֹתוֹ הַקֶּבֶר צָעַק מָרָה / וַיֹּאמַר “אָבִי אָבִי רֶכֶב יִשְׂרָאֵל וּפָרָשָׁיו / קוֹמַת אֲרָזָיו מִבְחַר בְּרוֹשָׁיו / רָד עִם־אֵל וְעִם־קְדוֹשָׁיו / מִי הֶאֱמִין כִּי תָמוּת בְּאֶרֶץ נָכְרִיָּה / וְתַעֲזֹב אֵשֶׁת חֵיקְךָ תַּאֲנִיָּה וַאֲנִיָּה / בָּכוֹ תִּבְכֶּה בַּלַּיְלָה וְדִמְעָתָהּ עַל לֶחֱיָהּ / אָבִי אָבִי כֶּתֶר רָאשִׁי / הֲלֹא קָרָאתָ לִשְׁפֹּךְ עָלֶיךָ אֶת־מְרִירוּת נַפְשִׁי / מִי יָעַץ זֹאת עָלֶיךָ אָבִי אָבִי / לָמָּה הָיְתָה מַכָּתִי אֲנוּשָׁה וְנֶצַח כְּאֵבִי / חָנֻּנִי חִנָּנִי אַתֶּם רֵעָי כִּי יַד־אֱלוֹהַּ נָגְעָה בִּי” / וַיֵּבְךְּ הָאִישׁ בְּכִי נִמְרָץ / עַד אֲשֶׁר גְּבוּל הַבְּכִי פָּרָץ / כִּי דְבַר הַמָּמוֹן נָתַן בְּפִיהוּ חָזוֹן נִפְרָץ / וְכַאֲשֶׁר יִמְנָעוּהוּ מִלִּבְכּוֹת אָז יֶחֱרָץ / וּלְהַכּוֹת עַל־פָּנָיו וְעַל־חֵיקוֹ לֹא יְרַחֵם / וְשַׂעֲרוֹתָיו יִמְרֹט וּבְיָדוֹ יִקָּחֵם / וַיָּקוּמוּ כָּל־הָאֲנָשִׁים אֲשֶׁר שָׁם לְנַחֲמוֹ וַיְמָאֵן לְהִתְנַחֵם / וַיֹּאמַר כִּי נֵרֵד אֲנִי וְאִמִּי רוֹשׂוֹלָה / אֶל אָבִי אָבֵל שְׁאֹלָה. / וַיְהִי כִּרְאוֹת הָאֲנָשִׁים קוֹל בִּכְיָתוֹ כְּאִלּוּ הָאֵל זֹעֵם / וּבְקוֹל כָּמֹהוּ יַרְעֵם / וְכִי מַלְבּוּשָׁיו בְּיָדָיו יִקְרָעֵם / וַיַּחֲזִיקוּ הָאֲנָשִׁים בִּשְׁתֵּי יָדָיו / לְבִלְתִּי יִקְרַע עוֹד בְּגָדָיו / וַיִּזְעַק הָאִישׁ וַיֹּאמַר “הִנְנִי נִשְׁבַּע בְּנֶפֶשׁ אָבִי / כִּי אִם תִּמְנְעוּנִי מִקְּרֹעַ בְגָדַי אֶקְרַע לְבָבִי” / וַיֵּבְךְּ הָאִישׁ עַד־אֲשֶׁר גְּרוֹנוֹ נִחַר / וַיִּשָּׁבַע כִּי לָמוּת עַל־אָבִיו יִבְחַר / וּמֵעֲשֹׁוֹת כָּזֶה יוֹם יוֹם לֹא יִתְאַחֵר / וְהָיָה כָזֶה יוֹם מָחָר / וַיִּפֹּל עַל־הַקֶּבֶר מְלֹא־קוֹמָתוֹ / וַיִּלְחֹךְ עַפְרוֹתָיו וְאַדְמָתוֹ / וַיִּהְיוּ הָאֲנָשִׁים בִּרְאוֹתָם צָרַת נַפְשׁוֹ תְּמֵהִים / וַיֹּאמְרוּ “אֵין זֶה כִּי אִם־אִישׁ אֱלֹהִים” / וַיְהִי כְּכַלֹּתוֹ לִבְכּוֹת בְּכָל־אַוַּת נַפְשׁוֹ / וְלִמְרֹט שְׂעַר רֹאשׁוֹ / וַיֹּאמַר “אֲחַלֶּה לִפְנֵיכֶם הַזְּקֵנִים יוֹדְעֵי צָרַת לְבָבִי / לִקְרֹא צִדּוּק הַדִּין עַל־קֶבֶר אָבִי / אוּלַי יֵחָשֵךְ כְּאֵבִי” / וַיִּהְיוּ הָאֲנָשִׁים לִקְרוֹתוֹ מְהִירִים / וְהֵם לֹא יָדְעוּ כִּי עָרֹם יַעֲרִים / וְעוֹד יַעֲשֶׂה עַל־מִיתַת הָאִישׁ הַזֶּה פּוּרִים / וַיְהִי כִּקְרֹא הַקּוֹרֵא “שׁוֹמְרֵי גִנְזֵי גַן עֵדֶן” וְהִזְכִּיר שֵׁם הַמֵּת / הִתְאַנַּח אֲנָחָה גְּדוֹלָה לְהַרְאוֹת הֶפֶךְ הָאֱמֶת.

וַיָּשָׁב אֶל־בֵּיתוֹ אָבֵל וַחֲפוּי רֹאשׁ / בְּפָנָיו תַּחֲנֶה הַדְּאָגָה וּבְלִבּוֹ אָוֶן יַחֲרֹשׁ / בְּחָשְׁבוֹ כִּי תַחַת הַנַּעֲצוּץ יַעֲלֶה בְרוֹשׁ / וְהַזְּקֵנִים וְכָל־הָעָם עַד־בֵּיתוֹ לִוּוּהוּ / וּמִלִּבְכּוֹת עוֹד יַזְהִירוּהוּ וַיְצַוּוּהוּ / וַיִּשָּׁאֵר הָאִישׁ בְּבֵיתוֹ מִתְעַסֵּק בְּאֶבְלוֹ / וּגְדוּדִי הַשְּׂמָחוֹת יַחֲנוּ בְלִבּוֹ אִישׁ אִישׁ עַל־דִּגְלוֹ / וְכָל־הָעָם שָׁב אִישׁ לְאָהֳלוֹ / וַיֹּאמֶר בְּלִבּוֹ הָאִישׁ / רָשָׁע מַחְפִּיר וּמַבְאִישׁ / יִזָּכֵר עֲוֹן אֲבֹתָיו אֶל־יְיָ וְחַטַּאת אִמּוֹ אַל־תִּמָּח / “עֵת לְפַזֵּר הַיּוֹם וּמָחָר אֶשְׂמָח / הִנְנִי עֹשֶׂה חֲדָשָׁה עַתָּה תִצְמָח” / וַיִּשְׁלַח וַיִּזְבַּח צֹאן וּבָקָר / לַעֲשׂוֹת לַעֲנִיֵּי הָאָרֶץ אוֹרָה וְשִׂמְחָה וְשָׂשׂוֹן וְיָקָר / וַיְחַלֵּק לְכָל־הָעָם לְמֵאִישׁ וְעַד־אִשָּׁה / אֶשְׁפָּר וַאֲשִׁישָׁה וַיֹּאמַר כִּי זֶה יַעֲשֶׂה לְכַפָּרַת נֶפֶשׁ אָבִיו הַקְּדוֹשָׁה / וַיִּשְׁלַח לִמְחַלְּקֵי הַצְּדָקָה אֶל־הָאֲמֵלָלִים / כֶּסֶף שְׁלשִׁים שְׁקָלִים / לְחַלֵּק אוֹתָם לְאֶבְיוֹנִים וְדַלִּים / וּמִשְׁלוֹחַ מָנוֹת לָאַלְמָנוֹת וְלַיְתוֹמִים / וַיַּעַשׂ לְאָבִיו אֵבֶל שִׁבְעֵת יָמִים.

וַיָּשֶׁת הָאִישׁ עֵצוֹת בְּנַפְשׁוֹ / לְהַצִּיב צִיּוּן עַל־קֶבֶר הָאִישׁ אֲשֶׁר חָשַׁב לְיַרְשׁוֹ / לִבְנוֹת כְּמוֹ־רָמִים מִקְדָשׁוֹ / בְּחָשְׁבוֹ כִּי בָזֶה יַגִּיעַ אֶל־מְבֻקָּשׁוֹ / וַיֹּאמֶר בְּלִבּוֹ אִם הוֹנִי הַיּוֹם חוֹלֵף וְאוֹזֵל / וַעֲמָלִי וִיגִיעַ כַפַּי אֲנִי גוֹזֵל / לְשַׁלֵּחַ אוֹתוֹ לַעֲזָאזֵל / תַּחַת הַכֶּסֶף אָבִיא זָהָב וְתַחַת הַנְּחֹשֶׁת אָבִיא כֶסֶף וְתַחַת הָאֲבָנִים בַּרְזֶל. / וַיִּשְׁלַח וַיִּקְרָא לֶחָרָשִׁים וְלַבּוֹנִים / וַיִּתֵּן אֶת־הַכֶּסֶף בִּידֵיהֶם כַּאֲבָנִים / וַיֹּאמֶר אֲלֵיהֶם “אֲנִי עִם־לְבָבִי / לִבְנוֹת צִיּוּן עַל־קֶבֶר אָבִי / לְכָבוֹד וּלְתִפְאָרֶת / וְלִכְתֹּב מִקָּצָה בַּחֹבָרֶת / בְּעֵט בַּרְזֶל וְעֹפָרֶת / שֵׁם הַצַּדִּיק אָבִי וְשֵׁם אָבִיו וְאִמּוֹ / וְאֶחָיו וְשַׁעֵר מְקוֹמוֹ” / וַיֹּאמְרוּ הָאֲנָשִׁים “חַיֶּיךָ, לֹא־נִשְׁאַל וְלֹא־נְנַסֶּה / כְּכֹל אֲשֶׁר תְּצַוֶּה אֶת־עֲבָדֶיךָ כֵּן נַעֲשֶׂה” / וַיִּבְנוּ עַל־הַקֶּבֶר בְּמִצְוַת הָאִישׁ צִיּוּן נִפְלָא / בְּכֶתֶם טָהוֹר לֹא יְסֻלָּא / וַיָּקִימוּ עָלָיו מַצֵּבָה גְדוֹלָה / לְשֵׁם וְלִתְהִלָּה / וַיִּכְתֹּב שָׁם הָאִישׁ בְּיָדוֹ / זֶה הַמִּכְתָּב לְהַעֲלִים סוֹדוֹ / “פֹּה שׁוֹכֵב נֵזֶר הַחֲסִידִים / וְתִפְאֶרֶת הַנְּגִידִים / מֹשֶׁה בֶּן־פְּלוֹנִי מֵעִיר רוֹמָא וּבָנָה זֶה הַצִּיּוּן לִשְׁמוֹ וְלִכְבוֹדוֹ / דָּנִיֵּאל הַקּוֹדֵר וְהַנִּכְאָב בְּנוֹ יְחִידוֹ / אֲשֶׁר אֲהֵבוֹ בְּכָל־לְבָבוֹ וּבְכָל־נַפְשׁוֹ וּבְכָל־מְאֹדוֹ / וַיְמָאֵן לְהִתְנַחֵם בְּעוֹדוֹ / עַל־אֲשֶׁר לֹא־הָיָה עַל־פָּנָיו בְּהִפָּרְדוֹ / לָנוּד לוֹ וּלְסָפְדוֹ / הוֹי אָב וְהוֹי אָדוֹן וְהוֹי הֹדֹה”.

וַיֵּעֲמֹד שָׁם הָאִישׁ חֹדֶשׁ יָמִים / כְּאִישׁ מִתְאַבֵּל עַל־מֵת נֶעֱצָב וּמַשְׁמִים / וְלֹא שָׁאַל עַל־יְרֻשַּׁת אָבִיו לְעוֹלָם / פֶּן יִתְגַּלֶּה סוֹדוֹ וְיָשֹׁב דַּךְ וְנִכְלָם / וּמִדֵּי יוֹם יוֹם יַעֲלֶה בֵית־הַתְּפִלָּה לְהַעֲלִים סוֹדוֹ / וַיִּלְבַּשׁ צְדָקָה כְּמַדּוֹ / וּתְפִלִּין בְּרֹאשׁוֹ וְצִיצִת בְּבִגְדוֹ / עַד־אֲשֶׁר בָּטְחוּ בּוֹ הַלְּבָבוֹת / וְנִפְלָא בְּעֵינֵי הַמַּחֲשָׁבוֹת. / וַיְהִי כִּרְאוֹת אַנְשֵׁי הַקְּהִלָּה / כִּי־זֶה הָאִישׁ חָרֵד עַל־מֵת חֲרָדָה גְּדוֹלָה / וְלֹא סָר מִלִּבְכּוֹת וּמֵהִתְאַבֵּל / וּמִלִּקְרֹא “עַל־נַהֲרוֹת בָּבֶל” / וְלֹא־שָׁאַל מָמוֹן הַמֵּת כְּאִלּוּ לֹא־הָלַךְ שָׁם לְקַבֵּל / וְכִי יְפַזֵּר כָּל־רְכוּשׁוֹ / לְתִתּוֹ לְכַפָּרַת נַפְשׁוֹ / וְכִי הֵקִים עַל־קִבְרוֹ מַצֶּבֶת אָבֶן / לֹא־חָשְׁבוּ כִּי לִבּוֹ יִקְבָּץ־אָוֶן / וַיֹּאמְרוּ אִישׁ אֶל־אָחִיו “מַדּוּעַ אֲנַחְנוּ מַחֲשִׁים / וְהִנֵּה הָאִישׁ הַלָּזֶה זֶרַע קְדוֹשִׁים / אֵין צֹרֶךְ לְאִישׁ כָּזֶה עֵדוּת וּכְתִיבָה / עֵד הַגַּל הַזֶּה וְעֵדָה הַמַּצֵּבָה / בָּרוּךְ יְיָ אֲשֶׁר לֹא־הִשְׁבִּית לְאָבִיו גּוֹאֵל / וְיִקָּרֵא שְׁמוֹ בְּיִשְׂרָאֵל / וְעַתָּה נִתֵּן לוֹ אֶת־יְרֻשָּׁתוֹ / אַחֲרֵי אֲשֶׁר יָדַעְנוּ אֶת־קְדֻשָּׁתוֹ” / וְהֵם לֹא־יָדְעוּ כִּי נִלְכְּדוּ בְרִשְׁתּוֹ.

וַיִּשְׁלְחוּ הָאֲנָשִׁים וַיִּקְרְאוּ־לוֹ / וְהוּא עוֹדֶנּוּ מַחֲזִיק בְּאֶבְלוֹ / וַיֹּאמְרוּ לוֹ “אֲדוֹנֵנוּ אֶשְׁכֹּל הַכֹּפֶר וּצְרוֹר הַמֹּר / אָבִיךְ צִוָּה לִפְנֵי מוֹתוֹ לֵאמֹר / הַכֶּסֶף אֲשֶׁר נָתַתִּי בְיֶדְכֶם לִשְׁמֹר / תְּנוּ אוֹתוֹ לְדָנִיֵּאל בְּנִי וְיָרַשׁ אוֹתוֹ / וּבִתְנַאי שֶׁיָּבִיא רְאָיָה אוֹ עֵדוּת מֵאֶרֶץ מוֹלַדְתּוֹ / וּבְעֵדוּת זִקְנֵי קְהָלוֹ יִהְיֶה נֶעֱזָר / לְמַעַן אֲשֶׁר לֹא־יִקְרַב אִישׁ זָר / וַאֲנַחְנוּ רָאִינוּ בְךָ נוֹרָאוֹת / וְנֶגֶד עֵינֵינוּ עָשִׂיתָ עַל־מִיתַת אָבִיךָ נִפְלָאוֹת / וּבָא הַמּוֹפֵת וְהָאוֹת / כִּי פְעָלֶיךָ הַנִּכְבָּדִים / וּדְבָרֶיךָ הַנְּגִידִים / וְאַנְחוֹתֶיךָ וְדִמְעוֹתֶיךָ כְּמֵאָה עֵדִים / וּבְצִּיּוּן אֲשֶׁר עַל־קִבְרוֹ בָנִיתָ/ וְאֶת־אֲשֶׁר עָלָיו הִתְעַנִּיתָ / כֹּל אֲשֶׁר לְאָבִיךְ קָנִיתָ / מִלְּבַד אֲשֶׁר גָּבַרְתָּ עַל־אַנְשֵׁי הַדּוֹר בְּטוּב שִׂיחֶיךָ / וְכוֹכְבֵי הָעֲנָוָה זָרְחוּ בְמִצְחֶךָ / וְעַתָּה קוּמָה הַשָׂר לִמְנוּחֶיךָ / הִנֵּה כַפְתּוֹרֶיךָ וּפְרָחֶיךָ / עֲלֵה רֵשׁ / וְהָאֲנָחוֹת מִלִּבְּךָ גָּרֵשׁ תְּגָרֵשׁ / כִּי אֵין זוּלָתְךָ גּוֹאֵל וְיוֹרֵשׁ” / וַיִּקֹּד הָאִישׁ וַיִּשְׁתָּחוּ / וְעֵינָיו יִבְכָּיוּן וִיצוּרָיו יִשְׂמָחוּ / וְרַעֲיוֹנָיו כֶּעָלֶה יִפְרָחוּ / וְעֶשְׁתֹּנוֹתָיו בְּגַנֵּי הַחֶדְוָה וַתִּרְעֶינָה בָּאָחוּ / וַיֹּאמֶר אֶל־הָאֲנָשִׁים “יַעֲנֶה יְיָ שְׁלוֹמְכֶם / וְעַל־טוֹב יִזָּכֵר שִׁמְכֶם / מַה־יָּקָר חַסְדְכֶם / אֲשֶׁר עֲשִׂיתֶם אֶת־עַבְדְּכֶם / וַאֲנִי מָה / יְשַׁלֵּם יְיָ פָּעָלְכֶם וּתְהִי מַשְׂכֻּרְתְּכֶם שְׁלֵמָה / וְעַתָּה הָאֲצִילִים וְהַשּׁוֹעִים / שִׁלְחוּ מִכֶּם אֲנָשִׁים לְשִׁבְטֵיכֶם יְדוּעִים / וִישׁוֹטְטוּ בְּחוּצוֹת הָעִיר / וְיִשְׁאֲלוּ גָדוֹל וְצָעִיר / אִם אָבִי נִתְחַיֵּב מָמוֹן / לְאִישׁ מִכָּל־הֶהָמוֹן / בְּמֶכֶר אוֹ בְמִקָּח / יָבֹא וְיִקָּח / וּמִי הָאִישׁ אֲשֶׁר שֵׁרֵת אוֹתוֹ בְּמַחֲלָתוֹ / אוֹ הִתְעַסֵּק בִּקְבוּרָתוֹ / הִנֵּה שְׂכָרוֹ אִתּוֹ וּפְעֻלָּתוֹ / וּמִי הָאִישׁ אֲשֶׁר נִתְחַיַּבְתִּי לוֹ דָבָר יִקָּחֵנוּ מִמֶּנִּי / בְּטֶרֶם אֵלֵךְ וְאֵינֶנִּי” / וַיַּעֲנוּהוּ הָאֲנָשִׁים “תְּבֹרַךְ מֵהָאֵל זֶרַע קְדוֹשִׁים / כִּי כַבֵּד כִּבַּדְתָּנוּ / וְלֹא עֲשַׁקְתָּנוּ וְלֹא רַצּוֹתָנוּ / וּדְבַר־עֶצֶב מִמְּךָ וּמֵאָבִיךָ לֹא־סָבַלְנוּ / אֶת־הַכֹּל הִתְקַבָּלְנוּ / בָּרוּךְ יִהְיֶה אֲשֶׁר הֱבִיאֲךָ הֲלוֹם / אָחִינוּ לֵךְ לְשָׁלוֹם”.

וַיִּקַּח הָאִישׁ מִיָּדָם אֶת־הַהוֹן / וַיֹּאמֶר בְּלִבּוֹ אַךְ עָשַׁרְתִּי מָצָאתִי אוֹן / שָׁמַע אֱלֹהִים קוֹל בִּכְיִי / הֲכִי קָרָא שְׁמִי אֶפְרַיִם כִּי־הִפְרַנִי אֱלֹהִים בְּאֶרֶץ עָנְיִי / וְהָפַךְ מִסְפְּדִי לְמָחוֹל / כַּאֲשֶׁר הֱבִיאַנִי נַחֲלַת אֲחֵרִים לִנְחֹל / בָּרוּךְ הַמַּבְדִּיל בֵּין קֹדֶשׁ לָחוֹל. / וַיְהִי כַּאֲשֶׁר הִגִּיעַ הַכֶּסֶף לְיָדוֹ / יָרֵא פֶּן־יָבוֹא שָׁם אִישׁ וְיֶחֱשֹׂף סוֹדוֹ / וַיֹּאמַר “אֵין־לִי טוֹב כִּי אִם לִרְכֹּב עַל־צִי / וְלָשׁוּב אֶל־מְחוֹז חֶפְצִי / וְאֵלְכָה אֶל־מְקוֹמִי וּלְאַרְצִי / פֶּן־יָבֹא בְּנוֹ הָאֲמִתִּי / אוֹ יְשַׁלַּח הֵנָּה אִישׁ עִתִּי / וְיוֹצִיאוּ אֶת־בִּלְעִי מִפִּי / וּמִבֵּין שִׁנַּי טַרְפִּי / וָאֶשֹּׁם עַל־עֵקֶב בָּשְׁתִּי / אִם יִקְּחוּ מִמֶּנִּי אֶת־אֲשֶׁר לָכַדְתִּי בְּרִשְׁתִּי / הַכֶּסֶף וְהַזָּהָב אֲשֶׁר לָקַחְתִּי בְּחַרְבִּי וּבְקַשְׁתִּי” / וַיְמַהֵר וַיִּרְתֹּם פִּרְדֵי הַפְּרִידָה / וַיִּקַּח הַנְּסִיעָה לוֹ לְצֵידָה / וַיָּשָׁב אֶל־בֵּיתוֹ קַל חִישׁ כְּנֶשֶׁר חָשׁ לֶאֱכֹל / בְּשִׂמְחָה וּבְטוּב לֵבָב מֵרֹב כֹּל.

וַיְהִי מִקֵּץ שְׁנָתַיִם יָמִים / וַיָּשֶׂם דָּנִיֵּאל בֶּן־הַמֵּת עַל־לִבּוֹ לִרְכֹּב יַמִּים / וּלְהוֹצִיא אֶת־הַכֶּסֶף וְאֶת־הַזָּהָב אֲשֶׁר הִנִּיחַ אָבִיו מִבֵּין קַרְנֵי רְאֵמִים / וְלֹא־יָדַע כִּי אֶפְרַיִם / לָקַח בִּרְכָתוֹ וְהוֹשִׁיבוֹ בֵּין תַּנּוּר וְכִירָיִם. / וַיִּקַּח דָּנִיֵּאל מִזִּקְנֵי קְהַל רוֹמָא עֵדוּת וּרְאָיָה / כִּי הוּא בֶן־פְּלוֹנִי אֲשֶׁר מֵת בְּאֶרֶץ נָכְרִיָּה / וְעַל־הַחָתוּם הָיוּ זִקְנֵי קְהַל רוֹמָא וְשׁוֹעֶיהָ / נְקוּבֵי רֵאשִׁית הַחָכְמָה וִידוּעֶיהָ / וַיֵּלֶךְ לְמַסָּעָיו וַיָּבֹא אֶל־חוֹף אֲרַם צוֹבָה / וַיִּמְצָא אֳנִיָּה וַיִּתֵּן שְׂכָרָהּ וַיֵּרֶד בָּהּ.

וַיֵּלֶךְ אַחַת הֵנָּה וְאַחַת הֵנָּה עַד־אֲשֶׁר הִגִּיעַ אֶל עִיר־קִבְרוֹת אֲבוֹתָיו / וּבְהַגִּיעוֹ שָׁם לֹא־שָׁאַל עַל־אָבִיו וְעַל־אֹדוֹתָיו / וְאִם נִקְבַּר בְּקִבְרֵי יִשְׂרָאֵל אוֹ בְאַשְׁפָּה / וְאִם עָשׂוּ לוֹ כָבוֹד בְּמוֹתוֹ אוֹ חֶרְפָּה / לֹא־בָכָה וְלֹא־הִתְאַבֵּל לְמִטּוֹב וְעַד־רָע / לֹא־מָרַט אֶת־רֹאשׁוֹ וּבְגָדָיו לֹא־קָרָע / רַק שָׁאַל בְּיַד מִי הִנִּיחַ אָבִיו הוֹנוֹ / וּמִקְנֵה קִנְיָנוֹ / כִּי הוּא כֹּחוֹ וְרֵאשִׁית אוֹנוֹ / לוֹ מִשְׁפַּט הַיְּרֻשָּׁה / אוֹי לָהּ לְאוֹתָהּ כְּלִמָּה אוֹי לָהּ לְאוֹתָהּ בּוּשָׁה / וַיֹּאמְרוּ “מַה־לְּךָ הַפֶּתִי אֲשֶׁר עַל־רִיב לֹא־לְךָ תִּתְעַבָּר / וְעַתָּה מַה־לַּתֶּבֶן אֶת־הַבָּר / כִּי תָבוֹא בְּנֵכֶל וּבְעָרְמָה / כְּאִישׁ בַּעַל זָדוֹן וּמִרְמָה / לִשְׁאֹל יְרֻשַּׁת אֲנָשִׁים צַדִּיקִים / כְּהִגָּלוֹת נִגְלוֹת אַחַד הָרֵקִים / וְלֹא־זָכַרְתָּ “אָרוּר אֲשֶׁר לֹא־יָקִים” / וְעַתָּה חֲדַל־לְךָ הַסָּכָל וְהַנִּבְזֶה / שׁוֹכֵב וְהוֹזֶה / וְאַל־תּוֹסֶף עוֹד דַּבֵּר אֵלֵינוּ בַּדָּבָר הַזֶּה” / כִּשְׁמֹעַ הָאִישׁ תְּשׁוּבַת הָאֲנָשִׁים עָמַד מַרְעִיד / וַיּוֹצֵא אִגֶּרֶת מֵחֵיקוֹ וַיֹּאמַר “הִנֵּה זֹאת עָלַי תָּעִיד / כִּי אָנֹכִי דָּנִיֵּאל בֶּן־פְּלוֹנִי הַמֵּת / וְאֵינֶנִּי בָא אֲלֵיכֶם בְּמִרְמָה כִּי אִם בֶּאֱמֶת” / וַיִּקְרְאוּ הָאֲנָשִׁים אֶת־הָאִגֶּרֶת / וְהִנֵּה הִיא עַל־אֹדוֹת הָאִישׁ הַלָּזֶה מְדַבֶּרֶת / וְעַל־הֶחָתוּם זִקְנֵי קְהַל רוֹמָא וְרַבָּנֶיהָ / אֲצִילֶיהָ זְקֵנֶיהָ וּקְצִינֶיהָ.

וַיִּתְמְהוּ הָאֲנָשִׁים אִישׁ אֶל־רֵעֵהוּ / וַיִּקָּהֲלוּ שָׁם קְהִלָּה גְּדוֹלָה כָּל־הָעָם מִקָּצֵהוּ / וַיְהִי כָּל־הָעָם נָדוֹן / לֵאמֹר “הָאִישׁ אֲשֶׁר לָקַח הַיְּרֻשָּׁה הָיָה אִישׁ מָדוֹן / וּבָא עָלֵינוּ בְדֶרֶךְ מִרְמָה וְזָדוֹן” / וּקְצָתָם חָשְׁבוּ רָעָה עַל־זֶה הָאַחֲרוֹן / וַיֹּאמְרוּ “מַה־מְּאֹד גָּדוֹל זֶה הָעִוָּרוֹן / אֲשֶׁר לֹא הִתְנַהַגְנוּ בִּכְבֵדוּת / בְּתִתֵּנוּ הַמָּמוֹן הַהוּא בְּלֹא רְאָיָה וְעֵדוּת / לְאִישׁ אֲשֶׁר לֹא־יָדַעְנוּ אוֹתוֹ וְחִין עֶרְכּוֹ” / זֶה אֹמֵר בְּכֹה וְזֶה אֹמֵר בְּכֹה / וַיֹּאמֶר הָאִישׁ אֶל־הָאֲנָשִׁים / “אֶרְאֶה אֶתְכֶם מִתְלַחֲשִׁים / הַאִם נְתַתֶּם יְרֻשָּׁתִי לְאַחֵר” / וַיֹּאמְרוּ “נָתַנּוּ עֹבֵר לַסֹּחֵר” / וַיֶּחֱרַד הָאִישׁ חֲרָדָה גְּדוֹלָה / וַיִּצְעַק צְעָקָה כְּקוֹל הֲמֻלָּה / וַיֹּאמַר “מִי הוּא אֲשֶׁר הִרְבָּה מְבוּכָתִי / וּבָא עַד־הֲלוֹם וְלָקַח אֶת־בִּרְכָתִי” / וַיֹּאמְרוּ “בָּא הֵנָּה אִישׁ־חַיִל רַב־פְּעָלִים לִבּוֹ בְּקִרְבּוֹ חָלַל / בּוֹכֶה וּמִתְאַבֵּל וּבִדְמֵי דְמָעָיו מִתְגּוֹלֵל / וַיַּעַט אֶל־הַשָּׁלָל / שָׁאַל עַל־קֶבֶר אָבִיךָ וּפַלְגֵי־מַיִם יָרְדוּ עֵינָיו / וַיִּקְרַע שִׂמְלֹתָיו וַיָּשֶׂם שַׂק בְּמָתְנָיו / וַיִּמְרֹט שַׂעֲרוֹת רֹאשׁוֹ / וַיְחַלֵּק כַּמָּה זְהוּבִים לְכַפָּרַת נַפְשׁוֹ / וּמִדֵּי שַׁבָּת בְּשַׁבַּתּוֹ לְטוֹבָה אוֹתוֹ יִזְכֹּר / סָפַד עָלָיו כְּמִסְפֵּד עַל־הַיָּחִיד וְהָמֵר עָלָיו כְּהָמֵר עַל־הַבְּכוֹר / בָּנָה צִיּוּן עַל־קִבְרוֹ מֵהוֹנוֹ לֹא־נַעֲשָׂה כֵן לְפָנִים / וּפִזֵּר וְנָתַן לָאֶבְיוֹנִים / וְעָמַד נֶעֱצָב וְנִכְאָב / וּמִדָּם כְּבֵדוֹ דְּמָעָיו יִשְׁאָב / וְעָשָׂה מַה־שֶׁלֹּא עָשָׂה לְעוֹלָם בֵּן עַל־אָב / לֹא שָׁאַל לְעוֹלָם יְרֻשָּׁה / רַק הָיָה עַל־מִיתַת אָבִיךָ כְּאֵבוֹ נֶצַח וּמַכָּתוֹ אֲנוּשָׁה / וְהָיוּ לְעוֹלָם צִיצִת בְּבִגְדוֹ וּתְפִלִּין בְּרֹאשׁוֹ / וְקָנָה הוֹן אָבִיךְ וּרְכוּשׁוֹ / בְּשַׂעֲרוֹת אֲשֶׁר תָּלַשׁ מֵרֹאשׁוֹ / וּבַזְּהוּבִים אֲשֶׁר חִלֵּק לְכַפָּרַת נַפְשׁוֹ / וְאֶת־כָּל־מִקְנֵה קִנְיָנָיו / בִּדְמָעוֹת אֲשֶׁר הוֹרִיד מֵעֵינָיו / וְאִם אַתָּה בְנוֹ כַּאֲשֶׁר עֵדִים הֵעַדְתָּ / וּכְפִי הַחִידָה אֲשֶׁר חָדְתָּ / אַיֵּה הַחֲרָדָה אֲשֶׁר עָלָיו חָרָדְתָּ / וּדְמָעוֹת אֲשֶׁר מֵעֵינֶיךָ הוֹרָדְתָּ / וְהַמִּסְפֵּד אֲשֶׁר עָלָיו סָפָדְתָּ / אַיֵּה בְגָדֶיךָ אֲשֶׁר קָרָעְתָּ / וְרֹאשְׁךָ אֲשֶׁר פָּרָעְתָּ / וּמַמְּרוֹרִים אֲשֶׁר עָלָיו שָׂבָעְתָּ. / וְאַף אִם נֶאֶמְנוּ דִּבְרֵי אִגְּרוֹתֶיךָ / מַדּוּעַ אֶחֱרוּ פַּעֲמֵי מַרְכְּבוֹתֶיךָ / בָּא אָחִיךָ בְּמִרְמָה וַיִּקַּח בִּרְכָתֶךָ”.

וַיֹּאמֶר הָאִישׁ “מַה־גָּדְלָה זֹאת הַמְּבוּכָה / אֵין לְמַכָּתִי תְּעָלָה וַאֲרוּכָה / הֲלֹא אֲצַלְתֶּם לִי בְּרָכָה” / וַיֹּאמְרוּ “אָצַלְנוּ לְךָ הַבְּרָכָה אֲשֶׁר בְּהַר עֵיבָל / וּבְשָׂרְךָ יִכְאַב וְנַפְשְׁךָ תֶּאֱבָל / כִּי עַל־מִיתַת אָבִיךָ לֹא תִּתְעַצָּב / וּלְבַקֵּשׁ יְרֻשָּׁתוֹ עַל־מָצוֹר תִּתְיַצָּב / וְעַתָּה אֱמָר־נָא הַחֹלֶד וְהָעַכְבָּר וְהַצָּב / מִי יַאֲמִין כִּי תִהְיֶה בְּנוֹ / וּמִי הוּא הַבֵּן אֲשֶׁר עַל־אָבִיו לֹא תָחוּס עֵינוֹ / וְאַף אִם אַתָּה בֶּן־אִשְׁתּוֹ שְׁמַע חֶרְפָּתֶךָ / זָנְתָה אִמְּךָ הוֹבִישָׁה הוֹרָתֶךָ / וְלָכֵן עַל־מִיתָתוֹ לֹא נִכְמְרוּ נִחוּמֶיךָ / רַק תְּבַקֵּשׁ הוֹנוֹ לְבָשְׁתֶּךָ וּלְבֹשֶת עֶרְוַת אִמֶּךָ / וְעַתָּה אַל־תַּשְׁמַע קוֹלְךָ עִמָּנוּ וְתִמְצָא נֹפֶשׁ / פֶּן־יִפְגְּעוּ בְךָ הָאֲנָשִׁים מָרֵי נָפֶשׁ” / וַיֹּאמֶר הָאִישׁ “חֲמָסִי עֲלֵיכֶם אַנְשֵׁי הַקְּהִלָּה הַקְּדוֹשָׁה / מַדּוּעַ מִהַרְתֶּם לָתֵת לְאִישׁ נָכְרִי הַיְרֻשָּׁה” / וַיַּעֲנוּ הָעָם אוֹתוֹ קָשָׁה / וַיֹּאמְרוּ “רָאֹה רָאִינוּ צָרַת נַפְשׁוֹ / וְהַתְּפִלִּין אֲשֶׁר בְּרֹאשׁוֹ / וְאֶת־אֲשֶׁר נָפַל עַל־קֶבֶר אָבִיךָ וְהִתְעַסֵּק בְּאֶבְלוֹ / עַד־אֲשֶׁר חָשַׁבְנוּ כִּי לֹא יִחְיֶה אַחֲרֵי נִפְלוֹ / לֹא כַאֲשֶׁר עָשִׂיתָ אַתָּה אֱוִיל מַשְׁרִישׁ / בֵּן מַחְפִּיר וּמֵבִישׁ / כִּי הוּא מִלִּקְרֹא “אָבִי אָבִי” לֹא יַחֲרִישׁ / יִקְרָא “אוֹי וַאֲבוֹי” וְיִשְׁאַג כְּאַרְיֵה / וְנָתַנּוּ לוֹ הַכֹּל בְּטֶרֶם תָּבוֹא וְלֹא אָמַר אַיֵּה / וַנְּבָרֲכֵהוּ גַּם־בָּרוּךְ יִהְיֶה” / וַיֹּאמֶר הָאִישׁ “אָרוּר יִהְיֶה מִן־הָעוֹלָם וְעַד־הָעוֹלָם / הָאִישׁ אֲשֶׁר הִתְאַבֵּל עַל־אָבִי וְהִקְשָׁה אֵלַי וַיִּשְׁלָם / וְלָקַח אֶת־בִּרְכָתִי וְאֶל־בֵּיתִי יְשִׁיבֵנִי דַּךְ וְנִכְלָם” / וַיִּפָּרֵד מֵהֶם וְהוּא בּוֹשׁ וְנֶאֱלָם / וְהָיְתָה לוֹ חֶרְפַּת עוֹלָם.

וַיְמַהֵר וַיִּרְכַּב אֳנִיּוֹת / וַיָּשָׁב אֶל־בֵּיתוֹ בְּיָדַיִם רֵיקָנִיּוֹת / וַתִּשְׁאָלֵהוּ אִמּוֹ עַל־דְּבַר הַיְּרֻשָּׁה / וַיֹּאמַר “הִנְנִי שָׁלוּחַ אֵלַיִךְ קָשָׁה / אֵין קוֹל עֲנוֹת גְבוּרָה רַק קוֹל עֲנוֹת חֲלוּשָׁה / כִּי הָלַךְ שָׁם אִישׁ בְּלִיָּעַל / וּמָעַל בָּנוּ מָעַל / אַל־יִדְרְשֵׁהוּ אֱלוֹהַּ מִמָּעַל / בָּכָה וְסָפַד עַל־אָבִי / וְלָקַח כַּסְפִּי וּזְהָבִי / וְתָלַשׁ שַׂעֲרוֹת רֹאשׁוֹ / וְחִלֵּק מָמוֹן לְכַפָּרַת נַפְשׁוֹ / וְיָרַשׁ כָּל־רְכוּשׁוֹ / הֵקִים עַל־קִבְרוֹ מַצֵּבָה / וְעָשָׂה כַוָּנִים לְהַעֲצִבָה / וּבָנָה אֶצְלוֹ צִיּוּן / וְלָדַעַת מִי הָיָה זֶה הַנָּבָל צָרִיךְ עִיּוּן / הָלַכְתִּי שָׁם לְקַבֵּל הוֹן / וְהֻכֵּתִי בְשִׁגָּעוֹן וּבְעִוָּרוֹן וּבְתִמָּהוֹן / קִבַּלְתִּי חֶרְפָּה מִכָּל־הַחֲרָפוֹת מְקֻבָּצֶת / וְקִלֲלוּנִי קְלָלָה נִמְרָצֶת / שָׂמוּ לִי עֲלִילוֹת / וַיְּבַקְּשׁוּ אֶת־נַפְשִׁי לְהַקְלוֹת / וּלְיַחֲמֶנָּה בַּמַּקְלוֹת / וְעָלַיִךְ דִּבְּרוּ כְּאַחַת הַנְּבָלוֹת / וַאֲנִי זֹאת נֶחָמָתִי אַחֲרֵי אֲשֶׁר הִדְבִּיקַתְנִי הָרָעָה / כִּי לֹא הוֹרַדְתִּי אֶל־אָבִי דִמְעָה / וְלֹא קָרַעְתִּי וְלֹא סָפָדְתִּי / וְעָלָיו חֲרָדָה לֹא חָרָדְתִּי / נָשָׂאתִי אִגֶּרֶת וְעֵדִים הֵעַדְתִּי / וּפָרַשְׂתִּי רִשְׁתִּי וְלֹא לָכָדְתִּי / וְכַאֲשֶׁר אָבַדְתִּי אָבָדְתִּי” / כִּשְׁמֹעַ אִמּוֹ חֶרְפָּתוֹ וְחֶרְפָּתָהּ / וַתֵּרֶא כִּי נוֹחֲלָה אָבְדָה תִּקְוָתָהּ / וַתָּסַר מֵעָלֶיהָ בִּגְדֵי אַלְמְנוּתָהּ / וַתֹּאמֶר כִּי מִלִּבְכּוֹת עָלָיו גַּם־הִיא תֶחֱרָשׁ / אַחֲרֵי אֲשֶׁר מֵהוֹנוֹ לֹא תִירָשׁ / וַתְּהִי לָהֶם לְנֶחָמָה / וְיַחְדָּו הָלְכוּ בִכְלִמָּה".

כִּשְׁמֹעַ הַשַּׂר קוֹל הַחִידָה דּוֹפֵק / לֹא־יָכֹל לְהִתְאַפֵּק / וְאָז נִמְלָא פִיו שְׂחֹק / וְהַשִּׂמְחָה פָּעֲרָה פִיהָ לִבְלִי־חֹק / וְהַקּוֹל נִשְׁמַע עַד־לְמֵרָחוֹק / כְּאִלּוּ נָתַן תְּהוֹם קוֹלוֹ / וְאֵל־הַכָּבוֹד הִרְעִים מִזְּבוּלוֹ / וַיִּשְׁמְעוּ אַנְשֵׁי הָעִיר אֶת־קוֹל הַשַּׂר בְּרֵעֹה / וַיִּשְׁמְעוּ מִצְרַיִם וַיִּשְׁמַע בֵּית פַּרְעֹה / וַיְהִי הָאֲנָשִׁים עַל־שְׂחֹק הַשַּׂר תְּמֵהִים / כִּי לְמַבָּרִאשׁוֹנָה הָיוּ עָלָיו פּוֹחֲדִים וְרוֹהִים / וַיֹּאמֶר הַשָּׂר “צְחֹק עָשָׂה לִי אֱלֹהִים / כָּל־הַשֹּׁמֵעַ יִצֲחַק־לִי / כִּי נַשַּׁנִי אֱלֹהִים אֶת־כָּל־עֲמָלִי” / וַיְהִי כַּאֲשֶׁר־תַּם כָּל־הָעָם לִשְׂחֹק לִרְצוֹנוֹ / וַיֹּאמֶר הַשָּׂר “הֲיָדַעְתָּ מֶה עָשָׂה אֶפְרַיִם הַבּוֹגֵד מֵהוֹנוֹ”.

וָאַעַן וָאֹמַר "זֶה פִּתְרֹנוֹ:

אֶפְרַיִם הָיָה עֻגָה בְּלִי הֲפוּכָה / לֹא הִשְׁאִיר אַחֲרָיו בְּרָכָה / כִּי הָיָה בְרֶשַׁע נוֹדָע / לְכָל־אִישׁ רַע וּבְלִיַּעַל גּוֹאֵל וּמוֹדָע / אָכְלוּ זָרִים כֹּחוֹ וְהוּא לֹא יָדָע / וְאַחֲרֵי־כֵן הִתְחַבֵּר עִם־אַנְשֵׁי לָצוֹן / וּבוֹגְדֵי בֶגֶד יַעֲלוּ עַל־מִזְבְּחוֹ לְרָצוֹן / הָרֹג בָּקָר וְשָׁחֹט צֹאן / יִשְׁתֶּה יַיִן לְשָׂבְעָה / וְיִקְרַב אֶל־כָּל־בְּהֵמָה לְרִבְעָה / עַד־אֲשֶׁר חָלַף יְגִיעוֹ וַעֲמָלוֹ / וַיִּשְׁכֹּן בְּאָהֳלוֹ מִבְּלִי־לוֹ / וְנִשְׁאַר בְּמַקְלוֹ וּבְתַרְמִילוֹ / כִּי הַגַּזְלָן זֶה־מִשְׁפָּטוֹ / וּדְבַר הָעוֹשֵׁק כֹּה מַבָּטוֹ / כְּבוֹלְעוֹ כֵּן פּוֹלְטוֹ".

וַיֹּאמֶר הַשָּׂר “אַף אִם יֵשׁ לִשְׂחֹק עַל־זֶה הַמְּאוֹרָע / הוּא עִנְיָן רָע / לְבֹא אִישׁ חָנַף וּמֵרָע / חָלְקוּ מַחֲמָאֹת פִּיו וּקְרָב־לִבּוֹ / יְקָרֵב כַּתַּנּוּר לִבּוֹ בְּאָרְבּוֹ / וַיִּקַּח אֶת־אֲשֶׁר לֹא־עָמַל בּוֹ / וְאֶחְשֹׁב כִּי זֹאת הָרָעָה לֹא הָיְתָה הָרִאשׁוֹנָה אֲשֶׁר אֶפְרַיִם עָשָׂה / רַק נִגְעוּ בְּעוֹר רַבִּים פָּשָׂה”. / וָאַעַן וָאֹמַר "חַיֶּיךָ אֶפְרַיִם הָיָה אֶבֶן מַעֲמָסָה / וּפַח הָיָה לְרַבִּים וְרֶשֶׁת פְּרוּשָׂה / נִשְׂרְטוּ בוֹ עוֹמְסָיו / וְנוֹקְשׁוּ וְנִלְכְּדוּ בְמַעֲשָׂיו / לֹא תוּכַל הָאָרֶץ לְהָכִיל אֶת־חֲמָסָיו / וְאִתּוֹ חֶבְרַת פָּרִיצִים / עֲצֶרֶת בּוֹגְדִים וּמְנַאֲצִים / יֵשְׁבוּ בְּמוֹשָׁב לֵצִים / מִלְּבַד הַתּוֹעֵבוֹת וְהַשִּׁקּוּצִים / וּבְרִית עִם־הַשִּׁכְּרוּת יִכְרֹתוּ / יוֹם וָלַיְלָה לֹא יִשְׁבֹּתוּ / וּמִן־הַפֶּלֶא אֵיךְ לֹא הָיָה בַזְּמָן כִּלָּיוֹן חָרוּץ / לְרֹב הֱיוֹת הַדּוֹר פָּרוּץ / כִּי רַבּוּ הָעֲבָדִים מֵאֲדֹנֵיהֶם מִתְפָּרְצִים / וְהַמּוֹרִים בְּקַשְּׁתוֹת הָרְכִילוּת וּבַעֲלֵי חִצִּים / אֲשֶׁר בִּידֵיהֶם זִמָּה / וִימִינָם מָלְאָה נֵכֶל וְעָרְמָה / וְחָפְרָה הַבִּינָה וּבוֹשָׁה הַחָכְמָה / כִּי מָלַךְ הַנֵּכֶל בְּעֶבְרַת זָדוֹן / וּנְטֵה יָדוֹ בַּכִּידוֹן / וְהֶחֱזִיק מַלְכוּת בַּחֲלַקְלַקּוֹת / וְהָיוּ דַדֵּי הַמִּרְמָה מֵינִיקוֹת / וּשְׁבוּ דַדֵּי הַתְּבוּנָה לִהְיוֹת צוֹמְקוֹת / וּפְנֵי הַשֵּׂכֶל מוֹרִיקוֹת / וְאֵין מַעֳמָד רַק לִלְשׁוֹנוֹת הַמַּחֲלִיקוֹת / וּלְקֹדְחֵי אֵשׁ וְלִמְאַזְּרֵי זִיקוֹת / וַתִּבְלַעְנָה הַפָּרוֹת הָרָעוֹת וְהַדַּקּוֹת / דֹּבֵר מֵישָׁרִים וְהֹלֵךְ צְדָקוֹת / וּמֹאֵס בְּבֶצַע מַעֲשַׁקּוֹת / וַיִּהְיוּ לְרֹאשׁ אַנְשֵׁי אָיִן / וַאֲחוֹת תּוּבַל־קָיִן / הַשֹּׁתִים בְּמִזְרְקֵי יָיִן / וּבִרְאוֹתִי כִּי בְכַנְפֵיהֶם נִמְצְאוּ דַם נַפְשׁוֹת אֶבְיוֹנִים נְקִיִּים / וְאַנְשֵׁי־צֶדֶק נְתוּנִים מַאֲכָל לְעַם לְצִיִּים / בָּחַרְתִּי מָוֶת מֵחַיִּים / וְלוּ שָׁתְקוּ הָאֱוִילִים וְהַחֲסֵרִים / וְשָׁמְרוּ מַחְסוֹם לְפִיהֶם הַשִּׁכּוֹרִים / אוֹ יְעָרוּ כַדָּם אֶל־הַשֹּׁקֶת / אָז נָשְׁתוּ מֵי־יָם הַמַּחֲלֹקֶת.

וָאֶשָּׂא מְשָׁלִי וָאֹמַר:

הָהּ כִּי מִדַּד / מִרְמָה תִּרְוֶה / תֵּבֵל וּשְׁדֵי / בִינָה צוֹמְקוֹת

וּפְנֵי הָמוֹן / פָּלַח רִמּוֹן / וּפְנֵי הַשָׂ־ / רִים מוֹרִיקוֹת

אֵיכָה הַיְקוּם / יֵעְמֹד יוֹם קוּם / כִּתּוֹת עַל־הַ־ / צֶּדֶק חוֹלְקוֹת

גָּזְלוּ מִיָּ־ / רָבְעָם עֶגְלֵי / זָהָב אַף מֵ־ / עָמְרִי חֻקּוֹת

וּלְשׁוֹנָם חֶ־ / רֶב וִידֵיהֶם / מָלְאוּ דָמִים / כַּמִּזְרָקוֹת

הָהּ עַל־רֹאשׁ אִישׁ / כַּעַשׂ יַחְרֹשׁ / הַפּוֹרֶה רֹאשׁ / אִישׁ מַעְשַׁקּוֹת

אֵין מָעֲמָד רַק / לַנִּרְגָּנִים / אוֹ לִלְשׁוֹנוֹת / הַמַּחְלִיקוֹת

וּלְאִישׁ עוֹיֵן / הַבּוֹחֵר יֵין / מַלְכוּת בִּכְלֵי / זָהָב הַשְׁקוֹת

נַפְשִׁי בִּרְאוֹת / כָּל־הַמֹּצְאוֹת / רַעְיוֹנֶיהָ / מָוֶת חוֹשְׁקוֹת

לוּלֵי יִרְאַת / אֵל עַל־פָּנַי / הָיוּ יָדַי / אוֹתִי חוֹנְקוֹת

כִּי מָה חֵפֶץ / עוֹד בַּחַיִּים / וִיאוֹרֵי הַ־ / בִּין שׁוֹקֵקוֹת

שָׁפַל רוּם הַ־ / בִּין וּפְתַיּוּת / הֶחְזִיק מַלְכוּת / בַּחְלַקְלַקּוֹת

מַדּוּעַ הַ־ / פָּרוֹת טוֹבוֹת / תִּבְלַעְנָה אוֹ־ / תָם הַדַּקּוֹת

מַה־לַּשֵּׂכֶל / וּלְצֹאן אוֹבְדוֹת / עָלוֹת וּגְמַ־ / לִּים מֵינִיקוֹת

אֵיךְ לֹא־יִשְׁתֹּק / אִישׁ סָכָל וּמְ־ / חַרְחֵר־רִיב וּמְ־ / אַזֵּר זִיקוֹת

אִלּוּ שָׁתַק / אִישׁ לֹא יֵדַע / אָז בָּטְלוּ כָּל־ / הַמַּחְלוֹקוֹת.
