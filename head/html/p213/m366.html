אופן

אֶרְאֶלִּים / וְחַשְׁמַלִּים / לַאֲלָפִים / וְרִבּוֹאוֹת

נִקְבָּצִים / וּמַעֲרִיצִים / בְּרֹב אֵימוֹת / וְנוֹרָאוֹת,

לְהַקְדִּישָׁם / יַחַד שֵׁם / אֵל עוֹשֵה / נִפְלָאוֹת –

אוֹמְרים “קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!”

וּשְׂרָפִים / אֲשֶׁר עָפִים, / בַּעֲלֵי שֵׁשׁ / כְּנָפַיִם,

מִמַּעַל / אֶל מַעַל / חוּג אֶרֶץ / וְשָׁמַיִם –

דִּבְרֵיהֶם / וְכַנְפֵיהֶם / כְּקוֹל הָמוֹן / כְּקוֹל מַיִם,

כְּנִיב נִמְרָץ / לְאֵל נַעֲרָץ / בִּנְבוּאוֹת / נִבָּאוֹת.

בְּכֵס נִקְבָּע / בְּרֹאשׁ אַרְבַּע / דְמוּת סַפִּיר / הַטְּהוֹרָה.

נִיב רַחְשָׁם – / וְעַל רֹאשָׁם / כְּעֵין קֶרַח / הַנּוֹרָא –

דְּמוּת בָּזָק / בְּקוֹל חָזָק / מְבָרְכִים בְּשָׂ- / פָה בְרוּרָה,

יְרוֹמֵמוּ / וְיַנְעִימוּ / קְדֻשָּׁה כַּ- / אֲשֶׁר תֵּאוֹת.
