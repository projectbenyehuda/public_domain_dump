ארץ ושמים וחילם

חתימה: אלשבזי משתא שצ

אֶרֶץ וְשָׁמַיִם וְחֵילָם עוֹמְדִים לַעְשׂוֹת / רְצוֹן קוֹנָם וְשִׁיר יַבִּיעוּ

לוֹ חוֹשְׁקִים וּמְהַלְּלִים וּמְיַחֲדִים / וּבְנֵי אֱלֹהִים בַּמְּרוֹם יָרִיעוּ

שְׂרָפִים סְבִיב כִּסְאוֹ בְּרַעַשׁ סוֹגְדִים / עִם גַּלְגְּלֵי מַעְלָה בְּקוֹל יָנִיעוּ

בָּהֶם שְׂכָלִים גַּם נְפָשׁוֹת סוֹעֲדִים/ מִזִּיו שְׁכִינָתוֹ וְאוֹר יַזְרִיעוּ

זַרְזִיף אֲרָצוֹת לַטְּלָלִים חוֹמְדִים / הֵמָה עֲלֵיהֶם אוֹר וְטָל יַשְׁפִּיעוּ

יַחַד בְּשֵׁם הָאֵל אֲזַי הֵם עוֹמְדִים / מוֹדִים שְׁבָחֶיךָ וְהֵם יָזוּעוּ

מִיָּם דְּאוּקְיָנוֹס אֵדִים צוֹעֲדִים / עוֹלִים כְּמוֹ נֹאדוֹת עֲדֵי יַגִּיעוּ

שֶׁפַע יְקַבֵּלוּ וְלַכֹּל יוֹרְדִים / גִּשְׁמֵי בְרָכָה בַּאַהֲבָה יוֹפִיעוּ

תַּרְוֶה אֲרָצוֹת בָּם זְרָעִים סוֹלְדִים / כָּל בַּעֲלֵי חַיִּים בְּטוֹב יַשְׂבִּיעוּ

אַרְבַּע תְּקוּפוֹת בָּם מְזָגִים נִצְמְדִים / וַארְבַּע יְסוֹדוֹת זֶה לְזֶה יַכְרִיעוּ

שְׁבָח אוֹמְרִים הַכֹּל לְאֵל חַי / עוֹבְדִים חַסְדּוֹ וְטוּבוֹ זֶה לְזֶה יוֹדִיעוּ

צוּרִי פְּדֵה עַם קָדְשְׁךָ הַנּוֹדְדִים / וּבְקוֹל מְבַשֵּׂר לִי בְּחֵן יַשְׁמִיעוּ:
