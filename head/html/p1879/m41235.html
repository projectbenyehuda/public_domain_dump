צְאוּ אַחַי

זה השיר ששלח להגביר השר ונגיד יצחק דוד ישועה גבאי ש׳ התקכ״ג

סימן: צאלח


צְאוּ אַחַי רְאוּ שִׁיר לֹא יְסֻלֶּה / בְּכֶתֶם טוֹב וְזֵר זָהָב לְמִסְגַּרְתּוֹ

אֲשֶׁר שַׁרְתִּי לְשַׂר נֵזֶר חֲבֵרָיו / וְכֻלָּם הוֹלְכִים סָרִים לְמִשְׁמַעְתּוֹ

לְכוּ אַחַי מְשָׁחוּהוּ עֲלֵיכֶם / לְשַׂר קָצִין וְלֹא תִשְׁגּוּ בְזוּלָתוֹ

חֲנוּ סָבִיב לְאָהֳלוֹ וְאִכְלוּ / לְשָׂבְעָה צוּף דְּבַשׁ חָלָב מְלִיצָתוֹ

לְמַעַן תִּינְקוּ מִשַּׁד אֲמָרָיו / וְתִרְעוּ מִבְּשָׂמֵי חֵן עֲרוּגָתוֹ

וְאִישׁ חָכָם אֲשֶׁר יִפְרַע דְּבָרַי / אֱמֶת סָכָל וְהוּא חוֹמֵס יְחִידָתוֹ

רְאוּ קוֹמָה אֲשֶׁר דָּמְתָה לְתָמָר / וְעֵץ זַיִת עֲלֵי מַיִם בְּנִצָּתוֹ

וְעֵינַיִם כְּעֵינֵי הַצְּבָיִים / וְנֹפֶת הַשְּׂפָתַיִם בְּדִבְרָתוֹ

וְאִם אָבוֹא לְסַפֵּר מַהֲלָלָיו / לְשׁוֹנִי יִהְיֶה נֶאְלָם לְחֻלְשָׁתוֹ

אֲבָל דֶּרֶךְ כְּלָל אֹמַר הֲכִי הוּא / בְּלִי חֶסְרוֹן בְּצַד מָה מִשְּׁלֵמוּתוֹ

וְלָכֵן נִקְשְׁרָה נַפְשִׁי בְּנַפְשׁוֹ / וְהוּא קֹטֶב וְנַפְשִׁי הִיא חֲגוֹרָתוֹ

וְהִשְׁקַנִי זְמַנִּי מֵי מְרִירוּת / עֲכָרַנִי לְכָדַנִי בְעָרְמָתוֹ

אֲשֶׁר סִבַּב וְגָדַר אֶת יְדִידִי / וְנִשְׁאַרְתִּי כְאִישׁ בָּדָד בְּצָרַעְתּוֹ

וּמִי יִשְׁמַע וּמִי יַקְשִׁיב דְּבָרַי / חֲמָסִי עַל זְמַנִּי לֹא עֲלֵי בִלְתּוֹ

תְּפִלָּתִי לְבוֹרֵא הַשְּׁחָקִים / וְרוֹכֵב עַל שְׁמֵי קֶדֶם בְּגַאְוָתוֹ

יְקַבֵּל אֶת תְּפִלָּתִי לְמַעַן / זְכוּת אַבְרָם וְיִצְחָק וַעֲקֵדָתוֹ

וְיַשְׁלִים אֶת רְצוֹן דּוֹדִי וְחֶפְצוֹ / יְמַלֵּא מִשְׁאֲלוֹת לִבּוֹ לְטוֹבָתוֹ

וְיַחְזֹר אֶל נְוֵה קָדְשׁוֹ בְּכָבוֹד / שְׂבַע רָצוֹן וְשַׁלְוָתוֹ בְּתוֹךְ בֵּיתוֹ

אֲזַי אוֹדֶה לְאֵל נוֹרָא עֲלִילָה / אֲדוֹן עוֹלָם וְאֵין קֵץ אֶל גְּדֻלָּתוֹ.
