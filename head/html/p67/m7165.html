[הָבָה נִתְחַדֵּשָה...]

(מֻקְדָש לְ“עֹנֶג שַׁבָּת” בְּאֶרֶץ יִשְרָאֵל)

הָבָה נִתְחַדֵּשָׁה, הָבָה נִתְקַדֵּשָׁה,

הַשַׁבָּת כִּי בָּאָה; ב

כָּל יְמוֹת הַחֹל נָשָׂאנוּ אֶת הָעֹל, }

וְלַשַׁבָּת – הַלְוָאָה }ב


הַלְוָאָה, הַלְוָאָה, הַלְוָאָּה.

אוֹי וַאֲבוֹי, אִם אָיִן; ב

בְּלִי בָּשָׂר וָדָג אֶפְשָר לַעֲשוֹת חָג }

אַךְ לֹא בְּלִי יָיִן. } ב


יַיִן, יַיִן, יָיִן.

יַיִן שְׁתוֹת אֲמַהֵרָה; ב

וְאִם אֶשְׁתֶּה, זֶה אַחַר זֶה, }

מְעַט לִי אֶשְׁתַּכֵּרָה. } ב

אֶשְׁתַּכֵּרָה, אֶשְׁתַּכֵרָה, אֶשְׁתַּכֵּרָה.

יַיִן אֵינוֹ מָיִם; ב

וְאִם אֶשְׁתֶּה, זֶה אַחַר זֶה, }

תִּסְתּוֹבֵב יְרוּשָלָיִם. } ב


יְרוּשָלַיִם, יְרוּשָׁלָיִם, יְרוּשָלָיִם.

כָּל גּוּפִי אַרְקִידָה; ב

וְאִם אֶשְׁתֶּה, זֶה אַחַר זֶה {

אֶרְקֹד גַּם עִם חֲסִידָה. } ב


חֲסִידָה, חֲסִידָה, חֲסִידָה.

אוֹי, זֶה חֵטְא לֹא מְעַט הוּא; ב

אֲבָל אֶשְׁתֶּה, זֶה אַחַר זֶה, }

כִּי הַיּוֹם שַבָּת הוּא! } ב


נִגּוּן עֲמָמִי
