כשמש מרומים

כְּשֶׁמֶשׁ מְרוֹמִים הֲכִי אַתְּ, גְּבִירָה, / וְאִם אַתְּ בְּפִי כָל אֲנָשִׁים צְעִירָה,

וְעָלִית, וְרָחַק מְקוֹמֵךְ, עֲדֵי כִי / חֲשָׁבוּךְ עֲלֵי רֹאשׁ שְׁחָקִים צְפִירָה.

וְאַל תִּתְמְהִי, כִּי אֲנִי כֵן בְּעֵינָם / כְּיֶלֶד בְּזוּי גֵַּו וְנֶפֶשׁ יְקָרָה.

וְאִישׁ בֶּאֱמֶת לֹא יְדָעֵךְ, יְדַמֶּה / אֲשֶׁר תֵּלְכִי אַט ־ וְלֶכְתֵּךְ מְהֵרָה

וְכָזֹאת יְדַמּוּ אֲחוֹתֵךְ בְּרָאשֵׁי / חֳדָשִׁים חֲסֵרָה ־ וְהִיא לֹא חֲסֵרָה

וְכָכָה יְשִׂימוּן יְדִידַי יְדִידוּת / יְדִידָם חֲסֵרָה ־ וּבִמְאֹד יְתֵרָה

אֲשֶׁר הָלְכוּ לַחֲזוֹת צִיץ עֲרוּגוֹת / וְשָׁם חִבְּקוּ אַחֲרֵי כֵן עֲקָרָה.

הֲקִנֵּאת לְאוֹרֵךְ בְּשׁוּרֵךְ פְּנֵיהֶם / וְהִלְבַּשְׁתְּ אוֹרָם כְּתֹנֶת שְׁחֹרָה

וְאֵיךְ תּוּכְלִי אוֹר לְהַסְתִּיר ־ וְצוּרַת / פְּנֵיהֶם פְּנֵי הָאֲדָמָה מְאִירָה?

וְרֵיחַ בְּשָׂמִים בְּרֵעַי מְרַגֵּל / לְאַפִּי, וְרוּחַ תְּרַחֵף ־ בְּשׂוֹרָה.

בְּגַדְתֶּם בְּרֵעַ עֲזַבְתֶּם, וְנַפְשׁוֹ / בְּנֶפֶשׁ צְרוֹר אַהֲבַתְכֶם צְרוּרָה

וְהָיָה לְבָבוֹ לְמַיִם ־ וְצָמֵא / יְאַוֶּה הֱיוֹת רֹאשׁ פְּתָנִים תְּמוּרָה

וְלוּ כָהֲתָה עֵין חֲסַר עַם בְּקִרְבּוֹ ־ / אֲזַי הָיְתָה מִכְּלוֹתָהּ סְתוּרָה

וְתֻשְׁלַךְ בְּלֶב יָם גְּוִיַּת אֲחִיכֶם ־ / מְנָעָהּ כְּלוֹת הַשְּׁאֵר מִנְּעוּרָהּ

וְלוּ דַק נְמָלִים נְשָׁכוֹ ־ נְשָׂאוֹ / בְּפִיהוּ וְאָבַד בְּפִי הַמְּעָרָה.

שְׁנוֹתָיו בְּתֵבֵל בְּשִׁשִּׁית שְׁמוֹנָה / וּמַחְצִית שְׁלֹשִׁים וְשִׁשִּׁית עֲשָׂרָה

וְכִמְעַט יְהִי אוֹת בְּשַׂעְרַת לְחָיָיו / אֲשֶׁר הִיא שְׁחֹרָה ־ וְתָשׁוּב צְחֹרָה

חֲמָסִי, יְדִידִי, עֲלֵי הָאֲדָמָה / אֲשֶׁר מִמְּחִיר יוֹם לְפוֹתִים מְכוּרָה

וְלוּלֵי כְסִילִים אֲשֶׁר נִשְּׁאוּ בָךְ ־ / נְשָׂאתִיךְ וְשַׂמְתִּיךְ לְכַפַּי נְעוּרָה

אֲדַמֶּה לְקַבֵּר יְגוֹנִי בְּנַפְשִׁי ־ / וְהָיָה יְגוֹנִי לְנַפְשִׁי קְבוּרָה

וְתִדַּד שְׁנָתִי לְזֹאת כָּל שְׁנָתִי, / כְּאִלּוּ שְׁנָתִי לְעֵינַי אֲסוּרָה

וְהָמוּ קְרָבַי עֲדֵי יָם חֲשַׁבְתִּים, / וְחַם, עַד קְרָאתִיו לְבָבִי מְדוּרָה

וְאֶבְעַר לְעַם בּוֹעֲרִים, הֵם בְּנַפְשָׁם / כְּזִמְרַת אֲנָשִׁים ־ וְהֵם כַּזְּמוֹרָה

וְשֵׁפֶל וּבוּז לָבְשׁוּ, לֹא! אֲבָל כִּי / כְּתֹנֶת כְּסִילוּת בְּעוֹרָם תְּפוּרָה

עֲווֹנָם יְגַלּוּ וְשִׂנְאָה יְכַסּוּ, / וְתֵרָא בְעֵינָם ־ וְעֵינָם סְגוּרָה

מְעֻוָּת כְּגָמָל, וְשָׁפָל כְּנָמָל, / וְנָלוֹז כְּשׁוּעָל, וְסוֹרֵר כְּפָרָה

אֱמֹר לַפְּתָאִים: שְׁבוּ, אַל תְּנוּעוּן / לְהָעִיר חֲמַת אִישׁ כְּנוֹעַ כְּבָרָה

אֲשֶׁר שָׁת תְּבוּנָה לְמַקֵּל בְּיָדוֹ / וּמֵזַח מְזִמּוֹת לְמָתְנָיו חֲגוֹרָה

וְאַל תַּאֲמֵן בָּם, אֲבָל כִּי בְעֵדוּת / יְעִידוּן לְאִמָּם אֲשֶׁר הִיא עֲקָרָה

וְאֵיךְ יִפְרְצוּ לַעֲלוֹת אֶל מְקוֹמִי ־ / וְיָדָם לְהַשִּׂיג לְגָדֵר קְצָרָה!

וְגֻלַּת שְׁחָקִים עֲצוּרָה בְלִבִּי / וְתֵבֵל בְּגֻלַּת שְׁחָקִים עֲצוּרָה

וְאִם הֵם יְקַלּוּ דְבָרִי אֲשֶׁר עַז ־ / וְחוֹלָה תְחַלֵּל, וְאִשָּׁה מְצֵרָה?

וְאִם הֵם כְּשַׁיִת ־ אֲנִי הַמְּדוּרָה, / וְאִם הֵם כְּרוּחַ ־ אֲנִי הַסְּעָרָה

וְאִם הֵם כְּלָבִים ־ אֲנִי הַמְּגָרֶה, / וְאִם הֵם כְּשָׁמִיר ־ אֲנִי הַמְּגֵרָה

וְלוּ יָדְעוּן כָּל יְִפִי הַוְּרָדִים ־ / עֲזָבוּם, וְלֹא נִשְׂרְפוּ בַבְּעֵרָה

וְאִם נָשְׁכָה הַדְּבוֹרָה בְּשַׂר אִישׁ ־ / וְלֹא יָדְעָה אַחֲרִיתָהּ דְּבוֹרָה!
