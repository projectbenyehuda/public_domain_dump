משל פד: סוּס וְסֹחֵר וְאִישׁ

אֲשֶׁר בָּאָדָם יֶחֱסֶה / לֹא יִשְׁאַל וְלֹא יְנַסֶּה


סוּס נָאֶה הָיָה לְסֹחֵר / וּשְׁכֵנוֹ אֶת פָּנָיו שֹׁחֵר / לֵאמֹר: מְכֹר לִי סוּסֶךָ / אֶת עֶרְכּוֹ קַח וְשִׂים בְּכִיסֶךָ / וְהִנֵּה הַסֹּחֵר עָנָהוּ / הַסּוּס בֵּינְךָ וּבֵינִי מַה הוּא / לְהַזְכִּיר דָּמָיו אַל תּוֹסֵף / הִנֵּה הוּא לְךָ בְּלֹא כָסֶף / וְזֶה עָנָה: לֹא כֵן יִהְיֶה / שֹׂנֵא מַתָּנוֹת יִחְיֶה / אַךְ אֶת דָּמָיו גַּלֵּה אָזְנִי / וְאֶת הַכֶּסֶף קַח מִמֶּנִּי / וַיַּעַן לוֹ הַסֹּחֵר / לְתִתּוֹ לְךָ אֶהְיֶה בֹחֵר / אָמְנָם זֶה הַסּוּס לֹא מְצָאתִיו / בְּעֶשְׂרִים זְהוּבִים קְנִיתִיו / וְהוּא שָׁוֶה כֵּפֶל / כִּפְלַיִם מֵאַרְבָּעִים זְהוּבִים לֹא נֹפֵל / וְאִם בְּחִנָּם לֹא תַחְפֹּץ אֹתוֹ / עַל פִּי מַעֲרִיךְ תִּהְיֶה מִקְנָתוֹ / וְנַשְׁכִּימָה לַכְּרָמִים שְׁנֵינוּ / וְרִאשׁוֹן הַיּוֹצֵא לִקְרָאתֵנוּ / כַּאֲשֶׁר יַעֲרִיךְ בֵּין טוֹב וּבֵין רָע / לֹא תוֹסִיף עָלָיו וְלֹא תִגְרָע / וַיָּקוּמוּ בַבֹּקֶר וּבִמְסִלּוֹתָם / אִישׁ יוֹצֵא לִקְרָאתָם / מֵעֵין יְמִינוֹ הָיָה נִשְׁחָת / וְאֵין לוֹ רַק עַיִן אַחַת / וַיֹּאמְרוּ לוֹ: אַל תִּתֵּן לַחֲטֹא חִכֶּךָ / הִנֵּה סוּס זֶה נִמְכָּר כְּעֶרְכֶּךָ / דִּין עֶרְכּוֹ אַתָּה פְלִילוֹ / כֶּסֶף עֶרְכְּךָ עָלָיו וְקָם לוֹ / וַיֹּאמַר: חָלִילָה לִי לְשַׁקֵּר בֶּאֱמוּנָתִי / עֵרֶךְ הַסּוּס בְּלִבִּי תַרְתִּי וְרָאִיתִי / כִּי הוּא שָׁוֶה זְהוּבִים חָמֵשׁ / סוּס כָּמֹהוּ רָאִיתִי אֶמֶשׁ / בְּחָמֵשׁ זְהוּבִים נִמְכָּר / עַד הָרֹעֶה וְהָאִכָּר / וַיִּזְעַף הַסֹּחֵר וַיֹּלֶךְ מִשָּׁם סוּסוֹ / וּשְׁכֵנוֹ הִזִּיל זָהָב מִכִּיסוֹ / חָמֵשׁ זְהוּבִים וְלֹא אָבָה / בַּעַל הַסּוּס, וְזֶה בְּאַהֲבָה / מֵבִיא הַזָּהָב בְּאָהֳלוֹ / אוּלַי יְפֻתֶּה וְיוּכַל לוֹ / וַיֹּאמַר: תֶּן לִי הַסּוּס, וְזֶה עֶרְכּוֹ / וְזֶה יַמְרֶה עָלָיו חִכּוֹ / הַקּוֹנֶה צוֹעֵק לַשֹּׁפְטִים / וְחַכְמֵי לֵב אֵלָיו נֹטִים / צִוּוּ לְהָשִׁיב הַסּוּס אֵלָיו / אִם כֹּפֶר יוּשַׁת עָלָיו / הַסֹּחֵר נִתְיָעֵץ בִּמְזִמָּה / בִּדְבַר מִרְמָה וְתֹךְ וְעָרְמָה / וַיְסַפֵּר אֲלֵיהֶם כֹּה וָכֹה / וְכִי הָאִישׁ אֲשֶׁר הֶעֱרִיכוֹ / אִם הָיוּ לוֹ שְׁתֵּי עֵינַיִם / יַעֲרִיכֶנּוּ פִי שְׁנַיִם / אַךְ בְּנִקְרַת עֵין יְמִינוֹ / עֵת נָתַן בַּסּוּס עֵינוֹ / בַּחֲצִי רְאִיָּה רָאָהוּ / עַל כֵּן בַּחֲצִי עֵרֶךְ יַעֲרִיכֵהוּ / אָז אָמְרוּ הַשֹּׁפְטִים: יְהִי לְךָ אֲשֶׁר לְךָ / כִּי הַשֵּׂכֶל וְהַבִּינָה נָפְלָה בְגוֹרָלְךָ / כַּאֲשֶׁר צָדַקְתָּ מִמֶּנּוּ / כִּי בְעֵינוֹ אַחַת לֹא יַעֲרִיכֶנּוּ.


וְהַמָּשָׁל כֵּן יִקְרֶה לָאִישׁ אֲשֶׁר בָּאָדָם יֶחֱסֶה / מַה תִּשְׁאַל וּמַה תְּנַסֶּה / כִּי בְּהִשָּׁעֶנְךָ עָלָיו תִּשָּׁבֵר / בְּלֵב וָלֵב לְךָ דֹבֵר / כַּאֲשֶׁר הַסֹּחֵר אָמַר לִשְׁכֵנוֹ / סוּסִי בְלֹא כֶסֶף לְךָ הִנּוֹ / וּפִי שְׁנַיִם עַל דָּמָיו אָבָה לְמָכְרוֹ / וְאִישׁ כָּזֶה לֹא חָפֵץ בִּיקָרוֹ / וְעוֹד תָּבִין מָשָׁל וּמְלִיצָה / אַל תָּבוֹא בְּמִשְׁפָּט בְּלֹא עֵצָה / הִתְיָעֵץ בִּמְזִמּוֹתֶיךָ / טֶרֶם תְּדַבֵּרְנָה שִׂפְתּוֹתֶיךָ / כַּאֲשֶׁר הַסֹּחֵר בְּהִשָּׁפְטוֹ / הוֹצִיא לָאוֹר מִשְׁפָּטוֹ.


וָאֶשָּׂא מְשָׁלִי וָאֹמַר:


אִם חֶסֶד תּוֹצִיא מִן כֵּלַי / מַיִם מִצּוּר הוֹצֵא תוּכַל

כִּי כָל מַסְמֵר מִבַּרְזֶל רַע / חֹשֵׁב לִכְפֹּף אֹתוֹ סָכָל.
