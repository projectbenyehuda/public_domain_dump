משל קג: אִישׁ חֹרֵשׁ חֲרִישׁוֹ

טוֹב לָתוּר וְלִדְרֹשׁ בִּלְבָבוֹ / מִי הֶעֱבִיר עַל פָּנָיו טוּבוֹ


אִישׁ חֹרֵשׁ חֲרִישׁוֹ, וּבְאֵתוֹ / יְפַתַּח וִישַׂדֵּד אַדְמָתוֹ / וַיִּמְצָא שָׁמָּה אֹסֶף / אֶבֶן טוֹבָה זָהָב וָכָסֶף / וִיקַטֵּר לְמִכְמַרְתּוֹ וִיזַבֵּחַ לְחֶרְמוֹ / וּמִצֹּאן וּמִבָּקָר אֲשֶׁר עִמּוֹ / הֶעֱלָה עוֹלוֹת לַמָּקוֹם הַהוּא / וּבְמַאֲמַצֵּי כֹחוֹ כִּבְּדָהוּ / וּלְעַמּוֹ קָרָא לְכַבֵּד הַמָּקוֹם: עֲשׂוּ כָמוֹנִי / זֹבֵחַ תּוֹדָה יְכַבְּדָנְנִי / כִּי כָּל יָמַי אֲכַבְּדֵהוּ בְמַהֲלָלִי / אַךְ עָשַׁרְתִּי מָצָאתִי אוֹן לִי / וְלֹא יָסַף עוֹד לַחֲרֹשׁ וְלִזְרֹעַ / לִקְצֹר וְלִתְפֹּר וְלִקְרֹעַ / וְלֹא לְהַשְׁלִיךְ עַל תַּלְמֵי שָׂדַי דֹּמֶן / חֻבַּל עֹל מִפְּנֵי שָׁמֶן / אַךְ יְנַשֵּׁק הַמָּקוֹם וִיכַבְּדֶנּוּ בְמַהֲלָלָיו / וְהִנֵּה קוֹל הַזְּמָן מְדַבֵּר אֵלָיו / מַה לְּךָ פֹּה כִּי תִשְׁתַּחֲוֶה וּתְכַבֵּד אַדְמָתֶךָ / הֲלֹא יִרְאָתְךָ כִּסְלָתֶךָ / מִמּוֹצָא הַכֶּסֶף אֲשֶׁר אָהַבְתָּ / וּבוֹרוֹת חֲצוּבִים אֲשֶׁר לֹא חָצַבְתָּ / אֹתִי לֹא זָכַרְתָּ בְרַעְיוֹנֶךָ / אֲשֶׁר הִקְרֵיתִי הָאוֹצָר לְפָנֶיךָ / לָכֵן אָשׁוּב וְלָקַחְתִּי מֵאִתְּךָ זְהָבִי וְכַסְפִּי / וְתַחְתֵּיהֶם אָעִיר עָלֶיךָ קִצְפִּי / וְלֹא תְכַבֵּד עוֹד הָאָרֶץ בְּמַהֲלָלָךְ / מִזְּהָבִי וְכַסְפִּי אֲשֶׁר נָתַתִּי לָךְ / כִּי מִיּוֹם הַהוּא נָשָׂאתָ רֹאשְׁךָ / וְנָתַתָּה הַכֶּסֶף בְּכֹל אֲשֶׁר תְּאַוֶּה נַפְשְׁךָ / וַאֲנִי עָלֶיךָ עוֹד לֹא אֲרַחֵם / בְּזֵעַת אַפֶּיךָ תֹּאכַל לֶחֶם / עַד שׁוּבְךָ אֶל הָאֲדָמָה / עָרֹם יָצָאתָ מִבֶּטֶן אִמְּךָ וְעָרֹם תָּשׁוּב שָׁמָּה / וְהַגָּאוֹן אֲשֶׁר בְּלִבְּךָ צָמַח / צֶמַח בְּלִי יַעֲשֶׂה־קֶּמַח / וַיִּרֶף מִמֶּנּוּ וַיְהִי לוֹ כֵן / בְּבֹאוֹ לִמְנוֹת אוֹצָרוֹ וְהָכֵן / וַיְהִי אַךְ יָצֹא יָצָא / וַיְחַפֵּשׂ וְלֹא מָצָא / וַיְשִׁיבֵהוּ עַל כַּנּוֹ כַּמִּשְׁפָּט הָרִאשׁוֹן / לַמַּחֲרֵשׁוֹת וְלָאֵתִים וְלִשְׁלֹשׁ קִלְּשׁוֹן / לִקְצֹר קְצִירוֹ וְלַחֲרֹשׁ חֲרִישׁוֹ / וַתִּכָּנַע בֶּעָמָל נַפְשׁוֹ.


וְהַמָּשָׁל לָתוּר וְלִדְרֹשׁ בִּלְבָבוֹ / מִי הֶעֱבִיר עַל פָּנָיו טוּבוֹ / וּלְהָשִׁיב לוֹ כִגְמוּלוֹ / וּלְהַלֲלוֹ בְּכָל מַהֲלָלוֹ / לְמַלֵּא שְׁאֵלָתוֹ וְלַעֲשׂוֹת בַּקָּשָׁתוֹ / לֹא יַחֲלִיפֶנּוּ וְלֹא יָמִיר אֹתוֹ / לְהַלֵּל אַחֵר בִּמְקוֹמוֹ / אַךְ זֶה יְכַבֵּד וְעַמּוֹ / וּפִיו וְיָדוֹ מֶנּוּ לֹא תִקְצָר / כַּאֲשֶׁר עָשָׂה בַּעַל הָאוֹצָר / אֲשֶׁר לֹא כִבֵּד הַמַּגְדִּיל כְבוֹדוֹ וְהוֹדוֹ / וַיְחַבֵּק וַיְנַשֵּׁק לַאֲשֶׁר לֹא כִבְּדוֹ.
