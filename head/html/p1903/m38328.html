משל עט: קוֹף וְשֻׁעָל

הוֹי הָאֹמֵר לָרֵעַ / קָצַר הַמַּצָּע מֵהִשְׂתָּרֵעַ


קוֹף רָאָה שֻׁעָל עַל נְתִיבוֹ / אֲרֻכָּה וּרְחָבָה זְנָבוֹ / עַד עָפָר יַגִּיעֶנָּה / וּבְרַגְלָיו יִרְמְסֶנָּה / קוֹף הָלַךְ עַד תְּכוּנָתוֹ / הִפִּיל לְפָנָיו תְּחִנָּתוֹ / לֵאמֹר: הִנְנִי קוֹף אֹהַבְךָ / תְּנָה לִי מְעַט קָט מִזְּנָבְךָ / בְּלִי זָנָב מִבֶּטֶן יָצָאתִי / וַחֲשׂוּפֵי שֵׁת עֶרְוָתִי / לְכַסּוֹת בָּהּ דַּי לְךָ וְהוֹתֵר / וְהַיִּתְרוֹן יְהִי לָךְ לְבָתֶר / אוֹ לִבְנִי גָדוֹל יִהְיֶה / וְיִתְפַּלֵּל בַּעַדְךָ וְתִחְיֶה / בָּחֳרִי אַף עָנָה הַשֻּׁעָל / כְּדֵי קֶצֶף וָחֳרִי וָמָעַל / אֲשֶׁר מִזְּנָבִי שָׁאַלְתָּ / אֲשֶׁר לֹא עָמַלְתָּ בּוֹ וְלֹא גִּדַּלְתָּ / אִם יִהְיֶה לִי כִפְלֵי כִפְלַיִם / לֹא אֶתֵּן לְךָ לְכַסּוֹת בּוֹ אֶת הַשּׁוּלַיִם / גַּם בִּנְךָ מִמֶּנָּה לֹא יַגְדִּיל הוֹדוֹ / וְהִגַּעְתִּיהוּ אֶל הָאָרֶץ וְנִגְלָה יְסוֹדוֹ.


וְהַמָּשָׁל בְּאַבִּירֵי לֵב הָרְחוֹקִים מִצְּדָקָה / עֵת יִשְׁמְעוּ מִדַּלִּים צְעָקָה / וְאֵין אֶחָד אֹמֵר הִנֵּה / שָׁם יִצְעֲקוּ וְלֹא יַעֲנֶה / וְעֵת יַעֲנוּ יֶחֱרֶה אַפָּם / כִּי יָקָר בְּעֵינֵיהֶם כַּסְפָּם / כִּיסָם רָחָב לִבָּם קָצָר / מִלְּחוֹנֵן דַּלִּים בַּצַּר / וְיִרְאוּ הַצַּדִּיק נֶעֱזָב / וְיָעָה בָרָד מַחְסֵה כָזָב / וְהוֹי הָאֹמֵר לָרֵעַ / קָצַר הַמַּצָּע מֵהִשְׂתָּרֵעַ.
