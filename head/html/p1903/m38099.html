משל לח: אַרְנֶבֶת וּכְלָבִים

אֲשֶׁר הַזְּמָן עָלָיו שָׁלַט / לֹא יָחִישׁ לוֹ מִפְלָט


אַרְנֶבֶת וּבָנֶיהָ אָמָרוּ / לַעֲזֹב מְקוֹמָם אֲשֶׁר בּוֹ גָּרוּ / כִּי לֹא מָצְאוּ שָׁם נֹפֶשׁ / וְהַכְּלָבִים עַזֵּי נֶפֶשׁ / כִּי תָמִיד אַחֲרֵיהֶם יִרְדֹּפוּ / יָרוּצוּ וְלֹא יִיגְעוּ יֵלְכוּ וְלֹא יִיעָפוּ / וַיֹּאמְרוּ: הֵן נִשְׁאַרְנוּ מְעָט מֵהַרְבֵּה / וְאֵשׁ זַעְמֵנוּ אֵין מְכַבֶּה / כִּי גַם בַּחֲדָרִים אֵימָה / וְרוּחַ עוֹד בָּנוּ לֹא קָמָה / נִסְעָה וְנֵלְכָה הַשָּׂדֶה / אֲשֶׁר אֵין שָׁם אֹרֵב וְצֹדֶה / וַיַּעַן אֶחָד מִן הַמִּשְׁפָּחָה / אֲשֶׁר רוּחַ חָכְמָה בוֹ נָחָה / אִם אֶת עֲצָתִי תֹאבוּ / וְרִבוֹת שְׂפָתַי תַּקְשִׁיבוּ / אַל נָא תָמוּשׁוּ מִזֶּה / כִּי הַרְבֵּה נֹסְעִים אֶחֱזֶה / מִמְּקוֹמָם וּמֵאֶרֶץ מוֹלַדְתָּם / וְשָׁבוּ עַל עֵקֶב בָּשְׁתָּם / וַאֲשֶׁר קִוּוּ לִמְצֹא אֶרֶץ שֹׁקֶטֶת וּשְׁלֵוָה / וּמָצְאוּ כִפְלַיִם לְזַעֲוָה / כִּי אֵין הַחַי בְּעַצְמוֹ שֹׁלֵט / מִתַּחְבֻּלּוֹת וְסִבּוֹת לְהִמָּלֵט / וְהַמַּכִּיר מְקוֹם תַּחֲנוֹתוֹ / לֹא יַחֲלִיפֶנּו וְלֹא יָמִיר אֹתוֹ / וְקוֹל פְּחָדִים יִהְיֶה בְאָזְנֵיכֶם / כָּל מָקוֹם אֲשֶׁר תִּדְרֹךְ כַּף רַגְלְכֶם / וְלֹא שָׁמְעוּ לַאֲשֶׁר אָמַר לָהֶם / וַיַּעַזְבוּ מְקוֹמָם וְאָהֳלֵיהֶם / וַיֵּלְכוּ עַד בֹּא הַשֶּׁמֶשׁ וְאֵין אוֹר / וַיַּחֲנוּ עַל שְׂפַת הַיְאוֹר / כֻּלָּם מִקְּטַנָּם וְעַד גְּדוֹלָם / וּבְחָשְׁבָם לִמְצֹא מָנוֹחַ לְכַף רַגְלָם / וְהִנֵּה מִן הַיְאֹר צְפַרְדְְּעִים בְּקוֹלָן הוֹמוֹת / וַיְהִי לְכָל הָאַרְנָבִים פַּחַד וְאֵימוֹת / זֶה בְּקוֹלוֹ מַר צֹרֵחַ / וְזֶה מִמְּקוֹמוֹ בֹרֵחַ / זֶה הֹלֵךְ דֶּרֶךְ מְעֻקָּל / וְאָבַד מָנוֹס מִקָּל / וּכְמוֹ הַשַּׁחַר עָלָה / וְהִנֵּה עַל הַמְסִלָּה / הֹלְכֵי דֶרֶךְ עֹבְרִים וְשָׁבִים / רֹכְבִים סוּסִים וְלִפְנֵיהֶם כְּלָבִים / אֲשֶׁר גָּדְלָה מְאֹד צַעֲקָתָם / כַּאֲשֶׁר הֱרִיחוּם בִּמְסִלּוֹתָם / עִקְּבוֹת הָאַרְנָבִים הַנְּטוּשִׁים / עַל פְּנֵי הַשָּׂדֶה, וְהַפָּרָשִׁים / הֵסִיתוּ הַכְּלָבִים לִשְׁלֹחַ בָּהֶם מְהוּמָה / וַיַּכּוּם וַיַּכְּתוּם עַד הַחָרְמָה / וְהַנִּשְׁאָרִים אָמְרוּ: נָשׁוּבָה אֶל עַמֵּנוּ / וְאֶל אֶרֶץ מוֹלַדְתֵּנוּ / וְהָהּ לַיּוֹם אֲשֶׁר נַפְשֵׁנוּ קָצָה / לִשְׁמֹעַ מִן הֶחָכָם עֵצָה / וּלְבָבֵנוּ אֶל הוֹלֵלוּת נִפְתָּה / כִּי טוֹב לָנוּ אָז מֵעַתָּה / וַיָּשׁוּבוּ יֶתֶר הַפְּלֵטָה בְּנֶפֶשׁ מָרָה וְנִכְאֶבֶת / עַד בּוֹאָם אֶל אֶרֶץ נוֹשֶׁבֶת.


וְהַמָּשָׁל בְּאִישׁ אֲשֶׁר הַזְּמָן עָלָיו שָׁלָט / וַיֹּאמַר: אַרְחִיק נְדֹד אָחִישָׁה לִי מִפְלָט / וַיַּעֲזֹב אֶת מְקוֹמוֹ וְאֶרֶץ מוֹלַדְתּוֹ / אֶפֶס כִּי לֹא תִהְיֶה תִפְאַרְתּוֹ / בַּדֶּרֶךְ אֲשֶׁר בָּהּ יֵלֵךְ / וְאִם יֵאָמֶר לוֹ: יְהָבְךָ הַשְׁלֵךְ / עַל בּוֹרַאֲךָ וּסְבֹל סִבָּתוֹ / כִּי מִי יוּכַל לְתַקֵּן אֶת אֲשֶׁר עִוְּתוֹ / חַכֵּה עַד יַעֲבֹר זַעַם מֵאֹהָלֶיךָ / אִם רוּחַ הַמּוֹשֵׁל תַּעֲלֶה עָלֶיךָ / לֹא יִשְׁמַע וְלֹא יֹאבֶה / חָלְפוּ עִם אֳנִיּוֹת אֵבֶה / רַבִּים אֲשֶׁר לֹא עֵצָה שָׁמְעוּ / וַיִּסְעוּ מֵאָהֳלֵיהֶם אֲשֶׁר תָּקָעוּ / וְטוֹב מִפָּז וּמִיָּשְׁפֵה מְשֻׁבֶּצֶת בְּטַבָּעַת / מוֹכִיחַ חָכָם עַל אֹזֶן שׁוֹמַעַת.
