משל צב: שׁוֹר נַגָּח וּבְעָלָיו

אִישׁ פְּתַלְתֹּל וְעִקֵּשׁ / אִוַּלְתּוֹ תְהֵא לוֹ לְמוֹקֵשׁ

שׁוֹר בִּבְעָלָיו יֶאְשַׁם אָשֹׁם / נַגָּח הוּא מִתְּמֹל שִׁלְשֹׁם / בְּכָל עֵת אֲשֶׁר לַחֲרֹשׁ נִסָּהוּ / נוֹדַע כִּי שׁוֹר נַגָּח הוּא / וַתַּעַל חֲמַת הַבַּעַל וַחֲרוֹנָיו / וַיִּכְרֹת בַּמַּשּׂוֹר אֶת קַרְנָיו / וַיֹּאמֶר לִשְׁכֵנָיו כַּאֲשֶׁר עֲסָקָיו רָעִים / וְרָמֵי הַקּוֹמָה גְּדֻעִים / וַיְהִי בַבֹּקֶר בְּחָרְשׁוֹ עַל גַּבּוֹ / וַיְשַׁלַּח הַשּׁוֹר אֶת שִׁנָּיו בּוֹ / וַיֹּאבֶה אֲדֹנָיו כָּל גְּאוֹנוֹ לְהַשְׁפִּיל / וְרֹב שִׁנָּיו מִפִּיו הִפִּיל / וַיְהִי כַאֲשֶׁר מִפִּיו נִשְׁמָטוּ / וּבָטְלוּ הַטֹּחֲנוֹת כִּי מִעֵטוּ / אַחַר כָּל אָדָם יִרְדֹּף / בְּצַד וּבְכָתֵף יֶהְדֹּף / וְלֹא נִכְנַע הַשּׁוֹר בַּעֲמָלוֹ / וּבַיּוֹם הַשְּׁלִישִׁי בְרַגְלוֹ / הִכָּה אֲדֹנָיו מַכָּה רַבָּה / וַיֹּאמְרוּ לוֹ שְׁכֵנָיו: זֹאת הַסִּבָּה / וּזְדוֹן לִבּוֹ וּמַעֲלָלָיו / לֹא תוּכַל לָסִיר מֵעָלָיו / לִכְרֹת מִמֶּנּוּ אֵבֶר אֵבֶר / אִם לֹא מִכֹּל וָכֹל יִשָּׁבֵר.


וְהַמָּשָׁל פֶּן תִּתְגָּרֶה בְּאִישׁ עִקֵּשׁ / כִּי אִם תִּהְיֶה לוֹ לְמוֹקֵשׁ / וְאִם בְּתוֹךְ הָרִיפוֹת תִּכְתְּשֶׁנּוּ / לֹא תָסוּר עִקְּשׁוּתוֹ מִמֶּנּוּ / אַךְ תַּרְחִיקֶנּוּ מִכָּל קָצֶה / פִּגּוּל הוּא לֹא יֵרָצֶה / כִּי לִבּוֹ שֹׁרֶשׁ רֹאשׁ וְלַעֲנָה פוֹרִיָּה / סוּרֵי הַגֶּפֶן נָכְרִיָּה.
