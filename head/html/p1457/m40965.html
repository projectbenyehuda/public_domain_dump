א

[שִׁיר…] אֱמוּנָה בְּלַחְשִׁי

יִשָּׁקֵנִי / בְּרֶכֶב רִבּוֹתַיִם כְּבָא לְתָלְשִׁי

לְרֵיחַ / גַּן נָעוּל […]י זָהָב פִּלְּשִׁי

מָשְׁכֵנִי / דֶּלֶת מַעְלָן וּמַטָּן בְּחִדּוּשִׁי


שְׁחוֹרָה / הִטְעַנִי הַבְדָּלַת קְדוֹשִׁי

אַל תִּרְאוּנִי / וְעַל מַיִם הִפְרִיחַ שָׁרְשִׁי

הַגִּידָה לִי / זֹאת גְּנָזֶיהָ לְשַׂמֵּחַ נַפְשִׁי

אִם לֹא תֵדְעִי / חֵירוּת עוֹלָם לְשַׁלֵּחַ חָפְשִׁי


לְסוּסָתִי / טְעָנִי כְּבוֹדִי וּמֵרִים רֹאשִׁי

נָאווּ / יְדִידִים זֹהַר מִמַּיִם לְהַמְשִׁי

תּוֹרֵי / כְּתָב שָׁלִישִׁים בְּמוֹעֵצוֹת פֵּירְשִׁי

עַד שֶׁהַמֶּלֶךְ / לִמֵּד דִּבְּרוֹתָיו בַּחֹדֶשׁ הַשְּׁלִישִׁי

ב

[צְרוֹר הַמֹּר] / מַלְכִּי בַקֹּדֶשׁ מְיַחֲדִים

אֶשְׁכּוֹל / נְדִיבֵי עַמִּים וְכָל יְדִידִים

הִנָּךְ … ס….. ….. ] בְּאֶבְרוֹתָיו מְאֻפָּדִים

הִנְּךָ יָפֶה / עִטּוּר בְּקוֹלוֹת וְלַפִּידִים


קוֹרוֹת / פֹּרְקוּ הָרִים מִקּוֹל פְּחָדִים

אֲנִי חֲבַצֶּלֶת / צַגְתִּי בְּיִרְאָה וּרְעָדִים

כְּשׁוֹשַׁנָּה / קִוִּיתִי לִמְתוּקִים וְנֶחֱמָדִים

כְּתַפּוּחַ / רִוִּיתִי ממ[…] בָּאֲשִׁישׁוֹת מְאֻפָּדִים
