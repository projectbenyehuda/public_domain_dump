שירה יתומה

הֲלִצְבִי חֵן גְּבוּרַת אוֹן וְעָצְמָה / לְהִתְעַטֵּף רְדִיד אֹפֶל כְּשַׂלְמָה

וְלִרְעוֹת כּֽוֹכְבֵי נֶשֶׁף וְלִתְעוֹת / בְּעִי מִדְבָּר מְעוֹן פַּחַד וְאֵימָה

וְלָצֵאת מֵחֲדָרִים אֶל חֲרָדָה / וּמֵהֶמְיַת נְבָלִים אֶל מְהוּמָה? –

עֲדֵי נִלְכַּד בְּחַבְלֵי הַחֲלוֹמוֹת / וְנִצְטַיֵּד בְּפַחֵי הַתְּנוּמָה,

וְאָרִיתִי – וְהוּא נִרְדָּם – בְּרָצוֹן / אֲשֶׁר יִמְנַע – וְהוּא מֵקִיץ – בְּחֵמָה,

וְהִשְׁקַנִי בְיַד שֵׁנָה עֲרֵבָה / עֲסִיס פִּיהוּ בְּכוֹסֵי אַחֲלָמָה.

וְשָׁכַבְתִּי – וּבֵין שָׁדַי קְוֻצּוֹת / מְרִיקוֹת מֹר עֲלֵי רַקָּה אֲדֻמָּה

וְהַיָּמִין מְחַבֶּקֶת לְבָנָה / וְהַשָּׂפָה מְנַשֶּׁקֶת לְחַמָּה

וְהַמִּטָּה מְקֻטֶּרֶת לְבוֹנָה / וְהָעֶרֶשׂ בְּכָל בֹּשֶׂם פְּטוּמָה.

וְנָעַמְתִּי בְחֶזְיוֹנִי, עֲדֵי כִי / הֱקִיצוֹתִי – וְהִנֵּה אֵין מְאוּמָה

אֲבָל רֵיחַ יְשׁוֹבֵב הַנְּפָשׁוֹת / וּמֹר עוֹבֵר יְחַיֶּה הַנְּשָׁמָה –

כְּשֵׁם נָגִיד וְיָחִיד רַב שְׁמוּאֵל / אֲשֶׁר מִלֵּא פְנֵי כָל הָאֲדָמָה

אֲשֶׁר גָּדַל בְּיִשְׂרָאֵל כְּמִגְדָּל / וְהֻקַּם עָל בְּתוֹךְ עַמּוֹ לְחוֹמָה,

וּבוֹ חֻדַּשׁ לְשִׁבְטוֹ הַֽמְקֻדָּשׁ / מְכוֹן כָּבוֹד עֲלֵי רֹאשׁ עָשׁ וְכִימָה.

וְאָמְנָם כִּי שְׁמוּאֵל הוּא שְׁמוּאֵל / אֲשֶׁר נִקְרָא בְהֵיכַל אֵל פְּנִימָה!

וּמִי יֵדַע – הֲנַעְלָה עוֹד בְּאוֹבוֹת / וְאִם עָמַד לְגוֹרָלוֹ בְעָצְמָה?

וְאִם אֵינוֹ – הֲלֹא הִנּוֹ בְצִדְקוֹ / וְתֻמָּתוֹ וְדַרְכּוֹ הַשְּׁלֵמָה!

שְׂפָתָיו יִשְׁמְרוּ דַעַת, וּמִפִּיו / תְבֻקַּשׁ דַּת אֱלֹהִים הַתְּמִימָה,

וּמַהְלָלוֹ כְיָרֵחַ בְּהִלּוֹ, / וּמַעְלָלוֹ כְמוֹ פֶלֶג בְּצָמָא,

וְהַמַּעְלָה עֲלֵי כִתְפוֹ צְרוּרָה / וְהַמִּשְׂרָה עֲלֵי שִׁכְמוֹ חֲתוּמָה.

וְתָר חָכְמָה – עֲדֵי עָמַד בְּסוֹדָהּ, / וְהַבִּינָה – עֲדֵי יָדַע מְקוֹמָהּ.

אֲחִי עֵצָה – כְּמוֹ שַׁחַר מְאִירָה / בְעֵת צָרָה לְצַלְמָוֶת מְדֻמָּה,

וְכָל מֶלֶךְ לְאוֹר פָּנָיו יְהַלֵּךְ, / וְכָל פָּנִים – תְּכַסֵּמוֹ כְלִמָּה,

וְכָל שָׂרִים לְפָנָיו כַּשְּׁוָרִים / וְכָל יוֹעֵץ כְּרָחֵל נֶאֱלָמָה.

אֲדוֹן נַפְשִׁי, אֲשֶׁר נַפְשִׁי שְׁכוּנָה / בְאַהְבָתוֹ, וְאֶל זִכְרוֹ – שְׁמָמָה,

וְאַהְבָתוֹ בְתוֹךְ לִבִּי כְתוּבָה / וּמִנֹּעַר עֲלֵי לוּחוֹ רְשׁוּמָה,

לְךָ חָכְמָה, לְךָ רוּחַ יְתֵרָה, / לְךָ דֵעָה, לְךָ בִינָה וְעָרְמָה,

וְרֹחַב יָד כְּמוֹ יוֹרֶה וּמַלְקוֹשׁ, / וְרֹחַב לֵב כְּבֵין מִזְרָח לְיָמָּה.

לְךָ הָעֵט אֲשֶׁר נִכְבָּד וְנֶחְמָד – / וְהוּא נִבְזֶה חֲדַל תֹּאַר וְקוֹמָה,

פְּרִי קָנֶה – וְקוֹנֶה הוֹד וּמִשְׂרָה, / אֲחִי רִמָּה – וּמָלֵא תֹךְ וּמִרְמָה,

נְשׂוּא פָנִים וְאִישׁ חַיִל בְּרָכְבוֹ, / וְלֹא יִשְׁוֶה – וְהוּא רַגְלִי – מְאוּמָה.

שְׁנֵי שִׁנָּיו – שְׁנֵי חִצִּים שְׁנוּנִים, / וְרֻקּוֹ – אִם לְחֶסֶד אוֹ נְקָמָה,

וְסַפִּירִים יְפַזֵּר אֶל סְפָרִים / מְשֻׁבָּצִים בְּכָל מֶשִׁי וְרִקְמָה.

וְאַהְבָתִי לְבֶן פֹּרָת יְהוֹסֵף / עֲלֵי כָל אַהֲבָה גָדְלָה וְעָצְמָה,

לְגוּר אַרְיֵה וְיֶלֶד שַׁעֲשׁוּעִים / וְצָפְנַת פַּעֲנֵחַ כָּל סְתוּמָה,

וְצִיץ פָּרַח בְּמַטֵּה הַקְּהָתִי / עֲלֵי מַעְיַן פְּלִילִיָּה וְחָכְמָה

צְעִיר שָׁנִים מְשַׁעְשֵׁעַ בְּמִשְׁנָה / וְעֶלֶם רַךְ מְבָאֵר תַּעֲלוּמָה:

כְּאִלּוּ כָל בְּנֵי הַדּוֹר תְּבוּאוֹת / וְהוּא מֵהֶם לְאֵל שַׁדַּי תְרוּמָה!

פְּדִיתִיהוּ בְנַפְשִׁי, הַקְּשׁוּרָה / בְנַפְשׁוֹ הַתְּמִימָה הַחֲכָמָה.

שְׁמָעֵנִי, נְגִיד עַמִּי, וְהַטֵּה / לְמִלָּתִי שְׁתֵּי אָזְנֵי מְזִמָּה!

קְחָה שִׁירָה מְיֻשֶּׁרֶת יְשָׁרָה, / תְּשׁוּרַת אַהֲבָה זַכָּה קְדוּמָה,

מְעֻלֶּפֶת בְּכֵלֶיהָ כְּכַלָּה, / מְעֻטֶּפֶת בְּעֶדְיֶהָ כְּעַלְמָה,

אֲרוּסָה אַתְּ – וְהִיא לָעַד בְּתוּלָה, / וְאִם לָהּ אָב – וְהִנֶּהָ יְתוֹמָה.

וּפִקְדוֹנִי בְיָדֶךָ, גְּאוֹנִי, / שְׁנֵי אַחִים פְּלֵיטַת הַנְּשַׁמָּה,

אֲשֶׁר הֵמָּה בְנַחְלָתָם כְּזָרִים / וְאַדְמָתָם בְּעֵינֵיהֶם כְּאַדְמָה,

וְלוּלֵא צִלְּךָ נָטוּי עֲלֵיהֶם – / אֲזַי נָדוּ אֱלֵי קַצְוֵי אֲדָמָה.

וְשָׂא שָׁלוֹם בְּמַהְלָלָךְ וְגָדְלָךְ / לְרָאשֵׁי הַֽיְשִׁיבָה הַנְּעִימָה,

אֲצִילַי הָֽרְחוֹקִים, הַקְּרוֹבִים / לְנַפְשִׁי מִתְּשׁוּקָתָם עֲגוּמָה.

בְּחַיֶּיךָ וְחַיֵּיהֶם יְמַהֵר / אֱלֹהֵינוּ לְנוֹפֶלֶת תְּקוּמָה!
