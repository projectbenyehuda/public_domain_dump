[שִׂפְתֵי רְנָנוֹת לִבְנֵי פַּאדוֹבָה]

Isac Raffael Finzi

[נולד בשנת 1728 בפֵֿרָרָה ומת בפָּדוֹבָה בשנת 1813]


שִׂפְתֵי רְנָנוֹת לִבְנֵי פַּאדוֹבָה / רָנֵּי פַלֵּט יְסוֹבֲבֵם סֶלָּה

הֵמָּה רָאוּ מַעֲשֵׂה יְיָ / וַיְסַפְּרוּ כְבוֹדוֹ בִּרְחוֹבוֹת קִרְיָה

רְעָדָה אֲחָזָתָם בְּאִישׁוֹן לָיְלָה / עוֹדָם בְּעַרְשׂוֹתָם יְשֵׁנִים

אֵין מִסְפָּר לְקוֹלוֹת הֲמוֹנָם / שַׁוְעָתָם הַשָּׁמַיִם עָלָתָה

וְהָיָה זְוָעָה הָבִין שְׁמוּעָה / כִּי צָעֲקוּ וְאֵין שׁוֹמֵעַ

וְקוֹל הָעָם בְּרֵעֹה / לֹא אֹזֶן מִלִּין תִּבְחָן

יַפְטִירוּ בְשָׂפָה הוֹי הוֹי / וְאֵין מֵבִין מַה־לָּעָם כִּי יִבְכּוּ

עַל־כֵּן רָחַק יְשׁוּעָה / וְלֹא יַשִּׂיגֶנּוּ רֶוַח וְהֵצָּלָה

עַד נֵעוֹרוּ מִשְּׁנָתָם / וְעַיִן רָאֲתָה וַתְּעִידֵהוּ

אָז נִבְהֲלוּ מֵרְאוֹת / וַאֲשֶׁר לֹא שָׁמְעוּ הִתְבּוֹנָנוּ

כִּי בָעֲרָה אֵשׁ בְּבֵית יְהוּדָה / לֶהָבָה תְלַהֵט קֹרוֹת בֵּיתוֹּ

רָאוּ נְעָרִים נֶחְבָּאוּ / מָנוֹס בִּקְשׁוּ לְנַפְשָׁם

שָׁכֵן קָרוֹב וְאָח רָחוֹק / פַּלָצוּת אֲחָזָם וְלִבָּם סְחַרְחַר

וּקְרוֹבִים מֵרָחוֹק עָמָדוּ / פֶּן־יִמּוֹטוּ עֲלֵיהֶם גֶּחָלִים

וַיָּקוּמוּ וַיָּנוּסוּ בַנֶּשֶׁף / עֲרוּמִים הָלְכוּ בְּלִי לְבוּשׁ

חֹק וּזְמָן לֹא נִתַּן לָהֶם / לְהַצִּיל כָּל־הוֹן יָקָר וְנָעִים

וַיַּעַזְבוּ אֶת־אָהֳלֵיהֶם / וְתוֹחֶלֶת הוֹנָם אָבָדָה

וְהֶחָצֵר סֹגֶרֶת וּמְסֻגֶּרֶת / וְאֵין פּוֹתֵחַ בְּרִיחַ וּדְלָתָיִם

וְרוּחַ לָבְשָׁה אֶחָד וּשְׁנַיִם מֵעַמֵּנוּ / וְלִפְנֵי שַׂר הָעִיר מְהֵרָה רָצוּ

וַיָּקֶם הַמּוֹשֵׁל לַיְלָה / וְלַעֲבָדִים צִוָּה וַיִּגְזַר אֹמֶר

לֵאמֹר צֵאוּ מַהֵר וּרְאוּ / וּבָתֵּי בְנֵי־יִשְׂרָאֵל הַצִּילוּ

וַיַּעֲבֹר מַלְכָּם לִפְנֵיהֶם / וְהֶחָלִיל מַכֶּה בְחוּצוֹתָיִם

גַּם מֵהַמִּגְדָּלִים קוֹלוֹת נִשְׁמָעוּ / בַּרְזֶל בְּבַרְזֶל יָחַד

וַיִּתְקְעוּ וַיָּרִיעוּ לֵאמֹר / הֵאָסְפוּ וּלְכוּ לִישׁוּעָתָה

וַיִּקָּהֲלוּ כְּאִישׁ אֶחָד וַיָּבֹאוּ / אוּמָנִים וּמַשְׁלִיכֵי בַּיְאֹר כֶּלִי

וַיִּפָּתְחוּ כָּל־בְּאֵרוֹת הַמָּיִם / וּבְכָל־פִּנּוֹת נֵרוֹת הִדְלִיקוּ

וַיִּשְׁאֲבוּ מַיִם בְּחִפָּזוֹן / וְלִמְקוֹם הַבְּעֵרָה הֵבִיאוּ

וְרוּחַ נָסַע מֵאֵת יְיָ / וְשַׁלְהֶבֶת אָחוֹר נָסוֹגָה

בְּכַדֵּי־מַיִם יָדוֹ פָּרַשׂ שָׂר / וּנְדִיבֵי עַמִּים אַחֲרָיו הֶחֱזִיקוּ

בְּבֵית הַתְּפִלָּה מוֹרִים הֶעְתִּירוּ / כִּי לַיְיָ הַיְשׁוּעָה וְנַעְתּוֹר לָמוֹ

תַּחַת הַהֲרִיסוּת תִּשְׁקַע הָאֵשׁ / וְלֹא יָסְפָה שׁוּב אֵלָיו עוֹד

מָגוֹר מִסָּבִיב לְבִטְחָה נֶהְפַּךְ / וּבַעֲלוֹת הַחַרְסָה חֶסֶד אֵל יְבֹאֵנוּ

תּוֹשָׁב וְגֵר אוּלָמוֹ שָׁב / וְכֻלָּם יִבְכּוּ אֶת־הַשְּׂרֵפָה

וַיָּעַר יְיָ אֶת־רוּחַ הַנָגִיד / וְהֶעֱמִיד מֵאֲנָשָׁיו לְשָׁמְרֵנוּ

וְהוּא כְּחָפֵץ חֶסֶד / הוֹלֵךְ וּבָא לְהַזְהִירָם

בָּרְכוּ עַמִּים אֱלֹהֵינוּ / אֲשֶׁר נָתַן כָּזֹאת בְּלֵב הַשָּׂר

כִּי לוּלֵא הִתְמַהְמַהּ / כִּמְעַט נָטָיוּ רַגְלֵינוּ

כִּי כְמִשְׁלשׁ שָׁעוֹת חָזַר / וְנֵעוֹר הָאֵשׁ מֵעֲרֵמוֹת הֶעָפָר

מַשְׂאֵת הֶעָשָׁן הֵחֵלָּה / מִתּוֹךְ הַמִּשְׂרְפוֹת גֻּלְתָה הֹעֲלָתָה

וּלְרָאשֵׁי הָעֵדָה נִתַּן תּוֹדָה / כִּי בְיִרְאַת יְיָ עֲלֵיהֶם

קִדְּשׁוּ צוֹם קָרְאוּ עֲצָרָה / וּמִנְדִיבֵי לֵב נְדָבָה הוֹצִיאוּ

וְכָל־הַקָּהָל עַל־עָמְדָם / מִמַּחֲצִית הַיּוֹם וְעַד־הַלָּיְלָה

כֻּלָּם בְּשִׁירֵי דָוִד / עוֹנִים וְאוֹמְרִים בְּיִרְאָה

עַל־כֵּן בְּיוֹם זֶה בְּכָל־שָׁנָה / יְשֻׁבַּח לַאדֹנָי לַדּוֹר

וּבְיוֹם חַ"ג לְיֶרַח מַתַּן דָּת / לְזֵכֶר עוֹלָם יִהְיֶה בָנוּ

כִּי בִלְתִּי מְקוֹם הַבְּעֵרָה / יְיָ בָתֵּינוּ הִצִּיל

וְהָאֵל הַנִּקְדָשׁ בִּצְדָקָה / לוֹ יְשֻׁלַּם נֵדֶר נָדַרְנוּ

וְאַחֲלֵינוּ יִכּוֹנוּ דְרָכֵינוּ / וְאֶת־הָאֵשׁ הַגְּדוֹלָה הַזֹּאת

לֹא נִרְאֶה עוֹד / וּבְחֶסֶד עֶלְיוֹן נִשְׂמְחָה וְנָגִילָה

יִצְחַק יְרַנֵּן כָּל־אִישׁ / וּבְעַד עֲדָתוֹ יַפְגִּיעַ

רְפָא אֵל נָא אֶת־שְׁבָרֶיהָ / וּגְאֻלָּתָם מְהֵרָה תַּצְמַח

יְבָרֵךְ יִשְׁמֹר יְיָ חֵילָם / וְזַרְעָם נָכוֹן לִפְנֵיהֶם

צֶאֱצָאֵיהֶם יִרְבּוּ בַבָּר / וּבְתוֹרָה יָרִימוּ רֹאשׁ

יְבִיאֵם לִגְבוּל קָדְשׁוֹ / הַר זֶה קָנְתָ"ה יְמִינוֹ.

(1795)
