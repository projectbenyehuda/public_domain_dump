[מעשה יונה הנביא]

וּבְכֵן – וַיְהִי דְּבַר יְיָ אֶל יוֹנָה בֶן אֲמִתַּי.


מִשִּׁמְעֲךָ רָהוּ / אֲנָשִׁים וַיִּתְמָהוּ,

אֲשֶׁר לוֹ הַיָּם וְהוּא עָשָׂהוּ.

אֵל נוֹרָא עֲלִילָה / דֶּרֶךְ לְבַעֲלֵי מְעִילָה

[5] הוֹרָה לִמְצֹא מְחִילָה,

נוֹרְאוֹתָיו גִּלָּה / לְהוֹדִיעַ כָּל פְּעֻלָּה,

כִּי לוֹ לְבַד הַגְּדֻלָּה.

יָזַם לְהָשִׁיב גְּמוּלָה / לְעִיר מִנִּי עַד שְׁתוּלָה,

נִינְוֵה הָעִיר הַגְּדוֹלָה

[10] יוֹם צִוָּה צִיר אֱמוּנַי / "קוּם קְרָא חֶזְיוֹנַי,

כִּי עָלְתָה רָעָתָם לְפָנַי".

צִיר בְּשׁוּבָם לִנְכֹחַ / בָּן כִּי יֶרֶב סְלֹחַ –

וַיָּקָם יוֹנָה לִבְרֹחַ.

חֶפְצוֹ בְבוֹא יָם קִיַּם, / וְנָשְֹאוּ מַיִם דָּכְיָם

[15] וַיְהִי סַעַר גָּדוֹל בַּיָּם.

קָרְאוּ יוֹרְדָיו בֵּינֵיהֶם / וַיָּטִילוּ כְלֵיהֶם

אֶל הַיָּם לְהָקֵל מֵעֲלֵיהֶם.

בְּיַרְכְּתֵי סְפִינָה קָדַם / וְלֹא נִתְמַנָּה בְסוֹדָם

וַיִּשְׁכַּב וַיֵּרָדַם.

[20] רַבָּיו קָרְאוּ כְמֵהִים / "קוּם, קְרָא לְשׁוֹכֵן גְּבֹהִים,

אוּלַי יִתְעַשֵּׁת הָאֱלֹהִים".

יָעֲצוּ עֵצָה נְכוֹנָה / לְהַפִּיל פּוּר בֶּאֱמוּנָה

וַיִּפֹּל הַגּוֹרָל עַל יוֹנָה.

הֵן לְחַוּוֹתָם בְּנִיבוֹ / שְׁאֵלוּהוּ: מַה טִּיבוֹ

[25] “מַה מְּלַאכְתְּךָ וּמֵאַיִן תָּבֹא?”

וַיַּעֲנֵמוֹ קָשָׁה / שְׁלֹשָׁה תַּחַת שְׁלֹשָׁה

“אֲנִי יָרֵא אֲשֶׁר עָשָׂה אֶת הַיָּם וְאֶת הַיַּבָּשָׁה”.

דִּבְּרוּ בְשָׁמְעָם בְּאָזְנֵיהֶם, / מִפְּנֵי אֵל, נוֹדַע לָהֶם,

הוּא בוֹרֵחַ, כִּי הִגִּיד לָהֶם.

[30] הֱשִׁיבוּהוּ בְלֵב סוֹעֵר / "בַּמֶּה יִשְׁתֹּק מִסְתָּעֵר,

כִּי הַיָּם הוֹלֵךְ וְסוֹעֵר?"

"גֹּשׁוּ יַחַד כֻּלְּכֶם / וַהֲטִילוּנִי בִכְלֵיכֶם

אֶל הַיָּם וַיִּשְׁתֹּק מֵעֲלֵיכֶם".

יַחַד הָמוּ וְחָלוּ / וְחָתְרוּ – אוּלַי יוֹעִילוּ

[35] אֶל הַיַּבָּשָׁה וְלֹא יָכֹלוּ.

אָז כְּנִכְבָּד כְּנִבְזֶה / צָעֲקוּ לְאֵל נֶעְלָם, כֹּל חוֹזֶה

“אַל נָא נֹאבְדָה בְנֶפֶשׁ הָאִישׁ הַזֶּה!”

תָּקְפוּ מִפֹּה וּמִפֹּה / טוֹרֵף נַפְשׁוֹ בְּאַפּוֹ

וַיְטִילוּהוּ אֶל הַיָּם וַיַּעֲמֹד הַיָּם מִזַּעְפּוֹ.

[40] אֵל יוֹדֵעַ סְתָרִים / אָמְנָם יָרְאוּ גְבָרִים

וַיִּדְּרוּ נְדָרִים.

אָז בַּמְּצוּלוֹת מִנָּה / נָאוֹר אֵין לוֹ תְמוּנָה

דָּג גָּדוֹל לִבְלֹעַ אֶת יוֹנָה.

וַיָּנַח בַּמְּצוּלוֹת / בִּמְעֵי דָג וַאֲפֵלוֹת

[45] שְׁלֹשָׁה יָמִים וּשְׁלֹשָׁה לֵילוֹת.
