[אֵל אֱמֶת עִמִּי]

אֵל אֱמֶת עִמִּי, / וַאֲנִי מִמִּי / אֶפְחֲדָה לִי?

יַעֲשֶׂה שָׁלוֹם / לִי, וְרֹב שָׁלוֹם / יַעֲשֶׂה לִי.


בּוֹ אֲדַבֵּר מִ- / דֵּי הֱיוֹתִי תַחַת צְלָלָיו,

מִי יְגַלֶּה, מִי, / תַּעֲצוּמוֹ אוֹ מַהֲלָלָיו?

מַעֲדֶה לִשְׁמִי / מֵהֲדָרוֹ וּשְׁמֵי גְלִילָיו.

מֵאֲזָי מַעְלִי / מִתְּהוֹמֵי / מָגוֹר וּמַעְלִי,

מַעֲבִיר נָפְלִי / צוּר, מְרִימִי, / אָחַז בְּשִׁפְלִי.


רַחֲשִׁי מוּלוֹ / אֶשְׁפְּכָה כַּמַּיִם, וְאָמְנָה

אֶזְכְּרָה גָדְלוֹ, / כִּי לְתַעֲצוּמוֹ אֵין תְּכוּנָה.

מַה אֲחַו? הֵן לֹא / קֵץ לְמַהֲלָלוֹ, בֶּאֱמוּנָה

אִם שְׁמוֹ פֶּלִיא – / רַחֲמָיו לִבְלִי / חֹק לְמוּלִי

הוּא לְבַדּ דִּגְלִי, / הוּא לְבַד חֵילִי, / יַעֲזֹר לִי.


הֶאֱמִירַנִי, / כִּי יְצָרַנִי / בַּעֲבוּרוֹ,

אַט נְשָׂאַנִי / עַד הֱבִיאַנִי / אֶל יְקָרוֹ.

אֶשְׂמְחָה אָנִי, / כִּי יְעָטַנִי / סוּת הֲדָרוֹ.

יַאֲזִין קוֹלִי, / כִּי בְעֵת סִכְלִי – / הוּא בְכִסְלִי,

אַךְ בְּיוֹם צַר לִי / שָׂם שְׁמוֹ צִלִּי / מִקְּרָב לִי.


מַה אֲדַבֵּר לוֹ? / אֵדְעָה כִּי לֹא / לִי שְׁבִילִי,

אַךְ כְּפִי יָכְלוֹ / יַעֲזֹב נִכְלוֹ / אִישׁ אֱוִילִי.

אֶהֱיֶה אֶצְלוֹ, / אֶעֱלֶה עָלֹה / בֵּית זְבוּלִי.

הוּא צְרִי חִילִי, / כִּי מְשׂוֹשׂ גִּילִי / הוּא בְאֶבְלִי,

הוּא מְנַת חֶבְלִי, / הוּא בְיוֹם אָפְלִי / נֵר לְרַגְלִי.
