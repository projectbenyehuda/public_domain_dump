אֵיךְ אָזְלַת יָדִי

חטאנו (?לט׳ באב)


אֵיךְ אָזְלַת יָדִי, / יוֹם גָּלָה כְבוֹדִי!

אֲהָה, נִגְלָה סוֹדִי, / וְעוֹד מָה עִמָּדִי?

אַחַר בָּחַרְתָּ, דּוֹדִי, / [……] די

אֲמָרֵר: אוֹי נָא לִי!


לִי אָפֵס מַפְנֶה; / מָה אַעַשׂ – וְהִנֵּה

שְׁבָרַנִי מִשְׁנֶה / שִׁבָּרוֹן שׁוֹנֶה?

אֶצְעַק בְּלֵב נַעֲנֶה, / אֶקְרָא יוֹמָם – וְלֹא תַעֲנֶה,

וְלַיְלָה וְלֹא דוּמִיָּה לִי.


בְּחֶטְאִי יְצָאַנִי / בַּעַל מְצָאַנִי,

וְעַבְדִּי רָאַנִי, / בְּכָל צַד דִּכְּאַנִי,

וְגַם הוּא הֶלְאַנִי, / בָּאֵשׁ וּבַמַּיִם הֱבִיאַנִי,

וְאָמַר: הֲלֹא מְצַחֵק אָנִי.


אֲנִי, הַנִּנְעֶרֶת, / בְּיָדוֹ נִמְסֶרֶת,

וְחִנָּם נִמְכֶּרֶת – / וְנַפְשִׁי זוֹכֶרֶת

בְּעַיִן נִגֶּרֶת: אָז הָיִיתִי גְבֶרֶת,

וּבְנֵי בַיִת הָיָה לִי.


רְאוּ, עוֹבְרֵי נְתִיבִי, / הֲיֵשׁ כְּאֵב כִּכְאֵבִי?

הֲדָפָנִי אָבִי, / טְרָפָנִי לָבִיא,

וְשָׁכַן מוֹשָׁבִי / וְאֵיךְ יַעֲמֹד לְבָבִי –

וַייָ עָנָה בִי?


בִּי קָצְרָה רוּחִי / וְהַיּוֹם מְרִי שִׂיחִי,

כִּי אָבַד נִצְחִי / וְתַם לָרִיק כֹּחִי,

וְעִם זָר אֵיךְ אֶחִי / וְרֹב חֲמָסִי עַל אָחִי,

כִּי מִתְאַנֶה הוּא לִי?


מוֹקֵשׁ הֶעֱמַדְתִּי / וְנַפְשִׁי לָכַדְתִּי,

וְאֵיךְ לֹא אָבַדְתִּי – / וְחַיַּי פֻּקַּדְתִּי,

וּמֵאַרְצִי נִדַּחְתִּי / וּמִמִּקְדָּשׁ נָדַדְתִּי,

לְקִצְבֵי הָרִים יָרַדְתִּי?


יָרַדְתִּי פֶלֶא / אֱלֵי בֵית כֶּלֶא,

וְהוּא רֶפֶשׁ מָלֵא / וְאֵין עָלַי חוֹלֶה;

כִּמְעַט קָט אֶכְלֶה / בְּאֹרֶךְ שְׁבִי, לוּלֵי

יְיָ עֶזְרָתָה לִּי / וּפַחַד יִצְחָק הָיָה לִי.
