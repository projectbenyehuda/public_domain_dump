אֱלֹהֵי הָרוּחוֹת לְכָל בָּשָׂר... רְצוֹנִי נֶגְדֶּךָ

אֱלֹהֵי הָרוּחוֹת / לְכָל בָּשָׂר אַתָּה

וְחָכְמָה בַּטּוּחוֹת / בְּחָכְמָתְךָ שַׁתָּה

וְגַם דַּבֵּר צָחוֹת / בְּיַד לָשׁוֹן תָּתָּה —

אֱמֶת כֹּל חוֹלַלְתָּ / וּבוֹ לֹא עָמַלְתָּ, / רַק בְּפִיךָ מִלָּלְתָּ!


בְּיָדְךָ עִתּוֹתַי / וּמַה זֶּה פִי יַעְנֶה —

וְקָצְרוּ שִׂפְתוֹתָי? / וְכָל רֶגַע הִנֵּה

בְקִרְבִּי אוֹתוֹתַי, / וְאָן לִבִּי יִפְנֶה —

וְאוֹתוֹ הִגְבַּלְתָּ / בְּדָת שֶׁהִנְחַלְתָּ, / לִי בְרָכָה אָצָלְתָּ!


רְצוֹנִי נֶגְדֶּךָ / וְטוֹב לִי בָחַרְתָּ,

וּבִנְפֹל עַבְדֶּךָ / יְמִינוֹ עָזַרְתָּ,

וּבִכְסוּת חַסְדֶּךָ / עֲוֹנִי הִסְתַּרְתָּ,

וְעָלַי חָמַלְתָּ / וְטוֹבוֹת גָּמַלְתָּ / לִי גְּמוּל לֹא שָׁאָלְתָּ!


הֲכִינוֹתָ סֵפֶר / לְכָל הַנּוֹלָדִים,

וּבִכְלֵי עֵט סוֹפֵר / כְּתוּבִים מַעְבָּדִים,

וּמָצָאתָ כֹפֶר / לְרוּחוֹת נִכְבָּדִים,

וְנַפְשִׁי הִצַּלְתָּ / וְחַיַּי גָּאַלְתָּ, / מֵעֲזֹר לֹא חָדָלְתָּ!


מְלֶאכֶת שָׁמֶיךָ / לְאוֹת לָךְ לֹא נִכְחָד,

וְכֵס בִּמְרוֹמֶיךָ / בְּכֻלָּם הִתְאֶחָד,

וְיַעְמֹד בִּשְׁמֶךָ / לְמַעַן הוּא אֶחָד,

וְהַכֹּל פָּעַלְתָּ / וּבַכֹּל מָשַׁלְתָּ / וַעֲלֵי כֹל גָּדָלְתָּ!
