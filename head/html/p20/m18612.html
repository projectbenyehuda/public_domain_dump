אֵל נֶאְדַּר בַּקֹּדֶשׁ

תוכחה (ליום הכיפורים)

אֵל נֶאְדַּר בַּקֹּדֶשׁ / צִוָּה אַנְשֵׁי קֹדֶשׁ

לְהִתְעַנּוֹת יוֹם קֹדֶשׁ / בֶּעָשׂוֹר לַחֹדֶשׁ.

בְּצֵאת כֹּהֵן לְאֵל עֶלְיוֹן / לְאוֹהֲבוֹ – קִדְּמוֹ בְהֶגְיוֹן

בְּרָכָה, אוֹתוֹ בִּשֵּׂר, / וַיִּתֶּן לוֹ מַעֲשֵׂר.

רָאָה תָם חֲלוֹמוֹתָיו / וְנֶדֶר פָּצוּ שְׂפָתָיו:

מַה תִּתֵּן מִשֶּׁלָּךְ – / עַשֵּׂר אֲעַשְּׂרֶנּוּ לָךְ!

הִזְהִיר הָאֵל עַמּוֹ / לְכַבֵּד כָּל מְכַבֵּד שְׁמוֹ:

תֵּן לָהֶם בִּשְׁנָתְךָ / כָּל מַעֲשַׂר תְּבוּאָתְךָ!

מַעֲלַת עֲבוֹדַת גְּבוֹהִים / וּמִשְׂרַת בֵּית אֱלֹהִים

לְמַקְטִיר קְטֹרֶת, יָסֵר / מַעֲשֵׂר מִן הַמַּעֲשֵׂר.

חֻקִּים צַדִּיקִים שָׂם / בְּכָל הַמַּעֲשִׂים, וּמִכְסָם –

לָתֵת לַשֵּׁם בְּרָצוֹן / מַעֲשַׂר בָּקָר וָצֹאן.

זְמַן בֹּקֶר וָעָרֶב / חַי וּתְבוּאָה לְקָרֵב –

כֶּבֶשׂ לְעוֹלָה שְׂרֵפָה / וַעֲשִׂירִית הָאֵיפָה.

קַוּוּ, נְגוּעִים, לְצוּרְכֶם / וְעוֹד יֵאָצֵר לָכֶם:

קְחוּ כִנּוֹרוֹת וְשׁוֹרֲרוּ, / בְּנֵבֶל עָשׂוֹר זַמְּרוּ!
