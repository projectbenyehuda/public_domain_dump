לו נעצרה

לוּ נֶעְצְרָה אוֹ נִבְצְרָה תַּאֲוָתִי / לוּ כָלְתָה אוֹ אָבְדָה תִּקְוָתִי

אוֹחִיל ־ נַפְשִׁי אָמְרָה נוֹאָשׁ, וְעוֹד / לֹא נוֹחֲלָה ־ כִּי נָחֲלָה מַכָּתִי

אֶגְעַר בְּשַׂרְעַפַּי לְהִכָּבֵד וְגַם / עַל מַחְשְׁבוֹתַי אַעֲלֶה מַשְׂאָתִי

הֲאִם אֲצַוֶּה עַל קְרָבִים יֶהֱמוּ: ״הִשָּׁמְרוּ!״ ־ לֹא שָׁמְרוּ מִצְוָתִי.

אֹמַר לְעֵינַי: שִׁחֲדוּ מִכֹּחֲכֶם, / אוּלַי תְּכַבֶּה הַיְקוֹד דִּמְעָתִי,

עַל כֵּן אֲנִי כֹּפֶר דְּמָעַי אֶתְּנָה / שְׂפָתִי, וְשִׁירָתִי פְּדוּת אֶנְקָתִי.

אֶבְכֶּה בְּמִסְתָּרִים עֲלֵי עֵינִי וְלֹא / רֵעַ מְעוֹדֵד לִי לְבַד אַנְחָתִי.

אוּלַי תְּלָאוֹת יִדְרְשׁוּנִי יוֹם לְיוֹם / אֶשָּׂא אֲבִירֵיהֶם עֲלֵי [אֶבְרָ]תִי

לוּ הַשְּׂמָלוֹת כַּנְּפָשׁוֹת נָשְׂאוּ ־ / מִשֵּׂאת תְּלָאוֹת נִקְרְעָה שִׂמְלָתִי

כִּמְעַט שְׁנוֹתַי בַּעֲצָמַי יִתְּמוּ / וּכְבֶן עֲזַרְיָה מִהֲרָה שֵׂיבָתִי

יִבְכֶּה לְבָבִי אַחֲרֵי נַפְשִׁי עֲדֵי / אֶחְשֹׁב נְעוּרַי אַחֲרֵי זִקְנָתִי

מַה לַּעֲשׂוֹת לוֹ כִּי יְעַדְתִּיו לִהְיוֹת / חַרְבִּי וּמָגִנִּי וְגַם צִנָּתִי

אִם אֹמְרָה קַשְׁתִּי ־ [וְהוּא] רִשְׁתִּי, וְהוּא / פִשְׁעִי, וְהוּא פִשְׁעִי וְחַבּוּרָתִי

פִּשְׁעִי וְחַטָּאתִי ־ תְבוּנָתִי, וּמִי / יִתֵּן זְמַנִּי יַעֲבִיר חַטָּאתִי

יִרְדֹּף וְגַם יֶהְדֹּף [בְּאַפּוֹ] הַזְּמַן / נַפְשִׁי וְיִרְמֹס בָּחֳרִי חַיָּתִי


דַּי כִּי [הֲרִימוֹ]תִי מְאֹד נִסִּי עֲדֵי / מִכָּל גְּבָעוֹת נִשְּׂאָה גִבְעִָתִי

הַאִם תְּמַלֵּט לִי שְׁאָר תַּשְׂכִּיל וְאִם / תַּשְׂכִּיל עֲשֹׂה אֶתֵּן לְךָ חֶלְקָתִי

אִם אֵין לְאֵל יָדִי לְהִשָּׂגֵב וְלוּ / חָכְמַת שְׁלֹמֹה תִּהְיֶה חָכְמָתִי

אוּלָם אֲנִי עַל בָּמָתֵי עָב אֶעֱלֶה / לוּ אָסְפוּ בַּדֵּי שְׁאוֹל כִּנְעָתִי

ב ה לה ל אֶעֱזֹב / עַד אֶפְּלָה אֶרֶץ מְלֹא קוֹמָתִי

לוּ שָׁפְטָה אֶרֶץ בְּצֶדֶק שׁוֹחֲרֵי / צֶדֶק ־ אֲזַי בִּי עָנְתָה צִדְקָתִי

מָה אַאֲרִיךְ שִׂיחִי וְתֵבֵל הָיְתָה / טוֹבָה אֲבָל בָּאֲחֲרוֹנָה בָּאתִי


וּמְאָרְבִים כִּי לָכְדוּ לִי מַעְבְּרוֹת / חָכְמָה, עֲנִיתִים: הַאֲ[נִי] אֶפְרָתִי?

הִרְחִיב פְּתַיוּת אָהֳלוֹ בָם וַאֲנִי / אַרְחִיב כְּמוֹ נֶשֶׁר לְזֹאת קָרְחָתִי

לוּ אֵשׁ כְּאֵשׁ מִתְאוֹנְנִים בָּם נָפְלָה / לֹא אֻכְּלוּ עַד אֻכְּלוּ קִנְאָתִי

תִּכְהֶה מְהֵרָה עֵין יְמִינָם בַּעֲבוּר / לֹא רָאֲתָה מִנִּי לְבַד צוּרָתִי

אַל נָא תְרִיבוּנִי, לְמַעַן קִלְלַת / אֵלֶּה תְכַפֵּר [לִי] בְּעַד אָשְׁמָתִי.
