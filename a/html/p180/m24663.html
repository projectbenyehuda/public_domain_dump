שְׁכוּלָה אֲכוּלָה

שְׁכוּלָה אֲכוּלָה / לָמֶה תִבְכִּי

הֲנוֹאַשׁ לִבֵּךְ / מֵאֲשֶׁר תְּחַכִּי

קִצִּי נִמְשַׁךְ / וְאָרַךְ חָשְׁכִּי

הוֹחִילִי עֲנִיָּה / עוֹד מְעַט כִּי

אֶשְׁלַח מַלְאָכִי / לְפַנּוֹת דַּרְכִּי

וְעַל הַר צִיּוֹן / אֶסֹּךְ מַלְכִּי

בּוֹא יָבוֹא מְבַשֵּׂר / לְפַנּוֹת שְׁבִילָךְ

אִמְרוּ לְצִיּוֹן / יְיָ מָלָךְ / הִנֵּה מַלְכֵּךְ עוֹד יָבוֹא לַָךְ.


לְאֵלִי אֵלִי / כַּמָּה אֲיַחֵל

וְעַד אָן תִּמְשֹׁךְ / גָּלוּּת הַחֵל

וּבְנֵי רָחֵל / גְּזוּזִים כְּרָחֵל

וַאֲנִי בְכָל זֹאת / תָּמִיד אֲיַחֵל

הוֹחִילִי עֲנִיָּה / לְפוֹדֶה וּמוֹחֵל

כִּי לֹא לְעוֹלָם / בָּךְ אֲנִי בוֹחֵל / עוֹד מְעַט וְהָיִית לִי וַאֲנִי לָךְ.


מָתַי יִקְרַב / וְיִגַּע הַתּוֹר

סָתוּם וְחָתוּם / מָתַי תִּפְתֹּר

אַרְמוֹן זָרִים / מָתַי תִּסְתֹּר

הוֹחִילִי עֲנִיָּה / לְמַחְסֶה וּמִסְתּוֹר

כִּי עוֹד יֶשׁ לָךְ / מַרְפֵּא וּמַעְתּוֹר

וְיִדֹּם כַּפְתּוֹר / כְּיוֹם אִי כַפְתּוֹר / וְיָצִיץ בְּמִצְחֵךְ צִיץ אֲשֶׁר נִשְׁלָךְ.


הֲמוֹנִי לְפָנִים / בְּאַחַת נִטְבַּע

מִנֹּף וּבָבֶל / בְּקָרוֹב נִתְבַּע

הַיּוֹם אֲנִי עוֹד / כְּעַיִט נִצְבַּע

נִקְבְּצוּ עָלַי / מַלְכֻיּוֹת אַרְבַּע

וְאָכַל בְּשָׂרִי / וְעוֹד לֹא נִשְׂבָּע

הוֹחִילִי עֲנִיָּה / לְצוּר אֲשֶׁר נִשְׁבַּע / כִּי בוֹא יָבוֹא דוֹד אֲשֶׁר הָלַָךְ.
