[גפן שתולה]

וכתב לאחד מרעיו שחזר מן הנסיעה וזה מדבריו המצוינים


גֶּפֶן שְׁתוּלָה עַל מַיִם וּפוֹרִיָּה / הַאַתְּ צְבִיָּה אוֹ עֶגְלָה יְפֵיפִיָּה?

בִּרְאוֹת אֱנוֹשׁ חַי מֵת תֶּאְסֹף שְׂחוֹק וּבְכִי / וּכְטַל עֲלֵי שׁוֹשָׁן דִּמְעָהּ עֲלֵי לֶחְיָהּ

תַּבִּיט לְמוּל שַׁחַק לַיִל, וְתַרְאֵנִי / שֶׁמֶשׁ וְיָרֵחַ יַחַד וְדוּמִיָּה.

מַה לָּהּ אֲשֶׁר תַּטֶּה לְהַטּוֹת לְבָבִי עוֹד / צַוָּאר כְּמוֹ מִגְדָּל בָּנוּי לְתַלְפִּיָּה

הַמְעַט רְאוֹתֵךְ אִישׁ עָנִי אֲחָזוּהוּ / חֶבְלֵי יְמֵי פֵרוּד, יָמוּת בְּלִי מִחְיָה

סָרוּ מְיֻדָּעָיו, נוֹתַר בְּלִי רֵעַ / מִבֵּין בְּנוֹת יָמִים יֶאְנַק בְּתַאְנִיָּה

אַל יִזְכְּרֵנִי אֵל אִם לֹא זְכַרְתִּיכֶם, / אָרִיד בְּשִׂיחִי עֵת אֶזְכֹּר וְאֶהְמָיָה

לוּ הַַיְּגוֹנִים אֶמְשֹׁל בָּם כְּמָשְׁלָם בִּי / בִּלְבַב אֱנוֹשׁ בּוֹגֵד שַׂמְתִּים תְּרוּמִיָּה.

שׁוֹכַנְתְּ עֲלֵי אָמִיר, מַה לָּךְ בְּשׁוּר דִּמְעַת / אִישׁ חַי כְּמֵת נִהְיָה, תִּנְהִי נְהִי נִהְיָה?

אֶנְהֶה עֲלֵי יוֹנָה אָחִי אֲשֶׁר נָדַד / אֶבְכֶּה כְּאִלּוּ אֵין תִּקְוָה וְצִפִּיָּה

נִכְאַב לְבָבִי עַד שֶׁלֹּא עֲזָבַנִי / כִּי יוֹם עֲזָבַנִי עוֹפֵף כְּמוֹ אַיָּה

יוֹם צֵאת מְתֵי מוּסָר אַחֲרַי לְבַקֶּשְׁךָ / אַךְ בַּאֲשׁוּרָךְ מָצָאנוּ פְלִילִיָּה

לוּ חָֽכְמוּ אַנְשֵׁי לֵב לֹא נְתָנוּךָ / לָסוּר אֲבָל לִבָּם עִמָּךְ כְּמוֹ שִׁבְיָה

אִם הָאֲנָשִׁים יֵחָצוּן שְׁנֵי רָאשִׁים / כִּשְׁתֵּי נְפָשׁוֹת הֵם בָּהֶם - וְאַתְּ חַיָּה

פֶּתִי זְמַנִּי בָךְ נִסְתַּר וְלֹא נִרְאֶה / כְּאִלּוּ כְסוּת [אַתְּ] עַל עֵרֹם וְעַל עֶרְיָה

נָבוֹן, חֲכַם לֵבָב, תָּמִים וְאֵין בּוֹ מוּם / כִּי אִם יְדַבֵּר - לֹא שֶׁקֶר וְלֹא רְמִיָּה

קַח זֹאת - וְאַל תִּקְטַן, אָחִי, בְּעֵינֶיךָ / כִּי דַעְתְּךָ תַחְסִיר דַּעַת וְתוּשִׁיָּה

הִיא כָל תְּהִלָּתָךְ, אוּלָם לְאַט עַל עֵט / שָׁגָה בְּכָתְבוֹ ״כָל״, לֹא כָל וְלֹא חֶצְיָהּ -

שִׁירָה כְּמוֹ חוֹתָם, עָלָיו שִׁמְךָ חָקוּק, / לַיּוֹם וְלָעֵדוּת תִּהְיֶה שְׁלִישִׁיָּה:

שֶׁמֶשׁ וְיָרֵחַ מִבֵּין מְאוֹרֶיהָ / וּבְבֵין פְּנִינֶיהָ רוּחַ פְּנִינִיָּה.
