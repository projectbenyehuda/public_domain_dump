מי נשקני ואני נרדפת

חתימה: משתא אלשבזי מכנא


מִי נִשְּׁקַנִי וַאֲנִי נִרְדֶּפֶת / דַּעְתִּי מְעֻרְבֶּבֶת וּמִטָּרֶפֶת

שׁוֹאֵף מְאוֹר שִׂכְלִי לְהַשִּׂיג מַחֲמַד / לִבִּי וְרַעְיוֹנַי בְּהוֹד חוֹפֶפֶת

תָּהָה זְמַנִּי כִּי מְשַׂנְאַי גָּזְרוּ / לִגְנוֹז סְפָרַי וַאֲנִי נֶחְרֶפֶת

אֵיכָה בְּתֵימָן אַהֲבָתִי נִשְׁכְּחָה / סָרָה עֲטָרָתִי וְהַמִּצְנֶפֶת

אִמְרִי אֲיֻמָּה אֶל ה׳ יִזְכְּרָה / לִבְרִית אֲבוֹתַי כִּי אֲנִי נִכְסֶפֶת

לִקְרֹא בְּסֵפֶר הַבְּרִית הַנֶּחֱקָק / עֵינַי בְּיוֹם מַעְמָד דְּמָע דּוֹלֶפֶת

שׁוּר מִמְּעוֹן קָדְשָׁךְ בְּרָצוֹן יוֹצְרִי / וּרְאֵה לְיַעְלַת חֵן שְׁהִיא מַשְׁקֶפֶת

בִּזְכוּת יְלוּד עַמְרָם רְצֵה עַל עַמְּךָ / עָלַי שְׁכִינָתָךְ תְּהֶא מַקֶּפֶת

זָרִים וְאַכְזָרִים בְּאַפֶּךָ נְתוֹשׁ / נִינֵי יְלוּד אָמָה שְׁהִיא נִטְנֶפֶת

יִתְנוֹדְדָה פֶּרֶא בְּאֵשׁ אַף יֶחֱלֶה / יִתְלַבְּשָׁה גּוּפוֹ חֳלִי שַׁחֶפֶת

מֵרֹב גְּזֵרוֹתָיו צְעָדַי קָצְרוּ / מֵהֵיכְלֵי דָּתִי אֲנִי נִדְחֶפֶת

כַּפֵּר עֲוֹן עַמָּךְ אֲשֶׁר נִתְפַּזְּרוּ / יוֹנָתְךָ חוֹלָה וּמִתְעַפְעֶפֶת

נַחֵם לְדַל נִרְדָּף וְגוֹלֶה בַּזְּמָן / עָזַב גְּדִיל בִּגְדוֹ וְהַטּוֹטֶפֶת

אוֹיְבָיו בְּכָל דֶּרֶךְ לְתָפְשׂוֹ אוֹרְבִים / שִׁלַּם לְהוֹנוֹ מַס וְעוֹד תּוֹסֶפֶת

מַהֵר בְּהַצָּלָה וְרֶוַח מִמְּךָ / רַחֵם לְבַת הוֹמָה וּמִתְעַלֶּפֶת.
