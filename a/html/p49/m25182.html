[היום מחר תמישון]

ולה אלי פעצל מן אכואנה במצר

ולו אל נכבד מרעיו במצרים


U – – – U – – – U //– – U – – – U – – – U – –

הֲיוֹם מָחָר תְּמִישׁוּן אָהֳלֵיכֶם? // תְּנוּ נָא אֶת לְבָבִי בֵּין כְּלֵיכֶם,

וְצַוּוּ עָב, אֲחוֹתְכֶם בַּנְּדִיבוּת, // בְּהִקָּדְרָהּ, בְּלֶכֶת יַעֲלֵיכֶם,

לְהִתְאַפֵּק, וְלֹא תִבְכֶּה לְנוּדָם, // וְלֹא תָרִיק עֲלֵיהֶם וַעֲלֵיכֶם,

אֲבָל עֵת נוּחֲכֶם תַּרְבֶּה בְּכִיתָהּ, // כְּיֶדְכֶם, אוֹ כְּמוֹ פִי שׁוֹאֲלֵיכֶם.

אֲנִי אֶדְאַג לְנוּד עֹפֶר, יְמָאֵן // שְׁמֹעַ לִי, וְהוּא יִשְׁמַע אֲלֵיכֶם,

וְיִתְהַלֵּךְ לְרָחוֹק אַחֲרֵיכֶם, // וְיָלִין בַּעֲרִיסֵי צֶאֱלֵיכֶם,

וְיִתְקַנֵּא לְבָבִי, בַּעֲבוּר מִן // עֲפַר רַגְלוֹ מְלֵאִים שַׁעֲלֵיכֶם,

וְכִי יֵלֵךְ בְּדֶרֶךְ זוּ הֲלַכְתֶּם // וְיִטֶּה מִן שְׁבִילֵי גוֹאֲלֵיכֶם.

עֲלֵי מָה תַחְפְּזוּ לָנוּס? תְּנוּ לִי // בְּמַתָּן נַעֲלוֹ אוֹ נַעֲלֵיכֶם

וְאָרוּץ בּוֹ אֱלֵי נֹף, עִיר מְגוּרֵי // תְּחוֹלֶלְכֶם וְקִרְיַת סִבְלֵיכֶם,

וְאֵלֵךְ עִם רְבִי סַהְלַאן אֲצִילָהּ // אֱלֵי צִיּוֹן, תְּהִלַת חוֹלֲלֵיכֶם,

וְאָשִׂיחָה לְשִׁכְנֵי הַר לְבָנוֹן // וְיַרְדֵּן וַאֲמָנָה נַחֲלֵיכֶם:

תְּנוּ כָבוֹד לְיוֹרֵשׁ מַהֲלָלֵי // אֲבוֹתֵיכֶם וּמוֹרִישׁ הוֹלֲלֵיכֶם,

כְּפִיר דֵּעָה וּמוֹעֵצָה, וְשַׁחַל // תְּבוּנָה, אַחֲרֵי מוֹת שַׁחֲלֵיכֶם,

צֳרִי הַמַּחֲלִיא מַרְפֵּי יְדֵיכֶם // וְצָרֵיכֶם, וּמַרְפֵּא לָחֳלִיכֶם,

וּמַקֵּל לַח בְּיֶדְכֶם הוּא – וְאֵיכָה // דְּאַגְתֶּם מִן יְבשֶׁת מַקְלֵיכֶם?

וְלָמָּה תִשְׁבְּרוּ מַיִם – וְלִבּוֹ // רְוָיַתְכֶם, וּבִלְשׁוֹנוֹ דְּלִיכֶם!

וְאֵיכָכָה חֲלִי כֶתֶם תְּבַקְשׁוּ // לְהִתְיַפּוֹת – וְהוּא מִבְחַר חֲלִיכֶם!

וְאֵיךְ יוּכַל מְכַבֶּה אוּר פְּחָרִים // וְהוּא אֶשְׁכֶם, לְכַבּוֹת גַּחֲלֵיכֶם?

וְאֶבְטַח כִּי עֲלֵי יָדוֹ יְבֻשַּׁל // קְצִיר צָרִים – קְחוּ אֶת מַגְּלֵיכֶם!

שְמָעוּנִי, עֲדַת זַכִּים, קְרוֹבָיו, // אֲנִי נִכְסָף, בְּחַיַּי, לוֹ וְלָכֶם,

לְמַעַן כִּי כְּמוֹ עֵץ קוֹר יְדֵיכֶם // וְיִרְאָה עַל פְּנֵיכֶם מֵי עֲלֵיכֶם,

וְסוּת צֶדֶק אֲשֶׁר מֵאָב נְחַלְתֶּם // לְבֵשׁוּהוּ בְּנֵיכֶם נוֹחֲלֵיכֶם,

וְשֶׁבֶר יִמְצְאוּן בָּאִים אֲלֵיכֶם, // וְלֹא רוּחַ מְלֵאוֹת שִׁבֳּלֵיכֶם,

וְנִלְאוּ הַחֲסִידִים הַנְּבוֹנִים // עֲשׂה חֶסֶד כְּמַעְשֵׂי נוֹאֳלֵיכֶם,

וְיַד רֶשַׁע בְּמַכְתֶּשְׁכֶם דְּכִיתֶם, // וְרֹאשׁ עַוְלָה שְׁבַרְתֶּם בֶּעֱלִיכֶם,

וְעַל זָדוֹן נְעַלְתֶּם בַּבְּרִיחִים // וּבִנְחשֶׁת וּבַרְזֶל מִנְעֲלֵיכֶם,

וְלָכֶם מִפְעֲלֵי ישֶׁר יְדוּעִים // בְּמֶרְחַקִּים וּבֵינוֹת מַקְהֲלֵיכֶם,

כְּזִקְנֵיכֶם כְּנַעְרֵיכֶם, כְּהוֹרֹות // בְּבָתֵּיכֶם צְעִירֵי עוֹלֲלֵיכֶם,

וּמַשְׂכֹּרֶת בְּיַד הָאֵל שְׁלֵמָה // וּמוּכָנָה לְשַׁלֵּם פָּעֳלֵיכֶם!
