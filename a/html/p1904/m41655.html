[בְּיָדְךָ אַפְקִיד רוּחִי בְּשָׂרִי וְדָמִי כְאַחַת (סליחה)]

סְלִיחָה לְשַׁחֲרִית יוֹם הַכִּפּוּרִים

(– – – – – –)

בְּיָדְךָ אַפְקִיד רוּחִי / בְּשָׂרִי וְדָמִי כְאַחַת

כִּי מָה אֲנִי וּמֶה חַיַּי / פַּחַד וָפַח וָפָחַת

וְיוֹם רוּחִי לְךָ יִקְרָא / וְרָגְזָה שְׁאוֹל מִתָּחַת

וַאֲסַפְתּוֹ בִצְרוֹרְךָ / צְרוֹר חַיִּים וְנָחַת

וִיחִי-עוֹד לָנֶצַח / לֹא יִרְאֶה הַשָּׁחַת.


נִשְׁמַת-רוּחַ חַיִּים / יוֹם נָפַחְתָּ וְהָיָה

מִצּוּר חַיִּים חֲצַבְתָּהּ / וּמַקֶּבֶת בּוֹר נְקִיָּה

וַתַּבֵּט אֵלֶיהָ / וְהִנָּהּ עֵרֹם וְעֶרְיָה

וַתַּלְבֵּשׁ עוֹר וּבָשָׂר / לִהְיוֹת לְנֶפֶשׁ חַיָּה

וַתְּצַוֶּה עָלֶיהָ / נְטוֹת מִדַּרְכֵי גְוִיָּה

לְמַעַן תָּסוּר מִשְּׁאוֹל / מַטָּה לְדַרְכֵי עֲלִיָּה

וַתְּלַמְּדֶהָ לָקַחַת / מוּסַר אֱמֶת וְתוֹכָחַת.


יֵשׁ מֵאַיִן גָּלְמִי / יוֹם הִמְצֵאתָ וְאִם סָר

הִלְבַּשְׁתָּ וְסוֹכַכְתָּ / גִּידִים וְעֶצֶם וּבָשָׂר

מְרֻקָּם בְּצֶלֶם אֱלֹהִים / לְכָל-מַעֲשֶׁיךָ הֱיוֹת שָׂר

וַתְּצַו עָלָיו חֻקִּים / וּמִצְוֹת, שֵׂכֶל וּמוּסָר

לְבַל-יֵט יָמִין וּשְׂמֹאל / וּמִנְּתִיבָם אַל-יָסָר

וְחַי בָהֶם לְעוֹלָם / וּכְבוֹדוֹ לֹא יֶחְסָר

יַחְשׂךְ מִנִּי-שָׁחַת / נַפְשׁוֹ וְהִיא כְּפֹרָחַת.


מֵאָז חֻבְּרוּ יַחְדָּו / נַפְשִׁי וּגְוִיָּתִי

יַחַד נוֹסְדוּ לָסוּר / מִנְּתִיבוֹת עֲלִיָּתִי

לָלֶכֶת נְתִיב תַּאֳווֹת / כִּבְהֶמְתִּי וְחַיָּתִי

וְהָיָה דָשֵׁן וְשָׁמֵן / מַרְעִיתִי בְּצִיָּתִי

וָאֶשְׁכַּח אֵל עוֹשַׂי / שָׁלֵו מִדֵּי הֱיוֹתִי

וְלֹא עָלְתָה עַל-לִבִּי / רֵאשִׁיתִי וְאַחֲרִיתִי

מְשׁוּבָתִי נִצַּחַת / צָרַעַת פֹּרָחַת.


נֶפֶשׁ כָּל-חַי בְּיָדְךָ / וְאִם הִיא כְלוּאָה בְמַאֲסַר

גֹּלֶם נִבְזֶה וְנִשְׁחַת / דַּעַת וְשֵׂכֶל חָסָר

וְאִם-עֲוֹנוֹת תִּשְׁמָר-יָהּ / וּכְפִשְׁעֵיהֶם תִּסָּר

מִי יַעֲמֹד לְפָנֶיךָ / אִם נָגִיד וְאִם טִפְסָר

אָנָּא לְמַעַן שְמֶךָ / סְלַח אֱלֹהֵי הַמּוּסָר

הֲלֹא הֵן מַעֲשֵׂה יָדֶיךָ / הַנֶּפֶשׁ וְהַבָּשָׂר

וּבְשׁוּבָה וָנַחַת / פְּדֵם מֵרֶדֶת שָׁחַת.


חֲטָאַי בְּפִי אֶתְוַדֶּה / וּבְלִבִּי אֶנָּחֵם

זְדוֹנִי כִּי עָשָׂה-לִי / אֲשֶׁר לֹא יַעֲשֶׂה לוֹחֵם

קְרָא-נָא לְהַנְעִים חַיַּי / בְּשֶׁקֶט וְשַׁאֲנָן לְהַנְחֵם

כָּל-עוֹד נִשְׁמָתִי בִּי / תִּזְכֹּר בְּרֹגֶז רַחֵם

וּבְשָׂרִי וְעַצְמוֹתַי / כַּדֶּשֶׁא הַפְרִיחֵם

וְאֶשְׂבְּעָה תְמוּנָתֶךָ / כַּהֲקִיצוֹתִי תְשִּׂיחֵם

שָׁלוֹם שָׁלוֹם וְנַחַת / אַל-תַּעֲרֹץ וְאַל-תֵּחָת.
