אלי שית רצון (תחנון)

אֵלִי, שִׁית רָצוֹן / לְעָם רָצוּץ, וּמִסִּינָי

מְנִיתָם אָז כְּעֶדְרֵי צֹאן, / נְחִיתָם עַל-יַד נֶאֱמָנָי;

וְשִׂים קֵץ זֶה קֵץ קִיצוֹן / לִבְנוֹת קִרְיַת אַרְמוֹנָי,

וּלְיוֹם זֶה תִּקְרָא צוֹם וְיוֹם רָצוֹן / לַאדֹנָי.


בְּתֹם לֵבָב וּבְנִקְיוֹן / כַּפַּיִם אֲשַׁחֲרֶךָ,

וּלְךָ אֶקְרָא עֶלְיוֹן / וּמֵהוֹדִי אֲפָאֲרֶךָ;

וְחֹם נַפְשִׁי כְּחֹם צָיוֹן, / לְךָ צָמְאָה לְשׁוֹרְרֶךָ.

וְאַתָּה, אֵלִי, שִׁית רַעְיוֹן / לְהָכִין לִבּוֹת יְצוּרֶיךָ –

וְיֶעֱרַב שִׂיחִי וְהֶגְיוֹן / לִבִּי לְפָנֶיךָ, אֲדֹנָי!


רְאֵה, אֵל, מַקְהֲלוֹתֶיךָ / חֲרֵדִים הַיּוֹם לְהַלְּלֶךָ –

וְהֵם עַמְּךָ וְנַחֲלָתֶךָ / אֲשֶׁר פָּדִיתָ בְּגָדְלֶךָ.

וּמִמְּעוֹן שִׁבְתֶּךָ / הַבֵּט נִצָּב לְמוּלֶךָ

וְעַל חַסְדְּךָ וַאֲמִתֶּךָ / הִדָּרֵשׁ לְשׁוֹאֲלֶיךָ;

וְעוֹרֲרָה נָא גְּבוּרָתֶךָ / וְתוֹשִׁיעַ מְיַחֲלֶיךָ –

מָכוֹן לְשִׁבְתְּךָ / פָּעַלְתָּ, אֲדֹנָי!


הֲלֹא אַתָּה מְנָת חֵלֶק – / שַׁפֵּר נַחֲלַת עַמֶּךָ!

וְאֵשׁ אֹכְלָה וְאוּר דּוֹלֵק – / וְתִגְאֶה לַהֲרֹס קָמֶיךָ!

הֲדֹף דּוֹחֵף, מְחַץ דּוֹלֵק / וְתִשְׁפֹּךְ עֲלֵיהֶם זַעְמֶךָ

וּמִתּוֹכָם, כְּיֶלֶק / לְלַקְלֵק דַּם תְּמִימֶיךָ,

מִדּוֹר דּוֹר בֹּק, בַּלֵּק, וְקַלְקֵל וּמְלֹק בְּזַעְמֶךָ!

וְיָד עַל כֵּס יָהּ: בַּעֲמָלֵק / מִלְחָמָה לַאדֹנָי.
