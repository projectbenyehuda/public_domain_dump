אֱלֹהַי שְׁמַע נָא

אֱלֹהַי, שְׁמַע נָא לְקוֹל עַבְדֶּךָ, / עֲשֵׂה לִי לְמַעְנָךְ כְּפִי חַסְדֶּךָ,

וְקַבֵּץ נְדוּחַי אֱלֵי אַרְצֶךָ / מְהֵרָה, וְתִגְאַל שְׁאָר עֶדְרֶךָ.


בְּשִׁמְךָ, אֱלֹהִים, שְׁלַח עֶזְרֶךָ / לְהַצִּיל שְׁאֵרִית כְּלַל עַמֶּךָ,

אֲשֶׁר הֵם מְחַכִּים לְיוֹם יִשְׁעֶךָ / שְׁלַח עֻזְּךָ צוּר וְלִמְשִׁיחֶךָ.


רְצֵה נָא תְפִלּוֹת בְּבֵית קָדְשֶׁךָ, / אֲשֶׁר הֵם חֲשׁוּבִים כְּקָרְבָּנֶךָ,

מְשׁוּלִים כְּאֶתְרוֹג בְּשֻׁלְחָנֶךָ, / עֲשׂוּיִם כְּקֶדֶם בְּהֵיכָלֶיךָ.


הֲכִינֵם, הֲבִינֵם בְּצִדְקָתֶךָ / וְתַהְרֹס לְזֵדִים כְּפִי כֹחֶךָ,

וְתַעְנֶה מְחַכִּים לְיוֹם יִשְׁעֶךָ, / לְהַצִּיל שְׁאֵרִית כְּלַל עַמֶּךָ.


מְדֻכִּים, מְעֻנִּים, בְּנֵי עִירֶךָ – / תְּכַפֵּר לְחֶטְאָם בְּחֶמְלָתֶךָ,

וְתִפְתַּח לְבָבָם לְתוֹרָתֶךָ / לְהָבִין לְהוֹרוֹת בְּבֵית קָדְשֶׁךָ.
