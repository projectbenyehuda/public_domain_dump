אֲנִי אָמָה לְאָמָה

אֲנִי אָמָה לְאָמָה, / עַל כֵּן דְּבָרַי לָעוּ —

מִי יִתֵּן בְּחָכְמָה — / וַיַּעֲבֹר עָלַי מָה.


בַּמִּקְדָּשׁ בְּעָמְדִי / עָמַדְתִּי בִּבְרִית שְׁלוֹמִים

עִם אָחִי — וְעַבְדִּי / הִנּוֹ; וְלָמָּה בְקָמִים

קָם עָלַי בְּאֵידִי / וַיַּעְקְבֵנִי פְעָמִים?

הָאֵיבָה וְשִׂטְמָה / נָטַר אֲשֶׁר נוֹדָעוּ,

בָּא אָחִי בְמִרְמָה / גַּם הוּא וְעָשָׂה בְעָרְמָה.


רַעְיַת חֵן יְשִׂיחָהּ / לִבָּהּ וְתֹאמַר בְּנַפְשָׁהּ:

אַיֵּה דוֹד לְקָחָהּ / וּפְאֵר נְבוּאָה חֲבָשָׁהּ,

צִיץ קֹדֶשׁ בְּמִצְחָהּ / וּצְנִיף מְלוּכָה בְּרֹאשָׁהּ?

וּבְיוֹם אַף וְחֵמָה / בִּגְדֵי פְאֵר נִקְרָעוּ,

עַד נוֹתְרָה עֲרֻמָּה / וַתַּעֲטֶה סוּת כְּלִמָּה.


הֵן צִירַי חֲזָקִים / עָלַי וְאֶצְעַק כְּחוֹלָה,

אַךְ צָרַי מְשַׂחֲקִים — / הֵן הִיא חֲטָאָה גְדוֹלָה!

אֵיךְ עָמְדוּ שְׁחָקִים / וּבְנֵי אֱלֹהִים בְּגוֹלָה?

עַד כַּמָּה וְכַמָּה / לַעַג וּבוּז יִשְׂבָּעוּ?

אֵיךְ רָעָב וְצָמָא / בִּדְבַר נְבוּאָה חֲתוּמָה!


מָתַי קֵץ פְּלָאוֹת — / וּדְבַר אֱלֹהִים צְבָאוֹת

‘הוֹאֶל נָא’ בְמַרְאוֹת / אָז, לֹא בְהָקִיץ בְּמַרְאוֹת.

עַל דִּבְרֵי נְבוּאוֹת / עוֹדָם עֲתִידוֹת וּבָאוֹת —

עַל אָזְנֵי מְזִמָּה / קוֹלוֹת אֱמֶת נִשְׁמָעוּ:

‘דֹּמִּי, בַּת תְּמִימָה, / וּתְהִי מַשְׂכֻּרְתֵּךְ שְׁלֵמָה.’
