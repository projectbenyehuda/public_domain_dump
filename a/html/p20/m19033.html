תִּשְׁבִּי שְׁלַח לִי אֵלִי

(הבדלה)

תִּשְׁבִּי שְׁלַח לִי, / אֵלִי, לְרַפֵּא מַחְלִי!


אַל נָא בְחוֹבִי / תַּרְחִיק זְּמַן הַיֶּשַׁע,

כִּי לֹא בְחַרְבִּי / אֶבְטַח – וְלִי לֹא תוֹשָׁע.

רִבָה יְרִיבִי / מַהֵר וְעֵינָיו הָשַׁע,

וּסְמֹךְ בְּנָפְלִי / יָדַי וְהַעְצֵם חֵילִי.


בִּי, אֵל, הֲתַחְרִישׁ / מִבֵּן יְקַו רַחְמֶיךָ?

וִיטוּר וְנָפִישׁ / אֵיךְ יִירְשׁוּ כַרְמֶךָ?

קוּמָה, וְעַל אִישׁ / שָׂעִיר שְׁפֹךְ זַעְמֶךָ

וּשְׁמַע לְקוֹלִי / וּנְשָׂא כְקֶדֶם דִּגְלִי.


רַב מִהְיוֹתִי / יוֹשֵׁב מְשׁוֹמֵם בִּשְׁבִי,

הָחֵשׁ פְּדוּתִי / עַל יַד נְבִיאָךְ תִּשְׁבִּי;

שׁוּב אֶל נְוָתִי / וּדְמֵה לְעֹפֶר וּצְבִי,

יַצֵּב גְּבוּלִי / וּבְנֵה דְּבִיר הֵיכָלִי.


׳מַה לָּךְ בְּשִׂיחֵךְ, / תִּצְעֲקִי כִשְׁבוּיָה?

הִנֵּה מְשִׁיחֵךְ / נִגְלָה וּבָא אֵלִיָּה!

רִקְחֵךְ וְרֵיחֵךְ / אָרַח בְּהַר מוֹרִיָּה,

וַעְלֵי חֲלִילִי / שִׁיר יַעֲנֶה נִין מַחְלִי.׳
