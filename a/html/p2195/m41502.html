נָשִׁיר לָאֵל בָּחַר

לברית מילה. סימן: נסים. מנגינה: חוסיני.


נָשִׁיר לָאֵל בָּחַר בָּנוּ / מִכָּל הָעַמִּים

וְתוֹרָתוֹ הִנְחִיל לָנוּ / רָחְבָה מִיַּמִּים

מִצְוָה רִאשׁוֹנָה צִוָּנוּ / יָהּ צוּר עוֹלָמִים

פְּרוּ וּרְבוּ בַּיִת תִּבְנוּ / וְתִהְיוּ תַמִּים

מוֹרָאֵנוּ וְחִתֵּנוּ / עַל כָּל לְאֻמִּים.


סַפְּרוּ הוֹד חֲתַן מִשְׂרָה / זֶרַע רְחוּמִים

אֲמִתּוֹ צִנָּה סֹחֵרָה / לֵילוֹת וְיָמִים

קִיֵּם מִצְוָה זוֹ חֲמוּרָה / כִּקְטֹרֶת סַמִּים.

פְּרוּ וּרְבוּ


יִגְדַּל יִנָּצֵל מִקָּטוֹב / מִמִּתְקוֹמְמִים

יְהִי זַרְעוֹ כְגַן רָטֹב / כֻּלָּם חֲכָמִים

מֵיְיָ יֵצֵא הַטּוֹב / שׁוֹכֵן מְרוֹמִים.

פְּרוּ וּרְבוּ


מַלְכִּי, בָּרֵךְ קְהָלֵנוּ / כֻּלָּם תְּמִימִים

וּזְכֹר לָנוּ אֱלֹהֵינוּ / אַהְבַת קְדוּמִים

נַקְרִיב קָרְבָּן בְּאַרְצֵנוּ / זִבְחֵי שְׁלָמִים.

פְּרוּ וּרְבוּ
