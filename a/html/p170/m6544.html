מִבֵּית מְלוֹנִי קַמְתִּי בְצוּקִי

לחן: אחות קטנה תפילותיה עורכ

סימן: משה הק[טן]

מִבֵּית מְלוֹנִי / קַמְתִּי בְצוּקִי

וּלְבֵית אֲדוֹנִי / בָּאתִי כְחֻקִּי

וְאֵלָיו עֵינִי / תְּלוּיָה בְצַעֲקִי

בְּקָרְאִי עֲנֵנִי / אֱלֹהֵי צִדְקִי: תה' ד, ב


שְׁרִירוּת יִצְרִי - / יַשְׁפִּילֵנִי

בְּהָלְכִי בְקָרִי / יַבְהִילֵנִי

וְחַטַּאת מֶרִי / יַנְחִילֵנִי

לֹא יַרְפֵּנִי / עַד בִּלְעִי רֻקִּי איו' ז, ב


הִבְאִישׁ רֵיחִי / כִּי בִי רָדָה

וְהֵמַר רוּחִי / וְעָלַי עָדָה

הִכְשִׁיל כֹּחִי / וְאוֹתִי צָדָה

שָׂם לַחֲרָדָה / נֶשֶׁף חִשְׁקִי: ישע' כא, ד


הָקֵץ מִשְׁנַת / יְגוֹנָיו הוֹזֶה

וְהַקְרִיבָה שְׁנַת / פְּדוּת לְעָם נִבְזֶה

וּרְצֵה רִנַּת / כּוֹאֵב וְרָזֶה

כִּי אַתָּה מְנָת / חֶבְלִי וְחֶלְקִי: תה' טז, ח
