[מָתַי יְשֵׁנֵי דֵעָה יֵעוֹרוּ]

סימן: משה


מָתַי יְשֵׁנֵי דֵעָה / יֵעוֹרוּ מִשְּׁנָתָם,

מָתַי שׁוֹכְחֵי נְסִיעָה / יָבִינוּ לְאַחֲרִיתָם,

מָתַי הֲלוּמֵי תוֹעֶה / יְסִירוּן יֵין תַּאֲוָתָם?

טַח מֵרְאוֹת עֵינֵיהֶם / מֵהַשְׂכִּיל לִבֹּתָם: [ישע' ד, יח]

שׁוֹקְטִים עַל שִׁמְרֵיהֶם, / אִמְרוּ: אֵיפֹה הֵמָּה?

וְטִיְרוֹתָם וְחַצְרֵיהֶם / בִּלְתָּם בְּמִדְבַּר שְׁמָמָה –

עֲזָבוּם יוֹם מַסְעֵיהֶם / לִשְׁכֹּן חוֹרֵי אֲדָמָה

וּזְהָבָם וְכַסְפֵּיהֶם / לְנִדָּה יוֹם הַחֵמָה

וְכָל עֹשֶׁר בְּאוֹצְרוֹתָם / לֹא הִצִּיל לְנַפְשׁוֹתָם:

הוֹי, הַמַּרְבֶּה וְלֹא לוֹ / וְעַבְטִיט מַכְבִּיד עָלָיו,

וְהַמִּתְהַלֵּל בְּחֵילוֹ / וְהוּא רֶשֶׁת לְמַעְגָּלָיו,

וּמֶה הֹוֶה בְּכָל עֲמָלוֹ / וְיַלְדֵי-יוֹם מַבְהִילָיו,

לֶאֱרוֹת תְּנוּבוֹת פָּעֳלוֹ / וְלִקְצֹר פְּרִי מַעֲלָלָיו

וְאִם יֵצְאוּ לִקְרָאתָם / דַּרְכָּם וַעֲלִילוֹתָם: [יחז' לו, יז]

חָנֻּנִי חָנֻּנִי / כִּי יַד אֵל נָגְעָה בִי,

מְתֵי סוֹדִי, הוֹרוּנִי / מָה אֶעֱשֶׂה לְיוֹם רִיבִי –

וִידֵי מַחְשְׁבוֹת אוֹנִי / יָסֹלּוּ אֶת נְתִיבִי?

אֲנִי טָמֵא בְּדַם עֲוֹנִי / וְאֵיךְ אֶעֱלֶה אֶל אָבִי –

נָא הַמֹּצְאוֹת אוֹתָם / יְכַפְּרוּ עַל חַטֹּאתָם:
