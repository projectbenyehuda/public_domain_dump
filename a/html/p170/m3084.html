הלנצח אהי תאב לרעי

הֲלָנֶצַח אֱהִי תָאֵב לְרֵעַי / וְנִגְעֵי הַנְּדֹד לָעַד נְגָעָי

וְיָשַׁבְתִּי עֲלֵי לַהַב חֲרָבוֹת / וְרִפַּדְתִּי עֲלֵי־גַחַל יְצוּעָי

וְהִכָּה הַנְּדֹד לִבִּי וּפָצַע / וּמִי בִלְתָּם לְחַבֵּשׁ אֶת־פְּצָעָי

עֲצֵי־עֵדֶן אֲשֶׁר מֵאָז נְטַעְתִּים / וּמָנַע הַזְּמָן לֶאְרוֹת נְטָעָי

יְפַחֵד כִּי־סְעִפֵיהֶם יְלַהֵט / יְקֹד קִרְבִּי וְתֹפֶת בֵּין צְלָעָי

וְאֵיכָה יֶחֱרַד לָזֹאת וְהִנֵּה / לְעִתּוֹתַי יְכַבּוּמוֹ דְמָעָי

לְךָ חָרוּת עֲלֵי־לוּחַ לְבָבִי / וְאָח דֹּדָיו כְּחוֹתָם עַל־זְרֹעָי

יְמֵי פֵרֽוּדְךָ נִצְחִי וְשָׁנַי / בְּסַפֵּר חַסְדְּךָ אֶחְשֹׁב רְגָעָי

וּמִדֵּי דַּבְּרִי שִׁמְךָ כְנֹפֶת / עֲלֵי חִכִּי וְרֹאשׁ־פֶּתֶן בְּמֵעָי

וְעֵת שָׁכְבִי יְצֻרֶיךָ שְׂשׂוֹנִי / וְזִכְרָךְ בַּהֲקִיצִי שַׁעֲשֻׁעָי.
