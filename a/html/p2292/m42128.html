שָׁבִים וְעוֹבְרִים עִמְדוּ

בארג׳יל טבת תרנ״ד. על מות מרדכי סמאג׳א.


שָׁבִים וְעוֹבְרִים עִמְדוּ בְצִיר וּמְגִנָּה / שְׂאוּ נְהִי עַל שׁוֹכֵב בְּאַדְמָתֵנוּ

פֹּה אִישׁ חַיִל אַרְיֵה שַׁחַל רָבַץ וְחָנָה / עָלְתָה נַפְשׁוֹ הַשָּׁמַיְמָה וְאֵינֶנּוּ

בְּיוֹם אַ״ךְ טֵבֵת יוֹם צָרָה וְתוֹכֵחָה / שְׁנַת נְהַר דֶּמַע תֵּרָדְ״נָה עֵינֵינוּ

עַל נָפְלוֹ, כַּנְּשָׁרִים נִקְרְחָה קָרְחָה / וּפִיק וְחִיל חַלְחָלָה בְכָל מְתָנֵינוּ

שָׂרֵינוּ מָרְדְּכַי סְמַאגָא כִי נֶאֱסַף / צְבִי בֵית מוֹעֵצָה הוּא בַעֲדָתֵנוּ

עִמּוֹ רוּחַ דִּמְעָה וְהַמַּדָּע אָסַף / מִמְשַׁח כְּרוּב בְּהִלּוֹ זִיו יִפְעָתֵנוּ

מִדְרַשׁ עֵץ חַיִּים בָּכֹה יִבְכֶּה בִדְאָבָה / עַל מְכוֹנֵן אָשְׁיוֹתָיו כִּי נֶעְדַּר מֶנּוּ

תַּלְמִידֵי חֲכָמִים שִמְחָתָם עָרְבָה / וּלְקוֹל צַעֲקָתָם צָלְלוּ אָזְנֵינוּ

הָעֲנִיִּים יִנְאֲקוּ כְּיוֹם שׁוֹד שַׁלְמַן / הוֹי זְמַן הִכָּה בְּחֵץ שָׁנוּן לִבֵּנוּ

אֲהָהּ כִּי חָסַרְנוּ מִשְׁעָן יָתֵד נֶאֱמָן / בּוֹ תָלִינוּ כָל תֶּלְיֵנוּ וְקַשְׁתֵּנוּ

שְׁלוּחֵי אֶרֶץ יִשְׂרָאֵל הֵמָּה עֵדָיו / הֱיוֹתוֹ אוֹהֵב לְאֶרֶץ מוֹלַדְתֵּנוּ

הָאֱלֹהִים יִתֵּן חֶבְלוֹ בְגַן חֲסִידָיו / לְהִתְעַדֵּן עִם נִשְׁמוֹת אֲבוֹתֵינוּ.
