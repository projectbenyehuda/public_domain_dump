להכנסת שבת.

שַׁבָּת, שַׁבָּת, / מֵאַיִן בָּאת?

שָׁלוֹם לָךְ, בַּת, / מַה־טּוֹב מַרְאֵךְ!

מַרְאֵךְ נָאוֶה, / אַתְּ גַּן רָוֶה:

כָּל־הַדָּוֶה / יַבִּיט וָחָי.

וָחַי גַּם־אַתָּה תִּהְיֶה מֵעָתָּה;

מַדּוּעַ חַתָּה? / עוּרָה אֵפוֹא.

אֵפוֹא קוּם גִּילָה; / רַק־זֹאת אוֹאִילָה:

הִנֵּה אוֹבִילָה / מַרְגּוֹעַ לָךְ.

לָךְ אֶשְׁמַע־נָא, / מִכְלַל עֶדְנָה;

מִמֵּךְ אָמְנָה / אֶדְרוֹשׁ דָּבָר.

דָּבָר אֶדְרוֹשׁ / קֶדֶם וָרֹאשׁ:

מָה רֵאשִׁיתֵךְ? / מָתַי? וָאָן?

וָאָן נִסַּכְתִּי, / וָאָן בֹּרַכְתִּי,

וָאָן מָלַכְתִּי, / הַקְשֵׁב דּוֹד צָח.

דּוֹד צַח עָשָׂנִי, / וַיַּמְלִיכַנִּי;

וַיִּקְרָאַנִּי / עֹנֶג לַכֹּל.

לַכֹּל? וָאֵיךְ / יֵחַל הוֹדֵךְ?

אַךְ מִי דוֹדֵךְ? / אַךְ מִי וָמִי?

וּמִי יִהְיֶה / אִתָךְ יִחְיֶה?

אַיֵה, אַיֵה / חַלָּמִישׁ צוּר?


צוּר רָם, רַב־דֵּעַ, / בִּי יִשְׁתַּעְשֵׁעַ;

וַיְּהִי לִי רֵעַ, / וַאֲנִי אֵלָיו.

אֵלָיו נִקְנֵיתִי, / אֵלָיו נִשֵּׂאתִי;

אוּלָם נִשְׁבֵּיתִי / מִגֶּבֶר טוֹב.


טוֹב הַדָּבָר: / משֶׁה גָבָר;

הָאִישׁ גָבָר / זָכָה וַיֵּיטֶב.

וַיֵּיטֶב לִי, / אַף לִקְהָלִי;

וּמְשׂוֹשׂ גִּילִי / הָיֹה הָיָה.


הָיָה אִישׁ יוֹשֶׁר, / הִדְרִיךְ בַּכּשֶׁר;

מָאַס הָעשֶׁר, / תּוֹרָה קִבֵּל.

קִבֵּל רַב־זֶבֶד / אָז בֶּן־יוֹכֶבֶד,

וַיְּכֻנֶּה עֶבֶד / אֶל־אֵל טָהוֹר.

טָהוֹר הַיּוֹם, / לָנוּ פִדְיוֹם;

גַּם סַבַּטְיוֹם / עֵדוּת נִתָּן.

נִתַּן אֵלֵינוּ / יוֹם שַׁבַּתֵּנוּ;

פִּצְחוּ, רַנֵּנוּ / עַל הַמַּתָן.
