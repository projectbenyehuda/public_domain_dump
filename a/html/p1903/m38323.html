משל עד: אַיִל וּכְלָבִים

הַמִּתְגָּאֶה בְרֹב חֵילוֹ / יִפֹּל וְאֵין עֹזֵר לוֹ


הָאַיִל עַל אֲפִיקֵי מַיִם כְּבֹאוֹ / רָצָה לִשְׁבֹּר אֶת צְמָאוֹ / וַיִּתֵן אֶת עֵינָיו בַּמַּיִם / וַיַּרְא בְּצִלּוֹ כִּי הוּא בַעַל קְרָנַיִם / וַיֹּאמַר: עָשַׁרְתִּי מָצָאתִי אוֹן לִי / מִי יַעֲרֹךְ אֵלַי לְהִתְהַלֵּל כְּמַהֲלָלִי / קַרְנַי לֹא כְקַרְנֵי הַשּׁוֹר / אַךְ נְטוּעוֹת בְּמִצְחִי בְמִישׁוֹר / וּסְדוּרוֹת בְּגֶרֶם הַמַּעֲלוֹת / וּבְחֵן וָיֹפִי שְׁתֵּיהֶן כְּלוּלוֹת / וּמִי הוּא אֲשֶׁר לְפָנַי יִתְיַצָּב / מִשְׁפַּט מָוֶת לוֹ חָרוּץ וְנֶחְצָב / לַאֲשֶׁר לִי עָלָיו תְּלֻנָּה / בָּהֶן אֲנַגַּח יָמָּה וְצָפוֹנָה / וְכַאֲשֶׁר רַגְלָיו בַּמַּיִם נָדִים / הִנֵּה כְּלָבִים הֹלְכִים צְמָדִים / וּמַנְהִיגֵיהֶם מְסִיתִים בּוֹ / וְהָאַיִל לָנוּס בְּחָפְזָה אָמַר בְּלִבּוֹ / וַיָּנָס וְלֹא פָנָה אֶל אֵם דֶּרֶךְ וְאֶל הַמִּשְׁעוֹלִים / אַךְ אֶל עֲבִי הַיַּעַר וְאֶל סִבְכֵי הַחֲרֻלִּים / וַתַּכְשִׁילֵהוּ עֲצָתוֹ / וַיִּנָּקֵשׁ בְּגַאֲוָתוֹ / כִּי בְּשִׁיתוֹ אֶל הַסְּבָכִים עֵינָיו / וְהִנֵּה נֶאֱחַז בַּסְּבָךְ בְּקַרְנָיו / וְשָׁם הַכְּלָבִים הִדְבִּיקוּהוּ / וְהַצַּיָּדִים נְתָחוּהוּ / וַיִּנָּקֵשׁ בְּקַרְנָיו אֲשֶׁר הָיוּ לוֹ לְאוֹן / לִפְנֵי שֶׁבֶר גָּאוֹן / וְכָל אֵבָרָיו כָּהֵם לְנִתּוּחַ נִמְסָרוּ / וְאִם הֵם עַל כֻּלָּם בְּעֵינָיו יָשָׁרוּ.


וְהַמָּשָׁל תְּהִי עֵינְךָ רֹאָה וְאָזְנְךָ שֹׁמַעַת / וְאַל תִּירָא לָגֶשֶׁת אֶל עֵץ הַדַּעַת / פֶּן תְּהִי מָהִיר בְּרַעְיוֹנֶיךָ / לִקְרַב לְכָל אֲשֶׁר יָשָׁר בְּעֵינֶיךָ / כִּי אֶת אֲשֶׁר תְּקָרֵב וְתִבְחַר בְּרַעְיוֹנִים / לִפְעָמִים הוּא כַּחֲמַת תַּנִּינִים וְרֹאשֵׁי פְּתָנִים / וְאִם מֵאַהֲבָה וְגַאֲוָה בָּהֶם תִּנָּקֵשׁ / יְהִי לְךָ לְפַח וּלְמוֹקֵשׁ / וַאֲשֶׁר תַּחֲשֹׁב כְּמִדְרַךְ נַעַל / עֹז וְתִפְאֶרֶת בְּלִבְּךָ יָעַל.
