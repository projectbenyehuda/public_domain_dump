משל כה: כְּבָשִׂים, אַיִל וָאֲרִי

אַל תִּהְיֶה לְלַעַג וְלָצוֹן / בִּרְאוֹתְךָ כִּי אֲמָרֶיךָ לֹא יַעֲלוּ לְרָצוֹן


כְּבָשִׂים בָּעֵדֶר נִשְׁאָרוּ / וְהָרוֹעִים הָלְכוּ בַאֲשֶׁר בָּחָרוּ / וְלֹא סָגְרוּ הַדֶּלֶת בַּעֲדָם / וּרְחוֹקִים הֵמָּה מֵאָדָם / וַיֵּצְאוּ מִן הָעֵדֶר כֻּלָּם / לָתוּר בַּשָּׂדֶה מַאֲכָלָם / הֵנָּה וָהֵנָּה בְּהִפָּזְרָם / וְרָעוּ כְבָשִׂים כְּדָבְרָם / וְעַל הָעֵשֶׂב לֶאֱכֹל סוֹד הִמְתִּיקוּ / הֵמָּה יָצְאוּ מִן הָעֵדֶר לֹא הִרְחִיקוּ / וְהִנֵּה אֲרִי נֹסֵעַ אַחַר הַמִּדְבָּר / וְלָרוּץ עֲלֵיהֶם יִתְגַּבָּר / וְכַאֲשֶׁר הִתְבּוֹנְנוּ לוֹ זֶה אֶל זֶה קָרָא / אַרְיֵה שָׁאַג מִי לֹא יִירָא / וַיִּרְאוּ כִּי אֹרֵב הוּא לָהֶם / וּמָנוֹס אָבַד מִנְהֶם / וַיֹּאמְרוּ אֶל הָאַיִל הַהֹלֵךְ בְּרֹאשָׁם / לֶךְ נָא אֶל הָאֲרִי אֶל הַדֶּרֶךְ אֲשֶׁר בָּא שָׁם / וּבְחֵלֶק לָשׁוֹן תְּפַתֶּנּוּ / אוּלַי יְפֻתֶּה וְיֵלֵךְ מֵעָלֵינוּ / וַיִּסַּע הָאַיִל מֵהֲמוֹנָיו / וַיָּשֶׁת אֶל הַמִּדְבָּר פָנָיו / וּבְטֶרֶם יִקְרַב אֵלָיו / וַיְסִיתֶנּוּ בְחֵלֶק מִלָּיו / וַיֹּאמֶר לוֹ: אֲדֹנִי הַמֶּלֶךְ בָּרוּךְ אַתָּה בְּבוֹאֶךָ / וְיוֹם טוֹב תִּמְצָא בְּעֵינֵי כָל רוֹאֶיךָ / וַיַּעַן לוֹ הָאֲרִי: יוֹם טוֹב אֲשֶׁר אֶהְיֶה מוֹצֵא / מִמְּךָ וּמֵחֲבֵרֶיךָ לִי יֵצֵא / מַה לְּךָ אֵלַי לִשְׁלוֹמִים / שַׁחֵת דְּבָרֶיךָ הַנְּעִימִים / הֲמֶלֶךְ כָּמוֹנִי מִדְּבַר פִּיךָ יְבֻשָּׂר / אֵין שִׂמְחָה אֶלָּא בְּבָשָׂר / אֲשֶׁר אֹכַל מֵעָלֶיךָ / וְלֹא מִן דְּבַר פִּיךָ אֲהַלֲלֶךָ.


וְהַמָּשָׁל בַּאֲשֶׁר יִרְאֶה עֵת צָרָה / וְכִיסוֹ וּתְכוּנָתוֹ בְעֵינָיו לָרֹב יְקָרָה / מִבְּלִי תֵת לְנַפְשׁוֹ כֹפֶר / וְיַחְשֹׁב לְהִמָּלֵט בְּאִמְרֵי שָׁפֶר / וְלָשׁוֹן רַךְ וַחֲלַקְלַקּוֹת / וְכָל דְּבָרָיו קֹדְחֵי אֵשׁ מְאַזְּרֵי זִיקוֹת / וְלֹא יִמָּלֵט בְּעֵת זַעַם / וְיֵשׁ אֲשֶׁר לוֹ יְהִי נֹעַם / בְּאָמְרוֹ: קַח אֶת הַחַיִל הַזֶּה / וְכַסְפּוֹ וּזְהָבוֹ בוֹזֶה / כֶּסֶף כֶּסֶף רִבָּה / אַל יוֹצִיאוּ עָלָיו דִּבָּה / כִּי בְעֵת צָרָה יִשָּׂא רֹאשׁוֹ / כֹּל אֲשֶׁר לָאִישׁ יִתֵּן בְּעַד נַפְשׁוֹ.
