משל לו: זְאֵב וּבְהֵמוֹת

אֲשֶׁר רַגְלָיו לָלֶכֶת בְּגָזֵל נִסָּה / עֵינוֹ עַל הוֹן זוּלָתוֹ לֹא חָסָה


זְאֵב אֲשֶׁר הָיָה מִשְׁנֶה לַמֶּלֶךְ וָשָׂר / מְגַמַּת פָּנָיו שָׂם לְשַׁחֵת כָּל בָּשָׂר / גֹּזֵל וְחֹמֵס טֹרֵף וְעֹקֵר / כָּל הַנִּמְצָא יִדָּקֵר / וְהַחַיָּה וְהָעוֹף וְהַבְּהֵמָה / אֲשֶׁר שֻׁלַּח בְּמִשְׁפְּחוֹתָם מְהוּמָה / לִצְעֹק עָלָיו הָלְכוּ לָאֲרִי / וַיֹּאמַר: אַךְ רַע וָמֶרִי / אִם כְּדִבְרֵיכֶם שִׁנָּיו גִּלָּה / הַכְּצַעֲקָתָהּ הַבָּאָה אֵלַי עָשָׂה כָּלָה / אֶשְׁפְּטֶנּוּ עַד אֲשֶׁר מִלְּהַשְׁחִית יִפְנֶה עֹרֶף / וּמִשִּׁנָּיו אַשְׁלִיךְ טָרֶף / וַיַּגֵּד לַזְּאֵב: בֹּא אֵלַי בְּיוֹם מָחָר / הַקְשִׁיבָה וַעֲשֵׂה אַל תְּאַחֵר / וַיָּבֹא אֵלָיו וַיְדַבֵּר עִמּוֹ הָאֲרִי עַזּוֹת / וַיֹּאמֶר לוֹ: לָמָּה עָשִׂיתָ כָזֹאת / אַל תּוֹסֵף עוֹד בַּבְּהֵמוֹת לַעֲרֹף / וְלֹא בַּחַיּוֹת טֶרֶף לִטְרֹף / וַאֲשֶׁר לְךָ לְמַאֲכָל הֵמָּה / כָּל נְבֵלָה וּטְרֵפָה מִן הָעוֹף וּמִן הַבְּהֵמָה / אֲשֶׁר תִּמְצָא מֵת בַּשָּׂדֶה / וְלַחַי לֹא תִהְיֶה אֹרֵב וְצֹדֶה / וְאִם לִשְׁמֹר דְּבָרַי לֹא תוּכַל / הִשָּׁבְעָה לִי פֶּן תֹּאכַל / בָּשָׂר עַד שְׁנָתַיִם יָמִים / לְכַפֵּר פְּשָׁעֶיךָ כְּתוּבִים וַחֲתוּמִים / וְזֶה מִשְׁפָּטְךָ אֲשֶׁר לְךָ אֶחֱזֶה / וַיִּשָּׁבַע לוֹ עַל הַדָּבָר הַזֶּה / לְבִלְתִּי אֲכֹל בָּשָׂר שְׁתֵּי שָׁנִים / מִיּוֹם יִטְרֹף כָּל אֲשֶׁר בַּחַיּוֹת חֹנִים / וַיִּפָּרֵד הַזְּאֵב מִשָּׁם וַיֵּלֶךְ לְדַרְכּוֹ / וְהָאֲרִי נִשְׁאַר כְּמֶלֶךְ בְּסֻכּוֹ / וְלֹא אָכַל הַזְּאֵב מִכָּל הֹלֵךְ עַל אַרְבַּע / מִשָּׁמְרוֹ אֶת הַשְּׁבוּעָה אֲשֶׁר נִשְׁבַּע / אִם לֹא מָצָא בַּשָּׂדֶה אוֹ בַּמְסִלָּה / טְרֵפָה מָשְׁלֶכֶת אוֹ נְבֵלָה / וַיְהִי הַיּוֹם וַיִּרְעַב וַיִּפֶן אָנֶה וְאָנָה / וַיַּרְא וְהִנֵּה רָחֵל שְׁמֵנָה / נֶחְמָדָה לְמַרְאֶה וְטוֹבָה לְמַאֲכָל / וַיֹּאמַר: לִשְׁמֹר מִצְוָה מִי יוּכַל / וַיְרִיבוּן בּוֹ רַעְיוֹנָיו / אִם יָשִׁית אֵלֶיהָ פָנָיו / וַיֹּאמֶר בְּלִבּוֹ: אִם תְּנַצְּחֵנִי תַאֲוָתִי / עוֹד לְהַכּוֹת כָּל חַי כַּאֲשֶׁר עָשִׂיתִי / מִן הַיּוֹם הַהוּא יִהְיֶה לִי שְׁתֵּי שָׁנִים / לְבִלְתִּי אֲכֹל בָּשָׂר מִיּוֹם אֶל יוֹם נִמְנִים / זֶה הַדָּבָר אֲשֶׁר לַמֶּלֶךְ נִשְׁבַּעְתִּי / לְקַיֵּם הַשְּׁבוּעָה בְלִבִּי תַרְתִּי / מִנְיַן יְמוֹת הַשָּׁנָה שְׁלֹשׁ מֵאוֹת שִׁשִּׁים וַחֲמִשָּׁה / פְּתִיחַת עֵינַי יִהְיֶה יוֹם וְהָעֲצִימָה לַיְלָה חֲדָשָׁה / וַיִּפְתַּח עֵינָיו אַחַר אֲשֶׁר עָצַם אוֹתָן יַחַד / וַיְהִי עֶרֶב וַיְהִי בֹקֶר יוֹם אֶחָד / וְכֵן עָשָׂה לִשְׁתֵּי שָׁנִים בַמִּסְפָּר / וְסָר עֲווֹנוֹ וְחַטָּאתוֹ יְכֻפָּר / וְעֵינָו פּוֹנוֹת לָעֵז אֲשֶׁר בָּחָרוּ / וְהִבִּיטוּ אֵלָיו אֲשֶׁר דָּקָרוּ / וַיֹּאמַר: הֵן קִדַּמְתִּי כַפָּרָתִי לִפְנֵי הַטֶּרֶף / וַיֹּאחֵז אֶת הָעֵז בָּעֹרֶף / וַיְנַתְּחֶהָ וַאֲכָלָהּ / כַּאֲשֶׁר מִנְהָגוֹ בַתְּחִלָּה / וְעוֹד יָדוֹ נְטוּיָה עַל הַחַיּוֹת / כִּימֵי עוֹלָם וּכְשָׁנִים קַדְמֹנִיּוֹת.


וְהַמָּשָׁל בְּאִישׁ אֲשֶׁר לִגְנֹב וְלִגְזֹל תָּמִיד נִסָּה / וְעֵינוֹ עַל הוֹן אֲחֵרִים לֹא חָסָה / שֹׁלֵל וּבוֹזֵז הוֹנָם וַעֲמָלָם / וְאִם יִשָּׁבַע בְּמַעֲמַד כֻּלָּם / יַעֲרִים בְּלִבּוֹ עַל שְׁבוּעָתוֹ / וְיַחְשֹׁב לִהְיוֹת נָקִי מֵאָלָתוֹ.
