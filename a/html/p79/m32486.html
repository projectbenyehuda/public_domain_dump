אָרַךְ גָּלוּתִי

כתובת בראש השיר: ואמר בשבח החכם אבי יצחק אברהם בן עטא (= בן נתן) יאריך ה' ימיו.

אָרַךְ גָּלוּתִי / בְּעָנְיִי וְדַלּוּתִי,

וְהִנְּנִי, אֱיָלוּתִי, / מְגַלֶּה־לְּךָ רִיבִ[י].

כֹּחִי עֲזָבַנִי / וְכַמֵּת חֲשָׁבַנִי

וְחֶטְאִי סְבָבַנִי / וְדִוְיִי וּמַכְאוֹבִי.

[וְ]תָמִיד בְּשַׁוְעָתִי / לְךָ צוּר יְשׁוּעָתִי,

וְאַשְׂחֶה בְּדִמְעָתִי / לְעַרְסִי וּמִשְׁכָּבִי.

[וּ]מֵרֹב עֲווֹנוֹתַי / וְעֹצֶם זְדוֹנוֹתַי

מְרֻבִּים יְגוֹנוֹתַי / בְּנַפְשִׁי וּבִלְבָבִי.

וּבֹשְׁתִּי בְתוֹכַחְתִּי / וְכַמֵּת נִשְׁכַּחְתִּי

בְּעָזְבִי לְמִשְׁפַּחְתִּי / וְאָחִי וּבֵית אָבִי.

[וְ]לוּלֵי אֲשֶׁר נוֹעַד / לְעַמּוֹ בְּעֶת־צַעַד

וְסִינַי מְאֹד רָעַד / בְּעָמְדִי בְמַצָּבִי,

[וְ]הֵקִים לְמוֹשִׁיעִים / בְּכָל־דּוֹר לְנוֹשָׁעִים

נְדִיבִים וְגַם שׁוֹעִים / מְפִיגֵי לְמַעְצָבִי.

[וּ]מֵהֶם נְשִׂיא דוֹרוֹ / נְסִיכוֹ וְכַבִּירוֹ

מְאִירוֹ וּמַנְהִירוֹ / וּפוֹדוֹ כְּמִשֶּׁבִי,

בְּחִירוֹ בְנוֹ נָתָן / נְדִיב לֵב כְּמוֹ אֵיתָן

מְהֻדָּר כְּמוֹ חָתָן / בְּחַדְרִי וּמוֹשָׁבִי.

[וּ]פָנָיו פְּנֵי חַמָּה / וְלִבּוֹ לְבַב חָכְמָה

וְגִבּוֹר בְּמִלְחָמָה / כְּאַרְיֵה וְגַם לָבִיא.

מְיַשֵּׁר מְסִלָּתוֹ / לְנֶגֶד קְהִלָּתוֹ,

וּמַנְהִיג בְּחֶמְלָתוֹ / קְטַנִּי וְגַם שָׂבִי.

וְהוּא שָׂר לְכָל־שָׂרַי, / מְחַזֵּק יְדֵי מוֹרַי,

וְחַרְבּוֹ עֲלֵי צָרַי / וְחוֹמָה לְהַסְבִּיבִי,

וְהוּא אוֹר שְׁנֵי עֵינַי, / וְהוּא אָב לְכָל־בָּנַי

וּמַשְׁעֵן לְנִשְׁעָנַי / וְתִקְרִי וּמַעְזָבִי.

בְּנוֹתַי בְּהוֹד עִלָּה / בְּקִשּׁוּט עֲדִי כַלָּה,

וְכִיסִי הֲלֹא מִלָּא / בְכַסְפִּי וּבִזְהָבִי,

וְצִוָּה לְהַלְבִּישִׁי / תְּכֵלֶת וְגַם מֶשִׁי

וְצָנִיף עֲלֵי רֹאשִׁי / וְסוּסִים לְמֶרְכָּבִי.

שְׁמוֹ טוֹב כְּאֶזְרָחִי / וְרֵיחוֹ כְּמֶרְקָחִי

מְהַלֵּךְ בְּמִזְרָחִי / צְפוֹנִי וּמַעְרָבִי.

לְבָנִים וְגַם־בָּנוֹת / יְתוֹמִים וְאַלְמָנוֹת

יְמִינוֹ בְמַתָּנוֹת / כְּמָטָר לְהַרְבִּיבִי.

וְהוּא אוֹר לְהִלּוּכִי / וְסַהַר בְּמַחְשַׁכִּי

וְגֶפֶן לְסִכּוּכִי / וּפִרְיוֹ לְהַתְנִיבִי.

לְדָת אֵל בְּפֶה גָמַר / וְחֻקָּיו מְאֹד שָׁמַר,

מְעֻלֶּה כְמוֹ תָמָר / וְעָנָיו כְּאֵזוֹבִי.

וּמַשְׁתִּיק לְמַדְמִימַי / וּמַפְעִים לְמַפְעִימַי

וְנִלְחָם בְּכָל־קָמַי / לְבַדּוֹ וּבִקְרָבִי.

אֲדוֹנִי אֲבָא אִסְחָק / קְחָה שִׁיר בְּךָ הוּחַק

קְרוֹבִים וּמִמֶּרְחָק / יְרוּצוּן……..בִי

הֲדַרְתִּיו בְּמוֹ טוּרִים / בְּנֹפֶךְ וְסַפִּירִים

וּמַשְׁתִּיק לְכָל……/ …………[בִי]

[וְ]יָרוּם נְשִׂיא הַדּוֹר / יְחִי עוֹד לְדוֹר [וָ]דוֹר

יִהְיֶה עָתִיד לִגְדּוֹר / …לִשְׂגּוֹב …[בִי].
