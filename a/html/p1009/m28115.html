גֶּשֶׁם נְדָבוֹת אוֹ יְאוֹר מִצְרַיִם

גֶּשֶׁם נְדָבוֹת אוֹ יְאוֹר מִצְרַיִם / עֵת יַעֲלֶה אוֹ יִזְּלוּ עֵינָיִם

אוֹ אֵשׁ מְלַהֶטֶת כְּמוֹפֵת הַסְּנֵה / תִּיקַד יְקוֹד אוֹ נֶחְמְרוּ מֵעָיִם

חָשַׁךְ מְאוֹר עֵינַי וְלֹא בַּנְתִּי לְזֹאת / הַאִם אֱמֶת אוֹ חָשְׁכוּ אוֹרָיִם

עַד שָׁב עֲלוֹת שַׁחַר כְּמוֹ נֶשֶׁף וְעוֹד / אֶרְאֶה לְצָהֳרַיִם כְּבֵין עַרְבָּיִם

גַּם רַעֲיוֹנַי חָרְדוּ גַּם רָעֲדוּ / זַעַף וְחַלְחָלָה בְּכָל מָתְנָיִם

וָאֶשְׁאֲלָה סִבָּה לְכָל עוֹבֵר וְשָׁב / אֵיךְ אֶמְצְאָה מָזוֹר לְפִיק בִּרְכָּיִם

וַיֹּאמְרוּ: אַתָּה פְּצוּעַ הַנְּדוֹד / לָקְחָה פְּרִידָה עַל פִּשְׁעֲךָ כִּפְלָיִם

וָאֶזְכְּרָה פֵּרוּד נְסִיכֵי מַעֲלָה / לִבִּי לְנָסְעָם נֶחֱלָק לִשְׁנָיִם

מֵהֶם מְאוֹר עֵינַי וְלֹא עֵינַי לְבַד / רַק הוּא לְכָל חָבֵר מְאוֹר עֵינָיִם

מֵאִיר בְּמַעֲנֵה פִּיו לְכָל שׁוֹאֵל כְּאוֹר / בֹּקֶר וְכַשֶּׁמֶשׁ בְּלֵב שָׁמָיִם

אָכֵן סְמָכָהוּ גְּבִיר הַדּוֹר אֲשֶׁר / גָּדַר לְכָל פֶּרֶץ כְּאִישׁ בֵּינָיִם

לֵוִי וּמַטֵּהוּ הֲלֹא פָּרַח וְצָץ / הָיָה כְּעֵץ שָׁתוּל עֲלֵי כָּל מָיִם

לָבַשׁ מְעִיל הַחֵן וְתִפְאָרָה וְהוֹד / גַּם פַּעֲמוֹן מִשְׂרָה עֲלֵי שׁוּלָיִם

גַּם מַחֲזִיקָיו עוֹזְרָיו גַּם אוֹהֲבָיו / מֵהֶם חֲכַם לֵב אוֹ לְבֶן שִׁנָּיִם

מִי יִתְנֵנִי חִישׁ בְּאֶבְרָה עַל כְּנַף / נֶשֶׁר וְאָטוּס לֹא בַּעֲצַלְתָּיִם

לִפְנֵי גְּדוֹלָם עִם קְטַנָּם אֶכְרְעָה / אֶקֹּד וְאֶשְׁתַּחֲוֶה עֲלֵי אַפָּיִם

אָמְנָם לְבֶן יַקִּיר אֲשׂוֹחֵחַ הֲכִי / הִפְצִיר וְנָתַן לִי כְּסוּת עֵינָיִם

אֵלֶּה אֲשֶׁר גָּנְבוּ לְבָבִי בֶּאֱמֶת / הִנְנִי בְּלִי לֵיבָב כְּבַד אָזְנָיִם

תָּנַע גְּוִיָּתִי כְּשִׁכּוֹר תֶּהֱמֶה / תִּדְמֶה לְפִסֵּחַ נְכֵה רַגְלָיִם

נִמְהָר בְּלִי שֵׂכֶל כְּפֶתִי מַחֲרִישׁ / נִרְאָה כְּאִישׁ נִדְהָם רְפֵה יָדָיִם

אֵל חַי בְּחַסְדֶּךָ פְּנֵה אֵלַי בְּחֵן / תֶּן לִי כְּיוֹם אֶחָד וְאִם יוֹמָיִם

תּוֹסִיף לְהַרְאוֹתִי תְּמוּנָתָם וְאָז / אֶת שֵׁם מְקוֹמָם אֶקְרָא מַחֲנָיִם

אָז הַלְּבָנָה תַּהֲפֹךְ אוֹרָהּ כְּאוֹר / חַמָּה וְאוֹר חַמָּה לְשִׁבְעָתָיִם.
