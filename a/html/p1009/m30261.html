שְׁבָט אַכְזָר וְעַז מֵצַח בְּקַרְחוֹ

שְׁבָט אַכְזָר וְעַז מֵצַח בְּקַרְחוֹ / וְזַלְעָפָה אֲחָזַתְנִי בְּרוּחוֹ

וְהַסּוֹבֵא בְּכוֹס יִגְעַל אֲבָל כִּי / דְּלִי יַחְפֹּץ אֱלֵי פִיהוּ בְּגִיחוֹ

וְיֹאמַר כִּי אֱלֹהִים הוּא בְחָרוֹ / לְמַזָּל טוֹב לְעֵת כָּזֹאת בְּיַרְחוֹ

וְיַעַן זֹאת מְנוּחָתִי עֲדֵי עַד / וּפֹה אֵשֵׁב וְלֹא הֵבִין בְּשִׂיחוֹ

אֲבָל מַשְׂכִּיל וְאִישׁ בַּעַל מְזִמּוֹת / בְּתוֹרָתוֹ יְהִי עָמֵל בְּטָרְחוֹ

לְזֶה נִקְרָא אֱמֶת צַדִּיק בְּדוֹרוֹ / לְזֶה נִקְרָא אֱמֶת מוֹשֵׁל בְּרוּחוֹ

וְיִבְטַח כִּי לְגַן עֵדֶן פְּעָמָיו / וְהַצֶּדֶק יְהֶא הוֹלֵךְ נְכוֹחוֹ.
