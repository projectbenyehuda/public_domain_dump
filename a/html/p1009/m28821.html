בְּמַרְחֶשְׁוָן יְהִי רוֹחֵשׁ לְבָבִי

בְּמַרְחֶשְׁוָן יְהִי רוֹחֵשׁ לְבָבִי / בְּדָבָר טוֹב וְאֵינֶנִּי לְנָבִיא

וּמִתִּגְרַת זְמָן שָׁלֵו וְשׁוֹקֵט / חֲצוֹת לֵיל יִזְּלוּ גִשְׁמֵי רְבִיבִי

וְלֹא אִירָא לְקוֹרוֹת הַזְּמַנִּים / וּבוֹ אֶשְׁתֶּה לְיֵינִי עִם חֲלָבִי

דְּבַשׁ חָכְמָה רְאוּ תַחַת לְשׁוֹנִי / וּמִמֶּנּוּ אֱהִי רוֹדֶה בְנִיבִי

וְאֵיךְ אִירָא לְעַקְרָב מֵעֲקִיצָה / וְהַנָּחָשׁ וְסִיעָתוֹ רְכוּבִי

וְעַד מָתַי חֲסַר לֵב בַּתְּנוּמוֹת / וְהָקִיצָה לְסוֹבֵב עַל נְתִיבִי

רְאֵה כִי אָרְכוּ לֵילוֹת לְהַשְׂכִּיל / וְאַל תֵּשֵׁב קְדוֹרַנִּית אֲהוּבִי

וְנֵרָךְ יִהֲיֶה דָלוּק בְּהִלּוֹ / וְתִתְנַשֵּׂא וְתָקוּמָה כְלָבִיא.
