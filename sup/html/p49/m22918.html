[שור כי]

— — ∪ — — — ∪— — — // — — ∪ — — — ∪ — — —

שׁוּר כִּי בְנוֹת צִיּוֹן1 בְּהַשְׁקִיפָן // אֵשׁ הַסְּנֶה תִּבְעַר2 בְּאוֹר אַפָּן: 3

אֵשׁ לֹא תְלַבֶּה אֶת סְבִיבֶיהָ, // כִּי אִם תְּלַבֶּה אֶת לְבַב צוֹפָן4.

נַחְנוּ הֲרוּגֵי אֵשׁ שְׁבִיב יֹפִי — // קִרְאוּ לְמִישָׁאֵל וְאֶלְצָפָן5!


  1. בנות ציון ־ בנות ישראל.  ↩

  2. אש הסנה תבער – על פי שמות ג ב: והנה הסנה בוער באש והסנה איננו אכל.  ↩

  3. אפן – פניהן.  ↩

  4. כסנה הבוער באש ואיננו אכּל כן אש זו: היא שורפת את החושקים אבל אינה פוגעת בבנות, הנשארות קרות ואדישות.  ↩

  5. למישאל ואלצפן – להוציאנו אל מחוץ למחנה כמו שמישאל ואלצפן הוציאו את נדב ואביהוא שרופי אש ה'. על פי ויקרא י ד: ויקרא משה אל מישאל ואל אלצפן…ויאמר אליהם קרבו שאו את אחיכם (נדב ואביהוא שנשרפו באש ה') מאת פני הקדש אל מחוץ למחנה.  ↩
