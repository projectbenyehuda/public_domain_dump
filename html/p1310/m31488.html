לְחוּצָה בְּאֹרֶךְ

״לְחוּצָה, בְּאֹרֶךְ / גָּלוּת מְשֻׁכֶּלֶת,

מְעֻנָּה בְפֶרֶךְ / מֹרְאָה וְנִגְאֶלֶת,

לְהֵיכַל דְּבִירֵךְ / אֲשִׁיבֵךְ לְמֶמְשֶׁלֶת

וְאֶגְדֹּר פִּרְצֵךְ / וְאָשִׂים לְמַחֲצֵךְ

אֲרוּכָה, וְאֶרְצֵךְ / וְאֹהֲבֵךְ נְדָבָה״.


״וְעַד אָן יְשׁוּעָה / אֶהְיֶה מְחַכָּה –

וַאֲנִי פְּרוּעָה / מִצֵּל וְסֻכָּה?

כַּשֶּׂה תוֹעָה, / כְּרָחֵל מְבַכָּה,

עָבַר קְצִירִי / וְכָלָה בְצִירִי

וְנוֹסַף צִירִי – / וְצִירִי לֹא בָא!״


״יְחִידָה, לִבְשִׁי / בִגְדֵי תִפְאֶרֶת

וְנֵזֶר חִבְשִׁי / לְכָבוֹד וַעֲטֶרֶת

כִּי עוֹד תִּנְשִׁי / בֹשֶׁת, וּמַשְׂכֹּרֶת

תִּקְחִי, וְיֵחַת / זֶרַח וְנַחַת,

תִּשְׂמְחִי תַּחַת / הֱיוֹתֵךְ עֲזוּבָה״.


״חָלְפוּ זְמַנַּי / בְּיָגוֹן וּמְהוּמָה

וּפָנַי – עֲווֹנַי / כִּסּוּ בִכְלִמָּה,

וּמִכָּל בָּנַי / נוֹתַרְתִּי עֲרֻמָּה.

לְרִיק אֶעֱמֹל, / אִם לֹא תִגְמֹל

בְּחַסְדְּךָ וְתַחְמֹל – / וְאָנָה אֲנִי בָא?״


״קוֹמֵם אֲקוֹמֵם / דְּבִירֵךְ הַנֶּחֱרָב

וְקַרְנֵךְ אֲרוֹמֵם / בְּקִרְיַת מֶלֶךְ רַב

וּלְדַל מְשׁוֹמֵם / יְשׁוּעָתִי תִקְרַב

וְאַשְׁמִיד רְשָׁעִים / זֶרַע מְרֵעִים –

וְעַל כָּל פְּשָׁעִים / תְּכַסֶּה אַהֲבָה״.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
