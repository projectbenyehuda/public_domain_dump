מַה לְּךָ בֶּן אָדָם

סימן: מ(שה בר) יע(ק)ב

מַה לְּךָ בֶּן אָדָם / נְשָׁמָה בְאַפּוֹ

מַה לְּךָ נִרְדָּם / וְהַיָּם בְּזַעְפּוֹ?

שׁוּר בָּשָׂר וָדָם / קָשֶׁה עָרְפּוֹ

שׁוּב וְיָשׁוּב אֱלֹהִים / מֵחֲרוֹן אַפּו: / דב' יג, יח


יָפְיְךָ הִשִּׁיאֲךָ / בִּימֵי שַׁחֲרוּת

וְתַאֲוָה תַשִּׁיאֲךָ / הֲלֹךְ בִּשְׁרִירוּת

לֵב, וְזֹקֶן יִרְאֲךָ / כְּתַב בִּמְרִירוּת

מָוֶת יִקְרָאֲךָ: / סוּרָה שְׁבָה-פֹה: / רו' ד, א


עֲתְּדָה לְךָ צֵדָה / לְדֶרֶךְ יִשְׁגֶּה

וּתְהִי לְךָ לְעֵדָה / בְּיֵצֶר מַשְׁגֶּה

בְּטֶרֶם בֹּא קְפָדָה / וְאוֹתְךָ יַגֶּה

כַּאֲשֶׁר עֵת יֶהְגֶּה / כְּפִיר עַל טַרְפּוֹ: / ישע' לא ד

בֶּן כְּעֶבֶד לִפְנֵי / אֲדֹנָיו נַעֲנֶה

בְּפִיהוּ אֵין מַעֲנֶה / מִלֵּב זוֹנֶה –

אַךְ לְחֶסֶד מִשְׁנֶה / חֲטָאָיו יִפְנֶה,

טוֹב תְּמַלֵּא כַפּוֹ / וְכוֹנֵן טַרְפּוֹ:
