כָּתְנוֹת פַּסִּים לָבַש הַגָּן

כָּתְנוֹת פַּסִּים / לָבַש הַגָּן / וּכְסוּת רִקְמָה / מִדֵּי דִשְׁאוֹ

וּמְעִיל תַּשְׁבֵּץ / עָטָה כָל עֵץ / וּלְכָל עַיִן / הֶרְאָה פִלְאוֹ

כָּל צִיץ חָדָשׁ / לִזְמָן חֻדַּשׁ / יָצָא שׂחֵק / לִקְרַאת בּוֹאוֹ

אַךְ לִפְנֵיהֶם / שׁוֹשָׁן עָבַר / מֶלֶךְ כִּי עָל / הוּרַם כִּסְאוֹ

5 / יָצָא מִבֵּין / מִשְׁמַר עָלָיו / וַיְשַׁנֶּה אֵת / בִּגְדֵי כִלְאוֹ

מִי לֹא יִשְׁתֶּה / יֵינוֹ עָלָיו / הָאִישׁ הַהוּא / יִשָּׂא חֶטְאוֹ.
