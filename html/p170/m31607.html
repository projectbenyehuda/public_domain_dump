תֵּבֵל בְּאֵיבַת לֵב

תנחומים אל אבו אלפרג – יאשׁיהו בן בזאז במות עליו אמו

תֵּבֵל בְּאֵיבַת לֵב וּבִצְדִיָּה / תִּשְׂנָא בְנֵי שֵׂכֶל וְתוּשִׁיָּה

כִּשְׁמָהּ פְּעָלֶיהָ, וְשִׁית זוֹנָה / נִקְרֵאת וְסֹרֶרֶת וְהֹמִיָּה.

תִּשְׂטֶה וְתִתְהַפֵּךְ כְּמוֹ חֹמֶר / חוֹתָם מְשַׂחֶקֶת וּבוֹכִיָּה.

לִבָּהּ לְבַב אַרְיֵה וְתֵרָא אֶל / פֶּתִי כְמוֹ עֶגְלָה יְפֵהפִיָּה.

וַיִּנְחֲלוּ דַרְכָּהּ יְלָדֶיהָ / פִּתּוּ בְאִמְרֵי שָׁוְא וּבִרְמִיָּה.

לֹא חָמְדוּ מֵהָאֱנוֹשׁ כִּי אִם / שֹׁנִים אֲשֶׁר פָּקוּ פְּלִילִיָּה.

בָּנוּ מְעוֹן עָוֶל וְלַחֹרֶב / קָרְאוּ מְכוֹנֵיהֶם וְלִשְׁאִיָּה.

הִתְהוֹלְלוּ עַד כִּי מְלָאָם לֵב / כֶּסֶל לְהָרִים יָד בְּיֹאשִׁיָּה.

פָּגְעוּ בְיוֹלַדְתּוֹ אֲשֶׁר מִכָּל / בָּנוֹת לְצוּר הָיְתָה תְרוּמִיָּה.

גֶּפֶן אֱמֶת עַל מַעְיְנֵי צֶדֶק / שֹׂגָה בְגַן יֹשֶׁר וּפוֹרִיָּה.

הֹלֵךְ אֲבֶל־אִמּוֹ בְלֵב קֹדֵר / צֹעֵק לְמִפְקָדָהּ בְּתַאְנִיָּה,

לָהּ חָל וְחָלוּ כָל־בְּנֵי תֵבֵל / לוֹ, גַּם לְבַב הַתֹּם מְאֹד נִהְיָה.

עַיִשׁ וְכִימָה אָבְלוּ, כִּי הִיא / הָיְתָה לְבַד לָהֶם שְׁלִישִׁיָּה.

גַּם הָאֲרָצוֹת רָגְזוּ, עַד כִּי / מֵהַבְּכִי לֹא שָׂגְבָה קִרְיָה.

הַט אָזְנְךָ, מִגְדַּל תְּבוּנוֹת עַל / אָשְׁיוֹת אֱמֶת בָּנוּי לְתַלְפִּיָּה,

וּלְטֹשׁ לְךָ עַיִן רְאוֹת יַד אֵל / מֵאָז אֲשֶׁר עַל הַיְּקוּם הוֹיָה.

עֵצָה יְעוּצָה הִיא, וּמִי יִהְיֶה / רָב בַּעֲצַת רַב־הָעֲלִילִיָּה.

לֹא נִבְצְרוּ מִיַּד שְׁאוֹל בִּקְרָב / חֶרֶב חֲנִית מַסָּע וְגַם שִׁרְיָה.

עַל הָאֲדָמָה הָאֱנוֹשׁ עֵירֹם / נוֹלָד וְשָׁמָּה יַהֲלֹךְ עֶרְיָה.

נַחְנוּ בְנֵי עָפָָר וְהָעָפָר / יָשׁוּב עֲלֵי אֶרֶץ כְּשֶׁהָיָה.




את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
