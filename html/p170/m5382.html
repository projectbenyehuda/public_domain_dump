[מַה-תַּשְׂמְאִיל דֹּודִי]

מַה-תַּשְׂמְאִיל דֹּודִי / אֹו תַאֲמִין

סוּרָה וּבֵין שָׁדַי / הֹואֵל וְלִין.

דֹּוד מִדְבְּרֵי פִיו עָ- / לַי עָתְקוּ

אָכֵן כְּצוּף דֹּדָיו / לִי מָתְקוּ

לִבִּי וּמֹורָשָׁיו / יִנָּתְקוּ

אִם יַחֲלִיף חֻקֹּו / אֹו יֵשְׂטְ מִן

דַּרְכֹּו וְאֶל-הֹוכַח / אִם יַאֲזִין.

מִיֹּום וּמִלַּיִל / צוּר חִבְּרֹו

שָׁחְרוּ קְוֻצֹּותָיו / צַח זָהֳרֹו

אִם-תִּשְׁאֲלוּ עָלָיו / מַה-תָּאֳרֹו

חֹבֵשׁ צְנִיף-אֹפֶל / עֹטֶה סְדִין

בֹּקֶר, בְּאַף יִטְרֹף / לֵב אָב וְנִין.
