מְרוֹמָם אֵל בְּיִשְׂרָאֵל

סימן: מש[ה]

מְרוֹמָם אֵל / בְּיִשְׂרָאֵל! – / בִּצְדָקָה שְׁמוֹ נִקְדָּש

וְיִתְעַלֶּה / בְּעַם אֵלֶּה, / שְׁנַת רָצוֹן כִּי תְחֻדָּשׁ, –

שְׁנַת חַיִים / שְׁנַת פִּדְיוֹם / לְבִנְיַן בֵּית הַמִּקְדָּשׁ

יִשְׁתַּבַּח / וְיִתְפָּאֵר / וְיִתּגַּדַּל וְיִתְקַדָּשׁ!


מְלֹךְ בְּאֵימָה / תְּמִים חָכְמָה, / וְהַכְנִיעַ כָּל מוֹלֵךְ

וְלִגְבוּלוֹ / בְּבֵית זְבוּלוֹ / הָשֵׁב עַם דַּל וְחֵלֶךְ

וּבִרְנָנִים / עֲלֵי מִנִּים / אֶל בֵּית אֲדֹנָי נֵלֵךְ

בְּשׁוֹפָרוֹת / וּבַחֲצוֹצְרוֹת / נָרִיעַ לִפְנֵי הַמֶּלֶךְ

בְּקוֹל הָמוֹן / בְּתוֹךְ אַרְמוֹן / וְנֶכֶד אַדְמוֹן יֻדָּשׁ:


שְׁלַח יֶשַׁע / לְעָם נוֹשַׁע / בְּעֵת תִּקְרָא שְׁנַת רָצוֹן

וְאָז אֶשְׂמַח / בְּבוֹא צֶמַח / לְשַׂמֵּחַ שְׁאָר הַצֹּאן

וְנִגְאָלִים / יְהוּ עוֹלִים / לְבֵית פְּנִימִי וְהַחִיצוֹן

וּבְמוֹעֲדִים / יְהוּ חַדִּים / לְעֵת הַקֵּץ וְהַקִּיצוֹן

וְהַחֹדֶשׁ / בְּהִתְחַדֵּשׁ / לִקְרֹא: מְקֻדָּש מְקֻדָּש

יִשְׁתַּבַּח / וְיִתְפָּאַר / וְיִתְגַּדַּל וְיִתְקַדָּשׁ!
