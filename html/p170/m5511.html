איכה יתאפק איש גלו גיליו

אֵיכָה יִתְאַפַּק / אִישׁ גָּלוּ גִילָיו / וּזְמַנּוֹ שָׁלַח / יָדוֹ בִּשְׁלֹמָיו

קַלּוּ מִנִּי־רָץ / לֵילוֹת חֶדְוָתוֹ / וּבְגֵרוּת כָּלוּ / בַהֶבֶל יָמָיו

מַכְמֹרֵי יָגוֹן / פָּרְשׂוּ־לוֹ יָמִים / גַּם־פַּחֵי פֵרוּד / טָמְנוּ לִפְעָמָיו

וּזְמָן אֵיךְ לֹא־חָס / עַל־מִפְרַד אַחִים / גַּם לֹא־נִכְמָרוּ / לָהֶם נִחוּמָיו.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
