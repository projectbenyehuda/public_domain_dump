חוּן וְרַחֵם אֵלִי

חתימה: חלפון [אברהם]


חוּן וְרַחֵם אֵלִי / עַל פְּזוּרַת שִׁבְיוֹן

מַהֲרָה חַכְלִילִי / שַׁלְּחָה לָהּ פִּדְיוֹן

הַעֲלָהּ הֵיכָלִי / תֶּחֱזֶה אַפִּרְיוֹן

תֹּאמְרָה שִׁיר פֶּלִ / בַּחֲצַר הַר צִיּוֹן


לָךְ אֲשַׁלֵּם צוּרִי / קָרְבְּנוֹת מִזְבֵּחַ

שׁוֹר וְכֶבֶשׂ בָּרִיא / כֹּהֲנִי נַתֵּחַ

עֵז וְתוֹר צִפּוֹרִי / יַעֲלוּ לָרֵיחַ

יָהּ שְׁמַע נָא קוֹלִי / מִבְטְחָה לָאֶבְיוֹן


פִּי בְּשִׁמְךָ יָרוֹן / אֵל גְּבִיר נַעֲרָצָה

צוּר הָסִירָה חָרוֹן / עַמְּךָ קַבֵּצָה

עִמְּךָ זִכָּרוֹן / לַאֲבוֹתָם נִמְצָא

כֹּהֲנִים תִּהְיוּ לִי / יוֹדְעִים בְּחֶזְיוֹן


וַאֲסַדֵּר שִׁירָה / לָךְ אֱלֹהֵי עוֹלָם

מִמְּךָ כָּל צוּרָה / יוֹדְעִי סוֹד נֶעְלָם

אֵין לְךָ מַכִּירָה / מִבְּרוּאִים כֻּלָּם

אוֹמְרִים שִׁירָתָם / אֵין לְךָ צוּר דִּמְיוֹן


נָפְלוּ נָא כּוֹרְעִים / כָּל בְּנֵי הָאָדָם

מַאֲזִינִים שׁוֹמְעִים / אֶל דְּבַר אֵל פּוֹדָם

וַהֲמוֹן הָרָעִים / הֵן בְּיוֹם זֶה אָבְדָם

בַּעֲלוֹת גּוֹרָלִי / אֶל נְוֵה חִזָּיוֹן.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
