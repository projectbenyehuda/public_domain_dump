שְׂחוֹק עֲשׂוֹת

שְׂחוֹק עֲשׂוֹת זִמָּה / כְּמוֹ שְׂאוֹר עִסָּה / לְהֶבֶל הוּא דָמָה / וְשִׂמְחָה מָה עוֹשָׂה

כְּאִישׁ עָלָיו חוֹבוֹת / וְעִנְיְנֵי רִיבוֹת / אֲלָפִים וּרְבָבוֹת / וְהָיוּ לְמַשָּׂא

וּמֶלֶךְ הַמִּשְׁפָּט / דְּאָגָה בִלְבָבוֹ / לְפָנָיו אֵיךְ יָבוֹא / וּפָנָיו אֵיךְ יִשָּׂא

מְזֻמָּן הוּא לִדּוֹן / בְּמִשְׁפַּט הָאָדוֹן / לְהַמְשִׁיכוֹ כְמוֹ / בְהֵמָה אֲבוּסָה

וְהֵכִין לִבְלִי חֹק / מְנָחוֹת הַשְּׂחוֹק / וְהָיָה חֲלִיפָה / סְגֻלָּה כְנוּסָה

הֲלֹא אוֹיָה עָלָיו / לְרֹעַ מִפְעָלָיו / הֲיֵשׁ לוֹ מְרַחֵם, / וְעַיִן לוֹ חָסָה

וְעֵת לִשְׂחֹק עָבַר, / וְעֵת לִבְכּוֹת גָּבַר / לְפֻרְעָנוּת הוּכַן / וְהָיָה לִמְשִׁסָּה.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
