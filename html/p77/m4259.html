[לוּ נַחֲלֵי שִׁיר]

כתובת: ושלחתי לו אלה השירים


לוּ נַחֲלֵי שִׁיר חָרְבוּ דָלָלוּ / וּמֵי זְמִיר מִדֶּלְיְךָ יִזָּלוּ

אוֹ יְסוֹדָיו מֵאֲזַי לֹא נוֹסְדוּ / תּוֹךְ לוּחֲךָ חָלִיל וְתוֹף חוֹלָלוּ

רוּחַ צְפוֹנָךְ קוֹל מְחוֹלוֹת חוֹלְלָה / וּלְשִׂיחֲךָ חֵיל חוֹלְלִים יִחָלוּ

לוּ שָׁמְעוּ לִבּוֹת עֲרֵלִים נִיב הֲגוּת / זִמְרָתְךָ עָרְלַת לְבָבָם מָלוּ

מִן נָאֳמָךְ לֶמֶךְ הֲמוֹן מִנִּים יְעַן / וּבְנִיבְךָ בֵּין כָּל חֲדָרָיו יַעְלוּ

נֹפֶת וְצוּף תַּטִּיף בְּעֵת תִּטֹּף עֲדֵי / מִלִּבְּךָ יָבְלֵי דְּבָשׁ יוּבָלוּ

יַעְרַת דְּבָשׁ זָבַת שְׂפַת עֵטָךְ וּבַת / שִׁירָךְ תְּנוֹבֵב הוֹלְלִים נוֹאָלוּ

שִׂפְתֵּי יְשֵׁנִים דּוֹבְבוֹת שִׁיר בַּחֲלוֹם / אוֹ בְּהָקִיץ זִכְרְךָ רָגָלוּ

תּוֹךְ חֵיק נְגִינוֹת גִּדְּלוּךָ נוֹגְנִים / וַיִּשְׂמְחוּ כִי בֶן זְאֵב גָּמָלוּ

אָב הַנְּבוֹנִים לוֹ כְבָנִים וַעֲלֵי / שֻׁלְחַן יְקָרוֹ בַר תְּבוּן אָכָלוּ

אָב לֹא יְלָדָם אַךְ יְנִיקֵמוֹ חֲלֵב /חָכְמָה וּמַדָּע עַד אֲשֶׁר יִגְדָּלוּ

אָב לִי יְהַנְחִיל הוֹן יְשֻׁדּוּן שׁוֹדְדֵי / לַיְלָה אֲבָל הַשּׁוֹדְדִים אֻכָּלוּ

הוֹן נָחֲלוּ בָזוּ לְנֶגְדּוֹ כָל זְהַב / אוֹפִיר וְסַפִּיר בַּעֲדוֹ זָלָלוּ

הוֹן אַחֲרֵי מָוֶת בְּטוֹב יִזָּכְרוּ / אוֹסְפָיו וּמִמַּסְגֵּר נְדוֹד נֻטָּלוּ

הוֹן לא יְבַלֵּהוּ זְמָן אַךְ הַזְּמָן / יִכְלֶה וְלֹא יִבְלֶה רְכוּשׁ נָחָלוּ

הוֹן לֹא יְכִילֵהוּ מְלֹא תֵבֵל וְתוֹךְ / לִבּוֹת מְבִינִים תַּעֲצוּמוֹ כָלוּ

הוֹן לא יְהִי נִמְעַט בְּהוֹצִיאוֹ אֲבָל / יִשְׂגֶּה בְתִתּוֹ לַאֲשֶׁר יִשְׁאָלוּ

הוֹן לֹא לְחָפְרוֹ יָרְדוּ אֶל יַרְכְּתֵי / בוֹרוֹת אֲבָל עַל בָּמֳתֵי עָב עָלוּ

עָלוּ בְמַעְלוֹת הַמְּזִמּוֹת עַד אֲשֶר / כִּימָה וְעָשׁ וּכְסִיל מְאֹד יִשְׁפָּלוּ

הַקּוֹטְפִים שִׂיחֵי רְנָנוֹת מִבְּלִי / זֵכֶר פְּאֵר נָעְמָךְ לְשָׁוְא עָמָלוּ

תַּחַת בְּרוֹשׁ קוֹץ יִקְטְפוּ וּמְקוֹם קְנֵה / בשֶׂם וְצוּף קָנֶה וְסוּף קָמָלוּ

הַשּׁוֹאֲפִים לִסְפּוֹר וְלִזְכּוֹר קֵץ יְקַר / מֶחְקַר תְּהִלּוֹת הוֹדְךָ סָכָלוּ

גַּם שָׁאֲפוּ רוּחַ וְלָרִיק יִיעֲפוּ / וּמְקוֹם תְּהִלָּה תָּהֳלָה פָעָלוּ

הַנּוֹאֲמִים לִמְנוֹת הֲמוֹן כוֹכְבֵי זְבוּל / אָז כָּל עֲפַר אָרֶץ הֲלֹא יֶהְבָּלוּ

בָּךְ מַעֲלוֹת רָמוֹת לְאֵין קֵצֵה וְלא / יִכְלוּ וְאִלּוּ הַתְּהִלּוֹת כָּלוּ

בֵּינוֹת בְּנוֹת בִּינוֹת צְלָעֶיךָ דְּבִיר / אִוּוּ וּבִנְוֵה לִבְּךָ אָהָלוּ

אֹהֶל אֲשֶׁר יָהֵל בְּמוֹ אֹפֶל קְצוֹת / תֵּבֵל לְמוֹשָׁב יִכְסְפוּ גַם כָּלוּ

עֵינֵי חֲשֵׁכִים יֶחֱזוּ אוֹר דַּת בְּאוֹר / שִׁכְלָךְ וּבִמְאוֹר הִלְּךָ הֻלָּלוּ

גַּם פַּעֲמֵי דַּלִּים בְּצַעְדֶּךָ עֲלֵי / רֻכְסֵי רְנָנוֹת כַּצְּבִי אָזָלוּ

בִּידֵי יְדִידוּתָךְ יְדֵי רָשׁ תָּמְכוּ / כִנּוֹר וּבִזַּת הַזְּמִיר שָׁלָלוּ

תִּמְשֹׁךְ לְבָבוֹת בַּעֲבוֹתוֹת אַהֲבָה / עַד מִסְּבֹל אַהְבָתְךָ נֶחְשָׁלוּ

כִּמְעַט לְבָבוֹת מִשְׁכְּנוֹתָם נָטְשׁוּ / וּבְיוֹם נְדוֹד בַּחְלָלְךָ יֻטָּלוּ

זֶה בָחֲנוּ טוּחַי בְּנוּדָם מִמְּךָ / כִּמְעַט בְּנַפְשָׁם מָאֲסוּ בָּחָלוּ

חַתּוּ וְכִמְעַט בִּלְתְּךָ אֶשְתּוֹלְלוּ / כִי אֵיךְ בְּלִי לֵב לַעֲמוֹד יוּכָלוּ

יִתְחַלְחֲלוּ לוּלֵי חֲזוֹתָךְ יִחֲלוּ / וּבְחַסְדְּךָ חֶבְלֵי נְדוֹד סֻכָּלוּ

מֵאֵשׁ אֲנָחוֹת מֵי מְנוּחוֹת יִמְצְאוּ / לוּ אֶל נְוֵה אָהָלְךָ נִנְהָלוּ

מִי אִתְּךָ גָּר לֹא יְגוּרוֹ גוּר אֲרִי / וּבְיַם חֲרָדוֹת צוֹלְלָיו יִצְלָלוּ

הַשּׁוֹאֲבִים מֵי שִׁירְךָ לֹא יִצְמְאוּ / אִלּוּ יְאוֹרֵי הַזְּמִיר דָּלָלוּ


סימן

קֶסֶת זְמִירִי זָחֲלָה מֵחִיל / נוֹגְנִים וְלִישׁוּעָתְךָ תּוֹחֵל

דָּיֵק זְמִיר בָּנוּ וְגַם שָׁפְכוּ / בָהּ סוֹלְלָה וַתַּעֲמֹד בַּחֵיל
