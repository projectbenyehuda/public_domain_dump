צִיץ הוֹד

זה השיר שלח אותו בכתב אשר שלח להחכם השלם יעקב [סוראגה] ז״ל שליח עה״ק ירושלם תובבא שנת התק״י

סימן: צאלח


צִיץ הוֹד לְרֹאשִׁי וַעֲטָרָה / אָב הוּא לְכָל מִדָּה יְשָׁרָה

לֵךְ נָא, כְּתָבִי, אֶל יְדֵי אִישׁ / חָכָם אֲרִי שֶׁבַּחֲבוּרָה

הִשְׁתַּחֲוֵה מוּלוֹ וְנַשֵּׁק / יָדָיו וְאַתָּה לוֹ תְמוּרָה

אֶתְאַו כְּנָפַיִם כְּיוֹנָה / אָעוּף לְחַצְרוֹתָיו מְהֵרָה

אֶרְאֶה וְאֶתְעַדֵּן בְּזִיווֹ / נִתְעַלְּסָה יַחַד בְּחֶבְרָה

אִינַק שְׁדֵי שִׂכְלוֹ בְתַלְמוּד / תּוֹרָה וּפִלְפּוּל גַּם סְבָרָה

אֵשֵׁב וְלֹא אָמוּשׁ לְעוֹלָם / אֶמְאַס חֲמוּדוֹתַי וְדִירָה

זָרַח יְקַר אוֹרוֹ לְבָבֶל / שָׂשָׂה וְשִׁית לַיְלָה לְאוֹרָה

פָּנָה בְצֵאתוֹ חוּץ גְּבוּלָהּ / זִיוָהּ וְהוֹדָהּ וַהֲדָרָהּ

שָׂם לֵב יְדִיד שׁוֹאֵף שְׁלוֹמוֹ / מֵצֵר כְּלֵב אִשָּׁה מְצֵרָה

אֶתְפַּלְּלָה לָאֵל יְקַבֵּץ / אֻמָּה אֲשֶׁר חִנָּם מְכוּרָה

אָז אֶשְׁכְּנָה תוֹךְ בֵּית יְדִידִי / הוּא לִי לְהוֹעִיל גַּם לְעֶזְרָה.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
