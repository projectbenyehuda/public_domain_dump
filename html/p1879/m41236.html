צְבִי הַחֵן עֲרוּגַת

גם זה שלח להגביר השר ונגיד יצחק דוד ישועה גבאי ש׳ התקכ״ג

סימן: צאלח


צְבִי הַחֵן עֲרוּגַת הַבְּשָׁמִים / בְּנוֹ שׁוֹעִים וְנֶכֶד הַתְּמִימִים

אֲבִי חָכְמוֹת וּמוֹעֵצוֹת וְעֵצוֹת / וּמֵבִין כָּל דְּבָרִים הָעֲלוּמִים

לְךָ נַפְשִׁי מְאֹד תִּכְסֹף וְתַעְרֹג / כְּאַיָּל עַל אֲפִיקֵי הַזְּרָמִים

חָמַדְתִּי לִי כְנָפַיִם כְּיוֹנָה / כְּעוֹף כָּנָף כְּצִפֹּרֶת כְּרָמִים

לְעוֹפֵף בָּם לְתוֹךְ אֹהֶל יְדִיד לַ/עֲבֹר מִדְבָּר, נְהָרוֹת וַאֲגַמִּים

אֲשַׂבַּע רַעֲבוֹן נֶפֶשׁ רְעֵבָה / בְּהוֹד שִׂכְלוֹ וּפָנָיו הַנְּעִימִים

בְּהוֹד פָּנִים כְּיֹפִי בֶן זְקוּנִים / וְזוּלָתוֹ בְעֶרְכּוֹ כַצְּלָמִים

אֲדַמֶּה נָתְנוּ לוֹ זִיו בְּמַתָּן / מֵהוֹד זִיוָם כְּסִילֵי הַמְּרוֹמִים

כְּאִלּוּ הַלְּבָנָה הִיא בְמִצְחוֹ / וּבֵין עֵינָיו מְאִירִים הַתְּאוֹמִים

לְשׁוֹן זָהָב וְנִיב מָתוֹק כְּנֹפֶת / דְּבָרָיו הֵם לְכָל נִכְשָׁל מְקִימִים

בְּשֵׂכֶל דַּק וְיֹשֶׁר הַסְּבָרָה / וּמֵשִׁיב בָּהּ לְאָחוֹר הַחֲכָמִים

וּמִדַּת הַנְּדִיבוּת לוֹ סְגֻלָּה / מְשַׂמַּח בָּהּ נְפָשׁוֹת הָעֲגוּמִים

אֲבַקֵּשׁ מֵאֱלֹהִים חַי יְרִימוֹ / וְיַשְׁלִיטוֹ עֲלֵי אוֹיְבִים וְקָמִים

נְשִׂיא אֶחָיו וְנֵזֶר כָּל קְהָלוֹ / יְחִי לָעַד בְּרֹב שָׁנִים וְיָמִים.
