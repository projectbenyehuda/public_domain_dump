[יודעי מכאובי]

יוֹדְעֵי מַכְאוֹבִי, / אַל תִּתְמַהְמֶהוּ / מֵהָבִיא

צָרְיִי מִגִּלְעָד ‑ / הוּא אֵלִיָּהוּ / הַנָּבִיא!


הַעתִּירוּ בַּעְדִי / אֶל אָבִי – אוּלַי / יֵעָתֵר

וִיגַלֶּה סוֹדִי / וְחָזוֹן מַשְׂכִּילַי / יִפָּתֵר.

עַד כַּמָּה, דּוֹדִי, / מִלִּפְנוֹת אֵלַי / תִּסָּתֵר?

וּצְבִי מָאַס בִּי ‑ / מָתַי אַעְלֵהוּ / בֵּית אָבִי,

הַדּוֹד הַנּוֹעָד / לִהְיוֹת מַחְנֵהוּ / בִּלְבָבִי?


וַיְבוֹשֵׁשׁ רִכְבּוֹ, / תַּמּוּ מוֹעֲדָיו / וָאָיִן,

מֵאַיִן יָבֹא / עֵזֶר לִידִידָיו, / מֵאָיִן?

אֶצְמָא אֶל נִיבוֹ, / כִּי טוֹבִים דֹּדָיו / מִיָּיִן.

מַלְכִּי וּכְרוּבִי, / כַּסְפִּי אֶעְדֵּהוּ / וּזְהָבִי,

וּבְבֵית הַוַּעַד / יֵינִי אַשְׁקֵהוּ / וַחְלָבִי.


דּוֹדִי, זָנַחְתָּ / אוֹתִי שׁוֹמֵמָה / תּוֹךְ מוֹקְשִׁי

אָכֵן שָׁלַחְתָּ / פֶּרֶא בֶּן‑אָמָה / לַחָפְשִׁי,

וֶאְדוֹם שִׂמַּחְתָּ, / כִּי יָדוֹ רָמָה / עַל קָדְשִׁי.

שִׂמַּחְתָּ אוֹיְבִי / וַתַרְכִּיבֵהוּ / מֶרְכָּבִי,

תֵּן מָתְנָיו הַמְעֵד / עַד תַּשְׁכִּיבֵהוּ / מִשְׁכָּבִי!


הַלֵּב לֹא יֵרַךְ, / אִם הַקֵּץ אָרַךְ / וַיִּרְבֶּה,

אַךְ בָּךְ יִתְבָּרַךְ / שְׁאֵרִית עֶדְרָךְ / הַנִּשְבֶּה.

גֵר מֵאָז נֶעְרַךְ / וְכוֹכָב דָּרַךְ / לֹא יִכְבֶּה.

אִם עָבַר טובִי / וְעֵינַי כָּהוּ ‑ / כּוֹכָבִי

לֹא יִכְבֶּה לָעַד, / כִּי הַנֵּר הַהוּא / מַעְרָבִי!


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
