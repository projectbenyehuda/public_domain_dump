אהבה / ר' אברהם אבן-עזרא

מִי יִתְּנֵנִי / כִימֵי אֱלוֹהַּ יִרְצֵנִי,

כִּי יַעֲנֵנִי, / אֶקְרָא – וְאָמַר: הִנֵּנִי!

אֵיךְ חִלְּלַנִי / דּוֹדִי וּבִי גָעְלָה נַפְשׁוֹ?

אָז הֶעֱלַנִי / מִנֹּף וּבַעְנָנוֹ וְאִשּׁוֹ

אַט נִהֲלַנִי, / וָאֶפְגְּעָה בִּמְקוֹם קָדְשׁוֹ,

וַיְקַדְּשֵׁנִי / וַיַעֲמֵד צִיר יוֹרֵנִי

בֵּינֹו וּבֵינִי / וְכָל יְקָר רָאֲתָה עֵינִי.

בְּנַחֲלָתִי, / אֶרֶץ טְהוֹרָה, רָאִיתִי

כָּל שֶׁאֱלָתִי, / וּבֵית זְבוּל שָׁם בָּנִיתִי

לָאֵל מְנָתִי, / עִמּוֹ שְׁלֵוָה הָיִיתִי –

וַיְפַרְפְּרֵנִי / עָוֹן אֲשֶׁר לֹא יַרְפֵּנִי,

וַיִּשְׁכָּחֵנִי / הַצּוּר אֲשֶׁר לֹא יִנְשֵׁנִי.

רוֹעִי לְפָנִים, / אַיֵּה זְרוֹעַ תִּפְאַרְתּוֹ?

אֵין קֵץ לְשָׁנִים / מָאַס בְּצִיּוֹן עִיר שִׁבְתּוֹ.

שׁוֹמֵר אֱמוּנִים / נָתַן בְּיַד עוֹבְדֵי בִלְתּוֹ.

הֵן יַחְקְרֵנִי / אִם אֶעֲבֹד בִּלְתּוֹ שֵׁנִי,

אֵיךְ יִטְּשֵׁנִי / עַד כֹּה וְלֹא יִגְאָלֵנִי?

מִדֵּי הֱיוֹתִי / חֶסֶד וּמִשְׁפָּט אָשִׁירָה,

אֶשְׁכַּח עֱנוּתִי, / וּמִלְּבָבִי אָסִירָה

אֹרֶךְ שְׁבוּתִי, / וּבְרִית נְעוּרִים אַזְכִּירָה,

הִיא תִּצְּרֵנִי / מִצַּר וְלֹא יִבְלָעֵנִי –

וִישׁוֹבְבֵנִי / אִישִׁי וְעוֹד יֶאְהָבֵנִי.
