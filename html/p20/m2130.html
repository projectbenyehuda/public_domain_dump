סליחה

אֶל בֵּית הַמֶּלֶךְ / לָבוֹא נִקְרֵאתִי

רְעָדָה אֲחַזָתְנִי, / כִּי פִיהוּ מָרִיתִי,

אֲדֹנָי, שָׁמַעְתִּי / שִׁמְעֲךָ יָרֵאתִי!


בַּמָּה, יְחִידָתִי, / לְהִתְנַצֵּל תּוּכְלִי

יוֹם עַל כַּף מֹאזְנֵי / הַמִּשְׁפָּט תַּעֲלִי?

מַה תִּלּוֹנִי? וְאֵל / בְּיָדֵךְ שָׂם כֶּלִי

לְהֵיטִיב – וְתַרְשִׁיעִי, / בְּהֶבֶל תֶּהְבָּלִי,

בְּלִבֵּךְ לֹא תַחְשְׁבִי, / בְּשִׂכְלֵךְ לֹא תִשְׂכְּלִי

לֵאמֹר: מַה זֶּה לִי / וְלָמָּה נִבְרֵאתִי?


רָצִית לָלֶכֶת / אַחֲרֵי תַאֲוָתֵךְ,

תִּתְעַנְגִי פֹה וְלֹא / זָכַרְתְּ אַחֲרִיתֵךְ –

יוֹם אֵל יַפְרִידֵךְ / מֵעַל גְּוִיָתֵךְ

וְאֶל חֵיקֵךְ יָשִׁיב / תַּגְמוּל פְּעֻלָּתֵךְ.

כִּי בְסֵפֶר רָשׁוּם / חֲקוּקָה אַשְׁמָתֵךְ –

הַיּוֹם לְעֻמָּתֵךְ / נְתִיב תֹּם הוֹרֵיתִי.


הֵלֶךְ דְּבַשׁ רָאִית / בִּגְוִיַת אַרְיֵה,

אָכַלְתְּ וַתִּטְמְאִי / וְלֹא אָמַרְתְּ "אַיֵּה

אֱלֹהֵי הַמִּשְׁפָּט, / יוֹם יֶאֱסֹף גּוֹיֵי

הָאֲרָצוֹת לַדִּין / וְהַמֵּתִים יְחַיֶּה?"

וַתֹּאמְרִי בְלִבֵּךְ: / אוּלַי לֹא יִהְיֶה –

אֶדְרֹשׁ מַאֲוַיֵּי / לִבִּי וּבָרִיתִי.


מַרְאוֹת עֵינַיִךְ / בִּשְׁאוֹל יַשְׁלִיכוּךְ,

וְתַאֲווֹתַיִךְ לְדֶ– / רֶךְ רַע יַדְרִיכוּךְ,

לֹא יוֹעִילוּךְ, אֲבָל / לְתֹפֶת יוֹלִיכוֹךְ.

מְקוֹם גַחֲלֵי כִידוֹ– / דֵי אֵשׁ יַחְשִׁיכוֹךְ.

תִּנָּחֲמִי וְתֹאמְרִי: / אוֹי כִּי נִטְמֵאתִי!


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
