אֵל נַעְרָץ בְּסוֹד גַּלְגִּלּוֹ

אֵל נַעְרָץ בְּסוֹד גַּלְגִּלּוֹ / וּקְדוֹשָׁיו בְּיָדוֹ,

שָׁם אוֹתוֹת שְׁמוֹ עַל דִּגְלוֹ — / בָּרוּךְ שֵׁם כְּבוֹדוֹ!


בַּקֹּדֶשׁ בְּרִיאוֹת רָמוֹת / נוֹצְרוּ מֵאֲמִתּוֹ,

הִנָּם עַל תְּמוּנוֹת שׁוּמוֹת / עַל מִשְׁמָר בְּדַעְתּוֹ;

יוֹצִיאֵם בְּמִסְפַּר שֵׁמוֹת, / אִישׁ אִישׁ מִמְּלַאכְתוֹ,

יִשְׂמַח בַּעֲלוֹת גּוֹרָלוֹ, / יַכִּיר מַעֲבָדוֹ —

וּלְמַלּאֹת אֲשֶׁר יֹאמַר לוֹ / לַמּוֹעֵד יְעָדוֹ.


רָקִיע אֲשֶׁר עַל רָאשֵׁי / הַחַיּוֹת נְשָׂאוֹ

הוּא כִסְאוֹ, וְעַל פִּי משֶׁה / הַכָּבוֹד קְרָאוֹ.

יִקְרָא אֶל שְׁמוֹ, לֹא יֶחְשֶׁה / כִּי הוּא שַׂר צְבָאוֹ,

יִתְרוֹצֵץ סְבִיב הֵיכָלוֹ — / עַל כֵּן הֶעֱמִידוֹ,

וַיִּפְקֹד אֲגֻדַּת חֵילוֹ / מִן הַבָּא בְיָדוֹ.


מִי יוּכַל סְפֹר מַעְבָּדָיו / נִגְלִים אוֹ צְפוּנִים?

אֵיךְ יַגִּיד — וְאֵין בִּלְעָדָיו / בֵּין מִחוּץ וְלִפְנִים?

הַיְכַחְשׁוּ אֲשֶׁר בָּם עֵדָיו / עֵדִים נֶאֱמָנִים?

כָּל פֶּה יַעֲרֹךְ מַהְלָלוֹ, / כָּל עַיִן תְּעִידוֹ,

עֻזּוֹ בַשְּׁחָקִים וּמְלֹא / הָאָרֶץ כְּבוֹדוֹ.
