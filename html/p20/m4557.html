אמרת יחידה ליחיד

אִמְרַת יְחִידָה לְיָחִיד יָאֲתָה, // נַפְשִׁי לְצוּרָה / נִכְסְפָה וְגַם כָּלְתָה.


בַּעְמֹד פְּעָלִים בְּכֹחַ אוֹנָהּ

אוֹת עַל הֱיוֹתָהּ, וְחַיָּה יֶשְׁנָהּ,

אֹמֶן יְסוֹד לָהּ וּמִקְרֶה אֵינָהּ.

עָמְדָה תְּכוּנָה לְגֵו אוֹ בָלְתָה – // חַיִּים מְקוֹרָהּ, / בִּנְוֵה אֱמֶת הָיְתָה.


רַבּוֹת תְּמוּנוֹת בְּגֵו נִרְכָּבוֹת

יַחַד, עֲרֵבוֹת בְּקִרְבָּהּ שָׁבוֹת.

עֵינָהּ כְּשֶׁמֶשׁ בְּאֶפֶס עָבוֹת,

הֵיכָל מְתֹאָר לְשִׁבְתָּהּ בָּנְתָה, // כִּי מִבְּשָׂרָהּ / הוֹד מְחוֹלְלָהּ חָזְתָה.


הַכֹּל בְּלִי עֵת בְּשִׂכְלָהּ רוֹאָה,

טֶרֶם בְּכֹחָהּ תְּהִי נִמְצָאָה

דֵעָה, וְאַחַר לְפֹעַל יָצָאָה.

תִּשָּׂא – וְהִיא כַנְּשׂוּאָה דָמְתָה, // עַד יַאֲדִירָהּ / אֵל לְרַחֲמָיו פָּנְתָה.


מָעוֹן, בְּךָ נִתְכְּנָה קִימָתִי,

אַיִן יְרִיבִי וְאַתְּ חוֹמָתִי,

אָשִׁית לְמָקוֹם לְךָ נִשְׁמָתִי.

אִם תַּחֲרִישׁ מִקְּרֹא לָךְ – מֵתָה // אָכֵן יְעִירָהּ/ זִכְרְךָ וּבוֹ חָיְתָה.
