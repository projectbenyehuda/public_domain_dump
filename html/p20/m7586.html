אֹהֶל רַחֲמֶיךָ נְטִיתוֹ

[אהבה]

אֹהֶל רַחֲמֶיךָ / נְטִיתוֹ לְמִכְסֶה

עַל עַם קִדְּמֶךָ / בְּסִינַי בְּנַעְשֶׂה,

יִקְרָא אֵל שְׁמֶךָ / כִּי נִפְזַר כְּמוֹ שֶׂה –

שׁוֹבְבָה / אֶל כָּבוֹד וְעֶדְנָה בַּת אֲהוּבָה.

בַּת נָדִיב בִּתְהִלּוֹת / כָּל לָשׁוֹן, בְּחוּנָה –

אֵי אַהְבַת כְּלוּלוֹת / וּבְרִית הָאֱמוּנָה?

נֶחְלֶקֶת גְּבוּלוֹת – / וּבְכָל עַם תְּמוּנָה

נִצְּבָה / וּלְךָ לְמָנָה אָהֳלִיבָה.

רַמְתָּ מִמְּעוֹנִי, / סָר כָּל מַחְמַדִּי,

עַל מוֹגַי וּמוֹנַי / אֵיךְ אֶשָּׂא לְבַדִּי?

כִּי מַלְכוּת שְׁכֵנַי – / כְּאָחִי כְּעַבְדִּי –

רָחֲבָה / כֹּה וָכֹה, וְאָנָה עוֹד אֲנִי בָא?

מֶה עָצְמוּ אֲשָׁמַי / לֵאמֹר צַר: יְכָלְתִּיו,

אַךְ עֵינַי לְשָׁמַי / עַד קֵץ דּוֹר מְשַׁכְתִּיו.

אָשׁוּב לַעֲלוּמַי / וּבְחֵיק דּוֹד אֲהַבְתִּיו

אֶשְׁכְּבָה, / מַה יָּפִית וּמַה נָּעַמְתְּ אַהֲבָה.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
