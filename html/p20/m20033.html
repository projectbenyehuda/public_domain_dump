אֵל הֶעָבִים דָּר בְּרוֹמָה

פיוט לשמיני עצרת


אֵל הֶעָבִים, דָּר בְּרוֹמָה, / צַו: הָרִיקוּ רְבִיבֵיכֶם,

טַל וּמָטָר, לָאֲדָמָה / בֹּאוּ בְּשָׁלוֹם שְׁנֵיכֶם!


בְּחַסְדְּךָ, דָּר עֲרָבוֹת, / שַׂמַּח בְּגִיל הַלְּבָבוֹת,

תֵּן טַל וּמָטָר לְהַשְׁתּוֹת / אֶרֶץ וְעוֹרֵר אַהֲבוֹת,

בָּרֵךְ תְּבָרֵךְ תְּבוּאוֹת / וְתִמָּלֵא תֵבֵל תְּנוּבוֹת.

רַחוּם, לְנִינֵי אֲיֻמָּה / בַּשֵּׂר: שָׁמַעְתִּי בְּקוֹלְכֶם.


הַמּוֹרִיד שִׁפְעַת רְבִיבִים / דָּבָר מִמֶּנּוּ לֹא נִכְחַד,

מְכַסֶּה שַׁחַק בֶּעָבִים, / תֵּן טַל וּמָטָר בְּיַחַד

שְׁנֵיהֶם לָאָרֶץ, טוֹבִים / הַשְּׁנַיִם מִן הָאֶחָד,

מוֹצִיא לָאוֹר תַּעֲלֻמָה / יִפְתַּח אֲרֻבּוֹת שְׁמֵיכֶם.


הָאֵל, הַנּוֹתֵן לְכָל בָּשָׂר / לֶחֶם, לְכָל דַּל וְאֶבְיוֹן,

דָּגָן וְתִירוֹשׁ וְיִצְהָר / בָּרֵךְ, מְרוֹמָם וְעֶלְיוֹן,

תְּמַלֵּא הַגְּרָנוֹת בָּר / בִּבְרָכָה, שׁוֹכֵן וְעֶלְיוֹן,

חַנּוּן בְּרִית הַקְדוּמָה / יִזְכֹּר בְּרַחֲמָיו אֲלֵיכֶם.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
