שם אלי מנת גורלי

[שם אלי מנת גורלי] / ר' אברהם אבן-עזרא

שָׁם אֵלִי מְנַת גּוֹרָלִי, / מִגְדָּלִי וְחֵילִי.

כִּי הוּא יַעֲשֶׂה שָׁלוֹם לִי,/ שָׁלוֹם יַעֲשֶׂה לִי.

אֶחָד אֵין לְפָנָיו אֶחָד, / גַּם כֵּן אַחֲרָיו לֹא

וּבְאֶחָד לְבַד הִתְאֶחָד / כָּל הֵפִיק וְכִלּוֹ,

יָעִיד עַל אֲשֶׁר לֹא נִכְחַד / מַעְגָּלוֹ וְדִגְלוֹ,

וּבְכָל מַעֲשָׂיו הוּא פֶלִאי, / הוּא מַרְפֵּא לְחֹלִי,

הוּא יִפְעַל בְּאֶפֶס כֶּלִי / בֶּאֱמֶת, לֹא בְּשֶׁלִי.

בִּי אוֹתוֹת אֱלֹהַי אֶרְאֶה, / אֵין עוֹד מִלְּבַדּוֹ,

הוּא עִמִּי וְאִם אֶשְׁתָּאֶה / כָּל רֶגַע לְחַסְדּוֹ,

לוֹ אֶכְרַע וּבוֹ אֶתְגָּאֶה, / כִּי רוֹצֶה בְעַבְדּוֹ;

הוּא עַל יַד יְמִינִי דִגְלִי, / הוּא גַם עַל שְׂמָאלִי,

הוּא הָרִי בְעִתּוֹת שִׁפְלִי, / הוּא מִסְעָד בְּנָפְלִי.

רוּחוֹ נוֹסְסָה בִלְבָבִי / וּבְכָל לֵב יְרֵאתִיו,

וּבְשֵׁם לִמְּדַנִי נָבִיא / לִי שָׁלַח – קְרָאתִיו,

רָחוֹק מִזְּבוּל מַחְשָׁבִי / אַךְ קָרוֹב מְצָאתִיו;

הוּא עֶזְרִי וְהוּא גוֹרָלִי, / הוּא חֶלְקִי וְחֶבְלִי,

הוּא מָנוֹס בְּיוֹם צָרָה לִי, / הוּא נֵרִי בְאָפְלִי.

מָה אֹמַר וְהַכֹּל יָדַע / טֶרֶם שֶׁאֲדַבֵּר?

אַךְ מִפִּי שְׁמוֹ לֹא אֶגְדַּע, / כִּי אֵלָיו אֲשַׂבֵּר,

נַפְשִׁי מִשְּׁאוֹל הוּא יִפְדַּע / וַחְיָלִים אֲגַבֵּר;

בּוֹ אָגִיל וְאֶשְׂמַח כֻּלִּי, / בּוֹ כָּל מַהֲלָלִי,

בּוֹ יִגְבַּהּ לְמַעְלָה קוֹלִי, / בּוֹ יָרֹן חֲלִילִי.
