אל צור קדושי (מיושב לראש השנה)

אֶל צוּר קְדוֹשִׁי / אֶשָּׂא נַפְשִׁי;

הַאֲזֵן לַחְשִׁי / וְהַעֲבֵר כַּחְשִׁי,

אֵל קְדוֹשִׁי, / כְּבוֹדִי וּמֵרִים רֹאשִׁי!


בְּמַעֲמָד וּמוֹשָׁב / שְׁעֵה גֵּר תּוֹשָׁב,

בְּכָל לֵבָב שָׁב / לָךְ, דֹם יָשָׁב;

מְהֵרָה יוּשָׁב / לְעִיר קְדוֹשִׁי!


הַעֲלֵה, צוּרִי, / צֳרִי לִמְזוֹרִי

וְנַהֵל עֶדְרִי / לְצִיּוֹן עִירִי,

עֲבוּר ׳מַהֲרִי / שְׁלֹשׁ סְאִים וְלוּשִׁי׳.


מְסוּרִים בִּידֵי / צוֹרֵר רוֹדֶה

חוּשָׁה וּפְדֵה; / וְסוּת יֶשַׁע עֲדֵה

לְבַשְּׂרָהּ: ׳בִּגְדֵי / יֶשַׁע לִבְשִׁי!׳


חָזוּת ׳וּלְקָחוּם / אֲשְֶׁר הִדִּיחוּם׳

תְּמַהֵר, רַחוּם, / לְדַלִּים נָחוּם,

וְחֲזוֹן נַחוּם / הָאֶלְקוֹשִׁי.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
