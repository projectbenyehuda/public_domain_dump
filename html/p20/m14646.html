הַבּוֹחֵר בְּשִׁירֵי זִמְרָה

הַבּוֹחֵר בְּשִׁירֵי זִמְרָה / אֶקְרָא, / יִתְגַּדֵּל שְׁמוֹ הַנּוֹרָא!


אֶתְבּוֹנֵן – / וּבַכֹּל שָׁפָר

וַאְרַנֵּן / לְאֵל כֹּל סָפָר,

וַיְכוֹנֵן / אֱנוֹשׁ מֵעָפָר

וּבְלִבּוֹ נְשָׁמָה בָרָה / בָּרָא, / כַּמֶּלֶךְ בְּתוֹךְ הַבִּירָה.


בַּכֹּל הוּא / וּמִיָּדוֹ כֹל,

כָּמֹהוּ / הֱיוֹת לֹא יָכֹל,

כַּתֹּהוּ / לְנֶגְדּוֹ הוּא כֹל!

מָה אֹמַר – וְיַד כָּל נִבְרָא / קָצְרָה, / דַּעְתּוֹ מֵאֱנוֹשׁ נִבְצָרָה?


רָם פִּלְאוֹ / וְסוֹד עֶלְיוֹנָיו,

עִם כִּסְאוֹ / מְשָׁרְתֵי פָנָיו,

מוֹרָאוֹ / עֲלֵי אוֹפַנָּיו,

בַּמַּיִם עֲלִיוֹת קֵרָה, / קָרָא – / שָׁמַיִם בְּרוּחוֹ שִׁפְרָּה.


מוֹרָאִים / בְּכָל מַעְבָּדוֹ,

כִּי נִרְאִים / מְעִידֵי הוֹדוֹ.

לִנְבִיאִים / יְגַלֶּה סוֹדוֹ,

כִּי נֶפֶשׁ בְּעֵינָיו יָקְרָה / הוֹרָה / לָלֶכֶת בְּדֶרֶךְ יְשָׁרָה.
