אָהֳלִיבָה לְאָהֳלָה כָּתְבָה

גאולה (לט׳ באב?)


אָהֳלִיבָה לְאָהֳלָה / כָּתְבָה עֲלֵי מְגִלָּה,

אָהֳלָה לְאָהֳלִיבָה / עַל כְּתָבָהּ מְשִׁיבָה.


׳אָן הַיּוֹם מְגוּרֵךְ? / אֶשֹּׁם בַּעֲבוּרֵךְ,

כִּי חָרַב דְּבִירֵךְ, / וְאוֹחִיל לְצִירֵךְ

בְּכָל מַסְלוּל וָדֶרֶךְ, / אוּלַי עוֹד אֲשׁוּרֵךְ

לָדַעַת דְּבָרֵךְ

אֲחוֹתִי הַגְּדוֹלָה.׳


׳בְּגוֹזָן אֲנִי עֲכוּרָה, / גְּזוּזָה כְּשֶׂה, פְּזוּרָה,

לִי בְחָבוֹר חַבּוּרָה, / אָבְדָה הַחֲבוּרָה;

אֲחוֹתִי הַצְּעִירָה, / הַעוֹדֵךְ גְּבִירָה

וְאַתְּ בְּבֵית הַבְּחִירָה,

יִזָּכֵר לְטוֹבָה?׳


׳רָעוֹת בְּכָל סְבִיבוֹת / מַקְהֲלוֹתַי מְרִיבוֹת,

וְכָל עָרַי עֲזוּבוֹת / וּמִשְׁכְּנוֹתַי חֲרֵבוֹת,

אַךְ נְבוּאוֹת כְּתוּבוֹת / נִחֲמוּ הַלְּבָבוֹת,

כִּי לְעִתִּים קְרוֹבוֹת

נָשׁוּב כְּבַתְּחִלָּה.׳


׳הֲכָתְבוּ נְבוּאוֹתָם / נְבִיאִים בְּשַׁלְוָתָם,

אוֹ אַחַר גָּלוּתָם / זָכְרוּ בְמַחֲזוֹתָם

אוֹתִי? בְּבוֹא פְּדוּתָם / שִׁלְחִי לִי מִשְׁמוֹתָם,

אוֹ מִשְׁמוֹת קְצוֹתָם

לִי מְגִלָּה כְתוּבָה.׳


׳מַרְאוֹת נָגָהוּ / מִפִּי יְשַׁעְיָהוּ,

יְחֶזְקֵאל, יִרְמְיָהוּ, / חֲבַקּוּק, צְפַנְיָהוּ,

חַגַּי, זְכַרְיָהוּ / וְדָנִיֵּאל – הֲלֹא הוּא

הַקֵּץ יְדָעָהוּ,

הוּא קֵץ הַגְּאוּלָה.׳


׳מוֹעֲדֵי קֵץ וְסוֹדָם / אִם נוֹדְעוּ לְאָדָם –

יוֹאֵל לְהַגִּידָם, / כִּי הֲמוֹנַי בְּנוּדָם

אֵין כֹּחַ בְּיָדָם, / וְשָׁכַב וְנִרְדָּם;

וְעֵת יָשׁוּב כְּבוֹדָם –

אָן הֲמוֹנָם יְהִי בָא?׳


׳שָׂמְחוּ רַעֲיוֹנַי / וּבֹשׁוּ כָל שְׁכֵנָי,

כִּי יְשׁוּעַת אֲדֹנָי / בָּאָה לַהֲמוֹנָי,

וְאוֹתוֹת חֶזְיוֹנַי / הֵן נִרְאוּ לְעֵינָי,

וְקוֹל קוֹרֵא בְאָזְנָי:

סֹלוּ הַמְּסִלָּה!׳


׳בְּרוּכָה אַתְּ, אֲחוֹתִי, / כִּי נִחַמְתְּ אוֹתִי,

כִּי תַם קֵץ שְׁבוּתִי / וּבָאָה עֵת פְּדוּתִי,

וְקָמָה מַלְכוּתִי / בְּיָדֵךְ, רַעְיָתִי;

וְאַתְּ, בַּעֲלַת בְּרִיתִי,

הֲיִי לְאַלְפֵי רְבָבָה!׳

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
