אמונים עומדים נגדך(פזמון)

אֱמוּנִים עוֹמְדִים נֶגְדְּךָ / בְּפַחַד וּמוֹרָא וְחִילָה,

נָא רַחֵם, צוּר, עֲלֵיהֶם / וְאַל תַּעֲשֶׂה בָם כָּלָה.

הוֹאֶל נָא וּבָרֵךְ, / עוֹטֶה מַעֲטֵה תְהִלָּה,

הָאֲנָשִׁים אֲשֶׁר בָּאוּ / אֵלֶיךָ הַלָּיְלָה.


בְּשִׁמְךָ בָטְחוּ וְחָסוּ – / כִּי לֹא בִגְבוּרָתָם,

וּבִמְקוֹם מִנְחָה שָׁפְכוּ / לְפָנֶיךָ דִּמְעָתָם,

וּסְלִיחָתְךָ דָּרְשׁוּ / לִמְחִילָה לְחַטָּאתָם.

חַדֶּשׁ נָא פְּלָאֶיךָ / וְהָחִישׁ יוֹם יְשׁוּעָתָם –

וּבָאִים בֵּיתְךָ בְּבֶכִי / יֵצְאוּ מֶנּוּ בְּגִילָה.


רְצֵה נָא אֶת דִּבְרֵיהֶם / וְאַל נָא תְשִׁיבֵם רֵיקָם

וּכְפָעֳלָם אַל תָּמֹד וְלֹא / כְרִשְׁעָתָם אֶל חֵיקָם;

וְהָאֵר אָפְלָם וְיֵצֵא / חִישׁ כַּנֹּגַהּ צִדְקָם,

וְאוֹיְבֵיהֶם מְעַנֵּיהֶם – / הֵמָּה יִקְחוּ חֶלְקָם,

וְעָם הָיוּ לַחֲרָפוֹת / יִהְיוּ לְשֵׁם וְלִתְהִלָּה.


מוֹדִים עַל חֲסָדֶיךָ / וּלְרַחֲמֶיךָ יְשַׂבֵּרוּ,

נְדִיבֵי עָם חֲצוֹת לַיְלָה / תְּהִלָּתֶךָ יְסַפֵּרוּ,

וּבְתַחֲנוּנִים נִגְּשׁוּ / וְנִיב רְנָנוֹת יְדַבֵּרוּ –

תְּהִי אָזְנְךָ קַשֶּׁבֶת / לִשְׁמֹעַ הַתְּפִלָּה.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
