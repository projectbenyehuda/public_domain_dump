אָבִי רֶכֶב יִשְׂרָאֵל

אהבה


אָבִי, רֶכֶב יִשְׂרָאֵל, / שָׂר אֶל מַלְאָךְ, רָד עִם אֵל –

נָפַל כָּרַע / מֵרַע / מֵרַע / מִסְתּוֹלֵל,

כִּסְאוֹ מִגֵּר, / יַגֵּר / חוֹגֵר / מִתְהַלֵּל.


בֵּן יַקִּיר מֵחֵיק נִגְרָשׁ, / גַּם בֶּן הָאָמָה יָרַשׁ,

יַשְׂכִִּיל עַבְדִּי / נֶגְדִּי, / בַּדִּי / בַּל שֹׁרַשׁ,

מַשְׁחִית הוֹדִי / רוֹדִי; / דּוֹדִי, / בַּל תֶּחְרָשׁ!


רַב לִמְשַׂנְאִי יִתְנַשָּׂא, / יִלְעַג – וּלְשׁוּר וּלְמַשָּׂא

גִּנַּת בִּיתָן / אֵיתָן / נָתַן / לִמְשִׁסָּה;

רוּחַ יִבְצֹר, / יַעְצֹר, / יִצֹּר / בַּמַּשָּׂא.


הָשֵׁב לַמִּזְרָח לַצְּבִי / בַּת נָדִיב מִשְּׁבִי,

וּצְבִי מֵעַם לוֹעֵז / הָעֵז, / צוּר, עֶזְ-/רָתִי בִי,

מִבֵּית כֶּלֶא / עוֹלֶה / אֶעְלֶה / אֶל אָבִי.


׳מוֹשִׁיעֵךְ הִנֵּה / זֶה בָּא, / בִּתִּי, יִרְתֹּם מֶרְכָּבָה,

עֵדֶן נַחְלִי / יַחְלִי, / בַּחֲלִי / פָז צָבָא,

יִקַּח רֵיחוֹ / רִקְחוֹ / לֵב חוֹ-/לַת אַהֲבָה.׳

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
