מאורה / ר' אברהם אבן-עזרא

אֲנִי בְאוֹנִי אֶצְעֲקָה לְדוֹד יִבְזֶה

לְקוֹל שְׁאוֹנִי, / יַחֲרִישׁ – וְאֵיךְ יֶחֱזֶה

עֵינִי בְעֵינִי / תִּגָּר? וְהוּא כְּצַר יַזֶּה

נִצְחִי, וְאוֹנִי / יַעֲנֶה בְפָנַי: זֶה

עֵקֶב אֲשֶׂר אֶשְׁכָּחֶךָ! / אַשְׁרֵי אֲשֶׁר שָׂכְלוּ.

מַה יַּעֲנוּ מוֹאֲסֵי חָכְמָה / יְרַדְּפוּ מְתִים סָכְלוּ?

בְּנֵי בְלִימָה / גִּדְּפוּ חֲסִידֶיךָ,

אֲשֶׁר מְאוּמָה / אֵין עֲדֶן בְּיָדֶיךָ.

קוּמָה וְרוּמָה / עַל שְׁמֵי כְבוֹדֶךָ,

וְיֶחֱזוּ מַה / נָּחֲלוּ עֲבָדֶיךָ

הַשּׁוֹאֲפִים לְרוּחֶךָ, / יִחֲלוּ – וְאִם נֶחְלוּ,

רַק יַחְדְּלוּ לַעֲנוֹת: עַל מָה / בְּבֵית הַשְּׁבִי הָגְלוּ.

רֵאשִׁית תְּבוּאָה / הֵן שְׁמִי, וְאֵל עֵדִי,

רוּחַ נְכֵאָה / בִי – וְהִיא לְבַד סוֹדִי.

נְכֵאָה וְנִלְאָה / כּוֹנֵן, וְיַעֲרֹץ מִדִּי.

כִּי אָח גָּאָה / עָלַי – וְאֵין לְאֵל יָדִי.

רַק אֶבְטְחָה בְכֹחֶךָ; / עַד יְצַו פְּדוּת כָּלוּ

עינַי לְךָ, אֵיךְ, גְּדֹל עָצְמָה, / מְשִׂיחִים בְּךָ דַּלּוּ?

מְרִי הֲמוֹנָךְ / יַעֲלִים פְּלָאֶיךָ,

הֲיַד יְמִינָךְ / קָצְרָה וְתַלְאֶךָ?

בְּנָךְ לְעֵינָךְ / נָתוּן בְּיַד מְשַׂנְּאֶיךָ.

הֲדַר גְּאוֹנָךְ / שׁוֹבֲבָה לְכִסְאֶךָ.

תָּקִים, בְּבוֹא מְשִׁיחֶךָ, / מִגְדְּלֵי יְקָר נָפְלוּ,

מַהֵר, שִׁית כְּאוֹר הַחַמָּה / מְאוֹרִים אֲשֶׁר אָפְלוּ!
