אִמְרָתְךָ אֵלִי הָקֵם לִשְׁבוּיָה

אִמְרָתְךָ, אֵלִי, / הָקֵם לִשְׁבוּיָה / כִּלְבָבִי:

‘הִנְנִי שׁוֹלֵחַ / לָכֶם אֵלִיָּה / הַנָּבִיא!’


בַּת צִיּוֹן אֵיכָה / בֵּינוֹת אַכְזָרִים / נֻטָּשָׁה?

אוֹ אֵיךְ רַעְיָתְךָ / מֵחֲדַר הוֹרִים / גֹּרָשָׁה?

וּנְוֵה קָדְשֶׁךָ / אֵיךְ לִבְנֵי זָרִים / מוֹרָשָׁה?

יָרְשׁוּ אֶת אָהֳלִי / מִקֶּדֶם הָיָה / מוֹשָׁבִי —

וַאֲנִי בוֹרֵחַ / אֶרֶץ נָכְרִיָּה / בֵּין אוֹיְבִי.


רֹאשִׁי הָרִימָה, / כִּי לְךָ מֵאַהֲבָה / מִשְׁתָּאֶה,

וּרְאֵה בֶן אָמָה / עַל בֶּן הָאֲהוּבָה / מִתְגָּאֶה.

יָמָּה וָקֵדְמָה / נִשְׁבַּר אוֹ נִשְׁבָּה, / אֵין רוֹאֶה.

הִכְבִּידוּ עֻלִּי / בּוֹזֵי תוּשִׁיָּה, / יִרְדּוּ בִי —

וַאֲנִי צוֹרֵחַ, נַפְשִׁי בּוֹכִיָּה / אֶל אָבִי.


'מֵרוֹם שָׁמַעְתִּי / קוֹלֵךְ, בַּת עַמִּי, / וָאֶעֱנֶה:

יִשְׁעִי קֵרַבְתִּי / לִפְדּוֹתֵךְ לִשְׁמִי, / וָאֶקְנֶה

אֶת בֵּית תִּפְאַרְתִּי / הָרַס אִישׁ חֶרְמִי / אֵין בּוֹנֶה.

לִמְקוֹם הֵיכָלִי / מִקַּצְוֵי נְשִׁיָּה / תָשׁוּבִי,

וּבְלֵב שָׂמֵחַ / הַר הַמּוֹרִיָּה / תִקְרָבִי!'
