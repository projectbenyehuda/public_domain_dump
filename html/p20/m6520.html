אֵיךְ עֵת פְּדוּת יְאַחֵר

אֵיךְ עֵת פְּדוּת יְאַחֵר / הָאֵל הַבּוֹחֵר / בְּעַמּוֹ?

בּוֹגֵד הֱיֵה עוֹקֵר / וְיַלְדְּךָ יַקֵּר / תַּעֲנוּג כְּמוֹ שִׁקֵּר

אֵל בֵּיתְךָ מְשַׁחֵר / וּרְצוֹנְךָ שִׁחֵר / בְּקוּמוֹ.

רַבֵּה צְבָא גִילִי, / אֵלִי, בְּחַכְלִילִי; / וְכֹהֵן בְּבֵית זְבוּלִי

תָּמִיד יְהִי סוֹחֵר / בְּמֶרְקָחוֹת סוֹחֵר / וּבָשְׂמוֹ.

מִלֵּב בְּנֵי רַהַב / יִכְבֶּה אֵשׁ לַהַב, / כִּי אוֹהֲבִי אָהַב

קִצִּי; וְאֵיךְ יוֹחֵר – וּבְנִי בְחֵיק אַחֵר / וְאִמּוֹ?
