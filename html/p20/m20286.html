אֵל אֵין שֵׁנִי

סליחה


אֵל אֵין [שֵׁנִי], מַעֲנֵה הֶגְיוֹן / שִׂפְתוֹתַי הַקְשֵׁב,

[….]ני עֶלְיוֹן / רֵיקָם אַל תָּשֵׁב,

וְאוֹתִי בְצִיּוֹן / [……] הוֹשֵׁב.

בְּצִדְקַת אֲבוֹתַי / תְכַפֵּר חוֹבוֹתַי

וְתָסִיר עַצְּ[בוֹ]תַי / ומ[ס…..ודוֹתי / [….] בִגְלָלָךְ –

רְפָאָה נַפְשִׁי, כִּי חָטָאתִי לָךְ.


תַּבִּיט בְּרָעַת עַבְדְּךָ […] וְתִשְׁמַע שוע[…] קוֹלִי

אֵל עֶלְיוֹן, וְתָחֹן דמע [……..לי]

[……..] לעולי

וְעֹצֶם שְׁגִיּוֹתַי / שני א[…]ם חֲסִידִים

[….תי / ….תי אלה [….] לך


בִּרְכוֹת אֱמוּנַי / שני[….] חֲסִידִים,

שְׁמַע תַּחְנוּנַי, / אֵל, [וְהוֹ]שִׁיב יְחִידִים,

וְ[חוּס] עַל [עַ]מִּים אֲשֶׁר אוֹתְךָ עוֹבְדִים;

לְבֵיתְךָ יִקָּווּ / וּבִדְבָרִים נָאווּ

גְּדֻלָּתְךָ יְחַוּוּ, / יַחַד יִשְׁתַּחֲווּ / לָךְ וִיזַמְּרוּ לָךְ.


רוּחִי וְנַפְשִׁי / אֶהֱרֹג עַל דָּתָךְ

וּבְקוֹל רַחְשִׁי / אֶצְעַק מַר ב[…ת]ך

עֲנֵנִי, קְדוֹשִׁי, / לְמַעַן תּוֹרָתָךְ,

וְהַמְצִיא לִי [חֲ]נִינָה / [וְדִי]נִי הָבִינָה,

וַהֲגִיגִי בִינָה / ומ[….] הַאֲזִינָה / קוֹלִי בְּקָרְאִי לָךְ.


הָאֵל הָרֵם / ת[…] כָּל מ[…]ינַי

וּמְחֵה זֶרֶם / דְּמָעַי מ[…ני]

מַהֵר בְּטֶרֶם / […]ות […]ל בָּנַי

וְעַל עַבְדְּךָ חוּסָה, / אֱלֹ[הַּ] רָם וְנִשָּׂא,

[……]כסה / וְלָנוּ כָּל מַשָּׂא / [אשר…..] לָךְ.
