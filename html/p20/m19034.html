אֵלִי אֵלִיָּה הַנָּבִיא הָבֵא נָא

(הבדלה)

אֵלִי אֵלִיָּה הַנָּבִיא הָבֵא נָא!


בּוֹ יִרְתֹּם רִכְבּוֹ / נָע בַּשְּׁבִי כִּי בוֹ

לֹא שָׁכַב לִבּוֹ, / גַּם לֹא רָאָה שֵׁנָה.

רַב מַחְלִי בִּרְאוֹת / כַּחְשִׁי – וּמְשַׂנְאוֹת

יָפוֹת וּבְרִיאֹת / בָּשָׂר, וַתִּרְעֶינָה.

הַשְׁקֵה, צוּר, מֵי רֹאשׁ / צַר עֵינָיו יִלְטֹשׁ,

יוֹם עֵינַי לִקְדוֹשׁ / יִשְׂרָאֵל תִּשְׁעֶינָה.

מָתַי תַּרְאֶה אוֹת / יֵשַׁע אֶל קוֹרְאוֹת

לָךְ, וּלְךָ נוֹשְׂאוֹת / קוֹלָן וַתִּבְכֶּינָה?

מַלְאַךְ הַגּוֹאֵל / לִפְנֵי דַל שׁוֹאֵל

אָנָּא, הָאֵל, אֵל / אַבְרָהָם, הַקְרֵה נָא!

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
