[כְּמוֹ זֹאת]

וְאָמַר בְּשׁוּבוֹ גוֹבֵר וּמְנַצֵּח מְהַמִּלְחָמָה הַהִיא, וְהָיְתָה בַחֹרֶף שֶׁל שְׁנַת תת"ו [4806]:


כְּמוֹ זֹאת תְּהִי חִידָה מְמֻשֶּׁלֶת / וְכָכָה תְּהִי גֵאוּת וּמֶמְשֶׁלֶת:

יְחַנֵּן בְּמִלּוּלוֹ – וְהוּא לֹא רָשׁ / וְנִקְלֶה, וְהִיא – גֵּאוּת מְמַלֶּלֶת

חֲכָמָה בְּצֵיד לִבּוֹת – וּמִי לִמֵּד / פְּעֻלַּת לְכֹד לִבּוֹת לְעַצֶּלֶת?

נְדִיבָה, נְבָלָה לֹא מְדַבֶּרֶת, / וְלֹא בוּז לְהוֹרֶיהָ מְגַלְגֶּלֶת,

בְּחוּרַת בְּנֵי צָדוֹק, בְּרִית רֵעִים / תְּחַלֵּל – וְהָאָב לֹֹא מְחַלֶּלֶת,

וְשׁוֹעָה, וְאוּלָם לֹא בְּיָפְיָה אוֹ / בְּלֶחֶם כְּהֻנָּה זָר מְכַלְכֶּלֶת,

וְתֵבוֹשׁ לְהָשִׁיב ‘לֹא’ וְלָכֵן אִם / תְּאֻלַּץ – יְמוֹתֶיהָ מְקַלֶּלֶת,

כְּבוּדָה פְּנִימִיָה, דְּבַר צַדִּיק / תְּסַלֵּף – וְשֹׁחַד לֹא מְקַבֶּלֶת,

חֲלָקָה כְּמוֹ שֶׁמֶן, וְנִמְתֶקֶת / כְּנֹפֶת, וְנִבְחָרָה כְּמוֹ סֹלֶת,

מְאִירָה וְנִשְׁקָפָה כְּמוֹ שַׁחַר, / אֲיֻמָּה וְנוֹרָאָה כְּנִדְגֶּלֶת,

וּבָרָה כְּמִשְׁפַּחְתִּי, בְּאֵין יוֹצֵאת / וּפֶרֶץ, בְּמִשְׁפָּחוֹת מְסֻבֶּלֶת,

וּמִי נֶחֱצַב מִצּוּר קְהָת אוֹ בוֹר / מְרָרִי, וּמִרְיָם לוֹ מְחוֹלֶלֶת,

וְצָעַד כְּמוֹתָם עַל מְרוֹמֵי שׁוּר / וְחוֹמָה, וְלֹא צָעַד עֲלֵי דֶלֶת,

וְרֻבָּה עֲלֵי יִרְאָה, וְהָנְחַל חֹק / אֱלֹהָיו, וְלֹא נַחְלָה מְבֹהֶלֶת,

וְגֻדַּל עֲלֵי תוֹרָה וְעַל מִשְׂרָה, / וּמָאַס בְּנַלוֹז אוֹ פְּתַלְתֹּלֶת –

תְּהִי לוֹ וְלִשְׁאֵרוֹ עֲדַת צָרָיו / וְחוֹמְדָיו בְּעַל כָּרְחָה מְהַלֶּלֶת.

וּמִי יַעֲלִים שַׁחַר בְּיָדוֹ, אוֹ / יְשַׁוֶּה יְשָׁרָה כַּעֲקַלְקֹלֶת?

וּמִי יַעֲנֶה כִּי אֵין לְשֶׁמֶש אוֹר, / וְרֵיחַ לְקִנָּמוֹן וְלִשְׁחֵלֶת?

יְהוֹסֵף בְּנִי, אַל תַּהֲרֹס בִּנְיָן / יְסַדְתִּיו בְּקַו יֹשֶׁר וּמִשְׁקֹלֶת,

וְקוּמָה עֲלֵי כַנִּי, וְאֵחָשֵׁב / כְּדָוִד, וְתֵחָשֵׁב כְּקֹהֶלֶת,

לְמַעַן בְּקוּם אִישִׁים, בְּנֵי בוֹנִים, / וְעוֹשִׁים כְּמוֹ עֵדָה מְחַבֶּלֶת,

וּבָזִים לְמוּסַר אָב נְבוֹן דָּבָר, / וְדַת אֵם מְחֻכָּמָה וּמַשְׂכֶּלֶת –

אֲדָמָה בְּמַר תִּצְרַח, וְהָאָרֶץ / תְּקוֹנֵן כְּמוֹ אִשָּׁה מְשַׂכֶּלֶת;

וְעֵת שׂוּם בְּנֵי אָבוֹת מְבִינִים אֶת / תְּבוּנָה לְנַפְשָׁם חֹק וּמַכֹּלֶת,

כְּמַעְשֵׂה בְּנֵי מָחוֹל וְלֹא כִבְנֵי / אֱלִיאָב וּבֶן-יִצְהָר וּבֶן-פֶּלֶת –

אֲדָמָה אֲזַי תִּפְרַח, וְיַעַל צִיץ / וְגוֹמֵל בְּנָכְריָּה וְנוֹבֶלֶת.

אֲלֹהַי לְךָ אֵיטִיב דְּבַר שִׁירָה / כְּפוּלָה עֲלֵי טוֹבָה מְכֻפֶּלֶת.

וְאֶשָׂא וְאָרִים קוֹל וְאֹמַר: אַתְּ / מְרוֹמָם בְּמַמְלָכָה וּבִיכֹלֶת,

לְמַעַן הֲרִיקוֹתָ וְהִפָּלְתָּ / לְפָנַי עֲדַת זָרָה וְנוֹכֶלֶת,

אֲשֶׁר דִּבְּרָה: אָנִי וְאַפְסִי עוֹד! / מְפִיצָה וְהוֹרֶגֶת וְשׁוֹלֶלֶת,

וּבָנָה, בְּבוֹאֵנוּ בְּאַרְצָהּ, כִּי / אֲשֶׁר נַעֲשָׂה בָהּ לֹא מְכַלְכֶּלֶת,

וְנִלְאֵית רְאֹות שׁוֹלָל וְאִישׁ חָלָל, / וְשִׂמְלָה בְּדַם עַמָּהּ מְגוֹלֶלֶת,

וְחוֹמָה מְפֹרָצָה, וְאַרְמוֹן רָם / מְנֻתָּץ, וְהָאָרֶץ מְחֻבֶּלֶת.

וְהָיָה שְׁמָהּ: רֹאשָׁה וּמוֹשֶׁלֶת – / וְהַיּוֹם שְׁמָהּ: רָשָׁה וְכוֹשֶׁלֶת.

שְׂרוּפָה – וְהִיא לֹא דִמְּתָה כָזֹאת / לְפָנִים, וְלֹא הָיְתָה מְפַלֶּלֶת.

בְּלוּעָה אֲכוּלָה, אַחֲרֵי הָיְתָה / מְלָכִים מְבַלַּעַת וְאוֹכֶלֶת.

וְשַׁבְתִּי נְטוּי גָּרוֹן, וְרָחָב פֶּה / וְלָשׁוֹן, וְנִשָּׂא רֹאש וְגֻלְגֹּלֶת.

וְהִנֵּה, עֲלֵי שִׂיאִי וְהִכָּנַע / מְשַׂנְאַי אֲשֶׁר הָיוּ לְמַפֹּלֶת,

וְעַל טוֹב וְעַל חֶסֶד אֲשֶׁר יַד אֵל, / בְּטוּבוֹ וְחַסְדּוֹ, לִי מְחַבֶּלֶת,

וְשִׁירִי עֲלֵי יִשְׁעוֹ, אֲשֶׁר לָעַד / יְחֻדַּש בְּכָל עֵדָה וּמַקְהֶלֶת –

יְשׂוּשׂוּם נְאוֹת מִדְבָּר וְגַם תָּגֵל / עֲרָבָה וְתִפְרַח כַּחֲבַצֶּלֶת,

וְתָרֹן לְשׁוֹן אִלֵּם, וּפִסֵּחַ / יְדַלֵּג בְּשִׂמְחָתוֹ כְּאַיֶּלֶת.
