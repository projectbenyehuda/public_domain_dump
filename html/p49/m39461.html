הֱיֵה שׁוֹמֵר בְּרִית

הֱיֵה שׁוֹמֵר בְּרִית רֵעִים / וְאַל תָּקֹץ בְּרֵעָתָם

וְאִם-קָרְאוּ בְּצָרָה לָךְ / הֱיֵה לָהֶם יְשׁוּעָתָם

וְאַל-תָּמֵר בְּדִבְרֵיהֶם / וְאַף כִּי-אֶת שְׁבוּעָתָם

וּמַלֵּא לַעֲנָה וָרֹאשׁ / מְשַׂנְאֵיהֶם לְשָׂבְעָתָם

וְאַל-תִּתְחַר בְּמַרְשִׁיעִים / וְאַל-תִּתְחַר בְּרִשְׁעָתָם

וְאַל-תַּסְגֵּר אֲשֶׁר נָס מִ/פְּנֵי רָעִים וְרָעָתָם

וְסוּר מֵאִישׁ אֲשֶׁר שָׁעָה / מְנָאֵץ הוּא וְשָׁעָה תָּם

וּמֵרוֹעִים אֲשֶׁר אָוֶן / וְתַהְפּוּכוֹת בְּמִרְעָתָם.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
