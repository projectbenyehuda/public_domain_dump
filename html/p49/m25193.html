[שמע רופא]

שְׁמַע, רוֹפֵא, אֲשֶׁר הֵנִיס כְּאֵבַי: // הֲיֵשׁ אֶצְלָךְ צֳרִי לִנְדֹד אֲהוּבַי?

וְאַתְּ, חוֹזֶה בְּכוֹכָבִים, הֲתָבִין // בְּכוֹכָבִים אֲשֶׁר צָפְנוּ קְרָבַי?

אֲנִי חַבְתִּי – וְאֵל אַמִּיץ לְקָחָךְ // וְהֶגְלָךְ מִשְּׁכוּנָתִי, בְּחוֹבַי,

וְהִכַּנִי בְּנוּדָךְ, וַאֳנִי רַךְ, // כְּמַכַּת רֹאשׁ עֲלוֹת לֶקֶשׁ, בְּגוֹבַי.

שְׁתַּיִם נִקְבְּצוּ עָלַי בְּלֶכְתָּךְ, // וְזֹאת אוֹ זֹאת תְּעוֹרֵר אֶת עֲצָבַי:

אֲנִי שׁוֹתֵק וּמִתְאַפֵּק פְּעָמִים, // וְעִתִּים, מִבְּכִי, נַחַל סְבִיבַי.

וְעֵת אֶדֹּם – יְרִיבוּנִי חֲבֵרַי, // וְאִם אֶבְכֶּה – יְרִיבוּנִי קְרוֹבַי,

וְאֵיךְ אוּכַל נְשׂא אַחַת, בְּרֹאשִׁי // תְּשׁוּפֵנִי, וְאַחַת עַל עֲקֵבַי?

וְלֹא אֶשְׁלַח אֲשֶׁר יִרְאֶה שְׁלוֹמָךְ, // וְלֹא אֶשְׁלַח לְךָ כִּתְבֵי אֲהָבַי,

לְמַעַן לֹא תְּשׁוּרֵנִי, וְאֶחְמֹד // לְכָל תָּשׁוּר, וְלוּ הָיוּ יְרִיבַי,

אֲשֶׁר לוּ יֶחֱזוּךָ – יֶחְדְּלוּ מִן // מְרִיבָתִי וְיֵלְכוּ עַל נְתִיבָי

וְעַל כָּכָה אֲנִי חוֹמֵד לְכָל מַה // תְּשׁוּרֵהוּ, עֲדֵי אֶחְמֹד כְּתָבָי.
