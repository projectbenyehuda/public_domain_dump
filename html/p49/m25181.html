[ידידיך בך ירבון]

ולה גוֹאב אלי נסים זקל

ולו תשובה אל רב נסים זכר קדוש לברכה


U – – – U – – – U //– – U – – – U – – – U – –

יֶדִידֶיךָ בְּךָ יַרְבּוּן שְׁשׁוֹנִים, // וְצָרֶיךָ יְנִיקוּן רֹאשׁ פְּתָנִים,

וְתוֹרָתָךְ שְׁמֵחָה בָךְ, וְשִׁמְךָ // בְּמֶרְחַקִּים כְּמוֹ רֵיחַ שְׁמָנִים.

לְךָ הַטּוֹב, לְךָ ישֶׁר, לְךָ תֹם, // לְךָ הֵמָּה בְּמַתָּנָה נְתוּנִים,

וּמִיּוֹם צֵאת שְׁמָךְ הַטּוֹב בְּבָבֶל // קְרָאוּךָ בְּשֵׁם רֹאשׁ הַמְּבִינִים.

לְךָ מִשְׂרָה, לְךָ כֶתֶר תְּהִלָּה // וְתִפְאֶרֶת, לְעוֹלָמִים נְכוֹנִים,

וְלָךְ נָתְנוּ בְּנֵי עִירָךְ שְׂרָרָה, // כְּמוֹ נָתְנוּ לְהוֹרָךְ, אִישׁ אֱמוּנִים,

זְקַן יִרְאָה, חֲבֵר תּוֹרָה וּמוּסָר, // יְשִישׁ תַּלְמוּד, שְׁאֵרִית הַנְּבוֹנִים,

וְהַנִּכָּר בְּעוֹשֵׂי הַגְּדִילִים, // וְהַנִּכְבָּד בְּעוֹטֵי הַסְּדִינִים.

וּבִרְאוֹתָם בְּאַפְרִיקֵי כְּבוֹדוֹ // חֲשָׁבוּהוּ כְּאַחַד הַגְּאוֹנִים,

וְחִלּוּהוּ, כְּמוֹ חִלּוּ אֲנָשִׁים, // בְּדוֹר הָלַךְ, פְּנֵי שִׁבְעִים זְקֵנִים!

וְיֵשׁ מִשְׂרָה בְּלִי אֶדֶן – וְהָיְתָה // שְרָרָתוֹ יְסוּדָה בַאֲדָנִים,

וּבָנִיתָ לְךָ בִּנְיַן יְקָר עַל // יְסוֹדוֹתָיו, וְלֹא בִנְיַן אֲבָנִים.

וְיֵשׁ בּוֹנֶה לְנַפְשׁוֹ בֵית גְּדֻלָּה // בְּלֹא חֹמֶר אֲדָמָה אוֹ לְבֵנִים,

כְּבִנְיָנָךְ, אֲשֶׁר בִּתְהוֹם יְסוֹדוֹ // וְתִקְרָתוֹ עֲלֵי רָאשֵׁי עֲנָנִים.

אֲהַבְתִּיךָ בְּטֶרֶם אֶחֱזֶה אֶת // כְּתָבֶיךָ, וְחַי שׁוֹכֵן מְעוֹנִים!

וְהוּסַף אָז לְךָ אַהַב בְּלִבִּי // עֲשָׂרָה מֵאֲשֶׁר הָיָה לְפָנִים.

תְּהִלָּתָךְ גְּבֶרֶת לַתְּהִלּוֹת // וּמִכְתָּבָהּ לְכָל מִכְתָּב אֲדוֹנִים,

מְאֻזֶּנֶת בְּמִשְׁקַל יִשְׁמְעֵאלִים // וּמוּכֶנֶת בְּחָכְמַת הַיְּוָנִים,

וּבָהּ מִלִּים כְּאִלּוּ הֵם כְּלִילִים, // וְעִנְיָנִים כְּאִלּוּ הֵם פְּנִינִים,

וְאֵיךְ אָשִׁיב גְּמוּל שִׁירָךְ, בְּשִׂיחִים // כְּמוֹ שָׁנִי, עֲלֵי שִׂיחִים לְבָנִים?

וּמִי יֵצֵא לְהִלָּחֵם, בְּמַקֵּל // בְּיָדוֹ, עִם מְתֵי חִצִּים שְׁנוּנִים?

קְחָה, חֵלֶף שְׁבָחֶיךָ, לְבָבִי // בְּמַתָּן עִם כְּסָלַי הַצְּפוּנִים,

וְשִׁירָה, כַּמְּנוֹרָה הַטְּהוֹרָה, // מְהֻדֶּרֶת, מְקוֹם טוּרִים, בְּקָנִים,

וְאִם אֵינָהּ כְּמוֹ שִׁירָךְ – בְּתוֹכָה // דְבָרִים, הֵם לְצָרֶיךָ צְנִינִים.

לְקַח אוֹתָהּ וְתָעִיד אֶצְלְךָ כִּי // אֲהָבַי לָךְ לְעוֹלָם נֶאֱמָנִים,

וְדַבֶּר בָּהּ, וְתֵחָשֵב כְּאִלּוּ // אֲדַבֵּר עִמְּךָ פָּנִים בְּפָנִים.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
