[יוֹסִיף דּוֹד שֵׁנִית יָדוֹ]

יוֹסִיף דּוֹד שֵׁנִית יָדוֹ / לִקְבֹּץ אֶת כָּל פְּזוּרִי

הֶחָפֵץ שְׁלוֹם עַבְדּוֹ / כִּי הוּא תִקְוָתִי וְגַם שִׂבְרִי

אָשִׁיר לוֹ שִׁירָה

זַכָּה וּבָרָה

יֹאמַר מְהֵרָה / הִתְנַעֲרִי

נִשְׂגָּב לְבַדּוֹ

יִקְבֹּץ נְדוּדִי

לְעִיר חֶמְדָּתִי / יִבְנֶה מִקְדָּשִׁי

וְלֹא יָבוֹא עוֹד שִׁמְשִׁי / וְגַם יְרֵחִי לֹא יֵאָסֵף

לְעֵדָה יָפָה תַמָּה שִׂימָה / עֲלֵי עַיִן בֵּן פֹּרָת יוֹסֵף.


וּשְׁכִינַת עֻזְּךָ, אֵלִי, / כְּקֶדֶם עָלַי תְּחוֹפֵף

אֶל בֵּית מִשְׁכַּן זְבוּלִי / שָׁאֲפָה נַפְשִׁי לְהִסְתּוֹפֵף

דּוֹד אֶשְׁכֹּל כֹּפֶר

יִדְמֶה אֶל עֹפֶר

בְּרִית אַל תָּפֵר / מִבֵּית יוֹסֵף

שׁוֹכֵן עֲלִיָּה

בַּשֵּׂר שְׁבִיָּה

עֵינָם צוֹפִיָּה / יָדְךָ תְנוֹפֵף

עַל הַר בַּת צִיּוֹן רַגְלֵי / מְבַשֵּׂר כָּעוֹף יְעוֹפֵף

לְעֵדָה יָפָה תַמָּה שִׂימָה / עֲלֵי עַיִן בֵּן פֹּרָת יוֹסֵף.


סַלְעִי מִשְׂגָּב בְּצָרָה / תָּאִיר אֶת דַּרְכִּי וּנְתִיבָתִי

אַתָּה תִהְיֶה לִי עֶזְרָה / מַהֵר תָּחִישׁ יְשׁוּעָתִי

עַמְּךָ תְּחַלֵּץ

מִיַּד מְאַלֵּץ

וּבְטוּבְךָ עַלֵּץ / לַעֲדָתִי

כֻּלָּם יִשְׂמְחוּ

רִנָּה יִפְצְחוּ

יַחַד יָנוּחוּ / בִמְנוּחָתִי

הָשֵׁב אֶת צִיץ תִּפְאָרַָה / וְאֶת חֶרְפָּתֵנוּ אַסֵּף

לְעֵדָה יָפָה תַמָּה שִׂימָה / עֲלֵי עַיִן בֵּן פֹּרָת יוֹסֵף.


פְּדֵה, צוּרִי, אֶת עֵדָה / אֲשֶׁר בְּיַד צַר לְכוּדָה

רוֹעָהּ אַתָּה מֵעוֹדָהּ / נָא הַבֵּט וּרְאֵה נְדוּדָהּ

גָּדוֹל וְנוֹרָא

מָגֵן וְסִתְרָה

הָשֵׁב הָעֲטָרָה / הַחֲמוּדָה

עַל רֹאשׁ עַבְדְּךָ

חִישׁ תְּנָה הוֹדְךָ

הַגְדֵּל חַסְדְּךָ / לְסַעֲדָהּ

בֵּית אֶפְרַיִם וִיהוּדָה / אֶלֶף פְּעָמִים עֲלֵיכֶם יוֹסֵף

לְעֵדָה יָפָה תַמָּה שִׂימָה / עֲלֵי עַיִן בֵּן פֹּרָת יוֹסֵף.


חָזָק אֵל אָבִינוּ הוּא / יִבְנֶה מִקְדָּשׁ עַל נְוֵהוּ

כַּלְּבָנוֹן מַרְאֵהוּ / אֱלֹהֵי אָבִי אֲרוֹמְמֶנְהוּ

רוֹכֵב עֲרָבוֹת

בּוֹחֵן לְבָבוֹת

בָּנִים וְאָבוֹת / יְהַלְלוּהוּ

מִזַּעֲקַת דַּל

אָזְנוֹ לֹא תֶחְדַּל

שְׁמוֹ יִתְגַּדַּל / כָּל חַי יוֹדֵהוּ

בּוֹטֵחַ בּוֹ אַשְׁרֵהוּ / נִבְחָר מִזָּהָב וָכֶסֶף

לְעֵדָה יָפָה תַמָּה שִׂימָה / עֲלֵי עַיִן בֵּן פֹּרָת יוֹסֵף.


מקור: שירה ופיוט של יהודי בבל, מכון בן צבי, תש״ל


מיהודי בבל בהודו. שיר אחד ממנו נדפס בהגדה של פסח בבומבאי תרנ״ו. סימנו: יוסף חזק, כנראה שם המהולל.

סימן: יוסף חזק. לחן: יונה הומה.
