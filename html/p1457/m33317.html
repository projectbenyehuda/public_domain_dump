יָהּ זַעֲמָךְ אַל יִיחַר

תמאם מתוך שבעתא לערבית

יָהּ / זַעֲמָךְ אַל יִיחַר

הַבֵּט לַבְּרִית / וְזַעְפָּךְ יִתְאַחַר

זְכוֹר / זָעַק תַּחַן בְּרֹאשׁ הָהָר

וּסְלַח / זְדוֹנוֹת עַם מֻבְחָר

וְזִבְחֵי אַיֶּלֶת הַשַּׁחַר

זְרַח וְצַחְצְחֵם כְּצֹהַר וַעֲשֵׂה אַל תְּאַחֵר.


יְיָ / רוֹעִי אֶקְרָא

הַבֵּט לַבְּרִית / וְרֶשַׁע אַל תְּבַקְּרָה

זְכוֹר א׳ / רָץ וְהִגִּישׁ קְטוֹרָה

וּסְלַח / רָעַת אוֹם נוֹטֵרָה

וּרְחוֹשׁ שֵׁמַע טוֹב לַעֲקָרָה

וְרֶגֶשׁ לָעִיר תִּקְרָא

לְמַעַנְךָ אֱלֹהַי כִּי שִׁמְךָ נִקְרָא.


מִי יַעֲמוֹד / הָדוּר בְּזַעֲמֶיךָ

הַבֵּט לַבְּרִית / וָתִיקֶיךָ לְהַצִּיל מִמִּתְקוֹמְמֶיךָ

זְכוֹר או׳ / דִּגְלֵי תְמִימֶיךָ

וּסְלַח / הַפֶּשַׁע לַחֲתוּמֶיךָ

וַחֲבוֹשׁ שֶׁבֶר אוּלַמֶּיךָ

וּזְכוֹר בַּעֲבוּר שְׁמֶיךָ

קָדוֹשׁ כִּי שִׁמְךָ נִקְרָא עַל עִירְךָ וְעַל עַמֶּיךָ.



את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
