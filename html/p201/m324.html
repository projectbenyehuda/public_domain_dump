מוחרך

יָפְתָה עַלְמַת / חֵן וּמֵעַיִן / נֶעֱלָמָה:

בַּעֲלֵי חָכְמַת / לֵב, דְּעוּ “אַיִן, / אֵיךְ וְלָמָּה”.


יָרְדָה אֶל בֵּית / טִיט בְּהִתְאַסֵּף / גֵּו וְצוּרָה

וַתְּהִי נֶחְבֵּאת / שָׁם בְּאֵין כֹּסֶף, / רַק עֲצוּרָה.

לֹא בְיָד נִשְׁבֵּית / גַּם בְּלִי כֶסֶף / הִיא מְכוּרָה

לַעֲבֹד אַדְמַת / גּוּף, וְלָשִׂים לוֹ / הוֹד וְאֵימָה

חֹק, וּבִמְזִמַּת / לֵב לְהַבְדִּילוֹ / מִבְּהֵמָה.


סוֹד פְּלִיאָתְךָ / מַה מְּאֹד יִגְבַּר, / צוּר, לְהָבֵן!

עֹז בְּרִיאָתְךָ / בָאֱנוֹשׁ – חֻבַּר / פָּז לְאֶבֶן

רֹאשׁ פְּעֻלָּתְךָ / הִיא – וְאֶת הַבָּר / מַה לְּתֶבֶן?

חִבְּרָה עָצְמַת / יָדְךָ גָלְמִי / אֶל נְשָׁמָה,

כּוֹנֲנָה רִקְמַת / מַחֲזֶה צַלְמִי / מֵאֲדָמָה.


שָׁאֲלָה חֹפֶשׁ / מֵאֲשֶׁר עֻבַּד / בָּהּ יְחִידָה

יוֹם לְהִנָּפֵשׁ / בּוֹ, וְאַל תִּכְבַּד / הָעֲבוֹדָה,

כִּי יְסוֹד נֶפֶשׁ / חַי בְּלִי יֹאבַד / יוֹם פְּרִידָה,

בַּאֲשֶׁר קִדְמַת / גּוּף יִהְיֶה נִמְצָא / הִיא קְדוּמָה,

אַךְ לְפִי תֻמַּת / מַעֲשִׂים נִרְצָה / אוֹ זְעוּמָה.


יָהּ, תְּהִלָּתְךָ – / צוּף, וּמִלָּתְךָ – / מָן וְנֹפֶת,

עֹז פְּעֻלָּתְךָ / עַל גְּדֻלָּתְךָ / אוֹת וּמוֹפֵת.

חֵן קְהִלָּתְךָ / בֵית תְּפִלָּתְךָ / נֶאֱסֶפֶת!

תַּעֲטֹף נִשְׁמַת / כָּל בְּנֵי חֶלֶד / לָךְ כְּאָמָה,

קוֹל בְּלִי נִצְמָת / “הַלֲלוּ יוֹצֵר / כָּל נְשָׁמָה!”
