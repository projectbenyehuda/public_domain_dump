אִתָּךְ בִּזְכוּת אֲמִיתָךְ

קרובה לספר בראשית ח א – ויזכור אלהים את נח (סימן א״ב)


אִתָּךְ בִּזְכוּת אֲמִיתָךְ / בְּרָכָה לָאָרֶץ בְּתִתָּךְ

גִּבּוֹר הָרֵק חֲנִי[תָךְ] / דּוֹרָךְ מָאַס בְּדַעְתָּךְ

הָמְחָה וְהִסְרַדְתִּי אוֹתָךְ / וְהֶחְבֵּאתִיךָ ל……תָךְ

זְכַרְתִּיךָ לְחַיּוֹתָךְ / חַנּוֹתִי [תַ]חֲנוֹתָךְ

טִהַרְתִּי טֻמְ[אָתָךְ] / יִיחַסְתִּי מִטָּתָךְ

כִּלַּלְתִּי [אַ]הֲבָתָךְ / לָךְ טַל יַלְדוּתָךְ

מִמָּךְ אֲמַלֵּא אַדְמָתָךְ / נִטְעֵי נַ[עֲמָנִים] בִּרְכָתָךְ

סַגִּיא מְאֹד תְּהִי אַחֲרִיתָךְ / עֲדֵי עַד קִיַּמְתִּי בְרִיתָךְ

פליט…………[תָךְ] / צָפִיתִי לַעֲתִידָתָךְ

קֵרַבְתִּי חִבָּתָךְ / רבץ………תָךְ

שָׁלוֹם וּמִישׁוֹר מְצִיאָתָךְ / תַּתִּי אוֹר חָדָשׁ לִיצִיאָתָךְ.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
