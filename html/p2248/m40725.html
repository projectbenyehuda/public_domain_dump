[אֶכְרַע אֶקֹּד לִפְנֵי מַלְכִּי] (רשות לנשמת)

[בן המשפחה De Synagoga או Casadio, חי ברוֹמָא במחצית הראשונה של המאה הי"ד]


רְשׁוּת לְנִשְׁמַת שֶׁל רֹאשׁ הַשָׁנָה

אֶכְרַע אֶקֹּד לִפְנֵי מַלְכִּי / אֶקְרָאָה מִפָּנָיו אַבְרֵךְ

שׁוֹפֵט יָשָׁר הוּא אֵלִי כִּי / יוֹרֶה חַטָּאִים בַּדָּרֶךְ.


יוֹם זֶה יוֹם הַדִּין הוּא נִקְרָא / מִשְׁפָּט הוּא לֵאלֹהֵי יַעְקֹב

יֹשֵׁב עַל־כִּסֵּא רָם נוֹרָא / וּלְמִישׁוֹר יַיְשִׁיר הֶעָקֹב

עַל־הֶאָח תּוֹךְ אֵשׁ בּוֹעֵרָה / יַבְעִיר כָּל־אָח עָקֹב יַעְקֹב


לָכֵן אַתְּ שׁוֹכֶבֶת חֵיקִי / הִתְנַחְמִי סִפְקִי עַל־יָרֵךְ

עַל־בָּמוֹת יֵצֶר לֵב דִּרְכִי / תִּרְדִּי־בוֹ בַּעְבֹדַת פָּרֶךְ.


וּבִתְרוּעַת מֶלֶךְ עוּרִי / וּבִתְקֹעַ שׁוֹפָר שִׁמְעִי

וּבִשְׁבָרִים יִצְרֵךְ שִׁבְרִי / הַשְׁפִּילִי אוֹתוֹ הַכְנִיעִי

וּזְדוֹן עָרְלַת־לֵב הָסִירִי / הוֹחִילִי לֵאלֹהֵי יִשְׁעִי


שׁוּבִי שׁוּלַמִּית עָדָיו כִּי / חַנּוּן וְרַחוּם הוּא אַף אֶרֶךְ

גּוֹאֵל מִשַּׁחַת חַיָּיְכִי / אֵין לוֹ דִמְיוֹן אַף־לֹא עֵרֶךְ.


אֵל דָּרַשׁ סֵפֶר וַיִּקְרָא / אַיֵּה שֹׁקֵל אַיֵּה סֹפֵר

וּלְצַדִּיק דּוֹרֵךְ הַיְשָׁרָה / דּוֹדִי לִי אֶשְׁכֹּל הַכֹּפֶר

וּרְשָׁעִים יִשְׁתּוּ מֵי מָרָה / אוֹכֶלֶת אֵשׁ כָּל־הַכּוֹפֵר


וּמִמַּחְצִית פִּשְׁעֵי עֶרְכִּי / מַעְבִיר רִאשׁוֹן צוּר מִתְבָּרֵךְ

וּמַכְרִיעַ מֹאזְנַי צִדְקִי / וּמֵשִׂים פַּעְמַי לַדָּרֶךְ.


בִּינוּ־נָא זֹאת הָאֵל דִּרְשׁוּ / מִיּוֹם דִּין עַד־יוֹם כַּפָּרָה

וּבֵינְתַּיִם פָּנָיו בַּקְשׁוּ / לֹא יַשְׁחִית בַּעְבוּר הָעְשָׁרָה

וּבְכָל־לֵב אוֹתוֹ תַקְדִּישׁוּ / הוּא יַצִּילְכֶם מִכָּל־צָרָה


גַּם אַתְּ נַפְשִׁי אֵלִי בָרְכִי / וּלְפָנָיו הַכְרִיעִי בָרֶךְ

הוּא יִגְמֹל חֶסֶד עָלָיְכִי / קוּמִי אוֹרִי כִּי־בָא אוֹרֵךְ.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
