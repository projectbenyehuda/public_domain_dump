אהבה / לוי אבן-מר-שאול

"לִבַּבְתִּנִי, אֲחוֹתִי / אֲשֶׁר פְּנֵי דַמֶּשֶׂק צוֹפָה,

בְּחַד עֵינַיִךְ בִּהְיוֹתִי / וְנַפְשִׁי מְאֹד לָךְ נִכְסָפָה

יוֹם קָרָאתִי, בִּרְאוֹתִי / כְּחוּט הַשָּׁנִי הַשָּׂפָה:

מִי זֹאת הַנִּשְׁקָפָה / כְּמוֹ שַׁחַר יָפָה?"

"בְּלֶכְתִּי כַּאֲשֶׁר נִפְרַדְתִּי – / וְעֵינַי כְּחוּלוֹת בְּלִי כַחַל –

רַצְתִּי וְחַשְׁתִּי וְחָרַדְתִּי / בְּשָׁמְעִי אֶת קוֹל הַשַּׁחַל.

עֲנִיתִיהוּ וְלֹא מָרַדְתִּי / וַיְהִי בְלִבִּי כְּמוֹ גַחַל:

אֶל גִּנַּת אֱגוֹז יָרַדְתִּי / לִרְאוֹת בְּאִבֵּי הַנַּחַל.

יוֹם עָמַדְתִּי וָאֵפֶן / בְּבֵיתִי בְּעַד הַחַלּוֹנִים,

צָפַנְתִּי לְךָ דוֹדַי, פֶּן / יִרְאוּנִי עֵינֵי יוֹנִים,

חוֹשְׁקָם צָפוּן וְצוֹפֵן / מִבֵּין הֲדַסִּים רַעֲנַנִּים –

לִרְאוֹת הֲפָרְחָה הַגֶּפֶן / הֵנֵצוּ הָרִמּוֹנִים".
