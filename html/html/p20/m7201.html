[אֵל אֱמֶת יְבַקְּשׁוּ נֶאֱמָנָיו]

אֵל אֱמֶת יְבַקְּשׁוּ נֶאֱמָנָיו, / הוּא יְאַשְׁרֵם,

יִזְכְּרוּ שְׁמוֹ, וּבְאוֹר פָּנָיו / הוּא יְפָאֲרֵם.


בֶּאֱמֶת נֱתִיבוֹת חָכְמָתוֹ / הוּא יְכוֹנְנֵם,

וַאֲשֶׁר יְבוֹאוּן לִקְָראתוֹ – / הוּא יְחוֹנְנֵם,

מַחְשְׁבֹות אֱנוֹשׁ יֵדַע אוֹתוֹ – / הוּא יְבוֹנְנֵם.

יַעֲמִיד עֲלֵי חַסְדּוֹ מָתְנָיו – / הוּא יְאַזְּרֵם,

יַעֲזֹב נְתִיב שָׁוְא וַעְוֹנָיו – / הוּא יְכַפְּרֵם.


רַחֲמָיו יְקַוּוּ עַם נָפְלוּ,/ הוּא יְקוֹמְמֵם,

נִכְנְעוּ וְלֶעָפָר שָׁפְלוּ, / הוּא יְרוֹמְמֵם,

וַעֲלֵי הֲדוֹם רַגְלָיו אָבְלוּ, / הוּא יְנַחֲמֵם.

יִשְׁמְרוּ בְשִׁבְיָם חֹק עָנָיו – / הוּא יִשְׁמְרֵם,

אֶל דְּבִיר אֲשֶׁר שָׁמָּה עֵינָיו / הוּא יְחַבְּרֵם.


מַה יְּחַפְּשׂוּ לִבּוֹת חַפִּים? / הוּא יְסוֹבְבֵם.

עַל שְׁמוֹ יְנוּסוּן נִרְדָּפִים, / הוּא יְשׁוֹבְבֵם.

מוֹעֲדִים אֲשֶׁר אָמְרוּ צוֹפִים – / הוּא יְקָרְבֵם.

הֵם יְחַזְּקוּ לֵב אֵיתָנָיו – / הוּא יְגַבְּרֵם,

כַּאֲשֶׁר יְיַסֵּר אִישׁ בָּנָיו / הוּא יְיַסְּרֵם.
