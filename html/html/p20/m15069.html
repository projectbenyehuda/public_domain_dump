אֲחַלֶּה פָנֶיךָ בְּאֵימָתָה

אֲחַלֶּה פָנֶיךָ בְּאֵימָתָה, / אֲדוֹן כָּל בְּרִיאָה נִבְרְאָתָה,

אָז בְּאַיִן יֵשׁ נִהְיָתָה, / אֶזְרוֹעֲךָ עָלֶיהָ נִגְלָתָה,

אַף מִמְּאוֹרָהּ נִכְרָתָה / אֱמוּנַת נֶפֶשׁ נִפְלְאָתָה –

הִיא הָיְתָה / אֵם כָּל חָי.


בְּשׁוּב הַשּׁוֹבָב לְשִׂכְלוֹ – / בְּטוֹבָתְךָ תִּסְלַח לוֹ!

בִּינוֹתִי בְאָב בְּמָעֳלוֹ / בָךְ מֵהָעֵץ בְּאָכְלוֹ,

בְּכֵן נִגְלָה רוּחַ נִגְלֹה, / בַּגָּן נִשְׁמַע קוֹלוֹ –

כִּי יְדַבֵּר אֱלֹ-/ הִים אֶת הָאָדָם וָחָי.


רַב עִוָּה מַסְלוּלוֹ, / רָצַח אָח וְחִבְּלוֹ,

רִאִיתָ הִנָּחֲמוֹ לְפָעֳלוֹ – / רָחֲמֶיךָ הִמְצֵאתָ בְּשָׁאֳלוֹ.

רֹאשׁ בָּנִים, יָצוּעַ בְּחַלְּלוֹ – / רְצִיתוֹ בְּשׁוּבוֹ וַתִּשָּׂא לוֹ,

כִּי לֹא / יִצְדַּק לְפָנֶיךָ כָל חָי.


הוֹדָה בְחֶטְאוֹ בְּנוֹ תָם, / הֲרָאִיתָ סוֹד סָתָם.

הִרְבּוּ שְׁלשָׁה מְשׁוּבָתָם, / הֲכִי רַבָּה תְּשׁוּבָתָם!

הָעֵת שֶׁנִּכְנְעוּ בְמַלְכוּתָם, / הָלכוּ בְדָתֶיךָ וְחֻקּוֹתָם,

אֲשֶׁר יַעֲשֶׂה אֹתָם / הָאָדָם וָחָי.


מִסְגְּרוֹת לִבִּי קָרָעְתִּי, / מִבֵּית שִׁבְיִי שִׁוָּעְתִּי,

מַלְכִּי, בְּךָ פָגָעְתִּי: / מִפְּשָׁעַי פְּדֵנִי וְנוֹשָׁעְתִּי!

מַה תִּתְפָּאֵר, צַר, אִם נָעְתִּי? / מְיַחֵל אֱהִי לוּ גָוָעְתִּי!

וַאֲנִי יָדַעְתִּי: / גֹּאֲלִי חָי.


חַנּוּן יַחְשׂךְ מְרִי- / חֲטָאַי בִּשְׂכַר אִמְרִי!

חָמַס סֻכִּי צָרִי, / חָרַב עֵדֶן חֶדְרִי,

חָסִיתִי בְךָ, שִׂבְרִי, / חֲבֹשׁ נֶגַע שִׁבְרִי –

לִבִּי וּבְשָׂרִי / יְרַנְּנוּ אֶל אֵל חָי.


זְבוּלִי יָרַשׁ יָרֹשׁ / זָר, עַל גַּבִּי יַחֲרֹשׁ,

זְקוּפָה קַרְנוֹ כִּבְרוֹשׁ, / זוֹמֵם לְהַחְרִים וְלִשְׁרוֹשׁ,

זֶרַע רִוָּה מֵי רֹאשׁ; / זְכֹר חֲמָסוֹ וּדְרֹשׁ

עַל רֹאשׁ / הַשָּׂעִיר הַחַי!


קַצְתִּי בְחַיַּי בְּמוֹקְשִׁי / קָשׁוּר בִּידֵי גוֹרְשִׁי,

קִוִּיתִי דִבְרֵי קְדוֹשִׁי / קִצִּי לְקָרֵב וְאֵצֵא חָפְשִׁי.

קַמְתִּי בְּשִׁפְלוּת רֹאשִׁי, / קִדַּמְתִּי לְתַאֲוָה לְשָׁרְשִׁי –

צָמְאָה נַפְשִׁי / לֵאלֹהִים, לְאֵל חָי.
