שְׂשׂוֹן לִבִּי

שְׂשׂוֹן לִבִּי / מִשְׂגַּבִּי / הַנִּשְׂגָּב שְׁמוֹ לָעַד

סְלַח פִּשְׁעִי / וְחִישׁ יִשְׁעִי / הָבֵא צִיר הַנּוֹעָד

אֵלִיָּה / הַתִּשְׁבִּי / מִתּוֹשָׁבֵי גִלְעָד


לְנִדְכָּאִים / וְחֵלְכָּאִים / שְׂבֵעֵי בוּז וְגַם לָצוֹן

פְּנֵה שַׁדַּי / וְתֹאמַר דַּי / לְאֹרֶךְ קֵץ הַקִּיצוֹן

שְׁלַח רוֹעֶה / לְצֹאן תּוֹעֶה / וְאַל תִּטֹּשׁ מְעַט הַצֹּאן

אֲשֶׁר נָגֹז / וְגַם נָבֹז / מִלִּפְנִים וְלַחִיצוֹן

וְהוּא מְיַחֵל / כְּמוֹ רָחֵל / לְיוֹם תִּקְרָא שְׁנַת רָצוֹן

וְצִיץ צַמַּח / וְאָז נִשְׂמַח / בְּשִׂמְחַת בֵּית הַוָּעַד


מְחָצוּנִי / לְחָצוּנִי / פְּרִיץ חַיּוֹת וַאֲרָיוֹת

הִנְנִי גָר / וְגַם נִסְגָּר / בְּיַד אַרְבַּע מַלְכֻיּוֹת

נְבוֹ קָרַס / וְקָם פָּרַס / וֶאֱלִיפַז וּנְבָיוֹת

זֶה יִתְפֹּשׂ / וְזֶה יִרְפֹּשׂ / וְזֶה יַדִּיחַ שֵׂיוֹת

וּבְכָל זֶה / אֲנִי אֶחֱזֶה / וְעֵינַי לְךָ צוֹפִיּוֹת

וְכָל דּוֹרֵשׁ / קֵץ יְפָרֵשׁ / וְיִסְפֹּר הָאוֹתִיּוֹת

אוּלַי תַּם / אֲשֶׁר נֶחְתַּם / וּ[בֵ]יתִי תָכִין עֲדֵי עַד.
