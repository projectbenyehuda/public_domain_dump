מֵאֵל עֵילוֹם

אל רב גדול אלמוני, וליעקב ושלמה


מֵאֵל עֵילוֹם יוּבַל שָׁלוֹם / רֵיחוֹ בֹשֶׂם טַעְמוֹ נֹפֶת,

רֻבּוֹ יִשָּׂא הָרַב, שֶׁהוּא / עַל רֹאשׁ תּוֹרָה בִּלְבַד צֶפֶת,

אַף תּוֹרָתוֹ כָּל יוֹם תּוֹסִיף / גַּם עַנְוָתוֹ כֵּן נוֹסֶפֶת.

כַּמָּה בְרַח הוּא מִשְּׂרָרָה / אָכֵן אְַחְרָיו הִיא רוֹדֶפֶת

יוֹם נֶעֶלְמָה מֵעֵין כָּל־חַי / הוּא נִמְצָא לָהּ אוֹת אוֹ מוֹפֵת,

גַּם הוּא לָהּ דּוֹד כִּי תוֹרָתוֹ / כַּשַּׁחַר עַל כֹּל נִשְׁקֶפֶת

וּבְמִשְׁפָּטָיו מָצְאוּ בָרִים / עֵדֶן וּבְנֵי עָוֶל תֹּפֶת.

וּלְרַב יַעְקֹב כִּי הַחָכְמָה / הָיְתָה לוֹ שִׁפְחָה נֶחְרֶפֶת,

קָשַׁר כֶּתֶר בִּין עַל רֹאשׁוֹ / וַיָשֶׂם תֹם לוֹ מִצְנֶפֶת.

גַּם לִשְׁלֹמֹה הַשַּׂר, כִּי בֵין / עֵינֵי שִׂכְלוֹ דָת טוֹטֶפֶת.

מַשְׂכִּיל, מִפִּיו עַב הַחָכְמָה / עַל כָּל צָמֵא אַט עוֹרֶפֶת.

וּלְכָל מֵבִין עִם תַּלְמִיד, כִּי / מִפִּיהֶם דָּת אֵל נוֹטֶפֶת.

חֶבְרַת שֵׂכֶל בִּרְאוֹת אוֹתָהּ / שֶׁמֶשׁ יֶחְרַד וְיִלָּפֶת,

נַפְשִׁי כֹפֶר לָהּ מֵאָז כִּי / לַחְזוֹת אוֹתָהּ הִיא נִכְסֶפֶת.

יִנּוֹן זִכְרָם לִפְנֵי שֶׁמֶשׁ / וַעֲלֵיהֶם יַד אֵל חוֹפֶפֶת.


מִי זֶה דִּמָּה

אל יעקב בן אלעזר


מִי זֶה דִּמָּה / כִּי סוֹד חָכְמָה / מִתּוֹךְ אֶרֶץ / חַיִּים נִגְזָר,

יָסוּר לִרְאוֹת / מוֹפֵת וָאוֹת / בִּכְתָב מַשְׂאוֹת / בֶּן אֶלְעָזָר.

סֵפֶר מַחְפִּיר / לִבְנַת סַפִּיר / בּוֹ אִישׁ מ[וּסָ]ר / עֹז יִתְאַזָּר.

וּצְנִיף חָכְמָה / שִׁתְיוֹ עָרְמָה / עֶרְכּוֹ תֻּמָּה / לֹא שֵׁשׁ מָשְׁזָר,

חָכְמוֹת יָוָן / בַּחֵיק שִׁוָּן / לִהְיוֹת אָמְרוֹ / לָהֶם מִנְזָר,

דֵּעוֹת לִשְׁמֹר / בִּינוֹת לִגְמֹר / כִּי מֹר וּכְפוֹר / בָּהֶם פִּזַּר.

מוּסָר אִתּוֹ / בֵּית תִּפְאַרְתּוֹ / אָכֵן בִּלְתּוֹ / יִהְיֶה מוּזָר.

צוּר כֵּס גָּדְלוֹ / יוֹם הַנְחִילוֹ / שָׂח: עַל כִּסְאוֹ / לֹא יֵשֵׁב זָר.
