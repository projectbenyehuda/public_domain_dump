עַד אָן

סימן: עבדאללה.


עַד אָן לְדִינִי לֹא דָן / חֲוִילָה וּבְנֵי דְדָן

בְּחַר אֵל נֶחְבָּא מִפַּח / (וזייני) חֶרֶב עִם הַדָּרְבָן / וְאַלָּה.


בְּאֶבְרָתוֹ יָסֶךְ לוֹ / שִׂמְחַת עוֹלָם נָתַן

עַל רֹאשׁוֹ, וּלְכָל חֵילוֹ / יֶאְסֹף גִּנַּת בִּיתָן / טוֹב סֶלָה.


דּוֹרֵךְ עַל בָּמֳתֵי יָם / יִמְחַץ רֹאשׁ לִוְיָתָן

לַזָּרִים יַצִּיל עֶדְיָם / יְעוֹרֵר לִוְיָתָן / בִּמְצוּלָה.


אֵל, מֻדָּח כַּצְּבִי / רֹב עֲצָמָיו אֵיתָן

תֵּן לַעֲטֶרֶת צְבִי / יִשְׁקֹט בִּנְוֵה אֵיתָן / תְּהִלָּה.


לְךָ שִׁיר וְרֹן נָבִיא / עֵת שׁוֹד צַר יְחִיתַן

גַּם כֹּהֵן וְגַם נָבִיא / יָשִׂישׂ כִּמְשׂוֹשׂ חָתָן / עַל כַּלָּה.


הֵן עַם נִמְשַׁל כְּלָבִיא / תִּשְׁכֹּן אֵל בְּקִרְבָּן

רִנָּה וְתוֹדָה נָבִיא / תִּשְׁעֶה בִּמְקוֹם קָרְבָּן / תְּפִלָּה.
