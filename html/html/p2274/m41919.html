עָלַץ לִבִּי

סימן: עבדיה.


עָלַץ לִבִּי / בָּךְ מִשְׂגַּבִּי / וּשְׁבָחַי

בְּקוֹל זִמְרָה / אֲזַמְּרָה / עִם אַחַי

עִם אַחַי

חַי אֱלֹהִים חַי.


בְּכָל לִבִּי / דְרַשְׁתִּיךָ / מוֹרִישָׁם

מְקוֹם מִשְׁכַּן / כְּבוֹדֶךָ / בְּרֹאשָׁם

בְּרֹאשָׁם

שָׁם יְיָ שָׁם.


דְּרָכֶיךָ / הוֹדִיעֵנִי / וּלְאַט

תְּנַהֵל, צוּר / אֶת עַם עָנִי / וְלִי הַט

וְלִי הַט

הַט אָזְנֶיךָ הַט.


יְפֵה נוֹף עִיר / צִיּוֹן תִּבְנֶה / וְתִגְדֹּר

סֻכַּת דָּוִד / תָּקִים לִשְׁנֵי / דוֹר וָדוֹר

דוֹר וָדוֹר

דּוֹר יְשָׁרִים דּוֹר.


הַטֵּה שָׁלוֹם / לְעַבְדֶּךָ / כְּמֵי גַל

וְאַבִּיטָה / נִפְלְאוֹתֶיךָ / עֵינַי גַּל

עֵינַי גַּל

גַּל כְּבוֹדִי גַל.
