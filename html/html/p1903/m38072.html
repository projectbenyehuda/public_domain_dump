משל יא: שֻׁעָל וְנֶשֶׁר

הַחוֹמֵס לְרֹב גָּדְלוֹ וְעָשְׁרוֹ / סוֹף יְאַבֵּד גַּם אֶת יְקָרוֹ


שֻׁעָל הָלַךְ לִרְאוֹת בְּאַחַד הַגַּנִּים / וְעִמּוֹ צֶאֱצָאָיו שֻׁעָלִים קְטַנִּים / וַיָּטָשׂ הַנֶּשֶׁר עַל הָאֶחָד וַיִּשָּׂאֶנּוּ / וַיַּרְא הַשֻּׁעָל כִּי הָאֶחָד אֵינֶנּוּ / וְלֹא יָדַע אִם גֻּנַּב אוֹ נֶחְבָּא / אוֹ מֵת אוֹ נִשְׁבַּר אוֹ נִשְׁבָּה / וַיִּשָּׂא עֵינָיו וַיַּרְא כִּי הַנֶּשֶׁר נוֹשְׂאוֹ בְּאֶבְרָתוֹ / וַיָּרֶם קוֹלוֹ וַיֵּבְךְּ וַיִּצְעַק לִקְרָאתוֹ / לְאַט לִי לִבְנִי וַהֲשִׁיבֵהוּ / וְהֶאֱרַכְתָּ יָמִים אִם תְּנַקֵּהוּ / לָמָּה תַכְרִיתֶנּוּ מִמִּשְׁפַּחְתּוֹ / אֲשֶׁר לֹא עָמַלְתָּ בּוֹ וְלֹא גִדַּלְתּוֹ / שְׁמַע הַשְׂכֵּל וְדֵעָה / לָמָּה תִתְגָּרֶה בְרָעָה / וַיַּעַן הַנֶּשֶׁר: מִי אַתָּה אֲשֶׁר אַחֲרַי דָּלַקְתָּ / וּמַה לְךָ כִּי נִזְעַקְתָּ / וְלא אָבָה שְׁמֹעַ בְּקוֹלוֹ / וְהַשֻּׁעָל הָלַךְ בַּדֶּרֶךְ לְמוּלוֹ / עַד בָּאוּ אֶל עֵץ יַעַר גְּבַהּ קוֹמָה / וַיִּמָּלֵא הַשֻּׁעָל חֵמָה / כִּי הַנֶּשֶׁר הִשְׁלִיכוֹ אֶל קִנּוֹ לְגוֹזָלָיו / וַיַּרְא הַשֻּׁעָל כִּי לֹא יוּכַל לַעֲלוֹת אֵלָיו / וַיִּקְרָא לוֹ: לַאֲדִיב אֶת נַפְשֶׁךָ / אֶשְׂרֹף קוֹמַת אֲרָזֶיךָ וּמִבְחַר בְּרוֹשֶׁיךָ / וַיֵּלֶךְ הַשֻּׁעָל וַיָּשֶׂם אֵשׁ בְּכַדָּיו / צֶמֶר וּפִשְׁתִּים יַחְדָּיו / עֲלֵי אִילָן וְעֵצִים יְבֵשִׁים / וּנְעֹרֶת וְקוֹצִים קָשִׁים / וְלַמְּדוּרָה עֵצִים הַרְבֵּה / וְהָאֵשׁ תּוּקַד בּוֹ לֹא תִכְבֶּה / וְשִׁלַּח אֶת בְּעֵרוֹ וּבִיעֵר בַּצֶּמֶר וּבַפִּשְׁתִּים / וַיֹּאמַר: תָּמוֹת נַפְשִׁי עִם פְּלִשְׁתִּים / וְיִשָּׂרֵף בְּנִי עִם הַנֶּשֶׁר וּבָנָיו / בְּטֶרֶם יִהְיֶה לְמַאֲכָל בֵּין שִׁנָּיו / יִהְיֶה עִמָּהֶם כָּלָה / הִנֵּה לָאֵשׁ נִתַּן לְאָכְלָה / וּבִרְאוֹת הַנֶּשֶׁר הָאֵשׁ הַגְּדוֹלָה / וַיִּזְעַק זְעָקָה גְדוֹלָה וּמָרָה / וַיִּקְרָא לַשֻּׁעָל: קַח אֶת בִּנְךָ אֲשֶׁר נַפְשׁוֹ לְךָ יְקָרָה / וְכַבֵּה אֶת הָאֵשׁ אֲשֶׁר הִצַּתָּהּ בַּחֲרוֹנֶךָ / לָמָּה נָמוּת לְעֵינֶיךָ.

וְהַמָּשָׁל לְמִתְהַלֵּךְ בַּאֲשָׁמָיו / לִשְׁלֹחַ יָד בִּשְׁלוֹמָיו / חֹמֵד וְחֹמֵס גֹּזֵל וְלֹקֵחַ / יַבְאִישׁ יַבִּיעַ שֶׁמֶן רוֹקֵחַ / לְבַעֲבוּר כִּי הוּא חָזָק וְעָשִׁיר וְעֶלְיוֹן / שֹׁלֵל וּבֹזֵז עָנִי וְאֶבְיוֹן / גַּם כִּי תַּחֲנוּנִים יְדַבֵּר לוֹ רָשׁ בִּלְשׁוֹנוֹ / כְּמוֹ פֶתֶן חֵרֵשׁ יַאְטֵם אָזְנוֹ / לְסוֹף נִבְעוּ מַצְפֻּנָיו / וַיִּקְטַן גַּם הוּא בְּעֵינָיו.

וָאֶשָּׂא מְשָׁלִי וָאֹמַר:


גֶּבֶר יְקָרֶה גְבַר / שָׁפָל וְרָשָׁע לְבַר

עֵת יִתְקְפוֹ זֶה וְהוּא / יֵרֵד וְיַחְלִיק דְּבַר

פִּיו: נָא אֲדֹנִי לְחֵטְא / אַל תִּפְקֹד כִּי כְבָר

נִחַמְתִּי, וֶאֱהִי / לָךְ אָב וְאַתְּ לִי לְבָר.
