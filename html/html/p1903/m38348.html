משל צ: זְבוּב וָשׁוֹר

יִתְדַמֶּה חָבֵר חַלָּשׁ לְחַבָּרִים / וְאִם הָיוּ חֲבֵרָיו גִּבּוֹרִים


זְבוּב מִתְהַלֵּךְ בַּשָּׂדֶה וַיַּרְא הַשּׁוֹר / אֲשֶׁר עַל צַוָּארוֹ הָאִכָּר מֹסֵרוֹת יִקְשׁוֹר / וַאֲגֻדּוֹת מוֹטָה וְחַרְצֻבּוֹת / וּמִלְּהַצִּיב הַדָּרְבָּן לְצַלְעוֹתָיו לֹא יִשְׁבּוֹת / וְהַשּׁוֹר לַחֲרֹשׁ שָׁת אֶת פָּנָיו / וְהַזְּבוּב הָלַךְ וַיֵּשֶׁב לוֹ בֵּין קַרְנָיו / וְאִם הַשּׁוֹר עַל תַּלְמֵי שָׂדַי הֹלֵךְ וָשָׁב / הַזְּבוּב תָּמִיד עַל קַרְנוֹ יָשָׁב / וַתַּבֵּט אֵלָיו דְּבוֹרָה קְרוֹבָתוֹ / וַתֵּתַצֵּב מֵרָחֹק לִרְאוֹתוֹ / מַה בֵּין קַרְנֵי הַשּׁוֹר דֹּרֵשׁ / הֲכֹל הַיּוֹם יַחֲרֹשׁ הַחֹרֵשׁ / וַתֹּאמֶר הַדְּבוֹרָה: הֲקַרְנֵי הַשּׁוֹר לְךָ לְמַחֲנָיִם / לָמָּה יָשַׁבְתָּ בֵּין הַמִּשְׁפְּתַיִם / וַיַּעַן הַזְּבוּב: דְּעִי כִּי כָל הַשָּׂדֶה אֲשֶׁר לְפָנֵינוּ / אֲנִי וְהַשּׁוֹר חֲרַשְׁנוּהוּ בְכֹחֵנוּ / וַעֲשִׂי כָמוֹנוּ אִם לָךְ הַגְּבוּרָה / עוּרִי עוּרִי דְּבוֹרָה.


וְהַמָּשָׁל בְּשָׁפָל הַהֹלֵךְ בֵּין הַגִּבּוֹרִים / אוֹ לְעַוָּל הַנִּסְפָּח בְּמַחֲנֵה הַיְשָׁרִים / וּבְסוֹדָם וּבִגְבוּרָתָם אֵינֶנּוּ עֹמֵד / אַךְ בְּמַאֲמַר פִּיו אִתָּם יִצָּמֵד / לְהַשְׁווֹת גְּבוּרָתוֹ לִגְבוּרָתָם / וְחָכְמָתוֹ לִתְבוּנָתָם / וְיֹאמַר: כֵּן עָשִׂינוּ בִגְבוּרָתֵנוּ / וְכֵן פָּעַלְנוּ בְחָכְמָתֵנוּ.
