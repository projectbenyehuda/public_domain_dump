משל קיג: אִישׁ וּזְאֵב

אֲשֶׁר עֵינוֹ וְלִבּוֹ עַל בִּצְעוֹ / פִּיו יַגִּיד רִשְׁעוֹ

אִישׁ לַזְּאֵב אוֹתִיּוֹת מְאַלֵּף / וַיֹּאמֶר לוֹ: אֱמֹר אָלֶ״ף / זְאֵב אַחֲרָיו אָלֶ״ף עָנָה / עוֹד אָמַר: בֵּית אֱמֹר נָא / הַזְּאֵב מוֹצָא שְׂפָתָיו שָׁמַר / בֵּי״ת וְגִמֶּ״ל כָּמוֹהוּ אָמַר / וַיֹּאמֶר הָאִישׁ: שְׁמַע נָא בְאָזְנֶיךָ / אֶת אֲשֶׁר אֶעֱרֹךְ לְפָנֶיךָ / וְתַכִּיר הָאוֹתִיּוֹת לְחַבֵּר / לִמְצֹא חֶפְצְךָ וְדַבֵּר / וְעֵת תְּחַבֵּר אוֹתָם יָחַד / וְהָיִינוּ לְעַם אֶחָד / אָלֶ״ף בֵּי״ת כַּאֲשֶׁר אֶעֱשֶׂה / זְאֵב עָנָה: הִנֵּה הַשֶּׂה.

וְהַמָּשָׁל בַּאֲשֶׁר עֵינוֹ וְלִבּוֹ עַל בִּצְעוֹ / פִּיו יַגִּיד רִשְׁעוֹ / וּשְפָתָיו יַעֲנוּ בוֹ / לְהִתְגַּלּוֹת זְדוֹן לִבּוֹ / וְתֵצֵא מִבִּטְנוֹ הָרְשָׁעָה / עֵת תִּמָּצֵא בְפִיו רָעָה / וְיֵשׁ מַחֲשַׁבְתּוֹ נִכֶּרֶת מִתּוֹךְ מַעֲשָׂיו / מֹאֵס בְּיַעֲקֹב וּבֹחֵר בְּעֵשָׂו / וְצַדִּיק צִדְקוֹ יְקַדֵּם / וְסֶלֶף בֹּגְדִים יְשָׁדֵּם / וּמִשְׁלֵי שְׁלֹמֹה לֹא יִגְרַע / הֲלוֹא יִתְעוּ חֹרְשֵׁי רָע.
