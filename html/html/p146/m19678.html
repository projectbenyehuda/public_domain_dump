יְרוּשְׁלַם אַנְהַרִי אֲרֵי מְטָא זְמָן פֻּרְקָנִיךְ

החתימה: שלם


יְרוּשְׁלַם אַנְהַרִי / מְטָא זְמָן פֻּרְקָנִיךְ

וְתֻשְׁבְּחָן סַדַּרִי / לֶאֱלָהּ רֻחְצָנִיךְ

וּבַשִּׁיר אֵימַרִי / לִדְדָּאֵין דִּינִיךְ

אֲרֵי כָל עַמְמַיָּא / יְתוּבוּן לְפֻלְחָנִיךְ:


שַׁבַּחִי בְחֶדְוָא / פֻרְקָנִיךְ מִתְגְּלֵי

דָּכְרָנִיךְ בְּרַעֲוָא / עָל לְתַרְעָא דְּהֵיכְלֵי

וְעַמִּיךְ דַּאֲבִיל הֲוַא /יְתוּב בְּנִבְליִן וְצִלְצְלֵי

וִיהוֹן מִזְדָּמְנִין / כֻּלְּהוֹן לְשֻׁלְחָנִיךְ:


לְבוּשׁ דִּיקָר יִלְבְּשׁוּן / עַמִּיךְ דְּקַדִּישִׁין

וְשָׂנְאֵיהוֹן יִכְבְּשׁוּן / כָּל עַמְמִין בִּישִׁין

וּמַלְכֵיהוֹן יַחְבְּשׁוּן / יְהוֹן כְּעַפְרָא כְּתִישִׁין

וְּכָל שַׁלִּיטֵיהוֹן / יְתוּבוּן יְשַׁמְּשׁוּנִּיךְ:


מְשַׁעְבְּדֵךְ בְּגָלוּתִיךְ / יֵיתוֹן לִיךְ זְקִיקִין

יְהוֹן מְחַמְּדִין לְדָתִיךְ / וּבֵיהּ יִהְוְיָן חֲשִׁיקִין

וְיִשְׁתַּעְבְּדוּן לְוָתִיךְ / וְלָא יִשְׁכְּחוּן פְּרִיקִין

וּשְׁלָמָא יִשְׁתְּמַע / מִכָּל סַחְרָנִיךְ:


יֵתוֹן קָרְבָּנְהוֹן / לְגַוִּיךְ עַמְמַיָּא

יְהוֹן עָלִין כֻּלְּהוֹן / בְּיוֹמֵי חַגַּיָּא

אִנּוּן וְשִׁלְטוֹנְהוֹן / וְכָל אַמַּרְכְּלַיָּא

וְדַהֲווֹן סָנְאָן לִיךְ / יְתוּבוּן יְרַחֲמוּנִיְך:


תַּמַּן יְהֵי נְיָחָא / לְיִשְׂרָאֵל כֻּלְּהוֹן

וְיִחְזוֹן יָת מְשִׁיחָא / וְתֶסְגֵּי חֶדְוַת לִבְּהוֹן

יְקָרְבוּן עַל מַדְבְּחָא / יָת קֻרְבָּנֵיהוֹן

וְכָסָא דְּפֻרְקָנָא / אֱלָהִיךְ יַשְׁקֵינִיךְ:
