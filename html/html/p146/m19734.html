שִׁירִין שְׁמַע תָּקְנַיָּא אַתְּעַר לְחַבְרַיָּא

החתימה: שלם (36: אבן יוסף)


שִׁירִין שְׁמַע תָּקְנַיָּא / אַתְּעַר לְחַבְרַיָּא

לֵית סוֹף נְהוֹרָא דְדָכְיָא / עִלַּאי וְתַתַּיָּא

שְׁפַע מַלְכְּתָא דָּא דְשָׁרְיָא / וּסְעוֹדְתַהּ סַגְיָא

פָּתוּר לְכוֹלָּא מְשַׁוְיָא / וּלְחֵים דְאַפַּיָּא

כָּל חַד וְחַד מַסְחַר וְנַפְשֵׁיהּ תִּסְבַּע

וּמְבָרְכִין לְמַלְכָּא דְכוֹלָּא דִקְבַע

וּלְאַרְבְּעָה דְּעִלַּאי תַּבַּע

מְטַטְרוֹן בְּרֹאשׁ מַטְּרַיָּא / סָבֵיב לְדִגְלַיָּא

וּמְקַבְּלִין גִּלְגְּלַיָּא / עֶסְרָא כְּבוּלַיָּא:


לִכַׄדְמַת אַלְחַקּ תֻּסאַרִע / כֻּלַּן בִּמַא טֻבִּע

עַאשִׁק לְמַעְשׁוּק תַּאבִּע / וַאלנּוּר תַׄם סַאטִע

וְכַׄמְס מִאַת עַאם וַאסִע / בַּין דַּא וַדַׄא רַאפִע

וַאלעַקְל פִיהֻם מֻצִיַּא / לִאלנּוּר יַתהַיַּא

דַוְרַאתהֻם דַאִים פַלַא תַתְמַיַּל

פִי נוּר מַבְּסוּטַן מְגַׄוְהַר יַשְׁעַל

תַמְזֻגׄ צִׄיַאהַא דִׄי מַן אַעְלַא תַקְבַּל

תְנִיר לְדַׄא אַלְעַקְל אִגִיַא / וַאלנַּפְס פִיה תַחְיַא

תֻּשְׁרִק בַּהוּ דִין וְדֻנְיַא / פִי דַׄאלִךּ אַלְאַחְיַא:


מִזִּיו מְרוֹם הֶאֱצִילָם / שַׁדַּי וְהִבְדִּילָם

יַרְעִים שְׁמֵי רוֹם לְקוֹלָם / מִמַּעֲלוֹת סֻלָּם

מַשְׁלִים וְחוֹזֵר חֲלִילָם / נוֹעֵע בְּעִגּוּלָם

עֲרָבוֹת לְרוּחִין דְּחֶדְיָא / מָרֵי זְכוּתַיָּא

אַתְרָא דְּצַדִּיקִין בְּשׁוּפְרָא, קַשִּׁיש

דִּזְמִין לְהַאי דֻּכְתָּא וְלָא עָבֵיד בִּישׁ

דְּאִיהוּ כְּבַר יוֹחַאי וּכְבַר לַקִּישׁ

וְדֻּכְתָּא חֲדָא אֶמְצְעַיָּא / לְמָרֵי חֲסִידַיָּא

וְעוֹד אִית סְגוֹ הֵיכְלַיָּא / עִילַּאי וְשִׁפְלַיָּא:


מִן דַׄא אַלבַּשַׁר תַנְצֻׄד אַשְׁיַאת / פִי חַדְּהַא מַחְדּוּד

אִנְסַאן וּמַעְדִן וְאַנְבַּאת / חַיוַאן תַׄם מַפְרוּד

וִאכׄתַץּ אַלְאִנְסַאן פִי אַללִּדַׄאת / הוּ גַאיַת אַלְמַקְצוּד

חִין נַסְל אַדַם תִהַיַּא / עַן אַלבְּהִימִיַּא

זַין אַלְאֻמַם הוּ נַסְל אַדַם קֻרִבּ

מִן מַצְר אַכְׄרַגְׄהֻם בִּאַמְרֻן גַאלִבּ

אַרַקּ לִפַרְעוּן הוּ וְקַוְמֹה אַלְמֻדְׄנִבּ

פִי יַד מוּסַי אַלְנַּבִּיַא / בִּמֻעְגִׄז אִלַאהִיַּא

קַאל אִבְּן יוּסֻף יֻחַיַּא / מִן לַדׄ רֻמוּזִיַּא:
