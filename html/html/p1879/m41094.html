אֲשׁוֹרֵר שִׁיר

עוד לו שלח אותו להנשיא שי׳יצחק דוד ישועה גבאי

סימן: אני צאלח


אֲשׁוֹרֵר שִׁיר יָפֶה / לְכָל בָּשָׂר מַרְפֵּא / בְּעֶזְרַת אֵל צוֹפֶה / וּמַבִּיט מִסְתָּרִים

נְאוּם גֶּבֶר גּוֹבֵר / בְּטוּרֵי שִׁיר חִבֵּר / בְּצֶדֶק הוּא דוֹבֵר / וְלֹא גֻזְמַת שָׁרִים

יְבֻשַּׂם לָךְ שִׁירִי / פְּאֵרִי גַם נִזְרִי / כְצִפֹּרֶן וּצְרִי / בְּיוֹם הַכִּפּוּרִים

צְבִי עֹפֶר הַטּוֹב / רְדֹף אוֹיְבָךְ וַחְטֹב / תְּהִי תָמִיד רָטֹב / לְיַלְדֵי הָעִבְרִים

אֲדַמֶּה הַמּוּסָר / לְקַב אַחַד עָשָׂר / עֲשֶׂרֶת לָךְ הַשַּׂר / וְאֶחָד לִיצוּרִים

לְכָה כּוֹכַב צֶדֶק / וְהַחְזֵק כָּל צֶדֶק / לְאִישׁ דּוֹבֵר צֶדֶק / וּמַגִּיד מֵישָׁרִים

חֲנֵה תָמִיד עִמּוֹ / בְּשָׁכְבוֹ וּבְקוּמוֹ / לְמַתֵּק אֶת טַעְמוֹ / וְאֶת נִסּוֹ תָרִים

אֲנָשִׁים יַרְחִיקוּ / כְּלֵיהֶם יָרִיקוּ / בְּעֶבֶד יַסְפִּיקוּ / בְיַלְדֵי הַנָּכְרִים

שְׂפָתֶיךָ שָׁבוּ / אֲנָשִׁים נָקָבוּ / לְעָבְדָךְ יִתְאָבוּ / וְלֹא יֵצְאוּ חוֹרִים

לְבָבִי שָׁבִיתָ / בְּשֵׂכֶל שָׂרִיתָ / וְאוֹתִי קָנִיתָ / לְעֶבֶד לְדוֹרִים

לְפָאֵר וּלְהַלֵּל / בְכָל יוֹם וּבְכָל לֵיל / וּבִשְׁמָךְ אֶתְפַּלֵּל / עֲרָבִים וּשְׁחָרִים

וְאַעְתִּיר אֶל קוֹנָךְ / לְהַרְבּוֹת אֶת הוֹנָךְ / וְהוּא יִתֵּן חִנָּךְ / בְּעֵינֵי הַשָּׂרִים.
