יאֲשׁוּנִי מֵרַחֲמֶיךָ

חתימה: יהודה הלוי


יִאֲשׁוּנִי מֵרַחֲמֶיךָ גּוֹיִים שְׁכֵחֵי אֱלֹהִים:


״הַעוֹד תַּעֲמֹד עַל מִצְפֶּה, / תְּיַחֵל מַרְפֵּא

וּפִתְחוֹן פֶּה – / גּוֹי נִדְכֶּה וְנִסְפֶּה,

נָגוּעַ מֻכֵּה אֱלֹהִים?


וּמַה תִּקְוַת גּוֹי מְפֹרָד וּמְפֻזָּר, / נָכְרִי וּמוּזָר,

אוֹהֲבוֹ נִגְזָר / וְאוֹיְבוֹ נֶעֱזָר –

אֵין יְשׁוּעָתָה לּוֹ בֵאלֹהִים?


דְּבִירוֹ חָרַב / וְיוֹמוֹ עָרַב / וְאֵידוֹ קָרַב,

וּבֵיתוֹ נָשַׁם / וְאָשֹׁם אָשַׁם / הַהֵיכָל, אֲשֶׁר שָׁם

נִגְלוּ אֵלָיו הָאֱלֹהִים.


הֶבֶל יִתְעַנֶּה / וְיַרְחֵי שָׁוְא יְמֻנֶּה

וְלָרִיק יִפְנֶה – / וְאֵין קוֹל וְאֵין עוֹנֶה,

כִּי אֵין מַעֲנֵה אֱלֹהִים!״


הֵשִׁיבוּ שְׂעִפַּי / וַיָּרִיבוּ שַׂרְעַפַּי

לְהָשִׁיב דָּבָר כְּנֶגְדּוֹ: / ״הֲגַם אֶת הַטּוֹב לְבַדּוֹ

נְקַבֵּל מֵאֵת הָאֱלֹהִים?


לֹא לָנֶצַח יָרִיב / וְאִם הֶחֱרִיב – יַרְחִיב,

וְאִם הִרְחִיק – יַקְרִיב / וְיַקְשִׁיב לְהָשִׁיב –

מִי יוֹדֵעַ יָשׁוּב וְנִחַם הָאֱלֹהִים.


וַהֲלֹא מִמִּצְרַיִם גְּאָלָנוּ / וּמִמַּעֲמַקֵּי יָם הֶעֱלָנוּ

צוּר אֲבוֹתֵינוּ וְעֶזְרָתָם, / קְרָאוּהוּ בְּעֵת צָרָתָם

וַתַּעַל שַׁוְעָתָם אֶל הָאֱלֹהִים״.


יַחֵל יִשְׂרָאֵל פִּדְיוֹן

מֵאֵל עֶלְיוֹן, / שׁוֹכֵן צִיּוֹן –

מִי יוֹדֵעַ יָשׁוּב וְנִחַם הָאֱלֹהִים.
