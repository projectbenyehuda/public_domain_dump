[בְּכִי נַפְשִׁי אֲשֶׁר שָׁחָה]

בְּכִי נַפְשִׁי אֲשֶׁר שָׁחָה / אֱלֵי עָפָר, וְנִזְנָחָה,

אֲשֶׁר רַבּוּ כְאֵבֶיהָ / בְּאַיָּלָה [מְשֻׁלָּחָ]ה.

אֲשֶׁר הִטָּת לְבָבֶךָ / בְּעֵינֶיהָ, וְהִדִּיחָה.

מְשָׁכַתְךָ בְּחַבְלֵי שָׁוְא / לְפַח יָקוּשׁ וְגַם שׁוּחָה.

וְשָׁגִיתָ בְּרֹב חֶלְקַת / שְׂפָתֶיהָ בְּעֵת שָׂחָה:

"לְכָה דוֹדִי, עֲלֵה נָא אֶל / עֲלִיַּת קִיר מְרֻוָּחָה,

"וְשָׁם נִרְוֶה אֲזַי דּוֹדִים / עֲדֵי בֹקֶר, וְנִשְׁמָחָה!

"וְשֻׁלְחָנִי לְךָ עָרוּךְ, / וְנַפְשִׁי לָךְ מְשֻטָּחָה;

אֲשֶׁר תִּתֵּן יְגִיעֶךָ / לְאַכְזָרִים – וְהִיא נָחָה.

וְאַתְּ תֵּדַע וְתָבִין כִּי / לְךָ תָפֵל אֲשֶׁר טָחָה.

וְאַל תִּהְיֶה יְחִידָתָךְ / בְּחֶרֶב דִּין מְנֻתָּחָה.

בְּ[שֶׁמֶן] מוֹר מְרֻקָּחִים, / וּמִקְטַרְתּוֹ מְמֻלָּחָה.

פְּרַע אָחִי נְתִיבָתָהּ, / מְשׁוּבָה הִיא וְנִצָּחָה.

וְאֶל נָכְרִי עֲצָבֶיךָ – / וְהִיא שָׂשָׂה וְשָׂמֵחָה.

יְרָא הָאֵל, וְשׁוּב אָחִי, / לְדַרְכּוֹ הַמְשֻׁבָּחָה.
