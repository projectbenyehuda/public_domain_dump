משל לד: צְפַרְדֵּעַ וּשְׁוָרִים

הַמִּתְפָּאֵר בִּפְעֻלָּתוֹ / יִזָּהֵר פֶּן תִּגָּלֶה עָלָיו עֶרְוָתוֹ


צְפַרְדֵּעַ רָאֲתָה שְׁוָרִים חֹרְשִׁים / אֵצֶל קָנֶה וְגֹמֶא אָהֳלָהּ נִגָּשִׁים / וַתְּמַהֵר לָלֶכֶת לִקְרָאתָם / וַתִּשְׁאַל שְׁמָם וְשֵׁם אֲבוֹתָם / כִּי הִיא לְהַכִּירָם רוֹצָה / וַתֹּאמַר: לִי הַחָכְמָה וְהָעֵצָה / לִמְצֹא לְכָל פֶּצַע וּמַחֲלָה / מָזוֹר וּרְפֻאוֹת תְּעָלָה / בְּמַרְאוֹת הַשֶּׁתֶן וּבְדֹפֶק הַזְּרֹעַ / אַכִּיר אִם לַטּוֹב אִם לָרֹעַ / חוֹלִים לָרֹב הוֹצֵאתִי לָרְוָיָה / כַּאֲשֶׁר פָּתַרְתִּי כֵן הָיָה / וְשׁוֹר זָקֵן הִבִּיט אֶל אוֹתָהּ / הַמִּתְהַלֶּלֶת בְּרֹב חָכְמָתָהּ / וַיֹּאמֶר לָהּ: אִם דְּבָרֵךְ אֱמֶת / לָמָּה מַרְאֵךְ דּוֹמֶה לְמֵת / מִכָּל רֶמֶשׂ תָּאֳרֵךְ מָשְׁחָת / כַּאֲשֶׁר מָחָר תֵּרְדִי שָׁחַת / וְחָכְמָה נִכֶּרֶת בְּמַעֲלָלֶיהָ / הַחָכְמָה תְּחַיֶּה בְעָלֶיהָ / אֵין בָּךְ אֹדֶם רַק כֹּל יָרֹק / מִפָּנַיִךְ לֹא אֶחְשֹׂךְ רֹק / וְחָפְרָה הַצְּפַרְדֵּעַ וּבוֹשָׁה / וַתָּשֶׂם בֵּין שְׁתֵּי יַרְכוֹתֶיהָ רֹאשָׁהּ / וַתֵּלֶךְ לְהֵחָבֵא בִּמְרוּצָה / בְּסֵתֶר קָנֶה וּבִצָּה.

וְהַמָּשָׁל בְּמִתְפָּאֵר בְּחָכְמָתוֹ וּבִפְעֻלָּתוֹ / יִזָּהֵר פֶּן יוֹכִיחוּ עָלָיו חֶרְפָּתוֹ / כִּי טוֹב לְהַחֲרִישׁ עַד יַגִּידוּ אֲחֵרִים הוֹדוֹ / מִלְּהָמִיר בְּקָלוֹן אֶת כְּבוֹדוֹ / וְיֵשׁ אֲשֶׁר אַחֵר יְהַלֲלֶנּוּ / וְלִפְנֵי גְדוֹלִים יַנְחֶנּוּ / וְאָמַר הֶחָכָם: יְהַלֶּלְךָ בְּבֹאֲךָ וּבְצֵאתֶךָ / זָר וְלֹא פִיךָ נָכְרִי וְאַל שְׂפָתֶיךָ.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
