משל מח: שֻׁעָל וְעָגוּר

לְיַשֵּׁר מְסִלַּת פָּעְלוֹ / יִתְחַבֵּר עִם בֶּן גִּילוֹ

הַשֻּׁעָל אָמַר לֶעָגוּר: לָמָּה כָּל־עֵת אַתָּה סָגוּר / בֹּא וּסְעַד לִבְּךָ עִמִּי / וּרְאֵה בֵיתִי וּבְנֵי עַמִּי / וַיְסִיתֵהוּ לָבֹא עִמּוֹ / לִשְׂמֹחַ בּוֹ וְלֶאֱכֹל בְּלַחְמוֹ / וְאֶל הַבָּשָׂר הַשֻּׁעָל רָץ / וּבְפֶתַע בַּהֲמוֹן תַּרְנְגֹל פָּרָץ / וַיַּךְ תַּרְנְגֹלֶת אֲשֶׁר מוּל הַשֶּׁמֶשׁ סָגְרָה עַפְעַפֶּיהָ / תִּתְיַפֵּחַ תְּפָרֵשׂ כְּנָפֶיהָ / וְהַמַּאֲכָל בְּעֵת הֱכִינוֹ / חֹשֵׁב בְּתַחְבֻּלּוֹת הִנּוֹ / וַיֹּאמֶר בְּלִבּוֹ: הִנְנִי סָכַל / אִם מֵאֲשֶׁר הֲכִינוֹתִי לֹא אֹכַל / וְהַמַּאֲכָל מָתוֹק לְחֵךְ / וַאֲנִי לֹא אֶהְיֶה רַק לֹחֵךְ / וְהֶעָגוּר בְּפַעַם אֶחָד / יִבְלַע כָּל הַמַּאֲכָל יַחַד / וַיִּוָּעֵץ בְּעָרְמָתוֹ / אֲשֶׁר הָיְתָה קְרוֹבָתוֹ / וְהַמַּאֲכָל דַּק דַּק טָחַן / וַיִּפְרְשֵׂהוּ עַל הַשֻּׁלְחָן / וַיִּקְרָא הֶעָגוּר לֶאֱכֹל / וְלָקַחַת בְּפִיו וְלֹא יָכֹל / כִּי לֶאֱכֹל דַּק מְפֻזָּר וּמְפֹרָד לֹא נִסָּה / וְעֵין הַשֻּׁעָל עָלָיו לֹא חָסָה / וַיַּשְׁאִיר אֶת הַשֻּׁלְחָן רֵיק / כִּלְחֹךְ הַשּׁוֹר אֶת יֶרֶק / וְהֶעָגוּר שָׁב יָגֵעַ וְיָעֵף / לְבֵיתוֹ רָעֵב וְסָר וְזָעֵף / וְלֹא דִּבֵּר עִם הַשֻּׁעָל / לֹא בְמֶרֶד וְלֹא בְמַעַל / אַךְ בְּכִלְיוֹתָיו יִשְׁתּוֹנֵן / וְלִנְקֹם נִקְמָתוֹ כּוֹנֵן / וּבִהְיוֹתוֹ בְּתַחְבֻּלּוֹת יוֹעֵץ / וַתּוֹרֵהוּ מְזִמָּתוֹ עֵץ / מָלֵא נְקָבִים בְּכָל עֲבָרָיו / וַיְמַלֵּא טֶרֶף חוֹרָיו / וְהַמַּאֲכָל אֲשֶׁר הֵכִין בַּנֶּקֶב יִתְקָעֶנּוּ / עָמֹק עָמֹק מִי יִמְצָאֶנּוּ / לוֹ יִהְיֶה אֵין זָר אִתּוֹ / וַיִּקֹּב חוֹר בְּדַלְתּוֹ / דֶּרֶךְ אֲשֶׁר יָבֹא בִּגְבוּלוֹ / שָׁם מִנָּה אֶת מַאֲכָלוֹ / וַיִּקְרָא אֶל הַשֻּׁעָל וַיַּגִּיעֵהוּ / לָבֹא עִמּוֹ וַיַּרְאֵהוּ / הַמַּאֲכָל וַיֹּאמֶר לוֹ: הֲרֵי שֶׁלְּךָ לְפָנֶיךָ / אֱכֹל כַּטּוֹב בְּעֵינֶיךָ / וְהַשֻּׁעָל לֹא יוּכַל שָׁם לִלְחֹךְ בִּלְשׁוֹנוֹ / לַחָפְשִׁי יְשַׁלְּחֶנּוּ תַּחַת שִׁנּוֹ / לֹא בַלָּשׁוֹן וְלֹא בָרֶגֶל / וְעַל זֶה יִשְׂמַח וְיָגֵל / הֶעָגוּר אֲשֶׁר לְהוֹצִיא יוּכַל / וּבְתַאֲוָתוֹ לְעֵינֵי הַשֻּׁעָל אָכַל / וְהֵשִׁיב לַשֻּׁעָל כִּגְמוּלוֹ / כַּאֲשֶׁר עָשָׂה כֵּן יֵעָשֶׂה לוֹ.

וְהַמָּשָׁל בִּמְבַקֵּשׁ תַּחְבֻּלּוֹת / וְסִבּוֹת וַעֲלִילוֹת / לְהוֹנוֹת אֶת רֵעֵהוּ / וְזֶה תָם וְיָשָׁר הוּא וְיוֹשֵׁב לָבֶטַח אִתּוֹ / בִּפְעֻלָּתוֹ וּבְמַתְכֻּנְתּוֹ / זֶה יִרְקַח כָּמֹהוּ / בַּדֶּרֶךְ אֲשֶׁר הוֹרָהוּ / אֶשְׁתּוֹלֲלוּ אַבִּירֵי לֵב / וְטָפַשׁ לִבָּם כַּחֵלֶב / חֹשֵׁב לִמְצֹא טוֹב וְעֹשֶׂה רָע / אֵיךְ יִקְצֹר חִטִּים קוֹצִים זָרַע / חֹרֵשׁ עַל רֵעֵַהוּ אֵידוֹ / כְּלִי מַפָּצוֹ בְּיָדוֹ.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
