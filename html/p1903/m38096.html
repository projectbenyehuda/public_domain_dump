משל לה: עַכְבָּר וְחוֹר

אֲשֶׁר עַל הוֹן זָרִים שָׁלַט / כְּבוֹלְעוֹ כָּךְ פָּלַט


עַכְבָּר דַּל בָּשָׂר וְשָׁחוֹר / בָּא בַגֹּרֶן בְּעַד הַחוֹר / וּבַתְּבוּאוֹת שִׁלַּח שֵׁן / וְאָכַל וְשָׂבַע וְדָשֵׁן / שָׁם כִסָּה בְחֶלְבּוֹ פָנָיו / וְקָטָנּוֹ עָבָה מִמָּתְנָיו / אֲשֶׁר הֵבִיא שָׁם בַּתְּחִלָּה / בִּרְצוֹתוֹ לָשׁוּב בַּמְּסִלָּה / וְלֹא יָכֹל לָצֵאת בְּשׁוּבוֹ / דֶּרֶךְ הַחוֹר אֲשֶׁר בָּא בוֹ / וַתַּבֵּט אֵלָיו הַחֻלְדָּה / וַתֹּאמַר: הַחוֹר לֹא לְךָ בַמִּדָּה / עַכְבָּר מַה זֹּאת עָשִׂיתָ / שָׁמַנְתָּ עָבִיתָ כָּשִׂיתָ / בְּטֶרֶם תָּשׁוּב בַּדֶּרֶךְ הַזֶּה / מִשְׁמַן בְּשָׂרְךָ יֵרָזֶה / לֹא תִרְאֶה עוֹד אֲבוֹתֶיךָ אֲשֶׁר יָדַעְתָּ / אִם לֹא תָקִיא אֲשֶׁר בָּלַעְתָּ / וְשֹׁמְרֵי הַגֹּרֶן שְׁמָעוּהוּ / רְדָפוּהוּ הִכּוּהוּ פְּצָעוּהוּ.

וְהַמָּשָׁל בְּאִישׁ אֲשֶׁר עַל הוֹן זָרִים לִבּוֹ הִשְׁלִיטוֹ / כְּבוֹלְעוֹ כָּךְ פּוֹלְטוֹ / חַיִל בָּלַע וַיְקִיאֶנּוּ / בַּחֲצִי יָמָיו יַעַזְבֶנּוּ.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
