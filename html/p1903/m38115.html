משל נד: אֶרֶז וּסְנֶה

אַל יִתְגָּאֶה אִישׁ בְּעָשְׁרוֹ / כִּי לֹא לָנֶצַח יָהֵל אוֹרוֹ


הָאֶרֶז אֲשֶׁר בַּלְּבָנוֹן / לִפְנֵי שֶׁמֶשׁ שְׁמוֹ יִנּוֹן / וַיִּתְגָּאֶה בְסַרְעַפָּיו וּבְפֹארֹתָיו / בְּגָבְהוֹ וּבְרֹב דָּלִיּוֹתָיו / וַאֲשֶׁר לֹא הָיָה בַיַּעַר כָּמֹהוּ / וַאֲרָזִים לֹא עֲמָמֻהוּ / וַיַּבֵּט סְבִיבָיו בַּמַּחֲנֶה / וַיַּרְא וְהִנֵּה אֶצְלוֹ הַסְּנֶה / וַיֹּאמַר: מַה לְּךָ פֹּה כִּי בָאתָ בִגְבוּלִי / וְאֵיךְ לֹא יָרֵאתָ לִשְׁכֹּן בְּצִלִּי / הֲיֻקַּח מִמְּךָ עֵץ לַעֲשׂוֹת מְלָאכָה / עַל כֵּן יֵאוֹת לִי הַמְּלוּכָה / לַעֲשׂוֹת פְּרִי וְלָשֵׂאת עָנָף / וּבִי יִשְׁכְּנוּ בַעֲלֵי כָנָף / וְקָטָנִי עָבָה מִמָּתְנֶיךָ / וְעַל מִי נָטַשְׁתָּ חֲרֻלֶּיךָ וְקִמְשׁוֹנֶיךָ / כִּי אֵינְךָ רַק נֶטַע אֹרֶן / וּמִמֶּנִּי יַעֲשׂוּ מָשׁוֹט וְתֹרֶן / הָעֹמֵד לְנֵס עַמִּים / וְשָׁרְשְׁךָ שֹׁרֶשׁ רְתָמִים / וַיַּעַן הַסְּנֶה: בְּאִמְרֵי פִיךָ נוֹקַשְׁתָּ / כִּי לְהִתְגָּאוֹת בְּיָפְיִךָ דָרַשְׁתָּ / זְכֹר כִּי מְקוֹמְךָ שָדֶה תֵחָרֵשׁ / וְיַשְׁחִיתוּ אֶת עֵצֶיךָ עַד הַשֹּׁרֶשׁ / וּפֹארוֹתֶיךָ אֲשֶׁר הֵם רַעֲנַנִּים וְלַחִים / יִכְרְתוּ לִקְרָשִׁים וְעַמּוּדִים וּבְרִיחִים / וְאֶתְבּוֹנֵן עַל מְקוֹמְךָ וְאֵינֶנּוּ / אַךְ לֹא יַעֲלֶה הַכּוֹרֵת עָלֵינוּ / וְהוֹדְךָ לְמַשְׁחִית נֶהְפַּךְ עָלֶיךָ / לִכְרֹת עֵצְךָ וְלַנְפִּיל עָלֶיךָ / וּפִרְיְךָ אָז כָּל עוֹבֵר יְקוֹסֵס וִילַקֵּט / בְּעֵת אֶהְיֶה עַל שְׁמָרַי בֹּטֵחַ וְשָׁקֵט.


וְהַמָּשָׁל פֶּן יִתְגָּאֶה אִישׁ בְּיָפְיוֹ וּבְעָשְׁרוֹ / עַל שְׁכֵנוֹ וְאָחִיו אוֹ גֵרוֹ / כִּי הַכְּפִירִים אֲשֶׁר רָעֲבוּ וְרָשׁוּ / אֹתוֹ יוֹם יִדְרְשׁוּ / וְהוּא נֶחְבָּא לֹא יוּכַל / וְכָל יָמָיו בַּחֹשֶׁךְ יֹאכַל / וְהַדַּל אֲשֶׁר עָלָיו דִּבֵּר עָתָק וְגַאֲוָה / אוּלַי יֵשׁ תִּקְוָה.

וָאֶשָּׂא מְשָׁלִי:

בְּרֹאשׁ כָּפוּף וְלֵב נִשְׁבָּר וְעַיִן / שְׁפָלָה צַעֲדֶיךָ יִצְעָדוּ

רְאֵה אֵשׁ בַּעֲלוֹתוֹ סוֹף יְכֻבֶּה / וְהַמַּיִם בְּרִדְתָּם יַעֲמֹדוּ.



את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
