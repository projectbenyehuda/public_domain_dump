משל מ: דֹּב וְאַיָּלָה

אַל תִּיקַר נֶפֶשׁ אִישׁ בְּעֵינָיו / לְהִמָּלֵט תַּחַת בָּנָיו


דֹּב רָאָה אַיָּלָה הָרָה / חֲבָלֶיהָ אֲחָזוּהָ וַתְּהִי מְצֵרָה / וַיּתְבּוֹנֵן הַדֹּב כִּי הִגִּיעָה עִתָּהּ / לָלֶדֶת וַתְּקַשׁ בְּלִדְתָּה / וַיֹּאמֶר לָהּ בְּמִרְמָה / מַהֲרִי שִׁלְחִי חֲבָלַיִךְ / כִּי עֵינַי לְטוֹבָה אָשִׂים עָלַיִךְ / וְעַל הָאַיָּלִים אֲשֶׁר תֵּלְדִי אִם אֵדַע אַיֵּה / עִזְבִי בָנַיִךְ אֲנִי אֲחַיֶּה / וְאַמְצִיא לָהֶם דֵּי מַאֲכָלָם / וּבְרִית שָׁלוֹם בֵּינִי וּבֵינֵךְ עַד עוֹלָם / וְהַדֹּב דִּבֵּר בְּעָקְבָה לֶאֱכֹל גּוּרֶיהָ / אַף הִיא בְּחָכְמָה תָּשִׁיב אֲמָרֶיהָ / יָדַעְתִּי כִּי בְתָם לְבָבְךָ וּבְנִקְיוֹן כַּפֶּיךָ / נֶגְדִּי יֵישִׁירוּ עַפְעַפֶּיךָ / וְשָׂש אָנֹכִי עַל אִמְרָתֶךָ / וּפָרַשְׂתָּ כְנָפֶיךָ עַל אֲמָתֶךָ / אַךְ אֲדֹנִי לַעֲשֹׂת בַּקָּשָׁתִי אַחֲלַי / הִתְרַחֵק כִּמְטַחֲוֵי קֶשֶׁת מֵעָלַי / כִּי בְעֵת לֶדֶת חֶרְפָּה הִיא לָנוּ / אִם הַזְּכָרִים יַבִּיטוּ אֵלֵינוּ / לְכָה נָא מֵאִתִּי וְתִהְיֶה זָרִיז וְנִשְׂכָּר / אַל תַּבִּיט אַחֲרֶיךָ וְאַל תַּעֲמֹד בְּכָל הַכִּכָּר / לֹא נָכוֹן לַזְּכָרִים לִהְיוֹת כְּהַיּוֹם עִם נְקֵבוֹת בְּמִגְרָשָׁם / לֹא אוּכַל לַעֲשׂוֹת דָּבָר עַד בֹּאֲךָ שָׁם / וּכְבָר יָלַדְתִּי לוּלֵא עָצַרְתִּי לִי מֵרֹב בֹּשֶׁת / וַיֵּלֶךְ וַיֵּשֶׁב לָהּ מִנֶּגֶד הַרְחֵק כִּמְטַחֲוֵי קֶשֶׁת / וּבִרְאוֹתָהּ כִּי הִרְחִיק וַתָּנָס בְּרַגְלֶיהָ / וּבְעָרְמָתָהּ מִלְּטָה נַפְשָׁהּ וְנֶפֶשׁ בָּנֶיהָ.


וְהַמָּשָׁל אַל תִּיקַר נֶפֶשׁ אִישׁ בְּעֵינָיו / לְהִמָּלֵט מִמֹּקְשֵׁי מָוֶת וְלָתֵת תַּחְתָּיו בָּנָיו / וְהָאִשָּׁה אֲשֶׁר כְּרֵסָהּ בֵּין שִׁנֶּיהָ / אַל תִּיקַר נַפְשָׁהּ בְּעֵינֶיהָ / לֵאמֹר: מִי יִתֵּן וּבֶטֶן הַמְלֵאָה אָרִיק / כִּי עַד עַתָּה יָגַעְתִּי בְדֵי־רִיק / כַּאֲשֶׁר הָאַיָּלָה תָּשִׂים נַפְשָׁהּ בְּכַפָּהּ / לְמַלֵּט צֶאֱצָאֶיהָ בְּמִרְמָה וּבַחֲנֻפָּה.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
