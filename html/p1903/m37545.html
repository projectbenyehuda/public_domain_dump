משל ג: זְאֵב וְשֶׂה בְנָהָר

אִישׁ לְבִצְעוֹ יִפְנֶה / וְאִם דַּל הוּא אֶת אָחִיו יוֹנֶה

זְאֵב הָלַךְ לִשְׁבֹּר צְמָאוֹ עַל הַנָּהָר לְבַדּוֹ / וַיַּרְא וְהִנֵּה מֵרָחוֹק שֶׂה עוֹמֵד לְנֶגְדּוֹ / וַיִּקְרָא לוֹ הַזְּאֵב: לָמָה עֲכַרְתַּנִי / לִי הַיְאֹר וַאֲנִי עֲשִׂיתִנִי / וַיַּעַן לוֹ הַשֶּׂה תָמִים / דְּבָרִים טוֹבִים וּנְעִימִים: אֲדֹנִי הַמֶּלֶךְ לֹא עֲכַרְתִּיךָ / אָדוֹן לְרֹאשִׁי כָּל יָמַי נְתַתִּיךָ / וְלִשְׁתּוֹת מֵי נָהָר יָצָאתִי מֵאָהֳלִי / אִם רַע בְּעֵינֶיךָ אָשׁוּבָה לִי / וּלְהָשִׁיב עַל דְּבָרֶיךָ עֵת לַחְשׁוֹת / וַיְדַבֵּר הַזְּאֵב אִתּוֹ קָשׁוֹת / וַיֹּאמֶר לוֹ: מִי אַתָּה אֲשֶׁר מִמֶּנִּי לֹא תִפְחָד / הַאִם זְאֵב וְטָלֶה יִרְעוּ כְאֶחָד / אוֹ מִי הוּא אֲשֶׁר יַחְדָּו רָאָם / עַל אֲפִיקֵי מַיִם לִשְׁבֹּר צְמָאָם / מֵימַי עֲכַרְתָּ וְלִשְׁתּוֹת לֹא יָכֹלְתִּי / וְהִנֵּה אָשׁוּב צָמֵא עַל עֵקֶב בָּשְׁתִּי / אָבִיךָ הָרִאשׁוֹן לִי חָטָא / בְּפֹעַל וּבְמִבְטָא / זֶה אַרְבַּע שָׁנִים מָרַד וּפָשַׁע / וַיַּעַן הַשֶּׂה: לָמָה אֲדֹנִי מִתְעוֹלֵל עֲלִילוֹת בְּרֶשַׁע / וַאֲנִי שֶׂה קָטָן עַתָּה בָאתִי / וַעֲדַיִן לֹא הִגִּיעָה שְׁנָתִי / מַה תַּחֲרֹשׁ בְּרֵעֲךָ דִּבְרֵי רִיבוֹת / בָּנִים לֹא יוּמְתוּ עַל אָבוֹת / וַיַּעַן לוֹ: כַּאֲשֶׁר אֲנִי בַעֲבוּרְךָ צָמֵא וְנִכְלָם / לֹא תוֹסִיף לִרְאוֹתִי עוֹד עַד עוֹלָם / וַיַּךְ אֶת הַשֶּׂה וַיַּפְשֵׁט עוֹרוֹ / וַיְשַׁסְּעוֹ וַיֹּאכַל אֶת בְּשָׂרוֹ.

וְהַמָּשָׁל לְכָל הוֹלֵךְ אַחַר בִּצְעוֹ / כָּל הַגָּדוֹל מֵחֲבֵרוֹ בּוֹלְעוֹ / וְהַשֹּׁטְרִים וְהַשֹּׁפְטִים / אֲשֶׁר מִשְׁפַּט אֲנָשִׁים מַטִּים / וְכַסְפָּם מִכִּיסָם יִלְקֹטוּן / וּמִמּוּל שַׂלְמָה אֶדֶר יַפְשִׁיטוּן / וּמַאֲמַר הֶחָכָם שָׁגוּר בַּפֶּה / יֵשׁ בְּלֹא מִשְׁפָּט נִסְפֶּה.

וָאֶשָּׂא מְשָׁלִי וָאֹמַר:


פְּתָאִים אָהֲבוּ פֶתִי / וְלֵצִים חָמְדוּ לָצוֹן

וְעָשִׁיר יַעֲנֶה עַזּוֹת / לְדֹרֵשׁ שִׂפְתֵי רָצוֹן.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
