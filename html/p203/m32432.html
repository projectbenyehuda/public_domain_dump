מחרך [נגדך אשים מגמתי]

נֶגְדְּךָ / אָשִׂים מְגַמָּתִי,

כִּי לְךָ / רוּחִי וְנִשְׁמָתִי.


יָקְרוּ / עָצְמוּ פְלָאֶיךָ,

חֻבְּרוּ / אֶל עַל קְרוּאֶיךָ,

[5] בָּחֲרוּ / אוֹתְךָ בְרוּאֶיךָ,

אֶחֱזֶה בִרְאִי מְזִמָּתִי

כִּי שְׁמָךְ חֵילִי וְחוֹמָתִי.


צוֹרֲרִים / שׁוֹעִים לְתַעְתּוּעִים,

דּוֹבְרִים / כָּזָב וּבוֹ תוֹעִים

[10] אוֹמְרִים / “שָׁוְא עַמְּךָ רוֹעִים!”

וַאֲנִי – שִׁבְתִּי וְקִימָתִי

זִכְרְךָ עָזִּי וְזִמְרָתִי!


חוֹלֲלִי, / אֶעְטֹף לְפָנֶיךָ,

מַעֲלִי / יִסְלַח רְצוֹנֶךָ,

[15] מַחֲלִי / תִרְאֶה בְעֵינֶיךָ

כִּי בְיוֹם שׁוּבִי לְאַדְמָתִי

אֶקְרְאָה אָחוֹת לְרִמָּתִי.


קוֹמֲמָה, / יָהּ, בֵּיתְךָ! מֵעִיר

שׁוֹמֲמָה / חֶזְיוֹנְךָ תָעִיר,

[20] רוֹמֲמָה / וּשְפֹט בְּהַר שֵׂעִיר,

עוֹרֲרָה וּנְקֹם לְנִקְמָתִי –

תֵּן פְּאֵר תַּחַת כְּלִמָּתִי!

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
