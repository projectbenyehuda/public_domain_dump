תוכחה

סימן: אלף־בית

מז


   מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מָה אָשִׁיב עַל תּוֹכַחְתִּי


   אֵיכָה בָשָׂר וָדָם / יַעֲמֹד פְּנֵי כָל קָדַם / מוֹכִיחַ אָדָם

   בְּשׂוּמָם נֶגֶד פְּנֵיהֶם / צְפוּנֵי תַּעֲלוּמֵיהֶם / וַיּוֹכַח עֲלֵיהֶם

   גְּמוּלָם יָקִים נוֹכַח / וְיִרְאֶה זָכוּר וְנִשְׁכַּח / אֱלֹהֵי אֲבוֹתֵימוֹ וַיּוֹכַח

   מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מַה אָשִׁיב עַל תּוֹכַחְתִּי

135    דְּבֵקַת עָפָר שָׁחָה / נֶפֶשׁ לָדִין לֻקָּחָה / תִּהְיֶה בְיוֹם תּוֹכֵחָה

   הֲלֹא בַחֲזוֹתָהּ פְתוּחוֹת / מְגִלּוֹת מְדַבְּרֵי צָחוֹת / אֵין בְּפִיהָ תּוֹכָחוֹת

   וּבְעֵת מָרוֹם צָוְחָה / תִּקְרָא אַנְשֵׁי שׁוּחָה / לְכוּ נָא וְנִוָּכְחָה


   מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מַה אָשִׁיב עַל תּוֹכַחְתִּי

   זְמַן בָּא לִרְאוֹת יֶשַׁע / לְאִישׁ אֱמֶת וְעַל פֶּשַׁע / מוֹכִיחַ לָרָשָׁע

140    חַלְתִּי כִּי שָׁמַעְתִּי / גּוֹשׁוּ לָרִיב אִתִּי / שִׁמְעוּ נָא תוֹכַחְתִּי

   טָמֹן חֵטְא מַצְפּוּנֶיךָ / אָז אֲגַלֶּה וְעַל פָּנֶיךָ / אוֹכִיחֲךָ וְאֶעֶרְכָה לְעֵינֶיךָ

   מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מַה אָשִׁיב עַל תּוֹכַחְתִּי

   יַד בְּדָמִים מִלֵּאתֶם / וּמֵעֲצָתִי נְטִיתֶם / וְתוֹכַחְתִּי לֹא אֲבִיתֶם

   כַּמָּה אָטְמוּ תְנוּכָם / וְאָזְנָם מִבּוֹא בְתוֹכָם / הוֹכֵחַ לְחָכָם

145    לְכוּ נַעֲרֹךְ דִּינֵנוּ / לַמּוֹכִיחִים בֵּינֵינוּ / וְיוֹכִיחוּ בֵין שְׁנֵינוּ

   מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מַה אָשִׁיב עַל תּוֹכַחְתִּי

   מִי זֶה פִיהוּ יָרִים / לִפְנֵי דָן כָּל יְצוּרִים / וְהוֹכֵחַ בִּדְבָרִים

   נַעֲנֶה אֵיכָה יִכּוֹן / לְעֻומַּת כָּל מָכוֹן / הוֹכֵחַ בְּדָבָר לֹא יִסְכֹּן

   סִכְלֵי פְעָלִים הִשְׁחוּךְ / נַפְשִׁי וּבִשְׁאוֹל הִנְחוּךְ / וּמְשׁוּבוֹתַיִךְ תּוֹכִיחוּךְ

150    מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מַה אָשִׁיב עַל תּוֹכַחְתִּי

   עִמְדִי וְצַפִּי צִיגָתוֹ / וְעֵת לַמִּשְׁפָּט שִׁבְתּוֹ / אַל תָּקוֹץ בְּתוֹכַחְתּוֹ

   פָּנָיו חַלִּי בְּשִׂיחַ / כִּי כָל נָע יָנִיחַ / אֲשֶׁר יֶאֱהַב יְיָ יוֹכִיחַ

   צוּר הַנּוֹדָע בִּשְׁמוֹ / אֵיךְ אֱנוֹשׁ גָּבַר מוּמוֹ / יָשָׁר נוֹכַח עִמּוֹ

   מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מַה אָשִׁיב עַל תּוֹכַחְתִּי

155    קוֹרֵא בְּחִיל וְדַאֲבוֹן / בַּדִּין מְיַסֵּר כְּסִיל וְנָבוֹן / בְּתוֹכָחוֹת עַל עָוֹן

   רְאֵה שִׂיחִי וּנְחֵנִי / בִּצְדָקָה וַהֲנִיחֵנִי / וְאַל בְּאַפְּךָ תּוֹכִיחֵנִי

   תִּרְאֶה בְעָמְדִי נֶגְדֶּךָ / כִּי בְחַנּוֹתְךָ וְחַסְדֶּךָ / הוֹכַחְתָּ לְעַבְדֶּךָ

   מְאֹד חַלְתִּי וְזָחַלְתִּי / לְתוֹכַחַת לֻקַּחְתִּי / מַה אָשִׁיב עַל תּוֹכַחְתִּי
