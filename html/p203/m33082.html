פזמון [יוֹם צָעֲקוּ שָׁבִים מֵחֲטָאֵיהֶם]

סימן: יתחק

ט

         יוֹם צָעֲקוּ שָׁבִים מֵחֲטָאֵיהֶם

         חֶסֶד אֲבוֹתֵיהֶם / לָקְחוּ בִידֵיהֶם

         כְּחֵלֶב זִבְחֵיהֶם / עַל מִזְבְּחֵיהֶם

         יִשְׂאוּ בְנֵי יִשְׂרָאֵל / יַעֲקֹב אֲבִיהֶם


5        בֵּן מִהַר טִהַר / לֵבָב עָקֹב

          בֵּן בְּשׁוּבוֹ נִחַם / וְיִצְרוֹ יִקֹּב

        זְכוּת יַעֲקֹב קָרֵב / לְיַשֵּׁר עָקֹב

         רַחְמָן לֹא עַתָּה / יֵבוֹשׁ יַעֲקֹב

         בְּקָרְבָם חֶלְבֵּיהֶם / וְשָׁפְכָם דְּמֵיהֶם

10       יִשְׂאוּ בְנֵי יִשְׂרָאֵל / יַעֲקֹב אֲבִיהֶם


         בָּנִים הִתְרַחֲצוּ / וְהִטֶּהָרוּ

         אַךְ פְּנֵיהֶם מִמְּךָ / מְאֹד נֶחְפָּרוּ

         זְכוּת יַעֲקֹב לָקְחוּ / וְנֶגְדָּךְ סָרוּ

         רַחְמָן לֹא עַתָּה / פָּנָיו יֶחֱוָרוּ

15       בְּנָסְכָם דְּמֵעיהֶם / מְקוֹם נִסְכֵּיהֶם

         יִשְׂאוּ בְנֵי יִשְׂרָאֵל / יַעֲקֹב אֲבִיהֶם


         בֵּן יָחִישׁ לְפַנּוֹת / לֵבָב זוֹנֶה

         בֵּן הֲתוּלִים יְנַקֶּה / לֹא אֲלֵיהֶם פּוֹנֶה

         בֵּן נַפְשׁוֹ כַיּוֹם / בְּצוֹם יְעַנֶּה

20       בְּשֵׁם יַעֲקֹב יִקְרָא / וּבִשְׁמוֹ יְכַנֶּה

         וְנוֹשְׂאִים כַּפֵּיהֶם / נֶגֶד שְׁמֵיהֶם

         יִשְׂאוּ בְנֵי יִשְׂרָאֵל / יַעֲקֹב אֲבִיהֶם


         זְכוּת הוֹרִים נָעֲמוּ / בָּם לָךְ יִכּוֹן

         וְצִדְקוּ לוֹ יִסְכֹּן / לֹא לָךְ יִסְכֹּן

25       בֶּטַח עֵין יַעֲקֹב / בָּדָד יִשְׁכֹּן

         לֵב טָהֹר כַּיּוֹם / וְרוּחַ נָכוֹן

         חַדֵּשׁ בְּקִרְבֵּיהֶם / וְקֶרֶב בְּנֵיהֶם

         יִשְׂאוּ בְנֵי יִשְׂרָאֵל / יַעֲקֹב אֲבִיהֶם


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
