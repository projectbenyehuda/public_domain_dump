עֲטֶרֶת פָּז לְרֹאשׁ

סימן: עבדאללה כ׳צ׳יר. זה השיר חקוק על אבן לפונדק המ׳ הגביר היקר המרומם דוד מרדכי הכהן נ״ע ובנה אותו היקר הר׳ שלמה הי״ו לכבוד הוריו.


עֲטֶרֶת פָּז לְרֹאשׁ כֹּהֵן וּמִשְׂרָה / וּמִזֶּרַע יְקָר נֻצָּר וְנִבְרָא

בְּהִתְנַדֵּב יְסוֹד דָּוִד לְהָקִים / לְפֻנְדָּקוֹ כְבִנְיַן עִיר נְצוּרָה

דְּבִיר מָקוֹם נְבִיאֵנוּ יְחֶזְקֵאל / אֲשֶׁר בָנוּ אֲבוֹתָיו כַאן לְדִירָה

אֱלֹהִים חַי יְגַדֵּל שֵׁם שְׁלֹמֹה / בְּנוֹ דָוִד לְמָרְדְּכַי עֲטָרָה

לְשַׁלֵּם לוֹ וּמַשְׂכֻּרְתּוֹ שְׁלֵמָה / תְּהִי מֵאֵל עֲרוּכָה גַם שְׁמוּרָה

לְבָרֵךְ אֶת פְּרִי בִטְנוֹ וְהוֹנוֹ / עֲדֵי זִקְנָה וְשֵׂיבָה הַהֲדוּרָה

הֲלֹא בָנָה לְשֵׁם הוֹרָיו וְהִקְדִּישׁ / לִיחֶזְקֵאל בְּחִיר הָאֵל וְנוֹרָא

כְּצִיר יָבֹא אֵלִיָּהוּ בְּיָמָיו / מְבַשֵּׂר טוֹב לְקַבֵּץ שֶׂה פְזוּרָה

וְעֵת נִשְׁלַם בְאֶלֶף שֵׁשׁ וּבִשְׁנַת / [ ] לַיְּצִירָה.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
