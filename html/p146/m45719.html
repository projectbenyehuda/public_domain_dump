אִם לִבִּי

חתימה: אלשבזי


אִם לִבִּי / בְּעַרְבִי / מִמַּיִם זֵדוֹנִים

וָאֶצְמָא / לְמֵימָה / וְאֵין בִּי כֹּחַ אוֹנִים

וְעֵין שִׂכְלִי / יוֹם חֶדְלִי / תַּזְכִּירֵנִי יְגוֹנִים

רַחֲמֵנוּ / צוּרֵנוּ / כְּרַחֵם אָב עַל בָּנִים:


לְהַקְבִּיל דּוֹד / אֶתְנוֹדֵד / וְאֵין בְּפַעֲמַי כּוֹחַ

כִּי פִשְׁעִי / שָׂם צַלְעִי / כְּשֶׂה מוּכָן לִטְבּוֹחַ

בְּצַלְמָוֶת / גּוּף עִוֵּת / עֲדֵי יִדְכֶּה יָשׁוֹחַ

וְכִמְעַט אֶמְ/צָא רוּחַ / וּמָשַׁל גּוֹי עַז פָּנִים:


שְׁכִינַת אֵל / רְצִי שׁוֹאֵל / וּזְכוּת לַמְּדִי עָלָיו

וְיִתְבּוֹנֵן / לְהִתְכּוֹנֵן / בְּאוֹת אַרְבַּעַת דְּגָלָיו

בְּנִיב רַחְשׁוֹ / וְסוֹד נַפְשׁוֹ / יְוֹיָ נִצָּב עָלָיו

וּבְטוֹבַת / מַהֲלָלָיו / יְדַבֵּר שִׁיר וּרְנָנִים:


בְּאַהְבַת בַּת / נִצֶּבֶת / לְמַעְלָה מִמֶּרְכָּבָה

רְצֵה שַׁוְעִי / צוּר יִשְׁעִי / לְעַמָּךְ כֹּפֶר הָבָה

בִּזְכוּת חֵן / בְּנֵי בִטְחוֹן / דְּרוֹשׁ סְגֻלָּה נֶאְהָבָה

רְפָא נֶפֶשׁ / נֶעְלָבָה / בֵּין אוֹיְבִים חַמָּנִים:


זֶה אֵלִי / שְׁמַע קוֹלִי / לְעַמָּךְ הַמְצִיא כֹפֶר

רְפָא חוֹלֶה / שְׁהוּא גּוֹלֶה / בְּרִית אָבוֹת אַל תּוּפַר

וְשִׂים שָׁלוֹם / לְדָל חוֹלֵם / כְּמֵי גִּיחוֹן וּמֵי פַרְפַּר

לְעֶזְרָתִי / צְלַח עוֹפֶר / וְרַחֵם עָם נֶאֱמָנִים:


יְצַו חַסְדּוֹ / לִידִידוֹ / רָם וְנִשָּׂא עֲלֵי הַכֹּל

וְיוֹפִיעַ / לְהַשְׁפִּיעַ / לִבְנֵי דַּעַת וְהַשְׂכֵּל

וְטוּב מָזוֹן / לְדָל נִזּוֹן / מֵחַסְדּוֹ וּמִתְכַלְכֵּל

וְהוּא נִשְׁעָן / עֲלֵי בַת כֹּל / אֲשֶׁר הִיא אֵם הַבָּנִים:



את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
