שְׁזָפַתְנִי יְפַת תֹּאַר בְּהֵיכַל אַהֲבָה

החתימה: שלם יוסף


שְׁזָפַתְנִי יְפַת תֹּאַר / בְּהֵיכַל אַהֲבָה

רְאִיתִיהָ עֲלוֹת שַׁחַר / בְּדַעַת רָחֲבָה

וְהִיא תַּבִּיע בְּקוֹל אֹמֶר / בְּקוֹמָה נִצְּבָה:

יְדִיד נַפְשִׁי, בְּשִׁיר מַהֵר / וְאֵלַי תִּכְתְּבָה:


לְקָחַנִי הֲדַר יָפְיָהּ / בְּרֹב חָכְמָה וְסוֹד

הֲדַר מֶלֶךְ יְקַר עֶדְיָהּ / כְּמוֹ נֶצַח וְהוֹד

אֲבָל חָלַת בְּבֵית שִׁבְיָהּ / בְּגָלוּת הַנְּדוֹד

וּמַרְאֶיהָ כְּאוֹר סַהַר / לְבָבִי לִבְּבָה:


מְחֻשֶּׁקֶת בְּאַחְלָמָה / וְזָהָב פַּעֲמוֹן

יְפַת עַיִן וְלָהּ קוֹמָה / מְלֻבֶּשֶׁת בְּחֵן

לְמוּלָהּ מַלְאֲכֵי רוּמָה / יְצַחְצֵחוּ בְרוֹן

בְּקוֹל הַלֵּל וְקוֹל זֶמֶר / לְבָבִי לִהֲבָה:


יְרוּצוּן מַלְאֲכֵי מַעְלָה / לְמוּלָהּ נִכְסְפִים

וְהִיא תַנְעִים בְּקוֹל צַהְלָה / וְהֵם שָׁם שׁוֹאֲפִים

מְקֻשֶּׁטֶת כְּמוֹ כַלָּה / כְּלִילַת הַיְּפִים

הֲדָרָהּ בַּמְּרוֹם נָהַר / עֲלֵי כָל שַׂר צְבָא:


וְדוֹדָהּ וִיעֲדָהּ מֵרוֹם / בְּהַשְׁפֵּעַ וְרוֹן

וְסוֹדָהּ עַד מְאֹד עָצוּם / בְּגוֹבַהּ אִדְּרוֹן

וְתָבִין מַעֲיָן חָתוּם / וְתַשִּׁיג סַהֲרוֹן

וּמִדּוֹת תִּקְשְׁרֵם קֶשֶׁר / בְּאֵיתָן תִּשְׁלְבָה:


סְגֻלַּת כָּל בְּנֵי חָכְמָה / בְּשַׁלְמוּת נִרְקְמָה

וְתָשׁוּט עַל פְּנֵי יָמָּהּ / בְּהוֹדָהּ תִּשְׁלְמָה

וְהִיא עַלְמָה מְאֹד תַּמָּה / בְּטוֹבָה נִפְלְגָה

וְטוּבָהּ רַב וְלֹא יֶחְסַר / בְּעֵת כִּי שָׂגְבָה:


פְּתוֹר דּוֹדִי לְשִׁירִי זֶה / וְשִׂים עַל לִבְּךָ

וְהִתְבּוֹנֵן כְּמוֹ חוֹזֶה / לְדֶרֶךְ רָחֲבָה

וְהִשְׁתּוֹנֵן לְיוֹם מַסָּע / וְהָכֵן דֵּעֲךָ

וְהָכִינָה לְבוּשׁ טוֹהַר / לְמַעַן תִּרְהֲבָה:

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
