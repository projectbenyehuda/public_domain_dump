מעמד ג: עֵינִי רַב

החתימה: בן יוסף


עֵינִי רַב בִּכְיָה / מִקּוֹל הַמִּסְפֵּד

מִי יְחַזֵּק רִפְיָהּ / כִּי דִמְעָה יֵרֵד

עַל סְגֻלַּת שִׁבְיָה / הָרָב הַמְלַמֵּד

אִם עֲווֹנוֹת תִּשְׁמָר-יָהּ / יְוֹיָ מִי יַעֲמוֹד:


בִּרְכַּי יוֹחִילוּ / עַל צְנִיף תִּפְאָרָה

בְּרַעַד יֵלִילוּ / וּבֵיהוּל וּגְעָרָה

וְאֵיכָה יוֹבִילוּ / כְּלִי שֶׁלַּתּוֹרָה

לְדִירַת הֶעָפָר / כִּי בָא הַשּׁוֹדֵד:


נְעִימָה לוֹ, אִם בָּא / לִדְרוֹשׁ בָּעִנְיָן

בְּשִׂמְחָה וּבְחִבָּה / וְלֵב טָהוֹר מְצֻיָּן

וּבְנֶפֶשׁ טוֹבָה / יְתַקֵּן הַבִּנְיָן

לְתַלְמִידֵי הַדּוֹר / מְלַמֵּד וּמְחַדֵּד:


יָשָׁר הוּא וְנָבוֹן / בְּשָׁעַת תַּלְמוּדוֹ

כְּמוֹ שָׂר אוֹ רִבּוֹן / אֲדַמֶּה לִכְבוֹדוֹ

וּמִסִּיג סוֹד חֶשְׁבּוֹן / כְּאוֹר שֶׁמֶשׁ הוֹדוֹ

הוֹמֶה לִי לִבִּי / עָלָיו גַּם חָרֵד:


וּמַה-יּוֹעִיל בִּכְיִי / אַחַר אִישׁ נֶחְמָד

וּמִי יְרַפֵּא חָלְיִי / בְּשָׁעַת כִּי יִלְמָד

יְחַזֵּק אֶת רִפְיִי / וּבַתּוֹרָה נִצְמָד

וְקוֹלוֹ כַּשַּׁחַל / בְּמִדְרָשׁוֹ שׁוֹקֵד:


סְגַן הַמִּשְׁפָּחָה / וְרֹאשׁ הוּא לַחֲכָמִים

בְּעֵת כִּי יִפְתָּחָה / בְּמִדְרָשׁ וּטְעָמִים

לְבָבוֹ יְפַעֲנֵחָה / לְסוֹדוֹת נֶעְלָמִים

וְחוּט חֶסֶד מָשׁוּךְ / עֲלֵי רֹאשׁוֹ עוֹנֵד:


פְּאֵר יֶשַׁע קָשׁוּר / תָּמִיד עַל זְרוֹעוֹ

וּבְחָכְמָה יָשׁוּר / לְכָל סוֹד יַשְׁמִיעוֹ

עֲלֵי דוֹרוֹ כַּשּׁוּר / הָיָה לְהַצְנִיעוֹ

אַחֲרָיו אֵיךְ נִשְׂמַח / בְּשַׁבָּת וּבְמוֹעֵד:


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
