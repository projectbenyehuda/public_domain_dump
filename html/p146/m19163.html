אני הדל ועזי עיר ומגדל

חתימה: אלשבזי משתא (שלם) – מאה יז


אֲנִי הַדַּל / וְעָזִּי עִיר וּמִגְדַּל / וּמִבְטַחִי וְחֶשְׁקִי שֵׁם אֲדֹנָי

לְחֵן שַׁדַּי / אֲנִי נִכְסָף בְּלִי דַּי / בְּאַהְבָתִי וְשֵׂכֶל רַעֲיוֹנָי

שְׁבִיב נוּרָא / דְּמַלְכָּא אֶתְקְטִירָא / בְּתוֹךְ לִבִּי לְהָסִיר אֶת יְגוֹנָי

בְּסוֹד אַהְבַת / הֲדָסָה הַמְּכַסָּה / עֲלֵי רֹאשִׁי בְּשָׁעַת הֶגְיוֹנָי

זְכוּת עוֹרֵר / יְדִיד עַל דַּל מְשׁוֹרֵר / בְּשִׁיר חָדָשׁ לְשַׂמֵּחַ הֲמוֹנָי

יְבִיאוּנִי / בְּחֵן חַסְדֵי אֲדֹנָי / לְהַר קָדְשׁוֹ בְּיוֹם בּוֹא רֹאשׁ גְּאוֹנָי

מְחַכֶּה קוֹל / תְּרוּעָה יוֹם יְשׁוּעָה / וְשִׁבְטֵי יָהּ וְדִגְלֵיהֶם לְפָנָי

שְׁאוֹן גַּלֵּי / זְמָן אֶשְׁכַּח בְּתֵימָן / וְדוֹד נֶאְמָן / יְכַסֶּה כָּל עֲוֹנָי

תְּנִי הַלֵּל / יְחִידָתִי חֲצוֹת לֵיל / לְצוּר גּוֹלֵל / לְאוֹר לִבִּי וְעֵינָי

אֲדוֹן הַכֹּל / וְשָׂם דַּעַת וְהַשְׂכֵּל / בְּפִי רַחְשִׁי וְגַם שֵׁמַע בְּאָזְנָי

וְשִׁין לַמַדּ / וּמִים בִּשְׁמִי יְלַמֵּד / זְכוּת עָלַי וְעַל חֶבְרַת זְקֵנָי.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
