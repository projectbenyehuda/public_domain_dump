בְּיוֹם שַׁבָּת אֲהַלֵּל אֵל

חתימה: בן יוסף חזק


בְּיוֹם שַׁבָּת אֲהַלֵּל אֵל / וְאָעִיר נוֹגְנִים

וְאָשִׁיר שִׁיר בְּנוֹעַם קוֹל / לְמוּל מִתְבּוֹנְנִים:


נֻמַגִּדּ רַבְּנַא אלעַאלִי / אִלַאהּ כֻּלּ אלוֻדּוּדּ

נֻסַבִּחהוּ בְּאגּלַאלִי / קִיַאמֻן תֻ׳ם קֻעוּד

נֻוַחִדּהוּ בְּאַכּלַאמִי / וְנַעבֻּדּ פִי סֶגּוּדּ

וְמַא אַחדּ יֻטִיק יַכּמֻל / בִּתַּסבִּיח אַלעְצִ׳ים:

[נהלל את אלוהינו הנעלה / אלוהי כל האהובים

נשבחהו בהדר / עומדים ואחרמ יושבים

נייחדו בדיבור / ונעבדהו בהשתחוויה

ואין אדם יכול להיות שלם / בתשבוחות לאדיר:]


יְדִידִים זַמְּרוּ הוֹדוּ / לְאֵל עֶלְיוֹן בְּשִׁיר

בְּמַעֲשֵׂיכֶם תְּיַחֵדוּ / לְרַעְיָה עִם גְּבִיר

וּבִשְׁבָחִים תְּסַלֵּדוּ / לְבוֹרֵא כָּל יְצִיר

אֲשֶׁר הוּא מַאֲצִיל הַכֹּל / מְקוֹר כָּל נֶאְצְלִים:


וֻגִּ׳בּ תַּעזִיז יַא אַכוַאן / לְדַא אליַוְם אַלפְצִ׳יל

אַמַר פִיהּ רַבְּנַא אלרַחמַאן / בִּקַוּלֻהּ אַלגְּלִיל

עְלַא מוּסַי וְלַדּ עִמרַאן / נַבִּי זַאכִּי פְצִ׳יל

אְדִי פִי חֻלמְהוּ מֻכּמִל / רְסוּלַן תֻם כְּלִים:

[חובה לרומם, הו, אחים / את היום הנכבד הזה

בו ציווה אלוהינו הרחמן / בקולו הנעלה

על משהבן עמרם / נביא טהור ונכבד

אשר במתינותו מושלם / שליח אף דובר:]


שְׂרִידִים הַמְּיֻדָעִים / בְּנֵי תוֹרָה וְסוֹד

דְּעוּ וּתְיַחֲדוּ רֵעִים / עֲטֶרֶת עִם יְסוֹד

וְגַם כִּי בָּהּ רְמַז נָעִים / כְּלָלוֹת עִם כְּבוֹד

וּבָהּ שָׁבַת אֲדוֹן הַכֹּל / וְכֻלָּהּ אִשְׁתְּלִים:


פַרַאיִץ׳ אלעֻקוּל אַדרֻס / בִּאַסרַארֻן גַמִיק

וַאַגכֻּר תֻ׳ם אַזִיד אַחרֻס / פִי עַקְלֻן וַתִ׳יק

וַאַצפִי אלקַלבּ מִן אַלאַנשַאש / וַכֻּלּהַמֻּן וַצִ׳יק

לִאַן אַליַוְם זַכִּי צַ׳אהִל / וַכֻּלּ כֻּרְבַּה עַדִים:

[מצוות השכל אלמד / בסודות עמוקים

ואזכור אף אוסיף ואשמור / בשכל איתן

ואטהר את הלב מן הזיופים / וכל דאגה ומצוקה

כי היום טהור בוהק / וכל יגון נשלל:]


חַיַאתֻן לַאלַּאדִ׳י חַאפִט׳ / לְהַאדַ׳א יַוְמַנַא

זַכִּיֻּן מֻכּרַמַא פִי חִטּ׳ / וַיַרזֻק פִי אלתַ׳נַא

קַבּוּלֻן תֻ׳ם כַ׳יְר פַאִיץ׳ / יַכֻ׳ץ אַחבַּארַנַא

סַלאמֻן וַארדֻן כַּאלסַּיִל / מִן אלרַבּ אַל כַּרִים:

[חיים לאשר ישמור / את יומנו זה

טהור מכובד בחלק טוב / ויתפרנס בכפליים

רצון אף טוב שופע / יוענק לחכמינו

שלום בא כזרם שוטף / מן האל הנכבד:]


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
