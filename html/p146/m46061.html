שְׂרִידֵי אַב

חתימה: שלם שץ


שְׂרִידֵי אַב הֲמוֹנִים / הַטּוּ אָזְנֵיכֶם

וְשׁוּבוּ אֶל יְוֹיָ / מֵחַטֹּאתֵיכֶם

מֵאֻמּוֹת שִׁבְעִים / הוּא בָּחַר בָּכֶם

וְאִם הֵפִיץ אֶתְכֶם / בְּגָלִיּוֹתֵיכֶם

וּבְקָרְאֲכֶם אֵלָיו / יַקְשִׁיב לַחֲנַנְכֶם

וְאַתֶּם הַדְּבֵקִים / בַּיְוֹיָ אֱלֹהֵיכֶם:


לְמוּלוֹ הִזַּכּוּ / מֵעָווֹן וְאַשְׁמָה

וַעֲשׂוּ מִצְווֹתָיו / בְּטוּב נֶפֶשׁ תַּמָּה

וּמִלֵּב הָסִירוּ / כָּל עָוֶל וְעָרְמָה

וְלִיגִיעֵי כֹחַ / הוּא יִתֵּן עָצְמָה

וּדְבָרִים טוֹבִים / קְחוּ עִמָּכֶם:


מְחִילָה בַּקְּשׁוּ / וְרַחֲמִים מֵאִתּוֹ

עֲלֵי רוֹעַ לְבַבְכֶם / בְּשָׁעַת תַּאְוָתוֹ

אֲשֶׁר חָמַס וְגָזַל / וְהֶעֱלָה צֵַחְנָתוֹ

וְרָדַף אַחַר הוֹן / בְּחֵשֶׁק אַהְבָתוֹ

כִּי מֵרוֹב הֲבָלָיו / הִרְבָה מַחֲלַתְכֶם:


שְׁעוּ אֶל אֱלֹהֵיכֶם / אֲשֶׁר יָדוֹ רְחָבָה

וְחַכּוּ אֶל חַסְדּוֹ / כִּי יְשׁוּעָה קְרוֹבָה

וְלֵב זֶה הַמְעַכֵּב / עֲלֵיכֶם בְּחוֹבָה

קִלְקוּלוֹ תְרַפְּאוּ / מְהֵרָה בִּתְשׁוּבָה

בְּעוֹדְכֶם חַיִּים / הַיּוֹם כֻּלְּכֶם:


צֶדֶק וַעֲנָוָה / תְּרַפֵּא נִדְכָּאִים

וְהִיא תָאִיר לָהֶם / וְאִם הֵם חֵלְכָּאִים

וּבַיּוֹתֵר אִם הֵם / נְקִיִּים זַכָּאִים

לָכֵן הָבִינוּ / בְּיָמִים הַבָּאִים

בְּחַיֵּי עוֹלָמִים / הַמּוּכָן לָכֶם:


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
