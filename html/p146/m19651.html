אָנָה אֲבַקֵּשׁ אִישׁ נְבוֹן לַחַשׁ

חתימה: אלשבזי משתא


אָנָה אֲבַקֵּשׁ אִישׁ נְבוֹן לַחַשׁ וְצִיר / יָפִיג חֲמַת לִבִּי שְׁהוּא רוֹתֵח כְּסִיר:

לִי קַצד אִלַי אַלאַחבַּאבּ אֻבַּלִּג זִיַארְתִי / וַאלגִׄסם צַח נַאחִל וְכַׄלַּאנִי חְקִיר:

שָׁכְנִי בְעִיר דַּלִּים וְאֵין לִי עוֹזְרִים / חָלַת גְּוִיָּתִי וְלֹא אָבוֹא בְעִיר:

בִּאלפֻקד אַתּדַׄכַּר וְדַׄלַּת בְּנִיְּתִי / אַלשֻּׁגׄן צַח פִינִי וּמַא בִּי לִאלמְסִיר:

זִכְרוּ יְדִידְכֶם עֵת תְּפִלָּה אוֹהֲבַי / אוּלַי בְּרָב עָם יְהֶא לִבִּי נְהִיר:

יַחלֻם זִיַארַתכֻם צְׄמִירִי פִי אַללְּיַאל / אַלגִׄסם יַתבַּקַּא וְנַפסִי תַׄם תְצִיר:

מַר הַזְּמָן גַָּרַם וְהֵפֵר אַחֲוָה / מָכַר עֲדַת קֹדֶשׁ בְּלֹא כֶסֶף מְחִיר:

שַׁוְק אַלנֻּפוּס תַבּלֻגִ מֻנַאהַא כַּאלקְדִים / פִי חִצן בַּית אַלקֻדס

תְהַיַּא וְתַסתְנִיר:

תִּרְצֶה אֲדוֹן עוֹלָם תְּעוֹרֵר חַסְדְּךָ / תַּצִּיל סְגֻלָּתָךְ

כְּקֶדֶם תַּאֲמִיר:

יַא חַאפִץ̇ אַלאַנפֻס תַחפֻץ̇ מְכֻׄוְּתִי / דִׄי חַל מַסכִּנהֻם פִי וַאדִי אַלעְדִׄיר:

וּסְגַל לְאַבְרָהָם בְּשָׁלוֹם רָב וְטוֹב / זִכְרוּ יְעִירֵנִי בְּסוֹד טַעַם וְשִׁיר:

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
