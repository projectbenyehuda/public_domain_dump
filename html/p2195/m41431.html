אֲשׁוֹרֵר שִׁיר לְשַׂר

לכבוד שיך ששון בן צאלח דוד

סימן: אני נשׂים.


אֲשׁוֹרֵר שִׁיר לְשַׂר וְרֹאשׁ קְהִלָּה / נְשִׂיא הַדּוֹר וְהוּא נֵזֶר לְגוֹלָה

יְהִי רָצוּי כְּקָרְבָּן שַׁי וְתוֹדָה / נְאוּמִי לוֹ כְמוֹ מִנְחָה בְלוּלָה

שְׂבַע רָצוֹן יְהִי דָשֵׁן וְשָׁמֵן / יְחִי לָעַד בְּרֹב טוֹבָה כְפוּלָה

מְרוֹמָם יִהְיֶה עַל כָּל בְּרָכוֹת / וְהַצְלָחוֹת בְּרֹב שִׂמְחָה וְגִילָה

יְדוּעָה לָאֲדוֹן מֵאָז קְדוּמָה / אֲשֶׁר יֵשׁ לִי עֲלֵי הַגּוֹי עֲלִילָה

בְּבוֹאוֹ פֹה לְעִיר בָּבֶל אֲזַי שָׂר / אֲבּוּ יוֹסֵף לְבַקֵּשׁ לוֹ מְחִילָה

אֲזַי סִפֵּר וְהִכְרִיחַ לְהַגּוֹי / לְשַׁלֵּם לִי קְרוּשׁ קוּ״ף בַּחֲבִילָה

וְשַׂר אָמַר שְׁהוּא הִתְחִיל בְּמִצְוָה / וְאָדוֹן הוּא אֲשֶׁר יָדַע שְׁבִילָהּ

וְהוּא יָבִיא לְגֶמֶר טוֹב לְדָבָר / בְּחָכְמָתוֹ הָרָמָה הַגְּדוֹלָה

וְהֵן עַכְשָׁיו לְחַסְדּוֹ הַמְּרֻבֶּה / אֲנִי צוֹפֶה וְהוּא מַמְצִיא תְעָלָה

וּמִצְוָה זוֹ יְקָרַָה הִיא בְּעֵרֶךְ / וּמִי הוּא זֶה אֲשֶׁר יֵדַע גְּבוּלָהּ

וּבִשְׂכַר זֹאת אֱלֹהִים חַי יְרִימוֹ / מְאֹד יִגְבַּר וְיִתְנַשֵּׂא בְמַעְלָה

יְמַלֵּא מִשְׁאֲלוֹת לִבּוֹ לְטוֹבָה / וְלֹא יִרְאֶה לְשׁוּם צַעַר וּמַחְלָה

וְיַחְזֹר אֶל נְוֵה קָדְשׁוֹ בְּכָבוֹד / בְּרֹב כֶּסֶף וְרֹב זָהָב וְשִׂמְלָה.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
