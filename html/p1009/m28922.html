בְּנֵי תוֹרָה הֱוִי שׁוֹקֵד לְפִתְחָם

בְּנֵי תוֹרָה הֱוִי שׁוֹקֵד לְפִתְחָם / וְשָׂאָם עַל זְרוֹעוֹתֶיךָ וְקָחָם

וְשִׂים אָזְנָךְ לְמֵיטַב מַעֲנֵיהֶם / וְחִידוֹתָם וְדִבּוּרָם וְשִׂיחָם

וְסִפְרֵיהֶם תְּשֵׁנֵּן בָּם לְעוֹלָם / וְתֵדַע מַעְגְּלֵי צֶדֶק בְּלִקְחָם

וְעַתָּה נִקְבְּצוּ כָּל הַגְּאוֹנִים / קְרָשִׁים עוֹמְדִים מֵעַל בְּרִיחָם

עֲלֵיהֶם מִשְׁכְּנוֹת הָאוֹר פְּרוּשִׂים / בְּבֵית וַעַד לְרַבֵּנוּ יְרוּחָם

מְשָׁכָם הוּא בְּחַבְלֵי אַהֲבָתוֹ / וּפָתַח אוֹצְרוֹת יֵינָם וְטִבְחָם

יְרוֹחָם מֵאֲדוֹן עוֹלָם יְרֻחָם / מְאַסֵּף מַחֲנוֹת תּוֹרָה בְּכֹחָם

נְתִיבוֹתָיו בְּשֵׁם הַטּוֹב דְּרָכָם / וְאָרְחוֹתָיו בְּשֶׁמֶן טוֹב מְשָׁחָם

קְנֵה חָכְמָה קְנֵה בִינָה חֲסַר לֵב / וְסוּר הֵנָּה לְדַלְתֵי בִין וּפִתְחָם

רְאֵה נָא תּוֹלְדוֹת אָדָם וְחַוָּה / בְּנֵי נֹחַ וְשִׁנּוּ אֶת שְׁבָחָם

וְאֵין הָדָר וְתִפְאָרָה לְיֶפֶת / בְּנֵי שֵׁם הֵם אֲשֶׁר מָשְׁלוּ בְּרוּחָם

סְגֻלָּה הִיא יְקָרָה מִפְּנִינִים / בְּתוּשִׁיָּה וְאַל תִּזְכֹּר בְּנֵי חָם

נְחָלוּהָ וְנִקְרְאוּ קְדוֹשִׁים / וְאַדִּירִים אֲשֶׁר חֶפְצִי לְנִכְחָם

בְּהִלּוּכָם עֲלֵי אֶרֶץ תְּנִיחֵם / וְאִם שָׁכְבוּ בְּהָקִיץ הִיא תְשִׂיחֵם

וְסוֹף אוֹדֶה לְרוֹקֵעַ הָאֲדָמָה / וְשָׁמַיִם כְּאֹהָלִים מְתָחָם

גְּבוּרוֹתָיו לְהַתִּיר הָאֲסוּרִים / וְעֵינַיִם לְעִוְרִים הוּא פְּקָחָם

אֲקַוֶּה אֶת יְשׁוּעָתוֹ בְּקָרוֹב / וּבָהּ אָגִיל וּבָהּ נַפְשִׁי תְנֻחָם

וְאָז יִזְרַח פְּנֵי שֶׁמֶשׁ צְדָקָה / לְכָל יָרֵא וְלֹא יָבוֹא יְרֻחָם

וְחָלִילָה לְאֵל חַי מִכְּזָבִים / וְלֹא דִבַּר בְּטוֹבָה עוֹד וְנִחָם

שְׁנַת הַשְׁלַ״ט יְהֶא שׁוּלְטָן לְעַם אֵל / וּבָהּ יִגָּאֲלוּ עַל יַד מְשִׁיחָם

וְאִם אֵיחַר עֲדֵי שִׁיל״ה בְּבוֹאוֹ / וְעָם לוֹ יִקְּהַת כּוֹרְעִים בְּשִׂיחָם.
