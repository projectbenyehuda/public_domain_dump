מֵהֵר הַשִּׁיר אֶחְצֹב נִיבוֹת

מֵהַר הַשִּׁיר אֶחְצֹב נִיבוֹת / לִבְנוֹת הַשִּׁיר תָּשׁוּר בָּבוֹת

תִּשְׂמַח תָּגִיל כָּל פִּיפִיוֹת / גִּילַת רַנֵּן עַל כָּל לִבּוֹת

רָקְדוּ הָרִים הֵם כָּאֵילִים / וּבְכָל גִּבְעָה עִם מוֹשָׁבוֹת

וַיָּשִׁירוּ כָל עֵץ יַעַר / וּבְרוֹשׁ תִּדְהָר וַעְצֵי עָבוֹת

גַּם שָׁמַיִם נָתְנוּ טַלָּם / וּבְטַל אוֹרוֹת תָּמִיד זָבוֹת

עַד כִּי תוֹךְ פִּי חָזָק בַּשִּׁיר / מַעְיָן לִפְרֹץ מֵאֵן לִשְׁבּוֹת

לִשְׁבָח הַשָּׂר הָרָב יִצְחָק / עַל פֵּרוּשׁוֹ נַחְלַת אָבוֹת

כִּי רַב חֵילוֹ כִּי זַךְ שִׂכְלוֹ / בִּלְשׁוֹן צֶחָה זַךְ מַחְשָׁבוֹת

הָרִים עֹפֶל הֵרִים אֹפֶל / כָּל סוֹד נִסְתָּר שָׁם נִכְתָּבוֹת

נִדְבוֹת פִּיהוּ רָצָה הַשֵּׁם / עַד כִּי בִזְבוּל הֵם נִצָּבוֹת

לִפְנֵי שָׂרֵי כָל עַם קֹדֶשׁ / הֵם לַאְלָפִים הֵם לִרְבָבוֹת

חַכְמֵי דוֹרוֹת הָרִאשׁוֹנִים / הֵן לִשְׁאָלוֹת הֵן לִתְשׁוּבוֹת

וַיּוֹכִיחוּ מֵרוּם מַעְלָה / מַה לִּי לִדְרֹשׁ בַּעְלוֹת אוֹבוֹת

אַף כִּי מֹשֶׁה אִישׁ הָאֱלֹהִים / בֵּרַךְ יִצְחָק בְּרָכוֹת רַבּוֹת

גַּם כָּל נָשִׂיא קָם מִכִּסְאוֹ / עַל דִּבְרוֹתָיו הַנֶּאְהָבוֹת

עִם מַיְמוּנִי מוֹרֶה צֶדֶק / שִמַּח נַפְשׁוֹת הַמַּדְאִיבוֹת

מַעְיַן חָכְמָה מִפִּי יִצְחָק / הִשְׁקָה אֶרֶץ עִם תַּלְאוּבוֹת

עַד כִּי סִירִים עָשׂוּ פֶרִי / גַּם סִלּוֹנִים הַמַּכְאִיבוֹת

נָתְנוּ פִרְיָם מָתוֹק לַחֵיךְ / גַּם לִנְפָשׁוֹת הַשּׁוֹבֵבוֹת

שָׁבוּ נָחִים כִּי נִרְפָּאוּ / כִּי חֻבָּשׁוּ מִכָּל טוֹבוֹת

סִפְרֵי יִצְחָק הַמַּתְאִימוֹת / עַד תַּכְלִיתָם וּמְשֻׁלָּבוֹת

לָכֵן לִבִּי פָנָה נֶגְדָּם / לַחְקֹר בָּהֶם אַךְ חַרְצֻבּוֹת

רֶשַׁע קָמִים עַל אֵם דֶּרֶךְ / תָּמִיד חוֹרְשִׁים עַל כָּל גַּבּוֹת

אֵין פּוֹתֵחַ חַרְצֻבּוֹתָם / עַל רֹב פֶּשַׁע עַל רֹב חוֹבוֹת

דַּלּוּ עֵינַי לִמְרוֹם קָדְשׁוֹ / יַעְלִים עֵינִי מִכָּל דִּבּוֹת

עָשְׁקָה לִי אֵל גַּם עָרְבֵנִי / מֵאֵשׁ תָּפְתֶּה מִלֶהָבוֹת

וּבְגַן עֶדְנָךְ נַפְשִׁי תִּזְכֹּר / מַהֵר אָבִין אֶחְצֹב נִיבוֹת.
