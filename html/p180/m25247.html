שְׁכוּלַת בָּנִים

שְׁכוּלַת בָּנִים / מְאוּסַת בַּעַל

לְכוּדָה נְדוּדָה / דְּרוּסַת נַעַל

מָתַי תְּשִׁיבֶנָּה / לְסַף רַעַל

הָאֵל הַמּוֹדֵד / מַיִם בְּשֹׁעַל


שֹׁעַל בְּמֹעַל / אֶשָּׂא לְמַעַל

לְמַעַן כִּי אֵין / בְּיָדִי פֹעַל

מִפְעָל אֵין בִּי / אֲבָל חֵטְא וָמַעַל

הָאֵל בְּפָנַי / דֶּלֶת אַל תִּנְעַל


רְאֵה נָא עָנְיִי / בְּיַד בְּנֵי בְלִיַּעַל

וְהַעַל לְמַחֲצִי / אֲרוּכָה וְתַעַל

וּזְכֹר זְנוּחֶיךָ / וְנַחֵם אֲנוּחֶיךָ

אֱלֹהִים אַל תָּשֵׁב / פְּנֵי מְשִׁיחֶךָ


לְמָתַי תְּאַחֵר / שְׁנַת שִׁלּוּמִים

וְעַד אָן תִּמְשֹׁךְ / קֵץ נִחוּמִים

הִנֵּה בָא עֵת / וְקָרְבוּ הַיָּמִים

וְעַמְּךָ כַּמֶּה / שָׁנִים זְעוּמִים


הַבֵּט וּזְכֹר / שִׁבְעַת קְדוּמִים

וּבָאֵר סְתוּמִים / וְגַלֵּה חֲתוּמִים

קַבֵּץ נְדוּחֶיךָ / לְבֵית מְנוּחֶךָ

אֱלֹהִים אַל תָּשֵׁב / פְּנֵי מְשִׁיחֶךָ.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
