מְלִיצֵי אַהֲבָה גָּרֵי לְבָבוֹת / לְהוֹרוֹת אֹהֲבִים דִּינֵי אֲהָבוֹת

בְּעֵת חֶבְרָה יְשׂוּשׂוּן אַךְ בְּלִבּוֹת / לְקִנְאַת הַזְּמָן הָרָע עֲצוּבוֹת

חֲזֵם יִתְנוֹדֲדוּ לִזְמָן בְּטֶרֶם / יְפַלֵּס לַנְּדֹד כַּיּוֹם נְתִיבוֹת

וְיֶהְמֶה יַם-בְּכִי לִפְנֵי יְדִידִים / וְלַהַב הַתְּשׁוּקָה תּוֹךְ קְרָבוֹת

וְזֶה יִסְעַר וְזֶה יִבְעַר בְּלֵב סַר / וְיוֹבִישׁ נוֹזְלֵי דֶמַע בְּבָבוֹת

לְךָ הַשָּׂר אֲשֶׁר אָסַף תְּבוּנוֹת / וְחָכְמוֹת כֶּאֱסֹף בֵּיצִים עֲזֻבוֹת

לְךָ חָרְדוּ לְבָבוֹת מִצְּלָעוֹת / וְאַנְחוֹתָם לְחַלְמִישׁ צוּר מְדִיבוֹת

וְאִם הִתְפָּֽרְדוּ הַיּוֹם גְּוִיּוֹת / הֲלֹא הִתְלַכְּדוּ הַמַּחֲשָׁבוֹת

וְאֵיכָה יִֽחְיוּ עוֹד אַחֲרֵי זֹאת / פְּגָרִים נוֹתְרוּ מֵאֵין לְבָבוֹת

וְאֵיךְ בָּכוּ תְמוֹל שׁוֹאַת זְמַנִּים / בְּהַטּוֹתָם מְתֵי-אַהַב לְאֵיבוֹת

וְהֵן נוֹסַף נְדֹד עַל-שֽׂנְאֵיהֶם / וְנָתַן מִשְׁכְּנוֹת חֶבְרָה חֳרָבוֹת

וְאָמְנָם כָּל-יְמֵי אַהַב עֲמָלִים / וְקָרְאוּ אִישׁ אֲהָבוֹת אִישׁ דְּאָבוֹת

יְמֵי חֶבְרָה בְּאֵשׁ-אַהַב מְשִׂיקִים / וְלֵילוֹת נֹד בְּמַר וּנְהִי מְעִיבוֹת

וְלָכֵן אֶפְּלָה-נָּא כָּל-יְמוֹתַי / בְּיַד חֶבְרָה לְרָעוֹת אוֹ לְטֹבוֹת

וְהֵן זַעַף נְדֹד אֶשָּׂא וְאוֹחִיל / לְשׁוּב יָמִים וְלֵילוֹת מִמְּשׁוּבוֹת

וְיוּנַף מִשְּׁמֵי חֶבְרַת יְדִידִים / אֲשֶׁר יֵעָֽצְרוּ גֶּשֶׁם נְדָבוֹת.
