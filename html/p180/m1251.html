אדני מה אדם

אֲדֹנָי מָה אָדָם / ר' שלמה אבן גבירול

אֲדֹנָי, מָה אָדָם? / הֲלֹא בָשָׂר וָדָם!

יָמָיו כְּצֵל עוֹבֵר / וְלֹא יֵדַע בְּנוּדָם,

פִּתְאֹם יָבֹא אֵידוֹ / וְיִשְׁכַּב וְיֵרָדֵם.

אֲדֹנָי, מָה אָדָם? / גֵּו נִרְפָּשׂ וְנִרְמָס,

מָלֵא תּוֹךְ וּמִרְמָה / וְגַם אָוֶן וְחָמָס,

כְּצִיץ נֹבֵל יָצָא ‑ / וְחַם הַשֶּׁמֶשׁ וְנָמָס.

אִם תִּפְקֹד עָלָיו / עֲוֹנוֹ הַנִּכְמָס, ‑

אַפְּךָ וְגַם קֶצְפְּךָ / אֵיךְ יִשָּׂא וְעָמָס?

לָכֵן חֲמֹל וּגְמֹל / כִּי אֵין לְאֵל יָדָם.

אֲדֹנָי, מָה אָדָם? / בַּטִּיט יִתְגֹּלֵל,

בְּשֶׁקֶר יִתְהֹלֵל / וּבְשָׁוְא יִתְהַלֵּל.

הֲטָהוֹר מִטָּמֵא / וְיָקָר מִזּוֹלֵל?

אִם תִּפְקֹד עָלָיו / עֲוֹן יֵצֶר מְעוֹלֵל

כְּמוֹ עֵשֶׂב יִיבָשׁ, / וּכֶחָצִיר יְמוֹלֵל:

לָכֵן חֲמֹל וּגְמֹל / חֶסֶד בְּיוֹם אֵידָם.

אֲדֹנָי, מָה אָדָם? / יָהִיר לֹא נִחָם,

מֵי חָמָס יִשְׁתֶּה / וְלֶחֶם רַע יִלְחָם,

וְכַיָּם הוּא נִגְרָשׁ / וְכַתָּנּוּר יֵחָם.

אִם תִּפְקֹד עָלָיו / חֵטְא אֲשֶׁר בּוֹ יוּחַם –

יִסָּפֶה כְּחַלָּשׁ / עִם גִּבּוֹר נִלְחָם.

לָכֵן חֲמֹל וּגְמֹל / וְכַפֵּר מַעְבָּדָם!

אֲדֹנָי, מָה אָדָם? / רַק נִתְעָב וְנֶאֱלָח,

כָּל‑היּוֹם יִבְגֹּד / וּפָנָיו בְרַע יִשְׁלָח.

אִם תִּפְקֹד עָלָיו ‑ / יְהִי נִרְפָּשׂ וְנִדְלָח,

וְכַבֶּגֶד יִבְלֶה / וּכְעָשָׁן נִמְלָח.

לָכֵן חֲמֹל וּגְמֹל / וְכַפֶּר‑נָא וּסְלַח

לְשֹׁכְנֵי בָתֵי‑חמֶר / אֲשֶׁר בֶּעָפָר יְסוֹדָם!

אֲדֹנָי, מָה אָדָם? / והוּא כָּעֵץ נִקְסָס,

בְבוֹא אַמַּת בִּצְעוֹ / יְהִי כְּמוֹ‑קַשׁ מְרֻסָּס

וְיִבְכֶּה מֵאֲשֶׁר שָׂשׂ / יוֹם יִרְקַב כְּמוֹ‑סָס,

אִם תִּפְקֹד עָלָיו / עֲוֹנוֹ הַנִּכְסָס –

כְּמוֹ שַׁבְּלוּל יַהֲלֹךְ / וּכַדּוֹנַג נִמְסָס,

לָכֵן חֲמֹל וּגְמֹל / וְאַל תְּדַקְדֵּק בַּעֲדָם!

אֲדֹנָי, מָה אָדָם? / לְבַד נִדָּף כְּעָלֶה,

בְּמֹאזְנַיִם לַעֲלוֹת / בַּהֶבֶל יְסֻלֶּה,

וְכִכְלוּב מָלֵא עוֹף / כֵּן מִרְמָה מָלֵא

אִם תִּפְקֹד עָלָיו / וְעַל לִבְּךָ יַעֲלֶה –

כְּמוֹ עָשָׁן יִכְלֶה / וּכְרָקָב יִבְלֶה.

לָכֵן חֲמֹל וּגְמֹל / כְּחַסְדְּךָ, לֹא כְחַסְדָּם!
