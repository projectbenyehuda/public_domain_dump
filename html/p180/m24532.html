בַּאֲפֻדַּת חֹשֶׁן

אֱמֶת מַה נֶּהְדַּר כֹּהֵן גָּדוֹל בְּצֵאתוֹ מִן הַקֹּדֶשׁ


בַּאֲפֻדַּת חֹשֶׁן נִקְבַּעַת / בְּאַבְנֵי טוּרִים אַרְבַּעַת

בְּנֹעַם אֹדֶם רְאוּבֵן לִטְעֹן / בְּנֹגַהּ פִּטְדָּה לְהַזְכִּיר שִׁמְעוֹן

בִּיפִי בָרֶקֶת לְלֵוִי צְמוּדָה / בְּיִקְרַת נֹפֶךְ לְהַזְכִּיר יְהוּדָה

בְּשֶׁפֶר סַפִּיר לְדָן מַפְלִיא / בְּשֶׁזֶף יַהֲלוֹם לְהַזְכִּיר נַפְתָּלִי

בְּלַהַט לֶשֶׁם לְגָד מְאַשֵּׁר / בְּלֶקַח שְׁבוֹ לְהַזְכִּיר אָשֵׁר

בְּמִלּוּאֵי אַחְלָמָה לְיִשָּׂשׂכָר יַאֲהִילוּן / בְּמַרְאֵה תַרְשִׁישׁ לְהַזְכִּיר זְבוּלוּן

בְּהַדְרַת שֹׁהַם לְיוֹסֵף מַזְמִין / בְּהִלַּת יָשְׁפֵה לְהַזְכִּיר בִּנְיָמִין

בְּקֶרֶן אֵפוֹד מְלַמֵּד לְהוֹעִיל / בְּקוֹל פַּעֲמוֹנִים עַל שׁוּלֵי הַמְּעִיל

בְּטֹהַר כְּתֹנֶת כְּצִיץ הַחוֹנֵט / בְּטֶכֶס אֲפֻדַּת הָאַבְנֵט

בְּנִקְיוֹן מִצְנֶפֶת כְּנִיצוֹץ הַלַּהַב / בְּנֹעַם צִיץ הַזָּהָב.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
