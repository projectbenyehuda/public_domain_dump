שְׁעֵה נִיבִי

שְׁעֵה נִיבִי / בְּהֵיטִיבִי / שִׁגְיוֹנוֹתַי וְהֶגְיוֹנָי

לְמַכְאוֹבִי / צְרִי תָבִיא / וְתַקְרִיב יוֹם פִּדְיוֹנָי

לְהוֹשִׁיבִי / בַּת נְדִיבִי / בְּאַרְמוֹנַי כְּקַדְמוֹנָי

מְחַץ רָבִי / וְסָרָבִי / וְעַקְרַבִּי וְסַלּוֹנָי

הֱיֵה סְבִיבִי / וְרִיב רִיבִי / וְהָשֵׁב גְּמוּל גְּאֵיוֹנָי

שְׁלַח אָבִי / מְשׂוֹשׂ לְבָבִי / הַמְבַשֵּׂר אֶבְיוֹנָי

אֵלִיָּה / הַנָּבִיא / לִפְנֵי בּוֹא יוֹם יְיָ


לְךָ הָאֵל / אֲנִי שׁוֹאֵל / וְגַם אֶפְרֹשׂ לְךָ יָמִין

קְרֹץ אַדְבְּאֵל / וּמַגְדִּיאֵל / וְכָל מַכְזִיב בְּךָ יַאֲמִין

וְשׁוֹבֵב [אֶ]ל / עִיר אֲרִיאֵל / בְּנֵי יְהוּדָה וּבִנְיָמִין

וְחִישׁ גּוֹאֵל / לְיִשְׂרָאֵל / וְקָרֵב קֵץ הַיָּמִין

וּמִיכָאֵל / וְגַבְרִיאֵל / יַעַמְדוּ בְּדוּכָנָי


מְעַד חוֹמְסַי / וְחֵיל בּוֹשְׁסַי / כְּטִיט חוּצוֹת הָדֵ[ק רְקַע

וְלִרְמוּסַי / וְלִרְפוּ]סַי / יָתֵד נֶאֱמָן תִּתְקַע

[וְאוֹ]ר חֲמוּסַי / וְנֵר עֲמוּסַי / כַּשַּׁחַר יִבָּקַע

וּמַכְעִיסַי / וּמוֹ[אֲ]סַי / שְׁחוֹחַ יֵלְכוּ לְפָנָי

הֲדַר יִשַׁי / נְשִׂיא אִישַׁי / כְּאוֹר נוֹצֵץ בְּאַפִּרְיוֹן

וְנֶעֱנָשַׁי / וְנֶאֱנָשַׁי / בְּיָמָיו יִמְצְאוּ פִדְיוֹן

וְנֶחְבָּשַׁי / וְנִכְבָּשַׁי / בְּרִנָּה יַעֲלוּ צִיּוֹן

וְיוּבַל שַׁי / לְמִקְדָּשַׁי / בְּהִגָּיוֹן וְשִׁגָּיוֹן

וְאֶל רָשַׁי / וְחַלָּשַׁי / שְׁלַל מוֹנַי וְשַׁלְמוֹנָי.


את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
