[אל יצחק השר – שיר־אזור]

מַה־לָּךְ תֵּלֶךְ־מָר / דּוֹדִי וְעִמִּי יַיִן חָמָר.


יִתְהַפֵּךְ עֵינוֹ / בַכּוֹס אֱלֵי שִׁבְעָה עֵינָיִם

חַלָּשׁ דִּמְיוֹנוֹ / כִּכְפִיר בְּבוֹאוֹ תּוֹךְ מֵעָיִם

נִלְעָג וּבְעֵינוֹ / יִלְעַג לְחַרְטֻמֵּי מִצְרָיִם

כִּלְיוֹ לֹא הוּמָר / עַל־כֵּן צְבִי רֵיחוֹ לֹא נָמָר.


יֵעוֹר מֵחֶבְיוֹן / כַּדּוֹ, וְלַהַב מָשׂוֹשׂ יָעִיר

נַפְשִׁי לוֹ פִדְיוֹן / מִיקוֹד יְגוֹנִים מִלֵּב יַבְעִיר

יָעֹז לֵב אֶבְיוֹן / אַחַר שְׁתוֹתוֹ מִלּוֹכֵד עִיר

אַךְ בּוֹ תִתְיַמָּר / כָּל־עוֹד אֱלֹהִים רוּחָךְ שָׁמָר.


בִּימֵי שַׂר אוֹתוֹ / בָחַר אֱלֹהִים מִכָּל‏־עַמִּים

תֵּאוֹר מִשְׂרָתוֹ / שִׁבְעָה כְּאוֹר שִׁבְעַת הַיָּמִים

הִבְדִיל אֵל בֵּיתוֹ / לִהְיוֹת לְפָנָיו בֵּית עוֹלָמִים

חָסִיד אִם אָמָר / חֶסֶד מְהֵרָה אֹמֶר גָּמָר.


הוּא יִצְחָק הַשָּׂר / אַגָּן אֲשֶׁר מֶזֶג לֹא יֶחְסָר

אַךְ מָזַג מוּסַר / הַשְׂכֵּל לְחַיּוֹת בּוֹ כָּל־בָּשָׂר

הַהוּא שֶׁנִּמְסַר / לָאֵל לְבַדּוֹ לִהְיוֹת טִפְסָר

וּבְחַרְבּוֹ זָמָר / כָּל־זֵד עֲלֵי עַמּוֹ יִתְאַמָּר.


הָאָרֶץ כֻּלָּם / נֶגְדּוֹ בְּקוֹל רִנָּה יִצְהָלוּ

וּנְזִירֵי עוֹלָם / מִגִּיל נְזִירוּתָם יִגְעָלוּ

וּכְסוֹבְאִים קוֹלָם / יִשְׂאוּ וְעַל־שֵׁכָר יִשְׁאָלוּ

סכראן יא עיאר; / ואין אלטריק אלי דאר אלכמאר.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
