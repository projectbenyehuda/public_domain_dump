קחה השיר

ולו ירחמהו האל. עוד לו


קְחָה הַשִּׁיר וּמַצְפּוּנָיו / וְדִקְדּוּקָיו וְעִנְיָנָיו

עֲלֵה וַעֲמֹד בְּאֶשְׁנַבָּיו / וְהִתְיַצֵּב בְּחַלּוֹנָיו

וְרַחֵף עַל פְּנֵי מוּסָר / כְּמוֹ נֶשֶׁר עֲלֵי בָנָיו

מְאַס כָּל שִׁיר וְשִׁפְלוּתוֹ / וְהִתְרַהֵב בְּעֶלְיוֹנָיו

בְּחַר הַשִּׁיר וְהַדָָּבָר / אֲשֶׁר דָּבוּר עֲלֵי אָפְנָיו

אֲדוֹן נַפְשִׁי, / יְהוּ לִבּוֹ / וְקַשּׁוּבוֹת יְהוּ אָזְנָיו

לְהָבִין אֶת מְלִיצָתִי / וְגַם שִׁירִי וְתַחְנוּנָיו

אֲשֶׁר יִדְמֶה אֱלֵי מֶלֶךְ / וְהוּא דֶגֶל עֲלֵי פָנָיו

וְאָז כָּל שָׁר לְךָ יַבִּיט / וְאָז יִשָּׂא לְךָ עֵינָיו

וְיֹאמְרוּ: הֵן כְּתָמָר זֶה / וּמִי יֹאחֵז בְּסַנְסִנָּיו?

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
