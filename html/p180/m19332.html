[חֲגֹר הַנִּיב אֵזוֹר]

ולה פי אלחץּّ עלי טלב אלעלם אלמנטקי [ולו בעוררו לדרוש את חכמת ההגיון]


חֲגֹר הַנִּיב אֵזוֹר / וְהַדִּבֵּר שִׁרְיָן

וְשִׂים הַשֵּׂכֶל עֵד / וְהַמַּדָּע דַּיָּן

הֲכִי נִבְחָר מִפָּז / וְטוֹב מִכָּל־קִנְיָן

וְאִם־תָּבִין תַּבְנִית / וְתָכִין הַבִּנְיָן

וְתַשְׂכִּיל הַמּוֹפֵת / אֲשֶׁר אֵין־בּוֹ מִדְיָן

תְּגַלֶּה כָל־עָמֹק / וְתִמְצָא כָל־עִנְיָן.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
