[חקר היטב ידידי יגעני]

ומן אנט ͘ ם תצניפה וארשק נחתה ותרציפה [ומן הפיוטיים שבחיבוריו והמהודרים שבעיצוביו וריצופיו]


È — — — / È — — — / È — — / / È — — — /È — — — / È — —

חֲקֹר הֵיטֵב יְדִידִי יִגְּעַנִי / מְצֹא פֶלֶס לְאַהְבָתְךָ וּמַעְגָּל

עֲדֵי שַׁרְתִּי בְּחָקְרִי הָאֲהָבִים / עֲדִינָה יָֽשְׁבָה אֶצְלִי כְּשֵׁגָל

וְאֹמֶרֶת וְעַד-אָן אַהֲבָתְךָ / מְסֻתֶּרֶת וְלֹא-תֵרָא וְתִגָּל

וְהַקָּצִיר כְּבָשַׁל יִקְצְרֻהוּ / וְשִׁית חֶרְמֵשׁ עֲלֵי קָמָה וּמַגָּל

וּבֶן-יִשַׁי בְּעֹצֶם אַהֲבָתוֹ / הֲלֹא שָׁלַח וְקָרָא לַאֲבִיגָל.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
