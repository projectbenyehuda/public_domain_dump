[הֲלֹא צִוָּה]

ולה איצ ͘א

È — — — / È — — — / È — — / / È — — — /È — — — / È — —

הֲלֹא צִוָּה אֲבִי כָל-הַחֲכָמִים / בְּסִפְרוֹ אַל-תְּהִי אִישׁ רִיב וּמָדוֹן

וְאַתָּה רִיב וּמִדְיָנִים תְּחַרְחַר / וְתַרְבֶּה חֵטְא וְגַם-פֶּשַׁע וְזָדוֹן

יְרָא מֵחֵץ מְאֹד שָׁחוּט וְשָׁנוֹן / וְגוּרָה מִלְּהַב חֶרֶב וְכִידוֹן

אֱהִי עַבְדָּךְ שְׁלוֹמִים אִם-תְּבַקֵּשׁ / וְאִם-לָרִיב אֱהִי לָךְ-שַׂר וְאָדוֹן.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
