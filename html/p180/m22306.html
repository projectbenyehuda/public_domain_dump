כאין נשאו

כְּאַיִן נִשְּׁאוּ מֵרֹב חֲלָקָיו / בְּחַלּוֹתוֹ פְּנֵי רֵעָיו חֲשׁוּקָיו

וְרָאוּ בַחֲצִי יוֹמָם דְּמוּתוֹ / וְעַל כֵּן יֹאמְרוּ: קַו אַחֲרֵי קָו

יְדִידַי, שַׁאֲלוּ אֵלָיו שְׁנָתִי / וְאוּלַי בַחֲלוֹם אֶרְאֶה בְרָקָיו

בְּעוֹד שֶׁמֶשׁ בְּעֵינִי מַחֲמַד בּוֹ / וְטֶרֶם יִכְלוּ מָחָר יְרָקָיו

וְנִבְכֶּה עַל יְמֵי נֹעַר כְּסוֹחֵר / אֲשֶׁר יִבְכֶּה עֲלֵי רֹעַ שְׁוָקָיו

יְרַוּוּ אַרְצְכֶם עָבֵי שְׁחָקִים / כְּמוֹ לִבִּי אֲשֶׁר יִרְוֶה מְצוּקָיו

וְאַל נָא תִשְׁאֲלוּ גִשְׁמֵי נְדָבוֹת / לְעָב, כִּי שַׁעֲלֵי יִצְחָק מְרִיקָיו

אֲשֶׁר לוּ שָׂם בְּיוֹם הַחֹם שְׁעָלָיו / לְמוּל סֶלַע, עֲדֵי יִפְתַּח נְקִיקָיו

תְּמִים דַּעַת, יְקַר נֶפֶשׁ, טְהָר לֵב, / בְּעָבְרוֹ בַזְּמַן הֵרַךְ חֲזָקָיו

וְלוּ עִמּוֹ יְרוּצוּן כּוֹכְבֵי רוֹם / אֲזַי לֹא דָרְכוּ רֵאשִׁית אֲבָקָיו

אֲשֶׁר עָשַׁק לְבַב חַכְמֵי לְבָבוֹת / וְהָיִיתִי אֲנִי מֵרֹאשׁ עֲשׁוּקָיו

וְקָשַׁר הַזְּמַן חֶפְצִי בְיָדוֹ / וְאָסַר כַּאֲנִי קוֹשֵׁר חֲלָקָיו

נְתָנַנִי אֲסִיר תִּקְוָה וְשֵׂבֶר / וּמִי יִתֵּן וְאֶהְיֶה מִנְּתוּקָיו

וְלֹא חָמַל עֲלֵי רֵעַ, לְבָבוֹ / כְּלֵב אֵַיָּל אֲשֶׁר יָבְשׁוּ אֲפִיקָיו

כְּבָר הָיָה תְמוֹל קָרוֹב לְנַפְשִׁי / וְהַיּוֹם גָּבְהוּ מֶנִּי שְׁחָקָיו

וְהַיּוֹם זָרְחָה שֶׁמֶשׁ וְנוֹדַד / וְלֹא הָיָה יְדִידִי מִיְּלָקָיו

וְאֵיךְ יָמֹד אֱלֵי חֵיק אַהֲבָתִי / בְּמִדַּת פּוֹחְזֵי שֵׂכֶל וְרֵקָיו

וְהוּא הַתָּם אֲשֶׁר גָּבַר בְּשִׂכְלוֹ / וְרִפָּה בַזְּמַן מֵזַח אֲפִיקָיו

וְהַצֶּדֶק וְהַחֶסֶד דְּבֵקָיו / וְהַדַּעַת וְהַמּוּסָר מְנִיקָיו

וְחָלִילָה הֱיוֹת כִּי אִם כְּרֵעַ / גְּמוּל הָאַהֲבָה שִׁלֵּם, וְחֻקָּיו

אֲהַלֶּלְךָ וְאַתְּ מַהְלַל אֲמָרַי / וְצַוָּאר טוֹב יְיַפֶּה אֶת עֲנָקָיו

וּמִי הוּא זֶה אֲשֶׁר יַעְרֹךְ וְיָשִׁיב / עֲלֵי מִשְׁקָל אֲמָרַי וַעֲלֵי קָו

וְהַסּוּס בַּקְּרָב יִגְאֶה לְבָבוֹ / וְאִם יוּכַל חֲמוֹר לָרוּץ בְּשׁוֹקָיו.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
