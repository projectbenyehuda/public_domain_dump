לְךָ יְיָ הַצְּדָקָה… מְעוֹנָה

לְךָ יְיָ הַצְּדָקָה / כִּי יְמִינְךָ פְשׁוּטָה לְקַבֵּל שָׁבִים

- - - - - - -

לְךָ יְיָ הַצְּדָקָה / מְעוֹנָה אֱלֹהֵי קֶדֶם

הוּא הַשֵּׁם הַנַּעֲרָץ / אֲשֶׁר הָיָה מִקֶּדֶם

הוּא קַדְמוֹן לְכָל קַדְמוֹן / וֵאלֹהִים מִקֶּדֶם

הוּא רִאשׁוֹן לְכָל רִאשׁוֹן / מֵאַחֲרִית וּמִקֶּדֶם

הוּא נָאוֹר טֶרֶם אוֹר / הֲלֹא אַתָּה מִקֶּדֶם

הוּא יָחִיד וְנִשְׂגָּב שְׁמוֹ / כִּי אֶזְכְּרָה מִקֶּדֶם

הוּא שִׁקַּע מוּצָק שְׁאוֹל / וְנוֹצֵץ אוֹר מִקֶּדֶם

הוּא הִסְגִּיר בְּאֵין סַגְרִיר / אֲשֶׁר הָיָה מִקֶּדֶם

הוּא עוֹלָם בְּאֵין עוֹלָם / גַּם אַחֲרוֹן מִקֶּדֶם

הוּא מָקוֹם לְכָל מָקוֹם / וּמוֹצָאוֹתָיו מִקֶּדֶם

הוּא שִׁפְרֵר חוּג עֲלִיָּה / וּמֵרֹאשׁ הַרֲרֵי קֶדֶם

בְּחוּג נִשְׁקָף הַנִּקָּף / גַּן בְּעֵדֶן מִקֶּדֶם

רְאֵה עֲנִיִּים הַשְּׁבוּיִים / זְכֹר עֲדָתָךְ קָנִיתָ קֶדֶם

אֵל יוֹשֵׁב הַכְּרוּבִים / גּוֹמֵל טוֹב לְחַיָּבִים.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
