שְׁלַח מַלְאָךְ מֵלִיץ

שְׁלַח מַלְאָךְ מֵלִיץ / וְהַפְלֵא אוֹת וְסִבָּה

לְטַהֲרֵנוּ בְלֵיל זֶה / מִשֶּׁמֶץ וְדִבָּה

הָאֵל הַגָּדוֹל / לְאֵין חֵקֶר וְקִצְבָּה

צִדְקָתְךָ כְּהַרְרֵי אֵל / מִשְׁפָּטֶיךָ תְּהוֹם רַבָּה


לְךָ גָלוּי וְצָפוּי / יֵצֶר לֵב וְסוֹדוֹ

הֵן בְּעָווֹן חוֹלָל / וְאֵיךְ יִצְדַּק בְּעָמְדוֹ

הֲיִצְדַּק אָבָק דַּק / וְהֶבֶל נֶחְשַׁב בְּעוֹדוֹ

וְאַף כִּי יֹאבַד וְיָשׁוּב / כָּל יְסוֹד אֶל יְסוֹדוֹ

יוֹם יְסֹעֵר כַּמֹּץ / וּכְעָשָׁן מֵאֲרֻבָּה


מִי יָקוּם לְעַמְּךָ / וּמִי יַעֲמֹד לְהַרְוִיחֵם

אִם לָרִיב תִגַּשׁ / וְאִם לְמִשְׁפָּט תִּקָּחֵם

אָנָּא שְׁפֹט כְּצִדְקְךָ / וְאַל בְּאַפְּךָ תוֹכִיחֵם

כִּי לֹא יוּכַל חַלָּשׁ / עִם גִּבּוֹר לְהִלָּחֵם

וְאֵיךְ יוּכַל לַעֲמֹד / קַשׁ יָבֵשׁ בְּלֶהָבָה


הֵן כְּמוֹ צִיץ יִמָּל / וּבָרוֹחַ יִבְרַח כַּצֵּל

וְהַבָּשָׂר וְהָרוּחַ / זֶה מִזֶּה יֵאָצֵל

וְאִם תָּעִיר פְּקֻדָּה / לֹא יוּכַל לְהִנָּצֵל

וְאִם תִּדְחַק הַרְבֵּה / וְהַפּוֹעֵל מְאֹד עָצֵל

וְגַם הַיּוֹם קָצַר / וְהַמְּלָאכָה מְרֻבָּה.

את הטקסט[ים] לעיל הפיקו מתנדבי פרויקט בן־יהודה באינטרנט.
